>

## 一、认知台阶链

学生从「知道 c=n/V」到「能完成 w↔c↔b↔x 之间的任意换算」，需要跨过 4 级台阶：

| 台阶 | 学生现状 | 目标状态 | 关键跨越 |
|:---:|:---|:---|:---|
| 1 | 只知道质量分数 w 和物质的量浓度 c | 认识四种浓度表示法：w（质量分数）、x（摩尔分数）、b（质量摩尔浓度）、c（物质的量浓度） | **理解"为什么需要四种"**：w 直观但不反映反应计量；c 方便取液但随温度变；b 不随温度变适合理论讨论；x 适合混合体系 |
| 2 | 浓度换算靠背公式 | 理解 c=1000ρw/M 的每一步推导，知道 ρ（密度）是连接质量世界和体积世界的桥梁 | **从定义出发推导一次**：c=n/V → n=m/M → m=w·m液 → m液/V=ρ → c=1000ρw/M。重点标注每一步的单位消去 |
| 3 | 稀释 = "加水" | 稀释是 n 不变、V 变的过程：c₁V₁ = c₂V₂。注意「加 1dm³ 水」≠「定容至 1dm³」 | **反例**：50% NaOH (ρ=1.525) 稀释到 1dm³——加水体积 ≠ 1000−V原，因为体积不等于可加性 |
| 4 | 浓度和温度无关 | b、w 不随温度变化；c 随温度变化（溶液体积热胀冷缩） | **数值对比**：同一溶液在 10°C 和 20°C 的 c 值略有不同——引出什么时候必须用 b 而不是 c |

> ⚠️ **台阶 2 是第一轮基础班的计算密集区**：c=1000ρw/M 的推导涉及 4 步代数变换，每一步都有单位。学生容易在「密度放分子还是分母」「乘 1000 还是不乘」上出错。

## 二、关键认知冲突

### 冲突 1：「加 1 dm³ 水」≠「定容至 1 dm³」（发生率 ~40%）

**制造方式**：题目说「配制 1 mol·dm⁻³ 的 NaCl 溶液 1 dm³」——学生直接称 58.5g NaCl，加 1 dm³ 水。但实际上应该先溶解，再定容至 1 dm³。

**突破策略**：对比两种操作的结果差异——「加 1 dm³ 水」得到的体积 > 1 dm³（NaCl 占据体积），浓度 < 1 mol·dm⁻³。口诀——「定容不是加水，是补到刻度线」。

### 冲突 2：c=1000ρw/M 中「1000」从哪来（发生率 ~30%）

**制造方式**：让学生自己推导 c=ρw/M，然后把单位代进去——发现 ρ 是 g·cm⁻³，c 是 mol·dm⁻³，cm³→dm³ 差 1000 倍。

**突破策略**：板书分步推导，每一步标注单位：
```
c = n/V = (m/M)/V = (w·m液/M)/V = w·(m液/V)/M = w·ρ/M
但 ρ 单位是 g·cm⁻³，c 单位是 mol·dm⁻³ → 乘 1000
∴ c = 1000ρw/M
```

### 冲突 3：四种浓度——「为什么不能只用一种？」（发生率 ~25%）

**制造方式**：给同一个溶液（如 10% NaCl），让学生分别用 w、c、b、x 表示。「为什么 w 和 c 不能直接反映反应计量？为什么 b 不随温度变？」

**突破策略**：对比表——w 直观但无量纲；c 方便取液但温度敏感；b 不随温度变但需要密度数据才能和 c 互算。**没有一种浓度是万能的，选哪种取决于你接下来要做什么。**

## 三、例题选用指南

| 序号 | 例题 | 教学功能 | 何时用 | 来源 |
|:---:|:---|:---|:---|:---|
| 1 | 浓硫酸标签解读：98%, ρ=1.84 g/cm³ → 求 c | **首次推导 c=1000ρw/M**：从标签信息出发，不直接给公式 | 讲完四种浓度定义后 | 教材 |

## 四、高频错误模式

| 错误 | 典型表现 | 发生率 | 根源 | 预防策略 |
|:---|:---|:---:|:---|:---|
| **加水 vs 定容混淆** | 配制溶液时「加入 1 dm³ 水」而不是「定容至 1 dm³」 | ~40% | 日常语言习惯：「加水到 1 升」vs 化学操作「定容至 1 升」 | 口诀「定容不是加水，是补到刻度线」+ 演示两种操作的结果差异 |
| **c=1000ρw/M 中 1000 位置搞错** | 1000 放在分母或忘乘 | ~30% | cm³↔dm³ 换算不熟练，1000 的来源不理解 | 板书分步推导 + 每步标注单位消去 |
| **密度数据单位不匹配** | ρ 用 g/cm³ 但 V 用 m³，算出差 10⁶ 倍 | ~20% | 密度有多种单位（g/cm³, kg/m³, g/mL），不同公式需要不同单位 | 强制写「ρ = ___ g·cm⁻³ → 代入 c=1000ρw/M → c = ___ mol·dm⁻³」 |
| **b 和 c 混淆** | 用 c 代替 b 代入依数性公式（ΔT=Kb·b，不是 Kb·c） | ~25% | b 和 c 的符号相似，公式中的 b = mol/kg 不是 c = mol/L | 依数性公式中的 b 用「质量摩尔浓度」全称而非缩写 |
| **稀释计算中体积单位不统一** | c₁V₁=c₂V₂ 中 V₁ 用 mL、V₂ 用 L 直接代入 | ~15% | 跳步：不注意单位一致性 | 强制写「V₁ = ___ dm³, V₂ = ___ dm³」再代入 |

## 五、有效类比与口诀

### 类比库

| 概念 | 类比 | 使用场景 |
|:---|:---|:---|
| 四种浓度 = 四种语言 | 「w 像中文——日常最直观（质量比）；c 像英语——实验室通用；b 像数学公式——严谨但抽象；x 像联合国投票——每人一票（按摩尔数）」 | 首次介绍四种浓度时 |
| 密度 ρ = 浓度换算的桥梁 | 「ρ 是把质量世界和体积世界连起来的汇率——你知道一种货币（w），要通过汇率（ρ）才能换成另一种货币（c）」 | 推导 c=1000ρw/M 时 |
| 定容 vs 加水 | 「定容是倒水到刻度线，不是用量筒量好了倒进去——因为溶质也占体积」 | 配制溶液例题时 |

### 口诀库

| 口诀 | 用途 |
|:---|:---|
| 「定容是补到刻度线，不是量好了倒进去」 | 配制操作的核心纠正 |
| 「c = 1000ρw/M，ρ 是 g/cm³，c 是 mol/dm³」 | 浓度换算公式的单位匹配 |
| 「b 不随温度变，c 热胀冷缩」 | 区分 b 和 c 的温度依赖性 |
| 「w 看质量，c 看体积，b 看溶剂，x 看摩尔」 | 四种浓度的核心区分 |
