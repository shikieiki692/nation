# 真断链处理清单 v1

> 目的：把 [[2026-06-07-断链检测]] 中 `2298` 处“真断链”压缩为可执行处理流，而不是继续停留在总数层。
> 当前阶段不追求“一次性清零”，而是优先把高频、可批量、对当前知识库使用价值高的部分先分桶。

---

## 一、处理原则

1. **先修会批量降噪的项**：优先处理能一次性消掉一串红链的目标。
2. **先修导航层，再修内容层**：很多真断链不是缺 KP，而是缺聚合页、导航页、标准别名。
3. **先修当前业务主线会用到的概念**：第四轮冲刺、有机深化、酸碱/相图/配位等当前高频模块优先。
4. **真缺页要慎建**：只有在“现有库里确实没有可承接页面”时，才进入新建候选。

---

## 三、建议执行顺序

### 第一批：一周内最值得处理

1. A 桶 `KP-` 前缀残留
2. B 桶 决赛要求 / 决赛编号导航缺口
3. C 桶 当前高频常用名 alias：
   - `红外光谱`
   - `Claisen缩合`
   - `Grignard反应`
   - `乙酰乙酸乙酯合成法`
   - `两性物质`
   - `弱酸 / 弱碱 / 酸碱强度`
4. D 桶 当前备课主线相关概念：
   - `炔负离子`
   - `七大晶系`
   - `临界点`

### 第二批：适合做成小专项

1. 有机总类名聚合：
   `醇 / 醛 / 酮 / 羧酸 / 羰基化学`
2. 相图与依数性概念：
   `三相点 / 临界点 / 凝固点降低 / 沸点升高 / 组分数 / 自由度`
3. 分析化学方法名：
   `配位滴定 / 标准曲线法`

## 四、对现有知识库的直接好处

1. **把维护工作从“看见 2298 很焦虑”变成“知道先修哪几十个最值”**
2. **能显著提高断链审计的可操作性**：以后不是只看总数，而是看四桶分别下降多少
3. **能降低 Agent 误判**：很多链接本来不是缺内容，而是缺 alias / 导航 / 标准名
4. **能提升备课与答疑效率**：当前高频概念一旦补齐，课件、提炼、日志里的跳转会顺很多
5. **能为季度审计提供稳定维度**：以后季度审计可以直接比较 A/B/C/D 四桶变化，而不是重新从头解释断链来源

---

## 五、下一步建议

如果继续推进，最优先的不是再读完整报告，而是：

1. 开一个 **A 桶专项**：批量清理 `KP-` 前缀残留
2. 开一个 **B 桶专项**：补 `决赛要求-*` 与 `决赛编号` 导航页
3. 开一个 **C 桶专项**：批量补第一批高频 alias
4. 再对 D 桶做“是否独立建页”的小范围评估

> 建议把后续执行拆成 3 张任务卡，而不是一张“大断链处理”总卡。
