## 一、课前须知：本专题在练什么

> 你在 Round 1-3 学会了**每个公式单独用**。本专题教你**把五个公式串联起来**。
> 这是 2025 年竞赛物化计算的核心能力——不再是"代入求答案"，而是"从 A 推到 D"。

**本课五条计算链**：

| 计算链 | 桥梁公式 | 2025 真题对应 |
|:---|:---|:---|
| **路径设计** | 状态函数与路径无关→设计可逆多步→分段加和 | 初赛热力学（过冷/过热相变） |
| **E° → K → 浓度** | $\Delta G^\circ = -nFE^\circ = -RT\ln K$ | 初赛第 9 题（普鲁士蓝电极+比容量） |
| **BDE 循环** | ET($E^\circ$) + PT($\text{p}K_\text{a}$) + solv = BDE | 2024 决赛第 4 题（N-H 键活化） |
| **Arrhenius + 速率** | $\ln(k_2/k_1) = -(E_a/R)(1/T_2-1/T_1)$ → $t = f(k)$ | 初赛动力学（温度→反应时间） |
| **材料计算** | 比容量 $C = nF/M$ (mAh/g) | 初赛第 9 题（普鲁士蓝储钠） |

### 2.2 热力学路径设计

#### 三步法模板

```
场景：非标准条件下的相变或过程（如 -10°C 结冰、110°C 蒸发）

Step 1: 升温/降温到平衡温度（能发生相变的 T）
  ΔH₁ = ∫C_P dT     ΔS₁ = ∫(C_P/T) dT = C_P ln(T₂/T₁)

Step 2: 在平衡温度进行相变/反应
  ΔH₂ = ΔH_trs      ΔS₂ = ΔH_trs/T_trs

Step 3: 升温/降温回到目标温度
  ΔH₃ = ∫C_P dT     ΔS₃ = C_P ln(T₁/T₂)

总: ΔX_total = ΔX₁ + ΔX₂ + ΔX₃
```

**关键原理**：$U, H, S, G$ 是状态函数→只取决于始态和终态→与路径无关。所以可以设计**任意可逆路径**来算——只要始态和终态对就行。

#### 过冷水结冰完整示例（-10°C 水→冰）

```
已知: ΔH_fus(0°C) = 6.01 kJ/mol
      C_P(H₂O, l) = 75.3 J/mol·K
      C_P(H₂O, s) = 37.7 J/mol·K

Step 1: H₂O(l) -10°C → 0°C
  ΔH₁ = 75.3 × 10 = 753 J
  ΔS₁ = 75.3 × ln(273/263) = +2.81 J/K

Step 2: H₂O(l) → H₂O(s) at 0°C
  ΔH₂ = -6010 J
  ΔS₂ = -6010/273 = -22.01 J/K

Step 3: H₂O(s) 0°C → -10°C
  ΔH₃ = 37.7 × (-10) = -377 J
  ΔS₃ = 37.7 × ln(263/273) = -1.40 J/K

总计:
  ΔH = 753 - 6010 - 377 = -5634 J
  ΔS = 2.81 - 22.01 - 1.40 = -20.60 J/K
  ΔG = ΔH - TΔS = -5634 - 263×(-20.60) = -216 J  ← ΔG<0 → 自发！
```

### 2.3 BDE 热化学循环（2024 决赛新题型）

#### 核心原理

**问题**：X-H 键的均裂能（BDE）不能直接测量。
**解决方案**：拆成三步可测量的步骤→循环闭合求和。

```
        X-H ───BDE───→ X• + H•     （不能直接测）
         │               ▲
         │ ① ET          │ ③ solv
         ▼               │
      X-H⁺• + e⁻ ──② PT──→ X• + H⁺ + e⁻

BDE = ΔG°_ET + ΔG°_PT + ΔG°_solv
```

**三步公式**：

| 步 | 过程 | 公式 | @298K 简化 |
|:---:|:---|:---|:---|
| ① ET | X-H → X-H⁺• + e⁻ | $\Delta G^\circ_\text{ET} = +nFE^\circ$（氧化！注意符号） | $= 96.485 \times E^\circ$ kJ/mol |
| ② PT | X-H⁺• → X• + H⁺ | $\Delta G^\circ_\text{PT} = 2.303RT \cdot \text{p}K_\text{a}$ | $= 5.71 \times \text{p}K_\text{a}$ kJ/mol |
| ③ solv | H⁺ + e⁻ → H• | $\Delta G^\circ_\text{solv}(\text{H}^\bullet)$（**题目给定**） | 在 MeCN 中 ≈ 218 kJ/mol |

> [!info]- 最关键的陷阱：$E^\circ$ 的符号！
> Nernst 方程中 $E^\circ$ 是**还原**电位（X⁺ + e⁻ → X）。
> 但在 BDE 循环的 ET 步，方向是 X-H → X-H⁺• + **e⁻**（**氧化**！）
> 所以 $\Delta G^\circ_\text{ET} = +nFE^\circ$（正号！不是 $-nFE^\circ$）
>
> **检查方法**：氧化 = 失去电子 = 需要能量 → $\Delta G_\text{ET} > 0$。如果算出负值→符号错了。

#### 2024 决赛 N-H BDE 计算

```
已知 (in MeCN):
  E°(ArNH₂⁺•/ArNH₂) = +0.85 V vs Fc⁺/Fc
  pKa(ArNH₂⁺•) = 5.2
  ΔG°_solv(H•) = 218 kJ/mol

① ET: ΔG° = +1 × 96.485 × 0.85 = +82.0 kJ/mol
② PT: ΔG° = 5.71 × 5.2 = +29.7 kJ/mol
③ solv: ΔG° = +218 kJ/mol

BDE(N-H) = 82.0 + 29.7 + 218 = 329.7 kJ/mol
```

### 2.4 E° → ΔG° → K → 浓度 全链

#### 五步执行流程

```
Step 1: 计算电池标准电动势
  E°_cell = E°_cathode - E°_anode

Step 2: E° → ΔG°
  ΔG° = -nFE°_cell
  n = 转移电子数（配平后两半反应电子的最小公倍数）

Step 3: ΔG° → K
  ln K = -ΔG°/RT  或  K = exp(-ΔG°/RT)
  @298K: lg K = nE°_cell/0.0591

Step 4: K → ICE 表
  写平衡表达式 → 代入初始浓度 → 设 x → 解方程

Step 5: 利用近似（如果 K >> 1 或 K << 1）
  K >> 1 → 反应几乎完全 → [反应物]_eq ≈ 0
  K << 1 → 反应几乎不发生 → [产物]_eq ≈ 0
```

#### 示例：Fe³⁺ + Cu → Fe²⁺ + Cu²⁺

```
已知: E°(Fe³⁺/Fe²⁺) = +0.771 V, E°(Cu²⁺/Cu) = +0.337 V
      初始 [Fe³⁺] = 0.10 M, [Cu²⁺] = 0.010 M, 过量 Cu(s)
      反应: 2Fe³⁺ + Cu ⇌ 2Fe²⁺ + Cu²⁺

Step 1: E°_cell = 0.771 - 0.337 = 0.434 V
Step 2: ΔG° = -2 × 96485 × 0.434 = -83.7 kJ/mol
Step 3: ln K = 83700/(8.314×298) = 33.8 → K = 4.8×10¹⁴
        (或用: lg K = 2×0.434/0.0591 = 14.7 → K = 10¹⁴·⁷)

Step 4: K = [Fe²⁺]²[Cu²⁺]/[Fe³⁺]² = 4.8×10¹⁴
        ICE: [Fe³⁺] = 0.10-2x, [Fe²⁺] = 2x, [Cu²⁺] = 0.010+x

Step 5: K >> 1 → x ≈ 0.05 → [Fe³⁺]_eq = √(0.10²×0.06/4.8×10¹⁴)
        ≈ 2.5×10⁻⁹ M （几乎完全反应）
```

### 2.5 Nernst 方程多场景——$Q$ 的四种写法

| 场景 | $Q$ 的形式 | Nernst 方程 | 竞赛出现 |
|:---|:---|:---|:---|
| **标准** | $Q = \Pi a_i^{\nu_i}$ | $E = E^\circ - (RT/nF)\ln Q$ | 所有电化学题的基础 |
| **pH 依赖** | $Q$ 含 $[\text{H}^+]^m$ | $E = E^\circ - (0.0591 \cdot m/n)\text{pH}$ @298K | 酸性高锰酸钾、重铬酸钾的氧化性 |
| **浓差电池** | $Q = c_2/c_1$（相同电极） | $E = (RT/nF)\ln(c_2/c_1)$ | 浓度对电动势的影响 |
| **沉淀/配合物影响** | $Q$ 中含 $[\text{M}^{n+}]$ = $f(K_\text{sp})$ 或 $f(K_f)$ | $E = E^\circ - (RT/nF)\ln(1/K_\text{sp})$ | $\text{AgCl/Ag}$ vs $\text{Ag}^+/\text{Ag}$ |

**关键区分**：
- 298K 简化：$E = E^\circ - (0.0591/n)\lg Q$（自然对数→常用对数）
- 非 298K：必须用完整形式 $E = E^\circ - (RT/nF)\ln Q$，$T$ 不能约掉！
- 生化教材 0.0651：因为 $T=310$K（人体温度），不是印刷错误！

### 2.6 材料计算速查

| 物理量 | 公式 | 单位 | 关键 |
|:---|:---|:---|:---|
| **比容量** | $C = \dfrac{nF}{M} = \dfrac{n \times 96485}{M \times 3600}$ | mAh/g | $n$ = 每摩尔活性物质转移的**电子数**：LiFePO₄ → $n=1$；普鲁士蓝储 Na⁺ → $n$ 取决于嵌入 Na⁺ 数 |
| **能量密度** | $E_\text{density} = C \times V$ | Wh/kg | $V$ = 电池工作电压（≈ $E^\circ_\text{cell}$ 近似） |
| **储氢密度** | $\rho_\text{H} = m_{\text{H}_2}/m_\text{total} \times 100\%$ | wt% | 区分"材料的储氢量"和"系统的储氢量" |
| **粒径（BET）** | $d = 6/(\rho S)$ | nm | $S$ = 比表面积 m²/g, $\rho$ = 密度 g/cm³ |

### 2.7 Arrhenius + 速率积分

#### 温度→速率常数

$$\ln\frac{k_2}{k_1} = -\frac{E_a}{R}\left(\frac{1}{T_2} - \frac{1}{T_1}\right)$$

**典型竞赛场景**：已知 $T_1$ 时的 $k_1$（或 $t_{1/2}$、完成时间）→升温到 $T_2$ → 求新条件下的 $k_2$ 和完成时间。

```
流程：
  T₁, t₁（或 k₁） → Arrhenius → k₂ → 速率积分 → t₂

一级反应: t = ln(c₀/c)/k
         t₁/t₂ = k₂/k₁（同一转化率下）
```

#### 示例

```
某一级反应在 25°C 半衰期 30 min, Ea = 85 kJ/mol。
求 45°C 时完成 99% 转化所需时间。

Step 1: ln(k₂/k₁) = -(85000/8.314)(1/318 - 1/298)
        = -10223 × (-0.000211) = +2.15
        k₂/k₁ = 8.6

Step 2: 25°C: t₁/₂ = 30 min → k₁ = ln2/30 = 0.0231 min⁻¹
        45°C: k₂ = 8.6 × 0.0231 = 0.199 min⁻¹

Step 3: 99% → ln(c₀/c) = ln(100) = 4.605
        t = 4.605/0.199 = 23.1 min
```

### 例题 3（⭐⭐⭐ 基础）：Nernst——pH 对氧化性的影响

**题目**：$E^\circ(\text{MnO}_4^-/\text{Mn}^{2+}) = +1.51$ V（酸性条件，$[\text{H}^+] = 1$ M）。
半反应：$\text{MnO}_4^- + 8\text{H}^+ + 5e^- \rightarrow \text{Mn}^{2+} + 4\text{H}_2\text{O}$

(1) 写出 Nernst 方程。
(2) 当 pH = 5 时，$E$ 下降多少？（假设 $[\text{MnO}_4^-] = [\text{Mn}^{2+}] = 1$ M）

**解答**：
(1) $E = E^\circ - \dfrac{0.0591}{5}\lg\dfrac{[\text{Mn}^{2+}]}{[\text{MnO}_4^-][\text{H}^+]^8}$

(2) $[\text{H}^+] = 10^{-5}$ → $E = 1.51 - \dfrac{0.0591}{5}\lg\dfrac{1}{(10^{-5})^8}$
   $= 1.51 - \dfrac{0.0591}{5} \times 40 = 1.51 - 0.473 = 1.037$ V

**下降了 0.47 V！** → pH=5 时高锰酸钾氧化性显著减弱。

### 例题 4（⭐⭐⭐ 基础）：Arrhenius——升温加速

**题目**：某药物降解为一级反应，25°C 时有效期（降解 10%）为 24 个月。$E_a = 100$ kJ/mol。若不慎存放在 40°C 环境中，有效期变为多少？

**解答**：
- $k_2/k_1 = \exp[-\frac{100000}{8.314}(\frac{1}{313} - \frac{1}{298})] = \exp(1.93) = 6.9$
- 保质期 $\propto 1/k$ → $t_2 = 24/6.9 = 3.5$ 个月

**24 个月的药→40°C 下 3.5 个月就失效！**

### 例题 5（⭐⭐⭐⭐ 拓展）：三层联立——$E^\circ + K_\text{sp} + K_f$

**题目**：已知 $E^\circ(\text{Ag}^+/\text{Ag}) = +0.799$ V。
(1) $K_\text{sp}(\text{AgCl}) = 1.8 \times 10^{-10}$ → 求 $E^\circ(\text{AgCl}/\text{Ag}, \text{Cl}^-)$
(2) $K_f(\text{Ag(NH}_3)_2^+) = 1.7 \times 10^7$ → 求 $E^\circ(\text{Ag(NH}_3)_2^+/\text{Ag})$

**分析框架**：
- (1) AgCl/Ag 电极：$[\text{Ag}^+]$ 由 $K_\text{sp}$ 决定→ $[\text{Ag}^+] = K_\text{sp}/[\text{Cl}^-]$
  $E^\circ(\text{AgCl}/\text{Ag}) = E^\circ(\text{Ag}^+/\text{Ag}) + 0.0591\lg K_\text{sp} = 0.799 - 0.577 = 0.222$ V
- (2) $\text{Ag(NH}_3)_2^+$/Ag：$[\text{Ag}^+] = [\text{Ag(NH}_3)_2^+]/(K_f[\text{NH}_3]^2)$
  $E^\circ = 0.799 + 0.0591\lg(1/K_f) = 0.799 - 0.427 = 0.372$ V

**规律**：沉淀/配合物→降低游离 $[\text{Ag}^+]$→降低电极电势（更难还原）。

  1. 循环闭合是否画对
  2. 氧化步符号是否正确
  3. 单位和数量级是否自检

---

## 五、本节形成的核心结论

1. **$\Delta G^\circ$ 是万能转接头**：左边接电化学（$-nFE^\circ$），右边接平衡（$-RT\ln K$），上边接热力学（$\Delta H^\circ - T\Delta S^\circ$）。任何一个数据入口都能打通全图。
2. **路径设计 = 状态函数性质 + 可逆路径**：始态终态相同→$\Delta S$ 相同（不管真实过程多不可逆）。设计可逆路径分段算→加和。
3. **BDE 循环三公式**：ET（$+nFE^\circ$，氧化→正号！）+ PT（$5.71 \times \text{p}K_\text{a}$）+ solv（题目给定）。循环必须闭合。
4. **Nernst 方程的 $Q$ 决定一切**：pH→$[\text{H}^+]^m$/沉淀→$K_\text{sp}$/配合物→$K_f$——所有非标准态的修正都体现在 $Q$ 的表达式中。
5. **Arrhenius 方程是"时间机器"**：已知一个温度下的速率→算出活化能→预测任何温度下的速率和完成时间。

## 六、⚠️ 易错点预警

| 错误想法 | 为什么错 | 正确理解 |
|:---|:---|:---|
| "$\Delta G^\circ = -nFE^\circ$ 永远是对的" | 方向！$\Delta G^\circ = -nFE^\circ$ 用于**还原**反应（得电子）。如果是氧化反应→$\Delta G^\circ = +nFE^\circ$ | 先判断反应方向：电池放电=自发=$\Delta G<0$=电动势 $E>0$→$\Delta G^\circ = -nFE^\circ$ |
| "$E^\circ$ 可以直接加减" | $E^\circ$ 不是状态函数！不能像 $\Delta G^\circ$ 那样加减 | 只能 $\Delta G^\circ$ 加减：$\Delta G^\circ_3 = \Delta G^\circ_1 + \Delta G^\circ_2$ → $E^\circ_3 = (n_1E^\circ_1 + n_2E^\circ_2)/n_3$ |
| "比容量 $n$ = 化学计量系数" | LiFePO₄ → FePO₄ + Li⁺ + e⁻，$n=1$（一个 Li⁺ 一个 e⁻）。但 Na₂FePO₄F 中 Na⁺=2→$n$ 可能是 2 也可能是 1——看实际可逆脱嵌 | $n$ = 实际转移的电子数，需要从充放电反应判断 |
| "298K 下 Nernst 系数永远是 0.0591" | 0.0591 = $(RT/F)\ln 10$。只有 $T=298$K 时！非 298K → 必须用完整式 $RT/nF$ | 生化教材 0.0651 是因为 $T=310$K（37°C）——不是印刷错误！ |
| "绝热反应 ΔH = 0" | 不是！绝热 = Q = 0。$\Delta H = Q_P$ 只在恒压无非体积功时成立 | 绝热反应温度用 $\Delta U \approx 0$（恒容）或设计等温反应+升温路径使总 $\Delta H \approx 0$ |
