# 第一轮结构化学初步 · 章后复习

> 本复习课面向第一轮结构化学初步全部 6 节新课完成后的学生。目标不是重讲新内容，而是把原子、分子、晶体、配位四层结构语言串成一条线，帮助你建立做题时的判断框架。

> 对应讲义：原子结构、元素周期律、分子结构基础、晶体学基础、晶体结构基础、配位化合物基础（均为超级充实版）

---

## 学习目标

完成本节复习后，你应该能够：

1. 从一道结构题中快速判断"这题在考哪一层结构语言"——原子层、分子层、晶体层还是配位层
2. 用统一的"中心原子电子排布 -> 杂化/轨道 -> 构型 -> 性质"链路解决跨模块综合题
3. 熟练运用本章最核心的 5 张速查表
4. 识别并避开本章最高频的 8 个陷阱
5. 明确第一轮结构化学与第二轮/第三轮深化内容的边界

---

## 一、全章结构总览

第一轮结构化学初步的 6 节课，本质上是在回答一个问题：**原子是怎么组成物质的？**

答案按尺度从小到大分四层：

| 尺度 | 核心问题 | 对应章节 | 核心工具 |
|:---|:---|:---|:---|
| 原子层 | 电子在原子中如何排布？ | 原子结构、元素周期律 | 量子数、Slater规则、周期律四趋势 |
| 分子层 | 原子之间如何成键？分子是什么形状？ | 分子结构基础 | Lewis结构、VSEPR、杂化、MO |
| 晶体层 | 大量原子/离子如何排列成固体？ | 晶体学基础、晶体结构基础 | 晶胞、堆积、Bragg方程 |
| 配位层 | 金属离子如何与配体结合？ | 配位化合物基础 | 价键理论、晶体场理论、CFSE |

这四层不是孤立的。做题时，你需要先判断题目在考哪一层，再调用对应的工具。

## 二、原子层核心速查

### 2.1 量子数规则

| 量子数 | 符号 | 取值范围 | 物理意义 |
|:---|:---|:---|:---|
| 主量子数 | n | 1, 2, 3, ... | 电子层，决定主要能量 |
| 角量子数 | l | 0, 1, ..., n-1 | 轨道形状（s/p/d/f） |
| 磁量子数 | m_l | -l, ..., 0, ..., +l | 轨道空间取向 |
| 自旋量子数 | m_s | +1/2 或 -1/2 | 电子自旋方向 |

合法判断：n > 0，0 <= l <= n-1，|m_l| <= l，m_s = +/-1/2。

### 2.2 电子排布三原则

| 原则 | 内容 | 易错点 |
|:---|:---|:---|
| 构造原理 | 按能级由低到高填充（1s, 2s, 2p, 3s, 3p, 4s, 3d, ...） | 填充顺序和书写顺序不同！书写按 n 排序：3d 在 4s 前 |
| Pauli不相容 | 同一轨道最多2个电子，且自旋相反 | 不是"同一能级"，是"同一轨道" |
| Hund规则 | 等价轨道先单占、自旋平行再配对 | Cr: [Ar]3d54s1（半满稳定），不是 3d44s2 |

### 2.3 离子失电子顺序

失电子先失 n 最大的轨道。Fe 先失 4s 再失 3d：

  Fe: [Ar]3d6 4s2
  Fe2+: [Ar]3d6（失4s2）
  Fe3+: [Ar]3d5（再失1个3d）

### 2.4 Slater规则求有效核电荷

对于指定电子，屏蔽常数 sigma = 其他电子对该电子的屏蔽贡献之和。有效核电荷 Z* = Z - sigma。

简化规则（sp 区）：
- 同组电子：每个贡献 0.35（1s 组内贡献 0.30）
- (n-1) 层电子：每个贡献 0.85
- (n-2) 及更内层：每个贡献 1.00

对于 d/f 电子：同组 0.35，内层全部 1.00。

### 2.5 周期律四趋势

| 性质 | 同周期（左到右） | 同族（上到下） | 反常点 |
|:---|:---|:---|:---|
| 原子半径 | 减小（Z*增大） | 增大（n增大） | IIIA < IIA（s2到p1，新增p轨道） |
| 第一电离能 | 增大 | 减小 | IIA > IIIA，VA > VIA（p3半满稳定） |
| 电负性 | 增大 | 减小 | — |
| 电子亲和能 | 增大（总体） | 减小 | VIA > VIIA（F原子小，电子间排斥大） |

重元素反常（6s区）：相对论效应导致 6s 轨道收缩。Au 金黄色、Hg 低熔点、Tl+/Pb2+ 稳定（惰性电子对效应）。

## 三、分子层核心速查

### 3.1 结构预测工具链（顺序不可颠倒）

  Step 1: 画 Lewis 结构
      -> 算总价电子（阴离子加、阳离子减）
      -> 端基原子优先满足八隅体
      -> 形式电荷最小化，选最优共振式

  Step 2: VSEPR 判断构型
      -> 写出 AXmEn（m=配体数，n=孤对数）
      -> 根据 m+n 确定电子对几何
      -> 孤对电子对键角有压缩作用

  Step 3: 杂化类型（由构型反推）
      -> m+n=2: sp（直线）；3: sp2（平面三角）；4: sp3（四面体）
      -> 5: sp3d（三角双锥）；6: sp3d2（八面体）

  Step 4: 需要 MO 吗？
      -> 双原子分子的键级/磁性 -> 用 MO 能级图
      -> 多原子分子的结构判断 -> 等电子体更快

### 3.2 VSEPR 速查表

| m+n | 电子对几何 | 孤对效应 | 典型例子 |
|:---|:---|:---|:---|
| 2 | 直线形 | 无 | CO2, BeCl2 |
| 3 | 平面三角形 | 1对孤对 -> V形（~120度 -> <120度） | SO2, O3 |
| 4 | 四面体 | 1对 -> 三角锥（~109.5度 -> ~107度）；2对 -> V形（~104.5度） | NH3, H2O |
| 5 | 三角双锥 | 孤对优先占赤道向！ | SF4(跷跷板), ClF3(T形), XeF2(直线) |
| 6 | 八面体 | 1对 -> 四方锥；2对 -> 平面正方形 | BrF5, XeF4 |

### 3.3 MO 能级图（第二周期双原子分子）

两类能级顺序：

| s-p混杂程度 | 能级顺序 | 代表分子 | 记忆口诀 |
|:---|:---|:---|:---|
| 显著（能级差小） | pi_2p < sigma_2p | B2, C2, N2 | "硼碳氮，pi在前" |
| 弱（能级差大） | sigma_2p < pi_2p | O2, F2 | "氧氟，sigma在前" |

键级 = (成键电子数 - 反键电子数) / 2。键级越大，键越短越强。

未成对电子数 > 0 -> 顺磁性；= 0 -> 抗磁性。

### 3.4 八隅体例外

| 类型 | 特征 | 实例 |
|:---|:---|:---|
| 缺电子 | <8 电子 | BF3, BeCl2（可接受配体） |
| 扩展八隅体 | >8 电子（第三周期+） | PCl5(10e), SF6(12e) |
| 奇电子 | 奇数电子 | NO, NO2 |

### 3.5 离域 pi 键

形成三条件：(a) 参与原子共平面；(b) 各有垂直于平面的平行 p 轨道；(c) 电子数 m < 2n（n 为参与原子数）。

表示：Pi_n^m（n=原子数，m=电子数）。

典型：NO3-, CO32-, BF3 均为 Pi_4^6。

### 3.6 等电子体原理

相同通式 AXm + 相同价电子总数 -> 相同结构。这是结构预测的最快路径。

| 等电子体组 | 价电子数 | 结构 |
|:---|:---|:---|
| CO2, N2O, N3-, OCN- | 16 | 直线形 |
| SO42-, PO43-, ClO4- | 32 | 正四面体 |
| CO32-, NO3-, BF3 | 24 | 平面三角形 |
| NH3, H3O+ | 8 | 三角锥 |

## 四、晶体层核心速查

### 4.1 七大晶系与晶胞参数

| 晶系 | 晶胞参数特征 | 对称性要求 |
|:---|:---|:---|
| 立方 | a=b=c, alpha=beta=gamma=90度 | 最高 |
| 四方 | a=b!=c, alpha=beta=gamma=90度 | |
| 正交 | a!=b!=c, alpha=beta=gamma=90度 | |
| 六方 | a=b!=c, alpha=beta=90度, gamma=120度 | |
| 三方 | a=b=c, alpha=beta=gamma!=90度 | |
| 单斜 | a!=b!=c, alpha=gamma=90度!=beta | |
| 三斜 | a!=b!=c, alpha!=beta!=gamma | 最低 |

### 4.2 金属堆积与配位数

| 堆积方式 | 配位数 | 空间利用率 | 晶胞中原子数 | 典型金属 |
|:---|:---|:---|:---|:---|
| 简单立方（sc） | 6 | 52% | 1 | Po |
| 体心立方（bcc） | 8 | 68% | 2 | Na, K, Fe, W |
| 面心立方（fcc, ccp） | 12 | 74% | 4 | Cu, Ag, Au, Al |
| 六方最密堆积（hcp） | 12 | 74% | 2 | Mg, Zn, Ti |

fcc 和 hcp 都是最密堆积，空间利用率相同（74%），区别在于堆积层的重复方式：fcc 为 ABCABC，hcp 为 ABAB。

### 4.3 典型离子晶体结构

| 结构类型 | 晶胞特征 | 阴离子堆积 | 阳离子填充 | 典型例子 |
|:---|:---|:---|:---|:---|
| NaCl型 | 面心立方 | ccp | 八面体空隙（全部） | NaCl, MgO, FeO |
| CsCl型 | 简单立方 | 简单立方 | 立方体空隙（全部） | CsCl, CsBr |
| 立方ZnS型 | 面心立方 | ccp | 四面体空隙（一半） | ZnS, ZnO, GaAs |
| 六方ZnS型 | 六方 | hcp | 四面体空隙（一半） | ZnS(纤锌矿) |
| CaF2型 | 面心立方 | 简单立方(阳离子) | 全部四面体空隙(阴离子) | CaF2, BaF2, UO2 |
| TiO2(金红石) | 体心四方 | — | 八面体空隙（一半） | TiO2, SnO2, MgF2 |

### 4.4 Bragg方程

  2d sin(theta) = n lambda

  d = 晶面间距，theta = 入射角（掠射角），n = 衍射级数，lambda = X射线波长

应用：由衍射角计算晶面间距，进而确定晶胞参数。

### 4.5 晶胞计数法则

原子在晶胞中的贡献：
- 顶点：1/8
- 棱：1/4
- 面心：1/2
- 体心：1

## 五、配位层核心速查

### 5.1 配合物组成

配合物 = [内界] 外界

内界 = 中心离子（通常是过渡金属）+ 配体（提供孤对电子或 pi 电子）

关键概念：配位数、配位原子、单齿/多齿配体、螯合物

### 5.2 晶体场分裂

八面体场：5个 d 轨道分裂为 t2g（低能3重简并）和 eg（高能2重简并）。

  t2g: dxy, dxz, dyz（瓣在轴之间，排斥小）
  eg: dz2, dx2-y2（瓣沿轴指向配体，排斥大）

  E(eg) = +0.6 Delta_o，E(t2g) = -0.4 Delta_o

四面体场：分裂模式相反，Delta_t 约等于 4/9 Delta_o。

平面四方场：dx2-y2 能量最高。

### 5.3 高自旋 vs 低自旋

| 条件 | 自旋态 | d电子排布策略 | 磁性 |
|:---|:---|:---|:---|
| Delta < P（弱场） | 高自旋 | 先单占所有轨道再配对 | 顺磁性强 |
| Delta > P（强场） | 低自旋 | 先填满 t2g 再填 eg | 顺磁性弱或抗磁性 |

4d/5d 金属几乎总是低自旋（Delta 本身很大）。

### 5.4 光谱化学序列（配体场强弱）

  I- < Br- < Cl- < F- < OH- < H2O < NH3 < en < NO2- < CN- < CO
  <-------- 弱场 ----------------------------- 强场 -------->

### 5.5 CFSE 计算（八面体场）

  CFSE = (-0.4x + 0.6y) Delta_o

  其中 x = t2g 电子数，y = eg 电子数

注意：CFSE 不包含成对能 P 的贡献。

### 5.6 异构体

| 类型 | 识别要点 | 典型例子 |
|:---|:---|:---|
| 几何异构 | 配体空间排布不同 | [PtCl2(NH3)2]：顺铂/反铂 |
| 面式/经式 | Ma3b3 型 | fac(共面) / mer(共经线) |
| 光学异构 | 无对称面/对称中心 -> 手性 | [Co(en)3]3+ 的 Lambda/Delta |
| 电离异构 | 内外界离子交换 | [CoBr(NH3)5]SO4 vs [CoSO4(NH3)5]Br |
| 键合异构 | 两可配体不同配位原子 | NO2-: N配位(硝基) vs O配位(亚硝酸根) |

### 5.7 18电子规则

中心金属的 d 电子 + 配体提供的电子 = 18（下一个稀有气体的电子数）。

适用于低氧化态的有机金属配合物（羰基配合物等）。

## 六、跨模块综合判断框架

做一道结构题时，按以下顺序判断：

  第一步：这题在考哪一层？
      -> 涉及量子数/电子排布/周期律 -> 原子层
      -> 涉及分子形状/键角/键型/杂化 -> 分子层
      -> 涉及晶胞/堆积/密度/Bragg -> 晶体层
      -> 涉及配合物/配位数/d轨道分裂/磁性 -> 配位层

  第二步：调用对应层的核心工具
      -> 原子层：三原则写排布 -> Slater求Z* -> 周期律比较
      -> 分子层：Lewis -> VSEPR -> 杂化 -> MO(如需)
      -> 晶体层：找晶胞 -> 计数 -> 算密度/配位数 -> Bragg(如需)
      -> 配位层：识别中心+配体 -> CFT分裂 -> 高/低自旋 -> CFSE -> 磁性

  第三步：检查跨层连接
      -> 原子层 -> 分子层：电子排布决定杂化类型
      -> 分子层 -> 晶体层：分子间力决定晶体堆积方式
      -> 原子层 -> 配位层：d电子数决定CFT行为
      -> 分子层 -> 配位层：Lewis酸碱是配位键的本质

## 七、高频陷阱 Top 8

**陷阱 1：书写顺序 vs 填充顺序**

错误：Fe 的电子排布写成 [Ar]4s2 3d6

正确：按 n 排序写 [Ar]3d6 4s2。填充时 4s 先于 3d，但书写时按主量子数 n 排列。

**陷阱 2：失电子先失 4s 不是 3d**

错误：Fe2+ 失去 2 个 3d 电子

正确：Fe 先失 4s2 再失 3d。Fe2+ = [Ar]3d6，Fe3+ = [Ar]3d5。

**陷阱 3：孤对电子必须算进 VSEPR**

错误：H2O 是直线形（因为 O 连了 2 个 H）

正确：O 有 2 对孤对电子，AX2E2 -> 四面体电子对几何 -> V 形（~104.5度）。

**陷阱 4：异类型沉淀不能直接比 Ksp**

（此陷阱属于化学平衡模块，但在配合物-溶解度耦合题中常出现）

不同类型沉淀的 s-Ksp 关系不同。1:1 型 s = sqrt(Ksp)，1:2 型 s = cbrt(Ksp/4)。

**陷阱 5：三角双锥中孤对优先占赤道向**

错误：SF4 中孤对占轴向

正确：SF4 为 AX4E，孤对占赤道向（3 个赤道位置中取 1 个），分子构型为跷跷板形。原因：赤道向孤对与相邻 2 个轴向配体和 2 个赤道配体的排斥总和最小。

**陷阱 6：MO 能级图两类顺序搞混**

错误：所有第二周期双原子分子都用同一套 MO 能级图

正确：B2/C2/N2 用 pi_2p < sigma_2p（s-p混杂显著）；O2/F2 用 sigma_2p < pi_2p（s-p混杂弱）。

**陷阱 7：晶体场分裂能 Delta 和成对能 P 混淆**

错误：Delta 大就一定是低自旋

正确：必须比较 Delta 和 P。Delta > P -> 低自旋；Delta < P -> 高自旋。3d 金属需要判断，4d/5d 金属 Delta 本身就大，几乎总是低自旋。

**陷阱 8：fcc 和 hcp 晶胞中原子数不同**

错误：fcc 和 hcp 都是最密堆积，晶胞中原子数相同

正确：fcc 晶胞含 4 个原子，hcp 晶胞含 2 个原子。虽然空间利用率都是 74%，配位数都是 12，但晶胞大小不同。
