## 4. 抽料规则

### 4.1 教师稿从哪里抽

- 课时结构、边界、台阶表：优先取自备课大纲
- 板书顺序、讲评口令、课堂问题链：优先取自新授课正文
- 若两者冲突：
  - 深度边界以备课大纲为准
  - 学生可见表达与例题顺序以新授课为准

### 4.2 学生讲义从哪里抽

- 正文、例题、易错点、自查清单：直接来自新授课
- 允许保留的“精品化强化项”：
  - `结构题统一四问`
  - `物性题统一三步`
  - 三组关键对照图的说明文字
- 不写入：
  - 过多晶体学高阶术语
  - 教师侧边界决策语言

### 4.3 PPT 从哪里抽

- 必须进入 PPT 的三组图：
  - `ccp / hcp`
  - `NaCl / CsCl / ZnS`
  - `冰 / 金刚石 / 石墨`
- 必须进入 PPT 的三个框架：
  - `堆积 → 填隙 → 配位数`
  - `半径比 / Pauling / 极化`
  - `结构 → 作用力 → 性质`
- 长推导如空间利用率、半径比几何解释：保留给板书，不整段塞进 PPT

## 5. 媒介分工决策

| 内容 | 最佳媒介 | 处理规则 |
|:---|:---:|:---|
| ccp / hcp 对比 | PPT | 一页并排，只抓堆积顺序和配位数 |
| NaCl / CsCl / ZnS 对照 | PPT | 一页并排，不拆成三页碎讲 |
| 半径比与极化解释 | 板书 + PPT | PPT 给框架，板书补“为什么” |
| 结构解释物性 | 讲义 + PPT | PPT 用对照表，讲义保留完整表达 |
| 综合题路径 | 板书 | 用“结构题四问 / 物性题三步”固定套路 |
