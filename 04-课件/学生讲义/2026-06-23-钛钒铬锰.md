# 过渡金属（一）——钛、钒、铬、锰

> **适用**：第一轮（基础）→ 第二轮（深化）
> **对应备课大纲**：[[04-课件/备课大纲/2026-06-03-钛钒铬锰-提高班]]
> **前置要求**：原子结构、元素周期律、氧化还原基本概念、配合物基础、VSEPR 理论
> **深度边界**：本讲聚焦 Ti/V/Cr/Mn 四种元素的核心化学性质与价态变化规律。不展开过渡金属配位场理论的完整 MO 分析，不展开多酸化学，不展开金红石能带结构与光催化机理。

---

## 学习目标

- ✅ 目标1：能说出各元素的常见氧化态及特征颜色，记住 TiCl₄ 水解反应、钒的多价态颜色序列、Cr₂O₇²⁻/CrO₄²⁻ 互变、KMnO₄ 在不同 pH 下的还原产物。
- ✅ 目标2：能从颜色信号出发，结合介质条件推断 V/Cr/Mn 的价态和存在形式。
- ✅ 目标3：能写出 V₂O₅ 催化氧化 SO₂、KMnO₄ 的三个还原半反应、Cr(VI) 废液处理等核心方程式。
- ✅ 目标4：能运用电极电势判断 Ti/V/Cr/Mn 的氧化还原方向。
- ✅ 目标5：能解释 Cr(III) 的动力学惰性及其与 Al(III) 的异同。

---

## 零、过渡金属通性回顾

### 0.1 第一过渡系（3d 系列）共性

| 共性 | 说明 |
|:---|:---|
| 可变氧化态 | 3d 电子可部分或全部参与成键，氧化态从 +2 到最高族序数 |
| 颜色丰富 | d-d 跃迁导致水合离子和配离子呈现特征颜色 |
| 易形成配合物 | 具有空 d 轨道接受配体孤对电子 |
| 催化活性 | 未填满的 d 轨道可吸附反应物，降低活化能 |
| 多为顺磁性 | 存在未成对电子（除 d¹⁰ 构型） |

### 0.2 Ti-V-Cr-Mn 氧化态递变规律

| 元素 | 族 | 最高氧化态 | 常见稳定氧化态 | 特征 |
|:---:|:---:|:---:|:---|:---|
| Ti | IVB | +4 | +4（最稳），+3 | d⁰ 构型稳定，高氧化态为主 |
| V | VB | +5 | +5，+4，+3，+2 | 多价态共存，颜色序列推断指纹 |
| Cr | VIB | +6 | +3（最稳），+6 | Cr(VI) 强氧化性，Cr(III) 动力学惰性 |
| Mn | VIIB | +7 | +7，+4，+2 | 价态最丰富，MnO₄⁻ 是最强氧化剂之一 |

> **核心规律**：从左到右，最高氧化态稳定性递减（Ti⁴⁺ 极稳 → MnO₄⁻ 极强氧化性），低氧化态稳定性递增（Ti³⁺ 强还原剂 → Mn²⁺ 极稳）。

## 一、钛（Ti）

### 1.1 电子构型与常见氧化态

- 电子构型：[Ar]3d²4s²
- **常见氧化态**：+4（最稳定），+3
- Ti⁴⁺：d⁰ 构型，无色
- Ti³⁺：d¹ 构型，紫色，具强还原性

### 1.2 单质性质

- 钛被称为"**第三金属**"（仅次于铁、铝）：强度接近钢，密度仅 4.54 g/cm³（约为钢的 60%），耐高温约 800 K
- **钝化特性**：钛在热力学上非常活泼（E°(Ti²⁺/Ti) = -1.63 V），但表面形成致密氧化膜（TiO₂），常温下极稳定，不与 O₂、Cl₂、冷酸反应
- 这一现象体现了**热力学 vs 动力学**的区分——热力学倾向反应，但动力学上因钝化膜阻挡而受阻
- 钛溶于热浓 HCl、热浓 H₂SO₄、HF 中：
  - Ti + 6HF → H₂[TiF₆] + 2H₂↑
  - Ti + 6HCl(热浓) → TiCl₆²⁻ + 2H₂↑ + 4H⁺

### 1.3 重要化合物

#### TiO₂（二氧化钛，钛白粉）

- **结构**：金红石型（四方晶系），Ti 配位数 6，O 配位数 3
- **性质**：白色粉末，是最重要的白色颜料（无毒，遮盖力强，替代铅白/锌白）
- **用途**：颜料、防晒霜（纳米级 TiO₂ 透明且吸收紫外线）、光催化（需紫外光激发）
- **酸碱性**：两性偏碱性，与浓 H₂SO₄ 反应：
  - TiO₂ + 2H₂SO₄(浓热) → Ti(SO₄)₂ + 2H₂O

#### TiCl₄（四氯化钛）

- **性质**：无色液体（熔点 -25°C，沸点 136°C），遇水剧烈水解产生白色烟雾
- **核心反应**（竞赛必考）：

  TiCl₄ + 2H₂O → TiO₂↓ + 4HCl↑（白色烟雾）

- 与 SiCl₄ 水解类似：中心原子电荷高、半径小 → 对水分子有强极化作用
- **钛白粉制备**（气相氧化法）：

  TiCl₄(气) + O₂(气) → TiO₂ + 2Cl₂（~900°C）

#### Ti³⁺ 的颜色

- [Ti(H₂O)₆]³⁺：紫色（d¹，t₂g¹，d-d 跃迁）
- Ti³⁺ 在空气中易被氧化为 Ti⁴⁺（褪色）
- **钛的定性检验**：Ti⁴⁺ + H₂O₂ 在酸性溶液中生成 [TiO(H₂O₂)]²⁺ 橘黄色配合物，可用于比色分析

### 1.4 钛与硅的类比

| 性质 | Ti | Si |
|:---|:---|:---|
| 氯化物水解 | TiCl₄ + 2H₂O → TiO₂ + 4HCl | SiCl₄ + 2H₂O → SiO₂ + 4HCl |
| 形成氟配离子 | TiF₆²⁻ | SiF₆²⁻ |
| 高氧化态稳定 | Ti⁴⁺ 极稳定 | Si⁴⁺ 极稳定 |

> 原因：Ti 和 Si 都有空 d 轨道，可接受 F⁻ 等小配体的孤对电子形成稳定配离子。

## 二、钒（V）

### 2.1 电子构型与常见氧化态

- 电子构型：[Ar]3d³4s²
- 氧化态：+2、+3、+4、+5
- **多价态颜色序列**（竞赛核心考点）：

  V²⁺（紫）→ V³⁺（绿）→ VO²⁺（蓝）→ VO₂⁺（黄）

  > **记忆口诀**："价态越高，颜色越浅 (紫→绿→蓝→黄)"

- 还原过程：V₂O₅ → VO₂⁺(黄) → VO²⁺(蓝) → V³⁺(绿) → V²⁺(紫)，可用 Zn 还原逐级观察

![[media/vanadium-latimer-diagram.jpg|V 的 Latimer 图]]
*图 V 的 Latimer 图（酸性条件）——展示了钒从 +5 到 0 各氧化态的标准电极电势，VO₂⁺(+5) → VO²⁺(+4) → V³⁺(+3) → V²⁺(+2) 的逐级还原电势依次递减，解释了 Zn 还原 V₂O₅ 可依次观察到黄→蓝→绿→紫的颜色递变（Weller《无机化学》第6版 图15.x）*

### 2.2 单质性质

- 银白色金属，中等硬度
- 常温下较稳定，表面形成氧化物钝化膜
- 溶于浓 HNO₃、浓 H₂SO₄，不溶于稀酸（钝化）

### 2.3 重要化合物

#### V₂O₅（五氧化二钒）

- **颜色**：橙红色固体
- **两性偏酸性**：以酸性为主，溶于碱生成钒酸盐
  - V₂O₅ + 6NaOH → 2Na₃VO₄ + 3H₂O
- **最重要的工业应用**：接触法制硫酸的催化剂

  V₂O₅ + SO₂ → V₂O₄ + SO₃（催化循环的简化表达）
  实际催化循环：V⁵⁺ ↔ V⁴⁺ 之间循环，SO₂ 被氧化为 SO₃

#### 钒酸根缩合

- 与磷酸根类似，钒酸根在不同 pH 下发生缩合：
  - pH > 13：VO₄³⁻（正钒酸根，无色）
  - pH 降低 → 缩合度增大 → 颜色加深
  - VO₄³⁻ → V₂O₆(OH)³⁻ → V₃O₉³⁻ → V₁₀O₂₈⁶⁻（十钒酸根，深红）
- 颜色变化：无色 → 淡黄 → 深红

#### NH₄VO₃（偏钒酸铵）

- 热分解制备 V₂O₅：

  2NH₄VO₃ →△→ V₂O₅ + 2NH₃↑ + H₂O↑

#### VOCl₃（三氯氧钒）

- 黄色液体，与水剧烈反应：

  2VOCl₃ + 3H₂O → V₂O₅ + 6HCl

### 2.4 钒的化合物小结

| 氧化态 | 存在形式 | 颜色 | 性质 |
|:---:|:---|:---:|:---|
| +5 | VO₂⁺（氧钒离子）| 黄 | 酸性介质中稳定 |
| +5 | VO₄³⁻（正钒酸根）| 无色~淡黄 | 碱性介质中稳定 |
| +4 | VO²⁺（氧钒(IV)离子）| 蓝 | 稳定 |
| +3 | V³⁺(aq) | 绿 | 还原性 |
| +2 | V²⁺(aq) | 紫 | 强还原性 |

## 三、铬（Cr）

### 3.1 电子构型与常见氧化态

- 电子构型：3d⁵4s¹（**注意**：不是 3d⁴4s²，半满 d⁵ 特殊稳定性！）
- 常见氧化态：+3（最稳定），+6
- Cr²⁺ 存在但不稳定（强还原性）
- **铬是金属单质中硬度最大的**（未成对电子多 → 金属键强）

### 3.2 单质性质

- 银白色金属，硬度最大
- 表面有致密氧化膜 → 常温下耐腐蚀
- 溶于稀 HCl、稀 H₂SO₄ 生成 Cr³⁺（绿色），不溶于浓 HNO₃（钝化）

### 3.3 Cr³⁺ 的化学

- **颜色**：[Cr(H₂O)₆]³⁺ 为紫色，常见盐（如 CrCl₃·6H₂O）为绿色
- **最稳定氧化态**：Cr(III) 在酸性、中性、碱性条件下均稳定
- **两性**：Cr(OH)₃ 与 Al(OH)₃ 类似

  Cr³⁺ + 3OH⁻ → Cr(OH)₃↓(灰绿) + OH⁻(过量) → [Cr(OH)₄]⁻(亮绿)

- **Cr(III) 的动力学惰性**（d³ 构型，配体交换极慢）：
  - [Cr(H₂O)₆]³⁺ 的配体交换半衰期约 10⁵ 小时！
  - 这与 Al³⁺ 形成鲜明对比：Al³⁺ 是热力学稳定但动力学活泼
  - Cr(III) 的水合异构体可分离：[CrCl₂(H₂O)₄]⁺ 绿、[CrCl(H₂O)₅]²⁺ 浅绿、[Cr(H₂O)₆]³⁺ 紫

### 3.4 Cr(VI) 的化学（竞赛核心）

| 物种 | 颜色 | pH 条件 | 氧化性 |
|:---|:---:|:---:|:---:|
| CrO₄²⁻（铬酸根）| **黄色** | 碱性/中性 | 弱 |
| Cr₂O₇²⁻（重铬酸根）| **橙色** | 酸性 | **强**（E° = 1.36 V）|

- **酸碱互变平衡**（必考）：

  2CrO₄²⁻(黄) + 2H⁺ ⇌ Cr₂O₇²⁻(橙) + H₂O，K = 4.2 × 10¹⁴

![[media/cro4-cr2o7-equilibrium.jpg|CrO₄²⁻/Cr₂O₇²⁻ 酸碱平衡]]
*图 CrO₄²⁻（黄色）与 Cr₂O₇²⁻（橙色）的酸碱互变平衡——加酸平衡右移（橙色加深），加碱左移（黄色加深），K = 4.2×10¹⁴ 表明平衡强烈偏向 Cr₂O₇²⁻ 一侧。该平衡是 Cr(VI) 化学的核心，也是竞赛中"加酸变橙，加碱变黄"现象的根源（Weller《无机化学》第6版）*

  > "加酸变橙，加碱变黄"——本质是二聚平衡

- **强氧化性**（酸性条件）：

  Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺(绿) + 7H₂O

- **重要反应**——重铬酸根氧化 Fe²⁺（用于测定铁含量）：

  Cr₂O₇²⁻ + 6Fe²⁺ + 14H⁺ → 2Cr³⁺ + 6Fe³⁺ + 7H₂O

- **Cr(VI) 有毒**，废液需用 FeSO₄ 还原为 Cr(III) 再处理：

  上述反应即可用于废液处理

### 3.5 铬酸盐沉淀

| 阳离子 | 沉淀 | 颜色 | 用途 |
|:---|:---|:---:|:---|
| Ag⁺ | Ag₂CrO₄ | **砖红色** | 沉淀滴定指示剂（Mohr 法）|
| Pb²⁺ | PbCrO₄ | **橙黄色** | 黄色颜料（铬黄）|
| Ba²⁺ | BaCrO₄ | **黄色** | 检验 CrO₄²⁻ 的存在 |

### 3.6 Cr(III) vs Al(III) 对比

| 性质 | Cr(III) | Al(III) |
|:---|:---|:---|
| 电子构型 | d³（动力学惰性）| p⁶（稀有气体构型）|
| 水合离子 | [Cr(H₂O)₆]³⁺ 紫/绿色 | [Al(H₂O)₆]³⁺ 无色 |
| 与 OH⁻ 反应 | Cr(OH)₃↓ 灰绿，过量溶为 [Cr(OH)₄]⁻ 亮绿 | Al(OH)₃↓ 白，过量溶为 [Al(OH)₄]⁻ 无色 |
| 配体交换速率 | 极慢（动力学惰性）| 快（动力学活泼）|
| 氧化物 | Cr₂O₃ 绿 | Al₂O₃ 白 |
| 与 H₂O₂ 反应 | Cr(OH)₄⁻ + H₂O₂ → CrO₄²⁻(黄) | 无类似反应 |

> **Cr(OH)₃ 三步色变**（鉴别 Cr 的核心流程）：
> Cr³⁺(绿) + OH⁻ → Cr(OH)₃↓(灰绿) + OH⁻(过量) → [Cr(OH)₄]⁻(亮绿) + H₂O₂ → CrO₄²⁻(黄)

### 3.7 H₂O₂ 检测 Cr(VI)

- 在酸性条件下，Cr₂O₇²⁻ 与 H₂O₂ 反应生成深蓝色 CrO₅（过氧化铬）：

  Cr₂O₇²⁻ + 4H₂O₂ + 2H⁺ → 2CrO₅(蓝) + 5H₂O

- CrO₅ 在乙醚层中富集，呈深蓝色，是鉴定 Cr(VI) 的特征反应

## 四、锰（Mn）

### 4.1 电子构型与常见氧化态

- 电子构型：[Ar]3d⁵4s²
- **氧化态极其丰富**：+7、+6、+4、+3、+2
- 在酸性条件下的稳定存在形式：Mn²⁺、MnO₂、MnO₄⁻
- Mn²⁺ 特别稳定的原因：**d⁵ 半满构型**（洪特规则，额外的交换能稳定化）

### 4.2 单质性质

- 银白色金属，较活泼
- 与 Mg 类似：溶于稀酸生成 Mn²⁺ 和 H₂，不与碱反应
- **无钝化特性**（区别于 Ti/V/Cr）：E°(Mn²⁺/Mn) = -1.19 V

### 4.3 重要化合物

#### Mn²⁺ 的化学

- **颜色**：淡粉色（极浅，近无色）
- **稳定性**：酸性条件下极稳定，不易被氧化
- **Mn²⁺ 的鉴定**（竞赛必考）：

  **NaBiO₃ 法**：
  2Mn²⁺ + 5NaBiO₃ + 14H⁺ → 2MnO₄⁻(紫红) + 5Bi³⁺ + 5Na⁺ + 7H₂O

  **(NH₄)₂S₂O₈ 法**（Ag⁺ 催化）：
  2Mn²⁺ + 5S₂O₈²⁻ + 8H₂O → 2MnO₄⁻ + 10SO₄²⁻ + 16H⁺

  反应现象：近无色 → 紫红色（MnO₄⁻ 生成），特征明显

#### KMnO₄（高锰酸钾）——竞赛最高频考点

- **颜色**：紫色（d⁰ 构型，LMCT 电荷转移跃迁）
- **强氧化性**：E°(MnO₄⁻/Mn²⁺) = 1.51 V（酸性条件）

**KMnO₄ 在不同 pH 下的还原产物（必背！）**：

| 介质 | 还原产物 | Mn 氧化态 | 颜色 | 转移电子数 | φ° |
|:---:|:---|:---:|:---:|:---:|:---:|
| **酸性** | Mn²⁺ | +2 | 近无色（浅粉）| 5e⁻ | 1.51 V |
| **中性/弱碱性** | MnO₂↓ | +4 | **棕黑色沉淀** | 3e⁻ | 0.59 V |
| **强碱性** | MnO₄²⁻ | +6 | **绿色** | 1e⁻ | 0.56 V |

**三个条件下的典型半反应**：

- **酸性**：MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
- **中性**：MnO₄⁻ + 2H₂O + 3e⁻ → MnO₂↓ + 4OH⁻
- **碱性**：MnO₄⁻ + e⁻ → MnO₄²⁻（绿色，锰酸根）

> **注意事项**：KMnO₄ 在碱性条件下还原为 MnO₄²⁻（绿色），学生极易与 Mn²⁺ 或 MnO₂ 混淆。记住：**碱性→锰酸根绿色**！

**KMnO₄ 的典型反应示例**：

- 酸性条件 + Na₂SO₃（亚硫酸钠）：

  2MnO₄⁻ + 5SO₃²⁻ + 6H⁺ → 2Mn²⁺ + 5SO₄²⁻ + 3H₂O

- 中性条件 + Na₂SO₃：

  2MnO₄⁻ + 3SO₃²⁻ + H₂O → 2MnO₂↓ + 3SO₄²⁻ + 2OH⁻

- 碱性条件 + Na₂SO₃：

  2MnO₄⁻ + SO₃²⁻ + 2OH⁻ → 2MnO₄²⁻(绿) + SO₄²⁻ + H₂O

*图 Mn 的 Frost 图（酸性条件）——清晰地展示了 Mn 从 -2 到 +7 各氧化态的吉布斯自由能相对值：Mn²⁺ 位于最低点（最稳定，d⁵ 半满），MnO₄⁻ 在较高位置（强氧化性），斜率越大的线段对应的电对氧化性越强。Frost 图是判断元素各氧化态稳定性和歧化趋势的核心工具（Weller《无机化学》第6版 图15.7）*

#### MnO₂（二氧化锰）

- **颜色**：黑色固体
- **性质**：两性（以碱性为主），氧化性
- **最重要的性质**：催化作用
  - 催化 H₂O₂ 分解：2H₂O₂ → MnO₂→ 2H₂O + O₂↑
  - 催化 KClO₃ 分解：2KClO₃ → MnO₂→ 2KCl + 3O₂↑
- **与浓盐酸反应**（实验室制 Cl₂）：

  MnO₂ + 4HCl(浓) →△→ MnCl₂ + Cl₂↑ + 2H₂O

#### Mn₃O₄（四氧化三锰）

- Mn₂O₃·MnO，黑色
- 碳酸锰热分解产物：

  MnCO₃ →△→ MnO + CO₂↑；4MnO + O₂ → 2Mn₂O₃；Mn₂O₃ + MnO → Mn₃O₄

> 工业电池材料 LiMn₂O₄ 中 Mn 的平均价态为 +3.5，实际为 Mn²⁺ 和 Mn⁴⁺ 的混合。

## 六、典型例题

### 例1 | 钒的颜色序列

**题**：某无色溶液 X 经 Zn 还原后依次呈现黄色、蓝色、绿色、紫色。已知 X 含钒元素，确定每一步的颜色对应物种并写出 Zn 还原的总反应（假定最终还原为 V²⁺）。

**解析**：

| 步骤 | 颜色 | 物种 | 钒氧化态 |
|:---:|:---:|:---|:---:|
| 起始 | 无色（或淡黄）| VO₄³⁻ 或 VO₂⁺ | +5 |
| 第一步 | **黄色** | VO₂⁺ | +5 |
| 第二步 | **蓝色** | VO²⁺ | +4 |
| 第三步 | **绿色** | V³⁺ | +3 |
| 第四步 | **紫色** | V²⁺ | +2 |

对应还原路径：VO₂⁺ → VO²⁺ → V³⁺ → V²⁺

总反应：2VO₂⁺ + 3Zn + 8H⁺ → 2V²⁺ + 3Zn²⁺ + 4H₂O

**考点**：钒的多价态颜色序列是竞赛推断题的重要"指纹"，记住"黄→蓝→绿→紫"的还原方向。

### 例2 | Cr(OH)₃ 的两性

**题**：向 Cr₂(SO₄)₃ 绿色溶液中逐滴加入 NaOH 溶液，描述现象并写出相关方程式。

**解析**：

- 开始时：Cr³⁺(绿) + 3OH⁻ → Cr(OH)₃↓(灰绿色沉淀)
- NaOH 过量：Cr(OH)₃ + OH⁻ → [Cr(OH)₄]⁻(亮绿色溶液)
- 再加入 H₂O₂：[Cr(OH)₄]⁻ + H₂O₂ + OH⁻ → CrO₄²⁻(黄色) + 4H₂O（Cr(III) 被氧化为 Cr(VI)）

**典型现象色变链**：绿 → 灰绿沉淀 → 亮绿 → 黄

### 例3 | KMnO₄ 与 Na₂SO₃ 在不同 pH 下的反应

**题**：写出 KMnO₄ 在酸性、中性、碱性条件下与 Na₂SO₃ 反应的离子方程式。

| 条件 | 反应（离子方程式）| 现象 |
|:---:|:---|:---:|
| **酸性** | 2MnO₄⁻ + 5SO₃²⁻ + 6H⁺ → 2Mn²⁺ + 5SO₄²⁻ + 3H₂O | 紫红色褪为近无色 |
| **中性** | 2MnO₄⁻ + 3SO₃²⁻ + H₂O → 2MnO₂↓ + 3SO₄²⁻ + 2OH⁻ | 紫红褪去，棕黑沉淀生成 |
| **强碱性** | 2MnO₄⁻ + SO₃²⁻ + 2OH⁻ → 2MnO₄²⁻ + SO₄²⁻ + H₂O | 紫红变绿（MnO₄²⁻）|

**考点**：这是竞赛中最经典的 KMnO₄ 考点，必须**根据 pH 判断产物再配平**，不能死记一个反应。

### 例4 | Cr₂O₇²⁻ 氧化 Fe²⁺ 的配平

**题**：配平反应 Cr₂O₇²⁻ + Fe²⁺ + H⁺ → Cr³⁺ + Fe³⁺ + H₂O。

**配平步骤**：

1. 确定电子转移：Cr：+6 → +3，每个 Cr 得 3e⁻，2个 Cr 共得 6e⁻
2. Fe：+2 → +3，每个 Fe 失 1e⁻
3. 得失电子相等：需 6 个 Fe²⁺
4. 配平电荷：左侧：-2 + 12 + n(+) = 右侧：6 + 6 + 0 → n(H⁺)=14
5. 检查原子守恒：

**配平结果**：
Cr₂O₇²⁻ + 6Fe²⁺ + 14H⁺ → 2Cr³⁺ + 6Fe³⁺ + 7H₂O

### 例5 | MnO₂ 的催化与氧化

**题**：写出 MnO₂ 与浓盐酸共热的反应方程式，并说明该反应在实验室中的用途。

**解**：
MnO₂ + 4HCl(浓) →△→ MnCl₂ + Cl₂↑ + 2H₂O

该反应是实验室制备 Cl₂ 的方法之一（另一方法：KMnO₄ + HCl）。

**易错点**：注意 MnO₂ 在此反应中作**氧化剂**（Mn⁴⁺ → Mn²⁺），而不是催化剂。MnO₂ 作为催化剂的典型例子是催化 H₂O₂ 或 KClO₃ 分解。

### 例6 | LiMn₂O₄ 中的 Mn 价态判断

**题**：LiMn₂O₄ 是一种锂离子电池正极材料。计算其中 Mn 的平均氧化态，并说明其实际价态分布。

**解**：
- Li⁺ 为 +1，O²⁻ 为 -2
- 设 Mn 的平均氧化态为 x：+1 + 2x + 4×(-2) = 0 → 2x = 7 → x = +3.5
- Mn 的平均价态为 **+3.5**
- 实际为 Mn²⁺ 和 Mn⁴⁺ 的混合（1:1）：(Mn²⁺ + Mn⁴⁺)/2 = +3.5

## 七、常见误区

| 序号 | 误区 | 正确认知 |
|:---:|:---|:---|
| 1 | 认为 Ti 不活泼（因钝化）| Ti 热力学很活泼（E°=-1.63 V），钝化是动力学惰性 |
| 2 | 混淆 V²⁺ 颜色序列方向 | 还原方向：黄(V⁵⁺)→蓝(V⁴⁺)→绿(V³⁺)→紫(V²⁺)；氧化方向相反 |
| 3 | Cr(OH)₃ 与 Al(OH)₃ 完全一样 | 都是两性，但 Cr(III) 配体交换极慢（动力学惰性），Al(III) 快 |
| 4 | KMnO₄ 碱性产物记成 Mn²⁺ | 碱性产物是 MnO₄²⁻（绿色），不是 Mn²⁺ 或 MnO₂ |
| 5 | 认为所有 Cr(VI) 都有强氧化性 | CrO₄²⁻ 在碱性条件下氧化性很弱，强氧化性需酸性 Cr₂O₇²⁻ |
| 6 | 认为 Cr 电子构型是 3d⁴4s² | 实际是 3d⁵4s¹（半满 d⁵ 的特殊稳定性）|
| 7 | 混淆 BaCrO₄ 与 BaSO₄ | BaCrO₄ 黄色（不是白色），可溶于酸 |
| 8 | 认为 Mn²⁺ 无色 | Mn²⁺ 是极浅的粉色（近无色，但仔细看可察觉）|
| 9 | MnO₂ 所有反应都是催化 | MnO₂ + 浓 HCl 中被还原（作氧化剂），非催化 |
| 10 | 混淆中性/碱性 KMnO₄ 产物 | 中性→MnO₂↓ 棕黑沉淀；碱性→MnO₄²⁻ 绿色溶液 |

### 必做题

1. 写出 TiCl₄ 水解的反应方程式，并说明为什么 TiCl₄ 遇水会产生白色烟雾。
2. 默写钒的多价态颜色序列（从 V²⁺ 到 VO₂⁺），指出对应的氧化态。
3. 写出 Cr₂O₇²⁻/CrO₄²⁻ 的酸碱互变平衡方程式，说明颜色变化。
4. 写出 KMnO₄ 在酸性、中性、碱性条件下与 Na₂SO₃ 反应的离子方程式。
5. 写出 Cr₂O₇²⁻ 氧化 Fe²⁺ 的离子方程式，配平并标出电子转移。
6. 写出 Mn²⁺ 的两种鉴定方法（NaBiO₃ 法和 (NH₄)₂S₂O₈ 法）的离子方程式。
7. 为什么 Cr(VI) 废液需要用 FeSO₄ 处理？写出相关反应。

### 选做题

8. 比较 Cr(OH)₃ 与 Al(OH)₃ 的两性异同，说明 Cr(III) 动力学惰性的表现。
9. 已知 Ti³⁺ 在空气中易被氧化为 Ti⁴⁺，写出氧化反应方程式，并预测颜色变化。
10. 解释为什么 Cr 在所有金属单质中硬度最大（从电子构型角度）。

### 参考答案

1. TiCl₄ + 2H₂O → TiO₂ + 4HCl↑。TiCl₄ 中 Ti⁴⁺ 电荷高、半径小，对水分子有强极化作用，水解剧烈产生 HCl 气体遇水蒸气形成白色酸雾。
2. V²⁺(紫, +2) → V³⁺(绿, +3) → VO²⁺(蓝, +4) → VO₂⁺(黄, +5)
3. 2CrO₄²⁻(黄) + 2H⁺ ⇌ Cr₂O₇²⁻(橙) + H₂O，K=4.2×10¹⁴。加酸变橙，加碱变黄。
4-6. 见第六节例题解析。
7. 因 Cr(VI) 有毒且具强氧化性，Fe²⁺ 可将其还原为无毒的 Cr³⁺。反应：Cr₂O₇²⁻ + 6Fe²⁺ + 14H⁺ → 2Cr³⁺ + 6Fe³⁺ + 7H₂O。
8. Cr(OH)₃ 与 Al(OH)₃ 均为两性氢氧化物，但 Cr(III) 的 d³ 构型导致配体交换极慢（半衰期约 10⁵ 小时），水合异构体可分离；Al(III) 配体交换快，异构体无法分离。
9. 2[Ti(H₂O)₆]³⁺(紫) + O₂ + 4H⁺ → 2[TiO(H₂O)₅]²⁺(无色) + 5H₂O 或其他 Ti⁴⁺ 物种。颜色由紫色变为无色。
10. Cr 的电子构型为 3d⁵4s¹，有 6 个未成对电子（最多），金属键非常强 → 硬度最大。
