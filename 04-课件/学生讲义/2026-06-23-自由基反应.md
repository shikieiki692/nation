# 自由基反应

> **适用**：第三轮（提高班）| **前置要求**：[[04-课件/学生讲义/2026-06-23-有机化学基础]]（电子效应）、[[活性中间体]]、[[反应机理表示法]]
> **对应备课大纲**：[[04-课件/备课大纲/2026-06-05-自由基反应-提高班]]
> **对应专题**：[[04-专题与题型/专题/专题-自由基反应]]
>
> 核心线索：**自由基不是"例外题"，而是一条完整的单电子链式机理赛道**。从卤代到过氧化物效应，从 NBS 到自由基环化，全都可以统一在"引发→增长→终止"的框架下。

## 学习目标

- ✅ 能识别常见自由基引发信号（hv、ROOR、AIBN、NBS、SET）
- ✅ 能用链式反应三阶段（引发→增长→终止）完整书写自由基机理
- ✅ 能解释 Cl₂ 与 Br₂ 卤代选择性的差异（Hammond 假设 + BDE）
- ✅ 能用自由基稳定性解释 HBr/ROOR 反马氏加成
- ✅ 能写出 NBS 烯丙位溴代的机理，区分取代与加成
- ✅ 能判断自由基环化的 5-exo-trig / 6-endo-trig 选择性
- ✅ 能比较自由基与碳正离子的稳定性差异，用电子效应解释

## 一、自由基的基本性质

### 1.1 自由基的结构

自由基（Free Radical）是带有**未成对电子**的中性物种。其中心碳原子外层有 7 个电子，通常采取接近 sp² 的平面构型（单电子在 p 轨道中）。

| 性质 | 描述 |
|:---|:---|
| 电荷 | 中性（非正非负） |
| 电子数 | 7 个价电子（比八隅体少 1 个） |
| 杂化方式 | 接近 sp²（平面或略微金字塔形） |
| 单电子位置 | 垂直于分子平面的 p 轨道 |
| 典型寿命 | μs ~ ms（溶液中），可因离域而延长 |
| 符号表示 | R·（点表示未成对电子） |

**与碳正离子的结构对比**：

| 对比项 | 碳正离子 R₃C⁺ | 碳自由基 R₃C· |
|:---|:---|:---|
| 电荷 | 正电 | 中性 |
| 价电子数 | 6 | 7 |
| 杂化 | sp²（平面三角形） | 接近 sp²（平面） |
| 稳定性排序 | 苄基/烯丙基 > 3° > 2° > 1° > CH₃⁺ | **烯丙基/苄基 > 3° > 2° > 1° > CH₃·** |
| 关键稳定机制 | 超共轭效应为主 | 离域（共轭）> 超共轭 |

> 💡 **核心差异**：自由基的离域稳定化比超共轭更重要——烯丙基自由基（离域）比叔丁基自由基（超共轭）更稳定；但叔丁基碳正离子比烯丙基正离子更稳定。这一差异是竞赛高频考点。

### 1.2 自由基的稳定性顺序与电子效应解释

**稳定性总顺序**：

```
苄基/烯丙基 > 叔(3°) > 仲(2°) > 伯(1°) > 甲基 > 乙烯基/苯基
    离域        超共轭     超共轭     超共轭       无       SOMO能量高
```

**电子效应来源**：

| 稳定化机制 | 本质 | 示例 | 稳定化程度 |
|:---|:---|:---|:---:|
| **共轭离域** | SOMO 与 π 体系重叠，单电子分散到多个原子 | 烯丙基自由基 ·CH₂-CH=CH₂ | 极强 |
| **超共轭效应** | 相邻 C-H σ 键与 SOMO 部分重叠，分散单电子 | 叔丁基自由基 ·C(CH₃)₃（9 个 C-H） | 强 |
| **杂原子邻位效应** | 电负性原子通过诱导或共轭效应稳定 SOMO | ·CH₂OH、·CH₂CN、·CH₂Cl | 强 |
| **位阻效应** | 大位阻基团阻碍自由基聚合，动力学稳定 | Ph₃C·（三苯甲基）、TEMPO | 极强 |

**自由基稳定性 5 规则**（ABOC §2.7.4）：

| 规则 | 内容 | 示例 |
|:---|:---|:---|
| **规则 1** | **离域优先**——只要能离域，就比不能离域的更稳定 | 烯丙基 > 叔丁基 |
| **规则 2** | **不能离域的：3° > 2° > 1°** | 叔丁基 > 异丙基 > 乙基 |
| **规则 3** | **杂原子邻位稳定**——给电子或吸电子基团都能提高稳定性 | ·CH₂OH、·CH₂CN 均比 ·CH₃ 稳定 |
| **规则 4** | **推拉效应最佳**——一边吸电子、一边给电子的稳定化最好 | ·CH(CN)(OEt) 极稳定 |
| **规则 5** | **位阻稳定**——大位阻自由基更稳定 | Ph₃C· 可稳定存在 |

> ⚠️ **易混点**：自由基与碳正离子的稳定性排序**不完全相同**。碳正离子：叔 > 烯丙基；自由基：烯丙基 > 叔。原因：碳正离子中烯丙基正离子的共振结构将一个正电荷放在更不稳定的伯碳上，因此不如叔碳正离子稳定；而自由基的离域不涉及电荷分离，因此离域效应带来的稳定化特别显著。

### 1.3 键解离能（BDE）与自由基稳定性

BDE（Bond Dissociation Energy）是衡量 C-H 键强度的指标，BDE 越小的键越容易均裂，生成的自由基越稳定。

| C-H 键类型 | BDE (kJ/mol) | 对应自由基 | 相对稳定性 |
|:---:|:---:|:---|:---:|
| CH₃–H（甲烷） | 439 | ·CH₃ 甲基 | 最不稳定 |
| 1° RCH₂–H | 420-425 | RCH₂· 伯自由基 | 不稳定 |
| 2° R₂CH–H | 410-415 | R₂CH· 仲自由基 | 中等 |
| 3° R₃C–H | 397-405 | R₃C· 叔自由基 | 较稳定 |
| 烯丙基 H | ~370 | ·CH₂-CH=CH₂ | 非常稳定（离域）|
| 苄基 H | ~375 | ·CH₂-Ph | 非常稳定（离域）|

## 二、自由基链式反应三阶段

所有自由基链式反应都遵循统一的**三段式**框架：

![[media/radical-chain.png|自由基链式反应的三阶段——引发、增长、终止]]
*图 自由基链式反应的通用机理。链引发（Initiation）产生第一个自由基，链增长（Propagation）阶段自由基通过抽氢/加成/转移等模式循环再生，链终止（Termination）阶段两个自由基结合消耗活性物种。动力学链长可达 10³–10⁵（Wikimedia Commons）*

### 2.1 引发（Initiation）

产生第一个自由基的过程。

| 引发方式 | 具体条件 | 示例 |
|:---|:---|:---|
| **光照均裂** | hv 直接断裂弱键 | Cl₂ → 2 Cl· |
| **热均裂** | Δ 加热分解 | BPO → 2 Ph· + 2 CO₂ |
| **氧化还原** | 单电子转移（SET） | Fe²⁺ + H₂O₂ → Fe³⁺ + HO· + HO⁻ |

**常见引发剂对比**：

| 引发剂 | 结构 | 分解温度 | 产生自由基 | 特点 |
|:---|:---|:---:|:---|:---|
| **AIBN** | (CH₃)₂C(CN)-N=N-C(CN)(CH₃)₂ | 65-80°C | 2 (CH₃)₂C·CN + N₂↑ | 放出 N₂，易追踪 |
| **BPO** | (PhCOO)₂ | 80-90°C | 2 PhCOO· → 2 Ph· + 2 CO₂ | 通用型，产生 CO₂ |
| **DTBP** | (CH₃)₃C-O-O-C(CH₃)₃ | 120-150°C | 2 (CH₃)₃CO· | 高温反应 |
| **H₂O₂/Fe²⁺** | Fenton 试剂 | 室温 | HO· + HO⁻ + Fe³⁺ | 水相氧化 |

### 2.2 链增长（Propagation）

自由基参与反应后生成**新自由基**，形成循环。这是决定产物结构和选择性的关键阶段。

**链增长的基本模式**：

| 模式 | 反应通式 | 示例 |
|:---|:---|:---|
| **抽氢（H Abstraction）** | R· + R'-H → R-H + R'· | Cl· + CH₄ → HCl + CH₃· |
| **加成（Addition）** | R· + C=C → R-C-C· | Br· + CH₂=CH₂ → BrCH₂-CH₂· |
| **卤素转移** | R· + X₂ → R-X + X· | CH₃· + Cl₂ → CH₃Cl + Cl· |
| **β-断裂（Fragmentation）** | R-O· → R'· + 羰基化合物 | PhCOO· → Ph· + CO₂ |
| **环化（Cyclization）** | 分子内 C=C 加成 | 5-exo-trig 环化 |

### 2.3 链终止（Termination）

两个自由基相遇，耗尽未成对电子，链结束。

| 终止方式 | 反应 | 产物 |
|:---|:---|:---|
| **偶联（Coupling）** | R· + R'· → R-R' | 新 σ 键 |
| **歧化（Disproportionation）** | RCH₂-CH₂· + R'CH₂-CH₂· → RCH=CH₂ + R'CH₂CH₃ | 烯烃 + 烷烃 |

**动力学特征**：
- 一个引发事件可导致 10³ ~ 10⁵ 次链增长循环（动力学链长）
- 稳态时 [自由基] 极低（约 10⁻⁸ M），因此终止概率远低于增长概率

> 💡 **记忆口诀**：链式反应三段式——"引发开赛道，增长定产物，终止收尾巴"。

## 三、卤代反应：自由基取代的经典入口

### 3.1 甲烷氯代的完整机理

**链引发**：
```
Cl₂ —hv/Δ→ 2 Cl·         (Cl-Cl 键均裂)
```

**链增长**：
```
Cl· + CH₄ → CH₃· + HCl   (Cl· 抽氢，生成甲基自由基)
CH₃· + Cl₂ → CH₃Cl + Cl· (甲基自由基夺取 Cl₂ 中的 Cl，再生 Cl·)
```
（循环往复，一个 Cl· 可产生大量 CH₃Cl）

**链终止**：
```
Cl· + Cl·   → Cl₂
CH₃· + Cl· → CH₃Cl
CH₃· + CH₃· → CH₃CH₃  (副产物乙烷)
```

### 3.2 Cl₂ 与 Br₂ 选择性对比

**核心问题**：为什么溴代的选择性远高于氯代？

| 对比项 | Cl₂ 氯代 | Br₂ 溴代 |
|:---|:---|:---|
| 反应活性 | 高（几乎见 H 就夺） | 低（只挑最弱的 C-H） |
| 3°:2°:1° 相对活性（室温） | 5.0 : 3.8 : 1.0 | 1600 : 82 : 1.0 |
| 选择性特征 | 中等（接近统计分布） | **极高** |
| 丙烷 2-卤代物 : 1-卤代物 | 约 57:43 | > 98:2 |
| 主控因素 | 统计因子 + 微弱活性差异 | C-H BDE 差异 + Hammonds 假说 |

**Hammond 假说解释**：
- **Cl· 反应性高** → 夺取 H 的过渡态**早**（接近反应物）→ 过渡态中 C-H 断裂程度小 → 不同 C-H 的 BDE 差异未被充分体现 → 选择性低
- **Br· 反应性低** → 夺取 H 的过渡态**晚**（接近产物）→ 过渡态中 C-H 断裂程度大 → BDE 差异被放大 → 选择性高

### 3.3 一卤代产物分布计算

异丁烷 (CH₃)₃CH 在光照下与 Cl₂ 反应：

| H 类型 | 位置 | H 个数(n) | 相对活性(a) | n × a | 产物比例 |
|:---:|:---:|:---:|:---:|:---:|:---:|
| 伯(1°) | 3 个 CH₃ 上的 9 个 H | 9 | 1 | 9 | 64% |
| 叔(3°) | 1 个 CH 上的 1 个 H | 1 | 5 | 5 | 36% |

若改为 Br₂/hv，3° 产物比例 > 99%（因为 1600 : 1 的选择性远大于统计劣势）。

**计算通式**：
```
某产物% = (该位点 H 数 × 相对活性) ÷ Σ(所有位点 H 数 × 相对活性) × 100%
```

> 💡 **竞赛技巧**：氯代时先数 H 个数再乘活性，叔产物一般为 25-40%；溴代时直接判断——溴几乎专一地取代在**三级 H** 或 **烯丙位/苄位 H** 处（如果有的话）。

### 3.4 四种卤素的反应特征

| 卤素 | 反应性 | 选择性 | 是否可控 | 特征 |
|:---:|:---:|:---:|:---:|:---|
| F₂ | 极剧烈 | 极低 | ❌ 爆炸性 | 释放大量热，不可控 |
| Cl₂ | 高 | 低 | ✅ | 工业常用，产物混合 |
| Br₂ | 中 | **高** | ✅ | 竞赛核心，专一性高 |
| I₂ | 极低 | — | ❌ 不反应 | 热力学不利（H-I 键弱） |

## 四、过氧化物效应：自由基改写区域选择性

### 4.1 什么是过氧化物效应

在过氧化物（ROOR、BPO、AIBN）存在下，HBr 对烯烃的加成表现出**反马氏规则**的取向——Br 加到含氢较多的双键碳上。

**本质**：反应机理从离子型（碳正离子）切换为**自由基链式**，选择性由**自由基稳定性**决定。

### 4.2 完整机理（以丙烯 + HBr/ROOR 为例）

**链引发**：
```
RO-OR →Δ 2 RO·              (过氧化物均裂)
RO· + HBr → ROH + Br·       (产生 Br·)
```

**链增长**：
```
Br· + CH₃-CH=CH₂ → CH₃-ĊH-CH₂Br    (Br· 优先进攻末端 CH₂，生成 2° 自由基)
CH₃-ĊH-CH₂Br + HBr → CH₃CH₂CH₂Br + Br·  (产物 1-溴丙烷 + 再生 Br·)
```

**链终止**：
```
2 Br· → Br₂
2 R· → R-R
R· + Br· → R-Br
```

### 4.3 区域选择性来源：自由基稳定性

| 进攻方式 | 生成的自由基 | 自由基稳定性 | 贡献 |
|:---|:---|:---:|:---:|
| Br· 进攻末端 CH₂（反马氏）| CH₃-ĊH-CH₂Br **仲(2°)自由基** | 较稳定 | **主路径** |
| Br· 进攻内部 CH（马氏）| CH₃-CHBr-ĊH₂ **伯(1°)自由基** | 较不稳定 | 极少 |

**关键**：Br· 优先进攻末端碳，因为生成的二级自由基比一级自由基更稳定 → 反马氏产物。

### 4.4 为什么只有 HBr 有过氧化物效应？

| 卤化氢 | H-X 键能 (kJ/mol) | X· 反应性 | 能否发生自由基链式反应 | 原因 |
|:---:|:---:|:---:|:---:|:---|
| **HF** | 569 | — | ❌ | H-F 键太强，无法均裂产生 F· |
| **HCl** | 431 | 较高 | ❌ | H-Cl 键偏强，Cl· 难以从 HX 中夺 H |
| **HBr** | 366 | **适中** | ✅ | **最佳平衡**——既能夺 H 生成 Br·，又能有效加成 |
| **HI** | 298 | 极低 | ❌ | I· 太稳定，难以进攻双键；且 HI 是还原剂 |

> ⚠️ **竞赛高频陷阱**：过氧化物效应**只对 HBr 有效**！题目中出现"过氧化物存在下 HX 与烯烃加成"，如果 X = Cl 或 I，不要写反马氏产物。

### 4.5 三种反马氏方法的对比

| 方法 | 条件 | 机理 | 产物 | 立体化学 |
|:---|:---|:---|:---|:---:|
| **过氧化物效应** | HBr + ROOR/hv | 自由基链式 | 反马氏溴代烷 | 无特殊要求 |
| **硼氢氧化-氧化** | BH₃/H₂O₂/OH⁻ | 协同四中心 | 反马氏醇 | **顺式加成** |
| **催化氢化**（不是反马氏）| H₂/Pd/C | 表面催化 | 顺式加氢 | 顺式 |

## 五、NBS 与烯丙位/苄位溴代

### 5.1 NBS 是什么？

NBS（N-Bromosuccinimide，N-溴代丁二酰亚胺）是**烯丙位/苄位选择性溴代试剂**。

```
       O
       ||
    N—Br
       ||
       O
```

N-Br 键较弱（容易均裂），是 Br· 的间接来源。

### 5.2 NBS 的独特角色——"低浓度 Br₂ 维持器"

NBS 的核心价值不是"提供 Br⁺ 或 Br·"，而是**通过以下反应维持体系中 Br₂ 处于极低浓度**：

```
NBS + HBr → 琥珀酰亚胺 + Br₂
```

**为什么要维持低 Br₂ 浓度？**

| 条件 | Br₂ 浓度 | 反应路径 | 产物 |
|:---|:---:|:---|:---|
| NBS + CCl₄ + 引发剂 | 极低 | **自由基取代（烯丙位）** | 烯丙基溴（双键保留）|
| Br₂ 直接（高浓度）| 高 | 亲电加成（AdE）| 邻二溴化物（双键消失）|

NBS 通过消耗 HBr 再生 Br₂，使 [Br₂] 始终极低 → **加成被抑制，取代占优**。

### 5.3 NBS 烯丙位溴代的完整机理

**链引发**：
```
AIBN →Δ 2 (CH₃)₂C·CN + N₂
(CH₃)₂C·CN + NBS → (CH₃)₂C(CN)Br + 琥珀酰亚胺自由基
琥珀酰亚胺自由基 → 分解 → Br· （实际通过多步）
```

简化的引发路径：自由基夺取 NBS 中的 Br 或引发 Br₂ 均裂。

**链增长**：
```
Br· +   环己烯    → HBr + 烯丙基自由基（共振稳定，3 个等价共振式）
                                   ↓
                              Br
                              |
                              ║
                             ║
                              |
                             (三个共振式，单电子分布在 C1/C3/C5)
                              
烯丙基自由基 + NBS → 3-溴环己烯 + 琥珀酰亚胺自由基
（琥珀酰亚胺自由基与 HBr 反应再生 Br₂，循环）
```

### 5.4 NBS 的条件与产物

| 条件 | 底物 | 产物 |
|:---|:---|:---|
| NBS + CCl₄ + AIBN/hv + Δ | 环己烯 | 3-溴环己烯（烯丙位） |
| NBS + CCl₄ + AIBN/hv + Δ | 甲苯 | 苄溴（PhCH₂Br） |
| NBS + CCl₄ + AIBN/hv + Δ | 丙烯 | 3-溴丙烯（CH₂=CH-CH₂Br） |
| Br₂ + CCl₄（无 NBS）| 环己烯 | 1,2-二溴环己烷（加成） |

> ⚠️ **易混点**：NBS 是**取代试剂**，不是加成试剂。条件中出现 NBS + hv/AIBN → 优先判断为烯丙位/苄位取代，而非双键加成。

### 5.5 位点选择性：烯丙基 > 苄基 > 叔 > 仲 > 伯

**原因**：BDE 决定优先度

| C-H 类型 | BDE (kJ/mol) | 自由基稳定性 |
|:---|:---:|:---|
| 烯丙位 C-H | ~370 | 烯丙基自由基（离域稳定）|
| 苄位 C-H | ~375 | 苄基自由基（离域稳定）|
| 叔 C-H | ~400 | 三级自由基（超共轭）|
| 仲 C-H | ~410 | 二级自由基 |
| 伯 C-H | ~420 | 一级自由基 |

**NBS 条件下 Br· 优先夺取 BDE 最小的 H 位点** → 烯丙位/苄位 >> 叔 > 仲 > 伯。

## 六、自由基聚合（要点）

自由基聚合是自由基反复加成到单体双键的链式反应：引发剂 → 2 R·；R· + n CH₂=CHX → R-(CH₂CHX)n-CH₂CHX·；终止于偶联或歧化。

**竞赛要点**：工业制备聚乙烯、聚氯乙烯、聚苯乙烯的重要方法。分子量分布较宽。**不要求掌握聚合动力学详细推导**。

---

## 七、自由基稳定化的电子效应总结

| 效应 | 作用方式 | 强度 | 典型稳定化能 | 示例 |
|:---|:---|:---:|:---:|:---|
| **共轭离域** | SOMO 与 π* 重叠，单电子分散 | 极强 | ~50-80 kJ/mol | 烯丙基、苄基自由基 |
| **超共轭** | σ(C-H) 与 SOMO 部分重叠 | 中等 | ~10-20 kJ/mol/每个 C-H | 叔 > 仲 > 伯 |
| **诱导效应** | 电负性原子通过 σ 键吸/给电子 | 弱但可叠加 | ~5-15 kJ/mol | ·CH₂Cl、·CH₂OH |
| **推拉效应** | 同时吸电子+给电子 | **极强** | ~80-120 kJ/mol | ·CH(CN)(OEt) |
| **位阻效应** | 大基团阻碍碰撞 | 动力学稳定 | 可变 | Ph₃C·、TEMPO |

**与碳正离子的关键区别**：

| 中间体 | 离域效应的重要性 | 超共轭效应 | 杂原子效应 |
|:---|:---:|:---:|:---:|
| **碳正离子 R₃C⁺** | 重要（但烯丙基正离子不如叔丁基稳定） | 主导 | 吸电子(-I)有害 |
| **碳自由基 R₃C·** | **更加重要**（烯丙基 > 叔丁基）| 重要 | 吸电子 OR 给电子均有利 |

## 八、典型例题

### 例1 异丁烷溴代的选择性 ⭐⭐

**题干**：异丁烷与 Br₂ 在光照下反应，写出主产物并解释。

**思路**：先判断反应类型（自由基链式），再比较不同位点 C-H 的 BDE，最后确定主要抽氢位点。

**解析**：

异丁烷结构为 (CH₃)₃CH，有两种氢：
- 9 个伯氢（CH₃ 上）
- 1 个叔氢（CH 上）

Br· 的高度选择性（3°:1° ≈ 1600:1）意味着叔氢被优先夺取的概率远超伯氢，即使伯氢数量多 9 倍。

```
(Br· 选择性主导)
Br· + (CH₃)₃CH → HBr + (CH₃)₃C·   (叔自由基，最稳定)
(CH₃)₃C· + Br₂ → (CH₃)₃CBr + Br·  (叔丁基溴 + 再生 Br·)
```

**答案**：主产物为**叔丁基溴**((CH₃)₃CBr)，含量 > 99%。Br· 的高选择性使得抽氢几乎专一发生在 BDE 最低的叔 C-H 位点。

**反思**：
- 如果换成 Cl₂，叔产物比例降为约 36%（伯产物约占 64%）——因为 Cl· 选择性低
- 溴代先看"有没有三级/烯丙位/苄位 H"，若有，产物几乎唯一

---

### 例2 丙烯的 HBr 加成对比 ⭐⭐⭐

**题干**：写出以下两个反应的主产物，并解释差异：
(a) CH₃CH=CH₂ + HBr（无过氧化物）
(b) CH₃CH=CH₂ + HBr/ROOR

**思路**：先判断反应条件→确定是离子型还是自由基型→再写出各自的区域选择性。

**解析**：

(a) **离子型加成（马氏规则）**：
- H⁺ 进攻双键：H⁺ 加在 CH₂ 端（生成更稳定的 2° 碳正离子）
- 产物：**2-溴丙烷**（Br 在 C2）
```
CH₃CH=CH₂ + H⁺ → CH₃-ĊH-CH₃  (2° 碳正离子)
CH₃-ĊH-CH₃ + Br⁻ → CH₃CHBrCH₃
```

(b) **自由基型加成（反马氏，过氧化物效应）**：
- ROOR 均裂产生 Br·（见 §4.2 机理）
- Br· 优先进攻末端 CH₂（生成更稳定的 2° 自由基）
- 产物：**1-溴丙烷**（Br 在 C1）
```
Br· + CH₃CH=CH₂ → CH₃-ĊH-CH₂Br  (2° 自由基)
CH₃-ĊH-CH₂Br + HBr → CH₃CH₂CH₂Br + Br·
```

**答案**：(a) 2-溴丙烷（马氏产物），(b) 1-溴丙烷（反马氏产物）。

**反思**：
- 同一个底物+不同条件→不同机理→不同产物，这是竞赛中"机理决定产物"的经典案例
- 注意两题中 Br 的位置正好相反——考试中常作为对比题考查

---

### 例3 完整链式反应机理书写 ⭐⭐⭐

**题干**：过氧化物存在下，写出 HBr 与 2-甲基丙烯反应形成 1-溴-2-甲基丙烷的完整链式机理（三步：引发、增长、终止）。

**思路**：先写引发产生 Br·，再写 Br· 对烯烃的加成（注意区域选择性），再写链转移生成产物和再生 Br·，最后写可能的终止方式。

**解析**：

**链引发**：
```
RO-OR →Δ 2 RO·              (过氧化物均裂)
RO· + HBr → ROH + Br·       (产生 Br·)
```

**链增长**：
```
                                CH₃
                                |
Br· + (CH₃)₂C=CH₂ → (CH₃)₂C-ĊH₂Br → (CH₃)₂Ċ-CH₂Br
                   (Br· 加到 CH₂ 上，生成 3° 自由基——更稳定！)
                                ↓
                                HBr
                                ↓
                     (CH₃)₂CH-CH₂Br + Br·   (产物 + 再生 Br·)
```

**链终止**（任写 2-3 种）：
```
Br· + Br·   → Br₂
R· + Br·    → R-Br
R· + R·     → R-R  (偶联副产物)
```

**反思**：
- 初学者最常见的错误是只写链增长，遗漏引发和终止
- 竞赛中链式反应机理题的标准评分通常按"三步各占若干分"

---

### 例4 NBS 烯丙位溴代 ⭐⭐⭐

**题干**：环己烯与 NBS/AIBN/CCl₄ 在回流条件下反应，写出主产物并解释为什么不是 Br₂ 的加成产物。

**思路**：识别 NBS 条件 → 考虑低 Br₂ 浓度 → 自由基链式取代 → 烯丙基自由基共振稳定 → 产物。

**解析**：

**主产物**：**3-溴环己烯**（烯丙位溴代，双键保留）

```
    ╕          ╕
   ║   + NBS/AIBN/CCl₄ →Δ   ║    + 琥珀酰亚胺
    ║                       ║
    ╙                       Br ╙
  (环己烯)               (3-溴环己烯)
```

**为什么不是 Br₂ 加成产物（1,2-二溴环己烷）？**

```
NBS + HBr → 琥珀酰亚胺 + Br₂    (该反应不断消耗 HBr)
```

NBS 与链增长中产生的 HBr 反应，使体系中的 Br₂ 始终处于**极低浓度**。高浓度 Br₂ 才会发生对双键的亲电加成（需要 Br₂ 直接进攻π键）；低浓度下 Br· 的抽氢路径（夺取烯丙位 H）才是主反应。

**链增长关键步**：
```
Br· + 环己烯 → HBr + 烯丙基自由基
   (Br· 夺取烯丙位 H 后产生的自由基有 3 个等价共振式，
    单电子分布在 3 个碳上，极稳定)
```

**答案**：3-溴环己烯。NBS 体系中极低 [Br₂] 使亲电加成路径被抑制，Br· 取代路径占优。

**反思**：区分 NBS 条件与 Br₂ 条件是竞赛常考的选择性判断——NBS/CCl₄/hv → 取代（烯丙位），Br₂/CCl₄ → 加成（双键）。

### 例5 自由基环化选择性 ⭐⭐⭐⭐

**题干**：5-己烯-1-基自由基（·CH₂-CH₂-CH₂-CH₂-CH=CH₂）发生分子内环化，预测主要产物并说明理由。

**思路**：识别为分子内自由基环化 → 两种竞争路径（5-exo-trig vs 6-endo-trig）→ 用 Baldwin 规则判断 → 分析动力学与热力学。

**解析**：

底物：·CH₂-CH₂-CH₂-CH₂-CH=CH₂

**路径 A：5-exo-trig（自由基进攻 C₅=C₆ 的末端）**
```
     ·CH₂    H₂C=CH₂           H₂C——CH₂
      |       |       →         |     |
    H₂C——CH₂  CH₂             H₂C——CH₂
          \  /                   \   /  \
           CH₂                   CH₂  CH₂
```
形成五元环 + 伯碳自由基（环戊烷甲基衍生物）

**路径 B：6-endo-trig（自由基进攻 C₅=C₆ 的内部）**
形成六元环 + 仲碳自由基（环己烷衍生物）

**Baldwin 规则判断**：

| 环化模式 | Baldwin 分类 | 是否有利 |
|:---:|:---:|:---:|
| 5-exo-trig | 5-exo-trig | ✅ **有利** |
| 6-endo-trig | 6-endo-trig | ✅ 可发生但较慢 |

**主产物**：**5-exo-trig 环化产物**（甲基环戊烷衍生物）

**理由**：
1. **Baldwin 规则**：5-exo-trig 是**有利**的环化模式，过渡态几何更优（轨道重叠更好）
2. **动力学优先**：即使 6-endo 生成更稳定的仲碳自由基，但 5-exo 的环化速率为更快（活化能低）
3. **实验证据**：大多数 5-己烯基自由基的环化以 5-exo 为主，5-exo:6-endo 比例通常 > 10:1

**答案**：主产物为 5-exo-trig 环化产物（五元环衍生物），少量 6-endo-trig（六元环衍生物）。

**反思**：
- 自由基环化的核心口诀：**"5-exo-trig 优先"**
- 这条规则在竞赛机理推断中经常用到——看到分子内既有自由基又有双键，优先考虑 5-exo 环化

---

### 例6 自由基稳定性综合排序 ⭐⭐⭐

**题干**：将下列自由基按稳定性从大到小排序，并用电子效应解释：
(a) ·CH₃、(b) ·CH₂CH=CH₂、(c) ·C(CH₃)₃、(d) ·CH₂CN、(e) ·CH₂Cl

**思路**：运用自由基稳定性 5 规则逐条分析——先找共轭离域（最优先），再看杂原子效应，然后比较超共轭。

**解析**：

| 自由基 | 稳定化机制 | 排序 |
|:---|:---|---:|
| (b) ·CH₂CH=CH₂ | **烯丙基——共轭离域**（单电子分散到 3 个碳） | 1 |
| (d) ·CH₂CN | **杂原子邻位**（CN 吸电子，降低 SOMO 能量） | 2 |
| (e) ·CH₂Cl | **杂原子邻位**（Cl 吸电子，降低 SOMO 能量；同时 Cl 孤对电子有 +C 效应） | 3 |
| (c) ·C(CH₃)₃ | **叔自由基**——9 个 C-H 键超共轭稳定 | 4 |
| (a) ·CH₃ | **无稳定化因素** | 5 |

**答案**：·CH₂CH=CH₂ > ·CH₂CN > ·CH₂Cl > ·C(CH₃)₃ > ·CH₃

**反思**：
- 烯丙基自由基凭借共轭离域稳居第一——这是与碳正离子排序的最大不同
- 杂原子邻位的稳定化（规则 3）比超共轭（规则 2）更强——·CH₂CN 比叔丁基更稳定
- 吸电子基和给电子基都能稳定自由基（规则 3）——这是自由基相对于碳正离子的独特之处

## 九、常见误区

> [!warning] **误区 1：把自由基稳定性顺序照搬碳正离子**
> 学生常认为"叔 > 烯丙基"在自由基中也成立。事实：自由基中烯丙基/苄基（离域稳定）> 叔丁基（超共轭）。关键区别在于自由基的离域不产生电荷分离，因此特别有利。

> [!warning] **误区 2：以为所有 HX 都有过氧化物效应**
> 学生常推广"ROOR → 反马氏"到 HCl、HI。事实：**只有 HBr** 有过氧化物效应。HCl 的 Cl· 活性高但夺 H 慢，HI 的 I· 太稳定不进攻。竞赛选择题中这是经典陷阱——看到 HCl/ROOR 不要写反马氏产物。

> [!warning] **误区 3：混淆 NBS 的取代与 Br₂ 的加成**
> 学生看到溴试剂就写邻二溴化物。事实：NBS + 引发剂/光照 的条件是**烯丙位取代**（保留双键），Br₂/CCl₄ 才是亲电加成（双键消失）。记住"NBS 是取代试剂，不是加成试剂"。

> [!warning] **误区 4：写自由基机理时使用双钩箭头**
> 这是最容易被扣分的技术性错误。自由基反应必须用**单电子鱼钩箭头（fishhook）**——一个箭头代表一个电子移动。离子型反应才用双钩箭头。检查标准：自由基步骤中所有箭头都应该是单钩。

> [!warning] **误区 5：链式反应只写增长不写引发和终止**
> 竞赛中链式机理题通常要求完整的三段式。只写链增长会被扣分。尤其注意链终止至少需要写出两种方式（偶联和歧化都写）。

> [!warning] **误区 6：忽略 O₂ 对自由基反应的抑制**
> O₂ 是三线态双自由基，能捕获几乎所有碳自由基。在空气环境下做自由基反应常失败——这是实验室常识，也是竞赛信息题的潜在考点。

> [!warning] **误区 7：自由基环化优先考虑六元环**
> 受"六元环最稳定"的直觉影响，学生常猜 6-endo-trig 产物。事实：自由基环化的主流是 **5-exo-trig 优先**（Baldwin 规则），即使生成的自由基不如 6-endo 产物稳定。

## 十、方法总结

### 10.1 自由基反应四问法

面对任何可能的自由基题，先问四个问题：

```
① 有没有自由基触发信号？(hv/ROOR/AIBN/NBS/SET)
    ↓ 有 → 进入自由基赛道
    ↓ 没有 → 先别急着判自由基
② 链增长的关键步骤是什么？（抽氢? 加成? 环化?）
    ↓
③ 哪个自由基/过渡态更稳定？（离域? 超共轭? 几何优势?）
    ↓
④ 题目要你回答什么？（主产物? 位点? 区域? 还是完整机理?）
```

### 10.2 卤代选择性决策树

```
判断卤代产物时，按以下顺序：
① 底物中是否有烯丙位/苄位 H？
   ├── 有 → 优先考虑烯丙位/苄位取代（BDE 最低）
   └── 无 → 进入下一步
② 是否存在三级 H？
   ├── 是 → 判断卤素种类
   │   ├── Br₂ → 几乎专一在三级位（选择性 > 1600:1）
   │   └── Cl₂ → 按比例计算（3°:2°:1° ≈ 5:4:1）
   └── 否 → 按仲/伯比例计算
```

### 10.3 自由基反应口诀

```
自由基，一条链，引发增长和终断；
稳定性，分三档，烯丙苄基王中王；
超共轭，按碳算，三级二级强于莽；
卤代中，要分辨，溴挑氢氯随便抢；
过氧化物 HBr 独，反马加成看自由基；
NBS 是取代将，低浓溴来调控场；
环化记牢五元环，Baldwin 规则握手掌。
```

| 卤素/条件 | 相对反应活性（3°:2°:1°） | 特征 |
|:---:|:---:|:---|
| Cl₂/hv | 5.0 : 3.8 : 1.0 | 中等选择性，需要计算 |
| Br₂/hv | 1600 : 82 : 1.0 | 高选择性，专一性 |
| NBS/AIBN/hv | 烯丙位/苄位 >> 3° > 2° > 1° | 烯丙位专一 |

```
苄基/烯丙基（离域） > 叔（超共轭） > 仲 > 伯 > 甲基 > 乙烯基/苯基
                              (与碳正离子最大区别)
```

|:---|:---|:---:|

| HBr + 烯烃 + ROOR/hv | 自由基链式 | 反马氏 |
| HCl/HI + ROOR | ❌ 无效 | — |

### 11.5 自由基环化 Baldwin 规则速查

| 模式 | 5-exo-trig | 6-endo-trig |
|:---:|:---:|:---:|
| 有利性 | ✅ **有利** | ✅ 可行（较慢）|
| 环大小 | 五元环 | 六元环 |
| 动力学 | 快（过渡态优）| 慢 |
| 生成的自由基 | 伯/仲 | 仲/叔 |

## 十二、练习题

### 12.1 基础巩固（⭐⭐）

1. 写出丙烷（CH₃CH₂CH₃）在光照下与 Cl₂ 反应时一氯代物的比例（已知 1°:2° 活性比 = 1:3.8）。

2. 完成反应式：环己烯 + NBS/AIBN/CCl₄/hv → ? 并判断反应类型。

3. 判断正误并说明理由：过氧化物存在下，HCl 与 1-丁烯的加成得到反马氏产物。

4. 写出 2-甲基丙烯 + HBr/ROOR 的主产物。

### 12.2 提高训练（⭐⭐⭐）

5. 写出以下反应的完整链式机理（三阶段）：
   (CH₃)₂C=CH₂ + HBr/BPO → (CH₃)₂CHCH₂Br

6. 比较以下自由基的稳定性，从大到小排序并解释：
   ·CH₃、·CH(CH₃)₂、·C(CH₃)₃、·CH₂-Ph、·CH₂-CH=CH₂

7. 5-戊烯-1-基自由基（·CH₂CH₂CH₂CH=CH₂）环化，预测主产物。与 5-己烯-1-基自由基相比，环化产物有何不同？

8. 解释现象：同样含一个双键和一个烯丙位 C-H 的底物，与 NBS/AIBN 反应得烯丙基溴，而与过量 Br₂/CCl₄ 反应得邻二溴化物。

---

## 参考答案

1. 丙烷有 6 个 1° H 和 2 个 2° H。1-氯丙烷：6×1 = 6，2-氯丙烷：2×3.8 = 7.6。总和 = 13.6。1-氯丙烷 = 44%，2-氯丙烷 = 56%。

2. 3-溴环己烯，自由基取代（烯丙位溴代）。

3. 错误。过氧化物效应只对 HBr 有效。HCl 不能发生自由基链式反应（H-Cl 键太强，Cl· 难以从 HCl 中夺 H）。

4. 1-溴-2-甲基丙烷（反马氏产物）。

5. 链引发：BPO →Δ 2 Ph· + 2 CO₂；Ph· + HBr → PhH + Br·。
   链增长：Br· + (CH₃)₂C=CH₂ → (CH₃)₂Ċ-CH₂Br（3° 自由基）；
   (CH₃)₂Ċ-CH₂Br + HBr → (CH₃)₂CH-CH₂Br + Br·。
   链终止：Br· + Br· → Br₂；R· + Br· → R-Br；R· + R· → R-R。

6. ·CH₂-Ph ≈ ·CH₂-CH=CH₂（离域，最稳定）> ·C(CH₃)₃（3°）> ·CH(CH₃)₂（2°）> ·CH₃（1°）。

7. 5-戊烯-1-基自由基只能发生 5-exo-trig 环化（五元环），因链长不足以形成 6-endo。与 5-己烯-1-基（可竞争 5-exo vs 6-endo）不同。

8. NBS 维持极低 Br₂ 浓度 → Br· 抽烯丙位 H → 烯丙基自由基（共振稳定）→ 低浓度 Br₂ 捕获得烯丙基溴（取代）。高浓度 Br₂ 直接亲电加成（溴鎓离子）→ 邻二溴化物（加成）。
