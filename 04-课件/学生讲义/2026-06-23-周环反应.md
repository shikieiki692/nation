# 周环反应

> **适用**：第三轮（提高班）/ 第四轮（冲刺班）| **前置要求**：[[04-课件/学生讲义/2026-06-23-有机化学基础]]（分子轨道、HOMO/LUMO）
> **对应考纲**：考纲49-周环反应
> **源素材**：ABOC Ch7（周环反应）+ Clayden Ch34（环加成）+ Clayden Ch35（σ重排与电环化）
>
> 周环反应是有机化学中"最美"的反应类型——没有中间体、没有正负电荷、电子绕环协同移动。理解周环反应的核心不是背产物，而是**绘制前线轨道、判断对称性匹配**。

---

## 学习目标

- ✅ 能用前线轨道理论（FMO）判断周环反应的热/光允许性
- ✅ 能预测 Diels-Alder 反应的区域选择性和 endo/exo 立体化学
- ✅ 能用电环化规则（4n 热旋光 vs 4n+2 热对旋）预测产物
- ✅ 能识别 [1,5]-H 迁移、[3,3]-σ 迁移（Cope/Claisen）并判断可行性
- ✅ 能区分 [2+2] 和 [4+2] 环加成的热/光条件差异
- ✅ 能将复杂多步反应拆解为若干独立的周环步骤

## 一、周环反应总论

### 1.1 什么是周环反应？

**定义**：电子绕一个环移动的一步协同反应，任何中间体上都没有正负电荷，**没有中间体**。

| 特征 | 说明 |
|:---|---|
| **协同机理** | 没有自由基/正离子/负离子中间体 |
| **环状过渡态** | 电子围绕一个环移动 |
| **轨道对称性控制** | Woodward–Hoffmann 规则决定可行性 |
| **立体专一性** | 反应物立体化学→产物立体化学直接传递 |

### 1.2 三大类型

![[media/wh-correlation.png|Woodward-Hoffmann 相关图——4π 电子体系的光化学电环化反应]]
*图 4π 电子体系的光化学电环化相关图。Woodward–Hoffmann 规则通过轨道对称性关联判断周环反应在热/光条件下的允许性。4n 体系热禁阻但光允许（对旋），4n+2 则相反（Wikimedia Commons）*

| 类型 | σ 键变化 | 特征 | 典型例子 |
|:---:|:---|---:|:---|
| **环加成** | Δσ = +2 | 两个 π 体系形成两根新 σ 键 | Diels-Alder [4+2] |
| **电环化** | Δσ = ±1 | 共轭多烯关/开环 | 丁二烯→环丁烯 |
| **σ 迁移** | Δσ = 0 | σ 键从一处迁移到另一处 | Cope、Claisen |

## 二、前线轨道理论（FMO）

### 2.1 核心思想

反应的发生需要**HOMO（最高占据轨道）与 LUMO（最低空轨道）的有效重叠**。

**两条件**：
1. **对称性匹配**：HOMO 与 LUMO 的轨道瓣同相重叠
2. **能量接近**：HOMO-LUMO 能隙越小反应越快

### 2.2 对称性分析步骤

```
1. 确定参与反应的 π 电子数
2. 画出轨道的节面和对称性
3. 判断 HOMO-LUMO 是否同相重叠
4. 同相 → 热允许；异相 → 热禁阻（光照可能允许）
```

### 2.3 Diels-Alder 的 FMO 分析

在 D-A 反应中：
- **二烯的 HOMO（ψ₂）** + **亲双烯体的 LUMO（π*）** → 对称性匹配 → **热允许**
- 给电子基（EDG）在二烯上 → 升高 HOMO 能量 → 反应更快
- 吸电子基（EWG）在亲双烯体上 → 降低 LUMO 能量 → 反应更快

> **竞赛口诀**："EDG 在二烯，EWG 在亲双烯体，越快。"

## 三、Diels-Alder 反应

### 3.1 概述

$$
\ce{共轭二烯 + 亲双烯体 ->[加热] 环己烯衍生物}
$$

- **[4+2] 环加成**——周环反应的"王冠"
- **条件**：加热（通常 100–150°C），无需催化剂
- **限制**：二烯必须能采取 **s-cis 构象**

### 3.2 立体化学——syn 加成

亲双烯体的 cis/trans 构型完全保留在产物中：

| 亲双烯体 | 产物 |
|:---|---|
| 顺式烯烃 | 顺式取代环己烯 |
| 反式烯烃 | 反式取代环己烯 |

### 3.3 endo 规则

亲双烯体上的吸电子取代基倾向于在产物的 **endo 位**（桥位，靠近新形成的双键）。

```
       EWG                EWG
         \                  /
          C                C
         /  \            /  \
  二烯 +     C  →  endo /    \  为主
          \ /          \    /
           C            \  /
                           \
```

**原因**：endo 过渡态中，亲双烯体的 π* 轨道与二烯的 π 体系存在**次级轨道相互作用**（额外的成键作用），降低过渡态能量。

| 产物 | 热力学 | 动力学 |
|:---|---:|:---:|
| **endo** | 较不稳定 | **过渡态更稳定 → 形成更快** |
| **exo** | 较稳定 | 过渡态能量较高 |

> **注意**：endo 是**动力学产物**，exo 是**热力学产物**。高温长时间反应可能得 exo。

### 3.4 区域选择性——邻位/对位规则

| 二烯取代模式 | 产物取向 |
|:---|---|
| EDG 在二烯 **1-位** | 与亲双烯体 EWG **邻位** |
| EDG 在二烯 **2-位** | 与亲双烯体 EWG **对位（1,4-位）** |

**FMO 解释**：最大轨道系数之间成键 → 决定区域选择性。

### 3.5 反向 Diels-Alder

环己烯衍生物在高温下裂解回二烯 + 烯烃。在竞赛中常作为"隐藏的二烯"出现——当看到环己烯衍生物在加热条件下给出共轭二烯产物时，立刻想到 Retro-D-A。

### 3.6 分子内 Diels-Alder（IMDA）

二烯 + 亲双烯体在**同一分子内** → 形成多环产物。

### 3.7 特殊变体

| 变体 | 说明 | 竞赛价值 |
|:---|---|:---:|
| **Lewis 酸催化 D-A** | SnCl₄ 等降低 LUMO 能量，提高选择性和速率 | ⭐⭐ |
| **水溶剂效应** | D-A 在水中速率可提高 700 倍（"on water"）| ⭐ 了解 |
| **[4+3] 环加成** | 氧烯丙基阳离子 + 二烯 → 七元环 | ⭐ 前沿 |

## 四、[2+2] 环加成

### 4.1 一般规则

| 条件 | 允许性 | 原因 |
|:---|:---:|:---|
| **热** | **禁阻** | 两个乙烯的 HOMO-LUMO 对称性不匹配（4e 反芳香过渡态）|
| **光** | **允许** | 光照激发一个乙烯 → 激发态 HOMO 与基态 LUMO 匹配 |

### 4.2 烯酮的 [2+2] 例外

**烯酮（Ketene, R₂C=C=O）** 的 [2+2] 是**热允许的**——烯酮的 LUMO 独特（弯曲的累积二烯体系），与烯烃 HOMO 可同相重叠。

**竞赛应用**：
- 烯酮 + 烯烃 → 环丁酮
- 烯酮 + 亚胺 → β-内酰胺（**Staudinger 合成**，是抗生素合成的关键反应）

### 4.3 [2+2] 光环加成

$$
\ce{2 烯烃 ->[hv] 环丁烷}
$$

- **热禁阻，光允许**的标准案例
- 可用于合成环丁烷骨架

## 五、电环化反应

### 5.1 定义

共轭多烯的末端碳之间形成 σ 键（关环）或断裂 σ 键（开环）。

### 5.2 Woodward-Hoffmann 规则

| π 电子数 | 热反应 | 光反应 |
|:---:|:---:|:---:|
| **4n** | **旋光（conrotatory）** | 对旋（disrotatory）|
| **4n+2** | **对旋（disrotatory）** | 旋光（conrotatory）|

**记忆口诀**：
> "**4n 热旋光，4n+2 热对旋**"

```
4n 体系（如 4π 丁二烯）：
  ──→  热：两个末端向相同方向旋转（旋光）→ 反式产物
  ──→  光：两个末端向相反方向旋转（对旋）→ 顺式产物

4n+2 体系（如 6π 己三烯）：
  ──→  热：两个末端向相反方向旋转（对旋）→ 顺式产物
  ──→  光：两个末端向相同方向旋转（旋光）→ 反式产物
```

### 5.3 典型例子

| 底物 | π 电子数 | 热反应方式 | 产物立体化学 |
|:---|:---:|:---:|:---:|
| 1,3-丁二烯 | 4 (=4×1) | 旋光关环 | 反式环丁烯 |
| 1,3,5-己三烯 | 6 (=4×1+2) | 对旋关环 | 顺式环己二烯 |

### 5.4 Nazarov 环化——阳离子 4π 电环化

$$
\ce{二乙烯基酮 ->[Lewis酸] 环戊烯酮}
$$

- **本质**：戊二烯基正离子（pentadienyl cation）的 **4π 电环化关环**
- 方式：**旋光关环**
- 产物：**环戊烯酮**——天然产物合成中的重要骨架

## 六、σ 迁移反应

### 6.1 定义

σ 键从一处迁移到另一处，伴随 π 体系的重组。过渡态环大小 = 方括号中数字之和。

### 6.2 分类

| 类型 | 通式 | 典型例子 |
|:---|---|:---|
| **[1,n]-氢迁移** | H 跨越 n 个原子迁移 | [1,5]-H 迁移（热允许）|
| **[1,n]-碳迁移** | 碳基团跨越 n 个原子迁移 | [1,3]-C 迁移（通常禁阻）|
| **[m,n]-σ 迁移** | 两端同时迁移 | [3,3]-σ 迁移（Cope, Claisen）|

### 6.3 [1,5]-H 迁移

- **热允许**（六电子过程）
- 在天然产物全合成中常见（如维生素 D 的生物合成）
- 同面迁移，构型保持

```
      H
      |
    C=C-C=C-C     ——→    C-C=C-C=C
      \ /                  \
       C                    H-C
```

### 6.4 [3,3]-σ 迁移——Cope 重排

**Cope 重排**：纯碳版本的 [3,3]-σ 迁移。

$$
\ce{1,5-二烯 ->[加热] 1,5-二烯（异构）}
$$

- 对称时反应可逆（起始原料 = 产物）
- 通过引入取代基或官能团打破对称性 → 单向进行
- 六元环过渡态，**芳香过渡态稳定化**

**立体化学**：椅式过渡态 → 立体专一性

### 6.5 [3,3]-σ 迁移——Claisen 重排

**Claisen 重排**：含氧版本——烯丙基乙烯基醚 → γ,δ-不饱和羰基化合物。

$$
\ce{O-CH2-CH=CH2 + CH2=CH ->[加热] HO-C(R)=CH-CH2-CH=CH2}
$$

**脂肪族 Claisen**（Claisen-Cope）：
- 烯丙基乙烯基醚 → γ,δ-不饱和羰基化合物
- 驱动力：C=O 的稳定性 > C=C

**芳香族 Claisen**：
- 芳基烯丙基醚 → 邻烯丙基酚（两步：σ 迁移 + 质子转移恢复芳香性）

### 6.6 选择规则总结

| 迁移类型 | 同面/异面 | 4n+2 体系 | 4n 体系 |
|:---:|:---:|:---:|:---:|
| **[1,n]-H** | 同面 | 热允许（n=5,9...）| 光允许 |
| **[1,n]-C + 构型保持** | 同面 | 热允许（n=5...）| 禁阻 |
| **[1,n]-C + 构型翻转** | 异面 | 禁阻 | 热允许（n=3...）|
| **[3,3]** | — | **热允许** | — |

> **竞赛重点**：[3,3]-σ 迁移（Cope/Claisen）是竞赛频率最高的 σ 迁移反应；[1,5]-H 迁移次之。

## 七、Ene 反应

### 7.1 定义

烯烃（ene）+ 亲烯体（enophile）之间的协同氢转移 + 双键迁移 + σ 键形成。

$$
\ce{R-CH=CH2 + X=Y -> R-C(CH3)=CH-CH2-X-Y-H}
$$

### 7.2 与 D-A 的关系

- **Ene 反应 = "只形成一根 σ 键的 Diels-Alder"**
- 都是六电子过程、六元环过渡态
- 协同机理，无中间体

## 八、竞赛应用与常见考法

### 8.1 五种考法

| 考法 | 特征 | 解题策略 |
|:---|---|:---|
| **产物预测** | 给反应物+条件（热/光）→ 预测产物 | 先判断反应类型 → 应用 WH 规则 → 注意立体化学 |
| **反应类型识别** | 给机理图 → 判断是哪类周环反应 | 数 π 电子数 → 看 σ 键变化 → 归类 |
| **立体化学判断** | 给产物结构 → 反推 endo/exo | 看桥头取代基方向：endo = 靠近双键侧 |
| **FMO 分析** | 画轨道图 → 判断允许性 | 画 HOMO/LUMO → 看是否同相重叠 |
| **多步串联** | 天然产物合成路线 → 拆解成多个周环步骤 | 找 D-A → 电环化 → σ 迁移的间断点 |

### 8.2 记忆工具箱

| 需要记什么 | 怎么记 |
|:---|---:|
| D-A：二烯+亲双烯体 → 环己烯 | s-cis 二烯必须，EWG 活化亲双烯体 |
| endo 规则 | **次级轨道相互作用**稳定 endo 过渡态 |
| 4n 热旋光，4n+2 热对旋 | "4 旋光，6 对旋" |
| [3,3]-σ 迁移 = Cope/Claisen | 六元环过渡态，热允许 |
| [2+2] 热禁阻、光允许 | 除**烯酮**外 |
| D-A → 热；[2+2] → 光 | 周环反应的"阴阳"关系 |

### 8.3 高频陷阱

| 陷阱 | 正确认识 |
|:---|---|
| "endo 产物更稳定" | endo 是**动力学**产物，exo 才是**热力学**更稳定的 |
| "[2+2] 加热也可以" | 一般不可。**烯酮的 [2+2] 是唯一的常见热允许例外** |
| "二烯必须稳定" | D-A 中二烯不需要稳定，亲双烯体需要 EWG 活化 |
| "电环化关环产物顺反无所谓" | 旋光/对旋**决定产物的立体化学**——一定要画出末端碳取代基的旋转方向 |
| "σ 迁移和自由基迁移一样" | σ 迁移是**协同的周环反应**，无自由基中间体 |

## 九、反应类型快速鉴别

拿到一道周环反应题，按以下顺序判断：

```
① 反应物数：
   - 2 个 → 环加成（[2+2] 或 [4+2]）
   - 1 个 → 看 σ 键变化：
     - σ 键断裂+形成在不同位置 → σ 迁移
     - 末端碳之间成键/断键 → 电环化

② 反应条件：
   - 加热 → 热允许（D-A, [3,3]-σ 迁移, 电环化 4n+2）
   - 光照 → 光允许（[2+2], 电环化 4n）

③ π 电子数（电环化用）：
   - 4n → 热旋光，光对旋
   - 4n+2 → 热对旋，光旋光
```
