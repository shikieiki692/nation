# 过渡金属催化

> **适用**：第三轮（提高班）/ 第四轮（冲刺班）| **前置要求**：[[04-课件/学生讲义/2026-06-23-配位化合物基础]]（晶体场、18电子规则）、[[04-课件/学生讲义/2026-06-23-有机化学基础]]（电子效应）
> **对应考纲**：考纲51-金属有机 / 决赛金属催化循环 / 初赛交叉偶联
> **源素材**：ABOC Ch8（过渡金属催化）+ Clayden Ch40（金属有机化学）+ 初赛讲义第5讲
>
> 过渡金属催化是现代有机合成化学的核心工具。本讲以 **Pd 催化**为主线，覆盖偶联反应、氧化、复分解、C–H 活化四大板块，重点是**催化循环的拆解与理解**。

---

## 学习目标

- ✅ 能拆解催化循环为基元反应（氧化加成/迁移插入/β-H 消除/还原消除）
- ✅ 能用 18e 规则/16e 规则判断配合物的稳定性与反应活性
- ✅ 能区分 Heck、Suzuki、Negishi、Stille、Sonogashira 的底物与机理
- ✅ 能画出 Grubbs 催化烯烃复分解的金属杂环丁烷机理
- ✅ 能理解 Wacker 氧化和 Buchwald-Hartwig 胺化的催化循环
- ✅ 能预测 Tsuji-Trost 反应的区域选择性和立体化学

## 一、金属有机化学基础

### 1.1 18 电子规则

稳定的过渡金属配合物，其中心金属的价壳层具有 **18 电子**（填满 s + 5d + 3p 轨道）。

| 项目 | 内容 |
|:---|---|
| **电子来源** | 金属自身的价电子 + 所有配体贡献的电子 |
| **计数方法** | 新族号（1–18）直接给出金属价电子数，Pd = 10, Cr = 6 |
| **16e 例外** | Ni、Pd、Pt 族可形成稳定的**平面正方形 16e 配合物**（D₄h，HOMO-LUMO 间隙大）|
| **反应活性** | <18e 为络合不饱和，具有反应活性（催化循环所需）|

**常见配体的电子贡献**：

| 配体类型 | η | 形式电荷 | 贡献电子数 |
|:---|---:|:---:|:---:|
| 卤素、烷基、芳基（σ 键）| η¹ | –1 | 2 |
| 烯烃 | η² | 0 | 2 |
| CO、PPh₃、胺 | η¹ | 0 | 2 |
| π-烯丙基阳离子 | η³ | +1 | 2 |
| π-烯丙基阴离子 | η³ | –1 | 4 |
| 共轭双烯 | η⁴ | 0 | 4 |
| 环戊二烯基（Cp）| η⁵ | –1 | 6 |
| 苯 | η⁶ | 0 | 6 |

**实例**：
- Pd(PPh₃)₄：10 + 4×2 = **18e ✅**
- PdCl₂(MeCN)₂（Pd²⁺：8e）：8 + 4×2 = **16e （稳定且活泼）**
- Cr(苯)(CO)₃：6 + 6 + 3×2 = **18e ✅**

### 1.2 催化循环四步曲

![[media/suzuki-cycle.png|Suzuki 交叉偶联的催化循环（Pd 催化）]]
*图 Suzuki-Miyaura 交叉偶联反应的催化循环。从 Pd(0) 开始，经历氧化加成（Ar-X → Ar-Pd-X）、转金属化（与硼酸酯）、还原消除（Ar-Ar' 偶联产物），再生 Pd(0) 催化剂（Wikimedia Commons）*

```
                       氧化加成
                  ┌───────────────┐
                  │  M(0) → M(II) │
                  └───────┬───────┘
                          ↓
                  ┌───────────────┐
                  │   迁移插入    │
                  │  配体迁移到    │
                  │  不饱和配体上  │
      ┌─────────→└───────┬───────┘
      │                   ↓           再一轮
  还原消除         ┌───────────────┐
  M(II) → M(0)    │  β-H 消除     │ ←─── 关键区别：
  新 σ 键生成      │  16e 有利     │     Heck 有 β-H 消除
      │           └───────┬───────┘    Suzuki 没有→还原消除
      └───────────────────┘
```

| 基元反应 | 正向变化 | 逆向 | 关键特征 |
|:---|---:|:---|:---|
| **氧化加成** | M⁰ → M²⁺，配位数 +2 | 还原消除 | M 插入 X–Y 键，氧化态升高 2 |
| **迁移插入** | 配体从不饱和配体旁迁移 | β-消除/脱羰 | 构建 C–C 键的主要方式 |
| **β-H 消除** | 烷基 → 烯烃 + M–H | 迁移插入 | 需**syn** 构型，16e 有利 |
| **还原消除** | M²⁺ → M⁰，偶联产物释放 | 氧化加成 | 两个配体必须**顺式** |

> **竞赛核心**：Heck 反应包含 β-H 消除（产物有双键）；Suzuki 交叉偶联是还原消除（产物为饱和 C–C 键）。这是区分两类 Pd 催化反应的关键。

### 1.3 反馈成键（Back-bonding）

金属的 d 电子反馈到配体的 π* 轨道：

```
σ 给电子：配体孤对电子 → 金属空 dsp 轨道
π 反馈：  金属充满 d 轨道 → 配体空 π* 轨道
```

**证据**：CO 配位后 C–O 键长增加，IR 伸缩频率下降（~2000 cm⁻¹ → ~1800 cm⁻¹）。

## 二、Pd 催化交叉偶联总览

### 2.1 通用机理

```
R²–X（芳基/烯基卤）+ R¹–M（有机金属试剂）
       │
       ▼
① 氧化加成：Pd(0) + R²–X → R²–Pd(II)–X
       │
       ▼
② 转金属：R²–Pd(II)–X + R¹–M → R²–Pd(II)–R¹ + MX
       │
       ▼
③ 还原消除：R²–Pd(II)–R¹ → R²–R¹ + Pd(0) (再生)
```

### 2.2 六种 Pd 催化偶联对比

| 反应 | 有机金属试剂 | 碱？| 产物 | 特点 |
|:---|---|:---:|:---|---|
| **Heck** | 无（直接 + 烯烃）| ✅（中和 HX）| 芳基烯烃 | 有 β-H 消除，产物含双键 |
| **Suzuki** | 硼酸 R–B(OH)₂ | ✅ 必须 | 联芳基 | **最常用**，硼酸稳定低毒 |
| **Negishi** | 有机锌 R–ZnX | ❌ | 交叉偶联产物 | 反应性最高 |
| **Stille** | 有机锡 R–SnR'₃ | ❌ | 交叉偶联产物 | 最温和但**有毒** |
| **Sonogashira** | 末端炔烃 | ✅（胺碱）| 芳基炔烃 | 需 CuI 共催化 |
| **Buchwald-Hartwig** | 胺（作为 N 源）| ✅ | 芳胺 | 构建 C–N 键 |

## 三、Heck 反应

### 3.1 概述

$$
\ce{Ar-X + R-CH=CH2 ->[Pd(0)/L, base] Ar-CH=CH-R + HX}
$$

- **底物**：芳基/烯基卤（或 OTf）+ 烯烃
- **不能用于**：烷基卤（烷基-Pd 中间体 β-H 消除太快）

### 3.2 催化循环

```
① Pd(0)Lₙ + Ar–X ──→ Ar–Pd(II)–X（氧化加成）
                                │
② Ar–Pd–X + 烯烃 ──→ Ar–CH₂–CH₂–Pd–X（迁移插入，碳钯化）
                                │
③ Ar–CH₂–CH₂–Pd–X ──→ Ar–CH=CH₂ + H–Pd–X（β-H 消除，syn 消除）
                                │
④ H–Pd–X + 碱 ──→ Pd(0) + HX + 碱·HX（还原再生）
```

### 3.3 区域选择性

| 烯烃类型 | 产物 | 解释 |
|:---|---|:---|
| 缺电子烯烃（丙烯酸酯）| 新 C–C 键在**远端** | 电子效应主导 |
| 富电子烯烃（烯基醚）| 新 C–C 键在**近端**（取代基端）| 近端进攻后 Pd 更稳定 |

### 3.4 立体化学

β-H 消除为 **syn 消除** → 产物双键几何由消除时的构象决定。

> **竞赛应用**：看到 ArX + 烯烃 + Pd(0)/碱 → 立刻想到 Heck → 预测产物为取代烯烃。

## 四、Suzuki 偶联

### 4.1 概述

$$
\ce{Ar-B(OH)2 + Ar'-X ->[Pd(0)/碱] Ar-Ar' + X-B(OH)2}
$$

### 4.2 为什么 Suzuki 最常用？

| 优势 | 说明 |
|:---|---|
| **硼酸稳定** | 对空气/水稳定，可长期储存 |
| **毒性低** | 远低于有机锡（Stille） |
| **条件温和** | 室温–80°C，弱碱即可 |
| **官能团耐受好** | 酯、酮、醛、OH、NH₂ 均可共存 |
| **立体专一性** | 双键几何在偶联中保持 |

### 4.3 碱的作用

碱的关键作用：活化硼酸，形成更亲核的 **"ate" 配合物**：

$$
\ce{R-B(OH)2 + OH- -> R-B(OH)3-}
$$

这个带负电的硼酸根更容易与 R'–Pd–X 发生转金属。

> **竞赛应用**：Suzuki 的条件必有**碱**（Na₂CO₃、KOH、NaOEt 等），这是与其他偶联的重要区分！

### 4.4 常见配体

| 配体 | 特点 |
|:---|---|
| PPh₃ | 经典，通用性好 |
| SPhos/XPhos | 大位阻配体，加速氧化加成 |
| NHC（N-杂环卡宾）| 强σ给电子，高热稳定性 |

## 五、Negishi / Stille / Sonogashira 偶联

### 5.1 Negishi 偶联

$$
\ce{R-ZnX + Ar-X ->[Pd(0)] R-Ar + ZnX2}
$$

- 有机锌试剂：反应性**最高**（> Suzuki > Stille）
- 选择性好，官能团容忍性较高
- 可用于 sp³–sp² 偶联（Suzuki 较难）

### 5.2 Stille 偶联

$$
\ce{R-SnR'_3 + Ar-X ->[Pd(0)] R-Ar + X-SnR'_3}
$$

- 有机锡试剂：对空气/水极稳定
- 可在敏感官能团存在下反应
- **主要缺点**：有机锡**剧毒**，后处理困难

### 5.3 Sonogashira 偶联

$$
\ce{Ar-X + \!\!\!\!-\!C\!\equiv\!C\!-\!R ->[Pd(0)/CuI, 胺碱] Ar-C\!\equiv\!C-R + HX}
$$

- 末端炔烃与芳基/烯基卤的偶联
- **CuI 共催化**：炔烃 + CuI → 炔基亚铜 → 转金属到 Pd
- 条件温和，常室温进行

| 偶联类型 | 金属试剂 | 竞赛考频 | 毒性 |
|:---|---|:---:|:---:|
| Suzuki | 硼酸 | ⭐⭐⭐ 最高 | 低 |
| Heck | 无（烯烃直接）| ⭐⭐⭐ | — |
| Sonogashira | 末端炔烃 | ⭐⭐ | 低 |
| Negishi | 有机锌 | ⭐⭐ | 低 |
| Stille | 有机锡 | ⭐ | **高** |

## 六、Wacker 氧化

### 6.1 概述

$$
\ce{R-CH=CH2 + O2 ->[PdCl2/CuCl, H2O] R-CO-CH3}
$$

- 末端烯烃 → **甲基酮**
- 内烯烃 → 内酮
- 工业应用：乙烯 → 乙醛

### 6.2 催化循环

```
① 烯烃配位到 Pd(II) ──→ 烯烃-Pd(II) 配合物
       │
② H₂O 亲核进攻取代较多的碳（马氏取向）──→ C–Pd 键形成
       │
③ β-H 消除 ──→ 烯醇 + H–Pd(II)–X
       │
④ 烯醇互变异构 ──→ 酮 + Pd(0)（催化剂失活！）
       │
⑤ CuCl₂ + ½O₂ + 2HCl ──→ CuCl₂ + H₂O（再氧化 Cu）
   CuCl₂ + Pd(0) ──→ CuCl + Pd(II)（Pd 再生）
```

### 6.3 CuCl 共催化体系

关键：Wacker 氧化中 Pd(0) 必须被**重新氧化为 Pd(II)** 才能完成催化循环。

```
CuCl + ½O₂ + 2HCl → CuCl₂ + H₂O  （Cu 的再氧化）
CuCl₂ + Pd(0) → CuCl + PdCl₂     （Pd 的再氧化）
```

> **竞赛提示**：Wacker 氧化是马氏水合！取代较多的碳连 OH（与反马氏水合区分）。

## 七、烯烃复分解

### 7.1 Grubbs 催化剂

| 代次 | 催化剂结构 | 特点 |
|:---|:---|---|
| **第一代** | Ru=CHPh(PCy₃)₂Cl₂ | 经典，对官能团较敏感 |
| **第二代** | NHC-Ru=CHPh(PCy₃)Cl₂ | **NHC 配体** → 更稳定，活性更高 |
| **Hoveyda-Grubbs** | 含螯合醚配体 | 可回收，热稳定性最好 |

### 7.2 三种类型

| 类型 | 反应 | 应用 |
|:---|---|:---|
| **RCM**（关环复分解）| 二烯 → 环烯 + 乙烯 | 大环合成（8–30 元环）|
| **CM**（交叉复分解）| 烯烃 A + 烯烃 B → 新烯烃 | 烯烃转化 |
| **ROMP**（开环复分解聚合）| 环烯 → 线性聚合物 | 高分子合成 |

### 7.3 机理——金属杂环丁烷

```
[M]=CHR                 [M]=CHR'       [M]=CHR'
   +                      +              +
 H₂C=CHR'    交换舞伴    H₂C=CHR'       H₂C=CHR'
   │                      │
   ▼                      ▼
[M]──CHR               [M]──CHR'      [M]──CHR'
   │                      │
 H₂C──CHR'              H₂C──CHR       H₂C──CHR'
   金属杂环丁烷          (释放 RCH=CHR')
```

1. 烯烃配位到金属卡宾
2. 形成 [2+2] 环加成的**金属杂环丁烷**中间体
3. 反向断裂 → 新卡宾 + 新烯烃（"交换舞伴"）

> **竞赛应用**：看到大环（8+ 元环）→ 逆合成分析中"切出"双键 → 想到 RCM。

## 八、Tsuji-Trost 反应（π-烯丙基钯化学）

### 8.1 概述

$$
\ce{R-CH=CH-CH2-X + NuH ->[Pd(0)/L] R-CH=CH-CH2-Nu + HX}
$$

- **底物**：烯丙基化合物（X = OAc, OCO₂R, OPO(OR)₂, Cl, Br）
- **亲核试剂**：软亲核试剂（β-二羰基化合物负离子、胺、酚）

### 8.2 催化循环

```
Pd(0) + R–CH=CH–CH₂–OAc
       │
① 氧化加成（反式，从 OAc 反面进攻）
       │
       ▼
   π-烯丙基-Pd(II) 阳离子配合物（η³，16e）
       │
② 软亲核试剂从 Pd 反面进攻（第二次翻转）
       │
       ▼
   产物 + Pd(0) 再生
```

**立体化学**：两次翻转 = **净构型保持**（氧化加成 + 亲核进攻各翻转一次）。

**区域选择性**：亲核试剂进攻**空阻较小**的烯丙基末端。

> **例外**：使用格氏试剂等**硬亲核试剂**时，经转金属→还原消除 → 净**构型翻转**。

## 九、Buchwald-Hartwig 胺化

### 9.1 概述

$$
\ce{Ar-X + HNR2 ->[Pd(0)/L, 碱] Ar-NR2 + HX}
$$

- **用途**：C–N 键形成，替代传统 SNAr
- **离去基团活性**：I > Br > Cl >> F（与 SNAr **相反**）
- **无区域限制**：无需邻对位吸电子基团

### 9.2 催化循环

```
Ar–X + Pd(0) → Ar–Pd(II)–X (氧化加成)
                  │
             + HNR₂ (胺配位)
                  │
             + 碱 (如 NaOtBu, 脱 HX)
                  │
             Ar–Pd(II)–NR₂
                  │
             → Ar–NR₂ + Pd(0) (还原消除)
```

### 9.3 与 SNAr 的对比

| 特征 | Buchwald-Hartwig | SNAr |
|:---|---|:---|
| 离去基团活性 | I > Br > Cl >> F | F > Cl > Br > I |
| 区域要求 | 无限制 | 需邻/对位吸电子基 |
| 底物范围 | 各种芳卤 | 含吸电子基芳卤 |

## 十、C–H 活化（前沿）

### 10.1 核心思想

直接用 C–H 键取代预官能团化的卤代烃 → **原子经济性更高**

### 10.2 导向基策略

导向基（DG）含孤对电子原子（N、O、S）→ 与金属配位 → 拉近 C–H 键 → 区域选择性活化

**常见导向基**：酰胺、吡啶、羧酸、肟醚

### 10.3 机理类型

| 机理 | 描述 |
|:---|---|
| **CMD**（协同金属化-去质子化）| 碱协助去质子化，同时 Pd–C 键形成 |
| **氧化加成** | Pd(II) → Pd(IV)，C–H 键插入 |
| **σ 键交换** | Ru/Rh 催化，直接交换 M–C 和 C–H |

## 十一、不对称催化

### 11.1 不对称氢化（Noyori, 2001 诺贝尔奖）

- **催化剂**：手性 BINAP–Ru/Rh 配合物
- **底物**：烯烃、酮
- **BINAP 的手性来源**：**轴手性**（atropisomerism），不是中心手性
- **产物**：高对映选择性（>99% ee）

### 11.2 不对称双羟化（Sharpless, 2001 诺贝尔奖）

- **试剂**：AD-mix-α / AD-mix-β
- **底物**：烯烃
- **产物**：手性邻二醇

### 11.3 不对称环氧化（Sharpless）

- **底物**：烯丙醇
- **条件**：Ti(OiPr)₄ / DET / t-BuOOH
- **特点**：烯丙醇羟基导向 → 高度对映选择性

## 十二、竞赛应用与常见考法

### 12.1 五种考法

| 考法 | 特征 | 解题策略 |
|:---|---|:---|
| **催化循环填空** | 给循环图+已知反应类型→推中间体 | 从氧化态变化判断氧化加成/还原消除 |
| **产物预测** | 给底物+催化剂→预测产物 | 区分 Heck（含双键）vs Suzuki（饱和）|
| **机理书写** | 写出完整催化循环 | 四步曲：氧化加成→迁移插入→β-H消除/转金属→还原消除 |
| **选择性判断** | 给两个底物→判断哪个反应 | Suzuki 必有碱，Heck 必在无 β-H 的碳上 |
| **18e 计数** | 给配合物→判断稳定性和活性 | 18e 稳定，<18e 反应活性 |

### 12.2 高频陷阱

| 陷阱 | 正确认识 |
|:---|---|
| "Heck 和 Suzuki 机理一样" | Heck 有 β-H 消除，Suzuki 是**还原消除**直接释放产物 |
| "Wacker 是反马氏水合" | Wacker 是**马氏**取向（取代多的碳连 OH，然后异构化为酮）|
| "所有 Pd 催化都用 PPh₃" | SPhos、XPhos、NHC 配体更常用 |
| "Suzuki 不需要碱" | **碱是必须的**——活化硼酸为 "ate" 配合物加速转金属 |
| "烯烃复分解需要高温" | 第二代 Grubbs 催化剂在**室温**即可反应 |

## 十三、关键数据速查

### 13.1 各金属催化反应的核心条件速查

| 反应 | 催化剂 | 配体 | 碱？| 溶剂 | 温度 |

| **Buchwald-Hartwig** | Pd₂(dba)₃ | XPhos/BINAP | ✅ NaOtBu | Toluene | 80–100°C |

### 13.2 氧化态变化速查

| 反应 | 金属变化 | 配位数变化 |
|:---|---:|:---:|
| 氧化加成 | M⁰ → M²⁺ | +2 |
| 还原消除 | M²⁺ → M⁰ | −2 |
| 迁移插入 | 不变 | −1（然后 +1 被新配体补上）|
| β-H 消除 | 不变 | +1 |

### 13.3 18e 计数模板

```
[金属价电子数] + [∑ 配体贡献的电子数] = 总电子数
```

**常见错误**：
- 忘记 η³ 配体的电子计数（阳离子 = 2e，阴离子 = 4e）
- 桥连配体只计半边（桥连 CO → 每侧 1e）
- 把中性配体算成阴离子的电子数

## 十五、参考答案与提示

### 🌱 14.1 基础巩固答案

**1.** (a) Pd(0)=10, 4×PPh₃=8 → **18e ✅**。(b) Pd(II)=8, 4×MeCN=8 → **16e**（平面正方形，稳定且活泼）。(c) Cr=6, 苯=6, 3×CO=6 → **18e ✅**。(d) Ni=10, 4×CO=8 → **18e ✅**。(e) Fe=8, 5×CO=10 → **18e ✅**。

**2.** 烯烃 η² = 2e；π-烯丙基阳离子 η³ = 2e；π-烯丙基阴离子 η³ = 4e；Cp⁻ η⁵ = 6e；苯 η⁶ = 6e。

**3.** (a) 氧化加成：M⁰ → M²⁺（氧化态升高 2，配位数 +2）。(b) 迁移插入：**氧化态不变**（配体从一个位置迁移到另一个位置）。(c) β-H 消除：**氧化态不变**（配位数 +1）。(d) 还原消除：M²⁺ → M⁰（氧化态降低 2，配位数 −2，释放偶联产物）。

**4.** Heck 反应包含**β-H 消除**步骤（烯烃配体经过迁移插入后，β-H 消除使产物含双键）。Suzuki 偶联没有 β-H 消除，而是**还原消除**直接释放饱和偶联产物。因此 Heck 产物含双键（芳基烯烃），Suzuki 产物为饱和 C–C 键（联芳基）。

**5.** (a) PhBr + CH₂=CHCO₂Me → **PhCH=CHCO₂Me**（肉桂酸甲酯，E 型为主）。缺电子烯烃的 Heck 产物中新 C–C 键在远端（电子效应主导）。(b) PhBr + CH₂=CHPh → **PhCH=CHPh**（1,2-二苯乙烯，E 型为主）。

**6.** 烷基卤经氧化加成后生成**烷基-Pd 中间体**，烷基-Pd 的 β-H 消除速率极快 → 在迁移插入发生之前就已分解 → 无法得到偶联产物。此外，烷基-Pd 的氧化加成本身也较慢（sp³ C-X 键的氧化加成不如 sp² C-X 有利）。

**7.** 碱的关键作用：活化硼酸，形成更亲核的 **"ate" 配合物**：$\ce{R-B(OH)2 + OH- -> R-B(OH)3-}$。带负电的硼酸根更容易与 R'–Pd–X 发生**转金属化**。若无碱，硼酸的亲核性不足，转金属步骤极慢。

**8.** Suzuki：有机硼酸 R–B(OH)₂，反应性中，毒性**低**。Negishi：有机锌 R–ZnX，反应性**最高**，毒性低。Stille：有机锡 R–SnR'₃，反应性低，毒性**高**（剧毒）。

### 🔥 14.2 竞赛入门答案

**9.** (a) $\ce{CH2=CH2 + 1/2 O2 ->[PdCl2/CuCl] CH3CHO}$。(b) CuCl 的作用：**将 Pd(0) 重新氧化为 Pd(II)**，完成催化循环。两步：$\ce{CuCl2 + Pd(0) -> CuCl + PdCl2}$（Pd 再氧化）；$\ce{2CuCl + 2HCl + 1/2 O2 -> 2CuCl2 + H2O}$（Cu 再氧化）。(c) 马氏取向——水亲核进攻取代较多的碳。乙烯 → 乙醛的机理中，H₂O 进攻 CH₂ 端 → 形成烯醇 → 互变异构为甲基酮（乙醛）。注：对于末端烯烃 RCH=CH₂，Wacker 氧化遵循马氏规则得到甲基酮 RCOCH₃。

**10.** 第一代 Grubbs：Ru=CHPh(PCy₃)₂Cl₂（两个 PCy₃ 配体）。第二代 Grubbs：NHC-Ru=CHPh(PCy₃)Cl₂（一个 PCy₃ 被 N-杂环卡宾 NHC 取代）。NHC 是**强 σ 给电子配体**，通过增强金属卡宾的电子密度 → 提高 Ru=C 键的反应活性 → 更容易与烯烃配位形成金属杂环丁烷。同时 NHC 的大位阻提供动力学稳定性 → 催化剂寿命更长。

**11.** (a) 12 元环含一个双键 → 逆合成分析：在双键处"切开"→ 二烯前体的两端各有一个烯烃 → RCM 关环。前体结构为 $\ce{CH2=CH-(CH2)8-CH=CH2}$（1,11-十二碳二烯），RCM 产物为 12 元环烯 + 乙烯。(b) 环己烯本身已是环 → RCM 无法直接在单环上进行。若将环己烯理解为含双键的环，RCM 不适用于此底物。若题目意为"从环己烯的 RCM 合成出发"，则环己烯可由 1,7-辛二烯经 RCM 合成：$\ce{CH2=CH-(CH2)4-CH=CH2 ->[Grubbs] 环己烯 + CH2=CH2}$。

**12.** (a) **反式**氧化加成——Pd(0) 从 OAc 的反面进攻烯丙基碳 → 翻转一次。(b) 软亲核试剂从 **Pd 的反面**进攻（第二次翻转）→ 亲核试剂进入与 Pd 相反的烯丙基末端。(c) 两次翻转 = **净构型保持**（翻转 + 翻转 = 保持）。但注意：使用硬亲核试剂（如格氏试剂）时经转金属→还原消除路径，结果为净构型翻转。

**13.** Buchwald-Hartwig 离去基团活性：**I > Br > Cl >> F**（与 SNAr 相反）。区域限制：**无限制**（无需邻/对位吸电子基）。底物范围：**各种芳卤**（包括无吸电子基的芳卤）。SNAr 的离去基团活性是 F > Cl > Br > I（因为速率决定步骤是加成，F 的强吸电子效应加速加成），且需要邻/对位吸电子基活化芳环。

### 🏆 14.3 真题挑战答案

**14.** (a) 排列顺序：**B → A → D → C → B**（再生）。完整循环：Pd(0)(PPh₃)₂ (B) → 氧化加成 → Ar–Pd(II)–Br(PPh₃)₂ (A) → 转金属化（碱活化硼酸）→ Ar–Pd(II)–B(OH)₃⁻(PPh₃)₂ (D) → 还原消除 → Ar–Pd(II)–Ar'(PPh₃)₂ (C) → 释放偶联产物 Ar–Ar' → Pd(0)(PPh₃)₂ (B)。注：中间体 D 实际上是转金属化的过渡态/中间体，具体结构取决于碱的作用方式。(b) B → A：**氧化加成**。Pd(0) → Pd(II)，氧化态升高 2。(c) A → D：**转金属化**（transmetalation）。碱（如 Na₂CO₃）活化硼酸为 $\ce{R-B(OH)3-}$（"ate" 配合物），使其更容易将有机基团转移到 Pd 上。

**15.** (a) **Heck 反应**——条件中有烯烃（丙烯酸甲酯）、芳基溴、Pd(0)/配体、碱（Et₃N），且无有机金属试剂。(b) 四个关键中间体：① Pd(0)(PPh₃)₂——催化剂前体。② Ar–Pd(II)–Br(PPh₃)₂——氧化加成产物（Ar = 4-MeO-C₆H₄）。③ Ar–CH₂CH₂–Pd(II)–Br(PPh₃)₂——迁移插入（碳钯化）产物。④ H–Pd(II)–Br(PPh₃)₂——β-H 消除后的 Pd-H 物种，经碱中和再生 Pd(0)。(c) 产物 X 为 **(E)-3-(4-甲氧基苯基)丙烯酸甲酯**：$\ce{4-MeO-C6H4-CH=CH-CO2Me}$。E 型构型的来源：β-H 消除为 **syn 消除**——Pd 和 β-H 必须处于顺式（syn）位置 → 消除后双键几何由过渡态构象决定 → 位阻最小化使芳基与酯基处于反式 → E 型为主。

**16.** (a) $\Omega = 14 + 1 - (12-0)/2 = 15 - 6 = 9$。(b) IR 1665 cm⁻¹ → **共轭酮 C=O**（与芳环共轭，波数从正常 1715 cm⁻¹ 降低 ~50 cm⁻¹，因为共轭使 C=O 键级降低）。1600 和 1500 cm⁻¹ → 芳环 C=C 双特征峰，确认苯环存在。(c) δ 2.6 (s, 3H) → **甲基酮** $\ce{-COCH3}$（化学位移 2.0–2.5，邻羰基甲基；单峰说明相邻碳上无 H）。(d) 综合推断：Ω=9（苯环 4 + C=O 1 + 另一个苯环 4 = 9），IR 有共轭酮和芳环，NMR 有 9 个芳氢 + 3 个甲基氢 = 12H 吻合分子式。结构为 **4-甲基二苯甲酮**（$\ce{CH3-C6H4-CO-C6H5}$）：一个苯环被甲基取代（4H），另一个未取代苯环（5H），芳氢共 9H；甲基酮 δ 2.6 (s, 3H)。验证：IR 1665 cm⁻¹ 为共轭酮（与两个苯环共轭），符合。该化合物可由 4-溴甲苯与苯硼酸经 **Suzuki 偶联**得到 4-甲基联苯，再经 Friedel-Crafts 酰化引入乙酰基；或由 4-碘苯乙酮与苯硼酸经 Suzuki 偶联直接得到。
