# 过渡金属催化

> **适用**：第三轮（提高班）/ 第四轮（冲刺班）| **前置要求**：[[04-课件/学生讲义/2026-06-23-配位化合物基础]]（晶体场、18电子规则）、[[04-课件/学生讲义/2026-06-23-有机化学基础]]（电子效应）
> **对应考纲**：考纲51-金属有机 / 决赛金属催化循环 / 初赛交叉偶联
> **源素材**：ABOC Ch8（过渡金属催化）+ Clayden Ch40（金属有机化学）+ 初赛讲义第5讲
>
> 过渡金属催化是现代有机合成化学的核心工具。本讲以 **Pd 催化**为主线，覆盖偶联反应、氧化、复分解、C–H 活化四大板块，重点是**催化循环的拆解与理解**。

---

## 学习目标

- ✅ 能拆解催化循环为基元反应（氧化加成/迁移插入/β-H 消除/还原消除）
- ✅ 能用 18e 规则/16e 规则判断配合物的稳定性与反应活性
- ✅ 能区分 Heck、Suzuki、Negishi、Stille、Sonogashira 的底物与机理
- ✅ 能画出 Grubbs 催化烯烃复分解的金属杂环丁烷机理
- ✅ 能理解 Wacker 氧化和 Buchwald-Hartwig 胺化的催化循环
- ✅ 能预测 Tsuji-Trost 反应的区域选择性和立体化学

## 一、金属有机化学基础

### 1.1 18 电子规则

稳定的过渡金属配合物，其中心金属的价壳层具有 **18 电子**（填满 s + 5d + 3p 轨道）。

| 项目 | 内容 |
|:---|---|
| **电子来源** | 金属自身的价电子 + 所有配体贡献的电子 |
| **计数方法** | 新族号（1–18）直接给出金属价电子数，Pd = 10, Cr = 6 |
| **16e 例外** | Ni、Pd、Pt 族可形成稳定的**平面正方形 16e 配合物**（D₄h，HOMO-LUMO 间隙大）|
| **反应活性** | <18e 为络合不饱和，具有反应活性（催化循环所需）|

**常见配体的电子贡献**：

| 配体类型 | η | 形式电荷 | 贡献电子数 |
|:---|---:|:---:|:---:|
| 卤素、烷基、芳基（σ 键）| η¹ | –1 | 2 |
| 烯烃 | η² | 0 | 2 |
| CO、PPh₃、胺 | η¹ | 0 | 2 |
| π-烯丙基阳离子 | η³ | +1 | 2 |
| π-烯丙基阴离子 | η³ | –1 | 4 |
| 共轭双烯 | η⁴ | 0 | 4 |
| 环戊二烯基（Cp）| η⁵ | –1 | 6 |
| 苯 | η⁶ | 0 | 6 |

**实例**：
- Pd(PPh₃)₄：10 + 4×2 = **18e ✅**
- PdCl₂(MeCN)₂（Pd²⁺：8e）：8 + 4×2 = **16e （稳定且活泼）**
- Cr(苯)(CO)₃：6 + 6 + 3×2 = **18e ✅**

### 1.2 催化循环四步曲

![[media/suzuki-cycle.png|Suzuki 交叉偶联的催化循环（Pd 催化）]]
*图 Suzuki-Miyaura 交叉偶联反应的催化循环。从 Pd(0) 开始，经历氧化加成（Ar-X → Ar-Pd-X）、转金属化（与硼酸酯）、还原消除（Ar-Ar' 偶联产物），再生 Pd(0) 催化剂（Wikimedia Commons）*

```
                       氧化加成
                  ┌───────────────┐
                  │  M(0) → M(II) │
                  └───────┬───────┘
                          ↓
                  ┌───────────────┐
                  │   迁移插入    │
                  │  配体迁移到    │
                  │  不饱和配体上  │
      ┌─────────→└───────┬───────┘
      │                   ↓           再一轮
  还原消除         ┌───────────────┐
  M(II) → M(0)    │  β-H 消除     │ ←─── 关键区别：
  新 σ 键生成      │  16e 有利     │     Heck 有 β-H 消除
      │           └───────┬───────┘    Suzuki 没有→还原消除
      └───────────────────┘
```

| 基元反应 | 正向变化 | 逆向 | 关键特征 |
|:---|---:|:---|:---|
| **氧化加成** | M⁰ → M²⁺，配位数 +2 | 还原消除 | M 插入 X–Y 键，氧化态升高 2 |
| **迁移插入** | 配体从不饱和配体旁迁移 | β-消除/脱羰 | 构建 C–C 键的主要方式 |
| **β-H 消除** | 烷基 → 烯烃 + M–H | 迁移插入 | 需**syn** 构型，16e 有利 |
| **还原消除** | M²⁺ → M⁰，偶联产物释放 | 氧化加成 | 两个配体必须**顺式** |

> **竞赛核心**：Heck 反应包含 β-H 消除（产物有双键）；Suzuki 交叉偶联是还原消除（产物为饱和 C–C 键）。这是区分两类 Pd 催化反应的关键。

### 1.3 反馈成键（Back-bonding）

金属的 d 电子反馈到配体的 π* 轨道：

```
σ 给电子：配体孤对电子 → 金属空 dsp 轨道
π 反馈：  金属充满 d 轨道 → 配体空 π* 轨道
```

**证据**：CO 配位后 C–O 键长增加，IR 伸缩频率下降（~2000 cm⁻¹ → ~1800 cm⁻¹）。

## 二、Pd 催化交叉偶联总览

### 2.1 通用机理

```
R²–X（芳基/烯基卤）+ R¹–M（有机金属试剂）
       │
       ▼
① 氧化加成：Pd(0) + R²–X → R²–Pd(II)–X
       │
       ▼
② 转金属：R²–Pd(II)–X + R¹–M → R²–Pd(II)–R¹ + MX
       │
       ▼
③ 还原消除：R²–Pd(II)–R¹ → R²–R¹ + Pd(0) (再生)
```

### 2.2 六种 Pd 催化偶联对比

| 反应 | 有机金属试剂 | 碱？| 产物 | 特点 |
|:---|---|:---:|:---|---|
| **Heck** | 无（直接 + 烯烃）| ✅（中和 HX）| 芳基烯烃 | 有 β-H 消除，产物含双键 |
| **Suzuki** | 硼酸 R–B(OH)₂ | ✅ 必须 | 联芳基 | **最常用**，硼酸稳定低毒 |
| **Negishi** | 有机锌 R–ZnX | ❌ | 交叉偶联产物 | 反应性最高 |
| **Stille** | 有机锡 R–SnR'₃ | ❌ | 交叉偶联产物 | 最温和但**有毒** |
| **Sonogashira** | 末端炔烃 | ✅（胺碱）| 芳基炔烃 | 需 CuI 共催化 |
| **Buchwald-Hartwig** | 胺（作为 N 源）| ✅ | 芳胺 | 构建 C–N 键 |

## 三、Heck 反应

### 3.1 概述

$$
\ce{Ar-X + R-CH=CH2 ->[Pd(0)/L, base] Ar-CH=CH-R + HX}
$$

- **底物**：芳基/烯基卤（或 OTf）+ 烯烃
- **不能用于**：烷基卤（烷基-Pd 中间体 β-H 消除太快）

### 3.2 催化循环

```
① Pd(0)Lₙ + Ar–X ──→ Ar–Pd(II)–X（氧化加成）
                                │
② Ar–Pd–X + 烯烃 ──→ Ar–CH₂–CH₂–Pd–X（迁移插入，碳钯化）
                                │
③ Ar–CH₂–CH₂–Pd–X ──→ Ar–CH=CH₂ + H–Pd–X（β-H 消除，syn 消除）
                                │
④ H–Pd–X + 碱 ──→ Pd(0) + HX + 碱·HX（还原再生）
```

### 3.3 区域选择性

| 烯烃类型 | 产物 | 解释 |
|:---|---|:---|
| 缺电子烯烃（丙烯酸酯）| 新 C–C 键在**远端** | 电子效应主导 |
| 富电子烯烃（烯基醚）| 新 C–C 键在**近端**（取代基端）| 近端进攻后 Pd 更稳定 |

### 3.4 立体化学

β-H 消除为 **syn 消除** → 产物双键几何由消除时的构象决定。

> **竞赛应用**：看到 ArX + 烯烃 + Pd(0)/碱 → 立刻想到 Heck → 预测产物为取代烯烃。

## 四、Suzuki 偶联

### 4.1 概述

$$
\ce{Ar-B(OH)2 + Ar'-X ->[Pd(0)/碱] Ar-Ar' + X-B(OH)2}
$$

### 4.2 为什么 Suzuki 最常用？

| 优势 | 说明 |
|:---|---|
| **硼酸稳定** | 对空气/水稳定，可长期储存 |
| **毒性低** | 远低于有机锡（Stille） |
| **条件温和** | 室温–80°C，弱碱即可 |
| **官能团耐受好** | 酯、酮、醛、OH、NH₂ 均可共存 |
| **立体专一性** | 双键几何在偶联中保持 |

### 4.3 碱的作用

碱的关键作用：活化硼酸，形成更亲核的 **"ate" 配合物**：

$$
\ce{R-B(OH)2 + OH- -> R-B(OH)3-}
$$

这个带负电的硼酸根更容易与 R'–Pd–X 发生转金属。

> **竞赛应用**：Suzuki 的条件必有**碱**（Na₂CO₃、KOH、NaOEt 等），这是与其他偶联的重要区分！

### 4.4 常见配体

| 配体 | 特点 |
|:---|---|
| PPh₃ | 经典，通用性好 |
| SPhos/XPhos | 大位阻配体，加速氧化加成 |
| NHC（N-杂环卡宾）| 强σ给电子，高热稳定性 |

## 五、Negishi / Stille / Sonogashira 偶联

### 5.1 Negishi 偶联

$$
\ce{R-ZnX + Ar-X ->[Pd(0)] R-Ar + ZnX2}
$$

- 有机锌试剂：反应性**最高**（> Suzuki > Stille）
- 选择性好，官能团容忍性较高
- 可用于 sp³–sp² 偶联（Suzuki 较难）

### 5.2 Stille 偶联

$$
\ce{R-SnR'_3 + Ar-X ->[Pd(0)] R-Ar + X-SnR'_3}
$$

- 有机锡试剂：对空气/水极稳定
- 可在敏感官能团存在下反应
- **主要缺点**：有机锡**剧毒**，后处理困难

### 5.3 Sonogashira 偶联

$$
\ce{Ar-X + \!\!\!\!-\!C\!\equiv\!C\!-\!R ->[Pd(0)/CuI, 胺碱] Ar-C\!\equiv\!C-R + HX}
$$

- 末端炔烃与芳基/烯基卤的偶联
- **CuI 共催化**：炔烃 + CuI → 炔基亚铜 → 转金属到 Pd
- 条件温和，常室温进行

| 偶联类型 | 金属试剂 | 竞赛考频 | 毒性 |
|:---|---|:---:|:---:|
| Suzuki | 硼酸 | ⭐⭐⭐ 最高 | 低 |
| Heck | 无（烯烃直接）| ⭐⭐⭐ | — |
| Sonogashira | 末端炔烃 | ⭐⭐ | 低 |
| Negishi | 有机锌 | ⭐⭐ | 低 |
| Stille | 有机锡 | ⭐ | **高** |

## 六、Wacker 氧化

### 6.1 概述

$$
\ce{R-CH=CH2 + O2 ->[PdCl2/CuCl, H2O] R-CO-CH3}
$$

- 末端烯烃 → **甲基酮**
- 内烯烃 → 内酮
- 工业应用：乙烯 → 乙醛

### 6.2 催化循环

```
① 烯烃配位到 Pd(II) ──→ 烯烃-Pd(II) 配合物
       │
② H₂O 亲核进攻取代较多的碳（马氏取向）──→ C–Pd 键形成
       │
③ β-H 消除 ──→ 烯醇 + H–Pd(II)–X
       │
④ 烯醇互变异构 ──→ 酮 + Pd(0)（催化剂失活！）
       │
⑤ CuCl₂ + ½O₂ + 2HCl ──→ CuCl₂ + H₂O（再氧化 Cu）
   CuCl₂ + Pd(0) ──→ CuCl + Pd(II)（Pd 再生）
```

### 6.3 CuCl 共催化体系

关键：Wacker 氧化中 Pd(0) 必须被**重新氧化为 Pd(II)** 才能完成催化循环。

```
CuCl + ½O₂ + 2HCl → CuCl₂ + H₂O  （Cu 的再氧化）
CuCl₂ + Pd(0) → CuCl + PdCl₂     （Pd 的再氧化）
```

> **竞赛提示**：Wacker 氧化是马氏水合！取代较多的碳连 OH（与反马氏水合区分）。

## 七、烯烃复分解

### 7.1 Grubbs 催化剂

| 代次 | 催化剂结构 | 特点 |
|:---|:---|---|
| **第一代** | Ru=CHPh(PCy₃)₂Cl₂ | 经典，对官能团较敏感 |
| **第二代** | NHC-Ru=CHPh(PCy₃)Cl₂ | **NHC 配体** → 更稳定，活性更高 |
| **Hoveyda-Grubbs** | 含螯合醚配体 | 可回收，热稳定性最好 |

### 7.2 三种类型

| 类型 | 反应 | 应用 |
|:---|---|:---|
| **RCM**（关环复分解）| 二烯 → 环烯 + 乙烯 | 大环合成（8–30 元环）|
| **CM**（交叉复分解）| 烯烃 A + 烯烃 B → 新烯烃 | 烯烃转化 |
| **ROMP**（开环复分解聚合）| 环烯 → 线性聚合物 | 高分子合成 |

### 7.3 机理——金属杂环丁烷

```
[M]=CHR                 [M]=CHR'       [M]=CHR'
   +                      +              +
 H₂C=CHR'    交换舞伴    H₂C=CHR'       H₂C=CHR'
   │                      │
   ▼                      ▼
[M]──CHR               [M]──CHR'      [M]──CHR'
   │                      │
 H₂C──CHR'              H₂C──CHR       H₂C──CHR'
   金属杂环丁烷          (释放 RCH=CHR')
```

1. 烯烃配位到金属卡宾
2. 形成 [2+2] 环加成的**金属杂环丁烷**中间体
3. 反向断裂 → 新卡宾 + 新烯烃（"交换舞伴"）

> **竞赛应用**：看到大环（8+ 元环）→ 逆合成分析中"切出"双键 → 想到 RCM。

## 八、Tsuji-Trost 反应（π-烯丙基钯化学）

### 8.1 概述

$$
\ce{R-CH=CH-CH2-X + NuH ->[Pd(0)/L] R-CH=CH-CH2-Nu + HX}
$$

- **底物**：烯丙基化合物（X = OAc, OCO₂R, OPO(OR)₂, Cl, Br）
- **亲核试剂**：软亲核试剂（β-二羰基化合物负离子、胺、酚）

### 8.2 催化循环

```
Pd(0) + R–CH=CH–CH₂–OAc
       │
① 氧化加成（反式，从 OAc 反面进攻）
       │
       ▼
   π-烯丙基-Pd(II) 阳离子配合物（η³，16e）
       │
② 软亲核试剂从 Pd 反面进攻（第二次翻转）
       │
       ▼
   产物 + Pd(0) 再生
```

**立体化学**：两次翻转 = **净构型保持**（氧化加成 + 亲核进攻各翻转一次）。

**区域选择性**：亲核试剂进攻**空阻较小**的烯丙基末端。

> **例外**：使用格氏试剂等**硬亲核试剂**时，经转金属→还原消除 → 净**构型翻转**。

## 九、Buchwald-Hartwig 胺化

### 9.1 概述

$$
\ce{Ar-X + HNR2 ->[Pd(0)/L, 碱] Ar-NR2 + HX}
$$

- **用途**：C–N 键形成，替代传统 SNAr
- **离去基团活性**：I > Br > Cl >> F（与 SNAr **相反**）
- **无区域限制**：无需邻对位吸电子基团

### 9.2 催化循环

```
Ar–X + Pd(0) → Ar–Pd(II)–X (氧化加成)
                  │
             + HNR₂ (胺配位)
                  │
             + 碱 (如 NaOtBu, 脱 HX)
                  │
             Ar–Pd(II)–NR₂
                  │
             → Ar–NR₂ + Pd(0) (还原消除)
```

### 9.3 与 SNAr 的对比

| 特征 | Buchwald-Hartwig | SNAr |
|:---|---|:---|
| 离去基团活性 | I > Br > Cl >> F | F > Cl > Br > I |
| 区域要求 | 无限制 | 需邻/对位吸电子基 |
| 底物范围 | 各种芳卤 | 含吸电子基芳卤 |

## 十、C–H 活化（前沿）

### 10.1 核心思想

直接用 C–H 键取代预官能团化的卤代烃 → **原子经济性更高**

### 10.2 导向基策略

导向基（DG）含孤对电子原子（N、O、S）→ 与金属配位 → 拉近 C–H 键 → 区域选择性活化

**常见导向基**：酰胺、吡啶、羧酸、肟醚

### 10.3 机理类型

| 机理 | 描述 |
|:---|---|
| **CMD**（协同金属化-去质子化）| 碱协助去质子化，同时 Pd–C 键形成 |
| **氧化加成** | Pd(II) → Pd(IV)，C–H 键插入 |
| **σ 键交换** | Ru/Rh 催化，直接交换 M–C 和 C–H |

## 十一、不对称催化

### 11.1 不对称氢化（Noyori, 2001 诺贝尔奖）

- **催化剂**：手性 BINAP–Ru/Rh 配合物
- **底物**：烯烃、酮
- **BINAP 的手性来源**：**轴手性**（atropisomerism），不是中心手性
- **产物**：高对映选择性（>99% ee）

### 11.2 不对称双羟化（Sharpless, 2001 诺贝尔奖）

- **试剂**：AD-mix-α / AD-mix-β
- **底物**：烯烃
- **产物**：手性邻二醇

### 11.3 不对称环氧化（Sharpless）

- **底物**：烯丙醇
- **条件**：Ti(OiPr)₄ / DET / t-BuOOH
- **特点**：烯丙醇羟基导向 → 高度对映选择性

## 十二、竞赛应用与常见考法

### 12.1 五种考法

| 考法 | 特征 | 解题策略 |
|:---|---|:---|
| **催化循环填空** | 给循环图+已知反应类型→推中间体 | 从氧化态变化判断氧化加成/还原消除 |
| **产物预测** | 给底物+催化剂→预测产物 | 区分 Heck（含双键）vs Suzuki（饱和）|
| **机理书写** | 写出完整催化循环 | 四步曲：氧化加成→迁移插入→β-H消除/转金属→还原消除 |
| **选择性判断** | 给两个底物→判断哪个反应 | Suzuki 必有碱，Heck 必在无 β-H 的碳上 |
| **18e 计数** | 给配合物→判断稳定性和活性 | 18e 稳定，<18e 反应活性 |

### 12.2 高频陷阱

| 陷阱 | 正确认识 |
|:---|---|
| "Heck 和 Suzuki 机理一样" | Heck 有 β-H 消除，Suzuki 是**还原消除**直接释放产物 |
| "Wacker 是反马氏水合" | Wacker 是**马氏**取向（取代多的碳连 OH，然后异构化为酮）|
| "所有 Pd 催化都用 PPh₃" | SPhos、XPhos、NHC 配体更常用 |
| "Suzuki 不需要碱" | **碱是必须的**——活化硼酸为 "ate" 配合物加速转金属 |
| "烯烃复分解需要高温" | 第二代 Grubbs 催化剂在**室温**即可反应 |

## 十三、关键数据速查

### 13.1 各金属催化反应的核心条件速查

| 反应 | 催化剂 | 配体 | 碱？| 溶剂 | 温度 |
|:---|---|:---:|:---:|:---|---:|
| **Heck** | Pd(OAc)₂/Pd₂(dba)₃ | PPh₃ | ✅ Et₃N | DMF, THF | 80–120°C |
| **Suzuki** | Pd(PPh₃)₄ | PPh₃/SPhos | ✅ Na₂CO₃ | THF/H₂O | RT–80°C |
| **Negishi** | Pd(PPh₃)₄ | — | ❌ | THF | 0°C–RT |
| **Stille** | Pd(PPh₃)₄ | AsPh₃（加速）| ❌ | Toluene | 60–100°C |
| **Sonogashira** | Pd(PPh₃)₂Cl₂/CuI | PPh₃ | ✅ 胺 | THF, Et₃N | RT |
| **Wacker** | PdCl₂/CuCl | — | ❌ | H₂O/THF | RT–50°C |
| **RCM** | Grubbs II | NHC | ❌ | CH₂Cl₂ | RT |
| **Buchwald-Hartwig** | Pd₂(dba)₃ | XPhos/BINAP | ✅ NaOtBu | Toluene | 80–100°C |

### 13.2 氧化态变化速查

| 反应 | 金属变化 | 配位数变化 |
|:---|---:|:---:|
| 氧化加成 | M⁰ → M²⁺ | +2 |
| 还原消除 | M²⁺ → M⁰ | −2 |
| 迁移插入 | 不变 | −1（然后 +1 被新配体补上）|
| β-H 消除 | 不变 | +1 |

### 13.3 18e 计数模板

```
[金属价电子数] + [∑ 配体贡献的电子数] = 总电子数
```

**常见错误**：
- 忘记 η³ 配体的电子计数（阳离子 = 2e，阴离子 = 4e）
- 桥连配体只计半边（桥连 CO → 每侧 1e）
- 把中性配体算成阴离子的电子数
