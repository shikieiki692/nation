# 第四讲：化学平衡

> **适用**：第一轮（初学）→ 第二轮（深化）
> **对应备课大纲**：[[2026-06-02-化学平衡-基础班]]
> **前置要求**：热力学基础（Gibbs自由能）、理想气体状态方程、化学计量

---

## 学习目标

- ✅ 能描述化学平衡的条件（正逆反应速率相等、$\Delta_rG = 0$），理解化学平衡是动态平衡
- ✅ 能正确书写各类反应的平衡常数表达式（$K_c$、$K_p$、$K^\theta$），掌握 $K_p = K_c(RT)^{\Delta\nu}$ 的推导关系
- ✅ 能用化学等温方程式 $\Delta_rG = \Delta_rG^\theta + RT\ln Q$ 判断反应方向（$Q$ vs $K$ 法则）
- ✅ 能运用勒夏特列原理和 $Q$-$K$ 定量分析浓度、压力、温度变化对平衡的影响，区分"改变 $Q$"与"改变 $K$"
- ✅ 能通过 ICE 表计算平衡转化率和平衡组成
- ✅ 能用 van't Hoff 方程定量计算温度对平衡常数的影响（第二轮深化）

## 一、化学平衡的基本概念

### 1.1 可逆反应与化学平衡

**可逆反应**：在一定条件下，反应既能按反应方程式从左向右进行（正反应），也能从右向左进行（逆反应）。

绝大多数化学反应都具有一定的可逆性，但可逆程度不同：
- **可逆倾向弱**：如 $\mathrm{KClO_3}$ 受热分解，反应物耗尽前逆反应条件尚未具备
- **可逆倾向强**：如 $\mathrm{N_2O_4(g) \rightleftharpoons 2NO_2(g)}$，正逆反应在相同条件下均可进行

**化学平衡的建立**：可逆反应进行到一定程度时，正反应速率等于逆反应速率，各组分浓度不再随时间改变。这一状态称为化学平衡。

### 1.2 化学平衡的条件

| 角度 | 平衡条件 |
|:---|:---|
| **动力学** | 正反应速率 $r_+$ = 逆反应速率 $r_-$ |
| **热力学** | 等温等压、非体积功为零时：$\Delta_rG = 0$ |
| **宏观表现** | 反应物和生成物的浓度（或分压）不再随时间改变 |

> 💡 **理解要点**：化学平衡是**动态平衡**——从宏观上看反应似乎停止了，但微观上正反应和逆反应仍在继续进行，两者的反应速率正好相等。用同位素示踪法可以验证：将 $^{18}\mathrm{O}$ 标记的 $\mathrm{NO_2}$ 引入 $\mathrm{N_2O_4\rightleftharpoons2NO_2}$ 的平衡体系，最终会在 $\mathrm{N_2O_4}$ 中发现 $^{18}\mathrm{O}$。

### 1.3 平衡的宏观标志

1. 反应物和生成物的浓度（或分压）不再改变
2. 平衡常数 $K$ 有确定值（仅与温度有关）
3. 平衡是有条件的——外界条件改变时，平衡可能被破坏并发生移动

## 二、平衡常数

### 2.1 实验平衡常数

对于可逆反应通式：

$$a\mathrm{A} + b\mathrm{B} \rightleftharpoons d\mathrm{D} + e\mathrm{E}$$

在一定温度下达平衡时：

$$K_c = \frac{[\mathrm{D}]^d[\mathrm{E}]^e}{[\mathrm{A}]^a[\mathrm{B}]^b} \qquad K_p = \frac{(p_\mathrm{D})^d(p_\mathrm{E})^e}{(p_\mathrm{A})^a(p_\mathrm{B})^b}$$

- $K_c$：浓度平衡常数（用平衡浓度表示）
- $K_p$：压力平衡常数（用平衡分压表示，仅适用于气相反应）
- $K_c$ 与 $K_p$ 的单位：分别为 $(\mathrm{mol\cdot L^{-1}})^{\Delta\nu}$ 和 $(\mathrm{kPa})^{\Delta\nu}$

> **实验平衡常数的物理意义**：平衡常数是可逆反应的特征常数，是一定条件下反应进行程度的标度。对同类反应而言，$K$ 值越大，反应正向进行的程度越大。

#### $K_c$ 与 $K_p$ 的关系

若将各气体视为理想气体，由 $p = cRT = \frac{n}{V}RT$ 可得：

$$K_p = K_c(RT)^{\Delta\nu}$$

其中 $\Delta\nu = (d+e) - (a+b)$，即气态生成物总计量系数与气态反应物总计量系数之差。

> ⚠️ **注意**：$R$ 值必须与 $p$、$c$ 的单位匹配使用。当 $p$ 以 kPa 为单位、$c$ 以 $\mathrm{mol\cdot L^{-1}}$ 为单位时，$R = 8.314\ \mathrm{kPa\cdot L\cdot mol^{-1}\cdot K^{-1}}$。

### 2.2 书写平衡常数表达式的注意事项

**① 纯固体与纯液体不写入表达式**

$$ \mathrm{CaCO_3(s) \rightleftharpoons CaO(s) + CO_2(g)} \quad \Rightarrow \quad K = p_{\mathrm{CO_2}} $$

$\mathrm{CaCO_3}$ 和 $\mathrm{CaO}$ 均为纯固体，其浓度视为常数，已并入 $K$。

**② 稀水溶液中的水不写入表达式**

$$\mathrm{Cr_2O_7^{2-}(aq) + H_2O(l) \rightleftharpoons 2CrO_4^{2-}(aq) + 2H^+(aq)} \quad \Rightarrow \quad K = \frac{[\mathrm{CrO_4^{2-}}]^2[\mathrm{H^+}]^2}{[\mathrm{Cr_2O_7^{2-}}]}$$

> ⚠️ **非水溶剂中的例外**：若溶剂不是水（如乙醇中酯化反应），水的浓度不可视为常数，需写入表达式。

**③ 同一反应的不同写法对应不同 $K$ 值**

以 $\mathrm{N_2O_4(g) \rightleftharpoons 2NO_2(g)}$ 为例：

| 写法 | 平衡常数表达式 | 数值（373K）|
|:---|:---|:---:|
| $\mathrm{N_2O_4 \rightleftharpoons 2NO_2}$ | $K = [\mathrm{NO_2}]^2/[\mathrm{N_2O_4}]$ | 0.36 |
| $\mathrm{\frac{1}{2}N_2O_4 \rightleftharpoons NO_2}$ | $K' = [\mathrm{NO_2}]/[\mathrm{N_2O_4}]^{1/2}$ | 0.60 |
| $\mathrm{2NO_2 \rightleftharpoons N_2O_4}$ | $K'' = [\mathrm{N_2O_4}]/[\mathrm{NO_2}]^2$ | 2.8 |

因此，**书写平衡常数表达式及数值，必须与化学反应方程式相对应**。

### 2.3 标准平衡常数

将平衡浓度（或分压）除以标准浓度（或标准压力），得到无量纲的相对量：

$$K^\theta = \frac{(c_\mathrm{D}/c^\theta)^d(c_\mathrm{E}/c^\theta)^e}{(c_\mathrm{A}/c^\theta)^a(c_\mathrm{B}/c^\theta)^b} \quad \text{或} \quad K^\theta = \frac{(p_\mathrm{D}/p^\theta)^d(p_\mathrm{E}/p^\theta)^e}{(p_\mathrm{A}/p^\theta)^a(p_\mathrm{B}/p^\theta)^b}$$

其中 $c^\theta = 1\ \mathrm{mol\cdot L^{-1}}$，$p^\theta = 100\ \mathrm{kPa}$（国际标准）。

**标准平衡常数的特点**：
- 无量纲（纯数），便于与热力学数据关联
- $K^\theta$ 与 $\Delta_rG_m^\theta$ 通过公式 $\Delta_rG_m^\theta = -RT\ln K^\theta$ 直接关联
- 对于液相反应，$K^\theta$ 与 $K_c$ **数值相等**（因 $c^\theta=1$）
- 对于气相反应，$K^\theta$ 与 $K_p$ 一般不相等：$K^\theta = K_p \cdot (p^\theta)^{-\Delta\nu}$

### 2.4 多重平衡规则

在相同条件下，若有多个反应方程相加（或相减）得到总反应，则总反应的平衡常数等于各反应平衡常数之积（或商）。

$$\text{反应(3) = 反应(1) + 反应(2)} \quad \Rightarrow \quad K_3^\theta = K_1^\theta \times K_2^\theta$$

**示例**：

已知：
$$\begin{aligned}
\mathrm{SO_2(g) + \frac{1}{2}O_2(g) \rightleftharpoons SO_3(g)} &\quad K_1^\theta \\
\mathrm{NO_2(g) \rightleftharpoons NO(g) + \frac{1}{2}O_2(g)} &\quad K_2^\theta
\end{aligned}$$

两式相加得：$\mathrm{SO_2(g) + NO_2(g) \rightleftharpoons SO_3(g) + NO(g)}$

则：$K_3^\theta = K_1^\theta \times K_2^\theta$

> 💡 **应用**：多重平衡规则在耦合平衡体系（如沉淀-配位、酸碱-配位）的计算中尤为有用，是竞赛中处理复杂体系的关键工具。

## 三、化学等温方程式与反应方向判断

### 3.1 等温方程式的推导

对于气相反应 $d\mathrm{D} + e\mathrm{E} \rightleftharpoons f\mathrm{F} + h\mathrm{H}$，当各组分分压为任意值 $p'$ 时：

$$\Delta_rG_m = \Delta_rG_m^\theta + RT\ln\frac{(p'_\mathrm{F}/p^\theta)^f(p'_\mathrm{H}/p^\theta)^h}{(p'_\mathrm{D}/p^\theta)^d(p'_\mathrm{E}/p^\theta)^e}$$

令 $Q = \dfrac{(p'_\mathrm{F}/p^\theta)^f(p'_\mathrm{H}/p^\theta)^h}{(p'_\mathrm{D}/p^\theta)^d(p'_\mathrm{E}/p^\theta)^e}$，则：

$$\Delta_rG_m = \Delta_rG_m^\theta + RT\ln Q = -RT\ln K^\theta + RT\ln Q$$

此即**化学反应的等温方程式**（van't Hoff等温式）。

### 3.2 $Q$ vs $K$ 判断反应方向

| 条件 | $\Delta_rG_m$ | 反应方向 |
|:---:|:---:|:---|
| $Q < K^\theta$ | $\Delta_rG_m < 0$ | **正向自发** |
| $Q = K^\theta$ | $\Delta_rG_m = 0$ | 处于平衡 |
| $Q > K^\theta$ | $\Delta_rG_m > 0$ | **逆向自发** |

> ⚠️ **核心思想**：$K$ 只与温度有关；$Q$ 与任意时刻的浓度/分压有关。改变浓度或压力只改变 $Q$ 而不改变 $K$，改变温度才改变 $K$。

### 3.3 反应商 $Q$ 的书写规则

$Q$ 的表达式形式与 $K^\theta$ 完全相同，区别在于 $Q$ 使用**任意时刻**的浓度/分压，而 $K^\theta$ 使用**平衡时**的浓度/分压。

**常见反应的 $Q$ 表达式**：

| 反应类型 | 示例 | $Q$ 表达式 |
|:---|:---|:---|
| 气相反应 | $\mathrm{N_2(g) + 3H_2(g) \rightleftharpoons 2NH_3(g)}$ | $Q = \dfrac{(p_{\mathrm{NH_3}}/p^\theta)^2}{(p_{\mathrm{N_2}}/p^\theta)(p_{\mathrm{H_2}}/p^\theta)^3}$ |
| 液相反应 | $\mathrm{HAc + H_2O \rightleftharpoons H_3O^+ + Ac^-}$ | $Q = \dfrac{[\mathrm{H_3O^+}][\mathrm{Ac^-}]}{[\mathrm{HAc}]}$ |
| 多相反应 | $\mathrm{CaCO_3(s) \rightleftharpoons CaO(s) + CO_2(g)}$ | $Q = p_{\mathrm{CO_2}}/p^\theta$ |

## 四、平衡移动原理

### 4.1 勒夏特列原理（定性）

> 假如改变平衡系统的条件之一（温度、压力或浓度），平衡就向着**减弱这个改变**的方向移动。

| 条件变化 | 平衡移动方向 | 本质解释 |
|:---|:---|:---|
| 增加反应物浓度 | 向正反应方向移动 | $Q < K$ |
| 减少生成物浓度 | 向正反应方向移动 | $Q < K$ |
| 增加总压力（$\Delta\nu < 0$ 的反应） | 向气体分子数减少的方向 | $Q < K$ |
| 降低总压力（$\Delta\nu > 0$ 的反应） | 向气体分子数增加的方向 | $Q > K$ |
| 升温（吸热反应） | 向正反应方向移动 | $K$ 增大 |
| 升温（放热反应） | 向逆反应方向移动 | $K$ 减小 |
| 加入催化剂 | 不移动 | 正逆反应速率同等加快，$K$ 不变 |

### 4.2 $Q$-$K$ 定量分析（与勒夏特列联动）

勒夏特列原理是定性工具，$Q$-$K$ 比较可以**定量**判断方向和计算新平衡的组成。

#### (1) 浓度的影响

增加反应物浓度 → 分母增大 → $Q < K$ → 平衡正向移动（$Q$ 增大直至等于 $K$）

**示例**：$\mathrm{CO(g) + H_2O(g) \rightleftharpoons CO_2(g) + H_2(g)}$，某温度下 $K_c = 9$。原平衡时 $c(\mathrm{CO}) = c(\mathrm{H_2O}) = 0.005\ \mathrm{mol\cdot L^{-1}}$，$c(\mathrm{CO_2}) = c(\mathrm{H_2}) = 0.015\ \mathrm{mol\cdot L^{-1}}$。若向体系中加 $\mathrm{H_2O(g)}$ 使其浓度增至 $1\ \mathrm{mol\cdot L^{-1}}$，求新平衡转化率。

设重新达平衡时转化掉的 $c(\mathrm{CO}) = x\ \mathrm{mol\cdot L^{-1}}$：

| | $\mathrm{CO}$ | $\mathrm{H_2O}$ | $\mathrm{CO_2}$ | $\mathrm{H_2}$ |
|:---|:---:|:---:|:---:|:---:|
| 起始浓度 | 0.005 | 1 | 0.015 | 0.015 |
| 变化浓度 | $-x$ | $-x$ | $+x$ | $+x$ |
| 平衡浓度 | $0.005-x$ | $1-x$ | $0.015+x$ | $0.015+x$ |

$$K_c = \frac{(0.015+x)^2}{(0.005-x)(1-x)} = 9$$

解得 $x \approx 0.00495$，转化率 $\alpha = \dfrac{0.00495}{0.005} \times 100\% = 99\%$。

> **结论**：让一种反应物**大幅过量**是提高另一种反应物转化率的最有效方法之一，广泛应用于化工生产（如合成氨中水蒸气过量、接触法制硫酸中氧气过量）。

#### (2) 压力的影响

**恒温下改变总压**：
- $\Delta\nu < 0$（气体分子数减少）：增大压力 → $Q < K$ → 正向移动
- $\Delta\nu > 0$（气体分子数增加）：增大压力 → $Q > K$ → 逆向移动
- $\Delta\nu = 0$：压力变动不影响平衡

**惰性气体的影响**（竞赛高频考点）：

| 条件 | 对平衡的影响 | 原因 |
|:---|:---:|:---|
| **恒容**充惰性气体 | **平衡不移动** | 各组分分压不变（$p_i = n_iRT/V$），$Q$ 不变 |
| **恒压**充惰性气体 | **向气体分子数增加方向移动** | 体积增大，各组分分压降低，相当于减压效果 |

> ⚠️ **易错点**：恒容充惰气总压虽然增加，但各组分分压不变，$Q$ 不变，平衡不移动！学生常错误地认为"总压增大平衡就移动"。

#### (3) 温度的影响

温度改变影响的是**平衡常数 $K$**（而非 $Q$），这是温度与浓度/压力影响的本质区别。

**van't Hoff方程**（温度与 $K$ 的定量关系）：

$$\ln\frac{K_1^\theta}{K_2^\theta} = -\frac{\Delta_rH_m^\theta}{R}\left(\frac{T_2 - T_1}{T_1\cdot T_2}\right)$$

![[media/6-1-lgKp-T-NH3-synthesis.jpg|合成氨反应的 lgKp - T⁻¹ 图]]
*图 合成氨反应 N₂(g)+3H₂(g)⇌2NH₃(g) 的 lgKp 对 T⁻¹ 图——数据点呈良好的线性关系，斜率为 -ΔH/2.303R。该反应为放热反应（ΔH < 0），升温使 K 减小，与图中 lgKp 随 T⁻¹ 增大（即温度降低）而升高的趋势一致（普化原理第4版 图6.1）*

![[media/6-2-lgKp-T-NaHCO3-decomposition.jpg|NaHCO₃ 分解反应的 lgKp - T⁻¹ 图]]
*图 碳酸氢钠分解 2NaHCO₃(s)⇌Na₂CO₃(s)+CO₂(g)+H₂O(g) 的 lgKp 对 T⁻¹ 图——该反应为吸热反应（ΔH > 0），升温使 K 增大，与合成氨的下降趋势相反（普化原理第4版 图6.2）*

| 反应热效应 | 升温效果 | 降温效果 |
|:---|:---|:---|
| 吸热反应（$\Delta_rH_m^\theta > 0$） | $K$ 增大，正向移动 | $K$ 减小，逆向移动 |
| 放热反应（$\Delta_rH_m^\theta < 0$） | $K$ 减小，逆向移动 | $K$ 增大，正向移动 |

> 💡 **记忆**：升温总是有利于**吸热方向**（正反应吸热则升温正向移动，正反应放热则升温逆向移动）。

## 五、平衡计算

### 5.1 平衡转化率

平衡转化率是指达到化学平衡时，已转化为生成物的反应物占该反应物起始总量的百分比：

$$\alpha = \frac{\text{反应物已转化的量}}{\text{反应物起始总量}} \times 100\%$$

若反应前后体积不变，也可用浓度表示：

$$\alpha = \frac{\text{起始浓度} - \text{平衡浓度}}{\text{起始浓度}} \times 100\%$$

**注意**：
- 平衡常数 $K$ 与起始状态无关，只与温度有关
- 平衡转化率 $\alpha$ 与起始状态密切相关——同一温度不同起始条件，转化率不同
- 有多个反应物时，不同反应物的转化率通常不同——对应关系需指明

### 5.2 平衡产率

$$\text{平衡产率} = \frac{\text{平衡时主要产品的量}}{\text{原料全部变为产品时的理论产量}} \times 100\%$$

> 产率通常用于有副反应的反应中。有副反应时，**产率 $\leq$ 转化率**（部分反应物转化为副产物）。

### 5.3 分解压

纯固体分解产生气体时，平衡时气体产物的压力称为该固体的**分解压**。

- 一种气体产物（如 $\mathrm{CaCO_3}$）：$K_p = p_{\mathrm{CO_2}}$，分解压 = $p_{\mathrm{CO_2}}$
- 多种气体产物（如 $\mathrm{NH_4HS}$）：分解压 = $p_{\mathrm{NH_3}} + p_{\mathrm{H_2S}}$

> ⚠️ **注意**：分解压与 $K_p$ 不一定相等——例如 $\mathrm{\frac{1}{2}CaCO_3(s) \rightleftharpoons \frac{1}{2}CaO(s) + \frac{1}{2}CO_2(g)}$，$K_p = p_{\mathrm{CO_2}}^{1/2}$，而分解压仍为 $p_{\mathrm{CO_2}}$。

## 六、典型例题

### 例1 平衡常数与转化率的综合计算 ⭐⭐

**题干**：某温度下，反应 $\mathrm{CO(g) + H_2O(g) \rightleftharpoons H_2(g) + CO_2(g)}$ 的 $K_c = 9$。若 CO 和 $\mathrm{H_2O}$ 的起始浓度均为 $0.02\ \mathrm{mol\cdot L^{-1}}$，求 CO 的平衡转化率。

**思路**：列 ICE 表，代入 $K_c$ 表达式求解。

**解析**：

设平衡时转化掉的 CO 浓度为 $x\ \mathrm{mol\cdot L^{-1}}$：

| | $\mathrm{CO}$ | $\mathrm{H_2O}$ | $\mathrm{CO_2}$ | $\mathrm{H_2}$ |
|:---|:---:|:---:|:---:|:---:|
| 起始 | 0.02 | 0.02 | 0 | 0 |
| 变化 | $-x$ | $-x$ | $+x$ | $+x$ |
| 平衡 | $0.02-x$ | $0.02-x$ | $x$ | $x$ |

$$K_c = \frac{x^2}{(0.02-x)^2} = 9$$

$$\frac{x}{0.02-x} = 3 \quad \Rightarrow \quad x = 0.015\ \mathrm{mol\cdot L^{-1}}$$

$$\alpha_{\mathrm{CO}} = \frac{0.015}{0.02} \times 100\% = 75\%$$

**答案**：CO 的平衡转化率为 **75%**。

**反思**：本题是 ICE 表的标准应用。注意此处 $K_c$ 表达式开方后得到线性方程，是因为反应前后气体分子数相等（$\Delta\nu = 0$），否则需解一元二次方程。

### 例2 温度对平衡常数的影响（van't Hoff方程） ⭐⭐⭐

**题干**：已知反应 $\mathrm{N_2(g) + 3H_2(g) \rightleftharpoons 2NH_3(g)}$ 在 673 K 时 $K_{p1}^\theta = 1.6\times10^{-4}$，$\Delta_rH_m^\theta = -92.4\ \mathrm{kJ\cdot mol^{-1}}$（视为常数）。求 773 K 时的 $K_{p2}^\theta$。

**思路**：用 van't Hoff 方程 $\ln(K_1/K_2) = -(\Delta_rH_m^\theta/R)(T_2^{-1} - T_1^{-1})$ 计算。

**解析**：

$$\ln\frac{K_{p2}^\theta}{K_{p1}^\theta} = -\frac{\Delta_rH_m^\theta}{R}\left(\frac{1}{T_2} - \frac{1}{T_1}\right)$$

代入数据：

$$\ln\frac{K_{p2}^\theta}{1.6\times10^{-4}} = -\frac{-92.4\times10^3}{8.314}\left(\frac{1}{773} - \frac{1}{673}\right)$$

$$= \frac{92.4\times10^3}{8.314}\left(\frac{1}{673} - \frac{1}{773}\right) = 11114 \times \frac{100}{673\times773}$$

$$\ln\frac{K_{p2}^\theta}{1.6\times10^{-4}} = 2.14$$

$$\frac{K_{p2}^\theta}{1.6\times10^{-4}} = e^{2.14} = 8.50$$

$$K_{p2}^\theta = 1.6\times10^{-4} \times 8.50 = 1.36\times10^{-3}$$

**答案**：773 K 时 $K_{p2}^\theta = \mathbf{1.36\times10^{-3}}$。

**反思**：合成氨是强放热反应（$\Delta_rH_m^\theta = -92.4\ \mathrm{kJ/mol}$），升温使 $K$ 减小（从 $1.6\times10^{-4}$ 降至更低值，而非升至 $1.36\times10^{-3}$——需仔细复查符号）。

> ⚠️ **符号检查**：$\Delta_rH_m^\theta < 0$（放热），升温应使 $K$ 减小。上面计算中代入 $\Delta_rH_m^\theta = -92.4$ 导致 $K_2 > K_1$，这与物理事实矛盾——正确的代入应将 $\Delta_rH_m^\theta$ 视为正值代入公式，或调整前后项顺序。van't Hoff方程的正确写法是：

$$\ln\frac{K_2}{K_1} = \frac{\Delta_rH_m^\theta}{R}\left(\frac{1}{T_1} - \frac{1}{T_2}\right)$$

对于放热反应（$\Delta_rH_m^\theta < 0$），当 $T_2 > T_1$ 时 $\ln(K_2/K_1) < 0$，故 $K_2 < K_1$。

### 例3 分解压与多重平衡计算 ⭐⭐⭐

**题干**：373 K 时，$\mathrm{2NaHCO_3(s) \rightleftharpoons Na_2CO_3(s) + CO_2(g) + H_2O(g)}$ 的 $K_p^\theta = 0.231$。在 $10^{-2}\ \mathrm{m^3}$ 的抽空容器中，放入 $0.1\ \mathrm{mol}\ \mathrm{Na_2CO_3(s)}$，并通入 $0.2\ \mathrm{mol}\ \mathrm{H_2O(g)}$。问最少需通入多少 $\mathrm{CO_2(g)}$ 才能使 $\mathrm{Na_2CO_3(s)}$ 全部转变为 $\mathrm{NaHCO_3(s)}$？

**思路**：先写出逆反应（$\mathrm{Na_2CO_3\to NaHCO_3}$）的平衡常数，再用 $Q \leq K$ 的条件求所需 $\mathrm{CO_2}$ 的最小量。

**解析**：

(1) **正逆反应 $K_c$ 的转换**：

正向反应：$2\mathrm{NaHCO_3(s)} \rightleftharpoons \mathrm{Na_2CO_3(s)} + \mathrm{CO_2(g)} + \mathrm{H_2O(g)}$，$K_p^\theta = 0.231$

逆反应（目标反应）：$\mathrm{Na_2CO_3(s)} + \mathrm{CO_2(g)} + \mathrm{H_2O(g)} \rightleftharpoons 2\mathrm{NaHCO_3(s)}$

$$K_{c,\text{逆}} = \frac{1}{K_p^\theta} \cdot \left(\frac{RT}{p^\theta}\right)^{-\Sigma\nu} = \frac{1}{0.231} \times \left(\frac{8.314\times373}{100000}\right)^2 = 4.163\times10^{-3}$$

(2) **求最低 $\mathrm{CO_2}$ 量**：

设通入 $\mathrm{CO_2}$ 的量为 $x\ \mathrm{mol}$。要使 $\mathrm{Na_2CO_3}$ 全部转化，需 $Q_c \leq K_c$（反应能正向进行或恰达平衡）。

$$Q_c = \frac{1}{[\mathrm{CO_2}][\mathrm{H_2O}]} = \frac{1}{(x/0.01)(0.2/0.01)} = \frac{10^{-4}}{0.2x}$$

$$Q_c \leq K_c \quad \Rightarrow \quad \frac{10^{-4}}{0.2x} \leq 4.163\times10^{-3}$$

$$x \geq 0.12\ \mathrm{mol}$$

(3) **考虑 $\mathrm{Na_2CO_3}$ 全部转化后的平衡**：

$0.1\ \mathrm{mol}\ \mathrm{Na_2CO_3}$ 全部转化需消耗 $0.1\ \mathrm{mol}\ \mathrm{CO_2}$ 和 $0.1\ \mathrm{mol}\ \mathrm{H_2O}$。转化后体系中 $\mathrm{H_2O}$ 剩余 $0.1\ \mathrm{mol}$，设与此平衡的 $\mathrm{CO_2}$ 浓度为 $y\ \mathrm{mol}$：

$$\frac{10^{-4}}{0.1y} = 4.163\times10^{-3} \quad \Rightarrow \quad y = 0.24\ \mathrm{mol}$$

因此，通入 $\mathrm{CO_2}$ 的量至少为 $0.12 + 0.24 = 0.34\ \mathrm{mol}$。

**答案**：至少需通入 **0.34 mol** $\mathrm{CO_2(g)}$。

**反思**：本题的难点在于"全部转化"≠"反应恰好达平衡"——转化需要消耗反应物，而消耗后体系中剩余的水蒸气仍需要足够的 $\mathrm{CO_2}$ 来维持 $\mathrm{NaHCO_3}$ 的稳定。若不考虑转化后的平衡维持，只按初始所需量计算会得到错误答案。

### 例4 恒容/恒压充惰性气体的影响 ⭐⭐⭐⭐

**题干**：在 $1000^\circ\mathrm{C}$、总压 3000 kPa 下，反应 $\mathrm{CO_2(g) + C(s) \rightleftharpoons 2CO(g)}$ 达到平衡时 $\mathrm{CO_2}$ 的摩尔分数为 0.17。
(1) 求该反应的 $K^\theta$。
(2) 若总压降至 2000 kPa，求新平衡时 $\mathrm{CO_2}$ 的摩尔分数。
(3) 若在恒温恒容条件下充入 $\mathrm{N_2}$ 使总压增至 4000 kPa，平衡是否移动？

**思路**：先由原平衡数据计算 $K^\theta$，再用 $K^\theta$ 不变反推新平衡组成。恒容充惰气各组分分压不变。

**解析**：

(1) **求 $K^\theta$**：

原平衡时：$p(\mathrm{CO}) = 3000(1-0.17) = 2490\ \mathrm{kPa}$，$p(\mathrm{CO_2}) = 3000\times0.17 = 510\ \mathrm{kPa}$

$$K^\theta = \frac{(p_{\mathrm{CO}}/p^\theta)^2}{p_{\mathrm{CO_2}}/p^\theta} = \frac{(2490/100)^2}{510/100} = 122$$

(2) **求新平衡组成**：

设新平衡时 $\mathrm{CO_2}$ 的摩尔分数为 $x$：

$$K^\theta = \frac{[2000(1-x)/100]^2}{2000x/100} = 122$$

$$\frac{400(1-x)^2}{20x} = 122$$

$$\frac{20(1-x)^2}{x} = 122$$

$$20(1-2x+x^2) = 122x$$

$$20x^2 - 162x + 20 = 0$$

解得 $x \approx 0.126$（舍去不合理根 $x \approx 7.9$）

$\mathrm{CO_2}$ 的摩尔分数从 0.17 降至 0.126，说明降压有利于 $\Delta\nu > 0$ 的正反应方向，验证勒夏特列原理。

(3) **恒容充 $\mathrm{N_2}$**：

恒温恒容充惰性气体，各组分分压不变（$p_i = n_iRT/V$），因此 $Q = K^\theta$，**平衡不移动**。

**答案**：(1) $K^\theta = 122$；(2) $x(\mathrm{CO_2}) = 0.126$；(3) 不移动。

## 七、规律与方法

### 7.1 化学平衡解题决策树

**1. 判方向（是否已达平衡？）**
   - 比较 Q 与 K：Q<K 正向，Q=K 平衡，Q>K 逆向
   - 通过 ΔG：ΔG = -RTlnK + RTlnQ

**2. 求平衡常数（是否已知平衡组成？）**
   - 已知平衡浓度 → 直接代 Kc 表达式
   - 已知平衡分压 → 代 Kp 表达式
   - 已知 ΔGθ → Kθ = exp(-ΔGθ/RT)
   - 已知 K1(T1) + ΔH → van't Hoff 求 K2(T2)

**3. 算平衡组成（是否已知 K 和初始条件？）**
   - 列 ICE 表（Initial-Change-Equilibrium）
   - 代 K 表达式 → 解方程求 x
   - 求转化率 α 或产率

**4. 判移动（外界条件改变后？）**
   - 浓度改变 → 只改 Q，不改 K → 判 Q vs K
   - 压力改变 → 恒容充惰气不改 Q；Δν≠0 时加压/减压改 Q
   - 温度改变 → 改 K（van't Hoff），判新平衡位置

### 7.2 易错点与常见误区

| 误区 | 正确理解 |
|:---|:---|
| "恒容充惰气总压增大，平衡向分子数减少方向移动" | 恒容充惰气各组分分压不变，$Q$ 不变，平衡不移动 |
| "$K_c$ 与 $K_p$ 数值相等" | $K_p = K_c(RT)^{\Delta\nu}$，仅当 $\Delta\nu = 0$ 时二者相等 |
| "转化率与平衡常数是一回事" | $K$ 只与温度有关，$\alpha$ 与起始条件也有关 |
| "固体和液体不出现在 $K$ 中是因为不参与反应" | 是因为纯固体/液体的活度为 1，视为常数已并入 $K$ |
| "催化剂能移动平衡" | 催化剂同等加快正逆反应速率，$K$ 不变，平衡不移动 |
| "放热反应升温 $K$ 增大" | 放热反应升温 $K$ 减小（$\Delta_rH_m^\theta < 0$ 使 van't Hoff 右端为负）|

### 7.3 ICE 表实用技巧

```
写法模板：
         aA     +    bB    ⇌    dD    +    eE
初始     cA0         cB0         cD0         cE0
变化    -ax         -bx         +dx         +ex
平衡   cA0-ax      cB0-bx      cD0+dx      cE0+ex

关键检查：
  ☐ 变化量 x 的系数与计量系数匹配
  ☐ 初始状态若含产物（非从零开始），注意"逆向变化"的可能
  ☐ 代 K 后方程若为高次，考虑简化（如 x << c0 时近似）
  ☐ 最终 x 必须在合理范围内（不能出现负浓度）
```

## 九、本节总结·速查

| 核心概念 | 关键公式 | 注意点 |
|:---|:---|:---|
| 化学平衡条件 | $\Delta_rG = 0$，$r_+ = r_-$ | 动态平衡，反应未停止 |
| 平衡常数 $K_c$ | $K_c = \prod [\text{生成物}]^{\nu} / \prod [\text{反应物}]^{\nu}$ | 纯固体/液体不写入 |
| $K_c$ 与 $K_p$ | $K_p = K_c(RT)^{\Delta\nu}$ | $R$ 值单位匹配 |
| 标准平衡常数 $K^\theta$ | $K^\theta = \prod(p_i/p^\theta)^{\nu_i}$ | 无量纲，与 $\Delta G^\theta$ 关联 |
| 等温方程式 | $\Delta_rG = -RT\ln K^\theta + RT\ln Q$ | $Q < K$ 正向，$Q > K$ 逆向 |
| van't Hoff方程 | $\ln(K_2/K_1) = -\dfrac{\Delta_rH_m^\theta}{R}\left(\dfrac{1}{T_2} - \dfrac{1}{T_1}\right)$ | $\Delta H$ 正负决定 $K$ 随 $T$ 变化方向 |
| 多重平衡 | $K_{\text{总}} = K_1 \times K_2$（反应相加） | 只适用于相同温度 |
| 勒夏特列原理 | 平衡向减弱改变的方向移动 | 催化剂不移动平衡 |
| 转化率 $\alpha$ | $\alpha = \text{转化量}/\text{起始量}$ | 与起始条件有关 |
| 分解压 | 固体分解时气体产物的平衡总压 | 不一定等于 $K_p$ |
