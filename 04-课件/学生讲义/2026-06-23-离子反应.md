# 第九讲：离子反应

> **适用**：第一轮（初学）→ 第二轮（深化）
> **对应备课大纲**：[[04-课件/备课大纲/2026-06-02-酸碱理论-基础班]] · [[04-课件/备课大纲/2026-06-03-沉淀溶解平衡-基础班]] · [[04-课件/备课大纲/2026-06-02-电化学-基础班]]
> **前置要求**：酸碱质子理论（Brønsted）、沉淀溶解平衡（Ksp）、配位平衡（K稳）、氧化还原反应基本概念
> **深度边界**：本讲为上述各平衡体系的综合应用与总结。第一轮聚焦（1）各类离子反应的特征与方向判断；（2）离子共存规则；（3）离子方程式的规范书写。不展开非水溶剂体系（液氨、液态SO₂、BrF₃、无水HF）的反应，不展开复杂双水解的完整计算。

---

## 学习目标

- ✅ 目标1：能区分酸碱反应、水解反应、沉淀反应、配位反应和氧化还原反应五类离子反应，掌握各类反应的方向判断依据。
- ✅ 目标2：能用平衡常数（Ka、Kb、Ksp、K稳、K°）计算离子反应进行的程度，判断反应方向。
- ✅ 目标3：能根据离子共存规则判断溶液中离子能否大量共存。
- ✅ 目标4：能规范书写离子方程式，正确拆分强电解质与保留化学式，处理反应物相对量、滴加顺序、反应条件对产物的影响。
- ✅ 目标5：能区分完全双水解与不完全双水解，写出对应的离子方程式。

## 一、离子反应的类型与特征

在溶液中有离子参加或有离子生成的反应称为离子反应。水溶液中的离子反应可归纳为以下五类：

### 1.1 酸碱反应

根据 Brønsted 质子理论，酸碱反应的本质是质子的传递：酸给出质子，碱接受质子。

**方向判断**：反应自发方向为**强酸制弱酸**（或强碱制弱碱），即反应平衡常数可由共轭酸碱对的 Ka 计算：

$$
K = \frac{K_{\text{酸1}}}{K_{\text{酸2}}}
$$

其中酸1为反应物中的酸，酸2为生成物中的酸。$K \gg 1$ 时正向进行完全。

**示例**：$\text{HF} + \text{CN}^- \rightleftharpoons \text{F}^- + \text{HCN}$

$K_a(\text{HF}) = 6.8 \times 10^{-4}$，$K_a(\text{HCN}) = 6.2 \times 10^{-10}$，$K = \frac{6.8 \times 10^{-4}}{6.2 \times 10^{-10}} = 1.1 \times 10^6 \gg 1$，反应正向进行。

### 1.2 水解反应

盐类水解的本质是盐的离子与水电离出的 H⁺ 或 OH⁻ 结合生成弱电解质的过程。从 Brønsted 酸碱理论看，水解是共轭酸碱对与水之间的质子传递反应。

**方向判断**：水解反应的平衡常数 $K_h$ 可由对应弱酸/弱碱的 Ka/Kb 计算：

| 水解类型 | 示例 | Kh 表达式 |
|:---|:---|:---|
| 弱酸根水解 | $\text{Ac}^- + \text{H}_2\text{O} \rightleftharpoons \text{HAc} + \text{OH}^-$ | $K_h = K_w / K_a(\text{HAc})$ |
| 弱碱阳离子水解 | $\text{NH}_4^+ + \text{H}_2\text{O} \rightleftharpoons \text{NH}_3·\text{H}_2\text{O} + \text{H}^+$ | $K_h = K_w / K_b(\text{NH}_3)$ |
| 弱酸弱碱盐水解 | $\text{NH}_4^+ + \text{Ac}^- + \text{H}_2\text{O} \rightleftharpoons \text{NH}_3·\text{H}_2\text{O} + \text{HAc}$ | $K_h = K_w / (K_b \cdot K_a)$ |
| 高价金属阳离子水解 | $\text{Fe}^{3+} + 3\text{H}_2\text{O} \rightleftharpoons \text{Fe(OH)}_3 + 3\text{H}^+$ | $K_h = K_w^3 / K_{\text{sp}}$ |

**完全双水解 vs 不完全双水解**：

| 特征 | 不完全双水解 | 完全双水解 |
|:---|:---|:---|
| Kh 大小 | $K_h \ll 1$ | $K_h \gg 1$ |
| 离子方程式符号 | 可逆符号（⇌） | 等号（=）和箭头（↑↓） |
| 离子共存 | 可大量共存 | 不能大量共存 |
| 示例 | $\text{NH}_4^+ + \text{Ac}^- + \text{H}_2\text{O} \rightleftharpoons \text{NH}_3·\text{H}_2\text{O} + \text{HAc}$ | $\text{Al}^{3+} + 3\text{HCO}_3^- = \text{Al(OH)}_3 \downarrow + 3\text{CO}_2 \uparrow$ |

常见发生**完全双水解**的离子对：
- 阳离子：$\text{Al}^{3+}$、$\text{Fe}^{3+}$
- 阴离子：$\text{CO}_3^{2-}$、$\text{HCO}_3^-$、$\text{S}^{2-}$、$\text{HS}^-$、$\text{AlO}_2^-$

> ⚠️ 注意：$\text{Fe}^{3+}$ 与 $\text{S}^{2-}$、$\text{SO}_3^{2-}$ 等既能发生双水解也能发生氧化还原反应，实际体系中两者均可发生，需根据实验条件（浓度、滴加顺序）判断主反应。

### 1.3 沉淀反应

沉淀反应的本质是离子生成难溶电解质，使溶液中离子浓度降低。

**方向判断**：利用溶度积规则——$Q > K_{\text{sp}}$ 时沉淀生成，$Q < K_{\text{sp}}$ 时沉淀溶解，$Q = K_{\text{sp}}$ 时平衡。

**沉淀转化的平衡常数**：
$$
\mathrm{BaSO_4(s) + CO_3^{2-} \rightleftharpoons BaCO_3(s) + SO_4^{2-}}
$$
$$
K = \frac{K_{\text{sp}}(\mathrm{BaSO_4})}{K_{\text{sp}}(\mathrm{BaCO_3})} = \frac{1.1 \times 10^{-10}}{2.6 \times 10^{-9}} = 4.2 \times 10^{-2}
$$

### 1.4 配位反应

配位反应的本质是 Lewis 酸碱反应：中心离子（Lewis 酸）与配体（Lewis 碱）通过配位键形成配合物。

**方向判断**：配位反应的平衡常数可由稳定常数 $K_{\text{稳}}$ 计算。配体取代反应的 $K$ 等于生成物 $K_{\text{稳}}$ 与反应物 $K_{\text{稳}}$ 之比。

### 1.5 氧化还原反应

氧化还原反应的本质是电子转移，反应方向由电极电势决定。

**方向判断**：$E_{\text{cell}} = \varphi_{(+)} - \varphi_{(-)} > 0$ 时反应自发。$E^\circ_{\text{cell}}$ 与平衡常数的关系：$\ln K = nFE^\circ_{\text{cell}}/RT$。

## 二、离子反应的实质与方向判断

### 2.1 复分解反应的方向

复分解反应（包括酸碱反应、沉淀反应、部分水解反应）发生的方向是**向溶液中离子浓度减小的方向进行**——生成更难解离、更难溶或更易挥发的物质。

> ⚠️ **例外**：配位反应和氧化还原反应不遵循此规律。配位反应和氧化还原反应中，总离子浓度可能增加，但反应仍可进行完全（由 $K_{\text{稳}}$ 或 $E$ 决定）。

### 2.2 多重平衡的方向判断

当沉淀平衡、配位平衡、酸碱平衡、氧化还原平衡耦合时，总反应的平衡常数等于各步平衡常数的乘积（或组合），$K_{\text{总}} \gg 1$ 时正向进行完全。

**示例**——$\text{AgCl}$ 沉淀在氨水中的溶解：

$$
\mathrm{AgCl(s) + 2NH_3 \rightleftharpoons [Ag(NH_3)_2]^+ + Cl^-}
$$

该反应可拆分为：
- $\mathrm{AgCl(s) \rightleftharpoons Ag^+ + Cl^-}$，$K_1 = K_{\text{sp}} = 1.8 \times 10^{-10}$
- $\mathrm{Ag^+ + 2NH_3 \rightleftharpoons [Ag(NH_3)_2]^+}$，$K_2 = K_{\text{稳}} = 1.1 \times 10^7$

$$
K_{\text{总}} = K_1 \times K_2 = 1.8 \times 10^{-10} \times 1.1 \times 10^7 = 2.0 \times 10^{-3}
$$

$K_{\text{总}} = 2.0 \times 10^{-3} < 1$，因此 $\text{AgCl}$ 在氨水中的溶解度有限，需较高浓度的氨水才能完全溶解。

> 更多多重平衡耦合的计算方法参见 [[04-课件/学生讲义/2026-06-23-水中的几种平衡#六、配位平衡]] 中的示例。

## 三、离子共存

### 3.1 不能共存的常见情况

| 反应类型 | 不能大量共存的离子对 | 原因 |
|:---|:---|:---|
| **酸碱反应** | H⁺ 与 OH⁻、Ac⁻、CO₃²⁻、S²⁻ 等 | 生成弱酸/弱碱或水 |
| | OH⁻ 与 HCO₃⁻、HS⁻、HPO₄²⁻、NH₄⁺ 等 | 酸式酸根进一步电离或 NH₄⁺ 结合 OH⁻ |
| **沉淀反应** | Ba²⁺/Ca²⁺ 与 SO₄²⁻/CO₃²⁻ | 生成难溶盐沉淀 |
| | Cu²⁺/Fe³⁺/Mg²⁺ 与 OH⁻ | 生成难溶氢氧化物 |
| **配位反应** | Fe³⁺ 与 SCN⁻、C₆H₅O⁻ | 生成配合物 |
| | Ag⁺ 与 NH₃（低浓度时）、S₂O₃²⁻ | 生成配合物 |
| **氧化还原** | MnO₄⁻、Cr₂O₇²⁻、ClO⁻ 等强氧化剂 | 氧化 I⁻、S²⁻、SO₃²⁻、Fe²⁺ 等还原剂 |
| | 酸性条件下的 NO₃⁻ | 相当于 HNO₃，氧化 Fe²⁺、I⁻、SO₃²⁻ 等 |
| **双水解** | Al³⁺/Fe³⁺ 与 CO₃²⁻/HCO₃⁻/S²⁻/AlO₂⁻ | 完全双水解反应 |

### 3.2 审题注意事项

1. **溶液颜色**：无色溶液中不含有色离子（$\text{MnO}_4^-$ 紫、$\text{Fe}^{3+}$ 黄、$\text{Fe}^{2+}$ 浅绿、$\text{Cu}^{2+}$ 蓝）
2. **酸碱性**：酸性（含 H⁺ 或 pH < 7）或碱性（含 OH⁻ 或 pH > 7）条件分别限制共存离子
3. **"一定能"还是"可能"**：注意题目是要求"一定能大量共存"还是"可能大量共存"
4. **隐含条件**：$\text{NO}_3^-$ + H⁺ 构成强氧化性环境，相当于 $\text{HNO}_3$

## 四、离子方程式的书写

### 4.1 基本规则

| 规则 | 说明 | 示例 |
|:---|:---|:---|
| **强电解质拆，弱电解质不拆** | 可溶性强电解质拆为离子；弱电解质、难溶物、气体、单质、氧化物保留化学式 | $\text{Na}_2\text{CO}_3$ 拆为 $2\text{Na}^+ + \text{CO}_3^{2-}$；$\text{CaCO}_3$ 不拆 |
| **微溶物视情况处理** | 澄清溶液中作反应物时拆；悬浊液中或作生成物时不拆 | 澄清 $\text{Ca(OH)}_2$ 拆；石灰乳不拆 |
| **酸式盐** | $\text{HSO}_4^-$ 拆为 $\text{H}^+ + \text{SO}_4^{2-}$；其他弱酸酸式根不拆 | $\text{HCO}_3^-$ 不拆成 $\text{H}^+ + \text{CO}_3^{2-}$ |
| **质量守恒+电荷守恒** | 配平须同时满足原子个数和电荷数平衡 | $\text{Fe} + 2\text{Fe}^{3+} = 3\text{Fe}^{2+}$（不能写成 $\text{Fe} + \text{Fe}^{3+} = 2\text{Fe}^{2+}$） |

### 4.2 反应物相对量的影响

#### (1) 生成的产物与过量物质继续反应

| 条件 | 离子方程式 |
|:---|:---|
| 少量 CO₂ 通入 NaOH | $\text{CO}_2 + 2\text{OH}^- = \text{CO}_3^{2-} + \text{H}_2\text{O}$ |
| 过量 CO₂ 通入 NaOH | $\text{CO}_2 + \text{OH}^- = \text{HCO}_3^-$ |
| 少量氨水滴入 AgNO₃ | $\text{Ag}^+ + \text{NH}_3·\text{H}_2\text{O} = \text{AgOH} \downarrow + \text{NH}_4^+$ |
| 过量氨水滴入 AgNO₃ | $\text{Ag}^+ + 2\text{NH}_3·\text{H}_2\text{O} = [\text{Ag(NH}_3)_2]^+ + 2\text{H}_2\text{O}$ |

#### (2) 滴加顺序不同

| 操作 | 离子方程式 |
|:---|:---|
| AlCl₃ 中滴加少量 NaOH | $\text{Al}^{3+} + 3\text{OH}^- = \text{Al(OH)}_3 \downarrow$ |
| AlCl₃ 中滴加过量 NaOH | $\text{Al}^{3+} + 4\text{OH}^- = \text{AlO}_2^- + 2\text{H}_2\text{O}$ |
| NaOH 中滴加少量 AlCl₃ | $4\text{OH}^- + \text{Al}^{3+} = \text{AlO}_2^- + 2\text{H}_2\text{O}$ |
| NaOH 中滴加过量 AlCl₃ | $\text{Al}^{3+} + 3\text{OH}^- = \text{Al(OH)}_3 \downarrow$ |

#### (3) 反应有先后顺序

氧化还原反应中，还原性强的离子优先被氧化。

**示例**——$\text{FeBr}_2$ 溶液中通入 Cl₂：
还原性顺序：$\text{I}^- > \text{Fe}^{2+} > \text{Br}^-$

| 条件 | 反应 |
|:---|:---|
| 少量 Cl₂ 通入 FeI₂ 溶液 | $2\text{I}^- + \text{Cl}_2 = \text{I}_2 + 2\text{Cl}^-$ |
| 适量 Cl₂ 通入 FeBr₂ 溶液 | $2\text{Fe}^{2+} + \text{Cl}_2 = 2\text{Fe}^{3+} + 2\text{Cl}^-$ |
| 过量 Cl₂ 通入 FeBr₂ 溶液 | $2\text{Fe}^{2+} + 4\text{Br}^- + 3\text{Cl}_2 = 2\text{Fe}^{3+} + 2\text{Br}_2 + 6\text{Cl}^-$ |

### 4.3 反应条件的影响

| 条件 | 影响 | 示例 |
|:---|:---|:---|
| **温度** | 冷/热 NaOH 与 Cl₂ 产物不同 | 冷：$\text{Cl}_2 + 2\text{OH}^- = \text{ClO}^- + \text{Cl}^- + \text{H}_2\text{O}$；热：$3\text{Cl}_2 + 6\text{OH}^- = \text{ClO}_3^- + 5\text{Cl}^- + 3\text{H}_2\text{O}$ |
| **浓度** | 浓/稀 HNO₃ 与 Cu 产物不同 | 浓：$\text{Cu} + 4\text{H}^+ + 2\text{NO}_3^- = \text{Cu}^{2+} + 2\text{NO}_2 \uparrow + 2\text{H}_2\text{O}$；稀：$3\text{Cu} + 8\text{H}^+ + 2\text{NO}_3^- = 3\text{Cu}^{2+} + 2\text{NO} \uparrow + 4\text{H}_2\text{O}$ |
| **酸碱性** | $\text{MnO}_4^-$ 在不同介质中还原产物不同 | 酸性→$\text{Mn}^{2+}$（无色），中性→$\text{MnO}_2$（棕），碱性→$\text{MnO}_4^{2-}$（绿） |
| **酸碱性强弱** | $\text{C}_6\text{H}_5\text{O}^-$ 与 $\text{CO}_2$ 只能生成 $\text{HCO}_3^-$ | $\text{C}_6\text{H}_5\text{O}^- + \text{CO}_2 + \text{H}_2\text{O} = \text{C}_6\text{H}_5\text{OH} + \text{HCO}_3^-$（酚的酸性比 $\text{HCO}_3^-$ 弱） |

### 4.4 隐含因素的挖掘

有些离子反应包含隐含条件——表面上看是一种反应，实际包含另一种反应。

**示例 1**：$\text{Fe(OH)}_2$ 溶于足量稀 $\text{HNO}_3$
错误写法：$\text{Fe(OH)}_2 + 2\text{H}^+ = \text{Fe}^{2+} + 2\text{H}_2\text{O}$
正确写法：$3\text{Fe(OH)}_2 + 10\text{H}^+ + \text{NO}_3^- = 3\text{Fe}^{3+} + \text{NO} \uparrow + 8\text{H}_2\text{O}$
原因：稀 $\text{HNO}_3$ 具有强氧化性，会将 $\text{Fe}^{2+}$ 进一步氧化为 $\text{Fe}^{3+}$。

**示例 2**：$\text{SO}_2$ 通入 $\text{Ba(NO}_3)_2$ 溶液
错误判断：$\text{SO}_2$ 与 $\text{BaCl}_2$ 不反应，所以与 $\text{Ba(NO}_3)_2$ 也不反应。
正确分析：$\text{SO}_2$ 溶于水呈酸性，$\text{NO}_3^-$ 在酸性条件下表现强氧化性，将 $\text{SO}_3^{2-}$ 氧化为 $\text{SO}_4^{2-}$，进而与 $\text{Ba}^{2+}$ 生成 $\text{BaSO}_4$ 沉淀。
离子方程式：$3\text{SO}_2 + 3\text{Ba}^{2+} + 2\text{NO}_3^- + 2\text{H}_2\text{O} = 3\text{BaSO}_4 \downarrow + 2\text{NO} \uparrow + 4\text{H}^+$

## 五、典型例题

### 例1 离子反应方向判断（用平衡常数计算）⭐⭐⭐

**题干**：根据电离平衡常数 K 的数据，判断下列离子反应能否发生，并计算平衡常数。
已知 298 K 时，$K_a(\text{CH}_3\text{COOH}) = 1.7 \times 10^{-5}$，$K_{a1}(\text{H}_2\text{CO}_3) = 4.3 \times 10^{-7}$，$K_{a2}(\text{H}_2\text{CO}_3) = 5.6 \times 10^{-11}$。

(1) $\text{CH}_3\text{COO}^- + \text{HCO}_3^- \rightleftharpoons \text{CH}_3\text{COOH} + \text{CO}_3^{2-}$
(2) $\text{CH}_3\text{COOH} + \text{CO}_3^{2-} \rightleftharpoons \text{CH}_3\text{COO}^- + \text{HCO}_3^-$

**思路**：将总反应拆分为两个半反应（共轭酸碱对间的质子传递），利用 $K_a$ 之比计算平衡常数。

**解析**：

(1) 反应中 $\text{HCO}_3^-$ 作酸（质子给体），$\text{CH}_3\text{COO}^-$ 作碱（质子受体）。
   $$
   K = \frac{K_{a2}(\text{H}_2\text{CO}_3)}{K_a(\text{CH}_3\text{COOH})} = \frac{5.6 \times 10^{-11}}{1.7 \times 10^{-5}} = 3.3 \times 10^{-6} \ll 1
   $$
   正向几乎不反应；逆向可以自发进行。

(2) 反应中 $\text{CH}_3\text{COOH}$ 作酸，$\text{CO}_3^{2-}$ 作碱（分步接受质子，先转化为 $\text{HCO}_3^-$）。
   $$
   K = \frac{K_a(\text{CH}_3\text{COOH})}{K_{a2}(\text{H}_2\text{CO}_3)} = \frac{1.7 \times 10^{-5}}{5.6 \times 10^{-11}} = 3.0 \times 10^5 \gg 1
   $$
   正向可以进行完全。

**答案**：(1) 不能发生（$K = 3.3 \times 10^{-6}$）；(2) 能发生（$K = 3.0 \times 10^5$）。

### 例2 离子共存判断 ⭐⭐

**题干**：下列各组离子在指定溶液中能否大量共存？说明理由。
(1) 无色透明溶液：$\text{MnO}_4^-$、$\text{K}^+$、$\text{SO}_4^{2-}$、$\text{Na}^+$
(2) 酸性溶液：$\text{Fe}^{2+}$、$\text{NO}_3^-$、$\text{H}^+$、$\text{Cl}^-$
(3) 含 $\text{Al}^{3+}$ 的溶液：$\text{HCO}_3^-$、$\text{Na}^+$、$\text{Cl}^-$、$\text{K}^+$
(4) $\text{Fe}^{3+}$ 和 $\text{S}^{2-}$ 在水溶液中

**解析**：

(1) **不能大量共存**。$\text{MnO}_4^-$ 显紫色，无色溶液中不能存在有色离子。
(2) **不能大量共存**。酸性条件下 $\text{NO}_3^-$ 具有强氧化性，可氧化 $\text{Fe}^{2+}$：$3\text{Fe}^{2+} + \text{NO}_3^- + 4\text{H}^+ = 3\text{Fe}^{3+} + \text{NO} \uparrow + 2\text{H}_2\text{O}$。
(3) **不能大量共存**。$\text{Al}^{3+}$ 与 $\text{HCO}_3^-$ 发生完全双水解：$\text{Al}^{3+} + 3\text{HCO}_3^- = \text{Al(OH)}_3 \downarrow + 3\text{CO}_2 \uparrow$。
(4) **不能大量共存**。$\text{Fe}^{3+}$ 与 $\text{S}^{2-}$ 可同时发生氧化还原反应和双水解反应，两种反应均使离子浓度大幅降低。

**答案**：均不能大量共存，原因见解析。

### 例3 反应物相对量对离子方程式的影响 ⭐⭐⭐

**题干**：写出下列反应的离子方程式：
(1) 少量 NaHCO₃ 溶液与澄清石灰水反应
(2) 过量 NaHCO₃ 溶液与澄清石灰水反应
(3) 4 mol FeBr₂ 溶液中通入 3 mol Cl₂

**思路**：先判断反应物相对量，确定哪种反应物过量；再根据过量物质与产物的后续反应，确定最终产物。

**解析**：

(1) 少量 $\text{NaHCO}_3$ → $\text{HCO}_3^-$ 少量，一个 $\text{HCO}_3^-$ 消耗一个 $\text{OH}^-$ 生成 $\text{CO}_3^{2-}$，$\text{Ca}^{2+}$ 与 $\text{CO}_3^{2-}$ 结合沉淀：
   $$
   \text{HCO}_3^- + \text{Ca}^{2+} + \text{OH}^- = \text{CaCO}_3 \downarrow + \text{H}_2\text{O}
   $$

(2) 过量 $\text{NaHCO}_3$ → $\text{Ca}^{2+}$ 全部沉淀，$\text{HCO}_3^-$ 与 $\text{OH}^-$ 全部反应：
   $$
   2\text{HCO}_3^- + \text{Ca}^{2+} + 2\text{OH}^- = \text{CaCO}_3 \downarrow + 2\text{H}_2\text{O} + \text{CO}_3^{2-}
   $$

(3) 4 mol $\text{FeBr}_2$ 含 4 mol $\text{Fe}^{2+}$ 和 8 mol $\text{Br}^-$。3 mol $\text{Cl}_2$ 得到 6 mol 电子。还原性 $\text{Fe}^{2+} > \text{Br}^-$，因此 $\text{Cl}_2$ 先氧化 4 mol $\text{Fe}^{2+}$（消耗 2 mol $\text{Cl}_2$ 和 4 mol 电子），剩余 1 mol $\text{Cl}_2$（2 mol 电子）氧化 2 mol $\text{Br}^-$：
   $$
   4\text{Fe}^{2+} + 2\text{Br}^- + 3\text{Cl}_2 = 4\text{Fe}^{3+} + \text{Br}_2 + 6\text{Cl}^-
   $$

**答案**：见解析。

## 六、规律与方法

### 6.1 离子方程式书写六步检查法

| 步骤 | 检查内容 | 常见错误 |
|:---:|:---|:---|
| ① | 强电解质是否拆为离子形式 | 难溶物/弱电解质错误拆分 |
| ② | 反应物相对量是否考虑 | 产物与过量物质继续反应被忽略 |
| ③ | 质量是否守恒 | 原子个数未配平 |
| ④ | 电荷是否守恒 | 左右电荷数不一致 |
| ⑤ | 电子转移是否守恒（氧化还原） | 得失电子数不相等 |
| ⑥ | 条件是否符合（温度/浓度/酸碱性） | 忽略 $\text{NO}_3^-$ 在酸性条件下的氧化性 |

### 6.2 离子共存快速判断逻辑

```
溶液中有无有色离子？                → 有则排除
↓ 无
溶液是酸性还是碱性？                → 与 H⁺/OH⁻ 反应的离子排除
↓ 已排除
检查氧化性离子与还原性离子共存      → 能氧化则排除
↓ 无氧化还原
检查双水解离子对                    → 完全双水解则排除
↓ 无双水解
检查配位反应                        → 生成稳定配合物则排除
↓ 无配位
→ 可以大量共存
```

## 八、本节总结 · 速查

| 反应类型 | 方向判断依据 | 核心常数 |
|:---|:---|:---|
| 酸碱反应 | 强酸制弱酸（强碱制弱碱） | $K_a$、$K_b$ |
| 水解反应 | $K_h = K_w/K_a$ 或 $K_w/K_b$；$K_h \gg 1$ 为完全双水解 | $K_h$、$K_w$ |
| 沉淀反应 | $Q > K_{\text{sp}}$ 沉淀生成；$Q < K_{\text{sp}}$ 沉淀溶解 | $K_{\text{sp}}$ |
| 配位反应 | $K_{\text{稳}}$ 越大，配合物越稳定 | $K_{\text{稳}}$ |
| 氧化还原反应 | $E_{\text{cell}} = \varphi_{(+)} - \varphi_{(-)} > 0$ | $E^\circ$、$K^\circ$ |
| 多重平衡 | $K_{\text{总}} = K_1 \times K_2 \times \cdots$，$K_{\text{总}} \gg 1$ 正向完全 | 各步 K 的乘积 |

## 十、参考答案与提示

**1**. (a) $\text{HAc} + \text{OH}^- = \text{Ac}^- + \text{H}_2\text{O}$
   (b) $\text{CaCO}_3 + 2\text{H}^+ = \text{Ca}^{2+} + \text{CO}_2 \uparrow + \text{H}_2\text{O}$
   (c) $\text{Ba}^{2+} + \text{SO}_4^{2-} = \text{BaSO}_4 \downarrow$

**2**. (a) 不能（H⁺ + OH⁻ = H₂O）；(b) 不能（Fe³⁺ 与 SCN⁻ 配位）；(c) 不能（Ba²⁺ 与 SO₄²⁻ 沉淀）

**3**. (a) $K = \frac{K_w}{K_b \cdot K_{a2}} = \frac{10^{-14}}{1.7 \times 10^{-5} \times 5.6 \times 10^{-11}} = 10.5 > 1$，反应可正向进行，但不完全。
   (b) $K = \frac{K_w^3}{K_{\text{sp}} \cdot K_a^3} = \frac{10^{-42}}{3.0 \times 10^{-34} \times (3.0 \times 10^{-8})^3} = 1.2 \times 10^{14} \gg 1$，反应完全。

**4**. (a) $\text{Al}^{3+} + 3\text{OH}^- = \text{Al(OH)}_3 \downarrow$
   (b) $\text{Al}^{3+} + 4\text{OH}^- = \text{AlO}_2^- + 2\text{H}_2\text{O}$
   (c) $\text{Al}^{3+} + 4\text{OH}^- = \text{AlO}_2^- + 2\text{H}_2\text{O}$（注意：顺序改变后，OH⁻ 最初过量，产物直接为 $\text{AlO}_2^-$）

**5**. $E^\circ_{\text{cell}} = 0.77 - 0.54 = 0.23 \, \text{V} > 0$，反应自发：$2\text{Fe}^{3+} + 2\text{I}^- = 2\text{Fe}^{2+} + \text{I}_2$。

**6**. $3\text{SO}_2 + 3\text{Ba}^{2+} + 2\text{NO}_3^- + 2\text{H}_2\text{O} = 3\text{BaSO}_4 \downarrow + 2\text{NO} \uparrow + 4\text{H}^+$。$\text{SO}_2$ 溶于水呈酸性，$\text{NO}_3^-$ 在 H⁺ 存在下具强氧化性，将 $\text{SO}_3^{2-}$ 氧化为 $\text{SO}_4^{2-}$，与 $\text{Ba}^{2+}$ 沉淀。

**7**. 以 $\text{AgCl} \rightarrow [\text{Ag(NH}_3)_2]^+$ 为例：$K = K_{\text{sp}} \times K_{\text{稳}} = 1.8 \times 10^{-10} \times 1.1 \times 10^7 = 2.0 \times 10^{-3}$。该值小于 1，说明需要较高浓度的氨水才能有效溶解 $\text{AgCl}$。其余步骤类似计算。
