> **前置要求**：[[04-课件/学生讲义/原子结构-超级充实版（自学完整）|原子结构]]（基态电子排布，Z≤36）

> **深度边界**：掌握键型分类与电负性判断，能用电负性差区分离子键/共价键/金属键，理解配位键的形成条件和σ-给予/π-反馈机制，能用三步法判断分子极性。Lewis结构写法、形式电荷、共振、VSEPR 详见分子结构基础。

## 一、化学键的分类与本质

### 1.1 三大基本键型

| 键型 | 本质 | 形成方式 | 典型电负性差 |
|:---|:---|:---|:---:|
| **离子键** | 【______】 | 电子转移 → 阴/阳离子 → Coulomb 引力 | ≥ 1.7 |
| **共价键** | 【______】 | 原子轨道重叠，电子共享 | < 1.7 |
| **金属键** | 【______】 | 电子在金属正离子间自由流动 | — |

> **注意**：以上分类是连续谱上的理想端点。电负性差在 1.5-2.0 之间的键（如 HF、LiI）既有离子键成分也有共价键成分。即使是经典离子键 CsF，也有约 8% 的共价成分。

**键型连续谱**：

```
非极性共价键 ← 极性共价键 ← 离子键（← 金属键）
   Cl₂         HCl   HF   LiI   NaCl   MgO    Na
   ΔEN=0     1.0   1.8   1.7   2.3    2.1    —
```

### 1.2 离子键

**形成条件**：金属（电负性小）+ 非金属（电负性大）→ 电子转移 → 阴、阳离子 → 静电吸引。

**影响离子键强度的因素**：

| 因素 | 影响 | 示例比较 |
|:---|:---|:---|
| 离子电荷 ↑ | 键强 ↑ | MgO (Z=±2) > NaCl (Z=±1) |
| 离子半径 ↓ | 键强 ↑ | NaCl > KCl |
| 电子构型 | 18e⁻ > 8e⁻ | AgCl 比 NaCl 更难溶（极化效应）|

**竞赛拓展：Fajans 规则**（判断离子键中共价成分的大小）

阳离子【______】能力越强 → 共价成分越大：

| 因素 | 极化力增强方向 | 实例对比 |
|:---|:---|---:|
| 阳离子电荷 ↑ | 高价 > 低价 | Mg²⁺ > Na⁺ |
| 阳离子半径 ↓ | 小 > 大 | Li⁺ > Na⁺ |
| 阳离子电子构型 | 18e⁻ > 9-17e⁻ > 8e⁻ | Ag⁺ > Fe²⁺ > Na⁺ |
| 阴离子半径 ↑ | 大阴离子更易被极化 | I⁻ > Cl⁻ |

> **Fajans 规则应用**：AgF 易溶而 AgCl/AgBr/AgI 越来越难溶——F⁻ 半径小、难极化 → AgF 离子性强 → 易溶；I⁻ 半径大、易极化 → AgI 共价成分大 → 难溶。

### 1.3 共价键

**形成条件**：非金属之间 → 共用电子对 → 原子轨道重叠 → 电子云密度在核间增大 → 降低体系能量。

**共价键的分类**（按轨道重叠方式）：

| 类型 | 重叠方式 | 电子云对称性 | 强度 | 出现位置 |
|:---|:---|:---|:---:|:---|
| **σ 键** | "头碰头"，沿键轴 | 圆柱形对称 | 最强 | 单键必为 σ；多重键含 1 σ |
| **π 键** | "肩并肩"，垂直键轴 | 键轴节面 | 弱于 σ | 双键含 1π，三键含 2π |
| **δ 键** | "面碰面" | 两个节面 | 最弱 | 过渡金属配合物 |

**键参数**：

| 参数 | 定义 | 意义 |
|:---|:---|---:|
| **键能** $E$ | 断开 1 mol 键所需的能量 | 键越强，键能越大 |
| **键长** $d$ | 两原子核间的平衡距离 | 键越强，键长越短 |
| **键角** $\theta$ | 两键之间的夹角 | 决定分子形状 |

### 1.4 金属键简介

**金属键**：金属正离子沉浸在自由电子"海洋"中，自由电子在整个金属晶体中离域运动。

| 特性 | 金属键的解释 |
|:---|:---|
| 导电性 | 自由电子在外电场下定向移动 |
| 导热性 | 自由电子传递动能 |
| 延展性 | 原子层滑动 → 电子海重新粘合 → 变形不断 |
| 金属光泽 | 自由电子与光相互作用 |

## 二、电负性与键的极性

### 2.1 什么是电负性？

**电负性**（Electronegativity, χ）：原子在分子中【______】的能力。

**三种常用标度**：

| 标度 | 提出者 | 方法 | 范围 | 特点 |
|:---|:---|:---|---:|:---|
| **Pauling 标度** | Pauling (1932) | 基于键能数据 | 0.7-4.0 | 最常用，竞赛标准 |
| **Mulliken 标度** | Mulliken (1934) | 电离能+电子亲和能平均 | 0-4.5 | 理论意义明确 |
| **Allred-Rochow 标度** | Allred & Rochow (1958) | 有效核电荷/共价半径² | 0-4.3 | 更接近物理本质 |

$$ \chi_M = \frac{1}{2}(I + A) \quad (\text{Mulliken}) $$

> **三标度关系**：Mulliken 标度理论意义最明确——I 衡量"留住自己电子"的能力，A 衡量"吸引外来电子"的能力，取平均反映争夺电子的综合能力。Pauling 标度可通过 $\chi_P \approx 0.336(\chi_M - 0.615)$ 近似换算。

### 2.2 电负性周期趋势

**核心规律**：
- 同一周期从左到右增大（F 最大 χ=4.0）
- 同一主族从上到下减小（F > Cl > Br > I）
- 金属电负性 < 2.0，非金属 > 2.0

**常见元素电负性**（Pauling标度，记忆底线）：

| H 2.1 | | | | | | | |
|:---|:---|:---|:---|:---|:---|:---|:---|
| Li 1.0 | Be 1.6 | B 2.0 | C 2.6 | N 3.0 | O 3.4 | F 4.0 |
| Na 0.9 | Mg 1.3 | Al 1.6 | Si 1.9 | P 2.2 | S 2.6 | Cl 3.2 |
| K 0.8 | Ca 1.0 | | | | Br 3.0 | |
| | | | | | I 2.7 | |

### 2.3 影响电负性的因素

1. **杂化状态**：s 成分越高，电负性越大
   - C(sp³) χ=2.6，C(sp²) χ=2.8，C(sp) χ=3.0
   - 解释：乙炔中 C-H 键的酸性强于乙烯和乙烷
2. **氧化态**：高氧化态 → 电负性增大
3. **基团电负性**：取代基可通过诱导效应影响相连原子的电负性

### 2.4 电负性差与键型判断

$$
\Delta\chi = |\chi_A - \chi_B|
$$

| $\Delta\chi$ | 键型 | 离子键成分 | 示例 |
|:---:|:---|:---:|:---|
| 0 | 非极性共价键 | 0% | Cl₂ (0), N₂ (0) |
| 0-0.4 | 弱极性共价键 | ~5% | CH₄ (0.4) |
| 0.4-1.7 | 极性共价键 | 5-50% | HCl (1.0), HF (1.8) |
| ≥ 1.7 | 离子键 | ≥ 50% | NaCl (2.2), MgO (2.1) |

> **重要**：Δχ ≥ 1.7 是经验判据，不是绝对分界线。HF 虽然 Δχ=1.8，但仍是共价分子——因为 H⁺ 和 F⁻ 尺寸不匹配，无法形成稳定离子晶体。

## 三、配位键

### 3.1 定义与表示

**配位键**：共用电子对由【______】的共价键。

**表示方法**：用箭头【______】表示，箭头从提供【______】的原子指向接受电子的原子。

**形成条件**（缺一不可）：
1. 一方有【______】（电子给予体，Lewis 碱）
2. 另一方有【______】（电子接受体，Lewis 酸）

**常见实例**：

| 配位键 | 给予体 | 接受体 | 说明 |
|:---|:---|:---|:---|
| NH₃ + H⁺ → NH₄⁺ | NH₃ 的孤对电子 | H⁺ 的 1s 空轨道 | 经典 Lewis 酸碱反应 |
| CO 中的 C→O | C 提供孤对 | O 接受 | CO 含一个配位键 |
| Fe + CO → Fe(CO)ₓ | CO 的 C 端孤对 | Fe 的空 d 轨道 | 金属羰基配合物 |

> **竞赛考点**：配位键形成后，与普通共价键**没有本质区别**。NH₄⁺ 中四个 N-H 键完全等价。

### 3.2 σ-给予/π-反馈协同机制

CO 与过渡金属形成稳定配合物的能力远超 N₂，为什么？

| 对比项 | CO | N₂ |
|:---|:---|:---|
| 等电子 | 14 电子 | 14 电子 |
| 前端孤对 | **C 端突出**，σ 轨道含高密度孤对 | 两端对称，无单独突出端 |
| σ-给予能力 | 强 | 弱 |
| π* 空轨道 | C 端有低能 π* 空轨道 | π* 轨道较高 |
| π-反馈能力 | 金属 d 电子可反馈至 CO 的 π* | 反馈能力弱 |
| **协同效应** | **σ-给予 + π-反馈 → 协同增强** | 无协同 → 配位能力弱 |

> **课堂口诀**："CO 是两栖配体——C 端孤对往前送（σ-给予），空的 π* 往后收（π-反馈），一送一收比 N₂ 稳得多。"

## 四、分子极性判断 —— 三步法

> **说明**：本节为分子结构基础中 VSEPR + 分子极性的预热。详细的构型判断在分子结构基础中展开。

**三步判断法**：
1. **看键**：是否有极性键（Δχ > 0）？→ 无则非极性分子
2. **看形**：分子构型是否对称？→ 对称则键矩抵消
3. **看矢量和**：所有键偶极矩的矢量和是否为零？→ 零则非极性

**分子极性判断流程**：
```
Step 1: 是否有极性键（Δχ > 0）？
  ├─ 否 → 非极性分子（N₂、Cl₂、O₂）
  └─ 是 → Step 2: 分子构型是否对称？
       ├─ 是 → 键偶极矩矢量和为零 → 非极性分子（CO₂、CCl₄、BF₃）
       └─ 否 → 键偶极矩矢量和不为零 → 极性分子（H₂O、NH₃、SO₂）
```

**经典对比**：

| 分子 | 键的极性 | 空间构型 | 分子极性 | 关键 |
|:---|:---:|:---|:---:|:---|
| CO₂ | 有 | 直线形（对称）| **非极性** | 键矩抵消 |
| SO₂ | 有 | V 形（不对称）| **极性** | 孤对破坏对称 |
| BF₃ | 有 | 平面三角形（对称）| **非极性** | 键矩抵消 |
| NH₃ | 有 | 三角锥（不对称）| **极性** | 孤对破坏对称 |
| CH₄ | 弱 | 正四面体（对称）| **非极性** | 完美对称 |
| CH₃Cl | 有 | 四面体（不对称取代）| **极性** | 取代破坏对称 |

> **口诀**："先看键、再看形、最后矢量和——对称抵消出非极，不对称就是极"

## 五、本节速查

| 概念 | 核心要点 | 常见陷阱 |
|:---|:---|:---|
| **离子键** | 金属+非金属，电子转移，无方向性 | NaCl 气态是共价分子 |
| **共价键** | 非金属间，共用电子对，σ/π/δ 分类 | 多重键键能 ≠ 单键键能简单加和 |
| **配位键** | 孤对电子 + 空轨道，形成后与普通键等价 | NH₄⁺ 中四个 N-H 等价 |
| **Fajans 规则** | 极化力越强 → 共价成分越大 | 离子化合物也有共价成分（CsF 8%）|
| **电负性** | Pauling 标度，F 最大(4.0) | 电负性 ≠ 电子亲和能 ≠ 电离能 |
| **键型判断** | Δχ < 1.7 共价，≥ 1.7 离子 | HF(Δχ=1.8) 是共价分子 |
| **分子极性** | 先看键→再看构型→最后矢量和 | CO₂ 有极性键但非极性 |

## 六、练习题

### 6.1 基础巩固

1. 判断以下化学键的离子/共价属性：KBr、HI、MgO、ClF₃、LiH
2. 指出以下分子中哪些有极性键但分子本身是非极性的：CO₂、H₂O、CH₄、NH₃、BF₃、CCl₄、SO₂
3. 什么是配位键？举两个配位键的实例。

### 6.2 提高训练

4. 已知 BF₃ 中 B-F 键有较强的极性（Δχ ≈ 2.0），但 BF₃ 的沸点很低（−100°C），而 NaF（Δχ ≈ 3.0）的熔点高达 993°C。解释为什么同样是强极性的 B-F 键，BF₃ 和 NaF 的性质差异如此之大。
5. 用 σ-给予/π-反馈 机制解释：为什么 CO 可以与过渡金属形成稳定的配合物（如 Fe(CO)₅），而 N₂ 的配位能力远弱于 CO？

### 6.3 挑战题

6. 已知 NO₂⁺ 与 CO₂ 等电子，请写出 NO₂⁺ 的 Lewis 结构，计算形式电荷，推断其几何构型。（提示：详见分子结构基础 §1 Lewis 结构）
7. **综合推理**：A、B、C、D 为四种短周期元素。A 的基态原子电子排布式为 1s¹；B 的基态原子最外层有 4 个电子；C 的电负性仅次于 F；D 的单质在常温下为黄绿色气体。
   (1) 写出 A、B、C、D 的元素符号和各原子的电子式。
   (2) A 与 B、C、D 分别形成的化合物中，哪个是离子化合物？哪个是极性分子？哪个是非极性分子？

## 七、答案速查

**6.1 基础巩固**

1. **KBr**：离子键（Δχ≈2.3）；**HI**：极性共价键（Δχ≈0.5）；**MgO**：离子键（Δχ≈2.1）；**ClF₃**：极性共价键（Δχ≈0.8）；**LiH**：离子键（Δχ≈1.2）。

2. **有极性键但分子非极性**：CO₂（直线形）、CH₄（正四面体）、BF₃（平面三角形）、CCl₄（正四面体）。H₂O、NH₃、SO₂ 是极性分子。

3. 配位键是共用电子对由一方单独提供的共价键。例：NH₄⁺ 中 N 提供孤对、H⁺ 提供空轨道；CO 中 C 提供孤对、O 接受。

**6.2 提高训练**

4. BF₃ 是分子晶体（分子间仅范德华力）→ 沸点低；NaF 是离子晶体（Na⁺ 与 F⁻ 通过离子键连接成三维网状结构）→ 熔点高。**键的极性**描述的是分子内的成键方式，**熔沸点**反映的是分子间/晶体中的作用力——二者不直接对应。

5. CO 的 C 端有突出孤对（σ-给予体）+ 低能 π* 空轨道（π-反馈受体）→ 一送一收协同增强。N₂ 两端对称，无突出孤对端，π* 轨道较高 → 配位能力弱。

**6.3 挑战题**

6. NO₂⁺ 总价电子 = 5+12−1=16 e⁻ → 与 CO₂ 等电子。Lewis 结构 O=N⁺=O，形式电荷 N(0)，O(0)。VSEPR：N 无孤对，2 个σ键 → AX₂ → **直线形**。

7. (1) A=H、B=C、C=F、D=Cl。(2) A 与 B(CH₄)——非极性分子；A 与 C(HF)——极性分子（含氢键）；A 与 D(HCl)——极性分子。均为共价化合物。
