## 一、电子式 —— 化学键的入门语言

### 1.1 什么是电子式？

**【电子式】——【______】**：用元素符号表示原子核和内层电子，用 **·** 或 **×** 表示【______】，用 **—** 表示【______】。

| 原子 | 电子式 | 价电子数 | 说明 |
|:---|:---:|:---:|:---|
| H | H· | 1 | 最简 |
| C | ·C· | 4 | 四个方向各一个电子 |
| N | ·N· | 5 | 三个单电子 + 一对孤对 |
| O | :O· | 6 | 两对孤对 + 两个单电子 |
| F | :F: | 7 | 三对孤对 + 一个单电子 |
| Na | Na· | 1 | 金属原子，一个价电子 |
| Cl | :Cl: | 7 | 三对孤对 + 一个单电子 |

### 1.2 分子的电子式书写规则

**规则**（高中→竞赛过渡版）：
1. 先算总价电子数（各原子价电子之和 ± 离子电荷）
2. 电负性小的原子居中
3. 原子间用一对电子（—）连接
4. 剩余电子优先满足端基原子八隅体（H 满足 2 电子）
5. 若中心原子未满足八隅体，将端基孤对改为多重键

**常见分子电子式示例**：

| 分子 | 电子式（文本表示） | 结构式 | 说明 |
|:---|:---|:---:|:---|
| H₂ | H—H | H—H | 单键，2 电子 |
| O₂ | O=O | O=O | 双键，4 电子（实际为三键+单电子，详见 MO 部分）|
| N₂ | N≡N | N≡N | 三键，6 电子 |
| HCl | H—Cl | H—Cl | 极性共价键，Cl 有三对孤对 |
| H₂O | H—O—H | H—O—H | 中心 O 有两对孤对（V 形结构）|
| CO₂ | O=C=O | O=C=O | 直线形，两个双键 |
| NH₃ | H—N—H | H—N—H | 三角锥形，N 有一对孤对 |

### 1.3 竞赛中的电子式要求

- **高中水平**：能写出 H₂O、CO₂、CH₄、NH₃、NaCl、MgCl₂ 等的电子式
- **竞赛入门**：能写出 N₂、O₂（含单电子）、CO（含配位键）、NO₃⁻、SO₄²⁻ 等的电子式
- **竞赛进阶**：能写出 XeF₂、ClF₃、IF₄⁻ 等超八隅体物种的电子式

**与 Lewis 结构式的区别**：电子式是 Lewis 结构式的特殊形式，通常特指高中化学中用于描述共用电子对的简化表示。竞赛中统一使用 Lewis 结构式（含形式电荷计算）。

## 二、化学键的分类与本质

### 2.1 三大基本键型

| 键型 | 本质 | 形成方式 | 典型电负性差 |
|:---|:---|:---|:---:|
| **离子键** | 【______】 | 电子转移 → 阴/阳离子 | ≥ 1.7 |
| **共价键** | 【______】 | 电子共享 | < 1.7 |
| **金属键** | 【______】 | 电子在金属正离子间自由流动 | — |

> **注意**：以上分类不是绝对的。电负性差在 1.5-2.0 之间的键（如 HF、HCl）既有离子键成分也有共价键成分——用"极性共价键"描述更准确。

### 2.2 离子键

**形成条件**：金属（电负性小）+ 非金属（电负性大）→ 电子转移 → 阴、阳离子 → 静电吸引。

**离子化合物的特征**：
- 常温固体（离子晶体）
- 熔点高（NaCl 801°C，MgO 2852°C）
- 熔融状态或水溶液导电
- 硬度大、脆性

**影响离子键强度的因素**（决定熔点/硬度）：

| 因素 | 影响 | 示例比较 |
|:---|:---|:---|
| 离子电荷 ↑ | 键强 ↑ | MgO (Z=±2) > NaCl (Z=±1) |
| 离子半径 ↓ | 键强 ↑ | NaCl (r=102+181pm) > KCl (r=138+181pm) |
| 电子构型 | 18e⁻ 壳层 > 8e⁻ 壳层 | AgCl 比 NaCl 更难溶（极化效应）|

### 2.3 共价键

**形成条件**：非金属之间 → 共用电子对。

**共价键的类型**：

| 分类标准 | 类型 | 特点 | 实例 |
|:---|:---|:---|:---|
| 共用电子对偏移 | 非极性共价键 | ΔEN=0，电子对均分 | H₂、Cl₂、N₂ |
| | 极性共价键 | ΔEN>0，电子对偏移 | HCl、H₂O、NH₃ |
| 共用电子对数 | 单键 | 1 对电子 | H—H、C—H |
| | 双键 | 2 对电子 | C=O、C=C |
| | 三键 | 3 对电子 | N≡N、C≡O |

**键参数**：

| 参数 | 定义 | 意义 |
|:---|:---|:---|
| **键能** $E$ | 断开 1 mol 键所需的能量（kJ/mol）| 键越强，键能越大 |
| **键长** $d$ | 两原子核间的平衡距离（pm）| 键越强，键长越短 |
| **键角** $\theta$ | 两键之间的夹角 | 决定分子形状 |

**键长与键能的关系**（以 C-C 键为例）：
- C—C 单键：154 pm，347 kJ/mol
- C=C 双键：134 pm，612 kJ/mol（≠ 2 × 347）
- C≡C 三键：120 pm，838 kJ/mol（≠ 3 × 347）

> 多重键的键能**小于**相同数目的单键键能之和，这是因为 π 键重叠程度小于 σ 键，键能较弱。

### 2.4 配位键（Coordinate Covalent Bond）

**定义**：共用电子对由【______】的共价键。

**表示方法**：用箭头【______】表示，箭头从提供【______】的原子指向接受电子的原子。

**形成条件**（缺一不可）：
1. 一方有【______】（电子给予体，Lewis 碱）
2. 另一方有【______】（电子接受体，Lewis 酸）

**常见实例**：

| 配位键 | 给予体 | 接受体 | 说明 |
|:---|:---|:---|:---|
| NH₃ + H⁺ → NH₄⁺ | NH₃ 的孤对电子 | H⁺ 的 1s 空轨道 | 经典 Lewis 酸碱反应 |
| CO 中的 C→O | C 提供孤对 | O 接受 | CO 的 Lewis 结构：C≡O（含一个配位键）|
| H₂O + H⁺ → H₃O⁺ | H₂O 的孤对电子 | H⁺ 的 1s 空轨道 | 水合氢离子 |
| Fe + CO → Fe(CO)ₓ | CO 的 C 端孤对 | Fe 的空 d 轨道 | 金属羰基配合物 |

> **竞赛考点**：配位键形成后，与普通共价键**没有本质区别**。NH₄⁺ 中四个 N-H 键完全等价，不能区分哪个是配位键。

## 三、电负性与键的极性

### 3.1 电负性标度

**电负性**（Electronegativity, EN）：原子在分子中【______】的能力。最常用的是【______】标度（1932）。

**核心规律**：
- 同一周期从左到右增大（F 最大 EN=4.0）
- 同一主族从上到下减小（F > Cl > Br > I）
- 金属电负性 < 2.0，非金属 > 2.0

**常见元素电负性**（Pauling标度，记忆底线）：

| H 2.1 | | | | | | | |
|:---|:---|:---|:---|:---|:---|:---|:---|
| Li 1.0 | Be 1.6 | B 2.0 | C 2.6 | N 3.0 | O 3.4 | F 4.0 |
| Na 0.9 | Mg 1.3 | Al 1.6 | Si 1.9 | P 2.2 | S 2.6 | Cl 3.2 |
| K 0.8 | Ca 1.0 | | | | Br 3.0 | |
| | | | | | I 2.7 | |

### 3.2 电负性差与键型判断

$$
\Delta\mathrm{EN} = |\mathrm{EN_A} - \mathrm{EN_B}|
$$

| $\Delta$EN | 键型 | 离子键成分 | 示例 |
|:---:|:---|:---:|:---|
| 0 | 非极性共价键 | 0% | Cl₂ (0), N₂ (0) |
| 0-0.4 | 弱极性共价键 | ~5% | CH₄ (0.4) |
| 0.4-1.7 | 极性共价键 | 5-50% | HCl (1.0, ~20%), HF (1.8, ~60%) |
| ≥ 【______】 | 离子键 | ≥ 50% | NaCl (2.2), MgO (2.1) |

> **注意**：$\Delta$EN ≥ 1.7 是经验判据，不是绝对分界线。实际键型还取决于成键原子的具体电子构型和极化能力。例如 HF 虽然 $\Delta$EN=1.8，但仍是共价分子而非离子化合物——因为 H⁺ 和 F⁻ 的尺寸不匹配，无法形成离子晶体。

### 3.3 键的极性与分子极性

**键的极性**（由 $\Delta$EN 决定）是**分子极性**的必要非充分条件。

- 双原子分子：键有极性 → 分子有极性（HCl、CO）
- 多原子分子：取决于分子对称性
  - 对称分子：键有极性但分子无极性（CO₂、CCl₄、BF₃）
  - 不对称分子：键有极性且分子有极性（H₂O、NH₃、CHCl₃）

**判断流程**：
```
是否有极性键（ΔEN > 0）？
  ├─ 否 → 非极性分子（N₂、Cl₂、O₂）
  └─ 是 → 分子结构是否对称？
       ├─ 是 → 非极性分子（CO₂、CCl₄、CH₄、BF₃）
       └─ 否 → 极性分子（H₂O、NH₃、CHCl₃、SO₂、H₂S）
```

## 五、典型例题

### 例 1 [电子式书写基础] ⭐

**题目**：写出以下物质的电子式：NaCl、MgCl₂、H₂O、NH₃、CO₂、N₂。

**解答**：

| 物质 | 电子式（文本表示） | 说明 |
|:---|:---|:---|
| NaCl | Na⁺ [:Cl:]⁻ | 离子键，Na 失 1e⁻，Cl 得 1e⁻ |
| MgCl₂ | [Cl:]⁻ Mg²⁺ [:Cl:]⁻ | 两个 Cl⁻ 围绕 Mg²⁺ |
| H₂O | H:O:H（O 上两对孤对）| 共价键，O 满足八隅体 |
| NH₃ | H:N:H（N 上一对孤对）| 共价键，三角锥形 |
| CO₂ | O::C::O（双键）| 两个 C=O 双键，直线形 |
| N₂ | :N≡N:（三键+两对孤对）| N≡N 三键，直线形 |

---

### 例 2 [电负性与键型判断] ⭐⭐

**题目**：根据电负性数据判断下列化学键的极性大小排序：Cl₂、HCl、NaCl、HF、MgO。

**解答**：

计算 $\Delta$EN：
- Cl₂：$\Delta$EN = 0 → 非极性共价键
- HCl：$\Delta$EN = |3.2-2.1| = 1.1 → 极性共价键
- NaCl：$\Delta$EN = |3.2-0.9| = 2.3 → 离子键
- HF：$\Delta$EN = |4.0-2.1| = 1.8 → 极性共价键（偏向离子键）
- MgO：$\Delta$EN = |3.4-1.3| = 2.1 → 离子键

**极性排序**：Cl₂ < HCl < HF < MgO < NaCl

**注意**：$\Delta$EN 越大，键的极性/离子性越强，但不等于分子极性。例如 NaCl 的 $\Delta$EN 大于 HF，但 NaCl 是离子晶体（固态无分子），HF 是极性分子。

---

### 例 3 [配位键识别] ⭐⭐

**题目**：判断以下物质中是否存在配位键。若存在，指出电子给予体和接受体。

(1) NH₄Cl (2) H₃O⁺ (3) CO (4) CH₄

**解答**：

| 物质 | 配位键 | 给予体 | 接受体 | 说明 |
|:---|:---:|:---|:---|:---|
| NH₄Cl | ✅ | NH₃（孤对）| H⁺（1s 空轨道）| NH₃ 与 H⁺ 形成配位键后，4 个 N-H 完全等价 |
| H₃O⁺ | ✅ | H₂O（孤对）| H⁺（1s 空轨道）| H₂O 上的孤对与 H⁺ 结合 |
| CO | ✅ | C 原子（孤对）| O 原子 | CO 的 Lewis 结构 C≡O 中含一个配位键（O 接收）|
| CH₄ | ❌ | — | — | 四个 C-H 均为普通共价键，无孤对参与 |

## 六、本节速查（课堂填空答案汇总）

| 概念 | 核心要点 | 常见陷阱 |
|:---|:---|:---|
| **电子式** | 用 ·/× 表示价电子，— 表示共用对 | 忘记标孤对电子 |
| **离子键** | 金属+非金属，电子转移，静电吸引 | NaCl 气态是共价分子（不是离子键）|
| **共价键** | 非金属间，共用电子对 | 多重键键能 ≠ 单键键能简单加和 |
| **配位键** | 孤对电子 + 空轨道，形成后与普通键等价 | NH₄⁺ 中四个 N-H 等价，无"配位键"和"共价键"之分 |
| **电负性** | Pauling 标度，F 最大(4.0) | 电负性 ≠ 电子亲和能 ≠ 电离能 |
| **键型判断** | $\Delta$EN < 1.7 共价，≥ 1.7 离子 | HF($\Delta$EN=1.8) 是共价分子，不是离子化合物 |
| **分子极性** | 对称结构 ≠ 极性分子 | CO₂ 直线形对称→非极性，H₂O V 形→极性 |

   (1) 写出 A、B、C、D 的元素符号和电子式。

## 八、答案速查

**6.1 基础巩固**

1. **提示**：H₂S 与 H₂O 类似（S 有两对孤对）；PH₃ 与 NH₃ 类似（P 有一对孤对）；CCl₄ 四面体结构；Na₂O 为 Na⁺ 和 O²⁻ 的离子排列；Mg₃N₂ 为 Mg²⁺ 和 N³⁻ 的离子排列。

2. **KBr**：离子键（$\Delta$EN≈2.3）；**HI**：极性共价键（$\Delta$EN≈0.5）；**MgO**：离子键（$\Delta$EN≈2.1）；**ClF₃**：极性共价键（$\Delta$EN≈0.8）；**LiH**：离子键（$\Delta$EN≈1.2，但 H⁻ 离子确已存在）。

3. **有极性键但分子非极性**：CO₂（直线形）、CH₄（正四面体）、BF₃（平面三角形）、CCl₄（正四面体）。H₂O 和 NH₃ 是极性分子。

**6.2 提高训练**

4. **提示**：BF₃ 是分子晶体，分子间仅存在范德华力（分子量小），所以沸点低；NaF 是离子晶体，Na⁺ 与 F⁻ 通过离子键连接成三维网状结构，需要克服大量离子键才能熔融 → 熔点高。BF₃ 中 B-F 键虽然是极性键，但这描述的是**分子内的**成键方式，而决定沸点和熔点的是**分子间的**作用力。

5. **提示**：CO 与 N₂ 等电子（均为 14 电子），具有相似的三键结构。但 CO 的 C 端有一对孤对电子占据 σ 轨道（主要是 C 的 2s 成分），且 C 端有空的 π* 轨道。C 端孤对 σ 电子可以给予过渡金属的空 d 轨道（σ 给予），同时金属的 d 电子可以反馈到 C 端的 π* 空轨道（π 反馈）→ **σ-给予 + π-反馈** 协同增强 → 形成稳定的配位键。N₂ 两端对称，没有单独突出的孤对电子端，给予能力不如 CO。

6. **提示**：X 为 S（Z=16），H₂S。中心 S 有 6 个价电子，两个 H 各贡献 1 个 → 总价电子数 8 → 形成 2 个 S-H 单键 → 剩余两对孤对 → VSEPR：AX₂E₂ → V 形 → S 的杂化方式为 sp³（不等性）→ 键角接近 109.5° 但被孤对压缩至 ~92°（实际）。

**6.3 挑战题**

7. **提示**：(1) A=H(·)、B=C(:C:)、C=F(:F:)、D=Cl(:Cl:)。(2) A 与 D(HCl)是极性分子；A 与 C(HF)是极性分子（且含氢键）；B 与 A(CH₄)是非极性分子；A 与 B/C/D 均为共价化合物。（离子化合物需要典型金属+非金属组合）。(3) B-D (C-Cl) 键中，Cl 电负性更大 → 电子对偏向 Cl；CCl₄ 为正四面体结构 → 键的极性被对称性抵消 → 非极性分子。
