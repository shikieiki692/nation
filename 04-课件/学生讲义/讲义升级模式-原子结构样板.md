# 讲义升级模式 · 原子结构样板

> **用途**：回答“如何把一份普通讲义稳定升级到原子结构级别”，而不是要求所有讲义机械照抄原子结构的长度。
> **核心原则**：复制的是升级方法，不是表面篇幅。

---

## 一、为什么原子结构适合作为样板

`原子结构-超级充实版（自学完整）` 之所以能当母版，不是因为它字多，而是因为它同时做对了 8 件事：

| 维度 | 原子结构样板的成熟信号 |
|:---|:---|
| 讲义定位 | 顶部就写清楚“这讲讲什么、接前面什么、后面去哪” |
| 概念链 | 正文不是散点堆知识，而是按“原子结构 -> 周期趋势 -> 应用/辨析”推进 |
| 动机段 | 不是上来就定义，而是先解释“为什么要学这个” |
| 图文密度 | 复杂概念基本都有图、表或读图支架 |
| 嵌入推导 | 关键公式和规则不是只给结论，而是给出可跟走的推理 |
| 例题链 | 例题覆盖从基础到综合，且答案和提示放在后面统一回收 |
| 易错点 | 易混概念和判断边界被显式写出来，而不是只靠老师口头补 |
| 前后衔接 | 不只告诉学生“本讲学什么”，还告诉学生“学完以后下一讲怎么接” |

---

## 二、升级不是一步到位，而是三层推进

| 层级 | 目标 | 典型动作 | 完成标志 |
|:---|:---|:---|:---|
| **S0 基线规范** | 先让旧讲义变得可维护 | 清格式、清图片路径、清旧 callout、清星级 | 讲义能稳定过预检和 PDF 编译 |
| **S1 标准讲义** | 让讲义成为一份成熟可用的课堂材料 | 补学习目标、补主链结构、补例题练习答案、补前后衔接 | 拿着这份讲义可以稳定上完一讲 |
| **S2 超级充实版** | 让讲义成为“母版” | 加图表、加读图、加嵌入推导、加易错辨析、加更强的总结速查 | 课堂可用，自学也能顺着学下来 |

> **判断顺序**：先问“有没有达到 S0”，再问“能不能算 S1”，最后才问“值不值得上 S2”。

---

## 三、从旧讲义升级时，先做这 4 个收口动作

### 3.1 先清旧口径

- 删除 `⭐ / ★`
- 删除 emoji callout
- 删除非 `media/` 的正式图片路径
- 删除把解释塞进图注的写法

### 3.2 再清结构噪声

- 把零散段落重新并入明确的 `## / ### / ####` 层级
- 把习题答案统一挪到后面
- 去掉“前面讲一点、后面又重复讲一遍”的双写结构
- 去掉重复图片和低信息量占位图

### 3.3 补顶部课程信息

至少补齐：

- `对应专题`
- `对应备课大纲`
- `本讲定位`
- `上节衔接`
- `下节衔接`
- `前置要求`
- `深度边界`

### 3.4 最后再加内容

不要一开始就盲目加字。  
顺序应当是：**先让结构稳定，再让内容变厚。**

---

## 四、S1 标准讲义的升级配方

把一份普通讲义升到 S1，至少补齐下面这 7 项：

1. **学习目标完整**  
   至少能回答“这讲学完学生能做什么”。

2. **主线清楚**  
   正文应当能看出“问题 -> 概念 -> 方法 -> 例题 -> 练习 -> 答案”的基本链条。

3. **例题与练习分工清楚**  
   例题负责教做法，练习负责检验掌握，答案统一放后面。

4. **前后衔接明确**  
   学生看完能知道它接的是哪一讲、后面会通到哪里。

5. **深度边界明确**  
   第一轮讲到哪里为止，哪些内容以后再讲，必须说清。

6. **图与表不是装饰**  
   每张图、每个表都要承担教学功能。

7. **统一写法**  
   callout、图片、图注、数学模式全部对齐现行规范。

---

## 五、S2 超级充实版的升级配方

在 S1 基础上，把一份讲义升级到 S2，重点看下面这 8 项：

| 维度 | 升级动作 | 原子结构样板对应表现 |
|:---|:---|:---|
| 动机层 | 每个大块都先解释“为什么学这个” | 不是直接扔量子数和排布规则 |
| 结构层 | 每节内部有小闭环 | 为什么 -> 是什么 -> 怎么用 -> 注意什么 |
| 图形层 | 复杂概念图示化 | 能级图、趋势图、结构图、对比图 |
| 读图层 | 图后立刻给读图任务 | 让图变成训练，不只是插图 |
| 推导层 | 关键规则给出可跟走的过程 | 不是只写最终结论 |
| 辨析层 | 易混概念专门拉出来处理 | 特例、反常点、边界条件都会显式提醒 |
| 例题层 | 例题链覆盖“基础 -> 进阶 -> 综合” | 不让最后只剩一两道象征性题目 |
| 收束层 | 有可回看的总结区和答案区 | 方便学生课后独立复盘 |

## 六、不要机械复刻原子结构，要按专题类型变形

| 专题类型 | 应该复制原子结构的什么 | 不该硬抄什么 |
|:---|:---|:---|
| **结构型** | 图文密度、对比表、读图支架、空间关系解释 | 不必强行塞大量历史叙述 |
| **物化型** | 变量意义、条件判断、推导过程、边界说明 | 不必为了“充实”堆太多重复定义 |
| **专项工具型** | 流程、判据、错误分类、步骤演示 | 不必写成百科式长文 |
| **桥梁收束型** | 前后串联、知识网络、题型归并 | 不必把每个基础概念从头重讲一遍 |

### 6.1 一个简单判断

- 如果这讲的核心难点是“看不见”，就优先加图。
- 如果这讲的核心难点是“不会判条件”，就优先加流程和边界。
- 如果这讲的核心难点是“会算但不会选模型”，就优先加辨析和变量表。

## 七、实操升级流程：建议按 3 轮做

### 第 1 轮：规范化

目标：`旧稿 -> S0`

- 清格式
- 清旧图片引用
- 清旧 callout
- 清旧星级
- 修答案位置
- 跑预检

### 第 2 轮：标准化

目标：`S0 -> S1`

- 补学习目标
- 补本讲定位和前后衔接
- 重组正文主链
- 补例题、练习、答案
- 修掉明显重复与断层

### 第 3 轮：超级充实化

目标：`S1 -> S2`

- 补图和对比表
- 补嵌入式推导 / 计算
- 补读图提示与理解方法论
- 补更强的总结速查
- 补跨讲义桥接和 deferred topics

---

## 八、S2 完成态的检查清单

- [ ] 顶部已经写清本讲定位、上节衔接、下节衔接
- [ ] 学习目标能覆盖本讲核心能力
- [ ] 正文主链清楚，没有大段重复
- [ ] 关键抽象概念已有图或表支撑
- [ ] callout 统一为 `> **标签**：内容`
- [ ] 例题链和练习链完整
- [ ] 习题采用三级分类：🌱基础巩固（≥8题）/ 🔥竞赛入门（≥5题）/ 🏆真题挑战（≥3题），合计≥16题
- [ ] 习题含子话题分组（`**加粗标题**`），全局连续编号
- [ ] 答案独立成节（`## 参考答案与提示`），含完整解题过程
- [ ] 答案统一放后面
- [ ] 深度边界写清楚
- [ ] 延伸阅读只保留高价值入口
- [ ] 跑过 PDF 预检和编译回归

---

## 九、下一批升级时的建议顺序

### P1：优先做成下一组 S2 母版

- 热力学初步
- 化学平衡
- 电化学
- 酸碱理论
- 溶液与相图

### P2：先稳定成高质量 S1，再视价值上 S2

- 化学动力学初步
- 沉淀溶解平衡
- 定量化学分析入门
- 有机分子结构初探
- 气体与溶液

### P3：专项和桥梁讲义

- 方程式书写专项
- 物化综合计算
- 真题模拟拆解
