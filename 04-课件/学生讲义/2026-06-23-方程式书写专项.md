# 方程式书写专项（基础班）

> **本讲义定位**：第一轮基础班专用，聚焦氧化还原方程式的离子-电子法（半反应法）配平。
> **适用**：高一竞赛入门 | 第一轮基础班
> **对应备课大纲**：[[04-课件/备课大纲/2026-06-02-方程式书写专项-基础班]]
> **前置要求**：掌握 [[氧化态]] 判定、基本 [[氧化还原反应方程式配平|氧化还原]] 与 [[电化学]] 概念
> **深度边界**：掌握离子-电子法配平氧化还原方程式；掌握介质判断；掌握非氧化还原反应观察法配平；掌握离子方程式书写规则。**不展开**：复杂多元素同时变价（As₂S₃型）、有机氧化还原、电化学系统训练。

## 学习目标

- 能熟练判定元素的氧化态，准确判断反应介质（酸/碱/中性）
- 掌握离子-电子法（半反应法）配平氧化还原方程式的五步骤
- 熟记常见氧化剂在不同介质中的还原产物
- 能独立完成常见氧化还原反应的方程式配平
- 建立"介质优先"的配平思维习惯

## 一、氧化态判定与介质判断

### 1.1 氧化态判定规则（复习）

氧化态（氧化数）是配平的基础。判定规则如下：

| 规则 | 内容 | 例外 |
|:---|:---|:---|
| 单质 | 元素氧化态为 0 | — |
| 离子化合物 | 氧化态 = 离子电荷 | — |
| 共价化合物 | 电负性大的原子得电子（负氧化态） | — |
| H | 通常为 +1 | 金属氢化物中为 -1（如 NaH、CaH₂） |
| O | 通常为 -2 | 过氧化物中为 -1（H₂O₂），超氧化物中为 -1/2（KO₂），OF₂ 中为 +2 |
| 分子总电荷 | 各原子氧化态代数和 = 分子总电荷 | — |

> **应用示例**：Cr₂O₇²⁻ 中 O 为 -2，总电荷 -2，设 Cr 氧化态为 x，则 2x + 7×(-2) = -2 → x = +6，即 Cr 为 +6 价。

### 1.2 介质判断原则

介质决定了产物的形态，是配平的第一步。

| 判断依据 | 酸性介质 | 碱性介质 | 中性介质 |
|:---|:---|:---|:---|
| 题目明确说明 | "酸性条件"、"稀 H₂SO₄" | "碱性条件"、"NaOH" | "中性"、"水溶液" |
| 反应物特征 | 含 H⁺、强酸 | 含 OH⁻、强碱 | 无强酸强碱 |
| 产物提示 | 生成 CO₂（碳酸盐分解） | 生成 CO₃²⁻ | 视情况而定 |

> **核心纪律**：**碱性条件绝不允许出现游离 H⁺，酸性条件绝不允许出现游离 OH⁻**

## 二、离子-电子法（半反应法）配平

### 2.1 配平总口诀

```
介质定 → 产物定 → 原子平 → 电荷平
```

### 2.2 五步骤详解

以 **MnO₄⁻ 在酸性介质中还原为 Mn²⁺** 为例：

| 步骤 | 操作 | 示例 |
|:---:|:---|:---|
| ① 写出骨架 | 写出氧化/还原半反应的骨架形式 | MnO₄⁻ → Mn²⁺ |
| ② 配非 H/O 原子 | 配平除 H、O 外的原子 | 已平（1 个 Mn） |
| ③ 配 O（加 H₂O） | 在缺 O 的一侧加 H₂O | MnO₄⁻ → Mn²⁺ + **4H₂O** |
| ④ 配 H（加 H⁺ 或 OH⁻） | 酸性加 H⁺，碱性加 OH⁻ | MnO₄⁻ + **8H⁺** → Mn²⁺ + 4H₂O |
| ⑤ 配电荷（加 e⁻） | 用电子平衡两侧电荷 | MnO₄⁻ + 8H⁺ + **5e⁻** → Mn²⁺ + 4H₂O |

**完整半反应**：

还原半反应：$\text{MnO}_4^- + 8\text{H}^+ + 5\text{e}^- \rightarrow \text{Mn}^{2+} + 4\text{H}_2\text{O}$
氧化半反应：$\text{SO}_3^{2-} + \text{H}_2\text{O} \rightarrow \text{SO}_4^{2-} + 2\text{H}^+ + 2\text{e}^-$

**合并**：最小公倍数法（10e⁻），两式分别乘以 2 和 5，相加消去电子：

$$2\text{MnO}_4^- + 5\text{SO}_3^{2-} + 6\text{H}^+ \rightarrow 2\text{Mn}^{2+} + 5\text{SO}_4^{2-} + 3\text{H}_2\text{O}$$

### 2.3 碱性介质处理技巧

> **技巧**：先按酸性条件配平，再两边加等量 OH⁻ 中和 H⁺，合并 H⁺ + OH⁻ → H₂O，最后化简。

**酸性配平结果**：$\text{MnO}_4^- + 8\text{H}^+ + 5\text{e}^- \rightarrow \text{Mn}^{2+} + 4\text{H}_2\text{O}$

**转为碱性**：两边加 8 个 OH⁻：
$\text{MnO}_4^- + 8\text{H}^+ + 8\text{OH}^- + 5\text{e}^- \rightarrow \text{Mn}^{2+} + 4\text{H}_2\text{O} + 8\text{OH}^-$

合并 H⁺ + OH⁻ → H₂O：
$\text{MnO}_4^- + 4\text{H}_2\text{O} + 5\text{e}^- \rightarrow \text{Mn}^{2+} + 8\text{OH}^-$

> **验证**：碱性半反应中不含游离 H⁺ ✓

## 三、常见氧化剂在不同介质中的还原产物

### 3.1 还原产物速查表

| 氧化剂 | 酸性介质 | 中性/弱碱性 | 强碱性介质 |
|:---|:---|:---|:---|
| KMnO₄ | **Mn²⁺**（无色） | **MnO₂↓**（棕黑） | **MnO₄²⁻**（绿色） |
| K₂Cr₂O₇ | **Cr³⁺**（绿色） | — | — |
| HNO₃（浓） | **NO₂** | — | — |
| HNO₃（稀） | **NO** | — | — |
| H₂SO₄（浓） | **SO₂** | — | — |
| H₂O₂ | **H₂O** | H₂O（或 O₂↑） | — |
| Cl₂ | Cl⁻ | Cl⁻ / ClO⁻（歧化） | Cl⁻ / ClO⁻（冷）或 ClO₃⁻（热） |

### 3.2 记忆要点

- **MnO₄⁻ 的降价规律**：酸性降 5 价（Mn⁷⁺→Mn²⁺），中性降 3 价（Mn⁷⁺→Mn⁴⁺），碱性降 1 价（Mn⁷⁺→Mn⁶⁺）
- **Cr₂O₇²⁻**：只在酸性条件下作氧化剂，每个 Cr 得 3e⁻，**整体得 6e⁻**（含 2 个 Cr）
- **Cl₂ 歧化**：冷稀碱 → Cl⁻ + ClO⁻；热浓碱 → Cl⁻ + ClO₃⁻（温度影响产物）

### 3.3 常见离子在溶液中的颜色

| 离子 | 颜色 | 离子 | 颜色 |
|:---|:---:|:---|:---:|
| MnO₄⁻ | 紫红 | MnO₄²⁻ | 绿 |
| Mn²⁺ | 浅粉（稀时无色） | Cr₂O₇²⁻ | 橙红 |
| Cr³⁺ | 绿/紫 | Fe²⁺ | 浅绿 |
| Fe³⁺ | 淡黄 | Cu²⁺ | 蓝 |

## 四、典型例题

### 例 1 — KMnO₄ 在不同介质中氧化 Na₂SO₃ ⭐⭐

**题目**：分别写出 KMnO₄ 在酸性、中性、碱性介质中氧化 Na₂SO₃ 的离子方程式。

**思路分析**：先判断介质 → 确定 MnO₄⁻ 的还原产物 → 用离子-电子法配平。

**完整解答**：

**（1）酸性介质**（MnO₄⁻ → Mn²⁺）：

还原半反应：$\text{MnO}_4^- + 8\text{H}^+ + 5\text{e}^- \rightarrow \text{Mn}^{2+} + 4\text{H}_2\text{O}$（×2）
氧化半反应：$\text{SO}_3^{2-} + \text{H}_2\text{O} \rightarrow \text{SO}_4^{2-} + 2\text{H}^+ + 2\text{e}^-$（×5）

合并：
$$2\text{MnO}_4^- + 5\text{SO}_3^{2-} + 6\text{H}^+ \rightarrow 2\text{Mn}^{2+} + 5\text{SO}_4^{2-} + 3\text{H}_2\text{O}$$

**（2）中性介质**（MnO₄⁻ → MnO₂）：

$$2\text{MnO}_4^- + 3\text{SO}_3^{2-} + \text{H}_2\text{O} \rightarrow 2\text{MnO}_2\downarrow + 3\text{SO}_4^{2-} + 2\text{OH}^-$$

**（3）碱性介质**（MnO₄⁻ → MnO₄²⁻）：

$$2\text{MnO}_4^- + \text{SO}_3^{2-} + 2\text{OH}^- \rightarrow 2\text{MnO}_4^{2-} + \text{SO}_4^{2-} + \text{H}_2\text{O}$$

**教学要点**：同一反应物在不同介质中产物完全不同，**"介质优先"** 是配平的第一原则。

> **相关真题**：[[题-038-2-1-2-KMnO4氧化KI离子方程式]]（酸性条件 KMnO₄ 氧化 I⁻ 的离子方程式）

---

### 例 2 — Cr₂O₇²⁻ 酸性氧化 Fe²⁺ ⭐⭐

**题目**：写出 K₂Cr₂O₇ 在酸性条件下氧化 FeSO₄ 的离子方程式。

**思路分析**：Cr₂O₇²⁻ 中 Cr 为 +6 价，还原为 Cr³⁺；Fe²⁺ 氧化为 Fe³⁺。

**完整解答**：

还原半反应：$\text{Cr}_2\text{O}_7^{2-} + 14\text{H}^+ + 6\text{e}^- \rightarrow 2\text{Cr}^{3+} + 7\text{H}_2\text{O}$
氧化半反应：$\text{Fe}^{2+} \rightarrow \text{Fe}^{3+} + \text{e}^-$（×6）

合并：
$$\text{Cr}_2\text{O}_7^{2-} + 6\text{Fe}^{2+} + 14\text{H}^+ \rightarrow 2\text{Cr}^{3+} + 6\text{Fe}^{3+} + 7\text{H}_2\text{O}$$

> **关键**：Cr₂O₇²⁻ 含 **2 个 Cr**，整体得 **6e⁻**（每个 Cr 得 3e⁻）。常见错误是误以为只得了 3e⁻。

> **相关真题**：[[题-030-1-3-保险粉除CrVI离子方程式]]（Cr(VI)还原为 Cr(III) 的离子方程式书写）

---

### 例 3 — Al 与 NaOH 溶液反应 ⭐⭐

**题目**：写出 Al 与 NaOH 溶液反应的离子方程式。

**思路分析**：Al 是两性金属；碱性条件下产物是四羟基合铝酸盐（[Al(OH)₄]⁻）；H₂O 中的 H⁺ 被还原为 H₂。

**完整解答**：

Al 失电子（氧化）：$\text{Al} + 4\text{OH}^- \rightarrow [\text{Al(OH)}_4]^- + 3\text{e}^-$
H₂O 得电子（还原）：$2\text{H}_2\text{O} + 2\text{e}^- \rightarrow \text{H}_2\uparrow + 2\text{OH}^-$

合并（最小公倍数 6e⁻）：
$$2\text{Al} + 2\text{OH}^- + 6\text{H}_2\text{O} \rightarrow 2[\text{Al(OH)}_4]^- + 3\text{H}_2\uparrow$$

**化学方程式形式**：
$$2\text{Al} + 2\text{NaOH} + 6\text{H}_2\text{O} \rightarrow 2\text{Na}[\text{Al(OH)}_4] + 3\text{H}_2\uparrow$$

> **注意**：Al 与碱反应的本质是 Al 还原 H₂O 中的 H⁺，而非直接还原 NaOH。产物是四羟基合铝酸盐，过去常写为 AlO₂⁻（偏铝酸盐），但现代化学中正确写法为 [Al(OH)₄]⁻。

### 例 4 — Cl₂ 与 NaOH 歧化反应 ⭐⭐⭐

**题目**：写出 Cl₂ 与冷稀 NaOH 溶液以及热浓 NaOH 溶液反应的离子方程式。

**思路分析**：歧化反应中 Cl₂ 中的 Cl 既是氧化剂又是还原剂，Cl 从 0 价既升高又降低。

**完整解答**：

**（1）冷稀碱**：
$$\text{Cl}_2 + 2\text{OH}^- \rightarrow \text{Cl}^- + \text{ClO}^- + \text{H}_2\text{O}$$

**（2）热浓碱**：
$$3\text{Cl}_2 + 6\text{OH}^- \xrightarrow{\Delta} 5\text{Cl}^- + \text{ClO}_3^- + 3\text{H}_2\text{O}$$

> **为什么温度影响产物？**：ClO⁻ 在加热条件下不稳定，进一步歧化为 Cl⁻ 和 ClO₃⁻：
> $$3\text{ClO}^- \xrightarrow{\Delta} 2\text{Cl}^- + \text{ClO}_3^-$$
> 将两式叠加即得热碱条件下的总反应。

---

### 例 5（拓展）— Zn 与 NaOH 反应 ⭐⭐⭐

**题目**：写出 Zn 与 NaOH 溶液反应的离子方程式。

**思路分析**：Zn 也是两性金属（与 Al 类似），碱性条件下生成 [Zn(OH)₄]²⁻。

**完整解答**：
$$\text{Zn} + 2\text{OH}^- + 2\text{H}_2\text{O} \rightarrow [\text{Zn(OH)}_4]^{2-} + \text{H}_2\uparrow$$

---

### 例 6（拓展）— Cu 与浓 HNO₃ 反应 ⭐⭐⭐

**题目**：写出 Cu 与浓 HNO₃ 反应的化学方程式。

**思路分析**：Cu 被氧化为 Cu²⁺；浓 HNO₃ 被还原为 NO₂。

**完整解答**：
$$\text{Cu} + 4\text{HNO}_3(\text{浓}) \rightarrow \text{Cu(NO}_3)_2 + 2\text{NO}_2\uparrow + 2\text{H}_2\text{O}$$

**离子方程式**：
$$\text{Cu} + 4\text{H}^+ + 2\text{NO}_3^- \rightarrow \text{Cu}^{2+} + 2\text{NO}_2\uparrow + 2\text{H}_2\text{O}$$

> **相关拓展**：Cu 与稀 HNO₃ 的反应产物不同（稀 → NO），体现了介质浓度对还原产物的影响。

## 五、方法总结

### 5.1 离子-电子法总流程

```
判断介质（酸/碱/中性）
    ↓
确定产物（熟记氧化剂还原产物表）
    ↓
写半反应 → 配原子 → 配电荷
    ↓
合并半反应 → 检查守恒
```

### 5.2 核心结论

1. **介质判断是配平的第一步**，决定产物形态
2. **离子-电子法四步走**：介质定 → 产物定 → 原子平 → 电荷平
3. **碱性条件绝不允许出现游离 H⁺**，酸性条件绝不允许出现游离 OH⁻
4. **歧化反应**中同一元素既被氧化又被还原
5. **配平后必须检查**原子守恒和电荷守恒

### 5.3 常见易错点

| 错误想法 | 为什么错 | 正确理解 |
|:---|:---|:---|
| "配平就是凑系数" | 缺乏系统方法，容易出错 | 使用离子-电子法，按步骤进行 |
| "碱性条件也可以写 H⁺" | 违反基本化学事实 | 碱性条件只能用 OH⁻ 和 H₂O 配平 |
| "Cr₂O₇²⁻ 得 3 个电子" | 忽略了 1 个 Cr₂O₇²⁻ 含 2 个 Cr | Cr₂O₇²⁻ 整体得 **6e⁻** |
| "MnO₄⁻ 总是还原成 Mn²⁺" | 忽略了介质影响 | 酸 → Mn²⁺，中 → MnO₂，碱 → MnO₄²⁻ |

## 七、参考答案与提示

**6.1 基础巩固**

1. 酸性条件：
   - $2\text{KMnO}_4 + 10\text{FeSO}_4 + 8\text{H}_2\text{SO}_4 \rightarrow 2\text{MnSO}_4 + 5\text{Fe}_2(\text{SO}_4)_3 + \text{K}_2\text{SO}_4 + 8\text{H}_2\text{O}$
   - 离子：$2\text{MnO}_4^- + 10\text{Fe}^{2+} + 16\text{H}^+ \rightarrow 2\text{Mn}^{2+} + 10\text{Fe}^{3+} + 8\text{H}_2\text{O}$
   - $\text{Cr}_2\text{O}_7^{2-} + 3\text{H}_2\text{S} + 8\text{H}^+ \rightarrow 2\text{Cr}^{3+} + 3\text{S}\downarrow + 7\text{H}_2\text{O}$
   - $2\text{MnO}_4^- + 5\text{H}_2\text{C}_2\text{O}_4 + 6\text{H}^+ \rightarrow 2\text{Mn}^{2+} + 10\text{CO}_2\uparrow + 8\text{H}_2\text{O}$

2. 碱性/中性：
   - $\text{Cl}_2 + 2\text{OH}^- \rightarrow \text{Cl}^- + \text{ClO}^- + \text{H}_2\text{O}$
   - $\text{Zn} + 2\text{OH}^- + 2\text{H}_2\text{O} \rightarrow [\text{Zn(OH)}_4]^{2-} + \text{H}_2\uparrow$
   - $2\text{MnO}_4^- + 3\text{SO}_3^{2-} + \text{H}_2\text{O} \rightarrow 2\text{MnO}_2\downarrow + 3\text{SO}_4^{2-} + 2\text{OH}^-$

3. 离子方程式：
   - $3\text{Cu} + 8\text{H}^+ + 2\text{NO}_3^- \rightarrow 3\text{Cu}^{2+} + 2\text{NO}\uparrow + 4\text{H}_2\text{O}$
   - $\text{Fe} + 4\text{H}^+ + \text{NO}_3^- \rightarrow \text{Fe}^{3+} + \text{NO}\uparrow + 2\text{H}_2\text{O}$（稀硝酸过量时）

**6.2 提高训练**

4. 碱性条件 MnO₄⁻ → MnO₂：$2\text{MnO}_4^- + 3\text{S}^{2-} + 4\text{H}_2\text{O} \rightarrow 2\text{MnO}_2\downarrow + 3\text{S}\downarrow + 8\text{OH}^-$

5. $5\text{I}^- + \text{IO}_3^- + 6\text{H}^+ \rightarrow 3\text{I}_2 + 3\text{H}_2\text{O}$（归中反应，I⁻ 和 IO₃⁻ 均生成 I₂）

6. $2\text{Cl}_2 + 2\text{Ca(OH)}_2 \xrightarrow{\Delta} \text{Ca(ClO)}_2 + \text{CaCl}_2 + 2\text{H}_2\text{O}$
   （注意：工业漂白粉有效成分是 Ca(ClO)₂）

7. $4\text{H}_2\text{O}_2 + \text{PbS} \rightarrow \text{PbSO}_4 + 4\text{H}_2\text{O}$
   （H₂O₂ 中 O 从 -1 降为 -2，PbS 中 S 从 -2 升为 +6）

**6.3 挑战题**

8. 提示：As₂S₃ 中 As(+3)→H₃AsO₄(+5)，S(-2)→H₂SO₄(+6)，一个 As₂S₃ 共失去 28e⁻；HNO₃ 中 N(+5)→NO(+2)，得 3e⁻。
