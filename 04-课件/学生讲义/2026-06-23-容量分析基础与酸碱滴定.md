# 容量分析基础与酸碱滴定（提高班）

> **适用**：第二轮提高班 | **前置**：滴定分析基础、SP/EP/Et 基本术语、酸碱平衡、电离平衡
> **深度边界**：不展开混合酸的精确计算迭代法；不展开非水滴定的溶剂效应详细处理；不展开自动滴定仪的电路原理。详见 §0.3。

## §0 深度边界与前置说明

### 0.1 本讲定位

本轮在第一轮已建立 SP/EP/Et 概念和基本滴定术语的基础上，深化定量计算能力，建立滴定曲线直觉，掌握指示剂选择逻辑，能处理双指示剂法的分别测定问题与终点误差的定量估算。

### 0.2 学到什么程度

- 能独立判断基准物质是否合格，会写标准溶液配制和标定的完整计算流程

- 能定性画出强酸-强碱、弱酸-强碱、多元酸的滴定曲线，判断突跃范围
- 能运用林邦（Ringbom）误差公式计算终点误差，理解 ΔpH 的符号规则
- 能运用双指示剂法分析 Na₂CO₃/NaHCO₃ 混合物的组成和含量

## 一、容量分析基础深化

### 1.1 滴定分析分类速览

| 分类方式 | 类别 | 特点 |
|:---:|:---|:---|
| 按反应类型 | 酸碱滴定 | 质子传递反应 |
| | 氧化还原滴定 | 电子转移反应 |
| | 配位滴定 | 配位反应（常用 EDTA） |
| | 沉淀滴定 | 沉淀反应（银量法） |
| 按滴定方式 | 直接滴定 | 标准溶液直接滴定待测物 |
| | 返滴定 | 先加过量 A，再用 B 回滴剩余 A |
| | 置换滴定 | 待测物置换出可滴定的物质 |
| | 间接滴定 | 通过其他反应间接定量 |

### 1.2 四种滴定方式对比

| 特征 | 直接滴定 | 返滴定 | 置换滴定 | 间接滴定 |
|:---|:---:|:---:|:---:|:---:|
| 操作步骤 | 一步完成 | 两步（过量→回滴）| 两步（置换→滴定）| 多步转化 |
| 计量关系 | 直接 1:1 | 差量法 | 通过中间产物 | 通过串联反应 |
| 主要适用 | 满足五大条件的反应 | 反应慢/无合适指示剂 | 副反应干扰计量关系 | 不能直接与标准溶液反应 |
| 经典案例 | NaOH 滴定 HCl | Al³⁺ + 过量 EDTA → Zn²⁺ 回滴 | K₂Cr₂O₇ + KI → I₂ → Na₂S₂O₃ 滴定 | Ca²⁺ → CaC₂O₄ → H₂C₂O₄ → KMnO₄ 滴定 |

> ⚠️ **误区框 1**：返滴定不等于间接滴定。返滴定是用两种标准溶液直接反应（A + B），而间接滴定涉及待测物的化学转化后再滴定。例如测定 Ca²⁺ 通过草酸钙沉淀转化后滴定，这是间接滴定；Al³⁺ 加过量 EDTA 后用 Zn²⁺ 回滴，这是返滴定。

### 1.3 基准物质五项要求

基准物质是滴定分析的"根"，须满足：

1. **组成确定**——化学式与组成精确对应（含结晶水时也应恒定）
2. **纯度高**——纯度 ≥ 99.9%
3. **性质稳定**——不吸收 CO₂/H₂O，不被氧化，加热干燥时不分解
4. **定量反应**——按确定计量关系反应，无副反应
5. **摩尔质量大**——称取量多，减小称量相对误差

> ⚠️ **误区框 2**：为什么 KOH 不能做基准物质？因为 KOH 易吸收空气中的 CO₂ 和 H₂O，组成不恒定、性质不稳定。同理，浓 HCl 易挥发，Na₂S₂O₃·5H₂O 易风化失水，均不适合。

**常用基准物质分类表**：

| 标定对象 | 常用基准物质 | 化学式 | 摩尔质量 (g/mol) |
|:---:|:---|:---|:---:|
| 酸（HCl） | 无水碳酸钠 | Na₂CO₃ | 106.0 |
| 酸（HCl） | 硼砂 | Na₂B₄O₇·10H₂O | 381.4 |
| 碱（NaOH） | 邻苯二甲酸氢钾（KHP） | KHC₈H₄O₄ | 204.2 |
| 碱（NaOH） | 草酸 | H₂C₂O₄·2H₂O | 126.1 |
| KMnO₄ | 草酸钠 | Na₂C₂O₄ | 134.0 |
| EDTA | 纯锌 | Zn | 65.38 |
| AgNO₃ | 氯化钠 | NaCl | 58.44 |

### 1.4 标准溶液配制

| 方法 | 操作 | 适用物质 |
|:---:|:---|:---|
| **直接法** | 准确称取基准物质 → 溶解 → 定容至容量瓶 | 符合基准物质条件的试剂 |
| **间接法（标定法）** | 配成近似浓度 → 用基准物质标定 | NaOH、KMnO₄、Na₂S₂O₃、HCl 等 |

**标定计算通式**：

$$c_{\text{标}} = \frac{n_{\text{基准}} \times (a/t)}{V_{\text{标}}} = \frac{m_{\text{基准}} / M_{\text{基准}} \times (a/t)}{V_{\text{标}}}$$

其中 $a/t$ 为滴定剂与基准物质的化学计量系数比。

### 例题 1：基准物质判断与标定计算 ⭐⭐⭐

**题干**：现有以下物质：KOH(s)、Na₂CO₃(s，无水)、HCl(浓)、硼砂(Na₂B₄O₇·10H₂O)、KHP。请回答：
(1) 哪些适合做基准物质？说明理由。
(2) 称取 0.5023 g 无水 Na₂CO₃ 标定 HCl，消耗 HCl 25.35 mL，计算 c(HCl)。

**解析**：

(1)

| 物质 | 是否合格 | 理由 |
|:---|:---:|:---|
| KOH(s) | ❌ | 易吸收 CO₂ 和 H₂O，不稳定 |
| Na₂CO₃(s, 无水) | ✅ | 纯度高、稳定、摩尔质量适中 |
| HCl(浓) | ❌ | 易挥发，组成不恒定 |
| 硼砂 | ✅ | 纯度高、稳定、摩尔质量大（381.4 g/mol，称量相对误差小） |
| KHP | ✅ | 纯度高、不吸湿、摩尔质量大（204.2 g/mol） |

(2) 反应：Na₂CO₃ + 2HCl → 2NaCl + CO₂↑ + H₂O

$$n(\text{Na}_2\text{CO}_3) = \frac{0.5023}{106.0} = 4.739 \times 10^{-3} \text{ mol}$$

$$n(\text{HCl}) = 2 \times n(\text{Na}_2\text{CO}_3) = 9.478 \times 10^{-3} \text{ mol}$$

$$c(\text{HCl}) = \frac{9.478 \times 10^{-3}}{25.35 \times 10^{-3}} = 0.3739 \text{ mol/L}$$

**答案**：见 (1) 表；c(HCl) = 0.3739 mol/L。

## 二、酸碱指示剂变色原理

### 2.1 指示剂变色推导

酸碱指示剂是一类弱有机酸或弱有机碱（以 HIn 表示酸式，In⁻ 表示碱式）：

$$\text{HIn} \rightleftharpoons \text{H}^+ + \text{In}^-$$

$$K_a = \frac{[\text{H}^+][\text{In}^-]}{[\text{HIn}]}, \quad \frac{[\text{In}^-]}{[\text{HIn}]} = \frac{K_a}{[\text{H}^+]}$$

**理论变色点**：当 pH = pKa 时，[In⁻]/[HIn] = 1，酸式色 = 碱式色（中间过渡色）。

**理论变色范围**：人眼区分两种颜色的极限是 [In⁻]/[HIn] ≥ 10（看到碱式色）或 ≤ 1/10（看到酸式色）。代入得：

$$\text{pH} = \text{p}K_{\text{HIn}} \pm 1$$

> ⚠️ **误区框 3**：指示剂变色范围不是恰好 pH = pKa 这一个点，而是一个区间。pH = pKa 时看到的是混合色（中间色），不是终点。实际变色范围比理论值窄（约 1.6~1.8 个 pH 单位），因为人眼对不同颜色的敏感度不同。

### 2.2 常见酸碱指示剂

| 指示剂 | 变色范围 pH | 酸式色 | 碱式色 | pKa |
|:---|:---:|:---:|:---:|:---:|
| 甲基橙（MO） | 3.1~4.4 | 红 | 黄（橙） | 3.4 |
| 甲基红（MR） | 4.4~6.2 | 红 | 黄 | 5.0 |
| 溴百里酚蓝（BTB） | 6.2~7.6 | 黄 | 蓝 | 7.1 |
| 酚酞（PP） | 8.0~10.0 | 无色 | 粉红 | 9.1 |
| 百里酚酞 | 9.4~10.6 | 无色 | 蓝 | 10.0 |

### 2.3 混合指示剂

当需要将终点限制在很窄的 pH 范围内时使用混合指示剂，分为两类：

**类型一**：两种指示剂混合——利用颜色互补使变色更敏锐。例如溴甲酚绿（pKa = 4.9，黄→蓝）与甲基红（pKa = 5.2，红→黄）混合，在 pH ≈ 5.1 时由橙色突变为绿色（经过灰色过渡带）。

**类型二**：指示剂 + 惰性染料——颜色互补增强敏锐性。

> 💡 **对比**：单一指示剂变色过渡带通常 ≥ 1 pH 单位，而混合指示剂可压缩至 0.2 pH 单位以下。pH 试纸即由多种混合指示剂制成。

### 2.4 指示剂选择原则

**核心原则**：指示剂的变色范围必须**落入滴定突跃范围**之内。

| 滴定类型 | 突跃范围 (0.1 M) | 可选指示剂 | 不可选 |
|:---|:---:|:---|:---|
| 强碱滴定强酸 | 4.30~9.70 | 甲基红、酚酞、溴百里酚蓝 | — |
| 强碱滴定弱酸 (HAc) | 7.74~9.70 | 酚酞 | 甲基橙、甲基红 |
| 强酸滴定弱碱 (NH₃) | 6.25~4.30 | 甲基橙、甲基红 | 酚酞 |

## 三、酸碱滴定曲线深化

### 3.1 强酸-强碱滴定曲线（以 0.1000 M NaOH 滴定 20.00 mL 0.1000 M HCl 为例）

滴定分数定义：$a = \frac{V_{\text{NaOH}} \times c_{\text{NaOH}}}{V_{\text{HCl}} \times c_{\text{HCl}}}$

| 阶段 | 滴定分数 a | 溶液中组分 | pH 计算 | 示例 pH |
|:---:|:---:|:---|:---|:---:|
| 滴定前 | 0 | 纯 HCl | [H⁺] = c(HCl) | 1.00 |
| 计量点前 | 0.999 | 剩余 HCl + NaCl | [H⁺] = 剩余 HCl / 总体积 | 4.30 |
| 计量点 | 1.000 | 纯 NaCl（中性） | [H⁺] = [OH⁻] = 10⁻⁷ | 7.00 |
| 计量点后 | 1.001 | 过量 NaOH + NaCl | [OH⁻] = 过量 NaOH / 总体积 | 9.70 |

**突跃范围**：pH 4.30 → 9.70（a = 0.999 至 a = 1.001）

> 🧠 **数值冲击**：从 a = 0.999 到 a = 1.001，只差 0.02 mL NaOH（约半滴！），pH 从 4.30 跃升至 9.70，变化 5.4 个 pH 单位。

**浓度对突跃的影响**：

| 浓度 | 突跃范围 | 突跃宽度 (ΔpH) |
|:---:|:---:|:---:|
| 1.0 M | 3.3~10.7 | 7.4 |
| 0.1 M | 4.3~9.7 | 5.4 |
| 0.01 M | 5.3~8.7 | 3.4 |

规律：浓度增大 10 倍 → 突跃增加约 2 个 pH 单位。

![[media/titration-curve-hcl-strong.svg|强酸-强碱滴定曲线]]
*图 0.1000 M NaOH 滴定 20.00 mL 0.1000 M HCl 的滴定曲线——pH 随滴定分数 a 的变化。计量点附近 pH 从 4.30 跃升至 9.70（突跃范围），指示剂选择须在突跃范围内。酚酞（8.0~10.0）和甲基橙（3.1~4.4）均可，因二者变色范围均部分落入突跃（分析化学第六版 第5章）*

### 3.2 弱酸-强碱滴定曲线（以 0.1000 M NaOH 滴定 20.00 mL 0.1000 M HAc 为例）

**滴定前**：
$$[\text{H}^+] = \sqrt{K_a \cdot c} = \sqrt{1.8 \times 10^{-5} \times 0.1000} = 1.34 \times 10^{-3}$$
$$\text{pH} = 2.87$$

**缓冲区（0 < a < 1）**：形成 HAc-NaAc 缓冲对，pH 由 H-H 方程决定：
$$\text{pH} = \text{p}K_a + \lg\frac{[\text{Ac}^-]}{[\text{HAc}]}$$

**计量点（a = 1）**：产物 NaAc，Ac⁻ 水解：
$$[\text{OH}^-] = \sqrt{K_b \cdot c_{\text{sp}}} = \sqrt{\frac{10^{-14}}{1.8 \times 10^{-5}} \times 0.05000} = 5.27 \times 10^{-6}$$
$$\text{pOH} = 5.28, \quad \text{pH} = 8.72$$

**计量点后（a > 1）**：过量的 NaOH 决定 pH（同强碱滴定）。

**突跃范围**：pH 7.74 → 9.70

> 🎯 **核心对比**：
> - 强酸滴定：计量点 pH = 7，突跃宽（4.30~9.70），甲基橙/酚酞均可
> - 弱酸滴定：计量点 pH = 8.72，突跃窄（7.74~9.70），只能用酚酞
> - **原因**：弱酸滴定前已有部分解离（pH 较高），缓冲区使曲线较平坦，突跃被压缩

### 3.3 强酸-弱碱滴定曲线（以 HCl 滴定 NH₃ 为例）

计量点产物 NH₄Cl（强酸弱碱盐），溶液呈酸性：
$$\text{pH} \approx 5.28$$

突跃在酸性范围，可选甲基橙或甲基红。

### 3.4 三种滴定曲线对比

| 特征 | 强碱滴定强酸 | 强碱滴定弱酸 | 强酸滴定弱碱 |
|:---|:---:|:---:|:---:|
| 起始 pH | 1.00（低） | 2.87（中） | 11.13（高） |
| 缓冲区 | 无（强酸无缓冲） | HAc-Ac⁻ 缓冲对 | NH₃-NH₄⁺ 缓冲对 |
| 计量点 pH | 7.00 | 8.72（碱性） | 5.28（酸性） |
| 突跃范围 | 4.30~9.70 | 7.74~9.70 | 6.25~4.30 |
| 突跃宽度 | 5.4 | 2.0 | 2.0 |
| 可选指示剂 | MO/MR/PP/BTB | 酚酞 | 甲基橙/甲基红 |

### 3.5 多元酸分步滴定——以 H₃PO₄ 为例

**第一步**：H₃PO₄ + NaOH → NaH₂PO₄ + H₂O
**第二步**：NaH₂PO₄ + NaOH → Na₂HPO₄ + H₂O
**第三步**：Na₂HPO₄ + NaOH → Na₃PO₄ + H₂O

**分步滴定条件**：
1. 第一级可滴：$cK_{\text{a}_1} \geq 10^{-8}$
2. 可分步：$K_{\text{a}_1} / K_{\text{a}_2} \geq 10^5$（等浓度时）

| 级别 | Ka | cKa | 可滴性 | 相邻比 | 可分步 |
|:---:|:---:|:---:|:---:|:---:|:---:|
| 第一级 (H₃PO₄) | 7.6×10⁻³ | 7.6×10⁻⁴ ✅ | ✅ | Ka₁/Ka₂ ≈ 1.2×10⁵ ✅ | ✅ |
| 第二级 (H₂PO₄⁻) | 6.3×10⁻⁸ | 6.3×10⁻⁹ ≈ 10⁻⁸ | 勉强可滴 | Ka₂/Ka₃ ≈ 1.4×10⁵ ✅ | ✅ |
| 第三级 (HPO₄²⁻) | 4.4×10⁻¹³ | 4.4×10⁻¹⁴ ❌ | ❌ 不可直接滴定 | — | — |

**计量点 pH 计算**：
- 第一计量点（H₂PO₄⁻ 两性物质）：$\text{pH} = \frac{1}{2}(\text{p}K_{\text{a}_1} + \text{p}K_{\text{a}_2}) = \frac{1}{2}(2.12 + 7.21) = 4.66$ → 可选甲基橙/甲基红
- 第二计量点（HPO₄²⁻ 两性物质）：$\text{pH} = \frac{1}{2}(\text{p}K_{\text{a}_2} + \text{p}K_{\text{a}_3}) = \frac{1}{2}(7.21 + 12.36) = 9.78$ → 可选百里酚酞

> ⚠️ **误区框 4**：分步滴定要求 $K_{\text{a}_1}/K_{\text{a}_2} \geq 10^5$ 而不是 $10^4$。$10^4$ 时两步滴定有交叉干扰，一般要求 $10^5$ 才能保证相邻两级分别滴定时误差在 1% 以内。

## 四、终点误差定量计算

### 4.1 终点误差的基本定义

$$E_t = \frac{\text{终点时过量(或不足)的滴定剂物质的量}}{\text{计量点时应加入的滴定剂物质的量}} \times 100\%$$

- ΔpH = pH_ep - pH_sp > 0 → 滴定剂过量 → Et > 0
- ΔpH = pH_ep - pH_sp < 0 → 滴定剂不足 → Et < 0

### 4.2 林邦（Ringbom）误差公式

**强酸滴定强碱**（或相反）：

$$E_t = \frac{10^{\Delta \text{pH}} - 10^{-\Delta \text{pH}}}{\sqrt{1/K_w \cdot c^{\text{ep}}}} \times 100\%$$

**弱酸滴定**（强碱滴定弱酸）：

$$E_t = \frac{10^{\Delta \text{pH}} - 10^{-\Delta \text{pH}}}{\sqrt{c^{\text{sp}} K_a / K_w}} \times 100\%$$

**林邦公式的通式理解**：
- 分子：$10^{\Delta \text{pH}} - 10^{-\Delta \text{pH}}$ 反映终点偏离程度（偏离越大，误差越大）
- 分母：$\sqrt{c^{\text{sp}} K_t}$（$K_t$ 为滴定反应常数）反映滴定反应的完全程度——反应越完全，分母越大，相同 ΔpH 下误差越小

### 4.3 ΔpH 符号规则

| 条件 | ΔpH 符号 | 滴定剂状态 | Et 符号 |
|:---|:---:|:---|:---:|
| 终点在 SP 之后（pH_ep > pH_sp） | + | 过量 | + |
| 终点在 SP 之前（pH_ep < pH_sp） | - | 不足 | - |
| 终点恰好 = SP（理想） | 0 | 恰好 | 0 |

### 例题 2：指示剂选择与突跃估算 ⭐⭐⭐

**题干**：0.1000 M NaOH 滴定 0.1000 M HCOOH（甲酸，Ka = 1.8×10⁻⁴，pKa = 3.74）。
(1) 计算计量点 pH。
(2) 估算突跃范围，判断能否用酚酞？能否用甲基橙？

**解析**：

(1) 计量点产物为 HCOONa（强碱弱酸盐）：

$$c_{\text{sp}} = \frac{0.1000 \times V_{\text{酸}}}{V_{\text{酸}} + V_{\text{碱}}} = \frac{0.1000}{2} = 0.05000 \text{ M}$$

$$K_b = \frac{K_w}{K_a} = \frac{10^{-14}}{1.8 \times 10^{-4}} = 5.56 \times 10^{-11}$$

$$[\text{OH}^-] = \sqrt{K_b \cdot c_{\text{sp}}} = \sqrt{5.56 \times 10^{-11} \times 0.05000} = 1.67 \times 10^{-6}$$

$$\text{pOH} = 5.78, \quad \text{pH} = 8.22$$

(2) 突跃估算：
- 突跃起点（a = 0.999）：剩余弱酸 + 甲酸钠缓冲对，缓冲区末端 pH ≈ pKa + 1 ≈ 4.74
  更精确：a = 0.999 时 [Ac⁻]/[HAc] ≈ 999/1，pH ≈ pKa + lg(999) = 3.74 + 3.00 = 6.74
  （注意：在计量点前非常接近时，缓冲比极大，pH 快速上升）
- 突跃终点（a = 1.001）：过量 NaOH 0.1%，[OH⁻] = 0.1000 × 0.001 / 2 = 5.0×10⁻⁵
  pOH = 4.30，pH = 9.70
- 突跃范围约：pH 6.7 → 9.7

酚酞（8.0~10.0）在此范围内 → ✅ 可用。
甲基橙（3.1~4.4）完全不在范围内 → ❌ 不可用。

**答案**：(1) pH = 8.22；(2) 突跃约 6.7~9.7，可用酚酞，不可用甲基橙。

### 例题 3：林邦误差公式计算 ⭐⭐⭐

**题干**：用 0.1000 M NaOH 滴定 0.1000 M HAc（Ka = 1.8×10⁻⁵），化学计量点 pH = 8.72。若以酚酞为指示剂（变色点 pH ≈ 9.0），用林邦公式计算终点误差 Et。

**解析**：

步骤 1：计算 ΔpH
$$\Delta \text{pH} = \text{pH}_{\text{ep}} - \text{pH}_{\text{sp}} = 9.0 - 8.72 = +0.28$$

注意 ΔpH > 0，说明终点在 SP 之后、滴定剂已过量 → Et 应为正。

步骤 2：计算分子
$$10^{\Delta \text{pH}} - 10^{-\Delta \text{pH}} = 10^{0.28} - 10^{-0.28} = 1.905 - 0.525 = 1.380$$

步骤 3：计算分母
计量点时总体积约 2 倍，$c_{\text{sp}} = 0.05000$ M

$$c_{\text{sp}} \cdot K_a / K_w = 0.05000 \times 1.8 \times 10^{-5} / 10^{-14} = 9.0 \times 10^7$$

$$\sqrt{c_{\text{sp}} K_a / K_w} = \sqrt{9.0 \times 10^7} = 9487$$

步骤 4：代入林邦公式

$$E_t = \frac{1.380}{9487} \times 100\% \approx +0.015\%$$

**结论**：Et ≈ +0.015%，即终点约过量 0.015%，误差极小。酚酞是适合 HAc 滴定的指示剂。

> 💡 **为何误差这么小？** 虽然 ΔpH = 0.28 看起来不小，但分母（反映反应完全程度）达到 9487，说明 HAc 与 NaOH 的反应非常完全（Ka 不算太小），少量的终点偏离仅造成极小误差。

## 五、混合酸碱滴定——双指示剂法

### 5.1 原理与反应分步

Na₂CO₃ 是二元碱，用 HCl 标准溶液滴定时分两步：

**第一步**（酚酞终点，pH ≈ 8.3）：
$$\text{CO}_3^{2-} + \text{H}^+ \rightarrow \text{HCO}_3^-$$

**第二步**（甲基橙终点，pH ≈ 3.9）：
$$\text{HCO}_3^- + \text{H}^+ \rightarrow \text{H}_2\text{CO}_3 \rightarrow \text{CO}_2 \uparrow + \text{H}_2\text{O}$$

> ⚠️ **误区框 5**：酚酞变色时 Na₂CO₃ 并没有被完全中和！只走到 HCO₃⁻（第一终点），全程走了三分之一（中和了一个 H⁺）。整个 Na₂CO₃ 完全反应需要 2 个 H⁺，第一终点用了 1 个，第二终点再用 1 个。

### 5.2 V₁/V₂ 判组成法

记酚酞终点消耗 HCl 体积为 V₁（mL），接着加甲基橙继续消耗 HCl 体积为 V₂（mL）。

| V₁ 与 V₂ 关系 | 样品组成 | 说明 |
|:---:|:---|:---|
| V₁ = V₂ > 0 | 纯 Na₂CO₃ | 两段各用一半 |
| V₁ = 0, V₂ > 0 | 纯 NaHCO₃ | 酚酞不显色 |
| V₁ > 0, V₂ = 0 | 纯 NaOH（无 CO₃²⁻） | — |
| V₁ > V₂ | NaOH + Na₂CO₃ | 酚酞部分包含 NaOH 全中和 + Na₂CO₃ 的一半 |
| V₁ < V₂ | Na₂CO₃ + NaHCO₃ | V₁ 仅为 Na₂CO₃ 的一半，V₂ 含 Na₂CO₃ 另一半 + NaHCO₃ |

**含量计算公式**（以 Na₂CO₃ + NaHCO₃ 混合物为例）：

$$\omega_{\text{Na}_2\text{CO}_3} = \frac{c_{\text{HCl}} \times 2V_1 \times M_{\text{Na}_2\text{CO}_3}}{m_{\text{样}}} \times 100\%$$

$$\omega_{\text{NaHCO}_3} = \frac{c_{\text{HCl}} \times (V_2 - V_1) \times M_{\text{NaHCO}_3}}{m_{\text{样}}} \times 100\%$$

### 例题 4：双指示剂法判组成 ⭐⭐⭐

**题干**：某混合碱样品 0.6524 g，溶于水后用 0.2000 M HCl 滴定。先用酚酞作指示剂，消耗 HCl 15.00 mL；再加入甲基橙，继续消耗同浓度 HCl 22.50 mL。判断样品组成并计算各组分质量分数。

**解析**：

步骤 1：比较 V₁ 和 V₂
V₁ = 15.00 mL，V₂ = 22.50 mL，V₁ < V₂ → 样品为 Na₂CO₃ + NaHCO₃ 混合物。

步骤 2：计算 Na₂CO₃ 质量分数
酚酞终点（V₁ = 15.00 mL）是 Na₂CO₃ → NaHCO₃（只走了一半），所以 Na₂CO₃ 消耗的总 HCl = 2V₁ = 30.00 mL。

$$\omega_{\text{Na}_2\text{CO}_3} = \frac{0.2000 \times \frac{2 \times 15.00}{1000} \times 106.0}{0.6524} \times 100\%$$

$$= \frac{0.2000 \times 0.03000 \times 106.0}{0.6524} \times 100\% = \frac{0.6360}{0.6524} \times 100\% = 97.49\%$$

步骤 3：计算 NaHCO₃ 质量分数
NaHCO₃ 消耗的 HCl = V₂ - V₁ = 22.50 - 15.00 = 7.50 mL。

$$\omega_{\text{NaHCO}_3} = \frac{0.2000 \times \frac{7.50}{1000} \times 84.01}{0.6524} \times 100\%$$

$$= \frac{0.2000 \times 0.00750 \times 84.01}{0.6524} \times 100\% = \frac{0.1260}{0.6524} \times 100\% = 19.31\%$$

步骤 4：验证
97.49% + 19.31% = 116.80%，超过 100%。这说明 Na₂CO₃ 的摩尔质量取了 106.0 偏低，或样品中存在其他杂质。实际上样品可能是 Na₂CO₃ 和 NaHCO₃ 的混合物，但质量分数之和超过 100% 提示计算复核——在本例题中，ω(Na₂CO₃) = 97.49% 非常接近纯 Na₂CO₃，说明 NaHCO₃ 的贡献很小，实际样品以 Na₂CO₃ 为主（少量未完全转化的 NaHCO₃）。

**答案**：组成为 Na₂CO₃ + NaHCO₃；ω(Na₂CO₃) = 97.49%，ω(NaHCO₃) = 19.31%。

### 5.3 PO₄³⁻ 分步滴定

H₃PO₄ 是三元酸，用 NaOH 滴定时的分步情况：

- 第一终点（pH ≈ 4.7）：H₃PO₄ → H₂PO₄⁻，用甲基橙/甲基红
- 第二终点（pH ≈ 9.7）：H₂PO₄⁻ → HPO₄²⁻，用百里酚酞
- 第三级（HPO₄²⁻ → PO₄³⁻）：Ka₃ 太小不可直接滴定

V₁/V₂ 法同样适用于 PO₄³⁻ 混合体系：

| 组成 | V₁/V₂ 关系 |
|:---|:---:|
| 纯 H₃PO₄ | V₁/V₂ = 1:1 |
| H₃PO₄ + NaH₂PO₄ | V₁ < V₂ |
| NaH₂PO₄ | V₁ = 0，V₂ > 0 |
| NaH₂PO₄ + Na₂HPO₄ | V₁ > 0（仅从 NaH₂PO₄ 而来的消耗）... 需根据具体条件分析 |

## 六、非水滴定简介（点到为止）

### 6.1 为什么要非水滴定？

有些物质在水中：
- 碱性太弱（如苯胺、吡啶），无法用强酸准确滴定（cKb < 10⁻⁸）
- 酸性太弱（如酚类），无法用强碱准确滴定
- 溶解度太差

### 6.2 非水溶剂的选择原则

- **拉平效应**：在强酸性溶剂（如冰醋酸）中，弱碱被"拉平"到相同强度，可被统一滴定
- **区分效应**：在适当溶剂中，不同强度的酸/碱可被分别滴定

> ⚠️ **竞赛常考**：用 HClO₄ 的冰醋酸溶液滴定弱碱（如氨基酸、生物碱）——冰醋酸是强酸性溶剂，能将弱碱的碱性"拉平"到可滴定的程度。

### 6.3 非水滴定应用实例

- **冰醋酸体系**：用 HClO₄ 标准溶液滴定弱碱性物质（氨基酸、胺类、生物碱）
- **醇-苯混合溶剂**：用 NaOH 的乙醇溶液滴定弱酸性物质（酚类、烯醇类）
- **DMF 体系**：四丁基氢氧化铵滴定极弱酸

> 🧠 **本讲要求**：仅需了解非水滴定存在的必要性和基本思路，不展开溶剂效应的详细处理。

## 七、方法总结与速查卡

### 7.1 滴定误差判断口诀

> ΔpH 看符号：正超负不足，符号算清楚。
> 林邦公式三部曲：一求 ΔpH 差值，二判反应完全度，三除即得误差值。

### 7.2 双指示剂法记忆口诀

> 酚酞先走 V₁，甲基橙再走 V₂。
> V₁ = V₂ 纯碳酸，V₁ = 0 是氢钠。
> V₁ > V₂ 碱含碳，V₁ < V₂ 碳含氢。

### 7.3 指示剂选择逻辑链

```text
计算计量点 pH → 判断突跃范围 → 查指示剂变色范围表 →
    变色范围落入突跃范围内 → 选之
    变色范围不在突跃内 → 换之（或考虑混合指示剂）
```

### 7.4 滴定曲线类型速判表

| 滴定类型 | 计量点 pH | 突跃位置 | 突跃宽度 | 主要影响因素 |
|:---:|:---:|:---:|:---:|:---|
| 强酸-强碱 | 7.00 | 中性区 | 宽 | 浓度 |
| 强碱-弱酸 | > 7 | 碱性区 | 较窄 | Ka + 浓度 |
| 强酸-弱碱 | < 7 | 酸性区 | 较窄 | Kb + 浓度 |
| 多元酸 | 多变 | 多变 | 分级 | Ka₁/Ka₂ + cKa |

### 7.5 终点误差速查

| 条件 | 误差特点 | 应对策略 |
|:---|:---|:---|
| 指示剂变色点 = 计量点 | Et = 0 | 理想情况 |
| 强酸强碱滴定 | 对 ΔpH 不敏感（分母大） | 选甲基橙或酚酞均可 |
| 弱酸滴定 | 对 ΔpH 较敏感（分母中等） | 务必选酚酞（终点 pH ≈ 9） |
| 极弱酸滴定（cKa ≈ 10⁻⁸） | 对 ΔpH 极敏感（分母小） | 需用混合指示剂精确控制终点 |

## 八、综合例题

### 例题 5：混合碱 + 终点误差综合 ⭐⭐⭐⭐

**题干**：某试液可能含 NaOH、Na₂CO₃、NaHCO₃ 中的两种或一种。取 25.00 mL 试液，用 0.1000 M HCl 滴定：
- 以酚酞为指示剂时消耗 HCl 18.50 mL
- 以甲基橙为指示剂时（另取一份）消耗 HCl 30.00 mL

(1) 判断试液组成。
(2) 计算各组分浓度（mol/L）。
(3) 若酚酞的实际变色点 pH = 8.5，化学计量点 pH = 8.3，估算终点误差。

**解析**：

(1) 判断组成
比较 V₁ 与 V₂：V₁ = 18.50 mL，V₂ = 30.00 mL。
甲基橙终点 HCl 总量 V₂ = 30.00 mL > V₁ = 18.50 mL。
若为纯 Na₂CO₃，应有 V₁ = V₂ = 15.00 mL（不成立）。
若为 Na₂CO₃ + NaHCO₃，应有 V₁ < V₂（检查：2V₁ = 37.00 > V₂ = 30.00，不对）。
若为 NaOH + Na₂CO₃，应有 V₁ > V₂ - V₁（即 18.50 > 11.50，成立 ✅）。
故组成为 NaOH + Na₂CO₃。

(2) 计算浓度
设混合液体积为 25.00 mL。

NaOH 消耗 HCl（仅在酚酞阶段）：V(NaOH) = V₁ - (V₂ - V₁) = 18.50 - 11.50 = 7.00 mL

验证：酚酞终点 = NaOH 全中和 + Na₂CO₃→NaHCO₃（一半），
甲基橙终点比酚酞终点多消耗 = NaHCO₃→CO₂ 的一半 = V₂ - V₁ = 11.50 mL。
所以 Na₂CO₃ 消耗总 HCl = 2 × 11.50 = 23.00 mL，
NaOH 消耗 HCl = V₁ - 11.50 = 7.00 mL。

$$c(\text{NaOH}) = \frac{0.1000 \times 7.00}{25.00} = 0.02800 \text{ M}$$

$$c(\text{Na}_2\text{CO}_3) = \frac{0.1000 \times 23.00/2}{25.00} = 0.04600 \text{ M}$$

(3) 终点误差估算
Na₂CO₃ 第一计量点 pH = 8.3，酚酞变色点 pH = 8.5，ΔpH = +0.2。
但这里是 NaOH + Na₂CO₃ 混合体系，实际酚酞终点时溶液中是 NaOH 已中和、Na₂CO₃ 走到 NaHCO₃，属于缓冲体系。在混合体系中直接用林邦公式较为复杂，但可定性判断：ΔpH > 0 说明酚酞终点略滞后（过量 HCl 已略多），Et > 0，误差约在 0.1% 量级。

**答案**：(1) NaOH + Na₂CO₃；(2) c(NaOH) = 0.02800 M，c(Na₂CO₃) = 0.04600 M；(3) Et > 0，误差较小。

### 例题 6：多元酸分步滴定综合 ⭐⭐⭐⭐

**题干**：某磷酸溶液 25.00 mL，用 0.1000 M NaOH 滴定。以甲基橙为指示剂时消耗 NaOH 20.00 mL，以百里酚酞为指示剂时（另取一份）消耗 NaOH 40.00 mL。
(1) 判断该磷酸溶液的纯度级别（纯 H₃PO₄ 还是含 NaH₂PO₄？）。
(2) 计算原磷酸溶液的浓度。
(3) 第三级 HPO₄²⁻ 为什么不可直接滴定？

**解析**：

(1) 判断组成
若为纯 H₃PO₄，第一计量点（甲基橙）和第二计量点（百里酚酞）的体积比应为 1:1，即 V₁:V₂ = 20.00:20.00 → 此处 V₁ = 20.00，V₂ - V₁ = 20.00，所以 V₁ = V₂ - V₁ = 20.00 mL，即 V₁/V₂ = 1:1 ✅ 成立。故为纯 H₃PO₄。

(2) 计算浓度
第一计量点：H₃PO₄ → H₂PO₄⁻，消耗 NaOH 20.00 mL。
计量关系 1:1，所以：

$$c(\text{H}_3\text{PO}_4) = \frac{0.1000 \times 20.00}{25.00} = 0.08000 \text{ M}$$

验证：第二计量点消耗 NaOH 总计 40.00 mL，H₃PO₄ → HPO₄²⁻ 也是 1:1 关系（第二步也是一分子的 H⁺），验证通过。

(3) 第三级不可直接滴定原因：
$K_{\text{a}_3} = 4.4 \times 10^{-13}$，$cK_{\text{a}_3} = 0.0800 \times 4.4 \times 10^{-13} = 3.52 \times 10^{-14} \ll 10^{-8}$，不满足准确滴定条件，且 pH 突跃极小，无合适指示剂。

**答案**：(1) 纯 H₃PO₄；(2) c(H₃PO₄) = 0.08000 M；(3) cKa₃ ≪ 10⁻⁸，无突跃。

## 十、参考答案与提示

**1**. (a) NaOH(s)：❌，易吸收 CO₂ 和 H₂O；(b) 草酸：✅，纯度高、组成恒定；(c) 浓 HCl：❌，易挥发，浓度不恒定；(d) As₂O₃：✅，常用于标定 I₂/Na₂S₂O₃。

**2**. 理论变色范围 pH = 3.4 ± 1 = 2.4~4.4；实际 3.1~4.4。差异原因：人眼对红色比对黄色更敏感（需更高的 [In⁻]/[HIn] 比才能看到碱式色黄色），故实际范围偏向酸性一侧。

**3**. (a) $K_b = 10^{-14}/(1.3\times 10^{-5}) = 7.69\times 10^{-10}$，$c_{\text{sp}} = 0.05000$ M，[OH⁻] = √(7.69×10⁻¹⁰ × 0.05000) = 6.20×10⁻⁶，pOH = 5.21，pH = 8.79。
(b) 甲基红变色范围 4.4~6.2 完全不在突跃内 → ❌。
(c) 突跃估算：a = 0.999 时 pH ≈ pKa + 3 = 4.89 + 3 = 7.89；a = 1.001 时 pH = 9.70；突跃约 7.9~9.7，可用酚酞。

**4**. 若甲基橙终点 pHep = 4.0，则 ΔpH = 4.0 - 8.72 = -4.72。
分子 = 10⁻⁴·⁷² - 10⁴·⁷² ≈ 0 - 5.25×10⁴ ≈ -5.25×10⁴（负值，因指数项 10⁴·⁷² 远大于 10⁻⁴·⁷²，实际符号由 −10⁴·⁷² 主导，注意公式是对称的）。代入得 Et ≈ -550%，即误差超过 500%。显然甲基橙不适合。

**5**. V₁ = 18.50，V₂ = 15.40，V₁ > V₂ → 组成为 NaOH + Na₂CO₃。
NaOH 消耗 HCl = V₁ - V₂ = 3.10 mL。
Na₂CO₃ 消耗总 HCl = 2V₂ = 30.80 mL（或 V₁ + V₂ = 33.90 mL，但需细致：酚酞终点含 NaOH 全中和 + Na₂CO₃ 一半，所以 Na₂CO₃ 占一半的体积 = V₂，总 Na₂CO₃ 消耗 HCl = 2V₂ = 30.80 mL）。
ω(NaOH) = (0.2500 × 0.00310 × 40.00/1.2650) × 100% = 2.45%
ω(Na₂CO₃) = (0.2500 × 0.03080 × 106.0/1.2650) × 100% = 64.5%

**6**. (a) 酚酞终点消耗 HCl 24.50 mL，甲基橙终点消耗 25.00 mL。NaOH 吸收 CO₂ 后部分转化为 Na₂CO₃。
V₁(酚酞) = 24.50 mL：NaOH → NaCl + Na₂CO₃ → NaHCO₃
V₂(甲基橙) = 25.00 mL：NaHCO₃ → CO₂
V₂ - V₁ = 0.50 mL 代表 Na₂CO₃→NaHCO₃→CO₂ 的全过程。
NaOH 与 Na₂CO₃ 的物质的量关系：Na₂CO₃ 消耗 HCl = 2 × 0.50 = 1.00 mL。
NaOH 消耗 HCl = 24.50 - 0.50 = 24.00 mL（或 25.00 - 1.00 = 24.00 mL）。
c(NaOH) 表观 = 0.1000 × 24.00/25.00 = 0.09600 M（未吸收时原应为 0.1000 M）。
c(Na₂CO₃) = 0.1000 × 0.50/25.00 = 0.002000 M（注意 Na₂CO₃ 作为二元碱，1 mol Na₂CO₃ 消耗 2 mol H⁺）。
(b) 用该 NaOH 滴定 HAc（酚酞指示剂）：结果偏低。因为 NaOH 中部分有效碱已转化为 Na₂CO₃（碱性较弱），实际消耗的 NaOH 体积偏大，计算时仍按原浓度计算，使 HAc 含量偏低。

## 十一、速查卡

### 公式速查

| 公式 | 适用场景 |
|:---|:---|
| pH = pKa + lg([A⁻]/[HA]) | 缓冲溶液 pH（H-H 方程） |
| [H⁺] = √(Ka·c) | 一元弱酸 pH 最简式（c/Ka ≥ 100） |
| pH = ½(pKa₁ + pKa₂) | 多元酸两性物质计量点 pH |
| Et = (10^ΔpH - 10^-ΔpH) / √(c·Ka/Kw) | 林邦公式（弱酸滴定误差） |
| cKa ≥ 10⁻⁸ | 一元弱酸可滴性判据 |
| Ka₁/Ka₂ ≥ 10⁵ | 多元酸分步滴定判据 |

### 易错点速记

| 场景 | 常见错误 | 正确理解 |
|:---|:---|:---|
| 指示剂变色 | 变色范围 = 一个点 | 是区间 pH = pKa ± 1 |
| 甲醇橙变色 | 颜色变化 | 红→黄，中间经过橙色（过渡区） |
| 双指示剂酚酞终点 | 以为 CO₃²⁻ 全中和 | 只到 HCO₃⁻（一半） |
| 弱酸滴定突跃 | 以为突跃从滴定前开始 | 从缓冲区末端开始（约 a=0.999） |
| 林邦公式 ΔpH | 忘记符号规则 | 正→滴定剂过量，负→不足 |
| 标定计算 | 忘记稀释因子 | 分取/定容比必须乘入 |

```
基准物质 → 标准溶液 → 滴定曲线（强酸/弱酸/多元酸）
                               ↓
```
