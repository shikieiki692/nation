# 学生讲义：氧化还原滴定与沉淀滴定

> **适用**：第二轮提高班
> **对应备课大纲**：[[04-课件/备课大纲/2026-06-03-氧化还原滴定与沉淀滴定-提高班]]

> **深度边界**：本讲聚焦（1）碘量法与KMnO₄法的原理与计算；（2）莫尔法和Volhard法的指示剂原理与适用条件；（3）沉淀滴定三种方法的对比。不展开氧化还原滴定的完整电位理论、沉淀滴定的完整溶度积推导、Fajans法的吸附指示剂详细理论、氧化还原指示剂的完整电化学理论、KMnO₄在非酸性条件下的复杂反应。

## 学习目标

- ✅ 目标1：能区分直接碘量法与间接碘量法的适用场景，写出核心方程式 I₂ + 2S₂O₃²⁻ → 2I⁻ + S₄O₆²⁻ 并确认计量比 1:2。
- ✅ 目标2：能进行碘量法的滴定计算（包括KI浓度测定、Na₂S₂O₃标定），理解误差来源与控制方法。
- ✅ 目标3：理解KMnO₄法需在酸性条件下进行的原因，掌握"自指示剂"原理（紫色→无色/淡粉），能进行KMnO₄滴定Fe²⁺/H₂C₂O₄的计量计算。
- ✅ 目标4：理解莫尔法（K₂CrO₄作指示剂，分步沉淀原理）和Volhard法（Fe³⁺作指示剂，返滴定逻辑）的指示剂原理与酸度条件。
- ✅ 目标5：能对比Mohr法、Volhard法、Fajans法的适用条件与局限，根据待测物性质选择合适的沉淀滴定方法。
- ✅ 目标6：了解K₂Cr₂O₇法、Ce⁴⁺法、溴酸盐法的基本特点和典型应用。

## 一、氧化还原滴定概况

### 1.1 氧化还原滴定的基本条件

氧化还原滴定法是以氧化还原反应为基础的滴定分析方法。与酸碱滴定（质子转移）不同，氧化还原滴定基于**电子转移**，因此对反应条件和速度有更严格的要求。

适用于氧化还原滴定的反应必须满足以下条件：

| 条件 | 具体要求 | 示例 |
|:---|:---|:---|
| 反应完全 | $\lg K \geq 6$（对于1:1型反应），即 $\varphi_1^\ominus - \varphi_2^\ominus \geq 0.35\ \mathrm{V}$ | KMnO₄滴定Fe²⁺，$\Delta\varphi^\ominus = 1.51 - 0.77 = 0.74\ \mathrm{V} > 0.35\ \mathrm{V}$ |
| 反应速度快 | 瞬间完成；否则需加热或加催化剂 | KMnO₄滴定H₂C₂O₄需加热至70~85℃，Mn²⁺自催化 |
| 选择性高 | 标准溶液只与被测物反应，杂质不干扰 | KMnO₄法避免使用HCl介质（Cl⁻被诱导氧化） |
| 终点可判 | 有合适指示剂或自身颜色变化指示终点 | KMnO₄自身指示剂、淀粉-I₂专属指示剂 |

### 1.2 条件电位（条件电极电位）

实际分析中，溶液的离子强度和副反应会影响电对的电极电位，因此引入**条件电位** $\varphi^{\ominus'}$：

$$\varphi = \varphi^{\ominus'} + \frac{0.059\ \mathrm{V}}{z}\lg\frac{c_{\mathrm{Ox}}}{c_{\mathrm{Red}}}$$

其中 $\varphi^{\ominus'} = \varphi^{\ominus} + \frac{0.059\ \mathrm{V}}{z}\lg\frac{\gamma_{\mathrm{Ox}}\alpha_{\mathrm{Red}}}{\gamma_{\mathrm{Red}}\alpha_{\mathrm{Ox}}}$

条件电位综合了离子强度（$\gamma$）与副反应（$\alpha$）的影响，类似于配位滴定中的条件稳定常数 $K'$。

> **🔑 比赛关键数据**：在 $1\ \mathrm{mol\cdot L^{-1}}\ \mathrm{HCl}$ 中，$\varphi^{\ominus'}_{\mathrm{Fe^{3+}/Fe^{2+}}} = 0.68\ \mathrm{V}$；$\varphi^{\ominus'}_{\mathrm{Ce^{4+}/Ce^{3+}}} = 1.28\ \mathrm{V}$。

### 1.3 氧化还原滴定曲线与突跃

化学计量点电势计算公式（对称电对 $z_1 = z_2$）：

$$\varphi_{\mathrm{sp}} = \frac{z_1\varphi_1^{\ominus'} + z_2\varphi_2^{\ominus'}}{z_1 + z_2}$$

滴定突跃范围：$\varphi_2^{\ominus'} + \frac{0.059\times 3}{z_2}$ 到 $\varphi_1^{\ominus'} - \frac{0.059\times 3}{z_1}$

![[media/redox-titration-curve.svg|氧化还原滴定曲线示意图]]
*图 氧化还原滴定曲线示意图——以 Fe³⁺/Sn²⁺ 滴定为例，电位 E 随滴定分数 a 的变化。化学计量点附近电位发生突跃，突跃范围取决于两电对的条件电位差 Δφ°′。Δφ°′ ≥ 0.35 V 时突跃明显，可准确滴定（分析化学第六版 第7章）*

### 1.4 氧化还原指示剂

| 指示剂类型 | 原理 | 典型实例 |
|:---|:---|:---|
| **自身指示剂** | 滴定剂或被测物本身颜色变化 | KMnO₄（紫红→无色/淡粉） |
| **专属指示剂** | 与滴定体系中某组分形成有色配合物 | 淀粉-I₂（蓝色） |
| **氧化还原指示剂** | 氧化态/还原态颜色不同，变色范围 $\varphi_{\mathrm{In}}^{\ominus'} \pm \frac{0.059}{z}\ \mathrm{V}$ | 二苯胺磺酸钠、邻二氮菲-亚铁 |

> **选择原则**：指示剂的 $\varphi_{\mathrm{In}}^{\ominus'}$ 应尽量接近化学计量点电势 $\varphi_{\mathrm{sp}}$。

## 二、高锰酸钾法

### 2.1 基本原理

高锰酸钾法是利用 KMnO₄ 作为氧化剂进行滴定的方法。KMnO₄ 是强氧化剂，在不同 pH 条件下还原产物不同：

| 条件 | 半反应 | $\varphi^\ominus$ | 产物颜色 |
|:---|:---|:---:|:---:|
| 强酸性（常用） | $\mathrm{MnO_4^- + 8H^+ + 5e^- \rightarrow Mn^{2+} + 4H_2O}$ | 1.51 V | Mn²⁺无色 |
| 中性/弱碱性 | $\mathrm{MnO_4^- + 2H_2O + 3e^- \rightarrow MnO_2\downarrow + 4OH^-}$ | 0.59 V | MnO₂棕色沉淀 |
| 强碱性 | $\mathrm{MnO_4^- + e^- \rightarrow MnO_4^{2-}}$ | 0.56 V | MnO₄²⁻绿色 |

**核心原则**：竞赛中仅考**酸性条件**下的KMnO₄滴定，产物唯一为 Mn²⁺。

### 2.2 自身指示剂原理

KMnO₄ 溶液呈紫红色，而还原产物 Mn²⁺ 在稀溶液中几乎无色。滴定过程中，KMnO₄ 被不断还原为 Mn²⁺，溶液保持无色；化学计量点后，稍过量的 KMnO₄ 使溶液呈现**淡粉色**（约 $2\times10^{-6}\ \mathrm{mol\cdot L^{-1}}$ 即可观察到），指示终点到达。

> ⚠️ 淡粉色即为终点，**无需另加指示剂**。

### 2.3 标准溶液配制与标定

KMnO₄ 不能直接配制标准溶液，必须采用**间接法**：

- 称取稍过量的 KMnO₄ → 煮沸 1h → 放置 2~3 天 → 微孔玻璃漏斗过滤 → 棕色瓶暗处保存
- **基准物**：$\mathrm{Na_2C_2O_4}$（草酸钠），105~110℃烘干2h

**标定反应**（H₂SO₄介质，酸度 0.5~1 mol/L，70~85℃）：

$$2\mathrm{MnO_4^-} + 5\mathrm{C_2O_4^{2-}} + 16\mathrm{H^+} \rightarrow 2\mathrm{Mn^{2+}} + 10\mathrm{CO_2\uparrow} + 8\mathrm{H_2O}$$

**计量比**：$n(\mathrm{KMnO_4}) : n(\mathrm{Na_2C_2O_4}) = 2 : 5$

> ⚠️ **关键条件**：① 温度 70~85℃（过低反应慢，过高 H₂C₂O₄ 分解）；② 0.5~1 M H⁺；③ 最初几滴褪色慢（Mn²⁺ 自催化效应）；④ **不能用 HCl 或 HNO₃ 介质**（HCl 被氧化，HNO₃ 本身是氧化剂）。

### 2.4 典型应用

| 应用 | 反应（酸性条件） | 计量关系 |
|:---|:---|:---|
| 直接滴定 Fe²⁺ | $5\mathrm{Fe^{2+}} + \mathrm{MnO_4^-} + 8\mathrm{H^+} \rightarrow 5\mathrm{Fe^{3+}} + \mathrm{Mn^{2+}} + 4\mathrm{H_2O}$ | $n(\mathrm{Fe^{2+}}) : n(\mathrm{KMnO_4}) = 5 : 1$ |
| 直接滴定 H₂O₂ | $5\mathrm{H_2O_2} + 2\mathrm{MnO_4^-} + 6\mathrm{H^+} \rightarrow 5\mathrm{O_2\uparrow} + 2\mathrm{Mn^{2+}} + 8\mathrm{H_2O}$ | $n(\mathrm{H_2O_2}) : n(\mathrm{KMnO_4}) = 5 : 2$ |
| 直接滴定 H₂C₂O₄ | $5\mathrm{H_2C_2O_4} + 2\mathrm{MnO_4^-} + 6\mathrm{H^+} \rightarrow 10\mathrm{CO_2\uparrow} + 2\mathrm{Mn^{2+}} + 8\mathrm{H_2O}$ | $n(\mathrm{H_2C_2O_4}) : n(\mathrm{KMnO_4}) = 5 : 2$ |
| 间接测定 Ca²⁺ | Ca²⁺→CaC₂O₄↓→溶解→KMnO₄滴定C₂O₄²⁻ | $n(\mathrm{Ca^{2+}}) : n(\mathrm{KMnO_4}) = 5 : 2$ |

## 三、碘量法

### 3.1 基本反应与分类

碘量法的核心反应（**计量比 1:2——高亮重点！**）：

$$\mathrm{I_2 + 2S_2O_3^{2-} \rightarrow 2I^- + S_4O_6^{2-}}$$

| 类型 | 原理 | 适用对象 | $\varphi$ 条件 |
|:---:|:---|:---|:---:|
| **直接碘量法** | I₂ 作氧化剂，直接滴定还原性物质 | $\mathrm{Sn^{2+}, S^{2-}, SO_3^{2-}, As(III), VitC}$ | $\varphi < 0.545\ \mathrm{V}$ |
| **间接碘量法** | I⁻ 还原氧化剂→I₂，再用 Na₂S₂O₃ 滴定 I₂ | $\mathrm{Cu^{2+}, Cr_2O_7^{2-}, MnO_4^-, IO_3^-, H_2O_2, ClO^-}$ | $\varphi > 0.545\ \mathrm{V}$ |

### 3.2 直接碘量法

- **指示剂**：淀粉（与 I₂ 形成蓝色包合物，灵敏度极高）
- **条件**：弱酸性或中性（碱性中 I₂ 发生歧化）

### 3.3 间接碘量法

**核心流程**（三步计量链）：

$$\text{氧化剂} \xrightarrow{\text{过量 KI}} \mathrm{I_2} \xrightarrow{\mathrm{Na_2S_2O_3}} \text{滴定}\]

**定量关系**：$n(\text{氧化剂}) : n(\mathrm{I_2}) : n(\mathrm{S_2O_3^{2-}}) = a : b : 2b$

> **经典示例**：测定 Cu²⁺ 时，$2\mathrm{Cu^{2+} + 4I^- \rightarrow 2CuI\downarrow + I_2}$，$n(\mathrm{Cu^{2+}}) : n(\mathrm{Na_2S_2O_3}) = 2 : 2 = 1 : 1$

### 3.4 间接碘量法的关键条件控制

| 控制项 | 原因 | 解决方法 |
|:---|:---|:---|
| **pH 控制** | 强酸中 S₂O₃²⁻ 分解（S + SO₂↑）；强碱中 I₂ 歧化 | 维持弱酸性或中性（pH ≈ 4~8） |
| **I₂ 挥发损失** | I₂ 易升华，导致结果偏低 | 加过量 KI 生成 I₃⁻（$I_2 + I^- \rightleftharpoons I_3^-$）；使用碘量瓶；低温操作；避免剧烈摇动 |
| **I⁻ 空气氧化** | $4\mathrm{I^-} + \mathrm{O_2} + 4\mathrm{H^+} \rightarrow 2\mathrm{I_2} + 2\mathrm{H_2O}$，导致结果偏高 | 避光操作；避免过强酸性；析出 I₂ 后立即滴定 |
| **淀粉加入时机** | 过早加入会包裹 I₂，形成难解离的包合物，终点滞后 | **间接碘量法在临近终点时**（溶液呈浅黄色）再加入淀粉 |

> 🧠 **记忆口诀**："间接碘量终点加，直接碘量随时加。"

### 3.5 经典应用——铜合金中铜的测定

$$2\mathrm{Cu^{2+}} + 4\mathrm{I^-} \rightarrow 2\mathrm{CuI\downarrow} + \mathrm{I_2}$$
$$\mathrm{I_2} + 2\mathrm{S_2O_3^{2-}} \rightarrow 2\mathrm{I^-} + \mathrm{S_4O_6^{2-}}$$

**关键操作**：近终点时加入 NH₄SCN，使 CuI 转化为溶解度更小的 CuSCN，释放被吸附的 I₂：

$$\mathrm{CuI + SCN^- \rightarrow CuSCN\downarrow + I^-}$$

## 四、重铬酸钾法与其他氧化还原滴定方法

### 4.1 重铬酸钾法（K₂Cr₂O₇ 法）

**半反应**：$\mathrm{Cr_2O_7^{2-} + 14H^+ + 6e^- \rightarrow 2Cr^{3+} + 7H_2O}$，$\varphi^\ominus = 1.33\ \mathrm{V}$

**突出优点**：
- ✅ K₂Cr₂O₇ 易提纯（140~250℃干燥后），**可直接配制标准溶液**
- ✅ 溶液非常稳定，可长期保存
- ✅ 在 1 mol/L HCl 中室温下不与 Cl⁻ 作用（优于 KMnO₄ 法）

**典型应用——全铁测定**：

$$\mathrm{Cr_2O_7^{2-} + 6Fe^{2+} + 14H^+ \rightarrow 2Cr^{3+} + 6Fe^{3+} + 7H_2O}$$

**操作流程**：铁矿样 → 酸溶 → SnCl₂ 还原 Fe³⁺→Fe²⁺ → HgCl₂ 氧化过量 Sn²⁺ → H₃PO₄-H₂SO₄ 介质中 K₂Cr₂O₇ 滴定

> 💡 **H₃PO₄ 的作用**：与 Fe³⁺ 生成无色 $[\mathrm{Fe(HPO_4)_2}]^-$，降低 Fe³⁺/Fe²⁺ 电势增大突跃，同时消除 Fe³⁺ 黄色对终点观察的干扰。

### 4.2 三种主要氧化还原滴定方法对比

| 对比项 | KMnO₄ 法 | K₂Cr₂O₇ 法 | 碘量法 |
|:---|:---:|:---:|:---:|
| $\varphi^\ominus$ | 1.51 V（强） | 1.33 V（中） | 0.545 V（弱） |
| 标准溶液配制 | 间接配制，需标定 | **直接配制** | Na₂S₂O₃ 间接配制 |
| 溶液稳定性 | 不稳定，易分解 | 非常稳定 | Na₂S₂O₃ 不稳定 |
| 指示剂 | **自身指示剂**（紫红→无色） | 氧化还原指示剂（二苯胺磺酸钠等） | **专属指示剂**（淀粉-I₂ 蓝色） |
| 与 Cl⁻ 关系 | 有诱导反应干扰 | 室温下不反应 | — |
| 选择性 | 差（氧化性太强） | 较好 | 好 |
| 主要应用 | Fe²⁺、H₂O₂、Ca²⁺（间接）、COD | 全铁测定、COD | Cu²⁺、漂白粉、有机物 |

### 4.3 Ce⁴⁺法与溴酸盐法（了解）

| 方法 | 半反应 | $\varphi^\ominus$ | 特点 |
|:---|:---|:---:|:---|
| **Ce⁴⁺ 法** | $\mathrm{Ce^{4+} + e^- \rightarrow Ce^{3+}}$ | 1.61 V | 氧化性强，稳定性好，可用**邻二氮菲-亚铁**作指示剂 |
| **溴酸盐法** | $\mathrm{BrO_3^- + 6H^+ + 6e^- \rightarrow Br^- + 3H_2O}$ | 1.44 V | 可测定 As(III)、Sb(III)、I⁻ 等，常与碘量法联用 |

## 五、沉淀滴定

### 5.1 沉淀滴定概述

沉淀滴定法是以**沉淀反应**为基础的容量分析方法。目前应用最为广泛的是以 AgNO₃ 为滴定剂的**银量法**（Argentometric Methods），用于测定卤素离子（Cl⁻、Br⁻、I⁻）和拟卤素离子（SCN⁻、CN⁻）等。

**银量法的三种方法对比如下**：

| 项目 | **Mohr 法（莫尔法）** | **Volhard 法（佛尔哈德法）** | **Fajans 法（法扬司法）** |
|:---|:---|:---|:---|
| 指示剂 | K₂CrO₄（铬酸钾） | 铁铵矾（Fe³⁺） | 吸附指示剂（荧光素等） |
| 滴定剂 | AgNO₃ | NH₄SCN（或 KSCN） | AgNO₃（或 NaCl） |
| 滴定方式 | **直接滴定** | **返滴定**（测卤素） | 直接滴定 |
| 酸度条件 | pH **6.5~10.5**（中性/弱碱） | **强酸性**（0.1~1 M HNO₃） | 取决于指示剂（pH 2~10） |
| 测定对象 | Cl⁻、Br⁻、CN⁻ | Ag⁺；**返滴定测 Cl⁻、Br⁻、I⁻** | Cl⁻、Br⁻、I⁻、SCN⁻ |
| 终点现象 | 白色沉淀→**砖红色**沉淀 | 白色沉淀→**红色**配合物 | 沉淀表面**颜色突变** |

### 5.2 Mohr 法（莫尔法）

**原理**：以 K₂CrO₄ 为指示剂，用 AgNO₃ 标准溶液直接滴定 Cl⁻（或 Br⁻）。

**两步沉淀反应**：
1. （主反应）$\mathrm{Ag^+ + Cl^- \rightarrow AgCl\downarrow}$（白色，$K_{\mathrm{sp}} = 1.8\times10^{-10}$）
2. （终点反应）$2\mathrm{Ag^+ + CrO_4^{2-} \rightarrow Ag_2CrO_4\downarrow}$（砖红色，$K_{\mathrm{sp}} = 1.1\times10^{-12}$）

**分步沉淀原理**：因为 $K_{\mathrm{sp}}(\mathrm{AgCl})$ 很小，Ag⁺ 先与 Cl⁻ 生成白色 AgCl 沉淀；当 Cl⁻ 被完全沉淀后，稍过量的 Ag⁺ 与 CrO₄²⁻ 反应生成**砖红色** Ag₂CrO₄ 沉淀，指示终点。

**K₂CrO₄ 指示剂浓度**：理论计算 [CrO₄²⁻] ≈ $1.1\times10^{-2}\ \mathrm{mol/L}$，实际使用约 $5.0\times10^{-3}\ \mathrm{mol/L}$（颜色适中）。

**条件限制**：
- ⚠️ **酸度控制**：pH 6.5~10.5。酸性过强：CrO₄²⁻ 转化为 Cr₂O₇²⁻（$2\mathrm{CrO_4^{2-}} + 2\mathrm{H^+} \rightarrow \mathrm{Cr_2O_7^{2-}} + \mathrm{H_2O}$），Ag₂CrO₄ 沉淀过晚；碱性过强：Ag⁺ 形成 AgOH/Ag₂O 沉淀。
- ❌ 干扰阴离子：PO₄³⁻、AsO₄³⁻、SO₃²⁻、C₂O₄²⁻（与 Ag⁺ 生成沉淀）
- ❌ 干扰阳离子：Ba²⁺、Pb²⁺（与 CrO₄²⁻ 生成沉淀）
- ❌ 有色离子 Cu²⁺、Co²⁺、Ni²⁺ 影响终点观察
- ❌ 不能有 NH₃（与 Ag⁺ 配位）

### 5.3 Volhard 法（佛尔哈德法）

**原理**：以 Fe³⁺（铁铵矾）为指示剂，在**强酸性**介质中用 NH₄SCN（或 KSCN）滴定 Ag⁺。

**直接滴定 Ag⁺**：
1. $\mathrm{Ag^+ + SCN^- \rightarrow AgSCN\downarrow}$（白色）
2. $\mathrm{Fe^{3+} + SCN^- \rightarrow [Fe(SCN)]^{2+}}$（红色，终点指示）

**返滴定测定 Cl⁻（重点⚠️）**：

$$\text{Cl}^- \xrightarrow{\text{加过量 AgNO}_3} \text{AgCl↓ + 剩余 Ag}^+ \xrightarrow{\text{NH}_4\text{SCN 滴定}} \text{由剩余 Ag}^+ \text{推算 Cl}^-$$

**操作步骤**：① 加入过量 AgNO₃ 标准溶液（使 Cl⁻ 完全沉淀）→ ② 用 NH₄SCN 标准溶液滴定剩余的 Ag⁺ → ③ 终点时 Fe³⁺ 与 SCN⁻ 生成红色配合物

**计算逻辑**：$n(\mathrm{Cl^-}) = n(\mathrm{AgNO_3}_{\text{总}}) - n(\mathrm{AgNO_3}_{\text{剩余}}) = c_{\mathrm{AgNO_3}}V_{\mathrm{AgNO_3}} - c_{\mathrm{NH_4SCN}}V_{\mathrm{NH_4SCN}}$

> 🧠 **记忆口诀**："总量 - 剩余量 = 反应量"——**返滴定的核心账本**。

**为什么必须在强酸性条件下？**
- 防止 Fe³⁺ 水解生成 Fe(OH)₃ 沉淀：$\mathrm{Fe^{3+} + 3H_2O \rightleftharpoons Fe(OH)_3\downarrow + 3H^+}$
- 通常用 **0.1~1 M HNO₃** 介质

**Volhard 法返滴定 Cl⁻ 的特别注意事项（高频考点⚠️）**：

由于 $K_{\mathrm{sp}}(\mathrm{AgSCN}) = 1.0\times10^{-12} < K_{\mathrm{sp}}(\mathrm{AgCl}) = 1.8\times10^{-10}$，终点附近会发生**沉淀转化**：

$$\mathrm{AgCl\downarrow + SCN^- \rightleftharpoons AgSCN\downarrow + Cl^-}$$

这导致终点红色出现后又会褪去，需要不断加入 SCN⁻ 才能维持红色——**使结果偏高**。

**解决方法**：
1. 煮沸凝聚 AgCl 后**过滤除去**
2. 加入**硝基苯**或 1,2-二氯乙烷包裹 AgCl 沉淀，阻止 SCN⁻ 接触

> 💡 测定 Br⁻ 和 I⁻ 时无此问题（AgBr、AgI 溶解度小于 AgSCN，不会被置换）。

### 5.4 Fajans 法（法扬司法）——了解

以**吸附指示剂**指示终点。化学计量点前，沉淀吸附过量 Cl⁻ 带负电，排斥指示剂阴离子；化学计量点后，沉淀吸附过量 Ag⁺ 带正电，吸附指示剂阴离子，颜色改变。

**卤化银对离子的吸附能力顺序**：
$$\mathrm{I^- > SCN^- > Br^- > 曙红 > Cl^- > 荧光素}$$

**常用吸附指示剂**：

| 指示剂 | 适用对象 | 适用 pH |
|:---|:---|:---:|
| 荧光素 | Cl⁻ | pH 7~10 |
| 二氯荧光素 | Cl⁻ | pH 4~10 |
| 曙红 | Br⁻、I⁻、SCN⁻ | pH 2~10 |

> ⚠️ 曙红不能用于滴定 Cl⁻（Cl⁻ 吸附能力弱于曙红，终点前即被吸附，终点提前）。

### 5.5 三种沉淀滴定法终局对比表

| 对比维度 | Mohr 法 | Volhard 法 | Fajans 法 |
|:---|:---|:---|:---|
| **原理类别** | 分步沉淀 | 返滴定+有色配合物 | 吸附指示剂 |
| **酸度优势** | 中性（不伤器皿） | 强酸（弱酸根不干扰） | 可调宽（取决于指示剂） |
| **酸度局限** | pH<6 时 CrO₄²⁻→Cr₂O₇²⁻ | 必须强酸，否则 Fe³⁺ 水解 | 阴离子型需控 pH 使指示剂电离 |
| **直接/返滴定** | 直接滴定 | 返滴定（测卤素时） | 直接滴定 |
| **主要局限** | 干扰多（许多离子干扰） | 返滴定 Cl⁻ 有沉淀转化误差 | 浓度不能太稀（>0.005 mol/L） |
| **选择性** | 较差 | **最好**（强酸抑制弱酸根干扰） | 居中 |
| **对 NH₃ 耐受** | ❌ 有干扰 | ✅ 无干扰 | ✅ |

### 例1 碘量法计量比——电子守恒推导 ⭐⭐⭐

**题干**：写出间接碘量法中 I₂ 与 Na₂S₂O₃ 反应的离子方程式，并用双线桥标出电子转移方向，确认计量比。

**思路**：先确定 I₂ 是氧化剂（I：0→-1），S₂O₃²⁻ 是还原剂（S：+2→+2.5）。

**解析**：

**步骤1**：写出反应物和产物
$$\mathrm{I_2 + S_2O_3^{2-} \rightarrow I^- + S_4O_6^{2-}}$$

**步骤2**：确定化合价变化
- I：0 → -1，每个 I 得 1e⁻，I₂ 共得 2e⁻
- S：+2 → +2.5，每个 S 失 0.5e⁻，S₂O₃²⁻ 中 2 个 S 共失 1e⁻

**步骤3**：电子守恒配平
- 1 个 I₂ 得 2e⁻，1 个 S₂O₃²⁻ 失 1e⁻
- 需要 2 个 S₂O₃²⁻ 才能提供 2e⁻

$$\mathrm{I_2 + 2S_2O_3^{2-} \rightarrow 2I^- + S_4O_6^{2-}}$$

**步骤4**：确认计量比
$$n(\mathrm{I_2}) : n(\mathrm{Na_2S_2O_3}) = \mathbf{1 : 2}$$

> ⚠️ **易错**：学生常凭直觉记为 1:1（一个 I₂ 对一个 S₂O₃²⁻），但电子守恒要求 1:2！

**答案**：$\mathrm{I_2 + 2S_2O_3^{2-} = 2I^- + S_4O_6^{2-}}$，计量比 $1:2$。

### 例2 碘量法测定 KI 浓度（39届Q2改编）⭐⭐⭐

**题干**：准确称取 0.5000 g 工业 KI 样品，溶解后置于碘量瓶中，加入 10 mL 1 mol/L H₂SO₄ 酸化后，加入 20.00 mL 0.05000 mol/L KIO₃ 标准溶液，充分反应后生成 I₂。然后用 0.1000 mol/L Na₂S₂O₃ 标准溶液滴定至淀粉终点，消耗 14.50 mL。
已知：$\mathrm{IO_3^- + 5I^- + 6H^+ \rightarrow 3I_2 + 3H_2O}$
求：该工业 KI 样品中 KI 的质量分数。

**思路**：这是一个**置换滴定 + 间接碘量法**的综合题，涉及两步反应，关键是建立从 KI → Na₂S₂O₃ 的完整计量链。

**解析**：

**步骤1**：写出两步反应方程式
① 置换反应：$\mathrm{IO_3^- + 5I^- + 6H^+ \rightarrow 3I_2 + 3H_2O}$
② 滴定反应：$\mathrm{I_2 + 2S_2O_3^{2-} \rightarrow 2I^- + S_4O_6^{2-}}$

**步骤2**：建立完整计量链
$$\mathrm{KIO_3 \sim 3I_2 \sim 6Na_2S_2O_3}$$
即：$n(\mathrm{KIO_3}) : n(\mathrm{Na_2S_2O_3}) = 1 : 6$

但这里 KIO₃ 是标准溶液，**与样品中 KI 反应**后生成 I₂，其中生成的 I₂ 来自两个来源：
- 样品中 KI 的 I⁻ 被 KIO₃ 氧化
- KIO₃ 自身的 I(+V) 也被还原为 I₂

**步骤3**：计算总生成的 I₂ 量与 Na₂S₂O₃ 的消耗关系
总 I₂ 消耗 Na₂S₂O₃：
$$n(\mathrm{Na_2S_2O_3}) = 0.1000 \times 14.50 \times 10^{-3} = 1.450 \times 10^{-3}\ \mathrm{mol}$$

由计量关系 $3\mathrm{I_2} \sim 6\mathrm{Na_2S_2O_3}$，即 $n(\mathrm{I_2}) = \frac{1}{2} n(\mathrm{Na_2S_2O_3})$：
$$n(\mathrm{I_2})_{\text{总}} = \frac{1}{2} \times 1.450 \times 10^{-3} = 7.250 \times 10^{-4}\ \mathrm{mol}$$

**步骤4**：计算 KIO₃ 自身产生的 I₂
加入的 KIO₃：$n(\mathrm{KIO_3}) = 0.05000 \times 20.00 \times 10^{-3} = 1.000 \times 10^{-3}\ \mathrm{mol}$
由 $\mathrm{IO_3^- \sim 3I_2}$，KIO₃ 生成的 I₂：$n(\mathrm{I_2})_{\text{KIO₃}} = 3 \times 1.000 \times 10^{-3} = 3.000 \times 10^{-3}\ \mathrm{mol}$

> 💡 注意观察！KIO₃ 产生的 I₂（$3.000\times10^{-3}$ mol）**大于**总消耗的 I₂（$7.250\times10^{-4}$ mol）——这说明 KIO₃ 是**过量**的，溶液中剩余的 I⁻ 来自 KI 样品的量需要用差量法反推。

等等，这里需要重新思考——KIO₃ 与 I⁻ 的反应是：IO₃⁻ 将 I⁻ 氧化为 I₂。在这个反应中，IO₃⁻ 不仅氧化了样品中的 I⁻，自身也转化为 I₂。实际上，我们要找的是**样品中 KI 的 I⁻** 消耗了多少。

**重新分析计量关系**：

在反应 $\mathrm{IO_3^- + 5I^- + 6H^+ \rightarrow 3I_2 + 3H_2O}$ 中：
- 1 个 IO₃⁻ 与 5 个 I⁻ 反应生成 3 个 I₂
- 即 $n(\text{样品 I⁻}) : n(\mathrm{IO_3^-}) = 5 : 1$
- $n(\text{样品 I⁻}) : n(\mathrm{I_2}) = 5 : 3$

总 I₂ 来自 KIO₃ 与样品 I⁻ 的反应（即两步反应在这里耦合了）。

由总 I₂ 摩尔数 $7.250\times10^{-4}$ mol，和初始 KIO₃ 的摩尔数 $1.000\times10^{-3}$ mol，我们可以通过反应方程式列方程组。

设样品 I⁻ 为 $x$ mol，则：
$$\mathrm{IO_3^- + 5I^- + 6H^+ \rightarrow 3I_2 + 3H_2O}$$

实际上，这个反应是一个反应，不是分步的。IO₃⁻ 与 I⁻ 之间的计量关系是固定的。

但从另一个角度看——滴定的 I₂ 等于反应生成的 I₂：
- 初始 KIO₃：$1.000\times10^{-3}$ mol
- 这些 KIO₃ 能氧化多少 I⁻？取决于样品中 I⁻ 的量。

更直接的解法——**物料守恒法**：

由反应 $\mathrm{IO_3^- + 5I^- + 6H^+ \rightarrow 3I_2 + 3H_2O}$：
$$\text{总生成 I₂ 的量} = 3 \times n(\text{反应的 KIO₃})$$

而 Na₂S₂O₃ 滴定的结果是总 I₂：
$$n(\mathrm{I_2})_{\text{总}} = 7.250\times10^{-4}\ \mathrm{mol}$$

由反应计量：$n(\mathrm{I_2}) : n(\mathrm{IO_3^-}) = 3 : 1$
所以消耗的 KIO₃ 的量：$n(\mathrm{KIO_3})_{\text{消耗}} = \frac{1}{3} \times 7.250\times10^{-4} = 2.417\times10^{-4}\ \mathrm{mol}$

由反应计量：$n(\mathrm{I^-}) : n(\mathrm{IO_3^-}) = 5 : 1$
所以样品中 I⁻ 的物质的量：$n(\mathrm{I^-}) = 5 \times 2.417\times10^{-4} = 1.208\times10^{-3}\ \mathrm{mol}$

**步骤5**：计算 KI 的质量分数
$m_{\mathrm{KI}} = 1.208\times10^{-3} \times 166.0 = 0.2006\ \mathrm{g}$
$$w_{\mathrm{KI}} = \frac{0.2006}{0.5000} \times 100\% = 40.12\%$$

**答案**：$w_{\mathrm{KI}} = 40.12\%$。

### 例3 KMnO₄ 滴定 Fe²⁺ 计算 ⭐⭐⭐

**题干**：某铁矿石样品 0.2500 g 经酸溶和还原处理后，Fe³⁺ 全部转化为 Fe²⁺。用 0.02000 mol/L KMnO₄ 标准溶液滴定，消耗 26.50 mL。已知反应：
$$\mathrm{MnO_4^- + 8H^+ + 5e^- \rightarrow Mn^{2+} + 4H_2O}$$
$$\mathrm{Fe^{2+} \rightarrow Fe^{3+} + e^-}$$
求该铁矿石中铁的质量分数。

**思路**：写出总反应方程式 → 确定计量比 → 代入计算。

**解析**：

**步骤1**：写出总反应方程式
$$5\mathrm{Fe^{2+}} + \mathrm{MnO_4^-} + 8\mathrm{H^+} \rightarrow 5\mathrm{Fe^{3+}} + \mathrm{Mn^{2+}} + 4\mathrm{H_2O}$$

**计量比**：$n(\mathrm{Fe^{2+}}) : n(\mathrm{KMnO_4}) = 5 : 1$

**步骤2**：计算 Fe²⁺ 的物质的量
$$n(\mathrm{KMnO_4}) = 0.02000 \times 26.50 \times 10^{-3} = 5.300 \times 10^{-4}\ \mathrm{mol}$$
$$n(\mathrm{Fe^{2+}}) = 5 \times 5.300 \times 10^{-4} = 2.650 \times 10^{-3}\ \mathrm{mol}$$

**步骤3**：计算铁的质量分数
$$m_{\mathrm{Fe}} = 2.650 \times 10^{-3} \times 55.85 = 0.1480\ \mathrm{g}$$
$$w_{\mathrm{Fe}} = \frac{0.1480}{0.2500} \times 100\% = 59.20\%$$

**答案**：$w_{\mathrm{Fe}} = 59.20\%$。

### 例4 Mohr 法测定 NaCl 浓度 ⭐⭐⭐

**题干**：取 25.00 mL 含 NaCl 的试液，以 K₂CrO₄ 为指示剂，用 0.1000 mol/L AgNO₃ 溶液滴定，消耗 22.50 mL 到达砖红色终点。求试液中 NaCl 的浓度（mol/L）。

**思路**：Mohr 法直接滴定的计量关系为 1:1。

**解析**：

**步骤1**：写出反应方程式
$$\mathrm{Ag^+ + Cl^- \rightarrow AgCl\downarrow}$$

**计量关系**：$n(\mathrm{AgNO_3}) = n(\mathrm{NaCl})$

**步骤2**：计算
$$n(\mathrm{AgNO_3}) = 0.1000 \times 22.50 \times 10^{-3} = 2.250 \times 10^{-3}\ \mathrm{mol}$$
$$c(\mathrm{NaCl}) = \frac{2.250 \times 10^{-3}}{25.00 \times 10^{-3}} = 0.09000\ \mathrm{mol/L}$$

**步骤3**：判断指示剂条件
若该溶液的 pH < 6，则 CrO₄²⁻ 会转化为 Cr₂O₇²⁻，导致 Ag₂CrO₄ 沉淀不明显，结果偏高。因此测定前应确认溶液 pH 在 6.5~10.5 之间。

**答案**：$c(\mathrm{NaCl}) = 0.09000\ \mathrm{mol/L}$。

### 例5 Volhard 法返滴定测定 Cl⁻（含误差分析）⭐⭐⭐⭐

**题干**：用 Volhard 法测定某 NaCl 试液中的 Cl⁻ 含量。取 20.00 mL 试液，加入 30.00 mL 0.1000 mol/L AgNO₃ 标准溶液（过量），待沉淀完全后用 0.08000 mol/L NH₄SCN 标准溶液滴定剩余的 Ag⁺，消耗 12.50 mL 到达红色终点。
(1) 求试液中 Cl⁻ 的浓度。
(2) 若在滴定剩余 Ag⁺ 前未过滤除去 AgCl 沉淀，结果会偏高还是偏低？为什么？

**思路**：返滴定核心是"总量 - 剩余量 = 反应量"。

**解析**：

**(1) 计算 Cl⁻ 浓度**

总 Ag⁺ 加入量：
$$n(\mathrm{Ag^+})_{\text{总}} = 0.1000 \times 30.00 \times 10^{-3} = 3.000 \times 10^{-3}\ \mathrm{mol}$$

剩余 Ag⁺ 的量（被 NH₄SCN 滴定的部分）：
$$n(\mathrm{Ag^+})_{\text{剩余}} = 0.08000 \times 12.50 \times 10^{-3} = 1.000 \times 10^{-3}\ \mathrm{mol}$$

与 Cl⁻ 反应的 Ag⁺：
$$n(\mathrm{Ag^+})_{\text{反应}} = n(\mathrm{Ag^+})_{\text{总}} - n(\mathrm{Ag^+})_{\text{剩余}} = 3.000\times10^{-3} - 1.000\times10^{-3} = 2.000\times10^{-3}\ \mathrm{mol}$$

Cl⁻ 浓度：
$$c(\mathrm{Cl^-}) = \frac{2.000\times10^{-3}}{20.00\times10^{-3}} = 0.1000\ \mathrm{mol/L}$$

**(2) 误差分析**

若未过滤除去 AgCl 沉淀，结果会**偏高**。

原因：$K_{\mathrm{sp}}(\mathrm{AgSCN}) = 1.0\times10^{-12} < K_{\mathrm{sp}}(\mathrm{AgCl}) = 1.8\times10^{-10}$，终点附近 NH₄SCN 会置换 AgCl 中的 Cl⁻：
$$\mathrm{AgCl\downarrow + SCN^- \rightleftharpoons AgSCN\downarrow + Cl^-}$$

这导致滴定的 SCN⁻ 一部分用于置换 AgCl 而非滴定剩余 Ag⁺，使消耗的 NH₄SCN 体积比理论值大，计算出的剩余 Ag⁺ 偏高，从而反应掉的 Ag⁺ 偏低——**Cl⁻ 浓度偏低（负误差）**。

> 💡 此处是高频易错点：误差方向取决于具体的测量逻辑——在 Volhard 法返滴定 Cl⁻ 中，沉淀转化使 SCN⁻ 用量偏大 → 计算剩余 Ag⁺ 偏大 → Cl⁻ 结果**偏低**（负误差）。

**答案**：$c(\mathrm{Cl^-}) = 0.1000\ \mathrm{mol/L}$；若未过滤除去 AgCl 沉淀，Cl⁻ 结果偏低（负误差）。

## 七、常见误区

### 误区1：I₂ 与 Na₂S₂O₃ 的计量比记成 1:1

- **表现**：学生凭直觉认为一个 I₂ 分子对一个 S₂O₃²⁻ 分子。
- **纠正**：通过电子守恒推导——I₂ 得 2e⁻，每个 S₂O₃²⁻ 失 1e⁻（S 从 +2→+2.5），因此需要 2 个 S₂O₃²⁻。
- **记忆法**："碘遇硫代，一比二来"。

### 误区2：混淆 KMnO₄ 法和碘量法的酸度条件

- **表现**：认为 KMnO₄ 可以在任何条件下滴定；间接碘量法也可以强酸性操作。
- **纠正**：KMnO₄ 滴定必须在**强酸性**（H₂SO₄介质）进行，否则产物不唯一（有 MnO₂ 沉淀）；间接碘量法必须在**弱酸性或中性**进行，强酸性下 S₂O₃²⁻ 分解。
- **对比**：KMnO₄ 法——要强酸；碘量法——防强酸。

### 误区3：Mohr 法和 Volhard 法的酸度条件混为一谈

- **表现**：用 Mohr 法在中性条件下测 Cl⁻，却误以为 Volhard 法也在中性或碱性条件下操作。
- **纠正**：Mohr 法中 CrO₄²⁻ 在酸性条件下转化为 Cr₂O₇²⁻，故必须 pH 6.5~10.5；Volhard 法中 Fe³⁺ 在近中性会水解，故必须强酸性（0.1~1 M HNO₃）。
- **记忆法**："Mohr 中性直接滴，Volhard 强酸返滴"。

### 误区4：混淆 Volhard 法返滴定 Cl⁻ 中沉淀转化的误差方向

- **表现**：认为沉淀转化使 Cl⁻ 结果偏高（或判断方向错误）。
- **纠正**：沉淀转化使 SCN⁻ 不仅滴定剩余 Ag⁺ 还置换 AgCl，SCN⁻ 消耗体积偏大 → 计算剩余 Ag⁺ 偏大 → Cl⁻ 计算结果**偏低**（负误差）。
- **解决方法**：过滤除去 AgCl 或加硝基苯包裹。

### 误区5：KMnO₄ 标定中忘记温度控制

- **表现**：认为室温下直接用 Na₂C₂O₄ 标定 KMnO₄。
- **纠正**：室温下 MnO₄⁻ 与 C₂O₄²⁻ 反应极慢，必须加热至 70~85℃，且最初几滴 Mn²⁺ 自催化后才能加速。
- **记忆法**："七十度以上，草酸才被高锰酸氧化"。

### 误区6：返滴定计算时直接用消耗体积对应待测物

- **表现**：将 Volhard 法返滴定中 NH₄SCN 的消耗体积直接与 Cl⁻ 建立关系，忽略"总量 - 剩余量"的中间步骤。
- **纠正**：牢记"**总量 − 剩余量 = 反应量**"的返滴定账本法。

## 八、方法总结

### 8.1 氧化还原滴定方法选择决策树

```text
待测物是什么类型？
├── 还原性物质（Fe²⁺、H₂O₂、C₂O₄²⁻等）→ KMnO₄ 法（强酸性，自指示剂）
│   └── 需避免 Cl⁻ 干扰 → 改用 K₂Cr₂O₇ 法或 Ce⁴⁺ 法
├── 氧化性物质（Cu²⁺、Cr₂O₇²⁻、MnO₄⁻等）→ 间接碘量法
│   （弱酸性，淀粉指示剂终点加）
└── 还原性弱还原剂（Sn²⁺、S²⁻、As(III)等）→ 直接碘量法
    （中性/弱酸性，淀粉指示剂随时加）
```

### 8.2 沉淀滴定方法选择决策树

```text
测定卤素离子
├── Cl⁻ 且无干扰 → Mohr 法（中性，K₂CrO₄ 指示剂）
│   └── 有干扰（PO₄³⁻、有色离子等）→ Volhard 法返滴定
├── Br⁻ / I⁻ → Volhard 法返滴定（强酸性，Fe³⁺ 指示剂）
└── 低浓度 Cl⁻ → Fajans 法（选合适吸附指示剂）
    └── Cl⁻ 浓度 > 0.005 mol/L

测定 Ag⁺
└── Volhard 法直接滴定（强酸性，Fe³⁺ 指示剂）
```

### 8.3 滴定分析通用解题框架

1. **写出配平的化学反应方程式**（氧化还原反应先配电子）
2. **确定物质的量计量关系**（核心：谁和谁比）
3. **代入已知数据列式计算**（$cV = n$ 关系式）
4. **注意单位统一**（mL 换算为 L，或直接用 mmol）
5. **最后检查有效数字**

## 九、核心速查卡

| 方法 | 关键条件 | 指示剂 | 终点颜色 | 核心计量比 | 主要误差 |
|:---|:---|:---|:---:|:---:|:---|
| **KMnO₄ 法** | 强酸性（H₂SO₄） | 自身指示剂 | 无色→淡粉 | Fe²⁺:MnO₄⁻ = 5:1 | Cl⁻ 诱导氧化、温度不适 |
| **直接碘量法** | 中性/弱酸性 | 淀粉 | 无色→蓝色 | I₂:S₂O₃²⁻ = 1:2 | I₂ 挥发 |
| **间接碘量法** | 弱酸性/中性 | 淀粉（终点加） | 蓝色→无色 | 见计量链 | I₂ 挥发、I⁻ 氧化、淀粉加早 |
| **K₂Cr₂O₇ 法** | 酸性（HCl 也可） | 二苯胺磺酸钠 | 绿→紫 | Cr₂O₇²⁻:Fe²⁺ = 1:6 | — |
| **Mohr 法** | pH 6.5~10.5 | K₂CrO₄ | 白色→砖红色 | Ag⁺:Cl⁻ = 1:1 | CrO₄²⁻ 酸度影响、干扰离子 |
| **Volhard 法** | 强酸性（HNO₃） | Fe³⁺（铁铵矾） | 白色→红色 | 见返滴定公式 | 沉淀转化（Cl⁻）|
| **Fajans 法** | 取决于指示剂 | 吸附指示剂 | 沉淀表面变色 | Ag⁺:X⁻ = 1:1 | 浓度太稀、指示剂选错 |

## 十一、参考答案与提示

**1**. (a) 直接碘量法（Sn²⁺ 为还原剂）；(b) 间接碘量法（Cu²⁺ 氧化 I⁻ 生成 I₂）；(c) 间接碘量法（MnO₄⁻ 氧化 I⁻ 生成 I₂）；(d) 间接碘量法（H₂O₂ 氧化 I⁻ 生成 I₂）

**2**. 酸性条件下 MnO₄⁻ 才还原为 Mn²⁺（无色，$\varphi^\ominus=1.51$ V）；中性条件下生成 MnO₂ 棕色沉淀（$\varphi^\ominus=0.59$ V），干扰终点观察且使计量关系不确定。

**3**. 主反应：$\mathrm{Ag^+ + Cl^- \rightarrow AgCl\downarrow}$（白色，$K_{\mathrm{sp}}=1.8\times10^{-10}$）；终点反应：$2\mathrm{Ag^+ + CrO_4^{2-} \rightarrow Ag_2CrO_4\downarrow}$（砖红色，$K_{\mathrm{sp}}=1.1\times10^{-12}$）。计算化学计量点时 [Cl⁻] = $\sqrt{K_{\mathrm{sp}}(\mathrm{AgCl})}$ = $1.34\times10^{-5}$ mol/L，此时需要 [Ag⁺] = $K_{\mathrm{sp}}(\mathrm{Ag_2CrO_4}) / [\mathrm{CrO_4^{2-}}]$，只有当 Ag⁺ 稍过量时才达到 Ag₂CrO₄ 的 Ksp。

**4**. $n(\mathrm{Fe^{2+}}) = 0.1000 \times 25.00/1000 = 2.500\times10^{-3}$ mol；$n(\mathrm{KMnO_4}) = n(\mathrm{Fe^{2+}})/5 = 5.000\times10^{-4}$ mol；$V(\mathrm{KMnO_4}) = 5.000\times10^{-4} / 0.02000 \times 1000 = 25.00$ mL。

**5**. $n(\mathrm{AgNO_3}) = 0.01000 \times 15.60/1000 = 1.560\times10^{-4}$ mol = $n(\mathrm{Cl^-})$；$m(\mathrm{Cl^-}) = 1.560\times10^{-4} \times 35.45 = 5.530\times10^{-3}$ g = 5.530 mg；质量浓度 = $5.530 / 100.00 \times 1000 = 55.30$ mg/L。

**6**. $n(\mathrm{Na_2S_2O_3}) = 0.1000 \times 24.50/1000 = 2.450\times10^{-3}$ mol；由计量链 $2\mathrm{Cu^{2+}} \sim \mathrm{I_2} \sim 2\mathrm{Na_2S_2O_3}$，$n(\mathrm{Cu}) = n(\mathrm{Na_2S_2O_3}) = 2.450\times10^{-3}$ mol；$m_{\mathrm{Cu}} = 2.450\times10^{-3} \times 63.55 = 0.1557$ g；$w_{\mathrm{Cu}} = 0.1557/0.5000 \times 100\% = 31.14\%$。

**7**. 总 Ag⁺ 加入量：$n_{\text{总}}(\mathrm{Ag^+}) = 0.1000 \times 40.00/1000 = 4.000\times10^{-3}$ mol。分取 25.00/100.0 = 1/4，滤液中剩余 Ag⁺：25.00 mL 滤液消耗 NH₄SCN 10.00 mL → $n_{\text{滤液}}(\mathrm{Ag^+}_{\text{剩余}}) = 0.08000 \times 10.00/1000 \times (100.0/25.00) = 3.200\times10^{-4} \times 4 = 1.280\times10^{-3}$ mol。与 Cl⁻ 反应的 Ag⁺：$n(\mathrm{Ag^+}_{\text{反应}}) = 4.000\times10^{-3} - 1.280\times10^{-3} = 2.720\times10^{-3}$ mol = $n(\mathrm{Cl^-})$。$m_{\mathrm{Cl^-}} = 2.720\times10^{-3} \times 35.45 = 0.09642$ g；$w_{\mathrm{Cl}} = 0.09642/0.3000 \times 100\% = 32.14\%$。
