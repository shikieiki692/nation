## 一、误差基本概念

### 1.1 误差的定义

**误差**（error）是测量值 $x$ 与真值 $x_{\mathrm{T}}$ 之间的差值：

$$\text{绝对误差：} E = x - x_{\mathrm{T}} \qquad \text{相对误差：} E_{\mathrm{r}} = \frac{E}{x_{\mathrm{T}}} \times 100\%$$

- 真值在严格意义上无法获得，分析化学中用**理论真值**、**计量学约定真值**或**标准参考值**代替。
- 相对误差是衡量准确度的常用指标，竞赛中通常要求 $\leqslant 0.1\%\sim 0.2\%$。

### 1.2 误差三分法

误差按其来源和性质分为三大类——这是分析化学数据评价的**第一道门槛**：

| 误差类型 | 特点 | 主要来源 | 典型实例 | 减免/处理方法 |
|:---|:---|:---|:---|:---|
| **系统误差** | 重复出现、单向性（恒正或恒负）、可测 | 方法、仪器、试剂、操作 | 天平不等臂、砝码腐蚀、滴定管刻度不准、试剂含杂质、指示剂变色点与SP不符 | 对照试验、空白试验、校准仪器、方法校正 |
| **随机误差** | 不可预测、双向性、服从统计规律 | 环境波动、读数不确定性 | 温度湿度微变、天平读数末位波动 | **增加平行测定次数**（误差随 $\sqrt{n}$ 减小） |
| **过失误差** | 明显错误、本可避免 | 操作失误 | 加错试剂、溶液溅出、读错刻度 | **剔除**该数据，规范操作 |

> 💡 **核心口诀**："系统误差可测可改，随机误差不可免但可统计，过失误差直接剔除。"

### 1.3 系统误差的四个子类

| 子类 | 说明 | 实例 | 减免 |
|:---|:---|:---|:---|
| **方法误差** | 分析方法本身不完善 | 反应不完全、副反应、终点与SP不一致 | 选择更优方法 |
| **仪器误差** | 仪器不够精确 | 天平不等臂、砝码腐蚀、容量器皿刻度不准 | 校准仪器 |
| **试剂误差** | 试剂不纯 | 试剂含微量被测组分 | 空白试验 |
| **操作误差** | 操作者个人习惯 | 终点判断偏深/偏浅、洗涤不当 | 规范操作、对照试验 |

### 1.4 准确度与精密度

**准确度**（accuracy）：测定值与真值的接近程度，用**误差**衡量。误差越小，准确度越高。

**精密度**（precision）：平行测定值之间的接近程度，用**偏差**衡量。

**偏差的计算**：

| 名称 | 公式 | 说明 |
|:---|:---|:---|
| 绝对偏差 | $d_i = x_i - \bar{x}$ | 单次测定值与平均值之差 |
| 平均偏差 | $\bar{d} = \frac{\sum|d_i|}{n}$ | 各次绝对偏差绝对值的平均值 |
| 相对平均偏差 | $\bar{d}_r = \frac{\bar{d}}{\bar{x}} \times 100\%$ | 平均偏差的相对值，百分比表示 |
| **标准偏差** | $s = \sqrt{\frac{\sum(x_i - \bar{x})^2}{n-1}}$ | 最重要的精密度的量度（第二轮重点） |
| **RSD（变异系数）** | $\mathrm{RSD} = \frac{s}{\bar{x}} \times 100\%$ | 相对标准偏差 |

> **标准偏差对比平均偏差**：$s$ 通过平方放大了大偏差的权重，对离散程度更敏感，是更科学精密度的量度。

#### 准确度与精密度的四象限关系（靶心图）

| 情景 | 数据分布 | 准确度 | 精密度 | 结论 |
|:---:|:---|:---:|:---:|:---|
| **甲** | 数据集中在靶心 | 高 | 高 | ✅ **理想结果**：系统误差和随机误差均小 |
| **乙** | 数据集中但偏离靶心 | 低 | 高 | ⚠️ **存在系统误差**：精密度高但准确度低 |
| **丙** | 数据分散偏离靶心 | 低 | 低 | ❌ **结果不可靠**：二者均差 |
| **丁** | 数据分散但均值在靶心 | 高 | 低 | ⚠️ **偶然凑巧**：不可信赖 |

> 🔑 **核心原则**：精密度是保证准确度的**前提**——高精密度不一定高准确度（乙组）；但准确度高必须以高精密度为前提。

### 1.5 误差传递

**系统误差传递**（已知误差方向）：

| 运算 | 绝对误差 | 相对误差 |
|:---|:---|:---|
| $R = A + B - C$ | $E_R = E_A + E_B - E_C$ | — |
| $R = AB/C$ | — | $\dfrac{E_R}{R} = \dfrac{E_A}{A} + \dfrac{E_B}{B} - \dfrac{E_C}{C}$ |
| $R = mA^n$ | — | $\dfrac{E_R}{R} = n\dfrac{E_A}{A}$ |
| $R = m\lg A$ | $E_R = 0.434m\dfrac{E_A}{A}$ | — |

**随机误差传递**（基于方差）：

| 运算 | 传递公式 |
|:---|:---|
| $R = A + B - C$ | $s_R^2 = s_A^2 + s_B^2 + s_C^2$（**方差加和**，不论加减！）|
| $R = AB/C$ | $\dfrac{s_R^2}{R^2} = \dfrac{s_A^2}{A^2} + \dfrac{s_B^2}{B^2} + \dfrac{s_C^2}{C^2}$ |

> ⚠️ 随机误差传递中，加减法传递的是**方差**（不是标准偏差直接相加），且**不论运算是加还是减**，方差都是相加！

**极值误差估计**（最不利情况）：

| 运算 | 极值误差 |
|:---|:---|
| 加减法 | $|E_R|_{\max} = |E_A| + |E_B| + |E_C|$ |
| 乘除法 | $\left|\dfrac{E_R}{R}\right|_{\max} = \left|\dfrac{E_A}{A}\right| + \left|\dfrac{E_B}{B}\right| + \left|\dfrac{E_C}{C}\right|$ |

### 1.6 平均值与置信区间

**算术平均值**：$\bar{x} = \dfrac{\sum_{i=1}^n x_i}{n}$

**标准偏差**：$s = \sqrt{\dfrac{\sum(x_i - \bar{x})^2}{n-1}}$

**置信区间**（定量表达测量结果的可靠范围）：

$$\mu = \bar{x} \pm \frac{t \cdot s}{\sqrt{n}}$$

其中 $t$ 为置信因子（与置信水平和自由度 $f = n-1$ 有关），95% 置信水平是最常用的选择。

> 💡 置信区间的含义：在 95% 置信水平下，真值 $\mu$ 落在 $\bar{x} \pm t\cdot s/\sqrt{n}$ 范围内的概率为 95%。

### 1.7 Q 检验法剔除离群值

当一组数据中有一个值明显偏离时，可用 Q 检验法判断是否应剔除：

$$Q = \frac{|x_{\text{可疑}} - x_{\text{最邻近}}|}{x_{\text{最大}} - x_{\text{最小}}}$$

- 若 $Q > Q_{\text{表}}$（查表值），则可疑值应**剔除**
- 若 $Q \leq Q_{\text{表}}$，则应**保留**

## 二、有效数字

### 2.1 有效数字的认定

有效数字（Significant Figures）指测量值中所有**确定的数字**加上**一位估计的不确定数字**。

| 数值 | 有效数字位数 | 说明 |
|:---|:---:|:---|
| 0.0340 g | 3 | 前两个 0 定位，末尾 0 有效 |
| $1.80 \times 10^{-3}$ | 3 | 科学记数法明确表达 |
| 25.00 mL | 4 | 滴定管精度，两个 0 均为有效 |
| 1008 | 4 | 数字中间 0 均为有效 |
| 1300 | 不明确 | 可能是 2/3/4 位，应写科学记数法 |
| pH = 11.02 | **2** | 整数部分为 10 的方次，仅小数部分算有效数字 |

### 2.2 修约规则：四舍六入五成双

**国家标准 GB/T 8170**：

- 被修约数 $\leq 4$ → 舍去
- 被修约数 $\geq 6$ → 进位
- 被修约数 $= 5$（且 5 后无数或全为 0）→ 5 前为**偶数则舍**、**奇数则进**
- 被修约数 $= 5$（且 5 后还有非零数字）→ 一律进位
- **只允许一次性修约**到所需位数，不可分步修约

| 原数 | 修约结果（1位小数） | 说明 |
|:---|:---:|:---|
| 1.05 | 1.0 | 5 前为 0（偶数），舍 |
| 0.15 | 0.2 | 5 前为奇数（1），进 |
| 0.25 | 0.2 | 5 前为偶数（2），舍 |
| 1.0501 | 1.1 | 5 后还有非零数字，进 |

### 2.3 运算规则

| 运算类型 | 规则 | 示例 | 关键记忆 |
|:---:|:---|:---|:---:|
| **加减法** | 以**小数点后位数最少**的数据为准 | $12.61 + 0.567 + 0.0142 \rightarrow 12.61 + 0.57 + 0.01 = 13.19$ | **"对齐小数点"** |
| **乘除法** | 以**有效数字位数最少**的数据为准 | $0.0121 \times 25.64 \times 1.05782 \rightarrow 0.0121 \times 25.6 \times 1.06 = 0.328$ | **"够位"** |
| **对数运算** | 结果小数部分位数 = 真数有效位数 | $[\mathrm{H^+}] = 1.3 \times 10^{-3}$（2 位）→ pH = 2.89（2 位小数） | **"pH 只算小数后"** |
| **混合运算** | 中间结果多保留 1 位，最后一次性取位 | — | **"中间留一手"** |

> 🔑 **实践指南**：竞赛中题目数据通常全部为 3~4 位有效数字，结果也取 3~4 位。但**严格遵循规则**比"看齐题目"更重要——有些压轴题故意通过数据位数差异设置陷阱。

## 三、分光光度法原理

### 3.1 Lambert-Beer 定律

**Lambert-Beer 定律**（朗伯-比尔定律）是分光光度法定量分析的基石：

$$A = \lg\frac{I_0}{I_{\mathrm{t}}} = \kappa b c$$

| 符号 | 名称 | 单位 | 物理意义 |
|:---|:---|:---:|:---|
| $A$ | 吸光度（absorbance） | 无量纲 | 反映溶液对光的吸收程度 |
| $I_0$ | 入射光强度 | — | — |
| $I_{\mathrm{t}}$ | 透射光强度 | — | — |
| $T$ | 透射比（透光度） | 无量纲（或 %） | $T = I_{\mathrm{t}}/I_0$ |
| $\kappa$（或 $\varepsilon$） | 摩尔吸收系数 | $\mathrm{L\cdot mol^{-1}\cdot cm^{-1}}$ | 反映吸收灵敏度的特征常数 |
| $b$ | 液层厚度（光程） | cm | 通常用 1 cm 吸收池 |
| $c$ | 浓度 | $\mathrm{mol\cdot L^{-1}}$ | — |

**吸光度与透射比的关系**：

$$A = \lg\frac{1}{T} = -\lg T$$

- $T$ 范围 0~1（或 0%~100%），$A$ 范围 0~∞
- $T$ 从 100% $\rightarrow$ 10% 时，$A$ 从 0 $\rightarrow$ 1（对数关系！）

**吸光度的加和性**：若溶液含多种吸光物质且无相互作用：
$$A_{\text{总}} = A_1 + A_2 + \cdots + A_n = \sum \kappa_i b c_i$$

这是**多组分同时测定**的理论基础。

![[media/beers-law-illustration.jpg|Lambert-Beer 定律示意图]]
*图 Lambert-Beer 定律示意图——入射光 I₀ 通过溶液后被吸收，透射光强度减弱为 Iₜ。吸光度 A = lg(I₀/Iₜ) 与溶液浓度 c 和光程 b 成正比。图中标注了入射光、吸收池、透射光的关系（分析化学第六版 第10章）*

### 3.2 吸收光谱

吸收光谱（吸收曲线）是以**波长为横坐标**、**吸光度为纵坐标**的曲线。关键特征：
- **最大吸收波长** $\lambda_{\max}$：此处吸光度最大，灵敏度最高
- 同一物质不同浓度时，吸收光谱形状和 $\lambda_{\max}$ 不变，仅 $A$ 值改变——这是定量分析的依据

![[media/absorption-spectrum-example.png|吸收光谱示意图]]
*图 吸收光谱（吸收曲线）示意图——以叶绿素为例，横坐标为波长（nm），纵坐标为吸光度。曲线上的峰值对应最大吸收波长 λmax，此处灵敏度最高。不同浓度的同一物质，吸收光谱形状不变，仅吸光度值成比例变化（Wikimedia Commons）*

### 3.3 偏离 Lambert-Beer 定律的原因

严格来说，Lambert-Beer 定律只在**稀溶液**和**单色光**条件下成立。实际测量中常见两种偏离：

#### (1) 光学因素——非单色光

- 实际仪器获得的是一定波长范围的复合光（非严格单色光）
- 物质对不同波长的 $\kappa$ 不同，导致 $A$-$c$ 曲线向吸光度轴弯曲（**负偏离**）
- **克服方法**：选择 $\lambda_{\max}$ 为测定波长（此处吸收曲线最平坦，不同波长的 $\kappa$ 差异最小）

#### (2) 化学因素

| 原因 | 说明 | 示例 |
|:---|:---|:---|
| **解离** | 酸碱型吸光物质随 pH 变化，酸式和碱式吸收不同 | 指示剂在不同 pH 下颜色不同 |
| **缔合** | 高浓度时分子间作用改变吸收性质 | 高浓度时标准曲线向下弯 |
| **配位** | 多级配合物的各级配合比不同，吸收不同 | Fe(SCN)$_n^{3-n}$ 随 SCN⁻ 浓度增大颜色加深 |
| **互变异构** | 同一物质不同结构吸收不同 | CrO₄²⁻ 在酸中变为 Cr₂O₇²⁻ |

> ⚠️ **竞赛高频考点**：高浓度时标准曲线**向下弯曲**（向 $c$ 轴弯曲）——因为分子间相互作用使有效吸光度小于预期。

### 3.4 最佳测量条件

- **测定波长**：选 $\lambda_{\max}$（灵敏度最高，且此处 $\kappa$ 随波长变化小，非单色光影响小）
- 若有干扰，则选"吸收最大、干扰最小"的波长
- **最佳吸光度范围**：$A = 0.2\sim 0.8$（$T = 15\%\sim 65\%$）
- **最佳吸光度**：$A = 0.434$（$T = 36.8\%$）时**浓度测量的相对误差最小**

## 四、显色反应与光度滴定

### 4.1 显色反应

将无色/浅色离子转化为有色配合物的化学反应。对显色反应的要求：

| 要求 | 标准 | 原因 |
|:---|:---:|:---|
| 灵敏度高 | $\kappa > 10^4$ | 可测定低浓度样品 |
| 选择性好 | 干扰少或易消除 | 避免分离步骤 |
| 组成恒定 | 配合比确定 | 保证计量关系稳定 |
| 性质稳定 | 配合物不易分解 | 保证测定重现性 |
| 色差大 | $\Delta\lambda > 60\ \mathrm{nm}$（与显色剂相比） | 容易区分 |

**常见显色剂**：

| 显色剂 | 测定离子 | $\lambda_{\max}$/nm | $\kappa_{\max}$ |
|:---|:---|:---:|:---:|
| 1,10-邻二氮菲 | Fe²⁺ | 508 | $1.1\times10^4$ |
| 丁二酮肟 | Ni²⁺ | 470 | $1.3\times10^4$ |
| 磺基水杨酸 | Fe³⁺ | 520 | $1.6\times10^3$ |
| 二苯硫腙 | Pb²⁺ | 520 | $6.6\times10^4$ |

**显色条件优化**：酸度（pH）→ 显色剂用量 → 反应时间 → 温度 → 溶剂

### 4.2 光度滴定简介

**光度滴定**（分光光度滴定）是用分光光度计监测滴定过程中吸光度变化来确定终点的方法：
- 不需要有色反应物
- 适用于反应慢或没有常规指示剂的体系
- 通过 $A$-$V$ 曲线确定终点（曲线转折点）

## 五、仪器简介（了解）

### 5.1 可见分光光度计基本结构

$$\text{光源} \rightarrow \text{单色器} \rightarrow \text{吸收池} \rightarrow \text{检测器} \rightarrow \text{显示记录}$$

| 部件 | 功能 | 可见区常用 |
|:---|:---|:---:|
| **光源** | 提供复合光 | **钨丝灯**（360~800 nm）|
| **单色器** | 分解复合光为单色光 | 棱镜（玻璃）或**光栅**（分辨率更优）|
| **吸收池** | 盛放试液 | **玻璃**吸收池（可见区）；石英池（紫外区）|
| **检测器** | 光信号→电信号 | **光电管**或**光电倍增管** |
| **数据处理** | 显示 $A$ 或 $T$ | 数字显示/计算机采集 |

![[media/spectrophotometer-schematic.png|可见分光光度计结构示意图]]
*图 可见分光光度计的基本结构——光源（钨丝灯）→ 单色器（棱镜或光栅）→ 吸收池（玻璃池）→ 检测器（光电管/光电倍增管）→ 数据显示。分光光度计的核心功能是将复合光分解为单色光，测定溶液对各波长单色光的吸收程度（分析化学第六版 第10章）*

### 5.2 参比溶液的选择原则

| 情况 | 参比溶液 | 目的 |
|:---|:---|:---|
| 试液和显色剂均无色 | 蒸馏水/溶剂空白 | 扣除吸收池和溶剂影响 |
| 显色剂无色，试液有色 | **不加显色剂的试液** | 扣除试液本底颜色 |
| 显色剂有色 | **试剂空白**（仅加显色剂） | 扣除显色剂颜色 |
| 显色剂和试液均有色 | **加掩蔽剂的试液** | 扣除试液+显色剂共同影响 |

> 💡 参比溶液的选择原则：**扣除一切非待测组分的吸收贡献**。

## 六、标准曲线法

### 6.1 操作步骤

```
① 配制系列标准溶液（4~6 个不同浓度）
② 在 λ_max 处测定各标准溶液的 A 值
③ 以 A 为纵坐标、c 为横坐标作 A-c 图（标准曲线）
④ 测定未知样品的 A 值
⑤ 在标准曲线上查得对应的浓度 c
```

**注意**：标准曲线必须在**线性范围**内使用（$A = 0.2\sim 0.8$），且每次测定需同时带标准曲线。

### 6.2 常见问题

- 标准曲线**不过原点**：可能原因——参比溶液选择不当、存在系统误差
- 标准曲线**弯曲**：偏离 Lambert-Beer 定律（高浓度或非单色光导致）
- 标准曲线**斜率异常**：可能波长不准或显色条件（pH、时间）不当

### 例2 有效数字综合运算 ⭐⭐

**题干**：计算下列各式，结果保留正确的有效数字：
(1) $12.34 + 0.5 + 2.678$
(2) $0.102 \times 3.45 / 2.1$
(3) $[\mathrm{H^+}] = 2.4 \times 10^{-4}\ \mathrm{mol/L}$ 求 pH

**思路**：分别运用加减法（对齐小数点）、乘除法（对齐有效位数）、对数运算规则。

**解析**：

**(1)** 加减法
$12.34$（2 位小数）$+ 0.5$（1 位小数）$+ 2.678$（3 位小数）
→ 以最少的 1 位小数为准
$$12.34 + 0.5 + 2.678 = 15.518 \rightarrow 15.5$$

**(2)** 乘除法
$0.102$（3 位有效）× $3.45$（3 位有效）/ $2.1$（2 位有效）
→ 以最少的 2 位有效为准
$$0.102 \times 3.45 / 2.1 = 0.16757 \rightarrow 0.17$$

**(3)** 对数运算
$$\mathrm{pH} = -\lg(2.4 \times 10^{-4}) = -(-3.62) = 3.62$$

$[\mathrm{H^+}]$ 为 2 位有效数字 → pH 小数点后保留 2 位。

**答案**：(1) 15.5；(2) 0.17；(3) pH = 3.62。

### 例3 Lambert-Beer 定律计算 ⭐⭐⭐

**题干**：某有色配合物在 $\lambda_{\max} = 510\ \mathrm{nm}$ 处的摩尔吸收系数 $\kappa = 2.8 \times 10^4\ \mathrm{L\cdot mol^{-1}\cdot cm^{-1}}$。使用 2 cm 吸收池，测得某溶液的吸光度为 0.56。
(1) 求该溶液中待测物质的浓度。
(2) 若将溶液稀释一倍，在 1 cm 吸收池中测得的吸光度为多少？
(3) 判断该浓度是否在最佳测量范围内（$A = 0.2\sim 0.8$）。

**思路**：直接应用 $A = \kappa b c$。注意浓度、光程、吸光度的正比关系。

**解析**：

**(1)** 由 $A = \kappa b c$：
$$c = \frac{A}{\kappa b} = \frac{0.56}{2.8 \times 10^4 \times 2} = 1.0 \times 10^{-5}\ \mathrm{mol/L}$$

**(2)** 稀释一倍后浓度减半：$c' = 0.5 \times 10^{-5} = 5.0 \times 10^{-6}\ \mathrm{mol/L}$
改用 1 cm 吸收池：
$$A' = \kappa b' c' = 2.8 \times 10^4 \times 1 \times 5.0 \times 10^{-6} = 0.14$$

**(3)** 原溶液 $A = 0.56 \in [0.2, 0.8]$，在最佳测量范围内。稀释后 $A' = 0.14$ 低于最佳范围，应调整（例如换回 2 cm 吸收池则 $A' = 0.28$，可接受）。

**答案**：(1) $c = 1.0 \times 10^{-5}\ \mathrm{mol/L}$；(2) $A' = 0.14$；(3) 原溶液在最佳范围内。

### 例4 标准曲线法求未知浓度 ⭐⭐⭐

**题干**：用邻二氮菲光度法测定 Fe²⁺，得到以下标准数据：

| $c(\mathrm{Fe^{2+}})/10^{-5}\ \mathrm{mol\cdot L^{-1}}$ | 0.00 | 2.00 | 4.00 | 6.00 | 8.00 | 10.00 |
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| $A$（508 nm，1 cm） | 0.000 | 0.220 | 0.440 | 0.660 | 0.880 | 1.100 |

测得某未知 Fe²⁺ 溶液的吸光度为 0.550。求未知液中 Fe²⁺ 的浓度。

**思路**：先确认线性关系，计算斜率，再由吸光度反推浓度。

**解析**：

**步骤1**：判断线性关系
数据过原点，$A$ 与 $c$ 呈正比（线性范围为 $A = 0\sim 1.100$，对应 $c = 0\sim 10.00\times10^{-5}\ \mathrm{mol/L}$）。

**步骤2**：计算斜率
$$k = \frac{\Delta A}{\Delta c} = \frac{1.100 - 0.000}{(10.00 - 0.00) \times 10^{-5}} = 1.100 \times 10^4\ \mathrm{L\cdot mol^{-1}}$$

实际上 $k = \kappa b = \kappa \times 1\ \mathrm{cm}$，所以 $\kappa = 1.100 \times 10^4\ \mathrm{L\cdot mol^{-1}\cdot cm^{-1}}$。

**步骤3**：求未知浓度
$$c_x = \frac{A_x}{k} = \frac{0.550}{1.100 \times 10^4} = 5.00 \times 10^{-5}\ \mathrm{mol/L}$$

**步骤4**：检查该浓度是否在线性范围内
$A_x = 0.550 \in [0.2, 0.8]$，在线性范围内，结果可靠。

**答案**：$c_x = 5.00 \times 10^{-5}\ \mathrm{mol/L}$。

### 例5 偏离 Lambert-Beer 定律的辨析 ⭐⭐⭐

**题干**：某学生在绘制某有色配合物的标准曲线时发现，高浓度区域（$c > 5.0 \times 10^{-4}\ \mathrm{mol/L}$）的 $A$-$c$ 点明显向 $c$ 轴弯曲。请解释：
(1) 可能的原因是什么？
(2) 如何验证你的判断？
(3) 在实验中应如何避免此类问题？

**思路**：从化学因素和光学因素两个角度分析高浓度偏离。

**解析**：

**(1) 可能原因**：**化学因素主导**
高浓度时，有色配合物可能发生：
- **解离**：高浓度下配合物部分解离，使有效吸光度降低向 $c$ 轴弯曲
- **缔合**：分子间相互作用形成二聚体或多聚体，吸收光谱改变
- **配合比改变**：高浓度下可能形成不同配比的配合物，$\kappa$ 改变

同时，光学因素（非单色光）也会造成负偏离。

**(2) 验证方法**：
- 用同样溶液在不同波长处测定标准曲线——若曲线形状相似，是化学因素；若在 $\lambda_{\max}$ 附近偏离较小，在边峰处偏离较大，则光学因素为主
- 测定时更换不同狭缝宽度的单色器，观察曲线变化

**(3) 避免措施**：
- 控制测量浓度在**线性范围**内（$A = 0.2\sim 0.8$）
- 必要时稀释样品使其浓度落在标准曲线的线性部分
- 选择 $\lambda_{\max}$ 为测定波长（此处 $\kappa$ 随波长变化小）
- 使用性能更好的单色器减少非单色光影响

**答案**：见解析。

## 八、常见误区

### 误区1：将"多次测量"当作万能方法

- **表现**：认为所有误差都可以通过增加平行测定次数来减小。
- **纠正**：多次测量只能减小**随机误差**（误差随 $\sqrt{n}$ 减小），**系统误差**不会因重复测量而减小。
- **反例**：天平两臂不等长（系统误差）——测 100 次，结果仍然全部偏低，平均值不会接近真值。

### 误区2：混淆有效数字加减法和乘除法的规则

- **表现**：加减法用有效数字总位数来取位，或乘除法用小数点位数取位。
- **纠正**：加减法看**小数点后位数最少者**（对齐小数点），乘除法看**有效数字总位数最少者**（够位）。
- **记忆法**："加减对齐点，乘除数够位"。

### 误区3：将 $A$ 与 $T$ 的关系简单化为线性

- **表现**：认为 $T$ 减半则 $A$ 减半，或浓度加倍则 $T$ 减半。
- **纠正**：$A = -\lg T$ 是对数关系，不是线性关系。浓度加倍 → $A$ 加倍 → $T$ 是原来的平方（不是减半）。
- **示例**：$T = 50\%$（$A = 0.30$），浓度加倍 → $A = 0.60$ → $T = 25\%$（是的平方，不是减到一半）。

### 误区4：标准曲线不经原点就认为实验失败

- **表现**：看到标准曲线截距不为 0，直接认为实验有问题，数据作废。
- **纠正**：标准曲线不经原点可能是参比溶液选择不当造成的**系统误差**，应分析原因（是否需扣除本底/空白），而非直接废弃数据。
- **正确处理**：检查参比溶液选择是否适当；若为恒定截距，可在计算时用标准曲线方程 $A = kc + b$ 代入（不强制过原点）。

### 误区5：认为 $\lambda_{\max}$ 永远是唯一最佳测定波长

- **表现**：不论什么情况，都无条件使用 $\lambda_{\max}$。
- **纠正**：在 $\lambda_{\max}$ 处有共存物质干扰时，应选"吸收最大、干扰最小"的波长——即使灵敏度稍低，但准确度更高。
- **反例**：丁二酮肟镍的 $\lambda_{\max} = 470\ \mathrm{nm}$，但 Fe 在此处有干扰，改选 520 nm 测定。

### 误区6：忽略显色反应的条件控制

- **表现**：套用 $A = \kappa b c$ 前未确认显色条件（pH、时间、显色剂用量）是否一致。
- **纠正**：标准曲线和样品测定必须在**完全相同的显色条件**下进行，否则 $\kappa$ 不可比。
- **示例**：磺基水杨酸-Fe³⁺ 配合物在 pH 2~3 时为紫红色（1:1），pH 4~7 时为橙色（1:2），pH 8~10 时为黄色（1:3）——$\kappa$ 完全不同。

## 九、方法总结

### 9.1 误差处理的"三部曲"

```
① 识别误差类型：系统？随机？过失？
② 选择应对策略：校正/统计/剔除？
③ 评估数据可靠性：准确度如何？精密度如何？置信区间多宽？
```

### 9.2 Lambert-Beer 定律应用流程图

```text
检查溶液是否均匀透明
├── 否 → 过滤或离心
└── 是 → 确认显色条件（pH、时间、显色剂量）
    ├── 不稳定 → 优化条件
    └── 稳定 → 选择 λ_max（或"吸收最大干扰最小"波长）
        ├── 在 A=0.2~0.8 范围？→ 直接测定
        └── 超出范围 → 稀释或浓缩
            └── 配制标准系列 → 绘制标准曲线 → 测定样品 → 查浓度
```

### 9.3 标准曲线法使用注意事项

1. 标准溶液与样品溶液必须在**相同条件**下显色和测定
2. 标准曲线的浓度范围应覆盖样品的预期浓度
3. 每个浓度至少平行测定 2~3 次取平均值
4. 定期重做标准曲线（显色剂批次、仪器状态变化后需重做）
5. 样品测定值应**在标准曲线的中间范围**，而非外推至曲线之外

## 十、核心速查卡

| 概念 | 核心公式/规则 | 关键点 |
|:---|:---|:---|
| 误差种类 | — | 系统（可校正）、随机（可统计）、过失（剔除）|
| 准确度 | $E_r = \dfrac{E}{x_T} \times 100\%$ | 与真值比较，误差小→准确度高 |
| 精密度 | $s = \sqrt{\dfrac{\sum(x_i-\bar{x})^2}{n-1}}$ | 数据离散程度，偏差小→精密度高 |
| 标准偏差 | $s = \sqrt{\dfrac{\sum(x_i-\bar{x})^2}{n-1}}$ | 较平均偏差更科学（平方放大偏差权重） |
| 置信区间 | $\mu = \bar{x} \pm \dfrac{t \cdot s}{\sqrt{n}}$ | 用于表达结果可靠性范围 |
| Q检验 | $Q = \dfrac{|x_{\text{可疑}}-x_{\text{邻}}|}{x_{\text{max}}-x_{\text{min}}}$ | 离群值取舍的快速检验 |
| 有效数字加减 | 以小数点后位数最少为准 | "对齐小数点" |
| 有效数字乘除 | 以有效数字总位数最少为准 | "够位" |
| 有效数字修约 | 四舍六入五成双 | 一次性修约 |
| $A = \kappa b c$ | $A = \lg(I_0/I_t) = \kappa b c$ | $A$ 与 $c$ 成正比 |
| $A$ 与 $T$ 关系 | $A = -\lg T$ | 对数关系，非反比 |
| 最佳测量范围 | $A = 0.2\sim 0.8$（$T = 15\%\sim 65\%$） | — |
| 最小误差点 | $A = 0.434$（$T = 36.8\%$） | 相对误差最小 |
| 偏离定律原因 | 化学因素（解离/缔合/配位）+ 光学因素（非单色光/散射） | 稀溶液+单色光+均匀介质 |
