# 第六讲：醛·酮·羧酸及其衍生物

> **适用**：第三轮（提高班） | **前置要求**：[[04-课件/学生讲义/2026-06-23-醇醚胺酚]]
> **对应备课大纲**：[[04-课件/备课大纲/2026-06-05-有机合成与金属有机-提高班]]
>
> 本讲是**有机合成的核心枢纽**。羰基(C=O)是自然界中最重要、反应最丰富的官能团之一。**醛/酮的亲核加成**和**羧酸衍生物的亲核加成-消除**构成了绝大多数C-C键和C-杂键构建的基础。加上α-H的碳负离子化学（羟醛缩合/Claisen缩合），本讲覆盖了竞赛有机合成路线设计80%以上的工具。

---

## 学习目标

- ✅ 能比较醛/酮/羧酸/酯/酰胺/酰卤的亲核加成（-消除）活性顺序
- ✅ 能画出醛酮与RMgX/HCN/ROH/NH₃衍生物的亲核加成机理
- ✅ 能应用Cram规则预测手性α-碳醛酮的亲核加成立体选择性
- ✅ 能写出羟醛缩合、Claisen缩合、Dieckmann缩合的机理与产物
- ✅ 能设计基于羰基化学的多步合成路线（碳链增长/官能团转化）

## 一、核心概念

### 1.1 醛与酮

#### 1.1.1 结构特征

羰基碳 sp² 杂化，键角~120°。C=O 键的极化使C带部分正电荷（δ⁺）→ 易受亲核试剂进攻，O带部分负电荷（δ⁻）→ 可被质子化。

**羰基的亲核加成活性顺序**：

```
HCHO > RCHO > ArCHO > R₂CO > ArCOR > Ar₂CO
```
- 醛比酮活泼（醛的羰基C位阻小，且烷基的+I效应降低C⁺电荷）
- 芳香醛酮活性降低（芳环的+C效应使羰基C的正电性减弱）

**羰基α-H的酸性**：
醛/酮的α-H有弱酸性（pKa≈17-20），可被强碱（如LDA、NaH、EtONa）夺去生成烯醇负离子——这是碳负离子化学的起点。

> 💡 **竞赛必考**：α-H的酸性：β-二羰基化合物(pKa≈9-13) > 醛/酮(pKa≈17-20) > 酯(pKa≈25)。乙酰乙酸乙酯的亚甲基H pKa≈11，可被乙醇钠夺去。

#### 1.1.2 醛酮的亲核加成（核心机理）

**通用机理**（以 RMgX 为例）：

```
R₂C=O + Nu⁻ → R₂C⁻(O⁻)Nu —H⁺→ R₂C(OH)Nu
```

**关键特征**：
- Nu⁻ 从四面体羰基平面的**任一侧**进攻→可能产生新手性中心
- 形成 tetrahedral 中间体（sp²→sp³ 杂化）
- 反应的可逆性：缩醛/半缩醛可逆（酸催化），Grignard加成不可逆

**亲核加成的立体化学——Cram规则**（⭐竞赛高频）：

当醛/酮的α-C为手性碳时，亲核试剂从位阻较小的一侧加成：

```
     O               Nu⁻主要从S侧进攻
     ‖
R—C—C—R'  (α-C构型固定)
  L  M
  (大) (中)
```

Cram规则预测：在α-C的3个基团L/M/S中，**亲核试剂主要从体积最小的基团S一侧进攻羰基**。

> 💡 竞赛深化：Cram规则的构象基础——羰基O与α-C的最大基团(L)处于**反式**（减小位阻），M在中间，S在另一侧。Nu⁻从S侧进攻。

#### 1.1.3 各类亲核试剂的加成

**① 含碳亲核试剂（C-C键构建核心）**：

| 试剂 | 产物 | 特点 |
|:---|:---|:---|
| RMgX（Grignard） | 醇（1°/2°/3°取决于底物） | 不可逆，最常用 |
| RLi（有机锂） | 醇（比Grignard更活泼） | 可与大位阻酮反应 |
| NaC≡CR（炔化物） | 炔醇 | 端炔烷基化的延伸 |
| HCN（弱碱催化） | α-羟基腈→α-羟基酸 | 可逆，增长一个碳 |

**② 含氧亲核试剂**：

| 试剂 | 产物 | 特点 |
|:---|:---|:---|
| H₂O（催化H⁺或OH⁻） | 偕二醇（gem-diol） | 平衡偏向反应物（除甲醛/Cl₃CCHO） |
| ROH（催化H⁺） | 半缩醛→缩醛 | **羰基保护**的重要手段 |
| HOCH₂CH₂OH（乙二醇） | 环状缩醛 | 最常用的羰基保护基之一 |

**缩醛作为保护基**：
- 形成：醛/酮 + 乙二醇/H⁺ → 环状缩醛（对碱/Grignard试剂稳定）
- 脱除：稀H⁺/H₂O → 恢复醛酮
- 竞赛常考：合成中需用Grignard试剂时，先用缩醛保护醛基

**③ 含氮亲核试剂**：

| 试剂 | 产物 | 用途 |
|:---|:---|:---|
| NH₃ / 1°胺(RNH₂) | 亚胺(RCH=NR') | 缩合/还原胺化 |
| 2°胺(R₂NH) | 烯胺(R₂C=CR-NR₂) | 烯胺烷基化（Stork烷基化） |
| NH₂OH（羟胺） | 肟(R₂C=NOH) | 鉴别/Beckmann重排 |
| NH₂NH₂（肼） | 腙(R₂C=NNH₂) | Wolff-Kishner还原 |
| 2,4-二硝基苯肼 | 腙（黄色沉淀） | **鉴别醛酮的经典方法** |
| NH₂CONHNH₂（氨基脲） | 缩胺脲（结晶） | 鉴别/分离 |

> ⚠️ **易错——亚胺与烯胺的区别**：
> - 1°胺→亚胺（C=N，可水解恢复）
> - 2°胺→烯胺（C=C-NR₂，有亲核性，可用于α-烷基化）
> - 三者在合成中形成**保护/活化**的体系

**④ 含硫亲核试剂**：

| 试剂 | 产物 | 特点 |
|:---|:---|:---|
| NaHSO₃ | 磺酸盐加成物(沉淀) | 鉴别醛/甲基酮/小环酮 |
| RSH（硫醇） | 缩硫醛/酮→Raney Ni/H₂→**亚甲基** | **羰基→亚甲基**的还原方法 |

> 💡 **Wolff-Kishner还原 vs Clemmensen还原**：
> - Wolff-Kishner：酮 + NH₂NH₂/KOH/Δ → 烷烃（碱性条件）
> - Clemmensen：酮 + Zn(Hg)/HCl → 烷烃（酸性条件）
> - Clemmensen还原本讲在芳香烃中提到过（FC酰基化后的还原）

#### 1.1.4 α-H的反应

**① α-卤代反应**

| 条件 | 机理 | 区域选择性 |
|:---|:---|:---|
| 酸催化(H⁺) | 烯醇中间体 | 取代基多的α-C优先（超共轭稳定烯醇） |
| 碱催化(OH⁻) | 烯醇负离子 | 取代基少的α-C优先（酸性更强） |

**② 卤仿反应**（竞赛经典）

甲基酮(CH₃COR) + X₂/NaOH → RCOONa + CHX₃↓

- **碘仿**是黄色沉淀→可用于鉴别甲基酮和可被氧化为甲基酮的化合物（如乙醇、2°醇RCHOHCH₃）
- 机理：甲基上的3个H依次被X取代→CCl₃强吸电子→C-C断裂

**③ 羟醛缩合 (Aldol Reaction)** ⭐⭐⭐⭐⭐

![[media/base-catalysed-aldol.svg|碱催化羟醛缩合机理]]
*图 碱催化羟醛缩合的完整机理。首先碱夺取 α-H 形成烯醇负离子，然后烯醇负离子进攻另一分子醛的羰基碳，形成烷氧负离子中间体，最后质子化得到 β-羟基醛（Wikimedia Commons）*

醛/酮在稀碱作用下，一分子作为碳负离子（烯醇负离子）进攻另一分子的羰基：

```
2 CH₃CHO —OH⁻→ CH₃CH(OH)CH₂CHO (β-羟基丁醛)
```

**关键规律**：
- **同分子缩合**：2分子相同醛酮
- **交叉缩合**：两种不同醛酮→最多4种产物（无合成价值，除非其中一种无α-H）
- **分子内缩合**：二羰基化合物→环状产物（五/六元环最佳）
- 产物可在酸或加热下脱水→α,β-不饱和醛酮

| 条件 | 产物 | 应用 |
|:---|:---|:---|
| 稀碱(OH⁻) | β-羟基醛/酮 → 可脱水 | C-C键构建 |
| 碱 + 无α-H的醛(如ArCHO) | 交叉缩合（单一产物） | 合成特定α,β-不饱和醛酮 |
| 酮+醛（有α-H的醛） | 醛作亲电试剂优先 | 产物由醛的α-C与酮的羰基反应 |

> ⚠️ **易错——不对称酮的缩合位点**：
> - 碱催化：烯醇负离子在取代少的α-C上生成（酸性强）→该位点进攻
> - 酸催化：烯醇在取代多的α-C上生成（超共轭稳定）→该位点进攻

### 1.2 羧酸及其衍生物

#### 1.2.1 结构与活性顺序

羧酸及其衍生物通式 R-CO-G（G = OH, X, OCOR', OR', NH₂, NHR', NR'₂等）

所有羧酸衍生物的共同特征：**G原子上的孤对电子与C=O形成p-π共轭**（Π₃⁴），使C=O的亲电性比醛酮弱。

**活性顺序（亲核加成-消除反应）**：

```
RCOCl(酰卤) > (RCO)₂O(酸酐) > RCOOR'(酯) > RCONH₂(酰胺) > RCOOH(羧酸) > RCN(腈)
```

**解释**：
- G的电负性越大、给电子共轭越弱 → C=O碳越正 → 越活泼
- G⁻的离去能力越强 → 越活泼
- 酰氯：Cl的-I > +C + Cl⁻是好离去基
- 酰胺：NH₂的+C很强 + NH₂⁻是差离去基 → 最不活泼

#### 1.2.2 羧酸的化学性质

**① 酸性**
羧酸(pKa≈4-5)是弱酸，但远强于酚(pKa≈10)和醇(pKa≈16)。

影响因素：
- 吸电子基(-I, -C) → 酸性增强（如Cl₃CCOOH pKa=0.66）
- 给电子基(+I) → 酸性减弱（如CH₃COOH pKa=4.76）
- 场效应：顺丁烯二酸(pKa₁=1.92) > 反丁烯二酸(pKa₁=3.02)

**② HVZ反应（α-卤代）**
羧酸的α-H在PBr₃催化下被Br取代：
```
RCH₂COOH + Br₂ —PBr₃→ RCHBrCOOH + HBr
```
机理：通过羧酸→酰溴→烯醇式中间体进行。

**③ 酯化反应**
```
RCOOH + R'OH ⇌ RCOOR' + H₂O
```

| 醇的类型 | 机理 | 断键方式 |
|:---|:---|:---|
| 1°/2°醇 | 加成-消除（AAC2） | 酸脱OH，醇脱H |
| 3°醇 | SN1（AAC1） | 酸脱OH，醇脱OH（通过叔碳正离子） |

> 💡 **竞赛考点**：酯化反应的断键位置可用¹⁸O标记法确定——标记醇的O在酯化后出现在酯中（氧来自醇）。

**④ 成酰氯/酸酐/酰胺**
```
RCOOH + SOCl₂ → RCOCl（酰氯，最常用的酰化试剂）
RCOOH + P₂O₅/Δ → (RCO)₂O（酸酐）
RCOCl + 2NH₃ → RCONH₂ + NH₄Cl（酰胺，由酰氯制）
```

#### 1.2.3 羧酸衍生物的亲核加成-消除

**通用机理（以酯的水解为例）**：

```
RCOOR' + Nu⁻ ⇌ R(Nu)(O⁻)OR' → RCONu + R'O⁻
```
总过程：加成→四面体中间体→消除（恢复C=O）

**酯的水解/醇解/氨解**：

| 反应 | 条件 | 产物 |
|:---|:---|:---|
| 水解(酸催化) | H⁺/H₂O/Δ | RCOOH + R'OH（可逆） |
| 水解(碱催化/皂化) | OH⁻/H₂O/Δ | RCOO⁻ + R'OH（不可逆） |
| 醇解 | R''ONa/R''OH | RCOOR'' + R'OH（酯交换） |
| 氨解 | NH₃/Δ | RCONH₂ + R'OH |

**Claisen缩合（竞赛核心）** ⭐⭐⭐⭐⭐

![[media/claisen-condensation.png|Claisen 缩合——两分子酯在强碱作用下形成 β-酮酸酯]]
*图 Claisen 缩合反应的通式。两分子酯在强碱（如 C₂H₅ONa）作用下，一分子酯作为烯醇负离子进攻另一分子酯的羰基碳，经加成-消除机理形成 β-酮酸酯，并失去一分子醇（Wikimedia Commons）*

两分子酯在强碱作用下缩合生成β-酮酸酯：

```
2 CH₃COOC₂H₅ —C₂H₅ONa→ CH₃COCH₂COOC₂H₅ + C₂H₅OH
```

- **分子内Claisen** → Dieckmann缩合（成环，五/六元环最佳）
- **交叉Claisen**：一种酯作碳负离子，另一种酯作亲电试剂（需至少一种无α-H）

**与羟醛缩合的对比**：

| 特征 | 羟醛缩合 | Claisen缩合 |
|:---|:---|:---|
| 底物 | 醛/酮 | 酯 |
| 亲电试剂 | C=O | C=OR'在酯中 |
| 产物 | β-羟基羰基化合物 | β-酮酸酯（消除RO⁻） |
| 条件 | 稀碱 | 强碱(RO⁻) |

#### 1.2.4 β-二羰基化合物的特殊反应

**乙酰乙酸乙酯的合成应用**：

乙酰乙酸乙酯(pKa≈11)的亚甲基H可被EtONa夺取→碳负离子→烷基化：
```
CH₃COCH₂COOEt —EtONa→ [CH₃COCHCOOEt]⁻ —R-X→ CH₃COCHRCOOEt
```
然后水解/脱羧→得到取代丙酮。

**Knoevenagel反应**：醛+含活泼亚甲基的化合物（丙二酸酯/乙酰乙酸酯）→α,β-不饱和化合物

#### 1.2.5 其他重要反应

**① 迈克尔加成 (Michael Addition)** ⭐⭐⭐⭐

α,β-不饱和羰基化合物在碱催化下与碳负离子发生1,4-共轭加成：

```
CH₂=CHCOCH₃ + CH₃COCH₂COOEt —碱→ CH₃COCH(CH₂CH₂COCH₃)COOEt
```

**特点**：
- 亲电试剂：α,β-不饱和醛/酮/酯/腈/硝基化合物（迈克尔受体）
- 亲核试剂：有α-H的醛/酮/酯/β-二羰基化合物（迈克尔给体）
- 通常与羟醛缩合或Claisen缩合联合使用→**Robinson增环反应**

**② Mannich反应** ⭐⭐⭐⭐

甲醛 + 2°胺 + 有α-H的羰基化合物 → β-氨基羰基化合物（Mannich碱）

```
R₂C=O + CH₂O + HN(CH₃)₂ —H⁺→ R₂C(=O)CH₂CH₂N(CH₃)₂
```

**机理**：甲醛+2°胺→亚胺正离子（iminium）→作为亲电试剂与烯醇负离子反应。

**③ Beckmann重排**
酮肟在酸性条件下→酰胺（通过烷基迁移至N上）：
环己酮肟 → H⁺ → 己内酰胺 → 开环聚合 → 尼龙-6

> 💡 **工业价值**：Beckmann重排是尼龙-6生产的关键步骤。

## 二、规律与方法

### 2.1 羰基反应活性总表

| 化合物 | C=O亲电性 | 特征反应 | 活性顺序 |
|:---|:---:|:---|:---:|
| 醛(RCHO) | 强 | 亲核加成 | HCHO > RCHO > ArCHO |
| 酮(R₂CO) | 中 | 亲核加成（活性低于醛） | R₂CO > ArCOR > Ar₂CO |
| 酰卤(RCOX) | 强 | 亲核加成-消除 | RCOF > RCOCl > RCOBr |
| 酸酐 | 中强 | 亲核加成-消除 | 混合酐 > 对称酐 |
| 酯(RCOOR') | 中 | 亲核加成-消除（需催化） | HCOOR > RCOOR |
| 酰胺(RCONH₂) | 弱 | 亲核加成-消除（难） | RCONH₂ > RCONHR > RCONR₂ |
| 羧酸(RCOOH) | 弱 | 需活化 | 通过酰氯活化 |
| 腈(RCN) | 很弱 | 水解→酰胺→酸 | 通过酸催化/碱催化活化 |

### 2.2 常见易错点

> ⚠️ **易错1——缩醛保护的选择性**：醛和酮都能形成缩醛，但在标准条件下（乙二醇/H⁺/Δ），**醛形成缩醛比酮容易得多**。因此可以选择性保护醛基而保持酮基不变。

> ⚠️ **易错2——Grignard与羧酸的反应**：Grignard试剂不能与羧酸直接反应（羧酸有活泼H→Grignard分解）。必须先做成酰氯或酯，再与Grignard试剂反应得到酮（但酮会进一步反应→叔醇）。正确的顺序：酰氯 + Gilman试剂(R₂CuLi) → 酮（不发生进一步反应）。

> ⚠️ **易错3——交叉羟醛缩合的条件**：只有一种有α-H时才能得到单一产物（如苯甲醛+乙醛）。两种都有α-H时产物复杂（四种可能产物），几乎没有合成价值。

> ⚠️ **易错4——Dieckmann缩合的环大小**：最优形成五/六元环。若有多种成环可能，优先选择取代基在热力学上更稳定的方向。

### 2.3 方法技巧

> 💡 **碳链增长工具箱**：
> - +1C: HCN加成 → α-羟基酸（醛/酮+1C）
> - +2C: 环氧乙烷+Grignard（增长2C）
> - (+nC): 羟醛缩合/Cannizzaro反应
> - 交叉缩合:"有α-H + 无α-H"搭配→单一产物

> 💡 **羰基还原方法选择**：
> - 羰基→CH₂：Wolff-Kishner(NH₂NH₂/KOH)或Clemmensen(Zn/Hg/HCl)
> - 羰基→CHOH：NaBH₄(温和，选择性还原醛酮)或LiAlH₄(强，还原所有C=O)

> 💡 **鉴别路线：醛 vs 酮 vs 羧酸**：
> - 所有：2,4-二硝基苯肼（黄色沉淀=羰基化合物）
> - 醛：银镜反应（Tollens试剂）或斐林试剂
> - 甲基酮：碘仿反应（黄色沉淀）
> - 羧酸：NaHCO₃（CO₂↑）

### 2.4 【竞赛深化】来自化学竞赛初赛讲义 ★

> 📖 来源：[[07-资料提炼/书籍提炼/提炼-化学竞赛初赛讲义-第11讲-人名反应与机理推断]]

**初赛讲义第11讲的"母机理"视角——碳负离子缩合家族的统一框架**：

本讲的核心（羟醛缩合/Claisen缩合/Michael加成/Mannich反应）是**碳负离子缩合家族**的五个变体：

| 命名 | 碳负离子来源 | 亲电试剂 | 产物 |
|:---|:---|:---|:---|
| 羟醛缩合 | 醛/酮的α-C | 醛/酮的C=O | β-羟基羰基化合物 |
| Claisen缩合 | 酯的α-C | 酯的C=O | β-酮酸酯 |
| Michael加成 | 各种碳负离子 | α,β-不饱和C=O/C≡N | 1,5-二羰基化合物 |
| Mannich反应 | 醛/酮的α-C | 亚胺正离子(iminium) | β-氨基羰基化合物 |
| Knoevenagel | 活泼亚甲基 | 醛/酮 | α,β-不饱和化合物 |

**统一识别方法**——只需要问两个问题：
1. **碳负离子在哪里**？（找有α-H或活泼亚甲基的位置）
2. **亲电试剂在哪里**？（找C=O/亚胺正离子/α,β-不饱和键）

## 三、典型例题

### 例1 亲核加成活性比较 ⭐⭐

**题干**：比较下列化合物与HCN发生亲核加成的活性并解释：
(a) CH₃CHO (b) CCl₃CHO (c) CH₃COCH₃ (d) HCHO

**思路**：羰基C的正电性越强、位阻越小→亲核加成活性越高。

**解析**：
- HCHO（d）：C两端连H，无位阻，正电性最大→最快
- CCl₃CHO（b）：CCl₃强吸电子(-I)→C=O中C正电性激增
- CH₃CHO（a）：一般醛→第二位
- CH₃COCH₃（c）：两个CH₃的+I使C正电性降低+位阻增大→最慢

**答案**：HCHO > CCl₃CHO > CH₃CHO > CH₃COCH₃

---

### 例2 羟醛缩合与交叉缩合 ⭐⭐⭐

**题干**：丙醛与苯甲醛在稀NaOH溶液中反应，写出主要产物。

**思路**：苯甲醛无α-H→只能作亲电试剂；丙醛有α-H→作碳负离子来源。

**解析**：
丙醛(CH₃CH₂CHO)在碱中形成烯醇负离子（CH₂=CH(O⁻)CHO），进攻苯甲醛(PhCHO)的羰基C：
→ PhCH(OH)CH(CH₃)CHO（β-羟基醛，交叉羟醛缩合产物）
→ 可在加热条件下脱水 → PhCH=C(CH₃)CHO（α,β-不饱和醛）

**答案**：主产物为 2-甲基-3-苯基-2-丙烯醛（脱水后）。

---

### 例3 酯的还原选择性 ⭐⭐⭐

**题干**：化合物 CH₃COCH₂CH₂COOC₂H₅ 用 NaBH₄ 处理后的产物是什么？用 LiAlH₄ 呢？

**思路**：区分两种还原剂的选择性。

**解析**：
- NaBH₄：只还原醛/酮的C=O，不还原酯
  → CH₃CH(OH)CH₂CH₂COOC₂H₅（酮被还原，酯保留）
- LiAlH₄：还原所有C=O（醛/酮/酯/羧酸）
  → CH₃CH(OH)CH₂CH₂CH₂OH（酮和酯均被还原为醇）

**答案**：NaBH₄→酮变醇；LiAlH₄→两种C=O都变为醇。

---

### 例4 乙酰乙酸乙酯的合成应用 ⭐⭐⭐⭐

**题干**：用乙酰乙酸乙酯合成2-庚酮。

**思路**：乙酰乙酸乙酯→烷基化→水解/脱羧→取代丙酮。

**解析**：
1. CH₃COCH₂COOEt + EtONa → [CH₃COCHCOOEt]⁻
2. + CH₃CH₂CH₂CH₂Br → CH₃COCH(C₄H₉)COOEt（烷基化）
3. 稀OH⁻/Δ → 水解
4. H⁺/Δ → 脱羧 → CH₃COCH₂C₄H₉（2-庚酮）

**答案**：2-庚酮由乙酰乙酸乙酯 + 1-溴丁烷经烷基化/水解/脱羧制得。

---

### 例5 Michael加成与Robinson增环 ⭐⭐⭐⭐⭐

**题干**：2-甲基环戊酮与 CH₂=CHCOCH₃ 在碱作用下反应，写出产物并解释。

**思路**：Michael加成→1,5-二羰基→分子内羟醛缩合(Robinson增环)→形成新环。

**解析**：
1. 2-甲基环戊酮 + 碱 → 在取代少的α-C上形成烯醇负离子
2. 进攻 CH₂=CHCOCH₃（Michael加成）→ 1,5-二羰基化合物
3. 分子内羟醛缩合（碱催化，六元环）→ 环己烯酮衍生物（Robinson增环）

**答案**：Robinson增环产物——一个稠合的双环[4.3.0]-壬烯酮衍生物。
