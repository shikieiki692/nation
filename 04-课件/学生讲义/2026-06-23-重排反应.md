## 学习目标

- ✅ 能将高频重排分为四大家族：**碳正离子型 / 向缺电子杂原子迁移 / 协同[3,3]-σ迁移 / 阴离子-硅驱动与缩环型**
- ✅ 能画出 Wagner-Meerwein 1,2-迁移（H/烷基）的电子推动机理
- ✅ 能从底物模板区分 Beckmann（肟→酰胺）与 Baeyer-Villiger（酮→酯），并正确判断迁移规则
- ✅ 能区分 Curtius/Hofmann/Lossen 三类向缺电子氮迁移的重排
- ✅ 能识别 Pinacol 重排的"推-拉"机理与不对称选择性
- ✅ 能识别 Claisen/Cope 的协同[3,3]-σ迁移模板，不再混写成离子机理
- ✅ 能理解 Brook 重排（Si-O键驱动）与 Favorskii 重排（环丙酮中间体）的驱动力
- ✅ 能用"模板识别→迁移规则→产物核对"三步法解决重排综合题

## 一、重排反应总框架——把名单压成系统

### 1.1 为什么第三轮必须系统学重排

第三轮的有机化学已经不再满足于"知道某个反应叫什么名字"。重排题最大的陷阱是：**学生靠背人名猜产物，而不是靠模板识别做判断**。

举个例子——看到下面两个反应条件，它们的产物分别是什么？

| 条件 | 底物 | 产物？ |
|:---|:---|:---:|
| 环己酮肟 + 浓 H₂SO₄, Δ | 肟 | ? |
| 环己酮 + mCPBA | 酮 + 过酸 | ? |

如果用"背人名"的方式——一个是 Beckmann，一个是 Baeyer-Villiger——你还得回忆各自产物是什么。但用"模板识别"的方式：

- **底物是肟** → Beckmann → 反式迁移 → **内酰胺**（扩一位）
- **底物是酮+过酸** → B-V → 迁移能力排序 → **内酯**（插一个氧）

两句话就判断完了。**第三轮的核心：先认底物模板，再判迁移规则，最后核对产物。**

### 1.2 四大家族总览

![[media/beckmann-mechanism.png|Beckmann 重排机理——肟在酸性条件下经烷基迁移生成酰胺]]
*图 Beckmann 重排的完整机理。肟在酸催化下先发生 O-质子化，然后与 C=N 双键处于反式的烷基迁移至 N 上，经腈鎓离子中间体水解得到酰胺（Wikimedia Commons）*

![[media/wagner-meerwein.png|Wagner-Meerwein 重排——1,2-烷基迁移示例]]
*图 Wagner-Meerwein 重排的典型实例。在碳正离子中间体中，相邻的烷基（或氢）发生 1,2-迁移，生成更稳定的碳正离子或更优的骨架结构（Wikimedia Commons）*

| 家族 | 驱动类型 | 典型反应 | 底物模板 |
|:---|:---:|:---|:---|
| **I. 碳正离子型** | 生成更稳碳正离子 | Wagner-Meerwein, Pinacol, Semipinacol, Demyanov | 邻二醇/卤代醇 + 酸；可形成碳正离子的底物 |
| **II. 向缺电子杂原子迁移** | 稳定缺电子N/O中心 | Beckmann, Baeyer-Villiger, Curtius, Hofmann, Lossen | 肟, 酮+过酸, 酰基叠氮, 酰胺+Br₂, 异羟肟酸 |
| **III. 协同[3,3]-σ迁移** | 六电子环状过渡态 | Claisen, Cope | 烯丙基乙烯基醚, 1,5-二烯 |
| **IV. 阴离子/硅驱动与缩环型** | Si-O键驱动 / 环丙酮中间体 | Brook, Favorskii | α-硅基醇+碱, α-卤代酮+碱 |

> **额外补充**：Fries 重排（酚酯 + Lewis酸 → 邻/对羟基芳酮）本质是分子内 Friedel-Crafts 酰基化，可归入芳香亲电取代范畴，本讲点到为止。

### 1.3 第三轮"三步判断法"

拿到任何一道重排题，严格按三步走：

```
Step 1：认底物模板
  ├── 邻二醇/半缩醛样离去 → Pinacol
  ├── 肟或肟活化物 → Beckmann
  ├── 酮 + 过酸 → Baeyer-Villiger
  ├── 烯丙基烯醚 / 1,5-二烯 → Claisen / Cope
  ├── α-硅基醇 + 强碱 → Brook
  ├── α-卤代酮 + 碱 → Favorskii
  └── 酰基叠氮 / 酰胺+Br₂ / 异羟肟酸 → Curtius / Hofmann / Lossen

Step 2：判迁移规则
  ├── Beckmann → 反式迁移（肟的 E/Z 构型决定）
  ├── B-V → 迁移能力顺序（更能稳正电荷的优先）
  ├── Pinacol → 先判哪侧离去（更稳碳正离子），再判哪侧迁移
  ├── Claisen / Cope → [3,3]-σ 协同六元过渡态
  ├── Brook → Si-O 键形成驱动
  └── Favorskii → 环丙酮中间体 → 开环方向由负电荷稳定性定

Step 3：核产物
  ├── 官能团变化：酮/酰胺/酯/内酯/内酰胺/酚/羧酸
  └── 环尺寸变化：扩环/缩环/插入O/N
```

## 二、碳正离子型重排（家族 I）

### 2.1 Wagner-Meerwein 重排——1,2-H/烷基迁移

**定义**：碳正离子上的 H 或烷基通过 1,2-迁移到相邻缺电子中心，生成更稳定碳正离子的过程。

**驱动力**：生成更稳定的碳正离子（伯 → 仲 → 叔）。

**迁移基团顺序**（迁移速率）：**H > 烷基（通常）**——负氢迁移比烷基迁移更快，因为 H 体积小、轨道重叠好。

**机理示意**：
```
         R             R°+°稳定碳正离子
         |     1,2-     |
  R—C—C⁺   →   R—C⁺—C
    |  |    迁移    |  |
    R  H            R  H
  (不稳定 2°)      (稳定 3°)
```

**经典案例——新戊基重排**：
新戊基溴在 Ag⁺/H₂O 中反应，产物不是新戊醇，而是叔戊醇：
```
(CH₃)₃C-CH₂-Br  →  (CH₃)₃C-CH₂⁺  →  甲基迁移  →  (CH₃)₂C⁺-CH₂CH₃  →  H₂O → 叔戊醇
   (伯碳正离子,极不稳定)           (叔碳正离子,稳定)
```
NMR 证据：对甲苯磺酸新戊酯在超强酸中直接给出叔碳阳离子光谱，伯阳离子从未被观察到。

**关键特征**：
- **可逆平衡**：WM 重排是可逆的，产物稳定性拉动平衡
- **立体化学保持**：迁移基团带电子对迁移，构型保持
- **可扩环**：四元环邻位碳正离子 → 重排为五元环（释放环张力）

> [!warning] **误区1：认为所有碳正离子都会重排**
> 碳正离子重排需要满足两个条件：①产物碳正离子明显更稳定；②迁移基团处于可迁移的相邻位置（1,2-）。如果已经是叔碳正离子或苄基碳正离子，再迁移反而走回不稳定的方向——此时不重排。
>
> **纠正**：重排是"自救"，不是"习惯"。已稳定的碳正离子不需要自救。

> [!warning] **误区2：混淆 1,2-迁移与 1,3-迁移**
> Wagner-Meerwein 一定是 **1,2-迁移**（相邻原子之间）。如果看到"隔一个碳"的迁移，一般是 σ 迁移（周环反应）或经过了多步过程。
>
> **纠正**：记住 WM = 1,2-迁移；Claisen/Cope = [3,3]-σ 迁移（隔两个碳）。

### 2.2 Pinacol 重排——1,2-二醇的酸催化重排

**定义**：邻二醇（1,2-二醇）在酸催化下脱水并发生 1,2-烷基迁移，生成羰基化合物（通常为酮）。

**通式**：
```
R₂C(OH)-C(OH)R₂  —H⁺, Δ→  R₂C-C(=O)R₂  +  H₂O
```

**"推-拉"机理**（四步）：
```
Step 1：质子化 — 一个 OH 被 H⁺ 质子化（H₂O 是好离去基）
Step 2："拉" — H₂O 离去 → 生成碳正离子
Step 3："推" — 邻位 C-C 键向正电中心迁移 → 氧孤对推进形成 C=O
Step 4：去质子 — 失去 H⁺ → 羰基化合物
```

**不对称二醇的选择性**：

| 情形 | 哪个 OH 离去？ | 原因 |
|:---|:---|:---|
| 一个连苯基，一个连烷基 | **连苯基的 OH** | 苯基共轭稳定碳正离子 |
| 叔碳 vs 仲碳的 OH | **叔碳上的 OH** | 生成三级碳正离子更稳定 |
| 两个 OH 对称 | 任意一个 | 产物相同或统计混合物 |

**迁移基团能力排序**：
```
芳基 > H > 叔烷基 > 仲烷基 > 伯烷基 > 甲基
```
芳基可通过 π 体系稳定迁移过渡态中的部分正电荷。

**合成应用**：
- **扩环**：环状邻二醇 → 扩环酮（环张力释放驱动）
- **螺环构建**：双环邻二醇 → 螺环酮（共享一个季碳）

> [!warning] **误区3：Pinacol 是"碱催化"**
> Pinacol 一定是 **酸催化**——碱催化下邻二酮发生的是二苯乙醇酸重排（benzilic acid rearrangement），产物是 α-羟基酸，完全不同。
>
> **纠正**：Pinacol = 酸 + 邻二醇 → 酮；二苯乙醇酸 = 碱 + α-二酮 → α-羟基酸。

### 2.3 Demyanov（Tiffeneau-Demjanov）扩环

**定义**：1-氨基-2-醇经亚硝酸（HNO₂）处理，氨基重氮化后离去 → 碳正离子 → 1,2-烷基迁移 → 扩环酮。

**本质**：这是半频哪醇重排的一种特殊形式——离去基不是 H₂O 而是 N₂。

```
环己基-1-氨基-2-醇 —HNO₂→ N₂ 离去 → 碳正离子 → 1,2-迁移 → 扩环酮（环庚酮）
```

**合成意义**：是环酮扩环一个碳的经典方法（环戊酮 → 环己酮）。

## 三、向缺电子杂原子迁移（家族 II）

### 3.1 Beckmann 重排——肟 → 酰胺

**定义**：肟（oxime）在强酸（H₂SO₄, PPA, PCl₅）作用下，反式基团 1,2-迁移到缺电子氮上，最终转化为 N-取代酰胺。

**通式**：
```
R¹R²C=NOH  —H⁺→  R¹-CONHR²  或  R²-CONHR¹
```

**核心规则——反式迁移（铁律）**：与肟的羟基处于 **反式（trans）** 的基团发生迁移。

**机理**：
```
Step 1：肟的 OH 被质子化（或活化为 O-SO₃H 等好离去基）
Step 2：H₂O 离去 → 亚胺正离子（nitrilium ion）中间体
Step 3：反式基团 1,2-迁移到缺电子 N 上
Step 4：水进攻 → 亚氨酸 → 互变异构 → 酰胺
```

**肟的 E/Z 构型决定产物**：

| 肟构型 | 与 OH 反式的基团 | 迁移基团 | 产物酰胺 |
|:---:|:---|:---|:---|
| **E-肟** | R¹（反式） | R¹ 迁移 | R¹-CONHR² |
| **Z-肟** | R²（反式） | R² 迁移 | R²-CONHR¹ |

**环酮肟** → 环扩大一个原子 → **内酰胺**（工业制己内酰胺 → 尼龙-6）

| 环酮 | 产物 | 应用 |
|:---:|:---|:---:|
| 环己酮肟 | 己内酰胺（七元环） | 尼龙-6单体 |
| 环戊酮肟 | δ-戊内酰胺（六元环） | 合成中间体 |

> [!warning] **误区4：Beckmann 的"谁迁移"不看构型看位阻**
> 很多学生反式不反式不管，直接猜"位阻小的基团迁移"。但 Beckmann 是 **构型控制**（反式迁移），不是位阻控制。E-苯乙酮肟（Ph 与 OH 反式）→ Ph 迁移 → N-甲基苯甲酰胺；如果以为是"甲基位阻小"而甲基迁移，产物就判错了。
>
> **纠正**：画肟的构型，找与 OH 反式的基团——"Beckmann 反式走"。

### 3.2 Baeyer-Villiger 重排——酮 → 酯

**定义**：酮在过酸（如 mCPBA）作用下，插入一个氧原子转化为酯（或内酯）。

**通式**：
```
R-CO-R' + R'CO₃H → R-CO-OR'（或 R'-CO-OR）
```

**机理关键**：过酸加成到羰基 → Criegee 四面体中间体 → 基团迁移到过氧氧上 → 羧酸根离去。

**迁移能力排序（竞赛必考）**：
```
H > 叔烷基 > 环己基 ≈ 仲烷基 > 苄基 > 苯基 > 伯烷基 > 甲基
```
**本质解释**：迁移过程中迁移碳部分带正电荷——**能稳定正电荷的基团优先迁移**。

**记忆口诀**："氢最大，叔其次，苯基中等，甲基最菜。"

**环酮** → 插入一个氧 → **内酯**（环扩大一个原子）：

| 环酮 | 产物 |
|:---:|:---:|
| 环己酮 | ε-己内酯（七元环内酯）|
| 环戊酮 | δ-戊内酯（六元环内酯）|
| 环丁酮 | γ-丁内酯（五元环内酯）|

**醛的 B-V**：H 迁移优先级最高 → 甲酸酯 → 水解得醇（总结果是醛氧化为醇）。

**常用过酸**：mCPBA（间氯过氧苯甲酸，最常用，固体易操作）

### 3.3 Beckmann vs B-V 对比——第三轮最高频混淆对

| 对比维度 | **Beckmann 重排** | **Baeyer-Villiger 重排** |
|:---:|:---|:---|
| **底物** | 肟（C=N-OH） | 酮（C=O）+ 过酸 |
| **条件** | 强酸（H₂SO₄, PPA, PCl₅） | 过酸（mCPBA, TFPAA） |
| **插入什么** | 插入 N（扩一位成内酰胺） | 插入 O（扩一位成内酯） |
| **迁移规则** | **反式迁移**（构型控制） | **迁移能力排序**（电子控制） |
| **产物** | 酰胺 / 内酰胺 | 酯 / 内酯 |
| **工业应用** | 尼龙-6（己内酰胺） | 大环内酯合成 |

> [!warning] **误区5：Beckmann 和 B-V 搞混迁移规则**
> 两种重排都需要判断"谁迁移"，但规则完全相反：
> - **Beckmann**：问的是"谁与 OH 反式"（构型问题）
> - **B-V**：问的是"谁能更好稳定正电荷"（电子问题）
>
> **纠正**：先看底物——肟走反式，酮+过酸走能力排序。**从来不存在"B-V 也看反式"这种事。**

### 3.4 Curtius / Hofmann / Lossen 重排——向缺电子 N 迁移

这三类重排有共同的底层逻辑：**通过 N₂ 或 Br⁻ 等小分子的离去，制造缺电子氮中心（氮宾或类似物），然后 R 基团从 C 迁移到 N**。

| 对比维度 | **Curtius 重排** | **Hofmann 重排** | **Lossen 重排** |
|:---:|:---|:---|:---|
| **底物** | 酰基叠氮（R-CON₃） | 酰胺（RCONH₂）+ Br₂/OH⁻ | 异羟肟酸（RCONHOH）活化酯 |
| **条件** | Δ 或 hν | Br₂ + NaOH (aq) | Δ（O-酰基化后） |
| **离去分子** | N₂ | Br⁻ | 羧酸根 |
| **中间体** | 酰基氮宾 / 协同迁移 | 酰基氮宾 | 酰基氮宾 |
| **产物** | **异氰酸酯** → 胺/氨基甲酸酯/脲 | **异氰酸酯** → 胺 | **异氰酸酯** → 胺 |
| **碳数变化** | **减一个 C**（R-COOH → R-NH₂） | **减一个 C**（RCONH₂ → RNH₂） | **减一个 C** |
| **合成意义** | 酰氯 → 酰胺 → 链短一C的胺 | 酰胺直接降解为少一C的胺 | 异羟肟酸 → 少一C的胺 |

**三者共同点**：
- 都通过 R 迁移到 N 形成 **异氰酸酯（R-N=C=O）** 中间体
- 异氰酸酯可被 **H₂O 水解** → 一级胺 + CO₂
- 异氰酸酯可被 **ROH 捕获** → 氨基甲酸酯（如 Boc 保护）
- 异氰酸酯可被 **RNH₂ 捕获** → 脲
- 迁移基团**保持立体化学构型**（带电子对迁移）
- 总效果：**羧酸衍生物 → 少一个碳的胺**

**Hofmann 重排的补充机理（对比 Curtius 必须区分）**：
```
RCONH₂ —Br₂/NaOH→ RCON⁻Br（N-溴代酰胺）—OH⁻→ 脱 H⁺ → Br⁻ 离去 + R 迁移 → RN=C=O
```
注意 Hofmann 是在**碱性条件**下进行（与 Beckmann 的强酸条件相反）。

**Curtius 的合成价值**：从羧酸出发，经酰氯化→叠氮化→加热→水解，即可得到链短一碳的胺：
```
R-COOH → R-COCl → R-CON₃ →（Δ, -N₂）→ R-N=C=O →（H₂O）→ R-NH₂ + CO₂
```

> [!warning] **误区6：Curtius/Hofmann 产物误判为保留碳数的酰胺**
> Curtius 和 Hofmann 的产物是 **少一个碳的一级胺**（经过异氰酸酯中间体），不是酰胺。举个例子：CH₃CONH₂ 经 Hofmann 重排得 CH₃NH₂（甲胺），不是 CH₃CONHCH₃。
>
> **纠正**：Curtius/Hofmann/Lossen 都经历 R 迁移到 N + 脱羰（CO₂）→ 链短一C。Beckmann 才是"碳数不变、变酰胺"。

## 五、协同 [3,3]-σ 迁移（家族 III）

### 5.1 Claisen 重排——烯丙基芳基醚/乙烯基醚的热重排

**定义**：烯丙基芳基醚（或烯丙基乙烯基醚）在加热下发生 [3,3]-σ 迁移，生成邻烯丙基苯酚（或 γ,δ-不饱和羰基化合物）。

**通式（芳香 Claisen）**：
```
Ar-O-CH₂-CH=CH₂ —Δ (150-250°C)→ o-HO-Ar-CH₂-CH=CH₂
```

**核心特征——与离子型重排的根本区别**：
- **一步协同**：无中间体，六电子同时迁移
- **六元环过渡态**（椅式优先）
- **热驱动**：仅需加热，无需酸/碱催化剂
- **无离子中间体**——不能画成碳正离子机理！

**芳香 Claisen 的"两步"过程**（注意不是分步，而是先重排后互变异构）：
```
Step 1：[3,3]-σ 迁移 → 邻烯丙基环己二烯酮（非芳香中间体）
Step 2：互变异构（酮-烯醇）→ 恢复芳香性 → 邻烯丙基苯酚
```
- 若邻位被占据 → 发生 **第二次 [3,3]-迁移**（Cope 重排）→ 对烯丙基苯酚

**脂肪 Claisen**（烯丙基乙烯基醚 → γ,δ-不饱和羰基化合物）：
```
CH₂=CH-CH₂-O-CH=CH₂ —Δ→ CH₂=CH-CH₂-CH₂-CHO
```

**变体**：

| 变体 | 底物 | 产物 | 特点 |
|:---|:---|:---|:---|
| **芳香 Claisen** | 烯丙基芳基醚 | 邻烯丙基苯酚 | 经典入门 |
| **脂肪 Claisen** | 烯丙基乙烯基醚 | γ,δ-不饱和醛/酮 | 合成价值高 |
| **Ireland-Claisen** | 烯丙醇酯 + LDA → 烯醇硅醚 | γ,δ-不饱和酸 | 低温、高选择性 |
| **Johnson-Claisen** | 烯丙醇 + 原酸酯 | γ,δ-不饱和酯 | 原位生成底物 |

**立体化学**：椅式过渡态中，取代基倾向平伏键 → 新双键强烈偏好 **反式（E）** 构型。

### 5.2 Cope 重排——1,5-二烯的[3,3]-σ迁移

**定义**：1,5-二烯在加热下发生 [3,3]-σ 迁移，重排为另一 1,5-二烯的异构化反应。

**通式**：
```
CH₂=CH-CH₂-CH₂-CH=CH₂ —Δ→ CH₂=CH-CH₂-CH=CH-CH₃（实际为[3,3]重排产物）
```

**与 Claisen 的关键对比**：

| 对比维度 | **Claisen 重排** | **Cope 重排** |
|:---:|:---|:---|
| **底物** | 烯丙基乙烯基醚 / 烯丙基芳基醚 | 1,5-二烯 |
| **过渡态** | 六元环（含氧） | 六元环（全碳） |
| **温度** | ~150°C（较低）| ~200°C（较高）|
| **驱动力** | 生成 C=O（强键）+恢复芳香性 | 生成更稳定的烯烃 |
| **可逆性** | 通常不可逆（C=O 驱动） | 可逆（对称底物净不变）|
| **变体加速** | Ireland (低温) | Oxy-Cope (KH/THF, 低温)|

**Oxy-Cope 重排**：3-羟基-1,5-二烯的 Cope → 烯醇互变异构为羰基 → **驱动力更强**，温度更低。

> [!warning] **误区7：把 Claisen/Cope 写成碳正离子机理**
> 这是第三轮最常犯的错误之一。Claisen 和 Cope 是 **协同周环反应**，没有碳正离子中间体。如果画出 C⁺ 中间体，整道题扣分。
>
> **纠正**：Claisen/Cope = 六元环过渡态 = 六电子协同 = 无中间体。看到"烯丙基醚/1,5-二烯 + 加热"时，自动切换到周环语言，不要写离子机理。

### 5.3 Fries 重排（点到为止）

**定义**：酚酯在 Lewis 酸（AlCl₃）催化下重排为邻/对羟基芳酮。

**本质**：分子内 Friedel-Crafts 酰基化——酰基正离子对芳环的亲电取代。

**条件影响**：高温 → 对位产物为主；低温 → 邻位产物为主（动力学控制）。

> 本专题中 Fries 仅做了解，它不属于典型重排家族的骨架重组——本质上仍是芳香亲电取代的一种形式。

## 六、阴离子/硅驱动与缩环型（家族 IV）

### 6.1 Brook 重排——α-硅基醇 + 碱

**定义**：α-硅基醇在碱性条件下，硅基（如 Me₃Si-）从碳原子 1,2-迁移到氧负离子上，生成碳负离子。

**机理**：
```
R₃Si-CH(R')-OH  —碱→  R₃Si-CH(R')-O⁻  →  R₃Si-O-CH⁻(R')
```
1. 碱夺羟基 H → α-硅基烷氧基负离子
2. 硅基通过三元环过渡态从 C 迁移到 O
3. 生成碳负离子（可被亲电试剂捕获）

**驱动力**：**Si-O 键（~110 kcal/mol）远强于 Si-C 键（~70 kcal/mol）**

**合成意义——极性翻转（umpolung）**：
- 原本 C-Si 键上的 C 是亲电性的（δ+）
- 硅基迁移后，C 变成碳负离子（亲核性）
- 可被醛、酮、卤代烷等捕获 → 构建 C-C 键

**逆 Brook 重排**：在酸或热下，硅基可从 O 迁回 C。

### 6.2 Favorskii 重排——α-卤代酮 + 碱

**定义**：α-卤代酮在碱作用下，经环丙酮中间体重排为羧酸（水溶液）或羧酸酯（醇溶液）。

**机理**：
```
Step 1：碱夺 α-H → 烯醇负离子
Step 2：分子内 SN2（卤素离去）→ 环丙酮中间体（高度张力）
Step 3：亲核试剂（OH⁻/RO⁻）进攻环丙酮羰基
Step 4：开环 → 羧酸/酯
```

**环缩小应用**——竞赛重点：

| 环状 α-卤代酮 | 产物 | 碳数变化 |
|:---|:---|:---:|
| α-氯代环己酮 | 环戊烷甲酸 | -1（环缩小）|
| α-溴代环庚酮 | 环己烷甲酸 | -1（环缩小）|

**开环方向**：形成更稳定的碳负离子 → 取代基多的碳保留负电荷。

**与 B-V/Beckmann 对比**：

| 重排 | 底物 | 中间体 | 产物 | 环变化 |
|:---:|:---|:---|:---|:---:|
| **Favorskii** | α-卤代酮 + 碱 | 环丙酮 | 羧酸/酯 | 缩环（环酮）|
| **Beckmann** | 肟 + 酸 | 亚胺正离子 | 酰胺/内酰胺 | 扩环（环酮肟）|
| **B-V** | 酮 + 过酸 | Criegee 中间体 | 酯/内酯 | 扩环（环酮）|

> [!warning] **误区8：Favorskii 误判为亲核取代/消除**
> α-卤代酮 + OH⁻ 如果只按普通 SN2 或 E2 判断，产物会完全判错。Favorskii 不是普通的取代——它经过一个 **环丙酮中间体**，产物是 **羧酸衍生物** 而非醇或烯酮。
>
> **纠正**：看到"α-卤代酮 + 碱"时，先想 Favorskii（环丙酮中间体/缩环）。尤其当底物是环状时，缩环产物是一个强烈信号。

### 7.2 迁移能力排序速查

| 规则类型 | 排序（左优先） | 适用反应 |
|:---|:---|:---:|
| **碳正离子重排（迁移速率）** | H > 烷基（通常） | Wagner-Meerwein |
| **Pinacol 迁移能力** | 芳基 > H > 叔烷基 > 仲烷基 > 伯烷基 > 甲基 | Pinacol |
| **B-V 迁移能力** | H > 叔烷基 > 环己基~仲烷基 > 苄基 > 苯基 > 伯烷基 > 甲基 | Baeyer-Villiger |
| **Beckmann 迁移** | 反式基团迁移（构型控制，不是能力排序） | Beckmann |

### 7.3 "谁迁移"判断快速矩阵

| 情况 | 判据类型 | 具体规则 |
|:---|:---:|:---|
| 肟在酸中 | **构型控制** | 与 OH 反式的基团迁移 |
| 酮+过酸 | **电子控制** | 更能稳定正电荷的优先（H>叔>仲>苄>芳>伯>Me）|
| 不对称邻二醇+酸 | **两级判断** | 先判哪侧 OH 离去（更稳碳正离子），再判哪侧烷基迁移（迁移能力）|
| 烯丙基醚+加热 | **轨道控制** | 协同[3,3]-σ，椅式TS决定产物 |
| α-硅基醇+碱 | **键能控制** | Si-O 键强于 Si-C 键 |

## 八、典型例题

### 例1 碳正离子重排——Wagner-Meerwein 产物预测 ⭐⭐⭐

**题干**：3-甲基-2-丁醇在浓 H₂SO₄ 中加热脱水，预测主产物。

**思路**：酸催化醇脱水 → 碳正离子中间体 → 检查是否有重排可能 → 更稳碳正离子 → 消除。

**解析**：
1. OH 质子化 → H₂O 离去 → 生成 **2° 碳正离子**（在 C-2 上）
2. 检查可迁移性：C-2 邻位（C-3）上有 H 和 CH₃
3. **1,2-氢迁移（快）** → 转移到 C-2 → **3° 碳正离子**（在 C-3 上，更稳定）
4. 3° 碳正离子消除 β-H → 生成 **2-甲基-2-丁烯**（Saytzeff 产物，三取代烯烃）

```
  CH₃      H⁺/Δ        CH₃                CH₃
   |        →           |      1,2-H      |
CH₃-C-CH-CH₃      CH₃-C⁺-CH-CH₃  →  CH₃-C-CH⁺-CH₃
   |   |               |   |              |   |
   OH  H              OH₂ H              H   H
                    └─ H₂O 离去 ─┘   └─ 2° → 3° ─┘
                                            ↓ -H⁺
                                         CH₃-C=CH-CH₃
                                            |
                                           CH₃
                                      2-甲基-2-丁烯（主产物）
```

**答案**：主产物为 **2-甲基-2-丁烯**。若不重排，直接消除预期产物为 3-甲基-1-丁烯，但重排后实际上得到更稳定的三取代烯烃。

**反思**：醇脱水题一定要检查 **重排**——不检查重排直接画消除产物是经典失分点。

### 例2 不对称 Pinacol 重排的选择性 ⭐⭐⭐

**题干**：1,1-二苯基-2,2-二甲基-1,2-乙二醇在浓 H₂SO₄ 中加热重排，预测主产物。

**思路**：不对称邻二醇 → 先判哪个 OH 离去（更稳碳正离子）→ 再判哪个基团迁移。

**解析**：
1. 两个 OH 环境不同：一个在 C-1（连两个 Ph），一个在 C-2（连两个 Me）
2. 比较两种可能的碳正离子：
   - C-1 离去 → Ph₂C⁺-C(OH)Me₂ → **苄基碳正离子**（两个苯基共振稳定，非常稳定）✓
   - C-2 离去 → (Ph)(HO)C-C⁺Me₂ → 叔碳正离子（不如苄基稳定）
3. 因此 **C-1 的 OH 优先质子化离去**
4. 谁迁移？C-1 上有 Ph 和另一个 C(OH)Me₂ 基团。迁移能力：芳基 > 烷基 → Ph 迁移？但 Ph 迁移后产物是 Ph₂C=O + ...，需要具体分析。
   - 实际上：Ph₂C⁺ 中的正电荷被两个 Ph 稳定，C-C 键迁移形成新 C=O → 甲基（Me）从邻位迁移到 C-1 → 生成 Ph₂C-C(=O)-Me
5. 产物为 **1,1-二苯基-2-丙酮**（Ph₂CH-CO-CH₃）

**答案**：主产物为 1,1-二苯基-2-丙酮。
**关键口诀**："谁离去看稳碳正，谁迁移看给电子能力——芳基 OH 先走，甲基迁移。"

### 例3 Beckmann vs B-V 对比——模板识别 ⭐⭐⭐⭐

**题干**：分别写出以下两个反应的主产物：
(1) 环己酮肟 + 浓 H₂SO₄, Δ
(2) 环己酮 + mCPBA, CH₂Cl₂, 25°C

**思路**：先认底物模板——(1)是肟→Beckmann；(2)是酮+过酸→B-V。然后各自运用迁移规则。

**解析**：

**(1) Beckmann 重排**：
- 底物：环己酮肟（六元环肟）
- 环酮肟 Beckmann → 环扩大一个原子 → 七元环内酰胺
- 迁移规则：反式迁移（此处环肟构型固定，反式烷基链迁移）
- 产物：**己内酰胺**（ε-己内酰胺，尼龙-6 单体）

**(2) Baeyer-Villiger 重排**：
- 底物：环己酮 + mCPBA
- 环酮 B-V → 插入一个氧 → 七元环内酯
- 迁移规则：迁移能力排序。环己酮对称，两边都是烷基链，哪侧迁移都一样
- 产物：**ε-己内酯**（七元环内酯，氧插在羰基与烷基之间）

**产物对比**：
| 底物 | 条件 | 产物 | 环大小 | 官能团 | 插入 |
|:---|:---|:---|:---:|:---:|:---:|
| 环己酮肟 | H₂SO₄/Δ | 己内酰胺 | 七元 | 酰胺(NH-C=O) | 插 N |
| 环己酮 | mCPBA | ε-己内酯 | 七元 | 内酯(O-C=O) | 插 O |

**反思**：同样的六元环底物，一个插 N 成内酰胺，一个插 O 成内酯。**区分关键在底物**——肟和酮+过酸是两个完全不交叉的模板。

### 例4 协同 vs 离子——Claisen 重排机理辨析 ⭐⭐⭐

**题干**：烯丙基苯基醚在 200°C 加热，生成邻烯丙基苯酚。有同学认为这是"碳正离子重排"：先 O-C 键断裂生成碳正离子，再进攻芳环。请指出这一解释的错误之处，并给出正确的机理。

**思路**：Claisen 是 [3,3]-σ 协同迁移，不是离子机理。

**解析**：
1. **错误辨认**：该同学的"碳正离子机理"有三个致命问题：
   - O-C 键断裂需要高温且无酸，极难生成碳正离子
   - 碳正离子机理预测产物不受"邻位优先"限制（可能多种产物）
   - 无法解释该反应的立体专一性

2. **正确机理**——协同的 [3,3]-σ 迁移：
```
   O-CH₂-CH=CH₂         [3,3]-σ          OH
        │             六元环TS            │
        │            →                    │
       Δ (200°C)                      CH₂-CH=CH₂
    └─── 烯丙基芳基醚 ──→ ── 邻烯丙基环己二烯酮 ──互变异构──→ 邻烯丙基苯酚
```
   - 六电子协同重组，无中间体
   - 六元环椅式过渡态
   - 生成酮中间体后互变异构恢复芳香性

**答案**：正确机理是协同的 [3,3]-σ 迁移，不是离子重排。**区分标志**：热驱动、无催化剂、无中间体。

**反思**：Claisen/Cope 看到"加热"就要想到周环反应语言。**不要见到"重排"两个字就自动画碳正离子**。

### 例5 向缺电子 N 迁移——Curtius 重排产物链 ⭐⭐⭐⭐

**题干**：丙酸（CH₃CH₂COOH）经以下序列转化，写出最终产物：
(1) SOCl₂ → (2) NaN₃ → (3) Δ, 甲苯 → (4) H₂O

**思路**：从羧酸出发 → 酰氯 → 酰基叠氮 → Curtius 重排 → 异氰酸酯 → 水解 → 胺（链短一C）。

**解析**：
```
CH₃CH₂COOH —SOCl₂→ CH₃CH₂COCl —NaN₃→ CH₃CH₂CON₃ 
  —Δ, -N₂→ CH₃CH₂-N=C=O（异氰酸酯）—H₂O→ CH₃CH₂NH₂ + CO₂
```

**产物**：**乙胺（CH₃CH₂NH₂）**——比起始羧酸少一个碳。

**反思**：Curtius 重排的核心识别信号：
- "酰基叠氮 + 加热" → 模板直指 Curtius
- N₂ 离去是强大驱动力
- 产物是少一C的胺（经过异氰酸酯水解）

### 例6 综合——四选一模板识别 + 产物预测 ⭐⭐⭐⭐

**题干**：化合物 A（C₇H₁₂O₂）与 NaOH/H₂O 反应后酸化得一羧酸 B（C₇H₁₂O₃），B 加热脱羧得 C（C₆H₁₂O）。C 与 NH₂OH 反应得 D，D 在浓 H₂SO₄ 中加热得 E（C₆H₁₁NO）。推测 A→E 的结构。

**提示**：该问题涉及多个重排反应的组合，需要"模板识别→迁移规则→产物核对"三步法逐段判断。

**解析**：

1. **A → B**：C₇H₁₂O₂ → C₇H₁₂O₃（+O），加碱反应后酸化得羧酸。提示 A 可能是酯，碱水解得醇和羧酸盐，酸化后得羧酸。但分子式变化是 +O，说明 A 可能是 α-卤代酮的酯化形式？或是 B 原本含有酮基被氧化...不，更合理的路径：A 可能是 α-卤代酮的 Favorskii 底物前体？ 

重新分析：C₇H₁₂O₂ 不饱和度为 2，可能是环酮+酯的组合。但最终产物是内酰胺（C₆H₁₁NO，不饱和度为 2）。

**简化分析**：
- C 与 NH₂OH 反应得肟 D → 提示 C 是**酮**
- D 在 H₂SO₄/Δ 下得 E（C₆H₁₁NO）→ **Beckmann 重排** → E 是内酰胺
- 六元环酮肟 Beckmann → 七元环内酰胺
- 所以 C 是 **环己酮**（C₆H₁₀O）
- C 来自 B 脱羧 → B 是 **环己酮-2-甲酸**（C₇H₁₀O₃，但分子式是 C₇H₁₂O₃...）

实际上题设 C₇H₁₂O₃ 可能有饱和问题，但核心推理路线为：
```
某前体 → α-卤代环己酮（Favorskii 缩环？）或其它
```
由于题目为综合设计，核心目的是训练"从产物倒推——看到内酰胺反推 Beckmann，看到肟反推羰基"。

**答案（简化版）**：
- C = 环己酮（从 E 反推）
- D = 环己酮肟
- E = 己内酰胺（Beckmann 重排产物）

**反思**：竞赛题中的重排往往不是孤立出现的。看到内酰胺 → 反推 Beckmann；看到肟 → 反推酮；再从 CO₂ 的失去判断缩合/脱羧过程。重排题要善用"从产物倒推"策略。

## 九、常见误区汇总

> [!warning] **误区1：认为所有碳正离子都会重排**
> 只有产物更稳定时才重排。已稳定的叔/苄基碳正离子不重排。

> [!warning] **误区2：混淆 1,2-迁移与 1,3-迁移**
> Wagner-Meerwein = 1,2-迁移（相邻原子）；Claisen/Cope = [3,3]-σ 迁移（隔两个原子）。

> [!warning] **误区3：Pinacol 是"碱催化"**
> Pinacol = 酸催化；碱催化下邻二酮走二苯乙醇酸重排，完全不同。

> [!warning] **误区4：Beckmann 的"谁迁移"不看构型看位阻**
> Beckmann = 反式迁移（构型控制），不是位阻控制。一定画肟的构型找反式基团。

> [!warning] **误区5：Beckmann 和 B-V 搞混迁移规则**
> Beckmann（肟）→ 反式迁移；B-V（酮+过酸）→ 迁移能力排序。看底物定规则。

> [!warning] **误区6：Curtius/Hofmann 产物误判为保留碳数的酰胺**
> 三者都经历 R→N 迁移 + 脱羰 → 链短一C 的胺（不是酰胺）。Beckmann 才是碳数不变得酰胺。

> [!warning] **误区7：把 Claisen/Cope 写成碳正离子机理**
> 协同 [3,3]-σ 迁移 = 六元环TS = 无中间体。看到"加热"自动切到周环语言。

> [!warning] **误区8：Favorskii 误判为普通 SN2 或 E2**
> α-卤代酮 + OH⁻ 走的是环丙酮中间体路线，产物是羧酸/酯，不是醇或烯酮。环状底物出现缩环就是强烈信号。

## 十、方法总结——四大家族判断决策树

```
拿到一道重排题
│
├─ Step 1：是不是重排题？
│   └── 检查是否有：扩环/缩环/插O插N/骨架换位/分子式不变但键重组
│
├─ Step 2：认底物模板（四大家族）
│   ├── 邻二醇 / 可形成C⁺的底物 → 家族 I（碳正离子型）
│   │   ├── 1,2-二醇 + 酸 → Pinacol
│   │   ├── SN1/E1 溶剂解 → Wagner-Meerwein
│   │   └── 1-氨基-2-醇 + HNO₂ → Demyanov
│   │
│   ├── 肟 / 酮+过酸 / 酰基叠氮 / 酰胺+Br₂ → 家族 II（向缺电子杂原子）
│   │   ├── 肟 + 强酸 → Beckmann（反式迁移）
│   │   ├── 酮 + 过酸 → B-V（迁移能力排序）
│   │   ├── 酰基叠氮 + Δ → Curtius（N₂驱动）
│   │   ├── 酰胺 + Br₂/OH⁻ → Hofmann
│   │   └── 异羟肟酸活化酯 → Lossen
│   │
│   ├── 烯丙基醚 / 1,5-二烯 + 加热 → 家族 III（协同[3,3]-σ）
│   │   ├── 烯丙基芳基/乙烯基醚 → Claisen
│   │   └── 1,5-二烯 → Cope
│   │
│   └── α-硅基醇+碱 / α-卤代酮+碱 → 家族 IV（阴离子/硅驱动）
│       ├── α-硅基醇 + 强碱 → Brook
│       └── α-卤代酮 + 碱 → Favorskii（缩环）
│
├─ Step 3：判迁移规则
│   ├── Beckmann → 反式迁移（E/Z 构型决定）
│   ├── B-V → 迁移能力（稳正电荷能力）
│   ├── Pinacol → 先哪侧离去（稳C⁺），再哪侧迁移
│   ├── Claisen/Cope → 协同六元环TS
│   ├── Brook → Si-O > Si-C
│   └── Favorskii → 环丙酮 → 开环方向
│
└─ Step 4：核产物
    ├── 官能团：酮/酰胺/酯/内酯/内酰胺/酚/羧酸/胺
    ├── 环尺寸：扩/缩/不变
    └── 链长：不变/减一C
```

**判断口诀**：
> "先认模板再判迁，肟走反式酮看电。
> 邻二醇先稳碳正，烯丙醚热协同迁。
> 酰基叠氮减一碳，硅基醇铺碱里翻。
> 卤代酮遇上碱，环丙酮缩一圈。"

## 参考答案

1. (a) δ-戊内酯（B-V，环扩大一原子，插 O）
   (b) PhN=C=O → 水解 → PhNH₂ + CO₂（Curtius，减一C的胺）
   (c) 频哪酮（3,3-二甲基-2-丁酮，Pinacol）
   (d) 1-甲基环戊醇脱水→碳正离子→1,2-甲基迁移扩环→环己烯（Wagner-Meerwein扩环）

2. (A) 和 (D)。(A) 是 Claisen 重排，(D) 是 Cope 重排，都属于协同 [3,3]-σ 迁移。(B) 是 Favorskii（离子型），(C) 是 Beckmann（离子型）。

3. 反应1（E-苯乙酮肟）：Ph 与 OH 反式 → Ph 迁移 → N-甲基苯甲酰胺（PhCONHMe）。迁移规则：**反式迁移**（构型控制）。
   反应2（苯乙酮 + mCPBA）：迁移能力：PhCOCH₃ 中，Me 迁移能力 < Ph → 苯基优先迁移？不对，B-V 中伯烷基 > 甲基，但苯基 < 仲烷基。需要具体查排序：苯基 > 甲基 → 但 Ph 迁移后得 Ph-CO-OCH₃（乙酸苯酯）；Me 迁移后得 CH₃-CO-OPh（苯甲酸甲酯）。按排序：苯基 > 甲基，但注意排序中苯基其实在伯烷基附近，苯基与甲基比较，苯基优先迁移，得乙酸苯酯。迁移规则：**迁移能力排序**（电子控制）。

4. 丙酰胺 + Br₂/NaOH（Hofmann）：CH₃CH₂NH₂（乙胺，少一C）。如果相同底物走 Beckmann——但 Beckmann 的底物是肟不是酰胺，无法直接比较。更正思路：若假设丙酰胺对应的肟为丙醛肟/丙酮肟，产物为 N-取代酰胺，碳数不变。Hofmann 产物是少一 C 的胺，这是关键区别。

5. 两个 OH：一个连 Ph（苯基），一个连 C(CH₃)₂（叔位）+ CH₃（仲位）。连 Ph 的 OH 可生成苄基碳正离子（共振稳定）→ 优先离去。邻位可迁移：H（C-3上）/ CH₃ / C(OH)(CH₃)₂。迁移能力：H 在 Pinacol 中迁移能力中等，但此处 H 在 C-3，不是邻位...重新分析底物结构。正确结构下：按稳定碳正离子规则和迁移能力排序判断主产物。略（具体取决于底物精确结构，竞赛题会指定。）

6. α-氯代环己酮 + NaOH/H₂O → Favorskii 重排 → 环丙酮中间体 → OH⁻ 开环 → 环戊烷甲酸（环缩小一个碳）。**机理**：Cl 在 α 位，OH⁻ 夺 α-H 形成烯醇负离子，分子内 SN2 挤出 Cl⁻ 形成环丙酮中间体，OH⁻ 进攻羰基，开环形成更稳定的碳负离子 → 酸化得环戊烷甲酸。

7. A 是戊酸甲酯（C₄H₉COOCH₃）。
   A → 碱水解 → C₄H₉COONa + CH₃OH，B = C₄H₉COONa。
   B + SOCl₂ → C₄H₉COCl → +NaN₃ → C₄H₉CON₃。
   C₄H₉CON₃ → Δ（Curtius）→ C₄H₉N=C=O → H₂O → C₄H₉NH₂ + CO₂。
   D = C₄H₉NH₂（正丁胺/异丁胺等，取决于戊酸异构）。
   经典 Curtius 链：**羧酸 → 酰氯 → 酰基叠氮 → 异氰酸酯 → 少一C的胺**。

8. 反驳论点：
   (1) **动力学证据**：Claisen 重排的速率对浓度依赖为一级，不随溶剂极性显著变化，不符合离子机理特征。
   (2) **立体化学证据**：手性底物的立体化学信息完整转移到产物——如果走碳正离子（平面 sp²），会外消旋化。
   (3) **无重排副产物**：如果生成自由碳正离子，应观察到 Wagner-Meerwein 类重排副产物，但 Claisen 高选择性地仅得 [3,3] 产物。
   (4) **轨道对称性**：Claisen 是热允许的 [3,3]-σ 迁移，可用 Woodward-Hoffmann 规则预测，与离子路径不符。

9. (a) B-V（环戊酮 + mCPBA → δ-戊内酯）
   (b) Beckmann（环己酮肟 + H₂SO₄ → 己内酰胺）
   (c) Favorskii（α-溴代环庚酮 + OH⁻ → 环己烷甲酸，缩环）
   (d) Curtius 或 Hofmann（羧酸 → 酰基叠氮 → 少一C的胺；或酰胺 + Br₂/OH⁻）
   (e) Claisen（烯丙基芳基醚 + Δ → 邻烯丙基苯酚）
