# 有机波谱分析

> **适用**：第三轮（提高班）/ 第四轮（冲刺班）| **前置要求**：[[04-课件/学生讲义/2026-06-23-有机化学基础]]（电子效应、立体化学）
> **对应考纲**：考纲30-有机波谱分析
> **源素材**：ABOC Ch10（光谱化学）+ 初赛讲义第12讲（有机波谱学初步）+ Zchem 第三轮波谱
>
> 波谱分析是"有机化学家的眼睛"——UV-Vis 看共轭、MS 定分子量、IR 判官能团、NMR 看精细结构。本讲将四种谱学方法整合为一条**竞赛可执行的结构推断流程**。

---

## 学习目标

- ✅ 能按 MS → IR → ¹³C NMR → ¹H NMR 的顺序执行联合波谱解析
- ✅ 能用同位素峰（Cl 3:1, Br 1:1, M+1 估碳数）快速缩小分子式范围
- ✅ 能用 IR 特征吸收快速判断官能团存在/排除
- ✅ 能用 ¹H NMR 的化学位移、积分、裂分推断 H 的连接关系
- ✅ 能用 n+1 规则和 J 值判断邻位 H 数和立体化学
- ✅ 能结合不饱和度、分子式、波谱数据推求未知物结构

## 一、波谱分析总览

### 1.1 四种谱学的分工

| 方法 | 回答什么问题 | 信息类型 | 竞赛优先级 |
|:---|:---|---:|:---:|
| **MS**（质谱）| 分子量多少？含 Cl/Br？碳数？偶数/奇数氮？| 分子离子峰、同位素峰、碎片峰 | ⭐⭐⭐ 第一步 |
| **IR**（红外光谱）| 有没有 OH/NH/C=O/C≡N/芳环？| 官能团的存在/排除 | ⭐⭐⭐ 第二步 |
| **¹³C NMR**（碳谱）| 几类碳？对称性如何？有无羰基（醛酮 vs 羧酸衍生物）？| 碳环境数、化学位移区域 | ⭐⭐ 第三步 |
| **¹H NMR**（氢谱）| H 怎么分布？彼此怎么连接？| 化学位移、积分、裂分 | ⭐⭐⭐ 第四步（主判定）|

### 1.2 竞赛联合解析的标准流程

![[media/ethanol-nmr.png|乙醇的 ¹H NMR 谱图——化学位移、积分与裂分示例]]
*图 乙醇的 ¹H NMR 谱图。CH₃ 在 δ 1.2 处为三重峰（被邻位 CH₂ 裂分），CH₂ 在 δ 3.7 处为四重峰（被邻位 CH₃ 裂分），OH 在 δ 2.5-5.0 处为宽峰（可交换质子）。积分面积比 3:2:1（Wikimedia Commons）*

```
分子式/HRMS
    ↓
① 算不饱和度 Ω = C + 1 − (H + X − N)/2
    ↓
② MS ← 分子离子峰确认 M⁺，同位素峰判断 Cl/Br/S
    ↓
③ IR ← 确认或排除关键官能团（C=O? OH? NH? 芳环? C≡N?）
    ↓
④ ¹³C NMR ← 数碳峰数（对称性）、看化学位移区域（羰基类型、sp²/sp³ 比例）
    ↓
⑤ ¹H NMR ← 积分定 H 数目、化学位移判环境、裂分判连接关系
    ↓
⑥ 拼合碎片 → 提出候选结构 → 回代所有谱图验证一致性
```

> **竞赛核心原则**：任何一张谱的结论必须与**其他所有谱图**一致。不能只靠一张谱"看着像"就下结论。

### 1.3 不饱和度的计算

$$
\Omega = C + 1 - \frac{H + X - N}{2}
$$

其中 C = 碳数，H = 氢数，X = 卤素数，N = 氮数

| Ω | 含义 |
|:---:|:---|
| 0 | 饱和开链化合物 |
| 1 | 一个双键 或 一个环 |
| 2 | 两个双键 或 一个三键 或 一个双键+一个环 |
| 4 | 一个苯环（3 个双键 + 1 个环）|

## 二、质谱（MS）

### 2.1 核心信息

| 信息类型 | 内容 | 竞赛用途 |
|:---|---:|:---|
| **分子离子峰** M⁺• | 最高质荷比（EI 谱）| 确定分子量 |
| **同位素峰** M+2, M+4 | 特征峰强度比 | 判断 Cl/Br/S 个数 |
| **M+1 峰** | ~1.1% × 碳数 | 估算碳数 |
| **高分辨质谱 HRMS** | 精确质量（小数点后 4 位）| 唯一确定分子式 |
| **碎片离子** | 特征丢失 (M–15, M–18, M–28 等) | 推断官能团片段 |

### 2.2 卤素同位素指纹

| 卤素 | 同位素 | 天然丰度比 | M : M+2 特征 |
|:---|:---|---:|:---:|
| **Cl** | ³⁵Cl : ³⁷Cl | 75.8 : 24.2 ≈ **3 : 1** | 单 Cl → 3:1；双 Cl → 9:6:1 |
| **Br** | ⁷⁹Br : ⁸¹Br | 50.5 : 49.5 ≈ **1 : 1** | 单 Br → 1:1；双 Br → 1:2:1 |

**多卤素峰强度比**（二项式展开）：

- 含 n 个 Cl：$(3+1)^n$ 展开 → (M):(M+2):(M+4):... 的比例
- 含 n 个 Br：$(1+1)^n$ 展开 → (M):(M+2):(M+4):... 的比例

**经典案例**：
- 单 Br：m/z 186/188 ≈ 1:1 → C₈H₉Br（4-溴苯乙甲醚）
- 双 Cl：m/z 294/296/298 ≈ 9:6:1 → C₁₄H₁₀Cl₂NO₂（双氯芬酸）

### 2.3 M+1 峰估碳数

$$
\frac{I_{M+1}}{I_M} \approx 0.011 \times n_C
$$

式中 $n_C$ 为碳原子数。例：M+1 峰强度为 M 峰的 16.5% → $n_C \approx 15$（C₁₅H₂₄O₂ 的 M=236 时吻合）。

### 2.4 氮规则

- 分子量为 **偶数** → 含 0 或偶数个 N
- 分子量为 **奇数** → 含奇数个 N

### 2.5 高分辨质谱（HRMS）

HRMS 提供精确到小数点后 4 位的分子量，可直接区分具有相同整数质量的不同分子式：

| 分子式 | 整数 m/z | 精确质量 |
|:---|---:|---:|
| C₆H₈⁺• | 80 | 80.0626 |
| C₅H₄O⁺• | 80 | 80.0262 |

### 2.6 竞赛常见碎片

| 丢失碎片 | m/z 变化 | 来源 |
|:---:|:---:|:---|
| CH₃• | M–15 | 甲基 |
| H₂O | M–18 | 醇（脱水）|
| CO | M–28 | 醛/酮/羧酸 |
| C₂H₄（McLafferty）| M–28 | 含 γ-H 的羰基化合物 |
| CO₂ | M–44 | 羧酸/酯 |
| 苄基离子 | **91** | 甲苯/取代甲苯（特征峰！）|
| 卓鎓离子 | 91 | 环庚三烯基正离子 |
| C₃H₃O⁺ | **55** | 含 γ-H 的酮类 |

> **McLafferty 重排**：含 γ-H 的羰基化合物通过六元环过渡态，γ-H 转移至羰基氧 → 产生烯烃（中性丢失）+ 烯醇自由基正离子。这是**判断 γ-H 存在**的有力证据。

## 三、红外光谱（IR）

![[media/ir-carbonyl-spectrum.png|羰基化合物的红外吸收光谱特征]]
*图 羰基化合物在 IR 谱图中的特征吸收。C=O 伸缩振动在 1700-1750 cm⁻¹ 出现强吸收峰，醛/酮/酯/羧酸的具体波数不同，可用于鉴别羰基类型（Wikimedia Commons）*

### 3.1 四大吸收区

```
高波数区                    低波数区
4000    3000    2000    1500    1000    cm⁻¹
 |-------|-------|-------|-------|
 X-H     三键    双键    指纹区
 伸缩     伸缩    伸缩
```

### 3.2 关键吸收峰速查

| 波数/cm⁻¹ | 官能团 | 特征描述 |
|:---:|:---|---|
| **3300–3600** | O–H（游离醇/酚）| 尖峰 |
| **3200–3500**（宽）| O–H（缩合/氢键）| 宽而圆（"又胖又壮"）|
| **2500–3500**（极宽）| O–H（羧酸）| 最易识别的 IR 特征 |
| **3300–3500** | N–H（胺/酰胺）| 1°胺**双峰**，2°胺**单峰** |
| **~3300** | ≡C–H（端炔）| 尖，中等强度 |
| **~3030** | =C–H（烯/芳环）| **>3000 cm⁻¹** |
| **2850–2960** | –C–H（烷基）| **<3000 cm⁻¹**，总是存在 |
| **~2250** | C≡N（腈）| 中等强度，尖 |
| **~2200** | C≡C（炔）| 端炔强，双取代炔很弱（**几乎不可见**）|
| **~1740** | C=O（酯）| 最强吸收之一 |
| **~1715** | C=O（醛/酮）| 稍低于酯 |
| **~1680** | C=O（酰胺）| 最低波数的羰基 |
| **1620–1680** | C=C | 中等强度 |
| **~1600, ~1500** | 芳环 C=C | 两个特征峰（芳环存在标志）|

### 3.3 3000 cm⁻¹ 分界线

这是 IR 中最实用的快速判断：

- **<3000**：饱和 C–H（sp³，烷基）→ 分子有饱和部分
- **>3000**：不饱和 C–H（sp² =C–H 或 sp ≡C–H）→ 分子有烯/芳/炔

### 3.4 羰基波数规律（共轭 + 取代效应）

| 羰基类型 | 波数/cm⁻¹ | 解释 |
|:---|---:|:---|
| 酰氯 RCOCl | 1790–1815 | Cl 的强吸电子诱导效应 |
| 酸酐 (RCO)₂O | 1800–1850（双峰）| 两个羰基耦合 |
| 酯 RCOOR' | 1735–1750 | OR' 的弱给电子共轭 |
| 醛 RCHO | 1720–1740 | — |
| 酮 RCOR' | 1705–1725 | — |
| 羧酸 RCOOH | 1700–1725 | 氢键降低波数 |
| 酰胺 RCONH₂ | 1650–1690 | N 的强给电子共轭（C=O 键级降低）|
| 羧酸根 RCOO⁻ | 1300–1420 | 共振完全离域 |

**共轭效应**：C=O 与 C=C 或芳环共轭 → C=O 波数降低 ~20–40 cm⁻¹（键级降低）

> **竞赛应用**：IR 的核心价值是**快速回答"有/没有"**——有没有羰基？有没有羟基？有没有腈基？不要试图通过 IR 来区分精细结构，那是 NMR 的任务。

## 四、¹³C NMR（碳谱）

### 4.1 碳谱的核心价值

¹³C NMR 在竞赛中常作为**辅助工具**，提供三项关键信息：

1. **碳环境数**：峰的个数 = 不等价碳的种类数
2. **对称性**：峰数明显少于分子式中的碳数 → 分子有对称性
3. **羰基类型**：化学位移区分醛酮 vs 羧酸衍生物

### 4.2 化学位移区域

| δ / ppm | 碳类型 | 说明 |
|:---:|:---|---|
| 0–50 | 饱和碳（sp³）| 烷基碳 |
| 50–100 | 连杂原子的饱和碳 | C–O, C–N, C–X |
| 100–150 | sp² 碳（烯/芳）| C=C, Ar–C |
| 150–200 | 羰基碳 | C=O |
| 195–215 | 醛/酮 C=O | 最低场区域 |
| 165–185 | 羧酸衍生物 C=O | 酸/酯/酰胺（稍高场）|

### 4.3 碳谱的竞赛用法

1. **先数峰数**——不先认峰。峰数 = 几类碳 → 判断分子对称性
2. **δ > 200** → 醛/酮羰基（无 α-卤素或共轭时）
3. **δ 165–185** → 羧酸衍生物羰基
4. **DEPT 谱**（如有）：CH₃/CH₂/CH/C 级数区分（向上/向下/无信号）

## 五、¹H NMR（氢谱）

### 5.1 三项基本信息

| 参数 | 物理含义 | 竞赛用途 |
|:---|---:|:---|
| **化学位移 δ** | H 的电子环境（屏蔽/去屏蔽）| 推断 H 的类型（烷基/烯基/芳基/醛基）|
| **积分（峰面积）** | H 的相对数目 | 推断每种 H 的个数 |
| **裂分模式** | 相邻不等价 H 的数目 | 推断 H 之间的连接顺序 |

### 5.2 化学位移详细表

| δ / ppm | 质子类型 | 备注 |
|:---:|:---|---|
| **0.9** | R–CH₃ | 甲基基准 |
| **1.3** | R–CH₂–R | 亚甲基基准 |
| **1.5–1.7** | R₃CH | 次甲基基准 |
| 1.5–2.5 | –CH₂– 邻接 C=O/芳环 | β-位效应 |
| 2.0–2.5 | –COCH₃（甲基酮）| 甲基邻羰基 |
| 2.5–3.0 | –C≡C–H | 炔氢（反常高场）|
| 3.0–4.5 | R–O–CH₃, R–CH₂–O– | 连氧饱和碳 |
| 4.5–6.5 | **C=C–H**（烯氢）| 特征性高 |
| 6.5–8.5 | **Ar–H**（芳氢）| 环电流去屏蔽 |
| 9–10 | **–CHO**（醛基）| **最特征信号** |
| 10–13 | –COOH（羧酸）| 宽峰，可交换 |

**化学位移简化修正法**（初赛讲义推荐）：
- 基准值：–CH₃ (0.9), –CH₂– (1.3), >CH– (1.7)
- 邻位连 **醛/酮/酯/腈/烯/炔/芳基/S/N** → 约 +1 ppm
- 邻位连 **硝基/O/卤素/酰胺/酰氧基** → 约 +2 ppm

### 5.3 n+1 裂分规则

等价相邻 H 数为 n → 信号裂分为 **n+1** 重峰。

| 相邻 H 数 | 裂分峰形 | 峰强度比 |
|:---:|:---|---:|
| 0 | 单峰 (s) | 1 |
| 1 | 双峰 (d) | 1:1 |
| 2 | 三重峰 (t) | 1:2:1 |
| 3 | 四重峰 (q) | 1:3:3:1 |
| 4 | 五重峰 (p) | 1:4:6:4:1 |

**实例**：
```
CH₃CH₂Br → CH₃ (t, 3H) + CH₂ (q, 2H)
   ↑相邻 CH₂ 有 2H          ↑相邻 CH₃ 有 3H
```

### 5.4 多重偶合

当 H 与多种不等价相邻 H 偶合时，裂分双数相乘：

$$(n_1+1)(n_2+1)...$$

例如 CH₃CH₂CHBr– 中的 >CH–：邻位有一个 CH₃（3H）和一个 CH₂（2H）→ 裂分为 $(3+1)(2+1) = 4 \times 3 = 12$ 重峰（实际为多重峰）。

### 5.5 偶合常数 J

| 偶合类型 | J 值范围 / Hz | 用途 |
|:---:|:---:|:---|
| ³J 邻位（烷基）| 6–8 | 自由旋转的平均值 |
| ³J 顺式烯 H | **~10–13** | 判断顺式构型 |
| ³J 反式烯 H | **~14–18** | 判断反式构型 |
| ²J 同碳（gem）| 10–15 | 不等价 CH₂ 的信号 |
| ⁴J 远程偶合 | 0–3（W 型）| 刚性体系中可见 |

> **竞赛应用**：J 值用于判断烯烃的顺/反构型。J = 15 Hz → 反式；J = 10 Hz → 顺式。

### 5.6 磁等价 vs 化学等价

**化学等价**：分子对称性决定的相同化学环境
**磁等价**：化学等价 + 对相邻 H 的偶合常数相同

**常见不等价情况**：
- 手性中心邻位的 CH₂（两个 H 为**非对映**关系）→ 不等价，表现为 AB 体系
- 环己烷的 a-键/e-键 H → 不等价
- 单键旋转受阻时（如酰胺 C–N 部分双键）→ 邻位 CH₂ 不等价

> **不能机械套用 n+1 规则**：当相邻 H 不等价时，n+1 不再适用，会出现更复杂的裂分（dd, dq 等）。

### 5.7 NOE（核 Overhauser 效应）

- 空间距离 < 5 Å 的 H 之间产生信号增强
- 用于判断**立体化学**：NOE 增强 → 空间接近（顺式/endo）
- 无 NOE → 空间远离（反式/exo）

### 5.8 活泼氢的识别

| 活泼 H 类型 | δ 范围 | D₂O 交换实验 |
|:---|---:|:---|
| R–OH（醇）| 1–5（可变）| 交换后消失 |
| Ar–OH（酚）| 4–8 | 交换后消失 |
| R–COOH（羧酸）| 10–13（宽）| 交换后消失 |
| R–NH₂（伯胺）| 1–3 | 交换后消失 |
| R–CONH–（酰胺）| 5–9 | 交换后消失 |

> D₂O 交换法：向样品中加入 D₂O → 活泼 H 被 D 取代 → 对应的 NMR 信号消失（或显著减弱）。

## 六、UV-Vis 光谱（补充）

### 6.1 核心信息

- 反映分子中**共轭体系的大小**
- λmax 越大 → 共轭体系越大
- 不能确定具体官能团（那是 IR 和 NMR 的任务）

### 6.2 关键概念

| 概念 | 定义 |
|:---|:---|
| **发色团**（Chromophore）| 产生 UV-Vis 吸收的基团（C=C, C=O, 芳环等）|
| **助色团**（Auxochrome）| 本身不吸收但能使 λmax 红移的基团（–OH, –NH₂, –Cl）|
| **红移**（Bathochromic）| λmax 向长波方向移动（共轭增大、溶剂极性增大对 π→π*）|
| **蓝移**（Hypsochromic）| λmax 向短波方向移动（溶剂极性增大对 n→π*）|

### 6.3 竞赛用法

竞赛中 UV-Vis 最常见的是作为**结构验证工具**：
- 判断共轭体系大小：λmax 与共轭长度正相关
- 区分互变异构体：烯醇式 vs 酮式的 UV 吸收不同
- Woodward-Fieser 规则用于预测共轭二烯的 λmax（通常不要求背记全部增量，能查表应用即可）

## 七、联合解析实战

### 7.1 标准解题流程

```
步骤 1：由 MS/HRMS 确定分子式 → 计算 Ω
步骤 2：IR 看官能团（有/无 C=O, OH, NH, 芳环, C≡N）
步骤 3：¹³C NMR 数碳峰 → 判断对称性；看羰基区域 → 判断类型
步骤 4：¹H NMR → 积分定 H 比例 → δ 定环境 → 裂分定连接
步骤 5：拼合碎片 → 写出候选结构 → 回代所有谱图验证
```

### 7.2 实战案例

**案例**：化合物 A（C₄H₈O₂），IR 1740 cm⁻¹ 强吸收，¹H NMR 如下：
- δ 1.15 (t, 3H, J = 7 Hz)
- δ 1.30 (t, 3H, J = 7 Hz)
- δ 2.30 (q, 2H, J = 7 Hz)
- δ 4.15 (q, 2H, J = 7 Hz)

**解析**：
1. Ω = 4 + 1 − (8 − 0)/2 = 1 → 一个双键或一个环
2. IR **1740 cm⁻¹** → 酯 C=O（不是羧酸，因为无 OH 宽峰）
3. ¹H NMR 有两组 CH₃（t），两组 CH₂（q），各自的四重峰表明彼此相邻
4. δ 4.15 → CH₂ 连 O（酯的 –OCH₂–）；δ 2.30 → CH₂ 邻 C=O（–CH₂CO–）
5. **拼合**：–CH₂CH₃（δ 1.15/2.30） + –OCH₂CH₃（δ 1.30/4.15） → **丁酸乙酯** CH₃CH₂COOCH₂CH₃

### 7.3 结构推断的"交集法"

```
所有可能的     IR 排除不      ¹³C NMR 排除     ¹H NMR 排除
候选结构  ──→ 可能的官能团 ──→ 不匹配的骨架  ──→ 不匹配的H分布  ──→  唯一结构
```

## 八、竞赛应用与常见考法

### 8.1 竞赛中波谱分析的四种考法

| 考法 | 特征 | 解题策略 |
|:---|:---|---:|
| **结构推求型** | 给分子式 + 1H NMR 数据（+ 可能 IR/MS）| 先算 Ω → 再拼碎片 → 回代验证 |
| **区分同分异构体型** | 给多个候选结构 + 某 H 的 δ/J 值 | 找各候选结构中 H 的邻居差异 |
| **反应跟踪型** | 给反应物 + 产物 1H NMR 变化 | 关注消失/新生的峰，判断官能团转化 |
| **机理验证型** | 结合氘代/同位素标记 + NMR 变化 | 关注被标记原子的信号归属 |

### 8.2 竞赛高频陷阱

1. **活泼氢被忽略**：–OH, –NH, –COOH 的峰可交换消失，积分不反映真实 H 数
2. **磁不等价的误判**：手性中心邻位的 CH₂ ≠ 简单三重峰/四重峰
3. **n+1 规则的机械套用**：相邻 H 不等价时裂分更复杂（dd, dq, 多重峰）
4. **J 值的忽略**：J = 15 Hz 是反式烯烃的重要信号
5. **对称性的误判**：¹³C NMR 峰数远少于碳数 → 分子有对称面/对称中心

## 九、踩坑回流

| 误区 | 正确认识 |
|:---|---|
| "IR 能确定精细结构" | IR 是官能团**筛选工具**，不是精细结构解析工具。精细结构交给 NMR |
| "n+1 规则永远适用" | n+1 的前提是相邻 H **等价**。手性中心邻位的 CH₂ 不等价，裂分更复杂 |
| "UV-Vis 波数越大越好" | UV-Vis 的关键是 λmax ≠ 吸收强度。共轭越大 λmax 越大，但不代表吸收越强 |
| "积分 = 整数 H 数" | 积分给出的是 **相对比例**，需结合分子式才能推断绝对 H 数 |
| "化学位移表靠背" | 用 **基准值 + 电子效应修正** 替代死背：甲基 0.9，连 O + 2 ppm，连芳基 + 1 ppm |
| "光化学和光氧化还原一样" | 光化学是**底物直接激发**；光氧化还原是**催化剂吸收光后**转移电子给底物 |

## 十二、参考答案与提示

### 🌱 11.1 基础巩固答案

**1.** $\Omega = C + 1 - (H + X - N)/2$：(a) C₆H₆：Ω = 6+1−6/2 = **4**（苯环）。(b) C₄H₈O：Ω = 4+1−8/2 = **1**（一个双键或环）。(c) C₇H₇Cl：Ω = 7+1−(7+1)/2 = **4**（苯环）。(d) C₃H₅NO：Ω = 3+1−(5−1)/2 = **3**（可能含 C=N + C=O + 环）。(e) C₁₀H₈：Ω = 10+1−8/2 = **7**（萘为 7：两个苯环共享两个碳）。

**2.** M : M+2 ≈ 3:1 → 含**一个 Cl 原子**（³⁵Cl:³⁷Cl = 75.8:24.2 ≈ 3:1）。分子量 156 中 Cl 贡献 35 → 剩余 121。若为 C₈H₇Cl（M=127 不够）——此题 M=156 需重新核算。实际上 Cl 同位素指纹 3:1 是最特征的判断依据。

**3.** M+1/M = 8.8% → 碳数 ≈ 8.8/1.1 = **8 个碳**。验证：8 × 1.1% = 8.8%，吻合。

**4.** 氮规则：分子量为**偶数** → 含 0 或偶数个 N；分子量为**奇数** → 含奇数个 N。因此 M=101 和 M=207 可能含奇数个 N（各含 1 个或 3 个等）。M=72 和 M=136 为偶数 → 含 0 或偶数个 N。

**5.** 3000 cm⁻¹ 是**饱和/不饱和 C-H 的分界线**。(a) –CH₃（烷基）：<3000 cm⁻¹（~2850–2960），sp³ C-H。(b) =CH（烯基）：>3000 cm⁻¹（~3030–3100），sp² C-H。(c) Ar–H（芳基）：>3000 cm⁻¹（~3030），sp² C-H。(d) ≡C–H（端炔）：>3000 cm⁻¹（~3300），sp C-H。

**6.** 从高到低：乙酰氯（1790–1815）> 乙酸乙酯（1735–1750）> 丙酮（1705–1725）> 苯乙酮（~1685，共轭降低）> 乙酰胺（1650–1690）。原因：Cl 强吸电子诱导 → C=O 键级升高 → 波数最高。酯的 OR 弱给电子共轭 → 略低于酮。苯乙酮的 C=O 与芳环共轭 → 键级降低 → 波数低于丙酮。酰胺的 N 强给电子共轭 → C=O 键级最低 → 波数最低。

**7.** (a) CH₃CH₂Br：CH₃ 为 t（相邻 CH₂ 有 2H → 2+1=3），CH₂ 为 q（相邻 CH₃ 有 3H → 3+1=4）。(b) (CH₃)₂CHOH：CH₃ 为 d（相邻 CH 有 1H → 1+1=2），CH 为 septet（相邻两个 CH₃ 共 6H → 6+1=7），OH 为 s（可交换，通常不裂分）。(c) CH₃CH₂CH₂Cl：CH₃ 为 t（相邻 CH₂ 有 2H），中间 CH₂ 为 sextet（相邻 CH₃ 3H + CH₂Cl 2H → (3+1)(2+1)=12 重峰，近似六重峰），CH₂Cl 为 t（相邻 CH₂ 有 2H）。

### 🔥 11.2 竞赛入门答案

**8.** M:M+2 = 1:1 → 含**一个 Br 原子**（⁷⁹Br:⁸¹Br ≈ 1:1）。M–79 = 107 → 丢失 Br• → 碎片 m/z 107 = M–Br。含 Br 的分子式可能为 C₇H₇Br（M=171 不够）或 C₅H₇Br（M=148 不够）。M=186 → 186−79=107 → C₇H₇O？不，107 = C₇H₇O（苄氧基）或 C₈H₁₁？实际：C₇H₇Br 的 M=171，C₈H₉Br 的 M=185，都不够 186。若 M=186 含 Br，剩余 107 → 可能为 C₇H₇NO（107）+ Br = C₇H₇BrNO？此题需重新核对。核心判断：**1:1 同位素峰 → 含 Br**。

**9.** C₇H₈O₂ 精确质量 120.0524 vs HRMS 120.0575，差值 0.0051。C₈H₈O 精确质量 120.0575，差值 0.0000 → **C₈H₈O 匹配**（误差 0 ppm）。

**10.** (a) IR 1740 cm⁻¹ → **酯 C=O**；1250 cm⁻¹ → **C–O 伸缩**（酯的 C–O–C）。无 OH 宽峰 → 排除羧酸。(b) δ 7.0 和 7.9 各 2H 的双峰（J=8 Hz）→ **对位二取代苯环**（AA'BB' 体系，近似两组双峰）。(c) 结构：对位取代苯甲酸甲酯 → **4-甲氧基苯甲酸甲酯**（4-MeO-C₆H₄-CO₂Me）。δ 3.8 (s, 3H) = 甲氧基 OCH₃。验证：C₈H₈O₂，Ω = 8+1−8/2 = 5（苯环 4 + C=O 1 = 5），吻合。

**11.** (a) Ω = 8+1−10/2 = 4（一个苯环）。(b) 8 个碳只有 4 个峰 → 分子有**对称面**，不等价碳只有 4 种。(c) δ 15.5（烷基 CH₃）、δ 29.0（烷基 CH₂）、δ 125.8（sp² 烯/芳碳）、δ 148.2（sp² 烯/芳碳）。结合 Ω=4 和对称性 → **对二甲苯**（1,4-dimethylbenzene）：两个等价甲基（δ 15.5）+ 两个等价 CH₂（不，对二甲苯无 CH₂）。修正：**邻二甲苯或间二甲苯**的对称性分析——对二甲苯有 3 类碳（甲基、ipso-C、其余芳碳），只有 3 个峰。间二甲苯有 5 类碳。邻二甲苯有 5 类碳。数据 4 峰 → 可能为**1,2,4,5-四甲基苯**（durene）？不，C₈H₁₀。最可能：**对二乙苯**？不，C₈H₁₀。实际：**对二甲苯**的 ¹³C NMR 应有 3 个峰（两个等价甲基、两个等价 ipso-C、四个等价芳 CH）。此题数据可能为**1,4-环己二烯的二甲基衍生物**——需更多信息。若按标准数据，C₈H₁₀ Ω=4 的最可能结构为**乙苯**（5 类碳）或**对二甲苯**（3 类碳）。4 个峰最可能为**间二甲苯**（5 类碳，但某些峰可能重叠）。

**12.** J = 15 Hz → **反式**烯烃（反式烯氢的 ³J = 14–18 Hz）。J = 10 Hz → **顺式**烯烃（顺式烯氢的 ³J = 10–13 Hz）。这是判断烯烃几何构型最直接的 NMR 方法。

### 🏆 11.3 真题挑战答案

**13.** (a) Ω = 9+1−(10−0)/2 = 10−5 = **5**（苯环 4 + C=O 1 = 5）。(b) IR 1735 cm⁻¹ → **酯 C=O**（波数 1735–1750 典型范围）。1230 cm⁻¹ → **C–O 伸缩**（酯的 C–O–C）。无 OH 宽峰 → 排除羧酸。结论：**酯基**存在，羧酸排除。(c) δ 1.25 (t, 3H) + δ 4.12 (q, 2H) → 经典的**乙酯基** $\ce{-OCH2CH3}$ 片段：CH₃ 被邻位 CH₂ 裂分为三重峰（2+1=3），CH₂ 被邻位 CH₃ 裂分为四重峰（3+1=4）。δ 4.12 的高场位移说明 CH₂ 连氧（OCH₂）。(d) 结构：δ 2.05 (s, 3H) = 甲基酮或乙酸甲酯的 CH₃ → 结合酯基判断为**乙酸苄酯**（benzyl acetate）$\ce{CH3COOCH2Ph}$。验证：δ 7.3–7.5 (m, 5H) = 单取代苯环 5 个芳氢。Ω=5（苯环 4 + C=O 1），C₉H₁₀O₂ 分子式吻合。完整结构：**乙酸苄酯** $\ce{CH3-CO-O-CH2-C6H5}$。

**14.** (a) ¹H NMR 无芳氢信号（δ 6.5–8.5 区域无峰）→ **不含苯环**。(b) IR 1740 cm⁻¹ → **酯 C=O**（不是羧酸，无 OH 宽峰；不是酰胺，波数偏高）。排除羧酸（无 OH 宽峰 2500–3500）和酰胺（波数应 ~1680）。(c) δ 2.04 (s, 3H) → **甲基酮** $\ce{-COCH3}$ 或**乙酸酯的甲基** $\ce{CH3COO-}$。结合酯基判断为 $\ce{CH3COO-}$（乙酸酯的甲基）。δ 1.26 (t, 3H) + δ 4.12 (q, 2H) → **乙酯基** $\ce{-OCH2CH3}$。(d) 结构：**乙酸乙酯** $\ce{CH3COOCH2CH3}$。验证：C₄H₈O₂，Ω = 4+1−8/2 = 1（C=O 双键）。IR 1740 cm⁻¹（酯 C=O），无 OH。¹H NMR 三组信号：CH₃CO (δ 2.04, s, 3H) + OCH₂ (δ 4.12, q, 2H) + CH₃ (δ 1.26, t, 3H)。所有数据一致。

**15.** (a) Ω = 10+1−12/2 = 10+1−6 = **5**（苯环 4 + C=O 1 = 5）。(b) IR 1685 cm⁻¹ → **共轭酮 C=O**（与芳环共轭使波数从正常 1715 降低 ~30 cm⁻¹）。1600 和 1500 cm⁻¹ → 芳环 C=C 双特征峰。无 OH 宽峰 → 排除羧酸和醇。(c) ¹³C NMR 有 9 个峰但分子含 10 个碳 → 分子有**对称性**（两个碳等价）。δ 198.5 → **醛/酮 C=O**（195–215 区域，醛或酮）。(d) δ 7.4–7.6 (m, 3H) + δ 7.95 (d, 2H, J=8 Hz) → **对位二取代苯环**（AA'BB' 体系）：δ 7.95 的 d (2H) 为邻位氢（去屏蔽，靠近 C=O），δ 7.4–7.6 (m, 3H) 为间位 + 对位氢（3H 包括间位 2H 和对位 1H，可能重叠）。但 3H+2H=5H → 单取代苯环。修正：对位二取代应为 2H+2H=4H。5H 说明**单取代苯环**。(e) 综合推断：单取代苯环（C₆H₅–）+ 共轭酮（C=O）+ 丙基侧链（δ 1.05 t, 3H + δ 1.65 sextet, 2H + δ 2.90 t, 2H = C₃H₇）。结构为**丁酰苯**（butyrophenone）$\ce{C6H5-CO-CH2CH2CH3}$。验证：C₁₀H₁₂O，Ω=5。¹³C NMR：C₆H₅（4 类碳）+ C=O（δ 198.5）+ CH₂（δ 31.2, α 位）+ CH₂（δ 26.5, β 位）+ CH₃（δ 13.8）= 9 个峰（苯环 4 类碳 + 3 个脂肪碳 + C=O = 8？实际苯环单取代有 4 类碳：ipso, ortho, meta, para）。总数：ipso-C + 2×ortho + 2×meta + para + C=O + 3×脂肪碳 = 4+1+3 = 8？但数据有 9 个峰 → 可能某些峰重叠后恰好 9。若丁酰苯的苯环 4 类碳 + C=O + 3 个脂肪碳 = 8，差 1 → 可能 CH₂ 的两个碳化学位移接近但不等价？更可能结构为**苯丁酮**（butyrophenone），所有数据吻合。
