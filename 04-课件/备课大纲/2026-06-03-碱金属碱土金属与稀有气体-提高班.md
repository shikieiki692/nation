### 0.2 讲到为止

- 过氧化物/超氧化物：掌握生成条件、与CO₂/H₂O反应、与酸反应产物差异，不展开O₂²⁻/O₂⁻的MO能级图
- 锂的特殊性：掌握Li与同族的差异（熔点/电极电势/燃烧产物/氢氧化物碱性），能用Li⁺半径小+水合热大解释
- 储氢材料：了解PdH和LaNi₅H₆的可逆吸放氢原理，掌握金属型氢化物的非整比特征
- 对角线规则：掌握Li-Mg、Be-Al、B-Si三对相似性的具体表现，能解释Be(OH)₂的两性、Li₂CO₃难溶等，不展开离子势的定量计算
- Xe化合物：掌握XeF₂/XeF₄/XeF₆/XeO₃的制备、水解反应、VSEPR结构判断，不展开Xe-F键的MO分析
- 焰色反应：定性解释（电子跃迁），不展开原子光谱项符号

### 0.3 禁止提前展开

- 禁止引入超氧化物的分子轨道理论解释顺磁性
- 禁止展开稀有气体化合物的完整分子轨道分析
- 禁止用晶体场理论解释碱金属/碱土金属化合物的颜色
- 禁止引入碱金属负离子盐（Na⁻等）的结构化学

### 0.4 结论策略

- 对角线规则 → 用"离子势相近→极化能力相近→化学性质相似"一句话收束
- 溶解度趋势 → 用"晶格能与水合能竞争"定性解释，不计算
- Xe化合物结构 → 用VSEPR统一判断，强调"孤对电子决定实际构型"

| 目标层级 | 具体内容 |
|---------|---------|
| 知识目标 | 掌握碱金属过氧化物/超氧化物/臭氧化物的生成条件与反应；理解碱土金属硫酸盐/碳酸盐溶解度趋势；掌握对角线规则三对相似性；掌握XeF₂/XeF₄/XeF₆/XeO₃的制备与反应 |
| 能力目标 | 能用VSEPR判断Xe化合物结构；能用对角线规则预测Be/Al化合物的相似性；能区分Na₂O₂与酸反应和与CO₂/H₂O反应的产物差异 |
| 素养目标 | 建立"结构决定性质"的分析框架；体会周期表中"反常"现象的内在规律性（对角线规则） |

### 1.3 重难点

| 类型 | 内容 | 突破策略 |
|-----|------|---------|
| 重点 | 对角线规则（Li-Mg、Be-Al、B-Si） | 列表对比+实例驱动（Be(OH)₂两性、Li₂CO₃难溶） |
| 重点 | Xe化合物的VSEPR结构判断 | 先画Lewis结构→算孤对→定构型→对照实际 |
| 难点 | Na₂O₂与酸反应生成H₂O₂（不是直接放O₂） | 分步写方程式，强调"过氧键保留" |
| 难点 | XeF₆水解生成XeO₃（易爆）的条件控制 | 强调"水解剧烈→需控制水量"，联系实验安全 |

## 二、知识脉络与授课路线

### 2.1 路线总览

```text
课时1：碱金属与碱土金属（45 min）
├─ 导入（3 min）：Na₂O₂在呼吸面具中的应用 → 引出碱金属"反常"氧化态
├─ 碱金属深化（15 min）
│  ├─ 过氧化物 Na₂O₂：与CO₂/H₂O反应（放O₂）、与酸反应（生成H₂O₂）
│  ├─ 超氧化物 KO₂：与CO₂反应（4KO₂ + 2CO₂ → 2K₂CO₃ + 3O₂）
│  └─ 臭氧化物 KO₃：不稳定，强氧化性
├─ 焰色反应与离子型氢化物（7 min）
│  ├─ 焰色反应：物理变化（电子跃迁），不是化学变化
│  └─ 离子型氢化物：NaH、CaH₂的还原性
├─ 碱土金属深化（10 min）
│  ├─ Be/Mg的轻质性（密度小）
│  └─ Ca/Sr/Ba的硫酸盐/碳酸盐溶解度趋势（从上到下降低）
├─ 对角线规则（8 min）
│  ├─ Li-Mg：Li₂CO₃难溶、LiF难溶
│  ├─ Be-Al：Be(OH)₂两性、BeCl₂共价性
│  └─ B-Si：B/Si的含氧酸弱酸性
└─ 课时1小结（2 min）："离子势相近→性质相似"

课时2：稀有气体（45 min）
├─ 导入（3 min）：稀有气体"惰性"被打破 → 1962年Bartlett合成XePtF₆
├─ Xe的氟化物（20 min）
│  ├─ XeF₂：直线形，Xe + F₂（过量）→ XeF₂
│  ├─ XeF₄：平面正方形，Xe + F₂（1:5）→ XeF₄
│  ├─ XeF₆：变形八面体，Xe + F₂（1:20）→ XeF₆
│  └─ VSEPR应用：AX₂E₃、AX₄E₂、AX₆E → 构型判断
├─ Xe的氧化物（10 min）
│  ├─ XeO₃：XeF₆水解（剧烈，易爆），三角锥形
│  └─ XeO₄：XeF₆ + H₂O（控制条件），四面体形
├─ 真题切片：39届Q5（10 min）
│  ├─ Xe化合物晶胞分析
│  └─ VSEPR综合应用
└─ 课时2小结（2 min）："VSEPR是判断稀有气体化合物结构的核心工具"
```

## 七、下节衔接

- 下节内容：卤素与氧族元素（第二轮）
- 衔接点：
  - 本节XeF₂/XeF₄/XeF₆中的F元素 → 下节卤素的氧化态与含氧酸
  - 本节Na₂O₂中的O元素 → 下节过氧化氢与臭氧
  - 本节对角线规则 → 下节卤素含氧酸的"反常"酸性（HClO₄ > HBrO₄）
- 前置提醒：学生需复习第一轮卤素的基础知识（Cl₂的制备、卤化氢的酸性）
