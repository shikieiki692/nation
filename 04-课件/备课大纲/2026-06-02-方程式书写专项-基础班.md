# 方程式书写专项（基础班）备课大纲

## 1. 课程定位与目标

### 1.1 定位
- 这是"贯穿全程的方程式书写专项训练"的第一课
- 聚焦：氧化还原方程式配平（离子-电子法）
- 为后续元素化学、电化学、分析化学奠定方程式书写基础

### 1.2 教学目标
| 目标层级 | 具体内容 |
|---------|---------|
| 知识目标 | 掌握氧化态判定、介质判断、离子-电子法配平步骤 |
| 技能目标 | 能独立完成常见氧化还原反应的方程式配平 |
| 方法目标 | 建立"介质优先"的配平思维习惯 |
| 竞赛目标 | 为初赛方程式书写题争取满分 |

### 1.3 课时分配（90分钟）
| 环节 | 时间 | 内容 |
|-----|------|------|
| 课堂引入 | 5min | 问题导入：为什么方程式书写是竞赛基本功？ |
| 知识讲解 | 35min | 氧化态判定、介质判断、离子-电子法 |
| 课堂例题 | 25min | 4道基础题 + 2道拓展题 |
| 即时演练 | 15min | 学生独立练习，教师巡视 |
| 总结与作业 | 10min | 核心结论、易错点、课后作业 |

> 🎯 切入点：技能训练课，以"初赛方程式题占15-25分，为什么很多学生'知道反应但写不对'？"开场，直击痛点，建立"系统方法替代凭感觉"的课堂价值
> ⚠️ 预警：(1) 碱性介质配平时误用H⁺——核心纪律"碱性条件绝不允许出现游离H⁺"；(2) Cr₂O₇²⁻误以为得3e⁻——1个Cr₂O₇²⁻含2个Cr，整体得6e⁻；(3) MnO₄⁻还原产物不随介质变化——酸性→Mn²⁺、中性→MnO₂、碱性→MnO₄²⁻
> 📋 例题链：KMnO₄在不同介质中氧化Na₂SO₃(A) → Cr₂O₇²⁻酸性氧化Fe²⁺(B) → Al与NaOH反应(C) → Cl₂歧化(D) → Zn与NaOH拓展(E) → Cu与浓HNO₃拓展(F)
> ✂️ 边界决策：
>   - 纳入：氧化态判定、介质判断、离子-电子法配平五步骤、常见氧化剂不同介质还原产物
>   - 砍掉：复杂多元素同时变价（As₂S₃型，第二轮）
>   - 后移：有机氧化还原方程式（第三轮）、非水体系反应（第二轮）

## 2. 教学内容设计

### 2.1 课堂引入（5分钟）

**引入问题：**
> 化学竞赛中，方程式书写题占多少分？为什么很多学生会"知道反应但写不对"？

**要点：**
- 初赛方程式题通常占15-25分
- 常见失分原因：产物写错、配平错误、介质判断错误、电荷不守恒
- 本节课目标：建立系统方法，杜绝"凭感觉写方程式"

### 2.2 核心知识讲解（35分钟）

#### 2.2.1 氧化态判定与介质判断（10分钟）

**氧化态判定规则（复习）：**
1. 单质中元素氧化态为0
2. 离子化合物中：氧化态 = 离子电荷
3. 共价化合物中：电负性大的原子得电子
4. H通常为+1（金属氢化物中为-1）
5. O通常为-2（过氧化物中为-1，超氧化物中为-1/2，OF₂中为+2）

**介质判断原则：**
| 判断依据 | 酸性介质 | 碱性介质 | 中性介质 |
|---------|---------|---------|---------|
| 题目明确说明 | "酸性条件"、"稀H₂SO₄" | "碱性条件"、"NaOH" | "中性"、"水溶液" |
| 反应物特征 | 含H⁺、强酸 | 含OH⁻、强碱 | 无强酸强碱 |
| 产物提示 | 生成CO₂（碳酸盐分解） | 生成CO₃²⁻ | 视情况而定 |

**核心纪律：**
> **碱性条件绝不允许出现游离H⁺，酸性条件绝不允许出现游离OH⁻**

#### 2.2.2 离子-电子法配平步骤（15分钟）

**配平口诀：** 介质定 → 产物定 → 原子平 → 电荷平

**详细步骤：**
1. **写出半反应**：分别写出氧化半反应和还原半反应
2. **配平原子**：用H₂O配平O，用H⁺（酸性）或OH⁻（碱性）配平H
3. **配平电荷**：用电子配平电荷
4. **合并半反应**：使电子得失相等，相加消去电子
5. **检查守恒**：原子守恒、电荷守恒

**酸性介质示例（MnO₄⁻ → Mn²⁺）：**
```
① 写出骨架：MnO₄⁻ → Mn²⁺
② 配平Mn：已平
③ 配平O（加H₂O）：MnO₄⁻ → Mn²⁺ + 4H₂O
④ 配平H（加H⁺）：MnO₄⁻ + 8H⁺ → Mn²⁺ + 4H₂O
⑤ 配平电荷（加e⁻）：MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
```

**碱性介质处理技巧：**
- 先在酸性条件下配平
- 两边加等量OH⁻中和H⁺
- 合并H⁺ + OH⁻ → H₂O
- 化简

#### 2.2.3 常见氧化剂在不同介质中的还原产物（10分钟）

| 氧化剂 | 酸性介质 | 中性/弱碱性 | 强碱性介质 |
|-------|---------|-----------|-----------|
| KMnO₄ | Mn²⁺（无色） | MnO₂↓（棕黑） | MnO₄²⁻（绿色） |
| K₂Cr₂O₇ | Cr³⁺（绿色） | — | — |
| HNO₃(浓) | NO₂ | — | — |
| HNO₃(稀) | NO | — | — |
| H₂SO₄(浓) | SO₂ | — | — |
| H₂O₂ | H₂O | — | — |
| Cl₂ | Cl⁻ | — | Cl⁻ / ClO⁻（歧化） |

**记忆要点：**
- MnO₄⁻：酸性降5价（Mn⁷⁺→Mn²⁺），中性降3价（Mn⁷⁺→Mn⁴⁺），碱性降1价（Mn⁷⁺→Mn⁶⁺）
- Cr₂O₇²⁻：只在酸性条件下作氧化剂（Cr⁶⁺→Cr³⁺，每个Cr得3e⁻，整体得6e⁻）

### 2.3 课堂例题设计（25分钟）

#### 例题1：KMnO₄在不同介质中氧化Na₂SO₃（⭐⭐，必讲）
- 酸性：$2\text{MnO}_4^- + 5\text{SO}_3^{2-} + 6\text{H}^+ \rightarrow 2\text{Mn}^{2+} + 5\text{SO}_4^{2-} + 3\text{H}_2\text{O}$
- 中性：$2\text{MnO}_4^- + 3\text{SO}_3^{2-} + \text{H}_2\text{O} \rightarrow 2\text{MnO}_2\downarrow + 3\text{SO}_4^{2-} + 2\text{OH}^-$
- 碱性：$2\text{MnO}_4^- + \text{SO}_3^{2-} + 2\text{OH}^- \rightarrow 2\text{MnO}_4^{2-} + \text{SO}_4^{2-} + \text{H}_2\text{O}$

**教学要点：** 同一反应物在不同介质中产物不同，强调"介质优先"

#### 例题2：Cr₂O₇²⁻酸性条件下氧化Fe²⁺（⭐⭐，必讲）
- $\text{Cr}_2\text{O}_7^{2-} + 6\text{Fe}^{2+} + 14\text{H}^+ \rightarrow 2\text{Cr}^{3+} + 6\text{Fe}^{3+} + 7\text{H}_2\text{O}$

**教学要点：** Cr₂O₇²⁻含2个Cr，整体得6e⁻；注意系数关系

#### 例题3：Al与NaOH溶液反应（⭐⭐，必讲）
- $2\text{Al} + 2\text{OH}^- + 6\text{H}_2\text{O} \rightarrow 2[\text{Al(OH)}_4]^- + 3\text{H}_2\uparrow$
- 或写成：$2\text{Al} + 2\text{NaOH} + 6\text{H}_2\text{O} \rightarrow 2\text{Na}[\text{Al(OH)}_4] + 3\text{H}_2\uparrow$

**教学要点：** Al是两性金属；碱性条件下产物是四羟基合铝酸盐

#### 例题4：Cl₂与NaOH反应（歧化反应）（⭐⭐，必讲）
- 冷稀碱：$\text{Cl}_2 + 2\text{OH}^- \rightarrow \text{Cl}^- + \text{ClO}^- + \text{H}_2\text{O}$
- 热浓碱：$3\text{Cl}_2 + 6\text{OH}^- \xrightarrow{\Delta} 5\text{Cl}^- + \text{ClO}_3^- + 3\text{H}_2\text{O}$

**教学要点：** 歧化反应中同一元素既升又降；温度影响产物

#### 例题5（拓展）：Zn与NaOH反应（⭐⭐⭐，选讲）
- $\text{Zn} + 2\text{OH}^- + 2\text{H}_2\text{O} \rightarrow [\text{Zn(OH)}_4]^{2-} + \text{H}_2\uparrow$

#### 例题6（拓展）：Cu与浓HNO₃反应（⭐⭐⭐，选讲）
- $\text{Cu} + 4\text{HNO}_3(\text{浓}) \rightarrow \text{Cu(NO}_3)_2 + 2\text{NO}_2\uparrow + 2\text{H}_2\text{O}$

### 2.4 即时演练（15分钟）

**练习题（学生独立完成，教师巡视）：**
1. 配平：KMnO₄ + H₂C₂O₄ + H₂SO₄ → （酸性条件）
2. 配平：Cl₂ + NaOH(热浓) → 
3. 写出离子方程式：Zn + NaOH + H₂O → 
4. 判断介质并配平：MnO₄⁻ + S²⁻ → （碱性条件）

### 2.5 课堂总结（10分钟）

**核心结论（4-5条）：**
1. 介质判断是配平的第一步，决定产物形态
2. 离子-电子法四步：介质定→产物定→原子平→电荷平
3. 碱性条件绝不允许出现游离H⁺，酸性条件绝不允许出现游离OH⁻
4. 歧化反应中同一元素既被氧化又被还原
5. 配平后必须检查原子和电荷守恒

**易错点预警：**
| 错误想法 | 为什么错 | 正确理解 |
|---------|---------|---------|
| "配平就是凑系数" | 缺乏系统方法，容易出错 | 使用离子-电子法，按步骤进行 |
| "碱性条件也可以写H⁺" | 违反基本化学事实 | 碱性条件只能用OH⁻和H₂O配平 |
| "Cr₂O₇²⁻得3个电子" | 忽略了1个Cr₂O₇²⁻含2个Cr | Cr₂O₇²⁻整体得6e⁻（每个Cr得3e⁻） |
| "MnO₄⁻总是还原成Mn²⁺" | 忽略了介质影响 | 酸性→Mn²⁺，中性→MnO₂，碱性→MnO₄²⁻ |

```
   1. 写出半反应

   4. 合并半反应
   5. 检查守恒

   碱性条件：禁止出现H⁺

```

## 6. 教学反思要点

### 6.1 预期困难
1. 学生在碱性介质配平时容易混淆H⁺和OH⁻的使用
2. 对Cr₂O₇²⁻整体得6e⁻的理解可能有困难
3. 离子方程式和化学方程式的转换不熟练

### 6.2 应对策略
1. 碱性介质配平：强调"先用酸性条件配平，再加OH⁻中和"的技巧
2. Cr₂O₇²⁻：用颜色标注2个Cr，强调"整体思维"
3. 离子方程式：先写离子方程式，再补充 spectator ions 得到化学方程式

### 6.3 评价要点
- 能否正确判断介质类型
- 能否正确使用离子-电子法
- 配平后的方程式是否满足原子和电荷守恒
- 是否理解"介质优先"原则
