# 模块连接推进-分析化学

> 本页用于把“分析化学”模块的连接工程落到可执行层。

## 一、当前模块判断

- `分析化学` 当前约有 `35` 个 KP。
- 体量不大，但教材提炼和网课提炼的可教学价值很高，适合做高密度深连接样板。
- 当前最适合先推进的，不是平均扫全模块，而是优先围绕 `误差 / 滴定 / 指示剂 / 滴定曲线` 这条课堂主链。

## 二、第一批优先专题

### 1. 误差与数据处理

**优先提炼来源**
- `提炼-分析化学第六版-第3章-误差与数据处理`
- `教学逻辑提炼-分析化学网课-容量分析概论与酸碱滴定`
- `教学逻辑提炼-周坤无机新课-定量化学分析-第一轮`

**优先回补方向**
- 误差 / 偏差 / 准确度 / 精密度 / 有效数字

### 2. 酸碱滴定主链

**优先提炼来源**
- `提炼-分析化学第六版-第8章-酸碱滴定法`
- `教学逻辑提炼-分析化学网课-容量分析概论与酸碱滴定`
- `教学逻辑提炼-周坤无机新课-定量化学分析-第一轮`

**优先回补方向**
- 酸碱滴定 / 滴定曲线 / 指示剂 / 标准溶液 / 基准物质

## 三、推进顺序

1. 先补 `误差与数据处理`
2. 再补 `酸碱滴定主链`
3. 然后向 `沉淀滴定 / 络合滴定 / 氧化还原滴定` 扩展

## 三点五、当前已推进到哪里（2026-06-18）

- 已补通：
  - `误差 / 偏差 / 准确度 / 精密度`
  - `酸碱滴定 / 滴定曲线 / 指示剂 / 标准溶液 / 基准物质`
  - `沉淀滴定 / 络合滴定 / 氧化还原滴定`
  - `数据处理 / 条件电势 / 条件稳定常数 / 终点误差 / 分光光度法 / Lambert-Beer定律 / 比色分析 / 副反应系数 / 电荷平衡 / 碘量法 / 重量分析`
- 说明：分析化学当前已经形成了“定义层 + 酸碱滴定主链”两层支点，不再是零散页状态。
- 下一步最值得继续补的，是：
  1. 分析模块中更偏边缘或决赛化的谱学页（如 NMR 相关）另行分流处理
  2. 个别历史页的尾部老结构再做轻量审计
  3. 继续把这些页与专题、讲义、题库抽料链压得更紧

## 四、执行标准

- 优先让提炼页中的“课堂判断规则、公式口径、误区类型、例题链”能沉到 KP。
- 优先补 `source_extracts`，而不是只补 `source_notes`。
- 尽量形成“方法论底层页 → 具体滴定页 → 题型/讲义可抽料”的链条。
