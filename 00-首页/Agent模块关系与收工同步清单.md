# Agent模块关系与收工同步清单

> **用途**：给 Agent 一份真正可在 1-2 分钟内读完的“模块地图 + 收工回写矩阵”。
> 当任务跨模块、或你不确定“该先读哪里 / 做完要回写哪里”时，优先看本页。

---

## 一、知识库模块关系（90 秒版）

```text
01/02 考纲与数据库
    ↓ 定义边界 / 提供筛选入口
03 知识点（KP）
    ↔ 04-题库（题目通过 knowledge_points 反向关联）
    ↔ 04-专题与题型（专题页 / 题型页聚合解题语言）
    ↑
07 资料提炼 / 教学逻辑提炼
    ↑
06 外部资料导入

03 + 04 + 备课思路
    ↓
04-课件（备课大纲 / 新授课 / 习题课 / 复习课）
    ↓
06-学生侧材料（闪卡 / 练习卷 / 策略卡）
    ↓
学生反馈 / 复盘 / 下一轮优化

00-首页系统层
    状态摘要：会话快照
    Agent战略方向：长期优先级
    活跃任务：短期执行真相源
    工作日志：过程与证据
```

### 一句话理解

| 模块 | 核心问题 | 主要输入 | 主要输出 |
|---|---|---|---|
| `06-外部资料导入` | 原始资料放哪 | 教材 / 网课 / 扫描件 / 真题 | 可提炼源文件 |
| `07-资料提炼` | 外部资料里哪些值得吸收 | 原始资料 | 知识、教学逻辑、可回流资产、成品闭环前置素材 |
| `03-知识点` | 这个概念到底是什么 | 提炼稿 / 既有 KP / 考纲 | 定义、原理、易错点、教学视角 |
| `04-专题与题型` | 多个 KP 怎么组织成可教可用的模块 | KP + 题库 + 提炼稿 | 专题页、题型页、解题引擎 |
| `04-题库` | 哪些题对应哪些知识、哪些题能直接拿来组卷 | 真题 / 习题 / 提炼稿 | 题目文件、knowledge_points 关联、[[02-数据库/数据库-真题]] 查询入口 |
| `04-课件` | 这节课到底怎么讲 | 专题页 + KP + 教学逻辑 + 题库 | 备课大纲、讲义、习题课、课堂执行页 |
| `非正文交付层` | 这些内容怎么真正进课堂 / 发到学生手里 | 成熟正文 + 加工页 + 样板专题 + 验收标准 | PPT、docx、教具、工具卡、评分表、反馈闭环 |
| `06-学生侧材料` | 学生课后怎么复用 | KP / 专题页 / 课件 / review_cycle / [[KP复习清单]] | 闪卡、练习卷、策略卡 |
| `00-首页系统层` | 现在库里什么状态、该做什么 | 上述所有层的结论 | 快照、方向、任务、证据 |

## 二、从用户任务反推读取路径

| 用户任务类型 | 开工前最少读取 | 产出后必须回写 |
|---|---|---|
| 资料提炼 / 教学逻辑提炼 | `状态摘要` + `活跃任务` + `Agent战略方向` + 对应模板 + `方法论-多源教学资料整合` | `工作日志` + `活跃任务` + 必要时 `状态摘要/Agent战略方向` |
| 新建 / 填充 KP | 上述三文件 + `模板-知识点` + 相关 KP 1-2 个 + 对应考纲 | `工作日志` + `活跃任务` + 相关索引 / 题目链接 |
| 专题页 / 题型页建设 | 上述三文件 + 相关 KP + 题库 + `模板-专题` | `工作日志` + `活跃任务` + 必要时 `备课与教学思路待办` |
| 备课 / 新授课 / 习题课 | 上述三文件 + `备课与教学思路待办` + 相关专题页 + KP + 教学逻辑提炼 + 对应模板 | `工作日志` + `活跃任务` + `备课与教学思路待办` + 必要时 `状态摘要/Agent战略方向/首页` |
| 非正文交付物（PPT / docx / 教具 / 工具卡 / 评分表） | 上述三文件 + [[备课思路/四轮内容完成度总表]] + [[备课思路/非正文剩余工作落地清单]] + 对应样板专题真相源 | `工作日志` + `活跃任务` + `04-课件/专题产出总览看板` + 必要时 `状态摘要/Agent战略方向/首页` |
| 学生侧材料 | 上述三文件 + `06-学生侧材料/` + 对应模板 + 相关课件 / 专题页 / KP | `工作日志` + `活跃任务` + 必要时 `状态摘要/Agent战略方向` |
| 系统维护 / 审计 / 工作流 | SOP / 导航说明 / 数据库指南 / 任务卡 | `00-首页/` + `11-模板/` | `工作日志` + `状态摘要`（若结论变化） |

> 默认理解：三文件指 `[[状态摘要]] → [[活跃任务]] → [[Agent战略方向]]`。其中 `[[活跃任务]]` 先看健康检查，再看 P1/P2/P3/blocked。

## 四、收工同步矩阵（硬规则）

| 本次改动类型 | 必须同步 | MCP 工具对应 |
|---|---|---|
| 新增 / 修改 `07-资料提炼` | `工作日志`；若改变主线或补齐重要盲区，再同步 `状态摘要` / `Agent战略方向` | `kb_write` + `kb_dirty mark` |
| 新增 / 批量修改 KP | `工作日志`；必要时 `知识点总索引` / `题库链接` / `活跃任务` | `kb_write` / `kb_edit` + `kb_dirty mark` |
| 新增 / 升级专题页 | `工作日志`；若改变备课闭环状态，同步 `备课与教学思路待办` 和 `活跃任务` | `kb_write` / `kb_edit` + `kb_dirty mark` |
| 生成任何 `04-课件/` 文件 | `工作日志` + `活跃任务` + `备课与教学思路待办`（硬规则） | `kb_write` + `kb_dirty mark` |
| 新增非正文交付物（PPT / docx / 教具 / 工具卡 / 评分表） | `工作日志` + `活跃任务` + `04-课件/专题产出总览看板`；若改变阶段判断，再同步 `状态摘要/Agent战略方向/首页` | `kb_write` + `kb_dirty mark` |
| 改动 `02-数据库/数据库-真题`、题库查询口径或组卷入口 | `工作日志` + `活跃任务`；若改变默认题库使用路径，再同步 `首页/状态摘要/Agent战略方向` 中的相关描述 | `kb_edit` + `kb_dirty mark` |
| 改动 `方法论-多源教学资料整合` 或提炼→备课闭环规则 | `工作日志` + 必要时 `Agent标准作业流程` / `Agent模块关系与收工同步清单` / `状态摘要` | `kb_edit` + `kb_dirty report` |
| `化学竞赛初赛讲义` 这类持续性资料线推进了默认顺序或方法 | `工作日志` + 对应任务卡；若影响入口判断，再同步 `状态摘要` / `Agent战略方向` / 本页 | `kb_write` + `kb_dirty mark` |
| 关闭 / 阻塞一张任务卡 | `活跃任务卡` frontmatter；必要时同步 `状态摘要` 的阻塞 / 下一步 | `kb_edit` + `kb_gate close` / `kb_gate verify` |
| 改变阶段判断、阻塞项、下一步 | `状态摘要` + `Agent战略方向`；大任务时同步 `首页.current_focus` | `kb_edit` + `kb_dirty report` |
| 模板 / SOP 版本变更 | 对应模板文件 + `状态摘要` 体系版本表 + `首页.sop_version`（若改 SOP） | `kb_edit` + `kb_dirty report` |
| 学生侧材料试点、反馈、导出工作流 | `工作日志` + `活跃任务`；若改变学生侧主线或阻塞，再同步 `状态摘要/Agent战略方向` | `kb_write` + `kb_dirty mark` |

### 快速判断

- **只改过程，不改结论**：通常只写 `工作日志`
- **改了待办状态**：必须写 `活跃任务`
- **改了阶段 / 阻塞 / 下一步**：必须写 `状态摘要`
- **改了长期优先级 / 待用户决策项**：必须写 `Agent战略方向`
- **改了课件进度**：必须写 `备课与教学思路待办`

## 五、最容易漏掉的 6 件事

1. 课件生成完了，但没回写 `备课与教学思路待办`
2. 任务正文里写“已完成”，frontmatter 还是 `active`
3. 审计结论已经接受“复用已有产物”，却还挂着一个等待“补做”的旧任务
4. 任务其实在等用户 / 打印 / 课堂反馈，却一直占着 `active`
5. 改了 SOP / 模板版本，却没同步 `状态摘要` 的体系版本表
6. 学生侧材料开始试点了，但 `06-学生侧材料/`、`KP复习清单`、Anki、反馈闭环仍各写各的

## 六、推荐使用方式

### 日常小任务

```text
状态摘要 → 活跃任务 → 本页 → 开工
```

### 跨模块任务

```text
状态摘要 → 活跃任务 → Agent战略方向 → 本页 → 方法论 / SOP 对应章节 → 开工
```

### 收工前 20 秒

```text
工作日志写了吗？
任务卡状态对了吗？
如果生成了 04-课件，待办表更了吗？
如果结论变了，状态摘要 / 战略方向更了吗？
```
