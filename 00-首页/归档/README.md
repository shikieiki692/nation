## 归档规则

1. 只有当文件已经不再是当前真相源、主入口或默认工作流组成部分时，才移入本目录。
2. 归档后优先保留原文件名，便于旧 wikilink 和历史日志继续解释上下文。
3. 当前页或系统页若仍需提到旧文件，应显式写成“已归档 / 历史稿 / 不再作为真相源”。
4. 归档不等于删除；若后续需要回看旧策略、旧可视稿或早期实验稿，统一从本目录进入。
