# 结构化学讲义低质手绘图替换与图片提质

## 目标

让结构化学主线讲义在不继续膨胀图片数量的前提下，优先提升学生可见的图片质量和讲图效率。

## 当前判断

- 前一阶段“有没有图”的问题已经基本收口。
- 现在真正影响观感和课堂使用的，是：
  - 少量低质手绘图或包装图
  - 明明图库已有更好素材，但讲义仍沿用自绘简图
  - 图片索引虽然可用，但分类与复用提示还不够利于后续 Agent 调图

## 优先处理对象

1. 分子结构基础
2. 晶体学基础
3. 晶体结构基础
4. 配位化合物基础

## 完成标准

- 学生可见低质图显著减少
- 新替换图片全部落到 `media/`
- 图片索引能指出更细的类别与稳定路径
- 替图后至少做一次对应 PDF 抽样回归
