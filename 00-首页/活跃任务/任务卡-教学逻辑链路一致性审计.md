# 任务卡 · 教学逻辑链路一致性审计

> **审计范围**：备课体系从顶层设计到课堂落地的完整链路，确保每一步有据可查、链接有效、内容一致。
>
> 比常规 wikilink 检查更深入——审计涵盖**教学洞察**（12-教学洞察/下的教学笔记是否被正确引用）、**教学思路**（备课思路/下的教学逻辑文件是否得到体现）和**考纲**（2024考纲考点是否在讲义中有对应落点）。

---

## 审计链路全景

```
班级授课计划                          ← 顶层：课次安排+时间分配
    ↓
教学过程总表                          ← 中层：KP级映射+教学策略
    ↓
备课大纲                              ← 执行层：depth_boundary+课时规划
    ↓
学生讲义                              ← 内容层：四步循环+配图+例题
    ↓
KP知识点                              ← 知识层：03-知识点/下的独立KP
    ↓
专题页                                ← 综合层：跨知识点综合应用
    ↓
教学洞察                              ← 经验层：教学笔记+错误模式+类比
    ↓
教学逻辑提炼                          ← 策略层：来源资料提炼
    ↓
考纲 / 真题                           ← 目标层：最终对齐
```

## 审计清单

### Layer 1：班级授课计划 → 教学过程总表

| 检查项 | 方法 | 通过标准 |
|:---|:---|---:|
| 课次对应一致性 | 逐课对比 | 班级授课计划中每课的📄讲义引用与教学过程总表讲义列完全一致 |
| 年级/课时信息一致性 | 检查frontmatter | class_profile/sessions/total_sessions字段一致 |
| 学生基线描述一致性 | 对比文本 | 对"已学内容""已掌握能力"的描述在两份文件中不矛盾 |

### Layer 2：教学过程总表 → 备课大纲

| 检查项 | 方法 | 通过标准 |
|:---|:---|---:|
| KP清单匹配 | 逐KP对照 | 总表中的KP清单与备课大纲中的核心KP/知识点集一一对应 |
| 课时规划一致性 | 对比时间分配 | 总表的课时规划与备课大纲的§一 课时分配不冲突 |
| depth_boundary一致性 | 检查深度边界 | 总表的"教学策略"深度与备课大纲的depth_boundary字段一致 |
| 备课大纲存在性 | Glob验证 | 总表中引用的每个备课大纲文件都存在且未deprecated |

### Layer 3：备课大纲 → 学生讲义

| 检查项 | 方法 | 通过标准 |
|:---|:---|---:|
| 讲义引用有效性 | Glob验证 | 备课大纲引用的每份讲义文件存在且非deprecated |
| KP → 讲义§ 映射 | 逐项对比 | 备课大纲中的知识点/§编号在讲义正文中有匹配内容 |
| depth_boundary与实际内容一致 | 抽检3处 | 讲义中内容的深度不超过备课大纲定义的depth_boundary |
| 前置知识检查清单有效 | 验证wikilink | §0.0中所有wikilink指向存在的文件 |

### Layer 4：讲义 → KP → 专题页

| 检查项 | 方法 | 通过标准 |
|:---|:---|---:|
| KP引用有效性 | 扫描🔗标记 | 讲义中🔗引用的每个KP文件存在且frontmatter正确 |
| 专题页引用有效性 | 扫描📋标记 | 讲义中📋引用的专题页文件存在 |
| KP覆盖完整性 | 对比教学过程总表KP清单 | 讲义正文确实覆盖了总表中列为⭐核心的所有KP |
| 专题页内容与讲义互补 | 抽检2个专题页 | 专题页不是讲义的简单重复，提供不同视角（综合/速查/归纳） |

### Layer 5：教学洞察引用审计

| 检查项 | 方法 | 通过标准 |
|:---|:---|---:|
| 🧠 教学洞察文件存在性 | Glob验证 | 12-教学洞察/下所有被引用的文件存在 |
| 🧠 callout内容的准确性 | 交叉验证 | 讲义中引用的认知冲突/类比/错误模式在对应的教学洞察文件中有出处 |
| 教学洞察覆盖完整性 | 对比教学过程总表附录B | 教学洞察索引表中列出的每份洞察文件确实有在对应课次的讲义中被引用 |

### Layer 6：教学逻辑提炼引用审计

| 检查项 | 方法 | 通过标准 |
|:---|:---|---:|
| 教学逻辑文件存在性 | Glob验证 | 07-资料提炼/下被引用的文件存在 |
| 教学台阶与讲义的匹配 | 对比台阶映射 | 讲义中的内容深度与教学逻辑提炼中定义的"第几个台阶"匹配 |
| 来源资料的可追溯性 | 抽检2份提炼 | 教学逻辑提炼中的原始来源（质心/周坤/Zchem）有明确的提炼日期和提炼范围 |

### Layer 7：考纲对齐审计

| 检查项 | 方法 | 通过标准 |
|:---|:---|---:|
| 2024考纲考点 → 讲义覆盖 | 逐考点比对 | 2024考纲中与结构化学/化学原理相关的每个考点，至少在一份讲义中有对应落点 |
| 考纲未覆盖内容 | 标记超纲 | 讲义中超出2024考纲的内容需有🌟扩展/🟢扩展标记 |
| 初赛真题频率 → 课时权重 | 对比真题频率统计 | 高频考点（近年每年出现）在教学中给予的课时权重合理 |

- 晶体结构：密堆积/晶胞/典型结构/半径比/晶格能
- 配合物：配位键/异构体/CFT/CFSE/高低自旋/光谱化学序
