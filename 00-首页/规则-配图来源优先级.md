# 规则：配图来源优先级

> **用途**：为学生讲义、备课页、知识点页提供统一的找图与落图规则。
> **核心目标**：提高图片复用率，减少重复截图，避免 `media/` 命名和引用失控。

---

## 一、先记住这 6 条硬规则

1. 学生讲义最终使用的图片，一律落到 vault 根目录 `media/`。
2. `[[图片索引]]` 是默认第一站；先复用已有图，再去新找图。
3. 新图片文件名使用稳定英文名，不再新增 `Pasted image ...`、中文临时名、带空格的混乱命名。
4. `mineru/`、教材 OCR、真题解析中的图，只是**来源仓**；进入学生讲义前先复制到 `media/`。
5. 学生侧图注只写“图中对象 + 本讲用途”，控制在 30 汉字左右；不在图注里写来源说明。
6. Excalidraw / Mermaid 只用于流程、逻辑、分类、对比，不用于伪造教材级结构图、机理图、谱图、晶胞图。

---

## 二、默认来源优先级

```text
第一优先  图片索引中的精标表 / 图注笔记 / 已沉淀讲义图
第二优先  mineru / 教材 OCR / 真题原图 / 真题解析中的已有图片
第三优先  其他已验证的库内资料（课件资产、附件图、专题沉淀）
第四优先  外部高质量来源（仅在库内没有合适图时）
第五优先  Excalidraw / Mermaid 自制图（仅限逻辑型图）
```

### 什么时候必须先查图片索引

- 写学生讲义配图
- 给 KP 补关键图
- 给专题页补“第二张教学图”
- 准备把某张图复用到第二个以上场景

---

## 三、不同图片类型的来源口径

| 图片类型 | 推荐来源 | 是否允许自绘 | 说明 |
|:---|:---|:---:|:---|
| 分子结构 / 构型图 | 教材原图 / OCR / 已验证外部图 | 否 | 不要用手绘替代正式结构图 |
| 轨道图 / 能级图 / MO 图 | 教材原图 / OCR | 否 | 尤其不要临时拼接伪图 |
| 晶胞 / 堆积 / 空隙图 | 教材原图 / 真题原图 / OCR | 否 | 结构类图优先复用知识库资产 |
| 反应机理图 | 教材原图 / 真题原图 / 已验证资料 | 原则上否 | 概念流程可画，正式机理箭头图优先用现成图 |
| 谱图 / 曲线 / 相图 | 教材原图 / 真题原图 | 否 | 这类图信息密度高，自绘失真风险大 |
| 实验装置图 | 教材原图 / 真题原图 / 已验证资料 | 尽量否 | 只有极简流程示意才考虑自绘 |
| 分类树 / 决策树 / 因果链 | 自制图优先 | 是 | 这是 Excalidraw / Mermaid 的主战场 |
| 对比表 / 流程图 / 方法论图 | 自制图优先 | 是 | 只要不冒充教材原图即可 |

## 四、学生讲义图片的落地规范

### 1. 最终引用格式

```markdown
![[media/atomic-structure-h-spectrum.png]]

*图 1 氢原子光谱与跃迁对应关系*
```

### 2. 文件命名规则

- 使用英文小写或 kebab-case
- 推荐结构：`topic-keyword-purpose.ext`
- 示例：`atomic-structure-h-spectrum.png`
- 示例：`slater-rule-na-example.png`
- 示例：`perovskite-tetragonal-distortion.jpg`

### 3. 格式规则

- 默认优先 `.png`
- 扫描图、照片、教材截图可用 `.jpg` / `.jpeg`
- `.svg` 仅在本地编译已验证时使用；未验证前先转 `.png`
- 同一张图只保留一个稳定投产名，不重复保存多个改名副本

### 4. 图注规则

- 只写图中对象和教学用途
- 控制在 30 汉字左右
- 不写“来源：某教材 / 某网站”
- 不把长解释塞进图注，长说明回正文

## 五、Agent 默认找图工作流

```text
讲义需要图
  ↓
先看 [[图片索引]] 的精标表
  ↓
未命中 → 看同主题已知检索 hint
  ↓
再去 mineru / 教材 OCR / 真题原图目录定位
  ↓
命中后复制到根目录 media/
  ↓
在讲义中用 ![[media/...]] 引用
  ↓
若值得复用，回写 [[图片索引]]
```

### 回写图片索引的触发条件

- 找到一张以后高概率复用的图
- 某主题第二次还得重复检索
- 新识别出一类 OCR 噪声图
- 新建了有价值的图注笔记

---

## 六、禁止项

- 不要在最终学生讲义里直接保留 `![[mineru/...]]`
- 不要继续新增 `Pasted image ...` 这类文件名
- 不要把来源说明写进学生侧图注
- 不要用 Excalidraw 临摹教材级轨道图、晶胞图、谱图
- 不要为了局部使用再复制出第二份近似同图
- 不要把图片资产散落到多个临时目录后再人工记忆引用

---

## 七、特殊说明

### 当前样板的豁免

[[原子结构-超级充实版（自学完整）.md]] 当前图片引用已单独修整完成，不再为“追求一致”强行改动。  
本规则主要约束**之后的新讲义和重构稿**。

### 外部图片的使用边界

- 只有在知识库内没有合适图时，才使用外部图片
- 外部图进入讲义前，同样要复制到 `media/`
- 若来源可靠但未来复用价值高，应回写到 `[[图片索引]]`
