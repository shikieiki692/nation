# 规则：配图来源优先级

> **用途**：为学生讲义配图时，明确不同图片类型的来源选择顺序。
> **适用范围**：所有讲义（含超级充实版）的配图工作。

---

## 核心规则

```
配图时按以下优先级选择来源：

🥇 第一优先：教材原图（从OCR原稿或PDF提取）
🥈 第二优先：网络下载（教材无合适图时）
🥉 第三优先：Excalidraw自绘（仅限流程图、思维导图、概念关系图）
```

---

## 各类型图的具体规则

### 1. 分子结构类 — 只能用教材原图或网络下载

| 图片类型 | 禁止方式 | 推荐来源 |
|:--------|:--------|:--------|
| 分子结构（Lewis结构/球棍模型/填充模型） | ❌ Excalidraw自绘 | 教材原图或SciFinder/Google图片 |
| 轨道形状（s/p/d轨道角度分布） | ❌ Excalidraw自绘 | 教材原图 |
| MO能级图（同核/异核双原子） | ❌ Excalidraw自绘 | 普化原理§12、§6 OCR原稿 |
| 晶体结构图（堆积/晶胞/空隙） | ❌ Excalidraw自绘 | 普化原理§13 OCR原稿 |
| VSEPR构型图（三角双锥/八面体等） | ❌ Excalidraw自绘 | 教材原图或OCR原稿 |
| σ/π键形成示意图 | ❌ Excalidraw自绘 | 教材原图 |
| 光谱/衍射图 | ❌ Excalidraw自绘 | 教材原图 |

### 2. 流程图/思维导图 — 可用Excalidraw

| 图片类型 | 允许方式 | 示例 |
|:--------|:--------|:-----|
| 概念关系图（层级/分类） | ✅ Excalidraw | 晶体分类树、杂化类型对比 |
| 流程步骤图 | ✅ Excalidraw | 晶体学三步分析流程、Born-Haber循环步骤 |
| 对比图（并列对比） | ✅ Excalidraw | A1/A2/A3金属堆积对比、四大晶体类型对比 |
| 矩阵图（二维分类） | ✅ Excalidraw | 14种布拉维格子矩阵 |
| 决策树/判断流程图 | ✅ Excalidraw | 七大晶系判定决策树 |
| 时间线/演化图 | ✅ Excalidraw | 化学键理论发展时间线 |

### 3. 数据/表格类 — 优先从提炼文件直接嵌入

| 图片类型 | 推荐方式 |
|:--------|:--------|
| 数据表格（键参数/VSEPR构型表等） | ✅ 从提炼文件的Markdown表格直接嵌入讲义 |
| 坐标图（势能曲线/分布曲线） | 教材原图 > 网络下载 > Excalidraw |

## 常见问题

### Q: 教材OCR原稿中的图片是链接形式（如 `![](...jpg)`），怎么提取？

```text
1. 用 Read 工具读取OCR原稿，找到图片链接
2. 图片文件通常保存在OCR目录下的 `{filename}_images/` 子目录
3. 用 Read 工具打开图片文件确认内容
4. 将图片文件复制到讲义所在 `media/` 目录
5. 用 `![[media/filename.jpg]]` 嵌入讲义
```

### Q: 找不到合适的教材原图怎么办？

```text
1. 用 WebFetch 搜索高质量替代图（优先 Wikipedia Commons / 教育网站）
2. 下载后保存到 `media/` 目录
3. 在图片链接旁标注来源（如 `*来源：Wikipedia*`）
4. 只有流程图/思维导图才考虑 Excalidraw 自绘
```

### Q: 如果教材原图和网络下载都找不到合适的图呢？

对于**结构类图**：标注 ⚠️ 缺图，记录在讲义frontmatter中，后续再补。
对于**流程/概念类图**：使用Excalidraw自绘。
