# 模块连接推进-无机和结构化学

> 本页用于把“无机和结构化学”模块的连接工程落到可执行层。

## 一、当前模块判断

- `无机和结构化学` 当前约有 `173` 个 KP。
- 这是当前桥梁任务最重的模块，因为它同时承担：
  - 第一轮基础班结构主线
  - 第二轮元素化学深化
  - 第三轮 / 决赛的配位与结构深化

## 二、第一批优先专题

### 1. 晶体结构类型与稳定性

**当前产出真相源**
- [[04-课件/备课大纲/2026-06-17-晶体结构类型与稳定性-基础班]]
- [[04-课件/新授课/2026-06-17-晶体结构类型与稳定性-基础班]]

**优先提炼来源**
- `提炼-普化原理-第13章-晶体与晶体结构`
- `提炼-无机化学第五版-第7章-晶体结构`

**优先回补的 KP 方向**
- 晶体结构 / 晶胞 / 点阵与晶胞 / 晶体密度公式
- 密堆积 / 等径球堆积 / 填隙 / 典型结构（NaCl、CsCl、CaF2、NiAs、ZnS、金刚石等）
- 晶格能 / 离子极化 / 非化学计量化合物 / 尖晶石 / 钙钛矿

**当前连接目标**
- 把“第一轮基础结构主线”和“教材更深的晶体解释”同时接入
- 让后续从基础班往结构深化走时，不再断层

### 3. 配位 / 元素化学深化预备

## 三、推进顺序

1. 先补 `晶体结构类型与稳定性`
2. 再补 `原子结构基础` 中无机补深部分
3. 再补 `配位 / 元素化学` 的桥梁层
