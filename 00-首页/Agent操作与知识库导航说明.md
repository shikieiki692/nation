# Agent操作与知识库导航说明

> **用途**：给 Agent 读的操作速查——执行流程、定位法、红线。
>
> 用户侧说明（文件命名 / Dataview / Frontmatter / FAQ / 脚本工具）已迁至 [[首页#二、建设模式]] 和 [[首页#四、资料原则与产出规范]]，此处不再保留。

---

## 一、Agent 执行流程（Phase 0-4）

```text
Phase 0: 加载上下文 → 默认读 [[状态摘要]] → [[活跃任务]] → [[Agent战略方向]]
    ↓
Phase 1: 制定计划（目标、工具、风险预判）
    ↓
Phase 2: 执行操作（创建/修改文件、提炼资料）
    ↓
Phase 3: 验证抽查（3-5 个产出文件的质量检查）
    ↓
Phase 4: 归档收尾（工作日志 → 活跃任务 → 索引 → 状态摘要/战略方向）
```

### 收工回写入口速查

| 记录位置 | 用途 |
|:---|:---|
| `00-首页/工作日志/YYYY-MM-DD.md` | 过程记录与证据 |
| `00-首页/活跃任务/` 对应任务卡 | 任务状态更新 |
| `00-首页/状态摘要.md` | 会话快照（阶段/阻塞/下一步变化时） |
| `00-首页/Agent战略方向.md` | 长期优先级变化时 |

## 二、Agent 90 秒定位法（跨模块任务）

```text
状态摘要
  ↓
活跃任务（先看健康检查，再看任务优先级）
  ↓
Agent模块关系与收工同步清单
  ↓
对应模板 / 对应模块真相源
```

| 用户说… | 先读什么 | 做完重点回写 |
|---|---|---|
| "把这套网课整理成下周可直接上课的材料" | 状态摘要 → 模块关系清单 → 方法论 → 备课待办 → 模板 | 工作日志 + 活跃任务 + 备课待办 + 必要时状态摘要 |
| "把这个专题做成学生可复习的材料" | 状态摘要 → 模块关系清单 → 专题页/课件/学生侧模板 | 工作日志 + 活跃任务 + 学生侧目录/阻塞状态 |
| "继续推进第四轮" | 状态摘要 → 活跃任务 → 战略方向 → 备课待办 | 工作日志 + 活跃任务 + 备课待办 + 必要时首页/状态摘要 |

## 三、Agent 红线（不会做的事）

1. **不会整体重写已填充 KP**：文件已存在且 `status = "已填充"` 时只追加/修正，不整体重写。例外：明确触发的 Agent 7（KP 优化）或模板迁移任务。
2. **不会删除文件**：如需清理，会建议你先确认。
3. **不会修改 `.obsidian/` 配置**：除非明确授权。
