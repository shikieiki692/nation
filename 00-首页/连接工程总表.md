# 连接工程总表

> 本页用于统筹当前知识库几大系统之间的连接关系。
>
> 当前阶段的核心任务，不再是单纯“补内容”，而是把以下几条链稳定打通：
>
> `外部资料 → 资料提炼 → 知识点 / 题库 / 专题 → 备课产出`

## 一、当前系统划分

| 系统层 | 主要目录 | 主要职责 | 当前判断 |
|:---|:---|:---|:---|
| 入口与治理层 | `00-首页/` `11-模板/` `09-审计报告/` | 给出入口、规则、任务、日志、模板、审计 | 已较成熟 |
| 原始资料层 | `06-外部资料导入/` | 存放教材、网课、真题、讲义、PDF、图片等原始资料 | 资料较丰富 |
| 提炼层 | `07-资料提炼/` | 将外部资料转成可消费的教材主干、教学逻辑、例题、误区、图片资产 | 内容丰富，但下游消费密度不足 |
| 知识组织层 | `03-知识点/` | 稳定承载定义、原理、易错点、教学视角 | 内容充实，但与提炼层挂接密度仍偏低 |
| 训练组织层 | `04-题库/` `04-专题与题型/` | 组织题目、题型、专题、方法链 | 题库结构化较强，专题/题型覆盖仍不足 |
| 教学产出层 | `04-课件/` `06-学生侧材料/` | 形成备课大纲、讲义、PPT、学生材料 | 已进入实跑阶段 |

## 二、当前最关键的连接链

| 连接链 | 理想状态 | 当前状态 | 下一步重点 |
|:---|:---|:---|:---|
| 外部资料 → 资料提炼 | 每份高价值资料都能形成可消费提炼页 | 基本成立 | 清理 OCR 状态与入口误差 |
| 资料提炼 → 知识点 | 提炼成果能稳定沉到 KP 的 `source_notes / source_extracts / 教学视角` | 偏弱 | 优先补挂接密度 |
| 知识点 → 题库 | KP 能稳定挂到题目、题型、难度 | 较强 | 继续保持一致性 |
| 资料提炼 → 专题/题型 | 提炼能沉到专题页和题型页，而不是只停留在笔记层 | 偏弱 | 建专题级消费清单 |
| 专题/题型 → 备课 | 专题页、题型页真正成为备课抽料中枢 | 中等 | 继续做厚重点专题 |
| 题库 → 备课 | 例题、易错题、变式题能稳定进入讲义/PPT | 中等偏强 | 补“题库反哺专题/讲义”机制 |

## 三、当前系统性判断

### 3.1 已经比较成熟的部分

- 首页与入口系统已经比较完整。
- 题库字段化程度较高。
- 备课模板和加工页模板已经能约束高质量产出。
- 资料提炼内容储备已经不薄。

### 3.2 当前最薄弱的部分

- 提炼层成果没有充分回流到 KP。
- 专题页 / 题型页还没有充分承担“连接中枢”的角色。
- OCR 待整理与“已转写未清理”的状态仍有混杂。
