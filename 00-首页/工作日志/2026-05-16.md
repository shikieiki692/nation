# 2026-05-16 考纲 redlink 系统性修复 v2

## 📊 今日核心成果

### 1. DataviewJS 误报修复
- **问题**：`[[知识库待补充清单]]` 自动化缺口扫描显示 372 个缺失 KP，大量为误报
- **根因**：`link.path`（带 `.md`）与 `file.path` 格式差异 + `targetName` 未去 `.md` 导致排除规则失效 + Windows 路径分隔符问题
- **修复**：
  - 弃用自建 `allPaths` 集合，改用 `dv.page()` + `app.vault.getAbstractFileByPath()` + `app.metadataCache.getFirstLinkpathDest()` 多层存在性检查
  - `targetName` 统一去掉 `.md` 后缀，使排除规则/专题后缀规则正常生效
  - 新增模糊匹配层：检查缺失目标是否是已有 KP 文件名的子串（如「分散相」→「分散相与连续相」）

### 2. 假性缺失批量替换
- **扫描结果**：高置信度假性缺失 30 条 + 低置信度 14 条（除 `18电子规则` 概念误配外）
- **执行**：PowerShell 批量替换 45 处链接，涉及 27 个考纲条目文件
- **关键映射**（示例）：
  - `[[氨基酸]]` → `[[氨基酸与等电点]]`
  - `[[主族元素]]` → `[[主族元素化学]]`
  - `[[电子转移与电子对转移]]` → `[[电子转移]]`
  - `[[芳香性与苯环稳定性]]` → `[[芳香性]]`
  - `[[中间体稳定性排序]]` → `[[中间体]]`
- **格式修复**：替换后部分链接出现 `]]]` 多余右括号，已批量修复 27 个文件
