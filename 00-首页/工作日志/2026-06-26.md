### 关键教训（已沉淀到 [[经验沉淀]]）

1. **Excalidraw 程序化渲染不可靠** — Python Pillow 和 Puppeteer 都在 CJK/公式情境下失败。配图来源优先级固化：教材原图→网络→Excalidraw（最后手段）
2. **JSON 内部嵌套引号** — Excalidraw 文本若有 `"s"` 等 ASCII 引号，需转义为 `\"s\"`，否则 JSON 解析失败
3. **CRLF 行尾问题** — Windows `\r\n` 与 PowerShell heredoc `\n` 不匹配，Edit 替换需用 ReadAllText + regex -replace，或用 Git 配置 autocrlf
