## 📌 关键决策记录

1. **放弃 HTML `<img>` 标签**：由于 html2pptx 在 Windows 环境下对本地图片路径处理存在无法修复的 bug（`file://` → `/C:/` → `C:\C:\`），决定所有图片（含公式 PNG）均通过 PptxGenJS `slide.addImage()` 在 JS 层面插入，HTML 只负责文字和布局。

2. **三层架构分离**：
   - 渲染层（`latex-render.js`）→ 纯工具，不修改
   - 合成层（`ppt-engine.js`）→ 纯工具，不修改
   - 内容层（`模板-PPT生成.js` + `Agent提示词-PPT生成.md`）→ 每节课复制/参考

3. **内容深度规范优先于技术炫技**：用户明确反馈"内容太少、逻辑不清楚"，因此将工作重心从"技术验证"转向"内容架构设计"，建立六类幻灯片模板和深度标准。
