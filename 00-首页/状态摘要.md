## ⚠️ 踩坑速查 Top 5（DataviewJS 自动，来自 [[经验沉淀]] 🔥 标记）

> 自动抽取 [[经验沉淀]] 中标了 🔥 的 H3 条目（高频踩坑）的前 5 条。
> 加 🔥/去 🔥 即可调整本表，无需改本文件。完整说明见原条目。

```dataviewjs
const exp = await dv.io.load("00-首页/经验沉淀.md");
const lines = exp.split("\n");

const tops = [];
let currentCategory = "";
for (const line of lines) {
  // 一级分类标题：## 一、Frontmatter / YAML 类
  const cat = line.match(/^##\s+([一二三四五六七八九十]+)、(.+?)\s*$/);
  if (cat) {
    currentCategory = cat[2].trim();
    continue;
  }
  // 🔥 标记的 H3 条目：### 1-1 全角冒号导致 Edit 失败 🔥
  const m = line.match(/^###\s+(\S+)\s+(.+?)\s*🔥\s*$/);
  if (m) {
    tops.push({ code: m[1], title: m[2].trim(), category: currentCategory });
    if (tops.length >= 5) break;
  }
}

if (tops.length === 0) {
  dv.paragraph("⚠️ 未在 [[经验沉淀]] 中找到 🔥 标记的 H3 条目。");
} else {
  dv.list(tops.map(t => `**${t.code}** ${t.title}（${t.category}）`));
}
```

完整版（按 8 大类分组）见 [[经验沉淀]]。
