# 自动汇总 · 最近任务摘要

> 本页优先从 `00-首页/活跃任务/` 的任务卡抽取最近变动的任务事实。
> 具体改动细节、验证过程和证据仍以对应任务卡 `evidence` 与 [[工作日志]] 正文为准。

```dataviewjs
const pages = dv.pages('"00-首页/活跃任务"')
  .where(p => p.type === "活跃任务卡")
  .array()
  .map(p => {
    const eventDate = p.completed ?? p.updated ?? p.file.mtime;
    const evidence = Array.isArray(p.evidence)
      ? p.evidence.filter(Boolean).slice(0, 2).map(item => typeof item === "string" ? item : item?.path ?? String(item))
      : (p.evidence ? [typeof p.evidence === "string" ? p.evidence : p.evidence?.path ?? String(p.evidence)] : []);

    return {
      page: p,
      eventDate,
      status: p.status ?? "-",
      area: p.area ?? "-",
      evidence
    };
  })
  .sort((a, b) => String(b.eventDate).localeCompare(String(a.eventDate)));

const top3 = pages.slice(0, 3);

if (top3.length === 0) {
  dv.paragraph("⚠️ 未在 `00-首页/活跃任务/` 中找到可用于汇总的任务卡。");
} else {
  dv.table(
    ["日期", "任务", "状态", "领域", "证据"],
    top3.map(t => [
      t.eventDate ?? "-",
      t.page.file.link,
      t.status,
      t.area,
      t.evidence.length ? t.evidence.join(" · ") : "-"
    ])
  );
}
```
