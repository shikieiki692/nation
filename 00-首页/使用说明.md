# 使用说明

> 本文件是新人/旧人重新熟悉本库时的简短导览。深度导航见 [[首页]]；Agent 会话默认从 [[状态摘要]] 起步，再看 [[活跃任务]]，必要时再读 [[Agent战略方向]]。

## 1. 知识库定位

本库是中国化学奥林匹克备考与教学知识库，已从“建设期”进入 **“使用期+深化期”**。覆盖率、库存与近期任务等事实口径，统一以 [[状态摘要]] 与 `[[自动汇总-*]]` 为准。组织方式：

- 按 **考纲条目** 导航（基础要求 + 决赛要求）
- 拆分为 **知识点（KP）**，每个 KP 一篇笔记
- 配套 **题库**、**外部资料**、**资料提炼**、**教学逻辑提炼**
- 由 **AI Agent** 协作填充、维护、备课、产出专题页与讲义

## 4. 内容分层逻辑

```
原始资料 (06-外部资料)
   ↓ 提炼
中间层  (07-资料提炼 · 顺序层)
   ↓ 教学逻辑提炼
教学路径 (07-资料提炼/教学逻辑提炼 · 传授层)
   ↓ 并入正式库
正式库  (03-知识点)  ← 核心
   ↓ 链接
训练层  (04-题库 → 05-错题与反思)
   ↓ 教学产出
教学层  (专题页 → 备课大纲 → 学生讲义)
```

## 5. 基本规则

- **一篇 KP 只讲一个核心知识点**（避免大杂烩）
- **KP 必须用 [[模板-知识点]] v1.3.1**（含「教学视角」段 + 图片字段，备课/答疑必填）
- **AI 生成内容默认进待审核**，不得直接覆盖已填充 KP
- **外部资料必须先导入到 `06-外部资料导入/`，再提炼到 `07-资料提炼/`，再并入 `03-知识点/`**
- **若资料仍在 `OCR待整理/`，只算“已入库”，不算“已接入工作流”**；完成提炼后还需补齐“备课抽料资产标记”
- **修改/合并/弃用走软弃用 + 7 天缓冲**（详见 [[Agent提示词-知识库维护]] Agent 8）
- **短期任务统一进活跃任务卡**：`00-首页/活跃任务/` 每张任务卡独立维护，由 `[[活跃任务]]` 聚合展示，不再在日志正文中手写维护长任务清单

## 6. 建议习惯

- 每个 KP，至少配 1 个题型链接（`problem_types` frontmatter 字段）
- 每遇到错题，回链到对应 KP（在错题笔记顶部加 `相关知识点：[[...]]`）
- 每读一本书或一份讲义：先建资料页 → 再建提炼页 → 最后并入 KP
- AI 协作前先看 [[状态摘要]] 的「踩坑速查 Top 5」，避免重复踩
- 若任务跨两个及以上模块，再补看 [[Agent模块关系与收工同步清单]]，不要直接凭目录名猜回写位置
- 备课或产出教学材料前，先检查对应专题的「教学逻辑提炼」是否已就绪
- 跨专题查看当前课件进度时，优先看 [[04-课件/专题产出总览看板]]
- 查看尚未真正接入工作流的原始 OCR 资料时，优先看 [[06-外部资料导入/OCR待整理清单]]
- 若想判断“资料提炼、知识点、题库、备课”几套系统现在到底连得怎么样，优先看 [[连接工程总表]]；若想按学科模块推进，继续看 [[模块连接覆盖表]]
