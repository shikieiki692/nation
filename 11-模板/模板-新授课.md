# 模板：新授课

> 📋 **使用方式**：本模板用于生成**学生可见的课堂讲义**（而非教师备课文档）。落到 `04-课件/新授课/<日期>-<主题>.md`。
>
> 🎯 **职责定位**：**学生视角的课堂跟随材料**。包含完整知识内容、例题、易错点、作业和本节形成的核心结论，学生课后可独立阅读复习。
>
> ⚠️ **与备课大纲的区别**：
> - **备课大纲**（[[模板-备课大纲]]）是**教师备课文档**，包含认知台阶、课时分配、媒介分工、教学反思和轮次深度边界，**不包含知识内容本身**（通过 wikilink 引用专题页/KP）。
> - **本模板（新授课讲义）**是**学生课堂材料**，包含**完整知识内容**和例题提示，是备课大纲中「课堂呈现」的落地产物。
> - **使用顺序**：教师先用备课大纲完成教学设计 → 再根据备课大纲的「媒介分工表」和「深度边界」决定哪些内容写入本讲义、哪些内容留在 PPT/板书或后续轮次。
>
> 🎯 **课型特征**：学生**首次接触**目标知识点。核心任务是**建立概念、理解机理、形成初步应用能力**。
>
> 📚 **内容密度指导（v1.7）**：
> - 知识内容应**足够充实**，学生课后可**独立阅读复习**，不必再翻教材补全推导。
> - 鼓励引用已提炼的教材/网课资料（`source_notes`、`source_extracts`）中的完整推导、数据表格、分类体系。
> - 核心公式建议保留**推导链条**（至少关键步骤），不只是抛结论。
> - 每个子主题下设置 **「完整推导折叠区」**（Obsidian callout），课堂时间紧可跳过，课后学生可自行展开。
> - 例题总量建议 **3-4 道基础 + 1-2 道拓展**（v1.7 上调），课后作业 **4-6 道必做 + 2-3 道选做**。
>
> ⚠️ **结论呈现策略（v1.6）**：基础班新授课默认不把最终核心结论放在最开头，而是在讲义末尾“本节形成的核心结论”中总结。提高班、复习课、习题课可保留结论先行。
>
> ⚡ **化学竞赛适配**：课时按轮次灵活配置——第一轮通常为 **90 min**（2 课时 × 45 min），提高轮次可扩展至 **120 min**（2 h）。知识密度高，以讲授 + 真题实战为主。
>
> 📚 **教材关联**：每个板块标注对应教材章节，便于后续导入资料后直接填充。拓展层内容用 `> 🔷 拓展` 标记，教师可根据学生接受度灵活取舍。

> ⚠️ **v1.8 内容边界约定**（依据 [[模板-课件产出规范]]）：
> - **禁止**在学生讲义中出现「本节深度边界」「本节暂不展开」「暂不要求掌握」等教师侧用语。
> - **必须**包含：学习目标（学生视角）、教材关联（每板块末尾）、拓展阅读（🔷 标记）。
> - 自查清单中的边界描述改为正面表述（如「我了解拓展内容：Graham 扩散定律、van der Waals 方程」）。
> - 公式优先使用 LaTeX 渲染为图片嵌入（见 [[模板-课件产出规范#二、公式与化学式渲染]]）。

## 二、知识内容

> 📷 **图片策略**：本专题类型为 `{topic_type}`，默认 `images_priority: {priority}`。
> - **概念类**（concept）：以公式和对比表格为主，图片仅用于辅助理解（如指示剂变色机理）
> - **结构类**（structure）：图片是核心教学媒介，必须在讲义中嵌入关键结构图

> 例题分两层：**⭐⭐ 基础**（必做，覆盖核心概念）和 **⭐⭐⭐ 拓展**（选做，覆盖综合应用）。

> - 每道例题包含「题目 + 分析提示 + 完整解答路径」，解答可放折叠区或板书呈现。

## 新授课 vs 其他课型速查

| 维度 | 新授课 | 复习课 | 习题课 |
|:---|:---|:---|:---|
| **核心目标** | 建立概念、理解机理 | 知识网络、易错点攻克 | 解题方法、速度训练 |
| **课时** | 90-120 min | 90-120 min | 90-120 min |
| **概念引入** | 必须有问题/现象/认知冲突 | 快速回顾（5 min） | 无 |
| **核心结论位置** | 基础班默认结论后置 | 可结论先行 | 可方法先行 |
| **机理讲解** | 详细推导 + 拓展层 | 纲要式回顾 | 题中遇到再讲 |
| **例题难度** | ⭐⭐（基础必做）+ ⭐⭐⭐（拓展选做） | ⭐⭐~⭐⭐⭐ | ⭐⭐~⭐⭐⭐⭐ |
| **例题数量** | 基础 3-4 道 + 拓展 1-2 道 | 5-8 道 | 10-15 道 |
| **即时演练** | ✅ 有（3-5 min 快速检测） | 可选 | 无 |
| **师生互动** | 精简（关键提问 + 真题演练） | 中等（讨论 + 对比） | 较少（做题 + 快讲） |
| **知识网络图** | 局部（本节） | 全局（跨章节） | 方法体系图 |
| **板书** | 极简（核心框架） | 中等（网络图） | 较少（方法要点） |
| **PPT** | 主要载体 | 辅助 | 题目呈现 |
| **教材关联** | 每板块标注 | 跨教材整合 | 无 |
| **拓展层** | 用 🔷 标记，可选 | 较少 | 无 |
| **自查清单** | ✅ 有 | 可选 | 无 |
