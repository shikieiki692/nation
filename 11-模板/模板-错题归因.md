# 模板：错题归因

> 📋 **使用方式**：学生在练习卷、作业、考试订正后，对单题或同类错误做一次结构化复盘，不只写“答案是什么”，更要写“为什么会错、下次怎么避免”。
>
> 🎯 **课型特征**：**元认知层工具**。核心任务是把“做错一道题”转化为“定位一个薄弱环节 + 制定一个可执行修正动作”。
>
> 📐 **纸质规格**：A4 单页或活页本；推荐每次只分析 1-3 个高价值错误，不追求把所有错题都写满。
>
> 💻 **电子辅助**：可在 Obsidian 中保留对应题目、KP、策略卡链接；后续如需汇总，可按错误类型做 Dataview 统计。

## 二、推荐归因分类

| 归因类别 | 典型表现 | 常见根因 | 优先修正动作 |
|:---|:---|:---|:---|
| **概念不清** | 术语混淆、判据乱用 | 基础定义没稳固 | 回看对应 KP §一/§四，补概念卡 |
| **条件漏看** | 忽略“过量/少量/限定条件” | 审题习惯差 | 给题面关键信号画线，重做同类题 |
| **方法选错** | 会做但第一步走偏 | 没识别题型信号 | 回看策略卡，补“适用信号” |
| **计算失误** | 公式对但结果错 | 过程分规范弱 | 用 [[模板-过程分评分表]] 逐项自检 |
| **表达失分** | 思路对但得分低 | 关键词/层次不清 | 重写答案，按“因果链 + 分点”表达 |
| **迁移失败** | 学过但换壳不会 | 知识孤立、缺变式 | 再做 1 道近邻变式题 |

## 三、填写流程

```text
Step 1：先订正答案
   ↓ 先搞清“正确做法是什么”，再做归因
Step 2：只选 1 个主因
   ↓ 不要把“粗心、不会、紧张”全部都写上
Step 3：写出错误发生点
   ↓ 具体到“哪一步、哪个信号、哪句判断”
Step 4：链接到 KB 资产
   ↓ 对应 KP / 策略卡 / 练习卷 / 闪卡
Step 5：给出 1 条立即动作
   ↓ 必须是今天能执行的动作，而不是空泛反省
```

## 四、标准填写示例

### 4.1 示例一：方法选错

| 项目 | 内容 |
|:---|:---|
| 题目来源 | 专题练习卷-化学平衡 |
| 错误现象 | 把“比较平衡移动方向”做成了机械套公式 |
| 主归因 | **方法选错** |
| 为什么会错 | 看到有数值就直接代公式，没有先判断题目是在问“趋势”还是“定量结果” |
| 正确信号 | 题目只要求比较增大/减小，应先用“反应商 vs 平衡常数”或极限判断 |
| 对应资产 | [[化学平衡]]、[[反应商]]、[[模板-方法策略卡]] |
| 立即动作 | 再做 1 道同类“只判断趋势”的题，并在题面圈出信号词 |

### 4.2 示例二：过程分失分

| 项目 | 内容 |
|:---|:---|
| 题目来源 | 专题练习卷-氧化还原滴定 |
| 错误现象 | 最终答案数值正确，但因为单位和有效数字问题被扣分 |
| 主归因 | **表达失分 / 计算失误** |
| 为什么会错 | 中间步骤直接代数值，没有保留符号，也没有检查单位换算 |
| 正确做法 | 按 [[模板-过程分评分表]] 的“公式先行 → 量纲同步 → 最终答案”顺序写 |
| 对应资产 | [[模板-过程分评分表]] |
| 立即动作 | 把原题完整重写一遍，只训练过程分规范，不换题 |

## 五、与知识库的联动规则

| 归因结果 | 后续动作 |
|:---|:---|
| 同一 KP 连续出现 2 次以上 | → 把该 KP 加入本周复习清单 |
| 同一方法反复误用 | → 补 1 张 [[模板-方法策略卡]] 或回看已有策略卡 |
| 同一类计算规范重复扣分 | → 用 [[模板-过程分评分表]] 做 1 次专项训练 |
| 同一专题错误率偏高 | → 回到专题练习卷做“变式重练” |
| 错误源于记忆断裂 | → 补 3-5 张 [[模板-学生闪卡]] |

## 六、教师汇总视角（可选）

如果教师要批量看学生归因，优先抓这 3 个维度：

| 维度 | 看什么 | 后续教学动作 |
|:---|:---|:---|
| **高频错因** | 是概念问题还是方法问题居多 | 决定下节课是补知识还是补策略 |
| **集中薄弱 KP** | 哪几个 KP 被反复提及 | 列入复习课或讲义加练 |
| **过程分损失点** | 是公式、单位还是表达最常出错 | 做一次规范书写示范 |

## 七、质量检查清单

- [ ] 是否只选择了 1 个主归因，而不是泛泛而谈
- [ ] 是否写清“错在第几步/哪个判断”
- [ ] 是否写出了正确思路，而不只是抄答案
- [ ] 是否链接到至少 1 个具体 KB 资产（KP / 策略卡 / 评分表）
- [ ] 是否给出了 1 条可立即执行的修正动作
- [ ] 是否避免把“粗心”当作最终归因
