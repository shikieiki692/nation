# 学生讲义排版规范

> **用途**：统一 66 份学生讲义的排版标准，确保 .md 源文件 → docx 输出的一致性。
> **适用范围**：`04-课件/学生讲义/` 下所有讲义。
> **排版管线**：.md 源文件 → [本规范] → 预处理脚本 → Pandoc → 后处理（docx_utils.py）。

---

## 三、标题规范

### 3.1 讲义标题（`#`）

**统一格式**：`# 第N讲：标题` 或 `# 主题名：子标题`

例外规则：
- 有讲次编号的 → `# 第N讲：标题`（如 `# 第十讲：滴定分析`）
- 无讲次编号的自编讲义 → `# 主题名`（如 `# 晶体结构基础`）
- 跨讲次整合的 → `# 主题名：子标题`（如 `# 热能篇：化学热力学基础`）

### 3.2 节标题 (`##`)

- 一级阿拉伯数字编号：`## 一、标题` / `## 二、标题`
- 学习目标：固定 `## 学习目标`
- 作业/练习：固定 `## 课后练习` 或 `## 思考题`

### 3.3 子节标题 (`###`)

- 编号格式：`### N.M 标题`（N = 节号，M = 子节号）
- 示例：`### 2.1 SN2（双分子亲核取代）——一步协同`

## 四、学习目标格式

**统一格式**（二选一，整份讲义保持一致）：

```
- ✅ 能{动词}{对象}，{限定条件}
```

**推荐**：不用 `目标1/目标2/目标3` 编号，直接用动作开头。

**示例**：
```markdown
- ✅ 能说出晶体的微观定义（点阵+结构基元），根据轴长/轴角判断7大晶系
- ✅ 能计算 ccp/hcp/bcc 的配位数、空间利用率和空隙类型/数目
```

## 五、图片规范

### 5.1 图片嵌入

```markdown
![[media/filename.jpg|替代文本]]
*图 说明文字——详细描述（来源名称）*
```

- 图片文件统一放在 `04-课件/学生讲义/media/`
- 替代文本用 `|` 写在 wikilink 内
- 图注用斜体 `*` 包裹，格式：`*图 内容描述——补充说明（来源）*`
- 图注字号 9.5pt，居中

### 5.2 图片命名

- 教材来源图：保持原文件名（如 `11-13-orbital-shapes-s-p-d.jpg`）
- 网络来源图：`descriptive-name-type.jpg`（如 `sn2-reaction.svg`）
- 自绘示意图：`topic-description.jpg`

## 六、表格规范

### 6.1 表格格式

```markdown
| 列1 | 列2 | 列3 |
|:---|:---:|---:|
| 左对齐 | 居中 | 右对齐 |
```

- **表头**：加粗 `**` 包裹，黑体 SimHei
- **表体**：普通，宋体 SimSun
- **字号**：10.5pt
- **行距**：1.15

### 6.2 表格编号

长表格（≥5行）建议加表格编号：
```markdown
*表1 量子数取值范围与物理意义*
```

## 七、公式规范

### 7.1 行内公式

```markdown
能量 $E = -\dfrac{13.6}{n^2}\ \mathrm{eV}$
```

- 使用 `$...$` 包裹
- 单位和纯数字用 `\mathrm{}` 包裹
- 化学式用 `\mathrm{}` 包裹（避免被当作变量斜体）

### 7.2 展示公式

```markdown
$$
\Delta G^\circ = -RT \ln K^\circ
$$
```

- 使用 `$$...$$` 包裹（不要用 `\[...\]`）
- 需要编号时在公式后加 `\tag{N}`

### 7.3 化学式

- 简单化学式用 `\mathrm{H_2O}` 或直接写 H₂O（有 Unicode 下标）
- 复杂反应式用 `$$...$$`  + `\ce{}`（预处理脚本会转为 `\text{}` 兼容 OMML）

## 九、Metadata 块引用规范

标题下方的 `>` 块引用，按以下顺序排列字段（可选字段标 `*`）：

```markdown
> **适用**：{轮次与班型} | **前置要求**：{前置知识点}
> **对应备课大纲**：[[链接]]
> * **建议使用方式**：{课前/课堂/课后分工} *
> * **核心概念**：{本讲核心概念列表} *
> **深度边界**：{覆盖范围与不展开内容}
```

**字段顺序**：适用 → 对应备课大纲 → 前置要求 → [建议使用方式] → [核心概念] → 深度边界
