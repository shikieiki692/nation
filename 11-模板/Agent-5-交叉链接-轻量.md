# Agent 5 · 交叉链接（轻量版）

## 一句话定位
在知识点文件和题目文件之间建立双向链接。

## 何时使用
- 一批题目拆分完成后，需要链接到对应知识点
- 知识点文件需要补充典型题型链接
- 发现链接缺失或断链

## 核心输入（必填）
| 参数 | 说明 |
|------|------|
| 科目/模块范围 | 要处理的科目或章节目录 |
| 工作方式 | 补缺失链接（默认）或 全面重建 |

## 执行流程（3步）
1. **题目→知识点**：读取每道题的 knowledge_points，在"知识点映射"section 添加 wiki-link
2. **知识点→题目**：搜索所有题目 knowledge_points，在知识点"典型题型"/"例题"中添加反向链接
3. **汇总报告**：列出成功链接数、无法链接的、缺失的知识点

## 完成定义（DoD）
- [ ] 抽查3-5个知识点文件，典型题型/例题中有有效题目 wiki-link
- [ ] 抽查3-5个题目文件，知识点映射中的 wiki-link 有效
- [ ] 无知识点名称与文件名不匹配导致的死链
- [ ] 不破坏已有链接（补缺失模式）

## 边界（不做的事）
- 不修改知识点或题目的正文内容
- 不新建知识点文件（只链接已有文件）
- 不处理不存在的知识点（记录到缺口清单）

## 快速命令
```
用"交叉链接 Agent"处理 {科目}
```
