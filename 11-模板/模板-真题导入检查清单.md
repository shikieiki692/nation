## 五、Phase 4：单题文件 frontmatter 检查

> 每道题拆分为独立文件后，逐题检查以下 frontmatter 字段。
> 当前 `[[模板-题目]]` 以字段完整性为主，**暂未引入 `version` 字段**；因此本清单此处不强制检查版本号。若后续题目模板新增 `version`，则统一按“`version` 为主、`template_version` 兼容且需一致”的口径执行。

### 5.1 必填字段（少一个 = 不通过）

- [ ] `type: 题目`
- [ ] `exam_stage: 初赛`（或 决赛 / 选拔赛）
- [ ] `year: ____`（考试年份，如 2025）
- [ ] `exam_date: ____-__-__`（精确日期，如 2025-09-07）
- [ ] `source: 第__届中国化学奥林匹克（__赛）第__题`（精确到届+赛段+题号）
- [ ] `subject: ____`（化学原理 / 无机和结构化学 / 有机化学 / 分析化学）
- [ ] `module: ____`（与 subject 一致或更细分）
- [ ] `submodule: ____`（如 电化学 / 晶体结构 / 自由基反应）
- [ ] `question_type: ____`（选择题 / 计算题 / 推断题 / 机理书写题 / 实验设计题 / 简答题）
- [ ] `difficulty: _`（1-5 整数）
- [ ] `teaching_level: ____`（入门 / 巩固 / 拓展 / 竞赛拔高）
- [ ] `syllabus_codes: ["__"]`（考纲条目编号，如 ["8", "12", "17"]）
- [ ] `knowledge_points: [__]`（直接考查的 KP，必须是现有 03-知识点/ 中的文件名）
- [ ] `tags: [化竞, 真题, __届]`
- [ ] `updated: ____-__-__`

### 5.2 推荐字段（强烈建议）

- [ ] `aliases: [__届初赛-{大题号}.{小题号}, {核心概念简称}]`
  - 示例：`[39届初赛-2.1, 普鲁士蓝-电化学]`
  - 作用：方便快速 wikilink 和搜索

- [ ] `source_file: 06-外部资料导入/真题/___.pdf`
  - 作用：精确回溯到原始 PDF 页

### 5.3 frontmatter 质量检查

- [ ] `knowledge_points` 中的每个名称都能在 `03-知识点/` 中找到对应文件
  - 如不匹配：要么改名，要么记录到提炼页的「待创建 KP」段
- [ ] `syllabus_codes` 与 `02-考纲条目/` 中的编号对应
- [ ] `difficulty` 与提炼页考纲映射总表中的难度一致
- [ ] frontmatter 检查只以当前 `[[模板-题目]]` 已声明字段为准；不要自行补造模板中不存在的“标准字段”

## 六、Phase 5：单题文件内容检查

### 6.1 题目段

- [ ] **来源声明完整**

### 6.2 参考答案段

- [ ] **有得分点列表**
  - 示例：`**得分点**：1. **写出电极反应式**（2分）：...`

### 6.3 知识点映射段（表格化）

- [ ] **使用表格格式**

```markdown
| 关联 KP | 考查角度 | 直接/间接 |
|---|---|:---:|

```

- [ ] **至少 1 个「直接」KP**
- [ ] **wikilink 有效**（Obsidian 中不显示为红链）

### 6.4 解题思路段

- [ ] **有「读题定位」第一步**

## 九、常见错误速查

| 现象 | 原因 | 修正 |
|---|---|---|
| Dataview 查不到新题 | `type` 字段缺失或拼错 | 补 `type: 题目` |
| Dataview 查到但 KP 页查不到 | KP 页查询用了旧路径 `04-题目` | 改为 `FROM "04-题库"` |
| 红链（broken link） | `knowledge_points` 中的 KP 名与文件名不一致 | 统一命名或创建别名 |
| 图片不显示 | 路径用了 `![](path)` 而非 `![[path]]` | 改为 wiki-link 格式 |
| 难度排序错乱 | `difficulty` 写了字符串 `"3"` 而非数字 `3` | 去掉引号 |
| teaching_level 筛选失效 | 拼写不一致（如「巩固」vs「鞏固」） | 统一用简体字，全库统一词表 |

```markdown
# 题-016：普鲁士蓝电极反应、颜色变化与比容量

## 题目

## 知识点映射

| 关联 KP | 考查角度 | 直接/间接 |
|---|---|:---:|
| [[Nernst方程]] | 电池比容量计算 $C = nF/M$ | 直接 |
| [[配合物]] | 普鲁士蓝混合价态结构、CN⁻ 桥联配位 | 直接 |
| [[晶体场理论]] | CT 跃迁解释颜色（vs d-d 跃迁） | 间接 |
| [[电极电势]] | 充放电方向判断 | 间接 |

## 解题思路

1. **读题定位**：看到「普鲁士蓝」「电极」「比容量」→ 识别为 **氧化还原电化学 + 配合物结构** 综合题
2. **判断价态**：PBA-1 为 $\mathrm{KFe^{III}Fe^{II}(CN)_6}$，混合价态
3. **🔑 关键转换**：比容量公式 $C = \frac{nF}{3.6M}$，其中 $n=2$
4. **颜色机制**：CT 跃迁 vs d-d 跃迁 → CT 跃迁对称性允许、吸收截面大
5. **验证**：$C \approx 174.65\ \mathrm{mAh \cdot g^{-1}}$，数量级合理

## 易错分析

| 错误 | 原因 | 纠正 | 课堂提问 |
|---|---|---|---|
| 比容量只算 1 个 $\mathrm{K}^{+}$ | 未理解可逆储钾是 2 电子过程 | 看化学式，2 个 Fe 各变 1 价 | 「PBA-1 中有几个 Fe？各几价？」 |
| 颜色解释为 d-d 跃迁 | 混淆 d-d 与 CT 跃迁强度 | CT 跃迁对称性允许，吸收更强 | 「为什么普鲁士蓝颜色这么深？」 |
```
