# Agent 3 · 习题提炼（轻量版）

## 一句话定位
从习题解析 PDF 提取考纲章节的习题，生成提炼笔记。

## 何时使用
- 新导入一本习题解析书，需要提取题目
- 需要补充某章节的习题内容

## 核心输入（必填）
| 参数 | 说明 |
|------|------|
| 源文件路径 | 06-外部资料导入/ 下的习题解析 Markdown |
| 目标章节 | 要提取的考纲章节（可多个） |

## 执行流程（3步）
1. **筛选章节**：对照考纲，只提取考纲要求的章节
2. **提取内容**：每章一个文件，包含内容提要 + 完整题目 + 参考答案 + 解析
3. **格式处理**：题号保持原书编号，图片改 wiki-link，公式保留原样

## 完成定义（DoD）
- [ ] 每章一个文件，不跨章合并
- [ ] frontmatter 含主版本字段 `version`；若模板仍保留兼容字段，则 `template_version` 与 `version` 一致；同时包含 `source_type`、`subject`、`chapter`、`syllabus_codes`
- [ ] 原书题号保持（如 2.1、5.16），未重新编号
- [ ] 多问小题（1)(2)(3)）保持在一起
- [ ] 随机抽1题对照源文件，答案和解析未篡改

## 边界（不做的事）
- 不跳过"内容提要"
- 不合并不同章节的习题
- 不修改题号
- 不提取非考纲章节

## 快速命令
```
用"习题提炼 Agent"从 {习题解析文件} 提取 {章节} 的习题
```
