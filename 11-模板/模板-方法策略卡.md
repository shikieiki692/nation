# 模板：方法策略卡

> 📋 **使用方式**：每个方法微专题（或专题页中的解题套路）产出 1 张策略卡。学生随身携带，解题时像查字典一样快速定位方法。
>
> 🎯 **课型特征**：**元认知层工具**。不是记知识，而是记"遇到什么情况用什么策略"。
>
> 📐 **纸质规格**：1/4 A4（约 105mm × 148mm，A6 大小），单面或双面印刷。建议打孔穿环，做成"方法手册"。
>
> 💻 **电子辅助**：Obsidian 中可用 Canvas 或 Mermaid 做交互式决策树；纸质卡可印二维码链接到方法微专题全文。

## 一、策略卡版式

### 正面（识别面）

```text
┌─────────────────────────────┐
│  🔧 方法策略卡               │
│                              │
│  【方法名】                  │
│  极限假设法                  │
│                              │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                              │
│  【一句话定位】              │
│  把变量推到极端，看趋势      │
│  是否自洽。                  │
│                              │
│  【适用信号】                │
│  ⭐ 题目问"趋势""比较""增大  │
│     还是减小"               │
│  ⭐ 选项是范围而非具体数值   │
│  ⭐ 变量可取 0 / ∞ / 极大值  │
│                              │
│  【不适用陷阱】              │
│  ❌ 变量不能取极端（如分母→0│
│     导致无意义）             │
│  ❌ 需要精确数值答案         │
│                              │
│  来源：[[方法微专题-极限假设 │
│  法]]  适用：化学原理/元素   │
│  化学                        │
└─────────────────────────────┘
```

### 背面（操作面）

```text
┌─────────────────────────────┐
│  【操作步骤】                │
│                              │
│  Step 1：识别可推极端的变量  │
│  Step 2：代入极端值，算结果  │
│  Step 3：对比选项/判断趋势   │
│  Step 4：验证中间值是否一致  │

│                              │
│  【口诀】                    │
│  "先推极端再看趋势，         │
│   中间验证不迷糊"            │
│                              │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                              │
│  【典例速查】                │
│  题：比较 Ca(OH)₂ 在纯水和   │
│      0.1M NaOH 中的溶解度   │
│  用：令 [OH⁻]→∞，s→0，直接 │
│      判断"溶解度骤降"       │
│  来源：[[题-XXX]]            │
│                              │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━  │

│                              │
│  【关联方法】                │
│  ← 量纲分析法  → 守恒法     │
```

## 二、策略卡分类

| 类别 | 特征 | 示例 | 卡片数量/轮次 |
|:---|:---|:---|:---:|
| **通用计算策略** | 跨专题可用的数学/物理技巧 | 极限假设法、量纲分析法、守恒法 | 3–4 张 |
| **元素推断策略** | 颜色/沉淀/气体/价态的组合逻辑 | 特征信号法、价态变化法、配合物生成法 | 3–4 张 |
| **有机解题策略** | 有机特有的结构分析和合成思维 | 逆合成分析法、官能团保护法、选择性反应法 | 4–5 张 |
| **结构分析策略** | 晶体/分子/配合物的判断流程 | VSEPR 速判法、晶胞计数法、CFSE 计算法 | 3–4 张 |
| **应试策略** | 时间分配和答题顺序 | 三轮答题法、先易后难、跳步得分法 | 2–3 张 |

## 三、从方法微专题到策略卡

```text
方法微专题（15–20 min 课堂）
   ├── 核心思想（1 min）      → 策略卡【一句话定位】
   ├── 适用场景（2 min）      → 策略卡【适用信号】+【不适用陷阱】
   ├── 操作步骤（3 min）      → 策略卡【操作步骤】+【检查清单】
   ├── 典例演示（5–6 min）    → 策略卡【典例速查】
   ├── 口诀总结（1 min）      → 策略卡【口诀】
   └── 迁移应用（1 min）      → 策略卡【关联方法】
```

> 💡 **原则**：策略卡不是方法微专题的缩印，而是"解题时的速查工具"。课堂上讲 15 分钟，卡片上只保留"能帮我快速决策"的信息。

## 四、使用方法（给学生）

### 4.1 日常解题流程

```text
拿到题目 → 卡住了？
   ├── 不知道用什么方法 → 翻策略卡「适用信号」栏
   │                         找到匹配信号 → 看背面「操作步骤」
   │
   └── 知道用什么方法但不记得步骤 → 直接翻该卡片背面「操作步骤」
                                  → 按 Step 执行
                                  → 做完对照「检查清单」验证
```

### 4.2 考前速查流程

考前 30 min，快速过一遍全部策略卡：
- 只看正面「适用信号」→ 自我检测："看到这个信号，我能想起方法名吗？"
- 想不起来的 → 翻背面重点看「操作步骤」和「口诀」
- 已熟练的 → 放一边，不重复看

### 4.3 与错题本联动

错题归因时，如果错误原因是"没想到用某方法"：
- 在错题旁标注："应查策略卡：[[极限假设法]]"
- 在策略卡正面「适用信号」栏补充："本题信号：____"（个性化批注）

## 六、质量检查清单

- [ ] 正面能在 10 秒内判断"这个方法是否适合当前题目"
- [ ] 适用信号 ≥ 3 条，且每条都是学生能从题面直接观察到的
- [ ] 不适用陷阱 ≥ 2 条，防止误用
- [ ] 操作步骤 ≤ 5 步，每步 ≤ 15 字
- [ ] 口诀押韵或有对称结构，易记忆
- [ ] 典例速查 ≤ 3 行，只保留"方法怎么用"的关键信息
- [ ] 检查清单 ≤ 4 项，可快速勾选
- [ ] 关联方法已链接（形成策略网络）
- [ ] 卡片尺寸适合随身携带（A6 或名片大小）
