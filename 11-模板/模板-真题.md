# 模板：真题

> 📋 **使用方式**：为每道竞赛真题创建一个文件，落到 `05-真题库/真题-<主题>-<编号>.md`。
> 真题文件通过 frontmatter 中的 `knowledge_points` 字段与专题页 §八 的 Dataview 查询关联。

## 模板正文

```markdown
---
title: "<题目简短描述>"
aliases: []
type: 真题
year: <YYYY>
source: "<竞赛名称与级别>"
type_tag: "<选择/填空/简答/计算/推断>"
difficulty: <1-5>
knowledge_points:
  - "<KP标题1>"
  - "<KP标题2>"
tags:
  - 化竞
  - 真题
related_notes:
  - "[[专题-XXX]]"
updated: <YYYY-MM-DD>
---

# <题目标题>

## 题目

<题目正文，使用 LaTeX 公式。如含图，用 ![[图名]] 嵌入>

## 解析

### 分析
<解题思路分析>

### 解答
<分步解答>

## 答案

<最终答案>

## 知识点链接

- [[KP1]]
- [[KP2]]
- [[题型-XXX]]
```

---

## 字段说明

| 字段 | 说明 | 示例 |
|:---|:---|:---|
| `title` | 题目简短描述 | `"酸碱质子理论-缓冲溶液pH计算"` |
| `year` | 竞赛年份 | `2024` |
| `source` | 竞赛名称与级别（省级初赛/全国初赛/决赛/IChO） | `"中国化学奥林匹克(初赛)"` |
| `type_tag` | 题型 | `选择` / `填空` / `简答` / `计算` / `推断` |
| `difficulty` | 难度 1-5 | `1`=基础 / `3`=初赛中等 / `5`=决赛 |
| `knowledge_points` | 关联的 KP 标题（数组），**必须与对应 KP 的 `title` 字段完全一致** | `["酸碱理论", "缓冲溶液"]` |
| `related_notes` | 关联的专题页/题型页 | `["[[专题-酸碱理论]]"]` |

> ⚠️ `knowledge_points` 字段的值必须精确匹配对应 KP frontmatter 中的 `title` 字段。Dataview 的 `contains()` 查询依赖这个匹配。
