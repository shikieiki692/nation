# 原子结构讲义 · 最终检查清单（每次都查）

## 1. 核反应表格
- [ ] 表格不在 `>` blockquote 内
- [ ] 对齐线：`|:---|:---|:---|`（三列对齐）
- [ ] 化学式用 `\ce{^{A}_{Z}X}` mhchem 格式
- [ ] $\alpha$ $\beta$ $\gamma$ 用数学模式（`$\alpha$`）
- [ ] 反中微子写为 `\bar{\nu}_e`
- [ ] gamma写为 `\gamma`

## 2. 图注（每次修完必须检查的）
- [ ] 图注中**不能有 `$...$`** 格式的LaTeX内联公式
- [ ] 图注中**不能有 `_`** → caption在`E_4s<E_3d`会断裂成`E\_{4s}<E_{3d}` → `fffff`
- [ ] 图注文字纯中文，最多出现`Z`、`D(r)`这种简短符号
- [ ] 图6（p轨道三维图）图注写"图 p轨道的三维形状与空间取向"
- [ ] 图7（4s/3d径向分布）图注写"图 4s与3d径向分布对比——4s在主峰外有靠近核的小峰"
- [ ] 图10+11（趋势图）图注写"图 XXX随Z的变化趋势——..."
- [ ] 每张图的文件实际内容与图注描述一致（用眼睛看！）

## 3. Caption 转义管线（convert_handout_to_pdf.py）
- [ ] `%` → `\%`
- [ ] `$` 外部的 `_` → `\_`（内部的`_`保留，因为LaTeX数学模式下`_`是合法下标）
