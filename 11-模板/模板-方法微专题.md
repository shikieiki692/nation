# 模板：方法微专题

> 📋 **使用方式**：备课 Agent 或教师手动按本模板生成方法微专题课件，落到 `04-课件/方法微专题/<轮次>-<方法名>.md`。
>
> 🎯 **课型特征**：**跨模块思维工具训练**，每次 15-20 min。核心任务是建立学生的"解题策略库"，而非传授新知识点。
>
> ⚡ **化学竞赛适配**：方法微专题通常插入在两节课之间，或作为某节课的最后15分钟。要求**讲练结合**：教师演示1道典例 → 学生独立演练1道变式 → 口头总结口诀。
>
> 📚 **与主课件的关系**：方法微专题不替代主课件，而是补充。同一方法可在多轮次中复现深化（如"极限假设法"第一轮讲趋势判断，第四轮讲快解技巧）。
