### 2.2 铺垫与结论出现时机
- **切入前置**：作者从什么概念/问题/现象切入本章？（如“由热力学 ΔG 引出平衡常数” vs “先实验测定再讲推导”）
- **铺垫段落**：哪些“看似偏题”的内容其实是后文铺垫？
- **核心结论何时出现**：开头 / 中段 / 例题后 / 末尾归纳？为什么？
- **一笔带过/暂时跳过**：哪些内容被作者刻意简化或跳过？（往往是作者认为的“前置已掌握”或“超纲”）
- **教学风格**（可选标签）：公式优先 / 实验优先 / 历史脉络 / 类比驱动 / 机理优先 / 题目驱动

## 六A、备课抽料资产标记（2026-06-18 新增）

> 用于告诉后续备课 Agent：这份提炼笔记到底能抽什么，不要只知道“提炼过”。

| 资产类型 | 本章是否有高价值内容 | 可直接抽料的代表项 | 未来优先服务的产物 |
|:---|:---:|:---|:---|
| 教材主干 | 是 / 否 | 定义、定律、结构化板块、典型物性解释 | 备课大纲 / 学生讲义 |
| 教学顺序 | 是 / 否 | 切入、铺垫、结论出现时机、安排顺序 | 备课大纲 / 教师稿 |
| 桥梁解释 | 是 / 否 | 机制、推导、连接前后主题的解释点 | 讲义正文 / 板书 / PPT |
| 例题链 | 是 / 否 | 最小辨析题、典型题、易错题 | 讲义 / PPT / 作业 |
| 易错点 | 是 / 否 | 高频误判、术语混淆、边界误用 | 讲义 / 教师提醒 |

## 十、🪤 踩坑回流

> ⭐ v1.1 强制：完成提炼后必须回答以下 4 个问题（即使全部“无”也要明示）。如有新坑则同步追加到 [[经验沉淀]] 第八类「外部资料导入类」(8-x 编号递增)。

| # | 问题 | 答案（无则填“无”）| kp_target（回流目标）| 是否追加到经验沉淀 |
|:---:|---|---|---|:---:|
| 1 | 本书的字符 / 公式 / 图片有什么**特有的转换坑**？（如 mineru 漏渲染、特殊符号被吞、上下标错位） |  | | ⬜ |
| 2 | 本书的术语有没有跟其他来源**冲突**？（同一概念两种叫法、不同书章节命名差异） |  | | ⬜ |
| 3 | 有没有遇到 mineru 漏掉、需要肉眼补的图？（章节号、对应原文位置） |  | | ⬜ |
| 4 | 哪些段落让 AI **困惑或卡住**？（公式歧义、章节跨页、原文笔误等） |  | | ⬜ |

> **kp_target 填写规范**（v1.2 新增）：若答案非“无”，在此列标注该坑点应回流到哪个 KP 的哪个段落。格式：`[[知识点名]]§段落名` 或 `[[知识点名]]§十一`（易错点）。如坑点涉及多个 KP，用逗号分隔。示例：`[[配合物]]§十一, [[晶体场理论]]§十一`
>
> **触发条件**：任一问题答案非“无” → ① 必须打开 [[经验沉淀]] 追加新条目；② 根据 kp_target 在对应 KP 的「十一、易错点」中插入 wikilink 引用该提炼笔记。如“无”则在工作日志的“经验沉淀候选”段落明示“本次提炼无新坑”。
