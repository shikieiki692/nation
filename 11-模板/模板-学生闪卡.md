# 模板：学生闪卡

> 📋 **使用方式**：从 KP 核心结论或专题页对比表中提取「问题→答案」对，生成可打印的纸质闪卡，同时保留 Anki 字段以便有条件的学生导入电子版。
>
> 🎯 **课型特征**：**记忆层工具**。每张卡片只承载 1 个最小知识单元，通过主动回忆+间隔重复固化长期记忆。
>
> 📐 **纸质规格**：A4 纸打印后裁切，推荐 8 张/页（每卡 96mm × 68mm，类似扑克牌大小），双面打印。也可 4 张/页（大卡片，适合公式/机理）。
>
> 💻 **电子辅助**：Anki 字段同步标注，学生可扫码/手动导入 Anki 实现算法驱动的间隔重复。

## 一、闪卡分类体系

| 类型 | 适用内容 | 正面形态 | 背面形态 | 推荐尺寸 |
|:---|:---|:---|:---|:---:|
| **概念卡** | 定义、分类、适用条件 | 术语 / "什么是 X？" | 定义 + 关键特征 3 条 | 标准 |
| **公式卡** | 公式、方程、关系式 | 公式名称 / "写出 X 的表达式" | 公式 + 各符号含义 + 适用条件 | 大号 |
| **判据卡** | 判断规则、阈值、边界条件 | 情境 / "何时可用简化公式？" | 判据 + 反例（错用后果） | 标准 |
| **方程式卡** | 化学反应方程式 | 反应物 + 条件（Cloze 挖空） | 配平方程式 + 现象/备注 | 大号 |
| **对比卡** | 易混概念辨析 | "X 与 Y 的 3 个区别？" | 对比表（≤3 行）+ 速记口诀 | 标准 |
| **信号卡** | 元素推断/解题信号 | 现象 / "蓝色溶液→？" | 推断路径 + 验证操作 | 标准 |

## 二、纸质闪卡版式

### 2.1 标准尺寸版（8 张/页 A4）

**正面（回忆面）**：

```text
┌─────────────────────────────┐
│  [类型标记] 概念卡            │
│                              │
│        什么是 Ksp？          │
│                              │
│   （写出定义 + 表达式）       │
│                              │
│  ─────────────────────────  │
│  来源：[[溶度积]]  难度：⭐⭐ │
└─────────────────────────────┘
```

**背面（答案面）**：

```text
┌─────────────────────────────┐
│  答案                        │
│  ─────────────────────────  │
│  Ksp 是沉淀溶解平衡的平衡常   │
│  数。对于 AmBn(s) ⇌ mAn+ +   │
│  nBm-，Ksp = [An+]^m[Bm-]^n │
│                              │
│  ⚠️ 注意：纯固体不写入表达式  │
│                              │
│  常见陷阱：                   │
│  ☐ 固体写入 Ksp 表达式       │
│  ☐ 忘记化学计量数变指数       │
│                              │
│  关联：[[溶解度]] [[同离子效应│
│  ]]                          │
└─────────────────────────────┘
```

### 2.2 大号尺寸版（4 张/页 A4）

用于公式推导、反应机理、多步计算等需要较大空间的场景。

**正面**：

```text
┌─────────────────────────────────────┐
│  [公式卡]  理想气体状态方程          │
│                                      │
│  写出 pV = nRT 的密度形式，并说明   │
│  各符号含义与单位匹配规则。          │
│                                      │
│  ─────────────────────────────────  │
│  来源：[[理想气体状态方程]]  ⭐⭐     │
└─────────────────────────────────────┘
```

**背面**：

```text
┌─────────────────────────────────────┐
│  答案                                │
│  ─────────────────────────────────  │
│  ρ = pM / (RT)                       │
│                                      │
│  ρ：密度 (kg·m⁻³)                    │
│  p：压强 (Pa)                        │
│  M：摩尔质量 (kg·mol⁻¹)              │
│  R：8.314 J·mol⁻¹·K⁻¹               │
│  T：热力学温度 (K)                   │
│                                      │
│  ⚠️ 单位匹配是最大陷阱！              │
│  若 p 用 kPa，需对应 R = 8.314       │
│  kPa·dm³·mol⁻¹·K⁻¹                  │
│                                      │
│  关联：[[Dalton分压定律]]            │
└─────────────────────────────────────┘
```

## 三、Anki 兼容字段（电子版辅助）

如学生使用 Anki，同一张卡片可导出为以下字段：

| Anki 字段 | 纸质对应 | 说明 |
|:---|:---|:---|
| `Front` | 正面问题 | 支持 LaTeX、图片嵌入 |
| `Back` | 背面答案 | 支持 LaTeX、图片嵌入 |
| `Tags` | 来源 KP + 专题 | 如 `化学原理::化学平衡::溶度积` |
| `Note Type` | 卡片类型 | `概念卡` / `公式卡` / `方程式卡` 等 |
| `Extra` | 常见陷阱 + 关联 KP | 仅背面底部小字 |
| `Source` | 来源 wikilink | `[[溶度积]]` |
| `Difficulty` | ⭐标记 | 用于筛选复习优先级 |
| `Round` | 轮次 | `第一轮` / `第二轮` 等，控制显示时机 |

> 💡 **导入方式**：在 Obsidian 中通过 Dataview 查询生成 CSV → 学生用 Anki 导入。或提供二维码链接到网盘 CSV。

## 四、批量生成工作流

```text
Step 1：选定目标专题
   ↓ 读取对应专题页 + 相关 KP
Step 2：提取「问题→答案」对
   ↓ 从 KP §四关键结论、§五常见陷阱、专题页对比表中提取
Step 3：按卡片类型分类
   ↓ 概念卡 / 公式卡 / 判据卡 / 方程式卡 / 对比卡 / 信号卡
Step 4：填入模板
   ↓ 每页排 8 张（标准）或 4 张（大号）
Step 5：输出
   ↓ 纸质：PDF 打印文件（A4，双面）
   ↓ 电子：CSV（Anki 导入）或 Markdown（Obsidian 闪卡插件）
```

## 五、使用指南（给学生）

### 5.1 纸质闪卡使用法（Leitner 盒子法）

准备 5 个盒子/夹子，标记为：**每天、每 2 天、每 4 天、每 8 天、已掌握**。

```text
复习流程：
1. 从"每天"盒子中抽卡，看正面回忆背面
2. 回忆正确 → 移入下一级盒子（每天→2天→4天→8天→已掌握）
3. 回忆错误或模糊 → 退回"每天"盒子
4. 每天只复习当天盒子中的卡片，不超量（建议 20-30 张/天）
```

### 5.2 与课堂进度配合

| 时机 | 动作 |
|:---|:---|
| **新授课后** | 从当课 KP 提取 5-10 张闪卡，当天制作 |
| **周末** | 复习本周全部闪卡（Leitner 法） |
| **专题结束后** | 汇总该专题全部闪卡，做一次完整复习 |
| **考前 2 周** | 只复习"8 天"和"已掌握"盒子中仍记错的卡片 |

## 六、与知识库的联动规则

| KB 资产 | 闪卡提取位置 | 示例 |
|:---|:---|:---|
| **KP §四关键结论** | → 概念卡 / 公式卡正面 | "写出 Ksp 表达式" |
| **KP §五常见陷阱** | → 判据卡 / 对比卡 | "Ksp 大=溶解度大？为什么错" |
| **专题页对比表格** | → 对比卡（每行 1 张） | "Brønsted 与 Lewis 酸的 3 个区别" |
| **专题页信号-响应矩阵** | → 信号卡 | "紫色溶液→可能物种+验证" |
| **新授课课后作业** | → 方程式卡（题目变正面，方程式变背面） | "写出 Cl₂ + NaOH（热）的反应" |
| **方法微专题口诀** | → 口诀卡 | "极限假设法的 3 个信号" |

## 七、质量检查清单

产出闪卡前逐项确认：

- [ ] 每张卡片只包含 1 个最小知识单元（不堆叠多个概念）
- [ ] 正面问题明确，不泄露答案信息
- [ ] 背面答案完整，包含关键细节和易错点
- [ ] 来源 KP 已标注 wikilink，方便学生点击深入
- [ ] 难度标记与 ⭐ 体系一致
- [ ] 公式卡已检查 LaTeX 渲染正确
- [ ] 方程式卡已配平并标注条件
- [ ] 纸质版已排版测试（8 张/页或 4 张/页）
- [ ] 如有 CSV，已测试 Anki 导入成功
