# 题型：SN1/SN2/E1/E2 竞争判断

## 一、题型特征

- 给出一个卤代烷（或醇/醚等可转化为离去基底的底物）和一个反应条件
- 要求判断主要产物及反应机理（SN1/SN2/E1/E2）
- 常伴随立体化学问题（构型翻转/外消旋/反式消除）

## 二、解题步骤

### Step 1：识别底物类型

| 底物 | 倾向 |
|------|------|
| 甲基 / 1° | SN2（强碱下E2） |
| 2° | SN2/E2 竞争 |
| 3° / 苄基 / 烯丙基 | SN1/E1（强碱下E2） |
| 桥头碳 | SN1/E1（SN2空间受阻） |

### Step 2：识别试剂性质

- **强亲核/弱碱**（I⁻, Br⁻, RS⁻, N₃⁻）→ SN2
- **强亲核/强碱**（HO⁻, RO⁻）→ SN2/E2
- **弱亲核/强碱**（t-BuO⁻）→ E2
- **弱亲核/弱碱**（H₂O, ROH）→ SN1/E1

### Step 3：识别溶剂类型

- 极性质子溶剂 → 有利于 SN1/E1
- 极性非质子溶剂 → 有利于 SN2/E2

### Step 4：识别温度

- 低温 → 取代有利（动力学）
- 高温 → 消除有利（热力学）

## 三、典型例题

### 例 1
**题目**：(R)-2-溴丁烷与 NaOH 水溶液反应，主要产物是什么？

**分析**：
- 底物：2° 溴代烷
- 试剂：OH⁻（强亲核/强碱）
- 溶剂：水（极性质子溶剂）

**答案**：SN2 为主，产物为 (S)-2-丁醇（构型翻转）。若温度较高，可能有 E2 副产物 1-丁烯/2-丁烯。

### 例 2
**题目**：叔丁基溴在乙醇中加热，主要产物是什么？

**分析**：
- 底物：3°
- 试剂：EtOH（弱亲核/弱碱）
- 条件：加热

**答案**：SN1/E1 竞争。EtOH 既是溶剂又是亲核试剂，生成乙基叔丁基醚（SN1）和异丁烯（E1）。高温下 E1 比例增加。

## 四、易错点

- 苄基卤虽然像 1°，但 SN1 也能进行（中间体稳定）
- 环己烷体系的 E2 要求反式共平面
- SN1 中的亲核试剂浓度不影响速率
