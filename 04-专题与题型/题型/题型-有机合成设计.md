# 题型：有机合成设计

## 一、题型特征

- 给出目标分子（TM）结构
- 要求设计从指定起始原料（或常见简单原料）到 TM 的合成路线
- 可能涉及：碳骨架构建、官能团引入与转化、立体化学控制、保护基使用

## 二、解题步骤

### Step 1：分析目标分子

1. **数碳原子**：确定需要增加/减少的碳数
2. **识别官能团**：标记所有官能团及其相对位置
3. **识别手性中心**：注意立体化学要求
4. **识别关键结构单元**：如 β-羟基羰基、1,3-二羰基等

### Step 2：逆合成分析

1. **寻找可断开的 C-C 键**
   - 优先断开带有官能团的碳附近的 C-C 键
   - 利用已知反应的逆过程
2. **寻找官能团转化关系**
   - FGI：如羧酸 ← 醛 ← 醇
   - FGA：如引入羰基作为活化基团
3. **简化到常见起始原料**

### Step 3：验证合成路线

1. **每步反应的可行性**：试剂、条件是否合理
2. **官能团兼容性**：前步引入的官能团是否会被后步破坏
3. **选择性**：当有多个反应位点时，是否选择性好
4. **是否需要保护基**

## 三、常见 C-C 键构建策略

| 目标结构 | 逆合成切断 | 构建反应 |
|---------|-----------|---------|
| 醇 | R-MgX + 醛/酮 | Grignard 反应 |
| β-羟基羰基 | α,β-切断 | 羟醛缩合 |
| α,β-不饱和羰基 | α,β-切断 + 脱水 | 羟醛缩合-脱水 |
| 1,3-二羰基 | C2-C3 切断 | Claisen 缩合 |
| 烯烃 | 磷叶立德 + 羰基 | Wittig 反应 |
| 联芳基 | 两个芳基切断 | Suzuki 偶联 |

## 四、典型例题

### 例 1
**题目**：由苯和不超过 4 个碳的有机物合成 4-苯基-2-丁酮。

**逆合成分析**：
```
Ph-CH₂-CH₂-CO-CH₃
          ↑
    可看作羟醛缩合产物
          ↑
Ph-CH₂-CHO + CH₃-CO-CH₃ 或类似组合
```

**合成路线**（示意）：
1. 苯 + CH₃Cl/AlCl₃ → 甲苯
2. 甲苯侧链氯化 → 苄基氯
3. 苄基氯 → 苯乙腈 → 苯乙酸 → 苯乙醛
4. 苯乙醛 + 丙酮/碱 → 羟醛产物 → 脱水 → 不饱和酮 → 氢化 → 目标产物

## 五、评分要点

1. **路线合理性**：每步反应是否正确
2. **原料限制**：是否符合题目要求
3. **选择性**：位置选择性、立体选择性
4. **简洁性**：步骤越少越好
