**建议用途：** 适合接在第一屏后，专门把“氧化数配平”和“电极电势判断”连接起来，避免学生把氧化还原和电化学学成两章。

```text
先配平氧化还原
  ↓
明确电子转移数 n
  ↓
区分谁被氧化 / 谁被还原
  ↓
写成半反应
  ↓
再接 E° / Nernst / 原电池方向
```

**板书跟讲顺序：**
- 先把“氧化还原方程式会不会写”当成前提检验。
- 再把同一反应拆成两个半反应。
- 最后才允许进入电势与自发性判断。

**本页课堂入口：** 先把“热力学能不能”与“非标准态怎么算”分开，不要一上来就代 `E°`。

**先问四个问题：**

1. 题目是要判断方向、写方程式，还是算电动势/平衡常数？
2. 当前是标准态还是非标准态，能不能直接用 `E°`？
3. 介质已经确定了吗，产物会不会因为酸碱性而变化？
4. 这题更像原电池、电解池，还是浓差电池？

**一屏判断卡：**

- 先判介质，再判产物，再配平。
- 给浓度、分压、pH 就优先想 Nernst。
- `E°` 只回答“能不能”，不回答“快不快”。
- 电化学题最后要做一次“电荷 / 原子 / 符号”三检。

## 一、核心结论汇总 {#core-conclusions}

**必须记住：**
1. **电极电势是热力学判据**：$E^{\ominus}$ 越正，氧化态氧化性越强；$E_{\text{池}}^{\ominus} > 0 \iff \Delta G^{\ominus} < 0 \iff$ 反应自发。但它只回答"能不能"，不回答"快不快"。
2. **非标准态必须用 Nernst 方程**：只要题目出现具体浓度、分压、pH 偏离 0（强酸性）或 14（强碱性），立刻从 $E^{\ominus}$ 切换到 $E = E^{\ominus} - \frac{0.0592}{n}\lg Q$。
3. **介质决定产物**：同一氧化剂在不同酸碱性下还原产物可能完全不同（如 $\mathrm{MnO_4^-}$ 酸性→$\mathrm{Mn^{2+}}$，中性→$\mathrm{MnO_2}$，强碱→$\mathrm{MnO_4^{2-}}$），配平前必须先判断介质。

**最高频决策路径：**

```mermaid
flowchart TD
  A[题目出现氧化还原/电池/电极] --> B{是否给出具体浓度或pH?}
  B -->|是| C[用Nernst方程求E<br/>查[[Nernst方程]]]
  B -->|否| D{要求判断方向或选择试剂?}
  D -->|是| E[查标准电极电势表比较E°<br/>查[[标准电极电势]]]
  D -->|否| F[要求写方程式?]
  F -->|是| G[先判断介质→再选产物→半反应配平<br/>查[[氧化还原反应方程式配平]]]
  F -->|否| H[可能为电解池/超电势/动力学问题<br/>查[[电化学]]]
```

## 二、对比表格 {#comparison-table}

| 触发条件（题目关键词） | 比较维度 | A | B | 常见陷阱 |
|:---|:---|:---|:---|:---|
| "标准电极电势" / "查表判断" | 使用场景 | 标准态（1 mol/L, 1 bar, 25°C） | 任意浓度/分压/pH | 题目给浓度却直接用 $E^{\ominus}$ 比较 |
| "电动势计算" / "非标准态" | 核心公式 | $E = E^{\ominus} - \frac{0.0592}{n}\lg Q$（电池） | $E = E^{\ominus} + \frac{0.0592}{n}\lg\frac{[\text{Ox}]}{[\text{Red}]}$（电极） | 混淆 $Q$ 与 $K$；气体未用分压 |
| "酸性 $\mathrm{KMnO_4}$" / "碱性 $\mathrm{KMnO_4}$" | 还原产物 | 酸性 → $\mathrm{Mn^{2+}}$（无色） | 强碱 → $\mathrm{MnO_4^{2-}}$（绿色） | 中性条件下写成 $\mathrm{Mn^{2+}}$；碱性条件下出现 $\mathrm{H^+}$ |
| "浓差电池" | 电动势来源 | 两电极相同，$E^{\ominus}_{\text{池}} = 0$ | $E = \frac{0.0592}{n}\lg\frac{c_{\text{浓}}}{c_{\text{稀}}}$ | 错误代入 $E^{\ominus}$，认为标准电势不为零 |
| "$E^{\ominus}$ 加减计算" | 操作方法 | 不能直接加减 $E^{\ominus}$ | 必须通过 $\Delta G^{\ominus} = -nFE^{\ominus}$ 转换 | 元素电势图（Latimer图）求中间电势时直接算术平均 |
| "原电池" vs "电解池" | 能量转化 | 化学能 → 电能（自发） | 电能 → 化学能（非自发，外加电压） | 电解池阴极=还原反应，但连接电源负极；阳极=氧化反应，连接电源正极 |
| "电池比容量/储能密度" | 计算基准 | 比容量 $C = nF/M$（mA·h/g） | 储能密度 $\rho = nFE/m_{\text{总}}$（Wh/kg） | 用产物的式量代替反应物；电子数 $n$ 取错 |

## 三、解题套路 / 决策流程 {#problem-solving-routine}

### Step 1：审题——提取氧化还原三要素
- **操作**：圈出氧化剂、还原剂、介质（酸性/碱性/中性）、是否有具体浓度或 pH。
- **依据 KP**：[[氧化还原反应方程式配平]]（介质判断与产物确定）
- **检查点**：☐ 介质已确认 ☐ 所有变价元素已标出氧化态

### Step 2：选择工具——标态 vs 非标态
- **操作**：若所有物种浓度≈1 mol/L、气体分压≈1 bar、pH≈0（酸）或 14（碱），则用 $E^{\ominus}$；否则启动 Nernst 方程。
- **依据 KP**：[[标准电极电势]]（查表比较）或 [[Nernst方程]]（非标准态修正）
- **检查点**：☐ 是否标准态已判断 ☐ 温度是否为 25°C（决定系数 0.0592）

### Step 3：写半反应并配平（离子-电子法）
- **操作**：拆分为氧化半反应和还原半反应 → 配平原子（O 用 $\mathrm{H_2O}$，H 用 $\mathrm{H^+}$/OH⁻）→ 配电荷（加 e⁻）→ 电子数相等后相加。
- **依据 KP**：[[氧化还原反应方程式配平]]
- **检查点**：☐ 原子守恒 ☐ 电荷守恒 ☐ 介质一致（碱性条件下无游离 $\mathrm{H^+}$）

### Step 4：计算与判断
- **操作**：$E_{\text{池}} = E_{\text{正}} - E_{\text{负}}$；若求 $K$，用 $\lg K = \frac{nE^{\ominus}_{\text{池}}}{0.0592}$。
- **依据 KP**：[[标准电极电势]] / [[Nernst方程]]
- **检查点**：☐ 正负极未颠倒 ☐ 电子转移数 $n$ 与配平方程式一致

### Step 5：验证合理性
- **操作**：结果是否满足基本化学直觉？（如电动势是否为正、产物颜色是否与题述一致）
- **依据 KP**：[[电化学]]（动力学限制、超电势）
- **检查点**：☐ $E_{\text{池}}^{\ominus} > 0$ 仅说明热力学可行，速率未知 ☐ 比容量/储能密度的单位换算正确

| 步骤 | 核心操作 | 依据 KP | 检查清单 |
|:---|:---|:---|:---|
| 1 | 提取氧化剂、还原剂、介质、浓度 | [[氧化还原反应方程式配平]] | ☐ 介质确认 ☐ 变价元素标出 |
| 2 | 判断标态/非标态，选择 $E^{\ominus}$ 或 Nernst | [[标准电极电势]] / [[Nernst方程]] | ☐ 标准态判断 ☐ 温度确认 |
| 3 | 离子-电子法写半反应、配平 | [[氧化还原反应方程式配平]] | ☐ 原子守恒 ☐ 电荷守恒 ☐ 介质一致 |

## 四、反应机理拆解（含检查表） {#mechanism-analysis}

> 氧化还原反应的本质是电子转移。以下以**离子-电子法配平酸性介质中 $\mathrm{MnO_4^-}$ 氧化 $\mathrm{Fe^{2+}}$**为例，演示"电子流"拆解。

#### 步骤 1：拆分半反应并确定电子流方向
- **攻击位点**：$\mathrm{MnO_4^-}$ 中 Mn 为 +7（高氧化态，缺电子），作为氧化剂接受电子；$\mathrm{Fe^{2+}}$ 作为还原剂失去电子。
- **形成键**：无新共价键形成（水溶液离子反应）。
- **断裂键**：无共价键断裂。
- **电子流**：$\mathrm{Fe^{2+} \to Fe^{3+} + e^-}$（氧化，电子流出）；$\mathrm{MnO_4^- + 5e^- \to Mn^{2+}}$（还原，电子流入）。
- **学生任务（接力点）**：下一位同学需要判断的是——**在碱性介质中，$\mathrm{MnO_4^-}$ 的还原产物将是什么？**
- **检查表**：
  - ☐ 氧化剂（得电子）与还原剂（失电子）已正确区分
  - ☐ 氧化数变化计算正确（Mn +7→+2 得 5e；Fe +2→+3 失 1e）
  - ☐ 电子得失守恒（5Fe²⁺ 失 5e = 1MnO₄⁻ 得 5e）

#### 步骤 2：配平半反应（原子与电荷）
- **还原半反应**：$\mathrm{MnO_4^- + 8H^+ + 5e^- \longrightarrow Mn^{2+} + 4H_2O}$
  - O 配平：左侧 4 个 O → 右侧 4 个 $\mathrm{H_2O}$
  - H 配平：右侧 8 个 H → 左侧 8 个 $\mathrm{H^+}$
  - 电荷配平：左侧 (-1) + (+8) + (-5) = +2；右侧 +2 → 已平衡
- **氧化半反应**：$\mathrm{Fe^{2+} \longrightarrow Fe^{3+} + e^-}$
- **合并**：×5 氧化半反应后与还原半反应相加，消去电子：
  $$\mathrm{MnO_4^- + 5Fe^{2+} + 8H^+ \longrightarrow Mn^{2+} + 5Fe^{3+} + 4H_2O}$$
- **检查表**：
  - ☐ 原子守恒（Mn、O、H、Fe 左右相等）
  - ☐ 电荷守恒（左侧 -1 + 10 + 8 = +17；右侧 +2 + 15 = +17）
  - ☐ 介质一致（酸性介质，无游离 $\mathrm{OH^-}$）

## 五、典型例题串讲 {#typical-examples}

### 例题 1：普鲁士蓝电极反应与比容量（39届初赛第2题，难度 ⭐⭐⭐⭐）

**题目：**
普鲁士蓝类化合物 PBA-1 的化学式为 $\mathrm{KFe_2(CN)_6}$，式量 306.92。**2.1.1** 写出生成普鲁士白的电极反应方程式；**2.1.3** 计算其作为正极材料的理论比容量（mA·h·g⁻¹）。

**分析：**
1. 先判断铁的价态：PBA-1 中 K 为 +1，CN 为 -1，故 2Fe 总价态为 +3 → 混合价态 $\mathrm{Fe^{III}Fe^{II}}$。
2. 嵌入 $\mathrm{K^+}$ 后变成普鲁士白（全白），说明所有 Fe 变为同一价态——被还原为 $\mathrm{Fe^{II}Fe^{II}}$，即每个化学式单元接受 1 个电子。
3. 比容量计算：关键是确定"每摩尔活性物质转移多少摩尔电子"。PBA-1 到普鲁士白只嵌入 1 个 $\mathrm{K^+}$ 和 1 个 e⁻，但题目后续提示可逆储钾过程为 $2\mathrm{K^+} + 2\mathrm{e^-}$（两个 Fe 均可变价），因此 $n = 2$。

**解答：**
- 电极反应：$\mathrm{KFe^{III}Fe^{II}(CN)_6 + K^+ + e^- \longrightarrow K_2Fe_2^{II}(CN)_6}$
- 理论比容量：$C = \dfrac{nF}{M} = \dfrac{2 \times 96485}{306.92} = 628.74\ \mathrm{A\cdot s\cdot g^{-1}} = 174.65\ \mathrm{mA\cdot h\cdot g^{-1}}$

**反思：**
- 比容量必须用**反应物（PBA-1）的式量**作分母，不能用产物。
- 电子数 $n$ 的判定要结合题目给出的"可逆储钾"全过程，不能仅看单步反应。
- 颜色变化解释需区分 d-d 跃迁与 CT 跃迁——混合价态化合物通常是 CT 跃迁主导。

### 例题 2：Zn-I₂ 二次电池储能密度与电动势计算（37届决赛第4题，难度 ⭐⭐⭐⭐⭐）

**题目（节选 4-3 与 4-4）：**
某 $\mathrm{Zn-I_2}$ 电池，负极区为 $\mathrm{ZnSO_4}$，正极区为 $\mathrm{CuCl_2}$。实验测得高电位放电 $E^{\ominus} = +1.20\ \mathrm{V}$（反应：$2\mathrm{ICl} + 2\mathrm{e^-} \rightleftharpoons \mathrm{I_2} + 2\mathrm{Cl^-}$），低电位放电 $E^{\ominus} = +0.70\ \mathrm{V}$（反应：$2\mathrm{Cu^{2+}} + \mathrm{I_2} + 4\mathrm{e^-} \rightleftharpoons 2\mathrm{CuI}$）。

**4-3-2** 计算基于正负极总质量的储能密度（Wh/kg）。

**4-4** 电池表示为 $(-)\ \mathrm{Zn|ZnSO_4(1.0\ M)\|\|CuCl_2(1.0\ M)|I_2(s)}\ (+)$，以 200 mA 恒电流放电 482 s 后，计算电池电动势。

**分析：**
1. **储能密度**：需要把整个放电过程的总电能除以总质量。总电能 = 各阶段 $n_i F E_i$ 之和；总质量 = 所有参与电极反应的物质质量之和（注意是正负极总质量，包括 $\mathrm{Zn}$、$\mathrm{CuCl_2}$、$\mathrm{I_2}$ 转化后的产物）。
2. **恒流放电后的电动势**：先算转移电子数，再判断正极区和负极区剩余各物种的浓度，最后用 Nernst 方程分别计算两个电极的 $E$，相减得 $E_{\text{池}}$。

**解答（4-4 重点）：**
- 转移电子：$n_e = \dfrac{It}{F} = \dfrac{0.200 \times 482}{96485} = 1.00\times 10^{-3}\ \mathrm{mol} = 1.00\ \mathrm{mmol}$
- 负极：$\mathrm{Zn - 2e^- \to Zn^{2+}}$，放电后 $[\mathrm{Zn^{2+}}] = 1.0 + \dfrac{1.00}{2 \times 1.0} = 1.5\ \mathrm{mol\cdot L^{-1}}$
- 正极：$2\mathrm{Cu^{2+}} + \mathrm{I_2} + 4\mathrm{e^-} \to 2\mathrm{CuI}$，消耗 $\mathrm{Cu^{2+}}$ 为 $\dfrac{1.00}{2} = 0.50\ \mathrm{mmol}$，剩余 $[\mathrm{Cu^{2+}}] = 1.0 - 0.50 = 0.50\ \mathrm{mol\cdot L^{-1}}$
- Nernst 方程：
  $$E_{\text{池}} = \left(0.70 + \frac{0.0592}{4}\lg[\mathrm{Cu^{2+}}]^2\right) - \left(-0.762 + \frac{0.0592}{2}\lg[\mathrm{Zn^{2+}}]\right) = 1.45\ \mathrm{V}$$

**反思：**
- 恒流放电题的核心是"电量守恒"：$Q = It = n_e F$。算出 $n_e$ 后，把它分配到两个电极的浓度变化中。
- 注意正极反应消耗 $\mathrm{Cu^{2+}}$ 的化学计量比：4e⁻ 对应 2Cu²⁺，即 2e⁻ 对应 1Cu²⁺。
- Nernst 方程中固体（$\mathrm{I_2}$、$\mathrm{CuI}$）活度取 1，不出现在对数项中。

### 例题 3：沉淀对电极电势的影响（周坤无机，⭐⭐⭐）

**题目：**
向标准 Ag-Ag⁺ 电极（φ° = 0.80 V）溶液中加入 KCl，使 c(Cl⁻) = 1.0×10⁻² mol/dm³。求此时电极电势 E（Ksp(AgCl) = 1.8×10⁻¹⁰）。

**分析：**
Ksp 与 Nernst 方程联用：沉淀降低 [Ag⁺]，导致电势下降。先由 Ksp 求 [Ag⁺]，再代入 Nernst 方程。

**解答：**
- [Ag⁺] = Ksp/[Cl⁻] = 1.8×10⁻¹⁰/10⁻² = 1.8×10⁻⁸ M
- E = E° + 0.0592lg[Ag⁺] = 0.80 + 0.0592lg(1.8×10⁻⁸)
- E = 0.80 + 0.0592×(−7.74) = 0.80 − 0.46 = **0.34 V**

**反思：**
- 沉淀使氧化型浓度大幅降低，电势显著下降（0.80 V → 0.34 V）
- 通式：φ°(AgX/Ag) = φ°(Ag⁺/Ag) + 0.0592lgKsp(AgX)。Ksp 越小，电势越低
- AgI 的 Ksp 更小，φ°(AgI/Ag) ≈ −0.15 V，说明 Ag 在 I⁻ 存在下还原性大增

### 例题 4：Latimer 图与歧化反应判断（周坤无机，⭐⭐⭐⭐）

**题目：**
碱性介质中锰的元素电势图：MnO₄⁻ →(0.56 V) MnO₄²⁻ →(2.26 V) MnO₂。判断 MnO₄²⁻ 能否歧化，写出反应式，并求 MnO₄⁻/MnO₂ 的标准电势。

**分析：**
歧化判断规则：φ右 > φ左 则中间氧化态歧化。元素电势图计算：E₃° = (n₁E₁° + n₂E₂°)/(n₁+n₂)。

**解答：**
1. **歧化判断**：φ(MnO₄²⁻/MnO₂) = 2.26 V > φ(MnO₄⁻/MnO₄²⁻) = 0.56 V → **MnO₄²⁻ 能歧化**
2. **反应式**：3MnO₄²⁻ + 2H₂O → 2MnO₄⁻ + MnO₂ + 4OH⁻
3. **MnO₄⁻/MnO₂ 电势**：
   - MnO₄⁻ → MnO₄²⁻：1e⁻，E₁° = 0.56 V
   - MnO₄²⁻ → MnO₂：2e⁻，E₂° = 2.26 V
   - 总反应 MnO₄⁻ → MnO₂：3e⁻
   - E₃° = (1×0.56 + 2×2.26)/(1+2) = 5.08/3 = **1.69 V**

**反思：**
- **电势不能直接加减**，必须通过 ΔG = −nFE 转换：ΔG₃° = ΔG₁° + ΔG₂° → n₃FE₃° = n₁FE₁° + n₂FE₂°
- 歧化反应的推动力来自 φ右 − φ左 > 0，差值越大歧化趋势越强
- Latimer 图是判断多氧化态元素稳定性的核心工具

## 七点五、真题链与讲评顺序 {#exam-sequence}

- `第 1 题`：先讲标准电极电势比较题，稳住“方向判断”和“热力学判据”。
- `第 2 题`：再讲 Nernst 非标准态题，把浓度、pH、分压怎样进入 `Q` 讲清。
- `第 3 题`：最后讲介质决定产物或电池综合题，把配平、电动势和平衡常数收成一条线。
- 课堂顺序建议：`标态判断题 → 非标态修正题 → 综合配平/联立题`，先立抓手再做长链。

### 图后立刻练 / 讲后 1 题 / 课后 2 题

- 图后立刻练：给一题短题，只要求学生先判“能不能直接用 `E°`”。
- 讲后 1 题：选一题典型 Nernst 真题，完整写出 `Q`、代入、判断方向。
- 课后 2 题：一题酸碱介质配平题，一题电池/平衡常数联立题，训练从方程到电势的完整链。

### 真题入口使用建议

- **第一轮基础班**：优先用"氧化数配平 + 标准电极电势表查阅"题热身，先让学生建立"电势=反应方向的数"的直觉，不急着上 Nernst。
- **第三轮深化**：把 Nernst 方程、浓差电池、pH-电势图三样打包讲——它们是同一公式的三个应用场景，不要拆成三节。
- **第四轮冲刺**：真题里的电化学几乎都是联立题（E° + pKa → BDE、E° + Ksp → 配合物稳定性），练的不是电化学本身，是"如何在热力学循环里用电化学数据搭桥"。

### 真题链与讲评顺序 {#exam-sequence}

- `第 1 题`：**氧化数配平 + 电极电势正负判断**。课堂用途：热启动，确认基本工具还在。
- `第 2 题`：**Nernst 方程计算（浓度/温度 → 电势变化）**。课堂用途：主讲题，重点训练"对数项里是谁、系数怎么进指数"。
- `第 3 题`：**电化学联立热力学循环**（E° → ΔG° → K / BDE）。课堂用途：收束题，把电化学从"独立一章"变成"热力学工具箱里的一把螺丝刀"。
- 课堂顺序建议：`配平+电势方向 → Nernst 单步 → 联立循环`，不先上最综合的联立题。选择性的坑（哪种介质先配平什么）放在第二题后半段自然引出。
