## 零点六、课堂投影速查卡 {#classroom-quick-card}

**本课课堂入口：** 不要从"Ksp 的定义"开始。先问一个陷阱题——"AgCl 的 Ksp = 1.8×10⁻¹⁰，Ag₂CrO₄ 的 Ksp = 1.1×10⁻¹²——谁的 Ksp 更小？那谁先沉淀？"——如果学生说"Ag₂CrO₄ 先沉淀"，他们就掉进了异类型不能直接比 Ksp 的经典陷阱。Mohr 法正是利用了这一反直觉现象。

**先问三个问题：**

1. 混合后浓度变了吗？— 题目给的是"混合前"浓度，Q 必须用混合后浓度算！
2. 同类型还是异类型沉淀？— 同类型直接比 Ksp；异类型必须算 s 或所需沉淀剂最低浓度
3. 有没有同离子/配位/pH 效应？— Ksp 不变，但溶解度 s 会变

**一屏判断卡：**

```
沉淀溶解平衡题 → 先写 Ksp 表达式
    │
    ├─ 判断是否生成沉淀
    │   ├─ 算混合后浓度（⚠️ 体积变化！）
    │   ├─ Q = 离子积
    │   └─ Q > Ksp → 沉淀；Q < Ksp → 不沉淀
    │
    ├─ 计算溶解度 s
    │   ├─ 纯水：s = f(Ksp, 类型)
    │   │   ├─ 1:1 型 → s = √Ksp
    │   │   ├─ 1:2 / 2:1 型 → s = ³√(Ksp/4)
    │   │   └─ 1:3 型 → s = ⁴√(Ksp/27)
    │   ├─ 同离子溶液：s ≈ Ksp/c₀（s≪c₀ 时）
    │   └─ pH/配位效应：需联立方程组
    │
    ├─ 分步沉淀
    │   ├─ 同类型 → Ksp 小者先沉淀
    │   ├─ 异类型 → 算各离子所需最小沉淀剂浓度，取最小者
    │   └─ "完全沉淀"判据：离子浓度降至 < 10⁻⁵ mol/dm³
    │
    └─ 沉淀转化
        └─ K = Ksp(旧)/Ksp(新)，K ≫ 10⁶ 视为定量转化
```

**沉淀类型与 s-Ksp 关系速查：**

| 类型 | 示例 | s-Ksp 关系 |
|:---|:---|:---|
| AB (1:1) | AgCl, BaSO₄ | s = √Ksp |
| AB₂ (1:2) | PbCl₂, CaF₂ | s = ³√(Ksp/4) |
| A₂B (2:1) | Ag₂CrO₄ | s = ³√(Ksp/4) |
| AB₃ (1:3) | Fe(OH)₃ | s = ⁴√(Ksp/27) |

**影响 s 的五大因素：**

| 因素 | Ksp | s | 机制 |
|:---|:---:|:---:|:---|
| 温度↑ | 变（通常↑） | ↑ | 溶解吸热 |
| 同离子 | 不变 | **↓** | 平衡左移 |
| 盐效应 | 不变 | 略↑ | 离子氛降低有效浓度 |
| pH↓（弱酸盐） | 不变 | ↑ | H⁺ 消耗阴离子 |
| 配位 | 不变 | ↑ | 配体消耗金属离子 |

**Mohr 法原理（经典考点）：** Cl⁻ 溶液中加 CrO₄²⁻ 作指示剂，滴加 Ag⁺。AgCl (Ksp=1.8×10⁻¹⁰) 先沉淀，Cl⁻ 耗尽后，过量一滴 Ag⁺ 与 CrO₄²⁻ 生成砖红色 Ag₂CrO₄——虽然 Ag₂CrO₄ 的 Ksp 数值更小，但 AgCl 反而先沉淀，因为两者是异类型！

**课堂三问：** ① 混合后浓度算对了吗？② 同类型还是异类型？③ Ksp 不变但 s 可变——谁在变 s？

## 一、核心结论汇总 {#core-conclusions}

**必须记住：**
1. **$K_{sp}$ 是溶解度在"平衡常数语言"中的表达**：$K_{sp}$ 只与温度有关，与溶液中离子浓度无关。但**溶解度 $s$ 受同离子效应影响**——$K_{sp}$ 不变，$s$ 可变。
2. **溶度积规则 $Q \lessgtr K_{sp}$ 是判断沉淀/溶解的唯一判据**：$Q > K_{sp}$ → 沉淀；$Q = K_{sp}$ → 饱和；$Q < K_{sp}$ → 不饱和（或溶解）。这不是"浓度大小"的问题，是热力学判据。
3. **分步沉淀的本质是"谁的 $K_{sp}$ 先被 $Q$ 突破"**：不是看谁的 $K_{sp}$ 小，而是看谁的离子积先达到 $K_{sp}$。对同类型沉淀（如 AgCl vs AgBr，均为 1:1），$K_{sp}$ 更小的先沉淀；对异类型沉淀（如 AgCl 1:1 vs Ag₂CrO₄ 2:1），必须用 $Q$ 具体计算。
4. **沉淀转化的方向是 $K_{sp}$ 更小的方向**：$\mathrm{AgCl}(s) + \mathrm{I^-} \rightleftharpoons \mathrm{AgI}(s) + \mathrm{Cl^-}$，$K = K_{sp}(\mathrm{AgCl})/K_{sp}(\mathrm{AgI}) \gg 1$，热力学上自发。

**最高频决策路径：**

```text
拿到沉淀溶解平衡题
    ├── 已知 Ksp 和各离子浓度 → 算 Q → 比 Ksp → 判沉淀/溶解
    ├── 求溶解度 s →
    │   ├── 纯水中：s = f(Ksp)（1:1型 s=√Ksp；1:2型 s=∛(Ksp/4)）
    │   ├── 同离子溶液中：Ksp = s·(c₀ + s) ≈ s·c₀ → s = Ksp/c₀
    │   └── pH 依赖（弱酸盐/氢氧化物）：s 与 [H⁺] 有关
    ├── 分步沉淀 → 计算各离子开始沉淀所需的 [沉淀剂]
    │   └── 所需 [沉淀剂] 最小的先沉淀
    └── 沉淀转化 → K = Ksp(旧)/Ksp(新)，若 K ≫ 1 则转化完全
```

## 二、对比表格 {#comparison-table}

### 表 1：常见沉淀类型的 $K_{sp}$-溶解度关系

| 沉淀类型 | 例 | 溶解平衡 | $s$ 与 $K_{sp}$ 关系 |
|:---|:---|:---|:---|
| 1:1 (AB) | AgCl, BaSO₄ | $\mathrm{AB}(s) \rightleftharpoons \mathrm{A^+} + \mathrm{B^-}$ | $s = \sqrt{K_{sp}}$ |
| 1:2 (AB₂) | PbCl₂, CaF₂ | $\mathrm{AB_2}(s) \rightleftharpoons \mathrm{A^{2+}} + \mathrm{2B^-}$ | $s = \sqrt[3]{K_{sp}/4}$ |
| 2:1 (A₂B) | Ag₂CrO₄ | $\mathrm{A_2B}(s) \rightleftharpoons \mathrm{2A^+} + \mathrm{B^{2-}}$ | $s = \sqrt[3]{K_{sp}/4}$ |
| 1:3 (AB₃) | Fe(OH)₃ | $\mathrm{AB_3}(s) \rightleftharpoons \mathrm{A^{3+}} + \mathrm{3B^-}$ | $s = \sqrt[4]{K_{sp}/27}$ |

> ⚠️ **关键区分**：比较溶解度时，同类型（如 AgCl vs AgBr，均为 1:1）可直接比 $K_{sp}$；异类型（如 AgCl 1:1 vs Ag₂CrO₄ 2:1）必须算 $s$ 再比。

### 表 2：影响溶解度的因素

| 因素 | 对 $K_{sp}$ | 对 $s$ | 机制 | 定量关系 |
|:---|:---:|:---:|:---|:---|
| **温度** | 变 | 变 | 溶解通常吸热 → $T\uparrow, K_{sp}\uparrow, s\uparrow$ | van't Hoff 方程 |
| **同离子效应** | 不变 | **降低** | 平衡左移 | $s = K_{sp}/c_{\text{同离子}}$（近似） |
| **盐效应** | 不变 | **略增大** | 离子氛降低有效浓度 | $K_{sp}^0 = \gamma_+ \gamma_- K_{sp}$，$\gamma < 1$ |
| **pH**（弱酸盐） | 不变 | **增大** | $\mathrm{H^+}$ 消耗 $\mathrm{A^-}$，平衡右移 | $s = \sqrt{K_{sp}(1 + [\mathrm{H^+}]/K_a)}$ |
| **配位效应** | 不变 | **增大** | 配体消耗金属离子，平衡右移 | 与配合物 $K_{\text{稳}}$ 耦合 |

### 表 3：分步沉淀判断方法

| 条件 | 判断方法 | 例 |
|:---|:---|:---|
| 同类型沉淀 | 直接比 $K_{sp}$，$K_{sp}$ 小者先沉淀 | 加 Ag⁺ 到含 Cl⁻/Br⁻/I⁻ 混合液 → AgI 先沉淀 |
| 异类型沉淀 | 分别计算各离子开始沉淀所需沉淀剂浓度，取最小者 | 加 Ag⁺ 到含 Cl⁻/CrO₄²⁻ → 分别算 [Ag⁺]min |
| 完全沉淀判据 | 离子浓度降至 $10^{-5}\ \mathrm{mol\cdot dm^{-3}}$ 以下 | 判断"能否定量分离"的标准 |

## 三、解题套路 / 决策流程 {#problem-solving-routine}

### Step 1：写出溶解平衡方程和 $K_{sp}$ 表达式
- **操作**：确认沉淀化学式 → 写出解离方程 → 写出 $K_{sp} = [\mathrm{M}^{m+}]^a[\mathrm{N}^{n-}]^b$
- **检查点**：☐ 固体不写入 $K_{sp}$ ☐ 指数与化学计量系数一致

### Step 2：判断是否生成沉淀 — 算 $Q$ vs $K_{sp}$
- **操作**：用**混合后**（不是混合前！）的离子浓度算 $Q$
- **判据**：$Q > K_{sp}$ → 生成沉淀；$Q < K_{sp}$ → 无沉淀
- **检查点**：☐ 混合体积变化已考虑 ☐ 若有副反应（水解/配位），需先算游离离子浓度

### Step 3：计算溶解度 $s$（在有同离子/配位/pH 效应时）
- **同离子效应**：$K_{sp} = s(c_0 + s) \approx s \cdot c_0$ → $s = K_{sp}/c_0$
- **弱酸盐+酸**：$K_{sp} = [\mathrm{M}^{2+}][\mathrm{A^-}]^2$，其中 $[\mathrm{A^-}]$ 由分布分数 $\delta_{\mathrm{A^-}}$ 决定
- **配位效应**：需同时满足 $K_{sp}$ 和 $K_{\text{稳}}$，通常建立联立方程组

### Step 4：分步沉淀/沉淀转化 — 算 $K$ 值
- **分步沉淀**：计算各离子所需最小沉淀剂浓度，排序
- **沉淀转化**：$K = K_{sp}(\text{旧}) / K_{sp}(\text{新})$，若 $K > 10^6$ 视为定量转化
- **检查点**：☐ 是否同类型沉淀（决定能否直接比 $K_{sp}$） ☐ 转化后离子是否会再反应

| 步骤 | 核心操作 | 依据 KP | 检查清单 |
|:---|:---|:---|:---|
| 1 | 写 $K_{sp}$ 表达式 | [[溶度积]] | ☐ 固体不写 ☐ 指数正确 |
| 2 | 算混合后浓度 → 算 $Q$ → 比 $K_{sp}$ | [[溶度积规则]] | ☐ 体积变化已考虑 |
| 3 | 溶解度计算（考虑同离子/pH/配位） | [[同离子效应]] | ☐ 近似条件是否满足 |
| 4 | 分步沉淀/沉淀转化 | [[分步沉淀]]、[[沉淀转化]] | ☐ 同类型/异类型区分 |

## 五、典型例题串讲 {#typical-examples}

### 例题 1：溶度积规则基础判断（⭐⭐）

**题目**：将 $10\ \mathrm{cm^3}\ 0.010\ \mathrm{mol\cdot dm^{-3}}\ \mathrm{BaCl_2}$ 与 $10\ \mathrm{cm^3}\ 0.010\ \mathrm{mol\cdot dm^{-3}}\ \mathrm{Na_2SO_4}$ 混合，是否有 $\mathrm{BaSO_4}$ 沉淀生成？已知 $K_{sp}(\mathrm{BaSO_4}) = 1.1 \times 10^{-10}$。

**解答**：
混合后体积加倍，浓度减半：
$$[\mathrm{Ba^{2+}}] = [\mathrm{SO_4^{2-}}] = \frac{0.010}{2} = 5.0 \times 10^{-3}\ \mathrm{mol\cdot dm^{-3}}$$
$$Q = [\mathrm{Ba^{2+}}][\mathrm{SO_4^{2-}}] = (5.0 \times 10^{-3})^2 = 2.5 \times 10^{-5}$$

$Q = 2.5 \times 10^{-5} \gg K_{sp} = 1.1 \times 10^{-10}$ → 大量沉淀生成。

**反思**：最常见的错误是忘记混合后体积变化（直接用混合前浓度算 $Q = 10^{-4}$，虽然结论一致但这道题系数正好减半两次）。养成"混合后第一步：算新浓度"的习惯。

---

### 例题 2：分步沉淀——同类型 vs 异类型（⭐⭐⭐）

**题目**：溶液中 $[\mathrm{Cl^-}] = [\mathrm{CrO_4^{2-}}] = 0.010\ \mathrm{mol\cdot dm^{-3}}$，逐滴加入 $\mathrm{AgNO_3}$。已知 $K_{sp}(\mathrm{AgCl}) = 1.8 \times 10^{-10}$，$K_{sp}(\mathrm{Ag_2CrO_4}) = 1.1 \times 10^{-12}$。问：(1) 哪种沉淀先生成？(2) $\mathrm{Ag_2CrO_4}$ 开始沉淀时，$[\mathrm{Cl^-}]$ 还剩多少？

**解答**：
(1) 计算各沉淀所需最低 $[\mathrm{Ag^+}]$：
- AgCl：$[\mathrm{Ag^+}]_{\text{min}} = K_{sp}/[\mathrm{Cl^-}] = 1.8 \times 10^{-10}/0.010 = 1.8 \times 10^{-8}\ \mathrm{mol\cdot dm^{-3}}$
- Ag₂CrO₄：$[\mathrm{Ag^+}]_{\text{min}} = \sqrt{K_{sp}/[\mathrm{CrO_4^{2-}}]} = \sqrt{1.1 \times 10^{-12}/0.010} = 1.0 \times 10^{-5}\ \mathrm{mol\cdot dm^{-3}}$

$1.8 \times 10^{-8} < 1.0 \times 10^{-5}$ → **AgCl 先生成**。

(2) Ag₂CrO₄ 开始沉淀时 $[\mathrm{Ag^+}] = 1.0 \times 10^{-5}\ \mathrm{mol\cdot dm^{-3}}$：
$$[\mathrm{Cl^-}] = K_{sp}(\mathrm{AgCl})/[\mathrm{Ag^+}] = 1.8 \times 10^{-10}/1.0 \times 10^{-5} = 1.8 \times 10^{-5}\ \mathrm{mol\cdot dm^{-3}}$$

$\mathrm{Cl^-}$ 已降至 $1.8 \times 10^{-5}\ \mathrm{mol\cdot dm^{-3}} \approx 10^{-5}$，可视为"已定量沉淀"。

**反思**：虽然 $K_{sp}(\mathrm{Ag_2CrO_4})$ 数值上小于 $K_{sp}(\mathrm{AgCl})$，但 AgCl 反而先沉淀——这就是异类型不能直接比 $K_{sp}$ 的原因。Mohr 法测定 Cl⁻ 正是利用这一原理：用 CrO₄²⁻ 作指示剂，AgCl 先沉淀完全，过量一滴 Ag⁺ 即生成砖红色 Ag₂CrO₄。

---

### 例题 3：沉淀转化（⭐⭐）

**题目**：$K_{sp}(\mathrm{AgCl}) = 1.8 \times 10^{-10}$，$K_{sp}(\mathrm{AgI}) = 8.5 \times 10^{-17}$。计算反应 $\mathrm{AgCl}(s) + \mathrm{I^-} \rightleftharpoons \mathrm{AgI}(s) + \mathrm{Cl^-}$ 的平衡常数，并判断转化是否完全。

**解答**：
$$K = \frac{[\mathrm{Cl^-}]}{[\mathrm{I^-}]} = \frac{K_{sp}(\mathrm{AgCl})}{K_{sp}(\mathrm{AgI})} = \frac{1.8 \times 10^{-10}}{8.5 \times 10^{-17}} = 2.1 \times 10^6$$

$K \gg 10^6$ → 转化完全。只需微量 $\mathrm{I^-}$ 即可将 $\mathrm{AgCl}$ 转化为 $\mathrm{AgI}$。

**反思**：沉淀转化的 $K$ 值就是两个 $K_{sp}$ 的比值。$K$ 越大，转化越彻底。此原理在元素分离和定性分析中广泛应用。

### 真题入口使用建议

- 沉淀溶解平衡真题的核心考点只有三个：**溶度积计算**、**分步沉淀判断**、**沉淀转化条件**。训练时不要散讲，要按这三个核心词组织。
- **基础班**：优先练"Ksp → 溶解度互算"，先建立 Ksp 和溶解度之间的换算直觉（注意单位：AB 型和 AB₂ 型公式不同）。
- **提高班**：加入分步沉淀（哪种离子先沉淀？完全沉淀的 pH 范围？）和沉淀转化（AgCl → AgI 的平衡常数计算）。
- 真题中的沉淀溶解平衡常和酸碱平衡、配位平衡联合——"硫化物在酸中是否沉淀"是最经典的跨平衡联立题。

### 推荐真题 {#recommended-exam-questions}

> ⚠️ 当前题库暂无直接匹配的真题。推荐讲评方向：(1) 从分析-滴定真题中提取"终点指示剂选择→分步沉淀原理"子任务（Mohr 法测氯的核心逻辑）；(2) 将 [[真题-物化-平衡-001]] 中的平衡常数计算框架迁移到 Ksp 场景，重点训练 AB 型与 A₂B 型的 s-Ksp 换算差异；(3) 补充"硫化物在酸中是否沉淀"的跨平衡（Ksp + Ka）联立计算题。

### 真题链与讲评顺序 {#exam-sequence}

- `第 1 题`：**Ksp ↔ 溶解度互算**。课堂用途：热启动，确认换算公式和单位不出错。
- `第 2 题`：**分步沉淀 + pH 控制**。课堂用途：主讲题，训练"画出 pM-pH 图→判断沉淀区间"的思维。
- `第 3 题`：**沉淀转化 + 跨平衡联立**（硫化物/氢氧化物在酸中的溶解判断）。课堂用途：收束题，把沉淀平衡嵌入更大的溶液平衡体系。
- 课堂顺序建议：`Ksp计算 → 分步沉淀 → 转化/联立`。
