## 零点六、课堂投影速查卡 {#classroom-quick-card}

**本课课堂入口：** 不要从"酸碱理论有四种"开始。先问学生"Na₂CO₃ 是不是碱？"——让学生吵 30 秒，再引出 Arrhenius/Brønsted/Lewis 的定义差异。学生吵的过程就是理论扩容的动机。

**先问三个问题：**

1. 这题是在考理论归属（Arrhenius/Brønsted/Lewis/HSAB），还是定量计算（pH/缓冲/平衡）？
2. 如果是理论题——题目给的"酸/碱"，是以谁为标准定义的？换个理论它还酸/碱吗？
3. 如果是计算题——c/Ka ≥ 400 吗？不判据就代公式，最容易出 [H⁺] > c 的荒谬结果。

**一屏判断卡：**

```
酸碱题 → 先判大类
    │
    ├─ 理论判断题
    │   ├─ "质子转移" → Brønsted（Ka·Kb=Kw）
    │   ├─ "配位/空轨道" → Lewis（缺电子=酸，富电子=碱）
    │   ├─ "稳定性/溶解度" → HSAB（硬亲硬、软亲软）
    │   └─ 注意：三个理论是递进扩容，不是互相推翻！
    │
    └─ 定量计算题
        ├─ 先判 c/Ka ≥ 400？→ YES：最简式 [H⁺]=√(cKa)
        ├─                → NO：解二次方程 Ka=x²/(c−x)
        ├─ 缓冲溶液 → pH = pKa + lg([A⁻]/[HA])
        └─ 多元酸 → Ka₁≫Ka₂ 时可分步近似
```

**HSAB 快查口诀：**

| | 硬 | 边界 | 软 |
|:---|:---|:---|:---|
| **特征** | 小/高电荷/难极化 | 中间 | 大/低电荷/易极化 |
| **酸** | H⁺, Fe³⁺, Al³⁺, BF₃ | Fe²⁺, Cu²⁺, Zn²⁺ | Ag⁺, Hg²⁺, BH₃ |
| **碱** | F⁻, OH⁻, H₂O, NH₃ | Br⁻, NO₂⁻, py | I⁻, S²⁻, CN⁻, CO |
| **匹配** | 硬+硬→离子键→晶格能大 | — | 软+软→共价键→稳定常数大 |

**计算三不原则：** ① 不判 c/Ka 不代公式 ② 不算 [H⁺] 不超过 c ③ 不强酸强碱不套最简式

## 一、核心结论汇总 {#core-conclusions}

**必须记住：**
1. **酸碱理论是递进扩容关系**：Arrhenius ⊂ Brønsted ⊂ Lewis ⊂ HSAB，不是互相推翻。看到"质子转移"用 Brønsted，看到"配位键/空轨道"用 Lewis，看到"稳定性比较"用 HSAB。
2. **"硬亲硬、软亲软"的本质是键型匹配**：硬-硬 → 静电/离子键；软-软 → 共价/轨道重叠。硬软与强弱无关。
3. **计算前先判据是竞赛基本功**：$c/K_a \geq 400$ 才可用简化公式，否则可能得出 [H⁺] > c 的荒谬结果。

**最高频决策路径：**

```text
拿到题目 → 第一步：判断题目类型
            ├── 理论判断（酸碱类型/稳定性预测）→ 查「对比表格」→ 选 Brønsted/Lewis/HSAB
            └── 定量计算（pH/缓冲/平衡浓度）→ 先判 c/Ka ≥ 400？→ 选简化/精确公式
```

## 四、反应机理拆解（含检查表）{#mechanism-analysis}

> 缓冲溶液抵抗 pH 变化的本质是其内部的共轭酸碱对能够"吸收"外加的少量强酸或强碱。以下以**Henderson–Hasselbalch 方程的推导**为例，演示"质子流"拆解。

#### 步骤 1：写出弱酸的电离平衡
- **操作**：$\mathrm{HA \rightleftharpoons H^+ + A^-}$，$K_a = \dfrac{[\mathrm{H^+}][\mathrm{A^-}]}{[\mathrm{HA}]}$
- **检查表**：
  - ☐ HA 为弱酸，A⁻ 为其共轭碱
  - ☐ $K_a$ 只与温度有关

#### 步骤 2：变形取负对数
- **操作**：$[\mathrm{H^+}] = K_a \cdot \dfrac{[\mathrm{HA}]}{[\mathrm{A^-}]}$，两边取 $-\lg$
- **检查表**：
  - ☐ 对数运算正确：$\lg(a/b) = \lg a - \lg b$
  - ☐ 符号不遗漏

#### 步骤 3：得到 H–H 方程并解释缓冲机理
- **操作**：$\mathrm{pH} = \mathrm{p}K_a + \lg\dfrac{[\mathrm{A^-}]}{[\mathrm{HA}]}$
- **检查表**：
  - ☐ 外加少量强碱时，OH⁻ 与 HA 反应生成 A⁻，$[\mathrm{A^-}]/[\mathrm{HA}]$ 比值变化小 → pH 变化小
  - ☐ 外加少量强酸时，H⁺ 与 A⁻ 反应生成 HA，比值同样变化小
  - ☐ 当 $[\mathrm{A^-}] = [\mathrm{HA}]$ 时，$\mathrm{pH} = \mathrm{p}K_a$（缓冲能力最强）

## 五、典型例题串讲 {#typical-examples}

### 例题 1：HSAB 预测配合物稳定性（⭐⭐⭐）

**题目**：解释为什么 [Ag(CN)₂]⁻ 的稳定常数远大于 [Ag(NH₃)₂]⁺？为什么 FeF₃ 比 FeI₃ 更稳定？

**分析**：
- 判断 Lewis 酸类型：Ag⁺（d¹⁰，易极化）→ 软酸；Fe³⁺（电荷高、半径小）→ 硬酸
- 判断 Lewis 碱类型：CN⁻（C 给电子，易极化）→ 软碱；NH₃（N 电负性高）→ 硬碱；F⁻（半径小、难极化）→ 硬碱；I⁻（半径大、易极化）→ 软碱
- 应用 HSAB 规则预测

**解答**：
- [Ag(CN)₂]⁻：Ag⁺（软酸）+ CN⁻（软碱）→ 软-软匹配 → 共价键成分大 → 稳定性高
- [Ag(NH₃)₂]⁺：Ag⁺（软酸）+ NH₃（硬碱）→ 软-硬不匹配 → 稳定性相对较低
- FeF₃：Fe³⁺（硬酸）+ F⁻（硬碱）→ 硬-硬匹配 → 离子键成分大 → 晶格能大 → 稳定
- FeI₃：Fe³⁺（硬酸）+ I⁻（软碱）→ 硬-软不匹配 → 稳定性差（实际上 FeI₃ 极不稳定，会自发分解）

**反思**：HSAB 不是定量工具，但能解释"为什么相反电荷离子组成的化合物稳定性差异巨大"。注意同一元素不同氧化态硬软不同——Fe³⁺（硬）vs Fe²⁺（边界）。

### 例题 2：酸碱平衡综合计算（⭐⭐⭐⭐）

**题目**：某溶液含有 0.10 mol/L HAc 和 0.10 mol/L NaAc（pKa = 4.76）。
(1) 计算该缓冲溶液的 pH。
(2) 向 100 mL 该溶液中加入 1.0 mL 1.0 mol/L NaOH，计算新的 pH。
(3) 若改为向 100 mL pH = 4.76 的 HCl 溶液中加入等量 NaOH，pH 变为多少？对比说明缓冲作用。

**分析**：
- (1) 直接用 Henderson-Hasselbalch 方程，[Ac⁻]/[HAc] = 1
- (2) 加入 NaOH 后，HAc 被消耗，Ac⁻ 增加，体积略有变化
- (3) HCl 无缓冲能力，pH 由加入的 OH⁻ 直接决定

**解答**：
- (1) $\mathrm{pH} = \mathrm{p}K_a + \lg\frac{[\ce{Ac-}]}{[\ce{HAc}]} = 4.76 + \lg(1) = \mathbf{4.76}$
- (2) 加入 NaOH 物质的量 = $1.0 \times 10^{-3}\ \mathrm{L} \times 1.0\ \mathrm{mol/L} = 1.0 \times 10^{-3}\ \mathrm{mol}$
  - 反应后：$n(\ce{HAc}) = 0.010 - 0.0010 = 9.0 \times 10^{-3}\ \mathrm{mol}$
  - $n(\ce{Ac-}) = 0.010 + 0.0010 = 1.1 \times 10^{-2}\ \mathrm{mol}$
  - 体积 ≈ 101 mL（忽略体积变化时比值不变）
  - $\mathrm{pH} = 4.76 + \lg\frac{1.1 \times 10^{-2}}{9.0 \times 10^{-3}} = 4.76 + 0.09 = \mathbf{4.85}$
  - pH 变化仅 **+0.09**
- (3) HCl 溶液中：$n(\ce{H+}) = 10^{-4.76} \times 0.100 \approx 1.74 \times 10^{-5}\ \mathrm{mol}$
  - 加入 $1.0 \times 10^{-3}\ \mathrm{mol}$ OH⁻ 完全中和 H⁺ 后还剩余 $9.83 \times 10^{-4}\ \mathrm{mol}$ OH⁻
  - $[\ce{OH-}] = \frac{9.83 \times 10^{-4}}{0.101} \approx 9.7 \times 10^{-3}\ \mathrm{mol/L}$
  - $\mathrm{pOH} = 2.01$，$\mathrm{pH} = 14 - 2.01 = \mathbf{11.99}$
  - pH 变化从 4.76 → 11.99，变化了 **+7.23**

**反思**：缓冲溶液抵抗少量酸碱的能力来自共轭酸碱对的"储备军"——HAc 和 Ac⁻ 大量共存，少量外加酸碱只改变比值而非耗尽一方。定量感受：缓冲体系 pH 变化 < 0.1，非缓冲体系 pH 变化 > 7。

### 例题 3：拉平效应与区分效应（⭐⭐⭐）

**题目**：在液氨溶剂中，HAc 是强酸；在水中，HAc 是弱酸。解释这一现象，并说明这是拉平效应还是区分效应的体现。

**分析**：
- 比较 H₂O 与 NH₃ 的碱性（接受质子能力）
- NH₃ 接受质子能力强于 H₂O → 对质子的"拉力"更大
- HAc 在 NH₃ 中几乎完全电离 → 被拉平到 NH₄⁺ 水平

**解答**：
- NH₃ 的碱性比 H₂O 强（NH₃ 的 pKb ≈ 4.75，H₂O 的 pKb ≈ 15.7）
- 在液氨中，NH₃ 强烈接受 HAc 的质子：$\ce{HAc + NH3 -> NH4+ + Ac-}$
- 由于 NH₃ 对质子的亲和力远大于 H₂O，HAc 在液氨中几乎完全电离，表现为强酸
- 这是**拉平效应**的体现：液氨将 HAc "拉平"到 NH₄⁺ 的酸性水平
- 对比：在水中，H₂O 接受质子能力较弱，HAc 只部分电离（Ka = 1.8×10⁻⁵），表现为弱酸——水对 HAc 具有**区分效应**

**反思**：溶剂的碱性决定其"拉平能力"——碱性越强的溶剂，对酸的拉平效应越显著。这也是超酸只能在极低碱性溶剂中表现的原因。

### 例题 4：一元弱酸 pH 计算——假设-验证法（质心 L5，⭐⭐⭐）

**题目：**
1.00 mol/L HF 溶液，Ka = 7.2×10⁻⁴，求 pH。

**分析：**
弱酸不完全电离，需建立 PBE/CBE/MBE 方程，再判断近似条件：cKa > 10Kw（可忽略水电离）、c/Ka > 400（可认为 [HA] ≈ c₀），最后验证假设的 "5% 误差原则"。

**解答：**
- 判据检查：cKa = 7.2×10⁻⁴ > 10Kw ✓；c/Ka = 1.39×10³ > 400 ✓
- 最简式：[H⁺] = √(cKa) = √(7.2×10⁻⁴) = 2.68×10⁻² M
- pH = −lg(2.68×10⁻²) = **1.57**
- 验证：电离度 α = [H⁺]/c = 2.68% < 5% ✓

**反思：**
- **条件判断优先于套公式**——这是分析化学的核心思想
- 若 c/Ka < 400，最简式失效，需解二次方程：Ka = x²/(c−x)
- HF 的 Ka 数量级（10⁻⁴）是常见酸的"边界案例"，适合训练近似判断

### 例题 5：H₂S 二元弱酸分步近似（质心 L6，⭐⭐⭐⭐）

**题目：**
0.10 M H₂S 溶液，Ka₁ = 8.9×10⁻⁸，Ka₂ = 1.2×10⁻¹³，求 pH 和 [S²⁻]。

**分析：**
二元弱酸分步近似：Ka₁ >> Ka₂（相差约 10⁵ 倍），忽略二级电离；先当一元弱酸处理求 [H⁺]，再利用总平衡 H₂S ⇌ 2H⁺ + S²⁻，K = Ka₁Ka₂ 求 [S²⁻]。

**解答：**
1. pH 计算：
   - cKa₁ = 8.9×10⁻⁹ > 10Kw ✓；c/Ka₁ = 1.1×10⁶ > 400 ✓
   - [H⁺] = √(cKa₁) = √(8.9×10⁻⁹) = 9.4×10⁻⁵ M
   - **pH = 4.03**

2. [S²⁻] 计算：
   - 总平衡：H₂S ⇌ 2H⁺ + S²⁻，K = Ka₁Ka₂ = 1.07×10⁻²⁰
   - [S²⁻] = Ka₁Ka₂[H₂S]/[H⁺]² ≈ (1.07×10⁻²⁰ × 0.10)/(9.4×10⁻⁵)² ≈ **1.2×10⁻¹³ M**

**反思：**
- [S²⁻] 受 pH 强烈控制，是后续硫化物沉淀平衡的基础
- 分步拆解思想：先算 [H⁺]（一级主导），再算 [S²⁻]（用总平衡耦合）
- Ka₁ 和 Ka₂ 相差 10⁵ 倍以上时，二级电离对 [H⁺] 的贡献可忽略

### 例题 6：Na₂S 二元弱碱——最简公式失效反例（质心 L6，⭐⭐⭐⭐）

**题目：**
0.10 M Na₂S 溶液，H₂S 的 Ka₁ = 8.9×10⁻⁸，Ka₂ = 1.2×10⁻¹³，求 pH。

**分析：**
二元弱碱 Kb₁ = Kw/Ka₂ = 8.3×10⁻²，Kb₂ = Kw/Ka₁ = 1.1×10⁻⁷。判断条件：c/Kb₁ = 1.2 << 400，最简公式失效，需列 PBE 或解二次方程。

**解答：**
- Kb₁ = 10⁻¹⁴/(1.2×10⁻¹³) = 8.3×10⁻²
- c/Kb₁ = 0.10/0.083 ≈ 1.2 < 400 ❌（最简式 [OH⁻] = √(cKb₁) 失效）
- 列方程：Kb₁ = x²/(c−x) = 0.083 = x²/(0.10−x)
- 解得 x = [OH⁻] ≈ 5.9×10⁻² M
- pOH = 1.23，**pH ≈ 12.77**

**反思：**
- **反例驱动**：让学生发现"不是什么时候都能套最简式"
- 条件判断的优先级高于公式选择：c/Kb > 400 是最简式的生命线
- 此题若误用最简式：[OH⁻] = √(0.10×0.083) = 9.1×10⁻²，pH = 12.96，误差约 0.2 pH 单位——在精确计算中不可接受

### 真题入口使用建议

- 酸碱理论的真题分两个层次：**概念判断**（Brønsted vs Lewis vs HSAB）和**定量计算**（pH/分布系数/缓冲溶液）。课前先判断今天的主线是哪条。
- **基础班**：优先练"共轭酸碱对识别 + 强酸强碱 pH + 缓冲溶液 Henderson-Hasselbalch"三连，建立"酸碱=质子转移"的核心框架。
- **提高班**：加入 Lewis 酸碱与软硬酸碱（HSAB）判断——这是竞赛区分度所在，尤其是"软亲软、硬亲硬"在配合物稳定性预测中的应用。
- 酸碱真题常与沉淀溶解平衡、配位平衡联合出题——讲评时要有意识地串联。

### 真题链与讲评顺序 {#exam-sequence}

- `第 1 题`：**共轭酸碱对 + 强酸强碱 pH**。课堂用途：热启动，确认基础框架还在。
- `第 2 题`：**弱酸弱碱 pH + 缓冲溶液计算**。课堂用途：主讲题，重点训练"近似条件的判断"（cKa > 10Kw? c/Ka > 400?）。
- `第 3 题`：**Lewis 酸碱 / HSAB 判断**。课堂用途：收束题，把酸碱概念从"质子"扩展到"电子对"，链接配位化学。
- 课堂顺序建议：`Brønsted计算 → 缓冲溶液 → Lewis/HSAB`，先算清楚，再扩概念。
