## 零点六、课堂投影速查卡 {#classroom-quick-card}

**本页课堂入口：** 先抓“颜色-价态-介质”三联动，不要一上来就背散乱反应。

**先问四个问题：**

1. 题目在问 `MnO4-`、`Cr(VI)`、`V` 的颜色序列，还是 `Ti(III)/Ti(IV)` 稳定性？
2. 当前是否已经明确酸性、中性、强碱性？
3. 这是价态转化题、颜色推断题，还是含氧酸根结构题？
4. 这里最该先判的是“产物是谁”，还是“颜色对应哪一级价态”？

**一屏判断卡：**

- `MnO4-` 先判介质，再判产物，最后才配平。
- 铬先分黄铬酸根和橙重铬酸根，再谈氧化性强弱。
- 钒题优先抓 `黄→蓝→绿→紫` 的还原序列。
- 课堂上先做 1 道颜色快判，再做 1 道介质控制方程式。

## 一、核心结论汇总 {#core-conclusions}

**必须记住：**

1. **Ti(IV) 稳定，Ti(III) 是强还原剂**：Ti 的最高氧化态 +4 最稳定（d⁰，无色），Ti³⁺（d¹，紫色）在空气中易被氧化。
2. **Cr(VI) 与 Cr(III) 的互化受 pH 控制**：酸性介质中 Cr₂O₇²⁻（橙色）是强氧化剂（E°=1.36 V）；碱性介质中 Cr(III) 可被氧化为 Cr(VI)。CrO₄²⁻（黄色）与 Cr₂O₇²⁻（橙色）的转化取决于 pH。
3. **MnO₄⁻ 的还原产物严格依赖介质**：酸性 → Mn²⁺（近无色，5e⁻）；中性/弱碱性 → MnO₂↓（棕黑，3e⁻）；强碱性 → MnO₄²⁻（绿色，1e⁻）。这是竞赛最高频考点。
4. **V 的价态颜色序列是推断指纹**：VO₂⁺（黄，+5）→ VO²⁺（蓝，+4）→ V³⁺（绿，+3）→ V²⁺（紫，+2），用 Zn 还原可依次观察。
5. **第一过渡系左部（Ti~Mn）高氧化态稳定，且含氧酸酸性递增**：Ti(OH)₄（两性偏碱）< H₃VO₄（两性偏酸）< H₂CrO₄（酸性）< HMnO₄（强酸）。

**最高频决策路径：**

```mermaid
flowchart TD
  A[题目给出颜色/价态变化] --> B{是否涉及 MnO₄⁻?}
  B -->|是| C[先判断介质酸碱性 → 确定还原产物 → 查[[锰]]]
  B -->|否| D{是否涉及 Cr 的橙/黄/绿色变化?}
  D -->|是| E[判断 pH → 确定 Cr(VI) 形态或 Cr(III)/Cr(VI) 互化 → 查[[铬]]]
  D -->|否| F{是否出现紫→蓝→绿→黄颜色序列?}
  F -->|是| G[推断为 V 的价态还原 → 查[[钒]]]
  F -->|否| H[检查 Ti(IV)/Ti(III) 或配合物生成 → 查[[钛]]或[[过渡金属通性]]]
```

### 表 2-5：V/Cr/Mn 含氧酸根对比（结构与性质）

| 触发条件（题目关键词） | 元素 | 含氧酸根 | 中心原子氧化态 | 结构特征 | 酸碱性 | 氧化性 | 常见陷阱 |
|:---|:---:|:---|:---:|:---|:---:|:---:|:---|
| "正四面体""黄色" | Cr | CrO₄²⁻ | +6 | 正四面体 | 碱性条件下稳定 | 弱（碱性） | 与 SO₄²⁻ 等电子体，结构相似 |
| "两个四面体共顶点""橙色" | Cr | Cr₂O₇²⁻ | +6 | 两个 CrO₄ 共用一个 O | 酸性条件下稳定 | **强**（酸性 E°=1.36 V） | pH 驱动平衡：2CrO₄²⁻ + 2H⁺ ⇌ Cr₂O₇²⁻ + H₂O |
| "正四面体""紫色" | Mn | MnO₄⁻ | +7 | 正四面体 | 强酸性（HMnO₄） | **最强**（E°=1.51 V） | 还原产物严格依赖介质 |
| "正四面体""绿色" | Mn | MnO₄²⁻ | +6 | 正四面体 | 强碱性条件下稳定 | 可歧化 | 仅在强碱中稳定，遇酸分解 |
| "四面体/链状""黄色" | V | VO₄³⁻（正钒酸根） | +5 | 正四面体 | 碱性 | 弱 | 酸化后缩合为 V₂O₇⁴⁻、V₃O₉³⁻ 等 |
| "VO₂⁺""黄色" | V | VO₂⁺（钒酰/ dioxovanadium） | +5 | 直线形 O=V=O | 酸性 | 弱氧化性 | 不要与 VO²⁺（V⁴⁺，蓝色）混淆 |

> **核心规律**：V/Cr/Mn 的 +5/+6/+7 含氧酸根均为四面体构型；酸性条件下缩合/聚合趋势：V > Cr > Mn。

### 表 2-6：氧化还原条件控制表（Cr/Mn 价态转化）

| 触发条件（题目关键词） | 转化方向 | 所需条件 | 典型试剂 | 反应示例 | 常见陷阱 |
|:---|:---|:---|:---|:---|:---|
| "酸性""橙色变绿" | Cr(VI) → Cr(III) | 酸性 + 还原剂 | FeSO₄、SO₂、乙醇 | Cr₂O₇²⁻ + 6Fe²⁺ + 14H⁺ → 2Cr³⁺ + 6Fe³⁺ + 7H₂O | Cr(VI) 酸性氧化性强，碱性弱 |
| "碱性""绿色变黄" | Cr(III) → Cr(VI) | 碱性 + 氧化剂 | H₂O₂、NaClO、Br₂ | 2[Cr(OH)₄]⁻ + 3H₂O₂ + 2OH⁻ → 2CrO₄²⁻ + 8H₂O | 必须碱性，酸性条件下 Cr(VI) 会氧化还原剂 |
| "酸性""紫色褪去" | Mn(VII) → Mn(II) | 酸性 + 还原剂 | Fe²⁺、C₂O₄²⁻、H₂O₂ | MnO₄⁻ + 5Fe²⁺ + 8H⁺ → Mn²⁺ + 5Fe³⁺ + 4H₂O | 5e⁻ 转移，电子数最多 |
| "中性/弱碱性""紫色→棕黑" | Mn(VII) → Mn(IV) | 中性/弱碱性 + 还原剂 | SO₃²⁻、S²⁻ | 2MnO₄⁻ + 3SO₃²⁻ + H₂O → 2MnO₂↓ + 3SO₄²⁻ + 2OH⁻ | 3e⁻ 转移，生成沉淀 |
| "强碱性""紫色→绿色" | Mn(VII) → Mn(VI) | 强碱性 + 还原剂 | SO₃²⁻、OH⁻/加热 | 2MnO₄⁻ + SO₃²⁻ + 2OH⁻ → 2MnO₄²⁻ + SO₄²⁻ + H₂O | 1e⁻ 转移，绿色锰酸根 |
| "强碱性""加热""歧化" | Mn(VI) → Mn(VII)+Mn(IV) | 酸性/中性条件下自发 | 加酸或放置 | 3MnO₄²⁻ + 4H⁺ → 2MnO₄⁻ + MnO₂↓ + 2H₂O | MnO₄²⁻ 仅在强碱中稳定 |

> **核心规律**：Cr(VI)/Cr(III) 互化受 pH 控制（酸性还原，碱性氧化）；MnO₄⁻ 还原产物受 pH 控制（酸性→Mn²⁺，中性→MnO₂，碱性→MnO₄²⁻）。

## 三、解题套路 / 决策流程 {#problem-solving-routine}

### 套路一：过渡金属元素推断（颜色+价态法）

```mermaid
flowchart TD
  A[观察到特征颜色/现象] --> B{紫色溶液?}
  B -->|强氧化剂，可褪色| C[可能是 MnO₄⁻ 或 Ti³⁺]
  B -->|加入还原剂后变色| D[判断：若还原后无色→MnO₄⁻→Mn²⁺; 若氧化后无色→Ti³⁺→Ti⁴⁺]
  C --> E{是否有沉淀生成?}
  E -->|棕黑色沉淀| F[MnO₄⁻ 中性还原 → MnO₂，查[[锰]]]
  E -->|无沉淀，无色| G[MnO₄⁻ 酸性还原 → Mn²⁺ 或 Ti³⁺ 氧化 → Ti⁴⁺]
  A --> H{橙色/黄色溶液?}
  H -->|橙色，强氧化性| I[Cr₂O₇²⁻ 酸性，查[[铬]]]
  H -->|黄色，弱氧化性| J[CrO₄²⁻ 碱性，查[[铬]]]
  H -->|黄→蓝→绿→紫变化| K[V 的价态还原序列，查[[钒]]]
  A --> L{绿色溶液+两性氢氧化物?}
  L -->|是| M[Cr³⁺ 或 V³⁺；Cr³⁺ 氢氧化物灰绿色，V³⁺ 直接绿色]
```

| 1 | **先看介质酸碱性**：找题目中 "H⁺"、"酸性"、"NaOH"、"碱性"、"中性" 等关键词 | [[锰]] | ☐ 介质已明确；☐ 无介质描述时考虑反应物是否自带酸碱 |

### 套路三：Cr(VI)/Cr(III) 价态转化判断

| 步骤 | 核心操作 | 依据 KP | 检查清单 |
|:---|:---|:---|:---|
| 1 | **判断当前 Cr 价态和介质**：Cr₂O₇²⁻（橙，酸性）或 CrO₄²⁻（黄，碱性）或 Cr³⁺（绿） | [[铬]] | ☐ 颜色-价态-介质三者对应正确 |

| 3 | **写出关键反应**：Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O；或 2Cr(OH)₄⁻ + 3H₂O₂ + 2OH⁻ → 2CrO₄²⁻ + 8H₂O | [[铬]] | ☐ 电子转移数正确（Cr +6→+3 得 3e⁻/Cr）；☐ 碱性氧化用 H₂O₂ 常见 |

## 四、反应机理拆解（含检查表）{#mechanism-analysis}

> 以下以 **MnO₄⁻ 在不同介质中的还原产物差异** 为例，演示过渡金属氧化还原反应中"介质 → 电子数 → 产物"的定量推理链。

#### 步骤 1：判断介质酸碱性
- **操作**：从题目条件中提取 H⁺/OH⁻/H₂O 信息，确定反应环境
- **检查表**：
  - ☐ 明确标注 "酸性" / "中性/弱碱性" / "强碱性"
  - ☐ 无明确标注时，检查反应物是否自带酸碱（如 H₂SO₄、Na₂CO₃）

#### 步骤 2：确定电子转移数与还原产物
- **操作**：根据介质选择标准电极电势，判断热力学可行的产物
  - 酸性：MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O（E° = +1.51 V，最强氧化性）
  - 中性/弱碱性：MnO₄⁻ + 2H₂O + 3e⁻ → MnO₂↓ + 4OH⁻（E° = +0.59 V）
  - 强碱性：MnO₄⁻ + e⁻ → MnO₄²⁻（E° = +0.56 V，仅 1e⁻ 转移）
- **检查表**：
  - ☐ 酸性条件电子转移最多（5e⁻），产物价态最低（Mn²⁺）
  - ☐ 碱性条件电子转移最少（1e⁻），产物价态最高（MnO₄²⁻）
  - ☐ 中性介于两者之间（3e⁻ → MnO₂）

#### 步骤 3：用半反应法配平并验证
- **操作**：分别写出氧化半反应和还原半反应，按电子守恒合并
- **检查表**：
  - ☐ Mn 原子守恒
  - ☐ O 原子用 H₂O 平衡，H 原子用 H⁺/OH⁻ 平衡
  - ☐ 电荷守恒（左右总电荷相等）
  - ☐ 沉淀符号（MnO₂↓）和气体符号标注完整

> **核心规律**：同一氧化剂，酸性越强 → 得电子能力越强 → 还原产物价态越低。这是 Latimer/Frost 图在解题中的直接应用。

## 五、典型例题串讲 {#typical-examples}

### 例题 1：Cr/Mn 价态转化推断题

**题目：**
某金属 M 的化合物 A 为暗红色固体，溶于水得橙色溶液 B。B 具有强氧化性，能氧化乙醇（司机酒驾检测原理）。向 B 中加入还原剂 FeSO₄，溶液由橙色变为绿色 C。向 C 中加入 NaOH 得灰绿色沉淀 D，D 溶于过量 NaOH 得绿色溶液 E。将 E 用 H₂O₂ 氧化，溶液又变为黄色 F。酸化 F 后重新得到橙色溶液。

(1) 推断 M 为何种元素，写出 A~F 的化学式。
(2) 写出 B 氧化乙醇的离子方程式。
(3) 写出 C → D → E → F 的各步反应方程式。

**分析：**
- "暗红色固体，溶于水得橙色溶液" → CrO₃（暗红色）溶于水形成 H₂CrO₄/H₂Cr₂O₇（橙色），或直接用 K₂Cr₂O₇（橙红色晶体）。
- "强氧化性，氧化乙醇" → 重铬酸根的经典应用（酒驾检测：橙色→绿色）。
- "橙色→绿色" → Cr₂O₇²⁻（橙）被还原为 Cr³⁺（绿）。
- "灰绿色沉淀，溶于过量 NaOH" → Cr(OH)₃ 的两性特征。
- "绿色溶液氧化后变黄" → [Cr(OH)₄]⁻ 碱性被氧化为 CrO₄²⁻（黄）。
- "黄色酸化后变橙" → CrO₄²⁻ + H⁺ → Cr₂O₇²⁻。

**解答：**

(1) **M = Cr（铬）**
- A：CrO₃（或 K₂Cr₂O₇）
- B：Cr₂O₇²⁻（酸性溶液中）
- C：Cr³⁺（或 Cr₂(SO₄)₃）
- D：Cr(OH)₃
- E：[Cr(OH)₄]⁻（或 CrO₂⁻）
- F：CrO₄²⁻

(2) **B 氧化乙醇**：
$$\ce{2Cr2O7^{2-} + 3C2H5OH + 16H+ -> 4Cr^{3+} + 3CH3COOH + 11H2O}$$
（或氧化为乙醛：$\ce{Cr2O7^{2-} + 3C2H5OH + 8H+ -> 2Cr^{3+} + 3CH3CHO + 7H2O}$）

(3) **各步反应**：
- C → D：$\ce{Cr^{3+} + 3OH- -> Cr(OH)3 v}$（灰绿色）
- D → E：$\ce{Cr(OH)3 + OH- -> [Cr(OH)4]-}$（绿色）
- E → F：$\ce{2[Cr(OH)4]- + 3H2O2 + 2OH- -> 2CrO4^{2-} + 8H2O}$（黄色）

**反思：**
- 本题覆盖了 Cr(VI)/Cr(III) 的完整循环：酸性还原（橙→绿）→ 沉淀 → 溶解（两性）→ 碱性氧化（绿→黄）→ 酸化（黄→橙）。
- **关键记忆点**：Cr(OH)₃ 的两性和 Cr(VI) 的 pH 依赖性是推断 Cr 的"指纹"。

### 例题 2：MnO₄⁻ 在不同介质下的产物判断

**题目：**
用 KMnO₄ 溶液分别与下列物质反应，写出完整的离子方程式并说明介质条件：

(1) 酸性条件下与 Fe²⁺ 反应；
(2) 中性条件下与 SO₃²⁻ 反应；
(3) 强碱性条件下与 SO₃²⁻ 反应；
(4) 加热条件下与浓 HCl 反应。

**分析：**
- 核心考点：MnO₄⁻ 的还原产物与介质严格相关。
- (1) 酸性 → Mn²⁺（无色），5e⁻ 转移；
- (2) 中性 → MnO₂（棕黑沉淀），3e⁻ 转移；
- (3) 强碱性 → MnO₄²⁻（绿色），1e⁻ 转移；
- (4) 浓 HCl 加热 → Mn²⁺ + Cl₂↑（浓 HCl 提供 H⁺ 和 Cl⁻，Cl⁻ 被氧化为 Cl₂）。

**解答：**

(1) **酸性介质，MnO₄⁻ → Mn²⁺**：
$$\ce{MnO4^- + 5Fe^{2+} + 8H+ -> Mn^{2+} + 5Fe^{3+} + 4H2O}$$
- 现象：紫色褪去，溶液变为浅黄色（Fe³⁺）/近无色（稀溶液）。

(2) **中性介质，MnO₄⁻ → MnO₂↓**：
$$\ce{2MnO4^- + 3SO3^{2-} + H2O -> 2MnO2 v + 3SO4^{2-} + 2OH-}$$
- 现象：紫色褪去，生成棕黑色沉淀。

(3) **强碱性介质，MnO₄⁻ → MnO₄²⁻**：
$$\ce{2MnO4^- + SO3^{2-} + 2OH- -> 2MnO4^{2-} + SO4^{2-} + H2O}$$
- 现象：紫色变为绿色（锰酸根）。

(4) **浓 HCl 加热，MnO₄⁻ 氧化 Cl⁻**：
$$\ce{2MnO4^- + 16H+ + 10Cl- -> 2Mn^{2+} + 5Cl2 ^ + 8H2O}$$
- 或写成与 MnO₂ 的反应（实验室制 Cl₂ 更常见）：
$$\ce{MnO2 + 4HCl(浓) ->[\Delta] MnCl2 + Cl2 ^ + 2H2O}$$
- 现象：紫色褪去，产生黄绿色刺激性气体（Cl₂）。

**反思：**
- **最高频错误**：不区分介质，一律写成 MnO₄⁻ → Mn²⁺。
- **记忆口诀**："酸五（5e⁻）中三（3e⁻）碱一（1e⁻），颜色对应无棕绿"。
- 当题目未明确介质时，检查反应物是否自带酸碱性（如 H₂SO₄、NaOH、CO₂ 溶于水呈弱酸性等）。

### 例题 3：Ti/V 推断与含氧酸根综合

**题目：**
某过渡金属 M 的化合物 A 为白色粉末，遇水产生大量白色烟雾。A 在高温下与 O₂ 反应生成白色固体 B，B 不溶于水和稀酸，但溶于浓热 H₂SO₄。金属 M 的另一种化合物 C 为暗红色固体，是工业上重要的催化剂。将 C 溶于酸后，用 Zn 还原，溶液颜色依次经历黄 → 蓝 → 绿 → 紫的变化。

(1) 推断 M 为何种元素，写出 A、B、C 的化学式。
(2) 写出 A 遇水反应的化学方程式。
(3) 写出 C 在酸中被 Zn 还原时，各颜色对应的离子方程式。
(4) 比较 M 与同周期的 Cr、Mn 的最高氧化态含氧酸酸性强弱。

**分析：**
1. "白色粉末，遇水产生大量白色烟雾" → TiCl₄ 的经典现象（水解生成 TiO₂ 和 HCl 气溶胶）
2. "白色固体 B，不溶于水和稀酸，溶于浓热 H₂SO₄" → TiO₂（金红石型）
3. "暗红色固体，工业催化剂" → V₂O₅（接触法制硫酸的催化剂）
4. "黄→蓝→绿→紫颜色变化" → V 的 +5→+4→+3→+2 价态还原序列
5. 综合判断 M = V（钒），但 A 是 TiCl₄ 说明 M 也可能是 Ti...重新审题：A 和 C 是"某过渡金属 M 的化合物"，但 A 描述的是 Ti，C 描述的是 V。这里题目应理解为"两种不同过渡金属"，或题目描述有歧义。
6. 修正理解：题目描述的是两种不同金属。A/B 对应 Ti，C 对应 V。或者更合理的理解：A = TiCl₄，B = TiO₂，而 C 是另一种金属的化合物。
7. 最合理的竞赛题型：A/B 为 Ti 的化合物，C 的颜色变化为 V 的特征。题目可能要求分别推断两种金属。

**解答（修正理解后）：**

**(1) 推断：**
- 由 A 和 B 的特征 → **M₁ = Ti（钛）**
  - A = TiCl₄（四氯化钛）
  - B = TiO₂（二氧化钛，金红石型）
- 由 C 的颜色变化 → **M₂ = V（钒）**
  - C = V₂O₅（五氧化二钒）

**(2) TiCl₄ 遇水反应：**
$$
\ce{TiCl4 + 2H2O -> TiO2 v + 4HCl}
$$
（产生白色烟雾：TiO₂·nH₂O 胶粒和 HCl 气体）

**(3) V₂O₅ 在酸中被 Zn 还原的颜色序列：**

| 颜色 | 离子 | 反应 | 价态 |
|:---:|:---|:---|:---:|
| 黄色 | VO₂⁺ | V₂O₅ + 2H⁺ → 2VO₂⁺ + H₂O | +5 |
| 蓝色 | VO²⁺ | 2VO₂⁺ + Zn + 4H⁺ → 2VO²⁺ + Zn²⁺ + 2H₂O | +4 |
| 绿色 | V³⁺ | 2VO²⁺ + Zn + 4H⁺ → 2V³⁺ + Zn²⁺ + 2H₂O | +3 |
| 紫色 | V²⁺ | 2V³⁺ + Zn → 2V²⁺ + Zn²⁺ | +2 |

简化总反应：
- VO₂⁺ → VO²⁺：$\ce{2VO2^+ + Zn + 4H+ -> 2VO^{2+} + Zn^{2+} + 2H2O}$
- VO²⁺ → V³⁺：$\ce{2VO^{2+} + Zn + 4H+ -> 2V^{3+} + Zn^{2+} + 2H2O}$
- V³⁺ → V²⁺：$\ce{2V^{3+} + Zn -> 2V^{2+} + Zn^{2+}}$

**(4) Ti/V/Cr/Mn 最高氧化态含氧酸酸性递变：**
$$
\ce{H4TiO4（两性偏碱）< H3VO4（两性偏酸）< H2CrO4（酸性）< HMnO4（强酸）}
$$
- 原因：同一周期从左到右，中心原子电荷增加、半径减小，对 O 的极化能力增强，O—H 键更易断裂，酸性增强。

**反思：**
- TiCl₄ 水解是推断 Ti 的"指纹"现象：白色烟雾 = TiO₂ 胶体 + HCl
- V 的颜色序列（黄→蓝→绿→紫）是竞赛推断题的经典信号，必须熟记
- 同周期过渡金属含氧酸酸性递变规律与主族类似，但 Ti 的两性偏碱需要特别注意
- 本题综合考查了 Ti 的水解性质和 V 的多价态颜色变化，是第二轮过渡金属推断的典型难度
