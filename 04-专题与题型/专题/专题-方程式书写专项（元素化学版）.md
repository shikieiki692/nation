## 零点六、课堂投影速查卡 {#classroom-quick-card}

**本课课堂入口：** 不要从"离子-电子法五步"开始。让全班同时做一道题——"写出酸性 KMnO₄ 与 Na₂SO₃ 反应的离子方程式"——30 秒后统计：多少人写出了 Mn²⁺？多少人写了 MnO₂？多少人忘了介质决定产物？从学生的混乱中引出本课第一铁律：**介质优先——先判介质，再定产物，最后配平。**

**先问三个问题：**

1. 介质是什么？— 酸性用 H⁺/H₂O 配平，碱性用 OH⁻/H₂O，产物也完全不同
2. 变价元素氧化态变了多少？— 多原子物种（Cr₂O₇²⁻、As₂S₃）必须乘原子个数！
3. 配平后检查了吗？— 碱性不能有游离 H⁺，酸性不能有游离 OH⁻

**一屏判断卡：**

```
方程式书写 → 先判介质！再判类型！
    │
    ├─ 氧化还原反应 → 离子-电子法（半反应法）
    │   ├─ 1. 拆分氧化半反应 + 还原半反应
    │   ├─ 2. 配平非H/O元素
    │   ├─ 3. O用H₂O配；H用H⁺(酸性)或OH⁻(碱性)配
    │   ├─ 4. 加e⁻配电荷 → 乘系数使e⁻相等
    │   ├─ 5. 合并化简
    │   └─ ⚠️ 碱性快捷法：先按酸性配，再两侧加OH⁻中和H⁺
    │
    ├─ 非氧化还原 → 观察法/守恒法
    │   ├─ 水解反应、沉淀反应、酸碱中和、配位反应
    │   └─ 从最复杂物质入手配平
    │
    └─ 最终验证（每步必做！）
        ├─ 原子守恒？电荷守恒？电子得失相等？
        ├─ 介质一致？碱性无游离H⁺？酸性无游离OH⁻？
        └─ 系数最简整数比？
```

**介质-产物对照（KMnO₄ 还原）：**

| 介质 | KMnO₄ 还原产物 | Mn 价态变化 | 得电子数 |
|:---|:---|:---:|:---:|
| **酸性** | Mn²⁺（近无色） | +7→+2 | 5e⁻ |
| **中性/弱碱** | MnO₂↓（棕色） | +7→+4 | 3e⁻ |
| **强碱性** | MnO₄²⁻（绿色） | +7→+6 | 1e⁻ |

**五大高频配平错误：**

| 错误 | 正确做法 |
|:---|:---|
| 多原子物种只算1个原子（Cr₂O₇²⁻→只算1个Cr得3e⁻） | **算总数**：2Cr×3e⁻=6e⁻ |
| 碱性条件出现游离 H⁺ | 用 OH⁻ 配 H，产物为 H₂O |
| 离子方程式中弱电解质拆成离子 | HAc、H₂O、NH₃·H₂O 保留化学式 |
| 忘记删除 spectator ions | Na⁺、K⁺、NO₃⁻ 等不参与反应的删去 |
| 歧化反应配平：同物质在两侧出现 | 分别写氧化/还原半反应，按电子守恒合并 |

**离子-电子法口诀：** "介质先判，产物后定；O靠水补，H靠介质；电荷加e，电子等量；合并化简，守恒验证。"

**课堂三问：** ① 介质判了吗？② 多原子物种电子数乘了吗？③ 碱性没有H⁺、酸性没有OH⁻？

## 一、核心结论汇总 {#core-conclusions}

**必须记住：**
1. **介质优先原则**：元素化学方程式书写的首要步骤是判断反应介质（酸性 / 碱性 / 中性）。同一氧化剂在不同介质中还原产物可能完全不同——必须在动笔前先定介质，再定产物，最后配平。
2. **配平四步闭环**：先判介质 → 再定产物 → 配平原子（O 用 H₂O，H 用 H⁺/OH⁻）→ 配电荷（加 e⁻）。每一步都必须同时检查原子守恒和电荷守恒。
3. **非氧化还原与氧化还原的配平策略不同**：非氧化还原反应（水解、沉淀、酸碱中和）直接用观察法或守恒法；氧化还原反应必须追踪氧化态变化，优先使用离子-电子法（半反应法）。
4. **碱性条件绝不允许出现游离 H⁺，酸性条件绝不允许出现游离 OH⁻**。这是竞赛中方程式书写最常见的致命错误。
5. **含多原子变价的物种必须整体计算电子转移**：如 Cr₂O₇²⁻ 含 2 个 Cr，As₂S₃ 含 2 个 As + 3 个 S——漏乘原子数会导致配平结果全错。

**最高频决策路径：**

```mermaid
flowchart TD
  A[题目要求书写方程式] --> B{是否氧化还原反应?}
  B -->|是| C[判断介质: 酸性/碱性/中性<br/>查[[氧化还原反应方程式配平]]]
  C --> D[确定产物: 查元素特征氧化态与介质对应关系<br/>查[[元素化学]]]
  D --> E[离子-电子法配平半反应]
  E --> F[合并半反应，检查原子+电荷守恒]
  B -->|否| G[非氧化还原: 水解/沉淀/酸碱<br/>查[[方程式书写]]]
  G --> H[观察法配平，检查原子守恒]
  F --> I[最终检查: 介质一致? 无H⁺/OH⁻冲突?]
  H --> I
```

## 三、解题套路 / 决策流程 {#problem-solving-routine}

### Step 1：审题——提取反应三要素
- **操作**：圈出反应物、生成物（或根据现象推断）、介质条件（酸性/碱性/中性/未注明）、是否有特殊条件（加热、催化剂、光照）。
- **依据 KP**：[[方程式书写]]（书写原则与条件标注）
- **检查点**：☐ 介质已确认 ☐ 所有变价元素已标出氧化态 ☐ 物质状态已标注（s/l/g/aq）

### Step 2：判断反应类型——氧化还原 vs 非氧化还原
- **操作**：计算各元素氧化态，若存在氧化态变化则为氧化还原反应；否则为非氧化还原反应（水解、沉淀、酸碱中和、配合物生成等）。
- **依据 KP**：[[氧化态]]（氧化态判定规则）
- **检查点**：☐ 反应类型已判断 ☐ 氧化剂/还原剂已识别（氧化还原时）

### Step 3A：氧化还原反应——离子-电子法配平
- **操作**：
  1. 拆分氧化半反应和还原半反应
  2. 配平除 H、O 外的元素
  3. O 用 H₂O 配平；H 用 H⁺（酸性）或 OH⁻（碱性）配平
  4. 用电荷平衡加 e⁻
  5. 两个半反应乘以适当系数使电子数相等
  6. 相加化简，删去重复项
- **碱性介质快捷处理**：先按酸性配平，再在两侧同时加 OH⁻ 中和 H⁺，合并 H⁺ + OH⁻ → H₂O
- **依据 KP**：[[氧化还原反应方程式配平]]（离子-电子法详细步骤）
- **检查点**：☐ 原子守恒 ☐ 电荷守恒 ☐ 介质一致（碱性无游离 H⁺，酸性无游离 OH⁻） ☐ 电子得失相等

### Step 3B：非氧化还原反应——观察法配平
- **操作**：从复杂物质或原子数最少的元素入手，逐步调整系数；利用守恒关系（质量守恒、电荷守恒）验证。
- **依据 KP**：[[方程式书写]]（观察法与最小公倍数法）
- **检查点**：☐ 原子守恒 ☐ 电荷守恒（离子反应时）

### Step 4：离子方程式化简（如要求）
- **操作**：将强酸、强碱、可溶性盐拆为离子；删去 spectator ions；化简系数为最简整数比。
- **依据 KP**：[[离子方程式]]（可拆 vs 不可拆规则）
- **检查点**：☐ 可拆物质正确拆分 ☐ spectator ions 已删去 ☐ 系数最简

### Step 5：最终验证
- **操作**：逐元素检查原子守恒；计算总电荷检查电荷守恒；确认介质与产物匹配；确认与实验现象一致。
- **依据 KP**：[[方程式书写]]（守恒检查）
- **检查点**：☐ 原子守恒 ☐ 电荷守恒 ☐ 介质-产物匹配 ☐ 与实验现象一致

## 四、反应机理拆解（含检查表） {#mechanism-analysis}

> 以**酸性介质中 KMnO₄ 氧化 Fe²⁺**为例，演示离子-电子法的完整"电子流"拆解。

#### 步骤 1：拆分半反应并确定电子流方向
- **攻击位点**：MnO₄⁻ 中 Mn 为 +7（高氧化态，缺电子），作为氧化剂接受电子；Fe²⁺ 作为还原剂失去电子。
- **电子流**：Fe²⁺ → Fe³⁺ + e⁻（氧化，电子流出）；MnO₄⁻ + 5e⁻ → Mn²⁺（还原，电子流入）。
- **学生任务（接力点）**：下一位同学需要判断的是——**若将介质改为强碱性，MnO₄⁻ 的还原产物是什么？配平时 H 和 O 应分别用什么物种平衡？**
- **检查表**：
  - ☐ 氧化剂（得电子）与还原剂（失电子）已正确区分
  - ☐ 氧化数变化计算正确（Mn +7→+2 得 5e；Fe +2→+3 失 1e）
  - ☐ 电子得失守恒（5Fe²⁺ 失 5e = 1MnO₄⁻ 得 5e）

#### 步骤 2：配平半反应（原子与电荷）
- **还原半反应**：MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
  - O 配平：左侧 4 个 O → 右侧 4 个 H₂O
  - H 配平：右侧 8 个 H → 左侧 8 个 H⁺
  - 电荷配平：左侧 (-1) + (+8) + (-5) = +2；右侧 +2 → 已平衡
- **氧化半反应**：Fe²⁺ → Fe³⁺ + e⁻
- **合并**：×5 氧化半反应后与还原半反应相加，消去电子：
  $$\mathrm{MnO_4^- + 5Fe^{2+} + 8H^+ \longrightarrow Mn^{2+} + 5Fe^{3+} + 4H_2O}$$
- **检查表**：
  - ☐ 原子守恒（Mn、O、H、Fe 左右相等）
  - ☐ 电荷守恒（左侧 -1 + 10 + 8 = +17；右侧 +2 + 15 = +17）
  - ☐ 介质一致（酸性介质，无游离 OH⁻）

## 五、典型例题串讲 {#typical-examples}

### 例题 1：KMnO₄ 在不同介质中氧化 Na₂SO₃（难度 ⭐⭐⭐）

**题目：**
写出 KMnO₄ 在以下三种介质中与 Na₂SO₃ 反应的离子方程式：
（1）强酸性介质；（2）中性/弱碱性介质；（3）强碱性介质。

**分析：**
1. 首先确定介质，这是本题的"题眼"。同一氧化剂 KMnO₄ 在不同介质中还原产物不同，必须分别处理。
2. 还原剂均为 SO₃²⁻，氧化产物均为 SO₄²⁻（S 由 +4 → +6，每个 S 失 2e）。
3. 氧化剂 KMnO₄ 中 Mn 的还原产物：酸性 → Mn²⁺（+7→+2，得 5e）；中性 → MnO₂（+7→+4，得 3e）；强碱性 → MnO₄²⁻（+7→+6，得 1e）。
4. 分别用离子-电子法配平三个半反应，再与 SO₃²⁻ 的氧化半反应合并。

**解答：**

**（1）强酸性介质：**
- 还原半反应：MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
- 氧化半反应：SO₃²⁻ + H₂O → SO₄²⁻ + 2H⁺ + 2e⁻
- 合并（还原×2，氧化×5）：
  $$\mathrm{2MnO_4^- + 5SO_3^{2-} + 6H^+ \longrightarrow 2Mn^{2+} + 5SO_4^{2-} + 3H_2O}$$

**（2）中性/弱碱性介质：**
- 还原半反应：MnO₄⁻ + 2H₂O + 3e⁻ → MnO₂ + 4OH⁻
- 氧化半反应：SO₃²⁻ + 2OH⁻ → SO₄²⁻ + H₂O + 2e⁻
- 合并（还原×2，氧化×3）：
  $$\mathrm{2MnO_4^- + 3SO_3^{2-} + H_2O \longrightarrow 2MnO_2 + 3SO_4^{2-} + 2OH^-}$$

**（3）强碱性介质：**
- 还原半反应：MnO₄⁻ + e⁻ → MnO₄²⁻
- 氧化半反应：SO₃²⁻ + 2OH⁻ → SO₄²⁻ + H₂O + 2e⁻
- 合并（还原×2，氧化×1）：
  $$\mathrm{2MnO_4^- + SO_3^{2-} + 2OH^- \longrightarrow 2MnO_4^{2-} + SO_4^{2-} + H_2O}$$

**反思：**
- **介质优先原则**在本题中体现得淋漓尽致：三个方程式产物完全不同，但配平方法完全一致。
- 最容易出错的是中性条件下写成 Mn²⁺ 或碱性条件下出现 H⁺——必须牢记"介质-产物"对应关系。
- 配平后务必检查：碱性条件下是否有游离 H⁺？酸性条件下是否有游离 OH⁻？

### 例题 2：Cr₂O₇²⁻ 酸性条件下氧化 Fe²⁺（难度 ⭐⭐⭐）

**题目：**
在酸性介质中，K₂Cr₂O₇ 与 FeSO₄ 反应，写出完整的离子方程式。

**分析：**
1. 介质：酸性（H⁺ 参与，H₂O 生成）。
2. 氧化剂：Cr₂O₇²⁻ 中 Cr 为 +6，还原产物为 Cr³⁺（酸性条件下 Cr 的典型还原产物）。
3. 还原剂：Fe²⁺ 氧化为 Fe³⁺。
4. **关键注意点**：Cr₂O₇²⁻ 含 **2 个 Cr**，每个 Cr 从 +6 → +3 得 3e，共得 6e。这是最常见的漏乘错误点。

**解答：**
- 还原半反应：Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O
  - Cr：左侧 2 个 → 右侧 2 个
  - O：左侧 7 个 → 右侧 7 个 H₂O
  - H：右侧 14 个 → 左侧 14 个 H⁺
  - 电荷：左侧 (-2) + (+14) + (-6) = +6；右侧 2×(+3) = +6
- 氧化半反应：Fe²⁺ → Fe³⁺ + e⁻
- 合并（氧化×6）：
  $$\mathrm{Cr_2O_7^{2-} + 6Fe^{2+} + 14H^+ \longrightarrow 2Cr^{3+} + 6Fe^{3+} + 7H_2O}$$

**反思：**
- **多原子变价**是竞赛配平的高频陷阱。Cr₂O₇²⁻、As₂S₃、Fe₃O₄ 等物种都必须整体计算电子转移。
- 若只算 1 个 Cr 得 3e，会得到错误的系数比（如 Cr₂O₇²⁻ : Fe²⁺ = 1:3 而非 1:6）。
- 酸性条件下 Cr³⁺ 是绿色溶液，与题目描述的"橙红色变绿色"现象一致——配平结果需与实验现象互证。

### 例题 3：金属铝与氢氧化钠溶液反应（难度 ⭐⭐）

**题目：**
写出铝与氢氧化钠溶液反应的化学方程式和离子方程式。

**分析：**
1. 这是一个**非氧化还原反应与氧化还原反应的复合反应**：Al 与 H₂O 反应生成 Al(OH)₃ 和 H₂，然后 Al(OH)₃ 与 NaOH 反应生成 NaAlO₂（或 Na[Al(OH)₄]）。
2. 也可以直接看作 Al 在碱性条件下被水氧化：Al 从 0 → +3，H₂O 中 H 从 +1 → 0。
3. 产物写法有两种等价形式：NaAlO₂（偏铝酸钠，简化写法）或 Na[Al(OH)₄]（四羟基合铝酸钠，更精确）。

**解答：**

**化学方程式（简化形式）：**
$$\mathrm{2Al + 2NaOH + 2H_2O \longrightarrow 2NaAlO_2 + 3H_2 \uparrow}$$

**化学方程式（精确形式）：**
$$\mathrm{2Al + 2NaOH + 6H_2O \longrightarrow 2Na[Al(OH)_4] + 3H_2 \uparrow}$$

**离子方程式：**
$$\mathrm{2Al + 2OH^- + 6H_2O \longrightarrow 2[Al(OH)_4]^- + 3H_2 \uparrow}$$

或简化写法：
$$\mathrm{2Al + 2OH^- + 2H_2O \longrightarrow 2AlO_2^- + 3H_2 \uparrow}$$

**反思：**
- 本题展示了"碱性条件下金属与水反应"的典型模式：活泼金属（Al、Zn、Si 等）在强碱中反应，实质是金属被水氧化，OH⁻ 仅提供碱性环境并溶解产物。
- 离子方程式中 Al 和 H₂ 不拆（单质、气体），NaOH 拆为 Na⁺ 和 OH⁻，Na[Al(OH)₄] 拆为 Na⁺ 和 [Al(OH)₄]⁻，删去 spectator ion Na⁺。
- 注意：Al 是两性金属，与酸反应也生成 H₂，但产物为 Al³⁺ 而非 [Al(OH)₄]⁻——再次体现介质对产物的决定性影响。

### 例题 4：歧化反应配平——Cl₂ 与 NaOH 溶液（难度 ⭐⭐⭐）

**题目：**
写出 Cl₂ 分别与 (1) 冷稀 NaOH 溶液；(2) 热浓 NaOH 溶液反应的离子方程式。

**分析：**
Cl₂ 中 Cl 为 0 价，处于中间氧化态，在碱性条件下可发生歧化反应：一部分 Cl 被氧化，一部分被还原。
- 冷稀条件：Cl 的歧化程度较小 → 生成 Cl⁻（−1 价）和 ClO⁻（+1 价）
- 热浓条件：高温促进更深度的歧化 → 生成 Cl⁻（−1 价）和 ClO₃⁻（+5 价）

**解答：**

**(1) 冷稀 NaOH 溶液**
- 还原半反应：$\mathrm{Cl_2 + 2e^- \longrightarrow 2Cl^-}$
- 氧化半反应：$\mathrm{Cl_2 + 4OH^- \longrightarrow 2ClO^- + 2H_2O + 2e^-}$
- 合并：
  $$\mathrm{Cl_2 + 2OH^- \longrightarrow Cl^- + ClO^- + H_2O}$$

**(2) 热浓 NaOH 溶液**
- 还原半反应：$\mathrm{Cl_2 + 2e^- \longrightarrow 2Cl^-}$（×5）
- 氧化半反应：$\mathrm{Cl_2 + 12OH^- \longrightarrow 2ClO_3^- + 6H_2O + 10e^-}$
- 合并（还原×5 + 氧化×1）：
  $$\mathrm{3Cl_2 + 6OH^- \longrightarrow 5Cl^- + ClO_3^- + 3H_2O}$$

**反思：**
- **歧化反应配平的技巧**：把同一反应物（Cl₂）同时写在氧化半反应和还原半反应中，分别按不同产物配平，最后按电子守恒合并。
- 温度对歧化深度的影响是竞赛高频考点：冷稀 → 次氯酸盐（消毒/漂白）；热浓 → 氯酸盐（火柴、火药）。
- 验证：碱性条件下无游离 H⁺，两个方程式均满足此要求。

### 例题 5：归中反应配平——H₂S 与 SO₂（难度 ⭐⭐⭐）

**题目：**
写出 H₂S 气体与 SO₂ 气体混合反应的化学方程式。

**分析：**
- H₂S 中 S 为 −2 价（最低价态，只能被氧化）
- SO₂ 中 S 为 +4 价（中间价态，可被还原）
- 两者反应属于**归中反应**：高价和低价反应生成中间价态 → S（0 价）

**解答：**

**氧化半反应**：$\mathrm{H_2S \longrightarrow S + 2H^+ + 2e^-}$（×2）

**还原半反应**：$\mathrm{SO_2 + 4H^+ + 4e^- \longrightarrow S + 2H_2O}$

合并（氧化×2 + 还原×1，消去电子和 H⁺）：
$$\mathrm{2H_2S + SO_2 \longrightarrow 3S \downarrow + 2H_2O}$$

**验证：**
- 原子守恒：左侧 H=4, S=3, O=2；右侧 H=4, S=3, O=2 ✓
- 电荷守恒：两侧均为电中性 ✓
- 氧化态：2 mol S(−2) 失 4e⁻ → 2 mol S(0)；1 mol S(+4) 得 4e⁻ → 1 mol S(0) ✓

**反思：**
- **归中反应与歧化反应是氧化还原反应的两种特殊类型**，竞赛中常以"混合气体产生黄色固体（硫）"的现象出现。
- 归中反应的配平关键是：先确定中间价态产物，再分别写氧化和还原半反应。
- 本题若用氧化数法更快捷：H₂S 中 S(−2→0) 升 2，SO₂ 中 S(+4→0) 降 4 → 最小公倍数 4 → H₂S 前配 2，SO₂ 前配 1。
