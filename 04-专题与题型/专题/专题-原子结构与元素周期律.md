## 零点六、课堂投影速查卡 {#classroom-quick-card}

**本页课堂入口：** 不要从"量子数有四个"开始。先给一道元素推断题——"某元素基态有 3 个未成对电子，它可能是谁？"——从学生的答案倒推出他们缺的是"排布写不对"还是"周期律不会用"。

**先问三个问题：**

1. 学生是先写电子排布再判断，还是直接猜？
2. 如果是性质比较题（半径/电离能/电负性），学生是在用周期律趋势，还是在背数字？
3. 反常点（Be>B、N>O、Cr/Cu 排布）学生能解释原因，还是只能说"特殊的"？

**一屏判断卡：**

```
原子结构题 → 第一步：写电子排布
    │
    ├─ 基态原子 → Aufbau + Pauli + Hund
    │   ├─ 正常排布 → 直接写
    │   └─ Cr/Cu 族 "异常" → 交换能解释：d⁵/d¹⁰ 半满/全满
    │
    ├─ 离子排布 → 先失 n 最大轨道（4s 先于 3d！）
    │
    ├─ 性质比较 → 同周期左→右：r↓ I₁↑ χ↑
    │   ├─ 注意反常：ⅡA>ⅢA (s²→p¹)、ⅤA>ⅥA (p³→p⁴)
    │   └─ 注意 6s 区：相对论效应 → 惰性电子对 (Tl⁺>Tl³⁺)
    │
    └─ 定量估算（提高班）→ Slater 规则求 Z* → 交换能对数
```

**三大书写陷阱（每届必错）：**

| 陷阱 | 错误做法 | 正确做法 |
|:---|:---|:---|
| 书写顺序 ≠ 填充顺序 | 按 Aufbau 写：…4s² 3d⁶ | 按 n 排序写：…**3d⁶ 4s²** |
| 失电子顺序 | 先失 3d（能级高）| 先失 **n 最大**的轨道（4s 先失） |
| 半满"更稳定" | "因为半满所以稳定" | "半满→交换能收益大→总能量更低" |

**元素推断五步法：**

| 步 | 操作 | 检查 |
|:---:|:---|:---|
| 1 | 写基态电子排布 | ☐ 三原则 OK？☐ 注意 Cr/Cu 族 |
| 2 | 判周期/族/区 | ☐ n_max=周期 ☐ 价电子=族号 |
| 3 | 比周期性性质 | ☐ 方向对？☐ 反常点已标注？ |
| 4 | 离子推断 | ☐ 先失 n 最大 ☐ 常见氧化态匹配？ |
| 5 | 验证 | ☐ 量子数合法？☐ 未成对电子数对？ |

## 一、核心结论汇总 {#core-conclusions}

1. **元素性质由电子排布决定**。电子排布遵循 **Aufbau 原理 + Pauli 不相容原理 + Hund 规则** 三原则，周期律的本质是电子构型的周期性重复。
2. **离子失电子先失 n 最大的轨道**（如 Fe 先失 4s，不是 3d），书写时按 n 排序、填充时按能级高低——两者顺序不同。
3. **最高频决策路径**：题目要求"推断元素"或"比较性质"时，第一步**先写出电子排布** → 第二步判断周期/族/区 → 第三步调用周期律比较。
4. **半满/全满稳定是交换能的定量结果**。平行自旋电子对数越多，交换能越负（体系越稳定），d⁵ 和 d¹⁰ 的交换能收益足以补偿能级差，导致 Cr、Cu 等"异常"排布。
5. **重元素（6s 区）的"反常"性质源于相对论效应**。6s 电子的相对论性质量增加导致轨道收缩，是 Au 的金黄色、Hg 的低熔点、Pb²⁺ 稳定性的根源。

### 2.1 量子数规则速查

| 触发条件（题目关键词） | 量子数   | 符号  | 取值范围         | 物理意义          | 常见陷阱           |
| :---------- | :---- | :-- | :----------- | :------------ | :------------- |
| "第几电子层"     | 主量子数  | n   | 1, 2, 3, ... | 电子层，决定主要能量    | 与周期号对应（n=最大层数） |
| "什么形状轨道"    | 角量子数  | l   | 0 ~ n-1      | 轨道形状（s/p/d/f） | l < n，不可等于 n   |
| "空间取向"      | 磁量子数  | m_l | -l ~ +l      | 轨道空间取向        | \|m_l\| ≤ l    |
| "自旋方向"      | 自旋量子数 | m_s | ±½           | 电子自旋          | 同一轨道两电子必须反平行   |

### 2.2 电子排布：书写顺序 vs 填充顺序

| 场景 | 顺序规则 | 示例 |
|:---|:---|:---|
| **书写**（化学式/离子式） | 按主量子数 n 递增 | Fe: [Ar] **3d⁶ 4s²** |
| **填充**（Aufbau 能级图） | 按能级高低（n+l 规则） | Fe 先填 4s² 再填 3d⁶ |
| **失电子**（电离） | 先失 n 最大的轨道 | Fe²⁺: [Ar] **3d⁶**（失 4s²） |

### 2.3 周期性性质趋势（同周期从左→右）

| 性质     | 趋势       | 核心原因        | 反常点                           |
| :----- | :------- | :---------- | :---------------------------- |
| 原子半径   | ↓ 减小     | Z* 增大，核引力增强 | —                             |
| 电离能 I₁ | ↑ 增大（总体） | Z* 增大，更难失电子 | ⅡA > ⅢA（p¹易失），ⅤA > ⅥA（p³半满稳定） |
| 电负性 χ  | ↑ 增大     | 吸引电子能力增强    | F 最大（4.0），稀有气体无定义             |
| 金属性    | ↓ 减弱     | 失电子能力减弱     | —                             |

### 2.4 异常电子排布的交换能解释

> 深化版核心表格。交换能不是定性口号，而是可估算的定量工具。

| 触发条件（题目关键词） | 元素 | "正常"预期 | 实际排布 | 交换能解释 | 交换能对数 |
| :---------------- | :--- | :------- | :----- | :------- | :----- |
| "Cr 的基态排布" | Cr | 3d⁴ 4s² | **3d⁵ 4s¹** | d⁵ 有 C(5,2)=10 对平行自旋，d⁴ 只有 6 对 | 10 > 6 |
| "Cu 的基态排布" | Cu | 3d⁹ 4s² | **3d¹⁰ 4s¹** | d¹⁰ 有 2×C(5,2)=20 对，d⁹ 只有 16 对 | 20 > 16 |
| "Mo 也异常吗？" | Mo | 4d⁴ 5s² | **4d⁵ 5s¹** | 同 Cr，主量子数增加使能级差缩小，交换能相对更重要 | 10 > 6 |
| "Zn 为什么不异常？" | Zn | 3d¹⁰ 4s² | 3d¹⁰ 4s²（正常） | d¹⁰ 已是全满，s²→s¹ 无交换能收益，且 4s→3d 能级差太大 | — |

**核心公式**：
$$E_{ex} \approx -K \cdot \sum \frac{n_i(n_i - 1)}{2}$$
其中 $n_i$ 为第 i 亚层中**平行自旋**电子数，K 为交换积分（正值）。平行自旋对数越多，交换能越负（体系越稳定）。

### 2.5 Slater 规则速查

> 估算有效核电荷 Z* 的半定量工具。注意：Slater 规则是经验拟合，不适合追求 3 位有效数字的精确定量计算。

**分组顺序**：
```
(1s) | (2s, 2p) | (3s, 3p) | (3d) | (4s, 4p) | (4d) | (4f) | (5s, 5p) | ...
```

| 屏蔽来源 | 被屏蔽电子为 (ns, np) | 被屏蔽电子为 (nd, nf) |
|:---|:---|:---|
| 同组电子 | σ = 0.35（1s 为 0.30） | σ = 0.35 |
| 左侧相邻组（n-1 层） | σ = 0.85 | σ = 1.00 |
| 更左侧组（≤ n-2 层） | σ = 1.00 | σ = 1.00 |
| 右侧组（n+1 层及以外） | σ = 0 | σ = 0 |

**公式**：$Z^* = Z - \sigma$

**速算示例**：Fe 中 3d 电子的 Z*
- 同组 3d：5 × 0.35 = 1.75
- 右侧 4s4p：2 × 0.35 = 0.70
- 左侧 ≤ 3s3p：18 × 1.00 = 18.00
- σ = 20.45 → **Z* = 26 - 20.45 = 5.55**

### 2.6 电负性三种标度对比

| 标度 | 公式 | 数值范围 | 优点 | 缺点 |
|:---|:---|:---:|:---|:---|
| **Pauling** | χ_P = 0.102√ΔD | ~0.7 ~ 4.0 | 从键能直接反推，最常用 | 需要大量键能数据，无法预测未知元素 |
| **Mulliken** | χ_M = (I + A)/2 | ~2 ~ 10 | 从原子参数直接计算，理论意义强 | 电子亲和能 A 数据不全，数值范围与 Pauling 不同 |
| **Allred-Rochow** | χ_AR = 0.359·Z*/r² + 0.744 | ~0.7 ~ 4.3 | 从 Z* 和半径计算，适合讨论趋势 | 对过渡金属的准确性有限 |

**使用原则**：
- 比较**相对顺序**时，三种标度几乎一致；比较**绝对数值**时，只能同标度内比较。
- 讨论"为什么某元素电负性高于另一元素"时，优先用 Allred-Rochow（可拆解为 Z* 和 r 两个因素）。

### 2.7 相对论效应对重元素（6s 区）的影响

> 决赛级别考点。定性理解即可，不需要定量计算。

| 元素 | "正常"预期 | 实际反常性质 | 相对论效应解释 | 竞赛关联 |
|:---|:---|:---|:---|:---|
| **Au** | 银白色（如 Cu、Ag）| **金黄色** | 6s 收缩使 6s→5d 跃迁能量落入可见光区 | 颜色/光谱题 |
| **Hg** | 常温固态（如 Cd）| **熔点 -39°C（液态）** | 6s² 收缩导致金属键显著减弱 | 物理性质反常 |
| **Tl** | Tl³⁺ 更稳定 | **Tl⁺ 更稳定**（惰性电子对）| 6s² 因收缩能量降低，难以失去 | 氧化态推断 |
| **Pb** | Pb⁴⁺ 更稳定 | **Pb²⁺ 更稳定** | 同上，6s² 惰性电子对 | 氧化态推断/方程式书写 |
| **Pt** | 类似 Ni、Pd | 化学惰性极高 | 6s 收缩使电离能异常升高 | 催化/配合物稳定性 |

**定性图像**：重元素内层电子速度接近光速 → 相对论性质量增加 → s 轨道（尤其 6s）收缩 → "惰性电子对效应"

## 四、反应机理拆解（含检查表）{#mechanism-analysis}

> 原子结构的能量排序与电子排布由量子力学原理和电子间相互作用共同决定。以下以**交换能的定量推导**为例，演示"能量流"拆解。

#### 步骤 1：写出电子组态并统计平行自旋电子数
- **操作**：确定 d 亚层的电子排布，按 Hund 规则最大化平行自旋数
- **检查表**：
  - ☐ 高自旋时单电子优先占据不同轨道且自旋平行
  - ☐ 配对后自旋相反，不产生额外交换能

#### 步骤 2：计算交换能稳定化
- **操作**：$E_{ex} = -K \cdot \sum \dfrac{n_i(n_i - 1)}{2}$，其中 $n_i$ 为第 $i$ 个亚层中平行自旋电子数
- **检查表**：
  - ☐ d⁵ 的交换能对数：C(5,2) = 10 对
  - ☐ d⁴ 的交换能对数：C(4,2) = 6 对
  - ☐ 差值 4K 足以补偿 4s→3d 能级差，导致 Cr 的异常排布

#### 步骤 3：与能级差竞争，确定基态排布
- **操作**：比较交换能收益与能级升高代价，选择总能量最低的排布
- **检查表**：
  - ☐ 交换能收益 > 能级差 → 异常排布（如 3d⁵4s¹）
  - ☐ 交换能收益 < 能级差 → 正常排布（如 3d⁸4s²）

## 五、典型例题串讲 {#typical-examples}

### 例题 1：电子排布书写与离子推断（★★ 初赛基础）

**题目**：写出 Fe（Z=26）的电子排布式、Fe²⁺ 和 Fe³⁺ 的电子排布式。

**分析**：
- 基态 Fe 按 Aufbau 填充：[Ar] 4s² 3d⁶
- 书写时按 n 排序：[Ar] **3d⁶ 4s²**
- Fe²⁺ 先失 4s 电子（n 最大），不是 3d
- Fe³⁺ 再失 1 个 3d 电子，3d⁵ 半满更稳定

**解答**：
- Fe: **[Ar] 3d⁶ 4s²**
- Fe²⁺: **[Ar] 3d⁶**
- Fe³⁺: **[Ar] 3d⁵**（半满稳定）

**反思**：
- 书写顺序 ≠ 填充顺序 ≠ 失电子顺序——三种场景三种规则，学生极易混淆
- 离子电荷与 d 电子数的关系是元素推断题的常见切入点

### 例题 2：量子数判断与元素推断（★★★ 综合应用）

**题目**：某元素基态原子价电子排布为 3d² 4s²。
(1) 该元素的原子序数是多少？
(2) 它位于周期表第几周期、第几族？
(3) 它有多少个未成对电子？

**分析**：
- [Ar] 核心 18e + 3d² 4s² = 22e → Z = 22（Ti）
- 最高 n = 4 → 第四周期；价电子 2+2 = 4 → ⅣB 族（过渡金属，族数 = s+d 电子）
- 3d² 按 Hund 规则分占两个 d 轨道 → 2 个未成对电子

**解答**：
1. **Z = 22**（Ti）
2. **第四周期、ⅣB 族、d 区**
3. **2 个未成对电子**

**反思**：
- 过渡金属的"族数"判断是高频陷阱：主族看 s/p 电子，过渡族看 s+d 电子
- Hund 规则直接决定未成对电子数，进而影响磁性判断

### 例题 3：综合推断 + 定量比较（★★★★ 深化应用）

**题目**：某元素 X 的基态原子中有 3 个未成对电子，其 +2 价离子有 2 个未成对电子。第一电离能介于 Ca 和 Ti 之间。
(1) 推断 X 的元素符号。
(2) 写出 X 和 X²⁺ 的基态电子排布式。
(3) 用 Slater 规则估算 X 的 4s 电子的 Z*，并与 Ca 的 4s 电子 Z* 比较，解释为什么 I₁(X) 介于 Ca 和 Ti 之间。

**分析**：
- 基态 3 个未成对电子 + I₁ 介于 Ca 和 Ti 之间 → 第四周期 d 区元素
- Ca(Z=20): 0 未成对；Sc(Z=21): 1 个；Ti(Z=22): 2 个；V(Z=23): 3 个未成对 ✅
- I₁ 实验值：Ca=6.11, Sc=6.56, Ti=6.83, V=6.75 eV → V 确实介于 Ca 和 Ti 之间 ✅
- +2 价离子有 2 个未成对：V²⁺ = [Ar]3d³ 实际有 3 个未成对，与题意"2 个"存在微小矛盾。竞赛中按最接近合理推断。
- Slater 规则估算：比较 Ca、V 的 4s 电子 Z*，定性说明 I₁ 趋势

**解答**：
1. **V（Z=23）**
2. V: **[Ar] 3d³ 4s²**；V²⁺: **[Ar] 3d³**
3. Z* 估算：
   - **V(4s)**：σ = 0.35(同组) + 6.80(3s3p) + 2.55(3d) + 10.00(内层) = 19.70 → **Z* = 3.30**
   - **Ca(4s)**：σ = 0.35 + 6.80 + 10.00 = 17.15 → **Z* = 2.85**
   - Z*(V) > Z*(Ca)，故 I₁(V) > I₁(Ca)。V 的 Z* 略小于 Ti(3.15)，但 I₁ 的精细差异还受钻穿效应和交换作用影响，Slater 规则只能给出半定量趋势。

**反思**：
- 这道题融合了"排布推断 + 离子稳定性 + 定量估算"三层能力，是第三轮/决赛前训练的标杆题型。
- **高价值陷阱**：题目条件"+2 价有 2 个未成对"与 V²⁺ 实际有 3 个未成对存在矛盾。竞赛中遇到这种情况，优先按"最接近合理"推断，并在解答中注明可能的题目瑕疵。
- Slater 规则的精度有限，不适合用于解释 I₁ 的细微差异（如 Ti vs V 仅差 0.08 eV），只能用于解释大尺度趋势。

### 例题 4：第一电离能顺序与半满/全满反常（周坤无机，⭐⭐⭐）

**题目：**
判断并解释以下第一电离能顺序：(1) B < C < N 正确，但 B < C < N < O 是否正确？(2) 为什么 Na < Mg > Al < Si？

**分析：**
半满/全满稳定性的电离能反常：N(2p³半满) > O(2p⁴)；Mg(3s²全满) > Al(3s²3p¹)。

**解答：**
- B < C < N < O 是**错误**的。正确顺序为 B < C < O < N。
  - N 的 2p³ 为半满稳定构型，失去电子需要额外能量
  - O 的 2p⁴ 多一个电子，该电子与另一电子配对，存在配对能排斥，反而比 N 更容易失去
- Na < Mg > Al < Si 中：
  - Mg 的 3s² 为全满构型，稳定性高，I₁ 反常高于 Al
  - Al 的 3p¹ 电子受 3s² 屏蔽，有效核电荷较低，易失去

**反思：**
- 电离能曲线不是单调递增的，"锯齿"是学生最容易忽视的考点
- 核心规律：**半满/全满构型 → 交换能收益 → 稳定性提升 → 电离能反常升高**
- 同周期中 I₁ 的典型反常点：ⅡA > ⅢA（s²→s²p¹）、ⅤA > ⅥA（p³→p⁴）
