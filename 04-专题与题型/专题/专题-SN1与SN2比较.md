## 零点六、课堂投影速查卡

**本页课堂入口：** 先判底物能不能形成稳定碳正离子，再看亲核试剂强弱与位阻，最后才决定是 `SN1` 还是 `SN2`。

| 维度 | SN1 | SN2 |
|:---|:---|:---|
| 决定步骤 | 离去基先走，形成碳正离子 | 亲核体背面进攻同步离去 |
| 底物偏好 | 三级、烯丙/苄基 | 甲基、一级 |
| 速率式 | `v=k[RX]` | `v=k[RX][Nu^-]` |
| 立体后果 | 消旋或部分消旋 | 构型翻转 |
| 溶剂偏好 | 极性质子溶剂 | 极性非质子溶剂 |

## 一、核心对比总表

| 比较维度 | SN2 | SN1 |
|---------|:---|:---|
| **机理类型** | 双分子协同 | 单分子分步 |
| **决速步** | Nu⁻ + R-X → [Nu···C···X]‡ | R-X → R⁺ + X⁻ |
| **速率方程** | rate = k[RX][Nu⁻] | rate = k[RX] |
| **反应级数** | 二级 | 一级 |
| **立体化学** | Walden翻转（100%构型反转） | 外消旋化（部分翻转+部分保持） |
| **底物活性** | CH₃X > 1° > 2° > 3° | 3° > 2° > 1° > CH₃X |
| **亲核试剂** | 强亲核试剂有利 | 亲核试剂浓度不影响速率 |
| **离去基** | 好的离去基有利 | 好的离去基必需 |
| **溶剂** | 极性非质子溶剂有利 | 极性质子溶剂有利 |
| **典型副反应** | E2（强碱/高温） | E1、重排 |

## 二、底物结构的影响

### 2.1 烷基卤代烷

```
CH₃-X    1°-R-X    2°-R-X    3°-R-X
  │         │         │         │
 SN2      SN2为主   SN1/SN2   SN1为主
         （强Nu⁻）  （竞争）
```

### 2.2 烯丙基/苄基体系

- 烯丙基卤、苄基卤：**SN1/SN2 均很快**
- 原因：过渡态/中间体可稳定正电荷（共振稳定）

### 2.3 芳基/乙烯基卤

- **SN1/SN2 均难发生**（C-X 键能大，碳正离子不稳定）
- 特殊条件下可通过加成-消除机理进行

## 三、亲核试剂的影响

### 3.1 亲核性规律

| 溶剂类型 | 亲核性顺序 |
|---------|-----------|
| 极性质子溶剂（如乙醇） | I⁻ > Br⁻ > Cl⁻ > F⁻（与碱性相反） |
| 极性非质子溶剂（如DMF、DMSO） | F⁻ > Cl⁻ > Br⁻ > I⁻（与碱性一致） |

### 3.2 亲核性 vs 碱性

- **亲核性**：给电子进攻碳的能力（动力学）
- **碱性**：给电子夺取质子的能力（热力学）
- 一般规律：同一周期从左到右，亲核性↓，碱性↓

## 四、溶剂效应

| 溶剂类型 | 特征 | 对SN2的影响 | 对SN1的影响 |
|---------|------|-----------|-----------|
| 极性质子溶剂 | 能形成氢键（H₂O, ROH） | 不利（Nu⁻被溶剂化） | 有利（稳定碳正离子） |
| 极性非质子溶剂 | 高极性，无活泼氢（DMF, DMSO,丙酮） | 有利（Nu⁻裸露） | 一般 |
| 非极性溶剂 | 低介电常数（己烷, 苯） | 不利（难以稳定过渡态电荷） | 不利 |

## 五、离去基能力

### 5.1 离去基能力顺序（常见）

$$
\mathrm{I^- > Br^- > Cl^- > F^-}
$$

$$
\mathrm{OTs^- \approx OMs^- > I^- > H_2O}
$$

### 5.2 判断标准

- 离去基越稳定（共轭碱越弱），离去能力越强
- pKa(H-X) 越小，X⁻ 越稳定，离去能力越强

## 六、立体化学详解

### 6.1 SN2 的 Walden 翻转

- 亲核试剂从 C-LG 背面进攻
- 过渡态为五配位三角双锥
- 产物构型 100% 反转

### 6.2 SN1 的外消旋化

- 碳正离子中间体为平面 sp² 构型
- Nu⁻ 可从两面等概率进攻
- 理论上 50:50 外消旋，实际常因离子对效应而有偏向

## 八、竞赛解题策略

### 8.1 判断流程图

```
底物是 3° 烷基卤？
  ├─ 是 → 强亲核试剂/弱碱 → SN1
  │      → 强碱/高温 → E1
  │
  └─ 否 → 底物是 1°/CH₃？
           ├─ 是 → 几乎总是 SN2（除非强碱/高温 → E2）
           └─ 否（2°）→ 看试剂和条件
                    → 强亲核/弱碱 → SN2
                    → 强碱 → E2
```

### 8.2 常见陷阱

- 溶剂判断错误：把 DMSO 当成质子溶剂
- 忽略离子对效应：SN1 产物并非完全外消旋
- 苄基/烯丙基底物：虽然像 1°，但 SN1 也能进行
