## 零点八、第三张教学图：选择性判据速查决策树 {#selectivity-decision-tree}

**建议用途：** 这张图是 Part B（立体化学）的课堂核心工具。学生面对 α-手性羰基加成题时最常见的卡点是”该用 Felkin-Anh 还是 Cram 螯合？该不该上 Evans 模型？Curtin-Hammett 什么时候出场？”——这张决策树把分散在 Clayden 各章的选择性模型压缩成一张”先问哪一句”的速查界面。

**决策树总入口 → 四个分叉：**

```text
拿到一道”α-手性底物 + 亲核/亲电试剂 → 预测主产物构型”的题

├─ 分叉 1：底物有没有螯合金属？
│   ├─ YES（有 TiCl₄ / Zn²⁺ / Mg²⁺ 等）→ Cram 螯合模型
│   │   → 金属同时螯合 C=O 和 Cα-OR → 固定构象 → 亲核试剂从位阻较小面进攻
│   │   → 检查：螯合环是五元环还是六元环？（五元环螯合最稳定）
│   │
│   └─ NO（无金属或金属不螯合）→ 进入分叉 2
│
├─ 分叉 2：底物是环状还是开链？
│   ├─ 环状（环己酮类）→ 环状酮面选择性规则
│   │   → 亲核试剂优先从 equatorial 面进攻（axial 进攻有 1,3-二轴位阻）
│   │   → 例外：如果底物有大的 axial 取代基，可能改变面偏好
│   │   → 检查：画椅式 → 标出两个面的位阻 → 判断 equatorial/axial 进攻面
│   │
│   └─ 开链（α-手性醛/酮）→ Felkin-Anh 模型
│       → Cα 最大取代基 L 垂直于 C=O 平面 → 亲核试剂从远离 L 的面进攻
│       → 检查：如果 Cα 有电负性取代基（F, OR, NR₂），偶极效应可能反转选择性！
│       → 课堂口诀：”大基团竖起来，小基团两边躺，亲核试剂打空旷那一侧”
│
├─ 分叉 3：题目问的是”产物比”还是”主产物构型”？
│   ├─ 只问主产物构型 → 用上面 Felkin-Anh / Cram 直接判断即可
│   │
│   └─ 问”产物 A:B 的比例”→ 进入 Curtin-Hammett 分析
│       → 产物比不由构象浓度决定，由 ΔΔG‡ 决定
│       → 即使低能构象占 99%，如果高能构象的活化能更低 → 主产物来自高能构象
│       → 真题标志词：”两个构象能量差 X kJ/mol，但产物比却是 Y:Z” → 立刻触发 C-H
│
└─ 分叉 4：底物是否有手性助剂（共价连接的手性源）？
    ├─ YES → 不对称助剂模型
    │   ├─ 噁唑烷酮（Evans）→ (Z)-烯醇 + 面选择性（苄基挡一面）
    │   ├─ 手性亚砜 → 亚砜的手性硫诱导面选择性
    │   ├─ 手性肟醚 → 肟的 E/Z 几何 + 手性辅助基
    │   └─ SAMP/RAMP 腙 → 手性吡咯烷 → 烷基化面选择性
    │
    └─ NO + 底物本身无手性 → 不对称催化模型
        ├─ Sharpless 环氧化（烯丙醇 + Ti(OiPr)₄ + 酒石酸酯）
        ├─ Noyori 氢化（BINAP-Ru → 酮的对映选择性氢化）
        └─ 有机催化（脯氨酸 → 醛的 α-官能团化，通过烯胺/亚胺离子）
```

**五模型速查卡：**

| 模型 | 触发条件 | 核心逻辑 | 一句口诀 | 最容易错的点 |
|:---|:---|:---|:---|:---|
| **Felkin-Anh** | α-手性醛/酮 + 亲核试剂 + 无金属 | Cα 最大基团 ⟂ C=O；Nu 从空旷面进攻 | “大基团竖起来，从空旷面打” | 有电负性取代基时偶极可能反转选择性 |
| **Cram 螯合** | α-烷氧基醛/酮 + 螯合金属 (Ti/Zn/Mg) | 金属螯合锁定构象 → Nu 从位阻小面进攻 | “有金属先螯合，锁死再判断” | 没金属时不要用 Cram——退回 Felkin-Anh |
| **Evans 助剂** | 手性噁唑烷酮 + 酰氯 → 烯醇 + E⁺ | (Z)-烯醇固定 + 苄基挡一面 | “苄基挡下面，亲电从上面来” | 只适用于酰基噁唑烷酮底物，不能泛化 |
| **Curtin-Hammett** | 题目明确给两个构象的能量差 + 产物比 | 产物比 = e^(-ΔΔG‡/RT)，不看构象浓度 | “不问谁多，只问谁快” | 不要用构象浓度比去预测产物比！ |
| **Sharpless AE** | 烯丙醇 + Ti(OiPr)₄ + DET + TBHP | OH 导向 + 手性酒石酸酯选面 | “(+)-DET 打上面，(-)-DET 打下面” | 非烯丙醇底物不适用——必须要有 OH 导向 |

**课堂使用方式：**
- 先投影完整决策树（1 页 PPT）→ 给 2 min 让学生”走一遍分叉”。
- 再投影五模型速查卡（1 页 PPT）→ 对照 Felkin-Anh 和 Cram 螯合的 “触发条件” 列，强调这是最高频混淆点。
- 练习：给 4 个底物条件（a. α-OMe 醛 + TiCl₄ / b. α-Ph 酮 + MeLi / c. Evans 助剂 + BnBr / d. 烯丙醇 + TBHP + (+)-DET）→ 限时 3 min 走决策树 → 报出模型名 + 主产物构型。

**板书收口句：** “第四轮不背模型名——看条件走分叉，四个问题问完模型自动出来。”

## A.1 核心结论汇总 {#core-conclusions}

**必须记住的 4 条物理有机思维链：**

1. **能量 → 方向**：ΔG = -RTlnK。ΔG < -8 kJ/mol 即反应完全。键能差 > 8 kJ/mol 反应单向。
2. **结构 → 速率**：Hammett ρ 的符号和大小告诉你过渡态的电荷需求——ρ > 0 意味着过渡态负电荷积累（吸电子基加速），ρ < 0 意味着过渡态正电荷积累（给电子基加速）。
3. **同位素 → 速控步**：KIE（$k_\mathrm{H}/k_\mathrm{D}$）> 1 意味着 C-H 键在速控步中断裂。一级 KIE 通常 2-7，超过 7 暗示隧穿效应。
4. **轨道对称性 → 反应允许/禁阻**：不要背 WH 规则结论——从 HOMO-LUMO 对称性匹配去推。同面-同面 = 同号重叠 = 允许。

**2025 命题新热点**：
- **Bent 规则**：电负性取代基 → 中心原子杂化轨道 p 成分增加 → 其他键 s 成分增加 → 键角增大（2025 初赛第 8 题核心考点）
- **构象→选择性因果链**：α-氟代羰基的偶极-偶极排斥主导构象能差 → 决定反应面选择性

## A.1.5 竞赛加厚：证据题的四种最短打法

| 证据类型 | 第一反应不要做什么 | 最短切入口 | 最常见结论 | 课堂收口句 |
|:---|:---|:---|:---|:---|
| **Hammett** | 不要一上来背取代基强弱 | 先看 $\rho$ 的正负，再看绝对值大小 | 过渡态是缺电子还是富电子；电荷积累强不强 | “Hammett 不是记忆题，是过渡态电性显影。” |
| **KIE** | 不要把“有 KIE”直接等同于“这步断键” | 先分一级/二级 KIE，再问 C-H 是否真的进速控步 | 能判速控步是否含断 C-H / 成 C-H | “同位素效应只回答速控，不自动回答全机理。” |
| **标记实验** | 不要见同位素去向就急着画整条路径 | 先问标签保留、交换还是迁移 | 能判重排、可逆步、亲核源/离去源 | “先追原子命运，再谈电子命运。” |
| **捕获实验** | 不要把捕获产物当主路径产物 | 先问被捕获的是自由基、正离子还是负离子 | 能证明中间体存在，但不一定证明其为唯一主路 | “捕获得到的是‘存在证据’，不是‘唯一证据’。” |

## A.2 物理有机工具箱 {#physical-organic-toolbox}

### Hammett 方程（σ/ρ 分析）

**定义**：
$$\log\left(\frac{k_\mathrm{X}}{k_\mathrm{H}}\right) = \rho \cdot \sigma_\mathrm{X}$$

$$\log\left(\frac{K_\mathrm{X}}{K_\mathrm{H}}\right) = \rho \cdot \sigma_\mathrm{X}$$

| 参数 | 含义 | 判断 |
|:---|:---|:---|
| **σₓ** | 取代基 X 的电子效应常数 | σ > 0：吸电子（NO₂, CN, CF₃）；σ < 0：给电子（NH₂, OMe, Me） |
| **σₘ vs σₚ** | 间位主要反映诱导效应；对位反映诱导+共轭 | σₚ(NH₂) = -0.66 但 σₘ(NH₂) ≈ 0（共轭给电子在对位才体现） |
| **ρ** | 反应常数，反映过渡态对取代基电子效应的敏感度 | ρ > 0：过渡态负电荷积累；ρ < 0：过渡态正电荷积累；\|ρ\| 越大越敏感 |
| **σ⁺ / σ⁻** | 特殊取代基常数 | σ⁺：过渡态直接共轭到给电子基（如 SN1 的苄基碳正离子）；σ⁻：过渡态直接共轭到吸电子基 |

**实战判断流程**：
```text
拿到 Hammett 数据 → 
  ρ > 0 → 吸电子基加速 → 过渡态负电荷多于起始态（如 SN2 的离去基带负电荷走）
  ρ < 0 → 给电子基加速 → 过渡态正电荷多于起始态（如 SN1 的碳正离子形成）
  |ρ| 大 → 取代基效应强 → 反应中心电荷变化大 → 过渡态"像产物"（晚过渡态）
  |ρ| 小 → 取代基效应弱 → 反应中心电荷变化小 → 过渡态"像反应物"（早过渡态）
```

### 动力学同位素效应（KIE）

| 类型 | 定义 | 典型值 | 告诉你的信息 |
|:---|:---|:---:|:---|
| **一级 KIE** | $k_\mathrm{H}/k_\mathrm{D}$，C-H/C-D 键在速控步断裂 | 2–7 | 速控步涉及该 C-H 键的断裂 |
| **二级 α-KIE** | D 连在反应中心上，但该 C-H 键不断裂 | 1.1–1.3 | 速控步中该碳的杂化状态变化（sp³→sp² 时正常 KIE；sp²→sp³ 时反 KIE < 1） |
| **二级 β-KIE** | D 连在反应中心的邻位 | ~1.0–1.2 | 超共轭效应参与稳定过渡态（β-D 的超共轭比 β-H 弱） |
| **溶剂 KIE** | $k(\mathrm{D}_2\mathrm{O})/k(\mathrm{H}_2\mathrm{O})$ | 变化大 | 区分一般酸催化(GAC) vs 特殊酸催化(SAC) |

**KIE 的物理本质**：C-H 和 C-D 的零点能不同（ZPE = ½hν）。C-D 的 ν 更低 → ZPE 更低 → 活化能更高 → 反应更慢。一级 KIE 源于速控步过渡态中该键的力常数变化。

### Curtin-Hammett 原理

> **产物比由过渡态能差决定，不由中间体浓度比决定。**

$$\\frac{[P_A]}{[P_B]} = e^{-(ΔG^‡_A - ΔG^‡_B)/RT}$$

**关键推论**：
- 即使两个构象的浓度比是 99:1，如果高能构象通往产物的活化能更低，**主要产物来自高能构象**。
- 2025 初赛第 8 题的核心逻辑：α-氟代羰基的 anti/syn 构象能差 → 反应面选择性不由构象浓度决定 → 由各自通往过渡态的活化能决定。

### Bent 规则

> **电负性取代基 → 中心原子指向该取代基的杂化轨道 p 成分增加 → 指向其他基团的轨道 s 成分增加 → 其他键角增大。**

| 实例 | 解释 |
|:---|:---|
| CH₂F₂ 的 ∠FCF < ∠HCH | F 电负性大 → C-F 键用更多 p → C-H 键用更多 s → ∠HCH 增大 |
| 环丙烷的"C=C"性质 | 环张力迫使 C-C 键用更多 p → 指向 H 的轨道用更多 s → C-H 键增强 |
| PH₃ 的键角 ~93°（接近纯 p） | P 的电负性低 → 成键几乎全用 p → 键角接近 90° |

## A.3 FMO 推导 Woodward-Hoffmann 规则 {#fmo-wh}

> **核心原则**：不要背结论。从 HOMO-LUMO 对称性匹配去推导。

### 三个基本步骤

```
Step 1: 画出反应物和产物的前线轨道（HOMO / LUMO）
Step 2: 确定轨道的对称性（相对于保持的对称元素）
Step 3: 判断 HOMO-LUMO 重叠是否同号 → 同号 = 成键相互作用 = 允许
```

### 周环反应四大类的统一判据

| 反应类型 | 判断方法 | 热反应 | 光反应 |
|:---|:---|:---|:---|
| **电环化** | 末端 p 轨道旋转方向（顺旋/对旋）× π 电子数 | 4n: 顺旋；4n+2: 对旋 | 相反 |
| **环加成** | 同面(supra)/异面(antara) × 电子数 | [π2s+π2s]: 禁阻(热)；[π4s+π2s]: 允许 | 光反应反转 |
| **σ迁移** | 迁移基团同面/异面 × 碳数 | [1,3]-C: 异面(antara)；[3,3]-C: 同面(supra) | 反转 |
| **螯变反应** | 螯变过渡态考虑位阻最小的构象 | 信封式 > 船式 | — |

### FMO 推导实例：Diels-Alder 反应

```
二烯 HOMO (ψ₂): 两端同号 → 与亲二烯体 LUMO 同面-同面重叠 → 同号 → 允许
→ 为什么 [4+2] 热允许、[2+2] 热禁阻？
  [4+2]: 二烯 HOMO 两端同号 + 亲二烯体 LUMO 两端同号 → 同面重叠 = 同号
  [2+2]: 烯烃 HOMO 两端同号 × 2 → 同面重叠 = 一端同号一端异号 → 净零 → 禁阻
```

| 中间体 | 结构特征 | 典型反应 | 竞赛考点 |
|:---|:---|:---|:---|
| **碳正离子** | sp²，空 p 轨道，6e⁻ | SN1、E1、重排 | 非经典碳正离子（2-降冰片基）；1,2-迁移倾向：Ph > H > alkyl |

# Part B：有机立体化学与构象分析

## B.1 核心结论汇总 {#stereo-core}

1. **构象决定反应性**：不是"哪个构象多"决定产物，而是"哪个构象通往产物的活化能低"决定产物（Curtin-Hammett）。
2. **A 值 = 构象偏好**：环己烷上取代基的 A 值（-ΔG°）越大，该取代基越倾向于占据 equatorial 位。A 值表是构象分析的"基本数据"。
3. **不对称诱导的本质**：手性环境 → 非对映过渡态能差 → 一个面比另一个面更容易被进攻。
4. **立体电子效应优先于位阻**：当电子效应和位阻效应预测相反时，在动力学控制下立体电子效应常占优。

## B.1.5 竞赛加厚：立体题的四个起手按钮

| 起手按钮 | 先看什么 | 常见应用 | 容易犯的错 | 课堂提醒 |
|:---|:---|:---|:---|:---|
| **构象优势** | 主导构象是否已被大取代基锁定 | 环己烷、烯丙位、半椅式过渡态 | 只看最稳定构象，不看反应构象 | “先找能反应的构象，不只找最舒服的构象。” |
| **进攻面选择** | 两个面的遮挡、偶极、螯合和导向基 | 羰基加成、烯烃加成、亲电环化 | 把位阻解释成万能答案 | “位阻只是一票，电子和配位也会投票。” |
| **E2 几何要求** | 抗共平面能否成立 | 环系消除、桥环体系、刚性底物 | 忘了“能否成 anti”比碱强弱更先判断 | “先问几何可不可能，再问反应愿不愿意。” |
| **糖/异头效应** | 杂原子孤对与反键轨道是否反式共线 | 糖化学、缩醛、杂环立体选择 | 机械套用“轴向一定不稳” | “遇到杂原子先想立体电子效应，别只想 1,3-二轴排斥。” |

### 环己烷 A 值速查

| 取代基 | A 值 (kJ/mol) | A 值 (kcal/mol) |  equatorial 占比 (298K) |
|:---|:---:|:---:|:---:|
| H | 0 | 0 | — |
| Me | 7.3 | 1.74 | 95% |
| Et | 7.5 | 1.79 | 95% |
| i-Pr | 9.2 | 2.21 | 97% |
| t-Bu | >20 | >4.8 | >99.9%（锁定构象） |
| Ph | 12.1 | 2.89 | 99% |
| OH | 2.5–4.2 | 0.6–1.0 | 70–84% |
| OMe | 2.5–3.3 | 0.6–0.8 | 77–82% |
| F | 0.6–1.0 | 0.15–0.25 | 56–63% |
| COOH | 5.9 | 1.41 | 92% |

**关键应用**：
- t-Bu 锁定构象 → 用作构象锚定基团 → 迫使邻位取代基取向
- OH 和 F 的 A 值很小 → 氢键和偶极效应可以反转构象偏好

### Newman 投影 → 构象能差定量

以 2025 初赛第 8 题的 α-氟代羰基为例：

```
anti 构象: C-F 与 C=O 反式 → 偶极-偶极排斥最小
syn 构象:  C-F 与 C=O 同侧 → 偶极-偶极排斥 ~33.6 kJ/mol
→ anti 比 syn 稳定 ~33.6 kJ/mol → anti 占绝对优势
→ 但反应从哪个面进攻取决于各自过渡态能量
→ 不要混淆"构象偏好"和"反应选择性"
```

## B.3 不对称合成策略速查 {#asymmetric-strategies}

### Felkin-Anh 模型（α-手性醛酮 + 亲核试剂）

```
前提：Cα 有手性中心，羰基的亲核加成

步骤：
1. 画出羰基 C=O 与 Cα 的 Newman 投影
2. 将 Cα 上的最大取代基置于与 C=O 垂直的位置（减少位阻）
3. 亲核试剂从位阻较小的一面进攻

判断规则：
- Cα 的 L（大基团）与 C=O 呈 ~90°（垂直）
- 亲核试剂从远离 L 的一侧进攻
- 如果 Cα 有电负性取代基（如 F, OR），偶极效应可能反转选择性
```

### Cram 螯合模型（α-烷氧基醛酮 + 金属）

```
前提：Cα 有 OR 基团且能螯合金属离子（如 TiCl₄, Zn²⁺）

与 Felkin-Anh 的关键区别：
→ Cram 螯合：金属螯合 C=O 和 Cα-OR → 固定构象 → OR 在 C=O 同侧
→ Felkin-Anh：无螯合 → 位阻主导

判断方法：
- 有螯合金属 → Cram 螯合模型
- 无螯合金属 → Felkin-Anh 模型
```

### Evans 助剂（噁唑烷酮法）

```
核心序列：
1. 手性噁唑烷酮 + 酰氯 → 酰基噁唑烷酮
2. LDA 或 NaHMDS 去质子化 → (Z)-烯醇负离子（Li⁺ 螯合固定构型）
3. 亲电试剂从位阻较小面进攻 → 高非对映选择性
4. LiOH/H₂O₂ 或 NaBH₄ 脱除助剂 → 手性羧酸或手性醇

Evans 助剂的"魔法"：
→ 噁唑烷酮同时提供：手性环境 + (Z)-烯醇选择性 + 面选择性
→ 苯基取代的噁唑烷酮（从苯丙氨酸得来）是最常用的
```

### Sharpless 不对称环氧化

```
体系：Ti(OiPr)₄ + 手性酒石酸酯（DET）+ tBuOOH + 烯丙醇

选择性：
(+)-DET → 环氧化从"上方"（面向你的纸面外）+ 对烯丙醇有面选择性
(-)-DET → 环氧化从"下方"

判断口诀：
→ 把烯丙醇画成 C=C 在水平面、CH₂OH 在右下角
→ (+)-DET：环氧化在纸面上方
→ (-)-DET：环氧化在纸面下方

适用范围：仅烯丙醇（需要 OH 导向）
```

## B.4 立体电子效应速查 {#stereoelectronic}

| 效应 | 定义 | 实例 | 竞赛考点 |
|:---|:---|:---|:---|
| **反式共平面** | E2 消除要求 H 和离去基处于反式共平面（180°） | 环己烷：只有 trans-diaxial 的 H 和 LG 才能 E2 | 环状底物的 E2 区域选择性：必须画椅式找反式共平面 H |
| **端基效应** | O-C-O 片段中，gauche 构象比 anti 更稳定 | 糖的异头碳：O-C-O 片段的孤对电子→σ* 给电子稳定 gauche | 糖的 α/β 构型偏好；缩醛的异头效应 |
| **Grobfragmentierung** | 反式共平面断裂产生三个片段 | 1,3-二取代环己烷的开环碎片化 | 必须在反式共平面位置 |
| **Baldwin 规则** | 环化反应的 exo/endo 和 trig/dig/tet 分类 | 5-exo-trig 有利；5-endo-trig 不利 | 给材料学习题中判断环化路径的合法性 |

### 端基效应详解

```
正常预期：O-C-O 片段中，两个 O 的孤对电子互相排斥 → anti 构象更稳定
实际观察：gauche 构象更稳定（异头效应）

原因：O 的孤对电子 n_O → 给到 C-O σ* 反键轨道 → 稳定化作用
      gauche 构象提供最佳 n→σ* 重叠角度
      anti 构象中 n 与 σ* 不共线 → 无此稳定化作用

后果：
- 糖的异头碳：电负性取代基倾向占据 axial 位（与常规 A 值预测相反）
- 缩醛的构象偏好也遵循此规则
```

## 六、解题套路：机理判断与选择性预测 {#problem-solving-routine}

### 套路 1：给出 Hammett 数据 → 反推机理

```
Step 1: 看 ρ 的符号
  ρ > 0 → 吸电子基加速 → 过渡态负电荷多于起始态
  ρ < 0 → 给电子基加速 → 过渡态正电荷多于起始态

Step 2: 看 |ρ| 的大小
  |ρ| > 2 → 强烈依赖 → 过渡态电荷集中 → "晚过渡态"
  |ρ| < 1 → 弱依赖 → 过渡态电荷分散 → "早过渡态"

Step 3: 看 σ vs σ⁺ 的相关性
  σ⁺ 相关更好 → 过渡态直接共轭到给电子基（碳正离子特征）
  σ 相关更好 → 诱导效应主导

Step 4: 综合 → 提出速控步的电荷分布 → 验证是否与推测机理一致
```

### 套路 2：给出 KIE 数据 → 判断速控步

```
一级 KIE > 2 → 该 C-H 键在速控步中断裂 → 速控步是：
  - SN1 的 C-H 邻位参与？
  - E2 的质子被拔？
  - 自由基攫氢？
  - 需结合其他证据判断

一级 KIE ≈ 1 → 该 C-H 键不在速控步中断裂 → 速控步是：
  - 碳正离子形成？
  - 离去基离去？
  - 构象变化？

二级 β-KIE → 超共轭参与过渡态稳定化 → 碳正离子特征
```

### 套路 3：预测反应选择性

```
拿到一个陌生反应 → 不要背"这个反应选什么"
                    → 从电子效应推到立体化学

1. 找亲核/亲电位点 → 画 FMO（HOMO-LUMO）
2. 看 FMO 系数 → 预测区域选择性
3. 画底物最优构象 → 判断位阻面
4. 检查立体电子效应 → 有没有反式共平面/端基效应的强迫
5. 综合 → 预测主产物
```

## 七、典型例题串讲 {#typical-examples}

### 例题 1：Bent 规则 + 构象分析综合（2025 初赛第 8 题）

**题目特征**：以氟化学为载体，系统考察 Bent 规则→构象分析→烯醇化学→KIE

**分析框架**：

```
Q1: 为什么 CH₂F₂ 中 ∠FCF < ∠HCH？
  → Bent 规则：F 电负性 > H → C-F 用更多 p 成分 → C-H 用更多 s 成分 → ∠HCH 增大

Q2: α-氟代羰基的 anti/syn 构象能差来源？
  → 偶极-偶极排斥：C-F 偶极 vs C=O 偶极
  → anti: C-F 与 C=O 反式 → 排斥最小 → 稳定
  → syn:  C-F 与 C=O 同侧 → 排斥 ~33.6 kJ/mol

Q3: 烯醇负离子的动力学 vs 热力学控制？
  → LDA（大位阻强碱）→ 动力学控制 → 攫取位阻较小的 α-H → 较少取代的烯醇负离子
  → NaOH（小位阻）→ 热力学控制 → 更稳定的烯醇负离子（更多取代的）为主

Q4: KIE 实验告诉我们什么？
  → k_H/k_D > 1 → C-H(D) 键在速控步断裂
  → 证明去质子化是速控步（而非碳负离子后续反应）
```

### 例题 2：FMO 推导周环选择性（37 届初赛第 8 题）

**题目特征**：电环化反应的立体化学——顺旋还是对旋？

**FMO 推导**：

```
底物：共轭三烯（6π 电子体系）
热反应条件

Step 1: 画出 HOMO (ψ₃)
  三烯的 ψ₃: + - + - + -（两端同号）

Step 2: 端位 p 轨道怎么旋转才能同号重叠？
  对旋(conrotatory): 两端同向转 → 同号重叠 → 允许
  顺旋(disrotatory): 两端反向转 → 异号重叠 → 禁阻

Step 3: 结论
  6π 电子 + 热反应 → 对旋 → 产物立体化学：两个端位同侧
  （4π 电子 + 热反应 → 顺旋 → 产物立体化学：两个端位异侧）

不背口诀的方法：
  数 π 电子 → 画 HOMO → 看两端符号 → 同号=对旋, 异号=顺旋
```

### 例题 3：Evans 助剂的不对称诱导

**题目特征**：给定 Evans 助剂 + 底物 → 预测产物绝对构型

**分析框架**：

```
Step 1: 画出酰基噁唑烷酮 → (Z)-烯醇负离子
  Li⁺ 螯合两个羰基氧 → (Z)-烯醇固定

Step 2: 确定面选择性
  手性噁唑烷酮的苄基挡住了下方（位阻面）
  → 亲电试剂从上方（位阻较小面）进攻

Step 3: 产物的绝对构型
  从上方进攻 → 新生成的手性中心为 R（依具体底物而定）

Step 4: 脱除助剂
  LiOH/H₂O₂ → 羧酸（保留构型）
  NaBH₄ → 醇（保留构型）
```

## 八点五、真题链与讲评顺序 {#exam-sequence}

- `第 1 题`：先讲“选择性冲突但信息不多”的题，目标是让学生先用电子效应 / 构象 / 轨道三层框架筛机理。
- `第 2 题`：再讲带有 `KIE / Hammett / 标记实验` 这类证据的题，目标是训练“证据在证明哪一步”。
- `第 3 题`：最后再讲需要把机理判断和立体结果一起落地的综合题，作为第四轮收口。
- 课堂顺序建议：`机理筛选题 → 证据判步题 → 立体/区域综合题`，不要一上来就讲最全的长题。

### Part B 推荐实跑真题（2026-06-19 收紧版）

| 课堂位置 | 推荐题 | 使用方式 | 主要目的 |
|:---|:---|:---|:---|
| 图后立刻练 | 2025 初赛第 8 题（α-氟代构象分析） | 只画优势构象和进攻面，不急着报答案 | 先建立“构象 ≠ 产物比”的课堂口径 |
| 讲后 1 题 | 2024 初赛第 10 题（亚砜立体化学） | 只抓翻转能垒、动态立体化学、构型保持/外消旋三个点 | 训练“手性中心也可能是动态的” |
| 压轴收口 | 2025 决赛第 9 题（四级碳手性中心） | 只保留 Newman 构象、邻基参与、环丙烷开环三条主线 | 把长题拆成可迁移模板 |

### 图后立刻练 / 讲后 1 题 / 课后 2 题

- 图后立刻练：给一题“两个机理都看似能走”的短题，只要求学生说出先调用哪条证据。
- 讲后 1 题：选一题带 `KIE / Hammett / 标记实验` 的真题，完整练“证据 → 速控步 → 主机理”。
- 课后 2 题：一题偏构象控制，一题偏轨道/选择性综合，要求都写“为什么不是另一条路”。
