## 零点六、课堂投影速查卡 {#classroom-quick-card}

**本页课堂入口：** 先让学生接受“人名反应不是词汇表，而是机理族地图”。

**先问四个问题：**

1. 最强线索在底物、试剂条件，还是关键中间体？
2. 这一步是在造亲核碳、造缺电子中心，还是在走协同/SET 路线？
3. 题目真正卡人的，是迁移方向、偶联对象，还是为什么这一步能发生而别的不能？
4. 如果去掉人名，你还能不能用“母机理”把它讲明白？

**一屏判断卡：**

- 先归族，再报名字。
- 先找统一母机理，再看特殊边界。
- 人名陌生不可怕，条件和中间体不会一起伪装。
- 冲刺题讲评必须补一句：“为什么不是另一个族？”

## 零点七、第二张教学图：人名反应分族映射板 {#second-teaching-figure}

**建议用途：** 这张图适合接在第一屏速查卡之后，用来把“人名很多”的心理压力降下来。核心不是多列名字，而是让学生看到同族反应共用同一种判断语言。

| 机理族 | 代表名字 | 先看什么 | 常见误判 |
|:---|:---|:---|:---|
| 羰基/烯醇负离子族 | Aldol, Mannich, Michael, Claisen | 供体是谁，受体是谁 | 把所有羰基题混成“亲核加成” |
| 重排族 | Pinacol, Beckmann, B-V, Favorskii | 迁移基团与驱动力 | 只背名字，不看迁移逻辑 |
| 周环族 | Diels-Alder, Claisen, Cope | 热/光条件与协同轨道 | 硬找中间体 |
| 金属有机/偶联族 | Grignard, Gilman, Wittig 等 | 碳负性来源与官能团兼容性 | 只记试剂，不记边界 |

**🧵 下一跳：** 学完八族后先看 [[#二点五、跨专题缝合表：缺电子中心 1,2-迁移统一视图 🔗|§二点五 跨专题缝合表]]——把第1族（碳正重排）、第2族（缺电子氮）、第6族（氧化插氧）缝成"同一判断链"，再进入 §三 跨族对比。

**板书跟讲顺序：**
- 先把题目归进某一“机理族”，不要先报名字。
- 再讲该族的共性按钮是什么。
- 最后再补具体人名，这样学生记的是结构化地图，不是散卡。

**本页课堂入口：** 先把学生从“背名字”切回“认机理族”。

**先问四个问题：**

1. 题干最强的信号是条件、底物骨架，还是关键中间体？
2. 这题更像重排、缩合、周环、光化学，还是金属偶联族？
3. 你能不能先说出“为什么不是另一个族”？
4. 归完族以后，统一母机理能不能直接推出产物大方向？

**一屏判断卡：**

- 人名是标签，机理族才是主干。
- 冲刺班最值钱的是跨族排除，不是继续堆名字。
- 陌生人名先看中间体和条件，不先想“有没有见过”。
- 每族至少保留一句统一母机理。

## 一、核心结论：为什么按机理族归类

**第38届决赛有机试题解析的明确结论**："许多有机人名反应实质相似"。

竞赛有机不是考你背了多少个人名——而是考你：**看到一个陌生反应，能不能从中间体类型和电子流向推断产物**。

**本专题的 8 族分类逻辑**：

```
反应物 → 中间体类型 → 反应走向 → 产物
                ↑
         这是分类的核心依据
```

| 分类依据 | 说明 |
|:---|:---|
| **按中间体** | 碳正离子？碳负离子？自由基？卡宾？氮宾？周环过渡态？ |
| **按电子流向** | 亲核→亲电？极性翻转？氧化还原？ |
| **按驱动力** | 芳香性过渡态？稳定产物（N₂/CO₂ 离去）？开环释放张力？ |

## 二、八大机理族速查

### 第 1 族：碳正离子 1,2-迁移重排 🟡

> **统一机理**：缺电子中心生成 → 相邻基团 1,2-迁移 → 更稳定碳正离子/羰基 → 后处理

| 反应 | 起始物 → 产物 | 迁移基团 | 驱动力 | 竞赛出现形式 |
|:---|:---|:---|:---|:---|
| **Wagner-Meerwein** | 醇 + H⁺ → 重排烯烃 | H, alkyl, aryl | 碳正离子稳定性↑ | 萜类/甾体骨架重排 |
| **Pinacol** | 邻二醇 + H⁺ → 酮 | alkyl, aryl, H | 羰基稳定 | 四取代乙二醇→频哪酮 |
| **Semipinacol** | α-羟基醇 → 扩环酮 | alkyl（环扩张） | 羰基+扩环 | 中环合成 |
| **Tiffeneau-Demjanov** | β-氨基醇 + HNO₂ → 扩/缩环酮 | alkyl（环扩张） | N₂ 离去驱动 | 环己酮→环庚酮 |
| **Wolff** | α-重氮酮 → 烯酮 → 羧酸衍生物 | alkyl, aryl（经卡宾） | N₂ 离去 | Arndt-Eistert 同系化 |

**一眼识别**："底物有 OH/NH₂/重氮 + 酸或 HNO₂ → 缺电子中心形成 → 邻位基团迁过来"

**已有深度内容** → [[专题-重排反应]]

### 第 2 族：缺电子氮重排 🟢

> **统一机理**：氮上构建离去基团 → 缺电子氮（氮宾类似物）→ R 基从 C 迁移到 N → 异氰酸酯中间体 → 水解得少一个碳的胺/酰胺

| 反应 | 起始物 → 中间体 | 产物 | 碳数变化 | 关键试剂 |
|:---|:---|:---|:---:|:---|
| **Beckmann** | 肟 → 酰胺 | RCONHR' | 不变 | H⁺/PCl₅/PPA |
| **Hofmann** | 酰胺 → 胺 | RNH₂ | **−1** | Br₂ + NaOH |
| **Curtius** | 酰基叠氮 → 胺 | RNH₂ | **−1** | Δ（热解） |
| **Lossen** | 羟肟酸 → 胺 | RNH₂ | **−1** | Δ 或碱 |
| **Schmidt** | 羧酸 + HN₃ → 胺 | RNH₂ | **−1** | H⁺ |
| **Schmidt**（酮） | 酮 + HN₃ → 酰胺 | RCONHR' | 不变 | H⁺ |

**一眼识别**："Beckmann=肟→酰胺（碳数不变），其他四个=从羰基化合物→少一个碳的胺"

**已有深度内容** → [[专题-重排反应]]

### 第 3 族：碳负离子缩合 🔵

> **统一机理**：烯醇负离子（或等效亲核碳）→ 进攻亲电羰基 → β-羟基/β-酮酯 → 脱水（可选）

| 反应 | 亲核组分 | 亲电组分 | 产物特征 | 碱 |
|:---|:---|:---|:---|:---|
| **Aldol** | 醛/酮 α-C | 醛/酮 C=O | β-羟基醛/酮 | NaOH, LDA |
| **Claisen** | 酯 α-C | 酯 C=O | β-酮酯 | NaOR（匹配醇） |
| **Dieckmann** | 二酯分子内 | 酯 C=O | 环状 β-酮酯 | NaOR |
| **Knoevenagel** | 活泼亚甲基 | 醛/酮 C=O | α,β-不饱和羰基 | 弱碱（哌啶） |
| **Perkin** | 酸酐 α-C | 芳醛 C=O | 肉桂酸衍生物 | RCO₂Na |
| **Stobbe** | 琥珀酸酯 α-C | 醛/酮 C=O | 亚烷基琥珀酸酯 | t-BuOK |
| **Michael** | 烯醇负离子 | α,β-不饱和羰基 | 1,5-二羰基 | 碱催化 |
| **Mannich** | 烯醇/烯胺 | 亚胺离子 | β-氨基羰基 | H⁺/碱 |

**一眼识别**："有羰基 α-H + 碱 + 另一个羰基 → 必是缩合族。Aldol=醛+醛，Claisen=酯+酯，Michael=α,β-不饱和+亲核"

### 第 4 族：极性翻转（Umpolung）🟣

> **统一机理**：正常亲电的羰基碳→通过临时转化变为亲核中心→与亲电试剂反应→水解恢复羰基

| 反应 | 翻转策略 | 亲核等价物 | 产物 |
|:---|:---|:---|:---|
| **安息香缩合** | CN⁻ 催化，醛→酰基阴离子等价物 | 氰醇中间体 | α-羟基酮 |
| **Stetter 反应** | 噻唑盐催化，醛→酰基阴离子 | Breslow 中间体 | 1,4-二羰基 |
| **Corey-Seebach** | 1,3-二噻烷（S,S-缩醛）→Li 化 | 二噻烷负离子 | 水解→醛/酮 |
| **Corey-Seebach 同系化** | 1,3-二噻烷 → RX 烷基化 → 水解 | "R⁻ 等价物" | 延长碳链的醛/酮 |

**一眼识别**："看到 CN⁻ + 醛（无 α-H）→ Benzoin；看到二噻烷 → Corey-Seebach Umpolung"

### 第 5 族：协同周环反应 🔴

> **统一机理**：环状过渡态 → 芳香性过渡态（4n+2 电子→加热允许；4n 电子→光照允许）→ 立体专一性产物

| 反应 | 类型 | 电子数 | 热/光条件 | 立体化学 |
|:---|:---|:---:|:---|:---|
| **Diels-Alder** | [4+2] 环加成 | 6π | 加热（同面-同面） | 顺式加成，endo 优先 |
| **1,3-偶极环加成** | [3+2] 环加成 | 6π | 加热 | 顺式加成 |
| **Alder-ene** | ene 反应 | 6π | 加热 | 烯丙基移位+H 转移 |
| **Cope 重排** | [3,3]-σ 迁移 | 6π | 加热 | 椅式过渡态优先 |
| **Claisen 重排** | [3,3]-σ 迁移 | 6π | 加热 | 椅式过渡态优先→芳构化驱动 |
| **电环化（丁二烯→环丁烯）** | 电环化 | 4π | 加热=顺旋 | 顺旋 vs 对旋由 WH 决定 |
| **电环化（己三烯→环己二烯）** | 电环化 | 6π | 加热=对旋 | 对旋 vs 顺旋由 WH 决定 |

**一眼识别**："没有离子中间体 + 环状过渡态 + 高度立体专一 = 周环族"

**已有深度内容** → [[专题-周环反应]]

### 第 6 族：氧化重排/插氧 🟠

> **统一机理**：过氧酸 O-插入 → 四中心过渡态或自由基 → 重排 → 酯/内酯/酚

| 反应 | 底物 | 氧化剂 | 产物 | 迁移能力顺序 |
|:---|:---|:---|:---|:---|
| **Baeyer-Villiger** | 酮 | RCO₃H（m-CPBA 最常见） | 酯/内酯 | H > 3°-alkyl > 2°-alkyl ≈ aryl > 1°-alkyl > CH₃ |
| **Dakin** | 邻/对羟基芳醛 | H₂O₂ + NaOH | 酚（脱甲酰基） | — |
| **Rubottom** | 硅基烯醇醚 | m-CPBA | α-羟基酮 | — |

**一眼识别**："酮 + 过氧酸 → 酯 = Baeyer-Villiger。迁移基团=与 C=O 反式共平面的那个。"

### 第 7 族：光化学反应 ⚪

> **统一机理**：光激发 → 激发态（n→π\* 或 π→π\*）→ 双自由基中间体 → 热反应无法实现的骨架重排
>
> **来源**：[[07-资料提炼/书籍提炼/提炼-ABOC-第10章-光谱与自由基|ABOC Ch.10 §10.2-10.3]]

#### 7.1 激发态基础

| 参数 | 基态 S₀ | 单线态 S₁ | 三线态 T₁ |
|:---|:---:|:---:|:---:|
| 电子自旋 | 配对 ↑↓ | 配对 ↑↓ | 平行 ↑↑ |
| 寿命 | — | 10⁻⁹–10⁻⁶ s | 10⁻⁶–10⁻³ s |
| 能量 | 最低 | 较高 | 中等（低于 S₁） |

**关键光物理过程**：
- **ISC**（系间窜越，Intersystem Crossing）：S₁ → T₁，自旋禁阻但可通过自旋-轨道耦合发生
- **荧光**（Fluorescence）：S₁ → S₀ + hν（快，~10⁻⁸ s），自旋允许
- **磷光**（Phosphorescence）：T₁ → S₀ + hν（慢，~10⁻³ s），自旋禁阻

> **竞赛要点**：题目中出现 hν 条件时，反应的**选择性来源于激发态电子分布与基态不同**，不能用基态热化学的直觉推断产物。

#### 7.2 经典光化学反应

| 反应 | 起始物 | 关键中间体 | 产物 | 条件 |
|:---|:---|:---|:---|:---|
| **Norrish I** | 酮 | α-断裂→酰基自由基 + 烷基自由基（双自由基） | 脱羰产物 + 自由基重组产物（如醛→烷烃+CO） | hν |
| **Norrish II** | 含 γ-H 的酮 | γ-H 夺取→1,4-双自由基→环化或断裂 | 环丁醇（环化路径）或 烯烃+烯醇（断裂路径，烯醇→酮） | hν |
| **Paternò-Büchi** | 羰基 + 烯烃 | 激发态 C=O + 烯→双自由基→闭环 | 氧杂环丁烷（oxetane） | hν |
| **di-π-methane** | 含两个 π 键的烯（如 1,4-二烯） | 双自由基重排 | 环丙烷衍生物（乙烯基环丙烷） | hν |

##### Norrish I 详解（α-断裂）

```
R-CO-R'  →[hν]→  R-CO• + •R'  （α-断裂产生酰基自由基+烷基自由基）
                  ↓
            R• + CO（脱羰）     ← 酰基自由基不稳定，易脱 CO
                  ↓
           R• + •R' → R-R'     （自由基重组）
           或 R• + 溶剂 H 供体 → R-H
```

**竞赛考点**：Norrish I 可作为**降碳策略**（通过脱羰去掉一个碳）。

##### Norrish II 详解（γ-H 夺取 → 六元环过渡态）

```
R-CO-CH₂-CH₂-CH₂-R'  →[hν]→  激发态 C=O（n→π*，双自由基性质）
                                    ↓
                            γ-H 被 O 夺取（六元环 TS）
                                    ↓
                            1,4-双自由基
                              ↙          ↘
                      环化（C-C 成键）    断裂（C-C 断裂）
                          ↓                  ↓
                       环丁醇          CH₂=CH-R' + CH₂=C(OH)-R
                                            ↓
                                    烯醇→酮（互变异构）
```

**竞赛考点**：Norrish II 的**γ-H 是必须的**——没有 γ-H 的酮不能发生 Norrish II。产物分两路：环化得环丁醇，断裂得烯烃+酮。

##### Paternò-Büchi 详解（[2+2] 光化学环加成）

```
C=O（激发态，双自由基性质） + C=C  →[hν]→  氧杂环丁烷（四元环醚）
```

**关键**：这是**光允许、热禁阻**的 [2+2] 环加成（基态 C=O 与 C=C 的 [2+2] 热反应 Woodward-Hoffmann 禁阻）。

##### di-π-methane 重排

```
Ar-C(C=C)=C  →[hν]→  双自由基重排 → 乙烯基环丙烷骨架
```

**应用**：从 1,4-二烯合成环丙烷衍生物——在天然产物全合成中用于构建三元环。

#### 7.3 光氧化还原催化（进阶）

> **来源**：[[07-资料提炼/书籍提炼/提炼-ABOC-第10章-光谱与自由基|ABOC Ch.10 §10.3]]

**与普通光化学的区别**：
- **普通光化学**：底物直接吸收光 → 激发态反应
- **光氧化还原催化**：光催化剂（PC）吸收光 → PC\* → **单电子转移（SET）** 给/从底物

**核心催化剂**：

| 催化剂 | 激发态还原电位 E\*(Red) | 特点 |
|:---|:---:|:---|
| **Ru(bpy)₃²⁺** | –0.81 V | 经典，水溶性好 |
| **Ir(ppy)₃** | –1.73 V | 更强还原性，近年更常用 |

**还原淬灭循环**（以 Ru(bpy)₃²⁺ 为例）：
```
Ru(II) + hν → *Ru(II)（激发态）
*Ru(II) + 给体（如 Et₃N）→ Ru(I) + 给体•⁺（还原淬灭，SET）
Ru(I) + 底物 A → Ru(II) + A•⁻（单电子还原底物，催化剂再生）
A•⁻ → 产物（自由基中间体继续反应）
```

> **竞赛识别**：反应条件中有 **hν + Ru/Ir 配合物（或曙红 Y 等有机染料）** → 光氧化还原催化，必须写 SET 机理，不能写极性机理。

#### 7.4 热反应 vs 光化学——根本区别

| 维度 | 热反应 | 光化学反应 |
|:---|:---|:---|
| **能量来源** | 分子碰撞（动能） | 光子吸收（电子激发） |
| **反应性** | 基态电子分布决定 | 激发态电子分布决定 |
| **选择性** | 热力学控制或动力学控制 | **激发态特有的选择性**——用基态规律推不出来 |
| **[2+2] 环加成** | **禁阻**（WH 规则） | **允许**（光化学 WH 规则相反） |
| **典型条件** | Δ（加热），酸/碱/催化剂 | **hν**（光照），通常无其他试剂 |

**一眼识别**："题目明确说光照（hν）→ 必是光化学族。Norrish I=α-断裂脱羰，Norrish II=γ-H 夺取→环丁醇或断裂，Paternò-Büchi=羰基+烯烃→氧杂环丁烷"

### 第 8 族：金属催化交叉偶联 🔵

> **统一机理**：氧化加成（M → M⁺²，配位数+2）→ 转金属化（配体交换）→ 还原消除（M⁺² → M，配位数−2）→ 催化循环闭合
>
> **来源**：[[07-资料提炼/书籍提炼/提炼-Clayden-第40章-金属有机化学|Clayden Ch.40]] + [[07-资料提炼/书籍提炼/提炼-ABOC-第8章-过渡金属催化|ABOC Ch.8]]

#### 8.1 催化循环基础——四步曲

```
① 氧化加成（Oxidative Addition）
   Pd(0) + R-X → R-Pd(II)-X
   氧化态 +2，配位数 +2
   要求：Pd(0) 络合不饱和（14e 或 16e）
   底物：Ar-I > Ar-Br > Ar-Cl >> Ar-F
         Ar-OTf 也可用
         不能用有 β-H 的烷基卤（会 β-H 消除！）

② 转金属化（Transmetallation）
   R-Pd(II)-X + R'-M → R-Pd(II)-R' + M-X
   有机金属试剂 R'-M 将 R' 转给 Pd
   通常是慢步骤（速率决定步）
   Suzuki 中碱活化硼酸：R-B(OH)₂ + OH⁻ → R-B(OH)₃⁻（更亲核的"ate"络合物）

③ 还原消除（Reductive Elimination）
   R-Pd(II)-R' → R-R' + Pd(0)
   氧化态 −2，配位数 −2
   两个配体必须顺式排列！反式不反应！
   Pd(0) 再生，催化循环闭合

④ β-H 消除（β-Hydride Elimination）——特殊步骤
   仅存在于 Heck 反应中（替代转金属化+还原消除）
   Pd-CH₂-CH₂-R → Pd-H + CH₂=CH-R
   要求：Pd 有空位（16e）+ β-H 与 Pd 处于 syn 构型
   碱脱 HX 再生 Pd(0)：HPd(II)X + Base → Pd(0) + Base·HX
```

> **核心概念**：Pd 在整个循环中穿梭于 Pd(0) ↔ Pd(II)。Pd(0)（d¹⁰）亲核性强、易氧化加成；Pd(II)（d⁸）亲电性强、易还原消除。催化循环的本质就是**用 Pd 在两个氧化态之间来回切换**。

#### 8.2 五大偶联反应详解

| 反应 | 金属 | 亲核组分 R'-M（识别码） | 亲电组分 R-X | 产物键 | 特征步骤 |
|:---|:---|:---|:---|:---|:---|
| **Suzuki** | Pd(0) | **B**（硼酸 R-B(OH)₂） | Ar'-X | C-C | 碱活化→"ate"络合物→转金属（速控步） |
| **Negishi** | Pd(0)/Ni(0) | **Zn**（有机锌 R-ZnX） | R'-X | C-C | 转金属快，官能团耐受好；缺电子卤代物反应更好 |
| **Heck** | Pd(0) | **烯烃**（末端烯） | Ar-X | C=C | 无转金属！用迁移插入 + β-H 消除替代 |
| **Sonogashira** | Pd(0)/Cu(I) | **炔**（端炔 RC≡CH，CuI 共催化） | Ar-X | C≡C-C | 炔基亚铜（RC≡CCu）转金属→还原消除 |
| **Buchwald-Hartwig** | Pd(0) | **N**（胺 R₂NH） | Ar-X | C-N | 胺配位→碱脱 HX→还原消除形成 Ar-N 键 |

##### Suzuki 偶联（最常用——"最温柔的交叉偶联"）

**优点**：硼酸毒性低、对水/空气稳定、条件温和、官能团耐受好。

**催化循环**：
```
① 氧化加成：Pd(0) + Ar'-Br → Ar'-Pd(II)-Br
② 碱活化：Ar-B(OH)₂ + NaOH → Ar-B(OH)₃⁻ Na⁺（"ate"络合物，更亲核）
③ 转金属：Ar'-Pd(II)-Br + Ar-B(OH)₃⁻ → Ar'-Pd(II)-Ar + B(OH)₃ + Br⁻
④ 还原消除：Ar'-Pd(II)-Ar → Ar-Ar' + Pd(0)（再生）
```

**配体**：PPh₃（经典）/ SPhos, XPhos（大位阻→加速氧化加成）。

##### Negishi 偶联（高反应性）

**优点**：有机锌试剂反应性高、选择性好，对缺电子卤化物比 Suzuki 更有效。

**催化循环**：
```
① 氧化加成：Pd(0) + R'-X → R'-Pd(II)-X
② 转金属：R'-Pd(II)-X + R-ZnX → R'-Pd(II)-R + ZnX₂
③ 还原消除：R'-Pd(II)-R → R-R' + Pd(0)
```

##### Heck 反应（唯一走 β-H 消除的偶联）

**Heck 与 Suzuki/Negishi 的根本区别**：没有转金属化步骤！用迁移插入 + β-H 消除替代。

**催化循环**：
```
① 氧化加成：Pd(0) + Ar-X → Ar-Pd(II)-X
② 烯烃配位 + 迁移插入（碳钯化，carbopalladation）：
   Ar-Pd(II)-X + CH₂=CH-R → Ar-CH₂-CH-Pd(II)-X（Pd 连在取代少的碳上）
                              |
                              R
③ β-H 消除（syn 消除）：
   Ar-CH₂-CH-Pd(II)-X → Ar-CH=CH-R + H-Pd(II)-X
④ 碱再生：H-Pd(II)-X + Et₃N → Pd(0) + Et₃NH⁺X⁻
```

**区域选择性**：
- 缺电子烯烃（丙烯酸酯等）：Pd 加到取代少的碳 → 产物为反式（E）烯烃
- 富电子烯烃（烯基醚等）：进攻发生在取代基一端 → 1,1-双取代产物

##### Sonogashira 偶联（炔烃专属）

**CuI 的作用**：末端炔烃在 CuI/碱条件下形成**炔基亚铜**（RC≡CCu），再与 Pd(II) 中间体转金属。

**催化循环**：
```
① 氧化加成：Pd(0) + Ar-X → Ar-Pd(II)-X
② 铜循环（并行）：RC≡CH + CuI →[碱]→ RC≡CCu + HI
③ 转金属：Ar-Pd(II)-X + RC≡CCu → Ar-Pd(II)-C≡CR + CuX
④ 还原消除：Ar-Pd(II)-C≡CR → Ar-C≡C-R + Pd(0)
```

**应用**：烯二炔抗生素合成、Bergmann 环化前体。

##### Buchwald-Hartwig 胺化（C-N 偶联）

**与传统 SₙAr 的对比**：

| 维度 | Buchwald-Hartwig | SₙAr |
|:---|:---|:---|
| **离去基团活性** | I > Br > Cl >> F | **F > Cl > Br > I**（完全相反！） |
| **区域要求** | 无限制 | 需要邻/对位吸电子基 |
| **机理** | Pd 催化循环 | 加成-消除（Meisenheimer 中间体） |

**催化循环**：
```
① 氧化加成：Pd(0) + Ar-X → Ar-Pd(II)-X
② 胺配位：Ar-Pd(II)-X + R₂NH → Ar-Pd(II)-NHR₂ + X⁻
③ 碱脱 HX：Ar-Pd(II)-NHR₂ + Base → Ar-Pd(II)-NR₂ + Base·HX
④ 还原消除：Ar-Pd(II)-NR₂ → Ar-NR₂ + Pd(0)
```

**关键配体**：大位阻双膦配体（Xantphos、BINAP）促进还原消除。

#### 8.3 配体与条件速查

| 维度 | Suzuki | Negishi | Heck | Sonogashira | B-H |
|:---|:---|:---|:---|:---|:---|
| **催化剂前体** | Pd(PPh₃)₄ | Pd(PPh₃)₄ | Pd(OAc)₂/PPh₃ | Pd(PPh₃)₂Cl₂ | Pd₂(dba)₃ |
| **配体** | PPh₃/SPhos | PPh₃ | PPh₃/P(o-tol)₃ | PPh₃/CuI | Xantphos/BINAP |
| **碱** | Na₂CO₃/NaOH | — | Et₃N/NaOAc | Et₃N/哌啶 | t-BuONa/Cs₂CO₃ |
| **溶剂** | 含水混合溶剂 | THF/Et₂O | DMF/MeCN | THF/Et₃N | 甲苯/二氧六环 |

#### 8.4 一眼识别口诀

**"Pd 催化剂 + 两种底物 → 交叉偶联。看亲核组分识反应"**：
- B（硼酸）→ **Suzuki**（最常用，最温柔）
- Zn（有机锌）→ **Negishi**（反应性最高）
- 烯烃（末端烯，无有机金属试剂）→ **Heck**（走 β-H 消除，不是转金属化）
- 炔（端炔 + CuI）→ **Sonogashira**（Pd/Cu 双催化）
- N（胺）→ **Buchwald-Hartwig**（C-N 偶联，大位阻配体必备）

## 二点五、跨专题缝合表：缺电子中心 1,2-迁移统一视图 🔗

> **定位**：第四轮冲刺支架「跨专题缝合线」的核心落地。前三轮里，Wagner-Meerwein 在重排章、Beckmann 在含氮章、Baeyer-Villiger 在氧化章——它们分属三个不同专题（[[专题-重排反应]] / [[专题-人名反应系统归类]] / 氧化还原），但底层逻辑完全相同：**缺电子中心生成 → 相邻基团 1,2-迁移 → 稳定化后处理**。这张表把分散在三处的"同一件事"缝成一张可对比的单一视图。

### 统一机理骨架

```
前体 ──[条件]──→ 缺电子中心 ──[1,2-迁移]──→ 稳定中间体 ──[后处理]──→ 产物
  ↑                   ↑                ↑               ↑
 变化的是：        缺电子的原子      迁移的基团      捕获方式
 不变的是：        邻位必须有可迁移基团（H/alkyl/aryl）
                  驱动力 = 迁移后的产物比迁移前更稳定
```

### 三族统一对比：碳正·氮宾·氧正

| 维度 | 碳正离子 1,2-迁移 | 缺电子氮重排 | 氧化插氧重排 |
|:---|:---|:---|:---|
| **对应族** | **第1族** | **第2族** | **第6族** |
| **缺电子中心** | 碳正离子 R₃C⁺ | 缺电子氮（氮宾等价物） | Criegee 中间体的缺电子 O |
| **怎么生成的** | -OH 质子化→脱 H₂O / -N₂⁺→脱 N₂ | 酰胺 + Br₂/NaOH 活化 N / 酰基叠氮 Δ 脱 N₂ | 酮 + m-CPBA → 四中心 Criegee 加合物 |
| **迁移基团** | H > Ph > 3°-alkyl > 2° > 1° > CH₃ | 与 N 相邻的 R 基（烷基/芳基） | 与 C=O **反式共平面**的基团（同上排序） |
| **稳定化方式** | 迁移后→脱 H⁺（烯烃）/ +H₂O（醇/酮） | 异氰酸酯 R-N=C=O → 水解得 **少一碳** 的胺 | O 插入 C-C 键间→酯/内酯 |
| **碳数变化** | 不变（骨架重排） | **−1**（Hofmann/Curtius/Lossen/Schmidt） | 不变（O 插在 C-C 之间） |
| **一眼识别** | 酸 + 醇/重氮 | Br₂/NaOH 或 HN₃ 或 Δ + 羰基化合物 | 酮 + 过氧酸（m-CPBA） |
| **代表反应** | Wagner-Meerwein, Pinacol, Tiffeneau-Demjanov | Beckmann, Hofmann, Curtius, Lossen, Schmidt | Baeyer-Villiger, Dakin |
| **典型真题** | 萜类骨架重排（38届决赛） | von Richter + Beckmann 组合（38届决赛） | 甾体 Baeyer-Villiger |

### 缝合后的认知转变

| 学生原来的认知（R3 状态） | 缝合后的认知（R4 目标） |
|:---|:---|
| Wagner-Meerwein = 醇 + 酸 → 重排烯烃（重排章） | **缺电子碳** 1,2-迁移——碳正离子版 |
| Beckmann = 肟 → 酰胺（含氮章） | **缺电子氮** 1,2-迁移——氮宾版 |
| Baeyer-Villiger = 酮 + m-CPBA → 酯（氧化章） | **缺电子氧** 1,2-迁移——氧正版 |
| "这三题考的是三个不同知识点" | "**同一判断链**：找缺电子中心 → 找邻位迁移基团 → 推稳定产物" |

### 课堂使用方式

1. **先平行展示三个代表反应**（各≤30 秒），让学生自己发现"形状一样"
2. **亮出统一骨架图**：缺电子中心 → 1,2-迁移 → 稳定化
3. **让学生填表**：给一个新反应（如 Schmidt 羧酸版），问——缺电子中心在哪？什么在迁移？怎么稳定化？
4. **穿插一句口诀**：*"缺电子就近借邻居——碳正借烷基，氮正借碳链，氧正借共平面那个"*

## 四、解题套路：人名反应推断三步法

### Step 1：从条件识别机理族
- 有 Pd/Ni/Cu 催化剂 → **第8族**（交叉偶联）。进一步看亲核组分：B→Suzuki, Zn→Negishi, 烯烃（无有机金属）→Heck, 炔+CuI→Sonogashira, 胺→Buchwald-Hartwig
- 有 hν（光照）+ Ru/Ir 催化剂 → **第7族**（光氧化还原催化，SET 机理）
- 有 hν（光照）无金属催化剂 → **第7族**（普通光化学：Norrish/Paternò-Büchi/di-π-methane）
- 有碱 + 羰基化合物 → **第3族**（缩合）或 **第4族**（Umpolung——看是否有 CN⁻ 或二噻烷）
- 有酸/H⁺ + 醇/重氮 → **第1族**（碳正离子重排）
- 有 Br₂+NaOH/HN₃/Δ → **第2族**（缺电子氮重排）
- 有过氧酸（m-CPBA）→ **第6族**（氧化重排）
- 无离子条件 + 高度立体专一 → **第5族**（周环）

### Step 2：查本族统一机理→画中间体
- 每族有统一机理（见上文各族第一条）
- 画出关键中间体（碳正离子/烯醇负离子/环状过渡态/氧化加成中间体）
- **依据 KP**：各反应对应的知识点文件

### Step 3：追踪电子流向→写产物
- 从中间体继续：迁移→消除→加成→水解→芳构化
- **依据 KP**：[[1,2-迁移与重排]]、[[直接加成与共轭加成]]
- ☐ 机理族已识别 ☐ 中间体已画出 ☐ 电子流向已追踪 ☐ 产物已写出

## 六点五、真题链与讲评顺序 {#exam-sequence}

- `第 1 题`：先选一题”熟名字但容易归错族”的题（推荐 §七 A1 碳正重排或 A3 Beckmann），先把”机理族 > 人名标签”的口径立住。
- `第 2 题`：再讲一题跨族排除题（推荐 §七 B2 重排+缩合 或 A12 F-C 反应），让学生说明”为什么不是周环 / 重排 / 缩合族”。
- `第 3 题`：最后讲光化学或金属偶联的新口子题（推荐 §七 A11 Grignard 或 B1 Norrish），作为第四轮新增内容收口。
- 课堂顺序建议：`熟反应重归类 → 跨族排除 → 新族补口`，这样学生更容易建立母类框架。

### 图后立刻练 / 讲后 1 题 / 课后 2 题（§七联动版）

- 图后立刻练：给 3 个熟反应名（从 §七 C1/C2/C4/C5 中选），让学生只做”归族 + 排除一族”的快判。
- 讲后 1 题：选一题跨族干扰最强的真题（推荐 §七 B2 或 A12），要求学生写出”归族理由 + 排除理由”。
- 课后 2 题：一题光化学/周环（§七 A10），一题金属偶联（§七 A11），让学生把第四轮新增族也并入 8 族母表。
