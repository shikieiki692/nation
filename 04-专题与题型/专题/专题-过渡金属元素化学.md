## 零点六、课堂投影速查卡 {#classroom-quick-card}

**本课课堂入口：** 本页是"地图"，三个分专题页是"教材"。过渡金属 30+ 种元素，竞赛常考约 15 种——先帮学生建立"三段地图"，知道每段考什么题型、出什么信号，再按需进入分专题深挖。不要从"第一过渡系从 Sc 开始"讲起——那是元素周期表，不是教学逻辑。

**先问三个问题：**

1. 题目考的是过渡金属的"氧化态多变性"还是"配合物稳定性"？— 前者走第一分段，后者走第二分段
2. 颜色信号来自 d-d 跃迁还是电荷转移？— d⁰/d¹⁰ 无色，d¹-d⁹ 有色（但强度取决于自旋允许/禁阻）
3. 推断题中"无色溶液"排除了谁、保留了谁？— d⁰: Ti⁴⁺/V⁵⁺/Sc³⁺；d¹⁰: Zn²⁺/Cu⁺/Ag⁺/Hg²⁺

**三段地图速查：**

```
过渡金属总览 → 三段地图
    │
    ├─ 第一分段（Sc Ti V Cr Mn）
    │   ├─ 核心考点：氧化态多样性、高价氧化性
    │   ├─ 颜色信号：Cr³⁺(绿/灰绿)、CrO₄²⁻(黄)、Cr₂O₇²⁻(橙)、MnO₄⁻(紫)
    │   ├─ 介质敏感性：Mn 的还原产物随 pH 变（酸性Mn²⁺/中性MnO₂/碱性MnO₄²⁻）
    │   └─ 进入 → [[专题-过渡金属（一）钛钒铬锰]]
    │
    ├─ 第二分段（Fe Co Ni Cu Zn）
    │   ├─ 核心考点：配合物、生物无机、稳定常数
    │   ├─ 颜色信号：Fe(SCN)²⁺(血红)、[Cu(NH₃)₄]²⁺(深蓝)、Co²⁺/Co³⁺(粉红↔蓝)
    │   ├─ 磁性判断：Fe²⁺(d⁶高自旋4未成对)、Ni²⁺(d⁸ 2未成对)、Zn²⁺(d¹⁰反磁)
    │   └─ 进入 → [[专题-过渡金属（二）铁钴镍铜锌]]
    │
    └─ 第三分段（Ag Au Hg Mo W）
        ├─ 核心考点：软硬酸碱、矿物溶解性、重元素相对论效应
        ├─ 颜色信号：Ag⁺(无色)、Au(金黄-相对论)、Hg²⁺(无色)
        ├─ HSAB 应用：Ag⁺(软酸)+I⁻(软碱)→AgI↓；Hg²⁺(软酸)+S²⁻(软碱)→HgS↓
        └─ 进入 → [[专题-过渡金属（三）银金汞钼钨]]
```

**过渡金属颜色速判——d 电子构型法：**

| dⁿ | 无色/浅色 | 有色 | 原因 |
|:---:|:---|:---|:---|
| d⁰ | Sc³⁺/Ti⁴⁺/V⁵⁺ | — | 无 d 电子→无 d-d 跃迁 |
| d⁵(高自旋) | Mn²⁺(极淡粉) | Fe³⁺(黄褐) | 自旋禁阻→颜色浅 |
| d¹⁰ | Zn²⁺/Cu⁺/Ag⁺/Hg²⁺ | — | d 轨道全满→无 d-d 跃迁 |
| 其他 d¹-d⁹ | — | 各有特征色 | d-d 跃迁允许 |

**课堂三问：** ① 考氧化态还是配合物？→ 决定进哪个分段 ② d 电子数→颜色/磁性能自洽吗？③ 介质条件→产物对吗？

## 八、真题入口与讲评顺序 {#exam-guide}

### 真题入口使用建议

- 过渡金属真题极少单独考"某一个元素"——几乎都嵌在**情境推断题**或**配合物/晶体综合题**中。因此本页真题入口不按元素排列，而是按"推断信号 → 元素锁定 → 性质验证"三层训练。
- **基础班**：先用一两道"颜色/磁矩/氧化态"信号题，帮学生建立"看到绿色溶液先想 Ni²⁺/Cr³⁺，看到紫色先想 MnO₄⁻/Cr³⁺"的条件反射。
- **提高班/冲刺班**：过渡金属真题的讲评重点是**推断题的排除逻辑**——不是"这道题答案是 Fe"，而是"为什么不是 Cr / Mn / Co / Ni / Cu"。
- 真题应优先从三个分专题页（[[专题-过渡金属（一）钛钒铬锰]]、[[专题-过渡金属（二）铁钴镍铜锌]]、[[专题-过渡金属（三）银金汞钼钨]]）各自的相关真题区调取，本页只做"跨段联合推断"的排题指导。

### 真题链与讲评顺序 {#exam-sequence}

- `第 1 题`：**单元素信号识别题**（颜色/磁矩/氧化态 → 锁定元素）。课堂用途：热启动，建立"看到信号→想到元素"的第一层反射。
- `第 2 题`：**双元素对比推断题**（两个过渡金属同时出现 → 用排除法区分）。课堂用途：主讲题，训练"不只是一一对应，而是交叉排除"。
- `第 3 题`：**跨族联合推断题**（过渡金属 + 主族元素 + 配体信息 → 多步推断）。课堂用途：收束题，验证学生能否在混合信号中不丢失过渡金属线索。
- 课堂顺序建议：`单元素信号 → 双元素排除 → 跨族联合`，不要一上来就做最复杂的多金属推断。
