## 零点六、课堂投影速查卡 {#classroom-quick-card}

**本页课堂入口：** 先抓“缺电子 / 两性 / 惰性电子对 / C-Si 分家”四条主线，不要试图一页背完两族所有细节。

**先问四个问题：**

1. 题目落点是硼族缺电子性，还是碳族价态与结构差异？
2. 这里需要先判两性、Lewis 酸，还是先判 `Sn(II)/Pb(IV)` 的氧化还原方向？
3. 题目是在问结构类型、硅酸盐比例，还是实验信号与沉淀现象？
4. 当前最值得强调的反常点是 `BF3/BCl3`、`CCl4/SiCl4`，还是 `Sn/Pb` 价态稳定性？

**一屏判断卡：**

- 硼先想缺电子和 3c-2e，铝先想两性，锡铅先想惰性电子对。
- `C` 和 `Si` 同族但不能硬类推，尤其是 π 键能力和氯化物水解性。
- 硅酸盐题先看 `Si:O` 比例，再倒推岛/链/层/架。
- 重元素题最后一定复查：高价是在做氧化剂，还是低价在做还原剂。

## 一、核心结论汇总 {#sec-conclusions}

**必须记住：**

1. **B 的缺电子性**：B 只有 3 个价电子，倾向于形成 4 键（如 BF₄⁻），通过 **3c-2e 桥键**（B₂H₆ 中的 B—H—B）、离域 π 键（BCl₃ 的 π₄⁶）或寡聚结构稳定缺电子中心。B 与 Si 遵循对角规则，性质相似。

2. **Al 的两性贯穿始终**：单质 Al、Al₂O₃、Al(OH)₃ 均具两性；Al³⁺ 水解使溶液呈酸性（pKa₁ ≈ 5.0）；无水 AlCl₃ 为共价二聚体（Al₂Cl₆，含 3c-4e Al—Cl—Al 桥），是 Lewis 酸催化剂。

3. **C/Si 的本质差异**：C 可形成 sp²/sp π 键（石墨、苯），Si 的 pπ-pπ 键极弱，故无"硅苯"；CO₂ 是分子气体，SiO₂ 是三维共价网络固体；CCl₄ 不水解，SiCl₄ 剧烈水解。

4. **Sn/Pb 的惰性电子对效应**：第六周期 Pb 的 6s² 电子不易参与成键，Pb(II) 稳定，Pb(IV) 强氧化性（PbO₂ + 4HCl → PbCl₂ + Cl₂↑ + 2H₂O）；Sn(II) 是还原剂，Sn(IV) 稳定。

5. **Tl⁺ 的异常稳定性**：III A 族自上而下 +1 价稳定性递增，Tl⁺(6s²) 比 Tl³⁺ 更稳定，Tl₂O₃ 是强氧化剂。

**最高频决策路径：**

```mermaid
flowchart TD
  A[题目条件] --> B{涉及哪族元素?}
  B -->|B/Al/Ga/In/Tl| C{判断缺电子或两性}
  C -->|缺电子化合物| D[查 B的3c-2e桥键 / BF₃的Lewis酸性 / 硼烷Wade规则]
  C -->|两性反应| E[查 Al的两性 / 氢氧化物溶解性]
  B -->|C/Si/Ge/Sn/Pb| F{判断价态或结构}
  F -->|同素异形体/晶体| G[查 碳的同素异形体 / 二氧化硅结构]
  F -->|氧化还原/价态| H[查 惰性电子对效应: Pb(IV)氧化性 / Sn(II)还原性]
```

## 三、解题套路 / 决策流程 {#sec-strategies}

### 套路 1：p 区元素推断题（颜色 + 沉淀 + 价态变化）

**Step 1：锁定元素族**
- **操作**：根据"两性""缺电子""惰性电子对"等关键词缩小范围
- **依据 KP**：[[硼族元素]]、[[碳族元素]]
- **检查点**：☐ 是否涉及 B/Al 的两性或缺电子性？☐ 是否涉及 Sn/Pb/Tl 的价态异常？

**Step 2：判断价态稳定性**
- **操作**：第六周期 p 区元素（Tl、Pb、Bi）优先考虑 +1/+2/+3 低价稳定；Sn 的 +2 价有还原性，+4 价稳定
- **依据 KP**：[[锡]]、[[碳族元素]]
- **检查点**：☐ 是否写出正确的氧化还原半反应？☐ 是否考虑惰性电子对效应？

**Step 3：验证两性或缺电子特征**
- **操作**：若涉及酸碱两性，检验该物质是否同时与强酸和强碱反应；若涉及缺电子，检查是否有 3c-2e 桥键或 Lewis 酸行为
- **依据 KP**：[[铝]]、[[硼化学]]
- **检查点**：☐ 两性反应方程式是否配平？☐ 配位数是否合理（B 常为 3 或 4）？

```mermaid
  I -->|水| L[CO₂ + H₂O ⇌ H₂CO₃]
```

## 四、反应机理拆解（含检查表） {#sec-mechanism}

> 本专题以元素化学为主，机理部分聚焦于氧化还原电子转移与缺电子化合物的成键机理。

### 4.1 PbO₂ 氧化 Cl⁻ 的电子转移

- **攻击位点**：Cl⁻ 的孤对电子进攻 Pb(IV) 中心
- **形成键**：Pb—Cl 配位中间体
- **断裂键**：Pb—O 键断裂，Cl—Cl 键形成
- **电子流**：Pb(IV) + 2e⁻ → Pb(II)；2Cl⁻ → Cl₂ + 2e⁻
- **学生任务**：判断 PbO₂ 在酸性还是碱性条件下氧化性更强？（答：酸性，因 H⁺ 促进 Pb—O 键断裂）
- **检查表**：
  - ☐ Pb 的氧化态从 +4 降至 +2
  - ☐ Cl 的氧化态从 -1 升至 0
  - ☐ 电荷守恒（酸性介质补 H⁺）

### 4.2 B₂H₆ 中 3c-2e 桥键的形成

- **攻击位点**：BH₃ 中 B 的 empty p 轨道接受另一 BH₃ 的 H⁻ 配位
- **形成键**：B—H—B 三中心两电子键
- **断裂键**：无（二聚体形成）
- **电子流**：2 个电子由 2 个 B 和 1 个 H 共享
- **学生任务**：计算 B₂H₆ 中 2c-2e 键和 3c-2e 键的数目（答：4 个端 B—H 为 2c-2e，2 个桥 B—H—B 为 3c-2e）
- **检查表**：
  - ☐ B 的价电子总数 = 2 × 3 = 6
  - ☐ H 的价电子总数 = 6 × 1 = 6
  - ☐ 总电子数 = 12，用于成键 = 12
  - ☐ 4 个 B—H 端键用 8 电子，剩余 4 电子形成 2 个 3c-2e 键

## 五、典型例题串讲 {#sec-examples}

### 例题 1：B/Al 缺电子与两性推断（综合推断题）

**题目：**
某无色晶体 A 溶于水得弱酸性溶液，加入 NaOH 至过量产生白色沉淀 B，B 溶于过量 NaOH 得无色溶液 C。向 C 中通入 CO₂ 又析出白色沉淀 B。晶体 A 的熔融物与 CoO 共热得蓝色玻璃珠 D。推断 A、B、C、D 的化学式，并写出关键反应方程式。

**分析：**
1. "弱酸性溶液" + "白色沉淀溶于过量 NaOH" → 提示含 Al³⁺ 或 B 的化合物
2. "熔融物与 CoO 共热得蓝色玻璃珠" → **硼砂珠试验**，这是 B 的特征反应
3. 综合判断 A 为硼砂 Na₂B₄O₇·10H₂O，而非铝盐（铝盐无硼砂珠反应）

**解答：**
- A = Na₂B₄O₇·10H₂O（硼砂）
- B = H₃BO₃（硼酸，白色沉淀/晶体）
- C = Na[B(OH)₄]（四羟基合硼酸钠）
- D = Co(BO₂)₂（偏硼酸钴，蓝色）

关键方程式：
1. Na₂B₄O₇ + 7H₂O → 2NaOH + 4H₃BO₃（或写作 [B₄O₅(OH)₄]²⁻ 的水解）
2. H₃BO₃ + NaOH → Na[B(OH)₄]（B 的缺电子性，接受 OH⁻ 配位）
3. 2Na[B(OH)₄] + CO₂ → 2H₃BO₃↓ + Na₂CO₃ + H₂O（弱酸制更弱酸）
4. Na₂B₄O₇ + CoO → Co(BO₂)₂ + 2NaBO₂（熔融，蓝色玻璃珠）

**反思：**
- 硼砂珠试验是 B 的专属鉴别反应，Al 无此性质
- H₃BO₃ 是一元弱酸（K = 5.4 × 10⁻¹⁰），酸性来自与 H₂O 配位而非电离
- 注意区分 Al³⁺ 和 B 的两性体系：Al 最终生成 [Al(OH)₄]⁻，B 生成 [B(OH)₄]⁻

---

### 例题 2：Sn/Pb 价态稳定性判断（氧化还原与惰性电子对）

**题目：**
向 PbO₂ 的悬浊液中滴加浓盐酸，观察到黄绿色气体产生，溶液变为无色。向 SnCl₂ 溶液中滴加 HgCl₂ 溶液，先产生白色沉淀，继续滴加沉淀变灰黑色。

(1) 写出上述两个过程的离子方程式。
(2) 解释为什么 Pb(IV) 具有强氧化性而 Sn(IV) 稳定。
(3) 若将 PbO₂ 与 SnCl₂ 在酸性条件下混合，预测反应产物。

**分析：**
1. "黄绿色气体" = Cl₂，说明 PbO₂ 氧化了 Cl⁻ → Pb(IV) 强氧化性
2. "白色沉淀→灰黑色" = Hg₂Cl₂ → Hg，说明 Sn²⁺ 还原了 Hg²⁺ → Sn(II) 还原性
3. 对比 Pb 和 Sn 的 +4 价稳定性 → 惰性电子对效应

**解答：**

(1) 离子方程式：
- PbO₂ + 4H⁺ + 2Cl⁻ → Pb²⁺ + Cl₂↑ + 2H₂O
- 2HgCl₂ + Sn²⁺ → Hg₂Cl₂↓（白）+ Sn⁴⁺ + 2Cl⁻
- Hg₂Cl₂ + Sn²⁺ → 2Hg↓（黑）+ Sn⁴⁺ + 2Cl⁻

(2) 解释：
Pb 位于第六周期，受镧系收缩影响，6s² 电子对不易参与成键（惰性电子对效应）。Pb(IV) 失去 6s² 电子需要很高能量，故 Pb(IV) 化合物倾向于获得电子回到稳定的 Pb(II) 状态，表现为强氧化性。Sn 位于第五周期，惰性电子对效应较弱，Sn(IV) 是稳定价态。

(3) 预测：
PbO₂ + Sn²⁺ + 4H⁺ → Pb²⁺ + Sn⁴⁺ + 2H₂O
Pb(IV) 氧化 Sn(II)，两者发生归中反应。

**反思：**
- 惰性电子对效应主要影响第六周期 p 区元素：Tl(I)、Pb(II)、Bi(III) 稳定，Tl(III)、Pb(IV)、Bi(V) 强氧化性
- Sn(II) 是常用还原剂，但易被空气氧化，溶液中需加 Sn 粒保护
- 写方程式时注意酸性介质补 H⁺，并检查电荷守恒

### 例题 3：碳硅结构差异推断（晶体与成键）

**题目：**
解释下列现象：
(1) CO₂ 是气体，SiO₂ 是熔点 1700°C 的固体；
(2) CCl₄ 不水解，SiCl₄ 剧烈水解；
(3) 苯（C₆H₆）稳定存在，但不存在稳定的"硅苯"（Si₆H₆）。

**分析：**
三道小题均指向 C 和 Si 的成键差异：C 的 2p 轨道可形成强 pπ-pπ 键，Si 的 3p 轨道扩散，pπ-pπ 键极弱。

**解答：**

(1) CO₂ 中 C=O 双键很强（2 个 σ + 2 个 π），形成独立的 O=C=O 分子，分子间仅 van der Waals 力，故为气体。SiO₂ 中 Si—O 单键很强（368 kJ/mol），但 Si 无法形成稳定的 Si=O 双键（pπ-pπ 弱），因此通过 Si—O—Si 单键构建三维共价网络，熔点极高。

(2) CCl₄ 中 C 的价层已满（8 电子），无空轨道接受 H₂O 的孤对电子，且 C—Cl 键较强。SiCl₄ 中 Si 有空的 3d 轨道，H₂O 的 O 可作为亲核试剂进攻 Si，形成五配位中间体，发生水解：SiCl₄ + 2H₂O → SiO₂ + 4HCl。

(3) 苯的稳定性来自 6 中心 6 电子离域 π 键（π₆⁶），要求 C 的 p 轨道有效侧向重叠。Si 的 3p 轨道比 C 的 2p 轨道大且扩散，Si—Si 键长较长，p 轨道侧向重叠极弱，无法形成稳定的离域 π 键，故"硅苯"不存在。

**反思：**
- C/Si 差异的核心是原子半径和轨道匹配：C(2p) 紧凑，适合 π 键；Si(3p) 扩散，不适合 pπ-pπ
- Si 可利用 3d 轨道扩展配位数（如 SiF₆²⁻），C 不能
- Si—O 单键极强是硅酸盐化学的基础

---

### 例题 4：硅酸盐结构类型推断（结构化学应用）

**题目：**
某硅酸盐矿物经分析含 Si 和 O 的原子数比为 1:3，且该矿物具有纤维状结构。
(1) 判断该硅酸盐的结构类型，写出其结构单元通式。
(2) 若将该矿物与过量 NaOH 共熔，写出主要反应方程式。
(3) 另一种硅酸盐中 Si:O = 2:5，判断其结构类型并举例一种矿物。

**分析：**
1. Si:O = 1:3 → 每个 Si 与 3 个 O 共享，形成单链结构（辉石类）
2. Si:O = 2:5 → 每个 Si 与 2.5 个 O 共享，即共用 3 个氧，形成层状结构（云母/滑石类）
3. 硅酸盐结构分类的核心是 Si:O 比例与共享氧原子数的关系

**解答：**

(1) **结构类型：链状（单链）**
- 结构单元通式：[SiO₃]²⁻（单链）
- 每个 [SiO₄] 四面体共用 2 个氧原子，形成一维无限链
- 纤维状结构是链状硅酸盐的典型宏观特征

(2) **与 NaOH 共熔反应：**
$$
\ce{[SiO3]^{2-}_n + 2nNaOH ->[\Delta] nNa2SiO3 + nH2O}
$$
或更一般地写成：
$$
\ce{SiO2 + 2NaOH -> Na2SiO3 + H2O}
$$
（链状结构破坏，生成可溶性硅酸钠）

(3) **Si:O = 2:5 → 层状结构**
- 每个 [SiO₄] 四面体共用 3 个氧原子，形成二维层状网络
- 代表矿物：云母（如白云母 KAl₂(AlSi₃O₁₀)(OH)₂）、滑石、高岭土
- 结构单元可写作 [Si₂O₅]²⁻

**反思：**
- 硅酸盐结构类型的判断口诀："1:4 岛，1:3 链，2:5 层，1:2 架"
- 链状硅酸盐（如石棉）具有纤维状形态，层状硅酸盐（如云母）可剥离，架状硅酸盐（如石英）硬度高
- 共熔反应破坏 Si—O—Si 网络，生成可溶性硅酸盐
