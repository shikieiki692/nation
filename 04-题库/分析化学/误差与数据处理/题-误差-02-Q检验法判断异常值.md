## 题目

4次平行测定某试样中Fe含量（%）：10.25, 10.30, 10.28, 10.60。用Q检验法（90%置信度）判断10.60是否应舍弃。（Q₀.₉₀,₄=0.76）

## 参考答案

**Q检验法步骤：**

1. 将数据从小到大排列：10.25, 10.28, 10.30, 10.60

2. 计算极差R：
R = x_max - x_min = 10.60 - 10.25 = 0.35

3. 计算可疑值与邻近值之差：
可疑值为10.60，邻近值为10.30
差值 = 10.60 - 10.30 = 0.30

4. 计算Q值：
Q = 差值 / R = 0.30 / 0.35 = 0.857

5. 查表比较：
Q₀.₉₀,₄ = 0.76（90%置信度，4个数据）

6. 判断：
Q = 0.857 > Q₀.₉₀,₄ = 0.76

因此，10.60这个数据应予舍弃。

**舍弃后的数据处理：**

剩余数据：10.25, 10.28, 10.30

平均值 x̄ = (10.25 + 10.28 + 10.30) / 3 = 10.28%

标准偏差 s = √[((10.25-10.28)² + (10.28-10.28)² + (10.30-10.28)²) / (3-1)]
= √[(0.0009 + 0 + 0.0004) / 2]
= √(0.00065)
= 0.025%

**结果：** Q检验表明10.60应予舍弃，试样中铁含量为10.28 ± 0.03%（90%置信度）。
