## 题目

硼酸H₃BO₃的Ka=5.8×10⁻¹⁰，不能用NaOH直接准确滴定。加入甘露醇后，硼酸与甘露醇形成络合酸（Ka'=5.8×10⁻⁷）。计算0.10 mol/L硼酸甘露醇溶液用0.10 mol/L NaOH滴定时的计量点pH和突跃范围下限。

## 参考答案

**可行性判断：**

c·Ka' = 0.10 × 5.8×10⁻⁷ = 5.8×10⁻⁸ ≥ 10⁻⁸

强化后的硼酸-甘露醇络合酸刚好可以准确滴定。

**计量点pH计算：**

计量点时产物为甘露醇硼酸钠（弱酸盐），溶液体积加倍，csp = 0.05 mol/L

pH = 7 + ½(pKa' + lg csp)
= 7 + ½(-lg(5.8×10⁻⁷) + lg 0.05)
= 7 + ½(6.24 + (-1.30))
= 7 + ½(4.94)
= 9.47

**突跃范围下限计算：**

突跃下限对应a = 0.999（即滴定至99.9%）时的pH：

[HA] = 0.001csp，[A⁻] = 0.999csp

pH = pKa' + lg([A⁻]/[HA])
= 6.24 + lg(0.999/0.001)
= 6.24 + 3.00
= 9.24

考虑稀释效应和水的电离，突跃下限pH ≈ 7.7

**结论：** 计量点pH为9.47，可选用酚酞（变色范围8.0~10.0）作指示剂。
