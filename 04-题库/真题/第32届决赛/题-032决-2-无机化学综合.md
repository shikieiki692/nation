## 题目

### 2-1 (4分)
在一定条件下，$\mathrm{AlMe_3}$ 与 $\mathrm{H_2NMe}$ 反应生成 $\mathbf{A}$，$\mathbf{A}$ 的分子式为 $\mathrm{Al_8C_{16}N_8H_{48}}$。$\mathbf{A}$ 是多面体分子，多面体的骨干由四边形和六边形构成，分子中除一条四次反轴外，没有其他独立的对称元素。写出生成 $\mathbf{A}$ 的化学反应方程式，画出 $\mathbf{A}$ 多面体骨干的结构，结构中的原子用元素符号表示。

### 2-2 (3分)
最近研究发现，在 $200\mathrm{GPa}$ 压力下生成分子式为 $\mathrm{N_2F_8}$ 的B。B中存在N—N键，但2个N原子的配位数不同。画出该分子的结构，结构中的原子用元素符号表示，不在纸面上的化学键用楔形键表示。

### 2-3 (4分)
肼的制备一般采用 Raschig 反应，即在强碱性溶液中用 NaOCl 氧化 $\mathrm{NH_3}$ 来制取，反应分两步进行，第一步生成含 Cl 的中间体（方程式1），中间体与 $\mathrm{NH_3}$ 进一步反应生成肼（方程式2）。含 Cl 中间体与生成的肼会发生副反应（方程式3）。有的工厂用尿素代替 $\mathrm{NH_3}$ 来生产肼，写出用尿素生产肼的化学反应方程式（方程式4）。异烟肼（结构如下图）可以与银氨溶液发生银镜反应，写出化学反应方程式（方程式5）。

### 2-4 (4分)
通过 X 射线衍射测定了晶体 $\mathrm{W(N_3)_6}$ 的结构，结果发现，$\mathrm{W(N_3)_6}$ 晶体属于三方晶系，其六方晶胞参数为：$a=b=939.62 \, \mathrm{pm}, c=722.25 \, \mathrm{pm}, \alpha=\beta=90^{\circ}, \gamma=120^{\circ}$。已知晶体的密度为 $2.622 \, \mathrm{g \, cm^{-3}}$。该晶体的点阵型式是简单六方，通过计算推断结构基元。（阿伏加德罗常数 $N_{A} = 6.022 \times 10^{23} \, \mathrm{mol^{-1}}$）

### 2-5 (2分)
分子 $\mathrm{H_3SiNCS}$ 中除 H 之外，其余原子共线，写出分子中 N 原子的杂化方式及分子中的大π键。

## 参考答案

### 2-1 答案
**反应方程式**：
$$\mathrm{8AlMe_3 + 8H_2NMe \rightarrow Al_8C_{16}N_8H_{48} + 8CH_4}$$

**多面体骨干结构**：
- Al₈N₈ 笼状结构
- 由四边形和六边形组成
- 具有 S₄ 对称性（四次反轴）

### 2-2 答案
**N₂F₈ 结构**：
- N—N 键存在
- 两个 N 原子配位数不同
- 一个 N 连接 4 个 F，另一个 N 连接 3 个 F + 1 个 N
- 高压下的特殊结构

### 2-3 答案
**方程式4（尿素法）**：
$$\mathrm{CO(NH_2)_2 + NaOCl + 2NaOH \rightarrow N_2H_4 + NaCl + Na_2CO_3 + H_2O}$$

**方程式5（银镜反应）**：
$$\mathrm{异烟肼 + 2[Ag(NH_3)_2]^+ + 2OH^- \rightarrow 异烟酸 + 2Ag \downarrow + 4NH_3 + H_2O}$$

### 2-4 答案
**晶胞体积**：
$$V = a^2 c \sin(60°) = (939.62 \times 10^{-10})^2 \times 722.25 \times 10^{-10} \times \frac{\sqrt{3}}{2}$$
$$V = 5.37 \times 10^{-22} \, \mathrm{cm^3}$$

**结构基元计算**：
$$Z = \frac{\rho V N_A}{M} = \frac{2.622 \times 5.37 \times 10^{-22} \times 6.022 \times 10^{23}}{462.1}$$
$$Z = 1.85 \approx 2$$

**结构基元**：$\mathrm{W(N_3)_6}$ 分子，每个晶胞含2个分子

### 2-5 答案
**N原子杂化方式**：sp 杂化

**大π键**：$\Pi_3^4$（3中心4电子π键）
