# 题-025决-3：恒压反应器化学平衡

## 题目

> **来源**：第25届中国化学奥林匹克（决赛）第3题(10分)

恒温、有催化剂条件下，容积可变反应器（恒压）中进行 N₂(g)+3H₂(g)⇌2NH₃(g)。

3-1 平衡时V=1.0L，[N₂]=0.11, [H₂]=0.14, [NH₃]=0.75 mol/L。注入4.0 mol N₂，计算Kc和Qc，判断方向。

3-2 一般化推导：注入xV体积的N₂后Qc的表达式和反应方向。

## 参考答案

**3-1**：
- Kc = [NH₃]²/([N₂][H₂]³) = (0.75)²/(0.11×0.14³) = **1.9×10³ (mol/L)⁻²**
- 加入4.0 mol N₂后体积增至5.0L
- Qc = (0.15)²/(0.822×0.028³) = **1.2×10³**
- Qc < Kc → **反应向右移动**

**3-2**：
$$Q_{c} = K_{c} \cdot \frac{(1+x)^{2}}{1 + \frac{x}{c(\mathrm{N}_{2})V_{0}}}$$

- 当 $x > \frac{1}{c(\mathrm{N}_{2})V_{0}} - 2$ 时，平衡左移
- 当 $x < \frac{1}{c(\mathrm{N}_{2})V_{0}} - 2$ 时，平衡右移

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| Kc计算 | 代入平衡浓度 |
| 恒压条件 | 体积随物质的量变化 |
| Qc推导 | 考虑体积膨胀的浓度修正 |

## 解题思路

1. Kc直接由平衡浓度计算
2. 加入N₂后：总mol增加→体积等比例增大→各物质浓度变化
3. Qc = 新浓度代入平衡表达式
4. 一般化：用x表示加入量，推导Qc/Kc的比值

## 考点抽象

核心难点：恒压条件下，加入气体会导致体积膨胀，所有物种浓度同时变化。与恒容条件本质不同。

## 变式拓展

- 若改为恒容条件加入N₂，Qc表达式更简单
- 实际工业合成氨：高压条件，Le Chatelier原理

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 体积变化 | 恒压→V∝n，不是恒容 |
| 浓度修正 | 所有物种浓度都被稀释 |
| Qc vs Kc | Qc<Kc→右移（不是左移） |
