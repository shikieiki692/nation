# 题-025决-1-2-1：双缩脲Cu配合物结构

## 题目

> **来源**：第25届中国化学奥林匹克（决赛）第1题(13分) 第(1-2)(1)小问

$\mathrm{Cu}^{2+}$ 硫酸盐在碱性溶液中加入缩二脲 $\mathrm{HN}(\mathrm{CONH}_{2})_{2}$，得到特征紫色物质，该物质为-2价的配位阴离子，具有对称中心和不通过A的二重旋转轴，无金属-金属键。画出该阴离子的结构。

## 参考答案

双核Cu配合物，两个Cu²⁺各由缩二脲的N原子螯合配位，两个Cu通过两个OH⁻桥联。整体为-2价阴离子。

结构要点：
- 2个Cu²⁺ + 2个缩二脲配体 + 2个OH⁻桥 → 总电荷: +4-2-2 = 0 → 需配平
- 实际：缩二脲作为双齿配体（去质子后N⁻配位），每个Cu配位4个N
- 两个OH⁻桥联两个Cu
- 具有对称中心（i）和不通过Cu的C₂轴

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 双缩脲反应 | 1833年发现的经典蛋白质鉴定反应 |
| 桥联配合物 | OH⁻桥连两个Cu中心 |
| 对称性分析 | 对称中心+i，C₂轴不通过Cu |

## 解题思路

1. "不通过A的二重轴"+"对称中心" → 至少2个Cu
2. 缩二脲适合做螯合配体，Cu倾向平面四方配位
3. 碱性条件 → OH⁻可能做桥联配体
4. -2价阴离子 → 2Cu²⁺ + 2(缩二脲²⁻) + 2OH⁻ = -2

## 考点抽象

本题核心是利用对称性信息推断配合物结构。关键推理链：对称性→多核→桥联配体→电荷平衡。

## 变式拓展

- 若改为"具有C₂轴但无对称中心"，结构可能不同
- 类似题型：给出对称元素+电荷，推断多核配合物结构

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| Cu数目 | "不通过A的C₂轴"暗示多核 |
| 桥联配体 | OH⁻（碱性条件），不是其他配体 |
| 电荷平衡 | -2价 = 总正电荷 - 总负电荷 |
