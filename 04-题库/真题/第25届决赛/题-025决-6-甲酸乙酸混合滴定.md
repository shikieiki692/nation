# 题-025决-6：甲酸乙酸混合滴定

## 题目

> **来源**：第25届中国化学奥林匹克（决赛）第6题(8分)

甲酸和乙酸混合溶液：20.00mL用0.1000 mol/L NaOH滴定消耗25.00mL。另取20.00mL加50.00mL 0.02500 mol/L KMnO₄（强碱性），反应完全后酸化，加40.00mL 0.2000 mol/L Fe²⁺，用KMnO₄滴定剩余Fe²⁺消耗24.00mL。

6-1 总量。6-2 氧化还原方程式。6-3 各酸浓度。

## 参考答案

**6-1 总量**：
n(HCOOH) + n(HAc) = 0.1000 × 25.00×10⁻³ = **2.500×10⁻³ mol**

**6-2 碱性条件KMnO₄氧化甲酸**：
$$\mathrm{HCOO}^{-} + 2\mathrm{MnO}_{4}^{-} + 3\mathrm{OH}^{-} = \mathrm{CO}_{3}^{2-} + 2\mathrm{MnO}_{4}^{2-} + 2\mathrm{H}_{2}\mathrm{O}$$

**6-3**（电子守恒法）：

共74.00 mL KMnO₄最终均被还原至Mn²⁺。还原剂：Fe²⁺ + HCOOH。

$$n(\mathrm{HCOOH}) = \frac{1}{2}(5 \times 0.02500 \times 74.00 - 0.2000 \times 40.00) \times 10^{-3} = 6.250 \times 10^{-4} \text{ mol}$$

- **c(HCOOH) = 0.03125 mol/L**
- **c(HAc) = 0.09375 mol/L**

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 酸碱滴定 | 总酸量 |
| KMnO₄氧化 | 碱性条件下氧化甲酸 |
| 电子守恒 | 最终状态统一计算 |

## 解题思路

1. NaOH滴定→总酸量
2. KMnO₄碱性氧化甲酸→MnO₄²⁻→酸化歧化→Fe²⁺还原所有Mn物种
3. 最终：所有KMnO₄ → Mn²⁺，所有Fe²⁺ → Fe³⁺，HCOOH → CO₂
4. 电子守恒一步到位

## 考点抽象

关键技巧：跳过中间步骤，直接用**最终态电子守恒**。这在复杂氧化还原滴定中是万能方法。

## 变式拓展

- 类似方法用于铁矿石中Fe²⁺/Fe³⁺的同时测定
- 碱性KMnO₄的选择性氧化在分析化学中广泛应用

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 电子守恒 | 不要逐步分析每步反应 |
| KMnO₄总量 | 50+24=74 mL（不是50 mL） |
| 甲酸氧化态 | HCOOH中C为+2→CO₂中C为+4，转移2e⁻ |
