# 题-025决-7：C₆₀全合成路线推断

## 题目

> **来源**：第25届中国化学奥林匹克（决赛）第7题(18分)

2002年Scott等人首次完成C₆₀化学全合成。给出合成路线图。

7-1 试剂a~d。7-2 中间体A~I结构。7-3 J的对称元素。7-4 I→J的反应机理。

## 参考答案

**7-1 试剂/条件**：
- a: Mg/无水乙醚（选择性制备Br端格氏试剂）
- b: CH₃CHO（乙醛）
- c: H₃O⁺（酸化得醇）
- d: Ph₃P=CH₂（Wittig试剂）

**7-2 关键中间体**：
- A: 格氏试剂前体（二卤代苯）
- B: 格氏试剂（仅Br端反应）
- C: 醇（与乙醛加成后）
- D: 烯（脱水）
- E: 醛（Wittig反应底物）
- F: 二烯（Wittig产物）
- G: 溴化产物
- H: 水解/卤代产物
- I: 分子内Friedel-Crafts酰基化产物

**7-3 J的对称元素**：
- **C₃轴**（三聚体对称）
- **对称面σ**（平面分子）

**7-4 I→J机理**：
Lewis酸催化下的羟醛缩合环化三聚（aldol cyclotrimerization）。

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 格氏反应 | 选择性制备（Br vs Cl活性差异） |
| Wittig反应 | 烯烃合成 |
| Friedel-Crafts | 分子内酰基化关环 |
| 分子对称性 | C₃轴 + σ面 |

## 解题思路

1. 逆合成分析：从已知产物反推每步
2. 关键细节：格氏试剂制备需用乙醚（区分Br和Cl反应性）
3. Wittig反应引入关键双键
4. 分子内关环构建多环骨架
5. 三聚→J（C₆₀前体）

## 考点抽象

本题是典型的有机合成推断题，核心能力：逆合成分析 + 反应类型识别 + 细节把控。

## 变式拓展

- C₆₀的其他合成路线（电弧放电法、激光蒸发法）
- Scott合成的意义：首次证明C₆₀可通过理性设计合成

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 格氏溶剂 | 必须用乙醚（不是THF）来选择性反应 |
| Wittig试剂 | Ph₃P=CH₂（不是其他ylide） |
| 对称元素 | J是平面分子→有σ面 |
| 三聚机理 | 羟醛缩合环化（不是Diels-Alder） |
