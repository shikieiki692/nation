# 题-025决-1-4：O₂AuF₆推断

## 题目

> **来源**：第25届中国化学奥林匹克（决赛）第1题(13分) 第(1-4)小问

元素C（Au）的单质与单一无机酸不起作用，但可溶于王水。C与O₂和F₂作用制得化合物X，在X中C的质量分数为57.43%，X的结构与1962年N. Barlett开创性工作的产物极为相似。给出X的化学式。

## 参考答案

**X = [O₂][AuF₆]**

N. Barlett的工作：O₂ + PtF₆ → O₂⁺[PtF₆]⁻（首个惰性气体化合物的前驱研究）

设X为[O₂][AuₘFₙ]：

$$w(\mathrm{Au}) = \frac{197.0m}{32.00 + 197.0m + 19.00n} \times 100\% = 57.43\%$$

令m=1，则n=6合理 → X = [O₂][AuF₆]

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| Barlett开创性工作 | O₂PtF₆的发现 |
| O₂⁺化学 | O₂作为Lewis碱 |
| 元素含量计算 | 设未知数→赋值→找合理整数解 |

## 解题思路

1. Barlett → O₂PtF₆ → 类比 → O₂AuF₆
2. Au与O₂和F₂反应 → 含Au、O、F的化合物
3. 设化学式 → 代入质量分数 → 求整数解

## 考点抽象

核心方法：已知质量分数 → 设通式 → 赋值求解。这是决赛中反复出现的万能方法。

## 变式拓展

- 同法可推断XeF₂、XeF₄等惰性气体化合物
- 实际O₂AuF₆尚未被合成，但理论预测合理

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| Barlett工作 | 不是XeF₂（那是1962年之后的），而是O₂PtF₆ |
| 通式选择 | X = [O₂][AuFₙ]，不是AuOₓFᵧ |
| 整数约束 | m和n必须为正整数 |
