# 题-025决-5：丙酮热分解动力学

## 题目

> **来源**：第25届中国化学奥林匹克（决赛）第5题(14分)

1000K下丙酮热分解，给出5个基元反应及活化能。

5-1 写出两个平行总反应方程式。
5-2 推出速率方程（提示：丙酮消耗以反应(3)为主，可忽略指前因子差异）。
5-3 计算表观活化能。
5-4 计算CH₄+H₂O→3H₂+CO在298K的ΔrGm⊖和K⊖。

## 参考答案

**5-1 两个平行反应**：
- CH₃COCH₃ → CH₄ + CH₂CO
- 2CH₃COCH₃ → C₂H₅COCH₃ + CH₄ + CO

**5-2 速率方程**（稳态近似）：

对三种自由基（CH₃·、CH₃CO·、CH₃COCH₂·）建立稳态方程，联立求解。利用提示k₃k₄/(k₁k₅) >> 1，简化得：

$$r = \sqrt{\frac{k_{1}k_{3}k_{4}}{k_{5}}} \cdot [\mathrm{CH}_{3}\mathrm{COCH}_{3}]$$

对丙酮为**一级反应**。

**5-3 表观活化能**：

$$E_{\mathrm{a}} = \frac{1}{2}[E_{\mathrm{a}}(1) + E_{\mathrm{a}}(3) + E_{\mathrm{a}}(4) - E_{\mathrm{a}}(5)] = \frac{1}{2}(351+63+200-21) = \mathbf{296 \, kJ/mol}$$

**5-4 热力学计算**：
- ΔrH⊖ = 205.7 kJ/mol
- ΔrS⊖ = 214.7 J/(mol·K)
- ΔrG⊖ = 141.7 kJ/mol
- K⊖ = 1.45×10⁻²⁵
- K随温度升高而**增大**（ΔrH>0，吸热反应）

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 稳态近似 | 自由基浓度恒定 |
| 表观活化能 | 由各基元反应Ea组合 |
| Arrhenius方程 | k=A·exp(-Ea/RT) |
| 热力学计算 | ΔG=ΔH-TΔS, K=exp(-ΔG/RT) |

## 解题思路

1. 消去自由基→两个总反应
2. 稳态近似建立三个方程→联立求解[CH₃·]
3. 利用提示简化→速率方程
4. 由速率方程中k的组合→表观Ea
5. 热力学数据→ΔH、ΔS→ΔG→K

## 考点抽象

本题综合了动力学（稳态近似、表观活化能）和热力学（ΔG、K）两大板块，是决赛的典型综合计算题。

## 变式拓展

- 类似机理：甲烷热分解、乙烷裂解
- 稳态近似在酶催化（Michaelis-Menten）中的应用

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 稳态近似 | 三种自由基各一个方程 |
| 简化条件 | k₃k₄/(k₁k₅)>>1 → 近似求解 |
| Ea公式 | 注意E₅前是负号 |
| K趋势 | 吸热反应→K随T↑而↑ |
