# 题-025决-1-3-1：Ag滴定终点方程式

## 题目

> **来源**：第25届中国化学奥林匹克（决赛）第1题(13分) 第(1-3)(1)小问

用 $\mathrm{Ag}^{+}$ 的标准溶液滴定 KCl 和 KSCN 的中性溶液，得到电位滴定曲线，拐点依次位于 M、N、P 处。分别写出在 M、N、P 处达到滴定终点的离子方程式。

## 参考答案

**M处**（首先反应，形成配合物）：

$$\mathrm{Ag}^{+} + 2\mathrm{SCN}^{-} = \mathrm{Ag(SCN)}_{2}^{-}$$

**N处**（配合物转化为沉淀）：

$$\mathrm{Ag}^{+} + \mathrm{Ag(SCN)}_{2}^{-} = 2\mathrm{AgSCN} \downarrow$$

**P处**（最后沉淀Cl⁻）：

$$\mathrm{Ag}^{+} + \mathrm{Cl}^{-} = \mathrm{AgCl} \downarrow$$

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 沉淀与配合物竞争 | Ag⁺先与SCN⁻形成配合物 |
| Ksp与Kf | 配合物→沉淀→沉淀的顺序 |
| 电位滴定 | 每个拐点对应一个计量点 |

## 解题思路

1. Ag⁺加入含Cl⁻和SCN⁻的溶液
2. SCN⁻先与Ag⁺形成可溶配合物Ag(SCN)₂⁻（Kf很大）
3. Ag⁺过量后配合物转化为AgSCN沉淀
4. 最后Ag⁺与Cl⁻生成AgCl沉淀
5. 顺序：配合物→AgSCN沉淀→AgCl沉淀

## 考点抽象

核心：多组分体系中沉淀-配合物竞争的先后顺序。需综合考虑Ksp和Kf。

## 变式拓展

- 若用CN⁻代替SCN⁻，顺序如何变化？
- 实际分析中：Volhard法（返滴定）避免AgSCN对AgCl的转化

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 反应顺序 | 先配合物，后沉淀（不是先沉淀） |
| 三步区分 | M=配合物形成，N=沉淀转化，P=Cl⁻沉淀 |
| Ksp比较 | AgSCN < AgCl，但先形成配合物 |
