# 题-025决-4：沸石分子筛结构描述

## 题目

> **来源**：第25届中国化学奥林匹克（决赛）第4题(13分)

4-1 层状硅酸根结构的数字描述（e~h）。
4-2 笼状结构的数字描述（c~h）。
4-3 将笼状结构以平面图方法画出（b~d）。

## 参考答案

**4-1 层状结构**（连接点处各多边形边数）：

| 图 | 描述 |
|:---|:---|
| (e) | (4.6.10)(6².10) |
| (f) | (4.6.12) |
| (g) | (4.6.8)(4.8.12) |
| (h) | (4.5.8)(4.5.12)(5².12)(5.8.12) |

**4-2 笼状结构**（各多边形个数）：

| 图 | 描述 |
|:---|:---|
| (c) | 4³.5⁶ |
| (e) | 5¹².6² |
| (f) | 4⁶.6¹² |
| (g) | 4⁶.6².6³.6⁶ |
| (h) | 4⁶.4⁶.6².8⁶ |

**4-3 平面投影图**：将最上层放最内圈，逐层向外，连接对应顶点。

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 拓扑描述法 | 数字表示多边形类型和数目 |
| 层状结构 | 二维三连接层 |
| 笼状结构 | 三维封闭多面体 |

## 解题思路

1. 层状：观察每个连接点周围多边形→边数从小到大→"."隔开
2. 笼状：计数各多边形→按边数排列→上标写个数
3. 平面图：最内圈=顶部→逐层向外→连对应顶点

## 考点抽象

本题本质是**信息给予+逻辑推理**，可完全脱离化学知识，作为数学题解答。关键：理解规则→精确执行。

## 变式拓展

- 类似描述法用于碳笼（富勒烯）、金属有机框架
- 沸石分子筛的工业催化应用

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 层状vs笼状 | 描述方法不同（连接点 vs 多边形计数） |
| 排序规则 | 边数从小到大，同边数按个数排列 |
| 括号 | 不同类型连接点用括号区分 |
