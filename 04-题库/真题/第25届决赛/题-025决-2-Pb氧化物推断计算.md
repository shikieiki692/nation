# 题-025决-2：Pb氧化物推断计算

## 题目

> **来源**：第25届中国化学奥林匹克（决赛）第2题(12分)

化合物A为非放射性主族元素M与氧的化合物。将13.712 g A与过量热稀硝酸反应，得不溶物B和溶液C。B洗净后与过量草酸混合，加过量稀硫酸，生成气体通入过量熟石灰溶液，得4.004 g不溶物质。M的单质与氧反应生成D，D含氧10.38%。

2-1 计算D的化学式。2-2 计算A的化学式。2-3 写出B与草酸反应的离子方程式。2-4 根据晶体结构图写出M氧化物的化学式。

## 参考答案

**M = Pb（铅）**

**2-1**：D = **Pb₂O₃**（设MO_n，16n/(M+16n)=10.38%，n=3/2时M=207.2=Pb）

**2-2**：A = **Pb₃O₄**（PbO·PbO₂ = 2PbO·PbO₂）
- CaCO₃: 4.004/100.1 = 0.04000 mol → H₂C₂O₄ = 0.02000 mol
- PbO₂: 0.02000 mol × 239.2 = 4.784 g
- PbO: (13.712-4.784)/223.2 = 0.04000 mol
- n(PbO):n(PbO₂) = 2:1 → A = 2PbO·PbO₂ = Pb₃O₄

**2-3**：$\mathrm{PbO}_{2} + 2\mathrm{H}^{+} + \mathrm{H}_{2}\mathrm{C}_{2}\mathrm{O}_{4} = \mathrm{Pb}^{2+} + 2\mathrm{CO}_{2}\uparrow + 2\mathrm{H}_{2}\mathrm{O}$

**2-4**：晶体中Pb有六配位（+4）和三配位（+2）两种，比例1:2。O被三个Pb共用。
→ Pb₃O₄

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 元素推断 | 设未知数→赋值→找合理解 |
| Pb氧化态 | +2(PbO)、+4(PbO₂)、混合(Pb₃O₄) |
| 晶体读图 | 配位数→化学式 |

## 解题思路

1. D含氧10.38% → 设MO_n → 赋值n → 找到Pb
2. CaCO₃→草酸量→PbO₂量→A的组成
3. 晶体图：两种Pb配位数→氧化态→电中性→化学式

## 考点抽象

核心方法：设未知数-赋值-找合理整数解，适用于所有"含量推断化学式"的决赛题。

## 变式拓展

- 同法可用于推断Bi、Sn等主族金属氧化物
- 晶体结构读图能力是决赛的高频考点

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| D不是PbO₃ | n=3/2时才合理 |
| A的写法 | 2PbO·PbO₂或Pb₃O₄均可 |
| 晶体分析 | 需综合三个图的信息 |
