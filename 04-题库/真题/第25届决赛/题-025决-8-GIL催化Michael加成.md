# 题-025决-8：GIL催化Michael加成

## 题目

> **来源**：第25届中国化学奥林匹克（决赛）第8题(12分)

胍盐离子液体(GIL)催化Michael加成，合成多官能团化合物H。

8-1 画A和C结构。8-2 画B、D、F、G结构。8-3 E→F的机理。8-4 G→H的周环反应类型和驱动力。

## 参考答案

**8-1**：
- **A**：丙二腈（NC-CH₂-CN），活泼亚甲基化合物
- **C**：醛类底物（提供Michael受体）

**8-2**：
- **B**：Knoevenagel缩合产物（A与C脱水）
- **D**：B再与丙二腈Michael加成的产物
- **F**：分子内关环产物（二腈碳负离子对腈的亲核加成）
- **G**：F脱HCN后的产物

**8-3 E→F机理**：
丙二腈部分的碳负离子（GIL碱催化生成）对分子内另一端腈基进行亲核加成→环化。

**8-4 G→H**：
- (1) **[1,5]氢迁移**（sigmatropic rearrangement）
- (2) 驱动力：**形成芳环**，芳香稳定化能

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| Michael加成 | 碳负离子对α,β-不饱和体系的共轭加成 |
| Knoevenagel缩合 | 活泼亚甲基+醛→烯 |
| 分子内关环 | 碳负离子对腈的亲核加成 |
| [1,5]H迁移 | 周环反应，驱动力为芳构化 |

## 解题思路

1. 丙二腈两次引入（观察E的结构）
2. GIL作为碱催化剂（不需要了解离子液体细节）
3. 活泼亚甲基→碳负离子→Michael加成/Knoevenagel
4. 分子内关环→脱HCN→[1,5]H迁移→芳构化

## 考点抽象

核心：识别反应类型（Michael、Knoevenagel、周环反应）并串联为完整合成路线。

## 变式拓展

- 离子液体作为绿色催化剂在有机合成中的应用
- [1,5]H迁移在维生素D合成中的关键作用

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| GIL角色 | 碱催化剂（不是亲核试剂） |
| 丙二腈引入次数 | 两次（观察E的结构） |
| [1,5]H迁移 | 不是[1,3]或[1,7]，由骨架决定 |
| 芳构化 | 驱动力=芳香稳定化 |
