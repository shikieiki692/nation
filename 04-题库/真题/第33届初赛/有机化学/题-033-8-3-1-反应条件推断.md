# 题-033-8-3-1：反应 a 和 b 的条件推断

## 题目

> **来源**：[[07-资料提炼/提炼-第33届初赛试题解析|第33届初赛第8题]]（原始文件：[[mineru/02-真题解析/33届初赛试题解析]]）

8-3 维生素 C 普遍存在于动植物中，在动物肝脏中以葡萄糖为原料经四步反应合成。工业生产的方式也是以葡萄糖经以下过程合成的：

![[mineru/02-真题解析/33届初赛试题解析_images/b03b9bea8cdbd92cb84d2155eb77f04df874d5ae8dd71e0e9507d01f34266bfc.jpg]]

8-3-1 写出反应 a 和 b 的具体条件（可以不写溶剂）。

## 参考答案

**得分点**：
1. **正确写出反应 a 的条件**（2分）：还原剂
2. **正确写出反应 b 的条件**（2分）：丙酮保护

从被保护的中间产物倒推，"L-呋喃山梨糖"的名称提示我们它具有某种环状结构的醛或酮基团，注意到中间产物羧基的 $\alpha$ 碳实际上是一个缩酮，因而其他的含氧基团应为醇羟基。

条件 b 或 c 中的一步应为丙酮保护，另一步应为氧化，反应顺序应为先保护再氧化。故：

- **条件 a**：$\mathrm{NaBH}_4$、$\mathrm{LiAlH}_4$、$\mathrm{BH}_3 \cdot \mathrm{THF}$ 或 $\mathrm{H}_2/\mathrm{Pt}$ 等（醛基还原为醇）
- **条件 b**：$\mathrm{Me}_2\mathrm{C}=\mathrm{O}, \mathrm{H}^+$；$\mathrm{Me}_2\mathrm{C}(\mathrm{OMe})_2, \mathrm{H}^+$；或 $\mathrm{H}_2\mathrm{C}=\mathrm{C}(\mathrm{OMe})\mathrm{Me}, \mathrm{H}^+$ 等（丙酮保护二醇）
- **条件 c**：$\mathrm{KMnO}_4, \mathrm{H}^+$；$\mathrm{CrO}_3, \mathrm{H}^+$；或 PDC 等（氧化为羧酸）

D-山梨醇逆推回起始物 D-葡萄糖，由于葡萄糖是多羟基醛，因而这里应该发生的是醛基的还原过程（条件 a）。

## 解题思路

1. **读题定位**：题目要求推断维生素 C 合成路线中反应 a 和 b 的具体条件。
2. **🔑 关键转换**：从产物维生素 C 和已知中间体倒推——D-葡萄糖经还原（a）得 D-山梨醇，D-山梨醇经氧化和异构化得 L-山梨糖，L-山梨糖先经丙酮保护（b）再氧化（c）得羧酸，最后脱保护和异构化得维生素 C。
3. **推断要点**：a 为醛基还原，b 为丙酮保护邻二醇，c 为醇氧化为羧酸。

## 易错分析

| 错误 | 原因 | 纠正 | 课堂提问 |
|---|---|---|---|
| a 写成氧化条件 | 未识别葡萄糖到山梨醇是还原 | 葡萄糖是多羟基醛，山梨醇是六元醇，醛基被还原 | 葡萄糖和山梨醇的结构有什么区别？ |
| b 和 c 顺序写反 | 未理解保护基的作用 | 应先保护再氧化，否则氧化会破坏邻二醇 | 为什么在氧化前需要先保护邻二醇？ |
| 保护试剂写错 | 不熟悉缩酮保护的条件 | 丙酮/酸、2,2-二甲氧基丙酮/酸等 | 邻二醇的丙酮保护有哪些常用试剂？ |
| 氧化试剂选择不当 | 选择了过于强烈的氧化剂 | KMnO₄、CrO₃、PDC 等均可将醇氧化为酸 | 醇氧化为羧酸的常用试剂有哪些？ |
