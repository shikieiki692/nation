# 题-027-3-1：硫脲结构式

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第3题(11分) 第(3-1)小问

白色固体 A，熔点 182℃，摩尔质量 76.12 g mol⁻¹，可代替氰化物用于提炼金的新工艺。A 的合成方法有：(1) 142℃下加热硫氰酸铵；(2) CS₂与氨反应；(3) CaCN₂和(NH₄)₂S水溶液反应（放出氨气）。画出 A 的结构式。

## 参考答案

**A = 硫脲（Thiourea），CS(NH₂)₂**

结构式：

$$\mathrm{H}_{2}\mathrm{N}-\overset{\overset{\textstyle S}{\parallel}}{C}-\mathrm{NH}_{2}$$

推断依据：硫氰酸铵（NH₄SCN，M=76.12）加热异构化→硫脲，摩尔质量相同，无气体放出。

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 硫脲 | 重要工业试剂，代替CN⁻提金 |
| 异构化 | NH₄SCN → SC(NH₂)₂ |
| 结构推断 | 由合成方法和摩尔质量推导 |

## 解题思路

1. 加热硫氰酸铵（76.12 g/mol）→ A（76.12 g/mol）：无气体放出，同分异构
2. CS₂ + NH₃ → 含C、S、N、H的化合物
3. CaCN₂ + (NH₄)₂S → 同样含C、S、N、H
4. 化学式CSN₂H₄，排除其他异构体 → 硫脲最稳定

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 异构体选择 | NH₄SCN是盐，硫脲是共价分子 |
| 摩尔质量 | 76.12恰好等于CSN₂H₄ |
| 与尿素类比 | SC(NH₂)₂类似OC(NH₂)₂ |
