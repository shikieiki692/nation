# 题-027-3-4：硫脲溶解金离子方程式

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第3题第(3-4)小问

酸性溶液中，A 在氧化剂（如 Fe³⁺、H₂O₂ 和 O₂）存在下能溶解金，形成 sp 杂化的 Au(I) 配合物。写出 A 在硫酸铁存在下溶解金的离子方程式。

## 参考答案

$$\mathrm{Au} + \mathrm{Fe}^{3+} + 2\mathrm{CS(NH}_{2})_{2} = \mathrm{Fe}^{2+} + \mathrm{Au[CS(NH}_{2})_{2}]_{2}^{+}$$

考虑酸性环境，也可写为：

$$\mathrm{Au} + \mathrm{Fe}^{3+} + 2\mathrm{CSN}_{2}\mathrm{H}_{5}^{+} = \mathrm{Fe}^{2+} + \mathrm{Au[CS(NH}_{2})_{2}]_{2}^{+} + 2\mathrm{H}^{+}$$

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| Au(I)配合物 | sp杂化→二配位 |
| 硫脲配位 | 通过S原子配位 |
| 软硬酸碱 | Au⁺(软酸)+S(软碱) |

## 解题思路

1. Au → Au⁺（被Fe³⁺氧化），失去1e⁻
2. Au⁺为18e⁻型软酸，与软碱S配位
3. sp杂化→2配位→Au[SC(NH₂)₂]₂⁺
4. Fe³⁺ + e⁻ → Fe²⁺（氧化剂）

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 配位数 | sp杂化只能2配位 |
| 配位原子 | S（不是N），符合软硬酸碱 |
| 氧化态 | Au(I)，不是Au(III) |
