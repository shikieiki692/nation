# 题-027-3-2：硫脲合成方程式

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第3题(11分) 第(3-2)小问

分别写出合成 A 的方法（2）、（3）中的化学反应方程式。

## 参考答案

**方法（2）** CS₂与氨反应：

$$\mathrm{CS}_{2} + 3\mathrm{NH}_{3} = \mathrm{CSN}_{2}\mathrm{H}_{4} + \mathrm{NH}_{4}\mathrm{HS}$$

或写为：

$$\mathrm{CS}_{2} + 4\mathrm{NH}_{3} = \mathrm{CSN}_{2}\mathrm{H}_{4} + (\mathrm{NH}_{4})_{2}\mathrm{S}$$

**方法（3）** CaCN₂与(NH₄)₂S反应：

$$\mathrm{CaCN}_{2} + (\mathrm{NH}_{4})_{2}\mathrm{S} + 2\mathrm{H}_{2}\mathrm{O} = \mathrm{CSN}_{2}\mathrm{H}_{4} + \mathrm{Ca(OH)}_{2} + 2\mathrm{NH}_{3} \uparrow$$

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 硫脲工业合成 | 两种经典路线 |
| 氨解反应 | CS₂的氨解 |
| 水解反应 | CaCN₂的水解 |

## 解题思路

1. 方法(2)：CS₂中C-S键被NH₃亲核进攻，逐步氨解
2. 方法(3)：CaCN₂水解释放NH₂-，与S²⁻结合形成硫脲

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 副产物 | 方法(2)生成NH₄HS或(NH₄)₂S |
| 水的参与 | 方法(3)需要水参与反应 |
| NH₃放出 | 方法(3)有NH₃气体放出 |
