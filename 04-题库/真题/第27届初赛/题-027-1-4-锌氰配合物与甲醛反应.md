# 题-027-1-4：锌氰配合物与甲醛反应

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第1题(12分) 第(1-4)小问

在碱性条件下，$\mathrm{Zn(CN)_{4}^{2-}}$ 和甲醛反应。

## 参考答案

$$\mathrm{Zn(CN)}_{4}^{2-} + 4\mathrm{HCHO} + 4\mathrm{H}_{2}\mathrm{O} = 4\mathrm{HOCH}_{2}\mathrm{CN} + \mathrm{Zn(OH)}_{4}^{2-}$$

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 掩蔽剂的解蔽 | EDTA滴定中Zn²⁺的掩蔽/解蔽 |
| 亲核加成 | CN⁻亲核进攻HCHO形成氰醇 |
| 软硬酸碱交换 | CN⁻（软碱）+ HCHO ↔ OH⁻（硬碱）+ Zn²⁺ |

## 解题思路

1. Zn(CN)₄²⁻ 存在解离平衡：Zn(CN)₄²⁻ ⇌ Zn²⁺ + 4CN⁻
2. CN⁻是优良的亲核试剂，亲核进攻HCHO生成HOCH₂CN（氰醇）
3. 氰醇的稳定常数远大于Zn(CN)₄²⁻，推动平衡右移
4. 释放的OH⁻与Zn²⁺配位形成Zn(OH)₄²⁻
5. 本质：软碱CN⁻与软酸HCHO选择性结合，硬碱OH⁻与交界酸Zn²⁺结合

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 反应来源 | 来自分析化学"掩蔽剂的解蔽"，非课本常见反应 |
| 氰醇生成 | CN⁻ + HCHO + H₂O → HOCH₂CN + OH⁻ |
| 配合物转化 | Zn(CN)₄²⁻ → Zn(OH)₄²⁻，配体完全替换 |
| 难度定位 | 此题为"高手过招"级别的区分题 |
