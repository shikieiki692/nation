# 题-027-4-2：硝酸汞水合物化学式

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第4题(7分) 第(4-2)小问

称取 1.713 g Hg(NO₃)₂·xH₂O，配制成 500 mL 溶液作为滴定剂。取 20.00 mL 0.0100 mol/L NaCl 标准溶液，用上述硝酸汞溶液滴定至紫色，消耗 10.20 mL。推断该硝酸汞水合物样品的化学式。

## 参考答案

**Hg(NO₃)₂·1.4H₂O**

计算过程：
- 计量关系：1 Hg²⁺ ~ 2 Cl⁻
- c(Hg(NO₃)₂) = ½ × (20.00/10.20) × 0.0100 = 9.80×10⁻³ mol/L
- n(Hg(NO₃)₂) = 9.80×10⁻³ × 0.500 = 4.90 mmol
- M = 1713/4.90 = 350 g/mol
- x = (350 - 324.6)/18.00 ≈ 1.4

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 汞量法 | Hg²⁺ + 2Cl⁻ → HgCl₂ |
| 滴定计算 | 标准溶液浓度→物质的量→摩尔质量 |
| 水合物组成 | 由摩尔质量差确定结晶水 |

## 解题思路

1. 由NaCl标准溶液标定Hg(NO₃)₂浓度
2. 由浓度和体积得物质的量
3. 由质量和物质的量得摩尔质量
4. 减去无水Hg(NO₃)₂的M → 结晶水

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 计量比 | Hg²⁺:Cl⁻ = 1:2（不是1:1） |
| 摩尔质量 | Hg(NO₃)₂ = 324.6 |
| x值 | 1.4不是整数，样品可能不均匀 |
