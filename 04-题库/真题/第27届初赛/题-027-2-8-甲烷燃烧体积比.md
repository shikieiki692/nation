# 题-027-2-8：甲烷燃烧体积比计算

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第2题(23分) 第(2-8)小问

甲烷在汽车发动机中平稳、完全燃烧。甲烷与空气按一定比例混合，氧气的利用率为 85%，计算汽车尾气中 $\mathrm{O}_{2}$、$\mathrm{CO}_{2}$、$\mathrm{H}_{2}\mathrm{O}$ 和 $\mathrm{N}_{2}$ 的体积比。（空气中 $\mathrm{O}_{2}$ 和 $\mathrm{N}_{2}$ 体积比按 21:79 计；设尾气中 $\mathrm{CO}_{2}$ 的体积为1。）

## 参考答案

$$\mathrm{CH}_{4} + 2\mathrm{O}_{2} = \mathrm{CO}_{2} + 2\mathrm{H}_{2}\mathrm{O}$$

设CH₄ = 1体积：
- 需O₂ = 2体积，利用率85% → 实际供O₂ = 2/0.85 = 2.353体积
- 剩余O₂ = 2.353 - 2 = 0.353体积
- 空气中N₂ = 2.353 × 79/21 = 8.860体积
- 产CO₂ = 1，产H₂O = 2

以CO₂=1为基准：

**O₂:CO₂:H₂O:N₂ = 0.35:1:2:8.86**

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 燃烧方程式 | CH₄完全燃烧 |
| 利用率计算 | 实际需氧量 = 理论量/利用率 |
| 空气组成 | O₂:N₂ = 21:79 |

## 解题思路

1. 写出燃烧方程式，确定理论需氧量
2. 利用率85% → 实际供氧 = 理论/0.85
3. 剩余O₂ = 供氧 - 消耗
4. N₂随空气进入，不参与反应
5. 统一以CO₂=1为基准

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 利用率方向 | 85%利用率 → 需更多空气（不是更少） |
| N₂计算 | 从空气来，与O₂的21:79关系 |
| H₂O状态 | 常温下为液态，但尾气中通常按气态计 |
