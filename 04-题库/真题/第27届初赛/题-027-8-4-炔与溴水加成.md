# 题-027-8-4：炔与溴水加成

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第8题第(8-4)小问

$\mathrm{CH}_{3}\mathrm{CH}_{2}\mathrm{C}{\equiv}\mathrm{CH}$ 与 Br₂/H₂O 反应，画出中间体7、8和产物9、10。

## 参考答案

**中间体7**：溴鎓离子（三元环正离子）

**中间体8**：H₂O开环后的烯醇

**产物9**：烯醇异构化后的酮（2-丁酮）

**产物10**：进一步加成的二溴产物

机理：
1. 炔与Br₂加成→溴鎓离子（中间体7）
2. H₂O开环→烯醇（中间体8）
3. 烯醇异构化→酮（产物9，互变异构）
4. 类似炔的羟汞化反应

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 炔烃亲电加成 | Br₂加成到三键 |
| 溴鎓离子 | 三元环中间体 |
| 烯醇互变异构 | 烯醇→酮 |

## 解题思路

1. Br₂对炔加成→溴鎓离子
2. H₂O作为亲核试剂开环
3. 生成的烯醇不稳定→互变异构为酮
4. 类比炔的羟汞化-脱汞反应

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 烯醇→酮 | 互变异构是关键步骤 |
| 产物选择 | 主产物为酮（不是醇） |
| 类比 | 类似炔的羟汞化反应 |
