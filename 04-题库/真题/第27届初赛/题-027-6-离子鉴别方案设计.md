# 题-027-6：离子鉴别方案设计

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第6题(10分)

某同学得知一定浓度的 $\mathrm{Fe}^{2+}$、$\mathrm{Cr(OH)}_{4}^{-}$、$\mathrm{Ni}^{2+}$、$\mathrm{MnO}_{4}^{2-}$ 和 $\mathrm{CuCl}_{3}^{-}$ 的水溶液都呈绿色。用蒸馏水、稀 H₂SO₄ 及简单器材，以尽可能少的步骤鉴别它们。

## 参考答案

**第1步**：将5种溶液分别滴到点滴板上，滴加稀H₂SO₄：
- $\mathrm{CuCl}_{3}^{-}$ → 蓝色溶液（配合物解离）：$\mathrm{CuCl}_{3}^{-} + 4\mathrm{H}_{2}\mathrm{O} = \mathrm{Cu(H_{2}O)}_{4}^{2+} + 3\mathrm{Cl}^{-}$
- $\mathrm{Cr(OH)}_{4}^{-}$ → 绿色沉淀：$\mathrm{Cr(OH)}_{4}^{-} + \mathrm{H}^{+} = \mathrm{Cr(OH)}_{3}\downarrow + \mathrm{H}_{2}\mathrm{O}$
- $\mathrm{MnO}_{4}^{2-}$ → 紫色溶液+棕色沉淀：$3\mathrm{MnO}_{4}^{2-} + 4\mathrm{H}^{+} = 2\mathrm{MnO}_{4}^{-} + \mathrm{MnO}_{2}\downarrow + 2\mathrm{H}_{2}\mathrm{O}$
- Fe²⁺和Ni²⁺ → 无明显变化

**第2步**：用已鉴出的 $\mathrm{Cr(OH)}_{4}^{-}$ 滴加到Fe²⁺和Ni²⁺中：
- Fe²⁺ → 沉淀迅速变红褐色（Fe(OH)₂→Fe(OH)₃氧化）
- Ni²⁺ → 绿色沉淀不变色

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 离子鉴别 | 基于特征反应的系统分析 |
| 配合物解离 | CuCl₃⁻在水中解离变蓝 |
| 歧化反应 | MnO₄²⁻在酸性中歧化 |
| 沉淀颜色变化 | Fe(OH)₂→Fe(OH)₃的氧化变色 |

## 解题思路

1. 利用稀H₂SO₄的酸性环境差异鉴别3种离子
2. 利用已知离子作为新试剂鉴别剩余2种
3. 关键区别：Fe²⁺的氢氧化物会被空气氧化变色，Ni²⁺不会

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 步骤优化 | 官方答案分两步先水后酸，实际一步酸即可 |
| 滴加顺序 | 必须说明将谁滴加到谁中 |
| 现象描述 | 每种离子的特征现象必须准确 |
