# 题-027-2-1：Bi₂Cl₈²⁻结构与杂化轨道

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第2题(23分) 第(2-1)小问

$\mathrm{Bi}_{2}\mathrm{Cl}_{8}^{2-}$ 离子中铋原子的配位数为5，配体呈四角锥形分布，画出该离子的结构并指出Bi原子的杂化轨道类型。

## 参考答案

2个Bi原子配位数均为5，共有8个Cl，因此必定有2个Cl为共用桥联配体。

**杂化轨道**：Bi为第15族元素，最外层5个电子。形成6个杂化轨道（1个空轨道 + 1个孤对电子轨道 + 4个成键轨道对应四角锥），故为 **sp³d² 杂化**。

结构示意：两个Bi通过2个Cl桥联，每个Bi额外连接3个端基Cl，形成两个四角锥配位环境。存在顺反两种异构体。

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 配位数 | Bi配位数=5，8个Cl中2个共用 |
| 杂化轨道 | sp³d²（含1个孤对电子和1个空轨道） |
| 桥联配合物 | 两个Cl作桥联配体 |
| 几何构型 | 四角锥形，有顺反异构 |

## 解题思路

1. 2个Bi + 8个Cl → 每个Bi需5个配位原子
2. 若无桥联：2×5=10个Cl位，但只有8个Cl → 有2个Cl桥联
3. 每个Bi：2个桥联Cl + 3个端基Cl = 5配位
4. Bi为15族，6个杂化轨道 = sp³d²

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 桥联数 | 8-Cl中2个必须是桥联的，否则配位数不够 |
| 杂化类型 | sp³d²（不是sp³d，因为含孤对电子占一个轨道） |
| 顺反异构 | 两个端基Cl可位于同侧或异侧 |
