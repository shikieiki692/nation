# 题-027-3-5：配位原子判断

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第3题第(3-5)小问

A 和 Au(I) 形成的配合物中配位原子是什么？

## 参考答案

配位原子是 **S（硫原子）**。

理由：Au⁺是典型的18e⁻型化合物，属于**软酸**。硫脲中有两类配位原子：N（较硬）和S（较软）。根据**软硬酸碱理论**（软亲软），软酸Au⁺优先与软碱S配位。

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 软硬酸碱理论 | 软酸-软碱优先结合 |
| Au⁺电子构型 | 18e⁻，典型软酸 |
| 两可配体 | 硫脲可经S或N配位 |

## 解题思路

1. Au⁺为18e⁻构型 → 极化率大 → 软酸
2. 硫脲中S（3p）比N（2p）极化率大 → S更软
3. 软酸+软碱 → S配位

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| N vs S | 不是N配位，而是S配位 |
| 两可配体 | 硫脲是经典的S/N两可配体 |
| 理论依据 | 软硬酸碱理论，不是简单的电负性 |
