# 题-027-1-5：Fe(OH)₂转化为Fe₃O₄

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第1题(12分) 第(1-5)小问

$\mathrm{Fe(OH)_{2}}$ 在常温无氧条件下转化为 $\mathrm{Fe}_{3}\mathrm{O}_{4}$。

## 参考答案

$$3\mathrm{Fe(OH)}_{2} = \mathrm{Fe}_{3}\mathrm{O}_{4} + \mathrm{H}_{2} \uparrow + 2\mathrm{H}_{2}\mathrm{O}$$

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 铁的氧化态变化 | Fe(II) → Fe(II)+Fe(III)混合 |
| 氧化剂判断 | 无氧条件下H⁺为氧化剂 |
| Fe₃O₄结构 | FeO·Fe₂O₃，混合价态 |

## 解题思路

1. Fe(OH)₂中Fe为+2价，Fe₃O₄中Fe为+2和+3混合（平均+8/3）
2. 无氧条件下无外部氧化剂，O已在最低价(-2)
3. 唯一可能的氧化剂是H⁺（来自水）
4. Fe(II)部分被H⁺氧化，放出H₂

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 氧化剂选择 | 不能写O₂参与（题目明确无氧） |
| H₂来源 | H⁺被还原为H₂，不是分解水 |
| 配平 | 3Fe(OH)₂ → Fe₃O₄ + H₂ + 2H₂O |
