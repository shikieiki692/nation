# 题-027-8-3：CH₃SeCl与烯烃反应

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第8题第(8-3)小问

画出 CH₃SeCl 与烯烃反应的中间体5和产物6的结构简式。

## 参考答案

**中间体5**：硒鎓离子三元环（类溴鎓离子）

**产物6**：反式加成产物（Cl⁻从背面进攻开环）

机理：
1. CH₃SeCl中Se带部分正电荷，类似BrCl中的Br
2. 烯烃亲核进攻Se，Cl⁻离去
3. 形成硒鎓离子三元环（类似溴鎓离子）
4. Cl⁻反式进攻开环→反式加成产物

可将 CH₃SeCl 类比为 BrCl 来理解。

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 硒鎓离子 | 类似溴鎓离子的三元环中间体 |
| 类比法 | CH₃SeCl ≈ BrCl |
| 反式加成 | Cl⁻从背面进攻 |

## 解题思路

1. Se-Cl键比C-Se键更容易断裂
2. Cl⁻离去后Se带正电→亲电性
3. 烯烃进攻Se→三元环（硒鎓离子）
4. Cl⁻反式进攻→反式开环

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 类比对象 | CH₃SeCl可类比BrCl |
| 断键位置 | Se-Cl（不是C-Se） |
| 立体化学 | 反式加成（anti-addition） |
