# 题-027-5-2：元素推断AMXY

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第5题第(5-2)小问

A、M、X、Y 各是什么？

## 参考答案

- **M = Na**（钠，M≈23 g/mol）
- **X = O**（氧，Z=8 < Z(Na)=11）
- **Y = Cl**（氯，Z=17 > Z(Na)=11）
- **A = H**（氢，形成A₂=H₂，无色无味）

验证：
- Z(X) < Z(M) < Z(Y)：8 < 11 < 17 ✓
- M₃XY = Na₃OCl（反钙钛矿结构）✓
- MXA = NaOH，MY = NaCl ✓
- A₂ = H₂（无色无味）✓

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 元素推断 | 由摩尔质量和原子序数关系确定 |
| 反钙钛矿 | Na₃OCl为典型反钙钛矿 |

## 解题思路

1. M = Na（由5-1得M≈23）
2. Z(X) < 11 < Z(Y)，均为短周期非金属
3. Y与Na 1:1结合 → NaCl → Y = Cl
4. X与Na、Cl形成Na₃XCl → X = O
5. A形成双原子分子 → A₂ = H₂

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 原子序数约束 | 必须满足Z(X)<Z(M)<Z(Y) |
| MY比例 | 1:1 → NaCl（不是Na₂O等） |
| A₂性质 | 无色无味 → H₂（不是O₂或Cl₂） |
