# 题-027-1-6：NaN₃制备方程式

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第1题(12分) 第(1-6)小问

将 $\mathrm{NaNO}_{3}$ 粉末小心加到熔融的 $\mathrm{NaNH}_{2}$ 中，生成 $\mathrm{NaN}_{3}$（没有水生成）。

## 参考答案

$$\mathrm{NaNO}_{3} + 3\mathrm{NaNH}_{2} = \mathrm{NaN}_{3} + \mathrm{NH}_{3} \uparrow + 3\mathrm{NaOH}$$

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 叠氮化物制备 | 工业制备NaN₃的方法之一 |
| 非水体系反应 | 熔融态反应，无水生成 |
| 氮的氧化态变化 | N(+5) + N(-3) → N(-1/3) |

## 解题思路

1. 反应物：NaNO₃（N为+5）+ NaNH₂（N为-3）
2. 产物：NaN₃（N平均为-1/3）
3. 注意约束条件：没有水生成
4. 副产物推断：NaNH₂中的H和O结合 → NaOH，多余的N和H → NH₃

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 水的约束 | 题目明确"没有水生成"，不能写H₂O |
| 副产物 | NH₃和NaOH，不是H₂O |
| 配平 | 1:3:1:1:3 |
