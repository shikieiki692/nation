# 题-027-2-7：钯配合物组成计算

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第2题(23分) 第(2-7)小问

实验得到一种含钯化合物 $\mathrm{Pd}[\mathrm{C}_{x}\mathrm{H}_{y}\mathrm{N}_{z}](\mathrm{ClO}_{4})_{2}$，该化合物中 C 和 H 的质量分数分别为 30.15% 和 5.06%。将此化合物转化为硫氰酸盐 $\mathrm{Pd}[\mathrm{C}_{x}\mathrm{H}_{y}\mathrm{N}_{z}](\mathrm{SCN})_{2}$，则 C 和 H 的质量分数分别为 40.46% 和 5.94%。通过计算确定其组成。

## 参考答案

**Pd[C₁₄H₂₈N₄](ClO₄)₂**

计算过程：
1. 由ClO₄⁻化合物：C/H摩尔比 = (30.15/12.01):(5.06/1.008) = 1:2 → y = 2x
2. 由SCN⁻化合物：(x+2)/y = (40.46/12.01)/(5.94/1.008) → 解得 x ≈ 14, y = 28
3. 由分子量计算z：z = [14×12.01/30.15% - (106.4+12.01×4+1.008×28+2×99.45)]/14.01 ≈ 4

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 元素分析 | 质量分数→摩尔比 |
| 联立方程 | 两个化合物的数据联立求解 |
| 配合物组成 | Pd + 配体 + 反离子 |

## 解题思路

1. 第一个化合物C:H = 1:2 → y = 2x
2. 第二个化合物含额外2个C（来自SCN⁻），故(x+2)/y = 已知比
3. 代入y=2x → 解出x和y
4. 由第一个化合物的总分子量 → 求出z

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| SCN⁻贡献 | 转化后C增加2个（2个SCN⁻） |
| Pd原子量 | 使用106.4 |
| ClO₄⁻分子量 | 99.45 |
