# 题-027-5-3：反钙钛矿水解方程式

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第5题第(5-3)小问

写出 $\mathrm{M}_{3}\mathrm{XY}$（即 $\mathrm{Na}_{3}\mathrm{OCl}$）发生水解的方程式。

## 参考答案

$$\mathrm{Na}_{3}\mathrm{OCl} + \mathrm{H}_{2}\mathrm{O} = 2\mathrm{NaOH} + \mathrm{NaCl}$$

Na₃OCl可看作 Na₂O·NaCl，其中Na₂O部分与水反应生成NaOH，NaCl不水解。

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 氧化物水解 | Na₂O + H₂O → 2NaOH |
| 复合氧化物 | Na₃OCl = Na₂O + NaCl |
| 水解方程式 | 配平原子和电荷 |

## 解题思路

1. Na₃OCl = Na₂O·NaCl（概念性拆分）
2. Na₂O + H₂O → 2NaOH
3. NaCl保留不变
4. 合并：Na₃OCl + H₂O → 2NaOH + NaCl

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 拆分方式 | Na₂O·NaCl（不是3Na + O + Cl） |
| NaCl不水解 | NaCl是中性盐，不参与水解 |
| 配平 | 1:1:2:1 |
