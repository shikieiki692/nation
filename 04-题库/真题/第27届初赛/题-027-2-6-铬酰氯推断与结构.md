# 题-027-2-6：铬酰氯推断与结构

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第2题(23分) 第(2-6)小问

向 $\mathrm{K}_{2}\mathrm{Cr}_{2}\mathrm{O}_{7}$ 和 NaCl 的混合物中加入浓硫酸制得化合物 X（154.9 g mol⁻¹）。X 为暗红色液体，沸点 117℃，有强刺激性臭味，遇水冒白烟，遇硫燃烧。X 分子有两个相互垂直的镜面，两镜面的交线为二重旋转轴。写出 X 的化学式并画出其结构式。

## 参考答案

**X = CrO₂Cl₂（铬酰氯）**

M = 52.00 + 2×16.00 + 2×35.45 = **154.9 g/mol** ✓

结构：四面体构型（类AB₂C₂型），两个O和两个Cl围绕Cr呈四面体分布。两个相互垂直的镜面 + 二重旋转轴 = C₂ᵥ对称性。

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 铬酰氯 | CrO₂Cl₂，强氧化性 |
| 分子对称性 | C₂ᵥ点群，2σ + C₂ |
| 元素推断 | 由制备方法+物理性质推断 |

## 解题思路

1. K₂Cr₂O₇ + NaCl + H₂SO₄ → 含Cr的化合物
2. 暗红色液体，强氧化性（遇硫燃烧）→ Cr(VI)化合物
3. M = 154.9 → CrO₂Cl₂（154.9）✓
4. 四面体AB₂C₂型：2σ（分别含Cr-O和Cr-Cl键）+ C₂轴
5. 遇水冒白烟：CrO₂Cl₂ + H₂O → CrO₃ + 2HCl↑

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 摩尔质量验证 | 154.9恰好等于CrO₂Cl₂ |
| 对称性 | C₂ᵥ点群，不是D₂ₕ |
| 遇水反应 | 产生HCl白烟，不是Cl₂ |
