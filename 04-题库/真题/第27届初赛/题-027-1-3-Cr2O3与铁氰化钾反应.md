# 题-027-1-3：Cr₂O₃与铁氰化钾碱性反应

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第1题(12分) 第(1-3)小问

在碱性溶液中，$\mathrm{Cr}_{2}\mathrm{O}_{3}$ 和 $\mathrm{K}_{3}\mathrm{Fe(CN)}_{6}$ 反应。

## 参考答案

$$\mathrm{Cr}_{2}\mathrm{O}_{3} + 6\mathrm{Fe(CN)}_{6}^{3-} + 10\mathrm{OH}^{-} = 2\mathrm{CrO}_{4}^{2-} + 6\mathrm{Fe(CN)}_{6}^{4-} + 5\mathrm{H}_{2}\mathrm{O}$$

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 酸碱性对氧化还原的影响 | 碱性条件有利于Cr(III)→Cr(VI) |
| 配位稳定化 | CN⁻稳定Fe(II)和Fe(III) |
| 重铬酸根的逆向反应 | 酸性中Cr₂O₇²⁻氧化Fe²⁺，碱性中逆向 |

## 解题思路

1. 酸性中经典反应：Cr₂O₇²⁻ + 6Fe²⁺ + 14H⁺ → 2Cr³⁺ + 6Fe³⁺ + 7H₂O
2. 此题条件倒置：碱性 + 配位CN⁻ → 逆向反应有利
3. Cr₂O₃（Cr为+3）被氧化为CrO₄²⁻（Cr为+6）
4. Fe(CN)₆³⁻（Fe为+3）被还原为Fe(CN)₆⁴⁻（Fe为+2）

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 反应方向 | 与酸性条件下的经典反应方向相反 |
| 配合物写法 | 保持配合物整体书写，不要拆开 |
| OH⁻配平 | 碱性介质用OH⁻配平电荷和原子 |
