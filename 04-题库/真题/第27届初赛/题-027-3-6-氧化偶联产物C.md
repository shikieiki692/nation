# 题-027-3-6：氧化偶联产物C结构

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第3题第(3-6)小问

在提炼金时，A 可被氧化成 C：2A → C + 2e⁻；C 能提高金的溶解速率。画出 C 的结构式。写出 C 和 Au 反应的方程式。

## 参考答案

**C = 硫脲氧化偶联产物（formamidine disulfide，二硫化双甲脒）**

$$\mathrm{C}_{2}\mathrm{S}_{2}\mathrm{N}_{4}\mathrm{H}_{8}^{2+}$$

结构：两分子硫脲通过S-S键偶联：

$$[\mathrm{H}_{2}\mathrm{N}{-}\mathrm{CS}{-}\mathrm{S}{-}\mathrm{CS}{-}\mathrm{NH}_{2}]^{2+} \text{（质子化形式）}$$

C与Au反应：

$$\mathrm{C}_{2}\mathrm{S}_{2}\mathrm{N}_{4}\mathrm{H}_{8}^{2+} + 2\mathrm{CS(NH}_{2})_{2} + 2\mathrm{Au} = 2\mathrm{Au[CS(NH}_{2})_{2}]_{2}^{+}$$

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 氧化偶联 | S-S键形成（类比半胱氨酸） |
| 二硫键 | 蛋白质中常见的交联方式 |
| 再生循环 | C可重新氧化Au，加速溶解 |

## 解题思路

1. 2A - 2e⁻ → C，即两分子硫脲失去2个电子
2. 可能偶联位点：S-S、N-N、C-C
3. N-N键（类似联氨）还原性太强，排除
4. S-S键偶联最合理（类比蛋白质中半胱氨酸→胱氨酸）
5. C是强氧化剂，可再氧化Au → 再生硫脲配合物

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 偶联位点 | S-S键（不是N-N或C-C） |
| 生物类比 | 类似蛋白质中二硫键形成 |
| 催化循环 | C氧化Au后再生硫脲，循环使用 |
