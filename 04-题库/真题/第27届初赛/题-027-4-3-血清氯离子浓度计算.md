# 题-027-4-3：血清氯离子浓度计算

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第4题(7分) 第(4-3)小问

取 0.500 mL 血清，加去离子水、HNO₃ 和指示剂，用硝酸汞溶液滴定至终点，消耗 1.53 mL。以十倍于血清样品体积的水为试样进行空白实验，消耗硝酸汞溶液 0.80 mL。计算该血清样品中氯离子的浓度（mg/100 mL）。

## 参考答案

**201 mg/100 mL**

计算过程：
- 空白校正：实际消耗 = 1.53 - 0.80×0.1 = **1.45 mL**（空白用5mL水但样品用0.5mL水，需按比例校正）
- c(Cl⁻) = 2 × c(Hg(NO₃)₂) × V(Hg)/V(样品)
- c(Cl⁻) = 2 × 9.80×10⁻³ × 1.45/0.500 = 5.68×10⁻² mol/L
- 换算：5.68×10⁻² × 35.45 × 1000/10 = 201 mg/100 mL

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 空白实验 | 校正指示剂消耗的金属离子 |
| 浓度换算 | mol/L → mg/100 mL |
| 临床分析 | 血清氯离子正常范围 |

## 解题思路

1. 空白实验校正指示剂（二苯卡巴腙）消耗的Hg²⁺
2. 空白用5mL水（10倍体积），消耗0.80mL → 0.5mL样品对应空白消耗0.08mL
3. 实际V = 1.53 - 0.08 = 1.45 mL
4. 计算浓度并换算单位

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 空白校正 | 样品消耗需扣除空白，但需按体积比例折算 |
| 体积比例 | 空白用5mL水，样品用0.5mL → 10倍关系 |
| 单位换算 | mol/L → mg/100mL需乘以M和换算因子 |
