# 题-027-2-5：多核钴配合物手性结构

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第2题(23分) 第(2-5)小问

向含 $\left[cis-\mathrm{Co}(\mathrm{NH}_{3})_{4}(\mathrm{H}_{2}\mathrm{O})_{2}\right]^{3+}$ 的溶液中加入氨水，析出含 $\left\{\mathrm{Co}\left[\mathrm{Co}(\mathrm{NH}_{3})_{4}(\mathrm{OH})_{2}\right]_{3}\right\}^{6+}$ 的难溶盐。该离子以羟基为桥键的多核配离子，具有手性。画出其结构。

## 参考答案

三个 $\left[cis-\mathrm{Co}(\mathrm{NH}_{3})_{4}(\mathrm{OH})_{2}\right]$ 单元相当于双齿配体，各以两个OH⁻与中心Co桥连。三个双齿配体围绕中心Co形成**三脚架（propeller）型结构**，必然具有手性（类似三叶螺旋桨）。

结构要点：
- 中心Co配位6个来自三个配体单元的OH
- 每个外围Co各自配位4个NH₃和2个OH（其中OH与中心Co桥联）
- 整体呈C₃对称性，有Δ和Λ两种对映体

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 多核配合物 | 羟基桥联多核钴配合物 |
| 手性判据 | 三脚架型配位→C₃对称→手性 |
| 立体化学 | Δ/Λ对映体 |

## 解题思路

1. 离子组成：1个中心Co + 3个外围Co(OH)₂(NH₃)₄
2. 每个外围单元提供2个OH桥联到中心Co
3. 3个双齿配体=6个配位点→中心Co配位数=6
4. 三脚架排列→螺旋手性

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 桥联方式 | OH是桥联配体，连接中心和外围Co |
| 手性来源 | 三脚架排列（不是中心Co的手性碳） |
| 配位数 | 中心Co=6，外围Co=6（各自完整） |
