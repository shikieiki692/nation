# 题-027-3-3：异构体B结构式

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第3题第(3-3)小问

常温下，A 在水溶液中可发生异构化反应，部分转化成 B。画出 B 的结构式。

## 参考答案

**B = 巯基异氰酸铵（异硫氰酸铵，NH₄SCN）** 或其共价形式 **HS-C≡N**

硫脲（A）与异硫氰酸（B）之间存在互变异构平衡：

$$\mathrm{H}_{2}\mathrm{N}-\mathrm{CS}-\mathrm{NH}_{2} \rightleftharpoons \mathrm{HS}-\mathrm{C}{\equiv}\mathrm{N} + \mathrm{NH}_{3}$$

但在水溶液中的异构化更准确地说是：

$$\mathrm{SC(NH}_{2})_{2} \rightleftharpoons \mathrm{NH}_{4}\mathrm{SCN}$$

B 的结构为 $\mathrm{HS}-\mathrm{C}{\equiv}\mathrm{N}$（或离子形式 $\mathrm{NH}_{4}^{+} + \mathrm{NCS}^{-}$）。

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 互变异构 | 硫脲⇌异硫氰酸 |
| 酸碱平衡 | 巯基有弱酸性 |
| 异构化 | 可逆反应，部分转化 |

## 解题思路

1. A = 硫脲 SC(NH₂)₂
2. 类似酮-烯醇互变异构，硫脲可异构化
3. N上的H迁移到S上 → 巯基 + 氰基
4. B含-SH和-C≡N基团

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 不是NH₄SCN | B是异构化产物，不是起始物 |
| 互变异构 | 类似酮-烯醇互变，而非分解 |
| 部分转化 | 平衡反应，不完全转化 |
