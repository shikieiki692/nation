# 题-027-9-1：同位素标记反应历程

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第9题(12分) 第(9-1)小问

利用 ¹⁸O 标记的甲醇钠研究反应，发现最终产物不含 ¹⁸O。画出中间体的结构简式。

## 参考答案

产物不含 ¹⁸O 说明 C-O 键**没有断裂**（不是简单的Sₙ2取代），而是 **S-O 键断裂**。

**反应历程**：
1. CH₃¹⁸O⁻ 进攻磺酰基S原子
2. 发生加成-消去反应（酯交换），S-O键断裂
3. 反应物以氧负离子形式离去
4. 得到的甲磺酸甲酯（含¹⁸O）是甲基化试剂
5. 甲基化新生成的氧负离子 → 产物（不含¹⁸O）

**中间体**：加成-消去反应的四面体中间体

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 同位素示踪 | ¹⁸O标记判断断键位置 |
| 加成-消去 | 磺酰基上的亲核取代 |
| 酯交换 | 本质是酯交换反应 |

## 解题思路

1. 产物不含¹⁸O → ¹⁸O没有保留在产物中
2. 若是C-O断裂（Sₙ2），¹⁸O应进入产物 → 不是Sₙ2
3. S-O断裂 → ¹⁸O留在甲磺酸酯上 → 不进入产物 ✓
4. 中间体为四面体加成产物

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 断键位置 | S-O键（不是C-O键） |
| ¹⁸O去向 | 留在副产物甲磺酸甲酯中 |
| 反应类型 | 酯交换（不是直接取代） |
