# 题-027-1-1：Mn₂O₃与CO反应方程式

## 题目

> **来源**：第27届中国化学奥林匹克（初赛）第1题(12分) 第(1-1)小问

写出下列化学反应的方程式：

加热时，三氧化二锰与一氧化碳反应产生四氧化三锰。

## 参考答案

$$3\mathrm{Mn}_{2}\mathrm{O}_{3} + \mathrm{CO} = 2\mathrm{Mn}_{3}\mathrm{O}_{4} + \mathrm{CO}_{2}$$

该反应与氧化铁的部分还原反应类似。Mn₂O₃中Mn为+3价，Mn₃O₄中Mn为+8/3价（混合价态），CO被氧化为CO₂。

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 锰的氧化物 | Mn₂O₃（+3）、Mn₃O₄（混合价态+2/+3） |
| CO的还原性 | CO → CO₂，碳从+2升至+4 |
| 方程式配平 | 需注意Mn的非整数氧化态变化 |

## 解题思路

1. 明确反应物：Mn₂O₃ + CO；产物：Mn₃O₄ + CO₂
2. Mn₂O₃中Mn为+3价，Mn₃O₄可视为MnO·Mn₂O₃（Mn为+2和+3混合价态）
3. CO → CO₂ 失去2e⁻
4. 2个Mn从+3降到+8/3（平均），需配平得 3:1:2:1

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| Mn₃O₄价态 | 不是简单的+4/3，而是+2和+3混合，平均+8/3 |
| 配平比例 | 正确比例为3:1→2:1，不是1:1 |
