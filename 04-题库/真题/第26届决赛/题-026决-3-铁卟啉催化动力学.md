# 题-026决-3：铁卟啉催化动力学

## 题目

> **来源**：第26届中国化学奥林匹克（决赛）第3题（20分）

铁卟啉（FeP）催化 $\beta$-胡萝卜素的氧化分解反应。实验测得不同温度下的速率常数。

**3-1**：利用Arrhenius方程，由 $\ln k$ 对 $1/T$ 作图求表观活化能 $E_{\mathrm{a,obs'}}$（结果为 $50.19\,\mathrm{kJ \cdot mol^{-1}}$）。

**3-2**：推导速率方程。假设反应机理为：FeP与底物S快速平衡形成中间体，然后中间体经过决速步转化为产物。利用题目给出的 "$y = ax + b$" 提示进行线性化处理。

**3-3**：计算第二步的活化能 $E_{\mathrm{a,2}}$（结果为 $42.43\,\mathrm{kJ \cdot mol^{-1}}$）。

**3-4**：比较催化反应与非催化反应的活化能，说明催化效率。

**3-5**：利用van't Hoff方程计算反应的 $\Delta_{\mathrm{r}}H^\ominus$ 和 $\Delta_{\mathrm{r}}S^\ominus$。

**3-6**：解释温度对催化反应速率的影响。

**3-7**：计算转向温度（催化效率最高时的温度）。

## 参考答案

### 3-1

Arrhenius方程：

$$k = A \exp\left(-\frac{E_{\mathrm{a}}}{RT}\right)$$

取对数：

$$\ln k = \ln A - \frac{E_{\mathrm{a}}}{R} \cdot \frac{1}{T}$$

由 $\ln k$ 对 $1/T$ 作图，斜率 $= -E_{\mathrm{a}}/R$：

$$E_{\mathrm{a,obs'}} = -R \times \text{斜率} = 8.314 \times 6032 = 50.19\,\mathrm{kJ \cdot mol^{-1}}$$

### 3-2

**反应机理**：

$$\mathrm{FeP + S \underset{k_{-1}}{\overset{k_1}{\rightleftharpoons}} [FeP \cdot S]}$$

$$\mathrm{[FeP \cdot S] \overset{k_2}{\to} 产物}$$

**快速平衡假设**：

$$K = \frac{k_1}{k_{-1}} = \frac{[\mathrm{FeP \cdot S}]}{[\mathrm{FeP}][\mathrm{S}]}$$

$$[\mathrm{FeP \cdot S}] = K[\mathrm{FeP}][\mathrm{S}]$$

**物料守恒**：

$$[\mathrm{FeP}]_0 = [\mathrm{FeP}] + [\mathrm{FeP \cdot S}] = [\mathrm{FeP}](1 + K[\mathrm{S}])$$

$$[\mathrm{FeP}] = \frac{[\mathrm{FeP}]_0}{1 + K[\mathrm{S}]}$$

**速率方程**：

$$v = k_2[\mathrm{FeP \cdot S}] = k_2 K[\mathrm{FeP}][\mathrm{S}] = \frac{k_2 K [\mathrm{FeP}]_0 [\mathrm{S}]}{1 + K[\mathrm{S}]}$$

**线性化**（$y = ax + b$ 形式）：

$$\frac{1}{v} = \frac{1}{k_2 K [\mathrm{FeP}]_0} \cdot \frac{1}{[\mathrm{S}]} + \frac{1}{k_2 [\mathrm{FeP}]_0}$$

令 $y = 1/v$，$x = 1/[\mathrm{S}]$，则：

$$a = \frac{1}{k_2 K [\mathrm{FeP}]_0}, \quad b = \frac{1}{k_2 [\mathrm{FeP}]_0}$$

### 3-3

表观活化能由两部分组成：

$$E_{\mathrm{a,obs'}} = E_{\mathrm{a,2}} + \Delta H_K$$

其中 $\Delta H_K$ 是快速平衡的焓变。

由实验数据：

$$E_{\mathrm{a,2}} = E_{\mathrm{a,obs'}} - \Delta H_K = 50.19 - 7.76 = 42.43\,\mathrm{kJ \cdot mol^{-1}}$$

### 3-4

非催化反应的活化能通常远高于催化反应。FeP催化使 $E_{\mathrm{a}}$ 从非催化的较高值降至 $42.43\,\mathrm{kJ \cdot mol^{-1}}$，速率提高倍数为：

$$\frac{k_{\text{催化}}}{k_{\text{非催化}}} = \exp\left(\frac{E_{\mathrm{a,非催化}} - E_{\mathrm{a,催化}}}{RT}\right)$$

### 3-5

van't Hoff方程：

$$\ln K = -\frac{\Delta_{\mathrm{r}}H^\ominus}{RT} + \frac{\Delta_{\mathrm{r}}S^\ominus}{R}$$

由 $\ln K$ 对 $1/T$ 作图：

- 斜率 $= -\Delta_{\mathrm{r}}H^\ominus/R$
- 截距 $= \Delta_{\mathrm{r}}S^\ominus/R$

$$\Delta_{\mathrm{r}}H^\ominus = -R \times \text{斜率}$$

$$\Delta_{\mathrm{r}}S^\ominus = R \times \text{截距}$$

### 3-6

温度升高对催化反应有双重影响：
1. **速率常数增大**：Arrhenius效应，$k_2$ 随温度升高而增大
2. **平衡常数减小**：若快速平衡放热（$\Delta H < 0$），升温使 $K$ 减小，中间体浓度降低

综合效应：在低温区，升温加速反应；在高温区，平衡常数减小的效应可能超过速率常数增大的效应。

### 3-7

转向温度是催化效率最高时的温度，满足：

$$\frac{d(E_{\mathrm{a,obs'}})}{dT} = 0$$

或由 $\Delta_{\mathrm{r}}G^\ominus = 0$ 条件：

$$T_{\text{转向}} = \frac{\Delta_{\mathrm{r}}H^\ominus}{\Delta_{\mathrm{r}}S^\ominus}$$

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| Arrhenius方程 | $\ln k$ vs $1/T$ 求活化能 |
| 速率方程推导 | 快速平衡+决速步机理 |
| 线性化处理 | $1/v$ vs $1/[S]$ 的双倒数图 |
| 表观活化能 | $E_{\mathrm{a,obs'}} = E_{\mathrm{a,2}} + \Delta H_K$ |
| van't Hoff方程 | $\ln K$ vs $1/T$ 求热力学参数 |
| 催化效率 | 活化能降低导致速率提高 |
| 转向温度 | 催化效率最优温度 |

## 解题思路

1. Arrhenius图的斜率直接给出活化能
2. 速率方程推导：快速平衡→中间体浓度→速率表达式
3. 线性化：双倒数图（类似Lineweaver-Burk图）
4. 表观活化能分解为化学步骤活化能+平衡焓变
5. van't Hoff图给出热力学参数
6. 转向温度：平衡焓变与熵变的比值

## 考点抽象

决赛最难动力学题之一，综合速率方程推导（快速平衡+决速步）、Arrhenius方程应用、van't Hoff方程热力学参数计算。核心是理解"表观活化能"包含化学步骤活化能和平衡焓变两部分，以及催化反应中温度的双重效应。双倒数线性化是处理酶催化/Michaelis-Menten型动力学的标准方法。

## 变式拓展

- 可改变催化体系：酶催化（Michaelis-Menten方程）、酸碱催化、金属催化
- 可引入竞争抑制或变构效应
- 可考查更复杂的机理：稳态近似替代快速平衡
- 可拓展至光催化、电催化动力学

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 表观活化能 | $E_{\mathrm{a,obs'}}$ 不等于真实活化能，需分解为 $E_{\mathrm{a,2}} + \Delta H_K$ |
| 线性化形式 | 双倒数图是 $1/v$ vs $1/[S]$，不是 $v$ vs $[S]$ |
| 快速平衡 vs 稳态 | 快速平衡假设 $k_{-1} \gg k_2$，稳态假设 $d[\text{中间体}]/dt = 0$ |
| van't Hoff符号 | 放热反应斜率为正（$\Delta H < 0$），注意符号 |
| 转向温度 | 是催化效率最优温度，不是反应速率最快温度 |
| 温度双重效应 | 升温既加速反应又可能降低平衡常数，需综合考虑 |
