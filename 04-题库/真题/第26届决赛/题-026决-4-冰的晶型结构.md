# 题-026决-4：冰的晶型结构

## 题目

> **来源**：第26届中国化学奥林匹克（决赛）第4题（15分）

冰有16种已知晶型，本题考查其中几种重要晶型的结构计算。

**4-1**：冰-Ih（六方冰）为六方晶系。已知O—O距离（氢键长）为 $276\,\mathrm{pm}$，六方晶胞参数 $a = 450\,\mathrm{pm}$，$c = 730\,\mathrm{pm}$。计算晶胞中水分子数 $Z$，并解释 $c$ 值偏离理想值的原因。

**4-2**：冰-XI（正交晶系）的晶胞参数 $a = 450\,\mathrm{pm}$，$b = 780\,\mathrm{pm}$，$c = 730\,\mathrm{pm}$，密度 $\rho = 0.92\,\mathrm{g \cdot cm^{-3}}$。计算 $Z$。

**4-3**：冰-Ic（立方冰）为立方晶系，面心立方点阵。O—O距离为 $276\,\mathrm{pm}$，求晶胞参数 $a$ 和 $Z$。

**4-4**：冰-X为高压冰，体心立方结构，$a = 238\,\mathrm{pm}$，密度 $\rho = 2.51\,\mathrm{g \cdot cm^{-3}}$。计算冰-X与冰-Ih的密度比，并解释高压下冰密度增大的原因。

**4-5**：笼状水合物中，水分子形成笼状结构包裹客体分子。已知某笼状结构中水分子的分数坐标，求笼的类型（如十二面体、十四面体等）。

## 参考答案

### 4-1

冰-Ih的晶胞中含有4个水分子（$Z = 4$）。

理想六方晶系中 $c/a = \sqrt{8/3} \approx 1.633$。

$$c_{\text{理想}} = 1.633 \times 450 = 735\,\mathrm{pm}$$

实际 $c = 730\,\mathrm{pm}$，偏差约 $0.7\%$。

**偏差原因**：
1. 氢键的非简谐性导致O—O距离不完全均匀
2. 晶体中存在质子无序（氢原子位置无序）
3. 零点振动能的影响

### 4-2

$$V = a \times b \times c = 450 \times 780 \times 730 = 2.553 \times 10^8\,\mathrm{pm^3} = 2.553 \times 10^{-22}\,\mathrm{cm^3}$$

$$Z = \frac{\rho V N_{\mathrm{A}}}{M(\mathrm{H_2O})} = \frac{0.92 \times 2.553 \times 10^{-22} \times 6.022 \times 10^{23}}{18.015} \approx 8$$

### 4-3

冰-Ic为面心立方点阵，每个晶胞含4个水分子（$Z = 4$）。

面心立方中，O—O最近距离 $= a/\sqrt{2}$：

$$a = \sqrt{2} \times 276 = 390\,\mathrm{pm}$$

验证：面心立方中每个顶点和面心原子与最近邻的距离为 $a/\sqrt{2}$，对应氢键长度。

### 4-4

冰-X的晶胞体积：

$$V = a^3 = (238 \times 10^{-10})^3 = 1.348 \times 10^{-23}\,\mathrm{cm^3}$$

体心立方 $Z = 2$：

$$\rho_{\text{冰-X}} = \frac{2 \times 18.015}{6.022 \times 10^{23} \times 1.348 \times 10^{-23}} = 4.44\,\mathrm{g \cdot cm^{-3}}$$

（注：题目给出2.51 g/cm³，需根据实际条件调整。）

密度比：

$$\frac{\rho_{\text{冰-X}}}{\rho_{\text{冰-Ih}}} = \frac{2.51}{0.92} \approx 2.73$$

**高压下密度增大原因**：
1. 高压压缩O—O距离，氢键变短
2. 分子排列更紧密，配位数增加
3. 氢键从弯曲变为对称（冰-X中H位于O—O中点）

### 4-5

笼状水合物中常见的笼类型：
- **5¹²**：十二面体（12个五边形面），由20个水分子组成
- **5¹²6²**：十四面体（12个五边形+2个六边形），由24个水分子组成
- **5¹²6⁴**：十六面体（12个五边形+4个六边形），由28个水分子组成

根据分数坐标可确定笼的类型和大小。

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 六方冰 Ih | 六方晶系，Z=4 |
| 立方冰 Ic | 面心立方，Z=4 |
| 高压冰 X | 体心立方，对称氢键 |
| 密度计算 | $\rho = ZM/(N_{\mathrm{A}}V)$ |
| 分数坐标 | 确定笼状结构类型 |
| 氢键几何 | O—O距离与晶胞参数关系 |

## 解题思路

1. 六方冰：利用 $c/a$ 比值判断结构偏离
2. 正交冰-XI：直接用密度公式求Z
3. 立方冰-Ic：面心立方几何关系 $d = a/\sqrt{2}$
4. 高压冰-X：密度比反映压缩程度
5. 笼状结构：根据面数和顶点数判断笼类型

## 考点抽象

晶体结构计算的综合应用，涵盖不同晶系（六方、正交、立方）的晶胞参数计算、点阵类型判断（面心立方、体心立方）、分数坐标应用以及密度比较。冰的多种晶型展示了同一物质在不同条件下可形成截然不同的晶体结构，是晶体化学的经典案例。

## 变式拓展

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 六方冰Z值 | 冰-Ih的Z=4（六方晶胞），不是Z=2 |
| 面心立方d=a/√2 | 最近邻距离是面对角线的一半，不是边长 |
| 体心立方Z=2 | 体心立方晶胞含2个原子/分子 |
| 密度单位 | pm³→cm³需乘(10⁻¹⁰)³=10⁻³⁰ |
| c/a理想值 | 六方冰理想比值为√(8/3)≈1.633，不是√3 |
| 笼状结构 | 5¹²表示12个五边形面，不是5个十二边形 |
