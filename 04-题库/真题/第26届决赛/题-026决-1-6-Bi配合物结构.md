# 题-026决-1-6：Bi配合物结构

## 题目

> **来源**：第26届中国化学奥林匹克（决赛）第1题（1-6）

甲基黄原酸铋配合物 $\mathrm{A}$ 是一种重要的铋配合物。

**1-6-1**：写出 $\mathrm{A}$ 的化学式。

**1-6-2**：写出由 $\mathrm{BiCl_2CH_3}$ 和 $\mathrm{NaS_2COMe}$ 制备 $\mathrm{A}$ 的反应方程式。

**1-6-3**：制备过程中能否用水作溶剂？为什么？

**1-6-4**：$\mathrm{La^{3+}}$ 能否取代配合物 $\mathrm{A}$ 中的 $\mathrm{Bi^{3+}}$？为什么？

## 参考答案

### 1-6-1

$\mathrm{A}$ 的化学式为 $\mathrm{CH_3Bi(S_2COCH_3)_2}$。

### 1-6-2

$$\mathrm{BiCl_2CH_3 + 2NaS_2COMe \to CH_3Bi(S_2COCH_3)_2 + 2NaCl}$$

### 1-6-3

**不能用水作溶剂**。

$\mathrm{BiCl_2CH_3}$ 是金属有机化合物，极易水解：

$$\mathrm{BiCl_2CH_3 + H_2O \to Bi(OH)_2CH_3 + 2HCl}$$

或进一步水解生成 $\mathrm{Bi(OH)_3}$ 和甲烷。因此制备必须在无水条件下进行，通常使用有机溶剂（如乙醇、乙醚等）。

### 1-6-4

**$\mathrm{La^{3+}}$ 不能取代 $\mathrm{Bi^{3+}}$**。

根据软硬酸碱理论（HSAB）：
- $\mathrm{Bi^{3+}}$ 是较软的酸（大半径、低电荷密度、易极化）
- $\mathrm{S}$ 原子是软碱
- $\mathrm{Bi^{3+}}$ 与 $\mathrm{S}$ 形成稳定的软-软配合物

而 $\mathrm{La^{3+}}$ 是典型的硬酸（小半径、高电荷密度、不易极化），与软碱 $\mathrm{S}$ 的结合能力很弱。因此 $\mathrm{La^{3+}}$ 无法取代 $\mathrm{Bi^{3+}}$。

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 软硬酸碱理论 | Bi³⁺(软酸)与S(软碱)的亲和性 |
| 金属有机化合物水解 | Bi-Cl键易受水进攻 |
| 配合物化学式推导 | 根据配体数目和电荷平衡 |
| 配位化学计量 | 1:2的金属:配体比 |

## 解题思路

1. 根据Bi的氧化态(+3)和黄原酸根(-1)的电荷平衡确定配体数
2. 金属有机Bi-Cl键具有共价性，易水解
3. 软硬酸碱匹配：Bi³⁺(软)+S(软) vs La³⁺(硬)+S(软)

## 考点抽象

软硬酸碱理论在配合物稳定性中的应用是本题核心。Bi³⁺作为大半径、低电荷密度的阳离子，属于软酸，与软碱S有强亲和力。La³⁺作为典型的硬酸，无法与软碱S形成稳定配合物。这一原理可推广至其他主族金属有机化合物的稳定性和反应性分析。

## 变式拓展

- 可延伸考查Pb²⁺、Cd²⁺等软酸与含S配体的配合物
- 可对比Bi³⁺与Ga³⁺、In³⁺在软硬度上的差异
- 可拓展至Bi的硫化物（Bi₂S₃）的难溶性（软酸+软碱→难溶）
- 金属有机化合物水解是主族有机金属化学的普遍问题

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 化学式配平 | Bi(+3)+CH₃(-1)+2×黄原酸(-1)=0，电荷平衡 |
| 水解产物 | Bi-CH₃键断裂产生CH₄（或Bi(OH)₂CH₃），不是简单复分解 |
| HSAB判断 | Bi³⁺是软酸不是硬酸，不要因为Bi在第VA族就认为是硬酸 |
| La³⁺的酸碱性 | La³⁺是典型的硬酸（镧系收缩前），与S配位能力极弱 |
