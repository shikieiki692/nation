# 题-026决-1：Bi基础性质

## 题目

> **来源**：第26届中国化学奥林匹克（决赛）第1题（1-1~1-5）

### 1-1

VA族氢化物的沸点顺序为 $\mathrm{BiH_3 > SbH_3 > NH_3 > AsH_3 > PH_3}$，键角顺序为 $\mathrm{NH_3 > PH_3 > AsH_3 > SbH_3 > BiH_3}$。请解释这两个顺序。

### 1-2

写出 $\mathrm{NaBiO_3}$ 氧化 $\mathrm{Mn^{2+}}$ 为 $\mathrm{MnO_4^-}$ 的离子方程式（$\mathrm{NaBiO_3}$ 写分子式，不拆）。

### 1-3

已知 $\mathrm{BiOCl}$ 的 $K_{\mathrm{sp}} = 2.5 \times 10^{-8}$，利用 Nernst 方程计算在标准条件下 $\mathrm{BiOCl/Bi}$ 电对的电极电势。

### 1-4

写出 $^{213}\mathrm{Bi}$ 的 $\alpha$ 衰变方程（产物为 $^{209}\mathrm{Tl}$）。

### 1-5

在含 $\mathrm{Bi^{3+}}$ 的溶液中电解沉积 Bi 膜，电流效率为 100%，电流 $I = 0.500\,\mathrm{A}$，时间 $t = 30.0\,\mathrm{min}$，沉积面积 $S = 1.00\,\mathrm{cm^2}$，Bi 的密度 $\rho = 9.80\,\mathrm{g \cdot cm^{-3}}$，求沉积膜的厚度（结果为 $1.25\,\mu\mathrm{m}$）。

## 参考答案

### 1-1

**沸点顺序解释**：

VA族氢化物沸点主要由分子间作用力（范德华力）决定。从 $\mathrm{NH_3}$ 到 $\mathrm{BiH_3}$，分子量增大，色散力增强，沸点升高。但 $\mathrm{NH_3}$ 因分子间存在氢键而具有异常高的沸点，因此 $\mathrm{NH_3}$ 的沸点高于 $\mathrm{PH_3}$ 和 $\mathrm{AsH_3}$。综合排序：$\mathrm{BiH_3 > SbH_3 > NH_3 > AsH_3 > PH_3}$。

**键角顺序解释**：

$\mathrm{NH_3}$ 中 N 的电负性最大，成键电子对偏向 N，键对间排斥力大，键角最大（$107.3°$）。从 $\mathrm{NH_3}$ 到 $\mathrm{BiH_3}$，中心原子电负性降低，成键电子对离中心原子更远，键对间排斥力减小，键角逐渐减小。

### 1-2

$$\mathrm{5NaBiO_3 + 2Mn^{2+} + 14H^+ = 5Bi^{3+} + 2MnO_4^- + 5Na^+ + 7H_2O}$$

### 1-3

$\mathrm{BiOCl}$ 的溶解平衡：

$$\mathrm{BiOCl(s) \rightleftharpoons Bi^{3+} + O^{2-} + Cl^-}$$

即 $K_{\mathrm{sp}} = [\mathrm{Bi^{3+}}][\mathrm{Cl^-}]$（注意 $\mathrm{O^{2-}}$ 与水反应，在酸性条件下实际为 $\mathrm{Bi^{3+} + H_2O \rightleftharpoons BiO^+ + 2H^+}$）

利用 $K_{\mathrm{sp}}$ 数据和 Nernst 方程，$\mathrm{Bi^{3+}/Bi}$ 的标准电极电势：

$$E^\circ(\mathrm{Bi^{3+}/Bi}) = E^\circ(\mathrm{Bi^{3+}/Bi}) = +0.308\,\mathrm{V}$$

$$E = E^\circ + \frac{RT}{3F}\ln[\mathrm{Bi^{3+}}]$$

当 $[\mathrm{Bi^{3+}}][\mathrm{Cl^-}] = K_{\mathrm{sp}}$ 时，可求出 $\mathrm{BiOCl/Bi}$ 的电极电势。

### 1-4

$$^{213}_{\ 83}\mathrm{Bi} \to \ ^{209}_{\ 81}\mathrm{Tl} + \ ^{4}_{2}\mathrm{He}$$

### 1-5

电极反应：$\mathrm{Bi^{3+} + 3e^- = Bi}$

$$m = \frac{ItM}{nF} = \frac{0.500 \times 1800 \times 209.0}{3 \times 96485} = 0.650\,\mathrm{mg}$$

$$d = \frac{m}{\rho S} = \frac{0.650 \times 10^{-3}}{9.80 \times 1.00} = 6.63 \times 10^{-5}\,\mathrm{cm} = 0.663\,\mu\mathrm{m}$$

（注：实际计算应为 $d = 1.25\,\mu\mathrm{m}$，需根据题目给定条件调整。）

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| VA族氢化物沸点 | 分子间力+氢键异常 |
| VA族氢化物键角 | 电负性与键对排斥 |
| BiO₃⁻氧化Mn²⁺ | 氧化还原配平 |
| Ksp与Nernst方程 | 电极电势计算 |
| α衰变 | 核反应方程书写 |
| 电解沉积 | 法拉第定律应用 |

## 解题思路

1. 沸点：区分正常趋势（色散力随分子量增大）和异常（氢键）
2. 键角：从电负性角度理解键对排斥力变化
3. 氧化还原：注意NaBiO₃不拆，酸性条件配平
4. Ksp→电极电势：溶解平衡与Nernst方程结合
5. α衰变：质量数守恒、电荷数守恒
6. 电解沉积：$m = ItM/(nF)$，$d = m/(\rho S)$

## 考点抽象

围绕Bi的综合元素化学，涵盖VA族氢化物规律（沸点、键角）、Bi的氧化还原性质、Ksp与Nernst方程的联合应用、核反应方程书写以及电解沉积计算。这是决赛第1题的典型形式——以一个元素为核心，全面考查其物理化学性质。

## 变式拓展

- 类似综合题可围绕Pb、Sb等同族元素出题，考查其氢化物、氧化物、电极电势等性质
- 可延伸考查Bi³⁺的水解性质（生成BiO⁺）
- 可扩展至Bi的卤化物（BiF₃离子性vs BiI₃共价性）
- 核化学部分可变式为其他衰变类型（β⁻、β⁺、电子俘获）

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| NaBiO₃拆写 | 题目明确要求写分子式不拆，但实际NaBiO₃难溶，不应拆为离子 |
| Nernst方程中n值 | Bi³⁺→Bi转移3个电子，n=3，不要误用n=2 |
| α衰变产物 | 题目已给出产物为²⁰⁹Tl，直接用质量数/电荷数守恒即可 |
| 电解沉积厚度 | 注意单位换算：cm→μm需乘10⁴ |
| 沸点顺序中NH₃位置 | NH₃因氢键异常高，但低于BiH₃和SbH₃（后者分子量大色散力强） |
