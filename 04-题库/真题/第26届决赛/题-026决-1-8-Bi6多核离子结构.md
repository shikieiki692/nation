# 题-026决-1-8：Bi₆多核离子结构

## 题目

> **来源**：第26届中国化学奥林匹克（决赛）第1题（1-8）

$\mathrm{Bi_6(OH)_{12}^{6+}}$ 是一个多核铋配合物离子。

- 6个Bi原子占据立方体的6个面心位置
- 12个OH⁻占据立方体的12条棱心位置
- Bi—Bi键长为370 pm（金属Bi中Bi—Bi键长为307 pm和353 pm）

**1-8-1**：画出该离子的结构示意图，标注对称性。

**1-8-2**：解释Bi—Bi键长（370 pm）与金属Bi中键长（307/353 pm）的差异。

**1-8-3**：计算该离子的形式电荷并解释其稳定性。

## 参考答案

### 1-8-1

结构示意图：

```
        OH    OH
         |    |
    OH---Bi---Bi---OH
   /|    |    |    |\
  OH Bi        Bi OH
   \|    |    |    |/
    OH---Bi---Bi---OH
         |    |
        OH    OH
```

6个Bi位于立方体6个面心，12个OH位于12条棱心。对称性为 $O_h$（立方体对称性）。

### 1-8-2

$\mathrm{Bi_6(OH)_{12}^{6+}}$ 中Bi—Bi键长（370 pm）比金属Bi中的键长（307/353 pm）长，原因：

1. **正电荷排斥**：离子带6+电荷，Bi原子间存在强烈的静电排斥，拉长了Bi—Bi键
2. **OH⁻的桥联作用**：OH⁻作为桥联配体，在Bi原子间形成Bi—OH—Bi桥，限制了Bi—Bi的接近
3. **电子结构**：Bi³⁺的6s²孤对电子可能参与成键，但正电荷使金属-金属相互作用减弱

### 1-8-3

形式电荷计算：

- 6个Bi³⁺：$6 \times (+3) = +18$
- 12个OH⁻：$12 \times (-1) = -12$
- 总电荷：$+18 - 12 = +6$

因此离子形式为 $\mathrm{Bi_6(OH)_{12}^{6+}}$。

稳定性来源：
1. 高对称性（$O_h$）降低了体系能量
2. OH⁻桥联形成稳定的六元环结构
3. Bi³⁺的6s²孤对电子可能提供额外的稳定化能（次级键作用）

## 知识点映射

| 知识点 | 对应内容 |
|:---|:---|
| 多核配合物 | 6个金属中心的簇合物 |
| 对称性分析 | 立方体Oh对称性 |
| 电荷约束 | 形式电荷=金属电荷+配体电荷 |
| 金属-金属键 | Bi-Bi键长与金属态对比 |
| 桥联配体 | OH⁻的μ₂桥联模式 |

## 解题思路

1. 根据面心立方体模型确定原子位置
2. 利用Bi的氧化态(+3)和OH的电荷(-1)计算形式电荷
3. 对比键长差异，从电荷排斥和配体桥联角度解释
4. 分析对称性对稳定性的贡献

## 考点抽象

利用对称性和电荷约束推断多核配合物结构是本题核心。给定原子位置（面心、棱心）可直接确定对称性（立方体Oh），再通过电荷平衡验证化学式的合理性。键长差异反映了多核离子中金属-金属相互作用受电荷和配体影响的特点。这类题目训练从对称性出发推断结构的能力。

## 变式拓展

- 可延伸至其他多核配合物：$\mathrm{[Mo_6Cl_8]^{4+}}$（八面体簇）、$\mathrm{[Re_6Se_8]^{2+}}$
- 可拓展至主族元素簇合物：$\mathrm{Pb_5^{2-}}$、$\mathrm{Bi_5^{3+}}$（三角双锥）
- 可结合分子轨道理论解释金属-金属多重键
- 桥联模式可扩展至μ₃、μ₄桥联

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 对称性判断 | 面心+棱心→立方体Oh，不要误判为Oh以外的对称性 |
| 形式电荷 | Bi为+3不是+5（Bi³⁺的6s²稳定），注意氧化态 |
| 键长比较 | 370pm比金属Bi长，说明多核离子中Bi-Bi作用减弱 |
| 桥联模式 | 每个OH桥接2个Bi（μ₂模式），12个OH正好连接6个Bi的12条棱 |
| 电荷验证 | 6×(+3)+12×(-1)=+6，与题目给出的6+一致 |
