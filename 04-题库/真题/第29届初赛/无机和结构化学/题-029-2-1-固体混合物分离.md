# 题-029-2-1：固体混合物分离

## 题目
> **来源**：[[07-资料提炼/提炼-第29届初赛试题解析|第29届初赛第2题]]（原始文件：[[mineru/02-真题解析/29届初赛试题解析]]）
> **小问关联**：本文件为第(2-1)小问；其余小问见 [[题-029-2-2-SO2与KI反应]]、[[题-029-2-3-1-聚乙二醇结构判断]]、[[题-029-2-3-2-聚乙二醇保水原理]]、[[题-029-2-3-3-环氧乙烷聚合方程式]]

实验室现有试剂：盐酸、硝酸、乙酸、氢氧化钠和氨水。从中选择一种试剂，分别分离以下各组固体混合物（不要求复原，括号内数据是溶度积），指出溶解的固体。

(1) $\mathrm{CaCO}_{3}(3.4 \times 10^{-9})$ 和 $\mathrm{CaC}_{2} \mathrm{O}_{4}(2.3 \times 10^{-9})$   
(2) $\mathrm{BaSO}_4$ $(1.1\times 10^{-10})$ 和 $\mathrm{BaCrO}_4$ $(1.1\times 10^{-10})$   
(3) $\mathrm{Zn(OH)}_2$ $(3.0\times 10^{-17})$ 和 $\mathrm{Ni(OH)}_2$ $(5.5\times 10^{-16})$   
(4) $\mathrm{AgCl}(1.8\times 10^{-10})$ 和 $\mathrm{AgI}(8.5\times 10^{-17})$   
(5) $\mathrm{ZnS}(2.5\times 10^{-22})$ 和 $\mathrm{HgS}(1.6\times 10^{-52})$

## 参考答案

(1) 选用乙酸，$\mathrm{CaCO}_{3}$ 溶解而 $\mathrm{CaC}_{2}\mathrm{O}_{4}$ 不溶。  
(2) 选用硝酸或盐酸，$\mathrm{BaCrO}_{4}$ 溶解而 $\mathrm{BaSO}_{4}$ 不溶。  
(3) 选用 NaOH，$\mathrm{Zn(OH)}_{2}$ 溶解而 $\mathrm{Ni(OH)}_{2}$ 不溶。  
(4) 选用氨水，$\mathrm{AgCl}$ 溶解而 $\mathrm{AgI}$ 不溶。  
(5) 选用盐酸或硝酸，$\mathrm{ZnS}$ 溶解而 $\mathrm{HgS}$ 不溶。

## 解析

固体混合物的分离方法常常用于化合物的定性与定量分析上，本题要求同学们通过分析沉淀的组成离子性质，结合基本的元素化学知识解答。

(1) $\mathrm{CaCO}_{3}$ 和 $\mathrm{CaC}_{2}\mathrm{O}_{4}$ 的 $K_{\mathrm{sp}}$ 接近，但 $\mathrm{H}_{2}\mathrm{C}_{2}\mathrm{O}_{4}$ 是比 $\mathrm{CH}_{3}\mathrm{COOH}$ 更强的酸，$\mathrm{H}_{2}\mathrm{CO}_{3}$ 是比 $\mathrm{CH}_{3}\mathrm{COOH}$ 更弱的酸，因此 $\mathrm{CaCO}_{3}$ 可以溶于 $\mathrm{CH}_{3}\mathrm{COOH}$ 而 $\mathrm{CaC}_{2}\mathrm{O}_{4}$ 不能。

(2) 两种沉淀也具有相近的 $K_{\mathrm{sp}}$，但 $\mathrm{CrO}_{4}^{2-}$ 是比 $\mathrm{SO}_{4}^{2-}$ 更强的碱，因此可以用硝酸或盐酸使 $\mathrm{BaCrO}_{4}$ 溶解。

(3) $\mathrm{Zn}^{2+}$ 具有两性，可以和 $\mathrm{OH}^{-}$ 形成配离子而 $\mathrm{Ni}^{2+}$ 不可，故可使用 NaOH 分离这两者的混合物。

(4)(5) 同学们可以通过基本的元素化学知识解题：AgCl 溶于氨水而 AgI 不溶。ZnS 溶于强酸，HgS 是极难溶的沉淀。据此，(4) 选用氨水使 AgCl 溶解，(5) 选用盐酸或硝酸使 ZnS 溶解。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 溶度积 | 利用 $K_{\mathrm{sp}}$ 和酸的强度差异选择分离试剂 |
| 酸碱性质 | 碳酸盐与草酸盐在弱酸中溶解性的差异 |
| 两性氢氧化物 | Zn(OH)₂的两性使其可溶于NaOH，Ni(OH)₂不溶 |
| 配位化学 | AgCl溶于氨水形成[Ag(NH₃)₂]⁺，AgI不溶 |
| 硫化物溶解性 | ZnS溶于强酸，HgS极难溶 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 仅根据 $K_{\mathrm{sp}}$ 大小判断 | 需结合组成离子的酸碱性质、配位性质综合分析 |
| 用强酸分离(1) | CaC₂O₄在强酸中也会溶解，需用弱酸乙酸选择性溶解CaCO₃ |
| 忽略Zn(OH)₂的两性 | Zn²⁺具有两性，可与过量OH⁻形成配离子 |
| 对HgS溶解性认识不足 | HgS极难溶，甚至不溶于强酸，需用王水溶解 |
