# 题-026-11-5：臭氧化产物D结构

## 题目

> **来源**：第26届中国化学奥林匹克（初赛）第11题(12分) 第(11-5)小问

画出化合物 D 的结构简式。

## 参考答案

C在NaOH/H₂O条件下：

1. C2半缩醛→碱夺取OH上的H→四面体中间体
2. 消除按方向b进行（方向a产生单线态氧，不稳定）
3. 碎片化反应→得到二羰基化合物
4. 碱性条件下分子内**羟醛缩合**→六元环α,β-不饱和酮 = D

**D = 六元环α,β-不饱和酮**（环己烯酮衍生物）

## 知识点映射

- [[碎片化反应]]
- [[羟醛缩合]]
- αβ-不饱和酮

## 解题思路

1. C2半缩醛→碱催化开环
2. 过氧负离子碎片化→二羰基中间体
3. 分子内羟醛缩合→六元环
4. 脱水→α,β-不饱和酮

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 消除方向 | 方向b（碎片化）vs 方向a（单线态氧，排除） |
| 最终步骤 | 二羰基→羟醛缩合→六元环→脱水 |
| 不是直接产物 | D不是碎片化直接产物，还需羟醛缩合 |
