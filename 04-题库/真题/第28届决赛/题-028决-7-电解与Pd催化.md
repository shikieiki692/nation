# 第 7 题 电解与 Pd 催化 (16 分)

## 题目

本题分为两部分：第一部分涉及 $\mathrm{PbCl_2/CuCl_2}$ 溶液的电解，第二部分涉及 Wacker 法催化乙烯氧化。

### 第一部分：电解

### 7-1 (2分)

以 $\mathrm{Cu}$ 为阴极电解 $\mathrm{PbCl_2}$ 和 $\mathrm{CuCl_2}$ 的混合溶液。写出阴极和阳极的反应，并计算理论分解电压。

### 7-2 (2分)

判断在阴极是否可能生成 $\mathrm{CuOH}$（或 $\mathrm{Cu_2O}$）。通过计算 $\mathrm{CuOH}$ 的溶度积和相关平衡进行分析。

### 7-3 (2分)

计算 $\mathrm{Pb^{2+}}$ 和 $\mathrm{Cu^{2+}}$ 在溶液中水解的平衡常数，并讨论 pH 对电解的影响。

### 7-4 (2分)

考虑 $\mathrm{Cl_2}$ 在阳极析出的超电势，讨论其对电解可行性的影响。

### 第二部分：Wacker 法催化

### 7-5 (8分)

Wacker 法以 $\mathrm{PdCl_2/CuCl_2}$ 为催化剂，将乙烯氧化为乙醛。总反应：

$$\mathrm{CH_2{=}CH_2 + \frac{1}{2}O_2 \xrightarrow{PdCl_2/CuCl_2} CH_3CHO}$$

(a) 实验测得速率方程为 $r = k[\mathrm{Pd^{2+}}]^x[\mathrm{Cu^{2+}}]^y[\mathrm{Cl^-}]^z[\mathrm{O_2}]^\omega[\mathrm{C_2H_4}]^1$，其中 $x = 1, y = 1, z = -1, \omega = -2$。写出完整速率方程并解释各指数的物理意义。

(b) 确定反应的决速步。

(c) 讨论 Wacker 法的立体化学选择性。

(d) 在催化循环中，$\mathrm{Cl^-}$ 与 $\mathrm{Pd}$ 的物质的量之比对反应速率有何影响？

(e) 解释 $\mathrm{CuCl_2}$ 在催化循环中的作用。

---

## 参考答案

### 7-1 答案

阴极反应：$\mathrm{Cu^{2+} + 2e^- \rightarrow Cu(s)}$（$E^\ominus = +0.34\,\mathrm{V}$，优先于 $\mathrm{Pb^{2+}}$ 还原）

阳极反应：$\mathrm{2Cl^- \rightarrow Cl_2(g) + 2e^-}$（$E^\ominus = +1.36\,\mathrm{V}$）

理论分解电压：

$$E_\text{分解} = E_\text{阳极} - E_\text{阴极} = 1.36 - 0.34 = 1.02\,\mathrm{V}$$ (2 分)

### 7-2 答案

$\mathrm{CuOH}$ 的 $K_{\mathrm{sp}}$ 约为 $10^{-14}$ 量级。在阴极附近，$\mathrm{Cu^{2+}}$ 还原为 $\mathrm{Cu}$，不会生成 $\mathrm{CuOH}$。但如果阴极电位过负，可能发生副反应 $\mathrm{Cu^{2+} + OH^- + e^- \rightarrow CuOH}$，这需要通过 Nernst 方程计算在实际 pH 下的可能性。(2 分)

### 7-3 答案

$\mathrm{Pb^{2+}}$ 水解：$\mathrm{Pb^{2+} + H_2O \rightleftharpoons PbOH^+ + H^+}$，$K_h \approx 10^{-8}$

$\mathrm{Cu^{2+}}$ 水解：$\mathrm{Cu^{2+} + H_2O \rightleftharpoons CuOH^+ + H^+}$，$K_h \approx 10^{-7}$

$\mathrm{Cu^{2+}}$ 水解程度略大于 $\mathrm{Pb^{2+}}$。在酸性溶液中（低 pH），水解被抑制，有利于电解的正常进行。若 pH 过高，可能生成氢氧化物沉淀。(2 分)

### 7-4 答案

$\mathrm{Cl_2}$ 在电极上的析出超电势通常较大（约 $0.1\text{-}0.3\,\mathrm{V}$，取决于电极材料和电流密度）。实际分解电压：

$$E_\text{实际} = E_\text{理论} + \eta_\text{阳极} + \eta_\text{阴极} + IR$$

若超电势过大，可能使实际分解电压超过 $\mathrm{O_2}$ 析出的电位（$1.23\,\mathrm{V}$），此时阳极可能发生水的氧化而非 $\mathrm{Cl^-}$ 的氧化。选择合适的电极材料（如石墨、$\mathrm{RuO_2}$）可降低 $\mathrm{Cl_2}$ 析出的超电势。(2 分)

### 7-5 答案

(a) 速率方程：

$$r = k \frac{[\mathrm{Pd^{2+}}][\mathrm{Cu^{2+}}][\mathrm{C_2H_4}]}{[\mathrm{Cl^-}][\mathrm{O_2}]^2}$$

- $x = 1$：$\mathrm{Pd^{2+}}$ 为活性催化中心，速率与其浓度成正比
- $y = 1$：$\mathrm{Cu^{2+}}$ 作为氧化剂再生 $\mathrm{Pd^{2+}}$，速率与其浓度成正比
- $z = -1$：$\mathrm{Cl^-}$ 与 $\mathrm{Pd^{2+}}$ 配位形成 $\mathrm{[PdCl_4]^{2-}}$，降低活性 $\mathrm{Pd^{2+}}$ 浓度，表现为抑制效应
- $\omega = -2$：$\mathrm{O_2}$ 参与 $\mathrm{Cu^{+}}$ 的再氧化，但高浓度 $\mathrm{O_2}$ 可能干扰催化循环中的某些步骤 (2 分)

(b) 决速步为 $\mathrm{[PdCl_3(C_2H_4)]^- + H_2O \rightarrow Pd(s) + CH_3CHO + 3Cl^- + 2H^+}$（即乙烯-Pd配合物的水解步骤）。该步骤涉及配体交换和 C-H 键的活化，活化能最高。(1.5 分)

(c) Wacker 法中乙烯氧化为乙醛，产物为反 Markovnikov 加成的醛（而非 Markovnikov 加成的醇）。该反应具有高度的区域选择性：$\mathrm{Pd^{2+}}$ 与乙烯配位后，水从反面进攻，最终生成乙醛。对于不对称烯烃，主要产物遵循反 Markovnikov 规则（生成醛而非酮）。(1.5 分)

(d) $\mathrm{Cl^-}$ 与 $\mathrm{Pd}$ 的比值增大（即 $\mathrm{Cl^-}$ 过量），会形成更多 $\mathrm{[PdCl_4]^{2-}}$，降低自由 $\mathrm{Pd^{2+}}$ 浓度，使速率降低（$z = -1$）。但 $\mathrm{Cl^-}$ 浓度过低又不利于 $\mathrm{PdCl_2}$ 的溶解和催化循环的稳定。最佳 $\mathrm{Cl^-/Pd}$ 比需通过实验优化。(1 分)

(e) $\mathrm{CuCl_2}$ 的作用是将还原产生的 $\mathrm{Pd(0)}$ 重新氧化为 $\mathrm{Pd(II)}$，实现催化循环。自身被还原为 $\mathrm{Cu(I)}$，再被 $\mathrm{O_2}$ 氧化回 $\mathrm{Cu(II)}$：

$$\mathrm{Pd(0) + 2Cu^{2+} \rightarrow Pd^{2+} + 2Cu^+}$$
$$\mathrm{2Cu^+ + \frac{1}{2}O_2 + 2H^+ \rightarrow 2Cu^{2+} + H_2O}$$ (2 分)

## 解题思路

1. 电解部分：阴极优先还原电位更高的 $\mathrm{Cu^{2+}}$，阳极氧化 $\mathrm{Cl^-}$
2. 理论分解电压 = $E_\text{阳极} - E_\text{阴极}$
3. 超电势使实际分解电压高于理论值
4. Wacker 法：根据速率方程中各指数判断各物质的作用
5. 决速步 = 活化能最高的步骤 = 乙烯-Pd配合物的水解
6. 立体化学 = 反 Markovnikov 加成

## 易错分析

- 电解时忘记比较各离子的标准电极电位，错误判断阴极产物
- 速率方程中 $\mathrm{O_2}$ 的指数为负：误认为反应物浓度增大一定加速反应
- Wacker 法中误认为 $\mathrm{Cu^{2+}}$ 直接参与乙烯的氧化
