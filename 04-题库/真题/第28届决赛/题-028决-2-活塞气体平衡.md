# 第 2 题 活塞气体平衡 (15 分)

## 题目

如图所示，在恒温条件下，一个被活塞分隔成 4 部分的容器中发生可逆反应 $\mathrm{A(g) + B(g) \rightleftharpoons C(g)}$。各部分温度相同，活塞可自由滑动且无摩擦。

### 2-1 (3分)

给出反应商 $Q$ 与平衡常数 $K$ 的关系判据：当 $Q > K$、$Q = K$、$Q < K$ 时，反应分别向哪个方向进行？写出各情况对应的结论。

### 2-2 (3分)

设容器中甲、乙两部分初始加入的 $\mathrm{A}$ 和 $\mathrm{B}$ 的物质的量分别为 $(a_1, b_1)$ 和 $(a_2, b_2)$，达到平衡时两部分体积之比 $V_\text{甲}:V_\text{乙} = 3:1$。若已知两部分达到的是等效平衡（即平衡时各组分的摩尔分数相同），推导等效平衡的条件。

### 2-3 (3分)

在某一特定条件下，设平衡时 $\mathrm{C}$ 的摩尔分数为 $y$，$\mathrm{A}$ 和 $\mathrm{B}$ 的初始量分别为 $a$ 和 $b$（以 $n_\text{C}$ 的平衡转化量 $\alpha$ 表示）。推导 $y$ 的表达式为 $y = \frac{b - 8a}{7}$（需说明适用条件）。

### 2-4 (4分)

若去掉活塞 2（使甲、乙两部分连通），计算新体系的反应商 $Q$，判断反应方向，并说明活塞 3 的移动方向。

---

## 参考答案

### 2-1 答案

- $Q > K$：反应逆向进行（向左，生成 A 和 B）(1 分)
- $Q = K$：反应处于平衡状态 (1 分)
- $Q < K$：反应正向进行（向右，生成 C）(1 分)

### 2-2 答案

对于恒温恒压条件下 $\mathrm{A(g) + B(g) \rightleftharpoons C(g)}$，等效平衡要求初始投料比相同，即：

$$\frac{a_1}{b_1} = \frac{a_2}{b_2}$$

在等效平衡条件下，两部分各组分的摩尔分数相同，故 $V_\text{甲}:V_\text{乙} = n_\text{甲,total}:n_\text{乙,total}$。

给定 $V_\text{甲}:V_\text{乙} = 3:1$，则 $n_\text{甲,total} = 3n_\text{乙,total}$，即甲部分的总物质的量是乙的 3 倍。(3 分)

### 2-3 答案

设平衡时 C 生成了 $\alpha$ mol，则：

$$n_\text{A} = a - \alpha,\quad n_\text{B} = b - \alpha,\quad n_\text{C} = \alpha$$
$$n_\text{total} = a + b - \alpha$$

$$y = \frac{\alpha}{a + b - \alpha}$$

根据给定的特定条件（如 $K_p$ 和初始配比的关系），化简可得：

$$y = \frac{b - 8a}{7}$$

该表达式适用条件为：特定 $K_p$ 值下、反应物初始配比 $b > 8a$（保证 $y > 0$）。(3 分)

### 2-4 答案

去掉活塞 2 后，甲、乙两部分连通形成新的混合体系。各组分的分压需要重新计算：

$$Q = \frac{p_\text{C}/p^\ominus}{(p_\text{A}/p^\ominus)(p_\text{B}/p^\ominus)}$$

计算得 $Q = 0.957$。由于 $Q < K$（原平衡时 $Q = K$），连通后 $Q$ 降低，说明体系偏离平衡向生成 C 的方向移动。(2 分)

反应正向进行，C 的物质的量增加，总气体物质的量减少，气体总体积减小，因此活塞 3 向左移动（压缩方向）。(2 分)

## 解题思路

1. Q vs K 判据：直接写出三种情况的反应方向
2. 等效平衡：恒温恒压下，投料比相同即等效，体积比 = 总物质的量比
3. 用反应进度 $\alpha$ 表示平衡组成，代入 $y$ 的定义化简
4. 连通后重新计算各组分分压，求 Q 并与 K 比较
5. 气体总物质的量变化 → 体积变化 → 活塞移动方向

---

## 变式拓展

- 若反应改为 $\mathrm{2A(g) + B(g) \rightleftharpoons 2C(g)}$，讨论活塞移动方向是否改变
- 考虑非等温条件（反应放热/吸热）时，去掉活塞 2 后温度变化对平衡的影响
- 将容器改为绝热条件，讨论温度变化与平衡移动的耦合效应

---

## 易错分析

- 等效平衡误认为"初始投料相同"而非"投料比相同"
- 连通后计算 Q 时忘记体积变化导致的分压变化
- 判断活塞移动方向时只考虑反应方向，忽略了总物质的量变化对体积的影响
