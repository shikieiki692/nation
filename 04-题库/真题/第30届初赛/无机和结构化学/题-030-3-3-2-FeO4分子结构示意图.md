# 题-030-3-3-2：FeO₄分子结构示意图

## 题目
> **来源**：[[07-资料提炼/提炼-第30届初赛试题解析|第30届初赛第3题]]（原始文件：[[mineru/02-真题解析/30届初赛试题解析]]）
> **小问关联**：本文件为第3-3-2小问；其余小问见 [[题-030-3-1-1-磷氧化合物中间体分子式]]、[[题-030-3-1-2-P4O8结构图键长分组键角比较]]、[[题-030-3-2-ZnNH3配合物中HNH键角变化]]、[[题-030-3-3-1-FeO4中铁价电子组态]]

画出该分子结构的示意图（用元素符号表示原子，用短线表示原子间的化学键）。

## 参考答案

由于该分子为四面体构型，Fe 的氧化态为 +6，分子中有一根过氧键，分子结构如下：

![[mineru/02-真题解析/30届初赛试题解析_images/26ee3df9d3b802f6c1cf508ce6d00183a71255e828c39788ea70a5f5fe229cbc.jpg]]

即 $\mathrm{FeO}_{2}(\mathrm{O}_{2})$，中心 Fe 原子与两个端基 O 原子形成双键，与一个过氧基团 $(\mathrm{O}_{2})^{2-}$ 配位。

## 解析

铁的四氧化物分子式为 $FeO_4$。根据题目所提供的条件，Fe 的氧化态是 +6，所以有两个 O 的氧化态是 -1，即分子中有一根过氧键。因此，四氧化物的分子式可以写成 $\mathrm{FeO}_{2}(\mathrm{O}_{2})$。由于该分子为四面体构型，故分子结构为中心 Fe 与两个端氧形成 Fe=O 双键，与一个过氧基团配位。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 四面体构型 | FeO₄分子构型为四面体 |
| 过氧键 | 分子中含O-O键，过氧基团为O₂²⁻ |
| 氧化态平衡 | Fe(+6) + 2×O(-2) + O₂(-2) = 0 |
| 结构示意图 | 用元素符号和短线表示原子连接方式 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 画成四个Fe-O单键 | 这样Fe氧化态为+8，与题意不符 |
| 未标出过氧键 | 必须明确画出O-O键以体现过氧基团 |
| 构型画错 | 应为四面体构型，不是平面四方形 |
| 忽略端氧与过氧氧的区别 | 端氧为O²⁻，过氧为O₂²⁻ |
