# 题-030-3-3-1：FeO₄分子中铁的价电子组态

## 题目
> **来源**：[[07-资料提炼/提炼-第30届初赛试题解析|第30届初赛第3题]]（原始文件：[[mineru/02-真题解析/30届初赛试题解析]]）
> **小问关联**：本文件为第3-3-1小问；其余小问见 [[题-030-3-1-1-磷氧化合物中间体分子式]]、[[题-030-3-1-2-P4O8结构图键长分组键角比较]]、[[题-030-3-2-ZnNH3配合物中HNH键角变化]]、[[题-030-3-3-2-FeO4分子结构示意图]]

量子化学计算预测未知化合物是现代化学发展的途径之一。2016年2月有人通过计算预言铁也存在四氧化物，其分子构型是四面体，但该分子中铁的氧化态是 +6 而不是 +8。

写出该分子中铁的价电子组态。

## 参考答案
$3d^{2}$

## 解析

零价的 Fe 原子的价电子组态为 $3d^{6}4s^{2}$，失去 6 个电子后，氧化态转化为 +6，价电子组态为 $3d^{2}$。

铁的四氧化物分子式为 $FeO_4$。根据题目所提供的条件，Fe 的氧化态是 +6，所以有两个 O 的氧化态是 -1，即分子中有一根过氧键。因此，四氧化物的分子式可以写成 $\mathrm{FeO}_{2}(\mathrm{O}_{2})$。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 价电子组态 | Fe原子基态为3d⁶4s²，失去6e⁻后为3d² |
| 氧化态确定 | Fe为+6价，而非按常规计算的+8价 |
| 过氧键存在 | FeO₄中Fe为+6说明含O₂²⁻过氧键 |
| 量子化学预测 | 现代计算化学方法可预测未知化合物 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 写成3d⁶ | 未考虑Fe失去6个电子形成+6氧化态 |
| 误认为Fe为+8价 | 题目已明确指出Fe氧化态为+6 |
| 忽略4s电子先失去 | 过渡金属失电子时4s先于3d失去 |
| 未理解过氧键的意义 | Fe+6价说明不是所有O都为-2价 |
