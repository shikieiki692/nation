## 题目

已知 $K_{\mathrm{a}_1}^{\ominus}(\mathrm{H}_2\mathrm{S}) = 1.1\times 10^{-7}, K_{\mathrm{a}_2}^{\ominus}(\mathrm{H}_2\mathrm{S}) = 1.3\times 10^{-3}, K_{\mathrm{sp}}^{\ominus}(\mathrm{ZnS}) = 2.5\times 10^{-22}$。假定本题两杯中的液体体积始终等于 $1.00~\mathrm{dm}^{3}$，且 $\mathrm{H_2S}$ 饱和溶液的浓度均等于 $0.100\mathrm{mol \, dm^{-3}}$。天平两盘各有一盛着 $1.0\mathrm{dm}^{3}\mathrm{H_2O}$ 的烧杯，天平平衡。

### 2-1 (3分)
向左杯中溶入 $2.72 \, \mathrm{g} \, \mathrm{ZnCl_{2}}$，试通过计算说明，为使天平再次平衡需向右杯中加入多少克 $\mathrm{Na_{2}O_{2}}$？

### 2-2 (3分)
天平再次平衡后，向左杯中通入 $\mathrm{H_{2}S}$ 气体，充分反应且使 $\mathrm{H_{2}S}$ 达到饱和。接着向右杯中通入 $\mathrm{H_{2}S}$ 气体。试通过计算求出，当天平第三次平衡时右杯中溶液的 pH。

### 2-3 (3分)
天平第三次平衡后，向左杯中缓缓通入 HCl 气体。试通过计算求出，当杯中的沉淀刚好溶解掉一半时溶液的 Cl⁻ 浓度。

### 2-4 (3分)
拟使天平第四次平衡，需向右杯中通入多少克 HCl 气体？试通过计算加以说明。

## 参考答案

### 2-1 答案
**左杯**：$\mathrm{ZnCl_2}$ 溶解
$$n_{\mathrm{ZnCl_2}} = \frac{2.72}{136} = 0.0200 \, \mathrm{mol}$$
质量增加：2.72 g

**右杯**：$\mathrm{Na_2O_2}$ 与水反应
$$\mathrm{2Na_2O_2 + 2H_2O \rightarrow 4NaOH + O_2 \uparrow}$$

设加入 x g Na₂O₂：
$$n_{\mathrm{Na_2O_2}} = \frac{x}{78}$$
$$\Delta m = x - \frac{x}{78} \times \frac{32}{2} = x \times \frac{78-16}{78} = x \times \frac{62}{78}$$

**平衡条件**：$\Delta m_{左} = \Delta m_{右}$
$$2.72 = x \times \frac{62}{78}$$
$$x = 3.42 \, \mathrm{g}$$

### 2-2 答案
**左杯**：Zn²⁺ + H₂S → ZnS↓ + 2H⁺
$$n_{\mathrm{ZnS}} = 0.0200 \, \mathrm{mol}$$
$$[\mathrm{H^+}] = 0.0400 \, \mathrm{mol/L}$$

**右杯**：NaOH 与 H₂S 反应
$$n_{\mathrm{NaOH}} = \frac{3.42}{78} \times 4 = 0.175 \, \mathrm{mol}$$

H₂S 饱和：0.100 mol

**反应**：$\mathrm{H_2S + 2NaOH \rightarrow Na_2S + 2H_2O}$
$$n_{\mathrm{Na_2S}} = 0.0500 \, \mathrm{mol}$$
$$n_{\mathrm{NaOH}}_{过量} = 0.175 - 0.100 = 0.075 \, \mathrm{mol}$$

**pH 计算**：
$$[\mathrm{OH^-}] = 0.075 \, \mathrm{mol/L}$$
$$\mathrm{pH} = 14 + \lg(0.075) = 12.88$$

### 2-3 答案
**左杯**：ZnS + 2H⁺ → Zn²⁺ + H₂S
沉淀溶解一半：0.0100 mol ZnS 溶解

$$\mathrm{ZnS + 2H^+ \rightleftharpoons Zn^{2+} + H_2S}$$

**平衡常数**：
$$K = \frac{K_{sp}}{K_{a1} \times K_{a2}} = \frac{2.5 \times 10^{-22}}{1.1 \times 10^{-7} \times 1.3 \times 10^{-3}} = 1.75 \times 10^{-12}$$

**溶解平衡**：
$$[\mathrm{Zn^{2+}}] = 0.0100 \, \mathrm{mol/L}$$
$$[\mathrm{H_2S}] = 0.0100 \, \mathrm{mol/L}$$

$$[\mathrm{H^+}] = \sqrt{\frac{[\mathrm{Zn^{2+}}][\mathrm{H_2S}]}{K}} = \sqrt{\frac{0.01 \times 0.01}{1.75 \times 10^{-12}}} = 0.240 \, \mathrm{mol/L}$$

$$[\mathrm{Cl^-}] = [\mathrm{H^+}] = 0.240 \, \mathrm{mol/L}$$

### 2-4 答案
**右杯**：HCl 与 NaOH 反应
$$n_{\mathrm{HCl}} = n_{\mathrm{NaOH}}_{过量} = 0.075 \, \mathrm{mol}$$

**质量计算**：
$$m_{\mathrm{HCl}} = 0.075 \times 36.5 = 2.74 \, \mathrm{g}$$

## 知识点映射
- **沉淀溶解平衡**: Ksp 的应用
- **酸碱平衡**: pH 计算
- **氧化还原**: Na₂O₂ 与水的反应
- **质量平衡**: 天平平衡条件

## 解题思路
1. **2-1**: 计算左右杯的质量变化，建立平衡方程
2. **2-2**: 分析两杯中的反应，计算 pH
3. **2-3**: 利用溶解平衡常数计算 [H⁺]
4. **2-4**: 计算中和过量 NaOH 所需 HCl

## 考点抽象
- **多重平衡**: 沉淀、酸碱、氧化还原平衡的综合
- **定量计算**: 从实验数据推断组成
- **平衡常数应用**: Ksp、Ka 的综合运用
