## 题目

酸性溶液中 $\mathrm{Ce(IV)}$ 是一种强氧化剂，被用于定量分析和有机合成等领域。

### 5-1 (8分)
尽管水溶液中 $\mathrm{Ce^{4+}}$ 强烈水解，但在 $\left[\mathrm{HClO_{4}}\right]>6.05\mathrm{mol \, kg^{-1}}$ 的高酸度溶液中 $\mathrm{Ce^{4+}}$ 仍然会发生反应：$\mathrm{Ce^{4+} + e^{-} \rightleftharpoons Ce^{3+}}$。高氯酸溶液中 $\mathrm{Ce^{4+}}$ 与 $\mathrm{OH^{-}}$ 形成 4 种配合物的累积生成常数的对数分别为 $\lg\beta_{1}^{\ominus}=14.76, \lg\beta_{2}^{\ominus}=28.04, \lg\beta_{3}^{\ominus}=40.53, \lg\beta_{4}^{\ominus}=51.86$，有关物质的标准摩尔生成 Gibbs 自由能 $\Delta_{f}G_{m}^{\ominus}(298.15\mathrm{K})$ 列于下表。

表5-1
| 物质 | $\Delta_{\text{f}}G_{\text{m}}^{\ominus}/(\text{kJ \, mol}^{-1})$ |
|------|-------|
| $\mathrm{H_{2}O(l)}$ | -237.10 |
| $\mathrm{OH^{-}(aq)}$ | -157.20 |
| $\mathrm{H^{+}(aq)}$ | 0 |
| $\mathrm{Ce^{4+}(aq)}$ | -503.80 |

#### 5-1-1 (2分)
写出 $\mathrm{Ce^{4+}}$ 水解反应的一般表达式、标准水解平衡常数的一般表达式。

#### 5-1-2 (2分)
写出 $\mathrm{Ce^{4+}}$ 与 $\mathrm{OH^{-}}$ 形成配合物反应的一般表达式、标准累积生成常数的一般表达式。

#### 5-1-3 (2分)
求 $\mathrm{Ce^{4+}}$ 水解反应生成的 4 种氢氧化物的 $\Delta_{f}G_{m}^{\ominus}$。

#### 5-1-4 (2分)
当 pH<8, $\mathrm{Ce^{3+}}$ 不水解。当 $\mathrm{HClO_{4}}$ 浓度 <6.05 mol kg⁻¹ 时，$\mathrm{Ce^{4+}/Ce^{3+}}$ 电对的电势是溶液的 pH、$\mathrm{Ce(IV)}$ 配合物的 $\beta_{i}^{\ominus}$、有关物种浓度的函数，试导出这一函数关系。

## 参考答案

### 5-1-1 答案
**水解反应**：
$$\mathrm{Ce^{4+} + nH_2O \rightleftharpoons Ce(OH)_n^{(4-n)+} + nH^+}$$

**水解平衡常数**：
$$K_{h,n} = \frac{[\mathrm{Ce(OH)_n^{(4-n)+}}][\mathrm{H^+}]^n}{[\mathrm{Ce^{4+}}]}$$

### 5-1-2 答案
**配合物形成**：
$$\mathrm{Ce^{4+} + nOH^- \rightleftharpoons Ce(OH)_n^{(4-n)+}}$$

**累积生成常数**：
$$\beta_n = \frac{[\mathrm{Ce(OH)_n^{(4-n)+}}]}{[\mathrm{Ce^{4+}}][\mathrm{OH^-}]^n}$$

### 5-1-3 答案
**ΔfGm° 计算**：
$$\Delta_fG_m^{\ominus}[\mathrm{Ce(OH)_n^{(4-n)+}}] = \Delta_fG_m^{\ominus}[\mathrm{Ce^{4+}}] + n\Delta_fG_m^{\ominus}[\mathrm{OH^-}] - RT\ln\beta_n$$

计算得：
- Ce(OH)³⁺：-345.2 kJ/mol
- Ce(OH)₂²⁺：-192.5 kJ/mol
- Ce(OH)₃⁺：-42.3 kJ/mol
- Ce(OH)₄：+105.8 kJ/mol

### 5-1-4 答案
**电极电势函数关系**：
$$\varphi = \varphi^{\ominus} + \frac{RT}{F}\ln\frac{[\mathrm{Ce^{4+}}]}{[\mathrm{Ce^{3+}}]}$$

考虑水解平衡：
$$[\mathrm{Ce^{4+}}]_{总} = [\mathrm{Ce^{4+}}](1 + \sum_{i=1}^{4}\beta_i[\mathrm{OH^-}]^i)$$

$$\varphi = \varphi^{\ominus} + \frac{RT}{F}\ln\frac{[\mathrm{Ce^{4+}}]_{总}}{(1 + \sum_{i=1}^{4}\beta_i[\mathrm{OH^-}]^i)[\mathrm{Ce^{3+}}]}$$

### 5-2 答案
**熵变分析**：
- [HClO₄] > 1.85 mol/dm³：ΔrSm > 0（水合壳层破坏）
- [HClO₄] < 1.85 mol/dm³：ΔrSm < 0（水合增强）

**焓变变化**：
ΔrHm 随 [HClO₄] 增大而增大（去水合吸热）

### 5-3-1 答案
**速率方程**：
$$r = k[\mathrm{Ce(OH)}^{3+}][\mathrm{ATF}]$$

**表观速率常数**：
$$k_{\text{表}} = \frac{kK_{OH}[\mathrm{ATF}]}{[\mathrm{H^+}] + K_{OH}}$$

### 5-3-2 答案
**[ATF] 的影响**：
$k_{\text{表}}$ 与 [ATF] 成正比，反应对 ATF 为**一级**

### 5-3-3 答案
**[H⁺] 的影响**：
$k_{\text{表}}$ 与 [H⁺] 成反比，[H⁺] 增大抑制水解，降低 Ce(OH)³⁺ 浓度
