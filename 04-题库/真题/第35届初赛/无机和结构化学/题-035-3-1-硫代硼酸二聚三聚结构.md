# 题-035-3-1：硫代硼酸二聚三聚结构

## 题目

> **来源**：[[07-资料提炼/提炼-第35届初赛试题解析|第35届初赛第3题]]（原始文件：[[06-外部资料导入/真题/第35届初赛试题解析]]）
> **小问关联**：本文件为第 3 题第(1)小问；第(2)小问见 [[题-035-3-2-BCl3反应与水解]]；第(3)小问见 [[题-035-3-3-卟吩类似物芳香性]]

$\mathrm{B(OH)_3}$ 是一种重要的无机化合物。在尝试合成硫代硼酸的过程中，得到了最简式为 $\mathrm{HBS_2}$ 的化合物。结构研究发现，这类物质主要以二聚 $(\mathrm{HBS_2})_2$ (**A**) 或三聚 $(\mathrm{HBS_2})_3$ (**B**) 的形式存在，其中的B原子均为三配位。

**3-1** 画出 **A** 和 **B** 的结构。

## 参考答案

**得分点**：
1. **正确推断A的结构**（2分）：两个B原子通过两个S桥连接，每个B连1个端基-SH，形成四元环 $\mathrm{B_2S_2}$
2. **正确推断B的结构**（2分）：三个B原子通过三个S桥连接，每个B连1个端基-SH，形成六元环 $\mathrm{B_3S_3}$

**完整推导**：

硼酸的化学式为 $\mathrm{B(OH)_3}$，因而硫代硼酸的化学式应为 $\mathrm{B(SH)_3}$。由于二聚体A与三聚体B均为合成硫代硼酸时的副产物，合理推测合成中不发生氧化还原反应，仅涉及单体不同的聚合方式。

结合题干中的信息「其中的B原子均为三配位」，可推测：
- 在A与B的结构中，每个B原子均连有3个S原子
- 若B原子与H原子或其他B原子相连，则必然涉及氧化还原过程
- 每种S元素均保持 $-2$ 价不变，其存在形式可以是端基形式的巯基 $-\mathrm{SH}$，也可以是桥基形式的 $-\mathrm{S}-$

**A（二聚体）**：可看作2分子硫代硼酸 $\mathrm{B(SH)_3}$ 单体脱去2分子 $\mathrm{H_2S}$ 得到的缩合产物，结构为含 $\mathrm{B_2S_2}$ 四元环的平面结构：

```
      SH           SH
       |             |
   S--B--S       S--B--S
       \             /
        S---------S
       /             \
   S--B--S       S--B--S
       |             |
      SH           SH
```

（简化表示：两个B通过两个S桥连接，每个B各连一个-SH端基）

**B（三聚体）**：可看作3分子硫代硼酸 $\mathrm{B(SH)_3}$ 单体脱去3分子 $\mathrm{H_2S}$ 得到的缩合产物，结构为含 $\mathrm{B_3S_3}$ 六元环的平面结构，与三聚硼酸的结构等同：

```
        SH
         |
    S--B--S
   /         \
  S           S
   \         /
    B--S--B
    |     |
   SH    SH
```

（简化表示：三个B通过三个S桥交替连接成六元环，每个B各连一个-SH端基）

> 为削弱中心B原子的缺电子性质，二聚体A与三聚体B结构中的环系均为平面结构，分别形成 $\pi_4^4$ 和 $\pi_6^6$ 的离域 $\pi$ 键。

## 解题思路

1. **读题定位**：看到「硫代硼酸」「二聚/三聚」「B原子均为三配位」→ 识别为 **缺电子化合物结构推断** 题
2. **推断单体**：硼酸 $\mathrm{B(OH)_3}$ → 硫代硼酸 $\mathrm{B(SH)_3}$
3. **推断聚合方式**：不发生氧化还原 → 缩合脱水（脱 $\mathrm{H_2S}$）
4. **🔑 关键转换**：B 为三配位，每个 B 连3个 S；S 为二配位，可作桥连或端基
   - 二聚体：2个 B，每个 B 需3个 S → 共需6个 S 配位点
   - 设 $x$ 个桥 S（每个提供2个配位点），$y$ 个端基 S（每个提供1个配位点）
   - $2x + y = 6$，$x + y = 4$（二聚体总S数）→ $x = 2, y = 2$ ✓
   - 三聚体类似：$2x + y = 9$，$x + y = 6$ → $x = 3, y = 3$ ✓
5. **验证**：结构平面，B 采取 $\mathrm{sp}^2$ 杂化，符合缺电子化合物的成键特征

## 易错分析

| 错误 | 原因 | 纠正 | 课堂提问 |
|---|---|---|---|
| B-B 直接成键 | 违反「不发生氧化还原」的假设 | B-B 键需要氧化数变化或特殊合成条件 | 「如果B和B直接相连，B的氧化数是多少？与单体相比变化了吗？」 |
| S 氧化态判断错误 | 误认为有 S-S 键 | 端基和桥基 S 均为 $-2$ 价，无 S-S 键 | 「结构中有 S-S 键吗？为什么？」 |
| 画成非平面结构 | 未考虑 B 的缺电子稳定化 | 平面结构可形成离域 $\pi$ 键，稳定缺电子中心 | 「B 是缺电子原子，平面结构如何帮助稳定它？」 |
| 漏画端基 -SH | 未注意 S 的总数守恒 | 二聚体 $(\mathrm{HBS_2})_2 = \mathrm{H_2B_2S_4}$，含2个端基-SH | 「$(\mathrm{HBS_2})_2$ 的分子式是什么？有几个 H？」 |

## 知识拓展

**硼的缺电子化合物的结构特征**：

硼元素是第二周期第三主族的元素，其化合物大多都是典型的缺电子化合物。为了削弱中心硼原子的缺电子程度，结构中通常形成：

1. **离域 $\pi$ 键**：如平面三角形的 $\mathrm{BCl_3}$（$\pi_4^6$）
2. **多中心键**：如二聚的乙硼烷 $\mathrm{B_2H_6}$（3c-2e 键）
3. **多聚结构**：如本题中的二聚/三聚硫代硼酸

硼化合物的结构多样性使得其常被作为考查点出现。
