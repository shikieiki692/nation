# 题-025-2-5：六氨合钴合成机理

## 题目

> **来源**：第25届中国化学奥林匹克（初赛）第2题(11分) 第(2-5)小问

橙黄色固体配合物A的名称是三氯化六氨合钴（Ⅲ），是将二氯化钴、浓氨水、氯化铵和过氧化氢混合，以活性炭为催化剂合成的。机理研究发现，反应过程中首先得到 $\mathrm{Co(NH_{3})_{6}^{2+}}$ 离子；随后发生配体取代反应，得到以新配体为桥键的双核离子 $B^{4+}$；接着发生桥键断裂，同时两个中心原子分别将一个电子传递到均裂后的新配体上，得到两个 $C^{2+}$ 离子；最后 $C^{2+}$ 离子在活性炭表面上发生配体取代反应，并与氯离子结合形成固体配合物A。写出合成配合物A的总反应方程式；画出 $B^{4+}$ 和 $C^{2+}$ 离子的结构式。

## 参考答案

**总反应方程式**：

$$
2\mathrm{CoCl_2} + 10\mathrm{NH_3} + 2\mathrm{NH_4Cl} + \mathrm{H_2O_2} = 2\mathrm{Co(NH_3)_6Cl_3} + 2\mathrm{H_2O}
$$

**B⁴⁺结构**：两个Co(II)通过H₂O₂的过氧键(-O-O-)桥连，每个Co仍保留5个NH₃配体。

**C²⁺结构**：过氧键均裂后，每个Co被氧化为Co(III)，配位一个OH⁻和5个NH₃。

## 知识点映射

- [[Co配位化学]]
- 过氧键均裂
- 活性炭催化
- 配体取代反应

## 解题思路

1. Co(II)氨配离子 → H₂O₂桥连双核B⁴⁺ → 过氧键均裂 → Co(III)的C²⁺
2. H₂O₂作为配体取代2个NH₃ → 桥键连接两个Co
3. 过氧键均裂：O-O断裂，每个O获得1个电子→OH⁻
4. Co(II)→Co(III)：每个Co失去1个电子给O
5. 最后NH₃取代OH⁻ → [Co(NH₃)₆]³⁺

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| B⁴⁺电荷 | 两个Co(II)=+4，H₂O₂中性，电荷守恒 |
| 均裂vs异裂 | 过氧键是均裂（对称断裂），不是异裂 |
| 活性炭作用 | 催化最后一步配体取代，无活性炭产物不同 |
