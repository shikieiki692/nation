# 题-025-1-5：高铁酸根氧化还原

## 题目

> **来源**：第25届中国化学奥林匹克（初赛）第1题(15分) 第(1-5)小问

已知 $E^{\ominus}(\mathrm{FeO}_{4}^{2-}/\mathrm{Fe}^{3+})=2.20\ \mathrm{V},\ E^{\ominus}(\mathrm{FeO}_{4}^{2-}/\mathrm{Fe}(\mathrm{OH})_{3})=0.72\ \mathrm{V}$ 。

① 写出氯气和三氯化铁反应形成高铁酸根的离子方程式。

② 写出高铁酸钾在酸性水溶液中分解的离子方程式。

③ 用高铁酸钾与镁等组成碱性电池，写出该电池的电极反应。

## 参考答案

① 碱性条件下Cl₂氧化Fe³⁺：

$$
2\mathrm{Fe}^{3+} + 3\mathrm{Cl}_2 + 16\mathrm{OH}^{-} = 2\mathrm{FeO}_4^{2-} + 6\mathrm{Cl}^{-} + 8\mathrm{H}_2\mathrm{O}
$$

> Cl₂/Cl⁻电极电势介于0.72～2.20V之间，只能在碱性条件下（E°=0.72V）氧化Fe³⁺。

② 酸性条件下高铁酸根氧化水：

$$
4\mathrm{FeO}_4^{2-} + 20\mathrm{H}^{+} = 4\mathrm{Fe}^{3+} + 3\mathrm{O}_2 + 10\mathrm{H}_2\mathrm{O}
$$

③ 碱性电池电极反应：

$$
\text{正极: } \mathrm{FeO}_4^{2-} + 4\mathrm{H}_2\mathrm{O} + 3\mathrm{e}^{-} = \mathrm{Fe(OH)}_3 + 5\mathrm{OH}^{-}
$$

$$
\text{负极: } \mathrm{Mg} + 2\mathrm{OH}^{-} - 2\mathrm{e}^{-} = \mathrm{Mg(OH)}_2
$$

## 解题思路

1. 高铁酸根在酸性条件下E°=2.20V（强氧化性），碱性条件下E°=0.72V（惰性）
2. Cl₂只能在碱性条件下氧化Fe³⁺→FeO₄²⁻
3. 酸性条件下FeO₄²⁻氧化H₂O生成O₂
4. 碱性电池中：FeO₄²⁻在正极被还原为Fe(OH)₃，Mg在负极被氧化为Mg(OH)₂

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 反应条件选择 | ①必须在碱性条件下，酸性条件下Cl₂无法氧化Fe³⁺到Fe(VI) |
| 电极反应配平 | 碱性条件下用OH⁻和H₂O配平，不要用H⁺ |
| 正负极电子数 | 正负极反应得失电子数不必相等（各自独立配平即可） |
