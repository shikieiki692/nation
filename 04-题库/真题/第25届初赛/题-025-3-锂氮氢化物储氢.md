# 题-025-3：锂氮氢化物储氢

## 题目

> **来源**：第25届中国化学奥林匹克（初赛）第3题(8分)

近年来，某些轻元素的含氢化合物及其复合体系作为氢源受到广泛关注。化合物 A ($XYH_{2}$) 和 B(XH)都是具有潜在应用价值的释氢材料。A 受热分解生成固体化合物 C 并放出刺激性气体 D, D 可使湿润的 pH 试纸变蓝。A 和 B 混合可优化放氢性能。研究发现，该混合体系的放氢反应分三步进行：

$$2\mathbf{A} = \mathbf{C} + \mathbf{D}\tag{1}$$

$$\mathbf{D} + \mathbf{B} = \mathbf{A} + \mathrm{H}_{2}\tag{2}$$

$$\mathbf{C} + \mathbf{B} = \mathbf{E} + \mathrm{H}_{2}\tag{3}$$

将 A 和 B 按 1:2 的摩尔比混合，在催化剂作用下，所含的氢全部以氢气放出，失重 10.4%。

A、C、E均能水解生成F和D。G是由X和Y组成的二元化合物，其阴离子是二氧化碳的等电子体，G分解生成E和一种无色无味的气体I。写出A、B、C、D、E、F、G和I的化学式。

## 参考答案

| 符号 | 化学式 | 说明 |
|:---:|:---:|:---|
| A | LiNH₂ | 氨基锂 |
| B | LiH | 氢化锂 |
| C | Li₂NH | 亚氨基锂 |
| D | NH₃ | 氨 |
| E | Li₃N | 氮化锂 |
| F | LiOH | 氢氧化锂 |
| G | LiN₃ | 叠氮化锂 |
| I | N₂ | 氮气 |

**关键推导**：D使pH试纸变蓝→D为NH₃→Y=N。由失重10.4%计算得X的Ar=6.9→X=Li。G的阴离子N₃⁻与CO₂等电子。

## 解题思路

1. D使pH试纸变蓝 → D = NH₃ → Y = N
2. A(XNH₂) + 2B(XH) → 总反应放H₂，失重10.4%
3. 由失重比例计算Ar(X) = 6.9 → X = Li
4. G阴离子与CO₂等电子 → N₃⁻ → G = LiN₃
5. LiN₃分解 → Li₃N + 4N₂（I = N₂，无色无味）

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 等电子体判断 | N₃⁻与CO₂等电子（均有3个原子+16个价电子） |
| 失重计算 | 10.4% = 4H/(3Ar(X)+14+6Ar(X)+...)，需要列方程 |
| G分解产物 | LiN₃ → Li₃N + 4N₂，不要遗漏N₂ |
