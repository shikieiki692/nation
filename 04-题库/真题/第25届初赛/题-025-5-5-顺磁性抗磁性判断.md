# 题-025-5-5：顺磁性抗磁性判断

## 题目

> **来源**：第25届中国化学奥林匹克（初赛）第5题(10分) 第(5-5)小问

用配合物的价键理论推测 C 和 E 显顺磁性还是抗磁性,说明理由。

## 参考答案

C和E均显**顺磁性**。

**理由**：C和E中Rh均为+3氧化态，价电子构型为4d⁶，dsp³杂化。5个4d轨道中只有1个参与杂化，剩余4个d轨道容纳6个电子。根据洪特规则和泡利不相容原理，必然存在2个未成对电子，因此均为顺磁性。

## 知识点映射

- [[顺磁性·抗磁性]]
- [[VB理论]]
- 洪特规则

## 解题思路

1. Rh(III) = 4d⁶
2. dsp³杂化：1个d + 1个s + 3个p = 5个轨道参与杂化
3. 剩余4个d轨道，容纳6个电子
4. 排布：2,2,1,1 → 2个未成对电子 → 顺磁性

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| d轨道分配 | dsp³用掉1个d，剩余4个d轨道（不是5个） |
| 电子排布 | 6个电子填入4个轨道→必然有未成对电子 |
| 与A对比 | A是d⁸dsp²→8个d电子填入4个d轨道→全配对→抗磁性 |
