# 题-025-9-1：有机推断结构式

## 题目

> **来源**：第25届中国化学奥林匹克（初赛）第9题(10分) 第(9-1)小问

画出化合物 A、B、C、D、E、M 和 N 的结构简式。

## 参考答案

**推导链路**：
1. A/B/C（C₇H₈O₂，不饱和度4）→ 加H₂得 D（C₇H₁₂O₂）
2. D碱性水解+酸化 → E（C₆H₁₀O₂）+ F（CH₄O = CH₃OH）
3. E含1个羧基+1个环（不饱和度=2）
4. E = 环戊基甲酸
5. D = 环戊基甲酸甲酯（COOMe）
6. A/B/C = 环戊二烯衍生物的甲酯（五元环中含2个双键）
7. A + CH₃MgCl → M（C₈H₁₂O）→ 浓H₂SO₄/Δ → N（C₈H₁₀）
8. N只有3种H（1:1:3）→ N为具有对称性的环烯烃

## 知识点映射

- [[格氏反应]]
- [[酯水解]]
- 不饱和度分析
- 对称性与NMR

## 解题思路

1. 由F=CH₃OH→E含-COOH→D为甲酯
2. E不饱和度=2（1个C=O + 1个环）
3. A/B/C为位置异构体（双键位置不同）
4. M为三级醇（格氏试剂加成2次）
5. N脱水后只有3种H→高对称性结构

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 格氏试剂加成次数 | 酯可与1或2当量格氏试剂反应，M的不饱和度=3→加了2次 |
| N的对称性 | N(C₈H₁₀)只有3种H(1:1:3)→需要仔细验证对称性 |
| A/B/C关系 | 位置异构体（双键位置不同），不是碳架异构 |
