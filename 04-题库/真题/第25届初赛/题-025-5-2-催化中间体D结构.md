# 题-025-5-2：催化中间体D结构

## 题目

> **来源**：第25届中国化学奥林匹克（初赛）第5题(10分) 第(5-2)小问

A 可能的催化机理如下页图所示(图中 16e 表示中心原子周围总共有 16 个电子)：画出 D 的结构式。

## 参考答案

D为丙烯与C加成的产物，在C的五配位结构基础上增加丙烯配体，形成六配位18电子化合物。八面体构型，Rh(III)为中心，4d⁶构型，d²sp³杂化。

配体：1个Cl + 2个H + 1个丙烯（2e配体）+ 2个PPh₃ = 六配位

## 知识点映射

- [[18e规则]]
- 烯烃配位
- 八面体配合物

## 解题思路

1. C为五配位16e物种 → 丙烯(2e配体)加成 → D为六配位18e
2. 六配位 → 八面体构型
3. Rh(III)，4d⁶，d²sp³杂化（强场→内轨型）

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| 电子计数 | 丙烯是2e配体，加成后电子数+2 |
| 氧化态 | D中Rh为+3（Cl贡献1，H贡献1，丙烯和PPh₃不改变氧化态） |
| 磁性 | d⁶强场→t₂g⁶→抗磁性（但题目只问结构） |
