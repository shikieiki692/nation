# 题-025-1-3：SNF与SNF₃结构

## 题目

> **来源**：第25届中国化学奥林匹克（初赛）第1题(15分) 第(1-3)小问

20世纪60年代维也纳大学V. Gutmann研究小组报道，三原子分子A可由 $\mathrm{SF}_4$ 和 $\mathrm{NH}_3$ 反应合成；A被 $\mathrm{AgF_2}$ 氧化得到沸点为 $27^{\circ}\mathrm{C}$ 的三元化合物B。A和B分子中的中心原子与同种端位原子的核间距几乎相等；B分子有一根三重轴和3个镜面。画出A和B的结构式(明确示出单键和重键，不在纸面上的键用楔形键表示，非键合电子不必标出）。

## 参考答案

**A = SNF**：S为中心原子，与N形成双键，与F形成单键，结构为 $\mathrm{F\text{-}S=N}$（弯曲形，VSEPR为三角形，孤对电子一对）。

**B = SNF₃**：S为中心原子，N和三个F为配位原子，$\mathrm{C_{3v}}$ 对称性（三重轴+3个镜面），S与N形成双键，S与三个F形成单键。

## 知识点映射

- [[VSEPR理论]]
- [[等电子体]]
- SF₄的反应性
- AgF₂的氟化作用

## 解题思路

1. 三原子分子A只可能是三种元素组成（两种元素无法满足题设）
2. 由反应物推断A含S、N、F三种元素 → A = SNF
3. AgF₂为氟化试剂，将A氟化为B = SNF₃
4. B有一根三重轴+3个镜面 → $\mathrm{C_{3v}}$ 对称 → S在中心，N与3个F等价
5. 用VSEPR验证：SNF₃中S有4个σ键+0孤对 → 四面体 → 满足$\mathrm{C_{3v}}$

## 易错分析

| 易错点 | 说明 |
|:---|:---|
| A的组成推断 | 不要漏掉"三原子"限制，两种元素无法形成合适结构 |
| B的对称性 | 三重轴+3个镜面 = $\mathrm{C_{3v}}$，排除$\mathrm{C_{2v}}$等可能 |
| 中心原子判断 | S在中心而非N（N的配位数通常不超过3） |
