## 题目

**第6题（9分）**

将一定量的水在 $\mathrm{100\,^{\circ}C}$、$1\,\mathrm{atm}$ 下可逆蒸发为水蒸气，然后将水蒸气在可逆条件下压缩，压缩过程满足 $pV = a$（$a$ 为常数）。

**6-1**（3分）若终态压强为 $p_2$，利用 $pV = a$ 约束，计算终态温度 $T_2$。

**6-2**（2分）判断终态是否有液态水存在。（比较终态压强 $p_2$ 与该温度下的饱和蒸气压）

**6-3**（2分）计算全过程的焓变 $\Delta H$。

**6-4**（2分）计算全过程的熵变 $\Delta S$。

已知：水的蒸发焓 $\Delta_\mathrm{vap}H = 40.67\,\mathrm{kJ/mol}$（在 $100\,^{\circ}\mathrm{C}$），水蒸气视为理想气体，$R = 8.314\,\mathrm{J/(mol \cdot K)}$。

## 参考答案

### 6-1 终态温度 $T_2$

**蒸发过程**：$T_1 = 373.15\,\mathrm{K}$，$p_1 = 1\,\mathrm{atm}$

初始水蒸气状态：$p_1 V_1 = nRT_1$

**可逆压缩过程**：$pV = a$（常数）

由于蒸发前后 $n$、$T$ 不变时 $p_1 V_1 = nRT_1$，所以：
$$a = p_1 V_1 = nRT_1$$

终态：$p_2 V_2 = a = nRT_1$

又 $p_2 V_2 = nRT_2$（理想气体状态方程）

因此：$nRT_2 = nRT_1$

$$\boxed{T_2 = T_1 = 373.15\,\mathrm{K}}$$

即 $pV = a$ 过程中温度保持不变（等温过程）。

**注意**：$pV = a$ 对于理想气体意味着 $nRT = a = \text{const}$，故此过程为等温过程。

### 6-2 判断是否有液态水

终态温度 $T_2 = 373.15\,\mathrm{K} = 100\,^{\circ}\mathrm{C}$

在 $100\,^{\circ}\mathrm{C}$ 时，水的饱和蒸气压 $p^* = 1\,\mathrm{atm}$

- 若 $p_2 > p^* = 1\,\mathrm{atm}$：蒸气将凝结为液态水
- 若 $p_2 \leq p^*$：终态全部为气态

由于压缩过程中 $p_2 V_2 = nRT_1$，且 $p_2 > p_1 = 1\,\mathrm{atm}$（压缩使压强增大），所以：

**当 $p_2 > 1\,\mathrm{atm}$ 时**，终态温度仍为 $100\,^{\circ}\mathrm{C}$，但压强超过饱和蒸气压，**有液态水存在**（部分水蒸气凝结）。

**当 $p_2 = 1\,\mathrm{atm}$ 时**，回到初始状态，无液态水（恰好饱和）。

### 6-3 焓变 $\Delta H$

全过程分两步：

**第一步：蒸发**
$$\Delta H_1 = n \times \Delta_\mathrm{vap}H = n \times 40.67\,\mathrm{kJ/mol}$$

**第二步：等温压缩（理想气体）**

理想气体的焓只与温度有关，$T$ 不变则 $\Delta H_2 = 0$

因此：
$$\boxed{\Delta H = n \times \Delta_\mathrm{vap}H = n \times 40.67\,\mathrm{kJ/mol}}$$

若考虑 $1\,\mathrm{mol}$ 水：$\Delta H = 40.67\,\mathrm{kJ}$

### 6-4 熵变 $\Delta S$

**总过程**：水在 $100\,^{\circ}\mathrm{C}$、$1\,\mathrm{atm}$ 下可逆蒸发，然后可逆等温压缩至 $p_2$。

**蒸发过程的熵变**：
$$\Delta S_1 = \frac{n \Delta_\mathrm{vap}H}{T_1} = \frac{n \times 40670}{373.15}\,\mathrm{J/(mol \cdot K)} = n \times 109.0\,\mathrm{J/(mol \cdot K)}$$

**等温压缩的熵变**（理想气体）：
$$\Delta S_2 = -nR\ln\frac{p_2}{p_1} = -nR\ln\frac{p_2}{p_1}$$

**总熵变**：
$$\boxed{\Delta S = n\left(\frac{\Delta_\mathrm{vap}H}{T_1} - R\ln\frac{p_2}{p_1}\right)}$$

$$= n\left(109.0 - 8.314\ln\frac{p_2}{p_1}\right)\,\mathrm{J/(mol \cdot K)}$$

当 $p_2 > p_1$ 时，$\ln(p_2/p_1) > 0$，压缩使熵减少，抵消部分蒸发的熵增。

## 解题思路

### 关键突破口
$pV = a$（常数）是理解本题的核心。对于理想气体：
$$pV = nRT = a = \text{const}$$

这意味着 $T$ 恒定，即过程为**等温过程**。

### 分步分析
1. **蒸发**：$T$、$p$ 不变（$100\,^{\circ}\mathrm{C}$，$1\,\mathrm{atm}$），液态水→水蒸气，吸热 $\Delta_\mathrm{vap}H$
2. **等温压缩**：$T$ 不变（仍为 $373.15\,\mathrm{K}$），$p$ 增大，$V$ 减小
3. 终态判断：比较 $p_2$ 与 $p^*$（$100\,^{\circ}\mathrm{C}$ 时饱和蒸气压 $= 1\,\mathrm{atm}$）

### 计算要点
- 焓变：只考虑蒸发（压缩对理想气体焓无贡献）
- 熵变：蒸发熵变 + 等温压缩熵变（$-nR\ln(p_2/p_1)$）

## 考点抽象

### 核心考点
1. **$pV = a$ 过程的物理意义**：对理想气体等价于等温过程
2. **相平衡条件**：$T$、$p$ 同时满足才能维持两相共存
3. **可逆过程热力学函数的计算**：分步计算再求和
4. **理想气体焓的温度依赖性**：$H$ 只与 $T$ 有关

### 方法论
- 先分析过程性质（等温？绝热？等压？），再选公式
- 相变过程和气体压缩过程分别处理
- 利用状态函数的加和性（$\Delta S_\text{总} = \Delta S_1 + \Delta S_2$）

## 变式拓展

### 变式1：绝热压缩

### 变式2：van der Waals 气体
- 若水蒸气视为 van der Waals 气体而非理想气体，$pV = a$ 过程还是等温过程吗？

### 变式3：临界点附近
- 若蒸发在水的临界点（$374\,^{\circ}\mathrm{C}$，$218\,\mathrm{atm}$）附近进行，相界面消失，上述分析需要怎样修正？

## 易错分析

1. **$pV = a$ 的含义**：很多学生会误认为这是多方过程的一般形式。实际上对于理想气体，$pV = nRT$，若 $pV = a = \text{const}$，则 $T = a/(nR) = \text{const}$，就是等温过程。不要被 $pV = a$ 的形式迷惑。
2. **终态温度**：等温压缩意味着 $T$ 不变，$T_2 = T_1 = 373.15\,\mathrm{K}$。容易误认为压缩一定升温。
3. **液态水存在的判断**：需要比较终态压强 $p_2$ 与同温度下的饱和蒸气压 $p^*$。$T_2 = 373.15\,\mathrm{K}$ 时 $p^* = 1\,\mathrm{atm}$，若 $p_2 > 1\,\mathrm{atm}$ 则有液态水。
4. **理想气体焓**：理想气体的焓只取决于温度。等温过程中 $\Delta H = 0$，即使压强和体积都改变了。不要将 $\Delta H = nC_p\Delta T$ 与 $Q_p$ 混淆。
5. **熵变符号**：压缩使气体有序度增加（熵减），蒸发使液体变气体（熵增）。两部分符号相反，最终总熵变的正负取决于 $p_2$ 的大小。
