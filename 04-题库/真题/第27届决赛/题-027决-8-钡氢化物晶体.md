## 题目

**第8题（11分）**

$\mathrm{BaH_2}$ 为四方晶系，晶胞参数：$a = b = 305.2\,\mathrm{pm}$，$c = 383.1\,\mathrm{pm}$。晶体结构中 Ba 原子占据特殊位置，H 原子位于由 Ba 构成的多面体空隙中。

**8-1**（2分）写出 Ba 和 H 原子的分数坐标，并指出结构基元的内容。

**8-2**（2分）画出 $\mathrm{BaH_2}$ 沿 $a$ 方向（即 $[100]$ 方向）的投影图，标出原子位置。

**8-3**（2分）计算 H 原子之间的最短距离。

**8-4**（2分）描述 Ba 原子周围 H 原子构成的多面体类型，以及多面体之间的连接方式。

**8-5**（3分）计算 $\mathrm{BaH_2}$ 的储氢密度（以 $\mathrm{kg\,H_2/m^3}$ 表示），并与液氢的密度（$70.8\,\mathrm{kg/m^3}$）比较。

## 参考答案

### 8-1 分数坐标与结构基元

$\mathrm{BaH_2}$ 具有 $\mathrm{PbCl_2}$ 型（$\mathrm{C19}$，正交/四方变体）结构。在一个晶胞中：

| 原子 | Wyckoff 位 | 分数坐标 |
|------|-----------|----------|
| Ba（2个） | $2c$ | $(0, 0, 0)$；$(\tfrac{1}{2}, \tfrac{1}{2}, \tfrac{1}{2})$ |
| H（4个） | $4e$ | $(0, 0, z_1)$；$(0, 0, \bar{z_1})$；$(\tfrac{1}{2}, \tfrac{1}{2}, \tfrac{1}{2}+z_1)$；$(\tfrac{1}{2}, \tfrac{1}{2}, \tfrac{1}{2}-z_1)$ |

其中 $z_1 \approx 0.33$（具体值由精修确定）。

另外还有 4 个 H 原子位于偏心位置（实际结构中 H 原子有多个不等价位）。

**结构基元**：包含 2 个 Ba 和 4 个 H（即 2 个 $\mathrm{BaH_2}$ 公式单位），$Z = 2$。

### 8-2 沿 $a$ 方向投影图

沿 $[100]$ 方向投影到 $bc$ 平面：

```
  b
  ↑
  |
  |    Ba(1)        H        H        Ba(2)
  |     ●     ·    ○    ·    ○     ·    ●
  |          z₁   |         z₁+½  |
  |         _____|         _____|
  |        |     |        |     |
  |    ·   ○     |   ·    ○     |
  |   z₁-½       |   z₁         |
  |              |              |
  +————————————————————————————→ c
  
  ● = Ba 原子    ○ = H 原子
```

投影特点：
- Ba 原子位于投影面的角点和面心位置
- H 原子投影在 Ba 原子之间，沿 $c$ 方向有一定偏移（$z$ 坐标不同）
- 沿 $a$ 方向看，同一 $b$-$c$ 平面上 Ba 和 H 的排列交替出现

### 8-3 H 原子最短距离

两个 H 原子分数坐标差最小的情况：

取 H1 = $(0, 0, z_1)$ 和 H2 = $(0, 0, -z_1)$，沿 $c$ 方向的距离：

$$d = |2z_1| \times c$$

取 $z_1 \approx 0.33$：

$$d_{c} = 2 \times 0.33 \times 383.1 = 252.8\,\mathrm{pm}$$

另一个可能的最短距离：H1 = $(0, 0, z_1)$ 和 H2 = $(\tfrac{1}{2}, \tfrac{1}{2}, \tfrac{1}{2}+z_1)$：

$$d = \sqrt{(a/2)^2 + (b/2)^2 + (c/2)^2} = \sqrt{(152.6)^2 + (152.6)^2 + (191.6)^2}$$

$$= \sqrt{23287 + 23287 + 36710} = \sqrt{83284} \approx 288.6\,\mathrm{pm}$$

因此 **H 原子最短距离约为 $253\,\mathrm{pm}$**（沿 $c$ 方向的同列 H 原子之间）。

### 8-4 多面体类型与连接方式

每个 Ba 原子被 **8 个 H 原子**包围，形成**四方反棱柱**（distorted square antiprism）配位多面体。

多面体的连接方式：
- 相邻 Ba 的 H 配位多面体通过**共面**（sharing faces）方式连接
- 沿 $c$ 方向的多面体通过**共棱**（sharing edges）连接
- 整体形成三维无限网络结构

### 8-5 储氢密度

晶胞体积：
$$V = a^2 \times c = (305.2 \times 10^{-10}\,\mathrm{cm})^2 \times (383.1 \times 10^{-10}\,\mathrm{cm})$$
$$= (3.052 \times 10^{-8})^2 \times 3.831 \times 10^{-8}\,\mathrm{cm}^3$$
$$= 9.315 \times 10^{-16} \times 3.831 \times 10^{-8}$$
$$= 3.568 \times 10^{-23}\,\mathrm{cm}^3$$

晶胞中含 2 个 $\mathrm{BaH_2}$，即 4 个 H 原子，相当于 2 个 $\mathrm{H_2}$ 分子。

质量：
$$m = \frac{2 \times 2.016}{6.022 \times 10^{23}} = 6.695 \times 10^{-24}\,\mathrm{g}$$

储氢密度：
$$\rho_\mathrm{H_2} = \frac{m}{V} = \frac{6.695 \times 10^{-24}}{3.568 \times 10^{-23}} = 0.1877\,\mathrm{g/cm}^3 = 187.7\,\mathrm{kg/m^3}$$

**$\mathrm{BaH_2}$ 的储氢密度约为 $188\,\mathrm{kg\,H_2/m^3}$**，约为液氢密度（$70.8\,\mathrm{kg/m^3}$）的 **2.65 倍**。

## 解题思路

### 8-1
$\mathrm{BaH_2}$ 的结构基于 $\mathrm{PbCl_2}$ 型结构。利用 $Z = 2$ 和四方对称性确定原子坐标。Ba 位于高对称位置（$0,0,0$ 和 $\tfrac{1}{2},\tfrac{1}{2},\tfrac{1}{2}$），H 原子位于偏离中心的位置。

### 8-2
投影到 $bc$ 平面，沿 $a$ 方向观察。注意不同 $a$ 值的原子在投影面上的位置重叠关系。

### 8-3
直接用分数坐标差乘以晶胞参数计算距离，取最小值。注意 H 原子间的距离可能沿不同方向（$a$、$b$、$c$ 或对角线方向）。

### 8-4
分析 Ba 的配位环境（8 配位），判断多面体类型（四方反棱柱），再看多面体之间的共享元素（面、棱、角）。

### 8-5
储氢密度 = 晶胞中氢的质量 / 晶胞体积。关键在于正确计算晶胞中 H 原子的个数（$Z \times 2 = 4$ 个 H）。

## 考点抽象

### 核心考点
1. **晶体结构分析**：从晶胞参数和空间群推导原子坐标
2. **晶体投影**：三维结构向二维平面的投影方法
3. **配位多面体**：金属氢化物中阳离子的配位环境
4. **储氢材料性能评估**：体积储氢密度的计算与比较

### 方法论
- 晶体结构分析流程：空间群 → Wyckoff 位 → 原子坐标 → 结构特征
- 距离计算：$\Delta d = \sqrt{(\Delta x \cdot a)^2 + (\Delta y \cdot b)^2 + (\Delta z \cdot c)^2}$
- 储氢密度比较：与液氢、其他储氢材料对比评估

## 易错分析

1. **$Z$ 值的确定**：$\mathrm{BaH_2}$ 的 $Z = 2$（每个晶胞含 2 个公式单位，即 2 个 Ba + 4 个 H）。容易误认为 $Z = 1$ 或 $Z = 4$。
2. **H 原子个数**：一个 $\mathrm{BaH_2}$ 含 2 个 H，$Z = 2$ 则晶胞含 4 个 H。计算储氢密度时用 4 个 H 原子的质量而非 2 个 $\mathrm{H_2}$ 分子的质量（数值相同，但概念要清楚）。
3. **单位换算**：$305.2\,\mathrm{pm} = 3.052 \times 10^{-8}\,\mathrm{cm}$，体积计算时注意指数运算。$1\,\mathrm{g/cm^3} = 1000\,\mathrm{kg/m^3}$。
4. **H-H 距离的最短值**：不是简单地取 $z_1$ 方向的距离，还要考虑其他方向（如对角线方向）可能更短。需要逐一比较。
5. **投影图方向**：沿 $a$ 方向投影意味着 $a$ 轴垂直于纸面，投影面是 $bc$ 平面。不要画成沿 $c$ 方向的投影。
