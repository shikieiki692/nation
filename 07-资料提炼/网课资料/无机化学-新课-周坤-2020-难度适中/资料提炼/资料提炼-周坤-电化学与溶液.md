### 6.1 核心公式

| 公式 | 来源 | 说明 |
|:---|:---|:---|
| Nernst方程（25℃） | 13笔记 | E = E⊖ + (0.0591/n)lg([氧化型]/[还原型]) |
| 电势-pH关系 | 13笔记 | 对于2H⁺ + 2e⁻ ⇌ H₂，E = -0.0591·pH |
| 难溶盐电极电势 | 13笔记 | φAgX/Ag⊖ = φAg+/Ag⊖ + 0.0591lgKsp(AgX) |
| 元素电势图计算 | 13/14笔记 | E₃⊖ = (n₁E₁⊖ + n₂E₂⊖)/(n₁+n₂) |
| 平衡常数与电势 | 14笔记 | lgK⊖ = nFE⊖/(2.303RT) = nE⊖/0.0591（25℃） |
| 缓冲溶液 | 09笔记 | pH = pKa + lg([共轭碱]/[弱酸]) |
| 磁矩公式 | 09笔记 | μ = √[n(n+2)] B.M. |

### 6.2 重要电池反应

| 电池类型 | 正极反应 | 负极反应 | 总反应 |
|:---|:---|:---|:---|
| Zn-Mn干电池 | 2NH₄⁺ + 2e⁻ → 2NH₃ + H₂ | Zn → Zn²⁺ + 2e⁻ | Zn + 2MnO₂ → ZnO + Mn₂O₃ |
| Pb蓄电池 | PbO₂ + SO₄²⁻ + 4H⁺ + 2e⁻ → PbSO₄ + 2H₂O | Pb + SO₄²⁻ → PbSO₄ + 2e⁻ | PbO₂ + Pb + 2H₂SO₄ ⇌ 2PbSO₄ + 2H₂O |
| 锂离子电池 | Li₁₋ₓCoO₂ + xe⁻ → LiCoO₂ | LiₓC₆ → xLi⁺ + C₆ + xe⁻ | Li₁₋ₓCoO₂ + LiₓC₆ → LiCoO₂ + C₆ |
| 燃料电池（碱性） | O₂ + 2H₂O + 4e⁻ → 4OH⁻ | H₂ + 2OH⁻ → 2H₂O + 2e⁻ | 2H₂ + O₂ → 2H₂O |

### 6.3 关键图像描述

- **水的电势-pH图**：氢线（E=-0.059pH）与氧线（E=1.23-0.059pH）划分水的稳定区，b线以上为O₂稳定区，a线以下为H₂稳定区
- **Cr体系的电势-pH图**：AD线（Cr₂O₇²⁻→Cr³⁺，斜线）、BE线（Cr³⁺→Cr²⁺，水平线）、DE竖直线（Cr₂O₇²⁻⇌CrO₄²⁻，非电极反应）
- **Latimer图（溴酸性）**：BrO₄⁻ →(+1.85)→ BrO₃⁻ →(+1.45)→ HBrO →(+1.60)→ Br₂ →(+1.07)→ Br⁻
