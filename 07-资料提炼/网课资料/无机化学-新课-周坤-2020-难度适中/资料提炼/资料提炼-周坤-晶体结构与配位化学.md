## 二、教学逻辑锚点（简要版）

### 晶体结构篇
**认知链条**：晶体的宏观对称性（旋转轴、镜面）→ 微观点阵描述（晶胞、结构基元）→ 7大晶系与14种布拉维格子 → 金属密堆积（A1/A2/A3）→ 空隙类型与坐标 → 离子晶体填充模型（NaCl/ZnS/CaF2/CsCl/TiO2）→ 半径比规则预测配位数 → Pauling规则判断稳定性 → 离子极化修正 → 其他晶体类型（原子/分子/混合）。

### 配位化学篇
**认知链条**：配合物定义与组成（中心离子/配体/配位数）→ 命名规则 → 异构现象（几何/光学/结构）→ 价键理论VBT（杂化类型、内轨/外轨、磁矩）→ 晶体场理论CFT（d轨道分裂、分裂能Delta、光谱化学序列）→ 高自旋/低自旋与CFSE计算 → Jahn-Teller效应 → 18电子规则 → 反馈pi键（CO/CN-/膦配体）→ 反位效应 → 催化循环（烯烃氢化、Wacker氧化、Monsanto乙酸合成）。

## 六、关键图表和反应方程式

### 6.1 晶体结构关键图表

1. **七大晶系特征表**
   - 立方：a=b=c, alpha=beta=gamma=90deg
   - 六方：a=b≠c, alpha=beta=90deg, gamma=120deg
   - 三方：六方晶胞或菱面体晶胞（a=b=c, alpha=beta=gamma≠90deg）
   - 四方：a=b≠c, alpha=beta=gamma=90deg
   - 正交：a≠b≠c, alpha=beta=gamma=90deg
   - 单斜：a≠b≠c, alpha=gamma=90deg≠beta
   - 三斜：a≠b≠c, alpha≠beta≠gamma≠90deg

2. **14种布拉维格子**
   - 立方：P、I、F（3种）
   - 六方：P、R（2种）
   - 四方：P、I（2种）
   - 正交：P、I、C、F（4种）
   - 单斜：P、C（2种）
   - 三斜：P（1种）

3. **密堆积参数对比**
   - A1(ccp)：配位数12，空间利用率74.05%，4原子/晶胞
   - A2(bcc)：配位数8，空间利用率68.02%，2原子/晶胞
   - A3(hcp)：配位数12，空间利用率74.05%，2原子/晶胞，c/a=1.633

4. **空隙比例（所有最密堆积）**
   - 球数 : 正四面体空隙 : 正八面体空隙 = 1 : 2 : 1

5. **典型离子晶体结构汇总**

| 结构类型 | 晶系 | 点阵 | 配位数比 | 负离子堆积 | 正离子空隙 |
|:---|:---|:---|:---:|:---|:---|
| NaCl | 立方 | cF | 6:6 | A1(ccp) | 八面体100% |
| CsCl | 立方 | cP | 8:8 | 简单立方 | 立方体100% |
| 立方ZnS | 立方 | cF | 4:4 | A1(ccp) | 四面体50% |
| 六方ZnS | 六方 | hP | 4:4 | A3(hcp) | 四面体50% |
| CaF2 | 立方 | cF | 8:4 | 简单立方(F-) | 立方体50% |
| 金红石TiO2 | 四方 | tP | 6:3 | 伪六方 | 变形八面体50% |

6. **半径比规则临界值**
   - r+/r- ≥ 0.155 → 三角形（3配位）
   - r+/r- ≥ 0.225 → 正四面体（4配位，ZnS）
   - r+/r- ≥ 0.414 → 正八面体（6配位，NaCl）
   - r+/r- ≥ 0.732 → 立方体（8配位，CsCl）
   - r+/r- ≥ 1.0 → 最密堆积（12配位）

1. **VBT杂化类型表**

2. **光谱化学序列**
   I- < Br- < Cl- ≈ SCN- < F- < OH- < H2O ≈ C2O4^2- < NCS- < EDTA < Py < NH3 < en < NO2- < CN- < CO

3. **八面体场d轨道分裂**
   - Delta_o = 10Dq
   - E(eg) = +3/5 Delta_o（升高）
   - E(t2g) = -2/5 Delta_o（降低）

4. **CFSE计算公式**
   - d3：CFSE = 6/5 Delta_o
   - d4高自旋：CFSE = 3/5 Delta_o
   - d4低自旋：CFSE = 8/5 Delta_o - P
   - d7高自旋：CFSE = 4/5 Delta_o
   - d7低自旋：CFSE = 9/5 Delta_o - P

5. **18电子规则应用**
   - 计算方法：金属价电子 + 配体提供电子 = 18
   - Fe(CO)5：Fe(8e) + 5×2e = 18e
   - Mn2(CO)10：Mn(7e) + 5×2e + M-M(1e) = 18e
