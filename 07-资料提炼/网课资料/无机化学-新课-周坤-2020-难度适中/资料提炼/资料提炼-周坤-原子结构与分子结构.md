## 二、教学逻辑锚点（简要版）

**原子结构**：历史演进（经典物理危机→玻尔→量子力学）→数学工具（薛定谔方程、量子数）→可视化（电子云）→多电子近似（屏蔽/钻穿）→操作规则（排布三原则）→性质映射（周期律）。

**分子结构**：Lewis结构→VSEPR→杂化轨道→MO理论→分子间力/氢键。

### 原子结构部分（≥6道）

#### 题A1：元素R的离子构成
- **原文位置**：01.原子结构_笔记 / 课后习题 / 第1题；02.分子结构（一）_笔记1 / 例题1
- **题目概要**：元素R核电荷数为16，质量数32，则R²⁻离子应包含（）电子、质子、中子。
- **考点抽象**：核电荷数=质子数=原子序数；质量数=质子数+中子数；阴离子电子数=质子数+电荷数。
- **对应知识点**：原子结构基本概念
- **难度**：⭐⭐
- **教学作用**：基础概念辨析，区分Z/N/e⁻
- **是否录入题库**：⬜

#### 题A2：锂的原子量计算
- **原文位置**：01.原子结构_笔记 / 课后习题 / 第2题；02.分子结构（一）_笔记1 / 例题2
- **题目概要**：锂含7.42%⁶Li(m=6.01513)和92.58%⁷Li(m=7.01601)，计算原子量。
- **考点抽象**：加权平均法计算原子量；同位素丰度与精确质量。
- **对应知识点**：同位素、原子量
- **难度**：⭐⭐
- **教学作用**：数值计算训练
- **是否录入题库**：⬜

#### 题A3：基态/激发态/错误构型判断
- **原文位置**：01.原子结构_笔记 / 课后习题 / 第3题；02.分子结构（一）_笔记1 / 例题3
- **题目概要**：判断(1)1s²2s²2p¹ (2)1s²2p² (3)1s²2s³ (4)1s²2s²2p⁶3s¹3d¹ (5)1s²2s²2p⁶4f¹ (6)1s²2s²2p⁵ 的归属。
- **考点抽象**：构造原理（能量最低）、泡利不相容原理（同轨道最多2电子）、激发态识别（电子跃迁至高能级）。
- **对应知识点**：电子排布规则
- **难度**：⭐⭐

#### 题A4：电子排布式书写

- **题目概要**：不查表写出C(6), N(7), P(15), Sc(21), Ni(28), Zn(30), Ga(31), As(33), Zr(40), Te(52)的电子排布式。
- **考点抽象**：能级填充顺序1s→2s→2p→3s→3p→4s→3d→4p...；Cr/Cu半满/全满特例；离子失电子顺序np→ns→(n-1)d。
- **对应知识点**：电子排布

#### 题A5：原子序数51的元素推测
- **原文位置**：02.分子结构（一）_笔记1 / 例题13

- **考点抽象**：以稀有气体为基准定位（36Kr<51<54Xe）；价电子构型5s²5p³→第五周期VA族；半金属特性；最高氧化态=主族序数。

#### 题A6：He的第一电离能讨论
- **原文位置**：02.分子结构（一）_笔记1 / 例题16
- **题目概要**：(1)为什么He的第一电离能最高？(2)第二电离能最高的是何种元素？
- **考点抽象**：第一电离能：1s²全满+最小半径→最强核束缚；第二电离能：Li⁺(1s²)形成He-like稳定结构→再失电子极难。
- **对应知识点**：电离能
- **难度**：⭐⭐⭐
- **教学作用**：深化电离能概念，关联电子构型稳定性
- **是否录入题库**：⬜

#### 题A7：同周期半径反常解释
- **原文位置**：02.分子结构（一）_笔记1 / 例题17
- **题目概要**：解释Cu(128pm)/Zn(134pm) > Ni(124pm)，Ar(191pm) > Cl(185pm)。
- **考点抽象**：全充满/半充满电子层屏蔽效应增强→有效核电荷降低→电子云扩张；范德华半径：满电子层原子间相互作用减弱。
- **对应知识点**：原子半径、屏蔽效应
- **难度**：⭐⭐⭐
- **教学作用**：用电子构型解释反常数据
- **是否录入题库**：⬜

#### 题A8：第一电离能顺序判断
- **原文位置**：02.分子结构（一）_笔记1 / 例题19-20
- **题目概要**：判断B<C<N（正确）vs B<C<N<O（错误）；Na<Mg>Al<Si。
- **考点抽象**：半满/全满稳定性的电离能反常：N(2p³半满)>O(2p⁴)、Mg(3s²全满)>Al(3s²3p¹)。
- **对应知识点**：电离能、元素周期律
- **难度**：⭐⭐⭐
- **教学作用**：识别并解释电离能曲线中的"锯齿"
- **是否录入题库**：⬜

#### 题A9：元素性质与核电荷数关系曲线
- **原文位置**：02.分子结构（一）_笔记1 / 例题23
- **题目概要**：将IIA族价电子数、VIIA族氢化物沸点、第三周期单质熔点、最高正化合价、离子半径等性质与曲线匹配。
- **考点抽象**：多性质综合：价电子数恒定→沸点HF异常高（氢键）→熔点先升后降（Si最高）→化合价+1到+7→离子半径随电荷增加而减小。
- **对应知识点**：元素周期律综合
- **难度**：⭐⭐⭐⭐
- **教学作用**：多性质曲线识别与解释
- **是否录入题库**：⬜

#### 题M4：N₂成键的价键理论解释
- **原文位置**：03.分子结构（二）_笔记 / 近代价键理论 / 例题
- **题目概要**：用经典价键理论解释N₂的成键。
- **考点抽象**：N原子2p³构型，3个单电子分布在px/py/pz；pz头碰头形成σ键，px/py肩并肩形成两个π键；结论：1σ+2π。
