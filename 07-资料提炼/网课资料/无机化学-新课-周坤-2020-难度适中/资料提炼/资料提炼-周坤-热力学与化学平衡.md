## 二、教学逻辑锚点（简要版）

**热力学篇**：建立热力学语言与动机（为什么需要热力学）→ 体系/环境/状态函数 → 第一定律（ΔU = Q + W）→ 热化学计算（生成热/盖斯定律/燃烧热）→ 过程方向判据（熵S / Gibbs自由能G）→ 绝对熵基准（第三定律）。

**化学平衡篇**：可逆反应与动态平衡 → 经验平衡常数Kc/Kp与标准平衡常数Kθ → 反应商Q与等温方程（ΔG = ΔG° + RTlnQ）→ ΔG° = -RTlnK 的桥梁作用 → Le Chatelier原理（浓度/压强/温度）→ van't Hoff方程（温度对K的影响）。

**电化学预告**（第12讲后半段）：原电池构造 → 电极电势概念 → 标准氢电极 → Nernst方程初步 → ΔG = -nFE 关联。

### 6.3 关键反应方程式

- C(石墨) + O₂(g) → CO₂(g)  ΔH° = -393.5 kJ/mol（燃烧热基准）
- H₂(g) + ½O₂(g) → H₂O(l)  ΔH° = -285.8 kJ/mol
- N₂(g) + 3H₂(g) ⇌ 2NH₃(g)（合成氨，平衡示例）
- PCl₅(g) ⇌ PCl₃(g) + Cl₂(g)（解离度经典题）
- CaCO₃(s) ⇌ CaO(s) + CO₂(g)（复相平衡，K=pCO₂）
- Zn + Cu²⁺ → Zn²⁺ + Cu（丹尼尔电池总反应）
- 2H⁺ + 2e⁻ → H₂(g)（标准氢电极，E°=0V）
