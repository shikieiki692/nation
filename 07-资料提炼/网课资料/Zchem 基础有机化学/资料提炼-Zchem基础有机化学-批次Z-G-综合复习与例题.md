## 二、📐 教学逻辑锚点

> Z-G批次的最大价值是填补了学而思有机化学基础在"综合复习+例题讲解+备考策略"方面的完全空白。学而思15讲以知识讲授为主，而Zchem的Z-G提供了四个维度的冲刺素材：①**解题方法论的系统化**（SN/E三步判断法、定位效应决策树）；②**近50道典型例题**覆盖全部高频考点；③**生物有机补充**（糖、氨基酸、蛋白质、聚合物）；④**量化备考策略**（冲省一/冲省队的目标分解和得分规划）。

### 2.1 四大批次的功能分工

| 文件 | 功能定位 | 核心内容 | 与学而思关系 |
|:---|:---|:---|:---|
| **上7** | 上册知识例题化 | 结构基础、酸碱、构象、立体化学、自由基、SN/E的22道例题 | 与学而思批次A-B重叠，但例题不同 |
| **下7** | 下册机理深化+系统总结 | 定位效应权重规则、SNAr、SN/E三步判断法、频哪醇重排、环氧开环 | 与学而思批次C-E重叠，但方法论更系统 |
| **下14** | 查漏补缺+生物有机+备考策略 | 波谱纠错、含氮化合物、糖类、氨基酸蛋白质、聚合物、得分规划 | **学而思完全未涉及** |
| **下15** | 全体系高频考点例题串讲 | 芳香性、立体化学、自由基、SN/E、烯烃加成、羰基化学等全考点覆盖 | **学而思完全未涉及** |

### 2.2 原文讲解顺序与铺垫策略

**上7**：以"知识点回顾→例题实战→解题要点→典型错误"四段式组织，每个例题都配有"易错点"警示。

**下7**：以"作业讲解→系统总结→方法论提炼→例题验证"四段式组织，强调从具体题目中抽象出通用判断框架。

**下14**：以"查漏补缺→知识补充→策略指导"三段式组织，先解决具体错误，再补充知识缺口，最后给出备考规划。

**下15**：以"知识点→核心内容→考试重点/易混淆点→难度系数→例题"五段式组织，每个知识点配有速查表。

### 7.1 SN/E竞争反应三步判断法（讲义核心工具）

**第一步：判试剂——四大类型**

| 类型 | 代号 | 典型试剂 | 特征 |
|:---|:---:|:---|:---|

| 强亲核强碱 | S_Nu/SB | RO⁻(EtO⁻等), NH₂⁻ | 亲核性和碱性都强 |
| 弱亲核弱碱 | WNu/WB | H₂O, ROH, RCOOH | 中性分子 |

**第二步：判机理——底物级别 × 试剂类型**

| 底物 | S_Nu/WB | WNu/SB | S_Nu/SB | WNu/WB |
|:---:|:---:|:---:|:---:|:---:|
| **1°-X** | **SN2** | **E2** | **SN2 >> E2** | N.R.（不反应） |
| **2°-X** | SN1/SN2竞争 | **E2** | **E2 > SN** | messy（混乱） |
| **3°-X** | **SN1** | **E2** | **E2 >> SN1** | E1 + SN1 |

**记忆口诀**：
- 一级取代多，三级消除强，二级两边倒
- 大位阻碱爱消除（E2），小位阻碱爱取代（SN2）
- 质子溶剂爱碳正（SN1/E1），非质子溶剂爱背面（SN2）

**第三步：判产物——温度效应**

```
ΔG = ΔH - TΔS
    ↓
SN反应：ΔS ≈ 0（双分子→双分子，或单分子→单分子）
E反应：ΔS > 0（分子数增加，生成烯烃）
    ↓
低温 → 取代有利（熵变贡献小，焓变主导）
高温 → 消除有利（TΔS项增大）
```

### 7.2 定位效应"最强致活基团"规则（讲义核心工具）

**核心规则**：多取代苯发生亲电取代时，**由最强致活基团（most strongly activating group）决定**主要产物位置。

**致活/致钝能力排序**：

| 强度 | 基团类型 | 典型基团 | 定位 | 机理 |
|:---|:---|:---|:---:|:---|
| 极强致活 | 含孤对电子的杂原子 | -NH₂, -NHR, -NR₂, -OH, -OR | 邻对位 | p-π共轭给电子 |
| 中等致活 | 烷基 | -R, -Ar | 邻对位 | 超共轭给电子 |
| 弱致钝 | 卤素 | -F, -Cl, -Br, -I | 邻对位 | 诱导吸电子+共振给电子（净致钝） |
| 中等致钝 | 吸电子基（无孤对电子） | -COCH₃, -CHO, -COOR | 间位 | π键极化吸电子 |
| 强致钝 | 强吸电子基 | -NO₂, -CN, -SO₃H, -NR₃⁺ | 间位 | 强诱导+共振吸电子 |

**决策流程**：

```
苯环上有多个取代基
    ↓
Step 1：识别每个基团的类型（致活/致钝+定位）
    ↓
Step 2：找出最强致活基团
    ↓
Step 3：确定该基团的邻/对位
    ↓
Step 4：考虑位阻效应（大基团邻位位阻大→对位优势）
    ↓
Step 5：确定主产物
```

**典型案例**：

| 取代基组合 | 最强致活基团 | 主要产物位置 | 原因 |
|:---|:---|:---:|:---|
| -CH₃ + -NO₂ | -CH₃（弱致活） | 邻对位（相对甲基） | 硝基是致钝基，效应被覆盖 |
| -OH + -CH₃ | -OH（极强致活） | 邻对位（相对羟基） | 羟基致活能力 >> 甲基 |
| -Br + -CH₃ | -CH₃（弱致活）>-Br（弱致钝） | 邻对位（相对甲基） | 甲基虽弱但仍是致活，溴是净致钝 |
| -NHCOCH₃ + -COCH₃ | -NHCOCH₃（中等致活） | 邻对位（相对酰胺基） | 乙酰氨基的孤对电子共轭给电子 |

**位阻修正**：当最强致活基团的邻位有大位阻基团时，对位产物比例增加。

### 7.3 多取代苯合成逆推框架（讲义高阶工具）

**四步法**：

```
目标分子
    ↓
【Step 1】识别目标分子中的官能团和取代模式
    ↓
【Step 2】判断取代基引入顺序（利用定位效应）
         ├─ 先引入邻对位定位基 → 再引入新基团到邻/对位
         └─ 先引入间位定位基 → 再引入新基团到间位
    ↓
【Step 3】检查是否需要占位基/保护基/氧化还原切换
         ├─ 占位基：-SO₃H（引入后稀酸水解除去）
         ├─ 保护基：-NH₂ → -NHCOCH₃（保护氨基不被氧化）
         └─ 氧化切换：-CH₃ → -COOH（将邻对位定位改为间位定位）
    ↓
【Step 4】检查傅克反应是否会发生重排
         ├─ 傅克烷基化：碳正离子中间体 → 可能重排
         └─ 若会重排 → 改用傅克酰基化 + Clemmensen/Wolff-Kishner还原
    ↓
写出正向合成路线（以苯为起始原料）
```

**经典案例：对位硝基正丙基苯的合成**

```
错误路线：
苯 + CH₃CH₂CH₂Br/AlCl₃ → 正丙基苯（❌ 实际得到异丙基苯，因碳正离子重排）
正丙基苯 + HNO₃/H₂SO₄ → 邻/对位硝基正丙基苯（比例不理想）

正确路线：
苯 + CH₃CH₂COCl/AlCl₃ → 苯丙酮（傅克酰基化，无重排）
苯丙酮 + HNO₃/H₂SO₄ → 对硝基苯丙酮（羰基是间位定位基，但烷基链远离苯环，实际硝基进入苯环相对羰基的间位=相对烷基的对位）
对硝基苯丙酮 + Zn-Hg/HCl (Clemmensen) → 对硝基正丙基苯 ✅
```

**占位基策略案例：制备特定位置的三取代苯**

```
目标：1,3-二溴-5-硝基苯
    ↓
分析：硝基是间位定位基，溴是邻对位定位基
    ↓
策略：先引入占位基-SO₃H控制位置

苯 + 浓H₂SO₄ → 苯磺酸
苯磺酸 + Br₂/FeBr₃ → 对溴苯磺酸（-SO₃H是间位定位基，Br进入间位=对位）
对溴苯磺酸 + Br₂/FeBr₃ → 3,5-二溴苯磺酸（-SO₃H导向间位，-Br导向邻对位，两者在3,5位重合）
3,5-二溴苯磺酸 + 稀H₂SO₄/△ → 1,3-二溴苯（水解去除-SO₃H）
1,3-二溴苯 + HNO₃/H₂SO₄ → 1,3-二溴-5-硝基苯 ✅（两个-Br共同导向5位）
```

### 7.4 频哪醇重排双规则（Zchem下7核心增量）

**反应通式**：
```
    OH  OH              O
    |   |               ||
R₁-C---C-R₂  ──H⁺──→  R₁-C-C-R₂  + R₂迁移产物
    |   |               |
   R₃   R₄            R₃（或R₄）
```

**规则一：稳定碳正离子的基团迁移更快**
- 迁移能力：Ph > R > H（与Wagner-Meerwein重排"越小越快"相反！）
- 原因：迁移基团需通过共轭效应或超共轭效应稳定过渡态中的正电荷
- 实例：苯基迁移快于甲基，因为苯基可通过共轭分散正电荷

**规则二：满足反式共平面迁移速率更快**
- 空间要求：迁移基团与离去基团（质子化的OH→H₂O）需处于反式共平面
- 轨道理论：迁移基团的σ键HOMO与碳正离子空p轨道LUMO需要最大重叠
- 环己烷体系：只有处于反式双直立键（diaxial）位置的基团才能有效迁移

**与Wagner-Meerwein重排的关键区别**：

| 特征 | W-M重排 | 频哪醇重排 |
|:---|:---|:---|
| 迁移顺序 | 越小越快（H > Me > Et） | 给电子能力越强越快（Ph > R > H） |
| 原因 | 轨道匹配性（小基团σ键与空p轨道重叠更好） | 正电荷稳定能力（过渡态需求） |
| 驱动力 | 生成更稳定碳正离子 | 生成更稳定羰基化合物（C=O键能大） |

### 7.5 环氧开环反应：酸性vs碱性条件对比（讲义核心对比表）

| 对比维度 | 酸性条件 | 碱性/亲核条件 |
|:---|:---|:---|
| **机理** | 质子化环氧氧 → 碳正离子特征 | 亲核试剂直接进攻中性环氧 |
| **主导因素** | 电子效应 vs 位阻效应**竞争** | **位阻效应绝对主导** |
| **进攻位点** | 取代度高的碳（正电荷更稳定）或位阻小的碳 | 始终是**位阻小的碳** |
| **立体化学** | 反面进攻（但中间体有部分碳正离子特征） | **严格反面进攻**，反式开环 |
| **产物特点** | 常得混合物，主产物需具体分析 | 产物单一，可预测 |

**酸性条件经验法则**：
- 当两个碳上取代基差异大时：电荷效应和位阻效应可能指向同一碳 → 产物单一
- 当两个碳都有给电子基时：电子效应竞争 → 可能得混合物
- 苯基 vs 甲基：苯基更能稳定正电荷 → 亲核试剂进攻苯基相连的碳（反直觉，因苯基位阻也大）

**碱性条件铁律**：
```
Nu:⁻ 总是进攻位阻较小的碳
    ↓
原因：SN2机理，背面进攻需要空间可及性
    ↓
立体化学：Nu从环氧环的反面进攻，与氧原子反式
```

### 7.6 气相vs液相酸性差异（Zchem上7理论深化）

**核心矛盾**：
- **液相pKa**：乙醇(15) < 叔丁醇(17) → 乙醇酸性更强
- **气相酸性**：叔丁醇 > 乙醇 → 叔丁醇酸性更强

**原因分析**：

| 相态 | 决定因素 | 乙醇 | 叔丁醇 | 结论 |
|:---|:---|:---|:---|:---|
| **气相** | 分子内部结构（诱导效应） | 乙基给电子诱导 > 甲基 | 三个甲基给电子更强 | 气相：t-BuOH > EtOH |
| **液相** | 溶剂化效应 + 位阻效应 | 乙氧负离子小，易溶剂化 | 叔丁氧负离子位阻大，难溶剂化 | 液相：EtOH > t-BuOH |

### 7.7 超共轭效应的轨道本质（Zchem上7理论深化）

**传统说法**：烷基是给电子基团，通过超共轭稳定碳正离子。

**Zchem的深层解释**（HOMO-LUMO相互作用）：

```
超共轭稳定碳正离子：
σ(C-H)成键轨道（HOMO） → 空p轨道（LUMO）
    ↓
能量匹配良好，有效重叠 → 电子离域 → 碳正离子稳定

超共轭不能稳定碳负离子：
σ(C-H)成键轨道（HOMO） → 碳负离子的满p轨道（已填满，不是LUMO）
    ↓
碳负离子的LUMO是σ*反键轨道，能量太高，与σ(C-H)匹配差
    ↓
无有效相互作用 → 烷基给电子反而 destabilize 碳负离子
```

**结论**：
- 超共轭的本质是σ键电子向**空轨道或半满轨道**的离域
- 碳正离子（空p轨道）✅ 可被超共轭稳定
- 自由基（半满p轨道）✅ 可被超共轭稳定
- 碳负离子（满p轨道）❌ 不能被超共轭稳定

**位阻效应与碳负离子稳定性**：
- 甲基负离子 > 叔丁基负离子（稳定性顺序）
- 原因：甲基负电荷暴露，可通过溶剂化或正离子静电作用稳定
- 叔丁基负离子位阻大，不易发生有效溶剂化作用

### 7.8 含氮化合物系统（Zchem下14纯增量内容）

#### 📌 胺的手性翻转（Pyramidal Inversion）

**现象**：三级胺氮原子连接三个不同基团时，在室温下可快速完成构型翻转（像雨伞翻转）。

**结果**：R和S构型快速互变 → 无法测出旋光性 → **通常不视为手性中心**。

**抑制翻转的条件**（此时氮可视为手性中心）：
1. **极低温度**：减缓分子热运动
2. **大位阻取代基**：如叔丁基，阻碍翻转路径
3. **桥环结构**：桥头氮因几何约束难以翻转
4. **配位固定**：孤对电子与路易斯酸形成强配位键

**NMR证据**：翻转被抑制时，原本等价的氢（如桥头CH₂的两个H）在低温NMR中显示为两个峰。

#### 📌 胺的碱性排序

| 胺类型 | 典型化合物 | pKb | 原因 |
|:---|:---|:---:|:---|
| 芳香胺 | 苯胺 | ~9.4 | 孤对电子参与芳香共轭，电子密度降低 |
| 一级胺 | 甲胺 | ~3.4 | 烷基诱导给电子 |
| 二级胺 | 二甲胺 | ~3.2 | 两个烷基给电子，但位阻略有增加 |
| 三级胺 | 三甲胺 | ~4.2 | 三个烷基给电子，但位阻大+溶剂化差 |
| 吡啶（SP²氮） | 吡啶 | ~8.8 | SP²杂化，孤对电子在SP²轨道，s成分高，束缚强 |

**排序规律**：脂肪胺（2° > 1° > 3°）> 氨 > 芳香胺 > 吡啶

#### 📌 胺的制备方法总结

| 方法 | 反应式 | 适用 | 特点 |
|:---|:---|:---|:---|
| **硝基还原** | Ar-NO₂ + Fe/Zn/Sn + HCl → Ar-NH₂ | 芳香胺 | 经典方法，条件剧烈 |
| **选择性还原** | Ar-NO₂ + SnCl₂/H⁺ 或 ZnCl₂/H⁺ → Ar-NH₂ | 含羰基的芳香硝基化合物 | 避免羰基同时被还原 |
| **卤代烃+NaN₃** | R-X + NaN₃ → R-N₃ → NaBH₄ 或 LAH → R-NH₂ | 脂肪胺 | 迭氮化物危险，不常用 |
| **Gabriel合成** | 邻苯二甲酰亚胺钾盐 + R-X → SN2 → 水解 → R-NH₂ | **伯胺（实验室最常用）** | 只产伯胺，纯度高 |
| **还原氨基化** | R₂C=O + R'NH₂ → 亚胺 → NaBH₃CN → R₂CH-NHR' | 1°/2°/3°胺均可 | 一锅法，操作简便 |

#### 📌 霍夫曼消除（Hofmann Elimination）

**反应通式**：
```
R₄N⁺ X⁻ + Ag₂O/H₂O → R₄N⁺ OH⁻ ──△──→ 烯烃 + R₃N + H₂O
```

**特点**：
- 生成**最少取代的烯烃**（霍夫曼产物）——与扎伊采夫规则相反！
- 原因：四级铵盐位阻极大，OH⁻优先进攻位阻最小的β-H（动力学控制）
- 本质：虽然OH⁻是小位阻碱，但离去基团(R₃N)位阻极大，导致过渡态位阻效应主导

**与E2消除的对比**：

| 条件 | 碱 | 离去基团 | 产物 | 控制因素 |
|:---|:---|:---|:---|:---|
| 普通E2 | t-BuOK（大位阻） | Br⁻（小位阻） | 霍夫曼产物 | 碱的位阻 |
| 霍夫曼消除 | OH⁻（小位阻） | R₃N（大位阻） | 霍夫曼产物 | 离去基团的位阻 |

#### 📌 重氮盐反应群

**制备**：Ar-NH₂ + NaNO₂ + HCl (0-5°C) → Ar-N₂⁺ Cl⁻

**重要反应**：

| 反应类型 | 试剂 | 产物 | 备注 |
|:---|:---|:---|:---|
| **Sandmeyer** | CuX (X=Cl, Br, CN) | Ar-X, Ar-CN | 不能引入F（CuF不稳定） |
| **Schiemann** | HBF₄ → △ | Ar-F | Balz-Schiemann反应 |
| **还原去氨基** | H₃PO₂ (次磷酸) | Ar-H | 将-NH₂完全去除 |
| **水解** | H₂O/△ | Ar-OH | 亲核取代 |

**定位基应用**：制备间位三溴苯
```
苯 → HNO₃/H₂SO₄ → 硝基苯 → Br₂/FeBr₃ → 间溴硝基苯 → Sn/HCl → 间溴苯胺
间溴苯胺 → NaNO₂/HCl → 重氮盐 → CuBr → 间二溴苯 → ...（重复溴化）
    ↓
或直接：苯胺 → 溴化 → 2,4,6-三溴苯胺 → 重氮化 → 次磷酸还原 → 1,3,5-三溴苯
```

### 7.9 糖类立体化学（Zchem下14纯增量内容）

#### 📌 D/L构型判定（Fischer投影式）

**规则**：
1. 氧化态最高的基团（CHO或COOH）置于最上方
2. 最下端手性碳的OH在**右侧为D型**，**左侧为L型**
3. 天然葡萄糖为**D型**

**葡萄糖构型**：碳2-碳5的OH方向为 **右-左-左-右**

#### 📌 环状结构与异头碳效应

**成环机制**：分子内半缩醛反应
- C5-OH进攻C1醛基 → **六元环（吡喃糖）**，为主产物
- C4-OH进攻C1醛基 → **五元环（呋喃糖）**，为次产物

**α/β异构体**：
- **α-D-吡喃葡萄糖**：半缩醛OH与C6-CH₂OH **反向**（异侧）
- **β-D-吡喃葡萄糖**：半缩醛OH与C6-CH₂OH **同向**（同侧）

**稳定性**：β型(64%) > α型(36%)，因为β型中大基团都处于平伏键-like位置

#### 📌 变旋光计算

**实验数据**：
- 纯α型：[α]D = +112.2°
- 纯β型：[α]D = +18.7°
- 平衡混合液：[α]D = +52.6°

**计算**：
```
112.2x + 18.7(1-x) = 52.6
解得 x = 0.36（α型36%，β型64%）
```

#### 📌 多糖差异（淀粉 vs 纤维素）

| 特征 | 淀粉 | 纤维素 |
|:---|:---|:---|
| 单体连接 | α-1,4-糖苷键 | β-1,4-糖苷键 |
| 人体消化 | ✅ 可消化 | ❌ 不可消化 |
| 原因 | 消化酶识别α型 | 无β型糖苷酶 |

### 7.10 氨基酸与蛋白质（Zchem下14纯增量内容）

#### 📌 天然氨基酸构型

- 大多数天然氨基酸为**L-构型**
- L/D命名规则：Fischer投影式中**NH₂在左侧为L型**，右侧为D型
- **注意**：与糖类D/L规则对比——糖是OH在右侧为D，氨基酸是NH₂在左侧为L
- 绝对构型：L-氨基酸的α-碳通常为**S构型**（半胱氨酸例外为R）

#### 📌 等电点（pI）计算

**定义**：氨基酸以两性离子（zwitterion）形态浓度最高时的pH值

**计算公式**：
```
pI = (pKa₁ + pKa₂) / 2
```
- pKa₁：羧基解离（较小值，~2-2.5）
- pKa₂：氨基质子化解离（较大值，~9-9.5）

**多官能团氨基酸**（如赖氨酸、谷氨酸）：
- 取**同类基团**的pKa值平均
- 赖氨酸：pKa(α-COOH)=2.18, pKa(α-NH₃⁺)=8.95, pKa(侧链-NH₃⁺)=10.53
- pI = (8.95 + 10.53) / 2 = 9.74（取两个碱性基团的平均）

**电泳方向判断**：
- pH > pI → 氨基酸带**负电** → 向**正极**移动
- pH < pI → 氨基酸带**正电** → 向**负极**移动

#### 📌 蛋白质结构层次

| 结构层次 | 定义 | 作用力 |
|:---|:---|:---|
| **一级结构** | 氨基酸通过肽键(-CO-NH-)连接的序列 | 共价键（肽键） |
| **二级结构** | 局部折叠（α-螺旋、β-折叠） | **氢键**（主链C=O与N-H之间） |
| **三级结构** | 整体三维构象 | 氢键、范德华力、疏水作用、**二硫键**(-S-S-) |
| **四级结构** | 多条肽链的组装 | 非共价相互作用 |

**二硫键**：两个半胱氨酸的-SH在氧化条件下形成-S-S-，还原条件下可断裂。

**蛋白质变性**：
- 重金属盐：与-SH、-NH₂结合破坏结构
- 加热：破坏氢键（二级结构）
- 强酸/强碱：改变电荷分布
- 盐析：改变溶解度（通常可逆）

### 7.11 聚合物合成（Zchem下14高频考点补充）

#### 📌 加成聚合反应（Chain-Growth Polymerization）

| 机理类型 | 引发剂 | 适用单体 | 中间体 |
|:---|:---|:---|:---|
| **自由基聚合** | ROOR (过氧化物), hν | 乙烯基单体（无明显EDG/EWG） | 自由基 |
| **阳离子聚合** | 路易斯酸, H⁺ | 带给电子基单体（异丁烯, 苯乙烯） | 碳正离子 |
| **阴离子聚合** | 强碱 (BuLi) | 带吸电子基单体（丙烯腈, 甲基丙烯酸甲酯） | 碳负离子 |

**天然橡胶**：1,3-二烯（异戊二烯）的自由基加成聚合

#### 📌 缩合聚合反应（Step-Growth Polymerization）

**经典案例：尼龙66**
```
HOOC-(CH₂)₄-COOH + H₂N-(CH₂)₆-NH₂ → [-CO-(CH₂)₄-CO-NH-(CH₂)₆-NH-]n + nH₂O
```
- 单体：己二酸 + 己二胺（各6个碳→尼龙66）
- 特征：释放小分子（水）
- 聚合物名称规律：尼龙XY = X碳二胺 + Y碳二酸

**橡胶硫化**：
- 线性聚异戊二烯 + 硫（S₈）→ 二硫键交联
- 硫含量越高，交联越多，材料越硬（从弹性体→硬质橡胶）

#### 📌 单体推断陷阱（竞赛高频）

**陷阱**：给出的聚合物结构可能经过消除、重排等后续反应，不能直接对应单体。

**正确思路**：
1. 识别聚合物中的重复单元
2. 判断是加成聚合还是缩合聚合
3. 加成聚合：将重复单元中的单键恢复为双键
4. 缩合聚合：找出被缩合掉的小分子，恢复官能团
