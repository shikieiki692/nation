## 二、教学逻辑锚点（简要版）

L4 的核心主线是 **"Q/K 判断方向"**：先用热力学公式（ΔG° = -RTlnK°）解释 K 的物理意义，再用 Q 与 K 的定量比较判断反应方向，最后把 Le Chatelier 原理作为定性近似收束。

核心决策路径：**写 Q → 比 K → 判断方向 → 再处理条件变化**。

## 六、关键图表和反应方程式

### 6.1 核心公式汇总

| 公式 | 适用条件 | 备注 |
|:---|:---|:---|
| ΔG = ΔG° + RTlnQ | 任意状态 | van't Hoff 等温式 |
| ΔG° = -RTlnK° | 平衡状态 | 连接热力学与平衡常数 |
| ln(K₂/K₁) = -ΔH°/R(1/T₂ - 1/T₁) | 温度变化 | ΔH° 视为常数 |
| ln(P₂/P₁) = -ΔHvap/R(1/T₂ - 1/T₁) | 液-气或固-气相变 | Clausius-Clapeyron |
| ΔG = RTln(Q/K°) | 方向判断 | Q>K° 则 ΔG>0 |
| Kc = Kp·(RT)^(-Δn) | Kp 与 Kc 换算 | Δn = Σν(气态产物) - Σν(气态反应物) |

### 6.2 重要反应方程式

- Br₂(l) ⇌ Br₂(g) —— 沸点/相变平衡示例
- NH₄HS(s) ⇌ NH₃(g) + H₂S(g) —— 固体分解 1:1 产气
- NH₄Cl(s) ⇌ NH₃(g) + HCl(g) —— 固体分解总压计算
- 2NO₂(g) ⇌ N₂O₄(g) —— 转化率/分压计算经典体系
- PCl₅(g) ⇌ PCl₃(g) + Cl₂(g) —— 平衡移动条件辨析经典体系
- 2SO₂(g) + O₂(g) ⇌ 2SO₃(g) —— Kp/Kc 换算示例
