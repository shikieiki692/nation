## 二、教学逻辑锚点（简要版）

质心 L1-L3 的核心讲授逻辑是：从"能量守恒"到"过程方向性"再到"平衡入口"的三段递进。

L1 打热力学语言地基：先建立体系/环境、状态量vs过程量、ΔU=Q+W 的能量账本，再引出焓 H=U+PV 及其恒压热效应意义，最后以 Hess 定律和标准生成焓把状态函数思想转化为反应热计算工具。

L2 从热化学计算升级到方向判据：通过 Qp/QV/ΔU/W/ΔH 同题辨析强化热化学计算，再以第二定律和熵 S 打破"能量守恒即可自发"的直觉，最终引入 Gibbs 自由能 ΔG=ΔH-TΔS 作为过程方向性的统一判据。

L3 把 Gibbs 判据接到化学平衡：以 ΔG=0 解释相变临界温度，再引入反应商 Q 和标准平衡常数 K°，通过等温方程 ΔG=ΔG°+RTlnQ 和 ΔG°=-RTlnK° 建立热力学到化学平衡的桥梁。

## 六、关键图表和反应方程式

### 核心公式

| 公式 | 来源 | 备注 |
|:---|:---|:---|
| ΔU = Q + W | L1 | 热力学第一定律，密闭体系 |
| W = -PΔV = -RTΔn（理想气体恒压） | L1/L2 | 化学中体积功的标准表达式 |
| H = U + PV；ΔH = ΔU + Δ(PV) | L1 | 焓的定义 |
| ΔH = Qp（恒压只做体积功） | L1 | 化学中最常用的热效应关系 |
| Qp = QV + RTΔn | L2 | 恒压与恒容热效应的换算 |
| ΔrH° = ΣnᵢΔfH°(产物) - ΣnⱼΔfH°(反应物) | L1/L2 | 标准生成焓计算反应焓 |
| ΔrH° ≈ ΣBE(反应物) - ΣBE(产物) | L2 | 键能估算（气态平均值） |
| S = klnΩ = (R/NA)lnΩ | L2 | 玻尔兹曼熵公式 |
| ΔrS° = ΣS°(产物) - ΣS°(反应物) | L2 | 反应熵变计算 |
| G = H - TS；ΔG = ΔH - TΔS（恒温） | L2 | Gibbs-Helmholtz 方程 |
| ΔG = ΔG° + RTlnQ | L3 | van't Hoff 等温式 |
| ΔG° = -RTlnK° | L3 | 标准自由能变与平衡常数关系 |
| ln(K₂/K₁) = -ΔH°/R(1/T₂ - 1/T₁) | L3 | van't Hoff 方程 |
| ln(P₂/P₁) = -ΔvapH°/R(1/T₂ - 1/T₁) | L3 | Clausius-Clapeyron 方程 |

### 重要反应方程式

- **水蒸发**：H₂O(l) → H₂O(g)，ΔvapH°(298K) = +43.98 kJ/mol
- **甲烷燃烧**：CH₄(g) + 2O₂(g) → CO₂(g) + 2H₂O(l)，ΔrH° ≈ -891 kJ/mol
- **甲醇燃烧**：CH₃OH(l) + ³/₂O₂(g) → CO₂(g) + 2H₂O(l)，ΔrH° = -727 kJ/mol
- **HI 分解**：2HI(g) ⇌ H₂(g) + I₂(g)
- **N₂O₄ 分解**：N₂O₄(g) ⇌ 2NO₂(g)，ΔrH° = +57.2 kJ/mol
- **碘升华**：I₂(s) ⇌ I₂(g)，正常升华温度 ≈ 432 K

## 十、踩坑回流

### 1. 本书的字符/公式/图片有什么特有的转换坑？

- **希腊字母乱码**：Δ 常被转为 "δ" 或乱码（如 ΔrH° 写成 δrH°），需人工校正
- **上下标错位**：标准态符号 "°" 常与数字上标混淆（如 ΔrH°m 写成 ΔrH₂°）
- **单位符号混乱**：kJ·mol⁻¹ 中的点乘号有时缺失或变成逗号
- **图片路径失效**：MD 中的图片以相对路径嵌入，Obsidian 中可能无法渲染，需对照 minerU/PDF
- **表格错位**：L3 中多重平衡的表格在 MD 中严重错位，需对照原图理解

### 2. 本书的术语有没有跟其他来源冲突？

- **标准态压强**：质心明确使用 1 bar = 10⁵ Pa，但部分老教材/真题仍用 1 atm = 101.325 kPa，需注意题目上下文
- **键能 vs 键焓**：质心提到教材中称为"键函"，多数文献用"键能"，本质相同
- **STP vs 标准态**：质心反复强调 "气体处于标准" ≠ "气体处于 STP"，这是学生高频混淆点
- **Q 的双重含义**：反应商 Q 与热量 Q 符号相同，质心特别提醒 "不是热(heat)"

### 3. 有没有遇到 mineru 漏掉、需要肉眼补的图？

- L1 中的 P-V 图（功的积分示意图）：MD 中有文字描述但无图，建议从 minerU/PDF 补图
