## 六、关键图表和反应方程式

### 6.1 重要反应方程式

```
# 酸碱滴定
2NaOH + CO2 → Na2CO3 + H2O
Na2CO3 + 2H+ → H2CO3 + 2Na+   (甲基橙终点)
Na2CO3 + H+ → NaHCO3 + Na+    (酚酞终点)

# 络合滴定
M + In → MIn（显色）
MIn + Y → MY + In（变色）

# 氧化还原滴定
MnO4- + 8H+ + 5e → Mn2+ + 4H2O    E°=1.51V（强酸性）
MnO4- + 2H2O + 3e → MnO2 + 4OH-   E°=0.595V（弱酸/弱碱）
MnO4- + e → MnO4^2-               E°=0.558V（强碱性）
Cr2O7^2- + 14H+ + 6e → 2Cr3+ + 7H2O  E°=1.23V
I3- + 2e → 3I-                    E°=0.536V
2Cu2+ + 4I- → 2CuI↓ + I2
I2 + 2S2O3^2- → S4O6^2- + 2I-
2CuI + SCN- → CuSCN + I-（近终点加 KSCN）

# 沉淀滴定
Ag+ + SCN- → AgSCN↓    Ksp=1.1×10^-12
Fe3+ + SCN- → FeSCN2+   K=1.4×10^2（血红色）
```

### 6.2 关键公式

| 公式 | 应用场景 | 来源 |
|:---|:---|:---|
| 四舍六入五成双 | 有效数字修约 | 14 后半 |
| 加减法：以小数点后位数最少为准 | 有效数字运算 | 14 后半 |
| 乘除法：以有效数字位数最少为准 | 有效数字运算 | 14 后半 |
| pH = pKa + lg([盐]/[酸]) | 缓冲溶液计算 | 14 后半 |
