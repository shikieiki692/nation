## 专题 A：硼烷簇合物与 Wade 规则

**目标 KP**：[[硼族元素]] · 新增专节「硼烷簇合物与 Wade 规则」

### A1 · 核心定义块（可直接粘贴）

```markdown
### 硼烷簇合物的分类

硼氢化物按结构分为三大类，由 **Wade 规则** 统一描述：

| 类型 | 通式 | 骨架电子对数 | 结构特征 | 实例 |
|:---|:---|:---:|:---|:---|
| **Closo**（闭合型）| [BₙHₙ]²⁻ | n + 1 | 完整 Δ 多面体，所有顶点均被占据 | [B₆H₆]²⁻（八面体）|
| **Nido**（巢形）| BₙHₙ₊₄ | n + 2 | 缺一个顶点的多面体 | B₅H₉（四方锥）|
| **Arachno**（蛛网形）| BₙHₙ₊₆ | n + 3 | 缺两个顶点的多面体 | B₄H₁₀（蝴蝶形）|

> **Wade 规则计数法**：
> - 每个 **BH 单元** 贡献 **2 个骨架电子**（3 个价电子中 1 个用于端基 B─H 键，剩余 2 个进入骨架）
> - **额外 H**（桥氢或端基多出的 H）各贡献 **1 个电子** 给骨架
> - **负电荷** 直接计入骨架电子总数
> - 骨架电子对数 = 骨架电子总数 ÷ 2
```

### A2 · 骨架电子计数例题块

```markdown
#### 例题：判断 [B₆H₆]²⁻ 的结构类型

**解**：
- 6 个 BH 单元 → 6 × 2 = 12 个骨架电子
- 电荷 2− → 加 2 个电子
- 骨架电子总数 = 14，骨架电子对数 = 7
- n = 6（B 原子数），n + 1 = 7 → **closo 型**
- 结构：正八面体，6 个 B 原子占据 6 个顶点

#### 例题：判断 B₅H₉ 的结构类型

**解**：
- 5 个 BH 单元 → 5 × 2 = 10 个骨架电子
- 额外 4 个 H（9 − 5 = 4）→ 4 × 1 = 4 个电子
- 骨架电子总数 = 14，骨架电子对数 = 7
- n = 5，n + 2 = 7 → **nido 型**
- 结构：缺一个顶点的八面体（四方锥）
```

### A3 · 碳硼烷扩展块

```markdown
### 碳硼烷

**CH 单元**与 **BH 单元**在骨架中等瓣等价（等电子、等轨道），因此 Wade 规则同样适用于碳硼烷。

**例**：C₂B₅H₇
- 总顶点数 n = 2(C) + 5(B) = 7
- 7 个 CH/BH 单元 → 7 × 2 = 14 个骨架电子
- 骨架电子对数 = 7 = n + 0？不对，重新计数：
  - C 贡献 4 个价电子，1 个用于端基 C─H，剩余 3 个 → 但等瓣等价规定 CH 与 BH 一样只贡献 2 个骨架电子
- 骨架电子对数 = 7，n = 7，n + 0？
- **修正**：C₂B₅H₇ 实际为 nido 型？不，参考 Weller 例题 13.8：C₂B₅H₇ 是 **closo 五角双锥**（n=7，n+1=8 对？需再核实）
  
> ⚠️ 实际计算：C₂B₅H₇ 中若按 BH = 2e, CH = 3e（C 多一个电子）
> - 5 BH → 10e；2 CH → 6e；总计 16e → 8 对
> - n = 7，n + 1 = 8 → **closo** ✓
```

### A4 · 易错点

```markdown
> **易错点**：
> 1. Wade 规则中的 n 是**总顶点数**（B + C + 金属原子），不是仅 B 原子数
> 2. BH 单元贡献 2e，但 **CH 单元贡献 3e**（C 比 B 多一个价电子）
> 3. 骨架电子只算用于成簇的电子，端基 B─H 键的电子不计入骨架
```

## 专题 B：硅酸盐结构化学

**目标 KP**：[[二氧化硅结构]] 或 [[硅]] · 新增专节「硅酸盐结构分类」

### B1 · 分类总表（可直接粘贴）

```markdown
### 硅酸盐的结构分类

硅酸盐以 **SiO₄ 四面体**为基本结构单元，通过共享 O 原子数不同形成不同维度的结构：

| 类型 | 共享 O 数 | 实验式 | Si:O 比 | 结构维度 | 典型矿物 |
|:---|:---:|:---|:---:|:---:|:---|
| **岛状** | 0 | [SiO₄]⁴⁻ | 1:4 | 0D 孤立四面体 | 橄榄石 Mg₂SiO₄ |
| **双四面体** | 1 | [Si₂O₇]⁶⁻ | 2:7 | 0D 二聚体 | 硅钙石 Ca₃Si₂O₇ |
| **环状** | 2 | [Si₃O₉]⁶⁻, [Si₆O₁₈]¹²⁻ | 1:3 | 0D 闭合环 | 绿柱石 Be₃Al₂Si₆O₁₈ |
| **单链** | 2 | [SiO₃]²⁻ₙ | 1:3 | 1D 无限链 | 硬玉 NaAl(SiO₃)₂ |
| **双链** | 2–3 | [Si₄O₁₁]⁶⁻ₙ | 4:11 | 1D 带状 | 透闪石 Ca₂Mg₅(Si₄O₁₁)₂(OH)₂ |
| **层状** | 3 | [Si₂O₅]²⁻ₙ | 2:5 | 2D 平面层 | 滑石、云母 |
| **架状** | 4 | SiO₂ | 1:2 | 3D 网络 | 石英、长石、沸石 |

> **判定口诀**：Si:O = 1:4 → 岛状；1:3 → 链/环；2:5 → 层状；1:2 → 架状
```

### B2 · 例题模板

```markdown
#### 例题：根据化学式判断硅酸盐结构类型

**(1) Mg₂SiO₄** —— Si:O = 1:4 → **岛状硅酸盐**

**(2) Be₃Al₂Si₆O₁₈** —— Si:O = 6:18 = 1:3，且 O 数恰好为 3 的整数倍 → **六元环状硅酸盐**
- 6 个 SiO₄ 四面体，每个共享 2 个 O，形成 [Si₆O₁₈]¹²⁻ 环
- Be²⁺ 和 Al³⁺ 填充在环间平衡电荷

**(3) Ca₂Mg₅(Si₄O₁₁)₂(OH)₂** —— Si:O = 8:22 = 4:11 → **双链状硅酸盐**（闪石类）
```

## 专题 C：磷的含氧酸体系

**目标 KP**：[[磷]] 或 [[氮族元素]] · 新增专节「磷的含氧酸」

### C1 · 含氧酸全表（可直接粘贴）

```markdown
### 磷的含氧酸体系

| 名称 | 化学式 | P 氧化态 | 结构特征 | 酸强度 | 氧化还原性 |
|:---|:---|:---:|:---|:---|:---|
| **次磷酸** | H₃PO₂ / H₂PO₂⁻ | +1 | 四面体，**2 个 P─H 键** | 中强一元酸 | **强还原剂** |
| **亚磷酸** | H₃PO₃ / HPO₃²⁻ | +3 | 四面体，**1 个 P─H 键** | 中强二元酸 | **强还原剂** |
| **连二磷酸** | H₄P₂O₆ / P₂O₆⁴⁻ | +4 | P─P 键，无 P─H | 四元酸 | 稳定 |
| **（正）磷酸** | H₃PO₄ / PO₄³⁻ | +5 | 正四面体，无 P─H | 中强三元酸 | 稳定（热力学）|
| **焦磷酸** | H₄P₂O₇ / P₂O₇⁴⁻ | +5 | 2 个 PO₄ 共顶点 | 四元酸 | 稳定 |
| **偏磷酸** | (HPO₃)ₙ | +5 | 环状或链状 | 强酸 | 稳定 |

> **记忆法**：氧化态 +1/+3 有 P─H → 还原性；+5 无 P─H → 热力学稳定
```

### C2 · 缩聚磷酸盐块

```markdown
### 缩聚磷酸盐

磷酸加热脱水形成 **P─O─P 桥**：

- **链状缩聚**：
  - 焦磷酸根 P₂O₇⁴⁻（2 单元）
  - 三聚磷酸根 P₃O₁₀⁵⁻（3 单元）
  - 聚磷酸根 [PO₃]ₙⁿ⁻（n 单元，玻璃态）

- **环状缩聚**：
  - 三偏磷酸根 P₃O₉³⁻（3 元环）
  - 四偏磷酸根 P₄O₁₂⁴⁻（4 元环）

- **滴定特性**：端基 P─OH（弱酸性）vs 桥 P─O─P（无酸性）→ 可由酸碱滴定测定平均链长

- **磷氮烯**：(NPCl₂)ₙ，无机橡胶，P=N 交替骨架，耐高温弹性体
```

### C3 · Latimer 图与 Frost 图块

```markdown
### 磷的氧化还原化学

**碱性溶液中的 Latimer 图**：
```
P₄ ──(−1.12V)──> PH₃ ──(−0.89V)──> P ──(−1.57V)──> H₂PO₂⁻ ──(−2.05V)──> HPO₃²⁻ ──(−1.56V)──> PO₄³⁻
```

**关键结论**：
- 白磷 P₄ 在碱性溶液中**歧化**：P₄ + 3 OH⁻ + 3 H₂O → PH₃ + 3 H₂PO₂⁻
- 低氧化态（+1, +3）在碱性中稳定，但在酸性中易被氧化到 +5
- 磷的含氧酸氧化还原**动力学极慢**，热力学趋势与实际反应性常不一致
```

## 专题 D：硫的含氧酸与多硫化学

**目标 KP**：[[氧族元素]] · 新增专节「硫的含氧酸」

### D1 · 含氧酸全表（可直接粘贴）

```markdown
### 硫的含氧酸体系

| 名称 | 化学式 | S 平均氧化态 | 结构特征 | 氧化还原性 |
|:---|:---|:---:|:---|:---|
| **硫代硫酸根** | S₂O₃²⁻ | +2 | S─S 键，四面体 | 中等还原剂 |
| **连二亚硫酸根** | S₂O₄²⁻ | +3 | S─S 键，椅式 | **强还原剂**（保险粉）|
| **亚硫酸根** | SO₃²⁻ | +4 | 三角锥 | 还原剂 |
| **连二硫酸根** | S₂O₆²⁻ | +5 | S─S 键，两个 SO₃ 共面 | **惰性**（不氧化不还原）|
| **硫酸根** | SO₄²⁻ | +6 | 正四面体 | 热力学稳定，动力学惰性 |
| **过氧二硫酸根** | S₂O₈²⁻ | +6 | O₃SO─OSO₃，含 O─O 键 | **极强氧化剂**（E° = +1.96 V）|
| **连多硫酸根** | SₙO₆²⁻ (n=3–6) | 可变 | Sₙ 链，两端各连 SO₃ | — |

> **核心反应**：
> - S₂O₃²⁻ + 2 Ag⁺ + H₂O → Ag₂S↓ + SO₄²⁻ + 2 H⁺（摄影定影）
> - S₂O₃²⁻ + 4 Cl₂ + 5 H₂O → 2 SO₄²⁻ + 8 Cl⁻ + 10 H⁺（除氯）
> - S₂O₈²⁻ + 2 e⁻ → 2 SO₄²⁻（过硫酸铵作引发剂）
```

### D2 · 硫代硫酸根氧化态解析

```markdown
> **教学要点**：硫代硫酸根 S₂O₃²⁻ 中两个 S 的氧化态**不同**
>
> - 中心 S（类似硫酸根中的 S）：**+5**
> - 端基 S（取代氧的位置）：**−1**
> - 平均氧化态 = (+5 + (−1)) / 2 = **+2**
>
> 与 Ag⁺ 反应时，端基 S 被氧化/沉淀为 Ag₂S，中心 S 被氧化为 SO₄²⁻
```

### D3 · 多硫与硫氮化合物

```markdown
### 多硫离子与硫氮化合物

- **多硫化物**：Sₙ²⁻（n = 2–6），链状结构，颜色随 n 增大加深（黄→橙→红）
  - 制备：Na₂S + (n−1) S → Na₂Sₙ
  - 酸化分解：Sₙ²⁻ + 2 H⁺ → H₂S↑ + (n−1) S↓

- **硫的聚阳离子**：
  - S₄²⁺：正方形平面（Se₄²⁺ 同构）
  - S₈²⁺：椅式八元环扭曲
  - 仅在超强酸介质中稳定（如 HSO₃F / SbF₅）

- **硫氮化合物**：
  - S₄N₄：摇篮形，8π 电子体系，**假芳香性**
  - (SN)ₓ：导电聚合物，金属光泽，低温超导转变
```

## 专题 E：卤素互化物与卤素含氧酸

**目标 KP**：[[卤素]] · 新增专节「卤素互化物」和「卤素含氧酸」

### E1 · 卤素互化物总表（可直接粘贴）

```markdown
### 卤素互化物

中心原子为电负性较小、原子半径较大的卤素，配位原子为 F（最常见）或 Cl。

| 类型 | 通式 | 实例 | 中心原子氧化态 | VSEPR 结构 |
|:---|:---|:---|:---:|:---|
| **一卤化物** | XY | ClF, BrF, ICl, IBr, BrCl | +1 | 直线形 |
| **三卤化物** | XY₃ | ClF₃, BrF₃, IF₃ | +3 | T 形（5 对电子，2 对孤对）|
| **五卤化物** | XY₅ | ClF₅, BrF₅, IF₅ | +5 | 四方锥（6 对电子，1 对孤对）|
| **七卤化物** | XY₇ | IF₇ | +7 | 五角双锥（7 对电子，无孤对）|

> **重要性质**：
> - ClF₃、BrF₃：强氟化剂，可在室温下腐蚀玻璃
> - IF₅：温和氟化剂，可在玻璃装置中使用
> - IF₇：极强的氟化剂，但热稳定性差
> - **¹⁹F-NMR**：不同化学环境的 F 产生不同信号，可用来验证结构
>   - ClF₃（T 形，C₂ᵥ）：2 种 F 信号（轴向 vs 赤道），积分比 2:1
>   - IF₇（五角双锥，D₅ₕ）：2 种 F 信号（轴向 vs 赤道），积分比 2:5
```

### E2 · 卤素含氧酸 Frost 图块

```markdown
### 卤素含氧酸的氧化还原趋势

**酸性溶液中的 Frost 图特征**：

- **Cl**：ClO₄⁻ → ClO₃⁻ → ClO₂ → HClO → Cl₂ → Cl⁻
  - HClO₂ 位于 ClO₃⁻ 和 HClO 连线**上方** → **歧化**：2 HClO₂ → ClO₃⁻ + HClO + H⁺
  - ClO₄⁻/ClO₃⁻ 电位很高（+1.20 V），但动力学极慢（需 O 原子转移）

- **pH 影响**：碱性中氧化性大幅减弱
  - ClO₄⁻/ClO₃⁻：+1.20 V（酸性）→ +0.37 V（碱性）
  - BrO₄⁻/BrO₃⁻：+1.85 V（酸性）→ +0.93 V（碱性）

> **教学要点**：
> 1. Frost 图中某点位于相邻两点连线**上方** → 该物种**不稳定，会歧化**
> 2. Frost 图中某点位于连线**下方** → 该物种**稳定，相邻物种会归中**
> 3. 酸性中卤素含氧酸多为强氧化剂；碱性中氧化性减弱，高氧化态更稳定
```

## 专题 F：对角线关系与惰性电子对效应

**目标 KP**：[[对角线规则]]、[[惰性对效应]] · 增量补充

### F1 · 对角线关系块

```markdown
### 对角线关系（第 2/3 周期）

| 关系对 | 相似性表现 |
|:---|:---|
| **Li ~ Mg** | ① 在空气中燃烧生成普通氧化物（Li₂O、MgO），不形成过氧化物 ② 与 N₂ 反应生成氮化物（Li₃N、Mg₃N₂）③ 碳化物水解生成甲烷？不，MgC₂ 不太稳定；但都与有机酸反应生成弱酸盐 |
| **Be ~ Al** | ① 两性氧化物和氢氧化物（BeO/Be(OH)₂、Al₂O₃/Al(OH)₃）② 形成共价性卤化物（BeCl₂ 链状聚合物、AlCl₃ 二聚体）③ 冷浓 HNO₃ 中钝化 |
| **B ~ Si** | ① 形成挥发性氢化物（B₂H₆、SiH₄）② 含氧酸为弱酸（H₃BO₃、H₂SiO₃/H₄SiO₄）③ 卤化物易水解（BCl₃ + 3 H₂O → H₃BO₃ + 3 HCl；SiCl₄ + 2 H₂O → SiO₂ + 4 HCl）④ 氧化物为玻璃态网络结构 |

> **成因**：离子势 φ = Z/r 相近 → 电荷密度相近 → 化学行为相似
```

### F2 · 惰性电子对效应块

```markdown
### 惰性电子对效应（Inert Pair Effect）

ns² 电子不易参与成键，导致低氧化态稳定性增强：

| 元素 | 常见氧化态 | 现象 |
|:---|:---|:---|
| **Tl** | Tl⁺ > Tl³⁺ | Tl⁺ 离子半径与 K⁺ 相近，可跨膜替代 K⁺ → **剧毒** |
| **Pb** | Pb²⁺ > Pb⁴⁺ | PbO 蓝黑/红色（立体化学活性 6s² 孤对电子）；Pb₃O₄ = 2 PbO·PbO₂（混合价）；PbO₂ 强氧化剂 |
| **Bi** | Bi³⁺ > Bi⁵⁺ | Bi(V) 为极强氧化剂，NaBiO₃ 可氧化 Mn²⁺ → MnO₄⁻（酸性条件）|

> **解释框架**：
> - 相对论效应使 6s 轨道收缩、能量降低 → 6s² 电子更难失去
> - Tl、Pb、Bi 的 **6s² 电子对** 表现为"惰性"
> - 但 Bi(V) 并非完全不存在，只是极强的氧化剂
```
