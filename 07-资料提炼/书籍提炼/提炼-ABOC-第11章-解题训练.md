## 五、各章精要

### 5.1 FINAL TEST 0 — [Jack] Mutilin 全合成（Luo, 2022）

**背景**：Mutilin 是双萜类真菌代谢产物，具有独特的 5-6-8 稠合骨架和多个相邻手性中心。北京大学罗佗平课题组 2022 年完成不对称全合成。

**核心考查点**：
- **逆合成分析**：De Mayo 反应切断、FGI（羟基→羰基）、烯丙基/甲基切断
- **联烯化学**：TMS-EBX 试剂构建联烯、加料次序设计
- **[3,3]-σ 重排**：构建稠环骨架的关键步骤
- **[2+2] 光环加成**：光照环化，过渡态立体选择性解释
- **差向异构体控制**：水解-脱羧-再环化的策略调整
- **动力学同位素效应（KIE）**：重水标记解决副反应问题
- **羟醛缩合过渡态**：手性中心选择性成因

**涉及 KP**：[[De Mayo反应]]、[[联烯化学]]、[[[3,3]-σ迁移]]、[[[2+2]环加成]]、[[Aldol缩合]]、[[动力学同位素效应]]

### 5.2 FINAL TEST 0 — [Tech] Jujuyane 全合成（Hee-Yoon Lee, 2021）

**背景**：Jujuyane 的全合成分析，以写产物为主的考查方式。

**核心考查点**：
- **呋喃衍生物合成**：从呋喃出发构建复杂骨架
- **D-A 环加成**：36 届初赛级别的环加成应用
- **产物预测**：多步转化的中间体推断

**涉及 KP**：[[Diels-Alder反应]]、[[呋喃化学]]、[[杂环合成]]

### 5.3 FINAL TEST 0 — [Speed] Taxol 全合成

**背景**：紫杉醇（Taxol）的全合成分析，出自第一届 ArCHO 联考。考查写产物和选择性控制。

**核心考查点**：
- **端炔化学**：炔基负离子的亲核加成
- **烯烃复分解（RCM）**：关环复分解构建环系
- **官能团转化**：保护基策略、氧化态调整
- **选择性控制**：化学选择性、区域选择性、立体选择性

**涉及 KP**：[[炔烃]]、[[烯烃复分解反应]]、[[保护基策略]]、[[选择性控制]]

### 5.4 FINAL TEST 0 — [Stream] 光化学与自由基反应

**背景**：改编自第三届和第七届 ArCHO 联考，考查光氧化还原催化和自由基反应。

**核心考查点**：
- **光氧化还原催化机理**：Ru/Ir 催化剂的 SET 循环、淬灭方式判断
- **自由基加成**：CO₂ 自由基负离子对双键的加成
- **Fe 催化的氢自由基转移**：FeH(acac)₂ 的原位制备与应用
- **反应条件替换的影响**：试剂替换后的产物预测

**涉及 KP**：[[光氧化还原催化]]、[[自由基]]、[[有机光化学]]、[[Giese反应]]

### 5.5 FINAL TEST 1 — PART 1：Organic Mechanisms（机理书写）

**5 道题的考点分布**：

| 题号 | 核心机理 | 涉及知识点 |
|:---:|:---|:---|
| EX.1 | CO₂ 自由基负离子催化循环 | 自由基链式反应、甲酸根氧化、硫醇引发 |
| EX.2 | Cu 催化烯醇化 + 自由基离去 | 烯醇化学、Cu(I)/Cu(II) 循环、自由基引发 |
| EX.3 | 双自由基关环 | 双自由基化学、轨道重叠、关环选择性 |
| EX.4 | [3,3]-重排 → D-A 反应 | Cope/Claisen 重排、Diels-Alder、热 vs 光环加成区分 |
| EX.5 | 重排关环（E-环己烯去质子）| 碱促进的重排、环化机理 |

> **关键辨析**：EX.4 中"不能发生 2+2 环加成"——热条件下 [2+2] 禁阻，必须经历 [3,3] 重排后再 D-A。

### 5.6 FINAL TEST 1 — PART 2：Organic Methodology（有机方法学）

**5 道题的考点分布**：

| 题号 | 主题 | 核心考查 |
|:---:|:---|:---|
| EX.1 | 分子间 vs 分子内反应 | 氢键导向的分子内反应、吡啶与苯环的反应性差异 |
| EX.2 | Rocaglate 合成中的碱效应 | Et₃N vs NaH 导致不同产物、碳正离子重排、开环反应 |
| EX.3 | 34 届决赛题（锂试剂 + D-A + 三组分反应）| LiTMP 去质子、[4+2] 环加成、硫杂环丁烷捕获、多组分反应 |
| EX.4 | 吡啶官能团化（"坑人警告"）| 吡啶 SNAr、PADI 催化机理、TBS⁺ 转移活化策略 |
| EX.5 | β-内酰胺不对称合成 | 立体选择性控制、过渡态分析、一锅串联反应 |

> **EX.4 的"坑人"设计**：PADI 作为质子酸与吡啶/烯亚胺反应，产生 N⁺ 中间体，TBS⁺ 转移活化吡啶——**进攻 TBS 而非看起来亲电性最强的烯亚胺**。

### 5.7 FINAL TEST 1 — PART 3：Organic Synthesis（Nodulisporic Acid D）

**背景**：COBP 模拟题经典名题，具有强烈杀虫活性的吲哚类化合物全合成。

**6 小问考点**：
1. **片段 1 合成**：还原碘代 → 不对称 SN2 → 臭氧氧化腙 → Tf 化 → Fe 单电子还原硝基 → ICl 碘代（对位选择性）
2. **片段 2 合成**：Cu 碳负离子 1,4-加成 → 甲基锂释放氧负离子 → 格氏加成 → RCM 关环 → 氢化还原 → LHMDS+PhNTf₂ → Pd 催化交换
3. **副产物推断**：Me₄Si（甲基锂进攻 TMS-O 反键轨道）
4. **过渡态分析**：逆 Aldol 缩合的协同过渡态（分子内氢键）
5. **溶剂效应**：DCM/H₂O 破坏分子内氢键 → 避免异构化
6. **偶联与关环**：Pd 催化 Br/NH₂ 偶联 → 分子内关环形成吲哚

**涉及 KP**：[[不对称合成]]、[[烯烃复分解反应]]、[[吲哚合成]]、[[臭氧氧化]]、[[单电子转移]]

## 六、关键创新点

### 6.1 ABOC 独特视角

1. **"原创命题 vs 汇编命题"双轨设计**：FT0 是作者两年原创命题的结晶（"集万千反应模式，自始至终未有重复"），FT1 是机构优质题的精选汇编——两套题风格互补。

2. **"选择性"作为全合成的灵魂**：FT0 的 [Jack] 以选择性控制为核心考点（TIPSOTf/MeI 的区域和立体选择性、羟醛缩合过渡态的选择性），体现了现代全合成化学的核心挑战。

3. **"坑人设计"是竞赛命题的常态**：FT1 EX.4 的 PADI 机理、EX.2 的碱效应差异——ABOC 通过真实竞赛题告诉学生："看起来最合理的进攻位点往往不是正确答案"。

4. **"逆推比顺推更快"**：FT1 PART 3 第 4 问明确提示——全合成推断题中，从已知中间体逆推比从原料顺推更有效。

## 十一、教学使用建议

### 11.1 学习路径

- **完成本章前，需要先掌握**：
  - Ch.1-10 全部核心 KP（尤其是 [[Diels-Alder反应]]、[[Aldol缩合]]、[[[3,3]-σ迁移]]、[[光氧化还原催化]]、[[自由基]]）
  - [[立体化学]]（CIP 规则、Fischer 投影、NOE）
  - [[反应机理表示法]]（箭头书写规范）
- **使用建议**：
  1. **FT0 作为"挑战题"**：不追求一次全对，重点是体会"现代全合成的选择性控制思维"
  2. **FT1 作为"模拟考"**：限时完成，训练时间管理和应试心态
  3. **FT0 解析视频 + FT1 交流群**：主动寻求反馈，不要闭门造车

### 11.2 教学重点

1. **"逆合成分析是解题的钥匙"**：FT0 每道题都给出逆合成切断策略，训练学生从产物倒推原料的能力。

2. **"立体化学不是附加题，是必答题"**：FT0 明确要求"要求立体化学"——意味着每写一个中间体都要考虑 R/S、E/Z、syn/anti。

3. **"坑人题的训练价值"**：FT1 EX.4 的 PADI 机理、EX.2 的碱效应——教会学生"质疑第一反应"，竞赛命题人常用"看起来最合理"的陷阱。

4. **"过渡态分析是解释选择性的唯一语言"**：FT0-Jack 的 J-1-4（[2+2] 过渡态）、FT1-EX.5（D→E 的过渡态）——能用过渡态画出选择性，才是真正的理解。

### 12.1 学生常见误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析 |
|:---|:---|:---|:---|
| "FT0 必须全对才算学会" | 追求满分 | FT0 是**挑战题**，做对 50% 已经很优秀 | 重点在于**思维过程**而非得分 |

| "全合成题必须从原料顺推" | 习惯正向思维 | **逆推往往更快**——从已知中间体或目标产物倒推 | FT1 PART 3 明确提示逆推 |

### 12.2 竞赛陷阱

1. **T-吡啶 SNAr 陷阱**：吡啶的 SEAr 困难、SNAr 优先在 2,4-位——FT1 EX.4 的"坑人"设计正是利用学生"看到芳环就 SEAr"的惯性思维。

2. **T-碱的选择陷阱**：Et₃N（弱碱，可逆）vs NaH（强碱，不可逆）——同一底物在不同碱条件下可能走完全不同的机理路径（FT1 EX.2）。

3. **T-分子内 vs 分子间陷阱**：苯环体系倾向于分子间反应，吡啶体系因氢键可拉成分子内——FT1 EX.1 的考查核心。

4. **T-[3,3] 重排后接 D-A 的串联陷阱**：学生可能直接写 [2+2] 光环加成，但热条件下 [2+2] 禁阻，必须先 [3,3] 重排得到适合 D-A 的底物（FT1 EX.4）。
