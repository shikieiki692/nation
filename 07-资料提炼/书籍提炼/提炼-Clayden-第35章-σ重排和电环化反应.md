# 提炼：Clayden 第35章 — σ 重排和电环化反应

---

## 一、核心概念与定义

### 1.1 周环反应的三大类型（区分标准）

| 类型 | σ键变化 | 特征 |
|:---|:---|:---|
| **环加成** | Δσ = +2（形成两根新σ键）| 两个组分 |
| **σ重排** | Δσ = 0（一根断裂，一根形成）| 一个组分内部 |
| **电环化** | Δσ = ±1（一根新σ键形成或断裂）| 一个共轭π体系 |

> 电环化反应：新σ键跨一个共轭多烯的末端形成的过程及其逆过程。（p. 916, line 2659）

### 1.2 σ重排（Sigmatropic Rearrangement）

**定义**：σ键看起来像是从一处地方移动到了另一处地方的协同周环反应。箭头中有一根开始于σ键而非π键。

**命名规则**：给旧σ键的两端都标为"1"，然后绕着两根方向，数到产物中新σ键的两端。例如新σ键两端的序号均为"3" → [3,3]-σ重排。

**过渡态环大小规律**：方括号中两个数字之和 = 过渡态环大小。如 [3,3] → 六元环；[2,3] → 五元环；[1,5] → 六元环。

### 1.3 电环化反应（Electrocyclic Reaction）

**定义**：在一个单独的共轭π体系中，一根新σ键的形成（关环）或断裂（开环）的过程。

**关键特征**：平面多烯要变为一个环（或反之），所需的旋转接近90°——这是电环化与其他周环反应的关键区别。

### 1.4 顺旋（Conrotatory）与对旋（Disrotatory）

| 术语 | 定义 |
|:---|:---|
| **顺旋 conrotatory** | 两个基团向**相同**方向旋转：都顺时针或都逆时针 |
| **对旋 disrotatory** | 一个基团顺时针旋转，一个基团**逆时针**旋转 |

## 二、重要反应与机理

### 2.1 [3,3]-σ重排

#### 2.1.1 芳香族 Claisen 重排（原始发现）

- **底物**：芳基烯丙基醚
- **条件**：无溶剂，加热
- **第一步**：[3,3]-σ重排 → 邻烯丙基环己二烯酮（非芳香性）
- **第二步**：简单离子型质子转移 → 恢复芳香性 → 邻烯丙基酚
- **不对称烯丙基醚**：重排时会"从里面翻出来"（立体专一性）

#### 2.1.2 脂肪族 Claisen 重排（Claisen-Cope 重排）

- **底物**：烯丙基乙烯基醚
- **产物**：γ,δ-不饱和羰基化合物
- **驱动力**：羰基的稳定性（C=O + C-C σ键 比 C=C π键 + C-O σ键更稳定）

**起始原料制备**（缩醛交换法）：
- 醛的缩醛与烯丙基醇的酸催化交换（丙酸作催化剂，高沸点）
- 第一分子甲醇由SN1失去，第二分子由E1失去
- 烯丙基乙烯基醚形成后立即重排

![[mineru/中文版clayden-chinese-34-36907-999_images/bcd53e3e8a2f2c8b8ac7c1887b3541d0d83997f5232a7a86906b6911299c26bb.jpg]]
![[mineru/中文版clayden-chinese-34-36907-999_images/7cf3087a7d17031f53438293d20de9ccf336392ced665e60efc1a543ec37c778.jpg]]

**应用**：合成γ,δ-不饱和醛、酮、酯、酰胺的常规方法。

#### 2.1.3 Cope 重排

- **定义**：环中仅涉及碳原子的 [3,3]-σ重排
- **最简单版本**：起始原料和产物相同（对称1,5-二烯）
- **驱动力策略**：
  - 通过羰基形成定向：3-羟基-1,5-二烯 → 烯醇 → 羰基化合物
  - 通过阴离子加速：烷氧基阴离子（KH处理）→ 产物烯醇钾更稳定
  - 通过张力释放：三元环中弱σ键断裂
  - 通过双键稳定性：末端双键→三取代双键

**Oxy-Cope / 阴离子加速 Cope**：
- 3-羟基-1,5-二烯用碱（最好是KH）处理得到烷氧基阴离子
- 反应被大幅加速（阴离子加速）
- 产物是烯醇钾，比烷氧基钾起始原料更稳定
- 可制得桥头烯烃（Bredt规则禁止的通常情况）

#### 2.1.4 Ireland-Claisen 重排

- **发明者**：R. E. Ireland，1970s
- **底物**：烯丙醇的酯 → 烯醇锂（LDA处理）
- **可选**：烯醇锂转化为烯醇硅醚（OSiMe3）后加热重排
- **产物**：不饱和酸（后处理后）
- **E选择性**：与普通Claisen相同的原因，双键强烈偏好反式(E)几何结构

#### 2.1.5 Fischer 吲哚合成中的 [3,3]-σ重排

- 苯肼 + 酮 → 苯腙 → 烯胺（可烯醇化的酮）
- 烯胺具有 [3,3]-σ重排的理想结构
- 断裂的σ键是弱的N-N σ键，一根π键在苯环内
- 产物：高度不稳定的双键亚胺 → 芳香性恢复 → 吲哚

#### 2.1.6 Cr(VI) 氧化中的共轭氧化（Conjugate Oxidation）

- 叔烯丙基醇 + CrO3/酸性溶液
- 第一步：铬酸酯形成
- 第二步：铬酸酯基通过 [3,3]-σ重排转移到烯丙基末端
- 两个氧和一个铬原子参与的 [3,3]-σ重排

### 2.2 [2,3]-σ重排

#### 2.2.1 基本概念

- 过渡态是**五元环**（2+3=5）
- 需要绕五元环行进的箭头，必须停止或开始于某一个原子（而非键）
- 该原子可以是：碳阴离子、碳阳离子、或愿意转换氧化态的杂原子

#### 2.2.2 Wittig 型 [2,3]-σ重排

- **底物**：苄基烯丙基醚 → 去质子化得碳阴离子
- **产物**：新C-C σ键 + 氧阴离子
- **平衡偏移**：氧阴离子比碳阴离子稳定得多
- **立体化学**：类椅式过渡态，新π键反式形成，R取代基倾向平伏位点

![[mineru/中文版clayden-chinese-34-36907-999_images/b39083f56a2976819ce147453966082c755801529782d63255a259f30b963eaa.jpg]]
![[mineru/中文版clayden-chinese-34-36907-999_images/bf975d13d35a4d4268462804e7abbfdd8ffd79a0ddb13a2b4f32f2ab03891c4b.jpg]]

#### 2.2.3 硫/硒参与的 [2,3]-σ重排

**烯丙基亚砜的制备（Mislow-Evans重排）**：
- 烯丙型醇 + PhSCl → 次磺酸酯（不稳定）
- 加热 → [2,3]-σ重排 → 烯丙基亚砜
- 箭头开始并结束于硫原子，S(II) → S(IV)
- **逆向重排**：亚砜 + (MeO)3P（亲硫亲核试剂）/ MeOH / 加热 → 逆向[2,3]-σ重排 → 次磺酸酯被捕获 → 另一个烯丙基醇
- **净结果**：烯丙型醇的"烷基化"（通常被认为不可能完成的转化）

![[mineru/中文版clayden-chinese-34-36907-999_images/af43e9dff2bb2f523e453f0a8f34def28bc89a9315315e593df294900abbf06b.jpg]]
![[mineru/中文版clayden-chinese-34-36907-999_images/f63fb2b1e7e160ce819ac3272edae335ab3a22bd7f0244fb9e56674332bb4960.jpg]]

**SeO2 氧化烯烃（Riley氧化）**：
- SeO2 + 烯烃 → [4+2]环加成 → 烯丙基亚硒酸
- [2,3]-σ重排 → 不稳定化合物 → 分解为烯丙醇
- 对甲基氧化 → 进一步氧化为醛/酮（烯基醛 enals）
- 催化量SeO2 + t-BuOOH（后续氧化剂）

### 2.3 [1,5]-σ氢移位

#### 2.3.1 基本概念

- 旧σ键和新σ键连在同一个原子上（氢原子）
- 过渡态是**六元环**（1+5=6）
- 氢原子"绕一个共轭体系的迁移"

#### 2.3.2 环戊二烯中的快速 [1,5]H 移位

- 环戊二烯基阴离子烷基化后 → 取代环戊二烯
- [1,5]H 移位非常迅速 → 三种环戊二烯异构体的混合物
- **后果**：Diels-Alder反应通常得到三种加合物的混合物
- **解决方案**：增强双烯活性（如与酯基共轭）→ Diels-Alder反应快于[1,5]H移位

![[mineru/中文版clayden-chinese-34-36907-999_images/1f48f760f3ebf915d5e1c54e8cb1be598ba0f8f3d09b505582d9631477e79f1f.jpg]]
![[mineru/中文版clayden-chinese-34-36907-999_images/05718182f8ac9a1f5f9b452d0951c916eef7f32a4f2f8725a7c9ea06991cb579.jpg]]

#### 2.3.3 前线轨道描述

- 双烯 LUMO (ψ3) + C-H键 HOMO（充满的σ轨道）
- 氢原子1s轨道是球形对称的
- **同面迁移**：氢从π体系的一面离开，从同一面再次加入

### 2.4 电环化反应

#### 2.4.1 基本例子

| 反应 | π电子数 | 条件 | 旋转方式 | 说明 |
|:---|:---:|:---|:---:|:---|
| 己三烯 → 环己二烯 | 6π | 热 (~500°C) | 对旋 | 驱动力：σ键比π键强 |
| 环丁烯 → 1,3-丁二烯 | 4π | 热 | 顺旋 | 驱动力：环张力释放 |
| 环庚三烯 ↔ 降替二烯 | 6π | 热 | — | 通常环庚三烯占优；特定取代模式下降替二烯有利 |

#### 2.4.2 Nazarov 环化

- **底物**：双α,β-不饱和酮
- **条件**：酸催化
- **机理**：
  1. 酮质子化 → 共轭二烯基阳离子（4π体系，5条p轨道，其中一条空置）
  2. 4π电环化关环（**顺旋**）→ 环戊烯基阳离子
  3. 质子失去 + 异构化 → 环戊烯酮
- **光化学条件**：对旋（与热反应相反）

![[mineru/中文版clayden-chinese-34-36907-999_images/363c4ea88b7507d0b4fee43d5196efc6a34b4de2b6c219ddcd8eef83dd3e1958.jpg]]
![[mineru/中文版clayden-chinese-34-36907-999_images/660535b7abfa6475f68e83465a91de18dfc3dfd68f9856fd49fb2401606e2835.jpg]]

#### 2.4.3 环辛二烯基阴离子的电环化

- 环辛二烯 + BuLi → 环辛二烯基阴离子
- 5条p轨道，6个π电子 → **对旋**关环
- 顺旋光化学环化被环张力阻止（5,5反式稠合体系不可能）
- 热电环化关环也被环张力阻止

![[mineru/中文版clayden-chinese-34-36907-999_images/85ca891046c548cf16e2e3c5299e1d0d59e21e66dc77e0e471ce97ad7cce7362.jpg]]

#### 2.4.4 小环的电环化开环

**环丙基阳离子 → 烯丙基阳离子**：
- 烯丙基阳离子（2π）热对旋关环 → 环丙基阳离子
- 但**实际上环丙基阳离子非常容易开环**为烯丙基阳离子
- 环张力促进电环化开环
- 尝试环丙烷环上的亲核取代 → 环丙基阳离子中间体迅速开环

**氮丙环（Aziridine）开环**：
- 带吸电子基团的氮丙环不稳定
- 电环化开环 → 甲亚铵叶立德（azomethine ylid）
- 叶立德可被亲偶极体通过[3+2]环加成捕捉
- 产物立体化学证实开环是**顺旋**的（叶立德是4π电子体系）

### 2.5 维生素D生物合成中的电环化

- **第一步（需光照）**：7-脱氢胆固醇 → 光化学电环化开环 → 预钙化醇（previtamin D3）
  - 6π体系，光化学 → **顺旋**开环
  - 对旋热6π环化会在六元环中形成不可能的反式双键
- **第二步（无需光照）**：预钙化醇 → [1,7]H移位（异面）→ 维生素D3
  - 三烯体系足够灵活使氢从顶面迁移到底面

## 三、关键结论与规则

### 3.1 电环化反应规则（核心）

> **所有电环化反应都是允许的。**
> - 涉及 **(4n+2)** 个π电子的热电环化反应是**对旋**的。
> - 涉及 **(4n)** 个π电子的热电环化反应是**顺旋**的。
> - 在顺旋反应中，两个基团向**相同**方向旋转。
> - 在对旋反应中，一个基团顺时针，一个基团逆时针。
> - **光化学反应中，顺旋/对旋规则颠倒。**

### 3.2 氢的σ迁移总结（热反应）

| 迁移类型 | 立体化学 | 可行性 | 原因 |
|:---|:---|:---|:---|
| [1,3]H 移位 | 异面 | **不可行** | 刚性三碳链太短，H无法从顶面到底面 |
| [1,5]H 移位 | 同面 | **容易** | 六元环过渡态，六电子 |
| [1,7]H 移位 | 异面 | **可行** | 链足够灵活 |

### 3.3 [3,3]-σ重排的立体化学规则

- **类椅式过渡态**优先
- 取代基倾向于**平伏位置**
- 新双键强烈偏好**反式(E)**几何结构
- 手性信息从起始原料**立体专一**地转移到产物

### 3.4 σ键的s/a对称性判定

对于σ键：
- **同面 σ2s**：两端都保持（retention）或两端都翻转（inversion）
- **异面 σ2a**：一端保持，另一端翻转

### 3.5 ω组分（用于[2,3]-σ重排）

- 碳阴离子：ω2组分（2个电子）
- 碳阳离子：ω0组分（0个电子）
- 同面：ω2s；异面：ω2a

### 3.6 允许的电环化反应会被空间因素阻止

> 虽然所有电环化反应在理论上都是允许的，但顺旋/对旋的空间需求可能导致其中一种模式不可行。（p. 922）

## 八、摘录的原始段落（附页码/位置标记）

### 8.1 σ重排的基本定义（line ~1803-1807）

> "环加成，上一章的主题，只是周环反应三种主要类别中的一类。在本章中，我们将考虑另两类：σ重排(sigmatropic rearrangements)和电环化反应(electrocyclic reactions)。我们会用类似环加成的方式分析它们。"

### 8.2 Claisen重排的原始发现（line ~1809-1831）

> "原始的σ重排反应，发生于在无溶剂的情况下加热芳基烯丙基醚的过程，所得的产物是邻烯丙基酚。这就是Claisen重排反应(rearrangement)。这个反应的第一步是一个您将要学着称其为[3,3]-σ重排的周环反应。"
> 
> "这是一个没有离子型中间体及任何电荷的一步机理，类似于环加成。箭头绕一个环移动。它和环加成的区别在于，它的箭头中有一根开始于σ键而非π键。"

### 8.3 [3,3]-σ重排的命名（line ~1857）

> "这些反应被称为'σ向的(sigmatropic)'，是因为反应过程中，一根σ键看起来像是从一处地方移动到了另一处地方。刚才的这一种被称作[3,3]-σ重排，因为新σ键与旧σ键处于3,3关系。给旧σ键的两端都标为'1'，然后绕着两根方向，数到产物中新σ键的两端。您会发现新σ键的两端的序号均为'3'。"

### 8.4 椅式过渡态与立体化学（line ~1867-1891）

> "这些[3,3]-σ重排都通过一个类椅式过渡态发生，这会使我们知晓正确的轨道并预测新双键的立体化学(如果有的话)。"
> 
> "如果在与氧原子相邻的饱和碳原子上有一个取代基的话，可能会产生立体化学。如果有，那么所得的双键强烈偏好反式(E)几何结构。这是因为取代基倾向于在椅式过渡态中处于平伏位置。"

### 8.5 σ键的s/a对称性（line ~1999-2001）

> "而对于σ键，对称性不那么明显。我们想要知道，它在各端做了同一件事(s)还是不同的事(a)。'事'指的是什么呢？它在反应时，会使用sp³轨道的大波瓣(构型保持retention)或小波瓣(构型翻转inversion)。如果它在各端都保持地反应，或都翻转地反应，那么它是同面的；如果它在一端保持地反应，在另一端翻转的反应，那么它是异面的。"

### 8.6 Cope重排的驱动力（line ~2031-2035）

> "Cope重排(rearrangement)是在环中仅涉及碳原子的[3,3]-σ重排。该反应最简单的版本根本不是一个反应。起始原料和产物是相同的。"
> 
> "如果我们在正确的位置放置一个OH取代基，我们也可以通过羰基的形成驱动反应。"

### 8.7 阴离子加速Cope（line ~2075-2087）

> "σ重排步骤的产物是最终产物的烯醇。结果表明，如果用碱(最好是KH)处理起始原料以得到烷氧基阴离子进行反应，那么反应会被加速。产物是烯醇钾，比简单的烷氧基钾起始原料更加稳定。"
> 
> "通过这种方法，可以制得一些引人注目的化合物。其中最奇怪的一种——一个'桥头'烯烃——可通过烷氧基钾加速的Cope重排制得，反应过程中，四元环被扩环为包含一根反式双键(绿色所示)的八元环。"

### 8.8 [2,3]-σ重排的ω组分（line ~2305-2329）

> "于描述简单p轨道的新符号是ω。碳阴离子是一个ω2组分，碳阳离子是一个ω0组分（含有零个电子）。如果两根新键形成在碳阴离子p轨道的同一波瓣上，那么我们具有的就是一个ω2s组分，如果它们在不同波瓣上形成，就是ω2a组分。"
> 
> "我们所要讨论的[2,3]-σ重排可以描述为一个ω2a + σ2s + π2a反应。包含一个(4q+2)s组分，没有(4r)a组分，因此该反应是热允许的。"

### 8.9 [1,5]H移位的轨道描述（line ~2509-2517）

> "氢原子划过平面状环戊二烯环的顶面。我们称这是一个同面迁移，意味着迁移基团从π体系的一面离开，并从同一面再次加入（此例子中均为顶面）。异面迁移则意味着，离去发生在顶面而加入发生在底面——显然，在这里是不可行的。"
> 
> "σ氢移位的轨道对称性的结果很简单。在热反应中，[1,5]H移位同面发生，但[1,3]H和[1,7]H移位都必须是异面的。异面的[1,3]H移位，即使是允许的，也是不可行的，因为刚性的三碳链太短，无法使H原子由顶部转移到底部。"

### 8.10 电环化反应的定义（line ~2610-2635）

> "在电环化反应中，总是有一个环断裂或消除。环，当然也可以通过环加成反应形成。但区别在于，电环化反应只是在一个单独的共轭π体系中一根新σ键的形成(或断裂)的过程；而在环加成中，往往形成(或断裂)两根σ键，σ重排反应则是在一根σ键形成的同时另一根断裂的过程。"

### 8.11 电环化反应规则（line ~2786-2794）

> "所有电环化反应都是允许的。
> - 涉及(4n+2)个π电子的热电环化反应是对旋的。
> - 涉及(4n)个π电子的热电环化反应是顺旋的。
> - 在顺旋反应中，两个基团向相同方向旋转：都顺时针或都逆时针。
> - 在对旋反应中，一个基团顺时针旋转，一个基团逆时针旋转。"
> 
> "这种旋转就是您必须仔细区分电环化和其他周环反应的原因。在环加成和σ重排反应中，键角在109°和120°之间变化，旋转很小，但在电环化反应中，由于平面多烯要变为一个环（或者与之相反），所需的旋转则会接近90°。"

### 8.12 允许的电环化被空间因素阻止（line ~2928-2933）

> "我们有必要提醒您，虽然所有的电环化反应，无论在热条件还是光条件下，只要旋转的方向选对了，就都会是允许的。但顺旋环化和对旋环化、开环的空间需求可能会导致其中一种模式变得不可行。"

### 8.13 环丙基阳离子的电环化开环（line ~2934-2948）

> "烯丙基阳离子是包含2π电子的共轭体系，因此如果您不知道本章以外的任何知识，您会认为，它们会通过一个对旋的电环化关环发生环化。产物会是一个环丙基阳离子。"
> 
> "事实上，正是环丙基阳离子会发生这种反应（属实非常容易——环丙基阳离子几乎不可被观察到），这是因为环张力促进它们通过电环化开环反应，得到烯丙基阳离子。"

### 9.1 高频题型

1. **电环化顺旋/对旋判断**（★★★★★）

   - 邻位被占时的对位产物（两次[3,3]）

## 十、存疑或待验证之处

1. **氧化态标注**：译者注指出硫的氧化态标注（S(II)→S(IV)）可能不严格正确，按电负性分配整数氧化态的方法重排前后都是S(0)。这在竞赛中通常不影响解题，但需注意概念严谨性。

2. **"旋光"与"对旋"的翻译**：本书使用"顺旋"(conrotatory)和"对旋"(disrotatory)，与常见教材一致。但需注意有些教材可能使用不同译名。

3. **环庚三烯/降替二烯平衡**：书中提到"通常环庚二烯是平衡的主要组分"，但随后又说"环庚三烯"——此处可能是原文笔误或翻译问题，应为"环庚三烯"（cycloheptatriene）与降替二烯（norcaradiene）的平衡。

4. **己三烯热环化温度**：书中提到"500°C"，这是非常高的温度，竞赛中通常不涉及具体数值，但需注意实际反应条件。

5. **阴离子Oxy-Cope的加速倍数**：现有KP提到"10^10–10^17倍"，本章未给出具体数值，仅说"被加速"。如需精确数据需查原始文献。

6. **美洲蜚蠊酮B合成中的细节**：本章末尾的合成路线涉及多个周环反应的串联，是极好的综合题素材，但部分步骤的立体化学解释需要结合Chapter 27的光异构化内容。
