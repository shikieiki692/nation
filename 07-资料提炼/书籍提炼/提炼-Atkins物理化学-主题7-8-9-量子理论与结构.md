## 二、讲解脉络

### 主题 7（量子理论）
> 从经典力学在微观尺度的失效出发，通过黑体辐射、光电效应、原子光谱等实验引出能量量子化和波粒二象性；建立薛定谔方程和波函数的玻恩诠释；引入算符、期望值和不确定原理；最后将量子力学应用于三种基本运动模型——平动（箱中粒子）、振动（谐振子）和转动（刚性转子），为后续原子结构和分子结构奠定数学基础。

- **切入前置**：7A 从19世纪末的经典物理学危机入手，用 Planck 量子假说、Einstein 光子理论和 de Broglie 关系建立波粒二象性图像。
- **全章高潮**：7B–7C 建立薛定谔方程的求解框架和算符形式主义，不确定原理是量子力学与经典力学最深刻的区别。
- **理论深化**：7D–7F 的三个模型系统（粒子在一维箱、谐振子、刚性转子）是理解原子轨道和分子振动/转动光谱的数学基础。
- **教学风格**：**物理图像 / 数学推导并重** — 每个模型都给出完整的薛定谔方程求解过程。

### 主题 8（原子结构）
> 将主题7的量子力学框架应用于原子体系：先精确求解类氢原子（单电子体系），得到原子轨道的量子数（n, l, m_l）和径向/角向分布函数；再处理多电子原子，引入自旋量子数 m_s、Pauli 不相容原理、Slater 轨道和自洽场方法；最后以原子光谱为出口，建立光谱项符号（term symbols）体系，连接量子数与实验观测。

- **切入前置**：8A 将主题7D的球坐标薛定谔方程求解应用于 Coulomb 势，自然导出 n, l, m_l 三个量子数。
- **全章高潮**：8B 的多电子原子结构是核心：自旋、Pauli 原理、Aufbau 原理、Hund 规则、周期表的结构解释。
- **理论深化**：8C 的光谱项符号（^{2S+1}L_J）是决赛阶段原子光谱的核心工具，也是群论在原子体系中的简化形式。
- **教学风格**：**从精确解到近似方法** — 类氢原子可严格求解，多电子原子必须引入近似。

### 主题 9（分子结构）
> 从两种互补的成键模型出发：价键理论（VB，定域键）强调电子配对和杂化；分子轨道理论（MO，离域键）强调轨道组合和对称性。从最简单的 H2+ 入手建立 LCAO-MO 方法，逐步扩展到同核/异核双原子分子和多原子分子，最后以休克尔方法处理共轭体系，并简要介绍 DFT 等现代方法。

- **切入前置**：9A 从 Heitler-London 对 H2 的处理出发，建立 VB 理论的电子配对图像。
- **全章高潮**：9B–9D 的 LCAO-MO 理论是理解双原子分子电子结构的核心：成键/反键轨道、键级、磁性判据。
- **理论深化**：9E 的多原子分子 MO 理论引入群论对称性（与主题10联动），休克尔方法（Huckel）是处理共轭体系的核心工具。
- **教学风格**：**两种模型对比** — VB 和 MO 的物理图像差异、适用范围和互补性贯穿全章。

## 五、关键公式和概念

### 主题7（量子理论）
- **Planck 量子假说**：E = hν
- **de Broglie 关系**：λ = h/p = h/(mv)
- **薛定谔方程（定态）**：Ĥψ = Eψ
- **玻恩诠释**：|ψ|² 是概率密度
- **不确定原理**：Δx·Δp ≥ ℏ/2；ΔE·Δt ≥ ℏ/2
- **一维箱中粒子**：E_n = n²h²/(8mL²)，n = 1,2,3,...
- **谐振子**：E_v = (v + 1/2)hν，零点能 E₀ = hν/2
- **刚性转子**：E_J = J(J+1)ℏ²/(2I)，J = 0,1,2,...

### 主题8（原子结构）
- **类氢原子能级**：E_n = -Z²R_H/n²，R_H = 13.6 eV（H原子）
- **量子数**：n（主量子数）、l（角量子数，0→n-1）、m_l（磁量子数，-l→+l）、m_s（自旋，±1/2）
- **Pauli 不相容原理**：同一原子中不能有四个量子数完全相同的两个电子
- **Aufbau 原理**：按能量最低原则填充
- **Hund 规则**：简并轨道上电子优先平行自旋占据不同轨道
- **光谱项符号**：^{2S+1}L_J，其中 L = 0(S),1(P),2(D),3(F),...
- **选律**：Δl = ±1，Δm_l = 0, ±1

### 主题9（分子结构）
- **LCAO-MO**：ψ = c_Aφ_A + c_Bφ_B
- **键级**：(成键电子数 - 反键电子数)/2
- **同核双原子分子轨道顺序**：
  - Z ≤ 7（N₂及更轻）：σ1s < σ*1s < σ2s < σ*2s < π2p < σ2p < π*2p < σ*2p
  - Z ≥ 8（O₂及更重）：σ1s < σ*1s < σ2s < σ*2s < σ2p < π2p < π*2p < σ*2p
- **O₂ 的顺磁性**：π*2p 上有两个未成对电子（^{3}Σ_g^-）
- **杂化轨道**：sp(180°), sp²(120°), sp³(109.5°), sp³d(三角双锥), sp³d²(八面体)
- **休克尔方法**：对共轭体系，α = H_ii（库仑积分），β = H_ij（共振积分，相邻原子），S_ij = 0（重叠忽略）
