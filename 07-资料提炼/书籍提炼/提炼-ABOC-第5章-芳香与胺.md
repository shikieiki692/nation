## 五、各章精要

### 5.1 芳香性（5.1）【→学生讲义：芳香烃与卤代烃 · 芳香性】

**休克尔规则（Hückel's Rule）**：
- 平面、环状、完全共轭的体系
- π 电子数 = 4n + 2（n = 0, 1, 2...）→ 芳香性
- π 电子数 = 4n → 反芳香性（极不稳定）

| 体系 | π 电子数 | 芳香性 |
|:---|:---:|:---:|
| 苯 | 6 | ✅ 芳香 |
| 环戊二烯负离子 | 6 | ✅ 芳香 |
| 环庚三烯正离子 | 6 | ✅ 芳香 |
| 环丁二烯 | 4 | ❌ 反芳香 |
| 环辛四烯 | 8 | ❌ 非芳香（折叠）|

**Möbius 芳香性**（ABOC 拓展）：
- 扭曲环状体系，节点数为奇数
- π 电子数 = 4n 时反而稳定
- 竞赛中偶尔涉及轮烯（annulene）的芳香性判断

**同芳香性（Homoaromaticity）**：
- 共轭链被一个 sp³ 碳打断，但轨道仍能重叠
- 经典案例：环丙烯正离子、降冰片二烯正离子

### 5.2 芳香亲电取代（SEAr）（5.2）【→学生讲义：芳香烃与卤代烃 · SEAr】

**机理两步**：
```
步骤 1：亲电试剂 E⁺ 进攻芳环 → σ 络合物（Wheland 中间体，正电荷）
步骤 2：失去 H⁺ → 恢复芳香性
```

**定位规则**：

| 取代基类型 | 例子 | 定位 | 活化/钝化 |
|:---|:---|:---:|:---:|
| **给电子诱导** | -R, -Ar | 邻/对位 | 弱活化 |
| **给电子共轭** | -OH, -OR, -NH₂ | 邻/对位 | 强活化 |
| **卤素** | -F, -Cl, -Br, -I | 邻/对位 | 弱钝化 |
| **吸电子共轭** | -NO₂, -CN, -CHO | 间位 | 强钝化 |
| **吸电子诱导** | -NR₃⁺ | 间位 | 强钝化 |

> ⭐ **ABOC 核心视角**：定位规则的本质是**σ 络合物中间体的稳定性**——给电子基稳定邻/对位中间体，吸电子基稳定间位中间体（相对而言）。

**Friedel-Crafts 反应限制**：
- **F-C 烷基化**：可重排（碳正离子中间体）、可逆、多取代
- **F-C 酰基化**：不重排、不可逆、单取代 → 更常用于合成
- 强钝化基（-NO₂, -CN）存在时 F-C 不反应

### 5.3 Birch 还原（5.5）【→学生讲义：芳香烃与卤代烃 · Birch还原】

**Birch 还原**：芳环在 **Na/NH₃(l)/醇** 条件下被还原为 **1,4-环己二烯**。

**区域选择性**：
- **给电子取代基（EDG）** → 还原发生在**间位**（产物：取代基在双键碳上）
- **吸电子取代基（EWG）** → 还原发生在**邻/对位**（产物：取代基在饱和碳上）

**本质**：单电子还原（SET），生成自由基负离子中间体。

### 5.4 [3,3]-σ 迁移（5.6）【→学生讲义：芳香烃与卤代烃 · Claisen/Cope重排】

**Claisen 重排**：烯丙基芳基醚 → 邻烯丙基苯酚（热驱动，协同）
- 机理：[3,3]-σ 迁移
- 是**周环反应**的经典入门案例

**Cope 重排**：1,5-二烯 → 异构 1,5-二烯
- 与 Claisen 类似，但无氧原子
- ABOC 此处引入"周环反应"概念，为 Ch.7 专章铺垫

### 5.5 胺的制备（5.8）【→学生讲义：醇醚胺酚 · 胺的制备】

| 方法 | 试剂 | 特点 |
|:---|:---|:---|
| **Gabriel 合成** | 邻苯二甲酰亚胺 + RX + 肼解 | 制备 **1° 胺**，无过度烷基化 |
| **Fukuyama-Mitsunobu** | 邻苯二甲酰亚胺 + 醇 + PPh₃/DIAD | 从醇制备 1° 胺，构型翻转 |
| **Ritter 反应** | 腈 + 醇/烯烃 + 强酸 | 制备 **酰胺**，水解得胺 |
| **还原胺化** | 醛/酮 + NH₃/胺 + NaBH₃CN | 最直接的方法 |
| **TosMIC** | 对甲苯磺酰甲基异腈 | 醛 → 多一个碳的腈 → 胺 |

### 5.6 重氮盐化学（5.9）【→学生讲义：芳香烃与卤代烃 · 重氮盐化学】

**重氮盐的制备**：ArNH₂ + NaNO₂/HCl（0-5°C）→ ArN₂⁺Cl⁻

**Sandmeyer 反应**：
- ArN₂⁺ + CuCl → ArCl
- ArN₂⁺ + CuBr → ArBr
- ArN₂⁺ + CuCN → ArCN
- ArN₂⁺ + KI → ArI（无需 Cu）

**Schiemann 反应**：ArN₂⁺ + HBF₄ → ArF（热分解）

**偶联反应**：
- 重氮盐 + 苯酚/苯胺 → 偶氮染料（azo compound）
- 是**亲电芳香取代**的变体（重氮正离子作为亲电试剂）

## 六、关键创新点

### 6.1 ABOC 独特视角

1. **"芳香性是一个连续谱"**：休克尔 → Möbius → 同芳香性 → 反芳香性，不是"是/否"的二元判断。

2. **"定位规则是中间体稳定性的可视化"**：不要死记"邻对位/间位"，而要画出 σ 络合物的共振式，看正电荷是否落在给电子基上。

3. **"Birch 还原的区域选择性由自由基负离子稳定性决定"**：EDG 稳定自由基负离子 → 出现在产物双键上；EWG 稳定 → 出现在饱和碳上。

4. **"胺的制备是一个策略网络"**：Gabriel（1°胺）、Ritter（酰胺）、还原胺化（最直接）、TosMIC（多一个碳）——不同场景选不同工具。

### 6.2 与传统教材的差异

| 主题 | 传统教材 | ABOC |
|:---|:---|:---|
| 芳香性 | 只讲休克尔 | 引入 Möbius、同芳香性、反芳香性 |
| SEAr 定位 | 死记规则 | 用 σ 络合物共振式解释 |
| 胺的制备 | 单独讲每种方法 | 用"策略网络"思维串联 |
| 人名反应 | 孤立记忆 | 用机理共性（如异腈化学串联 TosMIC/Ugi/Passerini）|

### 11.2 教学重点

1. **芳香性判据的连续谱**：不要让学生"数 π 电子"后机械判断，而要理解轨道重叠、平面性、节点数等多因素。

2. **定位规则的共振式解释**：画出 σ 络合物的三个共振式，看正电荷是否落在给电子基上——这是"为什么邻对位"的本质。

3. **Birch 还原的"反直觉"区域选择性**：EDG → 产物双键上（"似乎被还原的是没有取代基的碳"）。

4. **重氮盐是"万能中间体"**：ArN₂⁺ 可以变成 ArCl/ArBr/ArI/ArCN/ArF/ArOH/ArH —— 学会重氮盐 = 学会芳香取代的无限可能。

### 12.1 学生常见误区

### 12.2 竞赛陷阱

1. **T2（34 届初赛）**：多步 SEAr 合成中，取代顺序决定产物——酰基化先进行（定位+不会重排），然后还原烷基，再二次取代。

3. **T7（原创）**：Möbius 芳香性的判断需要数节点数 + π 电子数，不能只数电子。
