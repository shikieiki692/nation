## 二、📐 讲解脉络

### 主题 13（统计热力学）
> Atkins 从"等概率原理"出发，通过组合数学导出 Boltzmann 分布，揭示温度的统计本质；进而引入配分函数这一"热力学波函数"，展示如何用分子层面的信息（能级、简并度）计算宏观热力学量（内能、熵、Helmholtz能、Gibbs能）；最后将统计热力学应用于平衡常数的计算，建立微观光谱数据与宏观化学平衡之间的桥梁。

- **切入前置**：13A 从"构型与权重"的组合数学问题入手，通过 Stirling 近似和最概然分布方法，导出 Boltzmann 分布 $N_i/N = g_i e^{-\beta\varepsilon_i}/q$，其中 $\beta = 1/(kT)$。
- **全章高潮**：13B–13C 建立配分函数 $q = \sum_j g_j e^{-\beta\varepsilon_j}$ 及其与分子各运动模式（平动、转动、振动、电子）贡献的分解；13E 将 $q$ 与内能 $U$ 和熵 $S$ 严格关联，完成微观→宏观的跃迁。
- **理论深化**：13F 从 Helmholtz 能 $A = -NkT\ln q$ 出发，导出 Gibbs 能 $G = A + pV$，最终得到用配分函数计算平衡常数的公式 $K = (q_{\text{产物}}/V)^{\nu_{\text{产物}}} / (q_{\text{反应物}}/V)^{\nu_{\text{反应物}}} \times (RT/p^{\ominus})^{\Delta\nu}$。
- **教学风格**：**严格推导 / 数学驱动** — 大量组合数学、Stirling 近似、Lagrange 乘子法，每个公式都有"如何完成？"专栏。

### 主题 14（分子相互作用）
> 从分子的电性质（偶极矩、极化率）出发，建立分子间各种相互作用（取向力、诱导力、色散力、氢键）的定量理论；进而将这些相互作用应用于液体（表面张力、表面膜、凝结）和自组装体系（胶体、胶束、生物膜）的物理化学描述。

- **切入前置**：14A 定义电偶极矩 $\mu = QR$ 和极化率 $\alpha$，建立分子对外电场的响应图像（诱导偶极 $\mu^* = \alpha E$）。
- **全章高潮**：14B 给出分子间相互作用能的定量公式：Keesom（取向）$V \propto -\mu_1^2\mu_2^2/(kTr^6)$、Debye（诱导）$V \propto -(\mu_1^2\alpha_2 + \mu_2^2\alpha_1)/r^6$、London（色散）$V \propto -I_1I_2\alpha_1\alpha_2/(I_1+I_2)r^6$；氢键作为特殊的强偶极相互作用。
- **理论深化**：14C 用热力学论证推导表面张力、表面膜的状态方程（Gibbs膜）、凝结的 Kelvin 方程；14E 从疏水相互作用和熵驱动出发解释胶束形成。
- **教学风格**：**物理图像 / 定量推导并重** — 既有清晰的物理图像（如瞬时偶极诱导偶极），又有严格的能量公式推导。

## 六、关键图表和反应方程式

### 核心公式（竞赛必会）

#### 主题 13
- **Boltzmann 分布**：$N_i/N = g_i e^{-\varepsilon_i/kT} / q$
- **分子配分函数**：$q = \sum_j g_j e^{-\beta\varepsilon_j}$，$\beta = 1/(kT)$
- **内能**：$U - U(0) = -N(\partial\ln q/\partial\beta)_V = NkT^2(\partial\ln q/\partial T)_V$
- **熵**：$S = [U - U(0)]/T + kN\ln q$（定域子）或 $S = [U - U(0)]/T + kN\ln(qe/N)$（离域子）
- **Helmholtz 能**：$A - A(0) = -NkT\ln q$
- **各运动模式配分函数分解**：$q = q_T \cdot q_R \cdot q_V \cdot q_E$
- **平动配分函数**：$q_T = (2\pi mkT/h^2)^{3/2} V$

#### 主题 14
- **电偶极矩**：$\mu = QR$
- **诱导偶极**：$\mu^* = \alpha E$
- **Keesom（取向）相互作用**：$V = -\frac{2\mu_1^2\mu_2^2}{3(4\pi\varepsilon_0)^2 kT r^6}$
- **Debye（诱导）相互作用**：$V = -\frac{\mu_1^2\alpha_2 + \mu_2^2\alpha_1}{(4\pi\varepsilon_0)^2 r^6}$
- **London（色散）相互作用**：$V = -\frac{3}{2} \frac{I_1 I_2}{(I_1+I_2)} \frac{\alpha_1' \alpha_2'}{(4\pi\varepsilon_0)^2 r^6}$
- **总相互作用**：$V_{\text{总}} = V_{\text{取向}} + V_{\text{诱导}} + V_{\text{色散}}$（均 $\propto r^{-6}$）
- **Kelvin 方程**：$p = p^* \exp(2\gamma V_m / rRT)$
- **表面压**：$\pi = \gamma^* - \gamma$
