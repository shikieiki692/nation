## 二、讲解脉络

### 主题 2（热力学第一定律）
> 从系统与环境的能量交换出发，引入热和功的概念，建立内能U作为状态函数的定义；进而定义焓H以方便处理等压过程；讨论热化学中的标准焓变、Hess定律和Kirchhoff定律；通过全微分建立状态函数之间的定量关系（膨胀系数、压缩系数、Joule-Thomson效应）；最后讨论完美气体的绝热可逆过程。

- **切入前置**：2A 从功和热的分子诠释（有序运动 vs 无序运动）出发，建立能量交换的物理图像。
- **全章高潮**：2D 的状态函数和全微分是连接热力学第一定律与数学工具的桥梁，为后续Maxwell关系奠定基础。
- **理论深化**：2D.2–2D.3 利用全微分推导内能和焓随温度、压力变化的通式；2D.4 的 Joule-Thomson 效应是气体液化的理论基础。
- **教学风格**：**从物理图像到数学表述** — 每个概念先给定性理解，再给出严格定义和公式。

### 主题 3（热力学第二和第三定律）
> 从自发变化的方向性问题出发，通过Carnot循环和Clausius不等式引入熵S；建立熵的热力学定义和统计定义（Boltzmann公式）；讨论各类过程的熵变计算和熵的测量（第三定律）；引入Helmholtz能A和Gibbs能G作为等温等容/等压条件下的自发判据；最后将第一、第二定律结合，推导热力学基本方程和Maxwell关系式。

- **切入前置**：3A 从自发变化的日常经验（热从高温流向低温）出发，通过Carnot热机建立热力学温度的概念。
- **全章高潮**：3D 的Gibbs能是化学中最核心的热力学函数；3E 的Maxwell关系是热力学推导的"瑞士军刀"。
- **理论深化**：3E 将第一、第二定律结合为四个热力学基本方程，通过全微分性质导出四组Maxwell关系，实现"用易测量代替难测量"。
- **教学风格**：**从现象到判据** — 先回答"过程向哪个方向进行"，再给出"如何判断"。

## 五、关键公式和概念

### 主题2（热力学第一定律）
- **第一定律**：$\Delta U = q + w$（封闭体系）；隔离系统 $\Delta U = 0$
- **膨胀功**：$w = -\int_{V_1}^{V_2} p_{\text{ex}} \, dV$；可逆过程 $w = -\int_{V_1}^{V_2} p \, dV$
- **等外压膨胀**：$w = -p_{\text{ex}} \Delta V$
- **完美气体等温可逆膨胀**：$w = -nRT \ln(V_2/V_1) = -nRT \ln(p_1/p_2)$
- **焓的定义**：$H = U + pV$；等压过程 $\Delta H = q_p$
- **热容关系**：$C_p - C_V = nR$（完美气体）；$C_p - C_V = \alpha^2 TV/\kappa_T$（一般物质）
- **Hess定律**：反应焓变与途径无关
- **Kirchhoff定律**：$\Delta_r H^\circ(T_2) = \Delta_r H^\circ(T_1) + \int_{T_1}^{T_2} \Delta_r C_p^\circ \, dT$
- **内压/热力学状态方程**：$\pi_T = (\partial U/\partial V)_T = T(\partial p/\partial T)_V - p$
- **绝热可逆过程（完美气体）**：$TV^{\gamma-1} = \text{const}$，$pV^\gamma = \text{const}$，$T^\gamma p^{1-\gamma} = \text{const}$

### 主题3（热力学第二和第三定律）
- **Clausius不等式**：$dS \geq \delta q/T$；可逆过程取等号
- **熵的热力学定义**：$dS = \delta q_{\text{rev}}/T$
- **熵的统计定义（Boltzmann）**：$S = k_B \ln \Omega$
- **熵增原理**：$\Delta S_{\text{孤立}} \geq 0$
- **常见熵变**：
  - 等温膨胀：$\Delta S = nR\ln(V_2/V_1)$
  - 等压变温：$\Delta S = nC_p\ln(T_2/T_1)$
  - 可逆相变：$\Delta S = \Delta H_{\text{相变}}/T_{\text{相变}}$
- **第三定律**：完美晶体在0 K时 $S = 0$；$S(T) = \int_0^T (C_p/T')dT' + \sum \Delta H_{\text{相变}}/T_{\text{相变}}$
- **Helmholtz能**：$A = U - TS$；等温过程 $w_{\text{max}} = \Delta A$
- **Gibbs能**：$G = H - TS = A + pV$；等温等压 $w_{\text{add,max}} = \Delta G$
- **自发判据**：
  - 隔离系统：$dS_{U,V} \geq 0$
  - 等温等容：$dA_{T,V} \leq 0$
  - 等温等压：$dG_{T,p} \leq 0$
- **标准反应Gibbs能**：$\Delta_r G^\circ = \Delta_r H^\circ - T\Delta_r S^\circ = \sum_j \nu_j \Delta_f G^\circ(j)$
- **Gibbs-Helmholtz方程**：$[\partial(\Delta G/T)/\partial T]_p = -\Delta H/T^2$

### 主题3E（热力学基本方程与Maxwell关系）
- **四个热力学基本方程**：
  - $dU = TdS - pdV$
  - $dH = TdS + Vdp$
  - $dA = -SdT - pdV$
  - $dG = -SdT + Vdp$
- **四组Maxwell关系**：
  1. $(\partial T/\partial V)_S = -(\partial p/\partial S)_V$
  2. $(\partial T/\partial p)_S = (\partial V/\partial S)_p$
  3. $(\partial S/\partial V)_T = (\partial p/\partial T)_V$（最常用）
  4. $(\partial S/\partial p)_T = -(\partial V/\partial T)_p$
- **重要偏导数**：
  - $(\partial G/\partial T)_p = -S$
  - $(\partial G/\partial p)_T = V$
  - $(\partial H/\partial T)_p = C_p$
  - $(\partial U/\partial T)_V = C_V$
