## 二、📐 讲解脉络

> Weller 本章按「CFT（唯象）→ LFT（MO 解释）→ 光谱（实验手段）→ CT/选律/发光/磁性（深入专题）」四层递进。§20.1 先用点电荷模型建立 Δ₀、LFSE、高/低自旋、Jahn-Teller 等核心概念；§20.2 用 MO 理论解释光谱化学序列（σ + π 效应）；§20.3–20.4 引入 Russell-Saunders 偶合 → 光谱项 → Orgel 图 → Tanabe-Sugano 图，教会如何从光谱实验反推 Δ₀ 和 Racah 参数 B；§20.5–20.9 则是配位化学光谱/磁性/光化学的专题深化。

- **切入前置**：§20.1(a)–(c) 从八面体 d 轨道分裂图 + Ti(H₂O)₆³⁺ 光谱切入，以 Δ₀ 为核心变量建立 CFT 框架。这是整章的逻辑起点。
- **铺垫段落**：§20.1(d) 水合焓双峰曲线和 §20.1(i) Irving-Williams 序列用热力学/热化学数据验证 LFSE 概念，是连接理论与实验的关键桥梁。§20.2 的 σ/π 成键分析解释了 CFT 不能解释的光谱化学序列，是从"是什么"到"为什么"的转折。
- **全章高潮**：§20.4 的 Tanabe-Sugano 图 + 例题 20.8（从光谱反推 Δ₀ 和 B 值）是全章方法论的核心——竞赛中常考"给光谱数据，求 Δ₀/B/归属跃迁"类题型。§20.3 的光谱项推导（Clebsch-Gordan 系列 + 微状态表消去法）是竞赛理论题的难点。
- **一笔带过**：§20.5(b) MLCT（Ru-bpy 光化学）在竞赛中出现频率低于 LMCT；§20.7 发光和 §20.9 自旋交叉在初赛中极少涉及。
- **教学风格**：**理论-实验闭环**——每个理论概念都通过具体光谱/磁性实验数据验证，例题 20.1–20.9 贯穿全章（LFSE 计算 → 磁性判断组态 → 热化学解释 → PES → 光谱项 → 基谱项 → 谱项相关 → Tanabe-Sugano 图 → 选律归属）。

## 六、关键图表和反应方程式

- **d 轨道分裂能级图**：O_h 场（t₂g↓, e_g↑）、T_d 场（e↓, t₂↑，Δ_T≈(4/9)Δ₀）、D_4h 场（x²−y² 最高）
- **光谱化学序列**：I⁻<Br⁻<S²⁻<SCN⁻<Cl⁻<NO₂⁻<N₃⁻<F⁻<OH⁻<ox²⁻<O²⁻<H₂O<NCS⁻<CH₃CN<py<NH₃<en<bpy<phen<NO₂⁻<PPh₃<CN⁻<CO
- **配体分类（π 效应）**：π 给予体(I⁻,Br⁻,Cl⁻,F⁻) → 弱 π 给予体(H₂O) → 无 π 效应(NH₃) → π 接受体(PR₃,CO)
- **唯自旋磁矩公式**：μ = √[N(N+2)] μ_B，其中 μ_B = eℏ/2m_e = 9.274×10⁻²⁴ J·T⁻¹
- **Irving-Williams 序列**：Ba²⁺<Sr²⁺<Ca²⁺<Mg²⁺<Mn²⁺<Fe²⁺<Co²⁺<Ni²⁺<Cu²⁺>Zn²⁺
- **Racah 参数 d² 谱项能量**：E(¹S)=A+14B+7C；E(¹G)=A+4B+2C；E(¹D)=A−3B+2C；E(³P)=A+7B；E(³F)=A−8B
- **电子云重排参数**：β = B(络合物)/B(自由离子)；序列：Br⁻<CN⁻<Cl⁻<NH₃<H₂O<F⁻
- **Tanabe-Sugano 图使用法**：E/B 对 Δ₀/B 作图 → 跃迁能比值匹配 → 读出 Δ₀/B 和 E/B → 反推 Δ₀ 和 B
- **谱带强度四档**：自旋禁阻(ε<1) / Laporte 禁阻 d-d(20~100) / Laporte 允许 d-d(~250) / 对称性允许 CT(1000~50000)
- **LMCT 能量趋势**：[MnO₄]⁻<[TcO₄]⁻<[ReO₄]⁻（同族向下能量升高）；+7>+6>+5（氧化态降低能量升高）
- **红宝石激光机理**：⁴T₂g←⁴A₂g（绿吸收）+ ⁴T₁g←⁴A₂g（紫吸收）→ 系间跨越 → ²E→⁴A₂ 红光发射（627 nm）
- **磁化率-温度关系**：铁磁性（T_C 以上变顺磁）、反铁磁性（T_N 处 χ 骤降）、亚铁磁性（反平行但净磁矩≠0）
