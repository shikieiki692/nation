## 五、各章精要

### 5.1 UV-Vis 光谱（10.1）【→学生讲义：杂环糖氨基酸 · UV-Vis光谱】

**核心公式**：
$$
A = \varepsilon \cdot c \cdot l
$$

**关键概念**：
- **发色团**（Chromophore）：产生吸收的官能团（C=C、C=O、芳环等）
- **助色团**（Auxochrome）：本身不吸收但能使发色团吸收红移的基团（–OH、–NH₂、–Cl 等）
- **红移/蓝移**：共轭增加 → λmax 红移；溶剂极性增加 → n→π* 蓝移、π→π* 红移

**Woodward-Fieser 规则**（用于预测共轭二烯/三烯的 λmax）：
- 母体双烯基准值 + 延伸双键增量 + 取代基增量 + 环外双键增量
- 竞赛中常用于**验证结构**而非从头预测

**竞赛应用**：
- 判断共轭体系大小（λmax 与共轭长度正相关）
- 区分互变异构体（烯醇式 vs 酮式的 UV 差异）

### 5.2 有机光化学（10.2）【→学生讲义：杂环糖氨基酸 · 有机光化学】

**激发态基础**：

| 参数 | 基态 S₀ | 单线态 S₁ | 三线态 T₁ |
|:---|:---:|:---:|:---:|
| 电子自旋 | 配对 ↑↓ | 配对 ↑↓ | 平行 ↑ ↑ |
| 寿命 | — | 10⁻⁹–10⁻⁶ s | 10⁻⁶–10⁻³ s |
| 能量 | 最低 | 较高 | 中等（低于 S₁）|

**关键过程**：
- **ISC**（系间窜越）：S₁ → T₁，自旋禁阻但可发生
- **荧光**：S₁ → S₀ + hν（快，10⁻⁸ s）
- **磷光**：T₁ → S₀ + hν（慢，10⁻³ s）

**经典光化学反应**：

| 反应 | 类型 | 关键特征 |
|:---|:---|:---|
| **Norrish Type I** | α-裂解 | 羰基化合物 → 酰基自由基 + 烷基自由基 |
| **Norrish Type II** | γ-H 夺取 | 羰基 γ-位有 H → 环化 → 1,4-双自由基 → 烯烃 + 烯醇 |
| **Paternò-Büchi** | [2+2] 光环化 | 羰基 + 烯烃 → 氧杂环丁烷 |
| **[2+2] 光环化** | 协同/分步 | 两个烯烃 → 环丁烷（热禁阻、光允许）|

**竞赛应用**：
- [2+2] 光环化是**热禁阻、光允许**的典型——与热驱动的 D-A 形成互补
- 光化学的**选择性**来自激发态电子分布与基态不同

### 5.3 光氧化还原催化（10.3）【→学生讲义：杂环糖氨基酸 · 光氧化还原催化】

**核心催化剂**：

| 催化剂 | 激发态还原电位 E*(Red) | 激发态氧化电位 E*(Ox) | 特点 |
|:---|:---:|:---:|:---|
| **Ru(bpy)₃²⁺** | –0.81 V | +0.77 V | 经典，水溶性好 |
| **Ir(ppy)₃** | –1.73 V | +0.31 V | 更强还原性，近年更常用 |

**催化循环**（以还原淬灭为例）：
```
PC + hν → PC*（激发态光催化剂）
PC* + 给体（如胺）→ PC•⁻ + 给体•⁺（还原淬灭）
PC•⁻ + 底物 → PC + 底物•⁻（单电子还原底物）
```

**双催化体系**（Dual Catalysis）：
- **光氧化还原 + 过渡金属**（如 Pd）：光催化提供自由基，金属催化提供交叉偶联
- **光氧化还原 + 有机催化**：光催化 SET + 有机小分子活化

**竞赛应用**：
- 光氧化还原是**近年决赛热点**（2019-2024 多次出现）
- 关键识别点：反应条件中有 **hv + 光催化剂（Ru/Ir/曙红Y等）**
- 核心机理：**单电子转移（SET）** 而非极性机理

### 5.4 质谱（MS）（10.4）【→学生讲义：杂环糖氨基酸 · 质谱】

**核心信息**：
- **分子离子峰（M⁺•）**：提供精确分子量
- **碎片离子峰**：提供结构信息（特征碎片 = 特征官能团）
- **同位素峰**：Cl（3:1）、Br（1:1）的特征同位素分布

**常见碎片**（竞赛高频）：

| 碎片 | m/z | 来源 |
|:---:|:---:|:---|
| 失去 CH₃• | M–15 | 甲基 |
| 失去 H₂O | M–18 | 醇 |
| 失去 CO | M–28 | 醛/酮/羧酸 |
| 失去 C₂H₄ | M–28 | McLafferty 重排 |
| 苄基离子 | 91 | 甲苯/取代甲苯 |
| 卓鎓离子 | 91 | 环庚三烯基正离子 |

**McLafferty 重排**：
- 含 γ-H 的羰基化合物（醛/酮/酸/酯）
- γ-H 转移至羰基氧 → 六元环过渡态 → 烯醇自由基正离子 + 烯烃
- 是**判断 γ-H 存在**的有力证据

**高分辨质谱（HRMS）**：
- 区分相同整数质量不同精确质量的离子（如 C₆H₈⁺• 和 C₅H₄O⁺• 均为 m/z 80，但精确质量不同）

### 5.5 ¹H NMR（10.5）【→学生讲义：杂环糖氨基酸 · ¹H NMR】

**化学位移 δ（ppm）**——ABOC 附录 2 核心数据：

| 质子类型 | δ 范围 | 备注 |
|:---|:---:|:---|
| R–CH₃ | 0.9 | 烷基质子基准 |
| R–CH₂–R | 1.3 | — |
| R₃CH | 1.5 | — |
| C=C–H | 4.5–6.5 | 烯基质子 |
| Ar–H | 6.5–8.5 | 芳香质子 |
| R–CHO | 9–10 | 醛基质子（特征性极高）|
| R–COOH | 10–13 | 羧酸（宽峰，可交换）|
| R–OH / R–NH₂ | 1–5（可变）| 可交换质子，位置随浓度/温度变化 |

**影响化学位移的因素**：
- **诱导效应**：电负性基团去屏蔽 → δ 增大
- **磁各向异性**：芳环/双键/三键的感应磁场效应（芳环 H 去屏蔽、炔 H 屏蔽）
- **氢键**：分子内氢键 → δ 增大且稳定；分子间氢键 → δ 随浓度变化

**裂分与偶合**：
- **n+1 规则**：相邻 n 个等价 H → 裂分为 n+1 重峰
- **偶合常数 J**：
  - ³J（邻位）：6–8 Hz（反式 > 顺式）
  - ²J（同碳）：10–15 Hz
  - ⁴J（远程）：0–3 Hz（W-型偶合）

**磁不等价质子**：
- 同一个 CH₂ 上的两个 H 可能不等价（如手性中心邻位的 CH₂、环己烷的 a/e 键）
- 表现为 **AB 体系**（而不是简单的双峰）

**NOE（核 Overhauser 效应）**：
- 空间接近（< 5 Å）的质子之间产生信号增强
- 用于判断**立体化学**（顺式/反式、endo/exo）

### 5.6 自由基反应（10.6）【→学生讲义：烷烃烯烃炔烃 · 自由基反应】

**链式反应三段式**（ABOC 反复强调）：

```
引发（Initiation）：产生自由基
    ↓
增长（Propagation）：自由基反应 → 新自由基
    ↓
终止（Termination）：两个自由基结合或歧化
```

**竞赛中高频自由基反应**：

| 反应 | 核心特征 | 竞赛考查点 |
|:---|:---|:---|
| **NBS 烯丙位溴代** | 低 Br₂ 浓度 → 选择性烯丙位溴代 | 机理：Br• 夺取烯丙位 H → 烯丙基自由基（稳定）→ Br₂ 捕获 |
| **自由基加成（HBr/过氧化物）** | 反马氏加成 | 过氧化物效应：ROOR → 2 RO• → RO• + HBr → ROH + Br• → 反马氏加成 |
| **Barton 反应** | 亚硝酸酯光解 → 自由基 → δ-H 夺取 → 亚硝基醇 → 肟 | 甾体 remote functionalization |
| **自由基环化** | 5-exo-trig / 6-endo-trig 选择性 | Baldwin 规则在自由基反应中的应用 |

**自由基稳定性顺序**（ABOC 5 规则之一）：
$$
\text{苄基/烯丙基} > \text{三级} > \text{二级} > \text{一级} > \text{甲基}
$$

## 六、关键创新点

### 6.1 ABOC 独特视角

1. **"光谱是有机化学家的眼睛"**：UV-Vis 看共轭、MS 看分子量/碎片、NMR 看精细结构——三兄弟各有专攻，组合使用才能确证结构。

2. **"光化学是热化学的补集"**：热禁阻的反应（如 [2+2] 环加成）光照下可能允许；基态不发生的电子跃迁，激发态可以发生。

3. **"光氧化还原催化是 21 世纪有机合成的核心方法之一"**：ABOC 将光氧化还原与 Ch.8 的 Pd 催化并列，强调其在现代合成中的重要性。

4. **"自由基反应不是特例，是通例"**：从 Ch.2 的过氧化物效应到 Ch.10 的系统讲解，ABOC 将自由基定位为与离子型反应并列的两大反应类型之一。

### 6.2 与传统教材的差异

| 主题 | 传统教材 | ABOC |
|:---|:---|:---|
| NMR | 罗列化学位移表 | 强调**结构推求的逻辑链**（分子式 → 不饱和度 → 特征峰 → 拼接）|
| 光化学 | 公式推导（量子力学）| 强调**激发态反应性**与基态的对比 |
| 光氧化还原 | 可能不讲（太新）| 作为独立小节，与 FINAL TEST 0（Mutilin 全合成）直接关联 |
| 自由基 | 只讲 NBS | 系统化：引发-增长-终止 + 选择性控制 + Baldwin 规则 |

### 11.2 教学重点

1. **"光谱三兄弟分工明确"**：UV-Vis 看共轭（粗略）、MS 看分子量和碎片（中等）、NMR 看精细结构（最细）——竞赛结构推求题的标准流程。

## 十二、🪤 踩坑回流

### 12.1 学生常见误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析 |
|:---|:---|:---|:---|
| "UV-Vis 用来确定官能团" | 以为和 IR 一样 | UV-Vis 只反映**共轭体系大小**，不能确定具体官能团 | 确定官能团靠 IR，确定结构靠 NMR |
| "NMR 裂分严格遵循 n+1" | 教材只讲简单情况 | 磁不等价时（手性中心邻位 CH₂）会出现 **AB 体系** | n+1 的前提是相邻质子**等价** |
| "光化学就是加热的另一种形式" | 以为 hv 只是提供能量 | 光化学是**激发态反应**，产物选择性与热反应**不同** | [2+2] 热禁阻、光允许 |
| "光氧化还原和光化学是一回事" | 都有 hv | 光化学是**底物直接激发**；光氧化还原是**催化剂吸收光**后转移电子 | 光氧化还原不需要底物吸收光 |
| "McLafferty 重排只发生在醛酮" | 教材例子多用醛酮 | 羧酸、酯、酰胺**也有 γ-H 时**同样可以发生 | 关键是 **γ-H + 羰基** 的结构 |
| "自由基反应没有选择性" | 以为自由基很乱 | 自由基稳定性（苄基 > 三级 > 二级）和 Baldwin 规则**强烈控制选择性** | NBS 烯丙位溴代就是高选择性 |

### 12.2 竞赛陷阱

1. **T-NMR（结构推求）**：给出分子式 + ¹H NMR 数据 + 偶合常数 → 要求推断结构并标注立体化学。**陷阱**：J = 15 Hz 的反式烯烃 vs J = 10 Hz 的顺式烯烃；NOE 增强指示空间接近。

2. **T-MS（碎片推断）**：高分辨 MS 给出精确分子量 → 要求写出分子式。**陷阱**：注意同位素峰（Cl、Br 的特征分布）；注意 McLafferty 重排产生的偶数/奇数质量规律（氮规则）。

3. **T-光氧化还原机理**：给出反应式（hv + Ru(bpy)₃²⁺ + 胺 + 底物）→ 要求写出 SET 循环。**陷阱**：区分**氧化淬灭**（PC* 氧化底物）和**还原淬灭**（PC* 还原底物），看底物的氧化电位。

4. **T-自由基环化**：给出多烯底物 + 自由基引发剂 → 要求预测环化产物。**陷阱**：5-exo-trig vs 6-endo-trig 的竞争——通常 5-exo 占优势（Baldwin 规则 + 轨道重叠）。
