## 二、📐 讲解脉络

> Weller 本章按「通性 → 分族详述 → 专题深入」三层结构组织。先以 §19.1–19.2 建立 d 区元素的整体图景（存在/提取/物理化学趋势），然后 §19.3–19.12 逐族（Group 3→12）展开，每族统一按 (a) 存在和用途 → (b) 二元化合物 → (c) 复合物 → (d) 配位络合物 → (e) 金属有机化合物 五段式讨论。中间穿插多金属氧酸盐（应用文段 19.1）、金属-金属键（应用文段 19.2）、环境毒理学（应用文段 19.3）、半导体颜料（应用文段 19.4）四个专题。

- **切入前置**：§19.1–19.2 先给骨架——d 金属 ≠ 过渡金属（IUPAC 定义）、亲氧/亲硫转变、3d/4d/5d 趋势对比、氧化态变化规律。这个前置段是理解后续各族差异的基础。
- **铺垫段落**：§19.2 的原子半径/电负性表（表 19.2）和氧化态表（表 19.3）是整章的高频引用数据；§19.3 第 3 族相对简单，为学生熟悉分族叙述模式做铺垫。
- **全章高潮**：§19.6（Cr/Mo/W）+ §19.8（Fe/Ru/Os）+ §19.10（Ni/Pd/Pt）——三者分别是前过渡/中过渡/后过渡的精华，也是竞赛中最常考的三个族。应用文段 19.2 的金属-金属键（四重键、五重键）是决赛级高频考点。
- **一笔带过**：§19.9（Co/Rh/Ir）、§19.11（Cu/Ag/Au）、§19.12（Zn/Cd/Hg）中的金属有机部分在竞赛中出现频率较低，可选择性阅读。
- **教学风格**：**结构-性质-用途三位一体**——每个化合物都给出结构图示 + 化学方程式 + 实际用途，重视应用文段（4 个）和例题（3 道）的教学功能。

## 六、关键图表和反应方程式

- d 区氧化态变化规律：同族高氧化态稳定性自上而下增大；最大氧化态峰值出现在系列中部
- 亲氧→亲硫转变：3d 左→右：氧化物矿 → 硫化物矿；4d/5d 亲硫性强
- 多酸合成通式：$6[MoO_4]^{2-} + 10H^+ \rightarrow [Mo_6O_{19}]^{2-} + 5H_2O$
- M—M 四重键电子组态：σ²π⁴δ²（d⁴-d⁴），五重键：σ²π⁴δ⁴（d⁵-d⁵）
- Irving-Williams 序列：Ba²⁺ < Sr²⁺ < Ca²⁺ < Mg²⁺ < Mn²⁺ < Fe²⁺ < Co²⁺ < Ni²⁺ < Cu²⁺ > Zn²⁺
