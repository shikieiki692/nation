# 二、教学逻辑锚点

## 2.1 本章在学科体系中的位置

酸碱滴定法与配位滴定法是分析化学中最重要的两类滴定分析方法，构成了定量分析的核心基础。酸碱滴定法基于质子转移反应，配位滴定法基于配位反应，两者共同构成了容量分析（滴定分析）的理论框架。

- **网课常缺失或薄弱**:

  - 分布分数的计算与应用
  - 对数图解法的原理与应用
  - 林邦误差公式的推导与应用
  - 配位滴定中的副反应系数（酸效应、配位效应、共存离子效应）
  - 条件稳定常数的计算与应用

  - 掩蔽法的系统分类与应用实例
- **本书价值**: 提供了完整的滴定分析理论框架，是竞赛中滴定计算题和实验设计题的核心依据

# 四、知识点提取清单

## P0 级（必须掌握）

### 4.1 酸碱滴定法基础

#### 4.1.1 活度与活度系数

- **活度**: $a_i = \gamma_i c_i$
- **德拜-休克尔公式**: $-\lg \gamma_i = 0.51 z_i^2 \left(\frac{\sqrt{I}}{1 + B\dot{a}\sqrt{I}}\right)$
- **离子强度**: $I = \frac{1}{2}\sum c_i z_i^2$

#### 4.1.2 酸碱质子理论

- 酸：能给出质子的物质；碱：能接受质子的物质
- 共轭酸碱对：$K_a \cdot K_b = K_w$
- 溶剂质子自递：$K_s = [SH_2^+][S^-]$

#### 4.1.3 质子平衡方程（PBE）

以 $(NH_4)_2HPO_4$ 为例：
$$[H^+] + [H_2PO_4^-] + 2[H_3PO_4] = [OH^-] + [PO_4^{3-}] + [NH_3]$$

**书写要点**: 选择参考水准（原始酸碱组分+水），得质子产物=失质子产物

#### 4.1.4 分布分数

**一元酸**:
$$\delta_{HA} = \frac{[H^+]}{[H^+] + K_a}, \quad \delta_{A^-} = \frac{K_a}{[H^+] + K_a}$$

**多元酸**（以 $H_2B$ 为例）:
$$\delta_{H_2B} = \frac{[H^+]^2}{[H^+]^2 + K_{a1}[H^+] + K_{a1}K_{a2}}$$
$$\delta_{HB^-} = \frac{K_{a1}[H^+]}{[H^+]^2 + K_{a1}[H^+] + K_{a1}K_{a2}}$$
$$\delta_{B^{2-}} = \frac{K_{a1}K_{a2}}{[H^+]^2 + K_{a1}[H^+] + K_{a1}K_{a2}}$$

### 4.2 H+浓度计算

#### 4.2.1 一元弱酸

- **精确式**: $[H^+] = \sqrt{K_a[HA] + K_w}$
- **近似式**（$K_ac \geq 10K_w$）: $[H^+] = \sqrt{K_a(c - [H^+])}$
- **最简式**（$K_ac \geq 10K_w$ 且 $c/K_a \geq 100$）: $[H^+] = \sqrt{K_ac}$

#### 4.2.2 两性物质（酸式盐）

- **近似式**: $[H^+] = \sqrt{\frac{K_{a1}(K_{a2}c + K_w)}{K_{a1} + c}}$
- **最简式**（$K_{a2}c \geq 10K_w$ 且 $c \geq 10K_{a1}$）: $[H^+] = \sqrt{K_{a1}K_{a2}}$

#### 4.2.3 弱酸弱碱盐

$$[H^+] = \sqrt{\frac{c_{HA}}{c_B}K_{HA}K_{HB}}$$

### 4.3 缓冲溶液

#### 4.3.1 缓冲溶液pH计算

- **精确式**: $[H^+] = K_a\frac{c_{HB} - [H^+] + [OH^-]}{c_{B^-} + [H^+] - [OH^-]}$
- **最简式**: $pH = pK_a + \lg\frac{c_{B^-}}{c_{HB}}$

#### 4.3.2 缓冲容量

$$\beta = 2.30\frac{cK_a[H^+]}{([H^+] + K_a)^2} = 2.30\delta_0\delta_1c$$

- 最大缓冲容量: $\beta_{max} = 0.575c$（当 $[H^+] = K_a$ 时）
- 有效缓冲范围: $pK_a \pm 1$

### 4.4 酸碱指示剂

- **变色原理**: 酸碱式结构不同颜色
- **理论变色范围**: $pH = pK_a \pm 1$
- **理论变色点**: $pH = pK_a$（$[In^-]/[HIn] = 1$）
- **常用指示剂**:
  - 甲基橙（MO）: $pK_a = 3.4$，变色范围 $3.1 \sim 4.4$（红→黄）
  - 酚酞（PP）: 变色范围 $8.0 \sim 9.6$（无色→红）
  - 甲基红: 变色范围 $4.4 \sim 6.2$

### 4.5 酸碱滴定曲线

#### 4.5.1 强碱滴定强酸

- 突跃范围: $pH = 4.30 \sim 9.70$（0.1 mol/L）
- 指示剂: 甲基红、酚酞

#### 4.5.2 强碱滴定弱酸

- 化学计量点: $[OH^-] = \sqrt{\frac{K_w}{K_a}c}$
- 突跃范围: 碱性范围
- 准确滴定判据: $cK_a \geq 10^{-8}$

#### 4.5.3 多元酸滴定

- 分步滴定判据: $K_{a1}/K_{a2} > 10^5$
- 第一化学计量点: $[H^+] = \sqrt{K_{a1}K_{a2}}$（当 $cK_{a1} \geq 10^{-8}$）

### 4.6 配位滴定法基础

#### 4.6.1 EDTA性质

- EDTA（$H_4Y$）为六元酸（可形成 $H_6Y^{2+}$）
- 常用二钠盐 $Na_2H_2Y$
- 与金属离子配位比多为 1:1
- 稳定常数 $K_{MY}$ 很大

#### 4.6.2 配位平衡常数

- **逐级稳定常数**: $K_{稳1}, K_{稳2}, \dots$
- **累积稳定常数**: $\beta_n = K_{稳1} \cdot K_{稳2} \cdots K_{稳n}$
- **分布分数**: $\delta_{ML_n} = \frac{\beta_n[L]^n}{1 + \sum_{i=1}^{n}\beta_i[L]^i}$

### 4.7 副反应系数与条件稳定常数

#### 4.7.1 EDTA的酸效应系数

$$\alpha_{Y(H)} = 1 + \sum_{i=1}^{n}\beta_i^H[H^+]^i$$

#### 4.7.2 共存离子效应系数

$$\alpha_{Y(N)} = 1 + K_{NY}[N]$$

#### 4.7.3 金属离子配位效应系数

$$\alpha_{M(L)} = 1 + \sum_{i=1}^{n}\beta_i[L]^i$$

#### 4.7.4 条件稳定常数

$$\lg K_{MY}' = \lg K_{MY} - \lg \alpha_M - \lg \alpha_Y$$

### 4.8 配位滴定曲线

- 化学计量点: $pM_{sp} = \frac{1}{2}(pc_M^{sp} + \lg K_{MY}')$
- 影响突跃因素: $K_{MY}'$ 和 $c_M$

### 4.9 金属离子指示剂

| 指示剂 | 适用pH | 颜色变化 | 直接滴定离子 |
|---|---|---|---|
| 铬黑T (EBT) | 8~10 | 蓝→红 | Mg2+、Zn2+、Cd2+、Pb2+ |
| 二甲酚橙 (XO) | <6 | 亮黄→红 | Bi3+、Th4+、Zn2+、Pb2+ |
| 钙指示剂 (NN) | 12~13 | 蓝→红 | Ca2+ |
| PAN | 2~12 | 黄→紫红 | Cu2+、Ni2+、Pb2+、Cd2+、Zn2+ |

- **封闭**: 指示剂与金属离子形成过于稳定的配合物
- **僵化**: 指示剂与金属离子形成的配合物溶解度小，置换缓慢

## P1 级（重要拓展）

### 4.10 酸碱滴定终点误差

#### 4.10.1 林邦误差公式（强碱滴定强酸）

$$E_t = \frac{10^{\Delta pH} - 10^{-\Delta pH}}{\sqrt{\frac{1}{K_w}} \cdot c_{HCl}^{ep}} \times 100\%$$

#### 4.10.2 林邦误差公式（强碱滴定弱酸）

$$E_t = \frac{10^{\Delta pH} - 10^{-\Delta pH}}{\sqrt{\frac{K_a}{K_w}c_{HA}^{sp}}} \times 100\%$$

#### 4.10.3 多元酸滴定误差

第一终点: $E_{t1} = \frac{10^{\Delta pH} - 10^{-\Delta pH}}{\sqrt{K_{a1}/K_{a2}}} \times 100\%$

### 4.11 配位滴定终点误差

$$E_t = \frac{10^{\Delta pM'} - 10^{-\Delta pM'}}{\sqrt{K_{MY}'c_M^{sp}}} \times 100\%$$

### 4.12 准确滴定与分别滴定判别式

- **准确滴定**: $\lg(c_M^{sp}K_{MY}') \geq 6$（$E_t \leq 0.1\%$，$\Delta pM' = \pm 0.2$）
- **分别滴定**: $\Delta\lg(Kc) \geq 5$（$E_t \leq 0.3\%$）

### 4.13 酸度控制

- **最高酸度**: 由 $K_{MY}'$ 要求确定，$\lg \alpha_{Y(H)} = \lg K_{MY} - \lg K_{MY}'$
- **最低酸度（水解酸度）**: 由金属离子氢氧化物溶度积确定

### 4.14 掩蔽法

| 掩蔽方法 | 原理 | 实例 |
|---|---|---|
| 配位掩蔽 | 加入配位剂与干扰离子形成稳定配合物 | F-掩蔽Al3+，CN-掩蔽Cu2+、Zn2+ |
| 沉淀掩蔽 | 加入沉淀剂使干扰离子沉淀 | OH-掩蔽Mg2+（形成Mg(OH)2） |
| 氧化还原掩蔽 | 改变干扰离子价态 | Fe3+→Fe2+（抗坏血酸还原） |

### 4.15 滴定方式

| 滴定方式 | 适用情况 | 实例 |
|---|---|---|
| 直接滴定 | 反应快、有合适指示剂 | 水的硬度测定 |
| 返滴定 | 反应慢、无合适指示剂 | Al3+的测定 |
| 置换滴定 | 被测离子与EDTA不稳定 | Ag+的测定（置换Ni2+） |
| 间接滴定 | 被测物不与EDTA配位 | SO42-的测定 |

# 五、例题提取

## 例题 1：质子平衡方程书写（P0）

**题目**: 写出 $(NH_4)_2HPO_4$ 水溶液的质子平衡方程。

**考点抽象**: 多元酸碱盐的PBE书写——选择正确参考水准是关键。

**解答要点**:
- 参考水准: $NH_4^+$、$HPO_4^{2-}$、$H_2O$
- 得质子产物: $H^+$、$H_2PO_4^-$、$H_3PO_4$（系数2）
- 失质子产物: $OH^-$、$PO_4^{3-}$、$NH_3$
- PBE: $[H^+] + [H_2PO_4^-] + 2[H_3PO_4] = [OH^-] + [PO_4^{3-}] + [NH_3]$

## 例题 2：H+浓度计算（P0）

**题目**: 计算 $0.10\mathrm{mol\cdot L^{-1}}$ 乳酸溶液的pH。已知 $K_a = 1.4\times 10^{-4}$。

**考点抽象**: 一元弱酸H+浓度计算——判断使用最简式的条件。

**解答**:
- $K_ac = 1.4\times 10^{-5} > 10K_w$，且 $c/K_a = 714 > 100$
- 采用最简式: $[H^+] = \sqrt{K_ac} = \sqrt{1.4\times 10^{-4}\times 0.10} = 3.7\times 10^{-3}\mathrm{mol\cdot L^{-1}}$
- $pH = 2.43$

## 例题 3：缓冲溶液pH计算（P0）

**题目**: 计算 $0.10\mathrm{mol\cdot L^{-1}}NH_4Cl - 0.20\mathrm{mol\cdot L^{-1}}NH_3$ 缓冲溶液的pH。

**考点抽象**: 缓冲溶液pH计算——最简式的应用。

**解答**:
- $pK_a' = pK_w - pK_b = 14.00 - 4.74 = 9.26$
- $pH = pK_a' + \lg\frac{c_{NH_3}}{c_{NH_4^+}} = 9.26 + \lg\frac{0.20}{0.10} = 9.56$

## 例题 4：条件稳定常数计算（P0）

**题目**: 计算在pH 5.00的0.10 mol/L AlY溶液中，游离F-浓度为0.10 mol/L时AlY的条件稳定常数。

**考点抽象**: 条件稳定常数计算——考虑酸效应和配位效应。

**解答**:
- 查表: pH 5.00时 $\lg\alpha_{Y(H)} = 6.45$
- 计算 $\alpha_{Al(F)} = 10^{9.95}$（通过逐级配位计算）
- $\lg K_{AlY}' = \lg K_{AlY} - \lg\alpha_{Y(H)} - \lg\alpha_{Al(F)} = 16.3 - 6.45 - 9.95 = -0.1$
- $K_{AlY}' = 0.8$，说明AlY已被氟化物破坏

## 例题 5：配位滴定终点误差（P1）

**题目**: 在pH 10.00的氨性溶液中，以铬黑T为指示剂，用0.020 mol/L EDTA滴定0.020 mol/L Ca2+，计算终点误差。

**考点抽象**: 配位滴定终点误差——林邦误差公式的应用。

**解答**:
- $\lg K_{CaY}' = 10.69 - 0.45 = 10.24$
- $pCa_{sp}' = \frac{1}{2}(2.00 + 10.24) = 6.12$
- EBT的 $\lg K_{Ca-EBT}' = 5.4 - 1.6 = 3.8$（考虑酸效应）
- $pCa_{ep} = 3.8$，$\Delta pCa = 3.8 - 6.12 = -2.3$
- $E_t = \frac{10^{-2.3} - 10^{2.3}}{\sqrt{10^{10.24}\times 10^{-2}}} \times 100\% = -1.5\%$

## 例题 6：分别滴定判别（P1）

**题目**: 用0.020 mol/L EDTA滴定0.020 mol/L Zn2+和0.10 mol/L Mg2+混合溶液中的Zn2+，能否准确滴定？

**考点抽象**: 分别滴定判别式的应用。

**解答**:
- $\lg(K_{ZnY}c_{Zn}^{sp}) - \lg(K_{MgY}c_{Mg}^{sp}) = (16.5 - 2.0) - (8.7 - 1.0) = 7.1 > 6$
- 故能准确滴定Zn2+，Mg2+不干扰

## 例题 7：混合碱分析（P1）

**题目**: 用双指示剂法测定NaOH和Na2CO3混合碱。以酚酞为指示剂消耗HCl体积V1，以甲基橙为指示剂继续滴定消耗HCl体积V2。推导各组分含量计算公式。

**考点抽象**: 双指示剂法的原理与计算——竞赛常考实验设计题。

**解答**:
- $w_{Na_2CO_3} = \frac{\frac{1}{2}c_{HCl}\times 2V_2 \times M_{Na_2CO_3}}{m_s} \times 100\%$
- $w_{NaOH} = \frac{c_{HCl}\times(V_1 - V_2)\times M_{NaOH}}{m_s} \times 100\%$

| 配位滴定终点误差 | $E_t = \frac{10^{\Delta pM'} - 10^{-\Delta pM'}}{\sqrt{K_{MY}'c_M^{sp}}} \times 100\%$ | 配位滴定 |
| 准确滴定判别式 | $\lg(c_M^{sp}K_{MY}') \geq 6$ | 单一离子 |
| 分别滴定判别式 | $\Delta\lg(Kc) \geq 5$ | 混合离子 |

# 十、易错点回流

## 10.1 常见概念混淆

| 易混淆点 | 正确区分 |
|---|---|
| 分析浓度 vs 平衡浓度 | 分析浓度是总浓度，平衡浓度是某型体的实际浓度 |
| 分布分数 vs 摩尔分数 | 分布分数是某型体平衡浓度占总浓度的分数 |
| 活度 vs 浓度 | 活度=活度系数×浓度，稀溶液中近似相等 |
| 稳定常数 vs 条件稳定常数 | 条件稳定常数考虑了副反应的影响 |
| 酸效应 vs 配位效应 | 酸效应是H+对配体的影响，配位效应是其他配体对金属离子的影响 |

## 10.2 计算常见错误

1. **PBE书写**: 注意参考水准的选择，同一物质只能选一种型体作为参考水准
2. **近似条件判断**: 使用最简式前必须验证适用条件
3. **缓冲溶液pH**: 当缓冲组分浓度较小时，不能使用最简式
4. **条件稳定常数**: 注意区分 $\alpha_{Y(H)}$、$\alpha_{Y(N)}$ 和总 $\alpha_Y$
5. **分别滴定判别**: 注意是 $\Delta\lg(Kc) \geq 5$，不是 $\Delta\lg K \geq 5$
6. **终点误差计算**: 注意 $\Delta pM$ 的正负号

## 10.3 实验设计题陷阱

- 酸碱滴定中，指示剂选择必须考虑突跃范围
- 配位滴定中，酸度控制是关键，需同时考虑最高酸度和水解酸度
- 掩蔽剂的选择要考虑pH适用范围
- 返滴定法中，返滴定剂与被测离子的稳定性关系要恰当
- 混合碱分析中，双指示剂法的V1和V2关系要正确理解
