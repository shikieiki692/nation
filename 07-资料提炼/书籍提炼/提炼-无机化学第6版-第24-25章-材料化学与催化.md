## 四、高 ROI 内容详录（按专题归类）

### 专题 A：复合氧化物结构（24.6）

**核心内容**：
- **刚玉结构**（α-Al₂O₃）：hcp O²⁻，阳离子占 2/3 八面体穴。V₂O₃ 有金属-绝缘体转变。
- **ReO₃ 结构**：ReO₆ 八面体共顶点，与钙钛矿密切相关（移除 A 位离子）。WO₃ 有扭曲。
- **尖晶石**（AB₂O₄）：
  - 正常尖晶石 A[Al₂]O₄（A 四面体，B 八面体）
  - 反尖晶石 B[AB]O₄
  - **LFSE 决定分布**：d⁵ 高自旋 Fe³⁺ 无 LFSE，故常形成反尖晶石；d³ Cr³⁺ 强烈偏好八面体 → 正常尖晶石
  - 例题 24.3：MnCr₂O₄ 为正常尖晶石（Cr³⁺ d³ 大 LFSE，Mn²⁺ d⁵ 无 LFSE）
- **钙钛矿**（ABX₃）：
  - 理想立方：A 在顶点，B 在体心，O 在面心
  - BaTiO₃ 铁电性：Ti 偏离中心 → 自发极化，T_c = 120°C
  - YBa₂Cu₃O₇ 高温超导：CuO₂ 面 + Cu─O 链

**竞赛考点**：根据离子半径和电子构型判断尖晶石类型；钙钛矿结构推断（Goldschmidt 容差因子）；铁电/压电原理

**目标 KP**：[[材料化学]] — 新增 §3.2「典型无机晶体结构」

---

### 专题 B：层状化合物与嵌入化学（24.9）

**核心内容**：
- **MS₂ 层状结构**：TaS₂（CdI₂ 型，八面体）、NbS₂（三棱柱）、MoS₂（润滑性）
- **嵌入（intercalation）**：客体分子/离子进入层间，层间距增大但层内结构不变
  - 碱金属嵌入：n-BuLi + TiS₂ → Li_xTiS₂（锂电池原理！）
  - 有机分子嵌入：吡啶、胺类进入层间
- **CVT 法提纯**：TaS₂ + 2 I₂(g) ⇌ TaI₄(g) + S₂(g)，温度梯度驱动转移

**竞赛考点**：嵌入 vs 插入（insertion）区别；层状化合物作为固体润滑剂和电极材料

**目标 KP**：[[材料化学]] — 新增 §3.5「层状化合物与嵌入化学」

---

### 专题 C：沸石与分子筛（24.11）

**核心内容**：
- **沸石通式**：(Mⁿ⁺)_{x/n}[(AlO₂)_x(SiO₂)_{2−x}]·mH₂O
- **Lowenstein 规则**：溶液合成沸石中，AlO₄ 四面体不共顶点（无 Al─O─Al）
- **孔径调控**：模板剂决定孔径 → UTD-1（14元环）、CIT-5 等超大孔沸石
- **ALPO**：AlO₄/PO₄ 1:1 交替，电中性；掺杂 Co/Mn 得氧化还原活性
- **工业应用**：
  - 分子筛分离（N₂/O₂ 分离基于四极矩差异）
  - 洗涤剂助剂（Ca²⁺/Mg²⁺ 交换）
  - 放射性核素捕获（¹³⁷Cs, ⁹⁰Sr）

**竞赛考点**：沸石孔径与择形催化；Si/Al 比与酸性的关系；离子交换选择性

**目标 KP**：[[材料化学]] — 新增 §3.6「沸石与分子筛」

---

### 专题 D：半导体与光催化（24.17–24.19）

**核心内容**：
- **TiO₂ 光催化**：
  - 锐钛矿带隙 3.2 eV，金红石 3.1 eV（紫外吸收）
  - 光照 → e⁻ + h⁺ → H₂O 分解为 H₂ + O₂（需 >1.23 eV）
  - 效率瓶颈：光吸收低、电子-空穴复合快、表面积小
  - 改性：掺杂 N/S（缩小带隙）、纳米化（增大表面积）、助催化剂（NiO、Pt）
- **p-n 结**：n 型 Si（掺杂 P）+ p 型 Si（掺杂 B）→ 整流、LED、太阳能电池
- **掺杂调控**：
  - p 型：13 族掺杂（B、Al）→ 空穴导电
  - n 型：15 族掺杂（P、As）→ 电子导电

**竞赛考点**：带隙计算与吸收波长；p/n 型判断；光催化原理图解

**目标 KP**：[[材料化学]] — 新增 §3.7「半导体与光催化」

### 专题 E：固体电解质（24.4）

**核心内容**：
- **YSZ（Y₂O₃ 稳定的 ZrO₂）**：Y³⁺ 取代 Zr⁴⁺ 产生氧空位 → O²⁻ 离子导体
- **NASICON**：Na₁₊ₓZr₂SiₓP₃₋ₓO₁₂，三维 Na⁺ 传导通道
- **SOFC**：固体氧化物燃料电池，O²⁻ 从阴极迁移至阳极

**竞赛考点**：缺陷化学与离子电导关系；掺杂产生空位的电荷补偿机制

**目标 KP**：[[材料化学]] — 新增 §3.4「离子导体与固体电解质」

### 专题 F：烯烃复分解（25.3）

**核心内容**：
- **反应类型**：ROMP / ADMET / RCM / ROM / CM（交叉复分解）
- **Grubbs 催化剂演进**：
  - 第一代：RuCl₂(PCy₃)₂=CHPh（1992）
  - 第二代：一个 PCy₃ 换为 NHC（活性更高，官能团容忍度更好）
  - 第三代：膦配体换为吡啶（引发更快）
- **Schrock 催化剂**：Mo/W 次烷基络合物（更早，1980s），可用手性版本控制立体化学
- **机理**：金属环丁烷中间体；Grubbs 中 PCy₃ 解离为关键步骤

**竞赛考点**：催化剂代际差异；反应类型选择（RCM 成环 vs ROMP 开环聚合）；金属环丁烷中间体验证

**目标 KP**：[[烯烃复分解反应]] — 全面升级，补充催化剂结构细节和机理

### 专题 G：钯催化 C─C 偶联（25.8）

**核心内容**：
- **Heck 偶联**：R─X + 烯烃 → R─烯烃，Pd(0)/Pd(II) 循环
- **Suzuki 偶联**：R─B(OH)₂ + R'─X → R─R'，碱促进，容忍度高
- **Stille 偶联**：R─SnR₃ + R'─X → R─R'
- **Sonogashira 偶联**：端炔 + R'─X → 炔烃，Pd(0)/Cu(I) 双催化
- **共同机理**：氧化加成 → 配位/插入 → 还原消除；两碎片需顺位

**竞赛考点**：偶联反应选择（哪种底物用哪种）；催化剂设计（双膦螯合配体控制顺位）

**目标 KP**：建议挂载到 [[金属有机与羰基化学]] 或独立为「钯催化偶联反应」专题素材

---

### 专题 H：工业催化经典（25.12, 25.13, 25.15）

**核心内容**：
- **合成氨**（Haber-Bosch）：Fe 基催化剂 + Al₂O₃/K 助剂，400°C / 200 atm，N₂ 解离为决速步
- **接触法 SO₂→SO₃**：V₂O₅/K₂S₂O₇ 熔融催化剂，决速步为 V(IV)→V(V) 氧化
- **Fischer-Tropsch**：CO + 2H₂ → −CH₂− + H₂O，Fe/Co 催化剂，产物分布可控

**竞赛考点**：热力学-动力学权衡（放热反应为什么高温？因为动力学）；催化剂组成与选择性

---

### 专题 I：聚合催化（25.18）

**核心内容**：
- **Ziegler-Natta**：TiCl₄/AlEt₃ → TiCl₃ 表面活性位，Cossee-Arlman 机理（配位-插入-β-H 消除）
- **Kaminsky 茂金属**：Cp₂ZrCl₂/MAO，均相，立构规正度可控
- **Philips 催化剂**：CrO₃/SiO₂，无需烷基化试剂

**竞赛考点**：Cossee-Arlman 机理步骤；立构规正度（全同/间同/无规）控制
