## 二、📐 讲解脉络

> 主题17从反应速率的定义和测量出发，建立积分速率方程（零级/一级/二级），引入弛豫法测对峙反应速率常数，然后通过Arrhenius公式描述温度依赖性，进而深入反应机理的推导方法（稳态近似、速控步、预平衡），最后以单分子反应、聚合反应和酶催化作为机理示例，并以光化学作为光激发态动力学的延伸。主题18从微观层面深化：碰撞理论给出Arrhenius参数的物理图像，过渡态理论通过Eyring方程连接热力学与动力学，分子束和势能面则进入真正的反应动态学。主题19转向表面过程：从物理吸附/化学吸附的本质区别，到Langmuir/BET吸附等温式，再到多相催化的Langmuir-Hinshelwood/Eley-Rideal机理，最后以电极过程（Butler-Volmer方程、超电势、伏安法）收尾。

- **切入前置**：主题17A以反应进程监测的实验方法（光谱、电导、压力、色谱）为起点，强调恒温条件是动力学实验的首要前提。
- **全章高潮**：主题17E–17F将稳态近似、速控步、预平衡三种数学工具应用于复杂机理推导，是从"实验速率方程"到"分子机理"的关键跃迁；主题19C将表面吸附等温式与催化反应速率结合，体现多相催化动力学的核心逻辑。
- **理论深化**：主题18C的Eyring方程（$k_r = \kappa(kT/h)K^\ddagger$）是决赛阶段连接热力学活化参数与速率常数的最高层次理论；主题19D的Butler-Volmer方程是电化学动力学的核心。
- **教学风格**：**严格推导 / 示例驱动** — 每个公式都配以"如何完成？"专栏展示推导步骤，例题数据完整，自测题丰富。

## 六、关键图表和反应方程式

### 核心公式（竞赛必会）
- **一级反应积分速率方程**：$\ln[A] = \ln[A]_0 - k_r t$；$t_{1/2} = \ln 2 / k_r$
- **二级反应积分速率方程**：$1/[A] - 1/[A]_0 = k_r t$；$t_{1/2} = 1/(k_r[A]_0)$
- **弛豫时间**：$1/\tau = k_1 + k_{-1}$（一级对峙反应）
- **Arrhenius方程**：$\ln k_r = \ln A - E_a/(RT)$；$E_a = RT^2 \cdot d\ln k_r/dT$
- **活化能与反应热关系**：$\Delta H = E_{a,正} - E_{a,逆}$
- **Lindemann机理**：$\mathrm{A} + \mathrm{A} \rightleftharpoons \mathrm{A}^* + \mathrm{A}$（活化），$\mathrm{A}^* \rightarrow \mathrm{P}$（分解）；高压极限一级，低压极限二级
- **Michaelis-Menten方程**：$v_0 = V_{max}[S]/(K_M + [S])$；$1/v_0 = (K_M/V_{max})(1/[S]) + 1/V_{max}$
- **碰撞理论速率常数**：$k_r = P\sigma(8kT/\pi\mu)^{1/2}N_A e^{-E_a/RT}$
- **Eyring方程**：$k_r = \kappa(kT/h)K^\ddagger$；$\Delta^\ddagger G = -RT\ln K^\ddagger$
- **Stern-Volmer猝灭方程**：$\Phi_0/\Phi = 1 + k_q\tau_0[Q] = 1 + K_{SV}[Q]$
- **Langmuir等温式**：$\theta = \alpha p/(1 + \alpha p)$；$V/V_\infty = \alpha p/(1 + \alpha p)$
- **BET等温式**：$V/V_{mon} = cz/(1-z) \cdot 1/(1-(1-c)z)$，其中 $z=p/p^*$
- **LH机理速率**：$v = k_r\theta_A\theta_B = k_r\alpha_A\alpha_B p_A p_B / (1+\alpha_A p_A + \alpha_B p_B)^2$
- **Butler-Volmer方程**：$j = j_0(e^{(1-\alpha)f\eta} - e^{-\alpha f\eta})$，$f=F/RT$
- **Tafel方程**：$\ln j = \ln j_0 + (1-\alpha)f\eta$（阴极，$\eta<0$）
