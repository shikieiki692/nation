# 书籍提炼·源书-讲义映射总表

> **背景**：本仓库的书籍提炼最初是为 **KP 知识点填充** 而作，后逐步转向 **学生讲义内容支撑**。两类用途对提炼的要求不同：
> - **KP 填充**：需要核心概念定义、公式、数据表（框架即可）
> - **讲义支撑**：需要教学叙事、推导过程、量化对比、案例拓展（需要血肉）
>
> 本表从「讲义支撑」视角重新评估每份提炼的实际可用性。

## 四、Agent 使用指南

### 4.1 查找流程

```
要填充讲义内容
    ↓
查「源书-讲义映射表」（本文件 §一）
    ↓
找到对应该讲义主题的源书
    ↓
检查密度评级：
  🟢 高密度 → 直接读取提取文件，做格式转换（教材→教学语言）
  🟡 中密度 → 读取+补充叙事和例证
  🔴 低密度 → 跳过此源，或标记"需重新提炼"
    ↓
从提取中抽取：
  表格 / 数据 → 直接嵌入讲义
  推导过程 → 重写为教学叙事
  例题 → 保留或改编
  教学洞察 → 转化为 🧠 / ⚠️ / 💡 callout
```
