## 五、入库文件命名规范

```
04-题库/教材习题/ABOC/
├── 题-{NNN}-ABOC-Ch{M}-{描述}.md     # Ch.1-9 章末习题
├── 题-{NNN}-ABOC-FT0-{代号}.md       # FINAL TEST 0
└── 题-{NNN}-ABOC-FT1-P{P}-EX{N}-{描述}.md  # FINAL TEST 1
```

**frontmatter 标准**（参见 [[模板-题目]]）：
- `type: 题目`
- `exam_stage: 初赛` / `决赛`（FT0/FT1 按决赛标注）
- `source: ABOC 第{M}章 {类型}（ARX's Basic Organic Chemistry 第3版）`
- `subject/module: 有机化学`
- `submodule: {具体子模块}`
- `question_type: {选择题/计算题/推断题/机理书写题/合成设计题}`
- `difficulty: 1-5`
- `teaching_level: 入门/巩固/拓展/竞赛拔高`
- `syllabus_codes: ["{考纲编号}"]`
- `knowledge_points: [{KP1}, {KP2}]`
- `tags: [化竞, ABOC, 有机化学]`
- `aliases: [ABOC-{代号}]`
