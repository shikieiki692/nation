# 生物化学

> 生物化学研究生命体系中的化学过程，核心在于理解生物大分子的结构、功能及其相互作用。本章内容基于 Clayden 第42章全面提炼，涵盖糖类、氨基酸与蛋白质、核酸、脂类、辅酶与酶催化、代谢途径等竞赛常考内容。

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[生物化学]]

## 一、定义

**生物化学**（Biochemistry）是研究生物体内化学过程的学科，关注生物大分子（糖类、蛋白质、核酸、脂类）的结构与功能，以及维持生命所需的代谢途径和能量转换。从有机化学竞赛角度，重点掌握生物大分子的化学结构特征、酶催化的有机反应机理，以及代谢途径中的关键化学转化。

## 三、核心原理

### 3.1 糖类化学

#### 3.1.1 环状结构与开链平衡

- **葡萄糖**：在水溶液中以三种形式平衡共存：
  - 开链醛式（约0.02%）
  - α-D-吡喃葡萄糖（约36%）
  - β-D-吡喃葡萄糖（约64%）
- **核糖**：可采取五元呋喃糖或开链形式；RNA 中为呋喃糖

#### 3.1.2 异头碳效应（Anomeric Effect）

$$\text{环内氧的孤对电子} \rightarrow \text{直立取代基的 } \sigma^* \text{ 轨道：} n \rightarrow \sigma^* \text{ 相互作用}$$

- **定义**：在吡喃糖等缩醛体系中，异头碳上的吸电子取代基（如 OMe、OH）倾向于采取直立键构象，而非预期的平伏键
- **机理**：环内氧的孤对电子与直立取代基的 σ* 反键轨道发生给电子相互作用，稳定化直立构象
- **结果**：
  - 对 D-葡萄糖：β-异头体（平伏 OH）更稳定
  - 对某些修饰糖：α-异头体（直立 OR）可能更稳定
- **竞赛考点**：解释缩醛形成的热力学立体选择性；丙酮缩醛的顺式/反式稠合选择性

#### 3.1.3 糖苷（Glycosides）

- **定义**：糖的异头碳羟基与另一分子的羟基/巯基/氨基脱水形成的缩醛/缩酮类化合物
- **分类**：O-糖苷、S-糖苷、N-糖苷
- **实例**：
  - 蔗糖（葡萄糖 α-1,2-果糖糖苷）
  - 花青素（β-葡萄糖苷）
  - 黑芥子苷（S-糖苷）
- **水解**：糖苷酶选择性水解糖苷键

### 3.2 氨基酸与蛋白质

#### 3.2.1 20种天然氨基酸

- **通用结构**：$\text{H}_2\text{N-CH(R)-CO}_2\text{H}$
- **D/L 与 R/S**：天然蛋白质氨基酸均为 L-构型（除甘氨酸无手性）；但细菌细胞壁含 D-氨基酸（如 D-Ala）
- **特殊氨基酸**：
  - 脯氨酸（Pro）：仲胺，形成亚胺而非典型酰胺
  - 羟基脯氨酸（Hyp）：胶原特征氨基酸
  - 半胱氨酸（Cys）：含巯基，可形成二硫键

#### 3.2.2 肽键的刚性

$$\text{肽键：} \quad \text{-CO-NH-}$$

- **共轭效应**：酰胺键 C=O 与 N 孤对电子共轭 → 平面刚性结构
- **C-N 键部分双键性质**：键长约 1.33 Å（介于单键 1.47 Å 和双键 1.27 Å 之间）
- **反式构型主导**：约 99% 的肽键为反式（脯氨酸例外，顺式比例约 10%）
- **结构意义**：肽键刚性限制了蛋白质主链的自由度，是蛋白质二级结构（α-螺旋、β-折叠）形成的基础

#### 3.2.3 蛋白质折叠驱动力

- **疏水效应**：非极性侧链避开水相，聚集于蛋白质内部
- **氢键**：主链 C=O 与 N-H 之间形成规则的氢键网络（α-螺旋、β-折叠）
- **二硫键**：半胱氨酸残基之间形成共价交联，稳定三级结构
- **离子相互作用**：带相反电荷的侧链之间的盐桥

#### 3.2.4 谷胱甘肽（Glutathione）

- **结构**：γ-谷氨酰半胱氨酰甘氨酸（非典型 γ-肽键）
- **功能**：细胞内主要抗氧化剂
- **机理**：巯基（-SH）氧化为二硫化物（-S-S-），清除活性氧自由基

### 3.3 核酸

#### 3.3.1 核苷酸结构

- **组成**：碱基（嘌呤/嘧啶）+ 五碳糖（核糖/脱氧核糖）+ 磷酸
- **碱基配对**：
  - A-T（DNA）/ A-U（RNA）：两个氢键
  - G-C：三个氢键 → GC 含量越高，DNA 越稳定

#### 3.3.2 RNA 的碱性水解

- **机理**：2'-OH 分子内进攻相邻的 3'-磷酸酯 → 形成 2',3'-环状磷酸酯中间体 → 水解
- **几何要求**：需要顺式稠环交点构型
- **意义**：RNA 在碱性条件下不稳定，而 DNA（无 2'-OH）稳定

#### 3.3.3 环状核苷酸（cAMP）

- **结构**：3',5'-环状腺苷一磷酸
- **功能**：第二信使，参与细胞信号转导
- **立体化学**：糖环采取特定构象以形成磷酸二酯环

### 3.4 脂类

#### 3.4.1 甘油酯与磷脂

- **甘油三酯**：甘油 + 三分子脂肪酸
- **磷脂**：甘油 + 两分子脂肪酸 + 磷酸 + 极性头基
- **胶束形成**：磷脂分子在水中自组装，疏水尾向内，亲水头向外

#### 3.4.2 皂化反应

$$\text{甘油三酯} + 3\,\text{NaOH} \rightarrow \text{甘油} + 3\,\text{脂肪酸钠（肥皂）}$$

- 皂化值：皂化 1g 油脂所需 KOH 的毫克数，反映油脂的平均分子量

### 3.5 辅酶与生物氧化还原

#### 3.5.1 NAD⁺/NADH 与 NADP⁺/NADPH

- **结构**：烟酰胺腺嘌呤二核苷酸（磷酸）
- **机理**：吡啶鎓环接受负氢（$\text{H}^- + 2\text{e}^-$）→ 1,4-二氢吡啶
- **功能**：
  - NAD⁺/NADH：主要参与分解代谢（氧化）
  - NADP⁺/NADPH：主要参与合成代谢（还原）
- **与 NaBH₄ 的类比**：NADH 是"天然 NaBH₄"，但酶催化控制对映面选择性

#### 3.5.2 吡哆醛磷酸（PLP）

- **结构**：吡啶衍生物辅酶
- **参与的三种关键反应**：
  1. **还原胺化**：酮酸 → 氨基酸
  2. **转氨**：氨基酸₁ + 酮酸₂ ⇌ 酮酸₁ + 氨基酸₂
  3. **脱羧**：氨基酸 → 胺 + CO₂
- **共同机理**：吡哆醛与底物形成亚胺（Schiff 碱）→ 电子推拉促进 C-H/C-COOH 键断裂 → 酶控制对映面选择性

#### 3.5.3 辅酶 A（CoA）

- **结构**：泛酸 + 巯基乙胺 + 3'-磷酸 ADP
- **硫醇酯特性**：
  - 共轭差（硫的 3p 轨道与羰基 π* 重叠差）→ 更易烯醇化和水解
  - 是脂肪酸合成/聚酮合成的基本构建单元
- **核心反应**：
  - 乙酰 CoA + 丙二酰 CoA → Claisen 缩合（脂肪酸合成）
  - 乙酰 CoA 进入柠檬酸循环

### 3.6 酶催化的经典有机反应

#### 3.6.1 醛缩酶催化的羟醛反应

- **机理**：赖氨酸残基与磷酸二羟基丙酮（DHAP）形成亚胺 → 烯胺 → 进攻甘油醛-3-磷酸的醛基 → syn 羟醛产物
- **酶控制**：活性位点的几何约束决定了立体化学（syn vs anti）

#### 3.6.2 磷酸烯醇丙酮酸（PEP）

- **结构**：烯醇磷酸酯
- **高能磷酸键**：水解 ΔG°' ≈ -61.9 kJ/mol
- **作用**：糖酵解中的高能磷酸供体；丙酮酸激酶催化 PEP → 丙酮酸 + ATP

#### 3.6.3 柠檬酸合成酶催化的反应

- **反应**：乙酰 CoA + 草酰乙酸 + H₂O → 柠檬酸 + CoA-SH
- **机理**：乙酰 CoA 烯醇盐进攻草酰乙酸的酮基 → 羟醛型 C-C 键形成
- **立体化学**：酶活性位点控制，产生特定立体构型的柠檬酸

## 四、关键结论与公式

### 4.1 糖类核心公式

**变旋现象**：
$$[\alpha]_D^{20} = \frac{[\alpha]_{\alpha} \cdot [\alpha]_{\beta} \cdot K_{eq}}{[\alpha]_{\alpha} + [\alpha]_{\beta} \cdot K_{eq}}$$

- 葡萄糖：$[\alpha]_D$（α）= +112°，$[\alpha]_D$（β）= +18.7°，平衡混合物 = +52.7°

### 4.2 氨基酸等电点

$$\text{pI} = \frac{\text{p}K_{a1} + \text{p}K_{a2}}{2} \quad \text{（中性氨基酸）}$$

- 酸性氨基酸：$\text{pI} = (\text{p}K_{a1} + \text{p}K_{R}) / 2$
- 碱性氨基酸：$\text{pI} = (\text{p}K_{R} + \text{p}K_{a2}) / 2$

### 4.3 辅酶反应通式

**NAD⁺ 还原**：
$$\text{NAD}^+ + \text{H}^- + 2\text{e}^- \rightarrow \text{NADH}$$

**硫醇酯烯醇化**：
$$\text{R-CO-SCoA} \rightleftharpoons \text{R-C(OH)=CH-SCoA} \quad \text{（烯醇式）}$$

### 4.4 脂肪酸合成循环

每轮循环：
$$\text{乙酰-S-ACP} + \text{丙二酰-S-ACP} \xrightarrow{\text{缩合}} \beta\text{-酮酰-S-ACP}$$
$$\xrightarrow{\text{NADPH还原}} \beta\text{-羟酰-S-ACP} \xrightarrow{\text{脱水}} \alpha,\beta\text{-不饱和酰-S-ACP} \xrightarrow{\text{NADPH再还原}} \text{饱和酰-S-ACP} + 2\text{C}$$

## 五、常见分类或情形

### 5.1 糖类分类

| 类别 | 特征 | 代表化合物 |
|:---|:---|:---|
| 单糖 | 不能水解为更小的糖单位 | 葡萄糖、果糖、核糖 |
| 二糖 | 两分子单糖通过糖苷键连接 | 蔗糖、麦芽糖、乳糖 |
| 多糖 | 多个单糖聚合 | 淀粉（α-1,4）、纤维素（β-1,4）、糖原 |
| 氨基糖 | 羟基被氨基取代 | 葡糖胺、半乳糖胺 |

### 5.2 氨基酸分类（按侧链性质）

| 类别 | 代表氨基酸 | 侧链特征 |
|:---|:---|:---|
| 非极性/疏水 | Gly, Ala, Val, Leu, Ile, Pro, Phe, Trp, Met | 烷基或芳香基 |
| 极性不带电 | Ser, Thr, Cys, Tyr, Asn, Gln | 含 OH, SH, CONH₂ |
| 酸性（带负电） | Asp, Glu | 羧基 |
| 碱性（带正电） | Lys, Arg, His | 氨基、胍基、咪唑基 |

### 5.3 代谢途径概览

| 途径 | 位置 | 主要功能 | 关键中间体 |
|:---|:---|:---|:---|
| 糖酵解 | 细胞质 | 葡萄糖 → 丙酮酸 + ATP | G6P, F1,6BP, PEP |
| 柠檬酸循环 | 线粒体 | 乙酰 CoA → CO₂ + 还原力 | 柠檬酸, α-酮戊二酸, 草酰乙酸 |
| 磷酸戊糖途径 | 细胞质 | 产生 NADPH 和核糖-5-磷酸 | G6P, 核酮糖-5-磷酸 |
| 脂肪酸合成 | 细胞质 | 乙酰 CoA → 脂肪酸 | 丙二酰 CoA, 酰基-ACP |
| 莽草酸途径 | 质体 | 芳香氨基酸合成 | 莽草酸, 分支酸 |

## 六、适用条件与限制

- **酶催化反应**：在温和条件下进行（体温、中性 pH），但机理与实验室反应本质相同
- **辅酶作用**：辅酶作为"分子工具"，扩展了氨基酸侧链的化学能力
- **代谢调控**：代谢途径受别构调节和共价修饰调控，竞赛层面通常不涉及
- **立体化学**：酶提供的手性环境产生高度立体选择性，与实验室手性催化剂原理相通
- **竞赛限制**：生物化学在初赛中考查基础结构识别；决赛中可能涉及酶机理推断

## 七、常见比较与易混点

### 7.1 α-螺旋 vs β-折叠

| 特征 | α-螺旋 | β-折叠 |
|:---|:---|:---|
| 氢键模式 | 链内 i → i+4 | 链间或链内相邻段 |
| 每圈残基数 | 3.6 | 2（反平行）或 2.3（平行） |
| 侧链方向 | 向外 | 交替上下 |
| 稳定性因素 | 分子内氢键 | 分子间/段间氢键 |

### 7.2 DNA vs RNA

| 特征 | DNA | RNA |
|:---|:---|:---|
| 糖 | 脱氧核糖（2'-H） | 核糖（2'-OH） |
| 碱基 | A, T, G, C | A, U, G, C |
| 结构 | 双螺旋 | 单链（可折叠） |
| 碱性稳定性 | 稳定 | 易水解（2'-OH分子内进攻） |
| 功能 | 遗传信息存储 | 蛋白质合成、催化、调控 |

### 7.3 NADH vs NADPH

| 特征 | NADH | NADPH |
|:---|:---|:---|
| 磷酸位置 | 无 | 2'位腺嘌呤核糖 |
| 主要功能 | 分解代谢（氧化供能） | 合成代谢（还原力） |
| 代谢定位 | 线粒体/糖酵解 | 细胞质/磷酸戊糖途径 |

## 十、例题

### 例题1：异头碳效应

**题目**：葡萄糖与甲醇在酸性条件下反应，热力学控制下主要产物是 α-甲基葡萄糖苷（直立 OMe），而非预期的 β-异头体（平伏 OMe）。解释这一现象。

**解答要点**：

1. **异头碳效应**：异头碳上直立的 OMe 与环内氧的孤对电子发生 $n \rightarrow \sigma^*$ 相互作用
2. **轨道相互作用**：环内氧的孤对电子（n）给入 OMe 的 C-O σ* 反键轨道，产生稳定化作用
3. **热力学控制**：反应可逆，达到平衡时热力学更稳定的 α-异头体占优
4. **注意**：对葡萄糖本身（OH 为取代基），β-异头体更稳定（平伏 OH 的 1,3-双直立相互作用最小化）；但当取代基为吸电子基团（OMe、OAc 等）时，异头碳效应可能逆转优势

### 例题2：RNA 碱性水解机理

**题目**：RNA 在碱性条件下迅速水解，而 DNA 在相同条件下稳定。写出 RNA 水解的机理并解释几何要求。

**解答要点**：

1. **亲核进攻**：2'-OH 的氧负离子分子内进攻相邻的 3'-磷酸酯的磷原子
2. **五配位过渡态**：磷原子形成三角双锥过渡态
3. **离去基团**：5'-O 离去，形成 2',3'-环状磷酸酯中间体
4. **几何要求**：2'-OH 与 3'-磷酸酯需满足顺式几何关系，使亲核试剂能接近磷原子
5. **DNA 稳定原因**：脱氧核糖无 2'-OH，无法进行分子内亲核进攻

### 例题3：PLP 介导的转氨反应

**题目**：写出吡哆醛磷酸（PLP）介导的氨基酸转氨反应的机理。

**解答要点**：

1. **亚胺形成**：PLP 的醛基与氨基酸的氨基缩合形成亚胺（Schiff 碱，外部醛亚胺）
2. **质子转移**：亚胺的 α-氢被碱夺取，形成醌式中间体（quinoid intermediate）
3. **水解**：水分子进攻亚胺碳，释放酮酸产物，PLP 转化为吡哆胺磷酸（PMP）
4. **逆过程**：另一分子酮酸与 PMP 反应，生成相应氨基酸，PLP 再生

## 十一、易错点

1. **混淆 α/β 异头体定义**：α-异头体指异头碳上的取代基与 C6（或决定构型的碳）在同侧（D-系列中为直立），不是简单的上下关系
2. **忽略 D/L 与 R/S 的区别**：D/L 是相对于甘油醛的相对构型；R/S 是 CIP 绝对构型。两者无简单对应关系
3. **肽键刚性误解**：肽键的平面刚性来自 C=O 与 N 孤对电子的共轭，不是单键旋转受阻
4. **NADH 负氢转移方向**：NAD⁺ 的烟酰胺环在 4-位接受负氢（H⁻），不是 2-位或 6-位
5. **CoA 硫醇酯特性**：硫醇酯比氧酯更易烯醇化和水解，因为硫的 3p 轨道与羰基 π* 轨道重叠差，共轭稳定化弱
6. **糖苷键类型混淆**：α-1,4、β-1,4、α-1,6 等表示异头碳构型和连接位置，需仔细区分

### 12.2 学生易踩的认知误区

- **误区1**：认为生物化学反应与实验室有机反应完全不同。实际上，酶催化的反应机理与经典有机反应（羟醛、Claisen、SN2 等）本质相同，只是条件更温和、选择性更高。
- **误区2**：混淆糖类的 Fischer 投影式和 Haworth 透视式。需要明确两种表示法的转换关系。
- **误区3**：认为所有天然氨基酸都是 L-构型。忽略了细菌细胞壁中的 D-氨基酸。

### 12.3 入门级例题

**题目**：画出 D-葡萄糖的 Haworth 式，标出异头碳，并指出 α-和 β-异头体的区别。

**参考答案**：

- Haworth 式：六元吡喃糖环，氧原子在右上方
- C1 为异头碳（半缩醛碳）
- α-D-葡萄糖：C1-OH 向下（与 C6 在同侧，直立）
- β-D-葡萄糖：C1-OH 向上（与 C6 在异侧，平伏）

### 12.4 与现实/直觉的连接

- **维生素 C 与坏血病**：维生素 C（抗坏血酸）是胶原合成中脯氨酸/赖氨酸羟化酶的辅因子。缺乏时胶原交联不足，导致血管脆弱（坏血病）。
- **青霉素与细菌细胞壁**：理解青霉素的作用机理有助于认识抗生素的合理使用。
- **代谢疾病**：苯丙酮尿症（PKU）是苯丙氨酸羟化酶缺陷导致的代谢疾病，理解代谢途径有助于理解疾病机制。
