## 一、定义

化学式推断是指根据实验数据（元素质量分数、燃烧产物质量、气体体积、晶胞参数等）确定未知化合物的实验式（最简式）或分子式的过程。它是竞赛中元素分析和结构推断的基础技能。

## 三、核心原理

### 3.1 由元素质量分数求实验式

1. 假设样品质量为 100g，计算各元素的物质的量：
   $$n_i = \frac{m_i}{M_i} = \frac{w_i \times 100}{M_i}$$
2. 求各元素物质的量之比，化为最简整数比
3. 得到实验式（最简式）

### 3.2 由实验式求分子式

需要知道相对分子质量 $M_r$：
$$n = \frac{M_r}{M_{\text{实验式}}}$$
分子式 = $(\text{实验式})_n$

**求 $M_r$ 的方法**：
- 理想气体法：$M = \frac{mRT}{PV} = \rho \frac{RT}{P} = 2D_{H_2} = 29D_{\text{air}}$
- 依数性法（稀溶液）：$M = \frac{mRT}{\pi V}$ 或沸点升高/凝固点降低
- 质谱法：直接给出分子离子峰

### 3.3 燃烧分析法

对于仅含 C、H、O 的有机物：
- $n(C) = n(CO_2) = \frac{m(CO_2)}{44}$
- $n(H) = 2n(H_2O) = \frac{2m(H_2O)}{18}$
- $n(O) = \frac{m_{\text{样品}} - m(C) - m(H)}{16}$

对于含 N 的有机物：
- $n(N) = 2n(N_2)$（由氮气体积计算）
- 或利用差减法求 N 的质量

### 3.4 晶胞化学式推断

1. 确定晶胞中各原子的位置及贡献（顶点 1/8、棱 1/4、面心 1/2、体心 1）
2. 计算各原子的总数
3. 化为最简整数比
