# 碘量法

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-分析化学]]
- 关联笔记：[[氧化还原滴定]], [[指示剂]], [[容量分析计算]]

## 一、定义

碘量法（Iodometry）是利用 I₂ 的氧化性或 I⁻ 的还原性进行定量分析的氧化还原滴定方法。根据反应方向分为直接碘量法（碘滴定法）和间接碘量法（滴定碘法）。碘量法在竞赛中常用于测定还原性物质、氧化性物质和某些有机物。

## 三、核心原理

### 3.1 基本反应

碘量法核心反应（计量关系 1:2）：

$$I_2 + 2S_2O_3^{2-} = 2I^- + S_4O_6^{2-}$$

$S_2O_3^{2-}$（硫代硫酸钠）是碘量法最常用的滴定剂。

### 碘量法分类

| 类型 | 条件 | 适用对象 |
|------|------|---------|
| **直接碘量法** | $\varphi < \varphi_{\mathrm{I_2/I^-}}^{\ominus} = 0.545\,\mathrm{V}$ | S²⁻、SO₃²⁻、S₂O₃²⁻、As(III)、Sn(II)、维生素C |
| **间接碘量法** | $\varphi > \varphi_{\mathrm{I_2/I^-}}^{\ominus}$ | ClO⁻、Cr₂O₇²⁻、MnO₄⁻、IO₃⁻、Cu²⁺、H₂O₂ |

### 3.2 直接碘量法（碘滴定法）

- **原理**：用 I₂ 标准溶液直接滴定还原性物质
- **适用对象**：S²⁻、SO₃²⁻、S₂O₃²⁻、As(III)、Sn(II)、维生素C 等
- **指示剂**：淀粉（与 I₂ 形成蓝色络合物，灵敏度高）
- **条件**：弱酸性或中性（碱性中 I₂ 会歧化）

### 3.3 间接碘量法（滴定碘法）

- **原理**：氧化性物质与过量 KI 反应生成 I₂，再用 Na₂S₂O₃ 滴定生成的 I₂
- **适用对象**：ClO⁻、Cr₂O₇²⁻、MnO₄⁻、IO₃⁻、Cu²⁺、H₂O₂ 等
- **反应通式**：$2I^- \rightarrow I_2 + 2e^-$（被氧化剂氧化）
- **定量关系**：$n(\text{氧化剂}) : n(I_2) : n(S_2O_3^{2-}) = a : b : 2b$

### 3.4 注意事项

| 问题 | 原因 | 解决方法 |
|------|------|---------|
| **I₂ 挥发** | I₂ 易升华 | 低温操作、使用碘量瓶、避免剧烈摇动 |
| **I⁻ 被空气氧化** | $4I^- + O_2 + 4H^+ = 2I_2 + 2H_2O$ | 避免强光、酸性不宜过强、及时滴定 |
| **淀粉指示剂加入时机** | 过早加入会包裹 I₂ | 间接碘量法在临近终点时加入 |
| **pH 控制** | 强酸中 S₂O₃²⁻ 分解；强碱中 I₂ 歧化 | pH 控制在弱酸性或中性 |

### 经典应用实例

**铜合金中铜的测定**（间接碘量法经典竞赛题）：

$$2\mathrm{Cu^{2+}} + 4\mathrm{I^-} = 2\mathrm{CuI}\downarrow + \mathrm{I_2}$$

再用 Na₂S₂O₃ 滴定生成的 I₂。近终点时加 NH₄SCN 使 CuI → CuSCN，减少对 I₂ 的吸附。

**甲醛测定**（易错点：误认为生成 CO₃²⁻）：

$$\mathrm{HCHO} + \mathrm{I_2} + 3\mathrm{OH^-} = \mathrm{HCOO^-} + 2\mathrm{I^-} + 2\mathrm{H_2O}$$

注意 1:1 计量关系（非生成 CO₃²⁻）。

## 四、关键结论

- 直接碘量法用 `I2` 滴定还原剂，间接碘量法用 `S2O3^2-` 滴定析出的 `I2`
- 间接碘量法通常要求中性或弱酸性，既要防 `I2` 挥发，也要防 `I-` 被空气氧化
- 淀粉指示剂在间接碘量法中应临近终点加入
- 铜测定和甲醛测定是两类最值得课堂讲清的典型应用

## 五、常见分类或情形

- 直接碘量法：测定 `As(III)`、`Sn(II)`、维生素 C 等还原剂
- 间接碘量法：测定 `Cu2+`、漂白粉有效氯、`Cr2O7^2-`、`H2O2` 等氧化剂
- 特殊误差题：淀粉加入时机、CuI 吸附 `I2`、酸碱条件控制

## 六、适用条件与限制

- 强酸中 `S2O3^2-` 会分解，强碱中 `I2` 会歧化
- 操作时要避光、低温、尽量减少 `I2` 挥发
- 加入过量 `KI` 有助于形成 `I3-`，减少挥发损失

## 七、常见比较与易混点

- 直接碘量法和间接碘量法不要混淆滴定剂
- `I2 + 2S2O3^2- -> 2I- + S4O6^2-` 中产物是连四硫酸根，不是硫酸根
- 间接碘量法中淀粉不是越早加越好

1. **碘量法计算**：由滴定体积计算待测物含量

## 十、例题

### 例题 1
**题目：** 间接碘量法测铜时，为什么临近终点要加入 `NH4SCN`？

**解答：** 因为 `CuI` 沉淀会吸附 `I2`，导致结果偏低；加入 `NH4SCN` 可使 `CuI` 转化为更有利于释放 `I2` 的体系，减小吸附误差。

**反思：** 这类题不只是记操作，而是在考误差来源和修正手段。

## 十一、易错点

- 间接碘量法中淀粉加入过早——应在临近终点时加入
- 忽略 I₂ 的挥发性——需使用碘量瓶
- 强酸性条件下 S₂O₃²⁻ 分解——pH 应控制在弱酸性或中性
- 强碱性条件下 I₂ 歧化——同样需控制 pH

## 十二、🎯 教学视角

### 学生常见认知误区
1. **误区**：认为淀粉指示剂越早加入越好。
   - **原因**：直觉上"指示剂就是用来指示的，早加早看到"。
   - **正确认知**：间接碘量法中淀粉过早加入会包裹 I₂，导致终点滞后。
   - **记忆法**："间接碘量终点加，直接碘量随时加"。

2. **误区**：认为 I₂ 和 S₂O₃²⁻ 的反应是氧化还原反应。
   - **原因**：看到 I₂（氧化剂）和还原剂就联想到氧化还原。
   - **正确认知**：I₂ + 2S₂O₃²⁻ → 2I⁻ + S₄O₆²⁻ 确实是氧化还原反应，但 S₂O₃²⁻ 被氧化为 S₄O₆²⁻（连四硫酸根），不是 SO₄²⁻。
   - **易错点**：不要写成 SO₄²⁻！

### 入门级例题（教师引导提问）
**题目**：间接碘量法测定 Cu²⁺ 时，为什么要在近终点时加入 NH₄SCN？

**预期解答路径**：
- CuI 沉淀表面会吸附 I₂
- 吸附的 I₂ 不被 Na₂S₂O₃ 滴定，导致结果偏低
- 加入 NH₄SCN 使 CuI 转化为溶解度更小的 CuSCN
- CuSCN 对 I₂ 的吸附弱，释放出的 I₂ 可被滴定
- **核心结论**：NH₄SCN 的作用是减少 I₂ 的吸附误差。

### 与现实/直觉的连接
- **类比**：碘量法像"接力赛"——氧化剂把 I⁻ 氧化成 I₂（第一棒），I₂ 再被 S₂O₃²⁻ 还原（第二棒），通过测量第二棒的"成绩"推算第一棒。
- **生活实例**：碘伏（聚维酮碘）中的 I₂ 就是利用了碘的氧化性杀菌，而硫代硫酸钠（大苏打）是碘中毒的解毒剂。
