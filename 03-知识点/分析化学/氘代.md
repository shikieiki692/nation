# 氘代

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-分析化学]]
- 对应考纲条目：[[NMR谱学]]、[[同位素]]

## 一、定义

**氘代（Deuteration）** 是指将化合物中的氢原子（¹H，质子）替换为其同位素氘（²H 或 D，重氢）的过程或结果。氘代在化学竞赛和科研中有三大核心用途：

1. **NMR 溶剂**：消除溶剂质子的信号干扰
2. **结构鉴定**：利用同位素位移或氘交换追踪活泼氢
3. **机理研究**：通过 D 标记追踪反应路径（动力学同位素效应 KIE）

| 考纲条目 | 对应内容 | 考查深度 |
|:---|:---|:---:|
| NMR 谱学 | 氘代溶剂的溶剂峰位置；氘代消除质子信号的原理 | 掌握 |
| 同位素 | 氘与氢的质量差异（2.014 vs 1.008）；核自旋差异（I=1 vs I=1/2）| 了解 |
| 质谱 | 同位素峰 M+1、M+2 的识别（含 D 时质量偏移）| 了解 |

## 三、核心原理

### 3.1 氘代溶剂在 NMR 中的作用

普通有机溶剂（如 CHCl₃）含有质子，在 ¹H NMR 中会产生强溶剂峰，掩盖样品信号。氘代溶剂（如 CDCl₃）将溶剂分子中的 H 替换为 D：

- **D 的核自旋 I = 1**（而非 H 的 I = 1/2），在 ¹H NMR 的射频条件下**不共振**
- 因此氘代溶剂在 ¹H NMR 谱中**不产生信号**（或仅剩残余质子峰，如 CDCl₃ 中约 0.01% 的 CHCl₃ 在 δ 7.26 处显示单峰）

**常见氘代溶剂及其残余峰**：

| 氘代溶剂 | 化学式 | ¹H 残余峰 (δ) | 特点 |
|:---|:---|:---:|:---|
| 氘代氯仿 | CDCl₃ | 7.26 (s) | 最常用，溶解性广 |
| 氘代二甲亚砜 | DMSO-d₆ | 2.50 (五重峰) | 适用于极性化合物 |
| 氘代丙酮 | (CD₃)₂CO | 2.05 (五重峰) | 中等极性 |
| 氘代水 | D₂O | 4.79 | 水溶性样品 |
| 氘代苯 | C₆D₆ | 7.16 (s) | 芳香体系，特殊溶剂效应 |
| 氘代甲醇 | CD₃OD | 3.31, 4.87 | 含可交换 OD |

> **竞赛要点**：DMSO-d₆ 和丙酮-d₆ 的残余峰因 D 与残余 H 的耦合而呈**五重峰**（2nI+1 = 2×2×1/2+1 = 3？不对——实际是因为 CD₃ 中 D 的 I=1，三个 D 与 H 耦合：2nI+1 = 2×3×1+1 = **7**？实际上常见的是五重峰，源于部分氘化或仪器分辨率限制。竞赛中只需记住"D 耦合使峰分裂"即可）。

### 3.2 活泼氢的氘交换

含有活泼氢的基团（-OH、-NH、-COOH、-SH）可以与 D₂O 或氘代溶剂发生**快速交换**：

$$
\ce{R-OH + D2O <=> R-OD + HDO}
$$

**NMR 中的表现**：
- ¹H NMR 中，交换后的活泼氢信号**消失**或**大幅减弱**
- 加入 D₂O 后若某峰消失 → 可确认该峰对应活泼氢
- D 交换速率：-COOH > -OH > -NH > -SH

### 3.3 同位素效应（KIE）

氘的质量约为氢的 2 倍，导致：
- **C-D 键比 C-H 键更强**（零点能差异）
- **C-D 键断裂更慢**（动力学同位素效应，KIE ≈ 5-7）

利用 KIE 可以判断决速步是否涉及 C-H 键断裂，是机理研究的重要工具。

## 四、关键结论

1. 氘代溶剂消除溶剂质子信号，是 NMR 测试的**标配**
2. 氘代溶剂残余峰可作为**内标**用于化学位移校准
3. D₂O 交换是鉴别活泼氢的**经典实验手段**
4. KIE（k_H/k_D ≈ 5-7）表明决速步涉及 C-H 断裂
5. D 的 I=1，在 ²H NMR 中可用，但灵敏度远低于 ¹H NMR

## 六、适用条件与限制

| 情形 | 说明 |
|:---|:---|
| NMR 必须使用氘代溶剂 | 常规 ¹H NMR 测试要求；少量样品可用普通溶剂但需内标 |
| D₂O 交换不完全 | 空间位阻大的 -OH 或分子内氢键导致交换慢 |
| 氘代溶剂干扰 | 某些溶剂峰可能与样品峰重叠（如 CDCl₃ 的 7.26 与芳氢）|
| KIE 定量要求 | 需要高氘代纯度（>98%）；部分氘化会复杂化分析 |
| 成本因素 | 氘代溶剂价格昂贵（CDCl₃ 约为 CHCl₃ 的 50-100 倍）|

## 七、常见比较与易混点

### 7.1 氘代溶剂残余峰 vs TMS

| 特征 | TMS（内标） | 氘代溶剂残余峰 |
|:---|:---|:---|
| 位置 | δ 0.00（固定） | 各溶剂特定（如 7.26、2.50）|
| 用途 | 精确校准化学位移 | 粗略参考或备用校准 |
| 添加方式 | 额外加入 | 溶剂自带 |
| 信号 | 强而锐利的单峰 | 较弱，可能分裂 |

**竞赛注意**：若题目未加 TMS，可用氘代溶剂残余峰作为参照，但精度较低。

### 7.2 D₂O 交换 vs 其他去活泼氢方法

- **D₂O 交换**：加入后振摇，¹H NMR 中活泼氢峰消失
- **加热法**：加速交换，但可能引发副反应
- **干燥剂**：除去水但无法区分活泼氢类型

## 十、例题

### 例题 1：氘代溶剂残余峰归属

**题目**：某化合物的 ¹H NMR 谱在 δ 7.26 处有一个小单峰（非样品峰），且加入 D₂O 后 δ 3.5 处的宽峰消失。判断：(a) 使用何种氘代溶剂？(b) δ 3.5 的峰对应何种质子？

**解析**：
1. δ 7.26 的小单峰是 **CDCl₃ 的残余 CHCl₃ 峰**
2. 加入 D₂O 后消失的峰是**活泼氢**，δ 3.5 处宽峰典型为 **-OH**
3. 若该峰在加入 D₂O 后移到 δ 4.7（HDO），则进一步确认

**答案**：(a) CDCl₃；(b) 羟基 (-OH)

### 例题 2：氘代还原的区域选择性

**题目**：用 NaBD₄ 还原环己酮，产物中氘原子在羟基的哪个位置？写出产物的立体化学。

**解析**：
1. NaBD₄ 提供 D⁻（氘代氢负离子），进攻羰基碳
2. 环己酮的羰基平面两侧位阻相同（对称分子），D⁻ 从上方或下方进攻概率相等
3. 产物为 **反-2-氘代环己醇** 和 **顺-2-氘代环己醇** 的混合物（1:1）
4. D 最终位于与 OH 相邻的碳上

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| 认为氘完全没有 NMR 信号 | 混淆 ¹H NMR 与 ²H NMR | D 在 ¹H NMR 频率下不共振，但在专门的 ²H NMR 中有信号（I=1） | 为什么氘代溶剂在 ¹H NMR 中"消失"？ |
| 忽略残余峰强度 | 认为氘代溶剂"绝对纯" | 商业氘代溶剂氘代率约 99.8-99.96%，仍有 0.04-0.2% 的残余质子 | 若 CDCl₃ 氘代率 99.9%，1 mL 溶剂中约有多少个 H 原子？ |
| 混淆 D₂O 交换与酸碱反应 | 认为 D₂O 只与酸反应 | D₂O 交换是**所有**活泼氢的快速平衡，不限于酸 | 醇羟基为什么也能与 D₂O 交换？ |
| KIE 数值方向搞反 | 记错 k_H/k_D 还是 k_D/k_H | **k_H/k_D > 1**（C-H 断裂更快），通常称"KIE = 5"指 k_H/k_D = 5 | 若 k_H/k_D = 7，C-H 断裂比 C-D 快多少倍？ |
| 把氘代溶剂峰当杂质 | 不识別残余峰特征位置 | 熟记常见氘代溶剂的残余峰化学位移 | δ 2.50 的五重峰是什么溶剂？ |

## 十二、🎯 教学视角

### 12.1 学习路径

```
前置：NMR谱学基础（化学位移、峰面积、裂分）→ 同位素概念（质量差异）
     ↓
本节：氘代溶剂原理 → 活泼氢交换 → KIE 初步
     ↓
后续：有机反应机理（KIE 深度应用）→ 高级 NMR 技术（DEPT、COSY）
```

**建议课时**：1 课时（氘代溶剂 0.3 + D₂O 交换实验 0.3 + KIE 介绍 0.4）

### 12.2 学生易踩的认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "氘就是氢，只是重一点" | 忽略核自旋差异 | 强调 **I_H = 1/2, I_D = 1**，这是 NMR 中"消失"的根本原因 |
| "D₂O 交换后 H 跑到别处了" | 不理解交换平衡 | 用天平类比：交换前后总 H 数守恒，只是分配到了 HDO/D₂O 中 |
| "所有氘代溶剂残余峰都是单峰" | 忽略 D-H 耦合 | 展示 DMSO-d₆ 的五重峰实例，解释 CD₃ 中 D 对残余 H 的耦合 |
| "KIE 越大反应越快" | 混淆同位素效应与速率 | KIE 是**比值**（k_H/k_D），不是绝对速率；KIE 大说明 H 版本比 D 版本快得多 |

### 12.3 入门级例题

**题目**：将少量乙醇溶于 CDCl₃ 测定 ¹H NMR，预期会看到哪些特征？若加入 D₂O 振摇后重测，谱图如何变化？

**预期解答路径**：
1. 乙醇在 CDCl₃ 中：CH₃（三重峰，δ ~1.2）、CH₂（四重峰，δ ~3.7）、OH（宽单峰，δ ~2-5，位置不定）
2. CDCl₃ 残余峰：δ 7.26（单峰，小）
3. 加入 D₂O 后：OH 峰**消失**（交换为 OD），CH₃ 和 CH₂ 不变
4. 可能出现 HDO 峰（δ ~4.7）

**课堂引导提问**：
- 学生若忽略残余峰："除了样品峰，溶剂本身会不会有信号？"
- 学生若认为所有峰都消失："D₂O 会和哪些氢交换？CH₃ 上的氢活泼吗？"

### 12.4 与现实/直觉的连接

**"静音耳机"类比氘代溶剂**：
- 普通溶剂 = 嘈杂环境中的通话（溶剂峰 = 背景噪音）
- 氘代溶剂 = 降噪耳机（D 原子不响应 ¹H 的"频道"）
- 残余峰 = 耳机未完全消除的微弱底噪

**"贴标签"类比氘标记**：
- 研究反应机理时，用 D "标记"某个特定 H
- 追踪产物中 D 的位置 = 追踪原子的"足迹"
- KIE = 标记后反应变慢了，说明这个 H 在关键时刻被"扯掉"了

### 13.2 KIE 的定量分析

一级动力学同位素效应的理论上限约为 7（298 K），源于零点能差异：

$$
k_H/k_D = e^{(\Delta ZPE)/k_BT} \approx 6-7 \text{（C-H 键，298 K）}
$$

若实测 KIE < 2，通常表明 C-H 断裂**不是**决速步，或仅部分参与过渡态。
