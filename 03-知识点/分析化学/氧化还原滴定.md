# 氧化还原滴定

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-分析化学]]
- 对应考纲条目：[[18-容量分析]]

## 一、定义

氧化还原滴定法是以氧化还原反应为基础的滴定分析方法。通常根据所用滴定剂的名称来命名，包括高锰酸钾法、重铬酸钾法、碘量法、硫酸铈法、溴酸钾法等。

## 二、考纲对应

- 对应考纲条目：[[18-容量分析]]
- 所属模块：[[基础要求-分析化学]]
- 本知识点在考纲中的作用：氧化还原滴定是四大滴定之一，是竞赛计算题的常见出题来源，尤其碘量法和 KMnO4 法的多步反应计量关系推导是难点。

## 三、核心原理

**1. 条件电势（条件电极电位）**

条件电势 $\varphi^{\ominus'}$ 是在特定条件下，氧化态与还原态分析浓度均为 $1\ \mathrm{mol\cdot L^{-1}}$ 时的实际电势：

$$\varphi = \varphi^{\ominus'} + \frac{0.059\mathrm{V}}{z}\lg\frac{c_{\mathrm{Ox}}}{c_{\mathrm{Red}}}$$

其中 $\varphi^{\ominus'} = \varphi^{\ominus} + \frac{0.059\mathrm{V}}{z}\lg\frac{\gamma_{\mathrm{Ox}}\alpha_{\mathrm{Red}}}{\gamma_{\mathrm{Red}}\alpha_{\mathrm{Ox}}}$

条件电势综合了离子强度与副反应的影响，类似于配位滴定中的条件稳定常数 $K'$。关键数据：在 $1\mathrm{mol\cdot L^{-1}}\ \mathrm{HCl}$ 中，$\varphi^{\ominus'}_{\mathrm{Fe^{3+}/Fe^{2+}}} = 0.68\mathrm{V}$（标准态为 0.77V）；$\varphi^{\ominus'}_{\mathrm{Ce^{4+}/Ce^{3+}}} = 1.28\mathrm{V}$。

**2. 氧化还原反应平衡常数**

$$\lg K = \frac{(\varphi_1^{\ominus} - \varphi_2^{\ominus})z}{0.059\mathrm{V}},\quad \lg K' = \frac{(\varphi_1^{\ominus'} - \varphi_2^{\ominus'})z}{0.059\mathrm{V}}$$

其中 $z$ 为两电对电子转移数的最小公倍数。对 $z_1 = z_2 = 1$ 的反应，要求反应完全程度 $\geq 99.9\%$，需 $\lg K \geq 6$，即 $\varphi_1^{\ominus} - \varphi_2^{\ominus} \geq 0.35\mathrm{V}$。

**3. 影响氧化还原反应速率的因素**

- **反应物浓度**：增大浓度通常加快速率
- **温度**：通常每升高 $10^{\circ}\mathrm{C}$，速率增大 2~3 倍（如 KMnO4 滴定 H2C2O4 需加热至 75~85℃）
- **催化剂**：如 Mn^{2+} 催化 MnO4^- 与 C2O4^{2-} 的反应（自动催化反应）

**4. 诱导反应**

区别于催化反应：诱导反应中诱导体参加反应后变为其他物质。如 KMnO4 氧化 Fe^{2+} 时诱导 Cl^- 被氧化，产生正误差。加入 Mn^{2+} 可抑制此诱导反应。

## 四、关键结论

| 方法 | 滴定剂 | 半反应 / 关键条件 | 指示剂 | 主要应用 |
|------|--------|-------------------|--------|---------|
| 高锰酸钾法 | KMnO4 | MnO4^- + 8H^+ + 5e^- = Mn^{2+} + 4H2O，$\varphi^{\ominus}=1.51\mathrm{V}$（酸性） | 自身指示（紫红→无色） | Fe^{2+}, H2O2, Ca^{2+}(间接), COD |
| 重铬酸钾法 | K2Cr2O7 | Cr2O7^{2-} + 14H^+ + 6e^- = 2Cr^{3+} + 7H2O，$\varphi^{\ominus}=1.33\mathrm{V}$ | 二苯胺磺酸钠 | 全铁测定，COD |
| 碘量法 | I2 / Na2S2O3 | I3^- + 2e^- = 3I^-，$\varphi^{\ominus}=0.545\mathrm{V}$ | 淀粉 | Cu^{2+}, 漂白粉, 有机物 |

### 氧化还原指示剂分类（来源：周坤无机新课 Batch 5）

| 类型 | 原理 | 典型实例 |
|:---|:---|:---|
| **自身指示剂** | 滴定剂或被测物本身颜色变化 | KMnO4（紫红→无色） |
| **显色指示剂** | 与滴定体系某组分形成有色配合物 | 淀粉-I2（蓝色） |
| **氧化还原指示剂** | 氧化态/还原态颜色不同 | 二苯胺磺酸钠、邻二氮菲-亚铁 |

**关键原理要点**：
- 可逆电对（如 Fe^{3+}/Fe^{2+}、I2/I^-）迅速建立平衡，电势符合 Nernst 公式；不可逆电对（如 MnO4^-/Mn^{2+}、Cr2O7^{2-}/Cr^{3+}）实际电势与理论值相差较大
- 化学计量点电势计算（对称电对 $z_1 = z_2$）：$\varphi_{\mathrm{sp}} = \frac{z_1\varphi_1^{\ominus'} + z_2\varphi_2^{\ominus'}}{z_1 + z_2}$

## 五、常见分类或情形

**1. 高锰酸钾法三种滴定方式**
- **直接滴定**：还原性物质如 Fe^{2+}、H2O2、C2O4^{2-}、NO2^-、As(III)、Sb(III)
- **返滴定**：氧化性物质如 MnO2（加过量 Na2C2O4，用 KMnO4 返滴）
- **间接滴定**：非氧化还原物质如 Ca^{2+}（沉淀为 CaC2O4，溶解后用 KMnO4 滴定 C2O4^{2-}），还可测 Ba^{2+}、Mg^{2+}、Zn^{2+}、Pb^{2+}、Ag^+ 等

**KMnO4 在不同 pH 下的还原产物**：
- 酸性：Mn^{2+}，$\varphi^{\ominus} = 1.51\,\mathrm{V}$
- 中性：MnO2，$\varphi^{\ominus} = 0.59\,\mathrm{V}$
- 强碱性：MnO4^{2-}，$\varphi^{\ominus} = 0.564\,\mathrm{V}$

条件选择关键：酸性条件下氧化能力最强，是大多数滴定的首选。

**KMnO4 标定反应**（与 Na2C2O4）：

$$2\mathrm{MnO_4^-} + 5\mathrm{C_2O_4^{2-}} + 16\mathrm{H^+} = 2\mathrm{Mn^{2+}} + 10\mathrm{CO_2}\uparrow + 8\mathrm{H_2O}$$

条件：70~80℃，0.5~1 M H^+。最初几滴褪色慢（Mn^{2+} 自动催化）。

**KMnO4 标定草酸反应**（来源：周坤无机新课 Batch 5）：

$$2\mathrm{MnO_4^-} + 5\mathrm{H_2C_2O_4} + 6\mathrm{H^+} = 2\mathrm{Mn^{2+}} + 10\mathrm{CO_2}\uparrow + 8\mathrm{H_2O}$$

条件与草酸钠相同：70~80℃、硫酸介质、约 $1\ \mathrm{mol\cdot L^{-1}}$ H^+，Mn^{2+} 自催化。**注意**：温度过高会使草酸分解；不能用 HCl 或 HNO3 介质（HCl 被氧化，HNO3 本身是氧化剂）。

**KMnO4 溶液的配制**：不能直接配制，需称取稍过量的 KMnO4，煮沸 1h，放置 2~3 天，微孔玻璃漏斗过滤，棕色瓶暗处保存。标定用 Na2C2O4（基准物质，105~110℃烘干 2h），在 H2SO4 介质中，70~85℃ 滴定，酸度 0.5~1 mol/L。

**2. 碘量法**
- **直接碘量法**：I2 标准溶液滴定强还原剂（Sn^{2+}, S^{2-}, SO3^{2-}, AsO3^{3-} 等），$\varphi_{\mathrm{I_2/I^-}}^{\ominus}=0.545\mathrm{V}$，只能在酸性或中性溶液中进行
- **间接碘量法**：氧化剂 + 过量 I^- → I2，用 Na2S2O3 标准溶液滴定 I2：$\mathrm{I_2 + 2S_2O_3^{2-} = 2I^- + S_4O_6^{2-}}$

**歧化反应**（碱性 KMnO4 法后酸化）：

$$3\mathrm{MnO_4^{2-}} + 4\mathrm{H^+} = 2\mathrm{MnO_4^-} + \mathrm{MnO_2}\downarrow + 2\mathrm{H_2O}$$

**间接碘量法关键条件**：
- 必须在中性或弱酸性溶液中进行（碱性中 I2 歧化且 S2O3^{2-} 被氧化为 SO4^{2-}，强酸性中 S2O3^{2-} 分解）
- 加入过量 KI 使 I2 形成 I3^-，减少挥发；使用碘量瓶，避光，析出 I2 后立即滴定
- 淀粉指示剂在临近终点时加入（I2 浓度大时淀粉-I2 络合物解离慢，产生误差）
- 典型应用：Cu^{2+} 测定（$2\mathrm{Cu^{2+} + 4I^- = 2CuI\downarrow + I_2}$），近终点时加 NH4SCN 使 CuI 转化为 CuSCN，减少对 I2 的吸附

**3. 重铬酸钾法**
- 优点：K2Cr2O7 易提纯，可直接配制标准溶液（140~250℃ 干燥后）；溶液稳定可长期保存；在 1 mol/L HCl 中 $\varphi^{\ominus'} = 1.00\mathrm{V}$，室温不与 Cl^- 作用
- 全铁测定：SnCl2 还原 Fe^{3+}，加 H3PO4 使 Fe^{3+} 生成 $[\mathrm{Fe(HPO_4)_2}]^-$ 降低 Fe^{3+}/Fe^{2+} 电势、增大突跃并消除 Fe^{3+} 黄色干扰

**K2Cr2O7 测 Fe 反应**（来源：周坤无机新课 Batch 5）：
$$\mathrm{Cr_2O_7^{2-} + 6Fe^{2+} + 14H^+ = 2Cr^{3+} + 6Fe^{3+} + 7H_2O}$$

**铁矿石分析实例**（来源：周坤无机新课 Batch 5）：典型铁矿石样品经 SnCl2 还原后，用 K2Cr2O7 滴定，计算结果示例为 Fe% = 45.24，Fe2O3% = 64.68。此数据可用于检验学生对多步计量关系（$\mathrm{Fe_2O_3} \sim 2\mathrm{Fe} \sim \frac{1}{3}\mathrm{K_2Cr_2O_7}$）的掌握。

## 六、适用条件与限制

- 反应必须有足够大的平衡常数（$\lg K \geq 6$ 对于 1:1 型反应）
- 反应速率必须足够快，否则需加热或加催化剂
- 无副反应或诱导反应的干扰
- 滴定剂在空气中应稳定（因此还原性滴定剂较少，常用仅 Na2S2O3、FeSO4 等）
- KMnO4 法：不能用 HCl 或 HNO3 介质（HCl 被氧化，HNO3 本身是氧化剂）；强酸性中用 H2SO4

## 七、常见比较与易混点

| 对比项 | KMnO4 法 | K2Cr2O7 法 | 碘量法 |
|--------|----------|-----------|--------|
| 氧化能力 | 强 ($\varphi^{\ominus}=1.51\mathrm{V}$) | 中 ($\varphi^{\ominus}=1.33\mathrm{V}$) | 弱 ($\varphi^{\ominus}=0.545\mathrm{V}$) |
| 标准溶液 | 间接配制，需标定 | 直接配制 | Na2S2O3 间接配制 |
| 稳定性 | 不稳定，易分解 | 非常稳定 | Na2S2O3 不稳定 |
| 指示剂 | 自身 | 氧化还原指示剂 | 专属指示剂（淀粉） |
| 与 Cl^- 关系 | 有诱导反应干扰 | 室温不反应 | — |
| 选择性 | 差（氧化能力强） | 较好 | 好 |

- **氧化还原指示剂变色范围**：$\varphi_{\mathrm{In}}^{\ominus'} \pm \frac{0.059}{z}\mathrm{V}$，选择时应使 $\varphi_{\mathrm{In}}^{\ominus'}$ 尽量接近 $\varphi_{\mathrm{sp}}$
- **不可逆电对**（如 MnO4^-/Mn^{2+}、Cr2O7^{2-}/Cr^{3+}、S4O6^{2-}/S2O3^{2-}）不能瞬间建立 Nernst 平衡，实际电势与理论值偏差大

## 十、例题

### 例题 1：KMnO4 法测定 Ca^{2+} 的计算关系
**题目：** 用 KMnO4 法间接测定 Ca^{2+} 时，推导 Ca^{2+} 与 KMnO4 的计量关系。

**分析：** Ca^{2+} + C2O4^{2-} → CaC2O4↓，沉淀溶于稀 H2SO4 后，用 KMnO4 滴定 C2O4^{2-}。

**解答：** $1\ \mathrm{Ca^{2+}} \sim 1\ \mathrm{CaC_2O_4} \sim 1\ \mathrm{C_2O_4^{2-}} \sim \frac{2}{5}\ \mathrm{KMnO_4}$

**反思：** 此类多步反应关键是写出每一步的化学反应方程式，逐级建立计量关系。

### 例题 2：碘量法测定铜
**题目：** 间接碘量法测定 Cu^{2+} 时，为什么要在近终点时加入 NH4SCN？

**分析：** CuI 沉淀表面吸附 I2，使结果偏低。

**解答：** 加入 NH4SCN 使 CuI 转化为溶解度更小的 CuSCN，减少对 I2 的吸附，并使被吸附的 I2 释放出来。反应式：$\mathrm{CuI + SCN^- = CuSCN\downarrow + I^-}$。

## 十一、易错点

- 误以为所有氧化还原反应都能用于滴定——必须满足速率和完全程度要求
- 混淆直接碘量法和间接碘量法的酸度条件：间接碘量法必须中性和弱酸性
- KMnO4 标定时忽视温度条件（70~85℃），温度过低反应慢，过高 H2C2O4 分解
- 混淆电对的可逆性：计算滴定曲线时不可逆电对的理论电势与实际不符
- 碘量法中淀粉加入过早，I2-淀粉络合物不易解离导致终点滞后
