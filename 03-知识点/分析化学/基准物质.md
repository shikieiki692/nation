# 基准物质

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-分析化学]]
- 对应考纲条目：[[18-容量分析]]

## 一、定义

**基准物质**（primary standard substance）属于标准物质（reference material, RM）的一种，也称滴定分析标准物质。它是能用于直接配制[[标准溶液]]或标定溶液准确浓度的物质。

## 三、核心原理

基准物质必须同时满足以下条件：

1. **组成与化学式严格一致**：若含结晶水，如 $\mathrm{H}_2\mathrm{C}_2\mathrm{O}_4\cdot 2\mathrm{H}_2\mathrm{O}$、$\mathrm{Na}_2\mathrm{B}_4\mathrm{O}_7\cdot 10\mathrm{H}_2\mathrm{O}$ 等，其结晶水的含量均应符合化学式。
2. **纯度足够高**：质量分数在 $99.9\%$ 以上。
3. **性质稳定**：不易与空气中的 $\mathrm{O}_2$ 及 $\mathrm{CO}_2$ 反应，亦不易吸收空气中的水分。
4. **参加滴定反应时按反应式定量进行**，没有副反应。
5. **摩尔质量较大**：以减小称量误差。在分析天平（精度 $0.1\mathrm{~mg}$）上称量时，摩尔质量大则相对称量误差小。

## 四、关键结论

### 注意事项

需注意，有些标识为高纯、超纯和光谱纯的试剂，只是表明某些特定杂质项的含量很低，并不表明其主要成分质量分数在 $99.9\%$ 以上。有时因含有不定组成的水分和气体杂质，或试剂本身组成不固定，使主要成分质量分数达不到要求，故不可随意认定基准物质。

### 常见基准物质

| 基准物质 | 化学式 | 主要用途 |
|------|------|------|
| 邻苯二甲酸氢钾 | $\mathrm{KHC_8H_4O_4}$ | 标定碱（NaOH等） |
| 硼砂 | $\mathrm{Na_2B_4O_7\cdot 10H_2O}$ | 标定酸（HCl等） |
| 碳酸钠 | $\mathrm{Na_2CO_3}$ | 标定酸 |
| 草酸 | $\mathrm{H_2C_2O_4\cdot 2H_2O}$ | 标定碱或 $\mathrm{KMnO_4}$ |
| 重铬酸钾 | $\mathrm{K_2Cr_2O_7}$ | 标定 $\mathrm{Na_2S_2O_3}$ |
| 草酸钠 | $\mathrm{Na_2C_2O_4}$ | 标定 $\mathrm{KMnO_4}$ |
| 氯化钠 | $\mathrm{NaCl}$ | 标定 $\mathrm{AgNO_3}$ |
| 锌 | $\mathrm{Zn}$ | 标定 EDTA |
| 碳酸钙 | $\mathrm{CaCO_3}$ | 标定 EDTA |

此外，纯金属如 $\mathrm{Ag}$、Cu、Cd、Al 等及 $\mathrm{As_2O_3}$ 也可用作基准物质，它们的质量分数一般在 $99.9\%$ 以上，甚至可达 $99.99\%$ 以上。

## 七、常见比较与易混点

| 概念 | 辨析 |
|------|------|
| 基准物质 vs 优级纯试剂 | 基准物质要求已知组成且纯度 $\geqslant 99.9\%$；优级纯（GR）仅保证杂质含量低，不一定可直接作基准 |
| 基准物质 vs 标准溶液 | 基准物质是固体试剂；标准溶液是已知浓度的溶液 |

## 八、与其他知识点的联系

- 基准物质 $\xrightarrow{\text{直接配制}}$ [[标准溶液]]
- 基准物质 $\xrightarrow{\text{标定}}$ 间接配制的[[标准溶液]]
- 基准物质 $\subset$ 标准物质（reference material）
- 与[[误差]]的关系：摩尔质量大 $\to$ 称量误差小

## 十、例题

**例** 称取基准物质 $\mathrm{K_2Cr_2O_7}$ $4.903\mathrm{~g}$，用水溶解后置于 $1\mathrm{L}$ 容量瓶中定容，求此标准溶液的浓度 $\left(M_{\mathrm{K_2Cr_2O_7}} = 294.18 \mathrm{~g\cdot mol^{-1}}\right)$。

解：$n = 4.903 / 294.18 = 0.01667\mathrm{~mol}$，$c = 0.01667 / 1.000 = 0.01667\mathrm{~mol\cdot L^{-1}}$。

## 十一、易错点

- 误用非基准物质的优级纯试剂直接配制标准溶液——绝大多数优级纯试剂也不能直接配制
- 忽略含结晶水基准物质的保存条件（如硼砂长期放置易风化失去结晶水）
- 称量前未对基准物质进行必要的干燥处理

## 十二、🎯 教学视角

### 学生常见认知误区
1. **误区**：认为"高纯"="基准"。
   - **原因**：商品标签上的"高纯""光谱纯"等术语误导。
   - **正确认知**：基准物质除纯度高外，还需组成确定、性质稳定、摩尔质量大。
   - **记忆法**："纯+稳+大=基准"。

2. **误区**：认为基准物质可以直接使用，无需干燥。
   - **原因**：忽略了吸湿性和风化问题。
   - **正确认知**：许多基准物质（如邻苯二甲酸氢钾）需在 105~110℃ 烘干至恒重。
   - **反例**：硼砂含 10 个结晶水，风化后组成改变，不能直接称量使用。

### 入门级例题（教师引导提问）
**题目**：为什么 NaOH 不能作为基准物质？

**预期解答路径**：
- NaOH 易吸收空气中的 CO₂（生成 Na₂CO₃）
- NaOH 易吸收水分（潮解）
- 因此 NaOH 固体纯度不确定、组成不固定
- 不符合基准物质的"组成与化学式严格一致"要求
- **核心结论**：NaOH 必须用邻苯二甲酸氢钾等基准物质标定。

### 与现实/直觉的连接
- **类比**：基准物质像"化学世界的黄金标准"——它的准确性和稳定性是所有浓度测量的根基。
- **生活实例**：国际千克原器（虽已退役）就是质量的"基准物质"，所有质量测量都溯源到它。
