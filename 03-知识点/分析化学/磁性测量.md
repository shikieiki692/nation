# 磁性测量

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-分析化学]]
- 对应考纲条目：[[配合物]]、[[晶体场理论]]

## 一、定义

**磁性测量（Magnetic Susceptibility Measurement）** 是通过测定物质的磁化率（magnetic susceptibility）来研究其电子结构和成键性质的分析方法。在化学竞赛中，磁性测量主要用于：

1. **确定未成对电子数**：推断中心金属离子的氧化态和电子组态
2. **区分高自旋与低自旋**：验证晶体场理论的预测
3. **判断配合物类型**：内轨型（低自旋）vs 外轨型（高自旋）
4. **辅助元素推断**：配合未知化合物的其他分析数据确定化学式

核心物理量：
- **磁化率 $\chi$**：单位体积或单位质量的磁化强度与外磁场强度之比
- **有效磁矩 $\mu_{eff}$**：与未成对电子数直接相关的实验量

## 三、核心原理

### 3.1 物质的磁性分类

| 磁性类型 | 磁化率符号 | 特征 | 典型物质 |
|:---|:---:|:---|:---|
| **抗磁性**（Diamagnetism）| $\chi < 0$ | 所有电子成对，微弱排斥磁场 | 水、苯、大多数有机化合物 |
| **顺磁性**（Paramagnetism）| $\chi > 0$ | 含未成对电子，微弱吸引磁场 | O₂、过渡金属离子 |
| **铁磁性**（Ferromagnetism）| $\chi \gg 0$ | 自发磁化，强吸引磁场 | Fe、Co、Ni |
| **反铁磁性**（Antiferromagnetism）| $\chi > 0$（小）| 相邻自旋反平行排列 | MnO、Cr₂O₃ |

**竞赛重点**：抗磁性与顺磁性的区分，以及顺磁性物质中未成对电子数的确定。

### 3.2 有效磁矩与未成对电子数

对于顺磁性物质，有效磁矩主要由电子自旋贡献（自旋-only公式）：

$$
\mu_{eff} = \sqrt{n(n+2)} \, \mu_B
$$

其中：
- $n$ = 未成对电子数
- $\mu_B$ = 玻尔磁子（Bohr magneton），$9.274 \times 10^{-24}$ J·T⁻¹

| 未成对电子数 $n$ | $\mu_{eff}$（$\mu_B$）|
|:---:|:---:|
| 0 | 0（抗磁性）|
| 1 | 1.73 |
| 2 | 2.83 |
| 3 | 3.87 |
| 4 | 4.90 |
| 5 | 5.92 |
| 6 | 6.93 |

**竞赛常用近似值**：
- $n=1$ → $\mu_{eff} \approx 1.7 \, \mu_B$
- $n=2$ → $\mu_{eff} \approx 2.8 \, \mu_B$
- $n=3$ → $\mu_{eff} \approx 3.9 \, \mu_B$
- $n=4$ → $\mu_{eff} \approx 4.9 \, \mu_B$
- $n=5$ → $\mu_{eff} \approx 5.9 \, \mu_B$

### 3.3 晶体场理论与磁性

在八面体晶体场中，d 轨道分裂为 $t_{2g}$（低能）和 $e_g$（高能）：

**高自旋 vs 低自旋**：

| d 电子数 | 弱场（高自旋）| 未成对电子 | 强场（低自旋）| 未成对电子 |
|:---:|:---|:---:|:---|:---:|
| d⁴ | $t_{2g}^3 e_g^1$ | 4 | $t_{2g}^4$ | 2 |
| d⁵ | $t_{2g}^3 e_g^2$ | 5 | $t_{2g}^5$ | 1 |
| d⁶ | $t_{2g}^4 e_g^2$ | 4 | $t_{2g}^6$ | 0 |
| d⁷ | $t_{2g}^5 e_g^2$ | 3 | $t_{2g}^6 e_g^1$ | 1 |

**竞赛要点**：
- d¹~d³ 和 d⁸~d¹⁰：高自旋和低自旋电子排布相同
- d⁴~d⁷：高自旋和低自旋有明显区别，磁性是判断依据

### 3.4 磁矩的轨道贡献

自旋-only公式忽略了轨道角动量的贡献。对于第一过渡系金属，轨道贡献通常较小（quenched），自旋-only公式近似成立。但对于某些离子（如 Co²⁺、Ni²⁺），实测磁矩略高于自旋-only值。

**修正公式**（考虑轨道贡献）：

$$
\mu_{eff} = \sqrt{4S(S+1) + L(L+1)} \, \mu_B
$$

竞赛中通常使用自旋-only公式即可。

## 四、关键结论

1. **磁矩与未成对电子数一一对应**——是推断电子组态的直接实验证据
2. **抗磁性 = 所有电子成对**——可用于确认低自旋 d⁶ 等电子组态
3. **高自旋/低自旋判断**：d⁴~d⁷ 体系中，实测磁矩与理论值对比
4. **配合物推断**：磁矩 + 元素分析 + 配位数 → 确定氧化态和电子组态
5. **O₂ 的顺磁性**是分子轨道理论的经典证据（$\pi^*_{2p}$ 上有两个未成对电子）

## 五、常见分类或情形

### 5.1 按磁性来源分类

```
磁性测量应用
├── 顺磁性（含未成对电子）
│   ├── 过渡金属离子配合物 → 推断氧化态和自旋态
│   ├── 自由基 → 确认未成对电子存在
│   └── O₂、NO 等分子 → 验证分子轨道理论
├── 抗磁性（全部电子成对）
│   ├── 低自旋配合物 → 确认强场配体作用
│   ├── 主族元素化合物 → 验证电子配对
│   └── 有机化合物 → 作为背景信号扣除
└── 特殊磁性
    ├── 铁磁性 → 固体材料研究（竞赛极少涉及）
    └── 反铁磁性 → 过渡金属氧化物（竞赛极少涉及）
```

## 六、适用条件与限制

| 条件 | 说明 |
|:---|:---|
| 适用范围 | 含未成对电子的顺磁性物质 |
| 测量方法 | Gouy法、Faraday法、SQUID（竞赛中通常给出计算值）|
| 温度影响 | 磁矩通常与温度无关（自旋-only），但磁化率服从Curie定律 |
| 轨道贡献 | 第一过渡系可忽略；第二、三过渡系需考虑 |
| 磁相互作用 | 高浓度或固体中可能存在磁耦合，导致表观磁矩异常 |

## 七、常见比较与易混点

### 7.1 高自旋 vs 低自旋的磁性判据

| 特征 | 高自旋配合物 | 低自旋配合物 |
|:---|:---|:---|
| 未成对电子 | 多 | 少或为零 |
| 磁矩 | 较大 | 较小或为零 |
| 晶体场 | 弱场（$\Delta < P$）| 强场（$\Delta > P$）|
| 配体类型 | 卤素、H₂O | CN⁻、CO、NO₂⁻ |
| 价键理论 | 外轨型 | 内轨型 |
| 示例 | [Fe(H₂O)₆]²⁺ | [Fe(CN)₆]⁴⁻ |

### 7.2 磁矩计算中的常见错误

| 错误 | 正确做法 |
|:---|:---|
| 忽略自旋-only公式的 $n(n+2)$ 形式 | 记住公式 $\mu_{eff} = \sqrt{n(n+2)}$，不是 $\sqrt{n}$ 或 $n$ |
| d⁶ 高自旋算成 6 个未成对电子 | d⁶ 高自旋：$t_{2g}^4 e_g^2$，4 个未成对电子 |
| d⁷ 低自旋算成 0 个未成对电子 | d⁷ 低自旋：$t_{2g}^6 e_g^1$，1 个未成对电子 |
| 把抗磁性误认为磁矩为 0 | 抗磁性物质 $\chi < 0$，不是 $\mu = 0$（但通常说"无顺磁性"）|

## 十、例题

### 例题 1：由磁矩推断未成对电子数

**题目**：实验测得某 Fe(II) 配合物的磁矩为 0 $\mu_B$（抗磁性），推断其电子组态和自旋态。

**解析**：
1. Fe(II) 为 d⁶ 组态
2. 磁矩为 0 → 未成对电子数 $n = 0$
3. d⁶ 组态中 $n = 0$ 对应低自旋：$t_{2g}^6 e_g^0$
4. 需要强场配体（如 CN⁻、phen）才能使 d⁶ 低自旋
5. **结论**：该配合物为低自旋 Fe(II)，如 [Fe(CN)₆]⁴⁻ 或 [Fe(phen)₃]²⁺

### 例题 2：综合推断

**题目**：某配合物由 Co、NH₃ 和 Cl 组成。元素分析表明质量分数：Co 22.0%，NH₃ 31.4%，Cl 46.6%。磁矩测定为 0 $\mu_B$。推断该配合物的化学式。

**解析**：
1. 假设 100 g 样品：Co 22.0 g，NH₃ 31.4 g，Cl 46.6 g
2. 物质的量：
   - $n_{Co} = 22.0 / 58.93 = 0.373$ mol
   - $n_{NH_3} = 31.4 / 17.03 = 1.844$ mol
   - $n_{Cl} = 46.6 / 35.45 = 1.315$ mol
3. 比值：Co : NH₃ : Cl = 0.373 : 1.844 : 1.315 = 1 : 4.94 : 3.52 ≈ 1 : 5 : 3.5
   乘以 2：≈ 2 : 10 : 7，不太整齐

   重新考虑：若配合物为 [Co(NH₃)₆]Cl₃：
   - Co : NH₃ : Cl = 1 : 6 : 3
   - 质量比：58.93 : 6×17.03 : 3×35.45 = 58.93 : 102.18 : 106.35
   - 总质量 = 267.46
   - Co% = 58.93/267.46 = 22.0% ✓
   - NH₃% = 102.18/267.46 = 38.2%（与 31.4% 不符）

   若考虑 [Co(NH₃)₅Cl]Cl₂：
   - Co : NH₃ : Cl = 1 : 5 : 3
   - 总质量 = 58.93 + 85.15 + 106.35 = 250.43
   - Co% = 23.5%（接近但不完全匹配）

   **结合磁性**：磁矩为 0 → Co(III) d⁶ 低自旋（$t_{2g}^6$）
   Co(III) 配合物通常为低自旋，如 [Co(NH₃)₆]³⁺

   验证 [Co(NH₃)₆]Cl₃：
   - 摩尔质量 = 267.5 g/mol
   - Co% = 22.0%，NH₃% = 38.2%，Cl% = 39.8%

   题目数据可能有简化或实验误差，但结合磁性（抗磁性）和 Co(III) 的常见配合物，**最可能为 [Co(NH₃)₆]Cl₃**

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| 把磁矩公式记成 $\mu = \sqrt{n}$ | 混淆不同公式 | 正确公式：$\mu = \sqrt{n(n+2)}$ | n=2 时，$\sqrt{2}$ 和 $\sqrt{8}$ 分别是多少？ |
| d⁶ 高自旋算成 6 个未成对电子 | 误以为 d⁶ = 6 个电子都未成对 | d⁶ 高自旋：$t_{2g}^4 e_g^2$，4 个未成对 | 画一下 d⁶ 高自旋的轨道填充图 |
| 忽略 Co(III) 总是低自旋 | 不知道 Co(III) 的强场特性 | Co(III) 配合物几乎总是低自旋 | 为什么 [Co(NH₃)₆]³⁺ 是抗磁性的？ |
| 把 Fe(III) 低自旋误认为高自旋 | 不了解配体场强影响 | CN⁻ 等强场配体使 Fe(III) 低自旋 | [Fe(CN)₆]³⁻ 有几个未成对电子？ |
| 忽略抗磁性背景 | 所有物质都有抗磁性贡献 | 顺磁性物质的抗磁性背景通常可忽略 | 为什么顺磁性物质的磁化率为正？ |

### 12.2 学生易踩的认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "磁矩就是磁性强弱" | 混淆磁矩与磁化率 | 磁矩是微观量（每个离子），磁化率是宏观量（单位体积） |
| "dⁿ 就有 n 个未成对电子" | 忽略电子配对 | d⁶ 高自旋只有 4 个未成对电子，低自旋为 0 |
| "所有 Fe²⁺ 配合物磁矩相同" | 忽略配体场强影响 | [Fe(H₂O)₆]²⁺（高自旋，4.9 μB）vs [Fe(CN)₆]⁴⁻（低自旋，0） |
| "抗磁性 = 没有磁性" | 忽略抗磁性普遍存在 | 所有物质都有抗磁性，只是顺磁性物质中抗磁性被掩盖 |

### 12.3 入门级例题

**题目**：计算下列离子的自旋-only磁矩：
(a) Mn²⁺（高自旋）(b) Fe³⁺（高自旋）(c) Co³⁺（低自旋）

**预期解答路径**：
1. (a) Mn²⁺ d⁵ 高自旋：$t_{2g}^3 e_g^2$，$n = 5$
   $\mu = \sqrt{5(5+2)} = \sqrt{35} \approx 5.92 \, \mu_B$
2. (b) Fe³⁺ d⁵ 高自旋：$t_{2g}^3 e_g^2$，$n = 5$
   $\mu = \sqrt{35} \approx 5.92 \, \mu_B$
3. (c) Co³⁺ d⁶ 低自旋：$t_{2g}^6$，$n = 0$
   $\mu = 0$（抗磁性）

**课堂引导提问**：
- 学生若把 d⁵ 算成 5 个未成对但用错公式："自旋-only公式是什么？"
- 学生若对 Co³⁺ 困惑："Co³⁺ 为什么是低自旋？NH₃ 是什么类型的配体？"

### 12.4 与现实/直觉的连接

**"指南针"类比顺磁性**：
- 顺磁性物质中的未成对电子像小指南针
- 外磁场作用下，这些小指南针倾向于沿磁场方向排列
- 抗磁性物质中所有指南针成对反向，相互抵消

**"停车场"类比高/低自旋**：
- d 轨道像两层停车场（$t_{2g}$ 下层，$e_g$ 上层）
- 高自旋 = 先每层停一辆车（单电子），再配对
- 低自旋 = 下层停满（配对）后，再停上层

## 十三、竞赛拓展

### 13.1 磁性与生物化学

| 体系 | 磁性变化 | 生物学意义 |
|:---|:---|:---|
| 血红蛋白脱氧 → 氧合 | Fe(II) 高自旋 → 低自旋 | 氧结合导致构象变化（T → R 态）|
| 细胞色素氧化酶 | Fe 的氧化态变化伴随磁性变化 | 电子传递链 |

### 13.2 分子磁体（竞赛边缘）

- **单分子磁体（SMM）**：具有慢磁弛豫的单个分子
- **单链磁体（SCM）**：一维配位聚合物中的磁有序
- **竞赛相关性**：极低，仅作了解
