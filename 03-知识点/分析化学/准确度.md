# 准确度

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-分析化学]]
- 对应考纲条目：[[19-误差分析]]

## 一、定义

**准确度**（accuracy）表示测量值 $x$ 与真值 $x_{\mathrm{T}}$ 的接近程度，由**系统误差**决定，用**误差**衡量。

- 准确度高 $\Longleftrightarrow$ 系统误差小 $\Longleftrightarrow$ 绝对误差/相对误差小
- 准确度是评价分析方法的首要指标

## 二、考纲对应

- 对应考纲条目：[[19-误差分析]]
- 所属模块：[[基础要求-分析化学]]
- 本知识点在考纲中的作用：准确度是分析结果可靠性的最终判据，竞赛中需要能区分准确度与精密度的不同含义。

## 三、核心原理

准确度的本质是测量值与客观真实值的差异程度。由于真值不可绝对获得，实际工作中：

1. 用**标准物质**（已知标准值）来检验准确度
2. 用**对照试验**发现系统误差
3. 用**回收率试验**评价方法准确度

回收率公式：

$$\text{回收率} = \frac{\text{测得量}}{\text{加入量}} \times 100\%$$

## 四、关键结论

### 准确度与[[精密度]]的关系

用打靶类比（四人同时测定铁矿石中 $\mathrm{Fe_2O_3}$ 含量，真值 $50.36\%$）：

| 人员 | 精密度 | 准确度 | 评价 |
|------|:---:|:---:|------|
| 甲 | 高 | 高 | 结果可靠 |
| 乙 | 高 | 低 | 有系统误差 |
| 丙 | 低 | 低 | 完全不可靠 |
| 丁 | 低 | 平均值巧合接近真值 | 不可靠，纯属巧合 |

**核心关系：**
1. 精密度高 $\neq$ 准确度高（可能存在系统误差使所有数据都偏离同一方向）
2. 准确度高 $\Rightarrow$ 精密度必然高（精密是准确的前提）
3. 精密度低时，讨论准确度无意义
4. 在确认消除系统误差后，可用精密度表达准确度

### 系统误差的校正方法

| 方法 | 说明 |
|------|------|
| **对照试验** | 用标准物质或标准方法对照，检查系统误差 |
| **空白试验** | 不加试样做全程操作，扣除试剂/器皿引入的误差 |
| **校准仪器** | 对天平、砝码、容量器皿等定期校准 |
| **方法校正** | 如用回收率校正分析结果 |

## 五、常见分类或情形

### 不同准确度要求

- 高含量组分（$>10\%$）：相对误差一般要求 $\pm 0.1\%\sim 0.3\%$
- 中含量组分（$1\%\sim 10\%$）：相对误差 $\pm 1\%$ 左右
- 微量组分（$<1\%$）：相对误差可放宽至 $\pm 5\%\sim 50\%$

## 六、适用条件与限制

1. 准确度评价必须知道真值或标准值，真值不可得是根本限制
2. 回收率试验要求加入的组分形态与试样中相同
3. 空白试验只能扣除恒定系统误差，不能解决比例系统误差
4. 多次测量取平均值只能减小随机误差，不能消除系统误差

## 七、常见比较与易混点

| 对比项 | [[准确度]] | [[精密度]] |
|------|:---:|:---:|
| 含义 | 接近真值的程度 | 数据间相互接近程度 |
| 决定因素 | 系统误差 | 随机误差 |
| 衡量指标 | 误差（$E, E_{\mathrm{r}}$） | 偏差 $s$, RSD |
| 与测量次数关系 | 不随 $n$ 增加而改善 | 平均值精密度随 $\sqrt{n}$ 提高 |
| 打靶类比 | 弹着点距靶心距离 | 弹着点之间分散程度 |

## 十、例题

### 例题 1
**题目：** 测定纯 $\mathrm{BaCl_2 \cdot 2H_2O}$ 中 Ba 含量，4 次结果：56.14%, 56.16%, 56.17%, 56.13%。已知理论值为 56.22%，评价准确度。
**分析：** 求 $\bar{x}$ 和 $E_{\mathrm{r}}$。
**解答：**
$\bar{x} = 56.15\%$，$E = 56.15\% - 56.22\% = -0.07\%$
$E_{\mathrm{r}} = \frac{-0.07\%}{56.22\%} \times 100\% = -0.12\%$
相对误差仅为 $-0.12\%$，准确度良好。
**反思：** 负号表示结果偏低，可能是沉淀溶解损失（方法误差）。

### 例题 2
**题目：** 某方法测定样品回收率为 85%，分析结果应如何校正？
**解答：** 校正值 = 测量值 / 85%。但回收率过低说明方法有严重系统误差，应改进方法而非简单校正。

## 十一、易错点

1. 把"数据重现性好"等同于"准确度高"——可能所有数据都偏高或偏低
2. 认为多次测量取平均值可以提高准确度——只能提高精密度（减小随机误差），不能消除系统误差
3. 回收率接近 100% 不等于没有系统误差——可能存在互相抵消的误差
4. 混淆准确度（accuracy）和精密度（precision）的英文术语
