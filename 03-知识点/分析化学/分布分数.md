# 分布分数

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-分析化学]]
- 对应考纲条目：[[18-容量分析]]

## 一、定义

**分布分数**（distribution fraction, $\delta$）表示酸碱平衡体系中某特定存在形态的浓度占总浓度的分数。它描述了不同 pH 条件下各形态的相对含量分布。

## 二、考纲对应

- 对应考纲条目：[[18-容量分析]]
- 本知识点在考纲中的作用：分布分数是理解缓冲溶液、滴定曲线和形态分析的基础工具。

## 三、核心原理

### 一元酸分布分数

对于一元弱酸 HA：

$$\delta_{\mathrm{HA}} = \frac{[\mathrm{H}^+]}{[\mathrm{H}^+] + K_\mathrm{a}}, \quad \delta_{\mathrm{A}^-} = \frac{K_\mathrm{a}}{[\mathrm{H}^+] + K_\mathrm{a}}$$

当 $\mathrm{pH} = \mathrm{p}K_\mathrm{a}$ 时，$\delta_{\mathrm{HA}} = \delta_{\mathrm{A}^-} = 0.5$，两种形态各占一半。

### 多元酸分布分数

对于二元弱酸 $H_2A$：

$$\delta_{H_2A} = \frac{[\mathrm{H}^+]^2}{[\mathrm{H}^+]^2 + K_{\mathrm{a}_1}[\mathrm{H}^+] + K_{\mathrm{a}_1}K_{\mathrm{a}_2}}$$

$$\delta_{\mathrm{HA}^-} = \frac{K_{\mathrm{a}_1}[\mathrm{H}^+]}{[\mathrm{H}^+]^2 + K_{\mathrm{a}_1}[\mathrm{H}^+] + K_{\mathrm{a}_1}K_{\mathrm{a}_2}}$$

$$\delta_{\mathrm{A}^{2-}} = \frac{K_{\mathrm{a}_1}K_{\mathrm{a}_2}}{[\mathrm{H}^+]^2 + K_{\mathrm{a}_1}[\mathrm{H}^+] + K_{\mathrm{a}_1}K_{\mathrm{a}_2}}$$

### 酒石酸分布分数实例

酒石酸：$\mathrm{p}K_{\mathrm{a}_1} = 3.04$，$\mathrm{p}K_{\mathrm{a}_2} = 4.37$

- $\delta(\mathrm{HA}^-)$ 在 $\mathrm{pH} = 3.655$ 处达到最大值 $0.673$
- 这是竞赛中经典的多元酸分布计算训练题
- 计算验证：$\mathrm{pH} = (3.04 + 4.37)/2 = 3.705$ 时，$\delta_{\mathrm{HA}^-}$ 接近最大值（精确值需对 $\delta$ 求导）

## 四、关键结论

### 分布分数的核心性质

1. **归一性**：所有形态的分布分数之和为 1，即 $\sum \delta_i = 1$
2. **pH = pKa 转折点**：一元酸在 $\mathrm{pH} = \mathrm{p}K_\mathrm{a}$ 时 $\delta = 0.5$；多元酸在各 $\mathrm{p}K_\mathrm{a}$ 处相邻两形态的 $\delta$ 相等
3. **形态优势区**：$\mathrm{pH} < \mathrm{p}K_\mathrm{a} - 1$ 时酸式形态占主导；$\mathrm{pH} > \mathrm{p}K_\mathrm{a} + 1$ 时碱式形态占主导

## 五、竞赛常考题型

### 题型 1：给定 pH 求各形态分布分数

**题目特征**：已知弱酸的 $K_\mathrm{a}$ 值和溶液 pH，求某形态的分布分数 $\delta$。

**解题步骤**：
1. 由 pH 计算 $[\mathrm{H}^+]$
2. 代入分布分数公式（一元酸或多元酸）
3. 验证 $\sum \delta_i = 1$

**典型例题**：酒石酸（$\mathrm{p}K_{\mathrm{a}_1}=3.04$，$\mathrm{p}K_{\mathrm{a}_2}=4.37$）在 $\mathrm{pH}=4.00$ 时，求 $\delta_{\mathrm{HA}^-}$
- 计算：$\delta_{\mathrm{HA}^-} = \frac{K_{\mathrm{a}_1}[\mathrm{H}^+]}{[\mathrm{H}^+]^2 + K_{\mathrm{a}_1}[\mathrm{H}^+] + K_{\mathrm{a}_1}K_{\mathrm{a}_2}} = 0.58$

### 题型 2：判断优势形态及形态优势区

**题目特征**：给定 pH 范围或缓冲体系，判断哪种形态占主导。

**解题步骤**：
1. 比较 pH 与各 $\mathrm{p}K_\mathrm{a}$ 的关系
2. 利用规则：$\mathrm{pH} < \mathrm{p}K_\mathrm{a} - 1$ 时酸式形态主导；$\mathrm{pH} > \mathrm{p}K_\mathrm{a} + 1$ 时碱式形态主导
3. 在 $\mathrm{pH} = \mathrm{p}K_\mathrm{a}$ 处，相邻两形态 $\delta$ 相等

**典型例题**：$\mathrm{H}_2\mathrm{CO}_3$（$\mathrm{p}K_{\mathrm{a}_1}=6.35$，$\mathrm{p}K_{\mathrm{a}_2}=10.33$）在血液 pH 7.4 时的主要形态
- 分析：$\mathrm{p}K_{\mathrm{a}_1} < 7.4 < \mathrm{p}K_{\mathrm{a}_2}$，故 $\mathrm{HCO}_3^-$ 为主要形态

### 题型 3：分布分数极值点计算

**题目特征**：求多元酸中间形态（如 $\mathrm{HA}^-$）分布分数最大时的 pH。

**解题步骤**：
1. 对 $\delta_{\mathrm{HA}^-}$ 关于 $[\mathrm{H}^+]$ 求导，令导数为零
2. 解得 $[\mathrm{H}^+] = \sqrt{K_{\mathrm{a}_1}K_{\mathrm{a}_2}}$，即 $\mathrm{pH} = \frac{1}{2}(\mathrm{p}K_{\mathrm{a}_1} + \mathrm{p}K_{\mathrm{a}_2})$
3. 代入求最大 $\delta$ 值

**典型例题**：酒石酸 $\delta_{\mathrm{HA}^-}$ 最大时的 pH 和 $\delta_\mathrm{max}$
- $\mathrm{pH}_\mathrm{max} = (3.04 + 4.37)/2 = 3.705$（精确值 3.655，因近似处理有微小偏差）
- $\delta_\mathrm{max} = 0.673$

## 六、常见错误

### 错误 1：分布分数公式中 $K_\mathrm{a}$ 与 $[\mathrm{H}^+]$ 的指数对应错误

**错误做法**：多元酸分布分数中，将 $K_{\mathrm{a}_1}$ 与 $[\mathrm{H}^+]^{n-1}$ 的对应关系搞混，导致分子分母写错。
**正确做法**：对于 $n$ 元酸 $H_nA$，分母始终是 $[\mathrm{H}^+]^n + K_{\mathrm{a}_1}[\mathrm{H}^+]^{n-1} + K_{\mathrm{a}_1}K_{\mathrm{a}_2}[\mathrm{H}^+]^{n-2} + \cdots + K_{\mathrm{a}_1}\cdots K_{\mathrm{a}_n}$。$\delta_{H_nA}$ 的分子是 $[\mathrm{H}^+]^n$，每脱一个质子分子中 $[\mathrm{H}^+]$ 的指数减 1，同时多乘一个 $K_\mathrm{a}$。

### 错误 2：将分布分数与平衡浓度混淆

**错误做法**：直接用 $\delta$ 代替 $[\mathrm{A}^-]$ 参与电荷平衡或物料平衡计算，忘记乘以总浓度 $c$。
**正确做法**：$[\mathrm{A}^-] = \delta_{\mathrm{A}^-} \cdot c$。分布分数是无量纲的分数，必须乘以总浓度才是实际浓度。

### 错误 3：pH = pKa 时认为所有形态 $\delta$ 都等于 0.5

**错误做法**：认为多元酸在任意 $\mathrm{p}K_\mathrm{a}$ 处所有形态的 $\delta$ 都等于 0.5。
**正确做法**：在 $\mathrm{pH} = \mathrm{p}K_{\mathrm{a}_i}$ 处，**仅相邻两形态**（$H_{n-i+1}A^{(i-1)-}$ 和 $H_{n-i}A^{i-}$）的 $\delta$ 相等且为 0.5，其他形态的 $\delta$ 不为 0.5。

### 错误 4：忽略归一性验证

**错误做法**：计算完各形态 $\delta$ 后不加和验证，导致计算错误未被发现。
**正确做法**：计算后必须验证 $\sum \delta_i = 1$。若不为 1，说明公式代入或计算有误。

### 错误 5：极值点计算时直接取算术平均

**错误做法**：求 $\delta_{\mathrm{HA}^-}$ 最大值时，直接认为 $\mathrm{pH} = (\mathrm{p}K_{\mathrm{a}_1} + \mathrm{p}K_{\mathrm{a}_2})/2$ 就是精确极值点。
**正确做法**：$\mathrm{pH} = \frac{1}{2}(\mathrm{p}K_{\mathrm{a}_1} + \mathrm{p}K_{\mathrm{a}_2})$ 是近似值（假设中间形态与其他形态的差异对称）。精确极值点需对 $\delta$ 求导得到，对于酒石酸精确值为 3.655 而非 3.705。

## 十一、教学视角

### 学生最难理解的是什么

分布分数的**物理意义**（为什么可以用一个公式描述所有形态的相对含量）和**多元酸分母的构造规律**（为什么分母是 $[\mathrm{H}^+]^n + K_{\mathrm{a}_1}[\mathrm{H}^+]^{n-1} + \cdots$ 的形式）。学生往往能背一元酸公式，但面对多元酸时不知道分子该写什么。

### 如何铺垫

1. **从物料平衡出发**：设总浓度为 $c$，则 $c = [\mathrm{H}_n\mathrm{A}] + [\mathrm{H}_{n-1}\mathrm{A}^-] + \cdots + [\mathrm{A}^{n-}]$。将每一项用 $[\mathrm{H}^+]$ 和 $K_\mathrm{a}$ 表示，自然导出分布分数。
2. **用具体数值建立直觉**：以醋酸（$\mathrm{p}K_\mathrm{a}=4.76$）为例，分别计算 $\mathrm{pH}=3.76$、$4.76$、$5.76$ 时的 $\delta_{\mathrm{HA}}$ 和 $\delta_{\mathrm{A}^-}$，让学生看到"pH 每远离 pKa 一个单位，优势形态的占比增加约 10 倍"。
3. **再推广到多元酸**：以酒石酸为例，画出 $\delta$ 随 pH 变化的示意图，标注三个形态的优势区和两个交点。

### 常见误区

- **误区 1**：认为分布分数只适用于酸碱体系——实际上配位平衡中配体的各质子化形态、氧化还原中各氧化态的分布都可用类似方法处理。
- **误区 2**：将 $\delta$ 当作浓度使用——$\delta$ 是无量纲分数，必须乘以总浓度才是平衡浓度。
- **误区 3**：认为 $\mathrm{pH} = \mathrm{p}K_\mathrm{a}$ 时所有形态等量——仅相邻两形态等量（各 0.5），其他形态不为 0.5。

### 推荐例题

1. **入门**：计算 0.10 M HAc 溶液在 $\mathrm{pH}=4.76$ 时的 $[\mathrm{HA}]$ 和 $[\mathrm{A}^-]$（训练分布分数与浓度的转换）。
2. **进阶**：酒石酸在 $\mathrm{pH}=3.655$ 时 $\delta_{\mathrm{HA}^-}$ 的最大值计算（训练多元酸分布分数和极值概念）。
3. **综合**：$\mathrm{H}_3\mathrm{PO}_4$ 在 $\mathrm{pH}=7.20$（第二 $\mathrm{p}K_\mathrm{a}$）时各形态的 $\delta$ 值（训练三元酸分布分数，验证 $\mathrm{HPO}_4^{2-}$ 和 $\mathrm{H}_2\mathrm{PO}_4^-$ 的 $\delta$ 各为 0.5）。
