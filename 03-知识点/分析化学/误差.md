# 误差

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-分析化学]]
- 对应考纲条目：[[19-误差分析]]

## 一、定义

**误差**（error）是测量值 $x$ 与真值 $x_{\mathrm{T}}$ 之间的差值，反映[[准确度]]的高低。误差越小，准确度越高。

- **绝对误差**：$E = x - x_{\mathrm{T}}$（有单位，有正负）
- **相对误差**：$E_{\mathrm{r}} = \frac{E}{x_{\mathrm{T}}} \times 100\%$

真值在严格意义上无法获得，分析化学中用理论真值、计量学约定真值或标准值（相对真值）代替。

## 三、核心原理

### 误差三分法

误差按其来源和性质分为三类：

| 类型 | 特征 | 产生原因 | 处理方法 |
|------|------|------|------|
| **系统误差** | 固定方向、可重现、可测 | 方法缺陷、仪器不准、试剂不纯、操作习惯 | 对照试验、空白试验、校准仪器 |
| **随机误差** | 无固定方向、不可预测、服从统计规律 | 环境波动（温度、湿度、气压）等不可控因素 | 多次测量、统计处理 |
| **过失误差** | 明显异常、本可避免 | 操作失误（溅失、读错、算错） | 剔除该数据，重做 |

### 系统误差的四个子类

1. **方法误差**：反应不完全、副反应、终点与计量点不一致
2. **仪器与试剂误差**：天平砝码不准、容量器皿刻度不准、试剂含水或杂质
3. **操作误差**：洗涤过度或不足、灼烧温度不当、终点判断偏深或偏浅
4. **主观误差**：个人读数习惯、先入为主向第一次结果靠拢（网课将"主观误差"并入操作误差）

> **来源：周坤无机新课（Batch 5）**：分析天平每次称量绝对误差约 $0.1\ \mathrm{mg}$；称量两次（空盘+试样）最大误差 $\pm 0.2\ \mathrm{mg}$。若要求相对误差 $< 0.1\%$，则称量量应 $> 0.2\ \mathrm{g}$。这一数据体现了"结果不能比测量更精确"的定量思维。

## 四、关键结论

1. 系统误差决定准确度，随机误差决定[[精密度]]
2. 系统误差理论上可测可校，随机误差不可免但可统计
3. 消除系统误差后，可用精密度表达准确度
4. 公差是生产部门对误差的允许限量：高含量（>80%）允许相对误差约 $\pm 0.4\%$，微量（<1%）可达 $\pm 5\%\sim 100\%$

## 五、常见分类或情形

### 误差的传递

设 $R$ 为分析结果，$A, B, C$ 为测量值：

**系统误差传递：**
- 加减法 $R = A + B - C$：$E_R = E_A + E_B - E_C$（绝对误差代数和）
- 乘除法 $R = \frac{AB}{C}$：$\frac{E_R}{R} = \frac{E_A}{A} + \frac{E_B}{B} - \frac{E_C}{C}$（相对误差代数和）
- 指数关系 $R = mA^n$：$\frac{E_R}{R} = n\frac{E_A}{A}$
- 对数关系 $R = m\lg A$：$E_R = 0.434m\frac{E_A}{A}$

**随机误差传递：**
- 加减法：$s_R^2 = s_A^2 + s_B^2 + s_C^2$（方差之和，不论加减）
- 乘除法：$\frac{s_R^2}{R^2} = \frac{s_A^2}{A^2} + \frac{s_B^2}{B^2} + \frac{s_C^2}{C^2}$（相对方差之和）

**极值误差**（最不利估计）：
- 加减法：$|E_R|_{\max} = |E_A| + |E_B| + |E_C|$
- 乘除法：$\left|\frac{E_R}{R}\right|_{\max} = \left|\frac{E_A}{A}\right| + \left|\frac{E_B}{B}\right| + \left|\frac{E_C}{C}\right|$

## 六、适用条件与限制

1. 误差传递公式适用于各测量值相互独立的情况
2. 系统误差传递公式要求各误差符号已知
3. 随机误差传递基于正态分布假设
4. 称量时需读两次（空盘+试样），故称量标准偏差 $s_m = \sqrt{2}s \approx 0.14\text{ mg}$（$s = 0.10\text{ mg}$ 时）

## 七、常见比较与易混点

| 对比项 | 系统误差 | 随机误差 |
|------|------|------|
| 方向 | 单向（总是偏高或偏低） | 双向（忽高忽低） |
| 可测性 | 可测 | 不可测 |
| 可校正性 | 可通过方法校正 | 不可校正，只能统计 |
| 影响指标 | [[准确度]] | [[精密度]] |
| 与测量次数关系 | 不随 n 增加而减小 | 平均值误差随 $\sqrt{n}$ 减小 |

## 九、典型题型

- 题型-误差分析：判断某误差属于系统/随机/过失
- 误差传递计算：由各步测量误差推算最终结果误差
- 极值误差估计：估算最不利情况下的最大可能误差

## 十、例题

### 例题 1
**题目：** 用分析天平称样（$s = 0.10\text{ mg}$），求称量标准偏差。
**分析：** 每次称量需读两次平衡点，属加减运算。
**解答：** $s_m = \sqrt{s^2 + s^2} = \sqrt{2} \times 0.10 = 0.14\text{ mg}$
**反思：** 即使天平精度为 $0.1\text{ mg}$，称量误差也是 $0.14\text{ mg}$，这是随机误差传递的典型应用。

### 例题 2
**题目：** 滴定管读数误差 $\pm 0.01\text{ mL}$，滴定体积 $V = 22.05\text{ mL}$（两次读数），求极值误差。
**解答：** $\Delta V = |\pm 0.01| + |\pm 0.01| = 0.02\text{ mL}$，故 $V = (22.05 \pm 0.02)\text{ mL}$

## 十一、易错点

1. 将过失误差当作随机误差处理，纳入统计计算
2. 混淆"误差"（与真值比较）与"[[偏差]]"（与平均值比较）
3. 系统误差的传递中，加减法传递的是**绝对误差**，乘除法传递的是**相对误差**
4. 随机误差的传递中加减法用的是方差相加（不论加减），不是标准偏差直接相加
5. 极值误差按绝对值累加，用于最坏情况估计而非精确计算
