# 热重分析

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-分析化学]]
- 对应考纲条目：[[热分析]]、[[化学动力学]]

## 一、定义

**热重分析（Thermogravimetric Analysis, TGA）** 是在程序控制温度下，测量物质质量随温度或时间变化关系的热分析技术。TGA 曲线（热重曲线）以质量损失率（%）或剩余质量（mg）为纵坐标，温度（°C）或时间为横坐标。

在化学竞赛中，TGA 主要用于：

1. **确定水合物中结晶水数目**：通过阶梯失重计算
2. **推断热分解产物**：根据失重百分比推断分解路径
3. **确定化学式**：结合元素分析和失重数据推断化合物组成
4. **评估热稳定性**：比较不同物质的起始分解温度

## 三、核心原理

### 3.1 TGA 基本原理

$$
\text{质量损失率} = \frac{m_0 - m_t}{m_0} \times 100\%
$$

其中 $m_0$ 为初始质量，$m_t$ 为 t 时刻（或某温度下）的质量。

**典型 TGA 曲线特征**：
- **平台区**：质量恒定，无热分解或挥发
- **失重台阶**：质量骤降，对应某一分解或挥发过程
- **台阶高度**：与该步失重的物质含量成正比

### 3.2 热分解的典型过程

#### 3.2.1 结晶水合物脱水

$$
\ce{M_xA_y \cdot nH2O -> M_xA_y + nH2O ^}
$$

**理论失重**：
$$
\Delta m\% = \frac{n \times M_{H_2O}}{M_{\text{水合物}}} \times 100\%
$$

**示例**：CuSO₄·5H₂O 脱水
- 摩尔质量 = 249.7 g/mol
- 5 个 H₂O = 5 × 18.0 = 90.0 g/mol
- 理论失重 = 90.0 / 249.7 × 100% = 36.0%

实际脱水通常分步进行：
- 30~120°C：失去 2 个 H₂O（约 14.4%）
- 120~250°C：再失去 2 个 H₂O（约 14.4%）
- 250~300°C：失去最后 1 个 H₂O（约 7.2%）

#### 3.2.2 碳酸盐分解

$$
\ce{MCO3 -> MO + CO2 ^}
$$

**理论失重**（CO₂ 逸出）：
$$
\Delta m\% = \frac{M_{CO_2}}{M_{MCO_3}} \times 100\%
$$

| 碳酸盐 | 摩尔质量 | CO₂ 质量 | 理论失重 |
|:---|:---:|:---:|:---:|
| CaCO₃ | 100.1 | 44.0 | 44.0% |
| MgCO₃ | 84.3 | 44.0 | 52.2% |
| BaCO₃ | 197.3 | 44.0 | 22.3% |
| Na₂CO₃ | 106.0 | 44.0 | 41.5% |

#### 3.2.3 氢氧化物分解

$$
\ce{M(OH)2 -> MO + H2O ^}
$$

**理论失重**：
$$
\Delta m\% = \frac{M_{H_2O}}{M_{M(OH)_2}} \times 100\%
$$

#### 3.2.4 复杂分解（多步）

某些化合物经历多步分解，TGA 曲线显示多个台阶：

**示例**：碱式碳酸铜 Cu₂(OH)₂CO₃
- 200~300°C：分解为 CuO + H₂O + CO₂
- 理论失重 = (18.0 + 44.0) / 221.1 × 100% = 28.0%

### 3.3 失重数据的定量分析

**从失重推断化学式**：

已知某水合物 MX·nH₂O 失重 Δm%，求 n：

$$
n = \frac{\Delta m\% \times M_{MX}}{(100\% - \Delta m\%) \times M_{H_2O}}
$$

或更直接地：

$$
\frac{m_{H_2O}}{m_{MX}} = \frac{\Delta m\%}{100\% - \Delta m\%} = \frac{n \times 18.0}{M_{MX}}
$$

## 四、关键结论

1. **TGA 是定量工具**——失重百分比可直接换算为物质组成
2. **多步失重对应多步分解**——每个台阶高度反映该步失去的物质含量
3. **失重温度反映热稳定性**——分解温度越高，热稳定性越好
4. **结合其他分析（XRD、IR）可确定分解产物**
5. **空气中 vs 惰性气氛中 TGA 结果可能不同**——氧化/还原反应的影响

## 五、常见分类或情形

### 5.1 按失重类型分类

```
TGA 失重类型
├── 物理过程
│   ├── 吸附水/自由水脱附（~100°C）
│   └── 挥发性物质蒸发
├── 化学分解
│   ├── 结晶水脱水（100~300°C）
│   ├── 氢氧化物分解（200~400°C）
│   ├── 碳酸盐分解（400~900°C）
│   ├── 硫酸盐分解（高温，>800°C）
│   └── 有机化合物热分解（复杂多步）
└── 氧化/还原
    ├── 空气中氧化增重（金属 → 氧化物）
    └── 惰性气氛中还原失重
```

### 5.2 常见化合物的 TGA 特征

| 化合物类型 | 典型失重温度 | 失重产物 | 备注 |
|:---|:---|:---|:---|
| 水合物 | 100~300°C | H₂O | 可能分步脱水 |
| 氢氧化物 | 200~400°C | H₂O | 如 Fe(OH)₃ → Fe₂O₃ |
| 碳酸盐 | 400~900°C | CO₂ | 碱金属碳酸盐更稳定 |
| 草酸盐 | 200~500°C | CO + CO₂ | 多步分解 |
| 硝酸盐 | 200~600°C | NO₂ + O₂ | 可能爆炸 |
| 铵盐 | 100~300°C | NH₃ + 酸 | 如 NH₄Cl |

## 六、适用条件与限制

| 条件 | 说明 |
|:---|:---|
| 样品量 | 通常 5~20 mg（太多传热不均，太少称量误差大）|
| 升温速率 | 常用 5~20°C/min（影响分解温度）|
| 气氛 | N₂（惰性）、空气（氧化）、CO₂（碳酸盐稳定）|
| 检测限 | 约 0.1% 的质量变化 |
| 局限性 | 无法区分同时发生的多个过程；需结合 DSC/DTA 确定热效应 |

## 七、常见比较与易混点

### 7.1 TGA vs DSC vs DTA

| 特征 | TGA | DSC | DTA |
|:---|:---|:---|:---|
| 测量信号 | 质量变化 | 热流差 | 温差 |
| 信息类型 | 定量组成 | 热效应（焓变）| 热效应（定性）|
| 主要应用 | 失重/增重分析 | 相变热、反应热 | 相变温度、分解温度 |
| 定量性 | 质量定量 | 热量定量 | 半定量 |
| 联用 | 常与 DSC/DTA 联用（TG-DSC）| — | — |

### 7.2 空气中 vs 惰性气氛中的 TGA

| 气氛 | 特点 | 应用 |
|:---|:---|:---|
| **空气** | 氧化性气氛 | 有机物燃烧、金属氧化 |
| **N₂/Ar** | 惰性气氛 | 热分解、脱水（避免氧化干扰）|
| **CO₂** | 碳酸盐稳定 | 防止碳酸盐过早分解 |
| **O₂** | 强氧化 | 煤/生物质燃烧特性 |

## 十、例题

### 例题 1：结晶水数目的确定

**题目**：某水合物 CaCl₂·nH₂O 在 200°C 以下完全脱水，失重率为 49.3%。求 n。

**解析**：
1. CaCl₂ 摩尔质量 = 40.08 + 2×35.45 = 110.98 g/mol
2. 设水合物摩尔质量 = 110.98 + 18.02n
3. 失重率 = $\frac{18.02n}{110.98 + 18.02n} = 0.493$
4. 解方程：18.02n = 0.493 × (110.98 + 18.02n)
5. 18.02n = 54.71 + 8.88n
6. 9.14n = 54.71
7. **n ≈ 6**

验证：CaCl₂·6H₂O 理论失重 = 108.12 / 219.10 = 49.3% ✓

### 例题 2：碱式碳酸盐的分解

**题目**：某碱式碳酸铜在加热时发生两步失重：
- 第一步：200~300°C，失重 11.2%
- 第二步：300~500°C，失重 16.8%
推断该化合物的化学式。

**解析**：
1. 总失重 = 11.2% + 16.8% = 28.0%
2. 碱式碳酸铜通式：Cuₓ(OH)ᵧ(CO₃)z
3. 常见形式 Cu₂(OH)₂CO₃（孔雀石）：
   - 摩尔质量 = 2×63.55 + 2×17.01 + 60.01 = 221.13
   - 失去 H₂O + CO₂ = 18.02 + 44.01 = 62.03
   - 理论失重 = 62.03 / 221.13 = 28.1%
4. 分步失重：
   - 若先失去 H₂O：18.02 / 221.13 = 8.1%（与 11.2% 不完全吻合）
   - 若先失去部分 CO₂ 或同时失去，可能对应不同机理

5. 另一种形式 Cu₃(OH)₂(CO₃)₂（蓝铜矿）：
   - 摩尔质量 = 3×63.55 + 2×17.01 + 2×60.01 = 344.69
   - 失去 H₂O + 2CO₂ = 18.02 + 88.02 = 106.04
   - 理论失重 = 106.04 / 344.69 = 30.8%

6. 根据总失重 28.0% 最接近 Cu₂(OH)₂CO₃ 的 28.1%

**答案**：该化合物最可能为 **Cu₂(OH)₂CO₃**（孔雀石）

> 实际分步失重可能与理论值有偏差，因脱水/脱CO₂可能部分重叠。

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| 用无水物质量计算失重 | 混淆基准 | 失重百分比以初始样品总质量为基准 | 10 g CuSO₄·5H₂O 脱水后剩多少克？ |
| 忽略分步脱水 | 假设所有水一次失去 | 结晶水通常分步失去，结合力不同 | 为什么 CuSO₄·5H₂O 的 TGA 有多个台阶？ |
| 把增重当失重 | 氧化反应导致质量增加 | 注意气氛：空气中金属可能氧化增重 | Fe 粉在空气中加热，TGA 曲线会怎样？ |
| 忽略分解产物的摩尔质量 | 计算错误 | 确认分解方程式，计算正确的理论失重 | CaCO₃ 分解的理论失重是多少？ |
| 混淆 TGA 和 DSC | 两者测量不同物理量 | TGA 测质量，DSC 测热量 | TGA 台阶对应 DSC 的什么特征？ |

## 十二、🎯 教学视角

### 12.1 学习路径

```
前置：化学计量 → 热力学（分解反应）→ 化学动力学
     ↓
本节：TGA 原理 → 失重计算 → 分解产物推断 → 综合化学式推断
     ↓
后续：热分析联用技术（TG-DSC）→ 材料热稳定性评估
```

**建议课时**：1.5 课时（原理 0.5 + 计算练习 0.5 + 综合推断 0.5）

### 12.2 学生易踩的认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "失重就是质量减少" | 忽略可能增重 | 强调：TGA 可以测增重（氧化反应）|
| "所有水合物一步脱水" | 不了解结晶水结合力差异 | 展示 CuSO₄·5H₂O 的分步脱水实例 |
| "失重温度就是分解温度" | 忽略升温速率影响 | 说明：升温速率越快，表观分解温度越高 |
| "TGA 可以确定分解产物" | 忽略 TGA 只能给出质量变化 | 强调：需结合 XRD、IR 等确定产物 |

### 12.3 入门级例题

**题目**：FeC₂O₄·2H₂O 在惰性气氛中加热，经历以下失重：
- 100~200°C：失重 20.1%
- 200~400°C：失重 39.9%
推断各步分解产物。

**预期解答路径**：
1. FeC₂O₄·2H₂O 摩尔质量 = 55.85 + 2×12.01 + 4×16.00 + 2×18.02 = 179.90
2. 2H₂O = 36.04，理论失重 = 36.04/179.90 = 20.0% → 第一步失去 2H₂O
3. 剩余 FeC₂O₄（摩尔质量 143.86）
4. 第二步失重 39.9%，对应 143.86 × 0.399 = 57.4 g
5. FeC₂O₄ → FeO + CO + CO₂，失重 = 28.01 + 44.01 = 72.02？不对

   重新考虑：FeC₂O₄ → FeO + CO₂ + CO
   - 失重 = 44.01 + 28.01 = 72.02
   - 失重率 = 72.02/143.86 = 50.1%

   若第二步失重 39.9%，可能部分氧化或不同机理。
   更可能：FeC₂O₄·2H₂O → FeC₂O₄ → FeCO₃ + CO → FeO + CO₂

   简化处理：第二步失去 CO + CO₂（约 72 g），但实测 39.9% 对应 57.4 g，
   可能为 FeC₂O₄ → FeO + CO₂ + CO 的部分过程或测量条件不同。

   **竞赛中通常给出简化数据**，学生只需识别：
   - 第一步：脱水
   - 第二步：草酸盐分解

### 12.4 与现实/直觉的连接

**"烤面包"类比热分解**：
- 面团中的水分蒸发 → 类似水合物脱水（TGA 第一个台阶）
- 有机物碳化 → 类似有机化合物分解（TGA 后续台阶）
- 温度越高，分解越彻底 → TGA 最终平台为稳定氧化物

**"剥洋葱"类比多步分解**：
- 外层先失去（结合力弱的水）
- 内层后失去（结合力强的水或与骨架结合的部分）
- 每层对应 TGA 的一个台阶
