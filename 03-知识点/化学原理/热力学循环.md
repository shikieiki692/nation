# 热力学循环（Thermodynamic Cycle）

## 一、定义

热力学循环是指系统经过一系列状态变化后回到初始状态的过程。由于循环过程的**状态函数变化为零**（$\Delta U = 0$, $\Delta H = 0$），热力学循环是计算难以直接测量的热力学量（如晶格能、键能）的**核心工具**。

## 三、核心原理

### 3.1 Hess 定律

化学反应的热效应只与始态和终态有关，与路径无关：

$$\Delta H_{\text{总}} = \sum_i \Delta H_i$$

### 3.2 Born-Haber 循环

用于计算离子晶体的**晶格能**（$\Delta H_L$）：

以 NaCl 为例：
$$\Delta H_f = \Delta H_{\text{sub}} + \frac{1}{2}D + I + A + \Delta H_L$$

| 步骤 | 过程 | 符号 |
|:---|:---|:---:|
| 1 | Na(s) → Na(g) | 升华焓 $\Delta H_{\text{sub}}$ |
| 2 | Na(g) → Na⁺(g) + e⁻ | 电离能 $I$ |
| 3 | ½Cl₂(g) → Cl(g) | ½解离能 $\frac{1}{2}D$ |
| 4 | Cl(g) + e⁻ → Cl⁻(g) | 电子亲合能 $A$ |
| 5 | Na⁺(g) + Cl⁻(g) → NaCl(s) | 晶格焓 $\Delta H_L$ |

### 3.3 Born-Fajans-Haber 循环的变化

- 可用于计算电子亲合能、质子亲合能
- 关键：气相离子生成焓 + 晶格焓 = 晶体生成焓

## 四、关键结论

### 4.1 Carnot 循环

- **效率**：$\eta = 1 - \frac{T_c}{T_h}$
- **意义**：确定了热机效率的理论上限
- **逆 Carnot 循环**：制冷机和热泵的理想模型

### 4.2 制冷循环

- **蒸发制冷**：利用液体蒸发吸热
- **制冷系数**：$\varepsilon = \frac{Q_c}{W} = \frac{T_c}{T_h - T_c}$

### 4.3 实际循环

- **Otto 循环**：汽油机
- **Diesel 循环**：柴油机
- **Rankine 循环**：蒸汽动力装置

### 4.4 电化学中的热力学循环

$$\Delta G = -nFE$$
$$\Delta S = nF\left(\frac{\partial E}{\partial T}\right)_P$$
$$\Delta H = -nFE + nFT\left(\frac{\partial E}{\partial T}\right)_P$$

通过测定不同温度下的电动势，可同时获得 $\Delta G$、$\Delta S$、$\Delta H$。

## 十一、易错点

1. **晶格能的符号约定**：$U$（晶格能）通常取正值（拆散晶体所需能量），但 Born-Haber 循环中晶格形成焓为负值
2. **Carnot 效率是上限**：实际热机效率一定低于 Carnot 效率
3. **制冷系数可以 > 1**：制冷机移走的热量可以大于输入的功
4. **状态函数在循环中变化为零**：但过程量（Q, W）不为零

## 十二、🎯 教学视角

### 12.1 学生典型认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 口诀 |
|:---|:---|:---|:---|
| "Born-Haber 循环只用于离子化合物" | 课本示例都是 NaCl 型 | 热力学循环思想（状态函数法）适用于任何过程——溶解热、水合能、键能都可以用循环关联 | "循环是方法，不限于离子" |
| "循环方向随便画，反正加和为零" | 对路径无关性理解机械 | 循环必须**可执行**——每一步都要有明确的物理意义。不能把吸热步骤当放热步骤写 | "循环要可走，物理意义必须清" |
| "晶格能 = 升华热 + 电离能" | 把 Born-Haber 循环中几个大项混淆 | 晶格能是循环中的一个**步骤**（气态离子→晶体），不能和整个循环的 ΔH 混淆 | "晶格能是⼀步，不是全循环" |
| "原子化焓和升华焓是一回事" | 符号相似 + 都是"变成气态原子" | 升华焓：固体→气态原子；原子化焓：任何相态→气态原子（如 ½Cl₂(g)→Cl(g) 是解离不是升华）| "升华是固体变气态，原子化是变原子" |

### 12.2 入门级例题

**题目**：利用以下数据，通过 Born-Haber 循环计算 NaCl 的晶格能：
Na(s) 升华焓：+108 kJ/mol；½Cl₂(g) 解离焓：+121 kJ/mol；
Na(g) 电离能：+496 kJ/mol；Cl(g) 电子亲合能：−349 kJ/mol；
NaCl(s) 标准生成焓：−411 kJ/mol。

**预期解答路径**：
1. Born-Haber 循环：Na(s) → Na(g) → Na⁺(g)+e⁻；½Cl₂(g) → Cl(g) → Cl⁻(g)
2. $\Delta_f H = \Delta_\text{sub}H(\mathrm{Na}) + I(\mathrm{Na}) + \frac{1}{2}D(\mathrm{Cl_2}) + E_\mathrm{ea}(\mathrm{Cl}) + U$
3. −411 = 108 + 496 + 121 + (−349) + $U$
4. $U = -411 - 108 - 496 - 121 + 349 = -787\ \mathrm{kJ\cdot mol^{-1}}$

**教师引导提问**：晶格能 $U$ 为负意味着什么？（气态离子结合成晶体是强烈放热过程——这就是离子化合物稳定的根源。$U$ 越负，离子晶体越稳定）

### 12.3 与现实/直觉的连接

- **能量收支表**：Born-Haber 循环就像财务收支表——你不需要关心钱（能量）中间经过了谁的账户，只要每笔进出记录清楚，最终余额（ΔfH）一定对得上。
- **为什么 NaCl 存在而 NaCl₂ 不存在？** Born-Haber 循环可以定量回答：NaCl₂ 需要 Na 的第二电离能（+4562 kJ/mol！），而多一个 Cl 的晶格能和电子亲合能远不足以补偿。热力学循环优雅地解释了化学式的"为什么"。
- **状态函数思想的应用**：Born-Haber 循环体现了化学中最重要的思维方式之一——**用一个可计算的路径替代不可直接测量的过程**。晶格能不能直接测量，但可以通过循环间接求得。

## 十三、竞赛拓展

1. **Born-Haber 循环的"逆向"应用——从晶格能反推未知热力学量**：竞赛题不只让你算晶格能，更常见的是"已知晶格能和晶体生成焓，反推电子亲合能或质子亲合能"。例如：通过 Born-Haber 循环计算 $\mathrm{O(g) + 2e^- \to O^{2-}(g)}$ 的过程焓变（第二电子亲合能 +780 kJ/mol 是正的——O²⁻(g) 不稳定，仅在晶体中因 Madelung 能而存在）。还可以用类似思路计算质子亲合能（气相碱度）、水合焓等。关键是：热力学循环的每一步都必须是物理上可定义的过程。

2. **制冷循环与热泵的竞赛级计算**：Carnot 制冷机的性能系数（COP）$\varepsilon_C = \dfrac{T_C}{T_H - T_C}$，实际制冷循环（蒸汽压缩式）的 COP 受制冷剂性质限制。竞赛可能给制冷剂的 p-H 图（压焓图）要求计算制冷量、压缩机功和 COP。热泵本质上是反向 Carnot 机——从低温热源吸热向高温热源放热，其制热系数 $\mathrm{COP}_{heating} = \dfrac{|Q_H|}{|W|} = \dfrac{T_H}{T_H - T_C} > 1$。注意：热泵 COP 总是 >1（排出的热量大于消耗的功），这不是违反热力学第一定律——多出来的热量来自低温热源。

3. **熵的统计诠释及其在竞赛中的定性应用**：Boltzmann 熵公式 $S = k_B \ln\Omega$ 是解释熵变方向的微观基础。竞赛中可用于定性判断：气体分子数增加（$\Omega$ 增加 → $\Delta S > 0$）、刚性分子比柔性分子 $\Omega$ 小（如乙烷的分子内旋转使 $S^\ominus$ 大于刚性乙烯）、混合熵 $\Delta S_{mix} = -R\sum x_i\ln x_i > 0$ 总是正的。决赛可能涉及残余熵（如 CO 晶体在 0K 仍有 $S \approx 5$ J/mol·K——因为 CO 分子在晶格中可随机取向，$\Omega = 2^N$）。
