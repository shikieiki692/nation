# 溶液浓度

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[03-分散相与连续相]]

## 一、定义
常用浓度表示法：
- **物质的量浓度** $c = n/V$ (mol/L)
- **质量分数** $w = m_\text{质}/m_\text{液}$
- **质量摩尔浓度** $b = n_\text{质}/m_\text{剂}$ (mol/kg)
- **摩尔分数** $x_i = n_i/\sum n$

## 二、考纲对应
- 对应考纲条目：[[03-分散相与连续相]]
- 所属模块：[[基础要求-化学原理]]
- 本知识点在考纲中的作用：溶液配制与浓度换算是一切化学实验和计算的基础

## 三、核心原理
浓度的表示方法分为两大类：一类用溶质与溶剂（或溶液）的相对量表示（质量或物质的量），另一类用一定体积溶液中所含溶质的量表示。

### 第一类：基于质量的浓度表示（不随温度变化）
- **质量分数** $w$：$w = \frac{m_\text{溶质}}{m_\text{溶液}}$，无量纲，可用分数或百分数表示
  - 例：$10.0\ \mathrm{g\ NaCl}$ 溶于 $100.0\ \mathrm{g}$ 水，$w(\mathrm{NaCl}) = 10.0/(100.0+10.0) \times 100\% = 9.1\%$
- **摩尔分数** $x$：$x_i = n_i / \sum n$，无量纲，所有组分摩尔分数之和恒等于 1
  - 例：$10.0\ \mathrm{g\ NaCl}$（$0.171\ \mathrm{mol}$）溶于 $90.0\ \mathrm{g}$ 水（$5.00\ \mathrm{mol}$），$x(\mathrm{NaCl}) = 0.171/(5.00+0.171) = 0.033$，$x(\mathrm{H_2O}) = 0.967$
  - 优点：可以和化学反应直接联系起来（反应式中物质的量之比简单）
- **质量摩尔浓度** $m$（或 $b$）：$m = n_\text{溶质} / m_\text{溶剂}$，单位 $\mathrm{mol\cdot kg^{-1}}$
  - 例：$58.4\ \mathrm{g\ NaCl}$ 溶于 $1000\ \mathrm{g}$ 水，浓度为 $1.00\ \mathrm{mol\cdot kg^{-1}}$
  - 优点：数值不随温度变化，常用于理论讨论（如依数性）

### 第二类：基于体积的浓度表示（温度敏感）
- **物质的量浓度** $c$：$c = n_\text{溶质} / V_\text{溶液}$，单位 $\mathrm{mol\cdot dm^{-3}}$（或 $\mathrm{mol\cdot L^{-1}}$）
  - 实验室最常用，方便取液计算：如取 $25\ \mathrm{cm^3}\ 18\ \mathrm{mol\cdot dm^{-3}}$ 浓硫酸，所含 H$_2$SO$_4$ 为 $18 \times 25/1000 = 0.45\ \mathrm{mol}$
  - 缺点：密度或体积随温度略有变化（如 $10\%$ NaCl 溶液在 $10^\circ\mathrm{C}$ 时 $\rho = 1.07411\ \mathrm{g\cdot cm^{-3}}$，$20^\circ\mathrm{C}$ 时 $\rho = 1.07074\ \mathrm{g\cdot cm^{-3}}$）
- **体积分数** $\varphi$：$\varphi = V_\text{组分} / \sum V_\text{混合前}$，无量纲
  - 例：$20^\circ\mathrm{C}$ 时将 $70\ \mathrm{cm^3}$ 乙醇与 $30\ \mathrm{cm^3}$ 水混合，$\varphi(\mathrm{C_2H_5OH}) = 70\%$

### 其他粗略表示法
- **比例浓度**：如 $1:1$ 盐酸，指市售浓盐酸（$37\%$, $\sim 12\ \mathrm{mol\cdot dm^{-3}}$）与水按体积比 $1:1$ 配制，简单方便但粗略
- **ppm / ppb**：微量成分，$\mathrm{ppm} = 10^{-6}$，$\mathrm{ppb} = 10^{-9}$（IUPAC 已建议不再使用，因 ppb 中的 billion 在欧洲为 $10^{12}$ 而在美国为 $10^9$，存在歧义）
  - 对气态溶液常指物质的量或体积：如空气中 SO$_2$ 浓度 $0.2\ \mathrm{ppm}$ 指 $10^6\ \mathrm{mol}$ 空气中含 $0.2\ \mathrm{mol\ SO_2}$
  - 对液态溶液常指质量：如污水中含汞 $6\ \mathrm{ppm}$ 指 $10^6\ \mathrm{g}$ 水中含 $6\ \mathrm{g}$ 汞

## 四、关键结论
- **浓度换算通用公式**（以 $10\%$ NaCl 溶液为例，$\rho = 1.07\ \mathrm{g\cdot cm^{-3}}$，$M = 58.4\ \mathrm{g\cdot mol^{-1}}$）：
  - $c = \frac{\text{溶质物质的量}}{\text{溶液体积}} = \frac{10.0/58.4}{100.0/1.07} \times 1000 = 1.83\ \mathrm{mol\cdot dm^{-3}}$
  - 换算需知溶液密度和溶质摩尔质量
- **稀释公式**：稀释前后溶质质量（或物质的量）不变
  - 例：用密度 $1.84\ \mathrm{g\cdot cm^{-3}}$、$98\%$ 的浓硫酸配制 $1.0\ \mathrm{dm^3}\ 2.0\ \mathrm{mol\cdot dm^{-3}}$ 的硫酸，需取浓硫酸 $110\ \mathrm{cm^3}$
  - $1.0 \times 2.0 \times 98 = x \times 1.84 \times 0.98 \Rightarrow x = 1.1 \times 10^2\ \mathrm{cm^3}$
- **配制精确溶液**需用分析天平称量和容量瓶定容：如 $0.7212\ \mathrm{g\ NaCl}$ 用容量瓶配成 $100.0\ \mathrm{cm^3}$，$c = 0.1234\ \mathrm{mol\cdot dm^{-3}}$（4 位有效数字）
- 所有浓度表示方法之间可以相互换算，根据实际需要选用

## 五、常见分类或情形
| 表示法 | 符号 | 单位 | 温度依赖性 | 典型用途 |
|--------|------|------|-----------|---------|
| 质量分数 | $w$ | 无量纲 | 无 | 商品试剂标签 |
| 摩尔分数 | $x$ | 无量纲 | 无 | 理论推导（Raoult 定律）、化学反应计量 |
| 质量摩尔浓度 | $m$ 或 $b$ | $\mathrm{mol\cdot kg^{-1}}$ | 无 | 依数性讨论 |
| 物质的量浓度 | $c$ | $\mathrm{mol\cdot dm^{-3}}$ | 有 | 实验室常规操作 |
| 体积分数 | $\varphi$ | 无量纲 | 有 | 混合溶剂配制 |
| 比例浓度 | — | — | — | 粗略的实验室配制 |
| ppm / ppb | — | $10^{-6}$ / $10^{-9}$ | — | 微量/痕量分析 |

## 六、适用条件与限制
- 质量分数、摩尔分数、质量摩尔浓度的数值不随温度变化，适合精确理论工作
- 物质的量浓度（$c$）随温度略有变化（因溶液密度/体积随温度变化），在 3 位有效数字计算中影响不大，4 位以上需注意
- 比例浓度和 ppm/ppb 是粗略表示法，不适用于精确的化学计量计算
- IUPAC 已不推荐使用 ppm/ppb，建议用 $\mathrm{mg\cdot m^{-3}}$、$\mu\mathrm{g\cdot m^{-3}}$ 等单位体积质量代替
- 配制物质的量浓度溶液时需使用容量瓶，不能简单地将溶质加入一定体积溶剂（因为溶解过程有体积变化）

## 七、常见比较与易混点
- **质量分数 vs 质量摩尔浓度**：$w$ 的分母是溶液质量，$m$ 的分母是溶剂质量。稀水溶液中两者数值相近但不相等（$m$ 略大于 $w$ 对应的值）
- **物质的量浓度 vs 质量摩尔浓度**：$c$ 是溶质物质的量除以溶液体积（$\mathrm{mol\cdot dm^{-3}}$），$m$ 是溶质物质的量除以溶剂质量（$\mathrm{mol\cdot kg^{-1}}$）。稀水溶液中 $1\ \mathrm{kg}$ 水 $\approx 1\ \mathrm{dm^3}$，两者近似相等
- **$100\ \mathrm{cm^3}$ 水中溶质克数 $\neq$ 质量分数**：溶剂质量 $\neq$ 溶液质量，需加溶质质量（但很稀的溶液中可近似）
- **"配制 $1.00\ \mathrm{mol\cdot dm^{-3}}$ NaCl 溶液"的正确做法**：将 $58.4\ \mathrm{g\ NaCl}$ 先溶于水，再在容量瓶中冲稀到 $1.00\ \mathrm{dm^3}$，而非加入 $1.00\ \mathrm{dm^3}$ 水（因为溶解有体积变化）
- **% 的三种含义**：$w/w\%$（质量分数）、$v/v\%$（体积分数）、$w/v\%$（质量体积百分数）——需根据上下文判断

## 十二、🎯 教学视角

### 12.1 学生典型认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 口诀 |
|:---|:---|:---|:---|
| ""加入 1 dm³ 水" = "定容至 1 dm³"" | 日常语言中的"加水"对应"加至" | 溶解过程有体积变化（如乙醇+水体积收缩）→ 配制物质的量浓度溶液必须用**容量瓶定容**，不能说"加入 X 升水" | "先溶后定容，不直接加水" |
| "质量分数 = 质量摩尔浓度 × 摩尔质量" | 把两个"质量"开头的概念混淆 | $w = \frac{m_\text{质}}{m_\text{液}}$（分母含溶质）；$b = \frac{n_\text{质}}{m_\text{剂}}$（分母只含溶剂）。稀溶液近似相等，浓溶液差异显著 | "分数的分母是全家，摩尔浓度只算水" |
| "浓度换算公式 $c = 1000\rho w/M$ 只能死记" | 被 1000 这个因子迷惑 | 公式推导：$c = \frac{n}{V} = \frac{w \cdot m_\text{液}/M}{m_\text{液}/\rho} \times 1000$——1000 只是 dm³→cm³ 的单位换算 | "推一遍，不用背" |
| "ppm 就是 mg/L" | 把近似当定义 | ppm = $10^{-6}$（无量纲），稀水溶液中 $1\ \mathrm{ppm} \approx 1\ \mathrm{mg/L}$，但严格来说 ppm 是质量比或物质的量比 | "ppm 是比例，不是单位" |

### 12.2 入门级例题

**题目**：市售浓硫酸标签：$w(\mathrm{H_2SO_4}) = 98\%$，$\rho = 1.84\ \mathrm{g\cdot cm^{-3}}$，$M(\mathrm{H_2SO_4}) = 98.0\ \mathrm{g\cdot mol^{-1}}$。
(1) 计算浓硫酸的物质的量浓度 $c$。
(2) 要配制 $500\ \mathrm{cm^3}\ 2.00\ \mathrm{mol\cdot dm^{-3}}$ 的稀硫酸，需取浓硫酸多少 $\mathrm{cm^3}$？

**预期解答路径**：
1. 取 $1\ \mathrm{dm^3} = 1000\ \mathrm{cm^3}$ 浓硫酸：
   $m_\text{液} = 1000 \times 1.84 = 1840\ \mathrm{g}$
   $m(\mathrm{H_2SO_4}) = 1840 \times 0.98 = 1803\ \mathrm{g}$
   $n(\mathrm{H_2SO_4}) = 1803 / 98.0 = 18.4\ \mathrm{mol}$
   $c = 18.4 / 1 = 18.4\ \mathrm{mol\cdot dm^{-3}}$（或直接用 $c = 1000\rho w/M$ 验证）
2. 稀释前后溶质物质的量不变：$c_1 V_1 = c_2 V_2$
   $18.4 \times V_1 = 2.00 \times 0.500$
   $V_1 = 0.0543\ \mathrm{dm^3} = 54.3\ \mathrm{cm^3}$

**教师引导提问**：如果学生把"配制 $500\ \mathrm{cm^3}$"理解成"加入 $500\ \mathrm{cm^3}$ 水"——结果差多少？（浓硫酸 $54.3\ \mathrm{cm^3}$ + 水 $500\ \mathrm{cm^3}$ → 总体积 ≈ $554\ \mathrm{cm^3}$，浓度 ≈ $18.4 \times 0.0543 / 0.554 = 1.80\ \mathrm{mol\cdot dm^{-3}}$，比目标 $2.00$ 低了 10%！）

### 12.3 与现实/直觉的连接

- **生理盐水为什么是 0.9%？** $0.9\%\ \mathrm{NaCl}$（质量分数）→ $c \approx 0.154\ \mathrm{mol\cdot dm^{-3}}$，与人血浆渗透压相等。如果浓度不对，红细胞会胀破（低渗）或皱缩（高渗）——浓度换算在医学上关乎人命。
- **输液瓶上的"葡萄糖 5%"**：指的是 $5\ \mathrm{g}$ 葡萄糖 / $100\ \mathrm{mL}$ 溶液（$w/v\%$），和化学中的 $w/w\%$ 不同！这就是为什么竞赛题中的"%"需要根据上下文判断含义。
- **糖水的甜度**：$10\%$ 蔗糖溶液 vs $10\%$ 葡萄糖溶液——甜度不同为什么？因为 $M(\text{蔗糖})=342 > M(\text{葡萄糖})=180$ → 相同质量分数下葡萄糖的物质的量浓度更高 → 甜味受体感受到更多分子。
