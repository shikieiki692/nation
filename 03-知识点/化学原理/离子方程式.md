# 离子方程式

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 关联笔记：[[方程式书写]], [[电荷守恒]], [[物料守恒]], [[电解质溶液]]

## 一、定义

离子方程式是用实际参加反应的离子符号表示离子反应的式子。它比化学方程式更能反映反应的本质，因为它省略了未参与反应的 spectator ions（旁观离子），直接展示了反应的核心过程。

## 二、考纲对应

- 对应考纲条目：[[08-化学反应与方程式]]、[[09-水溶液中的离子平衡]]、[[10-电解质溶液]]
- 所属模块：[[基础要求-化学原理]]
- 本知识点在考纲中的作用：离子方程式的书写与配平是竞赛化学的基本功，要求能正确判断 spectator ions、掌握缺项配平（酸性/碱性介质中 H⁺/OH⁻/H₂O 的补充），是元素推断题和氧化还原滴定题中方程式书写的标准格式。

## 三、核心原理

### 3.1 书写步骤

1. **写出完整的化学方程式**
2. **将可溶性强电解质拆成离子形式**
   - 强酸（HCl、H₂SO₄、HNO₃、HClO₄ 等）
   - 强碱（NaOH、KOH、Ba(OH)₂、Ca(OH)₂ 等）
   - 可溶性盐（查溶解度表）
3. **删去方程式两边相同的 spectator ions**
4. **检查守恒**：原子守恒、电荷守恒
5. **化简系数为最简整数比**

### 3.2 可拆 vs 不可拆

| 类别 | 处理方式 | 例子 |
|------|---------|------|
| 强酸 | 拆 | $H^+ + Cl^-$ |
| 强碱 | 拆 | $Na^+ + OH^-$ |
| 可溶性盐 | 拆 | $Na^+ + SO_4^{2-}$ |
| 弱酸/弱碱 | 不拆 | $CH_3COOH$、$NH_3·H_2O$ |
| 难溶物 | 不拆 | $BaSO_4$、$AgCl$ |
| 氧化物 | 不拆 | $CO_2$、$CaO$ |
| 单质 | 不拆 | $Cl_2$、Fe |
| 气体 | 不拆 | $H_2$、$CO_2$ |

### 3.3 常见错误类型

- **拆错**：把弱电解质或难溶物拆了
- **漏写**：微溶物（如 $Ca(OH)_2$）在稀溶液中拆、浓溶液中不拆的处理不当
- **不守恒**：原子数或电荷数不相等
- **不符合事实**：如把 $Cu + H^+$ 写成 $Cu^{2+} + H_2$（稀硫酸不反应）

## 四、关键结论

1. **离子方程式反映反应的本质**：只写实际参与反应的离子或分子，省略不参与反应的旁观离子（spectator ions）。所有酸碱中和的实质都是 $H^+ + OH^- \rightarrow H_2O$，无论用什么酸和碱。

2. **书写三步骤——"写→拆→删"**：
   - **写**：先写出完整的化学方程式；
   - **拆**：将可溶性强电解质（强酸、强碱、可溶性盐）拆成离子；
   - **删**：删除方程式两边相同的离子（即未参与反应的旁观离子）；
   - 最后检查化简为最简整数比。

3. **"拆"的精确判断规则**：
   - **可拆**：强酸（$HCl$、$HNO_3$、$H_2SO_4$、$HBr$、$HI$、$HClO_4$）、强碱（$NaOH$、$KOH$、$Ba(OH)_2$）、可溶盐；
   - **不可拆**：弱酸、弱碱、水、气体、沉淀、难溶物、氧化物、单质；
   - 口诀："强溶才拆，弱难气氧单不拆"。

4. **三守恒原则**：离子方程式必须同时满足——
   - **原子守恒**：方程式两边各元素原子总数相等；
   - **电荷守恒**：方程式两边总电荷数相等；
   - **电子守恒**（氧化还原中）：还原剂失去的电子总数 = 氧化剂得到的电子总数。

5. **氧化还原离子方程式的配平顺序**：先配电子守恒（得失电子相等）→ 再配电荷守恒（用 $H^+$/$OH^-$ 调平）→ 最后配原子守恒（用 $H_2O$ 补齐 H、O），与传统化学方程式配平顺序不同。

6. **浓溶液中的特殊处理**：浓 $H_2SO_4$ 主要以分子形式存在，在离子方程式中**不拆**；浓 $HNO_3$ 虽可拆但还原产物与稀 $HNO_3$ 不同，书写时需注意产物判断。

## 五、常见分类或情形

### 5.1 酸碱中和反应

酸碱中和的实质离子方程式为 $H^+ + OH^- \rightarrow H_2O$（强酸强碱中和的最简形式）。

**示例**：
- $HCl + NaOH \rightarrow NaCl + H_2O$ → 离子：$H^+ + OH^- \rightarrow H_2O$
- $H_2SO_4 + 2KOH \rightarrow K_2SO_4 + 2H_2O$ → 离子：$H^+ + OH^- \rightarrow H_2O$

若有弱酸/弱碱参与，弱电解质不拆：
- $CH_3COOH + NaOH$ → 离子：$CH_3COOH + OH^- \rightarrow CH_3COO^- + H_2O$

### 5.2 沉淀反应

生成难溶物的离子反应，↓不可省略。

**示例**：
- $Ag^+ + Cl^- \rightarrow AgCl\downarrow$
- $Ba^{2+} + SO_4^{2-} \rightarrow BaSO_4\downarrow$
- $Ca^{2+} + CO_3^{2-} \rightarrow CaCO_3\downarrow$

### 5.3 氧化还原离子方程式

有化合价变化的离子反应，配平需兼顾电子守恒。

**示例**：
- $MnO_4^- + 5Fe^{2+} + 8H^+ \rightarrow Mn^{2+} + 5Fe^{3+} + 4H_2O$
- $2Fe^{3+} + Cu \rightarrow 2Fe^{2+} + Cu^{2+}$
- $Cl_2 + 2I^- \rightarrow 2Cl^- + I_2$

配平顺序：**电子守恒 → 电荷守恒 → 原子守恒**。

### 5.4 配位反应

生成配合物的离子反应，配离子不可拆。

**示例**：
- $Cu^{2+} + 4NH_3 \rightarrow [Cu(NH_3)_4]^{2+}$
- $Ag^+ + 2NH_3 \rightarrow [Ag(NH_3)_2]^+$
- $Fe^{3+} + 6CN^- \rightarrow [Fe(CN)_6]^{3-}$

### 5.5 水解反应

盐类水解的离子方程式，使用可逆号 $\rightleftharpoons$（不写↓和↑）。

**示例**：
- $CO_3^{2-} + H_2O \rightleftharpoons HCO_3^- + OH^-$
- $Fe^{3+} + 3H_2O \rightleftharpoons Fe(OH)_3 + 3H^+$
- $NH_4^+ + H_2O \rightleftharpoons NH_3·H_2O + H^+$

### 5.6 少量/过量问题（"不足量定系数法"）

当一种反应物少量、另一种过量时，产物的离子方程式不同。**核心方法**：设不足量反应物的系数为 1，按需配平过量一方。

**示例**——$NaOH$ 与 $AlCl_3$ 反应：
- $NaOH$ **少量**（不足）：$Al^{3+} + 3OH^- \rightarrow Al(OH)_3\downarrow$
- $NaOH$ **过量**（足量）：$Al^{3+} + 4OH^- \rightarrow AlO_2^- + 2H_2O$

**示例**——$NaHCO_3$ 与 $Ca(OH)_2$ 反应（见 §十 例题2）。

## 六、适用条件与限制

1. **主要适用于溶液中的离子反应**：离子方程式的"拆"规则基于水溶液中电解质的电离行为，非溶液中的反应（如气体反应、固相反应）不适用离子方程式写法。

2. **浓溶液环境下规则需调整**：浓 $H_2SO_4$（约 98%）主要以 $H_2SO_4$ 分子形式存在，在离子方程式中**不拆**。例如浓 $H_2SO_4$ 与 $NaCl$ 固体共热制 $HCl$：$H_2SO_4(浓) + NaCl \xrightarrow{\Delta} NaHSO_4 + HCl\uparrow$，该反应不以离子方程式表示（固-液非均相反应）。

3. **熔融态反应不能照搬溶液中的拆法**：如熔融 $NaCl$ 电解为 $2NaCl(l) \xrightarrow{电解} 2Na + Cl_2\uparrow$，此处 $NaCl$ 虽在熔融态电离为 $Na^+$ 和 $Cl^-$，但书写方式与溶液中不同——不存在水合离子，不能用 $(aq)$ 思维套用。

4. **非水溶剂中"拆不拆"的判断标准不同**：在水溶液中判断"可溶/难溶"的标准（溶解度表）仅适用于水为溶剂的情况。非水溶剂（如液氨、$SO_2$ 液态、有机溶剂）中电解质的电离行为完全不同，不可直接套用。

5. **微溶物的双向处理规则**：微溶物（如 $Ca(OH)_2$、$CaSO_4$、$Ag_2SO_4$）作**反应物**时拆（因以溶解部分参与反应），作**生成物**时不拆（因超过溶解度会沉淀出来）。这是最常见的判断失误点之一。

## 七、常见比较与易混点

### 7.1 化学方程式 vs 离子方程式

| 比较维度 | 化学方程式 | 离子方程式 |
|:---------|:----------|:----------|
| 表达内容 | 所有反应物和产物，含旁观离子 | 仅实际参与反应的离子/分子 |
| 适用范围 | 所有化学反应 | 主要适用于溶液中的离子反应 |
| 信息量 | 完整但不简洁 | 精炼反映反应本质 |
| $HCl + NaOH$ | $HCl + NaOH \rightarrow NaCl + H_2O$ | $H^+ + OH^- \rightarrow H_2O$ |
| 普适性 | 一种酸+一种碱对应一个方程式 | 所有强酸强碱中和对应同一离子方程式 |

### 7.2 "拆" vs "不拆"的判断——关键区分

这是学生最困惑的点。判断的核心依据是：**该物质在溶液中主要以什么形式存在**。

| 物质 | 拆否 | 原因 |
|:-----|:----|:-----|
| $BaSO_4$ | **不拆** | 难溶盐，$K_{sp}=1.1\times10^{-10}$，溶液中离子浓度极低，主要以固体形式存在 |
| $Ba(OH)_2$ | **拆** | 可溶性强碱，溶液中完全电离为 $Ba^{2+}$ 和 $OH^-$ |
| $CaCO_3$ | **不拆** | 难溶盐，大理石/石灰石的主成分 |
| $Na_2CO_3$ | **拆** | 可溶性碳酸盐 |
| $Mg(OH)_2$ | **不拆** | 难溶碱 |
| $NaOH$ | **拆** | 可溶性强碱 |

**为什么 $BaSO_4$ 不拆但 $Ba(OH)_2$ 拆？** 前者是难溶盐（溶解度约 0.00024 g/100g 水），后者是可溶碱（溶解度约 3.9 g/100g 水）——"钡"不是判断依据，"溶解性"才是。

### 7.3 浓 $H_2SO_4$ vs 稀 $H_2SO_4$

| 项目 | 稀 $H_2SO_4$ | 浓 $H_2SO_4$ |
|:-----|:------------|:------------|
| 存在形式 | $H^+$ + $SO_4^{2-}$（完全电离） | 主要以 $H_2SO_4$ 分子存在 |
| 离子方程式写法 | **拆**：$2H^+ + SO_4^{2-}$ | **不拆**：写 $H_2SO_4(浓)$ |
| 与 $Cu$ 反应 | 不反应 | $Cu + 2H_2SO_4(浓) \rightarrow CuSO_4 + SO_2\uparrow + 2H_2O$ |
| 还原产物 | — | $SO_2$（而不是 $H_2$） |

### 7.4 $Ca(OH)_2$——石灰水 vs 石灰乳

| 形态 | 处理方式 | 原因 |
|:-----|:--------|:-----|
| **澄清石灰水** | **拆** | 稀溶液，$Ca(OH)_2$ 溶解部分完全电离 |
| **石灰乳（悬浊液）** | **不拆** | 浓悬浊液，大部分 $Ca(OH)_2$ 以固体形式存在 |

> 考试中最常见的是"澄清石灰水"——默认**拆**。

### 7.5 弱酸——$H_3PO_4$ 在离子方程式中的处理

$H_3PO_4$ 是**中强酸**（$K_{a1}=7.5\times10^{-3}$），但**在离子方程式中不拆**——处理规则同弱酸。原因：$H_3PO_4$ 在水中不完全电离，主要以 $H_3PO_4$ 分子存在。

类似的还有 $HF$（弱酸，不拆）、$CH_3COOH$（弱酸，不拆）、$H_2CO_3$（弱酸且不稳定，写为 $H_2O + CO_2$）。

### 7.6 少量/过量——产物不同

同一组反应物，因相对用量不同产物不同（详见 §五.5.6）。核心原则：**不足量反应物的系数定为 1**，由此确定过量一方消耗的离子数。

| 反应体系 | 少量方不足 | 离子方程式 |
|:---------|:----------|:----------|
| $NaOH + AlCl_3$ | $NaOH$ 少量 | $Al^{3+} + 3OH^- \rightarrow Al(OH)_3\downarrow$ |
| $NaOH + AlCl_3$ | $NaOH$ 过量 | $Al^{3+} + 4OH^- \rightarrow AlO_2^- + 2H_2O$ |
| $Na_2CO_3 + HCl$ | $Na_2CO_3$ 少量 | $CO_3^{2-} + 2H^+ \rightarrow H_2O + CO_2\uparrow$ |
| $Na_2CO_3 + HCl$ | $HCl$ 少量 | $CO_3^{2-} + H^+ \rightarrow HCO_3^-$ |

### 7.7 氧化还原离子方程式中 $H^+$ / $OH^-$ / $H_2O$ 的补充规则

配平氧化还原离子方程式时，当原子种类（H、O）不守恒时：
- **酸性介质**：用 $H^+$ 调电荷 → 用 $H_2O$ 调 H、O
- **碱性介质**：用 $OH^-$ 调电荷 → 用 $H_2O$ 调 H、O
- **中性介质**：反应物一侧只能加水，产物一侧可出现 $H^+$ 或 $OH^-$

**注意**：不可在反应物一侧同时出现 $H^+$ 和 $OH^-$（酸碱不可共存于反应体系中）。

## 九、典型题型

1. **书写离子方程式**：给定反应物，写出正确的离子方程式。考查"拆不拆"的判断和守恒配平。是基础题型。

2. **离子方程式正误判断**（竞赛高频）：给四个离子方程式选项，判断哪一个正确。典型错误类型包括：拆错、电荷不守恒、产物不符合事实、缺少↓↑。常在选择题中出现。

3. **少量/过量问题**：同一组反应物在"A少量+B过量"和"A过量+B少量"两种情况下写出不同的离子方程式。竞赛中的经典区分题。

4. **氧化还原离子方程式配平**：给定氧化剂和还原剂，在指定介质（酸性/碱性/中性）中写出并配平离子方程式。需要掌握 $H^+$/$OH^-$/$H_2O$ 的正确补充方法。

5. **由离子方程式反推化学方程式**：已知离子方程式 $A^{2+} + B^- \rightarrow AB\downarrow$，写出两组符合条件的化学方程式。考查对"离子方程式代表一类反应"的理解。

6. **离子共存判断**：判断给定离子组在特定条件下（酸性/碱性/有沉淀剂等）能否大量共存——本质是离子方程式的逆向应用。常结合溶液颜色（$Cu^{2+}$蓝、$Fe^{3+}$黄、$MnO_4^-$紫）、pH 条件进行考查。

## 十、例题

### 例题 1 ★★ | 酸碱中和 + 沉淀的复合反应

**题目**：写出 $Ba(OH)_2$ 溶液与 $H_2SO_4$ 溶液反应的离子方程式。

**分析**：
- 反应物：$Ba(OH)_2$ 是可溶性强碱 → **拆**：$Ba^{2+} + 2OH^-$
- $H_2SO_4$ 是强酸 → **拆**：$2H^+ + SO_4^{2-}$
- 产物：$BaSO_4$ 是难溶盐 → **不拆**；$H_2O$ 是弱电解质 → **不拆**
- 检查旁观离子：$Ba^{2+}$ 与 $SO_4^{2-}$ 结合沉淀，$H^+$ 与 $OH^-$ 结合成水——四种离子**全部参与反应**，无旁观离子

**解答**：
$$Ba^{2+} + 2OH^- + 2H^+ + SO_4^{2-} \rightarrow BaSO_4\downarrow + 2H_2O$$

**反思**：
- 这是极少见的"零旁观离子"离子反应。大多数离子反应中，总有至少一种离子（如 $Na^+$、$NO_3^-$）不参与反应。
- 与 $H_2SO_4 + 2NaOH \rightarrow Na_2SO_4 + 2H_2O$ 对比：后者的离子方程式仅为 $H^+ + OH^- \rightarrow H_2O$，$Na^+$ 和 $SO_4^{2-}$ 都是旁观离子。
- **易错**：忘记 $BaSO_4$ 后标注 $\downarrow$。

---

### 例题 2 ★★★ | 少量/过量经典题

**题目**：写出下列两组反应的离子方程式：
- (1) $NaHCO_3$ 溶液与**少量** $Ca(OH)_2$ 溶液反应
- (2) $NaHCO_3$ 溶液与**过量** $Ca(OH)_2$ 溶液反应

**分析**——使用"不足量定系数法"：

**(1) $Ca(OH)_2$ 少量（不足量）**：
- 设 $Ca(OH)_2$ 系数为 1：即 1 个 $Ca^{2+}$ + 2 个 $OH^-$ 参与反应
- $OH^-$ 与 $HCO_3^-$ 反应：$OH^- + HCO_3^- \rightarrow CO_3^{2-} + H_2O$
- 但 $Ca^{2+}$ 会与生成的 $CO_3^{2-}$ 结合沉淀：$Ca^{2+} + CO_3^{2-} \rightarrow CaCO_3\downarrow$
- 2 个 $OH^-$ 消耗 2 个 $HCO_3^-$，生成 2 个 $CO_3^{2-}$
- 但只有 1 个 $Ca^{2+}$，所以只有 1 个 $CO_3^{2-}$ 沉淀，另 1 个 $CO_3^{2-}$ 游离
- 写在一起：$Ca^{2+} + 2OH^- + 2HCO_3^- \rightarrow CaCO_3\downarrow + CO_3^{2-} + 2H_2O$

**(2) $NaHCO_3$ 少量（不足量）** → 即 $Ca(OH)_2$ 过量：
- 设 $NaHCO_3$ 系数为 1：即 1 个 $HCO_3^-$ 参与反应
- 需要 1 个 $OH^-$：$HCO_3^- + OH^- \rightarrow CO_3^{2-} + H_2O$
- 生成的 $CO_3^{2-}$ 与 $Ca^{2+}$（过量中存在）沉淀：$Ca^{2+} + CO_3^{2-} \rightarrow CaCO_3\downarrow$
- 合并：$Ca^{2+} + OH^- + HCO_3^- \rightarrow CaCO_3\downarrow + H_2O$
- 注意：$Ca(OH)_2$ 过量，溶液中有多余的 $Ca^{2+}$ 和 $OH^-$，但不影响离子方程式（多余的旁观）

**解答**：
- (1) $Ca(OH)_2$ 少量时：$$Ca^{2+} + 2OH^- + 2HCO_3^- \rightarrow CaCO_3\downarrow + CO_3^{2-} + 2H_2O$$
- (2) $NaHCO_3$ 少量时（即 $Ca(OH)_2$ 过量）：$$Ca^{2+} + OH^- + HCO_3^- \rightarrow CaCO_3\downarrow + H_2O$$

**反思**：
- 两个方程式的直观区别：少量一方决定了对方离子的消耗量。
- **不足量定系数法**是解决少量/过量问题的通用方法：设不足量一方的系数为 1，按需配平过量方。
- 验算技巧：检查两边的电荷是否相等。如 (2) 式左边 $+2-1-1=0$，右边 $0$——电荷守恒成立。
- 此为竞赛高频考点，需要熟练掌握。

### 例题 3 ★★★★ | 氧化还原离子方程式配平

**题目**：在酸性介质中，$KMnO_4$ 与 $H_2O_2$ 反应生成 $Mn^{2+}$ 和 $O_2$，写出并配平离子方程式。

**分析**：
- **确定变价元素**：$Mn$：$+7$（$MnO_4^-$）→ $+2$（$Mn^{2+}$），得 $5e^-$；$O$：$-1$（$H_2O_2$）→ $0$（$O_2$），每分子 $H_2O_2$ 失 $2e^-$
- **电子守恒**：5 和 2 的最小公倍数为 10 → $2MnO_4^-$（得 $10e^-$）+ $5H_2O_2$（失 $10e^-$）
- **写出框架**：$2MnO_4^- + 5H_2O_2 + \text{?}H^+ \rightarrow 2Mn^{2+} + 5O_2\uparrow + \text{?}H_2O$
- **电荷守恒**：左边 $2\times(-1) + 0 + x\times(+1) = x-2$；右边 $2\times(+2) = +4$ → $x-2=+4$，$x=6$ → 需要 $6H^+$
- **原子守恒**（H、O）：左边 H 总数 = $5\times2 + 6 = 16$ → 右边需要 $8H_2O$；检查 O：左边 $2\times4 + 5\times2 = 18$，右边 $5\times2 + 8 = 18$ → 守恒

**解答**：
$$2MnO_4^- + 5H_2O_2 + 6H^+ \rightarrow 2Mn^{2+} + 5O_2\uparrow + 8H_2O$$

**反思**：
- 氧化还原离子方程式配平三步走：**电子守恒 → 电荷守恒 → 原子守恒**，顺序不可颠倒。
- 若在**碱性介质**中，用 $OH^-$ 代替 $H^+$ 调电荷：左边不足的正电荷由 $OH^-$ 补？不——碱性介质中右边加水、左边加 $OH^-$（$2MnO_4^- + 5H_2O_2 \rightarrow 2Mn^{2+} + 5O_2\uparrow + 6OH^- + 2H_2O$？需要重新算）。
- 验证方法：最后的原子守恒检查是兜底的——如果配平正确，H 和 O 应该自动对上。

## 十一、易错点

### 易错 1：浓 $H_2SO_4$ 被拆成离子

| 项目 | 内容 |
|:-----|:-----|
| **错误表现** | 将浓 $H_2SO_4$ 写为 $2H^+ + SO_4^{2-}$ |
| **学生为什么会这么想** | "硫酸是强酸，强酸就要拆"——将强酸=必拆的规则绝对化，忽略了浓度对电离度的影响 |
| **正确做法** | 浓 $H_2SO_4$（约 98%）主要以 $H_2SO_4$ 分子存在，离子方程式中**不拆**，写为 $H_2SO_4(浓)$ |
| **记忆口诀** | "浓硫不拆，稀硫才拆" |

### 易错 2：难溶盐被拆开

| 项目 | 内容 |
|:-----|:-----|
| **错误表现** | 将 $CaCO_3$ 写为 $Ca^{2+} + CO_3^{2-}$、将 $BaSO_4$ 写为 $Ba^{2+} + SO_4^{2-}$ |
| **学生为什么会这么想** | "碳酸钙是盐，盐就要拆"——混淆了"盐=离子化合物"和"可溶=拆"两个概念。盐在水溶液中能拆，前提是**可溶** |
| **正确做法** | $CaCO_3$、$BaSO_4$、$AgCl$ 等难溶盐**一律不拆**，保持化学式 |
| **记忆口诀** | "难溶不拆，管它是不是盐" |

### 易错 3：电荷不守恒

| 项目 | 内容 |
|:-----|:-----|
| **错误表现** | 如 $Fe^{3+} + Cu \rightarrow Fe^{2+} + Cu^{2+}$（左 +3，右 +4，电荷不平） |
| **学生为什么会这么想** | 配平时只数原子个数，忘记检查电荷。化学方程式配平只看原子，离子方程式多了一个维度 |
| **正确做法** | 正确写法：$2Fe^{3+} + Cu \rightarrow 2Fe^{2+} + Cu^{2+}$（左 +6，右 +6） |
| **检查方法** | "写完必查电荷"——形成肌肉记忆 |

### 易错 4：弱酸被拆成离子

| 项目 | 内容 |
|:-----|:-----|
| **错误表现** | 将 $CH_3COOH$ 写为 $H^+ + CH_3COO^-$、将 $H_3PO_4$ 写为 $3H^+ + PO_4^{3-}$ |
| **学生为什么会这么想** | "酸就是 $H^+$ + 酸根"——将所有酸等同于强酸处理，不理解弱酸的电离平衡 |
| **正确做法** | 弱酸在溶液中主要以分子形式存在，**不拆**。$CH_3COOH$、$H_3PO_4$、$HF$、$H_2CO_3$ 均不拆 |
| **记忆口诀** | "弱不拆——弱酸弱碱弱电解" |

### 易错 5：少量/过量题中"不足量定系数 1"后忘记化简

| 项目 | 内容 |
|:-----|:-----|
| **错误表现** | 设不足量为 1 后配出的方程式有公约数但未化简，如 $2Ca^{2+} + 4OH^- + 4HCO_3^- \rightarrow 2CaCO_3\downarrow + 2CO_3^{2-} + 4H_2O$ |
| **学生为什么会这么想** | "我系数都配对了为什么要化简"——不理解离子方程式系数必须是**最简整数比** |
| **正确做法** | 所有系数除以最大公约数，化简为最简整数比：$Ca^{2+} + 2OH^- + 2HCO_3^- \rightarrow CaCO_3\downarrow + CO_3^{2-} + 2H_2O$ |

### 易错 6：微溶物区分不清

| 项目 | 内容 |
|:-----|:-----|
| **错误表现** | 一律拆或一律不拆 |
| **学生为什么会这么想** | "微溶是什么？可溶和难溶之间？那到底拆不拆？"——对微溶物的模糊地带缺乏清晰判断标准 |
| **正确做法** | 微溶物作**反应物**时拆（以溶解部分参与反应），作**生成物**时不拆（会沉淀）。需特别记忆的有： |
| | $Ca(OH)_2$——澄清石灰水拆，石灰乳不拆 |
| | $CaSO_4$——微溶（溶解度约 0.2 g/100g），产物不拆，反应物视浓度 |
| | $Ag_2SO_4$——微溶，产物不拆 |
| **记忆口诀** | "微溶反应物拆，微溶产物不拆" |

### 易错 7：氧化还原中 $H^+$/$OH^-$/$H_2O$ 补充错误

| 项目 | 内容 |
|:-----|:-----|
| **错误表现** | 在酸性介质中加 $OH^-$，或在反应物一侧同时写 $H^+$ 和 $OH^-$ |
| **学生为什么会这么想** | "什么地方缺 H、缺 O 就加什么"——没有理解介质条件的约束 |
| **正确做法** | 酸性介质 → 用 $H^+$ 调电荷、$H_2O$ 调原子；碱性介质 → 用 $OH^-$ 调电荷、$H_2O$ 调原子。$H^+$ 和 $OH^-$ **不可同时出现在反应物一侧** |
| **记忆口诀** | "酸加氢，碱加氢氧，水来补"

## 十二、🎯 教学视角

### 12.1 学生典型认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 口诀 |
|:---|:---|:---|:---|
| "离子方程式就是把所有物质拆成离子" | 把"离子方程式"理解为"全拆式" | 只有**可溶性强电解质**才拆（强酸、强碱、可溶性盐）；弱酸、弱碱、难溶物、气体、氧化物、单质——统统不拆 | "强溶才拆，弱难气氧单不拆" |
| "微溶物一律拆开写" | 看到溶解度表上写"微溶"就当可溶处理 | 微溶物（如 Ca(OH)₂）的处理要分情况：在**稀溶液**中按可溶拆开；在**浓溶液或浑浊液**中按难溶不拆。Ca(OH)₂ 在澄清石灰水中拆，在石灰乳中不拆 | "微溶看浓度，稀拆浓不拆" |
| "配平离子方程式不用管电荷" | 只检查原子守恒，忘记电荷守恒 | 离子方程式必须同时满足**原子守恒**和**电荷守恒**。如 Fe³⁺ + Cu → Fe²⁺ + Cu²⁺ 是错的（左边 +3，右边 +4）→ 正确：2Fe³⁺ + Cu → 2Fe²⁺ + Cu²⁺ | "原子要平电荷也要平，两边总数相等" |
| "离子方程式不用写沉淀和气体符号" | 觉得↓和↑是化学方程式的"装饰" | 离子方程式中↓和↑**同样需要标注**——它们是实验现象的重要组成部分，也是检验产物判断是否正确的重要线索 | "沉淀气体不能省，箭头一样要写清" |

### 12.2 入门级例题

**题目**：写出下列反应的离子方程式：(1) Ba(OH)₂ 溶液与 H₂SO₄ 溶液反应；(2) 碳酸钙与盐酸反应。

**预期解答路径**：
1. Ba(OH)₂ + H₂SO₄：
   - 化学方程式：Ba(OH)₂ + H₂SO₄ → BaSO₄↓ + 2H₂O
   - 拆：Ba²⁺ + 2OH⁻ + 2H⁺ + SO₄²⁻ → BaSO₄↓ + 2H₂O
   - 删 spectator ions：无（四种离子全部参与了反应）
   - 离子方程式：Ba²⁺ + 2OH⁻ + 2H⁺ + SO₄²⁻ → BaSO₄↓ + 2H₂O

2. CaCO₃ + HCl：
   - 化学方程式：CaCO₃ + 2HCl → CaCl₂ + CO₂↑ + H₂O
   - 拆：CaCO₃ + 2H⁺ + 2Cl⁻ → Ca²⁺ + 2Cl⁻ + CO₂↑ + H₂O
   - 删 Cl⁻（spectator ion）
   - 离子方程式：CaCO₃ + 2H⁺ → Ca²⁺ + CO₂↑ + H₂O

**教师引导提问**：第一个反应中为什么没有 spectator ions？（因为 Ba²⁺ 和 SO₄²⁻ 生成沉淀，H⁺ 和 OH⁻ 生成水——四种离子全部参与了反应。这是比较少见的"零旁观离子"离子反应，离子方程式和化学方程式形式相同只是拆开了）

### 12.3 与现实/直觉的连接

- **观众和演员**：离子方程式就像一部电影的"演员表"——只列出真正参与剧情的演员（参加反应的离子），spectator ions 就像背景里的群众演员，虽然在场景中但不推动剧情。Na⁺ 和 NO₃⁻ 在酸碱中和反应中就是"永远的观众"。
- **净离子反应 = 最小表达**：离子方程式是化学反应的最简表达——它告诉你"到底发生了什么"。所有酸碱中和的实质都是 H⁺ + OH⁻ → H₂O，无论用 HCl+NaOH 还是 HNO₃+KOH，离子方程式都一样。这正是科学的简约之美。
- **离子方程式和水质检测**：检测水中的 Cl⁻ 时加入 AgNO₃——实质反应是 Ag⁺ + Cl⁻ → AgCl↓，无论水中还有什么其他离子。理解了离子反应的本质，就能理解分析化学中各类检测方法的原理。

## 十三、竞赛拓展

### 13.1 少量/过量问题的系统分析方法——"设不足量为 1"法

竞赛中少量/过量问题的难度远超课内，常涉及多元酸/碱、连续反应体系。

**系统步骤**：
1. 判断哪种反应物不足量（通常题干直接给出，或通过"逐滴加入"等描述暗示）；
2. 设不足量反应物的系数为 **1**；
3. 写出不足量一方提供的离子数（如 1 mol $Ca(OH)_2$ → $1Ca^{2+} + 2OH^-$）；
4. 按反应优先级配平（中和 > 沉淀 > 氧化还原 > 配位）；
5. 检查所有离子是否合理消耗——过量一方剩余的离子**不写入离子方程式**（成为旁观离子）；
6. 化简系数。

**示例**：$NaHSO_4$ 溶液与 $Ba(OH)_2$ 溶液：
- $NaHSO_4$ 少量：$Ba^{2+} + OH^- + H^+ + SO_4^{2-} \rightarrow BaSO_4\downarrow + H_2O$
- $Ba(OH)_2$ 少量：$Ba^{2+} + 2OH^- + 2H^+ + SO_4^{2-} \rightarrow BaSO_4\downarrow + 2H_2O$

### 13.2 非水溶剂中的离子方程式

在水溶液中学到的"拆不拆"规则**不可直接套用于非水溶剂体系**。

**液氨体系**（$l.NH_3$）：
- 液氨中 $NH_4Cl$ 是完全电离的，但溶解度的概念与水中完全不同
- $AgCl$ 在水中难溶，但在液氨中可溶（因生成 $[Ag(NH_3)_2]^+$）
- 离子方程式的书写需基于该溶剂中的实际溶解和电离情况

**启示**：离子方程式的本质是描述"实际参与反应的粒子"，不同溶剂中"实际存在的粒子"不同。竞赛中偶尔出现以信息题形式考查此概念。

### 13.3 熔融盐中的离子反应

熔融态下的离子反应不能照搬溶液中的书写方式。

**区别要点**：
- 熔融 $NaCl$ 中确实存在 $Na^+$ 和 $Cl^-$，但没有水合作用
- 电解熔融 $NaCl$：$2Na^+ + 2Cl^- \xrightarrow{电解} 2Na + Cl_2\uparrow$（这里拆是因为熔融态电离）
- **与溶液的区别**：水溶液中电解 $NaCl$ 时，阳极放电的是 $Cl^-$（产生 $Cl_2$），但在阴极放电的是 $H_2O$ 中的 $H^+$（产生 $H_2$ 和 $OH^-$），而非 $Na^+$——离子方程式完全不同

**竞赛要点**：电解熔融盐 vs 电解水溶液的离子方程式对比，是竞赛中的经典辨析题。
