# 分散相与连续相

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[03-分散相与连续相]]

## 一、基本概念
- **分散相**（dispersed phase）：被分散的物质
- **连续相**（continuous phase / 分散介质）：分散相所处的介质

## 二、分散体系分类
| 类型      |   分散相尺寸   | 实例       |
| ------- | :-------: | -------- |
| 真溶液     |  < 1 nm   | 盐水、糖水    |
| 胶体      | 1–1000 nm | 牛奶、豆浆、溶胶 |
| 悬浊液/乳浊液 | > 1000 nm | 泥水、油水混合物 |


## 三、核心原理
- **分散体系定义**：一种或多种物质分散在另一种物质中形成的体系。被分散的物质称为**分散相**（dispersed phase），分散相所处的介质称为**分散介质**或**连续相**（continuous phase / dispersion medium）
- **溶液的定义**：两种或两种以上的物质混合形成均匀稳定的分散体系叫做溶液（solution）。溶液可以是液态、气态或固态
  - 气态溶液示例：空气（O$_2$、N$_2$、Ar 等多种气体的均匀混合物），气体分子间作用力很小，各种分子互不干扰
  - 固态溶液示例：黄铜（Zn 溶于 Cu），属于合金范畴，原子间作用力较强，结构较复杂
- **分散体系按粒子尺寸分类**：
  - 真溶液：分散相粒子 $< 1\ \mathrm{nm}$（$10^{-9}\ \mathrm{m}$），如糖水中的糖分子、盐水中的 Na$^+$ 和 Cl$^-$
  - 胶体溶液：分散相粒子 $1\sim1000\ \mathrm{nm}$，如血液、淋巴液、墨水
  - 粗分散体系：分散相粒子 $> 1000\ \mathrm{nm}$，如悬浊液（泥浆）和乳浊液（牛奶、豆浆）
- **溶解过程的本质**：溶解不是机械混合的物理过程，而是特殊的物理化学过程
  - 过程（i）：溶质分子或离子的离散——需吸热，克服原有质点间的吸引力，溶液体积增大
  - 过程（ii）：溶剂分子与溶质分子的溶剂化——放热，体积缩小
  - 整个过程是放热还是吸热、体积是缩小还是增大，取决于两个因素的相对大小
  - 例：H$_2$SO$_4$ 溶于水放热，NH$_4$NO$_3$ 溶于水吸热；$50\ \mathrm{cm^3}$ 无水乙醇 + $50\ \mathrm{cm^3}$ 水总体积 $< 100\ \mathrm{cm^3}$；$50\ \mathrm{cm^3}$ 醋酸 + $50\ \mathrm{cm^3}$ 水总体积 $> 100\ \mathrm{cm^3}$

## 四、关键结论
- **溶剂与溶质的区分**：气体或固体溶于液体时，液体为溶剂，气体或固体为溶质；两种液体互溶时，量多的为溶剂
  - 例：啤酒乙醇含量约 $4\%$，水是溶剂；白酒乙醇含量可达 $60\%$，乙醇是溶剂
- **胶体溶液三种类型**：
  - 溶胶：固态胶体粒子分散于液态介质中（如 Fe(OH)$_3$ 溶胶、As$_2$S$_3$ 溶胶），热力学不稳定但动力学稳定（靠胶粒带电排斥维持）
  - 大分子溶液：橡胶、蛋白质、淀粉等溶于溶剂形成，属亲液胶体（lyophilic colloid），稳定性来自高度溶剂化，溶解与沉淀可逆
  - 缔合胶体：表面活性剂分子在浓度约 $0.01\sim0.02\ \mathrm{mol\cdot dm^{-3}}$ 时自组装形成胶束（micelle）
- **胶体特性**：Tyndall 效应（光散射）、电泳（带电胶粒在电场中迁移）、渗析（半透膜分离胶粒与小分子/离子）、聚沉（加电解质破坏胶粒电荷排斥）
- **颜色变化与溶剂化**：无水 CuSO$_4$ 白色粉末溶于水变蓝色——Cu$^{2+}$ 本身无色，与水生成 Cu(H$_2$O)$_4^{2+}$ 水合离子呈蓝色

## 五、常见分类或情形
- **按分散相尺寸**：真溶液（$< 1\ \mathrm{nm}$）、胶体（$1\sim1000\ \mathrm{nm}$）、粗分散体系（$> 1000\ \mathrm{nm}$）
- **按物态**：
  - 气态溶液（如空气）
  - 液态溶液（本章重点）
  - 固态溶液（如合金）
- **胶体的电性分类**：
  - 正电胶体：如 Fe(OH)$_3$ 溶胶（胶核吸附 FeO$^+$）
  - 负电胶体：如 As$_2$S$_3$ 溶胶
  - AgI 溶胶：KI 过量时形成负电胶体，AgNO$_3$ 过量时形成正电胶体
- **胶体的亲液/憎液分类**：
  - 亲液胶体（大分子溶液）：稳定性来自溶剂化，不带电荷，可逆
  - 憎液胶体（一般溶胶）：稳定性来自电荷排斥，不可逆

## 六、适用条件与限制
- 按粒子尺寸划分分散体系是粗略的分类，边界并非绝对严格
- 胶体粒子尺寸范围（$1\sim1000\ \mathrm{nm}$）与纳米材料尺寸范围一致，纳米科技关注这一尺度的特殊性质
- "相似者相溶"原理适用于分子型物质，对离子型化合物（如 BaCO$_3$、BaSO$_4$、LiF 虽属离子型但在水中溶解度很小）涉及更复杂的因素

## 七、常见比较与易混点
- **溶液 vs 胶体 vs 粗分散体系**：肉眼看来溶液和胶体都是透明均匀的，但胶体有 Tyndall 效应（光柱），溶液没有；粗分散体系粒子可用肉眼或放大镜看到
- **亲液胶体 vs 憎液胶体**：亲液胶体溶解沉淀可逆，稳定性来自溶剂化；憎液胶体不可逆，稳定性来自胶粒电荷。向憎液溶胶中加入足量大分子溶液可起保护作用
- **溶胶 vs 大分子溶液 vs 缔合胶体**：三者虽同属胶体范畴，形成机制和稳定机制不同
- **溶解 vs 化学反应**：溶解伴有能量、体积、颜色变化，但用蒸馏、结晶等物理方法仍能分离溶质与溶剂，故不同于纯化学过程
