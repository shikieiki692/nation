# 酸碱反应

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 关联笔记：[[酸碱理论]], [[质子守恒]], [[指示剂]], [[容量分析计算]]

## 一、定义

酸碱反应是指酸与碱之间发生的质子（H⁺）转移反应，其本质是 Brønsted-Lowry 酸（质子给体）向 Brønsted-Lowry 碱（质子受体）转移质子的过程。广义上还包括 Lewis 酸碱的电子对授受反应。

## 二、考纲对应

- 对应考纲条目：[[08-酸碱理论]]、[[09-水溶液中的离子平衡]]
- 所属模块：[[基础要求-化学原理]]
- 本知识点在考纲中的作用：酸碱反应是化学竞赛最基础的反应类型之一，Brønsted-Lowry 质子理论是理解水溶液平衡（pH、缓冲、水解、滴定）的核心框架。Lewis 酸碱理论进一步扩展到有机化学和配位化学领域。

## 三、核心原理

### 3.1 Brønsted-Lowry 酸碱反应通式

$$\text{HA} + \text{B} \rightleftharpoons \text{A}^- + \text{BH}^+$$

其中 HA/A⁻ 和 BH⁺/B 分别为共轭酸碱对。

### 3.2 酸碱反应的类型

| 类型 | 反应式 | 特征 |
|------|--------|------|
| **强酸+强碱** | HCl + NaOH → NaCl + H₂O | 完全中和，pH = 7（25°C） |
| **强酸+弱碱** | HCl + NH₃ → NH₄Cl | 生成弱酸（NH₄⁺），pH < 7 |
| **弱酸+强碱** | CH₃COOH + NaOH → CH₃COONa + H₂O | 生成弱碱（CH₃COO⁻），pH > 7 |
| **弱酸+弱碱** | CH₃COOH + NH₃ → CH₃COONH₄ | pH 取决于 Ka 和 Kb 的相对大小 |
| **多元酸碱** | H₂CO₃ + 2NaOH → Na₂CO₃ + 2H₂O | 分步中和，存在多个化学计量点 |

### 3.3 滴定分析中的酸碱反应

- **化学计量点**：$n(\text{酸}) = n(\text{碱})$（按反应计量比）
- **指示剂选择**：根据化学计量点 pH 选择变色范围合适的指示剂
- **滴定突跃**：浓度越大、Ka/Kb 越强，突跃范围越大

### 3.4 Lewis 酸碱反应

- **Lewis 酸**：电子对受体（如 BF₃、AlCl₃、金属离子）
- **Lewis 碱**：电子对给体（如 NH₃、H₂O、醚、烯烃）
- **应用**：配位反应、Friedel-Crafts 催化、亲电加成中的 π 络合物

## 四、关键结论

1. **酸碱反应的本质**：质子（H⁺）从 Brønsted 酸（质子给体）转移到 Brønsted 碱（质子受体）的过程。这是所有 Brønsted 酸碱反应的统一描述框架，详见 [[酸碱理论#3.2 Brønsted-Lowry 酸碱（质子理论）|Brønsted 酸碱理论]]。

2. **中和反应**：$\ce{H+ + OH- -> H2O}$ 是强酸+强碱中和的净离子方程式。但弱酸/弱碱的中和是**平衡反应**（非定量完成），如 $\ce{CH3COOH + OH- <=> CH3COO- + H2O}$，其完成度取决于弱酸的 $K_a$。

3. **水解反应的本质**：盐的离子与水发生质子转移，使溶液呈酸/碱性。这本质上是 Brønsted 酸碱反应——水既可作酸（对弱酸根离子）也可作碱（对弱碱阳离子）。例如 $\ce{CO3^{2-} + H2O <=> HCO3- + OH-}$ 中 $\ce{H2O}$ 是质子给体（酸），$\ce{CO3^{2-}}$ 是质子受体（碱）。

4. **缓冲作用机理**：弱酸+其共轭碱（或弱碱+其共轭酸）组成的体系能抵抗外加少量强酸/强碱引起的 pH 变化。外加 $\ce{H+}$ 时共轭碱消耗之，外加 $\ce{OH-}$ 时弱酸消耗之。详见 [[缓冲溶液]]。

5. **酸碱反应的方向**：质子总是从较强的酸转移到较强的碱上，生成较弱的共轭碱和较弱的共轭酸——即**强酸+强碱 → 弱酸+弱碱**。判断依据是比较双方 $pK_a$：$pK_a$ 小者给出质子。

6. **两性物质**（如 $\ce{HCO3-}$、$\ce{H2PO4-}$、$\ce{HS-}$、$\ce{H2O}$）既可作酸给出质子，也可作碱接受质子，具体行为取决于反应对方。其溶液 pH 由 $K_a$ 和 $K_b$ 的相对大小决定。

## 五、常见分类或情形

### 5.1 强酸 + 强碱 → 完全中和

$$\ce{HCl + NaOH -> NaCl + H2O}$$

净离子式：$\ce{H+ + OH- -> H2O}$，反应完全进行，$K \approx 1/K_w \approx 10^{14}$（水的自电离逆反应）。化学计量点时 pH = 7（25°C）。

### 5.2 弱酸 + 强碱 → 部分中和，终点呈碱性

$$\ce{CH3COOH + NaOH -> CH3COONa + H2O}$$

净离子式：$\ce{CH3COOH + OH- -> CH3COO- + H2O}$。生成的 $\ce{CH3COO-}$ 是弱碱（$K_b = K_w/K_a$），水解使化学计量点 pH > 7（约 8.7）。

### 5.3 弱碱 + 强酸 → 部分中和，终点呈酸性

$$\ce{NH3 + HCl -> NH4Cl}$$

净离子式：$\ce{NH3 + H+ -> NH4+}$。生成的 $\ce{NH4+}$（$pK_a \approx 9.25$）是弱酸，水解使化学计量点 pH < 7（约 5.1）。

### 5.4 盐的水解反应

$$\ce{Na2CO3 + H2O <=> NaHCO3 + NaOH}$$

净离子式（第一级）：$\ce{CO3^{2-} + H2O <=> HCO3- + OH-}$，$K_{h1} = K_w / K_{a2}(\ce{H2CO3})$。

第二级水解：$\ce{HCO3- + H2O <=> H2CO3 + OH-}$，$K_{h2} = K_w / K_{a1}(\ce{H2CO3})$。通常 $K_{h1} \gg K_{h2}$，以第一级为主。

### 5.5 缓冲反应

加入少量强酸（$\ce{H+}$）时：$\ce{A- + H+ -> HA}$（共轭碱消耗外来酸）

加入少量强碱（$\ce{OH-}$）时：$\ce{HA + OH- -> A- + H2O}$（弱酸消耗外来碱）

pH 由 Henderson-Hasselbalch 方程决定：$\mathrm{pH} = pK_a + \log\frac{[\ce{A-}]}{[\ce{HA}]}$。

### 5.6 两性物质的酸碱反应

两性物质可与酸或碱分别反应，不同方向表现出不同角色：

- 与酸反应（作碱）：$\ce{HCO3- + H+ -> H2CO3}$（继而分解为 $\ce{CO2\uparrow + H2O}$）
- 与碱反应（作酸）：$\ce{HCO3- + OH- -> CO3^{2-} + H2O}$

$\ce{H2PO4-}$ 同样：加酸 $\ce{-> H3PO4}$，加碱 $\ce{-> HPO4^{2-}}$。

## 六、适用条件与限制

1. **水解程度取决于 $K_a$ / $K_b$**：盐的水解强弱由对应弱酸/弱碱的强度决定——酸越弱（$K_a$ 越小），其共轭碱的水解越强（$K_h = K_w/K_a$ 越大）；碱越弱（$K_b$ 越小），其共轭酸的水解越强。这意味着极弱酸对应的盐（如 $\ce{Na3PO4}$）水解非常剧烈，溶液呈强碱性。

2. **缓冲范围 $\mathrm{p}K_a \pm 1$**：缓冲溶液的有效缓冲区间约为 $pK_a \pm 1$（即共轭酸碱对浓度比在 $1:10$ 到 $10:1$ 之间）。超出此范围，缓冲能力急剧下降，外加少量酸/碱即引起显著 pH 变化。这与 [[酸碱理论#3.3 酸碱强度与 pKa|pKa 的定义]] 直接相关——选择缓冲对时，$pK_a$ 应尽量接近目标 pH。

3. **温度影响 $K_w$**：水的离子积 $K_w$ 随温度升高显著增大（25°C: $1.0\times 10^{-14}$；100°C: $\sim 5.5\times 10^{-13}$）。这意味着中性 pH 在不同温度下不同（100°C 时 pH = 6.14 即为中性），且水解平衡常数 $K_h = K_w/K_a$ 也随温度变化——高温时盐水解更显著。

4. **浓溶液中活度偏离浓度**：在较浓溶液（$>0.1\ \mathrm{mol\cdot L^{-1}}$）中，离子间的静电相互作用导致活度系数显著偏离 1，使用浓度直接代入平衡常数计算会产生误差。竞赛级别的精确计算有时需要考虑 Debye-Hückel 极限公式进行活度校正。

5. **非水溶剂中酸碱定义不同**：在 [[酸碱理论#3.4 溶剂对酸碱行为的影响|非水溶剂]] 中，Brønsted 酸碱的强度排序可能因拉平/区分效应而改变。Lewis 酸碱反应在非质子溶剂（如乙醚、THF）中不受此限，但溶剂本身的 Lewis 碱性可能竞争配位。另外，非水体系中的"pH"概念需要用 Hammett 酸度函数 $H_0$ 等替代（见 [[酸碱理论#溶剂的自电离|溶剂自电离]]）。

## 七、常见比较与易混点

### 7.1 中和 vs 水解

| | 中和反应 | 水解反应 |
|:---|:---|:---|
| **反应物** | 酸 + 碱 | 盐 + 水 |
| **本质** | $\ce{H+ + OH- -> H2O}$（强-强）或质子转移平衡（弱-强/弱-弱） | 盐的离子与水的质子转移——本质也是 Brønsted 酸碱反应 |
| **热效应** | 强-强中和热 $\approx -57.3\ \mathrm{kJ\cdot mol^{-1}}$ | 通常较小 |
| **可逆性** | 强-强几乎不可逆；涉及弱酸/弱碱时部分可逆 | 均为平衡反应，水解度通常较小 |

### 7.2 水解 vs 水合

- **水解（Hydrolysis）**：化学变化——盐的离子与水发生质子转移，改变了溶液的 pH。例如 $\ce{CO3^{2-} + H2O <=> HCO3- + OH-}$。
- **水合（Hydration）**：物理过程——离子被水分子包围（溶剂化），不涉及质子转移。例如 $\ce{CuSO4}$ 溶于水变蓝是 $\ce{Cu^{2+}}$ 与 $\ce{H2O}$ 形成 $\ce{[Cu(H2O)6]^{2+}}$ 水合离子。
- 学生常将 $\ce{CuSO4}$ 溶于水变蓝（水合）与 $\ce{Na2CO3}$ 溶于水呈碱性（水解）混淆——前者是配位水合，后者是质子转移。

### 7.3 酸性盐 ≠ 酸式盐

- **酸式盐**：化学式中仍含可电离 H 的盐，如 $\ce{NaHCO3}$、$\ce{NaH2PO4}$、$\ce{NaHSO4}$——按**化学组成**分类。
- **酸性盐**：水溶液呈酸性（pH < 7）的盐，如 $\ce{NH4Cl}$、$\ce{FeCl3}$、$\ce{NaHSO4}$——按**溶液 pH** 分类。
- **关键反例**：$\ce{NaHCO3}$ 是**酸式盐**，但溶液是**碱性的**（$\ce{HCO3-}$ 的 $K_b > K_a$）。这恰恰说明酸式盐不等于酸性盐。

### 7.4 $\ce{NaHCO3}$ 是碱性，不是酸性！

这是最重要、最高频的迷思之一：

- $\ce{HCO3-}$ 有两性：$K_a(\ce{HCO3-}) = 4.7 \times 10^{-11}$（给出质子 → $\ce{CO3^{2-}}$），$K_b(\ce{HCO3-}) = K_w/K_a(\ce{H2CO3}) = 2.3 \times 10^{-8}$（接受质子 → $\ce{H2CO3}$）。
- 因为 $K_b \gg K_a$（差约 500 倍），$\ce{NaHCO3}$ 溶液呈**弱碱性**（pH ≈ 8.3）。
- 学生直觉「酸式盐 = 酸性」必须纠正。类似地，$\ce{NaH2PO4}$ 溶液呈**弱酸性**（$K_{a2} > K_b$），$\ce{Na2HPO4}$ 溶液呈**弱碱性**（$K_b > K_{a3}$）——一切以 $K_a$ 与 $K_b$ 的数值对比为准。

### 7.5 缓冲溶液 ≠ 中性溶液

- 缓冲溶液可以有任意 pH——只要包含了共轭酸碱对，pH 可以是 4、7、10 等。
- 例如 $\ce{CH3COOH/CH3COONa}$ 缓冲的是 pH ≈ 4.76 附近，$\ce{NH3/NH4Cl}$ 缓冲的是 pH ≈ 9.25 附近。
- 「缓冲 = 中性」是日常语义污染——"缓冲"在化学中仅仅是**抵抗 pH 变化**，不预设目标 pH。

### 7.6 酸碱指示剂的变色原理

- 酸碱指示剂（如酚酞、甲基橙）本身是**弱有机酸/碱**，其酸形式与共轭碱形式颜色不同。
- 变色区间约为 $\mathrm{p}K_a(\text{指示剂}) \pm 1$——这本质上是 Brønsted 酸碱反应的平衡移动。
- 例如酚酞：$\ce{HIn <=> H+ + In-}$，$\ce{HIn}$ 无色，$\ce{In-}$ 粉红，$pK_a \approx 9.1$，变色区间 pH 8.2–10.0。
- 详见 [[指示剂]]。

### 7.7 酸式盐溶液 pH 的判断准则

对于酸式盐 $\ce{NaHA}$（两性物质），其溶液 pH 由 $K_{a1}$ 和 $K_{a2}$ 共同决定：

$$\mathrm{pH} = \frac{1}{2}(pK_{a1} + pK_{a2})$$

- 若 $K_b > K_{a2}$（即 $K_h > K_{a2}$），溶液呈碱性（如 $\ce{NaHCO3}$）
- 若 $K_{a2} > K_b$（即 $K_{a2} > K_h$），溶液呈酸性（如 $\ce{NaH2PO4}$）
- 若两者相近，溶液近中性（如 $\ce{NH4Ac}$，$K_a(\ce{NH4+}) \approx K_b(\ce{Ac-})$）

## 九、典型题型

1. **酸碱反应方向判断**：给定酸和碱的 $K_a$ / $pK_a$，判断反应能否正向进行。原则：质子从强酸转移到强碱——比较双方 $pK_a$，$pK_a$ 更小者给出质子。经典考法："$\ce{CH3COOH + NaHCO3}$ 是否反应？"（$pK_a$: $\ce{CH3COOH}$ 4.76 < $\ce{H2CO3}$ 6.35，故醋酸给出质子 → 反应正向进行，产生 $\ce{CO2}$↑）。

2. **酸碱滴定终点判断与指示剂选择**：已知滴定反应类型（强-强/强-弱/弱-强），计算化学计量点 pH，选择合适指示剂。如 $\ce{NaOH}$ 滴定 $\ce{CH3COOH}$：终点为 $\ce{CH3COONa}$ 溶液，pH ≈ 8.7，应选**酚酞**（变色区间 8.2–10.0）而非甲基橙（3.1–4.4）。详见 [[指示剂]]、[[容量分析计算]]。

3. **多元酸分步滴定**：判断能否分步滴定（判据：$K_{a1}/K_{a2} \gtrsim 10^4$ 且 $cK_a \gtrsim 10^{-8}$）。如 $\ce{H3PO4}$ 的 $K_{a1}/K_{a2} \approx 10^5$（可分步），而 $\ce{H2SO4}$ 第一级为强酸、第二级 $K_{a2} \approx 10^{-2}$（不可分步滴定出两个突跃）。

4. **盐水解 pH 计算**：给定盐浓度和对应 $K_a$ 或 $K_b$，计算水解度与溶液 pH。覆盖四类盐：
   - 强酸弱碱盐（如 $\ce{NH4Cl}$）：$[\ce{H+}] = \sqrt{K_h \cdot C}$，$K_h = K_w/K_b$
   - 强碱弱酸盐（如 $\ce{NaAc}$）：$[\ce{OH-}] = \sqrt{K_h \cdot C}$，$K_h = K_w/K_a$
   - 酸式盐（如 $\ce{NaHCO3}$）：$\mathrm{pH} = \frac{1}{2}(pK_{a1}+pK_{a2})$
   - 弱酸弱碱盐（如 $\ce{NH4Ac}$）：$\mathrm{pH} = \frac{1}{2}(pK_a + pK_w - pK_b)$

5. **缓冲溶液配方与计算**：给定目标 pH，选择合适的缓冲对（$pK_a$ 接近目标 pH）并用 Henderson-Hasselbalch 方程计算所需共轭酸碱比例。常见考法：配制 pH = 5.00 的缓冲溶液需多少 $\ce{NaAc}$ 加入给定浓度的 $\ce{HAc}$ 中？或外加少量酸/碱后缓冲溶液 pH 的变化量。

6. **滴定曲线综合解读**：给定滴定曲线（pH-V 图），识别以下关键特征并反推酸的信息：
   - 突跃区间（→ 酸的浓度和强度）
   - 半中和点（$\mathrm{pH} = pK_a$，$[\ce{HA}] = [\ce{A-}]$）
   - 化学计量点 pH（→ 产物盐的类型和水解行为）
   - 多级电离的多个突跃和缓冲区（→ $pK_{a1}, pK_{a2}, \dots$）

7. **Lewis 酸碱识别**：判断反应中的 Lewis 酸（电子对受体）和 Lewis 碱（电子对给体）。如 $\ce{BF3 + NH3 -> F3B-NH3}$ 中 $\ce{BF3}$ 是 Lewis 酸（缺电子 B），$\ce{NH3}$ 是 Lewis 碱（N 有孤对电子）。竞赛中常延伸至催化循环中的 Lewis 酸角色（如 $\ce{AlCl3}$ 在 Friedel-Crafts 反应中的作用）。

## 十、例题

### 10.1 例题 1（★ 中和反应热化学）

**题目**：$25\mathrm{°C}$ 时，$100\ \mathrm{mL}$ 的 $1.0\ \mathrm{mol\cdot L^{-1}}\ \ce{HCl}$ 与 $100\ \mathrm{mL}$ 的 $1.0\ \mathrm{mol\cdot L^{-1}}\ \ce{NaOH}$ 在量热计中混合，溶液温度升高 $6.7\mathrm{°C}$。已知溶液比热容为 $4.18\ \mathrm{J\cdot g^{-1}\cdot °C^{-1}}$，密度为 $1.0\ \mathrm{g\cdot mL^{-1}}$。计算中和反应热 $\Delta H$（$\mathrm{kJ\cdot mol^{-1}}$）。

**分析**：
- 强酸强碱中和，净反应 $\ce{H+ + OH- -> H2O}$
- 反应放热 = 溶液吸热 = $mc\Delta T$
- 物质的量以限量反应物计（此处两者等量）

**解答**：
$$Q = mc\Delta T = (200\ \mathrm{g}) \times (4.18\ \mathrm{J\cdot g^{-1}\cdot °C^{-1}}) \times (6.7\mathrm{°C}) = 5.60\ \mathrm{kJ}$$

$$n(\ce{H2O}\text{生成}) = 0.100\ \mathrm{L} \times 1.0\ \mathrm{mol\cdot L^{-1}} = 0.100\ \mathrm{mol}$$

$$\Delta H = -\frac{Q}{n} = -\frac{5.60}{0.100} = -56.0\ \mathrm{kJ\cdot mol^{-1}}$$

**反思**：
- 标准中和热约为 $-57.3\ \mathrm{kJ\cdot mol^{-1}}$，本结果接近。实际测量中需考虑量热计热容、热损失等系统误差。
- 若换成弱酸+强碱（如 $\ce{CH3COOH + NaOH}$），由于弱酸电离吸热，放热量会偏小（中和热绝对值更小）——这部分能量用于驱动弱酸的电离平衡。
- 若换成弱碱+强酸或弱酸+弱碱，中和热进一步偏离 $-57.3\ \mathrm{kJ\cdot mol^{-1}}$——这是判断酸碱强弱的热化学方法。

---

### 10.2 例题 2（★★★ $\ce{Na2CO3}$ 双级水解 pH）

**题目**：计算 $0.10\ \mathrm{mol\cdot L^{-1}}\ \ce{Na2CO3}$ 溶液的 pH。已知 $\ce{H2CO3}$ 的 $K_{a1} = 4.5 \times 10^{-7}$，$K_{a2} = 4.7 \times 10^{-11}$。

**分析**：
- $\ce{CO3^{2-}}$ 是二元弱碱，其 $K_{b1} = K_w/K_{a2}$，$K_{b2} = K_w/K_{a1}$
- $K_{b1} \gg K_{b2}$（差约 $10^4$ 倍），只需考虑第一级水解

**解答**：
$$K_{b1} = \frac{K_w}{K_{a2}} = \frac{1.0 \times 10^{-14}}{4.7 \times 10^{-11}} = 2.13 \times 10^{-4}$$

$$\ce{CO3^{2-} + H2O <=> HCO3- + OH-}$$

$$K_{b1} = \frac{[\ce{HCO3-}][\ce{OH-}]}{[\ce{CO3^{2-}}]} = \frac{x^2}{0.10 - x}$$

$C/K_{b1} = 0.10 / (2.13 \times 10^{-4}) = 470 > 400$，可用近似：

$$x = [\ce{OH-}] = \sqrt{K_{b1} \cdot C} = \sqrt{(2.13 \times 10^{-4}) \times 0.10} = 4.62 \times 10^{-3}\ \mathrm{mol\cdot L^{-1}}$$

$$\mathrm{pOH} = -\log(4.62 \times 10^{-3}) = 2.34$$

$$\mathrm{pH} = 14 - 2.34 = 11.66$$

**反思**：
- 验证近似合理性：水解度 $\alpha = 4.62 \times 10^{-3} / 0.10 = 4.6\%$（< 5%），近似合理。
- 第二级水解 $K_{b2} = 1.0\times 10^{-14} / 4.5\times 10^{-7} = 2.2\times 10^{-8} \ll K_{b1}$，可忽略。
- 这就是为什么 $\ce{Na2CO3}$ 俗称"纯碱"——$0.10\ \mathrm{M}$ 溶液的 pH 高达 ~11.7。同理 $\ce{Na3PO4}$ 的碱性更强（$\ce{PO4^{3-}}$ 的 $K_{b1} = K_w/K_{a3} \approx 2.3 \times 10^{-2}$）。

### 10.3 例题 3（★★★★ $\ce{H3PO4}$ + $\ce{NaOH}$ 多级中和产物判断）

**题目**：向 $0.10\ \mathrm{mol}$ 的 $\ce{H3PO4}$ 溶液中逐次加入 $\ce{NaOH}$，分别加入 (1) $0.05\ \mathrm{mol}$，(2) $0.10\ \mathrm{mol}$，(3) $0.15\ \mathrm{mol}$，(4) $0.20\ \mathrm{mol}$ 的 $\ce{NaOH}$。判断各阶段的主要溶质成分与近似 pH（忽略体积变化）。已知 $\ce{H3PO4}$ 的 $pK_{a1} = 2.12$，$pK_{a2} = 7.20$，$pK_{a3} = 12.36$。

**分析**：
- $\ce{H3PO4}$ 是三元酸，分三步被中和：

$$\ce{H3PO4 + NaOH -> NaH2PO4 + H2O}$$
$$\ce{NaH2PO4 + NaOH -> Na2HPO4 + H2O}$$
$$\ce{Na2HPO4 + NaOH -> Na3PO4 + H2O}$$

- 物质的量比决定产物阶段。

**解答**：
设初始 $n(\ce{H3PO4}) = 0.10\ \mathrm{mol}$。

| 编号 | $n(\ce{NaOH})$ | 比值 | 阶段 | 主要溶质 | 近似 pH |
|:---:|:---:|:---:|:---|:---|:---:|
| (1) | $0.05\ \mathrm{mol}$ | $0.5$ | 第一半中和点 | $\ce{H3PO4}$ + $\ce{NaH2PO4}$（1:1 缓冲对） | $\approx 2.12$ |
| (2) | $0.10\ \mathrm{mol}$ | $1.0$ | 第一化学计量点 | $\ce{NaH2PO4}$（两性物质） | $\approx 4.66$ |
| (3) | $0.15\ \mathrm{mol}$ | $1.5$ | 第二半中和点 | $\ce{NaH2PO4}$ + $\ce{Na2HPO4}$（1:1 缓冲对） | $\approx 7.20$ |
| (4) | $0.20\ \mathrm{mol}$ | $2.0$ | 第二化学计量点 | $\ce{Na2HPO4}$（两性物质） | $\approx 9.78$ |

pH 计算公式：
- 半中和点：$\mathrm{pH} = pK_a$（缓冲比 1:1）
- 化学计量点（酸式盐）：$\mathrm{pH} = \frac{1}{2}(pK_{a\text{前}} + pK_{a\text{后}})$

**反思**：
- 多元酸的中和是**分步**的——强碱优先与最强的酸性质子反应。这是热力学次序：$K_{a1} \gg K_{a2} \gg K_{a3}$ 保证了三步几乎互不干扰。
- 相邻 $pK_a$ 差：$pK_{a2} - pK_{a1} = 5.08$，$pK_{a3} - pK_{a2} = 5.16$，均 $\gtrsim 4$ → 可清晰分步滴定出三个突跃。
- 第三化学计量点（$n(\ce{NaOH}) = 0.30\ \mathrm{mol}$）时生成 $\ce{Na3PO4}$，溶液呈强碱性（$\ce{PO4^{3-}}$ 的 $K_{b1} \approx 2.3 \times 10^{-2}$，pH ≈ 12.7）。
- 此题是竞赛中"滴定产物判断"和"多级缓冲体系"的经典模型，可进一步延伸至 $\ce{H2CO3}$（两级）或 $\ce{H4Y}$（EDTA，四级）体系。

## 十一、易错点

1. ❌ **"中和反应一定生成中性溶液"**
   - **学生为什么会这么想**：初中化学的简化模型——酸+碱=盐+水——让学生把"中和"（neutralization）与"中性"（neutral，pH = 7）画了等号。两者的英文同源更加深了这个混淆。
   - **正确认识**：只有**强酸+强碱**按化学计量比完全中和时，溶液才呈中性（pH = 7，25°C）。弱酸+强碱中和后溶液呈碱性（弱酸根水解），弱碱+强酸中和后呈酸性（弱碱阳离子水解）。"中和"描述的是**反应过程**（酸+碱反应），不是**产物的 pH**。详见 §七 7.1。

2. ❌ **"$\ce{NaHCO3}$ 是酸性盐，所以溶液呈酸性"**
   - **学生为什么会这么想**：看到化学式里有 H 原子，就认为能电离出 $\ce{H+}$，加上"酸式盐"中带个"酸"字，直觉归类为酸性物质。初中阶段 $\ce{NaHSO4}$（确实是酸性的）是唯一接触到的酸式盐例子，形成了错误泛化。
   - **正确认识**：$\ce{HCO3-}$ 确实能电离出 $\ce{H+}$（$K_{a2} = 4.7 \times 10^{-11}$），但它同时也能水解接受 $\ce{H+}$（$K_b = 2.3 \times 10^{-8}$）。因为 $K_b \gg K_{a2}$（约 500 倍），水解占主导，溶液实际呈**弱碱性**（pH ≈ 8.3）。命名中有"酸"字不等于溶液呈酸性。详见 §七 7.4。

3. ❌ **"所有盐类都呈中性"**
   - **学生为什么会这么想**：初中将"盐"定义为"酸碱中和的产物"，而"中和"又被误解为达到中性，因此盐就是中性的——完美（但错误）的三段论。
   - **正确认识**：盐的酸碱性取决于其组成离子的水解行为。四类盐分别对应四种 pH 情况：$\ce{NaCl}$ 中性、$\ce{Na2CO3}$ 碱性、$\ce{NH4Cl}$ 酸性、$\ce{NH4Ac}$ 近似中性（$K_a \approx K_b$）。

4. ❌ **"多元酸的中和是一次性完成的"**
   - **学生为什么会这么想**：平时写化学反应方程式习惯写总反应（如 $\ce{H3PO4 + 3NaOH -> Na3PO4 + 3H2O}$），忽略了分步过程。教材中有时也直接给出总方程式。
   - **正确认识**：多元酸/碱的中和是**逐步**进行的，质子按酸性强弱（$K_{a1} \gg K_{a2} \gg K_{a3}$）逐级被中和。当 $K_{a1}/K_{a2} \gtrsim 10^4$ 时，滴定曲线上可观察到明显分离的多个突跃——这是分步进行的实验证据。

5. ❌ **"缓冲溶液就是 pH = 7 的溶液"**
   - **学生为什么会这么想**：将"缓冲"理解为"维持中性"——受到"缓冲"日常语义（如"缓冲地带"暗示中间状态、"缓冲器"减少冲击）的影响。
   - **正确认识**：缓冲溶液可以在任意 pH 值工作——$\ce{HAc/NaAc}$ 缓冲 pH ≈ 4.76，$\ce{NH3/NH4Cl}$ 缓冲 pH ≈ 9.25。缓冲的是"pH 的稳定性（抗扰动）"，而非"中性的 pH 值"。详见 §七 7.5。

6. ❌ **"水解反应和中和反应是完全不同的两类反应"**
   - **学生为什么会这么想**：教科书中"中和"和"水解"通常在不同章节出现，学生将它们当作两个独立的知识点分别记忆，形成不同的心理"知识盒子"。
   - **正确认识**：水解反应的本质就是 Brønsted 酸碱反应——盐的离子（碱或酸）与水（酸或碱）之间的质子转移。$\ce{CO3^{2-} + H2O <=> HCO3- + OH-}$ 中 $\ce{H2O}$ 是质子给体（酸），$\ce{CO3^{2-}}$ 是质子受体（碱）。详见 [[酸碱理论#3.2 Brønsted-Lowry 酸碱（质子理论）|Brønsted 酸碱理论]] 和 §七 7.1。

7. ❌ **"两性物质 = 酸碱中和后的产物"**
   - **学生为什么会这么想**：对"两性"概念停留在 $\ce{Al(OH)3}$、$\ce{ZnO}$ 等"既能与酸又能与碱反应"的 Lewis 两性层面，未理解 Brønsted 两性的本质是"既能给出质子也能接受质子"。
   - **正确认识**：Brønsted 两性物质（如 $\ce{HCO3-}$、$\ce{H2PO4-}$、$\ce{H2O}$）具有可电离的 H 和可接受 H⁺ 的孤对电子/负电荷，能在酸和碱两种角色间灵活切换。不要将其与 Lewis 两性（如 $\ce{Al(OH)3}$ 与酸反应生成 $\ce{Al^{3+}}$、与碱反应生成 $\ce{[Al(OH)4]-}$）混为一谈——后者涉及的是 $\ce{OH-}$ 的给/受，而非质子转移。

8. ❌ **"强酸与弱酸盐反应一定发生且定量完成"**
   - **学生为什么会这么想**：记住了"强酸制弱酸"的口诀，便认为只要强酸遇到任何弱酸盐，反应就一定彻底进行。
   - **正确认识**："强酸制弱酸"是热力学方向判断，但反应是否定量完成取决于两酸 $K_a$ 的差距。当 $\Delta pK_a$ 不大时（如 $\ce{HAc}(4.76)$ 与 $\ce{H2CO3}(6.35)$ 差仅 ~1.6），反应只是平衡偏向产物，并非完全。若产物是气体（如 $\ce{CO2}$↑）或沉淀，因产物离开体系，反应才趋于完全。

### 12.3 与现实/直觉的连接

1. **胃酸中和（$\ce{Al(OH)3}$ 抗酸药）**：
   - 胃酸主要成分是 $\ce{HCl}$（pH ≈ 1–2），胃酸过多引起胃灼热（heartburn）。
   - 抗酸药中的 $\ce{Al(OH)3}$（两性氢氧化物）与胃酸反应：$\ce{Al(OH)3 + 3HCl -> AlCl3 + 3H2O}$，温和且不产生气体。
   - 为什么不用 $\ce{NaOH}$ 或 $\ce{Ca(OH)2}$？因为强碱会灼伤食道和胃黏膜——酸碱反应物的选择不仅要看化学计量，还要考虑反应速率、安全性和生物相容性。
   - $\ce{NaHCO3}$（小苏打）也是常见抗酸成分：$\ce{NaHCO3 + HCl -> NaCl + H2O + CO2}$↑，但会产生 $\ce{CO2}$ 气体引起胃胀和嗳气——这是一个化学原理直接影响用药体验的实例。

2. **酸雨与石灰石建筑**：
   - 酸雨中的 $\ce{H2SO4}$ 和 $\ce{HNO3}$ 与大理石（$\ce{CaCO3}$）发生中和反应：$\ce{CaCO3 + H2SO4 -> CaSO4 + H2O + CO2}$↑。
   - $\ce{CaSO4}$ 微溶且比 $\ce{CaCO3}$ 更容易被雨水冲刷，导致建筑表面逐年侵蚀。这是"强酸制弱酸"的生动体现——$\ce{H2SO4}$（强酸）将 $\ce{H2CO3}$（弱酸，继而分解为 $\ce{CO2}$↑）从 $\ce{CaCO3}$ 中赶出。
   - 这也是为什么酸雨对石灰石建筑和大理石雕像的破坏尤为严重——它们本身就是碳酸盐，是"天然的抗酸缓冲"被缓慢消耗殆尽的过程。

3. **血液中的缓冲系统**：
   - 人体血液的 pH 维持在 7.35–7.45 的极窄范围内，主要依靠 $\ce{H2CO3/HCO3-}$ 缓冲对。
   - $\mathrm{pH} = pK_{a1} + \log\frac{[\ce{HCO3-}]}{[\ce{H2CO3}]} = 6.1 + \log\frac{24}{1.2} \approx 7.4$。
   - 代谢产生的酸性物质（乳酸、酮体）被 $\ce{HCO3-}$ 中和：$\ce{H+ + HCO3- -> H2CO3 -> CO2}$↑（经肺呼出）——这是缓冲反应在生命系统中的核心应用。
   - 酸中毒（pH < 7.35）和碱中毒（pH > 7.45）是临床危重症——化学原理直接关系到生死。

## 十三、竞赛拓展

1. **HSAB 理论在有机/无机竞赛题中的预测应用**：Pearson 的硬软酸碱理论不只用于无机——有机竞赛中，Michael 加成（软亲核 RS⁻ → 软亲电 $\alpha,\beta$-不饱和羰基的 $\beta$-C）vs 直接加成到羰基（硬 Nu⁻ → 硬羰基 C）就是 HSAB 控制的区域选择性。竞赛题常见"这个亲核试剂进攻底物的哪个位点？"——用 HSAB 判断：软-软优先形成键（轨道控制），硬-硬优先（电荷控制）。更进阶的是 Klopman-Salem 方程：$\Delta E = -\dfrac{Q_{Nu}Q_{El}}{R\varepsilon} + \dfrac{2(c_{Nu}c_{El}\beta)^2}{E_{HOMO(Nu)}-E_{LUMO(El)}}$——第一项是硬-硬静电项，第二项是软-软共价项。

2. **超酸与 Hammett 酸度函数 $H_0$**：普通 pH 标度在浓酸中失效——$\mathrm{100\%\ H_2SO_4}$ 的 $H_0 \approx -12$，魔酸 $\mathrm{HSO_3F\cdot SbF_5}$ 的 $H_0 \approx -25$（比纯硫酸强 $10^{13}$ 倍）。$H_0 = \mathrm{p}K_{BH^+} - \log([\mathrm{BH^+}]/[\mathrm{B}])$ 通过弱碱指示剂的质子化比例测量。竞赛信息题可能给 $H_0$ 数据要求比较超酸强度或判断"该酸能否质子化甲烷/苯/CO？"。碳正离子化学（Olah 的贡献，1994 Nobel）——超酸中可稳定生成并 NMR 观测三级碳正离子、非经典碳正离子。

3. **酸碱化学中的前沿：受阻 Lewis 酸碱对（FLP）与"无金属"催化**：2006 年 Stephan 发现空间位阻大的 Lewis 酸（如 B(C₆F₅)₃）和 Lewis 碱（如 t-Bu₃P）因位阻无法形成经典加合物，但可以**协同活化 H₂**——酸接受 H⁻、碱接受 H⁺——实现无过渡金属的氢化反应。这是竞赛信息题的绝佳素材：给你 FLP 活化的机理图，要求判断谁是 Lewis 酸/碱，推电子流向。FLP 还可以活化 CO₂、NH₃、烯烃等小分子——开创了"不含金属的催化"新领域。
