# 平衡常数

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[07-化学平衡]]

## 一、定义
对于反应 $aA + bB \rightleftharpoons cC + dD$：
$$K = \frac{[C]^c[D]^d}{[A]^a[B]^b} \quad (\text{平衡时})$$

- 平衡常数是**温度的函数**（与浓度无关）
- K 值越大，平衡越偏向产物

## 三、核心原理

### K 与 ΔG 的关系
$$\Delta_r G^\circ = -RT \ln K$$
$$K = e^{-\Delta_r G^\circ / RT}$$

### 反应商 Q vs K
- Q < K：反应正向进行
- Q = K：平衡状态
- Q > K：反应逆向进行

### van't Hoff 方程（K 与 T 的关系）
$$\frac{d \ln K}{dT} = \frac{\Delta_r H^\circ}{RT^2}$$
$$\ln\frac{K_2}{K_1} = -\frac{\Delta_r H^\circ}{R}\left(\frac{1}{T_2} - \frac{1}{T_1}\right)$$

- ΔH > 0（吸热）：T↑ → K↑
- ΔH < 0（放热）：T↑ → K↓

### 平衡常数的实验测定
平衡常数可由实验直接测定（经验平衡常数 $K$），也可由 $\Delta_{\mathrm{r}}G_{\mathrm{m}}^{\ominus}(T)$ 间接求算（标准平衡常数 $K^{\ominus}$）。

**示例 -- N₂O₄ 分解平衡 (373 K)**：
$$\mathrm{N_2O_4(g)} \rightleftharpoons 2\mathrm{NO_2(g)}$$

将 0.100 mol N₂O₄ 充入 1 dm³ 密闭烧瓶，373 K 恒温下达到平衡，测得 [N₂O₄] = 0.040 mol·dm⁻³，则 [NO₂] = 0.120 mol·dm⁻³。无论从 N₂O₄ 出发还是从 NO₂ 出发，平衡时均有：
$$K = \frac{[\mathrm{NO_2}]^2}{[\mathrm{N_2O_4}]} = 0.36 \quad (373\ \mathrm{K})$$

若温度升至 423 K，$K$ 增大为 3.2。

**示例 -- 醋酸电离平衡 (293 K)**：
$$\mathrm{HAc} + \mathrm{H_2O} \rightleftharpoons \mathrm{H_3O^+} + \mathrm{Ac^-}$$

四种不同起始浓度（0.2129~0.01065 mol·dm⁻³）下，电离度 $\alpha$ 分别为 0.94%~3.9%，但 $K(\mathrm{HAc})$ 均为约 $1.8 \times 10^{-5}$，证明 $K$ 与起始浓度无关。

**示例 -- 合成氨反应的 K 随温度变化**：
$$\mathrm{N_2(g)} + 3\mathrm{H_2(g)} \rightleftharpoons 2\mathrm{NH_3(g)}$$

由 $\Delta_{\mathrm{f}}G_{\mathrm{m}}^{\ominus}$ 数据计算：
- 298 K：$\Delta G^{\ominus} = -32.8\ \mathrm{kJ \cdot mol^{-1}}$，$\lg K_p^{\ominus} = 5.76$，$K_p^{\ominus} = 5.8 \times 10^5$
- 673 K：$\Delta G^{\ominus} = +41.5\ \mathrm{kJ \cdot mol^{-1}}$，$\lg K_p^{\ominus} = -3.23$，$K_p^{\ominus} = 5.9 \times 10^{-4}$

这说明放热反应升温后 $K$ 大幅减小，与 van't Hoff 方程一致。

### $\Delta G^{\ominus}(T)$ 粗估反应方向
- 当 $\Delta G^{\ominus}(T) < -40\ \mathrm{kJ \cdot mol^{-1}}$ 时，可认为反应能正向自发进行
- 当 $\Delta G^{\ominus}(T) > +40\ \mathrm{kJ \cdot mol^{-1}}$ 时，可认为反应不能正向进行
- 介于两者之间需结合具体条件分析

## 四、关键结论
### 各类平衡常数
| 平衡类型 | 符号 | 说明 |
|------|:---:|------|
| 酸碱解离 | $K_a$, $K_b$ | 酸/碱强度 |
| 水的离子积 | $K_w$ | $1.0\times10^{-14}$ (25°C) |
| 溶度积 | $K_{sp}$ | 难溶盐的溶解平衡 |
| 稳定常数 | $K_f$（或 $\beta$）| 配合物形成 |
| 配位累积常数 | $\beta_n$ | 逐级配位 |

### Le Châtelier 原理
平衡体系受到外界扰动时，平衡向**减弱扰动**的方向移动。

| 扰动 | 平衡移动 | 对 K 的影响 |
|------|------|:---:|
| 增加反应物浓度 | → | 不变 |
| 增加产物浓度 | ← | 不变 |
| 升温（吸热 ΔH>0） | → | ↑ |
| 升温（放热 ΔH<0） | ← | ↓ |
| 增压（气体分子数↓方向） | → 气体少的一方 | 不变 |
| 催化剂 | 不移动（等速加速正逆） | 不变 |

### 书写平衡常数表达式的注意事项
1. **必须与化学方程式对应**。同一反应化学方程式写法不同，$K$ 的数值不同，但含义相同。例如：
   - $\mathrm{N_2O_4 \rightleftharpoons 2NO_2}$，$K = 0.36$ (373 K)
   - $\frac{1}{2}\mathrm{N_2O_4 \rightleftharpoons NO_2}$，$K' = \sqrt{0.36} = 0.60$
   - $2\mathrm{NO_2 \rightleftharpoons N_2O_4}$，$K'' = 1/0.36 = 2.8$

2. **$K_c$ 与 $K_p$ 的换算**：对于气相反应 $K_p = K_c(RT)^{\Delta n}$，其中 $\Delta n$ = 生成物气体系数和 $-$ 反应物气体系数和。例如 $2\mathrm{SO_2 + O_2 \rightleftharpoons 2SO_3}$ 在 1000 K、$K_p = 3.46$ 时，$K_c = K_p(RT)^{-\Delta n} = 3.46 \times (0.0831 \times 1000)^1 = 288$。

3. **多相平衡中固相不写入**。纯固体和纯液体的活度为 1：
   - $\mathrm{CaCO_3(s) \rightleftharpoons CaO(s) + CO_2(g)}$，$K_p = p(\mathrm{CO_2})$
   - $\mathrm{Fe_2O_3(s) + 3CO(g) \rightleftharpoons 2Fe(s) + 3CO_2(g)}$，$K_p = \frac{p^3(\mathrm{CO_2})}{p^3(\mathrm{CO})}$

4. **稀溶液中溶剂不写入**。如 HAc 电离：$K = \frac{[\mathrm{H_3O^+}][\mathrm{Ac^-}]}{[\mathrm{HAc}]}$，$[\mathrm{H_2O}]$ 视为常数并入 $K$。

5. **平衡常数是温度的函数**。HAc 电离：$K = 1.8 \times 10^{-5}$ (298 K)，$1.7 \times 10^{-5}$ (273 K)；$\mathrm{N_2O_4}$ 分解：$K = 0.36$ (373 K)，$3.2$ (423 K)。水溶液反应（如 HAc 电离、AgCl 沉淀）在室温范围内 $K$ 随温度变化很小，可不注明温度。

### 多重平衡规则
若反应 $③ = ① + ②$，则 $\Delta G_3^{\ominus} = \Delta G_1^{\ominus} + \Delta G_2^{\ominus}$，从而：
$$K_1 \times K_2 = K_3$$

**示例**：$\mathrm{SO_2 + \frac{1}{2}O_2 \rightleftharpoons SO_3}$ ($K_1 = 3 \times 10^{12}$)，$\mathrm{NO_2 \rightleftharpoons NO + \frac{1}{2}O_2}$ ($K_2 = 6.3 \times 10^{-7}$)，则 $\mathrm{SO_2 + NO_2 \rightleftharpoons SO_3 + NO}$ 的 $K_3 = K_1 \times K_2 = 1.9 \times 10^6$。

又如 $\mathrm{CoO(s) + H_2(g) \rightleftharpoons Co(s) + H_2O(g)}$ ($K_2 = 67$, 823 K)，$\mathrm{CO_2 + H_2 \rightleftharpoons CO + H_2O}$ ($K_1 = 0.14$)，则 $\mathrm{CoO + CO \rightleftharpoons Co + CO_2}$ 的 $K_3 = K_2/K_1 = 4.8 \times 10^2$。

### 平衡常数与转化率的关系
**平衡常数 $K$ 与转化率 $\alpha$ 是两个不同概念**：
- $K$ 在给定温度下为常数，与起始浓度无关
- $\alpha$ 随起始浓度变化而变化
- 示例：乙醇 + 醋酸酯化反应 (373 K)，无论起始浓度如何，$K$ 恒为 4.0，但各反应物的转化率可以不同（67%、83%、42% 等）

## 五、常见分类或情形

### 经验平衡常数 vs 标准平衡常数
| 类型 | 符号 | 定义 | 用途 |
|------|:---:|------|------|
| 浓度平衡常数 | $K_c$ | 以 mol·dm⁻³ 表示 | 溶液反应 |
| 压力平衡常数 | $K_p$ | 以 bar 或 atm 表示 | 气相反应 |
| 标准平衡常数 | $K^\ominus$ | 无量纲（由 $\Delta G^{\ominus}$ 求算） | 热力学计算 |
| 解离常数 | $K_a$, $K_b$ | 酸碱解离 | 酸碱平衡 |
| 溶度积 | $K_{sp}$ | 难溶盐溶解 | 沉淀平衡 |
| 稳定常数 | $K_f$, $\beta$ | 配合物形成 | 配位平衡 |

**经验平衡常数与标准平衡常数的统一**：实验测定的平衡常数只要将实测分压除以 $p^{\ominus} = 1\ \mathrm{bar}$ 用相对压力表示，就可作为无量纲处理。标准平衡常数由 $\Delta G^{\ominus} = -RT\ln K^{\ominus}$ 求算。实际工作中往往不严格区分 $K$ 和 $K^{\ominus}$。

**示例 -- 由 $\Delta G^{\ominus}$ 求 $K^{\ominus}$**：
$\mathrm{NH_3(aq) + H_2O(l) \rightleftharpoons NH_4^+(aq) + OH^-(aq)}$

由附录 $\Delta_{\mathrm{f}}G^{\ominus}$ 数据：$\Delta G^{\ominus}(298\ \mathrm{K}) = +27.1\ \mathrm{kJ \cdot mol^{-1}}$
$$\lg K_c^{\ominus} = -\frac{27.1 \times 10^3}{2.30 \times 8.31 \times 298} = -4.75,\quad K_c^{\ominus} = 1.8 \times 10^{-5}$$

溶液体系中浓度项为相对浓度 $c/c^{\ominus}$（$c^{\ominus} = 1\ \mathrm{mol \cdot dm^{-3}}$），在数值上与 $c$ 相等，$K$ 也呈无量纲。

### 多相平衡
有气体和固体共存的体系称为多相平衡。固相本身是其热力学标准态，不必写入平衡常数式：
- $\mathrm{AgCl(s) \rightleftharpoons Ag^+(aq) + Cl^-(aq)}$，$K_c = [\mathrm{Ag^+}][\mathrm{Cl^-}]$
- $\mathrm{CaCO_3(s) \rightleftharpoons CaO(s) + CO_2(g)}$，$K_p = p(\mathrm{CO_2})$
- 稀溶液中溶剂（如 $\mathrm{H_2O}$）视为常数并入 $K$

### 多重平衡与耦合反应
- 同时平衡体系：联立各平衡方程求解
- **耦合反应**：$\Delta G^\circ_{\text{总}} = \Delta G^\circ_1 + \Delta G^\circ_2$
  - $K_{\text{总}} = K_1 \times K_2$
  - 将一个热力学不利的反应用另一个有利反应"拉动"
- 典型例题：由 $K_1$ 和 $K_2$ 求 $K_3$（见上述 CoO 还原和 SO₂/NO₂ 体系示例）

### 酸碱/沉淀/配位的平衡联动
- 酸效应：降低 pH → 沉淀溶解（如 CaCO₃ + 2H⁺）
- 络合效应：加入配体 → 沉淀溶解（如 AgCl + 2NH₃ → [Ag(NH₃)₂]⁺ + Cl⁻）
- 这些是分析化学和元素化学中最常见的多平衡计算

## 六、适用条件与限制
- ✅ $K$ 在给定温度下是常数（与浓度、压力无关）
- ✅ $K_c$ 和 $K_p$ 的关系：$K_p = K_c(RT)^{\Delta n}$（$\Delta n$ = 气体生成物系数 $-$ 气体反应物系数），推导自 $pV = nRT$ 即 $p = cRT$
- ✅ 标准平衡常数无量纲（浓度除以 $c^{\ominus} = 1\ \mathrm{mol \cdot dm^{-3}}$，压力除以 $p^{\ominus} = 1\ \mathrm{bar}$）
- ✅ $K$ 的形式必须与化学方程式写法一一对应
- ⚠️ 活度系数在浓溶液中 $\neq 1$，用浓度代替活度会引入误差（竞赛通常忽略）
- ⚠️ $K$ 对温度敏感，但温度变化不大时可用 van't Hoff 方程近似；水溶液反应（如酸碱电离、沉淀溶解）在室温范围 $K$ 随温度变化很小
- ⚠️ 纯固体和纯液体的活度为 1，不出现在 $K$ 表达式中
- ⚠️ 稀溶液中水的浓度几乎恒定，不出现在 $K$ 表达式中
- ⚠️ 平衡常数表达式中各浓度项为**平衡浓度**，不是起始浓度
- ⚠️ 多相反应的 $K$ 只含气相和溶液相物种，固相不出现

## 七、常见比较与易混点
| 易混点 | 区分 |
|------|------|
| $K$ vs $Q$ | $K$ 是平衡常数（固定温度下恒定），$Q$ 是反应商（任意时刻） |
| $K$ vs $k$ | $K$ 是热力学量（平衡位置），$k$ 是动力学量（速率常数） |
| $K_c$ vs $K_p$ | $K_c$ 用浓度，$K_p$ 用分压；两者通过 $K_p = K_c(RT)^{\Delta n}$ 互换 |
| $\Delta_r G^\circ$ vs $\Delta_r G$ | $\Delta_r G^\circ$ 是标准态（$Q=1$），$\Delta_r G = \Delta_r G^\circ + RT\ln Q$ |
| 热力学 vs 动力学 | $K$ 大不一定反应快！$\mathrm{H_2 + O_2 \to H_2O}$ ($K$ 极大但室温不反应) |
| $K$ vs 转化率 $\alpha$ | $K$ 是温度的函数，$\alpha$ 还取决于起始浓度。同一 $K$ 可对应不同 $\alpha$（如乙酸乙酯化反应，$K = 4.0$ (373 K)，$\alpha$ 从 42% 到 83%） |
| 经验 $K$ vs $K^{\ominus}$ | 经验 $K$ 由实验测定，$K^{\ominus}$ 由 $\Delta G^{\ominus}$ 求算；用相对压力/浓度可统一为无量纲 |
| 方程式写法 vs $K$ 值 | $\mathrm{N_2O_4 \rightleftharpoons 2NO_2}$ 时 $K = 0.36$；若写为 $\mathrm{2NO_2 \rightleftharpoons N_2O_4}$ 则 $K = 2.8$；数值不同但含义相同 |

## 十、例题
### 例题 1：K 与转化率
**题目：** 反应 N₂O₄(g) ⇌ 2NO₂(g)，某温度下 Kp = 0.14（atm）。若起始 N₂O₄ 为 1.00 atm，求平衡转化率。

**分析：** 设转化率为 α。

**解答：**
N₂O₄ → 2NO₂
始：1.00  0
平：1−α  2α
$$K_p = \frac{(P_{NO_2})^2}{P_{N_2O_4}} = \frac{(2\alpha)^2}{1-\alpha} = 0.14$$
$$\frac{4\alpha^2}{1-\alpha} = 0.14$$
$$4\alpha^2 = 0.14 - 0.14\alpha$$
$$4\alpha^2 + 0.14\alpha - 0.14 = 0$$
$$\alpha = 0.18 \text{（舍去负根）}$$

平衡转化率 = **18%**

**反思：** 气体反应 Kp 计算要注意压力单位一致。转化率低说明平衡偏向左（Kp 小）。

### 例题 2：多重平衡联动
**题目：** 将 AgCl 溶于 NH₃ 水，求 1.0 mol/L NH₃ 中 AgCl 的溶解度。  
已知：$K_{sp}(AgCl)=1.8\times10^{-10}$，$K_f([Ag(NH_3)_2]^+)=1.7\times10^7$

**分析：** 这是沉淀平衡 + 配位平衡的耦合。

**解答：**
反应：AgCl(s) + 2NH₃ ⇌ [Ag(NH₃)₂]⁺ + Cl⁻
$$K = K_{sp} \times K_f = 1.8\times10^{-10} \times 1.7\times10^7 = 3.06\times10^{-3}$$
设溶解度为 s：
$$K = \frac{s^2}{(1.0-2s)^2}$$
$$s = \sqrt{K} \times (1.0-2s) \approx \sqrt{K} = 0.055 \text{ mol/L}$$

**反思：** 氯化银在 1 mol/L 氨水中的溶解度约 0.055 mol/L，远大于纯水中的 $1.3\times10^{-5}$ mol/L。配位显著提高溶解度。

### 例题 3：van't Hoff 方程
**题目：** 反应 CO(g) + H₂O(g) ⇌ CO₂(g) + H₂(g)，ΔH° = −41.2 kJ/mol。已知 500 K 时 K = 126，求 800 K 时的 K。

**分析：** 放热反应 → T↑ → K↓。用 van't Hoff 方程。

**解答：**
$$\ln\frac{K_2}{K_1} = -\frac{\Delta H^\circ}{R}\left(\frac{1}{T_2} - \frac{1}{T_1}\right)$$
$$\ln\frac{K_2}{126} = -\frac{-41200}{8.314}\left(\frac{1}{800} - \frac{1}{500}\right)$$
$$= 4954 \times (-0.00075) = -3.72$$
$$K_2 = 126 \times e^{-3.72} = 126 \times 0.0242 = 3.1$$

**反思：** 800 K 时 K 降至 3.1，平衡显著左移。这与 Le Châtelier 原理一致。

## 十一、易错点
- **❌ 错：** 平衡常数表达式中包含固体或纯液体 → 纯固/液体活度为 1，不出现
- **❌ 错：** 混淆 K 和 Q → K 是平衡状态的属性，Q 是任意时刻的状态函数
- **❌ 错：** 温度变化后仍用原来的 K → 必须用 van't Hoff 方程重新计算
- **❌ 错：** 认为 K 大反应就快 → K 是热力学量，与速率无关
- **❌ 错：** 酸碱平衡计算中混淆 Ka 和 pKa → pKa = −lg Ka；酸越强 pKa 越小
- **❌ 错：** $K_c$ 和 $K_p$ 混用不转换 → 注意用 $K_p = K_c(RT)^{\Delta n}$ 换算

## 十二、🎯 教学视角

### 12.1 学生典型认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 口诀 |
|:---|:---|:---|:---|
| "$K$ 大 = 反应快" | 把平衡位置和速率混淆 | $K$ 是热力学量（平衡时产物/反应物比），和速率（动力学量）无关。$K$ 极大也可能慢到观测不到 | "K管多远，不管多快" |
| "$K_c$ 和 $K_p$ 数值总是一样" | 觉得都是"平衡常数" | $K_p = K_c(RT)^{\Delta n}$，只有当 $\Delta n = 0$（气体分子数不变）时两者才相等 | "Δn非零要换算，Kp和Kc不一样" |
| "$K$ 有单位" | 从量纲分析推导 | 严格热力学定义中 $K$ 是**无量纲**的（每个浓度除以 $c^\ominus = 1\ \mathrm{mol\cdot L^{-1}}$）。竞赛中有时保留单位，但要知道量纲已被"除标准态"消去 | "严格K无量纲，竞赛带单位也行" |
| "平衡常数只取决于温度, 与压力无关 = Kp 也不随压力变" | 把两个"压力"混了 | $K_p$ 只取决于 T，不随**总压**变化。但改变总压会改变**平衡位置**（通过 Le Châtelier 原理），这两个概念不同 | "K值只跟T走，平衡位置看条件" |

### 12.2 入门级例题

**题目**：反应 $\mathrm{N_2O_4(g) \rightleftharpoons 2NO_2(g)}$ 在 298 K 时 $K_c = 4.61 \times 10^{-3}$。
(1) 若初始 $[\mathrm{N_2O_4}] = 0.10\ \mathrm{mol\cdot L^{-1}}$，$[\mathrm{NO_2}] = 0$，求平衡时各物质浓度。
(2) 计算该温度下的 $K_p$。

**预期解答路径**：
1. 设平衡时 $[\mathrm{NO_2}] = 2x$，$[\mathrm{N_2O_4}] = 0.10 - x$
   $K_c = \frac{(2x)^2}{0.10 - x} = 4.61 \times 10^{-3}$ → $4x^2 = 4.61 \times 10^{-4} - 4.61 \times 10^{-3}x$
   解二次方程 → $x \approx 0.0102$（因 $K_c$ 很小，$x \ll 0.10$，近似 $0.10-x \approx 0.10$ 也合理）
   $[\mathrm{N_2O_4}] \approx 0.090\ \mathrm{M}$，$[\mathrm{NO_2}] \approx 0.020\ \mathrm{M}$
2. $\Delta n = 2-1 = 1$，$K_p = K_c(RT)^1 = 4.61 \times 10^{-3} \times 0.0821 \times 298 = 0.113$

**教师引导提问**：如果题目中 $K_c$ 较大（如 $K_c=100$），还能用近似 $0.10-x \approx 0.10$ 吗？（不能！这时必须解完整的二次方程。让学生理解"近似只在 K 很小时有效"的边界。）

### 12.3 与现实/直觉的连接

- **化学平衡 = 商场里的自动扶梯**：上下扶梯上的人数看似不变（宏观平衡），但每时每刻都有人上下（微观仍在反应）。平衡是**动态**的，不是静止的。
- **为什么 $K$ 是温度的"指纹"？** 每个反应在特定温度下只有唯一的 $K$ 值——就像你的指纹。如果你测得的"K"在变，说明体系还没平衡。
- **氮氧化物与城市雾霾**：$\mathrm{N_2O_4 \rightleftharpoons 2NO_2}$ 是光化学烟雾的核心反应之一。夏天高温 → $K$ 增大 → NO₂ 浓度上升 → 棕色烟雾加重。这就是为什么光化学污染在炎热天气更严重——背后是 van't Hoff 方程 $\ln K = -\Delta H/RT + C$。

## 十三、竞赛拓展
- **同时平衡的系统求解**：含多个平衡的溶液体系（如 H₂S + 水溶液中的硫化物分布），需联立质量守恒、电荷守恒和平衡表达式
- **对数浓度图（pC-pH 图）**：在酸碱/配位体系中画各物种浓度对数对 pH 的分布曲线
- **金属离子水解与聚合**：Fe³⁺, Al³⁺ 等高价离子的多核羟基配合物形成是竞赛难点
- **生化标态**：生化中标准态为 ΔG°′（pH=7, [H₂O]=1），与化学标态不同
