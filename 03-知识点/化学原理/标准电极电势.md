# 标准电极电势

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[08-氧化还原与电化学]]

## 一、定义
**标准电极电势 E°**：在标准态（25°C，各物种活度为 1，气体分压 1 bar）下，以**标准氢电极（SHE）**为参比（E° = 0 V），测得的各电极的**还原电势**。

- E° 越正 → 氧化态越容易被还原 → **氧化性越强**
- E° 越负 → 还原态越容易被氧化 → **还原性越强**

## 三、核心原理

### 标准氢电极（SHE）与电极电势的定义
由于无法直接测定单个电极的绝对电势，IUPAC 规定以**标准氢电极（SHE）**为理想参比标准：
- 装置：镀铂黑的铂片插入 $\mathrm{[H^+] = 1\ mol\cdot dm^{-3}}$ 溶液，通入标准压力 $p^{\ominus}$ 的纯净 H₂
- 定义：$E^{\ominus}(\mathrm{H^+/H_2}) = 0.000\ \mathrm{V}$（298K）
- 电极反应：$2\mathrm{H^+ + 2e \rightleftharpoons H_2}$

### 由电池电动势求电极电势
公式：$E_{\text{池}} = E_{\text{正}} - E_{\text{负}}$ (10.1)

教材实例——测定 Zn、Cu、Ag 的标准电极电势：
1. **Zn-H₂ 电池**：$(-)\ \mathrm{Zn|Zn^{2+}\|\|H^+|H_2|Pt}\ (+)$，$E_{\text{池}}^{\ominus} = 0.76\ \mathrm{V}$
   - $E^{\ominus}(\mathrm{Zn^{2+}/Zn}) = E^{\ominus}(\mathrm{H^+/H_2}) - E_{\text{池}}^{\ominus} = 0 - 0.76 = \mathbf{-0.76\ V}$

2. **H₂-Cu 电池**：$(-)\ \mathrm{Pt|H_2|H^+\|\|Cu^{2+}|Cu}\ (+)$，$E_{\text{池}}^{\ominus} = 0.34\ \mathrm{V}$
   - $E^{\ominus}(\mathrm{Cu^{2+}/Cu}) = \mathbf{+0.34\ V}$

3. **H₂-Ag 电池**：$(-)\ \mathrm{Pt|H_2|H^+\|\|Ag^+|Ag}\ (+)$，$E_{\text{池}}^{\ominus} = 0.80\ \mathrm{V}$
   - $E^{\ominus}(\mathrm{Ag^+/Ag}) = \mathbf{+0.80\ V}$

直接测定关系：$E^{\ominus}(\mathrm{Ag/Ag^+}) - E^{\ominus}(\mathrm{Zn/Zn^{2+}}) = 0.80 - (-0.76) = 1.56\ \mathrm{V}$，与分别测定的 Zn-Cu 电池 (1.10V) 和 Cu-Ag 电池 (0.46V) 之和一致（1.10 + 0.46 = 1.56V）。

### E° 与 ΔG° 和 K° 的关系
恒温恒压下：$\Delta G^{\ominus}(T) = -nFE_{\text{池}}^{\ominus}$ (10.2)

- 法拉第恒量：$1F = 9.65 \times 10^4\ \mathrm{C\cdot mol^{-1}}$（1 mol 电子的电量）
- n 为电池反应中的电子转移数

298K 时：$E_{\text{池}}^{\ominus} = \dfrac{0.0591\ \mathrm{V}}{n} \lg K^{\ominus}$ (10.3)

**实例——Zn 置换 Cu²⁺**：
$$E_{\text{池}}^{\ominus} = 0.34 - (-0.76) = 1.10\ \mathrm{V}$$
$$\lg K^{\ominus} = \frac{2 \times 1.10}{0.0591} = 37.2,\quad K^{\ominus} = 2 \times 10^{37}$$

**自发反应判据**：
- $\Delta G^{\ominus} < 0 \iff E_{\text{池}}^{\ominus} > 0$ —— 反应自发
- 常用经验：$E_{\text{池}}^{\ominus} > 0.2\sim 0.4\ \mathrm{V}$ 即认为反应可自发进行（n=1 时 $E_{\text{池}}^{\ominus}=0.36\ \mathrm{V}$ 对应 $K^{\ominus} > 10^6$）

### 标准电极电势表的排列规律
- 表按 $E^{\ominus}$ 由**低到高**（负→正）排列（表10.1）
- $E^{\ominus}$ 为负 → 该电对的标准氢电极组成的电池中为**负极**（容易被氧化）
- $E^{\ominus}$ 为正 → 该电极在标准氢电极组成的电池中为**正极**（容易被还原）
- $E^{\ominus}$ 越**正** → 氧化态物质的氧化性越**强**
- $E^{\ominus}$ 越**负** → 还原态物质的还原性越**强**

**氧化还原反应发生的条件**：$E^{\ominus}$ 较高的氧化态 + $E^{\ominus}$ 较低的还原态 → 自发反应（表中左下方氧化态配右上方还原态）

### 电极电势的间接计算
电极电势是**强度量**，不具有加和性！必须通过 $\Delta G^{\ominus}$ 计算。

**例10.4——求 $E^{\ominus}(\mathrm{ClO_3^-/Cl_2})$**：
- 已知：$\mathrm{ClO_3^- + 6H^+ + 6e \rightleftharpoons Cl^- + 3H_2O}$，$E_1^{\ominus} = +1.45\ \mathrm{V}$
- $\frac{1}{2}\mathrm{Cl_2 + e \rightleftharpoons Cl^-}$，$E_2^{\ominus} = +1.36\ \mathrm{V}$
- 求：$\mathrm{ClO_3^- + 6H^+ + 5e \rightleftharpoons \frac{1}{2}Cl_2 + 3H_2O}$
- **正确算法**：$\Delta G_3^{\ominus} = \Delta G_1^{\ominus} - \Delta G_2^{\ominus} \to -5FE_3^{\ominus} = -6FE_1^{\ominus} - (-1FE_2^{\ominus})$
- $E_3^{\ominus} = \dfrac{6\times 1.45 - 1\times 1.36}{5} = \mathbf{1.47\ V}$
- **错误算法**：直接 $E_1^{\ominus} - E_2^{\ominus} = 0.09\ \mathrm{V}$（错！）

**同一电极反应**不同写法时 $E^{\ominus}$ 相同但 $\Delta G^{\ominus}$ 不同：
| 电极反应 | $E^{\ominus}(\mathrm{Cl_2/Cl^-})$ | $\Delta G^{\ominus}$ / kJ·mol⁻¹ |
|------|------|------|
| $\frac{1}{2}\mathrm{Cl_2 + e \rightleftharpoons Cl^-}$ | +1.36 V | −131 |
| $\mathrm{Cl_2 + 2e \rightleftharpoons 2Cl^-}$ | +1.36 V | −262 |

### 元素电势图（Latimer 图）
同种元素的不同氧化态按氧化态由高到低从左向右排列，连线上的数据为相关电对的 $E^{\ominus}$。

例——Cu 的元素电势图：
$$\mathrm{Cu^{2+}} \xrightarrow{0.15\ \mathrm{V}} \mathrm{Cu^+} \xrightarrow{0.52\ \mathrm{V}} \mathrm{Cu}$$

由已知数据求未知 $E^{\ominus}$：$E^{\ominus}(\mathrm{Cu^{2+}/Cu}) = \dfrac{n_1E_1 + n_2E_2}{n} = \dfrac{0.15\times 1 + 0.52\times 1}{2} = 0.34\ \mathrm{V}$

### 电极电势的应用示例

**例10.1——选择氧化剂**：在含 Cl⁻、Br⁻、I⁻ 的混合溶液中只氧化 I⁻ 为 I₂。查表：
- $E^{\ominus}(\mathrm{I_2/I^-}) = +0.54\ \mathrm{V}$
- $E^{\ominus}(\mathrm{Fe^{3+}/Fe^{2+}}) = +0.77\ \mathrm{V}$
- $E^{\ominus}(\mathrm{Br_2/Br^-}) = +1.07\ \mathrm{V}$
- $E^{\ominus}(\mathrm{Cr_2O_7^{2-}/Cr^{3+}}) = +1.36\ \mathrm{V}$
- $E^{\ominus}(\mathrm{Cl_2/Cl^-}) = +1.36\ \mathrm{V}$
- $E^{\ominus}(\mathrm{H_2O_2/H_2O}) = +1.78\ \mathrm{V}$

结论：**$\mathrm{Fe^{3+}}$** 的 $E^{\ominus} = +0.77\ \mathrm{V}$ 介于 I₂/I⁻ 和 Br₂/Br⁻ 之间，只能氧化 I⁻，不能氧化 Br⁻ 和 Cl⁻。H₂O₂ 和 Cr₂O₇²⁻ 均太强。

**例10.2——判断 O₂ 能否氧化 Fe²⁺ 和 Co²⁺**（标准态酸性溶液）：
- $E^{\ominus}(\mathrm{O_2/H_2O}) = +1.23\ \mathrm{V}$，$E^{\ominus}(\mathrm{Fe^{3+}/Fe^{2+}}) = +0.77\ \mathrm{V}$，$E^{\ominus}(\mathrm{Co^{3+}/Co^{2+}}) = +1.92\ \mathrm{V}$
- $E_{\text{池}}^{\ominus}(\text{Fe}) = 1.23 - 0.77 = +0.46\ \mathrm{V}$ → **能**氧化 Fe²⁺
- $E_{\text{池}}^{\ominus}(\text{Co}) = 1.23 - 1.92 = -0.69\ \mathrm{V}$ → **不能**氧化 Co²⁺

**例10.3——H₂C₂O₄ 标定 KMnO₄**：
- $E^{\ominus}(\mathrm{CO_2/H_2C_2O_4}) = -0.49\ \mathrm{V}$，$E^{\ominus}(\mathrm{MnO_4^-/Mn^{2+}}) = +1.51\ \mathrm{V}$
- $E_{\text{池}}^{\ominus} = 1.51 - (-0.49) = 2.00\ \mathrm{V}$ → 反应彻底，经典分析方法可行

### 氧化还原反应速率问题
有些反应虽 $E_{\text{池}}^{\ominus} > 0$ 但速率极慢，如：
- $(\mathrm{NH_4})_2\mathrm{S_2O_8}$ + $\mathrm{MnSO_4}$（$E_{\text{池}}^{\ominus}=+0.50\ \mathrm{V}$）— 几乎不反应，加 $\mathrm{Ag^+}$ 催化后才生成 $\mathrm{MnO_4^-}$
- $\mathrm{C_2O_4^{2-}}$ + $\mathrm{MnO_4^-}$ 开始慢，生成 $\mathrm{Mn^{2+}}$ 后自催化加速

**因此需从速率和平衡两方面结合考虑氧化还原反应的可行性。**

### 金属活动性顺序
将金属电极反应按 $E^{\ominus}$ 顺序排列即得金属活动性顺序：
$$\mathrm{Li > K > Na > Mg > Al > Zn > Fe > H > Cu > Ag}$$

## 五、常见分类或情形

### 按电极类型分类
| 电极类型 | 构成 | 实例 |
|------|------|------|
| 金属-金属离子电极 | M\|Mⁿ⁺ | Zn\|Zn²⁺, Cu\|Cu²⁺ |
| 气体电极 | 惰性金属\|气体\|离子 | Pt\|H₂\|H⁺, Pt\|Cl₂\|Cl⁻ |
| 氧化还原电极 | 惰性金属\|Ox, Red | Pt\|Fe³⁺, Fe²⁺ |
| 金属-难溶盐电极 | M\|MX\|X⁻ | Ag\|AgCl\|Cl⁻ |
| 参比电极 | 固定组成的半电池 | 甘汞电极（SCE, +0.244V） |

### 按 $E^{\ominus}$ 的正负和应用
| $E^{\ominus}$ 范围 | 典型物质 | 化学性质 |
|------|------|------|
| < −2 V | Li, K, Na, Ca | 极强还原剂（极易被氧化） |
| −2 to 0 V | Zn, Fe, H₂ | 中等还原剂 |
| 0 to +1 V | Cu, I⁻, Ag | 弱还原剂/弱氧化剂 |
| +1 to +2 V | Br₂, MnO₄⁻, Cr₂O₇²⁻ | 强氧化剂 |
| > +2 V | F₂ | 极强氧化剂 |

### 酸性表 vs 碱性表
- 标准电极电势表分为**酸性表**（附录C.5，[H⁺]=1 mol·dm⁻³）和**碱性表**（附录C.6，[OH⁻]=1 mol·dm⁻³）
- 对于含 H⁺ 或 OH⁻ 参与的电极反应，在两表中的 $E^{\ominus}$ 值不同
- 酸度不仅影响电极电势的大小，还影响**还原产物**（如 MnO₄⁻ 在酸性→Mn²⁺，中性→MnO₂，强碱→MnO₄²⁻）

## 六、适用条件与限制

- $E^{\ominus}$ 数据适用于 **298K**、各物质浓度（活度）为 $1\ \mathrm{mol\cdot dm^{-3}}$、气体分压为标准压力 $p^{\ominus}$（100 kPa）
- $E^{\ominus}$ 为**热力学**数据 → 可判断反应**可能性**，不能说一定发生（反应速率可能极慢）
- 浓度变化会使电极电势偏离 $E^{\ominus}$ → 需用 [[Nernst方程]] 校正
- 电极电势表给出的是水溶液中的数据，不适用于非水溶剂
- 同一电极反应的不同写法 $E^{\ominus}$ 值**相同**，但 $\Delta G^{\ominus}$ 不同（因 n 不同）
- 间接计算 $E^{\ominus}$ 时，必须通过 $\Delta G^{\ominus} = -nFE^{\ominus}$，**不能直接加减 $E^{\ominus}$**

## 七、常见比较与易混点

| 易混点 | 区分 |
|------|------|
| $E^{\ominus}$ 的正负意义 | 相对于 SHE 的电极电势高低；正 = 比 SHE 更易还原，负 = 比 SHE 更易氧化 |
| $E^{\ominus}$ vs $E$ | $E^{\ominus}$ 是标准态；$E$ 是任意浓度下的电极电势（含 Nernst 方程修正） |
| 氧化性/还原性强弱 | $E^{\ominus}$ 越正 → **氧化态**氧化性越强；$E^{\ominus}$ 越负 → **还原态**还原性越强 |
| 判断反应方向 | 比较氧化剂所在的电对 $E^{\ominus}$ 和还原剂所在的电对 $E^{\ominus}$：$E^{\ominus}_{\text{氧化剂}} > E^{\ominus}_{\text{还原剂}}$ → 自发 |
| 电极电势的加减 | **不可直接加减**！必须乘以 n（电子数）通过 $\Delta G^{\ominus} = -nFE^{\ominus}$ 转换后再操作 |
| 电池电动势计算 | $E_{\text{池}}^{\ominus} = E_{\text{正}}^{\ominus} - E_{\text{负}}^{\ominus}$ 是**正确的**，因为 n 相同可相消 |
| 氧化态 vs 还原态写法 | 标准写法先写氧化态再写还原态：如 $E^{\ominus}(\mathrm{H^+/H_2})$，但电极反应统一为还原半反应 |
| 酸性表 vs 碱性表 | 同一电对的数值可能不同，如 $E^{\ominus}(\mathrm{O_2/H_2O}) = +1.23\ \mathrm{V}$(酸) vs $+0.40\ \mathrm{V}$(碱) |

## 十一、易错点

- 不能直接加减不同半反应的 $E^\ominus$，必须先转成 $\Delta G^\ominus$
- 把 $E^\ominus$ 当作动力学快慢指标
- 混淆标准电极电势与非标准条件下的实际电极电势

## 十二、🎯 教学视角

### 12.1 学生典型认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 口诀 |
|:---|:---|:---|:---|
| "$E^\ominus$ 越大反应越快" | 把热力学电势和动力学速率混淆 | $E^\ominus$ 是热力学量（趋势），不是动力学量（速率）！$E^\ominus$ 大 = 反应趋势大 ≠ 反应快 | "电势看趋势，不看快慢" |
| "电极电势可以直接代数相加" | 把 $E^\ominus$ 当状态函数 | $E^\ominus$ 不是状态函数！必须通过 $\Delta G^\ominus = -nFE^\ominus$ 转换后才能加和。$\mathrm{Fe^{3+}+e^- \to Fe^{2+}}$ 和 $\mathrm{Fe^{2+}+2e^- \to Fe}$ 的 $E^\ominus$ 不能直接平均 | "电势不能加，要先转ΔG" |
| "标准氢电极在日常实验中实际使用" | 课本上的 SHE 示意图造成的错觉 | SHE（$2\mathrm{H^+}+2e^- \rightleftharpoons \mathrm{H_2}$）操作繁琐（需 H₂ 气流 + 铂黑电极），实际实验室使用**甘汞电极**或**Ag/AgCl 电极**作为参比 | "SHE是理论基准，实验用甘汞" |
| "金属活动性顺序 = $E^\ominus$ 从小到大的顺序" | 高中化学的简化近似 | 基本一致但有不符之处！如 Na(−2.71V) 确实比 Ca(−2.87V) 更正，但 Ca 在水中反应不如 Na 剧烈——因为 Ca(OH)₂ 微溶覆盖表面阻碍反应。电势≠实际活泼性 | "电势排趋势，活泼看实际" |

### 12.2 入门级例题

**题目**：已知 $E^\ominus(\mathrm{Zn^{2+}/Zn}) = -0.76\ \mathrm{V}$，$E^\ominus(\mathrm{Cu^{2+}/Cu}) = +0.34\ \mathrm{V}$。
(1) 计算 $\mathrm{Zn + Cu^{2+} \to Zn^{2+} + Cu}$ 的标准电池电动势 $E^\ominus_{\text{cell}}$。
(2) 判断反应在标准态下是否自发。

**预期解答路径**：
1. $E^\ominus_{\text{cell}} = E^\ominus_{\text{正}} - E^\ominus_{\text{负}} = E^\ominus(\mathrm{Cu^{2+}/Cu}) - E^\ominus(\mathrm{Zn^{2+}/Zn}) = 0.34 - (-0.76) = 1.10\ \mathrm{V}$
2. $E^\ominus_{\text{cell}} > 0$ → $\Delta G^\ominus = -nFE^\ominus_{\text{cell}} < 0$ → 反应**自发**

**教师引导提问**：如果题目给的是 $E^\ominus(\mathrm{Zn^{2+}/Zn}) = -0.76$ 和 $E^\ominus(\mathrm{Cu^{2+}/Cu}) = +0.34$，"Zn 还原 Cu²⁺"的正极是谁？（Cu²⁺ 被还原 → Cu²⁺/Cu 是正极（得电子）。"电势高的得电子"——让学生理解正负极的本质是电子流动方向）

### 12.3 与现实/直觉的连接

- **为什么不能用铁桶装 CuSO₄ 溶液？** $E^\ominus(\mathrm{Fe^{2+}/Fe}) = -0.44 < E^\ominus(\mathrm{Cu^{2+}/Cu}) = +0.34$ → Fe 能还原 Cu²⁺ → 铁桶会被"腐蚀出一个铜洞"。电极电势直接指导了化工储运的材料选择。
- **锂电池为什么电压高？** Li⁺/Li 的 $E^\ominus = -3.04\ \mathrm{V}$（极负）→ 和正极材料配对能产生 3-4V 的高电压。元素周期表左上角的碱金属天然是负极材料的候选——这是原子结构→电化学的跨章连接。
- **标准氢电极为什么是"零"？** 不是测量出来的，而是**人为定义**的——就像定义了海平面为 0 米。所有其他电极的电势都是相对于这个"海平面"的高度差。
