# 滴定分析基本原理

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[18-容量分析]]

---

## 一、定义

**滴定分析（Titrimetric Analysis）**：将已知准确浓度的标准溶液（滴定剂）通过滴定管逐滴加入到被测物质的溶液中，直到所加试剂与被测物质按化学计量关系完全反应为止，根据滴定剂的消耗量计算被测物质含量的分析方法。

---

## 二、基本术语

| 术语 | 定义 |
|:---|:---|
| **标准溶液** | 已知准确浓度的试剂溶液 |
| **滴定** | 将标准溶液逐滴加入被测溶液的操作过程 |
| **化学计量点** | 滴入的标准溶液与被测物质恰好完全反应的理观点 |
| **滴定终点** | 指示剂变色停止滴定的实际操作点 |
| **终点误差** | 滴定终点与化学计量点不一致引起的误差 |
| **滴定突跃** | 化学计量点附近 pH/电位等的急剧变化 |

---

## 三、滴定分析对反应的要求

滴定分析对化学反应有严格要求，需同时满足以下五个条件：

1. **明确的计量关系**：反应按确定的化学方程式进行，无副反应
2. **定量完成**：反应进行程度 $\geqslant 99.9\%$
3. **反应速率快**：滴定反应应瞬时完成（或可通过加热、催化剂加速）
4. **有合适的确定终点的方法**：如指示剂、电位法等
5. **选择性高**：共存物质不干扰测定或可被掩蔽

### 基准物质与标准溶液

**基准物质**是滴定分析的"根"，需满足五项要求（来源：周坤无机新课 Batch 5）：
- 组成确定（与化学式完全一致）
- 纯度高（$\geqslant 99.9\%$）
- 稳定（不吸湿、不分解、不氧化）
- 定量反应（无副反应）
- 摩尔质量大（减小称量相对误差）

**常用基准物质分类**（来源：周坤无机新课 Batch 5）：

| 标定对象 | 基准物质 |
|:---|:---|
| 酸 | $\mathrm{Na_2CO_3}$、硼砂 ($\mathrm{Na_2B_4O_7\cdot 10H_2O}$) |
| 碱 | KHP（邻苯二甲酸氢钾）、草酸 ($\mathrm{H_2C_2O_4\cdot 2H_2O}$) |
| EDTA | 纯金属（Zn、Cu、Pb）或其氧化物 |
| $\mathrm{KMnO_4}$ | 草酸钠 ($\mathrm{Na_2C_2O_4}$) |
| $\mathrm{I_2}$ / $\mathrm{Na_2S_2O_3}$ | $\mathrm{As_2O_3}$、$\mathrm{K_2Cr_2O_7}$ |
| $\mathrm{AgNO_3}$ | NaCl、KCl |

**滴定管读数误差**（来源：周坤无机新课 Batch 5）：常量滴定管每次读数误差约 $\pm 0.02\ \mathrm{mL}$（初读+终读），故滴定体积宜 $> 20\ \mathrm{mL}$，以控制相对误差在 $\pm 0.1\%$ 以内。

### 滴定方式四分法

| 方式 | 定义 | 适用场景 |
|------|------|---------|
| **直接滴定** | 标准溶液直接滴定被测物 | 满足上述五条件的反应 |
| **返滴定** | 先加过量标准溶液，再用另一种标准溶液回滴剩余量 | 反应慢、无合适指示剂 |
| **置换滴定** | 被测物与试剂反应生成可滴定物质 | 无明确定量反应 |
| **间接滴定** | 通过其他反应间接测定 | 非氧化还原物质（如 Ca²⁺ 通过 CaC₂O₄ 间接测定）|

---

## 四、四大滴定方法

| 方法 | 反应类型 | 典型应用 |
|:---|:---|:---|
| **酸碱滴定** | 质子转移反应 | 酸碱含量测定 |
| **氧化还原滴定** | 电子转移反应 | Fe²⁺、COD 等测定 |
| **配位滴定** | 配位反应 | 水的硬度、金属离子含量 |
| **沉淀滴定** | 沉淀反应 | Ag⁺、Cl⁻ 含量测定 |

## 五、核心原理（源自教材提炼）

### 5.1 溶液浓度与滴定计算基础

根据 [[提炼-普化原理-第4章-溶液]]，溶液浓度的准确表示是滴定分析的前提：

- **物质的量浓度** $c = n/V$（单位 $\mathrm{mol\cdot dm^{-3}}$）是滴定计算的核心
- 滴定分析中常用的浓度表示还包括**质量摩尔浓度** $m$（单位 $\mathrm{mol\cdot kg^{-1}}$），其优点是不随温度变化
- 滴定计算的基本关系：$n_{\text{被测物}} : n_{\text{滴定剂}} = \nu_{\text{被测物}} : \nu_{\text{滴定剂}}$（化学计量比）

### 5.2 酸碱滴定的突跃与指示剂选择

根据 [[提炼-普化原理-第8章-酸碱平衡]]：

- **滴定突跃的本质**：化学计量点附近 pH 的急剧变化源于缓冲容量的突变
- 强酸强碱滴定突跃范围约 4–10 pH 单位；弱酸弱碱滴定突跃明显变窄
- 指示剂的变色范围应**落在突跃范围内**，且尽量靠近化学计量点 pH
- 酸碱质子理论（Brønsted-Lowry）是理解滴定反应本质的基础——质子从酸转移到碱

### 5.3 沉淀滴定与溶度积控制

根据 [[提炼-普化原理-第9章-沉淀溶解平衡]]：

- 沉淀滴定（如 Mohr 法、Volhard 法）利用溶度积 $K_{\mathrm{sp}}$ 控制终点
- **分步沉淀原理**：$K_{\mathrm{sp}}$ 小的沉淀先析出，可用于选择性滴定（如 $\mathrm{Cl^-}$ 与 $\mathrm{I^-}$ 的分离滴定）
- 同离子效应可降低沉淀溶解度，提高滴定完全度

### 5.4 氧化还原滴定的电势控制

根据 [[提炼-普化原理-第10章-氧化还原电化学]]：

- 氧化还原滴定突跃由**两个电对的条件电极电势差**决定
- Nernst 方程是计算滴定过程中电势变化的基础：$E = E^\ominus + \frac{RT}{nF}\ln\frac{[\text{氧化态}]}{[\text{还原态}]}$
- 指示剂选择：氧化还原指示剂的变色电势应落在滴定突跃范围内

## 六、竞赛常见考点

1. **滴定计算**：利用 $c_1V_1 = c_2V_2$ 或化学计量关系计算
2. **指示剂选择**：根据滴定突跃范围选择合适的指示剂
3. **终点误差分析**：判断终点提前或滞后对结果的影响
4. **滴定突跃的影响因素**：浓度、平衡常数对突跃范围的影响

## 十、例题

### 例题 1：硼砂标定 HCl（来源：周坤无机新课 Batch 5）

**题目**：称取硼砂 $\mathrm{Na_2B_4O_7\cdot 10H_2O}$ $4.7671\ \mathrm{g}$，溶解后定容至 $250.0\ \mathrm{mL}$。移取 $25.00\ \mathrm{mL}$，用 HCl 标准溶液滴定，消耗 $25.56\ \mathrm{mL}$。求 $c(\mathrm{HCl})$。

**分析**：硼砂与 HCl 的反应计量比为 $1:2$（$\mathrm{B_4O_7^{2-} + 2H^+ + 5H_2O = 4H_3BO_3}$）。

**解答**：
$$n(\text{硼砂}) = \frac{4.7671}{381.37} = 0.01250\ \mathrm{mol}$$
$$c(\text{硼砂}) = \frac{0.01250}{0.2500} = 0.05000\ \mathrm{mol\cdot L^{-1}}$$
$$n(\mathrm{HCl}) = 2 \times 0.05000 \times 0.02500 = 0.002500\ \mathrm{mol}$$
$$c(\mathrm{HCl}) = \frac{0.002500}{0.02556} = 0.09781\ \mathrm{mol\cdot L^{-1}}$$

**反思**：讲义给出的 $0.0978\ \mathrm{mol\cdot L^{-1}}$ 正确（保留 4 位有效数字）。此题同时训练了定容-移取-滴定的多步计量关系。
