# 离子键

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[10-分子结构与化学键]]

## 一、定义
**离子键**：由**正负离子之间的静电引力**形成的化学键。通常形成于电负性差很大的原子之间（金属 + 非金属）。

- 无方向性、无饱和性（与共价键本质区别）
- 离子键强度用**晶格能**衡量

## 三、核心原理
- Born-Haber 循环：通过热力学循环计算晶格能
- 离子键的离子性程度：$\Delta\chi$ 越大，离子性越强
- 纯粹的离子键不存在（Fajans 规则：阳离子极化能力越强，共价性越大）

## 四、关键结论
### 影响离子键强度的因素
1. 离子电荷越大 → 键越强
2. 离子半径越小 → 键越强
3. 晶格类型

### Fajans 规则（离子极化）
阳离子极化能力↑（小半径、高电荷、非惰气型电子构型）→ 共价性↑：
- 实例：AgCl（难溶，共价性强于 NaCl）vs NaCl（易溶）

## 五、常见分类或情形

### 按离子电荷分类
| 离子电荷 | 实例 | 特点 |
|------|------|------|
| 单价离子 | Na$^+$, K$^+$, Cl$^-$, F$^-$ | 晶格能较小（如 NaCl: 787 kJ/mol），熔沸点相对较低 |
| 二价离子 | Mg$^{2+}$, Ca$^{2+}$, O$^{2-}$, S$^{2-}$ | 晶格能显著增大（如 MgO: 3791 kJ/mol），高熔点、高硬度 |
| 三价离子 | Al$^{3+}$, Fe$^{3+}$ | 晶格能极大，但共价成分增加（极化效应） |

### 按离子电子构型分类
| 构型 | 实例 | 特征 |
|------|------|------|
| 2电子构型 | Li$^+$, Be$^{2+}$ | 半径小，极化能力强 |
| 8电子构型（惰气型）| Na$^+$, K$^+$, Ca$^{2+}$, F$^-$, O$^{2-}$ | 最典型离子键 |
| 18电子构型 | Ag$^+$, Zn$^{2+}$, Hg$^{2+}$ | 极化能力强，共价性增强 |
| (18+2)电子构型 | Pb$^{2+}$, Sn$^{2+}$ | 具有孤对电子 |
| 9~17电子构型 | Fe$^{2+}$, Fe$^{3+}$, Cr$^{3+}$, Mn$^{2+}$ | 最外层不饱和，有颜色、磁性 |

### 典型离子晶体结构类型
| 结构类型 | 配位数 | 实例 | 晶格能 U (kJ/mol) |
|------|:---:|------|:---:|
| NaCl型 | 6:6 | NaF, MgO, CaO | NaF: 923, MgO: 3791 |
| CsCl型 | 8:8 | CsCl, CsBr, CsI | — |

## 六、适用条件与限制

1. **适用条件**：电负性差大的金属与非金属之间（$\Delta\chi$ 通常 $> 1.7$）。典型存在于碱金属、碱土金属与卤素、氧族元素的化合物中。
2. **理论限制**：纯粹的离子键不存在，即使是 CsF 也有约 8% 的共价成分。Fajans 规则指出：阳离子半径越小、电荷越高、电子构型越非惰气型，离子键中共价成分越大。
3. **Kossel 理论的局限**（1916）：能说明离子化合物的形成和特性，但不能说明相同原子如何形成单质分子（如 O$_2$, N$_2$），也不能说明电负性相近的元素如何形成化合物分子（如 H$_2$O, NH$_3$）。
4. 离子化合物在通常条件下是**晶体**，无法从晶体中划分出孤立的单个分子。NaCl 仅是化学式（表示 Na$^+$:Cl$^-$ = 1:1），而非分子式。

## 七、常见比较与易混点

| 对比项 | 离子键 | 共价键 |
|------|------|------|
| 成键本质 | 正负离子间静电引力 | 原子间共用电子对（轨道重叠） |
| 方向性 | 无 | 有 |
| 饱和性 | 无 | 有 |
| 强度衡量 | 晶格能 U | 键能（解离能） |
| 存在形式 | 离子晶体（巨大分子） | 共价分子或原子晶体 |
| 典型实例 | NaCl, MgO | H$_2$O, CH$_4$, O$_2$ |

| 对比项 | NaCl型 | CsCl型 |
|------|------|------|
| 配位数 | 6:6 | 8:8 |
| 决定因素 | 正负离子半径比 $r_+/r_-$ | 同左 |

### 离子半径变化规律
- 同族：$r$(Li$^+$) < $r$(Na$^+$) < $r$(K$^+$) < $r$(Rb$^+$) < $r$(Cs$^+$)
- 同一周期：$r$(Na$^+$) > $r$(Mg$^{2+}$) > $r$(Al$^{3+}$)
- 同元素高价态半径更小：$r$(Fe$^{2+}$) > $r$(Fe$^{3+}$)
- 配位数越大，离子半径越大

## 十一、易错点
- 离子键无方向性和饱和性，但晶体有确定配位数（由半径比决定，非价电子数决定）
- NaCl 不是分子式，是化学式
- 晶格能是绝对值（正值），$-\Delta H = U$

### 12.1 学生典型认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 口诀 |
|:---|:---|:---|:---|
| "离子键就是Na⁺和Cl⁻之间的静电引力" | 教科书从NaCl讲离子键的路径依赖 | 离子键本质是**球形电荷分布**间的Coulomb引力，无方向性和饱和性——这就是为什么离子晶体是紧密堆积而非"分子"。NaCl中每个Na⁺被6个Cl⁻包围，不是1个 | "离子键无方向，球吸球不是点对点" |
| "离子键和共价键是泾渭分明的两种键" | 教科书的分类教学导致的二分思维 | 纯离子键和纯共价键是**两个极端**，多数键介于两者之间。如LiI中离子性成分约65%，HI中约5%。Fajans规则：阳离子极化力越强→共价成分越高 | "纯离子纯共价是理想，实际都是混血儿" |
| "晶格能越大，溶解度越小" | 以为晶格能大="结合紧"=难溶 | 溶解过程取决于晶格能和水合能的**差值**。NaCl晶格能大（788 kJ/mol），但水合能也大 → 可溶；AgCl晶格能略小（905 kJ/mol），但水合能不够 → 难溶 | "能否溶解看差值，不是晶格能一家说了算" |
| "离子键没有方向性=离子排列是随机的" | 把"无方向性"误解为"无规律" | 无方向性意味着**球对称**电场→配位数由离子半径比（r⁺/r⁻）决定→CsCl(8配位)/NaCl(6)/ZnS(4)。排列极其规律，不是随机的 | "无方向不是无规律，半径比定配位数" |

### 12.2 入门级例题

**题目**：NaCl 易溶于水，AgCl 难溶于水。两者的晶格能分别为 788 和 905 kJ/mol，试用热力学循环解释溶解性差异。

**预期解答路径**：
1. 溶解 ΔH = 晶格能（吸热，正值）+ 水合能（放热，负值）
2. NaCl：晶格能 788 − 水合能(Na⁺+Cl⁻)≈784 ≈ ΔH≈+4 kJ/mol（近零）→ ΔS>0驱动溶解
3. AgCl：晶格能 905 − 水合能(Ag⁺+Cl⁻)≈850 ≈ ΔH≈+55 kJ/mol（吸热显著）
4. Ag⁺ 极化力强 → AgCl 共价成分大 → 水合能不足以补偿晶格能 → 难溶

**教师引导提问**：AgF 易溶而 AgCl/AgBr/AgI 越来越难溶，为什么？（F⁻ 难极化→AgF离子性强→水合能大→易溶。Cl⁻→Br⁻→I⁻ 体积递增→可极化性递增→共价成分递增→水合能不足以补偿→更难溶。Fajans规则的经典序列）

### 12.3 与现实/直觉的连接

- **盐溶于水的微观画面**：食盐投入水中，H₂O 分子以 O 朝向 Na⁺、H 朝向 Cl⁻ 的方式包围并"剥离"离子——水合能补偿晶格能的过程。溶解不是"盐消失了"，是"离子被水分子抢走了"。
- **Fajans规则——离子键中的"变色龙"**：小阳离子+大阴离子 → 阳离子极化力强 → 把阴离子的电子云拉偏 → 共价成分增大。如 LiI 的"离子性"远不如 CsF——同是"离子化合物"，共价程度天差地别。
- **为什么敲NaCl会碎？** 离子晶体受外力→同号离子滑到对齐→排斥→碎裂。而金属受外力→原子层滑动→电子海重新粘合→变形不断。键的类型直接决定了材料的力学行为——这是从微观到宏观的完美连接。
