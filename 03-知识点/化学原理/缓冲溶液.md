# 缓冲溶液

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[05-酸碱理论]]

## 一、定义
**缓冲溶液**：能抵抗少量外加酸、碱或适当稀释而保持 pH 基本不变的溶液。通常由**弱酸及其共轭碱**（或弱碱及其共轭酸）组成。

## 三、核心原理

### 缓冲作用机制

缓冲溶液抵抗 pH 变化的本质：**共轭酸碱对共存**，少量强酸/强碱被"吸收"转化为弱酸/弱碱，pH 变化小。

- 外加 H⁺ → 被共轭碱 A⁻ 消耗 → $\ce{A- + H+ -> HA}$
- 外加 OH⁻ → 被弱酸 HA 消耗 → $\ce{HA + OH- -> A- + H2O}$
- 稀释 → 共轭酸碱对浓度等比例下降，比值不变 → pH 不变

### Henderson-Hasselbalch 方程
$$\mathrm{pH} = \mathrm{p}K_a + \lg\frac{[\mathrm{A}^-]}{[\mathrm{HA}]}$$

## 四、关键结论

### 缓冲容量的影响因素
1. **总浓度越大 → 缓冲容量越大**
2. **$[\mathrm{A}^-]/[\mathrm{HA}]$ 越接近 1 → 缓冲容量越大**
3. **有效缓冲范围**：$\mathrm{pH} = \mathrm{p}K_a \pm 1$
4. **最大缓冲容量位置**：$\beta_\mathrm{max}$ 在 $\mathrm{pH} = \mathrm{p}K_a$ 处（此时 $[\mathrm{A}^-] = [\mathrm{HA}]$，缓冲能力最强）

### 缓冲溶液配制步骤
1. 选择 $\mathrm{p}K_a$ 接近目标 pH 的弱酸
2. 按 H-H 方程计算共轭酸碱比
3. 确定总浓度
4. 计算各组分的量并配制

### 常见缓冲体系
| 缓冲体系 | pKa | 缓冲范围 |
|------|:---:|------|
| HAc—NaAc | 4.76 | 3.8–5.8 |
| NH₃—NH₄Cl | 9.25 (pKa of NH₄⁺) | 8.3–10.3 |
| NaH₂PO₄—Na₂HPO₄ | 7.20 (pKa₂) | 6.2–8.2 |
| NaHCO₃—Na₂CO₃ | 10.33 (pKa₂) | 9.3–11.3 |

## 五、常见分类或情形

### 1. 弱酸-共轭碱型缓冲溶液
由弱酸及其共轭碱（弱酸盐）组成，最常见类型。

**定量关系**（教材式 8.13）：
$$[\ce{H3O+}] = K_a \cdot \frac{c(\text{弱酸})}{c(\text{共轭碱})}$$
$$\mathrm{pH} = \mathrm{p}K_a + \lg\frac{c(\text{共轭碱})}{c(\text{弱酸})}$$

**教材实验数据**（教材 I、II、III 的对比实验）：
| 实验 | 操作 | pH（测定） | pH（计算） |
|------|------|------|------|
| I | $5.0\ \mathrm{cm^3}\ 0.20\ \mathrm{mol\cdot dm^{-3}}\ \ce{HAc} + 9.0\ \mathrm{cm^3}\ 0.20\ \mathrm{mol\cdot dm^{-3}}\ \ce{NaAc}$ 混合 | 4.84 | 5.01 |
| II | 上述混合液 + 2滴 $1.0\ \mathrm{mol\cdot dm^{-3}}\ \ce{HCl}$ | 4.79 | 4.96 |
| II | 上述混合液 + 2滴 $1.0\ \mathrm{mol\cdot dm^{-3}}\ \ce{NaOH}$ | 4.91 | 5.09 |
| III | $10\ \mathrm{cm^3}\ \mathrm{pH}=5.00\ \ce{HCl} + 2$ 滴 $\ce{HCl}$ | 2.07 | 2.00 |
| III | $10\ \mathrm{cm^3}\ \mathrm{pH}=5.00\ \ce{HCl} + 2$ 滴 $\ce{NaOH}$ | 11.81 | 12.00 |

实验 II 中缓冲溶液 pH 仅变化 ~0.05 单位，而实验 III 中无缓冲的 HCl 溶液 pH 变化了 ~3~7 个单位。

### 2. 弱碱-共轭酸型缓冲溶液
由弱碱及其共轭酸（弱碱盐）组成，如 $\ce{NH3\cdot H2O - NH4Cl}$。

**定量关系**（教材式 8.14）：
$$[\ce{OH-}] = K_b \cdot \frac{c(\text{弱碱})}{c(\text{共轭酸})}$$
$$\mathrm{pOH} = \mathrm{p}K_b + \lg\frac{c(\text{共轭酸})}{c(\text{弱碱})}$$

### 3. 两性物质型缓冲溶液
如 $\ce{KH2PO4-Na2HPO4}$（$\mathrm{p}K_{a_2} = 7.21$）、$\ce{NaHCO3-Na2CO3}$（$\mathrm{p}K_{a_2} = 10.33$）。

### 4. 常用缓冲溶液一览（教材表 8.6）
| 缓冲溶液 | 共轭酸碱对 | $\mathrm{p}K_a$ | 缓冲范围 |
|----------|-----------|------|------|
| $\ce{HCOOH-NaOH}$ | $\ce{HCOOH-HCOO-}$ | 3.74 | 2.74~4.74 |
| $\ce{CH3COOH-CH3COONa}$ | $\ce{HAc-Ac-}$ | 4.76 | 3.76~5.76 |
| $\ce{KH2PO4-Na2HPO4}$ | $\ce{H2PO4- - HPO4^{2-}}$ | 7.21 | 6.21~8.21 |
| $\ce{Na2B4O7-HCl}$ | $\ce{H3BO3-B(OH)4-}$ | 9.27 | 8.27~10.27 |
| $\ce{NH3\cdot H2O-NH4Cl}$ | $\ce{NH4+ - NH3}$ | 9.26 | 8.26~10.26 |
| $\ce{NaHCO3-Na2CO3}$ | $\ce{HCO3- - CO3^{2-}}$ | 10.33 | 9.33~11.33 |
| $\ce{Na2HPO4-NaOH}$ | $\ce{HPO4^{2-} - PO4^{3-}}$ | 12.32 | 11.32~13.32 |

### 5. 自然界和生物体中的缓冲体系
- **人体血液**：主要缓冲体系为 $\ce{H2CO3 - HCO3-}$，正常 $\mathrm{pH} = 7.40\pm 0.05$。改变量超过 $0.4\ \mathrm{pH}$ 单位就有生命危险。
- **血液缓冲机制**：
  $$\ce{H+ + HCO3- <=> H2CO3 <=> H2O + CO2(g)}$$
  血液中氧含量上升 → 平衡右移 → 释放 $\ce{CO2}$；$\ce{CO2}$ 浓度上升 → 平衡左移 → 促进血红蛋白放氧。
- **土壤**：硅酸、磷酸、腐殖酸等及其共轭碱的缓冲作用，使 $\mathrm{pH}$ 保持在 $5\sim 8$。

### 6. 缓冲溶液的配制方法
**方法一**：直接使用共轭酸碱对配制（如教材例 8.7）
- 例：配制 $\mathrm{pH} = 9.20$，$c(\ce{NH3\cdot H2O}) = 1.0\ \mathrm{mol\cdot dm^{-3}}$ 的缓冲溶液 $500\ \mathrm{cm^3}$
  - 由 $\mathrm{pOH} = 4.80$，$[\ce{OH-}] = 1.6\times 10^{-5}$
  - $\frac{[\ce{NH3\cdot H2O}]}{[\ce{NH4+}]} = \frac{[\ce{OH-}]}{K_b} = \frac{1.6\times 10^{-5}}{1.8\times 10^{-5}} = 0.89$
  - 需 $\ce{NH4Cl}$：$0.50 \times 1.1 \times 53.5 = 29\ \mathrm{g}$，浓氨水 $33\ \mathrm{cm^3}$，加水至 $500\ \mathrm{cm^3}$

**方法二**：利用酸碱中和反应形成共轭酸碱对（如教材例 8.8）
- 例：用 $50\ \mathrm{cm^3}\ 1.0\ \mathrm{mol\cdot dm^{-3}}\ \ce{NaOH}$ 与适量 $1.0\ \mathrm{mol\cdot dm^{-3}}\ \ce{HAc}$ 混合配制 $\mathrm{pH}=4.70$ 缓冲液 $500\ \mathrm{cm^3}$
  - $[\ce{H3O+}] = 2.0\times 10^{-5}$，$\frac{[\ce{HAc}]}{[\ce{Ac-}]} = 1.1$
  - 需 $\ce{HAc}$：$105\ \mathrm{cm^3}$，加水 $345\ \mathrm{cm^3}$

## 六、适用条件与限制

### 有效缓冲范围
缓冲溶液最有效的缓冲范围：
$$\mathrm{pH} = \mathrm{p}K_a \pm 1, \quad \mathrm{pOH} = \mathrm{p}K_b \pm 1$$
即共轭酸碱对浓度比在 $1/10 \sim 10/1$ 之间。

### 缓冲容量的影响因素
1. **共轭酸碱对总浓度**：总浓度越大，缓冲容量越大。
   - 当共轭酸碱浓度各为 $0.1\ \mathrm{mol\cdot dm^{-3}}$，加少量酸碱时 $\mathrm{pH}$ 改变约 $0.1$ 单位
   - 当浓度各为 $1\ \mathrm{mol\cdot dm^{-3}}$，同样操作 $\mathrm{pH}$ 改变约 $0.01$ 单位
   - 一般建议浓度在 $0.1\sim 1\ \mathrm{mol\cdot dm^{-3}}$ 为宜

2. **共轭酸碱浓度比**：比值越接近 $1:1$，缓冲能力越大。
   - 当 $c(\text{共轭碱})/c(\text{弱酸}) = 1$ 时，加 $0.01\ \mathrm{mol\ H_3O^+}$（每 $\mathrm{dm^3}$），$\mathrm{pH}$ 改变仅 $0.01$ 单位
   - 当 $c(\text{共轭碱})/c(\text{弱酸}) = 1/99$ 时，同样操作 $\mathrm{pH}$ 改变 $0.3$ 单位（$[\ce{H3O+}]$ 改变约 2 倍）

### 适用条件
- Henderson-Hasselbalch 方程成立的前提：共轭酸碱浓度远大于 $[\ce{H3O+}]$ 和 $[\ce{OH-}]$（即缓冲组分浓度不太低）
- 缓冲溶液不能抵抗**大量**外加酸或碱——缓冲容量有限
- 稀释时，共轭酸碱对浓度等比例下降，比值基本不变，故 $\mathrm{pH}$ 近似不变（但过度稀释可能导致 $[\ce{H3O+}]$ 或 $[\ce{OH-}]$ 不可忽略而失效）
- 水的离子积 $K_w$ 应是常数（温度须恒定）

### 配制要点
1. 选择 $\mathrm{p}K_a$（或 $\mathrm{p}K_b$）**等于或接近**目标 $\mathrm{pH}$ 的共轭酸碱对
2. 调节酸碱比值以达到精确 $\mathrm{pH}$
3. 适当提高共轭酸碱对的总浓度（一般 $0.1\sim 1\ \mathrm{mol\cdot dm^{-3}}$）
4. 保持共轭酸碱对浓度比接近 $1:1$ 以获得最大缓冲能力

## 七、常见比较与易混点

### 1. 缓冲溶液 vs 强酸（碱）溶液的 pH 稳定性
| 溶液类型 | 加入少量强酸/强碱后 pH 变化 | 原因 |
|----------|------|------|
| 缓冲溶液（HAc-NaAc） | 变化极小（~0.05 单位） | 共轭碱/酸消耗外加 $\ce{H+}$ 或 $\ce{OH-}$ |
| 强酸溶液（$\mathrm{pH}=5.00\ \ce{HCl}$） | 变化极大（~3~7 单位） | 无缓冲能力，$[\ce{H+}]$ 直接改变 |

参见教材对比实验（I、II vs III）——这是理解缓冲作用最直观的数据。

### 2. 缓冲作用 vs 同离子效应
两者基于同一原理（共轭酸碱共存），但侧重点不同：
- **同离子效应**：外加共同离子抑制弱电解质电离 → 降低电离度
- **缓冲作用**：共轭酸碱对抵抗外加少量酸碱 → 维持 pH 稳定

HAc-NaAc 体系既体现同离子效应（HAc 电离度从 1.3% 降至 0.0018%），又体现缓冲作用（加酸碱 pH 几乎不变）。

### 3. 缓冲范围与指示剂变色范围
两者都涉及 $\mathrm{p}K_a \pm 1$ 的区间，但概念不同：
- **缓冲范围**：共轭酸碱对有效缓冲的 $\mathrm{pH}$ 区间
- **指示剂变色范围**：指示剂颜色转变的 $\mathrm{pH}$ 区间
- 两者的数学基础相同（浓度比 $1/10\sim 10/1$），但应用目的不同

### 4. 酸型缓冲 vs 碱型缓冲
| | 酸型缓冲（弱酸 + 弱酸盐） | 碱型缓冲（弱碱 + 弱碱盐） |
|------|------|------|
| 计算 $\mathrm{pH}$ | $\mathrm{pH} = \mathrm{p}K_a + \lg\frac{[\text{盐}]}{[\text{酸}]}$ | 先用 $\mathrm{pOH} = \mathrm{p}K_b + \lg\frac{[\text{盐}]}{[\text{碱}]}$，再 $\mathrm{pH} = 14 - \mathrm{pOH}$ |
| 实例 | HAc-NaAc，$\mathrm{p}K_a=4.76$ | $\ce{NH3\cdot H2O-NH4Cl}$，$\mathrm{p}K_b=4.75$ |
| 有效 $\mathrm{pH}$ 区间 | 酸性区域 | 碱性区域 |

注意：$\ce{NH4+}$ 的 $\mathrm{p}K_a = 9.25$，因此 $\ce{NH3\cdot H2O-NH4Cl}$ 缓冲范围也可写成 $\mathrm{pH} = 9.25 \pm 1$（用 $\mathrm{p}K_a$ 而非 $\mathrm{p}K_b$ 表示时）。

### 5. 缓冲溶液配制的常见错误
- **选错共轭酸碱对**：用 $\mathrm{p}K_a = 4.76$ 的 HAc-NaAc 配制 $\mathrm{pH} = 9$ 的缓冲液——超出缓冲范围，几乎无效。应选 $\mathrm{p}K_a = 9.26$ 的 $\ce{NH4+ - NH3}$。
- **浓度过低**：共轭酸碱总浓度 << 外加酸碱量 → 缓冲能力不足。
- **忽略温度**：H-H 方程中的 $\mathrm{p}K_a$ 仅在给定温度下为常数。
