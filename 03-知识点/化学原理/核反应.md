# 核反应（Nuclear Reaction）

## 一、定义

核反应涉及原子核的变化，与化学反应（只涉及核外电子）有本质区别。初赛考查的核反应主要包括**放射性衰变**、**核裂变**、**核聚变**以及**人工核转变**。掌握核反应方程式的书写规则和守恒定律是解题关键。

## 三、核心原理

### 3.1 衰变类型汇总

| 衰变类型 | 放出粒子 | 质量数变化 | 电荷数变化 | 例子 |
|:---|:---:|:---:|:---:|:---|
| $\alpha$ 衰变 | $^4_2\mathrm{He}$ | $-4$ | $-2$ | $^{238}_{92}\mathrm{U} \rightarrow {}^{234}_{90}\mathrm{Th} + {}^4_2\mathrm{He}$ |
| $\beta$ 衰变 | $^0_{-1}e$（电子） | $0$ | $+1$ | $^{14}_6\mathrm{C} \rightarrow {}^{14}_7\mathrm{N} + {}^0_{-1}e$ |
| $\gamma$ 衰变 | 光子 | $0$ | $0$ | 激发态 $\rightarrow$ 基态 |
| 正电子发射 | $^0_{+1}e$ | $0$ | $-1$ | $^{11}_6\mathrm{C} \rightarrow {}^{11}_5\mathrm{B} + {}^0_{+1}e$ |
| 电子俘获（EC） | 无 | $0$ | $-1$ | $^{40}_{19}\mathrm{K} + {}^0_{-1}e \rightarrow {}^{40}_{18}\mathrm{Ar}$ |

### 3.2 各种衰变的详细说明

**$\alpha$ 衰变**
- 放出氦核（$^4_2\mathrm{He}$），质量数减4，电荷数减2
- 常见于重核（$Z > 83$）
- $\alpha$ 粒子穿透能力弱，可被一张纸阻挡

**$\beta$ 衰变**
- 核内中子转化为质子：$n \rightarrow p + e^- + \bar{\nu}_e$
- 质量数不变，电荷数加1
- $\beta$ 射线穿透能力中等，可被铝板阻挡

**$\gamma$ 衰变**
- 原子核从激发态回到基态时放出高能光子
- 不改变元素种类，仅释放能量
- 穿透能力最强，需厚铅板阻挡

**正电子发射**
- 核内质子转化为中子：$p \rightarrow n + e^+ + \nu_e$
- 质量数不变，电荷数减1
- 正电子与电子相遇发生湮灭：$e^+ + e^- \rightarrow 2\gamma$

**电子俘获（EC）**
- 核内质子捕获K层电子转化为中子：$p + e^- \rightarrow n + \nu_e$
- 质量数不变，电荷数减1
- 伴随特征X射线发射（外层电子填补K层空位）

## 四、关键结论

### 4.1 四大守恒定律

1. **质量数守恒**：反应前后上标（质量数）之和相同
2. **电荷数守恒**：反应前后下标（电荷数/原子序数）之和相同
3. **能量守恒**：质量亏损转化为能量 $\Delta E = \Delta m c^2$
4. **动量守恒**

### 4.2 书写规范

- 元素符号左上角写**质量数**（A），左下角写**电荷数/原子序数**（Z）
- 电子表示为 $^0_{-1}e$ 或 $^0_{-1}\beta$
- 正电子表示为 $^0_{+1}e$ 或 $^0_{+1}\beta$
- 中子表示为 $^1_0n$，质子表示为 $^1_1p$ 或 $^1_1H$

### 4.3 配平示例

$$^{14}_7\mathrm{N} + {}^4_2\mathrm{He} \rightarrow {}^{17}_8\mathrm{O} + {}^1_1\mathrm{H}$$

验证：
- 质量数：$14 + 4 = 17 + 1 = 18$ ✓
- 电荷数：$7 + 2 = 8 + 1 = 9$ ✓

## 十一、易错点

1. **$\beta$ 衰变中质量数不变**：不要因为放出电子就认为质量数减1
2. **半衰期是统计概念**：对单个核无意义，对大量核才成立
3. **核反应方程式不用配平化学计量数**：只需守恒质量数和电荷数
4. **$\gamma$ 射线不改变元素种类**：只是释放能量，原子核回到基态
5. **正电子发射与电子俘获都使电荷数减1**：但前者放出正电子，后者捕获电子
6. **核反应中质量不守恒**：有质量亏损，转化为能量

## 十三、竞赛拓展

1. **核稳定性与"幻数"——核壳层模型入门**：具有 2, 8, 20, 28, 50, 82, 126 个质子或中子的核特别稳定（类似原子的稀有气体电子构型）。$^{4}_{2}\mathrm{He}$（2p+2n）、$^{16}_{8}\mathrm{O}$（8p+8n）、$^{40}_{20}\mathrm{Ca}$（20p+20n，双幻核）是幻数核。竞赛信息题可能给同位素丰度或稳定性数据，要求用幻数解释——例如 $^{208}_{82}\mathrm{Pb}$（Z=82, N=126 双幻核）是稳定核素链的终点；$^{56}_{26}\mathrm{Fe}$（N=30 近幻数）是恒星核合成的"灰烬"——比铁重的元素不能在普通恒星中通过聚变产生。

2. **放射性衰变的数学——半衰期与测年应用**：一级衰变动力学 $N = N_0 e^{-\lambda t}$，半衰期 $t_{1/2} = \ln 2 / \lambda$。竞赛应用：① $^{14}$C 测年（$t_{1/2} = 5730$ y）——适用于 500-50000 年的有机样品，通过比较样品中 $^{14}$C/$^{12}$C 与大气中的比值计算年代；② $^{238}$U-$^{206}$Pb 测年（$t_{1/2} = 4.47\times10^9$ y）——用于测定地球/陨石年龄。关键假设：体系封闭（无 U/Pb 的后期丢失或增益）。竞赛计算题可能给同位素比值数据要求计算年龄。

3. **核素图（Chart of Nuclides）与稳定谷**：以 N（中子数）为横轴、Z（质子数）为纵轴的核素图，稳定核素集中在"稳定谷"中——轻核 N ≈ Z，重核 N/Z ≈ 1.5。谷外 → 放射性衰变回归谷内：中子过多 → β⁻ 衰变（n→p+e⁻）；质子过多 → β⁺ 衰变或电子俘获（p→n+e⁺）；重核 → α 衰变。竞赛可能给不稳定核素，要求预测衰变模式。
