## 二、内/外轨型配合物
- **内轨型**：使用 (n−1)d 轨道杂化（d²sp³）→ 低自旋，配位键强
- **外轨型**：使用 nd 轨道杂化（sp³d²）→ 高自旋，配位键弱
- 与晶体场理论的高低自旋相互印证

## 三、核心原理

配合物价键理论（VB 理论）认为：中心离子与配位体之间的结合，是由于**配位原子孤对电子轨道与中心离子的空轨道重叠**，两者共享该电子对而形成配位键。中心离子采用杂化轨道来接纳配体的孤对电子。

**外轨型配合物的形成**（以 $\mathrm{FeF_6^{3-}}$ 为例）：
$\mathrm{Fe^{3+}}$ 的 1 个 4s、3 个 4p 和 2 个 4d 空轨道杂化为 6 个能量相等的 $\mathrm{sp^3d^2}$ 杂化轨道，分别与 6 个 $\mathrm{F^-}$ 离子的 p 轨道重叠形成配位键。所有电子对均由 $\mathrm{F^-}$ 提供，内层 3d 电子排布不变（5 个 d 电子分占 5 个轨道）。

**内轨型配合物的形成**（以 $\mathrm{Fe(CN)_6^{3-}}$ 为例）：
当 6 个 $\mathrm{CN^-}$ 接近 $\mathrm{Fe^{3+}}$ 时，$\mathrm{Fe^{3+}}$ 的 5 个价电子挤入 3 个 3d 轨道，其余 2 个 3d 空轨道加上 1 个 4s 和 3 个 4p 轨道组成 6 个 $\mathrm{d^2sp^3}$ 杂化轨道，分别与 6 个 $\mathrm{CN^-}$ 轨道重叠。

**分类依据**：
- 外轨型：使用 $ns\!-\!np\!-\!nd$ 杂化 → $(n-1)d$ 电子排布不变
- 内轨型：使用 $(n-1)\mathrm{d}\!-\!ns\!-\!np$ 杂化 → $(n-1)d$ 电子发生重排

## 四、关键结论

1. **内轨型比外轨型更稳定**：$(n-1)\mathrm{d}$ 轨道能量比 $nd$ 轨道低，内轨型配位键键长更短。

2. **配体对杂化类型的影响**（表 14.8~14.9）：
   - 卤素离子、$\mathrm{H_2O}$ 等 → 外轨型（$sp^3d^2$），配位原子电负性高，不易给出孤对电子
   - $\mathrm{CN^-}$、$\mathrm{NO_2^-}$ 等 → 内轨型（$d^2sp^3$），配位原子电负性低，易给出孤对电子
   - $\mathrm{NH_3}$ → 介于两者之间，随中心离子不同而有 HS 或 LS

3. **磁性检验法**：磁矩 $\mu = \sqrt{n(n+2)}\,\mu_{\mathrm{B}}$。实验测定磁矩可判断未成对电子数 $n$，进而推知内层 d 电子排布和杂化类型：
   - $\mathrm{FeF_6^{3-}}$：$\mu_{\text{实}} = 5.88$ B.M. → $n=5$ → 外轨型 $sp^3d^2$
   - $\mathrm{Fe(CN)_6^{3-}}$：$\mu_{\text{实}} = 2.3$ B.M. → $n \approx 1$ → 内轨型 $d^2sp^3$

4. **历史演变**：价键理论曾分为"电价配键"（对应外轨型/HS）和"共价配键"（对应内轨型/LS），但无绝对界限，直接分高自旋/低自旋更简明。

### 按配体倾向分类
| 配体类型 | 倾向 | 实例 |
|:---|:---|:---|
| 弱场配体 | 外轨型、高自旋 | $\mathrm{F}^-$、$\mathrm{Cl}^-$、$\mathrm{Br}^-$、$\mathrm{I}^-$、$\mathrm{H_2O}$ |
| 中间配体 | 视中心离子而定 | $\mathrm{NH_3}$ |
| 强场配体 | 内轨型、低自旋 | $\mathrm{CN}^-$、$\mathrm{NO_2^-}$、CO |

## 六、适用条件与限制

- **适用**：解释配合物的几何构型、配位数
- **适用**：通过磁矩区分外轨型（HS）和内轨型（LS）
- **适用**：解释内轨型比外轨型更稳定的一般趋势
- **局限**：无法解释配合物的紫外-可见吸收光谱和颜色
- **局限**：无法定量说明高/低自旋产生的原因
- **局限**：存在反例——$\mathrm{Cu(NH_3)_4^{2+}}$（平面正方形 $dsp^2$，有 1 个未成对电子在较高能级，但很稳定，不具有还原性），与价键理论的预测矛盾
- **局限**：无法解释配合物的热力学性质（如 CFSE）

## 七、常见比较与易混点

| 对比项 | 外轨型 ($sp^3d^2$) | 内轨型 ($d^2sp^3$) |
|:---|:---|:---|
| 所用 d 轨道 | 外层 $nd$ | 内层 $(n-1)d$ |
| 内层 d 电子 | 排布不变 | 重排（电子挤入少数轨道） |
| 未成对电子 | 多（与自由离子相同） | 少 |
| 自旋态 | 高自旋 | 低自旋 |
| 键长 | 较长 | 较短 |
| 稳定性 | 较低 | 较高 |
| 常见配体 | $\mathrm{F}^-$、$\mathrm{H_2O}$ | $\mathrm{CN}^-$、$\mathrm{NO_2}^-$ |

| 易混点 | 区分 |
|:---|:---|
| $sp^3$ vs $dsp^2$ | 均为 4 配位，前者为四面体（外轨），后者为平面正方形（内轨）；$d^8$ 中心离子强场下倾向于 $dsp^2$ |
| $sp^3d^2$ vs $d^2sp^3$ | 仅 d 的位置不同，但含义完全不同：外轨（HS）vs 内轨（LS） |
| $\mathrm{Co(NH_3)_6^{2+}}$ vs $\mathrm{Co(NH_3)_6^{3+}}$ | 前者 HS（$sp^3d^2$，磁矩 5.04），后者 LS（$d^2sp^3$，磁矩 0）——中心离子价态影响 |

### 12.1 学生典型认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 口诀 |
|:---|:---|:---|:---|
| "配合物的杂化和普通分子的杂化一样" | 前面学了VSEPR+杂化，直接类推 | 配合物杂化涉及**d轨道**参与，且中心和配体间电子提供方式不同。普通分子杂化：原子各出一个电子；配位键杂化：配体提供孤对进中心空轨道 | "配合物杂化多了d，配体给电子中心出空轨" |
| "只看配位数就能判断杂化类型" | 配位数4=sp³/dsp²，6=sp³d²/d²sp³的简化 | 同配位数可对应不同杂化：配位数4→sp³（四面体）或 dsp²（平面正方形）；配位数6→sp³d²（外轨，高自旋）或 d²sp³（内轨，低自旋）。必须结合配体场强弱和电子构型 | "配位数只定一半，配体和d电子定另一半" |
| "sp³d² 和 d²sp³ 没区别，就是写法不同" | 以为字母顺序无所谓 | sp³d² = 外层d（外轨型，HS）→ 使用 ns, np, nd 轨道；d²sp³ = 内层d（内轨型，LS）→ 使用 (n-1)d, ns, np 轨道。决定磁性和稳定性 | "外轨用外层d，内轨用内层d，字母前后有玄机" |
| "价键理论（VB）完美解释了所有配合物性质" | 教科书先讲VB再讲CFT的错觉 | VB只能解释几何构型和磁性的大致趋势，不能解释颜色（d-d跃迁）、光谱化学序列、Jahn-Teller效应。这些需要晶体场理论（CFT）和配位场理论 | "VB是入门砖，颜色和光谱要找晶体场" |

### 12.2 入门级例题

**题目**：用价键理论判断 [FeF₆]³⁻ 和 [Fe(CN)₆]³⁻ 的杂化方式、自旋态和磁性。

**预期解答路径**：
1. Fe³⁺：3d⁵
2. [FeF₆]³⁻：F⁻ 弱场配体 → d电子不重排 → 使用外层4d → sp³d²（外轨型）→ HS → 5个未成对电子 → 强顺磁性
3. [Fe(CN)₆]³⁻：CN⁻ 强场配体 → d电子重排成对 → 空出2个内层3d → d²sp³（内轨型）→ LS → 1个未成对电子 → 弱顺磁性
4. 同一个Fe³⁺，配体不同 → 杂化不同 → 磁性差异巨大

**教师引导提问**：VB理论认为外轨型用外层d、内轨型用内层d——但为什么F⁻是外轨而CN⁻是内轨？VB理论自身无法回答！这恰好引出了晶体场理论的必要性（配体场分裂能Δ vs 成对能P）

### 12.3 与现实/直觉的连接

- **内轨 = 好学生坐前排**：内轨型配合物用内层d轨道——电子"往里面挤"配对→反磁性；外轨型用外层d——电子"散开"各占一个轨道→顺磁性。磁铁吸不吸，就看你家d电子怎么排。
- **磁矩——看得见的电子排布**：高自旋 Fe³⁺ 配合物（5个未成对电子）磁矩 ≈5.9 μB，能被普通磁铁吸住。低自旋 Fe³⁺（1个未成对）磁矩 ≈1.7 μB。磁天平测出的数字直接告诉你配体场强弱——理论预测和实验测量的完美对接。
- **VB→CFT→LFT——理论的进化**：价键理论（Pauling, 1930s）→晶体场理论（Bethe, 1929）→配位场理论（MO+CFT合并）。三种理论层层递进，理解每种理论的适用范围和局限性，本身就是科学思维的训练。
