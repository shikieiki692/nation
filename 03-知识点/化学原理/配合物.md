# 配合物

- 总览：[[中国化学奥林匹克基本要求-总览]]

## 一、定义
**配合物（Coordination Compound）**：由**中心原子/离子**（通常为过渡金属）和围绕它的**配体**（含孤电子对的分子或离子）通过**配位键**形成的化合物。

- 中心原子：Lewis 酸（电子对受体）
- 配体：Lewis 碱（电子对给体）

### 配合物命名规则速查（周坤无机新课讲义 · 资产 B4-8 · 可信度：高）

> 来源：[[教学逻辑提炼-周坤无机新课-晶体配合物与气体-第一轮]]

**命名顺序规则**：
1. **配体顺序**：先无机后有机，先阴离子后中性分子
2. **同类配体**：按配位原子元素符号英文字母顺序排列
3. **配体数量**：二、三、四...（复杂配体用 bis, tris, tetrakis）
4. **中心原子氧化态**：用罗马数字标注
5. **内外界**：阴离子名称在前、阳离子名称在后

**常见配体命名**：

| 配体 | 命名 | 配体 | 命名 |
|:---|:---|:---|:---|
| H₂O | 水 | NH₃ | 氨 |
| CO | 羰基 | NO | 亚硝酰 |
| O²⁻ | 氧 | OH⁻ | 羟 |
| Cl⁻ | 氯 | CN⁻ | 氰 |
| en (乙二胺) | 乙二胺 | Cp (C₅H₅⁻) | 环戊二烯基 |

**命名实例**：

| 化学式 | 系统命名 |
|:---|:---|
| [Co(NH₃)₆]Cl₃ | 三氯化六氨合钴(III) |
| K₃[Fe(CN)₆] | 六氰合铁(III)酸钾 |
| [PtCl₂(NH₃)₂] | 二氯·二氨合铂(II) |
| [CrCl₂(H₂O)₄]Cl | 氯化二氯·四水合铬(III) |
| [Fe(CO)₅] | 五羰基合铁(0) |

## 三、核心原理

### 关键概念
| 术语 | 定义 |
|------|------|
| 配位数（CN） | 中心原子直接相连的配位原子数 |
| 配位原子 | 配体中直接与中心原子成键的原子（N, O, S, P, C...） |
| 螯合配体 | 含 ≥2 个配位原子的配体（如 en, EDTA） |
| 桥连配体 | 同时与两个中心原子成键的配体（如 OH⁻, Cl⁻） |

### 命名法要点
- 配体顺序：先阴离子后中性分子
- 配体数量词头：二、三、四...（复杂配体用 双、三...）
- 中心原子氧化态用罗马数字表示
- 实例：[Co(NH₃)₆]Cl₃ = 三氯化六氨合钴(III)

## 四、关键结论
### Werner 配位理论（1893）
- 主价（离子键）+ 副价（配位键）
- CoCl₃·6NH₃ → 溶液中 3Cl⁻ 游离 + [Co(NH₃)₆]³⁺，解释了导电性

### 常见配位数与构型
| CN | 构型 | 杂化 | 实例 |
|:---:|------|------|------|
| 2 | 直线形 | sp | [Ag(NH₃)₂]⁺ |
| 4 | 正四面体 | sp³ | [Zn(NH₃)₄]²⁺ |
| 4 | 平面正方形 | dsp² | [PtCl₄]²⁻, [Ni(CN)₄]²⁻ |
| 6 | 八面体 | d²sp³ / sp³d² | [Co(NH₃)₆]³⁺, [Fe(CN)₆]⁴⁻ |

## 五、常见分类或情形

### 按配体类型分类

| 类型 | 配体特征 | 实例 |
|:---|------|------|
| 简单配合物 | 单齿配体 | $\mathrm{K_2[PtCl_6]}$, $\mathrm{[Cu(NH_3)_4]SO_4}$ |
| 螯合物 | 多齿配体形成环状结构 | $\mathrm{[Cu(en)_2]^{2+}}$, $\mathrm{CaY^{2-}}$ (EDTA) |
| 多核配合物 | 两个以上中心原子 | $\mathrm{[(NH_3)_5Co\!-\!O\!-\!Co(NH_3)_5]^{4+}}$ |
| 簇状配合物 | 金属-金属键 | $\mathrm{Fe_2(CO)_9}$, $\mathrm{Re_2Cl_6^{2-}}$ |
| 夹心配合物 | 配体夹住中心原子 | 二茂铁 $\mathrm{(C_5H_5)_2Fe}$ |
| 混配配合物 | 多种不同配体 | 含多种配体的配合物 |

### 按成键类型分类
- $\sigma$ 配键配合物：普通配位键
- $\pi$ 键配合物：如 $\mathrm{PtCl_3(C_2H_4)^-}$
- 穴状和笼状配合物：如碱金属大环多醚配合物

### 配离子电荷
配离子的电荷数等于中心离子和配位体电荷的代数和。若配位体全部是中性分子（如 $\mathrm{NH_3}$、CO），则配离子的电荷数就等于中心离子的电荷数。

### 命名规则要点
1. 配离子命名顺序：（配位体数）配体合中心离子或原子（氧化数），氧化数用罗马数字
2. 阴离子名称在前、阳离子名称在后
3. 配体列出顺序：先无机后有机，先阴离子后中性分子
4. 同类配体按配位原子元素符号的英文字母顺序排列
5. 配体数量用二、三、四……表示（复杂配体用 bis, tris, tetrakis 等）
6. 实例：
   - $\mathrm{K_3[Fe(CN)_6]}$ — 六氰合铁(III)酸钾
   - $\mathrm{[Cu(NH_3)_4]SO_4}$ — 硫酸四氨合铜(II)
   - $\mathrm{[Ni(CO)_4]}$ — 四羰基合镍

## 六、适用条件与限制

- **配合物与复盐的区分**：配合物在水溶液中存在稳定的配离子（部分解离），而复盐（如明矾 $\mathrm{KAl(SO_4)_2\cdot12H_2O}$）完全解离为简单离子。但两者之间并无绝对界限。
- **配位数的影响因素**：中心离子电荷越高、半径越大 → 配位数越大；配位体负电荷越高、半径越大 → 配位数减小。此外配位数还与成键性质有关。
- **常见配位数**：2、4、6、8，最常见为 4 和 6。配位数可从 2 到 9，更高配位数（10~12）仅见于镧系和锕系元素。
- 中心离子不仅可以是金属离子，也可以是中性原子（如 $\mathrm{Ni(CO)_4}$ 中的 Ni）或高氧化态非金属元素（如 $\mathrm{SiF_6^{2-}}$ 中的 Si(IV)）。

## 七、常见比较与易混点

| 对比项 | 简单配合物 | 螯合物 |
|:---|:---|:---|
| 配体 | 单齿配体 | 多齿配体 |
| 环结构 | 无 | 有（通常五元环或六元环） |
| 稳定性 | 较低 | 更高（螯合效应） |
| 实例 | $\mathrm{[Cu(NH_3)_4]^{2+}}$ | $\mathrm{[Ni(en)_3]^{2+}}$ |

| 对比项 | 复盐 | 配合物 |
|:---|:---|:---|
| 溶液中 | 完全解离为简单离子 | 存在稳定的配离子 |
| 实例 | 明矾 $\mathrm{KAl(SO_4)_2\cdot12H_2O}$ | $\mathrm{[Cu(NH_3)_4]SO_4}$ |

| 易混点 | 区分 |
|:---|:---|
| 配位数 vs 配体数 | 单齿配体时二者相等；多齿配体时配位数 > 配体数（如 $\mathrm{[Pt(en)_2]^{2+}}$ 中配位数 = 4，配体数 = 2） |
| 配位体 vs 配位原子 | 配位体是整体（如 $\mathrm{NH_3}$），配位原子是直接与中心离子相连的原子（如 N） |
| 内界 vs 外界 | 内界写在方括号内（中心离子+配体），外界在方括号外 |

## 十、例题

### 例题 1：配合物命名（入门）
**题目**：命名下列配合物：
(1) $\mathrm{[Co(NH_3)_6]Cl_3}$
(2) $\mathrm{K_3[Fe(CN)_6]}$
(3) $\mathrm{[PtCl_2(NH_3)_2]}$

**分析**：
- 命名顺序：（配体数）配体合中心离子（氧化数）
- 配体顺序：先阴离子后中性分子
- 氧化数用罗马数字

**解答**：
- (1) 三氯化六氨合钴(III)
- (2) 六氰合铁(III)酸钾
- (3) 二氯·二氨合铂(II)

**反思**：配体数词头用"二""三""四"，配体间用"·"隔开。

### 例题 2：配位数判断（提高）
**题目**：判断下列配合物中中心离子的配位数：
(1) $\mathrm{[Pt(en)_2]^{2+}}$
(2) $\mathrm{[Ca(EDTA)]^{2-}}$
(3) $\mathrm{[Co(NH_3)_5Cl]^{2+}}$

**分析**：
- en（乙二胺）是双齿配体，每个 en 提供 2 个配位原子
- EDTA 是六齿配体
- $\mathrm{NH_3}$ 和 $\mathrm{Cl^-}$ 均为单齿配体

**解答**：
- (1) 配位数 = 4（2 个 en × 2 = 4）
- (2) 配位数 = 6（EDTA 六齿）
- (3) 配位数 = 6（5 个 $\mathrm{NH_3}$ + 1 个 $\mathrm{Cl^-}$ = 6）

**反思**：配位数 ≠ 配体数！多齿配体是高频陷阱。

## 十一、易错点
- **❌ 错**：认为配位数就是配体数。
  - **正**：单齿配体时二者相等；多齿配体时配位数 > 配体数。
- **❌ 错**：命名时配体顺序混乱。
  - **正**：先无机后有机，先阴离子后中性分子，同类配体按配位原子字母顺序。
- **❌ 错**：将复盐误认为配合物。
  - **正**：配合物在水溶液中部分解离出配离子，复盐完全解离为简单离子。
- **❌ 错**：认为中心原子一定是金属离子。
  - **正**：中心原子可以是金属离子、中性原子（如 Ni(CO)₄ 中的 Ni）或高价非金属（如 SiF₆²⁻ 中的 Si）。

## 十二、🎯 教学视角
### 学生常见认知误区
1. **误区**：配位数 = 配体数。
   - **原因**：日常语言中"配"容易联想到"配对"，误以为一个配体对应一个配位。
   - **正确认知**：配位数是**直接与中心原子相连的配位原子总数**。
   - **记忆法**："齿数×配体数 = 配位数"，en（双齿）×2 = 4。

2. **误区**：配合物就是"复杂的盐"。
   - **原因**：初中化学的"盐"概念泛化残留。
   - **正确认知**：配合物有**内界**（配离子）和**外界**，内界在溶液中部分稳定存在。
   - **反例**：[Cu(NH₃)₄]SO₄ 加 BaCl₂ 立即出沉淀（SO₄²⁻ 在外界），加 NaOH 不出 Cu(OH)₂（Cu²⁺ 在内界）。

### 入门级例题（教师引导提问）
**题目**：为什么 [Cu(NH₃)₄]SO₄ 溶液中加 BaCl₂ 立即产生白色沉淀，而加 NaOH 不产生蓝色沉淀？

**预期解答路径**：
- 引导学生写出电离方程式：[Cu(NH₃)₄]SO₄ → [Cu(NH₃)₄]²⁺ + SO₄²⁻
- SO₄²⁻ 在外界，自由存在 → 遇 Ba²⁺ 生成 BaSO₄↓
- Cu²⁺ 在内界，被 NH₃ 稳定配位 → 不释放 Cu²⁺，故无 Cu(OH)₂↓
- **核心结论**：内界稳定，外界自由——这是配合物与复盐的本质区别。

### 与现实/直觉的连接
- **类比**：配合物像"公司"——中心原子是 CEO（接受资源），配体是部门经理（提供资源），内界是核心团队（稳定绑定），外界是外包人员（自由流动）。
- **生活实例**：血红蛋白中的 Fe²⁺ 与卟啉环、O₂/CO 的配位——CO 中毒的本质是 CO 与 Fe²⁺ 的配位能力比 O₂ 强 200 倍。

## 十三、竞赛拓展

- **超分子配合物**：冠醚、穴醚对金属离子的选择性配位
