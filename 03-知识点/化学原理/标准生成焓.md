# 标准生成焓

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[06-热力学初步]]

## 一、定义
标准生成焓 $\Delta_f H^\circ$ 指在标准状态下，由**最稳定单质**生成 1 mol 某物质时的焓变。

最重要的定义约定：
- 最稳定单质的 $\Delta_f H^\circ = 0$

例如：
- 石墨的 $\Delta_f H^\circ = 0$
- $\mathrm{O_2(g)}$ 的 $\Delta_f H^\circ = 0$

## 二、考纲对应
- 要求层级：会用标准生成焓表计算反应焓变，并理解“稳定单质取零”的含义
- 课堂定位：是 [[盖斯定律]] 进入查表计算题的标准入口页

## 三、核心原理

### 3.1 为什么要引入标准生成焓
因为化学反应千千万万，不可能每一个都单独测反应焓。  
但若把常见物质的生成焓整理成表，就能通过统一公式快速计算大量反应。

### 3.2 它本质上来自 Hess 定律
任意反应都可以想成：
- 反应物“回到稳定单质”
- 再由稳定单质“生成产物”

因此：

$$
\Delta_r H^\circ = \sum \nu \Delta_f H^\circ(\text{产物}) - \sum \nu \Delta_f H^\circ(\text{反应物})
$$

### 3.3 为什么强调“稳定单质”
若一种元素有多种同素异形体，只有最稳定形态取零点。

比如：
- 石墨取零，金刚石不取零
- $\mathrm{O_2}$ 取零，$\mathrm{O_3}$ 不取零

## 四、关键结论
- 标准生成焓是一个**查表型核心量**
- 稳定单质取零是定义，不是实验偶然结果
- 利用生成焓表可以直接求反应焓变
- 物态一定要写清楚，否则数据会变

## 五、常见分类或情形

### 5.1 典型零点物质
| 物质 | 是否取零 |
|------|------|
| C(石墨) | 是 |
| C(金刚石) | 否 |
| O2(g) | 是 |
| O3(g) | 否 |
| H2(g) | 是 |

### 5.2 常见查表计算
- 燃烧热
- 分解热
- 合成反应焓变

### 5.3 物态的影响
例如：
- $\mathrm{H_2O(l)}$ 与 $\mathrm{H_2O(g)}$ 的生成焓不同
- 液态与气态之间差值本质上体现了相变焓

## 六、适用条件与限制
- 必须在标准状态数据体系下使用
- 使用前要统一物态、温度和反应方程式计量系数
- 标准生成焓只是标准条件下的数据，温度变化较大时不能机械照搬

## 七、常见比较与易混点
- **生成焓不等于燃烧焓**：一个是“由单质生成”，一个是“与氧完全燃烧”
- **稳定单质取零不等于元素所有形态都取零**
- **查表公式是产物减反应物，不要反过来**
- **物态不同，生成焓会不同**

## 十、例题
- **例题 1**：为什么石墨的标准生成焓为零，而金刚石的不为零？  
  **答案要点**：定义规定零点选最稳定单质；碳在标准状态下最稳定形态是石墨。
- **例题 2**：如何用生成焓求某燃烧反应的反应焓？  
  **答案要点**：直接套用“产物生成焓总和减去反应物生成焓总和”。

## 十一、易错点
- 忘记稳定单质才取零
- 漏写物态导致代错数据
- 产物与反应物前后代错
- 把“标准”两个字忽略掉，混用非标态概念

## 十二、🎯 教学视角
- 这一页最值得讲清的是“为什么查一个表就能算很多反应”
- 第一轮课堂要重点训练“零点选择 + 公式代入 + 物态检查”
- 和 [[盖斯定律]] 连起来讲，学生更容易理解公式不是凭空来的

## 十三、🧰 备课可抽料资产

### 13.1 主干结论
- 标准生成焓是热化学计算中的标准表数据入口
- 稳定单质取零是整个体系的基准
- 公式本质来自 Hess 定律

### 13.2 机制 / 推导资产
- 可直接复用的课堂链：反应物回到单质 -> 单质生成产物 -> 两段相减
- 可直接复用的桥句：“查表法本质上是 Hess 定律的批量化”

### 13.3 误区 / 纠偏资产
- “稳定单质才取零”
- “先看物态，再看数值”
- “产物减反应物，别写反”

### 13.4 题型 / 例题资产
- 建议题链：零点判断 -> 简单查表 -> 燃烧焓 -> 同素异形体陷阱
