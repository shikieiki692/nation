# 摩尔电导率 (Molar Conductivity)

## 一、定义

**摩尔电导率（Λₘ）** 是衡量电解质溶液导电能力的量，定义为 **溶液的电导率（κ）除以电解质浓度（c）**：

$$ \Lambda_m = \frac{\kappa}{c} $$

- 单位：S·m²·mol⁻¹
- 物理意义：1 mol 电解质在给定条件下所有离子贡献的电导之和
- **极限摩尔电导率（Λ₀）**：无限稀释时的摩尔电导率（离子间相互作用消失）

## 二、核心规律

### 2.1 Kohlrausch 定律
$$\Lambda_m = \Lambda_0 - A\sqrt{c}$$

- 适用于 **强电解质** 稀溶液
- A 为与溶剂、温度、电荷数相关的常数
- 外推至 c → 0 可得 Λ₀

### 2.2 Kohlrausch 离子独立迁移定律（无限稀释）
$$\Lambda_0 = \nu_+\lambda_+^0 + \nu_-\lambda_-^0$$

- λ⁰₊、λ⁰₋：阳离子和阴离子的极限摩尔电导率
- ν₊、ν₋：化学式中的离子计量数
- 可用于计算弱电解质的 Λ₀（无法直接测量）

### 2.3 弱电解质
- 不满足 Kohlrausch 平方根定律
- Λ₀ 通过离子独立迁移定律间接求得
- 解离度 α = Λₘ/Λ₀（Ostwald 稀释定律基础）

## 四、教学常见节点

- **Λₘ 随浓度变化**：强电解质缓慢下降（离子缩合），弱电解质快速下降（解离度降低）
- **应用：求弱酸的 Ka**：测 Λₘ → Λ₀(Kohlrausch) → α → Ka
- **应用：电导滴定**：酸碱中和过程中 Λₘ 突变点即为终点
- **水的电导**：纯水 Λ₀ = λ⁰(H⁺) + λ⁰(OH⁻) ≈ 5.5 × 10⁻⁶ S·m²·mol⁻¹
- **竞赛考查**：从 Λₘ 与 √c 图判断强弱电解质、用离子独立迁移定律计算

## 五、竞赛拓展

- **Walden 规则**：Λ₀η = 常数（η 为溶剂粘度）
- **Λ₀ 的温度系数**：温度升高，Λ₀ 增大（离子迁移率增加）
- **电导法测溶解度**：难溶盐饱和溶液的电导率 → 溶解度
