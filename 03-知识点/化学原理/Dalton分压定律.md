# Dalton分压定律

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[02-气体]]

## 一、定义
**Dalton 分压定律**（1807 年 Dalton 提出）：混合气体的总压力等于各组分气体分压之和。某组分气体的分压等于总压力乘以该气体的摩尔分数（或体积分数）。

**核心公式**：
$$p_{\text{总}} = \sum p_i$$

$$p_i = p_{\text{总}} \times \frac{n_i}{\sum n_i} = p_{\text{总}} \times \frac{V_i}{\sum V_i}$$

其中 $p_i$ 为组分 $i$ 的**分压**——该组分单独占据混合气体总体积时所显示的压力。$x_i = n_i / \sum n_i$ 为摩尔分数。

- 本知识点在考纲中的作用：分压定律是处理混合气体计算的唯一基础定律，贯穿气体分析、化学反应中气体计量、排水集气法、化学平衡分压计算等领域。

## 三、核心原理

### 分子运动论解释
压力是分子碰撞器壁而产生的。A、B 两种不起化学作用的气体混合在同一容器中时，A 气体分子给器壁的压力为 $p_A$，B 气体分子给器壁的压力为 $p_B$，器壁所受总压力自然是 $p_A$ 与 $p_B$ 之和。每种气体的分压与它单独存在时相同，互不干扰。

### 两种表述条件

**恒温恒容条件下**（$T, V_{\text{总}}$ 恒定）：
$$p_{\text{总}} = p_A + p_B, \quad p_A = \frac{n_A RT}{V_{\text{总}}}$$
$$p_A = p_{\text{总}} \times \frac{n_A}{n_{\text{总}}}$$

**恒温恒压条件下**（$T, p_{\text{总}}$ 恒定）：
$$V_{\text{总}} = V_A + V_B, \quad V_A = \frac{n_A RT}{p_{\text{总}}}$$
$$V_A = V_{\text{总}} \times \frac{p_A}{p_{\text{总}}}, \quad p_A = p_{\text{总}} \times \frac{V_A}{V_{\text{总}}}$$

其中 $V_A$ 称为 A 气体的**分体积**——在总压条件下该组分单独占有的体积。

### 实验验证
Ramsay 曾用钯管实验验证：钯(Pd)管允许 $\mathrm{H_2}$ 分子自由透过而 $\mathrm{Ar}$ 分子不能。将 Ar 与 $\mathrm{H_2}$ 混合气封于内管，外管通入 $\mathrm{H_2}$，当内外 $\mathrm{H_2}$ 压力相等时测定总压和 $\mathrm{H_2}$ 分压，证明二者之和恰等于混合气体总压力。

## 四、关键结论

### 分压与摩尔分数的关系
$$\frac{p_i}{p_{\text{总}}} = \frac{n_i}{n_{\text{总}}} = \frac{V_i}{V_{\text{总}}} = x_i$$

在相同温度压力下，气体物质的量与体积成正比，因此体积分数等于摩尔分数。

### 排水集气法的分压校正
用排水集气法收集的气体中含有水蒸气，需校正：
$$p_{\text{干燥气体}} = p_{\text{总}} - p(\mathrm{H_2O})$$
$$V_{\text{干燥气体}} = V_{\text{总}} \times \frac{p_{\text{干燥气体}}}{p_{\text{总}}}$$

其中 $p(\mathrm{H_2O})$ 为实验温度下水的饱和蒸气压。

**例**：在 $100.0\ \mathrm{kPa}$、$20^\circ\mathrm{C}$ 时用排水集气法收集 $28.4\ \mathrm{cm^3}$ 氢气，已知 $p(\mathrm{H_2O}) = 2.34\ \mathrm{kPa}$，则干燥后氢气体积为：
$$V(\mathrm{H_2}) = 28.4 \times \frac{100.0 - 2.34}{100.0} = 27.7\ \mathrm{cm^3}$$

### 混合气体计算实例
在 $25^\circ\mathrm{C}$、$101.0\ \mathrm{kPa}$ 时，丁烷气中含 $\mathrm{H_2S}$ 质量分数 $1.00\%$（$M_{\mathrm{H_2S}} = 34.1$，$M_{\mathrm{C_4H_{10}}} = 58.1$）：
$$n(\mathrm{H_2S}) = \frac{1000 \times 0.01}{34.1} = 0.293\ \mathrm{mol}$$
$$n(\mathrm{C_4H_{10}}) = \frac{990}{58.1} = 17.0\ \mathrm{mol}$$
$$p(\mathrm{H_2S}) = 101.0 \times \frac{0.293}{17.293} = 1.71\ \mathrm{kPa}$$

## 五、常见分类或情形
- **恒容混合**（$T, V$ 恒定）：分压相加，$p_{\text{总}} = \sum p_i$
- **恒压混合**（$T, p$ 恒定）：分体积相加，$V_{\text{总}} = \sum V_i$
- **不同初压的混合**：先用 Boyle 定律计算各气体膨胀至总体积后的分压，再求和
- **含可凝蒸气的混合气**：需扣除饱和蒸气压（排水集气法）
- **化学反应中的分压**：平衡常数计算需用各气体的分压

## 六、适用条件与限制
- 各组分气体须符合理想气体行为
- 各组分气体之间不起化学反应
- 恒温恒容与恒温恒压的公式不同，**注意区分条件**
- $p_A = \frac{n_A RT}{V_{\text{总}}}$ 仅在恒温恒容条件下的分压计算中成立
- $p_A V_{\text{总}} = p_{\text{总}} V_A$ 的关系需要注意公式推导的前提

## 七、常见比较与易混点
| 易混点 | 区分 |
|------|------|
| 分压 vs 分体积 | 分压：$T,V_{\text{总}}$ 恒定下该组分单独存在的压力；分体积：$T,p_{\text{总}}$ 恒定下该组分单独占有的体积 |
| $p_A = \frac{n_A RT}{V}$ vs $p_A = p_{\text{总}} \times \frac{n_A}{n_{\text{总}}}$ | 前者用于直接计算分压（已知 $n_A,V,T$），后者用于已知总压时由摩尔分数求分压 |
| 摩尔分数 vs 体积分数 | 对于理想气体两者相等，但前提是同一 $T,p$ 条件下 |
| 分压 vs 总压 | 混合气体中各组分的分压之和等于总压；分压不能直接测量（需通过摩尔分数换算） |
| 排水集气的"湿体积" vs "干体积" | 实验测得的体积是湿体积（含饱和水蒸气），计算纯气体量时必须做分压校正 |

## 十、例题
### 例题 1 — 不同初压气体混合
**题目：** 一个 $6\ \mathrm{dm^3}$、$9\ \mathrm{MPa}$ 的氧气储罐与另一个 $12\ \mathrm{dm^3}$、$3\ \mathrm{MPa}$ 的氮气储罐由活塞连接，打开活塞混合均匀后（温度不变），求氧气、氮气的分压与分体积。

**分析：** 混合后总体积为 $18\ \mathrm{dm^3}$，但总压不是 $9+3=12\ \mathrm{MPa}$。需先用 Boyle 定律求各气体膨胀后的分压。

**解答：**
氧气膨胀后分压：$p(\mathrm{O_2}) = 9 \times \frac{6}{18} = 3\ \mathrm{MPa}$
氮气膨胀后分压：$p(\mathrm{N_2}) = 3 \times \frac{12}{18} = 2\ \mathrm{MPa}$
总压：$p_{\text{总}} = 3 + 2 = 5\ \mathrm{MPa}$
分体积（在总压条件下）：
$$V(\mathrm{O_2}) = 18 \times \frac{3}{5} = 11\ \mathrm{dm^3}$$
$$V(\mathrm{N_2}) = 18 \times \frac{2}{5} = 7\ \mathrm{dm^3}$$

**反思：** 混合前各气体初压不同，直接加和求总压是错误的。必须先按等温膨胀计算各气体在总体积下的分压。

## 十一、易错点
- **忘记排水集气的水蒸气校正**：实验收集的气体是"湿气体"，含饱和水蒸气，计算时必须减去水蒸气分压
- **混淆恒容和恒压条件**：两种条件下的分压、分体积公式不同
- **混合气体总压直接相加**：不同初始压力的气体混合时，不能直接将初始压力相加，需先计算混合后的分压
- **$p_A = \frac{n_A RT}{V_A}$ 的错误使用**：正确的分压计算是 $p_A = \frac{n_A RT}{V_{\text{总}}}$（恒容）或 $p_A = p_{\text{总}} \times x_A$（任意条件）

## 十二、🎯 教学视角

### 12.1 学生典型认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 口诀 |
|:---|:---|:---|:---|
| "总压 = 各气体初始压力的和" | 直觉加法 | 不同初压气体混合后总压 ≠ $p_1 + p_2$！各气体等温膨胀到总体积后再算分压求和 | "先膨胀，再求和" |
| "排水集气 = 直接读数" | 高中实验习惯 | $p_{\text{干}} = p_{\text{总}} - p(\mathrm{H_2O})$，忘记扣水蒸气压是最常见的竞赛扣分点 | "湿气减水气，才是干气" |
| "分压定律 = 分体积定律，一回事" | 概念混淆 | 恒容：$p_i = x_i \cdot p_{\text{总}}$；恒压：$V_i = x_i \cdot V_{\text{总}}$。条件不同公式不同 | "恒容谈分压，恒压谈分体积" |

### 12.2 入门级例题

**题目**：$20^\circ\mathrm{C}$ 排水集气收集 $\mathrm{H_2}$，湿气体体积 $150\ \mathrm{cm^3}$，大气压 $100.0\ \mathrm{kPa}$。$20^\circ\mathrm{C}$ 时 $p(\mathrm{H_2O}) = 2.34\ \mathrm{kPa}$。求干燥 $\mathrm{H_2}$ 的物质的量。

**预期解答路径**：
1. $p(\mathrm{H_2}) = 100.0 - 2.34 = 97.66\ \mathrm{kPa}$
2. $n = \frac{pV}{RT} = \frac{97.66 \times 0.150}{8.31 \times 293} = 6.03 \times 10^{-3}\ \mathrm{mol}$

**教师引导提问**：如果忘记扣水蒸气压，会得到什么结果？（$n = 6.17 \times 10^{-3}\ \mathrm{mol}$，偏差约 2.3%——误差不大但系统错误）

### 12.3 与现实/直觉的连接

- **呼吸**：吸入的空气中 $p(\mathrm{O_2}) = 21.2\ \mathrm{kPa}$（总压 101 kPa × 21%），正是 Dalton 分压定律的日常应用
- **高原反应**：高原总压低 → O₂ 分压低 → 缺氧——不是 O₂ 比例变了，是分压变了

## 十二、竞赛拓展
- **真实气体的分压**：实际气体混合时可能偏离 Dalton 定律，尤其在高压条件下
- **与化学平衡的结合**：$K_p$ 表达式中各气体的分压需用 Dalton 定律展开计算——这是平衡计算题的必备技能
- **气体分析**：利用分压、分体积和体积分数的关系进行多组分气体混合物的定量分析
