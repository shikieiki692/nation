# 原电池

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[08-氧化还原与电化学]]

## 一、定义
**原电池（Galvanic Cell）**：利用**自发氧化还原反应**将化学能直接转化为电能的装置。

- **阳极（Anode）**：发生**氧化**反应（−极）
- **阴极（Cathode）**：发生**还原**反应（+极）
- 电子从阳极经外电路流向阴极

## 三、核心原理

### 锌-铜原电池（Daniel 电池）的结构与原理
图10.1所示的 Zn-Cu 电池：
- 锌片插入 $\mathrm{ZnSO_4}$ 溶液，铜片插入 $\mathrm{CuSO_4}$ 溶液
- 两烧杯用**盐桥**（盛有KCl饱和溶液胶冻的U形管）连接——构成离子流的通路
- 锌片和铜片用导线连接，中间接电压表

**工作原理**：
- 负极（锌极）：$\mathrm{Zn \rightleftharpoons Zn^{2+} + 2e^-}$（氧化，向外电路输出电子）
- 正极（铜极）：$\mathrm{Cu^{2+} + 2e^- \rightleftharpoons Cu}$（还原，从外电路接受电子）
- 电流方向：铜极 → 锌极（外电路）；电子方向：锌极 → 铜极
- 电池电动势（标准态）：$E_{\text{池}}^{\ominus} = 1.10\ \mathrm{V}$

### 电池符号表示法
IUPAC 规定负极写在左边，正极写在右边：
$$\mathrm{(-)\ Zn\ |\ Zn^{2+}(c_1)\ \|\ Cu^{2+}(c_2)\ |\ Cu\ (+)}$$

**符号规则**：
- `|` = 物相界面（如金属与溶液之间）
- `‖` = 盐桥
- `,` = 同一液相中不同离子之间的分隔
- 惰性电极（如 Pt）需注明，气体用分压标记
- 必要时注明浓度

**复杂电池符号示例**：
$$\mathrm{(-)\ Pt\ |\ H_2(p^{\ominus})\ |\ H^+(1\ mol\cdot dm^{-3})\ \|\ Fe^{3+}(1\ mol\cdot dm^{-3}),\ Fe^{2+}(1\ mol\cdot dm^{-3})\ |\ Pt\ (+)}$$

### 电动势 $E_{\text{池}}$ 与 Gibbs 自由能
$$\Delta G = -nFE_{\text{池}}$$

- 标准态：$\Delta G^{\ominus} = -nFE_{\text{池}}^{\ominus}$
- Faraday 恒量：$F = 9.65 \times 10^4\ \mathrm{C\cdot mol^{-1}}$（1 mol 电子电量）
- 298K 时：$E_{\text{池}}^{\ominus} = \dfrac{0.0591\ \mathrm{V}}{n} \lg K^{\ominus}$

$E_{\text{池}} > 0 \iff \Delta G < 0$ → 电池反应自发（即电池可对外做电功）

### 盐桥的作用
1. **闭合电路**：提供离子迁移的通路
2. **消除液接电势**：使两种不同溶液接触处的电势差降至最低
3. **维持电中性**：负极区 $\mathrm{Zn^{2+}}$ 增多，正极区 $\mathrm{Cu^{2+}}$ 减少，盐桥中 K⁺ 向正极区迁移，Cl⁻ 向负极区迁移
4. 常用盐桥：**KCl-琼脂**冻胶（U形管中）或 **KNO₃-琼脂**（当溶液含 Ag⁺ 时避免 Cl⁻ 沉淀）

### 电极电势理论
电极电势由两部分组成：
1. **表面电势**：金属表面自由电子逃逸的趋势
2. **相间电势**：金属与溶液界面处的双电层（如 Cu 表面富集 Cu²⁺，溶液侧富集 SO₄²⁻）

## 四、关键结论

### 电动势的实验测定
**Zn-Cu 电池**：$(-)\ \mathrm{Zn|Zn^{2+}(1M)\|Cu^{2+}(1M)|Cu\ (+)}$，$E_{\text{池}}^{\ominus} = 1.10\ \mathrm{V}$

**Cu-Ag 电池**：$(-)\ \mathrm{Cu|Cu^{2+}(1M)\|Ag^+(1M)|Ag\ (+)}$，$E_{\text{池}}^{\ominus} = 0.46\ \mathrm{V}$

由以上两个电池可推算：$E^{\ominus}(\mathrm{Ag^+/Ag}) - E^{\ominus}(\mathrm{Zn^{2+}/Zn}) = 0.46 + 1.10 = 1.56\ \mathrm{V}$

**重要注意事项**：测量电池电动势须用**高阻抗的晶体管伏特计或电位差计**，普通电压表因内阻不够大而有电流通过，读数不等于真实电动势。

### 常见原电池类型

| 类型 | 原理 | 实例 | 特征 |
|------|------|------|------|
| 化学电池 | 不同电极/不同氧化还原电对 | Daniel 电池（Zn/Cu） | $E_{\text{池}}^{\ominus} \neq 0$ |
| 浓差电池 | 相同电极，不同浓度 | $\mathrm{Cu|Cu^{2+}(c_1)\|Cu^{2+}(c_2)|Cu}$ | $E_{\text{池}}^{\ominus} = 0$，电动势完全来自浓度差 |
| 燃料电池 | 燃料气体为还原剂，O₂ 为氧化剂 | H₂-O₂ 燃料电池 | 高效率能量转化（>80%） |

### 电池电动势计算步骤
1. 确定正负极（$E^{\ominus}$ 高的为正极）
2. 标准态：$E_{\text{池}}^{\ominus} = E_{\text{正}}^{\ominus} - E_{\text{负}}^{\ominus}$
3. 非标准态：用 [[Nernst方程]] 分别计算各电极的 $E$，再相减

**浓差电池示例**（例10.5(2)）：
$\mathrm{Cu|Cu^{2+}(10^{-4}\ M)\|Cu^{2+}(1\ M)|Cu}$
$E_{\text{池}} = 0 - \dfrac{0.0591}{2} \lg \dfrac{10^{-4}}{1} = +0.118\ \mathrm{V}$

### 化学电源（实用原电池）

#### 一次电池（不可充电）
1. **锰锌干电池**（Leclanche 电池）：
   - 负极（锌筒）：$\mathrm{Zn + 2NH_4Cl \rightleftharpoons Zn(NH_3)_2Cl_2 + 2H^+ + 2e^-}$
   - 正极：$\mathrm{2MnO_2 + 2H^+ + 2e^- \rightleftharpoons 2MnO(OH)}$
   - 电动势 ≈ 1.5 V
   - 改进型：碱性锰锌电池（KOH 为电解质）

2. **锂碘电池**：
   - 负极：$\mathrm{2Li \rightleftharpoons 2Li^+ + 2e^-}$
   - 正极：$\mathrm{P_2VP \cdot nI_2 + 2Li^+ + 2e^- \rightleftharpoons P_2VP \cdot (n-1)I_2 + 2LiI}$
   - 电动势 ≈ 3 V，寿命长达 10 年，用于心脏起搏器

3. **银锌电池**（高能电池）：
   - 负极：$\mathrm{Zn + 2OH^- \rightleftharpoons Zn(OH)_2 + 2e^-}$
   - 正极：$\mathrm{AgO + H_2O + 2e^- \rightleftharpoons Ag + 2OH^-}$
   - 纽扣电池形式，用于电子手表、计算器等

#### 二次电池（可充电）
1. **铅蓄电池**：
   - 放电：$\mathrm{Pb + PbO_2 + 2H_2SO_4 \longrightarrow 2PbSO_4 + 2H_2O}$
   - 充电：$2\mathrm{PbSO_4 + 2H_2O \longrightarrow Pb + PbO_2 + 2H_2SO_4}$
   - 工作电压稳定、价格便宜，缺点是沉重

2. **镍镉电池**：$(-)\ \mathrm{Cd|Cd(OH)_2\|KOH\|Ni(OH)_2|NiO(OH)\ (+)}$

3. **镍氢电池**：以 $\mathrm{LaNi_5}$ 合金等替代镉电极，环保

4. **锂离子电池**：
   - 负极：嵌锂离子的层状石墨（$\mathrm{Li_xC_6}$）
   - 正极：嵌锂离子的金属氧化物（如 $\mathrm{LiCoO_2}$，磷酸铁锂 $\mathrm{LiFePO_4}$）
   - 反应：$\mathrm{Li_xC_6 + Li_{1-x}CoO_2 \xrightleftharpoons[充电]{放电} 6C + LiCoO_2}$
   - 无记忆效应，高容量；磷酸铁锂电池工作电压 3.2V，电容量大

#### 燃料电池
- 燃料气体（$\mathrm{H_2, CH_4, CH_3OH}$ 等）送到负极室，空气/O₂ 输入正极室
- 碱性氢氧燃料电池：以 KOH 为电解质，Pt、Ni 等贵金属催化
- 最大特点：能量转化率可达 **80%** 以上（柴油发电机 < 40%）
- 产物污染小（主要生成 H₂O 和 CO₂）

## 五、常见分类或情形

### 按电极构成分类
| 电池类型 | 两个电极的特点 | 实例 |
|------|------|------|
| 双化学电池 | 两个化学性质不同的电极 | Zn/Cu 电池 |
| 浓差电池 | 电极相同，浓度不同 | Cu²⁺(c₁)/Cu²⁺(c₂) 电池 |
| 电极浓差电池 | 电极材料浓度不同（如汞齐） | Zn(Hg)(c₁)/Zn(Hg)(c₂) |

### 按电解质分类
- **酸性电池**：H₂SO₄ 为电解质（铅蓄电池）
- **碱性电池**：KOH 为电解质（碱性锰锌电池、镍镉电池、氢氧燃料电池）
- **有机电解质电池**：锂离子电池（$\mathrm{LiClO_4}$ 或 $\mathrm{LiPF_6}$ 的 EC/DMC 溶液）
- **固态电解质电池**：锂碘电池（固态 LiI 薄膜）

### 按使用方式分类
- **一次电池**：用完废弃（锌锰干电池、锂碘电池）
- **二次电池**：可充放电循环（铅蓄电池、镍镉电池、镍氢电池、锂离子电池）
- **燃料电池**：连续供给燃料和氧化剂，持续发电

## 六、适用条件与限制

- 原电池中的氧化还原反应必须是**自发的**（$\Delta G < 0, E_{\text{池}} > 0$）
- 实际电池的内阻、超电势、电解液浓度变化等因素使工作电压低于电动势
- 盐桥不能完全消除液接电势，仅能降至最低
- 一次电池放电过程中反应物逐渐消耗，电压会下降
- 二次电池的充放电循环次数有限（电极材料会劣化）
- 燃料电池需要贵金属催化剂（Pt、Ni、Au、Ag），成本较高
- 电池符号表示法是约定俗成的，实验中必须明确标注浓度和压力

## 七、常见比较与易混点

| 易混点 | 区分 |
|------|------|
| 原电池 vs 电解池 | 原电池：化学能 → 电能（自发）；电解池：电能 → 化学能（非自发，需外加电压） |
| 负极 vs 阳极 | 原电池：负极 = 阳极（氧化反应，电子流出）；电解池：负极 = 阴极（还原反应） |
| 正极 vs 阴极 | 原电池：正极 = 阴极（还原反应）；电解池：正极 = 阳极（氧化反应） |
| 电池符号左 vs 右 | 左 = 负极（氧化）；右 = 正极（还原）——这是 IUPAC 约定 |
| 电动势 vs 电压 | 电动势 = 无电流通过时的开路电压；有电流时因内阻和极化，工作电压 < 电动势 |
| 一次电池 vs 二次电池 | 一次不可逆充电；二次可逆充放电 |
| 燃料电池 vs 普通电池 | 燃料和氧化剂从外部**持续供给**，不是预储存在电池内 |
| 盐桥 vs 半透膜 | 盐桥靠离子迁移保持电中性；半透膜选择性透过来实现分隔 |

## 十二、🎯 教学视角

### 12.1 学生典型认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 口诀 |
|:---|:---|:---|:---|
| "原电池和电解池是一回事，只是方向反了" | 两者都有"正负极"和"阴阳极" | 原电池：**负极=阳极**（氧化），**正极=阴极**（还原）；电解池：**负极=阴极**（还原），**正极=阳极**（氧化）——阴阳极的定义在两种装置中**恰好相反**！ | "原电池负阳氧，电解池负阴还" |
| "负极一定是阳极、正极一定是阴极" | 高中记忆的绝对化 | 原电池中负极=阳极（说得通）；电解池中负极=阴极——阴阳极由**反应类型**（氧化/还原）定义，正负极由**电位高低**定义，两者不是一回事 | "阴阳看反应，正负看电位" |
| "盐桥可有可无，就是为了好看" | 课本装置图中的盐桥像"装饰品" | 盐桥有三大作用：(1)闭合电路（离子通路）；(2)消除液接电势；(3)维持电中性——没有盐桥，电池很快停止工作！ | "没盐桥，电池秒停" |
| "电动势就是电压，用万用表直接测就行" | 日常"量电压"的经验 | 电动势是**无电流通过时**的开路电压；普通电压表内阻不够大，测量时有电流通过 → 读数偏低！必须用**高阻抗电位差计** | "测电动势要用高阻抗，万用表不行" |

### 12.2 入门级例题

**题目**：写出 Daniel 电池（Zn|Zn²⁺(1M)‖Cu²⁺(1M)|Cu）的电池符号，标出正负极和电子流向，已知 E°(Zn²⁺/Zn)=−0.76V，E°(Cu²⁺/Cu)=+0.34V，计算标准电动势。

**预期解答路径**：
1. 负极（氧化）：Zn → Zn²⁺ + 2e⁻（E°=−0.76V，更负 → 易氧化 → 负极）
2. 正极（还原）：Cu²⁺ + 2e⁻ → Cu（E°=+0.34V，更正 → 易还原 → 正极）
3. E°池 = E°正 − E°负 = 0.34 − (−0.76) = 1.10V
4. 电子流向：Zn极 → 导线 → Cu极（负极出、正极入）

**教师引导提问**：如果不用盐桥，直接用导线把两个烧杯里的溶液连起来行不行？（不行——没有离子通路，电路不闭合。而且两种溶液直接混合会产生液接电势。盐桥的妙处就在这里——只让离子通过，不让溶液混合）

### 12.3 与现实/直觉的连接

- **水果电池**：柠檬插铜片和锌片就能点亮 LED——这就是原电池的直观体验。柠檬汁是电解质溶液，Zn 负极氧化，Cu 正极上 H⁺ 还原出 H₂。1.10V 的 Daniel 电池和大家课桌上做过的水果电池本质上是同一回事。
- **手机锂电池为什么能充电？** 锂离子电池是二次电池（可逆充放电）。放电时 Li⁺ 从石墨负极脱出穿过电解质到正极；充电时外加反向电压使 Li⁺ 回到石墨。理解了原电池和电解池的关系，就能理解"放电=原电池、充电=电解池"。
- **盐桥 = 走廊**：把原电池想象成两个房间——一个房间（负极区）Zn²⁺ 越来越多（正电荷堆积），另一个房间（正极区）Cu²⁺ 越来越少（负电荷过剩）。盐桥就是连接两个房间的走廊，K⁺ 和 Cl⁻ 在里面穿梭维持电中性。
