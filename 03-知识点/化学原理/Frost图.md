# Frost图

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[08-氧化还原与电化学]]、[[决赛10-电化学]]

## 一、定义
- **Frost图**（Frost diagram，又称氧化态图）是以元素的氧化数 $N$ 为横坐标、以 $\nu E^{\ominus}$ 为纵坐标所作的图。
- 其中 $\nu$ 是从氧化态 $N$ 还原至单质（$N=0$）所转移的电子数，$E^{\ominus}$ 为该还原半反应的标准电位。
- 由于 $\nu E^{\ominus} = -\Delta_r G^{\ominus}/F$，Frost 图也可看作元素各氧化态的**标准生成 Gibbs 自由能**（除以 $F$）随氧化数的变化图。

![[mineru/无机化学第6版Weller/无机化学第6版氧化还原反应180-212_images/5d61dc284a5d225bd6931502b9a36a5d5b37fa88725149242af20ddce6776699.jpg|350]]
> **图 5.5** Frost 图一般形式：横轴为氧化数 $N$，纵轴为 $\nu E^{\ominus}$；位置最低 = 最稳定氧化态；折线"凸点"对应歧化，"凹点"对应反歧化。
> 来源：Weller《无机化学》第6版 §5.13

## 二、考纲对应
- 对应考纲条目：[[08-氧化还原与电化学]]、[[决赛10-电化学]]
- 所属模块：[[基础要求-化学原理]]
- 本知识点在考纲中的作用：作为 Latimer 图的图形化等价工具，更直观地判断氧化态稳定性、歧化/反歧化趋势和氧化还原强弱。

## 三、核心原理

### 3.1 图的构筑方法
1. 将单质 $X(0)$ 放在原点 $(N=0, \nu E^{\ominus}=0)$。
2. 对每一氧化态 $X(N)$，写出还原半反应 $X(N) + \nu e^- \longrightarrow X(0)$。
3. 从 Latimer 图数据累加：$\nu E^{\ominus}(N) = \sum_i \nu_i E_i^{\ominus}$（从 0 到 $N$ 逐段累加）。
4. 注意：氧化数为负时（如 $N=-1$），$\nu$ 取负值，$\nu E^{\ominus}$ 可能为正或负。

**示例：氧在酸性溶液中的 Frost 图**
| 氧化数 $N$ | 物种 | 累加过程 | $\nu E^{\ominus}/\mathrm{V}$ |
|:---:|:---|:---|:---:|
| 0 | $O_2$ | 原点 | 0 |
| $-1$ | $H_2O_2$ | $-1\times(+0.70) = -0.70$ | $-0.70$ |
| $-2$ | $H_2O$ | $-2\times(+1.23) = -2.46$ | $-2.46$ |

### 3.2 图的解读

#### （1）稳定性判断
- **位置越低越稳定**：$\nu E^{\ominus}$ 最小的氧化态是热力学上最稳定的氧化态。
- 例如图 5.6 中氮的 Frost 图：酸性条件下 $NH_4^+$ 位置最低，故最稳定。

#### （2）氧化剂/还原剂强弱（斜率 = $E^{\ominus}$）
- 连接相邻两点的**线段斜率**等于该氧化还原电对的标准电位：
$$\text{斜率} = \frac{\Delta(\nu E^{\ominus})}{\Delta N} = \frac{\nu E^{\ominus}}{\nu} = E^{\ominus}$$
- **斜率越陡（正值越大）**：该电对的氧化型是越强的氧化剂。
- **斜率越平（正值越小/负值越大）**：该电对的还原型是越强的还原剂。

#### （3）歧化判据（凸形曲线）
- 若某物种的点位于**相邻两点连线的上方**（凸形），该物种不稳定，发生歧化。
- 几何解释：中间物种的摩尔 Gibbs 自由能高于两侧物种的平均值，因此倾向于分解为两侧物种。

**示例**：图 5.6 中 $NH_2OH$（羟胺）在酸性 Frost 图中位于 $NH_4^+$ 和 $N_2$ 连线上方，故不稳定，歧化为 $NH_3$ 和 $N_2$。

#### （4）反歧化判据（凹形曲线）
- 若某物种的点位于**相邻两点连线的下方**（凹形），则两侧物种倾向于反歧化（归中）为该物种。
- 几何解释：中间物种的摩尔 Gibbs 自由能低于两侧物种的平均值，因此是热力学上更有利的存在形式。

**示例**：$N_2O$ 在氮的 Frost 图中位于 $NH_4^+$ 和 $NO_3^-$ 连线的下方，故 $NH_4^+$ 和 $NO_3^-$ 可反歧化为 $N_2O$（虽然动力学上固态 $NH_4NO_3$ 的分解是爆炸反应）。

### 3.3 不同 pH 的 Frost 图
- 酸性（pH = 0，红线）和碱性（pH = 14，蓝线）的 Frost 图形状可能显著不同。
- 关键规律：当还原过程需要脱去氧原子时，反应消耗 $H^+$，因此**酸性溶液中的氧合阴离子总是比碱性溶液中更强的氧化剂**（酸性图中对应线段更陡）。
- 例如 $NO_2^-$：在碱性图中位于 $NO$ 和 $NO_3^-$ 连线**下方**（稳定），在酸性图中接近直线（趋于歧化）。

## 四、关键结论
- Frost 图与 Latimer 图**信息完全等价**，只是表现形式不同：Latimer 是"一维线性"，Frost 是"二维图形"。
- Frost 图的优势在于**直观比较多个氧化态的相对稳定性**和**快速识别歧化/反歧化**。
- 斜率 = $E^{\ominus}$ 是 Frost 图最核心的数学关系，必须熟练。

## 五、常见分类或情形
- **酸性 Frost 图**（pH = 0）：标注为红线，使用 $E_A^{\ominus}$。
- **碱性 Frost 图**（pH = 14）：标注为蓝线，使用 $E_B^{\ominus}$。
- 同一元素在不同 pH 下可能呈现完全不同的稳定性顺序。

## 六、适用条件与限制
- 同 Latimer 图：仅适用于**热力学判断**。
- 标准态假设：若实际条件偏离标准态，需用 Nernst 方程校正。
- 对复杂体系（多元素、多种配位环境）信息不足，需配合 Pourbaix 图。

## 七、常见比较与易混点

| 比较项 | Latimer 图 | Frost 图 |
|:---|:---|:---|
| 形式 | 一维线性链 | 二维折线图 |
| 横轴 | 氧化态（隐含） | 氧化数 $N$ |
| 纵轴 | 无 | $\nu E^{\ominus}$（∝ $-\Delta_f G^{\ominus}$） |
| 歧化判据 | 右 $E^{\ominus}$ > 左 $E^{\ominus}$ | 中间点在连线上方（凸形） |
| 氧化剂强弱 | 比较相邻 $E^{\ominus}$ 大小 | 比较线段斜率陡度 |
| 优势 | 精确读数、计算方便 | 直观、多氧化态比较一目了然 |

## 十、例题

### 例题 1
**题目**：根据锰在酸性溶液中的 Frost 图（图 5.10），判断 $Mn^{3+}$ 在酸性溶液中是否稳定。

**分析**：观察 $Mn^{3+}$（$N=+3$）相对于相邻物种 $Mn^{2+}$（$N=+2$）和 $MnO_2$（$N=+4$）的位置关系。

**解答**：$Mn^{3+}$ 的点位于 $Mn^{2+}$ 和 $MnO_2$ 连线的上方（凸形），故不稳定，发生歧化：
$$2Mn^{3+}(aq) + 2H_2O(l) \longrightarrow Mn^{2+}(aq) + MnO_2(s) + 4H^+(aq)$$

**反思**：Frost 图的"上方 = 不稳定"与日常直觉相反——学生容易记成"越高越稳定"，需要强调纵坐标是 $\nu E^{\ominus}$（∝ Gibbs 自由能），**越低（越负）越稳定**。

### 例题 2
**题目**：参考氮的 Frost 图（图 5.6），比较 $NO_3^-$ 在酸性和碱性溶液中作为氧化剂的强弱。

![[mineru/无机化学第6版Weller/无机化学第6版氧化还原反应180-212_images/3ec6f85bbab0ee45a207978a1437f5e6aff9e27d662bf16f38f982170797b526.jpg|500]]
> **图 5.6** 氮元素在酸性（pH=0，上曲线）和碱性（pH=14，下曲线）溶液中的 Frost 图。
> 横轴 -3 ~ +5 全氧化态，关键物种：$NH_4^+$（N=-3）、$N_2H_4$、$NH_2OH$、$N_2$（N=0）、$N_2O$、$NO$、$NO_2^-$、$NO_3^-$（N=+5）。

**分析**：比较 $NO_3^- \to$ 较低氧化态物种的线段斜率。斜率越陡，$E^{\ominus}$ 越大，氧化性越强。

**解答**：酸性图中 $NO_3^- \to NH_4^+$ 的线段斜率远大于碱性图中对应线段的斜率，因此 $NO_3^-$ 在酸性溶液中是更强的氧化剂。

**反思**：这与化学直觉一致——硝酸在酸性条件下是强氧化剂，而在中性/碱性条件下氧化性显著减弱。

## 十一、易错点
- 纵坐标含义混淆：$\nu E^{\ominus}$ 不是电位本身，而是"电位 × 电子数"；位置低 = 稳定，不是位置高 = 稳定。
- 斜率符号搞反：斜率为正意味着 $E^{\ominus} > 0$（还原自发），氧化型是氧化剂。
- 电子数 $\nu$ 的符号：当 $N < 0$ 时，$\nu$ 为负值（因为从 $N$ 到 0 是氧化过程，还原半反应的电子数为负）。
- 不同 pH 的图混用：必须看清图例标注的是 pH = 0 还是 pH = 14。

### 12.2 学生易踩的认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析口诀/方法 |
|---|---|---|---|
| "点越高越稳定" | 日常直觉"越高越好" | 纵坐标是 $\nu E^{\ominus}$ ∝ $-\Delta G$，越低（越负）Gibbs 自由能越小越稳定 | "低点安稳，高点危险" |
| "斜率大就是还原性强" | 混淆斜率与电位 | 斜率 = $E^{\ominus}$，是还原电位；斜率大说明氧化型是强氧化剂 | "陡坡氧化强，缓坡还原强" |

### 12.3 入门级例题
**题目**：某元素 X 的 Frost 图显示 $X^{2+}$ 的点位于 $X^{3+}$ 和 $X^+$ 连线的上方。判断 $X^{2+}$ 的稳定性。

**预期解答路径**：
1. 识别几何特征：点在连线上方 → 凸形
2. 应用判据：凸形 → 中间物种歧化
3. 结论：$X^{2+}$ 不稳定，歧化为 $X^{3+}$ 和 $X^+$

**老师可在课堂上引导的提问**：
- 学生若卡在第 1 步：问 "$X^{2+}$ 的点和连接两侧物种的直线相比，是在上面还是下面？"
- 学生若卡在第 2 步：问 "凸形和凹形分别对应什么反应类型？"

### 12.4 与现实/直觉的连接
- 类比：Frost 图就像元素的"能量地形图"，氧化态是地形上的不同海拔点，水（最稳定的氧化态）总是往低处流。
- 反例：硝酸铵 $NH_4NO_3$ 看起来是普通的化肥——但固态时它是爆炸物，因为 $NH_4^+$（N=-3）和 $NO_3^-$（N=+5）在 Frost 图上有强烈的反歧化趋势生成 $N_2O$。
