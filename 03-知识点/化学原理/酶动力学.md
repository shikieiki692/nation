# 酶动力学（Enzyme Kinetics）

## 一、定义

酶动力学研究酶催化反应的速率规律，是连接化学动力学与生物化学的桥梁。CChO 决赛中的酶动力学题目通常要求推导速率方程、分析抑制机制、或从实验数据计算动力学参数。

## 三、核心原理

### 3.1 Michaelis-Menten 机理

$$E + S \underset{k_{-1}}{\stackrel{k_1}{\rightleftharpoons}} ES \stackrel{k_2}{\rightarrow} E + P$$

- **$E$**：游离酶
- **$S$**：底物
- **$ES$**：酶-底物复合物
- **$P$**：产物

### 3.2 稳态近似推导

对 $ES$ 应用稳态近似：
$$\frac{d[ES]}{dt} = k_1[E][S] - k_{-1}[ES] - k_2[ES] = 0$$

结合物料守恒 $[E]_0 = [E] + [ES]$，得到：
$$v = \frac{V_{\max}[S]}{K_M + [S]}$$

其中：
- **$V_{\max} = k_2[E]_0$**：最大反应速率
- **$K_M = \frac{k_{-1} + k_2}{k_1}$**：米氏常数（$K_M$ 等于速率 $V_{\max}/2$ 时的底物浓度）

### 3.3 $K_M$ 的动力学意义

| 条件 | $K_M$ 的近似意义 |
|:---|:---|
| $k_2 \ll k_{-1}$ | $K_M \approx k_{-1}/k_1 = K_d^{-1}$，即 $ES$ 解离常数 |
| $k_2 \gg k_{-1}$ | $K_M$ 反映催化效率，不等于 $K_d$ |

## 四、关键结论

### 4.1 Lineweaver-Burk 双倒数作图

$$\frac{1}{v} = \frac{K_M}{V_{\max}} \cdot \frac{1}{[S]} + \frac{1}{V_{\max}}$$

- 纵轴截距：$1/V_{\max}$
- 横轴截距：$-1/K_M$
- 斜率：$K_M/V_{\max}$

### 4.2 酶抑制

**竞争性抑制**
- **抑制剂 $I$** 与底物竞争酶的活性位点
- $E + I \rightleftharpoons EI$
- 表观 $K_M^{\text{app}} = K_M(1 + [I]/K_i)$，$V_{\max}$ 不变
- 双倒数图：直线交于纵轴

**非竞争性抑制**
- $I$ 结合于酶的非活性位点，$ES$ 也能结合 $I$
- $V_{\max}^{\text{app}} = V_{\max}/(1 + [I]/K_i)$，$K_M$ 不变
- 双倒数图：直线交于横轴

**反竞争性抑制**
- $I$ 只与 $ES$ 结合
- $V_{\max}$、$K_M$ 同比例减小
- 双倒数图：平行线

### 4.3 催化效率

$$k_{\text{cat}} = \frac{V_{\max}}{[E]_0} = k_2$$
$$\frac{k_{\text{cat}}}{K_M}$$：催化效率的量度，上限约 $10^8 \sim 10^9\ \mathrm{M^{-1}s^{-1}}$（扩散控制）

## 十一、易错点

1. **$K_M$ 不是 $ES$ 的解离常数**：只有当 $k_2 \ll k_{-1}$ 时才近似等于 $K_d$
2. **$V_{\max}$ 是渐近值**：实际实验中很难达到真正的 $V_{\max}$
3. **双倒数图的误差放大**：$1/v$ 对 $1/[S]$ 作图会放大低底物浓度时的误差
4. **抑制类型的判断依据**：看 $V_{\max}$、$K_M$ 哪个变化，而不是死记图形

## 十三、竞赛拓展

1. **Lineweaver-Burk 作图与抑制类型判断**：双倒数图（$1/v_0$ vs $1/[S]$）是竞赛中区分酶抑制类型的标准工具。**竞争性抑制**（抑制剂与底物竞争活性位点）：$V_{\max}$ 不变，$K_m^{\mathrm{app}} = K_m(1 + [I]/K_I)$ 增大 → 直线斜率增大，y 截距不变（各线交于 y 轴）。**非竞争性抑制**（抑制剂与 ES 复合物结合，不干扰底物结合）：$V_{\max}^{\mathrm{app}} = V_{\max}/(1 + [I]/K_I)$，$K_m$ 不变 → y 截距和斜率均增大（各线交于 x 轴）。**反竞争性抑制**（抑制剂只与 ES 结合）：$V_{\max}$ 和 $K_m$ 按相同比例减小 → 直线平移（平行线）。

2. **稳态近似在酶动力学推导中的数学**：Briggs-Haldane 稳态处理：$d[ES]/dt = k_1[E][S] - k_{-1}[ES] - k_2[ES] = 0$，结合 $[E]_0 = [E] + [ES]$ 导出 $v_0 = \dfrac{k_2[E]_0[S]}{K_m + [S]}$，其中 $K_m = (k_{-1} + k_2)/k_1$。注意 $K_m$ 不一定是 $K_S = k_{-1}/k_1$（真正的解离常数）——只有当 $k_2 \ll k_{-1}$ 时 $K_m \approx K_S$。竞赛题可能给一系列 [S] 和 $v_0$ 数据，要求用 Lineweaver-Burk 或 Eadie-Hofstee 作图法求 $K_m$ 和 $V_{\max}$。

3. **核酶与抗体酶——超越蛋白质的催化**：核酶（ribozyme）是具催化活性的 RNA——证明酶不一定是蛋白质（Cech & Altman, 1989 Nobel）。核糖体的肽基转移酶活性中心完全由 rRNA 构成——rRNA 才是真正的催化剂，蛋白质只是"脚手架"。抗体酶（abzyme）：利用过渡态类似物作为抗原免疫动物，产生的抗体可催化特定反应——这是人工设计酶的开端。竞赛信息题可能给这些"非经典酶"作为前沿素材。
