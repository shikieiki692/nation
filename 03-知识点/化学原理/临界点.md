# 临界点 (Critical Point)

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[15-物质结构与状态]]

## 零、考纲定位
- 课堂定位：真实气体、相图与超临界流体之间的桥梁知识点
- 能力要求：会从 p-V 等温线和状态方程识别临界条件，理解“气液界限消失”的物理含义
- 常见去向：[[van der Waals方程]]、[[相图]]、[[超临界流体]]

## 一、定义

**临界点** 是 P-T 相图中气-液平衡线的终点，标志着气液两相可以共存的最高温度和最高压力。在临界点之上，物质处于 **超临界状态**，无法区分气态和液态。

- **临界温度 (T_c)**：在该温度以上，无论多大压力都不能使气体液化
- **临界压力 (P_c)**：在临界温度下使气体液化所需的最小压力
- **临界密度 (ρ_c)**：临界点处的密度，通常介于气液密度之间

## 二、物理意义

1. **气液消失**：在临界点，液相和气相的密度相等，表面张力为零
2. **临界乳光**：临界点附近强密度涨落导致强烈散射光的现象
3. **van der Waals 方程预测**：vdW 方程给出的临界参数为：T_c = 8a/(27Rb)，P_c = a/(27b²)，V_c = 3b
4. **对比状态**：R_T = T/T_c，R_P = P/P_c，R_V = V/V_c — 压缩因子 Z_c = 0.375(vdW)

## 四、教学常见节点

- **临界参数的实验值与 vdW 理论值对比**：实际 Z_c ≈ 0.27，vdW 预测 0.375
- **对比状态原理**：不同物质在相同对比态下性质相似
- **P-V 等温线**：T < T_c 有水平段（相变），T = T_c 有拐点，T > T_c 光滑
- **竞赛考查**：从 P-V 等温线判断是否达到临界温度、从 vdW 参数推算临界常数

## 五、竞赛拓展

- **超临界 CO₂ 萃取**：在 T_c = 31.1°C, P_c = 7.38 MPa 条件下的应用
- **临界参数的统计力学解释**：分子间相互作用的宏观体现
- **对应状态方程**：基于对比态的普适性状态方程

- 题型-由 vdW 参数求临界常数

- 题型-比较理想气体与真实气体的偏差来源

## 八、例题
**例题**：已知某气体满足 van der Waals 方程，其参数为 $a,b$。求临界温度 $T_c$ 和临界压力 $P_c$。  
**思路**：临界点处等温线满足
$$\left(\frac{\partial P}{\partial V}\right)_T=0,\qquad \left(\frac{\partial^2 P}{\partial V^2}\right)_T=0$$
联立可得 $V_c=3b$，进一步得到
$$T_c=\frac{8a}{27Rb},\qquad P_c=\frac{a}{27b^2}$$
这是“临界点 = p-V 曲线拐点”的标准计算模板。

## 九、易错点
- 把“临界温度以上不能液化”误解为“分子间作用完全消失”
- 把临界点和三相点混淆：三相点是三相共存，临界点是气液界面终止
- 只会背公式，不理解临界条件来自等温线拐点

## 十、教学视角
- 第一轮建议把临界点放在真实气体之后讲，用“CO₂ 为什么既不像普通气体也不像普通液体”做直觉承接
- 板书上最好同时给出 p-T 图和 p-V 图，让学生建立“相图终点”和“等温线拐点”两种表征的对应关系
- 若课时有限，可弱化统计力学表述，保留临界参数、超临界流体和真实气体偏差三块
