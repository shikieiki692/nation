# 化学平衡

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[07-化学平衡]]

## 一、定义
**化学平衡（Chemical Equilibrium）**：可逆反应中，正反应和逆反应速率相等，体系组成不再随时间变化的动态状态。
$$
\text{a A + b B} \;\overset{k_+}{\underset{k_-}{\rightleftharpoons}}\; \text{c C + d D}
$$
- **正向速率** $v_+$ = **逆向速率** $v_-$（动态平衡，非静止）
- 体系中各物种浓度（或分压）保持恒定 $[A], [B], [C], [D] = $ 常数
- 此时反应商 $Q = $ 平衡常数 $K$

## 三、核心原理

### 平衡的两个等价判据
1. **动力学判据**：$v_+ = v_-$（净反应速率为 0）——中学阶段的认识
2. **热力学判据**：$\Delta G = 0$，即 $\Delta G° = -RT\ln K°$——大学化学的深化视角

> **从动力学到热力学的认知升级**（张丽荣《无机化学》§4.1）：中学从"速率相等"理解平衡，但这只是表象。热力学揭示的更深层本质是：反应进行过程中 $\left|\Delta G\right|$ 逐渐趋近于零，当 $\Delta G=0$ 时，反应不再以不可逆方式向某一方向进行，正逆反应同时以可逆方式继续，体系达成平衡。所有化学反应原则上都有可逆性，只是程度不同——如 $\mathrm{NH_3 + HCl \rightleftharpoons NH_4Cl}$ 在常温下可逆程度极小，而 $\mathrm{PCl_5 \rightleftharpoons PCl_3 + Cl_2}$ 可逆程度显著。

### 平衡常数（[[平衡常数]]）
对反应 $a\text{A} + b\text{B} \rightleftharpoons c\text{C} + d\text{D}$：
$$K_c = \frac{[C]^c[D]^d}{[A]^a[B]^b}$$
$$K_p = \frac{p_C^c \cdot p_D^d}{p_A^a \cdot p_B^b}$$
$$K° = \prod a_i^{\nu_i}$$（标准平衡常数，无量纲）

### 反应商 Q 与平衡的关系
$$Q = \frac{[C]_t^c[D]_t^d}{[A]_t^a[B]_t^b}$$
- $Q < K$：正反应自发（产物生成）
- $Q = K$：平衡（净速率 = 0）
- $Q > K$：逆反应自发（反应物生成）

### 影响平衡的因素（[[Le Chatelier 原理]]）
"若改变影响平衡的某个条件，平衡将向能够减弱这一改变的方向移动"
| 因素 | 平衡移动方向 |
|---|---|
| 增大反应物浓度 | 正向（消耗反应物） |
| 减小产物浓度 | 正向（生成产物补充） |
| 升温（吸热反应） | 正向（向吸热方向，[H⁺] 系统冷却） |
| 升温（放热反应） | 逆向（向吸热方向） |
| 加压（气体减少方向）| 气体物质量小的一侧（产物或反应物） |
| 加入催化剂 | **不移动平衡**，只改变到达时间 |
| 加入惰性气体（V 恒）| 不移动平衡 |
| 加入惰性气体（p 恒）| 等价于减压 |

## 四、关键结论

### 平衡的特征（"五特征"）
| 特征 | 含义 |
|---|---|
| **动态** | $v_+ = v_- \neq 0$，分子层面仍有反应 |
| **可逆** | 化学反应可向两个方向进行 |
| **限度** | 反应不会进行到完全；体系组成达到极限值 |
| **条件性** | 温度、压强、浓度变化会打破平衡 |
| **同一性** | 从正/逆方向出发，最终达到的平衡组成相同 |

### 平衡判断的"实用记忆"
- 浓度/分压不再变化 → 平衡
- $v_+ = v_-$ → 平衡
- $Q = K$ → 平衡
- $\Delta G = 0$ → 平衡

### 速率常数与平衡常数的联系
$$K = \frac{k_+}{k_-}$$
- $k_+$ 是正向速率常数，$k_-$ 是逆向速率常数
- 当温度升高，若正反应吸热，则 $k_+/k_-$ 增大 → $K$ 增大 → 平衡正移

### 平衡常数与 ΔG° 关系
$$\Delta G° = -RT\ln K°$$
| $\Delta G°$ | $K°$ | 含义 |
|:---:|:---:|---|
| $< 0$ | $> 1$ | 标准态下正向自发 |
| $= 0$ | $= 1$ | 标准态下恰好平衡 |
| $> 0$ | $< 1$ | 标准态下逆向自发 |

## 五、常见分类或情形

### 按反应物相分类
| 类型 | 实例 |
|---|---|
| 气相平衡 | $N_2 + 3H_2 \rightleftharpoons 2NH_3$ |
| 液相平衡 | $\text{H}^+ + \text{OH}^- \rightleftharpoons \text{H}_2\text{O}$ |
| 多相平衡 | $\text{CaCO}_3(s) \rightleftharpoons \text{CaO}(s) + \text{CO}_2(g)$ |
| 溶解平衡 | $\text{AgCl}(s) \rightleftharpoons \text{Ag}^+ + \text{Cl}^-$ |
| 配位平衡 | $\text{Fe}^{3+} + \text{SCN}^- \rightleftharpoons \text{Fe(SCN)}^{2+}$ |

### 按平衡常数表达分类
| K 类型 | 适用 | 是否无量纲 |
|---|---|---|
| $K_c$ | 浓度平衡 | 有单位（除非 $\Delta\nu = 0$）|
| $K_p$ | 气相分压 | 有单位 |
| $K°$ | 标准平衡 | 无量纲（活度比） |
| $K_{sp}$ | 难溶电解质 | 有单位 |
| $K_a, K_b$ | 弱酸/弱碱解离 | 有单位 |
| $K_w$ | 水自解离 | 有单位（10⁻¹⁴ mol²/L²） |

### 按反应方向的平衡常数
- $K_+$ 表正向，$K_-$ 表逆向
- 关系：$K_+ \cdot K_- = 1$
- 反应×n：$K_{(n)} = K_{(1)}^n$
- 反应相加：$K_{\text{total}} = K_1 \cdot K_2$

### 风化与潮解 —— 平衡常数的实际应用
复相反应的平衡常数可以定量解释**风化**和**潮解**现象。以 $\mathrm{Na_2SO_4 \cdot 10H_2O(s) \rightleftharpoons Na_2SO_4(s) + 10H_2O(g)}$ 为例：

- $K° = [p(\mathrm{H_2O})/p°]^{10}$，273 K 时 $K° = 4.08 \times 10^{-25}$
- 解得平衡水蒸气压 $p(\mathrm{H_2O}) = 0.364\ \mathrm{kPa}$
- 若环境 $\mathrm{H_2O}$ 分压 < 0.364 kPa → $Q < K°$ → 正向（风化）
- 若环境 $\mathrm{H_2O}$ 分压 > 0.364 kPa → $Q > K°$ → 逆向（潮解）
- 273 K 时水的饱和蒸气压为 0.611 kPa，故**相对湿度 < 59.6% 即发生风化**

> 这一模型是理解"为什么某些盐在干燥空气中失水"的定量基础。

## 六、适用条件与限制
1. **必须是可逆反应**：不可逆反应不存在平衡（如燃烧、强酸碱中和）
2. **封闭体系**：开放体系（不断进/出物质）不达平衡
3. **恒温**：温度变化打破平衡
4. **理想体系/稀溶液**：活度近似浓度
5. **达到平衡需时间**：动力学决定快慢

## 七、常见比较与易混点

### 化学平衡 vs 相平衡
| 项目 | 化学平衡 | 相平衡 |
|---|---|---|
| 过程 | 化学键断开/形成 | 物相变化（液/气/固） |
| 描述 | 平衡常数 K | 蒸气压、沸点等 |
| 实例 | $N_2 + 3H_2 \rightleftharpoons 2NH_3$ | $H_2O(l) \rightleftharpoons H_2O(g)$ |

### 化学平衡 vs 稳态
- **平衡**：$v_+ = v_-$，$\Delta G = 0$，热力学可达极限
- **稳态**（kinetic steady state）：中间产物 $v_{\text{生成}} = v_{\text{消耗}}$，但整体反应仍向前
- 平衡是稳态的特例

### Q < K vs Q > K
- $Q < K$：反应物多/产物少 → 正向自发（向 Q 增大方向）
- $Q > K$：产物多/反应物少 → 逆向自发（向 Q 减小方向）
- 用 $\Delta G = RT\ln(Q/K)$ 严格表达

### 改变 K 还是只移动平衡？
- **只温度**改变 K
- 浓度、压强、催化剂**不改变 K**（K 是温度的函数）
- 改变浓度/压强 → 改变 Q → 推动平衡移动

## 十、例题

### 例题 1：合成氨平衡
**题目：** 在 500°C 时，反应 $N_2(g) + 3H_2(g) \rightleftharpoons 2NH_3(g)$ 的 $K_p° = 1.5 \times 10^{-5}$。某时刻 $p(N_2) = 0.5, p(H_2) = 1.0, p(NH_3) = 0.2$ bar，判断反应方向。

**分析：** 比较 $Q$ 与 $K°$。

**解答：**
- $Q = \frac{(0.2)^2}{0.5 \times 1.0^3} = 0.08$
- $K° = 1.5 \times 10^{-5}$
- $Q > K°$ → **逆向反应自发**（NH₃ 分解）

**反思：** 工业上必须设法降低 Q（移走 NH₃），才能让反应继续正向。

### 例题 2：Le Chatelier 应用
**题目：** 反应 $2NO_2(g) \rightleftharpoons N_2O_4(g), \Delta H = -57$ kJ/mol。下列措施使平衡如何移动？
1. 升温
2. 加压
3. 加入 NO₂
4. 加催化剂

**解答：**
1. 升温（吸热方向，逆向）：平衡**逆移**，NO₂ 增多（颜色变深红）
2. 加压（气体减少方向，2→1，正向）：平衡**正移**
3. 加入 NO₂（消耗反应物方向）：平衡**正移**
4. 催化剂：**不移动**，只加速到达

**反思：** Le Chatelier 是定性工具，可与定量 K 计算配合使用。

### 勒夏特列原理的适用边界
- 勒夏特列原理是**定性**工具，不适用于所有情况
- **经典反例**（习题4-23）：某反应 $\Delta_r H° > 0, \Delta_r S° < 0$。按勒夏特列原理，升温→平衡向吸热方向（右移）。但 $\Delta_r G° = \Delta_r H° - T\Delta_r S°$，因 $\Delta_r S° < 0$，升温使 $\Delta_r G°$ **增大**（更不利于正向）。哪个正确？**ΔG° 增大≠平衡左移**——标准态 ΔG° 变化与平衡移动方向不可混淆，勒夏特列原理在此仍然成立（K°随T升高而增大因为 $\ln K° = -\Delta G°/RT$，分母T增大更快）
- **KOH 溶解度矛盾**（习题4-24）：KOH 溶解放热（ΔsolH<0），但溶解度随温度升高而增大（违反勒夏特列预期）。原因：溶解涉及复杂的离子水合熵变，不能简单套用气-固相变的勒夏特列直觉。

## 十一、易错点
- ❌ 错：平衡状态浓度不变 = 反应停止——平衡是动态的，正逆反应仍在
- ❌ 错：加催化剂能让 K 增大——催化剂只改变速率，不改变 K
- ❌ 错：加压一定能让平衡正移——只对气体减少的反应有效
- ❌ 错：平衡常数只与浓度有关——K 是 T 的函数，浓度变化不改变 K
- ❌ 错：Q = K 时 ΔG = 0 但反应停止——动态平衡是"看不见的平衡"
- ❌ 错：$\Delta_r G°$ 增大则平衡必左移——标准态 ΔG° 与平衡移动方向不可混淆；勒夏特列原理判断的是 **K° 的变化趋势**，而非 ΔG° 的绝对值变化

|---|---|---|---|

### 12.3 入门级例题
**题目**：$\text{H}_2 + \text{I}_2 \rightleftharpoons 2\text{HI}$ 在 700 K 下 $K = 50$。起始 $[H_2] = [I_2] = 1.0$ M, $[HI] = 0$。求平衡浓度。

**预期解答路径**：
1. 设转化 $x$ M H₂，则 [H₂] = [I₂] = 1-x, [HI] = 2x
2. $K = (2x)^2 / [(1-x)(1-x)] = 50$
3. $2x/(1-x) = \sqrt{50} = 7.07$
4. $2x = 7.07 - 7.07x \to x = 0.78$
5. [H₂] = [I₂] = 0.22 M, [HI] = 1.56 M

**老师可在课堂上引导的提问**：
- 学生若卡住：用 ICE 表（Initial-Change-Equilibrium）规范步骤
- 学生若答错：检查反应化学计量比 (2 mol HI 对应 1 mol H₂)

### 12.4 与现实/直觉的连接
- 类比：化学平衡像"两个银行同时存取款"——存款（正向）= 取款（逆向），余额不变但钱在动
- 反例：燃烧反应（如 CH₄ + 2O₂ → CO₂ + 2H₂O）几乎不可逆，K 极大，称"完全反应"

## 十三、竞赛拓展
- **van't Hoff 方程**：$\ln(K_2/K_1) = -\Delta H°/R \cdot (1/T_2 - 1/T_1)$
- **多重平衡的耦合常数**：$K_{net} = K_1 \cdot K_2 \cdot ...$
- **响应函数**：$\partial(\ln K)/\partial T$ 等热力学一阶量
- **绝缘体系的化学振荡**（如 BZ 反应）——远离平衡的奇特动力学
