# 物料守恒

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 关联笔记：[[电荷守恒]], [[质子守恒]], [[化学计量]], [[容量分析计算]]

## 一、定义

物料守恒（Mass Balance / Material Balance）是指在一个封闭化学体系中，某一元素或某一组分的总物质的量在反应前后保持不变。它是化学计算中最基本的守恒关系之一，常与电荷守恒、质子守恒联合使用，解决复杂平衡体系中的浓度计算问题。

- 对应考纲条目：[[04-热力学第一定律]]、[[08-化学反应与方程式]]、[[09-水溶液中的离子平衡]]
- 所属模块：[[基础要求-化学原理]]
- 本知识点在考纲中的作用：物料守恒与电荷守恒、质子守恒并称"三大守恒"，是电解质溶液计算（pH、缓冲、水解、沉淀溶解平衡、分布系数）的核心工具。初赛溶液平衡题几乎必用物料守恒建立方程。

## 三、核心原理

### 3.1 基本形式

对于某一元素 X，若其在体系中可能以多种形式存在（如游离离子、配合物、沉淀等），则：

$$c_{\text{X,总}} = \sum_i c_{\text{X}_i}$$

其中 $c_{\text{X}_i}$ 为 X 在第 i 种物种中的浓度。

### 3.2 典型应用场景

| 场景 | 物料守恒表达式 | 用途 |
|------|---------------|------|
| 弱酸溶液 | $c_{\text{HA}} = [\text{HA}] + [\text{A}^-]$ | 计算各型体分布 |
| 多元弱酸 | $c_{\text{H}_n\text{A}} = [\text{H}_n\text{A}] + [\text{H}_{n-1}\text{A}^-] + \dots + [\text{A}^{n-}]$ | 分布系数计算 |
| 缓冲溶液 | $c_{\text{酸}} + c_{\text{碱}} = [\text{HA}] + [\text{A}^-]$ | pH 精确计算 |
| 配合物体系 | $c_{\text{M}} = [\text{M}] + [\text{ML}] + [\text{ML}_2] + \dots$ | 配位平衡计算 |
| 滴定分析 | $n_{\text{被测物}} = n_{\text{标准溶液}} \times \text{化学计量比}$ | 滴定结果计算 |

### 3.3 与电荷守恒、质子守恒的关系

在酸碱平衡计算中，三大守恒常联立使用：

- **电荷守恒（CCE）**：溶液中正负电荷总量相等
  $$[\text{H}^+] + [\text{Na}^+] = [\text{OH}^-] + [\text{A}^-]$$

- **物料守恒（MBE）**：某元素各型体浓度之和等于初始浓度
  $$c = [\text{HA}] + [\text{A}^-]$$

- **质子守恒（PBE）**：得质子产物与失质子产物的浓度关系（由 CCE 和 MBE 推导）
  $$[\text{H}^+] + [\text{HA}] = [\text{OH}^-] + c_{\text{碱}}$$

### 3.4 竞赛中的快速技巧

- **分布系数法**：利用 $\delta_i$ 直接计算各型体浓度，避免解高次方程
- **化学计量点分析**：滴定终点时物料守恒关系最简洁
- **元素质量守恒**：燃烧分析、化学式推断中的核心工具

## 四、关键结论

1. **物料守恒（MBE）的本质**：某一组分的总浓度等于该组分所有存在形式的平衡浓度之和，并按化学计量系数加权：
   $$c_{\text{总}} = \sum_i n_i [\text{X}_i]$$
   其中 $n_i$ 为该形态中目标原子/原子团的个数。

2. **AB型电解质**：对于单一AB型电解质（如NaCl、KNO₃），阳离子和阴离子各自独立列MBE：
   $$c = [\text{A}^+] = [\text{B}^-]$$

3. **非1:1电解质注意系数**——以 $\mathrm{Na_2CO_3}$ 为例：
   $$c(\mathrm{Na^+}) = 2[\mathrm{Na^+}]$$
   $$c(\mathrm{CO_3^{2-}总}) = [\mathrm{CO_3^{2-}}] + [\mathrm{HCO_3^-}] + [\mathrm{H_2CO_3}]$$
   钠元素出现在一种形态中但化学式中含2个Na，故系数为2；碳酸根的碳原子则分散到三种形态。

4. **MBE与CCE的根本区别**：[[电荷守恒|CCE]]关注的是电荷层面——正负电荷总量相等；MBE关注的是原子/原子团层面——元素的总量守恒。两者维度不同，不可混淆。

5. **MBE的数量规则**：体系中独立组分的数量 = 所需MBE方程的数量。例如 $\mathrm{Na_2CO_3}$ 溶液需列出Na和 $\mathrm{CO_3^{2-}}$ 两个MBE方程。

6. **列出MBE的核心技巧**：追踪目标原子"去了哪里"。以碳酸根为例——碳原子进入溶液后分布在 $\mathrm{CO_3^{2-}}$、$\mathrm{HCO_3^-}$、$\mathrm{H_2CO_3}$ 三种形式中，因此MBE必须覆盖这三种形态。

## 五、常见分类或情形

1. **单一强电解质（最简单情形）**
   - 例：0.1 mol/L NaCl → $c(\mathrm{Na^+}) = [\mathrm{Na^+}] = 0.1$，$c(\mathrm{Cl^-}) = [\mathrm{Cl^-}] = 0.1$
   - 若发生水解或配合，形式增多，但仍以总浓度列式。

2. **弱酸/弱碱溶液（含分子态）**
   - 例：$\mathrm{HAc}$ 溶液中，HAc部分电离：
   $$c(\mathrm{HAc总}) = [\mathrm{HAc}] + [\mathrm{Ac^-}]$$
   - 关键：分子态HAc也必须计入MBE，不能只算离子。

3. **多元酸/碱（多级分布）**
   - 例：$\mathrm{H_3PO_4}$ 溶液：
   $$c(\mathrm{H_3PO_4总}) = [\mathrm{H_3PO_4}] + [\mathrm{H_2PO_4^-}] + [\mathrm{HPO_4^{2-}}] + [\mathrm{PO_4^{3-}}]$$
   - 又例：$\mathrm{Na_2CO_3}$ 溶液：
   $$c(\mathrm{CO_3^{2-}总}) = [\mathrm{CO_3^{2-}}] + [\mathrm{HCO_3^-}] + [\mathrm{H_2CO_3}]$$

4. **混合溶液（各自独立列MBE）**
   - 例：$\mathrm{HAc + NaAc}$ 缓冲溶液：
     - $c(\mathrm{Na^+}) = [\mathrm{Na^+}]$
     - $c(\mathrm{Ac总}) = [\mathrm{HAc}] + [\mathrm{Ac^-}] = c_{\mathrm{HAc}} + c_{\mathrm{NaAc}}$
   - 每种加入的组分各列一个MBE。

5. **配位/沉淀共存体系**
   - 例：$\mathrm{Ag^+} + \mathrm{NH_3}$ 体系：
   $$c(\mathrm{Ag^+总}) = [\mathrm{Ag^+}] + [\mathrm{AgNH_3^+}] + [\mathrm{Ag(NH_3)_2^+}]$$
   $$c(\mathrm{NH_3总}) = [\mathrm{NH_3}] + [\mathrm{AgNH_3^+}] + 2[\mathrm{Ag(NH_3)_2^+}]$$
   - 注意：$\mathrm{Ag(NH_3)_2^+}$ 含2个 $\mathrm{NH_3}$，故其浓度须乘以2。

## 六、适用条件与限制

1. **适用条件——封闭体系**
   - MBE严格适用于封闭体系——无物质交换，初始加入量已知。
   - 对于开放体系（如 $\mathrm{CO_2}$ 可逸出），MBE需修正或不能直接使用。

2. **系数由化学式决定**
   - 系数来源于化学式中的原子个数：$\mathrm{Na_2SO_4}$ 中Na的系数为2，$\mathrm{Al_2(SO_4)_3}$ 中Al的系数为2、$\mathrm{SO_4^{2-}}$ 的系数为3。
   - 系数与电荷数、化合价无关。

3. **忽略极微量物种的近似**
   - 在某些pH条件下，某些形态浓度极低（如强碱性溶液中 $[\mathrm{H_2CO_3}] \approx 0$），可近似忽略以简化计算。
   - 但忽略操作必须基于定量判断（$\delta_i < 1\%$），否则引入不可忽略的误差。

4. **配合物体系须列全所有配合形态**
   - 金属离子与配体形成的各级配合物（$\mathrm{ML}, \mathrm{ML_2}, \dots, \mathrm{ML_n}$）均须纳入MBE。
   - 漏掉某一级配合物将导致总浓度偏低，进而使平衡计算出错。

5. **MBE不直接涉及反应程度**
   - MBE描述的是"总量"关系，不直接反映各形态的相对多少（那是[[分布系数]]的任务）。
   - MBE + 平衡常数 → 才能求解各形态的具体浓度。

## 七、常见比较与易混点

1. **MBE vs CCE（物质守恒 vs 电荷守恒）**
   - MBE：追踪某一特定原子/原子团，列出其所有存在形式。
   - [[电荷守恒|CCE]]：溶液中正电荷总量 = 负电荷总量，关注的是离子的电荷数。
   - 易混：$\mathrm{Na_2CO_3}$ 的MBE是 $c(\mathrm{Na}) = 2[\mathrm{Na^+}]$，而CCE中 $\mathrm{Na^+}$ 只贡献 $+1$ 电荷项。学生容易把MBE中的系数"2"误解为电荷数。

2. **MBE vs PBE（原子团守恒 vs 质子守恒）**
   - [[质子守恒|PBE]]反映的是质子（$\mathrm{H^+}$）在体系中的得失关系。
   - MBE关注的是某一元素的"总量"，PBE关注的是质子的"净转移"。
   - 实际上PBE可由MBE和CCE联立推导得到。

3. **$\mathrm{Na_2CO_3}$ 中 $c(\mathrm{Na}) = 2c$ ——为什么有系数2？**
   - **学生常错**：写成 $c(\mathrm{Na}) = [\mathrm{Na^+}]$，忽略了1个 $\mathrm{Na_2CO_3}$ 含2个Na。
   - 正确理解：配制 0.1 mol/L $\mathrm{Na_2CO_3}$ 溶液时，$\mathrm{Na^+}$ 的分析浓度为 0.2 mol/L，故 $c(\mathrm{Na}) = 2 \times 0.1 = 0.2$。

4. **$\mathrm{H_2SO_4}$ 的MBE特殊处理**
   - $\mathrm{H_2SO_4}$ 一级完全电离、二级部分电离（$K_{a2} \approx 1.0 \times 10^{-2}$）。
   - MBE：$c(\mathrm{SO_4^{2-}总}) = [\mathrm{HSO_4^-}] + [\mathrm{SO_4^{2-}}]$
   - 注意：$[\mathrm{H_2SO_4}]$ 分子态浓度可忽略（一级完全电离），不同于普通弱酸。

5. **"浓度" vs "分析浓度"**
   - **分析浓度 $c$**：按配制时加入量计算的初始总浓度（如"配了0.1 mol/L的HAc"）。
   - **平衡浓度 $[\ ]$**：体系达到平衡后各形态的实际浓度。
   - MBE是用平衡浓度 $[\ ]$ 表达、等于分析浓度 $c$ 的等式。

6. **MBE等式的"左边"是什么？**
   - 左边是分析浓度（乘以化学计量系数），右边是各形态平衡浓度之和。
   - 学生有时把 $[\mathrm{Na^+}]$ 写在左边、$c$ 写在右边，逻辑上等价但形式上建议统一。

## 八、与其他知识点的联系

- **前置知识**：[[溶液浓度]]（分析浓度与平衡浓度的区分）、[[化学计量]]（系数来源）、[[酸碱理论]]（理解质子转移导致的多形态分布）。
- **并行知识**：[[电荷守恒]]（CCE，常与MBE联立求解）、[[质子守恒]]（PBE，由MBE + CCE推导）、[[分布系数]]（$\delta_i$ 将MBE中的各形态浓度与pH关联）、[[缓冲溶液]]（MBE是缓冲溶液精确pH计算的基础）。
- **后续知识**：[[配位平衡]]（配合物MBE + 累积稳定常数 $\beta$）、[[沉淀平衡]]（MBE + $K_{sp}$ 求溶解度）、[[滴定分析]]（化学计量点处MBE最简洁）、[[容量分析计算]]（滴定结果由MBE关系导出）。

## 九、典型题型

1. **弱酸各型体浓度计算**：已知 pH 和总浓度，求 [HA] 和 [A⁻]
2. **滴定突跃计算**：化学计量点时各物种浓度
3. **多平衡综合计算**：酸碱 + 配位 + 沉淀共存体系
4. **燃烧法化学式推断**：由 CO₂、H₂O 质量守恒求实验式

## 十、例题

### 例题1 ★★ | 0.1 mol/L $\mathrm{NaHCO_3}$ 溶液的MBE

**分析**：$\mathrm{NaHCO_3}$ 在溶液中完全电离为 $\mathrm{Na^+}$ 和 $\mathrm{HCO_3^-}$。$\mathrm{HCO_3^-}$ 既能电离（$\to \mathrm{CO_3^{2-}} + \mathrm{H^+}$）又能水解（$\to \mathrm{H_2CO_3} + \mathrm{OH^-}$），因此含碳物种有三种形态。

**解答**：
- Na元素：$c(\mathrm{Na}) = [\mathrm{Na^+}] = 0.1\ \mathrm{mol/L}$
- $\mathrm{HCO_3^-}$ 中的碳：$c(\mathrm{HCO_3^-总}) = [\mathrm{H_2CO_3}] + [\mathrm{HCO_3^-}] + [\mathrm{CO_3^{2-}}] = 0.1\ \mathrm{mol/L}$

**反思**：$\mathrm{NaHCO_3}$ 中 $\mathrm{Na^+}$ 和 $\mathrm{HCO_3^-}$ 的分析浓度都是 0.1 mol/L。含碳物种的MBE与 $\mathrm{Na_2CO_3}$ 的形式完全相同——因为碳原子在溶液中的存在形式只取决于pH，与初始加入的是 $\mathrm{CO_3^{2-}}$ 还是 $\mathrm{HCO_3^-}$ 无关。关键是把"碳原子去了哪里"追踪完备。

### 例题2 ★★★ | 0.1 mol/L $\mathrm{H_3PO_4}$ 的MBE与分布系数计算各形态浓度（pH = 4.0）

**分析**：$\mathrm{H_3PO_4}$ 是三元酸，$\mathrm{p}K_{a1} = 2.12$，$\mathrm{p}K_{a2} = 7.20$，$\mathrm{p}K_{a3} = 12.36$。pH = 4.0时，须先用[[分布系数]]公式计算各 $\delta_i$，再乘以总浓度 0.1 mol/L 得到各形态浓度。

**解答**：
- MBE：$c(\mathrm{H_3PO_4总}) = [\mathrm{H_3PO_4}] + [\mathrm{H_2PO_4^-}] + [\mathrm{HPO_4^{2-}}] + [\mathrm{PO_4^{3-}}] = 0.1$
- 由分布系数公式：
  $$\delta_0 = \frac{[\mathrm{H^+}]^3}{D},\ \delta_1 = \frac{K_{a1}[\mathrm{H^+}]^2}{D},\ \delta_2 = \frac{K_{a1}K_{a2}[\mathrm{H^+}]}{D},\ \delta_3 = \frac{K_{a1}K_{a2}K_{a3}}{D}$$
  其中 $D = [\mathrm{H^+}]^3 + K_{a1}[\mathrm{H^+}]^2 + K_{a1}K_{a2}[\mathrm{H^+}] + K_{a1}K_{a2}K_{a3}$
- pH = 4.0，$[\mathrm{H^+}] = 1.0 \times 10^{-4}\ \mathrm{mol/L}$：
  - $\delta_0 \approx 1.3 \times 10^{-3}$ → $[\mathrm{H_3PO_4}] \approx 1.3 \times 10^{-4}\ \mathrm{mol/L}$
  - $\delta_1 \approx 0.987$ → $[\mathrm{H_2PO_4^-}] \approx 9.87 \times 10^{-2}\ \mathrm{mol/L}$
  - $\delta_2 \approx 1.3 \times 10^{-2}$ → $[\mathrm{HPO_4^{2-}}] \approx 1.3 \times 10^{-3}\ \mathrm{mol/L}$
  - $\delta_3 \approx 0$ → $[\mathrm{PO_4^{3-}}] \approx 0$

**反思**：MBE保证四形态浓度之和为 0.1 mol/L。分布系数 $\delta_1 \approx 0.987$ 表明 pH = 4.0 时 $\mathrm{H_2PO_4^-}$ 占绝对优势。四个 $\delta$ 之和必须为 1——这是检验计算正确性的快速方法。变换pH值可快速评估不同条件下的主导形态（见[[分布系数]]的优势区域图）。

### 例题3 ★★★★ | AgCl在0.1 mol/L $\mathrm{NH_3}$ 中的溶解度

已知 $K_{sp}(\mathrm{AgCl}) = 1.8 \times 10^{-10}$，$\mathrm{Ag^+}$-$\mathrm{NH_3}$ 配合物的累积稳定常数 $\beta_1 = 10^{3.24}$，$\beta_2 = 10^{7.05}$。

**分析**：AgCl溶解产生的 $\mathrm{Ag^+}$ 与 $\mathrm{NH_3}$ 配合，设溶解度为 $s\ \mathrm{mol/L}$，需同时满足MBE、[[配位平衡]]和[[沉淀平衡]]。

**解答**：
- MBE——Ag元素：$c(\mathrm{Ag^+总}) = [\mathrm{Ag^+}] + [\mathrm{AgNH_3^+}] + [\mathrm{Ag(NH_3)_2^+}] = s$
- MBE——$\mathrm{NH_3}$：$c(\mathrm{NH_3总}) = [\mathrm{NH_3}] + [\mathrm{AgNH_3^+}] + 2[\mathrm{Ag(NH_3)_2^+}] \approx 0.1$（因 $s$ 较小，耗 $\mathrm{NH_3}$ 可忽略 → $[\mathrm{NH_3}] \approx 0.1$）
- 配位平衡：$[\mathrm{AgNH_3^+}] = \beta_1[\mathrm{Ag^+}][\mathrm{NH_3}]$，$[\mathrm{Ag(NH_3)_2^+}] = \beta_2[\mathrm{Ag^+}][\mathrm{NH_3}]^2$
- 代入MBE：$s = [\mathrm{Ag^+}](1 + \beta_1[\mathrm{NH_3}] + \beta_2[\mathrm{NH_3}]^2)$
- 由 $K_{sp} = [\mathrm{Ag^+}][\mathrm{Cl^-}]$ 且 $[\mathrm{Cl^-}] = s$，得 $[\mathrm{Ag^+}] = K_{sp}/s$
- 联立：$s = \frac{K_{sp}}{s}(1 + \beta_1[\mathrm{NH_3}] + \beta_2[\mathrm{NH_3}]^2)$ → $s = \sqrt{K_{sp}(1 + \beta_1[\mathrm{NH_3}] + \beta_2[\mathrm{NH_3}]^2)}$
- 代入 $[\mathrm{NH_3}] = 0.1$：$\alpha_{\mathrm{Ag(NH_3)}} = 1 + 10^{3.24} \times 0.1 + 10^{7.05} \times 0.01 \approx 1.12 \times 10^5$
- $s = \sqrt{1.8 \times 10^{-10} \times 1.12 \times 10^5} \approx 4.5 \times 10^{-3}\ \mathrm{mol/L}$

**反思**：此题展示MBE + 配位平衡 + 沉淀平衡的联合使用。"总浓度变换法"（$c_M = [M] \cdot \alpha_M$）将MBE化为简洁形式。注意 $\mathrm{Ag(NH_3)_2^+}$ 中 $\mathrm{NH_3}$ 的系数为2——这是MBE中系数由化学式决定的一个关键点。若不考虑配合，AgCl在纯水中溶解度仅 $1.34 \times 10^{-5}$ mol/L，$\mathrm{NH_3}$ 的存在使溶解度增大约300倍。

## 十一、易错点

1. **忘记化学计量系数**
   - **典型错误**：$\mathrm{Na_2SO_4}$ 溶液的MBE写成 $c(\mathrm{Na}) = [\mathrm{Na^+}]$。
   - **正确**：$c(\mathrm{Na}) = 2[\mathrm{Na^+}]$，因为1 mol $\mathrm{Na_2SO_4}$ 含2 mol Na。
   - **学生为什么会这么想**：直觉上认为"总浓度等于该离子浓度"，忽略了化学式中原子个数的系数。教学中应强调"追踪的是原子数，不是离子数"。

2. **多元酸漏掉某一级形态**
   - **典型错误**：$\mathrm{H_3PO_4}$ 的MBE漏掉 $\mathrm{H_2PO_4^-}$，写成 $c = [\mathrm{H_3PO_4}] + [\mathrm{HPO_4^{2-}}] + [\mathrm{PO_4^{3-}}]$。
   - **正确**：四级形态一个都不能少。
   - **学生为什么会这么想**：在给定pH下某形态浓度极低时，学生可能下意识地"跳过"它。应强调"MBE是理论等式，不因数值小而删项——近似简化是后续计算步骤的事"。

3. **分子态不计入MBE**
   - **典型错误**：HAc的MBE写成 $c = [\mathrm{Ac^-}]$。
   - **正确**：$c = [\mathrm{HAc}] + [\mathrm{Ac^-}]$，分子态HAc也是含Ac的形态。
   - **学生为什么会这么想**：受"浓度 = 离子浓度"的思维定式影响。强电解质100%电离的特例被过度泛化到弱电解质。

4. **配合物体系中只算游离金属离子**
   - **典型错误**：$\mathrm{Cu^{2+}} + \mathrm{NH_3}$ 体系的MBE写成 $c(\mathrm{Cu^{2+}}) = [\mathrm{Cu^{2+}}]$。
   - **正确**：$c(\mathrm{Cu^{2+}总}) = [\mathrm{Cu^{2+}}] + [\mathrm{CuNH_3^{2+}}] + [\mathrm{Cu(NH_3)_2^{2+}}] + \dots$
   - **学生为什么会这么想**：把"自由金属离子浓度"等同于"金属元素总浓度"，忽略了配合物中含有的金属。

5. **MBE与CCE混淆——把电荷关系当作物料关系**
   - **典型错误**：列 $\mathrm{Na_2CO_3}$ 的MBE时考虑 $\mathrm{H^+}$ 和 $\mathrm{OH^-}$。
   - **正确**：MBE只追踪Na和 $\mathrm{CO_3^{2-}}$，不涉及 $\mathrm{H^+}$ 和 $\mathrm{OH^-}$。
   - **学生为什么会这么想**：三大守恒一起学时容易混淆它们的对象。MBE是"元素守恒"，不是"离子守恒"。

6. **MBE中的浓度符号用混**
   - **典型错误**：$c(\mathrm{Na}) = [\mathrm{Na^+}]$ 和 $[\mathrm{Na^+}] = c(\mathrm{Na})$ 随意互换，但 $c$ 是分析浓度而 $[\ ]$ 是平衡浓度。
   - **正确**：MBE的标准形式是"分析浓度（左）= 各形态平衡浓度之和（右）"。
   - **学生为什么会这么想**：对 $c$ 和 $[\ ]$ 的区别意识薄弱，以为两者可以随意互换。

## 十二、🎯 教学视角

### 常见迷思概念

1. **"物料守恒 = 质量守恒"的误解**
   - 学生初学时容易把MBE等同于初中化学的"质量守恒定律"。
   - 纠正：质量守恒是反应前后总质量不变（宏观），MBE针对特定组分追踪其在溶液中的所有存在形式（微观分布）。两者不在同一层面。

2. **"系数就是电荷数"的误解**
   - 学生看到 $\mathrm{Na_2CO_3}$ 中Na的系数为2，误认为是因为 $\mathrm{Na^+}$ 带+1电荷而 $\mathrm{CO_3^{2-}}$ 带−2电荷。
   - 纠正：系数来自化学式中的原子个数，与电荷无关。$\mathrm{CaCl_2}$ 的MBE：$c(\mathrm{Ca}) = [\mathrm{Ca^{2+}}]$（系数1），$c(\mathrm{Cl}) = 2[\mathrm{Cl^-}]$（系数2），都是因为化学式中Cl有2个。

3. **"MBE不需要写完全部形态"的误解**
   - 学生可能认为"浓度很小就可以不写"。
   - 纠正：列出MBE是"定义方程"——必须先写全，近似简化是后续计算时基于定量判断之后的操作。

### 推荐教学路径

1. **从"碳原子跟踪"出发**：以一杯 $\mathrm{Na_2CO_3}$ 溶液为例——"你往水里加了0.1 mol的 $\mathrm{Na_2CO_3}$，那么水里应该有多少摩尔的碳原子？它们现在以哪些形态存在？"引导学生自然得出三种含碳形态。
2. **$\mathrm{Na_2CO_3}$ 范例巩固**：完整列出Na和 $\mathrm{CO_3^{2-}}$ 两条MBE，强调Na的系数2。
3. **推广到多元酸（$\mathrm{H_3PO_4}$）**：四级形态递进，让学生自己列出MBE，检查是否漏项。
4. **拓展到配合物体系**：以 $\mathrm{Ag^+}$-$\mathrm{NH_3}$ 为例，说明配离子也含中心原子，必须在MBE中体现。
5. **收官——三大守恒联立**：用一道综合题（如 $\mathrm{NaHCO_3}$ 溶液pH计算），展示MBE、CCE、PBE如何联立求解。

### 真实世界关联

- **海水碳酸盐体系与海洋酸化**：海水中溶解的 $\mathrm{CO_2}$、$\mathrm{HCO_3^-}$、$\mathrm{CO_3^{2-}}$ 之间的MBE关系是计算海洋吸收 $\mathrm{CO_2}$ 能力的核心工具。大气 $\mathrm{CO_2}$ 浓度上升 → 海水pH下降 → 碳酸盐各形态分布改变 → 影响珊瑚和贝类钙化。
- **人体血液钙平衡**：血液中钙以游离 $\mathrm{Ca^{2+}}$（约50%）、蛋白结合钙（约40%）、配合钙（约10%）三种形态存在。MBE关系：$c(\mathrm{Ca总}) = [\mathrm{Ca^{2+}}] + [\mathrm{Ca\text{-}蛋白}] + [\mathrm{Ca\text{-}配合物}]$。临床检测总钙和游离钙的差异可用MBE解释。

## 十三、竞赛拓展

1. **质子条件（PBE）的系统推导方法——零水准法**：竞赛中复杂体系的 PBE 推导是高频考点。零水准法（参考水准法）步骤：① 选取溶液中大量存在的、参与质子转移的物种作为"零水准"（通常是溶剂分子 + 加入的原始酸碱物种）；② 写出所有得质子的产物（零水准 + H⁺）和失质子的产物（零水准 − H⁺）；③ PBE：∑（得质子产物的浓度 × 得质子数）= ∑（失质子产物的浓度 × 失质子数）。例如 Na₂CO₃ 溶液——零水准：CO₃²⁻ 和 H₂O。得质子：H₂CO₃（+2H⁺）、HCO₃⁻（+1H⁺）、H₃O⁺（+1H⁺）。失质子：OH⁻（−1H⁺）。PBE：$2[\mathrm{H_2CO_3}] + [\mathrm{HCO_3^-}] + [\mathrm{H_3O^+}] = [\mathrm{OH^-}]$。决赛可能考"多质子酸碱混合体系的 PBE"或"氨基酸等两性物质的 PBE"。

2. **分布系数 $\delta$ 与优势区域图的竞赛应用**：对于 n 元酸 $\mathrm{H_nA}$，各级电离的分布系数 $\delta_i$（$i = 0,1,...,n$）仅取决于 pH 和各级 $K_a$。关键结论：$\mathrm{pH = p}K_{a1}$ 时 $\delta_{\mathrm{H_nA}} = \delta_{\mathrm{H_{n-1}A^-}} = 0.5$；$\mathrm{pH = p}K_{ai}$ 时第 $i$ 和第 $i+1$ 级型体浓度相等。各型体的优势 pH 范围在 p$K_a$ 之间。竞赛中可用于：① 判断给定 pH 下哪种型体为主（读数即可，不用算）；② 解释为什么 EDTA 的配位能力随 pH 变化（$\mathrm{Y^{4-}}$ 的 $\delta$ 在 pH > 10 才接近 1）；③ 氨基酸的等电点计算——pI = (p$K_{a1}$ + p$K_{a2}$)/2（对于中性侧链氨基酸）。

3. **质量守恒在多步反应体系中的进阶应用——"总浓度"变换法**：复杂体系（如金属离子 + 配体 + pH 缓冲的多重平衡）中，将未知浓度用 $[M]$ 和已知常数表示：$c_M = [M] + [ML] + [ML_2] + ... = [M](1 + \beta_1[L] + \beta_2[L]^2 + ...)$。这使本来需要解 n 元方程组的体系退化到只需解 1-2 个未知量。决赛级配位平衡/沉淀溶解综合题的核心解法就是"物料守恒 + 累积稳定常数 + 总浓度变换"。
