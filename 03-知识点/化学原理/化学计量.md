## 一、定义
- **物质的量 n**（mol）：$n = m / M = N / N_A$
- **摩尔质量 M**（g/mol）：1 mol 物质的质量，数值上等于该物质的相对分子质量或相对原子质量
- **阿伏伽德罗常数 N_A**：$6.022 \times 10^{23}\ \mathrm{mol^{-1}}$
- **摩尔体积 V_m**：标准状况（STP: $0^{\circ}\mathrm{C}$, $101.325\ \mathrm{kPa}$）下，理想气体的摩尔体积为 $22.414\ \mathrm{dm^3 \cdot mol^{-1}}$

## 三、核心原理

### 化学计量计算的核心概念
- 配平后的系数给出各物质的摩尔比，这是化学计量计算的核心
- **限量试剂**（limiting reagent）：最先消耗完的反应物，决定产物的最大量（理论产量）
- **过量试剂**：反应结束后仍有剩余的反应物
- **产率**：$\displaystyle \eta = \frac{\text{实际产量}}{\text{理论产量}} \times 100\%$
- **原子经济性**：$\displaystyle \mathrm{AE} = \frac{\text{目标产物摩尔质量之和}}{\text{反应物摩尔质量之和}} \times 100\%$

### 物质的量的桥梁作用
物质的量（$n$）是化学计量计算的中心桥梁。宏观可测量（质量、气体体积、溶液浓度）均通过 $n$ 与微观粒子数（分子数、离子数）建立联系：

$$m \xrightarrow{M} n \xrightarrow{N_A} N$$

$$V_\text{气体} \xrightarrow{V_m} n \xleftarrow{c} V_\text{溶液}$$

### 化学方程式的定量含义
以反应 $2\mathrm{H_2} + \mathrm{O_2} \longrightarrow 2\mathrm{H_2O}$ 为例：
- **微观层面**：2 个 $\mathrm{H_2}$ 分子与 1 个 $\mathrm{O_2}$ 分子反应生成 2 个 $\mathrm{H_2O}$ 分子
- **宏观层面**：2 mol $\mathrm{H_2}$ 与 1 mol $\mathrm{O_2}$ 反应生成 2 mol $\mathrm{H_2O}$
- **质量层面**：4.0 g $\mathrm{H_2}$ 与 32.0 g $\mathrm{O_2}$ 反应生成 36.0 g $\mathrm{H_2O}$（质量守恒）

### 限量试剂判别方法
1. **摩尔比法**：将各反应物的物质的量除以其化学计量系数，所得商最小者为限量试剂
2. **比较法**：任选一种反应物，计算与其完全反应所需的其他反应物的量，与实际量比较

$$R_{\text{限量}} = \min\left\{\frac{n_i}{\nu_i}\right\}$$

其中 $n_i$ 为第 $i$ 种反应物的物质的量，$\nu_i$ 为其化学计量系数。

## 四、关键结论

### 化学计量计算的通用步骤
1. **写出配平的化学方程式**
2. **将已知量转换为物质的量**（$n = m/M$、$n = cV$、$n = V/V_m$ 等）
3. **根据化学计量比确定待求物的物质的量**
4. **将 $n$ 转换回题目所需的物理量**（质量、体积、浓度、粒子数等）
5. **检验有效数字和单位**

### 气体反应的体积关系
- 同温同压下，气体体积比等于物质的量之比（Avogadro 定律的直接推论）
- 对于反应物和产物均为气体的反应，可直接用体积比计算（如 $\mathrm{N_2 + 3H_2 \to 2NH_3}$ 中体积比为 1:3:2）

### 溶液反应计算
- 利用 $n = cV$ 直接计算反应物的物质的量
- 对于滴定分析：$c_A V_A / \nu_A = c_B V_B / \nu_B$
- 稀释公式：$c_1 V_1 = c_2 V_2$（溶质的物质的量不变）

## 五、常见分类或情形

### 按计算类型分类
1. **质量-质量计算**：已知反应物质量，求产物的理论产量
2. **质量-体积计算**：已知反应物质量，求生成气体的体积（给定温度、压力）
3. **溶液浓度计算**：涉及溶液中的反应，使用 $n = cV$ 关系
4. **限量试剂与产率计算**：多种反应物并存时判断限量试剂并计算产率
5. **连续反应计算**：多步反应中的化学计量关系
6. **混合物分析计算**：通过反应消耗量反推混合物组成

### 按数据处理方式分类
- **直接计算**：所有数据已知，直接求算
- **逆推计算**：由产物量反推反应物量（如元素分析、燃烧分析）
- **方程组计算**：涉及多个未知数的复杂混合物分析

## 六、适用条件与限制
- 化学计量计算假定反应**完全按方程式进行**（无副反应、无平行反应）
- 限量试剂判断**仅适用于不可逆反应**（或 $K$ 很大的反应）；对于可逆平衡反应，需用化学平衡方法处理
- 产率计算时需考虑**实际反应条件**（温度、压力、催化剂等）对反应限度的影响
- 气体反应计算需注明温度、压力条件
- 计算结果的有效数字位数由测量数据的精度决定（参见[[有效数字]]）

## 七、常见比较与易混点
| 易混点 | 区分 |
|--------|------|
| 理论产量 vs 实际产量 | 理论产量由限量试剂按化学计量计算；实际产量受产率影响 |
| 限量试剂 vs 过量试剂 | 限量试剂量决定理论产量；过量试剂量不影响理论产量 |
| 产率 vs 转化率 | 产率 = 实际产量 / 理论产量；转化率 = 已反应的反应物 / 初始反应物 |
| 原子经济性 vs 产率 | 原子经济性是反应固有的（由方程式决定）；产率取决于实验操作 |
| 物质的量浓度 vs 质量摩尔浓度 | $c$ 单位 mol/L（随温度变化）；$m$ 单位 mol/kg（不随温度变化）|
| 化学计量比 vs 反应级数 | 对于基元反应二者一致；对于复杂反应二者无直接关系 |

## 十、例题

### 例题 1：限量试剂与产率
**题目：** 将 5.00 g $\mathrm{Fe}$ 与 3.00 g $\mathrm{S}$ 混合加热，反应生成 $\mathrm{FeS}$。实际得到 6.50 g $\mathrm{FeS}$，求产率。

**分析：** $M(\mathrm{Fe}) = 55.85\ \mathrm{g/mol}$，$M(\mathrm{S}) = 32.07\ \mathrm{g/mol}$，$M(\mathrm{FeS}) = 87.92\ \mathrm{g/mol}$
$$n(\mathrm{Fe}) = \frac{5.00}{55.85} = 0.0895\ \mathrm{mol},\quad n(\mathrm{S}) = \frac{3.00}{32.07} = 0.0935\ \mathrm{mol}$$
化学计量比 1:1，$\mathrm{Fe}$ 为限量试剂。

**解答：** 理论产量 $m_{\text{理}} = 0.0895 \times 87.92 = 7.87\ \mathrm{g}$
$$\eta = \frac{6.50}{7.87} \times 100\% = 82.6\%$$

**反思：** 限量试剂不一定质量最小，需按物质的量和化学计量系数综合判断。

### 例题 2：燃烧分析确定化学式
**题目：** 某碳氢化合物样品 0.500 g 完全燃烧，生成 1.540 g $\mathrm{CO_2}$ 和 0.756 g $\mathrm{H_2O}$。已知其摩尔质量约 58 g/mol，求分子式。

**分析：**
$$n(\mathrm{C}) = n(\mathrm{CO_2}) = \frac{1.540}{44.01} = 0.03500\ \mathrm{mol}$$
$$n(\mathrm{H}) = 2n(\mathrm{H_2O}) = 2 \times \frac{0.756}{18.02} = 0.08391\ \mathrm{mol}$$
$$m(\mathrm{C}) = 0.03500 \times 12.01 = 0.4204\ \mathrm{g};\quad m(\mathrm{H}) = 0.08391 \times 1.008 = 0.08458\ \mathrm{g}$$
质量和 $0.4204 + 0.08458 = 0.5050\ \mathrm{g} \approx 0.500\ \mathrm{g}$（验证：不含其他元素）

**解答：** 
$$\mathrm{C:H} = 0.03500 : 0.08391 = 1 : 2.40 = 5 : 12$$
实验式为 $\mathrm{C_5H_{12}}$，式量 $5 \times 12.01 + 12 \times 1.008 = 72.15$
$n = 58 / 72.15 \approx 1$，最合理的分子式：$\mathrm{C_4H_{10}}$（丁烷）——需结合摩尔质量更精确判断

**反思：** 燃烧分析法中，碳氢比由 $\mathrm{CO_2}$ 和 $\mathrm{H_2O}$ 的质量反推。注意氢原子数需乘以 2。

## 十一、易错点
- **忘记配平方程式**：系数不等于 1 时必须按配平系数计算物质的量比
- **直接用质量比计算**：化学方程式中的系数是摩尔比而非质量比，必须转换为物质的量
- **限量试剂判断方法错误**：需将物质的量除以化学计量系数后再比较，不能直接比较物质的量
- **忽略单位换算**：气体体积计算时必须注明温度和压力，使用正确的 $R$ 值和单位
- **有效数字处理不当**：中间计算保留 1-2 位多余数字，最终结果按数据中最小有效数字位数取舍

## 十二、🎯 教学视角

### 12.1 学生典型认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 口诀 |
|:---|:---|:---|:---|
| "限量试剂 = 物质的量最少的反应物" | 直觉：少=先用完 | 限量试剂 = $n_i/\nu_i$ 最小者。$n$ 少但计量系数也小的可能是过量的！ | "比比值，不比绝对量" |
| "产率 > 100% 意味着计算错误" | 高中刷题形成的机械认知 | 可能产物未干燥、含杂质、或发生了其他反应。产率 > 100% 恰恰是实验题中最值得分析的线索 | ">100% 不是错，是线索" |
| "气体反应计量中 V∝n，所以直接比较体积" | 初中化学迁移 | 同温同压下成立，但题目常混入不同温度/压力的气体，必须先转换为 $n$ | "先转 n，再比" |

### 12.2 入门级例题

**题目**：$2.00\ \mathrm{g}\ \mathrm{H_2}$ 与 $16.0\ \mathrm{g}\ \mathrm{O_2}$ 反应生成水。求：(1) 限量试剂；(2) 理论产量；(3) 若实际得水 $16.2\ \mathrm{g}$，求产率。

**预期解答路径**：
1. $n(\mathrm{H_2}) = 1.00\ \mathrm{mol}$，$n(\mathrm{O_2}) = 0.500\ \mathrm{mol}$
2. 限量试剂判断：$1.00/2 = 0.50$ vs $0.500/1 = 0.50$ → 恰好完全反应！
3. $n(\mathrm{H_2O}) = 1.00\ \mathrm{mol}$，$m = 18.0\ \mathrm{g}$
4. 产率 = $16.2/18.0 = 90.0\%$

**教师引导提问**：如果题目改成 $3.00\ \mathrm{g}\ \mathrm{H_2}$ 和 $16.0\ \mathrm{g}\ \mathrm{O_2}$，限量试剂是谁？（学生常见错误：认为 H₂ 更多所以 O₂ 是限量——正确判断：$1.50/2=0.75 > 0.50$，O₂ 仍是限量）

### 12.3 与现实/直觉的连接

- **烹饪类比**：限量试剂就像"做三明治时面包片数量限制了能做的三明治数"——即使火腿和奶酪堆成山，面包片用完就停了
- **生活中**：电池没电 = 负极材料（限量试剂）反应完了；汽车油箱见底 = 汽油是限量试剂，氧气（空气中的）大大过量

## 十三、竞赛拓展
- **多步合成总产率**：总产率 = 各步产率的乘积。如三步合成各步产率分别为 90%、80%、75%，总产率 = $0.90 \times 0.80 \times 0.75 = 54\%$
- **反应物配比优化**：当某反应物昂贵或有毒时，常使另一种廉价反应物过量以提高昂贵反应物的转化率
- **热重分析（TGA）**：通过加热失重数据计算结晶水数目和化合物组成，是竞赛中的常见拓展题型
