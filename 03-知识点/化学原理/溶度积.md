# 溶度积

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[07-化学平衡]]

## 一、定义
对于难溶电解质 $A_mB_n(s) \rightleftharpoons mA^{n+}(aq) + nB^{m-}(aq)$：
$$K_{sp} = [A^{n+}]^m [B^{m-}]^n$$

- $K_{sp}$ 越小 → 溶解度通常越小（但需注意化学计量比）
- **同离子效应**：外加同离子 → 溶解度降低
- **盐效应**：外加惰性盐 → 溶解度略微增大（活度系数变化）

## 三、核心原理

### 溶度积的定义与推导
对难溶电解质 $\mathrm{AgCl}$ 的沉淀溶解平衡：
$$\mathrm{AgCl(s)} \xrightleftharpoons[\text{沉淀}]{\text{溶解}} \mathrm{Ag^+(aq) + Cl^-(aq)}$$

由热力学数据（298K）计算标准平衡常数：
- $\Delta G_{\mathrm{f}}^{\ominus}$: AgCl(s) = −109.80, Ag⁺ = 77.12, Cl⁻ = −131.26 kJ·mol⁻¹
- $\Delta G^{\ominus}(298\mathrm{K}) = 55.66\ \mathrm{kJ\cdot mol^{-1}}$
- $\lg K^{\ominus} = \frac{-\Delta G^{\ominus}}{2.303RT} = -9.750$
- $K^{\ominus} = [\mathrm{Ag^+}][\mathrm{Cl^-}] = 1.78 \times 10^{-10}$

通式：$A_mB_n(s) \rightleftharpoons mA^{n+}(aq) + nB^{m-}(aq)$
$$K_{sp}^{\ominus} = [A^{n+}]^m [B^{m-}]^n \tag{9.1}$$

**$K_{sp}$ 的性质**：
- $K_{sp}$ 只与难溶电解质的**本性**和**温度**有关，与沉淀量和离子浓度变化无关
- 严格地说，应以离子**活度**（a）幂的乘积等于常数；只有当难溶物溶解度很小（如 AgCl、BaSO₄）且活度系数 ≈ 1 时，离子浓度幂的乘积才近似等于 $K_{sp}$
- 可由实验测定饱和溶液的离子浓度求 $K_{sp}$；例：SrSO₄ 在 25°C 时 $s = 7.35\times 10^{-4}\ \mathrm{mol\cdot dm^{-3}}$，$K_{sp} = (7.35\times 10^{-4})^2 = 5.40\times 10^{-7}$

### 溶度积规则（判断沉淀生成与溶解）
对于 $MA(s) \rightleftharpoons M^{n+}(aq) + A^{n-}(aq)$：

| 关系式 | 结论 |
|------|------|
| $[M^{n+}][A^{n-}] = K_{sp}$ (Q = Ksp) | 饱和溶液，沉淀与溶解处于平衡 |
| $[M^{n+}][A^{n-}] > K_{sp}$ (Q > Ksp) | 将有 MA(s) 沉淀析出，直至新平衡 |
| $[M^{n+}][A^{n-}] < K_{sp}$ (Q < Ksp) | 将有 MA(s) 溶解，直至新平衡 |

**溶度积规则的应用**：
- 判断是否有沉淀生成（如例9.2：CaCl₂ + Na₂C₂O₄ 混合后 Q=1.0×10⁻⁴ > Ksp(CaC₂O₄)=2.32×10⁻⁹ → 有 CaC₂O₄ 沉淀）
- 判断沉淀的完全程度：通常 $[M^{n+}] \leq 1\times 10^{-6}\ \mathrm{mol\cdot dm^{-3}}$ 认为沉淀"完全"（定量分析要求）；有些情况下 $\leq 1\times 10^{-5}$ 也可

## 四、关键结论

### 溶解度 s 与 $K_{sp}$ 的换算
| 难溶物类型 | s 与 $K_{sp}$ 关系式 | 实例 |
|------|------|------|
| AB 型 | $s = \sqrt{K_{sp}}$ | AgBr: $K_{sp}=5.35\times 10^{-13} \to s=7.31\times 10^{-7}$ |
| AB₂ 型 | $s = \sqrt[3]{K_{sp}/4}$ | Mg(OH)₂: $K_{sp}=5.61\times 10^{-12} \to s=1.12\times 10^{-4}$ |
| A₂B 型 | $s = \sqrt[3]{K_{sp}/4}$ | Ag₂CrO₄: $K_{sp}=1.12\times 10^{-12} \to s=6.5\times 10^{-5}$ |

**比较表**（教材表）：
| 难溶物 | s / mol·dm⁻³ | Ksp | s与Ksp关系 |
|------|------|------|------|
| SrSO₄ | 7.35×10⁻⁴ | 5.40×10⁻⁷ | s=√Ksp |
| AgBr | 7.31×10⁻⁷ | 5.35×10⁻¹³ | s=√Ksp |
| Mg(OH)₂ | 1.12×10⁻⁴ | 5.61×10⁻¹² | s=∛(Ksp/4) |

**关键限制**：
1. **同类型才可直接比较**：Ksp 越小溶解度越小仅适用于同类型（如 SrSO₄ 和 AgBr 均为 AB 型）；SrSO₄ 与 Mg(OH)₂ 分别为 AB 和 AB₂ 型，虽溶解度相近但 Ksp 差别很大
2. **完全电离假设**：凡溶解部分几乎完全电离的，s 与 Ksp 换算可靠；如 HgCl₂ 溶解部分主要以 HgCl₂ 分子存在，不适用
3. **分步电离**：如 Fe(OH)₃ 分三步电离（K₁, K₂, K₃），Ksp = K₁K₂K₃，溶液中 [Fe³⁺] 与 [OH⁻] 比例远不是 1:3
4. **水解效应**：弱酸弱碱生成的难溶盐（如 BaCO₃、PbS）的阴/阳离子会水解，使 s 与 Ksp 偏离

### 同离子效应（Common-Ion Effect）
加入含共同离子的强电解质，使难溶物的溶解度**降低**。

**定量示例**：Ag₂CrO₄ 在纯水中 $s = 6.5\times 10^{-5}\ \mathrm{mol\cdot dm^{-3}}$（Ksp = 1.12×10⁻¹²）
- 在 0.10 mol·dm⁻³ AgNO₃ 中：$s = \frac{K_{sp}}{[Ag^+]^2} \approx \frac{1.12\times 10^{-12}}{(0.10)^2} = 1.12\times 10^{-10}\ \mathrm{mol\cdot dm^{-3}}$（降约 10⁵ 倍）
- 在 0.10 mol·dm⁻³ Na₂CrO₄ 中：$s = \frac{1}{2}[Ag^+] = \frac{1}{2}\sqrt{\frac{K_{sp}}{[CrO_4^{2-}]}} \approx 1.7\times 10^{-6}\ \mathrm{mol\cdot dm^{-3}}$（降约 38 倍）

**沉淀完全度与 Ksp 的关系**：
- $K_{sp}$ 极小的物质（如 AgI）：等物质的量试剂即可沉淀完全
- $K_{sp}$ 中等的物质（如 BaSO₄，Ksp=1.08×10⁻¹⁰）：需借助同离子效应，稍过量试剂即可完全（例9.4：[Ba²⁺] 残留 = 1.1×10⁻⁸）
- $K_{sp}$ 较大的物质（如 CaSO₄）：即使过量试剂也难以完全
- **注意**：沉淀剂过量太多反而因生成络离子（如 AgCl₂⁻）或盐效应增大溶解度

### 盐效应（Salt Effect）
加入强电解质（不含共同离子）后，溶液中离子数目骤增，正负离子周围形成"离子氛"，束缚离子自由行动，使沉淀溶解度**增大**。如图9.1所示，BaSO₄ 和 AgCl 在 KNO₃ 溶液中 s/s₀ > 1，且 KNO₃ 浓度越大，s/s₀ 越高；价态越高（如 BaSO₄），盐效应越明显。

> 同离子效应和盐效应同时存在时，一般**同离子效应占主导**。在一般较稀的溶液中不必考虑盐效应。

### 沉淀的转化
由一种沉淀转化为另一种沉淀：$K = \frac{K_{sp}(\text{原沉淀})}{K_{sp}(\text{新沉淀})}$

- PbCrO₄ (黄色, Ksp=2.8×10⁻¹³) + S²⁻ → PbS (黑色, Ksp=8.0×10⁻²⁸)，$K = 3.5\times 10^{14}$，转化彻底
- BaSO₄ (Ksp=1.08×10⁻¹⁰) + CO₃²⁻ ⇌ BaCO₃ (Ksp=2.58×10⁻⁹)，$K = 1/24$，需要反复用新鲜饱和 Na₂CO₃ 处理多次才能转化完全

### 分步沉淀（Fractional Precipitation）
溶液中多种离子共存时，生成沉淀所需**试剂离子浓度越小**的越先沉淀，而非简单地 Ksp 小者先沉淀。

**同类型沉淀**：Ksp 小的先沉淀。如 Cl⁻ 和 I⁻（均为 0.010 mol·dm⁻³）中加 AgNO₃：
- AgI 先沉淀（需 [Ag⁺] = 8.52×10⁻¹⁵），AgCl 后沉淀（需 [Ag⁺] = 1.77×10⁻⁸）
- 当 Cl⁻ 开始沉淀时，[I⁻] = 4.8×10⁻⁹ mol·dm⁻³（早已沉淀完全）

**不同类型沉淀**：必须计算所需试剂离子浓度。如 Cl⁻ 和 CrO₄²⁻（均为 0.010 mol·dm⁻³）：
- AgCl 沉淀需 [Ag⁺] = 1.8×10⁻⁸（Ksp=1.77×10⁻¹⁰）
- Ag₂CrO₄ 沉淀需 [Ag⁺] = 1.1×10⁻⁵（Ksp=1.12×10⁻¹²）
- 虽然 Ag₂CrO₄ 的 Ksp 更小，但**AgCl 先沉淀**（因为所需 [Ag⁺] 更小）

**硫化物分步沉淀示例**：Zn²⁺ 和 Mn²⁺ 均为 0.10 mol·dm⁻³，通 H₂S：
- ZnS (Ksp=1.6×10⁻²⁴) 和 MnS (Ksp=2.5×10⁻¹³)
- 控制 pH 在 1.59~4.68 之间可使 ZnS 完全沉淀而 MnS 不析出
- 实际最佳 pH ≈ 3.5，需加缓冲溶液控制酸度

**氢氧化物分步沉淀示例**（例9.7）：Fe³⁺ 和 Mg²⁺ 均为 0.010 mol·dm⁻³：
- Fe(OH)₃ (Ksp=2.79×10⁻³⁹)：开始沉淀 pH=1.81，完全沉淀 pH=3.15
- Mg(OH)₂ (Ksp=5.61×10⁻¹²)：开始沉淀 pH=9.38，完全沉淀 pH=11.38
- 控制 pH 在 3.2~9.4 可使 Fe³⁺ 完全沉淀而 Mg²⁺ 不沉淀

## 五、常见分类或情形

### 按沉淀反应类型
| 类型 | 特征 | 典型实例 |
|------|------|------|
| 直接沉淀 | 加入沉淀剂直接生成沉淀 | Ag⁺ + Cl⁻ → AgCl↓ |
| 分步沉淀 | 先后沉淀不同离子实现分离 | I⁻ 先于 Cl⁻ 被 Ag⁺ 沉淀 |
| 沉淀转化 | 一种沉淀转化为另一种 | PbCrO₄ → PbS |
| 酸溶沉淀 | 酸与阴离子结合使沉淀溶解 | CaCO₃ 溶于 HCl |
| 氧化溶解 | 氧化性酸破坏阴离子 | CuS 溶于 HNO₃ |
| 络合溶解 | 络合剂与阳离子结合使沉淀溶解 | AgCl 溶于 NH₃·H₂O |

### 影响沉淀完全度的因素
1. 沉淀物本质（Ksp 大小）——决定性的
2. 同离子效应 —— 适当过量沉淀剂有利
3. 盐效应 —— 过量太多反而不利
4. 酸度 —— 弱酸盐受 pH 影响大
5. 络合效应 —— 沉淀剂过量可能形成络离子（如 AgCl + Cl⁻ ⇌ AgCl₂⁻）
6. 温度 —— 一般溶解为吸热过程，T↑ 溶解度↑

## 六、适用条件与限制

- 适用于**难溶强电解质**（溶解部分几乎完全电离）的沉淀溶解平衡
- **活度近似**：仅当溶解度很小（s < ~10⁻³ mol·dm⁻³）时，浓度可近似代替活度；溶解度较大时需用活度系数校正
- 溶解度与 Ksp 的简单换算仅适用于**完全电离、无分步电离、无水解副反应**的情形
- 同离子效应计算中，近似处理（如 x + 0.10 ≈ 0.10）的有效性需验证
- 酸溶平衡常数 $K = K_{sp} / (K_{a1}K_{a2})$ 仅适用于生成二元弱酸的硫化物等沉淀
- 分步沉淀的分离效果与**初始离子浓度**有关，不仅取决于 Ksp
- 实际工作中，碱式盐生成、氢氧化物聚合等使实际沉淀 pH 与理论计算有出入（见表9.2），需参考计算值并通过实验确定

## 七、常见比较与易混点

| 易混点 | 区分 |
|------|------|
| Ksp vs 溶解度 s | 同类型（如AB型）可直接由Ksp比较溶解度；不同类型必须换算为s再比较 |
| 同离子效应 vs 盐效应 | 同离子效应降低溶解度（化学作用）；盐效应增大溶解度（物理静电作用）。一般同离子效应更强 |
| Q > Ksp 沉淀 vs Q < Ksp 溶解 | Q 是起始浓度积（反应商），Ksp 是饱和常数；浓度改变使平衡移动但不改变 Ksp |
| 分步沉淀判断 | 关键是比较**开始沉淀所需试剂离子浓度**，而非直接比较 Ksp（尤其不同类型沉淀） |
| 沉淀完全的标准 | 定量分析：[ion] ≤ 1×10⁻⁶ mol·dm⁻³；一般化学：[ion] ≤ 1×10⁻⁵ mol·dm⁻³ |
| 沉淀转化的方向 | 总是由 Ksp 大的沉淀向 Ksp 小的沉淀转化；转化平衡常数 = Ksp(大)/Ksp(小) |
| 酸溶反应 | 取决于沉淀 Ksp 和生成酸的 Ka 两个因素：Ksp 越大、Ka 越小，酸溶越易进行 |
