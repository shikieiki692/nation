# 分子极性

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[10-分子结构与化学键]]

## 一、定义
**分子极性**：分子整体电荷分布不均所表现出的宏观极性，其定量表征是**分子总偶极矩**是否为零。

判断核心不是“有没有极性键”，而是：
- 键是否有极性
- 这些键偶极矩的**矢量和**是否为零

## 二、考纲对应
- 要求层级：能够结合键的极性与空间构型判断分子极性
- 课堂定位：是 [[偶极矩]]、[[范德华力]]、[[氢键]]、[[溶解度]] 的上游桥梁页

## 三、核心原理

### 3.1 两步判断法
1. **先看单键是否极性**：由成键原子电负性差决定
2. **再看整体构型是否抵消**：由 VSEPR / 杂化轨道理论决定

### 3.2 极性键不等于极性分子
- 极性键 + 高度对称构型 → 键矩抵消 → **非极性分子**
- 极性键 + 不对称构型 → 键矩不抵消 → **极性分子**
- 无极性键构成的双原子分子或对称多原子分子通常为非极性

### 3.3 孤电子对的作用
孤电子对本身不形成键，但会改变空间构型，从而改变键偶极矩能否抵消。

典型例子：
- CO2：直线形，抵消
- H2O：V 形，不抵消
- NH3：三角锥，不抵消
- BF3：平面三角形，抵消

## 四、关键结论

### 4.1 经典判断表
| 分子 | 键的极性 | 构型 | 分子极性 |
|------|:---:|------|:---:|
| CO2 | 有 | 直线形 | 非极性 |
| H2O | 有 | V 形 | 极性 |
| BF3 | 有 | 平面三角形 | 非极性 |
| NH3 | 有 | 三角锥 | 极性 |
| CH4 | 近似无/很弱 | 正四面体 | 非极性 |
| CH3Cl | 有 | 四面体但不对称取代 | 极性 |
| SF6 | 有 | 正八面体 | 非极性 |
| PF5 | 有 | 三角双锥 | 非极性 |
| SF4 | 有 | 翘翘板形 | 极性 |
| XeF4 | 有 | 平面正方形 | 非极性 |

### 4.2 顺反异构的极性差异
- `cis-1,2-二氯乙烯`：极性
- `trans-1,2-二氯乙烯`：非极性

这说明同一分子式、同一成键方式，**仅仅空间排列变化**就足以改变分子极性。

## 五、常见分类或情形

### 5.1 双原子分子
| 类型 | 键的极性 | 分子极性 | 实例 |
|------|:---:|:---:|------|
| 同核双原子 | 无 | 非极性 | H2, O2, N2, Cl2 |
| 异核双原子 | 有 | 极性 | HF, HCl, CO |

### 5.2 多原子分子中的“对称抵消”
| 分子 | 说明 |
|------|------|
| CO2 | 两个 C=O 键在一直线上，大小相等、方向相反 |
| CCl4 | 四个 C-Cl 键沿正四面体方向，矢量和为零 |
| BF3 | 三个 B-F 键在平面三角形中对称分布 |
| SF6 | 六个 S-F 键在八面体中完全对称 |

### 5.3 孤对打破对称的高频分子
| 分子 | 电子对排布 | 分子构型 | 极性判断 |
|------|------|------|------|
| NH3 | 四面体电子对排布 | 三角锥 | 极性 |
| H2O | 四面体电子对排布 | V 形 | 极性 |
| SF4 | 三角双锥电子对排布 | 翘翘板形 | 极性 |
| BrF5 | 八面体电子对排布 | 四方锥 | 极性 |

## 六、适用条件与限制
- 判断分子极性必须同时看“键极性”和“构型对称性”，缺一不可
- 不能把“对称”只理解成肉眼看着对称，必须落到**偶极矩矢量是否抵消**
- “极性溶剂/非极性溶剂”是更宏观的概念，还会受氢键能力、介电常数等影响

## 七、常见比较与易混点
- **有极性键不一定是极性分子**：CO2、BF3、CCl4 都是典型反例
- **直线形不一定非极性**：HCl、CO 虽线性但只有一个键矩，不会抵消
- **孤电子对虽不成键，却常决定极性**：NH3、H2O、SF4 都要靠它解释
- **判断前先画结构**：只看分子式很容易误判 PF5、XeF4、SF4 这类分子

## 十、例题
- **例题 1**：判断 PF5、SF4、XeF4 的分子极性。  
  **答案要点**：PF5 三角双锥且完全对称，为非极性；SF4 含孤对形成翘翘板形，为极性；XeF4 平面正方形且对称，为非极性。
- **例题 2**：为什么 CO2 非极性而 SO2 极性？  
  **答案要点**：二者均含极性键，但 CO2 为直线形、键矩抵消；SO2 为 V 形、键矩不抵消。

## 十一、易错点
- 把“有极性键”直接等同于“极性分子”
- 忽略孤电子对对分子构型的决定作用
- 不画 VSEPR 构型，直接凭分子式猜极性
- 把 `mu = 0` 误解为“每个键都没有极性”

## 十二、🎯 教学视角
- 第一轮最值得抓的是“先看键、再看构型、最后看矢量和”这条判断流程
- 这一页最好和 [[偶极矩]] 连起来讲，学生会更容易理解“为什么叫极性”
- 课堂上最适合拿来训练的分子是：CO2、H2O、NH3、BF3、PF5、SF4、XeF4

### 12.1 与现实/直觉的连接
- **拔河类比**：每根极性键像一股拉力，方向由正端指向负端；分子极性就是所有拉力的合力
- **相似相溶**：极性分子更容易和极性分子相互吸引，非极性分子更容易溶于非极性溶剂
- **顺反异构差异**：同样是二氯乙烯，仅空间排列不同，就能造成沸点和极性差异

## 十三、🧰 备课可抽料资产

### 13.1 主干结论
- 分子极性不是看单个键，而是看所有键偶极矩的矢量和
- 键极性与分子构型必须同时判断
- 孤电子对是第一轮判断中最常见的“破坏对称性”因素

### 13.2 机制 / 推导资产
- 可直接复用的流程：电负性差 → 键极性 → VSEPR 构型 → 矢量叠加 → 分子极性
- 可直接复用的课堂桥句：“极性键只是原料，空间构型才决定成品”

### 13.3 误区 / 纠偏资产
- “极性键不等于极性分子”
- “先画构型，再谈极性”
- “孤对虽然看不见，但最会改变结论”

### 13.4 题型 / 例题资产
- 建议题链：双原子分子 → 对称多原子分子 → 含孤对分子 → 顺反异构体
- 适合课堂快练：CO2 / SO2 / BF3 / NH3 / XeF4 / SF4
