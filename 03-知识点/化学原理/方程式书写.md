# 方程式书写

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 关联笔记：[[离子方程式]], [[氧化还原反应]], [[化学计量]]

## 一、定义

化学方程式是用化学式表示化学反应的式子，它不仅反映了反应物和生成物的种类，还体现了它们之间的定量关系（原子守恒、电荷守恒、质量守恒）。竞赛中方程式书写要求准确、规范，是解答计算题和推断题的基础。

## 二、考纲对应

- 对应考纲条目：[[04-热力学第一定律]]、[[08-化学反应与方程式]]
- 所属模块：[[基础要求-化学原理]]
- 本知识点在考纲中的作用：方程式书写是所有化学计算和推断的基础技能。竞赛中要求能正确书写陌生反应的化学方程式和离子方程式（含氧化还原配平、缺项配平），是元素推断题和合成路线题的必备前置能力。

## 三、核心原理

### 3.1 化学方程式的书写原则

1. **以客观事实为依据**：不能臆造不存在的反应
2. **遵守质量守恒**：反应前后各元素原子数相等（配平）
3. **遵守电荷守恒**：离子反应前后净电荷相等
4. **注明反应条件**：温度、压强、催化剂、光照等
5. **标注物质状态**：(s)、(l)、(g)、(aq)

### 3.2 常见配平方法

| 方法 | 适用类型 | 要点 |
|------|---------|------|
| **观察法** | 简单反应 | 从复杂物质入手，逐步调整系数 |
| **最小公倍数法** | 单质参与的反应 | 找出现次数最少的元素，配其原子数 |
| **氧化数法** | 氧化还原反应 | 氧化数升高总数 = 降低总数 |
| **离子-电子法** | 离子型氧化还原 | 分半反应配平，再合并 |
| **待定系数法** | 复杂反应 | 设未知数，列方程组求解 |

### 3.3 离子方程式书写规则

- **可拆物质**：强酸、强碱、可溶性盐写成离子形式
- **不拆物质**：难溶物、气体、单质、氧化物、弱电解质写化学式
- **删去 spectator ions**：反应前后未参与反应的离子
- **检查守恒**：原子守恒、电荷守恒

### 3.4 热化学方程式

- 注明反应温度和压强（默认 25°C, 101 kPa）
- $\Delta H$ 与方程式计量系数对应
- 注明物质状态（影响 $\Delta H$ 数值）

## 四、关键结论

1. **方程式书写四步法**：写出反应物和产物（以客观事实为依据）→ 配平原子数（质量守恒）→ 配平电荷（氧化还原中电子守恒 + 离子反应中净电荷守恒）→ 检查（三个守恒 + 条件/状态标注）。这是贯穿所有方程式书写的统一流程，无论反应类型如何变化，核心框架不变。

2. **氧化还原配平的核心是电子守恒**：氧化剂得电子总数必须等于还原剂失电子总数。这是氧化数法和离子-电子法的本质依据——氧化数升降总数相等，实际上就是电子转移守恒的体现。忘记检查电子是否守恒是配平失败最常见的原因。

3. **非氧化还原配平只需原子守恒**：复分解反应、酸碱中和、沉淀反应等非氧化还原过程，价态不变，只需要保证反应前后每种元素的原子数相等即可。通常用观察法或最小公倍数法就能配出，不必引入氧化数概念。

4. **溶液中反应优先写离子方程式**：离子方程式删去了不参与反应的 spectator ions（旁观离子），更能反映反应的本质。例如 $\ce{BaCl2 + Na2SO4 -> BaSO4 v + 2NaCl}$ 的本质是 $\ce{Ba^{2+} + SO4^{2-} -> BaSO4 v}$。竞赛阅卷中，如果要求写离子方程式而错写化学方程式，直接扣分。

5. **方程式配平 $\neq$ 热力学可行**：配平只是化学计量上的守恒，不代表该反应在实际条件下能自发进行。例如 $\ce{2NaCl + I2 -> 2NaI + Cl2}$ 配平没问题，但实际上碘的氧化性弱于氯，反应不会正向发生。永远先判断反应方向性再书写。

6. **"缺项配平"补充规则**：当反应物或产物中缺 H/O 原子时，需要补充介质离子/分子。判定规则为：
   - **酸性介质**：用 $\ce{H+}$ 和 $\ce{H2O}$ 补缺
   - **碱性介质**：用 $\ce{OH-}$ 和 $\ce{H2O}$ 补缺
   - **中性介质**：反应物侧用 $\ce{H2O}$ 补缺，产物侧可能出 $\ce{H+}$ 或 $\ce{OH-}$（由反应本质决定）
   
   口诀："酸补 $\ce{H+}$ 水来配，碱补 $\ce{OH-}$ 水到位"。

## 五、常见分类或情形

| 序号 | 情形 | 核心守恒量 | 典型方法 | 示例 |
|:---:|------|:----------:|---------|------|
| 1 | **非氧化还原方程式** | 原子守恒 | 观察法、最小公倍数法 | $\ce{NaOH + HCl -> NaCl + H2O}$ |
| 2 | **氧化还原方程式（分子）** | 原子守恒 + 电子守恒 | 氧化数法 | $\ce{2KMnO4 + 16HCl -> 2KCl + 2MnCl2 + 5Cl2 ^ + 8H2O}$ |
| 3 | **离子方程式（氧化还原型）** | 原子 + 电荷 + 电子三重守恒 | 离子-电子法（半反应法） | $\ce{MnO4- + 5Fe^{2+} + 8H+ -> Mn^{2+} + 5Fe^{3+} + 4H2O}$ |
| 4 | **有机反应方程式** | 主要关注官能团转化 | 写有机物结构变化，无机试剂可简化 | $\ce{CH3CH2OH ->[{浓H2SO4}][170^\circ C] CH2=CH2 ^ + H2O}$ |
| 5 | **热化学方程式** | 能量守恒（$\Delta H$ 与系数对应） | 标明状态和 $\Delta H$ | $\ce{2H2(g) + O2(g) -> 2H2O(l)}\quad \Delta H = -571.6\ \mathrm{kJ\cdot mol^{-1}}$ |
| 6 | **缺项配平** | 同上 + 质量守恒补缺项 | 先配变价元素，再看 H/O 差额补 $\ce{H+/OH-/H2O}$ | $\ce{2MnO4- + 5H2O2 + 6H+ -> 2Mn^{2+} + 5O2 ^ + 8H2O}$ |

> **注意**：第 6 种"缺项配平"不是独立反应类型，而是配平技巧——当给出反应物和产物但不完整（缺介质物种）时，需要根据反应条件和 H/O 原子数差来自行补全。

## 六、适用条件与限制

1. **客观性前提**：方程式书写必须基于真实的化学反应事实。不能仅凭化学式拼凑——例如 $\ce{Cu + 2HCl -> CuCl2 + H2 ^}$ 配平正确，但铜不与非氧化性酸反应，该方程式无效。这是最根本的限制：配平服务于真实反应，而非反过来。

2. **状态条件决定形式**：同一反应在不同条件下可能写出不同方程式。例如 $\ce{2NaOH + CO2 -> Na2CO3 + H2O}$（$\ce{CO2}$ 少量） vs $\ce{NaOH + CO2 -> NaHCO3}$（$\ce{CO2}$ 过量）。必须明确反应条件（浓度、温度、pH 等）才能正确书写。

3. **离子方程式适用范围**：离子方程式只适用于电解质溶液中的离子反应。固相反应（如 $\ce{CaO + SiO2 ->[{高温}] CaSiO3}$）、气相反应、非电解质溶液中的反应不能写离子方程式。

4. **化合价规则**：氧化还原方程式书写的前提是能正确判断各元素化合价。对于结构复杂的物质（如 $\ce{Fe3O4}$、$\ce{Na2S2O3}$、$\ce{S4O6^{2-}}$），化合价的准确判定是第一关。$\ce{Fe3O4}$ 可视作 $\ce{FeO \cdot Fe2O3}$（Fe 为 +2 和 +3），$\ce{Na2S2O3}$ 中 S 的平均氧化数为 +2（实际上是两个不等价 S）。

5. **热化学方程式的特殊性**：系数可以是分数（如 $\ce{1/2 N2(g) + 3/2 H2(g) -> NH3(g)}$），但普通化学方程式系数必须是最简整数比。热化学方程式必须注明温度、压强（默认 $25^\circ\mathrm{C}$、$101\ \mathrm{kPa}$ 除外）和物质聚集状态，因为 $\Delta H$ 依赖于这些因素。

## 七、常见比较与易混点

| 比较维度 | 概念 A | 概念 B | 辨析要点 |
|:---------|--------|--------|---------|
| **化学方程式 vs 离子方程式** | 写出所有物质化学式 | 仅写实际参与反应的离子/分子 | 离子方程式更能反映反应本质；竞赛中"写反应方程式"默认写化学方程式，"写离子方程式"明确要求才能写离子式 |
| **配平正确 vs 反应可行** | 计量系数满足守恒 | 反应在热力学/动力学上可行 | 这是学生最高频的混淆——用氧化数升降"凑出"一个配平的方程式就以为是真实反应。典型反例：$\ce{Fe + 2Al^{3+} -> 3Fe^{2+} + 2Al}$ 配平但不会发生 |
| **缺项配平中 $\ce{H+}$ vs $\ce{OH-}$** | 酸性介质加 $\ce{H+}$ | 碱性介质加 $\ce{OH-}$ | 判断关键：反应条件（题目可能明说"酸性条件"、"碱性条件"）或产物特征（若生成 $\ce{SO4^{2-}}$ 等含氧酸根为酸性，生成 $\ce{MnO2}$ 等为近中性或碱性） |
| **有机方程式中的 $\ce{H2O}$** | 有时写 | 有时不写 | 若 $\ce{H2O}$ 是有机反应的主要副产物（如消去、酯化），必须写；若只是无机试剂之间的中和等无关副反应，可不写，重点是表示有机物的转化 |
| **热化学方程式系数** | 可为分数 | 普通方程式必须整数 | $\ce{\frac{1}{2}H2(g) + \frac{1}{2}Cl2(g) -> HCl(g)}\ \Delta H = -92.3\ \mathrm{kJ\cdot mol^{-1}}$ 是合法的热化学方程式 |
| **氧化数法 vs 离子-电子法** | 通过氧化数升降找最小公倍数配系数 | 分别写氧化半反应和还原半反应再合并 | 离子-电子法更严谨（能处理复杂体系），但步骤多；氧化数法更快速，适合简单反应。竞赛中二法皆可，但缺项配平推荐用离子-电子法 |
| **反应条件标注位置** | 写在箭头上方/下方 | 写在 $\Delta$ 符号旁 | 催化剂、温度、压强等写在反应箭头上方（如 $\ce{->[{MnO2}][\Delta]}$）；热化学方程式的 $T$、$p$ 也可在 $\Delta H$ 后面括号注明 |

## 八、与其他知识点的联系

### 前置知识（学习方程式书写前必须掌握）
- [[化学计量]]：理解相对原子质量、摩尔、化学式与原子数关系，是配平的数学基础
- [[氧化还原反应]]：掌握化合价/氧化数判定、氧化剂与还原剂识别、得失电子概念——这是氧化还原配平的逻辑前提
- [[离子方程式]]：理解电解质电离、强弱电解质区分、spectator ion 概念
- [[化合价与化学键]]：能快速判断常见元素及变价元素在各物质中的化合价
- [[化学反应分类]]：能区分氧化还原反应和非氧化还原反应，选择合适的配平方法

### 并行知识（与方程式书写同步学习、互相强化）
- [[化学计量计算]]：配平后方可进行质量-物质的量-气体体积等计算，方程式系数即摩尔比
- [[化学反应速率]]：方程式是速率方程建立的基础（基元反应的速率方程与计量系数直接相关）
- [[化学平衡]]：平衡常数表达式中的指数来自方程式系数，系数变化导致 $K$ 值改变
- [[热化学]]：热化学方程式的 $\Delta H$ 与系数严格对应

### 后续知识（方程式书写熟练后支撑学习）
- [[元素化学]]：每个元素的反应方程式是元素化学知识的基本载体
- [[电化学]]：电极反应本质是半反应（氧化半反应/还原半反应），与离子-电子法配平完全一致
- [[分析化学]]：滴定反应的计量关系依赖于正确配平的方程式
- [[有机化学]]：有机反应机理中电子流向的表示与氧化还原半反应思路相通

## 九、典型题型

> 题型难度递进：基础配平 → 氧化还原配平 → 缺项配平 → 离子方程式书写 → 热化学方程式

| 题型 | 考察重点 | 难度 | 典型特征 |
|------|---------|:----:|---------|
| **基础配平**（非氧化还原） | 原子守恒、观察法/最小公倍数法 | ★★ | 复分解、中和、沉淀等，化合价不变 |
| **氧化还原配平** | 电子守恒、氧化数法 | ★★★ | 有变价元素，需计算氧化数升降并取最小公倍数 |
| **缺项配平** | 补充 $\ce{H+/OH-/H2O}$ + 电子守恒 | ★★★★ | 给出的反应物/产物不全，需根据 H/O 差额和介质条件补全 |
| **离子方程式书写** | 可拆/不拆判断 + 电荷守恒 + 电子守恒 | ★★★ | 常含 $\ce{MnO4-}$、$\ce{Cr2O7^{2-}}$ 等特征离子，需写半反应合并 |
| **热化学方程式** | 状态标注 + $\Delta H$ 计算与书写 | ★★~★★★ | 给定键能/生成焓数据，先计算 $\Delta H$ 再正确书写，系数可为分数 |

## 十、例题

### 例题 1（★★）—— $\ce{KClO3 -> KCl + O2}$ 分解配平

> **题目**：氯酸钾受热分解生成氯化钾和氧气，写出并配平化学方程式。

**分析**

学生拿到此题，第一反应可能是：$\ce{KClO3 -> KCl + O2}$，K 和 Cl 各 1 个已守恒，只需配 O 原子——左边 3 个 O，右边 2 个 O，取最小公倍数 6，那么 $2\ce{KClO3 -> 2KCl + 3O2}$。

**学生为什么会这么想**：受最小公倍数法训练的惯性思维——看到两边原子数不同就取最小公倍数。但这里有一个陷阱：$\ce{KClO3}$ 中的 $\ce{Cl}$ 是 +5 价，$\ce{KCl}$ 中的 $\ce{Cl}$ 是 -1 价，$\ce{O}$ 从 -2 价变为 0 价——这其实是一个**氧化还原反应**（$\ce{Cl}$ 被还原，$\ce{O}$ 被氧化），不能只用观察法。

**解答**

**方法一（氧化数法）**：
- $\ce{Cl}$：+5 → -1，降低 6 × 1 = 6
- $\ce{O}$：-2 → 0，升高 2 × 2 = 4（每分子 $\ce{O2}$ 含 2 个 O 原子）
- 电子守恒：最小公倍数 = 12，因此 Cl 的系数 = $12/6 = 2$，$\ce{O2}$ 的系数 = $12/4 = 3$
- 故 $2\ce{KClO3 -> 2KCl + 3O2 ^}$

**方法二**：直接用氧化数升降找出系数，结果为 $\ce{2KClO3 ->[{MnO2}][\Delta] 2KCl + 3O2 ^}$。

**反思**

这个例子说明：**不能因为配平简单就忽略氧化还原本质**。观察法/"凑"出结果是侥幸——换一个更复杂的变价反应（如 $\ce{KMnO4}$ 热分解），纯观察法就会失效。养成"先判断是否氧化还原 → 再选方法"的习惯，远比直接上手配更重要。

### 例题 2（★★★）—— $\ce{MnO4- + H2C2O4 + H+ -> Mn^{2+} + CO2 + H2O}$ 氧化还原配平

> **题目**：酸性高锰酸钾溶液与草酸反应的离子方程式配平。

**分析**

这是竞赛中的高频反应，也是离子-电子法（半反应法）的经典训练题。

- 氧化剂：$\ce{MnO4-}$（Mn +7 → +2，得 5e⁻）
- 还原剂：$\ce{H2C2O4}$（C +3 → +4，每分子草酸含 2 个 C，共失 2e⁻）
- 电子守恒：最小公倍数 10 → $\ce{MnO4-}$ 系数 = 2，$\ce{H2C2O4}$ 系数 = 5
- 然后补 $\ce{H+}$ 和 $\ce{H2O}$ 使 H、O 原子守恒

**解答**

**半反应法**：

还原半反应：$\ce{MnO4- + 8H+ + 5e- -> Mn^{2+} + 4H2O}$  … ①
氧化半反应：$\ce{H2C2O4 -> 2CO2 + 2H+ + 2e-}$  … ②

①×2 + ②×5，消去电子：
$$\ce{2MnO4- + 16H+ + 5H2C2O4 -> 2Mn^{2+} + 8H2O + 10CO2 + 10H+}$$

合并右端的 $\ce{H+}$：$16-10=6$，最终：
$$\boxed{\ce{2MnO4- + 5H2C2O4 + 6H+ -> 2Mn^{2+} + 10CO2 ^ + 8H2O}}$$

**反思**

- 半反应法中 $\ce{H+}$ 的数目在合并时可能部分抵消，这是学生最易出错的地方——一定要先写完整再合并，不要试图一步出结果。
- 本题 $n(\ce{MnO4-}):n(\ce{H2C2O4}) = 2:5$ 可直接用于氧化还原滴定计算——配平系数就是滴定中的摩尔比。
- 注意反应条件是"酸性"，所以用 $\ce{H+}$ 补缺；若题目改为"碱性 $\ce{KMnO4}$ 氧化草酸"，产物将是 $\ce{MnO2}$（而非 $\ce{Mn^{2+}}$），补缺用 $\ce{OH-}$，这是缺项配平的经典变体。

### 例题 3（★★★★）—— $\ce{As2S3 + HNO3 -> H3AsO4 + H2SO4 + NO}$ 缺项配平

> **题目**：$\ce{As2S3}$ 与硝酸反应，生成 $\ce{H3AsO4}$、$\ce{H2SO4}$ 和 $\ce{NO}$，配平化学方程式。

**分析**

这是缺项配平的难题——$\ce{As}$ 和 $\ce{S}$ 两种元素同时被氧化。

- $\ce{As}$：+3 → +5，每分子 $\ce{As2S3}$ 含 2 个 As，升高 $2 \times 2 = 4$
- $\ce{S}$：-2 → +6，每分子 $\ce{As2S3}$ 含 3 个 S，升高 $3 \times 8 = 24$
- 每分子 $\ce{As2S3}$ 总升高：$4 + 24 = 28$
- $\ce{N}$：+5 → +2（$\ce{HNO3 -> NO}$），降低 3
- 电子守恒：最小公倍数 $28 \times 3 = 84$ → $\ce{As2S3}$ 系数 = 3，$\ce{HNO3}$ 系数 = 28

**解答**

先确定氧化还原相关物质的系数：

$$3\ce{As2S3} + 28\ce{HNO3} \rightarrow 6\ce{H3AsO4} + 9\ce{H2SO4} + 28\ce{NO}$$

检查 H 原子：左边 $28 \times 1 = 28$；右边 $6 \times 3 + 9 \times 2 = 18 + 18 = 36$。右边多出 **8 个 H**。

检查 O 原子：左边 $28 \times 3 = 84$；右边 $6 \times 4 + 9 \times 4 + 28 \times 1 = 24 + 36 + 28 = 88$。右边多出 **4 个 O**。

右边多出的 8H + 4O 正好是 $4\ce{H2O}$。因此水应出现在**左边**作为反应物：

$$\boxed{\ce{3As2S3 + 28HNO3 + 4H2O -> 6H3AsO4 + 9H2SO4 + 28NO ^}}$$

**反思**

- 缺项配平的核心流程：**先配变价元素（电子守恒）→ 再补非变价元素 → 最后根据 H/O 差额确定缺项（$\ce{H2O}$ / $\ce{H+}$ / $\ce{OH-}$）**。这个顺序不能乱。
- 本题 H/O 差额恰好对应 $4\ce{H2O}$ 且水应补在左边，学生常见的错误是：发现 H 多就盲目在左边加 $\ce{H+}$ 或在右边加 $\ce{OH-}$——必须先看差额的 H:O 比例。若差额 H:O = 2:1，正好是 $\ce{H2O}$；若只有 H 多而 O 不多，则补 $\ce{H+}$；若只有 O 多，则视介质补 $\ce{H2O}$（提供 1O + 2H）或 $\ce{OH-}$。
- 此反应在元素化学中有实际意义：$\ce{As2S3}$ 溶于硝酸是砷的湿法冶金前处理步骤之一。

## 十一、易错点

| 序号 | 易错点 | 错误表现 | 正确做法 |
|:---:|--------|---------|---------|
| 1 | **忘记检查反应可行性** | 用氧化数升降拼凑出一个配平方程式但反应实际不发生 | 先判断反应能否发生（金属活动性顺序、氧化性/还原性相对强弱），再书写 |
| 2 | **离子方程式电荷不守恒** | 配平了原子却漏检净电荷，如 $\ce{Fe^{3+} + Cu -> Fe^{2+} + Cu^{2+}}$（左边 +3，右边 +4） | 写完后逐一核对：原子数守恒 + 净电荷守恒 + 电子守恒（氧化还原型），三重检查 |
| 3 | **缺项配平补缺方向错误** | H/O 多出时不分青红皂白乱补 $\ce{H+}$ 或 $\ce{OH-}$ | 算清差额 → 判断 H:O 比例 → 2:1 补 $\ce{H2O}$，只有 H 多补 $\ce{H+}$，只有 O 多则按介质规则补 |
| 4 | **氧化数判断失误** | 将 $\ce{Fe3O4}$ 中 Fe 全算作 +3 或全 +8/3 | $\ce{Fe3O4 = FeO \cdot Fe2O3}$，含一个 +2 和两个 +3；$\ce{Na2S2O3}$ 中两个 S 价态不同（-2 和 +6）或按平均氧化数 +2 处理后配电子 |
| 5 | **拆与不拆判断错误** | 把 $\ce{CH3COOH}$、$\ce{H2CO3}$ 等弱电解质拆成离子；把 $\ce{CaCO3}$（难溶）拆开 | 强酸、强碱、可溶性盐 → 拆；弱电解质、难溶物、气体、单质、氧化物 → 不拆 |
| 6 | **热化学方程式 $\Delta H$ 单位与系数不对应** | 方程式系数翻倍但 $\Delta H$ 不翻倍；单位误写 | $\Delta H$ 与系数严格成正比，单位 $\mathrm{kJ \cdot mol^{-1}}$ 指"每摩尔反应"即按方程式系数进行 1 mol 反应 |
| 7 | **有机方程式不写副产物** | 消去反应只写烯烃不写 $\ce{H2O}$ | 主要小分子副产物（$\ce{H2O}$、$\ce{HX}$、$\ce{NH3}$ 等）必须写出 |
| 8 | **反应条件遗漏** | 把 $\ce{->[{浓H2SO4}][170^\circ C]}$ 写成普通箭头 | 竞赛评分中反应条件是采分点，尤其是特殊条件（高温高压、催化剂、光、通电等） |

## 十二、🎯 教学视角

### 教学定位
方程式书写是化竞基础要求中最"硬"的技能——它不考理解深度，考的是**准确率和规范性**。初赛第一道大题几乎必然涉及方程式书写或配平，这是送分题也是必拿分题，做错任何一题都会直接拉低排名。

### 教学分层策略

**第一阶段（入门）**：
- 目标：能配平非氧化还原反应的简单方程式
- 方法：观察法、最小公倍数法的大量练习
- 材料：复分解反应、中和反应、金属与酸/盐溶液反应
- 标准：20 个方程式配平全对

**第二阶段（核心）**：
- 目标：掌握氧化还原配平（氧化数法 + 离子-电子法）
- 方法：先单独练氧化数升降计算，再练完整配平流程
- 材料：$\ce{KMnO4}$、$\ce{K2Cr2O7}$、$\ce{HNO3}$、$\ce{H2O2}$、$\ce{H2C2O4}$ 五大氧化还原反应体系的各类变式
- 标准：15 分钟内完成 10 道不同难度的氧化还原配平，正确率 100%

**第三阶段（竞赛级）**：
- 目标：缺项配平 + 陌生方程式书写
- 方法：半反应法 + 条件推理 + 缺项补充
- 材料：历年初赛真题中的陌生氧化还原反应（如 $\ce{S2O3^{2-}}$ 歧化、$\ce{MnO2 + KClO3 + KOH}$ 共熔等）
- 标准：能根据文字描述（如"在碱性条件下通入 $\ce{Cl2}$ 氧化 $\ce{Cr^{3+}}$ 生成 $\ce{CrO4^{2-}}$"）独立写出完整配平的方程式

### 教学常见问题
- **学生抗拒半反应法**：觉得氧化数法"更快"，但遇到缺项配平或电极反应就束手无策。教师应早引入半反应法，让学生体会到"过程虽多但不出错"的可靠性。
- **缺少"先判断是否氧化还原"的意识**：直接上手配就容易出错。训练时可先给混合题型（含氧化还原和非氧化还原），强制要求第一步标注每个元素的化合价。
- **补缺项的 H/O 计算不熟练**：建议教学生画"H-O 收支表"，左边＝反应物 H/O 总数，右边＝产物 H/O 总数，差值一目了然。

## 十三、竞赛拓展

### 1. 歧化反应与归中反应的配平
歧化反应（同一元素部分升高部分降低）和归中反应（同一元素高价+低价→中间价）在竞赛中常见。配平技巧：**从产物侧或反应物侧分别计算电子转移**。

示例：$\ce{Cl2 + NaOH -> NaCl + NaClO3 + H2O}$（歧化）
- $\ce{Cl}$：0 → -1（得 1e⁻，生成 $\ce{NaCl}$），0 → +5（失 5e⁻，生成 $\ce{NaClO3}$）
- 电子守恒：$\ce{NaCl}$ 系数 5，$\ce{NaClO3}$ 系数 1
- 最终：$\ce{3Cl2 + 6NaOH -> 5NaCl + NaClO3 + 3H2O}$

### 2. 有机氧化还原反应的电子计数
有机分子中 C 的氧化数需逐一计算：

| 官能团/环境 | C 的氧化数 |
|:-----------|:---------:|
| $\ce{C-H}$ | -1 |
| $\ce{C-C}$ | 0 |
| $\ce{C-OH}$ / $\ce{C-O-}$ | +1 |
| $\ce{C=O}$（羰基） | +2 |
| $\ce{COOH}$（羧基） | +3 |
| $\ce{CO2}$ | +4 |

例如乙醇 $\ce{CH3CH2OH}$ 被氧化为乙酸 $\ce{CH3COOH}$：甲基 C（-3 → -3，不变），羟甲基 C（-1 → +3，升高 4），共失 4e⁻。

### 3. 多核配合物及复杂含氧酸盐的配平
遇到 $\ce{[Fe(CN)6]^{4-}}$、$\ce{[Co(NH3)6]^{3+}}$ 等配合物参与的反应，配体也可能被氧化（如 $\ce{CN-}$ → $\ce{CO2 + NO3-}$）。竞赛中判断原则：**先看中心离子价态变化，再看配体价态变化，分别配电子再合并**。

### 4. 熔融/高温固相反应的方程式特点
高温固相反应（如 $\ce{Cr2O3 + Na2CO3 + NaNO3 ->[{熔融}] Na2CrO4 + NaNO2 + CO2}$）中，氧化还原仍在发生但无溶剂 $\ce{H2O}$ 参与补缺——缺项可能变为 $\ce{O2}$ 逸出或 $\ce{O^{2-}}$ 进入晶格。不要机械套用溶液缺项配平规则。

### 5. 初赛常见"方程式送分题"考点
- $\ce{MnO2 + 4HCl(浓) ->[\Delta] MnCl2 + Cl2 ^ + 2H2O}$（注意"浓盐酸"）
- $\ce{2Na2O2 + 2H2O -> 4NaOH + O2 ^}$（注意 $\ce{Na2O2}$ 中 O 为 -1 价，发生歧化）
- $\ce{3Cu + 8HNO3(稀) -> 3Cu(NO3)2 + 2NO ^ + 4H2O}$ vs $\ce{Cu + 4HNO3(浓) -> Cu(NO3)2 + 2NO2 ^ + 2H2O}$（硝酸浓度对比）
