# 过渡元素颜色与配位行为

- 总览：[[中国化学奥林匹克基本要求-总览]]

## 一、定义

**过渡元素颜色与配位行为**是研究 d 区金属离子在配位环境中因 d 轨道分裂而产生的颜色、磁性、热力学稳定性等性质的化学分支。其核心框架是**晶体场理论（CFT）**，将配体视为点电荷，通过 d 轨道在非球形场中的能量分裂来解释实验现象。

## 三、核心原理

### 3.1 d-d 跃迁与颜色

- **颜色来源**：d¹–d⁹ 组态金属的络合物中，d 电子可在分裂后的 d 轨道之间跃迁（d-d 跃迁），所需能量常落在可见光范围，因而化合物呈现颜色。
- **d⁰ 和 d¹⁰**：无 d-d 跃迁（无未成对 d 电子或 d 轨道全满），通常无色（如 Ti⁴⁺、Zn²⁺ 的水溶液）。
- **吸收与补色**：化合物吸收某一波长的光，呈现其补色。例如 [Ti(H₂O)₆]³⁺ 在 493 nm（蓝绿光区）有最大吸收，故显紫红色。
- **不是所有颜色都来自 d-d 跃迁**：LMCT/MLCT 电荷转移带常比 d-d 带强得多，很多“颜色很深”的体系首先应怀疑 CT 带参与。

### 3.1.1 为什么有的颜色很浅，有的很深
- **Laporte 选律**：在近似中心对称的八面体体系中，d-d 跃迁常受禁阻，所以颜色常偏浅。
- **自旋选律**：若跃迁前后自旋多重度改变，吸收会进一步变弱；典型如高自旋 d5 体系。
- **对称性降低 / 振动耦合**：畸变和振动会部分解除禁阻，所以“禁阻”不等于“完全看不见”。
- **四面体常更鲜艳**：Td 无反演中心，d-d 跃迁通常比 Oh 体系更强。

### 3.2 晶体场分裂与光谱化学序列

#### （1）八面体场分裂
- 6 个配体沿坐标轴接近，d 轨道分裂为：
  - **t₂g**（dxy, dyz, dzx）：能量较低，-0.4Δ₀
  - **eg**（dz², dx²-y²）：能量较高，+0.6Δ₀
- 分裂参数 **Δ₀**（八面体场）取决于：
  1. **配体性质**（光谱化学序列）
  2. **金属氧化态**（电荷越高，Δ₀ 越大）
  3. **金属周期**（同族自上而下 Δ₀ 增大：3d < 4d < 5d）

#### （2）光谱化学序列
$$
\mathrm{I^- < Br^- < S^{2-} < SCN^- < Cl^- < NO_2^- < N_3^- < F^- < OH^- < C_2O_4^{2-} < O^{2-} < H_2O < NCS^- < CH_3CN < py < NH_3 < en < bpy < phen < NO_2^- < PPh_3 < CN^- < CO}
$$
- **强场配体**（CO、CN⁻ 等）：Δ₀ 大，易形成低自旋络合物
- **弱场配体**（I⁻、Br⁻、Cl⁻ 等）：Δ₀ 小，易形成高自旋络合物

### 3.3 高自旋与低自旋

| dⁿ | 弱场（高自旋） | 未成对电子 | 强场（低自旋） | 未成对电子 |
|:---:|:---:|:---:|:---:|:---:|
| d⁴ | t₂g³eg¹ | 4 | t₂g⁴ | 2 |
| d⁵ | t₂g³eg² | 5 | t₂g⁵ | 1 |
| d⁶ | t₂g⁴eg² | 4 | t₂g⁶ | 0 |
| d⁷ | t₂g⁵eg² | 3 | t₂g⁶eg¹ | 1 |

- **判断依据**：Δ₀ vs 成对能 P
  - Δ₀ < P：高自旋（电子优先分占不同轨道，自旋平行）
  - Δ₀ > P：低自旋（电子优先占据低能级轨道，成对）
- **周期规律**：
  - 3d 金属：高低自旋均常见（Δ₀ 与 P 相当）
  - 4d、5d 金属：几乎总是低自旋（Δ₀ >> P，轨道更伸展，与配体作用更强）

### 3.4 配位场稳定化能（LFSE）

- **定义**：相对于球形场（d 轨道简并）而言，d 电子在分裂轨道中排布所获得的额外稳定化能。
- **计算公式**：
  $$LFSE = (-0.4n_{t_{2g}} + 0.6n_{e_g})\Delta_0$$
  其中 $n_{t_{2g}}$ 和 $n_{e_g}$ 分别为 t₂g 和 eg 轨道上的电子数。
- **成对能修正**：低自旋组态中，若相对于球形场有额外电子成对，需在 LFSE 中加上成对能 P 的修正项。

| dⁿ | 高自旋 LFSE/Δ₀ | 低自旋 LFSE |
|:---:|:---:|:---|
| d⁰ | 0 | — |
| d¹ | -0.4 | — |
| d² | -0.8 | — |
| d³ | -1.2 | — |
| d⁴ | -0.6 | -1.6Δ₀ + P |
| d⁵ | 0 | -2.0Δ₀ + 2P |
| d⁶ | -0.4 | -2.4Δ₀ + 2P |
| d⁷ | -0.8 | -1.8Δ₀ + P |
| d⁸ | -1.2 | — |
| d⁹ | -0.6 | — |
| d¹⁰ | 0 | — |

**典型应用**：
- 第一过渡系 M²⁺ 水合焓的双峰曲线（d³ 和 d⁸ 为峰值，d⁰ 和 d⁵ 为谷值），可用 LFSE 解释偏离线性趋势的部分。

### 3.5 磁性与唯自旋磁矩

- **顺磁性**：含未成对电子的络合物受磁场吸引。
- **抗磁性**：所有电子均成对的络合物受磁场排斥。
- **唯自旋磁矩公式**：
  $$\mu = \sqrt{N(N+2)} \, \mu_B$$
  其中 N 为未成对电子数，μ_B 为玻尔磁子。

| N | 组态示例 | μ/μ_B（计算）|
|:---:|:---|:---:|
| 1 | t₂g¹ | 1.73 |
| 2 | t₂g² | 2.83 |
| 3 | t₂g³ | 3.87 |
| 4 | t₂g³eg¹ | 4.90 |
| 5 | t₂g³eg² | 5.92 |

- **实验应用**：通过磁矩测量区分高自旋与低自旋。例如 Fe³⁺ 高自旋 μ ≈ 5.9 μ_B（5 个未成对电子），低自旋 μ ≈ 2.3 μ_B（1 个未成对电子，但轨道贡献显著，偏离唯自旋值）。

### 3.6 自旋交叉（Spin-Crossover）

- 某些 d⁴–d⁷ 络合物中，Δ₀ ≈ P，温度变化可导致高自旋 ↔ 低自旋转变。
- 伴随颜色、磁矩的显著变化。
- 典型例子：Fe(II) 的某些多吡啶络合物。

### 3.7 课堂判断顺序
遇到“某过渡金属配合物为什么有这个颜色/为什么颜色深浅不同”时，建议固定按下面顺序判断：

1. **先看 d 电子数**：d0 / d10 先排除 d-d 跃迁主导；
2. **再看构型与场强**：八面体、四面体、平面正方形决定分裂方式；
3. **再分 d-d 还是 CT**：很深的颜色、强吸收带往往提示电荷转移；
4. **再看允许/禁阻**：Laporte、自旋选律决定强度；
5. **最后才考虑定量升级**：如果需要由多条吸收带反推参数，再转去 [[Tanabe-Sugano图]]。

## 四、关键结论

1. **颜色 ≠ 稳定性**：颜色由 d-d 跃迁能量决定，稳定性由 LFSE 决定，二者有关联但不等同。
2. **高自旋/低自旋是竞争结果**：不是配体的绝对属性，而是 Δ₀ 与 P 竞争的结果。
3. **3d vs 4d/5d 差异巨大**：同族金属自上而下 Δ₀ 增大，高自旋几乎只在 3d 金属中出现。
4. **磁矩是判断自旋态的实验利器**：唯自旋公式对大多数 3d 络合物适用，低自旋 d⁵ 和高自旋 d⁶、d⁷ 可能有轨道贡献偏离。
5. **颜色判断先看跃迁类型，再看波长位置**：先区分 d-d 与 CT，再谈“为什么呈现这个补色”。

|:---|:---|:---|:---|
| 弱场 | H₂O、OH⁻、F⁻、Cl⁻ | Mn²⁺(d⁵)、Fe²⁺(d⁶)、Co²⁺(d⁷) | — |

## 六、适用条件与限制

- 晶体场理论是**静电模型**，不涉及金属-配体共价成键。
- 对强共价性络合物（如金属羰基化合物），需用**配位场理论**（LFT，引入分子轨道）补充。
- d⁰（Ti⁴⁺）和 d¹⁰（Zn²⁺）无 d-d 跃迁，颜色来自电荷转移（LMCT/MLCT）。

## 七、常见比较与易混点

| 比较项 | 高自旋 | 低自旋 |
|:---|:---|:---|
| 电子排布 | 优先分占轨道，自旋平行 | 优先占据低能级，成对 |
| 未成对电子 | 多 | 少 |
| 磁性 | 顺磁性强 | 顺磁性弱或抗磁性 |
| 稳定性 | LFSE 较小（d⁵、d¹⁰ 为 0）| LFSE 较大 |
| 典型配体 | 弱场（H₂O、Cl⁻）| 强场（CN⁻、CO）|
| 典型金属 | 3d 金属 | 3d（高氧化态）、4d、5d |

## 十、例题

### 例题 1
**题目**：解释为什么 [Ti(H₂O)₆]³⁺ 显紫红色，而 [Ti(H₂O)₆]⁴⁺ 无色。

**分析**：Ti³⁺ 为 d¹，有 d-d 跃迁；Ti⁴⁺ 为 d⁰，无 d-d 跃迁。

**解答**：
- [Ti(H₂O)₆]³⁺：d¹ 组态，t₂g¹ → eg¹ 跃迁吸收约 493 nm（蓝绿光），显紫红色。
- [Ti(H₂O)₆]⁴⁺：d⁰ 组态，无 d 电子，无 d-d 跃迁，故无色。

**反思**：颜色判断的第一步是确定 d 电子数。

### 例题 2
**题目**：某八面体 Co(III) 络合物呈抗磁性，推断其电子组态。

**分析**：Co³⁺ 为 d⁶。抗磁性意味着无未成对电子，故为低自旋 t₂g⁶。

**解答**：低自旋 d⁶，t₂g⁶eg⁰，N = 0，μ = 0，抗磁性。

**反思**：Co³⁺ 通常是强场低自旋（尤其是与 NH₃、en 等配体），但 Co²⁺(d⁷) 则常为高自旋。

### 例题 3
**题目**：用 LFSE 解释第一过渡系 M²⁺ 水合焓为何呈双峰曲线（d³ 和 d⁸ 为峰）。

**分析**：水合焓总趋势随离子半径减小而增大，但 LFSE 叠加其上造成偏离。

**解答**：
- 高自旋 LFSE：d⁰(0)、d¹(-0.4)、d²(-0.8)、d³(-1.2)、d⁴(-0.6)、d⁵(0)、d⁶(-0.4)、d⁷(-0.8)、d⁸(-1.2)、d⁹(-0.6)、d¹⁰(0)。
- LFSE 在 d³ 和 d⁸ 最负，额外稳定化使水合放热更多，形成峰值。

**反思**：d⁵ 和 d⁰、d¹⁰ 的 LFSE = 0，是谷值位置。

## 十一、易错点
- **d-d 跃迁与电荷转移混淆**：d⁰ 和 d¹⁰ 的无色/有色应由电荷转移解释，不要强行套用 d-d 跃迁。
- **高自旋/低自旋判断对象**：只有 d⁴–d⁷ 八面体络合物才存在高/低自旋之分。d¹–d³ 和 d⁸–d¹⁰ 只有一种组态。
- **唯自旋公式的局限**：低自旋 d⁵ 和高自旋 d⁶、d⁷ 的实验磁矩可能明显偏离唯自旋值，因轨道贡献不可忽略。
- **成对能的方向**：低自旋更稳定（LFSE 更负），但要**付出成对能代价**；净稳定化能 = LFSE - 额外成对能。

### 12.3 入门级例题
**题目**：预测 [FeF₆]³⁻ 和 [Fe(CN)₆]³⁻ 的自旋状态和磁矩。

**预期解答路径**：
1. Fe³⁺ 为 d⁵
2. F⁻ 为弱场 → 高自旋 t₂g³eg²，N = 5，μ = 5.92 μ_B
3. CN⁻ 为强场 → 低自旋 t₂g⁵，N = 1，μ ≈ 1.73 μ_B（实际约 2.3 μ_B，有轨道贡献）

### 12.4 与现实/直觉的连接
- 类比：晶体场就像给 d 电子安排的"座位"——配体（强场）把座位拉得很开（Δ₀ 大），电子宁愿挤在一起（成对）坐低座位；配体（弱场）座位间距小（Δ₀ 小），电子宁可坐高座位也要各占一个（高自旋）。
- 反例：血红蛋白中的 Fe²⁺ 在高自旋和低自旋之间切换，正是 O₂ 配位导致自旋态变化，从而影响配位几何和生物功能。
- 如果课堂要继续向上走，下一跳最自然不是直接堆更多颜色例子，而是分成两条：
  - `颜色/强度判断 -> [[配位场理论]] -> [[Tanabe-Sugano图]]`
  - `允许/禁阻判断 -> [[分子对称性初步]]`

## 十三、竞赛拓展
- 光谱化学序列的**理论解释**：配位场理论（LFT）通过 σ 给予和 π 反馈解释为什么 CO、CN⁻ 是强场而 I⁻、Br⁻ 是弱场。
