# 氧化还原反应方程式配平

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[08-氧化还原与电化学]]

## 一、定义
氧化还原反应配平的两大方法：
1. **氧化数法**：利用氧化数变化相等
2. **离子-电子法（半反应法）**：分别配平氧化半反应和还原半反应

离子-电子法是竞赛主流方法。

## 三、核心原理

### 氧化数法的三步规则（教材10.1.2）

**第一步：根据实验现象确定生成物并注明反应条件。**
这是写出正确方程式的**首要问题**。若不顾反应条件会写出表面上配平但实际错误的方程式。

- 酸性 KMnO₄ + FeSO₄：MnO₄⁻ 紫红色褪去，生成无色 Mn²⁺（不是 MnO₂ 棕色沉淀，也不是碱性条件下的 MnO₄²⁻）
  $$\mathrm{5Fe^{2+} + MnO_4^- + 8H^+ \longrightarrow 5Fe^{3+} + Mn^{2+} + 4H_2O}$$
  以下两个写法的**错误原因**：
  - $5\mathrm{Fe^{2+} + MnO_4^- + 4H_2O \rightleftharpoons 5Fe^{3+} + Mn^{2+} + 8OH^-}$：碱性介质中 MnO₄⁻ 不会被还原为 Mn²⁺，且 Fe²⁺、Fe³⁺ 会生成氢氧化物沉淀
  - $3\mathrm{Fe^{2+} + MnO_4^- + 4H^+ = 3Fe^{3+} + MnO_2 + 2H_2O}$：MnO₂ 是棕色沉淀，与无色溶液事实不符

- 碱性 $\mathrm{CrO_2^- + H_2O_2}$：生成黄色铬酸盐 $\mathrm{CrO_4^{2-}}$
  $$2\mathrm{CrO_2^- + 3H_2O_2 + 2OH^- \longrightarrow 2CrO_4^{2-} + 4H_2O}$$
  错误写法：$2\mathrm{CrO_2^- + 3H_2O_2 \rightleftharpoons 2CrO_4^{2-} + 2H^+ + 2H_2O}$——在碱性溶液中不应出现 H⁺，且酸性条件下 CrO₂⁻ 和 CrO₄²⁻ 均不能存在

**第二步：确定有关元素氧化数的变化。**
这是**关键步骤**，若有错，反应式无法配平。

例1：$\mathrm{K_2Cr_2O_7}$ 与浓盐酸反应：
$$\mathrm{Cr_2O_7^{2-} + 6Cl^- + 14H^+ \longrightarrow 2Cr^{3+} + 3Cl_2 + 7H_2O}$$
- Cr 氧化数 +6 → +3（每个 Cr₂O₇²⁻ 含 2 个 Cr → 总共得 6e）
- Cl 氧化数 −1 → 0（每个 Cl− 失去 1e → 共 6Cl⁻ → 6e）

注意：含多个变价原子的情况（如 Cr₂O₇²⁻ 含 2Cr），需乘以原子数。

**第三步：氧化剂得电子数 = 还原剂失电子数。**
在大多数反应中直接明了，少数情况需综合计算。

**复杂示例：$\mathrm{As_2S_3}$ 与浓硝酸反应**
$$\mathrm{3As_2S_3 + 10NO_3^- + 4H^+ \longrightarrow 6AsO_3^- + 9S + 10NO + 2H_2O}$$
- As：+3 → +5（每个 As 失 2e，每个 As₂S₃ 含 2As → 共失 4e）
- S：−2 → 0（每个 S 失 2e，每个 As₂S₃ 含 3S → 共失 6e）
- 每个 As₂S₃ 共失 4e + 6e = 10e
- N：+5 → +2（每个 N 得 3e）
- $\mathrm{3As_2S_3}$ 共失 30e，$\mathrm{10NO_3^-}$ 共得 30e

### 氧化数法 vs 离子-电子法
| 方面 | 氧化数法 | 离子-电子法（半反应法） |
|------|------|------|
| 核心思路 | 氧化数变化相等 | 分别配平氧化和还原半反应 |
| 适用范围 | 各类氧化还原反应 | 主要适用于水溶液中的离子反应 |
| 优势 | 对固相反应、有机反应直观 | 对复杂水溶液反应清晰可靠 |
| 竞赛地位 | 基础方法，必须掌握 | 主流方法，更常用 |

### 离子-电子法配平步骤回顾
1. 写出氧化半反应和还原半反应的骨架
2. 配平除 H、O 外的元素
3. O 用 H₂O 配平；H 用 H⁺ 配平（酸性介质）
4. 用电荷平衡（加 e⁻）
5. 两个半反应乘以适当系数使电子数相等
6. 相加化简

**碱性介质处理**：每多一个 O 原子 → 加 2OH⁻ 于左边（生成 H₂O）；或先按酸性配平，再在两侧同时加 OH⁻ 中和 H⁺

## 四、关键结论

### 歧化反应（Disproportionation）
同一元素一部分原子氧化数升高，另一部分原子氧化数降低——自身既是氧化剂又是还原剂。

**教材示例：$\mathrm{KClO_3}$ 热分解**
$$4\mathrm{KClO_3} \xrightarrow{\triangle} 3\mathrm{KClO_4} + \mathrm{KCl}$$
- Cl 的氧化数变化：
  - $3\mathrm{KClO_3} \to 3\mathrm{KClO_4}$：Cl 由 +5 → +7（升高 2），共失 3×2 = 6e
  - $1\mathrm{KClO_3} \to \mathrm{KCl}$：Cl 由 +5 → −1（降低 6），共得 1×6 = 6e
- 得失电子均为 6e，配平一致

**其他常见歧化反应**：
- $\mathrm{Cl_2 + 2OH^- \longrightarrow Cl^- + ClO^- + H_2O}$
- $\mathrm{2H_2O_2 \longrightarrow 2H_2O + O_2}$（O 由 −1 → −2 和 −1 → 0）

**热力学判据**：元素电势图（Latimer 图）中，若 $E^{\ominus}(\text{右}) > E^{\ominus}(\text{左})$，中间价态可歧化。

### 归中反应（Comproportionation）
高价态与低价态同种元素反应生成中间价态。

例：$\mathrm{5I^- + IO_3^- + 6H^+ \longrightarrow 3I_2 + 3H_2O}$
- I：−1 → 0 和 +5 → 0

### 介质对产物的影响（重要）
**$\mathrm{KMnO_4}$ 在三种不同介质中的还原产物**：
| 介质条件 | 还原产物 | Mn 氧化数 | 颜色 |
|------|------|------|------|
| 强酸性 | Mn²⁺ | +2 | 无色 |
| 中性/弱酸/弱碱 | MnO₂ | +4 | 棕色沉淀 |
| 强碱性 | MnO₄²⁻ | +6 | 绿色 |

具体反应（以 Na₂SO₃ 为例）：
- 强酸性：$2\mathrm{MnO_4^- + 5SO_3^{2-} + 6H^+ \longrightarrow 2Mn^{2+} + 5SO_4^{2-} + 3H_2O}$
- 中性/弱酸/弱碱：$2\mathrm{MnO_4^- + 3SO_3^{2-} + H_2O \longrightarrow 2MnO_2 + 3SO_4^{2-} + 2OH^-}$
- 强碱性：$2\mathrm{MnO_4^- + SO_3^{2-} + 2OH^- \longrightarrow 2MnO_4^{2-} + SO_4^{2-} + H_2O}$

### 氧化还原反应配平的常见陷阱
1. **含多原子变价**：如 Cr₂O₇²⁻ 含 2Cr，As₂S₃ 含 2As + 3S——必须全部计算
2. **介质不对**：必须根据实验条件确定是否有 H⁺ 或 OH⁻ 参与
3. **产物不对**：如 KMnO₄ 在不同介质中产物不同
4. **氧化数非整数**：如 Na₂S₄O₆ 中 S 平均 +2.5，但氧化数改变量的总计算仍然适用

## 五、常见分类或情形

### 按反应物中的氧化还原元素数量
| 情形 | 特征 | 实例 |
|------|------|------|
| 简单氧化还原 | 一种氧化剂 + 一种还原剂，各含一种变价元素 | Fe²⁺ + MnO₄⁻ |
| 同元素多价态 | 同种元素的不同氧化态之间反应 | 归中反应、歧化反应 |
| 多种元素变价 | 一种物质中多种元素同时变价 | As₂S₃ + HNO₃（As 和 S 均被氧化） |
| 自身氧化还原 | 同种物质的部分分子氧化另一部分 | KClO₃ 热分解 |

### 按介质条件
| 介质 | 配平方法 | 特征标志 |
|------|------|------|
| 酸性 | 用 H⁺ 和 H₂O 配平 H、O | 反应物出现 H⁺，或反应前注明"酸性" |
| 碱性 | 用 OH⁻ 和 H₂O 配平 H、O | 反应物出现 OH⁻，或反应前注明"碱性" |
| 中性 | 产物可能出现 H⁺ 或 OH⁻ | 需根据实际产物判断 |

## 六、适用条件与限制

- 氧化数法适用性广（气相、固相、水溶液、有机反应），但对复杂水溶液反应不够直观
- 离子-电子法仅适用于**水溶液中的离子反应**（需明确溶液中实际存在的物种形式）
- 配平前必须**已知实验确定的生成物**和反应条件——不能凭空推测
- 配平的是反应的**计量关系**，不保证反应一定按该方程式进行（需热力学/动力学验证）
- 对于含非整数氧化数的物种，氧化数变化量的计算可转化为"每 mol 物质转移电子数"来处理

## 七、常见比较与易混点

| 易混点 | 区分 |
|------|------|
| 氧化数法 vs 离子-电子法 | 前者以氧化数变化为核心，后者以半反应为核心。对水溶液离子反应，离子-电子法是竞赛主流方法 |
| 化合价 vs 氧化数 | 中学"化合价升降法"用的是氧化数概念，不是经典化合价 |
| 酸性介质 vs 碱性介质产物 | KMnO₄：酸性→Mn²⁺，中性→MnO₂，强碱性→MnO₄²⁻——必须在配平时正确对应产物 |
| 氧化数变化 vs 电子转移数 | 氧化数变化量 × 原子个数 = 电子转移数（如 Cr₂O₇²⁻ 中 2 个 Cr 各降 3 → 总得 6e） |
| H⁺ vs H₃O⁺ | 教材使用 H⁺ 或 H₃O⁺ 均可，但在酸溶反应等中，实际参与的是 H₃O⁺ |

## 十二、🎯 教学视角

### 12.1 学生典型认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 口诀 |
|:---|:---|:---|:---|
| "配平就是凑系数，原子守恒就行" | 初中化学配平的惯性——只关注原子个数相等 | 氧化还原配平的**核心是电子守恒**：氧化剂得电子数 = 还原剂失电子数。原子守恒是必要条件但不充分，必须电荷也守恒 | "先守电子再守原子，两个守恒不能偏" |
| "KMnO₄ 还原产物永远是 Mn²⁺" | 高中做过太多酸性 KMnO₄ 滴定实验 | KMnO₄ 的还原产物取决于**介质酸碱性**：酸性→Mn²⁺（无色），中性→MnO₂（棕色沉淀），强碱→MnO₄²⁻（绿色）。配平前**必须先确定介质** | "酸锰二、中锰四、碱锰六" |
| "介质里的 H⁺ 或 OH⁻ 随便加就行" | 觉得 H⁺ 和 H₂O 只是"填空工具" | 介质条件由**实验事实**决定，不能为提高配平方便而随意添加。酸性溶液产物中不应出现 OH⁻，碱性溶液产物中不应出现 H⁺——这是化学事实错误 | "酸性不出OH⁻，碱性不出H⁺" |
| "平均氧化数没法用来配平" | 觉得非整数氧化数是"不精确的" | Fe₃O₄（Fe平均+8/3）、Na₂S₄O₆（S平均+2.5）都可以用氧化数法配平——每mol物质的总电子转移数仍是整数 | "平均氧化数也能配，总数算对就行" |

### 12.2 入门级例题

**题目**：用离子-电子法配平酸性条件下 KMnO₄ 与 FeSO₄ 反应的离子方程式。

**预期解答路径**：
1. 写出骨架——氧化半反应：Fe²⁺ → Fe³⁺ + e⁻
2. 还原半反应：MnO₄⁻ → Mn²⁺
   - 配Mn：已平衡
   - 配O：MnO₄⁻ → Mn²⁺ + 4H₂O
   - 配H：MnO₄⁻ + 8H⁺ → Mn²⁺ + 4H₂O
   - 配电荷：MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
3. 电子数相等：氧化×5 + 还原×1 → 5Fe²⁺ + MnO₄⁻ + 8H⁺ → 5Fe³⁺ + Mn²⁺ + 4H₂O

**教师引导提问**：如果这个反应在中性条件下进行，产物会有什么不同？（Mn 的还原产物变成 MnO₂ 棕色沉淀，同时 Fe³⁺ 在中性条件下也会水解沉淀——正是实验观察到的现象）

### 12.3 与现实/直觉的连接

- **电子收支平衡**：氧化还原配平就像一个收支账本——氧化剂"收入"电子，还原剂"支出"电子，最后的方程式必须收支平衡。多原子变价的情况（如 As₂S₃ 中 As 和 S 同时被氧化）就像"一个人同时从两个口袋掏钱"——两个口袋的钱加起来才算总数。
- **不同 pH 下产物不同 = 化学变色龙**：KMnO₄ 在酸性→无色 Mn²⁺、中性→棕色 MnO₂、碱性→绿色 MnO₄²⁻——同一个氧化剂在不同介质中"变身"成不同产物。配平前如果不看反应条件，就像拿着地图却不知道自己在哪个城市。
- **离子-电子法 = 分而治之**：把复杂的氧化还原反应拆成"氧化半反应"和"还原半反应"分别配平，就像收拾房间时先分堆再整理——这正是科学思维的"分解-综合"方法。
