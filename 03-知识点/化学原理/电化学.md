# 电化学

> 电化学是研究化学电池中氧化还原反应过程以及化学能和电能相互变换规律的化学分支。利用自发氧化还原反应产生电流的装置叫**原电池**；利用电流促使非自发氧化还原反应发生的装置叫**电解池**。本章核心概念包括电极电势、Nernst方程、电动势与热力学的关系等。

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[08-氧化还原电化学]]

---

## 一、定义

**电化学（Electrochemistry）**：研究化学能与电能相互转化规律的科学。涉及：
- **原电池（Galvanic Cell）**：利用自发氧化还原反应产生电流
- **电解池（Electrolytic Cell）**：利用外加电流驱动非自发氧化还原反应
- **电极电势（Electrode Potential）**：衡量物质氧化还原能力的物理量

## 三、核心原理

### 3.1 氧化数与氧化还原方程式配平

**氧化数规则**：
1. 单质中元素氧化数 = 0
2. 二元离子化合物中，氧化数 = 离子电荷数
3. 共价化合物中：F = -1，O = -2（过氧化物等例外），H = +1（金属氢化物中例外）
4. 化合物中各元素氧化数代数和 = 0；离子中 = 离子电荷

**配平步骤**：
1. 根据实验现象确定生成物并注明反应条件
2. 确定有关元素氧化数的变化
3. 按电子得失数目相等原则配平，用H⁺/OH⁻/H₂O配平H和O

### 3.2 原电池与电池电动势

#### 3.2.1 电池组成

以锌铜电池为例：

$$
(-) \, \mathrm{Zn} \, | \, \mathrm{Zn^{2+}}(1\,\mathrm{mol\cdot dm^{-3}}) \, || \, \mathrm{Cu^{2+}}(1\,\mathrm{mol\cdot dm^{-3}}) \, | \, \mathrm{Cu} \, (+)
$$

- **负极（阳极）**：Zn → Zn²⁺ + 2e⁻（氧化反应）
- **正极（阴极）**：Cu²⁺ + 2e⁻ → Cu（还原反应）
- **电池反应**：Zn + Cu²⁺ → Zn²⁺ + Cu

#### 3.2.2 标准氢电极与电极电势

**标准氢电极（SHE）**：
$$
2\mathrm{H^+}(1\,\mathrm{mol\cdot dm^{-3}}) + 2\mathrm{e^-} \rightleftharpoons \mathrm{H_2}(p^\ominus) \quad E^\ominus = 0.00\,\mathrm{V}
$$

**电极电势的定义**：待测电极与标准氢电极组成电池时测得的电动势。

**常见标准电极电势**（298 K）：

| 电对 | 电极反应 | E°/V |
|:---|:---|:---:|
| Li⁺/Li | Li⁺ + e⁻ ⇌ Li | -3.04 |
| Zn²⁺/Zn | Zn²⁺ + 2e⁻ ⇌ Zn | -0.76 |
| Fe²⁺/Fe | Fe²⁺ + 2e⁻ ⇌ Fe | -0.45 |
| H⁺/H₂ | 2H⁺ + 2e⁻ ⇌ H₂ | 0.00 |
| Cu²⁺/Cu | Cu²⁺ + 2e⁻ ⇌ Cu | +0.34 |
| I₂/I⁻ | I₂ + 2e⁻ ⇌ 2I⁻ | +0.54 |
| Fe³⁺/Fe²⁺ | Fe³⁺ + e⁻ ⇌ Fe²⁺ | +0.77 |
| Ag⁺/Ag | Ag⁺ + e⁻ ⇌ Ag | +0.80 |
| Br₂/Br⁻ | Br₂ + 2e⁻ ⇌ 2Br⁻ | +1.07 |
| Cl₂/Cl⁻ | Cl₂ + 2e⁻ ⇌ 2Cl⁻ | +1.36 |
| MnO₄⁻/Mn²⁺ | MnO₄⁻ + 8H⁺ + 5e⁻ ⇌ Mn²⁺ + 4H₂O | +1.51 |
| F₂/F⁻ | F₂ + 2e⁻ ⇌ 2F⁻ | +2.87 |

### 3.3 电池电动势的计算

$$
E_{\text{池}} = E_{\text{正}} - E_{\text{负}} \quad \text{或} \quad E_{\text{池}}^\ominus = E_{\text{正}}^\ominus - E_{\text{负}}^\ominus
$$

**判断反应方向**：
- E池 > 0：反应自发进行（ΔG < 0）
- E池 < 0：反应非自发（ΔG > 0）
- E池 = 0：反应达到平衡

### 3.4 电动势与热力学的关系

$$
\Delta G^\ominus = -nFE_{\text{池}}^\ominus
$$

$$
E_{\text{池}}^\ominus = \frac{0.0591\,\mathrm{V}}{n} \lg K^\ominus \quad (298\,\mathrm{K})
$$

**实用判断标准**：
- n = 1：E池° ≈ 0.36 V 时，K° ≈ 10⁶
- n = 2：E池° ≈ 0.18 V 时，K° ≈ 10⁶
- n = 3：E池° ≈ 0.12 V 时，K° ≈ 10⁶

常用 **E池° > 0.2~0.4 V** 判断氧化还原反应能否自发进行。

### 3.5 Nernst方程

#### 3.5.1 电池电动势的Nernst方程

$$
E_{\text{池}} = E_{\text{池}}^\ominus - \frac{0.0591\,\mathrm{V}}{n} \lg Q
$$

其中 Q 为反应商。

#### 3.5.2 电极电势的Nernst方程

对于电极反应：$a\,\mathrm{ox} + n\mathrm{e^-} \rightleftharpoons c\,\mathrm{red}$

$$
E = E^\ominus + \frac{0.0591\,\mathrm{V}}{n} \lg \frac{(\mathrm{ox})^a}{(\mathrm{red})^c}
$$

**关键理解**：
- 氧化态浓度升高 → E升高
- 还原态浓度升高 → E降低
- 有H⁺参与的电极反应，酸度对E影响很大

### 3.6 电极电势的间接计算

**重要原则**：电极电势是**强度量**，不具加和性；必须通过ΔG° = -nFE°进行换算。

**Latimer图（元素电势图）**：

$$
\mathrm{Cu^{2+}} \xrightarrow{0.15\,\mathrm{V}} \mathrm{Cu^+} \xrightarrow{0.52\,\mathrm{V}} \mathrm{Cu}
$$

计算Cu²⁺/Cu的电势：
$$
E^\ominus(\mathrm{Cu^{2+}/Cu}) = \frac{n_1E_1^\ominus + n_2E_2^\ominus}{n} = \frac{1 \times 0.15 + 1 \times 0.52}{2} = 0.34\,\mathrm{V}
$$

### 3.7 浓度对电极电势的影响

#### 3.7.1 酸度的影响

以Cr₂O₇²⁻/Cr³⁺为例：
$$
\mathrm{Cr_2O_7^{2-} + 14H^+ + 6e^- \rightleftharpoons 2Cr^{3+} + 7H_2O} \quad E^\ominus = +1.36\,\mathrm{V}
$$

- (H⁺) = 10 mol·dm⁻³：E = +1.50 V
- (H⁺) = 1 mol·dm⁻³：E = +1.36 V
- (H⁺) = 10⁻³ mol·dm⁻³：E = +0.95 V

**结论**：酸度降低，Cr₂O₇²⁻的氧化能力明显减弱。

#### 3.7.2 沉淀的影响

以Cu²⁺/Cu⁺为例：
$$
\mathrm{Cu^{2+} + I^- + e^- \rightleftharpoons CuI} \quad E^\ominus = +0.85\,\mathrm{V}
$$

由于CuI的Ksp = 1.3 × 10⁻¹²，Cu⁺浓度极低，使Cu²⁺/Cu⁺电对的电势从+0.15 V升高到+0.85 V，从而Cu²⁺能氧化I⁻。

#### 3.7.3 络合的影响

配合物的形成降低游离金属离子浓度，从而改变电极电势。

### 3.8 电解与超电势

#### 3.8.1 分解电压

**理论分解电压**：由热力学计算得到的最低电解电压

**实际分解电压**：使电解反应顺利进行所需的最低电压，通常高于理论值

**超电势（η）**：实际分解电压超过理论分解电压的部分

$$
\eta = E_{\text{实际}} - E_{\text{理论}}
$$

#### 3.8.2 超电势的影响

超电势导致：
- 阴极上还原反应需要更负的电势
- 阳极上氧化反应需要更正的电势

**实例**：电解ZnSO₄溶液时，由于H₂在Zn电极上的超电势较高，阴极产物是Zn而不是H₂。

### 3.9 化学电源

| 类型 | 代表 | 特点 |
|:---|:---|:---|
| **一次电池** | 锌锰干电池、锂碘电池 | 不可充电 |
| **二次电池** | 铅蓄电池、镍氢电池、锂离子电池 | 可反复充放电 |
| **燃料电池** | 氢氧燃料电池 | 能量转化率高（>80%）|

**锂离子电池**：
$$
\mathrm{Li_xC_6 + Li_{1-x}CoO_2 \xrightleftharpoons[充电]{放电} 6C + LiCoO_2}
$$

## 四、关键结论

1. **电极电势越高，氧化态的氧化能力越强**；电极电势越低，还原态的还原能力越强
2. **电池电动势 E池 = E正 - E负**，用于判断反应自发性和计算平衡常数
3. **ΔG° = -nFE池°**，连接电化学与热力学
4. **Nernst方程描述浓度对电极电势的影响**，是电化学计算的核心工具
5. **有H⁺参与的电极反应，酸度对电极电势影响很大**
6. **沉淀和络合反应通过改变离子浓度间接影响电极电势**
7. **超电势使实际电解电压高于理论值**，影响电解产物
8. **Latimer图中电极电势的计算必须用ΔG°加和**，不能直接加减E°

## 五、常见分类或情形

### 5.1 原电池 vs 电解池

| 特征 | 原电池 | 电解池 |
|:---|:---|:---|
| **能量转化** | 化学能 → 电能 | 电能 → 化学能 |
| **反应自发性** | 自发（ΔG < 0） | 非自发（ΔG > 0） |
| **电极名称** | 正极/负极 | 阳极/阴极 |
| **正极/阴极** | 还原反应 | 还原反应 |
| **负极/阳极** | 氧化反应 | 氧化反应 |
| **电子流向** | 负极 → 正极（外电路） | 电源负极 → 阴极 |

### 5.2 常见电极类型

| 电极类型 | 组成 | 示例 |
|:---|:---|:---|
| **金属-金属离子电极** | M | Mⁿ⁺ | Zn | Zn²⁺ |
| **气体电极** | 惰性电极 + 气体 | Pt | H₂ | H⁺ |
| **氧化还原电极** | 惰性电极 + 两种氧化态离子 | Pt | Fe³⁺, Fe²⁺ |
| **金属-难溶盐电极** | M | MX | X⁻ | Ag | AgCl | Cl⁻ |

## 七、常见比较与易混点

### 7.1 氧化电势 vs 还原电势

- **还原电势（E°）**：氧化态 + ne⁻ → 还原态（IUPAC推荐）
- **氧化电势**：还原态 → 氧化态 + ne⁻（数值与还原电势相反）
- **竞赛统一使用还原电势**

### 7.2 电极名称辨析

| 体系 | 发生氧化反应的电极 | 发生还原反应的电极 |
|:---|:---|:---|
| **原电池** | 负极 | 正极 |
| **电解池** | 阳极 | 阴极 |

**记忆口诀**："阳氧阴还"（阳极氧化，阴极还原）——适用于所有电化学体系

### 7.3 学生常见误区

| 误区 | 正确认识 |
|:---|:---|
| "E°可以直接加减" | E°是强度量，必须通过ΔG° = -nFE°换算后才能加和 |
| "电池电动势总是正的" | 实验测定值为正；计算值可正可负，负值表示反应非自发 |
| "标准电极电势与计量系数有关" | E°与计量系数无关；ΔG°与计量系数有关 |
| "所有电解都按电极电势顺序进行" | 超电势可能改变实际电解顺序 |
| "Nernst方程中氧化态在分子" | 注意方程形式：E = E° + (0.0591/n)lg([氧化态]/[还原态]) |

## 十、例题

### 例题 1（难度 ⭐⭐⭐）
**题目**：已知E°(Fe³⁺/Fe²⁺) = +0.77 V，E°(I₂/I⁻) = +0.54 V。判断Fe³⁺能否氧化I⁻？若能，计算反应的平衡常数。

**答案**：
- 电池反应：2Fe³⁺ + 2I⁻ → 2Fe²⁺ + I₂
- E池° = E°(Fe³⁺/Fe²⁺) - E°(I₂/I⁻) = 0.77 - 0.54 = +0.23 V > 0
- 反应能自发进行
- lg K° = nE池°/0.0591 = 2 × 0.23/0.0591 = 7.78
- K° ≈ 6.0 × 10⁷

**解析**：E池° > 0说明反应自发；K° >> 10⁶说明反应进行得很彻底。

### 例题 2（难度 ⭐⭐⭐⭐）
**题目**：已知Cu²⁺/Cu⁺的E° = +0.15 V，Cu⁺/Cu的E° = +0.52 V。求Cu²⁺/Cu的E°，并判断Cu⁺在溶液中能否稳定存在。

**答案**：
- E°(Cu²⁺/Cu) = (1×0.15 + 1×0.52)/2 = +0.34 V
- 歧化反应：2Cu⁺ → Cu²⁺ + Cu
- E池° = E°(Cu⁺/Cu) - E°(Cu²⁺/Cu⁺) = 0.52 - 0.15 = +0.37 V > 0
- Cu⁺能发生歧化，在溶液中不能稳定存在

**解析**：利用Latimer图计算时，必须通过ΔG°加和，不能直接加减E°。Cu⁺的歧化是经典的无机化学问题。

### 例题 3（难度 ⭐⭐⭐⭐）
**题目**：在含有Cl⁻、Br⁻、I⁻的混合溶液中，如何选择氧化剂只氧化I⁻而不氧化Br⁻和Cl⁻？

**答案**：
- 查标准电极电势：E°(I₂/I⁻) = +0.54 V，E°(Br₂/Br⁻) = +1.07 V，E°(Cl₂/Cl⁻) = +1.36 V
- 需要选择E°介于+0.54 V和+1.07 V之间的氧化剂
- Fe³⁺/Fe²⁺的E° = +0.77 V，满足条件
- Fe³⁺能氧化I⁻，但不能氧化Br⁻和Cl⁻

**解析**：这是电极电势选择性应用的经典问题。氧化剂的E°必须高于被氧化电对的E°，但低于不被氧化电对的E°。

## 十一、易错点

1. **电极电势的加和性错误**
   - 错误：E₃° = E₁° - E₂°
   - 正确：n₃E₃° = n₁E₁° - n₂E₂°（通过ΔG°加和）

2. **原电池与电解池电极名称混淆**
   - 错误：原电池的正极叫阳极
   - 正确：原电池只有正/负极；电解池才有阴/阳极（但正极=阴极，负极=阳极）

3. **Nernst方程符号错误**
   - 错误：E = E° - (0.0591/n)lg([氧化态]/[还原态])
   - 正确：E = E° + (0.0591/n)lg([氧化态]/[还原态])

4. **忽略超电势对电解产物的影响**
   - 错误：完全按标准电极电势判断电解产物
   - 正确：实际电解需考虑超电势，如H₂在Zn电极上的高超电势使Zn²⁺优先还原

5. **温度对E°的影响**
   - 错误：认为E°与温度无关
   - 正确：E°随温度变化，但298 K附近变化较小

### 12.1 学习路径

1. **先掌握氧化数概念和氧化还原方程式配平**
2. **理解原电池的工作原理和电池符号表示法**
3. **掌握标准氢电极和标准电极电势的定义**
4. **熟练运用E池 = E正 - E负进行计算**
5. **学习Nernst方程及其应用**
6. **理解电动势与热力学的关系（ΔG° = -nFE°）**
7. **了解电解原理和超电势概念**
8. **通过计算题巩固所有公式**

### 12.2 学生易踩的认知误区

- **误区**："电极电势越高，氧化性越强"（忽略"氧化态"的前提）
- **纠正**：应该说"电极电势越高，该电对中氧化态的氧化能力越强"
- **误区**："电池反应达到平衡时E = 0"
- **纠正**：E池 = 0时反应达到平衡，但单个电极的E不一定为0

### 12.3 入门级例题

**题目**：为什么标准氢电极的电势被定义为0 V？

**答案**：单个电极的电势无法直接测量，只能测量相对值。为了建立统一的电极电势标度，IUPAC人为规定标准氢电极的电势为0 V，其他电极的电势都是相对于它的相对值。这类似于海拔以海平面为0的参照系。

### 12.4 与现实/直觉的连接

- **金属活动性顺序**：就是金属电极电势从低到高的排列
- **铁生锈**：Fe的电极电势低于O₂/H₂O，因此Fe容易被氧化
- **水果电池**：不同金属的电极电势差产生电动势
- **锂离子电池**：手机、电动车的核心，基于Li⁺的嵌入/脱嵌反应
