# 高分子重复单元（Polymer Repeat Unit）

## 一、定义

**高分子（聚合物，Polymer）**是由大量重复结构单元（重复单元，Repeat Unit）通过共价键连接而成的大分子化合物。重复单元是由**单体（Monomer）**通过聚合反应形成的、在聚合物链中周期性出现的最小结构片段。

- **重复单元（Repeat Unit）**：聚合物主链中重复出现的最小结构单元
- **单体（Monomer）**：形成聚合物的低分子原料
- **聚合度（Degree of Polymerization, DP）**：聚合物链中重复单元的数目
- **数均分子量**：$M_n = DP \times M_0$（$M_0$ 为重复单元分子量）

**注意**：
- 加聚反应中，重复单元通常与单体结构相同（仅双键打开）
- 缩聚反应中，重复单元通常比单体少去小分子（如 H₂O, HCl）

## 三、核心原理

### 3.1 聚合反应的两大类型

#### 3.1.1 加成聚合（Addition Polymerization / Chain-Growth Polymerization）

**特征**：
- 单体含不饱和键（C=C, C≡C, C=O 等）
- 无小分子副产物
- 反应通过链式机理进行（引发 → 增长 → 终止）
- 重复单元与单体分子式基本相同

**常见加聚单体与聚合物**：

| 单体 | 聚合物 | 重复单元 | 用途 |
|:---|:---|:---|:---|
| 乙烯 $\ce{CH2=CH2}$ | 聚乙烯（PE）| $\ce{-[CH2-CH2]-}$ | 薄膜、容器 |
| 丙烯 $\ce{CH2=CH-CH3}$ | 聚丙烯（PP）| $\ce{-[CH2-CH(CH3)]-}$ | 纤维、注塑 |
| 氯乙烯 $\ce{CH2=CHCl}$ | 聚氯乙烯（PVC）| $\ce{-[CH2-CHCl]-}$ | 管材、薄膜 |
| 苯乙烯 $\ce{CH2=CH-C6H5}$ | 聚苯乙烯（PS）| $\ce{-[CH2-CH(C6H5)]-}$ | 泡沫、包装 |
| 丙烯腈 $\ce{CH2=CH-CN}$ | 聚丙烯腈（PAN）| $\ce{-[CH2-CH(CN)]-}$ | 合成纤维 |
| 四氟乙烯 $\ce{CF2=CF2}$ | 聚四氟乙烯（PTFE）| $\ce{-[CF2-CF2]-}$ | 不粘涂层 |
| 甲基丙烯酸甲酯 | 有机玻璃（PMMA）| 含酯基侧链 | 透明材料 |
| 异戊二烯 | 顺/反式聚异戊二烯 | 含双键主链 | 天然/合成橡胶 |

#### 3.1.2 缩合聚合（Condensation Polymerization / Step-Growth Polymerization）

**特征**：
- 单体通常含两个或以上官能团
- 反应过程中失去小分子（H₂O, HCl, CH₃OH 等）
- 通过逐步反应增长（每一步都是独立反应）
- 重复单元与单体分子式不同（少了小分子）

**常见缩聚单体与聚合物**：

| 单体 | 聚合物 | 重复单元 | 失去小分子 | 用途 |
|:---|:---|:---|:---:|:---|
| 对苯二甲酸 + 乙二醇 | 聚对苯二甲酸乙二醇酯（PET）| $\ce{-[CO-C6H4-CO-O-CH2CH2-O]-}$ | H₂O | 饮料瓶、纤维 |
| 己二酸 + 己二胺 | 尼龙-66（PA-66）| $\ce{-[CO-(CH2)4-CO-NH-(CH2)6-NH]-}$ | H₂O | 纤维、工程塑料 |
| 己内酰胺 | 尼龙-6（PA-6）| $\ce{-[NH-(CH2)5-CO]-}$ | —（开环聚合）| 纤维 |
| 双酚 A + 光气 | 聚碳酸酯（PC）| 含碳酸酯键 | HCl | 防弹玻璃、光盘 |
| 苯酚 + 甲醛 | 酚醛树脂 | 含亚甲基桥 | H₂O | 电木 |
| 尿素 + 甲醛 | 脲醛树脂 | 含亚甲基桥 | H₂O | 粘合剂 |
| 马来酸酐 + 乙二醇 | 不饱和聚酯 | 含酯键和双键 | H₂O | 玻璃钢 |

### 3.2 单体与重复单元的关系

| 聚合类型 | 单体 → 重复单元 | 示例 |
|:---|:---|:---|
| **加成聚合** | 重复单元 ≈ 单体（双键变单键）| 乙烯 → $\ce{-[CH2-CH2]-}$ |
| **缩合聚合** | 重复单元 = 单体之和 − 小分子 | 对苯二甲酸 + 乙二醇 → PET + H₂O |
| **开环聚合** | 环打开，无小分子失去 | 己内酰胺 → 尼龙-6 |

### 3.3 共聚物（Copolymers）

由两种或以上单体共同聚合而成：

| 类型 | 结构特征 | 示例 |
|:---|:---|:---|
| **无规共聚物（Random）** | 单体无规则排列 | SBR 橡胶（苯乙烯-丁二烯）|
| **交替共聚物（Alternating）** | ABAB 严格交替 | 马来酸酐-苯乙烯 |
| **嵌段共聚物（Block）** | AAA...BBB... | SBS 热塑性弹性体 |
| **接枝共聚物（Graft）** | 主链一种单体，支链另一种 | 高抗冲聚苯乙烯 |

## 四、关键结论

1. **加聚反应：重复单元 = 单体残基（双键打开）**；缩聚反应：重复单元 = 单体残基之和 − 小分子
2. **聚合度 DP 决定分子量**：$M_n = DP \times M_0$；高分子通常 DP > 1000
3. **聚乙烯是最简单的高分子**：重复单元为 $\ce{-[CH2-CH2]-}$，但实际聚乙烯链末端有烷基
4. **缩聚反应需要严格化学计量比**：否则分子量受限（Carothers 方程）
5. **天然高分子也是聚合物**：纤维素、淀粉、蛋白质、DNA 都有重复单元结构

| 类型 | 特征 | 代表 |
|:---|:---|:---|

## 六、适用条件

### 6.1 单体推断重复单元的方法

**加成聚合**：
1. 写出单体结构
2. 打开 C=C 双键（或三键变双键）
3. 将打开的键连接成链，即得重复单元

**缩合聚合**：
1. 写出两种单体结构
2. 确定官能团反应位点（–COOH 与 –OH / –NH₂）
3. 脱去小分子（H₂O, HCl）
4. 连接剩余部分即得重复单元

### 6.2 重复单元推断单体的方法

| 聚合物类型 | 推断方法 | 示例 |
|:---|:---|:---|
| **加聚物** | 重复单元中恢复双键 | $\ce{-[CH2-CHCl]-}$ → 氯乙烯 |
| **聚酯** | 在酯键处断裂，加 H₂O | PET → 对苯二甲酸 + 乙二醇 |
| **聚酰胺** | 在酰胺键处断裂，加 H₂O | 尼龙-66 → 己二酸 + 己二胺 |
| **聚醚** | 在醚键处断裂 | 聚环氧乙烷 → 环氧乙烷 |

### 6.3 聚合度与分子量计算

**数均聚合度**：
$$\overline{DP}_n = \frac{\overline{M}_n}{M_0}$$

**数均分子量**（端基分析法、沸点升高、渗透压）：
$$\overline{M}_n = \frac{\sum N_i M_i}{\sum N_i}$$

**重均分子量**（光散射法）：
$$\overline{M}_w = \frac{\sum N_i M_i^2}{\sum N_i M_i}$$

**多分散指数（PDI）**：
$$PDI = \frac{\overline{M}_w}{\overline{M}_n}$$

- 单分散：PDI = 1（如蛋白质）
- 一般合成高分子：PDI = 1.5–20

## 七、常见比较

### 7.1 加聚 vs 缩聚

| 维度 | 加成聚合（Chain-Growth）| 缩合聚合（Step-Growth）|
|:---|:---|:---|
| **单体特征** | 含不饱和键 | 含两个以上官能团 |
| **副产物** | 无 | 有小分子（H₂O, HCl 等）|
| **增长方式** | 链式（引发→增长→终止）| 逐步（每一步都是双分子反应）|
| **分子量增长** | 单体转化率低时即有高分子 | 高转化率时才得高分子 |
| **重复单元** | 与单体基本相同 | 比单体少小分子 |
| **实例** | PE, PP, PS, PVC | PET, PA, PC, 酚醛树脂 |

### 7.2 常见塑料的缩写与用途

| 缩写 | 全称 | 主要用途 |
|:---|:---|:---|
| PE | 聚乙烯 | 包装袋、瓶、管 |
| PP | 聚丙烯 | 食品容器、纤维、汽车部件 |
| PS | 聚苯乙烯 | 泡沫塑料、餐具、玩具 |
| PVC | 聚氯乙烯 | 管材、地板、电线电缆外皮 |
| PET | 聚对苯二甲酸乙二醇酯 | 饮料瓶、纤维（涤纶）|
| PA | 聚酰胺（尼龙）| 纤维、齿轮、轴承 |
| PC | 聚碳酸酯 | 防弹玻璃、光盘、头盔 |
| PMMA | 聚甲基丙烯酸甲酯 | 有机玻璃、光学透镜 |
| PTFE | 聚四氟乙烯 | 不粘锅涂层、密封件 |
| PU | 聚氨酯 | 泡沫、弹性体、涂料 |

### 7.3 线形、支化与交联高分子

| 结构 | 特征 | 实例 |
|:---|:---|:---|
| **线形（Linear）** | 长链无分支 | HDPE、PET、尼龙 |
| **支化（Branched）** | 主链有侧支 | LDPE、支链淀粉 |
| **交联（Cross-linked）**| 链间有共价键连接 | 硫化橡胶、热固性树脂 |
| **网状（Network）** | 三维交联 | 环氧树脂固化物 |

## 十、例题

### 例题 1：单体 → 重复单元

**题目**：写出下列单体聚合后的重复单元结构。

1. 氯乙烯 $\ce{CH2=CHCl}$
2. 对苯二甲酸与乙二醇

**解析**：

1. **氯乙烯（加聚）**：
   - 打开 C=C 双键
   - 重复单元：$\ce{-[CH2-CHCl]-}$

2. **对苯二甲酸 + 乙二醇（缩聚）**：
   - 对苯二甲酸：$\ce{HOOC-C6H4-COOH}$
   - 乙二醇：$\ce{HO-CH2CH2-OH}$
   - 酯化脱水：$\ce{-[CO-C6H4-CO-O-CH2CH2-O]-}$

### 例题 2：重复单元 → 单体

**题目**：某聚合物的重复单元如下，推断其单体。

$$
\ce{-[NH-(CH2)5-CO]-}
$$

**解析**：

1. 观察到重复单元中含有酰胺键 $\ce{-[NH-CO]-}$
2. 这是聚酰胺类高分子
3. 在酰胺键处断裂，恢复为单体：
   - $\ce{-[NH-(CH2)5-CO]-}$ → $\ce{NH2-(CH2)5-COOH}$（氨基己酸）
   - 或环状单体：己内酰胺（开环聚合）
4. **单体**：**己内酰胺**（ε-caprolactam）或 **6-氨基己酸**
5. **聚合物**：**尼龙-6（PA-6）**

### 例题 3：加聚 vs 缩聚判断

**题目**：判断下列聚合反应属于加聚还是缩聚，并说明理由。

1. $\ce{n CH2=CH2 -> -[CH2-CH2]-_n}$
2. $\ce{n HOOC-(CH2)4-COOH + n HO-(CH2)6-OH ->}$ 尼龙-66 + 副产物

**解析**：

1. **乙烯 → 聚乙烯**：
   - **加聚反应**
   - 理由：单体含 C=C 双键，聚合后双键打开；无小分子副产物

2. **己二酸 + 己二胺 → 尼龙-66**：
   - **缩聚反应**
   - 理由：单体含双官能团（–COOH 和 –NH₂）；反应中失去小分子 H₂O；重复单元比单体之和小了 n 个 H₂O

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| "重复单元 = 单体" | 忽略缩聚反应 | 只有**加聚**的重复单元与单体基本相同；缩聚要减去小分子 | PET 的重复单元与单体分子式相同吗？ |
| "所有高分子都是加聚得到的" | 不了解缩聚 | 聚酯、聚酰胺、酚醛树脂都是**缩聚**产物 | 尼龙是加聚还是缩聚产物？ |
| "高分子链两端没有区别" | 忽略端基 | 高分子链有端基，只是分子量很大时端基影响小 | 聚乙烯链的末端是什么基团？ |
| "共聚物只有两种单体" | 理解过窄 | 共聚物可以有**两种或以上**单体 | ABS 塑料由几种单体组成？（三种：丙烯腈、丁二烯、苯乙烯）|
| "热塑性塑料不能交联" | 不了解改性 | 某些热塑性塑料可通过辐照等方法交联 | PE 可以交联吗？（可以，得到 XLPE）|

| "聚合物没有确定的分子式" | 混淆分子式与结构式 | 解释：聚合物有确定的**重复单元结构式**，但分子量是多分散的 |

### 12.3 入门例题

**最简判断练习**：
> 1. 加聚反应和缩聚反应的根本区别是什么？
> 2. 聚乙烯的重复单元是什么？
> 3. 尼龙-66 是由哪两种单体缩聚而成的？

**答案**：
1. 加聚**无小分子副产物**，单体含不饱和键；缩聚**有小分子副产物**，单体含双官能团
2. $\ce{-[CH2-CH2]-}$
3. **己二酸**（HOOC–(CH₂)₄–COOH）和 **己二胺**（H₂N–(CH₂)₆–NH₂）
