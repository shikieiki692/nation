# Favorskii 重排（Favorskii Rearrangement）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[39-重排反应]]、[[47-羧酸及羧酸衍生物]]

---

## 一、定义

**Favorskii 重排**：**α-卤代酮**在**碱**作用下发生重排，生成**羧酸**（水作溶剂）或**羧酸酯**（醇作溶剂）的反应。

**通式**：
$$
\ce{R-CO-CH2-X + OH^- -> R-CH2-COOH}
$$

**本质**：**环丙酮中间体的亲核开环**——α-卤代酮先环化为环丙酮，再被亲核试剂开环。

## 三、核心原理

### 3.1 反应机理

```
步骤 1：碱夺取 α-H → 烯醇负离子
步骤 2：分子内 SN2（卤素离去）→ 环丙酮中间体
步骤 3：亲核试剂（OH⁻ 或 RO⁻）进攻环丙酮羰基碳
步骤 4：开环 → 羧酸/酯
```

**关键中间体**：**环丙酮（cyclopropanone）**——高度张力的三元环酮，非常活泼。

### 3.2 为什么形成环丙酮？

**分子内 SN2**：
- α-碳上的卤素是好的离去基
- 烯醇负离子的氧或 α-碳进攻 α-碳 → 挤出卤素
- 形成三元环 → 环丙酮

**环丙酮的特殊性**：
- 羰基碳为 sp²，但三元环迫使 C-C=O 键角偏离 120°
- 高度环张力 → 极易被亲核试剂进攻
- 是"隐藏"的中间体，难以分离

### 3.3 环状 α-卤代酮的 Favorskii

**环酮的环缩小**：

| 底物 | 产物 | 碳数变化 |
|:---|:---|:---:|
| **α-氯代环己酮** | 环戊烷甲酸 | -1 |
| **α-溴代环庚酮** | 环己烷甲酸 | -1 |
| **α-氯代环辛酮** | 环庚烷甲酸 | -1 |

- 环酮 → 环丙酮稠环 → 开环 → 少一个碳的环状羧酸
- 这是**环缩小**的标准方法

### 3.4 开环的方向性

**亲核试剂进攻后，哪个 C-C 键断裂？**
- 通常形成**更稳定的碳负离子**
- 即：取代基多的碳保留负电荷 → 断裂另一根 C-C 键
- 本质是**负电荷稳定性**决定开环方向

### 3.5 与其他重排的对比

| 重排 | 底物 | 中间体 | 产物 | 碳数变化 |
|:---|:---|:---|:---|:---:|
| **Favorskii** | α-卤代酮 | 环丙酮 | 羧酸/酯 | 不变 |
| **Beckmann** | 肟 | 亚胺正离子 | 酰胺 | 不变 |
| **B-V** | 酮 + 过酸 | Criegee | 酯 | 不变 |
| **Benzilic** | α-二酮 | 苯偶酰 | 乙醇酸 | 不变 |

---

## 四、关键结论

1. **Favorskii = α-卤代酮 + 碱 → 羧酸/酯**
2. **环丙酮是关键中间体**——高度张力，极易开环
3. **环状底物发生环缩小**——环酮 → 少一个碳的环羧酸
4. **开环方向由负电荷稳定性决定**
5. **碱催化必需**——夺取 α-H 形成烯醇负离子

## 五、常见分类或情形

### 5.1 按底物分类

| 底物类型 | 例子 | 产物 | 特点 |
|:---|:---|:---|:---|
| **开链 α-卤代酮** | α-氯代丙酮 | 羧酸/酯 | 简单 |
| **环状 α-卤代酮** | α-氯代环己酮 | 环缩小羧酸 | 竞赛常考 |
| **α,α'-二卤代酮** | 1,3-二氯代丙酮 | 可能二次重排 | 复杂 |

### 5.2 按亲核试剂分类

| 条件 | 亲核试剂 | 产物 |
|:---|:---|:---|
| **水溶液** | OH⁻ | 羧酸 |
| **醇溶液** | RO⁻ | 羧酸酯 |
| **胺溶液** | RNH₂ / R₂NH | 酰胺 |

## 六、适用条件与限制

1. **必须有 α-H 和 α-卤素**——缺一不可
2. **需要碱**——夺取 α-H 形成烯醇负离子
3. **环丙酮张力大**——若有更稳定的消除路径，可能竞争
4. **对强吸电子基敏感**——可能影响烯醇负离子稳定性
5. **开链底物产物可能混合**——开环方向不唯一

---

## 七、常见比较与易混点

### 7.1 Favorskii vs Benzilic 重排

- Favorskii：α-卤代酮 + 碱 → 羧酸（环丙酮中间体）
- Benzilic：α-二酮 + 强碱 → 乙醇酸（1,2-芳基迁移）
- 两者底物不同，但都涉及碳骨架重组

### 7.2 Favorskii vs 简单水解

- 简单水解：α-卤代酮 → 羟基酮（取代反应）
- Favorskii：α-卤代酮 → 羧酸（重排反应）
- 条件不同（弱碱水解 vs 强碱重排）

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：α-氯代环己酮在 NaOH 水溶液中的产物是什么？

**解答**：
- α-氯代环己酮 → Favorskii 重排
- 环丙酮中间体 → 开环
- 产物：**环戊烷甲酸**（六元环缩小为五元环）

### 10.2 例题 2（★★ 开环方向）

**题目**：2-氯-2-甲基环己酮经 Favorskii 重排后的产物是什么？

**解答**：
- 环丙酮中间体被 OH⁻ 进攻
- 开环时，取代基多的碳（带甲基的碳）保留负电荷更稳定
- 断裂另一根键 → **1-甲基环戊烷甲酸**

### 10.3 例题 3（★★★ 机理）

**题目**：画出 α-氯代丙酮 Favorskii 重排的环丙酮中间体。

**解答**：
- α-氯代丙酮：Cl-CH₂-CO-CH₃
- 烯醇负离子形成 → 分子内 SN2 → 环丙酮
- 环丙酮结构：三元环，其中一个顶点是羰基碳

## 十一、易错点

1. ❌ **Favorskii 产物是羟基酮** → Favorskii 产物是**羧酸/酯**（重排，不是取代）
2. ❌ **环酮 Favorskii 环扩大** → 环酮 Favorskii → **环缩小**（少一个碳）
3. ❌ **没有 α-H 也能 Favorskii** → **必须有 α-H** 才能形成烯醇负离子
4. ❌ **开链底物产物单一** → 开链底物开环方向可能**不唯一**
5. ❌ **Favorskii 和 Beckmann 一样** → 底物和机理完全不同

### 12.3 入门例题

**题目**：为什么 Favorskii 重排中会形成环丙酮中间体？

**引导**：
- α-卤代酮有 α-H 和 α-卤素两个关键基团
- 碱夺去 α-H → 烯醇负离子
- 烯醇负离子的碳进攻连卤素的 α-碳（分子内 SN2）
- 卤素离去 → 形成三元环（环丙酮）
- 这是分子内反应，熵变有利

### 12.4 现实类比

**Favorskii 重排像"折叠椅变凳子"**：
- 六条腿的折叠椅 = 六元环酮
- 把其中两条腿折进去（形成环丙酮）
- 然后一屁股坐下去（亲核进攻开环）
- 结果变成了五条腿的凳子 = 五元环羧酸
- 椅子变矮了（环缩小），但还能坐（保持功能）

## 十三、竞赛拓展

### 13.1 准 Favorskii 重排（Quasi-Favorskii）
- 无 α-H 的 α-卤代酮也能发生类似重排
- 机理不同：直接亲核取代而非环丙酮中间体

### 13.2 无法烯醇化的酮的二苯乙醇酸类 Favorskii 机理（Clayden Ch.36）

**关键发现**：Favorskii 重排并非总是通过环丙酮中间体。

**当酮没有可用的酸性氢时**（即无法烯醇化）：
- 碱不能夺取 α-H → 无法形成烯醇负离子。
- 但 Favorskii 重排仍可工作！
- **机理**：亲核试剂（RO⁻）直接进攻羰基 → 四面体中间体 → 类似二苯乙醇酸重排的芳基/烷基迁移 → 开环 → 酯。

**与二苯乙醇酸重排的对比**：
| 特征 | 二苯乙醇酸重排 | 无法烯醇化的酮的 Favorskii |
|:---|:---|:---|
| 底物 | α-二酮（Ar-CO-CO-Ar）| α-卤代酮（无 α-H）|
| 亲核试剂 | OH⁻ | RO⁻ |
| 中间体 | 四面体中间体 | 四面体中间体 |
| 断裂键 | C=O π 键 | C-X σ 键 |
| 迁移基团 | 芳基 | 烷基/芳基 |

**竞赛意义**：
- 判断 Favorskii 机理时，首先检查是否有 α-H。
- **有 α-H** → 环丙酮中间体机理（经典）。
- **无 α-H** → 二苯乙醇酸类机理（亲核加成-迁移）。

### 13.3 环丙酮中间体的周环机理（Clayden Ch.36）

**替代机理**：烯醇盐失去氯离子 → 氧烯丙基阳离子（oxyallyl cation）→ 两电子对旋电环化 → 环丙酮。

**与分子内 SN2 的关系**：
- 两种机理给出相同的环丙酮中间体。
- 竞赛中通常以分子内 SN2 为主要表述，但需知周环机理也是可能的。

### 13.4 Favorskii 在天然产物合成中的应用
- 调整环大小
- 从易得的环酮制备少一个碳的环羧酸
- **立方烷合成**（1964）：两步 Favorskii 将五元环缩为四元环
