# Robinson 托品酮合成（Robinson Tropinone Synthesis）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-总览]]
- 对应考纲条目：[[50-有机合成]]

## 一、定义

**Robinson 托品酮合成**（1917 年）是有机化学史上最著名的全合成之一。它以最简单的起始原料——琥珀醛（C4 二醛）、甲胺和丙酮二羧酸——在水溶液（pH 7）中一锅法构建托品酮（tropinone）骨架，巧妙模拟了自然界生物合成的 Mannich + 羟醛串联策略。

## 二、核心原理

### 2.1 反应序列

```
OHC-(CH2)2-CHO  +  MeNH2  +  CH2(CO2H)2
        ↓ pH 7, H2O, 25°C
    托品酮（双环酮）
        ↓ 还原
    托品（tropine）
```

### 2.2 机理拆解

1. **Mannich 反应**：琥珀醛的一个醛基与 MeNH₂ 形成亚胺 → 被丙酮二羧酸的活性亚甲基进攻 → 形成 C–N 键和新的 C–C 键。
2. **第二次 Mannich**：另一醛基重复上述过程，构建双环骨架中的第二个 C–N 键。
3. **分子内羟醛缩合**：二羧酸部分发生分子内羟醛反应，关环形成酮。
4. **脱羧**：两个羧基相继脱去 CO₂，得到托品酮。

> 1970 年证实此路线可在"自然条件"（pH 7 水溶液）下进行，与生物合成异曲同工。

## 三、关键结论

1. **一锅法串联 = 效率的极致**：Mannich × 2 + 羟醛 + 双重脱羧，四步协同在一锅中完成。
2. **丙酮二羧酸的角色**：既是 Mannich 反应的活性亚甲基供体，又是羟醛反应的羰基组分，最后通过脱羧"消失"，只留下所需的酮。
3. **pH 7 的生物学意义**：证明复杂的有机合成可以在不依赖强酸强碱的条件下进行，与酶催化环境的兼容性启示了后来的仿生合成。
4. 托品酮是颠茄生物碱（如阿托品）的**核心骨架**。
