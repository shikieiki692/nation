# Diels-Alder 反应（Diels-Alder Reaction）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[49-周环反应]]

---

## 一、定义

**Diels-Alder 反应（D-A 反应）**：**共轭二烯（diene）**与**亲双烯体（dienophile）**之间发生的 **[4+2] 环加成**，生成 **环己烯** 衍生物的协同反应。

**通式**：
$$
\ce{共轭二烯 + 亲双烯体 -> 环己烯}
$$

**本质**：**六电子协同环化**——二烯提供 4 个 π 电子，亲双烯体提供 2 个 π 电子，形成两根新的 σ 键和一根新的 π 键。

## 三、核心原理

### 3.1 反应机理

**协同的 [4+2] 环加成**：
```
步骤：六个 π 电子同时重组
- 二烯的 C1-C2 和 C3-C4 π 键断裂
- 亲双烯体的 C5-C6 π 键断裂
- 形成两根新的 σ 键（C1-C6 和 C4-C5）
- 形成一根新的 π 键（C2-C3）
```

**关键特征**：
- **一步协同**：无中间体，无催化剂（热反应）
- **立体专一**：syn 加成——二烯和亲双烯体的立体化学保留在产物中
- **可逆**：高温下可发生反向 D-A（Retro-Diels-Alder）

### 3.2 前线轨道分析

**HOMO(diene) + LUMO(dienophile)**：
- 二烯的 HOMO 与亲双烯体的 LUMO 对称性匹配 → **热允许**
- 二烯的 LUMO 与亲双烯体的 HOMO 也匹配（次要贡献）

**给电子基（EDG）在二烯上** → 升高二烯 HOMO 能量 → 反应更快
**吸电子基（EWG）在亲双烯体上** → 降低亲双烯体 LUMO 能量 → 反应更快

### 3.3 endo 规则（Alder endo rule）

**定义**：亲双烯体上的吸电子取代基倾向于在产物的 **endo 位**（桥位，靠近新形成的双键）。

| 产物类型 | EWG 位置 | 稳定性 | 控制类型 |
|:---|:---|:---|:---|
| **endo** | 桥位（靠近新双键）| 动力学优势 | 低温、短时间 |
| **exo** | 反桥位 | 热力学优势 | 高温、长时间 |

**次级轨道相互作用**：
- endo 过渡态中，亲双烯体的 π* 轨道（LUMO）与二烯的 HOMO 有额外的稳定化相互作用
- 这不是空间位阻效应，而是**轨道相互作用**

**Clayden 的轨道图像**（Clayden Ch34）：
- 画出两个组分相遇时的前线轨道：导致新键形成的轨道（黑色）对称性正确；同时，双烯后部的轨道（绿色）与亲双烯体共轭基团的轨道也存在对称性合适的成键相互作用。
- 这种跨空间的成键相互作用不会导致新键形成，但会在产物的立体化学中留下痕迹——**endo 产物因此有利**。

### 3.4 区域选择性（Regioselectivity）

**"邻位/对位"规则**：
- 给电子基在二烯的 **1-位** → 与亲双烯体的 EWG **邻位**
- 给电子基在二烯的 **2-位** → 与亲双烯体的 EWG **对位（1,4-位）**

**FMO 系数解释**：
- 二烯 HOMO 中，C1 和 C4 的系数最大
- 亲双烯体 LUMO 中，β-碳（连 EWG）的系数最大
- 最大系数相连 → 形成主要产物

**Clayden 深入解释**（Clayden Ch34）：
- 不对称二烯的 HOMO 发生变形：与亲电试剂反应的一端有更大的系数。这可以通过考察二烯与 HBr 的加成来理解——H⁺ 优先进攻 HOMO 系数较大的一端，形成更稳定的烯丙基阳离子。
- 亲双烯体的 LUMO 同样被共轭 EWG 变形：β-碳上的轨道系数比 α-碳大（与 Michael 加成的区域选择性同源）。
- 在过渡态中，每个组分中**最大轨道系数的两端会更率先成键**，这决定了区域选择性。
- **实用技巧**：画出"离子型"的分步机理（二烯作为亲核试剂、亲双烯体作为亲电试剂），找出反应位点，即可推断正确的协同 D-A 取向。这种分步机理并不完全正确，但可以指向正确的取向。

### 3.5 立体化学

**syn 加成**：
- 二烯的 cis/trans 构型保留在产物中
- 亲双烯体的 cis/trans 构型也保留
- 这是协同反应的必然结果（不可能 anti 加成）

### 3.6 反向 D-A（Retro-Diels-Alder）

$$
\ce{环己烯衍生物 ->[\Delta] 共轭二烯 + 烯烃}
$$

- 高温下（通常 > 200°C）D-A 产物裂解
- 是合成中**保护/释放双烯**的策略
- 常用于将双烯"隐藏"在环己烯中，需要时再释放

---

## 四、关键结论

1. **D-A = [4+2] 环加成 = 共轭二烯 + 亲双烯体 → 环己烯**
2. **协同机理，一步完成，无中间体**
3. **endo 规则**——EWG 倾向于 endo 位（动力学控制）
4. **区域选择性**——"邻对位规则"，由 FMO 系数决定
5. **立体专一**——syn 加成，cis/trans 构型保留
6. **EDG 在二烯，EWG 在亲双烯体**——反应最快
7. **高温可逆**——反向 D-A 释放双烯

## 五、常见分类或情形

### 5.1 按二烯分类

| 二烯类型 | 反应性 | 备注 |
|:---|:---:|:---|
| **开链顺式二烯** | ✅ 优秀 | 最经典 |
| **环状二烯**（如环戊二烯）| ✅ 优秀 | 反应极快 |
| **反式二烯** | ❌ 不行 | 无法达到 s-cis 构象 |
| **固定 s-cis 二烯** | ✅ 优秀 | 如环戊二烯 |
| **固定 s-trans 二烯** | ❌ 不行 | 如某些稠环体系 |

### 5.2 按亲双烯体分类

| 亲双烯体 | EWG | 反应性 |
|:---|:---|:---:|
| **马来酸酐** | 两个羰基 | 极强 |
| **丙烯醛** | -CHO | 强 |
| **丙烯酸甲酯** | -COOMe | 强 |
| **苯醌** | 两个羰基 | 极强 |
| **乙烯** | 无 | 弱（需高温高压）|

### 5.3 特殊 D-A 变体

| 变体 | 特点 | 例子 |
|:---|:---|:---|
| **杂 D-A** | 二烯或亲双烯体含杂原子 | 二氢吡喃合成 |
| **分子内 D-A** | 二烯和亲双烯体在同一分子 | 桥环合成 |
| **反向电子需求 D-A** | EWG 在二烯，EDG 在亲双烯体 | 某些杂环合成 |
| **[4+3] 环加成** | 氧烯丙基阳离子 + 双烯 | 七元环合成 |
| **苯炔捕获** | 苯炔作为亲双烯体 | 笼状结构合成 |

#### 分子内 D-A 的 endo 规则例外（Clayden Ch34）

分子内 D-A 反应（IMDA）中，endo 规则**经常被破坏**：
- **通常倾向于外型产物**：因为跨空间的成键相互作用不那么重要，空阻因素占主导。
- **但若有共轭羰基**：分子可通过折叠使羰基与双烯后部享受成键重叠 → 得到较不稳定的**顺式环交点**（内型）。此时连接链不得不采取类船式构象。
- **关键结论**：分子内 D-A "可能给出内型产物，也可能相反！请准备好，可能得到外型、内型产物或它们的混合物。"
- 产物中正在形成的环必须始终采取**船式构象**。

#### [4+3] 环加成（Clayden Ch34）

**氧烯丙基阳离子（oxyallyl cation）**与双烯的环加成：
- 由 α,α'-二溴代酮经 Zn 处理制得：先形成烯醇锌，再失去 Br⁻ 得到烯丙型阳离子。
- 三个原子、两个电子的阳离子 + 双烯（4e）→ **七元环**。
- 电子总数为六，与 D-A 相同，是允许的 [4+3] 全同面环加成。
- WH 分析：一个 π4s 组分 + 一个 π2s 组分 → 一个 $(4q+2)_s$ 组分，总和 = 1（奇数）→ 允许。

#### 苯炔的 D-A 捕获（Clayden Ch34）

- 苯炔是非常不稳定的亲电分子，具有较低能的 LUMO（三键的 π*）。
- 在双烯存在下生成苯炔 → 有效的 D-A 反应。
- 蒽 + 苯炔 → 具有对称笼式结构的产物。
- 两个平面分子在相互正交的平面上彼此靠近，苯炔定域 π 键的轨道与蒽中心环上的 p 轨道相互作用。

### 5.4 杂环合成中的 Diels-Alder反应（Ch.9 视角）

**杂 Diels-Alder（Hetero-Diels-Alder, HDA）**是构建**含氧/含氮六元杂环**的核心方法，在 ABOC Ch.9 杂环合成中具有重要地位。

#### 含氧杂环的构建

| 反应 | 二烯 | 亲双烯体 | 产物 | 应用 |
|:---|:---|:---|:---|:---|
| **二氢吡喃合成** | 不饱和醛/酮（杂二烯）| 烯烃 | 二氢吡喃 | 糖类骨架 |
| **香豆素合成** | 邻羟基肉桂酸酯 | — | 香豆素 | 天然产物 |
| **吡喃酮合成** | α-吡喃酮 | 烯烃 | 双环内酯 | 抗生素骨架 |

#### 含氮杂环的构建

| 反应 | 二烯 | 亲双烯体 | 产物 | 应用 |
|:---|:---|:---|:---|:---|
| **二氢吡啶合成** | 1-氮杂-1,3-丁二烯 | 烯烃 | 二氢吡啶 | Hantzsch 合成中间体 |
| **四氢吡啶合成** | 亚胺 + 共轭二烯 | — | 四氢吡啶 | 生物碱骨架 |

#### 呋喃作为二烯的 D-A反应

$$
\ce{呋喃 + 马来酸酐 ->[D-A] 氧杂桥环 ->[加热/反向D-A] 苯衍生物}
$$

- **呋喃**是**二烯组分**：其 2,3-位双键可作为二烯参与 D-A
- 产物为**氧杂桥环**（oxanorbornene 体系）
- **可逆性**：加热可发生反向 D-A 释放呋喃 → 用于**保护/脱保护策略**
- **在 Ch.9 杂环合成中的应用**：
  - 呋喃 D-A → 氧化开环 → 1,4-二羰基 → Paal-Knorr 吡咯合成
  - 这是 **D-A + 杂环合成** 的串联策略

#### 反向电子需求 D-A 在杂环合成中的应用

$$
\ce{1,2,4,5-四嗪 + 烯烃 ->[反向D-A] 哒嗪 + N2}
$$

- 四嗪（tetrazine）是强缺电子二烯
- 与烯烃发生反向 D-A → 释放 N₂ → 生成哒嗪
- **Diels-Alder 反应释放 N₂** 是构建含氮杂环的独特方法

**竞赛要点**：
- 看到 **呋喃 + 亲双烯体** → 考虑 D-A 形成氧杂桥环
- 看到 **1,2,4,5-四嗪** → 考虑反向 D-A 释放 N₂
- 杂 D-A 的机理分析同经典 D-A（协同、endo 规则适用）

## 六、适用条件与限制

1. **二烯必须能采取 s-cis 构象**——反式二烯不行
2. **需要热或光驱动**——通常加热即可
3. **立体位阻大会降低反应性**——大位阻二烯反应慢
4. **高温可逆**——产物在高温下可能裂解
5. **某些二烯会发生副反应**——如环戊二烯的自聚

### 6.1 水溶剂效应（Clayden Ch34）

- 水对 D-A 反应有**显著的加速作用**。
- 例：某反应在异辛烷中相对速率为 1，在水中为 **700**；endo:exo 选择性从 80:20 提高到 **96:4**。
- 原因：不溶于水的试剂被水的作用聚集在油滴中，被迫靠得很近。这类反应被称作"**在水上 (on water)**"的反应。
- 即使只在有机溶剂中添加一些水，都能加速反应。

### 6.2 Lewis 酸催化（Clayden Ch34）

- Lewis 酸（如 SnCl₄、AlCl₃、TiCl₄）与亲双烯体上的 EWG 络合 → 进一步降低 LUMO 能量 → 催化反应。
- **额外优势**：增加 LUMO 在不同碳上轨道系数的差异 → **提高区域选择性**。
- 例：某反应在 120°C 下区域选择性为 71:29；在 SnCl₄ 催化下，低于 25°C 即可反应，选择性提高到 **93:7**。
- 催化条件下反应温度更低，且无需密封管。

## 七、常见比较与易混点

### 7.1 D-A vs Ene 反应

- D-A：[4+2]，形成两根 σ 键 + 一根 π 键
- Ene：[2+2+2]，只形成一根 σ 键 + 双键迁移
- 两者都是六电子过程，六元环过渡态

### 7.2 D-A vs [2+2] 环加成

- D-A：热允许，协同六电子
- [2+2]：热禁阻（除烯酮例外），光允许
- D-A 产物是环己烯，[2+2] 产物是环丁烷

### 7.3 endo vs exo

- endo：动力学产物（低温快）
- exo：热力学产物（高温稳定）
- 竞赛中通常问"主要产物"= endo（动力学控制）

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：环戊二烯 + 马来酸酐的主要产物是什么？

**解答**：
- 环戊二烯是优秀二烯
- 马来酸酐是极强亲双烯体
- endo 规则 → 马来酸酐的羰基在 endo 位
- 产物：**内型-降冰片烯二酸酐衍生物**

### 10.2 例题 2（★★ 区域选择性）

**题目**：1-甲氧基-1,3-丁二烯 + 丙烯醛的主要 D-A 产物？

**解答**：
- 甲氧基在二烯 1-位（EDG）
- 丙烯醛的 EWG（-CHO）与甲氧基呈"邻位"
- 产物：**5-甲氧基-3-环己烯-1-甲醛**（邻位产物）

### 10.3 例题 3（★★★ 立体化学）

**题目**：顺,顺-2,4-己二烯 + 顺丁烯二酸二甲酯的 D-A 产物立体化学？

**解答**：
- syn 加成 → 二烯的 cis 关系保留
- 亲双烯体的 cis 关系也保留
- 产物中两个 COOMe 在**同侧**（cis）

### 10.4 例题 4（★★★ 杂 D-A）

**题目**：呋喃与马来酸酐在室温下反应，写出产物结构。若将产物加热至 150°C，又会发生什么变化？

**解答**：

**步骤 1：D-A 反应（室温）**
- 呋喃作为二烯 + 马来酸酐作为亲双烯体
- endo 规则 → 马来酸酐的羰基朝向桥环内侧
- 产物：**氧杂桥环**（7-氧杂双环[2.2.1]庚-5-烯-2,3-二羧酸酐）

**步骤 2：加热（150°C）**
- 发生**反向 D-A 反应**
- 环断裂 → 释放呋喃 + 马来酸酐
- 或：若体系中有其他亲双烯体 trapped，可发生交换

**反思**：
- 呋喃 D-A 的**可逆性**是其特征
- 这种可逆性在合成中可用于保护双烯或构建特殊骨架

---

## 十一、易错点

1. ❌ **endo 产物更稳定** → endo 是**动力学**产物，exo 更稳定（热力学）
2. ❌ **反式二烯也能 D-A** → 二烯必须能采取 **s-cis 构象**
3. ❌ **D-A 是离子型反应** → D-A 是**协同周环反应**
4. ❌ **高温有利于 endo** → 高温有利于 **exo**（热力学产物）
5. ❌ **区域选择性靠背的** → 区域选择性由 **FMO 系数**决定
6. ❌ **杂 D-A 机理不同** → 杂 D-A 与经典 D-A **机理相同**（协同、endo 规则适用），只是原子类型不同
7. ❌ **呋喃 D-A 产物稳定** → 呋喃 D-A 产物**可逆**，加热可发生反向 D-A

### 12.3 入门例题

**题目**：为什么 D-A 反应中，给电子基在二烯上、吸电子基在亲双烯体上时反应最快？

**引导**：
- EDG 在二烯 → 升高二烯 HOMO 能量
- EWG 在亲双烯体 → 降低亲双烯体 LUMO 能量
- HOMO-LUMO 能隙减小 → 反应活化能降低 → 反应更快

### 12.4 现实类比

**D-A 反应像"握手"**：
- 二烯 = 张开双臂的人（4 个 π 电子 = 两只手 + 两只脚）
- 亲双烯体 = 伸出一只手的人（2 个 π 电子 = 一只手）
- 两人走近 → 同时握住两只手 + 两脚扣住 → 形成一个环
- 整个过程一气呵成（协同），没有"只握一只手停住"的中间状态

## 十三、竞赛拓展

1. **不对称 D-A 反应**：
   - 手性 Lewis 酸催化（如 Corey 的恶唑硼烷）
   - 有机催化（如 MacMillan 催化剂）
   - 高对映选择性构建手性环己烯

2. **分子内 D-A（IMDA）**：
   - 二烯和亲双烯体通过 tether 连接
   - 合成桥环和稠环的天然产物骨架
   - 是萜类全合成的核心策略
   - **注意**：IMDA 的 endo 规则经常被破坏，可能得到外型、内型或混合物（Clayden Ch34）

3. **[4+3] 环加成**：
   - 氧烯丙基阳离子 + 双烯 → 七元环
   - 电子总数为六，与 D-A 相同
   - 是合成七元环的罕见但有效的方法（Clayden Ch34）

4. **苯炔的 D-A 捕获**：
   - 苯炔具有较低能的 LUMO，可作为亲双烯体
   - 蒽 + 苯炔 → 对称笼式结构
   - 为苯炔的存在提供令人信服的证据（Clayden Ch34）

5. **"在水上"反应**：
   - 水可大幅加速 D-A 反应并提高 endo 选择性
   - 是一种绿色化学策略（Clayden Ch34）
