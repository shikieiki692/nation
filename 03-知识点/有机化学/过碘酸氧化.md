# 过碘酸氧化（HIO₄ 氧化、Malaprade 反应）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]、[[基础要求-无机和结构化学]]
- 对应考纲条目：[[40-氧化还原反应]]、[[50-有机合成]]、[[13-元素化学]]
- 外部资料：[[Arrow Pushing in Inorganic Chemistry-总索引]] §7.13

## 一、定义
**过碘酸氧化（Malaprade 反应）**：使用 **过碘酸 HIO₄（或六羟基碘酸 H₅IO₆）** 这一类 **λ⁷-碘烷（I(VII) 试剂）** 在水/醇相中将 **顺式 1,2-二醇（vicinal diol）** 氧化裂解为两个羰基化合物（醛、酮或酸）的反应。

$$
\ce{R^1R^2C(OH)-C(OH)R^3R^4 ->[\text{HIO4 / H2O}] R^1R^2C=O + R^3R^4C=O + HIO3 + H2O}
$$

类似机理也适用于：
- α-羟基醛、α-羟基酮（断裂为两个羰基）
- α-羟基酸、α-羰基酸（断裂为羰基 + CO₂）
- α-氨基醇（断裂为醛 + 亚胺/氨气）
- 顺式 1,2-二胺（断裂为亚胺 / 醛）
- 1,2-二酮、1,2-二肟

## 二、考纲对应
- 对应考纲条目：[[40-氧化还原反应]]、[[50-有机合成]]、[[13-元素化学]]（I(VII) 化学）
- 所属模块：[[基础要求-有机化学]]
- 本知识点在考纲中的作用：是 **碳-碳键氧化裂解** 的最经典方法，与 [[铅四乙酸酯氧化]]（Pb(OAc)₄）平行发展，**糖类结构鉴定**和**天然产物降解**的基础工具。机理为 **环状碘酸酯 + 协同 [2σ+2σ] 重排**。

## 三、核心原理（3 步：环化 + 协同 + 水解）

### 3.1 第 1 步：环状碘酸酯的形成

顺式 1,2-二醇与 HIO₄（或 H₅IO₆）的两个 OH 进行**双 SN2-Si 置换**，形成 **5 元环状碘酸酯** 中间体。两个 OH 上的 H 转移到碘上的两个 OH，水分子离去：

$$
\ce{R^1R^2C(OH)-C(OH)R^3R^4 + H5IO6 ->[-2H2O] \underset{\text{5 元环碘酸酯}}{R^1R^2C-O-I(O)(OH)2-O-CR^3R^4} }
$$

> **关键点**：
> - 必须是 **顺式 1,2-二醇**（cis-diol），因为 5 元环 strain 要求两个 OH 共面相邻；
> - 反式（trans）位置太远，无法形成稳定环状中间体，**反应极慢**或不反应。
> - 这一点是 Malaprade 反应的 **立体选择性** 的来源。

### 3.2 第 2 步：协同 [2σ+2σ] 逆环加成

环状碘酸酯经 **5 元环协同过渡态**：
- C–C 键断裂（释放两个羰基的电子）
- 两个 C–O 键变为 C=O 双键
- 碘从 I(VII) → I(V)，得到碘酸 HIO₃ + H₂O

> 这一步是 **完全协同的、不可逆的**——碘"接走"两个 OH 的 H，C-C 同时断裂。

### 3.3 第 3 步：水解（如必要）

I(V) 中间体（hydrated iodate）失水脱氢，最终得 IO₃⁻（碘酸根）。

整体方程：

$$
\ce{R^1R^2C(OH)-C(OH)R^3R^4 + HIO4 -> R^1R^2C=O + R^3R^4C=O + HIO3 + H2O}
$$

碘从 +7 → +5，每个 C-OH 失 1 H，共失 2 H 与 2 电子。

## 四、关键结论

- **底物要求**：必须有 **相邻** 的两个含氧官能团（1,2-二醇、α-羟基醛、α-羟基酸等），且最好是顺式构型（5 元环过渡态）。
- **机理本质**：**碘催化的环状协同 [2σ+2σ] 逆环加成**，类似 [[铅四乙酸酯氧化]]（PbOAc₄）。
- **碘的电子流**：I(VII) → I(V)，每次反应释放 2 电子，将一个 C-C 键氧化裂解。
- 反应条件：**水相、室温**，可加 1:1 H₂O/MeOH 或乙醚作为助溶剂。
- **碳数变化**：1,2-二醇 → 2 个羰基，**碳骨架 C-C 断裂**——这是 Malaprade 反应的核心特征。

## 五、常见分类或情形

| 底物 | 产物 | 备注 |
|---|---|---|
| 1,2-二醇 RR'C(OH)-C(OH)RR' | 2 个醛/酮 | 经典 Malaprade |
| α-羟基醛 R-CH(OH)-CHO | RCHO + HCOOH | 糖类降解 |
| α-羟基酸 R-CH(OH)-COOH | RCHO + CO₂ + H₂O | 含羧基则脱羧 |
| 1,2-二酮 RC(=O)-C(=O)R' | 2 个 RCOOH（罕见） | 需 H₂O₂ 协同 |
| α-氨基醇 R-CH(NH₂)-CH(OH)R' | RCHO + R'CHO + NH₃ | |
| 顺式 1,2-二胺 | 2 个亚胺 | 较少 |
| 蒽-9,10-二醇 | 蒽醌 | 特殊 |

## 六、适用条件与限制

### 6.1 适用条件
- 溶剂：H₂O（最常用）、H₂O/MeOH、H₂O/醋酸/THF 混合
- 温度：0 °C → 室温
- pH：弱酸性（pH 3-5）最佳；强碱可能引起异构化
- 底物：顺式 1,2-二醇活性 >> 反式 1,2-二醇

### 6.2 限制
- **反式 1,2-二醇**（如反-1,2-环己二醇）反应慢得多，需要加热
- 碳骨架上若有 -OR、-NHR 等保护基会保护 OH，反应受阻
- 不能直接氧化 1,3-二醇或更远位置的二醇
- **过量** HIO₄ 会进一步氧化产物为羧酸（罕见）
- 涉及糖类时，可能出现"重复降解"——每个新生成的 α-羟基醛会继续被氧化裂解

## 七、常见比较与易混点

### 7.1 HIO₄ vs Pb(OAc)₄（铅四乙酸酯）

| 对比项 | HIO₄（过碘酸） | Pb(OAc)₄（铅四乙酸酯） |
|---|---|---|
| 中心金属 | 碘 I(VII) | 铅 Pb(IV) |
| 溶剂 | 水相 | 苯、CH₂Cl₂（无水） |
| 价态变化 | I(VII) → I(V) | Pb(IV) → Pb(II) |
| 适用底物 | 顺式 1,2-二醇为主 | 顺式 + 反式 1,2-二醇 |
| α-羟基酸 | 可裂 | 可裂 |
| 顺反差异 | 顺 > 反，差异显著 | 顺 > 反，差异较小 |
| 副产物 | HIO₃ | PbOAc₂ + 2 AcOH |
| 重金属污染 | 无 | 有 Pb |
| 现代偏好 | **HIO₄** | 较少 |

> **关键**：HIO₄ **强偏好顺式 1,2-二醇**；Pb(OAc)₄ 也走环状中间体，但环可"扭转"，对反式也有效。这种顺反差异可用于 **立体化学探针**。

### 7.2 HIO₄ vs DMP/IBX（[[Dess-Martin氧化]]）

| 对比项 | HIO₄ | DMP / IBX |
|---|---|---|
| 价态变化 | I(VII) → I(V) | I(V) → I(III) |
| 底物 | **1,2-二醇** | **单 OH 醇** |
| 反应类型 | C-C 裂解 | C-H 氧化 |
| 产物 | 2 个羰基（碳数减少） | 1 个羰基（碳数不变） |
| 溶剂 | 水相 | CH₂Cl₂ |

> **机理同源、产物分流**：都用环状碘酸酯过渡态，但底物决定走"两个 O 取代 C-C"还是"一个 O 取代 + α-H 拔取"。

### 7.3 与 [[O3氧化]]（臭氧分解）的对比
- O₃ 也能裂解 C=C 给两个羰基，但需要还原后处理（Me₂S 或 Zn/AcOH）；
- HIO₄ 只能裂解 1,2-二醇（C-C 单键 + 双 OH），**不能直接裂解烯烃**。

### 7.4 容易混淆的"过碘酸"
- **HIO₄**：偏过碘酸，I(VII)，不带水；
- **H₅IO₆**：正过碘酸（六羟基碘酸），I(VII)，相当于 HIO₄·2H₂O；
- 二者可互相转化，**反应活性近似**。在水相中以 H₅IO₆ 形式存在更稳定。

## 十、例题

### 例题 1：机理写出
**题目：** 写出 HIO₄ 氧化乙二醇 HO-CH₂-CH₂-OH 的完整机理（3 步）。

**分析：**
- 第 1 步：两个 OH 与 HIO₄ 的两个 OH 进行 SN2 置换，形成 5 元环状碘酸酯 $\ce{HOCH2-O-I(O)(OH)2-O-CH2OH}$（中央 I(VII)）+ 2 H₂O
- 第 2 步：协同 [2σ+2σ] 逆环加成：C-C 断裂、两个 C=O 形成，I(VII) → I(V) → 2 HCHO + H₃IO₅ + H₂O
- 第 3 步：H₃IO₅ → HIO₃ + H₂O（脱水）

### 例题 2：糖结构鉴定
**题目：** D-葡萄糖（醛糖，含 5 个 OH）用过量 HIO₄ 氧化。预测产物，并计算 HIO₄ 消耗量与 HCOOH 生成量。

**分析：**
D-葡萄糖含 5 个 OH（2,3,4,5,6 位）+ 1 个 CHO（1 位）。
- 1-CHO 与 2-OH 形成 α-羟基醛 → 第 1 个裂解：HCHO + 中间产物
- 沿链每对相邻位继续被氧化裂解...
- 最终：5 个 HIO₄ 消耗 → 5 个 HCOOH（中间 4 个 C 全部变为 HCOOH）+ 1 HCHO（C-1 端）+ 1 HCHO（C-6 端）

具体计数：
- 6 个 C 全部裂开 → 5 个 C-C 键被打断 → 5 mol HIO₄ 消耗
- 4 个内部 C 原子各变为 HCOOH（4 mol）
- 2 个端 C 原子（C-1 和 C-6）变为 HCHO（2 mol）

### 例题 3：立体选择性
**题目：** 比较 HIO₄ 氧化顺式-环戊烷-1,2-二醇 与 反式-环戊烷-1,2-二醇 的速率。

**解答：**
- 顺式：两个 OH 位于同侧，可形成 5 元环碘酸酯，**反应快**；
- 反式：两个 OH 反式排列，C-C 键无法弯曲到形成 5 元环，反应**极慢**。

> 这种立体差异可用于"区分顺反 1,2-二醇"，是经典的立体化学探针。

## 十一、易错点
- ❌ 误以为 HIO₄ 能氧化任何 OH：**只能氧化相邻位 OH**（1,2-二醇），独立 OH 不反应。
- ❌ 把 HIO₄ 与 IBX 混淆：HIO₄ 是 I(VII) 无机酸；IBX 是 I(V) 含芳基有机碘。
- ❌ 误以为对反式二醇也快：**顺式快，反式慢**——5 元环要求决定。
- ❌ 把 HIO₃ 写成产物 IO₃⁻：HIO₃（碘酸）和 IO₃⁻（碘酸根）等价，主要产物之一。
- ❌ 算错 C-C 断裂数：n 个 OH 的多元醇可被裂为 n+1 块（但相邻 OH 才被打断）。

## 十三、竞赛拓展
- **HIO₄ + RuO₄ 协同**：可将烯烃直接裂解为羧酸（一步法 Lemieux-Johnson）；
- **HIO₄ + KMnO₄**（Lemieux-Tropic）：另一种烯烃裂解；
- **NaIO₄ + Os(VIII)**：催化用量 OsO₄ + 化学计量 NaIO₄，是 cis-二羟基化 + 裂解的串联反应；
- **碘的"价态阶梯"**：I(-1) → I(+1) → I(+3) → I(+5) → I(+7)，分别对应 [[卤素]]、HOI、IBX、IO₃⁻、HIO₄；
- **生物化学应用**：HIO₄ 在蛋白质化学、糖化学、核酸化学的 **Smith 降解** 中是经典工具；
- 与 [[糖的构型与变旋]] 联用：通过 HIO₄ 消耗量 + 产物分布推断糖结构。
