# Semipinacol 重排（Semipinacol Rearrangement）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[39-重排反应]]

---

## 一、定义

**Semipinacol 重排**：**离去基不是水**，而是其他预先活化的基团（如 OTs、卤素、N₂）的**频哪醇型 1,2-迁移重排**。

**与 Pinacol 的区别**：Pinacol 中 OH 需在强酸下质子化脱水；Semipinacol 中的 OH 或其他基团已预先转化为良好离去基，反应条件更温和，**区域选择性更优**。

**通式**：
$$
\ce{R2C(X)-C(OH)R2' -> R2C-C(=O)R2'}
$$
其中 X = OTs, I, Br, N₂⁺ 等良好离去基。

## 三、核心原理

### 3.1 反应机理

```
步骤 1：离去基离去
- OTs、X 或 N₂ 自发离去（已预先活化）
- 生成碳正离子

步骤 2：1,2-迁移
- 相邻的 C–C 键迁移到带正电荷的碳上
- 迁移基团带着成键电子对移动

步骤 3：去质子
- 氧上的质子离去 → 形成羰基化合物
```

**"推-拉"描述**：
- **"拉"**：离去基（OTs/X/N₂）离去 → 产生碳正离子，"拉"动迁移基团
- **"推"**：氧孤对电子向正电中心推进 → 形成羰基

> 半频哪醇重排中，仍是羟基"推"迁移基团，而"拉"迁移基团的则由水换成了其他离去基团。

### 3.2 区域选择性优势

这是 Semipinacol 相对于 Pinacol 的**最大优势**。

| 控制方式 | Pinacol | Semipinacol |
|:---|:---|:---|
| **离去基选择** | 质子化竞争（热力学控制）| 预先 tosylate 或卤化 |
| **精度** | 中等 | **高**——想让它在哪边离去就在哪边 |
| **示例** | 不对称二醇选择性差 | 仲醇在叔醇存在下被选择性 tosylate |

**典型策略**：
1. 底物同时含有仲醇和叔醇
2. 对甲苯磺酰化时，**仲醇优先被 tosylate**（空阻较小）
3. OTs 离去 → 在仲碳上生成碳正离子
4. 叔碳上的烷基迁移 → 扩环/螺环产物

### 3.3 Tiffeneau-Demjanov 重排

**定义**：由 **2-氨基醇**衍生的重氮盐发生的 Semipinacol 重排。

**步骤**：
$$
\ce{氨基醇 ->[HNO2] 重氮盐 ->[-N2] 碳正离子 ->[1,2-迁移] 羰基化合物}
$$

**特点**：
- 氨基比羟基更容易被转化为离去基（重氮化）
- N₂ 是极好的离去基
- 常用于**扩环反应**

### 3.4 重氮甲烷扩环

$$
\ce{环己酮 + CH2N2 -> 环庚酮}
$$

**机理**：
1. CH₂N₂ 对羰基加成 → 质子转移 → 三元环中间体
2. 类似于 Semipinacol 的 1,2-迁移 → 扩环

**注意**：环戊酮 → 环己酮时反应会混乱（产物复杂），并非所有环都适用。

## 四、关键结论

1. **Semipinacol = 离去基预活化 + 1,2-迁移 → 羰基**
2. **区域选择性优于 Pinacol**——通过预先选择离去基位置控制
3. **Corey 长叶烯合成**是经典应用：仲醇 tosylate → 扩环酮
4. **Tiffeneau-Demjanov = 氨基醇 + 重氮化 → 扩环**
5. **迁移基团保持立体化学**——与所有 1,2-迁移一致
6. **扩环策略**：环状 β-取代醇 → Semipinacol → 环扩大一的酮

## 五、常见分类或情形

### 5.1 按离去基分类

| 离去基 | 来源 | 反应条件 | 典型应用 |
|:---|:---|:---|:---|
| **OTs** | 对甲苯磺酰化醇 | 弱碱或加热 | Corey 长叶烯合成 |
| **I/Br** | 卤代醇（如碘代醇）| Ag⁺ 促进 | 佛手柑油烯合成 |
| **N₂** | 重氮化氨基醇 | 亚硝酸 | Tiffeneau-Demjanov 扩环 |
| **N₂（CH₂N₂）** | 酮 + 重氮甲烷 | 质子源 | 环扩大一 |

### 5.2 按合成目的分类

| 目的 | 底物设计 | 产物 |
|:---|:---|:---|
| **扩环** | 环状 β-羟基-α-取代物 | 环扩大一的酮 |
| **螺环构建** | 双环 β-羟基物 | 螺环酮 |
| **骨架重组** | 复杂多环 β-羟基物 | 重排多环酮 |

## 六、适用条件与限制

1. **需有良好离去基**——OTs、X、N₂ 等
2. **需有邻位羟基或等价物**——氧最终成为羰基
3. **离去基与迁移基团需反叠**——立体电子效应要求
4. **迁移基团需能稳定正电荷**——芳基 > 烷基 > H
5. **重氮甲烷扩环对环大小敏感**——环戊酮体系不适用

---

## 七、常见比较与易混点

### 7.1 Semipinacol vs Pinacol

| 维度 | Semipinacol | Pinacol |
|:---|:---|:---|
| **离去基** | OTs, X, N₂（预先活化）| H₂O（酸催化脱水）|
| **条件** | 温和（Ag⁺、加热）| 强酸（H₂SO₄）|
| **选择性** | **高**（可预先控制）| 中等（热力学竞争）|
| **合成可控性** | **优秀** | 一般 |
| **竞赛频率** | 高（合成设计常用）| 中等 |

### 7.2 Semipinacol vs Tiffeneau-Demjanov

- **Semipinacol**：离去基 = OTs/X；底物 = β-取代醇
- **Tiffeneau-Demjanov**：离去基 = N₂；底物 = 2-氨基醇
- 两者机理相同，只是离去基来源不同

### 7.3 Semipinacol vs Grob 碎片化

- **Semipinacol**：电子源和电子槽**相邻**→ 1,2-迁移 → 重排
- **Grob 碎片化**：电子源和电子槽**相间（1,2,3,4）**→ C–C 键断裂 → 碎片化
- 关键区别：几何距离和电子流向不同

## 十、例题

### 10.1 例题 1（★★ 区域选择性）

**题目**：某分子同时含有仲醇和叔醇，如何利用 Semipinacol 重排实现选择性扩环？

**解答**：
1. 对甲苯磺酰化：仲醇空阻小 → **优先被 tosylate**
2. 叔醇保留，作为"推"的来源
3. OTs 离去 → 仲碳上生成碳正离子
4. 叔碳上的烷基迁移 → 扩环酮
5. **产物单一**，无区域异构体竞争

### 10.2 例题 2（★★★ Corey 长叶烯合成）

**题目**：长叶烯合成中有一步将五元环扩为六元环，指出该步反应名称和机理。

**解答**：
- **反应**：Semipinacol 重排
- **机理**：
  1. 仲醇被对甲苯磺酰化 → OTs
  2. OTs 离去 → 碳正离子
  3. 环内 C–C 键迁移 → 五元环 → 六元环扩环
  4. 氧去质子 → 酮

### 10.3 例题 3（★★★ Tiffeneau-Demjanov 扩环）

**题目**：由环己酮合成环庚酮，写出关键步骤。

**解答**：
$$
\ce{环己酮 ->[NaBH4] 环己醇 ->[SOCl2] 氯代环己烷 ->[NaN3] 叠氮化物 ->[H2/Pd] 氨基环己醇 ->[HNO2] 重氮盐 ->[-N2, 1,2-迁移] 环庚酮}
$$

**反思**：Tiffeneau-Demjanov 是 Semipinacol 的氨基变体，N₂ 作为离去基非常高效。

## 十一、易错点

1. ❌ **"Semipinacol 是碱催化"** → 离去基已预先活化，**不需要强酸**
2. ❌ **"Semipinacol 和 Pinacol 选择性一样"** → Semipinacol **选择性更高**
3. ❌ **"所有卤代醇都 Semipinacol"** → 需要 **β-羟基**且卤素为良好离去基
4. ❌ **"Tiffeneau-Demjanov 是独立反应"** → 它是 Semipinacol 的**氨基变体**
5. ❌ **"重氮甲烷扩环万能"** → 环戊酮 → 环己酮会混乱，不是万能的

### 12.3 入门例题

**题目**：为什么 Semipinacol 重排比 Pinacol 重排更容易控制区域选择性？

**引导**：
- Pinacol：两个 OH 竞争质子化 → 热力学控制 → 选择性有限
- Semipinacol： chemist 预先决定 tosylate 哪个 OH → 离去基位置**人为控制**
- **结论**：Semipinacol 把"选择性"从热力学交给了**化学家**

### 12.4 现实类比

**"预约拆迁 vs 强拆"**：
- **Pinacol** = 强拆——政府（酸）来决定拆哪家（质子化竞争），居民（羟基）被动等待
- **Semipinacol** = 预约拆迁——房主（化学家）提前签好协议（tosylate），指定哪天拆（OTs 离去），拆完邻居（迁移基团）立刻搬进来，商铺（羰基）马上开业
- **结果**：Semipinacol 拆迁有序、可控、产物单一

## 十三、竞赛拓展

### 13.1 天然产物合成中的 Semipinacol

| 天然产物 | 应用 | 特点 |
|:---|:---|:---|
| **长叶烯（longifolene）**| Corey 合成中的扩环 | 五元环 → 六元环 |
| **佛手柑油烯（bergamotene）**| 碘代醇扩环 | 2-碘代醇 + Ag⁺ |
| **保幼激素（juvenile hormone）**| Eschenmoser 碎片化前体 | 两次碎片化控制双键立体化学 |

### 13.2 与环氧开环的竞争

β-碘代醇在碱性条件下可能先形成环氧，再开环。Ag⁺ 的作用是提高碘的离去性，使 Semipinacol 快于环氧形成。
