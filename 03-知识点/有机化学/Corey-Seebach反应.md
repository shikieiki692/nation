# Corey-Seebach反应

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：羰基化学拓展（APiIC 占位）

## 一、定义

**Corey-Seebach反应**是指利用 **1,3-二噻烷（1,3-dithiane）** 作为 **羰基等价物**，实现 **羰基极性反转（umpolung）** 的一类反应。由 E. J. Corey 和 D. Seebach 于 1965 年发展。

核心思想：
- 醛/酮 + 1,3-丙二硫醇 → **1,3-二噻烷**（羰基被保护为硫代缩醛）
- 二噻烷 α-位去质子 → **酰基负离子等价物**（极性反转！）
- 该碳负离子进攻亲电体 → 水解恢复羰基 → 得到 **原醛/酮的 α-烷基化/酰基化产物**

## 二、考纲对应

- 中国化学奥林匹克考纲未单独列出，属于 **有机合成** 和 **羰基化学** 的高阶内容。
- 竞赛合成题中作为 **极性反转** 的经典策略出现。

## 三、核心原理

### 3.1 三步法流程

```
        O                          S                           S
        ||                        / \                         / \
    R-C-H  +  HS-CH₂   →      R-C   + H₂O   →   强碱 →    R-C⁻
              HS-CH₂                CH₂                         CH₂
               |                    |                            |
              CH₂                  CH₂                          CH₂

        S                              O
       / \                             ||
    R-C   + E⁺ → ... → 水解 →      R-C-E
       CH₂
        |
       CH₂
```

1. **保护**：醛/酮 + 1,3-丙二硫醇 + Lewis 酸（BF₃）→ 1,3-二噻烷
2. **极性反转**：n-BuLi 夺取二噻烷 2-位 H → 碳负离子（酰基负离子等价物）
3. **亲电捕获**：碳负离子进攻 RX、RCHO、RCOR' 等亲电体
4. **去保护**：Hg²⁺ 或 Ag⁺/NBS 等脱保护 → 恢复羰基

### 3.2 极性反转的本质

| 物种 | 正常羰基碳 | 二噻烷负离子 |
|:---|:---|:---|
| **极性** | 亲电（δ+）| 亲核（δ-）|
| **行为** | 接受亲核进攻 | 进攻亲电体 |
| **等价于** | 醛/酮 | 酰基负离子 |

**关键点**：正常醛/酮的羰基碳是亲电的，无法直接进攻 RX 等亲电体。二噻烷方法实现了 "醛 = 亲核体" 的反转。

## 四、关键结论

1. **1,3-二噻烷是醛/酮的 "保护 + 极性反转" 双功能等价物**
2. **二噻烷 2-位 H 有酸性**（pKa ~ 30-32，可用 n-BuLi 去质子）
3. **去保护需用 Hg²⁺ 或 Ag⁺**（硫对重金属高亲和性，水解释放羰基）
4. **可与多种亲电体反应**：RX（烷基化）、醛/酮（加成）、环氧（开环）

## 五、常见分类或情形

| 应用 | 亲电体 | 产物特征 |
|:---|:---|:---|
| **α-烷基化** | 伯卤代烷（RX）| 醛 → α-烷基醛 |
| **羟烷基化** | 醛/酮 | 醛 → α-羟基酮 |
| **酰基化** | 酯、酰氯 | 醛 → 1,2-二酮 |
| **环氧化** | 环氧乙烷 | 醛 → α,β-不饱和醛（脱水后）|

## 六、适用条件与限制

| 条件 | 说明 |
|:---|:---|
| **保护** | 1,3-丙二硫醇 + BF₃·Et₂O，室温 |
| **去质子** | n-BuLi, LDA, NaH，无水 THF，-78°C |
| **去保护** | HgCl₂/H₂O/CH₃CN 或 AgNO₃/NBS |
| **限制** | 二噻烷体积大，位阻敏感底物收率低；去保护有毒金属 |

## 七、常见比较与易混点

### Corey-Seebach vs 烯醇负离子烷基化

| 维度 | Corey-Seebach | 烯醇负离子 |
|:---|:---|:---|
| **底物** | 醛 | 醛/酮/酯 |
| **极性反转** | ✅ 羰基碳变亲核 | ❌ α-碳亲核 |
| **产物位置** | 原羰基碳上引入基团 | α-碳上引入基团 |
| **保护** | 自带保护 | 需额外保护基 |

## 十、例题

### 例题 1：合成设计

**题目**：由乙醛合成 2-甲基丙醛。

**解析**：
正常路线：乙醛烯醇负离子 + MeI → 但产物在 α-位，不是直接羰基碳上。

Corey-Seebach 路线：
1. 乙醛 + 1,3-丙二硫醇 → 2-乙基-1,3-二噻烷
2. n-BuLi 去质子 → 碳负离子
3. + MeI → 2-甲基-2-乙基-1,3-二噻烷
4. Hg²⁺ 水解 → **2-甲基丙醛**

### 例题 2：极性反转概念

**题目**：为何不能直接用乙醛与溴乙烷反应制备 2-丁酮？如何用 Corey-Seebach 解决？

**解析**：
- 乙醛羰基碳是亲电的，不能进攻溴乙烷（也是亲电的）
- Corey-Seebach：将乙醛转化为二噻烷 → 去质子得亲核碳 → 进攻溴乙烷 → 水解得 2-丁酮

## 十一、易错点

| 错误 | 纠正 |
|:---|:---|
| 与烯醇负离子烷基化混淆 | Corey-Seebach 是在 **原羰基碳** 上引入基团；烯醇负离子是在 **α-碳** 上引入 |
| 忽略去保护的必要性 | 二噻烷是保护基，反应后必须水解才能恢复羰基 |
| 去保护条件错误 | 不能用普通酸水解（硫代缩醛对酸稳定），需用 Hg²⁺/Ag⁺ |
