# 共轭效应

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[22-分子轨道理论]]、[[28-有机物结构的稳定性]]、[[29-共振论]]、[[32-有机反应机理]]

## 一、定义

**共轭效应（Conjugative Effect / Mesomeric Effect, 简称 +M / -M 或 +C / -C）** 是指在共轭体系中，**π 电子或孤对电子**通过 **π 轨道重叠**发生**离域**而产生的电子效应。其本质是 [[给体-受体相互作用]] 的一种特殊形式（π 给体 → π\* 受体）。

> 在 ABOC 的统一框架下，传统"超共轭"和"共轭"被统一描述为"给体-受体相互作用"。本 KP 聚焦于**给体是 π 体系或孤对电子**、**受体是 π\* 反键轨道**的情况。

### 1.1 历史定义（传统视角）

经典定义：
- **+M / +C 效应（给电子共轭）**：将孤对电子或 π 电子推向共轭体系
  - 例：-OH, -NH₂, -OR, -NR₂, -F, -Cl, -Br, -I（多数卤素都有弱 +M）
- **-M / -C 效应（吸电子共轭）**：从共轭体系拉出 π 电子
  - 例：-NO₂, -CN, -CHO, -COR, -COOR, -SO₃R, -NR₃⁺

### 1.2 现代定义（ABOC 视角）

**统一描述**：

任何一个**填充电子的 p / π 轨道（给体）**与**未填充的较低能级 π\* 反键轨道（受体）**之间的电子离域。涉及四种亚类：

| 亚类 | 给体 | 受体 | 例子 |
|:---|:---|:---|:---|
| **p-π 共轭**（孤对电子→π\*）| n（杂原子孤对电子）| π\* 反键 | OH 给苯环、烯基醚 |
| **π-π 共轭**（π→π\*）| C=C π 键 | 邻位 C=O 或 C=C π\* | 1,3-丁二烯、α,β-不饱和酮 |
| **σ-π\* 超共轭**（[[超共轭效应]] 一类）| C-H σ | π\* | 烷基稳定烯烃 |
| **σ-σ\* 超共轭**（[[超共轭效应]] 一类）| C-H σ | σ\* | E2 反式共平面 |

> **统一前提**：给-受体能级匹配 + **共平面**（满足 π 轨道重叠的几何要求）。

### 1.3 共轭体系的共平面要求

> ⚠️ **请不要对"共平面"这个词进行缩写**（与 [[超共轭效应]] 中"反式共平面"同等强调）。

考察典型共轭分子的几何：

| 分子 | 是否共平面 | 共轭效果 |
|:---|:---|:---|
| 1,3-丁二烯 | 平面 s-trans / s-cis | 强 π-π 共轭 |
| 联苯 | 通过芳环旋转 0° → 共平面（强）；90° → 垂直（无）| 取决于位阻 |
| 苯乙烯（PhCH=CH₂）| 共平面 | 强 |
| 邻甲苯衍生物 | 邻位甲基阻碍共平面 → 共轭减弱 | 中等 |
| 桥环（如降冰片烯）| 不共平面 | 弱或无 |

> ABOC 行 4015：**三氟甲氧基苯**因 O 孤对电子优先与三组 C-F σ\* 形成强超共轭，**O 完全失去与芳环的共轭**（O 实际处于 sp³ 杂化）。

## 三、核心原理

### 3.1 共轭体系的电子离域

**关键原理**：在共轭体系中，π 电子（或孤对电子）不再固定在两个特定原子之间，而是**离域**到整个共轭范围。

**例：丁二烯**
- 4 个 sp² C 共平面，4 个 p 轨道并排
- 4 个 p 电子离域到 4 个 π 分子轨道（成键 ψ₁、ψ₂；反键 ψ₃\*、ψ₄\*）
- **分子轨道（MO）描述**：电子分布在 ψ₁、ψ₂ 上
- **共振论（VB）描述**：通过多个共振式（极限式）的叠加表达
- 两种描述等价

### 3.2 重叠程度的关键因素

ABOC 行 1096：

> 影响共轭效应的因素：共轭效应好不好取决于**碳杂原子之间原子半径的差距**，实质是**能量相近原则**。

**轨道重叠程度** ≈ 给体与受体的：
1. 能级匹配程度（能级差越小越好）
2. 轨道半径相似度（差距小 → 重叠面积大 → 重叠强）

**典型 C-X 共轭强度顺序**：

$$\text{C-C} > \text{C-N} > \text{C-O} > \text{C-S} > \text{C-Cl} > \text{C-Br} > \text{C-I} > \text{C-F}$$

> 解读：C-C 共轭最好（同元素）；C-N 次之（半径差小）；C-F 最差（半径差大、能级差大）。

### 3.3 共轭链的电荷模式（"-+-+" 振荡）

**ABOC 关键洞察**（行 1094）：

> 共轭链（共轭结构上的电荷分布）：**负-正-负-正-……**

**生成机制**：
- 给电子杂原子（如 OH、NH₂）将孤对电子离域到 α-C → α-C 上电子云密度增加（δ-）
- α-C 与 β-C 之间形成"双键"（极限式）→ β-C 在共振式中带正电（δ+）
- β-C 又与 γ-C 之间...

**典型例子：苯酚**

```text
HO-C₁-C₂=C₃-C₄=C₅-C₆=C₁  →  通过共振：
δ-     δ+   δ-   δ+   δ-   ......
```

苯酚的邻位、对位富电子（亲电反应优先位置）；间位**相对**贫电子。

**典型例子：硝基苯**

```text
O₂N⁺=C₁-C₂=C₃-C₄=C₅-C₆=C₁  →  共振：
δ+     δ-   δ+   δ-   δ+   ......
```

硝基苯邻、对位贫电子（亲电反应不利）；间位相对富电子。

> **"-+-+"模式是共轭效应的"DNA"**——记住这个模式，可以快速预测富/贫电子位置。

### 3.4 自学练习 1.2.2-5（烯基醚的电子云分布）

**题目**：画出左边烯基醚共振式的轨道重叠图，解释共轭效应为何只能把电子给到氧的 **β 位**（距离氧 2 个原子）而不是氧的 **α 位**（与氧相邻的 C）。

**分析**：
- 烯基醚：R-O-CH=CH₂（O 与 C=C 共轭）
- O 的孤对电子 n → C=C 的 π\* 反键

**解答**：
1. **氧的孤对电子（n）**填入到**邻位 C-C 双键的 π\* 反键轨道**
2. C-C 双键的 π\* 反键轨道在两端 C 上具有大波瓣（一个正一个负相位）
3. 给体（n）只能填入与其相位匹配的一端
4. 通过共振式分析：给电子的极限式中，**β-C 上有负电荷**（孤对电子转过去了）
5. 因此**β 位富电子**

**反思**：共轭中"-+-+"的模式直接给出富电子位置预测。烯基醚中 β-C 是富电子的，因此亲电反应（如卤化、酰基化）发生在 β-C 上。

## 四、关键结论

### 4.1 综合电子效应（共轭 + 诱导）

ABOC 行 1098-1100：

> ▲ 综合诱导效应和共轭效应后得到的给电子与吸电子能力如下：

#### 4.1.1 给电子能力

$$\text{NR}_2 > \text{OR} > \text{F}$$

> 解读：
> - 共轭：N（半径 0.75）> O（0.66）> F（0.64）：与 C 半径越接近，共轭越强
> - 诱导：F > O > N：与 C 电负性差越大，诱导吸电子越强
> - 综合：**N 共轭赢得很多 + 诱导吸电子较弱 → 净给电子最强**

#### 4.1.2 吸电子能力

$$\text{F} < \text{I} \sim \text{Cl} \sim \text{Br} < \text{NO}_2 < \text{NR}_3^+$$

> 解读：
> - F：诱导吸电子很强（电负性 4.0），但共轭也较强（弱给电子），**综合相对最弱**
> - I/Cl/Br：诱导和共轭都中等
> - NO₂：双重 -M（O=N⁻=O 的两个共振式）
> - NR₃⁺（季铵盐）：永久正电荷，**仅诱导吸电子**（无共轭），综合最强

### 4.2 形式电荷的"强吸电子/强给电子"作用

ABOC 行 1102：

> 具有**形式负电荷**的原子具有**高度给电子性**；具有**形式正电荷**的原子具有**高度吸电子性**。

**典型例子**：

| 基团 | 形式电荷 | 综合作用 |
|:---|:---|:---|
| -O⁻（alkoxide）| -1 | 极强给电子（共轭+诱导双重正贡献）|
| -NR₃⁺（季铵盐）| +1 | 极强吸电子（仅诱导，但电荷大）|
| -O⁻ 或 -S⁻ | -1 | 极强给电子，可形成稳定芳香环（如苯酚负离子）|
| -NO₂ | N 形式带 +1（共振式中）| **强吸电子**（部分诱导 + -M 共振）|

### 4.3 自学练习 1.2.2-6（NO₂ 在烯烃上的共振式）

**题目**：画出当硝基（-NO₂）连在乙烯上面时物质的共振式，解释哪些位置富电子，哪些位置缺电子。

**分析**：
- 乙烯硝基化合物：CH₂=CH-NO₂
- NO₂ 的两个共振式：N=O（一个）+ N⁻-O⁻（另一个）

**解答**：
1. **基本式**：CH₂=CH-N⁺(=O)(-O⁻)
2. **共振式 1**：⁻CH₂-CH=N⁺(=O)(-O⁻) （C₁ 位负电荷）
3. **共振式 2**：⁻CH₂-CH=N⁺(-O)(=O) （另一种 N 共振）
4. **结论**：**β-C（远离 N 的 C）δ-**；**α-C（靠近 N 的 C）δ+**；**N 永远 +**；**O 永远 -**

**反思**：与烯基醚（O 给电子）相比，NO₂ 让 β-C 缺电子，因此该位置容易被亲核试剂攻击。

### 4.4 卤素的"反常"行为

卤素（F、Cl、Br、I）特殊：
- **诱导**：强吸电子（-I）
- **共轭**：弱给电子（+C，因为孤对电子可与芳环 π 共轭）
- **综合**：**吸电子**（-I 主导）

但是！在芳香亲电取代中：
- 卤素是**邻、对位定位**（+M 共轭主导反应位置选择）
- 卤素是**钝化**（-I 诱导主导反应活性）
- **同时邻、对位定位 + 钝化**——经典反例！

### 4.5 共轭吸电子取代基与非共轭基团的对比

| 基团 | 诱导作用 | 共轭作用 | 综合 |
|:---|:---|:---|:---|
| -F, -Cl, -Br, -I | 强 -I | 弱 +M | 综合吸电子（-I 主导）|
| -OH, -OR | 中 -I | 强 +M | 综合给电子（+M 主导）|
| -NH₂, -NR₂ | 弱 -I | 强 +M | 综合给电子（+M 主导）|
| -CH=CH-R | 0 | 0 | 主要是超共轭，中性 |
| -CN | 中 -I | 强 -M | 综合吸电子 |
| -NO₂ | 中 -I | 强 -M | 综合吸电子（最强之一）|

## 五、常见分类或情形

### 5.1 共轭体系类型

| 类型 | 给体 | 受体 | 例子 |
|:---|:---|:---|:---|
| **π-π 共轭** | C=C 的 π | 邻位 π\* | 1,3-丁二烯 |
| **p-π 共轭** | n（孤对电子）| π\* | 苯酚、烯基醚、酰胺 |
| **超共轭**（独立 KP）| C-H σ | π\* / σ\* | 见 [[超共轭效应]] |
| **跨共轭体系（cross-conjugation）**| 多个 π 体系交叉 | 多个 π\* | 醌类、间苯二酚 |
| **同共轭（homoconjugation）**| π 体系隔一个 sp³ C | π\* | 不连续共轭，弱 |

### 5.2 经典共轭体系

#### 5.2.1 1,3-丁二烯
- s-trans 构象（180°）：稳定，主要构象
- s-cis 构象：强共轭，但相对位阻

#### 5.2.2 苯环
- 6 个 p 轨道完全离域 → 6 个 π 电子均匀分布
- **芳香性**（4n+2 = 6）

#### 5.2.3 α,β-不饱和酮（迈克尔受体）
- C=C-C=O 共轭
- 极限式：C⁺-C-C=O⁻ 和 C⁻-C=C-O⁻
- **β-C 为亲电位置**（迈克尔加成的反应位）

#### 5.2.4 共轭酰胺（lactam）
- N-C=O：N 孤对电子 → C=O π\*
- 旋转能垒高（~ 20 kcal/mol）
- 限制肽键的旋转（蛋白质结构基础）

#### 5.2.5 烯醇（enol）
- C=C-O-H：与酸性互变异构体
- 正常烯醇（中性）vs 烯醇负离子（强稳定化）

### 5.3 共轭对反应活性的影响

#### 5.3.1 芳香亲电取代中的定位效应

| 取代基 | 类别 | 定位 | 活性 |
|:---|:---|:---|:---|
| -OH, -OR, -NH₂, -NR₂ | 强 +M | 邻、对 | 活化 |
| -CH₃, -CH₂R | 弱 +I（超共轭）| 邻、对 | 弱活化 |
| -F, -Cl, -Br, -I | -I + 弱 +M | 邻、对 | **钝化但邻、对定位** |
| -NO₂, -CN, -COR, -COOH | -M | 间 | 钝化 |
| -NR₃⁺ | -I | 间 | 强钝化 |

#### 5.3.2 烯醇负离子的稳定性

烯醇负离子 RC(=CR')-O⁻ 比烷氧负离子 R'O⁻ 稳定 ~ 10⁷ 倍，因为：
- p-π 共轭：O 的孤对电子离域到 C=C
- 共振式：RC⁻-CR'=O 和 RC=CR'-O⁻
- 负电荷分散到 O 和 α-C

### 5.4 共轭对几何的影响

**键长平均化**：
- 苯：所有 C-C 键均为 1.40 Å（介于单键 1.54 Å 和双键 1.34 Å 之间）
- 1,3-丁二烯：C₂-C₃ 单键缩短到 1.48 Å（接近双键）

**酰胺旋转能垒**：
- 普通 C-N 单键旋转能垒 ~ 1-2 kcal/mol（自由旋转）
- 酰胺 C-N 旋转能垒 ~ 20 kcal/mol（共轭使其具有部分双键性质）

## 六、适用条件与限制

### 6.1 推荐应用场景

- ✅ 解释稳定性差异（如苯酚 vs 醇）
- ✅ 解释酸碱性（苯酚酸性比醇强 ~ 10⁹ 倍）
- ✅ 预测亲电反应区域选择性（邻、对 vs 间）
- ✅ 预测亲核反应区域选择性（迈克尔加成 β-C）
- ✅ 解释颜色（共轭体系长度 → 吸收波长）
- ✅ 解释芳香性

### 6.2 不推荐应用场景

- ❌ 共平面被破坏的体系（如邻位位阻强的取代联苯）
- ❌ 给-受体能级差极大的体系（如 C-F 与 C=C 共轭，几乎不发生）
- ❌ 反应过渡态高度扭曲的情况（共轭被破坏）

### 6.3 与超共轭效应的边界

> 在传统术语中"共轭"特指 π → π\*，"超共轭"特指 σ → 任意。
>
> 在 ABOC 框架下，二者本质相同（[[给体-受体相互作用]] 不同亚类），但**习惯上仍区分使用**。

## 七、常见比较与易混点

### 7.1 共轭 vs 超共轭 vs 诱导

| 效应 | 给体本质 | 受体 | 传递 | 距离 |
|:---|:---|:---|:---|:---|
| [[诱导效应]] | 极性键的偶极 | 同 | σ 键依次极化 | 强衰减 |
| **共轭效应** | π 键 / 孤对电子 | π\* | π 体系 | 不衰减 |
| [[超共轭效应]] | σ 键 / 孤对电子 | σ\*/π\*/p | σ 重叠 | 中度 |

### 7.2 +M 与 +I 的对照

| 基团 | +I（诱导给）| +M（共轭给）| 综合 |
|:---|:---|:---|:---|
| -CH₃（烷基）| 弱 +I | 0 | 净给电子（弱）|
| -OH | -I（强吸电子）| +M（强给电子）| 给电子（+M 主导）|
| -F | -I（极强吸电子）| +M（弱给电子）| 吸电子（-I 主导）|
| -NH₂ | -I（弱吸电子）| +M（极强给电子）| 给电子（+M 主导）|
| -NO₂ | -I（中吸电子）| -M（强吸电子）| 强吸电子 |

### 7.3 -M 与 -I 的对照

| 基团 | -I（诱导吸）| -M（共轭吸）| 综合 |
|:---|:---|:---|:---|
| -NO₂ | 中 | 强 | 强吸 |
| -CN | 中 | 强 | 强吸 |
| -CHO | 弱 | 中 | 中吸 |
| -COOH | 中 | 中 | 中吸 |

### 7.4 易混点对照

| 易混点 | 区别 |
|:---|:---|
| 共轭 vs 超共轭 | 给体类型不同（π vs σ）|
| 共轭 vs 诱导 | 通过 π 体系 vs 通过 σ 键 |
| **+M 与 +I 的合作 vs 矛盾** | -OH（合作）vs -F（矛盾）|
| 邻、对位定位 + 活化 vs 钝化 | -OH（活化邻、对）vs -F（钝化但邻、对）|
| 烯基醚 vs 烯基酮 | β-C 富电子 vs β-C 缺电子 |

### 7.5 共振论与共轭效应的关系

- **共振论**：用极限式（resonance structures）描述电子离域 —— **定性表象**
- **共轭效应**：用轨道重叠（HOMO-LUMO）描述电子离域 —— **底层机理**

二者**互补**：极限式画法形象，但只能定性比较；轨道分析量化，但需要轨道想象。

## 十、例题

### 例题 1（ABOC 自学练习 1.2.2-5，行 1092）

**题目：** 画出烯基醚（R-O-CH=CH₂）的共振式，解释共轭效应为何只能把电子给到氧的 **β 位**而不是 **α 位**，并画出共轭链中富电子（δ-）和缺电子（δ+）位置。

**分析：** 这是 p-π 共轭的经典例子。

**解答：**
1. **极限式**：
   - 主：R-O-CH=CH₂（中性）
   - 共振式：R-O⁺=CH-CH₂⁻（O 给电子，β-C 富电子）

2. **共轭链分析**：
   - O：δ+（共振式中 O 上有正电）
   - α-C（靠近 O 的 C）：δ-（接受 O 给的电子）
   - β-C（远离 O 的 C）：**δ-**（共振式中带负电）

   等等！按 -+-+ 模式，应该是：
   - O：δ-（基础）
   - α-C：δ+
   - β-C：δ-
   
3. **正确分析**：从能量上看，孤对电子离域到 β-C（与 O 隔一个原子），而不是 α-C（直接相邻）。这是因为：
   - α-C 与 O 之间已经是单键（无 π 键中间体）
   - 离域路径必须**通过 π 体系**：O 孤对电子 → C=C π → 终点是 β-C

4. **画图**：极限式中 β-C 有形式负电荷 → β-C **富电子**

**反思：** 烯基醚中 β-C 富电子，是亲电反应的攻击位置（如卤化、酰基化）。这与"-+-+ 模式"完全一致：起点是 O（给电子），通过 -O-α-β 路径，β 位富电子。

### 例题 2（ABOC 自学练习 1.2.2-6，行 1104）

**题目：** 画出当硝基（-NO₂）连在乙烯上面时物质的共振式，解释哪些位置富电子，哪些位置缺电子。

**分析：** 这是 -M 共轭的经典例子，与例题 1 相反。

**解答：**

CH₂=CH-NO₂ 的共振式：

1. **基本式**：CH₂=CH-N⁺(=O)(-O⁻)
2. **共振式 1**（C 给电子到 N=O）：⁻CH₂-CH=CH-N⁺(=O)(-O⁻)（β-C 上有负电？）
3. **修正共振式 1**：⁻CH₂-CH=N⁺(=O)(-O⁻)（C₁ 上有形式负电，N 中心 +）

实际共振式：

| 位置 | 主式（中性）| 共振 1（电子流向 N）| 共振 2（O 拉电子）|
|:---|:---|:---|:---|
| C₂（β）| CH₂= | -CH₂⁻（负电）| CH₂= |
| C₁（α）| =CH- | -CH= | =CH- |
| N | -N⁺(=O)(-O⁻) | =N⁺(=O)(-O⁻) | -N⁺(-O⁻)(=O) |

> **结论**：β-C 缺电子（共振式中 β-C 上有 +）；O 始终带负电；N 始终带 +。

**反思：** 与烯基醚相比，β-C 在硝基烯烃中是缺电子的，因此该位置是亲核加成（如 Michael 加成）的位点。

### 例题 3（甲酸 Z/E 构型，与超共轭对比）

**题目：** 解释甲酸（HCOOH）的两种构象 Z（较稳定）和 E 中酸性差异。

**分析：** Z 构型涉及 n-π\* 共轭；E 构型涉及 σ-π\* 超共轭。

**解答：**
1. **Z 构型**：H-O 朝向羰基 O 同侧 → **羟基 O 的孤对电子（n）→ C=O π\***（强 p-π 共轭）
2. **E 构型**：H-O 朝向羰基 O 异侧 → **H-O σ 键 → C=O π\***（σ-π\* 超共轭，弱）

**比较**：
- 稳定性：n-π\* 共轭强 > σ-π\* 超共轭弱 → Z 更稳定
- 酸性：E 构型中 H-O σ 给电子让 H 缺电子 → 酸性强；Z 构型中 H 不直接参与共轭 → 酸性弱
- **结论**：Z 稳定但 E 酸性强！

**反思：** 同一分子的两种构象，共轭和超共轭"一个稳定原物，一个让 H 易离去"。这是 ABOC 经典的反直觉案例。

### 例题 4（苯酚的酸性）

**题目：** 解释为何苯酚（PhOH，pKa ~ 10）比环己醇（pKa ~ 18）酸性强约 10⁸ 倍。

**分析：** 苯酚的负离子（PhO⁻）通过共轭得到稳定化。

**解答：**
1. 苯酚负离子 PhO⁻ 的共振式：
   - PhO⁻ ↔ 邻位 C 上有形式负电荷 ↔ 对位 C 上有形式负电荷
   - 共 4 个等价（或近似等价）极限式
2. 环己醇负离子（C₆H₁₁-O⁻）：负电荷集中在 O 上，无共轭
3. **结论**：负电荷分散到苯环 → 苯酚负离子更稳定 → 苯酚酸性更强

**反思：** 这是共轭"分散负电荷"的经典案例，是芳香化合物酸性的核心机制。

### 例题 5（迈克尔加成区域选择性）

**题目：** α,β-不饱和酮（CH₂=CH-COMe）与亲核试剂（如 RNH₂）反应，亲核试剂攻击哪个 C？

**分析：** 共振式分析。

**解答：**
1. CH₂=CH-COMe 的共振式：
   - 主：CH₂=CH-C(=O)Me
   - 共振 1：CH₂⁺-CH=C(-O⁻)Me（β-C 缺电子）
2. **β-C 为缺电子位（亲电位）**
3. 亲核试剂 RNH₂ 攻击 β-C → 形成 RNH-CH₂-CH=C(-O⁻)Me（烯醇式）
4. 烯醇互变异构 → RNH-CH₂-CH₂-C(=O)Me（迈克尔加成产物）

**反思：** -M 共轭的"远端缺电子"是迈克尔加成的核心。共轭链中"-+-+"模式：O δ-，C₁ δ+（即 β-C）。

## 十一、易错点

### 11.1 忘记共平面要求

**错误**：认为只要有 π 体系就有共轭。

**正确**：必须**共平面**！如果分子限制了共平面（如邻位位阻、扭曲），共轭被破坏。

### 11.2 把"-+-+ 模式"搞反

**错误**：硝基苯邻、对位富电子。

**正确**：硝基（-M）让邻、对位**贫电子**；间位**相对**富电子。但**间位的"富电子"是相对的，不如苯酚的邻、对位富电子那么强**。

### 11.3 卤素的 +M / -I 矛盾

**错误**：F 共轭给电子比 NH₂ 强。

**正确**：F 的 +M 较弱（半径差大、能级差大）。但 F 在芳香亲电取代中**仍是邻、对位定位**（因为 +M 决定**位置**），同时**钝化**（因为 -I 决定**活性**）。

### 11.4 烯基醚 vs 烯基硝基化合物的方向搞反

**错误**：烯基醚和烯基硝基化合物都让 β-C 富电子。

**正确**：烯基醚（O 给电子）让 β-C **富电子**；烯基硝基化合物（NO₂ 吸电子）让 β-C **缺电子**。

### 11.5 忽视形式电荷的重要性

**错误**：氨基（-NH₂）和铵基（-NH₃⁺）共轭差不多。

**正确**：-NH₂（中性）有 +M（强给电子）；-NH₃⁺（带正电）**仅有 -I**（无共轭，因 N 上无孤对电子）→ 强吸电子。

### 11.6 共轭与超共轭混为一谈

**错误**：σ-π\* 也叫共轭。

**正确**：共轭专指 π 给体；σ-π\* 是超共轭（[[超共轭效应]]）。

### 11.7 把共振式数量当成稳定性"绝对值"

**错误**：苯酚负离子有 4 个共振式，烯丙基负离子有 2 个，所以苯酚负离子比烯丙基负离子稳定一倍。

**正确**：共振式数量是**定性比较**指标，不是定量。各极限式的"贡献度"也不同（电荷分布在电负性大原子上的极限式贡献更大）。

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析口诀/方法 |
|:---|:---|:---|:---|
| "共轭只是画极限式"| 教科书强调画图练习 | 极限式只是表象，**底层是轨道重叠**| 极限式 + HOMO-LUMO 双重视角 |

| "邻、对位定位 = 活化"| 总把两者绑定 | **F、Cl、Br、I 钝化但邻、对位定位**（反例）| 定位看 +M / -M；活性看 -I 主导还是 +M 主导 |

| "共轭酰胺自由旋转"| 把它当一般 C-N 单键 | 共轭 → 部分双键 → 旋转能垒 ~ 20 kcal/mol | 蛋白肽键的来源 |

### 12.3 入门级例题（不同于第十节）

**题目**：苯胺（PhNH₂）和苯酚（PhOH）哪个更易做亲电芳香取代？为什么？

**预期解答路径**：
1. 第一步看到题目应该想到：**这是亲电芳香取代的活化/钝化判断**。
2. 然后联想到本 KP：-NH₂ 和 -OH 都是 +M 给电子基团（活化基团）。
3. **比较** +M 强度：N 半径 0.75 vs O 0.66，与 C 的差距相近，但 N 电负性比 O 小，孤对电子能量更高，所以 -NH₂ 的 +M 更强。
4. 应用得到：**PhNH₂ 比 PhOH 更易亲电反应**（如苯胺易溴化为对溴苯胺，溴水即可反应）。

**老师可在课堂上引导的提问**：
- 学生若卡在第 1 步：问 "什么基团让芳环富电子？"
- 学生若卡在第 2 步：问 "-NH₂ 和 -OH 都给电子，但谁给得多？为什么？"
- 学生若答错（认为 OH 给电子强）：让学生比较 N 和 O 的电负性，再想孤对电子能量。
- 学生若忽视位阻：补充"虽然 NH₂ 共轭强，但邻位位阻可能影响实际反应位"。

### 12.4 与现实/直觉的连接

> 用学生熟悉的现象/比喻 anchor 这个抽象概念。

- **类比 1**：共轭就像"**有线电视**"——一组芳环就是一个"频道线路"，孤对电子（OH 上的）就是从输入端送进信号，接收端（环上的 C）能听到 → 富电子。
- **类比 2**：-+-+ 模式就像"**多米诺骨牌**"——给电子者倒下（产生 -），下一块（α-C）受冲击（变 +），再下一块（β-C）受推动（变 -），如此交替。
- **反例/佯谬**：表面上看 F 是"卤素 → 应该都吸电子"，但**苯环上的 F 是邻、对位定位**，与 OH、NH₂ 看起来像同一"队伍"——这是因为定位和活性是不同的两个维度。

## 十三、竞赛拓展

### 13.1 现代物理有机化学定量

#### 13.1.1 Hammett σ 参数

将共轭和诱导效应分离量化：
- σₚ：对位 σ 参数
- σₘ：间位 σ 参数
- σ⁺、σ⁻：增强 +M 或 -M 的修正版本

| 取代基 | σₚ | σₘ |
|:---|:---|:---|
| -NH₂ | -0.66 | -0.16 |
| -OH | -0.37 | 0.12 |
| -F | 0.06 | 0.34 |
| -Cl | 0.23 | 0.37 |
| -NO₂ | 0.78 | 0.71 |

> 解读：σ < 0 是给电子，σ > 0 是吸电子。**-F 的 σₚ 几乎为 0**（+M 与 -I 几乎抵消），但 σₘ > 0（间位仅有 -I 影响，吸电子）。

#### 13.1.2 共振参数 R

共振参数 R = σₚ - σₘ：
- 纯 +M：R < 0
- 纯 -M：R > 0
- 纯 -I（如 -NR₃⁺）：R ≈ 0

### 13.2 NBO（自然键轨道）分析

NBO 方法可量化共轭强度：
- **二阶微扰能 E⁽²⁾**：每组 n → π\* 或 π → π\* 的稳定化能
- 苯酚：约 35-40 kcal/mol（O n → 苯环 π\*）
- 酰胺：约 25-30 kcal/mol（N n → C=O π\*）
- 烯基醚：约 30 kcal/mol

### 13.3 量子化学计算
