# Peterson 烯化反应（Peterson Olefination）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[48-膦化合物]]、[[51-金属有机]]

## 一、定义

**Peterson 烯化反应**：醛（或酮）与 **β-硅基碳负离子**反应，经中间体质子化/消除，生成**烯烃**和硅醇副产物的羰基烯化反应。

**通式**：

$$
\ce{R_3Si-CHR'^- + R''CHO -> R_3Si-CHR'-CHR''-O^- ->[H+ 或 碱] R'CH=CHR'' + R_3SiOH}
$$

> Peterson 反应是 [[Wittig反应]] 和 [[Horner-Wadsworth-Emmons反应]] 的硅基等价版本，由 Donald Peterson 于 1968 年报道。

## 三、核心原理

### 3.1 试剂制备

**β-硅基碳负离子**的生成：

1. **α-硅基卤代物 + 金属**：
   $$
   \ce{R_3Si-CH_2X + 2Li -> R_3Si-CH_2Li + LiX}
   $$

2. **硅基稳定化**：Si 的 **β-硅基效应**（超共轭/σ* 稳定化）使 α-碳上的负离子稳定

### 3.2 反应机理（双路径）

#### 路径 A：碱性条件（一步协同消除）

```
     SiR₃                        SiR₃
      |                           |
   H₂C-CH(R')-O⁻  ──碱──>  [TS]  ──>  R'CH=CHR'' + R₃SiO⁻
      |   |                      
     R''                           
```

- 强碱（如 KH）夺取 α-碳上的质子
- **协同 E2 消除**：C–H 断裂与 C–Si 断裂同步，O⁻ 协助
- 产物：**E-烯烃为主**（热力学控制）

#### 路径 B：酸性条件（两步消除）

1. **质子化**：氧负离子质子化 → β-羟基硅烷
2. **酸催化消除**：
   - 质子化硅基氧（R₃SiOH₂⁺）→ 形成硅鎓离子或促进 C–Si 断裂
   - 或形成环氧硅烷中间体（oxasiletanide）
3. 产物：**Z-烯烃为主**（动力学控制）

**关键区别**：

| 条件 | 机理 | 主产物 | 控制类型 |
|:---|:---|:---:|:---:|
| **碱性** | 协同 E2 消除 | **E-烯烃** | 热力学 |
| **酸性** | 两步消除（硅鎓离子）| **Z-烯烃** | 动力学 |

### 3.3 驱动力

- **C=Si → Si–O 键转化**：Si–O 键极强（~530 kJ/mol），是热力学驱动力
- 副产物 R₃SiOH（或进一步缩合为硅氧烷）容易除去

## 四、关键结论

1. **Peterson = Wittig 的硅基版本**——用 Si 替代 P，机理类似但条件更温和
2. **酸/碱条件调控 E/Z 选择性**——这是 Peterson 反应最核心的优势
   - 碱 → E
   - 酸 → Z
3. **副产物为硅醇/硅氧烷**——通常易挥发或易除去
4. **试剂制备简单**——α-硅基卤代物易得，碳负离子稳定性适中
5. **对位阻敏感**——大位阻酮反应性差

## 五、常见分类或情形

### 5.1 按底物分类

| 底物 | 反应性 | 产物特征 |
|:---|:---:|:---|
| **醛** | 优 | 高产率，E/Z 可控 |
| **脂肪酮** | 中 | 产率中等 |
| **芳香酮** | 中 | 产率中等 |
| **大位阻酮** | 差 | 消除困难 |
| **α,β-不饱和羰基** | 有限 | 1,2-加成为主 |

### 5.2 按硅基分类

| 硅基 | 特点 | 应用 |
|:---|:---|:---|
| **Me₃Si–** | 标准 | 最常用 |
| **Et₃Si–** | 稍大位阻 | 调控选择性 |
| **Ph₃Si–** | 更大位阻 | 特殊底物 |
| **Me₂PhSi–** | 中等位阻 | 优化 E/Z |

---

## 六、适用条件与限制

1. **β-硅基碳负离子对水敏感**——需无水条件
2. **酸/碱条件决定立体化学**——不可随意选择
3. **大位阻底物难反应**——消除步骤受阻
4. **与 Wittig 相比**：Peterson 试剂更稳定，但原子经济性仍不佳（R₃SiOH 副产）
5. **非稳定化碳负离子选择性一般**——若无 α-吸电子基，E/Z 控制不如稳定化试剂

## 七、常见比较与易混点

### 7.1 Peterson vs Wittig vs HWE 三合一对比

| 维度 | Wittig | HWE | Peterson |
|:---|:---|:---|:---|
| **核心元素** | P | P | **Si** |
| **试剂** | Ph₃P=CHR（叶立德）| (RO)₂P(O)CHR⁻ | R₃Si-CHR⁻ |
| **副产物** | Ph₃P=O（难除）| (RO)₂P(O)O⁻（水溶易除）| R₃SiOH（易挥发/缩合）|
| **E/Z 控制** | 依赖叶立德稳定性 | 稳定化试剂 → E；Still-Gennari → Z | **酸→Z；碱→E** |
| **控制可靠性** | 中等 | 高（E）| **高（酸/碱切换）**|
| **底物范围** | 醛/酮均可 | 醛为主 | 醛/酮（酮位阻敏感）|
| **试剂稳定性** | 较低（需原位制备）| 高 | 高 |
| **竞赛频率** | 高 | 中高 | 中等 |

### 7.2 Peterson 反应的独特优势

- **酸/碱切换 E/Z**：这是 Peterson **独有的优势**——Wittig 和 HWE 难以通过简单条件切换实现 E/Z 反转
- 在需要**特定构型烯烃**的合成中，Peterson 是首选工具之一

## 十、例题

### 10.1 例题 1（★★ 基础）

**题目**：苯甲醛与 Me₃Si-CH₂Li 反应后，分别用 (a) KH 处理和 (b) 稀酸处理，产物有何不同？

**解答**：
- 第一步：亲核加成 → β-三甲硅基乙醇盐中间体
- (a) **KH（碱条件）**：协同 E2 消除 → **E-苯乙烯** 为主
- (b) **稀酸（酸条件）**：质子化后酸催化消除 → **Z-苯乙烯** 为主

### 10.2 例题 2（★★★ 机理）

**题目**：解释为什么 Peterson 反应在酸性条件下主要生成 Z-烯烃。

**解答**：
1. 酸条件下，氧先质子化 → β-羟基硅烷
2. 进一步质子化硅基（或通过硅鎓离子）→ C–Si 键极化
3. 消除时，两个大基团（R 和 R'）倾向于处于**同侧**（避免在过渡态中严重位阻排斥）
4. 或者从 oxasiletanide 中间体看：小环迫使取代基采取特定构型
5. 动力学控制下，生成位阻较小的 **Z-烯烃**

### 10.3 例题 3（★★★ 对比选择）

**题目**：合成 (Z)-3-苯基-2-丙烯酸甲酯，应选用 Wittig、HWE 还是 Peterson？具体条件？

**解答**：
- 目标：Z-烯烃
- **Wittig**：需非稳定化叶立德 → 可得 Z，但试剂不稳定
- **HWE**：标准 HWE → E；Still-Gennari → Z（需特殊试剂）
- **Peterson**：酸条件 → Z，试剂简单（Me₃Si-CH₂CO₂Me 的碳负离子）
- **推荐**：Peterson 酸条件，或 Still-Gennari HWE

## 十一、易错点

1. ❌ **把 Peterson 当 Wittig 变体死记** → 虽然功能类似，但试剂是 **碳负离子**（不是叶立德），机理也不同
2. ❌ **酸/碱条件选择性记反** → **碱 = E；酸 = Z**
3. ❌ **忽略 β-硅基效应的作用** → Si 不仅作为离去基团，还稳定了 α-碳负离子
4. ❌ **认为 Peterson 总是高选择性** → 非稳定化试剂选择性可能一般
5. ❌ **副产物当废物** → R₃SiOH 可缩合为 (R₃Si)₂O，有时需专门除去
6. ❌ **与硅醚混淆** → Peterson 是 **C–Si** 键反应，不是 O–Si（硅醚保护基）

### 12.3 入门级例题

**题目**：写出环己酮与 Me₃Si-CH(Li)Ph 反应后在酸处理下的主产物及立体化学。

**预期解答路径**：
1. 亲核加成 → β-硅基烷氧负离子
2. 酸处理 → 质子化
3. 酸催化消除 → **Z-1-苯基亚甲基环己烷** 为主
4. 原因：酸条件下动力学控制，大位阻基团避免过渡态排斥 → Z

### 12.4 与现实/直觉的连接

- **类比**：Peterson 反应就像一个"可控开关"——拨到"碱"档，产物按热力学走向（E，更稳定）；拨到"酸"档，产物按动力学走向（Z，生成更快）。这种"酸/碱切换"是有机合成中少有的简单立体控制手段。
- **反例**：直觉认为"酸和碱只是催化量不同"。但在 Peterson 反应中，酸和碱不仅改变速率，还**完全翻转**了产物的立体化学——这是条件决定选择性的经典案例。

## 十三、竞赛拓展

1. **Tebbe 试剂 / Takai 反应**：
   - 钛基烯化试剂，可将醛酮转化为烯烃
   - 与 Peterson 形成互补（对酯也可反应）

2. **Julia-Kocienski 烯化**：
   - 砜基替代硅基/磷基
   - 主要生成 E-烯烃

3. **硅导向的串联反应**：
   - Peterson 烯化 + 原位环氧化
   - 构建复杂多官能团骨架

4. **不对称 Peterson 反应**：
   - 手性碱或手性硅基实现不对称烯化
   - 构建手性烯丙醇类化合物
