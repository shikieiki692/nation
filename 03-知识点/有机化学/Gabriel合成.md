# Gabriel 合成（Gabriel Synthesis）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[44-脂肪族亲核取代反应]]

---

## 一、定义

**Gabriel 合成**：使用 **邻苯二甲酰亚胺（phthalimide）** 与 **卤代烷** 反应，再经 **肼解（hydrazinolysis）**，制备 **1° 胺（伯胺）** 的经典方法。

**通式**：
$$
\ce{Phthalimide + RX + KOH -> N-alkylphthalimide + N2H4 -> R-NH2}
$$

**本质**：利用邻苯二甲酰亚胺的 **N-H 酸性（pKa ~8.3）**，在温和碱性条件下形成负离子，与卤代烷发生 SN2 反应；酰亚胺结构阻止了过度烷基化，确保只生成 1° 胺。

## 三、核心原理

### 3.1 反应机理（三步）

```
步骤 1：邻苯二甲酰亚胺 + KOH → 邻苯二甲酰亚胺钾盐
步骤 2：亚胺负离子 + RX（SN2）→ N-烷基邻苯二甲酰亚胺
步骤 3：肼解（N₂H₄）→ 伯胺 + 邻苯二甲酰肼副产物
```

**关键特征**：
- **SN2 机理**：亚胺负离子是良好的亲核试剂
- **单烷基化**：酰亚胺的 N 上只有一个 H，只能反应一次 → **避免过度烷基化**
- **肼解脱保护**：N₂H₄ 选择性地断裂酰亚胺 C-N 键

### 3.2 为什么能避免多取代？

**直接氨 + RX 的问题**：
- NH₃ + RX → RNH₂（1° 胺）
- RNH₂ + RX → R₂NH（2° 胺）
- R₂NH + RX → R₃N（3° 胺）
- 产物是混合物，难以分离

**Gabriel 的优势**：
- 邻苯二甲酰亚胺的 N 上只有一个可取代的 H
- 烷基化后，N 上不再有 H → **不能进一步烷基化**
- 肼解后只释放 **1° 胺**

### 3.3 Fukuyama-Mitsunobu 变体

**从醇制备 1° 胺**：
- 醇 + 邻苯二甲酰亚胺 + PPh₃/DIAD（Mitsunobu 条件）→ N-烷基邻苯二甲酰亚胺
- 构型翻转（inversion）
- 然后肼解 → 1° 胺

**优势**：从醇出发，避免使用卤代烷。

### 3.4 其他脱保护方法

| 方法 | 试剂 | 特点 |
|:---|:---|:---|
| **经典肼解** | N₂H₄·H₂O / EtOH | 最常用，条件温和 |
| **酸水解** | HCl / 加热 | 强酸条件，可能不适合敏感底物 |
| **甲胺解** | MeNH₂ / EtOH | 替代肼，副产物更易溶 |
| **NaBH₄** | NaBH₄ / EtOH | 还原脱保护 |

---

## 四、关键结论

1. **Gabriel 合成 = 邻苯二甲酰亚胺 + RX → N-烷基酰亚胺 → 肼解 → 1° 胺**
2. **核心优势是避免多取代**——酰亚胺的单烷基化特性
3. **Fukuyama-Mitsunobu 变体可从醇出发**——构型翻转
4. **肼解是经典的脱保护方法**
5. **1° 卤代烷效果最好**——2° 也可以，3° 因 SN2 位阻大而困难
6. **若路线里发生一次 SN2 就翻转，一次 Mitsunobu 加一次 SN2 则可能表现为“表观保留”**

---

## 五、常见分类或情形

### 5.1 按底物分类

| 卤代烷类型 | 可行性 | 备注 |
|:---|:---:|:---|
| **1° 卤代烷** | ✅ 优秀 | SN2 最佳底物 |
| **2° 卤代烷** | ⚠️ 中等 | 位阻大，反应较慢 |
| **3° 卤代烷** | ❌ 困难 | SN2 几乎不发生 |
| **烯丙基/苄基卤** | ✅ 优秀 | SN2 活泼 |
| **芳基卤** | ❌ 不行 | SN2 在芳环上不发生 |

### 5.2 与其他制胺方法的对比

| 方法 | 试剂 | 产物 | 特点 |
|:---|:---|:---:|:---|
| **Gabriel** | 邻苯二甲酰亚胺 + RX + N₂H₄ | 1° 胺 | 无过度烷基化 |
| **还原胺化** | 醛/酮 + NH₃ + NaBH₃CN | 1° 胺 | 最直接，醛酮为原料 |
| **Ritter** | 腈 + 醇/烯 + H⁺ | 酰胺 → 胺 | 从腈出发 |
| **直接氨解** | NH₃ + RX | 混合胺 | 多取代问题 |

## 六、适用条件与限制

1. **底物必须是 1° 或活泼 2° 卤代烷**——3° 不行
2. **需要无水条件**——KOH 和 RX 都对水敏感
3. **肼解需要加热**——通常回流数小时
4. **邻苯二甲酰肼副产物难除去**——需萃取或柱层析
5. **对酸敏感底物不适用酸水解脱保护**

### 6.1 立体化学路线设计提醒

第三轮合成设计中，Gabriel 合成不只是“制伯胺”，还经常承担立体信息传递：

- 若关键成键步骤是 **一次 SN2**，构型应按 **Walden 翻转** 判断
- 若先经 **Mitsunobu 翻转**，再做 **SN2**，两次翻转后可表现为**表观保留**

因此看到手性醇或手性卤代物进入 Gabriel 路线时，别只盯官能团转化，还要同步数“翻转次数”。

## 七、常见比较与易混点

### 7.1 Gabriel vs 还原胺化

| | **Gabriel** | **还原胺化** |
|:---|:---|:---|
| **原料** | 卤代烷 | 醛/酮 |
| **机理** | SN2 + 肼解 | 亚胺 + 还原 |
| **多取代** | 不会发生 | 需控制当量 |
| **适用范围** | 1° 卤代烷 | 醛/酮 |

### 7.2 Gabriel vs 直接氨解

- Gabriel：单取代，产物纯
- 直接氨解：多取代混合物，分离困难

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：用 Gabriel 合成从 1-溴丙烷制备丙胺。

**解答**：
1. 邻苯二甲酰亚胺 + KOH → 钾盐
2. + CH₃CH₂CH₂Br → N-丙基邻苯二甲酰亚胺
3. + N₂H₄ → **丙胺（CH₃CH₂CH₂NH₂）**

### 10.2 例题 2（★★ Fukuyama-Mitsunobu）

**题目**：如何从 (S)-2-丁醇制备 (R)-2-丁胺？

**解答**：
1. (S)-2-丁醇 + 邻苯二甲酰亚胺 + PPh₃/DIAD → N-((R)-2-丁基)邻苯二甲酰亚胺（Mitsunobu，翻转）
2. 肼解 → **(R)-2-丁胺**

### 10.3 例题 3（★★★ 合成设计）

**题目**：为什么不能直接用过量氨与溴乙烷制备乙胺？

**解答**：
- 直接反应：NH₃ + EtBr → EtNH₂（1°）+ Et₂NH（2°）+ Et₃N（3°）+ Et₄N⁺Br⁻
- 产物是混合物
- Gabriel 合成可确保只得到 1° 胺

---

## 十一、易错点

1. ❌ **Gabriel 能制备 2° 胺** → Gabriel 只能制备 **1° 胺**
2. ❌ **3° 卤代烷也能用** → 3° 卤代烷 SN2 困难，**不适用**
3. ❌ **酸水解比肼解好** → 肼解更温和，是**首选**
4. ❌ **Gabriel 和直接氨解一样** → Gabriel **避免多取代**，产物纯
5. ❌ **Fukuyama-Mitsunobu 构型保持** → Mitsunobu 是 **inversion（翻转）**

### 12.3 入门例题

**题目**：为什么邻苯二甲酰亚胺能避免多取代，而氨不能？

**引导**：
- 邻苯二甲酰亚胺的 N 上只有一个 H
- 烷基化后 N 上没有 H 了 → 不能再烷基化
- 氨的 N 上有三个 H → 可以连续三次烷基化
- 这是 Gabriel 合成的核心设计智慧

### 12.4 现实类比

**Gabriel 合成像"限量供应"**：
- 邻苯二甲酰亚胺 = 限量发售的商品（每人限购一件）
- 卤代烷 = 顾客
- 反应后每个 N 上只有一个烷基（限购成功）
- 肼解 = 拆包装 → 释放出纯的 1° 胺
