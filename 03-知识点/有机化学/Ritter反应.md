# Ritter 反应（Ritter Reaction）
- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[45-亲核加成反应]]、[[47-羧酸及羧酸衍生物]]

## 一、定义

**Ritter 反应**：**醇**或**烯烃**在**强酸**条件下生成**碳正离子**，与**腈（R-CN）**反应，经水解生成 **N-取代酰胺** 的反应。

**通式**：
$$
\ce{R-OH + H^+ -> R^+ + R'-CN -> R-NH-C(=O)-R'}
$$

**本质**：碳正离子对腈氮的亲电进攻，随后水解生成酰胺。

## 三、核心原理

### 3.1 反应机理（四步）

```
步骤 1：醇/烯烃 + H⁺ → 碳正离子
步骤 2：碳正离子进攻腈的 N → 腈鎓离子（nitrilium ion）
步骤 3：水进攻腈鎓离子 → 亚氨酸酯（imidate）
步骤 4：互变异构/水解 → N-取代酰胺
```

**关键中间体**：**腈鎓离子（R-C≡N⁺-R'）**——腈的 N 被质子化/烷基化后形成的亲电物种。

### 3.2 底物范围

| 碳正离子来源 | 例子 | 可行性 |
|:---|:---|:---:|
| **3° 醇** | t-BuOH | ✅ 优秀（稳定碳正离子）|
| **2° 醇** | 环己醇 | ✅ 良好 |
| **1° 醇** | 乙醇 | ⚠️ 较差（需强酸，可能竞争）|
| **烯烃** | 异丁烯 | ✅ 良好 |
| **苄醇/烯丙醇** | PhCH₂OH | ✅ 优秀（稳定碳正离子）|

### 3.3 腈的范围

| 腈类型 | 例子 | 备注 |
|:---|:---|:---|
| **脂肪腈** | MeCN, EtCN | 通用 |
| **芳香腈** | PhCN | 反应性稍低 |
| **氢氰酸** | HCN | 产物是甲酰胺 |

### 3.4 与 Gabriel 合成和还原胺化的关系

| | **Ritter** | **Gabriel** | **还原胺化** |
|:---|:---|:---|:---|
| **产物** | 酰胺（可水解为胺）| 1° 胺 | 1°/2°/3° 胺 |
| **原料** | 醇/烯 + 腈 | 卤代烷 | 醛/酮 |
| **机理** | 碳正离子 | SN2 | 亚胺 + 还原 |
| **特点** | 从醇出发 | 避免多取代 | 最直接 |

## 四、关键结论

1. **Ritter = 醇/烯 + H⁺ + 腈 → 酰胺**
2. **碳正离子是核心中间体**——3° 醇和苄醇效果最好
3. **腈鎓离子是亲电中间体**——水是亲核试剂
4. **产物酰胺可水解为胺**——Ritter 是间接制胺方法
5. **强酸是必要条件**——浓 H₂SO₄、H₃PO₄ 等

---

## 五、常见分类或情形

### 5.1 按产物用途分类

| 目标 | 路线 | 应用 |
|:---|:---|:---|
| **制备酰胺** | Ritter 直接产物 | 酰胺类化合物 |
| **制备胺** | Ritter → 酰胺 → 水解/还原 → 胺 | 间接制胺 |
| **引入氨基** | 烯烃 → Ritter → 水解 | 从烯烃引入氨基 |

### 5.2 与其他腈反应的对比

| 反应 | 条件 | 产物 |
|:---|:---|:---|
| **Ritter** | H⁺ / 醇或烯 | N-烷基酰胺 |
| **腈水解** | H⁺ 或 OH⁻ / H₂O | 羧酸 |
| **腈还原** | LiAlH₄ | 1° 胺 |
| **腈与格氏试剂** | RMgX | 酮 |

---

## 六、适用条件与限制

1. **需要强酸**——浓 H₂SO₄ 最常用
2. **碳正离子必须相对稳定**——1° 醇效果不好
3. **腈不能含敏感官能团**——强酸条件下易分解
4. **反应温度控制**——过高导致碳正离子重排或副反应
5. **水是后处理时必须加入的**——水解腈鎓离子

---

## 七、常见比较与易混点

### 7.1 Ritter vs 腈的水解

- Ritter：醇/烯 + 腈 + H⁺ → N-烷基酰胺
- 腈水解：腈 + H⁺/H₂O → 羧酸
- 两者都用酸，但 Ritter 需要碳正离子来源

### 7.2 Ritter vs Gabriel

- Ritter：从醇/烯出发，产物酰胺
- Gabriel：从卤代烷出发，产物 1° 胺
- 两者都是制胺方法，但原料和产物不同

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：t-BuOH + CH₃CN + H₂SO₄ 反应的产物是什么？

**解答**：
- t-BuOH → t-Bu⁺（稳定碳正离子）
- t-Bu⁺ 进攻 CH₃CN 的 N → 腈鎓离子
- 水解 → **N-叔丁基乙酰胺**

### 10.2 例题 2（★★ 合成设计）

**题目**：如何从异丁烯制备 N-叔丁基乙酰胺？

**解答**：
- 异丁烯 + H⁺ → t-Bu⁺
- + CH₃CN → Ritter 反应
- 产物：**N-叔丁基乙酰胺**

---

## 十一、易错点

1. ❌ **1° 醇也能很好进行 Ritter** → 1° 醇碳正离子不稳定，**效果不好**
2. ❌ **Ritter 直接得胺** → Ritter 得**酰胺**，需进一步水解/还原才得胺
3. ❌ **弱酸也能做** → 需要**强酸**生成碳正离子
4. ❌ **Ritter 和腈水解一样** → Ritter 需要碳正离子来源，产物是 N-烷基酰胺

### 12.3 入门例题

**题目**：为什么 Ritter 反应中 3° 醇比 1° 醇效果好？

**引导**：
- Ritter 的第一步是生成碳正离子
- 3° 碳正离子稳定，容易生成
- 1° 碳正离子极不稳定，难以生成
- 因此 3° 醇效果远优于 1° 醇

### 12.4 现实类比

**Ritter 像"介绍对象"**：
- 醇/烯 = 单身人士（需要媒人介绍）
- 强酸 = 媒人（撮合双方）
- 碳正离子 = 单身人士的"激活状态"
- 腈 = 另一位单身人士
- 腈鎓离子 = 双方见面的"约会状态"
- 水解 = 结婚后安定下来 → 酰胺（稳定产物）
