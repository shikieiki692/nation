# Ugi 反应（Ugi Reaction）
- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[45-亲核加成反应]]、[[47-羧酸及羧酸衍生物]]

## 一、定义

**Ugi 反应**：**醛/酮**、**胺**、**羧酸**、**异腈**四组分在**甲醇**等质子性溶剂中**一锅法**反应生成 **α-酰氨基酰胺** 的多组分反应。

**通式**：
$$
\ce{R^1-CHO + R^2-NH2 + R^3-COOH + R^4-NC -> R^1-CH(NH-CO-R^3)-NH-CO-R^4}
$$

**本质**：亚胺（Schiff 碱）与异腈和羧酸的串联亲核加成——是**多组分反应（MCR, Multi-Component Reaction）**的经典代表。

## 三、核心原理

### 3.1 反应机理（五步）

```
步骤 1：醛/酮 + 胺 → 亚胺（Schiff 碱）+ H₂O
步骤 2：羧酸质子化亚胺 → 亚胺正离子
步骤 3：异腈（R-NC）进攻亚胺正离子 → 腈鎓离子（nitrilium）
步骤 4：羧酸负离子进攻腈鎓离子 → 亚氨酸酯混合酸酐
步骤 5：Mumm 重排 → α-酰氨基酰胺
```

**关键中间体**：**腈鎓离子**——异腈的碳端（形式上带正电）对亚胺正离子亲核进攻后形成。

### 3.2 四组分角色

| 组分 | 角色 | 可变性 |
|:---|:---|:---:|
| **醛/酮** | 提供亚胺的碳骨架 | 高 |
| **胺** | 提供亚胺的氮 | 高 |
| **羧酸** | 酸催化剂 + 酰基供体 | 高 |
| **异腈** | 亲核试剂 + 酰胺碳供体 | 高 |

### 3.3 与 Passerini 反应的关系

| | **Ugi 反应** | **Passerini 反应** |
|:---|:---|:---|
| **组分** | 4 组分（醛 + 胺 + 酸 + 异腈）| 3 组分（醛 + 酸 + 异腈，无胺）|
| **产物** | α-酰氨基酰胺 | α-酰氧基酰胺 |
| **中间体** | 亚胺正离子 | 醛直接质子化 |
| **机理** | Mumm 重排 | 类似，无胺步骤 |

**关系**：Passerini = Ugi 去掉胺组分（醛不预成亚胺，直接反应）。

### 3.4 异腈的化学本质

**异腈（isonitrile, R-N≡C）**：
- 形式上碳带负电荷、氮带正电荷（:C≡N⁺-R）
- 碳端是**亲核位点**——进攻亲电中心
- 是 MCR 中的"C₁ 合成子"

---

## 四、关键结论

1. **Ugi = 醛 + 胺 + 酸 + 异腈 → α-酰氨基酰胺**
2. **一锅法反应**——四组分同时加入，无需分离中间体
3. **核心中间体是腈鎓离子**——异腈的碳端亲核进攻
4. **产物含两个酰胺键**——高度官能团化
5. **Passerini 是 Ugi 的三组分变体**（无胺）

---

## 五、常见分类或情形

### 5.1 按组分限制分类

| 类型 | 条件 | 产物特点 |
|:---|:---|:---|
| **经典 Ugi** | 4 组分均不同 | 高度多样化 |
| **Ugi-Smiles** | 含酚羟基 | 酚氧参与成环 |
| **Ugi-DC** | 脱羧条件 | 产物简化 |

### 5.2 与 Aldol/Mannich 的对比

| | **Ugi** | **Aldol** | **Mannich** |
|:---|:---|:---|:---|
| **组分数** | 4 | 2 | 3 |
| **产物** | α-酰氨基酰胺 | β-羟基羰基 | β-氨基羰基 |
| **核心键** | C-C + 两个酰胺 | C-C | C-C + C-N |
| **异腈** | 必需 | 无 | 无 |

## 六、适用条件与限制

1. **需要质子性溶剂**——甲醇最常用
2. **醛比酮反应性高**——酮的亚胺形成较慢
3. **异腈有毒且有异味**——实验室操作需注意
4. **产物复杂，纯化困难**——多组分反应的通病
5. **立体化学控制困难**——经典 Ugi 非对映选择性一般

---

## 七、常见比较与易混点

### 7.1 Ugi vs Passerini

- Ugi：有胺，产物是 α-酰氨基酰胺
- Passerini：无胺，产物是 α-酰氧基酰胺
- 两者都用异腈和羧酸

### 7.2 Ugi vs Mannich

- Ugi：用异腈，产物含两个酰胺
- Mannich：用烯醇/烯胺，产物是 β-氨基羰基
- 两者都是 MCR，但化学完全不同

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：苯甲醛 + 苄胺 + 乙酸 + 叔丁基异腈的 Ugi 产物是什么？

**解答**：
- 苯甲醛 + 苄胺 → N-苄基亚胺
- 异腈进攻 → 腈鎓离子
- 乙酸参与 → Mumm 重排
- 产物：**N-苄基-N-(叔丁基甲酰基)-2-乙酰氨基-2-苯基乙酰胺**
- （竞赛中通常不要求写出全名，画出结构即可）

### 10.2 例题 2（★★ Ugi vs Passerini 区分）

**题目**：以下两组反应条件，哪个是 Ugi，哪个是 Passerini？
- A：醛 + 羧酸 + 异腈
- B：醛 + 胺 + 羧酸 + 异腈

**解答**：
- **A = Passerini**（3 组分，无胺）
- **B = Ugi**（4 组分，有胺）

## 十一、易错点

1. ❌ **Ugi 产物只有一个酰胺** → Ugi 产物有**两个酰胺键**
2. ❌ **Passerini 有胺参与** → Passerini **无胺**，Ugi 才有胺
3. ❌ **异腈的氮端进攻** → 异腈是**碳端**亲核进攻
4. ❌ **Ugi 是缩合反应** → Ugi 是**多组分反应**，机理包含亲核加成 + Mumm 重排

| "异腈氮端亲核" | 碳端亲核 | "异腈碳端是亲核头" |

### 12.3 入门例题

**题目**：为什么 Ugi 反应被称为"一锅法"反应？

**引导**：
- 四组分同时加入同一反应瓶
- 中间体（亚胺、腈鎓离子）不需要分离
- 一步操作直接得到高度官能团化的产物
- 这是 MCR 相比分步合成的核心优势

### 12.4 现实类比

**Ugi 反应像"四人一桌的麻将"**：
- 四个人（四组分）同时上桌
- 各自的角色互补（醛给碳、胺给氮、酸催化+给酰基、异腈给酰胺碳）
- 一局下来直接出结果（一锅法产物）
- 缺一个人就是另一种玩法（Passerini = 缺胺）
