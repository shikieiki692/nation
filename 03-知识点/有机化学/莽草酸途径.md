# 莽草酸途径（Shikimic Acid Pathway）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-总览]]
- 对应考纲条目：[[50-有机合成]]（生物合成视角）

## 一、定义

**莽草酸途径**是植物、真菌和细菌中合成芳香氨基酸（苯丙氨酸 Phe、酪氨酸 Tyr、色氨酸 Trp）的核心代谢通路。其核心骨架 **莽草酸**（shikimic acid）是多个六元环天然产物（如奎宁酸咖啡醇酯、奥司他韦前体）的共同中间体。

## 二、核心原理

### 2.1 起始：羟醛反应构建 C7 骨架

- **亲核试剂**：磷酸烯醇丙酮酸（PEP，烯醇等价物）
- **亲电试剂**：4-磷酸赤藓糖（C4 糖）
- **产物**：七碳 α-酮酸，带有一个新的立体中心

### 2.2 环化：分子内羟醛 + 脱水

- 未环化的 C7 α-酮酸可通过**分子内羟醛反应**关环。
- 关键步骤：C5 位羟基先被 **NAD⁺ 氧化为酮** → 邻近质子酸性增强 → **E1cB 机理**消除磷酸酯 → 形成烯醇 → 环化。
- 此设计避免了直接消除不活泼质子的困难。

### 2.3 从莽草酸到芳香氨基酸

- 莽草酸经多步酶催化转化为 **分支酸**（chorismic acid），再分支为：
  - → **预苯酸** → Phe / Tyr
  - → **邻氨基苯甲酸** → Trp

## 三、关键结论

1. **莽草酸途径是芳香环的"生物工厂"**：所有植物来源的芳香族化合物（包括药物前体）几乎均由此途径衍生。
2. **PEP 是关键的烯醇等价物**：自然界利用磷酸酯的离去能力实现脱水/烯醇化，与实验室化学中的烯醇锂策略异曲同工。
3. **NAD⁺/NADH 的氧化还原调控**贯穿全程：氧化增强酸性以允许 E1cB，还原恢复醇羟基。
4. 人类缺乏此途径，因此 Phe/Tyr/Trp 为**必需氨基酸**，必须从食物中获取。

## 五、竞赛拓展

- 莽草酸途径是除草剂（如草甘膦）的作用靶点：草甘膦抑制 EPSP 合成酶，阻断 PEP 与分支酸的缩合。
- 在竞赛信息题中，常给出 PEP + 磷酸赤藓糖 → … → 芳香氨基酸的某几步，要求推断中间体结构或反应类型。
