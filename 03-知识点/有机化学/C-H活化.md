# C–H 活化（C–H Activation）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[50-有机合成]]、[[51-金属有机]]

---

## 一、定义

**C–H 活化（C–H Activation / C–H Functionalization）**：在**过渡金属催化剂**（通常是 Pd、Ru、Rh、Ir）存在下，**直接断裂有机分子中的 C–H 键**并转化为 C–X、C–C 或 C–杂原子键的反应。

**通式**：
$$
\ce{R-H + [M] -> R-[M]-H -> R-Y + [M]}
$$

**本质**：**有机合成的"圣杯"**——跳过预官能团化步骤（如卤代），直接从最丰富的 C–H 键构建新键，是绿色化学和原子经济性的极致体现。

## 三、核心原理

### 3.1 为什么 C–H 键难活化？

| 特征 | 说明 | 后果 |
|:---|:---|:---|
| **键能高** | C–H 键能约 90–110 kcal/mol | 断裂需要高能量 |
| **极性低** | C 和 H 电负性相近 | 亲电/亲核进攻都困难 |
| ** everywhere** | 有机分子中到处都是 C–H | 选择性难以控制 |

**解决策略**：
- **导向基（Directing Group）**：引导金属到特定位置
- **应变释放**：张力环的 C–H 更容易活化
- **特殊金属**：Pd(II)/Pd(IV) 循环可活化芳环 C–H

### 3.2 导向基策略

**常见导向基**：

| 导向基 | 配位原子 | 典型位置 |
|:---|:---:|:---|
| **酰胺（-CONHR）** | O, N | 邻位 C–H |
| **吡啶** | N | 邻位 C–H |
| **羧酸（-COOH）** | O | 邻位 C–H |
| **肟醚（-C=NOR）** | N, O | 邻位 C–H |
| **亚胺（-C=NR）** | N | 邻位 C–H |

**导向基的作用**：
1. **配位金属**——将金属催化剂"拉"到附近
2. **定位**——金属只能到达配位点附近的 C–H
3. **降低活化能**——形成环金属中间体，稳定过渡态

### 3.3 机理类型

**CMD 机理（Concerted Metalation-Deprotonation）**：
- 最普遍的机理
- 金属与导向基配位
- **协同的** C–H 键断裂 + 去质子化
- 碱协助脱质子

**氧化加成机理**：
- Pd(II) → Pd(IV) 的氧化加成
- 形成 C–Pd(IV)–H 中间体
- 还原消除得到产物

**σ 键交换机理**：
- Ru、Rh 等金属的 C–H 活化
- 金属与 C–H 形成 σ 络合物
- 交换为新的 σ 键

### 3.4 Pd 催化芳基 C–H 活化

**典型循环**：
```
步骤 1：导向基配位
  底物-DG + Pd(II) → Pd(II)-DG 络合物

步骤 2：CMD
  Pd(II) 与邻位 C–H 形成环钯中间体

步骤 3：偶联伙伴插入
  引入第二组分（如烯烃、硼酸、卤代物）

步骤 4：还原消除 / 质子解
  生成产物 + Pd(II) 再生
```

## 四、关键结论

1. **C–H 活化 = 直接断裂 C–H 键 → 新官能团**
2. **导向基是控制选择性的关键**
3. **CMD 是最普遍的机理**
4. **Pd(II)/Pd(IV) 循环是核心**
5. **是合成化学的前沿方向**
6. **绿色化学：减少预官能团化步骤**

## 五、常见分类或情形

### 5.1 按反应类型分类

| 类型 | 产物 | 例子 |
|:---|:---|:---|
| **C–H 芳基化** | 联芳基 | 芳环 + 芳基卤 |
| **C–H 烯基化** | 烯基芳烃 | 芳环 + 烯烃（类似 Heck）|
| **C–H 硼化** | 芳基硼酸酯 | 芳环 + B₂pin₂ |
| **C–H 胺化** | 芳胺 | 芳环 + 胺源 |
| **C–H 烷基化** | 烷基芳烃 | 芳环 + 烷基源 |

### 5.2 按导向基分类

| 导向基类型 | 选择性 | 去除难度 |
|:---|:---:|:---:|
| **羧酸** | 高（邻位）| 易（脱羧/还原）|
| **酰胺** | 高（邻位）| 中 |
| **吡啶** | 高（邻位）| 难 |
| **临时导向基** | 高 | 易（设计为可去除）|

## 六、适用条件与限制

1. **需要过渡金属催化剂**（Pd、Ru、Rh、Ir）
2. **通常需要导向基**——无导向基的 C–H 活化选择性差
3. **配体设计至关重要**
4. **氧化剂/条件需精确控制**
5. **底物浓度影响**——避免分子间副反应
6. **目前工业应用有限**——主要停留在实验室阶段

---

## 七、常见比较与易混点

### 7.1 C–H 活化 vs 传统偶联

| | **C–H 活化** | **Suzuki/Heck** |
|:---|:---|:---|
| **预官能团化** | ❌ 不需要 | ✅ 需要（卤代物/硼酸）|
| **步骤经济性** | **高**（一步）| 低（需预官能团化）|
| **选择性控制** | 难（需导向基）| 易（底物已定位）|
| **成熟度** | 发展中 | 非常成熟 |

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：C–H 活化中导向基的作用是什么？

**解答**：
- **配位金属**：将催化剂"拉"到特定区域
- **定位**：限制活化只能发生在导向基附近
- **稳定中间体**：形成环金属中间体，降低活化能
- 没有导向基，C–H 活化几乎没有选择性

### 10.2 例题 2（★★ 机理）

**题目**：CMD 机理的全称是什么？其核心步骤是什么？

**解答**：
- **全称**：Concerted Metalation-Deprotonation（协同金属化-去质子化）
- **核心步骤**：
  1. 金属与导向基配位
  2. 碱协助 C–H 键断裂
  3. 同时形成 C–M 键和脱去 H⁺
  4. 整个过程是**协同的**，没有独立的碳负离子中间体

### 10.3 例题 3（★★★ 设计）

**题目**：如何用一个 C–H 活化步骤替代 Suzuki 偶联中的预官能团化？

**解答**：
- Suzuki：芳基溴 + 芳基硼酸 → 联芳基（需要预制备溴代物和硼酸）
- C–H 活化：芳环 + 芳基溴 → 联芳基（芳环直接使用）
- 条件：Pd(OAc)₂ / 膦配体 / 碱 / 酰胺导向基
- 优势：**省一步预官能团化**，更绿色
- 劣势：选择性不如 Suzuki 高

## 十一、易错点

1. ❌ **C–H 活化不需要导向基** → 目前阶段**通常需要导向基**
2. ❌ **C–H 活化是自由基机理** → 主要是**金属催化协同机理**（CMD）
3. ❌ **C–H 活化已经很成熟** → 仍在**快速发展中**，工业应用有限
4. ❌ **所有金属都能 C–H 活化** → 主要是 **Pd、Ru、Rh、Ir**

### 12.3 现实类比

**C–H 活化像"遥控手术"**：
- 有机分子 = 人体
- C–H 键 = 全身无数的细胞
- 导向基 = GPS 定位器（放在需要手术的部位）
- 金属催化剂 = 手术机器人
- GPS 引导机器人到目标位置 → 精准操作
- 没有 GPS，机器人不知道去哪里（没有选择性）

## 十三、竞赛拓展

1. **光氧化还原 + C–H 活化（Dual Catalysis）**：
   - 光催化产生自由基
   - 金属催化进行 C–H 活化
   - 两者协同实现传统方法做不到的反应

2. **无导向基 C–H 活化**：
   - 通过位阻或电子效应控制选择性
   - 目前选择性仍不如导向基策略
   - 是领域的终极挑战
