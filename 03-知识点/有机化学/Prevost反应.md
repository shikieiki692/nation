# Prevost 反应

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[37-加成反应]]、[[43-亲电加成反应]]

## 一、定义

**Prevost 反应**：烯烃在 **I₂ / AgOAc**、**严格无水**条件下发生的**反式双羟化**反应，产物为反式邻二醇（anti-diol）。

**本质**：利用碘鎓离子中间体的可逆开环，通过两次背面进攻实现反式双羟化。

## 二、考纲对应

- 对应考纲条目：[[37-加成反应]]、[[43-亲电加成反应]]
- 所属模块：[[基础要求-有机化学]]
- 本知识点在考纲中的作用：Prevost 反应是制备反式邻二醇的经典方法，与 Woodward 反应（顺式）形成对照。

## 三、核心原理

### 3.1 反应机理

**第一步：碘鎓离子形成**
- 烯烃 + I₂ → **碘鎓离子**（三元环中间体）

**第二步：第一次背面进攻**
- AcO⁻ 进攻碘鎓离子（背面进攻，反式开环）
- 生成碘代醋酸酯

**第三步：关键——可逆性**
- 无水时：碘鎓离子开环**可逆**
- 若产物不利，可回到碘鎓离子
- 系统通过两次背面进攻达到最稳定状态

**第四步：第二次背面进攻**
- 第二个 AcO⁻ 取代 I（SN2，再次背面进攻）
- 生成**反式二醋酸酯**

**第五步：水解**
- 二醋酸酯水解 → **反式邻二醇**

### 3.2 可逆性的关键作用

**ABOC 平衡思维解释**：
- 碘鎓离子开环是可逆的平衡过程
- 无水时：AcO⁻ 进攻 → 若构型不利，可回到碘鎓离子重新选择
- 最终系统趋向于两次都是背面进攻的最稳定状态
- 结果：**反式加成**

> "用平衡思维解释：水使中间体不可逆，微量即可改变选择性。"（ABOC §3.2）

## 四、关键结论

1. **Prevost 反应 = 反式双羟化（anti）**
2. **Woodward 反应 = 顺式双羟化（syn）**
3. **两者都经历碘鎓离子中间体**
4. **无水条件使中间体可逆 → 反式**
5. **有水条件使中间体不可逆 → 顺式**

## 五、常见分类或情形

### 5.1 双羟化方法对比

| 方法 | 试剂 | 立体化学 | 中间体 |
|:---|:---|:---:|:---|
| **OsO₄/NMO** | 催化 OsO₄，NMO 共氧化 | **syn** | 锇酸酯 |
| **稀冷 KMnO₄** | 0°C，稀溶液 | **syn** | 高锰酸酯 |
| **Woodward** | I₂ / AgOAc / **H₂O** | **syn** | 碘鎓离子（被水捕获）|
| **Prevost** | I₂ / AgOAc，**无水** | **anti** | 碘鎓离子（可逆开环）|
| **环氧开环** | mCPBA → 酸/碱水解 | **trans** | 环氧 |

### 5.2 应用场景

- **需要 anti-二醇** → Prevost
- **需要 syn-二醇** → Woodward 或 OsO₄
- **需要最便宜** → 稀冷 KMnO₄（但需控温）

## 六、适用条件与限制

1. **严格无水**——水的存在会翻转立体化学为顺式
2. **AgOAc 的作用**：提供 AcO⁻ 亲核试剂，Ag⁺ 协助 I⁻ 离去
3. **碘鎓离子开环的位置**：遵循碳正离子稳定性规则
4. **后续水解步骤**：二醋酸酯需水解才能得到游离二醇
5. **与 OsO₄ 相比**：Prevost 可得到反式二醇，OsO₄ 只能得到顺式

## 七、常见比较与易混点

### 7.1 Prevost vs Woodward

| | **Prevost** | **Woodward** |
|:---|:---|:---|
| 试剂 | I₂ / AgOAc，**无水** | I₂ / AgOAc / **H₂O** |
| 立体化学 | **anti** | **syn** |
| 中间体可逆性 | 可逆 | 不可逆（水捕获）|
| 产物 | 反式二醇 | 顺式二醇 |
| 记忆口诀 | "P 字倒过来，反式没水算" | "W 像水波，顺式有波" |

### 7.2 Prevost vs 环氧开环

| | **Prevost** | **环氧开环** |
|:---|:---|:---|
| 立体化学 | anti | trans（酸/碱开环）|
| 中间体 | 碘鎓离子 | 环氧 |
| 路线 | 一步双羟化 | 两步（环氧化 + 开环）|
| 选择性 | 中等 | 高 |

## 十、例题

### 例题 1（★ 基础）

**题目**：环己烯用 Prevost 试剂（I₂/AgOAc，无水）处理，产物是什么？

**解答**：
- Prevost 反应 = 反式双羟化
- 产物：**反-1,2-环己二醇**（两个 OH 在环的异侧）

### 例题 2（★★ 对比）

**题目**：如何从顺-2-丁烯出发制备 meso-2,3-丁二醇？

**解答**：
- meso-2,3-丁二醇 = 反式加成产物（两个 OH 在异侧）
- 顺-2-丁烯 → 要得 meso，需要 **anti 加成**
- 用 **Prevost（无水 I₂/AgOAc）**

## 十一、易错点

1. ❌ **混淆 Prevost 和 Woodward 的条件** → Prevost 无水 = anti；Woodward 有水 = syn
2. ❌ **认为 Prevost 是顺式** → Prevost 是 **anti**
3. ❌ **忽略无水要求** → 微量水即可将反应转变为 Woodward 条件
4. ❌ **与环氧开环混淆** → 环氧开环也得 trans，但路线不同

### 12.3 入门级例题

**题目**：解释为什么 Prevost 反应得反式二醇。

**解答**：
1. 碘鎓离子开环是可逆的
2. 无水时：AcO⁻ 进攻 → 若产物不利，可回到碘鎓离子
3. 系统通过两次背面进攻达到最稳定状态
4. 两次背面进攻 = 两次翻转 = 总体反式加成

### 12.4 与现实/直觉的连接

- Prevost 反应就像"反复尝试直到最优"——因为可以"倒车"（可逆），系统最终找到最稳定的反式产物。
- 记忆口诀："P 字倒过来，反式没水算"——P 倒过来像反式的箭头，没有水才是 Prevost。
