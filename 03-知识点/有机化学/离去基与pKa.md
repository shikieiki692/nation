# 离去基与pKa

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[31-有机反应的本质]]、[[36-取代反应]]、[[38-消除反应]]

## 一、定义
**离去基（Leaving Group, LG）**：在取代反应或消除反应中**带着电子对离开**的基团。

**核心规则**：**离去基越稳定，反应越容易**。其稳定性可用其**共轭酸的 pKa** 来量化：
- **pKa 越低**（共轭酸酸性越强）→ 离去基越稳定 → **越好的离去基**
- **pKa 越高**（共轭酸越弱）→ 离去基越不稳定 → **越差的离去基**

## 二、考纲对应
- 对应考纲条目：[[31-有机反应的本质]]、[[36-取代反应]]、[[38-消除反应]]
- 所属模块：[[基础要求-有机化学]]
- 本知识点在考纲中的作用：判断 SN1/SN2/E1/E2 反应可行性的关键——好离去基是反应进行的必要条件之一。

## 三、核心原理

### 3.1 离去基稳定性的本质
- 离去基带走电子对 → 稳定的离去基 = 稳定的负电荷
- 稳定的负电荷 = 共轭酸是强酸（容易解离）
- 所以**共轭酸越强（pKa 越小）= 离去基越好**

### 3.2 关键的 pKa - LG 关联
| 离去基 | 共轭酸 | pKa | 离去能力 |
|---|---|---|---|
| $\ce{TfO^-}$（三氟甲磺酸根） | $\ce{TfOH}$ | −14 | 极强（神童） |
| $\ce{ClO4^-}$ | $\ce{HClO4}$ | −10 | 强 |
| $\ce{TsO^-}$（对甲苯磺酸根） | $\ce{TsOH}$ | −2.8 | 强 |
| $\ce{I^-}$ | $\ce{HI}$ | −10 | 强 |
| $\ce{Br^-}$ | $\ce{HBr}$ | −9 | 强 |
| $\ce{Cl^-}$ | $\ce{HCl}$ | −7 | 强 |
| $\ce{H2O}$（中性） | $\ce{H3O^+}$ | −1.7 | 中 |
| $\ce{F^-}$ | $\ce{HF}$ | 3.2 | 弱（在 C 上） |
| $\ce{AcO^-}$（乙酸根） | $\ce{AcOH}$ | 4.8 | 弱 |
| $\ce{CN^-}$ | $\ce{HCN}$ | 9.2 | 极弱 |
| $\ce{HO^-}$ | $\ce{H2O}$ | 15.7 | 极弱 |
| $\ce{RO^-}$ | $\ce{ROH}$ | ~16 | 极弱 |
| $\ce{H^-}$ | $\ce{H2}$ | ~36 | 不可能 |
| $\ce{R^-}$（碳负） | $\ce{R-H}$ | ~50 | 极不可能 |

### 3.3 推断规则
- **好离去基**（pKa < 0）：可在中性条件下离去
- **中等离去基**（pKa 0–5）：需要弱酸催化
- **差离去基**（pKa > 5）：需先质子化或活化

### 3.4 主族化学中的特殊情况
- **F⁻ 是"差离去基"**？是 — 但仅在 **C 中心**上
- 在 **Si 中心**上，F⁻ 是**好离去基**！因 Si-F 键能巨大（~565 kJ/mol），但反应中 Si-F **形成**比断裂更有利
- 因此 SiCl₄ 与 HF 反应：Cl⁻ 离去，F⁻ 进入

## 四、关键结论

### 4.1 经常用的"激活"策略
| 原始 OH 基 | 激活成 | 反应 |
|---|---|---|
| $\ce{ROH}$ | $\ce{ROTs}$（与 TsCl）| 然后 SN2 |
| $\ce{ROH}$ | $\ce{R-OPO_3^-}$（与磷酸） | 在生物体中 |
| $\ce{ROH}$ | $\ce{R-OH2^+}$（在酸中） | E1 / SN1 |
| $\ce{R-NH2}$ | $\ce{R-N2^+}$（与 HNO₂）| 几乎万能离去 |
| $\ce{R-CO-OH}$ | $\ce{R-CO-Cl}$（与 SOCl₂）| 酰化反应 |

### 4.2 在 SN1 / SN2 / E1 / E2 中的角色
- **SN1**：好离去基**至关重要**——离去为限速步
- **SN2**：好离去基有利但不必要——亲核体也驱动
- **E1**：与 SN1 同——好离去基
- **E2**：好离去基有利

### 4.3 为何 F⁻ 在 C 上是差离去基
- C-F 键能极高（~485 kJ/mol）
- HF 的 pKa = 3.2（中性）
- 在水中 F⁻ 很差离去，但其碱性介于硬-软之间
- 唯一例外：超强亲核条件（如 LDA + R-F → 消除）

### 4.4 离去基与亲核体的关系
- 同一物种 X⁻：同时是好亲核**和**好离去基的有 I⁻、Br⁻、Cl⁻
- 大多数好离去基（TsO⁻、TfO⁻）也是较差亲核体（共振稳定）

## 五、常见分类或情形

### 5.1 中性离去基（最优）
- $\ce{N2}$（重氮基离去）
- $\ce{H2O}$
- 烯烃、CO 等

### 5.2 阴离子离去基（按 pKa 排序）
按上方表格

### 5.3 主族化合物的离去基
- $\ce{Cl^-}$ 从 Si 上离去（在 SiCl₄ 水解中）
- $\ce{F^-}$ 从 P 上离去（在某些条件下）
- $\ce{R3PO}$（亚膦酰胺）和 $\ce{R3SO}$（亚砜）作离去基

## 六、适用条件与限制

### 适用条件
- 任何 SN/E 反应
- 主族元素的亲核取代

### 限制
- pKa 仅在水中或质子性溶剂中精确
- 在金属催化中，"离去"可能涉及配位场效应
- 立体因素和邻基参与可改变实际"离去"速率

## 七、常见比较与易混点

### 1. 离去基好坏 ≠ 亲核性强弱
| 物种 | 离去能力 | 亲核能力 |
|---|---|---|
| $\ce{TsO^-}$ | 极好 | 较差（共振分散） |
| $\ce{HO^-}$ | 极差 | 强 |
| $\ce{I^-}$ | 好 | 好（双重身份） |

### 2. pKa 与 LG 的方向
- pKa **越小** → 共轭酸越强 → **共轭碱（即离去基）越稳定** → **越好离去**
- 注意：是 **共轭酸的 pKa**，不是离去基本身的 pKa

### 3. F⁻ 在 C 上 vs 在 Si 上
- 在 C 上：F⁻ 是差离去基
- 在 Si 上：F⁻ 是**好离去基**（仅在某些反应中）
- 一般在 Si-X 中：X = Cl/Br/I 是好离去基；X = F 较差

## 十、例题

### 例题 1：离去基排序
**题目：** 排序下列基团作为离去基的能力：$\ce{F^-, Cl^-, Br^-, I^-, OH^-, OTs^-}$。
**分析：** 用共轭酸的 pKa：HF (3.2)、HCl (-7)、HBr (-9)、HI (-10)、H₂O (15.7)、TsOH (-2.8)。
**解答：**$\ce{I^- > Br^- > Cl^- > OTs^- > F^- >> OH^-}$。

### 例题 2：激活 OH
**题目：** 为什么 $\ce{ROH + Cl^- \to RCl + OH^-}$ 不可行？如何使其可行？
**分析：**
- $\ce{OH^-}$ 是极差的离去基
- 解决方法：用 SOCl₂ 把 OH 转化为更好的离去基
**解答：**$\ce{ROH + SOCl2 \to RCl + SO2 + HCl}$（先把 OH 与 S 形成 R-O-S(O)Cl，然后 Cl⁻ 攻击 R）。

### 例题 3：苯酚类的特殊性
**题目：** 苯酚的 pKa = 9.95，比醇低很多。这对作为离去基有何意义？
**分析：** 苯酚的负离子（苯氧基）有共振稳定。
**解答：** 苯氧基（$\ce{PhO^-}$）作离去基比烷氧基（$\ce{RO^-}$）强 ~10⁶ 倍——但仍不是好离去基。芳醚的反应通常需要强酸活化。

## 十一、易错点
- 用离去基本身的 pKa（错——应用其共轭酸的 pKa）
- 把"碱性强"等同于"好离去基"——完全相反
- 忽视溶剂——在质子性溶剂中，差离去基（如 OH⁻）会被氢键稳定，"看起来"略好
- 在 Si 化学中错误套用碳化学的 LG 顺序
