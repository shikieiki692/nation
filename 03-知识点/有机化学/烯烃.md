## 一、定义

**烯烃（Alkene）**：含碳碳双键（C=C）的不饱和烃，通式 CₙH₂ₙ（单烯烃）。

**结构特征**：
- sp² 杂化，平面构型，键角 ~120°
- C=C 键长 ~134 pm，由一个 σ 键和一个 π 键组成
- π 键电子云分布于平面上下方，是亲电攻击的靶点

> 烯烃是**有机化学中最核心的官能团之一**，几乎整个加成反应章节都围绕烯烃展开。

## 三、核心原理

### 3.1 π 键的反应性来源

π 键电子云暴露在外，能量高于 σ 键 → 易供给电子给亲电试剂（E⁺）

$$
\ce{R2C=CR2 + E+ -> [R2C^+-CR2-E] -> 产物}
$$

### 3.2 亲电加成机理总览

| 试剂 | 中间体 | 立体化学 | 区域选择性 |
|:---|:---|:---:|:---|
| HX | 碳正离子 | 无（平面）| 马氏 |
| X₂ (Cl₂, Br₂) | 卤鎓离子 | **反式** | — |
| H₂O/H⁺ | 碳正离子 | 无 | 马氏 |
| BH₃ | 四中心过渡态 | **syn** | **反马氏** |
| RCO₃H | 环氧化（无中间体）| **syn** | — |
| OsO₄ / KMnO₄(稀冷) | 环状锇酸酯/锰酸酯 | **syn** | — |
| O₃ | 臭氧化物 | — | 断裂 |
| H₂ / Pd | 表面吸附 | **syn** | — |

### 3.3 马氏规则与反马氏加成

- **马氏加成**：H⁺ 加到含 H 较多的碳 → 生成更稳定的碳正离子（见 [[马氏规则]]）
- **反马氏加成**：
  - 硼氢化-氧化：B 加到取代较少的碳（位阻 + 电子效应）
  - 过氧化物效应（HBr/ROOR）：自由基机理，Br· 加到取代较少的碳

### 3.4 立体化学控制

烯烃加成中立体化学是竞赛高频考点：

| 反应 | 立体化学 | 机理原因 |
|:---|:---:|:---|
| 溴水加成 | **反式** | 溴鎓离子中间体，Br⁻ 从背面进攻 |
| 硼氢化-氧化 | **syn** | 四中心过渡态，H 和 B 同侧加成 |
| 催化氢化 | **syn** | 金属表面吸附，H₂ 从同一侧加入 |
| 稀冷 KMnO₄ / OsO₄ | **syn** | 环状金属酸酯中间体，同侧双氧 |
| 环氧化 | **保持** | 过酸直接转移氧，烯烃构型保留 |

## 四、关键结论

1. **π 键是亲电加成的核心**——几乎所有烯烃特征反应都始于 E⁺ 对 π 的进攻
2. **中间体类型决定立体化学**——碳正离子（无立体控制）vs 鎓离子/环状中间体（syn 或 anti）
3. **马氏/反马氏取决于机理类型**——离子型一般马氏；硼氢化和自由基型反马氏
4. **烯烃是结构推断的关键节点**——臭氧化断裂可反推双键位置
5. **氧化程度递进**：环氧化 → 双羟化 → 臭氧化断裂（控制氧化深度是合成关键）

## 五、常见分类或情形

### 5.1 按取代度分类

| 类型 | 结构 | 反应性 | 备注 |
|:---|:---|:---|:---|
| 乙烯 | CH₂=CH₂ | 标准参照 | 无区域选择性 |
| 单取代 | R-CH=CH₂ | 高 | 有区域选择性 |
| 双取代（1,1-）| R₂C=CH₂ | 高 | 马氏产物单一 |
| 双取代（E/Z）| R-CH=CH-R' | 中 | 有立体化学 |
| 三取代 | R₂C=CH-R' | 中高 | 位阻增大 |
| 四取代 | R₂C=CR'₂ | 较低 | 位阻最大 |

### 5.2 按反应类型分类

| 反应类型 | 典型试剂 | 产物 | 竞赛重要性 |
|:---|:---|:---|:---:|
| **亲电加成** | HX, X₂, H₂O/H⁺ | 卤代烷、醇 | ★★★★★ |
| **硼氢化-氧化** | BH₃ / H₂O₂, NaOH | 醇（反马氏）| ★★★★★ |
| **环氧化** | mCPBA, RCO₃H | 环氧化物 | ★★★★ |
| **双羟化** | OsO₄, KMnO₄(稀冷) | 邻二醇（syn）| ★★★★ |
| **臭氧化断裂** | O₃ / Zn-H₂O 或 Me₂S | 醛/酮 | ★★★★★ |
| **催化氢化** | H₂ / Pd, Pt, Ni | 烷烃 | ★★★ |
| **自由基加成** | HBr / ROOR | 溴代烷（反马氏）| ★★★★ |
| **α-H 卤代** | NBS / hν 或 ROOR | 烯丙基溴 | ★★★★ |

## 六、适用条件与限制

1. **烯烃与 KMnO₄ 热浓 = 彻底氧化断裂**——与稀冷条件下的双羟化完全不同，条件决定产物
2. **臭氧化还原工作至关重要**——Zn/AcOH 或 Me₂S 做还原性工作得醛/酮；H₂O₂ 做氧化性工作得羧酸
3. **四取代烯烃难硼氢化**——位阻过大，(Sia)₂BH 等位阻硼烷可解决部分问题
4. **共轭二烯加成有 1,2- vs 1,4-选择**——见 [[直接加成与共轭加成]]
5. **烯烃对强氧化剂敏感**——选择合适氧化剂和条件是有机合成的基本功

## 七、常见比较与易混点

### 7.1 烯烃 vs 炔烃反应性

| 特征 | **烯烃** | **炔烃** |
|:---|:---|:---|
| 杂化 | sp² | sp |
| 亲电加成速率 | **快** | 慢（烯基正离子不稳定）|
| 亲核加成 | 一般不 | **可以** |
| 硼氢化产物 | 醇（反马氏）| 醛/酮 |
| 水合产物 | 醇 | 酮/醛（烯醇互变）|
| 还原控制 | 直接到烷烃 | 可停在烯烃（Lindlar/Na/NH₃）|

### 7.2 三大氧化方法对比

| 试剂 | 条件 | 产物 | 立体化学 | 用途 |
|:---|:---|:---|:---:|:---|
| **mCPBA** | 温和 | 环氧化物 | syn | 保护/转化为二醇 |
| **OsO₄ / NMO** | 温和 | 邻二醇 | **syn** | 高选择性双羟化 |
| **O₃ + Zn/H₂O** | 低温 | 醛/酮 | — | **结构推断/碳链切断** |
| **热浓 KMnO₄** | 强热 | 羧酸/CO₂ | — | 彻底氧化 |

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：写出 2-甲基-2-丁烯与下列试剂反应的主产物：
(1) HBr；(2) Br₂/CCl₄；(3) H₂O/H⁺；(4) 1) BH₃, 2) H₂O₂/NaOH

**解答**：
- (1) HBr：马氏加成 → **2-溴-2-甲基丁烷**（三级碳正离子中间体）
- (2) Br₂：反式加成，经溴鎓离子 → **2,3-二溴-2-甲基丁烷**（外消旋混合物）
- (3) H₂O/H⁺：马氏水合 → **2-甲基-2-丁醇**（烯醇不生成，因为直接生成三级碳正）
- (4) 硼氢化-氧化：反马氏 → **3-甲基-1-丁醇**

### 10.2 例题 2（★★ 立体化学）

**题目**：顺-2-丁烯与 Br₂/CCl₄ 反应，产物的立体化学如何？

**解答**：
- 生成溴鎓离子中间体
- Br⁻ 从背面进攻
- 产物为 **(2R,3R)-2,3-二溴丁烷 和 (2S,3S)-2,3-二溴丁烷** 的外消旋混合物
- 即 **meso 的相反情况**（对映体 pair）
- 若反-2-丁烯 → 则得 **meso-2,3-二溴丁烷**

### 10.3 例题 3（★★★ 结构推断）

**题目**：某烯烃 A（C₆H₁₂）臭氧化还原分解后得到丙酮和丙醛。求 A 的结构。

**解答**：
- 臭氧化还原分解：双键断裂，两端分别变成羰基
- 丙酮 (CH₃)₂C=O ← 对应双键一端为 (CH₃)₂C=
- 丙醛 CH₃CH₂CHO ← 对应双键另一端为 =CHCH₂CH₃
- A = **2-甲基-2-戊烯**：(CH₃)₂C=CHCH₂CH₃

## 十一、易错点

1. ❌ **认为所有 HX 都马氏** → HBr/ROOR 是自由基反马氏
2. ❌ **溴水加成写 syn** → 溴鎓离子中间体，是 **anti**
3. ❌ **硼氢化产物记混** → 末端烯烃 → 伯醇；内烯 → 仲醇
4. ❌ **臭氧化工作条件忽略** → 还原工作（Zn/AcOH, Me₂S）得醛/酮；氧化工作（H₂O₂）得酸
5. ❌ **催化氢化立体化学忽略** → syn 加成，双键两侧 H 同侧
6. ❌ **把烯醇当最终产物** → 烯醇会互变为酮/醛（除乙烯醇外都不稳定）
7. ❌ **四取代烯烃说不能反应** → 只是反应慢，强条件下仍可反应
8. ❌ **忽略 α-H 卤代** → NBS/hν 是烯丙位溴代，不是双键加成

### 12.2 学生易踩的认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析口诀 |
|---|---|---|---|
| "双键就是两个单键" | 名字联想 | π 键弱、暴露、易供电子 | "π 键露外面，亲电最喜欢" |
| "溴水加成是 syn" | 直觉同侧加成 | 溴鎓离子背面进攻 → anti | "溴锁门，从背后偷袭" |
| "反马氏就是错误" | 名字带"反" | 硼氢化和自由基都是正确的反马氏 | "硼氢自由基，反马也合理" |
| "臭氧化一定得醛酮" | 只记得还原工作 | 氧化工作得羧酸 | "还原醛酮，氧化变酸" |

### 12.3 入门级例题

**题目**：1-甲基环己烯与稀冷 KMnO₄ 反应，产物是什么？与热浓 KMnO₄ 呢？

**预期解答路径**：
1. 稀冷 KMnO₄：syn 双羟化 → **顺-1-甲基-1,2-环己二醇**
2. 热浓 KMnO₄：彻底氧化断裂 → 双键断裂，得 6-氧代庚酸（或进一步氧化产物）
3. 关键区分：**温度决定命运**

### 12.4 与现实/直觉的连接

- **类比**：烯烃像一个"张开双臂的人"（π 键电子云）——容易被人（亲电试剂）"拥抱"（进攻）。但拥抱的方式不同：有人从正面抱（syn），有人从背后抱（anti）。
- **反例**：直觉认为"双键断裂就是彻底断开"。但实际上，我们可以控制断裂程度：环氧化（只加一个氧）、双羟化（加两个 OH）、臭氧化（彻底切断）。就像切菜：切丁、切片、剁碎，刀法不同结果不同。

## 十三、竞赛拓展

1. **Sharpless 不对称双羟化（AD）**：
   - 用手性配体 + OsO₄ 实现烯烃的**对映选择性**双羟化
   - 2001 年 Sharpless 因此获诺贝尔化学奖

2. **烯烃复分解反应（Olefin Metathesis）**：
   - 两个烯烃交换烷叉片段
   - 2005 年 Nobel 奖（Chauvin, Grubbs, Schrock）
   - 见 [[烯烃复分解反应]]

3. **Wacker 氧化**：
   - Pd/Cu 催化下，烯烃氧化为酮/醛
   - 见 [[Wacker氧化]]

4. **Simmons-Smith 反应**：
   - 烯烃 + CH₂I₂/Zn(Cu) → 环丙烷
   - 见 [[Simmon-Smith反应]]
