# 双羟化反应（Dihydroxylation）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[37-加成反应]]、[[43-亲电加成反应]]

---

## 一、定义

**双羟化反应（Dihydroxylation）**：烯烃转化为**邻二醇（1,2-二醇）**的反应。根据试剂不同，可得到**顺式**或**反式**邻二醇。

**本质**：在双键的两端各加一个羟基（-OH），是烯烃氧化保留双键骨架的典型反应。

## 三、核心原理

### 3.1 顺式双羟化

#### OsO₄ / NMO 体系（Upjohn 双羟化）

**机理**：
1. OsO₄ 与烯烃发生 **[3+2] 环加成** → 生成锇酸酯五元环中间体
2. NMO（N-甲基吗啉-N-氧化物）氧化 Os(VI) → 再生 OsO₄
3. 水解锇酸酯 → **顺式邻二醇**

**特点**：
- **化学计量 OsO₄ 可降至催化量**（NMO 作共氧化剂）
- **顺式加成（syn）**：两个 OH 从双键同侧加入
- **反应温和**，几乎不发生副反应
- **有毒**：OsO₄ 挥发性强，剧毒，需在通风橱中操作

#### 稀冷 KMnO₄

**机理**：
1. KMnO₄ 与烯烃形成 **高锰酸酯环状中间体**（Mn 五元环）
2. 水解 → **顺式邻二醇**

**特点**：
- 反应条件苛刻：必须**稀溶液 + 低温（0°C 左右）**
- 温度稍高或浓度大 → 过度氧化断裂双键（类似臭氧化产物）
- 实验操作简便，但选择性不如 OsO₄
- ** syn 加成**

### 3.2 反式双羟化——Prevost 与 Woodward 反应

这两个反应都利用**碘鎓离子**作为中间体，但通过控制**水的有无**来调控立体化学——这是 ABOC 强调的"平衡思维"经典案例。

#### Prevost 反应（无水条件）

**试剂**：I₂ / AgOAc（或 Ag₂O）/ 无水条件

**机理**：
1. 烯烃 + I₂ → **碘鎓离子**
2. AcO⁻ 进攻碘鎓离子（背面进攻，反式开环）→ 碘代醋酸酯
3. 第二个 AcO⁻ 取代 I（SN2，再次背面进攻）→ **反式二醋酸酯**
4. 水解 → **反式邻二醇**

**总结果**：**反式加成**（两个 OH 最终位于双键两侧）

#### Woodward 反应（有水条件）

**试剂**：I₂ / AgOAc / **H₂O**

**机理**：
1. 烯烃 + I₂ → **碘鎓离子**
2. H₂O（或 AcO⁻）进攻碘鎓离子 → 碘代醇/碘代醋酸酯
3. **关键差异**：水使中间体**不可逆**
   - 无水时：碘鎓离子开环可逆，最终通过两次背面进攻得反式
   - 有水时：水捕获碳正离子特征中间体，使第一次开环后无法回到碘鎓离子
   - 结果：**顺式加成**（水和 I 从同侧加入的净结果）

**总结果**：**顺式加成**（两个 OH 最终位于双键同侧）

#### Prevost vs Woodward 对比

| | **Prevost（无水）** | **Woodward（有水）** |
|:---|:---|:---|
| **试剂** | I₂ / AgOAc，严格无水 | I₂ / AgOAc / H₂O |
| **中间体** | 碘鎓离子（可逆开环）| 碘鎓离子（被水捕获）|
| **立体化学** | **反式** | **顺式** |
| **本质** | 两次背面进攻 | 水使中间体不可逆，改变选择性 |

> ⭐ **ABOC 平衡思维**："用平衡思维解释：水使中间体不可逆，微量即可改变选择性。"

### 3.3 邻二醇切断（结构推断工具）

**HIO₄ / Pb(OAc)₄ 切断**：

```
R-CH(OH)-CH(OH)-R'  +  HIO₄  →  R-CHO  +  R'-CHO  +  HIO₃
```

**机理**：
1. 高碘酸与邻二醇形成**五元环酯中间体**
2. 电子转移 → C-C 键断裂
3. 生成两分子羰基化合物

**特点**：
- **顺式邻二醇切断更快**（五元环酯更易形成）
- 反式邻二醇也能反应，但速率慢得多
- 是**结构推断的经典工具**：从切断产物反推邻二醇结构

---

## 四、关键结论

1. **顺式双羟化**：OsO₄/NMO 或 稀冷 KMnO₄ → syn 加成
2. **反式双羟化**：Prevost（无水，I₂/AgOAc）→ anti 加成
3. **顺式双羟化（另一路线）**：Woodward（有水，I₂/AgOAc/H₂O）→ syn 加成
4. **Prevost vs Woodward 差异的本质**：**水使中间体不可逆**——微量条件变化即可翻转立体化学
5. **邻二醇切断**：HIO₄ 专切顺式邻二醇（五元环酯机理），是结构推断利器
6. **三种邻二醇制备路线互补**：
   - 要 **syn** → OsO₄ 或 Woodward
   - 要 **anti** → Prevost
   - 要 **最便宜** → 稀冷 KMnO₄（但需控温）

## 五、常见分类或情形

### 5.1 按产物立体化学分类

| 方法 | 试剂 | 立体化学 | 备注 |
|:---|:---|:---:|:---|
| **OsO₄ / NMO** | 催化 OsO₄，NMO 共氧化 | **syn** | 最常用，温和 |
| **稀冷 KMnO₄** | 0°C，稀溶液 | **syn** | 条件苛刻，易过度氧化 |
| **Prevost** | I₂ / AgOAc，无水 | **anti** | 反式二醇 |
| **Woodward** | I₂ / AgOAc / H₂O | **syn** | 顺式二醇 |
| **环氧开环** | mCPBA → 酸/碱水解 | **trans**（开环反式）| 间接路线 |

### 5.2 按应用目的分类

| 目的 | 推荐方法 | 原因 |
|:---|:---|:---|
| **合成 syn-二醇** | OsO₄/NMO | 条件温和，产率高 |
| **合成 anti-二醇** | Prevost | 经典方法 |
| **结构推断（切断）** | HIO₄ | 专切邻二醇，定量 |
| **低成本教学实验** | 稀冷 KMnO₄ | 试剂便宜 |

## 六、适用条件与限制

1. **OsO₄ 剧毒且贵**——实验室常用催化量 + NMO，工业上少用
2. **KMnO₄ 必须稀冷**——否则双键断裂（得到醛/酮/酸混合物）
3. **Prevost/Woodward 需严格控水**——水的有无直接决定立体化学
4. **邻二醇切断要求两个 OH 相邻**——1,3-二醇不被 HIO₄ 切断
5. **环状邻二醇的切断可打开环**——常用于环状结构推断
6. **Sharpless 不对称双羟化（AD-mix）**——烯烃的对映选择性 syn 双羟化，高阶内容

## 七、常见比较与易混点

### 7.1 四种邻二醇制备方法对比

| | **OsO₄/NMO** | **Prevost** | **Woodward** | **环氧开环** |
|:---|:---|:---|:---|:---|
| **中间体** | 锇酸酯 | 碘鎓离子 | 碘鎓离子 | 环氧 |
| **立体化学** | syn | anti | syn | trans（开环）|
| **条件敏感** | 低 | 高（无水要求）| 中（需水）| 低 |
| **成本** | 高 | 中 | 中 | 低 |

### 7.2 Prevost vs Woodward——为什么水能改变立体化学？

**核心理解**：
- 碘鎓离子开环是**可逆的**
- 无水时：AcO⁻ 进攻 → 若产物不利，可回到碘鎓离子 → 最终通过两次背面进攻得 **anti**
- 有水时：H₂O 进攻 → 形成 C-OH 键（不可逆）→ 后续步骤被迫围绕这个固定构型进行 → 最终得 **syn**

这就是 ABOC 强调的**"平衡思维"**——不是死记"无水反式、有水顺式"，而是理解"水如何改变中间体的可逆性"。

### 7.3 邻二醇切断 vs 臭氧化

| | **HIO₄ 切断** | **臭氧化** |
|:---|:---|:---|
| **底物** | 邻二醇 | 烯烃 |
| **断裂位置** | C-C（连两个 OH）| C=C |
| **产物** | 两分子醛/酮 | 两分子醛/酮/酸 |
| **应用** | 结构推断（从产物推邻二醇）| 结构推断（从产物推烯烃）|

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：环己烯分别用 OsO₄/NMO 和 Prevost 条件处理，产物的立体化学有何不同？

**解答**：
- **OsO₄/NMO**：syn 双羟化 → **顺-1,2-环己二醇**（两个 OH 都在环的同侧）
- **Prevost（无水 I₂/AgOAc）**：anti 双羟化 → **反-1,2-环己二醇**（两个 OH 在环的异侧）

### 10.2 例题 2（★★ Prevost vs Woodward）

**题目**：解释为什么 Prevost（无水）得反式二醇，而 Woodward（有水）得顺式二醇。

**解答**：
1. 两者都经历碘鎓离子中间体
2. 无水时：AcO⁻ 背面进攻碘鎓离子 → 产物可逆 → 系统通过两次背面进攻达到最稳定状态 → **anti**
3. 有水时：H₂O 进攻碘鎓离子 → C-OH 键形成后**不可逆** → 后续步骤围绕已固定的构型进行 → **syn**
4. **本质**：水的存在改变了中间体的可逆性，从而翻转了立体化学结果

### 10.3 例题 3（★★ 结构推断）

**题目**：某化合物 A 经 HIO₄ 切断后得到甲醛和环戊酮。A 的可能结构是什么？

**解答**：
- HIO₄ 切断邻二醇：R-CH(OH)-CH(OH)-R' → R-CHO + R'-CHO
- 产物为 HCHO + 环戊酮
- 反推：A 是 **1-(羟甲基)-1-羟基环戊烷** 的某种立体异构体
- 更准确地：A 为 **1,2-环戊二醇的某种衍生物**... 需要具体结构匹配
- 实际上：环戊酮来自环状部分，甲醛来自 CH₂OH → A 含 -CH(OH)-CH₂OH 与环戊烷相连
- A = **反-1-羟甲基-1-环戊醇**（或类似结构，需确保两个 OH 相邻且能形成五元环酯）

### 10.4 例题 4（★★★ 综合选择）

**题目**：如何从反-2-丁烯出发制备 meso-2,3-丁二醇？从顺-2-丁烯出发呢？

**解答**：
- **meso-2,3-丁二醇** = 反式加成产物（两个 OH 在异侧）
- 反-2-丁烯 → 要得 meso，需要 **syn 加成** → 用 **OsO₄/NMO** 或 **Woodward**
- 顺-2-丁烯 → 要得 meso，需要 **anti 加成** → 用 **Prevost（无水）**

## 十一、易错点

1. ❌ **混淆 Prevost 和 Woodward 的条件** → 无水 = Prevost = anti；有水 = Woodward = syn
2. ❌ **认为 KMnO₄ 总是氧化断裂双键** → 稀冷条件下是**syn 双羟化**
3. ❌ **忽略 OsO₄ 的毒性** → 竞赛实验设计题可能涉及安全考量
4. ❌ **邻二醇切断只记产物，不记立体要求** → 顺式邻二醇切断更快（五元环酯）
5. ❌ **把 Woodward 当反式** → Woodward 是 **syn**（有水使中间体不可逆）
6. ❌ **混淆环氧开环与双羟化的立体化学** → 环氧开环得 trans-二醇；OsO₄ 得 syn-二醇
7. ❌ **忘记 Sharpless 双羟化** → 高阶竞赛可能考不对称双羟化的对映选择性
8. ❌ **HIO₄ 切断非邻二醇** → HIO₄ 只切 1,2-二醇；1,3-二醇不反应

### 12.3 入门级例题

**题目**：写出环己烯用 OsO₄/NMO 和 Prevost 试剂分别处理后的产物，并画出立体化学。

**预期解答路径**：
1. OsO₄/NMO：syn 双羟化 → **顺-1,2-环己二醇**（两个 OH 均朝上或均朝下）
2. Prevost：anti 双羟化 → **反-1,2-环己二醇**（一个 OH 朝上，一个朝下）
3. 提示：用楔形式画， syn = 同侧，anti = 异侧

### 12.4 与现实/直觉的连接

- **类比**：双羟化就像给双键"戴上耳机"——syn 加成是两个耳机罩戴在同一边（都朝前或都朝后），anti 加成是戴在两边（一前一后）。Prevost 和 Woodward 的区别在于"戴耳机的规则"：无水时要求严格对称（anti），有水时可以同侧（syn）。
- **反例**：直觉认为"加水会稀释反应，降低产率"。但 Woodward 反应中，**微量水反而改变了立体化学**——这是化学中"微小条件变化导致巨大结果差异"的典型案例。

## 十三、竞赛拓展

1. **Sharpless 不对称双羟化（AD-mix）**：
   - OsO₄ / 铁氰化钾 / 手性配体（DHQD 或 DHQ）
   - 从双键的一面选择性双羟化
   - 与 Sharpless 环氧化、Sharpless 不对称氨羟化并称"Sharpless 三反应"

2. **Lemieux-Johnson 氧化**：
   - OsO₄ / NaIO₄ 组合
   - 先双羟化，后原位切断 → 直接从烯烃得醛/酮
   - 是"双羟化 + 切断"的一锅法

3. **不对称邻二醇的合成策略**：
   - 竞赛高阶题可能要求设计特定构型的邻二醇合成路线
   - 需要灵活组合：环氧化 → 开环、OsO₄、Prevost、Woodward
