# Aldol 缩合（Aldol Condensation）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[45-亲核加成反应]]、[[46-羰基α位反应]]

---

## 一、定义

**Aldol 缩合**：含 **α-H 的羰基化合物**在**酸或碱**催化下形成**烯醇/烯醇盐**，进攻另一分子**羰基**生成 **β-羟基羰基化合物**（aldol）的反应；若进一步脱水则得 **α,β-不饱和羰基化合物**。

**通式**：
$$
\ce{R-CH2-CHO + R'-CHO ->[base] R-CH(OH)-CH(R')-CHO}
$$

**本质**：**烯醇盐作为亲核试剂**对羰基的亲核加成——是羰基 α 位反应的基石。

## 三、核心原理

### 3.1 反应机理

**碱催化**：
```
步骤 1：碱夺取 α-H → 烯醇盐（enolate）
步骤 2：烯醇盐进攻另一分子羰基碳 → 烷氧负离子
步骤 3：质子化 → β-羟基羰基（aldol）
步骤 4（可选）：加热/脱水 → α,β-不饱和羰基
```

**酸催化**：
```
步骤 1：羰基质子化 → 亲电性增强
步骤 2：烯醇（enol）进攻另一质子化羰基
步骤 3：质子转移 → aldol
```

### 3.2 烯醇盐的 E/Z 构型与产物立体化学

**Zimmerman-Traxler 过渡态**（六元环椅式）：

| 烯醇盐构型 | 产物主要立体化学 |
|:---:|:---:|
| **E-烯醇盐** | anti（反式）|
| **Z-烯醇盐** | syn（顺式）|

**控制烯醇盐 E/Z 的方法**：
- 动力学条件（强碱、低温、短反应时间）→ 取代少的烯醇盐
- 热力学条件（弱碱、室温）→ 取代多的烯醇盐（更稳定）

### 3.3 交叉 Aldol 的选择性策略

**交叉 Aldol 的问题**：两种羰基化合物都含 α-H → 四种烯醇盐 + 两种羰基 → 产物复杂。

**解决策略**：

| 策略 | 具体操作 | 适用场景 |
|:---|:---|:---|
| **LDA 动力学控制** | LDA, -78°C → 只生成取代少的烯醇盐 | 醛 + 酮 |
| **烯醇硅醚法** | 预制成烯醇硅醚 + Lewis 酸活化羰基 | 需要立体控制 |
| **无 α-H 的羰基** | 苯甲醛、甲醛等作为受体 | 经典交叉 Aldol |
| **定向 Aldol** | Evans 手性辅剂、Corey 法等 | 不对称合成 |

### 3.4 分子内 Aldol

**二羰基化合物**的分子内环化：
- **1,5-二羰基** → 五元环
- **1,6-二羰基** → 六元环（最稳定，优先）
- 是 **Robinson 环化** 的第二步

### 3.5 脱水反应

**β-羟基羰基 → α,β-不饱和羰基**：
- 酸/碱催化，加热
- 驱动力：生成共轭双键（更稳定）
- 立体化学：通常生成 **E 型**（热力学稳定）

---

## 四、关键结论

1. **Aldol = 烯醇盐 + 羰基 → β-羟基羰基**
2. **E-烯醇盐 → anti，Z-烯醇盐 → syn**（Zimmerman-Traxler 过渡态）
3. **交叉 Aldol 需要选择性策略**——LDA、烯醇硅醚、或无 α-H 羰基
4. **分子内 Aldol 优先成五/六元环**
5. **脱水生成 E-型 α,β-不饱和羰基**（共轭稳定）
6. **Aldol 是 Robinson 环化、Mannich、Knoevenagel 的母反应**
7. **交叉 Aldol 先看两件事**：受体是否无 α-H，供体是否更易形成烯醇盐

## 五、常见分类或情形

### 5.1 按催化方式分类

| 类型 | 催化剂 | 特点 |
|:---|:---|:---|
| **碱催化 Aldol** | NaOH, LDA, K₂CO₃ | 生成烯醇盐，最常用 |
| **酸催化 Aldol** | HCl, H₂SO₄, TsOH | 生成烯醇，可逆性更强 |
| **硅醚法 Aldol** | 烯醇硅醚 + Lewis 酸 | Mukaiyama 变体，条件温和 |

### 5.2 按反应模式分类

| 模式 | 底物 | 产物 | 难度 |
|:---|:---|:---|:---:|
| **自身 Aldol** | 同种醛/酮 | 单一 β-羟基羰基 | ★★ |
| **交叉 Aldol** | 两种醛/酮 | 混合物（需策略控制）| ★★★ |
| **分子内 Aldol** | 1,5-或 1,6-二羰基 | 环状产物 | ★★★ |
| **脱水 Aldol** | 加热 aldol 产物 | α,β-不饱和羰基 | ★★ |

## 六、适用条件与限制

1. **底物必须有 α-H**——无 α-H 的羰基（如苯甲醛）只能做受体
2. **碱不能过强**——强碱可能导致副反应（Cannizzaro、Claisen 缩合）
3. **温度控制**——低温有利于动力学烯醇盐；高温有利于热力学产物
4. **交叉 Aldol 产物复杂**——需预制备烯醇盐或使用无 α-H 受体
5. **醛比酮反应性高**——醛的羰基更亲电，更容易被烯醇盐进攻

### 6.1 第三轮最该强调的交叉缩合成功条件

第三轮备课里，交叉 Aldol 不适合只讲“可能混合物很多”，更有用的是给出一眼判断口径：

- **受体最好无 α-H**：如苯甲醛、甲醛，这样它只做亲电体，不会自己生成烯醇盐
- **供体最好更容易去质子化**：如 β-二羰基或酸性更强的羰基底物，更容易定向生成单一烯醇盐

因此遇到交叉 Aldol，先判断“**谁只能做受体，谁更适合做供体**”，再谈条件控制。

## 七、常见比较与易混点

### 7.1 Aldol vs Mannich

- Aldol：烯醇盐进攻**羰基** → β-羟基羰基
- Mannich：烯醇盐进攻**亚胺正离子** → β-氨基羰基
- 两者都用烯醇盐作为亲核体

### 7.2 Aldol vs Knoevenagel

- Aldol：单吸电子基羰基 → β-羟基羰基
- Knoevenagel：**双吸电子基**活泼亚甲基 → 直接脱水得 α,β-不饱和产物
- Knoevenagel 更容易脱水（两个吸电子基稳定双键）

### 7.3 Aldol vs Claisen 缩合

- Aldol：醛/酮之间的缩合
- Claisen 缩合：**酯**之间的缩合 → β-酮酯
- 两者都是烯醇盐对羰基的亲核加成

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：乙醛在稀 NaOH 中反应，产物是什么？

**解答**：
- 乙醛形成烯醇盐 → 进攻另一分子乙醛
- 产物：**3-羟基丁醛**（aldol）
- 加热脱水 → **巴豆醛**（crotonaldehyde，E-2-丁烯醛）

### 10.2 例题 2（★★ 交叉 Aldol）

**题目**：苯甲醛 + 丙酮在碱中反应的主要产物？

**解答**：
- 苯甲醛无 α-H → 只能做受体
- 丙酮形成烯醇盐 → 进攻苯甲醛
- 脱水后 → **亚苄基丙酮**（benzalacetone，E-型）

### 10.3 例题 3（★★★ 立体化学）

**题目**：E-烯醇盐与醛在 Zimmerman-Traxler 过渡态中主要产物构型是什么？

**解答**：
- E-烯醇盐 → 六元环椅式过渡态中取代基倾向于假平伏键
- 产物：**anti**（两个新形成的手性中心反式）

## 十一、易错点

1. ❌ **Aldol 产物只有 β-羟基羰基** → 加热可**脱水**得 α,β-不饱和羰基
2. ❌ **交叉 Aldol 产物单一** → 无选择性策略时产物是**混合物**
3. ❌ **Z-烯醇盐得 anti** → Z-烯醇盐得 **syn**，E-烯醇盐得 **anti**
4. ❌ **分子内 Aldol 优先成四元环** → 优先成**五/六元环**
5. ❌ **脱水产物是 Z 型** → 脱水产物通常是 **E 型**（热力学稳定）

| "交叉 Aldol 很简单" | 无策略时是混合物 | "交叉要控制" |

### 12.3 入门例题

**题目**：为什么 Aldol 反应中，醛比酮更容易反应？

**引导**：
- 醛的羰基碳亲电性更强（烷基给电子少）
- 酮有两个烷基，空间位阻大
- 所以烯醇盐优先进攻醛

### 12.4 现实类比

**Aldol 缩合像"搭积木"**：
- 烯醇盐 = 带磁性的一端（亲核端）
- 羰基 = 带磁性的另一端（亲电端）
- 两者靠近 → 自然吸合（C-C 键形成）
- 脱水 = 把两块积木压紧，去掉中间的空气（水分子）
