# SN2 反应（Substitution Nucleophilic Bimolecular）

- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[32-有机反应机理]]、[[36-取代反应]]、[[44-脂肪族亲核取代反应]]

## 一、定义

**SN2（Substitution Nucleophilic Bimolecular）**：双分子亲核取代反应。**一步协同**进行——亲核试剂从 C-LG 反键轨道方向背面进攻，**同时**离去基团离去；过渡态为五配位三角双锥，**两种分子都参与决速步**。

$$\text{rate} = k[\mathrm{RX}][\mathrm{Nu}^-] \quad\text{（二级反应）}$$

> ⭐ **ABOC 视角（行 1894）**：从结构上看，卤代烷转变为过渡态时，"中心碳原子将由原来 sp³ 的四面体结构转为 sp² 的三角形的平面结构，碳上还有一个垂直于该平面的 p 轨道，该 p 轨道的一侧与亲核试剂(Nu)的轨道重叠，另一侧与离去基团(X)的轨道重叠。"
>
> 这是 SN2 的几何本质：**亲核试剂进攻 C-LG 反键轨道（σ\*），同时这个 σ\* 与离去基的成键轨道直线对齐**——这正是 [[反式共平面]] 在取代反应中的体现。

## 三、核心原理

### 3.1 协同机理与五配位过渡态

```
        Nu⁻               Nu                 Nu
         │                 ︙                 │
         ↓     →    ──C─（五配位平面）   →    ──C──
         C─X              ︙                  
                           X                  X⁻
        sp³            sp² + p 轨道           sp³
       四面体          三角双锥（TS）        四面体（构型翻转）
```

| 步骤 | 描述 |
|:---|:---|
| 1. 接近 | 亲核试剂沿 C-LG 反键轨道方向背面接近 |
| 2. 过渡态 | C 中心 sp³ → sp² 平面（含一根垂直 p 轨道）；Nu 与 LG 在 p 轨道两侧分别重叠（C 类似 sp²+p）|
| 3. 翻转 | C 三个键由原伞形向反方向偏转（"伞被风吹翻"）|
| 4. 产物 | 构型 100% 翻转（[[Walden翻转]]）|

> **能量学**：过渡态能量比起始物高 ~15-30 kcal/mol；反应可由放热（HBr → H+Br⁻ 解离驱动）或吸热（碱催化驱动）整体推动。

### 3.2 [[反式共平面]] 几何要求

> ⭐ ABOC §1.5 + §2.9 联动：SN2 是 [[给体-受体相互作用]] 框架下"反式共平面"几何要求的代表反应。

- 亲核试剂的 σ 给体（lp 或 σ键）必须**沿着 C-LG 反键轨道（σ\*）的最大波瓣方向**接近
- 这要求 Nu...C...LG 三者**共线**（180° 二面角）
- 这正是 [[反式共平面]] 的特殊情形——给体与受体 σ\* 在直线上对齐
- 与 [[E2反应]]（β-H 与 LG 反式共平面）几何同源

### 3.3 反极化与亲卤取代（ABOC 独特视角，行 1906）

> ⭐ **ABOC 关键创新**：SN2 不仅可以进攻 C-LG 反键轨道（标准方式），**也可以进攻 LG-C 反键的另一侧**（即 LG 那一头）：

```
标准 SN2：       Nu⁻ → C ← LG⁻       （进攻 C 一侧）

反极化 SN2：     Nu⁻ → LG → C⁻ → Nu  （进攻 LG 一侧 + 反向回攻）
                          ↓
                     得到稳定碳负 → 再进攻
```

- **当对应的碳负离子很稳定时**（如 ArCH⁻、α-杂原子稳定的负离子），亲核试剂可"先进攻离去基团"
- 得到碳负离子 + 离去-亲核杂化体（如 NuLG）
- 然后碳负离子从相反方向进攻 → **构型保持产物**
- 在**硫亲核试剂**条件下十分常见

| 项 | 标准 SN2 | 反极化 SN2 |
|:---|:---|:---|
| 进攻位点 | C 端（C-LG σ\*）| LG 端（LG-C σ\*）|
| 中间体 | 无（一步协同）| 短寿命碳负离子 |
| 立体化学 | **翻转** | **保持** |
| 典型亲核 | OH⁻、CN⁻ | RS⁻、RSe⁻、I⁻ 等软亲核 |

> 这是 ABOC §2.9 把"反极化与亲卤取代"作为主线介绍而非奇技淫巧的原因——它直接把 Ch.1 §1.4 给体-受体相互作用拉到取代反应里复用。

### 3.4 邻基参与（Neighboring Group Participation, NGP）

详见 [[邻基参与]]。SN2 体系常见 NGP：
- 含孤对的邻位基团（-COO⁻、-SR、-Br、-NHR 等）先分子内 SN2 进攻
- 形成三/五元环鎓离子中间体
- 第二步：外来 Nu 进攻鎓离子开环（第二次 SN2）
- 表观立体化学：**两次翻转 = 保持**
- 表观速率：常远超不含 NGP 体系（10⁵-10⁸ 倍加速）
- 经典例子：芥子气在碱中"看似 SN1 但产物保持"

### 3.5 分子内 SN2 优先原则（ABOC 行 1880）

> ⭐ ABOC 强调："分子内 SN2 和 SN1 反应更为有利，速率更快，**需要优先考虑**。"

- 熵效应：分子内反应少了"碰撞概率"环节
- 几何易实现：3-7 元环过渡态最佳
- 高频实战：邻基参与（NGP）、关环反应、阳离子环化

### 3.6 反应势能图特征

$$\text{SN2 势能图} = \begin{cases} \text{峰：五配位过渡态（一峰）} \\ \text{谷：起始物 + 产物（两侧）} \end{cases}$$

**一峰，无中间体** vs **SN1 两峰一谷**——这是 ABOC §2.9 章末习题 T7 的核心考点。

## 四、关键结论

### 4.1 有利于 SN2 的因素

| 因素 | 有利条件 | 物理本质 |
|:---|:---|:---|
| 底物结构 | 甲基 > 1° > 2° ≫ 3° | 位阻越小，五配位过渡态越易形成 |
| 离去基团 | 好的离去基（弱碱）：I⁻ > Br⁻ > Cl⁻ > F⁻；OTf > OTs > OMs | C-LG σ\* 越低，越易接受 σ 给体 |
| 亲核试剂 | **强亲核试剂**：HS⁻ > I⁻ > CN⁻ > N₃⁻ > RO⁻ > OH⁻ | 必须直接参与 RDS |
| 溶剂 | **极性非质子**（DMSO、DMF、丙酮、CH₃CN、HMPA）| 阳离子溶剂化 + 阴离子"裸露" |
| 几何 | 反式共平面接近（无空间阻挡）| 满足 σ→σ\* 几何重叠 |

### 4.2 SN2 vs SN1 关键区分

详见 [[SN1反应]] §4.2 全表对比。简表：

| 项 | SN2 | SN1 |
|:---|:---:|:---:|
| 速率方程 | rate = k[RX][Nu⁻] | rate = k[RX] |
| 中间体 | ❌ 无（协同）| ✅ 碳正离子 |
| 立体化学 | **完全翻转** | 外消旋偏翻转 |
| 底物偏好 | 甲基 > 1° > 2° | 3° > 2° |
| 亲核试剂要求 | 必须强 | 弱亦可 |
| 重排 | ❌ 无 | ✅ 可能 |
| 溶剂偏好 | 极性非质子 | 极性质子 |

### 4.3 位阻效应的"奇特规律"

| 底物类型 | SN2 速率比（vs CH₃Br = 1）| 备注 |
|:---|:---:|:---|
| CH₃Br | 1 | 基准 |
| CH₃CH₂Br（1°）| 0.04 | 一个 α-CH₃ 显著降速 |
| (CH₃)₂CHBr（2°）| 0.005 | 两个 α-CH₃ 进一步降速 |
| (CH₃)₃CBr（3°）| ~0 | 几乎完全阻断 |
| 新戊基 (CH₃)₃CCH₂Br | ~10⁻⁵ | β-叔丁基的"远程位阻" |
| PhCH₂Br（苄基）| 120 | π 体系在 TS 中稳定（共振）|
| CH₂=CHCH₂Br（烯丙基）| 60 | π 体系类似稳定 |

> **教学要点**：位阻效应不是只看 α-取代——β 位置（新戊基）的"远程位阻"也能使 SN2 速率降低 10⁵ 倍。

### 4.4 [[硬软亲核试剂]] 与 SN2 选择性

| 亲核试剂类型 | 进攻位点 | SN2 表现 |
|:---|:---|:---|
| 硬亲核（OH⁻、RO⁻、F⁻、NH₃）| 偏好 sp³ C-LG 直接进攻 | 标准 SN2 |
| 软亲核（HS⁻、RS⁻、I⁻、CN⁻）| 部分进攻 LG 端 | 反极化 SN2 + NGP 易发生 |
| 中等（Br⁻、Cl⁻、N₃⁻）| 标准进攻 | 标准 SN2 |

## 五、常见分类或情形

### 5.1 按底物类型

| 底物 | SN2 反应性 | 备注 |
|:---|:---:|:---|
| CH₃-X | ★★★★★ | 最优底物，无位阻 |
| RCH₂-X（1°）| ★★★★ | 良好底物 |
| R₂CH-X（2°）| ★★ | 与 E2 / SN1 / E1 多重竞争 |
| R₃C-X（3°）| ✗ | 几乎不反应（位阻），走 SN1 / E1 |
| PhCH₂-X（苄基）| ★★★★★ | π 共振稳定 TS，极快 |
| CH₂=CHCH₂-X（烯丙基）| ★★★★ | π 参与稳定 |
| 桥头碳 | ✗ | 几何无法五配位翻转 |
| Ph-X / 乙烯基-X | ✗ | sp² C，背面被 π 屏蔽（详见 [[SNAr]]）|

### 5.2 按亲核试剂

| 亲核试剂 | 强度 | 说明 |
|:---|:---:|:---|
| I⁻ / HS⁻ / RS⁻ | 强（可极化）| 软亲核，SN2 优秀；常见反极化 |
| CN⁻ / N₃⁻ / RCOO⁻ | 中强 | 常见合成试剂 |
| Br⁻ / Cl⁻ | 中等 | 卤素交换反应（[[Finkelstein反应]]）|
| H₂O / ROH | 弱 | 需要好底物 |
| RO⁻ / HO⁻ | 强（碱性）| 强亲核但伴随 E2 竞争 |
| t-BuO⁻ / 大位阻碱 | 弱（位阻）/ 强碱 | 几乎只走 E2 |

### 5.3 按溶剂效应

| 溶剂 | 对 SN2 的影响 |
|:---|:---|
| 极性非质子（DMSO、DMF、丙酮、HMPA、MeCN）| ✅ 加速：阳离子溶剂化，阴离子"裸露" |
| 极性质子（H₂O、ROH）| ❌ 减速：氢键包裹亲核试剂，降低反应性 |
| 非极性（己烷、苯）| ❌ 离子型亲核试剂不溶 |

> **ABOC 法则（行 1914）**：底物 → 决速步过渡态极性变化决定溶剂依赖性。
>
> SN2 中性 + 阴离子 → 阴离子分散过渡态（极性减小）→ **极性溶剂减慢**（与中性亲核试剂方向相反，需具体分析）。

## 六、适用条件与限制

### ✅ SN2 必备条件
- 底物位阻足够小（甲基、1°、部分 2°）
- 强亲核试剂参与
- 极性非质子溶剂最佳
- 反式共平面几何能实现（无空间阻挡）

### ⚠️ SN2 的限制
- 2° 底物：常伴随 E2 竞争（强碱亲核试剂时更显著）
- 3° 底物：SN2 不可行——走 SN1 或 E1/E2
- Ph-X、CH=CHX：背面被 π 电子云屏蔽，SN2 极慢；需走 [[SNAr]]
- 桥头碳：sp³ → sp² 平面化不可行（[[Bredt规则]]）
- 高温：可能切到 SN1 / E1（碳正离子也能形成）

### 🚫 SN2 绝不可能的情形
- 桥头碳（环状几何阻止 sp²+p 平面化）
- 芳基 / 乙烯基底物（背面无空间）
- 极强位阻底物（如新戊基的"远程位阻"）

## 七、常见比较与易混点

### 1. SN2 vs SN1（详见 §4.2）

### 2. SN2 vs E2

> 同一底物 + 同一强碱亲核试剂的两种命运。

| 项 | SN2 | E2 |
|:---|:---|:---|
| Nu/B 进攻 | C-LG（取代）| β-H（消除）|
| 几何要求 | 反式共平面（C-LG-Nu 直线）| 反式共平面（β-H 与 LG）|
| 影响因素 | 位阻 ↑ → SN2 ↓ | 位阻不影响（β-H 在外侧）|
| 温度 | 不敏感 | ↑ 利于 E2 |
| 碱性 | 中等亲核 | 强碱（t-BuO⁻、LDA）|

> 经典对照：**EtO⁻ + 2-BrCH(CH₃)₂**（2° 底物）：约 80% E2 + 20% SN2。

### 3. SN2 vs SN1 边界（ABOC 视角）

> ABOC 行 1865："R⁺ 与 X⁻ 结合比较紧密，亲核试剂从相反一面进攻居多，**其实就是把 SN1 当成一定性质的 SN2 看待**。"

- 紧密离子对的 SN1 ≈ "迟到的" SN2
- 反极化 SN2 ≈ "短寿命碳负离子" 的 SN2 + 回攻
- 邻基参与 SN2（芥子气）≈ 表观 SN1 速率 + 两次 SN2 实质

> **教学要点**：SN1/SN2 不是绝对二元，而是同一谱系上的两个端点。

### 4. SN2 vs SN2'

> 烯丙基体系的特例：

| 项 | SN2 | SN2' |
|:---|:---|:---|
| 进攻位点 | 含 LG 的 C | 远端烯丙位 C（γ 位）|
| 双键位置 | 不变 | 移至原 C-LG 处 |
| 立体化学 | C 上翻转 | γ 位翻转（更复杂）|
| 应用 | 常规取代 | 烯丙基取代位置切换 |

### 5. SN2 与反极化（亲卤取代）

> 详见 §3.3。关键标志：
> - 软亲核试剂（RS⁻、RSe⁻、I⁻）
> - 产物**构型保持**（不是翻转）
> - 中间体可分离（在某些体系中）

### 6. 易混点总览

| 易混 | 区分 |
|:---|:---|
| SN2 vs E2 | 进攻 C 是 SN2；进攻 β-H 是 E2。位阻 ↑ + 强碱 → E2 比例 ↑ |
| 翻转 vs 外消旋 | SN2 = 100% 翻转（除非有 NGP 或反极化）；SN1 = 部分外消旋 |
| 极性非质子 vs 极性质子 | 前者裸露阴离子 → SN2 ↑；后者氢键包裹阴离子 → SN2 ↓ |
| 标准 SN2 vs 反极化 SN2 | 标准 = 翻转；反极化 = 保持。前者进攻 C；后者进攻 LG |
| SN2 vs NGP-SN2 | 表面看 SN2 + 翻转；NGP 实际是分子内 SN2 + 外加第二次 SN2 = 保持 |

## 十、例题

### 例题 1：底物反应性排序

**题目**：将下列卤代烷按 SN2 反应速率排序：
(A) CH₃CH₂Br  (B) (CH₃)₃CBr  (C) CH₃Br  (D) (CH₃)₂CHBr

**分析**：SN2 速率取决于底物位阻。位阻越小越快。

**解答**：C（甲基）> A（1°）> D（2°）> B（3°，几乎不反应）

**反思**：排序题必须先判断机理类型。如果是 SN1，则排序反向：B > D > A > C。

### 例题 2：立体化学预测

**题目**：(R)-2-溴丁烷在 NaOH/DMSO 中发生 SN2 反应。写出产物的立体构型。

**分析**：SN2 → Walden 翻转。DMSO 是极性非质子溶剂，促进 SN2。NaOH 是强碱强亲核，但 2° 底物 + 极性非质子 → SN2 主导。

**解答**：产物为 (S)-2-丁醇（构型完全翻转）。

**反思**：SN2 的立体化学和底物构型翻转是竞赛高频考点。注意书写时用楔形/虚线键表达构型。

### 例题 3：竞争判断

**题目**：预测 (CH₃)₃CO⁻（叔丁醇钾）与 2-溴丙烷在叔丁醇中的主要反应路径。

**分析**：t-BuO⁻ 是**强碱、位阻大的弱亲核试剂**。2° 底物 + 强碱 → 倾向于 E2 消除而非 SN2。

**解答**：主产物为丙烯（E2 消除），SN2 产物 2-叔丁氧基丙烷为次要产物。

**反思**：SN2 vs E2 竞争是竞赛经典题型。关键看：亲核试剂碱性强弱 + 位阻 + 温度（高温更有利于消除）。

### 例题 4：反极化与亲卤取代（ABOC §2.9 视角）

**题目**：PhSH/NaOH 与 (R)-2-溴丁烷反应，产物的立体化学是什么？

**分析**：
- 亲核试剂：PhS⁻（软亲核试剂、可极化、α 位有共振稳定）
- 标准 SN2：进攻 C-Br σ\*  → 翻转产物 (S)-PhSCH(CH₃)CH₂CH₃
- 反极化 SN2：PhS⁻ 进攻 Br 端 → 形成 PhS-Br + (R)-CH₃CHCH₂CH₃⁻ 短寿命碳负离子 → 碳负从相反方向回攻 → **构型保持** (R)-PhSCH(CH₃)CH₂CH₃

**解答**：
- 在含 PhSH/NaOH 的体系中，**反极化 SN2 占主导**：实测产物为 80% (R) + 20% (S)，即**保持产物为主**
- 标志：硫亲核 + 较好的稳定碳负（2° 仲碳）→ 反极化路径

**反思**：ABOC §2.9 行 1906 强调反极化是 SN2 的"另一面"——硫亲核试剂常见构型保持产物（不是翻转）。这是 35-39 届初赛/决赛常考的"非典型 SN2"陷阱。

### 例题 5：分子内 SN2 优先（ABOC 行 1880）

**题目**：底物 4-氯-1-戊醇 在 NaH/THF 中加热。预测主产物。

**分析**：
- 醇 + NaH → 醇盐（强亲核 / 强碱）
- 分子内 SN2（5-exo-tet 关环，符合 [[Baldwin规则]]）：醇盐进攻 4 位 C，Cl⁻ 离去
- vs 分子间副反应：醇盐 + 别的分子的 4-氯，速率慢

**解答**：主产物为 2-甲基四氢呋喃（5 元环醚）。
$$\mathrm{4{-}Cl{-}1{-}OH{-}C_5H_{10}} + \text{NaH} \to \mathrm{2{-}CH_3{-}THF} + \text{NaCl} + H_2$$

**反思**：分子内 SN2（关环）优先于分子间——5/6 元环过渡态最佳，熵效应有利。这是 ABOC 行 1880 强调的"分子内反应优先考虑"原则的直接应用。

## 十一、易错点

| 错误 | 原因 | 纠正 |
|:---|:---|:---|
| ❌ 认为所有 2° 底物都走 SN2 | 看到 2° 直接归类 | 强碱条件（如 t-BuO⁻、LDA、NaH）时 E2 为主 |
| ❌ 在 SN2 中画出碳正离子中间体 | 把 SN1 机理代入 | SN2 是**一步协同反应**，无中间体；只有五配位过渡态 |
| ❌ 忽略溶剂效应 | 觉得溶剂不重要 | 极性非质子溶剂使阴离子亲核试剂活性大幅增强（10²-10⁵ 倍）|
| ❌ 忽略 NGP 导致的构型保持 | 默认 SN2 = 翻转 | 含邻基参与基团时（-COO⁻、-SR、-OH 等），表观构型保持 |
| ❌ 用 SN2 处理乙烯基卤和芳基卤 | 把"卤代物"一视同仁 | 这些底物的 SN2 极慢——必须走 [[SNAr]] 或金属催化 |
| ❌ 忽视反极化（亲卤取代） | 默认 SN2 都是进攻 C | 软亲核 + 稳定碳负条件下反极化常见，产物**保持**而非翻转 |
| ❌ "位阻只看 α 位" | 简化 | β 位（新戊基）的远程位阻可使速率降 10⁵ 倍 |
| ❌ 在 SN2 势能图上画两个峰 | 把 SN1 与 SN2 混淆 | SN2 是一峰；SN1 是两峰一谷（T7 经典考点）|

### 12.1 学习路径

- **学习本 KP 前，需要先掌握**：
  - "[[反式共平面]]（SN2 几何基础：Nu-C-LG 直线对齐）"
  - "[[反应机理表示法]]（一步协同的箭头推动 + Walden 翻转楔形键）"
  - "[[给体-受体相互作用]]（Nu σ 给体进攻 σ\*(C-LG) 受体）"
- **学完本 KP 后，自然过渡到**：
  - "[[SN1反应]]（对照对，理解"协同 vs 解离"差异）"
  - "[[E2反应]]（与 SN2 竞争，几何要求相同）"

### 12.2 学生易踩的认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析口诀/方法 |
|:---|:---|:---|:---|
| "SN2 = 翻转，SN1 = 外消旋" | 教科书二元对立 | 实际 SN1 也偏翻转（离子对效应）；SN2 在 NGP 或反极化下保持 | "翻转 vs 保持是结果，不是机理；机理看动力学和势能图" |
| "强碱 = 强亲核" | 把碱性等同于亲核性 | 强碱（如 t-BuO⁻）位阻大时是弱亲核（只能拔 H 走 E2）| "看大小：大位阻碱 → E2；小亲核 → SN2" |
| "极性溶剂 = 利于反应"| 极性是宽泛术语 | SN2 必须**极性非质子**（DMSO、DMF）；极性质子（H₂O、ROH）反而抑制 | "记住：质子坏（包阴离子），非质子好（裸阴离子）" |
| "位阻只看 α 位" | 把 SN2 看成 α-中心反应 | β-叔丁基的远程位阻使 SN2 慢 10⁵ 倍 | "中心碳要回头看远处" |
| "SN2 一定翻转，没例外" | 教科书"100% 翻转"绝对化 | NGP 体系两次翻转 = 保持；反极化体系也保持 | "见硫亲核 / 邻位带孤对 → 检查保持可能性" |
| "桥头碳的 SN2 慢" | 简化为"位阻"问题 | 桥头碳的 SN2 是**几何不可行**（sp³ 无法变 sp²+p 平面）| "桥头 = 0 速率，不是慢" |
| "看到 RX 就是 SN2" | 见 X 就用 SN2 | Ph-X、=CH-X、桥头-X 不可走 SN2；芳基要走 SNAr | "三类不能 SN2：芳/烯基/桥头" |

### 12.3 入门级例题（不同于第十节）

> 第十节例题面向"已学的人"做巩固；本段例题面向"刚学的人"做入门。

**题目**：CH₃Br 在 50% NaOH/H₂O 与 NaOH/DMSO 两种条件下都得到 CH₃OH。哪个条件下反应更快？为什么？

**预期解答路径**（把脑内推理写出来）：

1. 第一步看到题目应该想到：CH₃Br 是甲基底物（最优 SN2 底物）；OH⁻ 是强亲核试剂；问题是溶剂如何影响。
2. 然后联想到本 KP 的：SN2 在**极性非质子溶剂**（DMSO）中更快，因为：
   - 极性质子溶剂（H₂O）通过氢键包裹 OH⁻ → 削弱其亲核性
   - 极性非质子溶剂（DMSO）只溶剂化 Na⁺ → OH⁻ "裸露" → 亲核性大增
3. 应用：DMSO 条件下 SN2 速率比 H₂O 快 ~10³-10⁵ 倍。

**老师可在课堂上引导的提问**：
- 学生若卡在第 1 步：问 "CH₃Br 在 OH⁻ 下走什么机理？"
- 学生若卡在溶剂效应：问 "OH⁻ 在 H₂O 中是被氢键包围还是自由？" → 引导出 "包围 → 弱亲核"
- 进阶提问：如果换成 t-BuBr，DMSO 中还快吗？（不快，3° 底物根本不走 SN2，只走 SN1/E1，DMSO 反而抑制 SN1）

### 12.4 与现实/直觉的连接

- **类比**：SN2 像**"打台球的精确瞄准 + 推杆"** —— Nu（球杆）必须从 LG（被推的球）的**正背面**对准击球（C 中心球），才能让被推的球（LG）准确飞出，同时新球（Nu）填到原位。如果角度偏（不是反式共平面），力量分散，反应失败。SN1 则像**"等台球碰撞"** —— 球先自己散开（解离），再各自重组（亲核进攻）。
- **反例/佯谬**：表面上看 SN2 速率应该是"亲核越强越快"，但实际**强碱 + 位阻**（t-BuO⁻）几乎不参与 SN2，反而走 E2。这告诉我们：**亲核性 ≠ 碱性**——亲核性看是否能"伸到 C 上做共价键"；碱性看是否能"在远处拔 β-H"。位阻是分水岭。

## 十三、竞赛拓展

### 13.1 经典 SN2 反应（命名反应中的 SN2）

| 反应名 | 底物 + 试剂 | 备注 |
|:---|:---|:---|
| [[Finkelstein反应]] | RX + NaI/丙酮 → RI + NaX↓ | 卤素交换；NaI 溶于丙酮而 NaCl/NaBr 不溶 → 平衡推动 |
| [[相转移催化]]（PTC）| 季铵盐将水相 Nu⁻ 带入有机相 | 加速 SN2 数倍至千倍 |
| [[冠醚]] | 18-crown-6 络合 K⁺ 使 F⁻ "裸露" | 增强 F⁻ 亲核性 |
| [[Mitsunobu反应]] | ROH + Nu-H + PR₃ + DEAD → R-Nu | 醇构型翻转 SN2 |
| [[Williamson醚合成]] | RO⁻ + R'X → R-O-R' | 经典醚合成 |
| [[Gabriel合成]] | 邻苯二甲酰亚胺钾 + RX → R-NH₂ | 一级胺合成（避免多次烷基化）|

### 13.2 反极化与亲卤取代的"宽广视野"

> ABOC §2.9 把这一视角作为主线介绍，是有别于经典教材的关键创新。
>
> **应用领域**：
> - [[Pummerer反应]]：硫鎓中间体的"反极化 + 重排"
> - α-溴酮 + 软亲核：保持构型而非翻转
> - 某些 SE2 反应（亲电取代）的逆向 SN2 视角

### 13.3 邻基参与（NGP）的速率加速

| 邻基 | 加速倍数 | 经典案例 |
|:---|:---:|:---|
| -COO⁻ | 10²-10⁵ | 4-溴丁酸酯化 |
| -OH | 10²-10⁴ | 2-溴乙醇关环 |
| -SR / -S⁻ | 10⁵-10⁸ | 芥子气在水中的快速水解（最经典）|
| -Ar（π 参与）| 10²-10³ | 苯基亚乙基的 NGP |

### 13.4 同位素效应分析

> SN2 vs SN1 的实验区分手段：
> - **²H 动力学同位素效应**（α-H, KIE）：SN2 ≈ 1.0；SN1 ≈ 1.1-1.4
> - **¹⁸O 同位素分馏**（离去基中的 O）：SN2 全部翻转；SN1 部分外消旋
> - **温度依赖性**（活化能）：SN2 ΔH‡ ~ 15-25 kcal/mol；SN1 ΔH‡ ~ 20-30 kcal/mol
