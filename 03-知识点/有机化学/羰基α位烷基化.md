# 羰基α位烷基化

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[46-羰基α位反应]]

## 一、定义

**羰基α位烷基化**是指在碱作用下，羰基化合物（醛、酮、酯）α-位碳去质子形成**烯醇盐（enolate）**，随后烯醇盐作为亲核试剂进攻卤代烃（或其他亲电试剂），在 α-位引入烷基的反应。

- 总反应式：

$$
\ce{R-CO-CH2-R' + Base -> R-CO-CH^--R' \xrightarrow{R''-X} R-CO-CH(R'')-R' + X^-}
$$

- 核心：**烯醇盐是羰基α位烷基化的关键中间体**，其形成与稳定性决定反应条件

## 二、考纲对应

- 对应考纲条目：[[46-羰基α位反应]]
- 所属模块：[[基础要求-有机化学]]
- 作用：碳碳键构建的核心方法之一，与 Aldol、Claisen 并列的 α-位反应三大类型

## 三、核心原理

### 3.1 烯醇盐的形成

羰基 α-H 的 pKa ~ 19–25（酮）或 ~ 24–25（酯），需**强碱**才能去质子：

| 碱 | pKa(共轭酸) | 适用范围 | 备注 |
|:---|:---:|:---|:---|
| **LDA** | ~36 | 酮、酯、酰胺 | 动力学控制，低温 |
| **NaH** | ~35 | 1,3-二羰基、β-酮酯 | 不可逆去质子 |
| **NaOEt/EtOH** | ~16 | 1,3-二羰基（活性更高）| 热力学控制 |
| **t-BuOK** | ~18 | 动力学/热力学切换 | 取决于温度和溶剂 |

### 3.2 动力学 vs 热力学烯醇盐

- **动力学烯醇盐**：低温、强位阻碱（LDA）、动力学控制 → 取代较少的一侧去质子（位阻小）
- **热力学烯醇盐**：高温、弱碱、质子性溶剂 → 更稳定的取代较多一侧的烯醇盐

$$
\ce{CH3COCH2CH3 \xrightarrow[\text{-78°C}]{LDA} \text{动力学烯醇盐} \quad vs \quad \xrightarrow[\Delta]{NaOEt} \text{热力学烯醇盐}}
$$

### 3.3 烷基化步骤（SN2）

烯醇盐进攻卤代烃，典型的 **SN2 机理**：

- **R-X 要求**：1° 或 2° 卤代烷（甲基最好）；3° 卤代烷易消除
- **离去基**：I > Br > Cl（I 最佳但较贵，Br 常用）
- **溶剂**：极性非质子溶剂（DMF、DMSO）加速 SN2

## 四、关键结论

### 4.1 反应成功的四要素

1. **足够的烯醇盐浓度** → 强碱 + 无水条件
2. **合适的亲电试剂** → 1°/2° RX，避免 3°（消除竞争）
3. **避免 O-烷基化** → 烯醇盐有两个亲核位点（C 和 O），极性非质子溶剂有利于 C-烷基化
4. **控制聚合/二聚** → 过量碱或 RX 可能导致多烷基化

### 4.2 与 Aldol 的竞争

- **烯醇盐 + RX** → α-烷基化（SN2）
- **烯醇盐 + 羰基** → Aldol 缩合（亲核加成）

> 控制方法：先 fully 形成烯醇盐，再一次性加入 RX；或预先硅基化保护（烯醇硅醚）后再反应。

## 五、常见分类或情形

### 5.1 按羰基类型

| 底物 | 条件特点 | 典型产物 |
|:---|:---|:---|
| **单酮** | LDA, -78°C, THF | α-烷基酮 |
| **1,3-二羰基** | 弱碱即可（pKa ~9–13） | α-烷基-1,3-二羰基 |
| **酯** | LDA 或 NaHMDS | α-烷基酯 |
| **β-酮酯** | NaOEt/EtOH | 活泼亚甲基烷基化 |

### 5.2 特殊变体

- **丙二酸酯合成**：丙二酸二乙酯 → 二烷基化 → 水解脱羧 → 取代乙酸
- **乙酰乙酸乙酯合成**：类似丙二酸酯，但只保留一个羰基
- **Stork 烯胺烷基化**：烯胺（enamine）代替烯醇盐，避免 O-烷基化和聚合

## 六、适用条件与限制

- **醛通常不适用**：醛的 α-烷基化产率低，因为醛在碱性条件下易发生 Aldol 自身缩合
- **不对称酮的位置选择性**：必须通过动力学/热力学控制选择烷基化位置
- **手性控制**：无手性辅助时，产物为外消旋；用 Evans 手性辅基可实现对映选择性
- **酯的 O-烷基化竞争**：酯烯醇盐在极性质子溶剂中易发生 O-烷基化

## 七、常见比较与易混点

- **C-烷基化 vs O-烷基化**：C-产物是目标（碳链延长），O-产物是副产物；极性非质子溶剂（DMF/DMSO）有利于 C-烷基化
- **动力学 vs 热力学控制**：不是"哪个更好"，而是"想要哪个位置的产物"→ 选对应条件
- **α-烷基化 vs Aldol**：两者都用烯醇盐，但亲电试剂不同（RX vs 羰基）
- **α-烷基化 vs Michael 加成**：Michael 是烯醇盐进攻 α,β-不饱和羰基（1,4-加成），不是 RX

## 十、例题

### 例题 1

**题目：** 2-丁酮用 LDA 在 -78°C 下处理，随后加入碘甲烷，主要产物是什么？若改用 NaOEt/EtOH 回流，产物有何不同？

**分析：** 涉及动力学 vs 热力学烯醇盐的选择。

**解答：**
- **LDA, -78°C（动力学控制）**：去质子发生在位阻较小的甲基侧 → 形成取代较少的烯醇盐 → 甲基化后得 **3-甲基-2-丁酮**（CH₃COCH(CH₃)₂）
- **NaOEt, 回流（热力学控制）**：形成更稳定的取代较多的烯醇盐（亚甲基侧）→ 甲基化后得 **2-戊酮**（CH₃COCH₂CH₂CH₃）

**反思：** 同一底物、同一亲电试剂，**碱和温度决定位置选择性**。

## 十一、易错点

- 用 NaOH 或 KOH 做碱 → 这些碱不够强，无法充分去质子普通酮（pKa ~20），产率极低
- 3° 卤代烷 → 主要发生 E2 消除，不是 SN2 烷基化
- 忘记区分动力学/热力学条件 → 导致产物结构判断错误
- 醛的 α-烷基化 → 实际上在竞赛中几乎不用，因为 Aldol 竞争太严重

### 12.2 学生易踩的认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析方法 |
|:---|:---|:---|:---|
| "弱碱 NaOH 也能让酮烷基化" | 以为碱只是"催化" | 普通酮 α-H pKa~20，NaOH（H₂O pKa~15.7）热力学上不能充分去质子 | 查 pKa 表：碱的共轭酸 pKa 必须 > 底物 pKa + 2 |
| "3° 溴代烷也能 SN2" | 死记"卤代烷都能反应" | 3° RX 位阻太大，烯醇盐作为强碱会优先引发 E2 消除 | 记住：烯醇盐 = 强亲核 + 强碱，3° RX → 消除为主 |
| "动力学 = 快，热力学 = 慢" | 字面联想 | 动力学控制 = **低温下不可逆去质子**得到 kinetic 产物；热力学 = 高温平衡得到 thermodynamic 产物 | 记条件：LDA/-78°C = 动力学；NaOEt/Δ = 热力学 |

### 12.3 入门级例题

**题目：** 用丙二酸二乙酯合成 2-甲基丁酸。

**预期解答路径：**
1. 丙二酸二乙酯 + NaOEt → 烯醇盐
2. + CH₃CH₂I → α-乙基化产物
3. 重复：+ CH₃I → α,α-二甲基？不对...

修正：目标 2-甲基丁酸 = (CH₃)CH(COOH)CH₂CH₃
- 丙二酸二乙酯 + NaOEt → 烯醇盐
- + CH₃CH₂I → 单乙基化
- + CH₃I → 甲基化得 α-乙基-α-甲基丙二酸酯
- 水解、酸化、加热脱羧 → 2-甲基丁酸

**课堂引导提问：**
- 若学生不知道从何入手：问"丙二酸酯合成法的核心步骤是哪三步？"
- 若学生产物写错：提醒"脱羧后保留的是哪个羰基？"

### 12.4 与现实/直觉的连接

- 类比：烯醇盐的 C-烷基化 vs O-烷基化就像**拳击手的左右手**——同一个人（烯醇盐），打左边（C）还是右边（O）取决于对手站位（溶剂极性）
- 反例：学生以为"碱性越强越好"——实际上过强的碱（如丁基锂）可能引发副反应，LDA 是"刚刚好"的选择
