# 1,2-迁移与重排

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[32-有机反应机理]]、[[34-有机活性中间体]]、[[39-重排反应]]

## 一、定义
**1,2-迁移（1,2-Shift / 1,2-Migration）**：一个基团（H、烷基、芳基等）连同**键对电子**从一个原子（Cα）迁移到相邻原子（Cβ）的过程。

通式：
$$\ce{[X-Cα-Cβ]+ -> [Cα=Cβ-X]+}$$（X 带电子对一起迁移）

**驱动力**：生成更稳定的离子或物种（更稳定的碳正离子、消除张力等）。

## 三、核心原理

### 3.1 迁移的电子流动
- **迁移基带电子对**：单条箭头从迁移基的 σ 键 → 缺电子原子的空轨道
- 等同于"亲核体"（迁移基）攻击"亲电体"（缺电子原子）
- 经典例：碳正离子重排

```
     H              H
     |              |
 R3C-C(+)-R'  →  R3C(+)-C-R'
                       
 (氢迁移)  H 带电子对从 Cα 迁移到 Cβ
```

### 3.2 迁移基的种类
| 迁移基 | 特点 | 实例 |
|---|---|---|
| H（氢化物迁移） | 最常见，最快 | 仲碳正离子 → 叔碳正离子 |
| 烷基（R） | Wagner-Meerwein 重排 | 新戊基重排 |
| 芳基（Ar） | 通过桥连过渡态（苯鎓） | 联苯酸 → 二苯甲基重排 |
| 卤素（X） | 较少（破坏稳定 C-X） | 部分 1,2-溴迁移 |
| OH/NH₂ | 通常先去质子再迁移 | Pinacol、Beckmann 重排 |

### 3.3 迁移倾向（迁移倾向：H ≈ Aryl > Alkyl）
- H：单原子迁移，过渡态紧密，能垒低
- Aryl：通过 Φ-桥（苯鎓）过渡态，稳定
- Alkyl：通过普通三中心 TS

### 3.4 立体化学：保留构型
迁移基**带电子对**迁移 → 在迁移过程中**保持构型**（即手性中心保留 R/S）。

## 四、关键结论

### 4.1 经典重排反应
| 重排 | 关键步骤 | 实例 |
|---|---|---|
| **Wagner-Meerwein** | 1,2-烷基迁移在碳正离子上 | 莰烯重排 |
| **Pinacol 重排** | 二醇脱水后 1,2-烷基迁移 | 频哪醇 → 频哪酮 |
| **Pinacolone 重排** | 同上的逆 | — |
| **Beckmann 重排** | 肟 → 酰胺，1,2-Aryl 迁移 | 环己酮肟 → 己内酰胺（尼龙原料） |
| **Schmidt 重排** | 酮 + HN₃，1,2-R 迁移 | 酮 → 酰胺 |
| **Hofmann 重排** | 酰胺 → 胺，1,2-R 迁移 | 酰胺 → 异氰酸酯 → 胺 |
| **Curtius 重排** | 酰基叠氮 → 异氰酸酯 | — |
| **Wolff 重排** | 重氮酮 → 烯酮，碳烯中间体 | Arndt-Eistert 反应 |

### 4.2 主族化学中的 1,2-迁移
- **硼烷重排**：$\ce{R3B \to R-B-R-R}$ 在加热下重排
- **硅烷重排**：$\ce{Si-Si}$ 间的 1,2-迁移
- **磷化合物**：磷叶立德的 Wittig 反应中部分有 1,2-迁移

### 4.3 不发生 1,2-迁移的情形
- 已是最稳定结构（无热力学驱动力）
- 桥头碳正离子（结构限制）
- 立体障碍

## 五、常见分类或情形

### 5.1 碳正离子重排（最常见）
**示例：仲丁基 → 叔丁基**
$$\ce{(CH3)2CH-CH2(+) \xrightarrow{H \text{迁移}} (CH3)2C(+)-CH3}$$

### 5.2 Pinacol 重排
$$\ce{(CH3)2C(OH)-C(OH)(CH3)2 \xrightarrow{H+} (CH3)3C-CO-CH3 + H2O}$$
机理：
1. 一个 OH 质子化 → H₂O 离去 → 生成 3° 碳正离子
2. 邻位 CH₃ 迁移到正电荷碳上（1,2-甲基迁移）
3. 离子重排为 4° 碳正离子（次稳定）→ 失去 H⁺ 形成 C=O

**频哪醇重排双规则（Zchem 系统化工具）**：

**规则一：稳定碳正离子的基团迁移更快**
- 迁移能力：Ph > R > H（与 Wagner-Meerwein 重排"越小越快"相反！）
- 原因：迁移基团需通过共轭效应或超共轭效应稳定过渡态中的正电荷
- 实例：苯基迁移快于甲基，因为苯基可通过共轭分散正电荷

**规则二：满足反式共平面迁移速率更快**
- 空间要求：迁移基团与离去基团（质子化的 OH → H₂O）需处于反式共平面
- 轨道理论：迁移基团的 σ 键 HOMO 与碳正离子空 p 轨道 LUMO 需要最大重叠
- 环己烷体系：只有处于反式双直立键（diaxial）位置的基团才能有效迁移

**与 Wagner-Meerwein 重排的关键区别**：

| 特征 | W-M 重排 | 频哪醇重排 |
|:---|:---|:---|
| 迁移顺序 | 越小越快（H > Me > Et）| 给电子能力越强越快（Ph > R > H）|
| 原因 | 轨道匹配性（小基团 σ 键与空 p 轨道重叠更好）| 正电荷稳定能力（过渡态需求）|
| 驱动力 | 生成更稳定碳正离子 | 生成更稳定羰基化合物（C=O 键能大）|

### 5.3 Beckmann 重排
$$\ce{R2C=N-OH \xrightarrow{H+} R-CO-NHR}$$
机理：
1. OH 质子化 → 生成 OH₂⁺
2. **反式 R 基**伴随电子对迁移到 N 上（同时 H₂O 离去）
3. 生成 C=N⁺-R 阳离子
4. 水加成 → 互变异构 → 酰胺

### 5.4 Wolff 重排
$$\ce{R-CO-CHN2 \to R-CH=C=O \xrightarrow{H2O} R-CH2-COOH}$$
- 加热或光照下，N₂ 离去 → 卡宾中间体
- R 基从 C 迁移到卡宾 C 上
- 形成稳定烯酮

## 六、适用条件与限制

### 适用条件
- 存在缺电子中心（碳正离子、卡宾、缺电子氮等）
- 邻位有可迁移基团（H、R、Ar）
- 迁移后的产物更稳定

### 限制
- 桥头位置的 1,2-迁移受立体限制
- 双键 sp² 碳上不能成正离子（不稳定）
- 极端位阻或几何不利

## 七、常见比较与易混点

### 1. 1,2-迁移 vs σ-bond metathesis
- 1,2-迁移：在**同一分子内**进行，迁移基带电子对
- σ-bond metathesis：**分子间**协同的双键交换

### 2. 迁移基带电子对 vs 离去基带电子对
- 迁移基（H、R、Ar）**带电子对**——所以保留构型
- 离去基（X、H₂O）**带电子对**离开——所以是离子化

### 3. 1,2-迁移 vs 1,3-迁移
- 1,2-迁移：相邻两原子间，几何最有利
- 1,3-迁移：常需要σ-键沿环形断裂（周环反应）
- 一般情况下 1,2-迁移远比 1,3-迁移容易

### 4. 1,2-迁移 vs 取代
- 取代：**外来**基团进攻，离去基离开
- 1,2-迁移：**自身**基团迁移（无外来）

## 十、例题

### 例题 1：仲丁基 OBs 的 SN1 重排
**题目：** $\ce{(CH3)2CH-CH(OBs)-CH3}$（仲）经 SN1 反应，主产物是叔丁基取代物，解释。
**分析：**
- 仲碳正离子先形成
- 邻位 H 迁移 → 叔碳正离子（更稳定）
- 亲核体进攻叔位
**解答：** 1,2-氢迁移使产物从仲位转为叔位。

### 例题 2：Pinacol 重排
**题目：** 频哪醇 $\ce{(CH3)2C(OH)-C(OH)(CH3)2}$ 在酸催化下生成什么？写机理。
**分析：**
- 一个 OH 质子化 → 离去 → 3° 碳正离子
- 邻位 CH₃ 迁移 → 形成 OH 邻位碳上的正电荷
- 失水 → C=O
**解答：** 产物为频哪酮 $\ce{(CH3)3C-CO-CH3}$。

## 十一、易错点
- 忘记迁移基**带电子对**——错误画箭头
- 忽视 H 迁移的高速性——错误地认为只有烷基迁移
- 在 sp² 上画"1,2-碳正离子重排"——sp² 不能形成稳定碳正离子
- 把 Beckmann 重排中的迁移基误认为 cis 而非 trans
