# TosMIC（对甲苯磺酰甲基异腈，Van Leusen 试剂）

## 一、定义

**TosMIC**（p-Toluenesulfonylmethyl isocyanide，对甲苯磺酰甲基异腈）是一种**异腈类试剂**，在碱性条件下与**醛/酮**反应，可将醛转化为**多一个碳的腈**，或合成**噁唑**类杂环。

**通式（醛 → 腈）**：
$$
\ce{R-CHO + TosMIC + base -> R-CH2-CN + TsOH}
$$

**本质**：TosMIC 是**C₁ 合成子**——将醛的羰基碳与 TosMIC 的异腈碳结合，净增加一个碳原子得到腈。

## 三、核心原理

### 3.1 醛 → 腈的机理（Van Leusen 反应）

```
步骤 1：TosMIC 的活泼亚甲基在碱作用下脱质子 → 碳负离子
步骤 2：碳负离子进攻醛的羰基碳 → 烷氧负离子
步骤 3：分子内环化 → 5-噁唑啉中间体
步骤 4：消除对甲苯亚磺酸（TsH）→ 腈
```

**关键特征**：
- **C₁ 增量**：醛（R-CHO）→ 腈（R-CH₂-CN），净增 1 个碳
- **Ts⁻ 是好的离去基**：对甲苯磺酰基（Tos）稳定碳负离子，也促进消除
- **碱的选择**：通常用 K₂CO₃、NaH 或 DBU

### 3.2 噁唑合成

**TosMIC + 醛 → 噁唑**（在另一醛存在下或不同条件下）：
$$
\ce{R-CHO + TosMIC ->[base] 2-取代噁唑}
$$

- TosMIC 提供噁唑环的 **C-2** 和 **C-5**
- 醛提供噁唑环的 **C-4** 取代基

### 3.3 TosMIC 的结构与反应性

| 结构特征 | 作用 |
|:---|:---|
| **异腈基（-NC）** | 亲电性碳，接受亲核进攻 |
| **磺酰基（-SO₂Tol）** | 吸电子基，稳定 α-碳负离子 |
| **活泼亚甲基（-CH₂-）** | 在碱作用下形成碳负离子 |

---

## 四、关键结论

1. **TosMIC = 醛 + base → 多一个碳的腈**
2. **核心试剂是对甲苯磺酰甲基异腈**——兼具异腈和活泼亚甲基
3. **Ts 基团是灵魂**——既稳定碳负离子，又是离去基
4. **也可用于噁唑合成**——杂环化学中的重要试剂
5. **与 Ugi/Passerini 同属异腈化学家族**——但机理和应用完全不同

## 五、常见分类或情形

### 5.1 按产物分类

| 反应条件 | 产物 | 应用 |
|:---|:---|:---|
| **TosMIC + 醛 + 碱** | 多一个碳的腈 | 醛 → 腈的转化 |
| **TosMIC + 醛（另一条件）** | 2-取代噁唑 | 杂环合成 |
| **TosMIC + 酮** | 类似产物（位阻大时效率低）| 酮 → 腈 |

### 5.2 与其他醛→腈方法的对比

| 方法 | 试剂 | 碳数变化 | 特点 |
|:---|:---|:---:|:---|
| **TosMIC** | TosMIC, base | +1 | 条件温和，选择性好 |
| **肟脱水** | NH₂OH → 肟 → 脱水 | 0 | 经典方法，需强脱水剂 |
| **Sandmeyer** | ArN₂⁺ + CuCN | 0 | 从芳胺出发 |
| **酰胺脱水** | RCONH₂ + P₂O₅ | 0 | 强条件 |

## 六、适用条件与限制

1. **需要碱**——K₂CO₃、NaH、Et₃N 等
2. **醛比酮反应性好**——酮的位阻大，产率较低
3. **对甲苯亚磺酸是副产物**——需分离
4. **底物不能有对碱敏感的基团**
5. **TosMIC 本身有异味**——操作注意通风

---

## 七、常见比较与易混点

### 7.1 TosMIC vs Ugi/Passerini

- TosMIC：用**碱**，产物是**腈或噁唑**
- Ugi/Passerini：用**酸（羧酸）**，产物是**酰胺**
- 三者都用异腈，但条件和产物完全不同

### 7.2 TosMIC vs 肟脱水

- TosMIC：从醛出发，+1 碳得腈，一步完成
- 肟脱水：从醛出发，先成肟再脱水，碳数不变
- TosMIC 更直接，且增加一个碳

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：苯甲醛 + TosMIC / K₂CO₃ 的产物是什么？

**解答**：
- 苯甲醛（C₆H₅-CHO）+ TosMIC → 苯乙腈（C₆H₅-CH₂-CN）
- 净增加一个碳原子
- 产物：**苯乙腈**

### 10.2 例题 2（★★ 碳数计算）

**题目**：用 TosMIC 从正丁醛制备的腈是什么？

**解答**：
- 正丁醛：CH₃CH₂CH₂-CHO（4 个碳）
- TosMIC 提供 1 个碳（异腈碳）
- 产物：CH₃CH₂CH₂-CH₂-CN = **戊腈**（5 个碳）

---

## 十一、易错点

1. ❌ **TosMIC 产物碳数不变** → TosMIC 让醛**多一个碳**得腈
2. ❌ **TosMIC 和 Ugi 条件一样** → TosMIC 用**碱**，Ugi 用**酸/羧酸**
3. ❌ **TosMIC 只能做腈** → TosMIC 也能做**噁唑**
4. ❌ **酮和 TosMIC 反应很好** → **醛**比酮反应性好

### 12.3 入门例题

**题目**：为什么 TosMIC 能实现"醛 → 多一个碳的腈"？

**引导**：
- TosMIC 的异腈碳（:C≡N⁺）进入产物
- 醛的羰基碳也保留在产物中
- 两者结合后消除 TsH → 腈
- 所以净结果是 R-CHO → R-CH₂-CN（+1 碳）

### 12.4 现实类比

**TosMIC 像"带弹簧的嫁接器"**：
- 醛 = 砧木（需要嫁接的植物主干）
- TosMIC = 接穗 + 弹簧装置
- 碱 = 按下嫁接按钮
- 嫁接后弹簧脱落（TsH 离去）
- 最终植物长高了一节（多一个碳）
