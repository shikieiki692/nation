# BrF₃（三氟化溴）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：卤素化学拓展（APiIC 占位）

## 一、定义

**BrF₃（三氟化溴，Bromine Trifluoride）** 是一种 **强氟化试剂** 和 **强氧化剂**，属于 **卤素互化物**。在有机化学中主要用于 **芳环氟化** 和 **含氟有机物的制备**。

基本性质：
| 性质 | 数值/描述 |
|:---|:---|
| **分子式** | BrF₃ |
| **外观** | 无色至淡黄色液体 |
| **沸点** | 125.8°C |
| **结构** | T 形（Br 中心，两赤道 F，一轴向 F，两孤对电子）|
| **氧化态** | Br 为 +3 |
| **危险性** | 极强腐蚀性，与有机物接触可能燃烧或爆炸 |

## 二、考纲对应

- 中国化学奥林匹克考纲未单独列出，属于 **卤素互化物** 和 **氟化学** 的拓展内容。
- 竞赛中可能以无机结构（T 形、VSEPR）或有机氟化为背景出现。

## 三、核心原理

### 3.1 作为氟化试剂

BrF₃ 可将有机化合物中的 **C-H 键转化为 C-F 键**，或将其他卤素置换为氟：

$$\mathrm{R-H + BrF_3 \rightarrow R-F + HF + BrF}$$

**反应性特点**：
- BrF₃ 是 **亲电氟化试剂**（F⁺ 等价物）
- 反应通常伴随 **芳环亲电取代** 机理
- 对芳环电子密度敏感：**富电子芳环 > 缺电子芳环**

### 3.2 作为氧化剂

BrF₃ 中 Br 为 +3 氧化态，可被还原为 Br₂ 或 Br⁻：

$$\mathrm{BrF_3 + 2e^- \rightarrow BrF + 2F^-}$$
$$\mathrm{BrF_3 + 4e^- \rightarrow \frac{1}{2}Br_2 + 3F^-}$$

**氧化能力**：BrF₃ > ClF₃ > IF₅（在卤素氟化物中）

### 3.3 作为路易斯酸

BrF₃ 可接受 F⁻ 形成配离子：

$$\mathrm{BrF_3 + F^- \rightarrow BrF_4^-}$$

在超强酸体系中，BrF₃ 与 SbF₅ 等可形成 **氟离子供体/受体体系**。

## 四、关键结论

1. **BrF₃ 是强亲电氟化试剂和强氧化剂**
2. **T 形分子结构**（AX₃E₂，VSEPR）— 竞赛结构题考点
3. **主要用于芳环氟化** — 富电子芳环优先
4. **操作极危险** — 需在专业防护下进行，通常用惰性溶剂稀释

## 五、常见分类或情形

| 应用 | 底物 | 产物 | 条件 |
|:---|:---|:---|:---|
| **芳环氟化** | 苯、甲苯 | 氟苯、氟甲苯 | 低温，惰性溶剂 |
| **卤素交换** | 氯代芳烃 | 氟代芳烃 | 加热 |
| **无机合成** | 金属氧化物 | 金属氟化物 | BrF₃ 作为氟化剂 |
| **六氟化制备** | MoO₃ → MoF₆ | — | 高温 |

## 六、适用条件与限制

| 条件 | 说明 |
|:---|:---|
| **溶剂** | 必须在惰性溶剂中进行（如 CFCl₃、液态 N₂）|
| **温度** | 通常 -78°C → 0°C（控制放热）|
| **防护** | 需专业设备，避免与有机物、水、还原剂接触 |
| **限制** | 极危险，实验室很少直接使用；常改用更温和的氟化试剂（如 Selectfluor、F₂/N₂）|

## 七、常见比较与易混点

### BrF₃ vs 其他氟化试剂

| 试剂 | 类型 | 活性 | 安全性 | 应用场景 |
|:---|:---|:---:|:---:|:---|
| **BrF₃** | 亲电氟化 | 极高 | 极危险 | 工业无机氟化 |
| **F₂/N₂** | 亲电氟化 | 极高 | 危险 | 直接氟化 |
| **Selectfluor** | 亲电氟化 | 中等 | 较安全 | 实验室芳环氟化 |
| **HF/Py** | 亲核氟化 | 中等 | 较安全 | 醇 → 氟代烷 |
| **DAST** | 亲核氟化 | 中等 | 较安全 | 醇 → 氟代烷 |

## 十、例题

### 例题 1：结构与 VSEPR

**题目**：预测 BrF₃ 的分子几何构型，并说明键角偏离理想值的原因。

**解析**：
- Br 的价电子：7
- 成键：3 个 Br-F 键 → 3 对成键电子
- 孤对电子：(7-3)/2 = 2 对孤对电子
- VSEPR：AX₃E₂ → **T 形**
- 理想角：90°（赤道-轴向）和 180°（轴向-轴向）
- 实际：轴向 F-Br-F ≈ 86°（孤对电子排斥压缩键角）

### 例题 2：氧化态与反应

**题目**：BrF₃ 与水的反应产物是什么？写出方程式。

**解析**：
BrF₃ 剧烈水解：
$$\mathrm{BrF_3 + 2H_2O \rightarrow 3HF + HBrO_2}$$
或更常见：
$$\mathrm{3BrF_3 + 5H_2O \rightarrow HBrO_3 + Br_2 + 9HF + O_2}$$
（伴随歧化和氧化水）

## 十一、易错点

| 错误 | 纠正 |
|:---|:---|
| 误认为 BrF₃ 是 Br⁻ + F³⁻ | BrF₃ 是共价化合物，Br 氧化态为 +3 |
| 与 BrF₅ 混淆 | BrF₅ 是四方锥（AX₅E），Br 氧化态 +5 |
| 忽略安全警告 | BrF₃ 极强腐蚀性，实验室几乎不使用；竞赛中主要考结构和理论 |
