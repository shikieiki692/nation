# 超共轭效应
- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[22-分子轨道理论]]、[[28-有机物结构的稳定性]]、[[29-共振论]]、[[32-有机反应机理]]

## 一、定义

**超共轭效应（Hyperconjugation）** 是指**σ 成键电子（或孤对电子）**与**相邻 σ\*/π\*/p 空轨道**之间的离域相互作用。其本质是 [[给体-受体相互作用]] 的一种特殊形式。

> 在 ABOC 的统一框架下，传统"超共轭"和"共轭"被统一称为"给体-受体相互作用"——前者涉及 σ 键/孤对电子作给体，后者涉及 π 键作给体，**两者本质上是同一种轨道重叠现象**。

### 1.1 历史定义（传统视角）

在传统教科书中，超共轭被定义为：

> "C—H σ 键（或 C—C σ 键）的电子向相邻的 p 空轨道、π 体系或自由基单电子离域。"

经典例子：解释烷基对碳正离子、自由基的稳定化作用——
- 烷基越多 → C—H σ 键越多 → 超共轭越强 → 稳定性越大
- 顺序：3° > 2° > 1° > CH₃⁺
- 同位素效应：C—H > C—D > C—T 超共轭

### 1.2 现代定义（ABOC 视角，以本 KP 为主）

**统一描述**：

任何**填充电子的轨道（给体）**与**未填充的较低能级轨道（受体）**之间的电子离域，只要：

1. 两轨道**反式共平面**（trans-coplanar，二面角约 180°）
2. 轨道**有效重叠**（与给体/受体的"波瓣大小"匹配）
3. 给体能级 < 受体能级

就构成超共轭/共轭/给-受体相互作用。

参见数学形式：
$$\Delta E_\text{stab} \propto \frac{|H_{ij}|^2}{E_\text{acceptor} - E_\text{donor}}$$

### 1.3 经典亚类

| 亚类 | 给体 | 受体 | 例子 |
|:---|:---|:---|:---|
| **σ-p 超共轭** | C—H σ 键 | 空 p 轨道 | 烷基稳定碳正离子 |
| **σ-σ\* 超共轭** | C—H σ 键 | 邻位 C—X σ\* 反键轨道 | H-C-C-F 中的 H-C → C-F\* |
| **σ-π\* 超共轭** | C—H σ 键 | C=O π\* 反键轨道 | 烷基稳定共轭烯酮 |
| **n-σ\* 超共轭** | 杂原子孤对电子 n | 邻位 C—X σ\* 反键轨道 | [[端基效应]]、Bohlmann 效应 |
| **n-π\* 超共轭** | 杂原子孤对电子 n | C=O π\* 反键轨道 | 共振论中"+M 共轭" |

> 这五种亚类**本质相同**，只是给体/受体的具体身份不同。**重叠的几何（反式共平面）和能级匹配是核心，给体类型只是"换了个标签"**。

## 三、核心原理

### 3.1 几何要求：反式共平面（不可缩写！）

> ⚠️ **请不要对"反式共平面"这个词进行缩写！**（ABOC 原文强调三次）

考察 H-C-C-F 体系中 H-C σ 键给电子到 C-F σ\* 反键轨道：

- **反式共平面**（H-C-C-F 二面角 = 180°）：**重叠最大**
- **顺式共平面**（H-C-C-F 二面角 = 0°）：**重叠不利**
- **歪斜构象**（60°-120°）：**重叠介于两者之间**

**为什么必须是反式共平面？**

反键轨道（σ\*）的最大波瓣处在**成键的后方**（即键中心外侧）。给体 σ 键的成键电子云若想"无障碍"地填入反键轨道的最大波瓣处，必须从该方向接近——也就是与受体键反向（即反式）共平面。

![](mineru/ABOC202505_1-200_images/2b425c4025687d326176579411cbbfd2ea562c67018ff32bd04dc1a2f83ef06d.jpg)

### 3.2 HOMO 与 LUMO 的能级图分析

**关键概念**：HOMO（Highest Occupied Molecular Orbital，最高占据分子轨道）与 LUMO（Lowest Unoccupied Molecular Orbital，最低未占据分子轨道）。

> ABOC 习惯用 OMO/UMO（Occupied/Unoccupied）替代 HOMO/LUMO 强调"任意"前线轨道——但在初学阶段两者等价。

**对 σ 键的能级图**：

| 键 | 表观电负性 | HOMO（σ）相对位置 | LUMO（σ\*）相对位置 |
|:---|:---|:---|:---|
| C-C | 相同 | 偏中央 | 偏中央 |
| C-Si | C > Si | 偏 Si 侧（高） | 偏 C 侧（低） |
| C-H | C > H | 偏 H 侧（高） | 偏 C 侧（低） |
| C-F | F > C | 偏 F 侧（低） | 偏 C 侧（高） |

> **重要规律**：电负性大的原子拉低自己侧的轨道能量；电负性小的原子推高自己侧的轨道能量。

### 3.3 波瓣大小与重叠的不利性

为什么 H-C-C-F 中**反向超共轭**（F-C → C-H\*）几乎不发生？

**能量论证**：
- C-F 的 σ\*（受体）能量低（F 拉低受体侧轨道）→ 易接受电子
- C-H 的 σ\*（受体）能量高（H 推高受体侧轨道）→ 难接受电子

**轨道波瓣论证**：
- C-H 的 σ\*（反键）：C 贡献小（C 比 H 电负性大）→ 反键轨道波瓣偏 H 侧，C 侧波瓣**小**
- F-C 的 σ（成键）：F 贡献大（F 比 C 电负性大）→ 成键波瓣偏 F 侧，C 侧波瓣**小**
- 两个"小波瓣"相遇 → 重叠极小 → 不发生超共轭

**结论**（ABOC 行 933）：

> 超共轭**不是互相的**！如果一边比另一边有利很多，我们只需要考虑**有利的一组作用**，可以直接忽略不利的一组。

> 这一结论直到学习羰基亲核加成理论模型（Bürgi-Dunitz 角度）之前都是正确的。

## 四、关键结论

### 4.1 给体强弱顺序

ABOC 行 970-984 系统给出的给体顺序：

$$\text{F-C} < \text{O-C} < \text{N-C} < \text{C-C} < \text{H-C} < \text{S-C} < \text{P-C} < \text{Si-C} < \text{Li-C}$$

**解读**：
- **左端（F-C, O-C, N-C）**：电负性大，σ 成键电子被拉向 X 侧，C 侧的成键波瓣很小 → 难给电子
- **右端（Si-C, Li-C）**：电负性小，σ 成键电子被推向 C 侧，C 侧的成键波瓣很大 → 易给电子
- **C-C ≈ "中性参考"**：表观电负性相同，两侧波瓣对称
- **H-C > C-C 的原因**：H 电负性比 C 小（H 是 2.2，C 是 2.55），H-C 比 C-C 是更好的超共轭给体

> 卤素子顺序：**F-C < Cl-C < Br-C < I-C**（高周期电负性更小，碳-碘键已开始向 C 极化；轨道半径相差大，成键松散，更易重叠）

### 4.2 受体强弱顺序

#### 4.2.1 电负性影响

$$\text{C-F} > \text{C-O} > \text{C-N} > \text{C-C}$$

电负性大的原子使受体侧的反键轨道能级降低 → 易接受电子。

#### 4.2.2 可极化性（轨道弥散度）影响

$$\text{C-I} > \text{C-Br} > \text{C-Cl} > \text{C-F}$$
$$\text{C-S} > \text{C-O}$$
$$\text{C-P} > \text{C-N}$$

高周期元素轨道更弥散 → 重叠面积更大 → 接受电子能力更强。

> ⚠️ **特殊**：与高周期元素成键时反键轨道更松散，**这些键既是好的给体又是好的受体，无论正向（如 S-C）还是反向（如 C-S）**。例如 S-C-C-F 是一组有利的相互作用，**Si-C-C-F 则是极强的超共轭效应**——这就是 [[β-硅基效应]] 的本质。

### 4.3 重要推论

1. **乙烷的 3 组超共轭**：全交叉构象有 3 组反式共平面 H-C 对 C-H σ-σ\* 超共轭（每组都是 H-C → C-H\*），是其稳定性的来源。
2. **CF₃CH₃ vs CH₃CH₃ 的 C-C 键长**：CF₃CH₃（1,1,1-三氟乙烷）的 C-C 键比 CH₃CH₃ 短。原因：CF₃CH₃ 中 H-C → C-F 是**强超共轭**（受体一边电负性大），加强了中间 C-C 键。"一组好超共轭胜过三组普通超共轭"。
3. **苯基/烯丙基稳定碳正离子超过烷基**：因为 π → p 共轭比 σ → p 超共轭更强。
4. **β-Si 效应**：C-Si σ 键作给体特别好（Si 是优秀给体），可使邻位碳正离子稳定性增加 ≥ 10¹⁰ 倍。

### 4.4 优先离域原则

ABOC 行 992-1004 引入：

> **优先离域原则**：电负性小的原子上面的孤对电子能量更高，需要**优先**被离域。

**经典案例**：LiAlH₄ 还原 5 元环内酯 vs 内酰胺

| 反应物 | 产物 | 优先离域的孤对电子 |
|:---|:---|:---|
| 5 元环内酯（O 环内）| 二醇（开环）| 环外 O 的孤对电子（O 的 3 对）→ 推下离去基团 |
| 5 元环内酰胺（N 环内）| 环胺（不开环）| 环内 N 的孤对电子（N 比 O 电负性小）→ 形成亚胺正离子，再被还原 |

> 这是优先离域原则的经典验证案例！

### 4.5 α 效应

ABOC 行 1050 推论：

> 两对孤对电子相邻时（如 H₂O₂、H₂NOH、cis-N₂F₂），其中一对孤对电子无法找到反键轨道作受体（因为另一边是孤对电子），导致**亲核性大幅增强**——这就是 [[α效应]]。

但这种"被迫"的相邻孤对电子是**能量高、不稳定的**，导致这类分子选择**邻交叉构象**而非反式共平面构象。

### 4.6 超共轭与位阻效应的区分（Zchem 深化视角）

> 本节来源：Z-G 综合复习 §7.7

**传统说法**：烷基是给电子基团，通过超共轭稳定碳正离子。

**Zchem 的深层解释**（HOMO-LUMO 相互作用）：

```
超共轭稳定碳正离子：
σ(C-H) 成键轨道（HOMO）→ 空 p 轨道（LUMO）
    ↓
能量匹配良好，有效重叠 → 电子离域 → 碳正离子稳定

超共轭不能稳定碳负离子：
σ(C-H) 成键轨道（HOMO）→ 碳负离子的满 p 轨道（已填满，不是 LUMO）
    ↓
碳负离子的 LUMO 是 σ* 反键轨道，能量太高，与 σ(C-H) 匹配差
    ↓
无有效相互作用 → 烷基给电子反而 destabilize 碳负离子
```

**结论**：
- 超共轭的本质是 σ 键电子向**空轨道或半满轨道**的离域
- 碳正离子（空 p 轨道）✅ 可被超共轭稳定
- 自由基（半满 p 轨道）✅ 可被超共轭稳定
- 碳负离子（满 p 轨道）❌ 不能被超共轭稳定

**位阻效应与碳负离子稳定性**：
- 甲基负离子 > 叔丁基负离子（稳定性顺序）
- 原因：甲基负电荷暴露，可通过溶剂化或正离子静电作用稳定
- 叔丁基负离子位阻大，不易发生有效溶剂化作用

> **教学要点**：烷基"给电子"的现象在不同电荷体系中的**本质完全不同**——对碳正离子是超共轭（轨道匹配），对碳负离子则是位阻（溶剂化受阻）。这是学生最容易混淆的"同名异实"陷阱。

## 五、常见分类或情形

### 5.1 烷基对碳正离子的稳定化

经典例子：3°（叔）>  2°（仲）> 1°（伯）> CH₃⁺

**分析**：每多一个相邻 C-H σ 键，就多一组 H-C σ → C(+) p\* 超共轭，碳正离子越稳定。

注意：在 ABOC 框架下，**通过给体顺序 H-C > C-C 解释 Et₃C⁺ < Me₃C⁺ 的稳定性差异**——
- Me₃C⁺ 中相邻 9 个 C-H σ 键（强给体）
- Et₃C⁺ 中相邻部分 C-H 被 C-C 替换（弱给体）
- 因此 Et₃C⁺ 不如 Me₃C⁺ 稳定（ABOC 行 1115）

### 5.2 烷基对烯烃的稳定化

多取代烯烃比少取代烯烃稳定（Markovnikov 规则的反映）：
- 多取代烯烃：相邻 C-H 多，超共轭多
- 单/双取代烯烃：相邻 C-H 少，超共轭少

经典案例：自学练习 1.2.2-4

### 5.3 端基效应（n-σ\* 超共轭）

[[端基效应]] 是 n-σ\* 超共轭的经典应用：吡喃糖中 C1 上的取代基（如 OMe）优先位于直立键 a-位置，因为**环氧 O 的孤对电子可以反式共平面填入 C1-OMe σ\* 反键轨道**——这种超共轭只能在直立键时实现。

### 5.4 σ-σ\* 超共轭主导的反应

#### 5.4.1 E2 消除（反式共平面消除）

ABOC 行 3049：

> E2 消除：**反式共平面消除**（源于超共轭效应的极限考虑），如果消除时无法处于反式共平面，那么不能消除。

E2 反应实质：β-H 的 C-H σ 与离去基团 C-LG σ\* 在反式共平面时**重叠最大**，达到极限就是断键，得到 π 键。

#### 5.4.2 E2 选择性的"主要矛盾理论"

同顺-(2-甲基环己基)-N,N,N-三甲基季铵碱热消除：得到多取代 Hofmann 产物。

> 主要矛盾相同（环己烷构象翻转难度相同，碱拔氢位阻相同），次要矛盾（**过渡态甲基超共轭给电子稳定**）成为主要矛盾。

### 5.5 σ-π\* 超共轭

例子：羧酸 Z/E 构型（甲酸为例）
- **E 构型**：羟基 OH 的 H-O σ 键可以与羰基 C=O π\* 反式共平面 → σ-π\* 超共轭 → H 更缺电子 → 酸性更强
- **Z 构型**：羟基 O 的孤对电子（n）与羰基 C=O π\* 反式共平面 → n-π\* 共轭 → **离域更强**（孤对电子比成键电子是更好的给体）→ 更稳定

> 因此：**Z 构型更稳定，E 构型酸性更强**。

### 5.6 n-σ\* 超共轭主导的反应

#### 5.6.1 异头效应（[[端基效应]] 经典案例）

吡喃糖 C1 上 OMe 优先 a-位（直立键）：环氧 O 的孤对电子 → C1-OMe σ\*。

#### 5.6.2 内异头/外异头取舍

ABOC 行 1288：N,N,O-缩醛中**外异头优先**（N 孤对电子离域到 C-O σ\*），因为：
1. N 孤对电子能量比 O 孤对电子高（电负性 N < O）→ 优先离域
2. NMe₂ 1,3-双直立位阻大于 OMe → 直立位 NMe₂ 不利

### 5.7 n-π\* 共轭（"+M" 经典共轭）

**例子**：苯酚的 OH。
- O 孤对电子向苯环 π\* 离域 → "+M 共轭" → 苯环富电子 → 给电子取代基
- 同时 O 的孤对电子留下来一些被离去 → OH 酸性比 ROH 强

#### 5.7.1 接到双键上的 OMs/OTs

ABOC 行 1080：OMs 和 OTs 接到双键上时是**给电子基团**。
- 即使 O 的电子被共轭到硫那边
- 依旧可以给双键电子（O 电子总共 6 个，分给硫 1-2 个还剩 4-5 个）
- 同时双键对这个氧是吸电子的（O→C 共轭）

### 5.8 跨原子超共轭：β-硅基效应

ABOC 行 986：**Si-C-C-F 是极强的超共轭效应**。

**β-硅基效应**：当 C-Si 键与邻位空 p 轨道（碳正离子）反式共平面时，碳正离子稳定性可增加 10¹⁰ 倍。

应用：[[Sakurai烯丙基化]]、烯醇硅醚合成、TMS 保护策略中的稳定化效应。

## 六、适用条件与限制

### 6.1 推荐使用场景

- ✅ 解释烷基对碳正离子/自由基/烯烃的稳定化
- ✅ 解释构象稳定性（如乙烷全交叉、环己烷椅式优势）
- ✅ 解释立体选择性（如 E2 反式共平面消除）
- ✅ 解释酸碱性差异（如 Z/E 羧酸、HF 与 RF 的酸性）
- ✅ 解释官能团活化/钝化
- ✅ 跨章节工具：碳正离子、消除、亲电加成、构象、立体电子效应等

### 6.2 不推荐使用场景

- ❌ 离子键体系（电子定域，无超共轭）
- ❌ 反键轨道极高（如 C-Li σ\*）的给电子方向
- ❌ 没有反式共平面构象的体系
- ❌ 顺式或扭曲构象（重叠极差）
- ❌ 共轭效应（π-π\*）显著强于超共轭时，可忽略后者

### 6.3 与 [[共振论]] 的关系

- **共振论**：用极限式（resonance structures）描述电子离域 —— 是**定性表象**
- **超共轭**：用轨道重叠（HOMO-LUMO）描述电子离域 —— 是**底层机理**

二者**互补**：极限式画法形象，但只能定性比较；超共轭分析量化，但需要轨道想象。

## 七、常见比较与易混点

### 7.1 超共轭 vs 共轭 vs 诱导

| 效应 | 给体本质 | 受体本质 | 通过什么传递 | 距离敏感度 |
|:---|:---|:---|:---|:---|
| [[诱导效应]] | 极性化键的偶极 | 同上 | σ 键依次极化 | 强（指数衰减）|
| [[共轭效应]] | π 键 / 孤对电子 | π\* | π 体系 | 弱（不衰减）|
| **超共轭** | σ 键 / 孤对电子 | σ\* / π\* / p | σ-σ\* 重叠（反式共平面）| 中等 |

### 7.2 易混点对照

| 易混点 | 区别 |
|:---|:---|
| σ-σ\* 超共轭 vs σ-π\* 超共轭 | 受体不同（前者是 σ\*，后者是 π\*）|
| σ-π\* 超共轭 vs π-π\* 共轭 | 给体不同（前者是 σ，后者是 π）|
| **正向超共轭 vs 反向超共轭** | 给-受体身份**不可对调**！考察电负性才能判断哪一方向是有利的（重要陷阱！）|
| 顺式共平面 vs 反式共平面 | 顺式的"小波瓣相遇"导致重叠不利，必须用反式 |
| 同位素超共轭 | C-H > C-D > C-T 是因为 C-H σ 键能级稍低、波瓣稍大 |

### 7.3 与共振论的语言对照

| 传统共振论术语 | 给-受体语言 |
|:---|:---|
| 烷基 +I + 弱 +H（hyperconjugation 超共轭）| H-C σ → 邻位 σ\*/p\* |
| OMe 为 +M（mesomeric）| O 的 n（孤对电子）→ π\*/σ\* |
| C=O 为 -M（吸电子）| C=O π\* 接受 n 或 σ 给电子 |
| Cl 同时具有 -I 和 +M | C-Cl 的 σ 是受体（电负性论证），但 Cl 的孤对电子 n 也可作给体（共轭）|
| CF₃ 是 -I 和 -M | C-F σ\* 是受体（无给电子能力）|

## 十、例题

### 例题 1（ABOC 例 1.2.2，行 1044）

**题目：** 为什么两对孤对电子不希望自己处于相邻位置的反式共平面，从而组成给体-受体相互作用？

**分析：** 在给体-受体相互作用中，孤对电子是**最差的受体**（孤对电子能级很高）。

**解答：**
- 根据分子轨道理论，原子轨道组合为分子轨道时，**反键轨道上升的能量**大于成键轨道降低的能量
- 因此两对孤对电子相邻时（即 n + n 形成"双重占据"轨道）能量大幅上升
- 表现为：H₂O₂、H₂NOH 等分子选择**邻交叉构象**而非反式共平面构象
- 类似的，**cis-N₂F₂ 比 trans-N₂F₂ 稳定**（反直觉！）：因为反式构象下两对 N 上的孤对电子相邻，能量上升

**反思：** 这就是 [[α效应]] 的来源——孤对电子相邻虽然不稳定，但赋予体系超强的亲核性。

### 例题 2（ABOC 例 1.2.3，行 1064）

**题目：** 甲酸的 Z 构型和 E 构型，谁更稳定，谁酸性更强？

**分析：**
- E 构型：H-O σ 与 C=O π\* 反式共平面 → **σ-π\* 超共轭**
- Z 构型：羟基 O 的孤对电子（n）与 C=O π\* 反式共平面 → **n-π\* 共轭**

**解答：**
- **稳定性**：n-π\* 共轭 >> σ-π\* 超共轭（孤对电子是比成键电子更好的给体）→ **Z 构型更稳定**
- **酸性**：E 构型中 H 更缺电子 → 酸性更强 → **E 构型酸性更强**

**反思：** 稳定性和酸性是两个不同的问题，可能给出"反向"的答案。Z 稳定但 E 酸性强，**孤对电子稳定原物，但成键电子让 H 更易离去**。

### 例题 3（ABOC 自学练习 1.2.2-2 A，行 1030）

**题目：** 解释顺式 N₂F₂ 比反式在热力学上更加稳定（反直觉案例）。

**分析：**
- 反式 N₂F₂：两个 N 上的孤对电子反式共平面 → **n + n 重叠**（两对孤对电子相邻）→ 能量上升
- 顺式 N₂F₂：两个 F 在同一侧，两对 N 孤对电子分别远离 → 不形成 n + n 反键

**解答：**

| 构象 | N-F σ → C-F\*（外向）| N 孤对 → 空轨道（外向）| 总效应 |
|:---|:---|:---|:---|
| trans-N₂F₂ | 弱 | 反式共平面 N 孤对 + 邻 N 孤对 → **不利** | 高能 |
| cis-N₂F₂ | 弱 | 孤对电子各自取向不同 → **无 α 效应惩罚** | 较低能 |

因此 **cis-N₂F₂ 更稳定**。

**反思：** 这是 ABOC 用"超共轭+α效应"统一解释顺反式稳定性的经典反直觉案例，是理解给-受体的重要路标。

### 例题 4（自学练习 1.2.2-4，行 1074）

**题目：** 利用超共轭效应解释为什么多取代的双键比少取代的双键热力学上更加稳定。

**分析：** 烷基中的 C-H σ 键可以与邻位 C=C π\* 反式共平面 → σ-π\* 超共轭。

**解答：**
- 单取代烯烃（如 1-丁烯）：只有 1 端有烷基 → 只有 ~3 组 σ-π\* 超共轭
- 双取代烯烃（如 2-丁烯）：两端有烷基 → 6 组以上 σ-π\* 超共轭
- 三/四取代烯烃：超共轭组数更多

**结论：** 取代基越多 → 超共轭越多 → 热力学越稳定。

**反思：** 这就是 Saytzeff 规则（消除主要得到取代度高的烯烃）的电子学基础。

### 例题 5（T11，COBP 模拟题，🔥 高难综合）

**题目：** 双 TMS（三甲基硅基）取代的联苯较双叔丁基取代的联苯具有更大的旋转能垒，而 TMS 处于直立键的环己烷却比叔丁基处于直立键的环己烷具有更小的直立键 1,3-张力。两种结果反映出的位阻顺序看似自相矛盾，请解释。

**键长数据**：dC-Si = 186 pm，dC-C = 154 pm。

**分析：** 这道题考察"键长 vs 位阻"的双重作用。

**解答：**
- 联苯旋转能垒：旋转过渡态时两个邻位取代基相互"撞击"。**TMS 中 C-Si 键长（186 pm）比 C-C 键长（154 pm）长**，但 Si 上的 3 个甲基比 C-叔丁基中的 3 个甲基**距离更近**（因为 C-Si 长了 32 pm，但是 TMS 的 3 个 H/Me 都被推出去——这里是因为 C-Si 键长，Si 的"伞"撑得更开）。**结果**：TMS 的"伞"实际上比叔丁基的"伞"更大，旋转能垒更高。
- 环己烷直立键 1,3-张力：直立键上的取代基与邻位双直立 C-H 相互"撞击"。**TMS 中的 Si 离邻位 C-H 距离更远**（因为 Si 比 C 大 32 pm 在键长上），所以 1,3-张力**减小**。

**反思：** 关键在于**位阻效应取决于"邻位距离"，而不是单纯的键长**。同一个 TMS 在不同情境下表现"位阻大"或"位阻小"，本质上是几何关系的差异。本题虽然不直接是超共轭题，但体现了 ABOC "矛盾理论" 的训练价值。

## 十一、易错点

### 11.1 忘记反式共平面要求

**错误**：认为只要有 C-H σ 和 C-X σ\* 就有超共轭。

**正确**：必须**反式共平面**！如果分子限制了构象（如桥环、刚性环），可能没有超共轭。

### 11.2 反向超共轭的陷阱

**错误**：把 H-C-C-F 写成 F-C 给体 → C-H 受体。

**正确**：必须根据**电负性**判断哪一方向是有利的。F 电负性大 → C-F σ\* 是好受体；H 电负性小 → H-C σ 是好给体。**正向是 H-C → C-F σ\***。

### 11.3 把超共轭与诱导效应混淆

**错误**：用"诱导吸电子"解释烷基稳定碳正离子。

**正确**：烷基的 +I 是非常弱的（甚至有人认为可忽略），主要是**超共轭**给电子稳定碳正离子。

### 11.4 忘记 α 效应中孤对电子相邻的不稳定

**错误**：H₂O₂ 应该选反式共平面以最大化超共轭。

**正确**：两对孤对电子相邻反式共平面 → 反键填充能量上升。H₂O₂ 选**邻交叉构象**避免这一不利情况。

### 11.5 忘记优先离域原则

**错误**：用"O 的孤对电子优先离域"解释所有内酯/内酰胺反应。

**正确**：电负性越小，孤对电子能量越高，优先离域。例如**内酰胺中 N 的孤对电子优先离域**（N 比 O 电负性小），所以 LiAlH₄ 还原内酰胺得到环胺（N 孤对推下去 O 离去基团）而非二醇。

### 11.6 把超共轭与共轭混为一谈

**错误**：σ-σ\* 超共轭和 π-π\* 共轭是两种不同的现象，需要分开讨论。

**正确**：在 ABOC 框架下，二者是 [[给体-受体相互作用]] 的不同亚类，本质相同。但在传统术语中"共轭"特指 π → π\*，"超共轭"特指 σ → 任意。

### 11.7 同位素效应方向搞反

**错误**：C-D 比 C-H 是更好的超共轭给体。

**正确**：C-H > C-D > C-T。原因：C-H σ 键能级稍低（H 更轻、零点能更高），重叠更大。

| "H 在 H₂O 上写 σ-π\* 共轭"| 把 OH 当作给体 | OH 中孤对电子（n）才是给体，H-O σ 弱 | 区分 n vs σ；优先看孤对电子 |

### 12.3 入门级例题（不同于第十节）

**题目**：用超共轭效应解释为什么 (CH₃)₃C⁺ 比 (CH₃)₂CH⁺ 更稳定。

**预期解答路径**：
1. 第一步看到题目应该想到：**这是碳正离子稳定性比较**。
2. 然后联想到本 KP 的：**烷基越多 → 邻位 C-H 越多 → 超共轭越多 → 越稳定**。
3. 应用得到：
   - (CH₃)₃C⁺：3 个 CH₃，每个有 3 个 C-H σ → 共 9 组 H-C → C(+) p\* 超共轭
   - (CH₃)₂CH⁺：2 个 CH₃ + 1 个 H，共 6 个 C-H σ → 6 组超共轭
   - 9 > 6 → (CH₃)₃C⁺ 更稳定。

**老师可在课堂上引导的提问**：
- 学生若卡在第 1 步：问 "你看到 C(+) 想到什么？什么因素决定它的稳定性？"
- 学生若卡在第 2 步：问 "什么是超共轭？需要什么几何要求？"
- 学生若答错（认为是诱导效应）：问 "你认为 -I 和 +I 怎么比较？烷基真的有那么强 +I 吗？"
- 学生若混淆共轭与超共轭：让学生写出共振式（极限式），看看是否有 π 键参与
- 学生若忘记反式共平面：问 "C-H 和 C(+) 的 p 空轨道之间需要什么角度？"

### 12.4 与现实/直觉的连接

> 用学生熟悉的现象/比喻 anchor 这个抽象概念。

- **类比 1**：超共轭就像"邻居家的 WiFi"——你住进新家（碳正离子），周围邻居家（C-H σ 键）越多，你"借" WiFi 的机会越多，越能"上网"（稳定化）。
- **类比 2**：反式共平面就像"投篮"——你必须正对篮筐（反键轨道最大波瓣方向）才能投中，从侧面或反方向都不行。
- **反例/佯谬**：表面上看 σ 键很"稳定、不动"，但实际上**它是有机化学中第二大重要电子离域来源**，仅次于 π 键。

## 十三、竞赛拓展

### 13.1 NBO（自然键轨道）分析

NBO（Natural Bond Orbital）方法是定量化超共轭强度的标准方法：

| 量 | 含义 | 工具 |
|:---|:---|:---|
| $\sigma \to \sigma^*$ 离域能 | 一组超共轭的稳定化能（kcal/mol）| Gaussian/ORCA + NBO 程序 |
| 二阶微扰能 $E^{(2)}$ | 各种给-受体相互作用的能量贡献 | NBO 6.0+ |
| HOMO-LUMO gap | 反键能级与成键能级之差 | DFT 计算 |

ABOC 行 1148 提到：**亚胺正离子比甲醛正离子稳定 ~10¹⁰ 倍**——这种量化估计就来自 NBO 分析。

### 13.2 物理有机化学定量公式

**Hammett 方程**：
$$\log \frac{k}{k_0} = \rho \sigma$$

其中 $\sigma$ 包含 $\sigma_R$（共振项）、$\sigma_I$（诱导项），$\sigma_R$ 部分实际上包含了超共轭效应的贡献。

**Taft 方程**：分离出 $\sigma^\*$（电场效应）、$E_s$（位阻效应）、$\sigma_R$（共轭/超共轭）三项，更细致地划分各种效应。
