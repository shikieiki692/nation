# 逆合成分析

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[50-有机合成]]

## 一、定义

**逆合成分析**是由 E.J. Corey 于 1967 年系统提出的有机合成设计方法，核心思想是**从目标分子（Target Molecule, TM）出发，逆向推导合成路径**。通过一系列合理的"断键"（disconnection）操作，将复杂分子逐步简化为可购买的简单原料。

- **核心操作**：
  1. **断键（Disconnection）**：在 C-C 或 C-X 键处虚拟切断，将 TM 分为更简单的片段
  2. **官能团互换（FGI, Functional Group Interconversion）**：将目标官能团转换为更易引入或更稳定的等价形式
  3. **官能团添加（FGA, Functional Group Addition）**：添加必要的官能团以辅助断键
- **逆合成箭头**：⇒（空心箭头，读作"可以由……合成得到"）
- **合成子（Synthon）**：断键后得到的理论片段（可能是碳正离子、碳负离子、自由基等）
- **合成等价物（Synthetic Equivalent）**：合成子的实际试剂形式

## 三、核心原理

### 3.1 断键的基本原则

**好的断键应满足**：

| 原则 | 说明 |
|:---|:---|
| **反应可行性** | 断键对应的正向反应必须可靠、有文献支持 |
| **简化作用** | 断键后分子复杂度显著降低 |
| **官能团相容性** | 正向反应中其他官能团不受影响 |
| **立体化学可控** | 若涉及手性，正向反应应能控制立体化学 |
| **步骤经济性** | 避免过多保护-去保护步骤 |

**断键的优先顺序**：
1. **战略键（Strategic Bond）**：连接分子主要骨架的键
2. **C-X 键**：通常比 C-C 键更容易形成
3. **支点附近**：切断支链与主链的连接
4. **对称性利用**：利用对称性减少合成步骤

### 3.2 常见断键反应与合成子

| 反应类型 | 断键位置 | 合成子类型 | 合成等价物 |
|:---|:---|:---|:---|
| **Grignard/有机锂 + 羰基** | C-C(OH) | 亲核碳 + 亲电羰基 | RMgX/RLi + 醛/酮 |
| **Wittig 反应** | C=C | 磷叶立德 + 羰基 | Ph₃P=CHR + R'CHO |
| **羟醛缩合（Aldol）** | C-C(OH)-C=O | 烯醇盐 + 醛/酮 | LDA/醛 |
| **Claisen 缩合** | C-C(O)-C=O | 烯醇盐 + 酯 | NaOEt/酯 |
| **Michael 加成** | C-C-C=O | 烯醇盐 + α,β-不饱和羰基 | 铜锂试剂/烯酮 |
| **Diels-Alder** | 六元环中的两个键 | 二烯 + 亲双烯体 | 各种二烯/亲双烯体 |
| **亲核取代（SN2）** | C-C(Nu) | 亲核体 + 卤代烃 | Nu⁻ + R-X |
| **傅-克酰基化** | Ar-C=O | 芳环 + 酰氯 | ArH + RCOCl/AlCl₃ |
| **炔烃偶联** | C≡C-C | 炔负离子 + 亲电体 | R-C≡C⁻ + R'-X |

### 3.3 官能团互换（FGI）策略

**常见 FGI**：

| 目标官能团 | 前体官能团 | 转化方法 |
|:---|:---|:---|
| **醇（OH）** | 酮/醛 | NaBH₄, LiAlH₄ 还原 |
| **酮（C=O）** | 醇 | PCC, Swern, DMP 氧化 |
| **羧酸（COOH）** | 酯/腈 | 水解 |
| **胺（NH₂）** | 酰胺/腈 | LiAlH₄ 还原；Gabriel 合成 |
| **烯烃（C=C）** | 醇 | 脱水；Wittig 反应 |
| **卤代烃（C-X）** | 醇 | SOCl₂, PBr₃, Appel 反应 |
| **环氧** | 烯烃 | mCPBA 环氧化 |

## 四、关键结论

1. **逆合成分析是有机合成的"战略思维"**：好的逆合成路线比单纯的反应知识更重要
2. **断键选择决定合成效率**：同一分子可能有多种逆合成路线，需要评估比较
3. **对称性是最有力的简化工具**：对称分子通过对称断键可大幅缩短路线
4. **保护基策略是逆合成的一部分**：在逆合成阶段就应考虑何时保护、何时脱除
5. **Corey 的逆合成分析是 1990 年诺贝尔化学奖的基础**：是现代有机合成的核心方法论

## 五、常见分类

### 5.1 按断键类型分类

```
逆合成分析
├── C-C 键断键
│   ├── 亲核加成类（Grignard, Aldol, Wittig）
│   ├── 亲电芳香取代（Friedel-Crafts）
│   ├── 周环反应（Diels-Alder, [3,3]-σ 迁移）
│   └── 偶联反应（炔烃偶联, Ullmann）
├── C-X 键断键
│   ├── 亲核取代（SN2, SNAr）
│   ├── 酰基取代（酯化, 酰胺化）
│   └── 自由基取代
├── 环化断键
│   ├── 分子内 Aldol
│   ├── 分子内 SN2
│   └── 环加成（Diels-Alder 逆反应）
└── 重排反应的逆分析
    └── 某些重排可用于构建特定骨架
```

### 5.2 经典逆合成策略

| 策略 | 适用场景 | 说明 |
|:---|:---|:---|
| **基于官能团的策略** | 含羰基、羟基等官能团的分子 | 从官能团出发寻找合适的构建反应 |
| **基于转化的策略** | 已知某步转化特别有效 | 如 Diels-Alder 构建六元环 |
| **基于骨架的策略** | 复杂多环分子 | 识别核心骨架，从核心向外扩展 |
| **基于立体化学的策略** | 含多个手性中心的分子 | 选择能控制立体化学的关键步骤 |
| **收敛式合成** | 多片段分子 | 分别合成片段，最后拼接 |
| **线性合成** | 逐步构建的分子 | 一步一步延伸碳链 |

## 六、适用条件

### 6.1 逆合成分析的一般流程

| 步骤 | 操作 | 目的 |
|:---|:---|:---|
| 1 | **识别目标分子的官能团和骨架** | 了解分子结构特征 |
| 2 | **寻找战略键** | 找到最佳断键位置 |
| 3 | **尝试断键** | 用逆合成箭头表示 |
| 4 | **评估可行性** | 正向反应是否可靠 |
| 5 | **必要时进行 FGI/FGA** | 转换或添加官能团以辅助断键 |
| 6 | **重复 2-5** | 直到得到简单起始原料 |
| 7 | **写出正向合成路线** | 从原料到产物的完整路径 |
| 8 | **评估路线** | 步骤数、产率、选择性、经济性 |

### 6.2 收敛式 vs 线性合成

| 维度 | 线性合成（Linear）| 收敛式合成（Convergent）|
|:---|:---|:---|
| **路线形状** | A → B → C → D → TM | A → C; B → D; C + D → TM |
| **总步骤数** | 相同或更多 | 相同或更少 |
| **实际效率** | 低（每步产率累积衰减）| 高（片段分别合成后再拼接）|
| **中间体复杂度** | 逐步增加 | 前期并行处理简单片段 |
| **适用** | 简单分子 | 复杂分子、多片段分子 |
| **例子** | 逐步延长碳链 | 分别合成两个半分子，最后偶联 |

**产率计算示例**：
- 线性 9 步，每步 90% 产率：总产率 = 0.9⁹ ≈ **39%**
- 收敛式：两条线各 4 步（每步 90%）→ 中间体产率各约 66%，偶联 90% → 总产率 ≈ **59%**

## 七、常见比较

### 7.1 逆合成 vs 正向合成

| 维度 | 逆合成分析 | 正向合成 |
|:---|:---|:---|
| **方向** | 从目标分子 → 原料 | 从原料 → 目标分子 |
| **思维** | 逆向断键 | 正向成键/转化 |
| **目的** | 设计路线 | 执行路线 |
| **符号** | ⇒（空心箭头）| →（正常箭头）|
| **核心操作** | 断键、FGI | 反应、分离、纯化 |
| **评价标准** | 路线简洁性、可行性 | 实际产率、纯度 |

### 7.2 不同断键策略的比较

以 4-苯基-2-丁酮为例：

**策略 A：Aldol 断键**
```
Ph-CH₂-CH₂-CO-CH₃
        ⇒
Ph-CH₂-CHO + CH₃-CO-CH₃（交叉 Aldol）
```

**策略 B：Grignard 断键**
```
Ph-CH₂-CH₂-CO-CH₃
        ⇒
Ph-CH₂-CH₂-COCl + CH₃MgBr
或：Ph-CH₂-CH₂-MgBr + CH₃COCl
```

**评估**：策略 A 可能面临自身 Aldol 的竞争；策略 B 更清晰可控。

## 十、例题

### 例题 1：简单酮的逆合成分析

**题目**：对 4-苯基-2-丁酮（PhCH₂CH₂COCH₃）进行逆合成分析。

**解析**：

**方案 A：Aldol 断键（在 α-β 键处切断）**
```
Ph-CH₂-CH₂-CO-CH₃
         ⇒（Aldol 断键）
Ph-CH₂-CHO + CH₃-CO-CH₃
```
- 正向：苯乙醛 + 丙酮，碱催化 Aldol → 脱水 → 4-苯基-3-丁烯-2-酮 → 氢化
- 问题：自身 Aldol 竞争（两分子苯乙醛或两分子丙酮也可 Aldol）

**方案 B：酰基化断键（在羰基 α 位切断）**
```
Ph-CH₂-CH₂-CO-CH₃
         ⇒
Ph-CH₂-CH₂-COCl + (CH₃)₂CuLi
或：Ph-CH₂-CH₂-MgBr + CH₃COCl
```
- 正向：酰氯 + 铜锂试剂（1,4-加成不适用，这里用酰氯 + 有机金属得酮）
- 更清晰，避免交叉反应问题

### 例题 2：利用对称性简化

**题目**：设计 2,6-二甲基-2,6-辛二烯的逆合成路线。

**解析**：
- 分子有**对称性**：两个相同的 2-甲基-2-丁烯片段通过中间 C-C 键连接
- **对称断键**：
```
     CH₃          CH₃
      |            |
CH₃-C=CH-CH₂-CH₂-CH=C-CH₃
                ⇒
2 × CH₃-C(=CH₂)-CH₂-Br（或等价物）
```
- 正向：两分子通过偶联反应（如 Wurtz 型偶联或金属有机偶联）连接

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| "逆合成就是写反应式倒过来" | 理解肤浅 | 逆合成是**断键分析**，不是简单倒写反应式；需考虑合成子和等价物 | 逆合成箭头和正常反应箭头有何不同含义？ |
| "只选最短路线" | 忽略可行性和产率 | 最佳路线需综合考虑**步骤数、产率、选择性、原料可得性** | 两步 50% 产率的路线和四步 80% 产率的路线哪个更好？ |
| "所有断键都可行" | 忽略反应限制 | 断键必须有**可靠的正向反应**支持 | 某酮在某个 C-C 键处切断，但无合适反应构建该键，怎么办？ |
| "忽略保护基策略" | 线性思维 | 逆合成阶段就应**预判**哪些官能团需要保护 | 含羟基和醛基的分子做 Grignard 反应前需要做什么？ |
| "忽略立体化学" | 只关注骨架 | 手性分子的逆合成必须考虑**立体化学控制** | 手性中心的构建应在路线早期还是晚期？ |

### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "逆合成分析是'纸上谈兵'" | 认为不实用 | 强调：所有复杂分子的全合成**都必须**先做逆合成分析；竞赛合成题直接考查 |

### 12.3 入门例题

**最简逆合成练习**：
> 对 1-苯基-1-丁醇（PhCH(OH)CH₂CH₂CH₃）进行逆合成分析，给出至少两条路线。

**答案**：
**路线 A（Grignard）**：
PhCH(OH)CH₂CH₂CH₃ ⇒ PhCHO + CH₃CH₂CH₂MgBr（丙基格氏试剂）

**路线 B（Aldol + 还原）**：
PhCH(OH)CH₂CH₂CH₃ ⇒ PhCH=CHCH₂CH₃（脱水）⇒ PhCHO + CH₃CH₂CHO（交叉 Aldol）

**评估**：路线 A 更直接，一步完成；路线 B 需先 Aldol 再脱水再氢化，步骤多。
**引导**：问学生"如果目标分子是 PhCH(OH)CH₂CH₃ 呢？"（乙基格氏更简单）

### 12.4 现实类比

**"拆乐高"类比逆合成分析**：
- 目标分子像一座**复杂的乐高城堡**
- **逆合成分析** = **拆乐高说明书**——从成品倒推，看每个模块是怎么拼起来的
- **断键** = 找到积木之间的**拼接缝**——从缝隙处拆开
- **合成子** = **理论上的积木块**——可能是"带凸点的方块"或"带凹槽的长条"
- **合成等价物** = **实际买到的积木包**——理论和实际的桥梁
- **FGI** = "这个蓝色方块其实是用红色方块喷漆得到的"——先找到更容易买到的红色方块
- **收敛式合成** = **分工组装**——两个人分别拼城堡的左半和右半，最后合体；比一个人从头拼到尾快得多
- **好的合成化学家** = **优秀的乐高设计师**——不仅知道怎么拆，还知道哪些积木容易买到、哪些拼接最牢固
