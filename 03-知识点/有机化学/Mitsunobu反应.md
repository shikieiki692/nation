# Mitsunobu 反应（Mitsunobu Reaction）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[36-取代反应]]、[[44-脂肪族亲核取代反应]]

---

## 一、定义

**Mitsunobu 反应**：醇（ROH）在 **三苯基膦（PPh₃）** 和 **偶氮二甲酸二酯（DIAD 或 DEAD）** 存在下，与亲核试剂反应，发生 **构型翻转（inversion）**，生成取代产物的反应。

**通式**：
$$
\ce{R-OH + H-Nu + PPh3 + R'OC(=N)N=NC(=O)OR' -> R-Nu + Ph3P=O + R'OC(=N)NHC(=O)OR'}
$$

**本质**：醇的 **SN2 型取代**——PPh₃ 将 OH 转化为优秀的离去基（氧鎓中间体），亲核试剂从背面进攻，导致 **Walden 翻转**。

## 三、核心原理

### 3.1 试剂与作用

| 试剂 | 作用 | 为什么需要它 |
|:---|:---|:---|
| **PPh₃** | 亲核催化剂 | 进攻 DIAD → 形成活性中间体；最终被氧化为 **Ph₃P=O**（驱动力）|
| **DIAD / DEAD** | 偶氮试剂 | 接受 PPh₃ 的亲核进攻，释放活性物种 |
| **H-Nu** | 亲核试剂（需酸性）| pKa < 15 的酸：羧酸、酚、酰亚胺等 |
| **ROH** | 底物 | 1° 和 2° 醇均可；3° 醇通常不行 |

### 3.2 反应机理

**四步机理**：
```
步骤 1：PPh₃ 进攻 DIAD → 形成两性离子中间体（betaine）
步骤 2：醇的 OH 被活化 → 氧鎓中间体（RO-⁺PPh₃）
步骤 3：亲核试剂 Nu⁻ 从背面进攻 → SN2 型取代
步骤 4：构型翻转产物 + Ph₃P=O + 肼副产物
```

**关键特征**：
- **立体专一性**：2° 醇 → **完全构型翻转**（Walden inversion）
- **亲核试剂要求**：pKa < 15（酸性足够强，才能形成稳定的共轭碱去进攻）
- **P=O 键是引擎**：Ph₃P=O 的键能极高（~540 kJ/mol），是整个反应的不可逆驱动力

### 3.3 亲核试剂范围

**可用（pKa < 15）**：
- 羧酸（RCOOH）→ 酯
- 酚（ArOH）→ 芳基醚
- 酰亚胺（phthalimide）→ 胺（Gabriel 变体）
- 叠氮酸（HN₃）→ 叠氮化物

**不可用（pKa > 15）**：
- 醇本身（pKa ~16-18）
- 胺（pKa ~35）
- 普通羧酸酯（pKa ~25）

### 3.4 与 SN2 的对比

| | **Mitsunobu** | **SN2** |
|:---|:---|:---|
| **底物** | 醇（OH 被活化）| 卤代烷或磺酸酯 |
| **离去基** | Ph₃P=O（原位生成）| 卤素或磺酸根 |
| **立体化学** | **翻转** | **翻转** |
| **条件** | 中性、温和 | 需要碱性亲核试剂 |
| **官能团耐受性** | **极好**（中性条件）| 可能受酸/碱影响 |

## 四、关键结论

1. **Mitsunobu = 醇的 SN2 型翻转**——将不良离去基 OH 转化为优良离去基
2. **P=O 键形成是驱动力**——Ph₃P=O 的高键能让反应不可逆
3. **亲核试剂必须 pKa < 15**——这是判断能否 Mitsunobu 的关键
4. **1° 和 2° 醇可行，3° 通常不行**——3° 醇的空间位阻阻止背面进攻
5. **中性条件 → 官能团耐受性好**——是复杂分子后期修饰的利器

## 五、常见分类或情形

### 5.1 按产物分类

| 亲核试剂 | 产物 | 应用 |
|:---|:---|:---|
| **RCOOH** | 酯（R-OCOR'）| 醇的酰化 |
| **Phthalimide** | N-烷基邻苯二甲酰亚胺 | Gabriel 合成胺的前体 |
| **HN₃** | 叠氮（R-N₃）| 还原为胺（Staudinger 后处理）|
| **PhOH** | 芳基醚（R-OAr）| 酚醚合成 |

### 5.2 改良版本

| 改良 | 改动 | 目的 |
|:---|:---|:---|
| **Mitsunobu-I** | 用 DEAD 代替 DIAD | DEAD 更常见，但 DIAD 更易处理 |
| **Bis-Mitsunobu** | 双重活化 | 合成环状化合物 |
| **aza-Mitsunobu** | 氮亲核试剂 | 合成胺类 |

## 六、适用条件与限制

1. **醇必须是 1° 或 2°**——3° 醇空间位阻太大
2. **亲核试剂 pKa < 15**——酸性太弱则无法去质子化
3. **严格无水**——水会与 PPh₃/DIAD 反应竞争
4. **需惰性气氛**（有时）——防止 PPh₃ 被空气氧化
5. **Ph₃P=O 副产物难除去**——大量反应时后处理困难
6. **原子经济性差**——副产物多，不适合工业大规模

## 七、常见比较与易混点

### 7.1 Mitsunobu vs Appel vs SOCl₂

| | **Mitsunobu** | **Appel** | **SOCl₂** |
|:---|:---|:---|:---|
| **产物** | 醚/酯/胺等（多样）| 卤代烷 | 卤代烷 |
| **立体化学** | **翻转** | **翻转** | **翻转** |
| **条件** | 中性 | 中性 | 碱性（需加碱）|
| **适用** | 复杂分子后期修饰 | 卤代 | 卤代 |

### 7.2 Mitsunobu 与 Gabriel 合成的关系

- Mitsunobu + 邻苯二甲酰亚胺 → N-烷基邻苯二甲酰亚胺
- 肼解 → 伯胺
- 这是 **Gabriel-Fukuyama-Mitsunobu 合成胺** 的组合策略

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：(S)-2-丁醇经 Mitsunobu 反应（PPh₃/DIAD/苯甲酸）后的产物构型是什么？

**解答**：
- Mitsunobu 是 **SN2 型翻转**
- (S) → **(R)**-2-丁醇苯甲酸酯

### 10.2 例题 2（★★ 试剂选择）

**题目**：下列哪些亲核试剂可以用于 Mitsunobu 反应？
A. CH₃COOH  B. CH₃OH  C. PhNH₂  D. HN₃

**解答**：
- pKa < 15 的才能用
- A（pKa ~4.8）✓、D（pKa ~4.7）✓
- B（pKa ~15.5）✗、C（pKa ~30）✗

### 10.3 例题 3（★★★ 合成设计）

**题目**：设计从 (R)-1-苯乙醇到 (S)-1-苯乙胺的路线。

**解答**：
1. (R)-1-苯乙醇 + PPh₃/DIAD/HN₃ → **(S)-1-苯乙基叠氮**（Mitsunobu，翻转）
2. (S)-叠氮 + PPh₃/H₂O → **(S)-1-苯乙胺**（Staudinger 还原）

## 十一、易错点

1. ❌ **3° 醇也能 Mitsunobu** → 3° 醇 **通常不行**（位阻太大）
2. ❌ **胺可以作为亲核试剂** → 胺 pKa 太高（~35），**不行**
3. ❌ **构型保持** → Mitsunobu 是 **翻转**（inversion）
4. ❌ **忽略 P=O 驱动力** → Ph₃P=O 的高键能是反应不可逆的关键
5. ❌ **认为 Mitsunobu 只是取代** → 它是**醇的活化 + SN2 翻转**的组合

### 12.3 入门例题

**题目**：为什么 Mitsunobu 反应需要 PPh₃ 和 DIAD 两者？只用其中一个可以吗？

**引导**：
- PPh₃ 单独：无法活化醇
- DIAD 单独：无法提供亲核磷
- 两者协同：PPh₃ 亲核进攻 DIAD → 生成活性物种 → 活化醇 → SN2 翻转

### 12.4 现实类比

**Mitsunobu 像"快递公司"**：
- PPh₃ = 快递员（把包裹送到目的地）
- DIAD = 运单系统（提供能量/信息）
- 醇 = 寄件人（提供货物）
- 亲核试剂 = 收件人（接收货物）
- Ph₃P=O = 快递员的报酬（高能量释放，让整个过程不可逆）
