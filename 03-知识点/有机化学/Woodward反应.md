# Woodward 反应

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[37-加成反应]]、[[43-亲电加成反应]]

## 一、定义

**Woodward 反应**：烯烃在 **I₂ / AgOAc / H₂O** 条件下发生的**顺式双羟化**反应，产物为顺式邻二醇（syn-diol）。

**本质**：利用碘鎓离子中间体，通过水的捕获实现顺式双羟化——与 Prevost 反应（无水，反式）形成对照。

## 二、考纲对应

- 对应考纲条目：[[37-加成反应]]、[[43-亲电加成反应]]
- 所属模块：[[基础要求-有机化学]]
- 本知识点在考纲中的作用：Woodward 反应是双羟化反应的重要方法之一，与 Prevost 反应共同构成碘鎓离子介导的双羟化体系。

## 三、核心原理

### 3.1 反应机理

**第一步：碘鎓离子形成**
- 烯烃 + I₂ → **碘鎓离子**（三元环中间体）

**第二步：水进攻**
- H₂O（或 AcO⁻）进攻碘鎓离子
- 形成碘代醇/碘代醋酸酯

**第三步：关键差异——水使中间体不可逆**
- 有水时：H₂O 进攻 → 形成 C-OH 键（**不可逆**）
- 后续步骤围绕已固定的构型进行

**第四步：第二次取代**
- 第二个亲核试剂（AcO⁻ 或 H₂O）取代 I
- 最终水解得 **顺式邻二醇**

### 3.2 Woodward vs Prevost 的核心差异

**ABOC 平衡思维解释**：
- 无水时：碘鎓离子开环**可逆** → 系统通过两次背面进攻达到最稳定状态 → **反式**（Prevost）
- 有水时：H₂O 捕获碳正离子特征中间体 → 第一次开环后**不可逆** → 后续围绕固定构型 → **顺式**（Woodward）

> "用平衡思维解释：水使中间体不可逆，微量即可改变选择性。"（ABOC §3.2）

## 四、关键结论

1. **Woodward 反应 = 顺式双羟化（syn）**
2. **Prevost 反应 = 反式双羟化（anti）**
3. **两者都经历碘鎓离子中间体**
4. **水的有无决定立体化学**——微量条件变化即可翻转结果
5. **这是 ABOC "平衡思维"的经典案例**

## 五、常见分类或情形

### 5.1 双羟化方法对比

| 方法 | 试剂 | 立体化学 | 中间体 |
|:---|:---|:---:|:---|
| **OsO₄/NMO** | 催化 OsO₄，NMO 共氧化 | **syn** | 锇酸酯 |
| **稀冷 KMnO₄** | 0°C，稀溶液 | **syn** | 高锰酸酯 |
| **Woodward** | I₂ / AgOAc / **H₂O** | **syn** | 碘鎓离子（被水捕获）|
| **Prevost** | I₂ / AgOAc，**无水** | **anti** | 碘鎓离子（可逆开环）|
| **环氧开环** | mCPBA → 酸/碱水解 | **trans** | 环氧 |

### 5.2 应用场景

- **需要 syn-二醇** → Woodward 或 OsO₄
- **需要 anti-二醇** → Prevost
- **需要区分 syn/anti** → 通过条件控制（水的有无）

## 六、适用条件与限制

1. **需要严格控水**——Prevost 要求无水，Woodward 需要水
2. **AgOAc 的作用**：提供 AcO⁻ 亲核试剂，同时 Ag⁺ 协助 I⁻ 离去
3. **碘鎓离子开环的位置**：遵循碳正离子稳定性规则
4. **与 OsO₄ 相比**：Woodward 试剂更便宜，但立体选择性略低
5. **后续水解步骤**：二醋酸酯需水解才能得到游离二醇

## 七、常见比较与易混点

### 7.1 Woodward vs Prevost

| | **Woodward** | **Prevost** |
|:---|:---|:---|
| 试剂 | I₂ / AgOAc / **H₂O** | I₂ / AgOAc，**无水** |
| 立体化学 | **syn** | **anti** |
| 中间体可逆性 | 不可逆（水捕获）| 可逆 |
| 产物 | 顺式二醇 | 反式二醇 |
| 记忆口诀 | "W 像水波，顺式有波" | "P 字倒过来，反式没水算" |

### 7.2 Woodward vs OsO₄

| | **Woodward** | **OsO₄/NMO** |
|:---|:---|:---|
| 试剂成本 | 较低 | 较高（Os 昂贵且剧毒）|
| 立体化学 | syn | syn |
| 中间体 | 碘鎓离子 | 锇酸酯 |
| 操作安全性 | 较高 | 较低（OsO₄ 剧毒）|
| 选择性 | 中等 | 高 |

## 十、例题

### 例题 1（★ 基础）

**题目**：环己烯用 Woodward 试剂（I₂/AgOAc/H₂O）处理，产物是什么？

**解答**：
- Woodward 反应 = 顺式双羟化
- 产物：**顺-1,2-环己二醇**（两个 OH 都在环的同侧）

### 例题 2（★★ 对比）

**题目**：如何从反-2-丁烯出发制备 meso-2,3-丁二醇？

**解答**：
- meso-2,3-丁二醇 = 反式加成产物（两个 OH 在异侧）
- 反-2-丁烯 → 要得 meso，需要 **syn 加成**
- 用 **Woodward**（I₂/AgOAc/H₂O）或 **OsO₄/NMO**

## 十一、易错点

1. ❌ **混淆 Woodward 和 Prevost 的条件** → Woodward 有水 = syn；Prevost 无水 = anti
2. ❌ **认为 Woodward 是反式** → Woodward 是 **syn**
3. ❌ **忽略水的关键作用** → 水的存在使中间体不可逆，从而改变立体化学
4. ❌ **与环氧开环混淆** → 环氧开环得 trans-二醇，Woodward 得 syn-二醇

### 12.3 入门级例题

**题目**：解释为什么 Woodward 反应得顺式二醇，而 Prevost 反应得反式二醇。

**解答**：
1. 两者都经历碘鎓离子中间体
2. Woodward（有水）：H₂O 进攻 → C-OH 键不可逆 → 后续围绕固定构型 → **syn**
3. Prevost（无水）：AcO⁻ 进攻 → 可逆 → 两次背面进攻 → **anti**
4. 本质：水改变中间体的可逆性

### 12.4 与现实/直觉的连接

- Woodward 和 Prevost 的区别就像"有刹车 vs 无刹车"——有水时反应"刹住"在第一次开环的位置，无水时可以"倒车"重新选择。
- 记忆口诀："W 像水波，顺式有波"——W 字母像水波，有水就是顺式。
