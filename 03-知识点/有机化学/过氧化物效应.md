# 过氧化物效应（Peroxide Effect / Kharasch Effect）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[37-加成反应]]、[[38-自由基取代]]

---

## 一、定义

**过氧化物效应**：在过氧化物（如 ROOR、BPO、AIBN）存在下，HBr 对烯烃的加成表现出**反马氏规则**的取向——Br 加到含氢较多的双键碳上，H 加到含氢较少的双键碳上。

**本质**：反应机理从**离子型（碳正离子）**转变为**自由基型**，选择性由**自由基稳定性**决定，而非碳正离子稳定性。

> ⭐ **ABOC 行 1827**："过氧化物效应：烯烃经历自由基加溴的反应具有'反马氏规则'，本质是自由基稳定性。"

## 三、核心原理

### 3.1 自由基加成 HBr 的链式机理

**链引发**：
```
RO-OR → 2 RO·         （过氧化物均裂）
RO· + HBr → ROH + Br·  （产生溴自由基）
```

**链转移**（增长）：
```
CH₃CH=CH₂ + Br· → CH₃ĊH-CH₂Br   （Br· 加到末端，生成二级自由基）
CH₃ĊH-CH₂Br + HBr → CH₃CH₂-CH₂Br + Br·  （产物 + 新 Br·）
```

**链终止**：
```
2 Br· → Br₂
2 R· → R-R
R· + Br· → RBr
```

### 3.2 为什么是反马氏？——自由基稳定性解释

溴自由基进攻烯烃时，有两种可能的加成方式：

| 进攻位点 | 生成自由基 | 稳定性 | 结果 |
|:---|:---|:---:|:---|
| Br· 进攻末端 CH₂ | CH₃ĊH-CH₂Br（二级自由基）| 较稳 | **主路径** |
| Br· 进攻内部 CH | CH₃CH₂-ĊHBr（一级自由基）| 不稳 | 极少 |

- 二级自由基 > 一级自由基（稳定性）
- 所以 Br· 优先进攻末端 → Br 最终连在末端 → **反马氏产物**

> ⭐ **ABOC 行 1833**："显然，溴自由基发动加成时，进攻了导致自由基稳定的一端，从而实现了反马氏加成。"

### 3.3 为什么溴自由基不拔氢形成烯丙基自由基？

**因为可逆！**（ABOC 行 1833）
- Br· + 烯丙位 H → HBr + 烯丙基自由基
- 该步骤可逆，且加成步骤更快、更有利
- 所以链转移以加成为主，而非取代

---

## 四、关键结论

1. **过氧化物效应 = 机理切换**（离子型 → 自由基型），不是规则例外
2. **反马氏的本质 = 自由基稳定性**（二级 > 一级），类比马氏规则中碳正离子稳定性
3. **只有 HBr 有明显过氧化物效应**——见 §6 限制
4. **Br· 的氧化性是关键**——氧化性不如 Cl· 强，所以较易生成且选择性适中
5. **NBS 体系是过氧化物效应的"温和版"**——低浓度 Br₂ 避免亲电加成竞争

## 五、常见分类或情形

### 5.1 引发剂类型

| 引发剂 | 全称 | 分解产物 | 特点 |
|:---|:---|:---|:---|
| **BPO** | 过氧化苯甲酰 | 2 Ph· + 2 CO₂ | 常用，分解温度适中 |
| **AIBN** | 偶氮二异丁腈 | 2 (CH₃)₂Ċ-CN + N₂ | 分解产生 N₂，易追踪 |
| **ROOR** | 二烷基过氧化物 | 2 RO· | 通用 |

### 5.2 NBS 体系：过氧化物效应的变体

NBS（N-溴代丁二酰亚胺）在 BPO/AIBN 引发下，作为**"溴单质均匀释放器"**：

```
NBS + HBr → 琥珀酰亚胺 + Br₂   （维持低浓度 Br₂）
```

- **NBS 的意义**：保证 Br₂ 含量较低 → 避免亲电加成（Br₂ 对双键的亲电加成）
- **主要产物**：烯丙位溴代（自由基取代），而非加成
- 见 [[自由基取代]]、[[NBS]]

> ⭐ **ABOC 行 1801**："NBS 采用如下原理：NBS + HBr = 琥珀酰亚胺 + Br₂，因此称为'溴单质均匀释放器'。"

## 六、适用条件与限制

1. **只有 HBr 有明显过氧化物效应**
   - **HCl**：Cl· 氧化性太强，难以从 HCl 中夺取 H 生成 Cl·（ABOC 行 1835）
   - **HI**：I· 与双键加成活化能太高，且 I· 易自相结合 → 不反应
   - **HF**：H-F 键太强，无法均裂产生 F·

2. **需要自由基引发条件**
   - 过氧化物（ROOR, BPO, AIBN）或光照/加热
   - 无引发剂时，正常离子型马氏加成

3. **O₂ 可能干扰**
   - O₂ 是自由基捕获剂，可能抑制链式反应

4. **不适用于共轭二烯**
   - 共轭二烯的自由基加成更复杂，可能得 1,2- 和 1,4- 混合产物

## 七、常见比较与易混点

### 7.1 离子型加成 vs 自由基加成：一张总表

| | **离子型加成（马氏）** | **自由基加成（反马氏）** |
|:---|:---|:---|
| **条件** | 极性溶剂、无过氧化物 | 过氧化物 / 光照 / 加热 |
| **机理** | 碳正离子中间体 | 自由基链式反应 |
| **决速步** | 质子化生成碳正离子 | Br· 对双键加成 |
| **选择性来源** | 碳正离子稳定性 | 自由基稳定性 |
| **H 加到哪** | 多氢碳 | 少氢碳 |
| **Br 加到哪** | 少氢碳（多取代）| 多氢碳（少取代）|
| **立体化学** | 无特殊要求 | 无特殊要求 |

### 7.2 过氧化物效应 vs 硼氢化-氧化

| | **过氧化物效应（HBr/ROOR）** | **硼氢化-氧化（BH₃/H₂O₂,OH⁻）** |
|:---|:---|:---|
| **产物** | 溴代烷（反马氏） | 醇（反马氏） |
| **机理** | 自由基链式 | 协同四中心过渡态 |
| **立体化学** | 无特殊要求 | **顺式加成、反马氏** |
| **碳骨架** | 不改变 | 不改变 |

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：丙烯分别与 a) HBr b) HBr/ROOR 反应，主产物分别是什么？

**解答**：
- a) **离子型马氏加成**：2-溴丙烷（Br 在 C2，多取代碳）
- b) **自由基反马氏加成**：1-溴丙烷（Br 在 C1，末端碳）

### 10.2 例题 2（★★ 机理书写）

**题目**：写出过氧化物存在下，HBr 与 2-甲基丙烯反应的自由基加成机理。

**解答**：
- **链引发**：ROOR → 2 RO·；RO· + HBr → ROH + Br·
- **链转移**：Br· + (CH₃)₂C=CH₂ → (CH₃)₂Ċ-CH₂Br（三级自由基，最稳定）
- **链转移**：(CH₃)₂Ċ-CH₂Br + HBr → (CH₃)₂CH-CH₂Br + Br·
- **链终止**：2 Br· → Br₂（等）
- 产物：**1-溴-2-甲基丙烷**（反马氏：Br 在末端）

### 10.3 例题 3（★★★ 为何只有 HBr）

**题目**：为什么过氧化物效应主要对 HBr 有效，而对 HCl 和 HI 无效？

**解答**：
- **HCl**：Cl· 氧化性太强 → 难以从 HCl 中夺 H 生成 Cl· → 链引发困难（ABOC 行 1835）
- **HI**：I· 与双键加成要求较高活化能 → 加成步骤慢；且 I· 易自相结合 → 链增长难以维持
- **HBr**：Br· 氧化性适中 → 既能从 HBr 中夺 H，又能与双键有效加成 → 链式反应可行

### 10.4 例题 4（★★★ NBS 体系分析）

**题目**：环己烯 + NBS / BPO / CCl₄，主产物是什么？为什么不是加成产物？

**解答**：
- NBS 维持低浓度 Br₂ → 避免 Br₂ 对双键的亲电加成
- BPO 产生 Ph· → Ph· 夺取 HBr 中的 H → Br·
- Br· 进攻烯丙位 H → 烯丙基自由基（最稳定）
- 主产物：**3-溴环己烯**（烯丙位溴代，自由基取代而非加成）
- 这是 NBS 的经典应用，见 [[自由基取代]]

## 十一、易错点

1. ❌ **认为所有 HX 都有过氧化物效应** → 只有 HBr 明显有效
2. ❌ **把反马氏当成马氏规则的"例外"** → 其实是**不同机理**（自由基 vs 离子型）
3. ❌ **混淆过氧化物效应与催化加氢** → 催化加氢（H₂/Pd）不是反马氏，是无选择性的顺式加氢
4. ❌ **写机理时遗漏链终止** → 完整链式机理必须包含引发/转移/终止
5. ❌ **认为过氧化物效应只发生在加成** → NBS 体系中主要是烯丙位取代
6. ❌ **忽略 O₂ 的干扰** → O₂ 捕获自由基，可能抑制反应
7. ❌ **把"反马氏"等同于"末端取代"** → 对不对称内烯，需具体分析自由基稳定性
8. ❌ **Br· 进攻位点判断错误** → Br· 优先进攻能生成更稳定自由基的碳

### 12.3 入门级例题

**题目**：比较以下三个反应的主产物：
a) 1-丁烯 + HBr（无过氧化物）
b) 1-丁烯 + HBr / ROOR
c) 1-丁烯 + BH₃ / THF，然后 H₂O₂/OH⁻

**预期解答路径**：
1. a) 离子型马氏加成 → 2-溴丁烷
2. b) 自由基反马氏加成 → 1-溴丁烷
3. c) 硼氢化-氧化（协同反马氏）→ 1-丁醇
4. 注意 b) 和 c) 都是"反马氏"取向，但机理完全不同

**老师可引导的提问**：
- 卡在第 2 步：问 "Br· 进攻 1-丁烯的哪个碳？生成几级自由基？"
- 卡在第 3 步：问 "硼氢化是自由基还是协同机理？四中心过渡态中 H 和 B 分别带什么电荷？"
- 延伸问："若用 DBr/ROOR（D = 氘），产物中 D 在哪里？"

### 12.4 与现实/直觉的连接

- **类比**：离子型加成就像"磁铁吸引"——H⁺（正极）被双键电子云吸引，自然落到电子密度高的地方（多取代碳）。自由基加成就像"台球碰撞"——Br· 这颗台球滚过去，它更喜欢停在能量更低的位置（生成更稳定的自由基）。过氧化物就是那个"发球杆"，提供了初始的 Br·。
- **反例**：如果认为"反马氏 = 不遵循规则 = 错误"，那就错了。有机化学中没有"错误"的路径，只有"在不同条件下占主导"的路径。过氧化物效应恰恰说明：条件可以改变机理，机理决定选择性。

## 十三、竞赛拓展

1. **反马氏氯化氢的间接实现**：
   - ABOC 行 1837 思考题：可通过硅氢化/氧化或硼氢化-氧化后再转换实现
   - 或利用汞盐催化后再脱汞（oxymercuration-demercuration 的变体）

2. **自由基加成中的立体化学**：
   - 自由基平面 sp² 结构 → 亲电试剂可从两面进攻 → 通常得外消旋混合物
   - 若底物有手性中心，可能存在非对映选择性

3. **氧气的调控作用**：
   - 微量 O₂ 可以"调节"自由基链长
   - 过量 O₂ 完全抑制自由基反应（形成过氧自由基）
