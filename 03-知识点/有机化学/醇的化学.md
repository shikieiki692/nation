# 醇的化学

> 醇是有机合成中最核心的官能团之一，其制备、转化（卤代、氧化、消除）构成了合成化学的基础工具箱。

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[醇酚醚]]

## 一、定义

**醇（Alcohol）**：烃分子中的氢原子被羟基（-OH）取代形成的化合物，通式 R-OH。

- **伯醇（1°醇）**：-OH 连接在伯碳上（RCH₂OH）
- **仲醇（2°醇）**：-OH 连接在仲碳上（R₂CHOH）
- **叔醇（3°醇）**：-OH 连接在叔碳上（R₃COH）
- **烯醇**：-OH 连接在烯碳上（不稳定，互变异构为羰基）

## 三、核心原理

### 3.1 醇的制备

#### 3.1.1 格氏试剂法

$$
\ce{RMgX + R'CHO -> R-CH(R')-OMgX ->[H3O+] R-CH(R')-OH}
$$

**特点**：
- 格氏试剂与醛反应得 2°醇，与酮反应得 3°醇
- 与甲醛反应得 1°醇（多一个碳）
- 与酯反应得 3°醇（两分子格氏试剂加成）
- **限制**：格氏试剂对水、CO₂、活泼氢敏感

#### 3.1.2 硼氢化-氧化反应

$$
\ce{R-CH=CH2 + BH3 -> (R-CH2-CH2)3B ->[H2O2/OH-] R-CH2-CH2-OH}

$$

**特点**：
- **反马氏规则**：羟基加在取代较少的碳上
- **顺式加成**：H 和 OH 从同侧加成
- **立体选择性**：硼从位阻较小面进攻
- 与烯烃水合（酸催化，马氏规则）形成互补

#### 3.1.3 其他制备方法

| 方法 | 试剂/条件 | 产物 | 特点 |
|:---|:---|:---|:---|
| **烯烃水合** | H₂O/H⁺ 或 Hg(OAc)₂/NaBH₄ | 醇 | 马氏规则 |
| **卤代烷水解** | NaOH/H₂O | 醇 | SN2（1°）或 SN1（3°）|
| **醛/酮还原** | NaBH₄ 或 LiAlH₄ | 1°/2°醇 | 选择性还原 |
| **环氧开环** | 酸或碱催化 | 邻二醇 | 酸/碱催化位向不同 |
| **羟汞化-去汞** | Hg(OAc)₂/H₂O → NaBH₄ | 醇 | 马氏规则，不重排 |

### 3.2 醇的取代反应

#### 3.2.1 醇 → 卤代烷

**四种经典方法对比**：

| 试剂 | 条件 | 立体化学 | 适用范围 | 特点 |
|:---|:---|:---:|:---|:---|
| **SOCl₂** | 碱（吡啶/Et₃N）| **inversion** | 1°/2°醇 | 副产物 SO₂ + HCl（气体，易除）|
| **PBr₃** | 中性 | **inversion** | 1°/2°醇 | 副产物 H₃PO₃ |
| **Appel（CBr₄/PPh₃）** | 中性 | **inversion** | 1°/2°醇 | 温和，官能团耐受性好 |
| **DAST** | 低温 | 有时 **retention** | 1°/2°/3°醇 | 氟代首选，也可氯代 |

**机理**：
- SOCl₂/碱：先形成氯亚硫酸酯 → SN2 → inversion
- PBr₃：先形成亚磷酸酯 → SN2 → inversion
- DAST：特殊机理（有时通过碳正离子或邻基参与导致 retention）

#### 3.2.2 Mitsunobu 反应

$$
\ce{ROH + R'COOH + DIAD + PPh3 -> R'COOR（构型翻转）+ Ph3P=O + DIAD-H2}
$$

**特点**：
- 醇的 **inversion**，形成酯
- 本质：PPh₃ 活化醇 → 氧鎓中间体 → 羧酸根背面进攻
- P=O 键的形成是反应驱动力
- 可在温和条件下实现 2°醇的构型翻转

### 3.3 醇的消除反应

#### 3.3.1 酸催化脱水（E1 机理）

$$
\ce{R3C-OH + H+ -> R3C-OH2+ -> R3C+ -> 烯烃 + H+}
$$

**特点**：
- 3°醇 > 2°醇 > 1°醇（碳正离子稳定性）
- 遵循 **Zaitsev 规则**：多取代烯烃为主
- 可能伴随重排（碳正离子中间体）

#### 3.3.2 E2 消除

$$
\ce{R-CH2-CH2-OH + POCl3 ->[吡啶] R-CH=CH2}
$$

**特点**：
- **反式共平面（anti-periplanar）**要求
- 用 Newman 投影式分析构象
- 环己烷衍生物需画椅式构象判断可行性

#### 3.3.3 E1 vs E2 vs E1cb 对比

| | **E1** | **E2** | **E1cb** |
|:---|:---|:---|:---|
| **步骤** | 两步 | 一步协同 | 两步 |
| **决速步** | 离去基团离去 | C-H 断裂 + LG 离去同步 | C-H 断裂（去质子）|
| **碳正离子** | 有 | 无 | 无 |
| **碱的要求** | 弱碱即可 | 需要强碱 | 需要极强碱 |
| **底物** | 3° > 2° > 1° | 无限制（但位阻影响）| 需稳定碳负离子 |
| **立体化学** | 无特定 | **反式共平面** | 无特定 |
| **区域选择性** | Zaitsev | Zaitsev 或 Hofmann | Hofmann |

#### 3.3.4 Zaitsev vs Hofmann 规则

- **Zaitsev（热力学产物）**：多取代烯烃为主
- **Hofmann（动力学产物）**：少取代烯烃为主
- **控制因素**：
  - 大位阻碱 → Hofmann（难以接近多取代 β-H）
  - 高温/弱碱 → Zaitsev（热力学平衡）
  - 离去基团差（如 F）→ Hofmann（E1cb 特征）

### 3.4 醇的氧化

#### 3.4.1 氧化试剂选择性光谱

| 试剂 | 氧化强度 | 1°醇产物 | 2°醇产物 | 特点 |
|:---|:---:|:---:|:---:|:---|
| **PCC/PDC** | 温和 | **醛** | 酮 | 不进一步氧化醛 |
| **Swern** | 温和 | **醛** | 酮 | 低温，无金属 |
| **Dess-Martin** | 温和 | **醛** | 酮 | 高选择性，贵 |
| **Jones（CrO₃/H₂SO₄）** | 强 | **羧酸** | 酮 | 酸性，副反应多 |
| **Oppenauer** | 温和 | — | 酮 | 可逆，MPV 逆反应 |
| **MPV 还原** | 温和 | — | 2°醇 | 酮 + 异丙醇 → 醇 |

**核心思维**：氧化试剂的选择性——"想停在哪就停在哪"。

#### 3.4.2 Swern 氧化机理

$$
\ce{DMSO + (COCl)2 -> [Cl2S+(CH3)2] ->[ROH] 烷氧基锍盐 ->[Et3N] RCHO + (CH3)2S}
$$

**步骤**：
1. DMSO 与草酰氯形成氯锍盐
2. 醇进攻 → 烷氧基锍盐
3. Et₃N 去质子 → 消除 → 醛/酮 + 二甲硫醚

#### 3.4.3 其他氧化方法

| 试剂 | 特点 | 适用范围 |
|:---|:---|:---|
| **Collins 试剂** | CrO₃·吡啶复合物 | 1°醇 → 醛 |
| **Corey-Kim** | NCS/DMS + 醇 → 氧化 | 温和氧化 |
| **Ley-Griffith（TPAP/NMO）** | 高价钌催化 | 不饱和醇的选择性氧化 |
| **TEMPO/NaOCl** | 有机催化 | 1°醇 → 醛 |

## 四、关键结论

### 4.1 醇的反应性排序

| 反应类型 | 反应性顺序 | 原因 |
|:---|:---|:---|
| 亲核取代（SN1）| 3° > 2° > 1° | 碳正离子稳定性 |
| 亲核取代（SN2）| 1° > 2° > 3° | 位阻 |
| 消除（E1）| 3° > 2° > 1° | 碳正离子稳定性 |
| 消除（E2）| 3° > 2°（强碱）| 烯烃稳定性 + 位阻 |
| 氧化 | 1°/2°可氧化，3°不氧化 | 无 α-H |

### 4.2 氧化试剂选择决策树

```
目标产物是醛？
  ├─ 是 → PCC、Swern、Dess-Martin、Collins
  └─ 否 → 目标产物是羧酸？
           ├─ 是 → Jones、KMnO₄
           └─ 否 → 目标产物是酮？
                    ├─ 是 → PCC、Swern、Dess-Martin、Jones
                    └─ 否 → 3°醇不可氧化
```

### 4.3 卤代试剂选择

| 目标 | 首选试剂 | 注意 |
|:---|:---|:---|
| 1°醇 → 氯代烷 | SOCl₂/吡啶 | inversion |
| 1°醇 → 溴代烷 | PBr₃ 或 Appel | inversion |
| 1°/2°/3°醇 → 氟代烷 | DAST | 有时 retention |
| 构型翻转（2°醇）| Mitsunobu | 先成酯，再水解 |

## 五、常见分类或情形

### 5.1 按取代/消除竞争分类

| 条件 | 主要产物 | 原因 |
|:---|:---|:---|
| 低温、弱碱、稀溶液 | 取代 | SN2 有利 |
| 高温、强碱、浓溶液 | 消除 | E2 有利 |
| 3°醇 + 酸 | 消除（E1）| 碳正离子易形成 |
| 1°醇 + 强亲核试剂 | 取代（SN2）| 位阻小 |

### 5.2 按氧化场景分类

- **选择性氧化**：PCC/Swern/Dess-Martin（醛不停留）
- **彻底氧化**：Jones/KMnO₄（1°醇 → 羧酸）
- **逆反应**：MPV 还原（酮 → 2°醇）

## 六、适用条件与限制

### 6.1 格氏试剂制备醇的限制

- **底物不能含活泼氢**（-OH、-COOH、-NH₂ 等需先保护）
- **底物不能含羰基**（会自身反应）
- **CO₂ 敏感**：需在惰性气氛下操作

### 6.2 氧化试剂的选择性

- **PCC**：酸性，对酸敏感底物不适用
- **Swern**：低温（-78°C → 0°C），需无水条件
- **Dess-Martin**：中性，最贵，但选择性最好
- **Jones**：强酸性，可能破坏酸敏感官能团

### 6.3 Mitsunobu 反应的限制

- 需要 pKa < 15 的酸作为亲核体
- 2°醇可反应，但 3°醇通常不反应
- 原子经济性差（产生大量副产物）

## 七、常见比较与易混点

| 对比项 | SN1 | SN2 |
|:---|:---|:---|
| 底物 | 3° > 2° | 1° > 2° |
| 立体化学 | 外消旋化 | 构型翻转 |
| 重排 | 可能 | 不可能 |
| 溶剂 | 极性质子有利 | 极性非质子有利 |

| 对比项 | Zaitsev 规则 | Hofmann 规则 |
|:---|:---|:---|
| 产物 | 多取代烯烃 | 少取代烯烃 |
| 控制因素 | 热力学（稳定性）| 动力学（位阻）|
| 典型条件 | 高温、弱碱 | 大位阻碱 |
| 对应消除 | E1、E2（小位阻碱）| E2（大位阻碱）、E1cb |

| 对比项 | PCC | Jones |
|:---|:---|:---|
| 1°醇产物 | 醛 | 羧酸 |
| 条件 | 温和（CH₂Cl₂）| 强酸（H₂SO₄）|
| 选择性 | 高 | 低 |
| 官能团耐受 | 较好 | 较差 |

## 十、例题

### 例题 1：氧化试剂选择
**题目**：如何将 1-丁醇转化为丁醛？写出试剂和条件。

**解答**：
可选试剂：
- **PCC**：CH₂Cl₂，室温 → 丁醛（不停留）
- **Swern**：(COCl)₂/DMSO，-78°C → Et₃N → 丁醛
- **Dess-Martin 氧化**：CH₂Cl₂，室温 → 丁醛

不可用 Jones（会氧化为丁酸）。

### 例题 2：E2 消除构象分析
**题目**：某环己烷衍生物的 ³Jₐₓ₋ₐₓ = 10 Hz，³Jₐₓ₋ₑq ≈ 4 Hz，³Jₑq₋ₑq ≈ 2 Hz。解释这些数值并预测 E2 消除产物。

**解答**：
- 偶合常数对应二面角约 180°、60°、60°，符合椅式构象
- E2 消除要求 H 和离去基团 anti-periplanar（二面角 180°）
- 只有直立-直立（ax-ax）关系满足此几何要求
- 因此，消除只发生在离去基团和相邻直立氢之间

### 例题 3：醇的制备
**题目**：由丙烯制备 1-丙醇和 2-丙醇。

**解答**：
**1-丙醇（反马氏）**：
$$
\ce{CH3-CH=CH2 + BH3 -> (CH3-CH2-CH2)3B ->[H2O2/OH-] CH3-CH2-CH2OH}
$$

**2-丙醇（马氏）**：
$$
\ce{CH3-CH=CH2 + H2O ->[H+] CH3-CH(OH)-CH3}
$$
或
$$
\ce{CH3-CH=CH2 + Hg(OAc)2/H2O -> NaBH4 CH3-CH(OH)-CH3}
$$

## 十一、易错点

1. **"PCC 氧化 1°醇得羧酸"**：错误，PCC 停在醛；Jones 才氧化到羧酸
2. **"E2 总是 Zaitsev"**：错误，大位阻碱 → Hofmann 产物
3. **"消除和取代是竞争"**：正确，但高温/强碱 → 消除；低温/弱碱 → 取代
4. **"所有醇卤代都 inversion"**：错误，DAST 有时 retention
5. **"3°醇可被氧化"**：错误，3°醇无 α-H，不能被常规氧化剂氧化
6. **"Swern 氧化需要高温"**：错误，Swern 需低温（-78°C 开始）
7. **忽略 Mitsunobu 的 pKa 要求**：亲核体 pKa 必须 < 15

## 十二、🎯 教学视角

### 12.1 学习路径

- **前置**：掌握 SN1/SN2 机理、碳正离子稳定性、环己烷构象分析
- **核心**：理解"氧化还原是一个光谱"——从温和到强氧化的梯度控制
- **延伸**：醇的转化组合策略（氧化→卤代→消除→亲核取代）是全合成的基础

### 12.2 学生易踩的认知误区

- **"氧化就是彻底氧化"**：不理解选择性氧化试剂的意义
- **"E2 只记 Zaitsev"**：忽略碱的位阻和底物结构的影响
- **"所有醇卤代都 inversion"**：DAST 的特殊机理导致例外
- **"3°醇氧化得酮"**：3°醇根本不能被氧化

### 12.3 入门级例题

**题目**：用 Swern 氧化将环己基甲醇转化为环己基甲醛，写出关键试剂。

**解答**：(COCl)₂、DMSO（-78°C），然后 Et₃N。

### 12.4 与现实/直觉的连接

- 醇的氧化选择性 → 香水工业中醛类香料的合成
- Swern 氧化的低温要求 → 实验室中干冰-丙酮浴的应用
- 硼氢化-氧化的反马氏规则 → 合成特定区域异构体的工具
- 醇脱水制烯烃 → 石油化工中乙烯/丙烯的工业制备

## 十三、竞赛拓展

1. **Sharpless 不对称双羟基化**：烯烃 → 邻二醇，对映选择性控制
2. **Woodward/Prevost 双羟基化**：碘/羧酸银法，不同立体化学
3. **频哪醇重排**：邻二醇在酸催化下的 1,2-迁移
4. **Tishchenko 反应**：醛的歧化生成酯（无 α-H 或有 α-H 均可）
5. **Meerwein-Ponndorf-Verley 还原**：Oppenauer 的逆反应，酮 → 醇
