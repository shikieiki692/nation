# Fischer 投影式（Fischer Projection）

## 一、定义

**Fischer 投影式**是由德国化学家 Emil Fischer 于 1891 年提出的一种用于表示手性分子（尤其是糖类和氨基酸）立体结构的平面投影方法。它用二维平面表达三维立体信息，是糖化学和氨基酸立体化学的标准表示法。

- **核心约定**：
  - **十字交叉点** = 手性碳原子
  - **水平线（横线）** = 键指向观察者（纸面前方，楔形向前）
  - **竖直线（竖线）** = 键远离观察者（纸面后方，虚线向后）
  - **碳链竖排**，氧化态最高的基团（如醛基、羧基）通常置于上方
- **主要用途**：糖类、氨基酸的构型表达与 D/L、R/S 判定

## 三、核心原理

### 3.1 Fischer 投影式的绘制规则

**规则 1：主链竖排**
- 碳主链沿竖直方向排列
- 氧化态最高的碳原子置于**最上方**（醛糖的醛基、氨基酸的羧基）
- 编号最小的碳在最上方

**规则 2：横前竖后**
- 水平线上的基团：**指向观察者**（纸面前方）
- 竖直线上的基团：**远离观察者**（纸面后方）

**规则 3：不可旋转 90°**
- 纸面内旋转 **180°**：构型**保持不变**
- 纸面内旋转 **90°**：构型**翻转**（R↔S，D↔L）——**禁止操作**

### 3.2 Fischer 投影式的操作规则

| 操作 | 对构型的影响 | 说明 |
|:---|:---:|:---|
| **纸面内旋转 180°** | **不变** | 允许操作 |
| **纸面内旋转 90°** | **翻转** | 不允许 |
| **交换任意两个基团** | **翻转** | 奇数次交换 = 翻转 |
| **连续交换两次** | **不变** | 偶数次交换 = 复原 |
| **左右翻转（镜像）** | **对映体** | 得到镜像结构 |
| **上下翻转** | **通常改变** | 除非分子对称 |

### 3.3 Fischer 投影式与楔形式的转换

**Fischer → 楔形式**：
1. 水平线上的基团 → 实心楔形（朝前）
2. 竖直线上的基团 → 虚线楔形（朝后）
3. 保持碳链骨架不变

**楔形式 → Fischer**：
1. 将碳链调整为竖直方向
2. 确保左右手性碳的横前竖后关系正确
3. 必要时先旋转楔形式使两个主要基团处于适当位置

**注意**：直接转换时若碳链不在纸面内，需先进行 mentally rotation。

## 四、关键结论

1. **Fischer 投影式的本质是"约定"而非"真实"**：横前竖后是人为约定，同一分子可有多种合法投影式
2. **180° 旋转合法，90° 旋转非法**：这是 Fischer 投影式最容易出错的地方
3. **判断对映体的方法**：若一个投影式无法通过合法操作（180° 旋转）与另一个重合，且二者呈镜像关系，则为对映体
4. **判断同一化合物的方法**：若能通过偶数次基团交换或 180° 旋转重合，则为同一化合物
5. **Fischer 投影式特别适用于含多个相邻手性中心的链状分子**：如糖类、氨基酸、羟基酸

## 五、常见分类

### 5.1 常见分子的 Fischer 投影式

**甘油醛（最简单的醛糖）**：

```
    CHO          CHO
     |            |
 H — C — OH    HO — C — H
     |            |
    CH₂OH        CH₂OH

   D-甘油醛     L-甘油醛
```

**乳酸**：

```
    COOH         COOH
     |            |
 H — C — OH    HO — C — H
     |            |
    CH₃          CH₃

  (S)-乳酸    (R)-乳酸
```

**D-葡萄糖**：

```
     CHO
      |
  H — C — OH       (C2)
      |
  HO — C — H       (C3)
      |
  H — C — OH       (C4)
      |
  H — C — OH       (C5)
      |
     CH₂OH
```

### 5.2 合法操作 vs 非法操作

| 场景 | 操作 | 结果 | 是否允许 |
|:---|:---|:---:|:---:|
| 投影式 A 旋转 180° | 纸面内 180° | 与 A 相同 | ✅ |
| 投影式 A 旋转 90° | 纸面内 90° | A 的对映体 | ❌ |
| 交换 A 中左右两个基团 | 一次交换 | A 的对映体 | ⚠️（需知翻转）|
| 先交换左右，再旋转 180° | 奇数次交换 + 180° | A 的对映体 | ⚠️ |
| 投影式在纸面外翻转 | 三维翻转 | 需重新投影 | 需重画 |

## 六、适用条件

### 6.1 适用与不适用的场景

| 适用 | 不适用 |
|:---|:---|
| 链状手性分子（糖类、氨基酸）| 环状分子（环己烷椅式等）|
| 含多个相邻手性中心 | 仅含一个手性中心且需要直观空间感 |
| D/L 构型判断 | 需要展示构象的分子 |
| R/S 构型判断（配合规则）| 大环或稠环体系 |
| 糖类环化前的开链形式 | 三维空间关系复杂的分子 |

### 6.2 Fischer 投影式中的 R/S 判定

**口诀："横变竖不变"**

| 最低优先基团（4号）位置 | 1→2→3 顺时针 | 1→2→3 逆时针 |
|:---|:---:|:---:|
| **竖线**（朝后）| R | S |
| **横线**（朝前）| S | R |

**推导**：Fischer 中横线朝前，即最低优先基朝前。标准规则要求最低优先朝后，因此直接观察的结果与实际相反。

## 七、常见比较

### 7.1 Fischer 投影式 vs 楔形式

| 维度 | Fischer 投影式 | 楔形式 |
|:---|:---|:---|
| **维度** | 二维平面 | 三维立体 |
| **表达精度** | 固定约定（横前竖后）| 直观空间关系 |
| **适用分子** | 糖类、氨基酸、链状多手性中心 | 所有手性分子 |
| **操作复杂度** | 有严格操作规则（180°/90°）| 自由旋转 |
| **书写速度** | 快（简单线条）| 较慢（需画楔形）|
| **初学者友好度** | 需记忆规则 | 更直观 |

### 7.2 四种结构表达方式的比较

| 表达方式 | 特点 | 最佳适用 |
|:---|:---|:---|
| **Fischer 投影式** | 横前竖后，碳链竖排 | 糖类、氨基酸、多手性中心链 |
| **楔形式** | 实心朝前，虚线朝后 | 单/双手性中心、直观教学 |
| **锯架式** | 沿 C-C 键观察，展示构象 | 构象分析（丁烷、环己烷）|
| **纽曼投影式** | 沿 C-C 键观察，前后碳重叠 | 构象分析、旋转势能 |

## 十、例题

### 例题 1：判断同一化合物

**题目**：下列两个 Fischer 投影式是否代表同一化合物？

```
    CHO              CHO
     |                |
 H — C — OH        HO — C — H
     |                |
 H — C — OH        HO — C — H
     |                |
    CH₂OH           CH₂OH

      A                B
```

**解析**：

1. 观察 A 和 B：B 看起来是 A 的左右翻转（镜像）
2. 尝试对 A 进行合法操作：将 A **旋转 180°**
3. 旋转 180° 后 A 变为：

```
    CH₂OH
     |
 HO — C — H
     |
 HO — C — H
     |
    CHO
```

4. 再上下颠倒（等价于整体旋转 180° 后的竖排）：
   - 实际上，旋转 180° 后，A 的水平基团位置互换、上下基团互换
   - 得到的结构与 B **不完全相同**

5. **更准确的方法**：比较每个手性中心的构型
   - A 的 C2：—OH 在右 → D-型；C3：—OH 在右 → 与赤藓糖类似
   - B 的 C2：—OH 在左；C3：—OH 在左
   - B 是 A 的**对映体**

**结论**：A 和 B 代表**对映体**，不是同一化合物。

### 例题 2：Fischer ↔ 楔形式转换

**题目**：将下列 Fischer 投影式转换为楔形式，并判断构型。

```
    COOH
     |
 H — C — NH₂
     |
    CH₃
```

**解析**：

1. **识别结构**：丙氨酸的 Fischer 投影式
2. **转换规则**：
   - 横线（—H, —NH₂）→ 实心楔形（朝前）
   - 竖线（—COOH, —CH₃）→ 虚线楔形（朝后）

3. **楔形式**：
   - 中心碳连 —COOH（虚线，上）、—CH₃（虚线，下）
   - —H（实心，左）、—NH₂（实心，右）

4. **R/S 判断**：
   - CIP 顺序：—NH₂ > —COOH > —CH₃ > —H
   - Fischer 中 —H 在横线（朝前）
   - 1→2→3：NH₂(右) → COOH(上) → CH₃(下) → 逆时针
   - 横线规则取反：逆时针 → **S**

5. **结论**：该分子为 **L-丙氨酸 = (S)-丙氨酸**

### 例题 3：非法操作识别

**题目**：某学生将 D-甘油的 Fischer 投影式旋转了 90° 后认为构型不变。指出错误并说明正确结果。

**解析**：

1. **错误**：Fischer 投影式旋转 **90° 会翻转构型**
2. **D-甘油醛旋转 90° 后**：
   - 原来的 D-型（最下方手性碳 —OH 在右）
   - 旋转 90° 后 —OH 位置改变，变为 **L-型**
3. **正确操作**：如需调整投影式方向，应旋转 **180°**（构型不变）

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| "Fischer 投影式旋转任意角度都保持构型" | 混淆普通结构与投影式 | Fischer 只允许 **180°** 旋转；90° 会翻转 | Fischer 投影式旋转 90° 后构型变吗？ |
| "横线上的基团朝后" | 记忆颠倒 | 横线 = **朝前**，竖线 = **朝后** | Fischer 中水平线代表朝哪个方向？ |
| "Fischer 中最低优先基在横线上直接判断 R/S" | 忽略横变规则 | 横线上要**取反**：顺时针 → S，逆时针 → R | Fischer 中 H 在横线，1→2→3 顺时针是什么构型？ |
| "左右翻转后仍是同一化合物" | 混淆旋转与翻转 | 左右翻转（镜像）得到**对映体**，需通过 180° 旋转验证 | 两个投影式左右对称，是同一化合物吗？ |
| "把 Fischer 用于环状化合物" | 适用范围不清 | Fischer 主要用于**链状**分子；环状用椅式/楔形式 | 环己烷能用 Fischer 投影式表示吗？ |

### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "Fischer 就是普通化学结构式" | 忽略投影约定 | 强调：Fischer 是**有约定的投影**，横竖线有明确的立体含义 |
| "转换 Fischer 和楔形式很简单" | 低估空间想象难度 | 提供大量练习，强制"横前竖后"的对应关系 |

### 12.3 入门例题

**最简判断练习**：
> 1. Fischer 投影式中，竖直线上的键指向哪里？
> 2. 将 Fischer 投影式旋转 180° 后，构型是否改变？
> 3. Fischer 投影式中最低优先基在竖线上，1→2→3 逆时针，构型是？

**答案**：
1. **远离观察者**（纸面后方）
2. **不改变**（180° 旋转是合法操作）
3. **S**（竖线不变，逆时针 = S）
