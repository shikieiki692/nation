# 质谱（Mass Spectrometry, MS）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[30-有机波谱分析]]

---

## 一、定义

**质谱（MS）**：通过将有机分子离子化后按**质荷比（m/z）**分离并检测，获得分子离子峰和碎片离子峰信息，从而推断分子量和结构的波谱技术。

**本质**：**有机化学家的天平+锤子**——先称分子量（天平），再打碎看碎片（锤子），从碎片反推结构。

## 三、核心原理

### 3.1 基本原理

**EI-MS（电子轰击质谱）流程**：
```
样品气化 → 电子轰击（70 eV）→ 分子离子 M⁺• → 碎片化 → 质量分析器分离 → 检测
```

**关键峰类型**：

| 峰类型 | 符号 | 意义 |
|:---|:---:|:---|
| **分子离子峰** | M⁺• | 分子失去一个电子形成的自由基正离子，给出分子量 |
| **碎片离子峰** | — | 分子离子断裂产生的各种离子，提供结构信息 |
| **同位素峰** | M+1, M+2 | 由重同位素（¹³C、²H、³⁴S 等）引起 |
| **亚稳离子峰** | m* | 飞行途中分解的离子，用于验证裂解路径 |

### 3.2 氮规则（Nitrogen Rule）

- **含有奇数个氮原子**的化合物 → 分子离子峰为**奇数 m/z**
- **不含氮或含偶数个氮原子**的化合物 → 分子离子峰为**偶数 m/z**
- **竞赛应用**：快速判断分子式中是否含氮及氮原子奇偶性

### 3.3 常见碎片离子（竞赛高频）

| 碎片 | m/z | 来源/特征 |
|:---:|:---:|:---|
| 失去 CH₃• | M–15 | 甲基化合物 |
| 失去 H₂O | M–18 | 醇（脱水）|
| 失去 CO | M–28 | 醛、酮、羧酸、酯 |
| 失去 C₂H₄ | M–28 | McLafferty 重排 |
| 失去 C₂H₅• | M–29 | 乙基、醛基（–CHO）|
| 失去 CH₂=CH₂ | M–28 | 烯烃侧链 |
| 苄基离子 | 91 | C₆H₅CH₂⁺，甲苯/取代甲苯特征 |
| 卓鎓离子 | 91 | C₇H₇⁺，环庚三烯基正离子 |
| 酰基离子 | RCO⁺ | 醛/酮/酸的 α-裂解 |
| 异丙基 | 43 | C₃H₇⁺，高度稳定 |
| 叔丁基 | 57 | C₄H₉⁺，高度稳定 |

### 3.4 McLafferty 重排

**条件**：
1. 分子中含有羰基（醛/酮/酸/酯）
2. γ-位有氢原子（γ-H）

**机理**：
$$
\ce{γ-H 转移至羰基氧 → 六元环过渡态 → 烯醇自由基正离子 + 烯烃}
$$

**竞赛意义**：
- McLafferty 重排产生 **M–28（C₂H₄）或更大烯烃**的碎片
- 是判断 **γ-H 存在**的有力证据
- 常与 α-裂解竞争发生

### 3.5 同位素峰识别

**含卤素化合物的特征同位素分布**：

| 元素 | 天然同位素比 | M+2 相对强度 |
|:---|:---:|:---:|
| Cl | ³⁵Cl : ³⁷Cl = 3 : 1 | M+2 ≈ 1/3 M |
| Br | ⁷⁹Br : ⁸¹Br = 1 : 1 | M+2 ≈ M |

**竞赛应用**：
- 看到 M : M+2 ≈ 3 : 1 → 分子含 **1 个 Cl**
- 看到 M : M+2 ≈ 1 : 1 → 分子含 **1 个 Br**
- 看到 M : M+2 : M+4 ≈ 1 : 2 : 1 → 分子含 **2 个 Br**

### 3.6 高分辨质谱（HRMS）

- 精确到小数点后 4 位
- 可区分**相同整数质量、不同精确质量**的离子
- 例：C₆H₈⁺• 和 C₅H₄O⁺• 的整数质量均为 80，但精确质量不同
- **竞赛应用**：直接确定分子式（精确质量匹配）

## 四、关键结论

1. **结构推求第一步**：先看分子离子峰确定分子量，再看碎片峰推断结构片段
2. **氮规则快速判氮**：奇数 m/z 且含 N → 奇数个 N；偶数 m/z → 无 N 或偶数个 N
3. **卤素识别看 M+2**：Cl 的 3:1 和 Br 的 1:1 是最可靠的同位素指纹
4. **McLafferty 重排 = γ-H + 羰基** → 看到 M–28 且分子含羰基 → 优先考虑
5. **稳定碳正离子优先形成**：苄基、烯丙基、叔丁基等离子丰度最高

## 五、常见分类或情形

### 5.1 按裂解方式分类

| 裂解方式 | 特征 | 典型产物 |
|:---|:---|:---|
| **α-裂解** | 羰基化合物，Cα–Cβ 键断裂 | 酰基离子 RCO⁺ |
| **β-裂解** | 醇/醚，Cβ–Cγ 键断裂 | 氧鎓离子 |
| **McLafferty 重排** | γ-H 转移至羰基氧 | 烯醇离子 + 烯烃 |
| **逆 Diels-Alder** | 环己烯类 | 二烯 + 烯烃碎片 |

### 5.2 分子离子峰的稳定性顺序

$$
\text{芳环} > \text{共轭烯烃} > \text{环状化合物} > \text{硫化物} > \text{直链烷烃} > \text{支链烷烃} > \text{醇}
$$

- **醇的分子离子峰往往很弱或看不到**（容易脱水）
- **芳环化合物的 M⁺• 峰通常很强**

---

## 六、适用条件与限制

1. **需要气化**：不适用于热不稳定化合物（可用 ESI/MALDI 等软电离，但竞赛通常考 EI）
2. **分子离子峰可能缺失**：醇、支链烷烃等容易裂解，M⁺• 峰弱
3. **同分异构体区分困难**：质谱无法直接区分构造异构体（需结合 NMR）
4. **仅提供质量信息**，不提供氢数信息（需结合分子式）

---

## 七、常见比较与易混点

### 7.1 质谱 vs NMR vs UV-Vis

| | **质谱** | **NMR** | **UV-Vis** |
|:---|:---|:---|:---|
| **信息类型** | 分子量 + 碎片 | 氢环境 + 数量 + 偶合 | 共轭体系大小 |
| **精度** | 高（HRMS 到小数点后 4 位）| 中等（δ 到小数点后 2 位）| 低（λmax 到 nm）|
| **破坏样品** | 是（分子被打碎）| 否 | 否 |
| **竞赛用途** | 确定分子式 | 确定结构 | 验证共轭 |

### 7.2 碎片峰的误判

| 碎片 | 可能误判 | 正确判断 |
|:---|:---|:---|
| M–15 | 失 CH₃• | 确实是甲基 |
| M–18 | 失 H₂O | 醇类脱水（注意不是失 NH₃）|
| M–28 | 失 CO 或 C₂H₄ | 含 O → 可能 CO；含 H → 可能 McLafferty |
| m/z 91 | 总是苄基 | 也可能是卓鎓离子（环庚三烯基正离子）|

## 十、例题

### 10.1 例题 1（★★★ 基础）

**题目**：某化合物 MS 显示分子离子峰 m/z = 78，M+1 峰相对强度约 6.6%。推测分子式。

**分析**：
- M = 78（偶数）→ 不含 N 或含偶数个 N
- M+1 ≈ 6.6% → 约 6 个碳原子（每个 ¹³C 贡献约 1.1%）
- 若 C₆H₆：分子量 = 72 + 6 = 78 ✓

**解答**：
$$
\ce{C6H6} \quad \text{（苯）}
$$

**反思**：
- M+1 强度 ≈ 碳数 × 1.1% 是快速估算碳数的方法
- 78 是苯的经典分子离子峰

### 10.2 例题 2（★★★★ 进阶）

**题目**：某化合物 MS 显示 M : M+2 = 1 : 1，分子离子峰 m/z = 122。推断可能的结构类型。

**分析**：
- M : M+2 = 1 : 1 → 含 **1 个 Br**（⁷⁹Br : ⁸¹Br = 1 : 1）
- M = 122，减去 Br（79）→ 剩余基团质量 = 43
- 43 可能是 C₃H₇（丙基/异丙基）或 CH₃CO（乙酰基）

**解答**：
- 可能为溴丙烷（C₃H₇Br）或溴乙酸甲酯等
- 需结合 NMR 进一步确认

**反思**：
- Br 的同位素 1:1 是最可靠的识别标志
- Cl 的 3:1 也很特征，不要混淆

## 十一、易错点

1. ❌ **忽略氮规则** → 奇数 m/z 含奇数个 N，是快速判氮的利器
2. ❌ **把同位素峰当碎片峰** → M+2 可能是同位素，不是真正的碎片
3. ❌ **醇的 M⁺• 弱就以为分子量错了** → 醇容易脱水，M⁺• 本来就弱
4. ❌ **McLafferty 重排只记 M–28** → 实际失去的烯烃取决于 γ-取代基
5. ❌ **忽略高分辨 MS 的小数部分** → HRMS 的精确质量是确定分子式的金标准

### 12.2 学生易踩的认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析口诀/方法 |
|:---|:---|:---|:---|
| "质谱就是称分子量的" | 只关注 M⁺• | 碎片峰比分子离子峰更丰富的结构信息 | "质谱是天平+锤子" |
| "M+2 总是同位素" | 没考虑双电荷 | M+2 通常是同位素，但偶尔有双电荷离子（m/2z）| 先看元素组成，再想其他可能 |
| "分子离子峰越强越稳定" | 以为峰强就是结构稳定 | 分子离子峰强度反映的是**电离后稳定性**，不是分子稳定性 | 芳环 > 烷烃 > 醇 |
| "碎片峰是随机产生的" | 以为打碎是随机的 | 碎片遵循**固定规律**（α-裂解、McLafferty 等）| "裂解有规律，不是乱飞的" |

### 12.3 入门级例题

**题目**：某化合物 MS 显示 m/z = 58（M⁺•），主要碎片峰 m/z = 43。这是什么化合物？

**预期解答路径**：
1. M = 58 → 分子量较小
2. 碎片 m/z = 43 是 C₃H₇⁺（异丙基）或 CH₃CO⁺
3. 若碎片为 CH₃CO⁺ → 母体可能是丙酮（CH₃COCH₃，M = 58）
4. 验证：丙酮 α-裂解产生 CH₃CO⁺（m/z 43）和 CH₃•

**老师可在课堂上引导的提问**：
- 学生若卡在 43：问 "43 对应什么常见碎片？"
- 学生若答错：问 "如果是丁烷，M 是多少？碎片会是什么？"

### 12.4 与现实/直觉的连接

- **类比**：质谱就像考古中的"碳十四测年+文物碎片分析"——先确定年代（分子量），再看碎片风格（结构特征）推断原物
- **反例**：表面上看，打碎东西会丢失信息；实际上，质谱的碎片规律（如 McLafferty 重排）反而能揭示隐藏的结构特征（γ-H 的位置）

## 十三、竞赛拓展

1. **软电离技术（ESI, MALDI）**：不破坏分子，适用于大分子和热不稳定化合物。竞赛中仍以 EI-MS 为主。
2. **串联质谱（MS/MS）**：先选母离子，再打碎看子离子，用于复杂混合物分析。
3. **质谱成像**：空间分辨的质谱分析，超纲但属于前沿技术。
