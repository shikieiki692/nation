# Grob 碎裂化反应（Grob Fragmentation）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[28-有机物结构的稳定性]]、[[32-有机反应机理]]、[[38-消除反应]]、[[39-重排反应]]

> ⚠️ **命名提示**：本反应在不同教材里有不同译名——ABOC 用「碎裂化」，Clayden 中文版作「Grob 断裂」，中科大教材常作「Grob 裂解」。本 KP 已在 aliases 中收录三种译名，互为指向。

---

## 一、定义

**Grob 碎裂化反应（Grob fragmentation）**：在脂肪链上 **1-位有孤对电子**、**3-位有离去基团（LG）** 的化合物，通过 **两组反式共平面**（antiperiplanar）几何排布，**协同断裂**三根 σ 键，生成 **一个阳离子片段 + 一个烯烃 + 一个离去基团** 的反应。

通式（X = O / N / S 等含孤对电子的杂原子；LG = Cl、Br、OTs、OMs、OH₂⁺ 等）：

```
       X：           ┌──────── 协同三键断裂 ────────┐
        \           ↓             ↓             ↓
        a────b────c────d────e──── LG
                           
                  ↓  反式共平面 ×2

        X⁺=a       b=c        d=e         + : LG⁻
       (亚胺/    (新生        (新生
        氧鎓)    π 键)        π 键)
```

**形式上等同于**：1-位 X 的 n 轨道 → 经 a-b → c-d → e-LG 全程反式共平面 → 同时填入两个 σ\* 反键 → 切断中间 b-c 键、推走 LG。

> 📌 **关键三件套**（ABOC §2.11 行 1996 + 行 2004）：
> 1. **1-位孤对电子**（电子推动起点）
> 2. **3-位离去基团**（电子推动终点；3-位 = "γ-LG"）
> 3. **两组反式共平面同时满足**（n→σ\*(b-c) 反式共平面 + σ(c-d)→σ\*(e-LG) 反式共平面）

## 三、核心原理

### 3.1 协同三键断裂的电子推动图（ABOC §2.11 主线）

以最简底物 H₂N-CH₂-CH₂-CH₂-Cl（实际反应需更稳定的 X 与活化的 LG，此处仅示意）为例：

```
           ↓  n→σ*(C2-C3)            ↓  σ→σ*(C3-Cl)
                                      ↓
   H₂N: ────CH₂──CH₂──CH₂──Cl
   (孤对)   C1    C2    C3
                                      
   ─→ H₂N⁺=CH₂  +  CH₂=CH₂  +  Cl⁻
      (亚胺鎓)   (乙烯)      (氯负)
```

电子流向（一笔到底）：
- 第一推：N 的孤对 → C1=N 形成（→ 亚胺鎓）；C1-C2 σ 键 → 填入 C2-C3 σ\* 反键 → C2-C3 σ 键断
- 第二推：C2-C3 σ\* 被填的同时，C2-C3 σ 键也成为下一推动的供体 → 形成新的 C2=C3 π 键 → 同时把 C3-Cl σ\* 填满 → Cl⁻ 离去

**实际是同一时刻发生**（concerted），但电子推动图通常分两笔表示。

> ⭐ **ABOC 行 2006 原话**："这样**两次孤对电子填入**碳碳键和碳氧键的反键轨道，切断中间的碳碳键，得到两个双键。"

### 3.2 两组反式共平面的几何要求

| # | 反式共平面 | 涉及轨道 | 几何含义 |
|:---:|:---|:---|:---|
| 第一组 | **n(X) ⊥ σ\*(C2-C3)** | X 孤对 ↔ 中间 C-C 反键 | X-C1-C2-C3 四原子二面角 = 180° |
| 第二组 | **σ(C2-C3) ⊥ σ\*(C3-LG)** | 中间 σ 键 ↔ LG 反键 | C2-C3-LG 三原子需 anti（对应 H-C2-C3-LG 二面角 180°）|

**几何上等价于**：
- 把 X-C1-C2-C3-C4-LG 这条链摆成 **W 形锯齿构象**（all-anti）；或者
- 在六元环底物（环己烷类）里，X 在 1-位**直立键 (axial)**，LG 在 3-位**直立键 (axial)**，两者在 1,3-双直立位置（**1,3-diaxial**）。

> 📐 **核心口诀**：**1,3-双直立 + W 锯齿 = Grob 必备**。

### 3.3 与 E2、E1cb、Wharton 重排的关系（边界）

| 反应 | 1-位 | 3-位 | 4-位 | 协同断键数 | 产物 |
|:---|:---:|:---:|:---:|:---:|:---|
| **E2** | 含 H | LG | — | 2（C-H + C-LG）| 1 烯烃 |
| **E1cb** | C-H 酸性强 | LG | — | 2（先 H 断成碳负，再赶走 LG）| 1 烯烃 |
| **Grob** | X 孤对（n 给体） | C-C | LG | **3**（C1-C2 + C2-C3 + C3-LG 中两根；保留 C1=X）| 1 阳离子 + 1 烯烃 + LG⁻ |
| **Wharton** | 烯丙位 OH | OH→C | C-Br | 3（重排版的 Grob，n 给体换成烯丙位）| 1 烯烃迁移产物 |
| **α-溴酮重排** | 邻位羰基 n | C-C | C-Br | 3（半 Pinacol 风格）| 重排环缩的酮 |

> 🔑 **Grob 是 E2 的"加长版"**——E2 切两根键、Grob 切三根键；E2 给体是 C-H σ、Grob 给体是 X 孤对；E2 端只有一根 π、Grob 端要新生两根 π。

### 3.4 1-位允许的 X 杂原子（ABOC §2.11 行 1996 表外推）

| X 杂原子 | 产物阳离子 | 稳定性 | 反应活性 |
|:---:|:---:|:---:|:---:|
| **N**（胺）| 亚胺鎓 R₂N⁺=CR'₂ | ⭐⭐⭐ | 最高（最常用） |
| **O**（醇/醚）| 氧鎓 R-O⁺=CR'₂ | ⭐⭐⭐ | 高（醇盐 RO⁻ 更高）|
| **S**（硫醚）| 硫鎓 R-S⁺=CR'₂ | ⭐⭐ | 中 |
| **C**（烯丙位负电）| 烯丙阴离子 → 烯烃 | ⭐ | Wharton 类 |
| **Sn / Si**（β-效应推广）| 锡正/硅正 | ⭐ | 用于片段化合成中  |

ABOC §2.11 例题中给出的版本是 **β-氨基醇活化为 LG**（即把 OH 转为 OMs/OTs 后碎裂化），是合成中最常见的 Grob 用法。

### 3.5 LG 端的强度阶梯

LG 越强，碎裂化越易发生，且**对几何要求的"宽容度"略放宽**（接近极端 LG 时可能滑向 E1 + 重排）：

$$\mathrm{N_2^+} > \mathrm{OTf^-} > \mathrm{OMs^-} \approx \mathrm{OTs^-} > \mathrm{Br^-} > \mathrm{Cl^-} \gg \mathrm{OH_2^+} \approx \mathrm{OH(prot)}$$

### 3.6 ABOC 例题 2.11 的"硼氢化-碎裂化"组合（行 2002-2008）

ABOC 给出的核心例题底物是「**硼氢化反应产物 → 碎裂化**」串联：
- 硼烷加成到反式十氢化萘的烯烃后，得到 **1,3-双直立的 OH-X**（B 经氧化后变 OH，作为 1-位 X）+ **3-位另一个烯烃的 LG（如 OMs）**
- 此时分子内**两个反式共平面同时满足**，加热触发协同碎裂化
- **温度依赖**：低温下因构象最小改变原理（AAOC 讨论），优先得到顺式产物；高温下硼氢化可逆性使反式产物更稳定

> ⭐ **ABOC 行 2008 原话总结**："其实，无论从上面还是下面来，得到顺式还是反式产物，**最终都能碎开**，产物是相同的。"——这是 Grob 在合成中**容错性高**的体现。

### 3.7 反极化版本（"逆 Grob"，思路扩展）

如果反过来：1-位是亲电基团（如羰基）、3-位是富电子（如 H 或 C-M），则可由**亲核试剂触发的反向碎裂化**：

```
   Nu: ── C=O (亲电)  ──CH₂──CH₂──H
                
   ─→ Nu-C-O⁻  +  CH₂=CH₂ + H⁻ (或 H⁺ 转给 O⁻)
```

属于"反极化加成 + 碎裂"思路（参见 [[反极化与亲卤取代]]），ABOC 未直接以"Grob"称之，但 §2.10.2 αβ-不饱和羰基亲核加成的某些产物有类似机理。

## 四、影响因素与重要参数

### 4.1 几何门槛（前面 §3.2 阐释，此处量化）

| 二面角偏离 180° | 反应性损失 | 备注 |
|:---:|:---:|:---|
| 0° | — | 完全反式共平面 |
| ±15° | 损失 < 10% | 实际多数体系范围 |
| ±30° | 损失 ~50% | 边界 |
| ±60°（gauche）| 损失 > 95% | 几乎不发生 |
| 90°（垂直）| 损失 ≈ 100% | 不可能 |

### 4.2 X 与 LG 的"双激活"原理

- **X 端激活**：醇 < 醚 < 胺 < 醇盐 < 氮负
- **LG 端激活**：OH < Cl < Br < OMs/OTs < N₂⁺
- **协同效应**：两端都强 → 反应可在室温自发；两端都弱 → 即使几何完美也不发生

### 4.3 溶剂极性（中等极性最佳）

| 溶剂 | 极性 (ε) | Grob 速率（相对值）| 解释 |
|:---|:---:|:---:|:---|
| 己烷 | 1.9 | 0.01 | 极性太低，不能稳定离子产物 |
| 乙醚 | 4.3 | 0.5 | 偏低 |
| THF | 7.6 | 1（基准）| 偏低-中等 |
| 乙腈 | 37 | 5 | 中等极性最佳，能溶剂化离子产物 |
| 甲醇 | 33 | 3 | 中等极性，但氢键供体可能拉走 X 孤对 → 略降 |
| 水 | 78 | 0.1 | 极性过高，水合 LG 太死，反而抑制 |

> 💡 与 [[SN1反应]] 倾向极性质子溶剂（甲醇/水）不同，Grob 的"中等极性最佳"反映了 **协同 vs 分步** 的本质差异。

### 4.4 温度（活化能 30-40 kcal/mol，高温有利）

- 室温下：仅最强激活体系（如 1-N⁻、3-OTf）能进行
- 加热（80-150°C）：覆盖大多数实际合成场景
- 微波 / 路易斯酸辅助：可在低温实现

### 4.5 立体效应（环刚性是把双刃剑）

- **刚性环系**：构象锁定 → 若刚好满足 1,3-双直立，几乎是唯一通道（高产率，高立体选择性）
- **柔性链**：可旋转到 W 锯齿构象 → 可发生但产率较低，且与 E2 竞争
- **多取代环**：若双直立位被甲基占据 → 1,3-空间排斥（A^(1,3)-strain）反而有利于碎裂化（此即"碎裂化释放张力"）

## 五、关键例题

### 5.1 例题 1（★★ 基础识别）

**题目**：给出 4 个底物，判断哪些可发生 Grob 碎裂化（写出预期产物）：

```
A. CH₃-CH(NH₂)-CH₂-CH₂-OMs      → ✓
B. CH₃-CH(NH₂)-CH(CH₃)-CH₂-OMs  → ✓ (但需柔性 W 锯齿)
C. CH₃-CH(NH₂)-CH=CH-OMs        → ✗（C2-C3 已是双键，无法形成新 π）
D. CH₃-C(=O)-CH₂-CH₂-OMs        → ✗ (1-位无孤对推 a-b 键；C=O 是受体不是给体)
```

**思路**：检查"1-位 X 孤对 + 3-位 LG + 中间 C-C 单键 + 几何可达"四要素。
- **A**：β-氨基醇磺酸酯，柔性链，可达 W 锯齿 → 给亚胺鎓 + CH₂=CH₂ + OMs⁻
- **B**：同 A 但有 CH₃ 取代，需检查空间能否旋转到 anti
- **C**：C2-C3 是 π 键，已用电子，无法再形成
- **D**：C1=O 是 π 受体（σ\* 充电），不是孤对给体；不符合"1-位孤对推 → "

### 5.2 例题 2（★★★ ABOC 例题 2.11 改编）

**题目**：反式十氢化萘衍生物 **1**（以下结构）经硼氢化-氧化-成 OMs 处理后，加热碎裂化，预测产物：

```
            (反式)
       ┌─────────┐
       │         │
   HO─C-CH₂-CH₂-C-OMs    (1,3-双直立)
       │         │
       └─────────┘
```

**思路**：
1. **几何检验**：反式十氢化萘的两个环融合处 H 是反式（顺式十氢化萘则同一侧）；1,3-双直立排布天然形成两组反式共平面 ✓
2. **电子推动**：HO: 推 → C1=O（醛/酮）→ C1-C2 σ → C2-C3 σ\* → 切 C2-C3；C2-C3 σ → C3-OMs σ\* → OMs 离开
3. **产物**：双环开环，得到一个长链含两个新双键 + OMs⁻ 离去
   - C=O 为醛/酮（取决于 C1 上原本是 H 还是 R）
   - 内部得 1 根新 C=C
   - 外部 LG 端得 1 根新 C=C

**关键判据**：反式十氢化萘构象固定（无翻转），是 Grob 发生的"理想刚性平台"；顺式十氢化萘构象柔性，会有竞争反应。

### 5.3 例题 3（★★★ ABOC 自学练习 2.11-2 类型）

**题目**：以下 3 个底物在水中溶剂解，2 个发生 Grob 碎裂化，1 个不发生，分析原因：

```
A. 反式 1-N(CH₃)₂-3-OMs-环己烷       → ✓ Grob（1,3-双直立可达）
B. 顺式 1-N(CH₃)₂-3-OMs-环己烷       → ✗（1,3-双直立不可同时达到，仅能单边）
C. 反式 1-N(CH₃)₂-3-OMs-环戊烷       → ✓ Grob（环戊烷扁，可达）
```

**思路**：
- 顺式 1,3-环己烷取代：一直立一平伏，无法同时双直立 → ✗
- 反式 1,3-环己烷取代：可同时双直立（构象 1）或同时双平伏（构象 2）→ 取双直立构象 → ✓
- 反式 1,3-环戊烷取代：环戊烷扁平度高，二面角接近 180° → ✓

**产物**（A 为例）：
```
反式 1-N(CH₃)₂-3-OMs-环己烷 → CH₂=N(CH₃)₂⁺ + CH₂=CH-CH₂-CH₂-CH₃ + OMs⁻
                                (亚胺鎓)         (1-戊烯)
```

注：开环后链状产物的双键位置由具体环原子排列决定。

### 5.4 例题 4（★★★ Holton-Taxol 改编）

**题目**：在紫杉醇的 Holton 全合成中，关键的"骨架重组"步骤将一个 6-6-4 三环系（含 1-OAc 和 3-OMs）转化为 8-6 双环系。请用 Grob 碎裂化原理解释这步的可行性，并预测可能的副反应。

**思路**：
- **正反应**：4 元环张力高（约 26 kcal/mol） → 碎裂化释放张力 → 可发生
- 1-OAc 端的羰基 O 提供孤对，3-OMs 端 LG 离去，中间 4 元环的 C-C 键断裂，扩环为 8 元环
- **副反应**：① E2（消除给一个普通烯烃，未扩环）；② SN2（OMs 直接被亲核试剂取代，几何不利）；③ 1,2-迁移（重排到不同骨架）

**为什么 Grob 优先**：4 元环张力释放（~25 kcal/mol）+ 大环扩环熵成本可接受（环张力释放占据主导）+ 反式共平面几何天然由刚性多环骨架预设。

> 📚 **参考**：Holton R. A. et al., J. Am. Chem. Soc. 1994, 116, 1597; 1599. 这是有机合成史上 Grob 碎裂化最高光的应用。

### 5.5 例题 5（★ Wharton 重排）

**题目**：α,β-环氧酮 + 肼试剂 → 烯丙醇（Wharton 重排），用 Grob 框架解释：

```
       O    O                     OH
       ‖    ‖                     │
       C────C─CH─R    ─→        HC=C─CH─R
       │  ╲ │                     │       
       O    NHNH₂                 (烯丙醇)
        epoxide
                      经过 1,2-二醇 → 反 H₂O 形成 H₂N-NH-OH ↔ N₂ 离去
```

**思路**：肼把酮变成腙 → 重排为偶氮 → β-碳上的羟基（环氧打开后）作 LG → N=N⁻ 提供孤对 → 协同切断 → 生成 N₂ + 烯丙醇。框架上是 1-N 孤对、3-OH/O⁻ LG 的 Grob。

---

## 六、判定题型

### 6.1 是否为 Grob 反应？（识别题）

判别四要素同时满足：
1. **1-位有孤对**（X = N/O/S 等）✓
2. **3-位有 LG**（含离去基团或可活化为 LG）✓
3. **1,3 位之间有 σ 键链**（C-C 单键，非 π 键）✓
4. **两组反式共平面几何可达**（环刚性+柔性链 W 锯齿都行）✓

**陷阱**：
- 1,2-位有 LG（β-LG）→ 是 E2，不是 Grob
- 1,4-位有 LG（δ-LG）→ 是更长程的"远 Grob"，需要更复杂的协同（罕见）
- 1-位是孤对但 3-位被 π 键阻断 → 不可发生

### 6.2 Grob 与 E2 / SN2 / E1cb 的竞争

```
                    1,3-双直立? → Grob
                    │
   底物有 1-X + 3-LG  ─┤  几何不达 → SN2 (LG 端) 或 E2 (含 H 时)
                    │
                    └→ 强酸/强碱 → E1cb (从 X 端先成碳负)
```

### 6.3 产物预测（开环+多双键）

公式（n 元环 → n-1 长度链状产物 + LG⁻）：

$$\text{n 元环 (1-X, 3-LG)} \xrightarrow{\text{Grob}} (n-1) \text{-原子链状产物 + LG}^-$$

每发生一次 Grob，**多生 2 个双键 + 切 1 根 C-C 键 + 离去 1 个 LG**。

### 6.4 立体化学（构型保持/反转/外消旋？）

Grob 是协同反应，立体化学严格由几何决定：
- **C1-C2 端**：X⁺=C1 是 sp² → 平面化，C1 立体信息丢失（若有）
- **C3-C4 端**：新生成 C2=C3 或 C3=C4 双键 → C3 sp² → 立体信息丢失
- **C4-LG 端**：Walden 反式离去（与 SN2 类似但仅"形式上"，不真的 Walden）

**关键**：Grob 给出的烯烃**几何（E/Z）由两组反式共平面唯一锁定** → 几乎总是高度立体选择性。

## 七、易错点与陷阱

1. ❌ **混淆"1-位"和"3-位"的电子角色**：1-位是给体（孤对），3-位是受体（LG）；反过来不是 Grob（除非反极化版）。
2. ❌ **忘记两组反式共平面**：很多人只记得 1 组（X-C1-C2-C3 反式），漏了 σ(C2-C3) → σ\*(C3-LG) 这第二组；导致顺式 1,3-环己烷误判为可碎裂化。
3. ❌ **把 Grob 和 1,2-迁移混了**：1,2-迁移（W-M、Pinacol）只切 1 根 C-C 键且重组，Grob 切 3 根键且不重组。
4. ❌ **底物 1-OH 误以为不需要活化**：纯 OH（不带正电、未质子化）作 X 的话推力很弱；通常需要 OH⁻ → 醇盐或 R-O-推电子基团辅助。
5. ❌ **C2-C3 可以是双键**：错。C2-C3 必须是 σ 键，因为该键的 σ\* 是被填的反键；如果已经是 π，就没有 σ\* 可填了。
6. ❌ **3-位 LG 必须是 sp³ 碳上**：sp² 碳上的 LG（如乙烯基卤）几何上不允许形成所需反式共平面。
7. ❌ **想当然套到顺式十氢化萘**：顺式十氢化萘构象柔性大，会**竞争 E2**；只有反式十氢化萘是"理想刚性平台"。
8. ❌ **温度低未必产率高**：Grob 活化能高（30-40 kcal/mol），低温反应慢；若需立体专一性，加压加热常更好。

## 八、与其他知识点的关联

- **几何祖先**：[[反式共平面]]——Grob 是反式共平面应用的"双重升级版"（需要两组而非一组）
- **反应家族**：[[消除反应]]、[[E2反应]]、[[E1cb反应]]——Grob 可视为"加长版 E2"（切 3 根键而非 2 根）
- **协同思想**：[[周环反应]]——Grob 不是周环反应（不涉及环过渡态），但电子流是协同的；与 Cope/Claisen 重排的"协同"机理思想同源
- **碳正离子**：[[1,2-迁移与重排]]、[[碳正离子]]——Grob 可在某些条件下"漏"成碳正离子+E1，要看是否打"协同"（取决于 LG 强度和溶剂）
- **立体电子效应**：[[端基效应]]、[[给体-受体相互作用]]——同属"n→σ\* 推动"家族
- **类比反应**：[[SN1反应]]、[[SN2反应]]——Grob 与 SN2 在"反式协同"上同源；与 SN1 在"3-位 LG 离去"上同源
- **应用合成**：Holton-Taxol 全合成、Wharton 重排、Eschenmoser-Tanabe 碎裂化、α-溴酮重排
- **特殊变体**：Wharton 重排（1-OH/烯丙位 + 3-Br）、Eschenmoser-Tanabe（α,β-环氧酮 + 磺酰肼）、Beckmann 重排（α-肟）

## 九、特殊变体

### 9.1 Wharton 重排（α,β-环氧酮 + 肼）

α,β-环氧酮 + H₂N-NHTs → 烯丙醇。机理：肼对酮加成 → 腙 → 环氧重排为 β-OH → 偶氮 N=N 推走 → β-OH 作 LG → 协同碎裂化。

### 9.2 Eschenmoser-Tanabe 碎裂化（α,β-环氧酮 + 磺酰肼）

底物：α,β-环氧酮 + 磺酰肼（如 ArSO₂NHNH₂）。产物：炔烃 + 醛酮。机理：碎裂化生成炔烃片段 + 羰基片段。在天然产物合成中常用作"开环+生成炔基"的关键步。

### 9.3 Beckmann 重排（α-肟酸催化）

α-肟在酸催化下 → 酰胺。可看作"修饰版 Grob"——OH 走，N 上孤对推动 C-C 断+C-N 形成。

### 9.4 反式十氢化萘的 Grob（ABOC 例题 2.11）

ABOC 主例题。核心：**1,3-双直立锁定**，硼氢化-氧化-甲磺酸化序列产生 1-OH 与 3-OMs 双直立 → 协同碎裂化得到 8 元环或开链产物。

### 9.5 双 Grob（连续两次碎裂化）

某些超张力多环底物（如立方烷 cubane 衍生物）在适当 X-LG 取代下可发生**两次连续 Grob**：第一次切环成环戊烯-烯丙-LG → 第二次再切→开链产物。罕见但有报道。

## 十、解题模板

### 10.1 Grob 反应识别 SOP

```
Step 1: 标 1,3 位      ─→ 找 1-位 X 孤对 + 3-位 LG
Step 2: 检查中间 σ 键  ─→ C1-C2-C3 是单键链？(若 π 阻断 → 失败)
Step 3: 评估几何       ─→ 两组反式共平面可达？(刚性环锁定 vs 柔性链旋转)
Step 4: 电子推动写出   ─→ 一笔到底：n → σ*(b-c)，σ(c-d) → σ*(e-LG)
Step 5: 写产物         ─→ X⁺=C1 (鎓离子) + C2=C3 (新烯) + 链段-C4=C5 (再新烯) + LG⁻
                          (依底物结构，可能只生成 1 根新双键 + 1 根新 sp² 鎓)
```

### 10.2 Grob 与 E2 / SN2 竞争判别

| 情况 | 优先 | 原因 |
|:---|:---:|:---|
| 1-X 给体强 + 3-LG 强 + 几何完美 | **Grob** | 协同稳定 |
| 1-X 给体弱 + 2-H + 3-LG 强 | E2 | 双键端有 C-H 给体 |
| 几何不利 + LG 强 | SN1/E1 | 离子化先行 |
| 几何不利 + LG 弱 | 不反应 | 全部失活 |

### 12.2 常见认知误区（7 大）

| # | 误区 | 真相 |
|:---:|:---|:---|
| 1 | "Grob 是 E2 的复杂版" | ✗ Grob 切 3 根键而非 2 根；1-位是 X 孤对而非 C-H σ |
| 2 | "只要满足 1,3-双直立就一定碎裂化" | ✗ 还需 X 给体足够强 + LG 足够强 |
| 3 | "顺式十氢化萘也行" | ✗ 顺式构象柔性，竞争 E2；反式才是理想平台 |
| 4 | "C2-C3 可以是 π 键" | ✗ 必须 σ；π 键无 σ\* 可填 |
| 5 | "Grob 总能进行，只要看到 1-NH₂ 和 3-OMs" | ✗ 几何不达就不发生 |
| 6 | "产物就是 1 根烯烃 + 1 个 LG⁻" | ✗ 是 2 根烯烃 + 1 个鎓离子 + LG⁻（共 4 个产物片段）|
| 7 | "Grob 要在低温下进行" | ✗ Grob 活化能高，反而需加热（80-150°C）|

### 12.3 入门例题（"3 步搞定 Grob"）

**底物**：H₂N-CH(CH₃)-CH₂-CH(CH₃)-OMs

**Step 1**：标 1,3 位
- 1-位 = N (NH₂) ✓ 有孤对
- 3-位 = C-OMs ✓ 有 LG

**Step 2**：检查几何
- 链状底物，可旋转到 W 锯齿构象 ✓

**Step 3**：电子推动 + 写产物
```
H₂N: ─CH(CH₃)─CH₂─CH(CH₃)─OMs
 ↓                ↑
 ↘─→ C1=N         ↘─→ C3=C2
                  
─→ HN=CHCH₃ + CH₃-CH=CH₂ + OMs⁻
   (亚胺)        (丙烯)      (甲磺酸根)
```

**验证**：双键位置正确？亚胺无正电荷（HN= 不是 H₂N⁺=）？答：因为反应消耗一个 H（实际机理中 N 端有去质子步），产物可表示为亚胺或亚胺鎓视环境而定。

### 12.4 现实类比：组合家具的"一键拆解"

想象一套组合家具用 3 个连接螺丝固定（A-B 键、B-C 键、C-D 键）。如果三个螺丝都能**同时一齐松开**（协同），则整套家具瞬间散开成 3 个部分。Grob 碎裂化正是这种"协同三键松开"——
- **第 1 个螺丝**（C1-C2 σ）由"X 孤对"作为推杆（电子云推过去）
- **第 2 个螺丝**（C2-C3 σ）随之松动，转为新生 C2=C3 双键
- **第 3 个螺丝**（C3-LG σ）被推离，LG 离去

整个过程一气呵成（concerted），需要的"使用门槛"是**几何上的双重对齐**——所有螺丝要在同一直线上（两组反式共平面），就像组合家具的螺丝必须沿轴线对齐才能拧动。

**与 E2 的对比**：E2 像拆 2 颗螺丝的折叠桌（一边的 H 推 + 一边的 LG 走）；Grob 像拆 3 颗螺丝的伸缩床。后者难，但拆完得到 3 个部分。

## 十三、📚 竞赛拓展

### 13.1 与周环反应的边界

Grob 不是周环反应（不涉及环过渡态），但与 [[反式共平面]] / [[消除反应]] 一起构成**"协同断键"思维家族**。Cope 和 Claisen 重排是 [3,3]σ 重排，过渡态是 6 元环；Grob 是开链协同，无环过渡态。

### 13.2 张力驱动的 Grob

中环（8-12 元环）合成的"逆合成神剪刀"——把目标中环识别为"前体多环系经 Grob 碎裂化"，常用于：
- 紫杉醇（Taxol）的 8 元环
- 环辛二烯的 cis,cis-1,5- 异构体
- 大环内酯（macrolides）的中环骨架

**张力释放估算**：每开 1 个 4 元环约释放 26 kcal/mol；3 元环约 28；5 元环约 6。所以 4 元环和 3 元环开环驱动 Grob 最强。

### 13.3 阴离子加速版（α-LG 的协同氧鎓）

如果 1-位换成 RO⁻（醇负离子）而非 RO-H，则给体能力大幅提升，可在室温甚至零下完成 Grob：
$$\text{RO}^-\text{-CH}_2\text{-CH}_2\text{-CH}_2\text{-OMs} \xrightarrow[\text{r.t.}]{\text{KH/THF}} \text{RO}^+ = \text{CH}_2 + \text{CH}_2=\text{CH}_2 + \text{OMs}^-$$

### 13.4 双向版（"reversible Grob"）

某些 Grob 反应可逆：如果两侧都成离子对而非分离，则碎裂化产物可以重新合环。这是"动力学锁"在合成中的考虑——通过加水、加碱"淬灭"两端，让平衡偏向碎裂化产物。

### 13.5 立体专一性（Stereospecificity）

Grob 给出的 C2=C3 双键的几何构型（E 或 Z）严格由起始底物的立体化学决定：
- 顺式（threo）-1,3-取代 → E-烯烃
- 反式（erythro）-1,3-取代 → Z-烯烃

这是因为协同机理无中间体，完全由轨道对齐方式决定。

### 13.6 与异核 Grob（"杂 Grob"）

如果 a-b 键不是 C-C 而是 C-X（X = N、O、Si、Sn）：
- C1-N 可断（C-N → 阳离子 + 烯）—— "去 N Grob"
- C-Si 可断（β-Si 效应推广）—— "脱硅 Grob"
- C-Sn 可断 —— Stille 反应的"消除分支"

是合成中"功能基迁移"的关键工具。

### 13.7 与 Eschenmoser-Tanabe（特别处理）

底物 α,β-环氧酮 + 磺酰肼。机理 6 步：
1. 肼对酮加成成腙
2. 1,2-质子转移 → 腙稳定形式
3. 环氧 → β-OTs（活化）
4. N=N⁻ 形成（强 X 给体）
5. **协同碎裂化**：N₂ 离去 + 新 C=C + 新 C=O（其实是 C=C 旁边的 C-OH 经历 anomeric 重整）
6. 互变 → 终产物（含炔基）

此反应的关键创新：**炔烃和羰基同时生成**——是合成中的"开环+变官能团"利器。

### 13.8 Grob 在天然产物合成的"逆合成剪刀"思维

```
   逆合成思路：看到中环（8-12 元）→ 反推为多环 1,3-取代体 → 用 Grob 切开
   
   目标：环辛二烯
   ↓ retro-Grob
   1,3-(LG, X)-双环 [4.4.0] 系
   ↓ retro-硼氢化
   反式十氢化萘 + B₂H₆
   ↓ 来自...
   萘 + H₂ + 拆 OMs/X
```

### 13.9 反极化 Grob（"逆 Grob"）

α,β-不饱和羰基 + 软亲核试剂的 1,4-加成，从某种意义上是"反向 Grob"：
- 通常 Grob：n → σ\* → 切键（切的方向是 LG 端）
- 反向：Nu → π\* → 推走某个 LG（切的方向反过来）

ABOC §2.10.2 7×7 表（硬软亲核）实际上隐含了这个思路。

### 13.10 Grob 在"环张力释放炸药"中的应用

立方烷（cubane）经 Grob 碎裂化释放 ~166 kcal/mol 张力 → 用作高能材料合成中间体。

### 13.11 Clayden Ch.36 离子型碎片化视角（补充）

**Grob 碎裂化 vs Clayden 离子型碎片化的对比**：

| 维度 | Grob 碎裂化（ABOC）| Clayden 离子型碎片化 |
|:---|:---|:---|
| **机理类型** | 协同（concerted）| 分步（stepwise）或协同 |
| **电子源** | X 孤对电子（n→σ*）| 孤对电子或 σ 键电子 |
| **电子槽** | LG（离去基团）| 质子化的 OH、OTs、N₂ 等 |
| **断裂键数** | 3 根 σ 键协同断裂 | 通常 1-2 根 C-C 键断裂 |
| **立体化学** | 两组反式共平面 | 反叠式（anti-periplanar）|
| **产物** | 鎓离子 + 烯烃 + LG⁻ | 醛/酮 + 烯烃 + 其他碎片 |
| **典型底物** | 1-X-3-LG 链状/环状 | 1,2-二醇、环氧酮、氨基醇等 |

**Clayden 的 1,2,3,4 碎片化模式**：
- **1 = 电子源**（推）：孤对电子（OH、OR、NR₂）或 σ 键电子。
- **2-3 = 断裂的 C-C 键**。
- **4 = 电子槽**（拉）：离去基（H₂O⁺、OTs、N₂ 等）。
- 电子源和电子槽**相间**（而非相邻）→ 碎片化而非重排。

**Eschenmoser 碎片化作为离子型碎片化案例**：
- 底物：α,β-不饱和酮的环氧 + 对甲苯磺酰肼。
- 产物：炔烃 + 酮。
- 机理：1,2,3,4 碎片化——羟基"推" + Ts⁻/N₂"拉"。
- 所有断裂的键彼此平行，在两根双键两侧反叠。

**竞赛中的区分**：
- Grob 碎裂化强调**协同**和**两组反式共平面**——是消除反应的"加长版"。
- Clayden 离子型碎片化强调**推-拉电子**和**C-C 键极化**——是重排反应的"对立面"。
- 两者在立体化学要求上同源（都需要 anti-periplanar 几何）。
