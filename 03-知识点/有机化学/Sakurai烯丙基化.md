# Sakurai 烯丙基化（Hosomi-Sakurai Reaction）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[28-有机物结构的稳定性]]、[[32-有机反应机理]]、[[13-元素化学]]

## 一、定义

**Sakurai 烯丙基化**（Hosomi-Sakurai allylation, 1976）：在 **Lewis 酸催化**下，**烯丙基硅烷**（allylsilane, $\mathrm{R_3Si\!-\!CH_2\!-\!CH\!=\!CH_2}$）作为亲核体与亲电体（醛 / 酮 / 烷基卤 / α,β-不饱和羰基化合物等）反应，生成**新的 C–C 键**并伴随 **1,3-烯丙基迁移**（双键位置发生改变）和 $\mathrm{R_3Si^+}$ 的离去：

$$\mathrm{R_3Si\!-\!CH_2\!-\!CH\!=\!CH_2 \;+\; R'CHO \;\xrightarrow{\text{Lewis acid}}\; R'\!-\!CH(OH)\!-\!CH_2\!-\!CH\!=\!CH_2 \;+\; R_3SiX}$$

> **核心驱动力**：[[β-硅基效应]] — Lewis 酸活化亲电体后，烯丙基硅烷以 C=C 双键末端进攻，形成的碳正离子 **β 位恰好是 Si**，被 σ(C-Si) 强超共轭稳定（[[给体-受体相互作用]] §4.5），随后 $\mathrm{R_3Si^+}$ 由亲核体（卤负、烷氧负）抓走，整个过程 1,3-迁移**位置等价**于"烯丙基"被插入。

## 三、核心原理

### 3.1 总机理（标准三步）

```
        R₃Si           Lewis acid (M)        R₃Si  ⊕         (β-Si 稳定)
         |               活化亲电体              |   ⊖
   CH₂=CH-CH₂  +  R'CHO ─────────────→  CH₂—CH-CH₂—CH(OM)R'
                                          │
                                          β-硅基碳正离子
                                          │
                              ─────────→  CH₂=CH-CH₂—CH(OH)R' + R₃SiX
                                          (脱 R₃Si⁺，1,3-烯丙基迁移)
```

**三步详细分析**：

1. **Lewis 酸活化**：Lewis 酸（如 $\mathrm{TiCl_4, SnCl_4, BF_3 \cdot OEt_2}$）配位醛/酮的羰基氧 → 加大 C=O 极化，增强 C 的亲电性
2. **亲核进攻**：烯丙基硅烷以 **C=C 双键的末端 C（远离 Si 的一端）**作为亲核中心，攻击醛 C → 形成新的 C-C 键，**β 位的 Si** 稳定生成的碳正离子
3. **β-硅基离去**：阴离子（$\mathrm{Cl^-, F^-, OR^-}$）抓走 $\mathrm{R_3Si^+}$ → C=C 双键在原 Si 一端重新形成

> **关键观察**：起始烯丙基硅烷中 C=C 在"中间-末端"（离 Si 远）；产物中 C=C 在"中间-Si 端"（离原 Si 位）→ **1,3-烯丙基迁移**。

### 3.2 区域选择性的来源（为什么 β-硅而不是 α-硅）

烯丙基硅烷有两个潜在的进攻位点：
- **α 端**（连 Si 的 C）：进攻后 C+ 位于中间，β 位是 H（普通超共轭）→ 不显著稳定化
- **γ 端**（远 Si 的 C）：进攻后 C+ 位于中间-Si 位，**β 位是 Si**（[[β-硅基效应]] 加速 10²-10⁴）→ **唯一可行路径**

> 这就是为什么烯丙基硅烷反应**总是从远 Si 端开始**——β-硅基效应锁死了区域选择性。

### 3.3 1,3-烯丙基迁移的来源

```
   起始：    Si-Cα-Cβ=Cγ (Si 在 α，C=C 在 β-γ)

   亲电进攻 γ：  Si-Cα-Cβ⁺-Cγ-E
                       β-Si 稳定

   Si 离去：    Si⁻ + Cα=Cβ-Cγ-E   (C=C 在 α-β 位)

   总观察：双键从 β-γ 位 → α-β 位（净迁移 1,3）
```

> 这是与"普通烯丙位反应"截然不同的特征——双键**必然移位**。

## 四、关键结论

### 4.1 烯丙基硅烷与各类亲电体的反应矩阵

| 亲电体 | 催化条件 | 主产物 | 备注 |
|:---|:---|:---|:---|
| 醛 RCHO | TiCl₄ / BF₃·OEt₂ | 高烯丙醇（homoallyl alcohol） | 经典 Sakurai 醇 |
| 酮 R₂CO | TiCl₄（过量需要）| 三级烯丙醇 | 比醛慢 |
| 酰氯 RCOX | Lewis acid | 高烯丙基酮 | "Sakurai 酰化" |
| 烷基卤 R'X | TiCl₄（叔/烯丙/苄基 X）| 烯丙基烷烃 | 不能用伯卤 |
| α,β-不饱和酮 | TiCl₄ / TMSOTf | 1,4-加合产物（共轭加成）| Mukaiyama 型 |
| 缩醛/氧鎓盐 | TiCl₄ / BF₃ | 高烯丙基醚 | 链增长 |
| 亚胺 RCH=NR' | Lewis acid | 高烯丙基胺 | aza-Sakurai |

### 4.2 烯丙基硅烷的"反义性"（umpolung）特点

- 普通烯丙位（如 R-CH₂-CH=CH₂）作为亲核体时，从 **α 端**进攻
- 烯丙基硅烷由于 β-硅基效应，从 **γ 端**进攻 → 形式上类似 [[烯丙基金属]]（如 allyl-MgBr、allyl-SnBu₃）

> Sakurai 烯丙基化常与 [[烯丙基锡试剂]] 一起被归为"烯丙基-3 位反应"族。

### 4.3 与其他烯丙基化方法的对比

| 方法 | 中间体 | 选择性 | 优劣 |
|:---|:---|:---|:---|
| **Sakurai 烯丙基化** | β-Si 稳定的 sp³ 碳正离子 | γ-进攻 → 1,3-迁移 | 温和、宽容、易控立体 |
| Allyl-Mg / Allyl-Li | sp³ 阴离子 | α-进攻（直接） | 快但官能团容忍度差 |
| Allyl-Sn | 类似 Si 的 β 效应 | γ-进攻 | 选择性更好但毒性高 |
| Allyl-B / Allyl-Bpin（Roush）| 闭环过渡态 | 高度立体选择 | 不对称合成首选 |

### 4.4 立体选择性："Z 烯型"过渡态

> Sakurai 反应虽然**经过开链碳正离子**而非闭环过渡态，但通过精心设计取代基（如 Crotyl-SiMe₃，γ-甲基烯丙基硅烷），仍可实现**高 anti / syn 比**：
> - **γ-取代烯丙基硅烷 → anti 选择性**（开链过渡态偏好"反式椅式"排布）
> - 与 Roush 烯丙基硼烷相比，Sakurai 选择性较低但官能团容忍度更高

## 五、常见分类或情形

### 5.1 按亲电体分类

| 分类 | 例 | 重点 |
|:---|:---|:---|
| **Sakurai-Hosomi 醇化** | 醛/酮 + 烯丙基硅烷 | 最经典，TiCl₄ 催化 |
| **Sakurai 共轭加成** | α,β-不饱和羰基 + 烯丙基硅烷 | TMSOTf / Lewis 酸 |
| **Sakurai 酰化** | 酰氯 + 烯丙基硅烷 | 弱亲电体需要更强 LA |
| **aza-Sakurai** | 亚胺/亚胺盐 + 烯丙基硅烷 | 对手性亚胺立体选择性高 |
| **Mukaiyama-Aldol（相关）** | TMS-烯醇醚 + 醛 | 不是严格 Sakurai 但同源机理 |

### 5.2 按催化体系分类

| 催化剂类型 | 代表 | 特点 |
|:---|:---|:---|
| Lewis 酸（强）| $\mathrm{TiCl_4, SnCl_4}$ | 化学计量，需冷至 -78 °C |
| Lewis 酸（中）| $\mathrm{BF_3 \cdot OEt_2}$ | 较温和，常用于醛 |
| Brønsted 酸 | TfOH | 偶尔用于活化弱亲电体 |
| 氟离子催化 | $\mathrm{F^-/TBAF}$ | 五配位硅 → 烯丙基硅醚活化 |
| **手性 Lewis 碱**（Denmark）| 手性磷酰胺 | 不对称合成首选 |
| **手性 Lewis 酸**（CBS 类）| 手性 BINOL/Ti | 不对称烯丙基化 |

### 5.3 关于"中性"烯丙基硅烷与"激活"烯丙基硅烷

| 类型 | 例 | 反应活性 | 备注 |
|:---|:---|:---:|:---|
| 中性烯丙基硅烷 | $\mathrm{Me_3Si\!-\!CH_2\!-\!CH\!=\!CH_2}$ | 低 | 需强 Lewis 酸 |
| 烯丙基三氟硅烷 | $\mathrm{F_3Si\!-\!CH_2\!-\!CH\!=\!CH_2}$ | 高 | 不需 LA，可直接反应 |
| 烯丙基硅醚 | $\mathrm{R_3Si\!-\!OR'\!-\!CH_2\!-\!CH\!=\!CH_2}$ | 中 | 五配位中间体活化 |

## 六、适用条件与限制

### 适用条件
- ✓ 烯丙基硅烷需有完整的 Si-CH₂-CH=CH₂ 骨架
- ✓ 亲电体能被 Lewis 酸活化（醛/酮/烯酮 OK；酸根、酰胺通常不行）
- ✓ 反应温度低（-78 °C 至室温），避免烯丙基硅烷"自分解"
- ✓ 离去基团能稳定 $\mathrm{R_3Si^+}$（如 Cl⁻、F⁻ 形成强 Si-X 键）

### 不适用
- ✗ **碱性条件**：β-硅碳正离子不能在碱中存在
- ✗ **直接亲电进攻 α 端**：违反 β-硅基效应规律，几乎不发生
- ✗ **无 Lewis 酸的弱亲电体**：醛 + Allyl-SiMe₃ 在中性条件下基本不反应
- ✗ **手性硅烷无显著立体诱导**（除非引入 Roush 型手性辅助）

## 七、常见比较与易混点

### 1. Sakurai 反应 vs Wittig 反应

| 对比项 | Sakurai | Wittig |
|:---|:---|:---|
| 试剂 | 烯丙基硅烷 | 磷叶立德 |
| 中间体 | β-硅碳正离子（开链）| 偶联氧鎓盐（闭环）|
| 产物双键位置 | 1,3-烯丙基迁移（不在原位）| 与磷连接的 C 位 |
| 反应条件 | Lewis 酸催化 | 有时需要强碱 |

### 2. Sakurai 醇化 vs 烯丙基-Mg/Li 加成

| 对比项 | Sakurai-Hosomi | Allyl-Mg/Li |
|:---|:---|:---|
| 进攻位点 | **γ 端**（远 Si） | **α 端**（直接连 M）|
| 双键位置 | **1,3-迁移** | 不迁移 |
| 立体选择性 | Lewis 酸控制 | 较弱 |
| 对官能团容忍度 | 高 | 低 |

### 3. Sakurai 反应 vs Mukaiyama-Aldol

> 两者都用 TMS-X 试剂在 Lewis 酸催化下与醛反应：

| 对比项 | Sakurai | Mukaiyama |
|:---|:---|:---|
| 试剂 | 烯丙基硅烷 | TMS-烯醇醚 |
| 离去 Si | 与亲核 C **β** 位 | 与亲核 C **γ** 位 |
| 形式上 | 烯丙基化 | 醛醇缩合（aldol） |

> 尽管反应分类不同，**两者本质都是"β-Si 稳定的 SN1' 型"机理**——同源。

## 十、例题

### 例题 1（ABOC 自学练习 1.3.1-2 改编）

**题目**：写出以下反应的电子推动机理：

$$\mathrm{(CH_3)_3Si\!-\!CH_2\!-\!CH\!=\!CH_2 \;+\; PhCHO \;\xrightarrow{\mathrm{TiCl_4}}\; ?}$$

**分析**：
- 烯丙基硅烷 $\mathrm{(CH_3)_3Si\!-\!CH_2\!-\!CH\!=\!CH_2}$ 是亲核体；γ 端是 $\mathrm{=CH_2}$
- 醛 PhCHO 经 TiCl₄ 活化变为更强亲电体
- 进攻位点：γ 端（CH=CH₂ 末端）→ 形成 β-硅碳正离子 → β-Si 离去 → 双键位置改变

**解答**：

```
                        TiCl₄
   PhCHO  +  TiCl₄  ──────→  Ph-CH=O⁺-TiCl₄⁻  (活化)

        Me₃Si        Me₃Si    ⊕
          |            |    
   CH₂=CH-CH₂  +  Ph-CH=O⁺-TiCl₄⁻ → CH₂-CH-CH₂-CH(OTiCl₄)Ph
                                         (β-Si 碳正离子)

   Cl⁻ 抓走 Si:  CH₂=CH-CH₂-CH(OH)Ph + Me₃SiCl + TiCl₃(OH)
                          ↑
                      (1,3-迁移：双键从 β-γ → α-β)
```

**最终产物**：$\mathrm{Ph\!-\!CH(OH)\!-\!CH_2\!-\!CH\!=\!CH_2}$（1-苯基-3-丁烯-1-醇）

**反思**：此题展示了 Sakurai 反应的所有关键特征：① Lewis 酸活化羰基 ② 烯丙基硅烷 γ 端进攻 ③ β-Si 稳定中间体 ④ 1,3-烯丙基迁移到产物。

### 例题 2（APiIC Review Problem 4.8 类型 - 分子内 Sakurai）

**题目**：写出以下分子内 Sakurai 反应的机理：

$$\text{含烯丙基硅烷与 α,β-不饱和酮的桥连分子，在 Lewis acid 下闭环}$$

**分析**：
- 这是分子内共轭加成型 Sakurai
- 烯丙基硅烷 γ 端进攻 α,β-不饱和酮的 β 位（共轭加成）
- 形成 β-Si 稳定的烯醇负离子 + 已闭环
- β-Si 离去 → 烯酮回复 / 烯醇质子化

**解答（机理流程）**：

1. Lewis 酸活化烯酮 → 烯酮 LUMO 降低
2. 分子内烯丙基硅烷 γ-C 进攻烯酮 β-C → 形成 5-/6-元环 + β-Si 烯醇负离子
3. β-Si 离去（失去 R₃Si⁺）→ 烯酮 α 位双键，产物为环烷酮（双键消失）

**反思**：分子内 Sakurai 是合成 5-、6-元环的通用方法；β-Si 离去使其相对于普通分子内 Michael 加成有更好的位置选择性。

### 例题 3（区域选择性辨别）

**题目**：以下反应的产物中，烯丙基双键最终位于哪两个 C 之间？

$$\mathrm{Ph_3Si\!-\!CH(CH_3)\!-\!CH\!=\!CH_2 \;+\; CH_3CHO \;\xrightarrow{\mathrm{BF_3 \cdot OEt_2}}\; ?}$$

**分析**：
- 烯丙基硅烷的 α-C 上有取代（CH₃），γ-C 是 $\mathrm{=CH_2}$
- 进攻位点：γ-C（远 Si 端） → 形成 β-Si 碳正离子（α-C 上仍有 Si + CH₃）
- β-Si 离去 → 双键位于 α-C ↔ β-C 之间（即原烯丙基硅烷的"内位"）

**解答**：
- 进攻产生 $\mathrm{Ph_3Si\!-\!CH(CH_3)\!-\!CH^+\!-\!CH_2\!-\!CH(OH)CH_3}$
- 失去 Ph₃Si⁺ → $\mathrm{CH_3\!-\!CH\!=\!CH\!-\!CH_2\!-\!CH(OH)CH_3}$
- **烯丙基双键位于 C2-C3（原 α-β 位）**，含一个甲基取代
- 起始烯丙基双键在 C3-C4（原 β-γ 位），CH=CH₂ → 净 1,3-迁移成功

**反思**：当 α 位有手性中心时，迁移后双键也会"带走"该立体信息（前提是在反应过程中不被破坏）。

## 十一、易错点

| 错误 | 原因 | 纠正 |
|:---|:---|:---|
| 写产物时漏掉 1,3-迁移 | 把它当成普通烯丙基亲核加成 | **β-Si 必须离去** → 双键**不可能**在原位 |
| 把进攻位点弄错（α 而不是 γ）| 直觉"靠近金属的 C 是亲核中心"| Si 不是金属——**γ 端**才是真正的亲核中心 |
| 忘记加 Lewis 酸 | 觉得亲电体已经很活泼 | 中性烯丙基硅烷活性低，**几乎所有 Sakurai 都需 LA** |
| 试图在碱性条件做 Sakurai | 套用其他烯丙基化的"碱促"思路 | β-Si 碳正离子在碱中不能存在 → 反应**只能在酸或 LA 下** |
| 与 Wittig / Aldol 混淆 | 都是 C=C 形成反应 | Sakurai 经过**碳正离子**；Wittig 经过 betaine；Aldol 经过烯醇 |
| 用伯烷基卤做 Sakurai | 不知道 R'X 的限制 | **伯卤几乎不反应**；只能用叔/烯丙基/苄基卤（生成稳定碳正离子）|

## 十二、🎯 教学视角（v1.1 新增，备课/答疑必填）

> 本段是给学生用的"软包装"，不是给同行专家看的硬核机理。
> 由 [[Agent提示词-知识库使用]] Agent 10（备课）直接消费。

### 12.1 学习路径

- **学习本 KP 前，需要先掌握**：
  - [[β-硅基效应]]（核心驱动力——必须先理解 β-Si 为什么稳定 C+）
  - [[给体-受体相互作用]]（ABOC 框架，看懂 σ(C-Si) → p(C+) 的本质）
  - [[Lewis酸]]（理解为什么醛要被 Lewis 酸激活）
- **学完本 KP 后，自然过渡到**：
  - [[Peterson烯化反应]]（β-Si 离去型反应的"消除版本"）
  - [[Mukaiyama醛缩反应]]（同样依赖 TMS-X 的 β-Si 类机理）

### 12.2 学生易踩的认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析口诀/方法 |
|:---|:---|:---|:---|
| "烯丙基硅烷与醛反应，进攻位点是 α-C（连 Si 那个 C）"| 直觉"金属附近 = 反应中心" | **必须从 γ 端进攻**（远 Si 端），因为只有这样形成的 C+ 才有 β-Si 稳定 | "看双键末端，离 Si 远的那个 C 才是真亲核中心" |
| "Sakurai 反应像 Mg-allyl 一样，双键不动" | 以为所有烯丙基化反应都同源 | 必然伴随 **1,3-烯丙基迁移**（双键从 β-γ → α-β）；这是 β-Si 必须离去的几何后果 | "Si 一走，双键必移" |
| "Sakurai 不需要 Lewis 酸（烯丙基硅烷已经活泼）" | 看作"普通烯丙基试剂" | 中性烯丙基硅烷在没有 LA 时**几乎不反应**；醛活化是关键 | "三必备：烯丙基硅、亲电、Lewis 酸" |
| "反应过程中 Si⁺ 自由游离" | 把"Si 离去"误读为"独立离子" | $\mathrm{R_3Si^+}$ 极不稳定，离去时**必须被亲核体（Cl⁻ / F⁻ / OR⁻）抓走** | "Si 不能光着出门，必须有人接走" |
| "用乙基卤（伯卤）做 Sakurai 烷基化" | 套用其他取代反应思路 | 伯卤产生的伯碳正离子不稳定；**只能用产生稳定 C+ 的卤代物**（叔/烯丙/苄基）| "Lewis 酸 + 卤代物：要能离子化才行" |

### 12.3 入门级例题（不同于第十节）

> 第十节例题面向"已学的人"做巩固；本段例题面向"刚学的人"做入门。

**题目**：以下两个起始物分别与 PhCHO 在 TiCl₄ 催化下反应，**哪个反应快**？

A. $\mathrm{(CH_3)_3Si\!-\!CH_2\!-\!CH\!=\!CH_2}$（普通烯丙基三甲硅烷）

B. $\mathrm{(CH_3)_3C\!-\!CH_2\!-\!CH\!=\!CH_2}$（普通烯丙基叔丁基烷）

**预期解答路径**（把脑内推理写出来）：

1. 第一步看到题目应该想到：两个分子结构上唯一的区别是 **Si vs C**——其他都一样。
2. 然后联想到本 KP 的：Sakurai 反应靠 β-Si 稳定的碳正离子才能发生。Si 是关键。
3. 应用 [[β-硅基效应]]：σ(C-Si) → p(C⁺) 超共轭比 σ(C-C) → p(C⁺) 强 10²-10⁴ 倍。
4. 化合物 B 中 β 位是叔丁基的 C，σ(C-C) 弱给体——根本不形成可观测的中间体。
5. 结论：**A 快得多**（实际上 B 几乎不反应）。

**老师可在课堂上引导的提问**：
- 学生若卡在第 1 步：问 "把两个分子并排写出来，唯一不同的原子是哪个？"
- 学生若卡在第 2 步：问 "你最近学的 [[β-硅基效应]] 告诉我们什么？β 位的 Si 起什么作用？"
- 学生若答错（选 B 因"叔丁基大、+I 强"）：让学生写出"+I"在哪根键上、是否填入 LUMO？引导他们明白 +I 与超共轭是不同概念。

### 12.4 与现实/直觉的连接

- **类比**：Sakurai 反应像**接力赛跑** —— 烯丙基硅烷（A）把"接力棒"（碳链）交给亲电体（B），而 Si 在交棒过程中**先稳定碳正离子（暂时握棒）**，最后才放手离开。如果不是 Si（比如换成 C），他根本提不动这个棒。
- **反例/佯谬**：表面上看 Sakurai 反应是"烯丙基硅烷 + 醛 → 醇"——多了一个 OH 而已，看似平凡。但实际上**双键位置发生了改变**，这是"位置异构"的精妙变化——它告诉我们：表面上的官能团变化只是表象，**深层的电子流动决定了产物结构的所有细节**。

## 十三、竞赛拓展

### 13.1 不对称 Sakurai 烯丙基化

#### A. Roush 烯丙基硅烷（手性辅助）
- 在 Si 上引入手性辅助（如 d 或 l-酒石酸酯衍生物）
- 与醛反应给出高 ee（一般 80-95%）
- 与同样的 Roush 烯丙基硼烷类似但更适合大位阻底物

#### B. Denmark 不对称烯丙基化
- 使用手性 Lewis 碱催化剂（手性磷酰胺 / 双膦氧化物）
- 经过五配位 Si 的过渡态
- 对芳醛的 ee 可达 90% 以上

#### C. 手性 Lewis 酸催化
- BINOL-Ti、BINAP-Ag 等手性 Lewis 酸
- 适合醛 + 末端取代烯丙基硅烷的不对称 syn/anti 选择

### 13.2 烯丙基硅烷 + 不对称环外烯酮（Mukaiyama-Sakurai）

> Mukaiyama 在 1980 年代将 TMS 烯醇醚的反应扩展到烯丙基硅烷，建立了不对称环加成的"硅醚化学"基础。

### 13.3 重要文献

- Hosomi, A.; Sakurai, H. *Tetrahedron Lett.* **1976**, *17*, 1295–1298（首次报道）
- Hosomi, A. *Acc. Chem. Res.* **1988**, *21*, 200–206（综述）
- Denmark, S. E. *Chem. Rev.* **2003**, *103*, 2763–2794（Lewis 碱催化机理）
- Roush, W. R. *Comprehensive Organic Synthesis* (1991), Vol. 2, Chapter 1.1（不对称烯丙基化）
