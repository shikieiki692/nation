# 取代反应

## 一、定义

**取代反应（Substitution Reaction）** 是有机化学中最基本的反应类型之一：分子中的一个原子或基团被另一个原子或基团所替代，总体通式为：

$$
\ce{R-LG + Nu^- -> R-Nu + LG^-}
$$

- **核心特征**：旧键断裂与新键形成**同时或分步**发生，但**不饱和度不变**（区别于加成/消除）
- **判断标志**：反应前后分子的**不饱和键数目不变**，只是官能团或取代基发生交换
- **原子经济性**：通常低于 100%（有副产物 LG 离去），但在某些催化循环中可实现原子经济性优化

## 三、核心原理

### 3.1 电子推动的三种模式

取代反应的驱动力本质上是**电子密度重新分布**，按进攻物种的电子特性分为三大类：

| 类型 | 进攻物种 | 底物特征 | 箭头方向 | 典型场景 |
|:---|:---|:---|:---|:---|
| **亲核取代（SN）** | 富电子亲核体（Nu⁻）| 缺电子中心（sp³ 碳连 LG）| Nu → C → LG | 卤代烷转化 |
| **亲电取代（SE）** | 缺电子亲电体（E⁺）| 富电子体系（芳环/烯烃）| E → π 体系 → H⁺ | 芳香取代 |
| **自由基取代（SR）** | 中性自由基（R·）| 弱极化 σ 键 | 单电子转移链式 | 烷烃卤代 |

### 3.2 取代 vs 其他反应类型的本质区别

```
取代：A-B + C → A-C + B      （不饱和度不变）
加成：A=B + C-D → A-B-C-D    （不饱和度下降）
消除：A-B-C-D → A=B + C-D    （不饱和度上升）
重排：A-B-C → A-C-B          （骨架重排，无外来试剂）
```

**竞赛中的关键判别**：当题目给出产物与底物的不饱和度相同时，优先考虑取代或重排；若有试剂加入且碳骨架不变，则为取代。

## 四、关键结论

1. **SN 反应的核心是 LG 能力**：离去基越稳定（共轭酸 pKa 越小），取代越易发生
2. **SE 反应的核心是 π 电子云密度**：芳环上推电子基加速、吸电子基减速（Hammett σ 常数量化）
3. **SR 反应的核心是键解离能（BDE）**：选择性由 **BDE(C-H)** 与 **BDE(H-X)** 的相对大小决定
4. **取代与消除的竞争**是竞赛高频考点：强碱/高温/大位阻 → 消除；弱碱/低温/小位阻 → 取代

## 五、常见分类

### 5.1 亲核取代（Nucleophilic Substitution）

| 子类型 | 机理特征 | 底物偏好 | 立体化学 | 竞争反应 |
|:---|:---|:---|:---|:---|
| **SN1** | 分步，碳正离子中间体 | 3° > 2° > 1° | 外消旋化 | E1、重排 |
| **SN2** | 协同，五配位过渡态 | 1° > 2° > 3° | 构型翻转 | E2 |
| **SNAr** | 加成-消除，Meisenheimer 络合物 | 芳环连强吸电子基 + 邻对位 LG | 保持 | 苯炔机理 |
| **SN1'** | 烯丙基碳正离子离域 | 烯丙型底物 | 构型混杂 | SN2' |

### 5.2 亲电取代（Electrophilic Substitution）

| 子类型 | 亲电试剂 | 底物 | 定位规则 | 典型反应 |
|:---|:---|:---|:---|:---|
| **SEAr** | E⁺（NO₂⁺、R⁺、X⁺、SO₃）| 芳环 | 邻对位 / 间位 | 硝化、卤代、Friedel-Crafts |
| **SE1** | E⁺ | 烯醇/烯胺 | — | 烯醇 C-烷基化 |

### 5.3 自由基取代（Free Radical Substitution）

| 子类型 | 引发剂 | 选择性来源 | 典型反应 |
|:---|:---|:---|:---|
| **链式 SR** | 光 / 过氧化物 / AIBN | 键解离能（BDE）| 烷烃卤代、NBS 烯丙基溴代 |
| **非链式 SR** | 强氧化剂 / 热解 | 自由基稳定性 | Barton 脱羧 |

## 六、适用条件

### 6.1 取代反应发生的一般判据

| 条件维度 | 亲核取代 SN | 亲电取代 SE | 自由基取代 SR |
|:---|:---|:---|:---|
| **底物** | 连 LG 的 sp³ 碳 | 富电子 π 体系 | 含弱极化 C-H/C-X |
| **试剂** | 亲核体（负离子/孤对电子）| 亲电体（正离子/路易斯酸）| 自由基前体（X₂/ROOR）|
| **溶剂** | 极性（质子型促 SN1，非质子型促 SN2）| 非极性/弱极性 | 非极性 |
| **温度** | 低温（抑制消除）| 温和 | 高温或光照 |
| **催化剂** | 偶极非质子溶剂、冠醚 | Lewis 酸（AlCl₃、FeBr₃）| 光 / 过氧化物 |

### 6.2 ABOC 强调的"选择性控制"视角

- **硬软酸碱（HSAB）**：软亲核体（RS⁻、I⁻）易与软中心（烯丙基/苄基）发生取代；硬亲核体（RO⁻、F⁻）易与硬中心（甲基/伯碳）发生取代
- **位阻 vs 电子**：大位阻底物 → SN1/E1；小位阻底物 → SN2

## 七、常见比较

### 7.1 SN1 vs SN2 竞争判断树

```
底物结构
├─ 甲基 / 1° ──→ SN2（除非碳正离子特别稳定：烯丙基/苄基）
├─ 2° ──→ SN1/SN2 竞争（看溶剂和亲核体强度）
└─ 3° ──→ SN1（SN2 位阻过大）

溶剂
├─ 极性质子（H₂O、ROH）──→ 稳定碳正 → SN1
└─ 极性非质子（DMF、DMSO）──→ 裸露 Nu⁻ → SN2

亲核体
├─ 强 Nu⁻（RS⁻、I⁻）──→ SN2
└─ 弱 Nu⁻（H₂O、ROH）──→ SN1
```

### 7.2 取代 vs 消除的竞争

| 条件倾向 | 取代产物为主 | 消除产物为主 |
|:---|:---|:---|
| **碱的强度** | 弱碱（I⁻、CH₃COO⁻）| 强碱（t-BuO⁻、LDA）|
| **底物位阻** | 甲基 / 1° / 2° | 3° / β-H 位阻大 |
| **温度** | 低温 | 高温 |
| **溶剂** | 极性（稳定碳正/过渡态）| 非极性（不利于离子中间体）|
| **底物结构** | 伯卤代烷 | 叔卤代烷 |

> **ABOC 口诀**："大碱大温大位阻，消除；小碱低温小位阻，取代"

## 十、例题

### 例题 1：取代反应类型判断

**题目**：下列反应中，哪些属于取代反应？
1. CH₃CH₂Br + NaOH → CH₃CH₂OH + NaBr
2. CH₂=CH₂ + Br₂ → CH₂BrCH₂Br
3. C₆H₆ + HNO₃ → C₆H₅NO₂ + H₂O
4. (CH₃)₃CBr → (CH₃)₂C=CH₂ + HBr

**解析**：
- 反应 1：**SN2 取代**（Br⁻ 被 OH⁻ 取代，不饱和度不变）
- 反应 2：**加成**（双键消失，不饱和度下降）
- 反应 3：**SEAr 取代**（芳环 H 被 NO₂ 取代）
- 反应 4：**消除**（HBr 离去，生成双键，不饱和度上升）

**关键判别**：看不饱和度是否变化。

### 例题 2：SN1 vs SN2 综合判断（ABOC 2.9 改编）

**题目**：下列底物与 NaI/丙酮反应，按 SN2 速率从快到慢排序：
CH₃Br、CH₃CH₂Br、(CH₃)₂CHBr、(CH₃)₃CBr、PhCH₂Br

**解析**：
1. **CH₃Br**（最快，SN2 最优底物，无位阻）
2. **PhCH₂Br**（次快，苄基但 1°，SN2 可及；SN1 也易但 NaI/丙酮为极性非质子溶剂，SN2 占主导）
3. **CH₃CH₂Br**（1°，标准 SN2）
4. **(CH₃)₂CHBr**（2°，SN2 变慢）
5. **(CH₃)₃CBr**（最慢，SN2 几乎不可能，只能 SN1；但 NaI 是弱亲核体/弱碱，3° 底物在丙酮中可能不反应或极慢）

> **陷阱**：若溶剂改为 H₂O，则 PhCH₂Br 和 (CH₃)₃CBr 会因 SN1 而速率大幅上升，排序改变。

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| 把烯烃与 HX 反应归为取代 | 只看 "H 被 X 替代"，忽略 π 键断裂 | 加成反应的不饱和度下降是硬性判据 | 取代和加成的不饱和度如何变化？ |
| SN1/SN2 判断只盯底物位阻 | 忽略溶剂和亲核体强度的影响 | **三维判断**：底物 + 试剂 + 溶剂 + 温度 | 同样 2° 底物，换溶剂为什么机理变？ |
| 认为 3° 底物绝对不发生 SN2 | 极端条件下（如甲基锂）仍可 SN2 | 位阻是趋势不是绝对禁令；超强 Nu⁻ 可强制 SN2 | 什么条件下 3° 底物也能发生 SN2？ |
| 混淆 SNAr 与苯炔机理 | 都涉及芳环和亲核体 | SNAr 需强吸电子基 + LG；苯炔需无 LG 的邻位消除 | 硝基氯苯 vs 氯苯在强碱下机理为何不同？ |
| 自由基取代选择性只记 "3°>2°>1°" | 忽略温度对选择性的调控 | 高温降低选择性（动力学 vs 热力学控制）| 为什么高温下烷烃氯代选择性变差？ |

## 十二、🎯 教学视角

### 12.1 学习路径

```
前置：反应机理表示法 → 亲核体与亲电体 → 碳正离子稳定性
     ↓
本节：取代反应总论（分类 + 判别 + 竞争）
     ↓
后续分支：SN1/SN2 专论 → 芳香取代 → 自由基取代 → 消除反应（竞争）
```

**建议课时**：2 课时（总论 1 课时 + SN1/SN2 竞争实验分析 1 课时）

### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "取代 = 交换，加成 = 插入" | 从字面理解，忽略电子层面 | 用**不饱和度计数法**训练判别 |
| "SN2 一定比 SN1 快" | 把协同等同于高效 | 展示 3° 底物 SN2 的位阻能垒图 |
| "亲核体越强越好" | 认为强 Nu⁻ 总是优势 | 案例：强碱 t-BuO⁻ 在 3° 底物上只发生消除 |
| "芳环只能亲电取代" | 忽略 SNAr 和苯炔 | 对比硝基氯苯（SNAr）vs 氯苯（苯炔）的产物差异 |

### 12.3 入门例题

**最简判别题**：
> 下列哪个是取代反应？
> A. CH₄ + Cl₂ → CH₃Cl + HCl  
> B. CH₂=CH₂ + H₂ → CH₃CH₃  
> C. CH₃CH₂OH → CH₂=CH₂ + H₂O  
> D. CH₃CHO → CH₄ + CO

**答案：A**（H 被 Cl 取代，不饱和度不变）
**引导**：如果学生选错，先问"反应前后双键/三键数目有没有变化？"

### 12.4 现实类比

**「接力赛跑」类比取代反应**：
- 底物 = 拿着接力棒（LG）的运动员
- 亲核体 = 接棒的运动员
- 交接瞬间 = 过渡态（SN2 是同步交接；SN1 是先扔掉棒再等人捡）
- 如果场边有人大喊"别跑了"（强碱出现）→ 运动员直接退场（消除反应）

## 十三、竞赛拓展

### 13.1 竞赛中的特殊取代反应

| 反应 | 类型 | 竞赛价值 | 典型应用 |
|:---|:---|:---|:---|
| **Vilsmeier-Haack 甲酰化** | SEAr | 芳环引入醛基 | 全合成中醛基作为多功能手柄 |
| **Kolbe-Schmitt 反应** | SEAr | 酚 → 水杨酸 | 阿司匹林前体合成 |
| **Sandmeyer 反应** | 自由基取代 | 重氮盐 → 卤代/氰基 | 芳环官能团转化 |
| **Barton-McCombie 脱氧** | 自由基取代 | 醇 → 脱氧 | 糖化学中常用 |
| **Mitsunobu 反应** | SN2（特殊）| 醇的构型翻转 | 手性中心构型操控 |

### 13.2 与元素化学的交叉

- **硅取代**：Si-C 键极化使硅基成为特殊离去基（β-硅基效应）
- **磷取代**：Appel 反应中 PPh₃ 将 OH 转化为 LG
- **硫取代**：硫醇的强亲核性在 SN2 中常用于分子内关环
