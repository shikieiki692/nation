# Hantzsch 吡咯合成（Hantzsch Pyrrole Synthesis）

## 一、定义

**Hantzsch 吡咯合成**：**α-卤代酮**与**β-酮酯**在**氨（或伯胺）**存在下反应，通过**多组分缩合**，生成**高度取代吡咯**的反应。

**通式**：
$$
\ce{R-CO-CH2-X + R'-CO-CH2-COOR'' + NH3 → 吡咯衍生物}
$$

**本质**：**高度取代吡咯的合成法**——通过三组分（卤代酮 + β-酮酯 + 氨）一步构建多取代吡咯，是 Paal-Knorr 的重要补充。

## 三、核心原理

### 3.1 反应机理

**步骤 1：胺的亲核取代**
$$
\ce{氨进攻 α-卤代酮 → 氨基酮}
$$

**步骤 2：缩合**
$$
\ce{氨基酮 + β-酮酯 → 烯胺中间体}
$$

**步骤 3：环化脱水**
$$
\ce{烯胺环化 → 吡咯}
$$

### 3.2 产物特征

- **3-位有酯基**（来自 β-酮酯）
- **2-位和 5-位有取代基**
- 高度取代

---

## 四、关键结论

1. **Hantzsch = α-卤代酮 + β-酮酯 + 氨 → 吡咯**
2. **三组分缩合**
3. **产物高度取代**
4. **3-位有酯基**

---

## 五、常见分类或情形

### 5.1 与 Paal-Knorr 的比较

| | **Hantzsch** | **Paal-Knorr** |
|:---|:---|:---|
| **底物** | 卤代酮 + β-酮酯 + 氨 | 1,4-二羰基 |
| **步骤** | 三组分 | 单组分环化 |
| **产物** | 高度取代（3-位酯基）| 简单取代 |

---

## 六、适用条件与限制

1. **需要 α-卤代酮**
2. **需要 β-酮酯**
3. **氨或伯胺**

## 十三、竞赛拓展

1. **Hantzsch 杂环合成的"通用模板"逻辑——β-二羰基 + α-卤代酮 + 氨 = 吡咯/呋喃/噻吩**：Hantzsch 合成的核心是双组分缩合——一个 1,3-二羰基提供 C₂-C₃，一个 α-卤代羰基提供 C₄-C₅，NH₃（或 H₂S、H₂O）提供杂原子。当用 β-酮酯 + α-卤代酮 + NH₃ → 吡咯；换 H₂S → 噻吩；换 NH₂OH → 异噁唑。竞赛中可能给一个未知杂环的合成路线要求"逆推"反应物——识别这是 Hantzsch 型合成是快速解题的关键。

2. **Paal-Knorr 合成——互补但又不同的吡咯合成策略**：1,4-二羰基化合物 + NH₃（或胺）→ 吡咯（Paal-Knorr），机理是两次缩合+脱水。比 Hantzsch 更简单但需要预制的 1,4-二羰基（通常由 Stetter 反应或 α-卤代酮与烯胺反应制得）。竞赛中两种合成策略的选择取决于原料的可获得性和所需取代模式。
