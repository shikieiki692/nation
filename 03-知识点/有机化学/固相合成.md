# 固相合成

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-有机化学]]
- 对应考纲条目：[[50-有机合成]]、[[54-氨基酸与蛋白质]]

## 一、定义

**固相合成（Solid-Phase Synthesis）**：将反应物之一固定在不溶性固体载体（树脂）上，在固相上进行化学反应，反应完成后通过过滤分离产物，未反应试剂和副产物被洗除的合成方法。

**本质**：利用固液分离简化纯化步骤，特别适合多步重复合成（如多肽、寡核苷酸）。

## 二、考纲对应

- 对应考纲条目：[[50-有机合成]]、[[54-氨基酸与蛋白质]]

- 本知识点在考纲中的作用：固相合成是现代有机合成的重要方法，特别是多肽合成（SPPS）是生物化学与有机化学的交叉内容。

## 三、核心原理

### 3.1 固相肽合成（SPPS）基本流程

**Merrifield 树脂**：
- 聚苯乙烯树脂，氯甲基化
- 第一个氨基酸通过酯键连接到树脂上

**合成循环**：
1. **脱保护**：脱除氨基保护基（如 Fmoc 用哌啶，Boc 用 TFA）
2. **洗涤**：去除脱保护试剂
3. **偶联**：加入下一个保护的氨基酸 + 偶联试剂（DIC/HOBt 或 HATU）
4. **洗涤**：去除过量试剂和副产物
5. **重复**：循环步骤 1-4 直到肽链完成
6. **切肽**：用强酸（TFA）将肽从树脂上切下，同时脱除所有侧链保护基

### 3.2 Fmoc vs Boc 策略

| 特征 | **Fmoc 策略** | **Boc 策略** |
|:---|:---|:---|
| α-氨基保护基 | Fmoc（碱敏感）| Boc（酸敏感）|
| 脱保护条件 | 哌啶（碱）| TFA（酸）|
| 侧链保护基 | 酸敏感（如 tBu）| 碱敏感或氢解敏感 |
| 切肽条件 | TFA（酸）| HF（强酸，危险）|
| 正交性 | Fmoc（碱脱）与侧链（酸脱）正交 | Boc（酸脱）与侧链（碱/氢解脱）正交 |
| 安全性 | 较安全 | HF 危险 |
| 主要应用 | **现代 SPPS 主流** | 早期方法 |

### 3.3 固相合成的优势

1. **简化纯化**：每步只需过滤和洗涤
2. **过量试剂驱动反应完全**：可使用大大过量的试剂
3. **自动化**：适合机器自动合成
4. **组合化学**：可平行合成大量化合物库

### 3.4 局限性

1. **需要额外步骤**：将产物从树脂上切下
2. **反应监测困难**：不能直接用 TLC 监测
3. **树脂负载量有限**：每克树脂只能负载一定量底物
4. **非均相反应**：反应速率可能较慢

## 四、关键结论

1. **Fmoc 策略是现代 SPPS 的主流**——安全、正交性好
2. **保护基的正交性是 SPPS 的核心**——Fmoc（碱脱）与侧链保护基（酸脱）正交
3. **过量试剂 + 反复洗涤 = 高纯度**
4. **固相合成的革命性意义**：使长肽和寡核苷酸的常规合成成为可能
5. **Merrifield 因固相肽合成获 1984 年诺贝尔化学奖**

## 五、常见分类或情形

### 5.1 常见树脂类型

| 树脂 | 特点 | 应用 |
|:---|:---|:---|
| **Merrifield 树脂** | 氯甲基聚苯乙烯 | Boc 策略 |
| **Wang 树脂** | 对苄氧基苄醇 | Fmoc 策略，酸切肽 |
| **Rink 树脂** | 酰胺连接 | Fmoc 策略，生成酰胺 |
| **TentaGel** | PEG 接枝聚苯乙烯 | 亲水性好，适合大肽 |

### 5.2 偶联试剂

| 试剂 | 特点 |
|:---|:---|
| **DIC/HOBt** | 经典组合，经济 |
| **HATU** | 高效，消旋少 |
| **HBTU** | 常用，高效 |
| **PyBOP** | 磷鎓盐类，低消旋 |

## 六、适用条件与限制

1. **需要正交保护基策略**——α-氨基和侧链保护基脱除条件必须不同
2. **树脂必须对反应条件稳定**
3. **切肽条件不能破坏产物**
4. **长肽合成（>50 残基）仍具挑战**——累积产率下降
5. **某些氨基酸（如 Arg）偶联困难**

## 七、常见比较与易混点

### 7.1 固相合成 vs 液相合成

| | **固相合成** | **液相合成** |
|:---|:---|:---|
| 纯化 | 过滤洗涤 | 萃取、柱层析 |
| 过量试剂 | 可以 | 增加后处理负担 |
| 自动化 | 容易 | 较难 |
| 规模 | 中小规模 | 可放大 |
| 监测 | 较困难 | 容易（TLC、NMR）|
| 成本 | 树脂较贵 | 试剂成本 |

### 7.2 Fmoc vs Boc 脱保护

| | **Fmoc 脱保护** | **Boc 脱保护** |
|:---|:---|:---|
| 试剂 | 哌啶/DMF | TFA/DCM |
| 副产物 | 二苯并富烯（DBF）| 异丁烯、CO₂ |
| 与树脂兼容性 | Fmoc 策略树脂对碱稳定 | Boc 策略树脂对酸稳定 |

## 十、例题

### 例题 1（★ 基础）

**题目**：为什么 SPPS 中常用 Fmoc 保护 α-氨基？

**解答**：
1. Fmoc 用碱（哌啶）脱除
2. 肽链连接在树脂上，通常通过酸敏感的连接基
3. 碱脱除 Fmoc 不影响肽-树脂连接
4. 侧链保护基用酸敏感的（如 tBu）
5. Fmoc（碱脱）与侧链（酸脱）**正交**

### 例题 2（★★ 正交性）

**题目**：SPPS 中如何同时保护 Lys 的 α-氨基和 ε-氨基？

**解答**：
- α-氨基：Fmoc（碱脱）
- ε-氨基：Boc（酸脱）或 Alloc（Pd 催化脱）
- 两者正交，可分别脱除

## 十一、易错点

1. ❌ **混淆 Fmoc 和 Boc 的脱保护条件** → Fmoc 用碱，Boc 用酸
2. ❌ **忽略正交性要求** → α-氨基和侧链保护基脱除条件必须不同
3. ❌ **认为固相合成不需要保护基** → 保护基策略是 SPPS 的核心
4. ❌ **忽略树脂的化学敏感性** → 树脂必须对反应条件稳定

### 12.2 学生易踩的认知误区

|---|---|

| "所有树脂都一样" | 不同树脂适用于不同策略和切肽条件 |

### 12.3 入门级例题

**题目**：写出 SPPS 中一个偶联循环的步骤。

**解答**：
1. 脱保护：哌啶脱除 Fmoc
2. 洗涤：DMF 洗涤
3. 偶联：Fmoc-氨基酸 + HATU + DIEA
4. 洗涤：DMF 洗涤
5. 检测：Kaiser 测试或茚三酮测试确认偶联完全

### 12.4 与现实/直觉的连接

- 固相合成就像"在工厂流水线上生产"——底物固定在传送带（树脂）上，每站完成一个步骤，不合格品被洗掉，最后成品从传送带上取下。
- 正交保护基就像"不同密码的锁"——确保每一步只打开需要的锁。
