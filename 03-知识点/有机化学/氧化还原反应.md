# 有机氧化还原反应（Organic Redox Reactions）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[36-取代反应]]、[[38-消除反应]]、[[40-氧化还原反应]]、[[44-脂肪族亲核取代反应]]

---

## 一、定义

**有机氧化还原反应**：有机分子中碳原子的氧化态发生变化，或电子密度在原子间重新分布的反应。

**本质**：氧化 = 失电子/得氧/失氢；还原 = 得电子/失氧/得氢。

**ABOC 核心视角**：氧化还原不是"能"与"不能"的二元对立，而是**选择性的梯度控制**——从最强氧化剂到最温和试剂，构成一个"光谱"。

## 三、核心原理

### 3.1 氧化还原的"选择性光谱"

ABOC 将醇的氧化试剂按"强度"排列成光谱：

| 试剂 | 强度 | 1°醇产物 | 2°醇产物 | 特点 |
|:---|:---:|:---:|:---:|:---|
| **Dess-Martin** | 最温和 | 醛 | 酮 | 高选择性，贵，对酸敏感基团友好 |
| **Swern** | 温和 | 醛 | 酮 | 低温（-78°C），无金属，无过氧化 |
| **PCC/PDC** | 温和 | 醛 | 酮 | 不氧化醛到酸，操作简便 |
| **Jones（CrO₃/H₂SO₄）** | 强 | 羧酸 | 酮 | 酸性，会过氧化醛 |
| **KMnO₄（热浓）** | 最强 | 羧酸 | 酮/断裂 | 彻底氧化，破坏性 |

> ⭐ **核心思维**：不是"选哪个能氧化"，而是"选哪个**刚好氧化到目标阶段**"。

### 3.2 负氢迁移的统一框架

多个看似不同的人名反应，共享同一个核心机理——**负氢迁移（hydride shift）**：

```
Cannizzaro → MPV → Tishchenko → Meerwein-Ponndorf
     ↓           ↓           ↓              ↓
 醛歧化      酮还原      醛→酯        酮还原（旧名）
```

**共性**：
- 都涉及 **Al(III)** 或碱催化
- 都经历 **六元环过渡态**（负氢从供体转移到受体）
- 都由 **Le Chatelier 原理** 驱动可逆性

### 3.3 Oppenauer 氧化 / MPV 还原的可逆对

$$
\ce{R^1R^2CHOH + R^3R^4C=O <=>[	ext{Al(OiPr)3}] R^1R^2C=O + R^3R^4CHOH}
$$

- **Oppenauer 氧化**：向右（醇被氧化，酮被还原）
  - 用大量丙酮作氧化剂，移除异丙醇
- **MPV 还原**：向左（酮被还原，醇被氧化）
  - 用大量异丙醇作还原剂，移除丙酮

**本质**：同一反应的两个方向，由过量试剂驱动。

### 3.4 醇的卤代与氧化的关系

| 转化类型 | 试剂 | 氧化态变化 |
|:---|:---|:---:|
| 醇 → 卤代烷 | SOCl₂/PBr₃/Appel/DAST | 不变（取代）|
| 醇 → 醛/酮 | Swern/Dess-Martin/PCC | 氧化 |
| 醇 → 烯烃 | POCl₃/Burgess（脱水）| 消除（不涉氧化态）|

---

## 四、关键结论

1. **氧化还原是光谱，不是开关**——试剂选择决定氧化深度
2. **负氢迁移串联多个人名反应**——学会 Cannizzaro = 学会 MPV = 学会 Tishchenko
3. **Swern 和 Dess-Martin 停在醛**——是 1°醇 → 醛 的首选（不过氧化）
4. **PCC 是"实验室通用型"**——便宜、方便、停在醛
5. **Jones 氧化彻底**——1°醇 → 羧酸，2°醇 → 酮
6. **Oppenauer/MPV 是一对可逆反应**——方向由过量试剂决定

## 五、常见分类或情形

### 5.1 按氧化对象分类

| 底物 | 目标产物 | 推荐试剂 |
|:---|:---|:---|
| **1°醇 → 醛** | 醛（不进一步氧化）| Dess-Martin、Swern、PCC |
| **1°醇 → 羧酸** | 羧酸 | Jones、KMnO₄、CrO₃ |
| **2°醇 → 酮** | 酮 | 上述所有均可 |
| **醛 → 羧酸** | 羧酸 | Ag₂O、KMnO₄、CrO₃ |
| **烯烃 → 邻二醇** | syn-二醇 | OsO₄、KMnO₄（冷稀）|
| **烯烃 → 醛/酮** | 切断产物 | O₃（臭氧化）|

### 5.2 按还原对象分类

| 底物 | 目标产物 | 推荐试剂 |
|:---|:---|:---|
| **醛/酮 → 醇** | 1°/2°醇 | NaBH₄、LiAlH₄ |
| **醛（无α-H）→ 醇+酸** | 歧化 | 浓 NaOH（Cannizzaro）|
| **酮 → 醇（可逆）** | 2°醇 | MPV 还原（Al(OiPr)₃ + 异丙醇）|
| **炔烃 → Z-烯烃** | 顺式烯烃 | Lindlar 催化剂 |
| **炔烃 → E-烯烃** | 反式烯烃 | Na/NH₃(l) |

## 六、适用条件与限制

1. **Swern 需低温**（-78°C）——室温下中间体分解
2. **Dess-Martin 对酸敏感底物友好**——但试剂本身昂贵
3. **LiAlH₄ 与水剧烈反应**——必须在无水条件下使用
4. **NaBH₄ 选择性还原醛/酮**——不还原酯、酰胺
5. **Cr 试剂有毒**——实验室逐渐被淘汰，但竞赛仍考
6. **Lindlar 需 H₂ 气氛**——对硫敏感

---

## 七、常见比较与易混点

### 7.1 PCC vs Jones vs Swern

| | **PCC** | **Jones** | **Swern** |
|:---|:---|:---|:---|
| **1°醇产物** | 醛 | 羧酸 | 醛 |
| **2°醇产物** | 酮 | 酮 | 酮 |
| **条件** | CH₂Cl₂，室温 | 丙酮/水，酸性 | -78°C，无金属 |
| **特点** | 方便 | 强氧化 | 温和，无过氧 |

### 7.2 NaBH₄ vs LiAlH₄

| | **NaBH₄** | **LiAlH₄** |
|:---|:---|:---|
| **还原对象** | 醛、酮 | 醛、酮、酯、酰胺、羧酸 |
| **反应性** | 温和 | 强 |
| **溶剂** | 甲醇/乙醇 | 乙醚/THF |
| **对水** | 较稳定 | 剧烈反应 |

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：如何将 1-丁醇氧化为丁醛而不得到丁酸？

**解答**：
- 需要**停在醛阶段**的氧化剂
- 选择：**PCC** 或 **Swern** 或 **Dess-Martin**
- **不能用 Jones**（会过氧化为羧酸）

### 10.2 例题 2（★★ 机理）

**题目**：写出 Cannizzaro 反应的机理，并说明负氢迁移的步骤。

**解答**：
1. OH⁻ 进攻醛 → 四面体中间体
2. 另一分子醛的 H 以 H⁻ 形式迁移 → 醇醛盐 + 羧酸根
3. 质子交换 → 醇 + 羧酸盐

### 10.3 例题 3（★★★ 合成设计）

**题目**：设计从环己醇到己二酸的合成路线。

**解答**：
1. 环己醇 → **Jones 氧化** → 环己酮
2. 环己酮 → **Baeyer-Villiger** → 己内酯
3. 或：环己烯 → **臭氧化** → 己二醛 → **Jones** → 己二酸

## 十一、易错点

1. ❌ **PCC 氧化 1°醇得羧酸** → PCC **停在醛**，Jones 才到酸
2. ❌ **Swern 可以室温做** → Swern 必须 **-78°C**
3. ❌ **NaBH₄ 能还原酯** → NaBH₄ **不能**还原酯，LiAlH₄ 才能
4. ❌ **Cannizzaro 需要 α-H** → 恰恰需要 **无 α-H**
5. ❌ **Oppenauer 和 MPV 是不同反应** → 是**同一反应的两个方向**
6. ❌ **氧化就是加氧** → 有机化学中**失氢也是氧化**（如醇 → 醛）

### 12.3 入门例题

**题目**：下列试剂中，哪些能将 1-丁醇氧化为丁醛（不进一步氧化）？
A. Jones 试剂  B. PCC  C. KMnO₄  D. Swern

**答案**：B、D

### 12.4 现实类比

**氧化还原试剂像"温度调节器"**：
- Dess-Martin = 小火慢炖（温和，精准）
- Swern = 微波炉（快速，低温）
- Jones = 大火爆炒（猛烈，彻底）
- KMnO₄ = 熔炉（什么都烧光）
