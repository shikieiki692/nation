# 氧化反应

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[40-氧化还原反应]]
- 关联笔记：[[有机还原反应]], [[官能团转化]], [[有机合成]]

## 一、定义

有机化学中的氧化反应是指有机物分子中碳原子的氧化数升高、失去电子（或电子密度降低）的反应。通常表现为：加氧、去氢、或两者兼有。与无机化学中的氧化反应不同，有机氧化反应的判断主要基于碳的氧化数变化。

## 二、考纲对应

- 对应考纲条目：[[40-氧化还原反应]]
- 所属模块：[[基础要求-有机化学]]
- 本知识点在考纲中的作用：氧化反应是**有机官能团转化的核心工具**之一，贯穿从醇到醛/酮/酸、从烯烃到环氧化物/邻二醇的整个合成逻辑链。有机合成题中几乎每一次"官能团升级"都涉及氧化剂的选择与条件优化。掌握氧化剂的"选择性谱"是从基础要求迈向综合应用的必经之路。

## 三、核心原理

### 2.1 碳的氧化数变化规律

| 官能团 | 碳氧化数 | 氧化程度 |
|--------|---------|---------|
| 烷烃 R-CH₃ | -3 | 最低 |
| 烯烃/醇 R-CH₂OH | -1 | 低 |
| 醛 R-CHO | +1 | 中 |
| 羧酸/酯 R-COOH | +3 | 高 |
| CO₂ | +4 | 最高 |

### 2.2 常用氧化剂及适用范围

| 氧化剂 | 典型反应 | 选择性 | 备注 |
|--------|---------|--------|------|
| **KMnO₄（热、浓、碱性）** | 烯烃 → 邻二醇/裂解；芳环侧链 → COOH | 低 | 强氧化剂，几乎氧化一切 |
| **KMnO₄（冷、稀）** | 烯烃 → 邻二醇 | 中 | 顺式加成 |
| **O₃ / Zn-H₂O** | 烯烃 → 醛/酮（臭氧化-还原裂解） | 高 | 确定双键位置 |
| **CrO₃ / Jones试剂** | 醇 → 醛/酮 | 中 | 一级醇 → 醛（不过度氧化） |
| **PCC** | 一级醇 → 醛 | 高 | 不氧化双键 |
| **Swern氧化** | 一级醇 → 醛；二级醇 → 酮 | 高 | 温和，无重金属 |
| **OsO₄ / NMO** | 烯烃 → 邻二醇 | 高 | 顺式加成，对映选择性可用 AD-mix |
| **O₂ / Ag（乙烯）** | 烯烃 → 环氧 | 高 | 工业制环氧乙烷 |
| **HIO₄** | 邻二醇 → 醛/酮 | 高 | 用于糖类结构分析 |

### 2.3 选择性氧化策略

- **位阻控制**：大位阻氧化剂（如 PCC）避免过度氧化
- **官能团兼容**：Swern、Dess-Martin 氧化对双键、硫醚等敏感基团友好
- **对映选择性**：Sharpless 不对称双羟基化（AD）、Sharpless 环氧化

## 四、关键结论

1. **氧化反应的本质 = 碳氧化数升高**——在有机化学中，氧化不一定"加氧"。凡是使碳原子氧化数升高的转化都是氧化反应——加氧、去氢、或碳与电负性更强的原子成键，均属此类。判断标准是**计算氧化数的净变化**，而非仅看产物中是否出现了氧原子。

2. **氧化阶梯（Oxidation Ladder）是合成规划的核心框架**：烷烃（C 氧化数 $-3$）$\to$ 醇（$-1$）$\to$ 醛（$+1$）$\to$ 羧酸（$+3$）$\to$ $\mathrm{CO_2}$（$+4$），每一步上升都对应不同的氧化剂和条件。能够沿着这个阶梯"上到哪一层停住"，是区分合成水平高低的关键指标。

3. **化学选择性（Chemoselectivity）是氧化反应的第一挑战**——真实分子中往往同时存在多个可氧化官能团（如羟基 + 双键 + 硫醚），如何让氧化剂"指哪打哪"而不伤及其他部位，是合成化学的核心难题。PCC、Swern、Dess-Martin 等温和氧化剂的出现，正是为了解决 Cr(VI) 强氧化剂的"滥杀"问题。

4. **氧化剂存在明确的"反应性-选择性谱"**：从强到弱大致为 $\mathrm{KMnO_4}$（热、浓）$\gt$ Jones 试剂 $\gt$ PCC / PDC $\gt$ Swern $\approx$ Dess-Martin $\gt$ TEMPO / bleach。选择氧化剂的过程，本质上是在这个谱上找到**恰好**能氧化目标官能团、而**不触及**其他敏感基团的那一个"能量窗口"。

5. **立体化学是氧化反应的高级维度**——Sharpless 不对称环氧化（AE）和不对称双羟基化（AD）将氧化反应从"能不能做成"提升到"能不能做对"，引入对映选择性的考量。这些反应是国初压轴题和决赛/IChO 的常客。

## 五、常见分类或情形

氧化反应可按氧化剂类型、被氧化官能团、或反应机理进行分类。以下是三种最实用的分类框架。

### 5.1 按氧化剂类型分类

#### A. 金属基氧化剂（Metal-Based Oxidants）

| 氧化剂 | 活性物种 | 典型转化 | 特点 |
|:---|:---|:---|:---|
| **Cr(VI) 系列**（PCC、PDC、Jones） | $\mathrm{CrO_3}$ 或氯铬酸根 | 醇 $\to$ 醛/酮/酸 | 经典但有毒，选择性可通过配体调节 |
| **Mn(VII) 系列**（$\mathrm{KMnO_4}$） | $\mathrm{MnO_4^-}$ | 烯烃 $\to$ 邻二醇/裂解产物；芳环侧链 $\to$ COOH | 强氧化剂，条件决定产物 |
| **Os(VIII) 系列**（$\mathrm{OsO_4}$） | $\mathrm{OsO_4}$ | 烯烃 $\to$ 邻二醇（顺式） | 催化量使用 + NMO 共氧化剂；可用 AD-mix 实现对映选择性 |
| **Ru(VIII) 系列**（$\mathrm{RuO_4}$、TPAP） | $\mathrm{RuO_4}$ 或 $\mathrm{Pr_4N^+RuO_4^-}$ | 醇 $\to$ 醛/酮 | TPAP（Ley-Griffith）温和，催化量+NMO |

#### B. 非金属氧化剂（Non-Metal Oxidants）

| 氧化剂 | 活性物种 | 典型转化 | 特点 |
|:---|:---|:---|:---|
| **Swern 氧化** | $\mathrm{Me_2S^+{-}O{-}Cl}$（活化的 DMSO） | 一级醇 $\to$ 醛；二级醇 $\to$ 酮 | 条件苛刻（$-78\,^\circ\mathrm{C}$，无水），但选择性极好，无重金属 |
| **Dess-Martin 氧化** | $\lambda^5$-碘烷（Dess-Martin periodinane, DMP） | 一级醇 $\to$ 醛；二级醇 $\to$ 酮 | 室温、中性、操作简便；比 Swern 条件温和得多 |
| **TEMPO 氧化** | 氧铵离子（$\mathrm{TEMPO^+}$） | 一级醇 $\to$ 醛（选择性优于二级醇） | 催化循环，常用 bleach（NaOCl）为终端氧化剂 |
| **Corey-Kim 氧化** | $\mathrm{Me_2S^+{-}Cl}$ / $\mathrm{Me_2S\text{-}NCS}$ | 醇 $\to$ 醛/酮 | Swern 的前身，机理类似但试剂不同 |

#### C. 催化氧化（Catalytic Oxidations）

| 反应 | 催化剂 | 典型转化 | 对映选择性 |
|:---|:---|:---|:---|
| **Sharpless 不对称环氧化（AE）** | $\mathrm{Ti(OiPr)_4}$ + 酒石酸酯 | 烯丙醇 $\to$ 环氧醇 | 高 ee%（$>90\%$） |
| **Sharpless 不对称双羟基化（AD）** | $\mathrm{OsO_4}$ + 金鸡纳碱配体（AD-mix） | 烯烃 $\to$ 手性邻二醇 | 高 ee%（$>90\%$） |
| **Jacobsen 环氧化** | Mn-salen 配合物 | 非官能团化烯烃 $\to$ 环氧化物 | 高 ee%（顺式烯烃尤其好） |
| **Wacker 氧化** | $\mathrm{PdCl_2}$ / $\mathrm{CuCl_2}$ / $\mathrm{O_2}$ | 末端烯烃 $\to$ 甲基酮 | 无（区域选择性由机理决定） |
| **Ley-Griffith 氧化（TPAP）** | $\mathrm{Pr_4N^+RuO_4^-}$ + NMO | 醇 $\to$ 醛/酮 | 无（催化量 Ru，温和） |

### 5.2 按被氧化官能团分类

| 目标转化 | 常用氧化剂（按推荐顺序） | 备注 |
|:---|:---|:---|
| **一级醇 $\to$ 醛**（不过度氧化） | PCC、DMP、Swern、TEMPO | 关键在于**不能有水**且**不能过度** |
| **一级醇 $\to$ 羧酸** | Jones（$\ce{CrO3/H2SO4}$）、$\ce{KMnO4}$、PDC（DMF 中） | 需要更强条件或原位生成水 |
| **二级醇 $\to$ 酮** | PCC、DMP、Swern、Jones、$\ce{KMnO4}$ | 酮不易过度氧化，条件选择宽泛 |
| **烯烃 $\to$ 顺式邻二醇** | $\ce{OsO4}$（催化）/ NMO；冷稀 $\ce{KMnO4}$ | $\ce{OsO4}$ 产率高且条件温和；$\ce{KMnO4}$ 易过度氧化 |
| **烯烃 $\to$ 环氧化物** | mCPBA 等过氧酸；$\ce{H2O2/NaOH}$（$\alpha,\beta$-不饱和酮） | 立体专一性：顺式烯 $\to$ 顺式环氧，反式 $\to$ 反式 |
| **烯烃 $\to$ 醛/酮（裂解）** | $\ce{O3}$ / $\ce{Zn-H2O}$（臭氧化-还原裂解）；$\ce{KMnO4}$（热） | 臭氧化条件温和、官能团兼容性好 |
| **芳环侧链 $\to$ 羧酸** | $\ce{KMnO4}$（热、碱性） | 不论侧链多长，只要有 $\alpha$-H 就氧化到 COOH |
| **硫醚 $\to$ 亚砜** | $\ce{H2O2}$（1 eq）、$\ce{NaIO4}$（1 eq） | 控制当量，过量会到砜 |
| **邻二醇 $\to$ 醛/酮（裂解）** | $\ce{HIO4}$、$\ce{NaIO4}$ | 用于糖类结构分析（见 [[过碘酸氧化]]） |

### 5.3 按机理分类

- **氧转移机理**：金属氧配合物（$\mathrm{CrO_3}$、$\mathrm{MnO_4^-}$、$\mathrm{OsO_4}$）直接将氧原子转移到底物上
- **活化-消除机理**：Swern 氧化中 DMSO 被活化后与醇形成烷氧基锍盐，经消除得羰基
- **氢转移/电子转移**：TEMPO 氧化中醇先被氧化为醛，TEMPO 被还原后由终端氧化剂再生
- **金属催化循环**：Wacker 氧化中 $\mathrm{Pd^{2+}}$ 被还原为 $\mathrm{Pd^0}$，由 $\mathrm{CuCl_2/O_2}$ 再生

## 六、适用条件与限制

### 6.1 关键氧化反应的条件约束

| 反应/试剂 | 严格要求 | 违反后果 |
|:---|:---|:---|
| **Swern 氧化** | ① 无水无氧；② $-78\,^\circ\mathrm{C}$（干冰/丙酮浴）；③ 加料顺序：DMSO + (COCl)$_2$ $\to$ 醇 $\to$ Et$_3$N | 温度升高 $\to$ 副反应激增（Pummerer 重排等）；有水 $\to$ 活性物种水解失效 |
| **PCC 氧化** | 无水 $\ce{CH2Cl2}$；一级醇用 PCC（不用 Jones） | Jones 试剂含水 $\to$ 一级醇直接到酸，无法停在醛 |
| **Dess-Martin 氧化** | 室温、中性即可，避免强碱强酸 | 条件宽容度高，但久置 DMP 会分解（避光、干燥保存） |
| **$\ce{OsO4}$ 催化双羟化** | 必须加 NMO 共氧化剂（或 $\ce{K3Fe(CN)6}$）| 无共氧化剂则需化学计量 $\ce{OsO4}$（贵且剧毒） |
| **$\ce{KMnO4}$ 冷稀** | 温度 $<10\,^\circ\mathrm{C}$，稀溶液，中性或微碱性 | 温度高/浓度大 $\to$ 邻二醇被进一步氧化裂解 |
| **臭氧化-还原裂解** | ① $\ce{O3}$ 通入低温溶液（$-78\,^\circ\mathrm{C}$）；② 还原后处理（$\ce{Zn/H2O}$ 或 $\ce{Me2S}$） | 若用氧化后处理（$\ce{H2O2}$）$\to$ 得羧酸而非醛/酮 |
| **Wacker 氧化** | $\ce{PdCl2/CuCl2}$，$\ce{O2}$ 氛围，$\ce{H2O/DMF}$ | $\ce{CuCl2}$ 需协同再生 $\ce{Pd^{2+}}$；无 $\ce{Cu}$ 则 Pd 沉淀失活 |

### 6.2 溶剂兼容性

| 氧化体系 | 适用溶剂 | 禁用溶剂 |
|:---|:---|:---|
| PCC / PDC | $\ce{CH2Cl2}$、$\ce{CHCl3}$ | 醇类（会被氧化）、水（PDC 不耐水） |
| Swern | $\ce{CH2Cl2}$、$\ce{THF}$（低温） | 水、醇、DMSO 之外的高极性溶剂 |
| Dess-Martin | $\ce{CH2Cl2}$、$\ce{MeCN}$、$\ce{THF}$ | 无明显禁忌，但避免强配位溶剂 |
| $\ce{KMnO4}$ | $\ce{H2O}$、$\ce{H2O/acetone}$ | 纯有机溶剂（溶解度差） |
| $\ce{OsO4}$ / NMO | $\ce{H2O/acetone}$ 或 $\ce{H2O/tBuOH}$ | 纯非极性溶剂（NMO 不溶） |
| $\ce{O3}$ | $\ce{CH2Cl2}$、$\ce{MeOH}$（低温） | 高温溶剂（臭氧溶解度低且危险） |

### 6.3 过度氧化的预防策略

| 目标 | 策略 | 实例 |
|:---|:---|:---|
| 一级醇 $\to$ 醛（避免到酸） | 使用无水条件 + 温和氧化剂 | PCC（无水 $\ce{CH2Cl2}$）或 Swern / DMP |
| 一级醇 $\to$ 醛（无铬方案） | Swern 或 Dess-Martin | 避免 Cr 废物；条件温和 |
| 邻二醇 $\to$ 二醛（避免裂解） | 使用 $\ce{OsO4}$ 而非 $\ce{KMnO4}$ | $\ce{OsO4}$ 停在二醇阶段 |
| 烯烃 $\to$ 醛/酮（避免 $\to$ 酸） | 臭氧化 + 还原后处理（$\ce{Zn}$ 或 $\ce{Me2S}$） | 不用 $\ce{H2O2}$ 后处理 |

### 6.4 铬氧化剂的环境与安全限制

铬(VI)试剂（PCC、PDC、Jones）产生大量含铬废水——$\mathrm{Cr^{6+}}$ 是剧毒致癌物。实验室趋向于用 Swern、DMP、TEMPO 等无金属方案替代。但在化竞考题中，Cr 试剂仍因"经典选择性"而频繁出现，需知晓其使用但不必在实际操作中首选。

## 七、常见比较与易混点

### 7.1 PCC vs PDC vs Jones 试剂：一张表理清

| | **PCC** | **PDC** | **Jones 试剂** |
|:---|:---|:---|:---|
| **组成** | $\ce{CrO3}$ + HCl + 吡啶 | $\ce{CrO3}$ + 吡啶（二铬酸吡啶盐） | $\ce{CrO3}$ + $\ce{H2SO4}$ + 丙酮/水 |
| **酸性** | 近中性（吡啶缓冲） | 弱碱性（吡啶过量） | 强酸性 |
| **一级醇产物** | **醛**（不过度） | 醛（无水条件）或**羧酸**（含水 DMF） | **羧酸**（直接到酸） |
| **二级醇产物** | 酮 | 酮 | 酮 |
| **对双键兼容** | 好（不氧化孤立双键） | 好 | 差（强酸 + 强氧化） |
| **操作** | 无水 $\ce{CH2Cl2}$ 中室温反应 | 同 PCC | 滴加到醇的丙酮溶液 |
| **记忆口诀** | PCC = **P**yridinium **C**hloro**c**hromate → 停在醛（"PCC，停一停"） | PDC = 用于从一级醇直接制酸（DMF 中） | Jones = 把醇往"死里"氧化到酸 |

### 7.2 Swern 氧化 vs Dess-Martin 氧化

| | **Swern 氧化** | **Dess-Martin 氧化（DMP）** |
|:---|:---|:---|
| **活性物种** | $\ce{Me2S^+-O-Cl}$（活化的 DMSO） | $\lambda^5$-碘烷（DMP） |
| **温度** | **$-78\,^\circ\mathrm{C}$**（严格低温） | **室温** |
| **无水要求** | 极严格（有水即失效） | 不严格（可容忍微量水） |
| **底物适用范围** | 一级醇 $\to$ 醛；二级醇 $\to$ 酮 | 同 Swern，且对酸敏感底物更友好 |
| **副产物** | $\ce{Me2S}$（恶臭）、$\ce{CO}$、$\ce{CO2}$ | 碘苯甲酸（无臭、易除去） |
| **操作性** | 三步加料，温度控制要求高 | 一锅加入，室温搅拌 |
| **工业化** | 差（低温 + 恶臭副产） | 好（室温 + 固体试剂） |
| **化竞中** | 常考其机理和条件要求 | 常作为"温和氧化"的首选答案 |
| **选择原则** | 经典考题必会；机理是活化 DMSO 类反应的模板 | 操作简便优先选 DMP；实验室实际首选 |

### 7.3 $\ce{OsO4}$ vs $\ce{KMnO4}$ 对烯烃的双羟基化

| | **$\ce{OsO4}$（催化）+ NMO** | **$\ce{KMnO4}$（冷、稀）** |
|:---|:---|:---|
| **立体化学** | **顺式加成**（syn） | **顺式加成**（syn，环状锰酸酯中间体） |
| **产率** | 高（$>90\%$），邻二醇不再被氧化 | 中等到低，易过度氧化裂解 |
| **对映选择性** | 可用 AD-mix（金鸡纳碱配体）实现不对称 | 不可 |
| **毒性/成本** | Os 贵且剧毒（但催化量使用） | Mn 便宜、低毒 |
| **反应条件** | $\ce{H2O/acetone}$ 或 $\ce{H2O/tBuOH}$，室温 | 低温水溶液 |
| **化竞考点** | 顺式立体化学 + 对映选择性 | 顺式立体化学 + 易过度氧化的陷阱 |

### 7.4 环氧化：过氧酸 vs $\ce{H2O2/NaOH}$

| | **过氧酸（mCPBA 等）** | **$\ce{H2O2/NaOH}$** |
|:---|:---|:---|
| **适用底物** | 普通烯烃 | **$\alpha,\beta$-不饱和羰基化合物**（Weitz-Scheffer 环氧化） |
| **机理** | 协同氧转移（Bartlett 机理，"蝴蝶型"过渡态） | 共轭加成-消除（$\ce{HOO^-}$ 对烯酮的 Michael 型加成） |
| **立体化学** | 保持烯烃几何：顺式 $\to$ 顺式环氧，反式 $\to$ 反式 | 同上保持 |
| **选择性** | 富电子双键优先（给电子取代基加速） | 仅对缺电子双键（$\alpha,\beta$-不饱和体系）有效 |
| **化竞易错** | 混淆适用范围：普通烯烃用 mCPBA，烯酮用 $\ce{H2O2/NaOH}$ | |

### 7.5 醛制备：一级醇氧化路线对比

这是合成中最常见的转化，也是考试中最高频的氧化判断题：

| 路线 | 试剂 | 是否停在醛 | 关键限制 |
|:---|:---|:---:|:---|
| A | PCC / $\ce{CH2Cl2}$ | ✅ | 无水、铬废物 |
| B | Swern | ✅ | 需 $-78\,^\circ\mathrm{C}$ |
| C | Dess-Martin | ✅ | 综合最优，价贵 |
| D | TEMPO / bleach | ✅ | 对二级醇慢，可实现一级选择性氧化 |
| E | Jones | ❌（到酸） | 酸性含水，一级醇直接到酸 |
| F | $\ce{KMnO4}$ / $\ce{H2O}$ | ❌（到酸） | 太强，无选择性 |

> 核心判断法则：**能否在一级醇阶段停住，取决于体系中是否有水 + 氧化剂是否过强。** 无水 + 温和氧化剂 = 停在醛；有水 + 强氧化剂 = 走到酸。

## 八、与其他知识点的联系

- **前置知识**：[[氧化数]]（判断氧化发生的基础）、[[官能团]]（识别可氧化位点）、[[亲电加成]]（烯烃氧化的对比机理）
- **直接对立知识**：[[有机还原反应]]——氧化与还原互为逆过程，常在同一合成题中交替出现。氧化阶梯的反向就是还原阶梯（酸 $\to$ 醛 $\to$ 醇）。
- **具体氧化方法**（本 KP 的下游详细条目）：
  - "[[Swern氧化]]——活化 DMSO 的温和氧化，$-78\,^\circ\mathrm{C}$ 无水"
  - "[[Dess-Martin氧化]]——$\lambda^5$-碘烷，室温中性氧化"
  - "[[Corey-Kim氧化]]——$\ce{Me2S\text{-}NCS}$，Swern 的前身变体"
  - "[[Wacker氧化]]——$\ce{PdCl2/CuCl2/O2}$ 催化末端烯烃 $\to$ 甲基酮"
  - "[[臭氧化反应]]——$\ce{O3}$ 裂解烯烃 $\to$ 醛/酮，结构解析利器"
  - "[[过碘酸氧化]]——$\ce{HIO4}$ 裂解邻二醇，糖化学核心工具"
- **关联合成策略**：
  - "[[官能团转化]]——氧化是官能团升级的主要手段"
  - "[[有机合成]]——氧化反应在逆合成分析中作为切断后的官能团引入步骤"
  - "[[保护基]]——氧化前常需保护敏感的羟基、氨基"
- **关联人名反应**：
  - "[[Baeyer-Villiger氧化]]——酮 $\to$ 酯，过氧酸氧化"
  - "[[Oppenauer氧化]]——醇 $\to$ 酮，$\ce{Al(OtBu)3}$ / 丙酮（Oppenauer 氧化是 Meerwein-Ponndorf-Verley 还原的逆反应）"
  - "[[Riley氧化]]——$\ce{SeO2}$ 氧化烯丙位 $\ce{CH2}$ $\to$ 羰基"
- **无机化学交叉**：[[氧化还原反应]]（无机）——有机氧化与无机氧化共享电子转移的本质，但判断标准从"化合价升降"转到"碳的氧化数变化"

## 九、典型题型

1. **氧化剂选择**：从指定原料到目标产物选择合适的氧化步骤。核心考察对不同氧化剂**选择性谱**的掌握——例如：底物含双键 + 一级醇，目标为保留双键的醛，应选 PCC/Swern/DMP 而非 Jones/KMnO₄。

2. **产物预测**：给定氧化剂和底物，预测主要产物。核心考察对氧化剂**反应范围和立体化学**的理解——例如：冷稀 KMnO₄ 与烯烃得顺式邻二醇；OsO₄ 也得顺式；臭氧化-还原裂解得醛/酮。

3. **氧化数变化分析**：判断反应是否为氧化及氧化程度。核心考察**碳氧化数计算**——从给定结构计算每个碳的氧化数变化，判断净氧化/还原发生。

4. **合成路线中的氧化步骤设计**：在较长的合成路线中规划氧化反应的时机和条件。核心考察**官能团兼容性**——后续步骤若需还原，氧化是否该先做？伯醇氧化到醛后是否会被后续碱性条件破坏？

5. **机理判断**：给定氧化反应条件和产物，判断经历何种中间体/过渡态。核心考察 Swern（活化 DMSO + 消除）、OsO₄（[3+2]环加成）、mCPBA（"蝴蝶型"过渡态）等机理。

## 十、例题

### 10.1 例题 1（★ 基础）——氧化阶梯应用

**题目**：设计从 1-丁醇到丁酸的转化，写出试剂和条件。如果要停在丁醛阶段，应如何修改？

**解答**：
- **到丁酸**：1-丁醇 $\ce{->[Jones试剂 (CrO3/H2SO4, 丙酮)]}$ 丁酸。Jones 试剂的酸性含水条件使一级醇直接氧化到羧酸。
- **停在丁醛**：将 Jones 换为 **PCC / $\ce{CH2Cl2}$**（无水），或使用 **Swern 氧化**（$\ce{DMSO, (COCl)2, Et3N}$, $-78\,^\circ\mathrm{C}$）或 **Dess-Martin 氧化**（DMP, $\ce{CH2Cl2}$, 室温）。关键在于**无水条件 + 温和氧化剂**。

> **辨析**：一级醇 $\to$ 醛 vs 一级醇 $\to$ 酸的区别，本质是体系中有无水。有水时醛以水合物形式存在，更易被继续氧化。

### 10.2 例题 2（★★ 化学选择性判断）——多官能团底物的氧化

**题目**：以下底物含有两个可氧化官能团（一级醇 + 双键），选择合适的条件分别实现：
(a) 只氧化一级醇到醛，保留双键
(b) 将双键转化为顺式邻二醇，保留醇羟基

底物：$\ce{HOCH2-CH2-CH=CH2}$

**解答**：

**(a) 选择性氧化一级醇（不触及双键）**：
- 选择 **PCC / $\ce{CH2Cl2}$** 或 **Swern 氧化** 或 **Dess-Martin 氧化**
- 这些氧化剂不与孤立双键反应
- 产物：$\ce{OHC-CH2-CH=CH2}$（3-丁烯醛）
- ❌ 不能选 Jones（酸性 + 强氧化可能进攻双键）或 $\ce{KMnO4}$（既氧化醇又氧化双键）

**(b) 选择性双羟基化（保留羟基）**：
- 选择 **$\ce{OsO4}$（催化量）/ NMO**（$\ce{H2O/acetone}$, 室温）
- $\ce{OsO4}$ 对醇羟基无影响
- 产物：$\ce{HOCH2-CH2-CHOH-CH2OH}$，立体化学为顺式加成
- ❌ 若用冷稀 $\ce{KMnO4}$，可能因醇羟基的存在加速过度氧化

> **核心逻辑**：在多官能团底物中选氧化剂 = 在"选择性谱"上找到恰好氧化目标官能团而不伤及无辜的那个点。

### 10.3 例题 3（★★★ 氧化数计数 + 合成规划）

**题目**：以下列转化中，计算起点和终点碳骨架中每个碳的氧化数变化，判断哪些步骤发生了净氧化/净还原，并推荐实现该转化的氧化/还原试剂。

转化：$\ce{CH3CH2CH2OH -> CH3CH2COOH}$

**解答**：

**氧化数计算**（氧化数 = 与电负性更高原子的键数 − 与电负性更低原子的键数）：

| 碳原子 | 起始（丙醇） | 终点（丙酸） | 氧化数变化 |
|:---|:---|:---|:---|
| $\ce{CH3-}$（C1） | $-3$（3 C-H 键） | $-3$（3 C-H 键） | $0$ |
| $\ce{-CH2-}$（C2） | $-2$（2 C-H + 1 C-C） | $-2$（2 C-H + 1 C-C） | $0$ |
| $\ce{-CH2OH}$（C3） | $-1$（1 C-O + 2 C-H） | $+3$（1 C=O + 1 C-O + 1 C-C） | **+4** |

- 只有 C3 发生了氧化（氧化数从 $-1 \to +3$，净上升 $+4$）
- 推荐试剂：**Jones 试剂**（$\ce{CrO3/H2SO4}$，丙酮）或 $\ce{KMnO4}$（热）

若欲停在 $\ce{CH3CH2CHO}$（丙醛，C3 氧化数 $+1$，净上升 $+2$），则应选 PCC / Swern / DMP。

### 10.4 例题 4（★★★ 综合实战）——竞赛风格

**题目**：某化合物分子式为 $\ce{C6H12O}$，经下列反应序列转化为 $\ce{C6H10O4}$（二元酸），推断原始结构并写出各步反应。

- 步骤 1：$\ce{C6H12O ->[PCC/CH2Cl2] C6H10O}$（不饱和醛）
- 步骤 2：$\ce{C6H10O ->[O3;  then Zn/H2O] C6H10O3}$（含一个醛基和一个酮基）
- 步骤 3：$\ce{C6H10O3 ->[Jones] C6H10O4}$（二元酸）

**解答**：

- 步骤 1：PCC 只氧化醇到羰基，产物少了 2 个 H $\to$ 醇被氧化为醛/酮，且分子出现不饱和度。$\ce{C6H12O}$（1 不饱和度，可能为环醇或烯醇）$\to$ $\ce{C6H10O}$（2 不饱和度）。
- 步骤 2：臭氧化-还原裂解 $\to$ 双键裂解为两个羰基，从 $\ce{C6}$ 到 $\ce{C6}$（未丢碳），说明底物为**环己烯醇**（双键在环内），裂解得开链二羰基化合物。
- 步骤 3：Jones 将醛氧化为羧酸，得二元酸 $\ce{C6H10O4}$。

推断原始结构：**2-环己烯-1-醇**（$\ce{C6H9OH}$，羟基在烯丙位）。

反应序列：
$$
\ce{C6H9OH ->[PCC] C6H8O (环己烯酮) ->[O3; Zn/H2O] OHC-(CH2)3-COCH3 ->[Jones] HOOC-(CH2)3-COOH}
$$

## 十一、易错点

1. ❌ **把"氧化"等同于"加氧"** $\to$ 有机氧化反应的本质是**碳氧化数升高**。脱氢（如醇 $\to$ 醛放出 $\ce{H2}$）也是氧化。判断标准永远是氧化数，不是原子种类。

2. ❌ **用 Jones 试剂氧化一级醇却期望停在醛** $\to$ Jones 试剂含水和强酸，一级醇会直接走到羧酸。停在醛必须用 PCC / Swern / DMP。

3. ❌ **忘记 Swern 氧化需要严格无水和 $-78\,^\circ\mathrm{C}$** $\to$ Swern 对水极度敏感；温度升高会导致 Pummerer 重排等副反应。考试中常设"室温 Swern"的陷阱。

4. ❌ **混淆 $\ce{OsO4}$ 和过氧酸环氧化** $\to$ $\ce{OsO4}$ 与烯烃得**顺式邻二醇**（两个 OH）；mCPBA 与烯烃得**环氧化物**（一个 O 桥）。两种反应虽都与烯烃反应，产物完全不同。

5. ❌ **混淆 $\ce{OsO4}$（顺式）与 $\ce{KMnO4}$（冷稀，也是顺式）** $\to$ 两者都得顺式邻二醇（都经过环状酯中间体），但 $\ce{KMnO4}$ 易过度氧化裂解，产率往往较低。考试中"冷稀 $\ce{KMnO4}$ 产率"经常成为扣分点。

6. ❌ **把 $\ce{KMnO4}$ 当作万能氧化剂滥用** $\to$ $\ce{KMnO4}$ 在合成题中往往是"错误答案"——它太过猛烈，官能团兼容性极差。只有特定场景（芳环侧链 $\to$ COOH、烯烃的彻底裂解分析）才用它。

7. ❌ **臭氧化后处理选错** $\to$ 还原后处理（$\ce{Zn/H2O}$ 或 $\ce{Me2S}$）$\to$ 醛/酮；氧化后处理（$\ce{H2O2}$）$\to$ 羧酸。选错后处理直接导致产物错误。

8. ❌ **搞混 Wacker 氧化的区域选择性** $\to$ Wacker 氧化中氧加到取代较多的碳上（生成甲基酮），而非反马氏。机理决定了这一选择性与马氏规则无关——是 $\ce{Pd}$ 的配位和迁移插入决定的。

9. ❌ **忽略邻二醇裂解在结构解析中的应用** $\to$ $\ce{HIO4}$ 裂解邻二醇是推断糖类、多羟基化合物结构的核心工具。考题中常给裂解产物反推原始结构，学生往往忘记 $\ce{HIO4}$ 的裂解规律（相邻二羰基不反应，相邻二羟基才裂解）。

10. ❌ **在用 PCC/PDC 时不注意无水条件** $\to$ PCC 和 PDC 都应在无水 $\ce{CH2Cl2}$ 中使用。如果溶剂含水，PCC 会部分水解产生 Jones 型条件，导致一级醇过度氧化到酸。

### 12.2 核心教学框架：氧化阶梯

氧化阶梯是串联本节所有氧化剂的最佳教学框架。将常见官能团按碳氧化数排列：

```
烷烃 (-3) ──→ 醇/烯烃 (-1) ──→ 醛 (+1) ──→ 羧酸 (+3) ──→ CO₂ (+4)
                     │                  │
                     └──→ 酮 (0, 二级碳) └──→ 酯 (+3, 保留碳数)
```

**教学要点**：
1. 让学生记住每个台阶的**氧化数**（不是死记，是通过结构推导）
2. 在每个箭头旁标注典型氧化剂：醇 $\to$ 醛用 PCC/Swern/DMP；醇 $\to$ 酸用 Jones/KMnO₄
3. 引入"选择性窗口"概念：氧化剂越强越容易跨越多个台阶；温和氧化剂才能精确停在某一级

### 12.3 学生易踩的认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析口诀 |
|---|---|---|---|
| "氧化就是加氧" | 初中化学的惯性定义 | 有机氧化看氧化数，脱氢也算氧化 | "加氧脱氢都算氧化，氧化数升高才是本质" |
| "KMnO₄ 是万能氧化剂" | KMnO₄ 在无机化学中无处不在 | 有机合成中 KMnO₄ 选择性极差，只在特定场景使用 | "KMnO₄ 是好工具但不是瑞士军刀" |
| "PCC 和 Jones 差不多" | 都含 Cr(VI) | PCC 无水停在醛，Jones 含水走到酸，天壤之别 | "PCC 是手术刀，Jones 是锤子" |
| "Swern 和 DMP 没区别" | 都做同样的转化 | Swern 需 -78°C 无水，DMP 室温即可。考试中选 Swern 却写室温条件是典型错误 | "Swern 要冷，DMP 常温行" |
| "所有双羟基化都一样" | 记不住 OsO₄ vs KMnO₄ 的区别 | 两者都得顺式产物，但 OsO₄ 产率高且可选对映选择性 | "OsO₄ 高贵，KMnO₄ 粗暴" |

### 12.4 入门级例题（课堂上手题）

**题目**：以下三种一级醇分别用什么条件可氧化到醛？能否用 Jones 试剂？为什么？

a) $\ce{CH3CH2CH2OH}$（正丙醇）
b) $\ce{HOCH2CH2CH=CH2}$（3-丁烯-1-醇）
c) $\ce{HOCH2CH2CH2OH}$（1,3-丙二醇）

**预期解答路径**：
- a) 正丙醇：PCC/Swern/DMP 均可；不能用 Jones（会到丙酸）。若题目只要求到醛，PCC 是最经典答案。
- b) 含双键：PCC/Swern/DMP 均可（不氧化孤立双键）；Jones 不可（酸性可能使双键质子化/重排）。这里要引导注意**官能团兼容性**。
- c) 二元醇：两端一级醇都可被氧化。若只到一个醛，需 1 eq PCC；若到二醛，需 2 eq PCC。引导注意**化学计量**在多元醇氧化中的考量。

**老师可引导的提问**：
- "为什么 Jones 试剂里的水是'坏'的？"（水使醛以偕二醇形式存在，易被继续氧化）
- "如果不用铬试剂，有哪些替代方案？"（Swern、DMP、TEMPO——引出绿色化学概念）

### 12.5 与现实/直觉的连接

- **氧化阶梯 = 能量阶梯**：氧化数越高的官能团越"氧化"，能量越低、越稳定。从烷烃到 CO₂ 是热力学下坡路——有机氧化反应的挑战不是"能不能反应"，而是"怎么停在中间某一级"。
- **类比**：氧化剂的选择像调温旋钮——KMnO₄ 是"大火猛烧"（直接烧到 CO₂），PCC 是"小火慢炖"（停在醛），Swern/DMP 是"恒温精准控温"。合成化学家的工作就是为每个反应找到正确的那一档"火力"。
- **绿色化学视角**：传统铬氧化产生大量致癌废水，现代趋势用 Swern（副产物 DMSO）、DMP（副产物碘苯甲酸，可回收）或催化氧化（TEMPO/bleach，O₂ 为终端氧化剂）替代。化竞考题仍以 Cr 试剂为主，但在实际科研中已逐步退出。

## 十三、竞赛拓展

1. **Sharpless 不对称环氧化（AE, Asymmetric Epoxidation）**：$\ce{Ti(OiPr)4}$ + 酒石酸二乙酯（DET 或 DIPT）+ $\ce{tBuOOH}$，专一性地对**烯丙醇**进行对映选择性环氧化。酒石酸酯的手性（(+)-DET vs $(-)$-DET）决定环氧的绝对构型。这是化竞决赛级考点——要求学生从烯丙醇的几何构型预测环氧产物的立体化学（Sharpless 模型：将烯丙醇的 C=C 平面与酒石酸酯配位后，$\ce{tBuOOH}$ 从特定面对映选择性地传递氧）。

2. **Jacobsen 环氧化**：Mn(III)-salen 配合物催化**非官能团化烯烃**的对映选择性环氧化，用 bleach（NaOCl）为终端氧化剂。区别于 Sharpless AE：Jacobsen 不要求底物有导向基团（如烯丙醇的 OH），对顺式烯烃的 ee% 尤其出色。机理涉及 Mn(V)-oxo 活性物种。

3. **Sharpless 不对称双羟基化（AD, Asymmetric Dihydroxylation）**：$\ce{OsO4}$（催化量）+ 金鸡纳碱衍生物配体（(DHQ)$_2$PHAL 或 (DHQD)$_2$PHAL）+ NMO 或 $\ce{K3Fe(CN)6}$（共氧化剂）。AD-mix-$\alpha$ 和 AD-mix-$\beta$ 是商业化的预混试剂。烯烃的面选择性可以通过 mnemonic 模型预测。

4. **Ley-Griffith 氧化（TPAP）**：四丙基过钌酸铵（$\ce{Pr4N^+RuO4^-}$，TPAP）+ NMO 共氧化剂。催化量 Ru(VIII)，对一级醇和二级醇均有效，条件温和（室温），比 Cr 试剂环保。在化竞中超纲但在决赛/选拔赛中偶尔出现。

5. **NMO 作为共氧化剂**：N-甲基吗啉-N-氧化物（NMO）在 $\ce{OsO4}$ 和 TPAP 催化氧化中起关键作用——将被还原的低价金属（Os(VI)或 Ru(VI)）重新氧化为高价活性物种（Os(VIII)或 Ru(VIII)），实现催化循环。这是理解"催化氧化"概念的核心实例。

6. **高价碘氧化剂（Hypervalent Iodine Oxidants）**：
   - **DMP（Dess-Martin periodinane）**：$\lambda^5$-碘烷，最常用的温和氧化剂
   - **IBX（2-碘酰基苯甲酸）**：DMP 的前体，水溶性差但在某些场景下选择性不同
   - **PIDA / PIFA**：三氟乙酰氧基碘苯，用于氧化偶联和重排反应
   - 化竞初赛一般只要求 DMP；国决可能涉及 IBX 与 DMP 的比较。

7. **需氧氧化（Aerobic Oxidation）**：以 $\ce{O2}$（空气）为终端氧化剂的绿色氧化方法。典型体系：
   - **Wacker 氧化**：$\ce{PdCl2/CuCl2/O2}$，烯烃 $\to$ 甲基酮
   - **TEMPO 催化氧化**：$\ce{TEMPO + NaOCl}$（bleach）或 $\ce{TEMPO + O2/Cu}$ 体系
   - 这是现代合成化学的前沿方向，初赛不要求机理但可能以"绿色化学"形式出现在材料题中。

8. **烯丙位/苄位氧化**：
   - **$\ce{SeO2}$（Riley 氧化）**：烯丙位 $\ce{CH2}$ $\to$ $\ce{C=O}$，或 $\ce{CH3}$ $\to$ $\ce{CH2OH}$（烯丙位羟基化）
   - **$\ce{CrO3}$/吡啶**（条件依赖）：可氧化苄位 $\ce{CH2}$ $\to$ $\ce{C=O}$
   - 国决中烯丙位氧化的区域选择性判断是常见难点。

9. **Baeyer-Villiger 氧化**：酮 + 过氧酸（mCPBA 或 $\ce{CF3CO3H}$）$\to$ 酯。迁移倾向决定哪个基团迁移到氧上：$\ce{H > 叔烷基 > 仲烷基 > 苯基 > 伯烷基 > 甲基}$。这是氧化反应中涉及骨架重排的特例，国决必会。

10. **Rubottom 氧化**：烯醇硅醚 + mCPBA $\to$ $\alpha$-羟基酮（经环氧化-开环序列）。用于在酮的 $\alpha$ 位引入羟基，是合成 $\alpha$-羟基羰基化合物的重要策略。
