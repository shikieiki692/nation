# 邻二醇切断（Glycol Cleavage）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[37-加成反应]]、[[43-亲电加成反应]]

---

## 一、定义

**邻二醇切断（Glycol Cleavage）**：1,2-二醇（邻二醇）在**高碘酸（HIO₄）**或**四醋酸铅（Pb(OAc)₄）**作用下，C-C 键断裂生成两分子羰基化合物（醛或酮）的反应。

**本质**：邻二醇的 C-C 键被氧化断裂，每个碳上连有一个羟基的 C-C 键变为 C=O 双键。

## 三、核心原理

### 3.1 反应通式

```
R-CH(OH)-CH(OH)-R'  +  HIO₄  →  R-CHO  +  R'-CHO  +  HIO₃  +  H₂O
```

- 每个带 OH 的碳失去 H 和 OH，形成 C=O
- 中间的 C-C 键断裂

### 3.2 机理——五元环酯中间体

**HIO₄ 机理**：
```
步骤 1：HIO₄ 与邻二醇的两个 OH 配位
步骤 2：形成环状 **高碘酸酯五元环**
步骤 3：电子转移 → C-C 键断裂
步骤 4：生成两分子醛/酮 + HIO₃
```

**关键特征**：
- **顺式邻二醇切断更快**——两个 OH 在同一侧，更容易与 HIO₄ 形成五元环
- 反式邻二醇也能反应，但速率慢得多（需要翻转构象）
- **定量反应**——1 mol 邻二醇消耗 1 mol HIO₄

### 3.3 Pb(OAc)₄ 与 HIO₄ 的对比

| | **HIO₄** | **Pb(OAc)₄** |
|:---|:---|:---|
| **试剂状态** | 水溶液 | 有机溶剂（如苯、氯仿）|
| **溶剂** | 水/水-醇 | 非质子有机溶剂 |
| **机理** | 五元环高碘酸酯 | 五元环铅酸酯 |
| **选择性** | 对顺式更快 | 类似 |
| **应用** | 水溶底物 | 脂溶底物 |
| **毒性** | 较低 | 铅有毒 |

### 3.4 产物类型推断

| 邻二醇类型 | 切断产物 |
|:---|:---|
| **R-CH(OH)-CH(OH)-R'** | R-CHO + R'-CHO（两分子醛）|
| **R₂C(OH)-CH(OH)-R'** | R₂C=O + R'-CHO（酮 + 醛）|
| **R₂C(OH)-C(OH)R'₂** | R₂C=O + R'₂C=O（两分子酮）|
| **环状邻二醇** | 二醛 / 二酮 / 酮醛（环打开）|

---

## 四、关键结论

1. **邻二醇切断 = C-C 键断裂 + 各碳氧化为羰基**
2. **顺式邻二醇切断更快**——五元环过渡态更易形成
3. **1 mol HIO₄ 切 1 mol 邻二醇**——定量反应
4. **结构推断的利器**——从产物反推邻二醇，再反推原烯烃
5. **与臭氧化形成互补**——臭氧化从烯烃推断；邻二醇切断从邻二醇推断

---

## 五、常见分类或情形

### 5.1 按试剂分类

| 试剂 | 溶剂 | 适用底物 | 特点 |
|:---|:---|:---|:---|
| **HIO₄** | H₂O / MeOH | 水溶性邻二醇 | 最常用，可定量滴定 |
| **Pb(OAc)₄** | 苯 / CHCl₃ | 脂溶性邻二醇 | 无水条件，有机相 |
| **NaIO₄** | H₂O | 水溶性 | HIO₄ 的盐形式，更稳定 |

### 5.2 按底物分类

| 底物 | 产物 | 应用 |
|:---|:---|:---|
| **链状邻二醇** | 两分子醛/酮 | 结构鉴定 |
| **环状邻二醇（顺式）**| 二醛/二酮（环打开）| 环状结构推断 |
| **糖类** | 多醛片段 | 糖结构分析（经典方法）|

---

## 六、适用条件与限制

1. **必须是 1,2-二醇**——1,3-二醇不被切断
2. **α-羟基醛/酮也可被切断**——因为可以烯醇化为邻二醇形式
3. **氨基醇（α-氨基醇）也可被 HIO₄ 切断**——类似机理
4. **邻二醇必须可接近**——大位阻可能减慢反应
5. **HIO₄ 是强氧化剂**——其他易氧化基团可能被影响

---

## 七、常见比较与易混点

### 7.1 邻二醇切断 vs 臭氧化

| | **邻二醇切断（HIO₄）** | **臭氧化（O₃）** |
|:---|:---|:---|
| **底物** | 邻二醇 | 烯烃 |
| **断裂位置** | C-C（连两个 OH）| C=C |
| **产物** | 醛/酮 | 醛/酮/酸（取决于后处理）|
| **应用** | 从邻二醇推断结构 | 从烯烃推断结构 |
| **关系** | 烯烃 → 双羟化 → 邻二醇 → HIO₄ 切断 | 烯烃 → O₃ 切断 |

**经典推断链**：
```
烯烃 → [双羟化] → 邻二醇 → [HIO₄ 切断] → 醛/酮
     → [臭氧化] → 醛/酮
```

### 7.2 顺式 vs 反式邻二醇的切断速率

- **顺式**：两个 OH 在同一侧 → 易与 HIO₄ 形成五元环 → **快**
- **反式**：两个 OH 在异侧 → 需要构象翻转才能形成五元环 → **慢**
- 此差异可用于**区分顺/反邻二醇**

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：写出顺-1,2-环己二醇被 HIO₄ 切断的产物。

**解答**：
- 环状邻二醇切断 → 环打开
- 产物：**己二醛**（OHC-(CH₂)₄-CHO）

### 10.2 例题 2（★★ 结构推断）

**题目**：某邻二醇 A 被 HIO₄ 切断后得到甲醛和丙醛。A 的结构是什么？

**解答**：
- HCHO 来自 =CH₂ 片段
- CH₃CH₂CHO 来自 =CH-CH₂-CH₃ 片段
- 将两个片段的羰基去掉，用 C-C 单键连接（邻二醇）
- A = **1,2-丁二醇**（HO-CH₂-CH(OH)-CH₂-CH₃）

### 10.3 例题 3（★★★ 糖化学）

**题目**：D-葡萄糖用 HIO₄ 处理，产物是什么？消耗多少摩尔 HIO₄？

**解答**：
- D-葡萄糖有 5 个邻二醇结构（直链式中有 4 个邻位 OH 对）
- 实际上环状葡萄糖的邻二醇切断较复杂
- 直链式：HOCH₂-CH(OH)-CH(OH)-CH(OH)-CH(OH)-CHO
- 有 4 个相邻 C-C 键可被切断（C1-C2, C2-C3, C3-C4, C4-C5）
- 消耗 **5 mol HIO₄**（包括 C1 的醛基被氧化）
- 产物：5 HCOOH + HCHO

## 十一、易错点

1. ❌ **认为 1,3-二醇也能被切断** → HIO₄ **只切 1,2-二醇**
2. ❌ **忽略顺/反速率差异** → 顺式比反式快得多
3. ❌ **忘记 α-羟基醛/酮也可被切断** → 它们可烯醇化为邻二醇形式
4. ❌ **与臭氧化产物混淆** → 邻二醇切断和臭氧化是两条不同的推断链
5. ❌ **环状邻二醇切断后忘记环已打开** → 产物是链状二醛/二酮
