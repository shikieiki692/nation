# 加成反应

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[37-加成反应]]

## 一、定义

**加成反应（Addition Reaction）** 是指两个或多个分子结合形成一个产物的反应，其特征是**不饱和键（π 键或环张力键）断裂**，形成两个新的 σ 键，不饱和度降低。

$$
\ce{A=B + C-D -> A-B-C-D}
$$

- **核心特征**：反应前后分子的**不饱和度下降**（双键/三键减少，或环打开）
- **原子经济性**：理想情况下可达 **100%**（所有原子进入产物），是绿色化学的重要指标
- **判断标志**：产物比底物多了新的 σ 键，且少了 π 键或环

## 三、核心原理

### 3.1 电子推动的三种模式

加成反应按进攻物种的电子特性分为三大类：

| 类型 | 进攻物种 | 底物特征 | 电子流向 | 典型反应 |
|:---|:---|:---|:---|:---|
| **亲电加成（AdE）** | 缺电子亲电体（E⁺）| 富电子 π 键（烯烃、炔烃）| π 电子 → E⁺，随后 Nu⁻ 捕获碳正离子 | HX、Br₂、H₂O 对烯烃加成 |
| **亲核加成（AdN）** | 富电子亲核体（Nu⁻）| 极化重键（C=O、C≡N）| Nu⁻ → 羰基碳，随后质子化 | Grignard、NaBH₄、CN⁻ 对羰基加成 |
| **自由基加成（AdR）** | 中性自由基（·）| 不饱和键 | 单电子分步加成 | HBr/过氧化物反马氏加成 |
| **协同加成** | 无独立进攻物种 | 双烯 + 亲双烯体 / 卡宾 | 环状过渡态同步成键 | Diels-Alder、Simmon-Smith |

### 3.2 加成反应的立体化学本质

加成到不饱和键时，新键可从 **同面（syn）** 或 **异面（anti）** 形成：

| 加成模式 | 定义 | 典型实例 |
|:---|:---|:---|
| **syn-加成** | 两个新 σ 键从同一侧形成 | 催化氢化（H₂/Pd）、硼氢化-氧化 |
| **anti-加成** | 两个新 σ 键从相对侧形成 | Br₂/CCl₄ 对烯烃加成（经由溴鎓离子）|
| **无立体选择性** | 两种模式均有，得外消旋或混合物 | HX 对非环烯烃加成（碳正离子平面）|

## 四、关键结论

1. **亲电加成的区域选择性由碳正离子稳定性决定**（马氏规则的本质）
2. **亲核加成的立体选择性由位阻控制**（Nu⁻ 从位阻较小面进攻）
3. **共轭体系存在 1,2-加成 vs 1,4-加成的竞争**：动力学控制 → 1,2-；热力学控制 → 1,4-
4. **自由基加成的区域选择性由自由基稳定性决定**（反马氏规则的本质）
5. **协同加成严格遵守立体专一性**：syn 底物得 syn 产物，anti 底物得 anti 产物

## 五、常见分类

### 5.1 按进攻物种分类

```
加成反应
├── 亲电加成（AdE）
│   ├── 烯烃 + HX → 卤代烷
│   ├── 烯烃 + X₂ → 邻二卤代物
│   ├── 烯烃 + H₂O（酸催化）→ 醇
│   └── 炔烃 + 2HX → 偕二卤代物
├── 亲核加成（AdN）
│   ├── 醛/酮 + RMgX → 醇
│   ├── 醛/酮 + NaBH₄/LiAlH₄ → 醇
│   ├── α,β-不饱和羰基 + Nu⁻ → 1,2- 或 1,4-加成
│   └── 醛 + HCN → 氰醇
├── 自由基加成（AdR）
│   └── 烯烃 + HBr/ROOR → 反马氏溴代烷
└── 协同加成
    ├── [4+2] 环加成（Diels-Alder）
    ├── [2+2] 光环加成
    ├── 卡宾加成（Simmon-Smith）
    └── 氢硼化（BH₃·THF，协同但非环状过渡态）
```

### 5.2 按底物分类

| 底物类型 | 典型加成反应 | 主要类型 |
|:---|:---|:---|
| **孤立烯烃** | HX、X₂、H₂O、H₂、BH₃ | AdE、AdR、协同 |
| **共轭二烯** | Br₂、HX（1,2 vs 1,4）| AdE（含选择性）|
| **炔烃** | HX（分步，可停在一分子）、H₂（Lindlar 部分还原）| AdE |
| **醛/酮** | RMgX、NaBH₄、HCN、Wittig | AdN |
| **α,β-不饱和羰基** | 硬 Nu⁻（1,2-）、软 Nu⁻（1,4-）| AdN |
| **环丙烷** | H₂/Cat、Br₂（类似烯烃）| AdE |

### 6.2 加成 vs 取代的快速判别

| 判别维度 | 加成 | 取代 |
|:---|:---|:---|
| **不饱和度变化** | 下降（+2 H 或 +X-Y）| 不变 |
| **底物特征** | 含 π 键或环张力 | 含 LG 的饱和碳 |
| **试剂特征** | 常无双离去基 | 有明确 LG 离去 |
| **产物数** | 单一产物（理想）| 产物 + LG 副产物 |

## 七、常见比较

### 7.1 1,2-加成 vs 1,4-加成（共轭烯酮体系）

| 维度 | 1,2-加成（直接加成）| 1,4-加成（共轭加成 / Michael 加成）|
|:---|:---|:---|
| **进攻位点** | 羰基碳（C=O）| β-碳（Cβ）|
| **中间体** | 烷氧负离子（稳定）| 烯醇负离子（可逆）|
| **动力学/热力学** | 动力学控制（低温、短反应时间）| 热力学控制（高温、长反应时间）|
| **亲核体偏好** | 硬亲核体（RLi、RMgX、LiAlH₄）| 软亲核体（CuR、烯胺、硫醇）|
| **产物稳定性** | 烯丙醇（较不稳定，可逆）| 饱和羰基（更稳定）|
| **后续转化** | 直接得醇 | 可进一步烷基化（Robinson 环化）|

**ABOC 记忆法**："硬上頭，软下腳"——硬亲核体打羰基碳（上头），软亲核体打 β-碳（下脚）。

### 7.2 马氏 vs 反马氏加成

| 规则 | 本质 | 适用条件 | 典型反应 |
|:---|:---|:---|:---|
| **马氏规则** | 碳正离子稳定性 | 亲电加成（HX、H₂O）| HBr + 丙烯 → 2-溴丙烷 |
| **反马氏规则** | 自由基稳定性 | 自由基加成（HBr/ROOR）| HBr/过氧化物 + 丙烯 → 1-溴丙烷 |
| **硼氢化-氧化** | 位阻控制 + 协同 syn | BH₃·THF 后 H₂O₂/NaOH | 烯烃 → 反马氏醇（syn）|

## 十、例题

### 例题 1：加成反应类型与选择性判断

**题目**：预测下列反应的主产物，并指出加成类型：
1. (CH₃)₂C=CH₂ + HBr（无过氧化物）
2. (CH₃)₂C=CH₂ + HBr（ROOR）
3. CH₃CH=CHCHO + Me₂CuLi
4. CH₃CH=CHCHO + MeLi

**解析**：
1. **马氏亲电加成** → (CH₃)₂CHCH₂Br（叔碳正离子更稳定）
2. **反马氏自由基加成** → (CH₃)₂CHCH₂Br 的反向？不，HBr/ROOR 时 H· 先加，Br· 后加。H· 加在取代多的碳上（生成更稳定的 2° 自由基），Br· 加在末端 → **(CH₃)₂CHCH₂Br**？
   
   纠正：丙烯型。H· 加在末端 CH₂ 上，生成中间碳自由基（2°），Br· 再加 → **(CH₃)₂CBrCH₃**？不对，重新分析：
   
   底物是 (CH₃)₂C=CH₂（异丁烯）。
   - 马氏：H⁺ 加在 CH₂ 上，生成 (CH₃)₂C⁺CH₃（3° 碳正）→ Br⁻ 捕获 → **(CH₃)₂CBrCH₃**
   - 反马氏自由基：H· 加在 =C(CH₃)₂ 上，生成 ·CH₂-CH(CH₃)₂（1° 自由基？不，H 加到双键碳上：
     H· + (CH₃)₂C=CH₂ → (CH₃)₂CH-CH₂· 或 (CH₃)₂C·-CH₃？
     自由基稳定性：3° > 2° > 1°。H· 加到 =CH₂ 端，生成 (CH₃)₂C·-CH₃（2°？不，是 3° 自由基）——等等，异丁烯是 (CH₃)₂C=CH₂：
     - C1: (CH₃)₂C=
     - C2: =CH₂
     H· 加到 C1 → ·C(CH₃)₂-CH₃（这不是自由基了，是饱和碳）。H· 只能加到双键碳上：
     H· + C1 → (CH₃)₂CH-CH₂·（1° 自由基）
     H· + C2 → (CH₃)₂C·-CH₃（2° 自由基？不，(CH₃)₂C·-CH₃ 是叔丁基自由基，3°）
     所以 H· 加到 C2（=CH₂），生成 3° 自由基 (CH₃)₃C·，然后 Br· 加到这个自由基上？不，是自由基反应当中 Br· 从 HBr 中夺取 H，形成新的碳自由基？
     正确机理：Br· + HBr → Br₂ + H·？不。
     
     标准 HBr/ROOR 机理：
     1. ROOR → 2 RO·
     2. RO· + HBr → ROH + Br·
     3. Br· + CH₂=C(CH₃)₂ → ·CH₂-CBr(CH₃)₂？不，Br· 加到双键上：
        Br· 加到 C2（=CH₂）→ BrCH₂-C·(CH₃)₂（3° 自由基）
        Br· 加到 C1（=C）→ BrC·(CH₃)₂-CH₂·（不可能，双键断裂后一个碳上只有一个键）
        正确：Br· 加到 =CH₂ 上 → Br-CH₂-C·(CH₃)₂（3° 自由基）
     4. BrCH₂-C·(CH₃)₂ + HBr → BrCH₂-CH(CH₃)₂ + Br·
     
     所以反马氏产物是 **BrCH₂-CH(CH₃)₂**（1-溴-2-甲基丙烷）。
     
     马氏产物是 **(CH₃)₂CBrCH₃**（2-溴-2-甲基丙烷）。

3. **Me₂CuLi（软亲核体）→ 1,4-共轭加成** → CH₃CH=CHCOCH₃ 的 β-位？不，Me 加到 β-碳，羰基保留 → **CH₃CH(CH₃)CH₂CHO**（饱和醛）
4. **MeLi（硬亲核体）→ 1,2-直接加成** → Me 加到羰基碳 → **CH₃CH=CHCH(OH)CH₃**（烯丙醇）

### 例题 2：立体化学判断（ABOC 2.8-2 改编）

**题目**：顺-2-丁烯 + Br₂/CCl₄ 的产物立体化学？

**解析**：
- Br₂ 对烯烃加成 **经溴鎓离子中间体**
- 第二个 Br⁻ 从溴鎓离子背面进攻（anti-加成）
- 顺式底物 → **外消旋的反式-1,2-二溴-2-丁烯**？不，是 2,3-二溴丁烷。
- 顺-2-丁烯 + Br₂ → **(2R,3R) + (2S,3S)** 一对对映异构体（meso 的前体）
- 实际上顺-2-丁烯经 anti 加成得 **外消旋体（dl 对）**，反-2-丁烯经 anti 加成得 **meso 体**

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| 把 HX 对炔烃加成当成一步完成 | 忽略炔烃可停在一分子加成 | HX 对炔烃可控制条件停在一分子，得乙烯基卤化物 | 为什么炔烃与 HX 加成可停在一分子，而烯烃通常不停？ |
| 1,4-加成产物画错结构 | 误以为 1,4-加成后双键在 α,β 位 | 1,4-加成后双键移到 Cα-Cβ 之间？不，是 Cα=O 保留，Cβ-Cγ 成单键 | 画出 1,4-加成后产物的完整结构，标出羰基位置 |
| 混淆 syn-加成与马氏规则 | syn 是立体化学，马氏是区域化学 | 两者独立：syn 加成可以是马氏或反马氏（如硼氢化是 syn+反马氏）| 硼氢化-氧化同时满足什么区域选择性和立体选择性？ |
| 认为所有协同加成都是 syn | [2+2] 光环加成可有不同立体化学 | [4+2] 环加成一般是 syn（endo 规则）；[2+2] 光环加成也保持烯烃构型 | Diels-Alder 反应中双烯体构型是否在产物中保留？ |
| 把过氧化物效应推广到 HCl 和 HI | 只记住"ROOR → 反马氏"，未理解机理 | 只有 HBr 有过氧化物效应；HCl 的 H-Cl 键太强，HI 的 I· 太不稳定 | 为什么 HCl 和 HI 不能发生反马氏自由基加成？ |

|:---|:---|:---|
| "加成就是加东西，取代就是换东西" | 停留在字面理解 | 强制使用**不饱和度计数法**：反应前后数双键+环数 |
| "马氏规则是规律，反马氏是例外" | 把反马氏当成反常 | 强调两者都是稳定性原理：马氏 = 碳正稳定；反马氏 = 自由基稳定 |
| "亲核加成只发生在羰基" | 忽略烯烃的某些亲核加成 | 案例：卡宾对烯烃的加成本质上是亲电，但 Simmon-Smith 的 CH₂I₂/Zn(Cu) 是亲核性卡宾 |
| "1,4-加成就是 Michael 加成" | 混淆概念 | Michael 加成是 1,4-加成的一种（特定亲核体 + α,β-不饱和羰基）|

### 12.3 入门例题

**最简判别题**：
> 下列哪个是加成反应？
> A. CH₃CH₂Br + NaOH → CH₃CH₂OH + NaBr  
> B. CH₂=CH₂ + Br₂ → CH₂BrCH₂Br  
> C. CH₃CH₂OH → CH₂=CH₂ + H₂O  
> D. C₆H₆ + HNO₃ → C₆H₅NO₂ + H₂O

**答案：B**（双键断裂，两个 Br 加进去，不饱和度下降）
**引导**：如果学生选 A，问"Br 去哪了？是不是有副产物？加成反应的理想状态是没有小分子副产物。"

### 12.4 现实类比

**「拼图拼接」类比加成反应**：
- 不饱和键（双键）= 两块拼图之间只有一个连接点（π 键），容易拆开
- 加成试剂 = 另外两块小拼图
- 加成过程 = 把双键拆开，分别接上两块新拼图，形成更牢固的两点连接（两个 σ 键）
- 协同加成 = 四块拼图同时咬合（Diels-Alder 像两个拼图块从两侧同时扣上）

## 十三、竞赛拓展

### 13.1 竞赛中的特殊加成反应

| 反应 | 类型 | 竞赛价值 | 典型应用 |
|:---|:---|:---|:---|
| **硼氢化-氧化** | 协同 syn + 反马氏 | 烯烃 → 醇的区域/立体精准控制 | 全合成中立体构建 |
| **羟汞化-去汞** | AdE（马氏）| 烯烃 → 醇，不重排 | 比酸催化水化更温和 |
| **Oxymercuration-Demercuration** | AdE | 马氏水化，无碳正重排 | 复杂烯烃的水化 |
| **双羟化（OsO₄ / KMnO₄）** | 协同 syn | 烯烃 → 邻二醇 |  Sharpless AD 的前置 |
| **氢甲酰化（Hydroformylation）** | AdR + CO 插入 | 烯烃 → 醛 | 工业合成（Oxo 过程）|
| **Baylis-Hillman 反应** | 亲核加成 + 亲电加成串联 | α,β-不饱和羰基 + 醛 → 高烯丙醇 | 构建复杂骨架 |

### 13.2 与不对称合成的交叉

- **Sharpless 双羟化**：OsO₄ + 手性配体 → 烯烃不对称 syn-双羟化
- **Noyori 氢化**：Ru(BINAP) 催化 → 羰基不对称氢化（AdE 的催化不对称版）
