# 糖的构型与变旋（Sugar Configuration and Mutarotation）

## 一、定义

**糖的构型**指单糖分子中手性碳原子的空间排布，通常用 **D/L 标记法**表示。由于单糖分子含有多个手性中心，其构型由离羰基最远的那个手性碳（即编号最大的手性碳）上的羟基方向决定。

**变旋现象（Mutarotation）**是指某些旋光性化合物在溶液中放置时，其比旋光度逐渐变化最终达到恒定值的现象。对单糖而言，这是由于**环式结构的 α 型和 β 型之间通过开链形式发生动态平衡**所致。

- **D-型糖**：Fischer 投影式中，离羰基最远的手性碳上的 —OH 在**右侧**
- **L-型糖**：Fischer 投影式中，该 —OH 在**左侧**
- **天然糖几乎均为 D-型**（D-葡萄糖、D-果糖、D-半乳糖等）

## 三、核心原理

### 3.1 D/L 构型判定

**判定规则**（Fischer-Rosanoff 约定）：

1. 写出单糖的 Fischer 投影式（主链竖直，羰基在上）
2. 找到**编号最大的手性碳**（离羰基最远）
3. 观察该碳上 —OH 的方向：
   - **右侧 → D-型**
   - **左侧 → L-型**

**注意**：D/L 标记与 R/S 标记**没有固定对应关系**。一个 D-型糖可能含有 R 和 S 构型的手性中心。

**常见单糖的 D/L 类型**：

| 单糖 | D/L | 手性碳数 |
|:---|:---:|:---:|
| 甘油醛 | D 或 L | 1 |
| 葡萄糖 | D | 4 |
| 果糖 | D | 3 |
| 半乳糖 | D | 4 |
| 核糖 | D | 3 |
| 脱氧核糖 | D | 2 |

### 3.2 单糖的环式结构

**成环原理**：单糖分子中的羰基（醛基或酮基）与同一分子内的羟基发生分子内亲核加成，形成**半缩醛**（醛糖）或**半缩酮**（酮糖）。

| 环大小 | 名称 | 成环原子 | 示例 |
|:---:|:---|:---|:---|
| 5 元环 | **呋喃糖（Furanose）** | 4 个碳 + 1 个氧 | 果糖（主要）、核糖 |
| 6 元环 | **吡喃糖（Pyranose）** | 5 个碳 + 1 个氧 | 葡萄糖（主要）、半乳糖、甘露糖 |

**为什么葡萄糖主要以吡喃糖形式存在？**
- 吡喃糖环的键角张力更小（接近 109.5°）
- 热力学上更稳定
- 在水溶液中 > 99% 为吡喃糖形式

### 3.3 α/β 端基异构体（Anomers）

**定义**：环式单糖中，新形成的手性碳（原羰基碳，称为**端基碳 anomeric carbon**）上的 —OH 有两种构型，二者互为**端基异构体**。

**判定规则**：

| 构型 | Fischer 投影式 | Haworth 式（D-吡喃糖）| Chair 式（D-吡喃糖）|
|:---|:---|:---|:---|
| **α** | 端基 —OH 与 **C5 的 CH₂OH 在异侧**（向下）| 端基 —OH 在环平面**下方** | 端基 —OH 在**直立键（a 键）** |
| **β** | 端基 —OH 与 **C5 的 CH₂OH 在同侧**（向上）| 端基 —OH 在环平面**上方** | 端基 —OH 在**平伏键（e 键）** |

**关键规律（D-糖）**：
- Haworth 式中：端基 —OH 与 C5 的 CH₂OH **同侧为 β，异侧为 α**
- 对 D-吡喃糖：β 型更稳定（端基 —OH 在平伏键，空间位阻小）
- 平衡时：D-葡萄糖水溶液中 α : β ≈ 36 : 64

### 3.4 变旋现象的原理

**过程**：

```
        开链形式（~0.02%）
          ↙         ↘
    α-吡喃糖 ←————→ β-吡喃糖
```

- **起始状态**：纯 α-D-葡萄糖的比旋光度 [α]D = +112°
- **起始状态**：纯 β-D-葡萄糖的比旋光度 [α]D = +18.7°
- **平衡状态**：混合物的比旋光度 [α]D = +52.7°
- **变化方向**：纯 α → 逐渐降低；纯 β → 逐渐升高；最终都达到 +52.7°

**发生变旋的条件**：
- 分子中含有**游离的半缩醛/半缩酮羟基**（端基 OH）
- 能够发生**开环-成环**平衡
- **糖苷**（端基 OH 成醚）和**非还原糖**（如蔗糖）**不发生变旋**

### 3.5 Fischer 式 → Haworth 式的转换规则

**规则（D-糖）**：
1. Fischer 式中**右侧**的基团 → Haworth 式中在环平面**下方**
2. Fischer 式中**左侧**的基团 → Haworth 式中在环平面**上方**
3. **D-糖**：CH₂OH 在环平面上方
4. **L-糖**：CH₂OH 在环平面下方

**记忆口诀**："**右下左上**"（D-糖）

## 四、关键结论

1. **D/L 标记只看最远手性碳**：与 R/S 体系独立，不要混淆
2. **变旋的本质是 α/β 端基异构体的互变平衡**：通过微量开链形式实现
3. **D-吡喃葡萄糖的 β 型比 α 型更稳定**：β 型的端基 —OH 处于平伏键
4. **不发生变旋 = 端基 OH 被保护**：糖苷、非还原二糖（蔗糖）无变旋
5. **Haworth 式是半平面投影**：上下方代表环平面的两侧，不代表前后（与 Fischer 不同）

## 五、常见分类

### 5.1 常见单糖的环式偏好

| 单糖 | 主要环式 | 说明 |
|:---|:---|:---|
| D-葡萄糖 | 吡喃糖（>99%）| 六元环最稳定 |
| D-果糖 | 吡喃糖/呋喃糖 | 两者共存 |
| D-核糖 | 呋喃糖 | 五碳糖倾向于五元环 |
| D-半乳糖 | 吡喃糖 | 与葡萄糖类似 |

### 5.2 单糖的分类（按羰基位置）

| 类型 | 定义 | 示例 |
|:---|:---|:---|
| **醛糖（Aldose）** | 含醛基 | 葡萄糖、半乳糖、核糖 |
| **酮糖（Ketose）** | 含酮基（通常在 C2）| 果糖、山梨糖 |

### 5.3 椅式构象中的取向（D-吡喃葡萄糖）

| 位置 | α-D-葡萄糖 | β-D-葡萄糖 |
|:---|:---|:---|
| C1（端基）| —OH **直立键（a）**| —OH **平伏键（e）**|
| C2 | —OH 平伏键（e）| —OH 平伏键（e）|
| C3 | —OH 平伏键（e）| —OH 平伏键（e）|
| C4 | —OH 平伏键（e）| —OH 平伏键（e）|
| C5 | CH₂OH 平伏键（e）| CH₂OH 平伏键（e）|

**结论**：β-D-葡萄糖的所有大基团都在平伏键，是最稳定的吡喃糖构象。

## 六、适用条件

### 6.1 α/β 判定的不同表示法

| 表示法 | 判定方法 |
|:---|:---|
| **Fischer 投影式** | 端基 —OH 与 C5 的 CH₂OH：同侧 → β；异侧 → α |
| **Haworth 式（D-吡喃糖）**| 端基 —OH 在环上方 → β；在环下方 → α |
| **Haworth 式（L-吡喃糖）**| 端基 —OH 在环上方 → α；在环下方 → β |
| **椅式构象** | 端基 —OH 在平伏键 → β；在直立键 → α |

### 6.2 变旋的定量计算

若已知纯 α 和纯 β 的比旋光度，可计算平衡组成：

$$[\alpha]_{\text{平衡}} = x_\alpha [\alpha]_\alpha + x_\beta [\alpha]_\beta$$

其中 $x_\alpha + x_\beta = 1$（忽略微量开链和呋喃糖形式）。

**D-葡萄糖**：
- $[\alpha]_\alpha = +112°$
- $[\alpha]_\beta = +18.7°$
- $[\alpha]_{\text{平衡}} = +52.7°$
- 解得：$x_\alpha ≈ 36\%$, $x_\beta ≈ 64\%$

## 七、常见比较

### 7.1 α vs β 端基异构体

| 维度 | α-D-吡喃葡萄糖 | β-D-吡喃葡萄糖 |
|:---|:---|:---|
| **端基 —OH 位置** | 直立键（a）| 平伏键（e）|
| **[α]D** | +112° | +18.7° |
| **稳定性** | 较低 | 较高 |
| **溶解度** | 较低 | 较高 |
| **结晶形态** | α-型结晶 | β-型结晶 |

### 7.2 吡喃糖 vs 呋喃糖

| 维度 | 吡喃糖 | 呋喃糖 |
|:---|:---|:---|
| **环大小** | 6 元环（含 O）| 5 元环（含 O）|
| **稳定性** | 通常更稳定 | 较不稳定 |
| **存在形式** | 葡萄糖、半乳糖的主要形式 | 果糖、核糖的主要形式 |
| **环张力** | 小 | 稍大 |

### 7.3 D-糖 vs L-糖

| 维度 | D-糖 | L-糖 |
|:---|:---|:---|
| **Fischer 最远 —OH** | 右侧 | 左侧 |
| **Haworth CH₂OH** | 环上方 | 环下方 |
| **天然分布** | 广泛（葡萄糖、果糖等）| 稀少（L-阿拉伯糖、L-鼠李糖等）|
| **酶识别** | 生物系统主要识别 D-糖 | 少数特例 |

## 十、例题

### 例题 1：D/L 构型判断

**题目**：判断下列 Fischer 投影式中的糖是 D 型还是 L 型。

```
    CHO                 CHO
     |                   |
    H-C-OH            HO-C-H
     |                   |
    HO-C-H            H-C-OH
     |                   |
    H-C-OH            H-C-OH
     |                   |
    CH₂OH             CH₂OH
```

**解析**：

1. **左图**：离 CHO 最远的手性碳（C5）上 —OH 在**右侧** → **D-型**
2. **右图**：C5 上 —OH 在**左侧** → **L-型**

（左图为 D-葡萄糖，右图为 L-葡萄糖）

### 例题 2：Fischer → Haworth 转换

**题目**：画出 D-葡萄糖的 α-吡喃糖的 Haworth 式。

**解析**：

1. D-葡萄糖 Fischer 式：
   - C2 —OH：右
   - C3 —OH：左
   - C4 —OH：右
   - C5 —OH：右（D-型）

2. 转换规则（右下左上）：
   - C2 —OH（右）→ Haworth **下方**
   - C3 —OH（左）→ Haworth **上方**
   - C4 —OH（右）→ Haworth **下方**
   - C5 CH₂OH（右，D-型）→ Haworth **上方**

3. α 型：端基 —OH 在环平面**下方**

```
        CH₂OH
          |
      O — C — OH
     /       |
   HO        H
    |        |
    H        OH
     \      /
      — C —
          |
         H
```

或标准 Haworth 式：
```
      O
     / \
  HO     CH₂OH
   |      |
   H      OH
    \    /
     C——C
    /    \
   H      H
```

### 例题 3：变旋计算

**题目**：已知纯 α-D-葡萄糖 [α]D = +112°，纯 β-D-葡萄糖 [α]D = +18.7°。某葡萄糖溶液达到平衡时 [α]D = +52.7°。求平衡时 α 和 β 的比例。

**解析**：

设 α 的比例为 $x$，则 β 为 $1-x$。

$$112x + 18.7(1-x) = 52.7$$
$$112x + 18.7 - 18.7x = 52.7$$
$$93.3x = 34$$
$$x ≈ 0.364$$

**结果**：
- **α-D-葡萄糖 ≈ 36.4%**
- **β-D-葡萄糖 ≈ 63.6%**

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| "D/L 与 R/S 有对应关系" | 混淆两套标记系统 | D/L 和 R/S **完全独立**，不要试图对应 | D-葡萄糖的 C2 是 R 还是 S？（需要计算，不是固定的）|
| "α/β 判定与 D/L 无关" | 忽略关联 | α/β 判定需结合 D/L：对 D-糖，端基 —OH 在 Haworth 上方为 β | L-糖的 α/β 判定与 D-糖相同还是相反？ |
| "Haworth 上下 = Fischer 前后" | 误解投影规则 | Haworth 是**半平面投影**，上下不代表前后；Fischer 是前后投影 | Fischer 式右侧的基团在 Haworth 式中在哪边？ |
| "所有糖都变旋" | 忽略糖苷 | 只有含**游离端基 OH** 的糖才变旋；糖苷和非还原糖不变旋 | 蔗糖有变旋现象吗？为什么？ |
| "α 型总是比 β 型稳定" | 不了解构象分析 | 对 D-吡喃葡萄糖，**β 型更稳定**（端基 —OH 在平伏键）| α 和 β 哪个更稳定？为什么？ |

### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "开链糖是主要存在形式" | 不知道环式更稳定 | 强调：水溶液中 > 99% 的葡萄糖以**环式**存在，开链仅 ~0.02% |
| "α/β 是构型标记，与 R/S 同级" | 不了解层级关系 | 解释：D/L 描述**整个分子**的构型族；α/β 只描述**端基碳**的相对构型 |

### 12.3 入门例题

**最简判断练习**：
> 1. D-葡萄糖 Fischer 式中，C5 的 —OH 在哪一侧？
> 2. D-吡喃葡萄糖的 Haworth 式中，CH₂OH 在环平面上方还是下方？
> 3. 为什么 β-D-吡喃葡萄糖比 α-型更稳定？

**答案**：
1. **右侧**（D-型定义）
2. **上方**（D-糖规则）
3. β 型中端基 —OH 处于**平伏键（e 键）**，空间位阻更小

**引导**：问"为什么葡萄糖溶液放置后旋光度会变化？"（α 和 β 通过开链形式互变，最终达到平衡）
