# Eschenmoser 碎片化（Eschenmoser Fragmentation）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-有机化学]]
- 对应考纲：决赛级碎片化反应（超基础要求，属决赛拓展）

---

## 一、定义

**Eschenmoser 碎片化**：α,β-不饱和酮的**环氧**与**对甲苯磺酰肼（TsNHNH₂）**反应，发生**协同碎片化**，生成**炔烃**和**酮**（在不同位置上）的反应。

**核心特征**：
- 产物含有一根**炔键**——这是碎片化反应中非常罕见的结果
- 是 **1,2,3,4-碎片化模式**的经典实例
- 广泛用于**大环酮合成**和**酮炔构建**

**通式**：
$$
\ce{环氧-α,β-不饱和酮 + TsNHNH2 ->[碎片化] 炔烃 + 酮}
$$

## 三、核心原理

### 3.1 反应机理

```
步骤 1：腙的形成
- 环氧酮 + TsNHNH₂ → 对甲苯磺酰腙

步骤 2：环氧开环
- 腙不稳定 → 环氧通过消除开环 → 产生羟基

步骤 3：协同碎片化（1,2,3,4 模式）
- 羟基"推"（电子源）
- 对甲苯亚磺酸根（ArSO₂⁻）+ N₂ "拉"（电子槽）
- 2-3 位 C–C 键断裂
- 所有断裂的键彼此平行，在两根双键两侧反叠

步骤 4：产物
- 炔烃 + 酮
```

**注意**：离去基是**对甲苯亚磺酸根（ArSO₂⁻）**而非对甲苯磺酸根——这是一个重要的细节。

### 3.2 1,2,3,4 碎片化模式

| 位置 | 角色 | 功能 |
|:---|:---|:---|
| **1** | 羟基（OH）| **电子源（推）**——孤对电子推动断裂 |
| **2–3** | C–C 键 | **断裂键**——碎片化发生的核心 |
| **4** | 离去基（ArSO₂⁻/N₂）| **电子槽（拉）**——接受电子对 |

**立体化学要求**：
- 所有断裂的键必须**彼此平行**
- 在两根双键的两侧呈**反叠（anti-periplanar）**排列
- 这是碎片化的严格几何约束

### 3.3 为什么产物是炔烃？

碎片化同时断裂了两根 C–C 键并形成两根新的 π 键：
- 一侧形成 **C≡C 三键**（炔烃）
- 另一侧形成 **C=O 双键**（酮）

**驱动力**：
1. N₂ 离去（极好离去基）
2. 亚磺酸根离去
3. 炔键的形成（强键）
4. 酮的形成（稳定羰基）

---

## 四、关键结论

1. **Eschenmoser = 环氧酮 + TsNHNH₂ → 炔 + 酮**
2. **1,2,3,4 碎片化模式**：OH 推 + 离去基拉 + C–C 键断
3. **产物含炔键**——碎片化反应中非常罕见的特征
4. **立体化学严格**：所有断裂键需平行且反叠
5. **离去基是亚磺酸根（ArSO₂⁻）**而非磺酸根
6. **核心应用**：大环酮合成、酮炔构建

---

## 五、常见分类或情形

### 5.1 按应用分类

| 应用 | 策略 | 后续转化 |
|:---|:---|:---|
| **大环酮合成** | 碎片化得酮炔 → 选择性氢化炔键 | 大环酮（如麝香酮）|
| **酮炔合成** | 碎片化后直接分离 | 功能化的炔酮 |
| **天然产物全合成** | 两次串联碎片化 | 控制多取代双键立体化学 |

### 5.2 与其他碎片化的比较

| 反应 | 底物 | 产物 | 核心特征 |
|:---|:---|:---|:---|
| **Eschenmoser** | 环氧-α,β-不饱和酮 + TsNHNH₂ | 炔 + 酮 | 产物含炔键，协同碎片化 |
| **Grob 碎片化** | 1,2,3,4-体系（含 OH/OTs）| 烯 + 羰基/离去基 | 离子型或协同，反叠要求 |
| **常规碎片化** | 环状二醇 + OTs | 开环烯 | 反叠控制，E2 型 |

## 六、适用条件与限制

1. **必须有环氧-α,β-不饱和酮骨架**——底物限制较严格
2. **需对甲苯磺酰肼**——与羰基形成腙是启动步骤
3. **严格立体化学要求**——碎片化要求反叠几何
4. **产物炔酮可能敏感**——需控制后续反应条件
5. **大环合成中的优势环大小**——通常用于 10–16 元环

---

## 七、常见比较与易混点

### 7.1 Eschenmoser vs Grob 碎片化

| 维度 | Eschenmoser | Grob |
|:---|:---|:---|
| **底物** | 环氧-α,β-不饱和酮 + TsNHNH₂ | 1,2,3,4-体系（OH + 离去基）|
| **产物** | **炔 + 酮** | 烯 + 羰基 |
| **机理** | 协同碎片化 | 协同或离子型 |
| **离去基** | ArSO₂⁻ + N₂ | OTs, N₂ 等 |
| **竞赛频率** | 低（决赛级）| 中（进阶）|

### 7.2 Eschenmoser vs Semipinacol

- **Eschenmoser**：1,2,3,4 碎片化 → C–C 键断裂 → 分子切成碎片
- **Semipinacol**：1,2-迁移 → 骨架重组 → 分子保持完整
- **关键区别**：电子源和电子槽**相间**（Eschenmoser，碎片化）vs **相邻**（Semipinacol，重排）

## 十、例题

### 10.1 例题 1（★★★ 机理）

**题目**：画出 Eschenmoser 碎片化的完整机理，标出电子源、断裂键和电子槽。

**解答**：
1. 环氧酮 + TsNHNH₂ → 磺酰腙
2. 环氧开环 → 羟基
3. **1,2,3,4 碎片化**：
   - 1 = OH（推）
   - 2–3 = C–C 键（断）
   - 4 = ArSO₂⁻ + N₂（拉）
4. 产物：炔 + 酮

### 10.2 例题 2（★★★★ 合成设计）

**题目**：麝香酮（muscone，十五元环酮）的合成中有一步将环氧酮转化为酮炔，指出该反应。

**解答**：
- **反应**：Eschenmoser 碎片化
- **底物**：α,β-不饱和大环酮的环氧 + TsNHNH₂
- **产物**：十五元环酮炔
- **后续**：选择性氢化炔键 → 麝香酮

### 10.3 例题 3（★★★★ 立体化学）

**题目**：Eschenmoser 碎片化中，为什么所有断裂的键必须彼此平行？

**解答**：
- 碎片化是**协同反应**
- 所有电子流动需在同一过渡态中完成
- 反叠几何确保断裂的 σ 键电子能流入新形成的 π 键
- 若几何不匹配，碎片化不能发生

## 十一、易错点

1. ❌ **"产物是烯不是炔"** → Eschenmoser 的独特之处在于产物含**炔键**
2. ❌ **"离去基是磺酸根"** → 离去基是**亚磺酸根（ArSO₂⁻）**
3. ❌ **"所有环氧酮都能碎片化"** → 需要**α,β-不饱和**且**反叠几何匹配**
4. ❌ **"Eschenmoser 是重排"** → 它是**碎片化**，不是重排（C–C 键断裂）
5. ❌ **"碎片化没有立体要求"** → 反叠几何是**严格必需的**

### 12.3 入门例题

**题目**：Eschenmoser 碎片化与 Semipinacol 重排的本质区别是什么？

**引导**：
- 两者底物都含环氧和离去基
- **Semipinacol**：电子源和电子槽**相邻**→ 1,2-迁移 → 骨架重组（重排）
- **Eschenmoser**：电子源和电子槽**相间（1,2,3,4）**→ C–C 键断裂 → 分子切成碎片
- **结论**：距离决定命运——相邻则迁，相间则断

### 12.4 现实类比

**"拉链 vs 剪刀"**：
- **Semipinacol（重排）** = 拉链——相邻的两边咬合在一起，重新组合但不断开
- **Eschenmoser（碎片化）** = 剪刀——相间的两端被拉开，中间完全断开
- **Eschenmoser 的独特之处**：剪断后，两端的断口自动焊接成**钢丝（炔键）**和**弹簧（羰基）**
- **大环合成** = 用这把剪刀剪断小环，再把两端拉直，焊成大环

## 十三、竞赛拓展

### 13.1 天然产物全合成中的应用

| 天然产物 | 合成中的 Eschenmoser 步骤 | 特点 |
|:---|:---|:---|
| **麝香酮（muscone）**| 大环酮炔 → 氢化 → 十五元环酮 | 经典大环合成 |
| **环十五酮（exaltone）**| 类似麝香酮路线 | 香料工业 |
| **外-西部松小蠹集合素（exo-brevicomin）**| 碎片化构建关键中间体 | 昆虫信息素 |
| **保幼激素（juvenile hormone）**| **两次串联 Eschenmoser** | 控制三根三取代双键的立体化学——极其精妙的应用 |

### 13.2 两次串联碎片化的精妙之处

保幼激素合成中，两次 Eschenmoser 碎片化：
1. 第一次碎片化控制第一根双键的立体化学
2. 第二次碎片化控制第二根和第三根双键
3. 每次碎片化的反叠几何精确锁定双键的 E/Z 构型

这是碎片化反应在立体化学控制上的**巅峰应用**。
