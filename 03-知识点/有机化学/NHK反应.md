# NHK 反应（Nozaki-Hiyama-Kishi Reaction）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[50-有机合成]]、[[51-金属有机]]

---

## 一、定义

**NHK 反应（Nozaki-Hiyama-Kishi Reaction）**：在 **CrCl₂ 和 NiCl₂（或 Ni(acac)₂）**共同催化下，**烯基/芳基卤代物**与**醛**发生交叉偶联，生成**烯丙型/苄型醇**的反应。

**通式**：
$$
\ce{R-X + R'-CHO ->[CrCl2/NiCl2] R-CH(OH)-R'}
$$

**本质**：**Cr/Ni 双金属催化的醛交叉偶联**——对醛基具有独特的化学选择性，是复杂分子合成中的关键 C–C 键形成方法。

## 三、核心原理

### 3.1 催化循环

**Ni/Cr 协同机理**：

```
步骤 1：Ni(II) 被 Cr(II) 还原为 Ni(0)
  NiCl₂ + CrCl₂ → Ni(0) + Cr(III)

步骤 2：Ni(0) 与卤代物氧化加成
  Ni(0) + R-X → R-Ni(II)-X

步骤 3：转金属到 Cr(III)
  R-Ni(II)-X + Cr(III) → R-Cr(III) + Ni(II)

步骤 4：R-Cr(III) 进攻醛
  R-Cr(III) + R'-CHO → R-CH(OH)-R' + Cr(III)
```

**关键点**：
- **Ni 是"活化者"**——负责氧化加成（Cr 单独做不到）
- **Cr 是"执行者"**——生成的有机铬试剂进攻醛
- **Cr(II) 同时是还原剂**——还原 Ni(II) 为 Ni(0)

### 3.2 化学选择性

**NHK 的独特优势**：

| 官能团 | NHK 选择性 | 原因 |
|:---|:---:|:---|
| **醛（-CHO）** | ✅ 反应 | 有机铬亲核进攻醛 |
| **酮（C=O）** | ❌ 不反应 | 铬试剂对醛选择性高 |
| **酯（-COOR）** | ❌ 不反应 | 不反应 |
| **酰胺（-CONR₂）** | ❌ 不反应 | 不反应 |

**为什么对醛选择性高？**
- 有机铬试剂是**软亲核试剂**
- 醛比酮更亲电（位阻小 + 缺电子程度更高）
- Cr 与醛的氧配位 → 定向进攻

### 3.3 底物范围

**卤代物**：
- 烯基碘/溴（最常用）
- 芳基碘/溴
- 烯丙基卤

**醛**：
- 脂肪醛
- 芳香醛
- α,β-不饱和醛（1,2-加成）

---

## 四、关键结论

1. **NHK = 卤代物 + 醛 → 醇**
2. **CrCl₂/NiCl₂ 双金属协同**
3. **Ni 活化卤代物，Cr 执行亲核进攻**
4. **对醛有独特化学选择性**
5. **不还原酮、酯、酰胺**
6. **是全合成中构建烯丙醇/苄醇的利器**

---

## 五、常见分类或情形

### 5.1 按催化剂比例分类

| Ni/Cr 比例 | 效果 |
|:---|:---|
| **少量 Ni（1–5 mol%）** | 标准条件，Ni 作为共催化剂 |
| **无 Ni** | 反应极慢或根本不发生 |
| **过量 Ni** | 可能降低选择性 |

---

## 六、适用条件与限制

1. **需要大量 CrCl₂**（化学计量或过量）
2. **严格无水**（Cr(II) 对水敏感）
3. **溶剂**：DMF、THF
4. **温度**：室温至 60°C
5. **对酮不反应**——既是优点也是限制
6. **Cr(II) 需现制或商业无水试剂**

---

## 七、常见比较与易混点

### 7.1 NHK vs 格氏反应

| | **NHK** | **格氏反应** |
|:---|:---|:---|
| **试剂** | 有机铬（原位生成）| 有机镁 |
| **选择性** | **只对醛反应** | 对醛/酮/酯都反应 |
| **官能团耐受** | 好（酯、酮保留）| 差（需保护基）|
| **操作** | 较复杂（双金属）| 简单 |

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：NHK 反应中 Ni 和 Cr 各自的作用是什么？

**解答**：
- **Ni**：与卤代物发生**氧化加成**，生成有机镍中间体
- **Cr**：接收有机基团形成**有机铬试剂**，并**选择性进攻醛**
- **Cr(II)**：同时作为**还原剂**，将 Ni(II) 还原为 Ni(0)

### 10.2 例题 2（★★ 选择性）

**题目**：一个同时含有醛基和酮基的分子与烯基碘发生 NHK 反应，产物是什么？

**解答**：
- NHK 对**醛有选择性**，酮不反应
- 产物：**烯基连接到醛基碳上，形成烯丙醇，酮基保留**
- 这是 NHK 在复杂分子合成中的核心价值

---

## 十一、易错点

1. ❌ **NHK 是单金属催化** → NHK 是 **Cr/Ni 双金属协同**
2. ❌ **NHK 对酮也反应** → NHK **只对醛反应**
3. ❌ **NHK 不需要无水** → Cr(II) **对水敏感**
