# Grignard 试剂

## 一、定义

**Grignard 试剂（格氏试剂）** 通式为 **RMgX**（R = 烷基、烯基、芳基；X = Cl、Br、I），是由卤代烃与金属镁在无水乙醚（Et₂O）或四氢呋喃（THF）中反应制备的有机金属化合物。

$$
\ce{R-X + Mg ->[Et2O 或 THF] R-Mg-X}
$$

- **C—Mg 键高度极化**（Cδ⁻—Mgδ⁺），碳原子带部分负电荷，具有 **强亲核性** 和 **强碱性**
- **本质**：格氏试剂中的 R 相当于 **"碳负离子等价物"**（carbanion equivalent）
- **制备条件**：必须严格 **无水、无氧**（否则 RMgX 被质子化或氧化分解）
- **溶剂作用**：乙醚或 THF 的孤对电子与 Mg 配位，稳定试剂并促进反应

## 三、核心原理

### 3.1 制备机理

卤代烃与镁的反应在金属表面进行：

1. **电子转移**：Mg 向 R-X 的 σ* 反键轨道注入电子，形成 **自由基阴离子** [R-X]⁻·
2. **C-X 键断裂**：生成 R· 和 X⁻
3. **自由基与金属结合**：R· 和 X· 吸附在 Mg 表面，形成 RMgX

**注意**：
- **R-I > R-Br > R-Cl** 反应活性递减（I 的 C-X 键最弱）
- **烯基/芳基卤代烃**也可制备格氏试剂，但需 THF（比 Et₂O 配位能力强）
- **乙烯基卤**保持构型（反应在 C-X σ* 进行，不涉及 π 键）

### 3.2 反应机理通式

Grignard 试剂与羰基化合物的反应经由 **六元环过渡态**（配位-加成机理）：

```
       R                    R
       |                    |
   δ⁻C-Mgδ⁺-X           C-O-MgX
       |    \    →        |
       |     C=O          C-R'
       |    /             |
      R'                 OMgX
```

1. **配位**：羰基氧的孤对电子与 Mg 配位（Lewis 酸碱作用）
2. **加成**：R⁻（亲核）进攻羰基碳，C=O π 键电子推向氧
3. **质子化**：水解（H₃O⁺）或酸性后处理，得醇

## 四、关键结论

1. **Grignard 试剂 = 碳负离子等价物**：R⁻ 进攻亲电中心，与 C=O、CO₂、环氧等反应
2. **碱性足以夺取酸性 H**：任何含 O-H、N-H、S-H、C≡C-H 的化合物都会使格氏试剂分解
3. **与酯/酰氯反应需 2 当量**：第一当量加成得四面体中间体， collapse 后生成酮；酮立即与第二当量反应得 3° 醇
4. **与 α,β-不饱和羰基主要发生 1,2-加成**（硬亲核体特性），除非 Cu(I) 催化转 1,4-
5. **空间位阻影响**：大位阻酮（如二异丙基酮）与格氏试剂反应变慢，可能与副反应竞争

## 五、常见分类

### 5.1 按底物分类的主要反应

| 底物 | 产物 | 反应类型 | 当量要求 | 备注 |
|:---|:---|:---|:---:|:---|
| **甲醛（HCHO）** | 1° 醇（RCH₂OH）| 亲核加成 | 1 eq | 增加一个碳 |
| **其他醛（R'CHO）** | 2° 醇（RR'CHOH）| 亲核加成 | 1 eq | — |
| **酮（R'COR''）** | 3° 醇（RR'R''COH）| 亲核加成 | 1 eq | 位阻大时反应慢 |
| **CO₂** | 羧酸（RCOOH）| 亲核加成 | 1 eq | **增加一个碳** |
| **甲酸酯（HCOOR'）** | 2° 醇（RCH₂OH）| 两次加成 | 2 eq | 第一次得醛，第二次得醇 |
| **其他酯（R'COOR''）** | 3° 醇（RR'₂COH）| 两次加成 | 2 eq | 第一次得酮，第二次得醇 |
| **酰氯（R'COCl）** | 3° 醇（RR'₂COH）| 两次加成 | 2 eq | 与酯类似，更快 |
| **环氧乙烷** | 1° 醇（RCH₂CH₂OH）| 亲核开环 | 1 eq | **增加两个碳** |
| **取代环氧** | 醇（位阻小端开环）| 亲核开环 | 1 eq | Sₙ2 机理，进攻位阻小碳 |
| **腈（R'CN）** | 酮（RCOR'，水解后）| 亲核加成 | 1 eq | 亚胺镁盐水解得酮 |

### 5.2 格氏试剂的反应网络

```
Grignard 试剂 RMgX
├── 与羰基化合物
│   ├── 醛 → 1°/2° 醇
│   ├── 酮 → 3° 醇
│   ├── 酯/酰氯 → 3° 醇（2 eq）
│   └── CO₂ → 羧酸
├── 与环氧化物
│   ├── 环氧乙烷 → 增加 2 个碳的 1° 醇
│   └── 取代环氧 → 位阻小端开环
├── 与含活泼氢化合物（分解反应）
│   ├── H₂O → RH + Mg(OH)X
│   ├── ROH → RH + Mg(OR)X
│   ├── 羧酸 → RH + Mg(O₂CR)X
│   └── 端炔 → RH + Mg(C≡CR)X
└── 与金属盐（转金属化）
    ├── CuI → Gilman 试剂（R₂CuLi 的前体）
    └── CdCl₂ → R₂Cd（制备酮）
```

## 六、适用条件

### 6.1 制备条件

| 因素 | 要求 | 原因 |
|:---|:---|:---|
| **溶剂** | 无水 Et₂O 或 THF | 醚类溶剂与 Mg 配位，稳定 RMgX；质子性溶剂会分解试剂 |
| **卤代烃** | R-I > R-Br > R-Cl | C-X 键强度递减；乙烯基/芳基需 THF |
| **镁** | 镁屑或镁粉，表面活化（I₂ 或 1,2-二溴乙烷引发）| 去除表面氧化膜，暴露新鲜金属 |
| **温度** | 室温或微热（Δ）| 放热反应；高温可能导致偶联副反应（Wurtz）|
| **气氛** | N₂ 或 Ar 保护 | 防止 O₂ 氧化和 H₂O 分解 |

### 6.2 反应条件

| 底物 | 温度 | 溶剂 | 后处理 |
|:---|:---|:---|:---|
| **醛/酮** | 0°C → rt | Et₂O 或 THF | 饱和 NH₄Cl 或稀 HCl（质子化烷氧镁盐）|
| **CO₂** | 低温（干冰温度）| Et₂O | 酸性水解 |
| **酯** | rt → 回流 | Et₂O 或 THF | 酸性水解 |
| **环氧乙烷** | rt → 回流 | Et₂O | 酸性水解 |

### 6.3 绝对禁忌（会使格氏试剂分解的官能团）

| 官能团 | 反应 | 结果 |
|:---|:---|:---|
| -OH（醇/酚）| 酸碱反应 | RH + Mg(OR)X（格氏试剂被"淬灭"）|
| -COOH（羧酸）| 酸碱反应 | RH + Mg(O₂CR)X |
| -NH₂/-NHR（胺）| 酸碱反应 | RH + Mg(NHR)X |
| -SH（硫醇）| 酸碱反应 | RH + Mg(SR)X |
| -C≡CH（端炔）| 酸碱反应 | RH + Mg(C≡CR)X |
| -NO₂（硝基）| 氧化还原 | 复杂分解 |
| C=O（醛/酮，当共存于底物时）| 亲核加成 | 分子内/间副反应 |

**核心原则**：合成设计中，若底物含上述官能团，必须**预先保护**或**选择其他试剂**。

## 七、常见比较

### 7.1 Grignard vs 有机锂（RLi）

| 维度 | Grignard（RMgX）| 有机锂（RLi）|
|:---|:---|:---|
| **C-金属键极化** | 强（Cδ⁻—Mgδ⁺）| 更强（Cδ⁻—Liδ⁺）|
| **亲核性** | 强 | **更强** |
| **碱性** | 强 | **更强** |
| **与位阻酮反应** | 较慢，可能还原竞争 | 快，即使大位阻酮也能加成 |
| **与酯反应** | 2 eq → 3° 醇 | 2 eq → 3° 醇（更快）|
| **制备难度** | 较易（Mg 较不活泼）| 较难（Li 更活泼，需低温）|
| **对空气/水敏感度** | 高 | **极高** |
| **卤素-金属交换** | 不易 | 易（t-BuLi + ArX → ArLi + t-BuX）|
| **价格/操作** | 便宜，操作相对简单 | 较贵，需更严格无氧 |

### 7.2 Grignard vs 铜锂试剂（R₂CuLi）

| 维度 | Grignard（RMgX）| Gilman 试剂（R₂CuLi）|
|:---|:---|:---|
| **软硬酸碱** | **硬**亲核体 | **软**亲核体 |
| **与饱和羰基** | 1,2-加成（快）| 不反应或极慢 |
| **与 α,β-不饱和羰基** | 1,2-加成（动力学）| **1,4-加成**（热力学）|
| **与卤代烃** | 不反应（除非烯丙型）| **偶联反应**（Corey-House）|
| **与环氧** | 开环（SN2）| 开环（较慢）|
| **碱性** | 强 | 较弱 |
| **制备** | R-X + Mg | 2 RLi + CuI |

**选择策略**：
- 要 **1,2-加成** 到醛/酮 → **Grignard 或 RLi**
- 要 **1,4-加成**（Michael 加成）→ **R₂CuLi**
- 要 **偶联卤代烃** → **R₂CuLi**

## 十、例题

### 例题 1：Grignard 试剂与酯的反应

**题目**：写出 CH₃MgBr（2 eq）与 CH₃COOEt 反应的完整过程及最终产物。

**解析**：
1. **第一当量**：CH₃MgBr + CH₃COOEt → 四面体中间体 → collapse（ EtO⁻ 离去）→ **丙酮（CH₃COCH₃）**
2. **第二当量**：CH₃MgBr + CH₃COCH₃ → **叔丁氧基溴化镁（(CH₃)₃COMgBr）**
3. **后处理**：H₃O⁺ → **叔丁醇（(CH₃)₃COH）**

**陷阱**：若只写一当量，产物是丙酮；但格氏试剂过量时，酮不会停留，继续反应得 3° 醇。

### 例题 2：合成设计中的碳数增加

**题目**：如何以溴乙烷为原料，用格氏试剂法制备：
1. 丙酸
2. 1-丁醇
3. 2-丁醇

**解析**：
1. **丙酸**：CH₃CH₂Br → CH₃CH₂MgBr + CO₂ → CH₃CH₂COOH（增加一个碳）
2. **1-丁醇**：CH₃CH₂MgBr + 环氧乙烷 → CH₃CH₂CH₂CH₂OH（增加两个碳）
3. **2-丁醇**：CH₃CH₂MgBr + CH₃CHO → CH₃CH₂CH(OH)CH₃

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| 忽略酯需要 2 当量格氏试剂 | 以为和醛酮一样 1 当量 | 酯 + 1 eq → 酮；酮 + 1 eq → 3° 醇；总需 **2 eq** | 为什么酯与格氏试剂反应不会停在酮？ |
| 底物含 -OH 时直接用格氏 | 未保护活泼氢 | 任何含活泼氢的底物必须先保护（如 -OH → -OTBS）| 如何用格氏试剂与含羟基的醛反应？ |
| 与 α,β-不饱和酮反应只考虑 1,2-加成 | 忽略 1,4-加成可能 | 格氏试剂主要 1,2-加成；若要 1,4-加成，换 **铜锂试剂** | 如何选择性实现共轭加成而不碰羰基？ |
| 认为格氏试剂与腈反应得胺 | 混淆还原与加成 | RMgX + R'CN → 亚胺盐 → 水解得 **酮**（不是胺）| 格氏试剂与腈反应后水解，产物是酮还是胺？ |
| 环氧开环方向错误 | 忘记 Sₙ2 位阻控制 | 格氏试剂进攻环氧 **位阻较小** 的碳 | 2,2-二甲基环氧乙烷与格氏试剂反应，进攻哪个碳？ |

### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "格氏试剂就是碳负离子" | 忽略 MgX 的作用 | 强调是 **"碳负离子等价物"**，不是自由碳负离子；Mg 的配位作用影响反应性 |
| "所有卤代烃都能制格氏试剂" | 忽略官能团兼容性 | 含 -OH、-COOH、-NH₂、-NO₂ 的卤代烃不能直接制格氏 |
| "格氏试剂只和醛酮反应" | 忽略与 CO₂、环氧、腈的反应 | 用 **"增加碳数的工具"** 框架总结：+1C（CO₂）、+2C（环氧）、构建醇（醛酮）|
| "格氏试剂与酯反应得酮" | 不知第二当量继续反应 | 明确：除非严格控制 1 eq 且低温，否则一般直接得 3° 醇 |

### 12.3 入门例题

**最简产物预测**：
> 写出 CH₃MgBr 与下列底物反应、酸性后处理后的产物：
> 1. 甲醛  2. 丙酮  3. CO₂

**答案**：
1. **乙醇**（CH₃CH₂OH）
2. **叔丁醇**（(CH₃)₃COH）
3. **乙酸**（CH₃COOH）
**引导**：若学生第 3 题答错，问"CO₂ 的结构是什么？有几个羰基碳？格氏试剂的甲基加到哪？"

### 12.4 现实类比

**「快递员送货」类比格氏试剂**：
- 格氏试剂 = 一位强力快递员（R⁻），专门给羰基 "送货"（碳链）
- 醛/酮的羰基 = 收件人地址（缺电子中心）
- Mg = 快递员的电动车（稳定 R⁻，使其能安全到达）
- **无水条件** = 快递不能碰水（遇水快递员辞职：RMgX + H₂O → RH）
- **酯需要 2 个快递员** = 第一个送到后变成 "中转站"（酮），第二个继续送到最终地址（3° 醇）
- **CO₂ = 特殊收件人**：只有一个碳的地址，快递员送过去后增加一个碳（羧酸）

## 十三、竞赛拓展

### 13.1 竞赛中的格氏试剂变体与应用

| 反应/变体 | 特点 | 竞赛价值 |
|:---|:---|:---|
| **格氏试剂 + 原甲酸酯** | 制备醛（比直接 + DMF 更温和）| 醛的合成 |
| **Barbier 反应** | 原位生成格氏试剂并反应（Mg + R-X + 羰基，一锅法）| 对水敏感底物的替代方案 |
| **Nozaki-Hiyama-Kishi 反应（NHK）** | Cr(II) 催化有机卤化物对醛的加成 | 大位阻醛的烯丙基化 |
| **Kumada 偶联** | Ni 或 Pd 催化 RMgX 与 R'-X 偶联 | C-C 键构建（与 Corey-House 互补）|
| **格氏试剂 + Weinerb 酰胺** | 得酮（不继续加成）| 酮的选择性合成 |

### 13.2 与元素化学的交叉

- **硅保护基**：格氏试剂用于引入 TMS（TMSCl + RMgX）或脱保护（TBAF 不可与格氏共存）
- **硼化学**：格氏试剂与 B(OMe)₃ 反应制备有机硼酸（Suzuki 偶联前体）
