# Beckmann 重排（Beckmann Rearrangement）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[39-重排反应]]、[[47-羧酸及羧酸衍生物]]

---

## 一、定义

**Beckmann 重排**：**肟（oxime）**在**强酸**条件下发生**1,2-基团迁移**，转化为 **N-取代酰胺** 的反应。

**通式**：
$$
\ce{R^1R^2C=NOH ->[H2SO4] R^1-CONHR^2 \text{ 或 } R^2-CONHR^1}
$$

**本质**：**反式基团从碳迁移到氮**——迁移基团与肟羟基处于反式位置。

## 三、核心原理

### 3.1 反应机理

```
步骤 1：肟的 OH 被质子化或转化为更好的离去基（如 O-SO₃H）
步骤 2：离去基离去 → 亚胺正离子（nitrilium ion）中间体
步骤 3：反式基团 1,2-迁移到氮上
步骤 4：水进攻 → 亚氨酸 → 互变异构 → 酰胺
```

**核心规则**：**反式迁移**——与离去基（OH）处于**反式**的基团发生迁移。

### 3.2 为什么是反式迁移？

**立体电子效应**：
- 迁移基团的 σ 键需要与离去基的断裂轨道**反式共平面**
- 反式排列时，σ→p* 电子推动最有效
- 顺式基团的 σ 键与离去基轨道不共线，迁移难以进行

### 3.3 肟的构型与产物

**不对称酮的肟有 E/Z 两种构型**：

| 肟构型 | 与 OH 反式的基团 | 迁移基团 | 产物酰胺 |
|:---:|:---|:---|:---|
| **E-肟** | R¹ | R¹ | R¹-CONHR² |
| **Z-肟** | R² | R² | R²-CONHR¹ |

**竞赛考点**：给定肟的构型（或立体结构），判断哪个基团迁移，产物是什么。

### 3.4 工业应用——己内酰胺

**环己酮肟 → Beckmann 重排 → 己内酰胺**
- 己内酰胺是**尼龙-6**的单体
- 这是 Beckmann 重排最重要的工业应用
- 使用发烟硫酸（SO₃/H₂SO₄）或 PCl₅ 作为活化剂

### 3.5 与其他重排的关系

| 重排 | 底物 | 迁移规则 | 产物 |
|:---|:---|:---|:---|
| **Beckmann** | 肟 | 反式迁移 | 酰胺 |
| **Baeyer-Villiger** | 酮 | 取代基多的优先 | 酯 |
| **Neber** | 肟磺酸酯 | 不同机理 | 氨基酮 |
| **Favorskii** | α-卤代酮 | 环丙酮中间体 | 羧酸/酯 |

---

## 四、关键结论

1. **Beckmann = 肟 + 强酸 → N-取代酰胺**
2. **反式迁移是铁律**——顺式基团不迁移
3. **E/Z 肟构型决定产物**——控制肟构型 = 控制产物
4. **己内酰胺工业制备依赖 Beckmann**——尼龙-6单体
5. **离去基活化是关键**——OH 本身不好离去，需质子化或转化为磺酸酯
6. **第三轮看题三连问**：是不是肟、谁与 OH 反式、是否发生扩环

## 五、常见分类或情形

### 5.1 按肟的来源分类

| 肟类型 | 来源 | 产物 | 备注 |
|:---|:---|:---|:---|
| **醛肟** | RCH=NOH | 甲酰胺衍生物 | 较少见 |
| **酮肟（对称）** | RC(R')=NOH, R=R' | 单一酰胺 | 产物确定 |
| **酮肟（不对称）** | R¹C(R²)=NOH | 取决于 E/Z 构型 | 竞赛常考 |
| **环酮肟** | 环己酮肟 | 内酰胺（环扩大）| 工业制己内酰胺 |

### 5.2 活化试剂

| 试剂 | 条件 | 特点 |
|:---|:---|:---|
| **浓 H₂SO₄** | 加热 | 经典条件 |
| **PCl₅ / SOCl₂** | 室温 | 将 OH 转化为更好离去基 |
| **TsCl / 碱** | 室温 | 形成肟磺酸酯 |
| **多聚磷酸（PPA）** | 加热 | 脱水剂兼酸 |

## 六、适用条件与限制

1. **需要强酸或活化试剂**——将 OH 转化为好离去基
2. **肟必须有反式基团**——对称肟无所谓，不对称肟需考虑构型
3. **α-氢过多时可能副反应**——如 Beckmann 断裂（Beckmann fragmentation）
4. **环酮肟 → 内酰胺**——环扩大一个原子
5. **不能迁移的基团**：如叔丁基（位阻太大）

### 6.1 第三轮高频判断口径

碰到 Beckmann 重排时，建议固定按以下顺序判断：

1. **先确认底物是不是肟或肟的活化衍生物**
2. **再找与 OH 反式的那个基团**，它才是迁移基
3. **若是环酮肟**，直接联想到“**扩一个原子得到内酰胺**”

这样可以把构型判断、迁移方向和产物类型一次性串起来。

---

## 七、常见比较与易混点

### 7.1 Beckmann vs Baeyer-Villiger

- Beckmann：肟 → 酰胺，**反式迁移**
- B-V：酮 + 过酸 → 酯，**取代基多的优先迁移**
- 两者都是"谁迁移"的问题，但底物和规则不同

### 7.2 Beckmann vs Neber

- Beckmann：肟 → **酰胺**（碳骨架不变）
- Neber：肟磺酸酯 → **氨基酮**（碳骨架重组）
- 两者底物类似（肟衍生物），产物完全不同

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：二苯甲酮肟（Ph₂C=NOH）经 Beckmann 重排后的产物是什么？

**解答**：
- 二苯甲酮肟对称，两个苯基等价
- Beckmann 重排 → **N-苯基苯甲酰胺**（苯甲酰苯胺）

### 10.2 例题 2（★★ 反式迁移）

**题目**：苯乙酮肟（PhC(Me)=NOH）的 E-构型中，Beckmann 重排产物是什么？

**解答**：
- E-构型中，苯基与 OH 反式
- 反式基团迁移 → **苯基迁移到氮**
- 产物：**N-甲基苯甲酰胺**

### 10.3 例题 3（★★★ 环扩大）

**题目**：环己酮肟经 Beckmann 重排后的产物是什么？

**解答**：
- 环酮肟 Beckmann 重排 → **环扩大一个原子**
- 六元环 → 七元环内酰胺
- 产物：**己内酰胺**（尼龙-6 单体）

---

## 十一、易错点

1. ❌ **顺式基团迁移** → Beckmann 是**反式迁移**
2. ❌ **肟的 E/Z 不影响产物** → E/Z 构型**决定**哪个基团迁移
3. ❌ **Beckmann 得羧酸** → Beckmann 得**酰胺**
4. ❌ **环酮肟环缩小** → 环酮肟 Beckmann → **环扩大**（插入 N）
5. ❌ **所有肟都能 Beckmann** → 需要**反式有可迁移基团**，叔丁基等难迁移

### 12.3 入门例题

**题目**：为什么 Beckmann 重排中必须是反式基团迁移？

**引导**：
- 画出肟的平面结构，标出 OH 和反式基团
- 迁移需要 σ 键电子推向氮
- 反式时 σ 键与 N-O 键反式共平面，电子推动最有效
- 顺式时轨道不正交，迁移困难

### 12.4 现实类比

**Beckmann 重排像"换座位游戏"**：
- 肟的碳和氮 = 两个相邻座位
- OH = 坐在碳旁边的人（要起身离开）
- 反式基团 = 坐在碳对面的人
- OH 起身后，对面的人迅速坐到氮的旁边（迁移）
- 新来的人（水）坐到空位 → 最终安定成酰胺

## 十三、竞赛拓展

### 13.1 Beckmann 碎片化（Clayden Ch.36 深化）

**核心概念**：当迁移基团能**太好地稳定阳离子**时（如叔丁基），Beckmann 重排不发生迁移，而发生 **C-C 键断裂**——生成**腈 + 碳正离子**。

**机理**：
1. 肟 OH 转化为离去基 → 水离去。
2. 叔碳基团无法以迁移方式稳定过渡态 → 直接断裂 C-C 键 → 叔碳阳离子 + 腈。
3. 碳正离子被腈捕获 → Ritter 反应 → 酰胺（交叉产物）。

**实验证据**：
- 混合实验：两个不同的叔丁基肟化合物在酸中混合反应 → 得到交叉产物（证明分子间过程）。
- 樟脑衍生物：环结构使阳离子不能被腈捕获 → 可分离出碎片化产物（腈 + 含羰基的环状化合物）。

**命名注意**：
- 避免使用"反常 Beckmann"或"二级 Beckmann"——"**Beckmann 碎片化**"是更好的名称。

**竞赛判断**：
- 迁移基团是叔碳/苄基等能稳定阳离子的基团时，需考虑碎片化竞争。
- 碎片化产物 = 腈 + 碳正离子衍生物（可能被溶剂捕获）。

### 13.2 光化学 Beckmann
- 某些肟在光照下发生不同重排路径
- 生成不同区域异构体
