# Cannizzaro 反应（Cannizzaro Reaction）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[40-氧化还原反应]]

---

## 一、定义

**Cannizzaro 反应**：无 α-氢的醛在**浓碱**作用下发生**分子间氧化还原（歧化）**：一分子醛被**氧化**为羧酸盐，另一分子被**还原**为醇。

**通式**：
$$
\ce{2 RCHO + OH^- -> RCOO^- + RCH2OH}
$$

**本质**：是**负氢迁移（hydride shift）**的经典案例——一分子醛将自身的氢负离子（H⁻）转移给另一分子醛。

## 三、核心原理

### 3.1 反应机理

**三步机理**：
```
步骤 1：OH⁻ 进攻醛羰基 → 形成四面体中间体（烷氧负离子）
步骤 2：负氢迁移（H⁻ 从烷氧负离子迁移到另一分子醛的羰基碳）
        → 一分子成为羧酸根，另一分子成为醇负离子
步骤 3：质子交换 → 羧酸盐 + 醇
```

**负氢迁移的本质**：
- 不是自由基过程，也不是单电子转移
- 是**协同的两电子转移**：C-H 键的 σ 电子对直接流入另一分子羰基的 π* 轨道
- 过渡态近似为 **六元环结构**（两个醛 + 一个 OH⁻）

### 3.2 底物要求

| 底物 | 能否反应 | 原因 |
|:---|:---:|:---|
| **甲醛（HCHO）** | ✅ | 无 α-H |
| **芳香醛（ArCHO）** | ✅ | 无 α-H |
| **三甲基乙醛（t-BuCHO）** | ✅ | 无 α-H |
| **乙醛（CH₃CHO）** | ❌ | 有 α-H，优先羟醛反应 |
| **丙醛（CH₃CH₂CHO）** | ❌ | 有 α-H |

> ⭐ **核心判据**：有 α-H 的醛在碱中优先发生**羟醛缩合**（aldol），无 α-H 的醛才走 Cannizzaro。

### 3.3 交叉 Cannizzaro

**甲醛 + 其他无 α-H 的醛**：
$$
\ce{HCHO + ArCHO + OH^- -> HCOO^- + ArCH2OH}
$$

- **甲醛总是"被氧化"的一方**（生成甲酸根）
- **另一醛被还原为醇**
- 原因：甲醛的羰基更亲电，更容易接受负氢

### 3.4 与 Tishchenko 反应的关系

| | **Cannizzaro** | **Tishchenko** |
|:---|:---|:---|
| **试剂** | NaOH（强碱）| Al(OEt)₃（Lewis 酸）|
| **产物** | 醇 + 羧酸盐 | 酯 |
| **本质** | 歧化 | 歧化 |
| **负氢来源** | 醛自身 | 醛自身 |
| **结果差异** | 碱催化 → 醇 + 酸盐 | 铝催化 → 酯 |

---

## 四、关键结论

1. **Cannizzaro = 无 α-H 醛的歧化**
2. **负氢迁移是核心机理**——学会 Cannizzaro = 学会 MPV = 学会 Tishchenko
3. **甲醛在交叉 Cannizzaro 中总是被氧化**
4. **有 α-H 的醛不走 Cannizzaro**——羟醛反应优先
5. **浓碱是必要条件**——稀碱下反应极慢
6. **过渡态是六元环**——两个醛分子 + OH⁻ 协同作用
7. **第三轮判定顺序**：先看 α-H，再看是否有甲醛参与交叉体系

## 五、常见分类或情形

### 5.1 按醛类型分类

| 类型 | 例子 | 产物 |
|:---|:---|:---|
| **脂肪醛（无 α-H）** | 甲醛、新戊醛 | 甲酸盐 / 新戊醇 |
| **芳香醛** | 苯甲醛、糠醛 | 苯甲酸盐 / 苯甲醇 |
| **杂环醛** | 吡啶甲醛 | 吡啶甲酸盐 / 吡啶甲醇 |

### 5.2 交叉 Cannizzaro 组合

| 组合 | 氧化产物 | 还原产物 |
|:---|:---|:---|
| HCHO + ArCHO | HCOO⁻ | ArCH₂OH |
| HCHO + R₃CCHO | HCOO⁻ | R₃CCH₂OH |
| ArCHO + Ar'CHO | 混合酸盐 | 混合醇 |

---

## 六、适用条件与限制

1. **必须无 α-H**——有 α-H 则羟醛反应竞争
2. **需要浓碱**（通常 50% NaOH）——稀碱反应太慢
3. **高温加速**——但副反应也增加
4. **芳香醛比脂肪醛更易反应**——芳环稳定中间体
5. **甲醛反应性最高**——在交叉反应中总是"牺牲者"

### 6.1 第三轮高频判断口径

第三轮题目里，Cannizzaro 最实用的不是背机理细节，而是形成一个固定判定顺序：

1. **先看底物有没有 α-H**
2. **若无 α-H，再看是不是强碱条件**
3. **若有甲醛参与交叉体系，默认甲醛被氧化、另一组分被还原**

这样能最快把 Cannizzaro、交叉 Cannizzaro 和普通羟醛路线分开。

## 七、常见比较与易混点

### 7.1 Cannizzaro vs 羟醛反应

| | **Cannizzaro** | **羟醛反应** |
|:---|:---|:---|
| **底物** | 无 α-H 醛 | 有 α-H 醛 |
| **条件** | 浓碱 | 稀碱/弱酸 |
| **产物** | 醇 + 羧酸盐 | β-羟基醛/酮 |
| **机理** | 负氢迁移 | 烯醇负离子进攻 |

### 7.2 Cannizzaro vs MPV 还原

| | **Cannizzaro** | **MPV 还原** |
|:---|:---|:---|
| **氢来源** | 另一分子醛 | 异丙醇 |
| **催化剂** | 浓碱 | Al(OiPr)₃ |
| **产物** | 醇 + 酸盐 | 醇 + 丙酮 |
| **本质** | 歧化 | 负氢转移 |

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：苯甲醛在浓 NaOH 中加热，产物是什么？

**解答**：
- 苯甲醛无 α-H → **Cannizzaro 反应**
- 产物：**苯甲酸钠 + 苯甲醇**
- 2 PhCHO + NaOH → PhCOONa + PhCH₂OH

### 10.2 例题 2（★★ 交叉）

**题目**：甲醛 + 苯甲醛在浓 NaOH 中反应，产物是什么？

**解答**：
- **交叉 Cannizzaro**
- 甲醛被氧化 → **甲酸钠**
- 苯甲醛被还原 → **苯甲醇**
- HCHO + PhCHO + NaOH → HCOONa + PhCH₂OH

### 10.3 例题 3（★★★ 竞争）

**题目**：下列醛中，哪些能发生 Cannizzaro 反应？
A. 乙醛  B. 苯甲醛  C. 丙醛  D. 三甲基乙醛

**解答**：
- **无 α-H** 才能 Cannizzaro
- B（苯甲醛）✓、D（三甲基乙醛，(CH₃)₃CCHO）✓
- A（CH₃CHO，有 α-H）✗、C（CH₃CH₂CHO，有 α-H）✗

## 十一、易错点

1. ❌ **有 α-H 的醛也能 Cannizzaro** → 有 α-H 优先**羟醛反应**
2. ❌ **Cannizzaro 需要氧化剂/还原剂** → 是**歧化**（自身氧化还原），无需外加试剂
3. ❌ **交叉 Cannizzaro 中甲醛被还原** → 甲醛**被氧化**（生成甲酸根）
4. ❌ **稀碱也能做** → 需要**浓碱**
5. ❌ **负氢迁移是自由基过程** → 是**协同的两电子转移**，不是自由基

### 12.3 入门例题

**题目**：为什么乙醛不能发生 Cannizzaro 反应？

**引导**：
- 乙醛有 3 个 α-H
- 在碱中，α-H 被拔去 → 烯醇负离子 → 进攻另一分子醛 → 羟醛产物
- Cannizzaro 和羟醛是**竞争关系**，有 α-H 时羟醛胜出

### 12.4 现实类比

**Cannizzaro 像"互助小组"**：
- 两个人（两分子醛）都没有资源（无 α-H）
- 一个人给出自己的氢（负氢迁移）
- 给出氢的人变得更有价值（被氧化为酸）
- 接受氢的人也获得帮助（被还原为醇）
- 碱（OH⁻）是"组织者"——撮合了这个交换

1. **内分子 Cannizzaro（Intramolecular）**：
   - 二醛化合物（如邻苯二甲醛）

2. **不对称 Cannizzaro**：
