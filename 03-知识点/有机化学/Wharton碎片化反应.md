# Wharton 碎片化反应（Wharton Fragmentation）

> **Wharton 碎片化**是 **Eschenmoser 碎片化**的别称，特指 alpha,beta-不饱和酮的环氧与对甲苯磺酰肼（TsNHNH2）反应，发生协同碎片化，生成炔烃和酮的反应。属于 1,2,3,4-碎片化模式的经典实例，广泛用于大环酮合成和酮炔构建。

## 一、定义

**碎片化（Fragmentation）**：通过 C-C 键断裂将分子切为碎片的反应。

**Wharton 碎片化 / Eschenmoser 碎片化**：alpha,beta-不饱和酮的**环氧**与**对甲苯磺酰肼（TsNHNH2）**反应，发生**协同碎片化**，生成**炔烃**和**酮**（在不同位置上）的反应。

**核心特征**：
- 产物含有一根**炔键**——这是碎片化反应中非常罕见的结果
- 是 **1,2,3,4-碎片化模式**的经典实例
- 广泛用于**大环酮合成**和**酮炔构建**

**通式**：
$$
\ce{环氧-\alpha,\beta-不饱和酮 + TsNHNH2 ->[碎片化] 炔烃 + 酮}
$$

---

## 二、核心原理

### 2.1 碎片化的基本需求

碎片化需求**推电子**和**拉电子**——C-C 键需要极化才能断裂。

- **电子源（推）**：提供电子推动 C-C 键断裂
- **电子槽（拉）**：接受电子对，帮助断裂发生
- **1,2,3,4 碎片化模式**：电子源（1）→ C2-C3 键断裂 → 电子槽（4，离去基）

![[mineru/中文版clayden-chinese-34-36907-999_images/3e525df932cafb8d9af7811edc4f084178c2323b2c67ca2014d5f233c7d8a497.jpg]] — 碎片化推-拉原理

### 2.2 反应机理

**步骤 1：腙的形成**
- 环氧酮 + TsNHNH2 → 对甲苯磺酰腙

**步骤 2：环氧开环**
- 腙不稳定 → 环氧通过消除开环 → 产生羟基

**步骤 3：协同碎片化（1,2,3,4 模式）**
- 羟基"推"（电子源）
- 对甲苯亚磺酸根（ArSO2-）+ N2 "拉"（电子槽）
- 2-3 位 C-C 键断裂
- 所有断裂的键彼此平行，在两根双键两侧反叠

**步骤 4：产物**
- 炔烃 + 酮

**注意**：离去基是**对甲苯亚磺酸根（ArSO2-）**而非对甲苯磺酸根——这是一个重要的细节。

### 2.3 1,2,3,4 碎片化模式

| 位置 | 角色 | 功能 |
|:---|:---|:---|
| **1** | 羟基（OH）| **电子源（推）**——孤对电子推动断裂 |
| **2-3** | C-C 键 | **断裂键**——碎片化发生的核心 |
| **4** | 离去基（ArSO2-/N2）| **电子槽（拉）**——接受电子对 |

### 2.4 立体化学控制

碎片化受**立体化学严格控制**——需要**反叠式（anti-periplanar）**几何。

- 断裂的 C-C 键必须与离去基处于反叠式
- 顺式异构体（两个取代基都平伏）→ 可碎片化
- 反式异构体（OTs 直立）→ 无 C-C 键与 OTs 反叠 → 主要发生 E2 消除

![[mineru/中文版clayden-chinese-34-36907-999_images/d4605106647f0e586f292886375e0cf7d034b0edb077baa4ea75b12f65d17e5c.jpg]] — 碎片化立体化学控制

![[mineru/中文版clayden-chinese-34-36907-999_images/eef52eb25e4a3f5d1047069a3563a0c327ea6a5b22d12936b17bc7a5ff275639.jpg]] — 顺式构象（可碎片化）

![[mineru/中文版clayden-chinese-34-36907-999_images/6e4b26a2742645d004cca51143e3d155c86eee88027870e1b94fbbc83ff0fe77.jpg]] — 反式构象（不可碎片化，E2 为主）

碎片化对双键几何是**立体专一性**的——类似 E2 消除。

### 2.5 为什么产物是炔烃？

碎片化同时断裂了两根 C-C 键并形成两根新的 pi 键：
- 一侧形成 **C≡C 三键**（炔烃）
- 另一侧形成 **C=O 双键**（酮）

**驱动力**：
1. N2 离去（极好离去基）
2. 亚磺酸根离去
3. 炔键的形成（强键）
4. 酮的形成（稳定羰基）

## 三、关键结论

1. **Wharton / Eschenmoser = 环氧酮 + TsNHNH2 → 炔 + 酮**
2. **碎片化需求推电子和拉电子**——C-C 键需要极化才能断裂
3. **碎片化受立体化学控制**——反叠式几何是必需的
4. **碎片化对双键几何是立体专一性的**——类似 E2 消除
5. **产物含炔键**——碎片化反应中非常罕见的特征
6. **离去基是亚磺酸根（ArSO2-）**而非磺酸根
7. **核心应用**：大环酮合成、酮炔构建

---

## 四、适用条件与限制

1. **必须有环氧-alpha,beta-不饱和酮骨架**——底物限制较严格
2. **需对甲苯磺酰肼**——与羰基形成腙是启动步骤
3. **严格立体化学要求**——碎片化要求反叠几何
4. **产物炔酮可能敏感**——需控制后续反应条件
5. **大环合成中的优势环大小**——通常用于 10-16 元环

### 5.1 碎片化 vs 重排

| 概念 | 特征 | 关系 |
|:---|:---|:---|
| **碎片化** | C-C 键断裂，分子切成碎片 | 电子源和电子槽**相间**（1,2,3,4）时发生 |
| **重排** | 骨架重组，原子迁移到新位置 | 电子源和电子槽**相邻**时发生 |

### 5.2 Wharton 碎片化 vs Grob 碎片化

| 维度 | Wharton 碎片化 | Grob 碎片化 |
|:---|:---|:---|
| **底物** | 环氧-alpha,beta-不饱和酮 + TsNHNH2 | 1,2,3,4-体系（OH + 离去基）|
| **产物** | **炔 + 酮** | 烯 + 羰基 |
| **机理** | 协同碎片化 | 协同或离子型 |
| **离去基** | ArSO2- + N2 | OTs, N2 等 |
| **竞赛频率** | 低（决赛级）| 中（进阶）|

### 5.3 Wharton 碎片化 vs Semipinacol 重排

- **Wharton 碎片化**：1,2,3,4 碎片化 → C-C 键断裂 → 分子切成碎片
- **Semipinacol 重排**：1,2-迁移 → 骨架重组 → 分子保持完整
- **关键区别**：电子源和电子槽**相间**（Wharton，碎片化）vs **相邻**（Semipinacol，重排）

## 七、例题

### 7.1 例题 1（★★★ 机理）

**题目**：画出 Wharton 碎片化的完整机理，标出电子源、断裂键和电子槽。

**解答**：
1. 环氧酮 + TsNHNH2 → 磺酰腙
2. 环氧开环 → 羟基
3. **1,2,3,4 碎片化**：
   - 1 = OH（推）
   - 2-3 = C-C 键（断）
   - 4 = ArSO2- + N2（拉）
4. 产物：炔 + 酮

### 7.2 例题 2（★★★★ 合成设计）

**题目**：麝香酮（muscone，十五元环酮）的合成中有一步将环氧酮转化为酮炔，指出该反应。

**解答**：
- **反应**：Wharton / Eschenmoser 碎片化
- **底物**：alpha,beta-不饱和大环酮的环氧 + TsNHNH2
- **产物**：十五元环酮炔
- **后续**：选择性氢化炔键 → 麝香酮

### 7.3 例题 3（★★★★ 立体化学）

**题目**：Wharton 碎片化中，为什么所有断裂的键必须彼此平行？

**解答**：
- 碎片化是**协同反应**
- 所有电子流动需在同一过渡态中完成
- 反叠几何确保断裂的 sigma 键电子能流入新形成的 pi 键
- 若几何不匹配，碎片化不能发生

### 7.4 例题 4（★★★★★ 扩环设计）

**题目**：十氢化萘碎片化扩环的原理是什么？

**解答**：
- 6,6-稠合十氢化萘 → 外环是十元环
- 使稠合键作为 2-3 键断裂
- 碎片化后得到十元环产物
- 这是碎片化扩环的经典策略

## 八、易错点

1. **"产物是烯不是炔"** → Wharton 碎片化的独特之处在于产物含**炔键**
2. **"离去基是磺酸根"** → 离去基是**亚磺酸根（ArSO2-）**
3. **"所有环氧酮都能碎片化"** → 需要**alpha,beta-不饱和**且**反叠几何匹配**
4. **"Wharton 碎片化是重排"** → 它是**碎片化**，不是重排（C-C 键断裂）
5. **"碎片化没有立体要求"** → 反叠几何是**严格必需的**
6. **"碎片化与消除竞争时总是碎片化占优"** → 反式异构体中 OTs 直立时主要发生 E2 消除

## 九、竞赛拓展

### 9.1 天然产物全合成中的应用

| 天然产物 | 合成中的 Wharton 碎片化步骤 | 特点 |
|:---|:---|:---|
| **麝香酮（muscone）**| 大环酮炔 → 氢化 → 十五元环酮 | 经典大环合成 |
| **环十五酮（exaltone）**| 类似麝香酮路线 | 香料工业 |
| **外-西部松小蠹集合素（exo-brevicomin）**| 碎片化构建关键中间体 | 昆虫信息素 |
| **保幼激素（juvenile hormone）**| **两次串联 Wharton 碎片化** | 控制三根三取代双键的立体化学——极其精妙的应用 |

### 9.2 两次串联碎片化的精妙之处

保幼激素合成中，两次 Wharton / Eschenmoser 碎片化：
1. 第一次碎片化控制第一根双键的立体化学
2. 第二次碎片化控制第二根和第三根双键
3. 每次碎片化的反叠几何精确锁定双键的 E/Z 构型

这是碎片化反应在立体化学控制上的**巅峰应用**。

### 9.3 碎片化扩环策略

6,6-稠合十氢化萘 → 外环是十元环 → 使稠合键作为 2-3 键断裂 → 十元环产物。

这是碎片化反应在**中环合成**中的经典应用，通过"扩环而非关环"绕开中环（8-11元环）的跨环张力和不利熵效应。
