# σ 迁移反应（Sigmatropic Rearrangement）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[49-周环反应]]

---

## 一、定义

**σ 迁移反应（Sigmatropic Rearrangement）**：一个 **σ 键** 迁移到共轭 π 体系的新位置，同时 π 体系发生重组的协同周环反应。

**通式**：
$$
\ce{[1,n]-迁移：R-H 从 1-位迁移到 n-位}
$$
$$
\ce{[m,n]-迁移：σ 键从 (1,1') 迁移到 (m,n)}
$$

**本质**：**σ 键在 π 体系中的"行走"**——是周环反应三大类型（环加成、电环化、σ 迁移）之一。

## 三、核心原理

### 3.1 分类体系

**按迁移原子数分类**：

| 类型 | 名称 | 典型例子 |
|:---|:---|:---|
| **[1,n]-氢迁移** | 氢原子跨越 n 个原子迁移 | [1,5]-H 迁移（热允许）|
| **[1,n]-碳迁移** | 碳基团跨越 n 个原子迁移 | [1,3]-C 迁移（通常禁阻）|
| **[m,n]-σ 迁移** | σ 键两端分别迁移 m 和 n 个原子 | [3,3]-σ 迁移（Cope, Claisen）|

### 3.2 Woodward-Hoffmann 选择规则

**[1,n]-迁移**：

| 体系 | 同面 + 构型保持 | 同面 + 构型翻转 |
|:---|:---:|:---:|
| **4n+2**（如 [1,5]-H）| 热允许 | — |
| **4n**（如 [1,3]-C）| 禁阻 | 热允许 |

**[m,n]-迁移**：

| 体系 | 同面-同面 | 同面-异面 |
|:---|:---:|:---:|
| **m+n = 4n+2**（如 [3,3]）| 热允许 | 禁阻 |
| **m+n = 4n**（如 [1,3]）| 禁阻 | 热允许 |

### 3.3 [1,5]-氢迁移

**热允许的最常见 σ 迁移**：

$$
\ce{R-CH=CH-CH=CH-CH2-R' -> R-CH2-CH=CH-CH=CH-R'}
$$

**关键特征**：
- **六元环过渡态**（1+5=6）
- 协同的六电子过程
- 在天然产物合成中非常常见
- **同面迁移**：氢原子从π体系的一面离开，从同一面再次加入

**前线轨道描述**（Clayden Ch35）：
- 双烯 LUMO (ψ₃) + C-H键 HOMO（充满的σ轨道）
- 氢原子1s轨道是球形对称的，无论怎么画都意味着"保持"
- Woodward-Hoffmann：π4s + σ2s → 一个(4q+2)s组分，总数=1（奇数）→ 热允许

**重要干扰效应**（Clayden Ch35）：
- 环戊二烯中的[1,5]H移位非常迅速 → 取代环戊二烯的三种异构体混合物
- 这导致Diels-Alder反应通常得到多种加合物
- E,Z-双烯也容易发生[1,5]H移位 → 产物混合物 → 难以用于D-A反应

**类比**：像"氢原子跳过三个碳原子跳到了另一端"

### 3.4 [3,3]-σ 迁移

**最著名、最实用的 σ 迁移**：

| 反应 | 底物 | 产物 | 过渡态 |
|:---|:---|:---|:---|
| **Cope 重排** | 1,5-二烯 | 异构 1,5-二烯 | 六元环（全碳）|
| **Claisen 重排** | 烯丙基乙烯基醚 | γ,δ-不饱和羰基 | 六元环（含氧）|
| **Carroll 重排** | β-酮烯丙基酯 | γ,δ-不饱和酮 | 六元环（含氧+羰基）|
| **Ireland-Claisen** | 烯丙基酯烯醇硅醚 | γ,δ-不饱和酸 | 六元环（含氧+硅）|

**关键特征**：
- **六电子协同过程**
- **六元环过渡态**，椅式构象优先
- 热允许（m+n = 6 = 4n+2, n=1）
- **立体化学**：取代基倾向平伏位置，新双键偏好反式(E)

**Woodward-Hoffmann 处理**（Clayden Ch35）：
- 组分：一根σ键 + 两根π键
- 一个(4q+2)s组分（烯烃），没有(4r)a组分 → 总数=1 → 热允许
- σ键的s/a判定：两端都保持或都翻转 → σ2s；一端保持一端翻转 → σ2a

### 3.5 [2,3]-σ 迁移（Clayden Ch35 新增）

**过渡态**：五元环（2+3=5）
**需要**：绕五元环行进的箭头必须停止/开始于一个原子（碳阴离子、碳阳离子、或可变氧化态杂原子）

**典型例子**：

| 反应 | 底物 | 产物 | 特点 |
|:---|:---|:---|:---|
| **Wittig 重排** | 苄基烯丙基醚 + BuLi | 烯丙基苄醇 | 碳阴离子启动 |
| **Mislow-Evans 重排** | 烯丙基亚砜 | 烯丙醇（逆向） | S(II)/S(IV)互变 |
| **硒氧化** | 烯烃 + SeO₂ | 烯丙醇/烯醛 | [4+2] + [2,3]串联 |

**ω 组分概念**（Clayden）：
- 碳阴离子：ω2组分（2个电子）
- 碳阳离子：ω0组分（0个电子）
- 同面：ω2s；异面：ω2a
- [2,3]-σ重排可描述为 ω2a + σ2s + π2a → 一个(4q+2)s组分 → 热允许

**立体化学**：类椅式五元环过渡态，新π键反式形成，R取代基倾向平伏位点

### 3.6 [1,3]-碳迁移的特殊性

**[1,3]-C 迁移（同面 + 构型保持）**：禁阻
**[1,3]-C 迁移（同面 + 构型翻转）**：热允许

**本质**：
- 碳迁移涉及 sp³ 碳的轨道旋转
- 同面保持 = 轨道不翻转 → 4n 体系禁阻
- 同面翻转 = 轨道翻转一次 → 4n 体系允许
- 这就是 **"构型翻转"** 的原因

### 3.7 [1,7]-H 迁移（Clayden Ch35 补充）

- **立体化学**：异面迁移（氢从π体系顶面离开，从底面加入）
- **可行性**：可行（链足够灵活，七碳链可使氢从顶面到底面）
- **热反应**：允许但需异面
- **光反应**：同面（规则颠倒）
- **生物实例**：维生素D合成中的最后一步（预钙化醇 → 维生素D₃）

## 四、关键结论

1. **σ 迁移 = σ 键在 π 体系中迁移**
2. **[1,5]-H 迁移热允许**——六元环过渡态，六电子
3. **[3,3]-σ 迁移热允许**——Cope 和 Claisen 的核心
4. **[1,3]-C 迁移通常禁阻**（除非构型翻转）
5. **m+n = 4n+2 → 同面-同面热允许；m+n = 4n → 同面-异面热允许**
6. **立体化学由过渡态构象决定**——椅式优先

## 五、常见分类或情形

### 5.1 按迁移类型分类

| 迁移类型 | 电子数 | 热允许方式 | 典型例子 |
|:---|:---:|:---|:---|
| **[1,5]-H** | 6e | 同面 | 环戊二烯的 [1,5]-H 迁移 |
| **[1,5]-C** | 6e | 同面+保持 | 某些萜类重排 |
| **[3,3]-C** | 6e | 同面-同面 | Cope 重排 |
| **[3,3]-O** | 6e | 同面-同面 | Claisen 重排 |
| **[2,3]-σ 迁移** | 6e | ω2a+σ2s+π2a | Wittig 重排、Mislow-Evans |
| **[1,3]-C** | 4e | 同面+翻转 | 罕见 |
| **[1,7]-H** | 8e | 异面（热）/同面（光） | 维生素D合成 |

## 六、适用条件与限制

1. **需要共轭 π 体系**——σ 键必须在 π 体系旁边
2. **几何要求严格**——过渡态必须是特定的环大小
3. **[1,3]-H 迁移热禁阻**——四元环过渡态不利
4. **大迁移（如 [1,9]-H）困难**——大环过渡态不利
5. **构型翻转只在碳迁移时出现**——氢迁移无此问题

---

## 七、常见比较与易混点

### 7.1 σ 迁移 vs 电环化

- σ 迁移：σ 键在 π 体系中迁移
- 电环化：π 体系末端形成/断裂 σ 键
- 两者都是周环反应，但过程不同

### 7.2 Cope vs Claisen

- Cope：全碳 [3,3]-σ 迁移 → 1,5-二烯异构
- Claisen：含氧 [3,3]-σ 迁移 → 羰基化合物
- 两者机理完全相同，只是原子不同

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：[1,5]-氢迁移是热允许的还是热禁阻的？

**解答**：
- [1,5]-H 迁移 → 六元环过渡态
- 六电子过程（4n+2, n=1）
- **热允许**

### 10.2 例题 2（★★ [3,3]-σ 迁移）

**题目**：1,5-己二烯加热后的产物是什么？

**解答**：
- [3,3]-σ 迁移 = Cope 重排
- 产物：**异构的 1,5-己二烯**（若对称则产物相同）
- 若底物不对称 → 新结构的 1,5-二烯

### 10.3 例题 3（★★★ 立体化学）

**题目**：手性底物的 [3,3]-σ 迁移中，手性中心是否保持？

**解答**：
- [3,3]-σ 迁移通过椅式六元环过渡态
- 手性信息从烯丙基转移到产物的新手性中心
- **手性转移是立体专一的**
- 过渡态中取代基倾向于占据假平伏键位置

## 十一、易错点

1. ❌ **[1,3]-H 迁移热允许** → [1,3]-H 迁移**热禁阻**（四元环过渡态）
2. ❌ **所有 [3,3]-σ 迁移产物都不同** → 对称底物的 Cope 重排产物**相同**
3. ❌ **σ 迁移和自由基迁移一样** → σ 迁移是**协同周环反应**，无中间体
4. ❌ **[1,5]-C 迁移需要构型翻转** → [1,5]-C 迁移**保持构型**（6e 允许）
5. ❌ **只记规则不画过渡态** → 必须理解**过渡态的轨道重叠**

| "和自由基一样" | 是协同反应 | "σ迁移无中间体" |

### 12.3 入门例题

**题目**：为什么 [1,5]-H 迁移热允许而 [1,3]-H 迁移热禁阻？

**引导**：
- [1,5]-H 迁移：六元环过渡态 → 六电子 → 4n+2 → 热允许
- [1,3]-H 迁移：四元环过渡态 → 四电子 → 4n → 热禁阻（除非同面-异面）
- H 不能翻转构型 → [1,3]-H 无法同面-异面 → 完全禁阻

### 12.4 现实类比

**σ 迁移像"跳房子"**：
- π 体系 = 地上画的格子（共轭体系）
- σ 键 = 站在格子里的小孩
- σ 迁移 = 小孩从第 1 格跳到第 5 格
- [1,5]-H 迁移 = 一步跳 5 格（六元环过渡态 = 一个大弧线）
- [3,3]-σ 迁移 = 两个小孩同时跳（两端一起动），手拉手转一圈

## 十三、竞赛拓展

### 13.1 金属催化的 σ 迁移

- Pd、Rh 等金属可以催化某些热禁阻的 σ 迁移
- 金属改变过渡态的几何，使禁阻变为允许

### 13.2 串联 σ 迁移

- 多个 [3,3]-σ 迁移串联构建复杂骨架
- 在萜类和生物碱全合成中常见

### 13.3 Fischer 吲哚合成中的 [3,3]-σ重排（Clayden Ch35）

- 苯肼 + 酮 → 苯腙 → 烯胺（可烯醇化的酮）
- 烯胺具有 [3,3]-σ重排的理想结构
- 断裂的σ键是**弱的N-N σ键**，一根π键在苯环内
- 产物：高度不稳定的双键亚胺 → 芳香性恢复 → 吲哚
- 这是包含**两个氮原子**的[3,3]-σ重排

### 13.4 Cr(VI) 氧化中的共轭氧化（Clayden Ch35）

- 叔烯丙基醇 + CrO₃/酸性溶液
- 第一步：铬酸酯形成
- 第二步：铬酸酯基通过 **[3,3]-σ重排** 转移到烯丙基末端
- 两个氧和一个铬原子参与的 [3,3]-σ重排
- 一般的氧化反应最后一步：Cr(VI) → Cr(IV) → 歧化为 Cr(III)

### 13.5 柠檬醛的工业合成（Clayden Ch35）

- 维生素A合成的关键中间体
- **连续两个 [3,3]-σ重排**：Claisen → Cope
- Claisen的驱动力：羰基形成
- Cope的驱动力：末端双键→三取代双键 + 共轭结构
- 异戊烯基由分子一端迁到另一端，包含两次翻转

### 13.6 海藻信息素的自失活（Clayden Ch35）

- 褐海藻信息素（环丙烷结构）→ 自身 [3,3]-σ重排 → 环庚二烯（ectocarpene）
- 驱动力：三元环张力释放
- 常温下半衰期仅数分钟
- 早期研究者误认为ectocarpene是有效信息素

### 13.7 [2,3]-σ重排的合成应用（Clayden Ch35）

**Mislow-Evans 重排**：
- 烯丙型醇 + PhSCl → 次磺酸酯 → [2,3]-σ重排 → 烯丙基亚砜
- 亚砜 + (MeO)₃P / MeOH / 加热 → 逆向[2,3]-σ重排 → 另一个烯丙基醇
- **净结果**：烯丙型醇的"烷基化"（通常被认为不可能完成）

**SeO₂ 氧化（Riley氧化）**：
- 烯烃 + SeO₂ → [4+2]环加成 → 烯丙基亚硒酸 → [2,3]-σ重排 → 烯丙醇
- 催化量SeO₂ + t-BuOOH（后续氧化剂）
- 对甲基氧化 → 进一步氧化为烯醛/烯酮
