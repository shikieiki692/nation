# Fischer 吲哚合成（Fischer Indole Synthesis）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[52-杂环化合物]]、[[50-有机合成]]

---

## 一、定义

**Fischer 吲哚合成**：**芳基肼**与**酮或醛**在**酸催化**下反应，首先形成**苯腙**，随后经**[3,3]-σ 迁移**、**环化**和**芳构化**，生成**吲哚**衍生物的反应。

**通式**：
$$
\ce{Ar-NHNH2 + R-CO-R' ->[酸] 吲哚衍生物}
$$

**本质**：**最经典、最常用的吲哚合成法**——从简单的芳基肼和羰基化合物出发，一步构建吲哚骨架，是有机合成中构建吲哚环的"基石"反应。

## 三、核心原理

### 3.1 反应机理

**步骤 1：苯腙的形成**
$$
\ce{Ar-NHNH2 + R2C=O -> Ar-NH-N=CR2 + H2O}
$$
- 芳基肼的 NH₂ 进攻羰基碳
- 脱水形成苯腙

**步骤 2：互变异构为烯胺**
$$
\ce{Ar-NH-N=CR2 <-> Ar-N=N-CHR2}
$$
- 苯腙 ↔ 烯胺式互变异构
- 烯胺式是 [3,3]-σ 迁移的前体

**步骤 3：[3,3]-σ 迁移（关键步骤）**
```
烯胺式发生 [3,3]-σ 迁移：
- N-N 键断裂
- 芳环邻位碳与烯胺碳形成新键
- 生成 二氢吲哚中间体（indoline）
```

**步骤 4：芳构化（失去氨）**
```
二氢吲哚 → 失去 NH₃ → 芳构化为吲哚
```

### 3.2 区域选择性

**不对称酮（R ≠ R'）的 Fischer 合成**：

| 取代基 | 主要产物 | 原因 |
|:---|:---|:---|
| **R 更富电子（给电子）** | 2-R'-吲哚 | [3,3]-迁移时富电子侧优先 |
| **R 位阻更大** | 2-R'-吲哚 | 位阻大的基团倾向于留在 3-位 |

**规律**：
- **[3,3]-σ 迁移的方向性**决定了取代基的位置
- 通常 **2-位是较小/较吸电子的取代基**
- **3-位是较大/较富电子的取代基**

### 3.3 底物范围

**芳基肼**：
- 苯肼（最常用）
- 取代苯肼（给电子/吸电子取代基均可）
- 萘肼 → 苯并吲哚

**羰基化合物**：
- **醛**：产物是 **3-未取代吲哚**（R = H）
- **酮**：产物是 **2,3-二取代或 3-取代吲哚**
- **环酮**：产物是并环吲哚（如四氢咔唑）
- **1,3-二羰基**：产物是多取代吲哚

### 3.4 [3,3]-σ 迁移的周环反应视角（Ch.7 深化）

Fischer 吲哚合成中的 **[3,3]-σ 迁移步骤**属于**周环反应家族**，与 Ch.7 中的多种协同反应具有共同的轨道对称性基础。理解这种联系是竞赛中**跨章融会贯通**的关键。

#### 与 Cope 重排的对比

| 特征 | **Cope 重排** | **Fischer 中的 [3,3]-迁移** |
|:---|:---|:---|
| **底物** | 1,5-二烯 | 芳基肼烯胺式 |
| **协同电子数** | 6 个电子 | 6 个电子 |
| **过渡态** | 六元环椅式/船式 | 六元环椅式 |
| **杂原子参与** | 无（全碳）| N–N 键参与 |
| **结果** | C–C 键迁移 | C–C 键形成 + 稠环构建 |
| **竞赛考纲** | 条目 49（σ 迁移）| 条目 49 → 52（跨章桥梁）|

**核心共同点**：
- 都是 **热允许的 [3,3]-σ 迁移**——Woodward-Hoffmann 规则允许
- 都经历**六元环过渡态**
- 都是**协同、无中间体**的一步反应

#### 与 Claisen 重排的对比

| 特征 | **Claisen 重排** | **Fischer 中的 [3,3]-迁移** |
|:---|:---|:---|
| **底物** | 烯丙基乙烯基醚 | 芳基肼烯胺式 |
| **杂原子** | 氧（O）| 氮（N）|
| **产物** | γ,δ-不饱和羰基 | 二氢吲哚（含 N 杂环）|
| **后续转化** | 互变异构为羰基 | 芳构化为吲哚 |

**核心共同点**：
- 都是**杂原子参与的 [3,3]-σ 迁移**
- Fischer 中的 N–N 键断裂 + C–C 键形成，与 Claisen 中的 C–O 键断裂 + C–C 键形成**机理同构**
- ABOC 将 Fischer 机理归入 Ch.9 杂环合成，但其核心步骤的**物理本质**是 Ch.7 的周环反应

#### 前线轨道分析

**[3,3]-σ 迁移的 HOMO-LUMO 解释**：
- 烯胺式的 **HOMO**（N 孤对 + C=C π 键）与芳环的 **LUMO**（或反向组合）对称性匹配
- 热条件下，基态 HOMO 的 ψ₂ 轨道在两端具有**相同相位** → [3,3]-迁移热允许
- 这与 Cope、Claisen、Diels-Alder 的**对称性分析完全统一**

#### 竞赛中的跨章联系

**Ch.7 → Ch.9 的桥梁**：
- 考纲条目 **49-周环反应** 涵盖 σ 迁移反应
- 考纲条目 **52-杂环化合物** 涵盖吲哚合成
- Fischer 吲哚合成是**唯一同时横跨 49 和 52 两个考纲条目的经典反应**

**典型竞赛设问方式**：
> "写出 Fischer 吲哚合成中关键步骤的名称，并说明其属于哪一类反应。"
>
> 答：[3,3]-σ 迁移（或 [3,3]-sigmatropic rearrangement），属于**周环反应**中的**σ 迁移反应**。

## 四、关键结论

1. **Fischer = 芳基肼 + 羰基化合物 → 吲哚**
2. **[3,3]-σ 迁移是关键步骤**
3. **苯腙 ↔ 烯胺互变异构是前体**
4. **不对称酮有区域选择性**
5. **环酮 → 并环吲哚**
6. **最经典、最常用的吲哚合成法**

## 五、常见分类或情形

### 5.1 按羰基化合物分类

| 羰基类型 | 产物 | 例子 |
|:---|:---|:---|
| **醛** | 3-H 吲哚 | 苯肼 + 丙醛 → 2-甲基吲哚 |
| **对称酮** | 3-取代吲哚 | 苯肼 + 丙酮 → 2,3-二甲基吲哚 |
| **不对称酮** | 2,3-二取代吲哚（混合物）| 苯肼 + 甲乙酮 |
| **环酮** | 并环吲哚 | 苯肼 + 环己酮 → 四氢咔唑 |

### 5.2 按催化剂分类

| 酸 | 强度 | 适用 |
|:---|:---|:---|
| **HCl / H₂SO₄** | 强 | 常规底物 |
| **PPA（多聚磷酸）** | 强 | 难反应底物 |
| **TsOH** | 中强 | 温和条件 |
| **Lewis 酸（ZnCl₂）** | 中 | 对酸敏感底物 |

## 六、适用条件与限制

1. **需要酸催化**
2. **温度**：回流至 150°C
3. **芳基肼对空气敏感**——需新鲜制备或密封保存
4. **某些官能团不耐强酸**
5. **不对称酮产物可能为混合物**
6. **位阻太大的酮可能不反应**

## 七、常见比较与易混点

### 7.1 Fischer vs Larock vs Bartoli

| | **Fischer** | **Larock** | **Bartoli** |
|:---|:---|:---|:---|
| **底物** | 芳基肼 + 酮/醛 | 邻碘苯胺 + 炔烃 | 邻硝基甲苯 + 乙烯基格氏 |
| **催化剂** | 酸 | Pd(0) | 无（格氏反应）|
| **产物** | 2,3-二取代吲哚 | 2-或3-取代吲哚 | 7-取代吲哚 |
| **通用性** | **最广** | 现代方法 | 特殊位置 |

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：苯肼与丙酮在酸催化下的 Fischer 吲哚合成产物是什么？

**解答**：
- 苯肼 + 丙酮（对称酮）→ 苯腙
- [3,3]-σ 迁移 + 芳构化
- 产物：**2,3-二甲基吲哚**

### 10.2 例题 2（★★ 区域选择性）

**题目**：苯肼与甲乙酮（CH₃COCH₂CH₃）反应，主要产物是什么？

**解答**：
- 甲乙酮是不对称酮
- [3,3]-迁移方向：乙基（较大/较富电子）倾向于在 3-位
- 主要产物：**2-甲基-3-乙基吲哚**
- 次要产物：**3-甲基-2-乙基吲哚**

### 10.3 例题 3（★★★ 环酮）

**题目**：苯肼与环己酮的 Fischer 合成产物是什么？

**解答**：
- 环己酮 + 苯肼 → 苯腙
- [3,3]-迁移 + 芳构化
- 产物：**1,2,3,4-四氢咔唑**（并环吲哚）
- 后续脱氢 → 咔唑

## 十一、易错点

1. ❌ **Fischer 只适用于酮** → 醛**也适用**（产物 3-H 吲哚）
2. ❌ **[3,3]-迁移产物只有一种** → 不对称酮可能生成**混合物**
3. ❌ **苯腙就是产物** → 苯腙只是**中间体**，还需 [3,3]-迁移和芳构化
4. ❌ **所有酸都行** → 需要**足够强的酸**促进 [3,3]-迁移
5. ❌ **Fischer 是纯粹的离子型/缩合反应** → 关键的 [3,3]-σ 迁移步骤是**协同周环反应**，不是离子机理
6. ❌ **Fischer 的 [3,3] 与 Cope 是两类反应** → Fischer 的核心步骤**就是** [3,3]-σ 迁移，与 Cope、Claisen **同属周环反应家族**

### 12.3 入门例题

**题目**：为什么 Fischer 吲哚合成中 [3,3]-σ 迁移是关键步骤？

**引导**：
- 苯腙本身是"直链"分子——不含吲哚的稠环结构
- [3,3]-σ 迁移使芳环与侧链发生"折叠"
- 形成新的 C–C 键 → 构建稠环骨架
- 这是分子内" rearrangement"的魔力
- **与 Cope 重排对比**：Cope 是 1,5-二烯的 [3,3]-迁移；Fischer 的烯胺式可以看作"含氮的 1,5-二烯类似物"
- **前线轨道视角**：烯胺的 HOMO 与芳环 LUMO（或反向）对称性匹配 → 热允许
- 这就是 ABOC 将 Fischer 放在 Ch.9（杂环合成），但其核心步骤本质是 Ch.7（周环反应）的原因

### 12.4 现实类比

**Fischer 合成像"折纸鹤"**：
- 芳基肼 + 羰基 = 一张长方形纸
- 苯腙形成 = 纸上画好折痕
- [3,3]-σ 迁移 = 沿着折痕折叠
- 芳构化 = 最后整形
- 最终产物 = 一只立体精致的纸鹤（吲哚）
- 不同的"折法"（不同的酮）→ 不同样式的纸鹤（不同取代的吲哚）
