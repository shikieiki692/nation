# 周环反应

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[49-周环反应]]

## 一、定义

**周环反应**是指通过**环状过渡态**协同进行的反应，反应过程中化学键的断裂和形成同步发生，**无中间体生成**，反应的立体化学和区域选择性受**轨道对称性**严格控制。

- **核心特征**：
  1. **协同机理**：所有键的断裂和形成在同一过渡态中完成
  2. **环状过渡态**：反应物轨道以环状方式重叠
  3. **无中间体**：不同于离子型或自由基反应
  4. **轨道对称性控制**：反应的允许/禁阻由前线轨道对称性决定
- **三大类型**：
  1. **环加成（Cycloaddition）**：如 Diels-Alder [4+2]
  2. **电环化（Electrocyclic）**：共轭多烯 ↔ 环烯烃
  3. **σ 迁移（Sigmatropic）**：σ 键沿共轭体系迁移

## 三、核心原理

### 3.1 Woodward-Hoffmann 规则

**Woodward-Hoffmann 规则**是判断周环反应允许/禁阻的核心理论，基于**前线轨道对称性**和**芳香过渡态理论**。

**判断方法**：

| 方法                        | 核心思想                                    | 适用     |
| :------------------------ | :-------------------------------------- | :----- |
| **前线轨道法（FMO）**            | 考察 HOMO 与 LUMO 的对称性匹配                   | 所有周环反应 |
| **相关图法**                  | 跟踪反应物到产物的轨道相关                           | 电环化    |
| **芳香过渡态法（Hückel/Möbius）** | 4n+2 电子 Hückel 过渡态允许；4n 电子 Möbius 过渡态允许 | 所有周环反应 |

#### Clayden 中的 WH 规则正式表述（Ch34）

**版本一**：
> 在热周环反应中，$(4q+2)_s$ 和 $(4r)_a$ 组分的数目之和必须是**奇数**。

**版本二（更易记）**：
> 在一个允许的热周环反应中：
> 带有 2, 6, 或 10 个电子的同面组分的数目 + 带有 0, 4, 或 8 个电子的异面组分的数目 = **一个奇数**

- **组分**：作为一个单元参与周环反应的一根键或一条轨道。双键 = π2 组分，双烯 = π4 组分。
- **s** = 同面（suprafacial）：两端在同侧形成新键。
- **a** = 异面（antarafacial）：两端在对面形成新键。
- 不计入的组分：$(4q+2)_a$ 和 $(4r)_s$ 可以有任意多，它们不参与计数。

**Diels-Alder 的 WH 分析**：π4s（双烯，同面）+ π2s（烯烃，同面）→ 一个 $(4q+2)_s$ 组分，没有 $(4r)_a$ 组分 → 总和 = 1（奇数）→ **允许**。

**为什么 [4+4] 和 [2+2]（同面-同面）热禁阻**：它们具有偶数个 $(4q+2)_s$ 和 $(4r)_a$ 组分 → 总和为偶数 → **禁阻**。

#### 芳香过渡态与允许性

- 电子总数为 $(4n+2)$ 的环加成反应，如果是同面的，则往往被允许。
- 它们被认为具有"**芳香过渡态**"——与芳香性对 $(4n+2)$ 电子的需求明显有联系。
- D-A 过渡态包含六个离域的 π 电子，可看作一个留有全部 π 键、但丢失了两根 σ 键的苯环。

### 3.2 电环化反应的选择规则

**定义**：共轭多烯末端形成 σ 键（关环）或环烯烃断裂 σ 键（开环）。

**立体化学取决于 π 电子数和条件**：

| π 电子数 | 热反应（基态）| 光反应（激发态）|
|:---:|:---:|:---:|
| **4n**（如 1,3-丁二烯，n=1）| **顺旋（conrotatory）** | **对旋（disrotatory）** |
| **4n+2**（如 1,3,5-己三烯，n=1）| **对旋（disrotatory）** | **顺旋（conrotatory）** |

**记忆口诀**：
- **4n 热顺旋，光对旋**
- **4n+2 热对旋，光顺旋**

**解释**：
- **顺旋（conrotatory）**：两个末端 p 轨道**同向**旋转（都顺时针或都逆时针）
- **对旋（disrotatory）**：两个末端 p 轨道**反向**旋转（一个顺时针，一个逆时针）

### 3.3 环加成反应的选择规则

**定义**：两个不饱和片段通过环状过渡态形成两个新 σ 键。

**通式**：[m+n] 环加成

| 电子数（m+n）πe | 热反应 | 光反应 |
|:---:|:---:|:---:|
| **4n**（如 [2+2]）| **禁阻**（同面-同面）| **允许** |
| **4n+2**（如 [4+2]）| **允许** | **禁阻** |

**重要实例**：
- **[4+2] Diels-Alder**：热允许，协同进行
- **[2+2]**：热禁阻（同面-同面），光允许；烯酮和异氰酸酯的 [2+2] 热允许（2s+2a 途径，正交 π 体系）
- **[4+3]**：热允许（氧烯丙基阳离子 + 双烯，六电子）
- **[4+6]**：热允许（双烯 + 三烯，十电子）
- **[6+4]**、[8+2]：热允许

#### suprafacial（同面）与 antarafacial（异面）定义（Clayden Ch34）

- **同面（suprafacial, s）**：组分的两端在**同侧**形成新键。
- **异面（antarafacial, a）**：组分的两端在**对面**形成新键。
- 绝大多数环加成反应中，两个组分都**同面**反应。
- 烯酮的 [2+2] 是罕见的 **2s+2a** 例子：烯酮的 C=C=O 正交 π 体系允许一个组分同面、另一个异面。

### 3.4 σ 迁移反应的选择规则

**定义**：σ 键沿共轭 π 体系迁移到新位置。

**通式**：[i,j]-σ 迁移

| 类型 | 电子数（i+j）| 热允许的条件 |
|:---|:---:|:---|
| **[1,5]-H 迁移** | 6e（4n+2）| **同面（suprafacial）**允许 |
| **[1,3]-H 迁移** | 4e（4n）| 热禁阻（同面）；光允许 |
| **[3,3]-σ 迁移** | 6e（4n+2）| **同面-同面**允许（热）|
| **[1,7]-H 迁移** | 8e（4n）| 热同面禁阻，异面（antarafacial）允许 |

**经典 [3,3]-σ 迁移**：
- **Claisen 重排**：烯丙基芳基醚 → 邻烯丙基苯酚
- **Cope 重排**：1,5-二烯 → 异构 1,5-二烯
- **Carroll 重排**：β-酮烯丙基酯 → α-烯丙基-β-酮酸

## 四、关键结论

1. **周环反应的立体化学是严格预测的**：不是统计的，而是轨道对称性决定的
2. **热反应和光反应的选择性相反**：基态 HOMO 和激发态 HOMO 的对称性相反
3. **[4+2] Diels-Alder 是最重要、最常用的周环反应**：热允许、协同、高立体选择性
4. **[3,3]-σ 迁移在合成中极重要**：Claisen 和 Cope 重排是构建 C-C 键的强大工具
5. **电环化的顺旋/对旋决定产物的立体化学**：必须根据 π 电子数正确判断

## 五、常见分类

### 5.1 环加成反应

| 反应类型 | 电子数 | 热/光 | 说明 |
|:---|:---:|:---:|:---|
| **[4+2] Diels-Alder** | 6e | 热允许 | 最重要；二烯 + 亲二烯体 → 六元环 |
| **[2+2]（普通烯烃）** | 4e | 光允许 | 热同面-同面禁阻 |
| **[2+2]（烯酮/异氰酸酯）** | 4e | **热允许** | 2s+2a 途径，正交 π 体系 |
| **[3+2] 1,3-偶极环加成** | 6e | 热允许 | 如叠氮-炔（Click 化学）、OsO₄ 双羟基化 |
| **[4+3]** | 6e | 热允许 | 氧烯丙基阳离子 + 双烯 → 七元环 |
| **[4+6]** | 10e | 热允许 | 双烯 + 三烯 |
| **[6+4]** | 10e | 热允许 | 三烯 + 二烯 |
| **[8+2]** | 10e | 热允许 | 八隅体 + 亲双烯体 |
| **臭氧解** | — | 热 | 两次 [3+2] + 一次逆 [3+2] |

#### 臭氧解作为环加成（Clayden Ch34）

- 臭氧（O₃）是 1,3-偶极体，与烯烃发生 [3+2] 环加成 → 初级臭氧化物（极不稳定）。
- 逆 [3+2] 分解 → 醛 + 羰基氧化物（新的 1,3-偶极体）。
- 羰基氧化物 + 醛 → 二级臭氧化物（ozonide）。
- 这是**两次 1,3-偶极环加成 + 一次逆 1,3-偶极环加成**的串联过程。

#### OsO₄ 双羟基化作为 [3+2] 环加成（Clayden Ch34）

- OsO₄ 与烯烃的反应是 **[3+2] 环加成**（1,3-偶极环加成）。
- OsO₄ 可看作偶极体（锇有大量轨道容纳四根双键）。
- 协同加成 → 两个 OH 的相对立体化学必定保持 **syn**。
- Os 由 Os(VIII) 被还原到 Os(VI)。
- 与 m-CPBA 或 Br₂ 的亲电加成不同，OsO₄ 与缺电子和富电子烯烃反应几乎一样好。

| 反应名称 | 类型 | 电子数 | 热允许方式 | 说明 |
|:---|:---|:---:|:---:|:---|

## 六、适用条件

### 6.1 Diels-Alder 反应的最佳条件

| 因素 | 条件 | 原因 |
|:---|:---|:---|
| **二烯** | 需 s-cis 构象 | s-trans 无法关环 |
| **亲双烯体** | 缺电子（有 EWG）为佳 | 降低 LUMO 能级 |
| **二烯电子性质** | 富电子（有 EDG）为佳 | 升高 HOMO 能级 |
| **溶剂** | 非极性溶剂（如甲苯、己烷）| 协同反应对溶剂不敏感 |
| **温度** | 室温至回流 | 热允许，通常不需高温 |
| **Lewis 酸催化** | AlCl₃, TiCl₄, ZnCl₂ 等 | 络合亲双烯体的 EWG → 加速反应、增强选择性 |

### 6.2 电环化反应的条件

| 因素 | 热反应 | 光反应 |
|:---|:---|:---|
| **条件** | 加热 | UV 光照 |
| **控制方式** | π 电子数决定顺旋/对旋 | 与热反应相反 |
| **立体化学** | 由 HOMO 对称性决定 | 由激发态 HOMO 对称性决定 |
| **可逆性** | 开环/关环均可发生 | 开环/关环均可发生 |

### 6.3 σ 迁移反应的条件

| 反应 | 条件 | 注意 |
|:---|:---|:---|
| **Claisen 重排** | 加热（150-200°C）| 热驱动的 [3,3]-σ 迁移 |
| **Cope 重排** | 加热 | 可逆反应，平衡取决于取代基 |
| **[1,5]-H 迁移** | 加热 | 同面迁移，立体专一 |

## 七、常见比较

### 7.1 周环反应 vs 离子型反应 vs 自由基反应

| 维度 | 周环反应 | 离子型反应 | 自由基反应 |
|:---|:---|:---|:---|
| **机理** | 协同，环状过渡态 | 分步，有离子中间体 | 链式，有自由基中间体 |
| **中间体** | **无** | 碳正离子/碳负离子 | 自由基 |
| **立体化学** | **高度立体专一** | 可能外消旋化 | 通常非立体专一 |
| **溶剂效应** | 小 | 大（极性溶剂稳定离子）| 小 |
| **催化剂** | 通常不需要 | 酸/碱催化常见 | 引发剂 |
| **区域选择性** | 由轨道重叠决定 | 由电荷分布决定 | 由键强度决定 |

### 7.2 Diels-Alder vs 其他环加成

| 维度 | [4+2] Diels-Alder | [2+2] 环加成 |
|:---|:---|:---|
| **电子数** | 6e（4n+2）| 4e（4n）|
| **热反应** | **允许** | **禁阻**（多数情况）|
| **光反应** | 禁阻 | 允许 |
| **立体化学** | 顺式加成（syn）| 依条件而定 |
| **应用广泛性** | 极广 | 受限（光化学为主）|

## 十、例题

### 例题 1：电环化反应的立体化学

**题目**：预测 (2E,4Z,6E)-2,4,6-辛三烯在加热条件下的电环化产物的立体化学。

**解析**：
1. **π 电子数**：2,4,6-辛三烯有 6 个 π 电子 → **4n+2 体系（n=1）**
2. **热反应**：根据规则，4n+2 体系热反应为 **对旋（disrotatory）**
3. **对旋含义**：两个末端 p 轨道**反向**旋转
4. **底物构型**：(2E,4Z,6E)
   - C2-C3 是 E，C4-C5 是 Z，C6-C7 是 E
   - 对旋关环时，末端甲基的相对取向由对旋方式决定
5. **产物**：生成 **顺式-5,6-二甲基-1,3-环己二烯**（两个甲基在环同侧）

**注意**：必须画出三维旋转过程才能准确判断产物立体化学。

### 例题 2：Diels-Alder 反应的区域选择性

**题目**：1-甲氧基-1,3-丁二烯与丙烯醛（CH₂=CH-CHO）发生 D-A 反应，预测主产物。

**解析**：
1. **电子性质**：
   - 二烯：1-甲氧基 → **富电子**（EDG 提升 HOMO）
   - 亲双烯体：丙烯醛有 CHO（EWG）→ **缺电子**（降低 LUMO）
2. **正常 D-A**：富电子二烯 + 缺电子亲双烯体
3. **区域选择性**（"邻对位规则"）：
   - 二烯的 EDG（OMe）导向位置与亲双烯体的 EWG（CHO）最终呈 **1,2-关系（邻位）**
4. **内型规则**：亲双烯体的 EWG（CHO）倾向于处于 **内型**（endo）位置

**主产物**：甲氧基和醛基在环己烯环上呈 **邻位（cis）** 关系，且醛基处于内型。

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| "周环反应需要催化剂" | 与离子型反应混淆 | 周环反应**通常不需要催化剂**（热或光驱动）| Diels-Alder 需要酸催化吗？什么情况下加 Lewis 酸？ |
| "4n 体系热反应总是禁阻" | 过度概括 | 4n 体系热反应禁阻的是**同面-同面**方式；**同面-异面**或**异面-异面**可能允许 | [2+2] 环加成在什么情况下热允许？ |
| "电环化顺旋/对旋记反" | 记忆混淆 | **4n 热顺旋，4n+2 热对旋**；光反应相反 | 1,3-丁二烯热关环是顺旋还是对旋？ |
| "σ 迁移没有立体要求" | 忽略同面/异面 | σ 迁移也有**同面（suprafacial）**和**异面（antarafacial）**之分 | [1,5]-H 迁移是同面还是异面？为什么？ |
| "Claisen 重排是离子型反应" | 机理类型混淆 | Claisen 是**[3,3]-σ 迁移**，属于周环反应，**无中间体** | Claisen 重排和 Wagner-Meerwein 重排的本质区别是什么？ |

### 12.1 学习路径

```

     ↓

     ↓
后续分支：

  ├── 电环化反应（开环/关环的立体化学）

```

### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "周环反应很难预测" | 被轨道对称性吓退 | 强调：**记住选择规则表**即可预测大多数反应的允许性和立体化学 |
| "D-A 和 [2+2] 一样" | 都是环加成 | 明确：[4+2] 热允许，[2+2] 热禁阻，这是核心区别 |
| "光反应只是热反应的加速版" | 忽略激发态化学 | 强调：光反应涉及**激发态 HOMO**，对称性与基态相反 → 选择性相反 |

### 12.3 入门例题

**最简判断练习**：
> 判断下列反应在热条件下是否允许：
> 1. 1,3-丁二烯关环成环丁烯
> 2. 1,3,5-己三烯关环成 1,3-环己二烯
> 3. [2+2] 环加成

**答案**：
1. **允许**（4n=4，热顺旋允许）
2. **允许**（4n+2=6，热对旋允许）
3. **禁阻**（同面-同面 [2+2] 热禁阻）
**引导**：让学生画出每种情况的 HOMO，看末端 p 轨道的位相匹配。

### 12.4 现实类比

**"舞蹈编排"类比周环反应**：
- 周环反应像一场**精心编排的集体舞**——所有舞者（原子）同时移动，没有一个先动或后动
- **离子型反应** = 独舞/双人舞——一个舞者先跳（离去基离去），然后另一个接上（亲核进攻）
- **Diels-Alder [4+2]** = 四人和两人的圆圈舞 → 围成一个六人圈（六元环）
- **[2+2] 热禁阻** = 两个人想牵手围成圈，但总是"同手同脚"（对称性不匹配）→ 牵不上
- **电环化顺旋/对旋** = 两个人面对面，要手拉手围成圈——**顺旋** = 两人都往同一个方向转身；**对旋** = 一个往左转、一个往右转
- **光反应** = 舞台灯光变化（激发态）→ 舞者的"惯用手"换了 → 原来牵不上的现在能牵上了
