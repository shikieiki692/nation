# 重排反应

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[39-重排反应]]

## 一、定义

**重排反应**是指反应过程中分子骨架发生重组，一个原子或基团从分子中的一个位置迁移到另一个位置的反应。净结果是分子内基团的位置改变，通常伴随着邻近官能团的转化。

- **核心特征**：
  1. 迁移基团（Migrating group）带着其成键电子对移动
  2. 通常经过一个**缺电子中心**（碳正离子、氮宾、卡宾等）
  3. 迁移的驱动力是生成**更稳定的中间体或产物**
- **与消除/取代的区别**：重排反应不改变分子式（异构化），而是重新分配原子间的连接方式
- **竞赛地位**：重排反应是化竞合成题中**构建复杂骨架**的重要手段，也是机理题的高频考点

## 三、核心原理

### 3.1 重排反应的通式与驱动力

**通式**：
$$\ce{A-B-C <=> A-C-B}$$

**主要驱动力**：

| 驱动力 | 说明 | 典型例子 |
|:---|:---|:---|
| **生成更稳定的碳正离子** | 不稳定碳正离子 → 稳定碳正离子 | Wagner-Meerwein 重排 |
| **释放环张力** | 小环 → 大环或开环 | 环扩张/收缩重排 |
| **形成更稳定的官能团** | 如酮比二醇更稳定 | Pinacol 重排 |
| **芳香性恢复/获得** | 非芳香 → 芳香 | 某些芳构化重排 |
| **趋避反芳香性** | 反芳香中间体快速重排 | 某些环系重排 |

### 3.2 1,2-迁移的基本规律

**1,2-迁移**是最常见的重排模式：

$$\ce{R3C-CR2-X <-> R2C-CR3-X}$$

**迁移基团的亲核性排序**（迁移能力）：

$$\ce{Ph > H > R3C > R2CH > RCH2 > CH3}$$

**解释**：
- **芳基（Ph）迁移最快**：可通过苯环的 π 体系稳定过渡态中的部分正电荷
- **氢迁移也很快**：H 体积小，迁移能垒低
- **烷基迁移**：三级 > 二级 > 一级 > 甲基（超共轭稳定过渡态）

**迁移基团保持构型**：1,2-迁移通常是协同的，迁移基团在迁移过程中**保持其立体化学构型**。

### 3.3 重排反应的分类体系

| 分类维度 | 类型 | 说明 |
|:---|:---|:---|
| **按中间体** | 碳正离子重排 | 最常见，如 Wagner-Meerwein |
| | 碳负离子重排 | 如 Stevens 重排、Wittig 重排 |
| | 卡宾/氮宾重排 | 如 Wolff 重排、Curtius 重排 |
| | 自由基重排 | 较少见 |
| **按迁移距离** | 1,2-迁移 | 最常见 |
| | 1,3-迁移及更远 | 如 Claisen-Cope（σ 迁移）|
| **按迁移基团** | 氢迁移 | H 迁移 |
| | 烷基迁移 | R 迁移 |
| | 芳基迁移 | Ar 迁移 |

## 四、关键结论

1. **碳正离子重排是竞赛中最常见的重排类型**：Wagner-Meerwein 重排是几乎所有碳正离子反应中都可能伴随的副反应
2. **1,2-氢迁移通常比 1,2-烷基迁移更快**：因为 H 体积小，能垒低
3. **芳基迁移非常快**：苯环可稳定迁移过渡态的部分正电荷
4. **重排反应中迁移基团保持构型**：这是判断重排机理的重要依据
5. **Pinacol 和 Beckmann 重排是合成酮/酰胺的经典方法**

## 五、常见分类

### 5.1 碳正离子重排（最常用）

| 反应名称 | 底物 | 产物 | 关键特征 |
|:---|:---|:---|:---|
| **Wagner-Meerwein 重排** | 碳正离子邻位有迁移基团 | 更稳定的碳正离子 | 醇脱水、卤代烃 SN1 中的常见副反应 |
| **Pinacol 重排** | 邻二醇（频哪醇）| α-羟基酮（频哪酮）| 酸催化，一个 OH 质子化后离去 |
| **半频哪醇重排** | β-氨基醇/卤代醇 | 羰基化合物 | 与 Pinacol 类似，但离去基不同 |
| **Tiffeneau-Demjanov 扩环** | 1-氨基-2-醇 | 扩环酮 | 氨基重氮化后离去，引发重排 |
| **Baeyer-Villiger 氧化** | 酮 + mCPBA | 酯 | 插入 O，基团迁移能力决定产物 |

### 5.2 碳负离子/卡宾重排

| 反应名称 | 底物 | 产物 | 关键特征 |
|:---|:---|:---|:---|
| **Stevens 重排** | 季铵盐/锍盐（邻接羰基）| 迁移产物 | 碱催化，碳负离子邻位迁移 |
| **Wittig 重排** | 苄基/烯丙基醚 | 醇 | 强碱生成碳负离子后重排 |
| **Wolff 重排** | α-重氮酮 | 烯酮 | 光/热分解，卡宾中间体 |
| **Curtius 重排** | 酰基叠氮 | 异氰酸酯 | 热分解，氮宾中间体 |
| **Hofmann 重排** | 酰胺 + Br₂/碱 | 少一个碳的胺 | 氮宾中间体，烷基迁移 |
| **Lossen 重排** | 异羟肟酸 | 异氰酸酯 | O-酰基化后热分解 |

### 5.3 其他重要重排

| 反应名称 | 底物 | 产物 | 关键特征 |
|:---|:---|:---|:---|
| **Beckmann 重排** | 肟（酮肟）| 酰胺 | 酸催化，反式基团迁移 |
| **Favorskii 重排** | α-卤代酮 | 羧酸衍生物 | 碱催化，环缩小 |
| **Benzilic acid 重排** | 苯偶酰（二芳基-1,2-二酮）| 苯偶酰酸 | 碱催化，芳基迁移 |
| **Claisen 重排** | 烯丙基芳基醚 | 邻烯丙基苯酚 | [3,3]-σ 迁移（周环反应）|
| **Cope 重排** | 1,5-二烯 | 异构 1,5-二烯 | [3,3]-σ 迁移（周环反应）|

## 六、适用条件

### 6.1 碳正离子重排的条件

| 因素 | 条件 | 原因 |
|:---|:---|:---|
| **酸催化** | 强酸（H₂SO₄, HCl, AlCl₃ 等）| 质子化 OH 或生成碳正离子 |
| **离去基** | 需有好的离去基（H₂O, X⁻ 等）| 生成碳正离子的前提 |
| **温度** | 通常加热 | 提供重排所需活化能 |
| **溶剂** | 极性质子或非质子溶剂 | 稳定碳正离子中间体 |

### 6.2 特定重排的条件

| 重排反应 | 特定条件 | 注意 |
|:---|:---|:---|
| **Pinacol 重排** | 酸催化；频哪醇在 H₂SO₄ 中加热 | 两个 OH 中质子化更碱性的那个离去 |
| **Beckmann 重排** | 酸催化（PCl₅, H₂SO₄, SOCl₂）| **反式基团迁移**是关键立体化学特征 |
| **Baeyer-Villiger** | mCPBA, CF₃CO₃H 等过酸 | 迁移能力：H > Ph > 三级烷基 > 二级 > 一级 |
| **Wolff 重排** | 光解或加热（>150°C）| 从 α-重氮酮出发 |
| **Curtius/Hofmann** | 热分解（80-150°C）| 氮气离去驱动反应 |

## 七、常见比较

### 7.1 Wagner-Meerwein vs 其他碳正离子重排

| 维度 | Wagner-Meerwein | Pinacol 重排 |
|:---|:---|:---|
| **中间体** | 碳正离子 | 质子化后脱水生成的碳正离子 |
| **驱动力** | 生成更稳定碳正离子 | 生成更稳定的羰基化合物 |
| **底物** | 含碳正离子的化合物 | 邻二醇 |
| **产物** | 重排后的碳正离子（继续反应）| α-羟基酮（频哪酮）|
| **立体化学** | 迁移基团构型保持 | 迁移基团构型保持 |

### 7.2 基团迁移能力排序

**在 Baeyer-Villiger 氧化和 Pinacol 重排中**：

$$\ce{H > Ph > 叔烷基 > 环己基 > 仲烷基 > 伯烷基 > 甲基}$$

**在碳正离子重排中**：

$$\ce{芳基 > H > 烷基}$$

**注意**：不同重排反应的迁移能力排序可能略有不同，因为过渡态的结构不同。

## 十、例题

### 例题 1：Pinacol 重排

**题目**：频哪醇（2,3-二甲基-2,3-丁二醇）在浓硫酸作用下发生重排，写出产物和机理。

**解析**：
1. **质子化**：一个 OH 被质子化（两个 OH 等价，任选其一）
2. **离去**：水离去 → 生成三级碳正离子
3. **1,2-甲基迁移**：相邻甲基带着一对电子迁移到带正电荷的碳上
4. **去质子**：相邻 OH 上的质子离去 → 生成频哪酮（3,3-二甲基-2-丁酮）

**产物**：**频哪酮**（pinacolone）

### 例题 2：Beckmann 重排的立体化学

**题目**：苯乙酮肟（PhC(Me)=NOH）在 PCl₅ 作用下发生 Beckmann 重排，预测产物。

**解析**：
1. Beckmann 重排的关键规则：**与肟羟基反式（anti）的基团迁移**
2. 苯乙酮肟有两种构型：
   - **E 型**（Ph 与 OH 反式）：Ph 迁移 → 产物为 **N-苯基乙酰胺（PhNHCOCH₃）**
   - **Z 型**（Me 与 OH 反式）：Me 迁移 → 产物为 **N-甲基苯甲酰胺（MeNHCOPh）**
3. 实际反应中通常得到热力学更稳定的肟构型对应的产物

**关键记忆**：**Anti 迁移**——与 OH 反位的基团走。

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| "重排反应改变分子式" | 与分解反应混淆 | 重排反应**不改变分子式**，只是原子连接方式改变 | 重排反应和消除反应的本质区别是什么？ |
| "所有碳正离子都会重排" | 过度概括 | 只有当重排能生成**更稳定**的碳正离子时才会发生 | 甲基碳正离子邻位是季碳，会重排吗？ |
| "1,2-迁移中迁移基团构型翻转" | 与 SN2 混淆 | 1,2-迁移是**协同**的，迁移基团**保持构型** | 1,2-迁移与 SN2 的立体化学有何不同？ |
| "Beckmann 重排中顺式基团迁移" | 记忆错误 | Beckmann 是**反式（anti）**基团迁移 | 为什么 Beckmann 重排中 Z/E 构型决定产物？ |
| "Baeyer-Villiger 中烷基比 H 更易迁移" | 迁移能力记反 | Baeyer-Villiger 中：**H >> 芳基 > 三级烷基 > 二级 > 一级** | 环己酮的 Baeyer-Villiger 氧化产物是什么？ |

```

     ↓

     ↓
后续分支：
  ├── Wagner-Meerwein 重排（碳正离子化学）
  ├── Pinacol/Beckmann/Baeyer-Villiger（人名反应）

```

**建议课时**：1.5 课时（1,2-迁移原理 0.4 + 碳正离子重排 0.5 + 其他重排 0.3 + 综合应用 0.3）

### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "重排是'错误'的反应路径" | 认为重排是副反应 | 强调重排是**设计的合成工具**（如 Pinacol → 酮，Beckmann → 酰胺）|
| "所有迁移基团能力相同" | 忽略迁移能力差异 | 用实验数据说明芳基 > H > 烷基的迁移能力排序 |
| "重排只发生在碳正离子中" | 知识面窄 | 展示碳负离子重排（Stevens）、卡宾重排（Wolff）等 |

### 12.3 入门例题

**最简判断练习**：
> 判断下列反应是否可能发生重排，若可能写出重排产物：
> 1. 新戊醇（(CH₃)₃CCH₂OH）在 H₂SO₄ 中脱水
> 2. 频哪醇在 H₂SO₄ 中加热

**答案**：
1. **会发生重排**——先脱水生成一级碳正离子 (CH₃)₃CCH₂⁺，极不稳定，立即发生 1,2-甲基迁移 → 生成更稳定的三级碳正离子 (CH₃)₂C⁺CH₂CH₃ → 最终产物为 2-甲基-2-丁烯
2. **Pinacol 重排** → 频哪酮（3,3-二甲基-2-丁酮）
**引导**：第 1 题是典型的"一级碳正离子必重排"案例。

### 12.4 现实类比

**"搬家"类比重排反应**：
- 分子像一个家庭，重排是**家庭成员换房间**
- **碳正离子重排** = 家里有个空房间（碳正离子的空 p 轨道），隔壁邻居（迁移基团）带着家具（电子对）搬进来
- **驱动力** = 大家都想住得更舒服（更稳定的碳正离子 = 更好的"房型"）
- **Wagner-Meerwein** = 从小房间搬到大房间（不稳定 → 稳定碳正离子）
- **Pinacol 重排** = 两个相邻的储藏室（两个 OH）中，一个被清空了，另一个邻居趁机扩建（形成 C=O）
- **Beckmann** = 搬家规则很严格——只能**对面（anti）**的邻居搬过来
