# 芳香亲电取代反应

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[41-芳香亲电取代反应]]

## 一、定义

**芳香亲电取代（Electrophilic Aromatic Substitution, SEAr）** 是指亲电试剂（E⁺）进攻芳香环，**取代环上的氢原子**（或其他基团），同时**保留芳香体系**的反应。是芳香化合物最核心、最典型的反应类型。

$$
\ce{Ar-H + E^+ -> Ar-E + H^+}
$$

- **核心特征**：进攻的是 **E⁺**（不是 Nu⁻），产物 **恢复芳香性**
- **与亲电加成的本质区别**：烯烃 + E⁺ → 加成产物（不饱和度下降）；芳环 + E⁺ → 取代产物（芳香性保留）
- **原因**：芳香性的稳定化能（苯约 150 kJ/mol）远大于加成产物的稳定性，芳环 "宁愿" 恢复芳香性而释放 H⁺

## 三、核心原理

### 3.1 两步机理（所有 SEAr 通用）

**第一步（速率决定步，RDS）**：亲电试剂 E⁺ 进攻芳环 π 电子云，形成 **σ-络合物（Wheland 中间体 / 芳正离子）**

```
      E⁺                    E
       |                    |
   H-C   C-H     →      H-C   C-H   （+）
     ||     |              |     |
    芳环   H               芳环  H
                           |
                          sp³ 碳（芳香性暂时破坏）
```

- σ-络合物中，被进攻的碳变为 **sp³ 杂化**
- 正电荷通过共振离域到 **邻位和对位**（共 3 个位置带正电荷）
- 此步 **破坏芳香性**，能垒高，是 RDS

**第二步（快）**：碱（通常是反应体系中存在的负离子，如 HSO₄⁻、AlCl₄⁻、X⁻）夺取质子，恢复芳香性

- 此步 **重建芳香性**，释放大量能量，极快且不可逆

### 3.2 σ-络合物的共振结构

以苯的硝化为例，σ-络合物的正电荷离域：

```
      NO₂                      NO₂                       NO₂
       |                        |                         |
   H-C   C-H   (+)         H-C   C-H              H-C   C-H
     /     \    ──→        /  (+)  \    ⟷         (+)     \
    CH       CH            CH       CH              CH       CH
      \     /                \     /                  \     /
       CH                    CH                      CH
```

正电荷分布在 **邻位、对位**（相对于 E⁺ 所在碳），这是定位效应的**电子根源**。

## 四、关键结论

1. **定位效应的本质**：取代基通过 **诱导效应（I）** 和 **共轭效应（C）** 影响 σ-络合物在邻、间、对位的相对稳定性
2. **活化基团 = 邻对位定位**：给电子基（+I / +C）稳定 σ-络合物的正电荷 → 降低活化能 → 加速反应
3. **钝化基团 = 间位定位**：吸电子基（-I / -C） destabilize σ-络合物 → 提高活化能 → 减速反应
4. **卤素是特殊钝化但邻对位定位基**：-I > +C，整体钝化；但 +C 效应使邻对位的 σ-络合物相对间位更稳定
5. **多取代苯的定位**：**活化基团主导**；定位一致时加和；定位矛盾时活化基团优先
6. **Friedel-Crafts 烷基化可逆且易重排**；酰基化不可逆且无重排 → 合成上更常用酰基化后还原

## 五、常见分类

### 5.1 经典 SEAr 反应

| 反应 | 亲电试剂（E⁺） | 催化剂/条件 | 产物 | 备注 |
|:---|:---|:---|:---|:---|
| **硝化** | NO₂⁺ | 浓 HNO₃/浓 H₂SO₄ | 硝基苯 | 强放热，需控温 |
| **卤化** | X⁺（Cl⁺、Br⁺）| FeX₃、AlX₃ | 卤苯 | I₂ 需氧化剂（HNO₃）活化 |
| **磺化** | SO₃ | 浓 H₂SO₄ 或发烟 H₂SO₄ | 苯磺酸 | **可逆**（稀酸加热脱磺酸基）|
| **Friedel-Crafts 烷基化** | R⁺（碳正离子）| AlCl₃、FeCl₃ | 烷基苯 | **可逆、可重排**；多取代倾向 |
| **Friedel-Crafts 酰基化** | RCO⁺（酰基正离子）| AlCl₃（≥1 eq）| 芳酮 | **不可逆、无重排**；合成价值更高 |
| **氯甲基化（Blanc反应）** | ⁺CH₂Cl | HCHO/HCl + ZnCl₂ | 苄基氯 | 引入 -CH₂Cl 手柄 |
| **Gattermann-Koch 甲酰化** | ⁺CHO | CO/HCl + AlCl₃/CuCl | 芳醛 | 苯/烷基苯专用 |

### 5.2 定位基分类

| 类型 | 电子效应 | 代表基团 | 定位 | 活化/钝化 | σ-络合物稳定性分析 |
|:---|:---|:---|:---:|:---:|:---|
| **强活化** | +I, +C | -OH, -OR, -NH₂, -NHR, -NR₂ | o,p | 活化 | 孤对电子直接离域到 σ-络合物的正电荷中心 |
| **中等活化** | +I | -R, -Ar | o,p | 活化 | 超共轭效应稳定邻对位 σ-络合物 |
| **弱钝化** | -I < +C | -F, -Cl, -Br, -I | o,p | 钝化 | 卤素的 -I 使环整体电子密度下降，但 +C 在 o,p 位可离域正电荷 |
| **强钝化** | -I, -C | -NO₂, -CN, -SO₃H, -COOH, -CHO, -COR | m | 钝化 | 吸电子基使 σ-络合物正电荷更不稳定；间位 σ-络合物受吸电子影响最小 |
| **特殊** | +I（强） | -NR₃⁺ | m | 强钝化 | 正电荷直接排斥 E⁺ |

## 六、适用条件

### 6.1 各类 SEAr 反应的条件

| 反应 | 底物 | 试剂 | 催化剂 | 温度 | 溶剂 |
|:---|:---|:---|:---|:---:|:---|
| **硝化** | 芳烃 | HNO₃ | H₂SO₄（混酸）| 50-60°C | 混酸本身 |
| **溴化** | 芳烃 | Br₂ | FeBr₃ 或 AlBr₃ | 室温 | CCl₄、CS₂ |
| **氯甲基化** | 芳烃 | HCHO + HCl | ZnCl₂ | 60°C | 乙酸 |
| **F-C 烷基化** | 芳烃 | RX | AlCl₃ | 室温-回流 | CS₂、硝基苯 |
| **F-C 酰基化** | 芳烃 | RCOCl | AlCl₃（1.1-2 eq）| 室温-回流 | CS₂、CH₂Cl₂ |
| **磺化** | 芳烃 | 浓 H₂SO₄ | 无需 | 室温（苯）/ 80°C（甲苯）| H₂SO₄ |

### 6.2 Friedel-Crafts 反应的特殊限制

| 限制 | 原因 | 解决策略 |
|:---|:---|:---|
| **芳环上有强吸电子基（NO₂、CN）** | 环太缺电子，无法形成 σ-络合物 | 换用其他方法（如硝基还原后再 F-C）|
| **芳环上有 -NH₂、-OH（未保护）** | 与 Lewis 酸配位，使催化剂失活 | 先酰基化/烷基化保护氨基 |
| **F-C 烷基化多取代** | 烷基是活化基团，产物比底物更活泼 | 用大量底物；或改用 F-C 酰基化后还原 |
| **F-C 烷基化重排** | 碳正离子中间体可 1,2-迁移 | 改用 F-C 酰基化（酰基正离子不重排）|
| **乙烯基/芳基卤代烷不反应** | 乙烯基/芳基碳正离子极不稳定 | 换用其他偶联方法（Suzuki、Heck）|

## 七、常见比较

### 7.1 活化基 vs 钝化基的定位效应本质

以甲苯硝化 vs 硝基苯硝化为例：

**甲苯（-CH₃，活化，o,p-定位）**：
- -CH₃ 的 +I 和超共轭效应增加环电子密度
- σ-络合物在 **邻位/对位** 时，正电荷可离域到与 -CH₃ 相连的碳，超共轭稳定化
- σ-络合物在 **间位** 时，正电荷无法被 -CH₃ 直接稳定
- 结果：o/p 位 σ-络合物更稳定 → o/p 硝化快

**硝基苯（-NO₂，钝化，m-定位）**：
- -NO₂ 的 -I 和 -C 效应降低环电子密度
- σ-络合物在 **邻位/对位** 时，正电荷离域到与 -NO₂ 相连的碳，正电荷与缺电子的 NO₂ 相邻 → **极不稳定**
- σ-络合物在 **间位** 时，正电荷远离 -NO₂，相对最稳定
- 结果：m 位 σ-络合物相对最稳定 → m 硝化（虽然整体很慢）

### 7.2 F-C 烷基化 vs F-C 酰基化

| 维度 | F-C 烷基化 | F-C 酰基化 |
|:---|:---|:---|
| **亲电体** | R⁺（碳正离子）| RCO⁺（酰基正离子）|
| **重排** | 常见（碳正离子重排）| **无**（酰基正离子通过共振稳定，不重排）|
| **多取代** | 易多取代（产物更活化）| 不易多取代（酰基是钝化基）|
| **可逆性** | **可逆** | **不可逆** |
| **合成价值** | 中等（重排和多取代问题）| **高**（精确引入侧链）|
| **后续转化** | — | 羰基可还原为亚甲基（Clemmensen/Wolff-Kishner）|

**合成策略**：若要向芳环引入正丙基，不能直接 F-C 丙基化（因碳正离子重排为异丙基）。正确路线：
1. F-C **丙酰化** → 苯丙酮（无重排）
2. **Clemmensen 还原**（Zn/Hg, HCl）或 **Wolff-Kishner 还原**（NH₂NH₂, KOH）→ 正丙苯

## 十、例题

### 例题 1：多取代苯的定位规则

**题目**：预测下列化合物硝化的主产物：
1. 对甲基苯酚（p-cresol）
2. 间硝基甲苯

**解析**：

**1. 对甲基苯酚**：
- -OH（强活化，o,p-定位）vs -CH₃（弱活化，o,p-定位）
- 两者定位方向：-OH 的邻位 = -CH₃ 的邻位（一致）
- -OH 的活化能力 >> -CH₃ → **-OH 主导定位**
- 主产物：**2-硝基-4-甲基苯酚**（-OH 的邻位硝化）

**2. 间硝基甲苯**：
- -CH₃（o,p-定位）vs -NO₂（m-定位）
- -CH₃ 的 o,p 位：2,4,6 位
- -NO₂ 的 m 位：5 位（相对于 CH₃ 是间位）
- 两者定位矛盾：-CH₃ 要 4-位，-NO₂ 要 5-位
- **活化基团主导**：-CH₃ 是活化基（尽管弱），-NO₂ 是钝化基
- 主产物：**4-硝基-3-甲基硝基苯**（即 3-甲基-4-硝基硝基苯？更正：间硝基甲苯的 4-位是 -CH₃ 的对位，也是 -NO₂ 的间位，两者一致！）

**修正分析**：
间硝基甲苯：
```
    CH₃
     |
  6-C-2
   /   \
 5-C   C-3
   \   /
    C-4
     |
    NO₂
```
- -CH₃ 的 o,p 位：2,4,6（相对于 CH₃）
- -NO₂ 的 m 位：5（相对于 NO₂）
- 实际上 4-位同时是 -CH₃ 的对位和 -NO₂ 的间位 → **两者一致**
- 主产物：**4-硝基-3-甲基？不，是 4-硝基-1-甲基-3-硝基苯？**

重新编号：间硝基甲苯 = 1-甲基-3-硝基苯
- -CH₃ 在 1 位，-NO₂ 在 3 位
- -CH₃ 的 o,p 位：2,4,6
- -NO₂ 的 m 位：5（相对于 3 位的 NO₂，m 位是 5 和 1；1 位已被占）
- 所以一致位点是 4 和 6（相对于 CH₃ 是 o/p，相对于 NO₂：4 是邻位？不）

相对于 3-NO₂：
- 邻位：2,4
- 间位：5,1（1 被占）
- 对位：6

所以 -NO₂ 的 m 位 = 5。而 -CH₃ 的 o,p 位 = 2,4,6。
两者无交集！但 4 位是 -NO₂ 的邻位，2 和 6 是 -NO₂ 的邻位。

**正确结论**：间硝基甲苯中，-CH₃ 的 o,p 位（2,4,6）全是 -NO₂ 的邻/对位（被 -NO₂ 钝化）。
唯一相对可行的位点是 **5 位**（-NO₂ 的 m 位，-CH₃ 的 m 位）。
由于 -CH₃ 弱活化其 o,p 位，但那些位点又被 -NO₂ 强烈钝化，实际硝化极慢且发生在 **5-位**。

### 例题 2：F-C 烷基化的重排

**题目**：苯与 1-氯丙烷/AlCl₃ 反应的主产物是什么？

**解析**：
1. 1-氯丙烷 + AlCl₃ → 丙基碳正离子 CH₃CH₂CH₂⁺（1°，不稳定）
2. **重排**：1,2-氢迁移 → 异丙基碳正离子 (CH₃)₂CH⁺（2°，更稳定）
3. 异丙基碳正离子进攻苯环 → **异丙苯（枯烯）**

**若要得正丙苯**：改用 F-C **丙酰化** → 苯丙酮 → Clemmensen 还原 → 正丙苯。

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| 把 -Cl、-Br 当成间位定位基 | 只记 "吸电子 = 钝化 = m-定位"，忽略卤素的 +C | 卤素是 **o,p-定位但钝化**（唯一例外）| 为什么氯苯硝化比苯慢，但仍在邻对位？ |
| 多取代苯定位只看最近基团 | 未考虑活化基团优先和加和规则 | 三原则：① 活化基主导；② 一致时加和；③ 矛盾时看相对活化能力 | 间硝基苯酚的溴化主产物在哪？ |
| 忽略 F-C 烷基化的重排 | 以为烷基化直接引入直链烷基 | F-C 烷基化经碳正离子 → **必检查重排** | 如何用 F-C 反应向苯引入正丁基？ |
| 认为所有芳环都能发生 F-C | 未考虑强钝化基的影响 | 含 -NO₂、-CN、-COR 的芳环 **不发生 F-C** | 硝基苯为什么不能 Friedel-Crafts 烷基化？ |
| 磺化可逆性被忽略 | 只记磺化是 SEAr，忘记可逆 | 磺化是 **可逆的**；稀酸加热可脱磺酸基，用于占位/保护 | 如何利用磺化的可逆性合成邻溴甲苯？ |


### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "活化 = 反应快，钝化 = 不反应" | 把钝化等同于惰性 | 钝化只是相对苯慢，并非不反应；硝基苯在强条件下仍可硝化 |
| "定位效应只由电子效应决定" | 忽略空间位阻 | 大位阻基团（t-Bu）的邻位位阻极大 → 对位产物占绝对主导 |
| "F-C 烷基化和酰基化无区别" | 混淆两者 | 对比表强调：酰基化无重排、不可逆、不多取代 → 合成价值更高 |
| "SEAr 是唯一的芳环反应" | 忽略亲核取代和其他转化 | Birch 还原、SNAr、苯炔机理都是芳环的重要反应 |

### 12.3 入门例题

**最简定位题**：
> 甲苯硝化时，硝基主要进入哪个位置？为什么？

**答案**：**邻位和对位**（约 58% 对 + 37% 邻 + 5% 间）
**解析**：-CH₃ 是活化基团，+I 和超共轭效应。σ-络合物在 o/p 位时，正电荷可离域到与 -CH₃ 相连的碳，被超共轭稳定。
**引导**：画出 σ-络合物在对位时的共振式，标出正电荷如何离域到连有 -CH₃ 的碳。

### 12.4 现实类比

**「国王登基」类比 SEAr**：
- 芳环 = 一个王国，六员大臣（H）围坐
- E⁺ = 新候选人，想登基（取代 H）
- σ-络合物 = 短暂的摄政期（王国混乱，六边形结构破坏）
- 碱夺 H⁺ = 旧国王被劝退，新王正式登基
- 芳香性恢复 = 王国恢复稳定六边形结构
- **活化基团** = 王国本来富足（电子多），欢迎新王；**钝化基团** = 王国贫困（电子少），抗拒新王
- **定位效应** = 某些座位（o/p 或 m）更利于权力过渡（σ-络合物稳定）

## 十三、竞赛拓展

### 13.1 竞赛中的高级 SEAr 反应

| 反应 | 亲电体 | 特点 | 竞赛价值 |
|:---|:---|:---|:---|
| **Vilsmeier-Haack 甲酰化** | ⁺CH=NMe₂Cl⁻（氯代亚胺盐）| 温和，适用于活泼芳环和杂环 | 引入醛基 |
| **Gattermann-Koch 甲酰化** | ⁺CHO | CO/HCl + AlCl₃ | 工业制芳醛 |
| **Meerwein 芳基化** | 重氮盐 + 烯烃 → 芳基化产物 | 自由基-亲电机理混合 | 构建芳基取代烯烃 |
| **Schiemann 反应** | ArN₂⁺BF₄⁻ → Ar-F（热解）| 芳环引入氟 | 制备氟芳烃 |
| **Ipso 取代** | E⁺ 取代已有取代基（如 t-Bu、SO₃H、I）| 可逆取代基被替换 | 合成中的占位-脱位策略 |

### 13.2 与杂环化学的交叉

- **吡咯、呋喃、噻吩**：比苯更活泼（5 电子芳杂环，π 电子密度高），SEAr 在 2-位
- **吡啶**：比苯更惰性（6 电子芳杂环，N 的 -I 效应），SEAr 在 3-位；且 F-C 反应困难
- **吲哚**：苯并吡咯，SEAr 优先在吡咯环的 3-位
