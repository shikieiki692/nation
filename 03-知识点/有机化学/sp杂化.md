# sp 杂化（sp Hybridization）

## 一、定义

**sp 杂化**是由 1 个 s 轨道和 1 个 p 轨道线性组合形成的杂化方式，产生 **2 个等价的 sp 杂化轨道**，剩余 **2 个未杂化的 p 轨道**。

- **几何构型**：直线形，键角 **180°**
- **s 成分**：50%（s : p = 1 : 1）
- **轨道夹角**：两个 sp 轨道互为 180°
- **未杂化 p 轨道**：互相垂直，且均垂直于 sp 轨道轴，可形成 2 个 π 键

## 三、核心原理

### 3.1 杂化过程

```
        激发              杂化
2s ↑↓  ──→  2s ↑  2p ↑ ↑  ──→  2 个 sp ↑ ↑  +  2 个 p ↑ ↑
```

- 基态 C：2s² 2p² → 激发态：2s¹ 2p³ → 杂化：2 个 sp + 2 个 p
- 2 个 sp 轨道形成 **2 个 σ 键**
- 2 个未杂化 p 轨道分别与相邻原子的 p 轨道侧向重叠，形成 **2 个 π 键**

### 3.2 σ 键与 π 键的分布

| 键类型 | 数量 | 形成方式 | 特点 |
|:---:|:---:|:---|:---|
| **σ 键** | 2 | sp–sp 或 sp–s 头对头重叠 | 键能较高，可自由旋转 |
| **π 键** | 2 | p–p 侧向重叠 | 电子云分布在键轴两侧，不能自由旋转 |

**C≡C 三键的本质**：1 个 σ 键 + 2 个互相垂直的 π 键。

### 3.3 电负性与酸性

杂化轨道中 **s 成分越高，电负性越大**，C–H 键的酸性越强：

| 杂化类型 | s 成分 | C–H pKa（近似）| 酸性 |
|:---:|:---:|:---:|:---:|
| sp³ | 25% | ~50 | 最弱 |
| sp² | 33% | ~44 | 中等 |
| **sp** | **50%** | **~25** | **最强** |

**末端炔烃的酸性**：
$$\ce{RC≡C-H + NaNH2 -> RC≡C^-Na^+ + NH3}$$

## 四、关键结论

1. **sp 杂化碳呈直线形**：所有与之直接相连的原子共线
2. **C≡C 三键 = 1σ + 2π**：π 键电子云呈圆柱形分布在 σ 键周围
3. **sp 碳的电负性最大**：s 成分 50% 使轨道更靠近核，吸电子能力最强
4. **两个 π 键互相垂直**：限制了相邻基团的旋转自由度
5. **累积双键的中心碳为 sp 杂化**：如丙二烯 H₂C=C=CH₂（中心碳 sp，两端 sp²）

## 五、常见分类

### 5.1 典型含 sp 杂化碳的化合物

| 化合物 | sp 杂化碳位置 | 键型 | 几何特征 |
|:---|:---|:---|:---|
| **乙炔** HC≡CH | 两个碳均为 sp | C≡C（1σ+2π）| 直线形，H–C≡C–H |
| **腈类** R–C≡N | C 和 N 均为 sp | C≡N（1σ+2π）| 直线形 |
| **CO₂** | C 为 sp，O 可视为 sp | 2 个 C=O | O=C=O 直线形 |
| **丙二烯** H₂C=C=CH₂ | 中心 C 为 sp | 2 个 C=C | 两端 CH₂ 平面互相垂直 |
| **异腈** R–N≡C | C 和 N 均为 sp | C≡N | 直线形 |

### 5.2 sp 杂化与其他杂化的对比

| 特征 | sp³ | sp² | **sp** |
|:---|:---:|:---:|:---:|
| **参与杂化轨道** | s + pₓ + pᵧ + p₂ | s + pₓ + pᵧ | s + pₓ |
| **杂化轨道数** | 4 | 3 | **2** |
| **未杂化 p 轨道数** | 0 | 1 | **2** |
| **几何构型** | 正四面体 | 平面三角形 | **直线形** |
| **键角** | 109.5° | 120° | **180°** |
| **s 成分** | 25% | 33% | **50%** |
| **典型键型** | 4σ | 3σ + 1π | **2σ + 2π** |
| **实例** | CH₄ | C₂H₄ | **C₂H₂** |

## 六、适用条件

### 6.1 判断原子是否为 sp 杂化

1. **成键数 = 2**（只计 σ 框架）
2. **周围原子共线**（键角 180°）
3. **需要形成多重键**：三键需要 2 个 π 键 → 必须保留 2 个 p 轨道 → 只能杂化 2 个轨道 → sp 杂化

### 6.2 常见误判

| 结构 | 误判 | 正确判断 |
|:---|:---|:---|
| CO₂ | O 是 sp² | O 用 2 个等价杂化轨道成键，可视为 sp 或不等性 sp² |
| 丙二烯中心碳 | sp² | **sp**（需与两个 π 键体系正交）|

## 十、例题

### 例题 1：杂化类型判断

**题目**：判断丙二烯（H₂C=C=CH₂）中心碳原子的杂化类型。

**解析**：
1. 中心碳形成 2 个 σ 键（与两端碳）
2. 需形成 2 个 π 键（与每个端碳各 1 个）
3. 2 个 π 键需 2 个互相垂直的 p 轨道
4. 因此中心碳只能提供 2 个轨道杂化（s + pₓ → 2 个 sp）
5. 剩余 2 个 p 轨道（pᵧ, p₂）分别形成 2 个 π 键

**结论**：中心碳为 **sp 杂化**，两端碳为 **sp² 杂化**。两个 CH₂ 平面互相垂直。

### 例题 2：酸性排序

**题目**：比较下列化合物中 C–H 键的酸性：乙烷、乙烯、乙炔、苯。

**解析**：

| 化合物 | C 的杂化 | s 成分 | pKa |
|:---|:---:|:---:|:---:|
| 乙烷 | sp³ | 25% | ~50 |
| 乙烯 | sp² | 33% | ~44 |
| 苯 | sp² | 33% | ~43 |
| **乙炔** | **sp** | **50%** | **~25** |

**结论**：酸性顺序：**乙炔 > 苯 ≈ 乙烯 > 乙烷**

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| "CO₂ 中 O 是 sp²" | 误判氧的杂化 | CO₂ 中 O 用 2 个 σ 键 + 2 对孤对电子，可按不等性 sp 理解 | CO₂ 中碳和氧的杂化类型？ |
| "三键碳是 sp²" | 混淆双键和三键 | 双键 → sp²（1 个 π）；三键 → **sp**（2 个 π）| 为什么三键碳不是 sp²？ |
| "sp 碳的酸性弱" | 忽略 s 成分影响 | s 成分越高，电负性越大，C–H 酸性越强 | sp³/sp²/sp 的 C–H 酸性排序？ |

### 12.3 入门例题

**最简判断练习**：
> 1. sp 杂化的键角是多少？
> 2. 乙炔中有几个 σ 键、几个 π 键？
> 3. 为什么末端炔烃有酸性？

**答案**：
1. **180°**
2. **3 个 σ 键**（2 个 C–H + 1 个 C–C σ），**2 个 π 键**
3. sp 杂化碳的 **s 成分高（50%）**，电负性大，C–H 键极性强，H 易以质子形式离去
