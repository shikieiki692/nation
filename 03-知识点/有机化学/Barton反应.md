# Barton反应

> Barton反应是一类以自由基机理为核心的光化学反应，主要用于**远程官能团化**和**脱羧/去氧转化**。由英国化学家Derek Barton（1969年诺贝尔化学奖得主）发现，在甾体化学和复杂天然产物合成中具有重要应用价值。

---

## 一、定义

**Barton反应（Barton Reaction）** 是指**亚硝酸酯在光照条件下发生均裂，生成烷氧自由基，随后发生分子内1,5-氢迁移（或更远距离），最终导致远程官能团化**的反应。广义上，Barton反应家族还包括：

- **Barton脱羧反应（Barton Decarboxylation）**：羧酸转化为硫代羟肟酸酯（Barton酯），在自由基引发下发生脱羧，生成烷基自由基
- **Barton-McCombie反应（Barton-McCombie Reaction）**：醇经黄原酸酯转化为烷基自由基，实现醇的去氧还原

## 三、反应原理 / 机理

### 3.1 经典Barton反应（亚硝酸酯光解）

**底物**：醇的亚硝酸酯（R-O-N=O）
**条件**：光照（hν）
**核心步骤**：

```
Step 1: 光均裂（引发）
  R-O-N=O  --hν-->  R-O• + •N=O  （亚硝酰自由基）

Step 2: 分子内1,5-氢迁移（关键步骤）
  R-O• 夺取分子内δ位（1,5-位）的氢原子
  形成碳自由基 + 醇

Step 3: 与NO结合
  碳自由基与•N=O结合形成亚硝基化合物

Step 4: 互变异构
  亚硝基化合物互变为肟（oxime）
```

**机理特征**：
- **1,5-氢迁移**是最有利的几何构型（六元环过渡态）
- 1,6-氢迁移也可能发生，但效率较低
- 1,4-氢迁移因五元环过渡态张力较大而不利

### 3.2 Barton脱羧反应

**底物**：羧酸转化为硫代羟肟酸酯（Barton酯，R-CO-S-N(OH)-R'）
**条件**：加热或光照，自由基引发剂（如AIBN）
**机理**：

```
Step 1: 引发剂产生自由基
  AIBN --Δ--> R•

Step 2: 自由基进攻硫原子
  R• + R-CO-S-N(OH)-R' --> R-S-N(OH)-R' + R-CO•  （酰基自由基）

Step 3: 脱羧
  R-CO• --> R• + CO2  （放出CO2）

Step 4: 链增长
  新生成的R•继续进攻另一分子Barton酯
```

**特点**：
- 温和条件下实现脱羧
- 生成的烷基自由基可进行后续反应（如加成、取代）
- 避免了强酸/强碱条件

### 3.3 Barton-McCombie反应（去氧反应）

**底物**：醇转化为黄原酸酯（xanthate，RO-CS-SR'）
**试剂**：Bu₃SnH + AIBN（自由基引发剂）
**机理**：

```
Step 1: 引发
  AIBN --Δ--> R•
  R• + Bu₃SnH --> RH + Bu₃Sn•

Step 2: 增长
  Bu₃Sn• + RO-CS-SR' --> Bu₃Sn-S-CS-OR + R'•
  （或Bu₃Sn•攫取硫，生成烷氧自由基中间体）

Step 3: 去氧
  烷氧自由基经β-断裂释放CS₂，生成烷基自由基
  R-O• --> R• + 羰基硫

Step 4: 链传递
  R• + Bu₃SnH --> R-H + Bu₃Sn•
```

**总结果**：R-OH --> R-H（醇被还原为烷烃，实现去氧）

## 四、核心结论 / 公式

1. **Barton反应的核心是光引发的自由基链式反应**，1,5-氢迁移是最有利的几何构型
2. **六元环过渡态决定选择性**：1,5-H迁移 >> 1,6-H迁移 >> 1,4-H迁移
3. **Barton脱羧的优势**：温和条件、避免强酸碱、可与其他自由基反应串联
4. **Barton-McCombie反应是醇去氧的经典方法**，特别适用于对酸/碱敏感的复杂分子
5. **所有Barton反应都遵循自由基化学的基本原则**：键强度决定选择性，六元环过渡态优先

## 五、分类与变体

| 变体 | 底物 | 条件 | 产物 | 核心应用 |
|:---|:---|:---|:---|:---|
| **经典Barton反应** | 亚硝酸酯（R-ONO） | hν | 远程肟/亚硝基化合物 | **远程官能团化** |
| **Barton脱羧** | Barton酯（硫代羟肟酸酯） | Δ/hν, 引发剂 | 脱羧产物 + CO₂ | **温和脱羧** |
| **Barton-McCombie** | 黄原酸酯（RO-CS-SR'） | Bu₃SnH/AIBN | 去氧产物（R-H） | **醇的去氧还原** |

## 六、适用范围与限制

### 6.1 经典Barton反应

**适用**：
- 含有适当δ位C-H键的醇（1,5-位有可攫取的氢）
- 甾体等刚性骨架（有利于固定1,5-几何关系）

**限制**：
- 需要1,5-位有可被攫取的氢
- 柔性分子可能产生多种产物
- 亚硝酸酯对光和热敏感，需现制现用

### 6.2 Barton脱羧

**适用**：
- 对酸碱敏感的羧酸
- 需要在中性条件下脱羧的体系
- 脱羧后自由基可进一步用于C-C键形成

**限制**：
- 需要预先制备Barton酯
- 硫代羟肟酸酯的合成增加步骤

### 6.3 Barton-McCombie去氧

**适用**：
- 复杂天然产物中的选择性去氧
- 对酸/碱/氧化还原敏感的底物
- 糖类、核苷酸等多元醇的选择性去氧

**限制**：
- 使用有毒的锡试剂（Bu₃SnH）
- 需要无氧条件
- 现代方法可用硅烷/硫醇替代锡试剂

## 七、与其他反应的比较

### 7.1 Barton-McCombie vs 其他去氧方法

| 方法 | 试剂 | 条件 | 优点 | 缺点 |
|:---|:---|:---|:---|:---|
| **Barton-McCombie** | Bu₃SnH/AIBN | 中性，温和 | 选择性高，兼容性好 | 锡试剂有毒 |
| **直接还原** | HI/红磷 | 强酸性 | 简单廉价 | 酸性条件破坏敏感基团 |
| **甲磺酸酯还原** | LiAlH₄ | 强还原性 | 经典方法 | 强还原剂，选择性差 |
| **现代替代** | (TMS)₃SiH | 中性 | 无金属，低毒 | 试剂较昂贵 |

### 7.2 Barton脱羧 vs 其他脱羧方法

| 方法 | 条件 | 机理 | 适用场景 |
|:---|:---|:---|:---|
| **Barton脱羧** | 中性，自由基 | 自由基链式 | 敏感底物，串联反应 |
| **Kolbe电解** | 电解 | 自由基偶联 | 对称二聚体 |
| **Hunsdiecker反应** | Ag盐 + Br₂ | 自由基 | 少一个碳的溴代物 |
| **热脱羧** | 高温 | 协同/离子 | 简单羧酸 |

## 十、典型例题

### 例题 1（难度 ⭐⭐⭐）
**题目**：经典Barton反应中，为什么1,5-氢迁移比1,4-氢迁移更有利？

**答案**：1,5-氢迁移通过**六元环过渡态**进行，几何上容易达到，且张力小；而1,4-氢迁移需要通过五元环过渡态，虽然也能发生，但通常效率较低。1,6-氢迁移通过七元环过渡态，在某些刚性骨架中也可能发生，但不如1,5-有利。

**解析**：这是理解Barton反应选择性的核心。六元环过渡态在有机反应中普遍有利（类似椅式构象的稳定性），Barton反应也不例外。

### 例题 2（难度 ⭐⭐⭐⭐）
**题目**：为什么Barton-McCombie反应特别适合复杂天然产物的去氧？

**答案**：
1. **条件温和**：中性自由基条件，不破坏酸/碱敏感的官能团
2. **选择性高**：通过黄原酸酯的引入实现特定羟基的去氧
3. **兼容性好**：与多种保护基和官能团兼容
4. **可预测**：自由基机理相对容易预测产物

**解析**：在复杂天然产物（如糖类、甾体、抗生素）合成中，经常需要在多羟基环境中选择性去除某一个羟基。Barton-McCombie反应通过将特定羟基转化为黄原酸酯，实现了这种选择性。

### 例题 3（难度 ⭐⭐⭐⭐）
**题目**：比较Barton脱羧与Hunsdiecker反应的异同。

**答案**：

| 对比项 | Barton脱羧 | Hunsdiecker反应 |
|:---|:---|:---|
| **底物** | 羧酸 → Barton酯 | 羧酸银盐 |
| **试剂** | 自由基引发剂 | Br₂或I₂ |
| **条件** | 中性，温和 | 加热 |
| **产物** | 少一个碳的烷烃/自由基产物 | 少一个碳的卤代烷 |
| **机理** | 自由基链式 | 自由基 |
| **优势** | 可串联其他自由基反应 | 直接得到卤代物 |

**解析**：两者都是利用自由基实现脱羧，但Barton脱羧更温和，生成的烷基自由基可进一步进行多种转化；Hunsdiecker反应则直接得到卤代产物。

## 十一、易错点与常见误解

1. **混淆Barton反应的不同变体**
   - 错误：将所有Barton反应混为一谈
   - 正确：经典Barton（亚硝酸酯光解）、Barton脱羧、Barton-McCombie是三个不同的反应

2. **忽视1,5-氢迁移的几何要求**
   - 错误：认为任何δ位氢都能发生迁移
   - 正确：需要六元环过渡态的几何匹配，刚性骨架更有利

3. **Barton-McCombie的试剂毒性**
   - 错误：忽视Bu₃SnH的毒性问题
   - 正确：现代方法可用(TMS)₃SiH等低毒替代试剂

4. **将Barton脱羧与简单热脱羧混淆**
   - 错误：认为所有脱羧都需要强热
   - 正确：Barton脱羧在温和条件下进行，适合敏感底物

## 十二、🎯 教学视角

### 学习路径建议
1. **先掌握自由基反应基础**（引发/增长/终止，键强度）
2. **理解光化学引发的特殊性**（光均裂 vs 热均裂）
3. **学习Barton反应的三个变体及其应用场景**
4. **通过甾体化学实例理解远程官能团化的价值**
5. **比较Barton方法与其他脱羧/去氧方法的优劣**

### 认知误区预警
- **误区**："自由基反应不可控，不适合精细合成"
- **纠正**：Barton反应家族证明自由基反应可以实现高选择性的远程官能团化和去氧
- **误区**："去氧就是简单还原"
- **纠正**：在复杂分子中，选择性去氧需要精巧的化学策略，Barton-McCombie是经典解决方案

### 入门例题推荐
**题目**：为什么甾体化学特别青睐Barton反应？

**答案**：甾体具有刚性四环骨架，这种刚性结构有利于固定1,5-几何关系，使Barton反应中的1,5-氢迁移具有高度区域选择性。同时，甾体分子中常有多个相似的反应位点，Barton反应通过自由基机理实现了其他方法难以达到的远程选择性官能团化。

### 现实类比 / 直观解释
- **1,5-氢迁移的六元环过渡态**：想象分子内"伸手够东西"——1,5-位的距离刚好合适（六元环），1,4-位太近（五元环张力），1,6-位太远（七元环不够紧凑）
- **Barton脱羧的温和性**：如同用"手术刀"（自由基）精准切除羧基，而不是用"大锤"（强酸强碱）砸碎整个分子

## 十三、竞赛拓展

1. **现代Barton-McCombie替代方法**：
   - 使用(TMS)₃SiH替代Bu₃SnH，避免锡的毒性
   - 光氧化还原催化下的去氧反应

2. **Barton反应的串联应用**：
   - 脱羧后自由基与烯烃的加成（Giese反应）
   - 脱羧后自由基的分子内环化

3. **不对称Barton反应**：
   - 在手性辅助基控制下的不对称脱羧
   - 用手性硫代羟肟酸酯实现立体选择性

4. **生物体系中的类似反应**：
   - 维生素B₁₂依赖的酶催化脱羧（类似Barton脱羧的自由基机理）
