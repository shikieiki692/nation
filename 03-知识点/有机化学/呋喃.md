- 总览：[[中国化学奥林匹克基本要求-总览]]

## 一、定义

呋喃（Furan）是一种含有一个氧原子的五元芳香杂环化合物，分子式为 C₄H₄O。氧原子以 sp² 杂化形式存在，其两对孤对电子中的一对参与构成芳香六隅体，使呋喃具有芳香性；但氧的强电负性也使呋喃的芳香性较苯为弱。

## 三、核心原理

### 3.1 电子结构与芳香性

- **6π 芳香体系**：氧提供 2 个电子（一对孤对电子），四个碳各提供 1 个电子，共 6 个 π 电子，满足 Hückel 4n+2 规则
- **富电子芳环**：氧向环供电子，但氧的电负性大，供电子能力弱于氮（吡咯）
- **芳香性较弱**：由于氧的电负性，呋喃的芳香性不如苯和噻吩，表现出一定的共轭双烯性质
- **取代活性顺序**：吡咯 > 呋喃 > 噻吩 > 苯

### 3.2 共振结构

呋喃的共振式显示 α-位电子密度高于 β-位：

```
      O               O⁺              O⁺
     / \             / \             / \
    /   \     ↔     /   \     ↔     /   \
   /     \         /     \         /     \
  负电荷主要分布在 α,β 位，α-位更富电子
```

### 3.3 亲电取代反应

- **反应位置**：主要在 α-位（2-位或 5-位）
- **反应活性**：比苯活泼，但比吡咯稳定（不易聚合）
- **典型反应**：
  - 卤化：与卤素反应得 α-卤代呋喃
  - 硝化：使用温和硝化试剂（如硝酸乙酰酯）
  - 磺化：使用吡啶-三氧化硫络合物
  - Friedel-Crafts 酰基化：可顺利进行
- **注意**：强酸条件下呋喃可能发生开环或聚合

### 3.4 Diels-Alder 反应

- **双烯体角色**：呋喃是经典的 4π 双烯体，可与亲双烯体发生 [4+2] 环加成
- **反应可逆性**：呋喃芳香性较弱，D-A 加成产物在高温下可发生 Retro-Diels-Alder 反应，重新生成呋喃
- **应用**：利用可逆性保护双烯或构建复杂环系

```
    O                      O
   / \                    / \
  /   \  +  亲双烯体  →  [D-A 加成产物]
 /     \                (氧桥双环)
 呋喃（4π）
```

### 3.5 2-吡喃酮的关联

- 2-吡喃酮（六元环内酯）可视为呋喃的扩展，具有共轭二烯 + 内酯双重性质
- 与亲双烯体 D-A 加成后脱 CO₂，可构建苯衍生物（见 [[杂环化合物]] 总览）

## 四、关键结论

1. **芳香性与双烯性的平衡**：呋喃既有芳香性（亲电取代），又有共轭双烯性（D-A 反应），这是氧原子电负性造成的独特性质
2. **α-位优先**：亲电取代专一地发生在 α-位，与吡咯、噻吩一致
3. **D-A 可逆性**：呋喃的 D-A 反应可逆，是竞赛中常考的"陷阱"——高温下产物可能分解
4. **合成入口**：
   - "[[Paal-Knorr合成]]：1,4-二羰基化合物 + 酸/热 → 呋喃（脱水关环）"
   - 从糠醛（呋喃-2-甲醛）衍生：糠醛是工业上最重要的呋喃衍生物

## 五、与吡咯、噻吩的对比

| 性质 | 吡咯 | 呋喃 | 噻吩 |
|:---|:---|:---|:---|
| 杂原子 | N-H | O | S |
| 杂原子供电子能力 | 最强 | 中等（电负性大） | 较弱（3p-2p 重叠差） |
| 亲电取代活性 | 最高 | 高 | 较高 |
| 环稳定性 | 较低（易聚合） | 中等 | 最高 |
| 芳香性感受 | 最强 | 较弱（有双烯性质） | 强 |
| 典型副反应 | 聚合、氧化 | D-A 反应（可逆） | 较稳定 |
| D-A 反应 | 难（芳香性强） | **易，且可逆** | 难 |
| 工业来源 | 煤焦油（少量） | **糠醛（大量）** | 石油（少量） |

## 六、典型题型

1. **芳香性判断**：呋喃的 π 电子计数（氧提供 2 电子）
2. **反应位置预测**：亲电取代在 α-位
3. **D-A 产物预测**：呋喃与马来酸酐等亲双烯体的加成产物结构
4. **可逆性判断**：给定反应条件，判断 D-A 产物是否稳定
5. **合成路线设计**：1,4-二羰基底物设计 → Paal-Knorr 合成呋喃

### 7.2 常见误区

| 误区 | 正确认识 |
|:---|:---|
| "呋喃和吡咯一样不稳定" | 呋喃**比吡咯稳定**，不易聚合；D-A 是其特色反应 |
| "呋喃 D-A 不可逆" | 呋喃芳香性弱，D-A **可逆**；高温 Retro-D-A 是考点 |
| "呋喃氧有碱性" | 呋喃氧孤对参与芳香，**无碱性**；与醚的氧完全不同 |
| "五元杂环都只做亲电取代" | 呋喃还可做 **D-A 双烯体**，这是与吡咯/噻吩的重要区别 |
