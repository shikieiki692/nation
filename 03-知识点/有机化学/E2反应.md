# E2 反应

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[38-消除反应]]

## 一、定义

**E2 反应（Elimination, bimolecular）** 即**双分子消除反应**，是碱夺取 β-H 与离去基（LG）离去**协同进行**的一步反应：

$$
\ce{B^- + H-C_β-C_α-LG ->[协同] B-H + C_β=C_α + LG^-}
$$

- **速率方程**：$v = k[\text{底物}][\text{碱}]$（二级动力学，与底物和碱浓度均有关）
- **核心特征**：**一步协同**（无中间体），**anti-共平面立体要求**（H 与 LG 反式共平面）
- **区域选择性**：Zaitsev 或 Hofmann，取决于**碱的位阻**和**底物结构**

## 三、核心原理

### 3.1 协同过渡态

E2 的过渡态是一个 **"展开的四中心环"**：

```
      δ⁻      δ⁺
       B···H
        \   /
         C_β
        /   \
       C_α···LG
              δ⁻
```

- **C_β-H 键断裂** 与 **C_α-LG 键断裂** 同步进行
- **C_β=C_α π 键形成** 同步进行
- 负电荷分散在 B 和 LG 上（类似 SN2 的五配位过渡态，但无亲核进攻）

### 3.2 反式共平面要求（Anti-periplanar Requirement）

这是 E2 反应**最严格的立体化学要求**：

- **定义**：被夺取的 β-H 和离去基 LG 必须在 **Newman 投影中的反式（180°）二面角**
- **原因**：
  1. **轨道重叠**：断裂的 C-H σ 键电子需流入正在形成的 C=C π 键；反式共平面时 σ 键与 C-LG σ* 反键轨道完美重叠
  2. **构象最小改变**：anti-排列使过渡态中各基团的位阻最小

- **环己烷体系**：H 和 LG **必须都处于直立键（a,a）**。若 LG 在平伏键，则邻位直立 H 与 LG 为 gauche（60°），不满足 anti-要求 → **E2 被禁止**（或极慢）

**竞赛高频考点**：环己烷衍生物的 E2 消除速率受构象锁定效应强烈影响。

## 四、关键结论

1. **E2 是协同反应，无中间体** → **不重排**（与 E1 的本质区别）
2. **立体化学严格**：anti-共平面是硬性要求；syn-消除仅在特殊刚性体系（如环状锁定的季铵碱）中发生
3. **碱的位阻决定取向**：
   - 小位阻碱（OH⁻、EtO⁻）→ **Zaitsev 产物**（更取代的烯烃）
   - 大位阻碱（t-BuO⁻、LDA）→ **Hofmann 产物**（少取代的烯烃，因位阻小的 β-H 更易接近）
4. **与 SN2 竞争**：强亲核体/小位阻 → SN2；强碱/大位阻 → E2；高温 → E2 占优
5. **好 LG 加速 E2**：LG 越好，过渡态中 C-LG 键越易断裂
6. **环状体系中的反-Bredt 规则**：桥环化合物中，若双键桥头碳有 s 成分过高（Bridgehead 双键），则 E2 无法形成该烯烃

## 五、常见分类

### 5.1 按取向分类

| 类型 | 碱特征 | 产物 | 本质 |
|:---|:---|:---|:---|
| **Zaitsev E2** | 小位阻碱（NaOEt、KOH）| 更取代、更稳定的烯烃 | 过渡态中取代多的烯烃更稳定 |
| **Hofmann E2** | 大位阻碱（t-BuOK、LDA）| 少取代的烯烃 | 位阻小的 β-H 更易被大碱接近 |
| **Hofmann 消除（季铵碱）** | Me₃N 作为 LG | 末端烯烃 | 季铵碱的立体化学要求导致 |

### 5.2 按底物分类

```
E2 反应
├── 开链卤代烷
│   ├── 1° RX + 强碱 → 主要 E2（SN2 竞争）
│   ├── 2° RX + 强碱 → E2/SN2 竞争
│   └── 3° RX + 强碱 → 主要 E2（SN2 位阻过大）
├── 环己烷衍生物
│   ├── LG 在 a 键 + 邻位 a-H → 快速 E2
│   └── LG 在 e 键 → E2 被禁止（需翻转构象）
├── 季铵碱（Hofmann 消除）
│   └── R₄N⁺OH⁻ → 烯烃 + Me₃N
└── 砜/亚砜消除（Julia 烯化）
    └── 合成中的特殊 E2 变体
```

## 六、适用条件

### 6.1 促进 E2 的条件

| 因素 | 理想条件 | 原因 |
|:---|:---|:---|
| **底物** | 1°、2°、3° 均可；2° 最常见 | E2 不受碳正离子稳定性限制 |
| **碱** | 强碱（t-BuO⁻、NaNH₂、LDA）| 协同机理需强碱推动 C-H 断裂 |
| **溶剂** | 极性非质子（DMSO、DMF）或醇 | 稳定碱和 LG⁻；非质子溶剂促 E2 胜 SN2 |
| **温度** | 高温 | 消除的 ΔS > 0，TΔS 有利；也加速 SN2/E2 竞争中的 E2 |
| **离去基** | 好 LG（OTs > I > Br > Cl）| 降低协同过渡态能量 |

### 6.2 E2 vs SN2 的竞争调控

| 条件倾向 | E2 主导 | SN2 主导 |
|:---|:---|:---|
| **碱的特征** | 强碱、大位阻、弱亲核性 | 强亲核体、小位阻、弱碱性 |
| **温度** | 高温 | 低温 |
| **溶剂** | 极性非质子（促裸露碱）| 极性质子（溶剂化碱，降碱性）|
| **底物位阻** | 3°（SN2 被禁）| 1°（SN2 最快）|
| **底物 β-H 数** | β-H 多 | 无 β-H 或 β-H 位阻大 |

**ABOC 口诀**："大碱大温大位阻，消除；小碱低温小位阻，取代"

## 七、常见比较

### 7.1 E1 vs E2 vs E1cb 全对比

| 维度 | E1 | E2 | E1cb |
|:---|:---|:---|:---|
| **机理** | 分步（碳正离子）| 协同（一步）| 分步（碳负离子）|
| **动力学** | 一级 | 二级 | 一级（对底物）|
| **中间体** | 碳正离子 | 无 | 碳负离子 |
| **立体化学** | 无要求 | **anti-共平面** | 无严格要求 |
| **重排** | 常见 | **无** | 无 |
| **碱要求** | 弱碱 | 强碱 | 强碱 |
| **底物偏好** | 3° > 2° | 1°/2°/3° 均可 | β-吸电子基 |
| **RDS** | LG 离去 | 协同 | C-H 去质子 |
| **KIE** | 无 | 显著 | 显著 |

### 7.2 Zaitsev vs Hofmann 取向的定量理解

以 2-溴-2-甲基丁烷为例：

```
       Br                         Br
        |                          |
CH₃-C-CH₂-CH₃   +  Base  →   CH₃-C=CH-CH₃  (Zaitsev: 三取代)
        |                          |
       CH₃                       CH₃
        |
       CH₂=C-CH₂-CH₃  (Hofmann: 二取代，若用大位阻碱)
```

| 碱 | 位阻 | 主产物 | 原因 |
|:---|:---|:---|:---|
| EtO⁻ | 小 | Zaitsev（三取代）| 过渡态中三取代烯烃更稳定 |
| t-BuO⁻ | 大 | Hofmann（末端）| 只能接近位阻小的末端 CH₃ 的 H |

**竞赛技巧**：若题目给出"t-BuOK/DMSO"，第一反应是 **Hofmann 消除**。

## 十、例题

### 例题 1：环己烷衍生物的 E2 立体化学

**题目**：反-1-溴-4-叔丁基环己烷与 t-BuOK/t-BuOH 反应的主产物是什么？

**解析**：
1. 叔丁基（t-Bu）是**大位阻基团**，为最小化 1,3-双直立作用，t-Bu **锁定在平伏键（e）**
2. 在反式异构体中，Br 与 t-Bu 反式 → Br 处于 **直立键（a）**
3. Br（a）的邻位碳上，必须有一个 **直立 H（a）** 才能 anti-共平面 E2
4. 环己烷椅式中，C1-Br（a）的邻位是 C2 和 C6：
   - C2-H：若 Br 在 C1-a，则 C2 的 a-H 在 Br 的 anti 位置（二面角 180°）
   - C6-H：同理
5. 消除后得 **4-叔丁基环己烯**

**陷阱**：若底物是顺式异构体（Br 与 t-Bu 同侧），则 Br 在平伏键 → **无邻位 a-H** → **E2 极慢或不能发生**。

### 例题 2：Zaitsev vs Hofmann 条件判别

**题目**：2-溴-2-甲基丁烷分别在以下条件下消除，主产物是什么？
1. NaOEt/EtOH，60°C
2. t-BuOK/t-BuOH，60°C

**解析**：
1. **NaOEt（小位阻碱）→ Zaitsev 产物**：2-甲基-2-丁烯（三取代烯烃）
2. **t-BuOK（大位阻碱）→ Hofmann 产物**：2-甲基-1-丁烯（二取代烯烃）

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| 忽略 E2 的 anti-共平面要求 | 只考虑 β-H 的存在，不考虑空间排列 | 环己烷体系必须用椅式构象判断 a/e；开链体系用 Newman 投影 | 为什么顺-1-溴-4-叔丁基环己烷很难发生 E2？ |
| 所有 E2 都写 Zaitsev 产物 | 忽略碱的位阻效应 | 看到 **t-BuO⁻、LDA、季铵碱** → 第一反应 Hofmann | t-BuOK 和 NaOEt 对同一底物的产物为什么不同？ |
| 认为 3° 底物不能 E2 | 混淆 E2 与 SN2 的位阻要求 | E2 **可以**发生在 3° 底物（碱不进攻 α-碳，只夺 β-H）；3° 底物 E2 甚至比 SN2 更容易 | 为什么 (CH₃)₃CBr 与 t-BuOK 主要发生 E2 而不是 SN2？ |
| E2 过渡态画成碳负离子 | 混淆 E2 与 E1cb | E2 是**协同**，无碳负离子中间体；负电荷只在过渡态中分散 | E2 和 E1cb 的负电荷分别出现在哪？ |
| 忽略反-Bredt 规则 | 在桥环中强行画桥头双键 | Bredt 规则：桥环中若双键桥头碳的桥含 <8 个碳，则桥头双键极不稳定（或无法形成）| 降冰片烷衍生物的 E2 消除为什么总是外向双键？ |

## 十二、🎯 教学视角

### 12.1 学习路径

```
前置：SN2反应（协同机理）→ 反式共平面（立体电子效应）→ 消除反应总论
     ↓
本节：E2反应（协同 + anti-共平面 + 取向规则 + 与 SN2 竞争）
     ↓
后续分支：E1反应（分步对比）→ E1cb反应（碳负离子路径）→ 消除综合
```

**建议课时**：2 课时（协同机理与过渡态 0.5 + anti-共平面与构象 0.5 + 取向与竞争 0.5 + 环状体系专题 0.5）

### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "E2 和 SN2 条件完全一样" | 都是强亲核/强碱 | 区分 **亲核性**（SN2）和 **碱性**（E2）；大位阻碱亲核弱但碱性强 → E2 |
| "anti-共平面就是反式" | 混淆构象与构型 | 强调 anti-共平面是**构象要求**（二面角 180°），不是产物构型；产物可以是顺式或反式烯烃 |
| "Zaitsev 产物一定更稳定" | 忽略动力学控制 | 大位阻碱下，Hofmann 产物虽热力学不稳定，但动力学上更易形成 |
| "所有消除都有重排" | 把 E1 的特征推广到所有消除 | 明确：**只有 E1 有中间体 → 可能重排；E2 和 E1cb 无中间体 → 不重排** |

### 12.3 入门例题

**最简 E2 判别题**：
> 叔丁基氯与 NaOH/H₂O 反应，主要产物是什么？与 t-BuOK/t-BuOH 的产物有何不同？

**答案**：
- NaOH/H₂O：SN2 难以发生（3° 位阻），E2 碱不够强 → 主要是 **E1**（若高温）或 SN1/E1 混合
- t-BuOK/t-BuOH：强碱 + 大位阻 + 非质子 → **E2**，产物为 **异丁烯（Hofmann？不，叔丁基只有 9 个等价 H，无区域选择问题）**

**修正简化题**：
> 2-溴丁烷与 NaOEt/EtOH 在 60°C 反应，主产物是什么？

**答案**：**反-2-丁烯**（Zaitsev 产物，更稳定的反式二取代烯烃）
**引导**：如果换成 t-BuOK，产物变为 **1-丁烯**（Hofmann）。

### 12.4 现实类比

**「拔河」类比 E2 反应**：
- 碱（B⁻）= 一方队员，拉 β-H
- 离去基（LG）= 另一方队员，拉 α-C
- C_β-H 键和 C_α-LG 键 = 两根绳子
- 协同过渡态 = 两根绳子**同时断裂**，中间的横杆（C_β-C_α）被拉紧形成新的连接（π 键）
- **anti-共平面** = 两根绳子必须在**相反方向**（180°）才能同时发力；如果成 60° 角（gauche），两边队员互相干扰，无法同时使力

## 十三、竞赛拓展

### 13.1 竞赛中的特殊 E2 反应

| 反应 | 特点 | 竞赛价值 |
|:---|:---|:---|
| **Hofmann 消除（季铵碱）** | R₄N⁺OH⁻ → 烯烃 + Me₃N | 构建末端烯烃；测定胺结构（Hofmann 降解）|
| **Cope 消除** | 叔胺氧化物热解 → syn 消除 | **例外**：E2 通常是 anti，但 Cope 是 **syn**（环状 TS）|
| **Julia 烯化** | 砜 + 碱 → E2 消除得烯烃 | 合成中立体选择性构建反式烯烃 |
| **Ramberg-Bäcklund** | α-卤砜 + 碱 → 环状中间体 → 消除得烯烃 | 构建张力烯烃 |
| **Bamford-Stevens 反应** | 对甲苯磺酰腙 + 碱 → 卡宾或烯烃 | 烯烃合成 |

### 13.2 与立体电子效应的交叉

- **gauche 效应**：某些含杂原子底物中，gauche 构象比 anti 更稳定 → 可能影响 E2 的构象要求
- **A₁,₃ 张力**：环己烷中 1,3-双直立作用迫使某些底物采取特定构象 → 锁定或禁止某些 E2 路径
