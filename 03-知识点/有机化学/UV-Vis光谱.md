# UV-Vis 光谱（紫外-可见光谱）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[30-有机波谱分析]]

---

## 一、定义

**UV-Vis 光谱（紫外-可见光谱）**：利用分子对紫外光（200–400 nm）和可见光（400–800 nm）的吸收，测定有机分子中**电子跃迁**特征的波谱技术。主要用于判断**共轭体系的大小**和**发色团的类型**。

**本质**：**共轭体系的尺子**——共轭越长，吸收波长越长（红移）。

## 三、核心原理

### 3.1 电子跃迁类型

| 跃迁 | 能量 | 波长范围 | 摩尔吸光系数 ε | 典型化合物 |
|:---|:---:|:---:|:---:|:---|
| **σ→σ*** | 最高 | < 150 nm | 强 | 烷烃（远紫外）|
| **n→σ*** | 较高 | 150–250 nm | 弱 | 醇、胺、卤代烃 |
| **π→π*** | 中等 | 160–250 nm（孤立）| 强 | 烯烃、芳环 |
| **n→π*** | 最低 | 270–300 nm | 弱 | 羰基、亚硝基 |

**竞赛重点**：π→π* 和 n→π* 跃迁

### 3.2 λmax 与共轭的关系

**核心规律**：
- **共轭链增长** → π 轨道能级差减小 → **λmax 红移**（向长波方向移动）
- **共轭环增大** → 同样红移

**经验规律**：

| 化合物 | λmax（nm）| 共轭特征 |
|:---|:---:|:---|
| 乙烯 | 171 | 孤立 C=C |
| 1,3-丁二烯 | 217 | 共轭二烯 |
| 1,3,5-己三烯 | 258 | 共轭三烯 |
| 苯 | 255 | 芳香六元环 |
| 萘 | 275 | 稠环芳烃 |
| β-胡萝卜素 | 450 | 多烯长链（11 个双键）|

### 3.3 Woodward-Fieser 规则

**用于预测共轭二烯/三烯的 λmax**

**基准值**：
- 异环二烯（双烯在不同环）：214 nm
- 同环二烯（双烯在同一环）：253 nm
- 无环/开链二烯：217 nm

**增量**：

| 结构特征 | 增量（nm）|
|:---|:---:|
| 延伸双键 | +30 |
| 环外双键 | +5 |
| 烷基取代 | +5（每个）|
| –OCOCH₃（酰氧基）| +0 |
| –OR（烷氧基）| +6 |
| –SR（烷硫基）| +30 |
| –Cl / –Br | +5 |
| –NR₂（二烷氨基）| +60 |

**竞赛应用**：
- 验证结构：计算 λmax 与实测值对比
- 注意：Woodward-Fieser 规则**只适用于共轭二烯/三烯**，不适用于芳环

### 3.4 助色团与发色团

| 类型 | 定义 | 例子 | 效应 |
|:---|:---|:---|:---|
| **发色团**（Chromophore）| 产生吸收的基团 | C=C、C=O、N=N、芳环 | 本身有吸收 |
| **助色团**（Auxochrome）| 本身不吸收，但使发色团吸收位移 | –OH、–NH₂、–Cl、–OR | 红移或增色 |

### 3.5 溶剂效应

| 跃迁类型 | 极性溶剂增加 | 原因 |
|:---|:---|:---|
| **π→π*** | 红移 | 激发态极性更大，被溶剂稳定 |
| **n→π*** | 蓝移 | n 轨道被溶剂氢键稳定，能级降低 |

---

## 四、关键结论

1. **UV-Vis 看共轭，不看官能团**——不能确定具体是什么官能团，只能判断共轭体系大小
2. **共轭越长，λmax 越大**——是 UV-Vis 最核心的定性规律
3. **Woodward-Fieser 规则是验证工具**——用于验证结构，不是从头预测
4. **n→π* 弱吸收，π→π* 强吸收**——羰基化合物的 n→π* 峰通常很弱（ε < 100）
5. **溶剂极性影响**：π→π* 红移，n→π* 蓝移

---

## 五、常见分类或情形

### 5.1 按化合物类型分类

| 化合物类型 | 典型 λmax | 特征 |
|:---|:---:|:---|
| **孤立烯烃** | 170–190 nm | 远紫外，常规仪器可能测不到 |
| **共轭二烯** | 217–250 nm | Woodward-Fieser 规则适用 |
| **芳烃** | 250–280 nm | 精细结构（振动边带）|
| **α,β-不饱和羰基** | 310–350 nm | n→π* 弱吸收 |
| **多烯/类胡萝卜素** | > 400 nm | 进入可见区，有色 |

### 5.2 互变异构体的 UV 差异

| 异构体 | λmax | 原因 |
|:---|:---:|:---|
| **酮式** | 短 | 无共轭或共轭短 |
| **烯醇式** | 长 | C=C–OH 共轭 |

- 乙酰丙酮：酮式 λmax ≈ 270 nm；烯醇式 λmax ≈ 310 nm
- UV-Vis 可用于**判断互变异构平衡**

---

## 六、适用条件与限制

1. **仅反映共轭信息**：UV-Vis 不能区分构造异构体（如邻/间/对二取代苯的 λmax 相近）
2. **定量需要已知 ε**：A = εcl，定量分析需要已知摩尔吸光系数
3. **溶剂截止波长**：溶剂在远紫外区有吸收，限制了可测范围
4. **不能提供氢数信息**：UV-Vis 不提供分子中氢原子数目的信息

---

## 七、常见比较与易混点

### 7.1 UV-Vis vs NMR vs MS

| | **UV-Vis** | **NMR** | **MS** |
|:---|:---|:---|:---|
| **信息类型** | 共轭体系大小 | 氢环境 + 数量 | 分子量 + 碎片 |
| **精度** | 低（nm 级）| 中（δ ppm）| 高（精确质量）|
| **破坏样品** | 否 | 否 | 是 |
| **竞赛用途** | 验证共轭 | 确定结构 | 确定分子式 |

### 7.2 红移 vs 蓝移

| | **红移（Bathochromic）**| **蓝移（Hypsochromic）**|
|:---|:---|:---|
| **方向** | 向长波移动 | 向短波移动 |
| **π→π*** | 共轭增加、极性溶剂 | 共轭减小 |
| **n→π*** | 极性溶剂减小（蓝移）| 氢键给体溶剂 |

## 十、例题

### 10.1 例题 1（★★ 基础）

**题目**：某化合物在 230 nm 有强吸收（ε > 10000），在 320 nm 有弱吸收（ε ≈ 50）。推断可能的官能团。

**分析**：
- 230 nm 强吸收 → π→π* 跃迁 → 共轭体系
- 320 nm 弱吸收 → n→π* 跃迁 → 羰基或类似基团
- 两者同时存在 → **α,β-不饱和羰基化合物**

**解答**：
- 可能为 **α,β-不饱和酮/醛/酯**
- π→π*（C=C–C=O 共轭）≈ 230 nm
- n→π*（羰基）≈ 320 nm

**反思**：
- 强吸收 = π→π*，弱吸收 = n→π*
- α,β-不饱和羰基是 UV-Vis 中最典型的双吸收体系

### 10.2 例题 2（★★★ 进阶）

**题目**：用 Woodward-Fieser 规则计算以下化合物的 λmax：一个同环二烯，带一个延伸双键、两个烷基取代、一个环外双键。

**解答**：
- 同环二烯基准值：253 nm
- 延伸双键：+30 nm
- 两个烷基：+5 × 2 = +10 nm
- 环外双键：+5 nm
- **计算 λmax = 253 + 30 + 10 + 5 = 298 nm**

**反思**：
- 记住基准值：同环 253 / 异环 214 / 无环 217
- 不要漏加环外双键和烷基取代

## 十一、易错点

1. ❌ **UV-Vis 用来确定官能团** → UV-Vis 只反映**共轭体系大小**，不能确定具体官能团（IR 和 NMR 才能）
2. ❌ **Woodward-Fieser 规则适用于所有化合物** → 只适用于**共轭二烯/三烯**
3. ❌ **把 n→π* 的弱吸收忽略** → n→π* 虽然弱（ε < 100），但位置特征（270–300 nm）
4. ❌ **溶剂效应记反** → π→π* 红移，n→π* 蓝移
5. ❌ **孤立双键在常规 UV 区有强吸收** → 孤立 C=C 的 π→π* 在 ~170 nm，远紫外区，常规仪器测不到

### 12.2 学生易踩的认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析口诀/方法 |
|:---|:---|:---|:---|
| "UV 和 IR 一样，都能确定官能团" | 都是光谱 | UV 只能看**共轭**，IR 才能看**官能团** | "UV 看共轭，IR 看官能团，NMR 看结构" |
| "λmax 越大就是分子越大" | 以为波长和分子量相关 | λmax 只与共轭长度相关，与分子量无关 | "共轭长，波长长；分子大，不一定" |
| "Woodward-Fieser 规则万能" | 记了一个规则到处用 | 只适用于**共轭二烯/三烯** | "二烯三烯才适用，芳环不行" |
| "n→π* 峰强所以重要" | 以为强峰才重要 | n→π* 是**弱峰**（ε < 100），但位置特征 | "n→π* 弱但特征" |

### 12.3 入门级例题

**题目**：1,3-丁二烯和 1,4-戊二烯，哪个的 UV λmax 更大？为什么？

**预期解答路径**：
1. 1,3-丁二烯：共轭二烯，π 电子离域 → λmax ≈ 217 nm
2. 1,4-戊二烯：孤立双键，无共轭 → λmax ≈ 170–190 nm（孤立烯烃）
3. 结论：1,3-丁二烯的 λmax 更大

**老师可在课堂上引导的提问**：
- 学生若答错：问 "共轭是什么意思？1,3-丁二烯的双键之间隔了几个单键？1,4-戊二烯呢？"

### 12.4 与现实/直觉的连接

- **类比**：UV-Vis 就像"彩虹滤镜"——共轭体系短，只吸收紫外（人眼看不见）；共轭体系长，吸收进入可见区（人眼看见颜色）。β-胡萝卜素（11 个双键）吸收蓝光 → 显橙黄色
- **反例**：表面上看，苯比乙烯"大"，但苯的 λmax（255 nm）并不比共轭三烯（258 nm）大很多——说明芳环的"有效共轭长度"约等于三烯

## 十三、竞赛拓展

1. **荧光光谱**：激发光谱（类似 UV-Vis）和发射光谱的组合，用于高灵敏度检测。
2. **圆二色谱（CD）**：用于测定手性化合物的绝对构型，超纲但属于高级波谱技术。
3. **瞬态吸收光谱**：用于研究光化学反应中的短寿命中间体，前沿技术。
