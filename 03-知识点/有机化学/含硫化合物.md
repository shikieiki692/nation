# 含硫化合物（Organosulfur Compounds）

## 一、定义

**含硫化合物**是指有机分子中含有碳-硫键（C-S）或硫-杂原子键的化合物。硫与氧同属第 16 族（氧族），但 S 的原子半径更大、电负性更低、3p 轨道扩散性更强，导致含硫化合物的性质与含氧化合物既有相似性又有显著差异。

- **硫的氧化态范围**：从 −2（硫醇、硫醚）到 +6（硫酸酯），比氧更广
- **与含氧化合物的关键差异**：
  - S 可扩展八隅体（利用 3d 轨道参与）→ 可形成高价硫化合物（亚砜、砜、锍盐）
  - S-H/S-C 键比 O-H/O-C 键更弱 → 硫醇酸性比醇强，硫醚比醚更亲核
  - S 的可极化性远大于 O → 硫醚是软亲核体

## 三、核心原理

### 3.1 硫的电子结构特点

| 特征 | 硫（S）| 氧（O）| 影响 |
|:---|:---:|:---:|:---|
| **原子半径** | 105 pm | 73 pm | S 更易极化，轨道重叠差 |
| **电负性** | 2.58 | 3.44 | C-S 键极性弱于 C-O |
| **键能（C-X）** | ~272 kJ/mol | ~358 kJ/mol | C-S 键更易断裂 |
| **孤对电子** | 3p（扩散）| 2p（紧凑）| S 的孤对亲核性更强、更软 |
| **扩展八隅体** | 可以（3d）| 不可以 | 可形成 4/6 价硫 |

### 3.2 含硫化合物的分类与氧化态

| 氧化态 | 化合物类型 | 结构 | 例子 |
|:---:|:---|:---|:---|
| **−2** | 硫醇（Thiol）| R-SH | 半胱氨酸、辅酶 A |
| **−2** | 硫醚（Thioether/Sulfide）| R-S-R' | 甲硫氨酸、芥子气 |
| **−1** | 二硫化物（Disulfide）| R-S-S-R' | 胱氨酸、橡胶硫化 |
| **−1** | 次磺酸（Sulfenic acid）| R-SOH | 不稳定，酶中间体 |
| **0** | 亚砜（Sulfoxide）| R-S(=O)-R' | DMSO |
| **0** | 硫代亚磺酸酯 | R-S(=O)-S-R' | 大蒜素 |
| **+2** | 砜（Sulfone）| R-S(=O)₂-R' | 环丁砜、糖精 |
| **+2** | 锍盐（Sulfonium salt）| R₃S⁺ X⁻ | S-腺苷甲硫氨酸（SAM）|
| **+4** | 亚磺酸（Sulfinic acid）| R-SO₂H | 抗氧化剂 |
| **+6** | 磺酸（Sulfonic acid）| R-SO₃H | 对甲苯磺酸（TsOH）|

## 四、关键结论

1. **硫醇的酸性比醇强得多**：pKa ~10-11（硫醇）vs ~16-18（醇），因为 S-H 键更弱且 RS⁻ 的电荷更分散
2. **硫醚是优良的软亲核体和烷基化试剂**：可形成锍盐，进而生成硫叶立德
3. **DMSO 是 Swern 氧化的核心试剂**：DMSO 被活化后与醇反应生成烷氧基硫鎓盐，碱促进消除得醛/酮
4. **硫缩醛/硫缩酮是羰基保护的重要形式**：比缩醛更耐酸，需 Hg²⁺ 脱除
5. **Corey-Chaykovsky 反应是锍叶立德的经典应用**：硫叶立德 + 羰基 → 环氧化物（区别于 Wittig 的烯烃）

## 五、常见分类

### 5.1 硫醇（R-SH）

| 性质/反应 | 说明 |
|:---|:---|
| **酸性** | pKa ~10-11，比醇强 10⁶ 倍；可溶于稀碱 |
| **氧化** | 温和氧化（I₂, H₂O₂）→ 二硫化物 R-S-S-R |
| **强氧化** | KMnO₄, HNO₃ → 磺酸 R-SO₃H |
| **与重金属** | 与 Hg²⁺, Pb²⁺, Cu⁺ 形成不溶性盐 |
| **亲核性** | RS⁻ 是强亲核体、软碱；SN2 中比 RO⁻ 更快 |
| **共轭效应** | +M 弱于 OR（3p-2p 重叠差）|

### 5.2 硫醚（R-S-R'）

| 性质/反应 | 说明 |
|:---|:---|
| **亲核性** | S 是软亲核体，亲核性远强于醚氧 |
| **烷基化** | 与卤代烃反应 → 锍盐 R₃S⁺ X⁻ |
| **氧化** | H₂O₂ 或 mCPBA → 亚砜 → 进一步氧化 → 砜 |
| **脱硫** | Raney Ni（含氢）→ R-H + R'-H（脱硫氢解）|
| **Pummerer 重排** | 亚砜 + 酸酐 → α-酰氧基硫醚 |

### 5.3 亚砜与砜

| 性质 | 亚砜（R₂S=O）| 砜（R₂SO₂）|
|:---|:---|:---|
| **S 构型** | 金字塔形，可手性（如光学活性亚砜）| 四面体 |
| **碱性** | S=O 的 O 有弱碱性 | O 碱性更弱 |
| **α-H 酸性** | 中等 | 强（砜基的 −I/−M）|
| **典型应用** | DMSO（溶剂、Swern 氧化）| Julia 烯烃合成 |

### 5.4 锍叶立德与 Corey-Chaykovsky 反应

**锍叶立德**：R₂S⁺—C⁻R'₂

**制备**：锍盐 + 强碱（如 NaH, NaNH₂, n-BuLi）

**Corey-Chaykovsky 反应**：
$$\ce{R2S^+ - CHR'2 + R''COR''' -> R2S=O + 环氧化物}$$

**特点**：
- 与 Wittig 反应不同——**生成环氧化物而非烯烃**
- 驱动力：形成稳定的 S=O 键
- 非稳定化叶立德：环氧化；稳定化叶立德：也可生成烯烃（消除路径竞争）

### 5.5 重要人名反应

| 反应 | 底物 | 产物 | 试剂 | 说明 |
|:---|:---|:---|:---|:---|
| **Swern 氧化** | 醇 | 醛/酮 | (COCl)₂, DMSO, Et₃N | DMSO 为硫试剂 |
| **Corey-Chaykovsky** | 醛/酮 + 锍叶立德 | 环氧化物 | 锍盐 + 强碱 | 硫叶立德构建三元环 |
| **Julia 烯烃合成** | 砜 + 醛 | 烯烃（E-为主）| 砜负离子 + 醛 → 砜中间体 → 还原消除 | 立体选择性构建 E-烯烃 |
| **Pummerer 重排** | 亚砜 | α-酰氧基硫醚 | 酸酐（如 Ac₂O）| 亚砜的酰化重排 |
| **Lawesson 试剂** | 羰基化合物 | 硫代羰基 | Lawesson 试剂（二硫代磷杂环戊烷）| 酮 → 硫酮，酰胺 → 硫代酰胺 |

## 六、适用条件

### 6.1 Swern 氧化的条件

| 因素 | 条件 | 说明 |
|:---|:---|:---|
| **第一步** | DMSO + (COCl)₂, −78°C | 形成氯代硫鎓盐 |
| **第二步** | 加底物醇，−78°C | 形成烷氧基硫鎓盐 |
| **第三步** | 加 Et₃N（大过量），−78°C → rt | 碱促进消除 |
| **溶剂** | CH₂Cl₂ | 标准溶剂 |
| **注意** | 严格无水、通风 | 产生 CO、CO₂ |

### 6.2 硫醚氧化为亚砜/砜的条件

| 产物 | 试剂 | 条件 |
|:---|:---|:---|
| **亚砜（停在亚砜）** | 1 当量 H₂O₂ 或 NaIO₄ | 室温，控制当量 |
| **砜（彻底氧化）** | 2 当量 H₂O₂ 或 mCPBA 过量 | 加热 |
| **手性亚砜** | 手性过酸（如 Davis 试剂）| 不对称氧化 |

## 七、常见比较

### 7.1 含硫 vs 含氧化合物

| 维度 | 硫醇/硫醚 | 醇/醚 |
|:---|:---|:---|
| **酸性** | 强（pKa ~10-11）| 弱（pKa ~16-18）|
| **亲核性** | 强（软亲核）| 弱（硬亲核）|
| **氧化** | 易氧化（S 可扩展八隅体）| 难氧化（O 不能扩展）|
| **烷基化** | 易成锍盐 | 难成氧鎓盐 |
| **保护基** | 硫缩醛（耐酸，需 Hg²⁺ 脱）| 缩醛（酸脱）|
| **共轭效应** | +M 弱（3p-2p 差）| +M 强 |

### 7.2 硫叶立德 vs 磷叶立德

| 维度 | 硫叶立德 | 磷叶立德 |
|:---|:---|:---|
| **结构** | R₂S⁺—C⁻R'₂ | R₃P⁺—C⁻R'₂ |
| **与醛/酮反应** | **环氧化物** | **烯烃** |
| **驱动力** | S=O 形成 | P=O 形成 |
| **立体化学** | 受硫取代基影响 | 受磷取代基和稳定化影响 |
| **典型应用** | Corey-Chaykovsky | Wittig |

## 十、例题

### 例题 1：硫的氧化态

**题目**：判断下列化合物中硫的氧化态：H₂S, CH₃SH, CH₃SCH₃, CH₃S(O)CH₃, CH₃SO₂CH₃, H₂SO₄。

**解析**：

| 化合物 | S 氧化态 | 说明 |
|:---|:---:|:---|
| H₂S | −2 | 两个 H-S 键 |
| CH₃SH | −2 | 同硫醇 |
| CH₃SCH₃ | −2 | 硫醚 |
| CH₃S(O)CH₃ | 0 | 亚砜，S=O 中 O 得 2e |
| CH₃SO₂CH₃ | +2 | 砜，两个 S=O |
| H₂SO₄ | +6 | 四个 S=O 等价 |

### 例题 2：Corey-Chaykovsky 反应

**题目**：二甲基锍叶立德与环己酮反应的产物是什么？与 Wittig 反应有何不同？

**解析**：

**产物**：**螺环氧化物**（spiro-epoxide）——环己酮的羰基碳与叶立德的 CH₂ 形成环氧三元环

**与 Wittig 的区别**：
- **硫叶立德** → **环氧化物**（三元环，S=O 为副产物）
- **磷叶立德** → **烯烃**（双键，Ph₃P=O 为副产物）

**原因**：硫比氧大，S-C 键更容易断裂/重组，倾向于形成三元环；而磷叶立德的 P-C 键极性不同，倾向于形成烯烃。

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| "硫醇酸性比醇弱" | 同族从上到下酸性增强的误解 | 硫醇 pKa ~10，醇 pKa ~16 → **硫醇酸性强得多** | 为什么硫醇比醇酸性强？ |
| "硫叶立德和磷叶立德产物相同" | 混淆两类叶立德 | 硫叶立德 → **环氧**；磷叶立德 → **烯烃** | 硫叶立德和磷叶立德与醛反应的产物有何不同？ |
| "亚砜不可手性" | 忽略 S 的金字塔构型 | 亚砜的 S 是 **sp³ 金字塔**（三个取代基 + 孤对），可存在对映体 | DMSO 有手性吗？为什么某些亚砜有光学活性？ |
| "硫醚氧化只到亚砜" | 忽略过量氧化 | 控制条件 → 亚砜；过量/加热 → **砜** | 如何控制硫醚氧化只到亚砜而不继续到砜？ |

### 12.3 入门例题

**最简判断练习**：
> 比较：1. CH₃S⁻ 和 CH₃O⁻ 的亲核性；2. CH₃SH 和 CH₃OH 的酸性。

**答案**：
1. **CH₃S⁻ 亲核性更强**（S 更软、可极化性大）
2. **CH₃SH 酸性更强**（S-H 键更弱，RS⁻ 更稳定）
**引导**：问"为什么硫和氧的性质差异这么大？"（原子半径、电负性、可扩展八隅体）
