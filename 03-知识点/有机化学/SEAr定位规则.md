# SEAr 定位规则（SEAr Orientation Rules）
- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]

## 一、定义

**SEAr 定位规则**：芳香亲电取代反应中，苯环上已有取代基决定新取代基进入位置的规律。

**本质**：定位规则是 **σ 络合物（Wheland 中间体）稳定性** 的直接反映——新取代基进入能使中间体正电荷最稳定分散的位置。

## 三、核心原理

### 3.1 σ 络合物（Wheland 中间体）

**SEAr 机理两步**：
```
步骤 1：E⁺ 进攻芳环 → σ 络合物（正电荷在环上）
步骤 2：失去 H⁺ → 恢复芳香性
```

**σ 络合物的共振式**：
- 进攻邻位 → 正电荷可分布在 1, 2, 4 位
- 进攻对位 → 正电荷可分布在 1, 2, 4 位（与邻位类似）
- 进攻间位 → 正电荷可分布在 1, 3, 5 位

### 3.2 定位规则分类

| 取代基类型 | 例子 | 定位 | 活化/钝化 | 本质 |
|:---|:---|:---:|:---:|:---|
| **给电子共轭（+C > -I）** | -OH, -OR, -NH₂, -NHR, -NR₂ | 邻/对位 | **强活化** | 孤对电子稳定 σ 络合物正电荷 |
| **给电子诱导（+I）** | -R, -Ar | 邻/对位 | **弱活化** | 超共轭稳定正电荷 |
| **卤素（-I > +C）** | -F, -Cl, -Br, -I | 邻/对位 | **弱钝化** | 诱导吸电子钝化，但孤对电子可稳定邻/对位中间体 |
| **吸电子共轭（-C）** | -NO₂, -CN, -CHO, -COOR | 间位 | **强钝化** | 正电荷无法被稳定，间位中间体相对最稳定 |
| **吸电子诱导（-I）** | -NR₃⁺, -SO₃H | 间位 | **强钝化** | 正电荷被 destabilize |

### 3.3 为什么给电子基是邻/对位？

**以苯酚为例**：
- OH 的孤对电子可与 σ 络合物的正电荷共轭
- 当 E⁺ 进攻**邻位或对位**时，其中一个共振式的正电荷在连有 OH 的碳上
- OH 的孤对电子可以**直接稳定这个正电荷**
- 进攻**间位**时，正电荷无法被 OH 直接稳定

### 3.4 为什么吸电子基是间位？

**以硝基苯为例**：
- NO₂ 是强吸电子基，正电荷在其邻/对位会被进一步 destabilize
- 当 E⁺ 进攻**间位**时，正电荷离 NO₂ 最远
- 因此间位 σ 络合物**相对最稳定**（虽然整体仍比苯难反应）

### 3.5 多取代苯的定位策略

**规则**：
1. **两个取代基定位一致** → 进入共同指向的位置
2. **两个取代基定位冲突** → 由**更强活化基**主导
3. **空间位阻大** → 邻位产物减少

**合成设计示例**：
- 目标：对硝基苯甲醚
- 策略：先上 OCH₃（强活化，邻/对位定位），再硝化 → 主要得对位产物
- 错误策略：先硝化，再上 OCH₃ → 硝基是间位定位，OCH₃ 进入间位（相对于硝基），产物错误

### 3.6 定位效应"最强致活基团"权重规则（Zchem 系统化工具）

**核心规则**：多取代苯发生亲电取代时，**由最强致活基团（most strongly activating group）决定**主要产物位置。

**致活/致钝能力排序**：

| 强度 | 基团类型 | 典型基团 | 定位 | 机理 |
|:---|:---|:---|:---:|:---|
| 极强致活 | 含孤对电子的杂原子 | -NH₂, -NHR, -NR₂, -OH, -OR | 邻对位 | p-π 共轭给电子 |
| 中等致活 | 烷基 | -R, -Ar | 邻对位 | 超共轭给电子 |
| 弱致钝 | 卤素 | -F, -Cl, -Br, -I | 邻对位 | 诱导吸电子 + 共振给电子（净致钝）|
| 中等致钝 | 吸电子基（无孤对电子）| -COCH₃, -CHO, -COOR | 间位 | π 键极化吸电子 |
| 强致钝 | 强吸电子基 | -NO₂, -CN, -SO₃H, -NR₃⁺ | 间位 | 强诱导 + 共振吸电子 |

**决策流程**：

```
苯环上有多个取代基
    ↓
Step 1：识别每个基团的类型（致活/致钝 + 定位）
    ↓
Step 2：找出最强致活基团
    ↓
Step 3：确定该基团的邻/对位
    ↓
Step 4：考虑位阻效应（大基团邻位位阻大 → 对位优势）
    ↓
Step 5：确定主产物
```

**典型案例**：

| 取代基组合 | 最强致活基团 | 主要产物位置 | 原因 |
|:---|:---|:---:|:---|
| -CH₃ + -NO₂ | -CH₃（弱致活）| 邻对位（相对甲基）| 硝基是致钝基，效应被覆盖 |
| -OH + -CH₃ | -OH（极强致活）| 邻对位（相对羟基）| 羟基致活能力 >> 甲基 |
| -Br + -CH₃ | -CH₃（弱致活）>-Br（弱致钝）| 邻对位（相对甲基）| 甲基虽弱但仍是致活，溴是净致钝 |
| -NHCOCH₃ + -COCH₃ | -NHCOCH₃（中等致活）| 邻对位（相对酰胺基）| 乙酰氨基的孤对电子共轭给电子 |

**位阻修正**：当最强致活基团的邻位有大位阻基团时，对位产物比例增加。

## 四、关键结论

1. **定位规则的本质是 σ 络合物稳定性**——不是死记，而是画共振式
2. **给电子基 → 邻/对位**（活化中间体）
3. **吸电子基 → 间位**（避免 destabilize 中间体）
4. **卤素是特殊案例**——钝化但邻/对位定位（因为孤对电子可稳定中间体）
5. **多取代合成中，取代顺序决定成败**
6. **F-C 烷基化可重排、可逆、多取代**——F-C 酰基化更可控

## 五、常见分类或情形

### 5.1 常见定位基一览

| 定位基 | 类型 | 活化/钝化 |
|:---|:---|:---:|
| -NH₂, -NHR, -NR₂, -OH, -OR | 邻/对位 | 强活化 |
| -R, -Ar | 邻/对位 | 弱活化 |
| -F, -Cl, -Br, -I | 邻/对位 | 弱钝化 |
| -NO₂, -CN, -SO₃H, -COOH, -CHO, -COOR | 间位 | 强钝化 |
| -NR₃⁺ | 间位 | 强钝化 |

### 5.2 多取代苯的合成策略

| 目标产物 | 正确顺序 | 错误顺序 |
|:---|:---|:---|
| 对硝基甲苯 | 先甲基化 → 再硝化 | 先硝化 → 再甲基化 |
| 间溴硝基苯 | 先硝化 → 再溴化 | 先溴化 → 再硝化 |
| 邻氯苯酚 | 先氯化 → 再水解（特殊）| — |

## 六、适用条件与限制

1. **Friedel-Crafts 反应限制**：强钝化基（-NO₂, -CN）存在时，F-C 不反应
2. **空间位阻影响**：大位阻基团（如 t-Bu）会阻碍邻位取代
3. **温度影响**：高温下动力学控制 → 更稳定的产物（热力学控制）
4. **磺化可逆**——可用于占位-脱保护策略

---

## 七、常见比较与易混点

### 7.1 F-C 烷基化 vs 酰基化

| | **F-C 烷基化** | **F-C 酰基化** |
|:---|:---|:---|
| **中间体** | 碳正离子 | 酰基正离子（较稳定）|
| **重排** | **会重排** | **不重排** |
| **可逆性** | **可逆** | **不可逆** |
| **多取代** | 容易多取代 | 单取代为主 |
| **合成价值** | 较低 | 较高 |

### 7.2 活化 vs 定位

- 活化/钝化：决定反应**快慢**
- 定位：决定产物**位置**
- 卤素是"钝化但邻/对位定位"的典型矛盾统一体

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：甲苯硝化时，主要产物是什么？

**解答**：
- -CH₃ 是邻/对位定位基，弱活化
- 硝化主要产物：**邻硝基甲苯 + 对硝基甲苯**
- 比例约 2:1（邻位稍多，但空间位阻使对位也不少）

### 10.2 例题 2（★★ 合成设计）

**题目**：如何从苯合成间硝基苯甲酸？

**解答**：
1. 先 **F-C 酰基化**（CH₃COCl/AlCl₃）→ 苯乙酮
2. **氧化**侧链（KMnO₄）→ 苯甲酸
3. **硝化**（HNO₃/H₂SO₄）→ 间硝基苯甲酸
- 原因：-COOH 是间位定位基

### 10.3 例题 3（★★★ 竞争）

**题目**：对甲基苯酚溴化时，溴进入哪个位置？

**解答**：
- -OH（强活化，邻/对位）vs -CH₃（弱活化，邻/对位）
- 两者都指向**邻位（相对于 OH）**
- OH 的活化能力远强于 CH₃，因此 OH 主导定位
- 产物：**2-溴-4-甲基苯酚**（溴在 OH 的邻位，CH₃ 的对位）

## 十一、易错点

1. ❌ **死记"给电子基邻对位，吸电子基间位"而不理解机理** → 必须画 σ 络合物共振式
2. ❌ **忽略卤素的特殊性** → 卤素是**钝化但邻/对位定位**
3. ❌ **F-C 烷基化不重排** → F-C 烷基化**会重排**
4. ❌ **多取代时随意排序** → 取代顺序**决定成败**
5. ❌ **认为磺化不可逆** → 磺化是**可逆的**（可用于占位策略）
6. ❌ **忽略空间位阻** → t-Bu 等大基团会阻碍邻位取代

### 12.3 入门例题

**题目**：硝基苯再进行硝化，新硝基进入哪个位置？

**引导**：
- -NO₂ 是间位定位基，强钝化
- 画 σ 络合物的三个共振式
- 间位进攻时，正电荷离 -NO₂ 最远 → 相对最稳定
- 答案：**间二硝基苯**

### 12.4 现实类比

**SEAr 定位像"排队选座位"**：
- 给电子基 = 友善的人（活化）→ 新来的人喜欢坐在他旁边（邻/对位）
- 吸电子基 = 不友善的人（钝化）→ 新来的人尽量远离他（间位）
- 卤素 = 外表冷淡但内心友善 → 新人还是愿意坐旁边，但整体队伍气氛不热烈（钝化）
