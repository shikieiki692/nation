# 铜锂试剂（Gilman 试剂）

## 一、定义

**铜锂试剂（Gilman Reagent）** 通式为 **R₂CuLi**，是由两当量有机锂试剂（RLi）与一当量碘化亚铜（CuI）在低温下反应制备的有机金属化合物。

$$
\ce{2 RLi + CuI ->[THF, -40°C 至 0°C] R2CuLi + LiI}
$$

- **本质**：铜锂试剂中的 R 基团以 **"软亲核体"** 形式存在
- **与 RLi/RMgX 的核心区别**：铜锂试剂的亲核性 **更软、更弱**，碱性 **更弱**
- **结构特点**：在溶液中以 **二聚体 [R₂CuLi]₂** 形式存在，含 Cu-Cu 相互作用
- **热稳定性差**：通常在低温下制备并立即使用

## 三、核心原理

### 3.1 制备与结构

**制备条件**：
- **温度**：-40°C 至 0°C（CuI 与 RLi 的反应放热）
- **溶剂**：THF 或 Et₂O（THF 更优，稳定试剂）
- **CuI 质量**：需高纯度 CuI（商业 CuI 常含 Cu(II)，需先用 Me₂S 或 Bu₃P 处理还原）
- **计量**：严格 2 RLi : 1 CuI

**结构特征**：
- R₂CuLi 不是简单的离子对；Cu 处于 **+1 氧化态**
- 溶液中主要存在形式为 **二聚体 [R₂CuLi]₂**，含平面四配位的 Cu 中心
- 比 RLi 更稳定（对空气/水仍敏感，但稍好于 RLi）

### 3.2 反应性根源：软硬酸碱（HSAB）

| 试剂 | 软硬酸碱分类 | 对饱和羰基（硬）| 对 α,β-不饱和羰基 |
|:---|:---|:---|:---|
| **RLi/RMgX** | **硬**亲核体 | 1,2-加成（快）| 1,2-加成（动力学）|
| **R₂CuLi** | **软**亲核体 | 几乎不反应 | **1,4-加成**（热力学）|

**解释**：
- α,β-不饱和羰基有两个亲电位点：
  - **羰基碳（C=O）**：硬中心（高电荷密度、低极化性）
  - **β-碳（Cβ）**：软中心（低电荷密度、高极化性、与 π* 轨道相关）
- 硬亲核体（RLi）进攻硬中心 → 1,2-加成
- 软亲核体（R₂CuLi）进攻软中心 → **1,4-加成（共轭加成 / Michael 加成）**

## 四、关键结论

1. **铜锂试剂是实现 1,4-加成的首选试剂**：将 α,β-不饱和羰基转化为 β-烷基取代的饱和羰基
2. **与酰氯反应得酮（不继续加成）**：R₂CuLi + R'COCl → RCOR' + RCu + LiCl。这是合成酮的独特方法
3. **与环氧反应开环**：进攻位阻较小的碳，SN2 机理
4. **与卤代烃偶联（Corey-House 反应）**：R₂CuLi + R'X → R-R' + RCu + LiX。适用于 1°、烯丙基、苄基卤代烃
5. **不能直接制备**：必须由 RLi + CuI 原位制备；无法像格氏试剂一样商品化储存
6. **催化量 Cu(I) 可转化格氏试剂的加成模式**：RMgX + 催化量 CuI/LiCl → 原位生成 RCu，实现 1,4-加成

## 五、常见分类

### 5.1 主要反应

| 反应类型 | 底物 | 产物 | 机理 | 备注 |
|:---|:---|:---|:---|:---|
| **1,4-加成（共轭加成）** | α,β-不饱和醛/酮/酯/腈/酰胺 | β-烷基饱和羰基 | 软亲核进攻 β-碳 | 铜锂试剂**最经典应用** |
| **酰氯 → 酮** | R'COCl | RCOR' | 亲核加成-消除 | **不继续加成**（酮对 R₂CuLi 惰性）|
| **环氧开环** | 环氧乙烷/取代环氧 | 醇 | SN2（位阻小端）| 比 RLi 温和 |
| **Corey-House 偶联** | R'X（1°/烯丙基/苄基）| R-R' | 氧化加成-还原消除 | 构建 C-C 键 |
| **1,2-加成（特殊）** | 醛（某些条件下）| 2° 醇 | 直接加成 | 通常需 TMSCl 活化 |

### 5.2 1,4-加成的底物范围

```
铜锂试剂 1,4-加成
├── α,β-不饱和醛
│   └── 得 β-烷基醛（需控制，避免过度还原）
├── α,β-不饱和酮
│   └── 得 β-烷基酮（最常用）
├── α,β-不饱和酯
│   └── 得 β-烷基酯
├── α,β-不饱和腈
│   └── 得 β-烷基腈
├── α,β-不饱和酰胺
│   └── 得 β-烷基酰胺
├── 烯酮（R₂C=C=O）
│   └── 得甲基酮
└── 硝基烯烃
    └── 得 β-烷基硝基烷（Michael 受体）
```

## 六、适用条件

### 6.1 制备条件

| 因素 | 条件 | 原因 |
|:---|:---|:---|
| **温度** | -40°C 至 0°C | 防止 RLi 与 CuI 过度反应或分解 |
| **溶剂** | 无水 THF（最优）或 Et₂O | 稳定 R₂CuLi；醚类溶剂配位 Cu⁺ |
| **CuI 来源** | 高纯度 CuI 或 CuI·SMe₂ 络合物 | 避免 Cu(II) 杂质氧化 RLi |
| **RLi 选择** | 市售 n-BuLi、s-BuLi、MeLi、t-BuLi | 根据目标 R 基团选择 |
| **操作** | 低温下将 CuI 悬浮液加入 RLi 溶液 | 控制放热；反向加入可能导致副反应 |

### 6.2 1,4-加成的最佳条件

| 因素 | 条件 | 原因 |
|:---|:---|:---|
| **底物** | α,β-不饱和羰基（EWG 在 β-位增强反应性）| 软亲核体偏好低电荷密度中心 |
| **温度** | -78°C → rt | 低温启动，升温完成 |
| **溶剂** | THF 或 Et₂O | 非配位性过强会导致试剂分解 |
| **添加剂** | TMSCl 或 HMPA（小心，致癌）| 捕获烯醇盐、加速反应 |
| **后处理** | 饱和 NH₄Cl（淬灭）| 温和质子化 |

### 6.3 催化量 Cu(I) 法（更实用的替代方案）

在许多合成中，不制备纯 R₂CuLi，而是 **原位催化**：

$$
\ce{RMgX + 催化量 CuI/LiCl + α,β-不饱和酮 -> 1,4-加成产物}
$$

- **催化量**：5-10 mol% CuI + 10-20 mol% LiCl
- **优势**：无需制备和计量 R₂CuLi；格氏试剂直接转化为铜试剂中间体
- **机理**：RMgX 与 Cu(I) 发生转金属化 → RCu 或 R₂CuMgX（"ate" 络合物）→ 1,4-加成

## 七、常见比较

### 7.1 铜锂试剂 vs 格氏/有机锂

| 维度 | R₂CuLi | RLi/RMgX |
|:---|:---|:---|
| **亲核性硬度** | **软** | **硬** |
| **碱性** | 弱 | 强 |
| **与饱和酮** | 不反应 | 1,2-加成 |
| **与 α,β-不饱和酮** | **1,4-加成** | 1,2-加成 |
| **与酰氯** | 得 **酮**（不继续加成）| 得 3° 醇（继续加成）|
| **与卤代烃** | **偶联**（Corey-House）| 一般不反应 |
| **与环氧** | 缓慢开环 | 快速开环 |
| **制备难度** | 中（需低温制备）| 格氏较易，RLi 较难 |

### 7.2 1,2-加成 vs 1,4-加成的选择性控制

| 目标加成模式 | 试剂选择 | 条件 |
|:---|:---|:---|
| **1,2-加成** | RLi 或 RMgX | 低温，醚溶剂 |
| **1,4-加成** | R₂CuLi 或 RMgX + CuI(cat) | THF，可升温 |
| **1,2- then 1,4-** | 先 RLi（1,2-），再加 R₂CuLi（1,4-）| 分步加入 |
| **1,4- then 捕获** | R₂CuLi + TMSCl | TMSCl 捕获烯醇盐，防止副反应 |

**竞赛策略题**：若要将甲基同时加到 α,β-不饱和酮的羰基碳和 β-碳，如何设计？
**答案**：无法实现（同一碳骨架上同时 1,2- 和 1,4- 加同一基团无意义）。但可 **先 1,4- 加成（Me₂CuLi），再羰基还原（NaBH₄）**，或 **先保护羰基，再 1,4- 加成**。

## 十、例题

### 例题 1：1,4-加成产物预测

**题目**：写出 (CH₃)₂CuLi 与 3-丁烯-2-酮（甲基乙烯基酮）反应、酸性后处理后的产物。

**解析**：
1. (CH₃)₂CuLi 是软亲核体，进攻 α,β-不饱和酮的 **β-碳**
2. π 电子流向羰基氧，形成 **烯醇盐中间体**
3. 酸性后处理（NH₄Cl 或 H₃O⁺）质子化烯醇盐
4. 产物：**4-甲基-2-丁酮**（CH₃COCH₂CH₂CH₃）

**对比**：若用 CH₃Li，则进攻羰基碳（1,2-加成），产物为 CH₃C(OH)(CH₃)CH=CH₂（烯丙醇）。

### 例题 2：酰氯与铜锂试剂反应

**题目**：(CH₃)₂CuLi 与苯甲酰氯（PhCOCl）反应的产物是什么？与 PhCOCl + MeLi 的产物有何不同？

**解析**：
1. **(CH₃)₂CuLi + PhCOCl**：
   - 亲核加成-消除机理
   - Me 进攻羰基碳 → 四面体中间体 → Cl⁻ 离去
   - 产物：**苯乙酮（PhCOCH₃）**
   - 苯乙酮对 R₂CuLi 惰性（软亲核体不与饱和酮反应）→ **反应停在这里**

2. **MeLi + PhCOCl**：
   - 第一当量 → 苯乙酮（同上）
   - 第二当量 → 苯乙酮 + MeLi → **1,1-二甲基-1-苯基甲醇（3° 醇）**
   - 最终产物是 3° 醇，不是酮

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| 认为铜锂试剂与饱和酮反应 | 混淆亲核性 | R₂CuLi **不与饱和醛/酮反应**（太软，不匹配硬中心）| 铜锂试剂与丙酮混合会怎样？ |
| 1,4-加成后产物画错 | 忘记烯醇盐质子化 | 1,4-加成后得 **烯醇盐**，需酸性后处理得饱和羰基 | 1,4-加成后的中间体是什么？如何转化为最终产物？ |
| 与酰氯反应继续加成 | 与 RLi/RMgX 混淆 | R₂CuLi + 酰氯 → **酮**（不继续加成），这是合成酮的独特方法 | 为什么铜锂试剂与酰氯反应停在酮，而格氏试剂不停？ |
| 忽略 Corey-House 偶联的底物限制 | 认为所有卤代烃都能偶联 |  Corey-House 仅适用于 **1°、烯丙基、苄基**卤代烃；2°/3° 产率低 | 为什么 2-溴丙烷与 (CH₃)₂CuLi 偶联效果差？ |
| 催化量 Cu(I) 法与原位制备混淆 | 概念不清 | 催化量 Cu(I) 法 = 格氏试剂 + 少量 CuI → 原位生成铜试剂；纯 R₂CuLi = 2 RLi + CuI | 两种方法各有什么优缺点？ |

## 十二、🎯 教学视角

### 12.1 学习路径

```
前置：有机锂试剂 → 软硬酸碱（HSAB）→ 亲核加成
     ↓
本节：铜锂试剂（软亲核特性 + 1,4-加成 + 酰氯→酮 + 偶联）
     ↓
后续分支：Michael加成 → Robinson环化 → 全合成中的共轭加成策略
```

**建议课时**：1.5 课时（制备与 HSAB 原理 0.5 + 1,4-加成与应用 0.5 + 酰氯/偶联 0.5）

### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "铜锂试剂比有机锂弱，所以不好用" | 只看亲核性强弱 | 强调 "弱" 正是其**选择性来源**——专打 1,4-加成，不打 1,2-加成 |
| "1,4-加成就是 Michael 加成" | 混淆概念 | Michael 加成是 **1,4-加成的一种**（特定亲核体 + α,β-不饱和羰基）；1,4-加成还包括铜锂、烯胺等 |
| "铜锂试剂可以自己买" | 不知其不稳定性 | R₂CuLi **不能商品化储存**；必须实验室原位制备 |
| "催化量 CuI 法不如纯 R₂CuLi" | 认为纯度低效果不好 | 实际上催化量法在合成中**更常用**，因为操作简便、安全 |

### 12.3 入门例题

**最简产物预测**：
> 写出 (CH₃)₂CuLi 与下列底物反应、酸性后处理后的产物：
> 1. 环己烯酮  2. 乙酰氯  3. 丙酮

**答案**：
1. **3-甲基环己酮**（1,4-加成，Me 加到 β-碳）
2. **丙酮**（CH₃COCH₃，酰氯 → 酮）
3. **不反应**（饱和酮对铜锂试剂惰性）
**引导**：若学生第 3 题答错，问"丙酮是 α,β-不饱和酮吗？铜锂试剂的软亲核性匹配什么类型的中心？"

### 12.4 现实类比

**"外科医生"类比铜锂试剂**：
- 有机锂/格氏试剂 = 大力士（硬碰硬，见什么都打）
- 铜锂试剂 = 精准的外科医生（软刀子，只切特定部位）
- **1,4-加成** = 外科医生绕过坚固的城墙（羰基碳 = 硬中心），直接攻击防守薄弱的侧门（β-碳 = 软中心）
- **酰氯 → 酮** = 医生完成手术后立即停止（不继续切割），因为酮对软亲核体是 "免疫的"
- **Corey-House 偶联** = 两位医生握手合作（R₂CuLi + R'X → R-R'），把两个分子精确缝合

## 十三、竞赛拓展

### 13.1 竞赛中的铜锂试剂变体

| 变体/反应 | 特点 | 竞赛价值 |
|:---|:---|:---|
| **高阶铜试剂（R₂Cu(CN)Li₂）** | 氰基稳定铜，反应性更温和 | 敏感底物的 1,4-加成 |
| **铜催化共轭加成** | 催化量 Cu(I) + RMgX/RLi/ZnR₂ | 不对称共轭加成（手性配体控制）|
| **S_N2' 反应（烯丙基取代）** | R₂CuLi + 烯丙基卤/环氧 → γ-取代产物 | 烯丙位官能团化 |
| **共轭加成-捕获串联** | 1,4-加成后原位捕获烯醇盐（如与亲电体反应）| 构建复杂骨架 |
| **铜催化的环丙烷化** | 铜卡宾对烯烃的环丙烷化 | 构建三元环 |

### 13.2 与现代方法学的交叉

- **不对称共轭加成**：Cu(I)/手性双膦配体催化 RMgX 对硝基烯烃/烯酮的不对称 1,4-加成（ee 可达 99%）
- **有机铜化学的机理研究**：低温 NMR 和质谱可直接观测 R₂CuLi 的二聚体结构和反应中间体
