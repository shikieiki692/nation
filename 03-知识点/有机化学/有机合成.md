# 有机合成

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[50-有机合成]], [[51-金属有机]]
- 关联笔记：[[有机合成]], [[羧酸衍生物]], [[亲核取代]], [[羰基α位反应]]

## 一、定义

有机合成是指利用有机化学反应，从简单的起始原料出发，通过一系列官能团转化和碳骨架构建，制备目标有机化合物的过程。竞赛中的有机合成题通常要求设计合理的合成路线，考查对反应类型、选择性控制、官能团保护和逆合成分析（retrosynthesis）的掌握。

## 二、考纲对应

- 对应考纲条目：[[50-有机合成]]、[[51-金属有机]]
- 所属模块：[[基础要求-有机化学]]
- 本知识点在考纲中的作用：有机合成是**有机化学知识的综合应用**，几乎覆盖全部有机反应类型。竞赛中要求能从指定原料出发，利用逆合成分析设计 3~8 步的合理路线。这部分是区分"理解反应"与"运用反应"的分水岭。
- 考纲核心要求：

  - 能控制反应的化学选择性、区域选择性与立体选择性
  - 了解金属有机试剂（格氏试剂、有机锂、有机铜等）在合成中的应用

## 三、核心原理

### 2.1 逆合成分析（Retrosynthesis）

**基本思想**：从目标分子（Target, TGT）出发，通过逻辑上的"切断"（disconnection），将其分解为更简单的前体分子，直到得到可购买的起始原料。

**常用切断策略**：
- **C-C 键切断**：根据已知反应类型的成键位置进行切断
  - 醇 → 格氏试剂 + 羰基化合物
  - α,β-不饱和羰基 → Aldol 缩合切断
  - 1,3-二羰基 → Claisen 缩合切断
- **官能团转化（FGI）**：将目标官能团转化为更容易引入的前体官能团
- **官能团添加（FGA）**：在分子中引入辅助官能团以便切断

### 2.2 常见合成子与等价试剂

| 合成子 | 类型 | 等价试剂 |
|--------|------|---------|
| R⁺ | 亲电性 | R-X、烯烃 |
| R⁻ | 亲核性 | R-MgBr、R-Li、R₂CuLi |
| acyl⁺ | 亲电性 | RCOCl、(RCO)₂O |
| enolate⁻ | 亲核性 | 烯醇盐、烯醇硅醚 |
| a¹（α-给体） | 双官能团 | 1,3-二羰基化合物 |
| d²（β-受体） | 双官能团 | α,β-不饱和羰基 |

### 2.3 选择性控制

1. **化学选择性**：优先与某一官能团反应而不影响其他官能团
   - 醛 > 酮 > 酯 > 酰胺 的还原活性差异
   - 使用不同活性的试剂（如 NaBH₄ vs LiAlH₄）

2. **区域选择性**：在多个可能位置中选择特定位置反应
   - 不对称酮的烯醇化（动力学 vs 热力学控制）
   - 芳香取代的定位规则

3. **立体选择性**：控制产物的手性和构型
   - 顺式/反式加成（如硼氢化-氧化为反马氏顺式加成）
   - 手性辅助基团、手性催化剂

### 2.4 多取代苯合成逆推四步法（Zchem 高阶工具）

**四步法**：

```
目标分子
    ↓
【Step 1】识别目标分子中的官能团和取代模式
    ↓
【Step 2】判断取代基引入顺序（利用定位效应）
         ├─ 先引入邻对位定位基 → 再引入新基团到邻/对位
         └─ 先引入间位定位基 → 再引入新基团到间位
    ↓
【Step 3】检查是否需要占位基/保护基/氧化还原切换
         ├─ 占位基：-SO₃H（引入后稀酸水解除去）
         ├─ 保护基：-NH₂ → -NHCOCH₃（保护氨基不被氧化）
         └─ 氧化切换：-CH₃ → -COOH（将邻对位定位改为间位定位）
    ↓
【Step 4】检查傅克反应是否会发生重排
         ├─ 傅克烷基化：碳正离子中间体 → 可能重排
         └─ 若会重排 → 改用傅克酰基化 + Clemmensen/Wolff-Kishner 还原
    ↓
写出正向合成路线（以苯为起始原料）
```

**经典案例：对位硝基正丙基苯的合成**

```
错误路线：
苯 + CH₃CH₂CH₂Br/AlCl₃ → 正丙基苯（❌ 实际得到异丙基苯，因碳正离子重排）
正丙基苯 + HNO₃/H₂SO₄ → 邻/对位硝基正丙基苯（比例不理想）

正确路线：
苯 + CH₃CH₂COCl/AlCl₃ → 苯丙酮（傅克酰基化，无重排）
苯丙酮 + HNO₃/H₂SO₄ → 对硝基苯丙酮（羰基是间位定位基，硝基进入相对羰基的间位 = 相对烷基的对位）
对硝基苯丙酮 + Zn-Hg/HCl (Clemmensen) → 对硝基正丙基苯 ✅
```

**占位基策略案例：制备 1,3-二溴-5-硝基苯**

```
目标：1,3-二溴-5-硝基苯
    ↓
分析：硝基是间位定位基，溴是邻对位定位基
    ↓
策略：先引入占位基 -SO₃H 控制位置

苯 + 浓 H₂SO₄ → 苯磺酸
苯磺酸 + Br₂/FeBr₃ → 对溴苯磺酸（-SO₃H 是间位定位基，Br 进入间位 = 对位）
对溴苯磺酸 + Br₂/FeBr₃ → 3,5-二溴苯磺酸（-SO₃H 导向间位，-Br 导向邻对位，两者在 3,5 位重合）
3,5-二溴苯磺酸 + 稀 H₂SO₄/△ → 1,3-二溴苯（水解去除 -SO₃H）
1,3-二溴苯 + HNO₃/H₂SO₄ → 1,3-二溴-5-硝基苯 ✅（两个 -Br 共同导向 5 位）
```

### 2.4 官能团保护策略

| 保护基 | 保护对象 | 脱保护条件 |
|--------|---------|-----------|
| 缩醛/缩酮 | 醛/酮 | 稀酸水解 |
| TMS/THP | 醇 | TBAF / 稀酸 |
| 酯/酰胺 | 羧酸/胺 | 碱水解 / 催化氢解 |
| 苄基醚 | 醇 | H₂/Pd |
| Boc/Cbz | 胺 | TFA / H₂/Pd |

**保护原则**：
- 保护基在后续步骤中稳定
- 脱保护条件不破坏分子其他部分
- 上保护和脱保护的产率高、操作简便

## 四、关键结论

1. **逆合成分析是核心方法论**：碳骨架切断（disconnection）决定合成路线的骨架，官能团转化（FGI）填充细节。"好的切断是成功合成的一半。"

2. **汇聚式合成优于线性合成**：在分子对称/接近对称处切断，可最大化汇聚度（convergency），将 $N$ 步线性路线的产率问题转化为 $\log N$ 层汇聚路线的优势。总产率 = 最长支路各步产率之积，而非所有步数。

3. **保护基 = 代价**：每增加一个保护基，路线增加 2 步（上保护 + 脱保护）。评估合成路线时须将保护基开销计入总步数。无保护基合成（protecting-group-free synthesis）是重要的优化维度。

4. **氧化还原经济性（Redox Economy）**：理想的合成路线应尽量减少不必要的氧化态升降——例如避免"将醇氧化为酮再还原回去"的无意义循环。酮的 $\alpha$-烷基化直接通过烯醇盐完成，无需氧化-还原绕路。

5. **手性源的优先级**：手性合成优先考虑 (1) 底物控制（已有手性中心诱导），(2) 手性辅基，(3) 手性催化剂。天然手性源（氨基酸、糖类、萜类）提供廉价的手性池（chiral pool），是"chiron 策略"的基础。

6. **合成路线评估四要素**：**简明性**（步数最少）、**选择性**（每步产率与区域/立体控制）、**原料可得性**（是否易购/廉价）、**可放大性**（试剂成本与安全）。竞赛中前两项为核心考核点。

## 五、常见分类或情形

### 5.1 按路线拓扑分类

| 类型 | 示意图 | 总产率 | 特点 |
|:---|:---|:---|:---|
| **线性合成** | $\ce{A -> B -> C -> D -> TGT}$ | $Y^n$（$n$ 步，每步产率 $Y$） | 简单直接，但步数多时总产率急剧下降 |
| **汇聚式合成** | A→C↘ + B→D↗ → TGT | 减少步数依赖，总产率更高 | 适合对称/近似对称的分子 |
| **分支汇聚** | A→B→C↘ + D→E↗ → F → TGT | 多片段汇聚 | 复杂天然产物全合成的标准策略 |

### 5.2 按合成策略分类

- **目标导向合成（TOS, Target-Oriented Synthesis）**：围绕单一结构明确的分子设计路线。竞赛合成题的主流类型。
- **多样性导向合成（DOS, Diversity-Oriented Synthesis）**：从共同中间体出发，经不同分支反应生成结构多样的化合物库。竞赛中偶见于"利用同一原料合成多种产物"题型。
- **仿生合成（Biomimetic Synthesis）**：模仿生源途径设计合成，常用阳离子环化串联反应（如仿生甾体合成）。

### 5.3 按反应级联程度分类

- **逐步合成**：每步反应后分离纯化，再进行下一步。路线清晰但操作繁琐。
- **一锅法合成（One-Pot Synthesis）**：多步反应连续进行，中间不分离。对反应兼容性要求极高。
- **串联/级联反应（Tandem/Cascade Reaction）**：设计一个引发事件触发连续多步转化。高效的"多米诺"策略。

### 5.4 按保护基使用策略分类

- **全程保护策略**：对每个不相容官能团使用保护基 → 路线长但可靠性高
- **无保护基合成（PGFree Synthesis）**：利用底物自身反应性差异避免保护 → 步数精简但对选择性要求极高（参考 Baran 实验室风格）
- **正交保护策略**：使用可在不同条件下独立脱除的保护基组合（如 Boc（酸脱）/Cbz（氢解脱）/Fmoc（碱脱）），实现精准的官能团释放顺序

## 六、适用条件与限制

1. **逆合成分析的适用前提**：需要掌握足够丰富的可靠反应——切断必须对应已知的、高产率的正向反应。对反应库不够熟悉的学生，逆分析会陷入"切得开、合不拢"的困境。

2. **"步数陷阱"**：理论上最简单的切断路线可能在实际操作中因保护基开销、官能团不相容等原因变得冗长。逆合成分析需要**反复迭代**，尝试多条切断路线后综合比较。

3. **多官能团分子的兼容性问题**：当目标分子含多种互不相容的官能团时，某些"最优切断"在实际上不可行——例如同时含有胺基和羰基的分子，简单的 Aldol 切断可能因胺的亲核性干扰副反应而失败。

4. **选择性不可控的限制**：某些切断在纸面上合理，但在实践中因区域选择性（如不对称酮的烯醇化方向）、立体选择性（如 Aldol 的 syn/anti 选择）无法控制，需改用其他策略。

5. **工业可行性约束**：竞赛中虽不重点考核，但需了解——某些试剂在实验室可行但工业上禁用（如剧毒的 $\ce{OsO4}$、昂贵的 Pd 催化剂大量使用），以及低温（$-78\ ^{\circ}\mathrm{C}$）、无水无氧操作等放大限制。

6. **保护基的隐性成本**：每引入一个保护基 = +2 步 + 两次纯化产率损失 + 额外的正交性约束。过度依赖保护基是初学者合成路线的通病。

7. **适用范围的底线**：逆合成分析主要适用于**多步 C–C 键构建**和**复杂官能团编排**。对于简单的单官能团转化（如"醇→醛"），直接写正向路线即可，无需逆分析。

## 七、常见比较与易混点

### 7.1 逆合成 vs 正向合成

| | 逆合成分析（Retrosynthesis） | 正向合成（Synthesis） |
|:---|:---|:---|
| **思维方向** | 目标 $\to$ 原料（倒退） | 原料 $\to$ 目标（前进） |
| **核心操作** | 切断（disconnection） | 成键（bond formation） |
| **符号** | $\Rightarrow$（逆合成箭头） | $\to$（反应箭头） |
| **用途** | 路线设计阶段 | 实验操作与路线表述 |
| **回答格式** | 画出切断树 | 写出反应条件和中间体 |

### 7.2 合成子（Synthon） vs 等价试剂（Reagent）

| | 合成子 | 等价试剂 |
|:---|:---|:---|
| **本质** | 理想化的带电片段（概念工具） | 实际可购买的化合物（实物） |
| **表示法** | R$^+$, R$^-$, acyl$^+$, enolate$^-$ | R-Br, R-MgBr, RCOCl |
| **关系** | 一个合成子可对应多种等价试剂 | 选择取决于稳定性与反应性 |
| **易混点** | 初学者常把"合成子是切断结果"混淆为"合成子是试剂" | 合成子 = 逻辑构造，等价试剂 = 化学实现 |

### 7.3 Umpolung（极性反转） vs 正常极性

- **正常极性**：羰基碳为亲电中心（$\delta^+$），$\alpha$-位为亲核中心（经烯醇化）；这是绝大多数羰基反应的基础
- **极性反转（Umpolung）**：通过特定策略使羰基碳变为亲核性的 $\ce{C=O -> C(OH)-R}$（如使用氰化物、二噻烷、NHC 催化），从而与另一亲电试剂反应
- **典型例子**：苯甲醛 $\ce{PhCHO}$ → 苯偶姻（benzoin）缩合（CN$^-$ 催化使羰基碳 umpolung 为亲核碳）
- **易错点**：Umpolung 不是"反应机理不对"，而是**合成子极性的反转策略**——使原本亲电的碳表现为亲核

### 7.4 策略性切断 vs 战术性转化

| | 策略性决策（Strategic） | 战术性决策（Tactical） |
|:---|:---|:---|
| **关注层级** | 碳骨架的切断位置与顺序 | 具体反应条件与试剂选择 |
| **影响范围** | 决定路线的整体框架和步数 | 影响单步产率和选择性 |
| **例子** | "在 C3-C4 之间切断还是 C5-C6 之间" | "该步 Aldol 用 LDA 还是 NaOEt" |
| **优先顺序** | 先定策略，再选战术 | 不应以战术便利性倒推策略选择 |

### 7.5 保护基策略 vs 选择性控制策略

- **保护基路线**：将干扰官能团"屏蔽" → 安全但步数增加（每保护一处 +2 步）
- **选择性控制路线**：利用反应自身的选择性差异（如醛 vs 酮的反应活性差）避免保护 → 精简但有风险
- **判断标准**：若选择性控制可靠（产率 > 85%，单一产物），优先不保护；否则老老实实上保护基

## 九、典型题型
1. **给定目标分子设计合成路线**：从指定原料出发，设计 3~5 步合成
2. **补全合成路线**：给出部分步骤，填写中间体和反应条件
3. **判断合成路线缺陷**：分析给定路线的选择性错误、保护缺失等问题

## 十、例题

### 10.1 例题 1（★★ 基础：芳香合成中的定位策略）

**题目**：以苯为起始原料，合成对硝基苯乙酮 $\ce{p-O2N-C6H4-COCH3}$。

**分析（逆合成树）**：
```
目标：p-O2N-C6H4-COCH3
          ⇒（C–C 切断：Friedel-Crafts 酰基化）
    p-O2N-C6H4-H + CH3COCl
          ↑
    硝基苯的 Friedel-Crafts 酰基化？
    └─ 硝基是强间位定位基 + 钝化基团 → FC 反应困难！
          ⇒（调整策略：先酰基化，再硝化）
    C6H5-COCH3（苯乙酮）
          ⇒（FC 酰基化）
    苯 + CH3COCl/AlCl3
```

**解答（正向路线）**：
1. 苯 $\ce{+ CH3COCl/AlCl3}$ → 苯乙酮（Friedel-Crafts 酰基化，乙酰基为间位定位基）
2. 苯乙酮 $\ce{+ HNO3/H2SO4}$ → 对硝基苯乙酮（硝化反应，硝基进入乙酰基的间位 = 相对乙酰基的对位 ✓）

共 2 步，策略正确。

**反思**：
- 核心教训：**先判断定位效应再排顺序**。若先硝化后酰化，硝基钝化芳环导致 Friedel-Crafts 失败，且硝基为间位定位基会使乙酰基进入间位 = 产物为间硝基苯乙酮（错误）。
- 这是芳香合成中"活化基团先上，钝化基团后上"原则的经典体现。

### 10.2 例题 2（★★★ 中等：保护基与选择性）

**题目**：以环己酮为起始原料，合成 4-羟基环己酮 $\ce{HO-C6H10(=O)}$（对位关系）。

**分析**：
```
目标：4-羟基环己酮
          ⇒（官能团转化：氧化二级醇为酮？——但需要先有对位羟基）
    1,4-环己二醇（直接氧化没有选择性——两个羟基都会氧化）
          ⇒（需要：在酮存在下选择性引入对位羟基）
    → 先保护羰基 → 在对位引入羟基 → 再释放羰基
```

**解答（正向路线）**：
1. 环己酮 $\ce{+ HOCH2CH2OH/H+}$ → 环己酮缩乙二醇（保护羰基为缩酮）
2. 环己酮缩乙二醇 $\ce{+ NBS/h\nu}$（自由基溴化）→ 4-溴环己酮缩乙二醇
3. 4-溴环己酮缩乙二醇 $\ce{+ NaOH/H2O}$ → 4-羟基环己酮缩乙二醇（SN2 水解）
4. 4-羟基环己酮缩乙二醇 $\ce{+ H3O+}$ → **4-羟基环己酮**（稀酸脱保护）

共 4 步（2 步保护/脱保护 + 2 步官能团化）。

**反思**：
- 保护基在此处必不可少：直接对环己酮做自由基溴化会因羰基 $\alpha$-位的竞争反应导致选择性差。
- 缩酮保护基的选择：在自由基溴化（中性条件）和碱性水解中稳定，最后用稀酸脱除——正交性好。
- 评估路线时注意保护-脱保护的开销（+2 步）是否值得。本例中值得，因为没有更好的替代策略。

### 10.3 例题 3（★★★★ 综合：汇聚式对称切断）

**题目**：以不超过 4 个碳的有机物为原料，合成 2,6-二甲基-4-庚酮 $\ce{(CH3)2CH-CH2-CO-CH2-CH(CH3)2}$。

**分析（逆合成树）**：
```
目标：(CH3)2CH-CH2-CO-CH2-CH(CH3)2
          ⇒（对称分子！在羰基两侧切断 → 汇聚式）
    2 × (CH3)2CH-CH2-MgBr + (EtO)2CO
          ↑
    两侧片段完全相同 → 只需制备一种格氏试剂，用两次
          ⇒（格氏试剂制备）
    (CH3)2CH-CH2-Br ← (CH3)2CH-CH2-OH ← 异丁醇（市售或合成）
```

**解答（正向路线）**：
1. 异丁醇 $\ce{(CH3)2CHCH2OH}$ + $\ce{PBr3}$ → 溴代异丁烷 $\ce{(CH3)2CHCH2Br}$
2. 溴代异丁烷 + $\ce{Mg/Et2O}$ → 异丁基溴化镁 $\ce{(CH3)2CHCH2MgBr}$
3. 2 当量异丁基溴化镁 + $\ce{(EtO)2CO}$（碳酸二乙酯）→ 水解后得 **2,6-二甲基-4-庚酮**

共 3 步。利用汇聚策略，仅制备单一片段即可。

**反思**：
- 识别分子的**对称性**是逆合成分析最有效率的切入点。对称的酮应在羰基两侧同时切断。
- 碳酸酯 + 2 当量格氏试剂 → 对称酮是经典汇聚式酮合成法。若使用酯 $\ce{RCOOEt}$ + 格氏试剂则只能得到叔醇（格氏试剂对酯加成两次）。
- 对称切断将路线从 5~6 步（分别从两侧不同路线构建）锐减为 3 步（同一片段用两次）——这就是汇聚式合成的威力。

## 十一、易错点

1. ❌ **逆分析时只考虑"能切断"而不考虑"选择性"** → 在有多个相似官能团的位置切断时，正向反应可能无选择性。解决：先通过官能团转化制造反应性差异。

2. ❌ **忽略官能团相容性** → 初学者常设计出"让格氏试剂与含活泼氢的官能团（-OH, -NH$_2$, -COOH）共存"的路线。记住：**强碱/强亲核试剂不容许酸性氢存在**。

3. ❌ **保护基的"过度使用"与"完全不用"两个极端** → 不加取舍地给每个官能团上保护，导致 10+ 步的冗余路线；或完全拒绝保护基，导致大量副反应。正确做法：按需保护，能用选择性控制代替则不保护。

4. ❌ **只关注 C–C 键切断，忽略氧化还原层面的规划** → 合成路线中氧化态应与各步反应匹配。例如把一级醇写成直接与格氏试剂偶联的前体，需先氧化为醛/酮。

5. ❌ **认为"逆合成分析 = 写着玩，正向路线才是答案"** → 竞赛阅卷同时关注逆分析的逻辑和正向路线的准确性。推荐**双栏呈现**：左侧逆合成树，右侧正向路线。

6. ❌ **忽略芳香合成中的定位效应顺序** → 多取代苯的合成必须按定位效应严格排序引入顺序，否则得到错误的区域异构体。错误顺序不仅走不通，且无法通过"增加步数"修复。

7. ❌ **混淆合成子极性与实际反应机理** → 合成子是逻辑工具，不代表真实的反应中间体。例如 enolate$^-$ 合成子由羰基 $\alpha$-去质子化产生，实际反应中不以游离阴离子存在。

8. ❌ **忽略试剂当量和反应条件** → 只写"加格氏试剂"而不指定当量（如酯需要 $\ge$ 2 当量生成叔醇），这在阅卷中会被扣分。

### 12.2 学生易踩的认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析口诀 |
|:---|:---|:---|:---|
| "合成就是背反应" | 初学时通过记忆经典反应来解题 | 合成是创造性设计，核心是逆分析思维的运用 | "反应是字母，合成是作文" |
| "切断越多越好" | 认为碎片越简单合成越容易 | 每步切断引入一个反应 → 过多切断导致步数爆炸 | "少切断、多汇聚" |
| "保护基是万能的" | 教科书强调保护策略 | 保护基 = +2 步 + 2 次产率损失，应"不得已而用之" | "能不保护就不保护" |
| "只要路线通就对" | 忽视产率和选择性 | 路线需同时满足选择性（区域、立体）和可操作性 | "通只是最低要求，好才是目标" |
| "对称分子最好合成" | 直觉认为对称 = 简单 | 对称分子确实适合汇聚策略，但须识别对称面才能发挥优势 | "对称切断，事半功倍" |

### 12.3 入门级引导例题

**题目**：给出以下 2 步合成的逆分析和正向路线：以环己醇为原料合成环己酮。

**预期解答路径**：
1. 逆分析：环己酮 $\Leftarrow$ 环己醇（FGI：氧化）
2. 正向：环己醇 $\ce{+ PCC/CH2Cl2}$（或 $\ce{Na2Cr2O7/H2SO4}$） $\to$ 环己酮

**老师可引导的提问**：
- "目标分子和原料之间差了什么？"（多了 2 个 H = 少了 2 个氧化态单位 → 需要氧化）
- "除了 PCC 还能用什么条件？"（对比不同氧化剂的选择性）

**从这题自然过渡到多步合成**：
- 将题目扩展："以环己醇为原料合成 2-甲基环己酮" → 需先氧化为酮 → $\alpha$-甲基化 → 引入了第三步和区域选择性问题

### 12.4 思维训练方法

- **"反向讲故事"练习**：给学生一个正向合成路线，让他们"倒着画"逆合成树 → 训练逆向思维
- **"路线多选"练习**：给同一目标分子，要求画出至少 2 条不同的切断方案，比较优劣 → 培养策略比较意识
- **"步骤压缩"挑战**：给一条 8 步的教科书路线，要求在不改变关键切断的前提下压缩至 5 步 → 训练保护基经济性和反应顺序优化
- **"一砖多砌"练习**：以几种"通用建筑模块"（苯、乙酰乙酸乙酯、丙二酸二乙酯等）为限定原料，合成系列目标分子 → 强化汇聚式（convergent）思维

### 12.5 与现实/直觉的连接

- **类比**：逆合成分析类似"迷宫倒走"——从出口倒推到入口比从入口摸索出口更高效。复杂的迷宫（多官能团分子）有许多死胡同（不可行的切断），倒走可以帮助排除。
- **反直觉点**：初学者习惯"看到原料就往前推"，而逆合成要求"从目标往回看"。这个转向是学习的第一个坎——可以先用 1~2 步的简单转化练习"倒推"，让学生体会倒推的高效性。
- **学生常见疑问**："为什么不直接从原料一步步推到产物？"——回答：面对 5+ 步的合成，正向思维的搜索空间指数爆炸（每步都有 $N$ 种可能反应），而逆分析通过目标约束大幅缩减搜索空间。

## 十三、竞赛拓展

1. **Corey 形式化逆合成分析**：E. J. Corey 将逆合成分析系统化，提出"合成子-等价试剂-切断"的严格框架，并因此获 1990 年诺贝尔化学奖。竞赛中偶尔要求以 Corey 风格写出分析过程（明确标注合成子类型和等价试剂）。

2. **串联/级联反应（Tandem/Cascade Reactions）**：设计一个引发事件（如 Lewis 酸活化）触发连续多步反应，是高效合成的重要策略。经典例子：Johnson 阳离子环化构建甾体骨架；Robinson 环化（Michael + Aldol 串联）构建六元环。

3. **生物催化在合成中的应用**：酶催化反应具有极高的化学选择性、区域选择性和对映选择性。竞赛中偶见利用脂肪酶（lipase）拆分外消旋醇、利用面包酵母还原 $\beta$-酮酯的题型。

4. **无保护基全合成（Protecting-Group-Free Total Synthesis）**：Baran 实验室倡导的策略，利用底物自身反应性差异和巧妙反应设计避免保护基。代表例子：ingenol、phorbol 的全合成。竞赛思路：在设计路线时，把"能否不用保护基"作为一条优化准则。

5. **工业过程化学的额外约束**：原子经济性（atom economy）、E-factor（废物/产物质量比）、溶剂选择（绿色化学）、催化剂回收。竞赛初赛几乎不考，但决赛可能出现"从工业角度评价路线"的开放题。

6. **计算机辅助合成规划（CASP）**：如 Chematica（现 Synthia）、ASKCOS 等逆合成软件。了解其基本工作原理（反应规则数据库 + 启发式搜索），可能出现在信息题的背景介绍中。

7. **复杂度与步数关系**：Leadbeater 经验：天然产物全合成的步数大致 $N \approx 10 \times (\text{分子复杂度指数})^{0.5}$。理解为什么复杂分子（如紫杉醇）需要 40+ 步合成。
