# 系统命名法（IUPAC Nomenclature）

## 一、定义

**系统命名法（IUPAC Nomenclature）**是由国际纯粹与应用化学联合会（International Union of Pure and Applied Chemistry）制定的有机化合物标准化命名规则。其目标是为每个有机化合物提供**唯一、明确、可推导**的名称，避免俗名的歧义性和地域性。

- **核心原则**：
  1. **唯一性**：每个结构对应唯一的 IUPAC 名称
  2. **可逆性**：从名称可唯一推导出结构
  3. **系统性**：基于明确的规则和优先级
- **与俗名的关系**：俗名（trivial names）在历史上广泛使用（如醋酸、酒精），但系统命名是科学交流和竞赛的标准

## 三、核心原理

### 3.1 IUPAC 命名的基本步骤

**步骤 1：确定母体（Parent Structure）**
- 选择含有**最高优先官能团**的最长碳链作为主链
- 若官能团在环上，则以环为母体

**步骤 2：编号（Numbering）**
- 从使**最高优先官能团位次最小**的一端开始编号
- 若官能团位次相同，则使**取代基位次之和最小**（最低系列原则）
- 若有多种编号方式，按**字母顺序**使先出现的取代基位次较小

**步骤 3：命名取代基（Substituents）**
- 按字母顺序列出取代基（不考虑数量前缀 di-, tri- 等）
- 位置数字 + 取代基名称

**步骤 4：组合名称**
- 格式：**位置-取代基 + 母体名称**
- 示例：2-甲基丙烷、3-乙基-2-己醇

### 3.2 官能团优先顺序（从高到低）

| 优先级 | 官能团 | 后缀 | 作为取代基时的名称 |
|:---:|:---|:---|:---|
| 1 | 羧酸（—COOH）| -oic acid | 羧基（carboxy）|
| 2 | 磺酸（—SO₃H）| -sulfonic acid | 磺酸基（sulfo）|
| 3 | 酯（—COOR）| -oate | 烷氧羰基（alkoxycarbonyl）|
| 4 | 酰氯（—COCl）| -oyl chloride | 氯甲酰基（chloroformyl）|
| 5 | 酰胺（—CONH₂）| -amide | 氨基甲酰基（carbamoyl）|
| 6 | 腈（—CN）| -nitrile | 氰基（cyano）|
| 7 | 醛（—CHO）| -al | 甲酰基（formyl）|
| 8 | 酮（C=O）| -one | 氧代（oxo）|
| 9 | 醇（—OH）| -ol | 羟基（hydroxy）|
| 10 | 酚（Ar—OH）| -ol | 羟基（hydroxy）|
| 11 | 硫醇（—SH）| -thiol | 巯基（mercapto）|
| 12 | 胺（—NH₂）| -amine | 氨基（amino）|
| 13 | 醚（—OR）| — | 烷氧基（alkoxy）|
| 14 | 烯烃（C=C）| -ene | 烯基（alkenyl）|
| 15 | 炔烃（C≡C）| -yne | 炔基（alkynyl）|

**注意**：官能团优先顺序决定了母体的选择。高优先官能团作为母体后缀，低优先官能团作为取代基前缀。

### 3.3 多官能团命名示例

| 结构特征 | 母体选择 | 其他官能团处理 |
|:---|:---|:---|
| 同时含 —OH 和 C=O | 酮（-one 优先于 -ol）| 羟基作前缀（hydroxy）|
| 同时含 —COOH 和 —OH | 羧酸（最高优先）| 羟基作前缀 |
| 同时含 C=C 和 —OH | 烯醇（-en-ol）| — |
| 同时含 —CHO 和 C=C | 烯醛（-enal）| — |

## 四、关键结论

1. **母体选择 = 最高优先官能团 + 最长碳链**：这是命名的第一步也是最关键的一步
2. **编号优先满足高优先官能团**：即使取代基位次会因此变大
3. **取代基按字母顺序排列**：di-, tri-, tetra- 等数量前缀不参与字母排序
4. **立体化学标记放在名称最前面**：R/S, E/Z, D/L 等
5. **俗名在竞赛中仍常出现**：尤其是简单化合物（如甲苯、苯酚、丙酮）

## 五、常见分类

### 5.1 烷烃命名

| 碳数 | 母体名 | 碳数 | 母体名 |
|:---:|:---|:---:|:---|
| 1 | meth- | 6 | hex- |
| 2 | eth- | 7 | hept- |
| 3 | prop- | 8 | oct- |
| 4 | but- | 9 | non- |
| 5 | pent- | 10 | dec- |

**常见取代基**：

| 取代基 | 名称 | 取代基 | 名称 |
|:---|:---|:---|:---|
| —CH₃ | methyl | —C(CH₃)₃ | tert-butyl |
| —CH₂CH₃ | ethyl | —CH(CH₃)₂ | isopropyl |
| —CH₂CH₂CH₃ | propyl | —CH=CH₂ | vinyl |
| —CH(CH₃)CH₃ | isopropyl | —C≡CH | ethynyl |
| —CH₂CH₂CH₂CH₃ | butyl | —C₆H₅ | phenyl |
| —CH₂CH(CH₃)₂ | isobutyl | —C₆H₅CH₂ | benzyl |
| —CH(CH₃)CH₂CH₃ | sec-butyl | | |

### 5.2 不饱和烃命名

- **烯烃**：-ene；位置编号指向双键的第一个碳
- **炔烃**：-yne；位置编号指向三键的第一个碳
- **同时含双键和三键**：-en-yne；编号使不饱和键位次之和最小，双键优先（若位次相同）

### 5.3 环状化合物命名

| 类型 | 命名方式 |
|:---|:---|
| **环烷烃** | cyclo- + 烷烃名（如 cyclohexane）|
| **环烯烃** | cyclo- + 烯烃名（编号使双键位次最小）|
| **环上连烷基** | 若烷基简单，环为母体；若烷基复杂，烷烃为母体 |
| **二环化合物** | bicyclo[a.b.c] + 母体；a ≥ b ≥ c 为桥碳数（不含桥头）|
| **螺环化合物** | spiro[a.b] + 母体；a, b 为两环碳数（不含螺原子）|

### 5.4 俗名与系统名对照

| 俗名 | 结构 | IUPAC 系统名 |
|:---|:---|:---|
| **醋酸** | CH₃COOH | ethanoic acid |
| **酒精** | CH₃CH₂OH | ethanol |
| **甘油** | HOCH₂CH(OH)CH₂OH | propane-1,2,3-triol |
| **草酸** | HOOCCOOH | ethanedioic acid |
| **甲醛** | HCHO | methanal |
| **乙醛** | CH₃CHO | ethanal |
| **丙酮** | CH₃COCH₃ | propan-2-one |
| **甲苯** | C₆H₅CH₃ | methylbenzene |
| **苯酚** | C₆H₅OH | phenol |
| **苯甲酸** | C₆H₅COOH | benzoic acid |

## 六、适用条件

### 6.1 IUPAC 命名的"最低系列原则"

当存在多种编号方式时：

| 优先级 | 规则 |
|:---|:---|
| 1 | 高优先官能团位次最小 |
| 2 | 取代基位次之和最小 |
| 3 | 第一个有差异的位次最小 |
| 4 | 按字母顺序，先出现的取代基位次较小 |

### 6.2 立体化学标记的位置

| 标记 | 位置 | 示例 |
|:---|:---|:---|
| R/S | 名称最前，括号内 | **(2R,3S)**-2,3-二羟基丁二酸 |
| E/Z | 名称最前，括号内 | **(E)**-2-丁烯 |
| D/L | 名称前 | **D**-葡萄糖 |
| α/β（糖）| 名称中 | α-D-吡喃葡萄糖 |
| (+)/(−) | 名称中或后 | (+)-酒石酸 |

## 七、常见比较

### 7.1 系统命名 vs 俗名

| 维度 | 系统命名（IUPAC） | 俗名 |
|:---|:---|:---|
| **规则性** | 严格规则，可推导 | 历史遗留，需记忆 |
| **适用范围** | 所有化合物 | 常见简单化合物 |
| **信息完整性** | 可还原结构 | 不一定 |
| **国际通用性** | 强 | 弱（地域差异）|
| **竞赛要求** | 必须掌握 | 需认识但优先用系统名 |
| **文献使用** | 现代文献标准 | 早期文献、生物/医学文献 |

### 7.2 中文命名 vs 英文命名

| 维度 | 中文 IUPAC | 英文 IUPAC |
|:---|:---|:---|
| **数字位置** | 数字在前（2-甲基）| 数字在前（2-methyl）|
| **母体后缀** | 醇、酮、酸 | -ol, -one, -oic acid |
| **取代基前缀** | 甲基、乙基 | methyl, ethyl |
| **多取代基** | 二、三、四 | di-, tri-, tetra- |
| **竞赛要求** | 初赛常用 | 决赛/国际赛常用 |

## 十、例题

### 例题 1：烷烃命名

**题目**：为下列烷烃命名。

```
      CH₃
       |
CH₃ — CH — CH₂ — CH — CH₂ — CH₃
               |        |
              CH₃      CH₂CH₃
```

**解析**：

1. **找最长碳链**：
   - 直链：6 个碳（己烷）
   - 若包含乙基支链：7 个碳（庚烷）
   - 最长链为 **7 个碳**（庚烷）

2. **编号**：
   - 从左端：2-甲基, 4-甲基, 5-乙基 → 位次 (2,4,5)
   - 从右端：3-乙基, 4-甲基, 6-甲基 → 位次 (3,4,6)
   - 从左端位次和 = 11，从右端 = 13
   - **从左端编号**

3. **取代基排序**（按字母顺序）：
   - ethyl（乙基）在 methyl（甲基）之前
   - 但中文命名按"取代基复杂程度"或简单规则

4. **名称**：
   - 英文：**5-ethyl-2,4-dimethylheptane**
   - 中文：**5-乙基-2,4-二甲基庚烷**

### 例题 2：多官能团命名

**题目**：命名下列化合物。

```
    CHO
     |
HO — C — CH₂ — COOH
     |
    CH₃
```

**解析**：

1. **官能团识别**：—OH, —CHO, —COOH
2. **最高优先官能团**：—COOH（羧酸）
3. **主链**：含 —COOH 的最长链 = 4 个碳（丁酸）
4. **编号**：从 —COOH 端开始
   - C1 = COOH, C2 = CH₂, C3 = CH(OH)(CHO), C4 = CH₃
5. **取代基**：
   - 3-位：—OH（羟基）和 —CHO（甲酰基）
   - 按字母：formyl（甲酰基）在 hydroxy（羟基）之前

6. **名称**：
   - 英文：**3-formyl-3-hydroxybutanoic acid**
   - 或更准确地考虑醛基是否应作为母体
   - 实际上 —CHO 作为取代基是 formyl

### 例题 3：名称 → 结构

**题目**：画出 **(E)-3-甲基戊-2-烯-1-醇** 的结构。

**解析**：

1. **母体**：戊-2-烯-1-醇（5 个碳，双键在 C2-C3，—OH 在 C1）
2. **取代基**：3-甲基（C3 上有一个 —CH₃）
3. **立体化学**：E（双键两侧高优先基团在异侧）

4. **碳链骨架**：
   - C1：—CH₂OH
   - C2：=CH—
   - C3：—CH(CH₃)—
   - C4：—CH₂—
   - C5：—CH₃

5. **E 构型**：
   - C2 上：—CH₂OH 优先于 —H
   - C3 上：—CH₂CH₃ 优先于 —CH₃
   - 高优先基团在异侧

6. **结构**：

```
      CH₃
       |
   HO-CH₂   CH₂-CH₃
        \   /
         C=C
        /   \
       H     H
```

或更标准：

```
       CH₂OH
         |
    H — C = C — CH₂CH₃
             |
            CH₃
```

**注意**：E 构型要求 —CH₂OH 和 —CH₂CH₃ 在双键异侧。

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| "主链只选直链" | 忽略支链可能更长 | 主链是**含最高优先官能团的最长碳链**，不一定是直链 | 某分子主链是否一定要是直链？ |
| "编号使取代基位次和最小" | 忽略官能团优先级 | **先满足高优先官能团位次最小**，再考虑取代基 | 同时有 —OH 和 C=O，应从哪端编号？ |
| "di-, tri- 参与字母排序" | 误解排序规则 | 数量前缀**不参与**字母排序；按取代基本身字母排序 | dimethyl 和 ethyl 哪个排前面？ |
| "俗名不能用于竞赛" | 过度严格 | 竞赛接受**系统命名**，但需认识常见俗名 | 醋酸的系统名是什么？ |
| "二环化合物编号随意" | 不了解桥环规则 | 从桥头碳开始，沿**最长桥** → 次长桥 → 最短桥 | bicyclo[2.2.1]庚烷的编号顺序？ |
| "立体标记放在名称中间" | 位置错误 | R/S, E/Z 放在名称**最前面** | (R)-乳酸的正确写法是什么？ |

## 十二、🎯 教学视角
### 12.1 学习路径

```
前置：构造异构 → 官能团识别 → 碳链计数
     ↓
本节：系统命名法（母体选择 + 编号 + 取代基 + 立体标记）
     ↓
后续分支：
  ├── 复杂环系命名（桥环、螺环、稠环）
  ├── 杂环命名（IUPAC 杂环命名规则）
  └── 俗名与系统名对照
```

**建议课时**：1.5 课时（基本规则 0.4 + 官能团优先 0.3 + 环系命名 0.3 + 练习 0.5）

### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "命名就是背规则" | 忽视实践 | 强调：命名是**应用规则解决具体问题**，需要大量练习 |
| "中文命名和英文命名完全对应" | 忽略差异 | 对比：中文"甲、乙、丙"vs 英文"meth-, eth-, prop-"；但规则结构一致 |
| "俗名已经过时" | 不了解实际 | 说明：生物化学、药物化学中俗名 ubiquitous，系统名和俗名都需掌握 |

### 12.3 入门例题

**最简判断练习**：
> 1. 命名 CH₃CH(CH₃)CH₂CH₃。
> 2. 同时含有 —OH 和 —CHO 的分子，母体是什么？
> 3. 画出 2-甲基-2-丁醇的结构。

**答案**：
1. **2-甲基丁烷**
2. **醛**（—CHO 优先于 —OH，但作为取代基时 —CHO 是 formyl；若 —CHO 是最高优先则母体为醛）
   - 更准确：若同时有 —CHO 和 —OH，—CHO 更高优先 → 母体为醛
3.
```
     CH₃
      |
CH₃ — C — CH₂CH₃
      |
     OH
```
