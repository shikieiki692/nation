# E1 反应

## 一、定义

**E1 反应（Elimination, unimolecular）** 即**单分子消除反应**，是经由**碳正离子中间体**分两步完成的消除反应：

**第一步（速率决定步，RDS）**：离去基（LG）解离，形成碳正离子
$$
\ce{R-CH2-CR'2-LG ->[慢] R-CH2-CR'2^+ + LG^-}
$$

**第二步**：碱（B⁻）夺取 β-氢，形成烯烃
$$
\ce{R-CH2-CR'2^+ + B^- ->[快] R-CH=CR'2 + HB}
$$

- **速率方程**：$v = k[\text{底物}]$（仅与底物浓度有关，与碱浓度无关）
- **核心特征**：与 SN1 共享碳正离子中间体，**竞争关系**
- **区域选择性**：通常遵循 **Zaitsev 规则**（生成更稳定的、取代更多的烯烃）

## 三、核心原理

### 3.1 两步机理详解

```
        LG                      LG⁻
         |                       |
R-CH₂-C-R'₂  ──[慢]──→  R-CH₂-C⁺-R'₂  +  B⁻
         |                       |
        β-H                     β-H
                                |
                                └──[快]──→  R-CH=CR'₂  +  HB
```

**第一步**：C-LG 键异裂，电子对随 LG 离去。此步骤与 SN1 的决速步**完全相同**。

**第二步**：碱夺取 β-H，电子对用于形成 C=C π 键。此步极快，因为：
- 碳正离子是高能中间体，迅速寻求稳定化
- 只要存在任何可夺取的 β-H，消除即发生

### 3.2 碳正离子中间体的命运

碳正离子一旦形成，面临三条可能的出路：

| 路径 | 结果 | 条件偏好 |
|:---|:---|:---|
| **SN1**：亲核体捕获碳正离子 | 取代产物 | 高 [Nu⁻]、弱碱、低温 |
| **E1**：碱夺取 β-H | 消除产物 | 高 [Base]、强碱（即使弱碱也可，因第二步快）、高温 |
| **重排**：1,2-氢/烷基迁移 | 更稳定的碳正离子 | 邻位有提供更稳定碳正离子的结构 |

**竞赛核心**：E1/SN1 的产物比例由 **"离子对阶段"** 的微观环境决定——溶剂笼内的碱浓度、溶剂极性、温度共同调控。

## 四、关键结论

1. **底物要求**：3° > 2° >> 1°（1° 底物几乎不发生 E1，因 1° 碳正离子极不稳定）
2. **碱的要求**：理论上任何碱均可（第二步非决速步），但 **弱碱/低浓度碱** 更利于 E1（避免 E2 竞争）
3. **溶剂要求**：**极性质子溶剂**（H₂O、ROH）最利 E1——稳定碳正离子和 LG⁻
4. **温度效应**：高温促进 E1（消除的 ΔS > 0，TΔS 项有利）
5. **Zaitsev 规则的本质**：生成更取代的烯烃（热力学控制），但**若有空间位阻或环张力，可能偏离**
6. **E1 必伴随重排**：若生成的碳正离子可通过 1,2-迁移得到更稳定碳正离子，则消除前的碳正离子已重排

## 五、常见分类

### 5.1 按底物分类

| 底物类型 | E1 倾向 | 典型实例 | 备注 |
|:---|:---:|:---|:---|
| **3° 卤代烷** | 高 | (CH₃)₃CBr + EtOH/Δ | 经典 E1 教材反应 |
| **2° 卤代烷** | 中 | (CH₃)₂CHBr + H₂O | 与 SN1/E2 竞争 |
| **1° 卤代烷** | 极低 | CH₃CH₂Br | 几乎不发生 |
| **苄基/烯丙型** | 高 | PhCH₂CH₂Cl | 碳正离子特别稳定 |
| **醇（酸催化脱水）** | 高 | (CH₃)₃COH + H₂SO₄ | 工业制烯烃 |

### 5.2 酸催化脱水（E1 的最重要应用）

醇在酸催化下的脱水是**合成烯烃**的经典方法：

$$
\ce{R-CH2-CH(OH)-R' + H^+ -> R-CH2-CH^+-R' + H2O -> R-CH=CH-R' + H^+}
$$

**机理**：
1. OH 质子化（将 poor LG 转化为 good LG：H₂O）
2. H₂O 离去，形成碳正离子
3. 碱（H₂O 或 HSO₄⁻）夺取 β-H

**注意**：酸催化脱水常伴随 **Wagner-Meerwein 重排**，这是竞赛高频考点。

## 六、适用条件

### 6.1 促进 E1 的条件总结

| 因素 | 理想条件 | 原因 |
|:---|:---|:---|
| **底物** | 3° > 2°，苄基/烯丙型 | 稳定碳正离子 |
| **碱** | 弱碱（H₂O、ROH、吡啶）| 避免 E2 竞争；E1 的第二步不依赖碱强度 |
| **溶剂** | 极性质子溶剂（H₂O、EtOH）| 溶剂化稳定碳正离子和 LG |
| **温度** | 高温 | ΔS > 0，高温有利；也加速碳正离子形成 |
| **离去基** | 好 LG（I > Br > Cl > OH₂⁺）| 降低第一步活化能 |

### 6.2 E1 vs E2 的判别实验

| 实验 | E1 结果 | E2 结果 |
|:---|:---|:---|
| **改变 [Base]** | 速率不变 | 速率正比于 [Base] |
| **同位素效应（KIE）** | 无（第二步非 RDS）| 显著（C-H 断裂在 TS 中）|
| **重排产物** | 常有 | 无 |
| **立体化学** | 无要求（平面碳正离子）| **anti-共平面**要求 |

## 七、常见比较

### 7.1 E1 vs SN1 竞争：同一中间体的两条出路

```
        LG
         |
R-CH₂-C-R'₂
         |
        β-H
          \
           [失去 LG⁻] → R-CH₂-C⁺-R'₂  ← 碳正离子中间体
                          /    \
                    [Nu⁻ 进攻]  [B⁻ 夺 H]
                        |          |
                    SN1 产物    E1 产物
```

| 条件 | SN1 主导 | E1 主导 |
|:---|:---|:---|
| **亲核体/碱** | 强亲核体（I⁻、Br⁻）| 弱亲核体但碱性足够（H₂O、EtOH）|
| **温度** | 低温 | 高温 |
| **溶剂** | 极性非质子（有利于 SN，不利于 E）| 极性质子（两者皆可）|
| **底物 β-H 数** | 无 β-H 时只能 SN1 | β-H 越多，E1 概率越大 |

**ABOC 经验规则**：
- 3° 卤代烷 + 弱碱/极性质子溶剂/高温 → **E1 为主**
- 3° 卤代烷 + 强亲核体/低温 → **SN1 为主**

### 7.2 E1 vs E2 速查表

| 维度 | E1 | E2 |
|:---|:---|:---|
| **机理** | 分步（碳正离子）| 协同（一步）|
| **动力学** | 一级（v = k[RX]）| 二级（v = k[RX][B]）|
| **底物** | 3° > 2° | 1° > 2° > 3°（3° 易 E2，但位阻过大时难）|
| **碱要求** | 弱碱即可 | 强碱（t-BuO⁻、NaNH₂）|
| **立体化学** | 无特殊要求 | **anti-共平面** |
| **重排** | 常见 | 无 |
| **取向** | Zaitsev（热力学）| Zaitsev 或 Hofmann（看碱位阻）|
| **溶剂** | 极性质子 | 极性非质子（促 E2）或质子 |

## 十、例题

### 例题 1：醇的酸催化脱水与重排（ABOC 4.4 改编）

**题目**：3,3-二甲基-2-丁醇在 H₂SO₄/Δ 条件下的脱水主产物是什么？

**解析**：
1. **质子化**：OH → OH₂⁺（好离去基）
2. **离去**：形成 2° 碳正离子 (CH₃)₃C-CH⁺-CH₃
3. **重排**：邻位季碳发生 **1,2-甲基迁移** → (CH₃)₂C⁺-CH(CH₃)₂（3° 碳正离子）
4. **消除**：碱夺取 β-H，形成最稳定烯烃
5. **主产物**：**2,3-二甲基-2-丁烯**（四取代烯烃，Zaitsev 产物）

**陷阱**：若忽略重排，会错判为 3,3-二甲基-1-丁烯。

### 例题 2：E1/E2/SN2 判别

**题目**：下列条件下，(CH₃)₃CBr 主要发生什么反应？
1. NaOH/H₂O，25°C
2. EtOH/Δ
3. t-BuOK/t-BuOH，60°C

**解析**：
1. **NaOH/H₂O，25°C**：强碱/极性质子 → 主要是 **SN1**（OH⁻ 是良好亲核体），少量 E2
2. **EtOH/Δ**：弱碱（EtOH）/极性质子/高温 → **E1**（弱碱无法 E2，高温促消除）
3. **t-BuOK/t-BuOH**：强碱/大位阻/高温 → **E2**（强碱克服 E1；3° 底物 + 大位阻碱 = Hofmann 可能，但 3° 通常仍 Zaitsev）

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| 认为 E1 不需要碱 | 第二步确实需要碱夺 H | 任何可作为碱的物种（溶剂、LG⁻、HSO₄⁻）均可；只是第二步非 RDS，故碱浓度不影响速率 | E1 的第二步中，谁是碱？为什么速率方程里没有 [Base]？ |
| 1° 底物写 E1 机理 | 忽略碳正离子稳定性门槛 | 1° 碳正极不稳定，1° 底物几乎**不发生 E1** | 1° 卤代烷在 EtOH/Δ 下主要发生什么反应？ |
| 忽略 E1 中的重排 | 只画一步消除 | E1 有碳正离子中间体 → **必检查重排可能** | 什么时候 E1 产物与预期 Zaitsev 产物不同？ |
| 混淆 E1 与 SN1 的 "弱碱" 条件 | 条件重叠 | 看产物：**取代物 = SN1；烯烃 = E1**。高温/高 [Base] 偏 E1；低温/高 [Nu] 偏 SN1 | 同一底物、同一溶剂，换温度为什么产物从取代变消除？ |
| 认为 Zaitsev 规则绝对 | 忽略位阻和环张力 | 大位阻碱（t-BuO⁻）→ Hofmann；环状体系中反-Bredt 烯烃不可形成 | 什么条件下 E1 会违反 Zaitsev 规则？ |

## 十二、🎯 教学视角

### 12.1 学习路径

```
前置：碳正离子稳定性 → SN1反应 → 消除反应总论
     ↓
本节：E1反应（分步机理 + Zaitsev + 重排 + 与 SN1 竞争）
     ↓
后续分支：E2反应（协同对比）→ E1cb反应（碳负离子路径）→ 消除反应综合
```

**建议课时**：1.5 课时（机理 0.5 + 与 SN1 竞争 0.5 + 重排专题 0.5）

### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "E1 是 SN1 的副反应" | 认为 E1 不重要 | 强调在高温/合成条件下，E1 常常是**主反应**；SN1/E1 是平行路径 |
| "E1 只发生在卤代烷" | 忽略醇脱水 | 醇的酸催化脱水是**最重要 E1 应用**，工业上大规模生产烯烃 |
| "弱碱 = 不反应" | 把碱强度与反应性直接挂钩 | E1 的第二步极快，弱碱（如 H₂O）足以完成消除 |
| "Zaitsev 产物总是主产物" | 死记规则 | 展示反例：大位阻底物 + 大位阻碱 → Hofmann；环状体系中的构象限制 |

### 12.3 入门例题

**最简 E1 判别题**：
> 叔丁基溴在乙醇中加热，主要产物是什么？写出两步机理。

**答案**：
1. (CH₃)₃CBr → (CH₃)₃C⁺ + Br⁻
2. EtOH 夺取 β-H → (CH₃)₂C=CH₂ + EtOH₂⁺
**主产物**：异丁烯（2-甲基丙烯）
**引导**：为什么不是 SN1 产物（叔丁基乙基醚）？答：高温促进消除。

### 12.4 现实类比

**「拆弹」类比 E1 反应**：
- 底物 = 一颗定时炸弹（碳正离子前体）
- 离去基 = 引信（LG 离去 = 拔掉引信）
- 碳正离子 = 已激活的炸弹（高能、不稳定、迅速寻求出路）
- SN1 = 有人迅速把炸弹包起来（亲核体捕获）
- E1 = 炸弹自己爆炸，碎片重组为更稳定形态（消除得烯烃）
- 重排 = 炸弹在爆炸前内部零件重新排列（1,2-迁移），变成另一种爆炸方式

## 十三、竞赛拓展

### 13.1 竞赛中的 E1 变体与应用

| 反应/场景 | 特点 | 竞赛价值 |
|:---|:---|:---|
| **频哪醇重排-消除** | 邻二醇酸催化 → 碳正离子 → 重排 + 消除 | 构建螺环/迁移后的酮 |
| **Wagner-Meerwein + E1** | 萜类生物合成中的串联重排消除 | 天然产物全合成中的骨架重排 |
| **醇的酸催化脱水制烯烃** | 工业基础反应 | 全合成中构建双键的经典方法 |
| **E1 与碳正离子捕获的串联** | 碳正离子被烯烃捕获 → 新碳正 → 继续消除 | 构建多烯体系 |

### 13.2 与物理有机的交叉

- **溶剂化显色（Y 值）**：E1 速率与溶剂离子化能力（Y 值）呈线性关系（Grunwald-Winstein 方程）
- **同位素标记**：β-D 取代底物在 E1 中无动力学同位素效应（KIE ≈ 1），确证 C-H 断裂非 RDS
- **碳正离子寿命**：激光闪光光解可直接观测碳正离子的寿命（ns–μs 量级）
