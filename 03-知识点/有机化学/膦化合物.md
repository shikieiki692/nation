# 膦化合物

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[48-膦化合物]]

## 一、定义

**膦化合物（Phosphorus Compounds）** 是指含 **P-C 键** 的有机化合物。磷与氮同族，但原子半径更大、电负性更低，导致膦化合物的性质与胺类有显著差异。

主要类型：
| 类型 | 通式 | 特征 | 典型代表 |
|:---|:---|:---|:---|
| **膦（Phosphine）** | $\mathrm{R_3P}$ | 磷上孤对电子，强亲核性 | 三苯基膦 $\mathrm{Ph_3P}$ |
| **鏻盐（Phosphonium Salt）** | $\mathrm{R_4P^+X^-}$ | 磷为正四面体，稳定 | 甲基三苯基溴化鏻 |
| **膦叶立德（Ylide）** | $\mathrm{R_3P^+=CR_2^-}$ | 邻位电荷分离，强亲核性 | Ph₃P=CH₂ |
| **膦酸酯** | $\mathrm{(RO)_2P(O)R'}$ | P=O 键，Wittig-Horner 试剂 | HWE 试剂 |
| **氧化膦** | $\mathrm{R_3P=O}$ | P=O 极性强，稳定 | 三苯基氧化膦 |

## 三、核心原理

### 3.1 膦 vs 胺的性质差异

| 性质 | 膦（$\mathrm{R_3P}$）| 胺（$\mathrm{R_3N}$）|
|:---|:---|:---|
| **碱性** | 弱（pKa ~ -2 to 2）| 较强（pKa ~ 9-11）|
| **亲核性** | **极强**（P 原子大，可极化性高）| 中等 |
| **孤对电子** | 3s/3p 杂化，更弥散 | 2s/2p 杂化，更定域 |
| **氧化趋势** | 易氧化为氧化膦（P=O，强放热）| 不易氧化 |
| **作为配体** | 强 σ-给电子 + π-反馈 | 仅 σ-给电子 |

**竞赛核心**：膦的 **强亲核性** 和 **P=O 形成驱动力** 是其反应性的根源。

### 3.2 膦叶立德的形成

$$\mathrm{R_3P + R'X \xrightarrow{S_N2} R_3P^+R' X^- \xrightarrow{强碱} R_3P^+=CHR'^-}$$

1. **季鏻盐形成**：膦对卤代烷进行 SN2 → 鏻盐
2. **去质子**：强碱（如 n-BuLi、NaH、NaNH₂）夺取 α-H → 叶立德

**稳定性分类**：
| 类型 | 取代基 | 稳定性 | 反应性 |
|:---|:---|:---:|:---:|
| **非稳定化叶立德** | R = H, 烷基 | 低，活泼 | 高 |
| **稳定化叶立德** | R = COR', CO₂R', CN | 高（共轭稳定）| 低 |

### 3.3 Wittig 反应核心

$$\mathrm{R_3P^+=CR'_2^- + R''CHO \rightarrow R''CH=CR'_2 + R_3P=O}$$

- **本质**：叶立德的碳负离子进攻醛/酮羰基 → 氧杂磷杂环丁烷（oxaphosphetane）→ 协同分解为烯烃 + 氧化膦
- **驱动力**：P=O 键极强（~540 kJ/mol），反应不可逆
- **立体化学**：非稳定化叶立德 → **Z-烯烃**（动力学）；稳定化叶立德 → **E-烯烃**（热力学）

## 四、关键结论

1. **膦亲核性 >> 胺** —— P 原子大、可极化性高、孤对电子弥散
2. **P=O 形成是强驱动力** —— Wittig、Arbuzov、Mitsunobu 等反应的核心动力
3. **Wittig 反应选择性**：
   - 非稳定化叶立德 + 醛 → Z-烯烃
   - 稳定化叶立德 + 醛 → E-烯烃
4. **膦是优良配体** —— 过渡金属催化（如钯催化偶联）中常用

## 五、常见分类或情形

### 5.1 常见膦试剂

| 试剂 | 结构 | 用途 |
|:---|:---|:---|
| **三苯基膦（Ph₃P）** | $\mathrm{(C_6H_5)_3P}$ | Wittig 反应、Mitsunobu 反应、配体 |
| **三丁基膦（Bu₃P）** | $\mathrm{(n-C_4H_9)_3P}$ | 更强亲核性，用于偶联反应 |
| **BINAP** | 联萘二膦 | 手性配体，不对称催化 |
| **DPPP/DPPB** | 双齿膦 | 钯催化偶联的螯合配体 |

### 5.2 磷参与的人名反应

| 反应 | 试剂 | 转化 | 核心驱动力 |
|:---|:---|:---|:---|
| **Wittig** | Ph₃P=CR₂ | 醛/酮 → 烯烃 | P=O 形成 |
| **Arbuzov** | P(OR)₃ + R'X | 亚磷酸酯 → 膦酸酯 | P=O 形成 |
| **Mitsunobu** | Ph₃P + DEAD + ROH | 醇 inversion → 酯/醚 | P=O 形成 |
| **HWE** | (EtO)₂P(O)CH₂R | 醛 → α,β-不饱和酯 | P=O 形成 |
| **Appel** | Ph₃P + CCl₄ | 醇 → 氯代烷 | P=O 形成 |

## 六、适用条件与限制

| 反应 | 条件 | 限制 |
|:---|:---|:---|
| Wittig | 无水，低温（-78°C → rt）| 对水敏感；位阻大酮反应慢 |
| 叶立德制备 | 强碱，无水 THF | 空气敏感；需惰性气氛 |
| Mitsunobu | Ph₃P + DIAD/DEAD | 产物含氧化膦，难分离 |
| Arbuzov | 加热（150-200°C）| 仅适用于伯/仲卤代烷 |

## 七、常见比较与易混点

### 7.1 Wittig vs Horner-Wadsworth-Emmons (HWE)

| 维度 | Wittig | HWE |
|:---|:---|:---|
| 试剂 | Ph₃P=CHR | (EtO)₂P(O)CHR⁻ |
| 副产物 | Ph₃P=O（难除）| (EtO)₂P(O)O⁻（水溶性，易除）|
| E/Z 选择性 | 依赖叶立德稳定性 | 通常高 E-选择性 |
| 底物范围 | 醛/酮 | 醛（酮反应差）|
| 工业应用 | 较少 | 较多（副产物易除）|

### 7.2 稳定化 vs 非稳定化叶立德

| 特征 | 非稳定化 | 稳定化 |
|:---|:---|:---|
| 结构 | Ph₃P=CH₂ | Ph₃P=CH-COR |
| 制备碱 | n-BuLi, NaH | NaOMe, K₂CO₃（弱碱即可）|
| Wittig 产物 | Z-烯烃 | E-烯烃 |
| 原因 | 动力学控制 | 热力学控制（oxaphosphetane 分解前平衡）|

## 十、例题

### 例题 1：Wittig 产物与立体化学

**题目**：环己基甲醛分别与 (a) Ph₃P=CH₂ 和 (b) Ph₃P=CHCO₂Et 反应，产物和立体化学分别是什么？

**解析**：
**(a) 非稳定化叶立德**：
- 反应通过早期过渡态 → 动力学控制
- oxaphosphetane 分解前不达到平衡
- 产物：**亚甲基环己烷（Z-选择性为主）**

**(b) 稳定化叶立德**：
- 吸电子酯基稳定碳负离子
- oxaphosphetane 可逆，分解前达到热力学平衡
- 产物：**(E)-2-(乙氧羰基)乙烯基环己烷（E-选择性）**

### 例题 2：HWE vs Wittig 选择

**题目**：工业上由甲醛制备丙烯酸甲酯，应选 Wittig 还是 HWE？

**解析**：
选 **HWE**（Horner-Wadsworth-Emmons）：
- 试剂：(MeO)₂P(O)CH₂CO₂Me + NaH → 碳负离子 + HCHO
- 产物：CH₂=CHCO₂Me
- 原因：HWE 副产物为水溶性膦酸盐，易水洗除去；Wittig 副产物 Ph₃P=O 难除，成本高。

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| 认为 Wittig 反应总是 E-选择性 | 混淆稳定化/非稳定化 | 非稳定化 → Z；稳定化 → E | Ph₃P=CH₂ 与 Ph₃P=CHCO₂Me 的区别？ |
| 忽略 P=O 的驱动力 | 不了解键能 | P=O ~540 kJ/mol，是极强驱动力 | 为何 Wittig 反应不可逆？ |
| 把 HWE 当作 Wittig 的变体 | 概念混淆 | HWE 试剂是膦酸酯，不是鏻叶立德 | HWE 副产物为何易除？ |
| 认为所有膦都易氧化 | 未分类 | 三烷基膦 >> 三芳基膦（芳基给电子共轭稳定）| Ph₃P 为何可在空气中操作？ |

## 十二、🎯 教学视角

### 12.1 学习路径

```
前置：亲核取代 → 羰基极性 → 烯烃结构
     ↓
本节：膦化合物（分类 + Wittig + 叶立德 + HWE）
     ↓

```

**建议课时**：2 课时（膦基础 0.5 + Wittig 机理与选择性 1 + HWE/其他反应 0.5）

### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "膦和胺差不多" | 同族 = 同性质 | 强调原子半径和电负性差异；亲核性对比实验 |
| "Wittig 就是制烯烃" | 忽略选择性 | 稳定化/非稳定化叶立德的选择性是核心考点 |
| "P=O 和 C=O 一样" | 键级类比错误 | P=O 有 d-π 反馈，键能远高于 C=O |

### 12.3 入门例题

**最简 Wittig 题**：
> 苯甲醛与 Ph₃P=CH₂ 反应的主产物是什么？Z/E 比例倾向？
> 答案：苯乙烯；非稳定化叶立德 → Z-选择性为主。
