# Heck 反应（Mizoroki-Heck Reaction）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[50-有机合成]]、[[51-金属有机]]

---

## 一、定义

**Heck 反应（Mizoroki-Heck Reaction）**：在 **Pd(0) 催化剂**和**碱**存在下，**芳基/烯基卤代物（或三氟甲磺酸酯）**与**烯烃**发生交叉偶联，生成**芳基取代烯烃**的反应。

**通式**：
$$
\ce{Ar-X + R-CH=CH2 ->[Pd(0), 碱] Ar-CH=CH-R + HX}
$$

**本质**：**Pd 催化的芳基化烯烃反应**——通过氧化加成、迁移插入和 β-H 消除三步构建 C–C 键，同时保留烯烃的不饱和度。

## 三、核心原理

### 3.1 催化循环

**Pd(0)/Pd(II) 循环**：

```
步骤 1：氧化加成（Oxidative Addition）
  Pd(0)L₂ + Ar-X → Ar-Pd(II)-X·L₂

步骤 2：迁移插入（Migratory Insertion）
  烯烃配位 → Ar-Pd(II)-CH₂-CH₂-X
  （芳基从 Pd 迁移到烯烃的一个碳上）

步骤 3：β-H 消除（β-Hydride Elimination）
  Ar-CH₂-CH₂-Pd(II)-X → Ar-CH=CH-R + H-Pd(II)-X

步骤 4：催化剂再生
  H-Pd(II)-X + 碱 → Pd(0)L₂ + 碱·HX
```

**关键点**：
- Heck **没有还原消除**——产物中的双键来自 β-H 消除
- 这也是 Heck 与 Suzuki 的核心区别

### 3.2 区域选择性

**烯烃上两个可能的位置**：

| 烯烃类型 | 主要产物 | 原因 |
|:---|:---|:---|
| **末端烯烃（R-CH=CH₂）** | **Ar-CH=CH-R**（芳基连末端碳）| 电子效应：芳基连取代较少的碳 |
| **吸电子基取代烯烃** | 芳基连 β-碳 | 电子效应：芳基连远离 EWG 的碳 |
| **大位阻配体** | 可能改变选择性 | 位阻效应压倒电子效应 |

**规律**：
- **电子控制**：芳基倾向于连在烯烃的 **β-位**（取代较少的碳）
- **位阻控制**：大位阻配体（如 P(o-tolyl)₃）可使芳基连在 **α-位**

### 3.3 立体选择性

**迁移插入的立体化学**：
- 烯烃配位 + 迁移插入是 **syn** 过程
- β-H 消除也要求 **syn** 共平面
- 总结果：产物双键的构型由过渡态几何决定

**内烯烃的 Heck**：
- 可能生成 **E 型**或 **Z 型**产物
- 通常 **E 型为主**（热力学更稳定）

### 3.4 底物范围

**卤代物组分**：
- **芳基碘（Ar-I）**：最活泼
- **芳基溴（Ar-Br）**：常用
- **芳基氯（Ar-Cl）**：需活性更高的催化剂体系
- **芳基三氟甲磺酸酯（Ar-OTf）**：常用
- **烯基卤**：也可反应

**注意**：烷基卤 **不适合**（β-H 消除太快，无法可控进行）。

**烯烃组分**：
- 末端烯烃（最常用）
- 1,1-二取代烯烃
- 吸电子基取代烯烃（丙烯酸酯、苯乙烯等）
- 环烯烃

---

## 四、关键结论

1. **Heck = 芳基卤 + 烯烃 → 芳基烯烃**
2. **Pd(0)/Pd(II) 循环**：氧化加成 → 迁移插入 → β-H 消除
3. **无还原消除**——产物双键来自 β-H 消除
4. **末端烯烃 → 芳基连 β-位**（电子控制）
5. **E 型产物为主**（热力学控制）
6. **syn-迁移插入 + syn-β-H 消除**决定立体化学

## 五、常见分类或情形

### 5.1 按烯烃类型分类

| 烯烃 | 产物特征 | 选择性 |
|:---|:---|:---|
| **末端烯烃** | 芳基取代内烯 | β-位为主 |
| **丙烯酸酯** | 芳基在 β-位 | 电子控制 |
| **苯乙烯** | 二苯乙烯衍生物 | β-位为主 |
| **环烯烃** | 芳基取代环烯 | 位阻控制 |

### 5.2 分子内 Heck

| 类型 | 特点 | 合成价值 |
|:---|:---|:---|
| **分子内 Heck** | 同一分子内的芳基卤 + 烯烃 | 构建环状结构（5-30 元环）|
| **串联 Heck** | 多步 Heck 连续进行 | 构建多芳基烯烃 |

## 六、适用条件与限制

1. **需要无氧条件**
2. **需要碱**——捕获 HX（Et₃N、NaOAc、KOAc 等）
3. **溶剂**：DMF、NMP、MeCN、二氧六环、甲苯
4. **温度**：80–150°C
5. **烷基卤不适合**——β-H 消除不可控
6. **1,2-二取代烯烃反应性较低**
7. **可能有多重插入**——高温下烯烃可能多次插入

## 七、常见比较与易混点

### 7.1 Heck vs Suzuki

| | **Heck** | **Suzuki** |
|:---|:---|:---|
| **第二组分** | 烯烃 | 有机硼酸 |
| **产物** | 芳基烯烃（保留双键）| 双芳基（饱和连接）|
| **关键步骤** | 迁移插入 + β-H 消除 | 转金属 + 还原消除 |
| **是否有还原消除** | ❌ 无 | ✅ 有 |

### 7.2 Heck vs SEAr

| | **Heck** | **SEAr（芳香亲电取代）** |
|:---|:---|:---|
| **底物** | 芳基卤 + 烯烃 | 芳环 + 亲电试剂 |
| **产物** | 芳基烯烃 | 取代芳烃 |
| **机理** | Pd 催化周环型 | 亲电取代离子型 |
| **应用** | 烯基化芳环 | 官能团化芳环 |

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：写出 Heck 反应的催化循环，并指出与 Suzuki 的关键区别。

**解答**：
- Heck 循环：氧化加成 → 烯烃配位 → 迁移插入 → β-H 消除 → 碱脱 HX → Pd(0) 再生
- Suzuki 循环：氧化加成 → 转金属 → 还原消除 → Pd(0) 再生
- **关键区别**：Heck **没有还原消除**，产物双键来自 **β-H 消除**

### 10.2 例题 2（★★ 区域选择性）

**题目**：溴苯 + 丙烯酸甲酯的 Heck 反应产物是什么？

**解答**：
- 丙烯酸甲酯 = CH₂=CH-COOMe（EWG 在 α-位）
- 芳基倾向于连在 **β-位**（远离 EWG）
- 产物：**反式-肉桂酸甲酯衍生物**（Ph-CH=CH-COOMe）
- E 型为主

### 10.3 例题 3（★★★ 分子内 Heck）

**题目**：含有芳基碘和末端烯烃的同一分子发生分子内 Heck，可能生成什么？

**解答**：
- 分子内 Heck 构建环状结构
- 环大小取决于连接链长度
- 5–6 元环最易形成
- 产物：**芳基取代的环烯烃**

## 十一、易错点

1. ❌ **Heck 有还原消除** → Heck **没有还原消除**，产物双键来自 **β-H 消除**
2. ❌ **Heck 和 Suzuki 机理一样** → 机理**不同**，Heck 是迁移插入+β-H 消除
3. ❌ **芳基连在烯烃任意位置** → 有**区域选择性**（电子/位阻控制）
4. ❌ **烷基卤也能 Heck** → 烷基卤 **不适合**（β-H 消除不可控）
5. ❌ **Heck 产物是饱和的** → Heck 产物**保留双键**

### 12.3 入门例题

**题目**：为什么 Heck 反应的产物中总是保留一个双键？

**引导**：
- Heck 的最后一步是 **β-H 消除**
- β-H 消除在 Pd 的 β-碳上脱去一个氢
- 形成一个 C=C 双键
- Pd 带着 H 和 X 离去（被碱捕获）
- 所以产物中保留了烯烃的不饱和度
- 这也是 Heck 与 Suzuki 的根本区别（Suzuki 是还原消除，产物饱和）

### 12.4 现实类比

**Heck 像"嫁接树枝"**：
- 芳基卤 = 一根带芽的树枝（芽 = 卤素）
- Pd(0) = 园艺师
- 烯烃 = 主干树
- 氧化加成 = 园艺师摘下树枝上的芽
- 迁移插入 = 把树枝插入主干的适当位置
- β-H 消除 = 修剪多余的叶子，让嫁接处保持"活性"（双键）
- Suzuki 像"把两根树枝捆在一起"（还原消除 = 捆紧，没有活性芽）

1. **不对称 Heck 反应**：
