# [2+2] 环加成（[2+2] Cycloaddition）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[49-周环反应]]

---

## 一、定义

**[2+2] 环加成**：两个含双键的分子（各提供 2 个 π 电子）发生环加成，生成 **环丁烷** 或 **环丁烯酮** 衍生物的反应。

**通式**：
$$
\ce{烯烃 A + 烯烃 B -> 环丁烷衍生物}
$$

**本质**：**四电子环加成**——两个 2π 体系 → 一个 4 元环产物。

## 三、核心原理

### 3.1 Woodward-Hoffmann 规则

| 条件 | 允许方式 | 说明 |
|:---|:---|:---|
| **热反应** | ❌ 禁阻（同面-同面） | 轨道对称性不匹配 |
| **光反应** | ✅ 允许（同面-同面） | 激发态 HOMO 相位改变 |
| **热反应** | ✅ 允许（同面-异面） | 几何上通常不利 |

**核心原因**：
- 两个烯烃 HOMO 的末端瓣相位**相反**
- 同面-同面重叠 → 一个同相、一个反相 → **净禁阻**
- 光照 → 激发态 HOMO（原 LUMO）相位**相反** → 同面-同面允许

### 3.2 烯酮例外（Ketene [2+2]）—— Clayden 详细机理

**烯酮（R₂C=C=O）和异氰酸酯（R-N=C=O）的 [2+2] 环加成**：
- **热允许**！这是 [2+2] 的重要例外
- 产物：**β-内酰胺**（烯酮 + 亚胺 / 异氰酸酯 + 烯烃）或 **环丁酮**（烯酮 + 烯烃）

**为什么是例外？——正交 π 体系与 2s+2a 途径（Clayden Ch34）**
- 烯酮的 C=C=O 体系具有**两根成直角的 π 键**（正交 π 体系）。
- 普通烯烃热 [2+2] 禁阻的原因：HOMO 与 LUMO 在一端成键、另一端反键。
- 如果将一个烯烃转到与另一个成 **90°** 的位置，并使 LUMO 的一端添加一个与其成直角的额外 p 轨道，则 HOMO 的两个轨道可以同时与 LUMO 及额外 p 轨道成键。
- 结果：**四个成键相互作用 + 两个反键相互作用** → 平衡偏向反应。
- 烯酮中心的 **sp 碳**恰好具有这根与 C=C 成直角的额外 π 键（C=O）！
- 因此烯酮的 [2+2] 通过 **2s+2a 途径**（一个组分同面、一个异面）热允许。
- ![[mineru/中文版clayden-chinese-34-36907-999_images/f6565aacec7199ce4b6b3669cd6d598896548f8f696d1869339a1f40279fcef2.jpg]]

| 反应 | 亲电烯烃组分 | 另一组分 | 产物 |
|:---|:---|:---|:---|
| Staudinger 合成 | 烯酮 | 亚胺 | β-内酰胺 |
| 环丁酮合成 | 烯酮 | 烯烃 | 环丁酮 |
| 异氰酸酯 [2+2] | 异氰酸酯 | 烯烃 | β-内酰胺 |

#### Staudinger 合成细节（Clayden Ch34）

- **二氯烯酮**可由二氯乙酰氯在叔胺（如 Et₃N）作用下经 ElcB 消除制得。
- 环戊二烯 + 二氯烯酮 → [2+2] 环加成（区域和立体专一性）。
- 环戊二烯上最亲核的原子添加到烯酮最亲电的原子上；由于环戊二烯双键为顺式，环交点处得**顺式**几何结构。
- 令人印象深刻的是：即使是这样极好的双烯，也不能使烯酮发生 D-A 反应——[2+2] 环加成一定快得多！
- 产物中的两个氯原子可通过 Zn/HOAc 还原脱除 → 环戊烯酮衍生物。
- ![[mineru/中文版clayden-chinese-34-36907-999_images/a89f5b3ccd38e3c0fee7877b38fc12154868413b4e8b8cb8c670968c162c4585.jpg]]
- ![[mineru/中文版clayden-chinese-34-36907-999_images/3817d8bbb3fdada74ddccce1e4bcf6523e1e27dfe3ac67a6e6927a3cfaef5039.jpg]]

#### 掩蔽烯酮（Masked Ketene Equivalents）（Clayden Ch34）

- 若想要 D-A 产物（六元环）而非 [2+2] 产物（四元环），可使用**不是烯酮但可在之后转化为酮的化合物**。
- **最重要的两种类型**：
  1. **硝基烯烃**：先进行 [2+2] 环加成，再通过 Nef 反应（NaOH）将硝基转化为酮。
  2. **氰乙醇酯（cyanohydrin ester）类化合物**：类似策略。
- ![[mineru/中文版clayden-chinese-34-36907-999_images/778391dd598762b73379f759ba1eca000d831df66145de066e75d42e5c012ce1.jpg]]

#### 异氰酸酯 [2+2] 合成 β-内酰胺（Clayden Ch34）

- **氯代硫酰异氰酸酯（chlorosulfonyl isocyanate）**是商业可得的强亲电异氰酸酯。
- 可与简单烯烃反应 → β-内酰胺；硫酰基可在温和条件下水解去除。
- 烯烃的 HOMO 与异氰酸酯的 LUMO 相互作用：最亲电的原子是羰基碳，烯烃末端原子进攻此处。
- **分步 vs 协同**：富电子烯烃（如烯醇醚、烯基硫醚）与异氰酸酯的反应可能变为**分步机理**——即使使用 E/Z 混合物的起始原料，产物也只有反式立体化学（立体选择性而非立体专一性），表明存在可自由旋转的中间体。
- ![[mineru/中文版clayden-chinese-34-36907-999_images/025aa95291d7dbe4ee5b7afd9f252572f898ee9b6ac09be14e70b34197e443d9.jpg]]
- ![[mineru/中文版clayden-chinese-34-36907-999_images/896cd5b5f7ba28d4d57e63ee91c4e846fdecf01276164f7bb394b2b0efc7dca9.jpg]]

### 3.3 光化学 [2+2]——Clayden 深入分析

**最常见的 [2+2] 实现方式**：
- 烯烃直接光照 → 激发单重态/三重态
- 激发态 HOMO（原 LUMO）与基态 LUMO 匹配
- 产物：**环丁烷衍生物**

**光激发机理**（Clayden Ch34）：
- 光将一个电子从 π 激发到 π* 轨道 → 激发态。
- 将一个烯烃的激发态与另一个烯烃的基态组合：
  - 混合两个 π 轨道 → 两个电子能量降低，一个电子能量升高
  - 混合两个 π* 轨道 → 一个电子能量降低，没有电子能量升高
  - 结果：**三个电子能量降低，一个电子能量升高** → 成键可以发生。
- ![[mineru/中文版clayden-chinese-34-36907-999_images/155c7f954a73cd4f4e2482e7bd4612684bb1665df5b5111c124beb7269deabc1.jpg]]
- ![[mineru/中文版clayden-chinese-34-36907-999_images/20c56b22ac2db61ad54ce82bef7ff7729a58803a2cb42ed53652ce1ba59b7b28.jpg]]

**为什么产物不回到起始原料？**（Clayden Ch34）
- 逆反应和正反应一样受控于 Woodward-Hoffmann 规则。
- 但产物已经失去了 π 键，没有低能空轨道可以让光将电子带入其中。
- 因此光化学反应的逆反应**不可能发生**——即使得到有张力的四元环，产物也是稳定的。

**光化学 [2+2] 的区域选择性**（Clayden Ch34）
- 观察到的区域选择性与热反应中 HOMO/LUMO 分析的预料**正好相反**。
- 原因：在激发态反应中，真正重要的是 **HOMO/HOMO 和 LUMO/LUMO 相互作用**。
- 烯烃 LUMO 轨道系数的大小正好与其 HOMO 的相反。
- 由于共轭烯酮被激发，它的 LUMO 中含有一个电子，正是两个 LUMO 之间的重叠形成了新键。
- **快速判断技巧**：画出与一般的 HOMO/LUMO 或弯曲箭头控制反应中所预料的**相反**的结果。
- ![[mineru/中文版clayden-chinese-34-36907-999_images/463f1ca24d9a49019c466d52b5990855d06b8cbefe595a57e26a75b11b333f3c.jpg]]
- ![[mineru/中文版clayden-chinese-34-36907-999_images/32f02a3d9b0eb4d8dd910e1f8f9112b9441d51f3af59a18805745e3eda14a25e.jpg]]

**分子内 [2+2]**：
- 同一分子中的两个双键
- 常用于构建**笼状结构**或**桥环**
- 合成中非常有用

## 四、关键结论

1. **[2+2] = 四电子环加成 → 环丁烷**
2. **热同面-同面禁阻**（轨道对称性）
3. **光同面-同面允许**（激发态 HOMO）
4. **烯酮是热允许例外**（2s+2a 途径）
5. **分子内 [2+2] 是合成笼状结构的利器**

## 五、常见分类或情形

### 5.1 按条件分类

| 类型 | 条件 | 允许性 | 例子 |
|:---|:---|:---:|:---|
| **热 [2+2]** | 加热 | ❌ 禁阻 | 普通烯烃 |
| **热 [2+2] 烯酮** | 加热 | ✅ 允许 | 烯酮 + 烯烃 |
| **光 [2+2]** | 光照 | ✅ 允许 | 烯烃光二聚 |
| **光敏化 [2+2]** | 光敏剂 + 光 | ✅ 允许 | 三重态 [2+2] |

### 5.2 按分子间/分子内分类

| 类型 | 特点 | 合成应用 |
|:---|:---|:---|
| **分子间** | 两个独立烯烃 | 环丁烷合成 |
| **分子内** | 同一分子的两个双键 | 桥环、笼状结构 |

## 六、适用条件与限制

1. **热 [2+2] 对普通烯烃禁阻**
2. **烯酮 [2+2] 热允许**——但底物需含 C=C=O
3. **光 [2+2] 需合适波长**——避免副反应
4. **分子内 [2+2] 需要合适间距**——两个双键需能接近
5. **立体化学复杂**——可形成多个立体异构体

---

## 七、常见比较与易混点

### 7.1 [2+2] vs [4+2]

| | **[2+2] 环加成** | **[4+2] 环加成（D-A）** |
|:---|:---|:---|
| **电子数** | 4e | 6e |
| **热允许** | ❌（烯酮例外）| ✅ |
| **光允许** | ✅ | ✅（但少见）|
| **产物** | 环丁烷 | 环己烯 |
| **应用频率** | 较少（需光/烯酮）| 极多 |

### 7.2 热禁阻 vs 光允许的原因

- 热禁阻：基态 HOMO-HOMO 相互作用对称性不匹配
- 光允许：激发态 HOMO（原 LUMO）改变了相位关系
- 这是 Woodward-Hoffmann 规则的典型体现

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：两个乙烯分子在加热条件下能否发生 [2+2] 环加成？

**解答**：
- 两个乙烯 → 各 2π，总共 4e
- 热同面-同面 → 轨道对称性**禁阻**
- **不能**在热条件下发生 [2+2] 环加成
- 但光照条件下可以

### 10.2 例题 2（★★ 烯酮例外）

**题目**：烯酮与烯烃在加热条件下的反应产物是什么？

**解答**：
- 烯酮具有正交 π 体系
- [2+2] 可通过 2s+2a 途径热允许
- 产物：**环丁酮衍生物**
- 这是 [2+2] 的重要合成应用

### 10.3 例题 3（★★★ 分子内）

**题目**：1,6-环辛二烯光照后的产物是什么？

**解答**：
- 分子内两个双键
- 光化学 [2+2] → 分子内环加成
- 产物：**四环 [4.2.0.0²,⁵.0³,⁸] 辛烷**（笼状结构）
- 这是光化学构建笼状结构的经典例子

## 十一、易错点

1. ❌ **[2+2] 热允许** → 普通烯烃的 [2+2] **热禁阻**
2. ❌ **[2+2] 和 [4+2] 都是热允许** → [2+2] 热禁阻，[4+2] 热允许
3. ❌ **烯酮 [2+2] 也禁阻** → 烯酮是**热允许例外**
4. ❌ **光 [2+2] 随便照就行** → 需**合适波长**避免副反应
5. ❌ **所有 [2+2] 都成环丁烷** → 烯酮 + 亚胺 → **β-内酰胺**

### 12.3 入门例题

**题目**：为什么 [2+2] 环加成在热条件下禁阻，但在光条件下允许？

**引导**：
- 画两个烯烃的 HOMO
- 同面-同面重叠：一个瓣同相、一个瓣反相
- 净效果：反键相互作用 → 禁阻
- 光照：电子跃迁 → 激发态 HOMO 是原 LUMO
- 激发态 HOMO 的末端相位与基态相反
- 现在同面-同面 → 两个瓣都同相 → 允许

### 12.4 现实类比

**[2+2] 像"两人背靠背坐下"**：
- 两个人（两个烯烃）各伸出两只手（两个 p 轨道瓣）
- 热反应：两人面朝同一方向 → 左手拉左手可以，右手拉右手不行 → 无法同时手拉手坐下
- 光照：突然一个人转身（激发态）→ 现在可以四只手同时拉住 → 背靠背坐成一圈（环丁烷）
- 烯酮例外 = 其中一人是柔体操运动员，可以扭曲身体实现特殊姿势

## 十三、竞赛拓展

1. **Paternò–Büchi 反应**：
   - 羰基与烯烃的 [2+2] 光化学环加成
   - 产物：氧杂环丁烷（oxetane）
   - 是 [2+2] 在合成中的重要变体

2. **de Mayo 反应**：
   - 1,3-二酮与烯烃的光化学 [2+2]
   - 产物经逆 aldol 裂解生成 1,5-二酮
   - 用于天然产物合成

3. **掩蔽烯酮策略**（Clayden Ch34）：
   - 硝基烯烃或氰乙醇酯作为烯酮等价物
   - 先 [2+2] 环加成，再转化硝基为酮（Nef 反应）
   - 用于构建环丁酮骨架

4. **异氰酸酯 [2+2]**（Clayden Ch34）：
   - 氯代硫酰异氰酸酯与烯烃 → β-内酰胺
   - 硫酰基可水解去除
   - 富电子烯烃可能经历分步机理
