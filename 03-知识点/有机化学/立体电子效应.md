# 立体电子效应

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[21-杂化轨道理论与成键]]、[[22-分子轨道理论]]、[[28-有机物结构的稳定性]]、[[29-共振论]]、[[32-有机反应机理]]

## 一、定义

**立体电子效应（Stereoelectronic Effect）**：当分子内的轨道相互作用（[[给体-受体相互作用]]）受**几何位形**显著影响、并由此决定**构象偏好或反应选择性**时，所产生的总效应。

> 一句话：**几何决定轨道重叠，轨道重叠决定能量与活性**——立体电子效应就是把"几何 → 电子离域 → 现象"这条因果链显式化的命名。

### 1.1 与近邻概念的关系

| 概念 | 着眼点 | 与本 KP 的关系 |
|:---|:---|:---|
| [[给体-受体相互作用]] | 轨道重叠的能级与波瓣条件 | **底层物理**——立体电子效应是其几何依赖维度 |
| [[反式共平面]] | 轨道重叠的最优几何 | **几何前提**——立体电子效应最常见的入口 |
| [[超共轭效应]] | σ→σ*/σ→π*/n→σ* 的电子离域 | **典型子集**——多数立体电子效应可还原为超共轭 |
| [[共振论]] | 用极限式描述电子离域 | **不同语言**——共振论难以解释立体电子效应（无几何信息）|

### 1.2 历史脉络

**Pierre Deslongchamps**（1980s）在专著 *Stereoelectronic Effects in Organic Chemistry*（1983）中系统化"Stereoelectronic Effects"概念，将散落各处的"异头效应""β-消除立体专一性""羰基加成方位"等现象统一为同一物理图景：**几何决定电子。**

ABOC 教材进一步把它置于「给体-受体相互作用」的框架下，使之与超共轭、共振论彻底接轨。

## 三、核心原理

### 3.1 [[反式共平面]]：立体电子效应的"通用钥匙"

> ⚠️ **请不要对"反式共平面"这个词进行缩写！**（ABOC 原文强调三次）

任何 σ-σ\* / σ-π\* / n-σ\* / n-π\* 的电子离域要达到最大：

1. 给体与受体之间的二面角 **≈ 180°**（反式共平面）
2. 给体波瓣大的一端朝向受体波瓣大的一端
3. 给体能级 < 受体能级

**重叠强度的角度依赖**（cos²θ 关系）：

| 二面角 | 名称 | 重叠相对强度 | 立体电子效应 |
|:---:|:---|:---:|:---|
| 180° | 反式共平面（antiperiplanar）| **1.00** | 最强 |
| 120° | 反式邻位交叉 | ~0.25 | 较弱 |
| 90° | 垂直 | **0**（节面阻断）| 无 |
| 60° | 顺式邻位交叉（gauche）| ~0.25 | 较弱 |
| 0° | 顺式共平面（synperiplanar）| ~0.5（小波瓣相邻）| 较弱 |

> 核心数学：稳定化能 $\Delta E \propto |H_{ij}|^2 / \Delta E_{\text{orb}}$，重叠矩阵元 $H_{ij} \propto \cos^2\theta$。

### 3.2 给体-受体能级要求

立体电子效应不是只看几何——还要看能量匹配。即使几何理想，若给体能级远低于受体能级，作用也微弱。

**典型给体强弱**（[[给体-受体相互作用]] §4.1 摘要）：

$$\text{F-C} < \text{O-C} < \text{N-C} < \text{C-C} < \text{H-C} < \text{S-C} < \text{P-C} < \text{Si-C} < \text{Li-C}$$

**典型受体强弱**：

$$\text{C-F} > \text{C-O} > \text{C-N} > \text{C-C}\ ;\quad \text{C-I} > \text{C-Br} > \text{C-Cl} > \text{C-F}$$

### 3.3 静电相互作用：立体电子效应的"补丁"

当分子中存在**强极性键**（C-F、C=O、C-N⁺ 等）时，**静电（偶极-偶极、电荷-偶极）作用**也是构象决定因素，且**典型能量比超共轭大一个数量级**：

| 因素 | 本质 | 典型能量 |
|:---|:---|:---:|
| 静电相互作用 | 库仑力（偶极-偶极、电荷-偶极）| ~30-50 kJ/mol |
| 扭转张力 / $A^{1,3}$ | 经典空间排斥 | ~10-16 kJ/mol |
| 立体电子效应（超共轭）| σ → σ\* / n → σ\* 重叠 | ~3-10 kJ/mol |
| 氢键 | X-H···Y 静电 + 轨道 | ~3-20 kJ/mol |

> ABOC 注：当强极性键存在时，**静电相互作用通常压倒超共轭效应**，但二者**方向往往一致**（都倾向于反式共平面）——所以即使能量级差很大，常常**不矛盾**。

### 3.4 几何路径示意：从底物构象到产物结构

```
        构象 A                构象 B
     ┌──────────┐          ┌──────────┐
     │ 反式共平面│  ⟷      │ 顺式 / gauche│
     └────┬─────┘          └────┬──────┘
          │                     │
          ▼                     ▼
       超共轭最大              超共轭弱
       静电匹配                静电不利
          │                     │
          ▼                     ▼
       反应/构象偏 A         反应/构象偏 B
```

## 四、关键结论

### 4.1 七大典型立体电子效应

ABOC 体系下「立体电子效应」覆盖以下七大具体效应（每一项都是"几何依赖的轨道相互作用"的具体显化）：

| 编号 | 名称 | 几何要求 | 关键给体-受体 | 现象 |
|:---:|:---|:---|:---|:---|
| **(1)** | gauche effect | 1,2-二氟乙烷 gauche 二面角 | σ(C-H) → σ\*(C-F) | 反直觉 gauche > anti |
| **(2)** | 异头效应（[[端基效应]] 子集）| C1 取代基直立键 | n(O) → σ\*(C-O') | 糖化学 α-异头体优先 |
| **(3)** | [[端基效应]]（广义）| Y-C-Z 反共平面 | n(Y) → σ\*(C-Z) | 酰胺 sp³ 杂化反常态 |
| **(4)** | Bent's Rule | sp 轨道成分调整 | 杂化重新分配 | 电负性取代基倾向占用 p 性多的轨道 |
| **(5)** | [[β-硅基效应]] | C-Si 与 p+ 反共平面 | σ(C-Si) → p(C+) | 碳正离子稳定 10¹⁰ 倍 |
| **(6)** | Bürgi-Dunitz 角 | Nu→C=O 的 ~107° 进攻角 | n(Nu) → π\*(C=O) | 羰基加成立体偏好 |
| **(7)** | E2 反式共平面 | β-H 与 C-LG 反共平面 | σ(C-H) → σ\*(C-LG) | E2 立体专一性 |

### 4.2 静电主导 vs 立体电子效应主导

| 体系 | 主导因素 | 偏好构象 |
|:---|:---|:---|
| 1,2-二氟乙烷 | **立体电子效应**（gauche effect）| gauche |
| 1,2-二溴乙烷 | 空间位阻 | anti |
| α-氟代酰胺 | **静电相互作用**（C-F 与 C=O 偶极对）| anti（远离）|
| 含 NH₃⁺-C-C-F | **电荷-偶极吸引**（强）| gauche |
| 环己基取代 | 空间位阻 + 1,3-双轴排斥 | equatorial |

> 判断"哪个主导"的快速准则：① 看分子中是否有强极性键 → 有则静电先来；② 再看几何是否允许有效轨道重叠 → 允许则超共轭微调。

### 4.3 立体电子效应的能量量化（NBO 摘要）

| 立体电子效应 | 典型 $E^{(2)}$（kcal/mol）|
|:---|:---:|
| n(O) → σ\*(C-O)（异头）| 1-3 |
| σ(C-H) → σ\*(C-F)（gauche）| 0.5-1 |
| σ(C-H) → π\*(C=O)（α-π 超共轭）| 2-4 |
| n(N) → π\*(C=O)（酰胺平面性）| 60-80 |
| σ(C-Si) → p(C⁺)（β-Si）| 30-40 |

## 五、常见分类或情形

### 5.1 gauche effect — 1,2-二氟乙烷

1,2-二氟乙烷中 **gauche 比 anti 稳定 ~3.4 kJ/mol**，反直觉（按空间位阻应该 anti 占优）。

**机理**：

$$\sigma(\text{C-H}) \longrightarrow \sigma^*(\text{C-F})$$

- gauche 构象下，C-H σ 与远端 C-F σ\* 反式共平面
- anti 构象下，反而是 C-F 自己与远端 C-F σ\* 反共平面（弱给体！）—— 不利
- 因此 **gauche 通过 σ-σ\* 超共轭赢出**

> ⚠️ 注意：1,2-二溴乙烷不出现此效应——Br 体积大，空间位阻压倒 gauche effect。

### 5.2 异头效应（[[端基效应]] 子集，糖化学）

D-葡萄糖吡喃糖中：
- C1 上的 OAc/OMe 在 **α-异头体**（直立键，axial）
- 按空间位阻判断，应当偏好平伏键（β-异头体，equatorial）
- 实测 α : β ≈ 36 : 64（虽然 β 占多数，但 α 比"位阻预测"多得多）

**机理**：环氧 O 的孤对电子 → C1-O' σ\*

```
        n(O环氧)                     C1-O'
              ↘                    ↗ (σ*)
                C1 ── O' (axial)
              ↗
            n(O环氧) 反共平面 with C1-O' 仅在 axial 时
```

### 5.3 [[端基效应]]（广义 Y-C-Z 体系）

当 N 或 C 上同时连两个电负性大的杂原子（Y, Z），形成 Y-C-Z 体系：
- N(OR)(OR') 类酰胺
- 端基效应使 N 杂化 sp³ 而非 sp²
- N 上可发生亲核取代

详见 [[端基效应]] KP。

### 5.4 Bent's 规则

**Bent's Rule**：原子杂化时，**电负性大的取代基倾向占用 p 性更多的杂化轨道**；电负性小的取代基占用 s 性更多的轨道。

**应用**：
- CHCl₃ 的 H-C-Cl 键角 < 109.5°（H 占用更多 s 性，Cl 占用更多 p 性）
- 苯环上 ipso 碳的内 / 外角度差异

> Bent's Rule 是"立体电子效应"在杂化层面的体现——给体-受体相互作用使原子重新分配 s/p 成分。

### 5.5 [[β-硅基效应]]（参 [[β-硅基效应]] KP）

C-Si σ 给电子能力极强，使邻位碳正离子稳定性增加 10¹⁰ 倍（Lambert 实验，1999）。

### 5.6 Bürgi-Dunitz 角（羰基亲核加成）

亲核试剂进攻 C=O 的最优角度 = **107° ± 5°**（不是 90° 也不是 180°）。

**几何来源**：
- C=O 的 π\* 在 C 上有大波瓣（电负性大的 O 拉低反键）
- 但波瓣并非垂直于 C=O 轴——它**向 O 后方倾斜**
- 亲核试剂沿该倾斜方向进攻 → 重叠最大

**应用**：[[Felkin-Anh模型]] 中羰基 α 碳手性诱导的几何分析基础。

### 5.7 E2 反式共平面消除

ABOC §1.5/Ch.7：

> E2 反应**必须**在 β-H 与 C-LG 反式共平面构象下发生，否则消除不成立。

**机理**：

$$\sigma(\text{C-H}_{\beta}) \xrightarrow{\text{反共平面}} \sigma^*(\text{C-LG}) \quad\Rightarrow\quad \text{断 σ → 形成 π}$$

**经典反例**：环己基季铵碱热消除——若必需 anti H 处于 axial 受阻，**给出反 Saytzeff 产物**（多取代但不能反共平面）。

### 5.8 Baldwin 规则（分子内关环可行性）

**Baldwin 规则**是判断分子内亲核关环可行性的核心工具，本质是**立体电子效应在环化反应中的延伸**：亲核试剂进攻轨道的空间可达性决定关环能否发生。

**分类体系**：环大小 + endo/exo + 亲电试剂杂化态（tet/trig/dig）

| 规则 | 结论 | 竞赛重点 |
|:---|:---|:---|
| exo-tet | 全部有利 | — |
| exo-trig | 全部有利 | — |
| endo-tet | 5和6元不利 | — |
| **endo-trig** | **3、4、5元不利；6和7元有利** | ⭐ |
| endo-dig | 全部有利 | — |
| exo-dig | 3和4元不利；5-7元有利 | — |

> **最关键规则：5-endo-trig 是不利的！**

**endo/exo 判定**：
- **endo**：断开的键在**环内**
- **exo**：断开的键在**环外**

**tet/trig/dig 判定**：
- **tet** = sp³ 杂化（四面体，如饱和碳上的 SN2）
- **trig** = sp² 杂化（三角，如烯烃/羰基的共轭加成）
- **dig** = sp 杂化（直线，如炔烃的加成）

**轨道解释**：
- 5-endo-trig 不利：亲核试剂的 HOMO 与亲电中心（sp² 碳）的 LUMO 在空间上难以达到有效重叠——亲核试剂必须从π体系"背后"进攻，但5元环的几何约束使进攻角偏离最优轨迹。
- 5-exo-trig 有利：亲核试剂从外侧进攻 sp² 碳，几何上允许 Bürgi-Dunitz 类型的轨道重叠。

**例外情况**（Clayden 第31章）：
- 热力学驱动极强时（平衡可逆转）
- 阳离子中间体（非协同机理，不受轨道可达性约束）
- 周环反应（用 Woodward-Hoffmann 规则判断）
- 含第三周期元素（如硫）时：S 的 3d 轨道扩展了进攻角度范围

> 竞赛中遇到 Baldwin 规则题，先标出 endo/exo 和 tet/trig/dig，再查表判断。重点记住 **"5-endo-trig 不利，6-endo-trig 有利"**。

### 5.9 饱和杂环的亲核性增强（构象锁定效应）

饱和杂环中，杂原子孤对电子的取向被环构象**锁定**，产生反常的亲核性：

| 杂环 | 特征 | 原因 |
|:---|:---|:---|
| 哌啶、吡咯烷 | 比非环状胺更亲核 | 烷基被"绑在后面"，避开孤对电子 |
| 奎宁环 | 反应速率比三乙胺高 63 倍 | 桥连结构锁定构象，孤对电子完全暴露 |
| DABCO | 反应速率比三乙胺高 40 倍 | 桥连哌嗪，离去能力也较好 |

**本质**：非环状胺中 C-N 键可自由旋转，烷基可短暂遮挡孤对电子；环状胺中构象锁定使烷基远离亲核中心，有效碰撞频率提高。

### 5.11 α-氟代酰胺构象（39 届真题）

α-氟代酰胺 anti vs syn 能量差达 **~33.6 kJ/mol**：

- C-F 键和 C=O 键均为强极性键
- anti 构象：两极性键反式共平面 → 偶极矩部分抵消
- 含 N 的 +M 共振增强 C=O 极性 → anti 偏好放大

**酰胺 > 酯 > 酮 > 醛**：羰基极性递减 → 能量差递减（33.6 → 18.9 → 9.2 → 7.1 kJ/mol）。

> 该案例是"静电主导的立体电子效应"经典——构象偏好不是来自轨道重叠，而是**偶极矩排列**。

### 5.12 电荷-偶极吸引（含 -NH₃⁺）

含 $\mathrm{NH_3^+}$ 和 F 的体系中，**gauche 构象因 C-F(δ⁻) 与 N-H(δ⁺) 静电吸引而显著稳定**——电荷-偶极作用比偶极-偶极更强。

## 六、适用条件与限制

### 6.1 推荐使用场景
- ✅ 含强极性键的构象偏好分析
- ✅ 反应立体专一性（E2 反共平面、SN2 后侧进攻）
- ✅ 立体选择性合成（Felkin-Anh、Bürgi-Dunitz）
- ✅ 杂原子化合物的反常构象（HOOH、NH₂OH、N₂F₂）
- ✅ 糖化学异头偏好
- ✅ 分子内关环可行性判断（Baldwin 规则）
- ✅ 饱和杂环构象分析（Karplus 关系 + NOE）

### 6.2 不推荐使用场景
- ❌ 离子键体系（电子定域，无轨道相互作用）
- ❌ 高度对称的全 C-C 体系（电子分布对称，立体电子效应弱）
- ❌ 溶液中需考虑溶剂化时（气相规则可能反转）
- ❌ 周环反应（要用 [[FMO理论]] / Woodward-Hoffmann 而非立体电子效应）

### 6.3 饱和杂环的构象与 NMR 分析（Clayden 第31章）

饱和杂环的构象分析是立体电子效应的重要应用场景。NMR 提供了两种互补的几何信息来源：

#### 6.3.1 Karplus 关系（³J 与二面角）

³J 偶合常数与二面角 φ 的关系（Karplus 方程）：

$$^3J = A \cos^2\phi + B \cos\phi + C$$

**关键数值**（Clayden 第31章）：

| 二面角 φ | ³J 相对大小 | 构象含义 |
|:---:|:---:|:---|
| 0° | 最大（~8-10 Hz）| 顺式共平面 |
| 60° | 中等（~2-4 Hz）| 顺式邻位交叉 |
| 90° | ≈ 0 Hz | 垂直（节面阻断）|
| 120° | 中等（~2-4 Hz）| 反式邻位交叉 |
| 180° | 最大（~8-14 Hz）| **反式共平面** |

**在饱和杂环中的应用**：

| 环大小 | 特征偶合模式 |
|:---|:---|
| 三元环 | 顺式 > 反式，所有偶合都小（环张力使键角偏离理想值）|
| 四元环 | 顺式 > 反式 |
| 五元环 | 顺式 ≈ 反式（约 8-9 Hz）——信封式构象使二面角分散 |
| 六元环 | 直立-直立（~10-14 Hz）> 直立-平伏（~2-5 Hz）> 平伏-平伏（~2-3 Hz）|

> 六元环中 **ax-ax > ax-eq > eq-eq** 是椅式构象的"指纹"——可直接从 ³J 判断取代基取向。

#### 6.3.2 NOE（核的欧沃豪斯效应）

NOE 提供**空间邻近信息**（非沿键关系），与 Karplus（沿键二面角）互补：

- **原理**：照射某核使其饱和，通过偶极-偶极弛豫使空间邻近核的信号增强
- **增强程度 ∝ 1/r⁶**——只对近距离（< 5 Å）核敏感
- **应用**：
  - 确定烯烃几何（E/Z）：Z 式烯烃上同侧质子有 NOE，E 式无
  - 确定杂环三维立体化学：如糖化学中 C1-H 与 C3-H 的 NOE 可确认异头构型
  - 区分椅式/船式构象

> 竞赛中通常不要求 NOE 的定量计算，但需知道其提供"空间邻近"信息，可与 Karplus 联用解决构象问题。

### 6.4 与 [[共振论]] 的边界

立体电子效应**无法用共振论描述**：共振结构没有几何信息。例如 1,2-二氟乙烷 gauche 偏好——纸面共振结构看不出与 anti 的区别，必须用立体电子效应（轨道重叠几何）解释。

> **一句话边界**：共振论看"电子分布"，立体电子效应看"几何下的电子分布"。

### 6.5 环的开环与关环反应（Clayden 第31章）

#### 6.5.1 关环速率顺序

分子内关环的速率顺序（熵 + 焓综合）：

> **5 > 6 > 3 > 7 > 4 > 8-10**

| 因素 | 解释 |
|:---|:---|
| **焓（ΔH‡）** | 小环（3、4元）有环张力，提高活化能 |
| **熵（ΔS‡）** | 大环需更多有序化，熵减不利 |
| **五元环最快** | 环张力显著减小，两端仍不太远 |
| **四元环最慢** | 既有环张力又需较大有序化 |

**与 Baldwin 规则的关联**：
- 速率顺序是**热力学/动力学综合**结果
- Baldwin 规则是**轨道可达性**判断——即使 5-exo-trig 速率最快，若轨道几何不允许（如 5-endo-trig），反应仍不能发生

#### 6.5.2 Thorpe-Ingold 效应（偕二甲基效应）

- **现象**：环上取代基（尤其偕二甲基）增加关环速率和平衡常数
- **原因**：
  1. 小环中：取代基减小键角，缓解环张力
  2. 大环中：取代基限制构象自由度，减小活化熵损失（关环时"需要有序化"的代价降低）

> Thorpe-Ingold 效应是合成设计中常用的"关环促进策略"——在底物上引入偕二甲基可显著提高关环产率。

## 七、常见比较与易混点

### 7.1 立体电子效应 vs 立体效应（位阻）vs 电子效应

| 类型 | 来源 | 几何依赖性 | 例 |
|:---|:---|:---:|:---|
| **立体（位阻）效应** | 经典空间排斥 | 高 | tBu vs Me 在亲核取代中的差异 |
| **电子效应** | +I/-I、+M/-M、超共轭 | 低（取决于具体效应）| Cl 的 -I 比 H 强 |
| **立体电子效应** | 轨道相互作用 + 几何 | **核心** | gauche effect、异头效应 |

### 7.2 立体电子效应 vs 静电相互作用

| 维度 | 立体电子效应 | 静电相互作用 |
|:---|:---|:---|
| 物理 | 轨道重叠（量子力学）| 库仑力（经典电磁）|
| 量级 | 1-10 kJ/mol（多数情况）| 10-50 kJ/mol（强极性键） |
| 距离 | 中等（1-2 键间隔最强）| 长（电场延伸）|
| 几何 | 反式共平面最优 | 偶极反向 / 电荷邻近最优 |

> 二者的偏好**经常一致**（都偏好反式共平面排极性键）—— ABOC 用此说明二者可统一为"分子内电子-几何相互作用"。

### 7.3 与同名效应的对比表

| 效应 | 具体子类 | 几何要求 | 给体 → 受体 |
|:---|:---|:---|:---|
| gauche effect | 1,2-difluoroethane | gauche（60°）| σ(C-H) → σ\*(C-F) |
| 异头效应 | C1-Y-C-Z 体系 | axial | n(Y) → σ\*(C-Z) |
| 端基效应 | Y-C-Z 内部 | sp³ N + anti | n(Y) → σ\*(N-Z) |
| Bürgi-Dunitz | C=O 加成 | ~107° | n(Nu) → π\*(C=O) |
| Felkin-Anh | α-stereocenter 加成 | 大基团反共平面 | σ(C-X) align with π\*(C=O) |
| E2 | β-H + LG | anti | σ(C-H) → σ\*(C-LG) |

## 十、例题

### 例题 1（39 届初赛 8.2.1，原 v1.0 案例）

**题目**：α-氟代酰胺 anti-1 和 syn-1 哪种更稳定？

```
         O                         O
         ║                         ║
     C—C—N                     C—C—N
     │                         │
     F                         F
   (anti)                     (syn)
   F 远离 C=O                F 靠近 C=O
```

**分析**：
- 主导因素：**静电相互作用**（C-F 和 C=O 都是强极性键）
- anti：两个偶极反向 → 排斥最小
- syn：两个偶极同向 → 排斥大

**解答**：anti-1 更稳定，能量差约 33.6 kJ/mol。N 的 +M 共振增强 C=O 极性（电荷分离极限式），使 C=O 偶极更大 → anti vs syn 能量差被放大。

**反思**：
1. 这是"静电主导的立体电子效应"——构象偏好不来自轨道重叠，而来自偶极矩排列。
2. **酰胺 > 酯 > 酮 > 醛**（33.6 → 18.9 → 9.2 → 7.1 kJ/mol）正比于羰基极性。
3. 教学陷阱：若用"反共平面 = 超共轭最强"硬套，会得到错误结论——**这里超共轭项小于静电项**。

### 例题 2（39 届初赛 8.3，原 v1.0 案例）

**题目**：含 NH₃⁺ 和 F 的丁基铵盐，gauche-3 vs staggered-3 哪个更稳定？

**分析**：
- 主导因素：**电荷-偶极吸引**（NH₃⁺ 是带正电，C-F 是偶极）
- gauche：N⁺-H...F-C 距离近 → 吸引强
- staggered（anti）：N⁺ 与 F 远离 → 吸引弱

**解答**：gauche-3 更稳定。

**反思**：
1. 注意"gauche effect"在不同体系的不同来源：
   - 1,2-二氟乙烷的 gauche → σ-σ\* 超共轭
   - 1-氨基-2-氟体系的 gauche → 电荷-偶极吸引
2. 同一构象偏好可能来自完全不同的机理——必须根据具体官能团判断。

### 例题 3（ABOC §1.4 自学练习，立体电子效应实操题）

**题目**：解释为什么 trans-2-甲基-N,N,N-三甲基环己基季铵碱在加热消除时给出 Hofmann 产物（少取代烯烃）而非 Saytzeff 产物。

**分析**：
- 标准 E2：要求 β-H 与 C-N⁺ 反共平面
- trans-2-甲基的椅式构象：N⁺ axial / 2-Me equatorial（or 反之）
- 检查 axial N⁺ 的两个 β-H：一个在 C2（被 Me 遮挡，equatorial 而非 axial）；一个在 C6（无取代基，axial）
- C2 上的 H 是 equatorial → **不与 N⁺ 反共平面** → 不能消除
- C6 上的 H 是 axial → **与 N⁺ 反共平面** → 可消除

**解答**：因 C2 方向 anti H 不可达，反应只能从 C6 方向消除 → 得到端烯（Hofmann 产物，少取代）。

**反思**：这是"立体电子效应**主导**反应选择性"的经典案例：哪怕热力学上 Saytzeff 产物更稳定，机理上无法走 → 必须走 Hofmann。

### 例题 4（Baldwin 规则应用，Clayden 第31章）

**题目**：判断以下分子内关环是否可行：4-羟基-2-丁烯酸甲酯在碱催化下形成五元环内酯。

**分析**：
- 底物：HO-CH₂-CH=CH-CO₂Me
- 关环方式：O⁻ 进攻 C=O 的邻位碳（即双键碳）
- 环大小：5 元环
- endo/exo：断开的 C=C π 键在环内 → **endo**
- 亲电中心杂化：双键碳为 sp² → **trig**
- 综合：**5-endo-trig**

**解答**：5-endo-trig → **不利**，该关环在 Baldwin 规则下被禁阻。

**变式思考**：若改为 4-羟基丁酸甲酯（饱和，HO-(CH₂)₃-CO₂Me）：
- O⁻ 进攻羰基碳
- 5 元环，exo（C=O 的 π 键在环外断裂），tet（羰基碳被加成后变为 sp³）
- **5-exo-tet → 有利** ✓

**反思**：
1. Baldwin 规则的核心是**轨道可达性**——不是热力学，是几何约束。
2. 同一底物，仅改变饱和度（烯 vs 饱和），Baldwin 分类完全不同。
3. 竞赛中遇到"能否成环"问题，先画机理箭头，再标 endo/exo + tet/trig/dig。

### 例题 5（Karplus 关系与构象，Clayden 第31章）

**题目**：某六元环化合物的 ³Jₐₓ₋ₐₓ = 10-14 Hz，³Jₐₓ₋ₑq ≈ 2-5 Hz，³Jₑq₋ₑq ≈ 2-3 Hz。解释这些数值并推断构象。

**分析**：
- 椅式构象中：
  - ax-ax 二面角 ≈ 180° → Karplus 最大 → 10-14 Hz
  - ax-eq 二面角 ≈ 60° → Karplus 中等 → 2-5 Hz
  - eq-eq 二面角 ≈ 60° → Karplus 中等 → 2-3 Hz

**解答**：这些数值是**椅式六元环的 NMR 指纹**，证实化合物采取椅式构象。

**延伸**：若某化合物 ³J 全部约 8-9 Hz，可能是什么环？
- 五元环信封式构象中，二面角分散在 0°-120° 范围，³J 趋于平均化 → **五元环特征**

**反思**：
1. Karplus 关系是"从数字到几何"的桥梁——竞赛中可能给偶合常数让推断构象。
2. 六元环 ax-ax > ax-eq ≈ eq-eq 是椅式的强证据。
3. NOE 可与 Karplus 互补：若两个氢 ³J 小（eq-eq）但有 NOE → 可能为 1,3-双直立关系（空间邻近但二面角约 60°）。

## 十一、易错点

### 11.1 把"反式共平面"缩写

**错误**：写"反共平面"或"反式"。

**正确**：必须保持"反式共平面"完整。"反式"未指定共平面，"反共平面"省了"式"字。ABOC 强调三次禁止缩写。

### 11.2 用空间位阻解释 gauche effect

**错误**：1,2-二氟乙烷的 gauche 偏好"是因为 F 体积小所以可以靠近"。

**正确**：F 体积虽小但偏好不源自位阻——是 σ(C-H) → σ\*(C-F) 超共轭主导。

### 11.3 忽视静电效应的优先级

**错误**：所有构象题都只用超共轭/立体电子效应分析。

**正确**：含强极性键时，先看静电——典型能量差 30-50 vs 3-10 kJ/mol。例如 α-氟代酰胺如果只用立体电子效应分析会得错。

### 11.4 把立体电子效应与立体（位阻）效应混淆

**错误**：把"位阻"现象都归到"立体电子效应"。

**正确**：
- 位阻 = 经典空间排斥（核-核+电子云排斥）
- 立体电子效应 = 几何依赖的轨道电子离域
- 二者**有时偏好同向**（如轴向取代位阻 + 1,3-双轴）但本质不同。

### 11.5 在共振结构中"看"立体电子效应

**错误**：用共振结构判断 1,2-二氟乙烷 gauche 偏好。

**正确**：共振结构没有几何信息。立体电子效应**必须用三维构象 + 轨道重叠**分析。

### 11.6 忘记几何节面

**错误**：当二面角 = 90° 时认为还有一些超共轭。

**正确**：θ = 90° 时 cos²θ = 0 —— **完全无重叠**（节面阻断）。

### 11.7 把 Bürgi-Dunitz 角写成 90° 或 180°

**错误**：直觉认为 Nu 应垂直进攻（90°）或正对进攻（180°）。

**正确**：Nu 与 C=O 轴的角度 ≈ **107°**——这是 π\* 在 C 上的最大波瓣方向，向 O 后方倾斜。

### 11.8 Baldwin 规则 endo/exo 判断错误

**错误**：把 endo 和 exo 搞混，或分不清 tet/trig/dig。

**正确**：
- **endo** = 断开的键在环内；**exo** = 断开的键在环外
- **tet** = sp³（四面体）；**trig** = sp²（三角）；**dig** = sp（直线）
- 记忆锚点：**5-endo-trig 不利，6-endo-trig 有利**

### 11.9 混淆关环速率与 Baldwin 规则

**错误**：认为"5元环关环最快"所以"5-endo-trig 应该有利"。

**正确**：
- 关环速率顺序（5 > 6 > 3...）是**热力学/动力学统计结果**
- Baldwin 规则是**轨道可达性**判断——即使 5 元环通常快，若几何不允许（5-endo-trig），反应仍被禁阻
- 二者互补：Baldwin 判断"能不能"，速率顺序判断"有多快"

### 11.10 忽视 Thorpe-Ingold 效应

**错误**：关环产率低时只考虑 Baldwin 规则，不考虑取代基效应。

**正确**：Thorpe-Ingold 效应（偕二甲基）可显著提高关环速率和平衡常数——合成设计中常用此策略。

|:---|:---|:---|:---|

| "α-氟代酰胺 anti 是因为反共平面超共轭强"| 想当然套用超共轭 | 主导是**静电**（偶极反向）；超共轭只是修正 | 看到强极性键先想静电 |
| "1,2-二氟乙烷 gauche 是因为 F 小不挤"| 用经典位阻直觉 | 是 σ-σ\* 超共轭：H-C(给体) 与 C-F(受体)反共平面 | 看 1,2-二溴乙烷反例（Br anti）|
| "立体电子效应都很弱（< 10 kJ/mol）"| 看 NBO 表觉得小 | β-硅基效应 30-40 kcal/mol，酰胺平面性 60-80 kcal/mol —— **强不强看体系** | "看具体给-受体的能级差" |
| "Bürgi-Dunitz 角是经验"| 没看到底层原理 | 来自 π\*(C=O) 在 C 端波瓣向 O 后方的几何倾斜 | 画出 π\* 形状即可推导 |
| "立体电子效应等于 Deslongchamps 命名的少数效应"| 历史认知 | ABOC 视角下任何"几何 → 轨道重叠 → 现象"都是立体电子效应 | "广义 = 几何依赖的电子离域" |
| "异头效应只在糖里"| 教材出现位置 | 异头效应是 [[端基效应]] 的子集，普适于 Y-C-Z 体系 | 看到"两个杂原子夹一个 C"就警觉 |

### 12.3 入门级例题（不同于第十节）

**题目**：1,2-二氟乙烷的最稳定构象是 anti 还是 gauche？请用立体电子效应解释。

**预期解答路径**：
1. 第一步看到题目应该想到：**这是一个构象比较问题**。
2. 然后联想到本 KP 的：**几何决定轨道重叠 → 决定能量**。
3. 画两个构象的 Newman 投影：
   - anti（180°）：F-C 与 F-C 反共平面
   - gauche（60°）：H-C 与 F-C 反共平面
4. 找给体-受体对：
   - anti 中 σ(C-F) → σ\*(C-F)：F-C 是**最差给体**（电负性大）→ 弱
   - gauche 中 σ(C-H) → σ\*(C-F)：H-C 是**好给体**，C-F 是**好受体** → 强
5. 应用得到：gauche 的超共轭强 → **gauche 更稳定**。
6. 验证：实验测得 gauche 比 anti 稳定 ~3.4 kJ/mol。✓

**老师可在课堂上引导的提问**：
- 学生若卡在第 1 步：问 "你直觉上哪个稳定？为什么？"（多数会说 anti，由此引出"反直觉"）
- 学生若卡在第 2 步：问 "立体电子效应的核心是什么？几何还是能量？"
- 学生若卡在第 3 步：让学生**亲手画 Newman 投影**，标出每根键的方向
- 学生若答错（选 anti）：让他/她对比 1,2-二溴乙烷（实测 anti），引导发现"位阻 vs 立体电子效应"的边界
- 学生若忘记画反共平面：问 "F 想给电子还是接电子？谁是给体谁是受体？"

### 12.4 与现实/直觉的连接

> 用学生熟悉的现象/比喻 anchor 这个抽象概念。

- **类比 1**：立体电子效应像**电磁波天线**——你转动天线（改变构象），收到的信号强度（能量稳定化）会大幅变化。180° 是"信号最强"方向。
- **类比 2**：反共平面要求像**伞兵跳伞**——伞兵（给体电子）必须从特定方向落入预设区域（受体反键波瓣），否则错过。
- **类比 3**：静电 vs 立体电子效应像**磁铁 vs 拼图**——磁铁吸引（静电）是宏观长程作用，拼图卡进（轨道重叠）需要精确几何匹配。
- **反例/佯谬**：
  - 表面上看 1,2-二氟乙烷 anti 应该最稳定（F 远离 F），但实际上 gauche 更稳——这是**反直觉**的标志，需要立体电子效应解释。
  - 表面上看糖类 OAc/OMe 应当 equatorial（位阻小），但实际上 axial 比例反常高 —— 异头效应。
  - 表面上看羰基亲核加成应当 90° 进攻（垂直），但实际上 107° —— Bürgi-Dunitz 角。

### 13.1 NBO 二阶微扰能量分析

NBO（Natural Bond Orbital）分析提供立体电子效应的定量化工具：

| 工具 | 用途 |
|:---|:---|
| Gaussian + NBO 6.0 | 计算 $E^{(2)}$ 二阶微扰稳定化能 |
| ORCA | 替代方案，开源 |
| MultiWFN | 后处理与可视化 |

典型操作：
1. DFT 优化几何（B3LYP/6-311+G\*\*）
2. NBO 关键字 `pop=nboread, $NBO E2PERT $END`
3. 输出文件中找 `Second Order Perturbation Theory Analysis`
4. 提取每对 donor-acceptor 的 $E^{(2)}$（kcal/mol）

### 13.2 Deslongchamps 立体电子效应历史

**Pierre Deslongchamps**（拉瓦尔大学，加拿大）：
- 1972 年发表"立体电子效应在水解中的作用"系列论文
- 提出 ester 水解中正交效应（"orthogonal effect"）
- 1983 年专著 *Stereoelectronic Effects in Organic Chemistry*（Pergamon）系统化
- 在乙缩醛/缩酮/糖化学领域奠定立体电子效应学科地位

**Eusebio Juaristi**（墨西哥）：
- 进一步将立体电子效应应用到含 N 杂环

### 13.3 Felkin-Anh 模型详解

α-stereocenter 的 RCH(X)CHO 加成中：
- 大基团 R_L 与 C=O **反共平面**（最佳超共轭排列）
- Nu 进攻**反 R_L 方向**
- 由此预测 newly formed center 立体化学

> 关键：σ\*(C-R_L) 是良好的"反共平面"对手，使 R_L 远离过渡态。

### 13.4 不止有机：在配位化学中的立体电子效应

- 18 电子规则中 d-orbital 的方向选择
- Buchwald 配体设计：磷的 cone angle 与 bite angle
- Schrock-type 烯烃复分解：β-H 消除的几何要求

- **Alabugin, I. V.** *Stereoelectronic Effects: A Bridge Between Structure and Reactivity*. Wiley, 2016. 现代综述
- **Cieplak, A. S.** *J. Am. Chem. Soc.* 1981, 103, 4540. 反向 Cieplak 模型
