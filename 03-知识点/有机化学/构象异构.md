## 一、定义

**构象异构（Conformational Isomerism / Conformation）**：由于 **σ 键旋转** 导致分子中原子或基团在空间的不同排布方式。不同构象之间**不需要断裂化学键**即可互相转化。

**关键特征**：
- 转化能垒低（~3–20 kcal/mol），常温下**无法分离**
- 各构象在平衡中以**玻尔兹曼分布**共存
- 最稳定构象（优势构象）在平衡中占比最高

## 三、核心原理

## 四、关键结论

1. **交叉式稳定的核心原因是超共轭，不是位阻**——ABOC 反复强调的"给体-受体相互作用"视角
2. **环己烷椅式是绝对优势构象**——扭船式能量高 5.5 kcal/mol，常温下占比 < 0.1%
3. **大基团优先平伏键**——由 A 值决定，t-Bu 的 A 值 ≈ 5 kcal/mol 使其几乎 100% 平伏
4. **1,3-双直立作用 > gauche 作用**——环己烷中的直立位阻比链状构象的 gauche 更严重
5. **环翻转不改变相对构型**——顺/反关系不变，只是 a/e 互换

## 五、常见分类或情形

## 六、适用条件与限制

1. **构象分析只适用于单键旋转**——双键构型（E/Z）属于构型异构，不是构象
2. **酰胺 C-N 键具有部分双键性**——旋转能垒高（~20 kcal/mol），构象可分离
3. **大位阻体系可能锁定构象**——如联苯类在邻位有大取代基时，旋转受阻
4. **计算构象能量需用分子力学/DFT**——简单 A 值估算对复杂体系误差大
5. **溶剂效应可改变构象分布**——氢键供体溶剂可能稳定某些特定构象

## 十、例题

## 十一、易错点

1. ❌ **认为重叠式不稳定是因为位阻** → 乙烷中主要是超共轭不利，H-H 位阻极小
2. ❌ **把构象异构当构型异构** → 构象不能分离，构型可以分离（对映体除外需手性环境）
3. ❌ **环翻转后顺反关系改变** → 环翻转只换 a/e，不改变相对构型（顺/反）
4. ❌ **A 值只适用于烷基** → OH、Br 等也有 A 值，只是较小
5. ❌ ** Newman 投影式画错前后碳** → 前面碳（圆心）的三键与后面碳（圆周）的三键要对应连接
6. ❌ **忽略 g+g⁻ 戊烷作用** → 连续两个 gauche 的位阻远大于单个 gauche
7. ❌ **认为船式从不出现** → 在某些多环体系中（如降冰片烷），船式是强制构象
8. ❌ **构象分析只用于环己烷** → 链状分子的构象同样重要（如蛋白质折叠中的扭转角）

## 十二、🎯 教学视角

### 12.1 学生常见误区

1. ❌ **"交叉式稳定是因为位阻小"** → 这是最常见的教科书级误解。实际上，乙烷交叉式的稳定根源是**超共轭**（σ(C-H)→σ*(C-H) 给体-受体相互作用），而非邻位 H-H 位阻。ABOC 和 Clayden 均反复强调此点——位阻解释在丁烷体系中尚可，但对乙烷完全失效。

2. ❌ **"环己烷就是平面六边形"** → 学生初学时常把环己烷画成平面正六边形，不理解为什么需要椅式。关键是：sp³ 碳的键角 ~109.5°，平面六边形内角 120°→严重角张力。椅式恰好使所有 C-C-C 键角接近 109.5°，且所有 H 处于交叉式排列。

3. ❌ **"构象和构型是同一个概念"** → 混淆两类异构。构象通过σ键旋转可互变（无需断键），构型需要断键再生成。考察方式：给一个环己烷衍生物，问 chair flip 后是同一分子还是对映体——这考的是 chair flip 不改变相对构型（顺/反关系不变）。

### 12.2 讲授主线

**视觉化驱动**：全程使用分子模型或 3D 可视化——构象是"用眼睛学"的知识，纯文字描述远远不够。

**四阶梯讲授**：
1. **乙烷谜题**（5 min）：先问学生"交叉式为什么稳定？"收集回答（必有人说位阻）→ 再揭示超共轭是真正原因 → 建立"给体-受体"思维方式
2. **丁烷与 gauche 效应**（10 min）：Newman 投影 + 能量曲线图，重点讲解 gauche 构象中 C-C σ 键与 C-H σ 键的超共轭
3. **环己烷椅式**（15 min）：从平面六边形的角张力问题出发→推导椅式→a/e 键→1,3-双直立作用→A 值概念。板书推演 chair flip 全过程（a→e, e→a）
4. **应用收束**（10 min）：E2 消除的 anti-periplanar 要求 + 糖的端基效应（葡萄糖为什么 α/β 比例不是 1:1）

### 12.3 跨学科/现实连接

1. **蛋白质折叠**：Ramachandran 图本质上是 φ/ψ 二面角的构象分析——二级结构（α-螺旋/β-折叠）对应特定的构象角范围
2. **药物设计**：构象锁定策略——用大基团或桥环强制药物分子采取活性构象，提高选择性
3. **高分子物理**：聚合物的自由旋转链模型、特性粘度、弹性——都建立在 C-C 键构象分析之上

## 十三、竞赛拓展

1. **环癸烷的"双椅"构象**：
   - 环癸烷最稳定构象不是简单的椅式，而是"双椅-扭船"杂化
   - 存在跨环氢相互作用（transannular H-H interaction）
   - 是中环化学（8-12 元环）的典型特征

2. **构象锁定（Conformational Locking）**：
   - 用大基团（如 t-Bu）或桥环强制分子采取特定构象
   - 在不对称催化中至关重要（手性配体通过构象锁定控制立体选择性）
   - 例：反-1,4-二叔丁基环己烷的构象锁定使 E2 消除立体专一

3. **糖的椅式构象与端基效应**：
   - D-葡萄糖吡喃糖的 ⁴C₁ 椅式构象中，所有大基团（OH、CH₂OH）都处于 e 键
   - 这是葡萄糖在自然界中最丰富的原因——热力学最稳定
   - 端基异构体（α/β）的活性差异源于端基 C-O 键的 a/e 取向

4. **蛋白质二级结构的构象角**：
   - α-螺旋：φ = -57°, ψ = -47°（扭歪但重复）
   - β-折叠：φ = -139°, ψ = +135°（伸展）
   - Ramachandran 图就是构象分析在生物化学中的延伸

### 3.1 乙烷构象 — 超共轭 vs 位阻

乙烷沿 C-C 键旋转产生两种极限构象：

| 构象 | 二面角 | 相对能量 | 稳定性 | 原因 |
|:---|:---:|:---:|:---:|:---|
| **交叉式（staggered）** | 60° | 0（基准）| 最稳定 | **3 组反式共平面**的给体-受体相互作用 |
| **重叠式（eclipsed）** | 0° | ~3.0 kcal/mol | 最不稳定 | H-H 电子云排斥 + 超共轭不利 |

> ⭐ **ABOC 行 1024 核心解释**："全交叉比全重叠稳定。大部分书中写这样位阻比较小，但是全重叠式中 C-C 键更长，这里两个 H 基本没有相互作用，因此并不是主要原因。真实的原因是什么？**全交叉构象具有三组反式共平面，三组碳氢键电子填入对方的反键轨道，形成 3 组给体-受体相互作用**，从而使电子离域。这就是给体-受体相互作用的核心！"

**关键结论**：乙烷交叉式的稳定性**主要不是位阻**（H-H 距离足够远），而是**超共轭效应（给体-受体相互作用）**！

### 3.2 丙烷与丁烷的构象

**丙烷**：与乙烷类似，交叉式稳定，但有一个 CH₃ 替代 H → 位阻略增。

**丁烷（沿 C2-C3 键）**：

| 构象 | 二面角 | 相对能量 | 名称 |
|:---|:---:|:---:|:---|
| 反交叉（anti） | 180° | 0 | **最稳定** |
| 旁交叉（gauche） | ±60° | ~0.9 kcal/mol | 次稳定 |
| 重叠式（eclipsed） | 0° | ~4.5 kcal/mol | 不稳定（CH₃-H 重叠）|
| 全重叠（fully eclipsed）| 120° | ~6.0 kcal/mol | 最不稳定（CH₃-CH₃ 重叠）|

**gauche 相互作用**：在 gauche 构象中，两个 CH₃ 相距 60°，存在 ~0.9 kcal/mol 的位阻排斥。

### 3.3 环己烷构象 — 椅式、船式、扭船式、半椅式

| 构象 | 相对能量 | 特征 | 稳定性 |
|:---|:---:|:---|:---:|
| **椅式（chair）** | 0 | 所有 C-C-C 键角 ~111°（接近理想 tetrahedral），无角张力 | **最稳定** |
| **扭船式（twist-boat）** | ~5.5 kcal/mol | 船式扭动缓解 flagpole 相互作用 | 次稳定 |
| **船式（boat）** | ~6.4 kcal/mol | 两个 flagpole H 相距 1.83 Å，存在强排斥 | 不稳定 |
| **半椅式（half-chair）** | ~10.0 kcal/mol | 一个 C 翻出平面，角张力最大 | 过渡态 |

**椅式构象的关键特征**：
- **直立键（axial, a）**：垂直于环的平均平面，交替上下
- **平伏键（equatorial, e）**：大致沿环外切线方向伸展
- 每个碳上：一个 a 键，一个 e 键
- 环翻转（ring flip）：a → e，e → a，但**相对构型（上/下）不变**

### 3.4 1,3-双直立作用（1,3-diaxial interaction）

当环己烷椅式上有**直立取代基**时，该基团与同一侧的两个直立氢存在**位阻排斥**：

```
       R (a键，向上)
       |
   H(a)─C─H(a)    ← 1,3-双直立排斥
       |
   H(a)─C─H(a)    ← 同一侧
```

**每个 1,3-双直立作用的能量代价**：

| 取代基 R | ΔG° (kcal/mol) |
|:---|:---:|
| -H | 0（基准）|
| -CH₃ | ~1.7 |
| -C₂H₅ | ~1.8 |
| -i-Pr | ~2.1 |
| -t-Bu | ~5.0 |
| -OH | ~0.9 |
| -Br | ~0.5 |

> ⭐ **A 值（A-value）**：取代基在直立键相对于平伏键的自由能差。A 值越大，该取代基越倾向于平伏键。

**结论**：大基团优先处于**平伏键**（e 键），以最小化 1,3-双直立排斥。

### 3.5 g+g⁻ 戊烷相互作用

在链状烷烃中，若连续两个二面角都是 gauche（同为 +60° 或同为 -60°），则会产生**g+g⁻ 戊烷相互作用**——链末端两个甲基相距很近，产生额外位阻。

**能量代价**：~3.5 kcal/mol（比单个 gauche 大得多）

**在环己烷中**：若两个相邻取代基都处于直立键（且同向），会触发类似 g+g⁻ 的强排斥。

> ⭐ **ABOC 行 1200**："g+g⁻戊烷相互作用(AAOC 中详细讨论)：大基团受到这种作用的影响比小基团更加大。"

### 7.1 构象异构 vs 构型异构

| | **构象异构（Conformation）** | **构型异构（Configuration）** |
|:---|:---|:---|
| 转化方式 | σ 键旋转 | 断键重排 |
| 能垒 | 低（3–20 kcal/mol）| 高（~>30 kcal/mol）|
| 可否分离 | **不能**（常温）| **能**（对映体需手性环境）|
| 例子 | 乙烷交叉/重叠 | R/S、E/Z |
| 数目 | 无限连续（旋转）| 有限离散 |

### 7.2 直立键 vs 平伏键的判别口诀

**"上上下下，平伏画外；直立冲天，交替排排"**

- 椅式中，相邻碳的直立键一上一下交替
- 平伏键大致"平"着向外伸展
- 环翻转后，原来的上-直立变成上-平伏（方向不变，键型变）

### 10.1 例题 1（★ 基础）

**题目**：画出丁烷沿 C2-C3 键的 Newman 投影式，标出 anti、gauche、重叠式，并排序稳定性。

**解答**：

```
anti (180°):           gauche (60°):         重叠式 (0°):
    CH₃                    CH₃                    CH₃
     |                      |                      |
    H─C─C─H              H─C─C─H              H─C─C─H
     |   \                  |  /                   |  |
    H     CH₃             H    CH₃               H   CH₃

最稳定                   次稳定                  最不稳定
```

稳定性：anti > gauche > 重叠式

### 10.2 例题 2（★★ 环己烷 A 值应用）

**题目**：顺-1,2-二甲基环己烷和反-1,2-二甲基环己烷哪个更稳定？

**解答**：
- **反式**：一个 CH₃ a，一个 CH₃ e → 1 个 1,3-双直立作用 → 代价 ~1.7 kcal/mol
- **顺式**（ee 构象）：两个 CH₃ 都 e → 无 1,3-双直立作用，但 gauche 丁烷作用 → 代价 ~0.9 kcal/mol
- **顺式**（aa 构象）：两个 CH₃ 都 a → 2 个 1,3-双直立作用 → 代价 ~3.4 kcal/mol

反式更稳定（主要存在 ee 构象，实际反式优势构象是一个 a 一个 e；顺式优势构象是两个 e 或两个 a 中的 ee）。

> ⭐ **注意**：反-1,2-二甲基环己烷的稳定构象是 **diaxial 或 diequatorial 取决于相对取向**。对于反式，优势构象是 **ee**（两个都平伏）—— 实际上反-1,2-二取代环己烷的优势构象是 ee（两个平伏），顺式则必须一个 a 一个 e。

让我重新仔细分析：
- 反-1,2：可以 ee（两个都平伏）→ **最稳定**
- 顺-1,2：必须一个 a 一个 e → 有一个 1,3-双直立作用 → 次稳定

所以 **反-1,2-二甲基环己烷 > 顺-1,2-二甲基环己烷**

### 10.3 例题 3（★★★ ABOC 例题 1.4.1 改编）

**题目**：设计一个含有单一六元环骨架的分子，使其**最稳定构象为船式**。

**思路**：
- 船式不稳定的主要原因是两个 flagpole H 的排斥
- 若把 flagpole 位置换成**成桥的亚甲基**（如降冰片烷骨架），则船式被"锁定"
- 另一个思路：在 1,4-位连两个大基团，椅式中产生极强的 1,3-双直立作用，迫使环转为船式

**解答示例**：**顺式-1,4-二叔丁基环己烷**
- 在椅式中，一个 t-Bu 必须直立 → A 值 ~5 kcal/mol 的代价
- 若两个 t-Bu 同向（都向上），椅式中一个 a 一个 e，代价 ~5 kcal/mol
- 船式构象可能使两个 t-Bu 都处于较有利位置 → 总能量反而更低
- 更经典的答案：**双环[2.2.2]辛烷的某个衍生物**，或 **1,4-桥连环己烷**（如 1,4-二氧六环的衍生物）

> ⭐ **ABOC 行 1212**：设计题，鼓励创造性破坏椅式构象。

### 10.4 例题 4（★★★ 构象与反应性）

**题目**：反-1-溴-4-叔丁基环己烷在碱作用下发生 E2 消除，产物是什么？解释构象控制。

**解答**：
- t-Bu 的 A 值 ~5 kcal/mol → **必须处于平伏键**（e 键）
- 因此环被锁定在一个特定椅式构象（t-Bu 向下 e 键）
- Br 在 C1，与 t-Bu 处于反式 → Br 必须处于**直立键**（a 键，向上）
- E2 要求 H 和 Br 反式共平面 → C2 或 C6 上的 H 必须处于直立（与 Br 反式）
- 由于构象锁定，只有一侧的 H 满足条件 → **立体专一性消除**

**结论**：大基团（t-Bu）的构象偏好"锁定"了环，使 E2 消除具有高度立体选择性。
