# Suzuki 偶联（Suzuki-Miyaura Coupling）
- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[50-有机合成]]、[[51-金属有机]]
---

## 一、定义

**Suzuki 偶联（Suzuki-Miyaura Coupling）**：在 **Pd(0) 催化剂**和**碱**存在下，**有机硼酸（或硼酸酯）**与**有机卤代物（或三氟甲磺酸酯）**发生**交叉偶联**，生成新的 **C–C σ 键**的反应。

**通式**：
$$
\ce{R-B(OH)2 + R'-X ->[Pd(0), 碱] R-R' + X-B(OH)2}
$$

**本质**：**最温和、最实用的 Pd 催化交叉偶联**——因硼酸的稳定性、低毒性和高官能团耐受性而成为竞赛和合成中的首选。

## 三、核心原理

### 3.1 催化循环

**Pd(0)/Pd(II) 循环四步曲**：

```
步骤 1：氧化加成（Oxidative Addition）
  Pd(0)L₂ + R'-X → R'-Pd(II)-X·L₂

步骤 2：转金属（Transmetalation）
  R'-Pd(II)-X + R-B(OH)₂ → R'-Pd(II)-R + X-B(OH)₂

步骤 3：还原消除（Reductive Elimination）
  R'-Pd(II)-R → R-R' + Pd(0)L₂（催化剂再生）
```

**关键点**：
- **Pd(0)** 是真正的催化剂，通常由 Pd(II) 前体（如 Pd(OAc)₂）原位还原生成
- 碱的作用：**活化硼酸**（形成 boronate）+ **捕获 HX**

### 3.2 碱的作用机理

**硼酸的活化**：
$$
\ce{R-B(OH)2 + OH^- -> R-B(OH)3^-}
$$
- "ate" 络合物（boronate）比中性硼酸**更富电子**
- 转金属更容易发生

**常用碱**：
| 碱 | 强度 | 适用性 |
|:---|:---|:---|
| **Na₂CO₃ / K₂CO₃** | 弱 | 最常用，官能团耐受好 |
| **Cs₂CO₃** | 弱 | 对难反应底物更有效 |
| **NaOH / KOH** | 强 | 活性高，但可能副反应 |
| **K₃PO₄** | 中等 | 无水条件常用 |
| **Et₃N** | 弱有机碱 | 对酸敏感底物 |

### 3.3 配体的选择

| 配体 | 类型 | 特点 | 应用 |
|:---|:---|:---|:---|
| **PPh₃** | 单齿膦 | 经典、便宜 | 常规 Suzuki |
| **SPhos** | 联苯单齿膦 | 大位阻、富电子 | 难氧化加成底物 |
| **XPhos** | 联苯单齿膦 | 比 SPhos 位阻更大 | 芳基氯化物 |
| **BrettPhos** | 联苯单齿膦 | 最高位阻之一 | 最惰性底物 |
| **dppf** | 双齿膦 | 螯合稳定 | 烯基/炔基硼酸 |

**配体效应**：
- **大位阻 + 富电子** → 加速氧化加成 → 反应更快
- 这就是为什么 SPhos/XPhos 比 PPh₃ 更高效

### 3.4 底物范围

**有机硼酸组分**：
- 芳基硼酸（最常用）
- 烯基硼酸
- 烷基硼酸（较难，需特殊配体）

**有机卤代物组分**：
- **芳基碘（Ar-I）**：最活泼
- **芳基溴（Ar-Br）**：常用
- **芳基氯（Ar-Cl）**：较惰性，需大位阻配体
- **芳基三氟甲磺酸酯（Ar-OTf）**：常用替代卤代物

**注意**：烷基卤代物通常不适合 Suzuki（β-H 消除竞争）。

---

## 四、关键结论

1. **Suzuki = 有机硼酸 + 有机卤代物 → 交叉偶联产物**
2. **Pd(0)/Pd(II) 催化循环**：氧化加成 → 转金属 → 还原消除
3. **碱活化硼酸**是转金属的关键
4. **大位阻富电子配体**加速反应
5. **Ar-I > Ar-Br > Ar-Cl** 反应性顺序
6. **最温和、最实用的交叉偶联**

## 六、适用条件与限制

1. **需要无氧条件**——Pd(0) 易被氧化失活
2. **需要碱**——活化硼酸 + 捕获 HX
3. **溶剂选择**：THF、二氧六环、甲苯、DMF、水（或混合溶剂）
4. **烷基卤代物不适用**——β-H 消除副反应
5. **芳基硼酸可能脱硼**——在强碱/高温下
6. **邻位大位阻底物反应慢**

## 七、常见比较与易混点

### 7.1 Suzuki vs Negishi vs Stille

| | **Suzuki** | **Negishi** | **Stille** |
|:---|:---|:---|:---|
| **有机金属试剂** | 硼酸 R-B(OH)₂ | 有机锌 R-ZnX | 有机锡 R-SnR'₃ |
| **稳定性** | ✅ 空气/水稳定 | ⚠️ 对水敏感 | ✅ 极稳定 |
| **毒性** | ✅ 无毒 | ⚠️ 低毒 | ❌ **高毒** |
| **反应性** | 中等 | **最高** | 中等 |
| **竞赛频率** | **最高** | 中 | 低 |

### 7.2 Suzuki vs Heck

| | **Suzuki** | **Heck** |
|:---|:---|:---|
| **有机金属试剂** | 需要（硼酸）| 不需要 |
| **另一组分** | 卤代物 | 烯烃 |
| **产物** | 双芳基/芳基-烯基 | 芳基烯烃 |
| **机理** | 转金属 + 还原消除 | 迁移插入 + β-H 消除 |

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：Suzuki 偶联中碱的作用是什么？

**解答**：
- **活化硼酸**：R-B(OH)₂ + OH⁻ → R-B(OH)₃⁻（boronate，更亲核）
- **捕获 HX**：还原消除前生成的 HX 被碱中和
- **促进转金属**：活化的 boronate 更容易与 Pd(II) 发生转金属

### 10.2 例题 2（★★ 配体选择）

**题目**：为什么芳基氯化物的 Suzuki 偶联需要 SPhos 或 XPhos 而不是 PPh₃？

**解答**：
- Ar-Cl 的 C–Cl 键比 Ar-Br/Ar-I 更强
- 氧化加成是决速步
- SPhos/XPhos 是**大位阻、富电子**配体
- 富电子 → 提高 Pd(0) 的电子密度 → 促进氧化加成
- 大位阻 → 加速还原消除（避免副反应）

### 10.3 例题 3（★★★ 合成设计）

**题目**：如何用 Suzuki 偶联合成 4-甲氧基联苯？

**解答**：
- 目标：Ph-Ph-OMe（对位）
- 策略：苯基硼酸 + 4-碘苯甲醚（或 4-溴苯甲醚）
- 条件：Pd(PPh₃)₄, Na₂CO₃, THF/H₂O, 回流
- 反应：Ph-B(OH)₂ + MeO-C₆H₄-Br → MeO-C₆H₄-Ph

## 十一、易错点

1. ❌ **Suzuki 不需要碱** → **必须加碱**活化硼酸
2. ❌ **烷基卤也能 Suzuki** → 烷基卤 **不适合**（β-H 消除）
3. ❌ **Suzuki 和 Heck 机理一样** → Suzuki 是**转金属+还原消除**，Heck 是**迁移插入+β-H 消除**
4. ❌ **所有硼酸都一样** → 烷基硼酸 **很难** Suzuki
5. ❌ **Pd(II) 是直接催化剂** → 真正催化剂是 **Pd(0)**，Pd(II) 是前体

### 12.3 入门例题

**题目**：为什么 Suzuki 偶联被称为"最温和的"交叉偶联？

**引导**：
- 硼酸对空气和水稳定 → 操作方便
- 硼酸无毒 → 安全
- 反应条件温和 → 官能团耐受性好
- 与其他有机金属试剂（有机锂/格氏试剂）相比，硼酸不活泼到"见官能团就反应"

### 12.4 现实类比

**Suzuki 像"牵线搭桥"**：
- Pd(0) = 媒人
- 硼酸 = 男方（稳定、靠谱、好说话）
- 卤代物 = 女方（有点脾气，需要媒人撮合）
- 碱 = 礼金（活化男方，让他更主动）
- 氧化加成 = 媒人先去女方家说媒
- 转金属 = 男女双方见面
- 还原消除 = 两人牵手成功，媒人功成身退
- 整个过程媒人不消耗（催化循环）

## 十三、竞赛拓展

1. **Buchwald-Hartwig 胺化**：
   - Suzuki 的"氮版本"——Pd 催化的 C–N 键形成
   - 用胺替代硼酸，合成芳胺
   - 2010 年诺贝尔化学奖相关

2. **Chan-Lam 偶联**：
   - Cu 催化的硼酸与胺/酚的偶联
   - 不需要贵金属 Pd
   - 是 Suzuki 的补充
