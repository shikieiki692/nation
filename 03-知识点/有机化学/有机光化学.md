# 有机光化学（Organic Photochemistry）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]

## 一、定义

**有机光化学**：研究有机分子在**光（紫外-可见光）激发**下发生的化学反应的学科。核心特征是反应从**激发态**出发，产物选择性与热反应（基态反应）可能完全不同。

**本质**：**热化学的补集**——热禁阻的反应光照下可能允许；基态不发生的电子跃迁，激发态可以实现。

## 三、核心原理

### 3.1 激发态基础

**电子跃迁类型**：

| 跃迁 | 能量 | 特征 | 竞赛相关性 |
|:---|:---:|:---|:---:|
| **π→π*** | 较高 | 允许跃迁，强吸收 | 高（烯烃、芳环）|
| **n→π*** | 较低 | 禁阻跃迁，弱吸收 | 高（羰基化合物）|

**激发态分类**：

| 参数 | 基态 S₀ | 单线态 S₁ | 三线态 T₁ |
|:---|:---:|:---:|:---:|
| 电子自旋 | 配对 ↑↓ | 配对 ↑↓ | 平行 ↑ ↑ |
| 寿命 | — | 10⁻⁹–10⁻⁶ s | 10⁻⁶–10⁻³ s |
| 能量 | 最低 | 较高 | 中等（低于 S₁）|

**关键过程**：
- **ISC（系间窜越）**：S₁ → T₁，自旋禁阻但可发生
- **荧光**：S₁ → S₀ + hν（快，10⁻⁸ s）
- **磷光**：T₁ → S₀ + hν（慢，10⁻³ s）

### 3.2 光化学反应的选择性

**激发态电子分布与基态不同** → 反应位点和产物选择性不同

**经典案例**：[2+2] 环加成
- **热反应**：[2+2] 是**热禁阻**的（Woodward-Hoffmann 规则）
- **光反应**：[2+2] 是**光允许**的
- **竞赛意义**：看到反应条件有 **hv** → 考虑光化学 [2+2]

### 3.3 经典光化学反应

#### Norrish Type I（α-裂解）

$$
\ce{R-CO-R' ->[hv] R-CO• + R'•}
$$

- 羰基化合物吸收光 → α-裂解 → 酰基自由基 + 烷基自由基
- 发生于**气相或非极性溶剂**

#### Norrish Type II（γ-H 夺取）

$$
\ce{RCOCH2CH2CH2R' ->[hv] [六元环过渡态] -> 烯醇 + 烯烃}
$$

- 条件：**γ-位有氢原子**
- 机理：激发态羰基氧夺取 γ-H → 1,4-双自由基 → 断裂为烯醇 + 烯烃
- **竞赛识别**：羰基化合物 + hv → 断裂为小分子烯烃 → 优先考虑 Norrish Type II

#### Paternò-Büchi 反应

$$
\ce{R2C=O + R'2C=CR''2 ->[hv] 氧杂环丁烷}
$$

- 羰基（激发态）+ 烯烃（基态）→ [2+2] 光环化 → **氧杂环丁烷**
- 是**光化学特有的环加成反应**

#### [2+2] 光环化

$$
\ce{2 R2C=CR2 ->[hv] 环丁烷衍生物}
$$

- 两个烯烃在光照下形成环丁烷
- **热禁阻、光允许**的典型代表
- 常用于构建**四元环**（张力大，热方法难以实现）

---

## 四、关键结论

1. **光化学 ≠ 热化学 + 能量**：光化学是**激发态反应**，电子分布不同 → 选择性不同
2. **[2+2] 热禁阻、光允许**——看到 hv + 烯烃 → 优先考虑环丁烷
3. **Norrish Type II 需要 γ-H**——羰基化合物光解的标志性反应
4. **Paternò-Büchi 是羰基特有的 [2+2]**——生成氧杂环丁烷
5. **荧光 vs 磷光**：S₁→S₀（快）vs T₁→S₀（慢），寿命差异 10⁵ 倍

---

## 五、常见分类或情形

### 5.1 按反应类型分类

| 反应 | 底物 | 产物 | 识别标志 |
|:---|:---|:---|:---|
| **Norrish I** | 羰基化合物 | 自由基碎片 | 气相/非极性溶剂 |
| **Norrish II** | γ-H 羰基化合物 | 烯醇 + 烯烃 | 断裂为小分子烯烃 |
| **Paternò-Büchi** | 羰基 + 烯烃 | 氧杂环丁烷 | 含氧四元环 |
| **[2+2] 光环化** | 两个烯烃 | 环丁烷 | 四元碳环 |
| **光异构化** | 烯烃 | 顺反异构体 | 顺反互变 |

### 5.2 按激发态类型分类

| 类型 | 特点 | 典型反应 |
|:---|:---|:---|
| **S₁ 反应** | 寿命短，保持单线态自旋 | 直接光解、荧光竞争 |
| **T₁ 反应** | 寿命长，自由基特征强 | 系间窜越后的反应 |

## 六、适用条件与限制

1. **需要光源**：通常用紫外灯（汞灯）或可见光 LED
2. **需要发色团**：分子必须吸收特定波长的光才能反应
3. **三线态淬灭**：氧气（³O₂）是常见的三线态淬灭剂，光化学反应通常需除氧
4. **与热反应竞争**：激发态寿命短，必须在失活前完成反应

---

## 七、常见比较与易混点

### 7.1 光化学 vs 热化学

| | **热化学** | **光化学** |
|:---|:---|:---|
| **能量来源** | 热能（Δ）| 光能（hv）|
| **反应态** | 基态 S₀ | 激发态 S₁/T₁ |
| **选择性** | 由基态电子分布决定 | 由激发态电子分布决定 |
| **[2+2] 环加成** | 热禁阻 | 光允许 |
| **[4+2] D-A** | 热允许 | 光禁阻（通常）|

### 7.2 Norrish I vs Norrish II

| | **Norrish I** | **Norrish II** |
|:---|:---|:---|
| **裂解位置** | α-位（羰基旁边）| γ-位（通过六元环过渡态）|
| **产物** | 两个自由基 | 烯醇 + 烯烃 |
| **条件** | 气相/非极性溶剂优先 | 需要 γ-H |
| **识别** | 产生小分子碎片 | 产生烯烃碎片 |

## 十、例题

### 10.1 例题 1（★★★ 基础）

**题目**：2-戊酮在光照下发生什么反应？写出主要产物。

**分析**：
- 2-戊酮：CH₃COCH₂CH₂CH₃
- γ-位有 H（C4 上的 H）→ 满足 Norrish Type II 条件
- 激发态羰基氧夺取 γ-H → 1,4-双自由基 → 断裂

**解答**：
$$
\ce{CH3COCH2CH2CH3 ->[hv] CH3COH=CH2 + CH2=CHCH3}
$$
更准确：烯醇（丙酮烯醇）+ 丙烯，烯醇互变为丙酮

**反思**：
- 关键：识别 γ-H 的存在
- Norrish II 是酮类光解的最常见路径

### 10.2 例题 2（★★★★ 进阶）

**题目**：环己烯酮在光照下与另一分子乙烯反应，预测产物。

**分析**：
- 环己烯酮：含有 C=O 和 C=C
- 光照 + 烯烃 → Paternò-Büchi 或 [2+2] 光环化
- 若 C=O 被激发 → 与乙烯发生 Paternò-Büchi → 氧杂环丁烷

**解答**：
- 产物为**螺环氧杂环丁烷**（spiro-oxetane）
- 具体结构取决于烯酮的哪个双键参与反应

**反思**：
- 光条件下羰基化合物 + 烯烃 → 优先考虑 Paternò-Büchi
- 注意区分 C=O 参与的 [2+2] 和 C=C 参与的 [2+2]

## 十一、易错点

1. ❌ **光化学就是加热的另一种形式** → 光化学是**激发态反应**，产物与热反应**不同**
2. ❌ **所有 [2+2] 都是光允许** → 只有**某些** [2+2] 光允许（需考虑轨道对称性）
3. ❌ **Norrish II 不需要 γ-H** → γ-H 是**必要条件**
4. ❌ **荧光和磷光是一个东西** → S₁→S₀（荧光）vs T₁→S₀（磷光），寿命差 10⁵ 倍
5. ❌ **光化学只发生在紫外区** → 可见光也能驱动某些反应（如光氧化还原催化）

### 12.2 学生易踩的认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析口诀/方法 |
|:---|:---|:---|:---|
| "hv 只是提供能量，反应机理和热一样" | 能量守恒思维 | 光化学改变的是**电子分布**，不只是能量 | "光照换脑子，加热换速度" |
| "[2+2] 总是光允许" | 只知道一个规则 | [2+2] 热禁阻、光允许是**一般规律**，但需具体看轨道 | "一般光允许，具体看轨道" |
| "Norrish I 和 II 是随机竞争的" | 以为两者随机发生 | 有 γ-H 且液相 → II 占优；气相/无 γ-H → I | "有 γ-H 走 II，没 γ-H 走 I" |
| "磷光比荧光强" | 以为慢的就是强的 | 荧光快但强（量子产率高），磷光慢但弱 | "荧光快而亮，磷光慢而暗" |

### 12.3 入门级例题

**题目**：苯乙酮（C₆H₅COCH₃）在光照下会发生什么反应？能发生 Norrish Type II 吗？

**预期解答路径**：
1. 苯乙酮结构：Ph–CO–CH₃
2. α-位是 CH₃，β-位是 Ph（无 H），没有 γ-H
3. 不满足 Norrish Type II 条件
4. 主要发生 Norrish Type I：PhCO• + CH₃•

**老师可在课堂上引导的提问**：
- 学生若直接写 Norrish II：问 "γ-H 在哪里？苯环上的 H 算 γ-H 吗？"
- 学生若答对：追问 "如果换成苯丙酮（PhCOCH₂CH₃）呢？"

### 12.4 与现实/直觉的连接

- **类比**：光化学就像"给分子喝了一杯浓缩咖啡"——平时（基态）不想做的事，激发态后愿意做了（电子分布改变）
- **反例**：表面上看，光照和加热都是"给能量"；实际上，光照改变的是分子的"性格"（电子构型），而加热只是让分子"跑更快"（动能增加）

## 十三、竞赛拓展

1. **不对称光化学**：手性光催化剂实现不对称 [2+2] 环加成，近年方法学热点。
2. **连续流光化学**：微反应器中的光化学，工业应用前沿。
3. **上转换光化学**：将低能可见光转为高能紫外光，拓展光化学反应范围。
