# Wagner-Meerwein 重排

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[39-重排反应]]

## 一、定义

**Wagner-Meerwein 重排**：包含 **H 或烷基迁移**的碳正离子重排反应，通过 1,2-迁移生成更稳定的碳正离子。

**本质**：碳正离子的"自救"——不稳定的碳正离子通过相邻基团的迁移，转变为更稳定的碳正离子。

## 二、考纲对应

- 对应考纲条目：[[39-重排反应]]
- 所属模块：[[基础要求-有机化学]]
- 本知识点在考纲中的作用：W-M 重排是碳正离子化学的核心内容，是理解 SN1 溶剂解、频哪醇重排等反应的基础。

## 三、核心原理

### 3.1 基本特征

**驱动力**：生成更稳定的碳正离子（伯 → 仲 → 叔）

**迁移基团**：
- **负氢迁移（Hydride Shift）**：H⁻ 迁移
- **烷基迁移（Alkyl Shift）**：R 迁移（甲基、乙基等）

**迁移规律**：
- 1,2-迁移（相邻碳之间的迁移）
- 迁移基团的立体化学保持

### 3.2 分子轨道分析（Clayden Ch.36）

**甲基迁移**：
- LUMO = 阳离子空 p 轨道
- HOMO = 断裂的 C-C σ 键
- 电子从 C-C σ 键流向空 p 轨道

**负氢迁移**：
- HOMO = C-H σ 键
- 电子从 C-H σ 键流向空 p 轨道

### 3.3 可逆性（ABOC 平衡思维）

**重要观点**：W-M 重排是**可逆平衡**
- 稳定碳正离子 → 不稳定碳正离子可逆
- 可由产物拉动平衡
- 这是 ABOC 强调的"平衡思维"经典案例

> "不稳定碳正离子通过 [1,2]-烷基/H 迁移变稳定，作为可逆平衡。"（ABOC §1.3f）

### 3.4 判断准则（Clayden Ch.36）

1. **重排必须很快**，使其他反应不能完成
2. **产物阳离子必须比起始阳离子足够稳定**
3. **捕获阳离子的过程必须可靠**

## 四、关键结论

1. **W-M 重排的驱动力是碳正离子稳定性**——伯 → 仲 → 叔
2. **迁移基团：H > 烷基（通常）**——负氢迁移比烷基迁移更快
3. **1,2-迁移中迁移基团保持立体化学**
4. **W-M 重排是可逆平衡**——可被产物拉动
5. **新戊基重排是经典案例**——甲基迁移，伯 → 叔

## 五、常见分类或情形

### 5.1 按迁移基团分类

| 迁移类型 | 例子 | 结果 |
|:---|:---|:---|
| **负氢迁移** | 2° 碳正离子 → 3° 碳正离子 | 更稳定 |
| **甲基迁移** | 新戊基碳正离子 → 叔戊基碳正离子 | 伯 → 叔 |
| **芳基迁移** | 频哪醇重排中 | 芳基比烷基更易迁移 |
| **环扩大** | 四元环相邻碳正离子 → 五元环 | 释放环张力 |

### 5.2 经典案例

**新戊基重排**（Clayden Ch.36）：
- 新戊基碘在 Ag⁺/H₂O 条件下
- 甲基迁移 → 叔碳阳离子 → 水捕获
- NMR 证据：对甲苯磺酸新戊酯在超强酸中直接给出叔碳阳离子光谱，伯阳离子从未被观察到

**扩环重排**：
- 与阳离子相邻的四元环 → 重排为五元环
- 释放环张力
- 天然产物 α-石竹烯醇的合成应用

## 六、适用条件与限制

1. **必须有碳正离子中间体**——SN1、E1、酸催化等条件下
2. **迁移基团必须在相邻位置**——1,2-迁移
3. **产物碳正离子必须更稳定**——否则重排不利
4. **与消除反应竞争**——强碱条件下消除可能比重排快
5. **频哪醇/半频哪醇重排是 W-M 的变体**

## 七、常见比较与易混点

### 7.1 W-M 重排 vs 频哪醇重排

| | **W-M 重排** | **频哪醇重排** |
|:---|:---|:---|
| 底物 | 一般碳正离子 | 1,2-二醇 |
| 驱动力 | 碳正离子稳定性 | 碳正离子稳定性 + 羰基形成 |
| 产物 | 碳正离子（被捕获）| 酮 |
| 关系 | 一般形式 | 特殊形式 |

### 7.2 负氢迁移 vs 烷基迁移

| | **负氢迁移** | **烷基迁移** |
|:---|:---|:---|
| 速率 | 通常更快 | 较慢 |
| 原因 | H 体积小，轨道重叠好 | R 位阻大 |
| 优先级 | 通常优先 | 次优先 |
| 例外 | 芳基迁移可能比 H 快 | — |

## 十、例题

### 例题 1（★ 基础）

**题目**：新戊基溴在 Ag⁺/H₂O 条件下反应，产物是什么？

**解答**：
1. Ag⁺ 协助 Br⁻ 离去 → 新戊基碳正离子（伯，极不稳定）
2. **甲基迁移**（W-M 重排）→ 叔戊基碳正离子
3. H₂O 进攻 → **叔戊醇**

### 例题 2（★★ 负氢迁移）

**题目**：2-甲基-2-丁醇在酸催化下脱水，产物是什么？

**解答**：
1. 质子化 OH → 失去 H₂O → 2° 碳正离子
2. **负氢迁移** → 3° 碳正离子（更稳定）
3. 失去 H⁺ → **2-甲基-2-丁烯**（Zaitsev 产物）

## 十一、易错点

1. ❌ **忽略 W-M 重排的可逆性** → 是平衡，可被产物拉动
2. ❌ **认为所有碳正离子都重排** → 需要产物更稳定才发生
3. ❌ **混淆 1,2-和 1,3-迁移** → W-M 是 1,2-迁移
4. ❌ **忽略立体化学保持** → 迁移基团构型保持

| "所有 SN1 都伴随重排" | 只有产物更稳定时才发生 |

### 12.3 入门级例题

**题目**：解释为什么新戊基碳正离子从不被观察到。

**解答**：
1. 新戊基碳正离子是伯碳正离子，极不稳定
2. 一旦形成，立即发生甲基迁移（W-M 重排）
3. 转变为叔戊基碳正离子（稳定）
4. NMR 实验中只观察到叔碳正离子

### 12.4 与现实/直觉的连接

- W-M 重排就像"热土豆游戏"——不稳定的碳正离子把"烫手山芋"（正电荷）传给邻居，直到有人能"接住"（稳定的碳正离子）。
- 可逆性就像"跷跷板"——两边可以来回摆动，哪边有"重物"（后续反应）就偏向哪边。
