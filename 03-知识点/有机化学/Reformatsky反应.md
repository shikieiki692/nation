# Reformatsky 反应（Reformatsky Reaction）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[45-亲核加成反应]]

---

## 一、定义

**Reformatsky 反应**：**α-卤代酯**与**锌粉**反应生成**有机锌试剂**，进而与**醛/酮**反应生成 **β-羟基酯** 的反应。

**通式**：
$$
\ce{X-CH2-COOR' + Zn + RCHO -> R-CH(OH)-CH2-COOR'}
$$

**本质**：**有机锌试剂对羰基的亲核加成**——是 Aldol 反应的"温和锌版"。

## 三、核心原理

### 3.1 反应机理

```
步骤 1：α-卤代酯 + Zn → 有机锌试剂（烯醇式有机锌）
步骤 2：有机锌试剂进攻醛/酮的羰基碳
步骤 3：酸性水解 → β-羟基酯
```

**关键特征**：
- **有机锌试剂亲核性较弱**——不进攻酯羰基，只进攻醛/酮
- **锌的活化**——通常用 1,2-二溴乙烷或 TMSCl 活化锌粉
- **溶剂**：苯、甲苯、THF、1,4-二氧六环

### 3.2 与 Grignard 试剂的对比

| | **Reformatsky** | **Grignard** |
|:---|:---|:---|
| **金属** | Zn | Mg |
| **亲核性** | 较弱 | 强 |
| **与酯的反应** | 不反应（选择性）| 反应（生成叔醇）|
| **产物** | β-羟基酯 | 醇 |
| **条件** | 温和（回流）| 严格（无水乙醚）|

**核心差异**：Grignard 试剂会进攻酯基，而 Reformatsky 的有机锌试剂**只进攻醛/酮**，因此可以与酯共存。

### 3.3 与 Aldol 的关系

| | **Aldol** | **Reformatsky** |
|:---|:---|:---|
| **亲核体** | 烯醇盐 | 有机锌烯醇式 |
| **亲电体** | 醛/酮 | 醛/酮 |
| **产物** | β-羟基羰基 | β-羟基酯 |
| **特点** | 强碱条件 | 中性/弱酸性条件 |

**Reformatsky = Aldol 的"酯版"**——产物多了一个酯基。

### 3.4 底物范围

| α-卤代酯 | 可行性 | 备注 |
|:---|:---:|:---|
| **α-溴代乙酸乙酯** | ✅ 优秀 | 最经典 |
| **α-溴代丙酸酯** | ✅ 良好 | 引入 α-甲基 |
| **α-氯代酯** | ⚠️ 较差 | 活性低，需活化锌 |
| **α-碘代酯** | ✅ 良好 | 活性高，但较贵 |

| 羰基化合物 | 可行性 | 备注 |
|:---|:---:|:---|
| **醛** | ✅ 优秀 | 反应最快 |
| **酮** | ✅ 良好 | 位阻大时较慢 |
| **酯** | ❌ 不反应 | 有机锌亲核性不够 |

---

## 四、关键结论

1. **Reformatsky = α-卤代酯 + Zn + 醛/酮 → β-羟基酯**
2. **有机锌试剂亲核性弱于 Grignard**——不进攻酯基
3. **可与酯共存**——这是 Reformatsky 的选择性优势
4. **是 Aldol 的"酯版"**——机理类似，产物含酯
5. **α-溴代乙酸乙酯是最常用底物**

---

## 五、常见分类或情形

### 5.1 按产物分类

| 目标 | 路线 | 应用 |
|:---|:---|:---|
| **制备 β-羟基酯** | Reformatsky 直接产物 | 天然产物中间体 |
| **制备 α,β-不饱和酯** | Reformatsky → 脱水 | 类似 Aldol 脱水 |
| **制备 β-酮酯** | Reformatsky → 氧化 | 进一步转化 |

### 5.2  Reformatsky 变体

| 变体 | 改进 | 特点 |
|:---|:---|:---|
| **Blaise 反应** | α-卤代酯 + Zn + 腈 | 生成 β-酮酯 |
| **Luche 还原** | 烯酮 + CeCl₃/NaBH₄ | 1,2-还原（与 Reformatsky 无关但常一起讲）|

---

## 六、适用条件与限制

1. **需要无水条件**——有机锌对水敏感
2. **锌需活化**——旧锌粉可能表面氧化，需用酸或 1,2-二溴乙烷活化
3. **α-氯代酯活性低**——最好用 α-溴代或 α-碘代酯
4. **反应较慢**——通常需要回流数小时
5. **产物可能脱水**——加热条件下 β-羟基酯可能脱水为 α,β-不饱和酯

---

## 七、常见比较与易混点

### 7.1 Reformatsky vs Grignard

- Reformatsky：有机锌，**不进攻酯**，产物 β-羟基酯
- Grignard：有机镁，**进攻酯**，产物醇
- 两者都用金属有机试剂，但选择性不同

### 7.2 Reformatsky vs Aldol

- Reformatsky：有机锌烯醇式进攻醛/酮
- Aldol：烯醇盐进攻醛/酮
- 两者产物结构相似（β-羟基羰基），但 Reformatsky 产物含酯基

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：α-溴代乙酸乙酯 + 锌粉 + 苯甲醛 → ?

**解答**：
- 形成有机锌试剂 → 进攻苯甲醛
- 产物：**β-羟基-β-苯基丙酸乙酯**

### 10.2 例题 2（★★ 选择性）

**题目**：为什么 Reformatsky 试剂能与酯共存，而 Grignard 试剂不能？

**解答**：
- 有机锌试剂的亲核性弱于有机镁试剂
- C-Zn 键比 C-Mg 键更共价、更稳定
- 所以有机锌不能进攻酯羰基，只能进攻更活泼的醛/酮

### 10.3 例题 3（★★★ 合成设计）

**题目**：如何从苯甲醛制备 β-苯基-β-羟基丙酸？

**解答**：
- 苯甲醛 + α-溴代乙酸乙酯 + Zn → Reformatsky
- 产物 β-羟基酯
- 水解 → **β-苯基-β-羟基丙酸**

---

## 十一、易错点

1. ❌ **Reformatsky 试剂进攻酯基** → 有机锌**不进攻酯**，只进攻醛/酮
2. ❌ **Reformatsky 和 Grignard 一样** → 两者选择性不同（Zn vs Mg）
3. ❌ **α-氯代酯效果最好** → α-溴代酯效果最好
4. ❌ **Reformatsky 产物是醇** → 产物是 **β-羟基酯**
5. ❌ **不需要无水条件** → **必须无水**，有机锌对水敏感

### 12.3 入门例题

**题目**：什么时候应该用 Reformatsky 反应而不是 Grignard 反应？

**引导**：
- 当底物中含有酯基时，Grignard 会破坏酯基
- Reformatsky 的有机锌试剂只进攻醛/酮，不进攻酯
- 所以底物含酯时选 Reformatsky

### 12.4 现实类比

**Reformatsky 像"精准手术刀"**：
- Grignard = 大砍刀（见什么砍什么，醛酮酯全砍）
- Reformatsky = 手术刀（只切病灶——醛/酮，不伤及好组织——酯）
- 当你需要保留酯基时，必须用 Reformatsky 这把手术刀
