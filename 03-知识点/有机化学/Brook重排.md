# Brook重排

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[50-有机合成]]

## 一、定义
- **Brook重排**（Brook rearrangement）是指在碱性条件下，**硅基（silyl group, 如 Me₃Si, TBS, TIPS）从碳原子迁移到氧原子**（或反向）的分子内重排反应。
- 最经典的形式是 **[1,2]-Brook 重排**：α-硅基烷氧基负离子（$R_3Si–CH(R')–O^-$）发生硅基从碳到氧的 [1,2]-迁移，生成 **碳负离子**（$^–CH(R')–OSiR_3$）。
- 该反应由加拿大化学家 **Adrian G. Brook** 于 1958 年首次系统报道。
- 逆反应（硅基从氧迁移到碳）称为 **逆 Brook 重排**（retro-Brook rearrangement）。

## 三、核心原理

### 1. [1,2]-Brook 重排机理
$$R_3Si–CH(R')–OH \xrightarrow{\text{碱}} R_3Si–CH(R')–O^– \longrightarrow \,^–CH(R')–OSiR_3$$

**步骤解析**：
1. **去质子化**：碱（如 LDA、n-BuLi、KH）夺取 α-硅基醇的羟基氢，生成 **α-硅基烷氧基负离子**。
2. **硅基迁移**：硅基从相邻的碳原子迁移到氧负离子上，通过 **三元环过渡态**（或超价硅中间体）完成 [1,2]-σ 迁移。
3. **生成碳负离子**：迁移完成后，原硅基所在的碳原子变成 **碳负离子**（carbanion），可被亲电试剂捕获。

**驱动力**：
- **热力学**：Si–O 键（~110 kcal/mol）远强于 Si–C 键（~70 kcal/mol）和 C–O 键（~85 kcal/mol）。硅基对氧的亲和力强于对碳的亲和力。
- **动力学**：烷氧基负离子的强碱性促进了硅基的迁移。

### 2. [1,n]-Brook 重排（远程迁移）
- 当硅基与烷氧基之间相隔 n–1 个碳原子时，可发生 **[1,n]-Brook 重排**（n = 3, 4, 5...）。
- 机理上通过 **链行走**（chain walking）或 **超价硅中间体** 实现远程迁移。
- [1,4]-Brook 和 [1,5]-Brook 在合成中应用较广，常用于 **1,4-二官能团化** 或 **远程关环**。

### 3. 逆 Brook 重排（Retro-Brook）
$$\,^–CH(R')–OSiR_3 \xrightarrow{\text{酸或热}} R_3Si–CH(R')–O^–$$
- 在酸性条件下或高温下，硅基可从氧迁移回碳，实现 **碳负离子的原位再生** 或 **官能团位置的复原**。
- 逆 Brook 重排是热力学不利的过程（Si–O 键更强），但在特定条件下（如强 Lewis 酸 AlCl₃、高温、或硅基体积过大导致位阻排斥）可以发生。

## 四、关键结论
- Brook 重排的核心价值在于 **"硅基辅助的极性翻转"**：通过硅基的迁移，将原本稳定的 C–Si 键转化为高反应活性的 C⁻ 碳负离子，实现 **碳的亲核化**。
- **可逆性**是 Brook 重排的独特优势——正向 Brook（碱催化）和逆向 Brook（酸/热）的切换，使得硅基成为 **可逆的官能团定位器**。
- 硅基的大小（Me₃Si < TBS < TIPS < TBDPS）显著影响迁移速率和选择性：**大位阻硅基迁移较慢，但稳定性更高**。
- Brook 重排常与 **亲电捕获** 串联：重排生成的碳负离子可被醛、酮、卤代烷、CO₂ 等亲电试剂捕获，实现 **C–C 键形成**。

## 五、常见分类或情形
- **按迁移距离**：
  - "[1,2]-Brook：最经典，硅基在相邻碳上"
  - "[1,3]-Brook：隔一个碳"
  - "[1,4]-Brook：隔两个碳（合成中最有用之一）"
  - "[1,5]-Brook 及更远：较少见，需特定几何构型"
- **按硅基类型**：
  - Me₃Si（TMS）：迁移最快，但稳定性差，易水解
  - t-BuMe₂Si（TBS/TBDMS）：平衡选择，最常用
  - i-Pr₃Si（TIPS）：迁移较慢，但对酸碱稳定
  - t-BuPh₂Si（TBDPS）：迁移最慢，用于需要高稳定性的场景
- **按串联模式**：
  - Brook + 亲电捕获（最常见）
  - Brook + 氧化（碳负离子 → 醇/酮）
  - 多步 Brook（连续多次硅基迁移）

## 六、适用条件与限制
- **正向 Brook（硅基 C→O 迁移）**：
  - 条件：碱性（LDA、NaH、n-BuLi、KOtBu）
  - 底物：必须含有 **α-硅基醇** 或 **α-硅基醚** 结构
  - 溶剂：THF、Et₂O 等非质子极性溶剂；质子性溶剂会与硅基反应
- **逆向 Brook（硅基 O→C 迁移）**：
  - 条件：酸性（TFA、HF·pyridine）或高温（>150°C）
  - 限制：Si–O 键强，逆反应热力学不利，需要强驱动力（如释放环张力或形成共轭体系）
- **硅基选择性**：
  - 若分子中有多个可迁移硅基，**小硅基优先迁移**（TMS > TBS > TIPS）
  - 位阻大的环境可能完全阻止迁移

## 七、常见比较与易混点

| 比较对 | 常见混淆 | 辨析 |
|:---|:---|:---|
| Brook 重排 vs Wagner-Meerwein 重排 | 都是 [1,2]-重排 | Brook 是 **硅基从 C 到 O 的 σ 迁移**；Wagner-Meerwein 是 **碳正离子的 [1,2]-烷基/氢迁移**。底物和机理完全不同 |
| Brook 重排 vs 硅醚保护基脱除 | 都涉及 Si–O 键 | Brook 重排是 **分子内硅基的迁移**（C→O 或 O→C）；硅醚脱除是 **Si–O 键的断裂**（如 TBAF 裂解 Si–O）。前者是重排，后者是消除 |
| 正向 Brook vs 逆向 Brook | 方向搞反 | 正向：碱催化，C→O，生成碳负离子；逆向：酸/热，O→C，再生烷氧基。驱动力方向由 Si–O > Si–C 决定 |
| [1,2]-Brook vs [1,4]-Brook | 距离判断错误 | [1,n]  numbering 以 **迁移终点为 1**，起点为 n。即 [1,2] 是相邻碳，[1,4] 是隔两个碳（1-O, 2-C, 3-C, 4-C-Si） |

## 十、例题

### 例题 1
**题目：**
解释为什么在以下转化中，使用 TBS（t-BuMe₂Si）保护的底物在 LDA 条件下处理后，再与醛反应，可以得到预期产物；而若使用 TMS（Me₃Si）保护，则产率显著降低。

底物示意：$R–CH(OH)–CH_2–SiMe_2Bu^t$（α-硅基醇，TBS 在末端碳上）

**分析：**
本题考查 Brook 重排中 **硅基大小对迁移速率的影响**，以及迁移后碳负离子的捕获效率。

**解答：**
1. **TBS 体系（产率高）**：
   - LDA 夺取羟基氢，生成烷氧基负离子。
   - TBS 发生 **[1,2]-Brook 重排**，从末端碳迁移到氧上，生成碳负离子 $R–CH(OTBS)–CH_2^–$。
   - 碳负离子与醛发生 **亲核加成**，生成 β-羟基硅醚产物。
   - TBS 的迁移速率**适中**，既能在 LDA 条件下有效重排，又不会过快导致副反应。

2. **TMS 体系（产率低）**：
   - TMS 位阻小，迁移速率**极快**。
   - 在 LDA 去质子化瞬间，TMS 立即发生 Brook 重排，生成的碳负离子可能**在亲电试剂加入前**就与体系中微量质子源（溶剂杂质）反应，或被氧分子捕获，导致分解。
   - 此外，TMS 对碱性条件较敏感，可能发生 **Si–C 键的副反应**（如 Peterson 消除的副路径）。

**反思：**
硅基的选择是 Brook 重排成功的前提。口诀：**"要快选 TMS，要稳选 TBS，要慢选 TIPS"**。在需要分步操作（先 Brook 重排，后加亲电试剂）时，TBS 的"中等迁移速率"提供了最佳的时间窗口。

## 十一、易错点
- **[1,n] numbering 错误**：Brook 重排的 [1,n] 命名以 **氧原子为 1**，硅基所在碳为 n。不是以硅基为起点。
- **忽略可逆性**：正向 Brook（碱）和逆向 Brook（酸）是一对可逆反应。在多步合成中，若某步用碱、下步用酸，可能发生 **意外的逆向 Brook**，导致硅基回到原来位置。
- **与 Peterson 烯化反应混淆**：Peterson 反应也是 α-硅基碳负离子与羰基反应，但产物是 **烯烃**（通过 β-羟基硅烷的消除）；Brook 重排是 **硅基迁移**，产物是 **碳负离子/硅醚**。
- **溶剂选择错误**：Brook 重排需要 **非质子极性溶剂**（THF、Et₂O）。质子性溶剂（如 MeOH、H₂O）会与硅基或碳负离子反应，导致反应失败。

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析口诀/方法 |
|:---|:---|:---|:---|
| "硅基只是保护基，没有化学反应性" | 中学/普化阶段硅基=保护基=惰性 | 硅基在特定条件下（碱催化）可发生 **Brook 重排**，实现极性翻转和 C–C 键形成 | 硅基不是木头，碱性条件会跳舞 |
| "Brook 重排和硅醚脱保护是一回事" | 都涉及 Si–O 键的形成/断裂 | Brook 是 **分子内重排**（Si 从 C 到 O）；脱保护是 **键的断裂**（Si–O 被 F⁻ 或 H⁺ 裂解）。产物不同 | 重排是搬家，脱保护是拆房 |
| "逆 Brook 很容易发生，因为正逆都是重排" | 误以为所有可逆反应都容易双向进行 | 正向 Brook（C→O）热力学非常有利（Si–O 键强）；逆向 Brook（O→C）需要 **强酸或高温**，不易发生 | 正向顺流而下，逆流向水而上 |

### 12.3 入门级例题

**题目**：
画出 (CH₃)₃Si–CH₂–CH₂–OH 在强碱（如 n-BuLi）作用下发生 [1,2]-Brook 重排的产物，并说明该产物可如何与 CH₃I 反应得到最终产物。

**预期解答路径**：
1. 第一步看到题目应该想到 **Brook 重排的底物要求** → α-硅基醇（Si 在 OH 相邻碳上）。
2. 然后画出 [1,2]-迁移：TMS 从 C-1 迁移到 O 上，C-1 变成碳负离子。
3. 产物是 $^–CH_2–CH_2–OSiMe_3$（2-三甲基硅氧基乙基碳负离子）。
4. 碳负离子与 CH₃I 发生 **SN2 反应**，生成 $CH_3–CH_2–CH_2–OSiMe_3$（丙基硅醚）。

**老师可在课堂上引导的提问**：
- 学生若卡在第 1 步：问 "Brook 重排需要什么官能团组合？硅基和羟基的位置关系是什么？"
- 学生若画错迁移方向（O→C）：问 "Si–O 键和 Si–C 键哪个更强？热力学上硅基更愿意待在哪里？"
- 学生若忽略碳负离子的反应性：问 "Brook 重排后，原硅基所在碳变成了什么？碳负离子能做什么反应？"

### 12.4 与现实/直觉的连接
- 类比：Brook 重排就像一个 **"化学开关"**——硅基平时安静地待在碳上（像开关的 OFF 状态），遇到碱就跳到氧上，同时把碳变成高活性的碳负离子（像开关的 ON 状态）。这个 ON 状态可以被亲电试剂"读取"，完成信息传递（C–C 键形成）。
- 反例/佯谬：我们通常认为"强碱会破坏分子"，但 Brook 重排中强碱（LDA、BuLi）不仅没破坏分子，反而**精确激活了特定位置**，实现了传统方法难以完成的极性翻转。这是"破坏性创造"的化学典范。

## 十三、竞赛拓展
- **串联 Brook-aldol 反应**：Brook 重排生成的碳负离子原位与醛/酮发生羟醛缩合，一锅实现 **C–Si → C⁻ → C–C** 的转化，是构建 1,3-二醇的高效策略。
- **硅基导向的环化反应**：利用 [1,4]-Brook 重排生成远程碳负离子，诱导 **分子内 SN2 或 Michael 加成**，实现大环或桥环的合成。
- **不对称 Brook 重排**：使用手性硅基或手性碱（如金鸡纳生物碱衍生的相转移催化剂），可实现对映选择性的硅基迁移和后续亲电捕获。
- **天然产物全合成应用**：
  - **(+)-Patulolide C**（大环内酯）
  - **多种聚酮类天然产物**的 1,3-二羟基片段构建
  - **糖化学**中硅基辅助的糖苷化策略
