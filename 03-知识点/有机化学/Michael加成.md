# Michael 加成（Michael Addition）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[37-加成反应]]、[[45-亲核加成反应]]

---

## 一、定义

**Michael 加成**：碳负离子（或亲核体）对 **α,β-不饱和羰基化合物**（Michael 受体）的 **1,4-共轭加成**反应。

**通式**：

$$
\ce{Nu^- + CH2=CH-CO-R -> Nu-CH2-CH^--CO-R ->[H+] Nu-CH2-CH2-CO-R}
$$

**Michael 受体（受体）**：
- α,β-不饱和醛/酮
- α,β-不饱和酯/酰胺
- α,β-不饱和腈
- 硝基烯烃（强 Michael 受体）

**Michael 供体（亲核体）**：
- 烯醇盐（酮、酯、腈的 α-碳负离子）
- 丙二酸酯负离子
- 氰化物
- 硝基烷负离子
- 有机铜试剂 R₂CuLi

## 三、核心原理

### 3.1 机理

**三步机理**：

1. **亲核试剂生成**：碱夺取供体的 α-H → 碳负离子（烯醇盐）
2. **1,4-加成**：碳负离子进攻 Michael 受体的 **β-碳**（Cβ）
   - 不是进攻羰基碳（1,2-加成）——因为软亲核试剂偏好与 LUMO 系数更大的 β-碳重叠
3. **质子化**：烯醇盐中间体质子化 → 饱和羰基产物

```
     O                      O⁻                     O
     ‖                      |                      ‖
Nu⁻ + C-CH=CH₂  →  Nu-CH-CH=CH₂  →  Nu-CH₂-CH₂-C-H
          |                |   （烯醇盐，互变）
         R                R
```

### 3.2 为什么是 1,4-加成？

**轨道控制 vs 电荷控制**：

| | 1,2-加成 | 1,4-加成（Michael）|
|:---|:---|:---|
| **控制类型** | 电荷控制 | **轨道控制** |
| **进攻位点** | 羰基碳（最正电）| β-碳（LUMO 系数大）|
| **亲核试剂** | 硬亲核（RMgX, RLi）| **软亲核**（烯醇盐, R₂CuLi）|
| **中间体** | 烷氧负离子 | 烯醇盐 |
| **产物特征** | 烯丙醇（羰基破坏）| 饱和羰基（**羰基保留**）|

**关键**：Michael 加成的亲核试剂（烯醇盐、有机铜）是**软亲核试剂**，根据 HSAB 原理偏好与软亲电位点（β-碳）作用。

### 3.3 热力学优势

1,4-加成产物保留强 C=O 键（键能 ~178 kcal/mol），而 1,2-加成破坏 C=O 生成较弱的 C=C + C-O。

因此：
- **热力学上，1,4-加成产物更稳定**
- 高温有利于 Michael 加成

---

## 四、关键结论

1. **Michael 加成 = 软亲核试剂 + α,β-不饱和羰基 → 1,4-加成**
2. **亲核试剂进攻 β-碳**，不是羰基碳
3. **产物保留羰基**——这是后续转化的关键手柄
4. **碱催化即可**——不需要强碱（与 Wittig 等反应不同）
5. **立体化学**：新生成的 C–C 键可产生手性中心，不对称催化版本可高对映选择性

## 五、常见分类或情形

### 5.1 按 Michael 受体分类

| 受体类型 | 例子 | 反应性 | 产物 |
|:---|:---|:---:|:---|
| α,β-不饱和酮 | 甲基乙烯基酮 (MVK) | 高 | 1,5-二酮 |
| α,β-不饱和酯 | 丙烯酸乙酯 | 中 | δ-酮酯 |
| α,β-不饱和腈 | 丙烯腈 | 中高 | δ-氰基酮 |
| 硝基烯烃 | β-硝基苯乙烯 | **很高** | γ-硝基羰基 |
| α,β-不饱和醛 | 丙烯醛 | 高 | 醛（但易进一步反应）|

### 5.2 按 Michael 供体分类

| 供体 | 生成方法 | 特点 |
|:---|:---|:---|
| **烯醇盐** | LDA, NaH, K₂CO₃ | 最常用 |
| **丙二酸酯负离子** | NaOEt, Et₃N | 双吸电子稳定 |
| **氰化物** | KCN | 简单，但毒性大 |
| **硝基烷负离子** | 碱 | 强亲核性 |
| **有机铜锂 (R₂CuLi)** | RLi + CuI | **专做 1,4**，不 1,2 |

### 5.3 串联反应

**Robinson 增环 = Michael 加成 + 分子内羟醛缩合**

```
环己酮 + MVK ──碱──> Michael 加成产物 ──碱/热──> 六元环稠合产物
```

这是构建**双环骨架**的经典方法。

## 六、适用条件与限制

1. **需共轭体系**——非共轭羰基不发生 Michael 加成
2. **醛做受体时反应性过高**——易进一步聚合或发生副反应
3. **1,2-加成竞争**——用硬亲核试剂（RLi, RMgX）或低温时，1,2-加成可能为主
4. **加 Cu(I) 盐可抑制 1,2**——Gilman 试剂 (R₂CuLi) 专做 1,4
5. **碱不可过强**——可能引发自身缩合或其他副反应
6. **空间位阻大的 β-碳反应慢**——四取代 β-碳几乎不反应

## 七、常见比较与易混点

### 7.1 Michael 加成 vs 普通 1,2-加成

| | Michael 加成 | 1,2-加成 |
|:---|:---|:---|
| **试剂** | 软亲核（烯醇盐, R₂CuLi）| 硬亲核（RLi, RMgX）|
| **进攻位点** | **β-碳** | **羰基碳** |
| **产物** | 饱和羰基 | 烯丙醇 |
| **羰基保留？** | ✅ 保留 | ❌ 破坏 |
| **合成价值** | 高（羰基可进一步转化）| 中等 |

### 7.2 Michael vs 普通亲核取代

- **Michael 是加成**，不是取代——没有离去基团
- 驱动力是 **C=C → C–C** 的转化 + **羰基保留**
- 亲核试剂本身不是强碱（否则会发生 E2 消除）

## 十、例题

### 10.1 例题 1（★★ 基础）

**题目**：写出丙二酸二乙酯负离子与甲基乙烯基酮（MVK）的 Michael 加成产物。

**解答**：
- 供体：丙二酸二乙酯负离子（CH(CO₂Et)₂⁻）
- 受体：MVK（CH₂=CH-CO-CH₃）
- 亲核进攻 β-碳（CH₂=CH- 的末端 CH₂）
- 产物：**5-氧代-3,3-二(乙氧羰基)己烷** 或更准确地写结构：
  - CH₃-CO-CH₂-CH₂-CH(CO₂Et)₂

### 10.2 例题 2（★★★ 选择性）

**题目**：环己烯酮分别与 MeMgBr 和 Me₂CuLi 反应，产物有何不同？

**解答**：
- **MeMgBr（硬亲核）**：1,2-加成 → **1-甲基环己-2-烯醇**
- **Me₂CuLi（软亲核）**：1,4-加成（Michael）→ **3-甲基环己酮**（烯醇盐质子化后羰基恢复）

### 10.3 例题 3（★★★ 串联反应）

**题目**：2-甲基环己酮在碱催化下与 MVK 反应，写出最终产物。

**解答**：
这是 **Robinson 增环**：
1. Michael 加成：环己酮烯醇盐进攻 MVK 的 β-碳
2. 分子内羟醛缩合：新生成的羰基（来自 MVK）与环己酮的另一 α-位缩合
3. 脱水 → 形成 **稠环烯酮**

产物为 **双环[4.4.0]癸烯酮衍生物**（具体为 2-甲基-3-(3-氧代丁基)环己酮 → 环化脱水后的产物）

## 十一、易错点

1. ❌ **进攻羰基碳** → Michael 是进攻 **β-碳**
2. ❌ **认为所有亲核试剂都做 Michael** → 硬亲核（RLi, RMgX）做 1,2-加成
3. ❌ **忽略羰基保留** → Michael 产物保留羰基，这是后续反应的手柄
4. ❌ **碱过强导致副反应** → 烯醇盐自身缩合、聚合等
5. ❌ **混淆 Michael 和羟醛缩合** → Michael 是 1,4-加成；羟醛是 1,2-加成（两羰基之间）
6. ❌ **Robinson 增环只写 Michael 步** → 别忘了后面的分子内羟醛 + 脱水

### 12.3 入门级例题

**题目**：2-环己烯酮与丙二酸二乙酯在 NaOEt 条件下反应，主产物是什么？

**预期解答路径**：
1. NaOEt 夺取丙二酸二乙酯的 α-H → 烯醇盐（软亲核）
2. 软亲核进攻 2-环己烯酮的 **β-碳**（C3，远离羰基）
3. 形成烯醇盐中间体（环己酮的 C2 上带负电荷）
4. 质子化 → 产物：**3-(2,2-二乙氧羰基乙基)环己酮**

### 12.4 与现实/直觉的连接

- **类比**：Michael 加成就像一个"插队"行为——亲核试剂（插队者）不直接找"队长"（羰基碳），而是找"队尾"（β-碳），因为这样队伍更稳定（保留羰基）。
- **反例**：直觉认为"亲核试剂总是进攻最正电的碳"。实际上，软亲核试剂根据**轨道匹配**选择进攻位点，β-碳的 LUMO 系数可能更适合软亲核的 HOMO 重叠。

## 十三、竞赛拓展

1. **不对称 Michael 加成**：
   - 手性 Lewis 酸（如 Cu-BOX 配合物）或有机催化剂控制立体选择性
   - 可高 ee 值构建手性季碳中心

2. **vinylogous Michael 加成**：
   - 共轭体系延长时的远程加成
   - α,β,γ,δ-不饱和羰基的 1,6-加成

3. **aza-Michael 加成**：
   - 胺或亚胺作为亲核体的 Michael 变体
   - 构建 C–N 键的重要方法

4. **串联 Michael/Michael 反应**：
   - 双 Michael 加成构建复杂环系
   - 在生物碱和萜类合成中广泛应用
