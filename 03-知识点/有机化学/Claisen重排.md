# Claisen 重排（Claisen Rearrangement）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[49-周环反应]]

---

## 一、定义

**Claisen 重排**：**烯丙基芳基醚**在**加热**条件下发生**[3,3]-σ 迁移**，重排为 **邻烯丙基苯酚** 的反应。

**通式**：
$$
\ce{ArO-CH2-CH=CH2 ->[\Delta] o-HO-C6H4-CH2-CH=CH2}
$$

**本质**：是**周环反应**的经典入门案例——协同的六电子迁移，无中间体，无催化剂。

## 三、核心原理

### 3.1 反应机理

**协同的 [3,3]-σ 迁移**：
```
        O                       OH
        ||                      |
    Ar—O—CH2—CH=CH2  →  Ar—C—CH2—CH=CH2
          \    /               |
           [3,3]              o-位
```

**关键特征**：
- **一步协同**：六个电子同时迁移，无中间体
- **六元环过渡态**：椅式构象优先
- **热驱动**：通常 150-250°C
- **无催化剂**：纯粹的周环反应

### 3.2 为什么是邻位产物？

**第一次 [3,3]-迁移** → 生成邻烯丙基环己二烯酮（非芳香性中间体）
**互变异构（烯醇化）** → 恢复芳香性 → 邻烯丙基苯酚

**如果邻位被占据**：
- 发生**第二次 [3,3]-迁移**（Cope 重排）→ 对烯丙基苯酚
- 总结果是**两次 [3,3]-迁移** → 对位产物

### 3.3 脂肪族 Claisen 重排（Clayden Ch35 深化）

**烯丙基乙烯基醚** → γ,δ-不饱和羰基化合物

**起始原料制备——缩醛交换法**（Clayden Ch35）：
- 醛的缩醛与烯丙基醇的**酸催化交换**（丙酸作催化剂，高沸点）
- 第一分子甲醇由 **SN1** 失去，第二分子由 **E1** 失去
- 烯丙基乙烯基醚形成后立即重排
- 甲醇是最易挥发的组分，被蒸馏出来驱动平衡

**适用范围**：醛、酮、酯、酰胺均可
- 酮/醛：缩醛本身
- 酯：原酸酯（orthoesters）
- 酰胺：原酸酰胺（orthoamides，又称"氨基缩醛"）

**共同特征**：产物都是 **γ,δ-不饱和羰基**——合成中需要此单元时，请使用 Claisen 重排

### 3.4 立体化学——E 选择性的来源（Clayden Ch35）

- 在与氧原子相邻的饱和碳原子上有取代基时，产生立体化学
- 所得双键**强烈偏好反式(E)**几何结构
- 原因：取代基倾向于在**椅式过渡态中处于平伏位置**
- 产物中R保持平伏位点，新烯烃双键的反式构型已在过渡态中确定

### 3.5 Claisen 重排的变体

| 变体 | 底物 | 产物 | 特点 |
|:---|:---|:---|:---|
| **芳香 Claisen** | 烯丙基芳基醚 | 邻烯丙基苯酚 | 经典，需互变异构恢复芳香性 |
| **脂肪 Claisen** | 烯丙基乙烯基醚 | γ,δ-不饱和醛/酮 | 合成价值高 |
| **Ireland-Claisen** | 烯丙基酯烯醇硅醚 | γ,δ-不饱和酸 | 低温，高选择性 |
| **Johnson-Claisen** | 烯丙醇 + 原酸酯 | γ,δ-不饱和酯 | 原位生成烯丙基乙烯基醚 |
| **Carroll 重排** | β-酮烯丙基酯 | γ,δ-不饱和酮 | [3,3]-σ迁移后脱羧 |

**Ireland-Claisen 重排**（Clayden Ch35 详细补充）：
- **发明者**：R. E. Ireland，1970s
- **底物**：烯丙醇的酯 → LDA处理得**烯醇锂**
- **可选**：烯醇锂转化为**烯醇硅醚**（OSiMe₃）后加热重排
- **产物**：不饱和酸（后处理后）
- **E选择性**：与普通Claisen相同的原因，双键强烈偏好反式(E)

### 3.6 与 Cope 重排的关系

| | **Claisen 重排** | **Cope 重排** |
|:---|:---|:---|
| **底物** | 烯丙基乙烯基醚 / 烯丙基芳基醚 | 1,5-二烯 |
| **过渡态** | 六元环（含氧）| 六元环（全碳）|
| **驱动力** | 生成 C=O（强键）| 生成更稳定的烯烃 |
| **温度** | 较低（~150°C）| 较高（~200°C）|

## 四、关键结论

1. **Claisen 重排 = [3,3]-σ 迁移 = 周环反应**
2. **邻位产物为主**——邻位被占时经两次迁移得对位产物
3. **协同机理，无中间体**——典型的周环反应特征
4. **热驱动，无催化剂**——高温即可
5. **椅式六元环过渡态**——立体化学由过渡态构象决定
6. **脂肪族 Claisen 是合成 γ,δ-不饱和羰基的重要方法**

| | **Claisen** | **Diels-Alder** |
|:---|:---|:---|
| **类型** | [3,3]-σ 迁移 | [4+2] 环加成 |
| **过渡态** | 六元环 | 六元环 |
| **产物** | 开链（或恢复芳环）| 六元环 |
| **驱动力** | σ 键重组 | 形成两个 σ 键 |

## 六、适用条件与限制

1. **需要高温**（150-250°C）——周环反应的热驱动特征
2. **底物必须含烯丙基 + 乙烯基/芳基醚**结构
3. **对酸/碱敏感基团不友好**——高温下可能副反应
4. **邻位被占时得对位产物**——但需两次迁移，温度可能更高
5. **立体化学由椅式过渡态决定**——手性底物可得手性产物

---

## 七、常见比较与易混点

### 7.1 Claisen 与 SN2'

- Claisen：[3,3]-σ 迁移，协同，无离子中间体
- SN2'：烯丙基体系的亲核取代，分步，有中间体
- 两者都可能涉及烯丙基重排，但机理完全不同

### 7.2 芳香 Claisen 与脂肪 Claisen

- 芳香 Claisen：产物是苯酚，需互变异构恢复芳香性
- 脂肪 Claisen：产物是羰基化合物，直接生成

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：烯丙基苯醚加热后的产物是什么？

**解答**：
- [3,3]-σ 迁移 → 邻烯丙基环己二烯酮中间体
- 互变异构 → **邻烯丙基苯酚**

### 10.2 例题 2（★★ 邻位被占）

**题目**：2-甲基烯丙基苯醚加热后的产物是什么？

**解答**：
- 邻位被 -CH₃ 占据
- 第一次 [3,3]-迁移 → 无法互变异构（邻位被占）
- 第二次 [3,3]-迁移（Cope）→ **对烯丙基苯甲酚**

### 10.3 例题 3（★★★ 立体化学）

**题目**：手性烯丙基醚在 Claisen 重排中，手性是否保持？

**解答**：
- 椅式过渡态中，取代基倾向于占据假平伏键（equatorial-like）
- 手性信息从烯丙基转移到产物的新手性中心
- **手性转移是立体专一的**——周环反应的标志性特征

## 十一、易错点

1. ❌ **产物是烯丙氧基苯** → 产物是**邻烯丙基苯酚**（C-C 键形成）
2. ❌ **邻位被占时不反应** → 邻位被占时**两次迁移得对位产物**
3. ❌ **Claisen 是离子型反应** → Claisen 是**周环反应，无中间体**
4. ❌ **需要路易斯酸催化** → 经典 Claisen **不需要催化剂**
5. ❌ **与 SN2' 机理相同** → 机理完全不同（协同 vs 分步）

### 12.3 入门例题

**题目**：为什么 Claisen 重排是 [3,3]-σ 迁移而不是 [1,5]-σ 迁移？

**引导**：
- 画烯丙基乙烯基醚的结构
- 数迁移的 σ 键两端的原子数：一边是 O-C-C（3个原子），另一边是 C=C-C（3个原子）
- 3+3 → [3,3]-迁移
- [1,5] 迁移需要不同的几何排列

### 12.4 现实类比

**Claisen 重排像"交换舞伴"**：
- 两对舞伴（烯丙基和乙烯基醚的两条链）
- 音乐响起（加热）
- 六个人手拉手转一圈（六元环过渡态）
- 松开手后，每人都换了新舞伴（σ 键重组）
- 整个过程一气呵成，没有"落单"的中间状态（协同机理）

## 十三、竞赛拓展

### 13.1 不对称 Claisen 重排

- 手性助剂控制过渡态构象
- 高对映选择性构建手性中心

### 13.2 aza-Claisen 重排

- 氮原子替代氧原子
- 烯丙基烯胺 → γ,δ-不饱和亚胺

### 13.3 Fischer 吲哚合成中的 [3,3]-σ重排（Clayden Ch35）

- 苯肼 + 酮 → 苯腙 → 烯胺（可烯醇化的酮）
- 烯胺具有 [3,3]-σ重排的理想结构
- 断裂的σ键是**弱的N-N σ键**，一根π键在苯环内
- 产物：高度不稳定的双键亚胺 → 芳香性恢复 → 吲哚
- 这是包含**两个氮原子**的[3,3]-σ重排

![[mineru/中文版clayden-chinese-34-36907-999_images/36c9317b22a277071e5d18652014dead9c8a5a64c8e011d8a97757b07174e1c.jpg]]

### 13.4 Cr(VI) 氧化中的共轭氧化（Clayden Ch35）

- 叔烯丙基醇 + CrO₃/酸性溶液
- 铬酸酯形成后，通过 **[3,3]-σ重排** 转移到烯丙基末端
- 两个氧和一个铬原子参与的 [3,3]-σ重排
- 这是Claisen重排原理在氧化反应中的延伸

### 13.5 柠檬醛的工业合成（Clayden Ch35）

- 维生素A合成的关键中间体
- **连续两个 [3,3]-σ重排**：Claisen → Cope
- Claisen的驱动力：羰基形成
- Cope的驱动力：末端双键→三取代双键 + 共轭结构

![[mineru/中文版clayden-chinese-34-36907-999_images/5221b16b595e7d2f6660f2918e18d11399494f76d121a4f9285bcf1af0dd84bc.jpg]]

### 13.6 美洲蜚蠊酮 B 合成中的 Claisen/Cope 应用（Clayden Ch35）

- Schreiber (1984) 合成
- 光化学 [2+2] 环加成 → **Cope 重排**（被烷氧基取代基加速）
- 然后 4π 电环化开环 → 12元环
- 展示了周环反应在复杂天然产物合成中的威力
