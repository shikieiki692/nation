# Wacker 氧化（Wacker Oxidation）
- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[40-氧化还原反应]]、[[51-金属有机]]
---

## 一、定义

**Wacker 氧化（Wacker-Tsuji Oxidation）**：在 **PdCl₂/CuCl** 催化体系和**氧气（或空气）**存在下，**末端烯烃**被氧化为**甲基酮**（或内烯烃被氧化为相应酮）的反应。

**通式**：
$$
\ce{R-CH=CH2 + O2 ->[PdCl2/CuCl] R-CO-CH3}
$$

**本质**：**Pd(II) 催化的烯烃氧化**——将廉价的烯烃转化为有价值的羰基化合物，是工业上合成丙酮和丁酮的经典方法。

## 三、核心原理

### 3.1 催化循环

**双金属协同循环**：

```
步骤 1：烯烃配位 + 水分子亲核进攻
  PdCl₂ + R-CH=CH2 → [Pd(II)-烯烃络合物]
  H2O 进攻 → Pd-CH2-CH(OH)-R

步骤 2：β-H 消除
  Pd-CH2-CH(OH)-R → Pd(0) + 烯醇 → 互变异构为酮

步骤 3：CuCl 再氧化 Pd(0)
  Pd(0) + 2 CuCl2 → PdCl2 + 2 CuCl

步骤 4：O2 再生 Cu(II)
  2 CuCl + 2 HCl + 1/2 O2 → 2 CuCl2 + H2O
```

**总反应**：
$$
\ce{R-CH=CH2 + 1/2 O2 ->[PdCl2/CuCl] R-CO-CH3}
$$

**关键点**：
- **Pd(II)** 是真正的氧化剂（被还原为 Pd(0)）
- **CuCl** 是"电子传递员"——将 Pd(0) 再氧化为 Pd(II)
- **O₂** 是最终的氧化剂——再生 Cu(II)
- 整个体系中，**Pd 和 Cu 都是催化剂**，只有 O₂ 被消耗

### 3.2 区域选择性

**马氏取向（Markovnikov）**：
- 水分子进攻烯烃的 **取代较多的碳**
- OH 连在 β-碳上 → 烯醇互变异构后 → 羰基在 **取代较多的碳**上
- 末端烯烃 → **甲基酮**

| 烯烃 | 产物 |
|:---|:---|
| **丙烯** | 丙酮 |
| **1-丁烯** | 2-丁酮（甲乙酮）|
| **苯乙烯** | 苯乙酮 |
| **环己烯** | 环己酮 |

### 3.3 立体化学

- 水分子从 **Pd(II) 配位烯烃的反位**进攻
- 反式加成 → 烯醇 → 互变异构为酮
- 由于最终产物是平面型的酮，立体化学不重要

### 3.4 Wacker 氧化的变体

**Tsuji-Wacker 氧化**：
- 使用 **DMF/H₂O** 混合溶剂
- 用 **苯醌（BQ）** 替代 O₂ 作为共氧化剂
- 条件更温和，适用于敏感底物

**分子内 Wacker**：
- 含有烯烃和亲核体（如胺、醇）的同一分子
- 分子内亲核进攻 → 环化产物

---

## 四、关键结论

1. **Wacker = 烯烃 + O₂ → 酮**
2. **Pd(II) 催化，Cu(I)/Cu(II) 共催化**
3. **马氏取向**——羰基在取代较多的碳上
4. **末端烯烃 → 甲基酮**
5. **Pd 和 Cu 都是催化剂，O₂ 是最终氧化剂**
6. **Tsuji 变体用苯醌替代 O₂**

## 五、常见分类或情形

### 5.1 按底物分类

| 烯烃类型 | 产物 | 例子 |
|:---|:---|:---|
| **末端烯烃** | 甲基酮 | 丙烯 → 丙酮 |
| **对称内烯烃** | 单一酮 | 2-丁烯 → 2-丁酮 |
| **不对称内烯烃** | 混合物（两种酮）| 1-丁烯 + 2-丁烯混合物 |

### 5.2 按氧化剂分类

| 体系 | 氧化剂 | 特点 |
|:---|:---|:---|
| **经典 Wacker** | O₂（空气）| 最便宜、最绿色 |
| **Tsuji-Wacker** | 苯醌（BQ）| 条件温和 |
| **电化学 Wacker** | 电极 | 无需化学氧化剂 |

## 六、适用条件与限制

1. **需要 PdCl₂/CuCl 体系**
2. **需要水作为亲核试剂**
3. **需要 O₂ 或共氧化剂**
4. **溶剂**：DMF/H₂O、丙酮/H₂O、THF/H₂O
5. **pH 控制**：酸性条件（HCl）抑制 Pd 沉淀
6. **末端烯烃效果最好**
7. **某些官能团可能受影响**（如醛基可能被进一步氧化）

## 七、常见比较与易混点

### 7.1 Wacker vs 硼氢化-氧化

| | **Wacker 氧化** | **硼氢化-氧化** |
|:---|:---|:---|
| **试剂** | PdCl₂/CuCl/O₂ | BH₃ / H₂O₂, NaOH |
| **取向** | **马氏** | **反马氏** |
| **产物** | 酮 | 醇 |
| **机理** | Pd 催化氧化 | 硼烷加成 + 氧化 |

|:---|:---|:---|
| **产物** | 酮 | 醇 |
| **深度** | 氧化 | 不氧化 |
| **选择性** | 马氏 | 马氏 |

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：丙烯在 Wacker 氧化条件下的产物是什么？

**解答**：
- 丙烯 = CH₃-CH=CH₂（末端烯烃）
- 马氏取向 → 水进攻取代较多的碳（中间的碳）
- 产物：**丙酮（CH₃-CO-CH₃）**

### 10.2 例题 2（★★ 机理）

**题目**：Wacker 氧化中 CuCl 的作用是什么？

**解答**：
- Pd(II) 氧化烯烃 → 被还原为 Pd(0)
- CuCl₂ **再氧化 Pd(0) 为 Pd(II)**
- CuCl₂ 自身被还原为 CuCl
- CuCl 被 O₂ **再生为 CuCl₂**
- **结论**：CuCl 是 Pd 的"再生助手"，O₂ 是最终的氧化剂

### 10.3 例题 3（★★★ 不对称烯烃）

**题目**：1-甲基环己烯的 Wacker 氧化产物是什么？

**解答**：
- 环烯烃 → 环酮
- 马氏取向 → 羰基在取代较多的碳上
- 产物：**2-甲基环己酮**

## 十一、易错点

1. ❌ **Wacker 产物是醇** → Wacker 产物是 **酮**
2. ❌ **Wacker 反马氏** → Wacker 是 **马氏**取向
3. ❌ **CuCl 是催化剂，Pd 是共催化剂** → **Pd 是主催化剂**，CuCl 是共催化剂
4. ❌ **Wacker 和硼氢化一样** → 取向**相反**（Wacker 马氏，硼氢化反马氏）
5. ❌ **Wacker 不需要氧气** → **需要 O₂**再生 Cu(II)

### 12.3 入门例题

**题目**：为什么 Wacker 氧化中需要 CuCl？不能直接只用 PdCl₂ 和 O₂ 吗？

**引导**：
- 理论上可以，但 Pd(0) 被 O₂ 氧化的速度很慢
- CuCl₂ 氧化 Pd(0) 为 Pd(II) 的速度很快
- CuCl 被 O₂ 再生的速度也很快
- CuCl 充当了 Pd 和 O₂ 之间的"电子传递员"
- 就像接力赛跑，Cu 是中间接力棒，O₂ 是最后一棒

### 12.4 现实类比

**Wacker 像"氧化工厂"**：
- 烯烃 = 原材料（石油裂解产物）
- Pd(II) = 主要工人（负责氧化烯烃）
- Pd(0) = 工人干完活后累了（需要休息/充电）
- CuCl₂ = 充电器（把累倒的 Pd(0) 变回 Pd(II)）
- O₂ = 发电厂（给充电器供电）
- 整个工厂只消耗电（O₂）和原材料（烯烃），工人（Pd）和充电器（Cu）可以一直用

## 十三、竞赛拓展

1. **分子内 Wacker 氧化**：
   - 含有烯烃和胺/醇的底物
   - 分子内亲核进攻 → 含氮/氧杂环
   - 是合成生物碱和糖苷的有力工具

2. **不对称 Wacker 氧化**：
   - 手性配体控制的氧化
   - 构建手性酮
   - 目前仍在发展中，竞赛中较少涉及
