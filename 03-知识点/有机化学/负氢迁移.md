# 负氢迁移

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[39-重排反应]]、[[40-氧化还原反应]]

## 一、定义

**负氢迁移（Hydride Shift）**：氢原子以氢负离子（H⁻）的形式从一个碳原子迁移到相邻的缺电子中心（通常是碳正离子）的过程。

**本质**：C-H σ 键的电子对转移到相邻的空 p 轨道，是 Wagner-Meerwein 重排的最常见形式。

## 二、考纲对应

- 对应考纲条目：[[39-重排反应]]、[[40-氧化还原反应]]
- 所属模块：[[基础要求-有机化学]]
- 本知识点在考纲中的作用：负氢迁移是碳正离子重排的核心内容，也是多种氧化还原反应（如 Cannizzaro、MPV）的共性机理。

## 三、核心原理

### 3.1 分子轨道分析

**负氢迁移的轨道相互作用**（Clayden Ch.36）：
- HOMO = C-H σ 键（给体）
- LUMO = 碳正离子的空 p 轨道（受体）
- C-H σ 键电子对流入空 p 轨道 → H 迁移 → 正电荷转移

**几何要求**：
- 1,2-迁移（相邻碳之间）
- C-H 键与空 p 轨道需有效重叠

### 3.2 与烷基迁移的比较

| 迁移类型 | 速率 | 原因 |
|:---|:---:|:---|
| **负氢迁移** | 通常更快 | H 体积小，轨道重叠好 |
| **甲基迁移** | 较慢 | 位阻大，轨道重叠较差 |
| **芳基迁移** | 可能更快 | 可通过 π 体系离域过渡态电荷 |

**一般优先级**：芳基 > H > 烷基（但具体情况取决于底物）

### 3.3 负氢迁移在氧化还原反应中的应用

**ABOC §4.8 强调**：负氢迁移是多种反应的共性机理。

| 反应 | 负氢迁移的角色 |
|:---|:---|
| **Wagner-Meerwein 重排** | 碳正离子重排的核心步骤 |
| **Cannizzaro 反应** | 无 α-H 醛的歧化：一分子给负氢，一分子接受负氢 |
| **Meerwein-Ponndorf-Verley（MPV）还原** | 异丙醇铝将负氢转移给醛/酮 |
| **Oppenauer 氧化** | MPV 的逆反应：醇给负氢给丙酮 |
| **Tishchenko 反应** | 醛的歧化生成酯，涉及负氢转移 |

> ABOC："负氢迁移的普遍性：Cannizzaro、MPV、Tishchenko、Meerwein-Ponndorf 都共享同一个负氢转移机理——学会一个等于学会四个。"

## 四、关键结论

1. **负氢迁移是碳正离子重排的最常见形式**
2. **负氢迁移通常比烷基迁移快**——H 体积小，轨道重叠好
3. **负氢迁移的驱动力是生成更稳定的碳正离子**
4. **负氢迁移是多种氧化还原反应的共性机理**——Cannizzaro、MPV、Oppenauer、Tishchenko
5. **1,2-负氢迁移中，H 的立体化学保持**

## 五、常见分类或情形

### 5.1 按反应类型分类

| 类型 | 反应 | 负氢来源 | 负氢受体 |
|:---|:---|:---|:---|
| **碳正离子重排** | W-M 重排 | 相邻 C-H | 碳正离子 |
| **歧化反应** | Cannizzaro | 醛（无 α-H）| 另一分子醛 |
| **负氢转移** | MPV 还原 | 异丙醇 | 醛/酮 |
| **负氢转移** | Oppenauer 氧化 | 2° 醇 | 丙酮 |
| **歧化** | Tishchenko | 醛 | 另一分子醛 |

### 5.2 经典案例

**Cannizzaro 反应**：
- 底物：无 α-H 的醛（如苯甲醛、甲醛）
- 试剂：强碱（NaOH）
- 产物：一分子醇 + 一分子羧酸
- 机理：负氢迁移（一个醛给 H⁻，另一个醛接受 H⁻）

**MPV 还原**：
- 酮 + 异丙醇 → 2° 醇 + 丙酮
- Al(OiPr)₃ 催化
- 异丙醇给负氢给酮

## 六、适用条件与限制

1. **必须有缺电子中心接受负氢**——碳正离子或极化的羰基碳
2. **负氢供体必须有可迁移的 H**——通常需要 α-H
3. **1,2-迁移是最常见的**——远程迁移较少见
4. **与消除反应竞争**——强碱条件下消除可能更快
5. **芳基迁移有时比 H 迁移更快**——取决于具体底物

## 七、常见比较与易混点

### 7.1 负氢迁移 vs 质子转移

| | **负氢迁移** | **质子转移** |
|:---|:---|:---|
| 迁移物种 | H⁻（氢负离子，带电子对）| H⁺（质子，无电子）|
| 电子流动 | C-H 键电子对跟随 H | 无电子跟随 |
| 发生场景 | 碳正离子、氧化还原 | 酸碱反应 |
| 结果 | 正电荷转移 | 电荷中和 |

### 7.2 负氢迁移在 W-M 重排 vs 氧化还原中

| | **W-M 重排** | **氧化还原（MPV/Cannizzaro）** |
|:---|:---|:---|
| 驱动力 | 碳正离子稳定性 | 氧化态变化 |
| 受体 | 碳正离子 | 极化羰基碳 |
| 供体 | 相邻 C-H | 醇或醛 |
| 催化剂 | 酸 | 碱或金属烷氧基 |

## 十、例题

### 例题 1（★ 基础）

**题目**：2-甲基-2-丁醇在酸催化下脱水，写出涉及负氢迁移的机理。

**解答**：
1. 质子化 OH → 失去 H₂O → 2° 碳正离子
2. **负氢迁移** → 3° 碳正离子（更稳定）
3. 失去 H⁺ → 2-甲基-2-丁烯

### 例题 2（★★ 氧化还原）

**题目**：写出 Cannizzaro 反应的负氢迁移机理。

**解答**：
1. OH⁻ 进攻苯甲醛 → 四面体中间体
2. 中间体给 **负氢** 给另一分子苯甲醛
3. 给负氢的分子 → 苯甲酸
4. 接受负氢的分子 → 苯甲醇

## 十一、易错点

1. ❌ **混淆负氢迁移和质子转移** → 负氢是 H⁻（带电子对），质子是 H⁺
2. ❌ **认为负氢迁移只发生在碳正离子中** → 也发生在氧化还原反应中
3. ❌ **忽略负氢迁移的可逆性** → 在 W-M 重排中是平衡过程
4. ❌ **认为烷基迁移总是比 H 慢** → 芳基迁移有时比 H 快

|---|---|

| "负氢迁移只发生在酸中" | 碱中也可（如 Cannizzaro）|
| "所有重排都是负氢迁移" | 还有烷基迁移、芳基迁移 |

### 12.3 入门级例题

**题目**：为什么 2° 碳正离子常通过负氢迁移变成 3° 碳正离子？

**解答**：
1. 3° 碳正离子比 2° 更稳定
2. 负氢迁移的活化能低（H 体积小）
3. 只要相邻碳有 H，负氢迁移就很快发生
4. 结果是生成更稳定的碳正离子

### 12.4 与现实/直觉的连接

- 负氢迁移就像"传递火炬"——H 带着它的电子对（火炬）从一个人（碳）传给另一个人（碳正离子）。
- Cannizzaro 反应就像"互帮互助"——一个醛分子把"能量"（负氢）给另一个，自己变成酸，对方变成醇。
