# 烯醇

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 关联笔记：[[烯醇负离子]], [[羰基α位反应]], [[Aldol缩合]]

## 一、定义

烯醇（Enol）是指含有羟基直接连在双键碳上的结构（$\text{R}_2\text{C}=\text{C(OH)R}$），是羰基化合物（醛、酮、酯）的互变异构体。酮-烯醇互变异构（Keto-Enol Tautomerism）是有机化学中最重要的平衡之一，是羰基 α-位反应性的根源。

## 二、考纲对应

- 对应考纲条目：[[46-羰基α位反应]]
- 所属模块：[[基础要求-有机化学]]
- 本知识点在考纲中的作用：烯醇/烯醇负离子是**羰基 α-位所有反应性的根源**——卤代、Aldol 缩合、Claisen 缩合、烷基化等均以烯醇或烯醇负离子为关键中间体。理解烯醇化学是掌握羰基 α-反应机理的前提。

## 三、核心原理

### 3.1 酮-烯醇互变异构

$$\text{R}_2\text{CH-CO-R} \rightleftharpoons \text{R}_2\text{C}=\text{C(OH)-R}$$

- **酸催化机理**：质子化羰基氧 → 脱水形成烯醇式
- **碱催化机理**：碱夺取 α-H 形成烯醇负离子 → 质子化氧得烯醇

### 3.2 烯醇含量影响因素

| 因素 | 对烯醇含量的影响 | 例子 |
|------|----------------|------|
| **α-H 酸性** | 酸性越强，烯醇含量越高 | 1,3-二羰基化合物烯醇含量很高 |
| **共轭稳定** | 烯醇能形成分子内氢键或共轭体系时含量升高 | 乙酰丙酮烯醇式 > 80% |
| **溶剂** | 质子溶剂有利于酮式；非质子溶剂可能有利于烯醇 | — |
| **温度** | 一般低温有利于烯醇（形成氢键） | — |

### 3.3 烯醇的反应性

烯醇是羰基 α-位亲电取代反应的关键中间体：
- **α-卤代**：烯醇双键与 X₂ 反应，生成 α-卤代羰基化合物
- **Aldol 加成**：烯醇作为亲核试剂进攻另一分子羰基
- **Mannich 反应**：烯醇与亚胺正离子反应

## 四、关键结论

1. **烯醇负离子（enolate）是羰基 α-化学的"中枢中间体"**——几乎所有 α-位反应（卤代、烷基化、Aldol、Claisen、Mannich）都经过烯醇负离子或其等价物。谁控制了烯醇负离子的生成，谁就控制了 α-位的反应性。
2. **动力学 vs 热力学烯醇负离子的区域选择性由碱和条件决定**——用 LDA（强位阻碱，低温，THF）优先生成**动力学烯醇负离子**（较少取代的 α-碳去质子化）；用 NaH/NaOEt/EtOH 回流则生成**热力学烯醇负离子**（较多取代的 α-碳去质子化，烯醇负离子更稳定）。
3. **pKa 是预测烯醇负离子生成可行性的核心参数**——羰基 α-H 的 pKa 约 16-20（酮）或 23-25（酯），碱的共轭酸 pKa 必须高于底物 α-H 的 pKa 才能有效去质子化。LDA 的共轭酸（二异丙胺）pKa ~36，足以定量去质子化几乎所有羰基化合物。
4. **烯醇负离子的几何构型（E/Z）直接决定后续反应的立体化学**——在六元环过渡态（如 Zimmerman-Traxler 模型）中：Z-烯醇负离子 → syn 产物，E-烯醇负离子 → anti 产物。控制烯醇负离子几何构型是立体选择性合成的核心策略（Ireland 模型：THF/HMPA → E-烯醇负离子 vs THF → Z-烯醇负离子）。
5. **酮-烯醇互变异构平衡常数 $K_T = [\text{enol}]/[\text{ketone}]$ 极低（通常 $10^{-4}\sim 10^{-6}$），但足以驱动催化量的 α-反应**——一旦烯醇/烯醇负离子被亲电试剂捕获，平衡持续向右移动，使 α-位反应可以定量完成。
6. **β-二羰基化合物（如乙酰丙酮、丙二酸酯）烯醇含量极高（可达 80%+）**——原因是分子内氢键 + 共轭双键使烯醇式获得额外稳定化能（约 8-12 kcal/mol）。

## 五、常见分类或情形

### 5.1 动力学烯醇负离子 vs 热力学烯醇负离子

| | **动力学烯醇负离子** | **热力学烯醇负离子** |
|:---|:---|:---|
| **生成位点** | 较少取代的 α-碳（位阻较小的 C-H）| 较多取代的 α-碳（更稳定的 C=C）|
| **控制因素** | 去质子化速率（位阻控制）| 产物稳定性（热力学控制）|
| **典型条件** | LDA, THF, −78°C | NaH, NaOEt/EtOH, 回流或室温 |
| **碱的特征** | 强位阻碱，不可逆去质子化 | 较弱碱，可逆去质子化 |
| **不对称酮例子** | 2-甲基环己酮 + LDA → 少取代端烯醇负离子（6-位）| 2-甲基环己酮 + NaH → 多取代端烯醇负离子（2-位）|

### 5.2 按反离子分类的烯醇负离子

| 反离子 | 特征 | 典型用途 |
|:---|:---|:---|
| **Li⁺** | 紧密结合，倾向于形成聚集态（四聚体、六聚体），Z-选择性高（THF 中）| Aldol 缩合、烷基化 |
| **Na⁺ / K⁺** | 较松散离子对，E-选择性增加，反应性更强但选择性降低 | 热力学烯醇负离子生成 |
| **Mg²⁺ (MgBr⁺)** | 中等紧密度，可通过配位导向 | 区域选择性烷基化 |
| **Zn²⁺** | 温和，可用于 Reformatsky 反应（α-溴代酯 + Zn → Zn-烯醇负离子）| Reformatsky 反应 |

### 5.3 烯醇负离子等价物

| 等价物 | 结构通式 | 特点 | 典型反应 |
|:---|:---|:---|:---|
| **烯醇硅醚** | $\text{R}_2\text{C}=\text{C(OSiR}_3\text{)R}$ | 可"锁定"特定 E/Z 几何构型；稳定、可分离 | Mukaiyama Aldol 缩合 |
| **烯醇硼酸酯** | $\text{R}_2\text{C}=\text{C(OBR}_2\text{)R}$ | B-O 键强，构型高度保持 | 立体选择性 Aldol |
| **烯胺** | $\text{R}_2\text{C}=\text{C(NR}_2\text{)R}$ | N 的给电子使 β-C 亲核性更强；可催化循环 | Stork 烯胺烷基化/酰基化 |
| **烯醇锂** | $\text{R}_2\text{C}=\text{C(OLi)R}$ | 最常用，原位生成直接使用 | Aldol, Claisen, 烷基化 |

### 5.4 区域化学分类

| 类型 | 定义 | 控制策略 |
|:---|:---|:---|
| **specific enolate** | 不对称酮只有一个 α-位被去质子化（高区域选择性）| 动力学条件（LDA, −78°C）|
| **regiochemical mixture** | 两个 α-位都被去质子化（低区域选择性）| 热力学条件或非选择性碱 |

> ⚠️ **关键判断**：在竞赛中，看到"不对称酮 + 碱 + 亲电试剂"的组合，第一反应必须是判断条件属于动力学控制还是热力学控制——这直接决定取代发生在哪个 α-碳上。

## 六、适用条件与限制

1. **LDA 必须恰好 1 当量，低温（−78°C）操作**——LDA 本身是亲核性很低的强碱，过量的 LDA 不会继续去质子化但可能带来副反应。温度升高会导致动力学烯醇负离子异构化为热力学产物。
2. **酯的 α-去质子化比酮更困难**——酯的 α-H pKa ~23-25（酮 ~16-20），需要更强的碱（如 LDA）。此外酯的烯醇负离子容易发生自缩合（Claisen 型），需要控制计量和温度。
3. **溶剂对烯醇负离子几何构型有决定性影响（Ireland 模型）**——以酯的锂烯醇负离子为例：THF 中主要生成 Z-烯醇负离子（Li⁺ 与 O 和 N 螯合形成六元环过渡态）；THF + HMPA（或 DMPU）中主要生成 E-烯醇负离子（HMPA 破坏 Li⁺ 螯合，过渡态变为开放链状）。
4. **反离子效应不可忽略**——Li⁺ 因离子半径小、电荷密度高，与烯醇负离子 O 紧密结合，倾向于 Z-构型；Na⁺/K⁺ 离子半径大、结合松散，E-选择性增加。在实际合成中，可通过添加 HMPA、DMPU 或冠醚来调节反离子效应。
5. **不对称酮的区域选择性取决于碱的位阻**——LDA（i-Pr₂N⁻，大位阻）优先进攻位阻小的 α-H → 动力学产物；NaH/NaOEt（位阻小）允许可逆平衡 → 热力学产物。LiTMP（四甲基哌啶锂）位阻比 LDA 更大，区域选择性更高。
6. **β-二羰基化合物在质子溶剂中烯醇含量极高**——此时烯醇式通过分子内氢键和共轭稳定，pKa 降至 ~9-11（乙酰丙酮 pKa ~9），用 NaOH/EtOH 即可完全去质子化，无需 LDA。
7. **非对称酮在强碱下可能发生自身 Aldol 缩合**——如果亲电试剂添加不够快，烯醇负离子可能进攻未反应的酮。解决方案：预先生成烯醇负离子（LDA, −78°C, THF），然后缓慢加入亲电试剂。

## 七、常见比较与易混点

### 7.1 烯醇（Enol）vs 烯醇负离子（Enolate）

| | **烯醇 (Enol)** | **烯醇负离子 (Enolate)** |
|:---|:---|:---|
| **结构** | $\text{C}=\text{C(OH)}$，中性 | $\text{C}=\text{C(O}^-\text{)}$，负离子 |
| **生成** | 酸/碱催化的互变异构 | 强碱（LDA 等）去质子化 |
| **反应性** | 亲核性较弱（OH 是弱给电子基）| 亲核性极强（O⁻ 是强给电子基）|
| **浓度** | 平衡含量极低（$K_T \ll 1$）| 可定量生成 |
| **典型反应** | α-卤代（酸催化）| Aldol, Claisen, 烷基化 |
| **选择性控制** | 几乎无立体控制 | 可通过烯醇负离子几何构型精确控制 |

> ⭐ **核心区分**：烯醇是互变异构体（平衡中的微量组分），烯醇负离子是活性中间体（可定量制备）。竞赛中涉及 α-反应机理时，绝大多数情况下真正起作用的亲核物种是**烯醇负离子**而非烯醇。

### 7.2 动力学烯醇负离子 vs 热力学烯醇负离子（详解）

| | **动力学烯醇负离子** | **热力学烯醇负离子** |
|:---|:---|:---|
| **形成的 C=C 位置** | 取代较少的双键（端烯型）| 取代较多的双键（内烯型）|
| **决定的物理量** | $\Delta G^\ddagger$（活化能）| $\Delta G$（热力学稳定性）|
| **去质子化过程** | 不可逆（碱的共轭酸 pKa ≫ 底物 pKa）| 可逆（碱的共轭酸 pKa ≈ 底物 pKa）|
| **碱** | LDA, LiTMP, LiHMDS | NaH, NaOEt, t-BuOK/protic solvent |
| **温度** | −78°C（低温冻结动力学产物）| 0°C ~ 回流（允许平衡）|
| **不对称酮位置** | 少取代 α-位 | 多取代 α-位 |
| **2-甲基环己酮 + LDA** | 6-位去质子化（CH₂，位阻最小）| — |
| **2-甲基环己酮 + NaH/Δ** | — | 2-位去质子化（CH₃-C=，烯醇更稳定）|

### 7.3 Z-烯醇负离子 vs E-烯醇负离子（Ireland 模型）

| | **Z-烯醇负离子** | **E-烯醇负离子** |
|:---|:---|:---|
| **O⁻ 和大取代基的相对位置** | 同侧（Z = Zusammen）| 异侧（E = Entgegen）|
| **酯的生成条件** | LDA, THF, −78°C（Li⁺ 螯合控制）| LDA, THF/HMPA (23%), −78°C（螯合破坏）|
| **酮的生成条件** | LDA, THF, −78°C（一般为主）| LiHMDS, THF/HMPA |
| **Aldol 产物构型** | Z-烯醇负离子 → syn-Aldol（六元环椅式 TS）| E-烯醇负离子 → anti-Aldol |

### 7.4 烯醇负离子 vs 烯胺（Enolate vs Enamine）

| | **烯醇负离子** | **烯胺** |
|:---|:---|:---|
| **亲核原子** | α-C（软亲核位点）| β-C（N 的给电子使 β-C 更富电子）|
| **碱性** | 强碱，易发生消除副反应 | 碱性弱，副反应少 |
| **反应类型** | Aldol, Claisen, 烷基化 | Stork 烷基化/酰基化, Michael 加成 |
| **催化** | 化学计量（1 eq 碱消耗）| 可催化量（胺 + 酸，生成水）|
| **区域选择性** | 取决于烯醇负离子的区域化学 | 烯胺通常只在较少取代的 α-位形成 |
| **典型条件** | LDA, −78°C, 无水无氧 | 吡咯烷, 催化量 TsOH, Dean-Stark 除水 |

### 7.5 烯醇硅醚的 E/Z 与烯醇负离子的 E/Z 不是同一概念

- 烯醇负离子的 E/Z 是指 **O⁻ 与 α-取代基的相对空间关系**
- 烯醇硅醚的 E/Z 是指 **OSiR₃ 与 α-取代基的相对空间关系**
- 制备条件决定了烯醇硅醚的 E/Z 构型，该构型在后续 Mukaiyama 反应中保持不变——因此"锁定"了烯醇负离子等价物的几何构型
- 但 Mukaiyama 反应走**开放过渡态**（Lewis 酸催化），立体化学规律与六元环椅式 TS 的经典 Aldol 不同

## 十、例题

### 10.1 例题 1（★ 基础：动力学 vs 热力学烯醇负离子预测）

**题目**：2-甲基环己酮分别在以下两种条件下用碱处理，然后用 CH₃I 捕获。预测两种条件下的主产物，并解释差异。

a) LDA (1.0 eq), THF, −78°C, 30 min; then CH₃I
b) NaH (1.0 eq), THF, 回流, 2 h; then CH₃I

**解答**：

- 2-甲基环己酮有两个 α-位：2-位（CH₃-C=O 的 α-C，有一个甲基取代）和 6-位（CH₂-C=O，无取代）
- **条件 a（LDA, −78°C）**：动力学控制。LDA 位阻大，优先进攻 6-位的 CH₂（位阻最小，去质子化最快）。生成 6-烯醇负离子（较少取代的双键）。CH₃I 烷基化 → **2-甲基-6-甲基环己酮**（即 2,6-二甲基环己酮）。
- **条件 b（NaH, 回流）**：热力学控制。碱可逆去质子化，平衡倾向于生成更稳定的烯醇负离子——2-位去质子化得到更多取代的烯醇负离子（双键在 2,3-之间，四取代双键 vs 三取代双键）。CH₃I 烷基化 → **2,2-二甲基环己酮**。

> 📝 **口诀**：LDA 低温攻少取代，NaH 回流攻多取代。

### 10.2 例题 2（★★ 中阶：不对称酮的区域选择性烷基化）

**题目**：3-甲基-2-丁酮 $(CH_3)_2CHCOCH_3$ 与 LDA（1.0 eq, THF, −78°C）反应后加入 PhCH₂Br，预测主产物。如果改用 NaOEt/EtOH 回流条件呢？

**解答**：

- 该酮有两个 α-位：乙酰基侧的 CH₃ 和异丙基侧的 CH(CH₃)₂
- **LDA 条件（动力学控制）**：
  - LDA 优先进攻位阻较小的 CH₃ 基团上的 H → 生成 $\text{(CH}_3\text{)}_2\text{CHCOCH}_2^-$
  - PhCH₂Br 烷基化 → **(CH₃)₂CHCOCH₂CH₂Ph**（1-苯基-4-甲基-3-戊酮）
  - 注意：该烯醇负离子是最少取代的（端烯型），虽然热力学上不是最稳定的
- **NaOEt/EtOH 回流（热力学控制）**：
  - 可逆去质子化，平衡趋于更稳定的烯醇负离子 → (CH₃)₂C=C(O⁻)CH₃（更多取代的双键）
  - PhCH₂Br 烷基化 → **(CH₃)₂C(CH₂Ph)COCH₃**（3,3-二甲基-4-苯基-2-丁酮）

> 📝 **核心教义**：不对称酮有两个不同的 α-位时，"用什么碱 + 什么温度"就决定了官能团化在哪里发生。

### 10.3 例题 3（★★★ 高阶：Ireland-Claisen 立体化学）

**题目**：丙酸乙酯 $(\text{CH}_3\text{CH}_2\text{COOEt})$ 在以下两种条件下分别用 LDA 处理生成烯醇负离子，然后与 PhCHO 进行 Aldol 缩合。预测两种条件下产物的相对构型（syn 或 anti），并简要说明理由。

a) LDA, THF, −78°C
b) LDA, THF/HMPA (23% v/v), −78°C

**解答**：

- 丙酸乙酯烯醇负离子的 E/Z 由 Ireland 模型决定：
  - **条件 a（THF）**：Li⁺ 与酯羰基氧和烯醇氧同时螯合，形成六元环过渡态 → 主要生成 **Z-烯醇负离子**（O⁻ 与 CH₃ 在同侧）。Z-烯醇负离子经六元环椅式 TS（Zimmerman-Traxler）进攻 PhCHO → **syn-Aldol 产物**（β-OH 与 α-CH₃ 在同侧）。
  - **条件 b（THF/HMPA）**：HMPA 是强配位溶剂，将 Li⁺ 从螯合位点剥离 → 过渡态变为开放链状，位阻控制 → 主要生成 **E-烯醇负离子**（O⁻ 与 CH₃ 在异侧）。E-烯醇负离子经六元环椅式 TS → **anti-Aldol 产物**。

> 🔬 **深层原理**：这就是 Ireland 模型的威力——仅凭溶剂的改变（加不加 HMPA），就能翻转 Aldol 产物的全部相对立体化学。

## 十一、易错点

1. ❌ **混淆动力学和热力学条件** → 看到"不对称酮 + 碱 + 亲电试剂"就默认取代发生在多取代 α-位，而忽略了碱的种类和温度。记住：LDA/−78°C 是动力学控制（攻位阻小的 α-H），NaH/NaOEt/Δ 是热力学控制（攻生成更稳定烯醇的 α-H）。
2. ❌ **忘记 LDA 必须恰好 1 当量** → LDA 的共轭酸（二异丙胺，pKa ~36）远高于底物 α-H，去质子化不可逆。如果 LDA < 1 eq，部分底物未转化；如果 LDA > 1 eq，多余的 LDA 可能作为亲核试剂进攻底物羰基（虽然 LDA 亲核性低，但在无 α-H 的底物上会发生）。
3. ❌ **忽略烯醇负离子几何构型对产物立体化学的影响** → 在有六元环过渡态的反应中（Aldol 缩合、Ireland-Claisen），Z-烯醇负离子 → syn 产物，E-烯醇负离子 → anti 产物。如果无视 E/Z 构型，立体化学完全预测错误。
4. ❌ **酯、酮、酰胺的 α-H pKa 混为一谈** → 酮 α-H pKa ~16-20，酯 ~23-25，酰胺 ~28-30。用同样条件（如 NaOEt/EtOH）对酮有效，对酯可能去质子化不完全，对酰胺基本无效。
5. ❌ **认为烯醇负离子只在 α-C 上有反应性** → 烯醇负离子是**双位亲核试剂**——O-端和 C-端均可反应。O-烷基化/酰基化（生成烯醇醚/酯）和 C-烷基化/酰基化是竞争反应。Li⁺ 反离子倾向于 C-烷基化（紧密离子对），Na⁺/K⁺ 倾向于 O-烷基化（松散离子对）。
6. ❌ **烯醇和烯醇负离子混为一谈** → 烯醇是中性互变异构体，亲核性弱；烯醇负离子是负离子，亲核性强百倍。酸催化的 α-卤代走的是烯醇机理，碱催化的 Aldol 走的是烯醇负离子机理——两者的条件和选择性完全不同。
7. ❌ **不对称酮自身的 Aldol 缩合被忽视** → 在生成烯醇负离子后，如果亲电试剂加入不及时，烯醇负离子可能进攻未反应的酮底物发生自身 Aldol。实用对策：先将酮完全转化为烯醇负离子（LDA, −78°C），再加入亲电试剂。
8. ❌ **β-二羰基化合物当成普通羰基处理** → 乙酰丙酮的 pKa 约 9（不是 20），NaOH 水溶液就能定量去质子化，不需要 LDA。其烯醇含量 ~80%，不应与普通单羰基的烯醇含量（<0.01%）相提并论。

  - [[动力学控制与热力学控制]]（烯醇负离子是理解 kin vs therm 的经典范例）

### 12.2 学生易踩的认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析口诀 |
|---|---|---|---|
| "烯醇就是烯醇负离子" | 名字相近，初学时上课统称"烯醇式" | 烯醇中性、含量微量；烯醇负离子负电、可定量制备 | "烯醇是平衡态，烯醇负离子是中间体" |
| "碱强就一定攻多取代" | 直觉上"取代越多越稳定" | 动力学控制下碱选择**位阻最小**的 H，而非生成最稳定的产物 | "低温攻少，回流攻多" |
| "LDA 加多少都行" | 习惯"碱过量"的安全操作 | 必须 1 eq——多了可能亲核加成，少了转化不完全 | "LDA 要恰好，不多也不少" |
| "E/Z 烯醇不重要" | 平面结构式看不出立体信息 | E/Z 直接决定 Aldol 产物的 syn/anti | "Z 构型出 syn，E 构型出 anti" |
| "所有羰基 α-H pKa 差不多" | 书上只记了"~20" | 酮 ~16-20, 酯 ~23-25, 酰胺 ~28-30, β-二羰基 ~9-11 | "酮酸酯碱酰胺更碱，二羰基酸到没朋友" |

### 12.3 入门级例题（教学用）

**题目**：丙酮在 LDA（1 eq, THF, −78°C）存在下与 CH₃I 反应，写出反应过程并说明为什么需要低温。

**预期解答路径**：
1. 丙酮有 6 个等价的 α-H（两个 CH₃ 基团），不存在区域选择性问题
2. LDA（pKa(HNiPr₂) ~36）不可逆地夺取一个 α-H → 烯醇负离子 $\text{CH}_2=\text{C(O}^-\text{Li}^+\text{)CH}_3$
3. 烯醇负离子的 α-C 作为亲核位点进攻 CH₃I（SN2）→ 丁酮
4. 为什么需要低温？——低温防止：a) 烯醇负离子进攻未反应的丙酮（自身 Aldol）；b) 动力学烯醇负离子异构化为热力学产物（丙酮无此问题，但不习惯低温会导致其他酮出错）

**老师可引导的提问**：
- 卡在"碱的强度"：问 "LDA 的共轭酸 pKa 是多少？丙酮 α-H 的 pKa 是多少？差了多少个数量级？"
- 卡在"亲核位点"：问 "烯醇负离子的负电荷在 O 上？那为什么亲核进攻发生在 C 上？"（引导到共振式：$\text{CH}_2=\text{C(O}^-)-\text{CH}_3 \leftrightarrow {}^-\text{CH}_2-\text{C(=O)-CH}_3$）

### 12.4 与现实/直觉的连接

- **类比**：烯醇负离子就像"被锁起来的弹簧"——碱（LDA）把 α-H 拔掉，弹簧（电子）就蓄能了；一旦有亲电试剂（弹簧释放的靶子），反应立刻发生。低温则是"把弹簧冻住，不让它提前弹开（自身 Aldol）"。
- **pKa 比较可视化**：让学生画一条"酸度标尺"——从强酸（HCl, pKa −7）到弱酸（水, pKa 15.7）到极弱酸（丙酮 α-H, pKa 19.3）再到烷烃（pKa ~50）。LDA（二异丙胺 pKa ~36）作为"超级碱"在这条标尺上的位置一目了然。

## 十三、竞赛拓展

1. **烯醇化学中的立体电子效应——"Bürgi-Dunitz 轨迹"与醛醇缩合的立体选择性**：亲核试剂进攻羰基时，最优轨迹是与 C=O 平面呈 ~107° 的 Bürgi-Dunitz 角——这决定了烯醇负离子进攻醛时的过渡态立体化学。Zimmerman-Traxler 过渡态模型（六元环椅式）是解释醛醇缩合产物构型的标准工具：Z-烯醇 → syn 产物，E-烯醇 → anti 产物（前提是六元环过渡态占主导）。高级竞赛题可能要求从产物的 syn/anti 比反推烯醇的 E/Z 比。

2. **Evans 手性助剂与烯醇化学的不对称控制**：Evans 噁唑烷酮手性助剂通过形成手性烯醇负离子来控制 α-位亲电进攻的面选择性——使亲电试剂只能从位阻较小的一面进攻。这是竞赛/决赛中"不对称合成"板块的核心内容。典型例子：Evans 助剂的烯醇锂盐 + BnOCH₂Cl → >95% de 的苄氧甲基化产物 → 水解回收助剂 → 得到手性 α-取代羧酸。

3. **烯醇的"被保护的等价物"——烯醇硅醚和烯醇硼酸酯在竞赛合成题中的角色**：烯醇硅醚（TMS-烯醇醚）和烯醇硼酸酯是"锁定"特定烯醇几何构型的工具——制备条件决定了 E/Z 构型，后续反应保持该构型。Mukaiyama 醛醇缩合用 TMS-烯醇醚 + Lewis 酸（如 TiCl₄/BF₃），走的是开放过渡态而非六元环椅式，因此立体选择性规律与经典醛醇缩合不同。竞赛中可能给 TMS-烯醇醚 + 醛的 Mukaiyama 条件，要求预测产物构型。
