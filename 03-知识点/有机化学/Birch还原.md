# Birch 还原（Birch Reduction）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[40-氧化还原反应]]

---

## 一、定义

**Birch 还原**：芳香族化合物在 **碱金属（Li/Na/K）/ 液氨（NH₃(l)）/ 醇（如 t-BuOH）** 条件下被还原为 **1,4-环己二烯** 衍生物的反应。

**通式**：
$$
\ce{Ar-H + 2 M + 2 R-OH ->[\text{NH3(l)}] 1,4-环己二烯 + 2 MOR}
$$

**本质**：**单电子还原（SET）**——碱金属给出电子，生成自由基负离子中间体，然后逐步质子化。

## 三、核心原理

### 3.1 反应机理（四步）

```
步骤 1：M 给出 e⁻ → 芳环形成自由基负离子（radical anion）
步骤 2：醇（ROH）质子化 → 环己二烯基自由基
步骤 3：第二个 M 给出 e⁻ → 环己二烯基负离子（carbanion）
步骤 4：醇第二次质子化 → 1,4-环己二烯产物
```

**关键中间体**：**自由基负离子**——负电荷和自由基都离域在芳环上。

### 3.2 区域选择性——EDG vs EWG

**核心规则**：

| 取代基类型 | 例子 | 产物中取代基位置 | 本质 |
|:---|:---|:---:|:---|
| **给电子基（EDG）** | -OCH₃, -CH₃ | **连在双键碳上**（非还原位）| EDG 稳定自由基负离子的双键碳位置 |
| **吸电子基（EWG）** | -COOH, -CONH₂ | **连在饱和碳上**（还原位）| EWG 稳定带负电的饱和碳位置 |

**记忆口诀**：
- **EDG → "1,4-二烯的双键上"**
- **EWG → "1,4-二烯的饱和碳上"**

### 3.3 为什么是 1,4-还原而不是 1,2-还原？

**电子因素**：
- 自由基负离子的负电荷主要定域在 **1,4-位**（对位）
- 这是因为共轭体系的对称性使 1,4-位电子密度最高
- 质子化发生在这些位置 → 1,4-加成产物

**对比**：催化氢化（H₂/Pd）是完全还原为环己烷，而 Birch 是**部分还原**为 1,4-环己二烯。

### 3.4 与 Na/NH₃(l) 还原炔烃的区别

| | **Birch 还原（芳环）** | **Na/NH₃(l) 还原炔烃** |
|:---|:---|:---|
| **底物** | 芳香环 | 炔烃 |
| **产物** | 1,4-环己二烯 | E-烯烃 |
| **机理** | SET → 自由基负离子 | SET → 乙烯基负离子 |
| **立体化学** | 不涉及（环上）| **anti** 加成 → E-烯烃 |
| **醇的作用** | 质子源 | 质子源 |

---

## 四、关键结论

1. **Birch 还原 = Na/NH₃(l)/醇 → 芳环 → 1,4-环己二烯**
2. **EDG → 产物中取代基在双键碳上**
3. **EWG → 产物中取代基在饱和碳上**
4. **机理是单电子还原（SET），不是催化氢化**
5. **液氨提供低温环境（-33°C）**——防止过度还原
6. **醇是质子源**——必须用，否则生成多聚物

---

## 五、常见分类或情形

### 5.1 按取代基分类

| 底物 | 产物 | 区域选择性 |
|:---|:---|:---:|
| **苯甲醚**（EDG） | 1-甲氧基-1,4-环己二烯 | 甲氧基在双键碳 |
| **甲苯**（弱 EDG） | 1-甲基-1,4-环己二烯 | 甲基在双键碳 |
| **苯甲酸**（EWG） | 2,5-环己二烯-1-甲酸 | 羧基在饱和碳 |
| **苯甲酰胺**（EWG） | 类似，酰胺在饱和碳 | 酰胺在饱和碳 |

### 5.2 与催化氢化的对比

| | **Birch 还原** | **催化氢化（H₂/Pd）** |
|:---|:---|:---|
| **产物** | 1,4-环己二烯（保留两个双键）| 环己烷（完全还原）|
| **选择性** | 部分还原，保留共轭二烯 | 彻底还原 |
| **取代基影响** | EDG/EWG 决定区域选择性 | 无区域选择性 |
| **应用** | 合成特定位置的双键 | 彻底饱和 |

## 六、适用条件与限制

1. **必须无水**——水会与碱金属反应
2. **需要醇作质子源**——t-BuOH 最常用（pKa 适中，不易自身被还原）
3. **低温操作**——液氨沸点 -33°C
4. **EDG 取代基可能过度还原**——如甲氧基在强条件下可能被还原
5. **不适用于含活泼氢的底物**——会与碱金属反应
6. **EWG 底物区域选择性有时不完全**——可能混有少量异构体

---

## 七、常见比较与易混点

### 7.1 Birch 与催化氢化

- Birch：部分还原，保留 1,4-二烯结构，有区域选择性
- 催化氢化：完全还原为环己烷，无区域选择性

### 7.2 Birch 与 Na/NH₃ 还原炔烃

- 两者都用 Na/NH₃(l)，但底物不同
- 芳环 Birch → 1,4-环己二烯
- 炔烃 Na/NH₃ → E-烯烃

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：苯甲醚经 Birch 还原后的产物是什么？

**解答**：
- -OCH₃ 是 EDG
- Birch 还原 → 取代基在**双键碳**上
- 产物：**1-甲氧基-1,4-环己二烯**

### 10.2 例题 2（★★ 区域选择性）

**题目**：苯甲酸经 Birch 还原后的产物是什么？

**解答**：
- -COOH 是 EWG
- Birch 还原 → 取代基在**饱和碳**上
- 产物：**2,5-环己二烯-1-甲酸**（羧基连在饱和碳）

### 10.3 例题 3（★★★ 合成设计）

**题目**：设计从甲苯合成 3-甲基-2-环己烯-1-酮的路线。

**解答**：
1. 甲苯 → **Birch 还原** → 1-甲基-1,4-环己二烯
2. 烯丙位氧化 → 引入羰基
3. 或： Birch 还原苯甲醚 → 水解 → 氧化 → 目标分子

---

## 十一、易错点

1. ❌ **EDG 在饱和碳上** → EDG 在**双键碳**上
2. ❌ **EWG 在双键碳上** → EWG 在**饱和碳**上
3. ❌ **Birch 还原得环己烷** → Birch 得**1,4-环己二烯**
4. ❌ **Birch 机理是催化氢化** → Birch 是**单电子还原（SET）**
5. ❌ **不需要质子源** → **必须加醇**作质子源
6. ❌ ** Birch 和 Na/NH₃ 还原炔烃一样** → 底物不同，产物完全不同

### 12.3 入门例题

**题目**： Birch 还原中，为什么醇（如 t-BuOH）是必须的？

**引导**：
- 机理需要两次质子化（步骤 2 和步骤 4）
- 液氨虽然也是质子源，但酸性太弱（pKa 38）
- t-BuOH（pKa ~18）提供适中的质子源
- 没有醇，中间体负离子会聚集 → 多聚或副反应

### 12.4 现实类比

**Birch 还原像"精准拆迁"**：
- 芳环 = 坚固的古城墙（芳香性）
- Na/NH₃ = 爆破队（单电子精准打击）
- 爆破后只拆两面墙（1,4-位），留下两面（双键）
- EDG = 珍贵文物放在未拆的墙上 → 双键碳
- EWG = 垃圾站放在拆掉的废墟上 → 饱和碳

1. **Birch 还原的变体**：
   - Li/NH₃/THF（替代 Na，更温和）

2. **不对称 Birch 还原**：
   - 手性质子源或手性助剂
