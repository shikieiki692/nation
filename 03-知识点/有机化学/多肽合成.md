# 多肽合成（Peptide Synthesis）

## 一、定义

**多肽合成**是指通过化学方法将氨基酸残基按特定顺序连接形成肽链的过程。肽键（酰胺键）是氨基酸之间的连接单元，多肽合成是有机合成与生物化学的交叉领域，也是药物化学（如胰岛素、GLP-1 类似物）和生物材料的核心技术。

- **肽键特点**：平面结构，C-N 键具有部分双键性质（共振稳定），不可自由旋转
- **核心挑战**：
  1. **选择性**：同一分子中同时存在氨基和羧基，需防止自身聚合
  2. **方向性**：需控制从 N→C 还是 C→N 延伸
  3. **侧链保护**：20 种氨基酸侧链官能团各异，需正交保护策略
  4. **手性**：每一步偶联都不能引起 α-碳的消旋化

## 三、核心原理

### 3.1 肽键形成的核心问题

游离氨基酸直接混合**不能**得到目标肽键：

| 问题 | 原因 | 解决方案 |
|:---|:---|:---|
| **自身聚合** | 每个氨基酸都有 —NH₂ 和 —COOH | 保护其中一个官能团 |
| **缺少选择性** | 多个氨基/羧基竞争 | 分步保护-脱保护 |
| **羧基活性不足** | —COOH 与 —NH₂ 反应慢，产率低 | 羧基活化（DCC, EDC 等）|
| **消旋化风险** | α-H 在碱性条件下可消旋 | 控制碱性和温度，选用低消旋偶联剂 |
| **侧链干扰** | Lys(ε-NH₂), Asp/Glu(β/γ-COOH) 等 | 正交保护基 |

### 3.2 羧基活化策略

直接加热氨基酸无法高效成肽，需先将羧基活化：

| 活化方法 | 活化试剂 | 活性中间体 | 特点 |
|:---|:---|:---|:---|
| **碳二亚胺法** | DCC（二环己基碳二亚胺）| O-酰基异脲 | 最经典，副产物 DCU 需过滤除去 |
| **活性酯法** | NHS / HOBt / ONp | 活性酯 | 稳定可分离，低消旋 |
| **酸酐法** | 混合酸酐 | 羧酸酐 | 快速，但可能有副反应 |
| **鎓盐法** | HATU / HBTU / TBTU | 苯并三唑鎓盐 | 偶联效率高，低消旋 |
| **酰氯法** | SOCl₂, (COCl)₂ | 酰氯 | 活性过高，易消旋 |

**DCC 机理**：
$$
\ce{R-COOH + DCC -> [R-CO-O-C(=NH)-Cy2] ->[H2N-R'] R-CO-NH-R' + DCU}
$$

### 3.3 消旋化机理与抑制

**消旋化途径**：
- 羧基活化后，α-碳的酸性增强 → 碱夺取 α-H → 形成烯醇/烯醇盐 → 质子化时可得 D-或 L-型 → 消旋

**抑制策略**：
- 使用低消旋偶联剂（HATU, PyBOP）
- 加入添加剂（HOBt, HOAt）捕获 O-酰基异脲的 O-酰基，减少烯醇化
- 缩短反应时间，降低温度
- 使用 Fmoc 策略时，哌啶脱保护时间不宜过长

## 四、关键结论

1. **肽键形成的先决条件是保护基策略**：没有完善的保护基体系，就没有可控的多肽合成
2. **Boc 策略用酸脱保护，Fmoc 策略用碱脱保护**：二者是正交的，可组合使用
3. **DCC + HOBt 是液相肽合成的经典组合**：DCC 活化，HOBt 抑制消旋和副反应
4. **固相合成（Merrifield） revolutionized 多肽化学**：过量试剂、简单过滤、易于自动化
5. **每一步偶联后都必须检验是否反应完全**：Kaiser 测试（茚三酮）检测游离氨基

## 五、常见分类

### 5.1 氨基保护基（N-保护）

| 保护基 | 缩写 | 结构 | 脱除条件 | 特点 |
|:---|:---|:---|:---|:---|
| **叔丁氧羰基** | Boc | (CH₃)₃C-O-CO- | TFA（三氟乙酸）| 酸脱，经典 |
| **苄氧羰基** | Cbz / Z | PhCH₂-O-CO- | H₂/Pd 或 HBr/AcOH | 氢解脱 |
| **9-芴甲氧羰基** | Fmoc | 芴基-CH₂-O-CO- | 哌啶（碱）| 碱脱，对酸稳定 |
| **烯丙氧羰基** | Alloc | CH₂=CHCH₂-O-CO- | Pd(0) 催化 | 正交于 Boc/Fmoc |

### 5.2 羧基保护基（C-保护）

| 保护基 | 缩写 | 脱除条件 | 特点 |
|:---|:---|:---|:---|
| **甲酯** | Me | 碱水解或皂化 | 简单 |
| **乙酯** | Et | 碱水解 | 简单 |
| **叔丁酯** | tBu | TFA | 与 Boc 同脱 |
| **苄酯** | Bn / OBn | H₂/Pd | 与 Cbz 同脱 |
| **烯丙酯** | OAll | Pd(0) | 正交 |

### 5.3 常见侧链保护基

| 氨基酸 | 侧链官能团 | 保护基 | 脱除条件 |
|:---|:---|:---|:---|
| **Asp / Glu** | β/γ-COOH | OtBu, OBn | TFA / H₂ |
| **Lys** | ε-NH₂ | Boc, Cbz | TFA / H₂ |
| **Arg** | 胍基 | Pbf, Mtr, Tos | TFA |
| **Cys** | -SH | Trt, Acm, tBu | TFA / I₂ / Hg²⁺ |
| **Ser / Thr** | -OH | tBu, Trt | TFA |
| **Tyr** | -OH | tBu, Bn | TFA / H₂ |
| **Trp** | 吲哚-NH | Boc, For | TFA / 哌啶 |
| **His** | 咪唑-N | Trt, Boc, Tos | TFA |

### 5.4 合成策略分类

```
多肽合成
├── 液相肽合成（LPPS）
│   ├── 逐步合成（stepwise）
│   └── 片段缩合（fragment condensation）
└── 固相肽合成（SPPS）
    ├── Boc 策略（Boc/Bzl）
    │   └── 酸脱保护（TFA），最终 HF 切树脂
    └── Fmoc 策略（Fmoc/tBu）
        └── 碱脱保护（哌啶），最终 TFA 切树脂
```

## 六、适用条件

### 6.1 Boc 策略（SPPS）

| 步骤 | 操作 | 试剂 | 说明 |
|:---|:---|:---|:---|
| 1 | 脱 Boc | TFA / CH₂Cl₂ | 暴露游离氨基 |
| 2 | 洗涤 | CH₂Cl₂, MeOH | 去除 TFA 和盐 |
| 3 | 中和 | DIEA（二异丙基乙胺）| 使氨基游离 |
| 4 | 偶联 | Boc-AA + DCC / HOBt | 形成肽键 |
| 5 | 洗涤 | DMF, CH₂Cl₂ | 去除过量试剂 |
| 循环 | 重复 1-5 | — | 逐个延长肽链 |
| 最终 | 切树脂 | 无水 HF（剧毒！）| 从树脂上切下肽 |

**注意**：HF 是极强腐蚀性剧毒试剂，需专用设备。这是 Boc 策略的主要缺点。

### 6.2 Fmoc 策略（SPPS，现代主流）

| 步骤 | 操作 | 试剂 | 说明 |
|:---|:---|:---|:---|
| 1 | 脱 Fmoc | 20% 哌啶 / DMF | β-消除脱保护 |
| 2 | 洗涤 | DMF | 去除哌啶和副产物 |
| 3 | 偶联 | Fmoc-AA + HATU / DIEA | 或 HBTU / HOBt |
| 4 | 洗涤 | DMF | 去除过量试剂 |
| 循环 | 重复 1-4 | — | 逐个延长肽链 |
| 最终 | 切树脂 | TFA / 清除剂（TIS, 水）| 同时脱除 tBu 等侧链保护 |

**Fmoc 优势**：
- 碱脱保护（哌啶），条件温和
- 最终切树脂用 TFA，安全易操作
- 反应进程可用 UV 监测（Fmoc 有强 UV 吸收）
- 适合自动化合成仪

### 6.3 液相 vs 固相合成比较

| 维度 | 液相合成（LPPS）| 固相合成（SPPS）|
|:---|:---|:---|
| **规模** | 可大规模（kg 级）| 通常 mg-g 级 |
| **纯化** | 每步需纯化（重结晶/层析）| 简单过滤洗涤 |
| **过量试剂** | 尽量等量（经济）| 可大过量（2-10 倍）|
| **中间体表征** | 容易 | 较难（树脂上）|
| **偶联效率** | 要求高 | 允许稍低（过滤除去）|
| **自动化** | 困难 | 易于自动化 |
| **成本** | 原料成本低，人工成本高 | 树脂和试剂较贵 |
| **典型应用** | 短肽（<5 残基）、大规模 | 长肽（5-50 残基）、筛选 |

## 七、常见比较

### 7.1 Boc vs Fmoc 保护策略

| 维度 | Boc 策略 | Fmoc 策略 |
|:---|:---|:---|
| **N-脱保护** | TFA（酸）| 哌啶（碱）|
| **C-末端/侧链** | 苄基型（Bzl）| 叔丁基型（tBu）|
| **最终切树脂** | 无水 HF | TFA |
| **安全性** | 需 HF 设备，危险 | TFA 即可，安全 |
| **侧链兼容性** | 对酸敏感侧链受限 | 对碱敏感侧链受限 |
| **消旋控制** | 较好 | 一般（哌啶长时间接触不利）|
| **现代应用** | 逐渐减少 | 绝对主流 |

### 7.2 常用偶联试剂比较

| 试剂 | 类型 | 消旋化 | 偶联速度 | 适用 |
|:---|:---|:---:|:---:|:---|
| **DCC** | 碳二亚胺 | 中等 | 中等 | 通用，经典 |
| **DCC + HOBt** | 碳二亚胺 + 添加剂 | 低 | 中等 | 推荐组合 |
| **EDC·HCl** | 水溶性碳二亚胺 | 低 | 中等 | 液相合成 |
| **HATU** | 鎓盐 | 极低 | 快 | 困难偶联、长肽 |
| **HBTU** | 鎓盐 | 低 | 快 | 通用 |
| **PyBOP** | 磷鎓盐 | 极低 | 快 | 对酸敏感体系 |

## 十、例题

### 例题 1：二肽合成路线

**题目**：设计从 L-丙氨酸和 L-苯丙氨酸合成 Ala-Phe 的液相合成路线。

**解析**：

**方案（Boc 策略）**：
1. **保护 Phe 的氨基**：Phe + Boc₂O → Boc-Phe
2. **活化 Boc-Phe 的羧基**：Boc-Phe + DCC + HOBt → Boc-Phe-OBt（活性酯）
3. **偶联**：Boc-Phe-OBt + H-Ala-OMe → Boc-Phe-Ala-OMe
4. **脱 Boc**：TFA → H-Phe-Ala-OMe
5. **脱甲酯**：皂化 → H-Phe-Ala-OH（ Ala-Phe ）

**注意**：要得到 **Ala-Phe**（Ala 在 N-端），应该：
- 保护 Ala 的氨基：Boc-Ala
- 活化 Boc-Ala，与 H-Phe-OMe 偶联
- 脱保护得 Ala-Phe

### 例题 2：固相合成循环

**题目**：简述 Fmoc 固相合成中一个氨基酸偶联循环的步骤。

**解析**：

1. **脱 Fmoc**：20% 哌啶/DMF，10 min × 2 次 → 脱除 N-端 Fmoc，暴露游离氨基
2. **洗涤**：DMF 洗涤树脂 5-6 次 → 去除哌啶
3. **Kaiser 测试**：茚三酮检验氨基是否游离（蓝色 = 阳性）
4. **偶联**：Fmoc-AA-OH + HATU + DIEA in DMF，30-60 min
5. **洗涤**：DMF 洗涤去除过量试剂
6. **Kaiser 测试**：检验偶联是否完全（黄色/无色 = 阴性，反应完全）

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| "肽键就是普通酰胺键" | 忽略其特殊性 | 肽键具有**平面性**和**部分双键性**，C-N 不可自由旋转 | 肽键与普通酰胺键有何不同？ |
| "游离氨基酸可以直接缩合" | 忽略副反应 | 必须**保护**一个端基并**活化**另一端 | 为什么不保护就无法合成多肽？ |
| "Boc 和 Fmoc 可以随意混用" | 忽略正交性 | Boc 用**酸**脱，Fmoc 用**碱**脱——设计不当会导致同时脱除 | 什么叫正交保护？为什么需要正交？ |
| "偶联试剂越多越好" | 忽略消旋化 | 过度活化会加速 α-H 烯醇化 → **消旋** | 消旋化是如何发生的？如何抑制？ |
| "固相合成不需要纯化" | 忽略副产物累积 | 每步必须**洗涤完全**，否则杂质累积 | 固相合成为什么也要控制偶联效率？ |
| "所有侧链都不需要保护" | 忽略侧链反应性 | Lys(ε-NH₂), Asp(β-COOH), Cys(-SH) 等必须保护 | 哪些氨基酸侧链在多肽合成中需要保护？ |

### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "多肽合成只是生物化学" | 忽略有机合成本质 | 强调：多肽合成是**有机合成**的经典应用——保护基、活化、选择性控制全涵盖 |
| "固相合成就是'搅一搅、洗一洗'" | 低估技术难度 | 解释：每一步的偶联效率、消旋控制、洗涤完全度都影响最终产物纯度 |
| "肽键形成不需要催化剂" | 混淆生物与化学 | 核糖体合成用酶催化；化学合成必须用 DCC/HATU 等活化试剂 |

### 12.3 入门例题

**最简判断练习**：
> 1. 合成 Ala-Gly 时，应该保护哪个氨基酸的氨基？活化哪个氨基酸的羧基？
> 2. Fmoc 策略中，为什么可以用哌啶脱保护而不影响 tBu 侧链保护基？

**答案**：
1. **保护 Ala 的氨基**（Boc-Ala 或 Fmoc-Ala），**活化 Ala 的羧基**，与游离的 H-Gly-OR 偶联
2. **Fmoc 对碱敏感**（哌啶引发 β-消除），而 **tBu 对碱稳定**（tBu 需强酸 TFA 才能脱除）——二者是正交的

**引导**：问"如果反过来用酸处理 Fmoc 保护的肽会怎样？"（Fmoc 对酸稳定，不会脱除）

### 12.4 现实类比

**"串珠子"类比多肽合成**：
- 合成一条特定序列的肽链 = **按顺序串特定颜色的珠子**
- 游离氨基酸 = 每颗珠子**两头都有孔**（氨基和羧基）
- 如果不加控制 = 珠子会**乱连**（自身聚合、无序连接）
- **保护基** = 用**塞子堵住一端**——Boc/Fmoc 堵住氨基端，酯基堵住羧基端
- **活化试剂** = **给珠子一端涂强力胶**——DCC/HATU 让羧基变得极具反应性
- **固相合成** = 把第一颗珠子**固定在绳子上**（树脂）——加珠子、洗珠子、再加珠子，不需要把半成品取下来
- **正交保护** = 用**不同类型的塞子**——有些用酸拔（Boc），有些用碱拔（Fmoc），有些用氢气拔（Cbz），不会互相干扰

## 十三、竞赛拓展

| 方向 | 内容 | 竞赛价值 |
|:---|:---|:---|
| **天然化学连接（NCL）** | 硫酯与半胱氨酸的化学选择性连接 | 无保护基合成长肽 |
