# 亲核体与亲电体

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[31-有机反应的本质]]、[[32-有机反应机理]]

## 一、定义

### 亲核体（Nucleophile, Nu）
具有**孤对电子或负电荷**、能将电子对**送给**亲电中心的物种。本质上是 **Lewis 碱**。
- 富电子（电子密度高）
- 中性或带负电
- 具有可用的孤对/π 键

### 亲电体（Electrophile, E⁺）
具有**空轨道或缺电子中心**、能**接受**电子对的物种。本质上是 **Lewis 酸**。
- 缺电子（电子密度低）
- 中性或带正电
- 中心原子上有 σ*、π* 或空 p 轨道

## 三、核心原理

### 3.1 极性反应的统一表达
$$\ce{Nu^- + E^+ -> Nu-E}$$
- 箭头从 Nu 上的孤对/π → E 上的缺电子位置
- 这是**所有**极性有机反应的本质（也包括主族无机反应）

### 3.2 亲核体的"等级"
- 强亲核体：负电荷高、不被屏蔽（如 RO⁻、CN⁻）
- 弱亲核体：电中性、有少量孤对（如 H₂O、ROH）
- 极弱亲核体：被强吸电子基团降低（如 NO₃⁻、ClO₄⁻）

### 3.3 亲电体的"等级"
- 强亲电体：H⁺、Cl⁺、NO₂⁺、SO₃、AlCl₃、BF₃
- 中等亲电体：羰基 C、卤代烷的 C
- 弱亲电体：芳环（除非已被取代）、烯烃（无强活化）

## 四、关键结论

### 4.1 与 Lewis 酸碱的对应
| 概念 | Lewis 体系 | 极性反应体系 |
|---|---|---|
| 给电子者 | Lewis 碱 | 亲核体 |
| 受电子者 | Lewis 酸 | 亲电体 |
| 反应 | 加合 | 一般包括加合 + 离解 |

### 4.2 与 HSAB 的关联
- **硬亲核**：F⁻、OH⁻、NH₃ → 偏好硬亲电（H⁺、Mg²⁺）
- **软亲核**：I⁻、S²⁻、PR₃ → 偏好软亲电（Hg²⁺、Pt²⁺）
- 在主族化学反应中至关重要——见 [[软硬酸碱理论]]

### 4.3 常见亲核体清单
| 等级 | 实例 |
|---|---|
| 极强 | $\ce{R-^{(-)}}$、$\ce{R-Li}$、$\ce{HC≡C^-}$、$\ce{H^-}$（NaH 中） |
| 强 | $\ce{RO^-}$、$\ce{HO^-}$、$\ce{CN^-}$、$\ce{R2N^-}$ |
| 中 | $\ce{ROH}$、$\ce{R2NH}$、$\ce{Cl^-}$、$\ce{Br^-}$、$\ce{I^-}$ |
| 弱 | $\ce{H2O}$、$\ce{ROCH=CR2}$（烯醇醚）、$\ce{ArOH}$ |
| 极弱 | $\ce{NO3^-}$、$\ce{ClO4^-}$、$\ce{ClF^-}$ |

### 4.4 常见亲电体清单
| 类型 | 实例 |
|---|---|
| 质子酸 | $\ce{H^+}$、$\ce{H3O^+}$、$\ce{HCl}$ |
| Lewis 酸 | $\ce{BF3}$、$\ce{AlCl3}$、$\ce{ZnCl2}$ |
| 卤代烷 | $\ce{R-Cl}$、$\ce{R-Br}$、$\ce{R-I}$ |
| 羰基化合物 | $\ce{R-CO-R}$、$\ce{R-COOR}$ |
| 不饱和键 | 共轭烯酮的 β 碳、富电子双键被缺电子加成 |
| 主族高价物种 | $\ce{SiCl4}$、$\ce{PCl3}$、$\ce{XeF2}$（Xe 上 σ*） |

## 五、常见分类或情形

### 5.1 中性亲核体 vs 阴离子亲核体
- 中性：$\ce{H2O}$、$\ce{NH3}$、$\ce{ROH}$、$\ce{R3N}$
- 阴离子：$\ce{OH^-}$、$\ce{NH2^-}$、$\ce{RO^-}$、$\ce{X^-}$
- 阴离子通常更强（电荷集中）

### 5.2 单功能 vs 双功能亲核体
- 单功能：$\ce{F^-}$
- 双功能（双齿）：$\ce{NO2^-}$（O 端为硬，N 端为软）、$\ce{SCN^-}$（S 端软，N 端硬）

### 5.3 与亲核加成 vs 取代的关系
- 亲核加成：Nu 与 E⁺ 生成 Nu-E 加合物（无离去）
- 亲核取代：Nu 取代 E⁺ 上的离去基（一同离去）

## 六、适用条件与限制

### 适用条件
- 极性反应（占大多数有机反应）
- 主族元素的极性机理
- Lewis 加合反应

### 限制
- 自由基反应不适用（用单钩箭头描述）
- 协同周环反应不适用（用 FMO 描述）
- 配位化学中的"中性"概念可能与有机/主族不同

## 七、常见比较与易混点

### 1. 亲核性 vs 碱性
- **亲核性**：动力学概念——攻击速率
- **碱性**：热力学概念——质子化平衡常数
- 两者通常正相关但不一定一致：
  - 大极化亲核体 I⁻ 的亲核性高于碱性
  - 受位阻保护的碱（如 LDA）碱性高但亲核性低

### 2. 亲电性 vs 酸性
- 类似亲核性 vs 碱性的关系
- "Lewis 亲电性"对应"Lewis 酸性"

### 3. 亲核试剂 vs 离去基
- 亲核试剂：进入分子（带电子对来）
- 离去基：离开分子（带电子对走）
- **同一物种可既是好亲核又是好离去基**（如 I⁻）

## 十、例题

### 例题 1：识别 SN2 中的亲核体
**题目：** 在 $\ce{CN^- + CH3-Br \to CH3-CN + Br-}$ 中，谁是亲核体？谁是亲电体？
**分析：**
- $\ce{CN^-}$ 是带负电荷的物种，孤对在 C 上（也可在 N 上）
- $\ce{CH3-Br}$ 中的 C 上有 σ* 反键空轨道（Br 为离去基）
**解答：**
- 亲核体：$\ce{CN^-}$（C 端）
- 亲电体：$\ce{CH3-Br}$ 的 C
- 离去基：$\ce{Br^-}$

### 例题 2：识别 Lewis 加合中的亲核/亲电
**题目：** $\ce{NH3 + BF3 \to H3N\bond{->}BF3}$
**分析：**
- $\ce{NH3}$：N 上有孤对
- $\ce{BF3}$：B 上有空 p 轨道（缺电子）
**解答：** N 是亲核体（Lewis 碱），B 是亲电体（Lewis 酸）。

## 十一、易错点
- 把"带电子对"等同于"亲核"——某些极弱亲核体（如 NO₃⁻）虽带电荷但因高度共振稳定而失去亲核性
- 忽视"软硬"匹配——硬-硬、软-软更稳定（HSAB）
- 把碱性强 = 亲核性强（不一定）
