## 一、定义

吡咯（Pyrrole）是一种含有一个氮原子的五元芳香杂环化合物，分子式为 C₄H₅N。氮原子以 sp² 杂化形式存在，其孤对电子参与构成芳香六隅体，使吡咯具有显著的芳香性。

## 三、核心原理

### 3.1 电子结构与芳香性

- **6π 芳香体系**：氮原子提供 2 个电子（孤对电子），四个碳原子各提供 1 个电子，共 6 个 π 电子，满足 Hückel 4n+2 规则（n=1）
- **富电子芳环**：氮的孤对电子向环强烈供电子，使吡咯环上电子密度远高于苯环
- **取代活性顺序**：吡咯 > 呋喃 > 噻吩 > 苯（吡咯是五元杂环中亲电取代活性最高的）

### 3.2 共振结构

吡咯的主要共振式显示电子密度集中在 2,5-位（α-位）和 3,4-位（β-位），其中 α-位电子密度更高：

```
      NH              N⁺H             N⁺H
     /  \            /  \            /  \
    /    \    ↔    /    \    ↔    /    \
   /      \        /      \        /      \
  负电荷主要在 α,β 位分布，α-位更富电子
```

### 3.3 亲电取代反应

- **反应位置**：主要在 α-位（2-位或 5-位），β-位取代较少
- **反应条件**：比苯温和得多，常在室温或低温下进行
- **典型反应**：
  - 卤化：与卤素剧烈反应，常得多卤代产物（需控制条件）
  - 硝化：使用温和硝化试剂（如乙酰硝酸酯）
  - 磺化：使用吡啶-三氧化硫络合物
  - Friedel-Crafts 酰基化：活性足够，可顺利进行

### 3.4 碱性特征

- **极弱碱性**：吡咯氮上的孤对电子参与芳香六隅体，质子化会破坏芳香性
- **pKb ≈ 13.6**，远弱于吡啶（pKb ≈ 8.8）
- **N-H 酸性**：吡咯 N-H 具有一定酸性（pKa ≈ 17），可与强碱（如 NaNH₂、Grignard 试剂）反应生成吡咯负离子

### 3.5 对酸敏感

- 强酸条件下，吡咯易发生聚合反应
- 质子化优先发生在 α-碳上（而非氮上），因为 α-碳质子化后仍可保持部分共振稳定

## 四、关键结论

1. **芳香性最强但最不稳定**：吡咯的芳香六隅体中氮贡献 2 电子，环上电子密度最高，亲电取代活性最强；但高活性也使其易发生氧化、聚合等副反应
2. **α-位优先**：亲电取代几乎专一地发生在 α-位，这是电子密度分布和中间体稳定性共同决定的
3. **氮不碱性**：吡咯氮不能作为碱接受质子（与吡啶形成鲜明对比），这是判断五元/六元氮杂环的关键区别
4. **合成入口**：
   - "[[Paal-Knorr合成]]：1,4-二羰基 + 氨/伯胺 → 吡咯"
   - "[[Knorr吡咯合成]]：α-氨基酮 + β-酮酯 → 取代吡咯"
   - "[[Hantzsch吡咯合成]]：α-卤代酮 + β-酮酯 + 氨 → 高度取代吡咯"

## 五、与呋喃、噻吩的对比

| 性质 | 吡咯 | 呋喃 | 噻吩 |
|:---|:---|:---|:---|
| 杂原子 | N-H | O | S |
| 杂原子供电子能力 | 最强 | 中等 | 较弱 |
| 亲电取代活性 | 最高 | 高 | 较高 |
| 环稳定性 | 较低（易聚合） | 中等 | 最高 |
| 芳香性感受 | 最强 | 较弱（有双烯性质） | 强 |
| 典型副反应 | 聚合、氧化 | D-A 反应（可逆） | 较稳定 |
| 碱性 | 极弱（N-H 酸性） | — | — |

## 六、典型题型

1. **芳香性判断**：吡咯的 π 电子计数（注意氮提供 2 电子）
2. **反应位置预测**：亲电取代在 α-位还是 β-位
3. **碱性排序**：吡咯、吡啶、苯胺、脂肪胺的碱性比较
4. **合成路线设计**：给定目标吡咯衍生物，选择合适的合成方法

### 7.2 常见误区

| 误区 | 正确认识 |
|:---|:---|
| "吡咯和吡啶都易亲电取代" | 吡咯**富电子**易 SEAr；吡啶**缺电子**易 SNAr |
| "吡咯氮有碱性" | 吡咯氮孤对参与芳香，**几乎无碱性**；N-H 反而有弱酸性 |
| "五元杂环都稳定" | 吡咯**不稳定**，易聚合；噻吩最稳定 |
| "亲电取代位置随机" | α-位电子密度更高，中间体正电荷可分散到氮上，**α-位优先** |
