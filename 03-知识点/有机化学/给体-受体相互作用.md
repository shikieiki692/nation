# 给体-受体相互作用
- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[22-分子轨道理论]]、[[28-有机物结构的稳定性]]、[[32-有机反应机理]]

## 一、定义

**给体-受体相互作用（Donor-Acceptor Interaction）** 是 ABOC 教材统一描述各种电子离域现象的核心范式：当一个**填充电子的轨道（给体）**与一个**未填充的较低能级轨道（受体）**在空间上有效重叠时，电子从给体离域到受体，导致整个体系能量降低。

### 核心数学描述

电子从能级 $E_\text{donor}$ 流向 $E_\text{acceptor}$（且 $E_\text{donor} < E_\text{acceptor}$）所获得的稳定化能：

$$\Delta E_\text{stab} \propto \frac{|H_{ij}|^2}{E_\text{acceptor} - E_\text{donor}}$$

- $H_{ij}$ 为两轨道的耦合矩阵元（与轨道**重叠程度**成正比）
- 分母为给体-受体能级差

> 因此：① 重叠越大，作用越强；② 能级差越小，作用越强。

### 给体-受体类型（4 大类）

| 给体（提供电子）| 受体（提供空轨道）| 相互作用类型 | 别名 |
|:---|:---|:---|:---|
| 孤对电子（n）| 空 p 轨道 | n → p | 配位键、p-π 共轭起源 |
| 孤对电子（n）| σ\* 反键轨道 | n → σ\* | **端基效应**、Bohlmann 效应 |
| 孤对电子（n）| π\* 反键轨道 | n → π\* | **共振论中的 +M 共轭** |
| σ 成键电子 | σ\* 反键轨道 | σ → σ\* | **σ-σ 超共轭** |
| σ 成键电子 | π\* 反键轨道 | σ → π\* | **σ-π 超共轭** |
| π 成键电子 | π\* 反键轨道 | π → π\* | **π-π 共轭** |
| π 成键电子 | σ\* 反键轨道 | π → σ\* | β-硅基效应等 |

> 这一统一框架**取代了无机化学中"配位键"和有机化学中"共轭/超共轭"的人为分类**——它们本质上都是同一种轨道相互作用的不同名称。

## 三、核心原理

### 3.1 几何要求：[[反式共平面]]（trans-coplanar / antiperiplanar）

> **请不要对"反式共平面"这个词进行缩写**（ABOC 原文强调 ×3）。详见 [[反式共平面]] KP。

考察 H-C-C-F 体系中 H-C σ 键给电子到 C-F σ\* 反键轨道：

- **反式共平面**（二面角 H-C-C-F = 180°）：H-C 给体波瓣与 C-F 反键的最大波瓣朝向重合 → 重叠最大 → 作用最强
- **顺式共平面**（二面角 = 0°）：必须使用 H-C 成键轨道**后方**的小波瓣去重叠 → 重叠很差 → 作用极弱
- **垂直**（二面角 = 90°）：节面与轨道波瓣不匹配 → 完全无相互作用

> **结论**：分析任何 σ-σ / σ-π / n-σ\* 相互作用前，先检查几何条件是否满足反式共平面。

### 3.2 电子要求：HOMO-LUMO 能级关系

引入"前线轨道"思维（与 [[22-分子轨道理论]] 中的 FMO 一致）：

| 类型 | 中文 | 英文 | 缩写 |
|:---:|:---|:---|:---:|
| 占据 | 最高占据分子轨道 | Highest Occupied MO | HOMO |
| 未占据 | 最低未占据分子轨道 | Lowest Unoccupied MO | LUMO |

只有 **HOMO（给体）与 LUMO（受体）能量足够接近** 时，才能产生显著的稳定化作用。能量太低的 σ 键（如 C-C）或太高的 σ\*（如 C-H σ\*）不参与有效相互作用。

### 3.3 电负性差导致的轨道波瓣不对称

**关键原理**：当成键的两个原子电负性不同时：
- **HOMO（成键轨道）**：偏向**电负性大**的原子（即电负性大的原子贡献更大的成键波瓣）
- **LUMO（反键轨道）**：偏向**电负性小**的原子（即电负性小的原子贡献更大的反键波瓣）

```
        HOMO (σ)             LUMO (σ*)

  ●●●●● ─── ●●●         ●●● ─── ●●●●●
   电负大    电负小       电负大    电负小
   (大波瓣)  (小波瓣)     (小波瓣)  (大波瓣)
```

### 3.4 推论：超共轭不是互相的

考察 H-C-C-F 体系：
- ✅ **有利方向**（H-C → C-F σ\*）：F 上的反键波瓣大；C 上的成键波瓣大；两个大波瓣相邻 → 重叠好
- ❌ **不利方向**（F-C → C-H σ\*）：F 上的成键波瓣大但被电子占据；C 上的反键波瓣很小 → 重叠差，且 σ\*(C-H) 能级太高

**结论**：**两个原子电负性差异越大，超共轭越具有方向性**——只考虑有利方向，忽略反向。这与"互相平衡"的早期错误描述完全不同。

> 注：在乙烷（C-C 两端等价）中，两侧 H-C 对 C-H σ\* 的超共轭是等价的，因此都需要考虑——这就是乙烷全交叉构象的稳定性来源（3 组 σ-σ\* 相互作用）。

## 四、关键结论

### 4.1 给体强弱顺序

> ABOC §1.2.2 给出的统一顺序（从最差给体到最佳给体）：

$$\text{F-C} \;<\; \text{O-C} \;<\; \text{N-C} \;<\; \text{C-C} \;<\; \text{H-C} \;<\; \text{S-C} \;<\; \text{P-C} \;<\; \cdots \;<\; \text{Si-C} \;<\; \text{Li-C}$$

**规律**：
- 与碳成键的原子电负性**越小**（H、Si、Li），σ(X-C) 能级**越高**，越接近 LUMO，给电子能力越强。
- 高周期元素（S、P、Si）由于轨道半径大、成键松散，作为给体也较强。
- 单独考察孤对电子时：**孤对电子是最强的给体**（电子能量比 σ 键更高）；其中电负性小的原子上的孤对电子能量更高。

#### 量化对比（vs H-C）

| 给体类型 | 相对给电子能力 | 备注 |
|:---|:---:|:---|
| Li-C 孤对电子（碳负离子）| ★★★★★ | 最强 |
| Si-C σ | ★★★★ | β-硅基效应（≈ 甲醛正离子稳定性）|
| H-C σ | ★★★ | 标准对照 |
| C-C σ | ★★ | 弱于 H-C |
| O-C σ | ★ | 弱（电负性差导致波瓣不利）|
| F-C σ | ☆ | 几乎无给电子能力 |

> **重要例外**：H-C **比** C-C 更好的超共轭给体（因 H 电负性更小）。这就是为什么 $\mathrm{Et_3C^+}$ **不如** $\mathrm{Me_3C^+}$ 稳定——后者拥有更多的 H-C 键给电子。

### 4.2 受体强弱顺序

#### A. 电负性主导（同周期）

$$\text{C-F} > \text{C-O} > \text{C-N} > \text{C-C}$$

**原因**：与 C 成键的原子电负性越大，σ\* 反键轨道能级**越低**，越靠近 HOMO；同时 C 上的反键波瓣越大，重叠越好。

#### B. 可极化性主导（同族）

$$\text{C-I} > \text{C-Br} > \text{C-Cl} > \text{C-F}$$
$$\text{C-S} > \text{C-O}, \quad \text{C-P} > \text{C-N}$$

**原因**：周期增大 → 原子半径增大 → σ\* 轨道更加**松散** → 几何上更容易接受其他轨道电子的填入。

#### C. 综合：高周期键的"双向"性

> **重要现象**：与高周期元素（S, P, Si）成键时，σ 既是好的给体又是好的受体。

例：S-C-C-F 体系中：
- S-C 作为给体（向 C-F σ\* 给电子）—— 有利
- F-C 作为给体（向 C-S σ\* 给电子）—— 也有利（C-S σ\* 能级足够低）

而 Si-C 由于 Si 电负性极小，是**极强的给体**而**普通的受体**。这就是 β-硅基效应（β-Si effect）的来源——硅基稳定相邻碳正离子的能力堪比甲醛正离子。

### 4.3 优先离域原则

> 当一个分子有**多个**潜在的给体-受体相互作用时，按照下列优先级判断哪一组主导：

**电负性小的原子上的孤对电子能量更高，需要优先被离域。**

#### 经典案例：LiAlH₄ 还原内酯 vs 内酰胺的选择性

| 底物 | 中间体的孤对电子状况 | 优先离域结果 | 产物 |
|:---|:---|:---|:---|
| 5 元环内酯 | 环外醇氧负离子（O 的孤对电子，3 对，要打架）| 离域到 C-O σ\*，**直接开环** | 二醇（开环）|
| 5 元环内酰胺 | 环内 N 上的孤对电子 + 醇氧负离子 | **N 的孤对电子优先离域**（电负性小，能量高），形成亚胺正离子，醇氧离去 | 环胺（不开环）|

> 这一选择性是优先离域原则最经典的实验证据。

#### 推论应用：胺的碱性 > 醇 > 酸

- N 的孤对电子能量最高（电负性 3.0），最容易接受质子 → 胺最碱
- O 的孤对电子能量次之（电负性 3.5）→ 醇/醚为弱碱
- 当 O 已带负电（如 RO⁻）时孤对电子能量上升，碱性增强

### 4.4 [[α效应]]：孤对电子-孤对电子相邻不利

> **核心结论**：孤对电子是**最差的受体**。当两对孤对电子相邻处于反式共平面时，该相互作用极不利。

**反键轨道上升的能量 > 成键轨道下降的能量**，因此两对孤对电子彼此填入对方"反键"导致体系总能量上升。

#### 经典实验现象

| 物种 | 现象 | 解释 |
|:---|:---|:---|
| HOOH（H₂O₂）| 亲核性远 > H₂O；构象采用**邻交叉**而非对交叉 | 两个 O 的孤对电子被迫相邻 → 沿 O-O 轴扭转规避反式共平面 |
| H₂NOH（羟胺）| 亲核性 > 同浓度 NH₃ | N 上孤对电子被相邻 O 孤对推高，反应活性增强 |
| 顺式 N₂F₂ | 比反式更稳定**且**反应性更强（看似矛盾）| 顺式中两对孤对电子无法反式共平面 → 稳定；但同时孤对电子能量被推高 → 活泼 |

### 4.5 [[β-硅基效应]]

> 当碳正离子的 **β 位**有 Si 取代基时，C-Si σ 键作为强给体填入碳正离子的空 p 轨道（受体）：

```
    C(δ+) ─── C ─── Si        ⟶       C ═══ C    +    Si(δ+)?
       p空轨道       β位                π键        Si稳定化
```

- 给体：C-Si σ（最强 σ 给体之一）
- 受体：碳正离子的空 p 轨道（最佳受体）
- 几何：β-Si 处于碳正离子 p 轨道方向 → 反式共平面成立

**结果**：β-硅基稳定的碳正离子稳定性 ≈ 甲醛正离子（亚胺正离子级别）。

**典型反应**：[[Sakurai烯丙基化]]——烯丙基硅烷与亲电试剂反应，β-Si 稳定中间体并最终离去。

### 4.6 经典 vs 给体-受体表述对照

> ABOC 用给体-受体语言**重写**了所有传统的电子效应概念。

| 传统术语 | 给体 → 受体 表述 | 备注 |
|:---|:---|:---|
| 共振论 +M（如 -OR）| n(O) → π\*(C=C) | 共振式中 O 的孤对电子推到 β 位 |
| 共振论 -M（如 -NO₂）| π(C=C) → π\*(N=O) | 双键电子被拉到硝基 |
| 超共轭（碳正离子稳定化）| σ(C-H) → p(C⁺) | β-H 给电子到正离子空轨道 |
| 烯醇负离子稳定性 | n(C⁻) → π\*(C=O) | 碳负离子推到羰基 |
| 异头效应（外）| n(O) → σ\*(C-O') | 环外 O 给到环内 C-O 反键 |
| 端基效应（酰胺）| n(Y) → σ\*(N-Z) | 见 [[端基效应]] |
| β-硅基效应 | σ(Si-C) → p(C⁺) | 见上节 4.5 |
| α 效应 | n → n（受体是孤对）| 不利组合 → 反向稳定化失效 |

## 五、常见分类或情形

### 5.1 按几何关系分类

| 几何 | 二面角 | 重叠程度 | 是否有效 |
|:---:|:---:|:---:|:---:|
| 反式共平面（antiperiplanar）| 180° | 最大 | ✅ |
| 顺式共平面（synperiplanar）| 0° | 较小（小波瓣相邻）| 较弱 |
| 邻交叉（gauche）| 60° | 中等 | 部分有效 |
| 垂直（orthogonal）| 90° | 0 | ❌ |

### 5.2 按链长分类

| 链长 | 名称 | 例 |
|:---:|:---|:---|
| 直接（0）| 配位键 / 离子相互作用 | NH₃ + H⁺ |
| 1 键间隔 | n → σ\*（端基效应）/ p-π 共轭 | 烯醇醚、糖类 |
| 2 键间隔 | π-π 共轭 / σ-σ 超共轭 | 丁二烯、乙烷 |
| 3+ 键间隔 | 远程相互作用（衰减明显）| 共轭长链 |

### 5.3 按给受体性质分类（4 大组合）

```
                   受体
              p空    σ*     π*     n
          ┌─────┬─────┬─────┬─────┐
          │     │     │     │     │
       n  │  ★  │  ●  │  ●  │  ✕  │
          │配位键│端基│共轭+M│ α效应│
          ├─────┼─────┼─────┼─────┤
   给     │     │     │     │     │
       σ  │  ●  │  ●  │  ●  │  ✕  │
   体     │ 超 │ 超 │ 超 │  /  │
          ├─────┼─────┼─────┼─────┤
          │     │     │     │     │
       π  │  ●  │  ●  │  ●  │  /  │
          │π→p │ 反向│ π-π │  /  │
          └─────┴─────┴─────┴─────┘
```

★ = 形成新化学键级别；● = 显著稳定化；✕ = 不利（α 效应）；/ = 较少见。

## 六、适用条件与限制

### 6.1 适用条件
- 给体能级 < 受体能级（否则没有稳定化驱动力）
- 几何允许：反式共平面或同向重叠
- 能量差不能过大（典型 < 10 eV）

### 6.2 限制

- **羰基亲核加成**：上文"超共轭不是互相的"结论**对羰基加成模型不再成立**——羰基 LUMO（π\*）非常低，反向作用（α-σ → π\*）也变得有利（参 ABOC 后续章节）。
- **轨道节面**：苯基正离子 / 烯基正离子的 sp 杂化空轨道与共轭体系**垂直**，无法共轭，不稳定。
- **周环反应**：当骨架变化大时（D-A、Cope 等），不能简单用"两条键之间的相互作用"描述，需用 [[FMO 理论]] / [[28-有机物结构的稳定性]] 中的协同分析。

## 七、常见比较与易混点

| 对比项 | 共振论（传统）| 给体-受体相互作用（ABOC）|
|:---|:---|:---|
| 描述对象 | 极限共振结构（"假"分子）| 真实占据的轨道电子 |
| 优势 | 直观、易画 | 解释力更强、可定量 |
| 表达"重排"驱动力 | "趋向稳定共振式" | 占据轨道→低能受体的能级降低 |
| 解释立体选择性 | 较弱 | **强**（反式共平面要求）|
| 解释 α 效应 | ✕ | ✅ |

| 对比项 | 共轭效应 | 超共轭效应 | 端基效应 |
|:---|:---|:---|:---|
| 给体 | π / n | σ | n |
| 受体 | π\* | π\* / p / σ\* | σ\* |
| 范围 | 长（多原子）| 短（1-2 键）| 极短（同一碳）|
| ABOC 视角 | 都是给体-受体相互作用的不同子类 | 同 | 同 |

## 十、例题

### 例题 1（ABOC 例 1.2.2）

**题目**：为什么两对孤对电子不希望自己处于相邻位置的反式共平面？

**分析**：
- 给体 = 孤对电子 1（占据轨道）
- 受体 = 孤对电子 2（也是占据轨道，本不应作为受体）
- 当强行将其视为给体-受体时，"反键"上升的能量大于"成键"下降的能量

**解答**：根据分子轨道理论，由于原子轨道组合成分子轨道时，反键轨道上升的能量 > 成键轨道降低的能量。当两对孤对电子均被占据，且被强迫处于反式共平面时，系统的总能量反而升高（因为占据的"反键"轨道贡献更大）。这就是 α 效应的本质：H₂O₂ 选择**邻交叉构象**而非对交叉，正是为了规避两对孤对电子的反式共平面。

**反思**：α 效应同时解释了：
1. H₂O₂、NH₂OH 的高亲核性（孤对电子被推高）
2. 顺式 N₂F₂ 比反式稳定（顺式中孤对电子无法反式共平面）

### 例题 2（ABOC 例 1.2.3 - 甲酸 Z/E 构型）

**题目**：甲酸的 Z 构型和 E 构型，谁更稳定？谁酸性更强？

```
   O                    O
   ║                    ║
H—C—O—H        vs       H—C—O—H
       (Z)                       (E)
   O 与 OH 同侧            O 与 OH 异侧
```

**分析**：
- Z 构型：O 的一对孤对电子可以离域到 C=O π\*（共振式：双键转移到 C-OH）
- E 构型：O-H 的成键电子（σ）可以填入 C=O π\*（σ-π 超共轭）

**解答**：
- **稳定性**：Z 更稳定。**孤对电子是比成键电子更好的给体**。
- **酸性**：E 构型酸性更强。E 中 O-H σ 电子被离域到 C=O，O-H 键削弱，质子易解离。

**反思**：这道题展示了"给体强弱"在不同性质上的表现：稳定性看离域能（Z 胜），酸性看 O-H 削弱程度（E 胜）。同一个给体-受体框架解释了两种不同效应。

### 例题 3（ABOC 自学练习 1.2.2-2 A）

**题目**：解释顺式 N₂F₂ 比反式 N₂F₂ 在热力学上更稳定。

**分析**：考察沿 N=N 双键方向的纽曼投影，分析 N 上孤对电子和 N-F σ 给体-受体相互作用。

**解答**：
- 顺式 N₂F₂：两个 N 的孤对电子彼此处于**邻交叉**（不反式共平面），同时两个 N-F σ 键彼此**反式共平面**——形成有利的 σ(N-F) → σ\*(N-F) 给体-受体相互作用（虽然 σ(N-F) 是较弱的给体）。
- 反式 N₂F₂：两个 N 的孤对电子反式共平面 → α 效应不利；同时 N-F 键之间反式共平面消失。

> **结论**：顺式更稳定的本质是规避了 α 效应。但顺式同时也具有更高的孤对电子能量，所以反应活性更高。

## 十一、易错点

| 错误 | 原因 | 纠正 |
|:---|:---|:---|
| 认为超共轭"相互对等"（H-C ↔ C-F 双向都考虑） | 从对称的乙烷场景类推 | **电负性差异**导致波瓣不对称：只看有利方向（H-C → C-F）|
| 把"反式共平面"缩写为"反共面" | 中文习惯简化 | **ABOC 明确反对**——保持完整术语，免与简单的"对位"混淆 |
| 顺式共平面 = 反式共平面（90° 翻转就行） | 几何想象错误 | 反式 = 二面角 180°；顺式 = 0°；二者重叠程度截然不同 |
| 认为孤对电子"主动找"σ\* | 把电子视为"愿望" | 是**整个体系**能量降低；电子离域是结果而非原因 |
| 计算给体-受体顺序时忘记 H-C > C-C | 想当然认为 σ(C-C) 更强 | H 电负性小，**H-C σ 是更强的给体**；这是 $\mathrm{Et_3C^+ < Me_3C^+}$ 稳定性顺序的来源 |
| 共轭效应 vs 超共轭效应分开学 | 教材分章不连续 | ABOC 视角：**两者同源**，仅给体类型不同（n/π vs σ）|

### 12.2 学生易踩的认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析口诀/方法 |
|---|---|---|---|
| "超共轭就是 H-C 对什么 σ\* 都有效"| 高中习惯背"超共轭"现象不深究方向 | 必须满足三条件：① 反式共平面 ② 给体能级 < 受体能级 ③ 重叠不被节面阻断 | "三看：看角度、看能级、看波瓣" |
| "孤对电子越多越稳定"| 直觉上"电子多 = 稳定" | 反键轨道上升幅度更大；当孤对电子相邻时反而不稳（α 效应）| 想想 H₂O₂ 为什么不是平面 |
| "电负性大就吸电子，电负性小就给电子" | 简单二分 | 真相要分**情景**：作给体看 σ 键能级（电负性小给电子好）；作受体看 σ\* 能级（电负性大受电子好）| "一根键的两端：给电子靠 H 那侧，吸电子靠 F 那侧" |
| "C-F 是好给体（因为 F 孤对电子多）"| 把孤对电子和 σ 键混淆 | F 的孤对电子是好给体（n→σ\*），但 F-C **σ 键**是最差给体（电子能量太低，被 F 拉得很深）| "F 的孤对 vs F-C σ 键，是两回事" |
| "Wagner-Meerwein 重排只往稳定方向走"| 把它当成单向反应 | 它是**平衡**——稳定碳正离子也能可逆地变成不稳定的，只要后续有反应拉动 | ABOC §1.3f 的"平衡思维" |

### 12.3 入门级例题

**题目**：比较 $\mathrm{Et_3C^+}$ 和 $\mathrm{Me_3C^+}$ 哪个更稳定？

**预期解答路径**（把脑内推理写出来）：

1. 第一步看到题目应该想到：**两者都是三级碳正离子**，按"叔 > 仲 > 伯"已经分不出。
2. 然后联想到本 KP 的：碳正离子 = 受体（空 p 轨道），β 位的 σ 键 = 给体。
3. 比较 β 位有什么 σ 键：
   - $\mathrm{Me_3C^+}$ 的 β 位有 **9 根 H-C 键**（每个甲基 3 根）
   - $\mathrm{Et_3C^+}$ 的 β 位是 3 根 H-C + 6 根 C-C 键（每个乙基 1 根 H-C 在 α，2 根 H-C 在 β，外加 1 根 C-C 在 β）
4. 应用 H-C > C-C 的给体顺序：$\mathrm{Me_3C^+}$ 的 H-C 给电子更多 → 更稳定。
5. 结论：$\mathrm{Me_3C^+} > \mathrm{Et_3C^+}$。

**老师可在课堂上引导的提问**：
- 学生若卡在第 1 步：问 "两者结构上唯一的区别是什么？"
- 学生若卡在第 2 步：问 "你给我画一下，β 位（隔正电荷一根键）有哪些键？"
- 学生若答错（如选 $\mathrm{Et_3C^+}$ 因"乙基大、+I 强"）：让学生写出"+I 效应"在哪根键上传递，是否就是这里的超共轭？引导他们发现"+I"和"超共轭"的真正机制。

### 12.4 与现实/直觉的连接

- **类比**：给体-受体相互作用就像**水池间的水流**——水从高（HOMO）流向低（LUMO），整体水位下降（能量降低）；如果两个池子被一堵墙隔开（节面），水流就发生不了（轨道不重叠，作用为 0）。
- **反例/佯谬**：表面上看 H₂O 应该比 H₂O₂ 稳定（少一个 O-H 键，电子排布简单），但实际上 H₂O₂ 中两对孤对电子彼此推开（α 效应），反而把 N、O 的孤对电子能量推高，从而：① H₂O₂ 选择非典型构象（邻交叉）；② H₂O₂ 亲核性远 > H₂O。这告诉我们："基本现象的稳定性"和"反应活性"是两回事，但都源于同一个给体-受体框架。

## 十三、竞赛拓展

### 13.1 给体-受体相互作用的定量化

> 利用 NBO（自然键轨道）分析或 SOPT（二阶微扰）方法，可以从 DFT 计算结果中**定量提取**每对给体-受体的稳定化能（单位：kcal/mol）。

典型数值（参考值）：
| 相互作用 | 典型稳定化能 |
|:---|:---:|
| n(N) → π\*(C=O)（酰胺）| 60-80 kcal/mol |
| n(O) → σ\*(C-O)（异头效应）| 1-3 kcal/mol |
| σ(C-H) → π\*(C=C)（α-π 超共轭）| 2-4 kcal/mol |
| σ(C-H) → σ\*(C-F)（gauche effect）| 0.5-1 kcal/mol |
| σ(C-Si) → p(C⁺)（β-硅基效应）| 30-40 kcal/mol |

### 13.2 不止有机：在配位化学中的等价物

- 过渡金属-配体的 σ 给体（如 PR₃ 给电子）/ π 受体（如 CO 反馈键）描述
- 18 电子规则的电子起源
- 详见 [[18电子规则]] 与 [[配位化学]]
