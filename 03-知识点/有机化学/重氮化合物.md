# 重氮化合物（Diazonium Salts & Diazo Compounds）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[34-有机活性中间体]]、[[39-重排反应]]、[[51-金属有机]]

## 一、定义

### 1.1 三类相关化合物的区分（极易混淆，必须明确）
| 类别 | 通式 | 结构特点 |
|---|---|---|
| **重氮盐（Diazonium Salt）** | $\ce{R-N2^+ X^-}$ | 单个 R 接 N₂⁺，X⁻ 为对阴离子（如 Cl⁻、BF₄⁻） |
| **偶氮化合物（Azo Compound）** | $\ce{R^1-N=N-R^2}$ | 两个 R 各接一个 N，N=N 双键 |
| **重氮化合物（Diazo Compound）** | $\ce{R^1R^2C=N2}$ 或 $\ce{R^1R^2C=N^+=N^-}$ | 一个 C 双键到 N₂；带正/负偶极 |

> 三类均含 N₂ 单元，但**接到 R 的方式不同**：重氮盐是单端接 R，偶氮是两端接 R，重氮化合物是 C 双键到 N₂。

### 1.2 各类的代表
- **芳基重氮盐**：$\ce{Ar-N2^+ X^-}$（如 PhN₂⁺ Cl⁻）
- **偶氮化合物**：$\ce{Ar^1-N=N-Ar^2}$（如苯胺黄）、AIBN（自由基引发剂）
- **重氮化合物**：CH₂N₂（重氮甲烷）、α-重氮酮（Wolff 重排前体）

## 三、核心原理

### 3.1 芳基重氮盐的形成
**反应**：芳胺 + 亚硝酸（HNO₂，常由 NaNO₂ + HCl 现场制备）→ 芳基重氮盐

通式：
$$\ce{Ar-NH2 ->[NaNO2/\,HCl,\,0–5℃] Ar-N2^+ Cl^-}$$

#### 详细机理（4 步）
**步骤 1**：HNO₂ + H⁺ → NO⁺（实际亲电体）
$$\ce{HNO2 + H^+ -> H2O + NO^+}$$

**步骤 2**：芳胺氮上的孤对进攻 NO⁺，形成 N-亚硝基胺
$$\ce{Ar-NH2 + NO^+ -> Ar-N(H)(NO) + H^+}$$

**步骤 3**：异构化为重氮羟基
$$\ce{Ar-NH-N=O <=> Ar-N=N-OH}$$

**步骤 4**：氧上质子化 + 脱水 → 重氮盐
$$\ce{Ar-N=N-OH + H^+ -> Ar-N=N-OH2^+ -> Ar-N≡N^+ + H2O}$$

> 关键：必须在低温（0–5 ℃）下进行，否则重氮盐分解。

### 3.2 重氮盐的反应（两大类）

#### 类型 A：放氮反应（释放 N₂，得到 Ar-Nu）
**机理**：Ar-N₂⁺ 受热脱去 N₂ 形成**芳基正离子（aryl cation）**，被亲核体捕获：
$$\ce{Ar-N2^+ ->[\Delta,\,-N2] Ar^+ ->[Nu^-] Ar-Nu}$$

| 试剂 | 产物 | 名称反应 |
|---|---|---|
| H₂O, Δ | Ar-OH（酚） | 水解 |
| KI | Ar-I | 直接碘化 |
| HBF₄, Δ | Ar-F | **Balz-Schiemann 反应** |
| CuCl/HCl | Ar-Cl | **Sandmeyer 反应** |
| CuBr/HBr | Ar-Br | **Sandmeyer 反应** |
| CuCN | Ar-CN | **Sandmeyer 反应** |
| H₃PO₂ | Ar-H | 去氨基 |

> 注：Sandmeyer 反应实际上不是简单的 SN1 路径，而是经过 **Cu(I)/Cu(II) 单电子转移**机理产生 Ar 自由基。

#### 类型 B：偶联反应（保留 N，得到偶氮化合物）
高度活化的芳烃（酚、芳胺）作为亲核体进攻 ArN₂⁺ 末端 N，形成 Ar-N=N-Ar'：
$$\ce{Ar-N2^+ + Ar'-OH(\text{或 }Ar'-NH2) -> Ar-N=N-Ar'-OH}$$

例：苯胺黄
$$\ce{p-Cl-C6H4-N2^+ + PhNH2 -> p-Cl-C6H4-N=N-C6H4-NH2}$$

### 3.3 偶氮化合物的应用：自由基引发剂（AIBN）
**AIBN（azobisisobutyronitrile, 偶氮二异丁腈）** 在加热下脱 N₂，产生两个 2-氰基丙基自由基：

![[ArrowPushinginInorganicChemistry_101-230_images/224a7f279b8eb8841afe2ef19308bbc498c122b7489a90239702b89891d97ea4.jpg]]

驱动力：脱 N₂ + 形成稳定叔碳自由基（被 -CN 共振稳定）。这两个自由基可引发自由基聚合或 H 抽取反应。

### 3.4 重氮化合物的电子结构
**重氮甲烷（CH₂N₂）的共振式**：

![[ArrowPushinginInorganicChemistry_101-230_images/64878cf4e41f5faa43d5403efe1e328b1154a22810de0e631676c37eec872f7b.jpg]]

主要共振式：
- $\ce{H2C=N^+=N^-}$（C 上无电荷，端 N⁻）
- $\ce{H2C^- - N^+\equiv N}$（C 带负电，C 是亲核体性质）
- $\ce{H2C^+=N-N^-}$（C 带正电，C 是亲电性，少数情况）

> 重氮甲烷的"碳"既有亲核性，也可被 H⁺ 质子化生成 CH₃-N₂⁺（甲基化重氮盐，强甲基化剂）。

### 3.5 重氮甲烷的羧酸甲基化
$$\ce{R-COOH + CH2N2 -> R-COO-CH3 + N2}$$

#### 机理
**步骤 1**：羧酸的 OH 质子化重氮甲烷（C 接受 H）
$$\ce{R-COOH + CH2=N^+=N^- -> R-COO^- + CH3-N^+\equiv N}$$

**步骤 2**：羧酸根 O⁻ 进攻 CH₃-N₂⁺ 的 C，N₂ 离去（SN2）
$$\ce{R-COO^- + CH3-N2^+ -> R-COO-CH3 + N2}$$

> 重氮甲烷甲基化的优势：几乎中性条件，副产物只有 N₂，不影响其他官能团。但因爆炸危险性，工业上常用 **TMS-CHN₂（更稳定的重氮甲烷类似物）** 替代。

### 3.6 Wolff 重排（α-重氮酮经卡宾的 1,2-迁移）
**反应**：α-重氮酮在加热/光照/Ag₂O 催化下脱 N₂，形成酰基卡宾，1,2-迁移得到烯酮：
$$\ce{R^1-CO-CN2-R^2 ->[\Delta,\,h\nu,\,\text{or }Ag2O,\,-N2] R^1-CR^2=C=O\,(\text{烯酮})}$$

#### Arndt-Eistert 同系化（链碳延长一个 C）
完整路径——羧酸 → 一长一碳的羧酸：

![[ArrowPushinginInorganicChemistry_101-230_images/280d8611f5bd7949285e42f93be13cfe49c7d9d702bc40037c88ac78ff675b04.jpg]]

![[ArrowPushinginInorganicChemistry_101-230_images/ccad7a1e598588438a49a46db22cab3c9e0f71388b189b79baa356c67352bf70.jpg]]

机理：
1. R-COOH → R-COCl（SOCl₂）
2. R-COCl + CH₂N₂ → R-CO-CHN₂ + HCl（α-重氮酮形成）
3. R-CO-CHN₂ + H₂O/Ag₂O/Δ → R-CH=C=O（Wolff 重排，烯酮）
4. R-CH=C=O + H₂O → R-CH₂-COOH（水合，链长一）

### 3.7 重氮化合物作 1,3-偶极体
重氮甲烷可作 1,3-偶极体与烯烃发生环加成，初生 pyrazoline，脱 N₂ 给环丙烷：

![[ArrowPushinginInorganicChemistry_101-230_images/d253c8275664b3600043bd2a28493e9d3d9f8799e131a29a83ed6fec2161a405.jpg]]

> 这是制备环丙烷的"无金属催化"绿色方法。但许多底物（如富电子烯烃）反应慢。

### 3.8 重氮化合物 + Rh 催化的环丙烷化
含吸电子基（EWG，如 CO₂Et、COR）的重氮化合物如 EDA（重氮乙酸乙酯）在 Rh₂(OAc)₄ 催化下，作为"卡宾等价体"加到烯烃 C=C 上：
$$\ce{N2=CH-CO2Et + R^1-CH=CH-R^2 ->[\text{Rh2(OAc)4}] \text{cyclopropane derivative}}$$

> 这类反应是现代不对称合成中极重要的环丙烷构建工具。

### 3.9 TMS-CHN₂ 的合成
TMS-CHN₂ 比 CH₂N₂ 更稳定、更安全，常作为甲基化/碳烯前体。其合成路径：

## 四、关键结论

### 4.1 三类化合物的反应模式总结
| 类别 | 核心反应 | 驱动力 | 应用 |
|---|---|---|---|
| **芳基重氮盐** | 放氮（→Ar-Nu）/ 偶联（→Ar-N=N-Ar'） | 脱 N₂ | Sandmeyer、Balz-Schiemann、染料合成 |
| **偶氮化合物** | 受热脱 N₂ → 自由基 | 脱 N₂ | AIBN（自由基引发） |
| **重氮化合物** | 1,3-偶极环加成、卡宾前体、甲基化 | 脱 N₂ | Wolff 重排、Arndt-Eistert、环丙烷化 |

### 4.2 N₂ 离去的"无敌驱动力"
- N≡N 键能 ≈ **945 kJ/mol**（化学中最高）
- 几乎所有"重氮"系反应都靠脱 N₂ 推动
- 这是"含 N₂ 的有机分子"几乎都不稳定的本质原因

### 4.3 安全注意
- 重氮甲烷：剧毒、爆炸性气体（黄色）
- 芳基重氮盐：低温稳定、室温分解
- α-重氮酮：相对安全（但加热易爆）
- 现代趋势：**用 TMS-CHN₂、EDA 等"安全替代品"**

## 六、适用条件与限制

### 重氮化反应（芳胺 → 芳基重氮盐）
- ✓ 一级芳胺
- ✓ 0–5 ℃低温
- ✗ 一级脂肪胺：脂肪重氮盐不稳定，立即分解

### Sandmeyer 反应
- ✓ 各种取代芳基重氮盐
- ✗ 含强氧化敏感基的底物（如 -CHO）

### Wolff 重排
- ✓ α-重氮酮
- ✓ 加热、光照或 Ag₂O 催化
- ✗ α-烷基重氮（迁移倾向不一）

## 七、常见比较与易混点

### 1. 重氮盐 vs 重氮化合物
- **重氮盐**：阳离子，单端接 R，必有对阴离子
- **重氮化合物**：中性，C=N₂，电子结构是偶极
- 二者命名相似但化学差异巨大

### 2. ArN₂⁺ 中 N₂⁺ 是离去基 vs RN=N-R 中 N₂ 是中间双键
- ArN₂⁺：脱 N₂ 形成 Ar⁺ 或 Ar•
- R-N=N-R：脱 N₂ 形成 2 R• 自由基
- 二者机理上等同但功能不同

### 3. 偶氮 vs 重氮 vs 重氮化合物的命名
- **重氮（diazo-）**：R-N₂⁺ 或 R₁R₂C=N₂
- **偶氮（azo-）**：R-N=N-R'
- **二氮（diazene）**：HN=NH（diimide）

### 4. Wolff 重排 vs Curtius 重排
- **Wolff**：α-重氮酮 → 烯酮 → 链长一的酸（**[[1,2-迁移与重排]]**）
- **Curtius**：酰基叠氮 → 异氰酸酯 → 链短一的胺（**[[Curtius重排]]**）
- 二者都涉及脱 N₂ + 1,2 迁移，但迁移方向相反

### 5. 卡宾 vs 氮宾
- **卡宾**：R₂C: （二价 C）
- **氮宾**：R-N: （一价 N）
- 重氮化合物 → 卡宾；叠氮 → 氮宾

## 十、例题

### 例题 1：芳基重氮盐的形成机理
**题目：** 写出 PhNH₂ + NaNO₂ + HCl（0 ℃）→ PhN₂⁺Cl⁻ 的完整机理（4 步）。
**分析：** 见 3.1 节
**解答：**
1. NaNO₂ + HCl → HNO₂ + NaCl
2. HNO₂ + H⁺ → NO⁺ + H₂O
3. PhNH₂ + NO⁺ → PhNH-NO → PhN=N-OH（异构）
4. PhN=N-OH + H⁺ → PhN=N-OH₂⁺ → PhN≡N⁺ + H₂O

### 例题 2：Sandmeyer 合成对溴甲苯
**题目：** 由对甲基苯胺（p-toluidine）合成对溴甲苯。
**分析：**
- 重氮化：p-CH₃-C₆H₄-NH₂ + NaNO₂/HCl → p-CH₃-C₆H₄-N₂⁺Cl⁻
- Sandmeyer：+ CuBr/HBr → p-CH₃-C₆H₄-Br + N₂
**解答：**
$$\ce{p-CH3-C6H4-NH2 ->[NaNO2,\,HCl,\,0℃] p-CH3-C6H4-N2^+ Cl^- ->[CuBr,\,HBr] p-CH3-C6H4-Br + N2 + CuCl}$$

### 例题 3：重氮甲烷甲基化
**题目：** 写出 CH₃COOH + CH₂N₂ → CH₃COOCH₃ + N₂ 的机理。
**分析：**
- 步骤 1：CH₃COOH 的 H⁺ 质子化重氮甲烷的 C
- 步骤 2：羧酸根 O⁻ 进攻 CH₃-N₂⁺ 的 CH₃ 端，N₂ 离去（SN2）
**解答：** 见 3.5 节

### 例题 4：Arndt-Eistert 同系化
**题目：** 由乙酸合成丙酸（链碳延长一个 C）。
**分析：** 利用 Wolff 重排
**解答：**
1. CH₃COOH + SOCl₂ → CH₃COCl
2. CH₃COCl + CH₂N₂ → CH₃-CO-CHN₂ + HCl
3. CH₃-CO-CHN₂ + H₂O/Ag₂O/Δ → CH₂=C=O→ CH₃-CH=C=O（烯酮）
4. CH₃-CH=C=O + H₂O → CH₃-CH₂-COOH（丙酸）

## 十一、易错点
- 把重氮盐 ArN₂⁺ 与重氮甲烷 CH₂N₂ 混淆（一个阳离子、一个中性偶极）
- Sandmeyer 反应误以为是简单 SN1（实际是单电子转移）
- 忘记重氮化必须在低温（0–5 ℃）下
- 把 Wolff 重排与 Curtius 重排混淆（前者迁移到 C，后者迁移到 N）
- 忘记 N₂ 是反应所有重氮化学的"超级离去基"
- 不知道 TMS-CHN₂ 是 CH₂N₂ 的安全替代品

## 十三、竞赛拓展
