# Prevost 反应与 Woodward 反应
- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[37-加成反应]]、[[43-亲电加成反应]]
---

## 一、定义

**Prevost 反应**（1933）：烯烃与碘 / 醋酸银（I₂ / AgOAc）在**严格无水**条件下反应，经碘鎓离子中间体和两次背面进攻，生成**反式邻二醇**（经由二醋酸酯水解）。

**Woodward 反应**（1958）：烯烃与碘 / 醋酸银在**有水**条件下反应，同样经碘鎓离子中间体，但水捕获中间体使反应不可逆，最终生成**顺式邻二醇**。

**本质**：两者共享相同的碘鎓离子中间体，但**水的有无**决定了立体化学——这是有机化学中"微量条件改变导致巨大结果差异"的典范。

## 三、核心原理

### 3.1 共同起点——碘鎓离子的形成

```
C=C  +  I₂   →   [碘鎓离子]⁺
```

- 与溴鎓离子机理相同
- I⁺ 加到双键上，形成三元环碘鎓离子
- AgOAc 的作用是移除 I⁻，促进碘鎓离子形成

### 3.2 Prevost 反应（无水 → anti）

**试剂**：I₂ / AgOAc / **严格无水溶剂**（如 CCl₄、CHCl₃）

**机理**：
```
步骤 1：烯烃 + I₂ → 碘鎓离子
步骤 2：AcO⁻ 背面进攻碘鎓离子 → 碘代醋酸酯
         （开环可逆，但若从背面进攻则锁定反式关系）
步骤 3：第二个 AcO⁻ 取代 I（SN2，背面进攻）→ 二醋酸酯
步骤 4：水解（酸或碱）→ 反式邻二醇
```

**立体化学**：
- 两次都是 **背面进攻**
- 第一次：AcO⁻ 从碘鎓离子背面进攻 → 反式开环
- 第二次：AcO⁻ 从 C-I 键背面进攻 → 再次翻转
- 净结果：**anti 加成**（两个 OH 最终位于双键两侧）

### 3.3 Woodward 反应（有水 → syn）

**试剂**：I₂ / AgOAc / **H₂O**（或湿溶剂）

**机理**：
```
步骤 1：烯烃 + I₂ → 碘鎓离子
步骤 2：H₂O 或 AcO⁻ 进攻碘鎓离子 → 碘代醇/碘代醋酸酯
步骤 3：关键差异——水使中间体不可逆
         碘鎓离子开环后，C-OH 键形成，无法回到碘鎓离子
步骤 4：第二个亲核试剂（H₂O 或 AcO⁻）取代 I → 二醇单/双酯
步骤 5：水解 → 顺式邻二醇
```

**立体化学**：
- 水捕获中间体 → 第一次开环的构型被"锁定"
- 后续步骤围绕已固定的构型进行
- 净结果：**syn 加成**（两个 OH 最终位于双键同侧）

### 3.4 为什么水能翻转立体化学？——平衡思维

**ABOC 核心解释**：

> "用平衡思维解释：水使中间体不可逆，微量即可改变选择性。"

| | **Prevost（无水）** | **Woodward（有水）** |
|:---|:---|:---|
| **中间体可逆性** | 碘鎓离子开环可逆 | C-OH 键形成后不可逆 |
| **系统行为** | 寻找最稳定状态 → 两次背面进攻 | 被水"困住"在特定构型 |
| **立体化学** | **anti** | **syn** |
| **本质** | 热力学控制 | 动力学捕获 |

---

## 四、关键结论

1. **Prevost = 无水 = 反式二醇**；**Woodward = 有水 = 顺式二醇**
2. **差异的本质：水的存在使中间体不可逆**——平衡思维的经典案例
3. **两者都经历碘鎓离子**——共享相同的起始步骤
4. **Prevost 是热力学控制**（系统寻找最稳定构型）；**Woodward 是动力学捕获**
5. **AgOAc 的作用**：移除 I⁻，促进碘鎓离子形成，同时提供 AcO⁻ 亲核试剂

## 五、常见分类或情形

### 5.1 反应条件对比

| 条件 | Prevost | Woodward |
|:---|:---|:---|
| **I₂** | ✅ | ✅ |
| **AgOAc** | ✅ | ✅ |
| **水** | ❌ 严格无水 | ✅ 有水 |
| **溶剂** | CCl₄, CHCl₃ | 含水醋酸、湿溶剂 |
| **温度** | 室温 | 室温 |
| **产物** | 反式邻二醇 | 顺式邻二醇 |

### 5.2 与其他双羟化方法对比

| 方法 | 试剂 | 立体化学 | 特点 |
|:---|:---|:---:|:---|
| **OsO₄/NMO** | 催化 OsO₄ | **syn** | 最通用，温和 |
| **稀冷 KMnO₄** | 0°C 稀溶液 | **syn** | 便宜，但易过度氧化 |
| **Woodward** | I₂/AgOAc/H₂O | **syn** | 从碘鎓离子路线得 syn |
| **Prevost** | I₂/AgOAc/无水 | **anti** | 从碘鎓离子路线得 anti |
| **环氧开环** | mCPBA → 水解 | **trans** | 间接路线 |

## 六、适用条件与限制

1. **Prevost 必须严格无水**——微量水即可使反应偏向 Woodward 路径
2. **AgOAc 昂贵**——实验室小规模可用，工业上经济性差
3. **碘代中间体可能副反应**——若捕获不及时，可能发生消除或其他副反应
4. **不适用于对酸敏感的底物**——反应产生 AcOH/HI 等酸性物质
5. **环状烯烃效果较好**——开链烯烃的立体化学分析更复杂

---

## 七、常见比较与易混点

### 7.1 Prevost vs Woodward——记忆法

| 联想 | Prevost | Woodward |
|:---|:---|:---|
| **首字母** | P = "Pair" 反式（两基团成对分开）| W = "With" 同式（两基团在一起）|
| **水** | **P**revious（之前）没有水 | **W**ater（水）在里面 |
| **结果** | **P**ass（穿过）→ 反式 | **W**rap（包裹）→ 顺式 |

### 7.2 Prevost/Woodward vs 其他 syn/anti 双羟化

- 要 **syn** → 可用 OsO₄、稀冷 KMnO₄、**Woodward**
- 要 **anti** → 可用 **Prevost**、环氧开环（酸性/碱性）
- Prevost/Woodward 的独特价值：**从同一试剂体系（I₂/AgOAc）通过调控水含量获得不同立体化学**

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：环己烯分别用 Prevost 条件和 Woodward 条件处理，产物的立体化学是什么？

**解答**：
- **Prevost（无水）**：anti 双羟化 → **反-1,2-环己二醇**
- **Woodward（有水）**：syn 双羟化 → **顺-1,2-环己二醇**

### 10.2 例题 2（★★ 机理辨析）

**题目**：解释为什么 Prevost 和 Woodward 使用相同的试剂（I₂/AgOAc），但立体化学相反。

**解答**：
1. 两者都形成碘鎓离子中间体
2. **Prevost（无水）**：AcO⁻ 进攻碘鎓离子 → 产物可逆 → 系统经历两次背面进攻达到最稳定状态 → **anti**
3. **Woodward（有水）**：H₂O 进攻碘鎓离子 → C-OH 键形成后**不可逆** → 构型被锁定 → 后续步骤围绕固定构型进行 → **syn**
4. **本质**：水的存在改变了中间体的可逆性，从而翻转了立体化学

---

## 十一、易错点

1. ❌ **Prevost = 顺式** → Prevost 无水 = **anti**
2. ❌ **Woodward = 反式** → Woodward 有水 = **syn**
3. ❌ **认为水只是溶剂** → 水是**反应物**，直接参与并改变机理
4. ❌ **忽略 AgOAc 的双重作用** → 既除 I⁻ 又提供 AcO⁻
5. ❌ **与碘鎓离子开环混淆** → Prevost/Woodward 是两次亲核取代，不是单次开环
