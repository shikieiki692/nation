# 叶立德（Ylide）

## 一、定义

**叶立德（Ylide）**是指相邻两个原子上带有**相反形式电荷**、且每个原子都满足八隅体（或稳定电子构型）的中性分子。通常表示为 X⁺—C⁻，其中 X 是带有正电荷的杂原子（P, S, N 等），相邻碳带有负电荷。

- **核心结构**：X⁺—C⁻ ↔ X=C（共振式）
- **最重要的两类叶立德**：
  1. **磷叶立德（Phosphorus Ylide）**：R₃P⁺—C⁻R'₂ ↔ R₃P=CR'₂ —— Wittig 试剂
  2. **硫叶立德（Sulfur Ylide）**：R₂S⁺—C⁻R'₂ ↔ R₂S=CR'₂ —— Corey-Chaykovsky 试剂
- **本质**：叶立德的碳负离子端是强亲核体，可与醛/酮等亲电体反应

## 三、核心原理

### 3.1 叶立德的共振结构

叶立德存在两种主要共振式：

$$\ce{R3P^+ - CHR2^- <-> R3P=CR2}$$

- **叶立德式（Ylide）**：X⁺—C⁻，碳端具有强亲核性
- **叶立因式（Ylene）**：X=C，双键特征

**磷叶立德的稳定性**：
- 磷可形成稳定的 P=O 键 → Wittig 反应驱动力强
- P 的 3d 轨道可接受电子 → 稳定相邻负电荷

### 3.2 磷叶立德的分类

| 类型 | 结构特征 | 稳定性 | Wittig 产物 |
|:---|:---|:---:|:---:|
| **非稳定化叶立德** | R = H, 烷基；无 −I/−M 基 | 不稳定（高活性）| **Z-烯烃**（动力学控制）|
| **稳定化叶立德** | R 上有 EWG（CO₂R, CN, COR）| 较稳定 | **E-烯烃**（热力学控制）|
| **半稳定化叶立德** | R 上有芳基、烯基 | 中等 | Z/E 混合物 |

**原因**：
- 非稳定化叶立德反应快、可逆性低 → 动力学控制 → 内盐中间体分解时形成较不稳定的 Z-烯烃
- 稳定化叶立德反应慢、可逆性高 → 热力学控制 → 更稳定的 E-烯烃

### 3.3 Wittig 反应的机理

**经典机理**（非稳定化叶立德）：

```
步骤 1：叶立德碳进攻羰基碳 → 形成甜菜碱（betaine）
步骤 2：甜菜碱环化 → 形成氧杂磷杂环丁烷（oxaphosphetane, OPA）
步骤 3：OPA 协同分解 → 烯烃 + Ph₃P=O
```

**注意**：
- OPA 的分解是**协同的、高度立体专一**的
- 非稳定化叶立德形成的 OPA 分解不可逆 → Z-烯烃
- 稳定化叶立德的 OPA 形成可逆 → 热力学有利的 E-烯烃

## 四、关键结论

1. **Wittig 反应是醛/酮 → 烯烃的最可靠方法**：双键位置确定，无重排
2. **非稳定化叶立德 → Z-烯烃；稳定化叶立德 → E-烯烃**：这是 Wittig 反应的核心选择性规律
3. **Schlosser 改良**：用 PhLi 处理 Wittig 产物 betaine → 可翻转选择性得 E-烯烃
4. **Horner-Wadsworth-Emmons（HWE）反应**：磷酸酯负离子替代磷叶立德 → 主要得 E-烯烃
5. **硫叶立德与磷叶立德产物不同**：硫 → 环氧化物；磷 → 烯烃

## 五、常见分类

### 5.1 叶立德的类型

```
叶立德
├── 磷叶立德（最重要）
│   ├── 非稳定化：Wittig → Z-烯烃
│   ├── 半稳定化：Wittig → Z/E 混合
│   └── 稳定化：Wittig → E-烯烃
├── 硫叶立德
│   └── Corey-Chaykovsky → 环氧化物
├── 氮叶立德
│   └── 较少用；某些环加成反应
└── 砷叶立德
    └── 类似磷叶立德，但活性不同
```

### 5.2 Wittig 反应的变体

| 变体 | 试剂 | 特点 | 主要产物 |
|:---|:---|:---|:---:|
| **经典 Wittig** | R₃P=CHR' | 通用 | 非稳定化→Z；稳定化→E |
| **Schlosser 改良** | 经典 Wittig + PhLi | 可翻转选择性 | E-烯烃（从非稳定化）|
| **Horner-Wadsworth-Emmons（HWE）** | (RO)₂P(O)CH₂R' + 碱 | 磷酸酯，水溶性副产物 | **E-烯烃** |
| **Horner-Wittig** | Ph₂P(O)CH₂R' + 碱 | 氧化膦中间体 | E-烯烃 |
| **Tebbe 试剂** | Cp₂Ti=CH₂/AlMe₃ | 非碱性条件 | 醛/酮 → 末端烯烃 |
| **Peterson 烯烃化** | β-硅基碳负离子 | 硅基替代磷 | E 或 Z（条件控制）|

### 5.3 叶立德的制备

| 叶立德类型 | 制备方法 | 试剂 |
|:---|:---|:---|
| **磷叶立德** | 鏻盐 + 强碱 | Ph₃P⁺CH₂R Br⁻ + n-BuLi/NaHMDS |
| **硫叶立德** | 锍盐 + 强碱 | Me₂S⁺CH₂ I⁻ + NaH |
| **氮叶立德** | 铵盐 + 强碱 | R₃N⁺CH₂R' X⁻ + NaNH₂ |

**鏻盐制备**：
$$\ce{Ph3P + R-CH2-X -> Ph3P^+ - CH2R X^-}$$
- 通常是 SN2 反应：Ph₃P 进攻卤代烃的碳

## 六、适用条件

### 6.1 Wittig 反应的最佳条件

| 因素 | 非稳定化叶立德 | 稳定化叶立德 |
|:---|:---|:---|
| **碱** | n-BuLi, NaHMDS, KHMDS | 较弱碱（NaOH, K₂CO₃, 胺）|
| **溶剂** | THF, Et₂O | THF, DMF |
| **温度** | −78°C → rt（低温开始）| rt → 回流 |
| **当量** | 1.0-1.5 当量 | 1.0-2.0 当量 |
| **水/氧** | 严格无水无氧 | 通常也需干燥 |

### 6.2 影响 Wittig Z/E 选择性的因素

| 因素 | Z-选择性增强 | E-选择性增强 |
|:---|:---|:---|
| **叶立德类型** | 非稳定化 | 稳定化 |
| **盐效应** | 无盐（Li⁺ 络合 OPA）| 加盐（NaI, LiBr）|
| **溶剂** | 非极性（如甲苯）| 极性（如 DMF）|
| **温度** | 低温 | 高温 |
| **改良方法** | 无 | Schlosser, HWE |

## 七、常见比较

### 7.1 Wittig vs HWE vs Peterson

| 维度 | Wittig | HWE | Peterson |
|:---|:---|:---|:---|
| **试剂** | R₃P=CHR' | (RO)₂P(O)CHR'⁻ | β-硅基碳负离子 |
| **副产物** | Ph₃P=O（难除）| (RO)₂P(O)O⁻（水溶，易除）| 硅氧烷（R₃SiOH）|
| **E/Z 选择性** | 非稳定化→Z；稳定化→E | 主要 **E** | 碱→E；酸→Z |
| **底物兼容性** | 对碱敏感基团受限 | 较温和，兼容性好 | 对酸/碱条件不同 |
| **工业应用** | 较少（副产物问题）| 较多 | 中等 |

### 7.2 磷叶立德 vs 硫叶立德

| 维度 | 磷叶立德 | 硫叶立德 |
|:---|:---|:---|
| **结构** | R₃P⁺—C⁻ | R₂S⁺—C⁻ |
| **与醛/酮反应** | **烯烃** | **环氧化物** |
| **驱动力** | P=O 键形成 | S=O 键形成 |
| **典型应用** | 构建 C=C 双键 | 构建环氧三元环 |
| **手性控制** | 可实现（手性磷）| 可实现（手性硫）|

## 十、例题

### 例题 1：Wittig 反应的 E/Z 选择性

**题目**：写出苯甲醛分别与 (a) Ph₃P=CHCH₃（非稳定化）和 (b) Ph₃P=CHCO₂Et（稳定化）反应的 Wittig 产物。

**解析**：

**(a) 非稳定化叶立德**：
- 叶立德碳上连的是 H 和 CH₃（无 EWG）
- 反应不可逆，动力学控制
- 产物：**(Z)-1-苯基丙烯**为主（Z > E）

**(b) 稳定化叶立德**：
- 叶立德碳上连的是 H 和 CO₂Et（EWG）
- 反应可逆，热力学控制
- 产物：**(E)-3-苯基丙烯酸乙酯**为主（E > Z）

### 例题 2：Wittig 在合成中的应用

**题目**：设计从环己酮合成亚甲基环己烷（=CH₂ 在环上）的路线。

**解析**：

**方案**：使用 **Tebbe 试剂** 或 **Wittig 试剂（非稳定化）**

**Wittig 法**：
1. 制备鏻盐：Ph₃P + CH₃Br → Ph₃P⁺CH₃ Br⁻
2. 制备叶立德：Ph₃P⁺CH₃ Br⁻ + n-BuLi → Ph₃P=CH₂
3. Wittig 反应：环己酮 + Ph₃P=CH₂ → 亚甲基环己烷 + Ph₃P=O

**特点**：双键位置精确，无重排风险。

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| "所有 Wittig 反应都得 Z-烯烃" | 忽略叶立德类型 | **非稳定化→Z；稳定化→E** | 如何判断 Wittig 产物以 Z 还是 E 为主？ |
| "Wittig 和 HWE 产物相同" | 混淆选择性 | HWE 主要得 **E**，与 Wittig（非稳定化得 Z）不同 | 需要合成 E-烯烃，选 Wittig 还是 HWE？ |
| "硫叶立德也得烯烃" | 与磷叶立德混淆 | 硫叶立德 → **环氧化物**；磷叶立德 → **烯烃** | 硫叶立德和磷叶立德与醛反应产物有何不同？ |
| "Wittig 反应有重排" | 与碳正离子反应混淆 | Wittig **无重排**，双键位置确定 | Wittig 反应和 Grignard 加成-消除构建烯烃相比有何优势？ |
| "叶立德带正负电荷不稳定" | 忽略共振稳定 | 叶立德通过 X=C 共振和 P/S 的空 d 轨道获得稳定化 | 为什么磷叶立德比相应的碳负离子稳定？ |

### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "Wittig 反应太复杂，不如消去法制烯烃" | 不了解 Wittig 的优势 | 强调 Wittig 的**位置专一性**和**无重排**——消去可能得多重产物 |
| "叶立德就是碳负离子" | 简化理解 | 强调叶立德的**双性特征**：既有碳负离子的亲核性，又有叶立因的稳定性 |

### 12.3 入门例题

**最简判断练习**：
> 判断下列叶立德与苯甲醛反应的主要产物构型（Z/E）：
> 1. Ph₃P=CHCH₃
> 2. Ph₃P=CHCOPh

**答案**：
1. **Z-烯烃**为主（非稳定化叶立德）
2. **E-烯烃**为主（稳定化叶立德，COPh 是 EWG）
**引导**：问"为什么 EWG 会改变选择性？"（涉及 OPA 的可逆性和热力学控制）
