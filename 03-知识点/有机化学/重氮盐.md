# 重氮盐（Diazonium Salts）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]

## 一、定义

**重氮盐（Diazonium Salt）**：芳胺（ArNH₂）在 **NaNO₂/HCl（或 H₂SO₄）** 作用下，于 **0-5°C** 生成的 **ArN₂⁺X⁻** 盐。

**通式**：
$$
\ce{Ar-NH2 + NaNO2 + 2 HX ->[0-5^\circ C] Ar-N2^+X^- + NaX + 2 H2O}
$$

**本质**：是**芳香取代的"万能中间体"**——ArN₂⁺ 可以转化为 ArCl、ArBr、ArI、ArCN、ArOH、ArH、ArF 等几乎所有芳香衍生物。

## 三、核心原理

### 3.1 制备机理

**亚硝酸（HNO₂）原位生成**：
```
NaNO₂ + HCl → HNO₂ + NaCl
HNO₂ + HCl → NOCl + H2O
ArNH₂ + NOCl → ArN=N-OH → ArN₂⁺Cl⁻
```

**低温必要性**：
- 重氮盐在 **0-5°C** 稳定
- **室温分解**（放出 N₂，生成苯正离子或自由基）
- 因此制备后需**立即使用**

### 3.2 Sandmeyer 反应

**Cu(I) 盐催化的取代**：

| 试剂 | 产物 | 备注 |
|:---|:---|:---|
| **CuCl** | ArCl | 经典 |
| **CuBr** | ArBr | 经典 |
| **CuCN** | ArCN | 引入氰基 |
| **Cu₂O/Cu²⁺** | ArOH（水解）| 替代直接加热 |

**机理**：自由基机理（非纯 SNAr）

### 3.3 Schiemann 反应

**制备氟代芳烃**：
$$
\ce{ArN2^+Cl^- + HBF4 -> ArN2^+BF4^- ->[\Delta] ArF + N2 + BF3}
$$

- 氟硼酸重氮盐热分解 → ArF
- 是制备 **芳香氟化物** 的主要方法

### 3.4 碘代反应

**直接加 KI**：
$$
\ce{ArN2^+ + KI -> ArI + N2 + K^+}
$$

- **不需要 Cu 催化**
- I⁻ 还原性强，可直接与重氮盐反应

### 3.5 偶联反应

**与活泼芳环偶联**（偶氮染料）：
$$
\ce{ArN2^+ + Ar'OH -> Ar-N=N-Ar'-OH}
$$

- 重氮正离子作为**亲电试剂**
- 进攻苯酚/苯胺的**对位**（或邻位）
- 产物是**偶氮化合物**（azo compound）

### 3.6 还原为氢（去氨基）

$$
\ce{ArN2^+ + H3PO2 -> ArH + N2 + H3PO3}
$$

- 次磷酸（H₃PO₂）还原
- 用于**去除芳环上的氨基**

---

## 四、关键结论

1. **重氮盐是芳香取代的万能中间体**
2. **制备必须低温（0-5°C）**——室温分解
3. **Sandmeyer = Cu(I) 催化 → ArCl/ArBr/ArCN**
4. **Schiemann = HBF₄ → 热分解 → ArF**
5. **碘代不需要 Cu**——直接加 KI
6. **偶联反应 = 亲电芳香取代**——重氮正离子进攻苯酚/苯胺
7. **H₃PO₂ 还原 → 去氨基**

---

## 五、常见分类或情形

### 5.1 重氮盐转化一览

| 目标产物 | 试剂 | 反应名 |
|:---|:---|:---|
| **ArCl** | CuCl | Sandmeyer |
| **ArBr** | CuBr | Sandmeyer |
| **ArI** | KI | 直接取代 |
| **ArF** | HBF₄, Δ | Schiemann |
| **ArCN** | CuCN | Sandmeyer |
| **ArOH** | H₂O, Δ 或 Cu₂O | 水解 |
| **ArH** | H₃PO₂ | 还原去氨基 |
| **Ar-N=N-Ar'** | Ar'OH / Ar'NH₂ | 偶联反应 |

### 5.2 脂肪族重氮盐

- 脂肪族重氮盐**极不稳定**（立即分解为碳正离子）
- 竞赛中主要关注**芳香族重氮盐**

---

## 六、适用条件与限制

1. **必须低温制备**（0-5°C）
2. **需要立即使用**——不能长期储存
3. **Sandmeyer 需要 Cu(I)**——Cu(II) 需先还原为 Cu(I)
4. **偶联反应需要活泼芳环**（苯酚、苯胺等）
5. **强吸电子基（-NO₂）存在时**，重氮盐更稳定

---

## 七、常见比较与易混点

### 7.1 Sandmeyer vs Schiemann

- Sandmeyer：Cu 催化，得 Cl/Br/CN
- Schiemann：HBF₄，得 F
- 两者都是重氮盐的取代反应

### 7.2 重氮盐水解 vs 直接苯酚制备

- 重氮盐水解：ArN₂⁺ + H₂O → ArOH + N₂
- 条件温和，但需先制备重氮盐
- 适合从芳胺路线制备苯酚

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：苯胺 → 氯苯的最佳路线是什么？

**解答**：
1. 苯胺 + NaNO₂/HCl（0-5°C）→ 氯化重氮苯
2. + CuCl → **氯苯**（Sandmeyer 反应）

### 10.2 例题 2（★★ 氟代）

**题目**：如何从苯胺制备氟苯？

**解答**：
1. 苯胺 + NaNO₂/HCl → 氯化重氮苯
2. + HBF₄ → 氟硼酸重氮盐
3. 加热 → **氟苯**（Schiemann 反应）

### 10.3 例题 3（★★★ 合成设计）

**题目**：设计从苯胺制备间溴苯酚的路线。

**解答**：
**更佳路线**：
1. 苯胺 + Ac₂O → 乙酰苯胺（保护氨基）
2. 硝化 → 对硝基乙酰苯胺
3. 水解 → 对硝基苯胺
4. 重氮化 → 重氮盐
5. CuBr → 对溴硝基苯
6. Sn/HCl → 对溴苯胺
7. 重氮化 → 重氮盐
8. H₂O, Δ → **对溴苯酚**

（若需间位产物，需利用硝基定位后还原）

## 十一、易错点

1. ❌ **重氮盐可以室温储存** → 必须 **0-5°C**，立即使用
2. ❌ **碘代也需要 Cu 催化** → 碘代 **不需要 Cu**，直接加 KI
3. ❌ **Schiemann 用 CuF** → Schiemann 用 **HBF₄**
4. ❌ **偶联反应是亲核取代** → 偶联是**亲电取代**（重氮正离子进攻苯酚）
5. ❌ **去氨基用 Zn/HCl** → 去氨基用 **H₃PO₂**
6. ❌ **脂肪胺也能制备稳定重氮盐** → 脂肪族重氮盐**极不稳定**

### 12.3 入门例题

**题目**：为什么重氮盐被称为"芳香取代的万能中间体"？

**引导**：
- 一张表列出所有转化：Cl/Br/I/F/CN/OH/H/偶联
- 一个重氮盐可以变成几乎所有芳香衍生物
- 这是其他中间体（如卤代苯）无法做到的

### 12.4 现实类比

**重氮盐像"变形金刚"**：
- 芳胺 = 普通人
- 重氮盐 = 穿上变形金刚战衣（N₂⁺ 是一个很好的离去基）
- 根据不同指令（CuCl/KI/HBF₄/H₂O/H₃PO₂），变身成不同形态（Cl/I/F/OH/H）
- 但战衣不稳定，必须低温立即使用
