# 直接加成与共轭加成（1,2- vs 1,4-Addition）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[37-加成反应]]、[[45-亲核加成反应]]

---

## 一、定义

在共轭体系（如 1,3-丁二烯、α,β-不饱和羰基化合物）的加成反应中，亲电/亲核试剂有两种加成模式：

| 加成模式 | 名称 | 加成位点 | 结果 |
|:---|:---|:---|:---|
| **1,2-加成** | 直接加成 | 试剂的两部分加在**相邻**的两个原子上 | 保留一个双键 |
| **1,4-加成** | 共轭加成 | 试剂的两部分加在**共轭体系两端**（1-位和 4-位）| 中间形成新双键 |

**经典例子（1,3-丁二烯 + Br₂）**：
```
1,2-加成：Br-CH₂-CHBr-CH=CH₂
1,4-加成：Br-CH₂-CH=CH-CH₂Br
```

## 三、核心原理

### 3.1 1,3-丁二烯的亲电加成：动力学 vs 热力学控制

> ⭐ **ABOC §2.10.1 行 1926-1937 核心内容**：

**机理**：
1. 亲电试剂（如 H⁺）进攻 → 生成**烯丙基碳正离子**中间体（离域）
2. 共振式：正电荷在 C2 和 C4 上离域
3. C2 上取代更多 → 该共振式权重更高 → C2 正电荷密度更高

**势能图关键**：
- **1,2-加成路径**：活化能低（C2 正电荷密度高）→ **动力学产物**
- **1,4-加成路径**：产物更稳定（双键更多取代）→ **热力学产物**

**温度依赖**：
| 条件 | 主产物 | 控制类型 | 原因 |
|:---|:---|:---:|:---|
| **低温** | 1,2-加成 | **动力学控制** | 活化能低，反应不可逆 |
| **高温** | 1,4-加成 | **热力学控制** | 第二步可逆，最终达热力学平衡 |

> ⭐ **ABOC 行 1937**："这就是平衡思维的体现。"

### 3.2 α,β-不饱和羰基的亲核加成：电荷 vs 轨道

| | **1,2-加成（直接加成）** | **1,4-加成（共轭加成）** |
|:---|:---|:---|
| 进攻位点 | 羰基碳（C=O）| β-碳（Cβ）|
| 中间体 | 四面体烷氧负离子 | 烯醇盐 |
| 质子化后 | 烯丙醇（羰基被破坏）| 羰基恢复（饱和羰基）|
| 本质 | **电荷控制**（最缺电子碳）| **轨道控制**（软亲核的 LUMO 匹配）|
| 热力学 | 通常动力学产物 | 通常热力学产物（羰基保留）|

**为什么 1,4-加成保留羰基更稳定？**
- C=O 键能 ~178 kcal/mol，C=C 键能 ~146 kcal/mol
- 1,2-加成破坏强 C=O → 生成弱 C=C + C-O
- 1,4-加成保留 C=O → 只转化 C=C 为 C-C

### 3.3 动力学控制 vs 热力学控制

| | **动力学控制** | **热力学控制** |
|:---|:---|:---|
| 决定因素 | **活化能**（ΔG‡）| **产物稳定性**（ΔG）|
| 温度 | 低温 | 高温 |
| 可逆性 | 不可逆或很少可逆 | 可逆，达平衡 |
| 产物特征 | 生成最快的产物 | 最稳定的产物 |
| 本反应中 | 1,2-加成 | 1,4-加成 |

**判断实验**：
- 低温反应 → 升温 → 若产物比例变化 → 存在热力学控制路径
- 同位素标记追踪 → 若底物与产物间有交换 → 可逆步骤存在

---

## 四、关键结论

1. **1,2-加成 = 动力学，1,4-加成 = 热力学**（经典规律，对共轭二烯和 α,β-不饱和羰基均适用）
2. **低温 → 1,2；高温 → 1,4**——温度是切换控制类型的开关
3. **硬亲核 → 1,2；软亲核 → 1,4**——见 [[硬软亲核试剂]]
4. **1,4-加成保留羰基 → 热力学更稳定**（C=O 键能 > C=C）
5. **共轭加成的核心中间体：烯醇盐**——质子化后羰基恢复

## 五、常见分类或情形

### 5.1 按底物分类

| 底物类型 | 1,2-加成产物 | 1,4-加成产物 | 备注 |
|:---|:---|:---|:---|
| 1,3-丁二烯 + HX | 3-卤-1-丁烯 | 1-卤-2-丁烯 | 热力学产物更稳定（更取代双键）|
| α,β-不饱和醛 + RMgX | 烯丙醇 | — | 醛几乎只 1,2 |
| α,β-不饱和酮 + R₂CuLi | — | 饱和酮 | 铜锂试剂专做 1,4 |
| 醌 + H₂ | 氢醌（1,2+1,2）| — | 共轭体系大，倾向全还原 |

### 5.2 Michael 加成（1,4-共轭加成的经典）

**定义**：碳负离子（烯醇盐、氰化物、硝基烷负离子等）对 α,β-不饱和羰基的 1,4-加成。

**通式**：
```
   Nu⁻ + CH₂=CH-CO-R → Nu-CH₂-CH⁻-CO-R → Nu-CH₂-CH₂-CO-R
                            (烯醇盐)         (质子化)
```

**特征**：
- 亲核试剂必须是**软亲核试剂**或**稳定碳负离子**
- 碱催化（生成亲核的碳负离子）
- 是 C-C 键形成的重要方法

## 六、适用条件与限制

1. **非共轭体系不发生 1,4-加成**——必须有共轭 π 系统
2. **醛几乎总是 1,2**——羰基碳正电性太强
3. **大位阻羰基可能被迫 1,4**——即使硬亲核试剂也进攻 β-碳
4. **动力学产物在低温"锁定"**——若升温不够，可能观察不到热力学产物
5. **某些 1,2-产物不能互变为 1,4**——若质子化不可逆，则动力学产物固定

## 七、常见比较与易混点

### 7.1 动力学 vs 热力学控制（全局比较）

| 反应类型 | 动力学产物 | 热力学产物 | 切换条件 |
|:---|:---|:---|:---|
| 1,3-丁二烯 + HBr | 1,2-加成 | 1,4-加成 | 温度 |
| 共轭二烯 + Br₂ | 1,2-加成 | 1,4-加成 | 温度/溶剂 |
| α,β-不饱和酮 + RMgX | 1,2-加成（醇）| 1,4-加成（酮）| Cu 盐/温度 |
| 烯醇化 | 动力学烯醇盐 | 热力学烯醇盐 | 温度/碱 |
| 萘的硝化 | α-硝基萘 | β-硝基萘 | 温度 |

### 7.2 "1,2"和"1,4"的编号陷阱

- 1,3-丁二烯编号：C1=C2-C3=C4
  - 1,2-加成：试剂加在 C1 和 C2
  - 1,4-加成：试剂加在 C1 和 C4
- α,β-不饱和羰基编号：O=C1-C2=C3
  - 1,2-加成：试剂加在 C1（羰基碳）和 O
  - 1,4-加成：试剂加在 C3（β-碳）和 O（经烯醇互变）

**注意**：对 α,β-不饱和羰基，"1,2"和"1,4"的编号**不从氧开始**，而是从羰基碳开始！

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：1,3-丁二烯与 HBr 在 -80°C 和 40°C 下反应，主产物分别是什么？

**解答**：
- **-80°C（低温）**：动力学控制 → **3-溴-1-丁烯**（1,2-加成）
- **40°C（高温）**：热力学控制 → **1-溴-2-丁烯**（1,4-加成，双键更取代、更稳定）

### 10.2 例题 2（★★ 势能图分析）

**题目**：画出 1,3-丁二烯 + HBr 的势能图，标出 1,2-路径和 1,4-路径的过渡态与产物相对能量。

**解答**：
```
能量
  ↑
  │     TS(1,2)    TS(1,4)
  │      ↘         ↗
  │       ↘   烯丙基正离子   ← 共同中间体
  │        ↘     ↑     ↗
  │         ↘    │    ↗
  │   反应物 →   │   ← 1,2-产物（较高）
  │              └──→ 1,4-产物（较低）
  └────────────────────────→ 反应坐标
```

TS(1,2) < TS(1,4)（动力学优势）
1,4-产物 < 1,2-产物（热力学优势）

### 10.3 例题 3（★★★ 综合判断）

**题目**：以下反应的主产物是什么？

a) 环己烯酮 + MeMgBr（无 Cu 盐，0°C）
b) 环己烯酮 + Me₂CuLi（-78°C）
c) 1,3-丁二烯 + Br₂（CCl₄, 低温）

**解答**：
- a) 硬亲核 + 低温 → **1,2-加成**（1-甲基环己-2-烯醇）
- b) 软亲核 + 低温 → **1,4-加成**（3-甲基环己酮，经烯醇互变）
- c) 低温 + Br₂ → **1,2-加成**（3,4-二溴-1-丁烯）为主；高温则 1,4-产物增加

### 10.4 例题 4（★★★ 动力学锁定）

**题目**：某 α,β-不饱和酮与 RLi 在 -78°C 反应得到 1,2-加成产物。升温至 50°C 并保持 2 小时后，1,4-加成产物比例未增加。解释原因。

**解答**：
- RLi 的 1,2-加成**不可逆**（烷氧负离子极稳定，不易消除）
- 若 1,2→1,4 的互变需要断裂 C-C 键，活化能极高
- **没有可逆步骤 → 不存在热力学控制路径**
- 这与 1,3-丁二烯 + HBr 不同（后者的第二步是可逆的质子化/消除）

## 十一、易错点

1. ❌ **认为 1,4-加成破坏共轭** → 1,4-加成后中间仍有双键，共轭被破坏但新双键形成
2. ❌ **动力学产物 = 1,4 对共轭二烯** → 低温下 1,2 是动力学产物
3. ❌ **温度是唯一控制因素** → 亲核试剂软硬、Cu 盐同样关键
4. ❌ **所有 1,2-产物都能转为 1,4** → 若不可逆则动力学产物锁定
5. ❌ **1,4-加成的编号从氧开始** → α,β-不饱和羰基编号从羰基碳开始
6. ❌ **忽略烯醇互变** → 1,4-加成的第一步产物是烯醇盐，需质子化才得最终产物
7. ❌ **混淆 Michael 加成与 1,2-加成** → Michael 专指 1,4-共轭加成
8. ❌ **认为热力学产物总是 1,4** → 对醛而言，1,2 同时是动力学和热力学产物

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析口诀 |

| "1,4 总是热力学产物" | 教材强调规律 | 醛的 1,2 也是热力学产物 | "醛是堡垒，1,2 统治" |

### 12.3 入门级例题

**题目**：1,3-丁二烯与 HBr 在 0°C 和 80°C 下反应，主产物有什么不同？用势能图解释。

**预期解答路径**：
1. 0°C：低温不可逆 → 动力学控制 → 1,2-加成（3-溴-1-丁烯）
2. 80°C：高温可逆 → 热力学控制 → 1,4-加成（1-溴-2-丁烯）
3. 1,4-产物双键更取代 → 更稳定
4. 画势能图：TS(1,2) 较低，但 1,4-产物能量更低

**老师可引导的提问**：
- 卡在第 2 步：问 "高温下哪一步变得可逆？"
- 卡在第 3 步：问 "哪个产物双键取代度更高？"

### 12.4 与现实/直觉的连接

- **类比**：动力学控制就像"先到先得"——谁跑得快谁先到终点（1,2-产物）。热力学控制就像"优胜劣汰"——大家先都到终点，然后弱的被强的淘汰，最后只留下最强者（1,4-产物）。温度就是"裁判"：低温时不允许反悔（先到为准），高温时允许反悔（强者为王）。
- **反例**：看起来 1,4-加成"更远"应该更难发生，但实际上 β-碳的 LUMO 系数足够大，软亲核试剂可以有效重叠。有机化学中"距离远"不等于"作用弱"——轨道重叠的质量比距离更重要。

## 十三、竞赛拓展

1. **串联反应中的动力学捕获**：
   - 在串联反应中，1,2-加成产物若含活泼基团，可被原位捕获 → 避免互变
   - 例：RLi 对 α,β-不饱和醛的 1,2-加成 → 立即 TMS 保护 → 锁定烯丙醇

2. **不对称共轭加成**：
   - 手性铜催化剂 + R₂Zn 对环烯酮的 1,4-加成 → 高 ee 值
   - 2001 年 Noyori/ Knowles 相关工作获诺贝尔奖背景

3. **vinylogous 反应（插烯规则）**：
   - 共轭体系延长时，亲核位点可进一步延伸
   - α,β,γ,δ-不饱和羰基可能发生 1,6-加成
