# 亲核取代

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[36-取代反应]], [[44-脂肪族亲核取代反应]]
- 关联笔记：[[脂肪族亲核取代反应]], [[芳香亲核取代]], [[专题-SN1与SN2比较]], [[碳正离子]]

## 一、定义

亲核取代反应（Nucleophilic Substitution, SN）是指亲核试剂（Nu:⁻）进攻带有离去基团（L）的底物，取代离去基团形成新键的反应。它是竞赛有机化学中最基础、最核心的反应类型之一，按机理分为 **SN1**（单分子亲核取代）和 **SN2**（双分子亲核取代）两大类。

## 二、考纲对应

- 对应考纲条目：[[36-取代反应]], [[44-脂肪族亲核取代反应]]
- 所属模块：[[基础要求-有机化学]]
- 本知识点在考纲中的作用：亲核取代是**有机化学竞赛中最核心的反应机理模块之一**。初赛阶段重点考查 SN1 与 SN2 的基本判断（底物结构 + 亲核试剂 + 溶剂 + 离去基团四维分析），决赛阶段则常与立体化学、邻基参与、SN/E 竞争、动力学 vs 热力学控制等内容联合命题。国初和国决中，SN 机理题几乎是**每年必考**，且常以"判断机理类型 → 预测立体化学 → 解释产物分布"的递进式设问出现。
- 考纲层级说明：

  - **决赛拓展**：邻基参与（NGP）、SNi 机理、非经典碳正离子、两可亲核试剂的区域选择性（Kornblum 规则）、相转移催化、动力学同位素效应在机理判定中的应用

## 三、核心原理

### 2.1 SN2 机理

**特征**：一步协同反应，亲核试剂从离去基团的背面进攻碳原子。

**速率方程**：$v = k[\text{RX}][\text{Nu}^-]$（二级反应）

**立体化学**：**构型翻转**（Walden inversion），手性中心构型完全翻转

**影响因素**：
- **底物结构**：甲基 > 一级 > 二级；三级碳不发生 SN2（空间位阻太大）
- **亲核试剂**：电荷密度高、可极化性强的亲核试剂活性高（如 RS⁻ > RO⁻、I⁻ > Br⁻ > Cl⁻）
- **离去基团**：pKa 越低的共轭酸，离去能力越强（TsO⁻、I⁻ > Br⁻ > Cl⁻ > F⁻）
- **溶剂**：极性非质子溶剂（DMSO、DMF、丙酮）有利于 SN2

### 2.2 SN1 机理

**特征**：分步反应，先形成碳正离子中间体，再被亲核试剂捕获。

**速率方程**：$v = k[\text{RX}]$（一级反应）

**立体化学**：**外消旋化**，碳正离子为平面 sp² 杂化，亲核试剂从两侧进攻概率相近

**影响因素**：
- **底物结构**：三级 > 二级 > 一级 > 甲基（碳正离子稳定性决定）
- **亲核试剂**：浓度和种类不影响速率（但影响产物分布）
- **离去基团**：同 SN2
- **溶剂**：极性质子溶剂（水、醇）有利于 SN1（稳定碳正离子和离去基团）

### 2.3 SN1 与 SN2 的竞争

| 因素 | 有利于 SN2 | 有利于 SN1 |
|------|-----------|-----------|
| 底物 | 甲基、一级、二级 | 三级、烯丙基、苄基 |
| 亲核试剂 | 强亲核试剂（高浓度） | 弱亲核试剂（低浓度） |
| 溶剂 | 极性非质子 | 极性质子 |
| 离去基团 | 好离去基团 | 好离去基团（更重要） |

### 2.4 特殊亲核取代

- **邻基参与（NGP）**：分子内亲核基团协助离去，导致构型保留或重排产物
- **芳香亲核取代（SNAr）**：需吸电子基团活化 + 加成-消除机理
- **苯炔机理**：无强吸电子基团时的消除-加成路径

### 2.5 SN/E 竞争判断入口（与 [[消除反应]] 联动）

> 亲核取代与消除反应是**竞争关系**。快速判断流程：

```
第一步：看底物级别
├─ 一级卤代烃 → SN2 为主（强碱时 E2 竞争）
├─ 二级卤代烃 → E2 略优（SN2/SN1/E1 都可能）
└─ 三级卤代烃 → E1 为主（SN1 竞争，高温 E1 优势）

第二步：看试剂
├─ 强亲核弱碱（如 I⁻, RS⁻）→ SN2
├─ 强亲核强碱（如 OH⁻, RO⁻）→ SN2 + E2
├─ 大位阻强碱（如 t-BuOK, LDA）→ E2（霍夫曼产物）
└─ 弱亲核弱碱（如 H₂O, ROH）→ SN1 + E1

第三步：看溶剂与温度
├─ 极性非质子（DMSO, DMF）→ 促进 SN2
├─ 极性质子（H₂O, EtOH）→ 促进 SN1/E1
├─ 低温 → 取代有利
└─ 高温 → 消除有利
```

**口诀**："一级取代多，三级消除强，二级两边倒，溶剂温度帮"

> 详见 [[消除反应]] §3.5 完整的 SN/E 竞争三步判断法与速查表。

## 四、关键结论

1. **SN2 和 SN1 不是绝对二元**——离子对效应使 SN1 表观保留 SN2 的反向偏好（翻转 > 保持）
2. **碳正离子稳定性是 SN1 的核心判据**——不是离去基好坏（ABOC 反复强调）
3. **极性非质子溶剂是 SN2 的加速器**——阳离子溶剂化 + 阴离子"裸露"
4. **邻基参与和反极化是 SN2 的"例外面孔"**——产物构型可能保持而非翻转
5. **SN/E 竞争看四个维度**：底物级别、试剂亲核/碱性、溶剂极性、温度
6. **桥头碳 / 芳基 / 乙烯基底物**不可走标准 SN1/SN2——需走其他机理（SNAr、金属催化等）

## 五、常见分类或情形

### 5.1 按机理分类（核心二分法 + 扩展）

| 机理 | 动力学级数 | 立体化学 | 底物适用范围 | 关键特征 |
|:---|:---:|:---|:---|:---|
| **SN2** | 二级，$v = k[\text{RX}][\text{Nu}^-]$ | 构型翻转（Walden 翻转） | 甲基 > 1° > 2°；3° 不发生 | 背面进攻；一步协同；无中间体 |
| **SN1** | 一级，$v = k[\text{RX}]$ | 外消旋化（翻转略多于保持） | 3° > 苄基/烯丙基 > 2°；1°/甲基 极慢 | 碳正离子中间体；分步；可重排 |
| **SNi** | 与 SN2 相同（二级） | **构型保持**（表观） | SOCl₂/吡啶体系对 2° 醇 | 离去基在"前线"传递亲核试剂到同侧 |
| **SNAr**（加成-消除） | 二级 | — | 芳环含强吸电子基（-NO₂, -CN 等 o/p 位） | Meisenheimer 络合物中间体；吸电子基活化是必要条件 |
| **苯炔机理**（消除-加成） | — | — | 无强吸电子基的芳卤 + 强碱 | 苯炔中间体；亲核试剂可进攻苯炔两端 → 区域异构体混合物 |
| **SRN1**（单电子转移） | 链式自由基-离子机理 | — | 芳基/烯基卤 + 可作电子给体的亲核试剂 | 光照或碱金属引发；对位阻不敏感 |

### 5.2 按底物类型分类

| 底物类型 | 典型机理 | 特殊注意事项 |
|:---|:---|:---|
| **一级卤代烷** | SN2 | 强碱时 E2 竞争（如 t-BuOK → 霍夫曼消除） |
| **二级卤代烷** | SN2 / SN1 交界 | 溶剂和亲核试剂决定走向；是竞赛最常见"判断区" |
| **三级卤代烷** | SN1（E1 竞争） | SN2 不可能（空间位阻极限）；强碱时 E2 反而可走 |
| **烯丙基底物** | SN1 或 SN2 均可 | 烯丙基碳正离子共振稳定 → SN1 容易；SN2 也快（烯丙基位空阻小） |
| **苄基底物** | SN1 或 SN2 均可 | 苄基碳正离子共振稳定 → SN1 快；SN2 也快；但苄基溴是高催泪剂——实验安全 |
| **α-卤代羰基化合物** | SN2（I⁻ 催化） | 邻位 C=O 使 SN2 加速（LUMO 降低）；$\mathrm{BrCH_2COOEt}$ 比 $\mathrm{CH_3CH_2Br}$ SN2 快 $\sim 10^4$ 倍 |
| **桥头碳卤代物** | **不发生 SN1/SN2** | 刚性笼状结构不允许背面进攻（SN2）和平面碳正离子（SN1）；需走其他路径 |
| **芳基卤代物** | SNAr 或苯炔 | 普通芳卤（如氯苯）在无吸电子基活化时几乎不发生亲核取代 |
| **乙烯基卤代物** | 几乎不反应 | $\mathrm{C_{sp^2}-X}$ 键能高；乙烯基碳正离子能量极高；需金属催化 |

### 5.3 按亲核试剂类型分类

| 亲核试剂类型 | 代表试剂 | 典型产物 | 竞赛命名 |
|:---|:---|:---|:---|
| 卤素负离子 | I⁻, Br⁻, Cl⁻ | 卤代烷 | **Finkelstein 反应**（NaI/丙酮） |
| 氧亲核试剂 | OH⁻, R'O⁻, R'COO⁻ | 醇, 醚, 酯 | **Williamson 醚合成**（R'O⁻ + RX → R'OR） |
| 硫亲核试剂 | RS⁻, HS⁻ | 硫醚, 硫醇 | S 亲核性 $\gg$ O 亲核性（可极化性强） |
| 氮亲核试剂 | NH₃, RNH₂, N₃⁻ | 胺, 叠氮化物 | **Gabriel 胺合成**（邻苯二甲酰亚胺钾盐） |
| 碳亲核试剂 | CN⁻, RC≡C⁻, 烯醇负离子 | 腈, 炔烃, β-酮酯 | CN⁻ 是 SN2"一招鲜"——快 + 多一个碳 |
| 氢负离子供体 | LiAlH₄, NaBH₄ | 醇（还原产物） | 虽然不是严格 SN2，但卤代烷 → 烷烃是亲核的 H⁻ 转移 |

### 5.4 按溶剂体系分类

| 溶剂类型 | 促进机理 | 代表溶剂 | 作用机制 |
|:---|:---:|:---|:---|
| **极性非质子溶剂** | SN2 | DMSO, DMF, DMAc, HMPA, 丙酮, MeCN | 阳离子被溶剂化"包裹"→ 阴离子亲核试剂"裸露"→ 亲核性暴增（$\sim 10^3$–$10^6$ 倍） |
| **极性质子溶剂** | SN1（E1） | H₂O, ROH, RCOOH | H-键稳定离去基团和碳正离子 → 降低活化能 |
| **非极性溶剂** | 无特殊促进 | 己烷, CCl₄, 苯 | 既不加速 SN1 也不加速 SN2；但可避免溶剂参与副反应 |
| **相转移催化（PTC）** | SN2（两相体系） | CH₂Cl₂ + H₂O（+ R₄N⁺X⁻） | 季铵盐将水相中的亲核试剂"搬运"到有机相 → 使 SN2 在两相中进行 |

## 六、适用条件与限制

### 6.1 SN2 的适用窗口

| 条件维度 | 有利范围 | 边界/禁区 |
|:---|:---|:---|
| **底物 α-C 取代度** | 甲基（最快）、1°、2° | **3° 碳绝对不发生 SN2**——背面进攻的空间位阻使过渡态能量过高（比相应的 1° 底物高 $\sim 10^4$–$10^6$ 倍） |
| **底物 β-支化** | 直链 > β-支链 | 新戊基型底物（$\mathrm{Me_3CCH_2X}$）SN2 极慢——β-位三个甲基从侧面阻挡亲核试剂接近 |
| **亲核试剂** | 高电荷密度 + 高可极化性 | 大位阻亲核试剂（如 t-BuO⁻, LDA）更适合做碱走 E2 而非 SN2 |
| **溶剂** | 极性非质子溶剂 | 极性质子溶剂中亲核试剂被 H-键"锁住"→ SN2 速率下降 $10^3$–$10^6$ 倍 |
| **离去基团** | 好离去基（弱碱）| OH⁻, NH₂⁻, OR⁻ 是极差的离去基——需先转化为 OTs, OMs 等 |
| **温度** | 低温有利于取代 | 高温促进消除——制备取代产物时应尽量低温 |

### 6.2 SN1 的适用窗口

| 条件维度 | 有利范围 | 边界/禁区 |
|:---|:---|:---|
| **底物 α-C 取代度** | 3° > 苄基/烯丙基 > 2° | 1° 和甲基底物 SN1 极慢——1° 碳正离子能量极高（$\sim 160$–$170\ \mathrm{kJ/mol}$ 高于中性底物），几乎不形成 |
| **碳正离子稳定性** | 可共振稳定（烯丙基/苄基）> 3° > 2° | 乙烯基碳正离子和芳基碳正离子能量极高——相应卤代物不走 SN1 |
| **溶剂** | 极性质子溶剂（高介电常数 + H-键供体能力） | 非极性溶剂中离子对分离困难，SN1 速率极低 |
| **亲核试剂** | 弱亲核试剂即可；浓度不影响速率 | 强亲核试剂存在时 SN2 可能竞争——需综合判断 |
| **离去基团** | 必须有好的离去基 | OH 不能被直接取代——酸性条件下先质子化 → $\mathrm{H_2O}$（好离去基）|
| **重排风险** | **碳正离子重排是 SN1 的伴生风险** | 1,2-H⁻ 迁移、1,2-烷基迁移、Wagner-Meerwein 重排——产物可能不是预期的 |

### 6.3 禁止底物（既不走 SN2 也不走 SN1）

| 底物类型 | 为什么不行 | 替代路径 |
|:---|:---|:---|
| **桥头碳卤代物**（如 1-溴双环[2.2.1]庚烷） | SN2：笼状结构背面无法接近（角张力）；SN1：碳正离子无法平面化 | 无直接替代；合成中用其他路线避开桥头取代 |
| **芳基卤（无吸电子基活化）** | SN2：$\mathrm{C_{sp^2}-X}$ 键能高 + 苯环 π 电子云阻挡背面进攻；SN1：芳基碳正离子能量极高 | SNAr（需 -NO₂, -CN 等吸电子基活化）或苯炔机理 |
| **乙烯基卤（$\mathrm{C=C-X}$）** | 同上——$\mathrm{C_{sp^2}}$ 碳的亲核取代能垒极高 | 金属催化（如 CuI 催化的 Ullmann 型偶联）或消除-加成 |
| **环丙基卤代物** | 环丙烷的 Walsh 轨道具有部分 π 特征——与烯基卤类似，SN2 受阻 | Ag⁺ 辅助（Ag⁺ 拉走 X⁻ → 环丙基碳正离子？会重排为烯丙基碳正离子） |
| **新戊基底物（$\mathrm{Me_3CCH_2X}$）** | β-位三个甲基形成"空间墙"→ SN2 速率比正丙基底物慢 $\sim 10^5$ 倍 | 在实际合成中几乎不用此类底物走 SN2 |

### 6.4 SNAr 的活化条件（缺电子芳环）

- **必要条件**：芳环上必须有**强吸电子基团**（-NO₂, -CN, -SO₂R, -CF₃ 等）处于离去基团的 **o- 或 p- 位**
- **离去基团排序**：F > NO₂ > Cl > Br > I（反常！因为决速步是亲核加成而非离去基断裂——F 的强 -I 效应降低 Meisenheimer 络合物的 LUMO 能量）
- **若只有 m-位吸电子基**：不能有效稳定 Meisenheimer 络合物的负电荷 → SNAr 不反应
- **含 o/p-吸电子基的芳卤活性参照**：2,4-二硝基氯苯 > 对硝基氯苯 > 间硝基氯苯（几乎不反应）

### 6.5 溶剂效应的定量感

| 反应类型 | 溶剂换用 | 速率变化（数量级） | 原因 |
|:---|:---|:---|:---:|
| $\mathrm{CH_3I + Cl^- \to CH_3Cl + I^-}$（SN2） | 甲醇 → DMSO | $\sim 10^6$ 倍加速 | 极性非质子溶剂中 Cl⁻ 不被 H-键"锁住" |
| $\mathrm{t\text{-}BuCl + H_2O \to t\text{-}BuOH + HCl}$（SN1） | 乙醇 → 水 | $\sim 10^2$–$10^3$ 倍加速 | 水介电常数更大，更好稳定过渡态的电荷分离 |
| $\mathrm{PhCH_2Cl + H_2O}$（SN1/SN2 交界） | 丙酮 → 50% 乙醇/水 | 机理可能从 SN2 切换为 SN1 | 极性质子溶剂稳定碳正离子 → 改变机理 |

## 七、常见比较与易混点

### 7.1 SN1 vs SN2 全面对比（⭐ 竞赛最核心对比表）

| 对比维度 | **SN2** | **SN1** |
|:---|:---|:---|
| **动力学** | 二级，$v = k[\text{RX}][\text{Nu}^-]$ | 一级，$v = k[\text{RX}]$ |
| **步骤数** | 1 步（协同） | ≥2 步（碳正离子中间体） |
| **决速步** | C-Nu 键形成与 C-X 键断裂**同步** | C-X 键断裂（形成碳正离子） |
| **立体化学** | Walden 翻转（100% 翻转） | 外消旋化（实际翻转略多于保持，因离子对效应） |
| **底物活性顺序** | $\mathrm{CH_3X} > 1° > 2° \gg 3°$（禁止） | $3° > \text{苄基/烯丙基} > 2° \gg 1°$（极慢） |
| **亲核试剂影响** | 影响速率（在速率方程中） | **不影响速率**（但影响产物分布） |
| **亲核试剂活性顺序** | $\mathrm{RS^- > I^- > CN^- > N_3^- > Br^- > Cl^- > F^-}$ | 弱亲核试剂即可 |
| **溶剂偏好** | 极性非质子（DMSO, DMF, 丙酮） | 极性质子（H₂O, ROH, RCOOH） |
| **离去基团** | 重要：好离去基 $\gg$ 差离去基 | **比 SN2 中更重要**——因为决速步就是 C-X 断裂 |
| **重排** | 无（无碳正离子中间体） | **有**——1,2-H⁻/烷基迁移，Wagner-Meerwein 重排 |
| **邻基参与** | 可发生（构型保持！） | 可发生（加速反应） |
| **邻位大基团** | β-支化显著减速 | β-支化影响较小（甚至可促进——B 张力释放） |
| **α-位 C=O/C=C** | SN2 加速（C=O 拉低 LUMO） | SN1 也加速（烯丙基/α-羰基碳正离子共振稳定） |

### 7.2 亲核取代 vs 消除反应竞争（SN vs E）

| 底物 | 强亲核弱碱（I⁻, RS⁻, CN⁻） | 强亲核强碱（OH⁻, EtO⁻） | 大位阻强碱（t-BuOK） | 弱亲核弱碱（H₂O, EtOH） |
|:---|:---:|:---:|:---:|:---:|
| **1° RX** | SN2 | SN2 + E2 | **E2（霍夫曼产物）** | 几乎不反应 |
| **2° RX** | SN2 | SN2 + E2（E2 ≈ SN2） | **E2（霍夫曼产物）** | SN1 + E1（慢） |
| **3° RX** | 几乎不反应（SN2 禁止） | E2 > SN2（SN2 禁止，只剩 E2） | **E2（霍夫曼产物）** | SN1 + E1（E1 为主，尤其高温） |

> 竞赛阅读法："看试剂先分亲核/碱性，再看底物级别，最后确认溶剂温度"——三步走。详见 [[消除反应]] 和 [[SN1与SN2比较]]。

### 7.3 脂肪族 SN vs 芳香族 SNAr

| 维度 | **脂肪族 SN2** | **SNAr（加成-消除）** |
|:---|:---|:---|
| 底物活化需求 | 不需要（烷基卤本身即可） | **必须**有强吸电子基团（-NO₂, -CN 等）在 o/p 位 |
| 离去基团活性顺序 | I > Br > Cl > F | **F > NO₂ > Cl ≈ Br ≈ I**（反常！F 的 -I 促进加成） |
| 中间体 | 五配位过渡态（无真实中间体） | **Meisenheimer 络合物**（孤立的负离子中间体） |
| 决速步 | 键形成与断裂同步 | 亲核加成（第一步）——离去基断裂是快步骤 |
| 亲核试剂需求 | 需要强亲核试剂 | 中等亲核试剂即可（缺电子芳环 LUMO 低） |
| 溶剂效应 | 极性非质子有利 | 极性质子或非质子均可（对两步骤影响方向相反） |

### 7.4 亲核性 vs 碱性（⭐ 高频混淆）

| 维度 | **亲核性** | **碱性** |
|:---|:---|:---|
| 本质 | 动力学概念——进攻碳原子的**速率** | 热力学概念——夺取质子的**平衡** |
| 测量量 | 相对速率常数 $k_{\text{Nu}}$ | $\mathrm{p}K_a$（共轭酸的酸解离常数） |
| 主要影响因素 | 可极化性 + 溶剂化 + 碱性 + 位阻 | 电负性 + 电荷 + 杂化 + 共振/诱导效应 |
| 规律对齐？ | **不完全对齐** | — |
| 典型"错位"案例 | I⁻ 是弱碱（HI 的 $\mathrm{p}K_a \approx -10$）但是**极好**的亲核试剂 | t-BuO⁻ 是强碱但是**很差**的亲核试剂（位阻大） |
| 同族趋势 | $\mathrm{I^- > Br^- > Cl^- > F^-}$（可极化性主导） | $\mathrm{F^- > Cl^- > Br^- > I^-}$（电负性主导）——**完全相反！** |

> **关键判断口诀**："碱看质子的战斗，亲核看碳的进攻；卤素一族亲核性跟可极化性走，碱性跟电负性走。"

### 7.5 构型翻转（SN2）vs 外消旋化（SN1）——更精细的图景

| 现象 | 理想预期 | 实际情况 | 原因 |
|:---|:---|:---|:---|
| SN2 立体化学 | 100% 翻转 | **确实 100% 翻转**（Walden 循环可严格验证） | 背面进攻是唯一的反应路径 |
| SN1 立体化学 | 50:50（完全外消旋） | 翻转通常 $\sim 55$–$65\%$，保持 $\sim 35$–$45\%$ | **离子对效应**——离去基团在碳正离子一侧短暂停留，部分阻挡同侧进攻 |
| SNi 立体化学 | — | **表观保持** | 离去基团在"前线"将亲核试剂传递到同侧（如 SOCl₂/吡啶体系对醇） |

> 如果产物旋光度数据同时偏离"100% 翻转"和"50:50 外消旋"→ 考虑离子对效应或 SN2/SN1 混合机理。

### 7.6 离去基团——竞赛必须区分的四种判断

| 判断维度 | 规律 | 实例 |
|:---|:---|:---|
| **离去能力**（热力学） | 共轭酸 $\mathrm{p}K_a$ 越低 → 离去能力越强 | TsO⁻（$\mathrm{p}K_a \approx -7$）$\gg$ Cl⁻（$\mathrm{p}K_a \approx -7$）$\approx$ Br⁻ $\approx$ I⁻ $\gg$ F⁻（$\mathrm{p}K_a \approx 3$）$\gg$ OH⁻（$\mathrm{p}K_a \approx 16$）$\gg$ NH₂⁻ |
| **可极化性对 SN2 的额外加成** | 可极化性大的离去基团**额外加速 SN2**（过渡态中可更好地分散负电荷） | I⁻ > Br⁻ > Cl⁻（即使三者共轭酸 pKa 相近，I⁻ 做离去基时 SN2 最快） |
| **"不能走"的离去基** | OH, OR, NH₂ 等强碱性基团**不可直接**被取代 | 需先转化为好离去基：$\mathrm{ROH \xrightarrow{TsCl/py} ROTs}$ 或 $\mathrm{ROH \xrightarrow{SOCl_2} RCl}$ |
| **离去基的"再生"** | 有些反应表现上是"离去基回来了"→ 邻基参与 | 硫芥子气中的 S 参与——分子内 SN2 第一步，第二步 S 以硫鎓盐形式离去 |

## 十、例题

### 10.1 例题 1（★★ 基础 — SN2 构型翻转与 Finkelstein 反应）

**题目**：(R)-2-溴丁烷在干燥丙酮中用过量 NaI 加热回流，写出主要有机产物的结构（需标明立体化学），并说明判断依据。

**分析**：
- 底物：2° 卤代烷，(R)-2-溴丁烷，β-位无大位阻基团 → SN2 可行
- 试剂：I⁻，可极化性强，是很好的亲核试剂；共轭酸 HI 的 $\mathrm{p}K_a \approx -10$，碱性极弱 → 几乎无 E2 竞争
- 溶剂：干燥丙酮（极性非质子溶剂）→ 阳离子 Na⁺ 被溶剂化，I⁻ 保持"裸露"高活性 → 显著加速 SN2
- 综合判断：**纯 SN2 条件**

**解答**：
- 主产物：(S)-2-碘丁烷
- 反应方程式：$$\ce{(R)-CH3CH2CHBrCH3 + NaI ->[dry acetone][\Delta] (S)-CH3CH2CHICH3 + NaBr v}$$
- 立体化学：SN2 机理要求亲核试剂从离去基团**背面进攻** → Walden 翻转 → 手性中心构型完全翻转。Cahn-Ingold-Prelog 规则中 I > Br > CH₂CH₃ > CH₃，(R)-2-溴丁烷经背面翻转后变为 (S)-2-碘丁烷（翻转 + 取代基优先顺序改变共同决定了最终 R/S 符号）。

**反思**：
- 这是经典 **Finkelstein 反应**——利用 NaI/丙酮体系将溴代烷或氯代烷转化为碘代烷，驱动力是 NaBr 或 NaCl 在丙酮中沉淀析出
- **翻转是 SN2 的"身份证"**——如果产物保持构型（或部分保持），则必然有其他机理参与（如邻基参与）
- 如果把溶剂换成乙醇/水（极性质子），2° 底物会出现 SN1 竞争，产物将部分外消旋化

---

### 10.2 例题 2（★★★ 进阶 — 从立体化学数据反推 SN2/SN1 比例）

**题目**：光学纯的 (R)-2-溴辛烷（$[\alpha]_D = -34.6°$）在 60% 乙醇/水中于 50°C 水解。所得 2-辛醇经纯化后测得 $[\alpha]_D = +4.8°$。已知光学纯 (S)-2-辛醇的 $[\alpha]_D = +9.6°$。回答：
a) 计算该反应中 SN2 途径所占的比例（假设 SN1 途径给出完全外消旋产物）。
b) 若希望提高产物的光学纯度，应如何改变反应条件？给出两个具体调整方向。

**分析**：
- (R)-2-溴辛烷 → SN2 翻转 → (S)-2-辛醇（$[\alpha]_D = +9.6°$）
- SN1 → 平面碳正离子 → 外消旋 2-辛醇（$[\alpha]_D = 0°$）
- 实测 $[\alpha]_D = +4.8°$ 介于 0 和 +9.6° 之间 → **SN2 和 SN1 正在竞争**
- 2° 底物在极性质子溶剂中处于 SN1/SN2 的"灰色地带"

**解答**：
- a) 设 SN2 比例为 $x$（$0 \le x \le 1$），SN1 比例为 $1-x$：
  - SN2 产物：100% (S)，ee = 100%
  - SN1 产物：外消旋，ee = 0%
  - 混合产物 ee $= x \times 100\% + (1-x) \times 0\% = x = \dfrac{4.8}{9.6} = 0.50$
  - **SN2 占 50%，SN1 占 50%**
- b) 提高光学纯度 = 提高 SN2 比例，可采取：
  1. **换用极性非质子溶剂**：如用干燥丙酮或 DMSO 代替乙醇/水 → 阴离子亲核试剂"裸露"活性高 → SN2 加速
  2. **降低反应温度**：SN2 的活化焓通常低于 SN1（SN1 需断裂 C−X 键生成碳正离子），低温更有利于 SN2
  3. （补充）使用高浓度强亲核试剂（如 $\ce{NaN3}$/DMF），亲核性远大于水的试剂压倒 SN1 溶剂解

**反思**：
- 立体化学数据是区分 SN1/SN2 的"**金标准**"——比速率方程更可靠（因为有时动力学级数不能区分平行途径）
- 实际体系中，SN1 的碳正离子并非完全"自由"：离去基团阴离子在短时间内停留在碳正离子附近形成**紧密离子对**，部分遮蔽前侧进攻 → SN1 通常给出略多于 50% 的翻转产物（约 55%~70% 翻转）。本题假设"完全外消旋"是简化处理，实际操作中 SN2 占比会**低于**计算值
- 这在 §四第 1 点的"离子对效应"中已有论述——SN1 和 SN2 不是绝对的二元对立

### 10.3 例题 3（★★★★ 综合 — SN2/E2 竞争分析与条件优化）

**题目**：以 2-溴丁烷为底物，在不同条件下反应，产物分布如下表：

| 编号 | 试剂（1 eq） | 溶剂 | 温度 | 取代产物 | 消除产物 |
|:---|:---|:---|:---|:---|:---|
| ① | $\ce{NaOH}$ | $\ce{H2O/EtOH}$（1:1）| 25°C | 60% | 40% |
| ② | $\ce{NaOH}$ | $\ce{H2O/EtOH}$（1:1）| 80°C | 25% | 75% |
| ③ | $\ce{NaI}$ | 干燥丙酮 | 25°C | >95% | <5% |
| ④ | $\ce{t-BuOK}$ | $\ce{t-BuOH}$ | 25°C | 5% | 95% |
| ⑤ | $\ce{NaSCH3}$ | $\ce{CH3OH}$ | 25°C | 88% | 12% |

回答以下问题：
a) 解释 ① 与 ② 的产物比例差异——为什么升高温度有利于消除？
b) 比较 ①、③、④——为什么相同的 2° 底物在不同试剂/溶剂下产物比例截然不同？
c) 比较 ① 与 ⑤——$\ce{CH3S-}$ 和 $\ce{OH-}$ 都是带负电荷的亲核试剂，为什么 $\ce{CH3S-}$ 给出更多取代产物？

**分析**：
- 2° 底物是 SN2/E2 竞争的经典"战场"——两种途径均为二级动力学，条件决定胜负
- SN2 vs E2 的核心判据：试剂的**亲核性/碱性比** + 溶剂极性和类型 + 温度

**解答**：
- a) **$\Delta S^\ddagger$ 效应**：E2 过渡态涉及 C−H 键和 C−X 键同时断裂 + C=C 形成，过渡态结构比 SN2（仅 C−Nu 成键 + C−X 断键）更"松散"，$\Delta S^\ddagger_{\text{E2}} > \Delta S^\ddagger_{\text{SN2}}$。根据 $\Delta G^\ddagger = \Delta H^\ddagger - T\Delta S^\ddagger$，温度升高使 $\Delta G^\ddagger_{\text{E2}}$ 降幅更大 → E2 加速更多 → 消除比例上升。**口诀**："高温消，低温代。"
- b) 三种条件的比较：
  - ① $\ce{NaOH}$/$\ce{H2O-EtOH}$：OH⁻ 既是强亲核试剂又是强碱 → SN2 和 E2 均快 → 激烈竞争
  - ③ $\ce{NaI}$/丙酮：I⁻ 可极化性强（亲核性好），共轭酸 HI 的 $\mathrm{p}K_a \approx -10$（碱性极弱）→ **纯 SN2**，几乎无 E2
  - ④ $\ce{t-BuOK}$/$\ce{t-BuOH}$：叔丁氧基体积巨大，阻碍了背面进攻 SN2，但作为强碱仍可夺取 β-H → **几乎纯 E2**（Hofmann 产物为主）
- c) **可极化性差异**：$\ce{CH3S-}$ 中 S 原子半径大（3p），外层电子云松散、高度可极化 → 亲核性远强于 $\ce{OH-}$（O 为 2p，电子云紧密，可极化性低）。在极性质子溶剂（$\ce{CH3OH}$）中，可极化性对亲核性的贡献超过碱性，因此 $\ce{CH3S-}$ 是比 $\ce{OH-}$ 更好的亲核试剂 → 取代比例更高。可参考亲核性顺序：$\ce{RS- > I- > CN- > HO- > RO-}$（在极性质子溶剂中）。

**反思**：
- 这道题覆盖了 SN2/E2 竞争的**全部四个维度**：试剂亲核/碱性（③ vs ④）、溶剂（③ vs ①）、温度（① vs ②）、可极化性（① vs ⑤）
- 学生最容易犯的错（参见 §十一）：认为"强碱 = 好亲核试剂"——④ 的 t-BuOK 是极强的碱但极差的亲核试剂（位阻大），反而给出 95% 消除产物
- ⑤ 中 $\ce{CH3S-}$/$\ce{CH3OH}$ 虽是极性质子溶剂，但硫负离子的高可极化性使其即使在质子溶剂中仍能维持高亲核活性——这是**可极化性 > 碱性**的经典案例
- 完整的 SN/E 竞争判断流程参见本文 §三 §2.5 及 [[消除反应]]

## 十一、易错点

1. ❌ **"SN2 进攻可以从前侧进行"** → 错！SN2 的轨道要求是亲核试剂 HOMO（孤对电子）与底物 C−X σ* LUMO 的**背面重叠**（backside attack），前侧被离去基团占据且轨道对称性不匹配。凡是写"从前侧进攻的 SN2"均为机理错误。

2. ❌ **"强碱等于好亲核试剂"** → 错！碱性是热力学概念（与 H⁺ 结合的能力），亲核性是动力学概念（与碳结合的速度）。二者相关但不相等。典型反例：$\ce{t-BuOK}$ 是极强的碱（$\mathrm{p}K_a(\ce{t-BuOH}) \approx 18$），但因位阻巨大而亲核性极差。可极化性（如 $\ce{I-}$、$\ce{RS-}$）对亲核性的贡献往往超过碱性。

3. ❌ **"SN1 和 SN2 只由底物级别决定"** → 错！底物级别是第一判据但不是唯一判据。2° 底物在极性非质子溶剂 + 强亲核试剂下以 SN2 为主，在极性质子溶剂 + 弱亲核试剂下以 SN1 为主。必须综合四维判断（**底物/亲核试剂/溶剂/离去基团**）。参见 §三 的 SN/E 竞争判断流程图。

4. ❌ **"SN1 产物一定是完全外消旋的"** → 不准确。实际反应中，碳正离子常以**紧密离子对**或**溶剂分隔离子对**形式存在，离去基团阴离子在离子对寿命内遮蔽前侧 → 翻转产物比例通常高于保持产物（约 55%~70% 翻转）。完全"自由"的碳正离子才给出完美的 50:50。参见 §四第 1 点。

5. ❌ **"极性溶剂总是加速 SN1"** → 不准确！极性质子溶剂（水、醇）通过氢键稳定离去基团和碳正离子 → 加速 SN1；极性非质子溶剂（DMSO、DMF、丙酮）通过溶剂化阳离子使阴离子亲核试剂"裸露" → 加速 SN2。混淆"极性"和"质子"两个维度是常见错误。

6. ❌ **"3° 卤代烷绝对不能发生取代反应"** → 错！3° 卤代烷不能发生 SN2（位阻太大），但可发生 SN1（通过稳定的 3° 碳正离子）。在弱碱性亲核试剂（如 $\ce{H2O}$、$\ce{EtOH}$）中，3° 卤代烷的溶剂解（SN1）是主要途径。"3° = 不可取代"的说法混淆了 SN2 和取代反应的范畴。

7. ❌ **"速率方程可以唯一确定机理"** → 不准确。$v = k[\ce{RX}]$ 确实是 SN1 的特征，但 $v = k[\ce{RX}][\ce{Nu-}]$ 不仅可以是 SN2，也可能是 E2（同为二级），甚至 SN1 在某些条件下也可表观出二级动力学（亲核试剂参与决速步之后的平衡）。需结合立体化学、产物分布、同位素效应等综合判定。

8. ❌ **"离去基团越好，SN2 越有利"** → 不完全正确。好离去基团对 SN1 和 SN2 都加速，但**对 SN1 的促进作用更大**——因为 SN1 的决速步就是 C−X 键的异裂。在比较 SN1 vs SN2 时，极好的离去基团（如 $\ce{OTf-}$、$\ce{OTs-}$）实际上可能使 SN1 路径变得更有竞争力。

9. ❌ **"溶剂只影响反应速率，不影响机理选择"** → 错！溶剂是机理选择的关键变量。以 2° 溴代烷 + $\ce{N3-}$ 为例：在 DMSO 中纯 SN2，在水中主要是 SN1（水是弱亲核试剂 + 极性质子溶剂稳定碳正离子）。溶剂可以"翻转"占优机理。

10. ❌ **"消除反应和取代反应互不相关"** → 错！消除和取代是**竞争关系**，很多试剂（如 $\ce{OH-}$、$\ce{RO-}$）既是亲核试剂又是碱——同一反应条件下两种产物共存（如 §十例题 3 条件 ①）。判断产物分布必须同时考虑 SN 和 E 途径。完整的竞争判断树见 §三 §2.5。

### 12.2 教学组织建议

**推荐教学框架**：以"**SN2 → SN1 → 竞争体系**"的三级递进主线组织教学。

1. **第一层：SN2 — 一步协同的"干净"机理**（15 min）
   - 从最简单的反应入手：$\ce{CH3Br + OH- -> CH3OH + Br-}$
   - 关键讲透：背面进攻的轨道原因（HOMO-LUMO 重叠），构型翻转是必然结果——不是"习惯"而是**轨道对称性要求**
   - 以 Finkelstein 反应（$\ce{R-Br + NaI}$/丙酮）作为 SN2 的"标准示范"
   - 给出四维影响因素表格（底物/亲核试剂/离去基团/溶剂），强调**极性非质子溶剂**的作用

2. **第二层：SN1 — 分步的"离子对"机理**（15 min）
   - 引出问题："如果底物是 3° 卤代烷，SN2 走不通了怎么办？"
   - 碳正离子中间体：平面 sp² → 外消旋化，与 SN2 的 100% 翻转形成鲜明对比
   - 关键难点："为什么 SN1 不是完全外消旋？"——引入**离子对**概念（紧密离子对 → 溶剂分隔离子对 → 自由离子），这是 ABOC 和竞赛中的高频细节考点
   - 溶剂解反应（$\ce{t-BuBr + H2O/EtOH}$）作为 SN1 的"标准示范"

3. **第三层：SN2 vs SN1 的决策体系**（15 min）
   - 用 §三 §2.3 的对比表和 §2.5 的决策树（底物 → 亲核试剂 → 溶剂 → 离去基团）做系统梳理
   - 重点放在 **2° 底物的"灰色地带"**——这是竞赛命题最集中的区域，需要综合四维判断
   - 引入 SN/E 竞争作为高阶扩展，联动 [[消除反应]]

4. **第四层：例题演练与易错点辨析**（15 min）
   - 用 §十 的 3 道分层例题（★★ ~ ★★★★）训练四维判断能力
   - 集中辨析 §十一 中的 10 个高频错误认知，特别是"强碱 = 好亲核试剂"和"SN1 一定完全外消旋"

### 12.3 学生易踩的认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析口诀 |
|---|---|---|---|
| "强碱就是好亲核试剂" | 课本常说 OH⁻ 既是强碱又是好亲核试剂，学生过度泛化 | t-BuOK 是强碱但亲核性差；I⁻ 是弱碱但亲核性强 | "碱性看 pKa，亲核看速度" |
| "SN1 一定外消旋" | 碳正离子是平面 sp² | 离子对效应使翻转略多于保持（约 55%~70%） | "碳正不自由，翻转多一筹" |
| "3° 卤代烃只消除不取代" | 把"不能 SN2"等同于"不能取代" | 3° 可通过 SN1 取代（弱碱亲核试剂中） | "三级不走二，改走一也行" |
| "看速率方程就能定机理" | 二级 = SN2，一级 = SN1，死背结论 | E2 也是二级；SN1 在某些条件下也可表观二级 | "速率推机理，还需立体验" |
| "极性溶剂都加速 SN1" | 混淆"极性"和"质子"两个维度 | 极性非质子加速 SN2（阴离子裸露），极性质子加速 SN1 | "极性分两类，SN1 要质子配" |

### 12.4 入门级引导题

**题目**：$\ce{CH3Br}$ 在以下三种条件下分别与亲核试剂反应，预测主要产物并比较相对速率：
- a) $\ce{NaOH}$ / $\ce{H2O}$
- b) $\ce{NaI}$ / 干燥丙酮
- c) $\ce{NaSCH3}$ / $\ce{CH3OH}$

**预期解答路径**：
1. 底物识别：$\ce{CH3Br}$ 是甲基卤代烷 → 只能走 SN2（甲基碳正离子 $\ce{CH3+}$ 极不稳定，SN1 不可能）
2. 亲核性比较（在各自溶剂中）：在极性质子溶剂中 $\ce{CH3S- > I- > HO-}$，在极性非质子溶剂（b）中 I⁻ 活性更高
3. 结论：三个条件均发生 SN2 取代，相对速率 **b > c > a**（b 中 I⁻ 在非质子溶剂中活性最高；c 中 $\ce{CH3S-}$ 的可极化性使其在质子溶剂中仍保持高亲核性；a 中 OH⁻ 受氢键束缚最显著）

**老师可引导的提问**：
- 卡在机理判断："甲基溴能形成碳正离子吗？那它只能走什么机理？"
- 卡在亲核性比较："硫和氧都在第 VIA 族，为什么 $\ce{CH3S-}$ 的亲核性比 $\ce{OH-}$ 强？——引导到可极化性概念"
- 卡在溶剂效应："为什么同样的 I⁻，在丙酮中比在水中亲核性强？——水中 I⁻ 被氢键'锁住'，丙酮中'自由'"

### 12.5 与现实/直觉的连接

- **类比 1（SN2 翻转）**："SN2 的背面进攻就像雨伞在大风中翻转——离去基团被吹走的瞬间，亲核试剂从背面顶上来，整个伞（碳的四面体构型）翻转了过来。"
- **类比 2（SN1/SN2 竞争）**："SN2 像两人三足跑（协同，步调一致）；SN1 像接力赛（分步，交接棒时可能掉棒 = 重排）。2° 底物是那个在两种比赛方式之间纠结的运动员。"
- **反直觉点**：学生常以为"反应步骤越少越好"，但 SN1 虽是一级反应（动力学简单），却因碳正离子中间体的重排和离子对效应，产物分布反而比 SN2 更复杂。**简单动力学 ≠ 简单产物**。
- **工业连接**：环氧乙烷的开环（SN2 型机理）年产数千万吨，用于生产乙二醇（防冻液、聚酯纤维原料 PET）。卤代烷水解制醇是重要的工业路线。这些是学生能在生活中触摸到的"分子级反应"现实映射。

## 十三、竞赛拓展

1. **邻基参与（Neighboring Group Participation, NGP / Anchimeric Assistance）**
   - 分子内邻近基团（含孤对电子的杂原子、π 键、甚至 σ 键）在离去基团离开时从背面协助 → 形成**三元环（或更大环）中间体** → 外部亲核试剂从同侧（即原离去基团所在侧）进攻 → **构型保留**！
   - 经典案例：**芥子气**（$\ce{ClCH2CH2SCH2CH2Cl}$）水解中硫原子的邻基参与，形成环状锍离子（thiiranium）中间体
   - 另一经典案例：2-溴丙酸根负离子（$\ce{CH3CHBrCOO-}$）的水解——羧酸根负离子从背面邻基参与生成 α-内酯（α-lactone）中间体 → 水从同侧开环 → 构型保留
   - 判断标志三要素：① SN2 预期应翻转但产物**保持构型**；② 反应速率异常快（邻基协助降低了 $\Delta G^\ddagger$）；③ 分子内邻基的几何条件必须满足（通常形成 3 元或 5 元环过渡态最优）
   - 常见邻基：$\ce{-COO-}$、$\ce{-S-}$、$\ce{-O-}$、$\ce{-NR2}$、苯基（π 参与，形成苯鎓离子桥）、$\ce{C=C}$（π 参与）

2. **非经典碳正离子（Non-classical Carbocations）**
   - 碳正离子不以经典的 sp² 三配位形式存在，而是通过 σ 键或 π 键的电子离域形成**多中心键**
   - 最著名案例：**2-降冰片基正离子**（2-norbornyl cation）——是否是对称的非经典结构（σ 桥连）引发了有机化学史上最著名的论战之一（Brown vs Winstein 之争）
   - 竞赛考查方式：给出异常的产物分布（如预期多种产物但只得到单一产物，或产物分布不符合经典碳正离子重排预期），要求用非经典碳正离子解释
   - 判断线索：反应速率异常大（σ 参与稳定了过渡态/中间体）、产物保持构型、无经典重排产物

3. **SNi 机理（分子内亲核取代 / Substitution Nucleophilic Internal）**
   - 外观像 SN1（速率 $= k[\ce{ROH}]$，一级反应），但立体化学结果因体系而异（保持或翻转）
   - 典型反应：醇与 $\ce{SOCl2}$ 反应生成氯代烷——先形成氯代亚硫酸酯 $\ce{ROS(=O)Cl}$，离去基团（$\ce{SO2}$ + $\ce{Cl-}$）在分子内以 SNi 方式被 Cl⁻ 取代
   - 竞赛关键区别：
     - **无外加碱**（如无吡啶）：Cl⁻ 来自同一面的分子内回攻 → **构型保持**
     - **加入吡啶**（捕获 HCl）：释放出游离 Cl⁻ 做正常的分子间 SN2 → **构型翻转**
   - 这是经典的选择性控制考点：同一反应、同一底物，加不加吡啶决定产物构型

4. **两可亲核试剂的区域选择性（Ambident Nucleophiles / Kornblum 规则）**
   - **两可亲核试剂**：含有两个亲核位点的试剂
     - $\ce{CN-}$：C 端（烷基化 → 腈）vs N 端（烷基化 → 异腈）
     - $\ce{NO2-}$：N 端（烷基化 → 硝基化合物）vs O 端（烷基化 → 亚硝酸酯）
     - $\ce{SCN-}$：S 端 vs N 端
     - 烯醇负离子：C 端 vs O 端
   - **Kornblum 规则**：SN2 条件下，**可极化性更高的原子**优先进攻碳亲电中心（$\ce{CN-}$ 以 **C** 烷基化为主 → 腈；$\ce{NO2-}$ 以 **N** 进攻为主 → 硝基烷烃）
   - **Ag⁺ 催化的反转**：加入 $\ce{Ag+}$ 与卤素配位促进 C−X 异裂 → 反应走向 SN1 型 → $\ce{NO2-}$ 改为以 **O** 端进攻 → 亚硝酸酯。这是竞赛命题中区分 SN2/SN1 的经典"遥控"手段
   - **HSAB 理论解释**：软亲核位点（可极化性高，如 S、I、C-端 CN⁻）优先与较软的亲电碳反应；硬亲电位点（电负性高，如 O、N-端）倾向于质子转移或与硬亲电试剂反应

5. **相转移催化（Phase-Transfer Catalysis, PTC）**
   - **问题**：无机亲核试剂（$\ce{CN-}$、$\ce{N3-}$、$\ce{MnO4-}$）以钠盐/钾盐形式存在，难溶于有机溶剂 → SN2 在非均相体系中进行极慢
   - **解决**：加入季铵盐（如 $\ce{Bu4N+HSO4-}$）或冠醚（如 18-crown-6）作为相转移催化剂 → 将阴离子从水相"搬运"到有机相 → SN2 速率大幅提升
   - 竞赛考点：
     - PTC 条件下 SN2 反应不需要极性非质子溶剂（可用廉价的 $\ce{CH2Cl2}$、甲苯）
     - 催化剂在两相间循环（催化量即可）
     - 冠醚通过尺寸匹配选择性络合特定阳离子（如 18-crown-6 与 K⁺）→ 释放高活性"裸露"阴离子

6. **动力学同位素效应（KIE）在 SN 机理判定中的应用**
   - **一级氘代 KIE**（$k_H/k_D$）——用于区分 SN2 和 E2：
     - SN2：$k_H/k_D \approx 1.0$（β 位 C−H 键不参与决速步的键断裂）
     - E2：$k_H/k_D \approx 2\sim8$（β 位 C−H 键在决速步中断裂）→ 这是区分 SN2 与 E2 的强力证据
   - **二级 α-氘代 KIE**——用于区分 SN1 和 SN2：
     - SN1：$k_H/k_D \approx 1.10\sim1.25$（Cα 从 sp³ → sp²，面外弯曲振动频率降低，零点能差增大）
     - SN2：$k_H/k_D \approx 0.95\sim1.05$（Cα 在过渡态中配位数增加、拥挤度增大，与 SN1 方向相反）

7. **非经典底物的 SN 反应禁区**
   - **桥头碳卤代物**（如 1-溴双环[2.2.1]庚烷）：背面进攻被桥环结构完全阻挡 → SN2 不可能；桥头碳正离子因刚性结构难以达到平面 sp²（违反 Bredt 规则）→ SN1 也被严重抑制。这类底物是"**SN 的禁区**"，取代反应极慢。
   - **乙烯基卤代物**（$\ce{CH2=CHBr}$）和**芳基卤代物**（$\ce{C6H5Br}$）：C−X 键与 π 体系共轭增强键能，且 sp² 碳的 SN2 背面进攻受 π 电子云空间阻碍。需走其他机理（SNAr 加成-消除、苯炔消除-加成、或过渡金属催化偶联）。

3. **Clayden《Organic Chemistry》（第 2 版）**：第 15 章"Nucleophilic substitution"——现代视角的 SN 机理讲述，特别推荐其邻基参与和碳正离子重排的**分子轨道图示**（HOMO-LUMO 重叠在 SN2 背面进攻中的可视化），以及 SN2 过渡态的立体电子效应分析。

4. **March《Advanced Organic Chemistry》（第 7 版）**：第 10 章——脂肪族亲核取代的全面综述，涵盖经典机理、离子对理论（紧密/溶剂分隔/自由离子）、非经典碳正离子、两可亲核试剂、相转移催化等高级主题。适合 IChO 备赛深入阅读。

5. **《化竞教程-有机化学》**：第五章——SN 反应在竞赛中的常见考查角度、解题要点和易错辨析，含历年国初/国决真题分类训练。

6. **IChO 历年真题**：
   - 几乎每年 Preparatory Problems 或 Theoretical Exam 中都会涉及 SN 机理判断或立体化学预测
   - 常见命题形式：给定底物 + 试剂 + 溶剂组合 → 判断机理（SN1/SN2/E1/E2）→ 画产物立体结构 → 解释产物分布
   - 决赛级命题通常联合考查取代与消除的竞争（SN/E 竞争的综合分析）

7. **《Modern Physical Organic Chemistry》（Anslyn & Dougherty）**：第 8 章——有机反应机理的物理有机视角，含水动力学同位素效应（一级/二级 KIE）、Hammett 线性自由能关系、溶剂效应的定量处理（Grunwald-Winstein 方程）。适合对物理有机方向感兴趣的高阶学生阅读。
