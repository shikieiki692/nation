# 邻基参与（Neighboring Group Participation, NGP）

## 一、定义

**邻基参与**是指分子中与反应中心相邻的基团（含孤对电子或 π 电子）在反应过程中主动参与，通过分子内亲核进攻形成**环状中间体**（或过渡态），从而影响反应速率、产物分布和立体化学的现象。

- **别名**：Anchimeric Assistance（邻位协助）
- **核心特征**：
  1. **分子内过程**：邻基从同一分子的另一位置参与
  2. **形成环状中间体**：通常为三元环至五元环
  3. **加速反应**：邻基参与通常显著加快反应速率
  4. **立体化学控制**：常导致 **构型保持（net retention）**
- **与分子间亲核取代的竞争**：邻基参与本质上是**分子内 SN2**，与分子间亲核试剂的 SN2 竞争

## 三、核心原理

### 3.1 邻基参与的基本机理

**三阶段机理**：
```
步骤 1：离去基离去（或部分离去）→ 邻基从背面进攻
         → 形成环状中间体（鎓离子或桥式中间体）
步骤 2：外部亲核试剂从背面进攻环状中间体
         → 开环，形成新键
步骤 3：净结果取决于两次进攻的方向
```

**立体化学分析**：
- 邻基参与要求邻基与离去基处于 **反式共平面（anti-periplanar）**
- 邻基从背面进攻 → 反应中心 **构型翻转**
- 外部亲核试剂从背面进攻环状中间体 → 反应中心 **再次构型翻转**
- **净结果：构型保持（retention）**——两次翻转 = 回到原构型

### 3.2 常见的邻基参与基团

| 邻基类型 | 例子 | 形成的环状中间体 | 说明 |
|:---|:---|:---|:---|
| **含孤对电子的杂原子** | —OH, —OR, —NR₂, —SR | 环状鎓离子（如环氧鎓）| 最常见 |
| **羰基/酯基** | —COOR, —CONR₂ | 酰基鎓离子 | 酯的碱性水解中常见 |
| **π 体系** | 双键、苯环 | 桥式碳正离子（非经典）| 如降冰片烯体系 |
| **C—C σ 键** | 环丙基 | 环丙基甲基正离子 | 邻基参与的特殊形式 |
| **卤素** | —Cl, —Br, —I | 卤鎓离子 | 卤素的邻基参与 |

### 3.3 邻基参与的证据

| 证据类型 | 现象 | 说明 |
|:---|:---|:---|
| **速率加速** | 比无邻基时快 10²–10⁶ 倍 | 分子内反应的有效摩尔浓度极高 |
| **构型保持** | 产物构型与底物相同 | 两次背面进攻的净结果 |
| **重排产物** | 出现邻基迁移产物 | 邻基迁移到原离去基位置 |
| **同位素标记** | 标记原子出现在产物不同位置 | 证明环状中间体的存在 |
| **环状中间体捕获** | 在特定条件下分离到环状中间体 | 最有力证据 |

## 四、关键结论

1. **邻基参与是导致"SN1 反应中构型保持"的重要原因**：看似 SN1（碳正离子特征），实则通过环状中间体实现两次翻转
2. **邻基参与对立体化学有严格要求**：邻基与离去基必须处于 anti-periplanar 构象
3. **邻基参与的反应速率远快于普通 SN1/SN2**：分子内反应的有效浓度效应
4. **酯的碱性水解中的邻基参与（BAC2 vs BAC1）**：某些酯水解速率异常快，源于羰基氧的邻基参与
5. **非经典碳正离子的本质就是邻基参与**：环丙基、双键、苯环对碳正离子的稳定化属于广义的邻基参与

## 五、常见分类

### 5.1 按邻基类型分类

```
邻基参与
├── 杂原子邻基参与（最常见）
│   ├── 羟基/烷氧基（—OH, —OR）
│   │   └── β-卤代醇的溶剂解 → 环氧中间体
│   ├── 氨基（—NR₂）
│   │   └── 氮丙啶鎓离子
│   ├── 硫基（—SR）
│   │   └── 环状锍离子（如芥子气水解）
│   └── 卤素（—Cl, —Br, —I）
│       └── 卤鎓离子
├── 羰基邻基参与
│   ├── 酯/酰胺的羰基氧参与
│   │   └── 酯碱性水解加速
│   └── 羧酸根参与
│       └── 乳酸衍生物的取代
├── π 体系邻基参与
│   ├── 烯丙基双键参与
│   ├── 芳环参与
│   └── 环丙基参与
└── σ 键参与（非经典碳正离子）
    └── 环丙基甲基正离子
```

### 5.2 经典实例

| 反应 | 底物 | 邻基 | 中间体 | 产物特征 |
|:---|:---|:---|:---|:---|
| **β-卤代醇溶剂解** | Br-CH₂-CH₂-OH | —OH | 环氧鎓离子 | 构型保持 |
| **芥子气水解** | Cl-CH₂-CH₂-S-CH₂-CH₂-Cl | S | 环状锍离子 | 快速水解 |
| **2-溴丙酸水解** | Br-CH(CH₃)-COOH | COO⁻ | α-内酯中间体 | 构型保持 |
| **降冰片烯衍生物** | 2-降冰片烯基-X | C=C | 非经典碳正离子 | 外式（exo）产物 |
| **环丙基甲基溴** | 环丙基-CH₂-Br | 环丙基 | 双环丁基正离子 | 重排产物 |

## 六、适用条件

### 6.1 发生邻基参与的条件

| 因素 | 条件 | 说明 |
|:---|:---|:---|
| **几何要求** | 邻基与离去基 **anti-periplanar** | 背面进攻的立体要求 |
| **邻基亲核性** | 需有足够亲核性（孤对电子或 π 电子）| O, N, S, 双键, 芳环 |
| **离去基** | 需有可离去基团 | 卤素、磺酸酯等 |
| **溶剂** | 极性质子溶剂有利于鎓离子稳定 | 但亲核性溶剂可能直接进攻 |
| **温度** | 通常室温 | 邻基参与降低活化能 |

### 6.2 邻基参与 vs 普通 SN2/SN1 的竞争

| 条件 | 邻基参与占优 | 普通取代占优 |
|:---|:---|:---|
| **溶剂** | 非亲核性溶剂 | 强亲核性溶剂 |
| **邻基亲核性** | 强 | 弱或无 |
| **立体化学** | anti-periplanar 满足 | 不满足 |
| **温度** | 通常不影响选择 | 高温可能 favor SN1 |

## 七、常见比较

### 7.1 邻基参与 vs 普通 SN1 vs SN2

| 维度 | 邻基参与 | 普通 SN1 | SN2 |
|:---|:---|:---|:---|
| **机理** | 分子内两步（环状中间体）| 碳正离子中间体 | 协同一步 |
| **立体化学** | **构型保持** | 外消旋化 | 构型翻转 |
| **速率** | 很快（分子内加速）| 中等 | 快 |
| **重排** | 可能（开环方式不同）| 常见 | 无 |
| **溶剂效应** | 非亲核溶剂有利 | 极性质子溶剂有利 | 极性非质子溶剂有利 |

### 7.2 两次翻转 = 构型保持的机理

以 (S)-2-溴-1-丙醇为例：

```
(S)-HO-CH₂-CH(Br)-CH₃
         ↓ 邻基 —OH 从背面进攻
    环氧鎓离子（翻转一次）
         ↓ Nu⁻ 从背面进攻开环
(R)-HO-CH₂-CH(Nu)-CH₃
```

**注意**：如果亲核试剂进攻的是 CH₂（而非手性碳），则手性碳的构型保持。

## 十、例题

### 例题 1：β-卤代醇的溶剂解

**题目**：解释为什么 (S)-2-溴-1-丙醇在碱性水溶液中水解得到构型保持的产物。

**解析**：
1. **邻基参与**：—OH 在 C1，离去基 Br 在 C2，二者处于适当位置
2. **第一步**：—O⁻（或 —OH 去质子后）从背面进攻 C2，Br⁻ 离去
   - 形成 **环氧中间体**
   - C2 的构型 **翻转**（第一次）
3. **第二步**：OH⁻ 从背面进攻环氧的 C2（或 C1，取决于位阻）
   - 开环
   - C2 的构型 **再次翻转**（第二次）
4. **净结果**：两次翻转 = **构型保持**

### 例题 2：芥子气的水解

**题目**：芥子气（S(CH₂CH₂Cl)₂）的水解比相应的简单硫醚快得多，解释原因。

**解析**：
1. **邻基参与**：一个氯乙基的 Cl 离去时，硫原子上的孤对电子从背面进攻
2. **形成环状锍离子**：三元环锍离子中间体
3. **速率加速**：分子内反应，有效浓度极高
4. **第二步**：水分子进攻开环 → 水解产物
5. **循环**：剩余的氯乙基可再次发生邻基参与

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| "邻基参与导致构型翻转" | 只算了一步 | 邻基参与是**两次翻转** = **构型保持** | 邻基参与的净立体化学结果是什么？为什么？ |
| "所有邻位杂原子都参与" | 忽略立体要求 | 必须满足 **anti-periplanar** 几何 | syn-构型的 β-卤代醇能发生邻基参与吗？ |
| "邻基参与就是 SN1" | 机理混淆 | 邻基参与是**分子内两步**，不是碳正离子 SN1 | 邻基参与和 SN1 的本质区别是什么？ |
| "邻基参与只加速反应" | 忽略立体化学影响 | 邻基参与同时**控制立体化学** | 邻基参与除了加速，还带来什么影响？ |

### 12.3 入门例题

**最简判断练习**：
> 判断下列底物是否可能发生邻基参与：
> 1. HO-CH₂-CH₂-Cl
> 2. CH₃-CH(OH)-CH₂-Br（anti 构型）
> 3. CH₃-CH(OH)-CH₂-Br（syn 构型）

**答案**：
1. **可以**（邻基 OH 与 Cl 可形成 anti 构象）
2. **可以**（anti-periplanar 满足）
3. **不行**（syn 构型不满足背面进攻要求）
**引导**：问"为什么 syn 构型不行？"（背面进攻的立体要求）

### 12.4 现实类比

**"接力赛"类比邻基参与**：
- 普通 SN2 = **对手直接抢球**——一个球员（亲核试剂）从背后抢断持球者（底物）→ 球的方向翻转
- 邻基参与 = **队友先抢，再传球**——队友（邻基）先从背后抢断持球者（第一次翻转）→ 形成小圈子（环状中间体）→ 队友把球传给外援（亲核试剂），外援从背后接球（第二次翻转）→ 最终持球方向看似没变（构型保持）
- 芥子气 = 队长（S）有两个球（两个氯乙基）——队长可以先抢一个球形成小圈，再交给外援；然后抢第二个球，再交给外援——所以水解特别快
