# Knoevenagel 缩合（Knoevenagel Condensation）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[45-亲核加成反应]]、[[46-羰基α位反应]]

---

## 一、定义

**Knoevenagel 缩合**：含 **两个吸电子基（EWG）的活泼亚甲基化合物**与**醛/酮**在**弱碱**催化下缩合，生成 **α,β-不饱和二羰基/腈** 的反应。

**通式**：
$$
\ce{R-CHO + CH2(CN)2 ->[base] R-CH=C(CN)2 + H2O}
$$

**本质**：**双吸电子基稳定碳负离子**，使弱碱即可脱质子，产物高度不饱和且易脱水。

## 三、核心原理

### 3.1 反应机理

```
步骤 1：弱碱夺取活泼亚甲基的 H → 稳定碳负离子
步骤 2：碳负离子进攻醛/酮的羰基碳
步骤 3：脱水 → α,β-不饱和产物
```

**与 Aldol 的区别**：
- Knoevenagel：双 EWG 稳定碳负离子，**弱碱**即可
- Aldol：单 EWG，需要**较强碱**形成烯醇盐
- Knoevenagel 产物**自发脱水**（双 EWG 稳定双键）

### 3.2 活泼亚甲基化合物

| 化合物 | EWG 组合 | pKa（估算）|
|:---|:---|:---:|
| **丙二酸二乙酯** | 两个酯基 | ~13 |
| **氰基乙酸乙酯** | 酯 + 氰基 | ~9 |
| **丙二腈** | 两个氰基 | ~11 |
| **巴比妥酸** | 两个酰胺 | ~4 |
| **硝基甲烷** | 硝基 | ~10 |

### 3.3 催化剂

| 催化剂 | 强度 | 适用 |
|:---|:---|:---|
| **哌啶** | 弱碱 | 最常用 |
| **乙酸铵** | 弱碱/弱酸 | 可逆控制 |
| **吡啶** | 弱碱 | 温和条件 |
| **伯胺/仲胺** | 弱碱 | 原位形成亚胺 |

---

## 四、关键结论

1. **Knoevenagel = 活泼亚甲基 + 醛/酮 + 弱碱 → α,β-不饱和产物**
2. **双 EWG 使碳负离子极稳定**——弱碱即可催化
3. **产物自发脱水**——双 EWG 极大稳定双键
4. **是 Michael 受体的重要来源**

## 五、常见分类或情形

### 5.1 按醛/酮分类

| 羰基化合物 | 反应性 | 备注 |
|:---|:---:|:---|
| **醛** | ✅ 优秀 | 反应快 |
| **酮** | ⚠️ 中等 | 位阻大时较慢 |
| **脂肪醛** | ✅ 优秀 | 比芳香醛更快 |

---

## 六、适用条件与限制

1. **活泼亚甲基必须有两个 EWG**
2. **弱碱即可**——太强碱可能导致副反应
3. **常需脱水条件**——Dean-Stark 装置或分子筛
4. **酮反应性低于醛**

---

## 七、常见比较与易混点

### 7.1 Knoevenagel vs Aldol

- Knoevenagel：双 EWG，弱碱，自发脱水
- Aldol：单 EWG，较强碱，脱水需加热
- Knoevenagel 是 Aldol 的"超级活化版"

## 十三、竞赛拓展

1. **Knoevenagel 缩合的"酸强度调谐"——为什么要用弱碱？**：Knoevenagel 缩合用哌啶/吡啶（pKa ~11）等弱碱，而非 NaOH/NaOEt 等强碱——因为强碱会导致醛的自身 Aldol 缩合竞争。活性亚甲基的 pKa 需要在 10-14 范围（丙二酸酯 pKa ~13，乙酰乙酸乙酯 pKa ~11，氰基乙酸酯 pKa ~9）。酸太弱则去质子化不完全；碱太强则副反应泛滥。竞赛中选碱是 Knoevenagel 收率的关键——这也是"温和条件"在有机合成中重要性的最佳示例。

2. **Doebner 修饰——Knoevenagel + 原位脱羧的串联**：丙二酸 + 醛 + 吡啶/哌啶 → Knoevenagel 缩合 + 加热脱羧 → α,β-不饱和羧酸（肉桂酸型产物）。这是制备肉桂酸及其衍生物的标准路线。机理要点：吡啶作为碱去质子化丙二酸 → 缩合 → β-羟基二酸 → 碱催化脱羧（六元环过渡态）。竞赛中给"醛 + 丙二酸 → 不饱和酸"的条件即 Doebner 修饰。
