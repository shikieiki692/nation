# 乙酰乙酸乙酯合成法

## 一、定义

**乙酰乙酸乙酯合成法** 是以乙酰乙酸乙酯（CH₃COCH₂COOEt）为底物，利用其活泼 α-氢的酸性和亲核取代能力，经 **烷基化/酰基化 → 水解脱羧** 两步构建取代酮的经典合成方法。

- **核心原理**：乙酰乙酸乙酯中两个羰基之间的亚甲基（—CH₂—）的 pKa ≈ 11，可被乙醇钠等弱碱去质子化生成稳定的烯醇负离子
- **与丙二酸二乙酯合成法并列**为有机合成两大"β-二羰基合成策略"

## 二、反应流程

### 2.1 标准步骤

```
步骤 1：去质子化
  CH₃COCH₂COOEt + NaOEt → CH₃COCH⁻COOEt Na⁺ + EtOH

步骤 2：烷基化
  CH₃COCH⁻COOEt + R-X → CH₃COCHRCOOEt + X⁻

步骤 3：水解脱羧
  CH₃COCHRCOOEt → (1) OH⁻/H₂O, Δ → (2) H⁺ → CH₃COCH₂R + CO₂ + EtOH
```

**三步总结果**：R-X 将 R 基引入乙酰乙酸乙酯的 α-位 → 水解脱羧后得 **取代丙酮**（RCH₂COCH₃）

### 2.2 变体

| 变体 | 步骤 2 试剂 | 产物 |
|:-----|:-----------|:-----|
| 烷基化 | R-X (伯卤代烃) | 单取代丙酮 RCH₂COCH₃ |
| 二烷基化 | R-X 两次 | 二取代丙酮 R₂CHCOCH₃ |
| 酰基化 | R-COCl 或 R-COOR' | β-二酮 RCOCH₂COCH₃ |
| 烯丙基化 | CH₂=CHCH₂-X | 不饱和酮 |

## 四、教学常见节点

- **两步选择性**：先烷基化再脱羧 vs 先脱羧再烷基化——不能先脱羧
- **卤代烃限制**：必须用伯卤代烃或甲基卤（Sₙ2 要求）
- **与丙二酸酯对比**：乙酰乙酸乙酯→取代酮；丙二酸二乙酯→取代羧酸
- **二烷基化**：可连续两次引入不同 R 基（第一步单取代，第二步再取代）
- **分子烷基化**：二卤代物可得环酮（Dieckmann 缩合的先驱）

## 五、竞赛拓展

- **不对称合成中的应用**：手性相转移催化下的不对称烷基化
- **Michael 加成串联**：可与 α,β-不饱和羰基发生 Michael 加成
- **与丙二酸酯互补选择**：合成酮 vs 合成酸的路线选择判断
- **Carroll 重排**：β-酮酸烯丙基酯的 [3,3]-σ 迁移脱羧
