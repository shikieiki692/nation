# 炔烃（Alkyne）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[37-加成反应]]、[[43-亲电加成反应]]

---

## 一、定义

**炔烃（Alkyne）**：含碳碳三键（C≡C）的不饱和烃。与烯烃相比，炔烃具有独特的反应性——既能发生亲电加成（比烯烃慢），也能发生亲核加成（烯烃一般不）。

**本质**：sp 杂化碳的高电负性 + 炔烃的可极化性，赋予其介于烯烃和烷烃之间的独特反应谱。

> ⭐ **ABOC 行 2682**："提要：炔烃与烯烃反应性的对比，简要说明炔烃的独特反应性。"

## 三、核心原理

### 3.1 炔烃 vs 烯烃的反应性对比

| 特征 | **烯烃** | **炔烃** |
|:---|:---|:---|
| **杂化** | sp² | sp |
| **键角** | ~120° | 180° |
| **电负性** | 较低 | 较高（sp > sp²）|
| **亲电加成速率** | 快 | **慢**（烯基正离子更不稳定）|
| **亲核加成** | 一般不发生 | **可以发生**（端炔、缺电子炔）|
| **Lewis 酸催化** | 通常不需要 | 常需要（增强亲电性）|
| **可生成负离子** | 烯基负离子极不稳定 | **炔基负离子较稳定**（sp 碳负电性高）|

**亲电加成更慢的原因**：
- 亲电加成经历碳正离子中间体
- 烯基正离子（vinyl cation，sp 杂化）比烷基碳正离子（sp²/sp³）更不稳定
- 因此炔烃亲电加成需要更强的条件或 Lewis 酸催化

**亲核加成的原因**：
- sp 碳电负性高，使三键电子云更易被极化
- 端炔的 C-H 键有一定酸性（pKa ~25），可去质子形成炔基负离子

### 3.2 亲电加成

#### 加卤素（Br₂ / Cl₂）

- 比烯烃慢，通常需 Lewis 酸催化
- 经历卤鎓离子中间体（类似烯烃）
- **反式加成**
- 可控制在一加成或二加成

#### 加 HX

- 马氏加成
- 经历烯基正离子中间体（不稳定）
- 通常需要 Hg²⁺ 等催化剂
- 二加成可生成偕二卤化物

#### 水合反应

- **Hg²⁺/H₂SO₄ 催化**：马氏水合
  - 产物：烯醇 → 互变异构 → **酮**（或醛，对于乙炔）
- **硼氢化-氧化**：反马氏水合
  - 末端炔 → 醛；内炔 → 酮

### 3.3 亲核加成

**烯烃一般不发生亲核加成，但炔烃可以**——这是炔烃最核心的独特性。

#### 与醇加成（KOH 催化）

```
HC≡CH + ROH  --(KOH)-->  CH₂=CH-OR（乙烯基醚）
```

- 亲核试剂 RO⁻ 进攻三键
- 经历烯基负离子中间体（被共轭稳定）
- 是制备乙烯基醚的重要工业方法

#### 与羧酸加成

```
HC≡CH + RCOOH  --(Hg²⁺)-->  CH₂=CH-OCOR（乙烯基酯）
```

#### 与 HCN 加成（工业制备丙烯腈）

```
HC≡CH + HCN  --(CuCl/NH₄Cl)-->  CH₂=CH-CN（丙烯腈）
```

- Cu⁺ 催化下，CN⁻ 对炔烃的亲核加成
- 是重要的工业有机合成路线

#### 与硫醇加成

```
R-C≡CH + R'SH  --(碱)-->  R-C(S-R')=CH₂
```

### 3.4 炔基负离子的反应

端炔的酸性（pKa ≈ 25）使其可被强碱去质子：

```
R-C≡C-H  +  NaNH₂  →  R-C≡C⁻ Na⁺  +  NH₃
```

**炔基负离子的应用**：
1. **亲核取代**：R-C≡C⁻ + R'-X → R-C≡C-R'（炔烃偶联，增长碳链）
2. **亲核加成**：与羰基化合物反应 → 炔丙醇
3. **金属炔化物**：Cu/Ag 盐用于偶联反应（Glaser 偶联）

### 3.5 还原反应——立体化学控制

#### Lindlar 催化剂：顺式烯烃

**试剂**：Pd/CaCO₃，用醋酸铅或喹啉**毒化**

**机理**：
- 催化氢化（H₂ 加成）
- 毒化降低了 Pd 的活性，使其无法进一步还原烯烃
- **syn 加成** → **顺式烯烃**（Z-烯烃）

**特点**：
- 选择性停在烯烃阶段（不还原为烷烃）
- 产物：**Z-烯烃**
- 是制备顺式烯烃的经典方法

#### Na/NH₃(l)（Birch 还原条件）：反式烯烃

**机理**：
- **单电子还原**（自由基负离子机理）
- Na 给出一个电子 → 炔烃形成自由基负离子
- 质子化（NH₃ 提供 H⁺）→ 乙烯基自由基
- 第二个电子 → 乙烯基负离子
- 第二次质子化 → **反式烯烃**（E-烯烃）

**特点**：
- 不是催化氢化，而是**溶解金属还原**
- 产物：**E-烯烃**
- 机理与 Lindlar 完全不同

#### Lindlar vs Na/NH₃ 对比

| | **Lindlar** | **Na/NH₃(l)** |
|:---|:---|:---|
| **试剂** | H₂ / 毒化 Pd | Na / 液 NH₃ |
| **机理** | 催化氢化 | 溶解金属还原（单电子转移）|
| **立体化学** | **syn** → **Z-烯烃** | **anti** → **E-烯烃** |
| **选择性** | 停在烯烃 | 停在烯烃 |
| **本质** | 表面催化 | 溶液均相自由基负离子 |

> ⭐ **ABOC 踩坑回流**："Lindlar 和 Na/NH₃ 都是还原炔烃"——只记了结果，机理完全不同。

### 3.6 氧化反应

#### KMnO₄ 热/浓氧化

- 三键彻底断裂
- 末端炔 → CO₂ + 羧酸
- 内炔 → 两分子羧酸

#### 臭氧化

- 三键断裂
- 产物：α-二羰基化合物（或进一步氧化产物）

#### 弱氧化（O₂ / CuCl）

- 乙炔氧化偶联 → 乙烯基乙炔（工业制备）

## 四、关键结论

1. **炔烃亲电加成比烯烃慢**——烯基正离子不稳定
2. **炔烃可发生亲核加成**——烯烃一般不，这是炔烃最核心的独特性
3. **Lindlar = Z-烯烃（syn）**；**Na/NH₃ = E-烯烃（anti）**——机理完全不同
4. **端炔水合 → 醛（反马氏，硼氢化）或 甲基酮（马氏，Hg²⁺）**；**内炔水合 → 酮**
5. **端炔有酸性（pKa~25）**——可形成炔基负离子，用于碳链增长
6. **炔烃氧化断裂得羧酸**（KMnO₄）或 α-二羰基（O₃）

## 五、常见分类或情形

### 5.1 按反应类型分类

| 反应 | 试剂 | 产物 | 特征 |
|:---|:---|:---|:---|
| **亲电加成（马氏）** | Hg²⁺/H₂SO₄, H₂O | 酮/醛 | 烯醇互变 |
| **亲电加成（反马氏）** | (Sia)₂BH → H₂O₂ | 醛/酮 | 硼氢化-氧化 |
| **亲核加成** | ROH/KOH, HCN/Cu⁺ | 乙烯基醚/腈 | 烯烃不能 |
| **还原（顺式）** | H₂ / Lindlar | Z-烯烃 | 毒化 Pd |
| **还原（反式）** | Na / NH₃(l) | E-烯烃 | Birch 条件 |
| **氧化断裂** | KMnO₄ 热浓 | 羧酸 | 彻底断裂 |

### 5.2 按炔烃类型分类

| 类型 | 结构 | 特殊性质 |
|:---|:---|:---|
| **端炔** | R-C≡CH | 有酸性 C-H；水合得醛（反马氏）或甲基酮（马氏）|
| **内炔** | R-C≡C-R' | 无酸性 H；水合得酮 |
| **共轭炔** | R-C≡C-C=C-R' | 亲电/亲核选择性更复杂 |

## 六、适用条件与限制

1. **亲电加成常需 Lewis 酸催化**——纯 Br₂/HX 反应慢
2. **端炔去质子需强碱**——NaNH₂、RLi、RMgX 等（pKa ~25，比普通醇酸性强但比水弱）
3. **Lindlar 催化剂需精确毒化**——毒化不足会过度还原为烷烃
4. **Na/NH₃ 需在低温下进行**——液氨沸点 −33°C，操作不便
5. **炔烃亲核加成限于特定底物**——端炔、缺电子炔、乙炔等
6. **炔基负离子对羰基加成时，立体化学需考虑**——通常生成外消旋混合物

## 七、常见比较与易混点

### 7.1 炔烃 vs 烯烃——全面对比

| 反应 | **烯烃** | **炔烃** |
|:---|:---|:---|
| **+ Br₂** | 快，无需催化 | 慢，常需 Lewis 酸 |
| **+ HX** | 快 | 慢，Hg²⁺ 催化 |
| **+ H₂O（酸催化）** | 醇（马氏）| 酮/醛（烯醇互变）|
| **+ ROH（碱催化）** | 一般不反应 | 乙烯基醚（亲核加成）|
| **+ HCN** | 一般不反应 | 丙烯腈（亲核加成）|
| **硼氢化-氧化** | 反马氏醇 | 醛/酮 |
| **催化氢化** | 烷烃 | 可停在烯烃（Lindlar/毒化 Pd）|
| **KMnO₄ 氧化** | 二醇（稀冷）或断裂（热浓）| 羧酸（彻底断裂）|

### 7.2 Lindlar vs Na/NH₃——机理的本质差异

最容易混淆的概念：

- **Lindlar**：
  - 机理：表面催化氢化（H₂ 在 Pd 表面活化，syn 加成）
  - 立体：syn → Z
  - 关键：毒化阻止进一步还原

- **Na/NH₃**：
  - 机理：溶液中的单电子转移（自由基负离子 → 乙烯基负离子）
  - 立体：anti → E（负离子采取更稳定的反式构型）
  - 关键：不是催化氢化，是化学计量还原

### 7.3 端炔水合的产物辨析

| 条件 | 区域选择性 | 末端炔产物 | 内炔产物 |
|:---|:---:|:---|:---|
| **Hg²⁺/H₂SO₄** | 马氏 | 甲基酮 | 酮 |
| **硼氢化-氧化** | 反马氏 | **醛** | 酮 |

**端炔的两种水合路线产物不同**——这是竞赛常考的选择题。

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：苯乙炔（Ph-C≡CH）用 Lindlar 催化剂和 Na/NH₃(l) 分别还原，产物有何不同？

**解答**：
- **Lindlar**：syn 加成 → **顺式-β-甲基苯乙烯**（Z-1-苯基丙烯）
- **Na/NH₃**：anti 加成 → **反式-β-甲基苯乙烯**（E-1-苯基丙烯）

### 10.2 例题 2（★★ 水合产物）

**题目**：3-甲基-1-丁炔（(CH₃)₂CH-C≡CH）分别用 Hg²⁺/H₂SO₄ 和 (Sia)₂BH/氧化处理，产物是什么？

**解答**：
- **Hg²⁺/H₂SO₄（马氏）**：
  - OH 加到多取代碳（内部 C）
  - 烯醇 → 互变 → **4-甲基-2-戊酮**
- **(Sia)₂BH / H₂O₂（反马氏）**：
  - B 加到末端 C
  - 氧化 → 烯醇 → 互变 → **3-甲基丁醛**（异戊醛）

### 10.3 例题 3（★★ 亲核加成）

**题目**：为什么乙炔可以与乙醇在 KOH 催化下反应，而乙烯不能？

**解答**：
1. **炔烃 sp 碳电负性高**，使三键电子云更易被亲核试剂进攻
2. 亲核加成后形成 **烯基负离子**，其负电荷可被 sp 杂化碳较好地容纳
3. **烯烃 sp² 碳电负性较低**，亲核试剂进攻后形成的碳负离子不稳定
4. 此外，乙炔有一定酸性，KOH 可协助去质子形成炔基负离子，促进反应

### 10.4 例题 4（★★★ 综合推断）

**题目**：某化合物 A（C₆H₁₀）经 Lindlar 还原得 B，B 经臭氧化得 OHC-CHO 和 OHC-CH₂-CH₂-CHO。求 A 的结构。

**解答**：
- Lindlar 还原炔烃 → Z-烯烃
- B 臭氧化得 乙二醛 + 丁二醛
- 反推 B：将两个醛的羰基氧去掉，用双键连接
  - OHC-CHO → =CH-CH=
  - OHC-CH₂-CH₂-CHO → =CH-CH₂-CH₂-CH=
- B = Z-1,5-己二烯（但需验证双键位置）
- 更准确：B 的臭氧化产物是两分子醛，说明 B 有两个双键... 等一下，一个双键臭氧化得两分子单醛/酮
- OHC-CHO 是双醛（来自 =CH-CH=），说明 B 含 CH=CH-CH=CH 结构
- 重新分析：B 臭氧化得 OHC-CHO（来自一个双键）和 OHC-(CH₂)₂-CHO（来自另一个双键）
- 但一个双键臭氧化只应得两片段... 若 B 只有一个双键，不可能得四个片段
- 实际上：B 是环状烯烃？或 A 含两个三键？
- 再读题：A 分子式 C₆H₁₀，不饱和度 = (2×6+2−10)/2 = 2
- 两个不饱和度 → 可能是二炔、炔+烯、或环+炔
- Lindlar 还原只能还原炔 → B 含烯烃
- B 臭氧化得 OHC-CHO（乙二醛）和 OHC-(CH₂)₂-CHO（戊二醛？不对，4 个碳）
- OHC-CH₂-CH₂-CHO 是丁二醛（succinic dialdehyde，4 个碳）
- 乙二醛（2C）+ 丁二醛（4C）= 6C，匹配
- B 臭氧化得这两个，说明 B 有两个双键？不，一个双键只产生两个片段
- 等等：若 B 是 **环状二烯**？不，C₆H₁₀ 不饱和度 2...
- 若 B 是 **1,3-环己二烯**（C₆H₈，不对）
- 若 B 是 **1,5-己二烯**（C₆H₁₀，不饱和度 2，但两个双键）
  - 臭氧化得 2 HCHO + OHC-CH₂-CH₂-CHO → 三个片段，不匹配
- 重新考虑：OHC-CHO 来自 =CH-CH=（一个双键的两端都有 H）
- 若 B 只有一个双键：反推 B = CH₂=CH-CH₂-CH=CH-CH₃？臭氧化得 HCHO + OHC-CH₂-CH₂-CHO + CH₃CHO — 三片段
- 题目说只得两个产物... 那 B 可能是环状单烯？
  - 环己烯臭氧化得 OHC-(CH₂)₄-CHO（一个产物，己二醛）
  - 不匹配
- 也许 A 是 **1,5-己二炔**？Lindlar 还原得 Z,Z-1,5-己二烯
  - 臭氧化得 2 HCHO + OHC-(CH₂)₂-CHO → 三片段
- 或者 A 是 **1,3-己二炔**？C₆H₆，不对
- 此题可能数据有误或我理解有误。核心思路是：从臭氧化产物反推双键位置，再从烯烃反推炔烃。

## 十一、易错点

1. ❌ **认为炔烃亲电加成和烯烃一样快** → 炔烃更慢（烯基正离子不稳定）
2. ❌ **Lindlar 和 Na/NH₃ 只记产物不记机理** → 前者是催化氢化（syn），后者是单电子还原（anti）
3. ❌ **端炔水合产物死记"醛"** → Hg²⁺ 马氏水合得 **甲基酮**，硼氢化反马氏才得 **醛**
4. ❌ **忽略炔烃的亲核加成** → 烯烃一般不，炔烃可以（ROH、HCN、RSH）
5. ❌ **端炔酸性被低估** → pKa ~25，可被 NaNH₂ 去质子，用于碳链增长
6. ❌ **混淆端炔和内炔的硼氢化产物** → 末端炔 → 醛；内炔 → 酮
7. ❌ **认为炔基负离子只能做亲核试剂** → 也可发生氧化偶联（Glaser）
8. ❌ **炔烃氧化只记 KMnO₄** → 臭氧化得 α-二羰基，也是有用的结构推断工具

### 12.3 入门级例题

**题目**：写出 1-丁炔与下列试剂反应的产物：
(1) H₂ / Lindlar；(2) Na / NH₃(l)；(3) Hg²⁺/H₂SO₄, H₂O；(4) (Sia)₂BH / H₂O₂, NaOH

**预期解答路径**：
1. Lindlar：syn 还原 → **顺-1-丁烯**（Z-1-丁烯）
2. Na/NH₃：anti 还原 → **反-1-丁烯**（E-1-丁烯）
3. Hg²⁺/H₂SO₄：马氏水合 → 烯醇 → **2-丁酮**
4. 硼氢化-氧化：反马氏 → 烯醇 → **丁醛**

**老师可引导的提问**：
- 卡在第 3 步：问 "马氏规则在炔烃水合中如何体现？烯醇为什么要互变？"
- 卡在第 4 步：问 "为什么硼氢化端炔得醛，而不是像烯烃那样得伯醇？"

### 12.4 与现实/直觉的连接

- **类比**：炔烃就像一个"挑食的食客"——亲电试剂（酸）来时，它比烯烃"难伺候"（反应慢）；但亲核试剂（碱）来时，它又比烯烃"好说话"（可以反应）。这是因为 sp 杂化碳的"性格"介于 sp²（随和）和 sp³（顽固）之间。
- **反例**：直觉认为"还原剂都一样，只是强弱不同"。但 Lindlar 和 Na/NH₃ 还原炔烃的机理完全不同——一个是"表面贴膜"（催化氢化），一个是"溶液放电"（单电子转移），产物立体化学正好相反。

## 十三、竞赛拓展

1. **Glaser 偶联与 Eglinton 偶联**：
   - 端炔在 Cu⁺/O₂ 条件下氧化偶联 → 二炔
   - 是合成共轭二炔的重要方法

2. **炔烃与卡宾的反应**：
   - 炔烃 + 卡宾 → 环丙烯（小环似烯）
   - 环丙烯张力极大，可发生开环或重排

3. **Sonogashira 偶联**：
   - 端炔 + 芳基卤化物 / Pd-Cu 催化
   - 制备芳基炔的重要方法
   - 涉及炔基铜中间体

4. **炔烃的复分解反应（Alkyne Metathesis）**：
   - 高阶内容，决赛偶尔涉及
   - 炔烃在 Mo/W 催化剂下交换烷叉片段
