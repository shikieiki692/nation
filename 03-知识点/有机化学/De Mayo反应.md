# De Mayo 反应

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-有机化学]]
- 对应考纲条目：[[49-周环反应]]、[[50-有机合成]]

## 一、定义

**De Mayo 反应**：烯酮（enone）与烯烃在光照条件下发生的 **[2+2] 光环加成**反应，生成环丁烷衍生物，随后可经逆 Aldol 断裂生成 1,5-二羰基化合物的反应。

**本质**：光化学 [2+2] 环加成 + 后续逆 Aldol 断裂的串联反应。

## 二、考纲对应

- 对应考纲条目：[[49-周环反应]]、[[50-有机合成]]
- 所属模块：[[决赛要求-有机化学]]
- 本知识点在考纲中的作用：De Mayo 反应是竞赛全合成推断题中的高级切断策略，属于决赛难度内容。

## 三、核心原理

### 3.1 反应步骤

**第一步：[2+2] 光环加成**
- 烯酮（C=C-C=O）在光照下激发至激发态
- 与另一分子烯烃发生 [2+2] 环加成
- 生成环丁烷衍生物

**第二步：逆 Aldol 断裂**
- 环丁烷产物在碱性或加热条件下发生逆 Aldol 反应
- C-C 键断裂，生成 1,5-二羰基化合物

### 3.2 光化学 [2+2] 环加成的特殊性

**热禁阻 vs 光允许**：
- 热条件下：[2+2] 环加成是**禁阻**的（Woodward-Hoffmann 规则）
- 光条件下：一个组分处于激发态 → **允许**

**烯酮的优势**：
- 烯酮的 C=C 与 C=O 共轭 → 吸收 UV 光 → 易于激发
- 是光化学 [2+2] 环加成的理想底物

## 四、关键结论

1. **De Mayo 反应 = [2+2] 光环加成 + 逆 Aldol 断裂**
2. **光照是必需条件**——热条件下 [2+2] 环加成禁阻
3. **产物是 1,5-二羰基化合物**——可用于构建复杂环系
4. **逆合成切断**：1,5-二羰基 ← 环丁烷 ← 烯酮 + 烯烃
5. **竞赛中常作为全合成的关键步骤**——构建 5-6-8 稠环骨架

## 五、常见分类或情形

### 5.1 按烯酮类型分类

| 烯酮类型 | 反应性 | 产物特征 |
|:---|:---|:---|
| 环状烯酮 | 高 | 稠环环丁烷 |
| 链状烯酮 | 中 | 单环环丁烷 |
| α,β-不饱和醛 | 低 | 较少使用 |

### 5.2 ABOC Ch.11 中的应用

**Mutilin 全合成（FT0-Jack）**：
- De Mayo 反应是逆合成分析的核心切断
- 用于构建 5-6-8 稠合骨架中的关键环系
- 涉及烯酮的 [2+2] 光环加成 → 逆 Aldol 断裂策略

## 六、适用条件与限制

1. **必须光照**——热条件下不发生
2. **烯酮必须能吸收 UV**——共轭体系是必需的
3. **逆 Aldol 步骤需要碱性或加热条件**
4. **立体化学控制较难**——光化学反应常产生混合物
5. **与 Diels-Alder 竞争时**——D-A 在热条件下更快，De Mayo 需光照区分

## 七、常见比较与易混点

### 7.1 De Mayo vs 一般 [2+2] 环加成

| | **De Mayo** | **一般 [2+2]** |
|:---|:---|:---|
| 底物 | 烯酮 + 烯烃 | 任意两个烯烃 |
| 后续反应 | 逆 Aldol 断裂 | 无 |
| 最终产物 | 1,5-二羰基 | 环丁烷 |
| 合成价值 | 构建 1,5-二羰基 | 构建环丁烷 |

### 7.2 De Mayo vs Diels-Alder

| | **De Mayo** | **Diels-Alder** |
|:---|:---|:---|
| 条件 | 光照 | 加热 |
| 机理 | [2+2] 光环加成 | [4+2] 热环加成 |
| 产物 | 1,5-二羰基 | 环己烯 |
| 电子数 | 4e | 6e |
| WH 规则 | 光允许 | 热允许 |

## 十、例题

### 例题 1（★★ 基础）

**题目**：De Mayo 反应的最终产物是什么类型的化合物？

**解答**：**1,5-二羰基化合物**。反应经过 [2+2] 光环加成生成环丁烷，再经逆 Aldol 断裂得到 1,5-二羰基。

### 例题 2（★★★ 逆合成）

**题目**：某全合成目标分子含有 1,5-二羰基结构，如何逆合成切断？

**解答**：
1. 识别 1,5-二羰基关系
2. 逆 Aldol 切断 → 环丁烷中间体
3. [2+2] 逆环加成切断 → 烯酮 + 烯烃
4. 这就是 De Mayo 逆合成策略

## 十一、易错点

1. ❌ **混淆 De Mayo 与 Diels-Alder** → De Mayo 是光化学 [2+2]，D-A 是热 [4+2]
2. ❌ **忘记逆 Aldol 步骤** → De Mayo 的最终产物是 1,5-二羰基，不是环丁烷
3. ❌ **热条件下尝试 De Mayo** → [2+2] 热禁阻，必须光照
4. ❌ **忽略立体化学** → 光化学反应立体选择性较差

### 12.3 入门级例题

**题目**：为什么 De Mayo 反应需要光照？

**解答**：[2+2] 环加成在热条件下是 Woodward-Hoffmann 禁阻的。光照使烯酮激发至激发态，改变轨道对称性，使反应变为允许。

### 12.4 与现实/直觉的连接

- De Mayo 反应就像"光合作用中的能量捕获"——光能打破热力学禁阻，打开新的反应通道。
- 逆 Aldol 断裂就像"解开一个结"——环丁烷的紧张结构在碱性条件下松开，释放为更稳定的链状产物。
