## 一、定义

**立体化学（Stereochemistry）**：研究分子在三维空间中排列方式及其对物理、化学性质影响的化学分支。

在竞赛有机化学中，立体化学是**机理书写的"基础设施"**——几乎所有涉及 sp³ 碳形成的反应（加成、取代、重排）都必须分析立体化学后果。

> ⭐ **ABOC 行 2180**："立体化学分析应优先区分非对映异构体，再讨论其对映体。"

## 三、核心原理

### 3.1 立体化学分析的标准顺序

ABOC 强调的分析顺序（本章最核心的思维框架）：

```
第一步：区分非对映异构体（顺/反、内消旋等）
    ↓
第二步：讨论对映体关系（镜像、旋光方向）
    ↓
第三步：标记 R/S 构型（CIP 规则）
```

**为什么是这个顺序？**
- 非对映异构体的物理/化学性质**差异很大**（熔点、溶解度、反应速率都不同）
- 对映体在非手性环境中**性质几乎相同**（仅旋光方向相反）
- R/S 只是命名标签，不能替代对"顺/反关系"的直观理解

> 常见错误：学生一上来就标 R/S，却忽略了分子是 meso 还是手性。

### 3.2 手性的判断

**分子具有手性的充要条件**：不含 ** improper 旋转轴 Sₙ**。

**最实用的判断法**：
1. 找对称面 σ → 有则非手性
2. 找对称中心 i → 有则非手性
3. 找 S₄ 等旋转反映轴 → 有则非手性

**常见手性元素**：
| 手性元素 | 实例 | 特点 |
|:---|:---|:---|
| **手性中心** | 乳酸 C₂ | 最常见，sp³ 碳连 4 个不同基团 |
| **手性轴** | 联苯类（位阻大）| 单键旋转受阻 |
| **手性面** | 对环芳烷、螺烯 | 螺旋状结构 |

### 3.3 立体异构体的分类

```
构造异构（连接方式不同）
    └── 立体异构（连接方式相同，空间排列不同）
            ├── 构象异构（单键旋转，如乙烷交叉/重叠式）
            └── 构型异构（不能通过单键旋转互变）
                    ├── 顺反异构（非手性，如烯烃 Z/E）
                    └── 对映异构（手性）
                            ├── 对映体（互为镜像，等量混合 = 外消旋）
                            └── 非对映体（不互为镜像）
                                    ├── 差向异构体（仅一个手性中心构型不同）
                                    └── 内消旋体（meso，有手性中心但分子有对称面）
```

### 3.4 CIP 顺序规则（R/S 标记）

**Cahn-Ingold-Prelog 优先级规则**：

1. **直接相连原子的原子序数** → 大者优先
2. **距离为 2 的原子集合** → 先比最大原子序数；若相同再比次大
3. **同位素** → 质量数大者优先
4. **多重键展开为虚拟原子**：
   - C=C 视为 C 连有两个 C
   - C≡C 视为 C 连有三个 C
   - C=O 视为 C 连有两个 O

**R/S 判定**：
- 最低优先级基团朝向远离观察者
- 1→2→3 顺时针 = **R**（Rectus）
- 1→2→3 逆时针 = **S**（Sinister）

> **R/S 与 (+)/(-) 无关**：R 可以是左旋或右旋，必须通过实验测定。

### 3.5 Fischer 投影式

**规则**：
- 十字交叉点 = 手性碳
- **水平线** = 伸向观察者前方（楔形键）
- **竖直线** = 伸向观察者后方（虚线键）

**在 Fischer 投影式中判断 R/S**：
- 若最低优先级在**竖直线上**：直接按 1→2→3 方向判断（顺时针 = R）
- 若最低优先级在**水平线上**：判断结果取反（顺时针 = S）

**Fischer ↔ 楔形式转换**：
- 水平线转楔形（向前），竖直线转虚线（向后）
- 不可将 Fischer 投影式在纸面内旋转 90°（会改变构型），只可旋转 180°

### 3.6 差向异构体（Epimers）

定义：**仅一个手性中心构型不同**的非对映异构体。

- D-葡萄糖 与 D-甘露糖：C2 差向异构体
- D-葡萄糖 与 D-半乳糖：C4 差向异构体

差向异构体在糖类和甾体化学中极为常见。

### 3.7 外消旋体与内消旋体

| | **外消旋体（Racemic）** | **内消旋体（meso）** |
|:---|:---|:---|
| **组成** | 等量 R 和 S 对映体混合 | 单一化合物 |
| **旋光性** | 无旋光（互相抵消）| 无旋光（分子内对称面）|
| **手性中心** | 有 | 有（至少两个）|
| **整体手性** | 非手性（混合物）| 非手性（单化合物）|
| **拆分** | 可拆分（如用手性柱）| 不可拆分 |

**经典 meso 实例**：酒石酸的 meso 体（两个手性中心，分子有对称面）。

## 四、关键结论

1. **分析顺序 = 先非对映 → 再对映 → 最后 R/S**——这是 ABOC 反复强调的"正确思维顺序"
2. **手性判断先看对称面/对称中心**——不要只数手性中心（meso 体有手性中心但无手性）
3. **R/S 与 (+)/(-) 完全无关**——R 可以是左旋，S 也可以是右旋
4. **Fischer 投影式不可旋转 90°**——只能旋转 180°，否则构型反转
5. **CIP 规则中多重键要展开**——C=O 视为连两个 O，C≡C 视为连三个 C
6. **n 个手性中心最多 2ⁿ 个立体异构体**——meso 体会减少实际数目

## 五、常见分类或情形

### 5.1 按异构关系分类

| 关系 | 定义 | 性质差异 | 实例 |
|:---|:---|:---|:---|
| **对映体** | 互为镜像，不能重叠 | 旋光方向相反，其他相同 | (R)-乳酸 与 (S)-乳酸 |
| **非对映体** | 不互为镜像的立体异构体 | 物理/化学性质均不同 | D-葡萄糖 与 D-甘露糖 |
| **差向异构体** | 仅一个手性中心不同的非对映体 | — | D-葡萄糖 与 D-半乳糖 |
| **内消旋体** | 有手性中心但分子有对称面 | 无旋光 | meso-酒石酸 |
| **外消旋体** | 等量对映体混合物 | 无旋光，可拆分 | (±)-乳酸 |

### 5.2 按立体化学表示法分类

| 表示法 | 适用场景 | 特点 |
|:---|:---|:---|
| **楔形式（Wedge-Dash）** | 通用，尤其环状化合物 | 直观，但复杂分子难以表示 |
| **Fischer 投影式** | 糖类、氨基酸、多手性中心链 | 简洁，但转换规则严格 |
| **Newman 投影式** | 构象分析 | 看旋转构象 |
| **锯架式（Sawhorse）** | 构象/立体关系 | 介于楔形式和 Newman 之间 |

## 六、适用条件与限制

1. **CIP 规则仅适用于 sp³ 手性中心的基本判定**——复杂情况（如手性轴、手性面）需要扩展规则
2. **Fischer 投影式仅适用于链状多手性中心**——环状化合物用楔形式更清楚
3. **R/S 标记不能预测旋光方向**——(+)/(-) 必须通过实验测定
4. **meso 体判断不能只看手性中心数目**——必须检查分子整体对称性
5. **外消旋体 ≠ meso 体**——前者是混合物，后者是单一化合物

## 七、常见比较与易混点

### 7.1 对映体 vs 非对映体 vs 同一化合物

**判断流程**：
```
1. 构造是否相同？
   └─ 否 → 构造异构
   └─ 是 → 继续
2. 能否通过单键旋转重合？
   └─ 能 → 构象异构（通常视为同一化合物）
   └─ 否 → 继续
3. 互为镜像？
   └─ 是 → 对映体
   └─ 否 → 非对映体
```

### 7.2 Z/E 与 R/S 的区别

| | **Z/E** | **R/S** |
|:---|:---|:---|
| **适用对象** | 烯烃、双键（顺反异构）| 手性中心（对映异构）|
| **判断依据** | CIP 优先级比较双键两侧 | CIP 优先级 + 空间排列 |
| **与手性关系** | 非手性标记 | 手性标记 |
| **"Z = 顺"？** | 不一定（取决于取代基优先级）| — |

### 7.3 外消旋体 vs 内消旋体

最容易混淆的概念：
- **共同点**：都无旋光性
- **不同点**：
  - 外消旋体 = 混合物（R + S），可以拆分
  - 内消旋体 = 纯化合物，有对称面，不可拆分

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：判断酒石酸（HOOC-CHOH-CHOH-COOH）的立体异构体数目，并指出 meso 体。

**解答**：
- 两个相同的手性中心（C2 和 C3 都连有 -H, -OH, -COOH, -CHOHCOOH）
- 最大数目 2² = 4 种
- 但实际只有 **3 种**：(2R,3R)、(2S,3S)、meso-(2R,3S)
- meso 体：分子有对称面，两个手性中心构型相反但整体非手性

### 10.2 例题 2（★★ R/S 判断）

**题目**：用 CIP 规则判断下列化合物的 R/S：
```
    COOH
     |
H₃C-C*-CH₂OH
     |
     NH₂
```

**解答**：
- 直接相连原子：C(6), C(6), N(7), H(1)
- N 优先级最高（原子序数 7）
- 两个 C 相同，继续比较下一层：
  - -COOH 的 C 连 (O,O,O) — 实际展开 C=O 为连两个 O
  - -CH₂OH 的 C 连 (O,H,H)
  - (O,O,O) > (O,H,H)，所以 -COOH > -CH₂OH
- 优先级顺序：-NH₂ > -COOH > -CH₂OH > -H
- 若 -H 朝向远离观察者，1→2→3 顺时针 → **R**

### 10.3 例题 3（★★ Fischer 投影式）

**题目**：将下列 Fischer 投影式转换为楔形式，并判断 R/S。
```
   CHO
    |
H-C-OH
    |
HO-C-H
    |
   CH₂OH
```
（D-甘油醛）

**解答**：
- Fischer 中竖直线下端 -CH₂OH 优先级最低（C 连 H,H,O）
- 最低优先级在竖直线上 → 直接判断
- 1(-OH) → 2(-CHO) → 3(-CH₂OH) 顺时针 → **R**
- 楔形式：-CHO 和 -CH₂OH 用虚线向后，-H 和 -OH 用楔形向前

### 10.4 例题 4（★★★ 反应立体化学）

**题目**：顺-2-丁烯与 Br₂ 反应，产物的立体关系是什么？

**解答**：
- 形成溴鎓离子后，Br⁻ 从背面进攻
- 两个 Br 从双键两侧加入 → **反式加成**
- 顺-2-丁烯反式加成得到 **外消旋体**（一对对映体）
- 反-2-丁烯反式加成得到 **内消旋体**

> 关键：先判断非对映关系（顺/反产物），再讨论对映关系（R/S 组合）。

## 十一、易错点

1. ❌ **一上来就标 R/S，不判断 meso/非对映关系** → 正确顺序：先非对映 → 再对映 → 最后 R/S
2. ❌ **认为有手性中心就一定有手性** → meso 体有手性中心但无手性（有对称面）
3. ❌ **混淆外消旋体与内消旋体** → 前者是混合物，后者是纯化合物
4. ❌ **Fischer 投影式旋转 90°** → 只能旋转 180°，90° 会改变构型
5. ❌ **CIP 规则不展开多重键** → C=C 视为连两个 C，C=O 视为连两个 O
6. ❌ **R/S 与 (+)/(-) 混为一谈** → R 可以是左旋，S 也可以是右旋
7. ❌ **忽略虚线/楔形键的方向** → 画三维结构时必须标明，否则立体信息丢失
8. ❌ **差向异构体 = 对映体** → 差向异构体是非对映体的一种（仅一个中心不同）

### 12.3 入门级例题

**题目**：判断 2,3-二溴丁烷的所有立体异构体，指出 meso 体，并标记各手性中心的 R/S。

**预期解答路径**：
1. 两个相同的手性中心 → 最多 4 种，实际 3 种（含 meso）
2. (2R,3R) 与 (2S,3S) 互为对映体
3. meso-(2R,3S) 有对称面，非手性
4. meso 与 (2R,3R) 互为非对映体

**老师可引导的提问**：
- 卡在第 1 步：问 "两个手性中心相同意味着什么？"
- 卡在第 3 步：问 "meso 体为什么无旋光？画出对称面。"

### 12.4 与现实/直觉的连接

- **类比**：立体化学就像"左手和右手"——对映体如同左右手，互为镜像但不能重叠。非对映体则像"左手和左脚"——既不互为镜像，也不相同。
- **类比**：meso 体就像"一个左撇子戴上了右手手套"——局部有手性，但整体对称。
- **反例**：直觉认为"无旋光 = 无手性"，但外消旋体也无旋光却是手性分子的混合物。

## 十三、竞赛拓展

1. **Grossman 规则（碳标号法）**：
   - ABOC 在本章引入碳标号法追踪反应前后骨架对应
   - 尤其在鎓离子开环、重排反应中，碳标号可避免"少原子"或"骨架画错"
   - 规则：在机理书写中给每个碳编号，确保产物中每个碳都能追溯到反应物

2. **手性轴与手性面**：
   - 联苯类（6,6'-位大取代基）因单键旋转受阻产生手性轴
   - 螺烯（helicene）因螺旋结构产生手性面
   - 这些属于"不含手性中心的手性分子"，初赛较少涉及但决赛会考

3. **动态立体化学**：
   - 构象异构与构型异构的边界：某些环状体系中，构象翻转可能导致构型"表观"变化
   - 翻转环（ring flip）对取代基取向的影响（直立/平伏键互换）
