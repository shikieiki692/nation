# 马氏规则（Markovnikov's Rule）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[37-加成反应]]

---

## 一、定义

**马氏规则**：烯烃与卤化氢（HX）等不对称亲电试剂发生加成时，氢原子（或正电部分）优先加到**含氢较多的双键碳**上，卤素（或负电部分）加到**含氢较少的双键碳**上。

**经典表述**："-rich gets richer"（富者愈富）——氢上加氢。

**现代本质**：**碳正离子稳定性决定加成方向**。质子优先进攻能生成更稳定碳正离子的那个双键碳。

> ⭐ **ABOC 行 1809 核心结论**："马氏规则：本质是碳正离子稳定性决定加成方向。优先产生稳定的正离子！"

## 三、核心原理

### 3.1 机理：碳正离子中间体

烯烃加 HX 经历**两步机理**：

1. **质子化**（决速步）：H⁺ 进攻双键 → 生成碳正离子
2. **亲核捕获**：X⁻ 进攻碳正离子 → 得卤代烷

```
R-CH=CH₂ + H⁺ → [R-ĊH-CH₃]  ← 更稳定的二级碳正离子
                     ↓
                   X⁻
                     ↓
                 R-CHX-CH₃  ← 马氏产物
```

### 3.2 为什么是碳正离子稳定性决定方向？

**过渡态理论解释**（ABOC 行 1807）：
- 生成稳定碳正离子的**活化能更低**
- 烷基等给电子基团**稳定了质子加成的过渡态**
- 过渡态已带有部分正电荷 → 越能稳定正电荷的取代模式，过渡态能量越低

| 烯烃 | 质子化位点 | 生成碳正离子 | 稳定性 | 主产物 |
|:---|:---|:---|:---:|:---|
| CH₂=CH-CH₃ | C1 | CH₃-ĊH-CH₃（二级）| 较稳 | 2-卤丙烷 |
| CH₂=CH-CH₃ | C2 | ⁺CH₂-CH₂-CH₃（一级）| 不稳 | 少 |
| (CH₃)₂C=CH₂ | C1 | (CH₃)₂Ċ-CH₃（三级）| 最稳 | 叔卤代烷 |

### 3.3 给电子/吸电子基团的影响

| 取代基类型 | 例子 | 对马氏规则的影响 |
|:---|:---|:---|
| **烷基（给电子）** | -CH₃, -CH₂CH₃ | 遵循马氏，且更偏向多取代端 |
| **共轭给电子** | -OCH₃, -NR₂ | 遵循马氏，碳正离子被共振稳定 |
| **吸电子基** | -CF₃, -COOR, -CN | **反马氏倾向**（或降低反应性）|
| **卤素** | -Cl, -Br | 诱导吸电子使马氏倾向减弱，但共轭给电子使其仍遵循马氏 |

**例**：CF₃CH=CHCl 与 HI 反应
- CF₃ 强吸电子 → 使相邻碳正离子极不稳定
- 质子优先进攻远离 CF₃ 的一端

## 四、关键结论

1. **马氏规则的本质 = 碳正离子稳定性**——不是经验规律，而是有物理有机基础
2. **过渡态已带部分正电荷**——所以碳正离子的稳定因素在 TS 中就开始发挥作用
3. **吸电子基团可能导致"反马氏"行为**——电子效应压倒结构效应
4. **碳正离子中间体可重排**——马氏规则预测的是第一步，实际产物可能经 Wagner-Meerwein 重排（见 [[1,2-迁移与重排]]）
5. **烯烃加水制醇时易重排**——因为是 SN1 型碳正离子机理（ABOC 行 1916）

## 五、常见分类或情形

### 5.1 遵循马氏规则的典型反应

| 反应 | 试剂 | 中间体 | 马氏产物 |
|:---|:---|:---|:---|
| 烯烃 + HX | HCl, HBr, HI | 碳正离子 | 卤代烷（X 在少氢碳）|
| 烯烃 + H₂O/H⁺ | 酸催化水合 | 碳正离子 | 醇（OH 在少氢碳）|
| 烯烃 + HOX | 次卤酸 | 卤鎓离子 → 开环 | 卤醇（X 在少氢碳）|

### 5.2 反马氏规则的经典情况

| 反应 | 本质 | 与马氏规则的关系 |
|:---|:---|:---|
| **烯烃 + HBr/ROOR** | 自由基加成 | 溴自由基加成 → 反马氏（见 [[过氧化物效应]]）|
| **烯烃 + BH₃** | 硼氢化 | B 为正电性，H 加到少氢碳 → 反马氏 |
| **烯烃 + HBr（吸电子烯烃）**| 电子效应 | CF₃ 等强吸电子基导致反向质子化 |

## 六、适用条件与限制

1. **仅适用于经历碳正离子中间体的加成**——协同反应（如硼氢化的四中心过渡态）不遵循
2. **对称烯烃不适用**——如乙烯加 HBr 无选择性
3. **双键两端取代度相同时需比较其他效应**——如共轭稳定、诱导效应
4. **碳正离子可能重排**——马氏规则预测的是最初质子化位点，不是最终产物结构
5. **不适用于自由基加成**——自由基稳定性决定的是过氧化物效应（反马氏）

---

## 七、常见比较与易混点

### 7.1 马氏 vs 反马氏：一张表理清

| | **马氏加成** | **反马氏加成** |
|:---|:---|:---|
| **机理** | 离子型（碳正离子）| 自由基或特殊电子效应 |
| **典型试剂** | HX, H₂O/H⁺ | HBr/ROOR, BH₃ |
| **控制因素** | 碳正离子稳定性 | 自由基稳定性 / B 的电正性 |
| **H 加到哪** | 多氢碳 | 少氢碳 |
| **X/OH 加到哪** | 少氢碳（多取代）| 多氢碳（少取代）|

### 7.2 马氏规则与碳正离子重排

- 马氏规则决定的是**第一步质子化的位点**
- 若生成的碳正离子不稳定，可能发生 [[1,2-迁移与重排|Wagner-Meerwein 重排]]
- 例：(CH₃)₂CH-CH=CH₂ + HBr → 先质子化得二级碳正 → 可能重排为三级碳正 → 产物在更取代位

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：写出丙烯与 HBr 反应的主要产物。

**解答**：
- 质子优先进攻末端 CH₂ → 生成二级碳正离子 CH₃-ĊH-CH₃
- Br⁻ 进攻 → **2-溴丙烷**（马氏产物）
- 不是 1-溴丙烷（那需要生成不稳定的一级碳正离子）

### 10.2 例题 2（★★ 吸电子基影响）

**题目**：CF₃CH=CH₂ 与 HCl 反应，主产物是什么？

**解答**：
- CF₃ 强吸电子（-I 效应）
- 若质子进攻 CF₃ 所在碳 → 生成 CF₃-ĊH-CH₃（正电荷紧邻 CF₃，极不稳定）
- 若质子进攻末端 CH₂ → 生成 CF₃-CH₂-ĊH₂（正电荷远离 CF₃）
- 主产物：**CF₃-CH₂-CH₂Cl**（形式上"反马氏"，但本质是电子效应决定）

### 10.3 例题 3（★★★ 重排陷阱）

**题目**：3,3-二甲基-1-丁烯与 HBr 反应，写出主产物。

**解答**：
- 第一步：质子进攻末端 → 二级碳正离子 (CH₃)₃C-CH₂-ĊH₂
- 该碳正离子极不稳定（邻位叔碳上的甲基可迁移）
- [1,2]-甲基迁移 → 三级碳正离子 (CH₃)₂Ċ-CH(CH₃)₂
- 最终产物：**2-溴-2,3-二甲基丁烷**
- 若不考虑重排会错答为 2-溴-3,3-二甲基丁烷

### 10.4 例题 4（★★★ 综合判断）

**题目**：以下反应的主产物分别是什么？

a) (CH₃)₂C=CHCH₃ + HBr（CCl₄, 0°C）
b) CH₃CH=CH₂ + HBr（过氧化物存在）
c) CH₂=CH-COOCH₃ + HCl

**解答**：
- a) 遵循马氏，生成三级碳正 → **2-溴-2-甲基丁烷**
- b) 过氧化物 → 自由基加成 → 反马氏 → **1-溴丙烷**
- c) -COOCH₃ 吸电子，但质子仍优先进攻末端 → 马氏产物为主，但反应性降低

## 十一、易错点

1. ❌ **把马氏规则当经验口诀死记** → 本质是碳正离子稳定性，理解了才能处理例外
2. ❌ **忽略碳正离子重排** → 马氏规则只预测第一步，产物可能重排
3. ❌ **认为所有 HX 都遵循马氏** → HBr/ROOR 是反马氏（自由基机理）
4. ❌ **吸电子基团存在时仍机械套马氏** → CF₃, CN, COOR 等可能反转
5. ❌ **混淆马氏规则与区域选择性** → 马氏是区域选择性的一个特例
6. ❌ **认为反马氏就是"错误"的** → 反马氏有其独立的机理基础（自由基/电子效应）
7. ❌ **忽略卤素的特殊效应** → 卤乙烯（CH₂=CHCl）仍遵循马氏，因为 Cl 的 +C 效应

### 12.3 入门级例题

**题目**：异丁烯 (CH₂=C(CH₃)₂) 与 HBr 反应，主产物是什么？用碳正离子稳定性解释。

**预期解答路径**：
1. 异丁烯双键两端：C1（CH₂，2H），C2（C(CH₃)₂，0H）
2. H⁺ 进攻 C1 → 生成三级碳正离子 (CH₃)₃C⁺（最稳定）
3. H⁺ 进攻 C2 → 生成一级碳正离子 ⁺CH₂-C(CH₃)₂H（极不稳定）
4. 三级碳正远稳定于一级 → 选择路径 2
5. Br⁻ 进攻三级碳正 → **叔丁基溴**（2-溴-2-甲基丙烷）

**老师可引导的提问**：
- 卡在第 2 步：问 "哪个碳正离子取代度更高？"
- 卡在第 4 步：问 "三级和一级碳正离子稳定性差多少数量级？"

### 12.4 与现实/直觉的连接

- **类比**：马氏规则就像"水往低处流"——质子（H⁺）会"滚向"能让它最舒服的位置（生成最稳定的碳正离子）。碳正离子的稳定性就是"地势高低"。
- **反例**：硼氢化反应是"反马氏"的，但它不是"逆流而上"——而是换了一套规则（B 带正电，H 带负电，所以 H 反而加到少氢端）。这说明同一个现象可以用不同的"力场"解释。
