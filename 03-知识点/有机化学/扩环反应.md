# 扩环反应

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-有机化学]]
- 对应考纲条目：[[39-重排反应]]、[[50-有机合成]]

## 一、定义

**扩环反应（Ring Expansion）**：通过重排反应将较小环系（通常是四元环或五元环）转变为较大环系的反应。

**本质**：利用环张力释放或碳正离子稳定性作为驱动力，通过 1,2-迁移实现环的扩大。

## 二、考纲对应

- 对应考纲条目：[[39-重排反应]]、[[50-有机合成]]
- 所属模块：[[决赛要求-有机化学]]
- 本知识点在考纲中的作用：扩环反应是构建中环和大环的重要策略，在天然产物全合成中有重要应用。

## 三、核心原理

### 3.1 扩环的驱动力

**环张力释放**：
- 四元环（~110 kJ/mol 张力）→ 五元环（~25 kJ/mol 张力）
- 五元环 → 六元环（几乎无张力）
- 张力释放提供热力学驱动力

**碳正离子稳定性**：
- 与阳离子相邻的环烷基 → 重排后生成更稳定的碳正离子

### 3.2 常见扩环机理

#### Wagner-Meerwein 型扩环

**四元环 → 五元环**（Clayden Ch.36）：
- 与阳离子相邻的四元环
- 环上 C-C 键迁移 → 五元环
- 释放环张力
- 天然产物 α-石竹烯醇的合成应用

**频哪醇型扩环**：
- 1,2-二醇在环上 → 一个 OH 质子化离去 → 碳正离子
- 邻位 C-C 键迁移 → 环扩大
- 氧孤对电子稳定正电荷 → 酮

#### 半频哪醇型扩环

**Corey 长叶烯合成**（Clayden Ch.36）：
- 仲醇在叔醇存在下被对甲苯磺酰化
- 半频哪醇重排 → 扩环酮

**Tiffeneau-Demjanov 扩环**：
- 2-氨基醇 → 重氮化 → 烷基重氮盐
- N₂ 离去 → 碳正离子 → 半频哪醇重排 → 环扩大

#### 卡宾/重氮甲烷扩环

**重氮甲烷扩环**（Clayden Ch.36）：
- 环己酮 + CH₂N₂ → 环庚酮
- 机理：重氮甲烷加成 → 环氧中间体 → 重排
- 注意：环戊酮 → 环己酮会混乱（副反应多）

### 3.3 环扩大的选择性

**扩环 vs 缩环**：
- 扩环：四元环 → 五元环，五元环 → 六元环（有利）
- 缩环：Favorskii 重排（α-卤代酮 → 环丙酮中间体 → 缩环）

**扩环的限度**：
- 六元环 → 七元环：可行，但驱动力减小
- 七元环以上：扩环困难，通常采用其他策略（如 RCM）

## 四、关键结论

1. **四元环 → 五元环是最容易发生的扩环**——张力释放最大
2. **五元环 → 六元环也很常见**——六元环最稳定
3. **W-M 重排是扩环的核心机理**——1,2-烷基迁移
4. **频哪醇/半频哪醇重排是扩环的重要工具**
5. **重氮甲烷可用于酮的扩环**——但小环酮（如环戊酮）副反应多

## 五、常见分类或情形

### 5.1 按机理分类

| 机理 | 底物 | 扩环方式 | 例子 |
|:---|:---|:---|:---|
| **W-M 扩环** | 环烷基碳正离子 | 1,2-烷基迁移 | 四元环 → 五元环 |
| **频哪醇扩环** | 环状 1,2-二醇 | 半频哪醇重排 | 五元环 → 六元环酮 |
| **半频哪醇扩环** | 环状氨基醇/碘代醇 | N₂/I 离去 → 重排 | 长叶烯合成 |
| **Tiffeneau-Demjanov** | 2-氨基醇 | 重氮化 → 重排 | 环酮扩环 |
| **重氮甲烷扩环** | 环酮 | CH₂N₂ 加成 → 重排 | 环己酮 → 环庚酮 |
| **卡宾扩环** | 环烯烃 | 卡宾插入 C-C 键 | [待补充] |

### 5.2 按环大小变化分类

| 变化 | 可行性 | 驱动力 |
|:---|:---:|:---|
| 三元环 → 四元环 | 较少见 | 张力仍大 |
| 四元环 → 五元环 | ★★★★★ | 张力释放大 |
| 五元环 → 六元环 | ★★★★ | 六元环最稳定 |
| 六元环 → 七元环 | ★★★ | 驱动力减小 |
| 七元环 → 更大 | ★★ | 困难 |

## 六、适用条件与限制

1. **需要有合适的离去基或碳正离子**——引发扩环的活性中心
2. **扩环后环张力必须降低**——否则不利
3. **迁移基团必须在环上**——1,2-迁移要求相邻
4. **与消除反应竞争**——强碱条件下消除可能更快
5. **中环（8-11 元）合成困难**——扩环不是理想策略，常用 RCM

## 七、常见比较与易混点

### 7.1 扩环 vs 缩环

| | **扩环** | **缩环** |
|:---|:---|:---|
| 驱动力 | 环张力释放 | 通常不是张力（Favorskii 是形成羧酸酯）|
| 常见方法 | W-M、频哪醇、重氮甲烷 | Favorskii、Benzilic |
| 环大小变化 | 小 → 大 | 大 → 小 |
| 例子 | 四元环 → 五元环 | 环己酮 → 环戊基甲酸酯 |

### 7.2 扩环 vs 环加成

| | **扩环** | **环加成** |
|:---|:---|:---|
| 机理 | 重排（无原子增减）| 两个分子结合 |
| 环数变化 | 环大小改变 | 环数增加 |
| 例子 | 四元环 → 五元环 | D-A：[4+2] → 六元环 |

## 十、例题

### 例题 1（★★ 基础）

**题目**：与碳正离子相邻的四元环为什么容易发生扩环？

**解答**：
1. 四元环有约 110 kJ/mol 的环张力
2. 扩环为五元环后，张力降至约 25 kJ/mol
3. 张力释放提供热力学驱动力
4. 同时，迁移后可能生成更稳定的碳正离子

### 例题 2（★★★ 应用）

**题目**：如何用重氮甲烷将环己酮扩环为环庚酮？

**解答**：
1. 环己酮 + CH₂N₂ → 重氮甲烷对羰基加成
2. 形成环氧中间体
3. 重排 → 环庚酮
4. 注意：环戊酮 → 环己酮用此方法副反应多

## 十一、易错点

1. ❌ **认为所有环都可以扩环** → 需要合适的引发剂和驱动力
2. ❌ **混淆扩环和缩环** → 扩环是小 → 大，缩环是大 → 小
3. ❌ **忽略环张力因素** → 扩环的驱动力主要来自张力释放
4. ❌ **认为重氮甲烷对所有环酮都适用** → 环戊酮等小环酮副反应多

## 十二、🎯 教学视角

### 12.1 学习路径

- **前置**：[[环张力]]、[[Wagner-Meerwein重排]]、[[频哪醇重排]]

### 12.2 学生易踩的认知误区

| 误区 | 正确认识 |
|---|---|
| "扩环只增加一个碳" | 取决于机理，重氮甲烷增加一个碳，W-M 不增减原子 |
| "六元环最容易扩环" | 六元环最稳定，扩环驱动力小；四/五元环更容易 |
| "扩环和开环一样" | 扩环保留环状结构，开环变成链状 |

### 12.3 入门级例题

**题目**：为什么频哪醇重排可以实现扩环？

**解答**：
1. 环状 1,2-二醇中，一个 OH 质子化离去 → 碳正离子
2. 环上相邻 C-C 键迁移 → 环扩大
3. 氧孤对电子稳定正电荷 → 形成酮
4. 结果是环扩大并生成羰基

### 12.4 与现实/直觉的连接

- 扩环就像"解开太紧的橡皮筋"——小环（四元环）张力大，像绷得太紧的橡皮筋，稍微一拉（重排）就变成更松的环（五元环）。
- 频哪醇扩环就像"魔术变环"——一个环变成更大的环，同时还变出一个羰基。
