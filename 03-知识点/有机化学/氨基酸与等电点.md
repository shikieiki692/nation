# 氨基酸与等电点（Amino Acids and Isoelectric Point）

## 一、定义

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[54-氨基酸]]

**氨基酸（Amino Acid）**是同时含有氨基（—NH₂）和羧基（—COOH）的有机化合物，是构成蛋白质的基本单元。**等电点（Isoelectric Point, pI）**是氨基酸（或蛋白质）在溶液中净电荷为零时的 pH 值。

- **α-氨基酸**：氨基和羧基连接在同一碳原子上（通式 H₂N-CHR-COOH），是蛋白质的构建单元
- **两性离子（Zwitterion）**：在同一分子内同时带有正负电荷的形式，如 ⁺H₃N-CHR-COO⁻
- **等电点的本质**：pH = pI 时，分子在电场中不移动；偏离 pI 时，带正电（pH < pI）或负电（pH > pI）

## 三、核心原理

### 3.1 氨基酸的两性离子性质

氨基酸在水溶液中主要以**两性离子**形式存在，而非游离的 —NH₂ 和 —COOH：

$$
\ce{H2N-CHR-COOH <=> ^+H3N-CHR-COO^-}
$$

**原因**：
- 羧基的酸性（pKa₁ ~ 2）强于氨基的共轭酸酸性（pKa₂ ~ 9-10）
- 质子从羧基转移到氨基是热力学有利的
- 两性离子形式因离子-偶极相互作用而稳定

**pH 对氨基酸形态的影响**：

| pH 范围 | 主要存在形式 | 净电荷 |
|:---|:---|:---:|
| pH ≪ pI | H₂N⁺-CHR-COOH（完全质子化）| **+1** |
| pH = pKa₁ | H₂N⁺-CHR-COOH ⇌ H₂N⁺-CHR-COO⁻（各 50%）| +0.5 |
| pH = pI | H₂N⁺-CHR-COO⁻（两性离子）| **0** |
| pH = pKa₂ | H₂N⁺-CHR-COO⁻ ⇌ H₂N-CHR-COO⁻（各 50%）| −0.5 |
| pH ≫ pI | H₂N-CHR-COO⁻（完全去质子化）| **−1** |

### 3.2 等电点的计算

**中性氨基酸**（1 个 —NH₂, 1 个 —COOH，侧链无电离基团）：

$$
\mathrm{pI} = \frac{\mathrm{p}K_{a1} + \mathrm{p}K_{a2}}{2}
$$

**酸性氨基酸**（侧链含额外 —COOH，如 Asp, Glu）：
- 侧链羧基的 pKa 约为 3.9-4.3
- pI 落在两个羧基 pKa 之间：

$$
\mathrm{pI} = \frac{\mathrm{p}K_{a1}(\alpha\text{-COOH}) + \mathrm{p}K_{aR}(\text{侧链 COOH})}{2}
$$

**碱性氨基酸**（侧链含额外碱性基团，如 Lys, Arg, His）：
- 侧链氨基/胍基/咪唑的 pKa 约为 6.0-12.5
- pI 落在两个氨基 pKa 之间：

$$
\mathrm{pI} = \frac{\mathrm{p}K_{aR}(\text{侧链碱}) + \mathrm{p}K_{a2}(\alpha\text{-NH}_3^+)}{2}
$$

### 3.3 20 种标准氨基酸分类

| 类别 | 氨基酸 | 侧链特点 | 性质 |
|:---|:---|:---|:---|
| **非极性脂肪族** | Gly, Ala, Val, Leu, Ile, Met, Pro | 烷基、硫醚、环状 | 疏水 |
| **芳香族** | Phe, Tyr, Trp | 苯环、酚羟基、吲哚 | 疏水/弱极性 |
| **极性不带电荷** | Ser, Thr, Cys, Asn, Gln | —OH, —SH, 酰胺 | 亲水 |
| **酸性（带负电）** | Asp, Glu | 额外 —COOH | 亲水、负电 |
| **碱性（带正电）** | Lys, Arg, His | 额外 —NH₂, 胍基, 咪唑 | 亲水、正电 |

**需记忆的缩写**：Asp (D), Glu (E), Lys (K), Arg (R), His (H), Cys (C), Ser (S), Thr (T), Asn (N), Gln (Q), Tyr (Y), Trp (W), Phe (F), Gly (G), Ala (A), Val (V), Leu (L), Ile (I), Met (M), Pro (P)

## 四、关键结论

1. **氨基酸在生理 pH（~7.4）下主要以两性离子存在**：因为 pKa₁ < 7 < pKa₂
2. **中性氨基酸的 pI ≈ 5.5-6.0**：在 pKa₁ 和 pKa₂ 之间
3. **酸性氨基酸的 pI < 4**：两个羧基的平均，偏酸性
4. **碱性氨基酸的 pI > 7.5**：两个氨基的平均，偏碱性
5. **等电点是氨基酸分离（电泳、离子交换）的理论基础**：pH = pI 时溶解度最低，可等电沉淀

### 5.2 按侧链反应性分类

| 反应类型 | 涉及氨基酸 | 应用 |
|:---|:---|:---|
| **二硫键形成** | Cys (-SH) | 蛋白质三级结构稳定 |
| **磷酸化** | Ser, Thr, Tyr (-OH) | 信号转导 |
| **糖基化** | Asn, Ser, Thr | 蛋白质修饰 |
| **酰化** | Lys (ε-NH₂) | 组蛋白修饰 |
| **芳香亲电取代** | Tyr, Trp | 碘化等标记 |

### 5.3 氨基酸的合成方法

| 方法 | 反应物 | 产物特点 | 消旋控制 |
|:---|:---|:---|:---:|
| **Strecker 合成** | 醛 + NH₃ + HCN → 氨基腈 → 水解 | 最经典，但产物为外消旋 | ❌ 外消旋 |
| **Gabriel 合成** | 邻苯二甲酰亚胺 + α-卤代酸酯 → 肼解 | 可得光学纯 | ✅ 可拆分 |
| **丙二酸酯法** | 酰氨基丙二酸酯 + 卤代烃 → 脱羧 | 通用 | ❌ 外消旋 |
| **酶催化拆分** | 外消旋氨基酸衍生物 + 酰化酶 | 一种对映体被水解 | ✅ 高 ee |
| **不对称合成** | 手性辅助基或催化剂 | 直接得单一对映体 | ✅ 高 ee |

### 6.1 等电点测定与计算

| 方法 | 原理 | 适用 |
|:---|:---|:---|
| **电泳法** | pH = pI 时不移动 | 定性判断 |
| **离子交换层析** | pH = pI 时结合最弱 | 分离纯化 |
| **等电聚焦** | 在 pH 梯度电场中聚焦到 pI | 精确测定 |
| **溶解度法** | pI 时溶解度最低 | 粗略估计 |

### 6.2 电泳分离条件

| 氨基酸类型 | pI 范围 | 电泳缓冲液 pH = 6.0 时的迁移方向 |
|:---|:---:|:---|
| **酸性氨基酸**（Asp, Glu）| 2.8-3.2 | **向阳极**（负电荷）|
| **中性氨基酸** | 5.0-6.0 | 基本不移动或微向阳极 |
| **碱性氨基酸**（Lys, Arg, His）| 7.6-10.8 | **向阴极**（正电荷）|

## 七、常见比较

### 7.1 三种类型氨基酸的比较

| 维度 | 中性氨基酸 | 酸性氨基酸 | 碱性氨基酸 |
|:---|:---|:---|:---|
| **代表** | Gly, Ala, Val… | Asp, Glu | Lys, Arg, His |
| **可电离基团** | α-COOH, α-NH₂ | + 侧链 COOH | + 侧链碱基 |
| **pKa 值** | pKa₁~2, pKa₂~9-10 | pKa₁~2, pKaᵣ~4, pKa₂~9 | pKa₁~2, pKa₂~9, pKaᵣ~6-12 |
| **pI 计算公式** | (pKa₁+pKa₂)/2 | (pKa₁+pKaᵣ)/2 | (pKaᵣ+pKa₂)/2 |
| **pI 数值** | ~5.5-6.0 | ~2.8-3.2 | ~7.6-10.8 |
| **生理 pH 电荷** | 两性离子为主 | 负电荷 | 正电荷 |

### 7.2 D/L 与 R/S 的关系

| 氨基酸 | D/L 构型 | R/S 构型 |
|:---|:---:|:---:|
| **绝大多数 L-氨基酸** | L | **S** |
| **L-半胱氨酸** | L | **R**（因侧链 —CH₂SH 的 CIP 优先级高于羧基）|
| **甘氨酸** | — | —（无手性中心）|

**注意**：D/L 命名基于 Fischer 投影与甘油醛的关系；R/S 基于 CIP 规则。二者**不是一一对应**的。

## 十、例题

### 例题 1：等电点计算与电泳行为

**题目**：某氨基酸的 pKa 值如下：pKa₁(α-COOH) = 2.34，pKa₂(α-NH₃⁺) = 9.60。计算其 pI，并判断在 pH = 6.0 的电场中该氨基酸向哪个电极移动。

**解析**：

1. **判断类型**：只有 α-COOH 和 α-NH₃⁺ 两个可电离基团 → **中性氨基酸**

2. **计算 pI**：

$$
   \mathrm{pI} = \frac{2.34 + 9.60}{2} = 5.97
   $$

3. **电泳方向判断**：
   - pH = 6.0 > pI = 5.97
   - 溶液 pH 略高于 pI，氨基去质子化倾向稍强
   - 净电荷为**微弱负电荷**
   - **向阳极（正极）移动**

### 例题 2：酸性氨基酸的 pI 计算

**题目**：天冬氨酸（Asp）的 pKa 值为：pKa₁(α-COOH) = 1.88，pKaᵣ(β-COOH) = 3.65，pKa₂(α-NH₃⁺) = 9.60。计算其 pI。

**解析**：

1. **判断类型**：含额外侧链羧基 → **酸性氨基酸**

2. **pI 计算**：取两个羧基 pKa 的平均值

$$
   \mathrm{pI} = \frac{1.88 + 3.65}{2} = 2.77
   $$

3. **验证**：
   - pI ≈ 2.77，远小于 7 → 酸性氨基酸的 pI 偏酸性，符合规律
   - 在生理 pH 7.4 时，Asp 带明显负电荷

### 例题 3：碱性氨基酸的 pI 计算

**题目**：赖氨酸（Lys）的 pKa 值为：pKa₁(α-COOH) = 2.18，pKa₂(α-NH₃⁺) = 8.95，pKaᵣ(ε-NH₃⁺) = 10.53。计算其 pI。

**解析**：

1. **判断类型**：含额外侧链氨基 → **碱性氨基酸**

2. **pI 计算**：取两个氨基 pKa 的平均值

$$
   \mathrm{pI} = \frac{8.95 + 10.53}{2} = 9.74
   $$

3. **验证**：
   - pI ≈ 9.74，大于 7 → 碱性氨基酸的 pI 偏碱性，符合规律
   - 在生理 pH 7.4 时，Lys 带正电荷

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| "所有氨基酸的 pI 都在 7 附近" | 以为氨基酸都是"中性" | 酸性氨基酸 pI ~3，碱性氨基酸 pI ~10 | 天冬氨酸和赖氨酸谁的 pI 更高？ |
| "pI = (pKa₁ + pKa₂)/2 通用于所有氨基酸" | 忽略侧链电离基团 | 酸性氨基酸用 (pKa₁+pKaᵣ)/2；碱性用 (pKaᵣ+pKa₂)/2 | 含三个可电离基团的氨基酸如何算 pI？ |
| "D-氨基酸就是 R 构型" | 混淆命名体系 | D/L 基于 Fischer 投影与甘油醛的关系；R/S 基于 CIP 规则 | L-半胱氨酸是 R 还是 S？为什么？ |
| "pH = pI 时氨基酸不带电" | 忽略两性离子 | pI 时**净电荷为零**，但分子内有正负电荷（两性离子）| pI 时氨基酸真的"不带电"吗？ |
| "电泳时所有氨基酸都向阳极移动" | 忽略 pI 与 pH 的关系 | pH > pI → 阳极；pH < pI → 阴极 | pH=3 时 Glu 向哪边移动？Lys 呢？ |
| "Strecker 合成可得光学纯氨基酸" | 忽略反应机理 | Strecker 从平面醛出发 → **外消旋**混合物 | 如何从不具有手性的原料得到光学纯氨基酸？ |

### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |

### 12.3 入门例题

**最简判断练习**：
> 1. 丙氨酸的 pKa₁ = 2.34, pKa₂ = 9.69，计算 pI。
> 2. pH = 7.0 时，Lys（pI = 9.74）在电场中向哪极移动？
> 3. 写出 Strecker 合成丙氨酸的原料。

**答案**：
1. pI = (2.34 + 9.69)/2 = **6.02**
2. pH = 7.0 < pI = 9.74 → 净正电荷 → **向阴极**移动
3. **乙醛 + NH₃ + HCN** → 氨基丙腈 → 水解得丙氨酸（外消旋）

**引导**：问"为什么 Strecker 合成得到的是外消旋产物？"（醛的羰基平面两侧被 NH₃ 和 CN⁻ 进攻的概率相等）

### 12.4 现实类比

**"磁铁在磁场中"类比电泳**：
- 氨基酸分子 = **小磁铁**
- 两性离子状态 = 磁铁**横向放置**——不指向任何一极（pH = pI 时不移动）
- pH < pI = 磁铁** N 极朝下**——被正极（阴极）吸引（向阴极移动）
- pH > pI = 磁铁** S 极朝下**——被负极（阳极）吸引（向阳极移动）
- **不同氨基酸的 pI 不同** = 不同磁铁的"平衡点"不同——在同一 pH 下有的向阴极、有的向阳极、有的不动，因此可以分离
