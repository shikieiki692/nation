# 不对称氢化（Asymmetric Hydrogenation）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[50-有机合成]]、[[51-金属有机]]

---

## 一、定义

**不对称氢化（Asymmetric Hydrogenation）**：在**手性过渡金属催化剂**存在下，**前手性烯烃（或酮）**与**氢气**发生对映选择性加成，生成**单一对映体（或 enriched 对映体）**的反应。

**通式**：
$$
\ce{前手性烯酮 + H2 ->[手性催化剂] 手性产物}
$$

**本质**：**催化量的手性配体控制大量底物的立体化学**——是手性合成中最高效、最绿色的方法之一。2001 年诺贝尔化学奖授予 Knowles、Noyori 和 Sharpless（不对称催化氢化和氧化）。

## 三、核心原理

### 3.1 手性催化剂体系

**Noyori 体系（BINAP-Ru/Rh）**：
- **BINAP** = 2,2'-双(二苯基膦)-1,1'-联萘
- 是 **轴手性** 配体（atropisomeric）
- (R)-BINAP 和 (S)-BINAP 互为对映体
- Ru-BINAP 用于酮的氢化
- Rh-BINAP 用于烯烃的氢化

**其他手性配体**：

| 配体 | 手性类型 | 应用 |
|:---|:---|:---|
| **BINAP** | 轴手性 | 烯烃/酮氢化 |
| **DuPhos** | 中心手性（P原子）| 烯胺氢化 |
| **Josiphos** | 平面手性（二茂铁）| 工业氢化 |
| **PHOX** | 中心+平面手性 | 不对称氢甲酰化 |

### 3.2 对映选择性来源

**手性口袋（Chiral Pocket）**：
- 金属中心周围的手性环境
- 底物以特定构象进入口袋
- 两个对映面的活化能不同 → 一个产物占优

**对映过量（ee 值）**：
$$
\text{ee} = \frac{[R] - [S]}{[R] + [S]} \times 100\%
$$
- ee = 0%：外消旋
- ee = 100%：单一对映体
- 竞赛中通常 ee > 90% 认为"高对映选择性"

### 3.3 底物范围

**烯烃不对称氢化**：
- 脱氢氨基酸衍生物 → 手性氨基酸
- 烯酰胺 → 手性胺
- α,β-不饱和羧酸 → 手性羧酸

**酮不对称氢化**：
- β-酮酯 → 手性 β-羟基酯
- 芳基烷基酮 → 手性仲醇
- 环状酮 → 手性环醇

### 3.4 不对称双羟化（Sharpless AD）

虽然不是氢化，但同属不对称催化：
- **AD-mix-α** 和 **AD-mix-β**
- OsO₄ / 手性配体（DHQ 或 DHQD）/ NMO
- 烯烃 → 手性邻二醇
- 两种混合试剂给出相反构型

---

## 四、关键结论

1. **不对称氢化 = H₂ + 前手性底物 → 手性产物**
2. **手性配体控制对映选择性**
3. **BINAP 是轴手性配体的代表**
4. **ee 值衡量对映选择性**
5. **Noyori（Ru）用于酮，Knowles（Rh）用于烯烃**
6. **是绿色、高效的手性合成方法**

## 五、常见分类或情形

### 5.1 按底物分类

| 底物 | 产物 | 催化剂 |
|:---|:---|:---|
| **脱氢氨基酸** | 手性氨基酸 | Rh-DuPhos |
| **β-酮酯** | 手性 β-羟基酯 | Ru-BINAP |
| **芳基烷基酮** | 手性仲醇 | Ru-BINAP |
| **烯酰胺** | 手性胺 | Rh-DuPhos |

### 5.2 按配体分类

| 配体 | 金属 | 底物 | ee 值 |
|:---|:---:|:---|:---:|
| **BINAP** | Ru/Rh | 多种 | 90–99% |
| **DuPhos** | Rh | 烯胺 | >99% |
| **Josiphos** | Rh | 工业底物 | 95–99% |

## 六、适用条件与限制

1. **需要高压氢气**（1–100 atm）
2. **严格无水无氧**
3. **手性配体昂贵**——但催化量使用
4. **底物需纯化**——杂质可能毒化催化剂
5. **对映面区分度依赖底物结构**
6. **某些底物 ee 值不高**

---

## 七、常见比较与易混点

### 7.1 不对称氢化 vs 普通氢化

| | **不对称氢化** | **普通氢化** |
|:---|:---|:---|
| **催化剂** | 手性金属络合物 | Pd/C, PtO₂ 等 |
| **产物** | 单一对映体（ enriched）| 外消旋或无手性 |
| **条件** | 较苛刻 | 较温和 |
| **成本** | 高（配体贵）| 低 |

### 7.2 BINAP 的手性类型

- BINAP 是 **轴手性**（atropisomerism）
- 不是中心手性（没有手性碳）
- 手性来源于联萘骨架的旋转受阻
- 两个对映体：(R)-BINAP 和 (S)-BINAP

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：什么是对映过量（ee）？如何计算？

**解答**：
- **定义**：一个对映体超过另一个对映体的比例
- **公式**：ee = (|R – S| / (R + S)) × 100%
- ee = 0%：外消旋（1:1）
- ee = 100%：单一对映体

### 10.2 例题 2（★★ 配体选择）

**题目**：β-酮酯的不对称氢化应该选用什么催化剂体系？

**解答**：
- **Ru-BINAP** 体系
- Noyori 的经典体系
- 产物：**手性 β-羟基酯**
- ee 值通常 > 95%

### 10.3 例题 3（★★★ ee 计算）

**题目**：某不对称氢化产物中 R 构型占 95%，S 构型占 5%，ee 值是多少？

**解答**：
- ee = (95% – 5%) / (95% + 5%) × 100%
- ee = 90% / 100% × 100%
- **ee = 90%**

---

## 十一、易错点

1. ❌ **BINAP 是中心手性** → BINAP 是 **轴手性**
2. ❌ **ee = 95% 意思是产物纯度 95%** → ee = 95% 意思是 **对映体比例 97.5:2.5**
3. ❌ **不对称氢化不需要高压** → 通常需要 **H₂ 压力**
4. ❌ **任何烯烃都能不对称氢化** → 需要 **前手性**烯烃

### 12.3 现实类比

**不对称氢化像"安检门"**：
- 前手性底物 = 一群人（左右手都有）
- 手性催化剂 = 只有一个入口的旋转门
- 旋转门只让右手的人通过 → 产物主要是右手的
- 如果旋转门设计得很好（配体手性强）→ 几乎只有右手的人通过（ee ≈ 100%）
- 普通氢化 = 普通大门 → 左右手的人都通过（外消旋）

## 十三、竞赛拓展

1. **不对称转移氢化**：
   - 用异丙醇等氢供体代替 H₂
   - 更安全、更温和
   - Noyori 的 "手性催化剂 + 异丙醇" 体系

2. **动态动力学拆分（DKR）**：
   - 底物在反应条件下不断消旋化
   - 催化剂只选择其中一个对映体反应
   - 理论上可达 100% 转化率 + 高 ee
