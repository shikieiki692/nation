# Wurtz 偶联反应（Wurtz Reaction）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[51-金属有机化学]]

---

## 一、定义

**Wurtz 偶联反应**：卤代烷（RX）与金属钠（Na）反应，通过**单电子转移（SET）**机理生成**对称烷烃（R-R）**的反应。

**Wurtz-Fittig 反应**：Wurtz 反应的变体，使用**卤代芳烃 + 卤代烷**与 Na 反应，生成**烷基芳烃（Ar-R）**。

**本质**：碱金属（Na, K, Li）通过单电子还原将 C-X 键断裂，生成的自由基偶联形成 C-C 键。

## 三、核心原理

### 3.1 反应通式

```
2 R-X  +  2 Na   →   R-R  +  2 NaX

R-X  +  R'-X  +  2 Na  →  R-R' + R-R + R'-R' + 2 NaX  （交叉偶联，产物混合物）
```

### 3.2 单电子转移机理

**步骤 1：电子转移生成自由基**
```
R-X  +  Na  →  R·  +  NaX
```
- Na 给出单个电子 → R-X 的 C-X σ* 轨道接受电子 → 键断裂
- 生成 **烷基自由基（R·）** 和 NaX

**步骤 2：自由基偶联**
```
R·  +  R·  →  R-R
```
- 两个自由基结合 → 对称烷烃

**总过程**：
```
2 R-X  +  2 Na  →  2 R·  +  2 NaX  →  R-R  +  2 NaX
```

### 3.3 为什么交叉偶联不好？

当使用两种不同卤代烷（R-X 和 R'-X）时：
```
R· + R· → R-R
R· + R'· → R-R'
R'· + R'· → R'-R'
```

- 三种产物混合物，难以分离
- **合成价值低**
- 若要合成不对称产物，通常用 **Wurtz-Fittig**（芳基 + 烷基，芳基自由基较稳定，交叉产物比例较高）或 **Corey-House 合成**（有机铜试剂偶联）

### 3.4 溶剂与条件

- **溶剂**：无水乙醚、THF、干燥苯
- **温度**：回流或室温
- **必须在无水无氧条件下进行**——Na 极易被水和氧气破坏
- **底物限制**：
  - 伯卤代烷效果最好
  - 仲卤代烷产率中等，易发生消除
  - 叔卤代烷主要发生消除，偶联产率极低
  - 乙烯基卤和芳基卤可用（Wurtz-Fittig）

---

## 四、关键结论

1. **Wurtz 偶联 = 单电子转移 + 自由基偶联**
2. **只适用于对称烷烃的合成**——交叉偶联产物复杂
3. **伯卤代烷最优**——仲、叔卤代烷易发生消除副反应
4. **无水无氧是必须的**——Na 对水氧极其敏感
5. **Wurtz-Fittig 是交叉偶联的改良版**——芳基 + 烷基，交叉产物比例较高

## 五、常见分类或情形

### 5.1 按底物分类

| 底物 | 产物 | 可行性 | 备注 |
|:---|:---|:---:|:---|
| **2 R-CH₂-X** | R-CH₂-CH₂-R | ✅ 好 | 伯卤代烷最优 |
| **2 R₂CH-X** | R₂CH-CHR₂ | ⚠️ 中等 | 有消除竞争 |
| **2 R₃C-X** | R₃C-CR₃ | ❌ 差 | 主要消除 |
| **Ar-X + R-X** | Ar-R（Wurtz-Fittig）| ✅ 较好 | 交叉产物为主 |

### 5.2 按金属分类

| 金属 | 活性 | 特点 |
|:---|:---:|:---|
| **Na** | 高 | 最经典，反应剧烈 |
| **K** | 更高 | 更活泼，危险 |
| **Li** | 中等 | 用于制备有机锂试剂 |

---

## 六、适用条件与限制

1. **必须严格无水**——Na 与水剧烈反应生成 H₂
2. **必须严格无氧**——Na 被氧气氧化
3. **伯卤代烷最优**——叔卤代烷几乎不能用于 Wurtz 偶联
4. **不适用于含活泼氢的底物**——OH、NH、COOH 会先于反应消耗 Na
5. **交叉偶联产物复杂**——合成不对称烷烃时应选用其他方法

## 七、常见比较与易混点

### 7.1 Wurtz 偶联 vs 其他 C-C 偶联反应

| | **Wurtz** | **Wurtz-Fittig** | **Corey-House** | **Kolbe 电解** |
|:---|:---|:---|:---|:---|
| **试剂** | Na / RX | Na / ArX + RX | R₂CuLi / R'X | 羧酸盐电解 |
| **机理** | SET/自由基 | SET/自由基 | 氧化加成/还原消除 | 自由基 |
| **产物** | R-R | Ar-R | R-R' | R-R |
| **交叉偶联** | ❌ 差 | ⚠️ 中等 | ✅ 好 | ❌ 差 |
| **底物限制** | 伯卤代烷 | 芳基 + 烷基 | 广 | 羧酸 |

### 7.2 Wurtz 偶联 vs 格氏试剂制备

| | **Wurtz** | **格氏试剂** |
|:---|:---|:---|
| **试剂** | Na + RX | Mg + RX |
| **产物** | R-R（偶联）| RMgX（金属有机试剂）|
| **目的** | 合成烷烃 | 制备亲核试剂 |
| **金属** | Na（更活泼）| Mg（较温和）|

**关键区别**：同样是金属 + 卤代烷，Na 导致偶联，Mg 导致金属插入。原因是 Na 的单电子转移速率远快于 Mg 的氧化加成插入。

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：用 Wurtz 偶联合成正己烷，写出反应式。

**解答**：
```
2 CH₃CH₂CH₂-Br  +  2 Na  →  CH₃CH₂CH₂-CH₂CH₂CH₃  +  2 NaBr
```
- 使用 1-溴丙烷（伯卤代烷）
- 产物：正己烷（对称烷烃）

### 10.2 例题 2（★★ 机理）

**题目**：写出 Wurtz 偶联的单电子转移机理。

**解答**：
```
步骤 1：R-X + Na → [R-X]⁻· + Na⁺
         单电子转移 → 自由基负离子（短暂）
         
步骤 2：[R-X]⁻· → R· + X⁻
         键断裂 → 烷基自由基
         
步骤 3：R· + R· → R-R
         自由基偶联
```

### 10.3 例题 3（★★ 交叉偶联问题）

**题目**：为什么用 1-溴乙烷和 1-溴丙烷进行 Wurtz 偶联合成正戊烷不可行？

**解答**：
- 反应中同时生成乙基自由基和丙基自由基
- 三种偶联产物：丁烷（C₂-C₂）、戊烷（C₂-C₃）、己烷（C₃-C₃）
- 正戊烷只是三种产物之一，难以分离纯化
- **应改用 Wurtz-Fittig 或 Corey-House 合成**

---

## 十一、易错点

1. ❌ **用叔卤代烷做 Wurtz** → 主要发生消除，偶联产率极低
2. ❌ **交叉 Wurtz 期望单一产物** → 必然三种产物混合物
3. ❌ **忽略无水条件** → Na 遇水爆炸性反应
4. ❌ **混淆 Wurtz 和格氏试剂** → Na 偶联，Mg 插入

## 十三、竞赛拓展

1. **Wurtz-Fittig 的应用**：
   - 合成烷基苯类化合物
   - 在工业上有一定应用价值

2. **单电子转移（SET）机理的普遍性**：
   - 碱金属（Li, Na, K）与多种有机底物的反应都涉及 SET
   - 理解 SET 是理解有机金属化学的基础
