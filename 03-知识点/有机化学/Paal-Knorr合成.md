# Paal-Knorr 合成（Paal-Knorr Synthesis）
- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[52-杂环化合物]]、[[50-有机合成]]
---

## 一、定义

**Paal-Knorr 合成**：**1,4-二羰基化合物**在酸催化或加热条件下发生**分子内环化脱水**，生成**五元杂环**（呋喃、吡咯或噻吩）的反应。

**通式**：
$$
\ce{R-CO-CH2-CH2-CO-R' + [X] ->[酸或热] 五元杂环 + H2O}
$$
（X = O → 呋喃；X = NH → 吡咯；X = S → 噻吩）

**本质**：**五元杂环的万能合成法**——1,4-二羰基是"万能平台"，通过选择不同的亲核试剂（水/胺/硫），可以得到三种不同的五元杂环。

## 三、核心原理

### 3.1 反应机理

**共同前体：1,4-二羰基化合物**

```
步骤 1：烯醇化
  1,4-二羰基 → 烯醇式

步骤 2：亲核进攻
  烯醇氧/胺/硫醇进攻另一羰基碳

步骤 3：脱水关环
  形成五元环 + 失去水分子

步骤 4：芳构化（失去第二分子水或脱氢）
  生成芳香五元杂环
```

**关键中间体**：
- 先形成**羟基二氢呋喃**（或氮/硫类似物）
- 再脱水芳构化为芳香杂环

### 3.2 三种变体

| 杂环 | 试剂 | 反应条件 | 脱水方式 |
|:---|:---|:---|:---|
| **呋喃** | 酸（H₂SO₄、P₂O₅）| 加热 | 失去两分子水 |
| **吡咯** | 氨或伯胺（NH₃、RNH₂）| 加热或酸催化 | 失去两分子水 |
| **噻吩** | P₂S₅ 或 Lawesson 试剂 | 加热 | 硫代 + 脱水 |

### 3.3 1,4-二羰基化合物的制备

| 方法 | 反应 | 例子 |
|:---|:---|:---|
| **1,4-二酮的氧化** | 环氧化合物重排 | — |
| **乙酰乙酸酯法** | Claisen 缩合 | 乙酰乙酸乙酯 + 卤代酮 |
| **1,3-偶极加成** | — | — |
| **Stetter 反应** | 醛 + α,β-不饱和酮 | 噻唑鎓催化 |

---

## 四、关键结论

1. **Paal-Knorr = 1,4-二羰基 → 五元杂环**
2. **三种变体：呋喃（脱水）、吡咯（加胺脱水）、噻吩（硫代脱水）**
3. **1,4-二羰基是共同前体**
4. **机理涉及烯醇化 + 亲核关环 + 脱水芳构化**
5. **是五元杂环最通用的合成方法**

## 五、常见分类或情形

### 5.1 按杂原子分类

| 杂环 | 试剂 | 产物特征 | 例子 |
|:---|:---|:---|:---|
| **呋喃** | 酸脱水 | 氧杂五元芳环 | 2,5-二甲基呋喃 |
| **吡咯** | NH₃ / RNH₂ | 氮杂五元芳环 | N-苯基吡咯 |
| **噻吩** | P₂S₅ / Lawesson | 硫杂五元芳环 | 2,5-二甲基噻吩 |

### 5.2 按取代基分类

| 二羰基类型 | 产物取代模式 |
|:---|:---|
| **对称 1,4-二酮** | 2,5-二取代杂环 |
| **不对称 1,4-二酮** | 2,3,4,5-多取代杂环 |
| **含酯基** | 2-或5-位有酯基 |

## 六、适用条件与限制

1. **必须有 1,4-二羰基底物**
2. **酸催化或加热**
3. **吡咯合成需要氨或伯胺**
4. **噻吩合成需要硫化试剂**
5. **大位阻底物可能环化困难**
6. **某些官能团在酸性条件下不稳定**

---

## 七、常见比较与易混点

### 7.1 Paal-Knorr vs Hantzsch

| | **Paal-Knorr** | **Hantzsch** |
|:---|:---|:---|
| **底物** | 1,4-二羰基 | α-卤代酮 + β-酮酯 + 氨 |
| **产物** | 简单吡咯 | 高度取代吡咯（3-位有酯基）|
| **灵活性** | 高（三种杂环）| 低（仅吡咯）|

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：2,5-己二酮在浓硫酸加热条件下的产物是什么？

**解答**：
- 2,5-己二酮 = 1,4-二酮
- 酸催化脱水 → Paal-Knorr 呋喃合成
- 产物：**2,5-二甲基呋喃**

### 10.2 例题 2（★★ 吡咯合成）

**题目**：如何用 Paal-Knorr 法合成 N-甲基吡咯？

**解答**：
- 需要：1,4-二羰基 + 甲胺（CH₃NH₂）
- 底物：2,5-己二酮
- 试剂：甲胺
- 产物：**N-甲基-2,5-二甲基吡咯**

### 10.3 例题 3（★★★ 噻吩合成）

**题目**：2,5-己二酮与 Lawesson 试剂反应的产物是什么？

**解答**：
- Lawesson 试剂 = 硫代试剂
- Paal-Knorr 噻吩合成
- 产物：**2,5-二甲基噻吩**

## 十一、易错点

1. ❌ **Paal-Knorr 只能合成呋喃** → 可以合成**呋喃、吡咯、噻吩**
2. ❌ **吡咯合成用氨水就行** → 需要**无水条件**，水会抑制环化
3. ❌ **噻吩合成用硫粉** → 需要**Lawesson 试剂或 P₂S₅**

| 误区 | 正确认识 | 辨析口诀 |
|---|---|---|
| "只合成呋喃" | 三种杂环都可 | "二羰基底，氧氮硫都行" |
| "吡咯用氨水" | 需无水氨或胺 | "吡咯无水才能成" |

### 12.3 现实类比

**Paal-Knorr 像"万能模具"**：
- 1,4-二羰基 = 一个标准模具底座
- 水/胺/硫 = 三种不同的"填充材料"
- 同样的模具 + 不同材料 → 三种不同的产品（呋喃/吡咯/噻吩）
- 只需要换填充材料，不需要换模具
