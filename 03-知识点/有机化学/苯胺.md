# 苯胺

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[41-芳香亲电取代反应]]、[[42-芳香亲核取代反应]]

## 一、定义与结构

**苯胺（Aniline）**是最简单的芳香胺，结构为苯环直接连接氨基（Ar–NH₂）。

- **碱性**：pK_b ≈ 9.4（共轭酸苯铵离子 pK_a ≈ 4.6），碱性显著**弱于脂肪胺**（脂肪胺 pK_b ≈ 3–5）
- 氮上的孤对电子与苯环发生 **p–π 共轭**，电子离域到苯环，导致：
  - 碱性减弱（孤对电子不易结合质子）
  - 芳环**极度活化**（亲电取代比苯酚还快）

$$
\ce{C6H5NH2 + H+ <=> C6H5NH3+}
$$

> ABOC 核心视角："苯酚能进行的反应，苯胺一般都能进行而且更快"——但苯胺**更需要保护**。

## 二、制备方法

| 方法 | 试剂/条件 | 特点 | 备注 |
|:---|:---|:---|:---|
| **硝基苯还原** | Fe/Sn/Zn + HCl 或 H₂/催化剂 | 最工业化的方法 | 单电子逐步还原：硝基→亚硝基→羟胺→胺 |
| **卤代芳烃氨解** | ArX + NH₃（高温高压）| 需吸电子基活化 | 无活化基时反应极难 |
| **Gabriel合成变体** | 邻苯二甲酰亚胺 + ArX + 肼解 | 制1°芳胺 | 与脂肪族 Gabriel 原理相同 |
| **Hofmann重排** | 苯甲酰胺 + Br₂/NaOH | 少一个碳的胺 | 从羧酸衍生物制备 |
| **重氮盐水解** | ArN₂⁺ + H₂O（加热）| 由胺出发的循环制备 | 常用于特定位置的羟基/胺基互换 |

### 硝基苯还原的机理细节

单电子还原序列：

$$
\ce{Ph-NO2 ->[2e-, 2H+] Ph-NO ->[2e-, 2H+] Ph-NHOH ->[2e-, 2H+] Ph-NH2}
$$

> ⚠️ **关键细节**：还原必须在**酸性条件**下进行。若在中性或碱性条件下，苯胺负离子会进攻亚硝基苯，经 E1cb 消除得到**偶氮苯**（azobenzene），进一步还原得氢化偶氮苯，最终酸催化下发生**联苯胺重排**。

## 三、碱性

### 3.1 为什么苯胺碱性弱于脂肪胺？

| 对比 | 脂肪胺（如甲胺）| 苯胺 |
|:---|:---|:---|
| 氮孤对电子 | 定域在 N 上 | 离域到苯环（p–π 共轭）|
| 结合质子能力 | 强 | 弱 |
| pK_b | ~3.4 | ~9.4 |
| 共轭酸稳定性 | — | 苯铵离子通过诱导效应分散正电荷 |

### 3.2 取代基效应

- **给电子基**（–CH₃, –OCH₃）→ 碱性**增强**（邻/对位尤其明显）
- **吸电子基**（–NO₂, –COOH）→ 碱性**减弱**（间位影响较小，邻/对位显著）
- 硝基苯胺的 pK_b：对位 > 间位 > 邻位（邻位有分子内氢键特殊稳定化）

## 四、亲电取代反应

苯胺的芳环**极度活化**，亲电取代反应**不需要催化剂**，且**多取代**倾向极强。

### 4.1 卤代

- 与 Br₂ 反应：即使在低温、CS₂ 中，也立即得到 **2,4,6-三溴苯胺**（白色沉淀）
- 如需单取代，必须**先保护氨基**：将 –NH₂ 转化为 –NHCOCH₃（乙酰苯胺），此时活性降低，位阻增大，主要得**对位**单取代产物

> ABOC 特别提示：苯胺的卤代比苯酚更剧烈——苯酚溴代还需 Br₂/H₂O 或 CS₂ 低温，苯胺几乎无条件就三取代。

### 4.2 硝化

- **浓 HNO₃**：直接氧化破坏苯胺（勿用）
- **稀 HNO₃**：可反应，但产物复杂
- **正确策略**：先乙酰化保护，再用混酸硝化 → 得对硝基乙酰苯胺 → 水解得对硝基苯胺

### 4.3 磺化与 Friedel-Crafts 反应

- 磺化：氨基需先保护，否则产率低且主要在**间位**（质子化氨基成为间位定位基）
- F-C 酰基化/烷基化：**必须保护**，否则 Lewis 酸（AlCl₃）与氨基配位，导致芳环钝化

### 4.4 与苯酚反应的对比

| 反应类型 | 苯酚 | 苯胺 |
|:---|:---|:---|
| 卤代 | Br₂/H₂O → 三溴苯酚（无需催化剂）| Br₂/CS₂ 低温 → 三溴苯胺（更快）|
| 硝化 | 稀 HNO₃ 可行 | 必须保护，否则氧化破坏 |
| 磺化 | 可逆，低温邻位/高温对位 | 需保护，否则间位为主 |
| F-C 反应 | 酚羟基与 AlCl₃ 配位，需先成酯（Fries重排）| 氨基与 AlCl₃ 配位更强，必须保护 |
| 重氮盐偶联 | 作为偶联组分（亲核）| 作为偶联组分（亲核），也作为重氮盐前体 |

## 五、氧化反应

苯胺**极易被氧化**（氨基强给电子效应）。

| 氧化剂 | 产物 | 备注 |
|:---|:---|:---|
| **空气/O₂**（长期）| 颜色变深（复杂氧化产物）| 苯胺需避光避空气保存 |
| **Fremy 盐** | 对苯醌 | 高产率，选择性氧化 |
| **KClO₃/HCl** | 四氯苯醌（氯醌）| 同时氯化+氧化 |
| **KMnO₄ / 过氧酸** | 硝基苯 | 深度氧化 |
| **Na₂Cr₂O₇/H₂SO₄** | 黑色复杂产物 | 过于剧烈，合成上无价值 |

> 工业意义：苯胺氧化在染料工业中有应用，但控制条件极为关键。

## 六、重氮盐化学

苯胺最重要的转化之一：制备**芳香重氮盐**。

### 6.1 重氮化反应

$$
\ce{Ar-NH2 + NaNO2 + 2HCl ->[0-5^\circ C] Ar-N2+Cl- + 2H2O}
$$

- **温度控制**：0–5°C，温度过高重氮盐分解（放出 N₂，生成苯基正离子或酚）
- **酸过量**：保持酸性防止重氮酸盐生成
- **亚硝酸钠稍过量**：用淀粉-KI 试纸检测终点

### 6.2 重氮盐的主要转化（Sandmeyer 等）

| 条件 | 产物 | 反应名称 |
|:---|:---|:---|
| CuCl | Ar–Cl | Sandmeyer |
| CuBr | Ar–Br | Sandmeyer |
| CuCN | Ar–CN | Sandmeyer |
| KI（无 Cu）| Ar–I | 直接取代 |
| HBF₄, Δ | Ar–F | Balz–Schiemann |
| H₃PO₂ | Ar–H | 去氨基还原 |
| H₂O, Δ | Ar–OH | 水解（制备苯酚）|
| 苯酚/苯胺（偶联）| 偶氮化合物 | 偶联反应 |

## 七、偶联反应

重氮盐作为**弱亲电试剂**，与活化的芳香化合物（苯酚、苯胺）发生**亲电芳香取代**。

- **与苯胺偶联**：
  - 条件：pH = 5–7（弱酸性）
  - 位置：苯胺的**对位**优先；若对位被占，则邻位
  - 产物：氨基偶氮苯

- **与苯酚偶联**：
  - 条件：pH = 8–9（弱碱性，苯酚以酚氧负离子形式存在，活性更高）
  - 位置：酚氧负离子的对位优先
  - 产物：羟基偶氮苯（偶氮染料的基本结构）

> 偶氮染料工业：世界上约 70% 的有机染料含有 –N=N– 偶氮结构，重氮盐-苯胺/苯酚偶联是核心合成路径。

## 八、关键结论

1. **苯胺 ≈ 超级活化的苯 + 弱碱性的胺**：两个特征（活化芳环 + 弱碱性）决定了它所有的化学行为。
2. **保护策略是核心考点**：苯胺的多数有用取代反应都需要先把 –NH₂ 变成 –NHCOCH₃，否则只能得到多取代或氧化产物。
3. **重氮盐是万能中转站**：从苯胺出发，通过重氮盐可以引入 –Cl、–Br、–I、–F、–CN、–OH、–H，或连接到另一个芳环上（偶联）。
4. **碱性条件下的副反应**：硝基苯还原若不在酸性中进行，会得到偶氮苯和联苯胺重排产物——这是竞赛中常考的"条件控制"考点。

## 十、常见比较与易混点

| 比较项 | 正确认识 | 常见错误 |
|:---|:---|:---|
| 苯胺碱性 | 弱于脂肪胺、弱于氨 | "芳香胺碱性强"——恰恰相反 |
| 苯胺亲电取代 | 无需催化剂，多取代 | "像苯一样单取代"——需要保护 |
| 苯胺硝化 | 必须先保护（乙酰化）| "直接用混酸"——产物是氧化破坏 |
| 重氮化温度 | 0–5°C 严格控温 | "室温也行"——室温下放 N₂ 分解 |
| 偶联 pH | 苯胺 pH 5–7，苯酚 pH 8–9 | "都一样"——pH 决定物种形态和活性 |

## 十一、易错点

- ❌ **"苯胺比苯酚更稳定"**——苯胺更易被氧化，空气中变色更快。
- ❌ **"氨基是邻对位定位基，所以可以直接做 F-C 反应"**——F-C 的 Lewis 酸会与氨基配位，导致芳环钝化，必须先保护。
- ❌ **"硝基苯还原总得到苯胺"**——碱性条件下得到偶氮苯，这是 E1cb 消除的实例。
- ❌ **"重氮盐很稳定，可以长期存放"**——干燥重氮盐会爆炸，通常现制现用。
- ❌ **"偶联反应在强酸中进行"**——强酸使苯胺质子化（失去亲核性），强碱使重氮盐变成重氮酸盐（失去亲电性），必须弱酸性或弱碱性。

### 12.3 入门级例题

**题目**：如何由苯胺制备对溴苯胺？写出关键步骤并说明为什么需要这样的操作顺序。

**预期解答路径**：
1. 苯胺 + 乙酸酐 → 乙酰苯胺（保护氨基，降低活性）
2. 乙酰苯胺 + Br₂/FeBr₃ 或 Br₂/AcOH → 对溴乙酰苯胺（对位优先）
3. 对溴乙酰苯胺 + H₃O⁺/Δ 或 NaOH/Δ → 水解得对溴苯胺
4. 若不保护，直接溴代会得到 2,4,6-三溴苯胺，无法得到单取代产物

## 十三、竞赛拓展

- **联苯胺重排（Benzidine Rearrangement）**：氢化偶氮苯在酸催化下的 [5,5]-σ 迁移，产物是 4,4'-二氨基联苯。涉及 π–π 堆积和质子化调控。
- **芳香胺的致癌性**：某些芳香胺（如联苯胺、β-萘胺）具有强致癌性，与 DNA 形成加合物。工业安全中需严格防护。
- **苯胺在黑火药中的历史应用**：19 世纪苯胺紫（mauveine）的发现开启了合成染料工业。
