# 烯醇负离子

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[46-羰基α位反应]]

## 一、定义

**烯醇负离子（Enolate）** 是羰基化合物在碱作用下，**α-碳去质子化**形成的共振稳定的碳负离子-烯醇盐杂化体。

$$
\ce{R-CH2-CO-R' + B^- -> [R-CH^- -CO-R' \longleftrightarrow R-CH=C(O^-) -R'] + HB}
$$

- **本质**：羰基 α-H 被碱夺取后，负电荷通过共振离域到 **α-碳** 和 **羰基氧** 上
- **两种极限式**：
  - **碳负离子式**（⁻C–C=O）：负电荷在碳上，体现亲核性
  - **烯醇盐式**（C=C–O⁻）：负电荷在氧上，更稳定（氧电负性高）
- **实际结构**：共振杂化体，C–C 键有部分双键性，C–O 键有部分单键性

## 三、核心原理

### 3.1 形成条件与 α-H 酸性

羰基 α-H 的酸性来源于 **烯醇负离子的共振稳定化**：

| 羰基化合物 | 典型 pKa | α-H 酸性来源 | 去质子难度 |
|:---|:---:|:---|:---:|
| **醛/酮** | 20 | 烯醇负离子共振稳定 | 需强碱（LDA、NaNH₂）|
| **酯** | 25 | 烯醇负离子共振 + 烷氧基给电子稍弱 | 需更强碱（LDA、NaH）|
| **酰胺** | 30 | 烯醇负离子共振 + 氨基给电子 | 需极强碱 |
| **β-二酮** | 9 | 两个羰基共同稳定烯醇负离子 | 弱碱即可（NaOEt）|
| **β-酮酯** | 11 | 类似 β-二酮 | 弱碱即可 |
| **硝基烷** | 10 | 硝基强吸电子稳定碳负离子 | 弱碱即可 |

**核心结论**：
- 两个羰基夹一个 CH₂（β-二羰基）时，烯醇负离子被 **双重共振稳定**，酸性接近苯酚
- 单羰基化合物的 α-H 酸性弱于水，需 **强碱** 才能有效去质子

### 3.2 共振结构详解

以丙酮烯醇负离子为例：

```
      ⁻                      ⁻
   H₃C-C=CH₂   ⟷   H₃C-C=CH₂
      |                  |
      O                  O
   碳负离子式         烯醇盐式
   (亲核位点)       (更稳定的极限式)
```

- **负电荷分布**：约 2/3 在氧上，1/3 在碳上（取决于具体体系和计算级别）
- **亲核进攻位点**：虽然氧上负电荷更多，但 **碳是亲核进攻位点**（C-烷基化）；氧是亲电位点（O-硅基化/酰基化）
- **C- vs O-选择性**：
  - 强亲电体、极性非质子溶剂 → **C-烷基化**（动力学控制）
  - 弱亲电体、质子性溶剂 → **O-烷基化** 可能竞争

## 四、关键结论

1. **动力学烯醇 vs 热力学烯醇**：
   - **动力学烯醇**：取代较少的双键（较不稳定），在 **低温、强位阻碱（LDA）、不可逆条件** 下形成
   - **热力学烯醇**：取代较多的双键（较稳定），在 **高温、弱碱、可逆条件** 下形成
2. **烯醇负离子的两种反应模式**：
   - **作为亲核体**（C-位）：α-烷基化、Aldol 缩合、Michael 加成、Claisen 缩合
   - **作为碱**（O-位）：夺取其他质子（较少见，但可发生）
3. **硅烯醇醚（Silyl Enol Ether）**：烯醇负离子的 O-硅基化保护形式，可隔离储存，后续再生成烯醇负离子
4. **LUMO 控制**：烯醇负离子的 HOMO 与羰基 LUMO 相互作用是 Aldol/Michael 反应的核心

## 五、常见分类

### 5.1 按形成条件分类

| 类型 | 条件 | 产物特征 | 典型应用 |
|:---|:---|:---|:---|
| **动力学烯醇负离子** | LDA，THF，-78°C | 较少取代的双键 | α-烷基化（特定位置）|
| **热力学烯醇负离子** | NaOEt/EtOH，rt | 较多取代的双键 | 热力学控制烷基化 |
| **烯胺（Enamine）** | 仲胺 + 酸催化 | 类似烯醇负离子，但氮替代氧 | Stork 烯胺烷基化 |
| **硅烯醇醚** | LDA 后 + TMSCl | 烯醇负离子的 O-保护形式 | 可储存、后续 Lewis 酸活化 |
| **烯醇（Enol）** | 酸或碱催化互变异构 | 中性烯醇（非负离子）| 某些特殊反应的中间体 |

### 5.2 主要反应类型

```
烯醇负离子
├── 亲核进攻羰基（C-位）
│   ├── Aldol缩合（与醛/酮）→ β-羟基羰基
│   ├── Claisen缩合（与酯）→ β-酮酯
│   ├── Michael加成（与 α,β-不饱和羰基）→ 1,5-二羰基
│   └── 定向Aldol（LDA + 预-formed 烯醇 + 醛）→ 立体控制产物
├── 亲核取代（C-位）
│   ├── α-烷基化（与 RX，SN2）→ α-取代羰基
│   └── α-卤化（与 X₂）→ α-卤代羰基
├── O-反应
│   ├── O-硅基化（TMSCl）→ 硅烯醇醚
│   ├── O-酰基化 → 烯醇酯
│   └── O-烷基化（特殊条件）→ 烯醇醚
└── 氧化
    └── 硒氧化-消除（LDA + PhSeBr → 氧化消除）→ α,β-不饱和羰基
```

## 六、适用条件

### 6.1 烯醇负离子的制备条件

| 目标 | 碱 | 溶剂 | 温度 | 结果 |
|:---|:---|:---|:---:|:---|
| **动力学烯醇** | LDA、LiHMDS、NaHMDS | THF、DME | -78°C | 较少取代烯醇（动力学控制）|
| **热力学烯醇** | NaOEt、KOH、t-BuOK | EtOH、t-BuOH | rt 或 Δ | 较多取代烯醇（热力学控制）|
| **硅烯醇醚** | LDA/-78°C，然后 TMSCl | THF | -78°C → rt | 可分离的稳定烯醇等价物 |
| **烯胺** | 吡咯烷/吗啉 + TsOH | 苯或甲苯 | Δ（除水）| Stork 烷基化中间体 |

### 6.2 影响动力学/热力学选择性的因素

| 因素 | 动力学烯醇 | 热力学烯醇 |
|:---|:---|:---|
| **碱的位阻** | 大（LDA、LiHMDS）| 小（NaOEt）|
| **碱的离开能力** | 非亲核、不共轭 | 可逆、可质子交换 |
| **温度** | 低温（-78°C）| 高温（rt 或 Δ）|
| **溶剂** | 极性非质子（THF）| 质子性（EtOH）|
| **时间** | 短（立即加入亲电体）| 长（允许平衡）|
| **底物结构** | 位阻小的 α-H 优先被夺 | 更取代的双键更稳定 |

**竞赛核心**：
- 不对称酮（如 2-甲基环己酮）有两个 α-位：
  - C2（连甲基，位阻大，1 个 H）
  - C6（无甲基，位阻小，2 个 H）
- **LDA/-78°C**：夺 C6-H（位阻小，动力学）→ C1=C6 烯醇
- **NaOEt/rt**：平衡到 C1=C2 烯醇（三取代双键更稳定，热力学）

## 七、常见比较

### 7.1 烯醇负离子 vs 烯胺 vs 硅烯醇醚

| 维度 | 烯醇负离子 | 烯胺 | 硅烯醇醚 |
|:---|:---|:---|:---|
| **电荷** | 负电荷（活泼）| 中性（较稳定）| 中性（稳定，可储存）|
| **制备** | 强碱去质子 | 仲胺 + 酸催化 | LDA + TMSCl |
| **亲核位点** | C（主要）| C（β-碳）| C（需 Lewis 酸活化）|
| **单烷基化** | 易过度烷基化 | **单烷基化优异**（亚胺盐阻止第二次）| 可控 |
| **对醛反应** | Aldol 缩合 | Stork 烷基化（与 RX）| Mukaiyama Aldol |
| **储存性** | 不可储存 | 现场制备 | **可分离纯化** |

### 7.2 动力学 vs 热力学烯醇负离子的合成应用

以 2-甲基环己酮的 α-甲基化为例：

```
        O                      O                      O
        ||                     ||                     ||
   CH₃-C-CH₂-CH₂-CH₂   →   CH₃-C-CH-CH₂-CH₂   or   CH₃-C-C-CH₂-CH₂
        |                     |    |                     |   |
       CH₃                   CH₃  CH₃                  CH₃ CH₃
     底物                 动力学产物             热力学产物
                         (C6-甲基化)            (C2-二甲基化)
```

| 条件 | 产物 | 解释 |
|:---|:---|:---|
| **LDA, -78°C; then MeI** | 2,6-二甲基环己酮 | 动力学烯醇（C1=C6）在 C6 被捕获 |
| **NaOEt, rt; then MeI** | 2,2-二甲基环己酮 | 热力学烯醇（C1=C2）在 C2 被捕获 |

## 十、例题

### 例题 1：动力学与热力学烯醇的选择性

**题目**：2-甲基环己酮分别用 LDA/THF/-78°C 和 NaOEt/EtOH/rt 处理，然后加入 CH₃I。主产物分别是什么？

**解析**：

**LDA/-78°C（动力学控制）**：
- LDA 位阻大，优先夺取位阻小的 C6-H（2 个等价 H，无甲基取代）
- 形成 **C1=C6 烯醇负离子**（二取代双键，热力学较不稳定）
- CH₃I 捕获 → **2,6-二甲基环己酮**

**NaOEt/rt（热力学控制）**：
- 可逆条件，烯醇负离子可平衡
- **C1=C2 烯醇负离子**（三取代双键，热力学更稳定）占优
- CH₃I 捕获 → **2,2-二甲基环己酮**

### 例题 2：Aldol 缩合的完整机理

**题目**：写出乙醛在稀 NaOH 条件下的 Aldol 缩合机理。

**解析**：
1. **去质子**：OH⁻ 夺取乙醛的 α-H → 烯醇负离子 CH₂=CH-O⁻
2. **亲核加成**：烯醇负离子的 C 进攻另一分子乙醛的羰基碳 → 烷氧负离子
3. **质子化**：烷氧负离子从水中夺取 H⁺ → **3-羟基丁醛（Aldol 产物）**
4. **脱水（可选）**：加热或碱性条件下，β-羟基醛脱水 → **巴豆醛（CH₃CH=CHCHO）**

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| 认为烯醇负离子的亲核位点是氧 | 负电荷更多在氧上 | 虽然氧上负电荷多，但 **碳是亲核进攻位点**（C-C 键形成在合成上更有价值）| 烯醇负离子与 CH₃I 反应，产物是 C-甲基化还是 O-甲基化？ |
| 混淆动力学/热力学烯醇的形成条件 | 只记 "LDA = 动力学" | 强调三要素：**碱位阻大 + 低温 + 不可逆** = 动力学；**碱位阻小 + 高温 + 可逆** = 热力学 | NaH 对不对称酮去质子，主要得动力学还是热力学烯醇？ |
| 认为所有羰基化合物的 α-H 都易去质子 | 忽略 pKa 差异 | 单酮 α-H pKa ~20，需强碱；β-二酮 pKa ~9，弱碱即可 | 为什么乙酰丙酮能用 NaOEt 去质子，而丙酮需要 LDA？ |
| Aldol 缩合只写一步加成 | 忽略脱水步骤 | 碱性条件下 Aldol 产物常**自发脱水**得 α,β-不饱和羰基；合成题中需判断是否脱水 | 什么条件下 Aldol 产物会脱水？脱水驱动力是什么？ |
| 酯的烯醇负离子与酮混淆 | 未注意酯烯醇的特殊性 | 酯的烯醇负离子 **O-烷基化竞争更强**；Claisen 缩合需强碱和可逆条件 | 为什么 Claisen 缩合需要一分子酯被完全去质子？ |

## 十二、🎯 教学视角

### 12.1 学习路径

```
前置：互变异构 → 共轭效应 → 有机酸碱性 → 羰基极性
     ↓
本节：烯醇负离子（形成 + 共振 + 动力学/热力学 + 核心反应）
     ↓
后续分支：Aldol缩合 → Claisen缩合 → Michael加成 → Robinson环化 → 全合成
```

**建议课时**：3 课时（形成与共振 0.5 + 动力学/热力学 0.5 + Aldol/Claisen 1 + Michael/Robinson 1）

### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "烯醇负离子 = 烯醇" | 忽略负电荷 | 烯醇（enol）是中性；烯醇负离子（enolate）是负离子。两者反应性完全不同 |
| "烯醇负离子的氧不反应" | 忽略 O-反应 | 虽然 C-反应为主，但在特定条件下（TMSCl、酰氯）O-反应是重要路径 |
| "动力学产物 = 快形成的产物" | 字面理解 | 动力学产物是 **形成速率最快**的产物，但不一定是热力学最稳定的；强调 "动力学 = 速率控制" |
| "Aldol 就是两个醛在一起" | 简化理解 | Aldol 是 **烯醇负离子对羰基的亲核加成**；可以是醛+醛、酮+醛、酯+醛等 |

### 12.3 入门例题

**最简 Aldol 题**：
> 丙酮在稀 NaOH 条件下的 Aldol 缩合主产物是什么？

**答案**：**二丙酮醇**（4-羟基-4-甲基-2-戊酮）
**机理**：
1. 碱夺丙酮 α-H → 烯醇负离子
2. 烯醇负离子进攻另一分子丙酮的羰基
3. 质子化得 β-羟基酮
**引导**：若加热，产物会脱水得 **异亚丙基丙酮**（4-甲基-3-戊烯-2-酮）。

### 12.4 现实类比

**"弹簧发射器"类比烯醇负离子**：
- 羰基化合物 = 一个压紧的弹簧发射器（α-H 被羰基 "压住"）
- 碱 = 按下发射按钮（夺取 α-H）
- 烯醇负离子 = 弹簧释放，能量储存在两个方向（碳端和氧端像弹簧的两臂）
- **碳端** = 弹簧的 "撞击头"（亲核进攻，打出去形成新键）
- **氧端** = 弹簧的 "稳定把手"（负电荷主要在氧上，使弹簧不至于失控）
- **动力学/热力学** = 弹簧可以停在两种平衡位置：一种弹得快但不太稳（动力学），一种调整得慢但更稳（热力学）
- **LDA** = 快速猛击按钮，弹簧瞬间弹出到最近的位置（动力学）
- **NaOEt** = 慢慢按按钮，弹簧有时间寻找最舒服的位置（热力学）

## 十三、竞赛拓展

### 13.1 竞赛中的高级烯醇化学

| 反应/技术 | 特点 | 竞赛价值 |
|:---|:---|:---|
| **定向 Aldol（Evans Aldol）** | 用手性噁唑烷酮辅基控制烯醇的 syn/anti 选择性 | 不对称 Aldol，构建手性中心 |
| **Mukaiyama Aldol** | 硅烯醇醚 + Lewis 酸活化醛 → Aldol | 避免强碱，兼容敏感官能团 |
| **Mannich 反应** | 烯醇负离子 + 亚胺 → β-氨基羰基 | 构建含氮骨架 |
| **Michael 加成串联** | 烯醇负离子 + Michael 受体 → 1,5-二羰基 → 继续环化 | Robinson 环化 |
| **Tsuji-Trost 烯丙基化** | 烯醇负离子的烯丙基取代 | 构建季碳中心 |
| **Stork-Danheiser 反应** | 硅烯醇醚 + α,β-不饱和羰基 → 1,5-二羰基 | 环化前体 |

### 13.2 与现代方法学的交叉

- **有机催化 Aldol**：脯氨酸催化不对称 Aldol 反应（List-Houk 模型），无需金属
- **相转移催化（PTC）**：手性季铵盐催化烯醇负离子的不对称烷基化
- **光氧化还原催化**：烯醇负离子在光催化下发生自由基反应，拓展反应类型
