# 反应势能图（Reaction Energy Profile）

## 一、定义

**反应势能图**：描述化学反应过程中体系**势能（或自由能）随反应坐标变化**的曲线图。它是理解反应机理、动力学控制与热力学控制、过渡态与中间体关系的**核心可视化工具**。

**基本要素**：
- **横轴**：反应坐标（Reaction Coordinate）——从反应物到产物的"进程"
- **纵轴**：势能 / 吉布斯自由能（Energy / ΔG）
- **峰**：过渡态（Transition State, TS）——能量极大值（鞍点）
- **谷**：中间体（Intermediate）——能量极小值（局部稳定）

> ⭐ **ABOC 核心视角**：反应势能图是**平衡思维、动力学思维、热力学思维**的共同语言。一张图可以同时回答"反应快不快"（活化能）和"最终得什么"（产物能量）。

## 三、核心原理

### 3.1 单步反应 vs 多步反应的势能图

**单步反应（协同机理）**：
```
能量
  ↑        TS
  │       /  \
  │      /    \
  │   R /      \ P
  │    /        \
  └──/──────────\──────→ 反应坐标
```
- 1 个 TS，无中间体
- 例：SN2、E2、Diels-Alder

**两步反应**：
```
能量
  ↑    TS₁           TS₂
  │     \           /
  │      \   中间体 /
  │       \   ●   /
  │        \  ↑  /
  │      R  \│/  P
  └──────────┼────────→ 反应坐标
             ↑
          中间体
```
- 2 个 TS，1 个中间体（两峰夹一谷）
- 例：SN1、E1、加成-消除

> ⭐ **ABOC §2.9**：SN1 反应势能图（两峰夹谷）vs SN2 反应势能图（单峰）。

### 3.2 从势能图读取关键信息

| 图上要素 | 物理意义 | 决定什么 |
|:---|:---|:---|
| **峰高（ΔG‡）** | 活化能 | 反应速率（动力学） |
| **谷深（中间体能量）** | 中间体稳定性 | 中间体寿命、是否可捕获 |
| **终点高度（ΔG）** | 反应热力学 | 产物分布（热力学控制时）|
| **峰间距** | 各步活化能相对大小 | 哪步是决速步 |

### 3.3 动力学控制 vs 热力学控制的势能图语言

**经典图景（以 1,3-丁二烯 + HBr 为例，ABOC §2.10.1）**：

```
能量
  ↑
  │     TS(1,2)    TS(1,4)
  │      ↘         ↗
  │       ↘  烯丙基正离子
  │        ↘    ↑    ↗
  │         ↘   │   ↗
  │   R  →   │  │  ← 1,2-产物（较高）
  │          └──┴──→ 1,4-产物（较低）
  └────────────────────────→ 反应坐标
```

| 控制类型 | 图上判据 | 本例中的结果 |
|:---|:---|:---|
| **动力学控制** | 翻越**较低**的 TS | 1,2-加成（TS(1,2) < TS(1,4)）|
| **热力学控制** | 最终落在**更低**的能量谷 | 1,4-加成（产物能量更低）|

> ⭐ **ABOC §2.10.1**："势能图与产物选择性讨论：平衡思维的运用。"

### 3.4 Hammond 假说：过渡态像谁？

**Hammond 假说**：过渡态在结构上类似于与之**能量相近**的物种。

| 反应类型 | TS 位置 | TS 结构类似于 | 应用 |
|:---|:---|:---|:---|
| **放热反应** | 靠近反应物（早期 TS）| **反应物** | 推断 TS 结构，解释选择性 |
| **吸热反应** | 靠近产物（晚期 TS）| **产物/中间体** | 若中间体稳定 → TS 也稳定 → 活化能降低 |

**在势能图上的表现**：
```
放热反应              吸热反应
  能量                  能量
   ↑    TS               ↑
   │   / \               │          TS
   │  /   \              │         /  \
   │ R     \ P           │   R    /    \ P
   │/       \            │   /   /      \
   └─────────→           └──/──/────────→
   
   TS 像 R               TS 像 P
```

**竞赛应用**：
- 碳正离子越稳定 → 生成它的 TS 能量越低（Hammond：TS 像碳正离子）
- 这就是马氏规则、Zaitsev 规则的过渡态理论基础

## 四、关键结论

1. **一峰无谷 = 协同反应**（SN2、E2、周环）；**两峰夹谷 = 分步反应**（SN1、E1）
2. **峰高决定速率**（ΔG‡ → k）；**谷底决定平衡**（ΔG → K）
3. **动力学产物 = 翻越最低 TS 到达的产物**；**热力学产物 = 能量最低谷的产物**
4. **Hammond 假说是连接"稳定性"与"速率"的桥梁**——稳定的中间体意味着较低的 TS 能量
5. **反应势能图是"有机思维"的可视化**——平衡思维、动力学思维、热力学思维都可在图上表达

## 五、常见分类或情形

### 5.1 按反应步数分类

| 类型 | TS 数 | 中间体数 | 例子 |
|:---|:---:|:---:|:---|
| **单步协同** | 1 | 0 | SN2, E2, Diels-Alder, 硼氢化 |
| **两步** | 2 | 1 | SN1, E1, 烯烃 + HX, 缩醛形成 |
| **多步** | n | n-1 | 复杂串联反应、多步全合成 |

### 5.2 经典反应的势能图特征

| 反应 | 势能图特征 | 关键说明 |
|:---|:---|:---|
| **SN1** | 两峰夹谷 | 谷底 = 碳正离子；第一峰通常更高（决速步）|
| **SN2** | 单峰 | 五配位 TS；背面进攻导致构型翻转 |
| **E2** | 单峰 | 协同消除；反式共平面过渡态 |
| **1,2-加成** | 两峰夹谷（共同中间体）| TS(1,2) 较低 → 动力学产物 |
| **1,4-加成** | 两峰夹谷（共同中间体）| 产物能量较低 → 热力学产物 |

## 六、适用条件与限制

1. **势能图是简化的二维模型**——真实反应在多维势能面上进行
2. **纵轴通常是自由能（ΔG）而非内能（ΔE）**——热力学控制必须用 ΔG
3. **溶剂效应不直接显示在图上**——但溶剂会改变各点的相对能量
4. **Hammond 假说是定性工具**——不能定量预测 TS 结构
5. **不能从势能图直接读取反应速率常数**——需要 Eyring 方程（ΔG‡ → k）
6. **中间体的"谷"可能很浅**——浅谷意味着中间体寿命极短

## 七、常见比较与易混点

### 7.1 动力学 vs 热力学：势能图视角

| | **动力学控制** | **热力学控制** |
|:---|:---|:---|
| **决定因素** | ΔG‡（活化能） | ΔG（产物稳定性）|
| **图上判据** | 翻越**最低**的峰 | 落入**最深**的谷 |
| **温度条件** | 低温 | 高温 |
| **可逆性** | 不可逆 | 可逆，达平衡 |
| **典型例子** | 1,2-加成（低温）| 1,4-加成（高温）|
| **烯醇盐** | 动力学烯醇盐（少取代）| 热力学烯醇盐（多取代）|

### 7.2 过渡态 vs 中间体在图上的区别

| | **过渡态 (TS)** | **中间体 (Intermediate)** |
|:---|:---:|:---:|
| **势能面位置** | 峰（极大值）| 谷（极小值）|
| **能否画结构** | 不能（半断半成）| 能（接近正常分子）|
| **能否被捕获** | 不能 | 能 |
| **寿命** | ~10⁻¹³ s | ns ~ μs 或更长 |
| **口诀** | "山口擦身过" | "垭口可休息" |

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：画出 SN1 和 SN2 反应的势能图示意图，标注关键差异。

**解答**：

**SN1（两峰夹谷）**：
```
能量
  ↑    TS₁           TS₂
  │     \           /
  │      \  碳正离子 /
  │       \   ●   /
  │        \      /
  │   R  →  \    / ← P
  └──────────\──/────→ 反应坐标
```

**SN2（单峰）**：
```
能量
  ↑        TS
  │       /  \
  │      /    \
  │   R /      \ P
  │    /        \
  └──/──────────\──→ 反应坐标
```

关键差异：SN1 有中间体（碳正离子），SN2 无中间体。

### 10.2 例题 2（★★ 动力学/热力学判断）

**题目**：某反应有两条路径 A 和 B：
- 路径 A：ΔG‡ = 15 kcal/mol，产物能量 = -5 kcal/mol
- 路径 B：ΔG‡ = 20 kcal/mol，产物能量 = -12 kcal/mol

低温下主产物是什么？高温下呢？

**解答**：
- **低温（动力学控制）**：路径 A 活化能低（15 < 20）→ 翻越 TS_A 更快 → **产物 A**
- **高温（热力学控制）**：路径 B 产物更稳定（-12 < -5）→ 达平衡后 **产物 B** 占优
- 画势能图：TS_A 较低但产物 A 较高；TS_B 较高但产物 B 更低

### 10.3 例题 3（★★★ Hammond 假说应用）

**题目**：用 Hammond 假说解释：为什么烯烃质子化时，生成三级碳正离子的活化能低于生成一级碳正离子？

**解答**：
- 质子化是**吸热过程**（断裂 C=C π 键，形成 C-H σ 键，同时生成带正电的碳正离子）
- 根据 Hammond 假说：**吸热反应的 TS 类似于产物**（晚期 TS）
- 产物是碳正离子 → TS 已带有显著的碳正离子特征
- 三级碳正离子比一级稳定得多 → 生成三级碳正离子的 TS 也更稳定 → **活化能更低**
- 这就是马氏规则的过渡态理论基础

### 10.4 例题 4（★★★ 综合势能图分析）

**题目**：1,3-丁二烯与 HBr 反应的势能图显示：烯丙基碳正离子为共同中间体，从该中间体到 1,2-产物的 TS 较低，但 1,4-产物能量更低。解释为什么低温得 1,2-产物，高温得 1,4-产物。

**解答**：
- **低温**：反应不可逆，体系"翻越"活化能最低的 TS → 1,2-路径 → **动力学产物**
- **高温**：第二步（烯丙基正离子 → 产物）变得可逆，体系可在两产物间互变
- 最终达到热力学平衡 → 落在能量更低的 1,4-产物谷中 → **热力学产物**
- 这就是 ABOC 强调的"平衡思维"——高温允许"反悔"

## 十一、易错点

1. ❌ **把势能图当成"时间进程图"** → 横轴是反应坐标（结构变化），不是时间
2. ❌ **认为所有峰都是过渡态** → 中间体是谷，过渡态是峰
3. ❌ **用 ΔH 代替 ΔG 讨论热力学控制** → 必须用 ΔG（含熵贡献）
4. ❌ **忽略 Hammond 假说的适用范围** → 只适用于连续变化的反应系列
5. ❌ **认为动力学产物永远不变** → 若条件变为可逆，可能转为热力学产物
6. ❌ **把中间体的"谷深"等同于"稳定性"直接比较** → 还需看生成它的活化能
7. ❌ **画势能图时把 TS 画成谷** → TS 是能量极大值，不是极小值
8. ❌ **认为单步反应没有过渡态** → 单步反应有 TS，只是没有中间体

### 12.3 入门级例题

**题目**：画出"放热单步反应"和"吸热两步反应"的势能图示意图，标注反应物、产物、TS、中间体。

**预期解答路径**：
1. 放热单步：反应物能量 > 产物能量；单峰；无中间体
2. 吸热两步：反应物能量 < 产物能量；两峰夹一谷；谷底为中间体
3. 第一峰通常高于第二峰（若第一步为决速步）

**老师可引导的提问**：
- 卡在第 1 步：问 "放热反应的产物应该在反应物上方还是下方？"
- 卡在第 2 步：问 "两步反应有几个 TS？几个中间体？"
- 延伸问："若第二步的 TS 比第一步还高，决速步是哪一步？"

### 12.4 与现实/直觉的连接

- **类比**：势能图就像一张"山地地形图"。反应物在山谷 A，产物在山谷 B。要从 A 到 B，必须翻越山口（TS）。
  - 动力学控制 = "选最容易翻的山口"——不管对面的山谷有多深。
  - 热力学控制 = "最终住在最低的山谷"——如果可以来回翻（可逆），最后一定选在最低处。
  - 中间体 = "两山之间的垭口营地"——可以短暂停留。
- **反例**：直觉认为"能量低的产物一定生成快"——错误！产物能量低不代表去那里的路好走（TS 可能很高）。热力学产物可能生成极慢，动力学产物反而"捷足先登"。

## 十三、竞赛拓展

1. **Eyring 方程**：
   - k = (k_B T / h) · exp(-ΔG‡/RT)
   - 从势能图的 ΔG‡ 可计算速率常数
   - 温度每升高 10°C，速率约增 2-3 倍（经验规则）

2. **Marcus 理论**：
   - 电子转移反应的抛物线势能模型
   - 可以解释"内层/外层"电子转移、反转区（inverted region）
   - 是 Hammond 假说的数学化扩展

3. **More O'Ferrall-Jencks 图**：
   - 二维势能图（两个键同时变化）
   - 用于分析 SN1/SN2 边界、E1/E2 边界
   - ABOC 多次提及"之后学习 MoreO 图"（行 1865 等）
