# Lindlar 催化剂（Lindlar Catalyst）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[37-加成反应]]、[[43-亲电加成反应]]

---

## 一、定义

**Lindlar 催化剂**：将钯（Pd）沉积在碳酸钙（CaCO₃）载体上，并用**醋酸铅（Pb(OAc)₂）或喹啉毒化**的催化剂。用于**选择性将炔烃还原为顺式烯烃（Z-烯烃）**，而不进一步还原为烷烃。

**本质**：毒化降低了 Pd 的催化活性，使其足以活化 H₂ 加成到三键上，但**不足以活化 H₂ 加成到双键上**——从而"停在"烯烃阶段。

## 三、核心原理

### 3.1 催化剂组成与制备

**经典 Lindlar 催化剂组成**：
- **Pd**（活性金属，~5% 负载量）
- **CaCO₃**（载体，提供大表面积）
- **Pb(OAc)₂**（毒化剂，部分覆盖 Pd 表面）
- 或加 **喹啉**（额外毒化，增强选择性）

**毒化机理**：
- Pb 原子沉积在 Pd 表面，占据部分活性位点
- 剩余的 Pd 活性位点仍然可以吸附 H₂ 和炔烃
- 但**吸附烯烃的能力大大降低**（烯烃吸附需要更多相邻活性位点）
- 结果：炔烃 → 烯烃 可行；烯烃 → 烷烃 极慢

### 3.2 催化氢化机理

**表面催化机理**：
```
步骤 1：H₂ 在 Pd 表面解离吸附 → 2 H(ads)
步骤 2：炔烃在 Pd 表面吸附
步骤 3：两个 H 原子从同一面逐步转移到炔烃上
步骤 4：syn 加成 → Z-烯烃从表面脱附
```

**syn 加成的原因**：
- H₂ 在 Pd 表面解离后，两个 H 原子都吸附在金属同一侧
- 炔烃也吸附在金属表面
- H 原子从金属表面"滑"到炔烃上 → 两个 H 从同侧加入
- 结果：**Z-烯烃**（顺式）

### 3.3 与 Na/NH₃(l) 的根本区别

| | **Lindlar** | **Na/NH₃(l)** |
|:---|:---|:---|
| **试剂** | H₂ / 毒化 Pd-CaCO₃ | Na / 液氨 |
| **机理** | **表面催化氢化** | **溶解金属还原（SET）** |
| **H 的来源** | H₂ 分子 | NH₃ 质子化 |
| **电子转移** | 两电子（H⁻ 等价物）| 单电子逐步转移 |
| **立体化学** | **syn** → **Z-烯烃** | **anti** → **E-烯烃** |
| **本质** | 催化反应 | 化学计量还原 |

> ⭐ **ABOC 踩坑回流**："Lindlar 和 Na/NH₃ 都是还原炔烃"——只记了结果，机理完全不同。

---

## 四、关键结论

1. **Lindlar = 毒化 Pd = 炔烃 → Z-烯烃**
2. **毒化的本质是降低催化剂活性**——活性刚好够还原三键，不够还原双键
3. **syn 加成 → Z-烯烃**——表面催化的几何必然性
4. **与 Na/NH₃ 机理完全不同**——后者是单电子还原，得 E-烯烃
5. **喹啉可增强选择性**——额外毒化确保不进一步还原

## 五、常见分类或情形

### 5.1 按炔烃类型分类

| 底物 | 产物 | 备注 |
|:---|:---|:---|
| **末端炔** | 端烯（R-CH=CH₂）| 可能过度还原，需控制条件 |
| **内炔（对称）** | 顺式烯烃（Z-RCH=CHR）| 最经典应用 |
| **内炔（不对称）** | 混合 Z-烯烃 | 无区域选择性问题（两边都加 H）|

### 5.2 与其他选择性还原方法对比

| 方法 | 试剂 | 产物 | 特点 |
|:---|:---|:---|:---|
| **Lindlar** | H₂ / 毒化 Pd | **Z-烯烃** | 催化量，经济 |
| **Na/NH₃** | Na / 液 NH₃ | **E-烯烃** | 化学计量，机理不同 |
| **二亚胺（N₂H₂）** | N₂H₂（原位生成）| Z-烯烃 | 无金属，温和 |
| **硼氢化（部分）** | 9-BBN | 烯基硼 → 烯烃 | 反马氏 |

## 六、适用条件与限制

1. **需 H₂ 气氛**——常压或低压氢气即可
2. **严格无水**——毒化 Pd 对水敏感，水可能去除毒化
3. **末端炔可能过度还原**——需控制 H₂ 当量和反应时间
4. **不适用于含硫底物**——硫会毒化 Pd 催化剂（永久失活）
5. **不适用于含硝基底物**——硝基也会被还原
6. **喹啉的臭味**——实验操作需通风

---

## 七、常见比较与易混点

### 7.1 Lindlar vs 普通 Pd/C 催化氢化

| | **Lindlar** | **Pd/C（未毒化）** |
|:---|:---|:---|
| **炔烃还原** | ✅ 可以 | ✅ 可以 |
| **烯烃还原** | ❌ 几乎不 | ✅ 可以 |
| **产物** | 停在烯烃 | 继续到烷烃 |
| **应用** | 选择性半氢化 | 完全氢化 |

### 7.2 Z-烯烃 vs E-烯烃的制备方法

| 目标 | 方法 |
|:---|:---|
| **Z-烯烃** | Lindlar 还原炔烃；催化氢化（部分控制）|
| **E-烯烃** | Na/NH₃ 还原炔烃；Wittig 反应（稳定型叶立德）|

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：3-己炔经 Lindlar 还原后的产物是什么？写出立体化学。

**解答**：
- Lindlar：syn 加成 → **Z-3-己烯**（顺-3-己烯）
- 两个乙基在双键同侧

### 10.2 例题 2（★★ 选择）

**题目**：如何从苯乙炔制备反式-β-甲基苯乙烯？如何制备顺式-β-甲基苯乙烯？

**解答**：
- **顺式（Z）**：Lindlar 还原 → **Z-β-甲基苯乙烯**
- **反式（E）**：Na/NH₃(l) 还原 → **E-β-甲基苯乙烯**

### 10.3 例题 3（★★★ 合成设计）

**题目**：设计从 1-丁炔到顺-2-丁烯的合成路线。

**解答**：
```
1-丁炔  --(Lindlar, H₂)-->  顺-2-丁烯
```
- 注意：1-丁炔是末端炔，Lindlar 还原得 1-丁烯
- 若要得 2-丁烯，需要先异构化... 
- 更正：内炔（2-丁炔）Lindlar 还原才得 2-丁烯
- 若从 1-丁炔出发，需先异构化为 2-丁炔，再用 Lindlar

## 十一、易错点

1. ❌ **Lindlar 得 E-烯烃** → Lindlar 得 **Z-烯烃**（syn 加成）
2. ❌ **与 Na/NH₃ 机理混淆** → 两者机理完全不同
3. ❌ **忽略毒化的必要性** → 未毒化的 Pd/C 会继续还原到烷烃
4. ❌ **末端炔不注意过度还原** → 需控制条件
5. ❌ **含硫底物用 Lindlar** → 硫会使 Pd 永久中毒
