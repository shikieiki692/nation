# 有机还原反应

## 一、定义

**有机还原反应**是指有机分子中碳原子的氧化态降低的反应，通常表现为：
- C-O/C-N/C-X 键减少（脱氧/脱杂原子）
- C-H 键增加（加氢）
- C=C/C≡C 键氢化（π 键断裂）

- **广义定义**：有机化合物中**电子密度升高**的过程（获得电子或向电负性更小的原子转移电子）
- **与氧化反应的关系**：还原和氧化是**成对**的——一个物质被还原，必有另一个物质被氧化
- **核心地位**：还原反应是官能团转化和骨架构建的基本工具

## 三、核心原理

### 3.1 还原反应的分类

| 类型 | 特征 | 典型例子 |
|:---|:---|:---|
| **氢化物还原** | H⁻（亲核）进攻羰基等缺电子中心 | NaBH₄, LiAlH₄ |
| **催化氢化** | H₂ 在金属表面加成 | Pd-C, Pt, Ni |
| **溶解金属还原** | 电子从金属转移到底物 | Na/NH₃(l), Mg/MeOH |
| **均相催化氢化** | 手性配体 + 金属络合物 | BINAP-Ru, Noyori 不对称氢化 |

### 3.2 金属氢化物还原剂的活性比较

金属氢化物通过提供 **H⁻**（氢负离子）进行还原，但活性差异巨大：

| 还原剂 | 还原能力 | 可还原 | 不可还原/需注意 |
|:---|:---:|:---|:---|
| **NaBH₄** | 温和 | 醛、酮、酰氯 | 酯、酰胺、羧酸（不还原）|
| **NaBH₄/CeCl₃（Luche）** | 温和，选择性 | α,β-不饱和酮的 1,2-还原 | 1,4-还原被抑制 |
| **NaBH₃CN** | 温和，pH 选择 | 亚胺（还原胺化）| 对醛/酮在 pH 3-4 反应慢 |
| **LiAlH₄** | 极强 | 醛、酮、酯、羧酸、酰胺、腈 → 醇/胺 | 对硝基、卤代烃可能还原；需无水 |
| **LiAlH(OtBu)₃（LTBA）** | 中等 | 醛、酮；酰氯 → 醛 | 酯不还原 |
| **DIBAL-H** | 可控 | 酯 → 醛（低温）；腈 → 醛（低温）| 高温还原为醇 |
| **Red-Al** | 强 | 类似 LiAlH₄ | 比 LiAlH₄ 更稳定、操作更安全 |
| **BH₃·THF** | 特殊选择性 | 羧酸 → 醇；酰胺 → 胺 | 对酮/醛还原较慢 |

### 3.3 催化氢化的选择性

| 催化剂 | 可氢化 | 通常不氢化 | 特点 |
|:---|:---|:---|:---|
| **Pd-C** | 烯、炔、硝基、苄醚/胺、卤代芳烃 | 羰基、酯、酰胺 | 最强氢化催化剂 |
| **PtO₂（Adams）** | 烯、炔、硝基、羰基 | 苯环 | 酸性条件下活跃 |
| **Raney Ni** | 烯、炔、硝基、腈 → 胺 | 羰基、酯 | 便宜，可氢解脱硫 |
| **Lindlar** | 炔 → **顺式烯** | 烯、羰基 | 毒化 Pd，停止在烯烃 |
| **Crabtree** | 烯（三取代 > 末端）| — | 均相催化剂，无异构化 |

## 四、关键结论

1. **NaBH₄ 和 LiAlH₄ 是最核心的两个氢化物还原剂**：前者温和（只还原醛/酮），后者强（还原几乎所有羰基衍生物）
2. **LiAlH₄ 操作需严格无水**：遇水剧烈反应，释放 H₂
3. **DIBAL-H 是酯 → 醛的专用试剂**：低温（−78°C）控制可停在醛
4. **Lindlar 催化氢化是炔 → 顺式烯的经典方法**：与 Na/NH₃ 的反式还原形成互补
5. **Birch 还原是芳环选择性还原为 1,4-环己二烯的独特方法**：EDG → 取代基在双键碳；EWG → 取代基在饱和碳

## 五、常见分类

### 5.1 羰基化合物的还原

```
羰基还原
├── 醛/酮 → 醇
│   ├── NaBH₄（温和，最常用）
│   ├── LiAlH₄（强，通用）
│   ├── Meerwein-Ponndorf-Verley（MPV）：异丙醇铝，只还原醛/酮
│   ├── 催化氢化（Pd, Pt, Ni）
│   └── 不对称还原：Noyori, CBS, 酶催化
├── 酯 → 醇
│   └── LiAlH₄, Red-Al, BH₃
├── 酯 → 醛（停在醛）
│   └── DIBAL-H（−78°C）
├── 酰胺 → 胺
│   └── LiAlH₄, BH₃
├── 羧酸 → 醇
│   └── LiAlH₄, BH₃
└── 腈 → 胺（一级）
    └── LiAlH₄, 催化氢化
```

### 5.2 不饱和键的还原

| 底物 | 产物 | 试剂 | 选择性 |
|:---|:---|:---|:---|
| **烯烃** | 烷烃 | H₂/Pd-C, H₂/Pt, H₂/Ni | 一般无立体选择性（除非手性催化剂）|
| **炔烃 → 烯烃（顺式）** | 顺式烯 | H₂/Lindlar | 停止在烯烃 |
| **炔烃 → 烯烃（反式）** | 反式烯 | Na/NH₃(l), Li/NH₃(l) | 溶解金属还原 |
| **炔烃 → 烷烃** | 烷烃 | H₂/Pd-C（过量）| 完全氢化 |
| **芳环 → 1,4-二烯** | 1,4-环己二烯 | Na/NH₃(l)/EtOH | Birch 还原 |
| **α,β-不饱和羰基** | 饱和羰基（1,4-还原）| H₂/Pd-C, Na/NH₃ | 共轭还原 |

### 5.3 其他重要还原反应

| 反应 | 底物 | 产物 | 试剂 | 说明 |
|:---|:---|:---|:---|:---|
| **Birch 还原** | 芳环 | 1,4-环己二烯 | Na/NH₃(l)/EtOH | 区域选择性由取代基决定 |
| **Clemmensen 还原** | 酮 | 亚甲基（CH₂）| Zn-Hg/HCl | 强酸条件，对酸敏感基团不适用 |
| **Wolff-Kishner 还原** | 酮 | 亚甲基 | N₂H₄/KOH/高温 | 强碱条件，对碱敏感基团不适用 |
| **黄鸣龙改进** | 酮 | 亚甲基 | N₂H₄/KOH/二乙二醇/Δ | 中式改进，更温和 |
| **金属溶解还原** | 邻二卤代物 | 烯烃 | Zn/酸 | 脱卤 |
| **脱苄基** | 苄醚/苄胺/苄酯 | 游离 OH/NH₂/COOH | H₂/Pd-C | 氢解脱除保护基 |

## 六、适用条件

### 6.1 NaBH₄ 还原的最佳条件

| 因素 | 条件 | 说明 |
|:---|:---|:---|
| **溶剂** | MeOH, EtOH, THF/H₂O | 质子溶剂稳定硼氢化物 |
| **温度** | 0°C → rt | 通常不需加热 |
| **底物范围** | 醛、酮、酰氯 | 酯、酰胺、羧酸不反应 |
| **后处理** | 稀酸淬灭 | 分解过量 NaBH₄ |

### 6.2 LiAlH₄ 还原的最佳条件

| 因素 | 条件 | 说明 |
|:---|:---|:---|
| **溶剂** | 无水 Et₂O, THF | 严格无水！遇水分解剧烈 |
| **温度** | 0°C → rt → 回流 | 根据底物活性调整 |
| **操作** | 氮气保护下缓慢加入 | 安全！
| **底物范围** | 几乎所有羰基衍生物 | 醛/酮/酯/酸/酰胺/腈 |
| **后处理** | 小心！先加 EtOAc，再加水/稀酸 | 顺序不可颠倒 |

### 6.3 DIBAL-H 还原酯为醛的条件

| 因素 | 条件 | 说明 |
|:---|:---|:---|
| **温度** | −78°C | 低温是关键！
| **当量** | 1.0-1.2 当量 | 控制化学计量 |
| **溶剂** | 无水甲苯或 CH₂Cl₂ | 非配位性溶剂 |
| **后处理** | 加 MeOH 或稀酸淬灭 | 生成醛 |
| **升温后果** | 继续还原为醇 | 温度失控导致过度还原 |

### 6.4 Birch 还原的条件与选择性

**条件**：Na（或 Li）/ 液 NH₃ / 醇（EtOH, t-BuOH）

**区域选择性**：

| 取代基类型 | 位置 | 产物结构 | 原因 |
|:---|:---:|:---|:---|
| **给电子基（EDG）** | 间位被还原 | 取代基在双键碳上 | EDG 稳定自由基负离子的间位 |
| **吸电子基（EWG）** | 邻/对位被还原 | 取代基在饱和碳上 | EWG 稳定自由基负离子的邻/对位 |

## 七、常见比较

### 7.1 NaBH₄ vs LiAlH₄

| 维度 | NaBH₄ | LiAlH₄ |
|:---|:---|:---|
| **氢源** | H⁻（来自 B-H）| H⁻（来自 Al-H）|
| **还原能力** | 温和 | 极强 |
| **可还原** | 醛、酮、酰氯 | 醛、酮、酯、羧酸、酰胺、腈 |
| **溶剂** | MeOH, EtOH, THF/H₂O | 严格无水 Et₂O, THF |
| **操作性** | 安全、方便 | 危险（遇水爆炸）、需惰性气氛 |
| **后处理** | 稀酸淬灭 | 复杂（先弱酸/酯，再水）|
| **选择性** | 对醛/酮选择（酯不还原）| 几乎还原所有羰基 |

### 7.2 催化氢化 vs 溶解金属还原（炔烃）

| 维度 | H₂/Lindlar | Na/NH₃(l) |
|:---|:---|:---|
| **炔 → 烯** | **顺式烯烃** | **反式烯烃** |
| **机理** | 顺式加成（syn）| 电子转移 → 自由基负离子 → 反式质子化 |
| **过度还原** | 被毒化阻止 | 通常停在反式烯 |
| **其他影响** | 可能异构化双键 | 可能还原芳环（Birch）|

## 十、例题

### 例题 1：羧酸衍生物的选择性还原

**题目**：如何将乙酸乙酯还原为乙醛（而不是乙醇）？

**解析**：

**正确答案：DIBAL-H，−78°C**

$$\ce{CH3COOEt + DIBAL-H ->[-78°C, 甲苯] CH3CHO + EtOH（来自 DIBAL-H 的烷氧基）}$$

**机理**：
1. DIBAL-H 的 Al-H 进攻酯羰基 → 形成四面体中间体
2. 低温下中间体稳定（不坍塌）
3. 淬灭时（加 MeOH 或弱酸）→ 中间体分解为醛

**为什么 LiAlH₄ 不行**：LiAlH₄ 太强，即使在低温也会将酯直接还原为醇。

### 例题 2：Birch 还原的区域选择性

**题目**：预测苯甲醚（Anisole）和苯甲酸在 Birch 还原中的产物。

**解析**：

**苯甲醚（EDG：OCH₃）**：
- OCH₃ 是给电子基 → 还原发生在**间位**
- 产物：**1-甲氧基-1,4-环己二烯**（甲氧基在双键碳上）

**苯甲酸（EWG：COOH）**：
- COOH 是吸电子基 → 还原发生在**邻/对位**
- 产物：**1,4-环己二烯-1-甲酸**（COOH 在饱和碳上，即 2,5-环己二烯-1-甲酸）

**记忆法**：
- EDG → "想保持双键" → 在双键碳上
- EWG → "不想有双键" → 在饱和碳上

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| "NaBH₄ 可以还原酯" | 与 LiAlH₄ 混淆 | NaBH₄ **不能**还原酯、酰胺、羧酸 | NaBH₄ 和 LiAlH₄ 对酯的反应有何不同？ |
| "LiAlH₄ 可以用水溶液" | 忽略其遇水反应 | LiAlH₄ **严格无水**，遇水剧烈放氢甚至爆炸 | 为什么 LiAlH₄ 后处理要先加 EtOAc 再加水？ |
| "DIBAL-H 任何温度都得醛" | 忽略温度控制 | DIBAL-H **−78°C** 得醛；升温继续还原为醇 | DIBAL-H 还原酯时温度控制为什么关键？ |
| "催化氢化总是 syn 加成" | 忽略溶解金属还原 | H₂/Pd 是 syn；Na/NH₃ 是 **反式** | 炔烃还原为顺式烯和反式烯各用什么试剂？ |
| "Birch 还原给全氢化产物" | 与催化氢化混淆 | Birch 还原只给 **1,4-二烯**，不是全氢化 | Birch 还原和 H₂/Pd 还原芳环的产物有何不同？ |

### 12.3 入门例题

**最简试剂选择**：
> 选择合适试剂：1. 丙酮 → 异丙醇；2. 乙酸乙酯 → 乙醇；3. 苯 → 1,4-环己二烯

**答案**：
1. NaBH₄（温和、方便）
2. LiAlH₄（还原酯）
3. Na/NH₃(l)/EtOH（Birch 还原）
**引导**：问"为什么第三题不用 H₂/Pd？"（H₂/Pd 给环己烷，不是 1,4-二烯）

### 12.4 现实类比

**"拆弹 vs 装修"类比还原反应**：
- **LiAlH₄** = 重型推土机——力量巨大，什么墙都能推（还原所有羰基），但操作危险（遇水爆炸），需要专业人员
- **NaBH₄** = 家用电钻——安全方便，能打普通孔（醛/酮），但打不穿混凝土（酯/酰胺）
- **DIBAL-H** = 精密手术刀——可以精确地只切一层皮（酯 → 醛），但手一抖（升温）就切到肉（继续还原为醇）
- **H₂/Pd** = 自动喷漆机——均匀地给墙面刷漆（氢化），但不能选择性只刷一半
- **Lindlar** = 半自动喷漆——刷到一半自动停止（停在烯烃），不会刷满（烷烃）
- **Na/NH₃** = 反向喷涂——从背面喷，效果相反（反式烯烃）
- **Birch 还原** = 局部装修——只翻新客厅（芳环 → 1,4-二烯），不碰卧室（保留部分不饱和度）

## 十三、竞赛拓展

### 13.1 现代还原方法

| 方法 | 特点 | 竞赛价值 |
|:---|:---|:---|
| **Noyori 不对称氢化** | BINAP-Ru/Rh 催化，不对称还原酮/烯 | 2001 诺奖，不对称合成 |
| **Corey-Bakshi-Shibata（CBS）** | 硼烷 + 手性恶唑硼烷，不对称还原酮 | 高效、高 ee |
| **转移氢化** | 异丙醇/甲酸为氢源，无需 H₂ 气 | 安全、方便 |
| **酶催化还原** | 醇脱氢酶，极高选择性 | 生物有机 |
