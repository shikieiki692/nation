# 硼氢化反应（Hydroboration）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[37-加成反应]]、[[43-亲电加成反应]]

---

## 一、定义

**硼氢化反应（Hydroboration）**：烯烃或炔烃与硼烷（BH₃ 或衍生物）发生的**顺式加成**反应，生成**烷基硼化合物**。后续可通过氧化（H₂O₂/NaOH）转化为醇，或通过还原（RCOOH）转化为烷烃。

**本质**：非鎓离子路径的**协同反应**，经历**四中心过渡态**。与鎓离子机理形成鲜明对比。

> ⭐ **ABOC 行 2566**："Brown 硼氢化反应：烯烃和乙硼烷或其他含有 B-H 键的硼烷在低温下反应，经历'四中心过渡态'顺式加成得到烷基硼化合物。受电子效应影响——B 为正电性，硼氢化采取反马氏加成。"

## 三、核心原理

### 3.1 四中心过渡态

硼氢化是**一步协同反应**（concerted），没有中间体：

```
        H
         \
    C = C   +   B-H   →   [四中心过渡态]   →   -CH-CH-
         /                              |    |
        H                              BH₂  H
```

**四中心过渡态特征**：
| 特征 | 说明 |
|:---|:---|
| **协同性** | C-H 键形成与 C-B 键形成同时发生 |
| **立体化学** | **顺式加成**（syn）——B 和 H 从双键同侧加入 |
| **区域选择性** | **反马氏**——B 加到少取代碳，H 加到多取代碳 |
| **电子驱动** | B 是正电性（缺电子），H 是负电性（氢负离子特征）|

### 3.2 为什么是反马氏？——"反直觉"的本质

传统马氏规则的思维是 "H⁺ 加到多取代碳"。但硼氢化中：

- **B 原子是正电性的**（价层只有 6 电子，缺电子）
- **H 原子是负电性的**（B-H 键极化：B^δ⁺-H^δ⁻）
- 因此：**B（正电部分）寻找电子丰富的双键碳 → 少取代碳**（因烷基给电子使多取代碳电子密度更高）
- **H（负电部分）加到多取代碳**

> 关键记忆：**硼氢化 = 反马氏**，因为 B 带正电，不是 H 带正电。

### 3.3 硼烷试剂家族

| 试剂 | 结构特点 | 选择性 | 典型应用 |
|:---|:---|:---|:---|
| **BH₃·THF** | 单体 BH₃（实际以 B₂H₆ 二聚体存在，THF 解聚）| 位阻小，反应快 | 通用硼氢化 |
| **B₂H₆** | 乙硼烷，二聚体 | 与 BH₃ 等价 | 实验室常用 |
| **9-BBN** | 双环[3.3.1]壬烷-9-硼烷，大位阻 | **高区域选择性**，优先位阻小的双键 | 复杂底物 |
| **Sia₂BH** | 二异松蒎基硼烷 | 极高的区域/对映选择性 | 不对称合成 |
| **catecholborane** | 邻苯二酚硼烷 | 温和，可室温反应 | 官能团兼容性 |

**9-BBN 的位阻效应**：
- 大位阻硼烷不能接近多取代双键 → 优先与**末端/少取代双键**反应
- 当分子中有多个双键时，9-BBN 可选择性硼氢化位阻最小的那个

### 3.4 氧化阶段——烷基迁移

烷基硼 → 醇 的转化机理：

```
步骤 1：HOO⁻ 进攻 B（B 缺电子）→ 硼酸过氧负离子中间体
步骤 2：烷基从 B 迁移到相邻的 O（亲核重排）
步骤 3：OH⁻ 进攻 → 断裂 O-O 键 → 烷氧基硼
步骤 4：水解 → 醇 + 硼酸
```

**烷基迁移的本质**：
- B 的价层电子只有 6 个（不满足八隅体），非常不稳定
- B 总是"想方设法把电子推出去"
- 当 O-O 键被活化后，烷基带着电子对迁移到 O 上，B 的氧化态升高
- **迁移保持构型**（立体化学在 C 上保留）

### 3.5 还原阶段

烷基硼 + 羧酸（如丙酸）→ **烷烃**

- 机理：类似氧化，但 H 从羧酸迁移到烷基碳
- 总结果：C=C → C-C（加氢还原）
- 立体化学：**保持构型**（syn 加成的 H 和 B，B 被 H 取代）

## 四、关键结论

1. **硼氢化 = 四中心过渡态 = 顺式加成 + 反马氏**——与鎓离子机理形成鲜明对比
2. **反马氏的本质是 B^δ⁺ 进攻**——不是 H^δ⁺，所以和马氏规则相反
3. **氧化时的烷基迁移保持构型**——整个硼氢化-氧化序列：syn 加成 + 构型保持 → 得特定立体化学的醇
4. **9-BBN 用于位阻选择性**——大位阻硼烷优先反应位阻小的双键
5. **炔烃硼氢化**：末端炔 → 醛（烯醇互变），内炔 → 酮
6. **硼氢化可逆**——高温下烷基硼可脱氢恢复烯烃（竞赛高阶内容）

## 五、常见分类或情形

### 5.1 按底物分类

| 底物 | 产物（氧化后）| 马氏/反马氏 | 立体化学 |
|:---|:---|:---:|:---:|
| **末端烯烃** R-CH=CH₂ | R-CH₂-CH₂OH（伯醇）| **反马氏** | 顺式加成 |
| **内烯烃** R₂C=CHR | R₂CH-CH(OH)R（仲醇）| **反马氏** | 顺式加成 |
| **末端炔烃** R-C≡CH | R-CH₂-CHO（醛）| **反马氏** | 顺式加成 |
| **内炔烃** R-C≡C-R' | R-CO-CH₂-R'（酮）| **反马氏** | 顺式加成 |

### 5.2 按后续处理分类

| 处理条件 | 产物类型 | 关键机理步骤 |
|:---|:---|:---|
| **H₂O₂ / NaOH** | 醇 | 烷基迁移（B→O），构型保持 |
| **RCOOH** | 烷烃 | H 迁移，C-B 键断裂 |
| **H₂O / Δ** | 烯烃（可逆）| 脱硼氢化，恢复双键 |

## 六、适用条件与限制

1. **反应通常在无水条件下进行**——BH₃ 与水剧烈反应
2. **对空气敏感**——硼烷试剂需在惰性气氛下操作
3. **位阻极大时反应变慢**——可用 9-BBN 解决选择性问题
4. **含活泼氢的官能团需保护**——OH、NH、COOH 会与 BH₃ 反应
5. **炔烃硼氢化需控制当量**——过量 BH₃ 会导致二次硼氢化
6. **硼氢化不可用于共轭二烯的选择性单硼氢化**（除非用位阻大的硼烷）

## 七、常见比较与易混点

### 7.1 硼氢化 vs 鎓离子加成（以 Br₂ 为例）

| | **硼氢化** | **Br₂ 加成** |
|:---|:---|:---|
| **机理** | 协同，四中心过渡态 | 分步，鎓离子中间体 |
| **立体化学** | **顺式加成**（syn）| **反式加成**（anti）|
| **区域选择性** | **反马氏** | **马氏**（多取代一边开环）|
| **中间体** | 无 | 溴鎓离子 |
| **碳正离子重排** | 无（协同）| 通常无（鎓离子抑制重排）|

### 7.2 硼氢化-氧化 vs 烯烃水合（Hg²⁺ 或酸催化）

| | **硼氢化-氧化** | **酸催化水合** | **汞鎓离子-脱汞** |
|:---|:---|:---|:---|
| **区域选择性** | **反马氏** | 马氏 | 马氏 |
| **立体化学** | **syn** | 无特定 | **anti**（汞鎓离子背面进攻）|
| **重排** | 无 | 有（碳正离子）| 无（鎓离子）|
| **官能团耐受性** | 一般 | 较好 | 较好 |

### 7.3 末端炔烃硼氢化的产物辨析

**常见错误**：认为炔烃硼氢化-氧化得醛（因为烯烃得伯醇）。

**正确**：
- 末端炔 → 烯基硼 → 氧化得 **烯醇** → 互变异构 → **醛** ✓
- 但 **内炔** → 烯基硼 → 氧化得烯醇 → 互变异构 → **酮**（不是醛）

| 炔烃类型 | 硼氢化-氧化产物 | 最终产物 |
|:---|:---|:---|
| 末端炔 R-C≡CH | R-CH=CHOH（烯醇）| **醛** R-CH₂-CHO |
| 对称内炔 R-C≡C-R | R-CH=C(OH)-R（烯醇）| **酮** R-CO-CH₂-R |
| 不对称内炔 | 反马氏选择 | 酮（混合物）|

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：1-甲基环己烯经硼氢化-氧化后的主产物是什么？说明立体化学。

**解答**：
- 反马氏：B 加到双键的次甲基碳（C1，少取代），H 加到叔碳（C2，多取代）
- 但环状体系中，syn 加成意味着 B 和 H 从双键同侧加入
- 氧化后 B 变为 OH
- 产物：**反-2-甲基环己醇**（注意：这里 syn 加成导致 H 和 OH 为顺式关系）
- 实际产物是 **顺-2-甲基环己醇**（H 和 OH 同侧），但由于是反马氏，OH 在 C1，H 在 C2 同侧加入

> 更正：对于 1-甲基环己烯，双键在 C1-C2 之间。B 加到 C2（少取代，因为 C1 有甲基），H 加到 C1。syn 加成后，H 在 C1 与 B 在 C2 同侧。氧化后 B→OH，所以 C1 上的 H 和 C2 上的 OH 是 **反式关系**（因为双键两面，syn 加成都从上或都从下）。
> 实际为 **反-2-甲基环己醇**（trans-2-methylcyclohexanol）。

### 10.2 例题 2（★★ 区域选择性）

**题目**：3-甲基-1-戊烯（CH₂=CH-CH(CH₃)-CH₂-CH₃）与 9-BBN 反应后再氧化，主产物是什么？

**解答**：
- 双键为末端烯烃（CH₂=CH-）
- 9-BBN 大位阻不影响末端烯烃的选择
- 反马氏：B 加到末端 CH₂，H 加到内部 CH
- 氧化后：B → OH
- 产物：**3-甲基-1-戊醇**（伯醇，OH 在末端）

### 10.3 例题 3（★★★ 烷基迁移机理）

**题目**：解释烷基硼氧化时为什么发生烷基迁移，而不是 B-C 键直接断裂。

**解答**：
- B 的价层只有 6 电子，不满足八隅体 → 极不稳定
- 硼氢化后 B 仍缺电子，渴望获得电子对
- HOO⁻ 进攻 B 后，O-O 键极化
- **烷基带着电子对迁移到 O 上**：B 从三价变为四价（硼酸酯），获得稳定
- 同时 C 的构型保持（迁移是分子内同面过程）
- 本质：B "想方设法把电子推出去"，通过烷基迁移实现自身稳定化

### 10.4 例题 4（★★★ 炔烃硼氢化）

**题目**：苯乙炔（Ph-C≡CH）经 (Sia)₂BH 处理后氧化，产物是什么？若用 Hg²⁺/H₂O 呢？

**解答**：
- **硼氢化路径**：(Sia)₂BH 是位阻很大的硼烷，只与末端炔反应一次（不会二次加成）
  - 反马氏：B 加到末端 C，H 加到 Ph 相连的 C
  - 氧化得烯醇：Ph-CH=CHOH → 互变异构 → **苯乙醛**（Ph-CH₂-CHO）
- **汞鎓离子路径**：Hg²⁺/H₂O → 马氏水合
  - OH 加到 Ph 相连的 C（多取代/苄位稳定）
  - 产物：**苯乙酮**（Ph-CO-CH₃）
- 两者产物完全不同，体现了反马氏 vs 马氏的选择性差异

## 十一、易错点

1. ❌ **认为硼氢化是马氏加成** → B 是正电性，H 是负电性，所以是**反马氏**
2. ❌ **忘记 syn 加成** → 硼氢化是**顺式加成**，产物中 H 和 OH 来自同侧
3. ❌ **混淆氧化和还原的后续处理** → H₂O₂/NaOH = 醇；RCOOH = 烷烃
4. ❌ **认为炔烃硼氢化都得醛** → 只有**末端炔**得醛，**内炔**得酮
5. ❌ **忽略 9-BBN 的位阻选择性** → 9-BBN 优先反应位阻小的双键
6. ❌ **画机理不写四中心过渡态** → 必须画出协同的环状过渡态
7. ❌ **认为硼氢化经历碳正离子** → **没有碳正离子**，是协同机理
8. ❌ **忽略烷基迁移的构型保持** → 氧化后手性中心的构型保持不变

### 12.3 入门级例题

**题目**：写出 1-己烯硼氢化-氧化的产物和反应机理要点。

**预期解答路径**：
1. 1-己烯（CH₂=CH-CH₂CH₂CH₂CH₃）+ BH₃ → 四中心过渡态
2. B 加到末端 CH₂（反马氏），H 加到内部 CH
3. syn 加成：B 和 H 从同侧加入
4. H₂O₂/NaOH 氧化：烷基迁移，B → OH
5. 产物：**1-己醇**（CH₃CH₂CH₂CH₂CH₂CH₂OH）

**老师可引导的提问**：
- 卡在第 2 步：问 "B 和 H 哪个带正电？应该加到哪个碳？"
- 卡在第 4 步：问 "为什么是烷基迁移，而不是 B-C 键直接水解？"

### 12.4 与现实/直觉的连接

- **类比**：硼氢化的"反直觉"很像磁铁——你以为 H（像质子）应该带正电，但实际上在 B-H 键中 H 是负电端（氢负离子特征）。这就像把一个磁铁的南极涂成红色、北极涂成蓝色，结果它吸引的方向和你预期相反。
- **反例**：直觉认为"所有亲电加成都是 H⁺ 先进攻"。硼氢化打破了这一直觉——B 才是亲电中心，H 是亲核中心。

## 十三、竞赛拓展

1. **硼氢化的可逆性（36 届决赛）**：
   - 高温下烷基硼可发生 **脱硼氢化（dehydroboration）**，恢复烯烃
   - 机理：β-氢消除，类似 E2 但发生在 B-C 键上
   - 可逆性使得硼氢化可以作为烯烃异构化的工具

2. **不对称硼氢化**：
   - 使用手性硼烷（如 IpcBH₂）可以实现对映选择性硼氢化
   - 是制备手性醇的重要方法

3. **硼氢化在合成中的策略价值**：
   - 烯烃 → 醇（反马氏）与 烯烃 → 醇（马氏，通过汞鎓/酸催化）形成互补
   - 合成设计中常用"硼氢化-氧化"构建末端醇或特定立体化学的醇
