# 反式共平面（antiperiplanar / trans-coplanar）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[22-分子轨道理论]]、[[28-有机物结构的稳定性]]、[[32-有机反应机理]]、[[38-消除反应]]

## 一、定义

**反式共平面（antiperiplanar / trans-coplanar）**：在一个 X-A-B-Y 四原子链中，X-A 键与 B-Y 键的**二面角为 180°**（即 X 与 Y 处于 A-B 键的相反侧、并且 X-A-B-Y 四个原子共面）的几何关系。

```
        X
        │
       (A)──(B)
                │
                Y    ← 二面角 X-A-B-Y = 180°
```

> ⭐ **ABOC 教材在 §1.2.2 强调三次**：
> > "请不要对'反式共平面'这个词进行缩写！！"
>
> 这是因为：① 缩写为"反共面"会与"对位"语义混淆；② 中文常见错误简化为"反式平面"丢失"共"字（共线的几何含义）；③ 这是有机化学中最频繁使用的立体电子术语，必须保持完整以维持精度。

**重要的几何同义词**：
- 反式共平面（antiperiplanar）：二面角 180°
- 顺式共平面（synperiplanar）：二面角 0°
- 邻交叉（gauche / synclinal）：二面角 ±60°
- 反交叉（anticlinal）：二面角 ±120°
- 垂直（orthogonal / perpendicular）：二面角 90°（通常用于 sp³ 与 sp² 体系的分析）

## 三、核心原理

### 3.1 物理本质：成键-反键的最大波瓣重叠

> 为什么反式共平面下"σ 给体 + σ\* 受体"重叠最大？

考察 H-C-C-F 体系中 H-Cσ 与 C-Fσ\* 的相互作用：

```
       占据轨道 σ(H-C)            空轨道 σ*(C-F)
       
       ●●●●● ── ●●●           ●●● ── ●●●●●
        H        C              C       F
       (大波瓣)  (小波瓣)        (小波瓣)  (大波瓣)
```

- σ\*(C-F) 反键轨道**最大波瓣的位置在成键的后方**（即 C 那一侧的"后方"）
- σ(H-C) 成键轨道则在 H 那一侧主导

**反式共平面（H 与 F 二面角 180°）**：
- σ(H-C) 的 H 端波瓣指向"远离 C-F"方向 → 与 σ\*(C-F) 的 C 端"后方"波瓣**朝同一方向** → 完美重叠

**顺式共平面（H 与 F 二面角 0°）**：
- σ(H-C) 的 H 端波瓣与 σ\*(C-F) 的 C 端"后方"波瓣**朝相反方向** → 必须用 σ(H-C) 的"后方小波瓣"重叠 → 重叠极差

**垂直（90°）**：
- 节面与轨道波瓣不匹配 → 完全无相互作用

> **结论**：相互作用强度排序：**反式共平面 > 邻交叉 > 顺式共平面 > 垂直**

### 3.2 二面角与重叠程度的定量关系

> 重叠积分 $S$ 随二面角 $\theta$ 的变化大致为 $S \propto \cos^2\theta$ 形式：

| 二面角 | $\cos^2\theta$ | 相对重叠（vs 反式共平面）|
|:---:|:---:|:---:|
| 0°（顺式共平面）| 1.0 | 较弱（小波瓣相邻）|
| 30° | 0.75 | 0.75 |
| 60°（邻交叉）| 0.25 | 0.25 |
| 90°（垂直）| 0.0 | 0 |
| 120° | 0.25 | 0.25（反向）|
| 150° | 0.75 | 0.75（反向）|
| **180°（反式共平面）** | **1.0** | **1.0（最大）**|

> 实际相互作用强度还要看"波瓣方向"——反式共平面胜过顺式共平面是因为后者要用"小波瓣"。

### 3.3 反式共平面的"动力学放大效应"

> 当反应过渡态的几何要求是反式共平面时，构象差异会被**指数级放大**：

| 反应类型 | 反式共平面 vs 非 | 速率比 |
|:---|:---|:---:|
| [[E2反应]] | β-H 与离去基的二面角 | ~10²-10⁴（环刚性下放大）|
| [[β-硅基效应]] | β-Si 与正电荷 p 轨道 | **~10¹²**（Lambert 实验）|
| 异头效应 | n(O) 与 σ\*(C-O) | ~10²-10³（构象偏好显著）|
| 端基效应 | n(N/O) 与 σ\*(N-Z) | ~10²-10³ |

> **教学要点**：反式共平面不是"装饰性的几何要求"，而是**反应能否发生**的关键判据。

## 四、关键结论

### 4.1 反式共平面是给体-受体相互作用的"地基"

> ABOC 把所有传统的电子效应都归到 [[给体-受体相互作用]] 框架下；而它们**全部要求反式共平面或类似几何**：

| 效应名 | 给体 | 受体 | 反式共平面要求 |
|:---|:---|:---|:---:|
| 普通超共轭 | σ(C-H) | σ\*(C-X) / p / π\* | ✓ |
| **β-硅基效应** | σ(C-Si) | p(C⁺) | ✓（Lambert 实验：10¹² 倍）|
| **端基效应** | n(N/O) | σ\*(N-Z) / σ\*(C-O) | ✓（决定酰胺平面性）|
| **异头效应** | n(O) | σ\*(C-O) | ✓（控糖类构象）|
| **E2 反应** | β-Hσ | LG σ\*（即将断键）| ✓（反式比顺式快 ~10⁴）|
| **α 效应**（反例）| n | n（另一对孤对电子）| ✗（避免反式共平面，构象逃避）|
| n→π\* 共轭 | n | π\* | 类似要求（lp 与 π 平面平行）|
| π→σ\*（部分 σ-π）| π(C=C) | σ\*(C-X) | ✓（共平面）|

### 4.2 几何识别的三个关键问题

> 看到一个分子要判断"反式共平面是否成立"，按这三个问题：

1. **找两根关键键**：哪两根键参与给体-受体相互作用？（如 σ(C-Si) 与 σ\*(C-X)、或 n(N) 与 σ\*(C-O) 等）
2. **画 Newman 投影或锯架式**：沿着两根键之间的中央 C-C（或 C-X / N-C 等）投影
3. **测量二面角**：起点轨道（给体）与终点轨道（受体）的"波瓣朝向"是否在同一直线？

> **简化口诀**：「画纽曼，量角度，180° 才行」

### 4.3 环系中的反式共平面：椅式构象的力量

> 在环己烷椅式构象中，反式共平面的几何由**取代基的轴/赤道位**直接决定：

| 取代基位置组合 | 二面角 | 反式共平面？ |
|:---:|:---:|:---:|
| 1-轴 + 2-轴（trans-1,2-双轴）| **180°** | ✅ |
| 1-轴 + 2-赤道 | 60° | ✗ |
| 1-赤道 + 2-轴 | 60° | ✗ |
| 1-赤道 + 2-赤道（trans-1,2-双赤道）| 60° | ✗ |

> **椅式构象的"反式共平面"只在 1,2-双轴（双 ax）配置下才成立**——这正是环己基 E2 反应、β-硅基效应、异头效应等的几何基础。

#### Lambert 1987 实验（[[β-硅基效应]]）

| β-Si 与离去基的几何 | 相对速率 |
|:---|:---:|
| 普通环己基（无 β-Si）| 1 |
| β-Si 双赤道（gauche, 60°）| ~10² |
| β-Si **双轴（180°，反式共平面）** | **~10¹²** |

### 4.4 双键体系中的反式共平面：sp² 体系特有的"共面"

> 在 sp² 体系中，"反式共平面"含义略变：要求轨道与 π 体系的**节面同向**或**π 平面共面**。

例：n(N) → π\*(C=O) 共轭（酰胺）：
- N 上的 lp 轨道必须与 C=O 的 π 平面**平行**（即 lp 处于 C-N-C=O 平面内）
- 几何条件：N 上的 p 轨道（含 lp）与 π(C=O) 共平面 → "广义反式共平面"

### 4.5 反式共平面的"反例"：α 效应

> 反式共平面在大多数情况下**有利**（增加重叠），但在 [[α 效应]] 中却**不利**：

- 当两对孤对电子相邻处于反式共平面 → "反键轨道上升能量 > 成键轨道下降能量" → **总能量上升**
- 因此 H₂O₂、肼 NH₂NH₂、N₂F₂ 等**主动避开**反式共平面，选择邻交叉构象

> **教学陷阱**：当看到"反式共平面"时，要先判断它是 **"给体 + 受体（空轨道）"**（→ 有利）还是 **"两个占据轨道"**（→ 不利）。

## 五、常见分类或情形

### 5.1 按几何状态分类

| 二面角 | 名称 | 中文别名 | 备注 |
|:---:|:---|:---|:---|
| 0° | synperiplanar | 顺式共平面 / 重叠 | 重叠程度小（小波瓣相邻）|
| ±60° | gauche / synclinal | 邻交叉 / 顺旁 | 中度重叠 |
| 90° | orthogonal | 垂直 / 正交 | 0 重叠 |
| ±120° | anticlinal | 反旁 | 中度反向 |
| **180°** | **antiperiplanar** | **反式共平面** | **最大重叠**（重要！）|

### 5.2 按反应类型分类

#### A. 协同反应（要求严格反式共平面）
- [[E2反应]]：β-H 与离去基反式共平面
- [[Wittig重排]]：σ 与 σ\* 反式共平面
- [[Grob碎裂化反应]]：键的同时断裂要求两根键反式共平面
- [[1,4-消除]]：sp² 体系中的"广义反式共平面"

#### B. 离子型反应（反式共平面提供"动力学加速"）
- [[β-硅基效应]]：σ(C-Si) 与 p(C⁺) 反式共平面 → 加速 10⁹-10¹²
- [[Wagner-Meerwein重排]]：迁移基团与正电荷 p 轨道反式共平面
- [[邻基参与]]：邻位 lp 与离去基反式共平面

#### C. 构象选择（反式共平面决定主导构象）
- [[异头效应]]：糖环中 n(O) → σ\*(C-O) 反式共平面（不论环外烷氧基为 α-/β-异构）
- [[端基效应]]：酰胺平面性的反式共平面来源
- 含氮杂环的 nNσ\*CN 相互作用决定 N 的几何（平面 vs 棱锥）

### 5.3 按"反式共平面有利 vs 不利"分类

| 类型 | 给体 | 受体 | 反式共平面是否有利 |
|:---|:---|:---|:---:|
| n → 空 p（配位/共轭）| 占据 | 空 | **有利** |
| n → σ\*（端基效应）| 占据 | 反键 | **有利** |
| σ → π\* / σ → σ\* | 占据 | 反键 | **有利** |
| **n → n（α 效应）** | 占据 | 占据 | **不利**（避开）|

## 六、适用条件与限制

### 适用条件
- ✓ 两根键有明确的二面角（通常需要 sp³ 中心或 sp²-sp³ 接合）
- ✓ 给体能级 < 受体能级（基础 [[给体-受体相互作用]] §3.2）
- ✓ 节面与波瓣方向一致（反式共平面下波瓣才能重叠）

### 不适用 / 限制
- ✗ 完全孤立的两根 σ 键之间（无给体-受体特征）→ 反式共平面只是"几何描述"
- ✗ 两个 lp 之间（α 效应）→ 反式共平面**不利**，分子主动避开
- ✗ 节面阻断的体系（如苯基与垂直 sp 轨道）→ 即使共平面也无重叠

## 七、常见比较与易混点

### 1. 反式共平面 vs 顺式共平面（180° vs 0°）

| 项 | 反式共平面（180°）| 顺式共平面（0°）|
|:---|:---|:---|
| 几何 | 两键朝相反方向 | 两键朝相同方向 |
| 重叠 | 最大波瓣相遇 | 仅小波瓣相遇 |
| 在反应中 | **必备**（典型反应几何要求）| 通常不被采用 |
| 应力 | 低（无邻位作用力）| 高（取代基相眼）|

### 2. 反式共平面 vs 反式（trans / E）

> ⚠️ **核心混淆点**：

| 项 | 反式共平面 | 反式（trans/E）|
|:---|:---|:---|
| 含义 | 二面角 = 180° | 双键两侧大基团位置相反 |
| 适用对象 | sp³ 链中的两根键 | C=C 双键的取代基相对位置 |
| 示例 | H-C-C-Cl 的 H、Cl 反式共平面 | (E)-2-丁烯：两个 CH₃ 反式 |

> **反式共平面**强调"二面角"和"轨道波瓣方向"；**反式（trans）**强调"双键两侧的取代基"。两者**不可互换**。

### 3. 反式共平面 vs 共面（coplanar）

| 项 | 反式共平面 | 共面（coplanar）|
|:---|:---|:---|
| 含义 | 二面角 180° 且四原子共面 | 多个原子在同一平面 |
| 焦点 | 两键的"反向"几何 | 多原子的"平面"性 |
| 典型应用 | E2、β-Si、端基效应 | 共轭体系平面性、芳香性 |

### 4. 为什么不是"反式平行"或"反共面"？

> ABOC §1.2.2 强调**完整的"反式共平面"**：

| 错误简化 | 问题 |
|:---|:---|
| "反式平面" | 丢失了"共"——不强调四原子共面 |
| "反共面" | 与"对位"中文混淆 |
| "trans" | 失去了 sp³ 二面角的精确含义 |
| "antiperiplanar" 简称 | 全称中"peri-"指"相邻"，不能省 |

### 5. 反式共平面 vs 反式椅式（trans-decalin）

> 不要把"反式共平面"误用为"反式椅式"——后者是稠环的几何关系。

## 十、例题

### 例题 1（基础识别题）

**题目**：在以下分子的 Newman 投影中，找出所有反式共平面的"σ 给体 + σ\* 受体"对。

$$\mathrm{(CH_3)_2CH\!-\!CH(F)\!-\!CH_3}$$（异丁基氟化物）

考虑两个甲基与 F 之间的相互作用。

**分析**：
- 沿着 C₂-C₃ 键作 Newman 投影（前 C 是 CH(F)CH₃，后 C 是 CHMe₂）
- 主要给体：σ(C-H) 键（每个甲基有 3 个 H）
- 主要受体：σ\*(C-F) 反键轨道
- 找哪个 H 与 F 的二面角 180°（反式共平面）

**解答**：
- 在 antiperiplanar 构象中，前 C 的 F 轴向"上"，后 C 的某根 C-H σ 与 F 反式共平面 → 该 σ(C-H) → σ\*(C-F) 是有利的超共轭
- 前 C 的 CH₃ 与 F 也参与超共轭，需依据具体构象

**反思**：反式共平面 σ→σ\* 是典型的超共轭分析步骤，判断时要写出 Newman 投影。

### 例题 2（ABOC §1.2.2 例题改编）

**题目**：H-C-C-F 体系中，为什么超共轭只考虑 H-C → C-F σ\*，而不能反向考虑 F-C → H-C σ\*？

**分析**：
- 给体看能级：σ(H-C) 较高（电负性差异），σ(F-C) 较低（被 F 拉得很深）→ σ(H-C) 是更好的给体
- 受体看能级：σ\*(C-F) 较低（F 电负性大），σ\*(C-H) 较高 → σ\*(C-F) 是更好的受体
- 几何：H 与 F 反式共平面时，σ(H-C) 大波瓣 + σ\*(C-F) 大波瓣 朝同一方向 → 重叠最大

**解答**：
- "正向"（H-C → C-F σ\*）：能量差小、几何匹配 → 强有利
- "反向"（F-C → C-H σ\*）：能量差大、几何匹配的不是"大波瓣对大波瓣" → 弱
- **结论**：超共轭**不是相互的**——电负性差异使其变得"单方向"

> 这与传统"超共轭对等"描述不同，是 ABOC 的核心修正。

**反思**：本题是 ABOC §1.2.2 的关键案例，必须掌握"反式共平面 + 电负性差异 + 单方向超共轭"三位一体的分析方法。

### 例题 3（环己基 E2 立体化学题）

**题目**：cis-1-甲基-2-溴环己烷与 Br⁻ 在 EtO⁻ 作用下发生消除——给出主产物及其立体化学。说明为什么用 trans 异构体反应快得多。

**分析**：
- 环己基 E2 要求 β-H 与 Br 反式共平面
- 在椅式构象中，反式共平面只在 **1,2-双轴**（双 ax）配置下成立
- cis-1-甲基-2-溴：1-Me、2-Br 在环上同侧——必为一轴一赤道（永远不反式共平面）
- trans-1-甲基-2-溴：1-Me、2-Br 在环上异侧——可同时双轴或双赤道；双轴构象下满足反式共平面

**解答**：
- **trans 异构体**：可在双轴构象下进行 E2 → 反应快
- **cis 异构体**：永远无法实现 1,2-双轴 → 反式共平面失败 → E2 慢得多（10²-10⁴ 倍）
- 主产物：1-甲基环己烯（脱去 H 在 6 位，也满足反式共平面）

**反思**：环己基 E2 是反式共平面**动力学放大效应**的经典考点——速率差直接来自"是否能形成反式共平面构象"。

## 十一、易错点

| 错误 | 原因 | 纠正 |
|:---|:---|:---|
| 把"反式共平面"缩写为"反共面"或"反共平面" | 中文习惯简化 | **ABOC 明确反对**——保持完整术语 |
| 把"反式共平面"等同于"反式（trans/E）"| 都有"反式"二字 | 反式共平面是**几何关系**（180°）；反式（trans）是**构型关系**（取代基相对位置）|
| 忘记顺式共平面（0°）也是"共平面" | 把"共平面" = "反式共平面" | 顺式共平面（synperiplanar）= 0°，反式共平面（antiperiplanar）= 180°，**两者不同** |
| 在 sp³ 中找不出反式共平面（垂直构象）| 不画 Newman 投影 | **必须画**！立体构象决定一切 |
| 认为反式共平面**总是**有利 | 忽视 α 效应反例 | 当两个轨道**都已被电子占据**时（如 lp-lp），反式共平面**不利** |
| 把"反式共平面"与"反平行"混淆 | 翻译相似 | 反平行（antiparallel）是磁矩或键的"方向相反"；反式共平面是几何二面角概念 |
| 在 sp² 体系中用错"反式共平面" | sp³ 习惯延伸 | sp² 体系要求"轨道与 π 平面共面"——这是广义反式共平面 |

### 12.1 学习路径

- **学习本 KP 前，需要先掌握**：
  - "[[有机分子轨道]]（HOMO / LUMO 概念，反键轨道的"后方波瓣"位置）"
  - "[[Newman投影式]] / 二面角的画法"
  - "[[21-杂化轨道理论与成键]]（理解 σ 键的方向性）"
- **学完本 KP 后，自然过渡到**：
  - "[[给体-受体相互作用]]（反式共平面是其几何条件）"
  - "[[E2反应]]（直接应用反式共平面要求的反应）"

### 12.2 学生易踩的认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析口诀/方法 |
|:---|:---|:---|:---|
| "反式共平面 = 反式（trans）" | 都有"反式"二字 | **完全不同**：反式共平面是**两根 σ 键的二面角 = 180°**；反式（trans）是**双键两侧大基团相对位置** | "看几何还是看构型——σ 键看反式共平面，C=C 看 cis/trans" |
| "反式共平面随便扭一下分子就能实现" | 把它当成"灵活几何" | 在**刚性环系**中（如环己基），反式共平面**只在特定构象**下成立——不是想要就有 | "环己基里 1,2-反式只在双轴位置才反式共平面" |
| "反式共平面就是直线"| 把"共平面"误为"共线" | 反式共平面是 **4 个原子共面**且**两根键反向**——是平面而非直线 | "Newman 投影里画一画，原子在同一平面 + 反向 = 反式共平面" |
| "反式共平面总是稳定" | 忽视 α 效应（n-n 不利）| 当涉及**两个占据轨道**（如 lp-lp）时，反式共平面**不稳定**——分子会主动避开 | "看是给-受体（√）还是给-给（×）" |
| "学完反式共平面就够了，不用看角度"| 只记"180°" | 必须能从分子结构中**画出 Newman 投影并量出角度**；判断是否真的反式共平面 | "做题前先画构象图" |

### 12.3 入门级例题（不同于第十节）

> 第十节例题面向"已学的人"做巩固；本段例题面向"刚学的人"做入门。

**题目**：在以下分子中，**1-Br 和 2-CH₃** 是否处于反式共平面？

A. cis-1-甲基-2-溴环己烷
B. trans-1-甲基-2-溴环己烷

**预期解答路径**（把脑内推理写出来）：

1. 第一步看到题目应该想到：环己烷的椅式构象。1,2-取代基的几何关系由"双轴/双赤道/一轴一赤道"决定。
2. 然后联想到本 KP 的：反式共平面 = 二面角 180°。在环己烷椅式中，**只有 1,2-双轴或 1,2-双赤道**才是反式共平面（双轴：180°；双赤道：约 60°，不算反式共平面）。
3. 应用：
   - **A（cis）**：1-Me、2-Br 在环上同侧 → 必为"一轴一赤道"（不可能同时双轴/双赤道）→ **永远不反式共平面**
   - **B（trans）**：1-Me、2-Br 在环上异侧 → 可双轴构象（此时反式共平面 ✅）或双赤道（不反式共平面）
4. **答案**：A 不可以；B 可以（在双轴构象下）。

**老师可在课堂上引导的提问**：
- 学生若卡在第 1 步：让学生在草稿纸上画出椅式构象，把 Br 和 CH₃ 标出来。
- 学生若卡在"双轴 vs 双赤道"判断：问 "trans-1,2 在椅式里，两个取代基同时'朝下'还是同时'朝上' / 一上一下？"
- 进阶提问：在环己基 E2 中，cis 异构体反应慢的原因是不是"反式共平面失败"？

### 12.4 与现实/直觉的连接

- **类比**：反式共平面像**两个磁铁的"南北极对齐"** —— σ 键的成键波瓣（"南极"）对准 σ\* 反键的"北极"（最大波瓣那侧）才能"吸"住（轨道重叠最大）。如果磁铁朝向错了（顺式共平面 = 同极相邻），互相排斥（重叠差）；如果磁铁垂直放置（90°），完全不互动（节面对齐）。
- **反例/佯谬**：表面上看，"反式"听起来是远离 / 不接触的几何（cis 是同侧、trans 是异侧 → 远）；但实际上，**反式共平面恰恰是两个轨道波瓣靠最近的位置**——因为 σ\* 反键轨道的最大波瓣位于"键的另一侧"，所以 σ 给体要从"键的反方向"伸过来才能重叠。这告诉我们：**轨道几何与原子几何不一样**——原子可以在远处，但轨道波瓣已经在交流。

## 十三、竞赛拓展

### 13.1 反式共平面的 NBO 分析

> 用 NBO 二阶微扰法可以**直接计算**反式共平面下给体-受体相互作用的稳定化能：

| 系统 | 二面角 | 稳定化能 ΔE_NBO |
|:---|:---:|:---:|
| H-C-C-F 反式共平面 | 180° | -3 ~ -5 kcal/mol |
| H-C-C-F 顺式共平面 | 0° | -1 ~ -2 kcal/mol（约 1/3）|
| H-C-C-F 垂直 | 90° | ~ 0（无相互作用）|
| 双轴 β-Si-C-C-OBs | 180° | -25 ~ -35 kcal/mol（β-Si 效应）|

### 13.2 反式共平面的"经济性"和"协同性"

> 在多步反应中，**多个反式共平面同时实现**会带来巨大的能量降低：

- E2 反应：β-H 与 LG **同时反式共平面** → 协同断裂，能垒大幅降低
- Grob 碎裂：两根 σ 键 + 一对孤对电子 **三向同时反式共平面** → "电子推动如多米诺"
- Cope 重排：σ-σ\* 与 σ-π\* **同时反式共平面** → 协同 [3,3]-σ 迁移

### 13.3 反式共平面的"破坏者"——立体位阻

> 在某些情况下，反式共平面构象在能量上不利，分子被迫"打破"它：
> - **过大取代基**强迫使用赤道-赤道构象（环己基，trans-1,2）→ 反式共平面失败 → E2 慢
> - **桥头碳**：桥环结构使某些键无法旋转（[[Bredt规则]]）→ 反式共平面不可能

### 13.4 与协同反应的关联

> [[周环反应]] 中的"协同性"在某种意义上就是"多个反式共平面同时实现"——
> [[FMO理论]] 与反式共平面共同决定了 D-A、Cope、Claisen 等反应的位置/立体选择性。

### 13.5 重要文献

- Deslongchamps, P. *Stereoelectronic Effects in Organic Chemistry* (Pergamon, 1983) — 反式共平面在立体电子效应中的核心地位
- Reed, A. E.; Curtiss, L. A.; Weinhold, F. *Chem. Rev.* **1988**, *88*, 899 — NBO 理论与反式共平面的定量
- Lambert, J. B. et al. *J. Am. Chem. Soc.* **1987**, *109*, 7838 — β-Si 实验对反式共平面的最壮观验证
