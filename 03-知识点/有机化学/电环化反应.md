# 电环化反应（Electrocyclic Reaction）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[49-周环反应]]

## 一、定义

**电环化反应（Electrocyclic Reaction）**：共轭多烯的**末端碳之间形成 σ 键（关环）**或**σ 键断裂（开环）**的协同周环反应。

**通式**：
$$
\ce{共轭多烯 <-> 环烯烃}
$$

**本质**：**π 电子体系重组为 σ 键**——是 Woodward-Hoffmann 规则的经典应用。

## 三、核心原理

### 3.1 Woodward-Hoffmann 规则

| π 电子数 | 热反应 | 光反应 |
|:---:|:---:|:---:|
| **4n**（4, 8, 12...）| **旋光（conrotatory）** | **对旋（disrotatory）** |
| **4n+2**（2, 6, 10...）| **对旋（disrotatory）** | **旋光（conrotatory）** |

**记忆口诀**：
> "**4n 热旋光，4n+2 热对旋**"

### 3.2 为什么是这些规则？——轨道相位分析

**关环过程**：
- 共轭多烯的两端 p 轨道旋转，使同相瓣重叠 → 形成 σ 键
- 旋转方向取决于 HOMO 末端瓣的相对相位

**4n 体系（以 1,3-丁二烯为例，4π）**：
- HOMO（ψ₂）末端瓣相位**相反**
- 要同相重叠 → 两个末端碳**同向旋转**（旋光）

**4n+2 体系（以 1,3,5-己三烯为例，6π）**：
- HOMO（ψ₃）末端瓣相位**相同**
- 要同相重叠 → 两个末端碳**反向旋转**（对旋）

### 3.3 旋光（Conrotatory）vs 对旋（Disrotatory）

| | **旋光（Conrotatory）** | **对旋（Disrotatory）** |
|:---|:---|:---|
| **旋转方向** | 两个末端碳**同向**旋转 | 两个末端碳**反向**旋转 |
| **4n 热反应** | ✅ 允许 | ❌ 禁阻 |
| **4n+2 热反应** | ❌ 禁阻 | ✅ 允许 |
| **立体化学结果** | 反式双键（trans） | 顺式双键（cis） |

### 3.4 典型例子

**4π 体系（4n, n=1）**：

| 反应 | 条件 | 旋转方式 | 产物 |
|:---|:---|:---:|:---|
| 1,3-丁二烯 → 环丁烯 | 热 | 旋光 | 反式环丁烯 |
| 环丁烯 → 1,3-丁二烯 | 热 | 旋光 | 顺,反-二烯 |
| 1,3-丁二烯 → 环丁烯 | 光 | 对旋 | 顺式环丁烯 |

**6π 体系（4n+2, n=1）**：

| 反应 | 条件 | 旋转方式 | 产物 |
|:---|:---|:---:|:---|
| 1,3,5-己三烯 → 1,3-环己二烯 | 热 | 对旋 | 顺式环己二烯 |
| 1,3-环己二烯 → 1,3,5-己三烯 | 热 | 对旋 | 顺,顺,反-三烯 |
| 1,3,5-己三烯 → 1,3-环己二烯 | 光 | 旋光 | 反式环己二烯 |

**8π 体系（4n, n=2）**：

| 反应 | 条件 | 旋转方式 | 产物 |
|:---|:---|:---:|:---|
| 辛四烯 → 环辛三烯 | 热 | 旋光 | — |
| 辛四烯 → 环辛三烯 | 光 | 对旋 | — |

### 3.5 光反应的反转

**光照改变反应方式的原因**：
- 光照 → 电子从 HOMO 跃迁到 LUMO
- 激发态的 HOMO 是原来的 LUMO
- LUMO 的末端瓣相位与 HOMO **相反**
- 所以光反应的热允许/禁阻方式与热反应**相反**

### 3.6 电环化反应的空间限制（Clayden Ch35 补充）

> 虽然所有电环化反应在理论上都是允许的，但顺旋/对旋的空间需求可能导致其中一种模式不可行。

**典型例子**：
- **6π对旋热环化**：若产物会在六元环中形成反式双键，则对旋模式不可行 → 反应需通过光化学顺旋进行（如维生素D合成中的前一步）
- **环辛二烯基阴离子**：6π对旋热关环可行，但顺旋光化学关环被阻止（产物将是5,5反式稠合体系，张力太大）
- **环辛二烯基阳离子**：热电环化关环也被环张力阻止

### 3.7 Woodward-Hoffmann 处理电环化反应（Clayden Ch35 补充）

对于电环化反应，**只能用 Woodward-Hoffmann 规则**（至少对于关环反应），因为所涉及的分子轨道只有一条。

**处理步骤**：
1. 画出反应机理
2. 选择组分（任何参与机理的键都必须包含在内）
3. 画出组分聚集在一起的三维图，在各组分的末端画出轨道
4. 在要形成新键的地方连接组分
5. 根据新键在相同侧还是相反侧形成，标注 "s" 或 "a"
6. 将 $(4q+2)_s$ 和 $(4r)_a$ 组分的数目相加。若总数为奇数，反应允许。

**关键发现**：
- $(4n+2)\pi$ 共轭体系：必须与自己**同面**反应 → **对旋**
- $(4n)\pi$ 共轭体系：必须与自己**异面**反应 → **顺旋**

---

## 四、关键结论

1. **电环化 = 共轭多烯 ↔ 环烯烃**
2. **4n 热旋光，4n+2 热对旋**
3. **光反应方式与热反应相反**
4. **旋光产生反式双键，对旋产生顺式双键**
5. **规则来源于 HOMO 末端瓣的相对相位**

## 五、常见分类或情形

### 5.1 按 π 电子数分类

| π 电子数 | 类型 | 热反应 | 例子 |
|:---:|:---|:---:|:---|
| **2** | 4n+2 (n=0) | 对旋 | 乙烯 ↔ 无（2π 不环化）|
| **4** | 4n (n=1) | 旋光 | 1,3-丁二烯 ↔ 环丁烯 |
| **6** | 4n+2 (n=1) | 对旋 | 1,3,5-己三烯 ↔ 环己二烯 |
| **8** | 4n (n=2) | 旋光 | 辛四烯 ↔ 环辛三烯 |

### 5.2 按反应方向分类

| 方向 | 说明 | 驱动力 |
|:---|:---|:---|
| **关环** | 多烯 → 环烯 | 熵减，但可能释放应变能 |
| **开环** | 环烯 → 多烯 | 环张力释放 |

## 六、适用条件与限制

1. **需要热或光驱动**
2. **产物可能热不稳定**——环丁烯有环张力
3. **大环多烯的环化困难**——末端碳难以接近
4. **立体化学由规则严格决定**——不能违反 Woodward-Hoffmann 规则
5. **光反应需选择合适的波长**

---

## 七、常见比较与易混点

### 7.1 电环化 vs D-A

- 电环化：一个分子的 π 体系 → 一个分子的 σ+π 体系
- D-A：两个分子 → 一个分子
- 两者都是周环反应，但参与体系不同

### 7.2 旋光 vs 对旋

- 旋光：两个末端碳**同向**旋转
- 对旋：两个末端碳**反向**旋转
- 4n 热 = 旋光；4n+2 热 = 对旋

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：1,3,5-己三烯在加热条件下的电环化产物是什么？旋转方式如何？

**解答**：
- 6π → 4n+2 体系
- 热反应 → **对旋**
- 产物：**顺式-1,3-环己二烯**

### 10.2 例题 2（★★ 光反应）

**题目**：反,反,顺-1,3,5-己三烯光照后的电环化产物？

**解答**：
- 6π → 4n+2 体系
- 光反应 → **旋光**
- 旋光 → 反式双键
- 产物：**反式-1,3-环己二烯衍生物**

### 10.3 例题 3（★★★ 开环）

**题目**：顺式环丁烯加热开环后的产物构型？

**解答**：
- 环丁烯开环 = 电环化的逆反应
- 4π → 热 → **旋光**
- 旋光开环 → 产物中两端甲基在**反式**位置
- 产物：**顺,反-2,4-己二烯**

## 十一、易错点

1. ❌ **4n 热对旋** → 4n 热**旋光**
2. ❌ **光反应和热反应一样** → 光反应方式**相反**
3. ❌ **旋光产生顺式双键** → 旋光产生**反式**双键
4. ❌ **电环化可以离子型机理** → 电环化是**协同周环反应**
5. ❌ **只背规则不画轨道** → 必须理解**HOMO 末端瓣相位**

### 12.3 入门例题

**题目**：为什么 4n 体系的热电环化必须是旋光而不是对旋？

**引导**：
- 画 1,3-丁二烯的 HOMO（ψ₂）
- 末端两个 p 轨道的瓣相位相反
- 要形成 σ 键，必须同相瓣重叠
- 同向旋转（旋光）才能使同相瓣相遇
- 反向旋转（对旋）会使反相瓣相遇，能量不利

### 12.4 现实类比

**电环化像"关门"**：
- 共轭多烯 = 敞开的双臂
- 关环 = 双手合十
- 4n 体系 = 双手掌心朝向相反 → 必须同向转动才能合十（旋光）
- 4n+2 体系 = 双手掌心朝向相同 → 反向转动才能合十（对旋）
- 光照 = 突然换了一只手的朝向 → 原来的关门方式不行了，要换方式

## 十三、竞赛拓展

### 13.1 Nazarov 环化（Clayden Ch35 详细补充）

**机理**：
1. 双α,β-不饱和酮 + 酸 → 质子化 → 共轭二烯基阳离子
2. 4π 电环化关环（**热顺旋**）→ 环戊烯基阳离子
3. 质子失去 + 异构化 → **环戊烯酮**

**关键特征**：
- 5条p轨道，其中一条空置 → 4π体系
- 热条件：**顺旋**
- 光条件：**对旋**（与热相反）
- 是合成环戊烯酮的重要方法

### 13.2 维生素 D 合成中的电环化（Clayden Ch35 补充）

**第一步（需光照）**：
- 7-脱氢胆固醇 → 光化学电环化开环 → 预钙化醇（Previtamin D₃）
- 6π体系，光化学 → **顺旋**开环
- 对旋热6π环化会在六元环中形成不可能的反式双键 → 热反应不可行

**第二步（无需光照）**：
- 预钙化醇 → [1,7]H移位（异面）→ 维生素D₃

### 13.3 土楠酸（Endiandric Acids）的生物合成（Clayden Ch35）

非酶催化的电环化串联：
1. **8π 电环化**（非环状多烯前体）→ **顺旋** → 形成第一个环
2. **6π 电环化** → **对旋** → 形成第二个环
3. 分子内 Diels-Alder → 土楠酸 A（四个环，八个手性中心，一步合成）

土楠酸 D 和 E 是两种非对映体，来源于第二步对旋环化的两种可能方向。

### 13.4 小环的电环化开环（Clayden Ch35）

**环丙基阳离子 → 烯丙基阳离子**：
- 烯丙基阳离子（2π）热对旋关环 → 环丙基阳离子（理论上）
- 但实际上**环丙基阳离子非常容易开环**为烯丙基阳离子
- 环张力促进电环化开环
- 尝试环丙烷环上的亲核取代 → 环丙基阳离子中间体迅速开环

**氮丙环（Aziridine）开环**：
- 带吸电子基团的氮丙环不稳定
- 电环化开环 → 甲亚铵叶立德（azomethine ylid，4π体系）
- 叶立德可被亲偶极体通过 [3+2] 环加成捕捉
- 产物立体化学证实开环是**顺旋**的

### 13.5 环辛二烯基阴离子的电环化（Clayden Ch35）

- 环辛二烯 + BuLi → 环辛二烯基阴离子
- 5条p轨道，6个π电子 → **对旋**关环（热允许）
- 顺旋光化学环化被环张力阻止（5,5反式稠合体系不可能）
- 热电环化关环也被环张力阻止（若形成不利结构）
