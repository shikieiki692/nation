# Kulinkovich 反应（Kulinkovich Reaction）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[50-有机合成]]、[[51-金属有机]]

---

## 一、定义

**Kulinkovich 反应**：在 **Ti(IV) 催化剂**（通常是 Ti(OiPr)₄）存在下，**酯**与**过量格氏试剂**反应，生成 **环丙醇衍生物**的反应。

**通式**：
$$
\ce{R-COOR' + 2 EtMgBr ->[Ti(OiPr)4] 环丙醇衍生物}
$$

**本质**：**钛杂环丙烷中间体与酯的环加成**——将酯转化为三元环醇的独特方法，是构建环丙烷骨架的重要工具。

## 三、核心原理

### 3.1 反应机理

**步骤 1：钛杂环丙烷的生成**
```
2 EtMgBr + Ti(OiPr)₄ → 钛杂环丙烷 + 2 MgBr(OiPr)
```
- 两分子格氏试剂与 Ti(IV) 反应
- 失去两分子烷氧基 → 形成 **Ti(II) 杂环丙烷**
- 钛杂环丙烷 = 类卡宾物种（可看作 Ti=C 的等价物）

**步骤 2：与酯的加成**
```
钛杂环丙烷 + R-COOR' → 钛杂环氧化物中间体
```
- 钛杂环丙烷的 C–Ti 键进攻酯的羰基碳
- 形成 **五元环钛杂中间体**

**步骤 3：断裂生成环丙醇**
```
钛杂环氧化物 → 水解 → 环丙醇 + Ti 副产物
```
- 水解（酸性后处理）→ 断裂 Ti–O 键
- 生成 **环丙醇**

### 3.2 关键中间体——钛杂环丙烷

**钛杂环丙烷（Titanacyclopropane）**：
- 三元的 Ti–C–C 环
- Ti 处于低氧化态（Ti(II)）
- **高度亲核**——可与羰基化合物反应
- 是 Ti 化学中的"类卡宾"等价物

### 3.3 底物范围

**酯**：
- 脂肪族酯（最常用）
- 内酯 → 双环环丙醇
- α,β-不饱和酯 → 可能竞争反应

**格氏试剂**：
- 乙基格氏最经典
- 其他烷基格氏也可
- 芳基格氏效果较差

---

## 四、关键结论

1. **Kulinkovich = 酯 + 过量格氏试剂 + Ti(IV) → 环丙醇**
2. **钛杂环丙烷是核心中间体**
3. **两分子格氏试剂参与反应**
4. **是构建环丙醇的独特方法**
5. **水解后处理得到产物**

---

## 五、常见分类或情形

### 5.1 按底物分类

| 酯类型 | 产物 | 特点 |
|:---|:---|:---|
| **开链酯** | 单环环丙醇 | 经典 |
| **内酯** | 双环（并环）环丙醇 | 合成价值高 |
| **α,β-不饱和酯** | 可能复杂混合物 | 选择性差 |

### 5.2 变体

**Kulinkovich-De Meijere 变体**：
- 使用 TiCl₄ 代替 Ti(OiPr)₄
- 对酰胺也有效

**分子内 Kulinkovich**：
- 含有酯和烯烃的同一分子
- 钛杂环丙烷与分子内酯反应

---

## 六、适用条件与限制

1. **需要过量格氏试剂**（至少 2 当量）
2. **严格无水无氧**
3. **低温进行**（-78°C 至 0°C）
4. **需要酸性后处理**（水解 Ti–O 键）
5. **α,β-不饱和酯可能副反应**
6. **酰胺反应性较差**（需变体条件）

---

## 七、常见比较与易混点

### 7.1 Kulinkovich vs Simmons-Smith

| | **Kulinkovich** | **Simmons-Smith** |
|:---|:---|:---|
| **产物** | 环丙醇 | 环丙烷 |
| **底物** | 酯 | 烯烃 |
| **试剂** | 格氏 + Ti(IV) | CH₂I₂ + Zn(Cu) |
| **机理** | 钛杂环丙烷 | 类卡宾 |

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：Kulinkovich 反应的核心中间体是什么？

**解答**：
- **钛杂环丙烷（titanacyclopropane）**
- 由 2 分子格氏试剂与 Ti(IV) 反应生成
- 是 Ti(II) 的三元环中间体
- 可看作 Ti=C 的类卡宾等价物

### 10.2 例题 2（★★ 产物）

**题目**：丙酸乙酯与过量乙基格氏试剂在 Ti(OiPr)₄ 存在下的产物是什么？

**解答**：
- 丙酸乙酯 = Et-COOEt
- 2 EtMgBr + Ti(OiPr)₄ → 钛杂环丙烷
- 钛杂环丙烷 + Et-COOEt → 环丙醇衍生物
- 产物：**1-乙基-2-甲基环丙醇**（或相应结构）

---

## 十一、易错点

1. ❌ **Kulinkovich 只需要 1 当量格氏** → 需要 **2 当量以上**
2. ❌ **产物是环丙烷** → 产物是 **环丙醇**
3. ❌ **Kulinkovich 是自由基机理** → 是 **金属杂环机理**

### 12.2 现实类比

**Kulinkovich 像"折纸艺术"**：
- 酯 = 一张平铺的纸
- 钛杂环丙烷 = 一双手（把纸折成三角形）
- 格氏试剂 = 提供折纸所需的"能量"
- 最终产物 = 一个立体的三角结构（环丙醇）
- 整个过程中纸没有被撕碎（协同机理）
