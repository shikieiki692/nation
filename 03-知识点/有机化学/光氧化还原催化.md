# 光氧化还原催化（Photoredox Catalysis）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[40-氧化还原反应]]、[[50-有机合成]]

---

## 一、定义

**光氧化还原催化**：利用**可见光激发的光催化剂**（如 Ru(bpy)₃²⁺、Ir(ppy)₃ 等）通过**单电子转移（SET）**引发或促进有机反应的催化方法。反应中光催化剂吸收光子进入激发态，随后与底物发生电子转移，生成自由基或自由基离子中间体。

**本质**：**光化学与氧化还原催化的交叉**——用光能驱动传统热催化难以实现的单电子过程，是 21 世纪有机合成的核心方法之一。

## 三、核心原理

### 3.1 核心催化剂

| 催化剂 | 激发态还原电位 E*(Red) | 激发态氧化电位 E*(Ox) | 特点 |
|:---|:---:|:---:|:---|
| **Ru(bpy)₃²⁺** | –0.81 V | +0.77 V | 经典催化剂，水溶性好 |
| **[Ru(bpy)₃]Cl₂** | –0.81 V | +0.77 V | 与上者相同，常用形式 |
| **Ir(ppy)₃** | –1.73 V | +0.31 V | 更强还原性，近年更常用 |
| **Ir[dF(CF₃)ppy]₂(dtbbpy)⁺** | –1.21 V | +1.21 V | 氧化还原双强，应用最广 |
| **曙红 Y（Eosin Y）**| — | — | 有机染料，廉价替代 |

**电位含义**：
- **E*(Red)**：激发态作为**还原剂**的强弱（越负越强）
- **E*(Ox)**：激发态作为**氧化剂**的强弱（越正越强）

### 3.2 两种淬灭循环

#### 还原淬灭（Reductive Quenching）

```
PC + hν → PC*（激发态光催化剂）
PC* + 电子给体（如胺、烯醇醚）→ PC•⁻ + 给体•⁺
PC•⁻ + 底物 → PC + 底物•⁻（单电子还原底物）
```

- 激发态 PC 被**还原**（接受电子）→ 生成 PC•⁻
- PC•⁻ 是强还原剂 → 还原底物

#### 氧化淬灭（Oxidative Quenching）

```
PC + hν → PC*
PC* + 电子受体（如卤代烃）→ PC•⁺ + 受体•⁻
PC•⁺ + 底物 → PC + 底物•⁺（单电子氧化底物）
```

- 激发态 PC 被**氧化**（失去电子）→ 生成 PC•⁺
- PC•⁺ 是强氧化剂 → 氧化底物

### 3.3 双催化体系（Dual Catalysis）

**光氧化还原 + 过渡金属**：
- 光催化提供**自由基**（通过 SET）
- 金属催化（如 Pd、Ni）提供**交叉偶联**
- 两者协同实现传统方法难以完成的转化

**光氧化还原 + 有机催化**：
- 光催化提供 SET
- 有机小分子（如胺、硫醇）提供**氢原子转移（HAT）**或**质子耦合电子转移（PCET）**

### 3.4 Giese 反应（光氧化还原变体）

$$
\ce{R-X + 烯烃 ->[hv, PC] R-烯烃加成产物}
$$

- 传统 Giese 反应：自由基加成到缺电子烯烃
- 光氧化还原变体：通过 SET 产生自由基 → 加成到烯烃
- 是**C–C 键形成**的重要方法

## 四、关键结论

1. **光氧化还原 = 可见光 + 光催化剂 + SET**
2. **还原淬灭**产生 PC•⁻（强还原剂），**氧化淬灭**产生 PC•⁺（强氧化剂）
3. **Ru(bpy)₃²⁺** 和 **Ir(ppy)₃** 是最常用的两种光催化剂
4. **双催化**是光氧化还原的前沿方向：光催化 + 金属/有机催化协同
5. **识别标志**：反应条件中出现 **hv + Ru/Ir/曙红Y** → 立刻想到 SET 机理

## 五、常见分类或情形

### 5.1 按淬灭方式分类

| 淬灭方式 | 激发态命运 | 产物 | 适用底物 |
|:---|:---|:---|:---|
| **还原淬灭** | PC* + 给体 → PC•⁻ | PC•⁻ 还原底物 | 需要还原的底物（卤代烃等）|
| **氧化淬灭** | PC* + 受体 → PC•⁺ | PC•⁺ 氧化底物 | 需要氧化的底物（胺、烯烃等）|

### 5.2 按反应类型分类

| 反应类型 | 机理要点 | 竞赛相关性 |
|:---|:---|:---:|
| **C–H 官能团化** | SET 产生碳自由基 → 捕获 | 高 |
| **交叉偶联** | 双催化：光催化 + Pd/Ni | 高 |
| **Giese 加成** | 自由基加成到缺电子烯烃 | 中 |
| **环化反应** | 光产生的自由基/离子环化 | 中 |
| **去卤化** | 还原淬灭还原 C–X 键 | 中 |

## 六、适用条件与限制

1. **需要光源**：通常用蓝色 LED（450–470 nm）或白光 LED
2. **需要除氧**：氧气会淬灭三线态光催化剂
3. **底物需匹配电位**：底物的氧化/还原电位需在光催化剂激发态电位范围内
4. **溶剂选择**：需考虑光催化剂的溶解性和稳定性
5. **与极性机理竞争**：光氧化还原是 SET 机理，不是极性机理

---

## 七、常见比较与易混点

### 7.1 光氧化还原 vs 有机光化学

| | **光氧化还原** | **有机光化学** |
|:---|:---|:---|
| **光吸收者** | 光催化剂（Ru/Ir/染料）| 底物本身 |
| **底物要求** | 无需吸收光 | 必须有发色团 |
| **机理** | SET（单电子转移）| 直接激发态反应 |
| **产物** | 自由基/自由基离子 | 激发态产物 |
| **选择性控制** | 通过催化剂和淬灭剂 | 通过波长和敏化剂 |

### 7.2 还原淬灭 vs 氧化淬灭

| | **还原淬灭** | **氧化淬灭** |
|:---|:---|:---|
| **激发态被...** | 还原（接受电子）| 氧化（失去电子）|
| **光催化剂中间体** | PC•⁻（强还原剂）| PC•⁺（强氧化剂）|
| **底物变化** | 被还原 | 被氧化 |
| **常见给体/受体** | 胺、烯醇醚 | 卤代烃、酮 |

## 十、例题

### 10.1 例题 1（★★★★ 基础）

**题目**：光氧化还原催化中，Ru(bpy)₃²⁺ 的激发态可以发生还原淬灭。画出还原淬灭循环，并说明 PC•⁻ 的作用。

**解答**：

**步骤 1**：光激发
$$
\ce{Ru(bpy)3^{2+} + hν -> [Ru(bpy)3^{2+}]*}
$$

**步骤 2**：还原淬灭（激发态被还原）
$$
\ce{[Ru(bpy)3^{2+}]* + D（给体）-> Ru(bpy)3^+ + D•^+}
$$

**步骤 3**：单电子还原底物
$$
\ce{Ru(bpy)3^+ + 底物 -> Ru(bpy)3^{2+} + 底物•^-}
$$

**PC•⁻（此处为 Ru⁺）的作用**：
- Ru⁺ 是强还原剂 → 单电子还原底物 → 产生底物自由基阴离子
- 自身被氧化回 Ru²⁺，完成催化循环

**反思**：
- 还原淬灭中，激发态**接受**电子 → 生成低价金属 → **还原**底物
- 注意区分 "还原淬灭" 和 "氧化淬灭" 的命名逻辑

### 10.2 例题 2（★★★★★ 进阶）

**题目**：某反应条件为：芳基卤 + 烯烃 + Pd(PPh₃)₄ + Ru(bpy)₃Cl₂ + 胺 + hv（蓝光 LED）。这是什么类型的反应？写出关键催化循环。

**分析**：
- Pd(0) + 芳基卤 → 交叉偶联的经典组合
- Ru(bpy)₃²⁺ + hv → 光氧化还原
- 胺 → 还原淬灭的给体
- **双催化体系**：光氧化还原 + Pd 催化

**解答**：
- 这是**光氧化还原/Pd 双催化的交叉偶联反应**
- 光催化通过 SET 产生芳基自由基（或还原芳基钯中间体）
- Pd 催化完成传统的氧化加成-迁移插入-还原消除循环

**反思**：
- 看到 **hv + Ru + Pd** → 立刻想到双催化
- 胺的作用是还原淬灭剂，不是碱

## 十一、易错点

1. ❌ **光氧化还原和光化学是一回事** → 光氧化还原是**催化剂吸光+SET**，有机光化学是**底物直接激发**
2. ❌ **还原淬灭是底物被还原** → 还原淬灭是**激发态被还原**（接受电子），然后还原剂（PC•⁻）再去还原底物
3. ❌ **所有光催化剂都一样** → Ru(bpy)₃²⁺ 和 Ir(ppy)₃ 的电位不同，适用底物不同
4. ❌ **光氧化还原不需要除氧** → 氧气是三线态淬灭剂，必须除氧
5. ❌ **光氧化还原是极性机理** → 是 **SET（单电子转移）** 机理，不是极性机理

### 12.2 学生易踩的认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析口诀/方法 |
|:---|:---|:---|:---|
| "光氧化还原就是光化学" | 都有 hv | 光氧化还原是**催化剂吸光**，底物不一定吸光 | "催化剂是灯泡，底物是演员" |
| "还原淬灭 = 底物被还原" | 字面理解 | 还原淬灭 = **激发态被还原**（接受电子） | "先还原催化剂，再还原底物" |
| "Ru 和 Ir 催化剂没区别" | 只记名字 | 电位不同：Ir 还原性更强（–1.73 V vs –0.81 V）| "Ir 比 Ru 强（还原性）" |
| "双催化是两个独立反应" | 以为各催化各的 | 双催化是**协同**：光催化产生中间体 → 金属催化接续 | "接力赛，不是两场比赛" |

### 12.3 入门级例题

**题目**：Ru(bpy)₃²⁺ 的激发态氧化电位为 +0.77 V，还原电位为 –0.81 V。它可以氧化电位低于多少的给体？可以还原电位高于多少的受体？

**预期解答路径**：
1. 氧化给体：激发态作为氧化剂 → 氧化电位 +0.77 V → 可以氧化 E < +0.77 V 的给体
2. 还原受体：激发态作为还原剂 → 还原电位 –0.81 V → 可以还原 E > –0.81 V 的受体
3. 结论：氧化电位比 +0.77 V 更负的给体；还原电位比 –0.81 V 更正的受体

**老师可在课堂上引导的提问**：
- 学生若混淆氧化/还原方向：问 "氧化是让对方失电子，还原是让对方得电子——激发态的哪个电位对应哪个能力？"

### 12.4 与现实/直觉的连接

- **类比**：光氧化还原催化就像"太阳能充电电池"——白天（hv）充电（激发态），晚上（基态）放电（SET 转移电子），电池本身（催化剂）不被消耗
- **反例**：表面上看，加光催化剂是多此一举；实际上，没有催化剂的话，可见光能量不足以直接激发大多数有机底物——催化剂是"能量中介"

## 十三、竞赛拓展

1. **不对称光氧化还原**：手性光催化剂或手性配体实现不对称转化，近年热点。
2. **电光协同催化**：电化学 + 光氧化还原结合，避免化学计量氧化还原剂。
3. **天然光催化**：利用可见光驱动的人工光合作用，超纲但属于前沿方向。
