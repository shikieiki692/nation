# Kekulé 式（Kekulé Structure）

## 一、定义

**Kekulé 式**是用短线表示化学键的结构表达式，其中单线（—）表示单键、双线（=）表示双键、三线（≡）表示三键。Kekulé 式是 Lewis 结构的简化形式，省略了 C—H 键上的氢原子和非键孤对电子（或仅保留杂原子上的孤对电子），是有机化学中最常用的结构表示法之一。

- **历史背景**：由 Friedrich August Kekulé 于 19 世纪提出，奠定了有机化合物结构理论的基础
- **核心约定**：
  - 碳原子位于键的端点或交叉点（不写出 C）
  - 与碳相连的氢原子被省略（隐含补足四价）
  - 杂原子（O, N, S, P, 卤素等）必须明确写出
  - 杂原子上的孤对电子可省略或保留
- **与 Lewis 式的关系**：Kekulé 式是 Lewis 式的简化版，保留了成键信息但省略了大部分价电子细节

## 三、核心原理

### 3.1 Kekulé 式的书写规则

| 规则 | 说明 | 示例 |
|:---|:---|:---|
| **碳原子省略** | 键的端点和交叉点默认是碳 | —CH₂—CH₂— 写作 —— |
| **氢原子省略** | 碳上的氢不写出，隐含补足四价 | CH₃CH₂OH 中 CH₃、CH₂ 的 H 省略 |
| **杂原子写出** | O, N, S, 卤素等必须明确标出 | —OH, —NH₂, —Cl |
| **多重键保留** | 双键、三键用 = 和 ≡ 表示 | C=C, C≡C, C=O |
| **电荷标注** | 离子电荷需标出 | R₃N⁺, RCOO⁻ |

### 3.2 Kekulé 式 vs Lewis 式 vs 键线式

| 维度 | Lewis 式 | Kekulé 式 | 键线式（Skeletal）|
|:---|:---|:---|:---|
| **碳原子** | 写出 C | 写出 C | 省略（端点/折点）|
| **氢原子** | 写出 H | 省略 C-H | 全部省略 |
| **孤对电子** | 必须标出 | 可省略（杂原子上保留）| 省略 |
| **书写复杂度** | 最复杂 | 中等 | 最简洁 |
| **适用场景** | 机理书写、电子推动 | 常规结构表示 | 复杂分子的快速表达 |
| **示例（乙醇）** | H₃C:̈C:̈H₂:̈O:̈H | CH₃CH₂OH | 折线加 OH |

### 3.3 苯的 Kekulé 式与共振

苯的 Kekulé 式存在两种等价写法：

```
  CH           CH
  //           \\
HC    CH  ↔  HC    CH
  \\           //
   CH         CH
```

- 两种 Kekulé 结构是**共振式**，不是互变异构体
- 真实苯结构是两者的共振杂化体，所有 C-C 键等长（~1.39 Å）
- 现代常用**圈形符号**（中间画圆圈）或**Kekulé 共振式**表示苯

## 四、关键结论

1. **Kekulé 式是有机化学的标准"工作语言"**：比 Lewis 式简洁，比键线式信息完整
2. **碳和氢的省略是理解 Kekulé 式的关键**：每个碳必须满足四价，缺失的价由氢补齐
3. **杂原子和官能团必须明确写出**：不能省略 O, N, S 等杂原子及其上的氢
4. **苯的 Kekulé 式不代表真实结构**：真实苯中键长均等，不能用单双键交替描述
5. **Kekulé 式不适合表达离域体系**：共轭体系、芳香体系需要用共振式或分子轨道描述

## 五、常见分类

### 5.1 常见官能团的 Kekulé 表达

| 官能团 | Kekulé 式 | 键线式 | 备注 |
|:---|:---|:---|:---|
| **烷烃** | CH₃CH₂CH₃ | 锯齿折线 | 最简单 |
| **烯烃** | CH₂=CHCH₃ | 双键用双线 | 几何异构需标注 |
| **炔烃** | CH≡CCH₃ | 三键用三线 | 直线形 |
| **醇** | CH₃CH₂OH | 折线-OH | O-H 写出 |
| **醚** | CH₃OCH₃ | CH₃-O-CH₃ | 氧桥 |
| **醛** | CH₃CHO | CH₃-C(=O)-H | 醛基 H 写出 |
| **酮** | CH₃COCH₃ | CH₃-C(=O)-CH₃ | — |
| **羧酸** | CH₃COOH | CH₃-COOH | —OH 写出 |
| **酯** | CH₃COOCH₃ | CH₃-C(=O)-O-CH₃ | — |
| **胺** | CH₃NH₂ | CH₃-NH₂ | N-H 写出 |
| **酰胺** | CH₃CONH₂ | CH₃-C(=O)-NH₂ | — |
| **腈** | CH₃C≡N | CH₃-C≡N | 三键 |
| **硝基** | CH₃NO₂ | CH₃-NO₂ | N=O 双线 |

### 5.2 离域体系的 Kekulé 表达局限

| 体系 | Kekulé 式问题 | 更好表示法 |
|:---|:---|:---|
| **苯** | 单双键交替不反映键长均等 | 共振式 + 圆圈符号 |
| **丁二烯** | 两个孤立双键不反映共轭 | 共振式 |
| **烯丙基正离子** | 正电荷定域不符合实际 | 共振式（两个等价式）|
| **羧酸根** | 两个 C-O 键不等不符合实际 | 共振式（两个等价式）|
| **酰胺** | C=O 和 C-N 单键不反映部分双键性 | 共振式 |

## 六、适用条件

### 6.1 Kekulé 式的最佳使用场景

| 场景 | 说明 |
|:---|:---|
| **常规结构表示** | 大多数有机化合物的标准表示法 |
| **反应方程式** | 显示反应前后分子的变化 |
| **机理书写辅助** | 与电子推动箭头配合使用 |
| **教学入门** | 初学者理解分子结构的第一步 |
| **强调特定键型** | 需要明确显示双键/三键时 |

### 6.2 不适合使用 Kekulé 式的场景

| 场景 | 替代方案 |
|:---|:---|
| **复杂大分子** | 键线式 |
| **离域体系** | 共振式、分子轨道图 |
| **电子机理详细推导** | Lewis 式（保留孤对电子）|
| **快速草图** | 键线式 |

## 七、常见比较

### 7.1 三种主要结构式的对比

| 化合物 | Lewis 式 | Kekulé 式 | 键线式 |
|:---|:---|:---|:---|
| **环己烷** | 写出所有 C 和 H | 写出 C，省略 H | 六边形 |
| **苯酚** | 写出所有价电子 | C₆H₅OH 或画六元环 | 六边形 + OH |
| **乙酸乙酯** | 所有电子对 | CH₃COOCH₂CH₃ | 折线 + 酯基 |
| **复杂度** | 高 | 中 | 低 |
| **信息完整性** | 最高 | 中等 | 最低（但足够）|

## 十、例题

### 例题 1：Kekulé 式书写

**题目**：写出 2-甲基-2-丁醇的 Kekulé 式。

**解析**：

1. **主链**：丁烷（4 个碳）
2. **取代基**：
   - 2-位有甲基（—CH₃）
   - 2-位有羟基（—OH）
3. **Kekulé 式**：

```
     CH₃
      |
CH₃ — C — OH
      |
     CH₂CH₃
```

或更简洁：

```
     OH
      |
CH₃ — C — CH₂CH₃
      |
     CH₃
```

### 例题 2：隐含氢的计算

**题目**：某化合物的 Kekulé 式如下，求其分子式。

```
    O
    ‖
CH₃-C-CH=CH₂
```

**解析**：

1. **数碳原子**：
   - CH₃ = 1 个 C
   - C=O = 1 个 C
   - CH = 1 个 C
   - CH₂ = 1 个 C
   - 总 C = 4

2. **数氢原子**（从 Kekulé 式直接读出）：
   - CH₃ = 3 个 H
   - C=O 无 H
   - CH = 1 个 H
   - CH₂ = 2 个 H
   - 总 H = 6

3. **数氧原子**：
   - C=O = 1 个 O

4. **分子式**：**C₄H₆O**

**验证不饱和度**：Ω = (2×4 + 2 - 6)/2 = 2（一个 C=O + 一个 C=C，符合）

### 例题 3：价态检查

**题目**：检查下列 Kekulé 式是否有错误。

```
CH₃-CH=CH-CH₃
    |
    OH
```

**解析**：

1. **检查 C2**：
   - C2 连有：CH₃（单键）、CH（双键，算两个键）、OH（单键）
   - 键数 = 1 + 2 + 1 = 4 ✓

2. **检查 C3**：
   - C3 连有：C2（双键，算两个键）、CH₃（单键）
   - 键数 = 2 + 1 = 3
   - **还缺一个键**！应该是 CH，不是单纯的 C

3. **修正**：
   - 若 OH 在 C2 上，C3 应该是 CH
   - 原式写法中 C3 缺一个 H

**结论**：该 Kekulé 式**有误**，C3 应为 CH。

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| "Kekulé 式中省略的氢可以不管" | 忽略四价规则 | 每个碳必须满足**四价**，省略的氢在心里补齐 | 写出 CH₃CH=CH₂ 中每个碳的成键数 |
| "苯的单双键是真实的" | 误解 Kekulé 式的含义 | 苯的 Kekulé 式是**共振式**，真实键长均等 | 苯中 C-C 键长是多少？与 Kekulé 式一致吗？ |
| "杂原子上的氢也可以省略" | 混淆省略规则 | 只有**碳上的氢**可以省略；N-H、O-H 必须写出 | —NH₂ 的氢可以省略吗？ |
| "Kekulé 式就是 Lewis 式" | 混淆两种表示法 | Kekulé 式**省略了孤对电子和大部分氢**，是 Lewis 式的简化 | Lewis 式和 Kekulé 式的根本区别是什么？ |
| "键线式比 Kekulé 式更详细" | 颠倒信息密度 | 键线式最简洁；Kekulé 式保留了原子符号 | 三种结构式（Lewis/Kekulé/键线）哪个最简洁？ |

### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "Kekulé 式过时了，都用键线式" | 忽视 Kekulé 式的教学价值 | 强调：Kekulé 式在显示**官能团细节**和**反应变化**时比键线式更清晰 |
| "画出 Kekulé 式就是写出分子式" | 混淆结构式与分子式 | 训练：从 Kekulé 式精确数出 C、H、O 等原子数 |

### 12.3 入门例题

**最简判断练习**：
> 1. Kekulé 式中，哪些原子上的氢可以省略？
> 2. 苯的 Kekulé 式中，C-C 键是单键还是双键？真实苯呢？
> 3. 写出乙醇的 Kekulé 式和键线式。

**答案**：
1. **碳原子（C）上的氢**可以省略；杂原子（O, N 等）上的氢必须写出
2. Kekulé 式中是**单双键交替**；真实苯中所有 C-C 键**完全等同**（键长约 1.39 Å）
3. Kekulé：CH₃CH₂OH；键线：折线端点加 OH
