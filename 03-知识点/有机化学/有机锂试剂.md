# 有机锂试剂

## 一、定义

**有机锂试剂（Organolithium Reagent）** 通式为 **RLi**（R = 烷基、烯基、芳基、炔基），是由卤代烃与金属锂反应或通过**卤素-锂交换**制备的有机金属化合物。

$$
\ce{R-X + 2 Li -> R-Li + LiX}
$$

$$
\ce{R-X + R'-Li -> R-Li + R'-X} \quad \text{(卤素-锂交换)}
$$

- **C—Li 键极化极强**（Cδ⁻—Liδ⁺），碳原子带高度负电荷
- **亲核性、碱性均强于 Grignard 试剂**
- **聚集态结构**：在溶液中以多聚体形式存在（四聚体、六聚体），THF 或 TMEDA 可解聚增强反应性
- **对空气和水极度敏感**：需严格无水、无氧、低温操作

## 三、核心原理

### 3.1 制备方法

| 方法 | 反应式 | 适用范围 | 特点 |
|:---|:---|:---|:---|
| **金属锂法** | RX + 2 Li → RLi + LiX | R = 烷基、芳基 | 经典方法；副反应少 |
| **卤素-锂交换** | ArX + n-BuLi → ArLi + n-BuX | ArX、乙烯基 X（X = Br, I）| 低温（-78°C）；保留构型 |
| **锂-氢交换（金属化）** | Ar-H + n-BuLi → ArLi + n-BuH | 芳环上有定向金属化基团（DMG）| 区域选择性金属化 |
| **还原锂化** | R₂Hg + Li → 2 RLi | 特殊烷基 | 毒性大，较少用 |

**卤素-锂交换的关键**：
- **速度极快**：通常在 -78°C（干冰/丙酮浴）下进行，以控制选择性
- **构型保留**：乙烯基溴与 t-BuLi 在低温下交换，**保留双键构型**
- **I > Br >> Cl**：氟代烃不发生交换
- **t-BuLi 是最佳交换试剂**：位阻大、碱性极强，交换后生成的 t-BuX 不再与 ArLi 反应

### 3.2 反应机理通式

与 Grignard 类似，有机锂与羰基反应也经由 **配位-加成** 机理，但 C-Li 键更强极化使 R⁻ 更 "自由"：

```
   R⁻ Li⁺                  R
    |    \                 |
    |     C=O   →          C-O⁻Li⁺
    |    /                 |
   R'                     R'
```

- **配位**：Li⁺ 与羰基氧配位（比 Mg²⁺ 配位弱，但 R⁻ 更活泼）
- **加成**：R⁻ 进攻羰基碳
- **后处理**：H₃O⁺ 质子化得醇

## 四、关键结论

1. **有机锂比格氏试剂反应性更强**：可与位阻更大的酮反应；可与更惰性的底物反应
2. **与酯/酰氯反应也需 2 当量**：第一当量加成得酮中间体，第二当量继续加成得 3° 醇
3. **LDA 是最常用的非亲核性强碱**：大位阻（二异丙基氨基）阻止其对羰基的亲核进攻，专用于夺取 α-H 形成烯醇负离子
4. **与 α,β-不饱和羰基主要 1,2-加成**（硬亲核体），但比格氏试剂更容易发生副反应
5. **定向金属化（DoM）是竞赛高级技巧**：利用芳环上的 DMG（如 -CONR₂、-OCH₃、-NHCOR）引导 Li 在特定位置取代，实现选择性官能团化

## 五、常见分类

### 5.1 常见有机锂试剂

| 试剂 | 结构 | 特点 | 主要用途 |
|:---|:---|:---|:---|
| **n-BuLi**（正丁基锂）| CH₃(CH₂)₃Li | 最强碱之一；市售溶液（己烷/环己烷）| 卤素-锂交换、金属化、制备其他试剂 |
| **s-BuLi**（仲丁基锂）| CH₃CH₂CH(Li)CH₃ | 比 n-BuLi 更强碱性和亲核性 | 难金属化的底物 |
| **t-BuLi**（叔丁基锂）| (CH₃)₃CLi | **极强碱性和位阻**；昂贵 | 卤素-锂交换首选（交换后 t-BuX 不副反应）|
| **PhLi**（苯基锂）| PhLi | 亲核性适中 | 苯基化试剂 |
| **MeLi**（甲基锂）| MeLi | 亲核性强 | 甲基化 |
| **LDA**（二异丙基氨基锂）| (i-Pr)₂NLi | **非亲核性强碱** | 烯醇负离子制备；动力学去质子 |
| **LiHMDS** | (Me₃Si)₂NLi | 比 LDA 更弱碱、更大位阻 | 敏感底物的去质子 |

### 5.2 主要反应类型

```
有机锂试剂 RLi
├── 与羰基化合物
│   ├── 醛 → 2° 醇（比格氏快）
│   ├── 酮 → 3° 醇（可与位阻酮反应）
│   ├── 酯 → 3° 醇（2 eq）
│   └── CO₂ → 羧酸
├── 与环氧化物
│   └── 开环得醇（进攻位阻小端）
├── 与含活泼氢化合物（分解）
│   ├── H₂O → RH + LiOH
│   ├── 醇 → RH + LiOR
│   ├── 端炔 → RH + LiC≡CR
│   └── 胺 → RH + LiNR₂
├── 作为强碱
│   ├── 形成烯醇负离子（LDA）
│   ├── 定向金属化（DoM）
│   └── 夺取其他酸性 H（如苄位、烯丙位）
├── 卤素-锂交换
│   ├── ArBr + n-BuLi → ArLi（-78°C）
│   └── 乙烯基卤 → 乙烯基锂（保留构型）
└── 转金属化
    ├── 2 RLi + CuI → R₂CuLi（Gilman 试剂）
    └── RLi + ZnCl₂ → RZnCl（Negishi 偶联前体）
```

## 六、适用条件

### 6.1 制备与储存

| 因素 | 要求 | 原因 |
|:---|:---|:---|
| **溶剂** | 无水非极性（己烷、环己烷、乙醚、THF）| 质子性溶剂会淬灭 RLi |
| **温度** | 低温（-78°C 至 0°C）| 防止副反应；控制选择性 |
| **气氛** | N₂ 或 Ar（严格无氧）| O₂ 氧化 RLi 为过氧化锂或醇锂 |
| **浓度** | 市售 n-BuLi 约 1.6 M（己烷）| 高浓度易聚集，降低反应性 |
| **稳定剂** | 无（纯溶液）或少量共溶剂 | 某些配方含少量 LiCl 以稳定 |

### 6.2 卤素-锂交换的最佳实践

| 参数 | 条件 | 原因 |
|:---|:---|:---|
| **温度** | -78°C（干冰/丙酮）| 防止交换产物再交换（平衡控制）|
| **时间** | 5-30 min | 交换极快，但需确保完全 |
| **试剂选择** | t-BuLi > n-BuLi > s-BuLi | t-BuX 产物不进一步反应；n-BuBr 可能偶联 |
| **底物** | ArBr > ArI >> ArCl | I 交换最快，但 ArI 贵；Br 是最佳性价比 |
| **溶剂** | THF（优于 Et₂O）| THF 稳定 Li⁺，促进交换 |

### 6.3 LDA 制备烯醇负离子的条件

| 目标 | 条件 | 结果 |
|:---|:---|:---|
| **动力学烯醇负离子** | LDA，THF，-78°C | 夺取位阻小、酸性较强的 α-H（较少取代的 α-H）|
| **热力学烯醇负离子** | NaOEt/EtOH 或 KH/THF，rt | 夺取更取代的 α-H（更稳定烯醇）|
| **动力学控制烷基化** | LDA/-78°C 后加 RX | 动力学烯醇被捕获，得较少取代产物 |
| **热力学控制烷基化** | NaOEt/rt 后加 RX | 热力学烯醇被捕获，得较多取代产物 |

## 七、常见比较

### 7.1 有机锂 vs Grignard 试剂

| 维度 | 有机锂（RLi）| Grignard（RMgX）|
|:---|:---|:---|
| **C-金属键极化** | **更强** | 强 |
| **亲核性** | **更强** | 强 |
| **碱性** | **更强** | 强 |
| **与位阻酮** | 反应 | 慢或还原竞争 |
| **与酯** | 2 eq → 3° 醇（快）| 2 eq → 3° 醇 |
| **卤素-锂交换** | **可** | 不可 |
| **定向金属化** | **可** | 不可 |
| **对水和氧敏感度** | **极高** | 高 |
| **操作难度** | 更难（需更低温度）| 相对简单 |
| **价格** | 较贵 | 便宜 |

### 7.2 有机锂试剂之间的选择

| 试剂 | 碱性 | 位阻 | 亲核性 | 最佳用途 |
|:---|:---:|:---:|:---:|:---|
| **n-BuLi** | 极强 | 小 | 强 | 通用强碱/金属化/制备其他试剂 |
| **t-BuLi** | 最强 | 大 | 中等 | 卤素-锂交换（交换后副反应少）|
| **s-BuLi** | 极强 | 中等 | 强 | 难金属化底物 |
| **LDA** | 强 | **极大** | **极弱** | 非亲核性强碱（专一烯醇制备）|
| **LiHMDS** | 中等 | 极大 | 极弱 | 比 LDA 更温和的选择性去质子 |

## 十、例题

### 例题 1：LDA 制备动力学烯醇负离子

**题目**：2-甲基环己酮用 LDA（THF，-78°C）处理，主要形成哪种烯醇负离子？与 NaOEt/EtOH（rt）处理有何不同？

**解析**：

**LDA/-78°C（动力学控制）**：
- LDA 位阻极大，优先夺取 **位阻较小的 α-H**
- 2-甲基环己酮有两个 α-位：
  - C2（连甲基，位阻大，但有 1 个 α-H）
  - C6（无甲基，位阻小，有 2 个 α-H）
- LDA 优先夺取 **C6-H**（位阻小，动力学易达）
- 产物：**C1=C6 烯醇负离子**（较少取代的双键）

**NaOEt/rt（热力学控制）**：
- 可逆条件，烯醇负离子可相互转化
- **更稳定的烯醇负离子占优**：C1=C2 烯醇负离子（三取代双键 vs C1=C6 的二取代双键）
- 产物：**C1=C2 烯醇负离子**（较多取代，热力学稳定）

**合成意义**：
- 动力学烯醇 + MeI → 6-甲基-2-甲基环己酮
- 热力学烯醇 + MeI → 2,2-二甲基环己酮

### 例题 2：卤素-锂交换后的合成

**题目**：以对溴苯甲醚为原料，用 t-BuLi 处理后的产物与 DMF 反应，再酸性水解，最终产物是什么？

**解析**：
1. **卤素-锂交换**：p-Br-C₆H₄-OCH₃ + t-BuLi → **p-Li-C₆H₄-OCH₃** + t-BuBr（-78°C）
2. **与 DMF 反应**：芳基锂进攻 DMF 的羰基 → 四面体中间体 → collapse（Me₂N⁻ 离去）→ **p-甲氧基苯甲醛**
3. **产物**：**茴香醛（4-甲氧基苯甲醛）**

**注意**：-OCH₃ 是 **定向金属化基团（DMG）**，可引导锂在其邻位取代；但本题中 Br 在对位，交换直接在对位发生。

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| 混淆 n-BuLi 的 "丁基" 与丁烷 | 以为 n-BuLi 是丁烷衍生物 | n-BuLi = 正丁基锂（CH₃CH₂CH₂CH₂Li），是 **丁烷去掉一个 H 后连 Li** | n-BuLi 与 CO₂ 反应后水解，产物是什么？ |
| 认为 LDA 是亲核试剂 | 名字里有 "锂" | LDA 位阻极大，**不能**作为亲核体进攻羰基；只做碱 | 为什么 LDA 与丙酮混合后不会加成，而 MeLi 会？ |
| 卤素-锂交换温度太高 | 未理解交换平衡 | 必须在 **-78°C** 进行，否则交换产物可能再交换，导致混合物 | 为什么卤素-锂交换要干冰浴？ |
| 认为有机锂与格氏试剂对酯反应不同 | 未理解共性 | 两者与酯反应都需 **2 当量**，最终都得 3° 醇 | 有机锂与酯反应会停在酮吗？ |
| 忽略芳环上的 DMG 效应 | 只记卤素-锂交换 | 含 -OCH₃、-CONR₂ 的芳环可被 n-BuLi **直接金属化**（不经过卤素交换）| 苯甲醚与 n-BuLi 反应，锂进入哪个位置？ |

## 十二、🎯 教学视角
### 12.1 学习路径

```

     ↓

```

### 12.2 认知误区

| 误区 | 学生典型想法 | 纠正策略 |
|:---|:---|:---|
| "有机锂就是 Grignard 的加强版" | 只注意强度差异 | 强调有机锂独有的能力：**卤素-锂交换**、**定向金属化**、**LDA 非亲核碱性** |
| "t-BuLi 比 n-BuLi 弱" | 以为位阻大就弱 | t-BuLi 是 **最强**的常用有机锂之一；位阻阻止副反应，但碱性和交换能力极强 |
| "所有锂试剂都亲核" | 忽略 LDA 的特殊性 | LDA = **碱但不是亲核体**；这是其最核心的合成价值 |
| "卤素-锂交换会乱序" | 担心选择性 | 强调低温下的高选择性，以及 Br/I 的位点专一性 |

### 12.3 入门例题

**最简产物预测**：
> 写出 PhLi 与下列底物反应、酸性后处理后的产物：
> 1. 丙酮  2. CO₂  3. 环氧乙烷

**答案**：
1. **二甲基苯基甲醇**（PhC(OH)(CH₃)₂，3° 醇）
2. **苯甲酸**（PhCOOH）
3. **3-苯基-1-丙醇**（PhCH₂CH₂OH）
**引导**：若学生第 3 题错，问"环氧乙烷有几个碳？PhLi 的苯基进攻后，分子变成几个碳？"

### 12.4 现实类比

**"狙击手"类比有机锂试剂**：
- 有机锂 = 一位极其精准且强力的狙击手（R⁻）
- Grignard = 普通步兵（也强，但不如狙击手精准和致命）
- **LDA** = 狙击手的 "专用观察员"（只负责标记目标 = 去质子，不开枪 = 不亲核进攻）
- **卤素-锂交换** = 狙击手通过瞄准镜快速锁定目标（Br/I 位点），一击必中，且不留痕迹（t-BuX 离去，不干扰）
- **-78°C** = 狙击手的 "夜间作战"——低温下敌人（副反应）不活跃，狙击手行动最精准

## 十三、竞赛拓展

### 13.1 竞赛中的有机锂高级应用

| 技术/反应 | 特点 | 竞赛价值 |
|:---|:---|:---|
| **定向金属化（DoM）** | 利用 -OCH₃、-CONR₂、-SO₂NR₂ 等 DMG 引导锂化位置 | 芳环选择性官能团化 |
| **协同金属化-去质子（CMD）** | Pd 催化下碱协助的 C-H 活化 | 现代方法学 |
| **Shapiro 反应** | 对甲苯磺酰腙 + 2 eq RLi → 烯烃 | 酮 → 烯烃的转化 |
| **Corey-Seebach 反应** | 1,3-二噻烷 + RLi → 酰基负离子等价物 | 极性反转策略 |
| **Wittig 反应的锂盐变体** | 磷叶立德锂盐 | 烯烃合成 |

### 13.2 与现代方法学的交叉

- **Negishi 偶联**：RLi + ZnCl₂ → RZnCl + Pd 催化偶联 → 芳基-芳基键构建
- **光氧化还原催化**：某些有机锂试剂可在光催化下产生自由基，拓展反应类型
- **流动化学**：有机锂试剂在微反应器中的连续流合成，提高安全性和选择性
