# 手性面（Planar Chirality）

## 一、定义

**手性面（Planar Chirality）**是指分子的手性来源于某一平面（或准平面）结构两侧取代基的**不对称分布**，使得分子整体与其镜像无法重叠。手性面是除手性中心和手性轴之外的第三大类型手性来源。

- **核心特征**：
  1. 无经典手性中心（sp³ 杂化碳连四个不同基团）
  2. 无手性轴（无明显的轴状排列）
  3. 存在一个参考平面（如芳香环平面、金属配合物的配位平面）
  4. 平面两侧的取代基分布不对称，导致无对称面、无对称中心
- **标记法**：扩展 CIP 规则用 **Rₚ/Sₚ**（p = planar）标记，或 **M/P**（Minus/Plus）螺旋标记

## 三、核心原理

### 3.1 手性面产生的结构条件

手性面的产生需要三个条件：
1. 分子中存在一个**参考平面**（如芳香环、共轭体系、金属配位平面）
2. 平面上的取代基排列**破坏了对称性**
3. 平面**两侧**的取代基分布**不对称**

### 3.2 常见手性面类型

| 类型 | 参考平面 | 不对称来源 | 典型例子 |
|:---|:---|:---|:---|
| **环蕃型（Cyclophane）** | 苯环平面 | 桥链不在对称位置 | [2.2]对环蕃衍生物 |
| **二茂铁型** | 两个 Cp 环的平行平面 | Cp 环上取代基不对称 | 1,2-二取代二茂铁 |
| **轮烯型** | 轮烯的准平面 | 取代基导致平面扭曲 | 某些取代轮烯 |
| **金属配合物型** | 配位平面 | 配体排列不对称 | 平面四方型配合物 |
| **酮类/内酯型** | 羰基平面 | α-位不对称取代 | 某些环状酮 |

### 3.3 环蕃（Cyclophane）的手性面

**结构特征**：
- 两个苯环通过桥链（—CH₂—）连接
- 桥链的长度、位置和取代模式决定对称性
- 当桥链的位置和取代方式使分子**无对称面**时，产生手性面

**[m.n]对环蕃**：
- 两个对位取代的苯环通过 m 个和 n 个亚甲基桥连接
- 若桥链长度不同或取代不对称 → 手性面

### 3.4 二茂铁（Ferrocene）的手性面

**二茂铁结构**：
- Fe 夹在两个平行的环戊二烯（Cp）环之间
- Cp 环可自由旋转（能垒低）
- 但当 Cp 环上有合适取代基时，旋转受阻或取代基分布不对称 → 手性面

**产生手性面的条件**：
- 1,2-二取代二茂铁（邻位二取代）
- 两个 Cp 环上的取代模式使分子整体不对称
- 若取代基阻止 Cp 环自由旋转，则对映体可分离

```
        R¹          R²
         |          |
    Cp --Fe-- Cp
         |          |
        R³          R⁴

当 R¹, R², R³, R⁴ 的排列使分子无对称面时 → 手性面
```

## 四、关键结论

1. **手性面是"无手性中心也无手性轴的手性分子"**：手性来源于二维平面的两侧不对称
2. **环蕃是研究手性面的经典体系**：桥链的位置和长度精确控制对称性
3. **二茂铁衍生物在不对称催化中有应用**：平面手性二茂铁配体（如 Josiphos）用于工业化不对称合成
4. **手性面的标记比手性中心和手性轴更复杂**：需要明确参考平面和"前后"方向
5. **天然产物中手性面相对罕见**：大多数天然手性分子来源于手性中心或手性轴

## 五、常见分类

### 5.1 环蕃型手性面

| 化合物 | 桥链 | 手性来源 | 说明 |
|:---|:---|:---|:---|
| **[2.2]对环蕃** | 两个 —CH₂CH₂— | 桥链位置固定 | 若桥链两端取代不同 → 手性面 |
| **[m.n]环蕃** | m≠n | 桥链长度不同 | 天然不对称 |
| **取代环蕃** | 等长桥链 | 桥上或环上不对称取代 | 破坏对称面 |

### 5.2 二茂铁型手性面

| 化合物 | 取代模式 | 手性来源 | 应用 |
|:---|:---|:---|:---|
| **1,2-二取代二茂铁** | 一个 Cp 环上 1,2-位有取代基 | Cp 环平面两侧不对称 | 手性配体前体 |
| **1,1'-二取代二茂铁** | 两个 Cp 环各有一个取代基 | 若取代基不同且无对称面 | 手性配体 |
| **Josiphos 配体** | 复杂的二茂铁二膦配体 | 平面手性 + 手性中心 | 工业化不对称氢化 |

### 5.3 其他手性面类型

| 类型 | 例子 | 说明 |
|:---|:---|:---|
| **扭曲轮烯** | 某些取代 [n]轮烯 | 平面扭曲导致手性 |
| **杯芳烃** | 不对称取代杯[4]芳烃 | 锥形构象的手性 |
| **金属配合物** | 平面四方型 Pt(II) 配合物 | 四个不同配体 |

## 六、适用条件

### 6.1 手性面的判定

| 步骤 | 操作 |
|:---|:---|
| 1 | 识别分子中的参考平面 |
| 2 | 检查平面上是否有对称面 |
| 3 | 检查平面两侧取代基是否对称 |
| 4 | 若两侧不对称且无其他对称元素 → 手性面 |

### 6.2 Rₚ/Sₚ 标记法

**扩展 CIP 规则用于手性面**：

1. 确定**参考平面**
2. 选择平面外的一个** pilot atom**（引导原子，通常是离平面最近的取代基上的原子）
3. 从 pilot atom 向平面看，按 CIP 规则排序平面上的原子
4. 螺旋方向为顺时针 → **Rₚ**
5. 螺旋方向为逆时针 → **Sₚ**

**M/P 标记法（螺旋标记）**：
- 观察从参考平面的一侧到另一侧的"扭转"
- 右手螺旋（顺时针）→ **P**（Plus）
- 左手螺旋（逆时针）→ **M**（Minus）

## 七、常见比较

### 7.1 手性中心 vs 手性轴 vs 手性面

| 维度 | 手性中心 | 手性轴 | 手性面 |
|:---|:---|:---|:---|
| **手性来源** | 单个原子（点）| 轴状排列（线）| 平面两侧不对称（面）|
| **典型结构** | 乳酸、氨基酸 | 联苯、联烯、螺环 | 环蕃、二茂铁、轮烯 |
| **标记法** | R/S | Rₐ/Sₐ 或 M/P | Rₚ/Sₚ 或 M/P |
| **对映体关系** | 镜像不重叠 | 镜像不重叠 | 镜像不重叠 |
| **常见程度** | 最常见 | 较常见 | 相对罕见 |
| **竞赛考查** | 必考 | 常考 | 偶有涉及 |

### 7.2 环蕃 vs 二茂铁手性面

| 维度 | 环蕃 | 二茂铁 |
|:---|:---|:---|
| **参考平面** | 苯环平面 | Cp 环平面 |
| **手性来源** | 桥链位置不对称 | 取代基排列不对称 |
| **旋转自由度** | 低（桥链固定）| Cp 环可旋转（但取代基可阻止）|
| **对映体稳定性** | 通常很高 | 取决于旋转能垒 |
| **应用** | 超分子化学、材料 | 不对称催化配体 |

## 十、例题

### 例题 1：手性面判断

**题目**：判断下列分子是否具有手性面。

(a) [2.2]对环蕃（两个苯环通过两个 —CH₂CH₂— 桥在对位连接）  
(b) 1,1'-二甲基二茂铁  
(c) 1,2-二甲基二茂铁

**解析**：

**(a) [2.2]对环蕃**：
- 两个桥链等长，连接位置对称
- 分子有对称面（通过两个桥链的平面）
- **无手性面**

**(b) 1,1'-二甲基二茂铁**：
- 两个 Cp 环各有一个 —CH₃
- 若两个 —CH₃ 在对应位置，分子可能有对称面
- Cp 环可自由旋转
- **无稳定手性面**

**(c) 1,2-二甲基二茂铁**：
- 一个 Cp 环上 1,2-位有两个 —CH₃
- 另一个 Cp 环无取代（或不同取代）
- 分子无对称面
- **有手性面**（若 Cp 旋转受阻则对映体可分离）

### 例题 2：三类手性综合判断

**题目**：判断下列分子的手性类型（手性中心、手性轴、手性面，或无手性）。

(a) 6,6'-二硝基-2,2'-联苯二甲酸  
(b) 1,3-二苯基联烯  
(c) [2.2]间环蕃（桥在间位）

**解析**：

**(a)**：联苯衍生物，邻位有大位阻 → **手性轴**（阻转异构）

**(b)**：联烯，两端各有两个不同基团 → **手性轴**

**(c)**：间环蕃，桥链在间位连接 → 苯环平面两侧不对称 → **手性面**

### 例题 3：对称性分析

**题目**：分析 1,2-二取代二茂铁（一个 Cp 环上有两个不同取代基，另一个 Cp 环无取代）的对称元素，并判断是否具有手性面。

**解析**：

1. **参考平面**：Cp 环的平面
2. **取代基分布**：一个 Cp 环上有两个不同取代基（如 R¹ 和 R²），另一个 Cp 环无取代
3. **对称面检查**：
   - 无通过 Fe 和取代基的对称面（因为 R¹ ≠ R²）
   - 无垂直于 Cp 环的对称面
4. **对称中心检查**：无
5. **交替对称轴检查**：无
6. **结论**：分子**无对称面、无对称中心** → **具有手性面**

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| "环蕃都有手性面" | 忽略对称性 | 对位等长桥链的环蕃可能有对称面 | [2.2]对环蕃有手性面吗？ |
| "二茂铁都有手性面" | 忽略取代基条件 | 需要**不对称取代**才能产生手性面 | 二茂铁本身有手性吗？ |
| "手性面 = 平面分子" | 概念混淆 | 手性面分子的参考平面**两侧不对称**，整体不是平面的 | 手性面分子是平面的吗？ |
| "Rₚ/Sₚ 和 R/S 标记法相同" | 不了解扩展规则 | 手性面用 **Rₚ/Sₚ**（planar），与 R/S 不同 | 手性中心、轴、面的标记有何不同？ |
| "手性面比手性中心高级" | 主观判断 | 三类手性是**平等**的分类，无高低之分 | 手性中心、轴、面哪个"更基本"？ |

### 12.3 入门例题

**最简判断练习**：
> 1. 手性面产生的关键条件是什么？
> 2. [2.2]对环蕃有手性面吗？为什么？
> 3. 二茂铁本身有手性吗？什么情况下二茂铁衍生物会有手性面？

**答案**：
1. 参考平面 + 平面两侧取代基**不对称分布**
2. **没有**，因为等长桥链在对位连接，分子有对称面
3. 二茂铁本身**无手性**（高度对称）；当 Cp 环有**不对称取代**且旋转受阻时，可产生手性面
