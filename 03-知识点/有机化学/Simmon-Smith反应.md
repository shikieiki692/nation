# Simmon-Smith 反应（Simmon-Smith Reaction）
- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[37-加成反应]]、[[43-亲电加成反应]]
---

## 一、定义

**Simmon-Smith 反应**：烯烃与 **二碘甲烷 / 锌铜偶（CH₂I₂ / Zn(Cu)）**反应，生成**环丙烷衍生物**的反应。

**本质**：反应中生成的**碘甲基锌碘化物（ICH₂ZnI）**作为**类卡宾（carbenoid）**试剂，将 :CH₂ 等价物转移到双键上，形成三元环。

**特点**：
- **立体专一性**：syn 加成，保留烯烃的原有立体化学
- **无副反应**：不像自由卡宾那样发生插入反应
- **条件温和**：无需高温或强光

## 三、核心原理

### 3.1 试剂与活性物种

**试剂组合**：
- **CH₂I₂**（二碘甲烷）
- **Zn(Cu)**（锌铜偶，锌粉用 CuSO₄ 溶液处理活化）

**活性物种的形成**：
```
CH₂I₂  +  Zn(Cu)  →  ICH₂-ZnI  （类卡宾，carbenoid）
```

- 不是自由卡宾（:CH₂），而是**有机锌试剂**
- Zn-C 键使卡宾碳"稳定化"，降低活性
- 因此称为 **carbenoid（类卡宾）**

### 3.2 反应机理

**协同的 [2+1] 环加成**：
```
    R                   R
     \\                 /
      C=C    +  ICH₂ZnI   →    CH₂
     /      \\                 /    \
    R        \\               R      R
              \\             /
               [过渡态]    ---
```

**关键特征**：
- **一步协同**：没有中间体
- **syn 加成**：CH₂ 从双键的一侧加入
- **立体化学保留**：顺式烯烃 → 顺式取代环丙烷；反式烯烃 → 反式取代环丙烷

### 3.3 类卡宾 vs 自由卡宾

| | **Simmon-Smith（类卡宾）** | **自由卡宾（:CH₂）** |
|:---|:---|:---|
| **生成方式** | CH₂I₂ + Zn(Cu) | CH₂N₂ 热解或光解 |
| **活性物种** | ICH₂ZnI（有机金属）| :CH₂（中性活性中间体）|
| **立体化学** | **syn 加成** | 单线态 syn；三线态非立体专一 |
| **副反应** | 极少 | 插入反应（C-H）常见 |
| **选择性** | 高（只与双键反应）| 低（多种副反应）|
| **安全性** | 较安全 | 重氮甲烷易爆、剧毒 |

### 3.4 烯烃反应性

| 烯烃类型 | 反应性 | 原因 |
|:---|:---:|:---|
| **富电子烯烃** | 快 | 类卡宾亲电性，偏好富电子双键 |
| **缺电子烯烃** | 慢 | 双键电子密度低 |
| **位阻大的烯烃** | 慢 | 三元环过渡态位阻敏感 |
| **共轭二烯** | 可选择性单环丙烷化 | 电子密度差异 |

---

## 四、关键结论

1. **Simmon-Smith = 类卡宾环丙烷化 = syn 加成**
2. **活性物种是 ICH₂ZnI，不是自由卡宾**——因此无插入副反应
3. **立体化学完全保留**——顺式烯烃得顺式环丙烷，反式得反式
4. **富电子烯烃反应更快**——类卡宾具有亲电性
5. **Zn(Cu) 中的 Cu 作用**：活化 Zn 表面，促进 Zn 插入 C-I 键

---

## 五、常见分类或情形

### 5.1 按环丙烷化试剂分类

| 试剂 | 活性物种 | 特点 |
|:---|:---|:---|
| **CH₂I₂ / Zn(Cu)** | ICH₂ZnI | 最经典，syn 加成 |
| **CH₂I₂ / Et₂Zn** | ICH₂ZnEt | 更活泼，副反应稍多 |
| **CH₂N₂ / Pd/Cu** | :CH₂（金属稳定化）| 过渡金属催化变体 |
| **自由卡宾（:CH₂）** | :CH₂ | 副反应多，选择性差 |

### 5.2 按底物分类

| 底物 | 产物特征 | 备注 |
|:---|:---|:---|
| **单取代烯烃** | 单取代环丙烷 | 通用 |
| **1,2-二取代烯烃** | 1,2-二取代环丙烷 | 顺/反保留 |
| **环烯烃** |  bicyclo[n.1.0] 体系 | 形成桥头环丙烷 |
| **共轭二烯** | 单或双环丙烷化 | 可控制当量选择性 |

## 六、适用条件与限制

1. **必须无水**——Zn 试剂对水极其敏感
2. **Zn(Cu) 需新鲜制备**——放置后活性下降
3. **CH₂I₂ 昂贵且光敏感**——需避光保存
4. **不适用于含活泼氢的底物**——OH、NH、COOH 会消耗试剂
5. **温度通常 0°C ~ 室温**——过高导致副反应
6. **对端烯反应性较好**——内烯（尤其四取代）可能较慢

---

## 七、常见比较与易混点

### 7.1 Simmon-Smith vs 自由卡宾环丙烷化

| | **Simmon-Smith** | **自由卡宾** |
|:---|:---|:---|
| **立体化学** | 严格 syn | 单线态 syn；三线态混合 |
| **副反应** | 几乎无 | C-H 插入常见 |
| **条件** | 温和 | 剧烈（热/光）|
| **安全性** | 相对安全 | 重氮甲烷剧毒易爆 |

### 7.2 Simmon-Smith vs 催化氢化

- Simmon-Smith：在双键上加 CH₂ → 环丙烷（双键保留，成三元环）
- 催化氢化：在双键上加 H₂ → 烷烃（双键消失）
- 两者都是 syn 加成，但产物完全不同

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：顺-2-丁烯经 Simmon-Smith 反应后的产物是什么？

**解答**：
- Simmon-Smith 是 **syn 加成**
- 顺-2-丁烯 → **顺-1,2-二甲基环丙烷**
- 两个甲基在环丙烷同侧

### 10.2 例题 2（★★ 立体化学）

**题目**：环己烯经 Simmon-Smith 反应后的产物是什么？说明立体化学。

**解答**：
- 环己烯 + CH₂ → **二环[4.1.0]庚烷**（nortricyclane 骨架）
- syn 加成 → CH₂ 从双键的一面加入
- 产物是**环丙烷环与环己烷环顺式稠合**

## 十一、易错点

1. ❌ **认为 Simmon-Smith 用自由卡宾** → 用的是**类卡宾（carbenoid）**ICH₂ZnI
2. ❌ **忽略 syn 加成** → 顺式烯烃得顺式环丙烷，反式得反式
3. ❌ **与催化氢化混淆** → Simmon-Smith 加 CH₂ 成环丙烷，不是加 H₂
4. ❌ **忘记无水条件** → Zn 试剂遇水分解
5. ❌ **对缺电子烯烃期望过高产率** → 类卡宾亲电性，缺电子双键反应慢
