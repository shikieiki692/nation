# 有机波谱分析（Organic Spectroscopy）

## 一、定义

**有机波谱分析**是利用电磁波与有机分子相互作用产生的光谱信息，来推断分子结构的分析方法。在有机化学研究和竞赛中，波谱分析是**结构鉴定的核心手段**，通常需要综合使用多种波谱数据进行结构推导。

- **四大波谱方法**：
  - **红外光谱（IR）**：分子振动/转动跃迁，鉴定官能团
  - **核磁共振（NMR）**：核自旋能级跃迁，确定原子连接方式和化学环境
  - **质谱（MS）**：分子离子化后按质荷比分离，确定分子量和碎片结构
  - **紫外-可见光谱（UV-Vis）**：电子跃迁，判断共轭体系
- **分析策略**：先由 MS 确定分子量/分子式 → IR 确定官能团 → NMR 确定骨架 → UV-Vis 辅助判断共轭

## 三、核心原理

### 3.1 红外光谱（IR）

**原理**：分子吸收红外辐射后发生振动能级跃迁（伸缩振动、弯曲振动）。只有引起偶极矩变化的振动才产生红外吸收。

**关键波数区域**：

| 波数范围 (cm⁻¹) | 归属 | 典型官能团 |
|:---:|:---|:---|
| 3650–3200 | O–H, N–H 伸缩 | 醇、酚、羧酸、胺、酰胺 |
| 3300–2700 | C–H 伸缩 | sp³ C–H (~2900), sp² C–H (~3100), sp C–H (~3300), 醛基 C–H (~2720, 2820 双峰) |
| 2250–2100 | C≡C, C≡N 伸缩 | 炔烃、腈 |
| 1850–1650 | C=O 伸缩 | 醛、酮、羧酸、酯、酰胺、酰氯、酸酐 |
| 1680–1500 | C=C, C=N, N–H 弯曲 | 烯烃、芳环、胺 |
| 1500–1300 | C–H 弯曲 | CH₃ (~1465, 1375), CH₂ (~1465) |
| 1300–1000 | C–O, C–N 伸缩 | 醇、醚、酯、胺 |
| 900–650 | =C–H 面外弯曲 | 烯烃取代模式、芳环取代模式 |

**C=O 伸缩波数特征**：

| 羰基类型 | ν(C=O) / cm⁻¹ | 说明 |
|:---|:---:|:---|
| 酰氯 | ~1800 | 强吸电子 Cl 使频率升高 |
| 酸酐 | ~1820, ~1760 | 双峰 |
| 酯 | ~1740–1735 | — |
| 醛 | ~1740–1720 | 常伴随 ~2720 的醛基 C–H |
| 酮 | ~1725–1710 | — |
| 羧酸 | ~1720–1700 | 二聚体使频率降低 |
| 酰胺 | ~1680–1630 | 共振使 C=O 减弱 |

### 3.2 核磁共振（NMR）

#### 3.2.1 ¹H NMR（质子核磁共振）

**化学位移 δ（ppm）参考**：

| δ 范围 | 质子类型 | 示例 |
|:---:|:---|:---|
| 0.5–2.0 | 烷基 C–H | R–CH₃, R–CH₂–R, 环烷烃 |
| 1.5–3.0 | 烯丙位/炔丙位 C–H, 与电负性基团相邻 | R–CH₂–X, R–CH₂–Ar |
| 2.0–3.0 | 炔氢 | R–C≡C–H |
| 2.1–2.5 | 与羰基相邻 | CH₃–CO–R |
| 3.0–4.5 | 与 O/N/卤素相邻 | R–O–CH₃, R–CH₂–Cl |
| 4.5–6.5 | 烯氢 | R₂C=CH₂, RCH=CHR |
| 6.5–8.5 | 芳氢 | 苯环、杂芳环 |
| 9.0–10.0 | 醛氢 | R–CHO |
| 10.0–13.0 | 羧酸氢 | R–COOH |
| 1.0–5.0 | 活泼氢（O–H, N–H）| 位置可变，D₂O 交换后消失 |

**自旋-自旋耦合（n+1 规则）**：
- 相邻碳上有 n 个等价氢 → 裂分为 n+1 重峰
- 峰面积比符合二项式系数（单峰:双峰:三重峰:四重峰 = 1:1:2:1:3:3:1:4:6:4:1）

**常见耦合常数 J**：

| 耦合类型 | J / Hz | 特征 |
|:---|:---:|:---|
| ³J（邻位，sp³）| 6–8 | 自由旋转时约 7 Hz |
| ³J（邻位，烯烃，反式）| 12–18 | 反式 > 顺式 |
| ³J（邻位，烯烃，顺式）| 6–12 | — |
| ²J（同碳）| 12–15 | 同碳二氢 |
| ⁴J（W-耦合）| 1–2 | 特定几何构型 |
| 芳环邻位 | 6–9 | — |
| 芳环间位 | 1–3 | — |
| 芳环对位 | 0–1 | 通常不裂分 |

#### 3.2.2 ¹³C NMR（碳核磁共振）

| δ 范围 | 碳类型 | 示例 |
|:---:|:---|:---|
| 0–50 | sp³ 烷基碳 | CH₃, CH₂, CH, C |
| 50–100 | 与 O/N/卤素相连的 sp³ 碳 | C–O, C–N, C–Cl |
| 100–150 | sp² 芳碳/烯碳 | 苯环、烯烃 |
| 150–170 | 芳碳连杂原子 | Ar–O, Ar–N |
| 160–220 | 羰基碳 | C=O（醛/酮/羧酸衍生物）|

**DEPT 技术**：区分 CH₃（正峰）、CH₂（负峰）、CH（正峰）、季碳（不出现）。

### 3.3 质谱（MS）

**主要离子类型**：
- **分子离子峰 M⁺·**：失去一个电子形成的自由基阳离子，其 m/z = 分子量
- **碎片离子**：分子离子裂解产生的离子
- **同位素峰**：M+1（¹³C）、M+2（³⁷Cl、⁸¹Br 等）

**常见裂解规律**：

| 裂解类型 | 说明 | 示例 |
|:---|:---|:---|
| α-裂解 | 羰基化合物的特征裂解 | 醛、酮、酯、酰胺 |
| McLafferty 重排 | 含 γ-H 的羰基化合物 | 酮、醛、酸、酯 |
| 苄基裂解 | 苄位 C–C 键断裂 | 烷基苯 → C₇H₇⁺ (m/z 91) |
| 逆 Diels-Alder | 环己烯类化合物 | — |
| 脱水/脱醇 | 醇/醚失去 H₂O/ROH | M–18, M–ROH |

**特征碎片**：

| m/z | 归属 | 来源 |
|:---:|:---|:---|
| 15 | CH₃⁺ | 甲基 |
| 29 | CHO⁺, C₂H₅⁺ | 醛、烷基 |
| 43 | CH₃CO⁺, C₃H₇⁺ | 甲基酮、丙基 |
| 57 | C₄H₉⁺, C₃H₅CO⁺ | 丁基、乙基酮 |
| 77 | C₆H₅⁺ | 苯基 |
| 91 | C₇H₇⁺ | 苄基 |
| 105 | C₆H₅CO⁺ | 苯甲酰基 |

**卤素同位素特征**：
- 一个 Cl：M : M+2 ≈ 3 : 1
- 一个 Br：M : M+2 ≈ 1 : 1
- 两个 Cl/Br：M : M+2 : M+4 呈特征比

### 3.4 紫外-可见光谱（UV-Vis）

**原理**：π→π* 和 n→π* 电子跃迁。

**Woodward-Fieser 规则（二烯 λmax 估算）**：
- 基础值：非环/异环二烯 217 nm；同环二烯 253 nm
- 每增加一个共轭双键 +30 nm
- 烷基取代 +5 nm（每个）
- 环外双键 +5 nm
- 助色团（–OAc, –OR, –SR, –Cl/Br, –NR₂）各有一定增量

**特征吸收**：

| 跃迁类型 | λmax 范围 | ε (L·mol⁻¹·cm⁻¹) | 说明 |
|:---|:---:|:---:|:---|
| σ→σ* | < 150 nm | 强 | 远紫外，饱和化合物 |
| n→σ* | 150–250 nm | 弱 | 含杂原子饱和化合物 |
| π→π* | 160–250 nm | 强 (ε > 10⁴) | 孤立双键 |
| n→π* | 270–300 nm | 弱 (ε < 100) | 羰基化合物 |
| 共轭 π→π* | 210–400 nm | 强 | 共轭体系，红移 |

## 四、关键结论

1. **IR 的核心价值是官能团鉴定**：羰基区域（1850–1650 cm⁻¹）和指纹区（< 1500 cm⁻¹）最为关键
2. **¹H NMR 是结构解析的“骨架图”**：化学位移确定环境，积分确定氢数，裂分确定相邻氢数
3. **分子离子峰是质谱分析的起点**：结合同位素峰和氮规则（含奇数 N 时分子量为奇数）推断分子式
4. **共轭体系增长 → UV λmax 红移**：Woodward-Fieser 规则可估算共轭二烯的 λmax
5. **波谱综合解析是竞赛考查的核心能力**：单一波谱往往不足以确定结构，需要交叉验证

## 六、适用条件

### 6.1 各波谱方法的适用范围

| 问题 | 首选波谱 | 辅助波谱 |
|:---|:---|:---|
| 分子量/分子式 | MS | 元素分析 |
| 官能团鉴定 | IR | — |
| 碳骨架/氢分布 | ¹H NMR, ¹³C NMR | 2D NMR |
| 共轭体系判断 | UV-Vis | NMR |
| 立体化学 | ¹H NMR (J 值) | NOE |
| 手性判断 | CD/ORD | X-ray |

### 6.2 谱图解析的一般流程

1. **MS**：确定分子离子峰 → 分子量 → 同位素峰推断卤素等 → 氮规则
2. **元素分析/高分辨 MS**：确定分子式 → 计算不饱和度
3. **IR**：确定主要官能团（C=O, O–H, N–H, C≡C 等）
4. **¹H NMR**：
   - 计算总氢数（与分子式核对）
   - 根据 δ 值分类氢的环境
   - 根据积分确定各类氢的数目
   - 根据裂分确定相邻氢数
   - 推断结构片段
5. **¹³C NMR/DEPT**：确定碳骨架，区分 CH₃/CH₂/CH/季碳
6. **综合**：拼接所有信息，提出可能结构，验证所有数据

## 十、例题

### 例题 1：¹H NMR 谱图解析

**题目**：某化合物 C₄H₈O 的 ¹H NMR 数据如下，推断其结构。
- δ 1.05 (t, 3H), δ 2.47 (q, 2H), δ 9.79 (t, 1H)

**解析**：

1. **不饱和度**：Ω = (2×4 + 2 - 8)/2 = 1（一个双键或环）
2. **δ 9.79 (t, 1H)**：醛氢特征，说明是醛
3. **δ 2.47 (q, 2H)**：与醛基相邻的 CH₂（–CH₂–CHO）
4. **δ 1.05 (t, 3H)**：末端 CH₃，与 CH₂ 相邻（CH₃–CH₂–）
5. **结构**：**CH₃CH₂CHO（丙醛）**

### 例题 2：IR + 分子式推断

**题目**：某化合物 C₃H₆O，IR 在 1715 cm⁻¹ 有强吸收，无 2720 cm⁻¹ 吸收。判断结构。

**解析**：

1. **不饱和度**：Ω = (2×3 + 2 - 6)/2 = 1
2. **1715 cm⁻¹**：C=O 伸缩振动
3. **无 2720 cm⁻¹**：排除醛（醛基 C–H 特征双峰缺失）
4. **结论**：只能是 **酮**
5. **结构**：**CH₃COCH₃（丙酮）**

### 例题 3：MS 同位素峰分析

**题目**：某化合物 MS 中 M : M+2 = 3 : 1，说明什么？

**解析**：

1. **M : M+2 = 3 : 1** 是 **一个氯原子**（³⁵Cl : ³⁷Cl ≈ 3 : 1）的特征
2. 若为 Br：M : M+2 ≈ 1 : 1
3. 若为两个 Cl：M : M+2 : M+4 = 9 : 6 : 1
4. **结论**：分子中含有一个氯原子

## 十一、易错点

| 错误 | 原因 | 纠正 | 课堂提问 |
|:---|:---|:---|:---|
| "NMR 裂分只看相邻碳上的氢" | 忽略等价性要求 | 只有**等价氢**才符合 n+1 规则；不等价氢各自裂分 | 如果相邻碳上有 2 个 H 但不等价，峰如何裂分？ |
| "IR 中所有峰都要记住" | 过度焦虑 | 重点记**官能团区**特征峰，指纹区用于确认 | IR 中最重要的是哪个区域？ |
| "MS 中分子离子峰一定是最高峰" | 忽略例外 | 分子离子峰可能很弱甚至不出现（如醇、支链烷烃）| 什么类型的化合物分子离子峰较弱？ |
| "UV-Vis 可以判断所有双键" | 不了解选择定则 | 孤立 C=C 的 π→π* 在远紫外，常规 UV-Vis 看不到 | 乙烯在 200–400 nm 有吸收吗？ |
| "活泼氢位置固定" | 忽略氢键影响 | O–H, N–H 的 δ 受浓度、温度、溶剂影响极大 | 为什么乙醇的 O–H 化学位移会变化？ |

|:---|:---|:---|
| "波谱解析就是背表" | 忽视逻辑推理 | 强调：波谱解析是**信息整合与逻辑推理**，表格只是参考 |
| "NMR 只看化学位移" | 忽略积分和裂分 | 训练：每次看 NMR 必须同时看 δ、积分、裂分三个信息 |
| "质谱碎片没有规律" | 觉得裂解随机 | 总结常见裂解模式（α-裂解、McLafferty、苄基裂解）|

### 12.3 入门例题

**最简判断练习**：
> 1. 某化合物 IR 在 3300 cm⁻¹ 有宽吸收，¹H NMR δ 3.5 (s, 1H, 可交换)，判断官能团。
> 2. 某化合物 MS 分子离子峰 m/z = 60，¹H NMR 只有单峰 δ 2.1 (6H)，推断结构。
> 3. 某二烯 λmax = 273 nm（实测），按 Woodward-Fieser 规则估算是否与同环二烯一致。

**答案**：
1. **羟基（–OH）**，3300 cm⁻¹ 宽峰 + 可交换氢
2. **丙酮 CH₃COCH₃**，M = 58... 实际上 m/z 60 对应 **乙酸 CH₃COOH**（M=60），NMR δ 2.1 (s, 3H) 是甲基。重新看：若只有单峰 δ 2.1 (6H) 则 M=58 是丙酮。需修正：m/z 60 + δ 2.1 (s, 3H) → **乙酸**（羧酸氢可交换或宽峰未显示）。
3. 同环二烯基础值 253 nm，若有 4 个烷基取代 +20 → 273 nm，**一致**。
