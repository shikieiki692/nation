# Pummerer重排

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：含硫化合物相关（APiIC 占位）

## 一、定义

**Pummerer重排**是指 **亚砜（sulfoxide）在酸酐或酰卤作用下**，α-碳发生官能团化（酰氧基化或卤化）的反应。反应中亚砜的氧被取代，同时 α-碳引入新的亲电基团。

通式：
$$\mathrm{R_2S=O + (RCO)_2O \rightarrow R_2S^+ - OCOCR \rightarrow R-CH(SR')-OCOR}$$

## 二、考纲对应

- 中国化学奥林匹克考纲未单独列出，属于**含硫化合物**和**重排反应**的拓展内容。
- 在 APiIC 教材和竞赛高阶题中出现。

## 三、核心原理

### 3.1 反应机理（五步）

1. **O-酰基化**：亚砜氧进攻酸酐 → 酰氧基锍盐
2. **去质子**：碱（或分子内）夺取 α-H → 硫叶立德中间体
3. **分子内重排**：酰氧基从硫迁移到 α-碳
4. **水解（可选）**：酰氧基被水解为羟基

```
R-CH₂-S(=O)-R' + (Ac)₂O
  → R-CH₂-S⁺(OAc)-R'  (O-酰基化)
  → R-CH=S⁺(OAc)-R'   (去质子)
  → R-CH(OAc)-S-R'    (酰氧基迁移)
```

### 3.2 反应驱动力

- **硫的强亲电性**：酰基化后硫带正电，强烈吸引电子
- **α-H 的酸性增强**：邻近正电硫，α-H pKa 大幅下降
- **S-O 键断裂**：酰氧基离去能力优于氧负离子

## 四、关键结论

1. **亚砜 + 酸酐/酰卤 → α-酰氧基硫醚**（核心转化）
2. **反应经历硫叶立德中间体** —— 不是简单的 SN2
3. **α-H 是必须的** —— 无 α-H 的亚砜不发生 Pummerer 重排
4. **可扩展**：除酸酐外，TFAA、MsCl、SOCl₂ 等也可促进

## 五、常见分类或情形

| 变体 | 试剂 | 产物特征 |
|:---|:---|:---|
| **经典 Pummerer** | Ac₂O | α-乙酰氧基硫醚 |
| **硅基 Pummerer** | TMSCl / TFAA | α-硅氧基硫醚 |
| **还原 Pummerer** | TFAA + 还原剂 | 脱硫得醛 |
| **氧化 Pummerer** | 过酸 + 亲核试剂 | 伴随过度氧化 |

## 六、适用条件与限制

| 条件 | 说明 |
|:---|:---|
| **底物** | 必须有 α-H 的亚砜 |
| **试剂** | 酸酐（Ac₂O）、TFAA、酰氯（SOCl₂）|
| **溶剂** | 常用 CH₂Cl₂、DCE |
| **温度** | 通常 0°C → 回流 |
| **限制** | 无 α-H 不反应；过度氧化可能生成砜 |

## 七、常见比较与易混点

### Pummerer vs Swern 氧化

| 维度 | Pummerer 重排 | Swern 氧化 |
|:---|:---|:---|
| **底物** | 亚砜 | 醇 |
| **试剂** | 酸酐/酰卤 | DMSO + (COCl)₂ + Et₃N |
| **产物** | α-官能团化硫醚 | 醛/酮 |
| **共同点** | 都涉及硫的亲电活化 | — |

## 十、例题

### 例题 1：产物预测

**题目**：二甲基亚砜（DMSO）与乙酸酐反应的主要产物是什么？

**解析**：
DMSO 有两个甲基（都有 α-H）：
1. O-乙酰基化 → (CH₃)₂S⁺(OAc)
2. 去质子 → CH₂=S⁺(Me)(OAc)
3. 酰氧基迁移 → CH₂(OAc)-S-CH₃

产物：**甲硫基甲基乙酸酯**

## 十一、易错点

| 错误 | 纠正 |
|:---|:---|
| 忽略 α-H 的必要性 | 无 α-H 的亚砜不发生 Pummerer 重排 |
| 与 Swern 氧化混淆 | Swern 氧化醇 → 醛；Pummerer 重排亚砜 → α-酰氧基硫醚 |
