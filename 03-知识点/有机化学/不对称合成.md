# 不对称合成

> 不对称合成是获取单一对映体分子的核心策略，在药物、香料、农药等领域具有关键应用。生物体系是手性环境，对映体往往表现出截然不同的生物活性（如(S)-多巴可治疗帕金森病，而(R)-多巴具有毒性）。

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]

---

## 一、定义

**不对称合成（Asymmetric Synthesis）**：利用手性源（手性底物、手性试剂、手性催化剂或手性助剂）使前手性底物转化为富对映体产物的过程。核心目标是控制反应的**对映选择性（enantioselectivity）**，通常用**对映体过量（ee, enantiomeric excess）**衡量。

$$
\text{ee} = \frac{[R] - [S]}{[R] + [S]} \times 100\%
$$

---

## 二、考纲定位

- 对应考纲条目：有机合成中的立体化学控制
- 难度定位：进阶——需要掌握立体化学基础、金属有机化学等前置知识
- 竞赛权重：中等——决赛级别合成题中涉及手性中心构建时必需

## 三、核心原理

### 3.1 获取单一对映体的五种策略

| 策略 | 原理 | 优缺点 |
|:---|:---|:---|
| **手性池（Chiral Pool）** | 从天然手性化合物（氨基酸、糖、萜类等）出发 | 直接可靠，但结构受限，对映体可能不均等可得 |
| **拆分（Resolution）** | 用手性试剂将对映体混合物分离 | 最大产率50%（单一对映体），但两种对映体都可获得 |
| **手性助剂（Chiral Auxiliary）** | 将手性单元临时连接到底物上，反应后去除 | 化学计量消耗，但可回收；非对映选择性控制 |
| **手性试剂（Chiral Reagent）** | 用手性试剂直接对非手性底物反应 | 化学计量消耗，成本较高 |
| **不对称催化（Asymmetric Catalysis）** | 用手性催化剂催化前手性底物的反应 | 催化剂用量极少，最理想的方法 |

> **效率排序**：手性池 < 拆分 < 手性助剂 < 手性试剂 < 不对称催化

### 3.2 手性池策略

- **氨基酸**：天然L-氨基酸（除半胱氨酸为R外，其余大多为S构型）是最易得的手性池成员
- **羟基酸**：乳酸、扁桃酸、苹果酸、酒石酸等可从天然来源获取；酒石酸是Sharpless不对称环氧化的关键配体来源
- **糖类**：甘露糖 → 甘露醇 → 双缩醛保护 → 裂解得被保护的甘油醛
- **局限**：合成路线必须"硬塞进"可用起始原料的结构中；很多手性池成员只以一种对映体形式廉价可得

### 3.3 手性助剂策略

**核心逻辑**：通过临时连接光学纯辅助基团，将**对映选择性**问题转化为**非对映选择性**问题。

**流程**：
1. 将光学纯手性助剂连接到非手性底物上
2. 手性助剂使底物产生非对映异位面，发生非对映选择性反应
3. 去除手性助剂，得到光学纯产物（原则上可回收助剂）

**常见手性助剂**：

| 助剂 | 来源/特点 | 应用 |
|:---|:---|:---|
| **Evans噁唑烷酮** | 由(S)-缬氨酸廉价制得 | 不对称烷基化、羟醛反应（aldol） |
| **Oppolzer樟脑磺内酰胺（sultam）** | 基于樟脑骨架 | 共轭加成等反应 |
| **8-苯基薄荷醇** | 基于薄荷醇骨架 | Diels-Alder反应 |
| **伪麻黄碱助剂** | 基于伪麻黄碱 | 不对称烷基化 |

### 3.4 不对称催化策略

**核心逻辑**：手性催化剂或配体创造不对称环境，使前手性底物的两面产生能量差异。

#### 3.4.1 CBS还原（Corey-Bakshi-Shibata Reduction）

- **试剂**：手性硼杂环（由脯氨酸衍生的氨基醇）+ 硼烷（BH₃）
- **底物**：芳基-烷基酮（两个取代基空间差异明显时效果最佳）
- **机理**：硼杂环的Lewis酸性B与羰基O络合，Lewis碱性N与BH₃络合；负氢通过六元环状过渡态迁移
- **对映选择性**：大取代基（R_L）倾向于占据假平伏键位置
- **催化量**：通常约10 mol%

#### 3.4.2 Ru/TsDPEN催化转移氢化

- **催化剂**：Ru(II)配合物 + (S,S)-TsDPEN（N-对甲苯磺酰基-1,2-二苯基乙二胺）
- **还原剂**：H₂、异丙醇或甲酸
- **特点**：催化剂用量极低（<<1%），适用于工业生产
- **Nobel奖**：Noyori（Ru/Rh催化还原）、Sharpless（Os/Ti催化氧化）、Knowles（工业不对称氢化）获2001年诺贝尔化学奖

#### 3.4.3 Rh/BINAP不对称氢化

- **配体**：BINAP（2,2'-双二苯膦基-1,1'-联萘），手性源于联萘的阻转异构
- **特点**：适用于多种不饱和羧酸衍生物的高对映选择性还原

#### 3.4.4 Sharpless不对称环氧化（AE）

- **底物**：烯丙醇（局限性：仅对烯丙醇有效）
- **试剂**：Ti(Oi-Pr)₄ + 酒石酸二乙酯（DET）+ t-BuOOH
- **催化量**：钛和酒石酸酯都可用催化量
- **机理**：双核Ti配合物，烯丙醇与Ti络合后，过氧配体从特定面传递氧原子
- **对映选择性**：L-(+)-DET与D-(-)-DET给出相反构型的环氧
- **应用**：雌舞毒蛾信息素（disparlure）、普萘洛尔（propranolol）等合成

#### 3.4.5 Jacobsen环氧化

- **适用范围**：**简单烯烃**（弥补了Sharpless AE仅对烯丙醇有效的不足）
- **催化剂**：Mn(III)-salen配合物
- **特点**：对顺式烯烃效果较好，对反式烯烃选择性较低

#### 3.4.6 Sharpless不对称双羟基化（AD）

- **试剂**：K₂OsO₂(OH)₄ + K₃Fe(CN)₆ + 手性配体（DHQD₂PHAL 或 DHQ₂PHAL）
- **配体**：基于二氢奎宁（DHQ）或二氢奎宁定（DHQD）的酞嗪二聚体
- **催化量**：Os用量极低
- **特点**：对几乎所有烯烃都有效（富电子或缺电子），选择性极高（>98% ee常见）
- **AD-mix**：商品化试剂混合物。AD-mix-β含DHQD₂PHAL；AD-mix-α含DHQ₂PHAL
- **对映选择性模型**：底物按"大-中-小"排列，DHQD配体使Os从顶面进攻，DHQ从底面进攻

### 3.5 配体加速催化（Ligand Accelerated Catalysis）

- **核心思想**：手性配体不仅提供手性环境，还**加速**了反应
- **为什么重要**：如果无配体时"背景反应"速率也很快，则外消旋产物会稀释ee值
- **理想情况**：背景反应极慢，有手性配体时反应极快——此时对映选择性最高
- **实例**：Sharpless AD、二烷基锌对醛的加成

### 3.6 碳-碳键的不对称形成

- **二烷基锌对醛的加成**：Et₂Zn与醛在无催化剂时反应极慢；手性氨基醇催化剂显著加速反应，并赋予对映选择性；产物手性仲醇ee可达90%以上

## 四、关键结论

1. **不对称合成的五种策略效率递增**：手性池 < 拆分 < 手性助剂 < 手性试剂 < 不对称催化
2. **手性助剂的核心逻辑**：通过临时连接光学纯辅助基团，将**对映选择性**问题转化为**非对映选择性**问题
3. **不对称催化的核心逻辑**：手性催化剂或配体创造不对称环境，使前手性底物的两面产生能量差异
4. **CBS还原适用于芳基-烷基酮**；空间差异越大，选择性越好
5. **Sharpless AE仅对烯丙醇有效**，但极为可靠；Jacobsen环氧化适用于简单烯烃
6. **Sharpless AD几乎适用于所有烯烃**，且选择性极高，是合成手性二醇的首选方法
7. **配体加速催化**是不对称反应设计的重要原则——好的不对称反应应满足"有配体快、无配体慢"
8. **2001年诺贝尔化学奖**：Knowles、Noyori、Sharpless，表彰金属催化不对称反应

## 五、常见分类或情形

### 5.1 按控制方式分类

| 控制方式 | 手性来源 | 化学计量/催化 | 典型实例 |
|:---|:---|:---:|:---|
| **底物控制** | 底物本身的手性 | 化学计量 | 手性池合成 |
| **试剂控制** | 手性试剂 | 化学计量 | (-)-鹰爪豆碱介导的去质子化 |
| **催化剂控制** | 手性催化剂/配体 | 催化量 | CBS还原、Sharpless AE/AD |
| **手性辅基控制** | 手性助剂（临时连接） | 化学计量（可回收） | Evans助剂不对称烷基化 |

### 5.2 按反应类型分类

| 反应类型 | 代表性方法 | 底物范围 | 典型ee |
|:---|:---|:---|:---:|
| 不对称还原 | CBS还原、Ru/TsDPEN、Rh/BINAP | 酮、烯烃 | 90-99% |
| 不对称环氧化 | Sharpless AE、Jacobsen | 烯丙醇、简单烯烃 | 90-98% |
| 不对称双羟基化 | Sharpless AD | 几乎所有烯烃 | >98% |
| 不对称C-C键形成 | 二烷基锌加成、Evans Aldol | 醛、酮 | 90-99% |

## 十、例题

### 例题 1（难度 ⭐⭐⭐）
**题目**：为什么CBS还原对芳基-烷基酮的选择性优于二烷基酮？

**答案**：CBS还原的立体选择性来源于六元环状过渡态中取代基的空间排斥。芳基-烷基酮中芳基（大）与烷基（小）的空间差异显著，大取代基倾向于占据假平伏键位置，从而明确区分两个对映面。而二烷基酮中两个烷基空间差异较小，过渡态能量差不大，因此对映选择性较低。

**解析**：理解CBS还原的过渡态模型是解题关键。空间差异越大，两个对映过渡态的能量差越大，选择性越好。

### 例题 2（难度 ⭐⭐⭐⭐）
**题目**：某化合物含有非烯丙醇的双键，需要进行不对称环氧化，应选择哪种方法？

**答案**：应选择Jacobsen环氧化（Mn-salen催化）。因为Sharpless不对称环氧化（AE）仅对烯丙醇有效，而Jacobsen环氧化适用于简单烯烃，可以弥补Sharpless AE的不足。

**解析**：掌握两种不对称环氧化的适用范围差异是竞赛常考点。Sharpless AE对烯丙醇极为可靠，但对非烯丙醇烯烃无效；Jacobsen环氧化专门用于简单烯烃。

## 十一、易错点

1. **混淆五种策略的适用场景**
   - 错误：认为不对称催化适用于所有情况
   - 正确：手性池在特定骨架合成中更直接；拆分在需要两种对映体时更有优势

2. **Sharpless AE的底物限制**
   - 错误：认为Sharpless AE对所有烯烃都有效
   - 正确：**仅对烯丙醇有效**，这是其最大局限性

3. **忽略配体加速催化的重要性**
   - 错误：只关注手性配体的立体控制，忽略其对反应速率的加速作用
   - 正确：好的不对称反应应满足"有配体快、无配体慢"，才能最大化ee值

4. **CBS还原的底物选择**
   - 错误：对所有酮类底物都尝试CBS还原
   - 正确：CBS对芳基-烷基酮效果最佳，二烷基酮选择性差

## 十二、🎯 教学视角

### 12.1 学习路径

1. **先掌握立体化学基础**（R/S命名、对映体/非对映体、前手性）
2. **理解对映体过量的概念和计算**
3. **系统学习五种策略的原理和典型实例**
4. **重点掌握不对称催化的几种经典反应**（CBS、Sharpless AE/AD、Jacobsen）
5. **通过合成设计题练习手性中心的构建策略**

### 12.2 学生易踩的认知误区

- **误区**："不对称合成就是用手性催化剂"
- **纠正**：五种策略各有优劣，需根据底物特点和合成目标灵活选择
- **误区**："ee值越高越好，方法不重要"
- **纠正**：工业应用中催化剂成本、底物适用范围、操作简便性同样重要

### 12.3 入门级例题

**题目**：L-(+)-酒石酸二乙酯（DET）在Sharpless AE中起什么作用？

**答案**：DET作为手性配体与Ti形成双核配合物，创造不对称环境，使过氧配体从特定面传递氧原子，从而控制环氧的构型。L-(+)-DET与D-(-)-DET给出相反构型的产物。

### 12.4 与现实/直觉的连接

- **药物手性**：沙利度胺（反应停）事件——R-对映体有镇静作用，S-对映体致畸
- **自然界的手性偏好**：生物体系普遍使用L-氨基酸和D-糖，不对称合成帮助人类模仿自然

## 十三、竞赛拓展

1. **不对称共轭加成**：有机铜试剂的不对称SN2'选择性
2. **有机催化不对称反应**：小分子催化剂（如脯氨酸衍生物）催化的不对称Aldol反应
3. **动态动力学拆分**：在拆分过程中使不需要的对映体原位消旋化，理论上可达100%产率
4. **不对称氢甲酰化**：Rh催化的烯烃不对称氢甲酰化构建手性醛
