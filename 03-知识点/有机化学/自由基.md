# 自由基（Free Radical）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[34-有机活性中间体]]、[[32-有机反应机理]]、[[36-取代反应]]

---

## 一、定义

**自由基（Free Radical）**：带有**未成对电子**的中性物种。中心原子（通常为碳）外层有 7 个电子，通过超共轭和共轭效应趋于平面化（接近 sp² 杂化）。

**基本特征**：
- 电中性，但具有高度反应活性
- 单电子既可作为**氧化剂**（接受电子配对），也可作为**还原剂**（给出电子）
- 典型寿命：微秒 ~ 毫秒（溶液中），可因离域而大幅延长

> ⭐ **ABOC 行 746**："自由基是具有单电子的物种，如 CH₃^·，这里的点符号表示具有一个单电子。"

## 三、核心原理

### 3.1 自由基稳定性 5 规则（ABOC §2.7.4）

| 规则 | 内容 | 示例 |
|:---|:---|:---|
| **规则 1** | **离域优先**：只要能离域，就比不能离域的稳定 | 烯丙基自由基 > 叔丁基自由基 |
| **规则 2** | **不能离域的：3° > 2° > 1°** | 叔丁基 > 异丙基 > 乙基 > 甲基 |
| **规则 3** | **杂原子邻位稳定**：连给电子或吸电子基团都提高稳定性 | ·CH₂OH、·CH₂CN 都比 ·CH₃ 稳定 |
| **规则 4** | **推拉效应最佳**：一边吸电子、一边给电子的稳定化最好 | ·CH(CN)(OEt) 极稳定 |
| **规则 5** | **位阻稳定**：大位阻自由基更稳定 | Ph₃C·（三苯甲基自由基）可稳定存在 |

> ⭐ **ABOC 行 1739-1757**："规则 1：与碳正离子不同，自由基只要可以离域，就比不能离域的稳定...规则 2：不能离域的，3>2>1...规则 3：旁边连给电子或吸电子基团都能大大提高稳定性...规则 4：一边连着吸电子基团，另一边连着给电子基团的稳定化最好...规则 5：位阻大的自由基比较稳定..."

**与碳正离子稳定性的关键区别**：
- **碳正离子**：叔 > 烯丙基 > 苄基（超共轭主导）
- **自由基**：烯丙基 > 叔丁基（离域效应在自由基中更重要！）

> ⭐ **ABOC 行 1739 强调**："烯丙基自由基甚至比叔丁基自由基更加稳定，而叔丁基碳正离子比烯丙基正离子稳定很多。"

### 3.2 氧化性自由基 vs 还原性自由基

| 类型 | 定义 | 例子 | 反应行为 |
|:---|:---|:---|:---|
| **氧化性自由基** | 得到电子后更稳定 | ·O⁻、·OH、NO₂、TEMPO | 倾向于**夺取电子**或**氧化底物** |
| **还原性自由基** | 失去电子后更稳定 | 烷基自由基 R·、H· | 倾向于**给出电子**或**还原底物** |

**判别方法 — α 效应类比**：
- 若自由基中心连有电负性大的原子（O, N, 卤素）→ 电子云偏向该原子 → 缺电子 → **氧化性**
- 若自由基中心连有给电子基团（烷基）→ 电子云富集 → **还原性**

> ⭐ **ABOC 行 1671-1675**："氧化性自由基：得到电子稳定的自由基。还原性自由基：失去电子稳定的自由基。判断方法：α效应 (电子邻位排斥效应，增强亲核性，如 H₂O₂, N₂H₄)。"

### 3.3 链式反应三阶段

所有自由基链式反应都遵循 **引发 → 链转移 → 链终止** 三段式：

**① 引发（Initiation）**：
$$\text{BPO} \xrightarrow{\Delta} 2\,\text{Ph}^\cdot + 2\,\text{CO}_2$$
$$\text{AIBN} \xrightarrow{\Delta} 2\,(\text{CH}_3)_2\dot{\text{C}}\text{-CN} + \text{N}_2$$
- 热或光使引发剂均裂 → 产生初始自由基

**② 链转移（Propagation）**：
$$\text{Br}^\cdot + \text{R-H} \longrightarrow \text{HBr} + \text{R}^\cdot$$
$$\text{R}^\cdot + \text{Br}_2 \longrightarrow \text{R-Br} + \text{Br}^\cdot$$
- 自由基消耗 + 新自由基生成 → 自由基浓度维持
- 每一步都放热或轻微吸热 → 链可持续

**③ 链终止（Termination）**：
$$2\,\text{R}^\cdot \longrightarrow \text{R-R} \quad \text{(偶联)}$$
$$2\,\text{R}^\cdot \longrightarrow \text{R-H} + \text{烯烃} \quad \text{(歧化)}$$
- 两个自由基相遇 → 耗尽自由基 → 链结束

### 3.4 常见自由基反应类型

| 反应类型 | 机理要点 | 典型例子 |
|:---|:---|:---|
| **自由基取代** | 链转移中自由基抽氢 → 新自由基 | 烷烃卤代、NBS 烯丙位溴代 |
| **自由基加成** | 自由基对 π 键加成 → 新自由基 | HBr 过氧化物效应（反马氏）|
| **自氧化** | O₂ 与自由基结合 → 过氧自由基 | 油脂酸败、橡胶老化 |
| **聚合** | 自由基反复加成单体 → 高分子 | 聚乙烯、聚苯乙烯 |
| **碎片化** | 自由基 β-断裂 → 新自由基 + 烯烃 | Barton 脱羧、McMurry 偶联前步 |
| **Barton 反应** | 亚硝酸酯光解 → 烷氧自由基 → [1,5]-H 迁移 → 碳自由基 → NO 捕获 → 肟 | 甾体 remote functionalization |
| **自由基环化** | 不饱和自由基对分子内 π 键的加成 → 环状自由基 | 5-exo-trig 优先（Baldwin 规则）|

### 3.5 Barton 反应机理详解（Ch.10 视角）

**Barton 反应**是自由基化学中**远程官能团化**的经典策略，核心是通过 [1,5]-氢迁移实现距离反应中心 5 个键的位置的官能团化。

**机理步骤**：
```
① R-O-NO + hv → R-O· + ·NO  （亚硝酸酯光解）
     ↓
② R-O· → [1,5]-H 迁移 → ·C-H（δ-位的碳自由基）
     ↓
③ ·C-H + NO → C=NOH（肟）
```

**关键特征**：
- **[1,5]-H 迁移**通过六元环过渡态进行，几何要求与 Diels-Alder 类似
- 最终产物是**肟**（oxime），可进一步转化为胺、腈等
- 在甾体化学中尤为重要——可实现角甲基的远程氧化

**Barton-McCombie 脱氧**（变体）：
- RO-CS-咪唑 → 光解 → 烷氧自由基 → β-断裂 → 脱去 CS=O → 脱氧产物
- 是将醇还原为烷烃的温和方法

### 3.6 自由基环化与 Baldwin 规则（Ch.10 视角）

**自由基环化**是构建环状骨架的重要方法，机理为：
$$
\ce{•C-(CH2)n-CH=CH2 → 环状自由基 → 捕获 → 环化产物}
$$

**Baldwin 规则在自由基环化中的应用**：

| 环大小 | exo-trig | endo-trig | 说明 |
|:---:|:---:|:---:|:---|
| **3** | × 不利 | × 不利 | 三元环张力大 |
| **4** | × 不利 | × 不利 | 四元环张力大 |
| **5** | ✅ **有利** | △ 边界 | **5-exo-trig 是最有利的自由基环化** |
| **6** | △ 边界 | ✅ 有利 | 6-endo-trig 也可发生 |
| **≥7** | — | — | 大环通常不通过自由基环化构建 |

**竞赛要点**：
- **5-exo-trig 通常优先于 6-endo-trig**——这是自由基环化的核心选择性
- 原因：5-exo 的过渡态轨道重叠更好，且生成的自由基更稳定
- **识别标志**：分子内有双键 + 自由基引发条件 → 优先考虑 5-exo-trig 环化

### 3.7 持久自由基（Persistent Radicals）

普通自由基极活泼，但以下因**空间位阻 + 电子离域**而可长期稳定：

| 自由基 | 结构特点 | 应用 |
|:---|:---|:---|
| **TEMPO** | 氮氧自由基，大位阻 | 氧化催化剂、自由基捕获剂 |
| **DPPH** | 二苯基苦基肼基，高度离域 | 自由基定量检测（EPR 标准）|
| **加尔万氧基** | 苯氧自由基，共轭稳定 | 抗氧化剂 |
| **三苯甲基** Ph₃C· | 三个苯环位阻 + 离域 | 历史上首个被分离的自由基 |

> ⭐ **ABOC 行 1759**："一些稳定的自由基：NO，NO₂，O₂，TEMPO，加尔万氧基自由基，Fremy盐等。"

## 四、关键结论

1. **自由基稳定性 5 规则是核心判据**——尤其注意"离域 > 叔"与碳正离子的区别
2. **链式反应三段式必须掌握**——引发、链转移、链终止
3. **过氧化物效应 = 反马氏 HBr 加成**——自由基加成决定方向
4. **NBS 是"溴单质均匀释放器"**——维持低浓度 Br₂，避免加成副反应
5. **氧化性/还原性自由基的判别是竞赛新考点**——α 效应类比法

## 六、适用条件与限制

1. **对氧气敏感**——O₂ 是三线态双自由基，可与几乎所有自由基快速反应
2. **对金属离子敏感**——过渡金属可催化自由基的氧化/还原终止
3. **选择性有限**——自由基取代的位点选择性不如离子型反应精确
4. **链长有限**——动力学链长通常 10²~10⁴，随后终止
5. **NBS 只适用于烯丙位/苄位溴代**——普通烷烃位置选择性差

## 七、常见比较与易混点

### 7.1 自由基取代 vs 离子型取代

| | **自由基取代** | **离子型取代（SN2）** |
|:---|:---|:---|
| 中间体 | 自由基 | 碳正离子或五配位 TS |
| 条件 | 光照/加热/过氧化物 | 极性溶剂/酸碱 |
| 立体化学 | 无（平面自由基）| 翻转（SN2）或外消旋（SN1）|
| 选择性 | 3° > 2° > 1°（但烯丙位/苄位优先）| 甲基 > 1° > 2° > 3°（SN2）|
| 溶剂效应 | 不大 | 很大 |
| 对 O₂ | 极敏感 | 不敏感 |

### 7.2 过氧化物效应 vs 马氏规则

| | **正常 HBr 加成** | **过氧化物效应** |
|:---|:---|:---|
| 机理 | 离子型（碳正离子）| 自由基链式 |
| 产物 | 马氏产物 | **反马氏产物** |
| 引发 | 无需引发 | 需要过氧化物/BPO |
| 适用 | 所有 HX | **仅 HBr**（HF/HCl/HI 不行）|
| 原因 | 碳正离子稳定性 | 自由基稳定性 |

**为什么只有 HBr 有过氧化物效应？**
- HF：H-F 键太强，F· 无法有效抽氢
- HCl：Cl· 活性适中，但 H-Cl 键较强，链转移不利
- HBr：Br· 活性适中，H-Br 键适中 → **最佳平衡**
- HI：I· 活性太低，无法有效引发链转移；且 HI 是还原剂，会还原过氧化物

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：按稳定性排序：a) ·CH₃  b) ·CH₂CH=CH₂  c) ·C(CH₃)₃  d) ·CH₂CN

**解答**：
- b) 烯丙基自由基 → 离域稳定（规则 1）→ **最稳定**
- d) ·CH₂CN → 杂原子邻位（CN 吸电子，规则 3）
- c) 叔丁基 → 3°（规则 2）
- a) 甲基 → 无稳定化因素 → **最不稳定**

**答案**：b > d > c > a

> ⭐ **关键**：烯丙基 > 叔丁基！这是自由基与碳正离子的关键区别。

### 10.2 例题 2（★★ NBS 反应）

**题目**：环己烯 + NBS / (BPO, CCl₄, hv) → ?

**解答**：
1. BPO 热解 → Ph· + CO₂
2. Ph· + NBS → PhBr + 琥珀酰亚胺自由基
3. 琥珀酰亚胺自由基 → Br· + 琥珀酰亚胺
4. Br· 抽取烯丙位 H → 烯丙基自由基（离域稳定）
5. 烯丙基自由基 + NBS → 3-溴环己烯 + 琥珀酰亚胺自由基（链转移继续）

**产物**：3-溴环己烯（烯丙位溴代）

**关键点**：NBS 维持极低 Br₂ 浓度 → 避免 Br₂ 对双键的亲电加成 → 选择性发生烯丙位取代。

### 10.3 例题 3（★★ 过氧化物效应）

**题目**：丙烯 + HBr 在有过氧化物存在下的主产物是什么？写出链转移机理。

**解答**：
- **反马氏产物**：1-溴丙烷（Br 在末端碳）

链转移：
1. Br· + CH₃CH=CH₂ → CH₃CH·CH₂Br（2° 自由基，稳定）
2. CH₃CH·CH₂Br + HBr → CH₃CH₂CH₂Br + Br·（链继续）

**注意**：若第一步 Br· 加到末端碳 → CH₃CHBrCH₂·（1° 自由基，不稳定）→ 不利。

### 10.4 例题 4（★★★ 氧化性/还原性判别）

**题目**：判断以下自由基是氧化性还是还原性：·OH、·CH₃、TEMPO、·O₂⁻

**解答**：
- **·OH**：O 电负性大，得电子稳定 → **氧化性**
- **·CH₃**：烷基自由基，失电子成碳正离子稳定 → **还原性**
- **TEMPO**：N-O· 结构，N 电负性大，得电子成 N-O⁻ 稳定 → **氧化性**
- **·O₂⁻**：超氧阴离子自由基，已带负电，倾向于给出电子 → **还原性**（虽然 O 电负性大，但整体电荷决定）

---

### 10.5 例题 5（★★★ Barton 反应）

**题目**：写出亚硝酸环己酯在光照下的 Barton 反应机理，说明 [1,5]-H 迁移的关键作用。

**解答**：

**步骤 1：光解**
$$
\ce{环己基-O-NO + hv → 环己基-O· + ·NO}
$$

**步骤 2：[1,5]-H 迁移**（通过六元环过渡态）
$$
\ce{环己基-O· → [六元环TS] → δ-碳自由基 + 环己酮}
$$
更准确：烷氧自由基的 O· 抽取 δ-H → 在 δ-位形成碳自由基

**步骤 3：NO 捕获**
$$
\ce{δ-碳自由基 + ·NO → δ-亚硝基化合物 → 互变异构 → 肟}
$$

**反思**：
- [1,5]-H 迁移通过**六元环过渡态**，几何上与 Diels-Alder 的六元环 TS 类似
- 最终产物肟可水解为醛/酮，或还原为胺

### 10.6 例题 6（★★★★ 自由基环化）

**题目**：5-己烯-1-基自由基（·CH₂–CH₂–CH₂–CH=CH₂）发生分子内环化，预测主要产物并说明理由。

**分析**：
- 底物有末端双键 + 自由基 → 分子内环化
- 可能路径：
  - **5-exo-trig**：自由基进攻末端双键 → 五元环 + 伯碳自由基
  - **6-endo-trig**：自由基进攻内部双键碳 → 六元环 + 仲碳自由基

**解答**：
- **主要产物：5-exo-trig 环化产物（甲基环戊烷衍生物）**
- 原因：
  1. Baldwin 规则：5-exo-trig **有利**
  2. 轨道重叠：5-exo 的过渡态几何更优
  3. 虽然 6-endo 生成更稳定的仲碳自由基，但动力学上 5-exo 更快

**反思**：
- **5-exo-trig 通常优先于 6-endo-trig**——这是自由基环化的经典选择性
- 注意：热力学更稳定的产物不一定是动力学主要产物

## 十一、易错点

1. ❌ **把自由基稳定性顺序照搬碳正离子** → 自由基：烯丙基 > 叔；碳正离子：叔 > 烯丙基
2. ❌ **认为所有 HX 都有反马氏加成** → 仅 HBr，HF/HCl/HI 不行
3. ❌ **NBS 反应中把产物写成加成产物** → NBS 是取代（烯丙位），不是加成
4. ❌ **忽略 O₂ 对自由基反应的抑制** → 空气中做自由基反应常失败
5. ❌ **链终止写成离子型反应** → 终止是两个自由基相遇（偶联或歧化）
6. ❌ **认为持久自由基不反应** → TEMPO 虽稳定，但仍可氧化醇 → TEMPO/NaOCl 是温和氧化体系
7. ❌ **氧化性/还原性判别只看原子电负性** → 整体电荷和周围基团同样重要（如 ·O₂⁻）
8. ❌ **自由基取代位置选择性记混** → 自由基：3° > 2° > 1°，但烯丙位/苄位因离域而优先
9. ❌ **Barton 反应的 [1,5]-H 迁移记成 [1,3] 或 [1,7]** → Barton 反应特指 **[1,5]-H 迁移**（六元环过渡态），[1,3] 不利，[1,7] 几何不利
10. ❌ **自由基环化只考虑产物自由基稳定性** → 5-exo-trig 动力学优先，即使生成的自由基稳定性不如 6-endo
11. ❌ **忽略 Barton-McCombie 脱氧与 Barton 反应的区别** → Barton 反应 = 远程官能团化（生成肟）；Barton-McCombie = 脱氧（醇 → 烷烃）

### 12.3 入门级例题

**题目 1**：写出甲烷与氯气在光照下的链式反应机理（三段式）。

**预期解答路径**：
1. **引发**：Cl₂ →(hv) 2Cl·
2. **链转移**：
   - Cl· + CH₄ → HCl + ·CH₃
   - ·CH₃ + Cl₂ → CH₃Cl + Cl·
3. **终止**：2·CH₃ → CH₃CH₃（偶联）或 ·CH₃ + Cl· → CH₃Cl

**老师可引导的提问**：
- 卡在第 1 步：问 "光照提供了什么能量？Cl-Cl 键发生什么断裂？"
- 卡在第 2 步：问 "为什么 Cl· 抽 H 而不是加到 CH₄ 上？"
- 卡在第 3 步：问 "两个自由基相遇有几种方式？"

**题目 2（Ch.10 视角）**：某分子含有末端双键和自由基，为什么通常发生 5-exo-trig 环化而不是 6-endo-trig？

**预期解答路径**：
1. **引发**：Cl₂ →(hv) 2Cl·
2. **链转移**：
   - Cl· + CH₄ → HCl + ·CH₃
   - ·CH₃ + Cl₂ → CH₃Cl + Cl·
3. **终止**：2·CH₃ → CH₃CH₃（偶联）或 ·CH₃ + Cl· → CH₃Cl

**老师可引导的提问**：
- 卡在第 1 步：问 "光照提供了什么能量？Cl-Cl 键发生什么断裂？"
- 卡在第 2 步：问 "为什么 Cl· 抽 H 而不是加到 CH₄ 上？"
- 卡在第 3 步：问 "两个自由基相遇有几种方式？"

**题目 2 预期解答路径**：
1. 5-exo-trig 通过**五元环过渡态**，轨道重叠更好
2. Baldwin 规则明确指出 5-exo-trig 是**有利**的环化模式
3. 虽然 6-endo 可能生成更稳定的仲碳自由基，但**动力学上 5-exo 更快**

**老师可引导的提问**：
- 学生若答"六元环更稳定"：问 "热力学稳定和动力学有利是一回事吗？过渡态的几何谁更好？"

### 12.4 与现实/直觉的连接

- **类比**：自由基就像"独身主义者"——有一个单身电子渴望配对。它们会不择手段地"抢伴侣"（抽氢、加成），直到遇到另一个"独身主义者"（自由基-自由基相遇），两人配对结婚（偶联），反应链才终止。
- **反例**：油脂放久了会"哈喇"——这是自氧化链式反应。O₂ 与油脂中的烯丙位自由基结合 → 过氧自由基 → 过氧化物 → 醛酮臭味。抗氧化剂（如维生素 E）就是"自由基灭火器"——提供一个 H 让自由基饱和，从而中断链转移。

## 十三、竞赛拓展

1. **Barton 反应与 Barton-McCombie 脱氧**：
   - 亚硝酸酯光解 → 烷氧自由基 → [1,5]-氢迁移 → 碳自由基 → NO 捕获 → 肟
   - 是构建复杂多环体系的经典自由基策略

2. **Giese 反应（自由基对 C=C 的加成）**：
   - 碳自由基对缺电子烯烃的加成 → 新碳自由基 → 被 H-供体捕获
   - 形成 C-C 键的重要方法

3. **原子转移自由基聚合（ATRP）**：
   - Cu(I)/Ligand 调控的活性自由基聚合
   - 可控合成嵌段共聚物
   - 是自由基化学向高分子领域的延伸

4. **光氧化还原催化（Photoredox Catalysis）**：
   - Ru(bpy)₃²⁺ 或 Ir(ppy)₃ 等光敏剂 → 激发态 → 单电子转移 → 产生自由基离子
   - 2010 年代后的有机合成革命
   - 见 [[光氧化还原催化]]
