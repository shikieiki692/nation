# 卡宾（Carbene）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[34-有机活性中间体]]、[[32-有机反应机理]]

---

## 一、定义

**卡宾（Carbene）**：含有一个**二价碳原子**的中性物种，通式为 R₂C:，碳上只有两个取代基（R = H, 烷基, 芳基, 卤素等），并有一个**孤对电子**和一个**空 p 轨道**（单线态），或两个**未成对电子**（三线态）。

**电子特征**：
- 碳外层 **6 电子**（4 来自两根 σ 键 + 2 来自孤对电子或两个单电子）
- **不满足八隅律**，因此极活泼，通常作为反应中间体存在
- sp² 杂化为主（单线态），sp 杂化（三线态）

> ⭐ **ABOC 行 742**："卡宾是形如 :CH₂ 的物种，描述时需要画出孤对电子以表示碳外层 6 电子，只成两根键，不满足 8 电子规则，不稳定。"

## 三、核心原理

### 3.1 单线态 vs 三线态

| 特征 | **单线态卡宾（Singlet）** | **三线态卡宾（Triplet）** |
|:---|:---|:---|
| **电子排布** | 孤对电子在 sp² 杂化轨道配对，空 p 轨道 | 一个电子在 sp² 轨道，一个电子在 p 轨道，自旋平行 |
| **杂化** | sp² | sp（或近似 sp²）|
| **几何** | 弯曲，键角 ~102-103° | 弯曲，键角 ~130-150° |
| **磁性** | 抗磁性（无单电子） | 顺磁性（2 个单电子）|
| **反应性** | **协同反应**（类似亲电/亲核协同） | **分步反应**（类似双自由基）|
| **立体化学** | 环丙烷化：**保持**烯烃构型 | 环丙烷化：**部分外消旋化** |
| **能量** | 通常更低（尤其有给电子取代基时）| :CH₂ 基态为三线态（低 8 kcal/mol）|

**简单判别**：
- **给电子取代基**（如 :CCl₂, :C(OMe)₂）→ 孤对电子稳定 → **单线态**
- **吸电子/无取代**（如 :CH₂）→ 电子排斥小 → **三线态为基态**
- **NHC 类**（氮杂环卡宾）→ 强给电子 → **单线态**，且极其稳定

### 3.2 生成方式

**① α-消除（最常见）**：
$$\mathrm{CHCl}_3 + \mathrm{t\text{-}BuO}^- \longrightarrow \ :\mathrm{CCl}_2 + \mathrm{Cl}^- + \mathrm{t\text{-}BuOH}$$
- 强碱从碳上拔掉质子，同时离去基团离去
- 三卤甲烷（CHX₃）+ 强碱 → 二卤卡宾 :CX₂

**② 重氮化合物分解**：
$$\mathrm{CH}_2\mathrm{N}_2 \xrightarrow{h\nu\text{ 或 }\Delta} \ :\mathrm{CH}_2 + \mathrm{N}_2$$
- 光解或热解，放出 N₂（极好的离去基团）
- 可精确控制单线态/三线态（光敏剂选择性）

**③ 过渡金属催化分解**：
- 过渡金属（Rh, Cu, Pd）与重氮化合物配位 → 金属卡宾（类卡宾，Carbenoid）
- 比游离卡宾更温和、选择性更高

**④  Simmons-Smith 反应（类卡宾）**：
$$\mathrm{CH}_2\mathrm{I}_2 + \mathrm{Zn(Cu)} \longrightarrow \text{ICH}_2\text{ZnI} \quad (\text{类卡宾})$$
- 不生成游离卡宾，但实现环丙烷化

### 3.3 典型反应

**① 对 C=C 的环丙烷化（最重要）**：
- 单线态：协同 [2+1] 环加成，**保持烯烃 E/Z 构型**
- 三线态：分步（先形成一个 C-C 键，自由基关环），**部分外消旋化**

**② 对 C-H 的插入**：
$$\text{R-H} + \ :\mathrm{CR'}_2 \longrightarrow \text{R-CR'}_2\text{-H}$$
- 选择性：3° C-H > 2° C-H > 1° C-H（与自由基抽氢选择性类似）
- 分子内插入可构建环系

**③ 对杂原子孤对的加成（生成叶立德）**：
$$\text{R}_2\text{S} + \ :\mathrm{CR'}_2 \longrightarrow \text{R}_2\text{S}^+\text{-CR'}_2^- \quad (\text{硫叶立德})$$

**④ 二聚（副反应）**：
$$2\ :\mathrm{CH}_2 \longrightarrow \mathrm{CH}_2=\mathrm{CH}_2$$

### 3.4 稳定卡宾家族（ABOC 独特视角）

传统卡宾极不稳定，但以下物种具有"卡宾特征"且可被分离：

| 稳定卡宾 | 结构特征 | 稳定机制 |
|:---|:---|:---|
| **NHC**（氮杂环卡宾）| 咪唑啉-2-亚基 | 6 π 电子芳香环 + 两个 N 的给电子 |
| **CO** | :C≡O: | C 有孤对 + 空 p，但 O→C 的 π 反馈稳定 |
| **异腈 R-NC** | R-N⁺≡C⁻ | 与 CO 等电子，C 端有孤对 |
| **环状(烷基)(氨基)卡宾（CAAC）** | 五元环 + 一个 N + 一个 C | N 给电子 + 环张力释放 |

> ⭐ **ABOC 行 748**："近年来分离了一些稳定的卡宾，最常见的是 NHC(氮杂环卡宾)。这种卡宾通过芳香性和共轭效应被稳定。其实 CO 和异腈 R-NC 也是稳定卡宾。"

**NHC 的深远影响**：
- 2005 年 Grubbs 第二代催化剂用 NHC 替代 PR₃ → 稳定性大幅提升
- NHC 本身是有机催化剂（如安息香缩合的极性反转催化剂）
- NHC 作为配体在过渡金属催化中已成"标准配置"

### 3.5 乃春（Nitrene）——卡宾的氮类似物

$$\text{R-N:} \quad \text{或} \quad \text{R-N}^+\text{-R'}$$

- N 比 C 电负性更大，更需满足 8 电子 → **乃春比卡宾更不稳定**
- 生成方式：叠氮化合物热解（RN₃ → RN: + N₂）、异氰酸酯光解
- 反应：C-H 插入、对 C=C 的氮杂环丙烷化、Curtius 重排中间体

> ⭐ **ABOC 行 744**："乃春是形如 RN: 的卡宾氮类似物。由于氮的电负性更大，更需要满足 8 电子规则，因此更加不稳定。"

## 四、关键结论

1. **单线态 = 协同，三线态 = 分步**——这是判别环丙烷化立体化学的核心
2. **取代基决定自旋态**：给电子基 → 单线态；无取代/吸电子 → 三线态基态
3. **NHC 是有机催化革命的起点**——从"不稳定中间体"到"可分离催化剂"
4. **α-消除是竞赛中最常见的卡宾生成方式**——CHX₃ + 强碱 → :CX₂
5. **卡宾插入选择性 ≈ 自由基抽氢选择性**——3° > 2° > 1°，但机理不同（协同 vs 分步）

## 五、常见分类或情形

### 5.1 按取代基分类

| 类型 | 例子 | 自旋态 | 典型反应 |
|:---|:---|:---:|:---|
| 亚甲基 | :CH₂ | 三线态基态 | 环丙烷化、C-H 插入 |
| 二卤卡宾 | :CCl₂, :CBr₂, :CF₂ | 单线态 | 环丙烷化（与烯烃）|
| 烷氧基卡宾 | :C(OMe)₂ | 单线态 | 亲核性卡宾 |
| NHC | IMes, IPr | 单线态 | 催化、极性反转 |
| 金属卡宾 | Rh=CHCO₂Et | 取决于金属 | 环丙烷化、C-H 活化 |

### 5.2 环丙烷化的立体化学对照

```
顺式烯烃 + 单线态 :CCl₂  →  顺式环丙烷（协同，构型保持）
顺式烯烃 + 三线态 :CH₂   →  顺+反混合物（分步，自由基旋转）
```

### 5.3 金属卡宾：Fischer vs Schrock 分类（Zchem 中级无机结构化学网课素材 · 资产 T-2 · 可信度：高）

> 来源：[[教学逻辑提炼-Zchem-过渡元素的结构化学-第三轮深化]]

金属卡宾（Metal Carbene, M=CR₂）是有机金属化学的核心中间体，按电子性质分为两类：

| 对比维度 | **Fischer 卡宾** | **Schrock 卡宾** |
|:---|:---|:---|
| **金属氧化态** | 低氧化态（0, +1）| 高氧化态（+4, +5）|
| **取代基 R** | 吸电子基（CO₂Et, Ph）| 给电子基（H, 烷基）|
| **卡宾碳性质** | **亲电性**（δ⁺）| **亲核性**（δ⁻）|
| **成键特征** | 强 π 反馈，M→C 电子给予 | 强 σ 给予 + 弱 π 反馈 |
| **稳定性** | 较稳定，可分离 | 活泼，通常原位反应 |
| **典型实例** | (CO)₅W=C(OMe)Ph | Cp₂Ta=CH₂ |

**教学类比**：
- Fischer 卡宾 ≈ 羰基化合物（碳亲电）→ 发生亲核进攻
- Schrock 卡宾 ≈ 叶立德（碳亲核）→ 发生亲电进攻
- 两者分别对应 "M=C 双键的极性反转"

**结构化学本质**：
- 低氧化态金属富电子 → 强 π 反馈 → 卡宾碳电子密度高 → 但吸电子取代基拉走电子 → **净亲电**
- 高氧化态金属缺电子 → 弱 π 反馈 → 给电子取代基推高碳电子密度 → **净亲核**

## 六、适用条件与限制

1. **游离卡宾需原位生成**——不能储存，通常现场生成现场反应
2. **对空气/水敏感**——单线态卡宾可与 O₂ 反应
3. **插入反应选择性有限**——分子间插入常伴随多种副产物
4. ** Simmons-Smith 是替代方案**——当需要避免游离卡宾的副反应时用
5. **NHC 虽稳定但不具典型卡宾反应性**——作为配体/催化剂，不做环丙烷化

## 七、常见比较与易混点

### 7.1 卡宾 vs 碳正离子 vs 自由基

| | 卡宾 :CR₂ | 碳正离子 C⁺R₃ | 自由基 ·CR₃ |
|:---|:---|:---|:---|
| 价电子 | 6 | 6 | 7 |
| 电荷 | 中性 | +1 | 中性 |
| 空轨道 | 有（p）| 有（p）| 无 |
| 单电子 | 0（单线态）或 2（三线态）| 0 | 1 |
| 稳定性 | 极不稳定（NHC 除外）| 可稳定（离域）| 可稳定（离域）|
| 与烯烃反应 | 环丙烷化 | 亲电加成 | 烯丙位取代 |

### 7.2 类卡宾（Carbenoid）vs 游离卡宾

| | 游离卡宾 :CH₂ | 类卡宾 ICH₂ZnI |
|:---|:---|:---|
| 存在形式 | 独立物种 | 与金属配位的络合物 |
| 反应性 | 极高，副反应多 | 温和，选择性高 |
| 立体化学 | 单/三线态决定 | 通常类似单线态 |
| 副反应 | 二聚、插入 | 少 |

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：画出 :CCl₂ 的 Lewis 结构式，判断是单线态还是三线态，并解释原因。

**解答**：
- C 与两个 Cl 成单键，C 上有一对孤对电子，空 p 轨道
- 两个 Cl 的强吸电子诱导效应使 C 更缺电子 → 孤对电子倾向于配对（降低能量）
- **单线态**

```
    Cl
     \
      C:
     /
    Cl
```
（: 表示孤对电子在 sp² 轨道，空 p 轨道垂直于分子平面）

### 10.2 例题 2（★★ 环丙烷化立体化学）

**题目**：顺-2-丁烯分别与 :CCl₂（单线态）和 :CH₂（三线态）反应，产物的立体化学有何不同？

**解答**：
- **:CCl₂（单线态）**：协同 [2+1] 环加成，烯烃构型保持 → **顺式-1,1-二氯-2,3-二甲基环丙烷**
- **:CH₂（三线态）**：分步机理。先形成一个 C-C 键，生成双自由基中间体，此时 C-C 单键可旋转 → 产物为**顺+反混合物**

### 10.3 例题 3（★★★ 卡宾 vs 乃春）

**题目**：叠氮化合物 RN₃ 热解生成乃春 RN:，而重氮化合物 R₂CN₂ 光解生成卡宾 R₂C:。为什么乃春比卡宾更不稳定？

**解答**：
- N 的电负性（3.04）远大于 C（2.55）
- N 作为电负性大的原子，更"渴望"满足 8 电子规则
- 乃春中 N 只有 6 电子，这种"缺电子"在电负性大的原子上极不稳定
- 卡宾中 C 电负性较小，对 6 电子状态的容忍度更高

> ⭐ **ABOC 核心逻辑**：电负性大的原子越缺电子越不稳定。

### 10.4 例题 4（★★ NHC 稳定性来源）

**题目**：咪唑啉-2-亚基（一种 NHC）可在空气中稳定存在，而 :CH₂ 只能原位生成。用轨道理论解释。

**解答**：
- NHC 是五元芳香环的一部分，环上有 6 π 电子（符合 Hückel 4n+2）
- 卡宾碳的空 p 轨道是芳香 π 体系的一部分，被离域稳定
- 两个环上 N 原子向卡宾碳提供电子（给电子共轭），进一步降低能量
- 大的 N-取代基（如 Mes = 均三甲苯基）提供空间位阻保护，阻止二聚

## 十一、易错点

1. ❌ **认为所有卡宾都是三线态** → 给电子取代基使单线态稳定
2. ❌ **混淆卡宾与碳正离子** → 两者都是 6 电子，但卡宾中性、碳正离子 +1 价
3. ❌ **环丙烷化立体化学记反** → 单线态保持、三线态外消旋化
4. ❌ **认为 NHC 会做环丙烷化** → NHC 是稳定卡宾，主要作配体/催化剂，不做典型卡宾反应
5. ❌ **忽略乃春的存在** → Curtius 重排、Hofmann 重排中都涉及乃春中间体
6. ❌ **α-消除 vs β-消除混淆** → α-消除是同一碳上同时掉 H 和 LG，β-消除是相邻碳上掉 H
7. ❌ **类卡宾 = 卡宾** → Simmons-Smith 试剂不是游离卡宾，反应性不同
8. ❌ **卡宾插入选择性记成自由基选择性** → 虽然 3°>2°>1° 结果类似，但机理不同（协同 vs 分步）

### 12.3 入门级例题

**题目**：CHCl₃ 和 t-BuOK 反应生成什么卡宾？是单线态还是三线态？与顺-2-丁烯反应产物构型如何？

**预期解答路径**：
1. t-BuO⁻ 拔掉 CHCl₃ 的质子，Cl⁻ 离去 → :CCl₂（二氯卡宾）
2. 两个 Cl 的吸电子效应 → 孤对电子配对 → **单线态**
3. 单线态 + 顺式烯烃 → 协同环加成 → **构型保持**（顺式环丙烷）

**老师可引导的提问**：
- 卡在第 1 步：问 "这是 α-消除还是 β-消除？"
- 卡在第 2 步：问 "Cl 是给电子还是吸电子？对自旋态有何影响？"
- 卡在第 3 步：问 "单线态卡宾的环丙烷化是协同还是分步？"

### 12.4 与现实/直觉的连接

- **类比**：卡宾就像"社交能力极强但极其冲动的人"——空 p 轨道（想要电子）和孤对电子（可以给出）同时具备，所以见谁反应谁。单线态是"冲动但有条理"（协同），三线态是"冲动且混乱"（分步）。
- **反例**：看起来 CO 是"稳定分子"，但它的 Lewis 式中 C 有孤对 + 空 p —— 本质上就是"被 O 稳定了的卡宾"。一氧化碳的毒性恰恰来自 C 端孤对的强配位能力（与血红蛋白 Fe 配位）——这是"稳定卡宾"的配位化学本质。

## 十三、竞赛拓展

1. **金属卡宾的 Fischer vs Schrock 分类**：
   - **Fischer 卡宾**：金属低氧化态 + 卡宾碳连吸电子基（如 CO₂Et）→ 卡宾碳亲电
   - **Schrock 卡宾**：金属高氧化态 + 卡宾碳连给电子基（如烷基）→ 卡宾碳亲核
   - 这是金属有机化学的核心概念，与金属氢化物的"酸性/碱性"分类类似

2. **卡宾在 C-H 活化中的新兴应用**：
   - 金属卡宾对惰性 C-H 键的插入 → 直接官能团化
   - Rh(II) 催化的分子内 C-H 插入构建中环

3. **乃春在合成中的"隐形"角色**：
   - Curtius 重排：RCON₃ → RNCO（异氰酸酯），中间体 RN: 乃春
   - Hofmann 重排：RCONH₂ → RNH₂，中间体也是乃春
   - 这些重排的机理题常考乃春的形成与捕获

4. **:CF₂ 的特殊性**：
   - 二氟卡宾极其活泼，但可与烯烃选择性环丙烷化
   - :CF₂ 插入 C-F 键是制备含氟环丙烷的重要方法
