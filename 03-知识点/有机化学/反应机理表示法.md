# 反应机理表示法
- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：[[32-有机反应机理]]、[[29-共振论]]、[[33-有机反应的分类]]

## 一、定义

**反应机理表示法（Reaction Mechanism Notation, Arrow-Pushing）** 是用**电子推动箭头**（curly arrow / curved arrow）系统地表示反应过程中**键的断裂与形成**的方法。它是有机/无机反应机理的"语言"——和 [[Grossman规则]]（解题策略）、[[共振论]]（电子分布）共同构成机理推断的三大基石。

> 一句话：**箭头从电子源（高能填充轨道）出发，指向电子汇（低能空轨道）**——这是所有反应机理表示的元规则。

### 1.1 为什么需要这种表示法？

- ✅ **可视化电子流向**：让"看不见的电子"成为可推导的图形语言
- ✅ **强制清晰思考**：每一步基元反应必须用箭头明确表示
- ✅ **检查机理合理性**：箭头守恒（电子数、电荷、原子）保证机理无错
- ✅ **跨学科通用**：有机/无机/生化机理都用同一套语言
- ✅ **教学可推导**：从基本规则可以推出复杂多步机理

### 1.2 历史脉络

- **Robert Robinson**（1920s）：提出弯箭头表示电子流动
- **Christopher Ingold**（1930s）：系统化"亲核试剂/亲电试剂"语言
- **Robert B. Grossman**（1990s 后）：在《The Art of Writing Reasonable Organic Reaction Mechanisms》（1999 第 1 版）中提出"7 commandments"系统化书写规则
- **Stuart Warren**（1980s 起）：在《Designing Organic Syntheses》（1978）中将逆合成与机理书写结合

## 三、核心原理

### 3.1 四种箭头类型（按反应类型）

| 箭头形式 | 名称 | 表示电子数 | 适用反应 |
|:---:|:---|:---:|:---|
| ↷（双钩弯箭头）| **极性反应箭头** | 2 e⁻（一对）| SN1/SN2/E1/E2/亲核加成/亲电加成 |
| ⤳（单钩鱼钩箭头）| **自由基箭头** | 1 e⁻ | 自由基取代/加成/聚合 |
| 弧形圆环（多个箭头围成圈）| **协同周环箭头** | 多对 | Diels-Alder/电环化/σ-迁移 |
| → / ⇌（直箭头）| **平衡 / 转化箭头** | — | 表示反应进展（不是电子流）|

**关键区别**：
- **极性反应**用 ↷（**整对**电子移动）
- **自由基**用 ⤳（**单个**电子移动）
- **绝不混用**！

### 3.2 箭头的元规则（5 条）

1. **箭头必须从电子源出发**（占据轨道：σ/π/n）
2. **箭头必须指向电子汇**（空或低能轨道：σ\*/π\*/p）
3. **每条箭头表示一对电子（极性）或一个电子（自由基）**
4. **电子守恒**：箭头总数等于反应中转移的电子对数
5. **箭头方向永远从电子源流向电子汇**——绝不反向

### 3.3 电子源 vs 电子汇

| 类型 | 电子源（箭头起点）| 电子汇（箭头终点）|
|:---|:---|:---|
| **σ 键** | 占据 σ 键 | σ\* 反键 |
| **π 键** | 占据 π 键 | π\* 反键 |
| **孤对电子** | 杂原子 n（标"⋮"或"··"）| 空轨道 / σ\* / π\* |
| **空轨道** | — | 空 p（C+）、空 d、空价层 |
| **形式电荷** | 负电荷 | 正电荷 |

> 决定哪个是"源"哪个是"汇"的核心：**前线轨道能量**。HOMO 是源，LUMO 是汇。详见 [[给体-受体相互作用]]。

### 3.4 Grossman 7 Commandments（机理书写"七诫"）

Grossman 在其专著中列出了机理书写的 7 条核心规则：

1. **画出所有键、所有形式电荷、所有相关孤对电子**——不要"图省事"省略
2. **箭头从电子源出发，向电子汇方向画**——绝不反向
3. **每个箭头表示一对电子（极性）或一个电子（自由基）**——不混用
4. **每个原子的电子数和形式电荷在每一步都明确**——逐步检查
5. **过渡态/中间体的电子结构必须自洽**——不能"凭空产生"或"消失"电子
6. **保持原子骨架编号一致**（[[Grossman规则]]）——便于追踪迁移
7. **每一步基元反应都画出来**——不要"一步到位"跨过中间体

### 3.5 平衡箭头与基元步骤

```
   反应物 ──┐
            ├─→ 中间体 1 ──→ 中间体 2 ──→ 产物
   试剂  ──┘

   ↑          ↑           ↑           ↑
  画箭头    画箭头      画箭头      画箭头
   ↓          ↓           ↓           ↓
"基元反应1" "基元反应2"  "基元反应3"  "基元反应4"
```

每一步基元反应（elementary step）都需要独立画出电子流向，箭头指向**该步骤新生成的中间体**而非最终产物。

## 四、关键结论

### 4.1 四类典型机理及其箭头表示

#### 4.1.1 极性反应（双钩弯箭头）

```
亲核取代（SN2）：

   Nu⁻  ↷  C—LG  →  Nu—C + LG⁻
              ↺
   箭头 1：Nu⁻ 孤对电子 → C（背面进攻）
   箭头 2：C—LG σ 键 → LG（离去基）
```

#### 4.1.2 自由基反应（鱼钩箭头）

```
自由基取代：

   X• + H—C—R  →  X—H + •C—R
        ⤳  ⤳
   两个鱼钩：① X• 单电子 + ② H—C 中的一个电子 → 形成 X—H σ 键
   留下：C 上的另一个电子（形成自由基中间体）
```

#### 4.1.3 协同周环反应（环箭头）

```
Diels-Alder：

   ⌒              
  / \\         环箭头闭合
 ⌒    ↺       3 对箭头围成 6e⁻ 周环 
  \\ /         
   ⌒              
                
   全部箭头同步画，表示协同发生
```

#### 4.1.4 配位反应（实箭头 / 双向箭头）

```
配位形成：

   L: → M    （L 给出孤对电子，M 接受）
   实心箭头表示"形成"配位键

   L ↹ M    （L 与 M 反馈键，π-receiver 反向给电子）
   双向箭头表示"反馈"
```

### 4.2 形式电荷与孤对电子的标记规则

| 原子 | 中性时孤对电子 | 形式电荷 +1 时孤对 | 形式电荷 -1 时孤对 |
|:---|:---:|:---:|:---:|
| C | 0 | 0 | 1 对（碳负离子）|
| N | 1 对 | 0（如 NH₄⁺）| 2 对（如 R₂N⁻）|
| O | 2 对 | 1 对（如 H₃O⁺）| 3 对（如 RO⁻）|
| F | 3 对 | 2 对 | 4 对（罕见）|

> **机理书写中孤对电子是否画出**？Grossman 规则：**至少画出参与反应的孤对电子**——尤其是箭头起点处的孤对要画清楚。

### 4.3 箭头守恒法则

| 守恒量 | 检查方法 |
|:---|:---|
| **电子数** | 反应物总电子 = 产物总电子（包含 e⁻ / e⁻⁻ 计数）|
| **形式电荷总和** | 反应物总电荷 = 产物总电荷 |
| **原子核数** | 反应物原子数 = 产物原子数（原子守恒）|
| **箭头数** | 每个箭头 = 一对/单个电子的转移 |

### 4.4 中间体与过渡态的图示

| 物种 | 寿命 | 表示方法 |
|:---|:---|:---|
| **中间体（intermediate）** | 有限寿命，能量极小值 | 用方括号 `[ ]` 或单独画 |
| **过渡态（transition state）** | 瞬时（10⁻¹³ s），能量极大值 | 用 `‡` 上标，部分键用虚线 |

```
反应坐标图：

     E
     │       TS‡
     │      /\\
     │     /  \\
     │    /    \\___ INT
     │   /          \\___ Product
     │R/                 
     └─────────────────→ 反应进度
```

## 五、常见分类或情形

### 5.1 按反应物的电子需求分类

| 反应类型 | 电子源（亲核试剂）| 电子汇（亲电试剂）| 箭头方向 |
|:---|:---|:---|:---|
| **亲核进攻** | Nu⁻ 或 Nu: | C+ / C=X / C-LG | Nu → C |
| **质子转移** | 碱（B:）| H-A | B → H |
| **离去基离去** | C-LG 中的 σ 键电子 | LG⁻ 形成 | σ → LG |
| **重排迁移** | 迁移键 σ | 缺电子原子 | σ → C+ |

### 5.2 按反应物种数分类

| 类型 | 例子 |
|:---|:---|
| **单分子（unimolecular）** | SN1, E1, 重排 |
| **双分子（bimolecular）** | SN2, E2, 亲核加成 |
| **多分子（多步骤）** | 链式自由基反应、催化循环 |

### 5.3 按机理路径分类

| 路径 | 特征 | 箭头模式 |
|:---|:---|:---|
| **极性（heterolytic）** | 整对电子转移 | 双钩 |
| **自由基（homolytic）** | 单电子转移 | 鱼钩 |
| **协同（concerted）** | 同时发生 | 环箭头 |
| **逐步（stepwise）** | 多个基元步骤 | 一系列单箭头 |

### 5.4 ABOC 框架下的反应分类（简表）

ABOC 将反应分为 4 大类：极性 / 自由基 / 周环 / 金属催化。每一类对应不同的箭头模式：

| ABOC 反应类 | 主要箭头 | 例 |
|:---|:---|:---|
| **极性** | 双钩 ↷ | SN1/SN2/E1/E2/加成 |
| **自由基** | 鱼钩 ⤳ | 自由基聚合、Glauber 反应 |
| **周环** | 环 | Diels-Alder/Cope/电环化 |
| **金属催化** | 双向（配位+反馈）| Pd 催化、Heck/Suzuki |

## 六、适用条件与限制

### 6.1 推荐使用场景
- ✅ 任何键断裂/形成的过程
- ✅ 反应中间体的产生
- ✅ 共振结构间电子的"虚拟流动"（参见 [[共振论]]）
- ✅ 教学说明：从"反应物 → 产物"过渡到"为什么这样反应"

### 6.2 限制
- ❌ **静电相互作用**（无键变化时不画箭头）
- ❌ **构象变化**（只是几何变化，不涉及电子）
- ❌ **跨原子超共轭**（弱作用，机理书写中常忽略）
- ❌ **量子隧穿**（H 原子隧穿在某些反应中重要，但传统箭头表示不出）

### 6.3 与 [[Grossman规则]]、[[共振论]] 的边界

| 工具 | 用途 | 关系 |
|:---|:---|:---|
| **反应机理表示法**（本 KP）| 书写每一步的电子流（箭头）| **语言层** |
| [[Grossman规则]] | 标号反应物/产物原子，确定哪根键断/生 | **策略层**（提供"画箭头时该画哪些键"的依据）|
| [[共振论]] | 描述某中间体的电子分布 | **结构层**（提供"中间体长什么样"）|

**典型工作流程**：
```
1. 用 Grossman 规则标号原子   →  确定哪根键断/生
2. 在中间体处用共振论分析     →  画出关键共振结构
3. 用反应机理表示法           →  画箭头连接每一步基元反应
```

## 七、常见比较与易混点

### 7.1 极性箭头 vs 自由基箭头

| 项 | 极性箭头 ↷ | 自由基箭头 ⤳ |
|:---|:---|:---|
| 表示电子数 | 2（一对）| 1 |
| 形式 | 整钩、闭合 | 半钩、单边 |
| 适用 | 离子反应 | 自由基反应 |
| 电荷传递 | 形式电荷变化 | 自由基产生 / 消失 |

### 7.2 协同 vs 多步

| 协同（concerted）| 多步（stepwise）|
|:---|:---|
| 所有键断/生**同时**发生 | 分步骤发生 |
| 无中间体 | 有中间体 |
| 一组箭头闭合（环箭头）| 多组箭头序列 |
| 例：Diels-Alder、SN2 | 例：SN1、E1 |

### 7.3 共振箭头 vs 机理箭头

| 共振结构间的箭头 | 反应机理的箭头 |
|:---|:---|
| 双向 ↔ | 单向 → |
| 表示**虚拟**电子流（同一分子的不同表示）| 表示**真实**电子流（一个分子变成另一个分子）|
| 不发生原子核移动 | 可能发生原子核移动 |
| 用在同一行 | 用在不同行 |

### 7.4 路径标识符号

| 符号 | 含义 |
|:---|:---|
| **A_N** | 加成-亲核 |
| **D_N + A_N** | 离去 + 亲核（SN1）|
| **A_N D_N** | 协同（SN2）|
| **E_1cb** | 共轭碱负离子消除 |
| **AdN-E** | 加成 - 消除 |

> IUPAC 推荐符号系统能精确描述反应类型，但化学竞赛中使用频率较低。

## 十、例题

### 例题 1（SN2 机理书写）

**题目**：写出 $\mathrm{CH_3CH_2Br + OH^-}$ 的 SN2 反应机理。

**分析**：
- 亲核试剂（电子源）：OH⁻ 的孤对电子
- 离去基（电子汇）：C-Br 的 σ 键 → Br⁻
- 同步发生（协同）

**解答**：

```
                Br
                |
           HO⁻ ↷ C—H    ↺
                |  ↺
                H

   箭头 1：OH⁻ 孤对电子 → C（背面进攻，与 Br 在 180°）
   箭头 2：C—Br σ 键电子 → Br

   产物：HO—CH₂CH₃ + Br⁻
```

**反思**：
1. 一步反应，无中间体
2. 立体专一性：构型反转（Walden 翻转）
3. 反应速率 = $k[\mathrm{RBr}][\mathrm{OH^-}]$（双分子）

---

### 例题 2（SN1 机理书写）

**题目**：写出 $\mathrm{(CH_3)_3CCl}$ 在水中的 SN1 反应机理。

**分析**：
- Step 1（决速步）：自发离去 → 碳正离子中间体
- Step 2：水的孤对电子进攻碳正离子
- Step 3：质子转移到水分子

**解答**：

```
Step 1: 离去
   (CH₃)₃C—Cl  ↷  (CH₃)₃C⁺ + Cl⁻
              ↺
   箭头：C—Cl σ 键 → Cl

Step 2: 亲核加成
   (CH₃)₃C⁺  +  H₂O ↷ → (CH₃)₃C—OH₂⁺
                       ↺
   箭头：H₂O 孤对电子 → C⁺（碳正离子空 p 轨道）

Step 3: 去质子
   (CH₃)₃C—OH₂⁺  +  H₂O ↷ → (CH₃)₃C—OH + H₃O⁺
                              ↺
   箭头 1：O—H σ 键 → H₂O 的 O
   箭头 2：H₂O 的 O 孤对 → H
```

**反思**：
1. 三步基元反应，**每一步独立画箭头**
2. 中间体（C+）用方括号或单独画
3. 速率仅取决于 RCl 浓度（一级反应）

### 例题 3（E1cb 机理 - 共轭碱负离子消除）

**题目**：写出 $\mathrm{HOCH_2CH_2Cl}$ 在强碱（OH⁻）中的 E1cb 机理。

**分析**：E1cb 适用于 α-H 酸性比 β-LG 离去性强的情况（如 OH 是好的羧酸）。

**解答**：

```
Step 1: α-去质子
   HO—CH(α)—CH₂(β)—Cl  +  OH⁻  →  HO—C⁻(α)—CH₂(β)—Cl  +  H₂O
                                  ↺
   箭头：α-H 的 σ 键 → OH⁻ 的 O

Step 2: β-离去
   HO—C⁻(α)—CH₂(β)—Cl  →  HO—C(α)=CH₂(β) + Cl⁻
                          ↺
   箭头：α-C 的孤对电子 → α-β 键（形成 π 键）；β-C 的 C-Cl 键 → Cl 离去

   产物：CH₂=C(OH) → 互变 → CH₃CHO（乙醛）
```

**反思**：
1. 两步机理：先去质子（碳负离子中间体），后离去
2. E1cb 与 E2 区别：E2 是协同的（一步），E1cb 是两步
3. 实验区分：用 D 标记 α-H，看是否在 β 离去前被交换

### 例题 4（共振结构间的箭头）

**题目**：用箭头表示醋酸根 $\mathrm{CH_3COO^-}$ 的两个共振结构间的电子流。

**分析**：注意——这是**共振结构间的箭头**（双向 ↔），不是真实反应的箭头。

**解答**：

```
       O⁻              O                   
       |               ║                   
   CH₃—C   ←→  CH₃—C        
       ║               |                   
       O               O⁻                  
                                              
   箭头：① 上方 O 的孤对电子 → C-O σ 键间 (变成 π 键)
        ② 下方 C=O 的 π 键 → O (变成孤对电子)
```

**反思**：
1. 共振结构间用 **↔（双向短箭头）** 而非 → 或 ⇌
2. 原子核位置不变（只移电子）
3. 电荷在两个 O 上等价分布（每个 O 各 -½ 电荷）

### 例题 5（Curtius 重排）

**题目**：写出 RCON₃（酰基叠氮）在加热下重排为 RNCO（异氰酸酯）的机理。

**分析**：
- 关键步骤：N₂ 离去 + R 基团迁移到 N
- 协同重排（无氮宾中间体）或两步（经氮宾）—— 实验支持协同

**解答**：

```
              O                                  
              ║                                  
          R—C—N=N⁺=N⁻                          
              ↑↺                                
              │ 协同迁移                        
              ↓                                  
          R—N=C=O  +  N₂                       
                                                 
   箭头 1：C-R σ 键电子 → N (R 从 C 迁移到 N)
   箭头 2：N₂⁺ 上的 N≡N → N₂ (释放氮气)
```

**反思**：
1. 这是 [1,2]-烷基迁移的典型机理
2. 协同 vs 两步：实验显示反应中无氮宾被捕捉 → 协同
3. 与 [[Hofmann重排]] 类比：都是 R 从 C 迁移到 N，只是离去基不同

## 十一、易错点

### 11.1 箭头方向反向（最常见错误）

**错误**：箭头从电子汇出发，指向电子源。

**正确**：箭头**永远**从源（HOMO）出发，指向汇（LUMO）。

> 检查方法：箭头起点应是孤对电子、负电荷、π 键 σ 键；终点应是空轨道、正电荷、σ\*/π\*。

### 11.2 把双钩和鱼钩混用

**错误**：在自由基反应中用双钩，或在极性反应中用鱼钩。

**正确**：极性反应必须用 ↷（双钩，2 e⁻）；自由基必须用 ⤳（鱼钩，1 e⁻）。

### 11.3 漏画孤对电子

**错误**：从 H₂O 的 O 出发画箭头但没画 O 上的孤对电子。

**正确**：参与反应的孤对电子**必须画出**（用 `:` 或 `··` 表示）。

### 11.4 箭头跨原子飞跃

**错误**：箭头一次跨 2-3 个原子（如 C 到 5 个键远的 X）。

**正确**：箭头**只在相邻原子间**移动一对电子。多原子迁移要分多步画。

### 11.5 漏画形式电荷

**错误**：碳正离子没标 +，碳负离子没标 -。

**正确**：每一步的形式电荷必须明确标记。

### 11.6 共振箭头用错（→ 替代 ↔）

**错误**：共振结构间用 → 或 ⇌。

**正确**：共振结构是同一分子的不同表示，用 ↔（双向短箭头），表示"等价描述"。

### 11.7 把 5 价 C 当成正常

**错误**：在中间体中画出 C 有 5 根键。

**正确**：C 在过渡态可以"准 5 配位"，但中间体中**只能 4 价**。出现 5 价说明机理错。

### 11.8 协同与多步混淆

**错误**：用一组环箭头表示 SN1（实际两步）。

**正确**：协同反应用环箭头闭合（一步同时）；多步反应每步独立。

### 11.9 平衡箭头与共振箭头混用

**错误**：在表示酮-烯醇互变异构时用 ↔（共振箭头）。

**正确**：互变异构发生原子核位置变化（H 迁移），用 ⇌（平衡箭头）。共振论中 H 不能移动！

### 11.10 跳过基元反应步骤

**错误**：直接从反应物画箭头到最终产物，跳过中间体。

**正确**：每个基元反应都要独立画——跨过中间体是 Grossman 第 7 诫的违反。

  - "[[Grossman规则]]（解题策略）"
  - "[[共振论]]（中间体结构表示）"

### 12.2 学生易踩的认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析口诀/方法 |
|:---|:---|:---|:---|
| "箭头是从原子出发画的"| 直觉来自"原子是中心"| 箭头从**电子源**出发（孤对电子、键、负电荷），原子只是"标识"位置 | 想清楚"哪些电子要移动" |
| "箭头方向无所谓，画得好看就行"| 没意识到方向性 | 方向**严格**：源 → 汇；反向画即错 | "电子总是从富走到贫" |
| "极性反应可以用鱼钩偷个懒"| 没区分 1 vs 2 电子 | **绝对不能**——双钩与鱼钩对应不同反应类型 | "极性双钩、自由鱼钩" |
| "每步画几个箭头随意"| 没意识到守恒 | 必须严格电子守恒 | 数每步进出的电子数 |
| "共振箭头和反应箭头一样"| 形似 | 共振用 ↔（双向短）；反应用 →（单向）| "共振 ↔ 不动核；反应 → 真发生" |
| "孤对电子可以省略"| 偷懒 | 至少**参与反应的孤对**必须画 | "看起来'空白'的杂原子也有孤对" |
| "5 价 C 偶尔可以画"| 觉得过渡态时可以 | 中间体中 C **只能 4 价**；过渡态可"准 5 配位"用虚线 | C 永远 4 价（实键）|
| "迁移基团可以一步飞到底"| 简化思维 | 一根箭头只移动一对电子，多步迁移要分步骤 | "一根箭一对电子" |
| "机理画法是死规则不能变"| 教条思维 | 7 commandments 是基础，但具体反应可有合理变形 | "规则保证正确，灵活展现思想" |
| "看到 H 移动就用 ↔"| 没分清共振与互变 | H 移动 = 互变异构（⇌）; 不动核 = 共振（↔）| "看 H 是否移动" |

### 12.3 入门级例题（不同于第十节）

**题目**：写出乙醇钠 $\mathrm{NaOC_2H_5}$ 与溴乙烷 $\mathrm{C_2H_5Br}$ 的 SN2 反应机理（用箭头表示）。

**预期解答路径**：

1. 第一步看到题目应该想到：**这是 SN2 反应**——亲核试剂 EtO⁻ + 烷基卤 EtBr。
2. 然后联想到本 KP 的：**SN2 是一步协同反应**，箭头同时画。
3. 找电子源与电子汇：
   - 源：EtO⁻ 上的孤对电子（强亲核试剂）
   - 汇：C-Br 的 σ\*（弱键，好离去基）
4. 应用得到：

```
   EtO⁻  ↷  CH₂—Br    ↺
            |
            C₂H₅

   箭头 1：EtO⁻ 孤对 → C（背面进攻，与 Br 反方向 180°）
   箭头 2：C—Br σ 键电子 → Br

   产物：EtO—CH₂CH₃ + Br⁻ + Na⁺
```

5. 检查：电子数守恒（2 对箭头转移）；形式电荷守恒（EtO⁻ → 中性 + Br⁻）。

**老师可在课堂上引导的提问**：
- 学生若卡在第 1 步：问 "什么是亲核试剂？什么是离去基团？"
- 学生若卡在第 2 步：问 "SN2 是几步反应？同时发生什么？"
- 学生若卡在第 3 步：让学生**画出 EtO⁻ 上的孤对电子**（用 :）
- 学生若答错（用一根箭头）：让学生数电子，发现"两根键变化要两根箭头"
- 学生若忘记 EtO⁻ 的负电荷：问 "电荷总和应当多少？"
- 学生若画反向（C → EtO⁻）：问 "电子从哪里来？为什么会主动给电子？"

### 12.4 与现实/直觉的连接

> 用学生熟悉的现象/比喻 anchor 这个抽象概念。

- **类比 1**：电子推动箭头像**水流图**——水从高处流向低处（HOMO 高 → LUMO 低），无须强迫，自然发生。
- **类比 2**：极性 vs 自由基箭头像**对话方式**——
  - 极性：两个人合作完成（一对电子整体转移）= 双钩
  - 自由基：两个人各自单独行动（一个电子独立转移）= 鱼钩
- **类比 3**：协同 vs 多步像**饭店打菜**——
  - 协同（如 Diels-Alder）：一锅烩，所有食材同时下锅
  - 多步（如 SN1）：先煎再炖，每一道工序独立
- **反例/佯谬**：
  - 表面上看"原子是反应的主体"，但实际上**电子才是反应的真正主角**——这是初学者最大的误解。
  - 表面上看"画箭头是表面文章"，但实际上每根箭头都对应一个**真实的物理过程**（轨道电子流向 + 几何变化）。

## 十三、竞赛拓展

### 13.1 IUPAC 反应路径符号系统

IUPAC 提出的标准化机理表示法（详见 *Pure & Appl. Chem.* 1989, 61, 23）：

| 符号 | 含义 |
|:---|:---|
| $A_N$ | Addition + Nucleophilic（亲核加成）|
| $D_N$ | Dissociation + Nucleophilic（解离 + 亲核）|
| $D_E$ | Dissociation + Electrophilic（亲电解离）|
| $\mathrm{D_N + A_N}$ | 两步 SN1 |
| $\mathrm{A_N D_N}$ | 协同 SN2 |
| $E_1$ / $E_2$ / $E_{1cb}$ | 不同消除路径 |

化学竞赛中较少使用，但出题人可能引用。

### 13.2 计算化学验证机理

| 工具 | 用途 |
|:---|:---|
| Gaussian / ORCA | DFT 优化中间体与过渡态 |
| GRRM | 自动反应路径搜索 |
| IRC（内禀反应坐标）| 验证过渡态连接的反应物与产物 |
| QTAIM（量子原子理论）| 给出"键路径"的实空间表示 |

### 13.3 Marcus 理论与电子转移机理

对于**单电子转移（SET, Single Electron Transfer）**反应，使用 Marcus 理论：

$$\Delta G^\ddagger = \frac{(\Delta G + \lambda)^2}{4\lambda}$$

其中 $\lambda$ 是重组能。SET 机理用单根鱼钩 ⤳ 表示。

### 13.4 现代金属催化机理表示

含金属配合物的机理使用扩展箭头系统：

| 步骤 | 箭头 |
|:---|:---|
| **氧化加成** | M + R-X → MR(X)，M 氧化数 +2 |
| **配体交换** | L → M ← L'，双向 |
| **β-H 消除** | β-H 通过反共平面进入 M 的 d 轨道 |
| **还原消除** | 反向氧化加成 |

详见 [[氧化加成与还原消除]] 等 KP。
