# 缔合-离解过程

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[12-配合物]]、[[13-元素化学]]

## 一、定义
**缔合（A, Associative）**与**离解（D, Dissociative）**是两类基元反应，描述配合物或主族中心原子上的配体交换：

- **A 过程**：先形成**键**（配体加成），中间体配位数**升高**
- **D 过程**：先**断**键（配体离开），中间体配位数**降低**
- **I（Interchange，交换）过程**：键的形成和断裂**协同**进行——介于 A 和 D 之间

## 二、考纲对应
- 对应考纲条目：[[12-配合物]]、[[13-元素化学]]、[[36-取代反应]]
- 所属模块：[[基础要求-无机和结构化学]]
- 本知识点在考纲中的作用：决赛级"配合物取代反应机理"的核心分类工具，也用于解释主族元素的反应。

## 三、核心原理

### 3.1 A 过程（缔合，Associative）
$$\ce{[ML_n] + L'} \to \ce{[ML_n L']} \to \ce{[ML_{n-1} L'] + L}$$
- 第一步：L' 加成 → 中间体配位数 +1
- 第二步：L 离去 → 配位数恢复
- **类似有机的 SN2-Si**

### 3.2 D 过程（离解，Dissociative）
$$\ce{[ML_n]} \to \ce{[ML_{n-1}]} + L \to \ce{[ML_{n-1} L'] + L}$$
- 第一步：L 离去 → 中间体配位数 −1
- 第二步：L' 进攻
- **类似有机的 SN1**

### 3.3 I 过程（交换，Interchange）
- 键的形成和断裂**协同**进行
- 没有真正的中间体——只是过渡态
- 进一步细分：
  - **Iₐ（associative-like）**：进入键比离去键强
  - **I_d（dissociative-like）**：离去键比进入键强

### 3.4 速率定律的差异
| 机理 | 速率定律 | 说明 |
|---|---|---|
| A | $v = k[\text{ML}_n][\text{L'}]$ | 二级，依赖 L' 浓度 |
| D | $v = k[\text{ML}_n]$ | 一级，与 L' 浓度无关 |
| Iₐ | $v = k[\text{ML}_n][\text{L'}]$ | 类似 A 但无可分离中间体 |
| I_d | $v = k[\text{ML}_n][\text{L'}]/(1+...)$ | 类似 D 但有 L' 弱依赖 |

## 四、关键结论

### 4.1 主族元素中的对应关系
| 中心原子 | 倾向 | 实例 |
|---|---|---|
| Si、Ge | A（高价中间体稳定） | $\ce{SiF4 + 2F- \to [SiF6]^{2-}}$ |
| P、As、Sb | A | $\ce{PCl5 + Cl- \to [PCl6]-}$ |
| S | A | $\ce{SF4 + F- \to [SF5]-}$ |
| 卤素 | A | $\ce{ICl + Cl- \to [ICl2]-}$ |
| Xe | A | $\ce{XeF2 + F- \to [XeF3]-}$ |

### 4.2 过渡金属配合物中的应用
- d⁶ 强场配合物（如 $\ce{[Co(NH3)6]^{3+}}$）：通常 D 或 I_d
- d⁰、d¹⁰ 配合物：A 或 Iₐ 多见
- 典型例：$\ce{[PtCl4]^{2-}} + \ce{NH3} \to \ce{[Pt(NH3)Cl3]-} + \ce{Cl-}$（Iₐ 机理）

### 4.3 A 与 D 的 Hammond 后置
- A 过程：过渡态结构接近**中间体**（高配位）
- D 过程：过渡态结构接近**离去后的中间体**（低配位）

## 五、常见分类或情形

### 5.1 主族元素的 A 过程优先
- 第 3 周期及以下中心：基本走 A
- 因为高配位中间体可被 [[高价化合物的成键|3c-4e 键]] 稳定

### 5.2 D 过程的常见情形
- 中心原子很拥挤（如 $\ce{[Co(en)3]^{3+}}$）
- 离去基团特别好（如 $\ce{H2O}$）
- 进入基团亲核性弱

### 5.3 I 过程的实验判据
- 反应级数：1 或 2（取决于 Iₐ 或 I_d）
- 活化体积：$\Delta V^\ddagger > 0$ 表示 D 倾向；$\Delta V^\ddagger < 0$ 表示 A 倾向
- 进入基团的影响：A 对进入基团敏感，D 不敏感

## 六、适用条件与限制

### 适用范围
- ✓ 配合物的配体取代
- ✓ 主族元素的亲核取代
- ✓ 金属有机化学的"迁移-插入"分析

### 限制
- 真正的"纯 A"或"纯 D"较少——大多数是 Iₐ 或 I_d
- 速率定律的解读需结合活化参数（$\Delta H^\ddagger$、$\Delta S^\ddagger$、$\Delta V^\ddagger$）

## 七、常见比较与易混点

### 1. A vs SN2、D vs SN1
- A 过程 ≈ SN2-Si（重元素的两步加成-离解）
- D 过程 ≈ SN1（先离解后进攻）
- 但 A 过程中中间体**可分离**，SN2 中只是过渡态

### 2. A vs Iₐ
| | A | Iₐ |
|---|---|---|
| 中间体 | 真实存在 | 仅过渡态 |
| 键形成与断裂 | 分两步 | 协同（但进入键先于离去键） |

### 3. A 过程 vs σ-bond metathesis
- A 过程：中心原子先加成后离解，**中间体存在**
- σ-bond metathesis（[[配体交换反应]]）：四中心协同过渡态，**无中间体**

## 十、例题

### 例题 1：判断 $\ce{[Co(NH3)5(H2O)]^{3+} + Cl-}$ 取代机理
**题目：** 反应 $\ce{[Co(NH3)5(H2O)]^{3+} + Cl- \to [Co(NH3)5Cl]^{2+} + H2O}$ 的速率定律为 $v = k[\text{Co complex}]$（与 Cl⁻ 浓度无关）。判断机理。
**分析：**
- 一级动力学（与 Cl⁻ 无关）→ D 或 I_d
- d⁶ 低自旋 Co(III)，配位饱和 → 倾向 D
**解答：** D 机理：先 $\ce{H2O}$ 离去得 $\ce{[Co(NH3)5]^{3+}}$，然后 $\ce{Cl-}$ 进入。

### 例题 2：$\ce{SiF4 + 2F-}$ 的 A 过程
**题目：** 写出 $\ce{SiF4 + 2F- \to [SiF6]^{2-}}$ 的机理（用 A 过程表示）。
**分析：**
- 第一步：$\ce{F-}$ 加成 → $\ce{[SiF5]-}$（A 中间体，TBP）
- 第二步：第二个 $\ce{F-}$ 加成 → $\ce{[SiF6]^{2-}}$（八面体）
**反思：** 没有"离去步"——产物本身就是稳定高配位物种。这是 A 过程的极端情形。

## 十一、易错点
- 把 A、D 的"中间体"和过渡态混淆
- 用速率方程一步定机理——还需结合活化参数
- 假定所有配合物取代都是 A 过程——忽视 d 电子配置的影响
