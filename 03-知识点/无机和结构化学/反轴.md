# 反轴

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[14-对称性]]

## 一、定义
**反轴（Improper Rotation Axis）** 也称**旋转反演轴(Rotoinversion Axis)**,记作 $I_n$ 或 $\bar{n}$。

操作定义:先绕轴**旋转 $360°/n$**,再相对该轴上一点**作反演(经过中心)**,得到等价位置。

- 国际(Hermann-Mauguin)符号:$\bar{1}, \bar{2}, \bar{3}, \bar{4}, \bar{6}$
- Schoenflies 符号:**$S_n$(改进旋转,Improper Rotation)**——先**旋转**再**反映**(过垂直轴的镜面),与 $I_n$ 不完全等价但相关

## 二、考纲对应

## 三、核心原理

### 反轴 $I_n$ 的几何操作
1. 绕 $n$ 重对称轴旋转 $\frac{360°}{n}$
2. 关于轴上一点(对称中心)反演:$(x,y,z) \to (-x,-y,-z)$
3. 复合操作 = 反轴的元素操作

### 反轴的低阶等价性
| $I_n$ | 等价于 | 说明 |
|:---:|:---:|---|
| $I_1 = \bar{1}$ | $i$ (对称中心) | 转 360° + 反演 = 仅反演 |
| $I_2 = \bar{2}$ | $\sigma_h$ (水平镜面) | 转 180° + 反演 = 镜面 |
| $I_3 = \bar{3}$ | $C_3 + i$ | 不可约,独立元素 |
| $I_4 = \bar{4}$ | 不可约 | 独立,见 CH₄ 中心 $S_4$ |
| $I_6 = \bar{6}$ | $C_3 + \sigma_h$ | 等价于 $C_{3h}$ |

> 因此**反轴的独立元素**只有 $\bar{3}, \bar{4}, \bar{6}$(以及它们的低阶折算 $\bar{1} = i, \bar{2} = \sigma$)

### Schoenflies $S_n$ 与 H-M $I_n$ 对照
| Schoenflies $S_n$ (旋转-反映) | H-M $I_n$ (旋转-反演) |
|:---:|:---:|
| $S_1 = \sigma$ | $I_1 = i$ |
| $S_2 = i$ | $I_2 = \sigma$ |
| $S_3$ | $I_6$ |
| $S_4 = I_4$ | $I_4 = S_4$ |
| $S_6$ | $I_3$ |

> **重要**:$S_4 = I_4 = \bar{4}$,但 $S_3 \neq I_3$;$S_6 \neq I_6$。中国晶体学教材常用 H-M 记法 $\bar{n}$;有机化学/光谱常用 Schoenflies $S_n$。

### 反轴的"晶体学限制"
- 晶体平移对称只允许 $n = 1, 2, 3, 4, 6$
- 五重轴、七重轴等不能与晶格平移相容
- 因此晶体学反轴只有 $\bar{1}, \bar{2}, \bar{3}, \bar{4}, \bar{6}$ 共 5 种(再加恒等元 $E$,7 种对称操作类型)

## 四、关键结论

### 反轴的"五种独立元素"
| 元素 | 等价/不等价 | 实例分子 |
|:---:|:---|:---:|
| $\bar{1} = i$ | 等价于对称中心 | trans-1,2-二氯乙烯, CO₂ |
| $\bar{2} = \sigma$ | 等价于镜面 | H₂O, NH₃(垂直镜面)|
| $\bar{3}$ | 独立(=$C_3 + i$) | 倍半倍-不常见,见某些晶体 |
| $\bar{4} = S_4$ | **独立** | **CH₄(沿 S₄ 轴)**, allene |
| $\bar{6}$ | 独立(=$C_{3h}$) | 见某些晶体 |

### CH₄ 中的 $S_4 / \bar{4}$ 反轴
- CH₄ 是 $T_d$ 点群:含 $4C_3, 3C_2, 6\sigma_d, 3S_4$
- $S_4$ 轴穿过 CH₄ 中心,沿两对边中点连线
- 操作:旋转 90° + 反映 = 4 个 H 互换位置
- 重要意义:含 $S_4$ 但**无 σ 镜面通过 H** → **手性辨析关键**

### 反轴与手性的关系
- **反轴(包括 $S_n, i, \sigma$)的存在 = 该分子有手性消除元素 = 非手性**
- 严格地说,$\bar{n}$ 是含 i 或 σ 的对称操作之一
- 因此:**手性分子不含任何反轴**($i, \sigma, S_n$ 全无)

## 五、常见分类或情形

### 含反轴的常见点群
| 点群 | 含反轴 | 实例 |
|:---:|:---:|:---:|
| $C_i$ | $i = \bar{1}$ | trans-1,2-二氯环戊烷 |
| $C_s$ | $\sigma = \bar{2}$ | 顺式-CHFClBr |
| $S_4$ | $S_4 = \bar{4}$ | allene 衍生物 |
| $D_{3h}$ | $\sigma_h, S_3$ | BF₃, PCl₅ |
| $T_d$ | $S_4$ | CH₄, SiCl₄ |
| $O_h$ | $S_4, S_6, i$ | SF₆, PF₆⁻ |

### "纯旋转"点群(不含反轴 → 手性)
- $C_n$:仅含 n 重旋转轴 → 手性
  - $C_1$:无对称元素(典型手性分子)
  - $C_2$:仅 $C_2$ 轴(BINAP)
- $D_n$:n 重轴 + n 个垂直 $C_2$ → 手性
  - $D_2$:三个 $C_2$ 互相垂直(twistane)

### 反轴在 32 个晶体学点群中的应用
- 5 个反轴元素分布在 32 个晶体学点群中
- $\bar{1}$ 出现在 11 个中心对称点群
- $\bar{4}$ 关键于四方晶系
- $\bar{6}$ 关键于六方晶系

## 六、适用条件与限制
1. **仅限离散对称**:反轴是点群对称元素,晶体中需与平移群结合(空间群)
2. **晶体学限制**:晶体反轴只能是 $\bar{1}, \bar{2}, \bar{3}, \bar{4}, \bar{6}$
3. **准晶可例外**:准晶可有 5 重、10 重等反轴
4. **H-M vs Schoenflies**:不同体系定义略异($I_n$ vs $S_n$),需要注意对照
5. **判断顺序**:先找最高 $C_n$,再找有无 $i, \sigma, S_n$

## 七、常见比较与易混点

### $I_n$(H-M)vs $S_n$(Schoenflies)
- **$I_n = $ 旋转 + 反演**
- **$S_n = $ 旋转 + 反映**
- $S_4 = I_4$ 是巧合;其他 $n$ 不等价
- 注:$S_{2n} = I_{2n}$ 当 $n$ 奇;$S_n \neq I_n$ 其他情况

### 反轴 vs 旋转轴 $C_n$
- $C_n$:**只**旋转,**不**反演/反映
- $I_n, S_n$:旋转**+**第二操作(反演或反映)
- 重要:$C_n$ 单独存在 → 分子可有手性;$I_n, S_n$ 存在 → 分子非手性

### $\bar{4}$ vs $S_4$
- 在 Schoenflies 体系中,$S_4$ 是"旋转-反映"
- 在 H-M 体系中,$\bar{4}$ 是"旋转-反演"
- **$\bar{4} = S_4$ 等价**(数学上证明)
- 中国教材两种符号都见,需根据上下文识别

### 反轴 vs 对称中心
- 对称中心 $i = \bar{1}$(一阶反轴)
- 不是所有反轴都是对称中心($\bar{4}, \bar{3}$ 都不是)
- 含 $\bar{4}$ 不一定含 $i$:CH₄ 含 $S_4$ 但不含 $i$

## 十、例题

### 例题 1:辨识 CH₄ 中的反轴
**题目:** CH₄ 中含有哪些反轴?写出相应 H-M 和 Schoenflies 符号。

**解答:**
- $T_d$ 点群对称元素:$E, 8C_3, 3C_2, 6S_4, 6\sigma_d$
- CH₄ 含 **3 个 $S_4$ 轴**(等价于 3 个 $\bar{4}$ 轴)
- $S_4$ 轴穿过相对的两条 H-C-H 的中点连线
- H-M 符号:$\bar{4}$
- Schoenflies 符号:$S_4$
- 一致性:$S_4 = \bar{4}$ ✓

**反思:** $T_d$ 点群中常见 $S_4$ 反轴,但**无对称中心 $i$**——这是 $T_d$ 与 $O_h$ 的关键区别。

### 例题 2:用反轴判断手性
**题目:** 判断下列分子是否手性:① trans-1,2-二氯环戊烷;② allene 衍生物 C(=CHCl)₂(=CHCH₃);③ CHFClBr。

**解答:**
- ① trans-1,2-二氯环戊烷:含**对称中心 $i$**(即 $\bar{1}$)→ **非手性**
- ② allene 衍生物:正交两端 $\sigma$ 不存在,但含 $C_2$;**无任何反轴**(即无 i, σ, S₄)→ **手性**(allene 衍生物的经典手性例子)
- ③ CHFClBr:无任何对称元素 → **手性**

**反思:** 判断手性的快速法则:**无反轴(无 i, σ, S_n) ⇔ 手性**。

## 十一、易错点
- ❌ 错:$S_n$ = 旋转 + 反演——是 **旋转 + 反映(镜面)**,反演是 $I_n$
- ❌ 错:$\bar{4} \neq S_4$——它们 **等价**(数学证明)
- ❌ 错:有 $C_2$ 就有对称中心——$C_2$ 是旋转,不含反演
- ❌ 错:CH₄ 有对称中心 $i$——CH₄ **无对称中心**($T_d$),但有 $S_4$
- ❌ 错:晶体中可有 5 重对称轴——准晶可,但常规晶体不可

|---|---|---|---|

### 12.3 入门级例题
**题目**:写出含反轴 $\bar{4}$ 但不含对称中心 $i$ 的最简单分子。

**预期解答路径**:
1. 候选:CH₄($T_d$)
2. CH₄ 含 $S_4 = \bar{4}$:轴穿过两对 H 中点
3. CH₄ 不含 $i$:任意 H 反演不到等价 H
4. 答:$CH_4$ (或 SiCl₄, GeF₄)

**老师可在课堂上引导的提问**:
- 学生若卡住:画 CH₄ 立体图,标 $S_4$ 轴方向
- 学生若答错:对照 $T_d$ 点群对称元素表

### 12.4 与现实/直觉的连接
- 类比:旋转一下再翻面 = 反轴
- 反例:旋转一下不翻面 = 只是旋转轴
