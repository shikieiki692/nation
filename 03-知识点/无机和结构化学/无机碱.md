# 无机碱

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]

## 一、常见强碱
| 碱 | 说明 |
|------|------|
| NaOH | 最重要工业碱，易潮解，吸收 CO₂ |
| KOH | 比 NaOH 更强的碱，更易潮解 |
| Ca(OH)₂ | 微溶（石灰水），廉价碱 |
| Ba(OH)₂ | 可溶性强碱（碱土金属中唯一可溶的 M(OH)₂） |

## 二、重要反应
- NaOH + CO₂ → Na₂CO₃（碱石灰中 NaOH 吸收 CO₂）
- NaOH 与两性金属/氧化物/氢氧化物反应
- 熔融 NaOH 腐蚀玻璃（SiO₂ + 2NaOH → Na₂SiO₃ + H₂O）

## 三、核心原理

### 3.1 碱金属氢氧化物的碱性
碱金属氢氧化物都是强碱，同族从上到下碱性依次增强：$\mathrm{LiOH} < \mathrm{NaOH} < \mathrm{KOH} < \mathrm{RbOH} < \mathrm{CsOH}$。原因：离子半径增大，离子势 $\varphi = Z/r$ 减小，$\mathrm{M}-\mathrm{OH}$ 键更容易断裂释放 $\mathrm{OH}^-$。其中 NaOH 是最重要的工业碱，俗称烧碱；KOH 比 NaOH 更强且更易潮解。

### 3.2 碱土金属氢氧化物的碱性
碱土金属氢氧化物碱性从上到下增强：$\mathrm{Be(OH)_2}$（两性）$< \mathrm{Mg(OH)_2}$（中强碱，不溶）$< \mathrm{Ca(OH)_2}$（强碱，微溶）$< \mathrm{Sr(OH)_2}$（强碱，微溶）$< \mathrm{Ba(OH)_2}$（强碱，可溶）。溶解性也依次增大，$\mathrm{Ba(OH)_2}$ 是碱土金属中唯一可溶的 $\mathrm{M(OH)_2}$。

### 3.3 NaOH 的化学性质
- NaOH 易潮解，易吸收空气中 $\mathrm{CO_2}$：$\mathrm{2NaOH} + \mathrm{CO_2} \to \mathrm{Na_2CO_3} + \mathrm{H_2O}$
- NaOH 与两性金属/氧化物/氢氧化物反应生成含氧酸盐
- 熔融 NaOH 会腐蚀玻璃：$\mathrm{SiO_2} + 2\mathrm{NaOH} \to \mathrm{Na_2SiO_3} + \mathrm{H_2O}$（因此 NaOH 不能用玻璃塞的试剂瓶长期储存）
- NaOH 与卤素发生歧化反应（如 $\mathrm{Cl_2} + 2\mathrm{NaOH} \to \mathrm{NaCl} + \mathrm{NaClO} + \mathrm{H_2O}$）

### 3.4 “三酸三碱”
硫酸、盐酸、硝酸、烧碱（NaOH）和纯碱（$\mathrm{Na_2CO_3}$）被称为"三酸二碱"，实际上氨水 $\mathrm{NH_3}\cdot \mathrm{H_2O}$ 也是重要工业碱，合称"三酸三碱"，是无机化学工业的最基础原料。

## 四、关键结论
- 碱金属氢氧化物均为强碱，从上到下碱性增强
- 碱土金属氢氧化物：$\mathrm{Be(OH)_2}$ 两性，$\mathrm{Mg(OH)_2}$ 中强碱（不溶），其余为强碱
- NaOH 是最重要的工业碱，易潮解，吸收 $\mathrm{CO_2}$，熔融时腐蚀玻璃
- KOH 比 NaOH 更强，更易潮解，常用于需要强碱性或易溶性的场合
- $\mathrm{NH_3}\cdot \mathrm{H_2O}$ 是典型的弱碱，但工业用量大（制化肥、硝酸等）

## 五、常见分类或情形
| 碱 | 强度 | 溶解性 | 说明 |
|------|:---:|:---:|------|
| NaOH | 强碱 | 极易溶 | 最重要工业碱，易潮解，吸收 $\mathrm{CO_2}$ |
| KOH | 强碱 | 极易溶 | 比 NaOH 更强，更易潮解 |
| $\mathrm{Ca(OH)_2}$ | 强碱 | 微溶 | 石灰水，廉价碱，石灰乳 |
| $\mathrm{Ba(OH)_2}$ | 强碱 | 可溶 | 碱土金属中唯一可溶的 $\mathrm{M(OH)_2}$ |
| $\mathrm{Mg(OH)_2}$ | 中强碱 | 几乎不溶 | 悬浊液称镁乳，可作抗酸药 |
| $\mathrm{NH_3}\cdot \mathrm{H_2O}$ | 弱碱 | 易溶 | 氨水，最重要的弱碱 |

## 六、适用条件与限制
- 固体 NaOH 和 KOH 需密封保存，防止潮解和吸收 $\mathrm{CO_2}$
- NaOH 溶液不能用玻璃塞（磨口玻璃瓶），因 NaOH 与玻璃中 $\mathrm{SiO_2}$ 缓慢反应使瓶塞粘住
- $\mathrm{Ca(OH)_2}$ 微溶，饱和溶液（石灰水）浓度仅约 $0.02\mathrm{mol}\cdot\mathrm{L}^{-1}$
- 熔融 NaOH/KOH 极具腐蚀性，需要镍坩埚或银坩埚

## 七、常见比较与易混点
| 对比项 | NaOH | KOH | $\mathrm{Ca(OH)_2}$ |
|------|------|------|------|
| 碱性 | 强 | 更强 | 强 |
| 溶解性 | 极大 | 极大 | 微溶 |
| 价格 | 便宜 | 较贵 | 最便宜 |
| 潮解性 | 强 | 更强 | 无 |

| 对比项 | $\mathrm{Be(OH)_2}$ | $\mathrm{Mg(OH)_2}$ | $\mathrm{Ca(OH)_2}$ |
|------|------|------|------|
| 酸碱性 | 两性 | 中强碱 | 强碱 |
| 溶解性 | 几乎不溶 | 几乎不溶 | 微溶 |

- $\mathrm{Mg(OH)_2}$ 悬浊液可溶性大于 $\mathrm{Ca(OH)_2}$？不！$\mathrm{Ca(OH)_2}$ 微溶（约 $0.02\mathrm{mol/L}$），$\mathrm{Mg(OH)_2}$ 几乎不溶（$K_{sp}\approx 1.8\times10^{-11}$）
- 碱石灰（NaOH + CaO 混合物）：NaOH 吸收 $\mathrm{CO_2}$，CaO 吸水并防止 NaOH 潮解
