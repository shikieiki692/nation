# 方酸

> **方酸（Squaric acid）**是一类具有独特电子结构的碳氧环状化合物，其核心特征为平面四元碳环与强离域 $\pi$ 体系，在合成化学与材料科学中均有重要应用。本知识点覆盖方酸及其衍生物的结构、芳香性，以及其在全碳环（cycl[*n*]carbon）合成中的关键作用。

## 一、定义

### 1.1 方酸分子

方酸的分子式为 $H_2C_4O_4$，不饱和度为 4。其稳定结构以**烯醇-酮互变异构**形式存在：

- 四元碳环（环丁烯骨架）
- 两个对位 C 原子各连一个羟基 $-OH$
- 另外两个对位 C 原子各连一个羰基 $C=O$

方酸是一种**二元强酸**（p$K_{a1}$ ≈ 1.5，p$K_{a2}$ ≈ 3.5），其酸性远强于普通醇和羧酸，这源于其共轭碱——方酸根——具有极高的共振稳定化能。

### 1.2 方酸根 $C_4O_4^{2-}$

方酸根是一个**平面闭合环状体系**，具有以下关键结构特征：

| 特征 | 描述 |
|:---|:---|
| 几何 | 平面正方形（近似），四元碳环 |
| $\pi$ 电子数 | 6 个 $\pi$ 电子（每个 C 贡献 1 个，两个负电荷贡献 2 个） |
| 离域体系 | $\pi_4^6$，满足 Hückel 芳香性规则 $4n+2$（$n=1$） |
| 键长特征 | 所有 C–C 键等长（键级 1.5），所有 C–O 键等长（键级 1.5） |
| 电荷分布 | 负电荷离域分布在 4 个 O 原子上 |

由于 $\pi_4^6$ 芳香体系的存在，方酸根表现出**反磁性**和**化学稳定性**，其共振稳定化能极高。

## 三、核心原理

### 3.1 共振结构与芳香性

#### 主要共振式

方酸根的共振式需满足以下要求：
- 所有 C–C 键等长
- 所有 C–O 键等长
- 负电荷离域在 O 原子上

至少存在 **3 种等价共振式**，它们通过环上 $\pi$ 电子的旋转相互转换。

#### 芳香性判据

方酸根的芳香性可系统验证：

1. **环状**：四元碳环闭合
2. **平面**：所有原子共平面，确保 p 轨道有效重叠
3. **连续共轭**：每个 C 和 O 均贡献 p 轨道形成离域体系
4. **Hückel 电子数**：6 个 $\pi$ 电子，满足 $4n+2$（$n=1$）

> **注意**：方酸根本身是一个 4n 元环，但因其 $\pi$ 电子数为 $4n+2$，故仍具芳香性。这与环丁二烯（$\pi_4^4$，反芳香）形成鲜明对比。

### 3.2 方酸衍生物与合成应用

#### 典型衍生物

| 衍生物 | 结构特点 | 用途 |
|:---|:---|:---|
| 方酸二甲酯 | 两个 $-OH$ 被 $-OCH_3$ 取代 | 合成中间体 |
| 方酸二酰胺 | 两个 $-OH$ 被 $-NR_2$ 取代 | 染料、生物探针 |
| 半方酸 | 一个 $-OH$ 保留，一个被取代 | 不对称合成 |

#### 全碳环 cycl[*n*]carbon 的合成

方酸衍生物在全碳环合成中扮演关键角色：

**历史背景**：
- 1966 年，Hoffmann 首次预言 $C_{18}$ 的存在
- 2019 年，IBM-牛津大学联合团队利用原子力显微镜（AFM）首次捕获 $C_{18}$ 分子形貌

**合成策略（以 $C_{18}$ 为例）**：
1. 以方酸衍生物 $(CH_3)_2C_4O_4$ 和 $Li–C ≡ C–SiR_3$ 为试剂
2. 构建含方酸单元的环状前体 $P$（$C_{24}O_6$）
3. 通过原子尺度操作，逐步脱去 CO 分子
4. 最终得到全碳环 [18]（单叁键交替的 18 元碳环）

**关键前体 P 的结构特征**：
- 分子式：$C_{24}O_6$
- 含 3 个方酸单元，通过 6 个 $-C ≡ C–$ 链连接成环
- 具有三重旋转轴（$C_3$ 对称性）

### 3.3 全碳环的键型争论

全碳环上所有 C 原子均存在 2 条正交 p 轨道，形成：
- 垂直于环平面的离域 $\pi$ 体系
- 朝向环平面内的离域 $\pi$ 体系

这种**双芳香性**导致键型长期存在分歧：

| 模型 | 特征 | 实验证据 |
|:---|:---|:---|
| **键长平均化模型** | 累积双键（=C=C=） | 理论计算支持 |
| **键长交替模型** | 单叁键交替（–C≡C–） | AFM 显示 $C_{18}$ 有约 15 pm 键长差异 |

实验表明 $C_{18}$ 实际处于两种极限结构的中间态，体现了电子离域与环张力的动态平衡。

| 概念 | 方酸 | 方酸根 | 全碳环 [18] |
|:---|:---|:---|:---|
| 分子式 | $H_2C_4O_4$ | $C_4O_4^{2-}$ | $C_{18}$ |

| C–O 键长 | 不等长（C=O vs C–O） | 等长（键级 1.5） | 无 C–O 键 |
