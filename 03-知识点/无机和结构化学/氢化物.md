# 氢化物

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]

## 一、分类
| 类型 | 特征 | 实例 |
|------|------|------|
| 离子型（盐型） | 碱金属/碱土金属 + H⁻ | NaH, CaH₂ |
| 共价型（分子型） | 非金属氢化物 | CH₄, NH₃, H₂O, HCl |
| 金属型（间充型） | H 填入金属晶格间隙 | PdHₓ, LaNi₅H₆ |

## 二、共价氢化物趋势
- 热稳定性：同族↓（H₂O > H₂S > H₂Se > H₂Te）
- 还原性：同族↑（H₂S < H₂Se < H₂Te）
- 酸性（水溶液）：同族↑（H₂O < H₂S < H₂Se）

## 三、核心原理

### 3.1 氢化物的三种类型
氢化物按化学键类型可分为三大类：
- **离子型（盐型）氢化物**：碱金属和碱土金属（Be、Mg 除外）与 H 形成含 $\mathrm{H}^-$ 的化合物，如 NaH、$\mathrm{CaH_2}$。$\mathrm{H}^-$ 是强还原剂，遇水剧烈反应放出 $\mathrm{H_2}$：$\mathrm{NaH} + \mathrm{H_2O} \to \mathrm{NaOH} + \mathrm{H_2}\uparrow$
- **共价型（分子型）氢化物**：非金属元素与 H 形成的化合物，如 $\mathrm{CH_4}$、$\mathrm{NH_3}$、$\mathrm{H_2O}$、HCl。以分子晶体形式存在，熔沸点较低
- **金属型（间充型）氢化物**：$\mathrm{H_2}$ 体积小，可被 d 区金属和 f 区金属吸附在晶体孔隙之间而不改变晶体结构，如 $\mathrm{PdH_x}$、$\mathrm{LaNi_5H_6}$。Ni 及其合金吸氢能力很强，是优质的储氢材料

### 3.2 共价型氢化物的周期性规律
- **热稳定性**：同族从上到下递减（$\mathrm{H_2O} > \mathrm{H_2S} > \mathrm{H_2Se} > \mathrm{H_2Te}$；$\mathrm{HF} > \mathrm{HCl} > \mathrm{HBr} > \mathrm{HI}$）。原因：中心原子半径增大，与 H 的键能减小
- **还原性**：同族从上到下递增（$\mathrm{H_2S} < \mathrm{H_2Se} < \mathrm{H_2Te}$；$\mathrm{HF} < \mathrm{HCl} < \mathrm{HBr} < \mathrm{HI}$）。HI 是强还原剂
- **酸性（水溶液）**：同族从上到下递增（$\mathrm{H_2O} < \mathrm{H_2S} < \mathrm{H_2Se}$；$\mathrm{HF} \ll \mathrm{HCl} < \mathrm{HBr} < \mathrm{HI}$）

### 3.3 HF 的特殊性
HF 在氢卤酸中酸性最弱（$K_a = 6.6\times 10^{-4}$），与同族 HCl（强酸）、HBr（强酸）、HI（强酸）截然不同。原因是：F 的电负性大、半径小，HF 分子间存在氢键作用使解离困难；同时 HF(g) 解离为 H(g) 和 F(g) 的键能（$\Delta H$）特别大，$\mathrm{F}^-$ 水合能虽大但仍不足以补偿。$\mathrm{H_2O}$ 和 $\mathrm{NH_3}$ 与同族其他氢化物相比也有类似情况——第二周期元素的特殊性。

### 3.4 NH₃ 的化学性质
$\mathrm{NH_3}$ 接受质子的能力比 $\mathrm{H_2O}$ 强，在水溶液中部分电离成 $\mathrm{NH_4^+}$ 和 $\mathrm{OH}^-$ 而显碱性，是典型的弱碱。N 原子上有孤对电子，容易形成配位化合物（氨配合物）。$\mathrm{NH_3}$ 分子中的 H 可以被取代生成多种衍生物：$\mathrm{NH_2OH}$（羟氨）、$\mathrm{NH_2-NH_2}$（联氨/肼）、$\mathrm{NaNH_2}$（氨基化钠）等，这些化合物具有很强的还原性。

## 四、关键结论
- 氢化物分为离子型（盐型）、共价型（分子型）和金属型（间充型）三大类
- 共价氢化物热稳定性同族从上到下递减，还原性递增，酸性递增
- 第二周期氢化物（$\mathrm{H_2O}$、$\mathrm{NH_3}$、HF）因氢键和特殊电子结构，性质与同族其余元素差异显著
- HF 为弱酸而 HCl、HBr、HI 为强酸——第二周期特殊性
- 离子型氢化物中 $\mathrm{H}^-$ 是强还原剂，遇水放出 $\mathrm{H_2}$
- 过渡金属可以形成间隙氢化物，Ni、$\mathrm{LaNi_5}$ 等是储氢材料

## 五、常见分类或情形
- **离子型氢化物**：NaH、$\mathrm{CaH_2}$、LiH（$\mathrm{H}^-$ 具有强还原性）
- **共价型氢化物**：$\mathrm{CH_4}$、$\mathrm{NH_3}$、$\mathrm{H_2O}$、$\mathrm{H_2S}$、HF、HCl
- **金属型氢化物**：$\mathrm{PdH_x}$、$\mathrm{LaNi_5H_6}$（储氢材料）
- **含氮氢化物衍生物**：$\mathrm{NH_2OH}$（羟氨）、$\mathrm{N_2H_4}$（肼）、$\mathrm{NH_2N(CH_3)_2}$（偏二甲基肼，火箭燃料）

## 六、适用条件与限制
- $\mathrm{H}^-$ 遇水即反应，离子型氢化物必须在无水条件下保存和使用
- 金属型氢化物中 H 不改变金属晶体结构，吸放氢可逆，是储氢材料的基础
- $\mathrm{Li_3N}$ 是离子型氮化物（而非氢化物），但也被归入 H 被取代的 $\mathrm{NH_3}$ 衍生物
- 共价氢化物水溶液酸性取决于中心原子电负性和半径的平衡

## 七、常见比较与易混点
| 对比项 | $\mathrm{H_2O}$ | $\mathrm{H_2S}$ | $\mathrm{H_2Se}$ | $\mathrm{H_2Te}$ |
|------|:---:|:---:|:---:|:---:|
| 热稳定性 | 最高 | 高 | 中 | 低 |
| 还原性 | 很弱 | 弱 | 中等 | 强 |
| 酸性 | 中性（$K_w=10^{-14}$） | 弱酸（$K_{a1}\sim 10^{-7}$） | 弱酸 | 弱酸 |

| 对比项 | HF | HCl | HBr | HI |
|------|:---:|:---:|:---:|:---:|
| 酸性 | 弱酸（$K_a\sim 10^{-4}$） | 强酸 | 强酸 | 最强 |
| 氢键 | 有 | 无 | 无 | 无 |

- $\mathrm{NH_3}$（碱性）vs $\mathrm{PH_3}$（近中性）vs $\mathrm{AsH_3}$（近中性）：同族氢化物碱性递减
- 储氢材料利用金属型氢化物的可逆吸放氢特性
