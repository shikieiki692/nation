# 氮族元素（N, P, As, Sb, Bi）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]

## 一、概述
VA 族：ns²np³。N 和 P 是非金属，As 半金属，Sb 和 Bi 金属。性质递变显著。

- N₂：N≡N 最强键之一，化学惰性
- P：白磷(P₄ 四面体)、红磷、黑磷（竞赛热点：白磷结构和反应）

## 二、重要反应

### 氮的化合物
| 化合物 | 氧化态 | 关键性质 |
|------|:---:|------|
| NH₃ | −3 | 三角锥，Lewis 碱，与 H⁺→NH₄⁺ |
| N₂H₄ (肼) | −2 | 强还原剂，火箭燃料 |
| NO | +2 | 奇电子分子，顺磁性 |
| NO₂ | +4 | 奇电子，红棕色，二聚 N₂O₄（无色，低温） |
| HNO₃ | +5 | 强氧化性酸，王水（3HCl+1HNO₃） |

### 硝酸的反应
- Cu + 4HNO₃(浓) → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O
- 3Cu + 8HNO₃(稀) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O
- 王水：HNO₃ + 3HCl → NOCl + Cl₂ + 2H₂O → 可溶 Au, Pt

### 氨的配合性质
- Ag⁺ + 2NH₃ → [Ag(NH₃)₂]⁺
- Cu²⁺ + 4NH₃ → [Cu(NH₃)₄]²⁺（深蓝色）
- 液态 NH₃ 作为非水溶剂（类似水，自耦电离 2NH₃ ⇌ NH₄⁺ + NH₂⁻）

## 三、核心原理

### 3.1 氮的固定与合成氨
$\mathrm{N}_2$ 分子由叁键结合，键能高达 $945\mathrm{kJ}\cdot\mathrm{mol}^{-1}$。使空气中游离的 $\mathrm{N}_2$ 变为化合态称为固氮。

- $\mathrm{N}_2 + \mathrm{O}_2 \longrightarrow 2\mathrm{NO}$：$\Delta G^{\ominus} = 181 - 0.025T$，转变温度 $T = 7.2 \times 10^{3}\mathrm{K}$（在闪电瞬间或汽车发动机附近发生，不能工业化）
- $\mathrm{N}_2 + 3\mathrm{H}_2 \longrightarrow 2\mathrm{NH}_3$：$\Delta G^{\ominus} = -92.2 + 0.199T$，转变温度 $T = 190^{\circ}\mathrm{C}$。工业条件：压力 $20\sim 30\mathrm{MPa}$，温度 $400\sim 530^{\circ}\mathrm{C}$，催化剂主要成分 $\mathrm{Fe}_2\mathrm{O}_3$ 和 $\mathrm{FeO}$，辅助成分 $\mathrm{Al}_2\mathrm{O}_3$—$\mathrm{CaO}$—$\mathrm{K}_2\mathrm{O}$ 等

### 3.2 氨的性质
- 沸点 $-33^{\circ}\mathrm{C}$，临界温度 $132^{\circ}\mathrm{C}$，加压降温易液化
- 气化热 $23.4\mathrm{kJ}\cdot\mathrm{mol}^{-1}$，可用作制冷剂
- 接受质子能力比 $\mathrm{H}_2\mathrm{O}$ 强，水溶液中部分电离成 $\mathrm{NH}_4^+$ 和 $\mathrm{OH}^-$ 而显碱性
- $\mathrm{NH}_3$ 分子中 N 原子上有孤对电子，容易形成配位化合物
- 液氨是常用的非水溶剂

### 3.3 氨氧化法制硝酸
$4\mathrm{NH}_3 + 5\mathrm{O}_2 \longrightarrow 4\mathrm{NO} + 6\mathrm{H}_2\mathrm{O}$、$2\mathrm{NO} + \mathrm{O}_2 \longrightarrow 2\mathrm{NO}_2$、$3\mathrm{NO}_2 + \mathrm{H}_2\mathrm{O} \longrightarrow 2\mathrm{HNO}_3 + \mathrm{NO}$。$\mathrm{HNO}_3$ 是具有强氧化性的强酸。

### 3.4 王水
1 体积市售浓硝酸 ($70\%$) 和 3 体积市售浓盐酸 ($37\%$) 的混合酸叫"王水"，既有硝酸的氧化性又有盐酸的络合性，能溶解 Au、Pt：$\mathrm{Au} + 4\mathrm{HCl} + \mathrm{HNO}_3 \longrightarrow \mathrm{HAuCl}_4 + \mathrm{NO} + 2\mathrm{H}_2\mathrm{O}$

### 3.5 氮的氧化物
氧化数从 $+1$ 至 $+5$ 齐全：
- $\mathrm{N}_2\mathrm{O}$（+1）：笑气，直线形 N—N—O，有 $\pi_3^4$ 键
- $\mathrm{NO}$（+2）：曾被认为是大气污染元，后发现对血管有调节血压和舒张作用（1992 年 Science 明星分子，1998 年 Nobel 奖）
- $\mathrm{N}_2\mathrm{O}_3$（+3）：$\mathrm{HNO}_2$ 的酸酐
- $\mathrm{NO}_2$（+4）：红棕色，低温二聚为 $\mathrm{N}_2\mathrm{O}_4$
- $\mathrm{N}_2\mathrm{O}_5$（+5）：$\mathrm{HNO}_3$ 的酸酐

### 3.6 VA 族元素的酸碱性递变
N 和 P 的氧化物为酸性，As 和 Sb 的氧化物为两性，Bi 的氧化物为碱性。

## 四、关键结论
- $\mathrm{N}\equiv\mathrm{N}$ 叁键是已知最强的化学键之一（$945\mathrm{kJ}\cdot\mathrm{mol}^{-1}$），导致 $\mathrm{N}_2$ 化学惰性
- 合成氨是涉及化学、化工、电子、机械、冶金等多种行业的综合工业
- $\mathrm{NH}_3$ 产量的约 $90\%$ 用于制化肥；另一个重要用途是制硝酸
- 王水溶解 Au/Pt 的原理：氧化性（$\mathrm{HNO}_3$）+ 络合性（$\mathrm{HCl}$ 提供 $\mathrm{Cl}^-$ 配体）
- $\mathrm{Cu}$ 和稀 $\mathrm{HNO}_3$ 产生 $\mathrm{NO}$，和浓 $\mathrm{HNO}_3$ 产生 $\mathrm{NO}_2$（但实际浓稀无严格界线）
- VA 族自上而下：N(非金属) $\to$ P(非金属) $\to$ As(半金属) $\to$ Sb(金属) $\to$ Bi(金属)

## 五、常见分类或情形
- **氮的氢化物**：$\mathrm{NH}_3$、$\mathrm{N}_2\mathrm{H}_4$（肼）、$\mathrm{NH}_2\mathrm{OH}$（羟氨）、$\mathrm{NaNH}_2$（氨基化钠）、$\mathrm{Li}_3\mathrm{N}$（氮化锂）
- **氮的氧化物**：$\mathrm{N}_2\mathrm{O}$、$\mathrm{NO}$、$\mathrm{N}_2\mathrm{O}_3$、$\mathrm{NO}_2/\mathrm{N}_2\mathrm{O}_4$、$\mathrm{N}_2\mathrm{O}_5$
- **含氧酸**：$\mathrm{HNO}_3$（硝酸）、$\mathrm{HNO}_2$（亚硝酸）
- **氮肥**：$(\mathrm{NH}_4)_2\mathrm{SO}_4$、$\mathrm{NH}_4\mathrm{Cl}$、$\mathrm{CO}(\mathrm{NH}_2)_2$（尿素）

## 六、适用条件与限制
- 合成氨需高压（$20\sim 30\mathrm{MPa}$）、催化剂（铁系）、适当温度（$400\sim 530^{\circ}\mathrm{C}$）
- $\mathrm{H}_2$ 分子很小，在高压下能钻过普通钢材，设备需用特种钢
- $\mathrm{NO}$ 迅速与 $\mathrm{O}_2$ 化合生成 $\mathrm{NO}_2$，汽车尾气的主要有害成分之一
- 液氨作溶剂需在加压或低温条件下
- $\mathrm{Cu}$ 与浓/稀 $\mathrm{HNO}_3$ 的反应产物随浓度连续变化

## 七、常见比较与易混点
| 对比项 | $\mathrm{N}_2$ | $\mathrm{O}_2$ |
|------|------|------|

| 沸点 | $-196^{\circ}\mathrm{C}$ | $-183^{\circ}\mathrm{C}$ |

|:---|:---|:---|:---|:---|:---|
| 酸碱性 | 酸性 | 酸性 | 两性（偏酸） | 两性（偏碱） | 碱性 |

## 九、典型题型

- **题型-氮氧化物推断与价态分析**：从颜色、磁性、二聚行为推断 NO/NO₂/N₂O₄/N₂O₅
- **题型-硝酸与金属反应计算**：浓稀 HNO₃ 产物差异、电子守恒 + 氮守恒联立
- **题型-氮族氟化物的结构化学**：NF₃ 碱性弱于 NH₃ 的原因、N₂F₂ 顺反异构、NF₄⁺ 水解
- **题型-合成氨/制硝酸工业流程**：平衡移动、催化剂、多步氧化串联
- **题型-重元素惰性电子对效应**：Bi(V) 强氧化性、Bi(III) 稳定、Bi₂O₅ 不存在（区别于 Sb₂O₅）

## 十、例题

### 例题 1（基础 · 硝酸与金属反应计算）

**题目**：将 3.84g Cu 投入一定量浓 HNO₃ 中，Cu 完全溶解，收集到 NO₂ 和 NO 混合气体 1.792L（标准状况）。求混合气体中 NO₂ 和 NO 各自的体积。

**分析**：Cu 失电子 = N 得电子。设 n(NO₂)=x, n(NO)=y，用电子守恒 + 总量守恒建立方程组。

**解答**：
$n(Cu) = 3.84 / 64 = 0.0600 \text{mol}$，Cu → Cu²⁺ 失 2e⁻/Cu → 总失 e⁻ = 0.1200 mol

$n(气体总) = 1.792 / 22.4 = 0.0800 \text{mol}$

N(+5) → N(+4) 在 NO₂：得 1e⁻/N
N(+5) → N(+2) 在 NO：得 3e⁻/N

电子守恒：$x \cdot 1 + y \cdot 3 = 0.1200$
总量守恒：$x + y = 0.0800$

解得：$x = 0.0600$ mol, $y = 0.0200$ mol
$V(NO_2) = 1.344 \text{L}$, $V(NO) = 0.448 \text{L}$

**反思**：硝酸与金属反应的关键——Cu 一开始与浓 HNO₃ 反应生成 NO₂，随着 HNO₃ 被消耗变稀，开始生成 NO。混合气体是两种产物的叠加。电子守恒 + 物料守恒是最干净的解法。

---

### 例题 2（提高 · NF₃ 为什么不是好 Lewis 碱）

**题目**：NH₃ 是经典 Lewis 碱（键角 107.3°，偶极矩 4.74×10⁻³⁰ C·m）。NF₃（键角 102°，偶极矩 0.78×10⁻³⁰ C·m）却几乎不具有碱性。解释原因。

**分析**：问题的关键在于 N−F 键的偶极方向。

**解答**：
- NH₃：N 电负性(3.0) > H(2.2)，N−H 键偶极指向 N，与孤对电子偶极同向 → **总偶极矩大** → 孤对电子"暴露"在分子外
- NF₃：F 电负性(4.0) > N(3.0)，N−F 键偶极**指向 F**，与孤对电子偶极**反向** → 总偶极矩极小 → 孤对电子"被拉回"分子内部
- 结果：NF₃ 的 N 上孤对电子密度显著降低，接受 H⁺ 的能力大幅减弱

**反思**：这是"电负性反转配位能力"的经典案例。类似地，PF₃ 是强 π-受体配体（反馈 π 键），而 NF₃ 几乎不配位。同族从上到下，配位能力可能完全反转。

### 例题 3（竞赛 · 2004 年全国初赛 · NF₃ 系列化合物推断）

**题目**：在 Cu 催化下 NH₃ 和 F₂ 反应得到铵盐和三角锥分子 A（键角 102°，偶极矩极小）。A 与汞共热得到汞盐和互为异构体的 B 和 C（相对分子质量 66）。A 与 F₂、BF₃ 反应得到四氟硼酸盐，其正离子水解定量生成 A 和 HF，同时产生的 O₂ 和 H₂O₂ 量因条件不同而变化。

（1）写出 A 的分子式。（2）解释 A 质子化放热远小于 NH₃ 的原因。（3）写出 B 和 C 的分子式和立体结构。（4）写出四氟硼酸盐正离子的化学式并解释其水解产物。

**分析**：A 是 NH₃ 全氟代产物 NF₃。B/C 分子量 66 → 设 N₂F₂（M=66），存在顺反异构。

**解答**：
（1）A = **NF₃**。$4NH_3 + 3F_2 \xrightarrow{Cu} NF_3 + 3NH_4F$
（2）N−F 键偶极指向 F，与 N 孤对电子偶极反向，导致孤对电子不易给出（见例题 2）。
（3）B/C = **N₂F₂**（顺反异构体）：$2NF_3 + 2Hg \rightarrow N_2F_2 + 2HgF_2$
- 顺式-N₂F₂：两个 F 在 N=N 同侧
- 反式-N₂F₂：两个 F 在 N=N 异侧
（4）正离子 = **NF₄⁺**（四面体）。$NF_3 + F_2 + BF_3 \rightarrow NF_4^+BF_4^-$
水解：$NF_4^+ + H_2O \rightarrow NF_3 + HOF + H^+$（定量生成 NF₃）
$2HOF \rightarrow 2HF + O_2$ 或 $HOF + H_2O \rightarrow HF + H_2O_2$（取决于条件）→ 无论如何定量生成 HF。

**反思**：这道题串联了"电负性影响碱性→氟代氮烷烃的结构化学→高价氟正离子的水解"，是竞赛对氮族氟化物化学的经典考查方式。

### 例题 4（竞赛 · Bi 的惰性电子对效应）

**题目**：Bi 的熔点是 545K，熔融后体积缩小。沸点 1883K 下蒸气密度 3.011g/L，2280K 下 1.122g/L，2770K 下 0.919g/L。Bi(III)盐在中性介质中水解生成含氧酸盐沉淀。在酸性介质中 Bi(V) 化合物能将 Mn²⁺ 氧化为 MnO₄⁻。

（1）判断 1883K 和 2280K 时气态 Bi 的分子型体。（2）写出 Bi 与浓 H₂SO₄、稀 HNO₃ 反应的方程式。（3）写出酸性介质中 Bi(V) 氧化 Mn²⁺ 的离子方程式。（4）Bi 比 Sb 更易形成 +3 氧化态而非 +5，解释原因。

**分析**：通过蒸气密度（M = ρRT/p）判断分子型体。惰性电子对效应——6s² 电子对在重元素中不易参与成键。

**解答**：
（1）1883K：$M = 3.011 \times 8.314 \times 1883 / 101.4 = 465 \text{g/mol} \approx 2 \times 208.98$ → 以 **Bi₂(g)** 存在。2280K：$M = 209.75 \text{g/mol}$ → 以 **Bi(g)** 存在。$Bi_2(g) \rightleftharpoons 2Bi(g)$
（2）浓 H₂SO₄：$2Bi + 6H_2SO_4 \rightarrow Bi_2(SO_4)_3 + 3SO_2↑ + 6H_2O$
稀 HNO₃：$Bi + 4HNO_3 \rightarrow Bi(NO_3)_3 + NO↑ + 2H_2O$（Bi 只被氧化到 +3）
（3）$5BiO_3^- + 2Mn^{2+} + 14H^+ \rightarrow 2MnO_4^- + 5Bi^{3+} + 7H_2O$
（4）Bi 位于第六周期，**6s² 惰性电子对效应**显著——6s 轨道因相对论效应收缩 + 镧系收缩导致 6s² 难以参与成键。因此 Bi 的稳定氧化态是 +3（6s² 不参与），+5 极不稳定（强氧化性）。Sb 的 5s² 惰性较弱，Sb₂O₅ 稳定存在而 Bi₂O₅ 不存在。

**反思**：惰性电子对效应是竞赛区分 VA 族重元素化学行为的关键概念——Bi(III) 稳定、Bi(V) 强氧化；Tl(I) 稳定、Tl(III) 强氧化；Pb(II) 稳定、Pb(IV) 强氧化。

## 十一、易错点

| 易错点 | 学生为什么会这么想 | 正确认识 | 口诀 |
|:---|:---|:---|:---|
| "浓 HNO₃ 一定生成 NO₂，稀 HNO₃ 一定生成 NO" | 教材简化表述被当成绝对规则 | Cu 与 HNO₃ 反应，产物随浓度**连续变化**。浓时以 NO₂ 为主，稀时以 NO 为主，中间浓度是混合物。更稀时甚至产生 N₂O、N₂、NH₄⁺ | "浓度是滑尺不是开关——NO₂ 和 NO 可以一起出来" |
| "王水就是硝酸和盐酸混在一起各干各的" | 把混合酸理解为简单的"叠加" | 王水的威力来自**氧化性（HNO₃）+ 络合性（Cl⁻）的协同**。HNO₃ 氧化 Au → Au³⁺，Cl⁻ 立即配位形成 [AuCl₄]⁻，降低 Au³⁺ 活度 → 推动氧化反应持续正向 | "硝酸推一把，氯离子接住——接力赛不是各自跑" |
| "NF₃ 和 NH₃ 一样是 Lewis 碱" | 同族类比，以为氟代不影响碱性 | NF₃ **几乎不是 Lewis 碱**。F 的电负性把 N 上孤对电子"拉回"分子内部。同理，PF₃ 的配位能力也远弱于 PH₃（但 PF₃ 是强 π-受体，与金属的反馈 π 键使它能稳定低价金属） | "氟代之后氮变蔫——孤对电子被拉跑了" |
| "NO 是奇数电子所以不稳定" | 奇电子分子 = 自由基 = 活泼的直觉 | NO 虽然是奇电子分子，但在常温下**相当稳定**（是 1992 年 Science 明星分子，体内信号分子）。真正不稳定的是它**与 O₂ 反应**生成 NO₂。奇电子 ≠ 必然不稳定（O₂ 也稳定！） | "奇电子不一定是短命鬼——NO 和 O₂ 都是稳定自由基" |
| "铋也是 VA 族所以有 +5 氧化态" | 按同族从上到下类比 | Bi 的 **6s² 惰性电子对效应**极强，+3 才是稳定氧化态。Bi(V) 是强氧化剂（能把 Mn²⁺ → MnO₄⁻）。Bi₂O₅ 根本不存在，对应的 NaBiO₃ 是强氧化剂 | "铋的 6s² 是铁饭碗——打死不参与成键" |

## 十二、🎯 教学视角

### 12.1 学生典型认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 教学干预 |
|:---|:---|:---|:---|
| "氮气不活泼所以氮元素也不活泼" | 把单质的惰性推广到元素 | N≡N 叁键（945 kJ/mol）是导致 N₂ 惰性的原因，但 N **元素**极其活泼——NH₃、HNO₃、NOₓ 都是高活性化合物。N₂ 的惰性是热力学+动力学双重原因 | 对比"N₂（睡觉的氮）"vs"HNO₃（暴脾气的氮）" |
| "硝酸的还原产物只有 NO 和 NO₂" | 教材只讲了这两种 | 随 HNO₃ 浓度降低和还原剂活性增强，还原产物可以是：NO₂ → NO → N₂O → N₂ → NH₄⁺（氧化数 +4 → +2 → +1 → 0 → −3）。极稀 HNO₃ + 活泼金属（如 Zn）可产生 NH₄⁺ | 用 N 的 Latimer 图展示完整还原链 |

### 12.2 讲授主线

（来自 [[教学逻辑提炼-Zchem-主族元素的结构化学I-第二轮深化]]）

氮族元素按三层递进：
1. **N 的特殊性**：N≡N 叁键 → 合成氨（工业化固定氮）→ NH₃ → HNO₃ 工业链 → N 的氧化物全家福（+1～+5）
2. **P 的结构化学**：白磷(P₄)/红磷/黑磷 → P₄O₆/P₄O₁₀ 结构 → 磷的含氧酸（H₃PO₄/H₃PO₃/H₃PO₂ 的"几元酸"陷阱）
3. **重元素的惰性电子对效应**：As/Sb/Bi 从上到下 +3 越来越稳定、+5 氧化性越来越强

### 12.3 与现实/直觉的连接

- **笑气（N₂O）**：牙科麻醉剂，直线形 N−N−O，含 π₃⁴ 键。吸入后产生欣快感→"笑气"得名。
- **硝酸甘油**：C₃H₅(ONO₂)₃，既是炸药（分解产生大量气体）又是心脏病急救药（释放 NO 扩张血管）。同一物质，剂量决定是杀人还是救人。
- **酸雨的另一面**：NOₓ 也是酸雨贡献者（汽车尾气 → NO → NO₂ → HNO₃）。但闪电固氮（N₂ + O₂ → 2NO → NO₂ → HNO₃）是自然界氮循环的重要环节。

## 十三、竞赛拓展

1. **NF₄⁺ 和 N₂F⁺ 的高价氟正离子化学**：NF₄⁺ 是少数稳定的 N(+5) 氟正离子（四面体，与 BF₄⁻ 形成盐）。N₂F⁺ 是直线形 [F−N≡N]⁺（与 N₂O 等电子）。这些是竞赛中对"高价氮不稳定"直觉的反例。

2. **过氧亚硝酸根 ONOO⁻ 的生物学意义**：NO + O₂⁻ → ONOO⁻（1997 年全国初赛考点）。ONOO⁻ 是强氧化剂和硝化剂，与多种疾病（炎症、中风、动脉粥样硬化）相关。同时也是巨噬细胞杀灭癌细胞和微生物的武器——典型的"双刃剑"分子。

3. **惰性电子对效应的结构根源**：相对论效应导致 6s 轨道收缩 + 镧系收缩使 6s 电子穿透到近核区，能量显著降低 → 6s² 成为"惰性对"不参与成键。这解释了：Tl(I) > Tl(III)、Pb(II) > Pb(IV)、Bi(III) > Bi(V) 的稳定性反转。是竞赛从"描述化学"上升到"结构解释"的桥梁概念。
