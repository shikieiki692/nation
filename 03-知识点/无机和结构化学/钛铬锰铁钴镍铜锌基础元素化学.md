# 钛铬锰铁钴镍铜锌基础元素化学

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]

## 一、定义

**钛（Ti）、铬（Cr）、锰（Mn）、铁（Fe）、钴（Co）、镍（Ni）、铜（Cu）、锌（Zn）**是第四周期第一过渡系（3d）的代表元素。掌握它们的基础化学性质——常见氧化态、特征颜色、重要化合物及相互转化——是无机元素化学的核心内容。

## 三、核心原理

### 3.1 第一过渡系的共性规律

1. **可变氧化态**：3d 电子可部分或全部参与成键，氧化态从 +2 到最高族序数
2. **颜色丰富**：d-d 跃迁导致水合离子和配离子呈现特征颜色
3. **催化活性**：未填满的 d 轨道可吸附反应物，降低活化能
4. **易形成配合物**：d 轨道接受配体孤对电子，形成配位化合物

### 3.2 氧化态稳定性趋势

- **左部（Ti、V、Cr）**：高氧化态稳定（Ti⁴⁺、Cr⁶⁺）
- **中部（Mn、Fe）**：多种氧化态共存，MnO₄⁻ 是最强氧化剂之一
- **右部（Co、Ni、Cu）**：低氧化态更稳定，Cu⁺ 易歧化
- **最右（Zn）**：只有 +2 氧化态（3d¹⁰ 全满，无 d-d 跃迁，无色）

## 四、关键结论：八元素速查表

| 元素 | 常见氧化态 | 特征颜色 | 重要化合物/反应 | 竞赛重点 |
|:---|:---:|:---|:---|:---|
| **Ti** 钛 | +4 (最稳), +3 | Ti⁴⁺: 无色; Ti³⁺: 紫色 | TiO₂ (白颜料); TiCl₄ + AlEt₃ (Ziegler-Natta) | Ti(IV) 的 Lewis 酸性 |
| **Cr** 铬 | +3 (最稳), +6 | Cr³⁺: 绿/紫; Cr₂O₇²⁻: 橙; CrO₄²⁻: 黄 | 铬矾; 铬酸洗液; Cr(OH)₃ 两性 | Cr(VI)/Cr(III) 氧化还原 |
| **Mn** 锰 | +2, +4, +6, +7 | Mn²⁺: 浅粉; MnO₄⁻: 紫; MnO₂: 黑黑 | KMnO₄ (强氧化剂); MnO₂ (干电池) | KMnO₄ 的还原产物与 pH 关系 |
| **Fe** 铁 | +2, +3 | Fe²⁺: 浅绿; Fe³⁺: 黄/棕 | 赤铁矿 Fe₂O₃; 磁铁矿 Fe₃O₄; 血红蛋白 | Fe²⁺/Fe³⁺ 互化; 铁的腐蚀 |
| **Co** 钴 | +2, +3 | Co²⁺: 粉红; Co³⁺: 蓝/绿 | CoCl₂ (干燥剂指示剂: 蓝→粉红); Vit B₁₂ | Co(III) 配合物惰性 |
| **Ni** 镍 | +2 | Ni²⁺: 绿色 | Ni(OH)₂; 镍氢/镍镉电池; Raney Ni | Ni 的配位化学 (dsp²) |
| **Cu** 铜 | +1, +2 | Cu²⁺: 蓝; Cu⁺: 无色; Cu: 紫红 | CuSO₄·5H₂O; Cu₂O (红); CuO (黑) | Cu⁺ 歧化; Cu(II) 的氧化性 |
| **Zn** 锌 | +2 | Zn²⁺: 无色 | ZnO (白, 两性的); 黄铜 (Cu-Zn); 镀锌 | Zn 的还原性; Zn(OH)₂ 两性 |

## 五、常见分类或情形

### 5.1 按氧化还原活性分类

| 类别 | 代表 | 特征 |
|:---|:---|:---|
| **强氧化剂** | KMnO₄ (Mn⁷⁺), K₂Cr₂O₇ (Cr⁶⁺) | 酸性条件下氧化性最强 |
| **中等还原剂** | Fe²⁺, Cu, Zn | Fe²⁺ 易被氧化为 Fe³⁺; Zn 可还原多数金属离子 |
| **催化活性金属** | Fe, Ni, Cu, Zn | Haber 法 (Fe), 氢化 (Ni), 偶联 (Cu/Pd) |
| **惰性/稳定** | TiO₂, Cr₂O₃, ZnO | 氧化物保护膜，耐腐蚀 |

### 5.2 重要反应方程式

**MnO₄⁻ 的还原（与 pH 相关）：**
- 酸性：$\ce{MnO4^- + 8H^+ + 5e^- -> Mn^2+ + 4H2O}$ （紫色→无色）
- 中性/弱酸：$\ce{MnO4^- + 2H2O + 3e^- -> MnO2 + 4OH^-}$ （紫色→棕色沉淀）
- 强碱：$\ce{MnO4^- + e^- -> MnO4^2-}$ （紫色→绿色）

**Cr(VI)/Cr(III) 转化：**
- 氧化：$\ce{Cr2O7^2- + 14H^+ + 6e^- -> 2Cr^3+ + 7H2O}$ （橙→绿）
- 碱性条件下 Cr(III) 可被氧化为 Cr(VI)：$\ce{2Cr(OH)4^- + 3H2O2 + 2OH^- -> 2CrO4^2- + 8H2O}$

**Cu⁺ 歧化：**
- $\ce{2Cu^+ -> Cu^2+ + Cu}$ （K ≈ 10⁶，水溶液中 Cu⁺ 极不稳定）

## 六、适用条件与限制

- **Ti³⁺**：强还原剂，空气中易被氧化为 Ti⁴⁺
- **Cr(VI)**：致癌物，酸性条件下氧化性极强，但碱性条件下氧化性减弱
- **Fe²⁺**：易被空气氧化，需酸性保护；与 SCN⁻ 不显色（Fe³⁺ 才显血红色）
- **Cu⁺**：水溶液中不稳定，需配合剂（如 NH₃, Cl⁻）或固态才稳定
- **Zn²⁺**：无色，易形成四配位四面体配合物

## 七、常见比较与易混点

- **Fe²⁺ vs Fe³⁺ 颜色**：Fe²⁺ 浅绿（稀溶液几乎无色），Fe³⁺ 黄/棕（水解产物）；Fe³⁺ + SCN⁻ → 血红色，Fe²⁺ 不反应
- **Cu²⁺ vs Cu⁺**：Cu²⁺ 蓝色（水合），Cu⁺ 无色但水溶液中歧化；Cu⁺ 配合物如 [Cu(CN)₂]⁻ 稳定
- **Cr³⁺ vs Al³⁺**：两者都形成 M(OH)₃ 两性氢氧化物，但 Cr³⁺ 有颜色（绿/紫），Al³⁺ 无色
- **MnO₄⁻ 还原产物与 pH**：酸性→Mn²⁺（无色），中性→MnO₂（棕），碱性→MnO₄²⁻（绿）——这是推断题高频考点
- **Zn vs Al 两性**：Zn(OH)₂ 和 Al(OH)₃ 都溶于强碱，但 Zn 不是两性金属（Zn 溶于酸，不溶于碱；ZnO 和 Zn(OH)₂ 是两性）

## 九、典型题型

- 元素推断题：根据颜色变化（如"紫色溶液被还原后变为无色"→MnO₄⁻→Mn²⁺）推断元素
- 反应方程式书写题：KMnO₄ 在不同 pH 下的还原反应
- 颜色识别题：给出离子/化合物颜色，判断是哪种元素
- 性质比较题：比较 Ti/Cr/Mn/Fe/Co/Ni/Cu/Zn 的某一性质（如最高价态、离子颜色、氢氧化物酸碱性）

## 十、例题

### 例题 1

**题目：** 某金属 M 的硝酸盐溶液为绿色，加入 NaOH 生成灰绿色沉淀，该沉淀溶于过量 NaOH 得绿色溶液，酸化后又变回绿色溶液。加入氧化剂后溶液变为橙黄色，再加入还原剂又恢复绿色。推断 M 是什么元素，并写出关键反应方程式。

**分析：** 绿色→灰绿色沉淀→溶于过量 NaOH→酸化恢复→氧化变橙黄→还原恢复。这是 Cr³⁺ 的典型反应序列。

**解答：**
- **M = Cr（铬）**
- Cr³⁺ + 3OH⁻ → Cr(OH)₃↓（灰绿色）
- Cr(OH)₃ + OH⁻ → [Cr(OH)₄]⁻（绿色，溶于过量碱）
- 酸化：[Cr(OH)₄]⁻ + 4H⁺ → Cr³⁺ + 4H₂O（恢复）
- 氧化：2Cr³⁺ + 3H₂O₂ + 10OH⁻ → 2CrO₄²⁻ + 8H₂O（碱性条件下氧化为黄色铬酸根）
- 或酸性氧化：Cr₂O₇²⁻（橙黄色）
- 还原：Cr₂O₇²⁻ + 3SO₂ + 2H⁺ → 2Cr³⁺ + 3SO₄²⁻ + H₂O（恢复绿色）

**反思：** Cr³⁺ 的两性氢氧化物和 Cr(VI)/Cr(III) 的互化是元素推断题的"指纹"。

## 十一、易错点

- KMnO₄ 在酸性条件下被还原为 Mn²⁺（无色），不是 MnO₂（棕色）
- Cu⁺ 在水溶液中歧化，不能稳定存在——但 CuI、Cu₂O 固态稳定
- CoCl₂ 的颜色变化：无水 CoCl₂ 蓝色，水合 [Co(H₂O)₆]²⁺ 粉红——用于硅胶干燥剂指示
- Zn²⁺ 无色但 ZnO 白色——不要混淆离子颜色和化合物颜色
- Fe³⁺ 的氧化性：可氧化 Cu（Fe³⁺ + Cu → Fe²⁺ + Cu²⁺），用于蚀刻电路板

### 12.2 学生易踩的认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 辨析方法 |
|:---|:---|:---|:---|
| "Cu⁺ 蓝色" | 与 Cu²⁺ 混淆 | Cu²⁺ (d⁹) 蓝色; Cu⁺ (d¹⁰) 无色 | 记电子构型：d⁹ 有颜色，d¹⁰ 无色 |
| "KMnO₄  always 还原成 Mn²⁺" | 只记酸性条件 | 产物与 **pH 强相关**：酸→Mn²⁺, 中→MnO₂, 碱→MnO₄²⁻ | 做题先看溶液酸碱性 |
| "Zn 是两性金属" | 与 Al 混淆 | Zn 单质不溶于碱；**ZnO 和 Zn(OH)₂ 是两性** | 区分金属单质 vs 氧化物/氢氧化物 |
| "Fe²⁺ 与 SCN⁻ 显红色" | 与 Fe³⁺ 混淆 | 只有 Fe³⁺ + SCN⁻ → [Fe(SCN)]²⁺ 血红色 | Fe²⁺ 需先氧化为 Fe³⁺ 才显色 |

### 12.3 入门级例题

**题目：** 向含 Fe²⁺ 和 Fe³⁺ 的混合溶液中，如何分别检验两者？

**预期解答路径：**
1. 取一份溶液，加入 KSCN：显血红色 → 证明 Fe³⁺ 存在
2. 取另一份溶液，加入 K₃[Fe(CN)₆]（铁氰化钾）：生成深蓝色沉淀（Turnbull's blue）→ 证明 Fe²⁺ 存在
3. 或：加入 KMnO₄，紫色褪去 → 证明 Fe²⁺ 存在（Fe²⁺ 还原 MnO₄⁻）

**课堂引导提问：**
- 若学生说"都用 KSCN"：问"Fe²⁺ 与 SCN⁻ 反应吗？如何区分 Fe²⁺ 和 Fe³⁺？"

### 12.4 与现实/直觉的连接

- 类比：过渡金属的颜色变化像"化学红绿灯"——Cr₂O₇²⁻ 橙（停）、Cr³⁺ 绿（行）、MnO₄⁻ 紫（警告）
- 反例：学生以为"金属离子都有颜色"——实际上 Zn²⁺ (d¹⁰)、Ti⁴⁺ (d⁰)、Cu⁺ (d¹⁰) 都是无色的，因为无 d-d 跃迁
