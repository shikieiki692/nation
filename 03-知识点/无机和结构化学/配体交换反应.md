# 配体交换反应（σ-Bond Metathesis）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[12-配合物]]、[[13-元素化学]]、[[决赛11-金属有机化学]]

## 一、定义
**σ-键复分解（σ-Bond Metathesis）**：两个 σ 键在**四中心协同过渡态**中同时断裂、重新组合，得到两个新的 σ 键。无氧化态变化，无中间体。

通用形式：
$$\ce{A-B + C-D ->[\text{4-center TS}] A-C + B-D}$$

> 又称 **复分解反应**、**双置换反应**——但与有机的双置换稍有差异：σ-bond metathesis 通常涉及金属或高价主族中心。

## 二、考纲对应
- 对应考纲条目：[[12-配合物]]、[[13-元素化学]]、[[决赛11-金属有机化学]]

## 三、核心原理

### 3.1 四中心过渡态（4-center TS）
```
    A — B
    |   |
    D — C
```
- A、B、C、D 共面
- A-B 和 C-D 两键同时断裂
- A-C 和 B-D 两键同时形成
- **无中间体**，仅过渡态

### 3.2 与氧化加成-还原消除的区别
| | σ-bond metathesis | 氧化加成 + 还原消除 |
|---|---|---|
| 步骤 | 1 步（协同） | 2 步 |
| 氧化态变化 | 无 | 中间体氧化态 +2 |
| 适用金属 | 早过渡金属（d⁰）、主族 | 后过渡金属（d⁸/d¹⁰） |
| 实例 | $\ce{Cp2ZrH2 + H-X \to Cp2Zr(H)X + H2}$ | $\ce{Pd^0 + Ar-X \to Pd^{II}(Ar)X}$ |

### 3.3 主族复分解的常见模式
1. **酸碱复分解**：$\ce{HX + M-Y \to M-X + HY}$
2. **σ键复分解**：$\ce{R-M + H-X \to R-H + M-X}$
3. **多键复分解**（烯烃复分解，Grubbs 催化）

## 四、关键结论

### 4.1 早过渡金属（d⁰）特别擅长 σ-bond metathesis
- d⁰ 金属（如 Sc³⁺、Ti⁴⁺、Zr⁴⁺、Lu³⁺）无 d 电子可参与氧化加成
- 但通过 σ-bond metathesis 可活化 H-H、C-H、Si-H、O-H 等 σ 键
- 这是早过渡金属催化的关键步骤

### 4.2 主族元素的 σ-bond metathesis
- $\ce{R3Si-H + R-Li \to R3Si-R + Li-H}$ —— 类似 σ键复分解
- $\ce{H-X + Mg-R \to H-R + Mg-X}$ —— Grignard 试剂的水解

### 4.3 烯烃复分解（Olefin Metathesis）
- Grubbs 催化剂（Ru-亚甲基，2005 诺贝尔化学奖）
- 通过 [2+2] 环加成-逆环加成 → 烯烃 C=C 重排

### 4.4 与 Brønsted 酸碱反应的关系
- 所有 Brønsted 质子转移可视为"σ-bond metathesis 的特例"——其中 A=H、B=baseLP、C=D=acid
- 例：$\ce{HF + NaOH \to NaF + H2O}$ 是质子复分解

## 五、常见分类或情形

### 5.1 主族化合物的复分解
| 反应类型 | 实例 |
|---|---|
| 卤化物-氢氧化物 | $\ce{AlCl3 + 3H2O \to Al(OH)3 + 3HCl}$ |
| 卤化物互换 | $\ce{SiF4 + 4HCl \to SiCl4 + 4HF}$（不利，需高温） |
| 配体复分解 | $\ce{Cp2ZrCl2 + 2 RLi \to Cp2ZrR2 + 2 LiCl}$ |

### 5.2 共聚物合成中的烯烃复分解
- ROMP（开环烯烃复分解聚合）
- ADMET（无环二烯复分解聚合）

### 5.3 C-H 活化的 σ-bond metathesis
- 早过渡金属 + C-H：$\ce{Cp2*ScCH3 + CH4 \to Cp2*Sc-CH3 + CH4}$（甲烷-甲基的"看似无变化"实际有交换——同位素标记证实）
- 后过渡金属常通过氧化加成

## 六、适用条件与限制

### 适用条件
- 早过渡金属（d⁰）的 σ键复分解
- 主族重元素的高价中间体反应
- Brønsted 酸碱反应（广义）

### 限制
- 几何要求严格：需四中心共面过渡态
- 能量上需要"恰好相消"——若 σ键能差太大则不可行
- 对位阻敏感

## 七、常见比较与易混点

### 1. σ-bond metathesis vs 氧化加成-还原消除
- 前者协同 1 步，后者分 2 步
- 前者氧化态不变，后者中间金属氧化态升高 2

### 2. σ-bond metathesis vs SN2-Si
- 前者**两个σ键同时断裂**（A-B + C-D → A-C + B-D）
- 后者**一个亲核体进攻一个亲电体**（Nu + R-X → Nu-R + X⁻），且重元素中走 A-D 两步

### 3. 复分解 vs 加成
- 复分解：键数不变（两键 → 两键）
- 加成：键数增加（一键 + 一键 → 一键，π 变 σ）

## 十、例题

### 例题 1：金属酰胺的水解
**题目：** $\ce{LiNMe2 + H2O \to LiOH + HNMe2}$，写出协同复分解机理。
**分析：** 4 个原子（Li、N、O、H）在过渡态共面。
**解答：**
```
    Li — NMe2
    |    |
    O    H
    |    |
    H ———
```
Li-N 键和 O-H 键同时断裂，Li-O 键和 N-H 键同时形成。

### 例题 2：金属-甲烷活化
**题目：** $\ce{Cp2*Sc-CH3 + ^{13}CH4 \to Cp2*Sc-^{13}CH3 + CH4}$，证据 σ-bond metathesis。
**分析：** 同位素标记表明 Sc 上的甲基与外加甲烷甲基交换。
**解答：** 通过四中心过渡态，Sc-C 和 C-H 两键同时断裂、重组。

## 十一、易错点
- 把 σ-bond metathesis 和 SN2 混淆
- 忘记早过渡金属（d⁰）才是典型的复分解中心
- 把烯烃复分解（[2+2]）也叫 σ-bond metathesis（应叫 olefin/π-bond metathesis）

## 十二、竞赛拓展
- **Grubbs 催化剂**：第一代和第二代，不同活性谱
- **Schrock 复分解催化剂**：Mo/W 早过渡金属体系
- **C-H 活化**：σ-bond metathesis vs 氧化加成的竞争
- **酸碱复分解**：盐酸 + 氢氧化钠的本质
