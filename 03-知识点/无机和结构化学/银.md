# 银

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]

## 一、概述
- [Kr]4d¹⁰5s¹
- 最主要氧化态：+1
- 化学性质较不活泼，但比金更容易被氧化

## 二、重要反应
- 卤化银：AgCl（白）、AgBr（浅黄）、AgI（黄）
- Ag⁺ + 2NH₃ → [Ag(NH₃)₂]⁺
- Tollens 试剂：银镜反应的核心是 [Ag(NH₃)₂]⁺
- Ag⁺ 与 S₂O₃²⁻、CN⁻ 均可形成稳定配合物

## 三、核心原理

### 3.1 货币金属中的位置
Ag 位于 Cu-Ag-Au 一链中，最稳定价态为 +1。相比 Cu 和 Au，Ag 的化学表现兼具“较高电势”和“较强配位倾向”。

### 3.2 卤化银与配位溶解
AgCl、AgBr、AgI 的溶度积依次减小，因此在氨水中的行为不同：
- AgCl 易溶于 NH₃
- AgBr 微溶
- AgI 基本不溶

### 3.3 银镜反应与摄影化学
[Ag(NH₃)₂]⁺ 是有机化学与分析化学中常见的温和氧化剂。卤化银的感光性则是摄影化学的基础应用之一。

## 四、关键结论
- Ag 最稳定价态是 +1
- 卤化银是“沉淀平衡 + 配位平衡 + 感光性”三位一体的经典体系
- 银镜反应与 Tollens 试剂是课堂和题目中的高频节点
- Ag⁺ 常形成直线形二配位配合物

## 五、常见分类或情形
- **银盐**：AgNO₃、AgCl、AgBr、AgI
- **银的配合物**：[Ag(NH₃)₂]⁺、[Ag(S₂O₃)₂]³⁻、[Ag(CN)₂]⁻
- **银的氧化物**：Ag₂O

## 六、适用条件与限制
- AgOH 不能稳定分离，通常转化为 Ag₂O
- 卤化银的溶解/不溶需要同时考虑 Ksp 和配位常数
- Tollens 试剂需现配现用，避免副反应和安全问题

## 七、常见比较与易混点
- AgCl 能溶于氨水，并不代表所有卤化银都一样
- Ag 与 Cu 同族，但 Ag 最稳定常见价态是 +1，而 Cu 在水溶液中常见 +2
- 不能把 Ag₂O 与“AgOH 固体”混用

## 十、例题
- **例题**：为什么 AgCl 能溶于氨水，而 AgI 基本不溶？  
  **答案要点**：两者都能与 NH₃ 配位，但 AgI 的溶度积极小，配位带来的平衡推动不足以显著溶解；AgCl 的 Ksp 较大，因此更容易被拉入配合物形式。

## 十一、易错点
- 误以为所有卤化银在氨水中行为相同
- 只记银镜反应产物，不理解其中核心是 [Ag(NH₃)₂]⁺ 的温和氧化性
- 混淆 AgOH 与 Ag₂O，前者并不能稳定分离

## 十二、🎯 教学视角
- 第一轮讲银最适合抓“卤化银-氨配银-银镜反应”三联主线
- 银页是把沉淀溶解平衡、配位平衡和元素化学连接起来的好节点
- 若与分析化学联动，可顺带挂接莫尔法、佛尔哈德法等银盐分析体系
