# 金刚石型结构

![[mineru/03-教材书籍/无机化学第6版Weller/无机化学第6版Weller/无机化学第6版主族364-564_images/4eef2f5667ede1df13bd746adfe1a69ee77a18b816ee289ef2f9ecefad77102d.jpg]]

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[17-常见晶体结构]]

## 一、结构描述
- C 原子：fcc 位置 + 1/2 四面体空隙交错填充
- 每个 C：sp³ 杂化，正四面体配位 4 个 C
- = 两套 fcc 子格子沿体对角线偏移 1/4

## 二、参数
- 晶胞含：8 个 C 原子
- 空间利用率：34%（很低，因共价键有方向性）
- C—C 键长：154 pm

## 三、类似结构
- Si, Ge, α-Sn（灰锡）
- 立方 BN（闪锌矿型，B+N 替代 C+C）

## 四、关键结论
### 晶胞参数
- 金刚石属面心立方晶胞，含 **8 个 C 原子**（4 个 fcc + 4 个 Td 交替填充）
- 空间利用率仅 **34%**（共价键方向性导致，远低于金属最密堆积的 74%）
- C—C 键长 **154 pm**，配位数 **4**（sp³ 正四面体）

### 性质（§13.4.4）
- 自然界中**硬度最大**的晶体之一
- 精密机械中每年消耗**几百万克拉**（1 克拉 = 200 mg）
- 用做金属表面磨料、石油勘探钻头

### 共价晶体特征
金刚石是共价晶体（原子晶体）的典型代表——低配位数（4）、低空间利用率（34%）、高熔点、高硬度、不导电、整个晶体是一个巨大分子。

## 五、常见分类或情形
与立方 ZnS 型的关系（§13.4.2）：
- **立方 ZnS 型** = S²⁻ fcc + Zn²⁺ 在 1/2 Td
- **金刚石型** = C fcc + C 在 1/2 Td（**两套 fcc 子格子沿体对角线偏移 1/4**）
- 立方 BN（闪锌矿型）= B + N 替代 C + C

### 其他金刚石型晶体
Si、Ge、$\alpha$-Sn（灰锡）均为金刚石型结构，但禁带较窄，属于半导体。

## 六、适用条件与限制
- 更适用于方向性极强的共价晶体，不适合简单离子球模型解释
- 结构相似不等于物性相同，Si、Ge、C 的禁带和硬度差异仍很大

## 七、常见比较与易混点
- 金刚石型可视为“同种原子的立方 ZnS 型”，但成键本质是共价键而非离子键
- 低空间利用率并不代表结构松散，方向性共价键会显著提高硬度
