## 一、基本定义

**Tanabe-Sugano图**（田边-菅野图，简称TS图）是一种描述**多电子体系络合物**的电子态能量随配位场强度变化的相关图（correlation diagram）。TS图将弱场极限（自由离子光谱项）与强场极限（配位场组态）平滑连接，是定量分析和归属d区络合物电子吸收光谱的核心工具。

与Orgel图的关键区别：
- **Orgel图**：以基谱项能量为参照，不同组态分开绘制，仅标注自旋允许跃迁，**不能用来定量获取Δ₀值**
- **TS图**：以最低谱项能量为零点，一个图包含该$d^n$组态的**所有谱项**（含自旋禁阻），坐标轴无量纲化（E/B 对 Δ₀/B），可用于**从实验光谱反推Δ₀和Racah参数B**

TS图的横轴为 $\Delta_0 / B$（配位场强度），纵轴为 $E/B$（谱项能量），其中B为Racah电子排斥参数。所有能量均以基谱项为零参照。

## 二、核心内容

### 2.1 图的构造原理

1. **弱场极限**（左侧，$\Delta_0/B \to 0$）：谱项能量等于自由离子的光谱项能量（由Racah参数A、B、C确定）。各谱项按Russell-Saunders偶合标记（如 ${}^{3}F$、${}^{1}D$、${}^{3}P$ 等）
2. **强场极限**（右侧，$\Delta_0/B$ 很大）：谱项按配位场组态标记（如 $t_{2g}^{2}$ 导出的 ${}^{3}T_{1g}$，$t_{2g}^{1}e_g^{1}$ 导出的 ${}^{3}T_{2g}$ 等）
3. **中间场区**：相关线将两个极限平滑连接，**具有相同对称类型的谱项**发生混合，曲线弯曲——遵循**不相交规则**

### 2.2 不相交规则（Noncrossing Rule）

当配位场增强导致两个**相同对称类型**的谱项相互靠近时，它们不会相交，而是**彼此弯离**（avoided crossing）。这是因为相同对称性的波函数发生混合（组态相互作用）。例如 $d^{2}$ 组态TS图中，两个 ${}^{1}E$ 谱项、两个 ${}^{1}T_{2}$ 谱项、两个 ${}^{1}A_{1}$ 谱项都展示了不相交规则的效应。

### 2.3 零能量的选取与自旋交叉

TS图的零能量总是取**最低谱项**的能量。因此对于 $d^{4}$、$d^{5}$、$d^{6}$、$d^{7}$ 组态，当场强增大导致高自旋→低自旋转变（基谱项改变）时，图中线的**斜率会发生突然变化**——这是自旋交叉在TS图上的直观表征。

### 2.4 从光谱数据反推Δ₀和B的标准方法

TS图设计的根本目的：将**实验观测的跃迁能比值**与**TS图上理论比值**匹配，从而读出 $\Delta_0/B$ 和 $E/B$，再反推出 $\Delta_0$ 和 $B$。标准操作流程：

1. **识别$d^n$组态**：由中心金属离子的氧化态和配体场（高/低自旋）确定d电子数
2. **选择对应TS图**：$d^2$至$d^8$各有专属图（资源节6），$d^1$、$d^9$、$d^{10}$ 因电子-空穴对称性过于简单不需要TS图
3. **提取观测跃迁能**：从紫外-可见光谱中读取自旋允许跃迁的吸收峰位（以 $\mathrm{cm^{-1}}$ 为单位）
4. **计算跃迁能比值**：$\nu_2 / \nu_1$（取最低能自旋允许跃迁为分母）
5. **在TS图上定位**：在横轴上找到使纵轴跃迁能比值与实验值匹配的唯一点
6. **读出参数**：从该点读出 $\Delta_0/B$ 值，从纵轴读出各跃迁的 $E/B$ 值
7. **反推Δ₀和B**：$B = \nu_1 / (E_1/B)$，$\Delta_0 = (\Delta_0/B) \times B$

### 2.5 d²-d⁸组态各图的自旋允许跃迁数量

| $d^n$ | 基谱项 | 自旋允许跃迁数 | 典型Ts图特征 |
|:---:|:---|:---:|:---|
| $d^{2}$ | ${}^{3}T_{1g}$ | 3条（${}^{3}T_{2g}\leftarrow{}^{3}T_{1g}$，${}^{3}T_{1g}(P)\leftarrow{}^{3}T_{1g}$，${}^{3}A_{2g}\leftarrow{}^{3}T_{1g}$）| 基态线弯曲（$t_{2g}^{2}$与$t_{2g}^{1}e_g^{1}$混合）|
| $d^{3}$ | ${}^{4}A_{2g}$ | 3条（${}^{4}T_{2g}$，${}^{4}T_{1g}(F)$，${}^{4}T_{1g}(P)$）| 基态线水平（$t_{2g}^{3}$无分裂），竞赛最重要 |
| $d^{4}$ | ${}^{5}E_{g}$（高自旋）| 1条 | 存在自旋交叉转折点 |
| $d^{5}$ | ${}^{6}A_{1g}$（高自旋）| **0条** | 无自旋允许d-d跃迁（如Mn²⁺配合物颜色极淡）|
| $d^{6}$ | ${}^{5}T_{2g}$（高自旋）| 1条 | 存在自旋交叉转折点 |
| $d^{7}$ | ${}^{4}T_{1g}$（高自旋）| 3条 | 基态线弯曲，存在自旋交叉 |
| $d^{8}$ | ${}^{3}A_{2g}$ | 3条（${}^{3}T_{2g}$，${}^{3}T_{1g}(F)$，${}^{3}T_{1g}(P)$）| 基态线水平，与$d^{3}$有电子-空穴对应关系 |

### 2.6 Franck-Condon原理与谱带宽度

**Franck-Condon原理**：电子跃迁发生的时间极短，跃迁过程中原子核位置来不及改变（垂直跃迁）。因此电子跃迁"瞬间检测"了分子在各种振动状态下的能量分布。

在TS图上的体现：
- **平行于基态线的激发态线** → 跃迁能基本不受 $\Delta_0$ 变化（如分子振动引起的M—L键长变化）影响 → **谱带尖锐**
- **严重不平行于基态线的激发态线** → 跃迁能显著依赖于 $\Delta_0$ 的瞬时值 → **谱带宽泛**

以$d^{2}$组态为例：${}^{3}T_{2g}\leftarrow{}^{3}T_{1g}$谱带宽（${}^{3}T_{2g}$线不平行于基态${}^{3}T_{1g}$），${}^{3}A_{2g}\leftarrow{}^{3}T_{1g}$谱带**更宽**（${}^{3}A_{2g}$线更不平行），而能量较低的${}^{1}T_{2g}\leftarrow{}^{3}T_{1g}$自旋禁阻跃迁虽然强度弱，谱带却**非常尖锐**（${}^{1}T_{2g}$线近似平行于基态线）。

### 2.7 什么时候该从“配位场理论”升级到 TS 图
TS 图不是讲配合物颜色时的起点，而是讲到下面这些任务时才需要正式调用：

1. 已经明确中心金属的 $d^n$ 组态和大致构型；
2. 需要把多条吸收带归属到具体跃迁，而不是只判断“强场/弱场”；
3. 需要由实验光谱**反推** $\Delta_0$、B 或比较不同配体下的参数变化；
4. 题目开始出现“第三条带为何缺失/为何被遮蔽/为何极弱”等光谱细节。

如果课堂还停留在“为什么有颜色、为什么颜色深浅不同”，更合适的入口应是 [[配位场理论]] 和 [[过渡元素颜色与配位行为]]。

## 三、竞赛要点

1. **核心技能**：给定紫外-可见光谱数据和$d^n$组态，**用TS图反推Δ₀和B值**——这是竞赛决赛中光谱分析类题型的最高频考查方式
2. **比值匹配法**：跃迁能比值是唯一匹配判据。例如$d^3$体系，$\nu_2/\nu_1 \approx 1.32$（$[\mathrm{Cr(NH_3)_6}]^{3+}$的典型值）
3. **读图精度**：在TS图上定位时只需精确到$\Delta_0/B \pm 1$范围内，B值计算结果精确到个位数即可
4. **电子云重排参数β**：$\beta = B(\text{络合物})/B(\text{自由离子})$，是d电子离域程度的量度。β越小，共价性越强。电子云重排序列：$\mathrm{Br^-}<\mathrm{CN^-}<\mathrm{Cl^-}<\mathrm{NH_3}<\mathrm{H_2O}<\mathrm{F^-}$
5. **自旋禁阻的特殊情况**：$d^5$高自旋（如Mn²⁺）无自旋允许d-d跃迁，配合物颜色极淡（由极弱自旋禁阻跃迁产生）
6. **使用前检查表**：先排除“其实是电荷转移带主导”“其实是$d^0/d^{10}$无 d-d 跃迁”“其实只有定性颜色判断并不需 TS 图”这三类误用场景
7. **题型挂点**：TS 图最常见的不是“纯读图”，而是“比值定位 + 跃迁归属 + 遮蔽带解释 + 参数合理性回判”

## 四、例题

### 例题 1：完整TS图计算流程（Weller例题20.8）

**题目**：$[\mathrm{Cr(NH_3)_6}]^{3+}$ 的紫外-可见光谱显示两个低能配位场吸收带：$\nu_1 = 21550\ \mathrm{cm^{-1}}$，$\nu_2 = 28500\ \mathrm{cm^{-1}}$（第三个自旋允许跃迁被电荷转移带遮蔽）。请用TS图推断该络合物的 $\Delta_0$ 和 $B$ 值。

**分析**：$\mathrm{Cr^{3+}}$ 为 $d^{3}$ 组态，基谱项 ${}^{4}A_{2g}$，有3个自旋允许四重态跃迁：${}^{4}T_{2g}\leftarrow{}^{4}A_{2g}$（ν₁）、${}^{4}T_{1g}(F)\leftarrow{}^{4}A_{2g}$（ν₂）、${}^{4}T_{1g}(P)\leftarrow{}^{4}A_{2g}$（ν₃，被CT带遮蔽）。使用$d^{3}$ TS图。

**解答**：

**Step 1**: 跃迁能比 = $\nu_2/\nu_1 = 28500/21550 = 1.32$

**Step 2**: 在 $d^{3}$ TS图（图20.31）上寻找纵轴两个跃迁之比为1.32的横轴位置。该点位于图右部偏远处。

**Step 3**: 从该点读取 $\Delta_0/B = 33.0$

**Step 4**: 表示较低能跃迁（${}^{4}T_{2g}\leftarrow{}^{4}A_{2g}$）的箭头尖端在纵轴 $E/B = 32.8$ 处

**Step 5**: $32.8B = 21550\ \mathrm{cm^{-1}} \Rightarrow B = 21550/32.8 = 657\ \mathrm{cm^{-1}}$

**Step 6**: $\Delta_0 = 33.0 \times 657 = 21700\ \mathrm{cm^{-1}}$

**结果**：$\Delta_0 = 21700\ \mathrm{cm^{-1}}$，$B = 657\ \mathrm{cm^{-1}}$

**验证**：$\mathrm{Cr^{3+}}$ 自由离子的 $B \approx 1030\ \mathrm{cm^{-1}}$，电子云重排参数 $\beta = 657/1030 \approx 0.64$（64%），NH₃为中等偏强场配体，结果合理。

**反思**：
- 如果实际比值不是1.32，说明场强不同，在TS图上向左或向右移动
- $[\mathrm{Cr(OH_2)_6}]^{3+}$ 的 $\Delta_0 = 17600\ \mathrm{cm^{-1}}$（比NH₃小），因为H₂O在光谱化学序列中位于NH₃左侧

### 例题 2：自旋允许跃迁数的判断

**题目**：$[\mathrm{Mn(H_2O)_6}]^{2+}$（高自旋$d^{5}$）的吸收光谱中应观察到几个自旋允许的d-d跃迁？其颜色特征如何？

**分析**：$d^{5}$高自旋基谱项为 ${}^{6}A_{1g}$（六重态），而所有激发态谱项的多重度均≤4。从六重态到四重态/二重态的跃迁均违反自旋选律（ΔS ≠ 0）。

**解答**：**0个**自旋允许d-d跃迁。所有可能的11个跃迁均为自旋禁阻，因此Mn²⁺配合物颜色极淡（几乎无色或极浅粉色）。其极弱的颜色来自自旋-轨道耦合部分解除自旋禁阻后产生的极低强度吸收。

**反思**：$d^{5}$高自旋是唯一一个无任何自旋允许d-d跃迁的组态，这是TS图计算题中的常见干扰项——如果题目给的光谱中没有明显的配位场谱带，应首先考虑是否为$d^{5}$高自旋体系。

## 五、易错点

1. **混淆Orgel图和Tanabe-Sugano图**：Orgel图以基谱项为参照（能量不归零）、只示自旋允许跃迁、**不能用来求Δ₀**。TS图以最低谱项能量为零点、含全部谱项、**必须用于求Δ₀和B**。竞赛中若问"用哪种图求Δ₀"，答案永远是TS图。
2. **选错TS图**：$d^{n}$ 和 $d^{10-n}$ 使用不同的图（光谱项结构不同），不能混淆。例如 $d^{3}$ 和 $d^{7}$ 即使有电子-空穴对称性，TS也不同。
3. **跃迁能比值匹配不唯一时的处理**：若TS图上存在多个比值匹配点（少见），需要结合其他光谱特征（谱带强度、第三跃迁位置）和化学常识（配体场强合理范围）排除。
4. **B值的比较**：络合物的B值应**小于**自由离子的B值（电子离域降低排斥）。如果计算出$B > B_{\text{自由离子}}$，说明读数有误。
5. **忽略Franck-Condon宽度信息**：TS图不仅能求Δ₀和B，还能定性解释谱带宽度——这在归属跃迁时是重要的辅助判据。
