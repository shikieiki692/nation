# Pauling规则（Pauling's Rules）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[15-晶体结构]]

## 一、定义

Pauling规则是Linus Pauling提出的**一套经验规则**，用于预测和解释离子晶体的结构稳定性。它从离子半径、电荷、配位数等角度出发，为理解复杂的离子晶体结构提供了定性框架。

- 本知识点在考纲中的作用：解释离子晶体的配位数、配位多面体连接方式及结构稳定性

## 三、核心原理

### 3.1 Pauling第一规则（配位多面体规则）

**正离子周围形成一个负离子配位多面体，正负离子间的距离取决于半径之和，配位数取决于半径比。**

- 配位多面体类型由 $r_+/r_-$ 决定
- 详见 [[离子晶体]] 中的半径比规则表

### 3.2 Pauling第二规则（电价规则 / 静电价规则）

**在稳定的离子晶体结构中，每个负离子的电荷数近似等于或等于从其邻近正离子分配给该负离子的静电键强度之和。**

静电键强度定义：
$$s = \frac{Z_+}{CN_+}$$

其中 $Z_+$ 为正离子电荷数，$CN_+$ 为正离子配位数。

**电价规则表达式**：
$$Z_- = \sum_i s_i = \sum_i \frac{Z_{+,i}}{CN_{+,i}}$$

**示例**：
- NaCl 中：Na⁺ 配位数为6，$s = 1/6$。每个 Cl⁻ 与6个 Na⁺ 配位，总静电键强度 = $6 \times 1/6 = 1$，等于 Cl⁻ 电荷数。
- CaF₂ 中：Ca²⁺ 配位数为8，$s = 2/8 = 1/4$。每个 F⁻ 与4个 Ca²⁺ 配位，总静电键强度 = $4 \times 1/4 = 1$，等于 F⁻ 电荷数。

### 3.3 Pauling第三规则（配位多面体连接规则）

**配位多面体之间倾向于共顶点连接，其次共棱，共面最不稳定。**

| 连接方式 | 稳定性 | 原因 |
|:---|:---|:---|
| 共顶点 | **最稳定** | 正离子之间距离最大，排斥最小 |
| 共棱 | 中等 | 正离子距离较近，排斥较大 |
| 共面 | **最不稳定** | 正离子距离最近，排斥最强 |

### 3.4 Pauling第四、五规则（扩展规则）

**第四规则**：若晶体中存在多种正离子，高电价、低配位数的正离子配位多面体之间倾向于**互不连接**（彼此隔离）。

**第五规则**：晶体中本质不同的组分种类数趋于最少（即晶体倾向于高对称性）。

## 四、关键结论

1. **电价规则是Pauling规则的核心**，用于验证或预测离子晶体结构的合理性
2. **共顶点 > 共棱 > 共面** 的稳定性顺序源于正离子之间的静电排斥
3. 高电价正离子（如Si⁴⁺）倾向于低配位数（4，四面体），且彼此隔离（如硅酸盐中SiO₄四面体只共顶点）
4. 电价规则可用于预测复杂氧化物中氧离子的配位环境是否饱和

## 五、常见分类或情形

### 按规则应用分类

| 规则 | 应用场景 | 典型例子 |
|:---|:---|:---|
| 第一规则 | 预测配位多面体类型 | NaCl（八面体）、ZnS（四面体） |
| 第二规则 | 验证结构稳定性 | CaF₂、TiO₂（金红石） |
| 第三规则 | 解释配位多面体连接方式 | 硅酸盐中SiO₄只共顶点 |
| 第四规则 | 解释多元离子晶体结构 | 尖晶石 AB₂O₄ 中 A、B 占位 |

## 六、适用条件与限制

1. **适用于典型离子晶体**：共价成分过强时（如AgI），规则偏差较大
2. **电价规则为近似**：不要求严格等式，接近即可
3. **不适用于非整数价态**：如混合价态化合物需要修正
4. **第五规则有例外**：某些功能材料故意打破对称性以获得特殊性质

## 七、常见比较与易混点

- **Pauling规则 vs 半径比规则**：第一规则依赖半径比，但Pauling规则整体框架更丰富（含电价、连接方式）
- **电价规则计算时混淆 $Z_+$ 和 $CN_+$**：注意静电键强度 $s = Z_+/CN_+$，不是 $CN_+/Z_+$

## 十、例题

### 例题1：用Pauling第二规则验证CaF₂结构

**题目**：验证萤石（CaF₂）结构是否满足Pauling电价规则。

**解答**：
- Ca²⁺ 配位数 CN = 8，静电键强度 $s = 2/8 = 1/4$
- 每个 F⁻ 与4个 Ca²⁺ 配位（CaF₂中F⁻位于四面体空隙）
- F⁻ 获得的总静电键强度 = $4 \times 1/4 = 1$，恰好等于 F⁻ 的电荷数
- **满足Pauling第二规则**

### 例题2：预测SiO₂中Si的配位数

**题目**：已知Si⁴⁺半径约40 pm，O²⁻半径约140 pm，预测SiO₂中Si的配位数。

**解答**：
- $r_+/r_- = 40/140 \approx 0.286$
- 查半径比规则：0.225–0.414 对应配位数4（四面体）
- 因此Si⁴⁺形成SiO₄四面体
- 高电价（+4）和低配位数（4）使SiO₄四面体倾向于只共顶点连接（第三规则）

## 十一、易错点

- 混淆静电键强度公式：$s = Z_+/CN_+$（不是 $CN/Z$）
- 忽略电价规则是验证规则，不是严格等式
- 忘记第三规则的稳定性顺序：共顶点 > 共棱 > 共面
