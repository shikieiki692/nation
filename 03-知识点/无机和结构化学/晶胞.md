# 晶胞

## 一、定义
**晶胞**：能反映晶体对称性的最小重复单元。

- **晶胞参数**：a, b, c（边长）; α, β, γ（夹角）
- 七大晶系：立方、四方、正交、六方、三方、单斜、三斜
- 14 种 Bravais 格子：P(简单)、I(体心)、F(面心)、C(底心)

## 二、关键计算
- 晶胞中原子数 = 顶点×1/8 + 棱上×1/4 + 面上×1/2 + 体内×1
- 密度公式：$\rho = \dfrac{nM}{N_A \cdot V_\text{晶胞}}$

### 晶胞密度/空隙计算通用流程（周坤无机新课晶体结构习题课 · 资产 B4-19 · 可信度：高）

> 来源：[[教学逻辑提炼-周坤无机新课-晶体配合物与气体-第一轮]]

**通用计算步骤：**

1. **确定晶胞化学式**：根据原子坐标和占有率计算晶胞中各原子的实际数目
   - 顶点：× 1/8
   - 棱上：× 1/4
   - 面上：× 1/2
   - 体内：× 1

2. **计算晶胞质量**：$m = \dfrac{Z \cdot M}{N_A}$
   - $Z$：晶胞中化学式单位数
   - $M$：摩尔质量 (g/mol)
   - $N_A$：Avogadro 常数

3. **计算晶胞体积**：
   - 立方晶系：$V = a^3$
   - 六方晶系：$V = a^2 c \sin 120° = \dfrac{\sqrt{3}}{2}a^2 c$

4. **计算密度**：$\rho = \dfrac{m}{V} = \dfrac{Z \cdot M}{N_A \cdot V}$

**空隙占有率计算**：
- 先确定堆积方式 → 查空隙总数 → 根据填隙率计算被占据空隙比例

## 三、核心原理

### 晶胞的定义
晶胞（unit cell）是晶体结构中最小的平行六面体重复单元。晶胞在三维空间中通过平移可以**无间隙地堆砌**形成整个晶体。晶胞包含两个要素：
1. **形式**：晶格单位（平行六面体）的大小和形状
2. **内容**：对应于晶格结点的结构单元（原子、离子或分子）

### 晶胞参数的完整定义（表 13.1）
晶胞由 6 个参数完全确定：
- 3 个边长：$a, b, c$
- 3 个夹角：$\alpha = b \wedge c$（b、c 间夹角），$\beta = c \wedge a$（c、a 间夹角），$\gamma = a \wedge b$（a、b 间夹角）

### 晶胞中原子的归属
| 位置 | 占有率 | 说明 |
|------|:---:|------|
| 顶点 | 1/8 | 被 8 个相邻晶胞共享 |
| 棱上 | 1/4 | 被 4 个相邻晶胞共享 |
| 面上 | 1/2 | 被 2 个相邻晶胞共享 |
| 体内 | 1 | 独属于该晶胞 |

密度计算：$\rho = \dfrac{nM}{N_A \cdot V_{\text{晶胞}}}$，其中 $n$ 为一个晶胞中化学式单位数，$M$ 为摩尔质量。

### 7 种晶系概览（表 13.1）
| 晶系 | 晶胞参数 | 实例 |
|------|------|------|
| 立方 | $a=b=c,\ \alpha=\beta=\gamma=90^\circ$ | Cu, NaCl |
| 六方 | $a=b\neq c,\ \alpha=\beta=90^\circ,\ \gamma=120^\circ$ | Mg, AgI |
| 四方 | $a=b\neq c,\ \alpha=\beta=\gamma=90^\circ$ | $\beta$-Sn, SnO₂ |
| 三方 | 简单三方或菱面体格子 | $\alpha$-SiO₂, Bi |
| 正交 | $a\neq b\neq c,\ \alpha=\beta=\gamma=90^\circ$ | I₂, HgCl₂ |
| 单斜 | $a\neq b\neq c,\ \alpha=\gamma=90^\circ$ | S, KClO₃ |
| 三斜 | $a\neq b\neq c,\ \alpha\neq\beta\neq\gamma\neq90^\circ$ | CuSO₄·5H₂O |

## 四、关键结论
- 晶体结构 = 点阵（几何框架）+ 结构基元（原子/分子/离子在结点上的内容）
- 立方晶系中，$a=b=c$ 且 $\alpha=\beta=\gamma=90^\circ$ 是**必要但不充分**条件——必须有沿体对角线方向的 4 个三重轴
- **简单立方**：$a=2r$，含 1 个球，空间利用率 52%
- **体心立方**：$r=\sqrt{3}a/4$，含 2 个球，空间利用率 68%
- **面心立方**：$a=2\sqrt{2}r$，含 4 个球，空间利用率 74%

## 五、常见分类或情形
- 立方晶系：3 种 Bravais 点阵（P/简单、I/体心、F/面心）
- 正交晶系：4 种 Bravais 点阵（P、I、F、C/底心）
- 四方和单斜：各 2 种点阵型式
- 六方和三斜：各 1 种
- 三方：R 心（菱面体）或简单三方
- 共 **14 种 Bravais 空间点阵型式**（Bravais, 1866）

## 六、适用条件与限制
- 晶胞选取方案不唯一——通常选**对称性最高、体积最小**的格子
- 三方晶系的菱面体格子和 R 心六方格子等价，可以互相转换
- 晶胞参数数量级：原子间距约 $10^{-10}\,\mathrm{m}$（A），可用 Bragg 方程 $2d\sin\theta=n\lambda$ 通过 X 射线衍射测定

## 七、常见比较与易混点
- **晶格（lattice）vs 晶胞（unit cell）**：晶格是抽象的几何点阵，晶胞包含具体结构内容
- **CsCl 型不是 bcc**：CsCl 中 Cl⁻ 在顶点、Cs⁺ 在体心，两者是不同原子，属于简单立方点阵而非体心立方（体心立方要求顶点和体心是同类原子）
- **立方晶系必须验证三重轴**：不能仅凭 $a=b=c$ 和 $90^\circ$ 夹角判定

## 九、典型题型

1. **题型-晶胞原子计数**：根据顶点/棱/面/体内的原子归属规则计算晶胞化学式——最基础也最容易因"对称性判断失误"翻车
2. **题型-正当晶胞判断**：判断给定晶胞是素晶胞还是复晶胞（体心/面心/底心），关键验证"平移后环境是否完全等同"——如 CsCl 看似体心实则简单立方
3. **题型-晶胞密度计算**：$\rho = ZM/(N_A \cdot V)$，结合晶胞参数 $a$ 和堆积方式 $r$-$a$ 关系，是竞赛计算题主力
4. **题型-晶系与对称性归属**：从晶胞参数（$a,b,c,\alpha,\beta,\gamma$）和对称元素判断七大晶系——如 CaC₂ 初看面心立方，实为四方（C₂²⁻ 哑铃取向破坏了四个三重轴）
5. **题型-原子坐标与占有率综合**：给定原子坐标 → 判断多面体连接方式 + 验证化学式 + 计算占有率

## 十、例题

### 例题 1（基础 · 金刚石晶胞碳原子计数与坐标验证）

**题目**：金刚石晶胞中，碳原子坐标分别为 $(0,0,0)$、$(1/2,1/2,0)$、$(1/2,0,1/2)$、$(0,1/2,1/2)$、$(3/4,1/4,1/4)$、$(1/4,3/4,1/4)$、$(1/4,1/4,3/4)$、$(3/4,3/4,3/4)$。
（1）判断每个坐标位置上碳原子的归属份额（顶点/面心/体内）。
（2）计算一个金刚石晶胞中含几个碳原子。
（3）已知金刚石晶胞参数 $a = 356.7\text{ pm}$，C 的相对原子质量为 12.01，求金刚石的理论密度。

**分析**：前四组坐标（全为 0 或 1/2）是 fcc 格点位置（顶点+面心），后四组坐标（分母含 4）是四面体空隙中的碳原子。

**解答**：
（1）$(0,0,0)$ — 顶点（8 个顶点之一 → 1/8）；$(1/2,1/2,0)$、$(1/2,0,1/2)$、$(0,1/2,1/2)$ — 面心（各 1/2）；$(3/4,1/4,1/4)$ 等四组 — 体内（各 1）。

（2）顶点：$8 \times 1/8 = 1$，面心：$6 \times 1/2 = 3$，体内：$4 \times 1 = 4$。合计 **8 个碳原子**。

（3）$Z = 8$（晶胞含 8 个 C），$V = a^3 = (356.7 \times 10^{-12}\text{ m})^3 = 4.54 \times 10^{-29}\text{ m}^3$，
$\rho = \dfrac{8 \times 12.01\text{ g/mol}}{6.022 \times 10^{23}\text{ mol}^{-1} \times 4.54 \times 10^{-29}\text{ m}^3} = 3.52\text{ g/cm}^3$。

**反思**：金刚石结构 = fcc + 半填四面体空隙。关键——不是"面心立方中的碳原子"都是面心归属，后四组坐标是**体内**的（不属于顶点/棱/面任何边界）。

### 例题 2（基础 · Cu₂O 体心立方判断）

**题目**：赤铜矿 Cu₂O 的晶胞中，O 原子坐标为 $(0,0,0)$ 和 $(1/2,1/2,1/2)$，Cu 原子坐标为 $(3/4,1/4,1/4)$、$(1/4,3/4,1/4)$、$(1/4,1/4,3/4)$、$(3/4,3/4,3/4)$。判断该晶胞是不是体心立方晶胞。

**分析**：体心晶胞的关键判据——做体心平移 $(+1/2,+1/2,+1/2)$ 后，新晶胞与旧晶胞**完全等同**。注意必须看原子及其**周围环境**，不是仅看坐标。

**解答**：将各原子坐标做 $(+1/2,+1/2,+1/2)$：
- O：$(0,0,0) \to (1/2,1/2,1/2)$（O → O ✓），$(1/2,1/2,1/2) \to (0,0,0)$（O → O ✓）
- Cu：$(3/4,1/4,1/4) \to (1/4,3/4,3/4)$ ——但原晶胞中 $(1/4,3/4,3/4)$ 周围 Cu 原子排布与平移前 $(3/4,1/4,1/4)$ 周围的**取向不同**！

原因是体心位置的 O 原子周围 4 个 Cu 的四面体取向与顶点 O 原子不同（一个朝上，一个朝下）。因此 **Cu₂O 不是体心立方晶胞**，是简单立方（P）。

**反思**：CsCl 和 Cu₂O 是两大道判断"表面体心实际不是"的经典反例。CsCl：顶点 Cl⁻ + 体心 Cs⁺ — 不同离子，平移后不能重合。Cu₂O：坐标平移后取向不同。

### 例题 3（提高 · CaC₂ 正当晶胞判断——从"面心立方"到"体心四方"）

**题目**：（第 24 届江苏赛区选拔赛）CaC₂ 的晶体结构中，Ca²⁺ 占据 fcc 格点，C₂²⁻ 占据八面体空隙。C—C 键长 120 pm，C₂²⁻ 呈哑铃型，其键轴与 c 轴平行。问：该晶胞是正当晶胞吗？若不是，正当晶胞是什么？

**分析**：初看 CaC₂ = NaCl 型（Ca²⁺ 替 Na⁺，C₂²⁻ 替 Cl⁻）→ 面心立方。但 C₂²⁻ 是**哑铃型**（非球形），取向破坏了立方对称性。

**解答**：
（1）不是正当晶胞。因为哑铃型 C₂²⁻ 沿 c 轴取向 → c 轴变长 → a ≠ c → **4 个三重轴（$C_3$）被破坏** → 不再属于立方晶系。

（2）但 c 轴的 $C_4$ 轴对称性得以保留 → 降为**四方晶系**。做体心四方平移可以复原 → 正当晶胞为 **体心四方晶胞**。

**反思**：这道题是"晶胞对称性判断"的经典教学素材——晶系归属由**对称性**决定，不是由"看起来像什么"决定。分子/离子的形状（球形 vs 哑铃）能改变晶体对称性。

### 例题 4（竞赛 · 碳化硼 B₄C — 2010 年全国初赛）

**题目**：（2010 年全国初赛）碳化硼是除金刚石和氮化硼外最硬的固体。右图是碳化硼晶胞简图（含所有原子），除"三原子"（中心原子及相连的 2 个原子）外，其余原子都是 B₁₂ 多面体的 1 个原子（B₁₂ 的其他原子不在晶胞内）。

（1）若"三原子"都是碳原子，写出碳化硼的化学式。
（2）该结构有什么旋转轴？有无对称中心和镜面？
（3）该晶胞属于布拉维系七种晶胞中的哪一种？

**分析**：晶胞中"三原子" = 3 个 C（全在体内）。每个位于晶胞内部的 B 原子代表 B₁₂ 多面体中的一个顶点——"该图给出了晶胞中的所有原子" → 每个 B 计入 1（不按边界归属）。总计 12 个 B + 3 个 C → B₁₂C₃ → 最简整数比 B₄C。

**解答**：
（1）晶胞含 3 个 C 原子（"三原子"全在体内）和 12 个 B 原子（B₁₂ 多面体各给 1 个位于晶胞内的原子）→ 化学式 **B₄C**。

（注：若"三原子"的中心原子为 B，则得 B₁₃C₂。）

（2）对称元素：1 个 $C_3$ 旋转轴（通过"三原子"的直线）、对称中心 $i$（在晶胞中心）、3 个镜面 $\sigma$（均通过 $C_3$ 轴，面间角 60°）、3 个 $C_2$ 轴（通过 $i$，与 $C_3$ 轴垂直，位于镜面角平分面上）。属 **$D_{3d}$ 点群**（与交叉式乙烷分子相同）。

（3）最高对称轴为 $C_3$ → **三方晶系** → 晶胞为菱方或菱面体。

**反思**：这道初赛题同时考查了——晶胞计数（含"非完整多面体"的归属判断）、对称元素分析（点群归属）、晶系分类——是晶体结构章节最综合的真题之一。B₁₂ 二十面体是硼化学的标志性结构单元。

## 十一、易错点

| 易错点 | 学生为什么会这么想 | 正确认识 | 口诀 |
|:---|:---|:---|:---|
| "顶点原子属于这个晶胞" | 看到顶点上有原子 → 直觉上"这个晶胞包含它" | 顶点原子被 **8 个**相邻晶胞共用，每个晶胞只占 1/8。同理棱上 1/4、面上 1/2、体内 1。这是"晶胞原子计数"最基础的规则，但初学者几乎必错 | "顶点八分一，棱边四分一，面对半，体内归自己" |
| "CsCl 是体心立方" | 顶点+体心都有离子 → 拍照式记忆为"体心" | CsCl 是**简单立方**（P）而非体心立方（I）——因为顶点是 Cl⁻、体心是 Cs⁺，**不是同类原子**，体心平移后不能重合。体心立方的核心判据是：顶点和体心必须是**环境完全等同**的原子 | "铯氯虽在心和角——离子不同不是体心" |
| "$a=b=c$ 且 $\alpha=\beta=\gamma=90^\circ$ 就是立方晶系" | 把晶胞参数等同于晶系判据 | 立方晶系的**充要条件**是：$a=b=c$、$\alpha=\beta=\gamma=90^\circ$ **且存在沿体对角线方向的 4 个三重轴（$C_3$）**。CaC₂ 的 $a=b=c$ 看似成立，但 C₂²⁻ 哑铃取向破坏了 $C_3$ → 实为四方晶系 | "立方不只看三边等——还要四根三重轴" |
| "晶胞选取是唯一的" | 课本上每个晶体只画了一种晶胞 | 正当晶胞有选取标准：①对称性尽可能高 ②直角尽可能多 ③体积尽可能小。满足这些的仍可能不唯一（如三方晶系的菱面体和 R 心六方格子等价）。另外素晶胞×2/×4 可得复晶胞 | "晶胞选法不唯一——看谁对称高体积小" |
| "六方棱柱可以当晶胞" | 六方晶系的晶体画成六方棱柱很直观 | 晶胞必须是**平行六面体**——六方棱柱不是平行六面体，不能无隙并置堆砌整个空间。六方晶胞的正确形状是 $a=b\neq c, \alpha=\beta=90^\circ, \gamma=120^\circ$ 的平行六面体 | "六方柱子不是晶胞——平行六面体才能堆" |
| "密度公式 $\rho=nM/(N_A V)$ 中 $n$ 就是晶胞原子总数" | 把"晶胞原子数"和"化学式单位数 $Z$"混淆 | $n$ 是晶胞中**化学式单位数** $Z$，不是原子总数。如 NaCl 晶胞有 4 个 Na⁺ + 4 个 Cl⁻，$Z=4$（NaCl 单位），不是 8。金刚石晶胞 8 个 C，$Z=8$（因为是原子晶体，单位就是 C） | "$n$不是原子总数——是化学式有几个" |

## 十二、🎯 教学视角

### 12.1 学生典型认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 教学干预 |
|:---|:---|:---|:---|
| "晶体结构就是背几种典型结构" | 课本把 CsCl/NaCl/ZnS/CaF₂/钙钛矿 当"清单"教 | 晶体结构的核心不是记结构类型，而是**理解"堆积+填隙"的思维框架**——谁堆积（较大离子）、堆成什么方式（fcc/hcp/sc）、填哪种空隙（四面体/八面体/立方体）、填多少。掌握这套框架 > 背 20 种结构 | 第一节课就画"堆积方式→空隙类型→填隙率→化学式"的推导链——"你会推导就永远不用背" |
| "只要会算 $\rho=ZM/N_AV$ 就掌握了晶胞" | 把晶胞章节简化为一道密度计算题 | 竞赛区分度在于——对称性判据（CaC₂ 立方→四方）、正当晶胞选择（为什么没有四方面心？）、多面体连接与计数综合（B₁₂ 二十面体/LaFe₄Sb₁₂ 等非简单结构）。纯密度计算只是**及格线** | 给一道 CaC₂ 题——"算出来是面心立方？那你被坑了——再看看对称性" |

### 12.2 讲授主线

（整合自 [[教学逻辑提炼-周坤无机新课-晶体配合物与气体-第一轮]] 与上海中学竞赛教程 Vol 1 第六~八讲）

晶体结构的教学应按"三步走 + 一个核心思维"：

**第一步：晶胞基本语言（坐标→归属→计数）**
- 顶点(0,0,0) → ×1/8；棱心(1/2,0,0) → ×1/4；面心(0,1/2,1/2) → ×1/2；体内 → ×1
- 关键训练：金刚石晶胞 8 组坐标 → 逐一判断归属 → 验证总计 8 个 C

**第二步：对称性与正当晶胞（素/复晶胞判断）**
- 体心晶胞判据：顶点+ (1/2,1/2,1/2) → 环境完全等同？
- 面心晶胞判据：顶点+ (1/2,1/2,0) 等 → 环境完全等同？
- 核心反例：CsCl（不是 I，是 P）、干冰（不是 F，分子取向不同）、Cu₂O（不是 I，四面体取向不同）
- 为什么没有四方底心和四方面心点阵型式？→ 可由更小/更对称的格子代替

**第三步：密度计算与综合应用**
- 通用流程（周坤资产 B4-19）：确定化学式 → $m = ZM/N_A$ → $V$（立方 $a^3$ / 六方 $\frac{\sqrt{3}}{2}a^2c$）→ $\rho = m/V$
- 反向：已知 $\rho$ + 晶胞参数 → 求原子量 → 推断元素

**一个核心思维：堆积+填隙的统一框架**
- 谁堆积（通常较大的负离子/anion）
- 堆成什么方式（sc/fcc/hcp）
- 填哪种空隙（八面体/四面体/立方体）
- 填多少（填隙率：全部/一半/其他）

### 12.3 与现实/直觉的连接

- **X 射线衍射——从"看到晶体"到"算出原子位置"**：Bragg 方程 $2d\sin\theta = n\lambda$ 是连接宏观测量（衍射角 $\theta$）和微观结构（晶面间距 $d$）的桥梁。每种晶体都有独特的"衍射指纹"，这就是为什么能通过 XRD 鉴定未知物相。Rosalind Franklin 的 DNA X 射线衍射照片直接启发了 Watson 和 Crick 发现双螺旋结构。
- **药物多晶型——同一个分子，不同晶胞 = 不同疗效**：利托那韦（HIV 药物）在 1998 年因出现一种更稳定的新晶型（溶解度低 50%）导致药物失效，被迫撤市重新研发——损失超 2.5 亿美元。晶胞结构直接决定药物的生物利用度。
- **宝石的颜色——杂质进晶胞**：红宝石和蓝宝石都是 $\alpha$-Al₂O₃（刚玉），红宝石的红色来自 Cr³⁺ 取代了晶胞中部分 Al³⁺（~1%），蓝宝石的蓝色来自 Fe²⁺/Ti⁴⁺ 共取代。**杂质进入晶胞 → 改变电子能级 → 改变颜色**——这是晶体结构与材料性质的完美连接。

## 十四、竞赛拓展

1. **非整比化合物与晶胞缺陷化学**：实际晶体中存在 Schottky 缺陷（正负离子空位成对）、Frenkel 缺陷（离子位移到间隙位）、以及非化学计量比（如 Fe₀.₉₅O 实为 Fe²⁺/Fe³⁺ 混合价态补偿 O 不足）。晶胞密度计算中，缺陷会导致实测密度偏离理论值。竞赛可考查"给定实测密度，判断缺陷类型及浓度"。

2. **准晶（Quasicrystal）——五次对称性与传统晶胞的突破**：2011 年诺贝尔化学奖得主 Shechtman 发现的准晶具有传统晶体学"禁止"的五次对称性——不能用单一晶胞周期性重复来描述。准晶需要至少两个不同菱面体按 Fibonacci 序列排列。这拓展了"什么是晶体"的定义——国际晶体学联合会 1992 年将晶体定义从"周期性的"改为"能给出明确衍射图的"。

3. **粉末 XRD 指标化与晶胞参数精修**：在多相混合物中，通过粉末 XRD 谱图的峰位置（$2\theta$）→ 计算 $d$ 值 → 指标化（给每个峰分配 $(hkl)$）→ 最小二乘精修得到精确晶胞参数。这在材料化学中（如电池正极材料的相鉴定、钙钛矿太阳能电池的晶格应变分析）是常规操作。竞赛中出现的 XRD 题通常是信息给予题——给出衍射峰位置和强度，要求指标化或判断晶系。
