# 溶剂类型

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[11-分子间作用力]]

## 一、定义
**溶剂类型（Solvent Types）**：按溶剂分子的结构、极性和氢键能力对溶剂进行的分类，是选择合适反应介质的依据。

最常见的三类划分：
1. **极性质子性溶剂（Polar protic）**：高极性 + 能给 H 键（含 O-H 或 N-H）。例：水、醇、酸。
2. **极性非质子性溶剂（Polar aprotic）**：高极性，但不能给 H 键（无 O-H/N-H）。例：DMSO、DMF、乙腈。
3. **非极性溶剂（Nonpolar）**：低极性，分子近似对称。例：苯、己烷、CCl₄。

## 三、核心原理

### 三类溶剂的核心特征

**1. 极性质子性 (Polar Protic)**
| 特征 | 表现 |
|---|---|
| $\varepsilon$（介电常数） | 高（> 15） |
| $\mu$（偶极矩） | 中-高 |
| 含 O-H 或 N-H | 是 |
| 可作 H 键给体 | 是 |
| 可作 H 键受体 | 是 |
| 例子 | H₂O, MeOH, EtOH, HCOOH, NH₃ |

**2. 极性非质子性 (Polar Aprotic)**
| 特征 | 表现 |
|---|---|
| $\varepsilon$ | 高（> 15） |
| $\mu$ | 高（> 3 D） |
| 含 O-H 或 N-H | 否 |
| 可作 H 键给体 | 否 |
| 可作 H 键受体 | 是（通过 O, N 上孤对） |
| 例子 | DMF, DMSO, MeCN, 丙酮, THF |

**3. 非极性 (Nonpolar)**
| 特征 | 表现 |
|---|---|
| $\varepsilon$ | 低（< 5） |
| $\mu$ | 低（< 1 D 或 0） |
| 含 O-H/N-H | 否 |
| H 键能力 | 无 |
| 例子 | 苯, 甲苯, 己烷, CCl₄, 醚 |

### 溶剂效应（核心应用）

**SN1 反应**：碳正离子中间体需稳定 → 用**极性质子性**溶剂（如 MeOH, H₂O）
- 极性 + H 键 + 高 $\varepsilon$ → 稳定 C⁺ 和 X⁻
- 反应速率：水 > 甲醇 > 乙醇 > 异丙醇

**SN2 反应**：亲核试剂需高活性 → 用**极性非质子性**溶剂（如 DMF, DMSO）
- 高 $\varepsilon$ → 稳定离子；但不形成 H 键 → 不"包裹"亲核试剂
- 反应速率：DMSO > DMF > MeCN > 丙酮

**E1 vs E2 消除反应**：与 SN1/SN2 类似规律。

**Diels-Alder 反应**：低极性溶剂或纯净物中进行（如苯、二氯甲烷）。

### 溶剂对反应速率的影响（数量级）
| 反应类型 | 极性质子性 | 极性非质子性 | 非极性 |
|---|:---:|:---:|:---:|
| SN1 | **快**（10⁵ 以上）| 慢 | 极慢 |
| SN2 | 慢 | **快**（10⁴ 倍）| 极慢 |
| E1 | 快 | 慢 | 极慢 |
| Friedel-Crafts | 慢 | 中 | 快 |
| Grignard | — | 中 | **必须**（醚溶剂）|

## 四、关键结论

### 极性质子性溶剂的特点
- 对**阴离子有强溶剂化**：H 键包裹 → 阴离子稳定但活性下降
- 对**阳离子有溶剂化**：偶极静电
- 用于：SN1、E1、酸碱反应、电化学
- 实例：水、甲醇、乙醇、乙酸、HF

### 极性非质子性溶剂的特点
- 对阴离子**弱溶剂化**：无 H 键能力 → 阴离子"裸露"，活性高
- 对阳离子有溶剂化（高介电）
- 用于：SN2、E2、有机金属反应、亲核加成
- 实例：DMF, DMSO, MeCN, 丙酮, HMPA

### 非极性溶剂的特点
- 几乎不溶剂化离子 → 离子化合物不溶
- 适合：非极性溶质，Diels-Alder 反应
- 实例：苯、甲苯、己烷、CCl₄、乙醚

### "经验法则"汇总
| 溶质类型 | 推荐溶剂类型 |
|---|---|
| 离子化合物 | 极性质子性（水, 醇）|
| 阴离子反应试剂（CN⁻, NaH 等） | 极性非质子性（DMF, DMSO）|
| 自由基反应 | 非极性（苯, CCl₄）|
| Grignard 试剂 | 醚类（Et₂O, THF）|
| 有机合成（一般） | 多用极性非质子性（DMF, MeCN）|
| 萃取分离 | 用相反极性（"相反相分离"）|

## 五、常见分类或情形

### 按 $\varepsilon$ 简化分类
| 类型 | $\varepsilon$ 范围 | 实例 |
|---|---|---|
| 强极性 | > 30 | H₂O, MeCN, DMSO, DMF |
| 中极性 | 10-30 | EtOH, 异丙醇, 丙酮 |
| 弱极性 | 5-10 | 二氯甲烷, THF, CHCl₃ |
| 非极性 | < 5 | 苯, 己烷, 醚 |

### 按"四象限"分类（极性 + 质子性）
```
                  极性强 (ε > 15)
                       |
极性质子性 (H 键 +) — | — 极性非质子性 (H 键 -)
   水, MeOH, EtOH      |    DMSO, DMF, MeCN
   HCOOH, NH₃          |    丙酮, THF
                       |
极性弱 (ε < 15)
                       |
弱极性质子性          | — 弱极性非质子性
  乙酸 (略)            |    乙醚, 二氯甲烷
                       |
非极性
                       |
  苯, 甲苯, 己烷, CCl₄
```

### 按"绿色化学"分类
| 类型 | 优点 | 实例 |
|---|---|---|
| 绿色溶剂 | 低毒、易回收 | 水, EtOH, 超临界 CO₂ |
| 一般绿色 | 较易处理 | 丙酮, 异丙醇 |
| 红色（避免用） | 致癌、生态毒性 | 苯, CCl₄, HMPA |

### 用 Hansen 三参数定位
| 类别 | $\delta_d$ (分散) | $\delta_p$ (极性) | $\delta_h$ (H 键) |
|---|---|---|---|
| 水 | 15.5 | 16.0 | 42.3 |
| 甲醇 | 15.1 | 12.3 | 22.3 |
| DMSO | 18.4 | 16.4 | 10.2 |
| 苯 | 18.4 | 0 | 2.0 |
| 己烷 | 14.9 | 0 | 0 |

### 溶剂自电离与拉平/区分效应（来源：周坤无机新课 Batch 2）

不同溶剂具有不同的自电离（自质子化）行为，这直接影响酸碱在其中的表现。

| 溶剂 | 自电离方程式 | 自电离常数 | 备注 |
|---|---|---|---|
| 水 | $2\mathrm{H_2O} \rightleftharpoons \mathrm{H_3O^+} + \mathrm{OH^-}$ | $K_w = 10^{-14}$ (25°C) | 最常用参考 |
| 液氨 | $2\mathrm{NH_3} \rightleftharpoons \mathrm{NH_4^+} + \mathrm{NH_2^-}$ | $K \approx 10^{-29}$ (-33°C) | 强碱 $\mathrm{NH_2^-}$ 可稳定存在 |
| 冰醋酸 | $2\mathrm{HOAc} \rightleftharpoons \mathrm{H_2OAc^+} + \mathrm{OAc^-}$ | $K \approx 10^{-13}$ (25°C) | 弱碱性溶剂 |
| HF | $2\mathrm{HF} \rightleftharpoons \mathrm{H_2F^+} + \mathrm{F^-}$ | $K \approx 10^{-10}$ | 兼具酸性和配位能力 |

**拉平效应（Leveling Effect）**：在某溶剂中，所有比溶剂共轭酸更强的酸都被"拉平"到溶剂共轭酸的强度。
- 在水中：HClO₄、HCl、HNO₃ 都是强酸（被拉平到 H₃O⁺ 水平）
- 在冰醋酸中：HClO₄ 是强酸，HCl 是弱酸（可区分）

**区分效应（Differentiating Effect）**：溶剂能区分不同强酸的相对强度。
- 冰醋酸对 HClO₄、H₂SO₄、HCl、HNO₃ 有区分能力
- 液氨中，HAc 也是强酸（被拉平到 NH₄⁺ 水平）

## 六、适用条件与限制
1. **三类划分有"交界"**：异丙醇/正丁醇等位于"中等"极性
2. **分类边界不严格**：依不同教材有差异
3. **不能只看一种参数**：极性 + 质子性 + 介电常数综合判断
4. **温度依赖**：升温改变溶剂的 $\varepsilon$, $\mu$
5. **混合溶剂**：可调整溶质溶解度和反应选择性

## 七、常见比较与易混点

### 极性质子性 vs 极性非质子性
| 项目 | 极性质子性 | 极性非质子性 |
|---|---|---|
| 含 O-H/N-H | 是 | 否 |
| 给 H 键 | 是 | 否 |
| 阴离子溶剂化 | 强（包裹） | 弱（裸露）|
| 阳离子溶剂化 | 强 | 强 |
| SN1/E1 | **加快** | 减慢 |
| SN2/E2 | 减慢 | **加快** |

### 极性非质子性 vs 非极性
| 项目 | 极性非质子性 | 非极性 |
|---|---|---|
| $\varepsilon$ | 高 | 低 |
| $\mu$ | 高 | 低 |
| 溶解离子 | 是 | 否 |
| 例子 | DMSO, DMF | 苯, 己烷 |

### 三类溶剂记忆要点
- **质子性**：有 O-H 或 N-H 就行
- **极性**：看 $\varepsilon$ 或 $\mu$
- **三类**：质子性=极性+给H键；非质子性极性=极性但无给H键；非极性=都没

## 十、例题

### 例题 1：识别溶剂类型
**题目：** 判断下列溶剂的类型（极性质子性/极性非质子性/非极性）：
1. CHCl₃
2. CH₃COOH
3. (CH₃)₂NCHO (DMF)
4. CCl₄
5. C₆H₅OH (苯酚)
6. CH₃COCH₃ (丙酮)

**解答：**
1. CHCl₃ ($\varepsilon$=4.8, $\mu$=1.0)：**弱极性非质子性**
2. CH₃COOH (O-H 含)：**极性质子性**
3. DMF ($\varepsilon$=38, 无 O-H/N-H)：**极性非质子性**
4. CCl₄ ($\mu$=0, $\varepsilon$=2.2)：**非极性**
5. 苯酚 (O-H 含)：**弱极性质子性**
6. 丙酮 (无 O-H/N-H, $\varepsilon$=21)：**极性非质子性**

**反思：** 先看有无 O-H/N-H 决定质子性；再看 $\varepsilon$ 决定极性。

### 例题 2：选择 SN2 反应溶剂
**题目：** 反应 $\text{CH}_3\text{CH}_2\text{Br} + \text{CN}^- \to \text{CH}_3\text{CH}_2\text{CN} + \text{Br}^-$。比较该反应在乙醇 vs DMSO 中的速率，并解释。

**分析：** SN2 反应；亲核试剂 CN⁻。

**解答：**
- 乙醇：极性质子性溶剂。H 键包裹 CN⁻ → CN⁻ 活性下降。SN2 速率慢。
- DMSO：极性非质子性。高 $\varepsilon$ 稳定 Br⁻；不形成 H 键 → CN⁻ "裸露"高活性。SN2 速率快。
- 实测：DMSO 中速率比乙醇中**快 10⁴~10⁶ 倍**

**反思：** SN2 用极性非质子性溶剂是经典策略。

## 十一、易错点
- ❌ 错：所有极性溶剂都是质子性——DMSO/DMF/MeCN 都是极性**非质子性**
- ❌ 错：非极性溶剂都没用——非极性是 Diels-Alder/萃取的重要选择
- ❌ 错：乙醚是极性溶剂——乙醚 $\varepsilon = 4.3$，**非极性或极弱极性**
- ❌ 错：CHCl₃ 是非极性——$\mu = 1.04$ D, $\varepsilon = 4.8$，弱极性非质子性
- ❌ 错：质子性 = 酸性——质子性指能给 H 键，未必酸性强（如水）

### 12.3 入门级例题
**题目**：写出溶解 NaOH 的最佳溶剂（极性质子性/极性非质子性/非极性）并解释。

**预期解答路径**：
1. NaOH 是离子化合物，需要高 $\varepsilon$ 溶剂
2. 极性质子性如水、甲醇：$\varepsilon$ 高 + H 键稳定 OH⁻ → 易溶
3. 极性非质子性如 DMSO：$\varepsilon$ 高但不 H 键 → 部分溶
4. 非极性如苯：不溶
5. 最佳：**水**（极性质子性）

**老师可在课堂上引导的提问**：
- 学生若卡住：问"NaOH 是离子还是分子？"
- 学生若答错：从离子-偶极相互作用入手讲

### 12.4 与现实/直觉的连接
- 类比：质子性溶剂像"湿润布料"，能"擦"住带电物；非质子性极性溶剂像"光滑湿玻璃"，能容纳离子但不抓住
- 反例：在汽油（非极性）中放盐，盐就是不溶
