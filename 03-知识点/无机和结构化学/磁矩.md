# 磁矩（Magnetic Moment）

## 一、定义

磁矩是判断配合物电子构型和自旋态的**关键实验量**。通过测量磁矩（常用古埃磁天平），可以推断中心金属的未成对电子数，进而确定配体场强弱和配合物的几何构型。

## 三、核心原理

### 理论公式

#### 纯自旋磁矩（Spin-only Formula）
$$\mu_{\text{so}} = \sqrt{n(n+2)}\ \mu_B$$
其中 $n$ 为未成对电子数，$\mu_B$ 为玻尔磁子。

| 未成对电子数 $n$ | $\mu_{\text{so}}$ ($\mu_B$) |
|:---:|:---:|
| 0 | 0.00 |
| 1 | 1.73 |
| 2 | 2.83 |
| 3 | 3.87 |
| 4 | 4.90 |
| 5 | 5.92 |

#### 考虑轨道贡献的磁矩
对于第一过渡系，轨道贡献通常被猝灭（quenched），纯自旋公式已足够准确。

对于第二、三过渡系和镧系：
$$\mu_{\text{eff}} = \sqrt{4S(S+1) + L(L+1)}\ \mu_B$$

#### 温度依赖
- **居里定律**：$\chi = \frac{C}{T}$，$\chi$ 为磁化率
- **居里-外斯定律**：$\chi = \frac{C}{T - \theta}$

## 五、常见分类或情形

### 磁性的分类

| 类型    | 来源     | 磁化率 $\chi$ | 温度依赖               |
| :---- | :----- | :--------: | :----------------- |
| 反磁性  | 电子成对   |    负，小    | 与温度无关             |
| 顺磁性  | 未成对电子  |    正，小    | $\chi \propto 1/T$ |
| 铁磁性  | 磁矩有序排列 |    正，极大    | 居里温度以上变为顺磁         |
| 反铁磁性 | 反平行排布  |    正，小    | 奈尔温度以上变为顺磁         |

## 十一、易错点

1. **纯自旋公式的适用范围**：第一过渡系配合物通常适用；第二、三过渡系和镧系需考虑轨道贡献
2. **$d^4$-$d^7$ 的配置是关键**：这些电子数的高/低自旋差异最大
3. **磁矩为0不一定是低自旋**：$d^0$、$d^{10}$、低自旋 $d^6$ 都表现为反磁性
4. **温度对磁矩的影响**：测量磁矩时必须标注温度，高温有利于高自旋

## 十三、竞赛拓展

### 磁矩与配合物结构

#### 高自旋 vs 低自旋
- **强场配体**（$CN^-$, $CO$, $NO_2^-$）：$\Delta_o$ 大，低自旋
- **弱场配体**（$H_2O$, $Cl^-$, $F^-$）：$\Delta_o$ 小，高自旋
- 磁矩测量是区分二者的**最直接实验手段**

#### 典型例子
- $[Fe(H_2O)_6]^{2+}$：高自旋 $d^6$，$n=4$，$\mu_{\text{so}} = 4.90\ \mu_B$
- $[Fe(CN)_6]^{4-}$：低自旋 $d^6$，$n=0$，$\mu_{\text{so}} = 0$
- $[CoF_6]^{3-}$：高自旋 $d^6$，$n=4$，顺磁性
- $[Co(NH_3)_6]^{3+}$：低自旋 $d^6$，$n=0$，反磁性

### 决赛考查要点

1. **从磁矩推断未成对电子数**：熟练使用纯自旋公式
2. **判断高自旋低自旋**：结合配体场强弱和磁矩数量
3. **Jahn-Teller 效应与磁矩**：$d^9$（如 $Cu^{2+}$）的畸变与磁性
4. **磁矩异常**
   - 实测值 > 纯自旋值：可能有轨道贡献
   - 实测值 < 纯自旋值：可能存在金属-金属相互作用或反铁磁耦合
