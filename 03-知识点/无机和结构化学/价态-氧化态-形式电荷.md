# 价态-氧化态-形式电荷

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[10-分子结构与化学键]]、[[13-元素化学]]

## 一、定义

### 三个核心概念
| 概念 | 英文 | 定义 | 计算方法 |
|---|---|---|---|
| **价（数）** | Valence | 一个原子参与的**化学键数 + 形式电荷的绝对值** | $\text{V} = (\text{键数}) + |\text{FC}|$（特殊算法）|
| **氧化态** | Oxidation State | 假设所有共价键全部异裂、电子归于电负性更大的原子时该原子的电荷 | 按电负性归属电子对 |
| **形式电荷** | Formal Charge | 假设所有共价键均裂时该原子的电荷 | $\text{FC} = (\text{价电子数}) - (\text{孤对电子数}) - \frac{1}{2}(\text{成键电子数})$ |

> **本书核心强调**：这三个概念**完全不同**，必须分别使用——尤其在用箭头推动法时，**形式电荷**才是关键。

## 三、核心原理

### 3.1 三者的根本区别
| 特性 | 价 | 氧化态 | 形式电荷 |
|---|---|---|---|
| 关注 | 物理键数 | 电子归属（按电负性） | 电子分配（均匀） |
| 与电负性 | 无关 | 强相关 | 无关 |
| 单原子 | 0（如 Ar） | 0 | 0 |
| 单质 | 视分子 | 0 | 0 |
| 用途 | 命名（λ）、键合分析 | 氧化还原配平 | 箭头推动起点判断 |

### 3.2 用 NH₄⁺ 作典型例子
| 概念 | N | H |
|---|---|---|
| 价 | **5**（4 个 σ 键 + 1 个 N 上的 + 形式电荷）| 1 |
| 氧化态 | **−3**（H 给电子归 N，4 个 H 各给 1 个，但 N 失 1 个：4 − 1 = 3，反号 −3）| +1 |
| 形式电荷 | **+1**（5 价电子 − 0 孤对 − 4 共用 = +1）| 0 |

### 3.3 用 H₃O⁺ 作典型例子
| 概念 | O | H |
|---|---|---|
| 价 | **4**（3 个 σ 键 + 1 个 O 上的 + 形式电荷）| 1 |
| 氧化态 | **−2**（同水中）| +1 |
| 形式电荷 | **+1** | 0 |

> **关键洞察**：H₃O⁺ 中 O 的"价"达到 4，但氧化态仍是 −2——它们不是同一回事。

### 3.4 价的计算公式（书内强调）
$$\boxed{\text{价} = (\text{键数}) + |\text{形式电荷}|}$$
但要更精确地表述：
- 若形式电荷为正（缺电子），加上 |FC|
- 若形式电荷为负（多电子），减去 |FC|
- 例：$\ce{NO_3^-}$ 中央 N：3 σ + 1 π + 1 形式正电荷 = **价 5**

## 四、关键结论

### 4.1 同一原子，三个数值通常不同
| 物种 | 中心 | 价 | 氧化态 | 形式电荷 |
|---|---|---|---|---|
| $\ce{NH3}$ | N | 3 | −3 | 0 |
| $\ce{NH4+}$ | N | 5 | −3 | +1 |
| $\ce{NO3-}$ | N | 5 | +5 | +1 |
| $\ce{HNO3}$ | N | 5 | +5 | +1 |
| $\ce{N2}$ | N | 3 | 0 | 0 |
| $\ce{H2O}$ | O | 2 | −2 | 0 |
| $\ce{H3O+}$ | O | 3 | −2 | +1 |
| $\ce{OH-}$ | O | 1 | −2 | −1 |
| $\ce{O3}$ | 中央 O | 3 | 0 | +1 |
| $\ce{XeF2}$ | Xe | 2 | +2 | 0 |
| $\ce{SF6}$ | S | 6 | +6 | 0 |
| $\ce{H2SO4}$ | S | 6 (Lewis) 或 4 (离子式) | +6 | 0 (Lewis) 或 +2 (离子式) |

### 4.2 在箭头推动中如何使用
- **形式电荷**：判断箭头起点（孤对在哪原子）和终点（缺电子原子在哪）
- **氧化态**：用于判断反应是否为氧化还原反应
- **价**：用于命名（如 λ 体系）和高价化合物的辨识

### 4.3 多重键 vs 离子式选择对形式电荷的影响
对于 $\ce{H2SO4}$、$\ce{SOCl2}$、$\ce{SO2Cl2}$ 等：
| Lewis 结构 | 中央 S 的形式电荷 | 适用 |
|---|---|---|
| 多重键 ($\ce{S=O}$) | 0（满足八隅体扩展） | 命名、连接性 |
| 离子式 ($\ce{S^+-O^-}$) | +2 | **机理分析（推荐）** |

> 本书推荐**离子式**，因为它清楚显示孤对位置，使箭头推动更直观。

## 五、常见分类或情形

### 5.1 简单分子（多数化合物）
- 价 = 键数（中性、八隅体）
- 形式电荷 = 0
- 氧化态由电负性决定

### 5.2 带电离子或形式电荷不为零的体系
- 价 ≠ 键数
- 见 4.1 节的表格

### 5.3 高价化合物
- 价显然超过经典化合价
- 但通过 [[高价化合物的成键|3c-4e 键]] 解释——并非真的有 5 / 6 / 7 个普通 σ 键

### 5.4 离域体系（共振杂化）
- 形式电荷可能因共振结构不同而不同
- **平均**形式电荷反映分子的电荷分布

## 六、适用条件与限制

### 适用条件
- 任何能写出 Lewis 结构的体系
- 价：基于"键的几何"，对实际成键最敏感
- 氧化态：基于电负性，对水溶液氧化还原最有意义
- 形式电荷：基于均匀分配，对**机理**最直接

### 限制
- 价、氧化态、形式电荷都是**形式化**概念，不直接对应实测电荷
- 实际原子电荷需用 Mulliken/NBO/AIM 等量子化学方法计算
- 在过渡金属配合物中，三者更复杂——还需考虑非键 d 电子

## 七、常见比较与易混点

### 1. **"NH₄⁺ 中 N 是 5 价，所以氧化态 +5"** —— 错！
- 价 = 5（确实），但氧化态 = **−3**（H 比 N 电负性低，4 个 H 给电子归 N）
- 错误的根源：把"N 周围有 5 个键"等同于"N 失了 5 个电子"

### 2. **"形式电荷 = 真实电荷"** —— 错！
- 形式电荷只是 Lewis 结构的形式化标记
- 实测电荷（如 Mulliken）通常比形式电荷小得多
- 但形式电荷在**机理**讨论中比真实电荷更有用

### 3. **"价不能超过 4 / 不能超过 8 个电子"** —— 错！
- 第 3 周期及以下的元素可达到价 5、6、7
- 见 [[高价化合物的成键]] 和 [[λ符号体系]]

### 4. **价 vs 化合价（中学概念）**
- 中学的"化合价"是混合概念（有时等于价，有时等于氧化态）
- 本书要求严格使用"价（valence）"——纯粹的几何/键数概念

### 5. λ 命名法用的是哪个？
- λ⁵-P 表示 **价为 5 的磷**——使用**价**而非氧化态
- 详见 [[λ符号体系]]

## 十、例题

### 例题 1：N₂O 的三个数
**题目：** 计算 $\ce{N2O}$（线形 N=N=O）中两个 N 和 O 的价、氧化态、形式电荷。
**分析：** 共振结构为 $\ce{N(-)=N(+)=O}$ 或 $\ce{N#N(+)-O(-)}$
**解答：** 取右侧共振 $\ce{N#N^+-O^-}$
| 原子 | 价 | 氧化态 | 形式电荷 |
|---|---|---|---|
| 端 N | 3（三键） | −3 | 0 |
| 中 N | 4（三键 + 单键）| +3 | +1 |
| O | 1（单键 + 3 孤对） | −2 | −1 |

### 例题 2：$\ce{ClO4-}$ 的中心 Cl
**题目：** 在 $\ce{ClO4-}$ 中，Cl 的价、氧化态、形式电荷？
**分析：** 离子式 $\ce{Cl^{3+}(O^-)4}$ vs 多重键式 $\ce{Cl(=O)3-O^-}$
**解答：**
- 离子式：Cl 价 = 4，氧化态 = +7，形式电荷 = +3
- 多重键式：Cl 价 = 7，氧化态 = +7，形式电荷 = 0
- **氧化态在两种 Lewis 结构下保持 +7 不变**——这是氧化态的优势：与 Lewis 结构无关

## 十一、易错点
- 把 NH₄⁺ 中 N 的氧化态写成 +1（应为 −3）
- 把高价化合物（如 PF₅）中 P 的氧化态写成 +5 是对的，但说"价 5"也是对的——别把它们混淆为同一概念
- 计算形式电荷时漏了孤对
- 在共振结构中只取一个去算形式电荷（应取代表性结构或讨论平均）
