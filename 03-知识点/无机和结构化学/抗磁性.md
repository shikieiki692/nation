# 抗磁性

- 总览：[[中国化学奥林匹克基本要求-总览]]

## 一、定义

抗磁性（Diamagnetism）是指物质中所有电子均成对，在外加磁场作用下产生与外磁场方向相反的微弱磁矩，从而被磁场排斥的性质。抗磁性是所有物质的普遍属性，但在没有未成对电子的物质中表现得最为明显。

## 二、核心原理

### 2.1 产生机理

- **电子成对**：抗磁性物质中所有电子都成对填充在原子轨道或分子轨道中，净自旋 $S = 0$
- **感应磁矩**：外加磁场改变电子轨道运动，产生与外磁场方向相反的感应磁矩（楞次定律的微观表现）
- **磁化率**：$\chi < 0$，数量级约 $-10^{-6}$ ~ $-10^{-5}$，与温度无关

### 2.2 与顺磁性的对比

| 性质 | 抗磁性 | 顺磁性 |
|------|--------|--------|
| **电子结构** | 全部成对 | 有未成对电子 |
| **磁矩来源** | 感应磁矩（轨道运动改变） | 永久磁矩（自旋+轨道） |
| **磁化率符号** | $\chi < 0$ | $\chi > 0$ |
| **温度依赖性** | 基本与温度无关 | 服从居里定律，随温度升高而减弱 |
| **典型物质** | H₂O、N₂、NaCl、Zn²⁺（d¹⁰） | O₂、Fe³⁺（d⁵）、Cu²⁺（d⁹） |

### 2.3 竞赛中的判断要点

1. **过渡金属离子**：d⁰（Sc³⁺、Ti⁴⁺）和 d¹⁰（Zn²⁺、Cu⁺）构型通常为抗磁性
2. **强场配体 vs 弱场配体**：弱场下可能高自旋（顺磁），强场下可能低自旋甚至配对完全（抗磁）
3. **双原子分子**：$N_2$（抗磁）vs $O_2$（顺磁，有两个未成对电子）是经典对比
