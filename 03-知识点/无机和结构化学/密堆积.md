# 密堆积（Close Packing）

## 一、定义

等径球密堆积是理解金属晶体和离子晶体结构的**基础模型**。两种最密堆积方式——立方密堆积（ccp）和六方密堆积（hcp）——具有相同的空间利用率（74.05%），但堆积方式和空隙分布不同，导致衍生结构多样性。

## 二、考纲对应

密堆积在考纲中属于**基础工具性知识点**，是衔接[[16-晶体基本概念]]（等径球堆积与填隙类型）和[[17-常见晶体结构]]（NaCl型、立方ZnS型、钙钛矿型等衍生结构）的枢纽。初赛明确要求"了解等径球堆积的基本方式（简单立方、体心立方、立方密堆积和六方密堆积）与填隙类型"，而决赛在此之上进一步考查空隙计数、填隙率反推、晶胞参数与原子半径的几何关系。几乎所有离子晶体和金属晶体结构的分析都以密堆积模型为出发点，该条目同时也是[[金属晶体]]、[[离子晶体]]中配位数和空间利用率计算的直接前提。

**考纲条目直接对应**：
- [[16-晶体基本概念]]：等径球堆积的四种基本方式 + 填隙类型
- [[17-常见晶体结构]]：由密堆积衍生出的 NaCl 型、CsCl 型、立方ZnS 型、CaF₂ 型、金红石型、钙钛矿型、金刚石型等
- [[15-晶体结构]]：晶体类型（离子/原子/金属/分子晶体）的概念区分，密堆积是金属晶体和离子晶体的共同语言

**决赛深化方向**（见 [[决赛要求-结构与配位深化]]）：
- 空隙坐标的精确计算（如 hcp 中 (1/3, 2/3, 1/4) 八面体位置）
- 非理想 c/a 比的实际结构分析（NiAs 型压缩、CaC₂ 型拉长）
- 多型体（polytype）的堆叠序列判断（SiC、ZnS 等）

## 三、核心原理

### 两种最密堆积

#### 六方密堆积（hcp, ABAB...）

- **堆积序列**：ABAB...
- **晶胞参数**：a = b = 2r，c = 1.633a（理想值，c/a = √(8/3) ≈ 1.633）
- **配位数**：12
- **空间利用率**：74.05%
- **实际偏离案例**：NiAs 型结构中 c/a ≈ 1.39（显著压缩），源于 As–As 间次级键作用使层间距离缩短
- **空隙类型与数量**（每 N 个球）：
  - 四面体空隙：2N
  - 八面体空隙：N

#### 立方密堆积（ccp, ABCABC...）

- **堆积序列**：ABCABC...
- **晶胞类型**：面心立方（fcc）
- **配位数**：12
- **空间利用率**：74.05%
- **空隙类型与数量**（每 N 个球）：
  - 四面体空隙：2N
  - 八面体空隙：N

#### 体心立方堆积（bcc）

- **非最密堆积**，但常见于碱金属
- **配位数**：8（最近邻），6（次近邻，距离仅远15%）
- **空间利用率**：68.02%

### 空隙位置与坐标

#### 立方密堆积（fcc）中的空隙

| 空隙类型 | 位置 | 坐标 | 每晶胞数量 |
|:---|:---|:---|:---:|
| 八面体空隙 | 体心 + 棱心 | (1/2,1/2,1/2)；棱心 | 4 |
| 四面体空隙 | 八个小立方体体心 | 如 (1/4,1/4,1/4) | 8 |

#### 空隙大小

- **八面体空隙半径**：r_oct = 0.414r
- **四面体空隙半径**：r_tet = 0.225r

## 四、关键结论

1. **ccp 与 hcp 空间利用率相同（74.05%）**，但堆积序列不同：ccp 为 ABCABC...（面心立方），hcp 为 ABAB...（六方）。空间利用率相等的原因是两种堆积方式中密置层的排布密度完全相同，区别仅在层间相对平移。

2. **空隙数量关系**（ccp 与 hcp 均满足）：每 N 个等径球产生 **N 个八面体空隙 + 2N 个四面体空隙**。对于 fcc 晶胞（含 4 个原子）：4 个八面体 + 8 个四面体；对于 hcp 晶胞（含 2 个原子）：2 个八面体 + 4 个四面体。

3. **体心立方堆积（bcc）不是密堆积**：空间利用率仅 $\eta = \frac{\sqrt{3}\pi}{8} \approx 68.02\%$，配位数 8（最近邻），但有 6 个次近邻距离仅远约 15%，因此有时在讨论"有效配位环境"时被近似视为 14 配位。

4. **理想 hcp 的轴比**：$c/a = \sqrt{8/3} \approx 1.633$。推导来源于两密置层间距离 $h = \sqrt{\frac{2}{3}}a$，而 $c = 2h$，故 $c/a = 2\sqrt{2/3} = \sqrt{8/3}$。实际金属和化合物的 c/a 可偏离此值（如 Zn 的 c/a ≈ 1.856，NiAs 的 c/a ≈ 1.39）。

5. **空隙半径比**：
   - 八面体空隙可容纳的最大小球半径：$r_{\text{oct}} = (\sqrt{2} - 1)r \approx 0.414r$
   - 四面体空隙可容纳的最大小球半径：$r_{\text{tet}} = (\sqrt{3/2} - 1)r \approx 0.225r$
   - 该比值是判断正离子占据何种空隙的**关键判据**：$r_+/r_- > 0.414$ 时倾向八面体配位，$0.225 < r_+/r_- < 0.414$ 时倾向四面体配位。

6. **fcc 空隙位置记忆**：八面体空隙——体心 (1/2, 1/2, 1/2) + 12 条棱心（每条棱分占 1/4，等价于 3 个）= 共 4 个；四面体空隙——将晶胞分成 8 个等大的小立方体，取每个小立方体的体心 = 8 个。

7. **密堆积是"骨架模型"**：先有阴离子/金属原子的密堆积排列，再由阳离子填入空隙，是分析绝大多数无机晶体结构的统一方法论（NaCl 型 = ccp + 全部八面体空隙被占；闪锌矿型 = ccp + 一半四面体空隙被占；反萤石型 = ccp + 全部四面体空隙被占等）。

## 五、常见分类或情形

### 密堆积的变换与组合

#### 堆叠层错
- hcp 与 ccp 的混合堆叠（如 ABCAB...）形成**多型体**
- 常见于 SiC、ZnS 等化合物

#### 密堆积到非密堆积的转变
- **温度/压力诱导**：如 Co 在高温下从 hcp 转变为 ccp
- **电子因素**：过渡金属的 d 电子数影响最稳定堆积方式

### HCP 空隙坐标表（Zchem 中级无机结构化学网课素材 · 资产 I-4 · 可信度：高）

以理想 hcp 晶胞（a = b = 2r，c = 1.633a）为参考：

| 空隙类型 | 位置描述 | 原子坐标 (分数坐标) | 每晶胞数量 |
|:---|:---|:---|:---:|
| **八面体空隙** | 在 (1/3, 2/3, 1/4) 及其等效位置 | (1/3, 2/3, 1/4); (2/3, 1/3, 3/4) | 2 |
| **四面体空隙 (A)**| 在 (0, 0, 3/8) 等棱上位置 | (0, 0, 3/8); (0, 0, 5/8) | 4 |
| **四面体空隙 (B)**| 在 (1/3, 2/3, 7/8) 等位置 | (1/3, 2/3, 7/8); (2/3, 1/3, 1/8) | 4 |

**验证**：hcp 每 N 个原子含 N 个八面体空隙 + 2N 个四面体空隙。晶胞含 2 个原子 → 2 个八面体 + 4 个四面体 = 6 个空隙。

**教学提示**：
- 八面体空隙在 c 轴方向交替排列（z = 1/4 和 3/4）
- 四面体空隙分为两组，分别对应两种堆叠层下方的空隙
- 坐标记忆技巧：1/4 和 3/4 是八面体；3/8 和 5/8 是四面体（差 1/8）

## 六、适用条件与限制

密堆积模型是一个高度理想化的几何模型，其有效性依赖于以下前提，超出范围时必须谨慎：

1. **等径球假设**：模型假设所有原子（或离子）为半径相等且不可压缩的刚性球。实际原子半径随化合价、配位数和自旋态变化（见 [[离子半径]]、[[原子半径]]），阴离子-阳离子半径差可达数倍（如 Cs⁺ 和 F⁻），需用不等径球填隙模型修正。

2. **忽略键的方向性**：密堆积基于各向同性的球形力场，适用于金属键和离子键体系。对于有方向性的共价键（如金刚石中的 sp³ 杂化），密堆积模型失效——金刚石型结构虽可由 fcc + 填隙导出，但其本质是共价方向性决定的结构，不是密堆积的直接结果。

3. **最密堆积不等于唯一稳定排列**：许多金属的实际堆积方式由电子结构决定。例如过渡金属中 d 电子数影响六方/立方堆积的稳定性——Mn(γ)、Fe(γ) 为 fcc，而 Ti、Zr、Co(α) 为 hcp。bcc 堆积虽然空间利用率更低（68.02%），但在碱金属和高熵合金中反而稳定（参见 [[金属晶体]]）。

4. **hcp 的 c/a 比可显著偏离理想值 1.633**：实际金属中 Zn（c/a ≈ 1.856）、Cd（c/a ≈ 1.886）明显偏离理想球形假设，反映出 d 电子的非球形电荷分布和层间方向性作用。NiAs 型化合物中 As–As 次级键使 c/a 压缩至 ~1.39（见 §十二）。

5. **温度与压力的影响**：堆积方式可随温度/压力变化——Co 在 ~420°C 从 hcp 转变为 ccp；Fe 从 bcc → fcc → bcc 随温度变化。高压下碱金属可从 bcc 转变为更致密的 fcc 或更复杂的结构。密堆积的"几何最佳"与自由能最小的真实稳定相并非总是同一件事。

6. **填隙模型的半径比判据是必要条件而非充分条件**：$r_+/r_- > 0.414$ 时阳离子"可以"进入八面体空隙，但实际是否进入还取决于极化效应（[[离子极化]]）、晶体场稳定化能 CFSE（过渡金属离子倾向占据特定对称性位置）以及 Madelung 常数等能量因素（[[晶格能]]）。例如尖晶石结构中 Zn²⁺ 倾向占据四面体位，尽管半径比更支持八面体位（参见 [[尖晶石结构]]）。

7. **"空隙"本身易被误解为"空位"**：化学竞赛中常见混淆——"填隙"指阳离子占据阴离子堆积中的空隙位置（如 NaCl 中 Na⁺ 在 Cl⁻ 的八面体空隙中），而非指晶体中存在本应被占据却空缺的格点（后者是缺陷化学的内容，见 [[非化学计量化合物]]、[[非整比化合物]]）。

### 7.3 bcc 配位数的澄清：8 还是 14？

这是竞赛中常见的混淆点，需从定义出发澄清：

- **按最近邻定义**：bcc 中每个原子有 8 个最近邻（体心到 8 个顶点），因此**配位数为 8**。大多数教材采用此定义（[[配位数]]）。
- **"有效配位数 14"说的来源**：bcc 中次近邻（6 个，沿 \[100\] 方向）的距离仅比最近邻远约 15%（$a$ vs $\frac{\sqrt{3}}{2}a \approx 0.866a$）。若将次近邻也"视同"配位原子，则 8 + 6 = 14。但这不是标准配位数的定义方式。
- **竞赛答题建议**：明确回答"配位数为 8（若考虑次近邻则为 8 + 6）"。不可简单说"bcc 配位数是 14"——这会丢分。

**bcc 不是密堆积**的再次提醒：空间利用率 68.02% < 74.05%，其空隙类型和数量关系也与 ccp/hcp 完全不同——bcc 每晶胞含 2 个原子，产生 6 个八面体空隙和 12 个四面体空隙（但这些空隙并非正多面体，且空隙大小小于等径球模型下的可填隙条件）。

- [[体心立方堆积]]：bcc 作为非密堆积对照（配位数、空隙类型）
- [[等径球堆积]]：更宏观的堆积分类（包含简单立方堆积等非密堆积方式）
- [[填隙]]：阳离子填入空隙的系统方法论
- [[配位数]]：从堆积角度理解配位数的几何含义
- [[离子半径]] / [[原子半径]]：填隙半径比判据的数值来源

### 应用与衍生（密堆积是这些知识点的前置）
- [[NaCl型结构]]：ccp 阴离子 + 全部八面体空隙被阳离子占据 → 6:6 配位
- [[CsCl型结构]]：不是密堆积衍生结构（简单立方 + 体心填隙 → 8:8 配位）
- [[立方ZnS型结构]]（闪锌矿型）：ccp 阴离子 + 一半四面体空隙被阳离子占据 → 4:4 配位
- [[六方ZnS型结构]]：hcp 阴离子 + 一半四面体空隙被阳离子占据 → 4:4 配位
- [[CaF2型结构]]（萤石型）：ccp 阳离子 + 全部四面体空隙被阴离子占据 → 8:4 配位
- [[反萤石型结构]]：萤石型的反式（阳离子和阴离子角色互换）
- [[NiAs型结构]]：hcp 阴离子 + 全部八面体空隙被阳离子占据 → 6:6 配位，c/a 显著压缩
- [[钙钛矿结构]]：ccp 大阳离子 + 小阳离子填八面体空隙（octahedral factor/tolerance factor 分析）
- [[尖晶石结构]]：ccp 阴离子 + 1/8 四面体 + 1/2 八面体被占据，常式/反式由位择能决定
- [[金红石型结构]]：hcp 变形结构 + 一半八面体空隙被占据 → TiO₂ 型
- [[金属晶体]]：直接以密堆积（ccp/hcp/bcc）描述金属单质的结构类型分布
- [[碳的同素异形体]]：金刚石 = fcc 填隙（四面体）；石墨 = 层内六方 + 层间 π 堆积（非密堆积逻辑）
- [[天然冰结构]]：冰 I_h 的氧原子排列可参照 hcp 中的四面体空隙方向理解（虽然冰不是密堆积）
- [[堆垛层错]] / 多型体：SiC、ZnS 等多型现象的堆积序列视角
- [[离子极化]]：极化使离子偏离球形，半径比判据可能失效
- [[晶格能]]：堆积方式影响 Madelung 常数，进而影响晶格能大小

## 九、典型题型

1. **空间利用率计算**：给定堆积方式（fcc/hcp/bcc），推导空间利用率 $\eta = \frac{V_{\text{球}}}{V_{\text{晶胞}}}$。经典题为"证明 fcc 的空间利用率为 $\frac{\sqrt{2}\pi}{6}$"。

2. **填隙率及半径比判断**：已知阳离子和阴离子半径，判断阳离子填入何种空隙（八面体/四面体），并计算填隙后的空间利用率或密度。常结合 $0.414$ 和 $0.225$ 的半径比阈值。

3. **晶胞参数与原子半径互推**：由晶胞参数 a 反推原子半径 r（如 fcc 中 $a = 2\sqrt{2}r$，bcc 中 $a = \frac{4}{\sqrt{3}}r$），或由原子半径预测晶胞参数（如 hcp 中 $a = 2r$，$c = \sqrt{8/3}a$）。

4. **从堆积方式判断实际晶体结构**：给出晶体结构描述（哪种原子构成何种堆积、何种原子占据哪些空隙），要求写出晶胞中各类原子的数目和位置，进而确定化学式和结构类型（如"阴离子构成 fcc，阳离子占据全部八面体空隙 → NaCl 型"）。

5. **多型体与堆叠序列**：给出 X 射线衍射数据或堆叠序列文字描述，判断是 ccp 还是 hcp 或其混合型；识别 SiC、ZnS 等化合物的多型体（如 2H、4H、6H、3C 等符号的含义）。

6. **bcc 区别于密堆积的专项考查**：计算 bcc 的空间利用率（68.02%），比较 bcc 与 fcc 的配位数和空隙差异，分析碱金属为何偏好 bcc 而非密堆积。

7. **竞赛决赛进阶题型**：hcp 空隙坐标精确推导（如推导 (1/3, 2/3, 1/4) 的来源）、非理想 c/a 下的结构参数计算（NiAs 型）、尖晶石常式/反式的位择能分析（结合 CFSE）。

## 十、例题

### 例题 1：fcc 空间利用率推导

**题目**：证明面心立方密堆积（fcc / ccp）的空间利用率为 $\displaystyle \eta = \frac{\sqrt{2}\pi}{6} \approx 74.05\%$。

**分析**：
- 空间利用率 = 晶胞内所有原子的体积之和 / 晶胞体积
- fcc 晶胞含 4 个原子（$8 \times 1/8 + 6 \times 1/2 = 4$）
- 需要找到晶胞参数 a 与原子半径 r 的关系：fcc 中密堆积方向在面对角线

**解答**：
1. fcc 的密堆积方向为面对角线：面对角线上三个原子依次相切，故 $4r = \sqrt{2}a$，即 $r = \frac{\sqrt{2}}{4}a$。
2. 晶胞体积：$V_{\text{cell}} = a^3$。
3. 4 个原子的总体积：$V_{\text{atom}} = 4 \times \frac{4}{3}\pi r^3 = \frac{16}{3}\pi \left(\frac{\sqrt{2}}{4}a\right)^3 = \frac{16}{3}\pi \cdot \frac{2\sqrt{2}}{64}a^3 = \frac{\sqrt{2}\pi}{6}a^3$。
4. 空间利用率：$\eta = \frac{V_{\text{atom}}}{V_{\text{cell}}} = \frac{\sqrt{2}\pi}{6} \approx 0.7405 = 74.05\%$。

**反思**：
- 关键步骤是正确识别 fcc 的密堆积方向在 **面对角线**（而非体对角线或棱）。体对角线上原子不相切（$\sqrt{3}a = 4r$ 会导致不同的空间利用率，那实际上是 bcc 的关系）。
- 同样的方法可推导 bcc 的空间利用率：体对角线上相切 → $\sqrt{3}a = 4r$ → $\eta = \frac{\sqrt{3}\pi}{8} \approx 68.02\%$。注意区分"体对角线相切"是 bcc，"面对角线相切"是 fcc。
- 竞赛中常考变体：简立方 $\eta = \pi/6 \approx 52.36\%$（棱上相切，$a = 2r$）。

### 例题 2：填隙半径比计算与结构判断

**题目**：
已知某离子晶体 AB 中，阴离子 B⁻ 的半径为 181 pm，阳离子 A⁺ 的半径为 102 pm。阴离子按 fcc 密堆积排列。
（1）判断 A⁺ 倾向占据八面体空隙还是四面体空隙。
（2）若 A⁺ 恰好填满所有该类空隙，请给出该晶体的结构类型和化学式。
（3）计算该晶体的理论密度。已知 B⁻ 的摩尔质量为 35.5 g·mol⁻¹，A⁺ 的摩尔质量为 23.0 g·mol⁻¹。

**分析**：
- 半径比 $r_+/r_- = 102/181 \approx 0.564$，与阈值 0.414 和 0.225 比较
- fcc 每晶胞含 4 个阴离子，空隙数为 4 八面体 + 8 四面体
- 密度需要晶胞参数 a：由 fcc 中 $4r_- = \sqrt{2}a$ 求出 a

**解答**：
（1）$r_+/r_- = \frac{102}{181} \approx 0.564$。
   - $0.414 < 0.564 < 0.732$，故 $r_+/r_-$ 支持 **八面体配位**。阳离子倾向占据八面体空隙。

（2）fcc 晶胞含 4 个阴离子 B⁻，对应 4 个八面体空隙。若全部八面体空隙被占：每晶胞含 4 个 A⁺ + 4 个 B⁻ → 化学式 AB → **NaCl 型结构**。

（3）晶胞参数：$4r_- = \sqrt{2}a \Rightarrow a = \frac{4 \times 181}{\sqrt{2}} = 512\ \text{pm} = 5.12 \times 10^{-8}\ \text{cm}$。

   晶胞体积：$V = a^3 = (5.12 \times 10^{-8})^3 = 1.342 \times 10^{-22}\ \text{cm}^3$。

   晶胞质量：$m = \frac{4 \times (23.0 + 35.5)}{N_A} = \frac{234.0}{6.022 \times 10^{23}} = 3.886 \times 10^{-22}\ \text{g}$。

   密度：$\rho = \frac{m}{V} = \frac{3.886 \times 10^{-22}}{1.342 \times 10^{-22}} \approx 2.90\ \text{g·cm}^{-3}$。

**反思**：
- 该题数值对应 NaCl 晶体：Na⁺（r = 102 pm），Cl⁻（r = 181 pm），$r_+/r_- = 0.564$ 落在八面体区间。真实 NaCl 密度为 2.17 g·cm⁻³，偏差来自于离子半径取值（Pauling 半径 vs Shannon 半径）和晶胞参数的略微差异。
- **陷阱提示**：如果 $r_+/r_- < 0.414$，不能说"阳离子进不去八面体空隙"——半径比判据只保证"不撑大空隙前提下的几何可行性"，实际阳离子过大时会使阴离子间距增大（晶胞膨胀），仍可占据该空隙，只是空间利用率分析不再精确。

### 例题 3：从堆积方式判断实际晶体结构（CChO 风格）

**题目**：
某离子晶体的结构描述如下：阴离子 X²⁻ 按立方密堆积（ccp）排列；阳离子 A²⁺ 占据 1/8 的四面体空隙；阳离子 B³⁺ 占据 1/2 的八面体空隙。
（1）写出该晶体的化学式（用 A、B、X 表示）。
（2）判断该晶体属于哪种已知结构类型。
（3）若 X²⁻ 的半径为 140 pm，A²⁺ 的半径为 60 pm，B³⁺ 的半径为 65 pm，讨论 A²⁺ 和 B³⁺ 占据各自空隙的几何合理性。

**分析**：
- ccp 晶胞含 4 个阴离子 X²⁻ → 8 个四面体空隙 + 4 个八面体空隙
- 按占据比例反推各离子数目 → 确定化学式
- 半径比验证

**解答**：
（1）ccp 晶胞含 4 个 X²⁻，对应 8 个四面体空隙和 4 个八面体空隙。
   - A²⁺ 数目 = $8 \times 1/8 = 1$
   - B³⁺ 数目 = $4 \times 1/2 = 2$
   - 化学式：$\boxed{\text{AB}_2\text{X}_4}$

（2）A 占据 1/8 四面体 + B 占据 1/2 八面体 + ccp 阴离子框架 → 这是 **[[尖晶石结构]]**（spinel, $\text{MgAl}_2\text{O}_4$ 型）。
   - 常式尖晶石：A²⁺ 在四面体位，B³⁺ 在八面体位 → 符合本题描述。
   - 反式尖晶石：A²⁺ 在八面体位，一半 B³⁺ 在四面体位，另一半 B³⁺ 在八面体位 → 本题是常式。

（3）半径比验证：
   - A²⁺（四面体）：$r_A/r_X = 60/140 \approx 0.429 > 0.225$ ✓（几何可行，但偏大，需撑大空隙）
   - B³⁺（八面体）：$r_B/r_X = 65/140 \approx 0.464 > 0.414$ ✓（几何可行）

**反思**：
- 此题是典型竞赛题模式——"堆积 + 填隙率 → 化学式 → 结构类型"。掌握 ccp 空隙数（4 八面体 + 8 四面体）是解题第一步。
- 实际尖晶石中，ZnFe₂O₄ 为常式（Zn²⁺ 填四面体），而 Fe₃O₄ 为反式（Fe³⁺ 填四面体，Fe²⁺ + Fe³⁺ 平分八面体）。区分常式/反式的深层原因在于 CFSE（晶体场稳定化能）——这也是决赛常考延伸点。

### 例题 4：hcp 晶胞参数与空隙坐标推导

**题目**：
已知理想六方密堆积（hcp）中，原子半径为 r。
（1）推导晶胞参数 a 和 c（用 r 表示），并证明 $c/a = \sqrt{8/3}$。
（2）指出 hcp 晶胞中八面体空隙的分数坐标，并说明推导思路。
（3）计算 hcp 的空间利用率，证明其与 fcc 相同（均为 74.05%）。

**分析**：
- hcp 晶胞为六方，a = b = 2r（底面六边形边长为 2r）
- c 轴由两层密置层的间距决定：单层层间距 $h = \sqrt{\frac{2}{3}}a$，$c = 2h$
- 八面体空隙位于两层之间

**解答**：
（1）底面为正六边形排列，相邻球相切 → $a = 2r$。
   密置层内三个相邻球心构成等边三角形（边长 2r），其中心到底边的高为 $\sqrt{3}r$。在此三角形正上方或正下方的球心到底面的垂直距离：
   三角形中心到球心距离：$\frac{2}{3} \times \frac{\sqrt{3}}{2} \times (2r) = \frac{2\sqrt{3}}{3}r$（重心到顶点的距离）。
   上方的球心到三个底球组成四面体，四面体高就是层间距：
   $h = \sqrt{(2r)^2 - \left(\frac{2\sqrt{3}}{3}r\right)^2} = \sqrt{4r^2 - \frac{4}{3}r^2} = \sqrt{\frac{8}{3}}r = \frac{2\sqrt{2}}{\sqrt{3}}r$。
   而 $c = 2h = \frac{4\sqrt{2}}{\sqrt{3}}r$。
   又 $a = 2r$，故 $c/a = \frac{2\sqrt{2}}{\sqrt{3}} = \sqrt{8/3} \approx 1.633$。

（2）八面体空隙位于两层之间，在三个底球和三个顶球之间的位置。取理想的六方坐标：
   底面两个原子在 (0, 0, 0) 和 (1/3, 2/3, 1/2)（体心原子），八面体空隙中心在 (1/3, 2/3, 1/4) 和 (2/3, 1/3, 3/4)。

   **推导思路**：在 AB 两层之间，三个 A 层球（z = 0）和三个 B 层球（z = 1/2）围成八面体，其中心位于 z = 1/4 处；在 xy 平面的投影位于两个三角形交错后的"孔"位置（1/3, 2/3）或（2/3, 1/3）。

（3）hcp 晶胞体积：六方柱体积 $V = \frac{\sqrt{3}}{2}a^2 c = \frac{\sqrt{3}}{2}(2r)^2 \cdot \frac{4\sqrt{2}}{\sqrt{3}}r = \frac{\sqrt{3}}{2} \cdot 4r^2 \cdot \frac{4\sqrt{2}}{\sqrt{3}}r = 8\sqrt{2}r^3$。
   晶胞含 2 个原子（$8 \times 1/8 + 1 = 2$）。
   原子总体积：$V_{\text{atom}} = 2 \times \frac{4}{3}\pi r^3 = \frac{8}{3}\pi r^3$。
   空间利用率：$\eta = \frac{8\pi r^3/3}{8\sqrt{2}r^3} = \frac{\pi}{3\sqrt{2}} = \frac{\sqrt{2}\pi}{6} \approx 74.05\%$。

   fcc 的空间利用率同样为 $\frac{\sqrt{2}\pi}{6}$，证毕。

**反思**：
- hcp 的 c/a 推导是竞赛中的经典几何题，关键是正确建立底面等边三角形和上方球心的空间关系——本质是在计算正四面体的高。
- hcp 的空隙坐标比 fcc 更容易写错：注意晶胞体心原子在 (1/3, 2/3, 1/2)，八面体空隙在 z = 1/4 和 3/4 交替（而非 z = 0 和 1/2）。
- hcp 与 fcc 空间利用率相同的深层原因：两种堆积的**局域几何**完全相同——每个球都被 12 个最近邻以相同方式包围，只是层间平移方向不同。

## 十一、易错点

1. **hcp 与 ccp 的空隙数量相同**：都是 2N 四面体 + N 八面体，但空间分布不同
2. **fcc 晶胞中的空隙不要遗漏**：八面体空隙除了体心，还有 12 条棱心（各分占 1/4）
3. **体心立方不是密堆积**：尽管有时被误认为是"9 配位"，但空间利用率明显低于 74%

## 十二、🎯 教学视角

### 12.1 从"理想"到"实际"：NiAs 的 c/a 压缩

> 来源：[[教学逻辑提炼-Zchem-主族元素的结构化学I-第二轮深化]] · 资产 I-2 · 可信度：高

**冲突数据**：
- 理想 HCP 的 c/a = √(8/3) ≈ **1.633**（几何推导的"完美值"）
- NiAs 型结构的实际 c/a ≈ **1.39**（显著偏离理想值）

**学生已有认知**：
- HCP 和 CCP 是"标准"密堆积，晶胞参数由几何唯一确定
- 同种堆积方式的 c/a 应该相同

**认知冲突**：
- NiAs 也是 HCP 衍生结构，但 c/a 压缩了约 15%
- 原因：As 原子在层间形成 As–As 次级键，把层"拉"得更近
- **结论**：堆积序列（ABAB）只是骨架，层间相互作用可以显著改变几何参数

**教学策略**：
1. 先强调理想 HCP 的 c/a = 1.633 是几何必然结果
2. 再给 NiAs 的反例，制造冲突
3. 引导学生从"几何唯一"转向"相互作用决定结构"
4. 为后续理解 CaC₂（立方→四方）、尖晶石（常式/反式）等结构畸变铺垫

**适用轮次**：
- 第一轮：知道理想值 1.633 即可
- 第二轮：引入 NiAs 偏离，建立"理想模型→实际修正"的思维框架

### 12.2 学习路径

1. 先从等径球模型建立 ccp 和 hcp 的直观图像
2. 掌握空隙类型、数量和位置的计算
3. 理解"堆积序列 + 填隙率"决定衍生结构的思路
4. 最后引入实际偏离（NiAs、CaC₂ 等），建立"模型是起点不是终点"的认知
