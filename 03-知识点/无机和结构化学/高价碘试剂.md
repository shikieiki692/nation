# 高价碘试剂（Hypervalent Iodine Reagents）

- 总览：[[决赛要求-总览]]
- 所属模块：[[决赛要求-有机与无机拓展]]、[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]、[[40-氧化还原反应]]、[[50-有机合成]]、[[决赛11-金属有机化学]]
- 外部资料：[[Arrow Pushing in Inorganic Chemistry-总索引]] §7.10–7.12

## 一、定义
**高价碘试剂**：含有 **碘原子在 +3、+5 或 +7 形式氧化态** 的有机/无机化合物，统称 **hypervalent iodine compounds**。它们的结构特征是 **碘上配体数 > 1**（与单价 I⁻ 相对），常以 **λⁿ-iodane** 命名（n = 3, 5, 7）。

| 名称 | λ值 | 价态 | 配体数 | 代表 |
|---|---|---|---|---|
| **λ³-碘烷** | λ³ | I(III) | 3 | $\ce{PhICl2}$, $\ce{PhI(OAc)2}$ (PIDA), Koser, Togni |
| **λ⁵-碘烷** | λ⁵ | I(V) | 5 | IBX, DMP（[[Dess-Martin氧化]]） |
| **λ⁷-碘烷** | λ⁷ | I(VII) | 7 | $\ce{HIO4}$, $\ce{H5IO6}$（[[过碘酸氧化]]） |

> **λ 符号体系**：源自 IUPAC 命名（详见 [[λ符号体系]]），表示原子的"非常规价数"。如 PhICl₂ 中 I 形式上 3 配位、+3 氧化态，记为 **λ³-iodane**。

## 三、核心原理

### 3.1 整体反应模式：碘的"自身电子流"

高价碘试剂的反应几乎都遵循以下 3 类机理：

**(a) 配体交换（亲核取代 I）**
亲核体 Nu⁻ 取代碘上的某配体 X⁻：

$$
\ce{Ar-I^{n+}-X + Nu^- -> Ar-I^{n+}-Nu + X^-}
$$

类似 [[SN2-Si机理]] 的"五配位中间体"过程。

**(b) 还原消除**
配体之间相互成键，I 还原 2 价（如 I(V) → I(III)、I(III) → I(I)）：

$$
\ce{Ar-I(III)-X-Nu -> Ar-I + X-Nu}
$$

是 **氧化产物形成 + I 还原** 的同步过程。

**(c) 配位物上的协同消除（环状过渡态）**
如 IBX/DMP/HIO₄ 中的 **5/4 元环过渡态**：碘上的氧拔取相邻 C 上的 H，C–O–I 同时断裂。

### 3.2 λ³-碘烷（I(III) 试剂）

**代表**：
- **$\ce{PhI(OAc)2}$**（碘苯二乙酸酯，PhI(OAc)₂，PIDA、IBD）：常见 α-酯化、α-氢取代试剂
- **$\ce{PhI=O}$**（碘酰苯，iodosobenzene）：实际为聚合物 $(PhIO)_n$
- **Koser 试剂 $\ce{PhI(OH)OTs}$**（HTIB）：用于 α-甲苯磺酰氧基化
- **Togni 试剂 1, 2**：含 -CF₃，用于亲电三氟甲基化
- **$\ce{PhICl2}$**：α-氯化、芳基氯化
- **二芳基碘鎓盐 Ar₂I⁺X⁻**：α-芳基化

![[ArrowPushinginInorganicChemistry_231-336_images/e9e4ef7bf43fb8b11fb484124cbd1e5b6c0e32721c36fdb4197b6abd2bb66970.jpg]]

> 图：λ³-碘烷家族的典型成员（红框为五价 λ⁵）。

**Koser 试剂的 α-甲苯磺酰氧基化机理**：

![[ArrowPushinginInorganicChemistry_231-336_images/85413a8fb0257b4dc58ff474acfe347e51e2ed8e4c5db500b60d36a4efd775ed.jpg]]

第 1 步：酮的烯醇式作为亲核体进攻 I(III) 中心，置换 OH⁻；
第 2 步：OTs⁻ 反向 SN2 进攻 α-C，I 还原为 PhI（离去），得 α-OTs 酮：

![[ArrowPushinginInorganicChemistry_231-336_images/8f10747ccf8c75c2aeb43d22b9a10735a340b375d335146f0ea876ecf5864791.jpg]]

> α-OTs 酮可进一步被亲核体取代，得到 α-取代酮（这是 Koser 试剂的核心价值）。

### 3.3 λ³-碘鎓盐与碳烯/氮宾转移

PhI(OAc)₂ + 1,3-二羰基化合物 + 碱 → **碘鎓叶立德**（iodonium ylide）：

$$
\ce{PhI(OAc)2 + R(EWG)CH-EWG' ->[\text{KOH}] PhI=C(EWG)(EWG') + 2AcOH}
$$

这种碘鎓叶立德是 **碳烯前体**（Doyle 等人的卡宾化学）和 **氮宾前体**（Du Bois 等人的 C-H 胺化）。

> 大多数碳烯/氮宾转移反应需金属催化（Rh、Cu、Ag 等），属决赛深度内容。

### 3.4 Togni 试剂：电亲三氟甲基化

![[ArrowPushinginInorganicChemistry_231-336_images/8696f1dd67f2bf8acfc42c0ffa9a44feb1d088dbfe6dec1b6891af69f63b7f5b.jpg]]

Togni 试剂 1（环型）和 Togni 试剂 2（线型）含 **I-CF₃ 键**。亲核体（醇、酚、硫醇、烯醇）进攻 I(III) 中心，CF₃ 作为亲电"-CF₃⁺"（实际是 CF₃⁻ 的 σ-trans 转移）转移到亲核体：

$$
\ce{Nu-H + ArI-CF3 -> Nu-CF3 + ArI + H+}
$$

这是 Ruppert-Prakash 试剂（Me₃SiCF₃，亲核 -CF₃⁻）的"反向"对应物。

### 3.5 λ⁵-碘烷（I(V) 试剂）

详见 [[Dess-Martin氧化]]。代表：
- **IBX**（2-碘酰基苯甲酸）：醇 → 醛/酮，仅 DMSO 可溶
- **DMP**（Dess-Martin Periodinane）：醇 → 醛/酮，CH₂Cl₂ 可溶，操作友好

机理为 **5 元环协同过渡态**：醇配位 I(V) → 烷氧基-λ⁵-碘烷 → 协同消除 → 醛/酮 + λ³-碘烷。

### 3.6 λ⁷-碘烷（I(VII) 试剂）

详见 [[过碘酸氧化]]。代表：
- **HIO₄ / H₅IO₆**（过碘酸）：cis-1,2-二醇 → 二羰基（Malaprade 反应），机理为环状碘酸酯过渡态

### 3.7 高价碘的电子结构

I 的 5p 轨道与 5d 杂化形成 sp³d / sp³d² 杂化，可容纳 5 / 6 个电子对。但 IBX 中的 I(V) 实际只是 **5 个 σ-键 + 1 对孤对**，类似 Brown 描述的"3-中心-4-电子键"（[[高价化合物的成键]]）。

> **HSAB 视角**：高价 I 是相对软的亲电中心；软亲核体（S, Se, P, 烯醇）反应速率最快，硬亲核体（O, F⁻）次之。

## 四、关键结论

- **高价碘 = 主族版的过渡金属催化剂**：能进行氧化加成、还原消除、配体交换、α-官能化，但不需要重金属。
- **碘的"价态阶梯"**：
  - I(−1)：I⁻（碘离子）
  - I(+1)：I⁺（在 I₂ 中、碘鎓盐中）
  - **I(+3)：λ³-碘烷**（PIDA、Koser、Togni 试剂、二芳基碘鎓盐）
  - **I(+5)：λ⁵-碘烷**（IBX、DMP）
  - **I(+7)：λ⁷-碘烷**（HIO₄、IO₄⁻）
- 反应中**碘自身电子流**承担"氧化-还原"的载体作用：碘 +3 → +1 时释放 2 电子等价于氧化底物。
- **不含重金属、绿色、温和**：是现代合成的重要试剂体系。

## 五、常见分类或情形

| 试剂 | 价态 | 主要用途 | 机理类型 |
|---|---|---|---|
| **PhICl₂** | I(III) | 芳基氯化、α-氯化 | 配体交换 + 还原消除 |
| **PhI(OAc)₂（PIDA）** | I(III) | α-酰氧化、Hofmann 重排 | 配体交换 + 还原消除 |
| **Koser 试剂 PhI(OH)OTs** | I(III) | α-甲苯磺酰氧基化 | SN2-Koser |
| **PhI=O（碘酰苯）** | I(III) | 烯氧化、Mn-催化 | 与金属合作 |
| **二芳基碘鎓 Ar₂I⁺X⁻** | I(III) | α-芳基化 | 还原消除 |
| **Togni 试剂 1/2** | I(III) | 亲电三氟甲基化 | 配体交换 + 还原消除 |
| **PhI=NTs**（碘鎓亚胺） | I(III) | 氮宾转移 | 配体转移 |
| **IBX**（2-碘酰基苯甲酸） | I(V) | 醇氧化，加热苄位 C-H | 5元环协同消除 |
| **DMP** | I(V) | 醇氧化，室温 | SN2-Si + α-H 拔取 |
| **H₅IO₆ / HIO₄** | I(VII) | cis-二醇裂解 | 环状碘酸酯过渡态 |

## 六、适用条件与限制

### 6.1 适用条件
- 多数 λ³ 试剂在中性或弱酸/碱条件下使用
- λ⁵ 试剂（IBX/DMP）需无水有机溶剂
- λ⁷（HIO₄）水相为主，温和加热

### 6.2 限制
- **稳定性问题**：IBX 易爆（受热/撞击），需 SIBX 或 mIBX 替代；
- **价格问题**：λ³ 试剂便宜，λ⁵ 试剂（DMP）较贵；
- 含富电子芳烃可能被双重氧化；
- 含硫底物可能干扰 λ³ 试剂的配体交换。

## 七、常见比较与易混点

### 7.1 与过渡金属催化的对比

| 对比项 | 高价碘 | 过渡金属催化 |
|---|---|---|
| 中心原子 | 主族 I | Pd、Cu、Rh 等 |
| 反应步骤 | 氧化加成、还原消除、配体交换 | **完全相同** |
| 价态变化 | I(III) ↔ I(I)，I(V) ↔ I(III) | M(0) ↔ M(II)，M(II) ↔ M(IV) |
| 重金属污染 | 无 | 有 |
| 成本 | 中等 | 高 |
| 选择性 | 中等 | 高（手性催化） |

> **机理类比**：I(III) → I(I) 类似 Pd(II) → Pd(0)；I(V) → I(III) 类似 Pd(IV) → Pd(II)。这是"主族版的过渡金属化学"。

### 7.2 不同高价碘的"软硬"对比
- **λ³**：相对软，易被烯醇、硫醇、磷化氢攻击；
- **λ⁵**：相对硬，主要被醇 OH 进攻；
- **λ⁷**：很硬，主要进攻 1,2-二醇这种"双 OH"配位体。

### 7.3 容易混淆的"高价碘"
- $\ce{IO3^-}$ vs $\ce{IO4^-}$：前者 I(V)，后者 I(VII)；
- **IBX vs HIO₄**：IBX 是芳基有机碘 I(V)；HIO₄ 是无机碘 I(VII)。
- Togni 试剂 vs Ruppert-Prakash 试剂：前者亲电 -CF₃；后者亲核 -CF₃⁻。

## 十、例题

### 例题 1：Koser 试剂机理
**题目：** 写出乙酸苯乙酮 PhCOMe 与 Koser 试剂 PhI(OH)OTs 的反应机理。

**分析：**
- 第 1 步：PhCOMe 的烯醇式 PhC(OH)=CH₂ 作为亲核体进攻 I(III) → α-C-I(III) 中间体（OTs 仍在 I 上）
- 第 2 步：OTs⁻ 反向 SN2 进攻 α-C，PhI 离去（I(III) → I(I)）→ PhCOCH₂OTs

**反思：**
碘"接受" α-C，再"传递"OTs，自身从 I(III) → I(I) 还原 2 价，等价于 2 电子氧化 α-C。

### 例题 2：试剂选择
**题目：** 若要将一级醇 RCH₂OH 转化为：(a) 醛 RCHO；(b) 羧酸 RCOOH；(c) α-OTs 酮（先酮化再 α-OTs）。选择合适的高价碘试剂。

**分析：**
- (a) 醛：DMP（[[Dess-Martin氧化]]），室温 CH₂Cl₂
- (b) 羧酸：DMP 不会过氧化，需用 Jones 或 RuO₄；高价碘单独不能给羧酸
- (c) α-OTs 酮：先 DMP 醛化，再 Jones 酮化，再 Koser 试剂 α-OTs（或直接氧化为酮再 Koser）

### 例题 3：Togni 试剂机理
**题目：** 写出 RSH + Togni 试剂 → RSCF₃ 的可能机理。

**解答：**
RSH 的 S 作为软亲核体进攻 Togni 试剂的 I(III) → 配体交换，得 RS-I(III)-CF₃ 中间体；S→C(CF₃) 还原消除（迁移），得 RSCF₃ + ArI。

## 十一、易错点
- ❌ 误以为 λ³-iodane 的"3"指 3 个 OAc：**指碘 +3 价、3 配位**。
- ❌ 把 IBX 与 HIO₄ 混为一谈：IBX 是 **有机** I(V)（含芳基 + COOH）；HIO₄ 是无机 I(VII)。
- ❌ 误以为高价碘像金属一样"催化"：实际是 **化学计量** 试剂，自身被消耗（除非有"碘催化"循环用 m-CPBA 再氧化）。
- ❌ 把"碘鎓盐 Ar₂I⁺"当成 λ³-iodane 的别称：碘鎓盐是 **特殊的 λ³**，配体两侧均为 Ar。
- ❌ 忘记 λ⁷ 也算高价碘：HIO₄ 是 **决赛级 I(VII)**，与 IBX 是同家族。

## 十三、竞赛拓展
- **碘催化**：使用催化量 PhI + 化学计量 m-CPBA / Oxone 可循环再生 PhI(OAc)₂ 类试剂；
- **手性 λ³-碘烷**：含手性骨架的 λ³ 试剂可用于不对称 α-官能化（Wirth, 2013）；
- **聚合载体型 λ³**：Poly[hydroxy(tosyloxy)iodo]styrene 是 Koser 的固载版，便于回收；
- **Hofmann 重排** 中 PhI(OAc)₂ 替代 NaOBr，是温和现代版本；
- **"仿生"碘化学**：海洋天然产物中的高价碘合成；
- 与 [[β-硅基效应]]、[[硅基保护基]] 联用：现代多步合成中的常配。
