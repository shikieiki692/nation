# 主族元素无机箭头推动法

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]

## 一、定义
**主族元素无机箭头推动法（Inorganic Arrow Pushing）**：将有机化学中已成熟的电子流动表示法（curved-arrow notation）系统地推广到主族元素无机反应机理上。每条箭头从**电子源（孤对/键对）**指向**电子缺失位置（亲电中心或离去基）**，从而把"描述性的无机反应"转化为"机理性的电子流动过程"。

> **核心口号**：**Think Like a Lone Pair!**（"像一对孤对电子那样思考"）——如果你是一对孤对电子，你会进攻哪里？

## 二、考纲对应
- 对应考纲条目：[[13-元素化学]]、[[31-有机反应的本质]]、[[32-有机反应机理]]
- 所属模块：[[基础要求-无机和结构化学]]
- 本知识点在考纲中的作用：作为决赛级别"机理推断题"的统一思维工具，使主族元素的反应不再依赖死记，而能从电子结构出发推导。

## 三、核心原理

### 3.1 箭头规则（与有机相同）
1. 双钩箭头 (full-headed arrow) → 表示**两个电子**的流动
2. 单钩箭头 (half-headed/fishhook) → 表示**单个电子**的流动（自由基反应）
3. 箭头**起点**：电子源（孤对、σ 键、π 键）
4. 箭头**终点**：电子缺失中心（空轨道、σ* / π* 反键、可弱化的键）

### 3.2 主族无机反应的常见模式
| 模式 | 描述 | 典型实例 |
|---|---|---|
| 亲核进攻空轨道 | 孤对 → 缺电子的 B、Al、Si、P、S | $\ce{NH3 + BF3 -> H3N->BF3}$ |
| 亲电体接受电子 | π 键/孤对 → 进攻 H⁺、卤鎓 | 烯烃溴化、卤化氢加成 |
| 缔合-离解 | 加成形成高配位中间体后离解 | $\ce{SiF4 + F- -> [SiF5]- -> ...}$ |
| 协同 σ 键复分解 | 两个键同时断裂、生成 | $\ce{HCl + Si-H -> Si-Cl + H2}$ |
| 1,2-迁移 | 邻位基团带键对迁向缺电子原子 | 硼烷重排、卡宾插入 |

### 3.3 与有机反应的统一性
- 把 C 视为一种"碳原子亲电体/亲核体"，把 Si、P、S、Xe 视为"重元素亲电中心"
- 把 OH⁻、Cl⁻、CN⁻ 视为亲核体——和有机反应中完全相同
- HSAB 原理（[[软硬酸碱理论]]）告诉我们**哪一对电子流动更容易实现**

## 四、关键结论

### 4.1 与"经典 SN2"的关键区别
- 第二周期 C 的 SN2：**协同**取代（背攻 + 离去同步）
- 第三周期及以下（Si, P, S, Cl, Br, I, Xe）的"SN2"：通常**两步**——先加成生成高价中间体，再离解。详见 [[SN2-Si机理]]。

### 4.2 反应分类的统一框架
本书把所有主族反应归并到几大模式：
1. **酸碱（Lewis）反应**——最基础的"加成/离解"
2. **极性反应**——亲核体进攻亲电体
3. **氧化还原反应**——氧化加成 / 还原消除
4. **自由基反应**——单电子箭头
5. **协同反应**——周环、σ-bond metathesis

### 4.3 教学建议中的 Lewis 结构选择
- 优先用**带形式电荷的离子结构**而非"多重键结构"
- 例：$\ce{SOCl2}$ 写作 $\ce{Cl-S(+)(Cl)-O(-)}$ 比 $\ce{Cl-S(=O)-Cl}$ 更利于电子流动分析
- 优势：清晰显示孤对位置和形式电荷，使箭头起点更直观

## 五、常见分类或情形

### 5.1 极性反应（占主流）
- **亲核加成**到缺电子主族中心：$\ce{R3N: + BX3}$
- **亲核取代**（SN2-Si 型）：$\ce{F- + SiF4 -> [SiF5]- -> ...}$
- **β-消除**：$\ce{tBuO- + R3SiCH2CH2X -> R3Si-OtBu + CH2=CH2 + X-}$（E2-Si）
- **质子转移（PT）**：所有 Brønsted 酸碱反应

### 5.2 高价中间体
- 第三周期及以下原子的反应几乎都通过**高配位过渡态/中间体**进行
- 这是无机反应区别于经典有机的最关键特征

### 5.3 复分解（σ-Bond Metathesis）
- 两键同时断裂、生成的协同四中心过渡态
- 例：$\ce{R-M + H-X -> R-H + M-X}$（[[配体交换反应]]）

## 六、适用条件与限制

### 适用范围
- ✓ 主族元素（s/p 区）的极性反应、Lewis 酸碱反应
- ✓ 包含极少 d 轨道参与的高价化合物的成键解释
- ✓ 决赛级"机理推断题"

### 不适用或需谨慎
- ✗ 经典晶体学问题（与机理无关）
- ✗ 单纯描述性现象（颜色、晶系）
- ⚠ 真正的过渡金属反应——需配合金属有机化学的 16/18 电子规则
- ⚠ 强相对论效应体系（如 Pb²⁺、Bi³⁺）需结合 [[惰性对效应]]

## 七、常见比较与易混点

### 1. "无机箭头" vs "有机箭头"
- 规则**完全相同**——双钩箭头表两电子流动
- 不同点：**起点更多样**（孤对、σ 键、π 键，连金属-金属 σ 键也可作起点）

### 2. 离子结构 vs 多重键结构
| 结构 | 适合 | 不适合 |
|---|---|---|
| 离子（带形式电荷） | 机理分析、电子流动 | Lewis 结构最简化时 |
| 多重键 | 简明传达连接性 | 看不出电子流动起点 |

### 3. d 轨道参与的旧错误
- **旧观点**：$\ce{PF5}$、$\ce{SF6}$ 通过 sp³d、sp³d² 杂化使用 3d 轨道
- **新观点**：实际是 **3 中心 4 电子键（3c-4e）**，详见 [[高价化合物的成键]]

## 十、例题
### 例题 1：$\ce{SiF4 + 2F- -> [SiF6]^{2-}}$ 的机理
**分析：**
- 第一步：$\ce{F-}$ 孤对作箭头起点 → $\ce{Si}$ 的反键空轨道（A 过程，缔合）→ 形成五配位 $\ce{[SiF5]-}$
- 第二步：第二个 $\ce{F-}$ 同样进攻 → 形成八面体 $\ce{[SiF6]^{2-}}$
**反思：** 这是典型的两步缔合，没有协同的 SN2 取代。

### 例题 2：$\ce{NH3 + BF3 -> H3N\bond{->}BF3}$ 的机理
**分析：**
- $\ce{NH3}$ 上 N 的孤对 → $\ce{BF3}$ 中 B 的空 p 轨道
- 单箭头表示 Lewis 加成
**反思：** 这是 HSAB 中"硬-硬"匹配的典型实例。

## 十一、易错点
- 把箭头从原子画起（错）——必须从**电子**画起
- 忽视形式电荷——画错起点
- 把 SN2-Si 当作经典 SN2——会丢掉中间体
- 只盯着多重键，忽视孤对的作用

## 十二、竞赛拓展
- **FLP（受阻 Lewis 对）**：空间位阻使 Lewis 酸碱不能形成加合物，但可协同活化 H₂、CO₂ 等小分子（决赛热点）
- **碳硼烷的反应**：把"亲核体进攻碳硼烷簇"也用箭头描述
- **金属有机催化循环**：氧化加成、迁移插入、还原消除——每一步都可用箭头推动
