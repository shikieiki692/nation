# 旋转轴

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[14-对称性]]

## 一、定义
**n 重旋转轴 Cₙ**：绕轴旋转 360°/n 后分子复原。**主轴** = n 最大的轴（通常选为 z 轴）。

- H₂O: C₂（唯一）
- NH₃: C₃（主轴）
- BF₃: C₃（主轴）+ 3C₂（垂直主轴）
- 苯: C₆（主轴）+ 6C₂

## 二、考纲对应
- [[14-对称性]]：旋转轴作为基本对称元素

## 三、核心原理
### 晶体中的旋转轴限制
在三维周期性晶体中，由于**点阵平移对称性**的限制，只有以下轴次的旋转轴能够存在：
- **C₁, C₂, C₃, C₄, C₆**（1, 2, 3, 4, 6 重轴）
- **C₅ 和 n > 6 的旋转轴不可能出现在晶体中**

### 晶体中旋转轴与晶系的关系（表 13.1）
各晶系的特征对称元素以旋转轴为核心：
| 晶系 | 特征旋转轴 |
|------|------|
| 立方 | 沿体对角线有 **4 个 C₃** |
| 六方 | **1 个 C₆** |
| 四方 | **1 个 C₄** |
| 三方 | **1 个 C₃** |
| 正交 | **3 个互相垂直的 C₂**（或等价对称面） |
| 单斜 | **1 个 C₂**（或 1 个对称面） |
| 三斜 | 无旋转轴（仅有 C₁，即恒等操作） |

## 四、关键结论
- 分子中：Cₙ 轴可以取任意 n 值（如二茂铁交错构型有 C₅）
- 晶体中：Cₙ 严格限制为 n = 1, 2, 3, 4, 6
- 特征旋转轴是划分**晶系**的核心判据——需综合旋转轴约束判断晶系归属
