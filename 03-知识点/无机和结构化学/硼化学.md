# 硼化学（Boron Chemistry）

## 一、定义

硼（B）是第13族（硼族）的第一个元素，是典型的**缺电子原子**。硼的化合物以多中心键、簇状结构和独特的Lewis酸性为核心特征，在有机合成（如硼氢化反应、Suzuki偶联）和材料科学（如硼氮化物、硼簇）中占据重要地位。

## 三、核心原理

### 缺电子与多中心键

- 硼只有3个价电子，但倾向于形成4个键（如BF₄⁻）
- 常见多中心键：
  - **3c-2e键**：如乙硼烷B₂H₆中的桥氢
  - **3c-4e键**：如BeCl₂中的桥氯（与Be类似）
- **Styx数**：用于描述硼烷结构（B₂H₆为2002型）

### 硼烷（Boranes）

- **closo-硼烷**（BₙHₙ²⁻）：闭合多面体，n个顶点
- **nido-硼烷**（BₙHₙ₊₄）：缺一顶点的closo结构
- **arachno-硼烷**（BₙHₙ₊₆）：缺两顶点的closo结构
- **Wade规则**：通过骨架电子对数预测硼烷结构类型

#### Wade规则详解

- 骨架电子对数 = n + 1 → closo
- 骨架电子对数 = n + 2 → nido
- 骨架电子对数 = n + 3 → arachno
- 每个BH单元贡献2个骨架电子；额外H各贡献1个电子；负电荷计入骨架电子

#### 碳硼烷与金属硼烷

- **碳硼烷**：CH与BH等叶瓣等价，Wade规则同样适用（如C₂B₅H₇为五角双锥closo）
- **金属硼烷**：M─B直接键合，金属可取代多面体顶点

### 重要的硼化合物

#### 三氟化硼（BF₃）
- **Lewis酸性**：强Lewis酸，常用于催化
- **结构**：平面三角形，sp²杂化
- **配位**：易形成BF₄⁻（四面体）

#### 乙硼烷（B₂H₆）
- **结构**：非BH₃的二聚体，含两个桥氢
- **桥氢**：形成B─H─B 3c-2e键
- **端氢**：正常的2c-2e B─H键

#### 硼氢化钠（NaBH₄）
- **还原性**：温和还原羰基化合物
- **选择性**：不还原酯、酰胺（与LiAlH₄不同）

#### 9-BBN
- **大位阻硼烷**：用于区域选择性的硼氢化反应
- **反马氏加成**：对烯烃的硼氢化遵循反马氏规则

### 硼氮化合物
- **氮化硼（BN）**：
  - 类石墨结构：层状，白色润滑剂
  - 类金刚石结构（立方BN）：超硬材料
- **硼氮杂环**：与碳杂环的等电子关系

## 十一、易错点

1. **BF₃的Lewis酸性反常**：由于F→B的π反馈键，BF₃实际Lewis酸性弱于BCl₃和BBr₃
2. **硼氢化反应的区域选择性**：H加在取代较多的碳上（反马氏），B加在取代较少的碳上
3. **Wade规则的应用范围**：不仅适用于硼烷，也适用于碳硼烷、金属硼烷
4. **Wade规则中的n**：对于BₙHₙ₊₄，n = B原子数 = 顶点数；碳硼烷中n为总顶点数（B + C）
5. **缺电子不等于不稳定**：硼的缺电子特性正是其丰富化学的根源

## 十三、竞赛拓展

### 决赛考查要点

1. **多中心键的识别与计数**：区别3c-2e与3c-4e键
2. **硼烷的结构类型**：运用Wade规则判断closo/nido/arachno
3. **Lewis酸性排序**：BBr₃ > BCl₃ > BF₃（反常的π反馈键效应）
4. **硼氢化反应的机理与立体化学**：顺式加成、反马氏规则
5. **硼在Suzuki偶联中的作用**：RB(OH)₂作为亲核试剂
6. **骨架电子计数**：根据化学式判断硼烷/碳硼烷结构类型
