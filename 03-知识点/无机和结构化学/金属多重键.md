## 一、基本定义

**金属-金属多重键**是指两个过渡金属原子之间通过d轨道的**σ、π、δ**重叠形成的共价多重键，键级可达4（四重键）甚至5（五重键）。与有机化学中C—C多重键由s和p轨道组成不同，M—M多重键的核心成键轨道来自金属的**d轨道**。

- **历史起点**：第一个被确认含M—M键的物种是 $\mathrm{Hg_2^{2+}}$（如 $\mathrm{Hg_2Cl_2}$ 中的 $\mathrm{Hg}$—$\mathrm{Hg}$ 单键，但d轨道不参与成键，仅s轨道形成σ单键）
- **里程碑**：$[\mathrm{Re_2Cl_8}]^{2-}$ 是最早被确认含**四重键**的络合物（Cotton, 1964），其重叠构象证明了δ键的存在
- **新前沿**：2005年首次分离出含**五重键**的稳定络合物（Cr(I)二聚体，Cr—Cr距离仅183.5 pm）

## 二、核心内容

### 2.1 σ/π/δ成键框架

**坐标系设定**：设两个金属原子沿 **z 轴**方向靠近形成 M—M 键。取 z 轴为重叠方向，根据 d 轨道的空间取向，可以系统分析各轨道如何重叠形成 σ、π、δ 键。

**推导步骤**：

1. **σ 键**：$\mathrm{d_{z^2}}$ 轨道沿 z 轴呈"头碰头"（端对端）重叠 → 形成最强的 σ 成键/反键轨道对
2. **π 键**：$\mathrm{d_{xz}}$ 和 $\mathrm{d_{yz}}$ 轨道在 xz/yz 平面内"肩并肩"重叠 → 形成两对简并的 π 成键/反键轨道
3. **δ 键**：$\mathrm{d_{xy}}$ 和 $\mathrm{d_{x^2-y^2}}$ 轨道在 xy 平面内"面对面"**四瓣重叠** → 形成两对 δ 成键/反键轨道。δ 键是 M—M 多重键区别于 C—C 多重键的**标志性特征**——C—C 键体系中没有 δ 键的类似物

| 键类型 | 参与d轨道 | 重叠方式 | 成键轨道数 | 反键轨道数 | 类比有机键 |
|:---:|:---|:---|:---:|:---:|:---:|
| **σ** | $\mathrm{d_{z^2}}$ + $\mathrm{d_{z^2}}$ | 头对头（端对端）| 1 | 1 | C—C σ键 |
| **π** | $\mathrm{d_{xz}}$ + $\mathrm{d_{xz}}$，$\mathrm{d_{yz}}$ + $\mathrm{d_{yz}}$ | 肩并肩（侧面）| 2 | 2 | C—C π键 |
| **δ** | $\mathrm{d_{xy}}$ + $\mathrm{d_{xy}}$，$\mathrm{d_{x^2-y^2}}$ + $\mathrm{d_{x^2-y^2}}$ | 面对面（四瓣）| 2 | 2 | **无有机类似物** |

δ键是M—M多重键区别于C—C多重键的标志性特征。δ键的形成要求两个 $\mathrm{d_{xy}}$（或 $\mathrm{d_{x^2-y^2}}$）轨道**面对面平行排列**，重叠积分较小，因此δ键远弱于σ和π键。δ键将络合物**锁定在重叠构象**，这一构象从空间位阻角度看是不利的——这正是δ键存在的强有力实验证据。

**能量排序与完整MO能级**：由于 d 轨道重叠积分的大小差异，成键强度排序为 $\sigma > \pi > \delta$。将两金属原子所有 d 轨道组合后，简单的双原子定性分子轨道从低到高为：

$$
\sigma\ \pi\ \pi\ \delta\ \delta\ \delta^*\ \delta^*\ \pi^*\ \pi^*\ \sigma^*
$$

其中 δ/δ* 能隙最小——δ 键是最弱的 M—M 多重键组分，δ* 容易被额外电子占据，导致键级逐步下降。这就是为什么 $\sigma^{2}\pi^{4}\delta^{2}$（键级=4）→ $\sigma^{2}\pi^{4}\delta^{2}\delta^{*1}$（键级=3.5）→ $\sigma^{2}\pi^{4}$（键级=3）的键级递减是通过逐步占据 δ* 和 π* 实现的。

### 2.2 分子轨道能级图

M—M相互作用的近似MO能级（从低到高）：

$$\sigma < \pi < \delta < \delta^* < \pi^* < \sigma^*$$

当配体使用 $\mathrm{d_{x^2-y^2}}$ 轨道参与M—L成键时，$d_{x^2-y^2}$ 不参与M—M δ成键，此时只有 $\mathrm{d_{xy}}$ 轨道形成一对δ/δ*，最大键级为4。

### 2.3 四重键（Quadruple Bond）

- **电子组态**：$\sigma^{2}\pi^{4}\delta^{2}$
- **要求**：每个金属提供4个d电子，即 $d^4$ 组态的中心原子
- **经典实例**：

**$[\mathrm{Re_2Cl_8}]^{2-}$**（四棱柱结构，重叠构象）：
- Re(III)，$d^4$，组态 $\sigma^{2}\pi^{4}\delta^{2}$
- Re—Re距离：224 pm（vs. 金属Re中约275 pm）
- $\mathrm{d_{x^2-y^2}}$ 轨道用于与Cl配体成键，不参与M—M键

**$[\mathrm{Mo_2(O_2CCH_3)_4}]$**（四重键合醋酸钼(II)）：
- 制备：$2[\mathrm{Mo(CO)_6}] + 4\mathrm{CH_3COOH} \rightarrow [\mathrm{Mo_2(O_2CCH_3)_4}] + 2\mathrm{H_2} + 12\mathrm{CO}$
- 是制备其他Mo—Mo化合物的优良起始物
- 可转化为四重键合氯络合物：$[\mathrm{Mo_2(O_2CCH_3)_4}] + 4\mathrm{H^+} + 8\mathrm{Cl^-} \rightarrow [\mathrm{Mo_2Cl_8}]^{4-} + 4\mathrm{CH_3COOH}$

### 2.4 五重键（Quintuple Bond）

- **电子组态**：$\sigma^{2}\pi^{4}\delta^{4}$
- **要求**：每个金属提供5个d电子，即 $d^5$ 组态，且无桥配体占用成键d轨道
- **经典实例**：Cr(I)二聚体（18），两个Cr原子距离仅**183.5 pm**（大块金属Cr中为258 pm），无桥配体支撑
- 五重键的成键d轨道全部填满（$\sigma^{2}\pi^{4}\delta^{4}$），所有5个d电子均投入M—M成键

### 2.5 键级判断方法 —— 通过电子组态

键级 = (成键电子数 − 反键电子数) / 2：

| 组态 | 键级 | M—M键长趋势 | 典型实例 |
|:---|:---:|:---|:---|
| $\sigma^{2}\pi^{4}\delta^{2}$ | **4** | 最短 | $[\mathrm{Re_2Cl_8}]^{2-}$ (224 pm) |
| $\sigma^{2}\pi^{4}\delta^{1}$ | **3.5** | | $[\mathrm{Tc_2Cl_8}]^{2-}$ (217 pm) |
| $\sigma^{2}\pi^{4}$ | **3** | | $[\mathrm{Mo_2(SO_4)_4}]^{4-}$ (222 pm) |
| $\sigma^{2}\pi^{4}\delta^{2}\delta^{*1}\pi^{*2}$ | **2.5** | | |
| $\sigma^{2}\pi^{4}\delta^{2}\delta^{*2}\pi^{*2}$ | **2** | | |
| $\sigma^{2}\pi^{4}\delta^{2}\delta^{*2}\pi^{*3}$ | **1.5** | | |
| $\sigma^{2}\pi^{4}\delta^{2}\delta^{*2}\pi^{*4}$ | **1** | 最长 | |

键级下降的原因：δ*被占据（δ键是弱键）→ π*被占据 → 键级逐步降至1。

### 2.6 多重键的反应性

M—M多重键是有机金属反应的枢纽。与C—C多重键的加合反应类似但产物结构更丰富：
- HI可加合于 $\mathrm{Mo}\equiv\mathrm{Mo}$ 三键，形成含3c,2e M—H—M桥和I桥的产物
- $[\mathrm{Pt(PPh_3)_4}]$ 可加合于Mo≡Mo三键，失去2个PPh₃配体生成三金属簇

## 三、竞赛要点

1. **d轨道重叠方式**：能画出σ（$\mathrm{d_{z^2}}$ 头对头）、π（$\mathrm{d_{xz}/d_{yz}}$ 肩并肩）、δ（$\mathrm{d_{xy}}$ 面对面）的重叠示意图
2. **键级定量计算**：给定电子组态 → 数出成键和反键电子数 → 键级 = (成键 − 反键) / 2
3. **构象与δ键锁定**：$[\mathrm{Re_2Cl_8}]^{2-}$ 采取重叠构象（空间不利）是δ键存在的直接证据
4. **d电子数与键级上限**：$d^4$ 中心（四重键上限，因$\mathrm{d_{x^2-y^2}}$被配体占用）；$d^5$ 且无桥配体（五重键上限）
5. **δ键特性**：δ键是弱键，δ/δ*能隙小，δ*容易被占据导致键级下降
6. **例题19.2 经典判断**：$[\mathrm{Hg_2}]^{2+}$的M—M键级。Hg(I)为 $d^{10}s^1$，d轨道全满不参与成键，仅6s轨道形成σ单键，键级=1

## 四、例题

### 例题 1：键级与电子组态（改编自例题19.2）

**题目**：$[\mathrm{Hg_2}]^{2+}$ 中存在M—M键，但其d轨道不参与成键。请写出成键电子组态并计算键级。若 $[\mathrm{Mo_2(SO_4)_4}]^{4-}$ 的组态为 $\sigma^{2}\pi^{4}$，其键级是多少？是否含δ键？

**分析**：Hg(I)为 $d^{10}s^1$，d¹⁰全满不参与M—M成键，仅6s¹形成σ单键。Mo(II)络合物中Mo为$d^4$，但仅有σ和π成键轨道被占据，δ轨道未填充。

**解答**：
- $[\mathrm{Hg_2}]^{2+}$：成键组态为 $\sigma^{2}$（来自6s-6s重叠），键级 = (2−0)/2 = **1**（单键），d轨道不参与。
- $[\mathrm{Mo_2(SO_4)_4}]^{4-}$：组态 $\sigma^{2}\pi^{4}$，键级 = (6−0)/2 = **3**（三重键）。不含δ键（δ轨道未被占据，可能是配体场或氧化态使δ轨道能量过高）。

**反思**：M—M键级不一定需要d轨道全部参与——Hg₂²⁺是纯s轨道σ键的极端例子。判断键级的关键是识别哪些轨道实际参与成键。

### 例题 2：组态与键长关联

**题目**：$[\mathrm{Re_2Cl_8}]^{2-}$（组态 $\sigma^{2}\pi^{4}\delta^{2}$）的Re—Re键长为224 pm。另一Re(III)二聚体组态为 $\sigma^{2}\pi^{4}\delta^{2}\delta^{*1}$，请预测其Re—Re键长的变化趋势并给出理由。

**分析**：δ*被部分占据 → 降低了δ成键的净效果 → 键级从4降至3.5 → 键长增加。

**解答**：组态 $\sigma^{2}\pi^{4}\delta^{2}\delta^{*1}$ 的键级 = (8−1)/2 = **3.5**。由于δ成键贡献减弱，Re—Re键间电子密度降低，键长应**增加**（约217 pm，参见表B19.1中Tc类似物数据）。

**反思**：δ*占据对键级的影响仅为0.5，但由于δ键本身很弱，δ*反键效应相对显著——这是M—M多重键化学的特有现象。

## 五、易错点

1. **混淆δ键与d-pπ键**：δ键是d-d轨道面对面重叠（四瓣对四瓣），不是d-p的π型重叠。C—C键体系中完全没有δ键的类似物。
2. **认为所有$d^4$中心都形成四重键**：必须考虑配体占用了哪些d轨道。如果 $\mathrm{d_{x^2-y^2}}$ 被M—L σ键占用，δ成键只剩$\mathrm{d_{xy}}$一对，最大键级为4。但如果配体也占用$\mathrm{d_{xy}}$，则可能无法形成δ键。
3. **Hg₂²⁺的键级判断**：学生常因Hg是"过渡金属"而认为d轨道参与成键。事实上Hg(I)为$d^{10}s^1$，d¹⁰全满不参与成键，只有6s¹形成σ单键，键级=1。
4. **键级与键长的非线性关系**：由于δ键很弱，三重键→四重键的键长缩短远小于双重键→三重键的缩短。三重键络合物和四重键络合物的键长常常相近。
5. **重叠构象vs.交错构象**：学生常见误区是认为交错构象空间位阻更小因此更稳定。但$[\mathrm{Re_2Cl_8}]^{2-}$采取重叠构象正是δ键存在的证据——δ键只能在$\mathrm{d_{xy}}$轨道共面时形成，其能量收益超过了空间位阻代价。

### 8.2 认知误区与纠正

- **误区**：学生认为"金属之间只有金属单键，不可能存在多重键"。
- **纠正**：$\mathrm{[Re_2Cl_8]^{2-}}$ 是最经典的反例，其中两个 Re 原子之间形成四重键（$\sigma + 2\pi + \delta$），键级为4，Re–Re键长仅224 pm，远短于Re金属中的原子间距（约275 pm）。金属-金属多重键的存在已被X射线晶体学充分证实，是金属簇合物化学的核心概念之一。

### 8.3 入门例题

> **例题**：为什么 $\mathrm{[Re_2Cl_8]^{2-}}$ 采取重叠式构型而非交错式构型？
>
> **解答**：$\mathrm{[Re_2Cl_8]^{2-}}$ 中Re–Re键为四重键（$\sigma + 2\pi + \delta$），其中 δ 键由两个 $\mathrm{d}_{xy}$ 轨道的侧面重叠形成。只有在**重叠式（eclipsed）构型**下，两个 $\mathrm{d}_{xy}$ 轨道的瓣才能正对，实现有效的 δ 重叠；若采取交错式（staggered）构型，两个 $\mathrm{d}_{xy}$ 轨道相对旋转45°，δ 重叠为零，键级将从4降为3。尽管重叠式构型会增加配体-配体之间的排斥，但 δ 键的形成带来的能量收益足以补偿这一不利因素，因此分子整体采取重叠式构型。
