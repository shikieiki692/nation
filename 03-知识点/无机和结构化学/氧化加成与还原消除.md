# 氧化加成与还原消除

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[12-配合物]]、[[决赛11-金属有机化学]]

## 一、定义

### 氧化加成（Oxidative Addition, OA）
一个分子（A-B）加合到金属或主族中心，**两键变三键**（一个 A-B 断裂，两个 M-A 和 M-B 形成）。中心原子的**氧化态升高 +2**，配位数增加 +2。

通式：
$$\ce{LnM + A-B -> LnM(A)(B)} \quad \text{(M 氧化态 +2)}$$

### 还原消除（Reductive Elimination, RE）
氧化加成的逆过程：金属上的两个配体（A 和 B）组合成 A-B 并离开，金属**氧化态降低 −2**，配位数 −2。

通式：
$$\ce{LnM(A)(B) -> LnM + A-B} \quad \text{(M 氧化态 -2)}$$

- 本知识点在考纲中的作用：决赛"金属催化循环"的核心步骤——几乎所有过渡金属催化都涉及 OA + RE。

## 三、核心原理

### 3.1 OA 的几种机理
1. **协同（concerted）机理**：三中心过渡态，**保留 A-B 立体化学**（如 H₂、Si-H 等非极性键）
2. **SN2 机理**：背攻取代，**反转**立体化学（如 Ar-X 中的 X 离去）
3. **自由基机理**：一步加一个原子（如 Pt 与 RX 中的 R 自由基）
4. **离子（Ionic）机理**：先解离生成 A⁻/B⁺，再分别加成（适用于强极性键）

### 3.2 OA 反应的电子流动（用箭头推动）
对于协同 OA：
- A-B 的 σ 键 → M（金属空轨道）
- 同时 M 上的 d 电子对 → A-B 的 σ*（反键）
- 结果：A-B 断裂，M-A 和 M-B 形成

### 3.3 适合 OA 的金属
| 金属 | 配置 | 实例 |
|---|---|---|
| Vaska 配合物 IrCl(CO)(PPh₃)₂ | d⁸（16e） | $\ce{H2}$、$\ce{O2}$ 加成 |
| Pd⁰(PPh₃)₄ | d¹⁰（18e） | C-X 键加成 |
| Pt⁰、Ni⁰ | d¹⁰ | C-X 键加成 |
| Rh⁺(PPh₃)₃Cl | d⁸（Wilkinson） | $\ce{H2}$ 加成 |

### 3.4 RE 反应的条件
- 中心金属高氧化态（M^(n+2)）
- 两个相邻配体（cis 位）
- 离去配体可形成稳定 A-B 键
- 金属配位饱和（>16 e）则更倾向 RE 释放空位

### 3.5 16/18 电子规则的判读
- 16 e ↔ 18 e 之间的振荡 = 催化循环
- OA：从 16 e（d⁸）→ 18 e（d⁶）
- RE：从 18 e（d⁶）→ 16 e（d⁸）

## 四、关键结论

### 4.1 OA + RE = 催化循环
经典 Pd 催化的 Suzuki 偶联：
1. **OA**：$\ce{Pd^0L4 + ArX \to Pd^{II}(Ar)(X)L2}$
2. **配体交换**：与 ArB(OH)₂ 反应
3. **转金属化**：$\ce{Pd^{II}(Ar)(Ar')L2}$
4. **RE**：得 $\ce{Ar-Ar' + Pd^0L2}$
5. 循环

### 4.2 OA-RE 中"双向"的意义
- OA 和 RE 是同一过程的双向
- 哪一方向有利取决于：
  - 金属氧化态的稳定性
  - A-B 键能 vs M-A + M-B 键能总和
  - 配位场效应

### 4.3 主族化学中的 OA 类比
- $\ce{XeF2 + 2 X- \to Xe + X2 + 2F-}$ 可看作 Xe(II) 的"被还原" + 2F⁻ 的释放
- $\ce{PCl5 \to PCl3 + Cl2}$ 是磷的"还原消除"，Cl₂ 离开

## 五、常见分类或情形

### 5.1 H₂ 的 OA（最经典）
$$\ce{IrCl(CO)(PPh3)2 + H2 \to IrCl(CO)(H)2(PPh3)2}$$
- Vaska 配合物 Ir(I) → Ir(III)
- 协同机理，cis 加成
- 用于催化氢化反应

### 5.2 Ar-X 的 OA（Suzuki/Heck/Stille 偶联）
$$\ce{Pd(PPh3)4 + Ar-Br \to Pd(Ar)(Br)(PPh3)2 + 2 PPh3}$$
- Pd(0) → Pd(II)
- 通常 SN2 机理
- 用于交叉偶联反应

### 5.3 C-H 活化中的 OA
- 经典 Cp*Ir(III) + 烷烃 → Cp*Ir(V)(H)(R)
- 早期被认为是 σ-bond metathesis，但有些是真正的 OA

### 5.4 RE 中的"双中心 vs 单中心"
- 单中心 RE：常见，A 和 B 在同一金属上
- 双中心 RE：A 在 M₁，B 在 M₂（如金属簇催化剂）

## 六、适用条件与限制

### OA 的限制
- 金属需有可被氧化的电子对（d 电子）
- 配位数需有空位（< 18e）
- A-B 键能不能太高（如 N₂ 的 941 kJ/mol 极难 OA）

### RE 的限制
- 两个配体需 cis 位
- 反应需 16e 中间体作驱动力
- 部分配体（如 PR₃）位阻可阻止 RE

## 七、常见比较与易混点

### 1. OA vs σ-bond metathesis
| | OA | σ-bond metathesis |
|---|---|---|
| 氧化态变化 | +2 | 不变 |
| 中间体 | 18e 加合物 | 仅过渡态 |
| 适用金属 | d⁸/d¹⁰ 后过渡 | d⁰ 早过渡 |

### 2. OA vs Lewis 加成
- OA：同时改变氧化态（电子转移）
- Lewis 加成：仅形成配位键（电子方向：配体 → 金属）

### 3. RE vs 还原（reduction）
- RE：金属内的两个配体合并离开，金属氧化态降低
- 还原：外加还原剂使金属电子增加
- 完全不同概念

## 十、例题

### 例题 1：Vaska 配合物 + H₂
**题目：** 写出 $\ce{IrCl(CO)(PPh3)2 + H2 \to IrCl(CO)(H)2(PPh3)2}$ 的电子流动机理。
**分析：**
- 起始：Ir(I)，d⁸，16e
- H-H σ 键 → Ir 空 d 轨道
- 同时 Ir 上 d 电子对 → H-H σ*
**解答：** 协同三中心过渡态，cis 加成，Ir(I) → Ir(III)，氧化态 +2。

### 例题 2：Pd 催化 Suzuki 中的 RE
**题目：** Pd(II)(Ar)(Ar')L₂ → Ar-Ar' + Pd(0)L₂，用箭头表示电子流动。
**分析：** 两个 cis 的 Ar 之间形成 C-C 键，Pd 失去 2e。
**解答：**
- Pd-Ar σ 键 → 与另一 Ar 中的 C 间形成新 C-C 键
- 同时另一 Pd-Ar' σ 键断裂，电子回到 Pd
- 净结果：Pd(II) → Pd(0)，Ar-Ar' 释放

## 十一、易错点
- 把 OA 当作配位（OA 改变氧化态）
- 忽视 RE 中的 cis 要求——trans 配体不能 RE
- 把 OA-RE 等同于 σ-bond metathesis（机理不同）
- 用错电子数判断（OA 配位数 +2，电子数 +2）

## 十二、竞赛拓展
- **CO 插入反应**：与 RE 类似，但金属-碳键 + CO → 酰基
- **β-氢消除**：与 RE 类似但产物为烯烃 + 金属-氢
- **均相催化的"全图"**：OA + 配体交换 + 插入 + RE = 完整循环
- **2010 诺贝尔化学奖**：Heck、Negishi、Suzuki 偶联（基于 OA-RE）
