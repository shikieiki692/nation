# 镉

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]

## 一、概述
- 镉（Cd）位于锌副族中部，最常见的氧化态为 +2
- 在基础层，镉主要通过 Zn → Cd → Hg 纵向递变出现

## 二、核心要点
- Cd 的 hcp 结构、Cd2+ 离子及其配合物
- CdS 呈柠檬黄至橙黄色，在色彩显色方面有直观价值
- 镉有毒，教学时不一定需要深度展开
