# 硼族元素（B, Al, Ga, In, Tl）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]

## 一、定义
IIIA 族：ns²np¹。B 是非金属（缺电子特征），Al 是两性金属，Tl 有惰性电子对效应。

- B：缺电子化合物核心元素，形成多中心键
- Al：地壳含量最高金属元素
- Tl：Tl⁺(6s²) 比 Tl³⁺ 更稳定（惰性电子对）

## 二、考纲对应
- 对应考纲条目：[[13-元素化学]]
- 所属模块：[[基础要求-无机和结构化学]]
- 本知识点在考纲中的作用：承接缺电子结构、Lewis 酸碱与主族递变，是第二轮元素化学和结构化学的桥梁页之一。

## 三、核心原理

### 重要反应

### 硼化学（竞赛热门）
| 化合物 | 结构要点 |
|------|------|
| BF₃ | 平面三角形，Lewis 酸（缺电子 B），BF₄⁻ 四面体 |
| B₂H₆ | **3c-2e B—H—B 桥键**（经典案例） |
| H₃BO₃ | 硼酸，层状结构（氢键），一元弱酸，B(OH)₃ + H₂O ⇌ [B(OH)₄]⁻ + H⁺ |
| NaBH₄ | 强还原剂，选择性还原羰基 |
| Na₂B₄O₇·10H₂O | 硼砂，[B₄O₅(OH)₄]²⁻ |

### 硼砂珠试验
Na₂B₄O₇ + CoO → Co(BO₂)₂（蓝色），用于鉴定金属离子

### 铝
- 两性：2Al + 6H⁺ → 2Al³⁺ + 3H₂↑；2Al + 2OH⁻ + 6H₂O → 2[Al(OH)₄]⁻ + 3H₂↑
- Al(OH)₃ 两性：溶于酸 → Al³⁺，溶于碱 → [Al(OH)₄]⁻
- 铝热反应：2Al + Fe₂O₃ → Al₂O₃ + 2Fe

## 三、核心原理

### 3.1 硼的特殊性（对角规则 B-Si）
B 和同族的 Al 相比，一是非金属，一是金属，物理性质、化学性质都不相似，而与右下角的硅比较相似（对角规则）。B 可以有三配位（"$\mathrm{BO}_3$"平面三角形）和四配位（"$\mathrm{BO}_4$"四面体），类似于 $\mathrm{SiO}_4$ 四面体。

### 3.2 硼砂和硼酸
- 硼砂为四硼酸钠，阴离子为 $[\mathrm{B}_4\mathrm{O}_5(\mathrm{OH})_4]^{2-}$，结构中既有"$\mathrm{BO}_4$"四面体又有"$\mathrm{BO}_3$"平面三角形
- 热的浓硼砂溶液和过量 $\mathrm{HCl}$ 起反应，冷却后析出硼酸 $\mathrm{H}_3\mathrm{BO}_3$：$\mathrm{Na}_2\mathrm{B}_4\mathrm{O}_7 + 2\mathrm{HCl} + 5\mathrm{H}_2\mathrm{O} \longrightarrow 2\mathrm{NaCl} + 4\mathrm{H}_3\mathrm{BO}_3$
- 硼酸形式上似三元酸，实际是一元弱酸：$\mathrm{H}_3\mathrm{BO}_3 + \mathrm{H}_2\mathrm{O} \Longrightarrow \mathrm{B(OH)}_4^{-} + \mathrm{H}^{+}$，$K = 5.4 \times 10^{-10}$

### 3.3 硼烷和硼氮化合物
- 硼能与氢形成一系列硼烷：$\mathrm{B}_2\mathrm{H}_6$、$\mathrm{B}_4\mathrm{H}_{10}$、$\mathrm{B}_5\mathrm{H}_9$、$\mathrm{B}_5\mathrm{H}_{11}$、$\mathrm{B}_6\mathrm{H}_{10}$、$\mathrm{B}_{10}\mathrm{H}_{14}$ 等
- 硼烷在空气中燃烧生成 $\mathrm{B}_2\mathrm{O}_3$ 和 $\mathrm{H}_2\mathrm{O}$ 并大量放热
- 氮化硼 (BN) 的性质与石墨相似；$\mathrm{B}_3\mathrm{N}_3\mathrm{H}_6$ 和 $\mathrm{C}_6\mathrm{H}_6$ 性质相似，被称为"无机苯"

### 3.4 含氧酸的酸性规律（适用于同周期比较）
同一周期的几种最高氧化态含氧酸相比较：非羟基氧越多，R 电负性越大，氧化态越高，对羟基氧吸引力越大，酸性越大。
- $\mathrm{HClO}_4$（$\mathrm{(HO)ClO}_3$）$K_a = 1\times 10^{10}$
- $\mathrm{H}_2\mathrm{SO}_4$（$\mathrm{(HO)}_2\mathrm{SO}_2$）$K_{a_1} = 1\times 10^{3}$
- $\mathrm{H}_3\mathrm{PO}_4$（$\mathrm{(HO)}_3\mathrm{PO}$）$K_{a_1} = 7\times 10^{-3}$
- $\mathrm{H}_4\mathrm{SiO}_4$（$\mathrm{(HO)}_4\mathrm{Si}$）$K_{a_1} = 1\times 10^{-10}$

### 3.5 缩合酸
缩合酸的酸性总是大于原正酸（非羟基氧增多），如焦磷酸 $\mathrm{H}_4\mathrm{P}_2\mathrm{O}_7$ 的 $K_{a_1} = 3.2\times 10^{-2}$，大于正磷酸的 $K_{a_1} = 7\times 10^{-3}$。

## 四、关键结论
- B 是 p 区唯一的非金属（与同族的 Al 金属性质差异大），遵循对角规则与 Si 相似
- $\mathrm{H}_3\mathrm{BO}_3$ 是一元弱酸（非三元），$K = 5.4\times 10^{-10}$
- $\mathrm{B}_2\mathrm{H}_6$ 中存在 3c-2e B—H—B 桥键（缺电子特征）
- $\mathrm{B}_3\mathrm{N}_3\mathrm{H}_6$（无机苯）与 $\mathrm{C}_6\mathrm{H}_6$ 等电子、等结构
- 同周期含氧酸酸性：$\mathrm{HClO}_4 > \mathrm{H}_2\mathrm{SO}_4 > \mathrm{H}_3\mathrm{PO}_4 > \mathrm{H}_4\mathrm{SiO}_4$

## 五、常见分类或情形
- **硼的氢化物（硼烷）**：$\mathrm{B}_2\mathrm{H}_6$、$\mathrm{B}_4\mathrm{H}_{10}$、$\mathrm{B}_5\mathrm{H}_9$ 等
- **硼酸盐**：硼砂 $\mathrm{Na}_2[\mathrm{B}_4\mathrm{O}_5(\mathrm{OH})_4]\cdot 8\mathrm{H}_2\mathrm{O}$
- **硼酸**：$\mathrm{H}_3\mathrm{BO}_3$
- **硼的卤化物**：$\mathrm{BF}_3$（平面三角形，Lewis 酸）
- **硼氮化合物**：BN（类石墨）、$\mathrm{B}_3\mathrm{N}_3\mathrm{H}_6$（无机苯）
- **配位氢化物**：$\mathrm{LiBH}_4$、$\mathrm{Al(BH}_4)_3$

## 六、适用条件与限制
- 硼砂溶于水，重结晶即可纯化（不同温度得不同结晶水合物）
- $\mathrm{H}_3\mathrm{BO}_3$ 的酸性来自与 $\mathrm{H}_2\mathrm{O}$ 的配位而非自身电离 $\mathrm{H}^{+}$
- 硼烷在空气中易燃，需无水无氧处理
- 缩合酸的形成与浓度、酸度、温度有关

## 七、常见比较与易混点
| 易混点 | 区分 |
|------|------|
| B vs Al | B 为非金属（对角规则与 Si 相似），Al 为金属 |
| $\mathrm{H}_3\mathrm{BO}_3$ 是一元还是三元酸 | 形式上三元，实际一元弱酸（加合 $\mathrm{OH}^{-}$ 而非电离 $\mathrm{H}^{+}$） |
| $\mathrm{B}_3\mathrm{N}_3\mathrm{H}_6$ vs $\mathrm{C}_6\mathrm{H}_6$ | 等电子体，分别称为"无机苯"和苯 |
| 缩合酸 vs 原正酸 | 缩合酸非羟基氧更多，酸性更强 |

## 十、例题

### 例题 1（基础 · H₃BO₃ 是一元酸还是三元酸）

**题目**：硼酸的化学式写作 H₃BO₃，形式上有三个 H。但实际上 H₃BO₃ 是一元弱酸（Kₐ = 5.8×10⁻¹⁰）。解释原因，并写出 H₃BO₃ 在水中的电离方程式。为什么加入甘油（丙三醇）后硼酸的酸性显著增强？

**分析**：问题的关键在于——硼酸不是通过”给出 H⁺”来体现酸性，而是通过”接受 OH⁻”来间接增大溶液中 H⁺ 的浓度。这是缺电子化合物的特征行为。

**解答**：
（1）B 是缺电子原子（价电子 2s²2p¹，3 个电子 4 个轨道）。H₃BO₃ 中 B 以 sp² 杂化与 3 个 OH 形成 σ 键，还有一个空的 2p 轨道。这个空轨道接受 H₂O 中 OH⁻ 的孤对电子，形成 [B(OH)₄]⁻，使水的电离平衡右移，H⁺ 浓度增大：
$$\mathrm{H_3BO_3 + H_2O \rightleftharpoons [B(OH)_4]^- + H^+}$$

**不是** $\mathrm{H_3BO_3 \rightleftharpoons H^+ + H_2BO_3^-}$（这是三元酸的电离方式！）

（2）加入甘油（CH₂OH-CHOH-CH₂OH）后，甘油的两个 OH 与 H₃BO₃ 的 B 螯合，释放出 H⁺：
$$\mathrm{H_3BO_3 + 甘油 \rightarrow [硼酸-甘油螯合物]^- + H^+}$$
酸性显著增强（可用于酸碱滴定定量测定硼酸）。

**反思**：这是缺电子化学的第一个重要教训——**缺电子化合物的”酸性”可能不是给出 H⁺，而是接受电子对**。这与 BF₃ + NH₃ → F₃B←NH₃ 的 Lewis 酸碱性是同源的。

### 例题 2（提高 · B₂H₆ 的三中心二电子键）

**题目**：乙硼烷 B₂H₆ 的结构长期困扰化学家——按价键理论，B 有 3 个价电子，与 3 个 H 形成 BH₃ 刚好，但实验测得 B₂H₆ 是桥式结构（两个 B 通过两个 H 桥连接），且所有 B—H 键等价。

（1）画出 B₂H₆ 的结构，标出端 H 和桥 H。
（2）B₂H₆ 中共有多少个价电子？多少根”键”？
（3）桥 B—H—B 的成键方式如何描述？这是一种什么类型的化学键？
（4）写出 B₂H₆ 与 LiH 反应的方程式，指出产物中 B 的杂化方式变化。

**分析**：B₂H₆ 是缺电子化合物的”图腾”——正是因为 B 原子只有 3 个价电子却有 4 个价轨道（2s+2pₓ+2pᵧ+2p𝓏），无法形成 4 根正常 2c-2e 键，才产生了”三中心二电子键”这种特殊的成键方式。

**解答**：
（1）结构：两个 B 各以 sp³ 杂化（不等性）与 2 个端 H 形成 2c-2e σ 键。4 个端 H 和 2 个 B 共面。上下各 1 个桥 H，其连线 ⟂ 分子平面。
（2）价电子总数 = 2×3(B) + 6×1(H) = 12 e⁻。4 根端 B—H（2c-2e）×4 = 8 e⁻，剩下 4 e⁻ 分配给 2 个桥 B—H—B → 每个桥 = **3c-2e 键**（三个原子共用两个电子）。
（3）桥 B—H—B：每个 B 各提供一个 sp³ 杂化轨道 + H 的 1s 轨道 → 三个轨道线性组合 → 成键 MO 填 2 个 e⁻。这叫做**三中心二电子键（3c-2e）**，是缺电子化合物的标志性成键方式。
（4）$\mathrm{B_2H_6 + 2LiH \longrightarrow 2Li[BH_4]}$。B 从 B₂H₆ 中的 sp³（配位数 4）→ [BH₄]⁻ 中的 **sp³（正四面体，配位数 4）**——杂化方式未变，但成键方式从”3c-2e 桥键 + 2c-2e 端键”全部转为 2c-2e σ 键。Li[BH₄] 是重要的还原剂。

**反思**：B₂H₆ 的结构是”电子不够用怎么办”的化学答案——不是少一根键，而是让两个电子为三个原子服务。类似地，Al₂Cl₆ 也用 3c-2e 桥 Cl 键（但那是配位共价键而非缺电子键）。

### 例题 3（竞赛 · BF₃ 系列结构与 Lewis 酸性）

**题目**：BF₃ 是平面三角形分子，B—F 键长 130 pm。

（1）BF₃ 与乙醚（Et₂O）作用后，B—F 键长增加到 141 pm。解释原因并指出新化合物的几何构型变化。
（2）BF₃ 的一水合物 BF₃·H₂O 的液相中存在与分子数相同的离子，其中一半为 +1 价、一半为 −1 价。写出其结构。
（3）BF₃ 与弱酸 HF 作用得到一个很强的酸。写出反应方程式并指出该酸的结构。
（4）BF₃ 与 NH₃ 反应得到加合物 H₃N→BF₃，后者在 125°C 以上分解得到两种晶体——一种结构与石墨相似，另一种含两种均为正四面体的离子（1:1）。写出分解反应的方程式。

**分析**：BF₃ 是检验 Lewis 酸碱理论综合运用能力的经典分子。

**解答**：
（1）BF₃ 中 B 是 sp² 杂化（平面三角形）。乙醚的 O 提供孤对电子与 B 的空 2p 轨道配位 → B 变为 **sp³ 杂化（四面体）**。B—F 键长增加是因为：B 从 sp²（s 成分 33%）变 sp³（s 成分 25%），与 F 的 σ 键减弱。产物：Et₂O→BF₃（四面体）。
（2）**自耦电离**：$2\mathrm{BF_3\cdot H_2O \rightleftharpoons [BF_3OH]^- + [BF_3\cdot OH_2]^+}$。即 BF₃ + H₂O → 一半 BF₃ 接受 OH⁻ 形成 [BF₃OH]⁻（四面体），一半 BF₃ 把 H₂O 整体配位形成 [H₂O→BF₃]⁺。
（3）$\mathrm{BF_3 + HF \longrightarrow H[BF_4]}$。**氟硼酸 HBF₄** 是强酸（与 H₂SO₄ 强度相当）。[BF₄]⁻ 为**正四面体**。
（4）$\mathrm{4H_3NBF_3 \xrightarrow{125^{\circ}C} 3NH_4BF_4 + BN}$（类比 NH₄Cl 的热分解 NH₄Cl → NH₃ + HCl，但此处是氧化还原+缩合）。BN（氮化硼）——与石墨等电子，六方层状结构。[NH₄]⁺ 正四面体 + [BF₄]⁻ 正四面体。

**反思**：这道题串联了 BF₃ 的”平面三角形→四面体”构型转换、自耦电离、超酸化学、等电子体原理（BN↔C₂），是竞赛对硼化学考查的经典深度。

### 例题 4（竞赛 · NaBH₄ 的合成与性质）

**题目**：NaBH₄ 被称为有机化学家的”万能还原剂”。60 年代德国拜耳药厂将 NaBH₄ 的合成发展为工业规模：

$$\mathrm{Na_2B_4O_7 + Na + H_2 + SiO_2 \longrightarrow NaBH_4 + Na_2SiO_3}$$

（1）配平上述方程式。
（2）NaBH₄ 易溶于水并与水反应产生 H₂。设 B 元素以 BO₂⁻ 形式存在，写出该反应的离子方程式。
（3）NaBH₄ 与水反应的速率受 pH 影响。pH 升高时反应速率如何变化？为什么？
（4）NaBH₄ 可将 Ru³⁺ 还原为金属 Ru 沉积在玻璃上。写出配平的离子方程式（设 B 元素以 BO₂⁻ 形式存在）。

**分析**：NaBH₄ 是缺电子化学的应用延伸——B₂H₆ + LiH → LiBH₄，NaBH₄ 的结构和性质都与 [BH₄]⁻ 的稳定性有关。

**解答**：
（1）$\mathrm{Na_2B_4O_7 + 16Na + 8H_2 + 7SiO_2 \longrightarrow 4NaBH_4 + 7Na_2SiO_3}$
氧化数追踪：Na(0→+1)，H₂(0→−1 in BH₄⁻)，SiO₂ 不作氧化还原。

（2）$\mathrm{BH_4^- + 2H_2O \longrightarrow BO_2^- + 4H_2\uparrow}$
BH₄⁻ 中 H 的氧化数为 −1，H₂O 中 H 为 +1 → H(−1) + H(+1) → H₂(0)，归中反应。

（3）pH 升高 → 反应速率**降低**。原因是 BH₄⁻ 的水解需要 H⁺ 参与（H(−1) + H(+1) → H₂），pH 高时 [H⁺] 低，水解被抑制。实际上，NaBH₄ 在碱性水溶液中相当稳定（pH > 10 时几乎不水解），这允许它在水溶液中使用。

（4）$8\mathrm{Ru^{3+} + 3BH_4^- + 24OH^- \longrightarrow 8Ru\downarrow + 3BO_2^- + 18H_2O}$
B(−5 → +3) 失 8e⁻，Ru(+3 → 0) 得 3e⁻，LCM = 24。

**反思**：NaBH₄ 是”缺电子化学→实际应用”的完美例子。理解 BH₄⁻ 的稳定性需要回到 B 的缺电子原子特征——B 有 4 个价轨道，形成 4 根 2c-2e B—H 键后恰好”饱和”，因此 [BH₄]⁻ 比 BH₃（缺电子）稳定得多。

## 十一、易错点

| 易错点 | 学生为什么会这么想 | 正确认识 | 口诀 |
|:---|:---|:---|:---|
| “H₃BO₃ 有三个 H，是三元酸” | 化学式写几个 H 就是几元酸——这是中学阶段的简化规则 | H₃BO₃ 的酸性来自 B 的**空轨道接受 OH⁻**，而非自身电离 H⁺。是一元弱酸（Kₐ = 5.8×10⁻¹⁰）。加甘油后酸性增强可用于定量滴定 | “硼酸只有一个氢在干活——另外两个是围观群众” |
| “B₂H₆ 就是两个 BH₃ 粘在一起” | 把聚合当成简单的”粘合”，不理解 3c-2e 键的本质 | BH₃ 不存在（B 只有 3 个价电子，配位数 3 时留下空轨道 → 极其不稳定）。B₂H₆ 中每个 B 是 sp³ 杂化，端 B—H 是正常 2c-2e 键，桥 B—H—B 是**三中心二电子键（3c-2e）** | “BH₃ 不存在——缺电子逼它找邻居借电子” |
| “BF₃ 和 BCl₃ 水解行为一样” | 把同族卤化物当成同质 | BF₃ 水解不完全（生成 HBF₄ 等一系列中间产物），BCl₃ 水解彻底（BCl₃ + 3H₂O → H₃BO₃ + 3HCl）。原因是 B—F 键太强、F⁻ 太稳定，BF₃ 的 Lewis 酸性更强，与水解产物 HF 结合为 HBF₄ 抑制进一步水解 | “氟硼难水解——氟离子抓住硼不撒手” |
| “B 和 Al 同族，性质差不多” | 从上到下类比惯性 | B 是**非金属**（缺电子化合物、共价键为主），Al 是**金属**（两性、Al³⁺ 存在于水溶液）。B 与右下方的 **Si** 更相似（对角线规则：B₂O₃↔SiO₂、H₃BO₃↔H₄SiO₄、硼砂↔硅酸盐） | “硼跟铝是远亲，跟硅才是近邻——对角线比同族亲” |
| “铝热反应就是铝和氧化铁的反应” | 把铝热法局限在一个反应 | 铝热法（Aluminothermy）是通用方法：Al 与几乎任何过渡金属氧化物反应都放出巨大热量，可冶炼 Cr、Mn、V 等难熔金属。原理是 Al₂O₃ 的生成焓（−1669.7 kJ/mol）远超大多数金属氧化物 | “铝热不是一个人的表演——所有过渡金属氧化物都被它抢走氧” |
| “Tl 的稳定氧化态应该是 +3，因为它和 Al 同族” | 又是”从上到下类比”的惯性 | Tl 因 **6s² 惰性电子对效应**，+1 比 +3 稳定得多。Tl(III) 是强氧化剂（Tl³⁺ + 2e⁻ → Tl⁺，易被 SO₂、S²⁻ 还原）。TlOH 是强碱（碱性接近 KOH），Tl⁺ 的许多盐与 Ag⁺ 的盐相似 | “铊的 6s² 也是铁饭碗——三价铊是氧化剂，一价铊像银” |

## 十二、🎯 教学视角

### 12.1 学生典型认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 教学干预 |
|:---|:---|:---|:---|
| “硼酸是三元酸，三个 OH 各出一个 H⁺” | 中学把”酸”简化为”有 H 就能电离” | H₃BO₃ 是 Lewis 酸而非 Brønsted 酸——B 的空轨道接受 OH⁻ 而非给出 H⁺。这是缺电子化学的第一课 | 课堂上投影 H₃BO₃ 和 H₃PO₃ 的结构对比——让学生自己找出”为什么一个是三元一个是一元” |
| “缺电子就是电子不够，所以不稳定” | 直觉上”缺”= 不好 | 缺电子化合物恰恰因为”缺”而产生独特的化学——3c-2e 键、Lewis 酸性、桥键聚合。缺电子不是缺陷，是特征 | 用”拼图少一块→别人来补”的比喻替代”电子不够”的负面表述 |

### 12.2 讲授主线

（来自 [[教学逻辑提炼-Zchem-主族元素的结构化学I-第二轮深化]]）

硼族元素按三层递进：
1. **B 的缺电子化学**：B 只有 3 个价电子却有 4 个价轨道 → 缺电子原子 → 3c-2e 键（B₂H₆）→ Lewis 酸性（BF₃ + :NH₃）→ H₃BO₃ 的”假三元酸” → 硼砂的缓冲作用
2. **Al 的两性与工业化学**：对角线规则 B-Si 和 Be-Al → Al 的亲氧性（铝热法、Al₂O₃ 生成焓）→ Al(OH)₃ 的两性 → AlCl₃ 的缺电子二聚（Al₂Cl₆）→ 无机阻燃剂（Al(OH)₃/Mg(OH)₂）
3. **重元素的惰性电子对效应**：Ga/In 两性递减 → Tl(I) 极其稳定、Tl(III) 强氧化 → TlOH 强碱（与 KOH 相当）→ TlX 与 AgX 的相似性（难溶、光解）

### 12.3 与现实/直觉的连接

- **硼砂与”硼珠试验”**：Na₂B₄O₇·10H₂O 的熔珠溶解金属氧化物产生特征颜色——Co²⁺ 蓝色、Cr³⁺ 绿色、Cu²⁺ 蓝色、Mn²⁺ 紫色。这是竞赛定性分析中与”焰色反应”并列的鉴定方法。化学原理是硼砂的偏硼酸盐骨架可以包裹过渡金属离子
- **铝热反应**：Fe₂O₃ + 2Al → Al₂O₃ + 2Fe，温度可达 3000°C。野外焊接铁轨就是靠它。”铝热剂”（Thermite）= 铝粉 + 金属氧化物粉末。本质是 Al₂O₃ 的巨大晶格能驱动
- **无机阻燃剂**：Al(OH)₃ 在 300°C 分解吸热 + 释放水蒸气稀释可燃气体 + 产生 Al₂O₃ 覆盖层隔绝 O₂。这是”化学原理→材料应用”的典型案例

## 十三、竞赛拓展

1. **硼烷簇合物的 Wade 规则**：从 B₂H₆ 的 3c-2e 键升级到 B₅H₉、B₆H₁₀、B₁₀H₁₄ 等更高硼烷的簇合物结构。Wade 规则（骨架电子对数 = n+1 为闭式、n+2 为巢式、n+3 为网式）是竞赛第二轮结构化学的核心内容之一。Carborane（碳硼烷）是硼烷的 C 取代衍生物，[B₁₂H₁₂]²⁻ 是完美的正二十面体闭合笼

2. **BF₃ 的超酸化学**：BF₃ + HF → H[BF₄]（氟硼酸，强酸）。进一步——SbF₅（超强 Lewis 酸）与 HF 形成 HSbF₆，与 HSO₃F 形成 Magic Acid（魔酸）。竞赛可能考查 Lewis 酸强度比较（SbF₅ > BF₃ > AlCl₃）以及对碳正离子稳定化的应用

3. **无机苯 B₃N₃H₆ 的芳香性**：与 C₆H₆ 等电子但不等性——B—N 键有极性（电负性 N > B），导致亲电取代反应活性不同于苯。B₃N₃H₆ 与 HCl 的加成反应位置选择性（H 加到 N 上、Cl 加到 B 上←电负性控制），是竞赛将”有机反应规则”迁移到无机体系的经典交叉。与 [[碳族元素]] §十三 第 1 条（BN 等电子体）和 [[氧族元素]] §十三 第 3 条（Te₄²⁺ 芳香性）形成”无机芳香性”的完整知识网
