# 钙钛矿

- 总览：[[中国化学奥林匹克基本要求-总览]]

## 一、定义

**钙钛矿（Perovskite）**是一类化学通式为 $\mathrm{ABX_3}$ 的化合物，其名称来源于天然矿物钙钛矿 $\mathrm{CaTiO_3}$。这类结构具有极高的结构可调性，是固体化学和材料科学中最重要的结构类型之一。

### 基本结构特征

- **A 位**：较大的阳离子（通常是碱土金属、稀土元素离子），配位数为 **12**，位于由 X 阴离子构成的立方八面体中心
- **B 位**：较小的阳离子（通常是过渡金属离子），配位数为 **6**，位于由 X 阴离子构成的八面体中心
- **X 位**：阴离子（通常是 O²⁻、F⁻、Cl⁻ 等），位于 A 和 B 位离子之间，形成八面体配位环境

## 三、核心原理

### 晶胞描述

理想立方钙钛矿属于 **$Pm\bar{3}m$** 空间群，具有以下特征：

- **A 离子**：位于立方晶胞的顶点（8 × 1/8 = 1）
- **B 离子**：位于立方晶胞的体心（1）
- **X 离子**：位于立方晶胞的棱心（12 × 1/4 = 3）

化学式：**$\mathrm{ABO_3}$**

### 配位多面体连接方式

- **[BX₆] 八面体**：B 离子位于中心，6 个 X 离子位于八面体顶点
- **八面体连接方式**：共顶点连接，每个八面体通过 6 个顶点与相邻八面体相连
- **[AX₁₂] 多面体**：A 离子位于 12 个 X 离子形成的立方八面体中心

## 四、关键结论

### Goldschmidt 容差因子

钙钛矿结构能否稳定存在，可以用 **Goldschmidt 容差因子 $t$** 来判断：

$$
t = \frac{r_\mathrm{A} + r_\mathrm{X}}{\sqrt{2}(r_\mathrm{B} + r_\mathrm{X})}
$$

| $t$ 范围 | 结构类型 | 实例 |
|:---:|:---|:---|
| $0.80 \leq t \leq 1.00$ | 立方/四方/正交钙钛矿 | $\mathrm{SrTiO_3}$、$\mathrm{BaTiO_3}$ |
| $t > 1.00$ | 六方钙钛矿（A 离子过大） | $\mathrm{BaNiO_3}$ |
| $t < 0.80$ | 非钙钛矿结构（如橄榄石型） | $\mathrm{MgSiO_3}$ |

## 五、常见分类或情形

### 典型实例

| 化合物 | A 位 | B 位 | 特点 |
|------|:---:|:---:|------|
| $\mathrm{CaTiO_3}$ | Ca²⁺ | Ti⁴⁺ | 天然钙钛矿矿物 |
| $\mathrm{BaTiO_3}$ | Ba²⁺ | Ti⁴⁺ | 铁电材料，居里温度 120°C |
| $\mathrm{SrTiO_3}$ | Sr²⁺ | Ti⁴⁺ | 顺电性，高介电常数 |
| $\mathrm{LaFeO_3}$ | La³⁺ | Fe³⁺ | 催化材料 |
| $\mathrm{CH_3NH_3PbI_3}$ | MA⁺ | Pb²⁺ | 有机-无机杂化钙钛矿，光伏明星材料 |

### 钙钛矿的衍生结构

#### 1. Ruddlesden-Popper（RP）相

将钙钛矿结构沿垂直于四重轴的平面切开，插入岩盐型 [AO] 层，形成层状结构：

**通式**：$\mathrm{A_{n+1}B_nO_{3n+1}}$ 或 $(\mathrm{ABO_3})_n \cdot (\mathrm{AO})$

- $n = 1$：单层钙钛矿 + 岩盐层 → $\mathrm{A_2BO_4}$
- $n = 2$：双层钙钛矿 + 岩盐层 → $\mathrm{A_3B_2O_7}$
- $n \to \infty$：趋近于三维钙钛矿 $\mathrm{ABO_3}$

> 详见真题：[[题-035-6-钙钛矿及RP相]]

#### 2. 双钙钛矿（Double Perovskite）

两种不同 B 位离子有序排列：

**通式**：$\mathrm{A_2BB'O_6}$

- 实例：$\mathrm{Sr_2FeMoO_6}$（半金属磁电阻材料）

#### 3. 氧缺陷型钙钛矿

部分氧位缺失，形成非化学计量化合物：

**通式**：$\mathrm{ABO_{3-\delta}}$

- 实例：$\mathrm{SrCoO_{3-\delta}}$（固体氧化物燃料电池阴极材料）

## 六、适用条件与限制

- **Goldschmidt 容差因子 $t$ 是第一道门槛**：$t = (r_A + r_X) / [\sqrt{2}(r_B + r_X)]$。$t < 0.80$ → 不能形成钙钛矿结构（如 MgSiO₃ 以橄榄石型存在）；$0.80 \leq t \leq 1.00$ → 立方/四方/正交钙钛矿；$t > 1.00$ → A 离子过大，倾向六方钙钛矿
- **八面体倾斜（Octahedral Tilting）**：当 $t < 1$ 时，[BX₆] 八面体发生协同倾斜以优化 A—X 键长，导致对称性降低——立方 $Pm\bar{3}m$ → 四方 $I4/mcm$ → 正交 $Pbnm$（Glazer 倾斜体系）
- **Jahn-Teller 畸变**：B 位为 $d^4$（如 Mn³⁺）或 $d^9$（如 Cu²⁺）时，八面体会发生拉长/压扁畸变，进一步降低对称性
- **B 位阳离子 $d^0$ 性**：B 为 $d^0$ 离子（Ti⁴⁺、Nb⁵⁺、W⁶⁺）时，二阶 Jahn-Teller 效应使 B 离子偏离八面体中心 → 铁电性（如 BaTiO₃）
- **A 位/B 位离子价态组合必须保持电中性**：$A^{n+}B^{m+}O_3$ 要求 $n + m = 6$。常见组合——I-V（如 KNbO₃）、II-IV（如 CaTiO₃、BaTiO₃）、III-III（如 LaFeO₃）
- **有机-无机杂化钙钛矿（HOIP）的限制**：A 位为 MA⁺/FA⁺ 等有机阳离子时，受限于 Goldschmidt 容差因子的同时，还需考虑氢键作用和湿度/热不稳定性

## 七、常见比较与易混点

| 易混点 | 区分 |
|------|------|
| 钙钛矿 vs 尖晶石 | 钙钛矿：ABX₃，B 位八面体共顶点；尖晶石：AB₂O₄，B 位八面体共边 |
| 理想钙钛矿 vs 畸变钙钛矿 | 理想：立方对称；畸变：B 离子偏离八面体中心（如铁电体的 Ti⁴⁺ 位移） |
| RP 相 vs 层状双氢氧化物 | RP 相是钙钛矿衍生，层间为岩盐型；LDH 是层状氢氧化物，层间为阴离子 |
| 双钙钛矿 vs 有序合金 | 双钙钛矿中 B/B' 的岩盐型有序排列产生超晶胞（$2a_p \times 2a_p \times 2a_p$），XRD 出现超晶格衍射峰 |
| 钙钛矿 vs 反钙钛矿 | 反钙钛矿（如 Mn₃GaN）中阴离子在 B 位、较大的阳离子在 A 位——"正负角色互换" |

## 九、典型题型

1. **题型-钙钛矿晶胞计数与化学式验证**：已知 CaTiO₃ 晶胞中 Ca²⁺（顶点×1/8=1）、Ti⁴⁺（体心×1=1）、O²⁻（棱心×12×1/4=3）→ 验证 ABO₃ 化学式
2. **题型-Goldschmidt 容差因子计算与结构预测**：给定 $r_A$、$r_B$、$r_O$，计算 $t$ 并判断结构类型（立方/四方/六方钙钛矿 vs 非钙钛矿）
3. **题型-钙钛矿衍生结构推断（RP 相、双钙钛矿）**：从通式和 $n$ 值写出化学式，或从化学式反推 $n$
4. **题型-鲍林电价规则在钙钛矿中的应用**：验证 CaTiO₃ 中 O²⁻ 的静电键强度之和 $4 \times 2/12 + 2 \times 4/6 = 2 = Z(O^{2-})$，判断结构稳定性
5. **题型-钙钛矿功能材料综合**：结合铁电性（BaTiO₃ 的 Ti⁴⁺ 位移）、光伏（CH₃NH₃PbI₃）、催化（LaFeO₃）出材料情境推断题

## 十、例题

### 例题 1（基础 · CaTiO₃ 晶胞计数与配位数）

**题目**：CaTiO₃ 是钙钛矿结构的原型。（1）在一个 CaTiO₃ 晶胞中，Ca²⁺、Ti⁴⁺、O²⁻ 各有几个？（2）Ca²⁺ 和 Ti⁴⁺ 的 O²⁻ 配位数各是多少？为什么不同？

**分析**：按"顶点 Ca²⁺ → 1/8 × 8 = 1，体心 Ti⁴⁺ → 1，棱心 O²⁻ → 1/4 × 12 = 3"直接计数。

**解答**：
（1）Ca²⁺：8 个顶点 × 1/8 = **1**；Ti⁴⁺：体心 × 1 = **1**；O²⁻：12 条棱心 × 1/4 = **3**。化学式 CaTiO₃ 得到验证。

（2）Ca²⁺ 的 O²⁻ 配位数：**12**——Ca²⁺ 位于 12 个最近邻 O²⁻ 构成的立方八面体中心。Ti⁴⁺ 的 O²⁻ 配位数：**6**——Ti⁴⁺ 位于 [TiO₆] 八面体中心。

配位数不同的原因：Ca²⁺ 半径（100 pm）> Ti⁴⁺ 半径（60.5 pm），更大的 A 位离子能容纳更多的配位 O²⁻。

**反思**：钙钛矿晶胞有两种等价画法——① Ca²⁺ 在顶点（常见于"离子堆积"描述）、② Ti⁴⁺ 在顶点（原点移到 Ti 上，则 Ca²⁺ 移到体心、O²⁻ 移到面心）。两种画法描述的是**同一个结构**——判断时只需看哪个离子更直观。

### 例题 2（提高 · Goldschmidt 容差因子计算）

**题目**：已知离子半径（pm）：$r(\mathrm{Sr^{2+}}) = 118$，$r(\mathrm{Ba^{2+}}) = 135$，$r(\mathrm{Ti^{4+}}) = 60.5$，$r(\mathrm{Fe^{3+}}) = 55$（LS），$r(\mathrm{O^{2-}}) = 140$。

（1）计算 SrTiO₃ 和 BaTiO₃ 的 Goldschmidt 容差因子 $t$。判断哪种结构更接近理想立方钙钛矿。
（2）SrFeO₃ 是否能形成钙钛矿结构？用 $t$ 值判断。

**分析**：$t = (r_A + r_O) / [\sqrt{2}(r_B + r_O)]$。$t$ 越接近 1，结构越接近理想立方。

**解答**：
（1）SrTiO₃：$t = (118 + 140) / [1.414 \times (60.5 + 140)] = 258 / 283.3 = 0.911$
BaTiO₃：$t = (135 + 140) / [1.414 \times (60.5 + 140)] = 275 / 283.3 = 0.971$
BaTiO₃ 的 $t = 0.971$ 更接近 1 → **BaTiO₃ 更接近理想立方钙钛矿**。SrTiO₃（$t = 0.911$）在室温下是立方，但已有轻微八面体倾斜倾向。

（2）SrFeO₃：$t = (118 + 140) / [1.414 \times (55 + 140)] = 258 / 275.7 = 0.936$
$t = 0.936$，在 $0.80 \sim 1.00$ 范围内 → **可以形成钙钛矿结构**（实际 SrFeO₃ 确实为立方钙钛矿，且因 Fe⁴⁺ 的 $d^4$ 电子构型表现出有趣的磁性和电子性质）。

**反思**：容差因子虽然是经验判据，但准确度很高。BaTiO₃（$t \approx 0.97$）→ 接近理想立方 → Ti⁴⁺ 有足够空间位移 → 铁电性。如果 $t$ 太小，八面体被"夹紧"，B 离子无法位移 → 顺电性。

### 例题 3（提高 · Ruddlesden-Popper 相化学式推导）

**题目**：Ruddlesden-Popper（RP）相的通式为 $A_{n+1}B_nO_{3n+1}$。

（1）写出 $n = 1, 2, 3, \infty$ 时 RP 相的化学式。
（2）某 RP 相材料中 A = Sr、B = Ti，$n = 2$。写出其化学式，并描述其结构特征（钙钛矿层厚度与岩盐层层数）。
（3）对比 $n = 1$ RP 相 Sr₂TiO₄ 与三维钙钛矿 SrTiO₃ 中 Ti 的配位环境有何不同。

**分析**：代入 $n$ 值即可得化学式。结构特征——"$n$ 层钙钛矿 + 1 层岩盐层 AO"交替。

**解答**：
（1）
- $n = 1$：$\mathrm{A_2BO_4}$
- $n = 2$：$\mathrm{A_3B_2O_7}$
- $n = 3$：$\mathrm{A_4B_3O_{10}}$
- $n \to \infty$：趋近于 $\mathrm{ABO_3}$

（2）化学式：**$\mathrm{Sr_3Ti_2O_7}$**。结构——2 层 $[\mathrm{TiO_6}]$ 八面体钙钛矿层 + 1 层岩盐型 SrO 层，沿 c 轴交替堆叠。钙钛矿层厚度 = 2 个八面体，岩盐层层数 = 1。

（3）SrTiO₃ 中 Ti⁴⁺ 位于 $[\mathrm{TiO_6}]$ 八面体中心，轴向和赤道向的 O²⁻ 完全等价（6 个等同 Ti—O 键）。Sr₂TiO₄ 中，$n = 1$ 意味着单层钙钛矿——Ti⁴⁺ 仍为 6 配位八面体，但**顶端的 O²⁻（朝向岩盐层）和赤道面的 O²⁻ 不再等价**，Ti—O(apical) 键比 Ti—O(equatorial) 键短——这就是"层间应力"的表现。

**反思**：RP 相是"层状钙钛矿"——从三维到二维的结构切分。$n$ 控制钙钛矿层厚度 → 控制电子维度（3D → 2D → 准2D）→ 是调控超导、磁性、催化性能的核心结构参数。

### 例题 4（竞赛 · 双钙钛矿与有序占位推断）

**题目**：双钙钛矿 $\mathrm{Sr_2FeMoO_6}$ 是一种半金属磁电阻材料。

（1）在 $\mathrm{Sr_2FeMoO_6}$ 的晶胞中，Fe 和 Mo 如何排列？写出其有序排列的化学式通式。
（2）若 Fe 和 Mo 在 B 位完全无序排列，晶胞大小与有序排列有何不同？
（3）用鲍林电价规则验证 $\mathrm{Sr_2FeMoO_6}$ 中 O²⁻ 的静电键强度之和。

**分析**：双钙钛矿 A₂BB'O₆ → B 和 B' 在原始钙钛矿的 B 位呈**岩盐型有序排列**（B 和 B' 沿三个方向交替出现），导致晶胞扩大为 $2a_p \times 2a_p \times 2a_p$。

**解答**：
（1）Fe³⁺（$3d^5$）和 Mo⁵⁺（$4d^1$）在 B 位**岩盐型有序排列**——每个 FeO₆ 八面体周围 6 个最近邻 B 位全部是 MoO₆（反之亦然）。通式：$\mathrm{A_2BB'O_6}$。此有序排列使晶胞扩大为原钙钛矿的 8 倍（$2 \times 2 \times 2$）→ 超晶胞。

（2）完全无序时：B 位被 (Fe₀.₅Mo₀.₅) 统计占据 → 每个 B 位等价 → 晶胞大小 = 单钙钛矿 $a_p \approx \sqrt{2}(r_{Sr} + r_O)$（无超晶胞）。有序使晶胞参数翻倍。

（3）Fe³⁺（配位数 6）：静电键强度 $S_{Fe} = 3/6 = 1/2$
Mo⁵⁺（配位数 6）：$S_{Mo} = 5/6$
Sr²⁺（配位数 12）：$S_{Sr} = 2/12 = 1/6$
O²⁻ 配位多面体为 $[\mathrm{OSr_4Fe_1Mo_1}]$（4 个 Sr + 1 个 Fe + 1 个 Mo）：
$\sum S = 4 \times 1/6 + 1 \times 1/2 + 1 \times 5/6 = 2/3 + 1/2 + 5/6 = 2.00 = Z(O^{2-})$ ✓

**反思**：鲍林第二规则（电价规则）是判断钙钛矿相关结构稳定性的关键工具——每当你面对一个新的钙钛矿衍生结构，先算 O²⁻ 的 $\sum S$，等于 2 才合理。偏离 2 意味着结构有应力、缺陷或错误判断。

## 十一、易错点

| 易错点 | 学生为什么会这么想 | 正确认识 | 口诀 |
|:---|:---|:---|:---|
| "钙钛矿晶胞中 O²⁻ 在面心" | CaTiO₃ 常见两种画法——把原点放在 Ca²⁺（O²⁻ 在棱心）vs 原点放在 Ti⁴⁺（O²⁻ 在面心）→ 混淆两种画法的归属 | 两种画法描述**同一个结构**。原点在 Ca²⁺：O²⁻ 在 12 条棱心 → × 1/4 = 3。原点在 Ti⁴⁺：O²⁻ 在 6 个面心 → × 1/2 = 3。结果一致，但判断归属时要先确定**原点在哪** | "原点不同位置换——氧从棱心跑面心，总数还是三" |
| "$t$ 在 0.8~1.0 就一定形成理想立方钙钛矿" | 把 $t$ 的"允许范围"等同于"立方范围" | $t$ 在 0.8~1.0 只说明**可以形成钙钛矿结构**，但不一定是立方。实际上只有 $t > 0.95$ 才接近立方；$t = 0.85 \sim 0.95$ 通常形成四方/正交钙钛矿（八面体倾斜） | "容差保钙钛矿——但不保立方，0.9 以下八面体就歪了" |
| "A 位离子必须大于 B 位离子" | 直观上"顶点放大的、体心放小的" | 这是正确的直观，但不绝对——A 位配位数 12 自然需要较大半径。但必须注意**具体数值**：$r_A > r_B$ 是必要条件但非充要条件，关键还要看 $t = (r_A+r_O)/[\sqrt{2}(r_B+r_O)]$ | "A 大 B 小是大方向——具体看容差算一算" |
| "CH₃NH₃PbI₃ 是有机物，跟钙钛矿没关系" | 把"钙钛矿"当成无机材料的专属名词 | 钙钛矿是**结构类型**，不是化学组成限制。CH₃NH₃PbI₃ 中 MA⁺ 占据 A 位、Pb²⁺ 占据 B 位、I⁻ 占据 X 位 → 是标准的有机-无机杂化钙钛矿（HOIP）。"钙钛矿"定义的是**ABX₃ + 配位几何**，不限制元素种类 | "钙钛矿是个结构——有机无机都能住" |
| "双钙钛矿中 B 和 B' 是随机排列的" | 把"两种离子都在 B 位"等同于"无序占位" | 双钙钛矿中 B/B' 呈**岩盐型有序排列**——每个 BO₆ 八面体的 6 个最近邻 B 位都是 B'O₆。这是**焓驱动**的有序化（电荷差异越大、有序化驱动力越强）。若无序，结构退化为单钙钛矿 $AB_{0.5}B'_{0.5}O_3$ | "双钙钛矿 B 有序——两个 B 像 NaCl 的 Na 和 Cl 一样交替" |

## 十二、🎯 教学视角

### 12.1 学生典型认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 教学干预 |
|:---|:---|:---|:---|
| "钙钛矿 = CaTiO₃，就是一种矿物" | 名字带"钙钛"→ 以为是一种特定化合物 | 钙钛矿是一个**庞大的结构家族**——从 BaTiO₃（铁电）到 CH₃NH₃PbI₃（光伏）到 Sr₂FeMoO₆（磁电阻）——都是钙钛矿。"钙钛矿"描述的是 ABX₃ 的**结构类型**，不是化学式的固定组合 | 一上来就把 5 个应用甩出来——"铁电、光伏、催化、超导、磁电阻——全叫钙钛矿，但它们做的事完全不同" |
| "Goldschmidt 容差因子就是背公式" | 当成又一个需要死记的经验参数 | $t$ 的物理本质是**几何匹配**——A—X 键长（$(r_A+r_O)$）和 B—X 键长（$(r_B+r_O)/\sqrt{2}$）的比较。A 位太大 → $t > 1$ → A 位"撑破"了八面体框架 → 六方钙钛矿。A 位太小 → $t < 0.8$ → A 位"晃荡"在过大的空隙中 → 结构坍塌为其他类型 | 用"鞋子和脚"的比喻——"A 是脚，八面体框架是鞋。太大了挤不进（$t > 1$），太小了晃荡（$t < 0.8$）" |

### 12.2 讲授主线

钙钛矿的教学应按"结构→调控→应用"三层递进：

**第一层：基础结构（看懂 CaTiO₃ 晶胞）**
- 两种等价画法统一：顶点 A + 体心 B + 棱心 X₃ ↔ 顶点 B + 体心 A + 面心 X₃
- 原子归属计数验证 ABX₃
- 配位多面体：$[\mathrm{BX_6}]$ 八面体共顶点 → 三维网络；A 在 12 配位立方八面体中心

**第二层：结构调控（容差因子 + 衍生结构）**
- Goldschmidt $t$：从几何匹配预测结构稳定性
- 八面体倾斜（Glazer 体系）：$t < 1$ 时对称性降低
- 衍生结构家族：RP 相（层状钙钛矿）、双钙钛矿（B 位有序）、氧缺陷型（非化学计量）
- 鲍林电价规则验证 $\sum S = Z(O^{2-})$

**第三层：功能-结构关联（为什么钙钛矿是"无机材料之王"）**
- BaTiO₃ 铁电性：Ti⁴⁺($d^0$) 的位移型铁电 → 电容器/存储器
- CH₃NH₃PbI₃ 光伏：直接带隙 + 长载流子扩散长度 → 太阳能电池效率 ~26%
- Sr₂FeMoO₆ 磁电阻：半金属 + 室温低场磁电阻 → 自旋电子学
- LaFeO₃/LaCoO₃ 催化：氧空位 + 变价 B 位 → 固体氧化物燃料电池/汽车尾气催化

### 12.3 与现实/直觉的连接

- **从矿物到光伏明星——钙钛矿太阳能电池**：2009 年 CH₃NH₃PbI₃ 首次用于光伏（效率 3.8%），到 2026 年效率已超 26%——这是太阳能电池史上最快的发展速度。钙钛矿太阳能电池的**核心优势**是"缺陷容忍"——不同于硅需要极高纯度，钙钛矿可以在溶液中自组装形成高质量薄膜，成本极低。
- **为什么汽车尾气催化器含钙钛矿**：LaFeO₃/LaCoO₃ 等钙钛矿在高温下稳定且表面有大量氧空位——这些空位就是催化活性位点。CO + O(空位) → CO₂ + 空位（Mars-van Krevelen 机理）。钙钛矿是少数能在 1000°C 汽车尾气环境中保持活性的材料。
- **"铁电体"与打火机**：按压式打火机里就是一小块压电陶瓷（通常是 PZT——Pb(Zr,Ti)O₃ 钙钛矿）。按压 → 机械应力 → Ti⁴⁺/Zr⁴⁺ 位移 → 极化 → 火花 → 点燃气体。钙钛矿结构把机械能直接转化成电能。

## 十三、竞赛拓展

1. **八面体倾斜的 Glazer 记号与对称性分析**：Glazer 体系用 $a^b c^d$ 记号（上标 +/0/− 表示相邻层八面体同向/不倾斜/反向倾斜）系统分类 23 种倾斜模式。如 $a^0a^0a^0$ = 理想立方 $Pm\bar{3}m$，$a^-a^-a^-$ = 三方 $R\bar{3}c$，$a^+b^-b^-$ = 正交 $Pbnm$。竞赛中如出现钙钛矿对称性分析题，Glazer 记号是判断晶系归属的标准工具。

2. **容差因子的扩展——八面体因子与"可形成性图"**：仅用 $t$ 还不够——$\mu = r_B/r_X$（八面体因子）保证了八面体的几何合理性（通常 $0.44 < \mu < 0.90$）。以 $t$ 为横轴、$\mu$ 为纵轴的"结构可形成性图"（structure formability map）是当前钙钛矿设计的前沿工具。竞赛信息给予题可能给出此图，要求判断某新化学式是否形成钙钛矿。

3. **卤化物钙钛矿量子点与发光**：CsPbX₃（X = Cl, Br, I）纳米晶（量子点）因量子限域效应表现出窄带发光（FWHM ~20 nm）、发光颜色可通过卤素比例在全可见光谱连续调谐。这是 2015 年后钙钛矿领域最热的研究方向之一。竞赛中可能以信息给予题出现——给定 CsPb(ClₓBr₁₋ₓ)₃ 的发光波长，结合 Vegard 定律推断成分。
