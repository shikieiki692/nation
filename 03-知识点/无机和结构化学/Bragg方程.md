# Bragg方程

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 相关主页：[[晶体结构]]、[[晶体结构深化]]

## 一、定义

Bragg 方程描述 X 射线在晶体晶面上发生相长干涉的条件：

$$
n\lambda = 2d\sin\theta
$$

其中：

- $n$ 为衍射级数
- $\lambda$ 为入射 X 射线波长
- $d$ 为相应晶面的晶面间距
- $\theta$ 为 Bragg 角，等于入射线与晶面的夹角；实验图谱常给的是 $2\theta$

## 二、考纲对应

主要服务晶体结构、晶胞参数、XRD 初步判断与结构题到实验表征题的桥梁入口。

## 三、核心原理

### 3.1 干涉条件的物理意义

把晶体看成一组平行晶面。相邻晶面反射出的两束 X 射线若光程差恰好为整数倍波长，就会相长干涉并出现衍射峰。

### 3.2 课堂最常用的三种调用方式

1. 已知 $\lambda$ 和 $\theta$，求晶面间距 $d$
2. 已知 $d$ 与立方晶系的 $(hkl)$，反推晶胞参数 $a$
3. 把 Bragg 方程与允许反射序列、系统消光合用，做粉末衍射的结构判型

### 3.3 在竞赛课里的真实定位

- 第一轮：只保留“XRD 能用来反推结构”的现象入口
- 第三轮：开始训练 $d$、$a$、$(hkl)$ 和峰位之间的互算
- 第四轮/决赛预备：接入 `系统消光 / 结构因子 / 结构到物性` 的完整链条

## 四、关键结论

1. 图谱常给的是 $2\theta$，代公式时要先换成 $\theta$
2. Bragg 方程给出的是衍射峰位置条件，不直接给峰强
3. 峰位主要反映晶面间距，峰强则还受结构因子等影响
4. 在立方晶系中，Bragg 方程常与 $d_{hkl} = \dfrac{a}{\sqrt{h^2+k^2+l^2}}$ 联用

## 五、常见分类或情形

### 5.1 基础计算型

已知某个衍射峰的 $\lambda$ 与 $2\theta$，先求 $d$，再由晶体模型求半径、密度或晶胞参数。

### 5.2 指标化入口型

给一串峰位或 $\sin^2\theta$ 比值，先借 Bragg 方程转成晶面间距关系，再判断对应的晶格类型。

### 5.3 结构-物性桥梁型

已通过 XRD 判断结构后，再追问配位数、填隙方式、导电性、相变或非整比特征。

## 六、适用条件与限制

1. Bragg 方程本身只给出几何衍射条件，不能单独完成完整结构解析
2. 复杂结构的峰强解释需要结构因子，不能只靠峰位
3. 粉末图谱、单晶图谱和电子/中子衍射的实验细节不同，课堂中不必混讲

## 七、常见比较与易混点

| 易混点 | 正确认识 |
|:---|:---|
| 把 $\theta$ 当成图谱上的读数 | 图谱通常读的是 $2\theta$ |
| 认为 $n$ 一定要保留高阶反射 | 竞赛题多数先按 $n=1$ 处理，再视题意判断 |
| 认为 Bragg 方程能直接判断 bcc/fcc | 结构判型还要结合允许反射序列或系统消光 |
| 把峰位和峰强混为一谈 | 峰位看 $d$，峰强还与结构因子、原子散射能力有关 |

## 九、典型题型

1. 由 $2\theta$ 求 $d$
2. 由 $d_{hkl}$ 和 $(hkl)$ 求立方晶胞参数 $a$
3. 由峰位比判断简单立方 / bcc / fcc
4. 结构已知后，把 XRD 结果接回密度、配位数和物性判断

## 十、例题

已知某立方晶体的 `(111)` 峰出现在 $2\theta$，波长已知。常规做法是：

1. 先由 Bragg 方程求 $d_{111}$
2. 再用 $d_{111} = a/\sqrt{3}$ 求 $a$
3. 最后回到晶胞原子数、密度或结构类型判断

## 十一、易错点

1. 忘记把角度从 $2\theta$ 改成 $\theta$
2. 把 Bragg 方程当成“所有 XRD 题的全部内容”
3. 只会代公式，不会把结果接回晶体结构语言

## 十二、🎯 教学视角

- 最好把 Bragg 方程讲成“结构题进入实验表征题的第一座桥”，而不是单独一节公式课
- 对竞赛学生最有用的口径不是推导细节，而是：
  - `峰位 -> d -> 晶胞参数 -> 结构`
  - `结构 -> 允许反射/系统消光 -> 判型`
- 第7讲这类初赛资料特别适合承担这个入口，因为它天然把投影、计数、相变和 XRD 放在同一条线上
