# NiAs型结构

## 一、定义
**NiAs 型结构（砷化镍型结构）**:以砷化镍 NiAs 为原型的典型 AB 型离子-金属混合性晶体结构。

- **阴离子 As**:占据**六方密堆积（hcp）**位置
- **阳离子 Ni**:占据所有**八面体空隙**(填充率 100%)
- 空间群:$P6_3/mmc$(六方晶系)
- 化学计量比:1:1(AB 型)
- 典型实例:NiAs, FeS, CoS, NiS, FeSe, CoSb, MnAs

## 三、核心原理

### 结构描述
- **阴离子层（As）**:沿 c 轴方向按 ABAB 交替排列(hcp 堆积),每个 As 周围有 12 个同类
- **阳离子（Ni）**:全部填入八面体空隙,沿 c 轴形成连续直线串(Ni-Ni 距离短)
- **配位环境**:
  - **Ni 周围**:6 个 As,组成正八面体
  - **As 周围**:6 个 Ni,组成**三方棱柱（trigonal prism）**(非八面体!)

### 与 NaCl 型对比

| 项目 | NaCl 型 | NiAs 型 |
|---|---|---|
| 阴离子堆积 | fcc(立方密堆积 ccp) | hcp(六方密堆积) |
| 阳离子位 | 八面体空隙(全填) | 八面体空隙(全填) |
| 配位数 | Na 6 / Cl 6 | Ni 6 / As 6 |
| Na/Ni 周围几何 | 正八面体 | 正八面体 |
| Cl/As 周围几何 | 正八面体 | **三方棱柱** |
| M-M 短接触 | 无 | **有(沿 c 轴 Ni-Ni 直线串)** |
| 导电性 | 绝缘 | **半金属(部分有金属性)** |
| 空间群 | $Fm\bar{3}m$ | $P6_3/mmc$ |

### 结构成因
- **NaCl 型 vs NiAs 型选择**取决于:
  1. 共价性程度高 → 倾向 NiAs(更多金属-金属相互作用)
  2. 离子电负性差异小 → 倾向 NiAs
  3. 阳离子尺寸接近 → 倾向 NiAs(短 M-M 接触不冲突)
- 一般规律:**强离子性(NaCl)** vs **过渡金属硫化物/砷化物(NiAs)**

### 短 M-M 接触的物理意义
- 沿 c 轴方向,相邻八面体共面共享 → Ni-Ni 距离短
- 形成"金属性导带"→ 部分 NiAs 型化合物表现金属性(FeS 高温相、NiAs)

## 四、关键结论

### 结构参数(NiAs 原型)
- 晶系:六方
- $a = 3.61$ Å, $c = 5.04$ Å, $c/a = 1.40$
- 空间群:$P6_3/mmc$ (No.194)
- Z = 2 (每晶胞 2 个 NiAs)
- Ni 占 (0, 0, 0) ; As 占 (1/3, 2/3, 1/4)

### NiAs 型晶体的特征
1. 仅阴离子作密堆,阳离子填全部八面体空隙
2. 配位数 6:6,但阴离子周围非八面体而是三方棱柱
3. 沿 c 轴的 M-M 接触短,部分化合物有金属性
4. 典型:过渡金属与 S/Se/Te/P/As/Sb 形成的二元化合物

### 典型化合物分类
| 类型 | 实例 |
|:---:|:---:|
| 硫化物 | FeS, NiS, CoS, MnS(高温相) |
| 硒化物 | FeSe, NiSe, CoSe |
| 锑化物 | NiSb, CoSb |
| 砷化物 | NiAs(原型), FeAs, MnAs |
| 碲化物 | NiTe, MnTe |

## 五、常见分类或情形

### 与其他 AB 型结构的比较
| 结构类型 | 阴离子堆积 | 阳离子位 | 配位数 | 实例 |
|---|:---:|:---:|:---:|:---:|
| NaCl 型 | fcc | 八面体全填 | 6:6 | NaCl, MgO, FeO |
| CsCl 型 | sc | 立方空隙全填 | 8:8 | CsCl, NH₄Cl, CuZn |
| 立方 ZnS | fcc | 四面体半填 | 4:4 | ZnS, GaAs, CdS |
| 六方 ZnS | hcp | 四面体半填 | 4:4 | ZnS, CdS, BeO |
| **NiAs 型** | **hcp** | **八面体全填** | **6:6** | NiAs, FeS, NiSe |

### "六方密堆积 + 全填八面体"的结构家族
- NiAs 型(1:1): As/Ni
- 反式 NiAs(其实是 Cd I₂ 型,半填): 1:2 比

### 缺陷型 NiAs
- 实际化合物中常出现阳离子缺陷,如 $\text{Fe}_{1-x}\text{S}$(磁黄铁矿,$x \approx 0.05-0.15$)
- 形成非整比化合物

## 六、适用条件与限制
1. **半径比**:虽然 r⁺/r⁻ 落在八面体范围(0.414-0.732),但 NiAs 型偏向半径比更大的体系
2. **化学计量比**:严格 1:1(纯净相);允许阳离子缺陷形成非整比
3. **极化性**:阳离子极化能力较强 → 偏向 NiAs 而非 NaCl
4. **温度依赖**:某些化合物高温为 NiAs 型,低温变形(如 FeS)
5. **不适用强离子型化合物**(NaCl/MgO 等仍取 NaCl 型)

## 七、常见比较与易混点

### NiAs 型 vs NaCl 型
- ⚠️ **最大易混点**:都是 AB 型、6:6 配位、八面体空隙全填,但堆积方式不同(fcc vs hcp),导致阴离子周围环境差异(八面体 vs 三方棱柱)
- 速判法:含**过渡金属 + 重负离子(S, Se, Te, As)**→ NiAs 型;含**碱金属/碱土金属 + O/F/Cl/Br**→ NaCl 型

### 三方棱柱 vs 八面体
- **八面体**:6 个原子绕中心,上 3 + 下 3 旋转 60°相错排列
- **三方棱柱**:6 个原子绕中心,上 3 + 下 3 对齐排列(无旋转)
- 区别在上下两个三角形面是"正对"还是"错位"

### NiAs vs WC 型
- WC 型:简单六方堆积(P),非密堆积
- NiAs 型:六方密堆积(hcp),阴离子层 ABAB

## 十、例题

### 例题 1:NiAs 晶胞计算
**题目:** NiAs 晶胞 $a = 3.61$ Å, $c = 5.04$ Å, $Z = 2$。求 NiAs 的密度。NiAs 摩尔质量 = 133.6 g/mol。

**解答:**
- 晶胞体积:$V = a^2 c \sin 120° = (3.61)^2 \times 5.04 \times \frac{\sqrt{3}}{2} = 56.85$ Å³ $= 5.685 \times 10^{-23}$ cm³
- 晶胞内质量:$m = \frac{2 \times 133.6}{6.022 \times 10^{23}} = 4.437 \times 10^{-22}$ g
- 密度:$\rho = \frac{m}{V} = \frac{4.437 \times 10^{-22}}{5.685 \times 10^{-23}} = 7.80$ g/cm³

**反思:** 注意六方晶胞体积公式 $V = a^2 c \sin 120°$。

### 例题 2:辨析配位几何
**题目:** 在 NiAs 型晶体中,As 周围有 6 个 Ni,这 6 个 Ni 形成什么几何构型?

**解答:**
- As 周围 6 个 Ni:**三方棱柱(trigonal prism)**,非正八面体
- 这是 NiAs 型的特征:阳离子周围八面体,阴离子周围三方棱柱
- 区分关键:NaCl 型中两者均八面体;NiAs 型阴离子环境不对称

**反思:** 阳离子和阴离子的配位几何在 NiAs 型中不对称,这正是与 NaCl 型的根本区别。

## 十一、易错点
- ❌ 错:NiAs 型阴离子周围是八面体——是**三方棱柱**!
- ❌ 错:NiAs 型和 NaCl 型完全等价——堆积方式不同(hcp vs fcc),衍生出 M-M 接触差异
- ❌ 错:所有 1:1 离子化合物都取 NaCl 型——过渡金属硫化物/砷化物常取 NiAs 型
- ❌ 错:NiAs 型只允许整比化合物——常见非整比缺陷(如 $\text{Fe}_{1-x}\text{S}$)
- ❌ 错:NiAs 型无金属性——沿 c 轴 Ni-Ni 短接触可产生金属性导电

### 12.3 入门级例题
**题目:** 判断下列化合物可能采取的晶体结构类型:① NaCl;② FeS(铁硫化合物高温相);③ MgO;④ NiAs。

**预期解答路径**:
1. NaCl:碱金属 + 卤素 → 强离子型 → **NaCl 型**
2. FeS:过渡金属 + 重负离子 → 半金属性 → **NiAs 型**
3. MgO:碱土金属 + 氧 → 强离子型 → **NaCl 型**
4. NiAs:原型 → **NiAs 型**

**老师可在课堂上引导的提问**:
- 问"两者配位数都是 6,为什么不同?"→ 引出堆积差异
- 问"什么样的化合物倾向 NiAs?"→ 引出过渡金属共价性

### 12.4 与现实/直觉的连接
- 类比:NaCl 型像"两组正方网叠起来";NiAs 型像"两组三角网叠起来"
- 实际应用:磁黄铁矿($\text{Fe}_{1-x}\text{S}$)是 NiAs 型变体,自然界常见

## 十三、竞赛拓展
- **磁黄铁矿(Pyrrhotite)**:$\text{Fe}_{1-x}\text{S}$,空位有序排列形成超结构
- **半金属性**:NiAs 型化合物 Ni-Ni 距离短,部分有金属电导率
- **磁性**:许多 NiAs 型化合物有磁有序态(铁磁、反铁磁)
- **CdI₂ 型**:阳离子半填(只填一半八面体空隙)→ AB₂ 型,与 NiAs 同源
