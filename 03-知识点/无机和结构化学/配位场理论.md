# 配位场理论

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[提高要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]

## 一、定义
配位场理论研究配体在金属中心周围形成非球形电场后，如何使金属离子的 d 轨道发生能级分裂，并进一步决定配合物的**电子排布、磁性、颜色与稳定性**。

课堂总线可以压成一句话：

`构型 -> d轨道分裂 -> 电子排布 -> 高低自旋 -> 磁性/颜色`

## 二、考纲定位
- 要求层级：理解 d 轨道分裂、高低自旋、CFSE 与颜色/磁性的关系
- 课堂定位：是 [[配位化学]]、[[光谱化学序列]]、[[高自旋与低自旋]] 的核心桥梁页

## 三、核心原理

### 3.1 八面体场分裂
在八面体场中，5 个 d 轨道分裂成两组：
- $t_{2g}$：较低能
- $e_g$：较高能

分裂能记为 $\Delta_o$ 或 $10Dq$。

$$
E(t_{2g})=-\frac{2}{5}\Delta_o,\quad E(e_g)=+\frac{3}{5}\Delta_o
$$

### 3.2 四面体场分裂
四面体场中的分裂顺序与八面体相反，且分裂能更小：

$$
\Delta_t \approx \frac{4}{9}\Delta_o
$$

因此四面体配合物通常更容易表现为**高自旋**。

### 3.3 分裂能与成对能
- $\Delta$：分裂能
- $P$：成对能

比较二者即可判断自旋状态：
- $\Delta < P$：高自旋
- $\Delta > P$：低自旋

### 3.4 高低自旋的高频体系
主要要盯住 $d^4$ 到 $d^7$：
- 强场：更容易低自旋
- 弱场：更容易高自旋

### 3.5 晶体场稳定化能 CFSE
CFSE 反映 d 电子进入较低能轨道后带来的稳定化。

八面体场下可写作：

$$
\mathrm{CFSE}=(-4n_{t_{2g}}+6n_{e_g})Dq+(m_1-m_2)P
$$

课堂上第一轮不必把全部电子数逐个背完，但要能读懂：
- 低能轨道占得越多，通常越稳定
- 但若多出电子对，也要付成对能代价

### 3.6 配位光谱的最小课堂链
一旦课堂开始讨论“为什么某配合物有颜色、颜色深浅为什么不同”，就不能只停在“有分裂所以有颜色”，而要把链条补完整：

`d电子数与构型 -> 分裂能大小 -> 跃迁类型 -> 允许/禁阻 -> 吸收位置与强度 -> 显色`

| 关心的问题 | 最低判断动作 | 课堂高频结论 |
|------|------|------|
| 有没有 d-d 跃迁？ | 先看 d 电子数与构型 | d0 / d10 通常无 d-d 跃迁 |
| 为什么有的颜色很浅？ | 看选律 | 八面体中心对称体系的 d-d 跃迁常受 Laporte 禁阻，颜色往往偏浅 |
| 为什么四面体常更鲜艳？ | 看对称性 | Td 无反演中心，d-d 跃迁禁阻较弱，吸收强度常更大 |
| 为什么有的带特别强？ | 区分 d-d 与 CT | 电荷转移带通常比 d-d 带强得多 |
| 什么时候要升级到 TS 图？ | 看是否进入多电子定量归属 | 需要从多条吸收带反推 $\Delta_o$、B 或归属跃迁时，转去 [[Tanabe-Sugano图]] |

这里最值得学生先建立的不是复杂图，而是两个判断：
- **颜色位置主要看 $\Delta$**
- **颜色深浅主要看跃迁是否允许**

## 四、关键结论
- 八面体场中 $t_{2g}$ 低、$e_g$ 高
- 四面体场分裂更小，因此通常高自旋
- 强场/弱场配体通过改变 $\Delta$ 来改变电子排布
- 高低自旋会直接影响磁性、颜色和稳定性
- 颜色不只看 $\Delta$，还要看跃迁是否允许；八面体 d-d 带常弱，CT 带常强

## 五、常见分类或情形

### 5.1 八面体、四面体与平面正方形
| 构型 | 分裂特点 | 课堂高频结论 |
|------|------|------|
| 八面体 | $t_{2g}$ / $e_g$ 分裂 | 最常见，最适合讲高低自旋 |
| 四面体 | 分裂更小且顺序相反 | 通常高自旋 |
| 平面正方形 | d8 体系高频 | 常见于强场 Ni(II)、Pd(II)、Pt(II) |

### 5.2 典型例子
- $[FeF_6]^{3-}$：弱场，高自旋
- $[Fe(CN)_6]^{3-}$：强场，低自旋
- $[Ni(CN)_4]^{2-}$：d8 强场，常见平面正方形
- Cu2+：d9，高频出现 Jahn-Teller 畸变

### 5.3 Jahn-Teller 效应
当电子在简并轨道中分布不对称时，分子会通过几何畸变降低能量。  
最常见课堂例子：Cu2+ 的八面体配合物常出现轴向拉长。

### 5.4 配位光谱里的三种高频吸收
| 吸收类型 | 课堂来源 | 强度特征 | 课堂判断口径 |
|------|------|------|------|
| d-d 跃迁 | 分裂后的 d 轨道之间跃迁 | 常较弱 | 先看 d 电子数、构型与选律 |
| LMCT | 配体到金属的电荷转移 | 常很强 | 高氧化态金属、易给电子配体常高频 |
| MLCT | 金属到配体的电荷转移 | 常很强 | 低价金属配强受体配体时常见 |

## 六、适用范围与限制
- 对过渡金属配合物的磁性、颜色、构型判断非常有效
- 纯晶体场理论偏静电模型，解释强场/弱场本质时不如配位场/分子轨道图景完整
- 对主族配合物和 f 区体系，不宜机械套用这一套

## 七、常见比较与易混点
- **强场不是“电荷更大”**：关键是让 d 轨道分裂得更大
- **四面体场不是八面体场简单缩小版**：分裂顺序也反过来
- **不是所有 d 电子构型都要谈高低自旋**：课堂高频主要在 $d^4$-$d^7$
- **CFSE 不是只看低能轨道占了几个**：还要注意成对能

## 十、例题
- **例题 1**：判断 $[FeF_6]^{3-}$ 与 $[Fe(CN)_6]^{3-}$ 的高低自旋。  
  **答案要点**：F- 弱场，$\Delta<P$，高自旋；CN- 强场，$\Delta>P$，低自旋。
- **例题 2**：为什么四面体配合物通常高自旋？  
  **答案要点**：$\Delta_t$ 较小，通常不足以克服成对能。

## 十一、易错点
- 把“强场/弱场”误解成配体电荷大小
- 只记结论，不先判断构型和 $\Delta$ 与 $P$
- 把八面体与四面体的分裂顺序记反
- 忽略 Jahn-Teller 畸变在 Cu2+ 等体系中的高频性

## 十二、🎯 教学视角
- 第一轮不必做成长篇决赛推导，重点是把“分裂 -> 排布 -> 性质”链条讲稳
- 最有效的课堂抓手是：八面体分裂图、强弱场配体、高低自旋、Cu2+ 畸变
- 最适合和 `Fe / Co / Ni / Cu` 单元素页联动使用
- 讲到颜色时，先补“d-d vs CT、允许 vs 禁阻”两层判断，不要立刻把学生推去背 TS 图
- 真正进入 [[Tanabe-Sugano图]] 的时机，是“多电子体系吸收带归属与参数反推”，不是“只要提到颜色就上 TS 图”

## 十三、🧰 备课可抽料资产

### 13.1 主干结论
- d 轨道分裂是过渡金属配合物磁性和颜色的核心来源
- 分裂能与成对能的比较决定高低自旋
- 高低自旋再进一步决定磁矩和稳定性差异

### 13.2 机制 / 推导资产
- 可直接复用的课堂链：构型 -> 分裂 -> 排布 -> 未成对电子数 -> 磁性/颜色
- 可直接复用的桥句：“先分裂，再排布，最后看性质”
- 可直接复用的配位光谱链：`d电子数 -> d-d/CT -> 允许性 -> 颜色深浅 -> 是否升级到 TS 图`

### 13.3 误区 / 纠偏资产
- “强场不是电荷强，而是分裂强”
- “四面体几乎总高自旋”
- “先定构型，再谈电子排布”
- “颜色深不一定因为 $\Delta$ 大，也可能只是 CT 带更强”

### 13.4 题型 / 例题资产
- 建议题链：八面体 d5 -> d6 -> 四面体高自旋 -> d8 平面正方形
- 建议桥题：Mn2+ 为什么颜色极浅 -> Co2+ 四面体为什么常更鲜艳 -> Cr3+ 为什么适合上 TS 图
