# 硅

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]

## 一、定义
- 地壳中含量第二的元素（26.3%，仅次于 O）
- 自然界以 SiO₂ 和硅酸盐形式存在
- **结构与碳的本质不同**：Si 不会形成 sp²/sp π 键（d 轨道可用但 pπ-pπ 弱）→ 没有"硅苯"类似物

## 二、考纲对应
- 对应考纲条目：[[13-元素化学]]
- 所属模块：[[基础要求-无机和结构化学]]

## 三、核心原理

### 3.1 硅氧骨干结构
硅的化学以 Si—O 为骨干。硅酸盐的基本结构单元是硅氧四面体 $\mathrm{SiO}_4^{4-}$（Al 可取代部分 Si 的位置成铝硅酸盐）。按硅氧四面体结合方式不同分为四大类：
- **有限硅氧团**：单个 $\mathrm{SiO}_4^{4-}$（如锆英石 $\mathrm{ZrSiO}_4$、镁橄榄石 $\mathrm{Mg}_2\mathrm{SiO}_4$），或 2 个四面体共用 1 个顶点的 $\mathrm{Si}_2\mathrm{O}_7^{6-}$，或 3 个四面体各有 2 个共用顶点成环的 $\mathrm{Si}_3\mathrm{O}_9^{6-}$
- **链型**：2 个氧原子分别与其他四面体共用成链，$\mathrm{Si}:\mathrm{O} = 1:3$，写作 $(\mathrm{SiO}_3)_n^{2n-}$，如透辉石 $\mathrm{MgCa}(\mathrm{SiO}_3)_2$
- **层型**：3 个氧原子分别共用成片状，$\mathrm{Si}:\mathrm{O} = 1:2.5$，即 $(\mathrm{Si}_2\mathrm{O}_5)_n^{2n-}$，如白云母
- **骨架型**：4 个氧原子全部共用，$\mathrm{Si}:\mathrm{O}= 1:2$，即 $(\mathrm{SiO}_2)_n$（石英）。钾长石 $\mathrm{K(AlSi}_3\mathrm{O}_8)$ 中有 1/4 的硅位置为 Al 所取代

### 3.2 碳硅对比
碳和硅虽然都容易形成四价化合物，但性质并不相似：
- $\mathrm{CO}_2$ 是气体，$\mathrm{SiO}_2$ 是固体
- $\mathrm{CH}_4$ 稳定，$\mathrm{SiH}_4$ 不稳定
- $\mathrm{H}_2\mathrm{CO}_3$ 存在于水溶液，$\mathrm{H}_2\mathrm{SiO}_3$ 不溶于水
- $\mathrm{CCl}_4$ 不水解，$\mathrm{SiCl}_4$ 容易水解

### 3.3 超纯硅的制备
电子工业用硅纯度要求达 $99.9999999\%$（"9 个 9"）。冶炼过程：
1. 电弧炉中焦炭还原石英砂：$\mathrm{SiO}_2 + 2\mathrm{C} \xrightarrow{1500^{\circ}\mathrm{C}} \mathrm{Si} + 2\mathrm{CO}$（纯度 $96\%\sim 97\%$）
2. 粗硅制三氯氢硅：$\mathrm{Si} + 3\mathrm{HCl} \xrightarrow{280\sim 330^{\circ}\mathrm{C}} \mathrm{SiHCl}_3 + \mathrm{H}_2$（精馏得"7 个 9"纯度）
3. 气相沉积 (CVD)：$\mathrm{SiHCl}_3 + \mathrm{H}_2 \xrightarrow{1050\sim 1150^{\circ}\mathrm{C}} \mathrm{Si} + 3\mathrm{HCl}$（得"9 个 9"多晶硅）
4. 区域熔融精炼制成单晶硅

## 四、关键结论
- 硅酸盐共性：水中溶解度小、耐高温、耐腐蚀、非导体
- Si—O 键 ($368\mathrm{kJ}\cdot\mathrm{mol}^{-1}$) 是最强的硅键，决定了硅化学以硅氧骨干为特征
- 硅不能像碳那样形成 $\mathrm{sp}^2$/$\mathrm{sp}$ 多重键（$p\pi-p\pi$ 键弱），因此没有"硅苯"类似物
- $\mathrm{SiO}_2$ 是酸性氧化物，可溶于碱和 HF
- 硅是重要的半导体材料，"硅谷"得名于此

## 五、常见分类或情形
- **有限硅氧团**：$\mathrm{SiO}_4^{4-}$（正硅酸盐）、$\mathrm{Si}_2\mathrm{O}_7^{6-}$（焦硅酸盐）、$\mathrm{Si}_3\mathrm{O}_9^{6-}$（环状）
- **链型硅酸盐**：$(\mathrm{SiO}_3)_n^{2n-}$（透辉石等）
- **层型硅酸盐**：$(\mathrm{Si}_2\mathrm{O}_5)_n^{2n-}$（云母、黏土）
- **骨架型硅酸盐**：$\mathrm{SiO}_2$（石英）、长石类等
- **分子筛**：多孔铝硅酸盐（4A、5A、3A 等）

## 六、适用条件与限制
- 硅酸盐在水中溶解度小，矿石分解需用酸法、碱法或火法
- 超纯硅的 CVD 制备需严格控制温度和无水无氧条件
- 分子筛的孔径和 Si/Al 比可通过制备条件（温度、配比）调控
- Si—Si 键很弱 ($197\mathrm{kJ}\cdot\mathrm{mol}^{-1}$)，硅烷长链不稳定

## 七、常见比较与易混点
| 对比项 | $\mathrm{CO}_2$ | $\mathrm{SiO}_2$ |
|------|------|------|
| 状态 | 气体 | 原子晶体（固体） |
| 结构 | 直线形分子 | 三维骨架（Si—O—Si） |
| 与水反应 | 生成 $\mathrm{H}_2\mathrm{CO}_3$ | 不溶于水 |
| 与 NaOH | 生成 $\mathrm{Na}_2\mathrm{CO}_3$ | 生成 $\mathrm{Na}_2\mathrm{SiO}_3$ |

- 硅酸盐 vs 传统含氧酸盐：不能用正、偏、焦等简单命名表明其结构差异，需按硅氧四面体连接方式分类
- B—Si 对角规则：B 和 Si 性质相似，$\mathrm{H}_3\mathrm{BO}_3$ 和 $\mathrm{H}_2\mathrm{SiO}_3$ 都是弱酸

## 九、典型题型

- 元素化学比较题：比较 C 和 Si 的氧化物、氢化物、氯化物稳定性与水解性
- 结构分类题：根据 Si:O 比或顶点共享数判断硅酸盐属于岛状、链状、层状还是骨架型
- 工业流程题：围绕粗硅提纯、SiHCl3 精馏和 CVD 沉积进行方程式与流程判断

## 十、例题

### 例题 1
**题目：** 为什么 SiCl4 易水解，而 CCl4 基本不水解？

**分析：** 关键不在“都四氯化”，而在中心原子大小、成键和受进攻能力不同。

**解答：** Si 原子半径更大、空轨道更易接受水分子进攻，且生成强 Si-O 键有明显热力学驱动力，因此 SiCl4 易水解；CCl4 中碳原子较小且难以形成稳定过渡结构，同时生成新键驱动力弱得多，因此基本不水解。

**反思：** 碳硅对比题不能只背现象，要回到成键本质。

## 十一、易错点

- 误把 SiO2 当作分子晶体，忽略其三维骨架网络
- 误认为硅和碳同族就会像碳一样稳定形成双键、三键
- 只会背 $(SiO3)_n^{2n-}$、$(Si2O5)_n^{2n-}$，却不会从共享顶点数解释来源

## 十二、🎯 教学视角

- 第一轮只要把“Si-O骨架”和“碳硅对比”讲透，很多零散现象会自动串起来
- 结构分类时优先从四面体共享顶点数入手，比直接背矿物名称更稳
