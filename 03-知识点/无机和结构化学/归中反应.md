## 一、定义

**归中反应**（comproportionation，又称归中反应、均化反应）是指同一种元素的不同氧化态物种反应，生成该元素中间氧化态产物的反应。归中反应是歧化反应（disproportionation）的逆过程。

**歧化反应**：同一元素的一种氧化态 → 更高氧化态 + 更低氧化态
**归中反应**：更高氧化态 + 更低氧化态 → 中间氧化态

典型例子：
- $2\mathrm{H}_2\mathrm{S} + \mathrm{SO}_2 \rightarrow 3\mathrm{S} + 2\mathrm{H}_2\mathrm{O}$（S从-2和+4归中为0）
- $6\mathrm{HCl} + \mathrm{KClO}_3 \rightarrow \mathrm{KCl} + 3\mathrm{Cl}_2 + 3\mathrm{H}_2\mathrm{O}$（Cl从-1和+5归中为0）
- $2\mathrm{Cu}^{2+} + \mathrm{Cu} \rightarrow 2\mathrm{Cu}^+$（高温或特定条件下）

## 三、核心原理

### 3.1 热力学判据——元素电势图

利用**Latimer元素电势图**可判断归中反应是否自发：

对于某元素三种氧化态 A（高）→ B（中）→ C（低）：
- 若 $E^\circ(\mathrm{A/B}) > E^\circ(\mathrm{B/C})$，则B不稳定，会**歧化**
- 若 $E^\circ(\mathrm{A/B}) < E^\circ(\mathrm{B/C})$，则B稳定，A和C会**归中**

**规则**：在元素电势图中，某物种右侧电势大于左侧电势时，该物种歧化；反之则归中。

### 3.2 具体判断方法

以氯元素在酸性介质中的电势图为例：
$$\mathrm{ClO}_3^- \xrightarrow{1.47\mathrm{V}} \mathrm{Cl}_2 \xrightarrow{1.36\mathrm{V}} \mathrm{Cl}^-$$

- $E^\circ(\mathrm{ClO}_3^-/\mathrm{Cl}_2) = 1.47\mathrm{V} > E^\circ(\mathrm{Cl}_2/\mathrm{Cl}^-) = 1.36\mathrm{V}$
- 因此Cl₂在酸性中**歧化**趋势不大，但ClO₃⁻和Cl⁻可发生**归中**反应生成Cl₂

### 3.3 与歧化反应的关系

| 反应类型 | 方向 | 电势图特征 | 例子 |
|:---|:---|:---|:---|
| 歧化 | 中间→高+低 | $E^\circ_\text{右} > E^\circ_\text{左}$ | Cl₂ + 2OH⁻ → Cl⁻ + ClO⁻ + H₂O |
| 归中 | 高+低→中间 | $E^\circ_\text{右} < E^\circ_\text{左}$ | ClO₃⁻ + 5Cl⁻ + 6H⁺ → 3Cl₂ + 3H₂O |

## 四、关键结论

1. **归中反应的本质**：氧化剂（高氧化态）和还原剂（低氧化态）是同一种元素
2. **产物氧化态**：归中产物氧化态一定在反应物氧化态之间
3. **pH影响**：许多元素的电势图随pH变化，酸碱条件下归中/歧化行为可能不同
4. **卤素系统**：ClO₃⁻/Cl⁻、BrO₃⁻/Br⁻、IO₃⁻/I⁻在酸性中均可发生归中反应生成单质

## 五、常见分类或情形

### 5.1 按元素分类

| 元素 | 归中反应实例 | 条件 |
|:---|:---|:---|
| 硫 | $2\mathrm{H}_2\mathrm{S} + \mathrm{SO}_2 \rightarrow 3\mathrm{S} + 2\mathrm{H}_2\mathrm{O}$ | 气相或溶液 |
| 氯 | $\mathrm{KClO}_3 + 6\mathrm{HCl} \rightarrow \mathrm{KCl} + 3\mathrm{Cl}_2 + 3\mathrm{H}_2\mathrm{O}$ | 酸性 |
| 氮 | $2\mathrm{NO}_2 + 2\mathrm{NaOH} \rightarrow \mathrm{NaNO}_3 + \mathrm{NaNO}_2 + \mathrm{H}_2\mathrm{O}$（歧化为主）| 碱性 |
| 磷 | $4\mathrm{P} + 3\mathrm{NaOH} + 3\mathrm{H}_2\mathrm{O} \rightarrow 3\mathrm{NaH}_2\mathrm{PO}_2 + \mathrm{PH}_3$（歧化）| 碱性 |
| 锰 | $2\mathrm{KMnO}_4 + 3\mathrm{MnO}_2 + 2\mathrm{H}_2\mathrm{O} \rightarrow 5\mathrm{MnO(OH)}_2$ | 特定条件 |

### 5.2 按反应条件分类

- **酸性归中**：卤素含氧酸与卤化物反应
- **碱性歧化**（逆过程）：卤素在碱中歧化
- **热归中**：高温下氧化态趋于平均化

## 六、适用条件与限制

1. 归中反应要求中间氧化态在热力学上稳定
2. 动力学因素可能阻止热力学允许的归中反应
3. 过强氧化剂/还原剂可能导致完全氧化/还原，不归中

## 七、常见比较与易混点

| 对比项 | 归中反应 | 歧化反应 |
|:---|:---|:---|
| 反应物氧化态 | 两种不同氧化态 | 一种中间氧化态 |
| 产物氧化态 | 中间氧化态 | 更高和更低氧化态 |
| 电势图判据 | $E^\circ_\text{右} < E^\circ_\text{左}$ | $E^\circ_\text{右} > E^\circ_\text{左}$ |
| 本质 | 同元素自身氧化还原 | 同元素自身氧化还原 |
| 关系 | 互为逆反应 | 互为逆反应 |

**易混点**：
- 不要把归中反应与一般的氧化还原反应混淆——归中反应的反应物和产物必须含同一种元素的不同氧化态
- 判断归中还是歧化，关键是看中间氧化态物种在电势图中的位置

## 十、例题

**例题1**：根据氯元素在碱性中的电势图判断Cl₂能否歧化。
$$\mathrm{ClO}^- \xrightarrow{0.40\mathrm{V}} \mathrm{Cl}_2 \xrightarrow{1.36\mathrm{V}} \mathrm{Cl}^-$$

**解**：$E^\circ(\mathrm{Cl}_2/\mathrm{Cl}^-) = 1.36\mathrm{V} > E^\circ(\mathrm{ClO}^-/\mathrm{Cl}_2) = 0.40\mathrm{V}$
- 右侧电势 > 左侧电势
- Cl₂在碱性中**歧化**：$\mathrm{Cl}_2 + 2\mathrm{OH}^- \rightarrow \mathrm{Cl}^- + \mathrm{ClO}^- + \mathrm{H}_2\mathrm{O}$

**例题2**：实验室用KClO₃和浓HCl制Cl₂，写出反应方程式。

**解**：
- Cl在KClO₃中为+5，在HCl中为-1
- 归中产物为Cl₂（0价）
- $\mathrm{KClO}_3 + 6\mathrm{HCl} \rightarrow \mathrm{KCl} + 3\mathrm{Cl}_2 + 3\mathrm{H}_2\mathrm{O}$

## 十一、易错点

1. 配平归中反应时，注意电子得失守恒
2. 不要混淆归中反应物和产物的氧化态关系
3. 元素电势图中"右侧"和"左侧"的判断要准确

## 十三、竞赛拓展

1. **Latimer 图判断归中/歧化的定量方法**：对于三物种 A —E₁— B —E₂— C，若 $E_1 > E_2$ 则 B 歧化为 A+C（$\Delta G < 0$）；若 $E_1 < E_2$ 则 A+C 归中为 B。竞赛计算：$\Delta G^\ominus = -nF[E^\ominus(A/B) - E^\ominus(B/C)]$。当 $E_1 < E_2$，A 氧化 C 的 $\Delta G^\ominus < 0$ → 归中自发。典型竞赛题：已知 $E^\ominus(\mathrm{ClO_3^-/Cl_2})=1.47$ V、$E^\ominus(\mathrm{Cl_2/Cl^-})=1.36$ V → $\mathrm{ClO_3^- + 5Cl^- + 6H^+ \to 3Cl_2 + 3H_2O}$ 的 $\Delta G^\ominus$ 和平衡常数 $K$。

2. **pH 依赖的归中/歧化反转**：某些元素的电势对 pH 敏感 → 在酸性和碱性条件下归中/歧化行为可能反转。例如碘：酸性中 IO₃⁻/I₂/I⁻ → $E_1 \approx 1.20$ V, $E_2 \approx 0.54$ V → 归中自发（$E_1 > E_2$ → IO₃⁻ 氧化 I⁻ 生成 I₂）。碱性中：IO₃⁻/I₂(OH⁻) → $E$ 值变化 → 可能改变方向。这类题目需要学生结合 Nernst 方程计算不同 pH 下的条件电势。
