# 铬

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]

## 一、定义
- [Ar]3d⁵4s¹（半满稳定！异常排布）
- 重要氧化态：+3（最稳定）、+6（强氧化性）
- Cr 表现多种鲜艳颜色（"chromium"原意即颜色）

## 二、考纲对应
- 对应考纲条目：[[13-元素化学]]
- 所属模块：[[基础要求-无机和结构化学]]

## 三、核心原理

### 3.1 Cr(III) 与 Cr(VI) 的相互转换
Cr 的常见氧化数为 $+3$ 和 $+6$。在酸性介质中 Cr(VI) 的氧化性较强，在碱性介质中 Cr(III) 的还原性较强：
- 酸性：$\mathrm{Cr}_2\mathrm{O}_7^{2-} + 14\mathrm{H}^{+} + 6\mathrm{e} \rightleftharpoons 2\mathrm{Cr}^{3+} + 7\mathrm{H}_2\mathrm{O}$，$E^{\ominus} = 1.36\mathrm{V}$
- 碱性：$\mathrm{CrO}_4^{2-} + 4\mathrm{H}_2\mathrm{O} + 3\mathrm{e} \rightleftharpoons \mathrm{Cr(OH)}_3 + 5\mathrm{OH}^{-}$，$E^{\ominus} = -0.13\mathrm{V}$

因此，将 Cr(VI) 还原到 Cr(III) 宜在酸性溶液中进行；将 Cr(III) 氧化为 Cr(VI) 则宜在碱性条件下进行。

### 3.2 铬酸根-重铬酸根平衡
$$2\mathrm{CrO}_4^{2-} + 2\mathrm{H}^{+} \rightleftharpoons \mathrm{Cr}_2\mathrm{O}_7^{2-} + \mathrm{H}_2\mathrm{O}$$
（黄色 $\mathrm{CrO}_4^{2-}$，橙色 $\mathrm{Cr}_2\mathrm{O}_7^{2-}$），平衡常数大约 $10^{14}$。低 pH 时以 $\mathrm{Cr}_2\mathrm{O}_7^{2-}$ 为主（橙色），高 pH 时以 $\mathrm{CrO}_4^{2-}$ 为主（黄色）。

### 3.3 铬的过氧化合物
向上层有乙醚的 $\mathrm{H}_2\mathrm{O}_2$ 稀 $\mathrm{H}_2\mathrm{SO}_4$ 溶液中加入 $\mathrm{Cr}_2\mathrm{O}_7^{2-}$，乙醚层显深蓝色（过氧化铬 $\mathrm{CrO}_5$ 的乙醚复合物），常用此反应鉴定 Cr(VI)：
$$\mathrm{Cr}_2\mathrm{O}_7^{2-} + 4\mathrm{H}_2\mathrm{O}_2 + 2\mathrm{H}^{+} \longrightarrow 2\mathrm{CrO}_5 + 5\mathrm{H}_2\mathrm{O}$$

### 3.4 铬的冶炼（碱法）
铬的主要矿石是铬铁矿 $\mathrm{FeCr}_2\mathrm{O}_4$。用 $\mathrm{Na}_2\mathrm{CO}_3$ 和 $\mathrm{NaOH}$ 与它共熔，同时 Cr(III) 被空气中 $\mathrm{O}_2$ 氧化为 Cr(VI)：
$$4\mathrm{FeCr}_2\mathrm{O}_4 + 8\mathrm{Na}_2\mathrm{CO}_3 + 7\mathrm{O}_2 \longrightarrow 2\mathrm{Fe}_2\mathrm{O}_3 + 8\mathrm{Na}_2\mathrm{CrO}_4 + 8\mathrm{CO}_2$$
用水将 $\mathrm{Na}_2\mathrm{CrO}_4$ 溶出，加酸变为 $\mathrm{Na}_2\mathrm{Cr}_2\mathrm{O}_7$，可用 C 还原为 $\mathrm{Cr}_2\mathrm{O}_3$，再用 Al 还原为金属 Cr。

### 3.5 多酸化学
重铬酸 ($\mathrm{H}_2\mathrm{Cr}_2\mathrm{O}_7$) 属同多酸类的二聚铬酸。同族的 Mo 和 W 也容易形成多酸，如七钼酸根 $\mathrm{H}_6\mathrm{Mo}_7\mathrm{O}_{24}$、十二钼磷杂多酸 $\mathrm{H}_3\mathrm{PMo}_{12}\mathrm{O}_{40}$（Keggin 型）、十二钨硅杂多酸 $\mathrm{H}_4\mathrm{SiW}_{12}\mathrm{O}_{40}$ 等。

## 四、关键结论
- Cr 的电子排布为 $[\mathrm{Ar}]3\mathrm{d}^5 4\mathrm{s}^1$（半满稳定，异常排布）
- $\mathrm{Cr}_2\mathrm{O}_7^{2-}$（橙色）和 $\mathrm{CrO}_4^{2-}$（黄色）的相互转化取决于 pH
- 酸性介质中 $\mathrm{Cr}_2\mathrm{O}_7^{2-}$ 是强氧化剂（$E^{\ominus} = 1.36\mathrm{V}$）
- Cr(III) 在碱性条件下可被氧化为 Cr(VI)；反之在酸性条件下 Cr(VI) 被还原为 Cr(III)
- 过氧化铬乙醚复合物呈深蓝色，用于鉴定 Cr(VI)
- Cr 是最硬的金属；铬铁矿冶炼用碱法（$\mathrm{Na}_2\mathrm{CO}_3$ 共熔）

## 五、常见分类或情形
- **Cr(III) 化合物**：$\mathrm{Cr}_2\mathrm{O}_3$（绿色，两性）、$\mathrm{Cr(OH)}_3$（灰绿，两性）、$\mathrm{Cr}^{3+}$ 盐
- **Cr(VI) 化合物**：$\mathrm{CrO}_3$（暗红色固体）、铬酸盐（黄）、重铬酸盐（橙）
- **同多酸/杂多酸**：$\mathrm{H}_2\mathrm{Cr}_2\mathrm{O}_7$（二聚）、$\mathrm{H}_2\mathrm{Cr}_3\mathrm{O}_{10}$（三聚）、十二钼磷杂多酸
- **铬的其他化合物**：$\mathrm{CrO}_2\mathrm{Cl}_2$（铬酰氯）、$\mathrm{CrO}_5$（过氧化铬）

## 六、适用条件与限制
- $\mathrm{CrO}_3$ 和重铬酸盐是强氧化剂，使用时注意安全
- 铬酸洗液（$\mathrm{K}_2\mathrm{Cr}_2\mathrm{O}_7$ + 浓 $\mathrm{H}_2\mathrm{SO}_4$）有强氧化性和腐蚀性
- $\mathrm{CrO}_5$ 不稳定，放置后分解生成 $\mathrm{Cr}^{3+}$ 和 $\mathrm{O}_2$
- $\mathrm{Cr(OH)}_3$ 具有两性，在过量强碱中溶解得 $\mathrm{Cr(OH)}_4^{-}$ 或 $\mathrm{CrO}_2^{-}$
- 铬铁矿焙烧需在高温和碱性条件下进行

## 七、常见比较与易混点
| 介质 | Cr(VI) 形态 | 颜色 | 氧化性 |
|------|------|:---:|:---:|
| 酸性 | $\mathrm{Cr}_2\mathrm{O}_7^{2-}$ | 橙色 | 强（$E^{\ominus}=1.36\mathrm{V}$） |
| 碱性 | $\mathrm{CrO}_4^{2-}$ | 黄色 | 弱（$E^{\ominus}=-0.13\mathrm{V}$） |

- $\mathrm{Cr}_2\mathrm{O}_7^{2-}$ vs $\mathrm{MnO}_4^{-}$：都是常用氧化剂，前者 $E^{\ominus}=1.36\mathrm{V}$，后者 $E^{\ominus}=1.51\mathrm{V}$
- 同多酸 vs 杂多酸：同一种酸根缩合为同多酸；不同酸根缩合为杂多酸

## 十、例题
- **例题**：向黄色 $\mathrm{K_2CrO_4}$ 溶液中逐滴加入酸，颜色如何变化？再加碱又如何？  
  **答案要点**：加酸后平衡向 $\mathrm{Cr_2O_7^{2-}}$ 方向移动，溶液由黄变橙；再加碱则逆向移动，橙变黄。核心是
  $$2\mathrm{CrO_4^{2-}}+2\mathrm{H^+}\rightleftharpoons \mathrm{Cr_2O_7^{2-}}+\mathrm{H_2O}$$

## 十一、易错点
- 误把铬酸根/重铬酸根转化当成氧化还原，实际上这里首先是酸碱平衡与缩合平衡
- 误认为 Cr(VI) 在任何条件下都是强氧化剂，忽略介质对电势的显著影响
- 只记颜色不记条件，导致黄橙转化题和鉴定题容易丢分

## 十二、🎯 教学视角
- 第一轮建议突出“颜色 + 介质 + 价态”三联线索，学生最容易形成稳定判断
- 最好与锰酸根/高锰酸根做并列比较，帮助学生理解“强氧化性依赖酸性介质”的普遍性
- 若要加深，可把异常排布 $3d^5 4s^1$ 作为电子结构与元素性质联系的一个经典例子
