# 分数坐标

> 核心内容基于《普通化学原理》第13章、Atkins物理化学主题15及竞赛真题提炼。分数坐标是晶体结构计算的核心工具。

## 一、定义

**分数坐标**（fractional coordinates）是指晶体中原子位置相对于晶胞边长的无量纲坐标。设晶胞的三条边为基矢 **a**、**b**、**c**，则晶胞中任意一点的位置可表示为：

$$\mathbf{r} = x\mathbf{a} + y\mathbf{b} + z\mathbf{c}$$

其中 $x, y, z$ 为分数坐标，取值范围通常为 $[0, 1)$。

**特点**：
- 分数坐标与晶胞大小无关，只与原子在晶胞中的相对位置有关
- 同一晶体结构，无论晶胞参数如何变化，分数坐标保持不变
- 是晶体学数据（如ICSD、CSD数据库）中记录原子位置的标准方式

## 三、核心原理

### 3.1 分数坐标与直角坐标的转换

**直角坐标** = 分数坐标 × 晶胞参数（需考虑晶胞夹角）

对于正交/立方晶系（$\alpha = \beta = \gamma = 90°$）：
$$\mathbf{r}_\text{直角} = (xa, yb, zc)$$

对于一般晶系，需用度量张量转换：
$$\mathbf{r} = x\mathbf{a} + y\mathbf{b} + z\mathbf{c}$$

### 3.2 常见位置的分数坐标

**立方晶系**：

| 位置 | 分数坐标 | 说明 |
|:---|:---|:---|
| 原点 | (0, 0, 0) | 晶胞顶点 |
| 体心 | (1/2, 1/2, 1/2) | BCC晶胞中心 |
| 面心 | (1/2, 1/2, 0) 等 | FCC晶胞面心 |
| 棱心 | (1/2, 0, 0) 等 | 棱中点 |
| 四面体空隙 | (1/4, 1/4, 1/4) 等 | FCC中8个位置 |
| 八面体空隙 | (1/2, 1/2, 1/2) 等 | FCC中4个位置 |

### 3.3 晶胞中原子间距的计算

对于立方晶系，两原子 $(x_1, y_1, z_1)$ 和 $(x_2, y_2, z_2)$ 之间的距离：

$$d = a\sqrt{(\Delta x)^2 + (\Delta y)^2 + (\Delta z)^2}$$

其中 $\Delta x = x_2 - x_1$（需考虑周期性边界条件，取最小值）。

**周期性边界条件**：若 $|\Delta x| > 1/2$，则用 $1 - |\Delta x|$ 替代。

## 四、关键结论

1. **分数坐标是晶体学的"通用语言"**：同一结构的不同晶胞选取，分数坐标可能不同，但原子间相对位置等价
2. **对称操作对分数坐标的影响**：旋转、反映、反演等对称操作可通过矩阵作用于分数坐标
3. **等效位置**：由对称操作联系的一组分数坐标称为等效位置（等效点系）
4. **晶胞中原子数计算**：通过分数坐标和占位率（occupancy）计算

## 五、常见分类或情形

### 5.1 按晶系分类的坐标特点

| 晶系 | 坐标特点 | 独立参数 |
|:---|:---|:---:|
| 立方 | a = b = c, 所有角90° | 1个 (a) |
| 六方 | a = b ≠ c, γ = 120° | 2个 (a, c) |
| 四方 | a = b ≠ c, 所有角90° | 2个 (a, c) |
| 正交 | a ≠ b ≠ c, 所有角90° | 3个 (a, b, c) |
| 单斜 | a ≠ b ≠ c, β ≠ 90° | 4个 (a, b, c, β) |
| 三斜 | 全部不等 | 6个 |

### 5.2 典型晶体结构的分数坐标

**NaCl型**（面心立方，Fm-3m）：
- Na⁺: (0, 0, 0); (1/2, 1/2, 0); (1/2, 0, 1/2); (0, 1/2, 1/2)
- Cl⁻: (1/2, 1/2, 1/2); (0, 0, 1/2); (0, 1/2, 0); (1/2, 0, 0)

**CsCl型**（简单立方，Pm-3m）：
- Cs⁺: (0, 0, 0)
- Cl⁻: (1/2, 1/2, 1/2)

**金刚石型**（面心立方，Fd-3m）：
- C: (0, 0, 0); (1/4, 1/4, 1/4) + FCC平移

## 六、适用条件与限制

1. 分数坐标只适用于周期性晶体结构
2. 不同晶胞选取（如原胞vs惯用晶胞）分数坐标不同
3. 非整数坐标表示原子在晶胞内部，不是缺陷

## 七、常见比较与易混点

| 对比项 | 分数坐标 | 直角坐标 |
|:---|:---|:---|
| 单位 | 无量纲 | 长度单位（Å, nm等）|
| 与晶胞大小关系 | 无关 | 直接相关 |
| 适用范围 | 晶体学标准 | 实际距离计算 |
| 数据库记录 | 常用 | 不常用 |

**易混点**：
- 分数坐标中的1/2不等于"在晶胞中间"——需考虑具体晶胞形状
- 六方晶系的(1/3, 2/3, 1/2)是常见的特殊位置，与六重对称有关
- 计算距离时必须考虑周期性边界条件（最近像原则）

## 九、典型题型

1. 由分数坐标判断原子位置（顶点、面心、体心等）
2. 计算两原子间的距离
3. 由分数坐标推导晶胞中化学式
4. 判断原子是否接触（距离 = 半径和）
5. 六方晶系中的分数坐标与距离计算

## 十、例题

**例题1**：立方晶系中，某原子位于(1/4, 1/4, 1/4)，另一原子位于(0, 0, 0)，晶胞参数a = 0.5 nm。求两原子距离。

**解**：
$$d = a\sqrt{(1/4)^2 + (1/4)^2 + (1/4)^2} = a\sqrt{3}/4 = 0.5 \times 0.433 = 0.217 \text{ nm}$$

**例题2**（第36届初赛第二场）：某立方晶体中Mg²⁺位于(0, 0, 0)和(2/3, 1/3, 1/3)等位置，写出所有Mg²⁺的分数坐标。

**解**：
- 需考虑晶胞的平移对称性
- (0, 0, 0)型：由FCC平移产生4个位置
- (2/3, 1/3, 1/3)型：需确定是否为等效位置
- 具体答案取决于空间群对称性

## 十一、易错点

1. 计算距离时忘记考虑周期性边界条件（如Δx = 0.9时，实际最近距离是0.1）
2. 六方晶系中距离计算需用六方晶系的度量公式，不是简单立方公式
3. 分数坐标(1, 0, 0)等价于(0, 0, 0)（周期性），但习惯上写(0, 0, 0)

## 十三、竞赛拓展

1. **从分数坐标到键长键角的竞赛计算**：这是竞赛结构题的核心能力。已知两原子的分数坐标 $(x_1,y_1,z_1)$ 和 $(x_2,y_2,z_2)$ + 晶胞参数，键长 $d = \sqrt{(\Delta x \cdot a)^2 + (\Delta y \cdot b)^2 + (\Delta z \cdot c)^2 + 2\Delta x\Delta y \cdot ab\cos\gamma + 2\Delta y\Delta z \cdot bc\cos\alpha + 2\Delta z\Delta x \cdot ca\cos\beta}$。对于正交/立方晶系简化为 $d = a\sqrt{(\Delta x)^2 + (b/a)^2(\Delta y)^2 + (c/a)^2(\Delta z)^2}$。三个连续原子 → 用余弦定理算键角。竞赛典型题：已知 NaCl 的 a=564 pm，Na⁺(0,0,0)，Cl⁻(½,0,0) → 键长 = a×½ = 282 pm。

2. **CIF 文件的"阅读"——竞赛信息题的新趋势**：近年竞赛出现给出 CIF（晶体学信息文件）片段的信息题。CIF 包含空间群符号、晶胞参数、原子分数坐标和占有率。考生需要从 CIF 中提取：① 独立原子数 vs 晶胞总原子数（多重性）；② 配位环境（最近邻距离）；③ 结构验证（密度/BVS）。不需要学生自己解析 XRD，但需要能读懂 CIF 的基本字段。
