# 石墨结构

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[17-常见晶体结构]]

## 一、结构描述
- **六方晶系**（hexagonal P）
- 层内：C sp² 杂化 + 离域 πₙ 键（类苯环），C—C 142 pm
- 层间：范德华力，间距 335 pm（远大于层内键长）

## 二、堆垛方式
- ABAB...（Bernal 石墨，最稳定）：一半 C 正对下层环心，一半正对下层 C
- ABCABC...（菱形石墨）

## 三、性质
- 导电（沿层面）：离域 π 电子
- 润滑性：层间滑移
- C—C 键比金刚石还短（π 键强化）

## 四、关键结论
### 混合键型晶体（§13.5.1，图 13.16）
石墨是**典型的混合键型晶体**，含有三种类型的相互作用：
1. **层内 C—C**：sp² 杂化 $\sigma$ 键，键长 $142\,\mathrm{pm}$（比金刚石的 154 pm 还短，因 $\pi$ 键强化）
2. **层内 $\pi$ 键**：各 C 的 p 轨道垂直于平面，相互重叠组成**离域大 $\pi$ 键**，$\pi$ 电子可在整层自由活动 → 类金属导电性
3. **层间**：**范德华力**，C 原子间距 $340\,\mathrm{pm}$（远大于层内 142 pm）→ 层间易滑动，具有润滑性

### 性质
- **导电性**（沿层面）：$\pi$ 电子自由移动，是优良电极材料
- **润滑性**：层间范德华力弱，工业上用做固体润滑剂
- C—C 键比金刚石**更强**（键长更短）

## 五、常见分类或情形
- 六方石墨（Bernal 石墨，ABAB...）：最稳定，一半 C 正对下层环心，一半正对下层 C
- 菱形石墨（ABCABC...）：亚稳相

## 七、常见比较与易混点
- **石墨 vs 金刚石**：同素异形体。石墨导电、软、润滑；金刚石不导电、极硬
- **石墨键型 vs 金刚石键型**：石墨为混合键型（共价 + 金属 + 分子间力），金刚石为纯共价
- **层内键长 142 pm < 金刚石 C—C 154 pm**：$\pi$ 键使层内键更强更短
