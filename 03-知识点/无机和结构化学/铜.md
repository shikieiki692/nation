# 铜

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]

## 一、概述
- [Ar]3d¹⁰4s¹（全满 d¹⁰ 稳定）
- 重要氧化态：+1（固态稳定，水中歧化）、+2（溶液中最稳定）
- 特征：d⁹ → Jahn-Teller 效应，配合物常为变形八面体

## 二、重要反应

### 铜的颜色体系（竞赛经典）
| 物种 | 颜色 | 说明 |
|------|:---:|------|
| Cu | 紫红 | 金属光泽 |
| Cu₂O | 红 | +1 氧化物 |
| CuO | 黑 | +2 氧化物 |
| Cu²⁺(aq) | **蓝色** | [Cu(H₂O)₆]²⁺，Jahn-Teller 变形八面体 |
| Cu(OH)₂ | 浅蓝 | 沉淀 |
| Cu₂S | 黑 | |
| CuS | 黑 | 极难溶（Ksp ~10⁻³⁶） |
| [Cu(NH₃)₄]²⁺ | **深蓝** | 鉴定 Cu²⁺ |
| [CuCl₄]²⁻ | 黄绿 | 四面体 |

### Cu⁺ 的歧化
2Cu⁺ → Cu + Cu²⁺（水溶液中 K ~10⁶，所以 Cu⁺ 在水中不稳定）
但 CuCl, CuI, Cu₂O 因难溶而在固态稳定。

### 重要配合物
- Cu²⁺ + 4NH₃ → [Cu(NH₃)₄]²⁺（深蓝，鉴定）
- [CuCl₄]²⁻（黄绿，四面体，与 [Cu(NH₃)₄]²⁺ 的平面正方形不同）
- 斐林试剂（Fehling）：Cu²⁺ + 酒石酸根（碱性）→ 鉴定还原糖
- 双缩脲反应：Cu²⁺ + 肽键 → 紫色（鉴定蛋白质）

### 与酸的反应
- Cu + 4HNO₃(浓) → Cu(NO₃)₂ + 2NO₂ + 2H₂O
- Cu + H₂SO₄(浓,热) → CuSO₄ + SO₂ + 2H₂O
- 不与稀 HCl/H₂SO₄ 反应

## 三、核心原理

### 3.1 Cu⁺ 的歧化
铜的电势图：$\mathrm{Cu}^{2+} \xrightarrow{0.153\mathrm{V}} \mathrm{Cu}^{+} \xrightarrow{0.521\mathrm{V}} \mathrm{Cu}$

$E^{\ominus}(\mathrm{Cu}^{+}/\mathrm{Cu}) > E^{\ominus}(\mathrm{Cu}^{2+}/\mathrm{Cu}^{+})$，所以在水溶液中 $\mathrm{Cu}^{+}$ 容易歧化变成 $\mathrm{Cu}^{2+}$ 和 $\mathrm{Cu}$。但 CuCl、CuI、$\mathrm{Cu}_2\mathrm{O}$ 因难溶而在固态稳定。在高温情况则不同，黑色的 CuO 加热到 $1026^{\circ}\mathrm{C}$ 以上则分解为红色的 $\mathrm{Cu}_2\mathrm{O}$。

### 3.2 铜的冶炼
现代需要大量纯度高的金属铜作电线，通过电解法进行精炼。电解过程中有多种贵金属（如 Au、Ag、Pt、Pd 等）富集在电解精制铜的阳极泥中。

铜与酸的反应：Cu 不溶于稀 $\mathrm{H}_2\mathrm{SO}_4$ 或稀 HCl，但可溶于 $\mathrm{HNO}_3$ 和浓热 $\mathrm{H}_2\mathrm{SO}_4$。反应不是与 $\mathrm{H}^{+}$ 发生的置换，而是与氧化性酸根（$\mathrm{NO}_3^{-}$、浓 $\mathrm{H}_2\mathrm{SO}_4$）发生的氧化还原。

### 3.3 ds 区与 s 区的区别
IA、IIA 族元素都是活泼金属，而 IB 族元素（Cu、Ag、Au）在常温常压是稳定的金属（怕水怕酸怕氧化的程度远小于碱金属）。铜的常见稳定氧化态为 $+2$（与 IA 族只有 $+1$ 价不同），因为 ns 和 $(n-1)\mathrm{d}$ 电子能量差别小，而水合能、配位能较大等因素导致。

### 3.4 Ellingham 图与冶炼方法比较
从 $\Delta G^{\ominus}-T$ 图（Ellingham 图）可知：
- Cu—CuO 线位于上部，在 $1000^{\circ}\mathrm{C}$ 以下 C 和 CO 都能使 CuO 还原为 Cu
- 炼铜技术比炼铁容易，历史上世界各地冶炼金属的发展过程都是铜先于铁
- 青铜时代：Cu-Sn 合金（青铜），黄铜：Cu-Zn 合金

## 四、关键结论
- Cu⁺ 在水溶液中歧化（$K\sim 10^6$），所以水溶液中 Cu²⁺ 比 Cu⁺ 更常见
- Cu 不溶于稀 HCl/稀 $\mathrm{H}_2\mathrm{SO}_4$（$E^{\ominus}(\mathrm{Cu}^{2+}/\mathrm{Cu})=+0.337\mathrm{V} > 0$）
- 铜族（IB）有货币金属之称：铜钱、银元、金元宝自古就是贸易货币
- 铜的常见稳定氧化态为 $+2$，Ag 为 $+1$，Au 为 $+3$
- 电解精炼铜时贵金属富集在阳极泥中

## 五、常见分类或情形
- **Cu(I) 化合物**：$\mathrm{Cu}_2\mathrm{O}$（红色）、CuCl（白色沉淀）、CuI（白色沉淀）
- **Cu(II) 化合物**：CuO（黑色）、$\mathrm{CuSO}_4\cdot 5\mathrm{H}_2\mathrm{O}$（蓝矾）、$\mathrm{Cu(OH)}_2$（浅蓝沉淀，两性以碱性为主）
- **Cu 的配合物**：$\mathrm{[Cu(NH}_3)_4]^{2+}$（深蓝，鉴定 Cu²⁺）、$\mathrm{[CuCl}_4]^{2-}$（黄绿）
- **铜合金**：青铜（Cu-Sn）、黄铜（Cu-Zn）

## 六、适用条件与限制
- Cu⁺ 盐在固态稳定（难溶于水），在水溶液中歧化
- Cu 与稀 $\mathrm{HNO}_3$ 产生 NO，与浓 $\mathrm{HNO}_3$ 产生 $\mathrm{NO}_2$（实际是混合物）
- 粗铜电解精炼需控制电压和电流密度
- Cu₂O 在高温（$>1026^{\circ}\mathrm{C}$）比 CuO 稳定

## 七、常见比较与易混点
| 对比项 | IA (K, Rb, Cs) | IB (Cu, Ag, Au) |
|------|------|------|
| 与氧作用 | 剧烈 | Cu → CuO；Ag/Au 不反应 |
| 与水作用 | 剧烈 | 不起作用 |
| 氧化态 | 只有 +1 | 有变价 |
| 化合物类型 | 离子型 | 共价型 |
| 配合物形成 | 不易 | 很容易 |

- Cu²⁺ vs Cu⁺ 稳定性：水溶液中 Cu²⁺ 稳定（Cu⁺ 歧化），高温固态 Cu⁺ 稳定（CuO → Cu₂O）
- Cu vs Ag vs Au：常见氧化态分别为 +2、+1、+3（与 ns 和 (n-1)d 电子能量差及水合能有关）

## 十、例题
- **例题**：为什么 Cu 不与稀盐酸反应，却能与浓硝酸反应？  
  **答案要点**：Cu 不能置换非氧化性酸中的 $\mathrm{H}^+$；而浓硝酸中的 $\mathrm{NO_3^-}$ 具有氧化性，可将 Cu 氧化为 $\mathrm{Cu}^{2+}$，自身被还原为 $\mathrm{NO_2}$。

## 十一、易错点
- 误把 Cu(I) 视为水溶液中最稳定价态，忽略其歧化倾向
- 误认为“金属与酸反应一定放氢”，忽略氧化性酸体系
- 只记深蓝色氨配合物，不理解颜色变化背后是配体置换与几何变化

## 十二、🎯 教学视角
- 第一轮讲授最适合抓三条线：Cu+/Cu2+ 稳定性、颜色体系、与酸和配体的反应性
- 可用“蓝矾-氢氧化铜-氨配铜”的连续实验现象串起课堂，学生感受最直观
- 若时间允许，可顺带引入 Jahn-Teller 效应解释六配位 Cu2+ 为何常表现出拉长八面体
