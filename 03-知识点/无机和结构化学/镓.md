# 镓

## 一、定义

**镓（Gallium, Ga）** 是第31号元素，位于元素周期表第4周期ⅢA族（硼族元素），属于p区金属。它是一种银白色软金属，是自然界中典型的**稀散元素**。

- [[基础要求-无机和结构化学]]：p区金属元素、稀散元素

## 三、核心原理

### 3.1 基本物理性质

| 性质 | 数据 |
|:---|:---|
| 原子序数 | 31 |
| 相对原子质量 | 69.72 |
| 价电子构型 | $4s^2 4p^1$ |
| 熔点 | 29.76°C |
| 沸点 | 2204°C |
| 密度 | $5.91\ \mathrm{g \cdot cm^{-3}}$（固态） |

**熔点异常低**：镓的熔点仅29.76°C，是常见金属中熔点最低的之一（低于人体体温），握在手中即可熔化。但沸点高达2204°C，是已知物质中**液态温度范围最大**的元素（约2174°C）。

### 3.2 存在与提炼

- **稀散性**：在自然界高度分散，几乎没有集中的矿石
- **主要来源**：铝矾土（含镓约 $3 \times 10^{-5}$），作为铝工业的副产品回收
- **其他来源**：燃煤烟囱的烟道灰中可富集

### 3.3 化学性质

#### 氧化态

镓常见氧化态为 **+1** 和 **+3**：

| 氧化物 | 性质 |
|:---|:---|
| $Ga_2O$（氧化亚镓） | 低价氧化物，黑色，不稳定 |
| $Ga_2O_3$（氧化镓） | 高价氧化物，白色，两性 |

**惰性电子对效应**：与铝不同，镓可以形成稳定的+1氧化态化合物，这是p区较重元素的一个特征。

#### 与酸碱反应

- 与酸反应：$2Ga + 6HCl \rightarrow 2GaCl_3 + 3H_2 \uparrow$
- 两性氢氧化物：$Ga(OH)_3$ 具有两性，既可溶于酸也可溶于强碱
- 与铝的差异：$Ga(OH)_3$ 的酸性比 $Al(OH)_3$ 更强

#### 特殊性质

- **浸润玻璃**：液态镓能浸润玻璃，这是它与汞的相似之处
- **过冷现象**：液态镓极易过冷，可冷却到0°C以下而不凝固

### 3.4 重要化合物与应用

#### 半导体材料

- **砷化镓（GaAs）**：重要的Ⅲ-Ⅴ族化合物半导体
  - 直接带隙半导体，光电转换效率高
  - 电子迁移率比硅高5-6倍
  - 用于高频器件、光电子器件、太阳能电池
- **氮化镓（GaN）**：宽禁带半导体，用于蓝光LED、激光二极管、高功率电子器件
- **磷化镓（GaP）**：用于红色和绿色LED

## 四、关键结论

1. **镓是最典型的稀散元素**：在自然界高度分散，无独立矿石，主要从铝工业副产物中提取。
2. **熔点极低、沸点极高**：液态温度范围最大，这一独特性质有重要应用价值。
3. **+1和+3两种氧化态**：+1氧化态的稳定性反映了p区重元素的惰性电子对效应。
4. **两性特征**：$Ga_2O_3$ 和 $Ga(OH)_3$ 具有两性，但酸性比铝的对应化合物更强。
5. **半导体应用核心元素**：GaAs、GaN 是现代光电子和微电子产业的关键材料。

## 六、适用条件与限制

1. **镓的保存**：因熔点低，常温下可能为液态，需密封保存；液态镓对玻璃有腐蚀性，不宜长期存放于玻璃瓶中。
2. **稀散性限制**：镓的产量受铝工业规模制约，价格相对较高。
3. **氧化态判断**：在化合物中镓通常为+3，但在某些特殊条件下可形成+1氧化态。

## 十、例题

### 例题1：镓的液态温度范围

**题目**：镓的熔点为29.76°C，沸点为2204°C。计算其液态温度范围，并说明这一特性有什么实际意义。

**解答**：
- 液态温度范围 = $2204 - 29.76 = 2174.24$°C
- 这是已知元素中最大的液态温度范围
- 实际意义：可用于高温温度计（代替汞，因无毒且温度范围更大），也适用于某些传热介质

### 例题2：GaAs的半导体类型

**题目**：GaAs属于哪种类型的半导体？与单晶硅相比有何优势？

**解答**：
- GaAs是**直接带隙**Ⅲ-Ⅴ族化合物半导体
- 优势：
  1. 电子迁移率高（约为Si的5-6倍），适合高频器件
  2. 直接带隙，光电转换效率高，适合光电子器件
  3. 带隙较宽（1.42 eV），耐高温性能优于Si

## 十一、易错点

1. **误认为镓只有+3氧化态**：实际上+1氧化态在某些化合物中也是稳定的。
2. **混淆镓与铝的氢氧化物酸性**：$Ga(OH)_3$ 的酸性比 $Al(OH)_3$ 更强，更易溶于碱。
3. **忽视稀散元素的分布特点**：镓在自然界高度分散，没有独立矿床。

- 混淆稀散元素与稀有元素：镓是稀散（分布分散），不是稀有（地壳总量并不少）

## 十三、竞赛拓展

1. **惰性电子对效应**：6s²电子的相对论性稳定效应，导致Tl⁺、Pb²⁺、Bi³⁺等低价态特别稳定。
2. **镓的液态合金**：Ga-In-Sn合金（熔点约-19°C）可作为低温液态金属导热材料。
3. **GaN基功率器件**：宽禁带特性使GaN在电动汽车、5G基站等高功率应用中逐渐取代硅。
4. **镓的配位化学**：$GaCl_3$ 作为温和的Lewis酸催化剂，在某些有机合成中有应用。
