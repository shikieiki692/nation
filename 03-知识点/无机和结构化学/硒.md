# 硒

## 一、定义

**硒（Selenium, Se）** 是第34号元素，位于元素周期表第4周期VIA族（氧族元素，也称硫族元素或硫属元素），属于典型的**稀散元素**和**半金属**。

## 三、核心原理

### 3.1 基本物理性质

| 性质 | 数据 |
|:---|:---|
| 原子序数 | 34 |
| 相对原子质量 | 78.96 |
| 价电子构型 | $4s^2 4p^4$ |
| 常见氧化态 | -2, +4, +6 |
| 电负性 | 2.55（Pauling） |

### 3.2 同素异形体

硒具有多种同素异形体，这是其重要特征：

#### 灰色硒（Gray Selenium，也称金属硒）

- **结构**：由螺旋状Se原子链构成的六方晶系
- **性质**：具有金属光泽、脆性、光电导性
- **稳定性**：最稳定的热力学形态
- **导电性**：不良导体，但光照下导电性显著增强（**光电导效应**）

#### 红色硒（Red Selenium）

- **结构**：$Se_8$ 环状分子（皇冠构型）
- **形态**：无定形或单斜晶体
- **性质**：溶于 $CS_2$，加热时软化

#### 黑色硒（Black Selenium）

- **结构**：无定形
- **性质**：脆性玻璃态物质

**同素异形体转化**：
- 红色硒 $\xrightarrow{\text{加热}}$ 灰色硒（热力学稳定态）
- 熔融硒快速冷却 $\rightarrow$ 黑色玻璃态硒

### 3.3 化学性质

#### 氧化态与化合物

| 氧化态 | 代表化合物 | 性质 |
|:---|:---|:---|
| -2 | $H_2Se$、$Na_2Se$ | 氢硒酸（酸性比 $H_2S$ 强） |
| +4 | $SeO_2$、$H_2SeO_3$（亚硒酸） | 两性偏酸，氧化剂 |
| +6 | $SeO_3$、$H_2SeO_4$（硒酸） | 强酸，强氧化剂 |

**重要反应**：
- 与卤素反应：$Se + 2F_2 \rightarrow SeF_6$（+6价）
- 被氧化：$Se + O_2 \xrightarrow{\Delta} SeO_2$（+4价）
- 与金属反应：$Fe + Se \xrightarrow{\Delta} FeSe$（-2价）

#### 氧化还原性

- 硒酸 $H_2SeO_4$ 是**强氧化剂**，可氧化 $Cl^-$ 为 $Cl_2$
- $SeO_2$ 在有机化学中可用作**选择性氧化剂**
- $Se^{2-}$ 是**强还原剂**，在空气中易被氧化

### 3.4 存在与提炼

- **稀散性**：在自然界高度分散，常与硫共生
- **主要来源**：铜冶炼的阳极泥（含硒约5-25%）
- **其他矿物**：硒铜矿 $Cu_2Se$、硒铅矿 $PbSe$

## 四、关键结论

1. **硒是稀散半金属**：地壳丰度低，高度分散，主要从铜冶炼副产物中回收。
2. **多种同素异形体**：灰色硒（螺旋链，光电导）、红色硒（$Se_8$ 环）、黑色硒（无定形）。
3. **氧化态为-2、+4、+6**：与硫相似，但高氧化态的氧化性更强。
4. **硒酸是强氧化剂**：$H_2SeO_4$ 的氧化性强于 $H_2SO_4$。
5. **生物必需微量元素**：参与谷胱甘肽过氧化物酶的活性中心，但过量有毒。

## 五、常见分类或情形

| 性质 | 氧(O) | 硫(S) | 硒(Se) | 碲(Te) |

| 常见氧化态 | -2 | -2, +4, +6 | -2, +4, +6 | -2, +4, +6 |

**递变趋势**：
- 从上到下，非金属性减弱，金属性增强

- 高氧化态含氧酸氧化性：$H_2SO_4 < H_2SeO_4 > H_6TeO_6$（Se异常）

## 六、适用条件与限制

1. **硒的毒性**：必需微量元素与有毒元素之间剂量范围很窄，过量摄入会导致硒中毒。
2. **光电导的条件**：灰色硒的光电导效应需要特定晶型，非晶态硒也有类似性质但机制不同。
3. **氧化态判断**：在含氧酸中，Se的最高氧化态为+6，但+4氧化态的化合物更常见且更稳定。

## 七、常见比较与易混点

### 硒 vs 硫

| 比较项 | 硫(S) | 硒(Se) |
|:---|:---|:---|
| 元素类型 | 非金属 | 半金属 |
| 同素异形体 | 斜方硫、单斜硫、弹性硫 | 灰色硒、红色硒、黑色硒 |
| 氢化物稳定性 | 较稳定 | 不稳定，易分解 |
| 氢化物酸性 | $H_2S$ 弱酸 | $H_2Se$ 较强酸 |
| 含氧酸氧化性 | $H_2SO_4$ 常温不氧化 $Cl^-$ | $H_2SeO_4$ 可氧化 $Cl^-$ |

### 氧族元素氢化物酸性递变

$$H_2O \ll H_2S < H_2Se < H_2Te$$

原因：键能 $H-O > H-S > H-Se > H-Te$，键越易断裂，酸性越强。

## 十、例题

### 例题1：硒的氧化态

**题目**：判断下列化合物中硒的氧化态：$H_2Se$、$SeO_2$、$H_2SeO_4$、$Na_2SeO_3$

**解答**：
- $H_2Se$：Se为 **-2**（H为+1）
- $SeO_2$：Se为 **+4**（O为-2）
- $H_2SeO_4$：Se为 **+6**（H为+1，O为-2）
- $Na_2SeO_3$：Se为 **+4**（Na为+1，O为-2）

### 例题2：氧族元素氢化物酸性比较

**题目**：解释为什么氢化物酸性顺序为 $H_2O \ll H_2S < H_2Se < H_2Te$。

**解答**：
- 氢化物酸性强弱取决于 $H-X$ 键的强度和极性
- 键能顺序：$H-O > H-S > H-Se > H-Te$
- 键长顺序：$H-O < H-S < H-Se < H-Te$
- 键长越长、键能越小，越易释放 $H^+$，酸性越强
- 因此酸性顺序为 $H_2O \ll H_2S < H_2Se < H_2Te$

## 十一、易错点

1. **误认为硒是非金属**：硒实际上是半金属（类金属），性质介于非金属和金属之间。
2. **混淆硒酸的氧化性**：$H_2SeO_4$ 的氧化性强于 $H_2SO_4$，可以氧化盐酸。
3. **忽视硒的稀散性**：硒没有独立矿床，主要从铜冶炼副产物中回收。
4. **氢化物稳定性判断错误**：$H_2Se$ 比 $H_2S$ 更不稳定，更易分解。
