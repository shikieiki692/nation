# λ 符号体系

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]

## 一、定义
**λ（lambda）符号体系**：IUPAC 推荐的高价化合物命名方法。在元素符号前加上 **λⁿ-**（其中 n 为该原子的**价**），用于明确高价或非常规价态的元素位置。

> 例：**λ⁵-磷烷** 表示价为 5 的磷化合物（如 $\ce{PF5}$、$\ce{PCl5}$）

## 三、核心原理

### 3.1 λⁿ 的含义
- λ 来自希腊字母（λ = "lambda"）
- 上标 n = 该原子的**价**（valence）= **键数**（不是氧化态、不是形式电荷）
- λⁿ 写在元素的**英文母词根**前

### 3.2 默认价（不需要写 λ）
| 元素 | 默认价 |
|---|---|
| H | 1 |
| C | 4 |
| N、P、As | 3 |
| O、S、Se | 2 |
| F、Cl、Br、I | 1 |

只有**与默认价不同**时才需要 λ⁻ 前缀。

### 3.3 常见 λ 标记
| 元素及前缀 | 意义 | 实例 |
|---|---|---|
| λ⁵-P | 价 5 的 P | $\ce{PF5}$、磷酸酯中央 P（在多重键式中） |
| λ⁴-S | 价 4 的 S | $\ce{SF4}$、亚砜 |
| λ⁶-S | 价 6 的 S | $\ce{SF6}$、砜 |
| λ²-N | 价 2 的 N | 氮宾（nitrene）、亚硝基 |
| λ⁵-Bi | 价 5 的 Bi | $\ce{BiPh5}$ |
| λ²-C | 价 2 的 C（卡宾） | $\ce{:CCl2}$ 等 |

### 3.4 命名规则（IUPAC 2013）
1. 对于含有非默认价原子的化合物，用 λⁿ 表明该原子的价
2. λⁿ 通常放在主链命名的对应位置
3. 无机化合物的 IUPAC 命名（红皮书）和有机化合物（蓝皮书）都接受 λ 标记

## 四、关键结论

### 4.1 与"价"的关系（牢牢绑定）
- λⁿ-X 中的 n **就是 X 的价**
- 不要与氧化态混淆——氧化态可能不同
- 例：λ⁵-P 在磷酸 $\ce{H3PO4}$ 中
  - P 的价 = 5（4 个 σ + 1 个 P=O 双键的额外贡献，按 IUPAC 计为 5）
  - P 的氧化态 = +5

### 4.2 命名实例
| 化合物 | λ 命名 | 解释 |
|---|---|---|
| $\ce{PF5}$ | λ⁵-磷烷-五氟化物 / 五氟化磷 | P 价 5 |
| $\ce{SF4}$ | λ⁴-硫烷-四氟化物 / 四氟化硫 | S 价 4 |
| $\ce{SF6}$ | λ⁶-硫烷-六氟化物 / 六氟化硫 | S 价 6 |
| $\ce{IF7}$ | λ⁷-碘烷-七氟化物 / 七氟化碘 | I 价 7 |
| $\ce{BiPh5}$ | 五苯基-λ⁵-铋烷 | Bi 价 5 |
| 氮宾 ($\ce{R-N:}$) | λ²-氮烷衍生物 | N 价 2 |

### 4.3 何时不需要 λ
- 默认价的化合物：$\ce{NH3}$（N 价 3，默认），$\ce{H2O}$（O 价 2，默认），$\ce{CH4}$（C 价 4，默认）
- 这些直接命名即可

## 五、常见分类或情形

### 5.1 高价磷化合物
- λ⁵-磷烷：$\ce{PF5}$、$\ce{PCl5}$、$\ce{P(OR)5}$
- 磷酸盐 $\ce{H3PO4}$：本身一般不用 λ，但其 IUPAC 名为"磷酸"或"四氢氧基-λ⁵-磷烷"

### 5.2 高价硫化合物
- λ⁴-硫烷：$\ce{SF4}$、亚砜 ($\ce{R2S=O}$)
- λ⁶-硫烷：$\ce{SF6}$、砜 ($\ce{R2SO2}$)

### 5.3 卡宾、氮宾、碳宾
- λ²-甲烷（卡宾）：$\ce{:CR2}$
- λ¹-氮烷（氮宾）：$\ce{R-N:}$
- λ⁰-（非常少见）

### 5.4 杂环命名
- 高价杂环：如 λ⁴-硫杂环（含 SF₂R₂ 单元）
- 用于精确描述杂原子的成键状态

## 六、适用条件与限制

### 适用范围
- ✓ 高价主族化合物
- ✓ 活性中间体（卡宾、氮宾）
- ✓ 杂环化合物（特殊价态杂原子）

### 限制
- 不适用于过渡金属配合物（用配位命名法）
- 不适用于离子化合物（主要是命名共价分子或基团）
- 中文命名中常用"X 价某烷"代替

## 七、常见比较与易混点

### 1. λ vs 氧化态符号
- $\ce{PF5}$ 中 P 是 **λ⁵**（价 5）= **氧化态 +5**——这两个数值在此恰巧相等
- 但 $\ce{NH4+}$ 中 N 是 **λ⁵** 但 **氧化态 −3**——值完全不同
- λ 始终表示**价（键数）**

### 2. λ vs 配位数
- 配位数：周围其他原子的数目（包括 σ 键和配位键的中心金属算法）
- λ：本原子参与的键数（更广义，包括多重键）

### 3. 中英文表达
| 英文 | 中文 |
|---|---|
| pentavalent phosphorus | 五价磷 / λ⁵-磷 |
| pentafluoro-λ⁵-phosphane | 五氟化-λ⁵-磷烷 / 五氟化磷 |
| octahedral SF₆ | 六氟化-λ⁶-硫烷 / 六氟化硫 |

## 十、例题

### 例题 1：$\ce{SF4}$ 的命名
**题目：** 给出 $\ce{SF4}$ 的 IUPAC 命名（含 λ）。
**分析：** S 通常价为 2，但在 $\ce{SF4}$ 中价为 4，需用 λ⁴。
**解答：** **四氟化-λ⁴-硫烷** 或常用 **四氟化硫**。

### 例题 2：磷酸的非常规命名
**题目：** $\ce{H3PO4}$ 的 IUPAC 全名（用 λ）。
**分析：** P 在磷酸中通常算价 5（含一个 P=O）。
**解答：** **三羟基(氧代)-λ⁵-磷烷** 或常用 **磷酸**。

## 十一、易错点
- 把 λⁿ 中的 n 写成氧化态——错（应为价）
- 在默认价的化合物上滥用 λ
- 忘记在中文命名中用 λ⁵-、λ⁴- 等
