# 稀有气体化合物

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]、[[15-晶体结构]]

## 一、定义
- 稀有气体化合物是指由稀有气体元素（主要是 Xe，少数 Kr、Rn）与其他元素形成的化合物
- 1962 年 N. Bartlett 首次合成 $\mathrm{XePtF_6}$，打破了"惰性气体不反应"的认知
- Xe 的化合物最丰富，因其电离能最低（12.13 eV）

## 二、考纲对应
- 对应考纲条目：[[13-元素化学]]、[[15-晶体结构]]、[[17-常见晶体结构]]
- 所属模块：[[基础要求-无机和结构化学]]
- 本知识点在考纲中的作用：掌握 Xe 的氟化物和氧化物的结构、制备与性质，是晶体结构分析的高频考点

## 三、核心原理

### 3.1 主要 Xe 化合物

| 化合物 | 氧化数 | 几何构型（VSEPR） | 特征 |
|:---|:---:|:---|:---|
| $\mathrm{XeF_2}$ | +2 | 直线形（sp³d，3 对孤对电子） | 无色晶体 |
| $\mathrm{XeF_4}$ | +4 | 平面正方形（sp³d²，2 对孤对） | 无色晶体 |
| $\mathrm{XeF_6}$ | +6 | 变形八面体（1 对孤对） | 最活泼氟化物 |
| $\mathrm{XeO_3}$ | +6 | 三角锥形（sp³，1 对孤对） | 无色晶体，爆炸性 |
| $\mathrm{XeO_4}$ | +8 | 正四面体 | 极不稳定，易爆炸 |
| $\mathrm{XeO_2}$ | +4 | — | 正交晶系 |

### 3.2 Bartlett 的发现（1962）

N. Bartlett 受到 $\mathrm{O_2}$ 能和 $\mathrm{PtF_6}$ 化合生成 $\mathrm{O_2^+[PtF_6]^-}$ 的启发，联想到 Xe 的第一电离能（$1170\ \mathrm{kJ\cdot mol^{-1}}$）比 $\mathrm{O_2}$ 的第一电离能（$1177\ \mathrm{kJ\cdot mol^{-1}}$）还要小，而理论计算 $\mathrm{Xe[PtF_6]}$ 的晶格能又比 $\mathrm{O_2[PtF_6]}$ 大 $42\ \mathrm{kJ\cdot mol^{-1}}$，因此 Xe 与 $\mathrm{PtF_6}$ 有可能反应。1962 年他成功制得稳定的橙黄色固体 $\mathrm{XePtF_6}$——人类获得的第一个稀有气体化合物。此后又合成了 $\mathrm{XeRuF_6}$ 和 $\mathrm{XeRhF_6}$，打破了"惰性气体不反应"的百年认知。

### 3.3 合成方法

$$\mathrm{Xe + F_2 \xrightarrow{h\nu\ or\ heat} XeF_2\ (过量\ Xe)}$$
$$\mathrm{Xe + 2F_2 \xrightarrow{>250^\circ C} XeF_4}$$
$$\mathrm{XeF_4 + 2H_2O \longrightarrow XeO_2 + 4HF\ (水解)}$$

### 3.4 已合成的化合物

- **Xe 的化合物**（最多）：$\mathrm{XeF_2}$、$\mathrm{XeF_4}$、$\mathrm{XeF_6}$、$\mathrm{XeCl_2}$、$\mathrm{XeCl_4}$、$\mathrm{XeBr_2}$、$\mathrm{XeO_3}$、$\mathrm{XeO_4}$、$\mathrm{XeOF_4}$、$\mathrm{Xe_2O_2F_2}$ 以及含氧酸盐（$\mathrm{HXeO_4^-}$、$\mathrm{XeO_6^{4-}}$、$\mathrm{XeO_3F^-}$ 等）
- **Kr 的化合物**：$\mathrm{KrF_2}$（唯一稳定的 Kr 化合物）
- **Rn 的化合物**：$\mathrm{RnF_2}$、$\mathrm{RnF_6}$
- **He、Ne、Ar**：化合物至今尚未制得

### 3.5 氧化性

$\mathrm{XeF_2}$、$\mathrm{XeF_4}$ 是强氧化剂和氟化剂：
$$\mathrm{XeF_2 + 2I^- \longrightarrow Xe + I_2 + 2F^-}$$

### 3.6 决赛级机理：水解、歧化与 APiIC 视角

> 来源：[[Arrow Pushing in Inorganic Chemistry-总索引]] §8.1–8.7（Ghosh & Berg, 2014）

#### 3.6.1 $\mathrm{XeF_2}$ 的碱性水解（含 HOF/HOOF 中间体）

$$
\ce{2XeF2 + 4OH^- -> 2Xe + O2 + 4F^- + 2H2O} \tag{8.17}
$$

机理（4 步）：
1. $\ce{OH^-}$ 亲核进攻 Xe-F → **HOF**（次氟酸）+ Xe + F⁻
2. $\ce{HOF + OH^-}$ → $\ce{FO^-}$ + H₂O（去质子化）
3. $\ce{FO^- + HOF}$ → **HOOF**（不稳定）+ F⁻
4. HOOF 经 $\ce{OH^-}$ E2 消除 → $\ce{O2}$ + F⁻ + H₂O

> 关键：Xe 的 +2 价被还原为 0 价的同时，O 由 −2 升至 0，是 **协同氧化还原**。

#### 3.6.2 $\mathrm{XeF_6}$ 与 $\mathrm{XeF_4}$ 的水解机理

**(a) $\mathrm{XeF_6}$**（极烈反应，分两步）：

$$
\ce{XeF6 + H2O -> XeOF4 + 2HF} \tag{8.21}
$$

$$
\ce{XeOF4 + 2H2O -> XeO3 + 4HF} \tag{8.22}
$$

机理：水/$\ce{OH^-}$ 进行 **SN2-Si 类亲核进攻 Xe**，置换 F 离去 → Xe-OH 中间体 → 去质子化得 Xe=O。

**(b) $\mathrm{XeF_4}$**（伴随歧化）：

$$
\ce{6XeF4 + 12H2O -> 2XeO3 + 4Xe + 3O2 + 24HF} \tag{8.38}
$$

机理本质：$\ce{XeF4}$ 上的孤对作为亲核体进攻另一个 $\ce{XeF4}$ 的 F → 同时得 **+6 价 Xe**（继续水解为 XeO₃）和 **+2 价 Xe**（继续按 8.17 路径还原至 0 价）。

#### 3.6.3 氙酸盐与高氙酸盐的歧化（决赛热点）

**XeO₃ 在碱性溶液中：**

$$
\ce{XeO3 + OH^- -> HXeO4^-} \tag{8.25, A 过程}
$$

> 简单的"亲核体–亲电体" association（A process）。

**氙酸盐自歧化为高氙酸盐：**

$$
\ce{2HXeO4^- + 2OH^- -> XeO6^{4-} + Xe + O2 + 2H2O} \tag{8.27}
$$

机理（关键步）：
1. 一个 $\ce{HXeO4^-}$ 上的 Xe 孤对作为亲核体进攻另一个 $\ce{HXeO4^-}$ 上的 O → 一步同时生成 **Xe(VIII)** 中间体 + **Xe(IV)** 中间体；
2. Xe(VIII) 加 OH⁻ + 去质子化 → 高氙酸根 $\ce{XeO6^{4-}}$；
3. Xe(IV) 中间体经 OH⁻ 进攻 → $\ce{Xe(OH)2}$ + $\ce{H2O2}$；
4. $\ce{H2O2}$ 进攻 $\ce{Xe(OH)2}$ → Xe + $\ce{H2O3}$（trioxidane，不稳定）→ $\ce{H2O3}$ 自分解为 H₂O + O₂。

> **关键洞察**：歧化的核心机理是 **"一步把 Xe(VI) 拆成 Xe(VIII) + Xe(IV)"**——利用一个 Xe 的孤对，进攻另一个 Xe 上的 O，同时实现氧化与还原。这是主族化学中最经典的"分子内电子对配对"思想。

#### 3.6.4 XeO₃ + O₃ 替代制备高氙酸盐

碱性条件下 XeO₃ 也可被臭氧氧化为高氙酸盐：

$$
\ce{XeO4^{2-} + O3 + 2OH^- -> XeO6^{4-} + O2 + H2O} \tag{8.28}
$$

机理：Xe 的孤对亲核进攻 O₃ 的 **中央 O**（带正电）→ Xe(VIII) 中间体 + O₂ → +OH⁻ → $\ce{XeO6^{4-}}$。

#### 3.6.5 高氙酸的脱水

$$
\ce{H4XeO6 -> XeO4 + 2H2O}
$$

机理：分子内两次脱水 + 形成两个 Xe=O 双键。$\ce{XeO4}$ 是 **正四面体**（Td 对称性，与 $\ce{ClO4^-}$、$\ce{SO4^{2-}}$ 同构型）。

> **关键提醒**：$\ce{XeO4}$ 是稀有气体里**唯一稳定的 +8 价**化合物，但 **极易爆炸**——存放需 −78 °C 干冰条件。

#### 3.6.6 KrF₂ 的特殊性

$$
\ce{KrF2 -> Kr + F2 \quad \Delta G^0 = -63 \text{ kJ/mol}} \tag{8.1}
$$
$$
\ce{XeF2 -> Xe + F2 \quad \Delta G^0 = +105 \text{ kJ/mol}} \tag{8.2}
$$

> $\ce{KrF2}$ 是 **唯一稳定的 Kr 化合物**，但热力学上自发分解（$\Delta G < 0$），仅能在低温（< −78 °C）下保存。这反映了 **Kr 比 Xe 更难氧化**——Kr 的电离能更高（13.99 vs Xe 的 12.13 eV）。


## 四、关键结论
- Xe 化合物的氧化数：+2（$\mathrm{XeF_2}$）→ +4（$\mathrm{XeF_4}$/$\mathrm{XeO_2}$）→ +6（$\mathrm{XeF_6}$/$\mathrm{XeO_3}$）→ +8（$\mathrm{XeO_4}$）
- $\mathrm{XeO_3}$ 是 $\mathrm{Xe(VI)}$ 最稳定的氧化物，但仍有爆炸性
- 所有 $\mathrm{XeO_x}$ 化合物均属**正交晶系**（$a \neq b \neq c$，$\alpha=\beta=\gamma=90^\circ$）
- $\mathrm{Xe_3O_2}$（$\mathrm{Xe_2^0Xe^{IV}O_2}$）中存在两种化学环境的 Xe

## 五、常见分类或情形
- **氟化物**：$\mathrm{XeF_2}$、$\mathrm{XeF_4}$、$\mathrm{XeF_6}$
- **氧化物**：$\mathrm{XeO_2}$、$\mathrm{XeO_3}$、$\mathrm{XeO_4}$
- **氟氧化物**：$\mathrm{XeOF_4}$、$\mathrm{XeO_2F_2}$、$\mathrm{XeO_3F_2}$
- **复合物**：$\mathrm{XePtF_6}$、$\mathrm{XeF_2\cdot XeF_4}$ 共晶

## 六、适用条件与限制
- $\mathrm{XeO_3}$ 和 $\mathrm{XeO_4}$ 极易爆炸，需小心处理
- $\mathrm{XeF_6}$ 与水反应剧烈，而 $\mathrm{XeF_2}$、$\mathrm{XeF_4}$ 水解较缓
- $\mathrm{XeF_x}$ 的 $x$ 值与温度、$\mathrm{F_2}$ 压力有关

## 七、常见比较与易混点
- **$\mathrm{XeF_2}$ vs $\mathrm{XeF_4}$ vs $\mathrm{XeF_6}$**：直线 → 平面正方形 → 变形八面体
- **$\mathrm{XeO_3}$ vs $\mathrm{XeO_4}$**：三角锥（+6价）vs 正四面体（+8价）
- **$\mathrm{Xe_3O_2}$**：不是简单氧化物，含 $\mathrm{Xe^0}$ 和 $\mathrm{Xe^{IV}}$

## 十、例题
### 例题 1（39 届初赛 5.1）
**题目：** 三种 $\mathrm{XeF_x}$ 晶体的晶胞结构如图所示，写出化学组成。
**分析：** 按原子位置计数（顶点 1/8、棱 1/4、面 1/2、内部 1）。
**解答：** A = $\mathrm{XeF_2}$，B = $\mathrm{XeF_4}$，C = $\mathrm{XeF_2 \cdot XeF_4}$（$\mathrm{XeF_3}$）。

### 例题 2（39 届初赛 5.2）
**题目：** 三者中 Xe 化合价最高的是哪个？晶系？
**解答：** $\mathrm{XeO_3}$（B），Xe 为 **+6** 价；**正交晶系**。

## 十一、易错点
- $\mathrm{XeF_2}$ 不是弯曲形（VSEPR：5 对电子，3 对孤对 → 直线形）
- 将 $\mathrm{XeO_3}$ 的氧化数写为 +3（应为 +6：3×(-2) = -6）
- 晶系判断混淆正交与四方

## 十二、竞赛拓展
- $\mathrm{Xe}$ 在地幔 $\mathrm{FeO_2}$ 矿物中的吸附（"失踪的氙"假说）
- $\mathrm{XeF_2}$ 作为温和氟化剂在有机合成中的应用
- $\mathrm{KrF_2}$ 的合成与性质（Kr 的唯一稳定化合物）
- 与 [[高价碘试剂]] 的对比：Xe(VIII) 与 I(VII) 都是"极高价主族元素"，但 Xe(VIII) 是孤立分子（XeO₄）、I(VII) 多以含氧酸（HIO₄）存在
- **Xe-N 键化合物**：$\ce{F-Xe-N(SO2F)2}$、$\ce{[Xe-N(CH3CN)]^+}$ 等含 Xe-N 键化合物的合成与稳定性
- **Xe-C 键化合物**：$\ce{[(C6F5)Xe]^+ [B(C6F5)4]^-}$ 是首个含 Xe-C 键的稳定化合物（1989 年）
