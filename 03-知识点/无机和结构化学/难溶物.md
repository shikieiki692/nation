# 难溶物

## 一、溶解性规律（竞赛必记）

### 硝酸盐
**全部可溶**

### 硫酸盐
CaSO₄(微溶)、SrSO₄、BaSO₄、PbSO₄ → 难溶

### 氯化物
AgCl、Hg₂Cl₂、PbCl₂ → 难溶（PbCl₂ 溶于热水）

### 硫化物
| 类型 | 实例 | Ksp 范围 |
|------|------|:---:|
| 溶于稀酸 | MnS, FeS, ZnS | 较大（~10⁻¹⁰–10⁻²⁰） |
| 不溶于稀酸 | CdS, PbS, CuS | 很小（~10⁻²⁸–10⁻³⁶） |
| 溶于 Na₂S/(NH₄)₂S | As₂S₃, Sb₂S₃, SnS₂, HgS | 生成硫代酸盐 |

### 碳酸盐
除碱金属和 NH₄⁺ 外均难溶（CaCO₃, BaCO₃, PbCO₃...）

### 铬酸盐
BaCrO₄(黄)、PbCrO₄(黄)、Ag₂CrO₄(砖红) 难溶

### 氢氧化物
除 NaOH/KOH/Ba(OH)₂ 外多数难溶

## 三、核心原理

### 3.1 溶解性规律总结
从离子化合物的溶解性数据中总结出以下经验规律：
- **硝酸盐**：全部可溶
- **碱金属盐**：除少数外几乎全部可溶（$\mathrm{Na^+}$、$\mathrm{K^+}$ 盐绝大多数易溶）
- **硫酸盐**：$\mathrm{CaSO_4}$（微溶）、$\mathrm{SrSO_4}$、$\mathrm{BaSO_4}$、$\mathrm{PbSO_4}$ 难溶；其余多数可溶
- **氯化物**：AgCl、$\mathrm{Hg_2Cl_2}$（甘汞）、$\mathrm{PbCl_2}$ 难溶（$\mathrm{PbCl_2}$ 溶于热水）；其余多数可溶
- **碳酸盐**：除碱金属和 $\mathrm{NH_4^+}$ 外均难溶（$\mathrm{CaCO_3}$、$\mathrm{BaCO_3}$、$\mathrm{PbCO_3}$ 等）
- **铬酸盐**：$\mathrm{BaCrO_4}$（黄）、$\mathrm{PbCrO_4}$（黄）、$\mathrm{Ag_2CrO_4}$（砖红）难溶
- **氢氧化物**：除 NaOH、KOH、$\mathrm{Ba(OH)_2}$ 外多数难溶

### 3.2 硫化物的分级溶解
硫化物按 $K_{sp}$ 大小可分为三类：
- **溶于稀酸**：MnS、FeS、ZnS（$K_{sp}$ 较大，约 $10^{-10}\sim 10^{-20}$）
- **不溶于稀酸**：CdS、PbS、CuS（$K_{sp}$ 很小，约 $10^{-28}\sim 10^{-36}$）。其中 CuS 极难溶（$K_{sp} \sim 10^{-36}$）
- **溶于 $\mathrm{Na_2S}$ 或 $(\mathrm{NH_4})_2\mathrm{S}$**：$\mathrm{As_2S_3}$、$\mathrm{Sb_2S_3}$、$\mathrm{SnS_2}$、HgS（生成硫代酸盐：$\mathrm{As_2S_3} + 3\mathrm{S}^{2-} \to 2\mathrm{AsS_3^{3-}}$；$\mathrm{HgS} + \mathrm{S}^{2-} \to \mathrm{HgS_2^{2-}}$）

### 3.3 IA/IIA 族盐类溶解性
IA 族（碱金属）盐类绝大多数易溶。IIA 族（碱土金属）的溶解性规律：
- 氯化物、溴化物、硝酸盐易溶于水
- 碳酸盐、草酸盐难溶
- $\mathrm{MgSO_4}$ 可溶，$\mathrm{CaSO_4}$ 微溶，$\mathrm{SrSO_4}$、$\mathrm{BaSO_4}$ 难溶
- $\mathrm{Mg(OH)_2}$ 不溶，$\mathrm{Ca(OH)_2}$、$\mathrm{Sr(OH)_2}$ 微溶，$\mathrm{Ba(OH)_2}$ 可溶
- IIA 族氢氧化物和氟化物溶解度随阳离子半径的增大而增大

### 3.4 卤化银的溶解性和配位性
AgF 可溶（因 $\mathrm{F}^-$ 半径小，晶格能适中同时水合能大），AgCl（白）、AgBr（浅黄）、AgI（黄）为难溶沉淀，$K_{sp}$ 依次递减。利用配位反应可溶解卤化银：
- $\mathrm{AgCl}$ 溶于 $\mathrm{NH_3}$：$\mathrm{AgCl} + 2\mathrm{NH_3} \to [\mathrm{Ag(NH_3)_2}]^{+} + \mathrm{Cl}^{-}$
- $\mathrm{AgBr}$ 微溶于 $\mathrm{NH_3}$，溶于 $\mathrm{Na_2S_2O_3}$（定影原理）
- $\mathrm{AgI}$ 不溶于 $\mathrm{NH_3}$，溶于 $\mathrm{Na_2S_2O_3}$ 或 $\mathrm{CN}^-$

### 3.5 沉淀的转化
溶解度大的沉淀可以转化为溶解度更小的沉淀：如 $\mathrm{CaCO_3}$ 的 $K_{sp}$ 虽小，但 $\mathrm{CaSO_4}$ 的 $K_{sp}$ 更小，不能用 $\mathrm{CaCO_3}$ 沉淀 $\mathrm{Ba^{2+}}$。可以通过加入沉淀剂或络合剂拉动平衡（如 $\mathrm{Hg_2^{2+}}$ 的歧化）。

## 四、关键结论
- 硝酸盐全部可溶，碱金属盐几乎全部可溶
- $\mathrm{BaSO_4}$、$\mathrm{PbSO_4}$、AgCl、$\mathrm{Hg_2Cl_2}$ 为竞赛必记难溶盐
- 硫化物溶解度差异大：MnS 溶于稀 HCl，CuS 不溶于稀 HCl，$\mathrm{As_2S_3}$ 溶于 $\mathrm{Na_2S}$
- AgF 可溶，AgCl、AgBr、AgI 难溶且依次更难溶
- 利用配位反应可使难溶盐溶解（如 $\mathrm{AgCl} + \mathrm{NH_3}$、$\mathrm{AgBr} + \mathrm{S_2O_3^{2-}}$）
- IIA 族碳酸盐、草酸盐难溶，硫酸盐溶解性随阳离子半径增大而减小

## 五、常见分类或情形
### 硫酸盐
$\mathrm{CaSO_4}$（微溶）、$\mathrm{SrSO_4}$（难溶）、$\mathrm{BaSO_4}$（难溶）、$\mathrm{PbSO_4}$（难溶）

### 氯化物
AgCl（白）、$\mathrm{Hg_2Cl_2}$（甘汞，白）、$\mathrm{PbCl_2}$（溶于热水）

### 硫化物
| 类型 | 实例 |
|------|------|
| 溶于稀酸 | MnS（肉色）、FeS（黑）、ZnS（白） |
| 不溶于稀酸 | CdS（黄）、PbS（黑）、CuS（黑）、$\mathrm{Ag_2S}$（黑） |
| 溶于 $\mathrm{Na_2S}$ | $\mathrm{As_2S_3}$（黄）、$\mathrm{Sb_2S_3}$（橙）、$\mathrm{SnS_2}$（黄）、HgS（黑/红） |

### 铬酸盐
$\mathrm{BaCrO_4}$（黄）、$\mathrm{PbCrO_4}$（黄）、$\mathrm{Ag_2CrO_4}$（砖红）

### 氢氧化物
除 NaOH、KOH、$\mathrm{Ba(OH)_2}$ 外多数难溶；$\mathrm{Ca(OH)_2}$、$\mathrm{Sr(OH)_2}$ 微溶

## 六、适用条件与限制
- 硫化物的分级溶解在定性分析中用于分组分离金属离子（$\mathrm{H_2S}$ 系统分析法）
- $\mathrm{PbCl_2}$ 溶于热水而 AgCl 不溶，可用于分离
- $\mathrm{HgS}$ 是所有金属硫化物中溶解度最小的（$K_{sp}$ 极小），需用王水溶解
- 硫代酸盐只在碱性溶液中稳定，酸化后重新生成硫化物沉淀

## 七、常见比较与易混点
| 对比项 | $\mathrm{BaSO_4}$ | $\mathrm{BaCO_3}$ | $\mathrm{BaCrO_4}$ |
|------|:---:|:---:|:---:|
| 颜色 | 白 | 白 | 黄 |
| 溶于 HCl | 不溶 | 溶（放出 $\mathrm{CO_2}$） | 溶 |
| 溶解性 | 极难溶 | 难溶 | 难溶 |

| 对比项 | AgCl | $\mathrm{Hg_2Cl_2}$ | $\mathrm{PbCl_2}$ |
|------|------|------|------|
| 颜色 | 白 | 白 | 白 |
| 与 $\mathrm{NH_3}$ | 溶（生成配合物） | 生成黑色 $\mathrm{Hg} + \mathrm{Hg(NH_2)Cl}$ | 生成白色 $\mathrm{Pb(OH)Cl}$ |
| 溶于热水 | 不溶 | 不溶 | 溶 |

- CuS（黑）vs ZnS（白）：鲜明对比，可用于鉴定
- $\mathrm{AgCl}$ 溶于 $\mathrm{NH_3}$ vs $\mathrm{AgBr}$ 微溶 vs $\mathrm{AgI}$ 不溶：$K_{sp}$ 递减
