## 一、定义
- **晶胞投影图**：将三维晶胞结构沿某一晶轴方向投影到二维平面上，类似于工程制图中的"三视图"
- 是近年化竞初赛/决赛的**固定题型**，考查空间想象力和晶体学基本功
- 出题方式：给出沿 $a$、$b$、$c$ 方向的一个或多个投影图 → 推导化学组成、晶系、原子坐标

## 二、考纲对应

- 本知识点在考纲中的作用：晶胞投影图是连接二维信息与三维结构的桥梁，是晶体学核心能力

## 三、核心原理

### 3.1 原子位置计数规则

| 位置 | 占有率 | 说明 |
|:---|:---:|:---|
| **顶点** | **1/8** | 被 8 个晶胞共享 |
| **棱上** | **1/4** | 被 4 个晶胞共享 |
| **面上** | **1/2** | 被 2 个晶胞共享 |
| **内部（体心等）** | **1** | 独属于该晶胞 |

### 3.2 投影图的"遮挡关系"判定

判断原子到底在什么位置的关键：**原子与边框线的遮挡关系**

| 投影图中位置 | 实际位置 |
|:---|:---|
| 顶点处、边框线**前方** | 晶胞顶点 |
| 顶点处、边框线**后方** | 棱上（被边框线挡住的顶点） |
| 边框线处、边框线**前方** | 该棱上 |
| 边框线处、边框线**后方** | 该面上（$a=0$ 或 $a=1$ 等） |
| 投影图内部 | 晶胞内部或面上 |

### 3.3 投影重叠的判断

- 两个原子投影重合：原子连线平行于投影方向
- 需结合**多个方向**的投影图才能唯一确定原子位置
- 利用化学合理性排除过近距离的重叠（如原子半径）

### 3.4 晶系判断依据

| 晶系 | 晶胞参数特征 | 特征对称元素 |
|:---|:---|:---|
| 立方 | $a=b=c$，$\alpha=\beta=\gamma=90^\circ$ | 4 条三重轴 |
| 四方 | $a=b\neq c$，$\alpha=\beta=\gamma=90^\circ$ | 1 条四重轴 |
| **正交** | $a\neq b\neq c$，$\alpha=\beta=\gamma=90^\circ$ | 3 条互相垂直的二重轴或镜面 |
| 三方 ($hR$) | $a=b=c$，$\alpha=\beta=\gamma\neq 90^\circ$ | 1 条三重轴 |
| 六方 | $a=b\neq c$，$\alpha=\beta=90^\circ,\gamma=120^\circ$ | 1 条六重轴 |

## 四、关键结论

### 4.1 分析步骤（"四步法"）

1. **找对称性** → 初步判断晶系（看是否有直角、等长）
2. **定位原子** → 利用遮挡关系和多个投影图确定每个原子的位置
3. **计数原子** → 按位置类型乘以占有率，求和
4. **验证合理性** → 检查原子间距、化合价、配位数

### 4.2 常见陷阱

- 投影图中"重合"的原子容易漏数 → 用多个方向交叉验证
- 分子晶体 vs 原子晶体计数不同（分子晶体中每个分子独立）
- $\mathrm{FeO_2}$ 型结构：O 可能以 $\mathrm{O_2^{2-}}$ 形式存在

## 五、常见分类或情形
- **分子晶体投影**：$\mathrm{XeF_2/XeF_4}$ 晶胞（原子不跨晶胞）
- **共顶点多面体投影**：$\mathrm{FeO_6}$ 八面体 → $\mathrm{FeO_2}$ 型
- **密堆积+填隙投影**：$hR$ 点阵中 $\mathrm{Xe}$ 吸附于 $\mathrm{FeO_2}$ 骨架
- **Pauling 配位多面体规则**：$\dfrac{n_+}{n_-} = \dfrac{\text{阴离子配位数}}{\text{阳离子配位数}}$

## 六、适用条件与限制
- 需要空间想象力，建议平时用软件（Diamond/VESTA）辅助练习
- 投影图通常只给晶胞内的信息，晶胞外原子不画出
- 必须结合化学常识排除不合理结构

## 七、常见比较与易混点
- **顶点 vs 棱心**：都在投影图顶点处，区分靠遮挡关系（前方 = 顶点，后方 = 棱）
- **面上 vs 内部**：都在投影图内部，区分靠另一方向的投影
- **正交 vs 四方**：都需要 $\alpha=\beta=\gamma=90^\circ$，四方有 $a=b$ 和四重轴

## 十、例题
### 例题 1（39 届初赛 5.1）
**题目：** 三种 $\mathrm{XeF_x}$ 晶胞投影图 → 化学组成。
**分析：** 分子晶体，原子不跨晶胞，计数即可。
**解答：** A = $\mathrm{XeF_2}$（2个/晶胞），B = $\mathrm{XeF_4}$（2个/晶胞），C = $\mathrm{XeF_2\cdot XeF_4}$。

### 例题 2（39 届初赛 5.3.1）
**题目：** $\mathrm{Fe_xO_y}$ 沿 (001) 投影图 → 化学式。
**分析：** 共顶点 $\mathrm{FeO_6}$ 八面体，每个 O 被 3 个八面体共享 → Fe:O = 3:6 = 1:2。
**解答：** $\mathrm{FeO_2}$。

## 十一、易错点
- 混淆顶点和棱心（遮挡关系判断错误）
- 投影重叠导致漏数或多数原子
- 晶系判断未使用特征对称元素而仅凭晶胞参数猜测
- 对 $hR$（菱心六方）点阵的计数规则不熟悉
