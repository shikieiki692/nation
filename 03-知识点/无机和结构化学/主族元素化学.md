# 主族元素化学

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 关联笔记：[[元素周期律]], [[元素推断]], [[过渡金属通性]], [[酸碱反应]]

## 一、定义

主族元素是指元素周期表中 s 区（第1~2族）和 p 区（第13~18族）的元素，其价电子仅填充在 s 和 p 轨道中。主族元素化学是竞赛元素化学的重要组成部分，涵盖碱金属、碱土金属、硼族、碳族、氮族、氧族、卤素和稀有气体的性质、反应和重要化合物。

## 二、考纲对应
- 对应考纲条目：[[13-元素化学]]、[[14-元素结构与性质]]

## 三、核心原理

### 3.1 各族通性概述

| 族 | 元素 | 价电子构型 | 主要氧化态 | 竞赛重点 |
|----|------|-----------|-----------|---------|
| **IA（碱金属）** | Li~Fr | $ns^1$ | +1 | 锂的特殊性（与镁对角线相似）；钠钾的焰色反应 |
| **IIA（碱土金属）** | Be~Ra | $ns^2$ | +2 | 铍的两性；镁的还原性；钙锶钡的焰色反应 |
| **IIIA（硼族）** | B~Tl | $ns^2np^1$ | +3（Tl 有+1） | 硼的缺电子性；铝的两性；镓铟的低温熔点 |
| **IVA（碳族）** | C~Pb | $ns^2np^2$ | +4, +2（Pb²⁺ 更稳定） | 碳的多样性（金刚石/石墨/C₆₀）；硅的半导体性质；Sn/Pb 的氧化态稳定性 |
| **VA（氮族）** | N~Bi | $ns^2np^3$ | -3~+5 | 氮的成键多样性；磷的同素异形体；砷锑铋的金属性递增 |
| **VIA（氧族）** | O~Po | $ns^2np^4$ | -2~+6 | 氧的特殊性；硫的同素异形体；过氧化氢的性质 |
| **VIIA（卤素）** | F~At | $ns^2np^5$ | -1~+7 | 氟的强氧化性；氯的歧化；卤素互化物；拟卤素 |
| **VIIIA（稀有气体）** | He~Rn | $ns^2np^6$ | 0（Xe/Kr 有化合物） | Xe 的化合物（XeF₂、XeF₄、XeF₆、XeO₃） |

### 3.2 重要规律

- **惰性电子对效应**：Tl⁺、Pb²⁺、Bi³⁺ 比高氧化态更稳定
- **对角线规则**：Li~Mg、Be~Al、B~Si 性质相似
- **第二周期反常**：N、O、F 的键能异常（由于原子半径小、孤对电子排斥）

### 3.3 重要化合物与反应

- **过氧化物/超氧化物**：Na₂O₂、KO₂ 的生成和反应
- **氢化物**：NH₃、PH₃、H₂O、H₂S 的酸碱性和还原性
- **含氧酸**：HNO₃、H₃PO₄、H₂SO₄、HClO₄ 的酸性变化规律
- **卤素含氧酸**：HClO~HClO₄ 酸性递增、氧化性递减

## 四、关键结论

1. **族内递变主线**：自上而下——金属性↑、共价性↓、低价氧化态稳定性↑（惰性电子对效应）、氢化物稳定性↓、含氧酸酸性↓、含氧酸盐热稳定性↑
2. **周期横向主线**：同周期自左至右——电负性↑、最高氧化态含氧酸酸性↑（非羟基氧增多）、氢氧化物碱性→两性→酸性
3. **三个跨族效应**：
   - **惰性电子对效应**（Tl⁺/Pb²⁺/Bi³⁺ 稳定）→ 影响 IIIA/IVA/VA 重元素
   - **对角线规则**（Li~Mg、Be~Al、B~Si）→ 连接 IA/IIA、IIA/IIIA、IIIA/IVA
   - **第二周期反常**（N/O/F 键能异常、无 d 轨道）→ 区分第二周期与第三周期及以下元素
4. **竞赛主族元素的主轴**：卤素（Cl 为中心）→ 氧族（S 为中心）→ 氮族（N 为中心）——这是出题频率最高的三个主族，应优先掌握

## 五、常见分类或情形

### 按元素类型分
- **s 区金属**（IA/IIA）：强还原剂，盐型氢化物（H⁻），焰色反应鉴定
- **p 区非金属**（右上角）：共价化合物为主，含氧酸体系丰富
- **p 区金属/半金属**（左下角）：两性、惰性电子对效应、低熔点（Ga 29.8°C）

### 按化合物类型分
- **氢化物**：分子型（p 区共价）→ 盐型（IA/IIA，H⁻）→ 金属型（d/f 区间隙）
- **氧化物**：酸性（非金属）→ 两性（半金属/轻金属）→ 碱性（重金属）
- **含氧酸**：氧化性 vs 酸性不平行（如 HClO₄ 酸性最强但氧化性最弱）
- **卤化物**：水解行为取决于中心原子有无空 d 轨道（CCl₄ 不水解 vs SiCl₄ 易水解）

## 六、适用条件与限制

- 主族元素性质的递变规律是**总体趋势**，存在诸多例外（如 N₂ 的惰性、HF 的弱酸性、H₃BO₃ 的一元弱酸性）
- 惰性电子对效应主要影响**第六周期** p 区元素（Tl, Pb, Bi），对第四、五周期影响较小
- 对角线规则适用于**第二周期第一个元素**与**第三周期第二个元素**之间的相似性，不能无限外推
- 含氧酸氧化性与酸性的分离——**非羟基氧越多酸性越强、中心原子氧化态越高氧化性不一定越强**（如 HClO₄ 氧化性 < HClO）
- 卤素在水/碱中的歧化方向取决于**介质的酸碱性**——同一反应在不同 pH 下方向可能相反

## 七、常见比较与易混点

| 对比维度 | 规律 | 典型反例 |
|:---|:---|:---|
| 氢化物酸性 | 同族 ↓ 酸性 ↑（HF < HCl < HBr < HI） | HF 是弱酸（pKₐ=3.18），其余是强酸 |
| 含氧酸酸性 | 同周期 → 酸性 ↑（非羟基氧增多） | H₃BO₃ 形式上三元实际一元弱酸 |
| 氢氧化物酸碱性 | 同族 ↓ 碱性 ↑（Be(OH)₂ 两性 → Ba(OH)₂ 强碱） | Al(OH)₃ 两性，TlOH 强碱 |
| 含氧酸盐热稳定性 | 同族 ↓ 稳定性 ↑（MgCO₃ < CaCO₃ < SrCO₃ < BaCO₃） | Li₂CO₃ 易分解（对角线规则 Li~Mg） |
| 卤素含氧酸氧化性 | HClO > HClO₂ > HClO₃ > HClO₄（氧化态越高氧化性越弱） | 与酸性顺序恰好相反 |
| 硫酸盐溶解度 | IIA ↓（BeSO₄ 易溶 → BaSO₄ 难溶） | M(OH)₂ 溶解度方向相反（↑） |
| 氯化物水解 | 有空 d 轨道者易水解（SiCl₄, PCl₅, SCl₄） | CCl₄、NF₃ 不水解（无 d 轨道） |

## 九、典型题型

- **题型-元素推断（主族综合）**：从颜色/状态/反应现象/定量数据推断未知主族元素的单质和化合物——这是化竞初赛最核心的题型之一
- **题型-跨族性质比较**：比较不同主族氢化物的酸性/还原性/热稳定性，或含氧酸的酸性/氧化性
- **题型-惰性电子对效应推断**：给定未知高价氧化物，从"它在周期表中的位置"判断其氧化还原行为
- **题型-对角线规则应用**：利用 Li~Mg、Be~Al、B~Si 的对角相似性预测化合物性质和反应
- **题型-主族元素反应方程式书写**：卤素歧化/归中、过氧化物/超氧化物反应、硝酸与金属反应等

## 十、例题

### 例题 1（基础 · 跨族氢化物酸性比较）

**题目**：比较下列氢化物的酸性强弱，并解释原因：HF、HCl、H₂O、NH₃、CH₄。

**分析**：氢化物酸性取决于中心原子对 H⁺ 释放的"支持程度"——电负性越大、半径越大（键越弱），酸性越强。

**解答**：
酸性排序：**HCl > HF > H₂O > NH₃ > CH₄**

- HCl（强酸）：Cl 大半径（99 pm）→ H—Cl 键弱（431 kJ/mol），H⁺ 极易释放
- HF（弱酸，pKₐ=3.18）：虽然 F 电负性最大，但 H—F 键太强（565 kJ/mol）+ F⁻ 水合能大 + HF 分子间氢键 → 综合效应使 HF 的 ΔG 解离为正值
- H₂O（极弱酸，pKₐ≈15.7）：O 电负性大但 H—O 键较强
- NH₃（几乎不电离 H⁺，反而是碱！）：N 电负性较小（3.0），更倾向于接受 H⁺ 而非释放
- CH₄（完全不电离）：C 电负性 2.5，C—H 键是非极性共价键

**反思**：氢化物酸性是周期律的绝佳演示——同周期左→右酸性↑（CH₄→NH₃→H₂O→HF），同族上→下酸性↑（HF→HCl→HBr→HI）。HF 的"反常弱酸性"是竞赛高频考点。

### 例题 2（提高 · 主族元素推断）

**题目**：某元素的单质 A 为银白色金属，在空气中燃烧得到白色粉末 B。B 与水剧烈反应放出大量热并生成气体 C（有刺激性气味）。C 的水溶液呈强碱性。A 与稀盐酸反应生成 D 和气体 E，D 焰色反应为砖红色。E 在空气中燃烧产生淡蓝色火焰。

（1）写出 A~E 的化学式。（2）B 与水反应的方程式。（3）A 在周期表中的位置和族名。

**分析**："砖红色焰色 + 银白色金属 + 与水反应放出有刺激气味气体"→ 钙（Ca）。CaO（B）+ H₂O → Ca(OH)₂——但这里"气体 C 有刺激性气味且水溶液呈强碱性"→ NH₃。所以 B 不是 CaO！重新考虑——Ca₃N₂ + H₂O → Ca(OH)₂ + NH₃。

但题中说"在空气中燃烧得到 B"→ 钙在空气中燃烧主要得 CaO 和少量 Ca₃N₂。结合"B 与水反应产生 NH₃"→ B = Ca₃N₂（或 CaO/Ca₃N₂ 混合物）。不过更简洁的判断——银白色金属+砖红焰色→Ca；B 与水产生有刺激性气味气体→Ca₃N₂（C=NH₃）；A+HCl→CaCl₂+H₂。

**解答**：
（1）A = **Ca**，B = **Ca₃N₂**（钙在空气中燃烧会与 N₂ 反应生成氮化钙），C = **NH₃**，D = **CaCl₂**，E = **H₂**
（2）$\mathrm{Ca_3N_2 + 6H_2O \longrightarrow 3Ca(OH)_2 + 2NH_3\uparrow}$
（3）第四周期、IIA 族，碱土金属

**反思**：元素推断题的多重验证——焰色反应（砖红=Ca/Sr 之一）、与水反应产生 NH₃（含 N 的化合物→可能是氮化物）、气体可燃（H₂ 或 NH₃ 都符合）。用多重限制条件收敛到唯一答案。

### 例题 3（竞赛 · 跨族惰性电子对效应推断）

**题目**：X 是第六周期 p 区元素，其稳定氧化态为 +1（而非按族预期的最高氧化态）。X 的 +1 价化合物中，卤化物与 AgX 性质相似（难溶、光解），硫酸盐易溶于水且易成矾。X 的 +3 价化合物是强氧化剂——能将 SO₂ 氧化为 H₂SO₄，将 S²⁻ 氧化为 S。

（1）推断 X 是什么元素，解释其稳定氧化态异常的原因。
（2）写出 X 与稀 H₂SO₄、浓 HNO₃ 反应的方程式。
（3）写出 X³⁺ 被 SO₂ 还原的离子方程式。
（4）同周期的 IVA 和 VA 族元素也有类似现象，分别写出它们"意外稳定"的低价氧化态。

**分析**：第六周期 p 区元素 + 稳定 +1 氧化态 + 卤化物与 AgX 相似 → **Tl**（铊）。惰性电子对效应（6s² 不参与成键）。Tl(I) 与 Ag(I)、K(I) 的相似性是经典考点。

**解答**：
（1）X = **Tl**（铊，IIIA 族）。Tl 的价电子构型为 6s²6p¹，因 **6s² 惰性电子对效应**（相对论效应导致 6s 轨道收缩 → 能量降低 → 不易参与成键），Tl 的稳定氧化态是 +1（6p¹ 失去），而非 +3。Tl(III) 是强氧化剂，容易被还原为 Tl(I)。

（2）与稀 H₂SO₄：$\mathrm{2Tl + H_2SO_4 \longrightarrow Tl_2SO_4 + H_2\uparrow}$（Tl 只被氧化到 +1！）
与浓 HNO₃：$\mathrm{Tl + 2HNO_3 \longrightarrow TlNO_3 + NO_2\uparrow + H_2O}$（即使浓 HNO₃ 也不能将 Tl 氧化到 +3）

（3）$\mathrm{Tl^{3+} + SO_2 + 2H_2O \longrightarrow Tl^+ + H_2SO_4 + 2H^+}$

（4）IVA 族：**Pb**——+2（Pb²⁺）稳定，+4（PbO₂）强氧化性。VA 族：**Bi**——+3（Bi³⁺）稳定，+5（NaBiO₃）强氧化性

**反思**：惰性电子对效应不是"一个族的特例"，而是第六周期 IIIA→IVA→VA→VIA（Tl→Pb→Bi→Po）的**系统性趋势**。竞赛常要求跨族串联解释——"为什么 Tl(I)、Pb(II)、Bi(III) 都反常稳定？"

### 例题 4（竞赛 · 对角线规则综合）

**题目**：Li 的许多性质与 Mg 更相似（而非同族的 Na），Be 与 Al 更相似，B 与 Si 更相似。这种"对角线相似"被称为对角线规则。

（1）举出 Li 与 Mg 性质相似的 3 个实例。
（2）举出 Be 与 Al 性质相似的 3 个实例。
（3）从离子势 φ = z/r 的角度解释 Li⁺~Mg²⁺、Be²⁺~Al³⁺、B³⁺~Si⁴⁺ 为何相似。
（4）为什么第三周期之后的对角线规则不显著？

**分析**：对角线规则的物理本质——对角线上两个离子的离子势 φ = z/r 相近，导致它们对外层电子的约束力相近 → 化学性质相似。

**解答**：
（1）Li~Mg 对角线：
- Li₂CO₃ 和 MgCO₃ 均易分解（Na₂CO₃ 高温才分解）
- Li 和 Mg 均能与 N₂ 直接反应生成氮化物（Li₃N, Mg₃N₂）
- LiOH 和 Mg(OH)₂ 碱性较弱（NaOH、KOH 为强碱）

（2）Be~Al 对角线：
- Be(OH)₂ 和 Al(OH)₃ 均为两性氢氧化物
- Be 和 Al 均能被浓 HNO₃ 钝化
- BeCl₂ 和 AlCl₃ 均为共价氯化物（Lewis 酸催化剂）

（3）离子势比较：
Li⁺：z=1, r=60 pm → φ=0.0167 → √φ≈0.129
Mg²⁺：z=2, r=65 pm → φ=0.0308 → √φ≈0.175
——虽然不等，但比 Li⁺~Na⁺（r 差更大）更接近。本质上是对角线上相邻周期的两个离子，其 **z/r 比的变化被 r 的增大所部分抵消**。

（4）第三周期之后，原子轨道（d/f 轨道）的填充使半径变化更加复杂，离子势的简单对应关系被打破。另外，第二周期元素的特殊性（无 d 轨道）是其与第三周期元素性质差异巨大的根源，对角线规则恰好"跨过"了这种差异。

**反思**：对角线规则是"记住特例，理解本质"的绝佳教学素材——"不是死记 Li 像 Mg、Be 像 Al、B 像 Si，而是理解离子势 φ=z/r 才是幕后推手。"

## 十一、易错点

| 易错点 | 学生为什么会这么想 | 正确认识 | 口诀 |
|:---|:---|:---|:---|
| "同族元素性质递变是均匀的" | 把周期律理解成"线性渐变" | 第二周期与第三周期之间的性质差异往往**大于**第三与第四周期之间的差异——因为第二周期没有 d 轨道。同时，第四与第五周期之间因镧系收缩也会出现突变 | "周期表不是调色板——二周期跳到三周期是断崖，不是缓坡" |
| "含氧酸酸性越强，氧化性也越强" | 把"酸"的两种"强"混为一谈 | 酸性取决于**非羟基氧数量**（质子释放难易），氧化性取决于**中心原子接受电子的倾向**。HClO₄ 酸性最强（非羟基氧 3 个）但氧化性最弱；HClO 酸性最弱但氧化性最强 | "高氯酸——强酸弱氧化；次氯酸——弱酸强氧化。别把强弱串了" |
| "周期表中从上到下金属性递增，所以 Tl 应该比 Al 更活泼" | 金属性=活泼性的等式 | Tl 因 6s² 惰性电子对效应，+1 稳定而 +3 极不稳定；Al 是两性活泼金属。**金属性 ≠ 化学活泼性**——Tl 在空气中比 Al 更稳定（表面氧化膜保护 + 惰性电子对减少反应通道） | "铊虽然金属性强——但它懒，6s² 不干活" |
| "对角线规则就是死记 Li~Mg、Be~Al、B~Si" | 当成三条孤立的"对应关系" | 对角线规则的本质是**离子势 φ=z/r 的匹配**——Li⁺(60pm,+1) 与 Mg²⁺(65pm,+2) 的 z/r 值相近。有本质就可以预测更多——例如 Na⁺~Ca²⁺ 有部分相似性（但不如 Li~Mg 显著） | "不是背三对——是算 z/r" |
| "主族元素化学就是各族的堆砌" | 学完 IA→IIA→…→VIIA → 主族化学 = 各族之和 | 主族元素的横向联系（同周期趋势、跨族效应）远比纵向联系（同族递变）容易被忽略——但竞赛的区分度恰恰在"跨族"上。**元素推断题不可能只考一个族** | "合起来不是加法——是乘法" |

## 十二、🎯 教学视角

### 12.1 学生典型认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 教学干预 |
|:---|:---|:---|:---|
| "学完八个主族就掌握了主族化学" | 把"学过"当成"掌握了联系" | 主族元素化学的真正难点不在单个族的知识，而在**跨族比较、横向串联、综合推断**。八个族的独立知识 ≈ 砖块，但考试考的是"用这些砖盖房子" | 每学完两个族就做一次跨族对比表，建立"横向神经连接" |
| "只要记住反应方程式就够了" | 把化学当成方程式记忆比赛 | 主族元素化学中，**理解"为什么会发生这个反应"比记住方程式本身更重要**——惰性电子对效应解释 PbO₂ 的氧化性、d 轨道解释 SiCl₄ 的水解、缺电子解释 B₂H₆ 的结构 | 每讲一个反应必追问"为什么"——不是"Cu+HNO₃ 生成 NO"，而是"为什么稀 HNO₃ 的还原产物是 NO 而不是 NO₂" |

### 12.2 讲授主线

（整合自 [[教学逻辑提炼-Zchem-主族元素的结构化学I-第二轮深化]] 与各提炼页）

主族元素化学的整体教学应按"四条纵线 + 三条横线"组织：

**四条纵线（按族推进）**：
1. **卤素** → 氧族 → 氮族（出题频率最高的三个主族，优先讲透彻）
2. **碳族** → 硼族（缺电子化学 + 惰性电子对效应）
3. **碱金属** → 碱土金属（s 区金属 + 对角线规则）
4. **氢** → 稀有气体（特殊位置 + 储氢/稀有气体化合物）

**三条横线（跨族串联）**：
1. **惰性电子对效应线**：Tl(I) > Tl(III) → Pb(II) > Pb(IV) → Bi(III) > Bi(V)。从 IIIA→IVA→VA→VIA 横向走一遍
2. **对角线规则线**：Li~Mg → Be~Al → B~Si。从 IA/IIA→IIA/IIIA→IIIA/IVA 对角线走一遍
3. **第二周期特殊性线**：N₂ 惰性（N≡N）、O₂ 顺磁性（MO）、F₂ 无正氧化态、HF 弱酸——四个"反直觉"全部集中在前两个周期

### 12.3 与现实/直觉的连接

- **合成氨养活世界**：N₂ + 3H₂ → 2NH₃ → 化肥 → 全球 ~50% 的粮食依赖于 Haber-Bosch 反应。这个反应连接了氮族（N₂）、氢和工业催化——是主族元素化学"改变人类命运"的最震撼例子
- **铅蓄电池与电动车**：Pb + PbO₂ + 2H₂SO₄ ⇌ 2PbSO₄ + 2H₂O。这是主族元素 + 电化学 + 惰性电子对效应（Pb(0) 和 Pb(IV) 归中为 Pb(II)）的完美交叉
- **从食盐到氯气**：NaCl（海水晒盐）→ 电解 → Cl₂ + NaOH + H₂。这三个产物分别走向：Cl₂ → PVC 塑料/漂白剂/消毒；NaOH → 造纸/制皂；H₂ → 合成氨。一条电解食盐水串联起三个主族（IA/卤素/氢）的工业化学

## 十三、竞赛拓展

1. **主族元素的配位化学**：传统观念中"主族元素不形成配合物"是错误的。冠醚（18-crown-6 + K⁺）、穴醚（cryptand + Na⁺）、EDTA + Ca²⁺/Mg²⁺——主族元素的配位化学在分析化学（络合滴定）、生物无机（Na⁺/K⁺-ATP 酶、Mg²⁺-叶绿素）和材料化学中极其重要。竞赛可考查"为什么 18-crown-6 选择性结合 K⁺ 而不选 Na⁺"（孔径匹配）

2. **稀有气体化合物的结构化学**：XeF₂（直线形，3c-4e 键）、XeF₄（平面正方形）、XeF₆（畸变八面体，VSEPR 预测失败案例——因 Xe 的孤对电子在 7 配位时不再保持立体化学活性）。XeO₃ 是三角锥（与 SO₃²⁻ 等电子），XeO₄ 是正四面体。这些是 VSEPR 理论的"终极考题"

3. **主族元素化学的"反常"总汇与结构根源**：将全主族的"反常"汇总——N₂ 反常惰性（N≡N 叁键 945 kJ/mol）、O₂ 顺磁性（MO 理论 π* 轨道两个未成对电子）、F₂ 无正氧化态（F 是电负性最大元素，永远得电子）、HF 弱酸（H—F 键解离能大 + 氢键）、H₃BO₃ 一元弱酸（Lewis 酸机制而非 Brønsted 酸）、NF₃ 几乎非碱性（N—F 键偶极反向拉回孤对电子）、HClO₄ 弱氧化性（+7 氧化态动力学稳定）——给学生一个统一的"结构解释框架"
