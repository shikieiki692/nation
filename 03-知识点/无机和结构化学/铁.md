# 铁

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]

## 一、概述
- [Ar]3d⁶4s²
- 重要氧化态：+2（亚铁）、+3（铁）
- 地壳中含量第四的元素，最重要工业金属

## 二、重要反应

### Fe²⁺ vs Fe³⁺
| 性质 | Fe²⁺ (d⁶) | Fe³⁺ (d⁵) |
|------|------|------|
| 水合离子颜色 | 浅绿 | 浅紫(水解后黄褐) |
| 稳定性 | 还原性（易氧化） | 氧化性（较弱） |
| pKa | ~9.5 | ~2.2（强水解） |

### 铁的配合物（竞赛热门）
| 配合物 | 颜色 | 自旋态 | 说明 |
|------|:---:|:---:|------|
| [Fe(H₂O)₆]²⁺ | 浅绿 | HS | Δo 小 |
| [Fe(H₂O)₆]³⁺ | 浅紫 | HS | 易水解 |
| [Fe(CN)₆]⁴⁻ | 黄 | LS | Δo 大 |
| [Fe(CN)₆]³⁻ | 红 | LS | |
| [Fe(SCN)]²⁺ | **血红色** | — | 鉴定 Fe³⁺ |
| [Fe(phen)₃]²⁺ | 深红 | LS | 邻菲罗啉铁（ferroin） |

### 鉴定反应
- Fe³⁺ + SCN⁻ → [Fe(SCN)]²⁺（血红色）
- Fe³⁺ + K₄[Fe(CN)₆] → Fe₄[Fe(CN)₆]₃（普鲁士蓝）
- Fe²⁺ + K₃[Fe(CN)₆] → Fe₃[Fe(CN)₆]₂（滕氏蓝 = 普鲁士蓝同物）
- Fe²⁺ + 邻菲罗啉(phen) → [Fe(phen)₃]²⁺（深红）

## 三、核心原理

### 3.1 铁的配位化合物
Fe(II) 和 Fe(III) 都可与卤素、类卤离子（$\mathrm{CN}^{-}$、$\mathrm{CNS}^{-}$）、$\mathrm{NH}_3$、$\mathrm{H}_2\mathrm{O}$ 等配体形成配合物。

**普鲁士蓝与滕氏蓝**：黄血盐 $\mathrm{K}_4[\mathrm{Fe(CN)}_6]$ 与 $\mathrm{Fe}^{3+}$ 反应，赤血盐 $\mathrm{K}_3[\mathrm{Fe(CN)}_6]$ 与 $\mathrm{Fe}^{2+}$ 反应都生成蓝色沉淀。X 射线结构分析证明普鲁士蓝和滕氏蓝是同一种物质，化学组成 $\mathrm{KFe}[\mathrm{Fe(CN)}_6]$。结构中 $\mathrm{Fe}^{2+}$ 和 $\mathrm{Fe}^{3+}$ 分别占角，CN 占边，N 原子与 Fe(III) 键合，C 原子与 Fe(II) 键合，$\mathrm{K}^{+}$ 间隔位于立方体中心。

### 3.2 Fe(II) 和 Fe(III) 的相对稳定性
- 水溶液中 $E^{\ominus}(\mathrm{Fe}^{3+}/\mathrm{Fe}^{2+}) = 0.771\mathrm{V}$，$\mathrm{Fe}^{3+}$ 稳定，$\mathrm{Fe}^{2+}$ 可作还原剂
- 碱性条件下 $\mathrm{Fe}^{2+}$ 和 $\mathrm{OH}^{-}$ 生成白色 $\mathrm{Fe(OH)}_2$，很快被空气氧化成棕红色 $\mathrm{Fe(OH)}_3$
- 配制 $\mathrm{FeSO}_4$ 或 $\mathrm{FeCl}_2$ 溶液不仅要保持酸性，还要加铁屑或铁丝防氧化

### 3.3 铁的羰基化合物
大部分 d 区元素都可生成 CO 配合物（羰基化合物），如 $\mathrm{Fe(CO)}_5$、$\mathrm{Ni(CO)}_4$、$\mathrm{Mo(CO)}_6$。成键情况：C 上的孤对电子与金属空轨道形成 $\sigma$ 配键；同时金属 d 轨道电子和 CO 分子的空反键轨道 $\pi_{2\mathrm{p}}^{*}$ 重叠形成反馈 $\pi$ 键。羰基化合物一般较易挥发、毒性较大、受热易分解。

### 3.4 高炉炼铁原理
铁矿石以赤铁矿 $\mathrm{Fe}_2\mathrm{O}_3$ 为主。焦炭和热空气产生 CO，CO 在高炉上部（温度较低）使 $\mathrm{Fe}_2\mathrm{O}_3$ 和 $\mathrm{Fe}_3\mathrm{O}_4$ 还原为 FeO，在温度较高的高炉中部下部使 FeO 还原为金属铁。石灰石 $\mathrm{CaCO}_3$ 分解为 $\mathrm{CaO}$，与脉石中 $\mathrm{SiO}_2$ 生成炉渣 $\mathrm{CaSiO}_3$，与铁水分离。

### 3.5 实验室制备铁的化合物
- **无水 $\mathrm{FeCl}_3$**：干法为 $\mathrm{Cl}_2$ 与热铁丝反应；湿法先用铁屑和 HCl 制 $\mathrm{FeCl}_2$ 溶液，通 $\mathrm{Cl}_2$ 氧化，在 HCl 气体保护下浓缩结晶得 $\mathrm{FeCl}_3\cdot 6\mathrm{H}_2\mathrm{O}$，再用 $\mathrm{SOCl}_2$ 脱水
- **硫酸亚铁铵**：$\mathrm{FeSO}_4$ 和 $(\mathrm{NH}_4)_2\mathrm{SO}_4$ 溶液按 $1:1$ 摩尔比混合，蒸发浓缩冷却结晶，稳定不易氧化
- **三草酸合铁(III)酸钾**：$\mathrm{K}_3[\mathrm{Fe(C}_2\mathrm{O}_4)_3]$，$\mathrm{Fe(C}_2\mathrm{O}_4)_3^{3-}$ 的 $K_{\text{稳}} = 2\times 10^{20}$

## 四、关键结论
- Fe 是地壳中丰度第四的元素（质量克拉克 $10^0$ 量级），最重要工业金属
- 普鲁士蓝 = 滕氏蓝（同一种物质），是 $\mathrm{Fe(II)}$ 和 $\mathrm{Fe(III)}$ 的混合价态（同素异价）化合物
- 高炉炼铁主要还原剂是 CO（非焦炭直接还原）
- $\mathrm{FeSO}_4\cdot(\mathrm{NH}_4)_2\mathrm{SO}_4\cdot 6\mathrm{H}_2\mathrm{O}$（硫酸亚铁铵/莫尔盐）是实验室常用的稳定亚铁盐
- 含不同结晶水的 $\mathrm{FeCl}_3$ 脱水需用 $\mathrm{SOCl}_2$ 而非直接加热（防止水解）

## 五、常见分类或情形
- **铁的氧化物**：$\mathrm{FeO}$（黑色）、$\mathrm{Fe}_2\mathrm{O}_3$（赤铁矿，红棕）、$\mathrm{Fe}_3\mathrm{O}_4$（磁铁矿，含 Fe(II) 和 Fe(III)）
- **铁的配合物**：黄血盐 $\mathrm{K}_4[\mathrm{Fe(CN)}_6]$、赤血盐 $\mathrm{K}_3[\mathrm{Fe(CN)}_6]$、普鲁士蓝 $\mathrm{KFe}[\mathrm{Fe(CN)}_6]$
- **铁的羰基化合物**：$\mathrm{Fe(CO)}_5$
- **铁盐**：$\mathrm{FeCl}_3$（水解性强）、$\mathrm{FeSO}_4\cdot 7\mathrm{H}_2\mathrm{O}$（绿矾）、硫酸亚铁铵（莫尔盐）

## 六、适用条件与限制
- $\mathrm{FeCl}_2$ 溶液需保持酸性并加铁屑防氧化
- $\mathrm{FeCl}_3\cdot 6\mathrm{H}_2\mathrm{O}$ 直接加热会水解生成 $\mathrm{Fe(OH)Cl}$ 而非无水 $\mathrm{FeCl}_3$
- 无水 $\mathrm{FeCl}_3$ 极易水解，必须在无水条件下制备和保存
- 高炉炼铁温度需足够高（$\mathrm{CaCO}_3$ 分解、炉渣熔融）

## 七、常见比较与易混点
| 对比项 | Fe(II) | Fe(III) |
|------|------|------|
| 水合离子颜色 | 浅绿 | 浅紫（水解后黄褐） |
| 稳定性 | 还原性（易氧化） | 氧化性（较弱） |
| $\mathrm{CN}^{-}$ 配合物 | $\mathrm{K}_4[\mathrm{Fe(CN)}_6]$（黄血盐） | $\mathrm{K}_3[\mathrm{Fe(CN)}_6]$（赤血盐） |
| $E^{\ominus}$ | — | $0.771\mathrm{V}$ |

- 普鲁士蓝 vs 滕氏蓝：同一物质！制备原料不同但产物相同
- $\mathrm{Fe(OH)}_2$（白色）$\to$ $\mathrm{Fe(OH)}_3$（棕红）是鉴定 $\mathrm{Fe}^{2+}$ 的特征转化

## 十、例题
- **例题**：向 $\mathrm{FeSO}_4$ 溶液中先滴加 NaOH，再暴露在空气中，现象如何变化？  
  **答案要点**：先生成白绿色 $\mathrm{Fe(OH)}_2$ 沉淀，随后被空气氧化为棕红色 $\mathrm{Fe(OH)}_3$。该过程体现 $\mathrm{Fe}^{2+}$ 易被氧化、$\mathrm{Fe}^{3+}$ 易水解两条主线。

## 十一、易错点
- 误以为普鲁士蓝和滕氏蓝是两种完全不同的物质
- 误把高炉中焦炭当作唯一还原剂，忽略 CO 才是主要还原剂
- 误认为 $\mathrm{Fe}^{3+}$ 一定比 $\mathrm{Fe}^{2+}$“更稳定”，没有区分介质和配体环境

## 十二、🎯 教学视角

- 与分析化学连接时，可顺接邻菲罗啉显色和重铬酸钾法中的亚铁还原性
