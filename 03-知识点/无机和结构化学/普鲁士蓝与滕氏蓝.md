# 普鲁士蓝与滕氏蓝

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[12-配合物]]、[[08-氧化还原与电化学]]、[[17-常见晶体结构]]

## 一、定义
- **普鲁士蓝（Prussian blue, PB）**：$\mathrm{KFe^{III}[Fe^{II}(CN)_6]}$（或 $\mathrm{KFe_2(CN)_6}$），混合价态配合物
- **滕氏蓝（Turnbull's blue）**：与普鲁士蓝实际为同一物质（$\mathrm{Fe^{II}}$ + $\mathrm{[Fe^{III}(CN)_6]^{3-}}$ → $\mathrm{Fe^{III}[Fe^{II}(CN)_6]}$，发生电子转移）
- **普鲁士白（Prussian white）**：$\mathrm{K_2Fe_2^{II}(CN)_6}$（完全还原态）

- 对应考纲条目：[[12-配合物]]、[[08-氧化还原与电化学]]
- 所属模块：[[基础要求-无机和结构化学]]

## 三、核心原理

### 3.1 传统制备与鉴别

- **黄血盐**（亚铁氰化钾）：$\mathrm{K_4[Fe(CN)_6]}$，与 $\mathrm{Fe^{3+}}$ 反应生成普鲁士蓝沉淀
- **赤血盐**（铁氰化钾）：$\mathrm{K_3[Fe(CN)_6]}$，与 $\mathrm{Fe^{2+}}$ 反应生成滕氏蓝沉淀
- 这两个反应是检验 $\mathrm{Fe^{3+}}$ 与 $\mathrm{Fe^{2+}}$ 的经典方法

### 3.2 晶体结构

- X 射线结构分析证明，普鲁士蓝和滕氏蓝实际上是**同一种物质**，化学组成均为 $\mathrm{KFe[Fe(CN)_6]}$，属同素异价化合物
- 简单立方结构，$\mathrm{Fe(II)}$ 和 $\mathrm{Fe(III)}$ 位于立方体顶点
- $\mathrm{CN^-}$ 为立方体的边（C 端配位 $\mathrm{Fe(II)}$，N 端配位 $\mathrm{Fe(III)}$）
- $\mathrm{K^+}$ 占据立方体空腔（体心位置，间隔分布），$\mathrm{K:Fe:(CN)} = 1:2:6$
- 相邻 Fe-Fe 距离 $a = 0.510\ \mathrm{nm}$

### 3.3 颜色来源：电荷转移（CT）跃迁

$$\mathrm{Fe^{II}} \rightarrow \mathrm{C\!\equiv\!N} \rightarrow \mathrm{Fe^{III}}$$

- $\mathrm{CN^-}$ C 端：强场配体 → $\mathrm{Fe^{II}}$：$t_{2g}^6 e_g^0$（低自旋）
- $\mathrm{CN^-}$ N 端：中强场配体 → $\mathrm{Fe^{III}}$：$t_{2g}^3 e_g^2$
- $\mathrm{Fe^{II}}$ → $\mathrm{Fe^{III}}$ 的 CT 跃迁：**对称性允许**，吸收截面大
- 红光区宽带吸收 → 透射蓝光 → **深蓝色**
- 还原为普鲁士白（全部 $\mathrm{Fe^{II}}$）→ CT 通道破坏 → **白色**

### 3.4 电化学储钾

$$\mathrm{KFe^{III}Fe^{II}(CN)_6} + \mathrm{K^+} + \mathrm{e^-} \longrightarrow \mathrm{K_2Fe_2^{II}(CN)_6}$$

$$\mathrm{Fe_2(CN)_6} + 2\mathrm{K^+} + 2\mathrm{e^-} \rightleftharpoons \mathrm{K_2Fe_2(CN)_6}\ (\text{可逆})$$

理论比容量：$C = \dfrac{2F}{M} = \dfrac{2 \times 96485}{306.92} = 174.65\ \mathrm{mA\ h\ g^{-1}}$

### 3.5 扩散动力学

离子在 PBA 颗粒中的扩散服从 Fick 定律：

$$Q(t) = Q_0 \exp\!\left(-\frac{\pi^2 D}{4L^2}t\right)$$

- 扩散时间常数 $\tau \sim L^2/D$
- 纳米化可显著加快充放电速率
- 接触不良时转为欧姆控制：$Q(t) = Q_0 - (U/R)t$

## 四、关键结论
- 普鲁士蓝 = 滕氏蓝（同一物质），$\mathrm{KFe^{III}Fe^{II}(CN)_6}$
- 蓝色来源于 **$\mathrm{Fe^{II} \to Fe^{III}}$ 的 CT 跃迁**（非 d-d 跃迁）
- 作为钾离子电池正极：理论比容量 **~175 mAh/g**
- 扩散控制 vs 接触电阻控制：两种不同的动力学模式

## 五、常见分类或情形
- **普鲁士蓝类化合物 (PBA)**：通式 $\mathrm{A_xM_y[M'(CN)_6]_z}$，A 为碱金属
- **普鲁士蓝**：$\mathrm{KFe_2(CN)_6}$，蓝色
- **普鲁士白**：$\mathrm{K_2Fe_2(CN)_6}$，白色
- **普鲁士绿**：部分氧化态中间体

## 六、适用条件与限制
- 颜色判断：d-d 跃迁 vs CT 跃迁 → CT 吸收更强（对称性允许）、带更宽
- 比容量计算：$n$ 为**可逆电子数**（2 mol $\mathrm{K^+}$/mol $\mathrm{Fe_2(CN)_6}$ 单元）
- 扩散系数 $D$ 的数量级：~$10^3\ \mathrm{nm^2/s}$

## 七、常见比较与易混点
- **普鲁士蓝 vs 滕氏蓝**：同一物质！制备原料不同，但产物一致
- **CT 跃迁 vs d-d 跃迁**：CT 对称性允许（强吸收）vs d-d 对称性禁阻（弱吸收）
- **扩散控制 vs 接触控制**：指数衰减 vs 线性衰减

## 十、例题
### 例题 1（39 届初赛 2.1.2）
**题目：** 解释 PBA-1（蓝色） → 普鲁士白（白色）的颜色变化。
**分析：** $\mathrm{Fe^{II} \to Fe^{III}}$ CT 通道 → 吸收红光 → 蓝色；还原后全部 $\mathrm{Fe^{II}}$，CT 通道消失 → 白色。
**解答：** CT 跃迁（电荷转移）的建立与破坏。

### 例题 2（39 届初赛 2.2）
**题目：** 边长为 180 nm 的立方体颗粒，$Q_0 = 3.52 \times 10^{-12}\ \mathrm{C}$，$t=5.00\ \mathrm{s}$ 时 $Q=0.964 \times 10^{-12}\ \mathrm{C}$，求 $D$。
**分析：** $Q = Q_0\exp(-\pi^2 D t / 4L^2)$ → $D = -(4L^2/\pi^2 t)\ln(Q/Q_0)$。
**解答：** $D = 3.40 \times 10^3\ \mathrm{nm^2\ s^{-1}}$。

## 十一、易错点
- 将颜色归因于 d-d 跃迁（实际是 CT 跃迁）
- 比容量只算 1 个 $\mathrm{K^+}$（应算可逆的 2 个）
- 颗粒质量计算忘记化学式中有 2 当量 Fe（式量应除以 2）
