# 氧族元素（O, S, Se, Te）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]

## 一、定义
VIA 族：ns²np⁴。O 是第二周期（特殊），S/Se/Te 是第三周期及以下。

- O₂：顺磁性（MO 理论经典案例），$\Pi_3^2$ 双自由基
- O₃：臭氧，V形，$\Pi_3^4$ 离域π键，强氧化剂
- S：多种同素异形体（S₈ 环状最稳定，弹性硫 Sₙ 链状）

## 二、考纲对应
- 对应考纲条目：[[13-元素化学]]
- 所属模块：[[基础要求-无机和结构化学]]
- 本知识点在考纲中的作用：是主族元素化学中最能承接氧化还原、结构与工业应用的主链之一，尤其要服务硫体系和含氧酸盐相关题目。

## 三、核心原理
### 硫的化合物（竞赛重中之重）
| 化合物 | 氧化态 | 结构要点 |
|------|:---:|------|
| H₂S | −2 | 弱酸，强还原剂 |
| S₈ | 0 | 环状 S₈ 分子 |
| SO₂ | +4 | V形，还原性为主 |
| SO₃ | +6 | 平面三角形，强烈吸水 |
| H₂SO₄ | +6 | 高沸点，强氧化性（浓热），脱水性 |

### 硫的含氧酸
| 酸 | 式 | 特点 |
|------|------|------|
| 亚硫酸 H₂SO₃ | — | 仅存在于溶液，SO₂ 的水溶液 |
| 硫酸 H₂SO₄ | — | 最强二元酸之一 |
| 硫代硫酸 H₂S₂O₃ | — | 仅存在于溶液 |
| 焦硫酸 H₂S₂O₇ | — | 发烟硫酸 |
| 过硫酸 H₂S₂O₈ | — | 含 −O−O− 桥，强氧化剂 |
| 连二亚硫酸 H₂S₂O₄ | — | 强还原剂（保险粉 Na₂S₂O₄） |

### 硫代硫酸钠 Na₂S₂O₃
- 中心 S(+6) 四面体 + 端 S(−2)
- 还原性：2S₂O₃²⁻ + I₂ → S₄O₆²⁻ + 2I⁻（碘量法定量）
- 定影：AgBr + 2S₂O₃²⁻ → [Ag(S₂O₃)₂]³⁻ + Br⁻

## 三、核心原理

### 3.1 氧元素的特殊性
氧的价层电子为 $2\mathrm{s}^{2}2\mathrm{p}^{4}$，电负性仅次于氟（F 为 4.0，O 为 3.5）。作为第二周期元素没有可供成键用的 d 轨道，所以氧化数最高为 $+2$（仅在 $\mathrm{OF}_2$ 中），一般化合物中 O 的氧化数总是 $-2$。S、Se、Te 都有氧化数为 $+4$ 和 $+6$ 的化合物。

### 3.2 氧气的工业制备
空气中有取之不尽的 $\mathrm{O}_2$，沸点 $-183^{\circ}\mathrm{C}$，$\mathrm{N}_2$ 的沸点 $-196^{\circ}\mathrm{C}$，利用两者沸点之差用分馏法分离，可得纯度为 $99.5\%$ 的 $\mathrm{O}_2$，储于蓝色高压钢瓶中（压力约为 $1.5\times 10^{4}\mathrm{kPa}$）。

### 3.3 臭氧
$\mathrm{O}_3$ 是 $\mathrm{O}_2$ 的同素异形体，折线状结构，键角 $116.8^{\circ}$，键长 $127.8\mathrm{nm}$，中间的 O 以 $\mathrm{sp}^2$ 杂化轨道和另外 2 个 O 形成 $\sigma$ 键，还有一个离域的 $\pi_3^4$ 键。$\Delta_{\mathrm{r}}G_{\mathrm{m}}^{\ominus}(3\mathrm{O}_2 \to 2\mathrm{O}_3) = 326\mathrm{kJ}\cdot\mathrm{mol}^{-1}$。

臭氧层位于离地面约 $15\sim 35\mathrm{km}$ 的高度，吸收波长在 $180\sim 310\mathrm{nm}$ 的太阳强紫外线辐射，保护地球生物。臭氧是很好的消毒剂、漂白剂，可用于污水处理：$5\mathrm{O}_3 + 2\mathrm{CN}^{-} + \mathrm{H}_2\mathrm{O} \longrightarrow 2\mathrm{HCO}_3^{-} + \mathrm{N}_2 + 5\mathrm{O}_2$

### 3.4 过氧化物和超氧化物
- 过氧化物含过氧离子 $\mathrm{O}_2^{2-}$（O 的氧化数为 $-1$），如 $\mathrm{Na}_2\mathrm{O}_2$、$\mathrm{H}_2\mathrm{O}_2$
- 超氧化物含超氧离子 $\mathrm{O}_2^{-}$（O 的氧化数为 $-1/2$），如 $\mathrm{KO}_2$
- $\mathrm{Na}_2\mathrm{O}_2$ 和 $\mathrm{KO}_2$ 用作供氧剂：$2\mathrm{Na}_2\mathrm{O}_2 + 2\mathrm{CO}_2 \longrightarrow 2\mathrm{Na}_2\mathrm{CO}_3 + \mathrm{O}_2$；$4\mathrm{KO}_2 + 2\mathrm{CO}_2 \longrightarrow 2\mathrm{K}_2\mathrm{CO}_3 + 3\mathrm{O}_2$

### 3.5 含氧酸盐的热稳定性
以钙盐为例：$\mathrm{CaSiO}_3 > \mathrm{CaSO}_4 > \mathrm{CaCO}_3$（热稳定性依次减弱）
- $\mathrm{CaCO}_3$ 分解温度约 $826^{\circ}\mathrm{C}$（工业石灰窑约 $900^{\circ}\mathrm{C}$）
- $\mathrm{CaSO}_4$ 分解需约 $1800^{\circ}\mathrm{C}$，不可取
- $\mathrm{CaSiO}_3$ 非常稳定，高炉炼铁成渣利用 $\mathrm{CaO} + \mathrm{SiO}_2 \to \mathrm{CaSiO}_3$

## 四、关键结论
- 氧是丰度最大的元素，质量几乎占地壳的一半（约 $46\%$）；空气中 $\mathrm{O}_2$ 约占 $21\%$
- O 的电负性 (3.5) 仅次于 F (4.0)，大多数化合物中 O 为 $-2$ 氧化态
- $\mathrm{O}_3$ 的分子结构为折线形（非正三角形），含离域 $\pi_3^4$ 键
- 过氧化物和超氧化物既是强氧化剂又是供氧剂
- 含氧酸盐的热稳定性：硅酸盐 > 硫酸盐 > 碳酸盐 > 硝酸盐

## 五、常见分类或情形
- **普通氧化物**：O 的氧化数 $-2$
- **过氧化物**：含 $\mathrm{O}_2^{2-}$（O 的氧化数 $-1$），如 $\mathrm{Na}_2\mathrm{O}_2$、$\mathrm{H}_2\mathrm{O}_2$、$\mathrm{H}_2\mathrm{S}_2\mathrm{O}_8$（过二硫酸）
- **超氧化物**：含 $\mathrm{O}_2^{-}$（O 的氧化数 $-1/2$），如 $\mathrm{KO}_2$
- **含氧酸**：$\mathrm{H}_2\mathrm{SO}_4$、$\mathrm{H}_2\mathrm{SO}_3$、$\mathrm{H}_2\mathrm{S}_2\mathrm{O}_3$（硫代硫酸）等
- **常见矿石**：赤铁矿 $\mathrm{Fe}_2\mathrm{O}_3$、磁铁矿 $\mathrm{Fe}_3\mathrm{O}_4$、软锰矿 $\mathrm{MnO}_2$、黄铁矿 $\mathrm{FeS}_2$ 等

## 六、适用条件与限制
- $\mathrm{H}_2\mathrm{O}_2$ 市售试剂为 $30\%$ 左右的水溶液，微量金属离子（$\mathrm{Fe}^{3+}$ 或 $\mathrm{MnO}_2$）会催化其分解
- 过氧乙酸稀溶液（约 $0.5\%$）是高效广谱杀菌消毒剂
- $\mathrm{Na}_2\mathrm{O}_2$ 兼有碱性和氧化性，是很好的熔矿剂
- $\mathrm{CaCO}_3$ 分解温度 $826^{\circ}\mathrm{C}$ 是标准态计算值，实际操作需更高温度

## 七、常见比较与易混点
| 对比项 | 过氧化物 | 超氧化物 |
|------|------|------|
| 阴离子 | $\mathrm{O}_2^{2-}$ | $\mathrm{O}_2^{-}$ |
| O 的氧化数 | $-1$ | $-1/2$ |
| 实例 | $\mathrm{Na}_2\mathrm{O}_2$, $\mathrm{BaO}_2$ | $\mathrm{KO}_2$, $\mathrm{RbO}_2$, $\mathrm{CsO}_2$ |
| 与水反应 | $\mathrm{Na}_2\mathrm{O}_2 + 2\mathrm{H}_2\mathrm{O} \to 2\mathrm{NaOH} + \mathrm{H}_2\mathrm{O}_2$ | $2\mathrm{KO}_2 + 2\mathrm{H}_2\mathrm{O} \to 2\mathrm{KOH} + \mathrm{H}_2\mathrm{O}_2 + \mathrm{O}_2$ |

- $\mathrm{O}_2$ (顺磁性，双自由基) vs $\mathrm{O}_3$ (抗磁性)
- S 有多种同素异形体（$\mathrm{S}_8$ 环状最稳定），O 同素异形体只有 $\mathrm{O}_2$ 和 $\mathrm{O}_3$

## 十、例题

### 例题 1（基础 · 过氧化物与超氧化物判别）

**题目**：Na₂O₂、KO₂、BaO₂ 分别与水、CO₂ 反应，写出方程式并指出哪个最适合潜艇供氧。

**分析**：供氧剂的核心指标：单位质量释 O₂ 量 + 是否同时吸收 CO₂。

**解答**：
- Na₂O₂ + 2H₂O → 2NaOH + H₂O₂（H₂O₂ 随后分解：2H₂O₂ → 2H₂O + O₂）
  总反应：2Na₂O₂ + 2H₂O → 4NaOH + O₂↑
  与 CO₂：2Na₂O₂ + 2CO₂ → 2Na₂CO₃ + O₂↑
- 2KO₂ + 2H₂O → 2KOH + H₂O₂ + O₂↑
  与 CO₂：4KO₂ + 2CO₂ → 2K₂CO₃ + 3O₂↑
- BaO₂ 与水/CO₂ 反应也放 O₂，但 Ba 摩尔质量大，单位质量产 O₂ 少

**最适合潜艇的是 KO₂**：同时吸收 CO₂ 并放出 O₂，且 O₂/质量比最高（4KO₂ → 3O₂）。Na₂O₂ 次之。

**反思**：O₂²⁻ 中 O 氧化数 −1，O₂⁻ 中 O 氧化数 −1/2。超氧化物比过氧化物"更不还原"——相同条件下释出更多 O₂。

### 例题 2（提高 · 硫代硫酸钠的定量反应）

**题目**：取某工业废水（含 I₂）50.00mL，用 0.1000mol/L Na₂S₂O₃ 滴定至淀粉褪色，消耗 24.30mL。求废水中 I₂ 的质量浓度（g/L）。

**分析**：碘量法的核心反应——2S₂O₃²⁻ + I₂ → S₄O₆²⁻ + 2I⁻。连四硫酸根 S₄O₆²⁻ 中 S 的平均氧化数为 +2.5。

**解答**：
$n(Na_2S_2O_3) = 0.1000 \times 24.30 \times 10^{-3} = 2.430 \times 10^{-3} \text{mol}$
$n(I_2) = \frac{1}{2}n(Na_2S_2O_3) = 1.215 \times 10^{-3} \text{mol}$
$m(I_2) = 1.215 \times 10^{-3} \times 254 = 0.3086 \text{g}$
$\rho(I_2) = 0.3086 / 0.05000 = 6.17 \text{g/L}$

**反思**：碘量法测 I₂ 是定量分析最经典的氧化还原滴定之一。记住计量比：$I_2 \sim 2S_2O_3^{2-}$。淀粉指示剂——I₂ 存在时蓝色，滴定终点褪为无色。

### 例题 3（竞赛 · 等电子体推断 — S₂O）

**题目**：A 与 SO₂ 属于等电子体，常温下为无色气体，冷却到 77K 变为橙红色液体。A 的一种制法：在真空中将 CuO 和硫混合加热，生成 A 和黑色固体 B。A 在碱性条件下发生歧化：$A + OH^- \rightarrow S^{2-} + S_2O_4^{2-} + SO_3^{2-} + H_2O$。若 A 和 OH⁻ 的物质的量比为 4:7，配平该离子方程式。

**分析**：与 SO₂ 等电子 → 分子式 S₂O（S=S=O，V 形）。由 CuO + S → S₂O，CuO 被还原 → 黑色固体为 Cu₂S。

**解答**：
（1）A = **S₂O**，结构为 V 形：S=S→O（中心 S 用 sp²，有离域 π₃⁴ 键）
（2）$2CuO + 5S \xrightarrow{\Delta} 2S_2O + Cu_2S$
（3）歧化可拆为两个独立反应：
- $5S_2O + 14OH^- = 4S^{2-} + 3S_2O_4^{2-} + 7H_2O$
- $S_2O + 4OH^- = S^{2-} + SO_3^{2-} + 2H_2O$
合成 4:7 比例：$12S_2O + 42OH^- = 11S^{2-} + 3S_2O_4^{2-} + 7SO_3^{2-} + 21H_2O$

**反思**：等电子体原理是竞赛推断题的利器——SO₂（V 形，18e⁻）→ S₂O（也是 V 形，18e⁻）。氧化数平均法在硫的歧化配平中极为高效。

### 例题 4（竞赛 · 2005 年全国初赛备用 · H₂O₂ 在超酸中的行为）

**题目**：将 H₂O₂ 慢慢加入 SbF₅ 的 HF 溶液中得到白色固体 A（盐类，负离子呈八面体结构）。A 在室温以上定量分解，产物之一 B 也含八面体结构。若将 H₂S 通入 SbF₅ 的 HF 溶液中得晶体 C。

（1）确定 A、B、C 的结构简式。（2）若将 H₂O₂ 投入到液氨中，得到白色固体 D，其中一种离子呈正四面体。确定 D 的结构简式。（3）上述实验说明 H₂O₂、H₂S 共同具有什么性质？

**分析**：SbF₅ 是强 Lewis 酸，夺取 F⁻ 形成八面体 [SbF₆]⁻。H₂O₂ 接受 H⁺ 形成 [H₃O₂]⁺ → 类比 NH₃ + H⁺ → NH₄⁺。

**解答**：
（1）A = **[H₃O₂]⁺[SbF₆]⁻**。$H_2O_2 + HF + SbF_5 \rightarrow [H_3O_2]^+[SbF_6]^-$
（2）A 分解：$2[H_3O_2]^+[SbF_6]^- \rightarrow 2[H_3O]^+[SbF_6]^- + O_2↑$
    B = **[H₃O]⁺[SbF₆]⁻**（O: sp³，Sb: sp³d²）
（3）C = **[SH₃]⁺[SbF₆]⁻**。$H_2S + HF + SbF_5 \rightarrow [SH_3]^+[SbF_6]^-$
（4）D = **[NH₄]⁺[HOO]⁻**。H₂O₂ 在液氨中表现弱酸性 → 给出 H⁺ 给 NH₃，自身变为 [HOO]⁻
（5）共同性质：H₂O₂ 和 H₂S 都可在不同条件下体现**酸性和碱性**（两性）。

**反思**：这道题的核心是 Lewis 酸碱理论的灵活运用——H₂O₂ 在 SbF₅/HF 体系中是碱（接受 H⁺），在液氨中是酸（给出 H⁺）。超酸化学是竞赛热门方向。

## 十一、易错点

| 易错点 | 学生为什么会这么想 | 正确认识 | 口诀 |
|:---|:---|:---|:---|
| "过氧化物和超氧化物差不多" | 名字只差一个字，直觉上性质接近 | 过氧化物含 O₂²⁻（O 氧化数−1），超氧化物含 O₂⁻（O 氧化数−1/2）。超氧化物水解**同时放 H₂O₂ 和 O₂**，过氧化物只放 H₂O₂。与 CO₂ 反应时 KO₂ 释 O₂ 量是 Na₂O₂ 的 1.5 倍 | "超氧更超——氧化数不是整数，放氧量更猛" |
| "SO₂ 使品红褪色 = Cl₂ 的漂白原理" | 中学把"漂白"都归为一类 | SO₂ 漂白是**化合型**（与品红生成无色加合物，加热可逆）；Cl₂ 漂白是**氧化型**（不可逆破坏色素）。SO₂ 漂白过的草帽加热会恢复红色 | "二氧化硫是粘上去的，氯气是烧掉的——一个可逆一个不可逆" |
| "浓硫酸是强氧化剂所以任何金属都钝化" | 把"氧化性"和"反应活性"混淆 | 浓热 H₂SO₄ 能氧化 Cu、Ag、C、S 等，但 Al、Fe、Cr 在**冷浓** H₂SO₄ 中钝化（表面生成致密氧化膜）。活泼金属 Zn 反而和稀 H₂SO₄ 反应更快 | "浓硫爱钝不爱活——铝铁在浓硫里睡着的，铜在浓硫里反而被吃了" |
| "Na₂S₂O₃ 中两个 S 氧化数相同" | 按平均氧化数(= +2)理解 | Na₂S₂O₃ = Na₂[S₂O₃]：中心 S(+6) 四面体 + 端 S(−2)。两个 S 的氧化数**截然不同**。I₂ 氧化的是端 S(−2) → S₄O₆²⁻ 中 S 平均 +2.5 | "硫代硫酸盐里有一个皇帝硫(+6)和一个奴隶硫(−2)——碘专吃奴隶" |
| "氧族=氧和硫，Se/Te 不重要" | 竞赛中 Se、Te 出现少，容易忽视 | Se 有 +4/+6 氧化态稳定性反转（H₂SeO₄ > H₂SeO₃，与 S 相反！）；Te₄²⁺ 有芳香性。Se/Te 是竞赛区分度的关键点 | "硒碲虽少但刁钻——硫酸的姐妹反而更稳定" |

## 十二、🎯 教学视角

### 12.1 学生典型认知误区

| 误区 | 学生为什么会这么想 | 正确认识 | 教学干预 |
|:---|:---|:---|:---|
| "O₂ 是双键所以抗磁性" | 价键理论画出 O=O 双键，所有电子配对 → 直觉上应该抗磁 | O₂ **顺磁性**——MO 理论中两个 π* 轨道各有一个未成对电子（三重态基态）。这是推翻价键理论最有力的实验证据之一 | 课堂演示：液态 O₂ 被磁铁吸引的视频 |
| "硒的化学和硫一样" | 同族类比直觉 | H₂SeO₄ **比 H₂SeO₃ 更稳定**，与 H₂SO₃（不稳定，仅存于溶液）vs H₂SO₄（稳定）相反。Se 的 +6 氧化态稳定性反常 | 用"周期律也有反例"制造认知冲突 |

### 12.2 讲授主线

（来自 [[教学逻辑提炼-Zchem-主族元素的结构化学I-第二轮深化]]）

氧族元素教学应分三条线：
1. **O 的特殊性**：顺磁性（MO 证据）→ O₃/过氧化物/超氧化物 → 供氧剂应用
2. **S 化学的丰富性**：−2(H₂S) → 0(S₈) → +4(SO₂/H₂SO₃) → +6(H₂SO₄) → 硫代硫酸盐(S₂O₃²⁻)/连二亚硫酸盐(S₂O₄²⁻)——一条完整的"氧化态走廊"
3. **Se/Te 的特殊性质**：作为竞赛区分度补充，只讲与 S 的差异（H₂SeO₄ 稳定性、Te₄²⁺ 芳香性）

### 12.3 与现实/直觉的连接

- **酸雨**：SO₂ → SO₃ → H₂SO₄。化石燃料（煤含 S）燃烧 → 酸雨 → 石灰石建筑腐蚀（CaCO₃ + H₂SO₄ → CaSO₄ + CO₂ + H₂O）。化学原理 + 环境问题的经典案例。
- **黑白照片定影**：AgBr（未感光部分）+ Na₂S₂O₃ → [Ag(S₂O₃)₂]³⁻（可溶配合物）。配位化学 + 氧族元素的交叉应用。
- **臭氧层空洞**：CFCs → Cl 自由基催化 O₃ 分解（$Cl + O_3 \rightarrow ClO + O_2$, $ClO + O \rightarrow Cl + O_2$），一个 Cl 原子可破坏 10 万个 O₃ 分子。

## 十三、竞赛拓展

1. **硫的"氧化态走廊"与电极电势串联**：H₂S → S → S₂O₃²⁻ → SO₂ → SO₄²⁻ 的完整 Latimer 图（酸性 + 碱性介质）。竞赛核心能力：从电势图判断任意两种氧化态之间是否发生氧化还原、歧化或归中。

2. **连二亚硫酸钠（保险粉 Na₂S₂O₄）的结构与反应性**：含 S−S 键，二价阴离子 S₂O₄²⁻ 具有 $C_{2v}$ 对称性（两个 SO₂⁻ 以 S−S 键连接）。强还原剂：$S_2O_4^{2-} + 2H_2O \rightarrow 2HSO_3^- + 2H^+ + 2e^-$。竞赛可能考察其作为一电子还原剂的特殊用途。

3. **Te₄²⁺ 的芳香性**：Te₄²⁺ 是平面正方形四元环，每个 Te 用 sp² 杂化，垂直于平面的 p 轨道共 6 个 π 电子 → 满足 Hückel 4n+2 规则（n=1）→ **无机芳香性**。这是化竞从有机芳香性向无机拓展的经典桥梁。
