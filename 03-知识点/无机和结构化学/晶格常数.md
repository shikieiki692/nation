## 一、定义
晶胞的六个参数：a, b, c（棱长）和 α, β, γ（棱间夹角）。

### 七大晶系的晶格常数特征
| 晶系 | 边长 | 夹角 |
|------|------|------|
| 立方 | a = b = c | α = β = γ = 90° |
| 四方 | a = b ≠ c | α = β = γ = 90° |
| 正交 | a ≠ b ≠ c | α = β = γ = 90° |
| 六方 | a = b ≠ c | α = β = 90°, γ = 120° |
| 三方 | a = b = c | α = β = γ ≠ 90° |
| 单斜 | a ≠ b ≠ c | α = γ = 90°, β ≠ 90° |
| 三斜 | a ≠ b ≠ c | α ≠ β ≠ γ ≠ 90° |

## 三、核心原理
### 晶胞参数的完整描述（§13.2）
六面体晶胞由六个参数完全确定：
- **边长** $a, b, c$：三个周期方向的重复距离
- **夹角**：$\alpha = b \wedge c$（$b$ 轴与 $c$ 轴的夹角），$\beta = c \wedge a$（$c$ 轴与 $a$ 轴的夹角），$\gamma = a \wedge b$（$a$ 轴与 $b$ 轴的夹角）

### X 射线衍射测定晶格常数——Bragg 方程
晶面间距 $d$ 与晶格常数直接相关。Bragg 方程：

$$2d\sin\theta = n\lambda$$

- $\theta$：衍射角（由实验测得）
- $\lambda$：X 射线波长（已知）
- $n$：衍射级数（整数）
- $d$：晶面间距（可由此推导 $a, b, c$）

### 实例数据
- Cu（面心立方）：$a=b=c=0.356\,\mathrm{nm}=356\,\mathrm{pm}$
- Na（体心立方）：由 $r=(\sqrt{3}/4)a$ 关系测定
- Po（简单立方）：$a=2r$

## 四、关键结论
- 晶格常数不仅决定晶胞大小，还与**原子/离子半径**直接关联
- 对于立方晶系，$a$ 和 $r$ 的关系：
  - 简单立方：$a=2r$
  - 体心立方：$r=\sqrt{3}a/4\qquad a=4r/\sqrt{3}$
  - 面心立方：$a=2\sqrt{2}r\qquad r=\sqrt{2}a/4$
- 三方晶系有两种取法：简单三方（$a=b\neq c,\gamma=120^\circ$）和菱面体（$a=b=c,\alpha=\beta=\gamma\neq90^\circ$），两者等价
- 六方密堆积的理想轴比：$c/a = \sqrt{8/3} \approx 1.633$（实际金属常偏离此值）

## 六、适用条件与限制
- 晶胞参数的选取必须符合对称性要求（如表 13.1 所列）
- 单凭 $a=b=c,\ \alpha=\beta=\gamma=90^\circ$ 不足以判定立方晶系——必须检查是否存在 4 个沿体对角线的三重轴
- X 射线波长 $\lambda$ 的数量级应与待测晶面间距 $d$ 相当（$10^{-10}\,\mathrm{m}$ 量级）

## 七、常见比较与易混点
- **晶胞参数 vs 原子间距**：晶胞参数是重复周期（宏观），原子间距是两相邻原子核间距离的一半（原子半径定义）
- **三方晶系的两种表述**等价——仅取法不同
- 七大晶系的"对称性要求"是高于"几何关系"的判据——先看对称元素，再看几何参数
