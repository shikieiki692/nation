# 吸附

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-物理化学深化]]
- 对应考纲条目：[[决赛09]]

## 一、定义

吸附是指气体、液体中的分子或离子在固体或液体表面富集的现象。本质上，界面处粒子所处环境与体相不同，导致表面自由能可通过粒子富集而降低。吸附过程是表面化学和胶体化学的基础，也是竞赛中连接热力学、动力学和结构效应的重要入口。

## 二、考纲对应

本条目对应决赛物理化学中表面现象与吸附的主干内容。竞赛要求通常包括：区分物理吸附与化学吸附，理解吸附量与压力、浓度、温度的关系，掌握典型等温式的物理意义，并能在具体材料或催化背景中讨论表面覆盖与活性位点问题。

## 三、核心原理

1. 表面原子或分子配位不饱和、受力不均衡，因此表面具有较高自由能；吸附可通过表面粒子重新分布降低这一自由能。
2. 吸附常用“表面覆盖度” `\theta`、单位质量吸附量或单位面积吸附量描述。不同定义适用于不同实验表征方式。
3. 在气体吸附中，压强升高通常促进吸附；在溶液吸附中，溶质浓度升高通常促进吸附，但最终会受到位点数或相互作用限制。
4. 物理吸附主要依赖范德华力等弱相互作用，通常可逆、吸附热较低、易形成多层；化学吸附涉及化学键形成，往往更强、更具选择性，多表现为单层吸附。
5. 理想化单层吸附模型可导向 [[Langmuir吸附等温式]]，而复杂多孔材料、表面不均一体系则需要更一般的经验式或多层模型处理。

## 四、关键结论

1. 吸附是界面自由能驱动的表面富集过程，不能简单等同于“溶解”或“进入体相”。
2. 升压通常有利于气体吸附，而升温通常不利于放热吸附过程，因此多数物理吸附在高温下减弱。
3. 化学吸附更强调位点特异性和活化过程，因此其温度依赖性与动力学特征比物理吸附更复杂。
4. 吸附等温线的形状可反映材料表面性质、孔结构和吸附机理，是实验判别的重要信息源。

## 五、常见分类或情形

1. `物理吸附`：低温下更明显，可形成多层，常见于活性炭吸附气体。
2. `化学吸附`：涉及成键与活化，常见于催化剂表面对反应物的定向结合。
3. `单层吸附`：当位点数有限且每个位点最多容纳一个吸附粒子时，常用 Langmuir 模型近似。
4. `多层吸附`：在较高压强下，第一层之外还可继续堆积，常引出 BET 等模型。
5. `溶液吸附`：如表面活性剂在界面上的富集、离子在固体表面的选择性吸附，常和胶体稳定性、表面电性联动讨论。

## 六、适用条件与限制

1. 任何吸附模型都依赖具体假设，不能脱离材料表面结构、孔分布和位点均一性盲目套用。
2. 将实验数据拟合为 Langmuir 直线并不自动证明体系严格满足单层、均一位点、无相互作用假设。
3. 高覆盖度下，吸附粒子之间的横向相互作用会显著改变理想模型预测。
4. 对多孔材料而言，实验测得的“吸附量”常混合了表面吸附、孔填充、毛细凝聚等效应，需要谨慎解读。

## 七、常见比较与易混点

1. `吸附`发生在界面，`吸收`则进入体相内部，两者位置和机理不同。
2. `物理吸附吸附热低`不等于`一定弱到可忽略`；在高比表面积材料中，累积效应仍可非常显著。
3. `化学吸附更强`不等于`吸附量一定更大`，因为位点数目、表面积和动力学限制同样关键。
4. `吸附平衡`与`吸附速率`是两层问题；一个体系可热力学上有利但动力学上建立很慢。

## 八、与其他知识点的联系

1. 与 [[Langmuir吸附等温式]] 直接相连，后者是吸附基础模型中的核心定量表达。
2. 与 [[表面张力]]、[[临界胶束浓度]] 相连，都是界面自由能与表面富集效应的不同表现。
3. 与 [[分子间相互作用深化]] 相连，物理吸附的强弱常由色散力、偶极作用和氢键等决定。
4. 与 [[化学动力学深化]]、催化主题相连，吸附常是异相催化机理中的首步或控速步骤。

## 九、典型题型

1. 根据实验现象判断物理吸附还是化学吸附。
2. 结合吸附量随压强或温度的变化分析吸附机理。
3. 利用等温式线性化关系求吸附常数、饱和吸附量或覆盖度。
4. 在催化背景下分析“吸附过弱 / 过强”对反应速率的影响。

## 十、例题

某固体对气体 A 的吸附在低温下显著增强，且吸附过程可逆、吸附热较小；升高温度后吸附量下降。判断该过程更接近哪类吸附，并说明理由。

解题思路：

1. 低温更有利、升温吸附减弱，说明过程多半为放热且不需明显活化。
2. 吸附过程可逆、吸附热较小，说明相互作用偏弱。
3. 因此更符合物理吸附特征，而非典型化学吸附。

这类题的关键是把“温度效应 + 可逆性 + 吸附热大小”作为一组证据联合判断，而不是只抓住单一现象。

## 十一、易错点

1. 把吸附与吸收、溶解、润湿混为一谈。
2. 误认为所有吸附都随温度升高而增强，忽略大多数吸附过程是放热的。
3. 看到单层模型就默认实际材料表面均一，忽视真实表面的复杂性。
4. 用一个经验拟合式解释全部机理，忽略模型只是近似表达。

## 十二、🎯 教学视角

教学上可先从“为什么表面原子更不稳定”切入，把吸附放回“体系自发降低表面自由能”的主线，再区分物理吸附和化学吸附。这样学生更容易把零散经验现象串成统一解释框架，后续再接 Langmuir 等温式会自然很多。

## 十三、竞赛拓展

1. 可进一步扩展到 BET 多层吸附、Freundlich 等温式及其适用背景。
2. 在异相催化中，吸附-表面反应-脱附三步常共同决定表观速率，吸附强度存在“过强不利、过弱也不利”的火山型关系。
3. 在纳米材料和多孔材料中，孔径分布、表面官能团和缺陷位点会显著改变吸附选择性。
