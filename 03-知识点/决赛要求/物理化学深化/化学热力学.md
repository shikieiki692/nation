# 化学热力学

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-物理化学深化]]
- 对应考纲条目：[[决赛04]]

## 一、定义

化学热力学研究化学体系在宏观平衡条件下能量、熵和自由能之间的关系，关注“变化能否自发发生”“平衡位置由什么决定”“状态函数之间如何相互联系”。它不直接描述变化有多快，而是回答变化在热力学上是否有驱动力，以及平衡态应满足什么条件。

## 二、考纲对应

本条目对应决赛物理化学中热力学第一、第二、第三定律，焓、熵、Gibbs 自由能，偏摩尔量、化学势以及平衡判据等主干内容。竞赛要求不仅会列公式，更要能用状态函数思维处理复杂过程，并把热力学判据迁移到相平衡、溶液平衡和化学反应平衡中。

## 三、核心原理

1. 热力学从宏观状态量出发描述体系，不追踪单个粒子的具体轨迹，而关注平衡态与过程前后的状态变化。
2. 内能 `U`、焓 `H`、熵 `S`、Helmholtz 自由能 `A`、Gibbs 自由能 `G` 都是核心状态函数，不同势函数适用于不同约束条件。
3. 第一性原理层面上，第一定律给出能量守恒，第二定律给出自发过程的方向判据，第三定律为绝对熵标定提供基准。
4. 在恒温恒压条件下，Gibbs 自由能最常作为化学问题的判据：`ΔG<0` 表示过程可自发，`ΔG=0` 对应平衡。
5. 多组分体系中，化学势是组分层面的自由能描述；相平衡、化学平衡和传质方向都可统一到自由能与化学势框架下。

## 四、关键结论

1. 热力学只告诉我们“能不能发生、平衡在哪里”，通常不直接告诉我们“多久发生完”，这要交给动力学。
2. 状态函数的变化只取决于初末态，与具体路径无关，因此 Hess 定律和各种热力学循环成立。
3. 恒温恒压下用 `ΔG` 判断自发性最方便；恒温恒容下则更常看 Helmholtz 自由能变化。
4. 平衡不是“什么都不变”，而是宏观无净变化、对应合适势函数达到极值或稳定条件。

## 五、常见分类或情形

1. `热化学问题`：围绕反应焓、相变焓、键能估算和 Hess 定律展开。
2. `熵变与自发性问题`：比较体系有序度、粒子分散程度和能量分配趋势。
3. `自由能与平衡问题`：用 `ΔG`、`ΔG^\ominus`、`K` 和反应商判断方向与平衡位置。
4. `多组分与相平衡问题`：引入化学势、偏摩尔量、相律和相图。
5. `非理想体系修正问题`：进一步引入活度、逸度和标准态处理真实体系。

## 六、适用条件与限制

1. 热力学结论通常建立在平衡态或准静态近似上，强非平衡快速过程不能机械套用简单平衡公式。
2. 自发性判据要和约束条件匹配，不能把某个势函数在所有条件下都当作万能判据。
3. “热力学允许”不等于“实验上立刻发生”，若活化能很高，过程仍可动力学上几乎停滞。
4. 对真实溶液、真实气体或电解质体系，若忽略活度、逸度等修正，定量结果可能明显偏离。

## 七、常见比较与易混点

1. `放热`不等于`自发`，还要综合考虑熵变与温度效应。
2. `ΔG<0` 表示在给定条件下有自发趋势，不代表反应会瞬间完全进行。
3. `平衡`不是“反应停止”，而是正逆过程速率相等、宏观量不再净变化。
4. `状态函数`与`过程函数`要分清：`U、H、S、G` 是状态函数，而热 `q`、功 `w` 依赖路径。

## 八、与其他知识点的联系

1. 与 [[化学势与平衡]] 相连，后者是化学热力学在多组分和平衡问题中的核心展开。
2. 与 [[双组分相图]] 相连，相图可看作热力学稳定相区和相平衡条件的图像表达。
3. 与 [[Raoult与Henry定律]] 相连，溶液热力学中的活度与标准态选择都属于其延伸。
4. 与 [[化学动力学深化]] 相对照，前者给方向与平衡，后者给速率与机理。

## 九、典型题型

1. 利用 Hess 定律或标准生成焓求反应焓变。
2. 结合 `ΔH`、`ΔS`、`T` 判断某过程在不同温度下的自发性。
3. 由 `ΔG^\ominus=-RT\ln K` 推断平衡常数大小规律或温度影响。
4. 结合相平衡、活度或化学势讨论真实体系平衡偏移。

## 十、例题

某反应的 `ΔH<0`、`ΔS<0`，判断其在低温和高温下的自发性趋势。

解题思路：

1. 恒温恒压下看 `ΔG=ΔH-TΔS`。
2. 因为 `ΔS<0`，所以 `-TΔS>0`，温度越高，这一不利项越大。
3. 低温时放热项可能占主导，过程更可能自发；高温时熵项惩罚增强，自发性可能减弱甚至消失。

这类题的关键是把焓和熵放进同一个自由能判据中，而不是孤立背“放热就自发”。

## 十一、易错点

1. 把热力学和动力学混成一个问题，用“反应慢”否定“热力学自发”。
2. 看到 `ΔH<0` 就默认一定自发，忽略熵项和温度因素。
3. 在不同约束条件下混用 `ΔG`、`ΔA` 等判据。
4. 只记公式不看标准态和适用条件，导致平衡常数或自由能关系误用。

## 十二、🎯 教学视角

教学上建议始终把化学热力学讲成“判方向、定平衡、做状态函数账”。先用第一定律建立能量守恒，再用第二定律回答“为什么有些过程会自发”，最后把自由能作为化学问题最常用的综合判据收束起来，学生会更容易理解后面的化学势、相图和溶液平衡。

## 十三、竞赛拓展

1. 可进一步引入 Maxwell 关系、Legendre 变换和自然变量，帮助理解不同热力学势为何出现。
2. 在统计热力学框架下，可把熵与配分函数联系起来，从微观上解释宏观判据。
3. 对真实体系，可扩展到活度、逸度、电化学势和非理想混合自由能等更精细内容。
