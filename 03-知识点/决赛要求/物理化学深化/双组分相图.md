# 双组分相图

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-物理化学深化]]
- 对应考纲条目：[[决赛05]]

## 一、定义

双组分相图是用温度、组成、压力等变量描述由两个组分构成体系在不同条件下稳定相区、相界和相变关系的图示工具。它把“哪些相能共存”“各相组成如何变化”“加热或冷却时先后出现什么现象”集中表达出来，是相平衡知识的核心载体。

## 二、考纲对应

本条目对应决赛物理化学中二元体系相平衡、杠杆规则、相区判读及典型相图类型分析。竞赛要求不止是会认图，还要能把相图和自由能、化学势平衡、相律、蒸馏或结晶过程联系起来。

## 三、核心原理

1. 双组分体系的相图建立在相平衡条件之上：共存各相中同一组分的化学势分别相等。
2. 在恒压二元体系中，最常见的是温度-组成相图；横坐标通常表示组成，纵坐标表示温度。
3. 两相区中的任一点不对应单一均一相，而对应两相共存；此时需借助连线端点读取两相组成，再用杠杆规则求各相相对量。
4. 相图中的液相线、固相线、沸点曲线、露点曲线、共晶点、最高/最低恒沸点等，分别对应不同类型的平衡边界或特殊不变点。
5. 因而“读相图”本质上是在回答三件事：体系中有哪些相、每个相的组成是什么、各相数量比例如何。

## 四、关键结论

1. 单相区内体系均一；进入两相区后，整体组成不等于各相组成，必须区分“总组成”和“相组成”。
2. 杠杆规则只适用于两相共存区，且必须在同一温度或压力下沿系连线使用。
3. 共晶点、包晶点、恒沸点等特殊点往往代表自由度最低、过程最敏感的区域，是竞赛高频考点。
4. 相图不是单纯经验图像，而是化学势平衡与相律约束的可视化结果。

## 五、常见分类或情形

1. `完全互溶液-固体系`：液相完全混溶，固相也可形成连续固溶体。
2. `简单共晶体系`：液相互溶而固相互溶有限，最常见于金属或盐类体系的基础竞赛模型。
3. `形成化合物的体系`：中间化合物可能对应相图中的新相区或特殊转变点。
4. `气液平衡相图`：常见于蒸馏问题，重点关注泡点线、露点线和恒沸组成。
5. `液液部分互溶体系`：会出现溶解度曲线、临界溶解温度等特征。

## 六、适用条件与限制

1. 读图前必须先确认相图类型，是固液相图、气液相图还是液液相图，不同图上的线含义并不相同。
2. 杠杆规则不能越相区乱用，也不能在单相区讨论“两相比例”。
3. 相图通常对应特定外压条件；若压力变化显著，原有相图可能不再适用。
4. 实际体系中的过冷、过热、动力学迟滞不属于平衡相图本身内容，不能直接用相图解释所有实验现象。

## 七、常见比较与易混点

1. `整体组成`是样品总比例，`相组成`是共存某一相内部的比例，二者经常被混淆。
2. `共晶点`不是“任意比例同时凝固”，而是特定组成在固定温度下液相同时转化为两种固相。
3. `恒沸混合物`沸腾时液相与气相组成相同，但这并不意味着体系成为纯物质。
4. `相图上的线`表示平衡边界，不是实际升温降温过程中所有路径都必然严格贴线进行。

## 八、与其他知识点的联系

1. 与 [[化学势与平衡]] 相连，相图中的相界本质上来自化学势相等条件。
2. 与 [[Raoult与Henry定律]] 相连，气液相图常从理想或偏离理想的蒸气压关系出发建立。
3. 与 [[化学热力学]] 相连，相图判据背后是自由能最小化与相律约束。
4. 与结晶、蒸馏、材料相变等应用场景相连，竞赛题常将相图与分离过程结合。

## 九、典型题型

1. 判断某一点位于单相区还是两相区，并写出存在的相。
2. 根据系连线读出两相组成，再用杠杆规则求两相比例。
3. 识别共晶点、恒沸点、临界互溶点等特殊位置并解释物理意义。
4. 结合升温或冷却路径分析“先析出什么相、剩余液相组成如何变化”。

## 十、例题

在一个简单共晶相图中，某总组成为 `x_0` 的液态混合物缓慢冷却进入液 + 固 A 两相区。说明此时如何确定固相和液相组成，以及如何求两相相对量。

解题思路：

1. 在给定温度下作水平系连线。
2. 水平线与液相线交点给出液相组成，与固相线或纯固相端对应位置给出固相组成。
3. 用杠杆规则，以总组成点到两端组成点的线段长度比求各相相对量。

核心不是记结论，而是建立“相区判定 → 读端点组成 → 再算比例”的稳定步骤。

## 十一、易错点

1. 把两相区中的横坐标直接当成其中某一相的组成。
2. 杠杆规则分子分母写反，导致比例颠倒。
3. 只会认共晶点名称，但说不清冷却路径上各相先后出现顺序。
4. 把平衡相图与实际快速冷却得到的非平衡组织简单等同。

## 十二、🎯 教学视角

教学上最好先训练“相图读图三问”：有什么相、各相什么组成、各相多少。只要学生形成这套固定流程，之后面对固液、气液、液液不同图型时就不容易慌，也更容易把相图和化学势、自由度这些抽象概念连起来。

## 十三、竞赛拓展

1. 可进一步引入 Gibbs 自由能-组成曲线，用公切线法解释两相共存与相界来源。
2. 真实体系偏离理想时，气液相图可出现最高或最低恒沸点，并改变蒸馏可分离性。
3. 材料化学中还会遇到包晶、偏晶、调幅分解等更复杂二元或多元相图问题。

## 十九、标准例题

在简单共晶相图中，某体系总组成为 `x_0`，在温度 `T` 下位于 `L + A` 两相区。已知同温度系连线两端对应的液相组成为 `x_L = 0.70`，固相 `A` 端可近似看作 `x_A = 0.20`，体系总组成为 `x_0 = 0.50`。求液相与固相 `A` 的相对量之比。

解题思路：

1. 先确认这是两相区，才能使用杠杆规则。
2. 液相所占比例与其“对边”线段成正比，固相所占比例同理。
3. 因而有
   `n_L : n_A = (x_0 - x_A) : (x_L - x_0) = (0.50 - 0.20) : (0.70 - 0.50) = 0.30 : 0.20 = 3 : 2`。

这道题最核心的训练点不是算数，而是把“总组成”“相组成”“相比例”三件事彻底分开。
