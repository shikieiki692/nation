## 三、核心原理

### 3.1 复杂反应动力学

#### 3.1.1 对峙反应（可逆反应）

$$A \underset{k_{-1}}{\stackrel{k_1}{\rightleftharpoons}} B$$

净反应速率：

$$-\frac{\mathrm{d}[A]}{\mathrm{d}t} = k_1[A] - k_{-1}[B]$$

平衡时 $-\frac{\mathrm{d}[A]}{\mathrm{d}t} = 0$，故：

$$K = \frac{k_1}{k_{-1}} = \frac{[B]_{eq}}{[A]_{eq}}$$

**弛豫法测速率常数**：对微扰后的体系，偏离平衡的量 $\Delta[A] = [A] - [A]_{eq}$ 呈指数衰减：

$$\Delta[A] = \Delta[A]_0 \exp(-t/\tau)$$

其中弛豫时间 $\tau = \frac{1}{k_1 + k_{-1}}$（对1-1级对峙反应）。

#### 3.1.2 平行反应

$$A \xrightarrow{k_1} B, \quad A \xrightarrow{k_2} C$$

总消耗速率：

$$-\frac{\mathrm{d}[A]}{\mathrm{d}t} = (k_1 + k_2)[A]$$

产物分布比：

$$\frac{[B]}{[C]} = \frac{k_1}{k_2}$$

**表观活化能**（Arrhenius形式）：

$$k_{obs} = k_1 + k_2, \quad E_{a,obs} = \frac{k_1 E_{a1} + k_2 E_{a2}}{k_1 + k_2}$$

#### 3.1.3 连续反应

$$A \xrightarrow{k_1} B \xrightarrow{k_2} C$$

各组分浓度随时间变化：

$$[A] = [A]_0 e^{-k_1 t}$$

$$[B] = \frac{k_1[A]_0}{k_2 - k_1}(e^{-k_1 t} - e^{-k_2 t})$$

$$[C] = [A]_0 - [A] - [B]$$

**中间产物B的极大值**：当 $k_1 \neq k_2$ 时，$[B]$ 在 $t_{max} = \frac{\ln(k_2/k_1)}{k_2 - k_1}$ 时达到最大。

### 3.1.4 弛豫法（Atkins 17C.2 补充）

**弛豫**（relaxation）指体系受微扰后恢复平衡的过程。弛豫法是研究快速反应动力学的重要实验方法，通过突然改变外部条件（温度、压力、电场等），使体系偏离原平衡，然后追踪浓度趋于新平衡的指数衰减过程。

#### 实验原理

对处于平衡的反应体系施加一个快速微扰（如温度突跃 temperature jump、压力突跃 pressure jump 或电场突跃），平衡位置发生移动，各物种浓度随之调整至新条件下的平衡值。若微扰足够小，偏离平衡的量呈指数衰减。

#### 一级对峙反应的弛豫

对正、逆向均为一级反应的对峙平衡：

$$A \underset{k_{-1}}{\stackrel{k_1}{\rightleftharpoons}} B$$

设 $[A]$ 偏离新平衡浓度的量为 $x$，即 $[A] = [A]_{\mathrm{eq}} + x$，$[B] = [B]_{\mathrm{eq}} - x$。在新的温度下：

$$\frac{\mathrm{d}[A]}{\mathrm{d}t} = -k_1[A] + k_{-1}[B] = -(k_1 + k_{-1})x$$

因 $\mathrm{d}[A]/\mathrm{d}t = \mathrm{d}x/\mathrm{d}t$，解得：

$$x = x_0 e^{-t/\tau}, \quad \frac{1}{\tau} = k_1 + k_{-1}$$

其中 $\tau$ 为**弛豫时间**（relaxation time），$x_0$ 为微扰后初始偏离量。

#### 更一般反应的弛豫时间

对反应 $A \rightleftharpoons B + C$（正向一级、逆向二级），弛豫时间满足：

$$\frac{1}{\tau} = k_{\mathrm{r}} + k_{\mathrm{r}}'([B]_{\mathrm{eq}} + [C]_{\mathrm{eq}})$$

引入平衡常数 $K = k_{\mathrm{r}}/k_{\mathrm{r}}'c^{\ominus}$，可进一步关联速率常数与热力学量。

#### 温度跃迁技术

- **放电法**：使导电样品经受放电，约 $1\ \mu\mathrm{s}$ 内温度突跃 $5\sim10\ \mathrm{K}$
- **脉冲激光法**：高能量脉冲激光可在 $1\ \mathrm{ns}$ 内使水溶液温度跃升 $10\sim30\ \mathrm{K}$

**弛豫法的核心优势**：通过测量弛豫时间 $\tau$，结合平衡常数 $K$，可同时确定正、逆反应的速率常数 $k_1$ 和 $k_{-1}$。

### 3.2 稳态近似（Steady-State Approximation, SSA）

对于活泼中间体 $I$（如自由基、原子等），其浓度很低且变化缓慢，可假设：

$$\frac{\mathrm{d}[I]}{\mathrm{d}t} \approx 0$$

**适用条件**：
- 中间体高活性、低浓度
- 中间体的生成速率与消耗速率近似相等
- 反应进行一段时间后（诱导期之后）

**竞赛应用示例**：

反应 $2NO + O_2 \rightarrow 2NO_2$，实验速率方程为 $v = k[NO]^2[O_2]$。

假设机理：
$$\text{(1)} \quad NO + NO \underset{k_{-1}}{\stackrel{k_1}{\rightleftharpoons}} N_2O_2 \quad \text{(快)}$$
$$\text{(2)} \quad N_2O_2 + O_2 \xrightarrow{k_2} 2NO_2 \quad \text{(慢)}$$

对中间体 $N_2O_2$ 应用稳态近似：

$$\frac{\mathrm{d}[N_2O_2]}{\mathrm{d}t} = k_1[NO]^2 - k_{-1}[N_2O_2] - k_2[N_2O_2][O_2] = 0$$

$$[N_2O_2] = \frac{k_1[NO]^2}{k_{-1} + k_2[O_2]}$$

总速率：

$$v = k_2[N_2O_2][O_2] = \frac{k_1 k_2 [NO]^2 [O_2]}{k_{-1} + k_2[O_2]}$$

若 $k_{-1} \gg k_2[O_2]$（第一步快速平衡），则 $v = \frac{k_1 k_2}{k_{-1}}[NO]^2[O_2] = k_{obs}[NO]^2[O_2]$。

### 3.3 速控步（Rate-Determining Step, RDS）

多步反应中，**最慢的一步**决定总反应速率。速控步之前的步骤可视为快速平衡，之后的步骤不影响总速率。

**与稳态近似的区别**：
- 速控步近似：明确指定某一步为RDS，其前面的步骤达平衡
- 稳态近似：对中间体浓度做数学近似，不预设哪一步是RDS

### 3.4 活化能与反应热的关系

对于基元反应：

$$\Delta_r H_m = E_{a,正} - E_{a,逆}$$

**图示理解**：

```
能量
  ↑
  |      ※ 过渡态
  |     / \
  |    /   \
  |   /     \
  |  /       \
  | /   ΔH   \
  |/__________\____→ 反应坐标
  反应物    产物
```

- 放热反应：$E_{a,正} < E_{a,逆}$，$\Delta H < 0$
- 吸热反应：$E_{a,正} > E_{a,逆}$，$\Delta H > 0$

### 3.5 微观可逆性原理

**原理表述**：基元反应的逆反应也必然是基元反应，且正逆反应经过相同的过渡态（或活化络合物）。

**推论**：
1. 正逆反应的活化能之差等于反应热
2. 正逆反应的频率因子相近（对于简单反应）
3. 平衡常数 $K = k_{正}/k_{逆}$

### 3.6 Lindemann单分子反应机理（Atkins 17F.1 补充）

#### 问题背景

许多气相反应（如环丙烷异构化为丙烯）服从一级动力学：

$$\mathrm{C}_3\mathrm{H}_6(\mathrm{g}) \longrightarrow \mathrm{CH}_3\mathrm{CH}=\mathrm{CH}_2(\mathrm{g}), \quad v = k_{\mathrm{r}}[\mathrm{C}_3\mathrm{H}_6]$$

但分子自身无法凭空获得反应所需能量，必须通过碰撞活化。Lindemann（1921）与Hinshelwood提出了第一个成功的解释。

#### Lindemann-Hinshelwood机理

**步骤1：活化（双分子碰撞）**

$$A + A \underset{k_{-a}}{\stackrel{k_a}{\rightleftharpoons}} A^* + A$$

一个A分子通过与另一个A分子碰撞成为富能激发态 $A^*$：

$$\frac{\mathrm{d}[A^*]}{\mathrm{d}t} = k_a[A]^2$$

**步骤2：去活化（双分子碰撞）**

$$A + A^* \xrightarrow{k_{-a}} A + A$$

富能分子通过与另一分子碰撞失去多余能量：

$$\frac{\mathrm{d}[A^*]}{\mathrm{d}t} = -k_{-a}[A][A^*]$$

**步骤3：单分子衰变**

$$A^* \xrightarrow{k_b} P$$

富能分子通过振动分解或原子重排生成产物：

$$\frac{\mathrm{d}[A^*]}{\mathrm{d}t} = -k_b[A^*]$$

#### 稳态近似推导总速率方程

对活泼中间体 $A^*$ 应用稳态近似：

$$\frac{\mathrm{d}[A^*]}{\mathrm{d}t} = k_a[A]^2 - k_{-a}[A][A^*] - k_b[A^*] = 0$$

解得：

$$[A^*] = \frac{k_a[A]^2}{k_b + k_{-a}[A]}$$

产物生成速率：

$$\frac{\mathrm{d}[P]}{\mathrm{d}t} = k_b[A^*] = \frac{k_a k_b [A]^2}{k_b + k_{-a}[A]}$$

#### 高压极限与低压极限

**高压极限**（$k_{-a}[A] \gg k_b$）：

去活化速率远大于单分子衰变速率，分母中 $k_b$ 可忽略：

$$\frac{\mathrm{d}[P]}{\mathrm{d}t} = \frac{k_a k_b}{k_{-a}}[A] = k_{\mathrm{r}}[A]$$

表现为**一级反应**，有效速率常数 $k_{\mathrm{r}} = k_a k_b / k_{-a}$。

**低压极限**（$k_{-a}[A] \ll k_b$）：

单分子衰变速率远大于去活化速率：

$$\frac{\mathrm{d}[P]}{\mathrm{d}t} = k_a[A]^2$$

表现为**二级反应**，活化步骤成为速率控制步骤。

#### 有效速率常数的倒数形式

将总速率写作 $v = k_{\mathrm{r}}[A]$，则：

$$k_{\mathrm{r}} = \frac{k_a k_b [A]}{k_b + k_{-a}[A]}$$

取倒数得：

$$\frac{1}{k_{\mathrm{r}}} = \frac{k_{-a}}{k_a k_b} + \frac{1}{k_a [A]}$$

**实验检验**：将 $1/k_{\mathrm{r}}$ 对 $1/[A]$ 作图，应得直线。此行为常在低浓度时观察到，高浓度时因机理更复杂而有偏差。

#### 为什么单分子反应需要碰撞活化？

单分子反应并非真正的"单分子"过程。分子要发生反应，必须先获得足够能量以克服活化能垒。在气相中，能量只能通过分子间碰撞传递。因此，虽然决定产物生成的步骤是单分子衰变，但整个机理中必须包含双分子的活化步骤。

### 3.7 聚合反应动力学（Atkins 17F.2 补充）

聚合反应主要分为两类：逐步聚合（stepwise polymerization）和链聚合（chain polymerization），其动力学特征截然不同。

#### 逐步聚合

**特征**：反应混合物中任意两个含官能团的分子（无论单体还是已形成的聚合物链）均可相互反应，聚合不受已形成的链的限制。

**典型反应**：缩聚反应，如尼龙-66、聚酯的形成。每一步消去一个小分子（如 $\mathrm{H}_2\mathrm{O}$）。

**速率方程**：

以含—COOH官能团的物种A表示，无催化剂时缩合为对A的二级反应：

$$\frac{\mathrm{d}[A]}{\mathrm{d}t} = -k_{\mathrm{r}}[A]^2$$

积分得：

$$[A] = \frac{[A]_0}{1 + k_{\mathrm{r}}t[A]_0}$$

**反应程度** $p$（已聚合官能团的分数）：

$$p = \frac{[A]_0 - [A]}{[A]_0} = \frac{k_{\mathrm{r}}t[A]_0}{1 + k_{\mathrm{r}}t[A]_0}$$

**数均聚合度** $\bar{X}_n$（每个聚合物分子中平均单体数）：

$$\bar{X}_n = \frac{[A]_0}{[A]} = \frac{1}{1-p}$$

或用速率常数表示：

$$\bar{X}_n = 1 + k_{\mathrm{r}}t[A]_0$$

**关键结论**：逐步聚合中，平均链长随时间线性增长；单体在反应早期即被大量消耗；要获得高分子量产物，反应程度 $p$ 必须非常接近1。

#### 链聚合

**特征**：单体M进攻一个正在生长的聚合物链，链快速延长。单体随与增长链的交联而被消耗。聚合物的平均摩尔质量在反应初期即已确定，随时间延长主要增加产率而非分子量。

**三个特征步骤**：

**1. 引发（initiation）**：

$$\mathrm{In} \longrightarrow \mathrm{R}\cdot + \mathrm{R}\cdot \quad v_i = k_i[\mathrm{In}]$$

$$\mathrm{M} + \mathrm{R}\cdot \longrightarrow \cdot\mathrm{M}_1$$

**2. 增长（propagation）**：

$$\mathrm{M} + \cdot\mathrm{M}_{n-1} \longrightarrow \cdot\mathrm{M}_n \quad v_p = k_p[\mathrm{M}][\cdot\mathrm{M}]$$

**3. 终止（termination）**（以相互终止为例）：

$$\cdot\mathrm{M}_n + \cdot\mathrm{M}_m \longrightarrow \mathrm{M}_{n+m} \quad v_t = k_t[\cdot\mathrm{M}]^2$$

#### 稳态近似推导聚合速率

对自由基链载体 $\cdot\mathrm{M}$ 应用稳态近似：

$$\frac{\mathrm{d}[\cdot\mathrm{M}]}{\mathrm{d}t} = 2fk_i[\mathrm{In}] - 2k_t[\cdot\mathrm{M}]^2 \approx 0$$

其中 $f$ 为成功引发链的自由基分数。解得：

$$[\cdot\mathrm{M}] = \left(\frac{fk_i}{k_t}\right)^{1/2}[\mathrm{In}]^{1/2}$$

**聚合总速率**（等于链传递速率）：

$$v = k_p[\cdot\mathrm{M}][\mathrm{M}] = k_{\mathrm{r}}[\mathrm{In}]^{1/2}[\mathrm{M}]$$

其中 $k_{\mathrm{r}} = k_p(fk_i/k_t)^{1/2}$。

#### 动力学链长

**定义**：消耗的单体单元数与链引发步骤中生成的自由基数之比，即一个引发自由基所生成的链中分子的平均数目：

$$\nu = \frac{\text{链传递速率}}{\text{自由基生成速率}} = \frac{k_p[\cdot\mathrm{M}][\mathrm{M}]}{2fk_i[\mathrm{In}]}$$

代入稳态浓度表达式：

$$\nu = \frac{k_p[\mathrm{M}]}{2(fk_i k_t)^{1/2}[\mathrm{In}]^{1/2}} = k_{\mathrm{r}}'[\mathrm{M}][\mathrm{In}]^{-1/2}$$

其中 $k_{\mathrm{r}}' = k_p(4fk_i k_t)^{-1/2}$。

#### 链式聚合 vs 逐步聚合的本质区别

| 特征 | 链式聚合 | 逐步聚合 |
|------|---------|---------|
| 活性中心 | 自由基/离子等活性中心 | 官能团反应，无特殊活性中心 |
| 单体消耗 | 单体随链增长逐步消耗，早期即有高分子量产物 | 单体在反应早期大量消耗 |
| 分子量增长 | 平均分子量在反应初期即确定，后期增加产率 | 平均分子量随时间线性增长 |
| 反应步骤 | 引发、增长、终止三步 | 任意两分子间均可反应 |
| 典型产物 | 聚乙烯、聚苯乙烯等加聚物 | 尼龙、聚酯等缩聚物 |

#### 聚合度对聚合物性质的影响

- **低聚合度**（$\bar{X}_n < 100$）：通常为油状液体或低熔点固体，机械强度差
- **中等聚合度**（$\bar{X}_n \sim 10^2\sim10^3$）：具有一定强度和弹性的材料
- **高聚合度**（$\bar{X}_n > 10^3$）：高强度、高韧性的工程塑料或纤维

聚合度直接影响聚合物的玻璃化转变温度、熔点、黏度和机械性能。逐步聚合中，$\bar{X}_n = 1/(1-p)$ 表明：$p$ 从0.99增至0.999，$\bar{X}_n$ 从100增至1000，分子量提高10倍。

## 四、关键结论

1. **对峙反应**：平衡常数等于正逆速率常数之比，$K = k_1/k_{-1}$
2. **平行反应**：产物比例由速率常数比决定，与初始浓度无关
3. **连续反应**：中间产物浓度存在极大值，可通过控制反应时间优化产率
4. **稳态近似**：活泼中间体的浓度近似不随时间变化
5. **活化能关系**：$\Delta H = E_{a,正} - E_{a,逆}$（基元反应）

## 五、常见分类

**复杂反应类型**：

| 类型 | 特征 | 速率方程特点 |
|-----|------|------------|
| 对峙反应 | 正逆反应同时进行 | 净速率 = 正反应速率 - 逆反应速率 |
| 平行反应 | 同一反应物多路径进行 | 总速率 = 各路径速率之和 |
| 连续反应 | 产物继续反应 | 中间体浓度有极大值 |
| 链反应 | 自由基循环再生 | 引发、传递、终止三步 |

## 六、适用条件

| 方法/公式 | 适用条件 | 不适用情况 |
|----------|---------|-----------|
| 稳态近似 | 中间体活泼、浓度低 | 中间体较稳定或积累明显 |
| 速控步近似 | 各步速率差异显著 | 多步速率相近 |
| 快速平衡假设 | 某步远快于其他步 | 各步速率同数量级 |
| $K = k_1/k_{-1}$ | 基元反应组成的对峙反应 | 非基元反应 |

## 七、常见比较

**稳态近似 vs 平衡假设**：

| 特征 | 稳态近似（SSA） | 平衡假设（PEA） |
|-----|----------------|----------------|
| 核心假设 | $\mathrm{d}[I]/\mathrm{d}t = 0$ | 某步快速达到平衡 |
| 适用中间体 | 高活性、低浓度 | 不限 |
| 数学处理 | 解代数方程 | 直接用平衡常数 |
| 典型应用 | 链反应、酶催化 | 速控步前有快速平衡 |

## 十、例题

### 例题1（对峙反应弛豫时间）

某对峙反应 $A \rightleftharpoons B$，$k_1 = 10 \mathrm{s^{-1}}$，$k_{-1} = 4 \mathrm{s^{-1}}$。体系受微扰后偏离平衡，求弛豫时间 $\tau$。

**解**：
对于1-1级对峙反应：

$$\tau = \frac{1}{k_1 + k_{-1}} = \frac{1}{10 + 4} = 0.071 \mathrm{s}$$

即体系将在约0.07秒内恢复到平衡态的 $1/e$。

### 例题2（连续反应优化）

连续反应 $A \xrightarrow{k_1} B \xrightarrow{k_2} C$，$k_1 = 0.1 \mathrm{min^{-1}}$，$k_2 = 0.2 \mathrm{min^{-1}}$。求 $[B]$ 达到最大值的时间及最大产率。

**解**：

$$t_{max} = \frac{\ln(k_2/k_1)}{k_2 - k_1} = \frac{\ln(0.2/0.1)}{0.2 - 0.1} = \frac{\ln 2}{0.1} = 6.93 \mathrm{min}$$

最大 $[B]$：

$$[B]_{max} = [A]_0 \left(\frac{k_1}{k_2}\right)^{\frac{k_2}{k_2-k_1}} = [A]_0 \left(\frac{0.1}{0.2}\right)^{\frac{0.2}{0.1}} = [A]_0 \times 0.5^2 = 0.25[A]_0$$

即最大产率为25%。

### 例题3（弛豫法测速率常数，Atkins 17C.2 补充）

某对峙反应 $A \rightleftharpoons B$ 在298 K下达平衡。对该体系施加温度突跃微扰后，测得弛豫时间 $\tau = 2.0 \times 10^{-4}\ \mathrm{s}$。已知该温度下平衡常数 $K = 2.5$，求正、逆反应的速率常数 $k_1$ 和 $k_{-1}$。

**分析**：
弛豫法的关键是利用弛豫时间 $\tau$ 与速率常数的关系，结合平衡常数 $K = k_1/k_{-1}$，联立求解两个未知量。

**解**：

对于正、逆向均为一级反应的对峙平衡，弛豫时间满足：

$$\frac{1}{\tau} = k_1 + k_{-1}$$

代入数据：

$$k_1 + k_{-1} = \frac{1}{2.0 \times 10^{-4}\ \mathrm{s}} = 5.0 \times 10^{3}\ \mathrm{s^{-1}}$$

又由平衡常数：

$$K = \frac{k_1}{k_{-1}} = 2.5$$

联立两式：

$$k_1 = 2.5 k_{-1}$$

$$2.5 k_{-1} + k_{-1} = 3.5 k_{-1} = 5.0 \times 10^{3}\ \mathrm{s^{-1}}$$

解得：

$$k_{-1} = \frac{5.0 \times 10^{3}}{3.5} = 1.43 \times 10^{3}\ \mathrm{s^{-1}} \approx 1.4 \times 10^{3}\ \mathrm{s^{-1}}$$

$$k_1 = 2.5 \times 1.43 \times 10^{3} = 3.57 \times 10^{3}\ \mathrm{s^{-1}} \approx 3.6 \times 10^{3}\ \mathrm{s^{-1}}$$

**反思**：
1. 弛豫法的核心优势在于：只需测量一个宏观量 $\tau$，结合热力学量 $K$，即可同时确定 $k_1$ 和 $k_{-1}$。这比分别测量正、逆反应速率要高效得多。
2. 温度跃迁技术适用于弛豫时间在 $\mu\mathrm{s}\sim\mathrm{ms}$ 量级的快速反应。对于更快或更慢的反应，需选用其他弛豫技术（如压力跃迁、电场跃迁或闪光光解）。
3. 推导中假设微扰足够小，使偏离平衡的量 $x$ 满足线性速率方程。若微扰过大，高阶项不可忽略，指数衰减规律将失效。

### 例题4（Lindemann机理分析，Atkins 17F.1 补充）

气相异构化反应 $A \rightarrow P$ 遵循Lindemann-Hinshelwood机理。300 K时测得两组数据：
- 当 $[A]_1 = 5.21 \times 10^{-4}\ \mathrm{mol\cdot dm^{-3}}$ 时，$k_{\mathrm{r},1} = 2.50 \times 10^{-4}\ \mathrm{s^{-1}}$
- 当 $[A]_2 = 4.81 \times 10^{-6}\ \mathrm{mol\cdot dm^{-3}}$ 时，$k_{\mathrm{r},2} = 2.10 \times 10^{-5}\ \mathrm{s^{-1}}$

求活化步骤的速率常数 $k_a$。

**分析**：
Lindemann机理的有效速率常数满足倒数关系：

$$\frac{1}{k_{\mathrm{r}}} = \frac{k_{-a}}{k_a k_b} + \frac{1}{k_a [A]}$$

两组数据相减可消去截距项，直接求出 $k_a$。

**解**：

由式（17F.8）：

$$\frac{1}{k_{\mathrm{r},2}} - \frac{1}{k_{\mathrm{r},1}} = \frac{1}{k_a}\left(\frac{1}{[A]_2} - \frac{1}{[A]_1}\right)$$

整理得：

$$k_a = \frac{1/[A]_2 - 1/[A]_1}{1/k_{\mathrm{r},2} - 1/k_{\mathrm{r},1}}$$

计算各项：

$$\frac{1}{[A]_2} = \frac{1}{4.81 \times 10^{-6}} = 2.079 \times 10^{5}\ \mathrm{dm^{3}\cdot mol^{-1}}$$

$$\frac{1}{[A]_1} = \frac{1}{5.21 \times 10^{-4}} = 1.919 \times 10^{3}\ \mathrm{dm^{3}\cdot mol^{-1}}$$

$$\frac{1}{k_{\mathrm{r},2}} = \frac{1}{2.10 \times 10^{-5}} = 4.762 \times 10^{4}\ \mathrm{s}$$

$$\frac{1}{k_{\mathrm{r},1}} = \frac{1}{2.50 \times 10^{-4}} = 4.000 \times 10^{3}\ \mathrm{s}$$

代入：

$$k_a = \frac{2.079 \times 10^{5} - 1.919 \times 10^{3}}{4.762 \times 10^{4} - 4.000 \times 10^{3}} = \frac{2.060 \times 10^{5}}{4.362 \times 10^{4}} = 4.72\ \mathrm{dm^{3}\cdot mol^{-1}\cdot s^{-1}}$$

**反思**：
1. 两组数据分别对应不同压力（浓度）区域：$[A]_1$ 较高，接近高压极限（表现一级）；$[A]_2$ 较低，接近低压极限（表现二级趋势）。
2. $1/k_{\mathrm{r}}$ 对 $1/[A]$ 作图应为直线，斜率为 $1/k_a$，截距为 $k_{-a}/(k_a k_b)$。本题利用差值法避免了截距项的影响。
3. 该机理的预测在低浓度区常与实验有偏差，后续RRKM理论对此进行了修正，考虑了能量在分子内振动模式间的重新分配。

### 例题5（聚合反应动力学，Atkins 17F.2 补充）

某逐步聚合反应中，单体官能团A的起始浓度 $[A]_0 = 4.00 \times 10^{-3}\ \mathrm{mol\cdot dm^{-3}}$，速率常数 $k_{\mathrm{r}} = 1.00\ \mathrm{dm^{3}\cdot mol^{-1}\cdot s^{-1}}$。

（1）计算 $t = 1.5 \times 10^{4}\ \mathrm{s}$ 时的反应程度 $p$ 和数均聚合度 $\bar{X}_n$。

（2）若要在相同时间内使 $\bar{X}_n$ 达到200，起始单体浓度应调整为多少（假设 $k_{\mathrm{r}}$ 不变）？

**分析**：
逐步聚合的核心公式为 $[A] = [A]_0/(1 + k_{\mathrm{r}}t[A]_0)$，由此导出 $p$ 和 $\bar{X}_n$。链式聚合与逐步聚合的本质区别在于：逐步聚合的分子量随时间线性增长，而链式聚合的分子量在反应初期即已确定。

**解**：

**（1）计算 $p$ 和 $\bar{X}_n$**

由式（17F.10）：

$$[A] = \frac{[A]_0}{1 + k_{\mathrm{r}}t[A]_0} = \frac{4.00 \times 10^{-3}}{1 + 1.00 \times 1.5 \times 10^{4} \times 4.00 \times 10^{-3}}$$

$$= \frac{4.00 \times 10^{-3}}{1 + 60} = \frac{4.00 \times 10^{-3}}{61} = 6.56 \times 10^{-5}\ \mathrm{mol\cdot dm^{-3}}$$

反应程度：

$$p = \frac{[A]_0 - [A]}{[A]_0} = 1 - \frac{[A]}{[A]_0} = 1 - \frac{1}{61} = \frac{60}{61} = 0.984$$

数均聚合度：

$$\bar{X}_n = \frac{[A]_0}{[A]} = 61$$

或由 $\bar{X}_n = 1/(1-p) = 1/(1-0.984) = 62.5$（因四舍五入略有差异，精确值为61）。

**（2）求达到 $\bar{X}_n = 200$ 所需的 $[A]_0$**

由 $\bar{X}_n = 1 + k_{\mathrm{r}}t[A]_0$：

$$[A]_0 = \frac{\bar{X}_n - 1}{k_{\mathrm{r}}t} = \frac{200 - 1}{1.00 \times 1.5 \times 10^{4}} = \frac{199}{1.5 \times 10^{4}} = 1.33 \times 10^{-2}\ \mathrm{mol\cdot dm^{-3}}$$

**反思**：
1. 逐步聚合中，$\bar{X}_n$ 随时间线性增长：$\bar{X}_n = 1 + k_{\mathrm{r}}t[A]_0$。要提高分子量，可延长反应时间或提高起始浓度。
2. 反应程度 $p$ 必须非常接近1才能获得高分子量。$p = 0.98$ 时 $\bar{X}_n = 50$；$p = 0.99$ 时 $\bar{X}_n = 100$；$p = 0.999$ 时 $\bar{X}_n = 1000$。这就是缩聚反应需要高真空除去小分子副产物的原因。
3. 与链式聚合对比：链式聚合的聚合度主要取决于动力学链长 $\nu$，而 $\nu$ 与引发剂浓度的平方根成反比。引发剂越少，动力学链越长，聚合物分子量越高，但聚合速率也越慢。

### 例题6（稳态近似推导）

臭氧分解反应 $2O_3 \rightarrow 3O_2$ 的机理为：
$$O_3 \underset{k_{-1}}{\stackrel{k_1}{\rightleftharpoons}} O_2 + O \quad \text{(快)}$$
$$O + O_3 \xrightarrow{k_2} 2O_2 \quad \text{(慢)}$$

用稳态近似推导速率方程。

**解**：
对中间体 $O$ 应用稳态近似：

$$\frac{\mathrm{d}[O]}{\mathrm{d}t} = k_1[O_3] - k_{-1}[O_2][O] - k_2[O][O_3] = 0$$

$$[O] = \frac{k_1[O_3]}{k_{-1}[O_2] + k_2[O_3]}$$

$O_3$ 的消耗速率：

$$-\frac{\mathrm{d}[O_3]}{\mathrm{d}t} = k_1[O_3] - k_{-1}[O_2][O] + k_2[O][O_3]$$

利用稳态条件简化：

$$-\frac{\mathrm{d}[O_3]}{\mathrm{d}t} = 2k_2[O][O_3] = \frac{2k_1 k_2 [O_3]^2}{k_{-1}[O_2] + k_2[O_3]}$$

若 $k_{-1}[O_2] \gg k_2[O_3]$，则：

$$-\frac{\mathrm{d}[O_3]}{\mathrm{d}t} = \frac{2k_1 k_2}{k_{-1}} \frac{[O_3]^2}{[O_2]}$$

## 十一、易错点

1. **稳态近似的适用时机**：反应初期（诱导期）不适用，需等中间体浓度建立稳态。
2. **连续反应中间体极大值**：仅当 $k_1 \neq k_2$ 时公式适用，$k_1 = k_2$ 时需用极限处理。
3. **活化能与反应热关系**：仅对基元反应严格成立，复杂反应的"表观活化能"无直接物理意义。
4. **平行反应表观活化能**：是各路径活化能的加权平均，不是简单算术平均。
5. **对峙反应平衡常数与速率常数**：$K = k_1/k_{-1}$ 仅对基元反应组成的对峙反应成立。

### 12.2 认知误区

- **误区1**：认为稳态近似意味着中间体浓度为零。纠正：浓度不为零，只是变化率为零。
- **误区2**：将表观活化能等同于基元反应活化能。纠正：复杂反应的表观活化能是组合量。
- **误区3**：认为所有反应的逆反应都容易进行。纠正：需满足微观可逆性，但速率可能极慢。
