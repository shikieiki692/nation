# 半衰期

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-物理化学深化]]
- 对应考纲条目：[[决赛07-化学动力学]]
- 关联笔记：[[化学动力学深化]]
- 关联提炼笔记：[[提炼-Atkins物理化学-主题17-19-动力学与表面过程]]

## 一、定义



半衰期 t_{1/2} 是指反应物浓度降至初始浓度一半所需的时间。它是衡量反应速率的重要参数，与反应级数密切相关。

## 二、考纲对应

半衰期是动力学积分形式的直接应用点，竞赛中常与一级、二级反应的积分方程联动考查，也常作为“由实验数据反推级数或速率常数”的入口量。对于决赛层面的要求，不仅要会套公式，更要理解不同级数下半衰期为何呈现不同的初始浓度依赖性。

## 三、核心原理


### 2.1 不同级数反应的半衰期

| 反应级数 | 速率方程 | 半衰期公式 | 特点 |
|---------|---------|-----------|------|
| 零级 | v = k | t_{1/2} = [A]_0/(2k) | 与初始浓度成正比 |
| 一级 | v = k[A] | t_{1/2} = ln 2/k | 与初始浓度无关 |
| 二级 | v = k[A]^2 | t_{1/2} = 1/(k[A]_0) | 与初始浓度成反比 |
| n级 | v = k[A]^n | t_{1/2} ∝ 1/[A]_0^(n-1) | 一般规律 |

**一级反应的放射性衰变**：
N = N_0 e^(-kt), t_{1/2} = ln 2/k = 0.693/k

### 2.2 半衰期的应用

1. **放射性定年**：利用 14C 的 t_{1/2} = 5730 年进行考古定年
2. **药物代谢**：药物的生物半衰期决定给药间隔
3. **反应级数判断**：通过测定不同初始浓度下的 t_{1/2} 变化规律判断反应级数

## 四、关键结论

1. 一级反应的半衰期恒定，与初始浓度无关，因此常被用作识别一级过程的重要特征。
2. 零级与二级反应的半衰期都依赖初始浓度，且依赖方向相反：零级随初始浓度增大而增大，二级随初始浓度增大而减小。
3. 若连续经历多个半衰期，一级反应每经过一个半衰期都再减少一半，因此剩余分数为 `(1/2)^n`。
4. 半衰期只是某一特定转化程度下的时间刻画，不应与“反应完成时间”混为一谈。

## 五、常见分类或情形

1. `零级反应`：常见于表面位点饱和、光照强度受限等体系，浓度随时间线性下降。
2. `一级反应`：如放射性衰变、部分分子分解反应，最典型特征是半衰期恒定。
3. `二级反应`：常见于双分子碰撞控制情形，半衰期对初始浓度敏感。
4. `伪一级反应`：当某一反应物大量过量时，可把高阶过程近似成一级反应，此时实验上也会表现出近似恒定半衰期。

## 六、适用条件与限制

1. 半衰期公式必须建立在已知或可近似确定反应级数的前提下，不能脱离速率方程孤立使用。
2. 对可逆反应、并联反应、连串反应或机理切换过程，简单单一半衰期往往不能完整描述全过程。
3. 当实验信号并不正比于浓度时，需先完成数据换算，不能直接把仪器读数带入半衰期公式。
4. 若体系温度改变，速率常数随之变化，半衰期也会变化，因此不同温度下的数据不能直接混用。

## 七、常见比较与易混点

1. `半衰期`与`平均寿命`不同：一级过程的平均寿命为 `1/k`，而半衰期为 `ln2/k`。
2. `一级反应半衰期恒定`不等于`反应速率恒定`；一级反应速率仍随浓度下降而减小。
3. `转化一半所需时间`与`浓度减少固定值所需时间`不是同一概念，前者是比例尺度，后者更适合零级过程讨论。
4. 对二级反应而言，初始浓度越高半衰期越短，这一点容易与“浓度越高反应物越多所以应该更慢耗尽”的直觉混淆。

## 八、与其他知识点的联系

1. 与 [[反应级数]] 直接相连，半衰期的浓度依赖性是判断级数的重要依据。
2. 与 [[速率方程]] 相连，半衰期公式本质上来自积分速率方程。
3. 与 [[Arrhenius方程]] 相连，温度通过影响 `k` 进一步影响半衰期。
4. 与 [[化学动力学深化]] 相连，竞赛题常将半衰期与机理分析、近似处理、实验设计一起综合考查。

## 九、典型题型


1. **由半衰期求速率常数**：k = ln 2 / t_{1/2}
2. **判断反应级数**：观察 t_{1/2} 与 [A]_0 的关系
3. **放射性衰变计算**：求剩余量或衰变时间

## 十、例题

某反应满足一级动力学，测得半衰期 `t_{1/2}=20 min`。

1. 求速率常数。
2. 求 1 h 后反应物剩余分数。

解题思路：

1. 一级反应 `k=\ln2/t_{1/2}=0.693/20 min^{-1}=3.47\times10^{-2} min^{-1}`。
2. `1 h = 60 min = 3 t_{1/2}`，因此剩余分数为 `(1/2)^3=1/8`。

这类题的关键不在代数运算，而在于先识别“一级反应可直接按半衰期个数处理”。

## 十一、易错点

1. 把所有反应都默认按一级半衰期公式处理。
2. 忘记检查时间单位与速率常数单位是否一致。
3. 将“经过两个半衰期剩余 1/4”误写成“减少 1/4”。
4. 用单个半衰期数据机械下结论，却忽略反应可能并非单一步骤或并非单一级数。

## 十二、🎯 教学视角

教学时可以先让学生比较零级、一级、二级反应在浓度减半时所需时间是否相同，再回到积分方程解释差异来源。这样更容易把“公式记忆”转成“图像化理解”，也能顺手带出如何用实验数据判定反应级数。

## 十三、竞赛拓展

1. 在复杂机理中可讨论表观半衰期，它反映的是给定实验条件下的整体衰减特征，而不是单一步元反应常数。
2. 对连串反应或平行反应，可通过早期近似、稳态近似等方法得到近似半衰期表达式。
3. 在生化和药代动力学中，半衰期常与清除过程、分布过程耦合，呈现多指数衰减行为。
