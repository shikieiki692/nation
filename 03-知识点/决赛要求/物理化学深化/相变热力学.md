# 相变热力学

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-物理化学深化]]
- 对应考纲条目：[[决赛05]]

## 一、定义

相变热力学研究体系在不同相之间转变时的能量、熵、自由能和平衡条件变化。它关注的不只是“相变发生了”，而是“在什么条件下发生”“伴随哪些热力学量变化”“相界如何随温压变化移动”。

## 二、考纲对应

本条目对应决赛物理化学中单组分相平衡、相变焓、Clapeyron/Clausius-Clapeyron 方程和相界分析部分。竞赛要求通常包括：理解相变平衡条件，掌握相变焓与熵变关系，能用相变方程分析相界斜率、蒸气压随温度变化以及特殊相变现象。

## 三、核心原理

1. 相变在平衡时满足两相中同一组分化学势相等，因此相变边界本质上是化学势相等条件随温度和压力变化的轨迹。
2. 一阶相变通常伴随焓、体积和熵的突变，如熔化、汽化、升华；其相变焓满足 `\Delta H = T\Delta S`（在平衡相变温度下）。
3. 相界斜率由 Clapeyron 方程给出：`dp/dT = \Delta H /(T\Delta V)`，它把相变热与体积变化联系起来。
4. 对液-气或固-气相变，若蒸气近似理想且凝聚相体积可忽略，可进一步得到 Clausius-Clapeyron 近似形式，用于分析蒸气压与温度关系。
5. 因而相变热力学的核心任务，是把“相变是否平衡、相界往哪边移动、热量和体积怎么变”统一到一个框架里。

## 四、关键结论

1. 相变温度不是任意常数，而是特定压力下两相化学势相等时的平衡温度。
2. 相变焓越大，通常表示完成该相变所需打破或重组的相互作用越显著。
3. Clapeyron 方程给出相界斜率符号与大小，因此能解释某些物质熔点随压强升高而升高或降低。
4. 蒸气压随温度升高而增大，是气液/固气平衡中最基础的结果之一。

## 五、常见分类或情形

1. `熔化/凝固`：关注固液平衡、熔点和熔化焓。
2. `汽化/液化`：关注蒸气压、沸点和汽化焓。
3. `升华/凝华`：常见于固体直接与蒸气平衡的体系。
4. `异常熔化线情形`：如冰融化时体积减小，因此熔化线斜率可呈特殊符号。
5. `近似蒸气压积分问题`：常通过 Clausius-Clapeyron 方程处理蒸气压与温度的定量变化。

## 六、适用条件与限制

1. Clapeyron 方程是一般相界方程，但使用前需正确判断对应的 `\Delta H` 与 `\Delta V`。
2. Clausius-Clapeyron 积分式依赖额外近似，如蒸气近似理想、相变焓在所讨论温区内近似常数。
3. 若温区较宽或体系偏离理想显著，简单线性积分关系可能只提供近似趋势。
4. 实际快速相变中的过热、过冷和成核障碍不属于平衡相变热力学的直接结论。

## 七、常见比较与易混点

1. `相变热`与`反应热`都可表现为焓变，但前者不改变物质化学组成，后者涉及化学转化。
2. `沸点升高到某温度`与`蒸气压变大`是同一平衡问题的不同表述，不应割裂理解。
3. `相界斜率`由 `\Delta H` 和 `\Delta V` 共同决定，不能只看吸热还是放热。
4. `相变发生`不等于`体系已达平衡相变条件`，实际过程中可能有迟滞现象。

## 九、典型题型

1. 用 Clapeyron 或 Clausius-Clapeyron 方程分析相界斜率或蒸气压变化。
2. 比较不同物质的熔化焓、汽化焓与相变难易程度。
3. 解释为什么某些物质的熔点随压强变化方向与直觉不同。
4. 由给定温压条件判断相变方向或平衡条件。

## 十、例题

为什么冰的熔点在增压时会降低，而多数物质的熔点在增压时会升高？

解题思路：

1. 对固液平衡使用 Clapeyron 方程：`dp/dT = \Delta H /(T\Delta V)`。
2. 熔化通常吸热，因此 `\Delta H > 0`；关键在于体积变化项 `\Delta V = V_l - V_s`。
3. 对冰而言，液态水密度更大，故熔化时体积减小，即 `\Delta V < 0`，因此相界斜率符号异常。
4. 这意味着增压会更有利于体积较小的液相，从而使熔点降低。

关键是别只盯着“熔化吸热”，而要同时看体积变化方向。

## 十一、易错点

1. 把相变热力学问题简单当成记蒸气压公式，而忽略其根源是两相化学势平衡。
2. 机械套用 Clausius-Clapeyron 积分式，却不检查所需近似条件是否成立。
3. 只看 `\Delta H` 不看 `\Delta V`，导致相界斜率判断错误。
4. 将过冷、过热等动力学现象误认为平衡相图本身的结论。

## 十二、🎯 教学视角

教学上适合先从学生熟悉的“沸腾、熔化、升华”现象入手，再不断追问“为什么在这个温压下发生、为什么换条件就变”。这样能把相变从经验现象拉回到“化学势相等 + 焓熵体积变化”的热力学主线。
