# 反应机理与近似

## 一、定义

**反应机理**（Reaction Mechanism）是对化学反应历程的详细描述，即反应物分子经过哪些基元步骤转化为产物。**基元反应**是指一步完成的反应，其速率方程可由化学计量系数直接写出。大多数化学反应是由多个基元反应组成的**非基元反应**（复杂反应）。

决赛阶段要求掌握：基元反应的判断、反应机理的推测方法、稳态近似与平衡近似两种数学处理手段、速控步概念，以及链反应机理的分析。

## 三、核心原理

### 3.1 基元反应与质量作用定律

**基元反应**（Elementary Reaction）：反应物分子通过一次碰撞直接转化为产物的反应。

对于基元反应 $aA + bB \rightarrow cC$，其速率方程直接由计量系数决定：

$$v = k[A]^a[B]^b$$

这就是**质量作用定律**（Law of Mass Action）。基元反应的正逆速率常数之比等于平衡常数：

$$K = \frac{k_{正}}{k_{逆}}$$

**重要区分**：
- 基元反应：速率方程可由计量系数直接写出
- 非基元反应：速率方程必须由实验测定，不能直接由总反应式写出

### 3.2 稳态近似（Steady-State Approximation, SSA）

对于活泼中间体 $I$（如自由基、原子等），其浓度很低且变化缓慢，可假设：

$$\frac{\mathrm{d}[I]}{\mathrm{d}t} \approx 0$$

**适用条件**：
- 中间体高活性、低浓度
- 反应进行一段时间后（诱导期之后）
- 中间体的生成速率与消耗速率近似相等

**竞赛应用示例**：$\mathrm{H}_2 + \mathrm{Br}_2 \rightarrow 2\mathrm{HBr}$

机理：
$$\text{(1)} \quad \mathrm{Br}_2 \underset{k_{-1}}{\stackrel{k_1}{\rightleftharpoons}} 2\mathrm{Br} \quad \text{(快)}$$
$$\text{(2)} \quad \mathrm{Br} + \mathrm{H}_2 \xrightarrow{k_2} \mathrm{HBr} + \mathrm{H} \quad \text{(慢)}$$
$$\text{(3)} \quad \mathrm{H} + \mathrm{Br}_2 \xrightarrow{k_3} \mathrm{HBr} + \mathrm{Br} \quad \text{(快)}$$

对中间体 $\mathrm{Br}$ 和 $\mathrm{H}$ 应用稳态近似：

$$\frac{\mathrm{d}[\mathrm{Br}]}{\mathrm{d}t} = 2k_1[\mathrm{Br}_2] - 2k_{-1}[\mathrm{Br}]^2 - k_2[\mathrm{Br}][\mathrm{H}_2] + k_3[\mathrm{H}][\mathrm{Br}_2] = 0$$

$$\frac{\mathrm{d}[\mathrm{H}]}{\mathrm{d}t} = k_2[\mathrm{Br}][\mathrm{H}_2] - k_3[\mathrm{H}][\mathrm{Br}_2] = 0$$

由第二式得 $k_2[\mathrm{Br}][\mathrm{H}_2] = k_3[\mathrm{H}][\mathrm{Br}_2]$，代入第一式：

$$2k_1[\mathrm{Br}_2] = 2k_{-1}[\mathrm{Br}]^2 \quad \Rightarrow \quad [\mathrm{Br}] = \sqrt{\frac{k_1}{k_{-1}}[\mathrm{Br}_2]}$$

总速率：

$$v = \frac{\mathrm{d}[\mathrm{HBr}]}{\mathrm{d}t} = k_2[\mathrm{Br}][\mathrm{H}_2] + k_3[\mathrm{H}][\mathrm{Br}_2] = \frac{2k_2\sqrt{k_1/k_{-1}}[\mathrm{H}_2][\mathrm{Br}_2]^{1/2}}{1 + k^{\prime}[\mathrm{HBr}]/[\mathrm{Br}_2]}$$

当 $[\mathrm{HBr}]$ 很小时，简化为 $v = k[\mathrm{H}_2][\mathrm{Br}_2]^{1/2}$。

### 3.3 平衡近似（Pre-Equilibrium Approximation, PEA）

当速控步之前存在一个快速平衡时，可用平衡常数表示中间体浓度。

**适用条件**：
- 速控步之前有快速可逆步骤
- 该可逆步骤远快于速控步
- 中间体浓度由平衡关系决定

**竞赛应用示例**：$\mathrm{H}_2 + \mathrm{I}_2 \rightarrow 2\mathrm{HI}$

机理：
$$\text{(1)} \quad \mathrm{I}_2 \underset{k_{-1}}{\stackrel{k_1}{\rightleftharpoons}} 2\mathrm{I} \quad \text{(快平衡)}$$
$$\text{(2)} \quad 2\mathrm{I} + \mathrm{H}_2 \xrightarrow{k_2} 2\mathrm{HI} \quad \text{(慢，RDS)}$$

由步骤①的平衡：$K_1 = \frac{[\mathrm{I}]^2}{[\mathrm{I}_2]} \quad \Rightarrow \quad [\mathrm{I}]^2 = K_1[\mathrm{I}_2]$

总速率由步骤②决定：

$$v = k_2[\mathrm{H}_2][\mathrm{I}]^2 = k_2K_1[\mathrm{H}_2][\mathrm{I}_2] = k[\mathrm{H}_2][\mathrm{I}_2]$$

这与实验测定的二级反应速率方程一致。

### 3.4 速控步（Rate-Determining Step, RDS）

多步反应中，**最慢的一步**决定总反应速率。

**特征**：
- 速控步之前的步骤可视为快速平衡
- 速控步之后的步骤不影响总速率
- 总速率方程由速控步的速率方程决定

**与稳态近似的区别**：

| 特征 | 速控步近似 | 稳态近似 |
|-----|-----------|---------|
| 核心假设 | 某一步为RDS | 中间体浓度不随时间变化 |
| 中间体处理 | 由前平衡决定 | 解代数方程 |
| 适用条件 | 各步速率差异显著 | 中间体活泼、低浓度 |
| 预设性 | 需预设哪一步是RDS | 不预设RDS |

### 3.5 链反应机理

链反应由三个基本步骤组成：

**（1）链引发**（Initiation）：产生自由基或活性粒子
$$\mathrm{Cl}_2 \xrightarrow{h\nu} 2\mathrm{Cl}$$

**（2）链增长**（Propagation）：自由基与反应物作用，生成产物并产生新的自由基
$$\mathrm{Cl} + \mathrm{H}_2 \rightarrow \mathrm{HCl} + \mathrm{H}$$
$$\mathrm{H} + \mathrm{Cl}_2 \rightarrow \mathrm{HCl} + \mathrm{Cl}$$

**（3）链终止**（Termination）：自由基结合使链中断
$$\mathrm{Cl} + \mathrm{Cl} \rightarrow \mathrm{Cl}_2$$
$$\mathrm{H} + \mathrm{H} \rightarrow \mathrm{H}_2$$
$$\mathrm{H} + \mathrm{Cl} \rightarrow \mathrm{HCl}$$

**支链反应**：一个自由基反应产生多个自由基，导致反应速率急剧增加，甚至爆炸。如 $\mathrm{H}_2 + \mathrm{O}_2$ 反应。

### 3.5a 预平衡近似深化（Atkins 17E补充）

预平衡近似（Pre-equilibrium Approximation）是处理复杂反应机理的另一种重要方法，与稳态近似形成互补。当反应机理中存在一个快速可逆步骤，且该步骤远快于后续的慢步骤时，中间体与反应物之间可视为维持热力学平衡。

**典型机理**：

$$\mathrm{A} + \mathrm{B} \underset{k_{-1}}{\stackrel{k_1}{\rightleftharpoons}} \mathrm{I} \xrightarrow{k_2} \mathrm{P}$$

**适用条件**：
- 中间体 $\mathrm{I}$ 返回反应物的速率常数远大于其生成产物的速率常数，即 $k_{-1} \gg k_2$
- 快速可逆步骤与后续慢步骤之间存在明确的速率分离
- 中间体 $\mathrm{I}$ 与反应物 $\mathrm{A}$、$\mathrm{B}$ 确实处于（或非常接近）热力学平衡

**推导过程**：

由于 $\mathrm{A}$、$\mathrm{B}$ 和 $\mathrm{I}$ 处于平衡，平衡常数为

$$K = \frac{[\mathrm{I}]/c^{\ominus}}{[\mathrm{A}][\mathrm{B}]/c^{\ominus 2}} = \frac{k_1}{k_{-1}}$$

因此中间体浓度可表示为

$$[\mathrm{I}] = \frac{k_1}{k_{-1}}[\mathrm{A}][\mathrm{B}]$$

产物 $\mathrm{P}$ 的生成速率由慢步骤决定：

$$\frac{\mathrm{d}[\mathrm{P}]}{\mathrm{d}t} = k_2[\mathrm{I}] = k_2 \frac{k_1}{k_{-1}}[\mathrm{A}][\mathrm{B}] = k_{\mathrm{r}}[\mathrm{A}][\mathrm{B}]$$

其中总速率常数为

$$k_{\mathrm{r}} = \frac{k_1 k_2}{k_{-1}}$$

**与稳态近似的对比**：

对同一机理应用稳态近似，设 $\mathrm{d}[\mathrm{I}]/\mathrm{d}t = 0$：

$$\frac{\mathrm{d}[\mathrm{I}]}{\mathrm{d}t} = k_1[\mathrm{A}][\mathrm{B}] - k_{-1}[\mathrm{I}] - k_2[\mathrm{I}] = 0$$

解得

$$[\mathrm{I}] = \frac{k_1[\mathrm{A}][\mathrm{B}]}{k_{-1} + k_2}$$

代入速率方程得

$$\frac{\mathrm{d}[\mathrm{P}]}{\mathrm{d}t} = \frac{k_1 k_2}{k_{-1} + k_2}[\mathrm{A}][\mathrm{B}]$$

当 $k_{-1} \gg k_2$ 时，上式还原为预平衡近似的结果。因此：

| 特征 | 预平衡近似 | 稳态近似 |
|-----|-----------|---------|
| 核心假设 | 中间体与反应物处于热力学平衡 | 中间体浓度不随时间变化 |
| 适用条件 | $k_{-1} \gg k_2$ | 中间体活泼、低浓度 |
| 中间体浓度 | 由平衡常数直接决定 | 解代数方程求得 |
| 预设性 | 需明确快速平衡步骤 | 不预设具体步骤性质 |

**何时选用预平衡而非稳态？**
- 当 $k_{-1} \gg k_2$ 时，中间体确实处于热力学平衡，预平衡近似更物理、更简洁
- 当 $k_{-1}$ 与 $k_2$ 同数量级时，必须使用稳态近似
- 预平衡近似数学上更简便，但适用条件更严格

**负活化能现象**：

预平衡机理中，总速率常数 $k_{\mathrm{r}} = k_1 k_2 / k_{-1}$。若各基元步骤服从阿仑尼乌斯公式，则总反应的有效活化能为

$$E_a = E_{a,1} + E_{a,2} - E_{a,-1}$$

若 $E_{a,-1} > E_{a,1} + E_{a,2}$，则 $E_a < 0$，反应速率随温度升高而减小。这提醒我们在预测多步反应的温度效应时必须谨慎。

**例题（预平衡近似推导与稳态近似对比）**

反应 $2\mathrm{NO} + \mathrm{O}_2 \rightarrow 2\mathrm{NO}_2$ 的实验速率方程为 $v = k[\mathrm{NO}]^2[\mathrm{O}_2]$。提出如下机理：

$$\text{(1)} \quad \mathrm{NO} + \mathrm{NO} \underset{k_{-1}}{\stackrel{k_1}{\rightleftharpoons}} \mathrm{N}_2\mathrm{O}_2 \quad \text{（快平衡）}$$
$$\text{(2)} \quad \mathrm{N}_2\mathrm{O}_2 + \mathrm{O}_2 \xrightarrow{k_2} 2\mathrm{NO}_2 \quad \text{（慢，RDS）}$$

**(a) 用预平衡近似推导速率方程。**

**解**：由步骤①的平衡：

$$K_1 = \frac{[\mathrm{N}_2\mathrm{O}_2]}{[\mathrm{NO}]^2} \quad \Rightarrow \quad [\mathrm{N}_2\mathrm{O}_2] = K_1[\mathrm{NO}]^2 = \frac{k_1}{k_{-1}}[\mathrm{NO}]^2$$

总速率由步骤②决定：

$$v = \frac{1}{2}\frac{\mathrm{d}[\mathrm{NO}_2]}{\mathrm{d}t} = k_2[\mathrm{N}_2\mathrm{O}_2][\mathrm{O}_2] = k_2 \frac{k_1}{k_{-1}}[\mathrm{NO}]^2[\mathrm{O}_2] = k[\mathrm{NO}]^2[\mathrm{O}_2]$$

其中 $k = k_1 k_2 / k_{-1}$，与实验速率方程一致。

**(b) 对中间体 $\mathrm{N}_2\mathrm{O}_2$ 应用稳态近似，并讨论两种近似的关系。**

**解**：稳态近似下：

$$\frac{\mathrm{d}[\mathrm{N}_2\mathrm{O}_2]}{\mathrm{d}t} = k_1[\mathrm{NO}]^2 - k_{-1}[\mathrm{N}_2\mathrm{O}_2] - k_2[\mathrm{N}_2\mathrm{O}_2][\mathrm{O}_2] = 0$$

解得

$$[\mathrm{N}_2\mathrm{O}_2] = \frac{k_1[\mathrm{NO}]^2}{k_{-1} + k_2[\mathrm{O}_2]}$$

总速率：

$$v = k_2[\mathrm{N}_2\mathrm{O}_2][\mathrm{O}_2] = \frac{k_1 k_2[\mathrm{NO}]^2[\mathrm{O}_2]}{k_{-1} + k_2[\mathrm{O}_2]}$$

当 $[\mathrm{O}_2]$ 浓度很高，使得 $k_2[\mathrm{O}_2] \gg k_{-1}$ 时，上式简化为 $v = k_1[\mathrm{NO}]^2$，此时步骤①为速率控制步骤。当 $k_{-1} \gg k_2[\mathrm{O}_2]$ 时，还原为预平衡近似的结果 $v = (k_1 k_2/k_{-1})[\mathrm{NO}]^2[\mathrm{O}_2]$。

**反思**：预平衡近似是稳态近似在 $k_{-1} \gg k_2$ 条件下的特例。预平衡假设中间体确实处于热力学平衡，而稳态仅假设其浓度不变——后者是更弱的条件，因此适用范围更广。竞赛中若题目明确给出"快平衡"，优先使用预平衡近似；若仅给出中间体活泼，则使用稳态近似。

### 3.5b 动力学控制与热力学控制（Atkins 17E补充）

当反应物可以通过不同路径生成多种产物时，产物分布可能由**动力学**或**热力学**因素决定，这是有机反应和复杂反应体系中极为重要的概念。

**定义**：

- **动力学控制**（Kinetic Control）：产物比例由各路径的相对反应速率决定，即由速率常数之比控制
- **热力学控制**（Thermodynamic Control）：产物比例由各产物的相对热力学稳定性（标准吉布斯能）决定，即由平衡常数之比控制

**竞争反应模型**：

假定由以下竞争反应可生成两种产物 $\mathrm{P}_1$ 和 $\mathrm{P}_2$：

$$\mathrm{A} + \mathrm{B} \xrightarrow{k_1} \mathrm{P}_1 \quad v(\mathrm{P}_1) = k_1[\mathrm{A}][\mathrm{B}]$$
$$\mathrm{A} + \mathrm{B} \xrightarrow{k_2} \mathrm{P}_2 \quad v(\mathrm{P}_2) = k_2[\mathrm{A}][\mathrm{B}]$$

在反应达到平衡之前，已生成的两种产物的相对比例为

$$\frac{[\mathrm{P}_2]}{[\mathrm{P}_1]} = \frac{k_2}{k_1}$$

该比值代表**动力学控制**下的产物比例。若反应允许达到平衡，则产物比例由热力学控制，浓度比率由标准反应吉布斯能控制。

**判据与条件**：

| 控制类型 | 有利条件 | 产物特征 |
|---------|---------|---------|
| 动力学控制 | 低温、短时间、不可逆条件 | 生成速率快，活化能较低 |
| 热力学控制 | 高温、长时间、可逆条件 | 热力学更稳定，能量更低 |

**典型实例分析**：

**实例1：1,3-丁二烯与HBr的加成**

低温下主要发生1,2-加成（动力学产物），因为碳正离子中间体在C-2位进攻的活化能较低；高温下主要发生1,4-加成（热力学产物），因为生成的1,4-产物具有更稳定的共轭双键结构，热力学上更有利。

**实例2：单取代苯的硝化反应**

根据初始取代基的导向能力，可以得到邻位、间位和对位取代产物。在反应初期（未达到平衡前），产物比例由各位置硝化的相对速率（动力学控制）决定；若反应条件允许异构化达到平衡，则比例由各异构体的相对稳定性（热力学控制）决定。

**势能图解释**：

在反应势能剖面图中：
- **动力学产物**对应较低活化能 $E_a$ 的反应路径，反应速率快，但产物能量可能较高
- **热力学产物**对应较低能量状态的产物，但生成路径的活化能可能较高

```
势能
  ^
  |    反应物          动力学产物        热力学产物
  |      *                *                 *
  |     / \              /                   \\n  |    /   \    E_a,小  /      E_a,大         \\n  |   /     \\    /     /        /              \\n  |  *       \\--*-----*--------*---------------*
  |  A+B      过渡态1   P_1      过渡态2         P_2
  |
  +-------------------------------------------> 反应坐标
```

**可逆反应中产物比例随时间的变化**：

对于可逆竞争反应，产物比例随时间演化：
- 初始阶段：动力学控制主导，$[\mathrm{P}_2]/[\mathrm{P}_1] \approx k_2/k_1$
- 长时间后：若产物间可相互转化，系统趋向平衡，$[\mathrm{P}_2]/[\mathrm{P}_1] \approx K_{eq}$

**例题（动力学控制与热力学控制分析）**

某化合物 $\mathrm{A}$ 在酸性条件下可异构化为 $\mathrm{B}$ 或 $\mathrm{C}$，反应如下：

$$\mathrm{A} \underset{k_{-1}}{\stackrel{k_1}{\rightleftharpoons}} \mathrm{B} \quad \mathrm{A} \underset{k_{-2}}{\stackrel{k_2}{\rightleftharpoons}} \mathrm{C}$$

已知：$k_1 = 2.0 \times 10^{-2} \ \mathrm{s}^{-1}$，$k_2 = 1.0 \times 10^{-2} \ \mathrm{s}^{-1}$，$k_{-1} = 5.0 \times 10^{-4} \ \mathrm{s}^{-1}$，$k_{-2} = 1.0 \times 10^{-4} \ \mathrm{s}^{-1}$。

**(a) 反应初期（动力学控制），$\mathrm{B}$ 与 $\mathrm{C}$ 的浓度比是多少？**

**解**：反应初期，逆反应可忽略，产物生成速率之比等于速率常数之比：

$$\frac{[\mathrm{B}]}{[\mathrm{C}]} = \frac{k_1}{k_2} = \frac{2.0 \times 10^{-2}}{1.0 \times 10^{-2}} = 2.0$$

即动力学控制下 $\mathrm{B}$ 是主要产物，占产物总量的 $2/3$。

**(b) 反应达到平衡后（热力学控制），$\mathrm{B}$ 与 $\mathrm{C}$ 的浓度比是多少？**

**解**：平衡时，各反应的平衡常数为

$$K_1 = \frac{k_1}{k_{-1}} = \frac{2.0 \times 10^{-2}}{5.0 \times 10^{-4}} = 40$$
$$K_2 = \frac{k_2}{k_{-2}} = \frac{1.0 \times 10^{-2}}{1.0 \times 10^{-4}} = 100$$

由于两个反应共享反应物 $\mathrm{A}$，平衡时：

$$K_1 = \frac{[\mathrm{B}]_{eq}}{[\mathrm{A}]_{eq}}, \quad K_2 = \frac{[\mathrm{C}]_{eq}}{[\mathrm{A}]_{eq}}$$

因此

$$\frac{[\mathrm{B}]_{eq}}{[\mathrm{C}]_{eq}} = \frac{K_1}{K_2} = \frac{40}{100} = 0.4$$

即热力学控制下 $\mathrm{C}$ 是主要产物。

**(c) 解释为何动力学控制与热力学控制给出不同的主要产物。**

**解**：动力学控制下，$\mathrm{B}$ 的生成速率快（$k_1 > k_2$），因此初期 $\mathrm{B}$ 占优。但 $\mathrm{B}$ 的热力学稳定性较差（$K_1 = 40 < K_2 = 100$），其逆反应速率相对较快。长时间后，系统趋向热力学平衡，更稳定的 $\mathrm{C}$ 成为主要产物。这体现了"速率快≠产物稳定"的核心思想。

**反思**：在合成化学中，若需要动力学产物，应采用低温、短时间、不可逆条件（如使用非平衡溶剂或快速淬灭）；若需要热力学产物，则应采用高温、长时间或加入催化剂促进平衡建立。理解这两种控制模式对优化反应条件、提高目标产物选择性至关重要。

### 3.6 催化剂对反应机理的影响

催化剂通过改变反应历程、降低活化能来加快反应速率。

**均相催化**：催化剂与反应物同相
- 例：$\mathrm{I}_2$ 催化 $\mathrm{CH}_3\mathrm{CHO}$ 分解，活化能从 $190 \rightarrow 136 \mathrm{kJ \cdot mol^{-1}}$

**非均相催化**：催化剂与反应物不同相
- 例：合成氨用铁催化剂，反应物为气态，催化剂为固态
- 通过吸附作用削弱反应物化学键，降低活化能

**酶催化**：生物催化剂，具有极高的专一性和效率
- 蔗糖水解：酸催化 $E_a = 107 \mathrm{kJ \cdot mol^{-1}}$，酶催化 $E_a = 36 \mathrm{kJ \cdot mol^{-1}}$

## 四、关键结论

1. **基元反应**：一步完成，速率方程由计量系数直接决定
2. **稳态近似**：活泼中间体 $\mathrm{d}[I]/\mathrm{d}t \approx 0$，适用于高活性中间体
3. **平衡近似**：速控步前有快速平衡，中间体浓度由平衡常数决定
4. **速控步**：最慢步骤决定总速率，其前后步骤分别视为快速平衡和快速反应
5. **链反应**：引发 $\rightarrow$ 增长 $\rightarrow$ 终止三步循环
6. **催化剂**：改变反应历程，降低活化能，不改变平衡位置

## 五、常见分类

**反应机理类型**：

| 类型 | 特征 | 典型例子 |
|-----|------|---------|
| 连串反应 | 产物继续反应 | $A \rightarrow B \rightarrow C$ |
| 平行反应 | 同一反应物多路径 | $A \rightarrow B$，$A \rightarrow C$ |
| 对峙反应 | 正逆反应同时进行 | $A \rightleftharpoons B$ |
| 链反应 | 自由基循环再生 | $\mathrm{H}_2 + \mathrm{Cl}_2 \rightarrow 2\mathrm{HCl}$ |
| 催化反应 | 催化剂参与又再生 | 合成氨、酶催化 |

## 六、适用条件

| 近似方法 | 适用条件 | 不适用情况 |
|----------|---------|-----------|
| 稳态近似 | 中间体活泼、浓度低、寿命短 | 中间体较稳定或积累明显 |
| 平衡近似 | 速控步前有快速平衡 | 各步速率相近 |
| 速控步近似 | 各步速率差异显著 | 多步速率同数量级 |
| 快速平衡假设 | 某步远快于其他步 | 各步速率同数量级 |

## 七、常见比较

**稳态近似 vs 平衡近似**：

| 特征 | 稳态近似（SSA） | 平衡近似（PEA） |
|-----|----------------|----------------|
| 核心假设 | $\mathrm{d}[I]/\mathrm{d}t = 0$ | 某步快速达到平衡 |
| 中间体浓度 | 解代数方程求得 | 由平衡常数直接表示 |
| 适用中间体 | 高活性、低浓度 | 不限 |
| 典型应用 | 链反应、自由基反应 | 速控步前有快速平衡 |
| 数学复杂度 | 较高（解方程组） | 较低（直接代入） |

## 九、典型题型

1. **推导题**：利用稳态近似或平衡近似推导复杂反应的速率方程
2. **分析题**：由实验速率方程推断可能的反应机理

4. **链反应题**：推导链反应的速率方程，分析链长

## 十、例题

### 例题1（平衡近似推导）

反应 $2\mathrm{NO} + \mathrm{O}_2 \rightarrow 2\mathrm{NO}_2$ 的实验速率方程为 $v = k[\mathrm{NO}]^2[\mathrm{O}_2]$。提出如下机理：

$$\text{(1)} \quad \mathrm{NO} + \mathrm{NO} \underset{k_{-1}}{\stackrel{k_1}{\rightleftharpoons}} \mathrm{N}_2\mathrm{O}_2 \quad \text{(快平衡)}$$
$$\text{(2)} \quad \mathrm{N}_2\mathrm{O}_2 + \mathrm{O}_2 \xrightarrow{k_2} 2\mathrm{NO}_2 \quad \text{(慢)}$$

用平衡近似推导速率方程。

**解**：
由步骤①的平衡：$K_1 = \frac{[\mathrm{N}_2\mathrm{O}_2]}{[\mathrm{NO}]^2} \quad \Rightarrow \quad [\mathrm{N}_2\mathrm{O}_2] = K_1[\mathrm{NO}]^2$

总速率由步骤②决定：
$$v = k_2[\mathrm{N}_2\mathrm{O}_2][\mathrm{O}_2] = k_2K_1[\mathrm{NO}]^2[\mathrm{O}_2] = k[\mathrm{NO}]^2[\mathrm{O}_2]$$

其中 $k = k_2K_1$，与实验速率方程一致。

### 例题2（稳态近似推导）

臭氧分解反应 $2\mathrm{O}_3 \rightarrow 3\mathrm{O}_2$ 的机理为：
$$\text{(1)} \quad \mathrm{O}_3 \underset{k_{-1}}{\stackrel{k_1}{\rightleftharpoons}} \mathrm{O}_2 + \mathrm{O} \quad \text{(快)}$$
$$\text{(2)} \quad \mathrm{O} + \mathrm{O}_3 \xrightarrow{k_2} 2\mathrm{O}_2 \quad \text{(慢)}$$

用稳态近似推导速率方程。

**解**：
对中间体 $\mathrm{O}$ 应用稳态近似：
$$\frac{\mathrm{d}[\mathrm{O}]}{\mathrm{d}t} = k_1[\mathrm{O}_3] - k_{-1}[\mathrm{O}_2][\mathrm{O}] - k_2[\mathrm{O}][\mathrm{O}_3] = 0$$

$$[\mathrm{O}] = \frac{k_1[\mathrm{O}_3]}{k_{-1}[\mathrm{O}_2] + k_2[\mathrm{O}_3]}$$

$\mathrm{O}_3$ 的消耗速率：
$$-\frac{\mathrm{d}[\mathrm{O}_3]}{\mathrm{d}t} = k_1[\mathrm{O}_3] - k_{-1}[\mathrm{O}_2][\mathrm{O}] + k_2[\mathrm{O}][\mathrm{O}_3]$$

利用稳态条件简化：
$$-\frac{\mathrm{d}[\mathrm{O}_3]}{\mathrm{d}t} = 2k_2[\mathrm{O}][\mathrm{O}_3] = \frac{2k_1k_2[\mathrm{O}_3]^2}{k_{-1}[\mathrm{O}_2] + k_2[\mathrm{O}_3]}$$

若 $k_{-1}[\mathrm{O}_2] \gg k_2[\mathrm{O}_3]$：
$$-\frac{\mathrm{d}[\mathrm{O}_3]}{\mathrm{d}t} = \frac{2k_1k_2}{k_{-1}} \frac{[\mathrm{O}_3]^2}{[\mathrm{O}_2]}$$

### 例题3（机理推断）

实验测定反应 $\mathrm{S}_2\mathrm{O}_8^{2-} + 3\mathrm{I}^- \rightarrow 2\mathrm{SO}_4^{2-} + \mathrm{I}_3^-$ 为二级反应，速率方程为 $v = k[\mathrm{S}_2\mathrm{O}_8^{2-}][\mathrm{I}^-]$。推断反应机理。

**解**：
总反应计量系数为1:3，但实验速率方程为1:1，说明不是基元反应。提出如下机理：

$$\text{(1)} \quad \mathrm{S}_2\mathrm{O}_8^{2-} + \mathrm{I}^- \xrightarrow{k_1} \mathrm{S}_2\mathrm{O}_8\mathrm{I}^{3-} \quad \text{(慢，RDS)}$$
$$\text{(2)} \quad \mathrm{S}_2\mathrm{O}_8\mathrm{I}^{3-} + \mathrm{I}^- \xrightarrow{k_2} 2\mathrm{SO}_4^{2-} + \mathrm{I}_2 \quad \text{(快)}$$
$$\text{(3)} \quad \mathrm{I}_2 + \mathrm{I}^- \xrightarrow{k_3} \mathrm{I}_3^- \quad \text{(快)}$$

步骤①为速控步，故 $v = k_1[\mathrm{S}_2\mathrm{O}_8^{2-}][\mathrm{I}^-]$，与实验一致。

## 十一、易错点

1. **混淆基元反应与非基元反应**：非基元反应的速率方程不能由计量系数直接写出
2. **稳态近似的适用时机**：反应初期（诱导期）不适用
3. **平衡近似的适用条件**：必须有明确的快速平衡步骤和速控步
4. **中间体与过渡态的区别**：中间体有一定寿命和浓度，过渡态是能量极大点
5. **催化剂不改变平衡**：催化剂同等程度加快正逆反应，不改变平衡常数
6. **链反应中的稳态处理**：链载体（自由基）适用稳态近似，但需注意引发和终止步骤

### 12.2 认知误区

- **误区1**：认为所有反应的速率方程都可由计量系数写出。纠正：只有基元反应可以。
- **误区2**：认为稳态近似意味着中间体浓度为零。纠正：浓度不为零，只是变化率为零。
- **误区3**：混淆催化剂与引发剂。纠正：催化剂在反应中再生，引发剂被消耗。

$$\text{(1)} \quad \mathrm{N}_2\mathrm{O}_5 \underset{k_{-1}}{\stackrel{k_1}{\rightleftharpoons}} \mathrm{NO}_2 + \mathrm{NO}_3 \quad \text{(快平衡)}$$
$$\text{(2)} \quad \mathrm{NO}_2 + \mathrm{NO}_3 \xrightarrow{k_2} \mathrm{NO} + \mathrm{O}_2 + \mathrm{NO}_2 \quad \text{(慢)}$$
