# 迁移数

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-物理化学深化]]
- 对应考纲条目：[[决赛10-电化学]]

## 一、定义

迁移数是指在电解质溶液中，某一种离子所承担的总电流分数，常记为 `t_+` 或 `t_-`。它反映了不同离子在导电过程中对总电流贡献的相对大小，因此是连接“总电导”和“单个离子迁移行为”的关键量。

## 二、考纲对应

本条目对应决赛物理化学中电解质溶液输运性质部分。竞赛要求通常包括：理解迁移数的物理意义，掌握 `t_+ + t_- = 1` 等基本关系，能将迁移数与离子迁移率、电导和电解过程中的浓度变化联系起来分析。

## 三、核心原理

1. 在电场作用下，阳离子和阴离子都参与导电，但各自迁移速度和携带电荷能力不同，因此承担的电流比例也不同。
2. 对二元电解质溶液，正、负离子的迁移数满足 `t_+ + t_- = 1`。
3. 迁移数与离子迁移率或离子摩尔电导有关，本质上反映“谁对总电流贡献更大”。
4. 迁移数并不表示某离子迁移了多少物质的量，而是指其对电流贡献的比例。
5. 在电解和浓差变化问题中，迁移数常决定电极附近溶液组成如何随通电过程发生改变。

## 四、关键结论

1. 迁移数越大，说明该离子在总电流承担中所占比例越高。
2. 迁移数是相对贡献量，不是单独的迁移速度本身。
3. 迁移数与离子电荷数、迁移率和浓度共同相关，但在给定体系中常作为实验可测或题设已知量使用。
4. 通电时电极附近浓度变化常不能只看电极反应，还要结合迁移数分析离子补给或流失。

## 五、常见分类或情形

1. `二元强电解质`：最常见的基础迁移数分析对象。
2. `阳离子迁移数`：记作 `t_+`，表示阳离子承担电流比例。
3. `阴离子迁移数`：记作 `t_-`，表示阴离子承担电流比例。
4. `极限稀释情形`：迁移数可由离子极限摩尔电导近似求得。
5. `电解分析情形`：常结合 Hittorf 方法、电极附近浓度变化或电解产物量讨论。

## 六、适用条件与限制

1. 迁移数是对某一体系、某一浓度和某一温度条件下的相对输运行为描述，不一定是普适常数。
2. 在多离子复杂体系中，迁移数分析比简单二元体系更复杂，不能机械套基础公式。
3. 浓度较高时离子间相互作用增强，迁移数可能随条件明显变化。
4. 用迁移数分析电解过程时，必须同时考虑电极反应化学计量和电流分担。

## 七、常见比较与易混点

1. `迁移数`不是`迁移率`，前者是电流分担比例，后者更接近单离子的运动能力参数。
2. `t_+` 大不一定意味着阳离子“数量更多”，也可能是其迁移能力更强。
3. `t_+ + t_- = 1` 只表示电流分担完毕，不表示两种离子迁移物质的量一定相等。
4. 不能只根据电极反应方向判断溶液组成变化，还要看离子迁移补偿过程。

## 八、与其他知识点的联系

1. 与 [[电导]] 直接相连，迁移数描述不同离子对总电导的分担。
2. 与离子迁移率、极限摩尔电导和 Kohlrausch 定律密切相关。
3. 与电解过程中的浓度变化分析相连，是 Hittorf 思想的核心。
4. 与 [[电极过程热力学]] 形成对照：迁移数更偏输运分担，不是平衡电势量。

## 九、典型题型

1. 已知离子极限摩尔电导，求阳离子或阴离子迁移数。
2. 根据电解前后某区浓度变化反推迁移数。
3. 比较不同离子在导电过程中的相对贡献。
4. 结合电极反应与迁移数判断溶液局部组成变化。

## 十、例题

某二元电解质溶液中阳离子迁移数 `t_+ = 0.40`，求阴离子迁移数，并说明哪一类离子承担了更多电流。

解题思路：

1. 二元体系满足 `t_+ + t_- = 1`。
2. 代入 `t_+ = 0.40`，得 `t_- = 0.60`。
3. 因此阴离子承担的电流比例更大。

关键是理解迁移数是电流分担比例，而不是物质量多少。

## 十一、易错点

1. 把迁移数和迁移率混为一谈。
2. 误以为迁移数大的离子其物质的量一定更多。
3. 忘记二元体系中正负离子迁移数之和为 1。
4. 在电解浓度变化题里只算电极反应，不算迁移补偿。

## 十二、🎯 教学视角

教学上可以把迁移数讲成“总电流这张账，正负离子各付了多少”。学生一旦接受这种分账视角，很多电解浓度变化题就会从混乱的离子流动叙事变成清晰的守恒题。

## 十三、竞赛拓展

## 十九、标准例题

某二元 1:1 电解质在无限稀释时，阳离子极限摩尔电导为 `40 S·cm^2·mol^{-1}`，阴离子极限摩尔电导为 `60 S·cm^2·mol^{-1}`。求 `t_+` 与 `t_-`。

解题思路：

1. 无限稀释下总极限摩尔电导
   `\Lambda_m^\infty = \lambda_+^\infty + \lambda_-^\infty = 40 + 60 = 100`。
2. 因而
   `t_+ = \lambda_+^\infty / \Lambda_m^\infty = 40/100 = 0.40`，
   `t_- = \lambda_-^\infty / \Lambda_m^\infty = 60/100 = 0.60`。

这道题用来把“迁移数是电流分担比例”讲清最合适，也正好和本页定义完全对齐。
