# 电导

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-物理化学深化]]
- 对应考纲条目：[[决赛10-电化学]]

## 一、定义

电导是体系传导电流能力的量度，常记为 `G`，是电阻 `R` 的倒数。对电解质溶液而言，电导本质上反映溶液中离子在电场作用下定向迁移所形成的导电能力，因此既取决于离子数目，也取决于离子的运动能力。

## 二、考纲对应

本条目对应决赛物理化学中电解质溶液导电、比电导、摩尔电导及浓度依赖行为部分。竞赛要求通常包括：区分不同电导量的定义，理解强弱电解质电导随浓度变化的规律，并能把电导与离子迁移、迁移数和电解质离解程度联系起来分析。

## 三、核心原理

1. 溶液导电依赖自由移动离子的存在，电流由正、负离子在电场中反向迁移共同承担。
2. 电导 `G` 与导体几何尺寸有关，因此更常引入比电导（电导率）`\kappa` 来表征单位长度和单位截面积归一化后的导电能力。
3. 对电解质溶液，还常用摩尔电导 `\Lambda_m = \kappa / c`（按单位统一写法）来比较单位物质的量电解质所贡献的导电能力。
4. 随浓度降低，强电解质的摩尔电导通常增大，这是因为离子间相互作用减弱、迁移更自由；弱电解质则还叠加离解程度增加的效应。
5. 因而电导问题既是“离子多少”的问题，也是“离子能跑多快”的问题。

## 四、关键结论

1. 电导率高不一定只因为浓度高，也可能因为离子迁移率大。
2. 摩尔电导是比较不同电解质导电贡献的更常用量，因为它部分剥离了浓度和几何因素。
3. 强电解质与弱电解质在稀释时摩尔电导变化趋势都上升，但成因并不相同。
4. 无限稀释摩尔电导是电解质和离子导电性质的重要基准量。

## 五、常见分类或情形

1. `金属导电`：靠电子迁移，不属于电解质溶液电导机制。
2. `电解质溶液导电`：靠正负离子迁移。
3. `强电解质`：离解程度高，摩尔电导随浓度降低平滑上升。
4. `弱电解质`：稀释时因离解度显著增加，摩尔电导上升更明显。
5. `无限稀释极限`：离子间相互作用最弱，常用于定义极限摩尔电导。

## 六、适用条件与限制

1. 不同教材对比电导、摩尔电导的单位写法可能略有差异，计算时必须先统一单位体系。
2. 用摩尔电导比较体系时，要确认浓度定义和体积单位一致。
3. 电导测量结果常受温度显著影响，比较数据时必须说明温度条件。
4. 强电解质和弱电解质的电导浓度关系不能用同一简单经验式机械处理。

## 七、常见比较与易混点

1. `电导 G`、`电导率 \kappa`、`摩尔电导 \Lambda_m` 是三个不同层级的量，不能混用。
2. `浓度更高`不一定意味着`摩尔电导更大`，因为高浓度下离子间相互作用反而可能压低单摩尔贡献。
3. `强电解质导电强`不等于在任何浓度下其摩尔电导都一定比弱电解质大很多，需结合具体离子迁移率分析。
4. 不能把电子导电和离子导电的机制混为一谈。

## 八、与其他知识点的联系

1. 与 [[迁移数]] 直接相连，电流由各离子分担的比例可由迁移数描述。
2. 与 [[活度与活度系数]] 相连，浓溶液中离子相互作用会同时影响热力学行为和输运行为。
3. 与电解质离解平衡、弱电解质电离度分析密切相关。
4. 与 [[电极过程热力学]] 形成对照：前者更偏输运，后者更偏平衡驱动力。

## 九、典型题型

1. 已知电阻或电导池参数，求电导率或摩尔电导。
2. 比较强弱电解质摩尔电导随浓度变化的差异。
3. 由无限稀释摩尔电导推断电离度或电解质性质。
4. 结合迁移数分析不同离子对总电流的贡献。

## 十、例题

为什么强电解质溶液在稀释时摩尔电导通常会增大？

解题思路：

1. 强电解质本身已高度离解，因此稀释带来的“新增离子数”效应不显著。
2. 但稀释后离子间静电相互作用减弱，离子迁移阻碍减少。
3. 因而单位物质的量电解质所贡献的导电能力增强，摩尔电导上升。

关键是区分“离解度变化”与“迁移自由度变化”两种不同来源。

## 十一、易错点

1. 把电导率和摩尔电导当成同一个量。
2. 只看浓度不看迁移率，就武断判断导电能力。
3. 不统一单位，导致 `\kappa`、`\Lambda_m` 计算错误。
4. 把强电解质和弱电解质随稀释的电导变化原因混为一谈。

## 十二、🎯 教学视角

教学上适合先把问题拆成“有多少载流离子”和“每个离子跑得快不快”两问。学生一旦接受这个框架，后面看强弱电解质差异、迁移数和极限摩尔电导时就不会只背结论。

## 十九、标准例题

某电导池池常数为 `1.20 cm^{-1}`，测得某溶液电阻 `R = 250 \Omega`。求该溶液的电导 `G` 与电导率 `\kappa`。

解题思路：

1. 电导为 `G = 1/R = 1/250 = 4.0 × 10^{-3} S`。
2. 由 `\kappa = G ×` 池常数，得
   `\kappa = 4.0 × 10^{-3} × 1.20 = 4.8 × 10^{-3} S·cm^{-1}`。

这道题的核心是先把 `G` 和 `\kappa` 分层，再谈摩尔电导，避免三个量从一开始就混在一起。
