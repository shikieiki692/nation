# 热力学第一定律深化

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-物理化学深化]]
- 对应考纲条目：[[决赛04]]

## 一、定义

热力学第一定律是能量守恒在热力学体系中的表达，指出体系内能的变化等于体系吸收的热量与体系所受功的代数和。在化学问题中，它回答的是“能量怎么记账”，而不是“过程是否自发”。

## 二、考纲对应

本条目对应决赛物理化学中内能、焓、热、功及热化学基础部分。竞赛要求通常包括：掌握第一定律数学表达，区分状态函数和过程函数，处理恒压/恒容热效应、体积功和焓变，并能把第一定律作为后续热力学势分析的能量基础。

## 三、核心原理

1. 对封闭体系，第一定律常写为 `\Delta U = q + w`，符号约定下 `q` 是体系吸收的热，`w` 是外界对体系所做的功。
2. 内能 `U` 是状态函数，只由体系状态决定；热 `q` 和功 `w` 则依赖具体过程路径。
3. 对只含体积功的可逆过程，`w = -\int p_{ext} dV`（按常见化学符号体系，膨胀时体系对外做功，`w` 常取负）。
4. 焓定义为 `H = U + pV`，因此在恒压且仅有体积功时，体系吸热量常与焓变直接对应：`q_p = \Delta H`。
5. 第一定律不区分过程能否自发，它只要求任何变化都必须满足总能量守恒。

## 四、关键结论

1. 热和功是能量传递方式，内能和焓是体系状态量，二者层级不同。
2. 恒容条件下更常直接联系内能变化，恒压条件下更常直接联系焓变化。
3. 反应热、相变热和混合热等量，本质上都可看作特定条件下的能量收支结果。
4. 第一定律只能告诉我们能量是否守恒，不能单独判断过程方向。

## 五、常见分类或情形

1. `恒容过程`：如刚性容器中的反应，常直接用 `q_V = \Delta U`。
2. `恒压过程`：如敞口或恒压装置中的过程，常用 `q_p = \Delta H`。
3. `绝热过程`：`q = 0`，体系能量变化完全来自做功。
4. `循环过程`：状态函数总变化为零，但热与功一般都不为零。
5. `热化学问题`：常结合 Hess 定律、标准生成焓和燃烧焓处理。

## 六、适用条件与限制

1. 使用 `q_p = \Delta H` 时要确认过程确为恒压且仅考虑体积功。
2. 不同教材对功的符号约定可能不同，解题时必须先统一约定再计算。
3. 第一定律适用于宏观能量守恒，但单靠它无法判断变化是否能自发发生。
4. 对开放体系、非体积功显著体系或流动体系，需要使用更完整的能量平衡形式。

## 七、常见比较与易混点

1. `内能`与`焓`不同：焓多了一个 `pV` 项，适用于恒压热效应表达更自然。
2. `热`不是体系“拥有”的量，而是过程中的能量交换方式。
3. `吸热/放热`不等于`内能一定增/减同样多`，因为过程还可能伴随做功。
4. `第一定律成立`不代表过程一定会发生，能量守恒与自发性是两层问题。

## 八、与其他知识点的联系

1. 与 [[化学热力学]] 相连，第一定律是全部热力学势分析的能量基础。
2. 与 [[热力学第二定律深化]] 相连，前者管守恒，后者管方向。
3. 与 [[相变热力学]] 相连，相变焓和潜热计算直接建立在能量平衡上。
4. 与热化学、Hess 定律和标准反应焓计算密切相关。

## 九、典型题型

1. 已知热量和功，求体系内能变化。
2. 比较恒压与恒容条件下反应热、内能变化和焓变化的关系。
3. 结合气体膨胀/压缩过程计算体积功。
4. 使用 Hess 定律或标准生成焓求反应焓变。

## 十、例题

某体系吸收热量 `50 kJ`，同时对外做功 `20 kJ`，求体系内能变化。

解题思路：

1. 按常见化学符号约定，体系对外做功则 `w = -20 kJ`。
2. 吸热 `q = +50 kJ`。
3. 由第一定律 `\Delta U = q + w = 50 - 20 = 30 kJ`。

关键是先把热和功的正负号约定统一，再代入公式。

## 十一、易错点

1. 混淆“体系对外做功”和“外界对体系做功”的符号。
2. 把状态函数和过程函数混为一谈。
3. 机械记 `q_p = \Delta H`，却忽略其适用条件。
4. 用第一定律去判断过程是否自发。

## 十二、🎯 教学视角

教学上适合把第一定律讲成“热力学记账法”。先让学生稳定掌握“体系吸热、做功、内能变化”三者的收支关系，再引入焓这一更适合恒压化学过程的势函数，后续接第二定律和自由能时会顺很多。

## 十三、竞赛拓展

1. 可进一步引入非体积功、开放体系焓流和流动过程能量平衡。
2. 在统计热力学层面，可把内能与配分函数联系起来。
3. 在高阶热化学题中，可将第一定律与 Kirchhoff 方程、热容修正联合使用。
