# 平衡近似

- 总览：[[决赛要求-总览]]
- 所属模块：[[决赛要求-物理化学深化]]
- 对应考纲条目：[[决赛08-反应动力学深化]]

---

## 一、定义

**平衡近似（Pre-equilibrium Approximation）**：假设反应机理中存在一个快速达到平衡的基元步骤，其后的慢步骤为速率控制步骤。利用平衡常数表达式消去中间体浓度。

---

## 二、适用条件

1. 机理中存在快速可逆步骤
2. 可逆步骤之后的基元反应很慢（速率控制步骤）
3. 中间体在快速步骤中维持平衡浓度

---

## 三、与稳态近似的比较

| 方法 | 核心假设 | 适用场景 |
|:---|:---|:---|
| **稳态近似** | $d[I]/dt = 0$ | 中间体活泼、浓度低 |
| **平衡近似** | 快速步骤达平衡 | 存在快速可逆步骤 + 慢步骤 |

---

## 四、核心原理深化

### 4.1 预平衡近似的严格推导

根据 [[反应机理与近似]] 中 Atkins 主题 17E 补充内容：

对于机理：

$$\mathrm{A} + \mathrm{B} \underset{k_{-1}}{\stackrel{k_1}{\rightleftharpoons}} \mathrm{I} \xrightarrow{k_2} \mathrm{P}$$

当中间体 $\mathrm{I}$ 返回反应物的速率远大于其生成产物的速率（$k_{-1} \gg k_2$）时，快速可逆步骤与后续慢步骤之间存在明确的速率分离，中间体与反应物之间可视为维持热力学平衡。

平衡常数：

$$K = \frac{[\mathrm{I}]}{[\mathrm{A}][\mathrm{B}]} = \frac{k_1}{k_{-1}}$$

产物生成速率由慢步骤决定：

$$\frac{\mathrm{d}[\mathrm{P}]}{\mathrm{d}t} = k_2[\mathrm{I}] = \frac{k_1 k_2}{k_{-1}}[\mathrm{A}][\mathrm{B}] = k_{\mathrm{r}}[\mathrm{A}][\mathrm{B}]$$

### 4.2 与稳态近似的数学关系

对同一机理应用稳态近似：

$$[\mathrm{I}] = \frac{k_1[\mathrm{A}][\mathrm{B}]}{k_{-1} + k_2}$$

当 $k_{-1} \gg k_2$ 时，上式还原为预平衡近似的结果。因此**预平衡近似是稳态近似在 $k_{-1} \gg k_2$ 条件下的特例**。

### 4.3 负活化能现象

预平衡机理中，总速率常数 $k_{\mathrm{r}} = k_1 k_2 / k_{-1}$。若各基元步骤服从 Arrhenius 公式，则总反应的有效活化能为：

$$E_a = E_{a,1} + E_{a,2} - E_{a,-1}$$

若 $E_{a,-1} > E_{a,1} + E_{a,2}$，则 $E_a < 0$，反应速率随温度升高而减小。

### 4.4 何时选用预平衡而非稳态？

| 条件 | 推荐方法 |
|:---|:---|
| $k_{-1} \gg k_2$，且题目明确给出"快平衡" | **预平衡近似**（更物理、更简洁） |
| $k_{-1}$ 与 $k_2$ 同数量级 | **稳态近似**（适用范围更广） |
| 仅知中间体活泼，无速率常数信息 | **稳态近似** |

## 五、竞赛常见考点

1. **判断使用哪种近似**：根据机理特点选择稳态近似或平衡近似
2. **推导速率方程**：利用平衡常数 $K = k_1/k_{-1}$ 消去中间体
3. **表观速率常数与平衡常数的关系**
4. **与稳态近似的对比推导**：同一机理分别用两种近似处理，讨论其等价条件

## 八、与其他知识点的联系

1. 与 [[稳态近似]] 直接相连，预平衡近似可看作稳态近似在特定速率分离条件下的特例。
2. 与 [[反应机理与近似]] 相连，它是复杂机理处理中最核心的两种近似方法之一。
3. 与 [[基元反应]] 相连，预平衡近似建立在明确的多步基元机理基础上。
4. 与 [[速率控制步骤]] 相连，预平衡近似通常默认其后的慢步骤为速控步。

## 九、典型题型

1. 已知“快平衡 + 慢步骤”机理，推导总速率方程。
2. 比较预平衡近似与稳态近似对同一机理的结果差异。
3. 判断题目所给信息更适合用预平衡还是稳态近似。
4. 结合 Arrhenius 参数讨论表观活化能为何可能出现异常符号。

## 十、例题

对机理 `A + B <=> I (快平衡)`，`I -> P (慢)`，推导产物生成速率。

解题思路：

1. 由前一步快平衡得 `K = [I]/([A][B]) = k_1/k_-1`。
2. 故中间体浓度 `[I] = (k_1/k_-1)[A][B]`。
3. 慢步骤决定总速率：`v = k_2[I] = (k_1k_2/k_-1)[A][B]`。

关键是先确认“前一步真的可视为平衡”，再用平衡常数消去中间体。

## 十一、易错点

1. 看到有中间体就直接套预平衡，而不核查是否存在明确的快平衡步骤。
2. 把“快平衡”与“稳态”混成同一个假设。
3. 用预平衡法时漏写后续慢步骤才是真正控制总速率的环节。
4. 把平衡常数表达和速率常数关系写反。

## 十二、🎯 教学视角

教学时可以把预平衡近似讲成“先把很快的前戏当成已经协调好，再看真正慢的一步怎么往前走”。学生一旦抓住这个图像，和稳态近似的边界也会更容易分清。

## 十三、竞赛拓展

1. 可进一步引入吸附-表面反应机理中 Langmuir-Hinshelwood 型前平衡处理。
2. 在光化学和酶催化机理中，也常出现“先形成复合物/激发态，再进入慢步骤”的预平衡结构。
3. 若继续深入，可讨论何时必须回到完整微分方程而不能依赖任何近似。
