# 多相催化

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-物理化学深化]]
- 对应考纲条目：[[决赛07]]

## 一、定义

多相催化是指催化剂与反应物处于不同相时发生的催化过程，最常见的是固体催化剂促进气相或液相反应。其核心特征在于反应往往发生在催化剂表面或界面上，因此吸附、表面扩散、表面反应和脱附成为理解机理的关键步骤。

## 二、考纲对应

本条目对应决赛物理化学中表面过程与复杂反应动力学部分。竞赛要求通常包括：理解多相催化的一般步骤，认识吸附与表面覆盖度的重要性，能结合速控步或 Langmuir 型近似分析表观速率，并将多相催化与均相催化、酶催化加以比较。

## 三、核心原理

1. 多相催化常经历“外扩散 → 吸附 → 表面反应 → 脱附 → 内扩散/外扩散”中的若干关键步骤。
2. 催化剂通过提供新的反应路径和较低活化能的表面中间体，使总反应速率提高，但并不改变平衡位置。
3. 反应物在表面的吸附强弱十分关键：吸附过弱不易活化，吸附过强又会导致位点被占据、产物难脱附。
4. 表面位点有限，因此表观速率常与覆盖度相关，低覆盖和高覆盖区的速率规律可能不同。
5. 因而多相催化问题的主线，通常是“哪些步骤在控制表面位点占用与表面反应通量”。

## 四、关键结论

1. 多相催化的真正反应场所是界面，不是简单把催化剂“混进去”就结束。
2. 催化剂改变反应历程和速率，但不改变平衡常数。
3. 吸附是多相催化的核心前提，速率常受表面覆盖度和位点竞争影响。
4. 多相催化常可出现速率饱和、反应级数变化或最优覆盖度等现象。

## 五、常见分类或情形

1. `Langmuir-Hinshelwood 机理`：两个反应物先吸附后在表面反应。
2. `Eley-Rideal 机理`：一个反应物吸附，另一个直接从气相/液相与之反应。
3. `吸附控制`：吸附或脱附成为瓶颈。
4. `表面反应控制`：真正成键/断键步骤最慢。
5. `扩散控制`：外扩散或孔内扩散限制总表观速率。

## 六、适用条件与限制

1. 多相催化题中常需区分“本征表面动力学”与“传质限制”，两者不能混淆。
2. 简化机理常默认表面位点均一、吸附行为理想，真实催化剂往往更复杂。
3. 若表面覆盖度高、位点不均一或存在副反应，简单 Langmuir 型表达式可能只保留趋势意义。
4. 实际工业催化中温度梯度、孔结构和失活问题也会显著影响表观速率。

## 七、常见比较与易混点

1. `多相催化`与`均相催化`不同，前者强调界面位点和表面覆盖，后者在同一相中进行。
2. `催化剂提高速率`不等于`改变平衡产率极限`。
3. `吸附越强越好`是错误直觉，过强吸附同样会抑制催化循环。
4. 不能把扩散限制下的表观速率规律误当作本征反应级数。

## 八、与其他知识点的联系

1. 与 [[吸附]]、[[Langmuir吸附等温式]] 直接相连，表面覆盖度是多相催化分析的基础。
2. 与 [[速率控制步骤]] 相连，多相催化常要判断吸附、表面反应或脱附哪一步是瓶颈。
3. 与 [[分子间相互作用深化]] 相连，吸附和表面活化本质上依赖界面相互作用。
4. 与 [[酶催化]] 形成对照：二者都通过中间体稳定化降活化能，但一个靠表面位点，一个靠活性中心。

## 九、典型题型

1. 根据机理判断属于 Langmuir-Hinshelwood 还是 Eley-Rideal。
2. 分析不同覆盖度下速率式为何变化。
3. 判断吸附、表面反应、脱附或扩散哪一步更可能控速。
4. 解释为什么催化剂表面积、孔结构或中毒会影响表观速率。

## 十、例题

为什么某些多相催化反应中，随着反应物压强继续升高，反应速率不再近似正比增加，甚至趋于平台？

解题思路：

1. 在低压区，位点尚未接近饱和，更多反应物吸附会明显提高覆盖度和速率。
2. 当表面位点接近被占满后，新增反应物难以继续有效增加反应位点占用。
3. 此时速率对压强的依赖减弱，表观上出现饱和趋势。

关键是把“位点有限”作为多相催化区别于简单均相动力学的核心特征。

## 十一、易错点

1. 忽略吸附步骤，直接把多相催化当普通均相反应处理。
2. 认为催化剂只要更多表面积就一定无限增速，忽略传质和覆盖度限制。
3. 把扩散控制误判为化学反应本身控速。
4. 认为催化剂改变平衡位置。

## 十二、🎯 教学视角

教学上适合把多相催化讲成“先把人带到工位上，再在工位上完成反应，再把人送走”。学生一旦接受“位点有限、上岗有序”的表面图像，就会更容易理解覆盖度、饱和和控速步骤。

## 十三、竞赛拓展

1. 可进一步引入火山型关系，解释“吸附既不能太弱也不能太强”的最佳催化区间。
2. 在工业催化中，可延展到毒化、烧结、载体效应和孔扩散问题。
3. 若继续深化，可连接表面科学中的 d-band 理论和原位表征思路。
