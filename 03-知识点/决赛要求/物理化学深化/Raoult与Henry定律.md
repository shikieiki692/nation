# Raoult 与 Henry 定律

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-物理化学深化]]
- 对应考纲条目：[[决赛05-稀溶液的通性]]

## 一、定义



**Raoult定律**和**Henry定律**是描述溶液中组分蒸气压与浓度关系的两个基本定律。Raoult定律适用于理想溶液或稀溶液中的溶剂，Henry定律适用于稀溶液中的溶质（尤其是挥发性溶质）。两者共同构成了溶液热力学的理论基础。

决赛阶段要求掌握：两个定律的数学表述、适用条件、微观解释，以及实际溶液对理想行为的偏离（正偏差、负偏差）。

## 三、核心原理



### 3.1 Raoult定律

**定律表述**：在一定温度下，稀溶液中溶剂的蒸气压等于纯溶剂的蒸气压乘以溶剂的摩尔分数。

$$p_1 = x_1 p_1^*$$

其中：
- $p_1$：溶液中溶剂的蒸气压
- $p_1^*$：纯溶剂的蒸气压
- $x_1$：溶剂的摩尔分数

**等价形式**：

$$\frac{p_1^* - p_1}{p_1^*} = x_2 \quad \text{或} \quad \Delta p = p_1^* x_2$$

**适用条件**：
- 理想溶液（全浓度范围）
- 稀溶液中的溶剂
- 溶质非挥发性时，溶液总蒸气压 $p = p_1$

**微观解释**：溶质分子的存在减少了单位表面上溶剂分子的数目，从而使溶剂的气化速率降低，蒸气压下降。

### 3.2 Henry定律

**定律表述**：在一定温度下，稀溶液中挥发性溶质的蒸气压与其在溶液中的摩尔分数（或浓度）成正比。

$$p_2 = K_{H,x} x_2$$

其中 $K_{H,x}$ 为Henry常数，与温度、溶剂、溶质本性有关。

**其他形式**：

$$p_2 = K_{H,c} c_2 \quad \text{或} \quad p_2 = K_{H,m} m_2$$

注意：不同浓度单位对应的Henry常数数值不同。

**适用条件**：
- 稀溶液中的溶质
- 溶质在气相和溶液中分子状态相同
- 气体溶解度较低时

**Raoult定律与Henry定律的比较**：

| 特征 | Raoult定律 | Henry定律 |
|-----|-----------|-----------|
| 适用对象 | 溶剂（大量组分） | 溶质（少量组分） |
| 比例常数 | $p_1^*$（纯溶剂蒸气压） | $K_H$（Henry常数） |
| 适用范围 | 理想溶液全浓度范围 | 稀溶液 |
| 物理意义 | 溶剂分子间作用力占主导 | 溶质-溶剂相互作用占主导 |

### 3.3 理想溶液

**定义**：在全浓度范围内都遵守Raoult定律的溶液。

**特征**：
- $\Delta_{mix}H = 0$（混合焓为零）
- $\Delta_{mix}V = 0$（混合体积为零）
- 各组分的蒸气压在全浓度范围符合Raoult定律

**典型例子**：苯-甲苯、甲醇-乙醇、正己烷-正庚烷等结构相似的物质。

**理想溶液的蒸气压-组成图**：

```
p
↑
p_A* |      /\  气相线
     |     /  \
p_B* |    /    \____
     |   / 液相线
     |  /
     +----------------→ x_B
     0              1
```

### 3.4 实际溶液的偏差

#### 3.4.1 正偏差（$p > p_{Raoult}$）

- 实际蒸气压高于Raoult定律计算值
- 原因：A-B分子间作用力弱于A-A和B-B
- 特征：$\Delta_{mix}H > 0$，混合吸热
- 例子：乙醇-水、丙酮-二硫化碳

**具有最低恒沸点的体系**：正偏差大到一定程度时，$p$-$x$图上出现极大值，对应最低恒沸点。

#### 3.4.2 负偏差（$p < p_{Raoult}$）

- 实际蒸气压低于Raoult定律计算值
- 原因：A-B分子间作用力强于A-A和B-B
- 特征：$\Delta_{mix}H < 0$，混合放热
- 例子：氯仿-丙酮、硝酸-水

**具有最高恒沸点的体系**：负偏差大到一定程度时，$p$-$x$图上出现极小值，对应最高恒沸点。

#### 3.4.3 偏差图示

```
正偏差：            负偏差：
  p↑                 p↑
   |  /\               |    __
   | /  \ 气相线       |   /  \ 气相线
   |/    \____         |  /    \____
   | 液相线            | / 液相线
   |/                  |/
   +------→ x          +------→ x
```

### 3.5 化学势表达式

**理想溶液中各组分**：

$$\mu_i = \mu_i^*(T,p) + RT\ln x_i$$

**稀溶液中**：
- 溶剂：$\mu_1 = \mu_1^* + RT\ln x_1$（Raoult定律）
- 溶质：$\mu_2 = \mu_2^\ominus + RT\ln x_2$（Henry定律）

其中溶质的标准态是假想的纯溶质状态（$x_2 = 1$时仍符合Henry定律的状态）。

### 3.6 相似相溶原理

**表述**：结构相似的物质容易相互溶解。

**解释**：
- 极性溶剂溶解极性溶质（如水溶解乙醇）
- 非极性溶剂溶解非极性溶质（如苯溶解碘）
- 溶解过程涉及分子间作用力的匹配

**应用**：
- 萃取：利用物质在不同溶剂中溶解度的差异
- 重结晶：利用溶解度随温度的变化
- 选择适当的溶剂进行化学反应

## 四、关键结论



1. **Raoult定律**：$p_1 = x_1 p_1^*$，适用于理想溶液或稀溶液的溶剂
2. **Henry定律**：$p_2 = K_H x_2$，适用于稀溶液的溶质
3. **理想溶液**：$\Delta_{mix}H = 0$，$\Delta_{mix}V = 0$，全浓度范围遵守Raoult定律
4. **正偏差**：A-B作用力弱，$\Delta_{mix}H > 0$，可能出现最低恒沸点
5. **负偏差**：A-B作用力强，$\Delta_{mix}H < 0$，可能出现最高恒沸点

## 五、常见分类或情形



- **理想溶液情形**：溶剂与近理想组分常更接近 Raoult 定律
- **稀溶液情形**：溶质在无限稀释附近常更接近 Henry 定律
- **正偏差/负偏差情形**：真实溶液因相互作用差异偏离理想线
- **挥发性双组分气液平衡情形**：两条定律共同决定总蒸气压与组成关系

## 六、适用条件与限制



- Raoult 定律更适用于作为“多数组分”的溶剂或彼此作用相近的近理想混合物
- Henry 定律通常只在溶质无限稀释极限附近可靠，浓度升高后偏差可能迅速增大
- 常数的定义方式与单位体系很多，解题时必须先看教材采用的是哪一种 Henry 常数
- 真实体系若有缔合、解离、氢键或强极性相互作用，偏差往往不能用简单理想模型解释

## 七、常见比较与易混点



- **Raoult 定律 vs Henry 定律**：前者常以纯物质蒸气压为参照，后者常以无限稀释线性比例为参照
- **溶剂 vs 溶质视角**：同一体系里，溶剂更常用 Raoult，稀溶质更常用 Henry
- **理想溶液 vs 理想稀溶液**：二者不是同一个概念，后者允许溶质 obey Henry 而溶剂 obey Raoult
- **正偏差/负偏差**：不能只看蒸气压大小，还要回到 A-A、B-B、A-B 相互作用强弱比较

## 八、与其他知识点的联系




## 十、例题



### 例题1（Raoult定律应用）

已知20℃时水的饱和蒸气压为 $2.34 \mathrm{kPa}$，将 $17.1 \mathrm{g}$ 蔗糖（$\mathrm{C_{12}H_{22}O_{11}}$）溶于 $100 \mathrm{g}$ 水。计算溶液的蒸气压。

**解**：
蔗糖的摩尔质量 $M = 342 \mathrm{g \cdot mol^{-1}}$

$$n_{蔗糖} = \frac{17.1}{342} = 0.0500 \mathrm{mol}$$

$$n_{水} = \frac{100}{18.0} = 5.56 \mathrm{mol}$$

$$x_{水} = \frac{5.56}{5.56 + 0.0500} = 0.991$$

$$p = p_{水}^* x_{水} = 2.34 \times 0.991 = 2.32 \mathrm{kPa}$$

### 例题2（Henry定律应用）

20℃时，氧气在水中的Henry常数 $K_{H,c} = 1.3 \times 10^{-3} \mathrm{mol \cdot dm^{-3} \cdot kPa^{-1}}$。计算常压（$p_{O_2} = 21 \mathrm{kPa}$）下氧气在水中的溶解度。

**解**：
由Henry定律：

$$c_{O_2} = K_{H,c} \cdot p_{O_2} = 1.3 \times 10^{-3} \times 21 = 2.73 \times 10^{-2} \mathrm{mol \cdot dm^{-3}}$$

### 例题3（理想溶液蒸气压）

苯和甲苯在90℃时的蒸气压分别为 $136 \mathrm{kPa}$ 和 $54 \mathrm{kPa}$。求含 $0.5$ 摩尔分数苯的理想溶液在90℃时的总蒸气压和气相组成。

**解**：
$$p_{苯} = x_{苯} p_{苯}^* = 0.5 \times 136 = 68 \mathrm{kPa}$$

$$p_{甲苯} = x_{甲苯} p_{甲苯}^* = 0.5 \times 54 = 27 \mathrm{kPa}$$

$$p_{总} = 68 + 27 = 95 \mathrm{kPa}$$

气相组成：

$$y_{苯} = \frac{p_{苯}}{p_{总}} = \frac{68}{95} = 0.716$$

$$y_{甲苯} = \frac{27}{95} = 0.284$$

## 十一、易错点



1. **Raoult定律的适用对象**：适用于溶剂（大量组分），不是溶质
2. **Henry常数的单位**：不同浓度单位对应的$K_H$数值不同
3. **理想溶液的判断**：不能仅凭"相似"就断定理想，需实验验证
4. **正偏差与负偏差的混淆**：正偏差$p > p_{Raoult}$，负偏差$p < p_{Raoult}$
5. **恒沸混合物的组成**：恒沸组成随压力变化

### 12.2 认知误区

- **误区1**：认为Raoult定律适用于所有溶液的所有组分。纠正：仅适用于理想溶液或稀溶液的溶剂。
- **误区2**：混淆Henry常数与纯溶质蒸气压。纠正：$K_H$一般不等于$p_2^*$，是实验测定值。
- **误区3**：认为正偏差溶液总是不稳定的。纠正：正偏差是常见现象，不一定导致相分离。

## 五、常见分类

**溶液类型**：

| 类型 | 特征 | 蒸气压行为 | 例子 |
|-----|------|-----------|------|
| 理想溶液 | 结构相似，作用力相近 | 遵守Raoult定律 | 苯-甲苯 |
| 正偏差溶液 | A-B作用力弱 | $p > p_{Raoult}$ | 乙醇-水 |
| 负偏差溶液 | A-B作用力强 | $p < p_{Raoult}$ | 氯仿-丙酮 |
| 稀溶液 | 溶质很少 | 溶剂Raoult，溶质Henry | 糖水 |

## 六、适用条件

| 定律 | 适用条件 | 不适用情况 |
|-----|---------|-----------|
| Raoult定律 | 理想溶液全浓度；稀溶液的溶剂 | 浓溶液中的溶质；非理想溶液 |
| Henry定律 | 稀溶液中的溶质；气体溶解 | 浓溶液；溶质发生缔合或解离 |
| 理想溶液假设 | 结构相似的物质 | 结构差异大的物质 |

## 七、常见比较

**Raoult定律 vs Henry定律**：

| 比较项 | Raoult定律 | Henry定律 |
|-------|-----------|-----------|
| 数学形式 | $p_i = x_i p_i^*$ | $p_i = K_H x_i$ |
| 比例常数 | 纯组分蒸气压 | Henry常数 |
| 适用组分 | 大量组分（溶剂） | 少量组分（溶质） |
| 标准态 | 纯液体 | 假想纯液体 |
| 浓度范围 | 全浓度（理想溶液） | 稀溶液 |
