# 化学势与平衡

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-物理化学深化]]
- 对应考纲条目：[[决赛04]]

## 一、定义

化学势 `\mu_i` 是体系吉布斯自由能对组分物质的量的偏导数，即在恒温恒压且其他组分量不变时，向体系中加入单位物质的量组分 `i` 所引起的自由能变化。它是描述物质迁移、相平衡和化学平衡驱动力的核心量；平衡的本质就是相关方向上化学势不再具有净下降趋势。

## 二、考纲对应

本条目对应决赛物理化学中“偏摩尔量、化学势、相平衡与化学平衡条件”的主干内容。竞赛要求不仅会写 `\mu=\mu^\ominus+RT\ln a`，还要能把它用于推导平衡条件、判别自发方向、建立逸度/活度近似下的定量关系。

## 三、核心原理

1. 对多组分体系，吉布斯自由能微分式可写为 `dG=-SdT+Vdp+\sum \mu_i dn_i`，因此化学势是组分层面的“自由能价格”。
2. 在恒温恒压下，体系自发变化方向对应总吉布斯自由能降低；若把自由能变化拆到各组分，就可用化学势差来判断迁移与反应方向。
3. 对纯物质或理想体系，化学势可表示为标准态项与活度项之和：`\mu_i=\mu_i^\ominus+RT\ln a_i`。活度刻画了“有效浓度”，是从理想到真实体系的桥梁。
4. 化学反应达到平衡时，反应吉布斯自由能 `\Delta_r G=\sum \nu_i \mu_i = 0`；相平衡达到平衡时，同一组分在各相中的化学势相等。
5. 因而“平衡常数”“蒸气压关系”“分配关系”本质上都可追溯到化学势相等或反应自由能为零这一统一框架。

## 四、关键结论

1. 同一组分在两相平衡时满足 `\mu_i^\alpha=\mu_i^\beta`，这是相平衡最根本的判据。
2. 化学反应平衡条件为 `\Delta_r G=0`，偏离平衡时其符号决定正逆反应的自发方向。
3. 对理想气体，`\mu_i=\mu_i^\ominus+RT\ln(p_i/p^\ominus)`；对理想溶液，常写成 `\mu_i=\mu_i^*+RT\ln x_i`。
4. 平衡常数来源于标准反应吉布斯自由能：`\Delta_r G^\ominus=-RT\ln K`。

## 五、常见分类或情形

1. `纯物质相平衡`：如液-气、固-液平衡，重点是同一物质跨相化学势相等。
2. `理想气体混合物`：以分压表达化学势，常用于推导反应平衡常数表达式。
3. `理想溶液与稀溶液`：以摩尔分数、浓度或活度近似表示化学势，常连接 Raoult 与 Henry 定律。
4. `真实体系`：需引入活度、逸度和相应标准态，竞赛中常以“接近理想”或“给定活度系数”形式出现。

## 六、适用条件与限制

1. `\mu=\mu^\ominus+RT\ln a` 这一写法普适，但关键在于活度定义和标准态选取，不能把浓度、分压、摩尔分数随意混写。
2. 纯用摩尔分数代替活度通常只适用于理想溶液或足够接近理想的情形。
3. 对电解质溶液、多相非理想体系、高压真实气体，若忽略活度系数或逸度系数会带来系统误差。
4. “化学势相等”是平衡条件，不是任意时刻恒成立的关系；非平衡传质过程恰恰由化学势差驱动。

## 七、常见比较与易混点

1. `化学势`是单组分偏摩尔自由能，`吉布斯自由能`是体系整体状态函数，两者层级不同。
2. `化学势相等`适用于同一组分跨相平衡，`反应平衡`则更常通过 `\sum \nu_i \mu_i=0` 表述，二者相关但不等同。
3. `浓度高`不一定意味着`化学势高`，还要看标准态和非理想修正。
4. `标准化学势`不是固定不变常数，而是随温度、压力和所选标准态定义而变的参照量。

## 八、与其他知识点的联系

1. 与 [[化学热力学]] 相连，化学势来自吉布斯自由能微分关系。
2. 与 [[Raoult与Henry定律]] 相连，两类定律都可从不同标准态下的化学势表达式推导。
3. 与 [[双组分相图]] 相连，相图中的相界本质上反映不同相之间化学势条件。
4. 与 [[电化学深化]] 相连，电极电势与电化学势、反应自由能密切相关。

## 九、典型题型

1. 由 `\mu_i` 表达式推导气相或溶液中的平衡常数关系。
2. 利用“同一组分在两相中化学势相等”推导蒸气压、分配或溶解平衡式。
3. 在给定活度或逸度修正时，比较真实体系与理想体系的平衡偏移。
4. 结合反应商 `Q` 与 `\Delta_r G=\Delta_r G^\ominus+RT\ln Q` 判断反应自发方向。

## 十、例题

设理想气体反应 `A(g) \rightleftharpoons B(g)` 在恒温下进行，写出平衡条件并说明平衡常数的来源。

解题思路：

1. 反应平衡时有 `\mu_B-\mu_A=0`。
2. 对理想气体，`\mu_i=\mu_i^\ominus+RT\ln(p_i/p^\ominus)`。
3. 代入得 `(\mu_B^\ominus-\mu_A^\ominus)+RT\ln(p_B/p_A)=0`。
4. 整理可得 `RT\ln(p_B/p_A)=-\Delta_r G^\ominus`，于是 `K=p_B/p_A=\exp(-\Delta_r G^\ominus/RT)`。

这道题体现的核心不是公式变形，而是“平衡常数来自化学势表达式与平衡条件的联立”。

## 十一、易错点

1. 直接把活度写成浓度，却没有声明理想近似或标准态。
2. 把“同一物质在两相中浓度相等”误当成相平衡条件，真正相等的是化学势而非数值浓度。
3. 在推导平衡常数时漏掉标准态项，导致量纲混乱。
4. 只会背 `\Delta_r G^\ominus=-RT\ln K`，却说不清它是如何从化学势定义推出的。

## 十二、🎯 教学视角

教学上适合把化学势讲成“组分加入体系时带来的自由能边际成本”。先让学生用“水往低处流”类比理解自由能下降驱动，再过渡到“粒子往低化学势方向迁移”，会比直接给偏导定义更容易建立直觉。之后再把反应平衡、相平衡统一到同一个框架里，整章会顺很多。

## 十三、竞赛拓展

1. 可进一步引入电化学势 `\tilde{\mu}_i=\mu_i+z_iF\phi`，处理带电粒子在电场中的平衡与迁移。
2. 在真实气体体系中，用逸度替代分压可保留与理想体系同构的化学势表达式。
3. 在统计热力学中，化学势还可与粒子数变化对应的配分函数变化联系起来，为巨正则系综提供桥梁。
