## 一、定义

催化循环（Catalytic Cycle）是催化剂在反应中经历一系列基元反应步骤、最终再生并释放产物的闭合反应路径。在过渡金属有机催化中，催化循环是理解反应机理的核心框架——它将复杂的金属催化反应拆解为氧化加成、迁移插入、β-H 消除、还原消除等可重复的基元步骤的组合。掌握催化循环的读写能力，是化学竞赛决赛水平与初赛水平的分水岭。

## 三、核心原理

### 3.1 催化循环四步曲

过渡金属催化的有机反应，绝大多数可以拆解为以下四个基元反应的组合：

```
┌─────────────────────────────────────────────────────────────┐
│                    催化循环四步曲                              │
├─────────────────────────────────────────────────────────────┤
│  ① 氧化加成（Oxidative Addition, OA）                         │
│     MLₙ + A—B  →  M(A)(B)Lₙ                                  │
│     金属氧化态 +2，配位数 +2                                   │
│     例：Pd(0) + Ar—X → Ar—Pd(II)—X                           │
├─────────────────────────────────────────────────────────────┤
│  ② 迁移插入（Migratory Insertion, MI）                        │
│     M—R + L  →  M—L—R（L 插入 M—R 键）                        │
│     金属氧化态不变，配位数不变                                  │
│     例：R—Pd—X + 烯烃 → R—CH₂—CH₂—Pd—X                       │
├─────────────────────────────────────────────────────────────┤
│  ③ β-H 消除（β-Hydride Elimination, β-HE）                    │
│     M—CH₂—CH₂—R → M—H + CH₂=CH—R                            │
│     金属氧化态不变，生成金属氢化物和烯烃                        │
│     例： Heck 反应中产物烯烃的生成                             │
├─────────────────────────────────────────────────────────────┤
│  ④ 还原消除（Reductive Elimination, RE）                      │
│     M(A)(B) → M + A—B                                        │
│     金属氧化态 −2，配位数 −2                                   │
│     例：Ar—Pd—R → Ar—R + Pd(0)                               │
└─────────────────────────────────────────────────────────────┘
```

**四步曲的氧化态变化规律**：

| 步骤 | 氧化态变化 | 配位数变化 | 电子数变化 |
|:---|:---:|:---:|:---:|
| 氧化加成 | +2 | +2 | 0（18e → 18e，但中间可能 16e）|
| 迁移插入 | 0 | 0 | 0 |
| β-H 消除 | 0 | 0 | 0 |
| 还原消除 | −2 | −2 | 0 |

**关键理解**：整个催化循环中，金属的氧化态经历"升-平-平-降"的循环，最终回到起始态。这是催化剂不被消耗的本质。

### 3.2 催化剂与催化循环的关系

**催化剂的定义特征**：
- 在反应中不被消耗（化学计量上）
- 降低反应活化能
- 参与反应但循环再生

**周转数（TON, Turnover Number）与周转频率（TOF, Turnover Frequency）**：

$$
\text{TON} = \frac{\text{产物摩尔数}}{\text{催化剂摩尔数}} \quad \text{（每个催化剂分子催化多少底物）}
$$

$$
\text{TOF} = \frac{\text{TON}}{\text{时间}} \quad \text{（催化效率）}
$$

**催化剂前体 vs 活性催化剂**：
- 催化剂前体：加入反应体系的初始配合物（如 Pd(OAc)₂、PdCl₂(PPh₃)₂）
- 活性催化剂：原位生成的真正催化物种（如 Pd(0)Lₙ）
- 例：Pd(OAc)₂ + 2PPh₃ + 还原剂 → Pd(0)(PPh₃)₂（活性催化剂）

**配体解离与催化活性**：
- Pd(PPh₃)₄（18e）在溶液中解离：Pd(PPh₃)₄ ⇌ Pd(PPh₃)₃ + PPh₃
- Pd(PPh₃)₃（16e，配位不饱和）才是真正参与氧化加成的活性物种
- 配体过多会抑制反应（Le Chatelier 原理）

### 3.3 配体在催化循环中的作用

**电子效应**：
- 富电子配体（如 PMe₃、NHC）加速氧化加成（OA 需要金属向底物反馈电子）
- 贫电子配体（如 P(OPh)₃）加速还原消除（RE 需要金属中心电子密度降低）

**位阻效应（Cone Angle）**：
- 大位阻配体（如 P(t-Bu)₃、SPhos、XPhos）促进还原消除（RE 需要配位层拥挤）
- 大位阻配体也可加速氧化加成（通过稳定低配位中间体）

| 配体 | 锥角 (°) | 电子性 | 主要作用 | 典型应用 |
|:---|:---:|:---:|:---|:---|
| PPh₃ | 145 | 中等 | 通用配体 | Suzuki, Heck |
| P(o-tolyl)₃ | 194 | 中等 | 大位阻，加速 OA | 早期偶联 |
| P(t-Bu)₃ | 182 | 富电子 | 加速 OA 和 RE | Buchwald-Hartwig |
| SPhos | ~180 | 富电子 | 大位阻，加速 OA | 芳基氯的 Suzuki |
| XPhos | ~190 | 富电子 | 极大位阻 | 难反应底物 |
| BINAP | — | 中等 | 手性配体 | 不对称催化 |
| NHC (SIMes) | — | 极强 σ 给予 | 稳定低价金属 | 第二代 Grubbs |

### 3.4 转金属（Transmetalation）

转金属是交叉偶联中连接两种有机片段的关键步骤：

$$
\ce{Ar-Pd(II)-X + R-B(OH)2 ->[碱] Ar-Pd(II)-R + X-B(OH)2}
$$

**在 Suzuki 偶联中的作用**：
- 有机硼酸被碱活化：R-B(OH)₂ + OH⁻ → [R-B(OH)₃]⁻（"ate" 络合物）
- "ate" 络合物更亲核，向 Pd(II) 转移 R 基团
- 同时 X 基团转移到硼上，形成 B-X 键

**转金属的驱动力**：
- 形成更强的 M-X 键（B-O 键强于 B-C 键在特定条件下）
- 热力学驱动力：金属-电负性差异

## 四、关键结论

1. **催化循环的本质是金属氧化态的"升降电梯"**——从低氧化态出发，经氧化加成升高，经还原消除降低，完成一轮循环。

2. **氧化加成和还原消除是氧化态变化的唯一两步**——迁移插入和 β-H 消除不改变氧化态，但改变配体连接方式。

3. **配体是催化循环的"调速器"**——电子效应调控 OA/RE 速率，位阻效应影响配位/解离平衡。

4. **Suzuki 偶联的转金属需要碱活化**——这是 Suzuki 与 Heck 机理的根本区别之一。

5. **Wacker 氧化的独特之处在于 CuCl 共催化**——Cu(I)/Cu(II) 循环负责将 Pd(0) 再氧化为 Pd(II)，实现 O₂ 作为终端氧化剂。

### 5.1 按基元反应组合分类

| 反应类型 | 基元反应组合 | 典型反应 |
|:---|:---|:---|
| 交叉偶联型 | OA → 转金属 → RE | Suzuki, Negishi, Stille, Sonogashira |
| 烯烃转化型 | OA → MI → β-HE → RE/脱 HX | Heck, Wacker |

| 氢化型 | OA(H₂) → MI(烯烃) → RE | 不对称氢化 |

### 5.2 按金属分类

| 金属 | 典型催化循环 | 特征 |
|:---|:---|:---|
| Pd | Suzuki, Heck, Stille, Negishi | 最通用的偶联金属，Pd(0)/Pd(II) 循环 |
| Ni | Kumada, Negishi（部分）| 廉价替代 Pd，可活化芳基醚 |
| Cu | Ullmann, Chan-Lam | Cu(I)/Cu(II) 或 Cu(0)/Cu(I)/Cu(II) 循环 |
| Rh | 氢甲酰化、氢化 | Rh(I)/Rh(III) 循环 |

| 配体角色 | 功能 | 例子 |
|:---|:---|:---|

| 助催化剂 | 再生催化剂 | CuCl（Wacker 中再生 Pd(II)）|
| 碱 | 活化试剂/捕获 HX | Et₃N、Na₂CO₃、K₃PO₄ |

## 六、适用条件与限制

### 6.1 各基元反应的适用条件

**氧化加成（OA）**：
- 需要配位不饱和的低价金属（16e 或 14e）
- 底物需要有可断裂的 σ 键（C-X, H-H, C-H 等）
- 富电子配体加速 OA
- 芳基/烯基卤 > 烷基卤（烷基卤 β-H 消除竞争）

**迁移插入（MI）**：
- 需要配位的 unsaturated 配体（CO、烯烃、炔烃）
- M-R 键和配位的不饱和配体需处于 cis 位置
- 插入方向：CO 总是插入 M-R；烯烃插入遵循马氏/反马氏规则

**β-H 消除（β-HE）**：
- 需要 β-碳上有氢原子
- M-R 键和 β-H 需处于 syn 共平面
- 是 Heck 反应产物形成的必要步骤，但也是烷基卤偶联失败的原因

**还原消除（RE）**：
- 需要两个待消除基团处于 cis 位置
- 配位层拥挤有利于 RE（大位阻配体促进）
- 金属氧化态通常 ≥ +2

### 6.2 典型催化反应的适用条件

| 反应 | 催化剂前体 | 配体 | 碱/添加剂 | 溶剂 | 温度 |
|:---|:---|:---|:---|:---|:---:|
| Suzuki | Pd(OAc)₂, Pd₂(dba)₃ | PPh₃, SPhos | Na₂CO₃, K₃PO₄ | 二氧六环/水 | 80-100°C |
| Heck | Pd(OAc)₂, Pd(PPh₃)₄ | PPh₃ | Et₃N, KOAc | DMF, MeCN | 80-120°C |
| Wacker | PdCl₂ | — | CuCl/O₂ | 水/丙酮 | 室温-50°C |
| 氢甲酰化 | RhCl(PPh₃)₃ | PPh₃ | — | 甲苯 | 80-120°C |
| RCM | Grubbs I/II | NHC (内置) | — | CH₂Cl₂ | 室温-40°C |

## 七、常见比较与易混点

### 7.1 Heck vs Suzuki 机理对比

| 特征 | Heck 反应 | Suzuki 偶联 |
|:---|:---|:---|
| 底物 | Ar-X + 烯烃 | Ar-X + R-B(OH)₂ |
| 关键基元 | OA → MI → β-HE → 脱 HX | OA → 转金属 → RE |
| 氧化态循环 | Pd(0) → Pd(II) → Pd(0) | Pd(0) → Pd(II) → Pd(0) |
| 产物特征 | 芳基烯烃（新 C=C） | 联芳基/烷芳基（新 C-C σ）|
| 是否需要碱 | 是（脱 HX） | 是（活化硼酸 + 脱 HX）|
| 区域选择性 | 取代较少碳连芳基 | 无区域选择问题（对称偶联）|
| 能否用于烷基卤 | 不能（β-HE 太快） | 困难（β-HE 竞争）|

### 7.2 各交叉偶联反应比较

| 偶联类型 | 有机金属试剂 | 是否需要碱 | 毒性 | 官能团耐受性 | 竞赛频率 |
|:---|:---|:---:|:---:|:---:|:---:|
| Suzuki | R-B(OH)₂ | 是 | 很低 | 很好 | ★★★★★ |
| Heck | 烯烃（无需有机金属） | 是 | — | 好 | ★★★★ |
| Negishi | R-ZnX | 否 | 低 | 好 | ★★ |
| Stille | R-SnR'₃ | 否 | **高** | 很好 | ★ |
| Kumada | R-MgX | 否 | 低 | 较差 | ★★ |

### 7.3 第一代 vs 第二代 Grubbs 催化剂

| 特征 | 第一代 Grubbs | 第二代 Grubbs |
|:---|:---|:---|
| 结构 | RuCl₂(PCy₃)₂=CHPh | RuCl₂(NHC)(PCy₃)=CHPh |
| 活性 | 中等 | 高 |
| 热稳定性 | 较好 | 更好 |
| 对官能团耐受性 | 好 | 很好 |
| 可聚合烯烃 | 有限制 | 范围更广 |
| NHC 配体 | 无 | SIMes 或 IMes |

## 十、例题

### 例题 1（Heck 反应催化循环补全）

**题目**：补全下列 Heck 反应的催化循环，标注每一步的基元反应类型和 Pd 的氧化态。

$$
\ce{Ph-Br + CH2=CH-CO2Et ->[Pd(0)/Et3N] Ph-CH=CH-CO2Et + HBr}
$$

**解答**：

```
步骤 1：配体解离
    Pd(0)(PPh₃)₄  ⇌  Pd(0)(PPh₃)₃ + PPh₃
    （氧化态：Pd(0)，16e，配位不饱和）

步骤 2：氧化加成（OA）
    Pd(0)(PPh₃)₃ + Ph-Br  →  Ph-Pd(II)(Br)(PPh₃)₂
    （氧化态：Pd(0) → Pd(+2)，18e）

步骤 3：烯烃配位
    Ph-Pd(II)(Br)(PPh₃)₂ + CH₂=CH-CO₂Et  →  Ph-Pd(II)(Br)(η²-烯烃)(PPh₃)
    （氧化态：Pd(+2)，18e）

步骤 4：迁移插入（MI）
    Ph-Pd(II)(Br)(η²-烯烃)  →  Ph-CH₂-CH(Pd(II)(Br))-CO₂Et
    （氧化态：Pd(+2)，16e；Ph 迁移到取代较少的碳）

步骤 5：β-H 消除（β-HE）
    Ph-CH₂-CH(Pd(II)(Br))-CO₂Et  →  Ph-CH=CH-CO₂Et + H-Pd(II)(Br)(PPh₃)
    （氧化态：Pd(+2)，16e）

步骤 6：碱促进的还原消除/脱 HX
    H-Pd(II)(Br) + Et₃N  →  Pd(0)(PPh₃)₂ + Et₃NH⁺Br⁻
    （氧化态：Pd(+2) → Pd(0)，催化剂再生）
```

**关键说明**：
- 迁移插入方向：苯基迁移到取代较少的烯烃碳（电子效应控制）
- β-H 消除后，Pd 上连有 H 和 Br，Et₃N 脱去 HBr 使 Pd(0) 再生
- 整个循环 Pd 从 0 → +2 → 0，符合催化剂定义

### 例题 2（Suzuki 偶联机理与碱的作用）

**题目**：Suzuki 偶联中碱的作用是什么？写出完整的催化循环，并解释为什么有机硼酸需要碱活化。

**解答**：

**碱的三重作用**：
1. **活化有机硼酸**：R-B(OH)₂ + OH⁻ → [R-B(OH)₃]⁻（四配位 "ate" 络合物），B 更富电子，转金属更容易
2. **捕获还原消除前的 HX**：促进还原消除向产物方向进行
3. **促进催化剂前体还原**：Pd(II) 前体在碱存在下被还原为 Pd(0) 活性物种

**Suzuki 催化循环**：

```
步骤 1：催化剂活化
    Pd(II)(OAc)₂ + 2PPh₃ + 还原剂/碱  →  Pd(0)(PPh₃)₂（活性催化剂）

步骤 2：氧化加成（OA）
    Pd(0)(PPh₃)₂ + Ar-X  →  Ar-Pd(II)(X)(PPh₃)₂
    （Pd(0) → Pd(+2)）

步骤 3：有机硼酸活化（溶液中）
    R-B(OH)₂ + OH⁻  ⇌  [R-B(OH)₃]⁻

步骤 4：转金属（Transmetalation）
    Ar-Pd(II)(X) + [R-B(OH)₃]⁻  →  Ar-Pd(II)(R) + [X-B(OH)₃]⁻
    （氧化态不变，Pd(+2)）

步骤 5：还原消除（RE）
    Ar-Pd(II)(R)  →  Ar-R + Pd(0)(PPh₃)₂
    （Pd(+2) → Pd(0)，催化剂再生）
```

**为什么需要碱活化**：
- 三配位硼酸 R-B(OH)₂ 中，B 为 sp² 杂化，缺电子，C-B 键极性不够强
- 四配位 "ate" 络合物 [R-B(OH)₃]⁻ 中，B 为 sp³ 杂化，负电荷使 B 更富电子
- R 基团的亲核性增强，更容易向电正性的 Pd(II) 转移

### 例题 3（Wacker 氧化机理与共催化循环）

**题目**：Wacker 氧化将乙烯转化为乙醛，使用 PdCl₂/CuCl/O₂ 体系。写出完整的催化循环，并解释 CuCl 的作用。

**解答**：

**总反应**：

$$
\ce{CH2=CH2 + 1/2 O2 ->[PdCl2/CuCl] CH3CHO}
$$

**Pd 催化循环**：

```
步骤 1：烯烃配位
    Pd(II)Cl₄²⁻ + CH₂=CH₂  ⇌  [Pd(II)Cl₃(η²-CH₂=CH₂)]⁻ + Cl⁻

步骤 2：亲核进攻（水分子）
    [Pd(II)Cl₃(烯烃)]⁻ + H₂O  →  [Pd(II)Cl₂(OH)(CH₂CH₂)]⁻ + H⁺ + Cl⁻
    （OH⁻ 进攻取代较多的碳——马氏取向）

步骤 3：β-H 消除
    [Pd(II)Cl₂(OH)(CH₂CH₂)]⁻  →  Pd(0) + CH₂=CH(OH) + 2Cl⁻
    （烯醇互变异构为乙醛）
```

**CuCl 共催化循环**：

```
步骤 4：Pd(0) 的再氧化
    Pd(0) + 2CuCl₂  →  Pd(II)Cl₂ + 2CuCl
    （Cu(II) 将 Pd(0) 氧化回 Pd(II)）

步骤 5：Cu(I) 的再氧化
    2CuCl + 2HCl + 1/2 O₂  →  2CuCl₂ + H₂O
    （O₂ 将 Cu(I) 氧化回 Cu(II)，实现终端氧化剂的循环利用）
```

**CuCl 的双重作用**：
1. **化学计量氧化剂**：Cu(II) 将 Pd(0) 再氧化为 Pd(II)，使 Pd 能够循环使用
2. **O₂ 的电子传递媒介**：Cu(I) 被 O₂ 氧化再生，避免了 Pd(0) 直接与 O₂ 反应生成 PdO 沉淀

**马氏取向的解释**：
- 水分子进攻配位烯烃时，进攻取代较多的碳
- 这是因为 Pd-烯烃配合物中，取代较多的碳带有更多正电性（Pd 的电子反馈使该碳更亲电）
- 最终产物为甲基酮（末端烯烃）或内酮（内烯烃）

## 十一、易错点

| 易错点 | 错误认知 | 正确理解 |
|:---|:---|:---|
| Heck 与 Suzuki 机理混淆 | "Heck 也有转金属" | Heck 是 OA → MI → β-HE，无转金属；Suzuki 是 OA → 转金属 → RE，无 β-HE |
| β-H 消除的方向 | "β-H 消除总是生成最稳定的烯烃" | β-H 消除要求 syn 共平面几何，热力学稳定烯烃不一定符合几何要求 |
| 还原消除的立体化学 | "还原消除保持构型" | 还原消除要求两个基团 cis，若底物为 trans 需先异构化 |
| Wacker 的产物 | "Wacker 氧化得到醇" | Wacker 氧化得到 **酮/醛**（烯醇互变异构），不是醇 |
| 催化剂前体 = 活性催化剂 | "加入 Pd(OAc)₂ 就是 Pd(II) 催化" | Pd(OAc)₂ 通常被原位还原为 Pd(0) 才是活性催化剂 |
| 配体越多越有利 | "多加 PPh₃ 提高产率" | 过量膦配体会抑制配体解离，反而降低催化活性 |
| 氧化加成的底物范围 | "所有卤代烃都能做 Heck/Suzuki" | 烷基卤因 β-H 消除竞争而难以使用；芳基/烯基卤是标准底物 |
| 转金属的碱需求 | "所有偶联都需要碱" | Suzuki 需要碱活化硼酸；Negishi/Stille 通常不需要碱 |

### 12.2 认知误区

**误区 1："催化循环就是反应机理"**
- 辨析：催化循环是机理的简化模型，只展示金属中心的转化。完整的机理还包括配体解离/配位、溶剂效应、离子对等细节。

**误区 2："所有 Pd 催化循环都一样"**
- 辨析：虽然都是 Pd(0) ↔ Pd(II)，但 Heck 有 β-HE，Suzuki 有转金属，反应性完全不同。看到 Pd 催化剂要先判断是哪类反应。

**误区 3："β-H 消除是副反应，要尽量避免"**
- 辨析：β-H 消除在 Heck 反应中是 **必需的产物生成步骤**；但在 Suzuki/Negishi 中，若烷基卤底物发生 β-HE，则会导致偶联失败。是"必要的恶"还是"要避免的恶"，取决于反应类型。

**误区 4："Wacker 氧化中 CuCl 是催化剂"**
- 辨析：CuCl 是 **共催化剂**（cocatalyst）或 **化学计量再氧化剂**。它在 Cu(I) 和 Cu(II) 之间循环，但每轮消耗的是 O₂。严格来说，Pd 是催化剂，Cu 是媒介。

### 12.3 入门例题

**入门题 1**：判断下列转化属于哪种基元反应。

(1) Pt(0) + H₂ → Pt(H)₂  
(2) R-Pd-CO → R-CO-Pd  
(3) R-CH₂-CH₂-Pd-H → R-CH=CH₂ + Pd(0) + H⁺  
(4) Ar-Pd-R → Ar-R + Pd(0)

**答案**：
(1) 氧化加成  
(2) 迁移插入（CO 插入 M-R）
(3) β-H 消除  
(4) 还原消除

**入门题 2**：在 Heck 反应中，为什么烷基卤（如碘乙烷）不能作为底物？

**答案**：烷基卤与 Pd(0) 氧化加成后形成烷基-Pd(II) 中间体，该中间体有 β-H，会发生快速的 β-H 消除生成烯烃，无法进入迁移插入步骤，因此不能得到 Heck 产物。

**入门题 3**：Suzuki 偶联中，为什么有机硼酸比有机锡试剂（Stille）更常用？

**答案**：有机硼酸（1）对水和空气稳定，操作简便；（2）毒性很低，环境友好；（3）官能团耐受性好；（4）碱活化后反应性适中。有机锡虽然也很稳定，但有机锡化合物剧毒，限制了其应用范围。
