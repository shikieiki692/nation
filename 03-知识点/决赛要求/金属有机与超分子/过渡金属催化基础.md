# 过渡金属催化基础（Transition Metal Catalysis）

## 一、定义

**过渡金属催化（Transition Metal Catalysis）**：利用 d 区金属（主要是 Pd, Ni, Cu, Rh, Ru 等）的可变氧化态和配位能力，通过**催化循环**实现 C–C、C–H、C–杂原子键的构建与转化的反应体系。

**核心特征**：
- 金属中心可在**相邻氧化态间循环**（如 Pd(0) ↔ Pd(II)）
- 配体调控金属的电子性质和空间环境
- 催化量即可驱动化学计量反应

## 三、核心原理

### 3.1 催化循环的五大基元反应

所有过渡金属催化偶联反应都由以下基元反应组合而成：

#### ① 氧化加成（Oxidative Addition, OA）

$$
\ce{L_nM^{(0)} + R-X -> L_nM^{(II)}(R)(X)}
$$

- 金属氧化态 **+2**
- 配位数 **+2**
- 是 **Pd(0)** 启动催化循环的第一步
- 活性顺序：**I > OTf > Br > Cl**（碳卤键断裂难易）

#### ② 还原消除（Reductive Elimination, RE）

$$
\ce{L_nM^{(II)}(R)(R') -> L_nM^{(0)} + R-R'}
$$

- 金属氧化态 **−2**
- 配位数 **−2**
- C–C 键形成的最后一步
- 要求两个有机配体处于 **cis** 位置

#### ③ 迁移插入（Migratory Insertion, MI）

$$
\ce{L_nM-R + CH2=CH2 -> L_nM-CH2-CH2-R}
$$

- 不饱和配体（烯烃、CO）插入 M–C 键
- 氧化态**不变**，配位数**不变**
- Heck 反应的核心步骤

#### ④ β-H 消除（β-Hydride Elimination, β-HE）

$$
\ce{L_nM-CH2-CH2-R -> L_nM-H + CH2=CH-R}
$$

- 金属氧化态 **+2**
- 需要 **syn** 共平面构型（M–C–C–H 二面角 ≈ 0°）
- Heck 反应产物双键的来源

#### ⑤ 转金属（Transmetalation, TM）

$$
\ce{L_nM^{(II)}-X + R'-M' -> L_nM^{(II)}-R' + X-M'}
$$

- 有机基团从一种金属转移到另一种金属
- Suzuki（B → Pd）、Stille（Sn → Pd）、Negishi（Zn → Pd）的核心步骤

### 3.2 常见催化循环模式

**模式 A：氧化加成 → 转金属 → 还原消除**
- 代表：Suzuki、Stille、Negishi、Sonogashira
- 特征：Pd(0)/Pd(II) 循环，最终生成 R–R' 并再生 Pd(0)

**模式 B：氧化加成 → 迁移插入 → β-H 消除**
- 代表：Heck 反应
- 特征：无还原消除，产物保留双键

**模式 C：氧化加成 → 迁移插入 → 还原消除**
- 代表：羰基化反应、氢甲酰化
- 特征：CO 或 H₂ 插入后还原消除

### 3.3 配体的作用

| 配体类型 | 例子 | 作用 |
|:---|:---|:---|
| 单齿膦 | PPh₃, P(o-tolyl)₃ | 调控空间位阻和电子密度 |
| 双齿膦 | dppe, dppp, BINAP | 螯合稳定，控制 cis/trans |
| N-杂环卡宾 (NHC) | IMes, SIPr | 强 σ-给电子，高稳定性 |
| 双齿 N-配体 | bpy, phen | 稳定低价态金属 |

**配体电子效应**：
- 富电子配体 → 加速氧化加成（Pd(0) 更富电子，易氧化）
- 大位阻配体 → 促进还原消除（cis 配体被迫靠近）

## 四、关键结论

1. **五大基元反应是理解所有过渡金属催化反应的"字母表"**——任何复杂循环都可拆解为这五种
2. **Pd(0) 启动，Pd(II) 工作**——Pd 催化的核心氧化态循环
3. **还原消除要求 cis 构型**——trans 构型无法发生 RE
4. **β-H 消除要求 syn 共平面**——这是 Heck 产物立体化学的根源
5. **配体不是旁观者**——直接调控反应速率、选择性、稳定性

## 五、常见分类或情形

### 5.1 按金属分类

| 金属 | 代表反应 | 特点 |
|:---|:---|:---|
| **Pd** | Suzuki, Heck, Stille, Negishi | 最常用，循环明确 |
| **Ni** | Kumada, Negishi（廉价替代）| 可活化芳基氯 |
| **Cu** | Ullmann, Glaser, Click | 低价 Cu(I) 主导 |
| **Rh** | 氢甲酰化、氢化 | Rh(I)/Rh(III) 循环 |
| **Ru** | 烯烃复分解、氢化 | 卡宾化学 |

### 5.2 按产物键型分类

| 键型 | 反应 | 基元组合 |
|:---|:---|:---|
| C(sp²)–C(sp²) | Suzuki, Heck | OA + TM + RE / OA + MI + β-HE |
| C(sp²)–C(sp) | Sonogashira | OA + TM + RE |
| C(sp²)–C(sp³) | Kumada, Negishi | OA + TM + RE |
| C–N | Buchwald-Hartwig | OA + 胺配位 + RE |
| C–O | 醚化反应 | OA + 醇盐转金属 + RE |

## 六、适用条件与限制

1. **需要无氧条件**——Pd(0) 易被 O₂ 氧化失活
2. **碱的作用多样**——捕获 HX（Suzuki/Heck）、活化硼酸、促进转金属
3. **溶剂配位性影响**——DMF、DMSO 等配位溶剂可能竞争配位，降低活性
4. **卤代物活性差异大**——Ar-I > Ar-Br >> Ar-Cl，氯化物常需特殊催化剂
5. **烷基卤问题**——β-H 消除太快，难以控制（除非无 β-H，如 CH₃I）
6. **催化剂前体**——常用 Pd(OAc)₂、Pd₂(dba)₃ 等，原位还原为 Pd(0)

## 七、常见比较与易混点

### 7.1 五大基元反应速查表

| 基元反应 | Δ氧化态 | Δ配位数 | 键变化 | 代表反应 |
|:---|:---:|:---:|:---|:---|
| 氧化加成 | +2 | +2 | 断裂 R–X | 所有 Pd 催化起点 |
| 还原消除 | −2 | −2 | 形成 R–R' | Suzuki/Stille 终点 |
| 迁移插入 | 0 | 0 | 形成 C–C，断裂 M–C | Heck 核心 |
| β-H 消除 | +2 | 0 | 形成 C=C，生成 M–H | Heck 终点 |
| 转金属 | 0 | 0 | 交换有机基团 | Suzuki/Stille 核心 |

### 7.2 Heck vs Suzuki 催化循环对比

| 步骤 | Heck | Suzuki |
|:---|:---|:---|
| 1 | 氧化加成 | 氧化加成 |
| 2 | 烯烃配位 | 转金属（B → Pd）|
| 3 | 迁移插入 | — |
| 4 | β-H 消除 | — |
| 5 | 碱脱 HX → Pd(0) | 还原消除 → Pd(0) |
| **产物特征** | 保留双键 | 饱和连接 |

## 十、例题

### 10.1 例题 1（★★ 催化循环）

**题目**：画出 Suzuki 偶联的催化循环，标注每一步的金属氧化态。

**解答**：

```
Pd(0)L₂ + Ar-Br  ──OA──>  Ar-Pd(II)(Br)L₂
      ↑                              │
      │                              │ TM
   RE（还原消除）                     ↓
      │                    Ar-Pd(II)(Ar')L₂
      │                              ↑
      └──── Pd(0)L₂ + Ar-Ar'  ←──  RE
              + 碱·HBr
```

1. Pd(0)L₂ + ArBr → Ar-Pd(II)(Br)L₂ （OA，+2）
2. Ar-Pd(II)(Br)L₂ + Ar'B(OH)₂ → Ar-Pd(II)(Ar')L₂ + B(OH)₂Br （TM）
3. Ar-Pd(II)(Ar')L₂ → Ar-Ar' + Pd(0)L₂ （RE，−2）

### 10.2 例题 2（★★★ 机理分析）

**题目**：为什么 Suzuki 反应中需要碱？碱的作用是什么？

**解答**：
- **活化硼酸**：ArB(OH)₂ + OH⁻ → ArB(OH)₃⁻，提高亲核性/转金属速率
- **捕获副产物**：中和还原消除前生成的 HBr，防止酸性条件使 Pd(0) 质子化失活
- **促进转金属**：硼酸负离子比中性硼酸更易向 Pd 转移有机基团

### 10.3 例题 3（★★★ 选择性解释）

**题目**：Heck 反应中，为什么末端烯烃的芳基化主要发生在 β-位？

**解答**：
1. 氧化加成后：Ar-Pd(II)-X
2. 烯烃配位 → 迁移插入：Ar 迁移到烯烃的一个碳上
3. 两种可能的迁移插入产物：
   - α-插入：Ar-CH₂-CH(R)-Pd-X（Pd 在 α-位）
   - β-插入：Ar-CH(R)-CH₂-Pd-X（Pd 在 β-位）
4. 下一步 β-H 消除要求 Pd 邻位有 H
5. 若 Ar 连在 α-位（取代较少的碳），β-位有 H → 可顺利 β-H 消除
6. 若 Ar 连在 β-位（取代较多的碳），α-位可能无 H → 难以消除
7. **电子效应**：芳基连在取代较少的碳，形成的碳正离子特征中间体更稳定

### 10.4 工业催化循环案例（来自初赛讲义第5讲）

#### Monsanto 乙酸合成（Rh 催化）

工业上以 $\ce{[Rh(CO)2I2]-}$ 为催化剂、$\ce{CH3I}$ 为助催化剂，从 $\ce{CH3OH + CO}$ 合成乙酸。催化循环：

1. **氧化加成**：$\ce{[Rh(CO)2I2]- + CH3I -> [Rh(CO)2I3(CH3)]-}$（Rh(I)→Rh(III)，+2）
2. **迁移插入**：$\ce{CH3}$ 迁移到 CO 配体 → 形成乙酰基 $\ce{Rh-COCH3}$ 中间体
3. **CO 配位**：外部 CO 补充配位空位
4. **还原消除**：$\ce{CH3COI}$ 脱离，Rh(III)→Rh(I)，催化剂再生
5. **水解**：$\ce{CH3COI + H2O -> CH3COOH + HI}$

> 关键：$\ce{CH3I}$ 将甲醇转化为活性更高的碘甲烷，HI 在体系中循环。Cativa 法（Ir 替代 Rh）步骤类似但配位顺序不同（先 CO 取代 I 再甲基迁移）。

#### Wacker 乙醛氧化（Pd/Cu 双循环）

在 $\ce{PdCl2-CuCl2-O2}$ 体系中，乙烯被氧化为乙醛（末端烯烃→甲基酮，符合马氏规则）：

$$
\ce{C2H4 + 1/2 O2 ->[PdCl2/CuCl2] CH3CHO}
$$

**Pd 循环**（主催化循环）：
1. **配位 + 插入**：$\ce{C2H4}$ 配位到 Pd(II)，水分子促进下烯烃插入 Pd—OH → $\ce{Pd-CH2CH2OH}$
2. **1,3-消除**：β-H 消除（但消除的是 OH 邻位的 H）→ 烯醇式中间体
3. **异构化**：烯醇 → 乙醛（酮式更稳定）
4. **还原消除**：Pd(II) 被还原为 Pd(0)

**Cu 循环**（再生循环）：
- $\ce{Pd(0) + 2 CuCl2 -> PdCl2 + 2 CuCl}$（Pd 再生）
- $\ce{2 CuCl + 2 HCl + 1/2 O2 -> 2 CuCl2 + H2O}$（Cu 再生）

> 🔑 核心洞察：末端烯烃总是生成**甲基酮**（马氏规则——OH 加在取代更多的碳上，随后异构化为酮）。非末端烯烃则生成取代更多的酮。这是 Wacker 氧化的标志性区域选择性。

![[00-附件/第5讲-Wacker催化循环图.jpg]]

> Wacker 催化循环示意图（来自初赛讲义第5讲）：Pd 循环（乙烯→乙醛）+ Cu 循环（Pd(0) 再生）

#### 氢化甲酰化（Hydroformylation，Co 催化）

$\ce{HCo(CO)4}$ 催化下，烯烃与 $\ce{CO + H2}$ 反应生成醛（多一个碳）：

$$
\ce{R-CH=CH2 + CO + H2 -> RCH2CH2CHO}
$$

六步催化循环：

| 步骤 | 反应 | 基元类型 |
|:---:|:---|:---:|
| (1) | $\ce{HCo(CO)4 -> HCo(CO)3 + CO}$ | **解离**（产生活性 16e 物种） |
| (2) | $\ce{HCo(CO)3 + C2H4 -> HCo(C2H4)(CO)3}$ | **配位** |
| (3) | $\ce{HCo(C2H4)(CO)3 -> Co(CH2CH3)(CO)3}$ | **插入**（H 迁移到烯烃） |
| (4) | $\ce{Co(CH2CH3)(CO)3 + CO -> EtCOCo(CO)3}$ | **插入**（乙基迁移到 CO → 酰基） |
| (5) | $\ce{H2 + EtCOCo(CO)3 -> EtCOCoH2(CO)3}$ | **氧化加成**（H—H 断裂，Co 氧化态 +2） |
| (6) | $\ce{EtCOCoH2(CO)3 -> EtCHO + HCo(CO)3}$ | **还原消除**（产物醛脱离，催化剂再生） |

> 🔑 氢化甲酰化是均相催化在工业上最成功的应用之一，年产量 >1000 万吨醛。后来发展的 Rh 催化体系（低压氢化甲酰化）选择性更高。

## 十一、易错点

1. ❌ **认为催化循环中金属氧化态不变** → 实际是循环变化的（如 Pd(0) ↔ Pd(II)）
2. ❌ **混淆迁移插入和还原消除** → MI 不生成新键离开金属；RE 生成产物并再生催化剂
3. ❌ **忽略 β-H 消除的 syn 要求** → 只有 syn 共平面才能发生，这决定了 Heck 产物的立体化学
4. ❌ **认为所有配体都加速反应** → 配位太强的溶剂（如 Py）可能抑制反应
5. ❌ **转金属和氧化加成混淆** → 转金属氧化态不变；氧化加成 +2
6. ❌ **忽略 cis 要求对还原消除的限制** → trans 双膦配体可能抑制 RE

### 12.3 入门级例题

**题目**：在 Pd(PPh₃)₄ 催化的 Suzuki 反应中，Pd 的起始氧化态是多少？转金属后呢？还原消除后又是什么？

**预期解答路径**：
1. Pd(PPh₃)₄ 中 Pd 是 **Pd(0)**（四个中性膦配体）
2. 氧化加成 ArBr 后 → **Pd(II)**（Ar 和 Br 各贡献 −1）
3. 转金属后 → 仍是 **Pd(II)**（交换配体，氧化态不变）
4. 还原消除后 → **Pd(0)**（失去两个有机配体，回到低价）

**老师可引导的提问**：
- 卡在第 2 步：问 "Ar 和 Br 各带什么电荷？Pd 需要多少正电荷来平衡？"

### 12.4 与现实/直觉的连接

- **类比**：过渡金属催化就像一个"媒人"——自己不结婚（催化量），但撮合两家人（R 和 R'）在一起。撮合的手段包括：先跟一方握手（氧化加成），再介绍给另一方（转金属），最后功成身退（还原消除）。
- **反例**：直觉认为"催化剂就是加快速度不参与反应"。实际上，过渡金属催化剂的每个原子都深度参与了化学键的断裂和形成，是真正的"分子演员"。
