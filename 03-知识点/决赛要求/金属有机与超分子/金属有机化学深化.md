## 一、定义

金属有机化学（Organometallic Chemistry）是研究含有金属-碳键（M-C）的化合物及其反应的化学分支。过渡金属有机化学则聚焦于 d 区元素与有机配体形成的配合物，其核心特征在于金属中心可经历可逆的氧化态变化，并通过配体的配位与解离实现催化循环。18 电子规则是判断金属有机配合物稳定性的首要工具，而配体的 X/L 分类法则是理解电子计数与反应性的基础框架。

## 三、核心原理

### 3.1 18 电子规则

过渡金属有机配合物倾向于达到 18 价电子构型（对应下一个稀有气体的电子构型），这一规则与主族元素的 8 电子规则 analogous。

**电子计数公式**：

$$
\text{总价电子数} = \text{金属的价电子数（族数）} + \text{配体提供的电子数} + \text{电荷修正}
$$

**常见配体的电子贡献**：

| 配体 | 类型 | 电子数 | 备注 |
|:---|:---:|:---:|:---|
| CO | L 型 | 2 | 强 π 受体，反馈键 |
| PR₃（膦） | L 型 | 2 | 电子密度调节 |
| H⁻ | X 型 | 1 | 氢负离子 |
| Cl⁻, Br⁻, I⁻ | X 型 | 1 | 卤素离子 |
| η¹-烷基（R） | X 型 | 1 | σ 键 |
| η²-烯烃（C=C） | L 型 | 2 | 配位后电子数不变 |
| η⁵-Cp（环戊二烯基） | LX 型 | 6 | 5 个碳各贡献约 1.2e |
| η⁶-C₆H₆（苯） | L 型 | 6 | 六电子给予体 |
| η³-烯丙基 | LX 型 | 3 | 烯丙基配体 |
| N₂ | L 型 | 2 | 与 CO 等电子 |

**电子计数方法——共价法 vs 离子法**：

```
共价法（Covalent Method）：
- 金属：中性原子，贡献价电子数 = 族数
- 配体：中性配体，贡献成键电子数
- 例：Fe(CO)₅ → Fe(8) + 5×CO(2) = 18e ✓

离子法（Ionic Method）：
- 金属：按氧化态计算 d 电子数
- 配体：按带电配体计算
- 例：[Mn(CO)₅]⁻ → Mn(-I), d⁸ + 5×CO(2) = 18e ✓
```

两种方法最终结果一致，竞赛中推荐掌握共价法，因其更直观。

**16 电子与 18 电子的互换**：
- 16 电子配合物常具有配位不饱和性，是催化活性的来源
- 18 电子配合物通常稳定，需要配体解离才能反应
- Pd(PPh₃)₄ 是 18e，但溶液中解离为 Pd(PPh₃)₃（16e）后参与氧化加成

### 3.2 配体分类——X 型与 L 型

Covalent Bond Classification（CBC）法由 Malcolm Green 提出，将配体分为：

**L 型配体（中性配体）**：
- 通过配位键（dative bond）向金属提供孤对电子
- 配位前后配体本身不带电荷
- 例：CO、PR₃、NH₃、H₂O、η²-烯烃、η⁶-芳烃

**X 型配体（阴离子配体）**：
- 与金属形成共价键，各贡献 1 个电子
- 配体视为从金属"夺走"1 个电子，故金属氧化态 +1
- 例：H、Cl、Br、I、烷基（R）、芳基（Ar）、OR、NR₂

**Z 型配体（阳离子配体，竞赛较少涉及）**：
- 接受金属电子对的配体（如 BR₃）

**LX 型配体**：
- 同时具有 L 和 X 特征的配体
- 例：η⁵-Cp（提供 6e，其中 1e 视为共价，5e 视为配位）

**氧化态确定规则**：
1. 所有 X 型配体视为阴离子（R⁻, Cl⁻, H⁻）
2. 所有 L 型配体视为中性
3. 配合物总电荷 = 金属氧化态 + 配体电荷之和

$$
\text{氧化态} = \text{配合物电荷} - \sum(\text{配体电荷})
$$

例：在 [Pd(PPh₃)₂(Ph)(Br)] 中，PPh₃ 为 L 型（0），Ph 和 Br 为 X 型（各 -1），配合物中性，故 Pd 氧化态为 +2。

### 3.3 金属-碳键类型

**σ 键（M-C σ bond）**：
- 最常见的 M-C 键类型
- 烷基、芳基、酰基与金属形成 σ 键
- 键能一般较弱（约 150-250 kJ/mol），热稳定性有限
- β-H 的存在导致易消除（见 β-H 消除）

**π 键（M-C π bond）——金属卡宾与卡拜**：

| 类型 | 结构 | 电子特征 | 典型例子 |
|:---|:---|:---|:---|
| Fischer 卡宾 | M=C(R)OR' | 金属低氧化态，卡宾碳亲电 | Cr(CO)₅=C(OMe)Me |
| Schrock 卡宾 | M=CHR | 金属高氧化态，卡宾碳亲核 | Tebbe 试剂 |
| 金属卡拜 | M≡CR | 三键，高氧化态 | ClW≡CPh |

**金属-烯烃 π 配合物**：
- Dewar-Chatt-Duncanson 模型：烯烃的 π 电子 → 金属空 d 轨道（σ 给予），金属 d 电子 → 烯烃 π* 轨道（π 反馈）
- 反馈键越强，烯烃 C=C 键越长，反应性越高
- Zeise 盐 K[PtCl₃(C₂H₄)] 是经典例子

### 3.4 金属有机化合物的反应性

**热稳定性规律**：
- 无 β-H 的烷基配合物较稳定（如 CH₃、Ph、CH₂SiMe₃）
- 有 β-H 的配合物易通过 β-H 消除分解
- 金属-氢键（M-H）通常较强，是许多催化循环的关键

**亲电/亲核反应性**：
- 低氧化态、富电子金属中心：亲核性，易被氧化加成
- 高氧化态、缺电子金属中心：亲电性，易被还原消除

### 3.5 常见有机金属试剂

**Grignard 试剂（RMgX）**：
- 制备：RX + Mg → RMgX（无水乙醚或 THF）
- 结构：Schlenk 平衡 2RMgX ⇌ R₂Mg + MgX₂
- 反应性：强亲核/强碱，与醛酮加成、与酯反应两次、与 CO₂ 羧化
- 竞赛注意：格氏试剂对水、氧极敏感，制备需无水无氧条件

**有机锂试剂（RLi）**：
- 制备：RX + 2Li → RLi + LiX；或 R-H + n-BuLi → RLi + 丁烷
- 反应性：比格氏试剂更强亲核/更强碱
- 金属化反应（Li-H exchange）： directed ortho metalization（DoM）
- 竞赛常考：有机锂与羰基化合物的加成、与卤代烃的偶联

**有机铜试剂（Gilman 试剂，R₂CuLi）**：
- 制备：2RLi + CuI → R₂CuLi + LiI
- 反应性：温和亲核，对 α,β-不饱和羰基进行 1,4-共轭加成（Michael 加成）
- 不与酮的羰基直接反应（与格氏试剂区别）
- 竞赛核心：1,4- vs 1,2-加成的选择性

**有机锌试剂（R₂Zn 或 RZnX）**：
- 制备：RI + Zn → RZnI；或 2RLi + ZnX₂ → R₂Zn + 2LiX
- 反应性：温和，对官能团耐受性好
- Negishi 偶联：R-ZnX + R'-X → R-R'（Pd 催化）
- 竞赛注意：有机锌是 Negishi 偶联的核心试剂

**有机硼试剂（R-B(OR')₂ 或 RB(OH)₂）**：
- 制备：RB(OMe)₂ 水解；或硼氢化-氧化后转化
- 反应性：稳定、低毒、对水/空气不敏感
- Suzuki 偶联：R-B(OH)₂ + R'-X → R-R'（Pd/碱催化）
- 竞赛核心：Suzuki 是竞赛中最常考的交叉偶联

| 试剂 | 反应性 | 选择性 | 官能团耐受性 | 毒性 | 竞赛优先级 |
|:---|:---:|:---:|:---:|:---:|:---:|
| RMgX | 强 | 低 | 差 | 低 | ★★★ |
| RLi | 很强 | 低 | 很差 | 低 | ★★★ |
| R₂CuLi | 温和 | 高（1,4-加成）| 较好 | 低 | ★★★ |
| R₂Zn | 温和 | 高 | 好 | 低 | ★★ |
| R-B(OH)₂ | 很温和 | 很高 | 很好 | 很低 | ★★★★★ |

### 3.6 基元反应机理

过渡金属有机化合物的反应性由五种基元反应构成，它们是理解催化循环的"基本积木"。

#### （1）配体取代反应
- **18e 配合物**：需先解离配体形成 16e 不饱和中间体，再缔合新配体。
  - 例：[Cr(CO)₆] + L → [Cr(CO)₅L] + CO（热或光化学促进）
- **16e 配合物**：可直接缔合新配体形成 18e 过渡态。
  - 例：[IrCl(CO)(PPh₃)₂] + PEt₃ → [IrCl(CO)(PPh₃)₂(PEt₃)] → [IrCl(CO)(PPh₃)(PEt₃)] + PPh₃
- **空间位阻效应**：大位阻配体加速解离、抑制缔合（Tolman 锥角）。

#### （2）氧化加成（Oxidative Addition, OA）
- **定义**：X—Y 分子断裂并加合到金属中心，形成 M—X 和 M—Y 两个新键。
- **结果**：金属氧化态 +2，配位数 +2，电子数 +2。
- **典型底物**：H₂、R—X、Ar—X、H—X、C—H（活化）。
- **机理类型**：
  - **协同机理**（最常见）：X—Y 同时与金属成键，两进入配体最终处于**顺位**。
    - 例：H₂ 与 Pt(II) 或 Rh(I) 的加成，先形成 σ 络合物，再断裂 H—H 键。
  - **S_N2 机理**：金属亲核进攻 R—X 的碳，X 被取代后再配位到金属。
    - 两进入配体可处于**反位**，手性碳发生**构型翻转**。
  - **自由基机理**（罕见）：经自由基中间体进行。
- **常见金属**：d⁸ 16e 平面四方形配合物（Pd(0)、Pt(0)、Rh(I)、Ir(I)）最易发生。

#### （3）还原消除（Reductive Elimination, RE）
- **定义**：氧化加成的逆反应。金属上的两个配体（X 和 Y）偶合并脱离金属，形成 X—Y 键。
- **结果**：金属氧化态 −2，配位数 −2，电子数 −2。
- **立体化学要求**：消除的两个配体必须处于**顺位**。
- **与氧化加成的关系**：二者在催化循环中配对出现，如交叉偶联中：
  - Pd(0) + R—X → Pd(II)(R)(X)（氧化加成）
  - 转金属 → Pd(II)(R)(R') → R—R' + Pd(0)（还原消除）

#### （4）σ 键复分解（σ-Bond Metathesis）
- **发生条件**：前 d 区金属（Ti、Zr 等）电子不足，无法发生氧化加成（所有电子已参与成键）。
- **机理**：四元环过渡态，协同交换两个 σ 键片段。
  - 例：[Cp₂Zr(H)(Me)] + H₂ → [Cp₂Zr(H)₂] + CH₄
- **特点**：金属氧化态不变，不是氧化加成-还原消除的替代路径。

#### （5）迁移插入（Migratory Insertion）
- **定义**：配位在金属上的一个基团（如烷基、氢负离子）迁移到相邻的不饱和配体（如 CO、烯烃）上，同时金属的配位数减少 1（留出一个空位）。
- **电子数变化**：金属总电子数减少 2（从 18e → 16e 或 16e → 14e）。
- **氧化态**：金属氧化态**不变**。
- **机理证据**（¹³CO 标记实验）：
  - 在 [Mn(Me)(CO)₅] + ¹³CO 反应中，产物显示新进入的 ¹³CO 与酰基处于**顺位**。
  - 证明是**甲基迁移**到相邻 CO，而非 CO "插入" Mn—Me 键。
  - 因此虽然术语叫 "insertion"，实际机理是 **migration**。
- **立体化学**：迁移基团的构型保持。

##### 1,1-迁移插入
- X 基团迁移到与其以单键相连的配体原子上。
- 典型：烷基迁移到 CO → 形成酰基（acyl）。
  - "[Mn(Me)(CO)₅] + PPh₃ → [Mn(MeCO)(CO)₄(PPh₃)]"

##### 1,2-迁移插入
- X 基团迁移到与其相隔两个键的原子上。
- 典型：H 或 R 迁移到 η²-烯烃 → 形成烷基。
  - M—H + CH₂=CH₂ → M—CH₂CH₃（氢插入，即氢化反应的第一步）
- **逆反应**：β-H 消除（β-hydride elimination）
  - M—CH₂CH₃ → M—H + CH₂=CH₂
  - 要求：β 位有氢，且 M—H 与 β-C—H 处于**顺位**（syn）。

#### （6）β-H 消除
- **定义**：烷基配体 β 位的氢以 H⁻ 形式迁移到金属上，同时金属-碳键断裂，生成烯烃。
- **结构要求**：
  1. 烷基必须有 β-H（无 β-H 的烷基如 CH₃、新戊基、Ph 特别稳定）。
  2. 金属上必须有空配位点（或能解离出配体）。
  3. 消除过程通过**顺位（syn）**过渡态进行。
- **与 1,2-插入的关系**：二者互为微观可逆反应，共同构成烯烃异构化和聚合循环的核心步骤。

### 3.7 金属茂（Metallocenes）

#### （1）定义与基本结构
- **金属茂**：含两个 η⁵-环戊二烯基（Cp = C₅H₅）配体的夹心型金属有机化合物。
- **通式**：[MCp₂] 或 [MCp₂Lₙ]
- **最著名代表**：二茂铁 [Fe(Cp)₂]（1951 年发现，结构超出经典价键理论预期）。

#### （2）二茂铁的结构与成键
- **几何**：两个平行的 Cp 环将 Fe 夹在中间，Fe 位于环心连线上。
- **电子计数**：
  - Fe(II)：d⁶，贡献 6e
  - 每个 η⁵-Cp⁻：贡献 6e（离子法）或 5e（共价法，L₂X）
  - 总计：6 + 6 + 6 = **18e**（离子法）或 8 + 5 + 5 = 18e（共价法，Fe(0) 视角）
  - 实际上二茂铁是 **20e 化合物**（若按每个 Cp 贡献 6e，Fe 贡献 8e）—— 这是 18 电子规则的著名例外，因夹心结构和 Cp 环的芳香性而获得额外稳定化。
- **芳香性**：每个 Cp 环有 6 个 π 电子（5 个来自 C，1 个来自 Fe），满足 Hückel 4n+2 规则，具有芳香性。
- **价键模型**：
  - Fe 的 d 轨道与 Cp 环的 π 轨道相互作用，形成强共价-配位混合键。
  - 涉及 σ、π 甚至 δ 键成分（因 Cp 环的多原子轨道重叠）。

#### （3）二茂铁的反应性
- **亲电芳香取代**：Cp 环表现出苯类似的芳香性，可发生 Friedel-Crafts 酰基化等反应。
- **氧化**：可被氧化为二茂铁阳离子 [Fe(Cp)₂]⁺（蓝色，18e），用于电化学标准参考。
- **稳定与用途**：对空气稳定，可用作汽油抗爆剂、火箭燃料添加剂、不对称催化配体前体。

#### （4）其他金属茂
| 金属茂 | 电子数 | 自旋态 | 结构特点 |
|:---|:---:|:---:|:---|
| [Fe(Cp)₂] | 18/20* | 低自旋 | 平行夹心，最稳定 |
| [Co(Cp)₂] | 19 | 低自旋 | 倾斜夹心（因单电子排斥）|
| [Ni(Cp)₂] | 20 | 低自旋 | 倾斜夹心 |
| [Ti(Cp)₂Cl₂] | 16 | — |  bent 夹心（Cp-Ti-Cp 角 ~130°）|
| [Zr(Cp)₂Cl₂] | 16 | — | 烯烃聚合催化剂前体 |

*20e 若按中性计数，18e 若按 Fe(II) + 2Cp⁻ 计数。两种视角均可，但二茂铁确实是 18 电子规则的例外。

## 四、关键结论

1. **18 电子规则是稳定性的必要条件，但不是充分条件**——有些 16 电子配合物因几何约束而稳定（如 d⁸ 平面四边形配合物）。

2. **氧化态变化是催化循环的核心驱动力**——Pd(0)/Pd(II) 循环是交叉偶联的基础。

3. **无 β-H 的烷基配合物更稳定**——这是设计稳定金属有机化合物的基本原则（如 CH₃、新戊基、Ph）。

4. **有机硼酸是竞赛中 C-C 键形成的首选试剂**——稳定性、低毒性、条件温和使其在 Suzuki 偶联中独占鳌头。

5. **配体决定反应性，金属决定可能性**——膦配体的位阻和电子效应直接调控氧化加成/还原消除的速率。

## 五、常见分类

### 5.1 按金属分类

| 金属 | 常见氧化态 | 特征反应 | 代表催化剂 |
|:---|:---:|:---|:---|
| Pd | 0, +2 | 交叉偶联、Heck | Pd(PPh₃)₄, Pd(OAc)₂ |
| Ni | 0, +2 | 偶联、聚合 | NiCl₂(dppe) |
| Cu | +1, +2 | Ullmann、偶联 | CuI, Cu(OTf)₂ |
| Rh | +1, +3 | 氢化、氢甲酰化 | RhCl(PPh₃)₃ (Wilkinson) |
| Ru | +2, +4 | 复分解、氢化 | Grubbs 催化剂 |
| Ti | +2, +4 | 环丙烷化、McMurry | TiCl₄, Ti(OiPr)₄ |
| Cr | +2, +3 | NHK 反应、Fischer 卡宾 | CrCl₂ |

### 5.2 按配体类型分类

| 配体类别 | 代表 | 电子效应 | 在催化中的作用 |
|:---|:---|:---|:---|
| 膦配体 | PPh₃, BINAP, SPhos | σ 给予/π 接受可调 | 调节电子密度、控制立体化学 |
| NHC 配体 | IMes, SIMes | 强 σ 给予、弱 π 接受 | 稳定低价金属、提高催化活性 |
| 烯烃 | 乙烯、降冰片烯 | π 给予/π* 接受 | 参与配位与插入 |
| CO | — | 强 π 接受 | 稳定低氧化态、参与迁移插入 |
| 卤素 | Cl⁻, Br⁻, I⁻ | 弱 σ 给予 | 影响氧化加成速率、作为离去基 |

### 5.3 按反应类型分类

| 反应类别 | 典型反应 | 涉及的基元步骤 |
|:---|:---|:---|
| 交叉偶联 | Suzuki, Heck, Negishi, Stille | 氧化加成、转金属、还原消除 |
| 烯烃转化 | Wacker 氧化、氢甲酰化 | 配位、迁移插入、β-H 消除 |
| 复分解 | RCM, CM, ROMP | 金属杂环丁烷形成与断裂 |
| 环丙烷化 | Simmons-Smith, Kulinkovich | 卡宾/类卡宾转移 |
| 氢化 | 不对称氢化（Noyori）| 氧化加成、迁移插入、还原消除 |

## 六、适用条件

### 6.1 18 电子规则的适用范围

- **适用**：d 区过渡金属（族 3-12）的有机配合物
- **不适用**：
  - 主族金属有机化合物（如 AlMe₃，遵循 8 电子规则）
  - f 区元素（镧系、锕系，电子计数复杂）
  - 某些 d¹⁰ 配合物（如 Zn、Cd、Hg 的烷基化合物）
  - 高配位数的早期过渡金属（如 Ti、Zr 可超过 18e）

### 6.2 各有机金属试剂的使用条件

| 试剂 | 溶剂 | 温度 | 对水氧敏感性 | 特殊要求 |
|:---|:---|:---:|:---:|:---|
| RMgX | 无水 Et₂O, THF | 室温-回流 | 极高 | 严格无水无氧 |
| RLi | 无水 THF, 己烷 | -78°C ~ 0°C | 极高 | 低温操作 |
| R₂CuLi | 无水 Et₂O, THF | -78°C ~ -20°C | 高 | 现制现用 |
| R₂Zn | 无水 THF | 0°C ~ 室温 | 中等 | 可短期暴露 |
| R-B(OH)₂ | 水/有机混合 | 室温-80°C | 低 | 需碱活化 |

## 七、常见比较

### 7.1 共价法 vs 离子法电子计数

| 特征 | 共价法 | 离子法 |
|:---|:---|:---|
| 金属电子来源 | 中性原子（族数）| 按氧化态的 d 电子数 |
| 配体处理 | 中性配体，贡献成键电子 | 带电配体，按电荷计算 |
| 直观性 | 更直观 | 需先确定氧化态 |
| 适用范围 | 有机金属化学常用 | 配位化学常用 |
| 结果 | 一致 | 一致 |

### 7.2 Fischer 卡宾 vs Schrock 卡宾

| 特征 | Fischer 卡宾 | Schrock 卡宾 |
|:---|:---|:---|
| 金属氧化态 | 低（0 或 +1）| 高（+4 或 +5）|
| 配体类型 | π 受体为主（CO）| σ 给予体为主（烷基）|
| 卡宾碳性质 | 亲电 | 亲核 |
| 典型金属 | Cr, W, Fe | Ti, Ta, Mo |
| 合成方法 | 金属羰基 + 亲电试剂 | 金属烷基 + α-H 消除 |

### 7.3 有机铜 vs 有机锌 vs 有机硼

| 特征 | Gilman 试剂 (R₂CuLi) | 有机锌 (R₂Zn) | 有机硼酸 (RB(OH)₂) |
|:---|:---|:---|:---|
| 反应性 | 温和 | 温和 | 很温和 |
| 主要反应 | 1,4-共轭加成 | Negishi 偶联 | Suzuki 偶联 |
| 对羰基反应 | 选择性 1,4-加成 | 较少直接加成 | 不直接反应 |
| 官能团耐受性 | 较好 | 好 | 很好 |
| 毒性 | 低 | 低 | 很低 |
| 竞赛出现频率 | ★★★ | ★★ | ★★★★★ |

## 十、例题

### 例题 1（电子计数）

**题目**：计算下列配合物的价电子数，并判断是否满足 18 电子规则。

(1) Fe(CO)₅
(2) [Mn(CO)₅]⁻
(3) (η⁵-Cp)₂Fe（二茂铁）
(4) Pd(PPh₃)₄

**解答**：

(1) Fe(CO)₅：Fe 为第 8 族，贡献 8e；5 个 CO 各贡献 2e。总计 8 + 10 = **18e**，满足 18 电子规则。

(2) [Mn(CO)₅]⁻：Mn 为第 7 族，贡献 7e；5 个 CO 贡献 10e；负电荷 +1e。总计 7 + 10 + 1 = **18e**，满足 18 电子规则。

(3) (η⁵-Cp)₂Fe：Fe 贡献 8e；每个 η⁵-Cp 贡献 6e。总计 8 + 12 = **20e**。二茂铁是 20 电子配合物，但因其夹心结构和芳香性而稳定，是 18 电子规则的例外。

(4) Pd(PPh₃)₄：Pd 为第 10 族，贡献 10e；4 个 PPh₃ 各贡献 2e。总计 10 + 8 = **18e**，满足 18 电子规则。但在溶液中会解离 1 个 PPh₃，形成 16e 的 Pd(PPh₃)₃，这是其催化活性的来源。

**点评**：二茂铁是经典例外，需牢记；Pd(PPh₃)₄ 的配体解离是理解催化循环的关键。

---

### 例题 2（氧化态与配体分类）

**题目**：确定下列配合物中金属的氧化态，并指出各配体的类型（X/L/LX）。

(1) [Pd(PPh₃)₂(Ph)(Br)]
(2) [RhCl(PPh₃)₃]（Wilkinson 催化剂）
(3) (η⁵-Cp)TiCl₃

**解答**：

(1) [Pd(PPh₃)₂(Ph)(Br)]：
- PPh₃：L 型（中性，0）
- Ph（苯基）：X 型（-1）
- Br：X 型（-1）
- 配合物中性，设 Pd 氧化态为 x：x + 0×2 + (-1) + (-1) = 0，x = **+2**

(2) [RhCl(PPh₃)₃]：
- Cl：X 型（-1）
- PPh₃：L 型（0），共 3 个
- 配合物中性：x + (-1) + 0×3 = 0，x = **+1**

(3) (η⁵-Cp)TiCl₃：
- η⁵-Cp：LX 型（按离子法视为 Cp⁻，-1；按共价法贡献 6e）
- Cl：X 型（-1），共 3 个
- 配合物中性：x + (-1) + (-1)×3 = 0，x = **+4**

**点评**：η⁵-Cp 的归类在不同教材中有差异，竞赛中按 LX（6e）或 X（1- 电荷）均可，关键是电子计数和氧化态计算自洽。

---

### 例题 3（有机金属试剂选择性）

**题目**：化合物 A 为环己烯酮，分别与下列试剂反应，写出主要产物结构。

(1) MeMgBr（1 当量）
(2) Me₂CuLi（1 当量）
(3) MeLi（1 当量）

**解答**：

```
      O                          O
      ‖                          ‖
   ┌──┴──┐                    ┌──┴──┐
   │     │    + MeMgBr  →     │     │
   └──┬──┘                    └──┬──┘
      │    1,2-加成               │
      ▼                          ▼
   OH  Me（1,2-产物，烯丙醇）
```

(1) **MeMgBr**：强亲核试剂，与羰基发生 **1,2-加成**，产物为 1-甲基-1-羟基环己-2-烯（烯丙醇）。

(2) **Me₂CuLi**：有机铜试剂，软亲核试剂，对 α,β-不饱和羰基发生 **1,4-共轭加成**，产物为 3-甲基环己酮。

(3) **MeLi**：比格氏试剂更强的亲核试剂，主要发生 **1,2-加成**，产物与 (1) 类似，但反应更快、产率可能更高。

**关键区分**：
- 硬亲核试剂（RLi, RMgX）→ 1,2-加成（动力学控制，进攻羰基碳）
- 软亲核试剂（R₂CuLi）→ 1,4-加成（热力学控制，进攻 β-碳）

## 十一、易错点

| 易错点 | 错误认知 | 正确理解 |
|:---|:---|:---|
| 18 电子规则的绝对化 | "所有稳定配合物都满足 18e" | 16e 配合物（如 d⁸ 平面四边形）可稳定存在；早期过渡金属可超 18e |
| 氧化态计算混乱 | "CO 是 -2，PR₃ 是 -3" | CO、PR₃ 等中性配体不改变氧化态；只有 X 型配体（Cl⁻, R⁻, H⁻）使氧化态 +1 |
| Fischer vs Schrock 混淆 | "Fischer 卡宾亲核，Schrock 亲电" | 恰恰相反：Fischer 卡宾碳亲电（低氧化态金属），Schrock 卡宾碳亲核（高氧化态金属）|
| 有机铜的 1,2-加成 | "有机铜与格氏试剂一样加成羰基" | Gilman 试剂主要做 1,4-共轭加成，不直接加成饱和酮 |
| 二茂铁电子数 | "二茂铁 18e" | 二茂铁是 20e，但因芳香性稳定，是例外 |
| Pd(0) 与 Pd(II) 的混淆 | "Pd 催化循环中氧化态不变" | Pd(0) ↔ Pd(II) 是交叉偶联的核心；Cu(I)/Cu(II) 在 Wacker 中同理 |
| 配体解离忽视 | "Pd(PPh₃)₄ 直接参与氧化加成" | 溶液中先解离为 Pd(PPh₃)₃（16e），再与底物配位 |

## 十二、🎯 教学视角
### 12.1 学习路径

**前置要求**：
1. [[配合物]]——配位键、配位数、配位几何
2. [[分子轨道理论]]——d 轨道分裂、π 反馈键
3. [[氧化还原反应]]——氧化态的定义与计算
4. [[Grignard试剂]]——主族金属有机化学基础

**本章学习顺序**：
```
18 电子规则 → 配体分类（X/L）→ 氧化态确定 → 金属-碳键类型
    ↓
常见有机金属试剂（格氏、锂、铜、锌、硼）
    ↓
基元反应（氧化加成、还原消除、迁移插入、β-H 消除）
    ↓
催化循环入门（各反应的组合应用）
```

**后续延伸**：
- [[催化循环入门]]——将基元反应组合成完整循环
- [[Suzuki偶联]]、[[Heck反应]]——具体反应的机理与应用
- [[全合成]]——金属有机反应在复杂分子合成中的战略地位

### 12.2 认知误区

**误区 1："金属有机就是有机金属，没有区别"**
- 辨析：Organometallic 强调 M-C 键；Metal-organic 广义上包括配位化合物。竞赛语境下二者常混用，但严格来说金属有机化学是研究 M-C 键的分支。

**误区 2："18 电子规则像 8 电子规则一样严格"**
- 辨析：18 电子规则是经验规则，有众多例外（二茂铁 20e、16e 平面四边形配合物、早期过渡金属超配位）。它更像"指导原则"而非"定律"。

**误区 3："氧化态越高，配合物越活泼"**
- 辨析：反应性取决于具体反应类型。氧化加成需要低氧化态金属（富电子），还原消除需要高氧化态金属。催化循环正是利用氧化态的升降实现反应。

**误区 4："有机硼试剂不活泼，所以不好用"**
- 辨析：有机硼的"不活泼"恰恰是其优势——稳定、低毒、官能团耐受性好。在 Pd 催化和碱活化下，有机硼酸是最高效的 C-C 键形成试剂之一。

### 12.3 入门例题

**入门题 1**：判断下列配体属于 X 型、L 型还是 LX 型，并给出电子贡献数。

(1) H⁻  (2) CO  (3) η⁵-Cp⁻  (4) η²-C₂H₄  (5) CH₃⁻

**答案**：
(1) X 型，1e  
(2) L 型，2e  
(3) LX 型，6e（或 X 型，按 Cp⁻ 计）
(4) L 型，2e  
(5) X 型，1e

**入门题 2**：计算 Cr(CO)₆ 的价电子数。Cr 为第 6 族。

**答案**：Cr(6) + 6×CO(2) = 18e，满足 18 电子规则。

**入门题 3**：解释为什么 (CH₃)₃C-CH₂-MgBr（新戊基格氏试剂）比 n-BuMgBr 更稳定。

**答案**：新戊基没有 β-H（β-碳为季碳），无法发生 β-H 消除分解；而正丁基有 β-H，在加热条件下易消除生成 1-丁烯。
