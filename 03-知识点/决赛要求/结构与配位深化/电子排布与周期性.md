## 一、定义

**电子排布与周期性**研究多电子原子中电子在轨道上的填充规律（Aufbau原理、Pauli原理、Hund规则），以及由此产生的元素性质周期性变化。决赛阶段要求不仅掌握"怎么排"，更要理解"为什么这么排"——即钻穿效应、屏蔽效应、交换能等因素如何共同决定能级顺序和电子构型。

## 三、核心原理

### 3.1 决定电子排布的三个原理

**Pauli不相容原理**：同一原子中不能有两个电子具有完全相同的四个量子数。一个轨道最多容纳2个自旋相反的电子。

**Aufbau原理（构造原理）**：按能量从低到高的顺序填充轨道。实际顺序由[[钻穿效应]]和[[屏蔽效应]]共同决定：

$$1s < 2s < 2p < 3s < 3p < 4s < 3d < 4p < 5s < 4d < 5p < 6s < 4f approx 5d < 6p$$

**Hund规则**：
1. 电子优先以平行自旋占据不同的简并轨道（最大S）
2. 在S相同的情况下，L最大的态能量最低
3. 半满前J最小的态最低；半满后J最大的态最低

### 3.2 反常排布的物理解释

| 元素 | 反常排布 | 原因 |
|-----|---------|------|
| Cr | $[Ar]3d^54s^1$ 而非 $3d^44s^2$ | 半满d层（$d^5$）有额外交换能稳定化 |
| Cu | $[Ar]3d^{10}4s^1$ 而非 $3d^94s^2$ | 全满d层（$d^{10}$）有额外稳定化 |
| Mo | $[Kr]4d^55s^1$ | 同Cr，半满d层稳定 |
| Pd | $[Kr]4d^{10}$ | 全满d层极稳定，甚至不填5s |

**交换能**（Hund规则第一条的能量根源）：平行自旋电子间的交换作用使能量降低。平行自旋对数越多，交换能越低。

### 3.3 周期性的电子结构根源

| 周期性 | 电子结构根源 |
|-------|------------|
| 周期数 = 最外层主量子数 | 每填满一层开始新周期 |
| 主族元素 | s/p区，价电子在最外层 |
| 过渡元素 | d区，(n-1)d电子逐步填充 |
| 镧系/锕系 | f区，(n-2)f电子逐步填充 |
| 原子半径递减 | 同周期$Z_{eff}$增加 |
| 电离能增大 | 同周期$Z_{eff}$增加 |
| 电负性增大 | 同周期$Z_{eff}$增加 |

## 四、关键结论

1. **电子排布不是死记**，而是理解钻穿→能级→填充→稳定化的逻辑链
2. **半满和全满有额外稳定化**：交换能 + 对称性
3. **离子排布≠原子排布**：失去电子时总是先失最外层（n最大），即使先填时4s先于3d
4. **镧系收缩**：4f电子屏蔽极差，导致后续过渡元素半径异常小

## 五、典型竞赛题型

### 题型1：书写基态电子排布

**例**：写出Ag（Z=47）的基态电子排布。

**答**：$[Kr]4d^{10}5s^1$。类似Cu，全满d层有额外稳定化。

### 题型2：解释反常排布

**例**：为什么Cr是$3d^54s^1$而不是$3d^44s^2$？

**答**：$d^5$为半满构型，有6对平行自旋（$C_5^2=10$对？不对，重新算：5个电子各占一个d轨道，平行自旋，交换对数为$C_5^2=10$）。而$d^4$只有$C_4^2=6$对交换。额外的交换能使$3d^54s^1$更稳定。此外，半满构型的电子云分布更对称，也贡献稳定性。
