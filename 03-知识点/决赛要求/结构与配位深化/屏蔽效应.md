## 一、定义

**屏蔽效应**（Shielding Effect）是指多电子原子中，内层电子对外层电子的排斥作用，部分抵消了原子核正电荷对最外层电子的吸引，使外层电子感受到的有效核电荷 $Z_{eff}$ 小于实际核电荷 $Z$ 的现象。

与[[钻穿效应]]共同决定了多电子原子中轨道的能级顺序和电子排布规律。

## 三、核心原理

### 3.1 有效核电荷

$$Z_{eff} = Z - sigma$$

- $Z$：原子序数（核电荷）
- $sigma$：屏蔽常数（被内层电子屏蔽掉的核电荷）
- $Z_{eff}$：有效核电荷（外层电子实际感受到的核电荷）

### 3.2 Slater规则（屏蔽常数计算）

Slater规则提供了一种估算屏蔽常数的半经验方法：

**分组顺序**：$(1s), (2s,2p), (3s,3p), (3d), (4s,4p), (4d), (4f), dots$

**对给定电子的屏蔽常数$sigma$**：

| 屏蔽电子位置 | 贡献值 |
|------------|-------|
| 同组（右侧不算） | 0.35（1s组为0.30） |
| $n-1$ 壳层 | 0.85 |
| $n-2$ 及更内层 | 1.00 |
| 对于d或f电子：所有左侧组 | 1.00 |

**示例**：计算Sc（Z=21, $[Ar]3d^14s^2$）中4s电子的$Z_{eff}$
- 同组（4s）：1个其他4s电子 → $1 	imes 0.35 = 0.35$
- n-1壳层（3s,3p,3d）：9+1=10个电子 → $10 	imes 0.85 = 8.50$
- n-2及更内层（1s,2s,2p）：10个电子 → $10 	imes 1.00 = 10.00$
- $sigma = 0.35 + 8.50 + 10.00 = 18.85$
- $Z_{eff} = 21 - 18.85 = 2.15$

### 3.3 屏蔽效应的周期性变化

**同一周期（从左到右）**：
- 核电荷Z增加，但新增电子进入同一层
- 同层电子相互屏蔽较弱（$sigma approx 0.35$）
- $Z_{eff}$ 显著增加 → 原子半径减小、电离能增大

**同一族（从上到下）**：
- 新增一层电子，内层屏蔽增强
- $Z_{eff}$ 增加缓慢 → 原子半径增大、电离能减小

### 3.4 与钻穿效应的关系

| | 屏蔽效应 | 钻穿效应 |
|--|---------|---------|
| **方向** | 升高轨道能量 | 降低轨道能量 |
| **本质** | 电子间排斥（被动） | 电子云穿透（主动） |
| **决定因素** | 内层电子数 | 轨道形状（l值） |
| **竞争结果** | 共同决定能级顺序 | 共同决定能级顺序 |

## 四、关键结论

1. **屏蔽常数$sigma$越大**，$Z_{eff}$越小，电子能量越高（越不稳定）
2. **同层电子相互屏蔽弱**（0.35），因此同周期$Z_{eff}$增加显著
3. **内层电子屏蔽强**（1.00），因此新增壳层导致半径骤增
4. **d电子屏蔽能力差**：d电子弥散，对内层屏蔽弱；同时d电子被外层s电子屏蔽强
5. **Slater规则为半经验**：简化模型，定量结果有误差，但定性趋势可靠

## 五、典型竞赛题型

### 题型1：Slater规则计算$Z_{eff}$

**例**：用Slater规则计算Fe（Z=26）中3d和4s电子的$Z_{eff}$，解释为什么Fe原子先填4s后填3d。

**解**：
- Fe: $[Ar]3d^64s^2$
- 对4s电子：$sigma = 1 	imes 0.35 + 14 	imes 0.85 + 10 	imes 1.00 = 22.25$，$Z_{eff} = 26 - 22.25 = 3.75$
- 对3d电子：$sigma = 5 	imes 0.35 + 18 	imes 1.00 = 19.75$，$Z_{eff} = 26 - 19.75 = 6.25$

注意：虽然单个3d电子感受到的$Z_{eff}$更大，但3d电子之间的排斥（6个d电子 crowded在较小空间）使整体能量升高。在Fe原子中，4s的钻穿效应使$E_{4s} < E_{3d}$，因此先填4s。

### 题型2：解释周期趋势

**例**：为什么Na的电离能远小于Mg，但Mg²⁺到Al³⁺的电离能跃迁不如Na⁺到Mg²⁺大？

**答**：Na失去1个3s电子后，Na⁺为Ne型稳定结构。Mg失去2个3s电子后，Mg²⁺也为Ne型。但Al的3p电子受到3s和3p电子的屏蔽，且3p钻穿能力弱于3s，因此$Z_{eff}$较小，电离能增长放缓。

## 七、易错点

1. **Slater规则分组**：3d和4s/4p分属不同组，(3d)在(4s,4p)左边
2. **d/f电子的特殊处理**：计算d或f电子的$sigma$时，左侧所有组都贡献1.00
3. **同组电子数**：计算某电子的$sigma$时，同组中"该电子本身"不计入，右侧电子不计入
4. **$Z_{eff}$与能量的关系**：$Z_{eff}$大意味着能量低（更稳定），但对不同主量子数的轨道不能直接比较
