# d轨道分裂

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-结构与配位深化]]
- 对应考纲条目：[[决赛03-配合物深化]]
- 关联笔记：[[晶体场分裂与磁性]]

## 一、定义

在配合物中，中心金属离子的5个简并d轨道由于配体场的静电排斥作用，能量发生分裂，形成不同能级的d轨道组。这是晶体场理论和配位场理论的核心内容。

## 二、核心原理

### 2.1 八面体场中的分裂

八面体配合物中，配体沿 $x, y, z$ 轴方向接近中心离子：

- **$e_g$ 轨道**（高能组）：$d_{z^2}$ 和 $d_{x^2-y^2}$，电子云直接指向配体，受排斥较强
- **$t_{2g}$ 轨道**（低能组）：$d_{xy}$、$d_{xz}$、$d_{yz}$，电子云位于配体之间，受排斥较弱

**分裂能记号**：$\Delta_o$（或 $10Dq$）

**能量重心守恒**：
$$2E(e_g) + 3E(t_{2g}) = 0 \Rightarrow E(e_g) = +0.6\Delta_o, \quad E(t_{2g}) = -0.4\Delta_o$$

### 2.2 四面体场中的分裂

四面体场可看作立方体中交替四个顶点的配体场：

- **$e$ 轨道**（低能）：$d_{z^2}$、$d_{x^2-y^2}$
- **$t_2$ 轨道**（高能）：$d_{xy}$、$d_{xz}$、$d_{yz}$

分裂方向与八面体相反：$\Delta_t \approx \frac{4}{9}\Delta_o$

### 2.3 光谱化学序列

配体按分裂能力从弱到强排列：

$$\text{I}^- < \text{Br}^- < \text{SCN}^- < \text{Cl}^- < \text{NO}_3^- < \text{F}^- < \text{OH}^- < \text{C}_2\text{O}_4^{2-} < \text{H}_2\text{O} < \text{NCS}^- < \text{NH}_3 < \text{en} < \text{bipy} < \text{phen} < \text{NO}_2^- < \text{PPh}_3 < \text{CN}^- < \text{CO}$$

**规律**：
- 卤素 < 氧配体 < 氮配体 < 碳配体
- $\pi$-受体配体（CO、CN⁻）分裂能力最强

## 三、典型题型

1. **绘制不同配位场的d轨道能级分裂图**
2. **根据光谱化学序列判断分裂能大小**
3. **比较配合物的颜色与分裂能的关系**：$\Delta = hc/\lambda$

## 五、常见分类或情形

- **八面体场**：最常见、最标准的分裂模型，适合训练高低自旋与 CFSE 判断
- **四面体场**：分裂方向与八面体相反，且 $\Delta_t$ 较小，通常更易高自旋
- **平方平面场**：常见于 d$^8$ 后过渡金属体系，分裂极不均匀，是强场低自旋的重要来源
- **畸变场**：Jahn-Teller 畸变会让本已分裂的轨道进一步失去简并

## 六、适用条件与限制

- 纯晶体场理论把配体视作点电荷，只能给出一阶静电图像，无法完整描述共价成分
- 对 $\pi$-供体、$\pi$-受体配体较多的体系，仅靠静电排斥不足以解释全部分裂趋势
- 分裂图只能告诉你轨道相对高低，真正电子排布还要与成对能比较后才能判断高低自旋
- 竞赛题中若给出具体配体类型、金属氧化态和几何构型，必须三者一起看，不能只背光谱化学序列

## 七、常见比较与易混点

- **$e_g/t_{2g}$ vs $e/t_2$**：前者属于八面体场，后者属于四面体场，符号和高低顺序都不同
- **分裂能大 vs 一定低自旋**：只有当 $\Delta$ 大于成对能时，低自旋才真正成立
- **晶体场理论 vs 配位场理论**：前者偏静电模型，后者把金属-配体轨道相互作用也纳入
- **颜色深浅 vs 分裂能大小**：颜色不只由 $\Delta$ 决定，还受允许/禁阻跃迁和摩尔吸光系数影响

## 八、与其他知识点的联系

- 是[[晶体场分裂与磁性]]的基础页，后者更偏电子排布、磁矩与颜色应用
- 与[[配合物稳定性]]、[[光谱化学序列]]、[[高自旋与低自旋]]直接相连
- 与[[d轨道]]、[[分子轨道理论深化]]和配位场理论有理论承接关系
- 与[[原子光谱与光谱项]]也有关联：自由离子项进入配体场后会进一步分裂

## 九、典型题型

1. **分裂图绘制题**：根据几何构型画出八面体、四面体或平方平面场的 d 轨道能级图
2. **分裂能大小规律题**：比较不同金属、不同氧化态或不同配体下的 $\Delta$
3. **高低自旋判断题**：结合 $\Delta$ 与成对能推断电子排布和磁性
4. **颜色/吸收波长解释题**：利用 $\Delta = hc/\lambda$ 建立配位场颜色直觉

## 十、例题

**例**：为什么 [Fe(CN)$_6$]$^{4-}$ 常为低自旋，而 [FeF$_6$]$^{4-}$ 更倾向高自旋？

**分析路径**：
1. 两者金属中心与氧化态相同，先锁定决定差异的主因是配体场强
2. 用光谱化学序列比较 CN$^-$ 与 F$^-$ 的分裂能力
3. 判断 $\Delta$ 与成对能大小关系

**结论**：CN$^-$ 是强场配体，使 $\Delta_o$ 更大，电子更倾向先在低能 $t_{2g}$ 成对，形成低自旋；F$^-$ 是弱场配体，$\Delta_o$ 较小，更易保持高自旋排布。

**反思**：这类题的关键顺序是“**先几何，再场强，再比成对能**”

## 十一、易错点

- 把四面体场仍按八面体的高低顺序去填电子
- 只背光谱化学序列，不会把它转化成高低自旋和颜色判断
- 认为所有 d$^8$ 配合物都平方平面或所有四面体配合物都低自旋
- 看到颜色就只联想到“有 d-d 跃迁”，却不去判断跃迁是否拉波特禁阻或自旋禁阻

## 十二、🎯 教学视角

- 这页最适合当作“**配位化学从形状走向电子结构**”的入口
- 课堂上可先固定两张母图：八面体与四面体，再逐步引出高低自旋和光谱化学序列
- 学生最容易混的是“谁高谁低”和“什么时候低自旋”，建议所有例题都强制画能级图
- 若时间有限，可优先掌握八面体场，因为多数竞赛推断题都从它展开

## 十三、竞赛拓展

- 平方平面 d$^8$、Jahn-Teller 畸变和配位场稳定化能（CFSE）是常见深化方向
- $\pi$-受体配体（CO、CN$^-$）与 $\pi$-供体配体对 $t_{2g}$ 的影响，可进一步连到配位场理论
- Tanabe-Sugano 图是在 d 轨道分裂基础上讨论自旋允许跃迁的重要工具
- 更高阶可继续连到配位场理论与金属-配体 MO 图的统一解释
