# 配位平衡

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-结构与配位深化]]
- 对应考纲条目：[[决赛03-配合物深化]], [[决赛12-配合物]]
- 关联笔记：[[配合物稳定性]], [[溶度积]]

## 一、定义

配位平衡是指溶液中金属离子与配体形成配合物的可逆过程达到平衡时的状态。通常用稳定常数（形成常数）$K_f$ 或不稳定常数 $K_d$ 来定量描述。

## 二、核心原理

### 2.1 逐级稳定常数与累积稳定常数

对于 $M + nL \rightleftharpoons ML_n$：

**逐级稳定常数**：
$$K_1 = \frac{[ML]}{[M][L]}, \quad K_2 = \frac{[ML_2]}{[ML][L]}, \quad \dots$$

**累积稳定常数**：
$$\beta_n = \frac{[ML_n]}{[M][L]^n} = K_1 K_2 \cdots K_n$$

### 2.2 常见配合物的稳定常数

| 配合物 | $\lg \beta$ | 说明 |
|--------|------------|------|
| $[\text{Ag(NH}_3)_2]^+$ | 7.05 | 弱配体 |
| $[\text{Ag(CN)}_2]^-$ | 21.1 | 强配体 |
| $[\text{Fe(CN)}_6]^{4-}$ | 35 | 极稳定 |
| $[\text{Cu(NH}_3)_4]^{2+}$ | 12.6 | 常见配合物 |
| $[\text{EDTA}]^{4-}$ 配合物 | 15–25 | 螯合效应显著 |

### 2.3 配位平衡的竞争与移动

**与沉淀平衡的竞争**：
$$\text{AgCl}(s) + 2\text{NH}_3 \rightleftharpoons [\text{Ag(NH}_3)_2]^+ + \text{Cl}^-$$

总平衡常数 $K = K_{\text{sp}} \cdot \beta_2$，利用此可判断沉淀能否溶于配位剂。

**与酸碱平衡的竞争**：
配体若为弱酸根（如 $F^-$、$C_2O_4^{2-}$），pH降低会使配体质子化，导致配位平衡向左移动（配合物解离）。

### 2.4 螯合效应

多齿配体（如 en、EDTA）形成的螯环配合物比单齿配体更稳定。原因：
1. **熵增效应**：多齿配体取代多个单齿配体，粒子数增加
2. **结构因素**：螯环（五元环、六元环最稳定）减少了配合物的溶剂化损失

## 三、典型题型

1. **配位平衡与沉淀平衡的竞争计算**
2. **由分布系数求各物种浓度**：利用累积稳定常数
3. **EDTA滴定中的条件稳定常数**：$K'_f = K_f / \alpha_Y$

## 五、常见分类或情形

- **逐级配位情形**：金属离子逐步接受配体，常用于写 $K_1, K_2, \dots$ 与分布分数
- **沉淀-配位竞争情形**：配位剂通过降低自由金属离子浓度促使沉淀溶解
- **酸碱-配位竞争情形**：配体被质子化后配位能力下降，条件稳定常数显著变小
- **多配体竞争情形**：不同配体同时存在时，由稳定常数、浓度和质子化程度共同决定主导物种

## 六、适用条件与限制

- 稳定常数表中的数值通常对应特定温度、离子强度和介质，换条件后不能生搬硬套
- 配位平衡题中真正参与计算的常是**条件稳定常数**，而不是文献给出的理想 $\beta$
- 当配体存在多级质子化、金属水解或沉淀副反应时，单一平衡式往往不够
- 极大或极小稳定常数的体系可做近似，但必须先说明哪一步是主要控制平衡

## 七、常见比较与易混点

- **逐级稳定常数 $K_n$ vs 累积稳定常数 $\beta_n$**：前者对应单步加一个配体，后者对应从游离金属一步到最终配合物
- **热力学稳定性 vs 动力学惰性**：平衡常数大不代表配体交换一定慢
- **稳定常数大 vs 主体物种占绝对优势**：若自由配体浓度很低或被强烈质子化，主配合物仍可能不占优
- **EDTA 稳定常数 vs 条件稳定常数**：后者必须扣除 Y$^{4-}$ 分数 $\alpha_Y$

## 九、典型题型

1. **逐级/累积常数换算题**：由 $K_n$ 求 $\beta_n$，或反向分解
2. **沉淀是否溶解题**：把 $K_{sp}$ 与配位稳定常数组合成总平衡判断
3. **条件稳定常数题**：考虑 pH、配体质子化和副反应后判断滴定或分离可行性
4. **主导物种判断题**：在给定金属、配体和 pH 条件下判断主要存在形式

## 十、例题

**例**：为什么 AgCl 沉淀能溶于氨水，而不一定能溶于同浓度的弱配位剂？

**分析路径**：
1. 写出 $AgCl(s) \rightleftharpoons Ag^+ + Cl^-$ 的溶解平衡
2. 再写 $Ag^+ + 2NH_3 \rightleftharpoons [Ag(NH_3)_2]^+$ 的配位平衡
3. 用配位作用降低自由 $Ag^+$ 浓度，判断总平衡是否被强烈推向右侧

**结论**：氨与 Ag$^+$ 的配位稳定常数足够大，使自由银离子浓度显著下降，从而推动 AgCl 继续溶解；若配位剂太弱，则这种拉动作用不足。

**反思**：这类题的统一思路是“**谁能更有效地吃掉自由金属离子**”

## 十一、易错点

- 把 $K_1, K_2$ 和 $\beta_2$ 混成一个量
- 只盯稳定常数大小，不去看自由配体浓度与配体是否被质子化
- 计算沉淀-配位竞争时忘记把两个平衡常数相乘
- 看到 EDTA 就直接套总稳定常数，忽略不同 pH 下 Y$^{4-}$ 分数可能极低
