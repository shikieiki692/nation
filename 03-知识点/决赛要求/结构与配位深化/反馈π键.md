# 反馈π键（Back-bonding / d→π* Feedback）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-结构与配位深化]]
- 对应考纲条目：[[决赛03-配合物深化]]

## 一、定义

**反馈π键（Back-bonding）**是指：中心金属原子将其 filled d 轨道（或杂化轨道）上的电子**反馈**给配体的空 π* 反键轨道（或空的 p/d 轨道），形成的**π型配位键**。

- 与正常的 σ 配位键方向相反：σ 键是配体→金属的电子给予，反馈π键是金属→配体的电子回馈
- 两者协同作用，** synergistically 增强配合物稳定性**

## 三、核心原理

### 3.1 形成条件

反馈π键的形成需要同时满足：

1. **金属条件**：中心金属具有 filled 或 partially filled 的 d 轨道（通常为低氧化态、后过渡金属）
2. **配体条件**：配体具有**能量较低的空 π* 轨道**（或空 p 轨道），能接受电子
3. **对称性匹配**：金属 d 轨道与配体 π* 轨道具有合适的对称性，可有效重叠

### 3.2 典型配体类型

| 配体类型 | 代表配体 | 空轨道类型 | 反馈π键强度 | 场强 |
|:---|:---|:---|:---:|:---:|
| **π-酸配体 (π-acid)** | CO, CN⁻, NO⁺, N₂ | π* 反键轨道 | **强** | **强场** |
| **膦类配体** | PR₃, PPh₃ | σ* 或空 d 轨道 | 中等 | 中强场 |
| **烯烃/炔烃** | C₂H₄, C₂H₂ | π* 反键轨道 | 中等 | 弱-中场 |
| **卡宾/卡拜** | :CR₂, CR | p 空轨道 | 强 | 强场 |
| **卤素/含氧阴离子** | Cl⁻, OH⁻, H₂O | —（无空π*轨道）| **无** | 弱场 |

### 3.3 对光谱化学序列的解释

反馈π键是理解**光谱化学序列**的关键：

- **强场配体**（CO, CN⁻）：空的 π* 轨道能量低，能有效接受金属 d 电子 → 强烈的 d→π* 反馈 → t₂g 能量显著降低 → Δ₀ 增大
- **弱场配体**（I⁻, Br⁻, Cl⁻）：filled p 轨道与金属 d 轨道形成 π 键（给电子方向相反）→ t₂g 能量升高 → Δ₀ 减小
- **中间配体**（H₂O, NH₃）：无显著 π 相互作用 → Δ₀ 适中

### 3.4 对配合物性质的影响

| 性质 | 反馈π键的作用 |
|:---|:---|
| **稳定性** | 增强M–L键，降低配体解离速率 |
| **键长** | M–L键缩短（反馈π键增加键级） |
| **振动光谱** | 配体内部键（如C≡O）键长增加、伸缩频率降低（电子进入π*轨道削弱配体内键）|
| **氧化态** | 使低氧化态金属稳定（如Fe(0)在Fe(CO)₅中） |

## 四、关键结论

1. **反馈π键是强场配体之所以强的根本原因**
2. **CO是最经典的π-酸配体**：C–O键因反馈π键而减弱（红外频率从2143降至~1900 cm⁻¹），但M–C键显著增强
3. **低氧化态金属更易形成反馈π键**：因为d电子更富集
4. **反馈π键与σ配位键协同**：配体→金属的σ给予使金属富电子，促进金属→配体的π反馈；π反馈又使金属缺电子，促进σ给予

## 五、常见分类或情形

### 按配体分类

| 配体 | 反馈π键类型 | 典型配合物 | 特征现象 |
|:---|:---|:---|:---|
| CO | d→π* | Ni(CO)₄, Fe(CO)₅ | ν(CO) 显著降低 |
| CN⁻ | d→π* | [Fe(CN)₆]⁴⁻ | 强场，低自旋 |
| N₂ | d→π* | [Ru(NH₃)₅N₂]²⁺ | N≡N键活化 |
| 烯烃 | d→π* | Zeise盐 [PtCl₃(C₂H₄)]⁻ | C=C键伸长 |
| PR₃ | d→σ* | [RhCl(PPh₃)₃] (Wilkinson催化剂) | 配体场强可调 |

## 六、适用条件与限制

1. **需要d电子**：主族元素（无d电子）通常不形成经典反馈π键
2. **氧化态影响**：高氧化态金属d电子少，反馈π键弱；低氧化态金属反馈π键强
3. **与电负性相关**：配体原子电负性低（如C vs O），其空π*轨道能量低，更易接受电子
4. **不能定量**：竞赛层面只需定性判断，定量计算需要计算化学

## 七、常见比较与易混点

### 反馈π键 vs 正常π键
| | 反馈π键 (Back-bonding) | 正常π键 |
|:---|:---|:---|
| 电子方向 | 金属 → 配体 | 双方共享 |
| 涉及轨道 | 金属 d → 配体 π* | p–p 或 d–d 重叠 |
| 对配体键级 | **降低**（电子进入配体反键轨道）| 不变或升高 |

### 反馈π键 vs π-给体作用
| | 反馈π键 (π-acceptor) | π-给体 (π-donor) |
|:---|:---|:---|
| 电子方向 | 金属 → 配体 | 配体 → 金属 |
| 典型配体 | CO, CN⁻, PR₃ | Cl⁻, Br⁻, OH⁻, O²⁻ |
| 对Δ₀影响 | **增大**（t₂g降低）| **减小**（t₂g升高）|
| 光谱化学序列位置 | 强场端 | 弱场端 |

## 十、例题

### 例题1：判断配体类型

**题目**：判断下列配体是π-酸、σ-给体还是π-给体：CO、Cl⁻、H₂O、PR₃。

**解答**：
- **CO**：π-酸（强）。有空的π*轨道，强d→π*反馈
- **Cl⁻**：π-给体。Filled p轨道与金属d形成π键，给电子方向为配体→金属
- **H₂O**：σ-给体为主。无显著π相互作用
- **PR₃**：σ-给体 + 弱π-酸。P的空d轨道或σ*轨道可接受少量电子

### 例题2：解释Fe(CO)₅的稳定性

**题目**：Fe(0)的氧化态很低，为什么Fe(CO)₅却很稳定？

**解答**：
1. Fe(0)为d⁸构型，具有丰富的d电子
2. CO是强π-酸配体，空的π*轨道能量低
3. Fe→CO的d→π*反馈π键很强
4. 反馈π键与CO→Fe的σ给予协同作用，形成稳定的协同键（synergic bond）
5. 同时满足18电子规则：Fe(0)提供8e + 5×CO提供10e = 18e

## 十一、易错点

1. **认为所有含多重键的配体都是π-酸**：如C₂H₄是π-酸，但O₂通常不是（电负性高，π*能量高）
2. **混淆反馈π键与正常π键**：反馈π键电子方向是金属→配体，且进入配体的**反键**轨道
3. **忽略氧化态的影响**：Mn(CO)₆⁻比Mn(CO)₆⁺的反馈π键更强（前者金属更富电子）
4. **认为PR₃只有σ给体作用**：PR₃也有弱的π-接受能力（通过P的d轨道或σ*轨道）
5. **用反馈π键解释所有强场现象**：NH₃是强场但无显著反馈π键，主要靠σ给体作用

### 12.3 入门例题
**题目**：为什么CO配合物中C–O伸缩振动频率比自由CO低？

**答案**：金属d电子反馈到CO的π*反键轨道→C–O键级降低→键力常数减小→振动频率降低。

## 十三、竞赛拓展

1. ** Dewar-Chatt-Duncanson 模型**：烯烃与过渡金属配位的经典模型（σ给予 + π反馈）
2. **N₂活化与固氮**：通过强反馈π键削弱N≡N三键，是生物固氮和工业催化模拟的核心理论
3. **金属卡宾/卡拜配合物**：M=C和M≡C多重键中的强反馈π键成分
