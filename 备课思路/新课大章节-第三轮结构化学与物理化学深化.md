# 新课大章节：第三轮结构化学与物理化学深化

> **定位**：把第一轮的结构与原理从“会做基础题”提升到“会解释复杂体系、会看表征、会处理更深模型”的新课大章。
> **边界**：本章仍然属于系统新授层；第四轮的冲刺压缩、跨专题缝合和真题策略不提前并入本章。

## 一、本章定位

- 这章承担第三轮的结构波谱线和物化定量线。
- 当前知识库里这章的主干已闭环，但“桥梁层”还应继续显式补出。
- 与 `[[新课大章节-第三轮有机化学]]` 并列构成第三轮双主章。

## 二、主框架教材与辅助来源

| 层级 | 主要来源 | 作用 |
|:---|:---|:---|
| 主框架教材 | Atkins / 结构化学基础 / 第三轮深化框架 | 决定深化层顺序 |
| 第一辅助 | Zchem 中级无机结构、物化相关提炼 | 补解释路径和高阶桥梁 |
| 知识库主产物 | 晶体、配位、动力学、热平衡专题页 | 转化成课堂语言 |
| 媒体与图示 | 晶体投影图、配位分裂图、能量图、曲线图 | 建立高阶表征直觉 |
| 训练层 | 结构推断题、配位题、综合物化计算题 | 构成章后习题课主轴 |
