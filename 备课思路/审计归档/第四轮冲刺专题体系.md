| 趋势 | 具体表现 | 对第四轮的影响 |

## 专题体系（6 核心 + 2 可选）

### 4-1 🔥 高等有机：反应机理与轨道理论

**为什么放第一位**：2025 初赛有机转向"分子轨道角度解析机理"，决赛有机以机理推断为核心。

| 维度 | 内容 |
|:---|:---|
| **物理有机基础** | Hammett 方程（σ/ρ 值的含义与应用）、动力学同位素效应（KIE，$k_\mathrm{H}/k_\mathrm{D}$ 用于判断速控步是否涉及 C-H 断裂）、Curtin-Hammett 原理（产物比由过渡态能差决定，不由中间体浓度比决定）、**Bent 规则**（电负性取代基→中心原子杂化轨道 p 成分增加→其他键 s 成分增加→键角增大） |
| **轨道对称性控制** | Woodward-Hoffmann 规则的 FMO 推导（不背结论，从 HOMO-LUMO 对称性匹配推导）；周环反应选择性的统一解释（DA/电环化/σ迁移）；**给材料学习型机理推断**（从文献催化循环→独立推断完整机理路径，跟踪电子流向→识别中间体→应用 Baldwin 规则/电性匹配） |
| **反应选择性统一框架** | 化学选择性/区域选择性/立体选择性 的三层判据：电子效应（FMO 系数）→ 位阻效应 → 构象效应（Felkin-Anh、Cram 规则）；**构象→反应选择性的因果推理链**（如 α-氟代羰基 anti/syn 构象能差 → 反应面选择性） |
| **活性中间体深度** | 碳正离子的非经典结构（2-降冰片基阳离子）、碳负离子的软硬酸碱理论、卡宾的单线态/三线态反应差异、氮宾（Nitrene）的重排与插入、芳炔（Benzyne）的区域选择性；**自由基中间体**（攫氢选择性、HAT 迁移、β-碎裂消除） |
| **考纲映射** | 决赛08-反应机理、32-有机反应机理、34-活性中间体、49-周环反应 |

**关键人名反应（按机理族选代表，不求全）**：
- 碳正离子重排族：Wagner-Meerwein → Pinacol → Semipinacol（统一框架：缺电子中心的1,2-迁移）
- 碳负离子缩合族：Aldol → Claisen → Dieckmann → Knoevenagel（统一框架：烯醇负离子亲核进攻 + 脱水）
- 周环反应族：Diels-Alder → 1,3-偶极环加成 → Cope/Claisen重排 → 电环化（统一框架：Woodward-Hoffmann 规则）

### 4-2 🔥 有机立体化学与构象分析

**为什么独立成专题**：2025 初赛有机重心明确转向"立体化学分析"，是提分性价比最高的有机子方向。

| 维度 | 内容 |
|:---|:---|
| **构象分析深度** | 环己烷取代基的 A 值（-ΔG°）定量应用；中环（7-11元环）的跨环张力和 trans-annular 反应；阻转异构（Atropisomerism）——BINAP 类配体的轴手性来源与翻转能垒 |
| **动态立体化学** | 氮的锥形翻转（氨/胺的翻转能垒 ~25 kJ/mol vs 磷化氢 ~132 kJ/mol）；联苯类阻转异构的半衰期与温度关系；环翻转的低温 NMR 检测（$T_c$ 聚并温度） |
| **不对称合成策略** | Felkin-Anh 模型（α-手性醛酮的亲核加成）；Cram 螯合模型（α-烷氧基醛酮）；Evans 助剂（噁唑烷酮→烯醇化→烷基化→脱除，控制 α-位绝对构型）；Sharpless 不对称环氧化（(+)-DET vs (-)-DET 的面选择性） |
| **立体电子效应** | 反式共平面（anti-periplanar）要求：E2 消除、Grobfragmentierung、Beckmann 重排；端基效应（Anomeric Effect）：糖类 O-C-O 片段的 gauche 偏好 |
| **考纲映射** | 24-异构现象、25-互变异构、27-手性、28-结构稳定性、决赛08-反应机理 |

### 4-3 🔥 人名反应系统归类

**为什么不能零散讲**：第38届决赛有机试题解析明确指出"许多有机人名反应实质相似"，需要按机理族归类。

**分类体系（按中间体/机理族，非按字母顺序）**：

| 机理族 | 核心人名反应 | 统一机理 | 代表性应用 |
|:---|:---|:---|:---|
| **碳正离子1,2-迁移** | Wagner-Meerwein、Pinacol、Semipinacol、Tiffeneau-Demjanov、Wolff重排（经由酰基卡宾） | 缺电子中心 + 迁移基团1,2-迁移 → 更稳定碳正离子/羰基 | 甾体骨架重排、扩环/缩环 |
| **缺电子氮的重排** | Beckmann（肟→酰胺）、Hofmann（酰胺→胺）、Curtius（酰基叠氮→异氰酸酯→胺）、Lossen（羟肟酸→胺）、Schmidt（羧酸/酮+ HN₃→胺/酰胺） | 氮上缺电子 → R 基迁移到 N → 水解 | 羧酸→少一个碳的胺（Curtius）、酮→酰胺（Schmidt） |
| **碳负离子缩合** | Aldol（羟醛）、Claisen（酯缩合）、Dieckmann（分子内 Claisen）、Knoevenagel、Perkin、Stobbe | 烯醇负离子 + 亲电羰基 → β-羟基/β-酮酯 | C-C 键构建核心工具 |
| **极性翻转（Umpolung）** | 安息香缩合（Benzoin）、Stetter 反应、Corey-Seebach（1,3-二噻烷法） | 醛的极性翻转：亲电→亲核 | 1,4-二羰基化合物合成 |
| **协同周环** | Diels-Alder、Alder-ene、1,3-偶极环加成、Cope 重排、Claisen 重排、电环化 | 环状过渡态 + 芳香性过渡态（4n+2） | 六元环/五元环高效构建 |
| **氧化重排/插氧** | Baeyer-Villiger（酮→酯/内酯）、Dakin（芳醛→酚）、Rubottom（硅醚→α-羟基酮） | 过氧酸 O-插入 → 重排 | 酮的区域选择性氧化 |
| **光化学反应** | Norrish I（α-断裂）、Norrish II（γ-H 夺取）、Paternò-Büchi（羰基+烯→氧杂环丁烷）、di-π-methane 重排 | 激发态双自由基中间体 | 热反应无法实现的骨架重排 |
| **金属催化交叉偶联** | Suzuki（Pd, 硼酸）、Negishi（Pd/Ni, 有机锌）、Heck（Pd, 烯烃+卤代物）、Sonogashira（Pd/Cu, 端炔）、Buchwald-Hartwig（Pd, C-N 偶联） | 氧化加成→转金属化→还原消除 | C-C/C-N 键催化构建 |

**考纲映射**：36-取代、38-消除、39-重排、44-SN、45-亲核加成、46-羰基α位、48-膦化合物、49-周环、50-有机合成、51-金属有机、决赛08-反应机理、决赛11-金属有机化学

### 4-4 🔥 晶体结构深度与推断

**为什么从计算型→定性推理型**：2025 初赛第5题（18分）给出三个晶体结构图 → 推断化学组成 + 判断晶系，不再考传统堆积效率/密度计算。

| 维度 | 内容 |
|:---|:---|
| **结构推断核心技能** | 俯视图+侧视图→三维结构还原；**不同晶向投影图识别**（如 [112] 投影的对称性分析——2024/2025 新题型）；共顶点多面体（如共顶点 TO₆ 八面体）的连接方式识别；晶胞中独立原子的不等价格位判断；从配位数反推结构类型（NaCl型/CsCl型/ZnS型/萤石型/钙钛矿型） |
| **晶系判断** | 从晶胞参数（a,b,c,α,β,γ）快速判断七大晶系；对称元素（镜面/旋转轴/反演中心）→点群→晶系的逻辑链 |
| **典型复杂结构** | 钙钛矿（ABX₃）——容忍因子 $t = (r_A + r_X)/\sqrt{2}(r_B + r_X)$ 与结构畸变；尖晶石（AB₂O₄）——正/反尖晶石的 CFSE 判据；萤石/反萤石——CaF₂ vs Na₂O；石榴石（A₃B₂Si₃O₁₂）——多面体共顶点连接；**MAX相/MXene**（层状三元碳化物→选择性刻蚀→二维材料） |
| **缺陷化学** | Frenkel 缺陷（等量空位+间隙）vs Schottky 缺陷（等量阴/阳离子空位）；Kröger-Vink 记号法；非化学计量化合物（如 Fe₁₋ₓO）的缺陷簇；离子导体的缺陷介导扩散；**氧空位的位置判定**（从投影图+对称性反推——2024 CeO₂ 题型） |
| **XRD 基础** | Bragg 方程：$n\lambda = 2d\sin\theta$；立方晶系指标化：$\sin^2\theta \propto (h^2+k^2+l^2)$；系统消光规律：P vs I vs F vs C 的反射条件 |
| **考纲映射** | 14-对称性、15-晶体结构、16-晶体基本概念、17-常见晶体结构、决赛01-原子结构补充、决赛02-分子结构补充 |

### 4-5 🔥 配合物与配位化学深化

**为什么独立成专题**：决赛03和决赛12都要求配合物深化，第38届决赛涉及稀土配合物的催化作用。

| 维度 | 内容 |
|:---|:---|
| **配体场理论统一** | 晶体场（静电模型）→ 配体场（引入共价成分，MO-LCAO）→ 角重叠模型（AOM，定量预测 d 轨道分裂）；σ给体 / π给体 / π受体 三类配体的光谱化学序列差异（如 CO 是强场 π 受体，I⁻ 是弱场 π 给体） |
| **Jahn-Teller 效应** | 一级 JT（基态简并，如Cu²⁺ d⁹）vs 二级 JT（激发态与基态耦合）；拉长 vs 压扁八面体的能量比较（拉长更常见）；JT 效应对 ESR 谱和电子光谱的影响 |
| **磁性深度** | 自旋态交叉（SCO）：Fe²⁺ 配合物在温度/压力/光照下的高自旋↔低自旋转变；反铁磁耦合：Cu₂(OAc)₄·2H₂O 的双核结构，$J$ 值与磁化率公式；分子磁性材料（单分子磁体 SMM，如 Mn₁₂-acetate）；**d 轨道分裂的精细分析**（如直线形 Co²⁺ 配合物中 d 轨道三组能级的电子分布几率——2025 决赛 SMM 题） |
| **σ/π 成键轨道分析** | **轨道级别的成键标注**：具体到"哪个 d 轨道 + 哪个配体轨道 → 什么对称性的 σ/π 键"（如 Co 的 dₓ₂₋ᵧ₂ 搭配 N 的 2s/2pₓ 形成 σ 键；Co 的 dₓᵧ 搭配 N 的 2pᵧ 形成 π 键——2025 决赛 Co 配合物题）；σ 给体/π 给体/π 受体三类配体的成键轨道组合图 |
| **稀土配合物** | 4f 轨道收缩（不参与成键）→ 配体场分裂极小 → 磁性主要由 f 电子决定，光谱呈线状；Eu³⁺/Tb³⁺ 的发光（天线效应）；Gd³⁺ 的 MRI 造影剂；Ce⁴⁺/Ce³⁺ 的氧化还原催化 |
| **配合物反应动力学** | 惰性（inert）vs 不稳定（labile）——与热力学稳定性的区分；Taube 的分类（d³ Cr³⁺、低自旋 d⁶ Co³⁺ 为惰性）；配体取代的 D 机理 vs Iₐ/I_d 机理；反位效应（trans effect）：Pt²⁺ 配合物的反位效应序列 |
| **考纲映射** | 12-配合物、决赛03-配合物深化、决赛12-配合物、决赛11-金属有机化学 |

### 4-6 🔥 元素化学深度 + 结构推断综合

**为什么从背诵→推断转型**：2025 初赛元素化学压缩，转向"给出陌生物质 → 推导结构"；第38届决赛以 Be-Be 键配合物入题。

| 维度 | 内容 |
|:---|:---|
| **无机簇合物与多酸** | 多金属氧酸盐（POM）：Keggin 型 [XM₁₂O₄₀]ⁿ⁻、Dawson 型 [X₂M₁₈O₆₂]ⁿ⁻ 的结构与合成；Zintl 相：碱金属/碱土金属与后过渡金属/半金属形成的簇阴离子（如 [Sn₉]⁴⁻、[Pb₉]⁴⁻）；金属-金属多重键：Cr₂(OAc)₄·2H₂O（四重键：σ+2π+δ） |
| **稀土与锕系** | 镧系收缩：原子/离子半径从 La→Lu 递减 ~15 pm，导致 Zr/Hf、Nb/Ta 原子半径几乎相等；Ce⁴⁺ 的异常稳定性（f⁰ 电子构型）→ 强氧化剂；UO₂²⁺（铀酰离子）的直线结构和配位化学 |
| **铂族金属化学** | Pt/Pd/Rh/Ir 的配合物催化循环；Wilkinson 催化剂（RhCl(PPh₃)₃）的加氢机理；Vaska 配合物（IrCl(CO)(PPh₃)₂）的氧化加成反应 |
| **非经典化学键** | 三中心两电子键（B₂H₆ 的桥氢）；Be-Be 金属-金属键（第38届决赛入题）；主族多重键（P=P、Si=Si 的空间位阻稳定化策略） |
| **热力学与元素推断融合** | Born-Haber 循环用于判断未知氧化态的稳定性（如 CuI₂ vs CuI 的 ΔH_f 对比）；Latimer/Frost 图用于判断歧化/归中；Ellingham 图用于判断金属氧化物还原温度 |
| **考纲映射** | 13-元素化学、决赛01/02/03 |
