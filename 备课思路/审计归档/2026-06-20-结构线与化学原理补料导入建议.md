## 二、按专题看的真实缺口

## 2.1 第一轮化学原理主线

### 已经有主干，但还可以补厚的部分

- `气体 / 溶液`：
  - 普化主干已在位，但 `Henry / Raoult / 依数性 / 活度直觉` 仍偏“列名词”，还没充分借助 Atkins 的混合物视角把“为什么成立、何时失效”讲透。
- `热力学 / 平衡 / 电化学`：
  - 普化够用，但还缺“教师看的更高视角解释层”，例如化学势、Van't Hoff、Nernst 与平衡常数之间的统一叙述。
- `原子结构 / 分子结构`：
  - 课堂主线已基本形成，但“反常结构”“离域解释”“MO 与性质联动”的素材还可以更系统地从 Atkins 量子结构部分补入。

## 2.2 第一轮结构线

### 当前最值得补的不是“再写更多结论”，而是四类桥梁

1. **对称性桥梁**
- 目前课堂正文已经能讲到 VSEPR、杂化、MO，但“为什么多原子 MO 和配位场要引入对称性”这层桥还不够稳。
- `[[提炼-Atkins物理化学-主题10-分子对称性]]` 已明确：现有 `[[分子对称性初步]]` 大体覆盖 10A，但 10B/10C（群论基础、特征标表、SALC、选律）仍属于高价值增量。

2. **晶体结构桥梁**
- 第一轮晶体页已能讲结构类型、堆积、半径比、极化修正，但对 `Miller 指数 / Bragg 方程 / 结构因子 / 系统消光 / 能带图像` 还没有稳定的教师参考层。
- `[[提炼-Atkins物理化学-主题15-固体]]` 已明确把这些内容标为高优。

3. **配位理论桥梁**
- 第一轮配合物页目前可以到命名、构型、异构、高低自旋、基础 LFSE，这已经不错。
- 但如果想让教师备课时更踏实，还缺：
  - `π 给体 / π 受体` 对 `Δ₀` 的影响
  - `Tanabe-Sugano 图`
  - `电荷转移带 / 颜色强度 / Laporte 与自旋选律`
  - `电子云重排效应`
- 这些内容在 `[[提炼-无机化学第6版-第20章-配位化学理论]]` 中已经被明确提炼出来，但尚未完全沉到知识库主链。

4. **元素化学—结构解释桥梁**
- 过渡金属、主族元素很多“为什么会这样”的课堂解释，其实要借助结构化学。
- 例如：
  - Co(III) 络合物为什么稳定
  - Cu(II) 为什么常见 Jahn-Teller 畸变
  - Prussian blue 为什么适合做经典例子
  - AgI 为什么从半径比预测偏离到 ZnS 型
- 这类桥梁在 `[[提炼-无机化学第6版-第19章-d区金属]]` 中已有大量材料，但还没有系统变成专题桥。

## 2.3 第三/四轮与决赛预备层

### 现在缺的已经不主要是“有没有页”，而是“有没有和竞赛题源接上”

- 本地复核后需要纠偏：
  - `[[Tanabe-Sugano图]]`
  - `[[多金属氧酸盐]]`
  - `[[金属-金属多重键]]`
  - `[[晶体结构深化]]`
  - `[[分子对称性初步]]`
  这些核心页并不是完全空白，而是**已经存在**。
- 当前真正偏弱的，是它们与以下三层之间的耦合仍不够紧：
  - `IChO syllabus` 的深度边界
  - `IChO preparatory problems / competition problems` 的题型母源
  - `04-课件/备课大纲 + 新授课` 的教师调用入口
- 因此后续动作不应再简单理解成“新建几个专题页”，而应理解为：
  - 给现有高价值页补 `竞赛例题入口`
  - 给备课页补 `引用这些高价值页的明确位置`
  - 给第一轮/第三轮/第四轮分别标清“只做入口 / 可展开 / 决赛深化”的层级边界

## 3.1 优先级 A：应优先继续抽，不需要先找新书

### A1. `Arrow Pushing in Inorganic Chemistry`

**本地已核实**：
- 该资料已存在于 `06-外部资料导入/书籍/`，且**已经做过总索引模式提炼**，并回流到一批 `03-知识点/有机化学` 与 `03-知识点/无机和结构化学` 页面。
- 但它和当前这套“第一轮/第三轮课堂调用链”的衔接仍不算完全收口，尤其是：
  - 主族反常结构如何服务第一轮结构线
  - 电子流语言如何服务第二轮主族元素化学
  - 哪些内容适合作为教师解释层，哪些适合直接进入学生讲义

**为什么高优**：
- 它能补上当前体系里最缺的一层：**无机/主族反应里的“电子流语言”**。
- 对以下内容特别有价值：
  - Lewis 酸碱与配位成键的动态理解

  - 主族元素的成键与反应机理解释
- 这会直接提升：
  - 第一轮 `分子结构基础`
  - 第二轮 `主族元素化学`
  - 后续 `结构—反应桥梁` 的讲法

**建议产出**：
- 当前不必重做整本提炼，而应优先补“课堂调用链”
- 再集中补两类页/桥梁：
  - `主族反应电子流基础`

## 四、网页核验：外部课程给出的信号

## 4.1 MIT OCW 的固体化学课程说明了什么

**网页核验来源**：
- [MIT OpenCourseWare: 3.091SC Introduction to Solid-State Chemistry](https://ocw.mit.edu/courses/3-091sc-introduction-to-solid-state-chemistry-fall-2010/)

**读到的关键信号**：
- 这门官方课程把以下内容放在同一条主链里：
  - chemical bonding
  - crystal structures
  - x-ray diffraction
  - semiconductors
  - point defects
  - electrical properties
  - phase diagrams
- 这说明“晶体结构”如果只停在 NaCl/CsCl/ZnS 和记忆结构类型，广度仍然偏窄；更成熟的课程会自然把**衍射、缺陷、能带、半导体与相图**视为同一大块固体化学语言。

**对当前知识库的意义**：
- 你的结构线后续完全可以把“晶体结构”往更完整的“固体化学”方向拉，而不是永远只停在竞赛结构判断。
- 这并不意味着第一轮要讲完全部，但意味着教师侧参考链最好提前补好。

## 4.2 MIT 官方参考书目带来的具体导入线索

**网页核验来源**：
- [MIT OCW 3.091SC Reference Materials](https://ocw.mit.edu/courses/3-091sc-introduction-to-solid-state-chemistry-fall-2010/pages/reference-materials/)

**网页里明确列出的参考方向**：
- `Shackelford, Introduction to Materials Science for Engineers`
- `Cullity & Stock, Elements of X-Ray Diffraction`

**对知识库的直接启发**：
- 如果要给 `晶体结构深化 / XRD / 固体物性` 补料，`Cullity & Stock` 一类 XRD 资料非常值得导入。
- 如果要把“结构—缺陷—电学—材料性质”的语言补宽，`Shackelford` 这种材料学导论类教材也可以作为辅助，不一定逐章深挖，但非常适合补“教师视角解释层”。

> 说明：这里是**网页核验到的官方课程参考书**，不是我主观猜测。

- `Weller`
- `Atkins`
- `Arrow Pushing in Inorganic Chemistry`

> 也就是说，**竞赛相关资料负责告诉你“会怎么考、会怎样组合”；教材负责告诉你“为什么、怎么讲清楚”。**

## 5.4 IChO syllabus 对当前知识库的首批映射

> 这里的目的不是做逐条考纲翻译，而是把**最值得先抽的 IChO Level 3 高价值点**对应到你当前最需要补强的知识链。

| IChO 纲目主轴 | 官方信号 | 当前最该补强的本地页 | 应用方向 |
| --- | --- | --- | --- |
| `2.1.2 / 2.2 / 2.3 / 2.4` 超价、离域、杂化、MO、顺磁性 | 结构不只停在 VSEPR，明确要求 `octet rule` 例外、离域和 O₂ 系 MO | `[[分子结构基础]]`、`[[分子对称性初步]]`、后续 `主族反应电子流基础` | 第一轮结构线解释层 + 第二轮主族结构反应桥梁 |
| `5.6.4 / 5.6.5` 配位场、高低自旋、配位立体异构 | 配位不是只会命名，官方直接把 `ligand field theory` 和 `enantiomers` 列为 3 级 | `[[配合物]]`、`[[配位场理论]]`、`[[Tanabe-Sugano图]]` | 第一轮教师解释层 + 第三/四轮配位冲刺 |
| `6.2 / 6.3 / 6.4 / 7` 热力学、平衡、电化学、动力学 | `ΔG-K`、`Nernst`、`Henry/Raoult`、速率方程与机理都在纲目里有明确 3 级项目 | `[[热力学基础]]`、`[[化学平衡]]`、`[[电化学基础]]`、`[[化学动力学基础]]` | 第一轮普化主干补厚 |
| `8.3` X-ray、Bragg、unit cell、NaCl/CsCl/金属 | 官方把 `Bragg's Law`、`coordination number`、`unit cell`、典型固体结构并列 | `[[晶体结构基础]]`、`[[晶体结构深化]]` | 第一轮晶体线 + 第三轮结构深化 |
| `8.4 / 8.5 / 9` NMR、MS、有机反应族 | 光谱与有机结构确证是正式纲目的一部分，不是可有可无的附带内容 | 第三轮 `有机波谱与结构确证` 主链相关页 | 第三轮结构确证与合成设计 |

## 5.5 IChO 资料导入顺序建议

### 第一批：最该先抽的，不是年份最多，而是最能反推课堂缺口的四条线

1. `结构 / 主族成键 / MO`
- 先用 `IChO syllabus` 校正深度上限，再配 `Arrow Pushing in Inorganic Chemistry`
- 目标不是加很多稀奇内容，而是把“超价、离域、电子流、主族反常结构”讲顺

2. `配位 / 光谱 / 颜色`
- 先用 `IChO syllabus` + `preparatory problems` 看题型组合，再回灌 `Weller Ch20`
- 重点不是机械加知识点，而是让 `高低自旋 -> 颜色 -> 跃迁归属 -> TS 图` 形成一条教师能调用的链

3. `晶体 / XRD / 固体结构`
- 先用 `IChO syllabus` 确认 `Bragg / unit cell / NaCl-CsCl-金属结构` 是正式竞赛边界
- 再让 `Atkins 15` 和后续 `Cullity` 候选资料补齐教师解释层

4. `热力学 / 平衡 / 电化学 / 动力学`
- 先抽 `IChO Level 3` 里最常见的统一语言：`ΔG-K`、`Nernst`、`Henry/Raoult`、积分速率式
- 目标是让第一轮化学原理真正达到“竞赛主干课”而不是“名词入口课”

### 第二批：更适合做第三/四轮桥梁的竞赛专题

5. `有机光谱与结构确证`
- 围绕 `NMR / IR / UV / MS` 的 IChO 纲目和真题组合，继续补第三轮结构确证主链

6. `d区结构—性质解释`
- 用历届 prep/problems 中的 `颜色、磁性、氧化态、配位环境` 组合题，反推元素化学页如何补结构解释

7. `多酸 / 金属-金属键 / 内外层电子转移`
- 这类更像第四轮或决赛预备层，不该先挤占第一轮主线时间，但教师侧应提前建好题源挂点

## 5.6 对 IChO preparatory problems 的实际使用建议

- 先把它当成“**教师备课反查库**”，不是立刻全量 OCR 的“题库工程”
- 第一轮先查四类题：
  - `Bragg / 晶胞 / 结构类型`
  - `高低自旋 / 颜色 / 配位平衡`
  - `Henry / Raoult / 相图 / 稀溶液`
  - `ΔG / K / E / 速率式` 的联动题
- 第三轮再查两类题：
  - `NMR / IR / MS / UV` 结构确证
  - `有机机理 + 合成设计`
- 第四轮或决赛预备层再查：
  - `多酸 / 金属-金属键 / 复杂配位光谱`

> 这样做的好处是：先把 IChO 当作“判断哪些内容值得进入教师备课链”的上游校准器，而不是一上来被海量题目拖进 OCR/整理泥潭。

**理由**：

- 并且这类内容与 IChO 历届 prep/problems 中的晶体与结构题天然同频，属于“竞赛相关性很高”的补料，而不是偏科研的无关扩展。

### 2. 固体化学 / 材料化学导论

**建议对象**：
- 与 MIT 参考链一致的材料学导论类教材
- 若只能先导一套，优先选“晶体结构 + 缺陷 + 能带 + 相图”兼顾的版本

**理由**：
- 目前你的晶体线已经有“结构判断”，但还缺“结构如何连到物性”的系统材料。

## 6.2 第二档：高价值，但属于我的建议性判断

> 这一档是**我的推荐**，不是本次网页直接核验到的官方指定书单。

### 3. 固体化学化学向教材

**建议对象**：
- `A. R. West, Solid State Chemistry and Its Applications`

**理由**：
- 相比材料学导论，它更偏化学语言，特别适合把 `晶体结构 / 缺陷 / 非化学计量 / 离子导体 / 能带` 统一到化学课堂里。
- 如果后续要做“老师看的一体化结构化学参考”，这本书比单纯材料学导论更贴近你的体系。

### 4. 群论 / 分子对称教材

**建议对象**：
- `Cotton, Chemical Applications of Group Theory`
- 或更易讲授的 `Molecular Symmetry and Group Theory` 类教材

**理由**：
- 当前 Atkins 10 已经足够做群论入口，但若要把 SALC、特征标表、光谱选律讲得更稳，仍需要一本更聚焦群论应用的参考书。

### 5. 结构化学教学型无机教材

**建议对象**：
- `Housecroft & Sharpe` 或 `Miessler/Fischer/Tarr` 一类教学型无机教材

**理由**：
- Weller 很强，但它更像“全景+丰富案例”的老师用书。
- 如果想补“课堂讲法”和“概念组织的顺滑度”，再配一本教学型无机教材会更舒服。

**注意**：
- 这类书不急着马上导入；优先级仍低于把 Weller/Atkins/Arrow Pushing 吃透。
