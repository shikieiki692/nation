# 新课大章节：第二轮分析化学

> **定位**：用“样品如何被定量”这条流程线，把误差、基准物、标准溶液、四大滴定、重量分析和分光光度法组织成一章。
> **边界**：本章讲到竞赛所需的经典分析主线，不提前系统展开更深的仪器分析。

## 一、本章定位

- 这章不是“几种滴定法的并列合集”，而是一条测量流程线。
- 第一轮的平衡、电势、酸碱和沉淀语言，在这里变成真正的定量工具。
- 与 `[[新课大章节-第二轮元素化学]]` 并列构成第二轮双主章。

## 二、主框架教材与辅助来源

| 层级 | 主要来源 | 作用 |
|:---|:---|:---|
| 主框架教材 | 《分析化学》第六版相关章节 | 决定测量流程和章节顺序 |
| 第一辅助 | 第二轮相关专题、滴定真题、误差与光度法资料 | 补题型和课堂化表达 |
| 知识库主产物 | 分析专题页、KP、教学洞察 | 转化成课堂语言 |
| 媒体与图示 | 滴定曲线、误差示意、标准曲线 | 帮学生建立测量直觉 |
| 训练层 | 滴定综合题、误差题、Lambert-Beer 题 | 构成章后习题课主轴 |

## 五、章后复习课骨架

**建议文件名**

- `第二轮分析化学复习.md`

**建议目标**

- 梳理四大滴定、误差、光度分析的共同测量逻辑
- 对比终点、突跃、条件常数、误差来源
- 建立“样品 -> 方法 -> 曲线 / 数据 -> 结果”的统一视角

## 六、章后习题课骨架

**建议文件名**

- `第二轮分析化学习题课.md`

**建议目标**

- 训练滴定综合计算、误差判别、标准曲线判读
- 强化“先判断体系，再选方法”的做题顺序
- 为更高轮次的分析/实验思维做预热
