# 新课大章节：第三轮有机化学

> **定位**：第三轮最大的新课母章。负责把结构基础、电子效应、立体化学、机理、反应族和合成策略组织成一条真正的系统课程。
> **边界**：本章目标是“初赛题全会 + 部分决赛题有入口”，不把第四轮物理有机冲刺层与全合成层提前平铺进来。

## 一、本章定位

- 这章不该写成“官能团百科”，而要保持“结构 -> 机理 -> 选择性 -> 合成”的主链。
- 这是后续复习课、习题课、速查卡最需要章群化管理的一章。
- 与 `[[新课大章节-第三轮结构化学与物理化学深化]]` 并列构成第三轮双主章。

## 二、主框架教材与辅助来源

| 层级 | 主要来源 | 作用 |
|:---|:---|:---|
| 主框架教材 | 《基础有机化学》 + Clayden | 决定系统顺序与机理深度 |
| 第一辅助 | ABOC、学而思/Zchem 有机提炼 | 补讲法、补判断链、补例题 |
| 知识库主产物 | 有机专题页、教学洞察、真题链 | 转化成课堂语言 |
| 媒体与图示 | 机理图、构象图、选择性流程图 | 解决电子流和空间关系可视化 |
| 训练层 | 机理题、结构推断题、路线设计题 | 构成章后习题课主轴 |
