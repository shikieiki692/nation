# 新课大章节：第一轮结构化学初步

> **定位**：第一轮结构语言母章。负责建立“原子 -> 分子 -> 晶体 -> 配位”的连续结构链，是后续讲义样板最重要的一章。
> **边界**：本章允许达到《无机化学》常规结构章节的难度，但不提前铺开群论、Tanabe-Sugano、完整配位场 MO 等第三轮内容。

## 一、本章定位

- 这是当前知识库里最适合继续做母版群的一章。
- 后续其他讲义若要“对标原子结构水平”，优先参考本章的成组样板，而不是单篇模仿。
- 与 `[[新课大章节-第一轮化学原理]]` 并列构成第一轮双主章。

## 二、主框架教材与辅助来源

| 层级 | 主要来源 | 作用 |
|:---|:---|:---|
| 主框架教材 | 《普通化学原理》结构相关章节 | 提供章节骨架 |
| 第一辅助 | 《无机化学》、初赛讲义、上海中学资料 | 补图景、补解释、补结构例题 |
| 知识库主产物 | 原子/分子/晶体/配位专题页与教学洞察 | 转化成课堂语言 |
| 媒体与图示 | 结构图、MO 图、晶胞图、Excalidraw | 让抽象空间关系可视化 |
| 训练层 | 结构推断题、磁性题、晶胞题、配位题 | 构成章后习题课主干 |
